Détectez les modèles malveillants dans les compétences AI Agent : 13 détecteurs de portes dérobées, de vol d'identifiants, d'exfiltration de données et d'attaques de la chaîne d'approvisionnement. Basé sur les renseignements sur les menaces ClawHub de SlowMist (plus de 472 compétences malveillantes). Python pur, zéro dépendance.
