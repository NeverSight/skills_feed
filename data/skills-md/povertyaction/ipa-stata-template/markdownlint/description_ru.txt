Этот навык следует использовать, когда пользователям необходимо отформатировать, очистить, проверить или проверить файлы Markdown с помощью инструмента командной строки markdownlint-cli2. Используйте этот навык для задач, связанных с проверкой качества файлов Markdown (включая Quarto Markdown `.qmd`), автоматическим исправлением форматирования, соблюдением правил стиля Markdown или выявлением проблем синтаксиса Markdown.
