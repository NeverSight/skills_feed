Comprendre les principes fondamentaux, l'architecture et les concepts de base du protocole ContextVM. À utiliser lorsque les utilisateurs ont besoin d'en apprendre davantage sur les bases de ContextVM, sur la façon dont il relie MCP avec Nostr, sur les principes de conception de protocole, sur les types d'événements ou sur la relation entre MCP et Nostr dans une communication décentralisée.
