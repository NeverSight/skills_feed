أنشئ تطبيقات وكيلة باستخدام GitHub Copilot SDK. يُستخدم عند تضمين وكلاء الذكاء الاصطناعي في التطبيقات، أو إنشاء أدوات مخصصة، أو تنفيذ استجابات التدفق، أو إدارة الجلسات، أو الاتصال بخوادم MCP، أو إنشاء وكلاء مخصصين. المشغلات على Copilot SDK، وGitHub SDK، والتطبيق الوكيل، وتضمين Copilot، والوكيل القابل للبرمجة، وخادم MCP، والوكيل المخصص.
