Exécutez des audits Lighthouse sur des sites Web à l'aide de l'API Foghorn. À utiliser lorsque l'utilisateur souhaite vérifier les performances, l'accessibilité, les meilleures pratiques ou les problèmes de référencement d'un site. Couvre l'inscription, l'authentification, la configuration de l'équipe/du site et la récupération des problèmes.
