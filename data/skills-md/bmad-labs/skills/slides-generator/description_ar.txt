أنشئ شرائح عرض تقديمي تفاعلية باستخدام React + Tailwind، وقم بالتصدير إلى ملف HTML مستقل بذاته. يتم تشغيلها عند استخدام كلمات رئيسية مثل "شرائح"، أو "عرض تقديمي"، أو "PPT"، أو "عرض توضيحي"، أو "معياري"، أو عندما يطلب المستخدم التصدير. يستخدم مهارة متصفح الوكيل للتحقق من المتصفح قبل التصدير (التثبيت باستخدام مهارات npx، أضف vercel-labs/agent-browser إذا لم يكن متاحًا).
