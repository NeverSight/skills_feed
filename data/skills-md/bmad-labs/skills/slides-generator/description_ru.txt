Создавайте интерактивные слайды презентаций с помощью React + Tailwind и экспортируйте их в отдельный HTML-файл. Запускается по таким ключевым словам, как «слайды», «презентация», «PPT», «демо», «тестовый показатель» или когда пользователь запрашивает экспорт. Использует навык агента-браузера для проверки браузера перед экспортом (установите с помощью `npxkills add vercel-labs/agent-browser`, если он недоступен).
