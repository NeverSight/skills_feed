Générez des diapositives de présentation interactives à l'aide de React + Tailwind et exportez-les vers un fichier HTML autonome. Se déclenche sur des mots-clés tels que « diapositives », « présentation », « PPT », « démo », « benchmark » ou lorsque l'utilisateur demande une exportation. Utilise la compétence agent-navigateur pour la vérification du navigateur avant l'exportation (installer avec `npx skills add vercel-labs/agent-browser` s'il n'est pas disponible).
