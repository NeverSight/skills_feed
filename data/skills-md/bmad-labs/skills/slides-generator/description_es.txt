Genere diapositivas de presentación interactivas usando React + Tailwind y expórtelas a HTML de un solo archivo independiente. Se activa con palabras clave como "diapositivas", "presentación", "PPT", "demostración", "punto de referencia" o cuando el usuario solicita la exportación. Utiliza la habilidad agente-navegador para la verificación del navegador antes de la exportación (instale con `npx skills add vercel-labs/agent-browser` si no está disponible).
