Конвертируйте книги EPUB в высококачественный формат Markdown с помощью Pandoc и форматирования с помощью искусственного интеллекта. Используйте, когда пользователь предоставляет путь к файлу EPUB и хочет преобразовать его в профессионально отформатированный Markdown, аналогично форматированию коллекции чистого кода. Этот навык позволяет выполнять весь рабочий процесс: от извлечения EPUB до форматирования контента с помощью искусственного интеллекта, включая исправление артефактов преобразования PDF, объединение разделенных абзацев, исправление блоков кода, стандартизацию заголовков и создание правильного оглавления.
