Konvertieren Sie EPUB-Bücher mithilfe von Pandoc und KI-gestützter Formatierung in hochwertig formatiertes Markdown. Verwenden Sie diese Option, wenn der Benutzer einen EPUB-Dateipfad angibt und ihn in professionell formatiertes Markdown konvertieren möchte, ähnlich der Formatierung der Clean Code Collection. Diese Fähigkeit übernimmt den gesamten Arbeitsablauf von der EPUB-Extraktion bis hin zur KI-gesteuerten Inhaltsformatierung, einschließlich der Behebung von PDF-Konvertierungsartefakten, dem Zusammenfügen geteilter Absätze, der Korrektur von Codeblöcken, der Standardisierung von Überschriften und der Erstellung eines geeigneten Inhaltsverzeichnisses.
