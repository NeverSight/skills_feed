Convierta libros EPUB a Markdown formateado de alta calidad utilizando pandoc y formato asistido por IA. Úselo cuando el usuario proporciona una ruta de archivo EPUB y desea convertirlo a Markdown con formato profesional, similar al formato Clean Code Collection. Esta habilidad maneja el flujo de trabajo completo desde la extracción de EPUB hasta el formateo de contenido impulsado por IA, incluida la reparación de artefactos de conversión de PDF, la unión de párrafos divididos, la corrección de bloques de código, la estandarización de encabezados y la creación de una tabla de contenido adecuada.
