مبادئ التعليمات البرمجية النظيفة والممارسات المهنية وسير العمل لمطوري TypeScript. استنادًا إلى كتابي "Clean Code" و"The Clean Coder" لروبرت سي. مارتن.

استخدم هذه المهارة عندما:
- كتابة وظائف أو فئات أو وحدات نمطية جديدة لـ TypeScript/JavaScript
- مراجعة الكود أو طلبات السحب
- إعادة هيكلة التعليمات البرمجية الموجودة
- بعد التطوير القائم على الاختبار (TDD)
- إصلاح الخلل مع تغطية الاختبار المناسبة
- تخطيط استراتيجية اختبار الميزات
- تقدير المهام بدقة
- التعامل مع المواعيد النهائية والالتزامات بشكل احترافي
- العمل بفعالية مع الفرق
