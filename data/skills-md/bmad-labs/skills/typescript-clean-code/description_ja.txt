TypeScript 開発者のためのクリーン コードの原則、プロフェッショナルな実践、ワークフロー。 Robert C. Martin の書籍「Clean Code」および「The Clean Coder」に基づいています。

このスキルは次の場合に使用します。
- 新しい TypeScript/JavaScript 関数、クラス、またはモジュールを作成する
- コードまたはプルリクエストのレビュー
- 既存のコードのリファクタリング
- テスト駆動開発 (TDD) 後
- 適切なテストカバレッジによるバグの修正
- 機能のテスト戦略の計画
- タスクを正確に見積もる
- 期限と約束を専門的に処理する
- チームと効果的に連携する
