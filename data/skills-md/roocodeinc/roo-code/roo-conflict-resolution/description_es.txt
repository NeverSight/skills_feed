Proporciona pautas integrales para resolver conflictos de fusión de manera inteligente utilizando el historial de git y el contexto de confirmación. Úselo cuando las tareas impliquen conflictos de fusión, rebase, conflictos de relaciones públicas o resolución de conflictos de git. Esta habilidad analiza los mensajes de confirmación, la culpa de git y la intención del código para tomar decisiones de resolución inteligentes.
