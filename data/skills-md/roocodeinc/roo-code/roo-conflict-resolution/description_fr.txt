Fournit des directives complètes pour résoudre intelligemment les conflits de fusion à l’aide de l’historique git et du contexte de validation. À utiliser lorsque les tâches impliquent des conflits de fusion, un rebasage, des conflits PR ou une résolution de conflits git. Cette compétence analyse les messages de validation, le blâme git et l'intention du code pour prendre des décisions de résolution intelligentes.
