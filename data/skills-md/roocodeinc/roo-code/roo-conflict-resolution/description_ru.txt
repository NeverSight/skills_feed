Предоставляет подробные рекомендации по разумному разрешению конфликтов слияния с использованием истории git и контекста фиксации. Используйте, когда задачи включают конфликты слияния, перебазирование, конфликты PR или разрешение конфликтов git. Этот навык анализирует сообщения коммитов, обвинения git и намерения кода, чтобы принимать разумные решения по разрешению проблем.
