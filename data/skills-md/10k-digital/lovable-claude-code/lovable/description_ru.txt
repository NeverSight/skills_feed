- Проекты Lovable.dev с синхронизацией с GitHub.
- Функции Supabase Edge, требующие развертывания.
- Миграция базы данных для Lovable Cloud
- Проекты со структурой супабазы/каталогов
- Любое упоминание о «Lovable», «развертывании пограничной функции», «применении миграции».

Предоставляет точные подсказки Lovable для серверных операций, которые невозможно выполнить только через GitHub.
