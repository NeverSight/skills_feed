Guide de migration de bout en bout de Vercel vers Cloudflare Workers. À utiliser lors de la migration de n'importe quelle application (Next.js, React SPA, Astro, Remix, sites statiques, routes API) de l'hébergement Vercel vers Cloudflare Workers. Couvre l'évaluation du projet, la configuration de Wrangler, OpenNext pour Next.js, la migration du stockage (KV, D1, R2, Hyperdrive), les variables d'environnement, le basculement DNS, l'optimisation des images, la compatibilité Node.js, l'optimisation de la taille du bundle et le dépannage des erreurs de migration courantes. Déclencheurs sur : migrer depuis Vercel, passer à Cloudflare, Vercel vers Workers, déployer Next.js sur Cloudflare, remplacer Vercel par Cloudflare ou toute tâche de migration de Vercel vers Cloudflare.
