End-to-End-Migrationsleitfaden von Vercel zu Cloudflare Workers. Zur Verwendung bei der Migration beliebiger Anwendungen (Next.js, React SPA, Astro, Remix, statische Sites, API-Routen) vom Vercel-Hosting zu Cloudflare Workers. Behandelt Projektbewertung, Wrangler-Konfiguration, OpenNext für Next.js, Speichermigration (KV, D1, R2, Hyperdrive), Umgebungsvariablen, DNS-Cutover, Image-Optimierung, Node.js-Kompatibilität, Bundle-Größenoptimierung und Fehlerbehebung bei häufigen Migrationsfehlern. Auslöser bei: Migration von Vercel, Wechsel zu Cloudflare, Vercel zu Workers, Bereitstellung von Next.js auf Cloudflare, Ersetzen von Vercel durch Cloudflare oder einer beliebigen Migrationsaufgabe von Vercel zu Cloudflare.
