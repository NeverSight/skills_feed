Vercel から Cloudflare Workers へのエンドツーエンドの移行ガイド。アプリケーション (Next.js、React SPA、Astro、Remix、静的サイト、API ルート) を Vercel ホスティングから Cloudflare Workers に移行するときに使用します。プロジェクトの評価、ラングラー構成、OpenNext for Next.js、ストレージ移行 (KV、D1、R2、Hyperdrive)、環境変数、DNS カットオーバー、イメージの最適化、Node.js の互換性、バンドル サイズの最適化、および一般的な移行エラーのトラブルシューティングについて説明します。トリガー: Vercel からの移行、Cloudflare への移行、Vercel から Workers への移行、Cloudflare への Next.js のデプロイ、Vercel から Cloudflare への置き換え、または Vercel から Cloudflare への移行タスク。
