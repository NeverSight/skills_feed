Создавайте локальные и автономные приложения TypeScript с помощью PowerSync. Используйте при реализации синхронизации в реальном времени между SQLite и серверными базами данных (Postgres, MongoDB, MySQL, SQL Server). Охватывает определение схемы, настройку базы данных, операции CRUD, перехватчики React/Vue, запросы наблюдения и интеграцию ORM Kysely/Drizzle.
