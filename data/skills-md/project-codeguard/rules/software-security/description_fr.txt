Une compétence de sécurité logicielle qui s'intègre à Project CodeGuard pour aider les agents de codage d'IA à écrire du code sécurisé et à prévenir les vulnérabilités courantes. Utilisez cette compétence lors de l'écriture, de la révision ou de la modification du code pour garantir le respect des pratiques de sécurité par défaut.
