Guía para patrones avanzados de Next.js App Router, incluidos controladores de ruta, rutas paralelas, rutas de intercepción, acciones del servidor, límites de error, modo borrador y transmisión con suspenso. CRÍTICO para las acciones del servidor (action.ts, archivos action.ts, directiva 'usar servidor'), configuración de cookies de componentes del cliente y manejo de formularios. Úselo cuando los requisitos involucren acciones del servidor, envíos de formularios, cookies, mutaciones, rutas API, `route.ts`, rutas paralelas, rutas de interceptación o transmisión por secuencias. Esencial para separar las acciones del servidor de los componentes del cliente.
