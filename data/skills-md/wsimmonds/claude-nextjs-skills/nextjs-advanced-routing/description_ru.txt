Руководство по расширенным шаблонам маршрутизации приложений Next.js, включая обработчики маршрутов, параллельные маршруты, перехватывающие маршруты, действия сервера, границы ошибок, черновой режим и потоковую передачу с приостановкой. КРИТИЧНО для действий сервера (файлы action.ts, action.ts, директива use server), установки файлов cookie из клиентских компонентов и обработки форм. Используйте, когда требования включают действия сервера, отправку форм, файлы cookie, мутации, маршруты API, Route.ts, параллельные маршруты, перехватывающие маршруты или потоковую передачу. Необходим для отделения действий сервера от клиентских компонентов.
