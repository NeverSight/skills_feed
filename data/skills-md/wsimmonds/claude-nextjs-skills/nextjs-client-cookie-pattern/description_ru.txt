Шаблон для клиентских компонентов, вызывающих действия сервера для установки файлов cookie в Next.js. Охватывает двухфайловый шаблон клиентского компонента с взаимодействием с пользователем (onClick, отправка формы), который вызывает действие сервера для изменения файлов cookie. Используйте при создании таких функций, как аутентификация, настройки или управление сеансами, где триггеры на стороне клиента должны устанавливать/изменять файлы cookie на стороне сервера.
