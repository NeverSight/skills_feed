Руководство по выбору между серверными и клиентскими компонентами в маршрутизаторе приложений Next.js. КРИТИЧНО для useSearchParams (требуется Suspense + «использовать клиент»), навигации (ссылка, перенаправление, useRouter), доступа к файлам cookie/заголовков и директивы «использовать клиент». Активируется, когда в подсказке упоминается использованиеSearchParams, Suspense, навигация, маршрутизация, компонент Link, перенаправление, путь, searchParams, файлы cookie, заголовки, асинхронные компоненты или «использовать клиент». Необходим для предотвращения смешивания API-интерфейсов сервера и клиента.
