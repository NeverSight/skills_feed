Guía para elegir entre componentes de servidor y componentes de cliente en Next.js App Router. CRÍTICO para useSearchParams (requiere Suspense + 'usar cliente'), navegación (Enlace, redireccionamiento, useRouter), acceso a cookies/encabezados y directiva 'usar cliente'. Se activa cuando el mensaje menciona useSearchParams, Suspense, navegación, enrutamiento, componente de enlace, redirección, nombre de ruta, searchParams, cookies, encabezados, componentes asíncronos o "usar cliente". Esencial para evitar mezclar API de servidor/cliente.
