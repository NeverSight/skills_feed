دليل للاختيار بين مكونات الخادم ومكونات العميل في جهاز توجيه تطبيق Next.js. ضروري لـ useSearchParams (يتطلب Suspense + "استخدام العميل")، والتنقل (الرابط، وإعادة التوجيه، useRouter)، والوصول إلى ملفات تعريف الارتباط/الرؤوس، وتوجيه "استخدام العميل". يتم التنشيط عندما تشير المطالبة إلى useSearchParams، أو Suspense، أو التنقل، أو التوجيه، أو مكون الارتباط، أو إعادة التوجيه، أو اسم المسار، أو searchParams، أو ملفات تعريف الارتباط، أو الرؤوس، أو المكونات غير المتزامنة، أو "استخدام العميل". ضروري لتجنب الخلط بين واجهات برمجة التطبيقات للخادم/العميل.
