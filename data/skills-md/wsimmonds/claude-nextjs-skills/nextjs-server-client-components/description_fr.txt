Guide pour choisir entre les composants serveur et les composants clients dans Next.js App Router. CRITIQUE pour useSearchParams (nécessite Suspense + 'use client'), la navigation (lien, redirection, useRouter), l'accès aux cookies/en-têtes et la directive 'use client'. S'active lorsque l'invite mentionne useSearchParams, Suspense, navigation, routage, composant Link, redirection, chemin d'accès, searchParams, cookies, en-têtes, composants asynchrones ou « utiliser le client ». Indispensable pour éviter de mélanger les API serveur/client.
