Guía para los patrones de implementación de Vercel AI SDK v5, incluidos generateText, streamText, useChat Hook, llamadas de herramientas, incrustaciones e integración de MCP. Úselo al implementar interfaces de chat de IA, transmitir respuestas, llamar a herramientas/funciones, incrustar texto o trabajar con convertToModelMessages y toUIMessageStreamResponse. Se activa para la integración del SDK de AI, el uso del enlace de chat, la transmisión de mensajes o las tareas de llamada de herramientas.
