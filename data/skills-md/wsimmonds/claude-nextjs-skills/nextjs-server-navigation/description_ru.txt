Руководство по реализации навигации в серверных компонентах Next.js с использованием компонента Link и функции redirect(). Охватывает разницу между серверными и клиентскими методами навигации. Используйте при добавлении ссылок, перенаправлений или логики навигации в серверные компоненты без ненужного преобразования их в клиентские компоненты.
