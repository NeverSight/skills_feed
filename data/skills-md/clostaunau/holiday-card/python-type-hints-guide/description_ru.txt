Комплексное справочное руководство по подсказкам типов Python, статической проверке типов с помощью mypy, современным шаблонам аннотаций типов (PEP 484, 585, 604, 612, 613) и лучшим практикам подсказок типов для Python 3.9+. Используйте во время проверки кода, чтобы обеспечить правильное использование аннотаций типов, оценить конфигурацию mypy и выявить антишаблоны подсказок типов.
