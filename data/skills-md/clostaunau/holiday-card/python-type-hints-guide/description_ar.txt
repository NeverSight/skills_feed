دليل مرجعي شامل لتلميحات نوع Python، والتحقق من النوع الثابت باستخدام mypy، وأنماط التعليقات التوضيحية الحديثة (PEP 484، 585، 604، 612، 613)، وأفضل ممارسات تلميح النوع لـ Python 3.9+. يُستخدم أثناء مراجعات التعليمات البرمجية لضمان الاستخدام الصحيح للتعليق التوضيحي للنوع، وتقييم تكوين mypy، وتحديد الأنماط المضادة لتلميحات الكتابة.
