دليل شامل لأنماط Python غير المتزامنة/الانتظار وأفضل الممارسات والأنماط المضادة. يغطي أساسيات عدم المزامنة، وcoroutines، ومديري السياق غير المتزامن، وإدارة المهام، والمكتبات العامة (aiohttp، aiofiles، asyncpg)، وتكامل إطار العمل (FastAPI، وDjango)، واعتبارات الأداء، والتعامل مع الاستثناءات المناسبة. يُستخدم عند مراجعة أو كتابة كود Python غير المتزامن.
