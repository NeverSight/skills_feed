Ermöglicht die Interaktion mit Webseiten durch das Ausführen von Aktionen wie das Klicken auf Schaltflächen, das Ausfüllen von Formularen und das Navigieren in Links. Es funktioniert durch Fernsteuerung von Google Chrome- oder Chromium-Browsern mithilfe des Chrome DevTools Protocol (CDP). Wenn Claude im Internet surfen muss, kann er diese Fähigkeit nutzen.
