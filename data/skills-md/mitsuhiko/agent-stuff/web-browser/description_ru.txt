Позволяет взаимодействовать с веб-страницами, выполняя такие действия, как нажатие кнопок, заполнение форм и переход по ссылкам. Он работает путем удаленного управления браузерами Google Chrome или Chromium с использованием протокола Chrome DevTools (CDP). Когда Claude нужно просмотреть веб-страницы, он может использовать для этого этот навык.
