Комплексное руководство по интеграции биллинговой платформы Chargebee. Предоставляет шаблоны интеграции API, обработку веб-перехватчиков, использование SDK и ссылки на схемы для операций выставления счетов. Используйте при работе с Chargebee для (1) интеграции API и вызовов конечных точек REST, (2) обработки событий веб-перехватчика, (3) операций по управлению клиентами, (4) обработки жизненного цикла подписки, (5) обработки платежей и счетов, (6) любых других задач интеграции, связанных с выставлением счетов, с платформой Chargebee.
