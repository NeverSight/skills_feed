Outil de construction Vite 8 (propulsé par Rolldown) : configuration, API de plugin, migration SSR et Rolldown. À utiliser lorsque vous travaillez avec des projets Vite, vite.config.ts, HMR, plugins Vite, mode bibliothèque, builds SSR ou lors de la migration de Vite 5/6 vers Vite 8. Déclenchez lorsque l'utilisateur mentionne Vite, vite.config, vitest (côté configuration), HMR, Rolldown, Rollup plugin, ou demande de configurer, déboguer ou optimiser un projet basé sur Vite (Vue, React, Svelte, Nuxt, SvelteKit).
