Опыт структурированного рабочего процесса коммитов Git с соблюдением соглашений китайской команды. Используйте этот навык, когда пользователь просит «зафиксировать», «提交», «стадия», «暂存», «показать статус», «статус git» или любой вариант, связанный с управлением изменениями git. Пользователю НЕ нужно явно упоминать «git-commit-workflow» — ЛЮБОЙ запрос на фиксацию изменений или управление состоянием git должен автоматически активировать этот навык. Обеспечивает автоматическое создание сообщений о фиксации с помощью PMS и отслеживания проблем GitHub, соблюдая строгие ограничения на длину строки в 80 символов.
