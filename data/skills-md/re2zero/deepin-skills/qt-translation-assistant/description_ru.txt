Используйте, когда пользователь запрашивает перевод файлов локализации проекта Qt (файлы TS), автоматизацию рабочих процессов перевода или настройку многоязычной поддержки для приложений Qt. Этот навык использует параллельную обработку с помощью ThreadPoolExecutor для эффективного перевода файлов TS (источник перевода).
