أنماط واصطلاحات نموذج ActiveRecord لـ Rails. يتم استدعاؤه تلقائيًا عند العمل مع النماذج أو الارتباطات أو عمليات التحقق من الصحة أو النطاقات أو عمليات الاسترجاعات أو تصميم مخطط قاعدة البيانات. مشغلات على "النموذج"، و"ActiveRecord"، و"الارتباط"، و"has_many"، و"belongs_to"، و"validation"، و"validates"، و"scope"، و"callback"، و"migration"، و"schema"، و"index"، و"external key".
