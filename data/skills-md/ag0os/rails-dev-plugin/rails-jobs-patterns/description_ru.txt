ActiveJob и шаблоны фоновой обработки для Rails. Автоматически вызывается при работе с фоновыми заданиями, Sidekiq, асинхронной обработкой, очередями заданий, планированием или каталогом app/jobs. Запускается по «заданию», «фоновому заданию», «ActiveJob», «Sidekiq», «async», «queue», «perform_later», «работнику», «запланированному заданию», «cron», «повторной попытке», «идемпотенту».
