Шаблоны Stimulus и Turbo для разработки интерфейса Rails. Автоматически вызывается при работе с контроллерами Hotwire, Stimulus, турбо-фреймами/потоками, прогрессивным улучшением или современным Rails JavaScript. Триггеры по «Стимулу», «Турбо», «Hotwire», «turbo_frame», «turbo_stream», «Контроллеру стимула», «контроллеру данных», «действию данных», «прогрессивному улучшению», «SPA-подобному».
