مساعد الأتمتة Xiaohongshu (RED/XHS). يوفر إمكانات تشغيلية كاملة لـ Xiaohongshu: تسجيل الدخول، ونشر الصور/مقاطع الفيديو، وملاحظات البحث، وتصفح التفاصيل، والإعجاب، والمفضلة، والتعليق، وعرض ملفات تعريف المبدعين، وتخطيط المحتوى.
استخدم هذه المهارة عندما يذكر المستخدم Xiaohongshu، أو Red Book، أو XHS، أو RED، أو ملاحظات النشر، أو ملاحظات البحث، أو عمليات Xiaohongshu، أو أي إجراءات متعلقة بـ Xiaohongshu - حتى إذا لم يقل المستخدم صراحةً "Xiaohongshu" ولكن السيناريو الموصوف يتعلق بذلك بوضوح (على سبيل المثال، "انشر ملاحظة عن زراعة العشب"، "تحليل هذا المنشئ").
