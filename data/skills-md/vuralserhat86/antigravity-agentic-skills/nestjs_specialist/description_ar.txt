دليل إطار عمل NestJS الشامل مع تكامل Drizzle ORM. يُستخدم عند إنشاء تطبيقات NestJS، أو إعداد واجهات برمجة التطبيقات، أو تنفيذ المصادقة، أو العمل مع قواعد البيانات، أو دمج Drizzle ORM. يغطي وحدات التحكم، والموفرين، والوحدات النمطية، والبرمجيات الوسيطة، والحراس، والمعترضات، والاختبار، والخدمات الدقيقة، وGraphQL، وأنماط قواعد البيانات.
