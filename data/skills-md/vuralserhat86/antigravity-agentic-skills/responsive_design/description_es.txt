Cree diseños responsivos y pensados ​​para dispositivos móviles utilizando contenedores fluidos, unidades flexibles, consultas de medios y un diseño táctil que funcione en todos los tamaños de pantalla. Utilice esta habilidad al crear o modificar diseños de interfaz de usuario, cuadrículas responsivas, estilos de puntos de interrupción, navegación móvil o cualquier interfaz que deba adaptarse a diferentes tamaños de pantalla. Aplicar cuando se trabaja con CSS responsivo, consultas de medios, configuraciones de ventana gráfica, diseños de flexbox/cuadrícula, estilos móviles primero, definiciones de puntos de interrupción (móvil, tableta, computadora de escritorio), tamaño del objetivo táctil, unidades relativas (rem, em, %), optimización de imágenes para diferentes pantallas o pruebas de diseños en múltiples dispositivos. Úselo para cualquier tarea que implique soporte para múltiples dispositivos, patrones de diseño responsivos o diseños adaptables.
