Crea layout reattivi e incentrati sui dispositivi mobili utilizzando contenitori fluidi, unità flessibili, query multimediali e un design touch-friendly che funzioni su schermi di tutte le dimensioni. Utilizza questa competenza durante la creazione o la modifica di layout dell'interfaccia utente, griglie reattive, stili di punti di interruzione, navigazione mobile o qualsiasi interfaccia che deve adattarsi a diverse dimensioni dello schermo. Applicare quando si lavora con CSS reattivi, query multimediali, impostazioni del viewport, layout flexbox/griglia, stile mobile-first, definizioni di punti di interruzione (mobile, tablet, desktop), dimensionamento del touch target, unità relative (rem, em, %), ottimizzazione delle immagini per schermi diversi o test di layout su più dispositivi. Utilizzalo per qualsiasi attività che coinvolga supporto multi-dispositivo, modelli di progettazione reattivi o layout adattivi.
