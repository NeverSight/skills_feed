Genera documentazione API REST completa utilizzando SpringDoc OpenAPI 3.0 e l'interfaccia utente Swagger nelle applicazioni Spring Boot 3.x. Da utilizzare durante l'impostazione della documentazione API, la configurazione dell'interfaccia utente Swagger, l'aggiunta di annotazioni OpenAPI, l'implementazione della documentazione sulla sicurezza o il miglioramento degli endpoint REST con esempi e schemi.
