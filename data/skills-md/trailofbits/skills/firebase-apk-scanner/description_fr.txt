Analyse les APK Android pour détecter les erreurs de configuration de sécurité Firebase, notamment les bases de données ouvertes, les compartiments de stockage, les problèmes d'authentification et les fonctions cloud exposées. À utiliser lors de l'analyse des fichiers APK pour les vulnérabilités Firebase, de la réalisation d'audits de sécurité des applications mobiles ou du test de la sécurité des points de terminaison Firebase.
