Выполняет дифференциальную проверку изменений кода с учетом безопасности (PR, коммиты, различия). Адаптирует глубину анализа к размеру кодовой базы, использует историю git для контекста, рассчитывает радиус взрыва, проверяет тестовое покрытие и генерирует подробные отчеты об уценке. Автоматически обнаруживает и предотвращает снижение безопасности.
