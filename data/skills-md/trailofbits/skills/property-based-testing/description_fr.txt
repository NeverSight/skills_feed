Fournit des conseils pour les tests basés sur les propriétés dans plusieurs langues et contrats intelligents. À utiliser lors de l'écriture de tests, de la révision de code avec des modèles de sérialisation/validation/analyse, de la conception de fonctionnalités ou lorsque les tests basés sur les propriétés fourniraient une couverture plus complète que les tests basés sur des exemples.
