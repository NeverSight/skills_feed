Bietet Anleitungen für eigenschaftsbasierte Tests in mehreren Sprachen und Smart Contracts. Verwenden Sie diese Option, wenn Sie Tests schreiben, Code mit Serialisierungs-/Validierungs-/Parsing-Mustern überprüfen, Funktionen entwerfen oder wenn eigenschaftsbasierte Tests eine stärkere Abdeckung bieten würden als beispielbasierte Tests.
