Détecte les vulnérabilités de canal secondaire de synchronisation dans le code cryptographique. À utiliser lors de l'implémentation ou de la révision de code cryptographique, en cas de division sur les secrets, de branches dépendantes des secrets ou de questions de programmation en temps constant en C, C++, Go, Rust, Swift, Java, Kotlin, C#, PHP, JavaScript, TypeScript, Python ou Ruby.
