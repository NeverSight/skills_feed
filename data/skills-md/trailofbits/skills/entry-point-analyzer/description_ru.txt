Анализирует кодовую базу смарт-контрактов для определения точек входа, меняющих состояние, для аудита безопасности. Обнаруживает вызываемые извне функции, которые изменяют состояние, классифицирует их по уровню доступа (общедоступные, административные, с ограничениями по ролям, только по контракту) и генерирует структурированные отчеты об аудите. Исключает функции просмотра/чистые/только для чтения. Используйте при аудите смарт-контрактов (Solidity, Vyper, Solana/Rust, Move, TON, CosmWasm) или когда вас просят найти точки входа, потоки аудита, внешние функции, шаблоны управления доступом или привилегированные операции.
