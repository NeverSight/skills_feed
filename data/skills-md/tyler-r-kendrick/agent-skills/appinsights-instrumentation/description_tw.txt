偵測 Web 應用程式以將遙測資料傳送至 Azure Application Insights 以進行可觀察和監視。用於：使用應用程式見解檢測應用程式、新增應用程式見解偵測、設定應用程式洞察、設定遙測監控、啟用應用程式見解自動偵測、向 azure web 應用程式新增可觀察性、偵測 webapp 以將資料傳送至應用程式洞察、為應用程式服務設定遙測。請勿用於：非 Azure 監控（針對 AWS 使用 CloudWatch，針對第三方使用 Datadog）、日誌分析（使用 azure-kusto）、成本監控（使用 azure-cost-optimization）、安全性監控（使用 azure-security）。
