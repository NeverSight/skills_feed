Web アプリケーションを計測して、可観測性と監視のためにテレメトリ データを Azure Application Insights に送信します。用途: アプリ インサイトを使用してアプリを計測する、appinsights インストルメンテーションを追加する、アプリケーション インサイトを構成する、テレメトリ モニタリングをセットアップする、アプリ インサイトの自動インストルメンテーションを有効にする、Azure Web アプリに可観測性を追加する、アプリ インサイトにデータを送信するために Web アプリを計測する、アプリ サービスのテレメトリを構成する。次の用途には使用しないでください: Azure 以外の監視 (AWS には CloudWatch、サードパーティには Datadog を使用)、ログ分析 (azure-kusto を使用)、コスト監視 (azure-cost-optimization を使用)、セキュリティ監視 (azure-security を使用)。
