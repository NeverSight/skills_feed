Инструментируйте веб-приложения для отправки данных телеметрии в Azure Application Insights для наблюдения и мониторинга. ИСПОЛЬЗОВАНИЕ ДЛЯ: инструментирования приложения с аналитикой приложений, добавления инструментов Appinsights, настройки аналитики приложений, настройки мониторинга телеметрии, включения автоматического инструментирования аналитики приложений, добавления возможности наблюдения в веб-приложение Azure, инструментирования веб-приложения для отправки данных в аналитику приложений, настройки телеметрии для службы приложений. НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: мониторинга, не связанного с Azure (используйте CloudWatch для AWS, Datadog для сторонних поставщиков), анализа журналов (используйте azure-kusto), мониторинга затрат (используйте azure-cost-optimization), мониторинга безопасности (используйте azure-security).
