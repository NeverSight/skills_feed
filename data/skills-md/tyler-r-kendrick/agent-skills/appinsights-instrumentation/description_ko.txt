관찰 가능성 및 모니터링을 위해 Azure Application Insights에 원격 분석 데이터를 보내도록 웹 애플리케이션을 계측합니다. 사용 대상: 앱 인사이트로 앱 계측, appinsights 계측 추가, 애플리케이션 인사이트 구성, 원격 측정 모니터링 설정, 앱 인사이트 자동 계측 활성화, Azure 웹앱에 관찰 가능성 추가, 앱 인사이트에 데이터를 보내기 위해 웹앱 계측, 앱 서비스용 원격 분석 구성. 사용하지 마십시오: 비 Azure 모니터링(AWS용 CloudWatch, 타사용 Datadog 사용), 로그 분석(azure-kusto 사용), 비용 모니터링(azure-cost-optimization 사용), 보안 모니터링(azure-security 사용).
