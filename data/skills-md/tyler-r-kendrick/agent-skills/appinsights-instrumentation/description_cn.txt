检测 Web 应用程序以将遥测数据发送到 Azure Application Insights 以进行可观察和监视。用于：使用应用程序见解检测应用程序、添加应用程序见解检测、配置应用程序见解、设置遥测监控、启用应用程序见解自动检测、向 azure web 应用程序添加可观察性、检测 webapp 以将数据发送到应用程序见解、为应用程序服务配置遥测。请勿用于：非 Azure 监控（针对 AWS 使用 CloudWatch，针对第三方使用 Datadog）、日志分析（使用 azure-kusto）、成本监控（使用 azure-cost-optimization）、安全监控（使用 azure-security）。
