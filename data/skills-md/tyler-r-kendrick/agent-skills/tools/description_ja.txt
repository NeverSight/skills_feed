すべての言語およびプラットフォームにわたるソフトウェア開発に不可欠な基本的な CLI ツールおよびユーティリティを使用する場合に使用します。シェル、バージョン管理、システム パッケージ マネージャー、コンテナー、リモート アクセス、HTTP クライアント、データ処理、ビルド ランナーをカバーします。
用途: CLI ツール、開発者ツール、シェル スクリプト、バージョン管理、システム パッケージ マネージャー、コンテナ、リモート アクセス、ビルド自動化、テキスト処理、クロスプラットフォーム開発ツールの選択
次の用途には使用しないでください: 言語固有のパッケージ マネージャー (npm/pip/cargo などの言語固有のスキルを使用)、IDE 構成、言語固有のビルド ツール (言語固有のスキルを使用)
