Используйте при работе с основными инструментами и утилитами CLI, которые необходимы для разработки программного обеспечения на всех языках и платформах. Охватывает оболочки, контроль версий, менеджеры системных пакетов, контейнеры, удаленный доступ, HTTP-клиенты, обработку данных и средства сборки.
ИСПОЛЬЗОВАНИЕ ДЛЯ: инструментов CLI, инструментов разработчика, сценариев оболочки, контроля версий, менеджеров системных пакетов, контейнеров, удаленного доступа, автоматизации сборки, обработки текста, выбора кроссплатформенных инструментов разработки.
НЕ ИСПОЛЬЗОВАТЬ ДЛЯ: менеджеров пакетов, специфичных для языка (используйте навыки, специфичные для языка, такие как npm/pip/cargo), настройки IDE, инструментов сборки, специфичных для языка (используйте навыки, специфичные для языка).
