ابحث عبر قواعد معارف الشركة (Confluence وJira والمستندات الداخلية) للعثور على المفاهيم والعمليات والتفاصيل الفنية الداخلية وشرحها. عندما يحتاج Claude إلى: (1) البحث عن معلومات حول الأنظمة أو المصطلحات أو العمليات أو النشر أو المصادقة أو البنية التحتية أو البنية أو المفاهيم التقنية أو البحث عنها، (2) البحث في الوثائق الداخلية أو قاعدة المعرفة أو مستندات الشركة أو مستنداتنا، (3) شرح ماهية شيء ما، أو كيفية عمله، أو البحث عن معلومات، أو (4) تجميع المعلومات من مصادر متعددة. يبحث بالتوازي ويقدم الإجابات المذكورة.
