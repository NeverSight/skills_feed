Sviluppo dell'infrastruttura AWS CDK con TypeScript. Da utilizzare durante la creazione o la modifica di stack CDK, costrutti, tabelle DynamoDB, servizi ECS/Fargate, funzioni Lambda, bucket S3, reti, ruoli IAM o qualsiasi risorsa CloudFormation. Copre modelli di configurazione, riferimenti cross-stack tramite SSM, convenzioni di denominazione e integrazione Bedrock AgentCore.
