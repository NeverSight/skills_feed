تطوير البنية التحتية لـ AWS CDK باستخدام TypeScript. يُستخدم عند إنشاء أو تعديل مجموعات CDK، أو البنيات، أو جداول DynamoDB، أو خدمات ECS/Fargate، أو وظائف Lambda، أو مجموعات S3، أو الشبكات، أو أدوار IAM، أو أي موارد CloudFormation. يغطي أنماط التكوين والمراجع المتداخلة عبر SSM واصطلاحات التسمية وتكامل Bedrock AgentCore.
