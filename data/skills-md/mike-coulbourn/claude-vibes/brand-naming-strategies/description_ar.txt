يوفر أطر تسمية العلامات التجارية ومعايير التقييم والقوالب الخاصة بأعمال تسمية الشركات الناشئة. يتم التنشيط تلقائيًا أثناء تطوير اسم العلامة التجارية، وتقييم الاسم، والتحقق من النطاق، وأبحاث العلامات التجارية. يُستخدم عند مناقشة اسم العلامة التجارية، أو اسم الشركة، أو اسم المنتج، أو استراتيجية التسمية، أو إطار عمل SMILE SCRATCH، أو توفر النطاق، أو العلامة التجارية، أو تقييم الاسم، أو الرمزية الصوتية، أو مصفوفة التسمية.
