يوفر أطر قيم العلامة التجارية، واختبارات التمايز، وأسئلة الاكتشاف، وقوالب رسم خرائط السلوك. يتم التنشيط تلقائيًا أثناء تعريف القيم، وتنظيم القيم الأساسية، واختبار القيم. يُستخدم عند مناقشة قيم العلامة التجارية، أو القيم الأساسية، أو فئات Lencioni الأربع، أو الاختبار المعاكس، أو القيمة المضادة، أو اختبار فصل شخص ما، أو تفعيل القيم، أو عملية قيم Brene Brown، أو اكتشاف القيم، أو قيم إذن التشغيل.
