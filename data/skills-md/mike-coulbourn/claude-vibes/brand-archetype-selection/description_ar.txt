يوفر إطارات نموذجية للعلامة التجارية Jungian، و12 نموذجًا أصليًا مع ملفات تعريف، وقاعدة 70/30 الأولية/الثانوية، ومجموعات النماذج الأصلية، وقوالب الاختيار. يتم التنشيط تلقائيًا أثناء اختيار النموذج الأصلي للعلامة التجارية، وتحديد الموقع العاطفي، وعمل شخصية العلامة التجارية. يُستخدم عند مناقشة نماذج العلامة التجارية، أو نماذج يونج، أو 12 نموذجًا، أو بطل، أو خارج عن القانون، أو ساحر، أو مبدع، أو عاشق، أو مهرج، أو كل رجل، أو مقدم رعاية، أو حاكم، أو حكيم، أو مستكشف، أو بريء، أو قاعدة 70/30، أو مارك-بيرسون، أو مجموعات النماذج الأصلية.
