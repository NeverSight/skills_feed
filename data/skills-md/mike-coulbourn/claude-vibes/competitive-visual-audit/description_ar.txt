يوفر أطر وقوالب للتدقيق البصري التنافسي لعمل هوية العلامة التجارية. يتم التنشيط تلقائيًا أثناء التوجيه البصري واختيار اللون واختيار الطباعة وتحديد المواقع. يُستخدم عند مناقشة المنافسين، أو التدقيق التنافسي، أو التمايز، أو المساحة البيضاء، أو رسم الخرائط الإدراكية، أو المشهد البصري، أو إستراتيجية التعرج، أو تدقيق العلامة التجارية، أو تدقيق الألوان، أو تدقيق الطباعة، أو تحليل الهوية المرئية، أو خرائط تحديد المواقع.
