Преобразуйте структурированные спецификации UX и контекст продукта в упорядоченный файл Prompts.md для Claude Code. Используйте, когда пользователь завершил проектное мышление (постановка задачи, PRD, UX-спецификация) и ему необходимо преобразовать это в пошаговые подсказки, которые агенты кодирования могут выполнять постепенно. Этот навык объединяет артефакты проектирования с генерацией кода.
