Genere documentos de requisitos de producto ágiles y procesables a partir de artefactos de pensamiento de diseño ascendentes o aportes sin procesar. Úselo cuando un usuario necesita definir lo que está creando con suficiente estructura para guiar el desarrollo pero sin sobrecarga empresarial. Genera un PRD que alimenta directamente las especificaciones de UX y las indicaciones de desarrollo.
