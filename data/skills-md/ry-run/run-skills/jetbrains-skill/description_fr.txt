Utilisez le serveur JetBrains IDE MCP (IntelliJ IDEA 2025.2+) pour permettre à un client externe de piloter des actions soutenues par l'IDE : exécuter des configurations, exécuter des commandes dans le terminal IDE, lire/créer/modifier des fichiers de projet, rechercher via des index IDE (texte/regex), récupérer des inspections de code pour un fichier, récupérer des informations sur les symboles, effectuer une refactorisation du nom, répertorier les modules/dépendances/dépôts, ouvrir des fichiers dans l'éditeur et reformater le code. À utiliser lorsque vous souhaitez une indexation/refactorisation/inspection de qualité IDE au lieu de scripts shell bruts.
