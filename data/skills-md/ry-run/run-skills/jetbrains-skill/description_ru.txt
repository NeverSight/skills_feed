Используйте сервер JetBrains IDE MCP (IntelliJ IDEA 2025.2+), чтобы позволить внешнему клиенту управлять действиями, поддерживаемыми IDE: запускать конфигурации, выполнять команды в терминале IDE, читать/создавать/редактировать файлы проекта, выполнять поиск по индексам IDE (текст/регулярное выражение), получать проверки кода для файла, получать информацию о символах, выполнять рефакторинг переименования, выводить список модулей/зависимостей/репозиториев, открывать файлы в редакторе и переформатировать код. Используйте, если вам нужна индексация/рефакторинг/проверка уровня IDE вместо необработанных сценариев оболочки.
