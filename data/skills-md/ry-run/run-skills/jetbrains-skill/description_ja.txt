JetBrains IDE MCP サーバー (IntelliJ IDEA 2025.2+) を使用して、外部クライアントが IDE ベースのアクションを実行できるようにします: 実行構成の実行、IDE ターミナルでのコマンドの実行、プロジェクト ファイルの読み取り/作成/編集、IDE インデックス (テキスト/正規表現) による検索、ファイルのコード インスペクションの取得、シンボル情報の取得、名前変更リファクタリングの実行、モジュール/依存関係/リポジトリのリスト、エディターでファイルを開く、コードの再フォーマット。生のシェルスクリプトではなく、IDE グレードのインデックス作成/リファクタリング/検査が必要な場合に使用します。
