Verwenden Sie den JetBrains IDE MCP Server (IntelliJ IDEA 2025.2+), um einen externen Client IDE-gestützte Aktionen steuern zu lassen: Konfigurationen ausführen, Befehle im IDE-Terminal ausführen, Projektdateien lesen/erstellen/bearbeiten, über IDE-Indizes (Text/Regex) suchen, Codeinspektionen für eine Datei abrufen, Symbolinformationen abrufen, Umbenennungs-Refactoring durchführen, Module/Abhängigkeiten/Repos auflisten, Dateien im Editor öffnen und Code neu formatieren. Verwenden Sie diese Option, wenn Sie eine Indizierung/Refactoring/Inspektion auf IDE-Niveau statt roher Shell-Skripterstellung wünschen.
