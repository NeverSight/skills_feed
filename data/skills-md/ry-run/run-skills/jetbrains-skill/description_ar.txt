استخدم خادم JetBrains IDE MCP (IntelliJ IDEA 2025.2+) للسماح للعميل الخارجي بقيادة الإجراءات المدعومة من IDE: تشغيل Run Configurations، وتنفيذ الأوامر في محطة IDE، وقراءة/إنشاء/تحرير ملفات المشروع، والبحث عبر فهارس IDE (نص/regex)، واسترداد عمليات فحص التعليمات البرمجية لملف، وجلب معلومات الرمز، وإجراء إعادة تسمية، وسرد الوحدات النمطية/التبعيات/repos، وفتح الملفات في المحرر، و رمز إعادة التنسيق. استخدمه عندما تريد الفهرسة/إعادة البناء/الفحص على مستوى IDE بدلاً من البرمجة النصية الأولية.
