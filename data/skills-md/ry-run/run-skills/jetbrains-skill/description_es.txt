Utilice el servidor JetBrains IDE MCP (IntelliJ IDEA 2025.2+) para permitir que un cliente externo impulse acciones respaldadas por IDE: ejecutar ejecutar configuraciones, ejecutar comandos en la terminal IDE, leer/crear/editar archivos de proyecto, buscar a través de índices IDE (texto/regex), recuperar inspecciones de código para un archivo, obtener información de símbolos, realizar refactorización de cambio de nombre, enumerar módulos/dependencias/repositorios, abrir archivos en el editor y reformatear el código. Úselo cuando desee indexación/refactorización/inspección de nivel IDE en lugar de scripts de shell sin formato.
