Desktop Commander MCP (通常は「mcp__desktop-commander__*」などのツール) を使用して、ローカル ファイルと長時間実行プロセスを管理します。ファイルの読み取り/書き込み/検索、正確な編集の適用、Excel/PDF の操作、ターミナル コマンドの実行と REPL (Python/Node/SSH/DB) との対話、プロセスの検査/終了、ツール呼び出し履歴の確認を行います。タスクがマシン上で実際の作業 (コード/構成の編集、リポジトリの検索、CSV/Excel の分析、PDF の生成/変更、ストリーミング出力でのコマンドの実行) を実行する必要がある場合に使用します。
