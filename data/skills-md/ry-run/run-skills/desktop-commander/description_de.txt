Verwenden Sie Desktop Commander MCP (normalerweise Tools wie „mcp__desktop-commander__*“), um lokale Dateien und lang laufende Prozesse zu verwalten: Dateien lesen/schreiben/durchsuchen, präzise Bearbeitungen vornehmen, mit Excel/PDFs arbeiten, Terminalbefehle ausführen und mit REPLs (Python/Node/SSH/DB) interagieren, Prozesse überprüfen/beenden und den Tool-Aufrufverlauf überprüfen. Verwenden Sie es, wenn die Aufgabe echte Arbeit auf dem Computer erfordert (Bearbeiten von Code/Konfigurationen, Durchsuchen eines Repositorys, Analysieren von CSV/Excel, Generieren/Ändern von PDFs, Ausführen von Befehlen mit Streaming-Ausgabe).
