Utilice Desktop Commander MCP (normalmente herramientas como `mcp__desktop-commander__*`) para administrar archivos locales y procesos de larga duración: leer/escribir/buscar archivos, aplicar ediciones precisas, trabajar con Excel/PDF, ejecutar comandos de terminal e interactuar con REPL (Python/Node/SSH/DB), inspeccionar/terminar procesos y revisar el historial de llamadas de herramientas. Úselo cuando la tarea requiera realizar un trabajo real en la máquina (editar código/configuraciones, buscar en un repositorio, analizar CSV/Excel, generar/modificar archivos PDF, ejecutar comandos con salida de streaming).
