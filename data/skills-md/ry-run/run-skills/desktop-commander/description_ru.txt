Используйте Desktop Commander MCP (обычно такие инструменты, как `mcp__desktop-commander__*`) для управления локальными файлами и длительными процессами: чтение/запись/поиск файлов, применение точных изменений, работа с Excel/PDF, запуск команд терминала и взаимодействие с REPL (Python/Node/SSH/DB), проверка/завершение процессов и просмотр истории вызовов инструментов. Используйте, когда задача требует выполнения реальной работы на компьютере (редактирование кода/конфигураций, поиск в репозитории, анализ CSV/Excel, создание/изменение PDF-файлов, запуск команд с потоковым выводом).
