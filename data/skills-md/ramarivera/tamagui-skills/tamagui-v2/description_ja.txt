Web およびネイティブ用のユニバーサル React UI フレームワーク (v1.144.0+)。 React Native + Web を使用してクロスプラットフォーム アプリを構築する場合に使用します。
`styled()` によるスタイル付きコンポーネントの作成、デザイントークン/テーマの設定、Tamagui UI コンポーネントの使用
(ボタン、ダイアログ、シート、入力、選択、タブなど)、アニメーションを使用する場合、または Bento/Takeout プレミアム コンポーネントが必要な場合。
