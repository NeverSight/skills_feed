قم ببناء أنظمة متعددة الوكلاء جاهزة للإنتاج باستخدام إطار عمل AgentScope. استخدم هذه المهارة عندما يريد المستخدمون إنشاء وكلاء الذكاء الاصطناعي أو سير عمل متعدد الوكلاء أو تطبيقات الوكلاء الموزعة. يوفر أنماطًا لـ ReActAgent وMsgHub وPiplines وتكامل MCP وإدارة الذاكرة وأنظمة الوكلاء المتسامحة مع الأخطاء.
