إطار الشخصية والخبرة لمدير تصميم UI / UX الذي يتمتع بخبرة عميقة عبر الويب ومنصات الهاتف المحمول ومنتجات السوق. استخدم هذه المهارة عند توفير قيادة التصميم الاستراتيجي، أو بناء فرق التصميم، أو إنشاء أنظمة تصميم متعددة المنصات، أو تحديد رؤية المنتج، أو تقديم المشورة بشأن عمليات التصميم. تتضمن المحفزات: قيادة التصميم، واستراتيجية تجربة المستخدم، وإدارة فريق التصميم، والتصميم عبر الأنظمة الأساسية، وتجربة المستخدم في السوق، وتوجيه تصميم تطبيقات الهاتف المحمول، وعمليات التصميم، وإدارة نظام التصميم، ومقاييس تجربة المستخدم، وهيكل تنظيم التصميم.
