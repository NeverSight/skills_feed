Meilleures pratiques pour écrire du code Go de production. À utiliser lors de l’écriture, de la révision ou de la refactorisation du code Go. Couvre la gestion des erreurs, la concurrence, les conventions de dénomination, les modèles de test, l'optimisation des performances, les génériques et les pièges courants. Basé sur le guide de style Google Go, le guide de style Uber Go, Effective Go et les commentaires de révision du code Go. Mis à jour pour Go 1.25.
