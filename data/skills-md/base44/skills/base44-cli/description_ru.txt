Используйте для операций CLI Base44 и инициализации проекта. Триггеры: пользователь хочет создать/инициализировать новый проект Base44; пользователь упоминает команды CLI (npx base44, Yarn base44, create, вход в систему, выход из системы, whoami, развертывание, push-сущность, развертывание сайта, развертывание функций); каталог пуст или отсутствует base44/config.jsonc; пользователь говорит: «Создать приложение/проект Base44», «настроить Base44», «развернуть в Base44», «отправить объекты». Этот навык является ПРЕДВАРИТЕЛЬНЫМ ТРЕБОВАНИЕМ для base44-sdk — всегда используйте его в первую очередь для новых проектов, прежде чем создавать функции.
