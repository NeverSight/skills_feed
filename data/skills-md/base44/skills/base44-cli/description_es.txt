Úselo para operaciones CLI Base44 e inicialización de proyectos. Desencadenantes: el usuario desea crear/inicializar un nuevo proyecto Base44; el usuario menciona los comandos CLI (npx base44, Yarn base44, crear, iniciar sesión, cerrar sesión, whoami, implementar, enviar entidades, implementar sitios, implementar funciones); el directorio está vacío o falta base44/config.jsonc; el usuario dice "crear una aplicación/proyecto Base44", "configurar Base44", "implementar en Base44", "enviar entidades". Esta habilidad es el PRERREQUISITO para base44-sdk; utilícela siempre primero para nuevos proyectos antes de crear funciones.
