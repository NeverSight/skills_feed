À utiliser pour les opérations CLI Base44 et l’initialisation du projet. Déclencheurs : l'utilisateur souhaite créer/initialiser un nouveau projet Base44 ; l'utilisateur mentionne les commandes CLI (npx base44, fil base44, créer, se connecter, se déconnecter, whoami, déployer, entités push, déploiement de site, déploiement de fonctions) ; le répertoire est vide ou manquant base44/config.jsonc ; l'utilisateur dit « créer une application/un projet Base44 », « configurer Base44 », « déployer sur Base44 », « envoyer des entités ». Cette compétence est le PRÉREQUIS pour base44-sdk - utilisez-la toujours en premier pour les nouveaux projets avant de créer des fonctionnalités.
