Úselo para escribir código JavaScript/TypeScript con el SDK Base44 en proyectos EXISTENTES. Desencadenantes: el usuario desea implementar funciones, crear funciones o escribir código utilizando Base44; el código utiliza importaciones '@base44/sdk'; el usuario menciona métodos SDK como base44.entities, base44.auth, base44.agents, base44.functions, base44.integrations; el usuario dice "agregar una característica", "implementar", "compilar componente", "escribir código para". NO para: crear nuevos proyectos, comandos CLI (npx base44 create/deploy/login) o directorios sin base44/config.jsonc; use base44-cli en su lugar.
