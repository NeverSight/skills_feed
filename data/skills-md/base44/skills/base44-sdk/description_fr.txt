À utiliser pour écrire du code JavaScript/TypeScript avec le SDK Base44 dans des projets EXISTANTS. Déclencheurs : l'utilisateur souhaite implémenter des fonctionnalités, créer des fonctionnalités ou écrire du code à l'aide de Base44 ; le code utilise les importations '@base44/sdk' ; l'utilisateur mentionne les méthodes du SDK telles que base44.entities, base44.auth, base44.agents, base44.functions, base44.integrations ; l'utilisateur dit "ajouter une fonctionnalité", "implémenter", "construire un composant", "écrire du code pour". PAS pour : la création de nouveaux projets, de commandes CLI (npx base44 create/deploy/login) ou de répertoires sans base44/config.jsonc - utilisez plutôt base44-cli.
