Используйте для написания кода JavaScript/TypeScript с помощью Base44 SDK в СУЩЕСТВУЮЩИХ проектах. Триггеры: пользователь хочет реализовать функции, построить функциональность или написать код с использованием Base44; код использует импорт @base44/sdk; пользователь упоминает такие методы SDK, как base44.entities, base44.auth, base44.agents, base44.functions, base44.integrations; Пользователь говорит: «Добавить функцию», «Реализовать», «Создать компонент», «Написать код». НЕ для: создания новых проектов, команд CLI (npx base44 create/deploy/login) или каталогов без base44/config.jsonc — вместо этого используйте base44-cli.
