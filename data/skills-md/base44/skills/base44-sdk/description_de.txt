Zum Schreiben von JavaScript-/TypeScript-Code mit dem Base44 SDK in VORHANDENEN Projekten verwenden. Auslöser: Der Benutzer möchte Features implementieren, Funktionen erstellen oder Code mit Base44 schreiben. Code verwendet „@base44/sdk“-Importe; Benutzer erwähnt SDK-Methoden wie base44.entities, base44.auth, base44.agents, base44.functions, base44.integrations; Der Benutzer sagt „Funktion hinzufügen“, „Implementieren“, „Komponente erstellen“, „Code schreiben für“. NICHT für: Erstellen neuer Projekte, CLI-Befehle (npx base44 create/deploy/login) oder Verzeichnisse ohne base44/config.jsonc – verwenden Sie stattdessen base44-cli.
