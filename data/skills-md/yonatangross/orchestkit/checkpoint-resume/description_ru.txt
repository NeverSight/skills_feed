Устойчивый к ограничению скорости конвейер с контрольной точкой/возобновлением для длительных многофазных сеансов. Сохраняет прогресс в .claude/pipeline-state.json после каждого этапа. Используйте при запуске сложной многоэтапной задачи, которая может привести к превышению ограничений скорости, при возобновлении прерванного сеанса или при организации работы, охватывающей коммиты, проблемы GitHub и большие изменения файлов.
