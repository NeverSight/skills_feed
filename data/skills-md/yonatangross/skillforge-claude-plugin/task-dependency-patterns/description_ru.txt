CC 2.1.16 Шаблоны управления задачами с помощью инструментов TaskCreate, TaskUpdate, TaskGet, TaskList. Разбивайте сложную работу на отслеживаемые задачи с помощью цепочек зависимостей. Используйте при управлении многоэтапными реализациями, координации параллельной работы или отслеживании статуса завершения.
