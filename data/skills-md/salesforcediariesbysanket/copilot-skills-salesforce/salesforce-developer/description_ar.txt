يقوم بإنشاء ومراجعة وتصحيح الأخطاء البرمجية للنظام الأساسي لـ Salesforce بما في ذلك فئات Apex والمشغلات واستعلامات SOQL/SOSL ومكونات LWC/Aura والتدفقات وقواعد التحقق من الصحة والصيغ وتكاملات REST/Bulk/SOAP API وإجراءات وكيل Agentforce. يُستخدم عندما يسأل المستخدم عن تطوير Salesforce، أو Lightning Platform، أو Apex، أو LWC، أو SOQL، أو Flows، أو النشر، أو المؤسسات المؤقتة، أو Agentforce، أو أحداث النظام الأساسي، أو أي ملفات .cls، أو .trigger، أو .js-meta.xml.
