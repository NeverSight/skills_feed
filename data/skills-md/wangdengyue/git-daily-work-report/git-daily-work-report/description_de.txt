Erstellen Sie automatisch tägliche Arbeitsberichte, indem Sie Git-Repositorys scannen. Verwenden Sie diese Fähigkeit, wenn der Benutzer Folgendes verlangt: (1) Erstellen eines täglichen Berichts aus Git-Commits, (2) Zusammenfassen der an einem bestimmten Datum durchgeführten Arbeit basierend auf Codeänderungen, (3) Überprüfen von Commits und Erstellen einer Arbeitszusammenfassung für ein Datum. Der Skill durchsucht alle Git-Repositorys in einem Stammverzeichnis, filtert nach Autor, ruft Commit-Datensätze mit Dateiänderungen ab und generiert mithilfe der LLM-Analyse eine zusammengefasste Beschreibung des Arbeitsinhalts.
