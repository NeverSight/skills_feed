Générez automatiquement des rapports de travail quotidiens en analysant les référentiels Git. Utilisez cette compétence lorsque l'utilisateur demande à : (1) Générer un rapport quotidien à partir des commits git, (2) Résumer le travail effectué à une date spécifique en fonction des modifications de code, (3) Vérifier les commits et créer un résumé du travail pour une date. La compétence analyse tous les référentiels git sous un répertoire racine, filtre par auteur, récupère les enregistrements de validation avec les modifications de fichiers et génère une description résumée du contenu du travail à l'aide de l'analyse LLM.
