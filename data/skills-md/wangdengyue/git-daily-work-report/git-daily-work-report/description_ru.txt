Автоматически создавайте ежедневные отчеты о работе, сканируя репозитории Git. Используйте этот навык, когда пользователь просит: (1) создать ежедневный отчет по коммитам git, (2) обобщить работу, выполненную за определенную дату, на основе изменений кода, (3) проверить коммиты и создать сводку работы за определенную дату. Этот навык сканирует все репозитории git в корневом каталоге, фильтрует по автору, извлекает записи коммитов с изменениями файлов и генерирует сводное описание содержимого работы с использованием LLM-анализа.
