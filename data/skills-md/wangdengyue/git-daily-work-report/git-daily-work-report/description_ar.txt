قم بإنشاء تقارير العمل اليومية تلقائيًا عن طريق مسح مستودعات Git. استخدم هذه المهارة عندما يطلب المستخدم ما يلي: (1) إنشاء تقرير يومي من التزامات git، (2) تلخيص العمل المنجز في تاريخ محدد بناءً على تغييرات التعليمات البرمجية، (3) التحقق من الالتزامات وإنشاء ملخص عمل للتاريخ. تقوم هذه المهارة بمسح جميع مستودعات git ضمن الدليل الجذر، والتصفية حسب المؤلف، واسترداد سجلات الالتزام مع تغييرات الملف، وإنشاء وصف ملخص لمحتوى العمل باستخدام تحليل LLM.
