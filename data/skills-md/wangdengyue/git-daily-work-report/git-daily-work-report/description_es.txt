Genere automáticamente informes de trabajo diarios escaneando repositorios de Git. Utilice esta habilidad cuando el usuario solicite: (1) Generar un informe diario a partir de confirmaciones de git, (2) Resumir el trabajo realizado en una fecha específica en función de los cambios de código, (3) Verificar las confirmaciones y crear un resumen de trabajo para una fecha. La habilidad escanea todos los repositorios de git en un directorio raíz, filtra por autor, recupera registros de confirmación con cambios de archivos y genera una descripción resumida del contenido del trabajo mediante el análisis LLM.
