Stellen Sie eine kostenpflichtige API-Anfrage an einen x402-Endpunkt mit automatischer USDC-Zahlung. Verwenden Sie diese Option, wenn Sie oder der Benutzer eine kostenpflichtige API aufrufen, eine x402-Anfrage stellen, einen kostenpflichtigen Dienst nutzen oder für einen API-Aufruf bezahlen möchten. Verwenden Sie diese Option, nachdem Sie einen Dienst mit der Funktion „Suche nach Dienst“ gefunden haben.
