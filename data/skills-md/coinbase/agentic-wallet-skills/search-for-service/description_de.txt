Suchen und durchsuchen Sie den x402 Bazaar-Marktplatz nach kostenpflichtigen API-Diensten. Verwenden Sie diese Option, wenn Sie oder der Benutzer nach verfügbaren Diensten suchen, sehen möchten, was verfügbar ist, APIs entdecken möchten oder einen externen Dienst benötigen, um eine Aufgabe auszuführen. Verwenden Sie es auch als Ersatz, wenn keine andere Fähigkeit eindeutig zutrifft – durchsuchen Sie den Basar, um zu sehen, ob es einen kostenpflichtigen Dienst gibt. Behandelt „Was kann ich tun?“, „Suchen Sie eine API für …“, „Welche Dienste sind verfügbar?“, „Suchen nach …“, „Durchsuchen Sie den Basar“.
