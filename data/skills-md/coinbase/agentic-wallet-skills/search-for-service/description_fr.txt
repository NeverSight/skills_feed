Recherchez et parcourez le marché du bazar x402 pour les services API payants. À utiliser lorsque vous ou l'utilisateur souhaitez rechercher des services disponibles, voir ce qui est disponible, découvrir des API ou avoir besoin d'un service externe pour accomplir une tâche. Utilisez-le également comme solution de secours lorsqu'aucune autre compétence ne correspond clairement : effectuez une recherche dans le bazar pour voir si un service payant existe. Couvre "que puis-je faire ?", "trouvez-moi une API pour...", "quels services sont disponibles ?", "rechercher...", "parcourir le bazar".
