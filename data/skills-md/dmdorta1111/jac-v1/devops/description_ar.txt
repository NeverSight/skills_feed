نشر البنية الأساسية السحابية وإدارتها على Cloudflare (Workers وR2 وD1 وKV وPages وDurable Objects وBrowser Rendering) وحاويات Docker وGoogle Cloud Platform (Compute Engine وGKE وCloud Run وApp Engine وCloud Storage). يُستخدم عند نشر وظائف بدون خادم على الحافة، أو تكوين حلول حوسبة الحافة، أو إدارة حاويات وصور Docker، أو إعداد خطوط أنابيب CI/CD، أو تحسين تكاليف البنية التحتية السحابية، أو تنفيذ إستراتيجيات التخزين المؤقت العالمية، أو العمل مع قواعد البيانات السحابية، أو إنشاء تطبيقات سحابية أصلية.
