Boîte à outils d'automatisation Java pour Creo Parametric (J-Link/Object TOOLKIT Java). À utiliser lorsque les utilisateurs posent des questions sur : l'écriture d'applications Jlink, la gestion de sessions, les opérations de modèle/fonctionnalité/paramètre, la sélection interactive, l'automatisation d'assemblage, la génération de dessins, l'intégration d'interface utilisateur/menu, le traitement par lots, la gestion des erreurs, l'optimisation des performances ou toute automatisation de conception Creo par programmation via Java.
