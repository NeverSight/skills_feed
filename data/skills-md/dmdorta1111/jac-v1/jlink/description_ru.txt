Набор инструментов автоматизации Java для Creo Parametric (J-Link/Object ToolKIT Java). Используйте, когда пользователи спрашивают о: написании приложений Jlink, управлении сеансами, операциях с моделью/функцией/параметрами, интерактивном выборе, автоматизации сборки, создании чертежей, интеграции пользовательского интерфейса/меню, пакетной обработке, обработке ошибок, оптимизации производительности или любой программной автоматизации проектирования Creo с помощью Java.
