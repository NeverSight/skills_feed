Gérer les serveurs MCP (Model Context Protocol) : découvrez, analysez et exécutez des outils/invites/ressources à partir de serveurs MCP configurés. À utiliser lorsque vous travaillez avec des intégrations MCP, devez découvrir les fonctionnalités MCP disponibles, filtrer les outils MCP pour des tâches spécifiques, exécuter les outils MCP par programme, accéder aux invites/ressources MCP ou implémenter la fonctionnalité client MCP. Prend en charge la sélection intelligente d'outils, la gestion multi-serveurs et la découverte de capacités contextuelles.
