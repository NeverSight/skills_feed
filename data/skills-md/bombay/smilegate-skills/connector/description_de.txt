Eine Schritt-für-Schritt-Anleitung zur Einrichtung, um die Arbeitstools von Smilegate (Slack, Jira, Confluence, BISKIT, API Docs) mit Claude Code zu verbinden. So konzipiert, dass auch Nicht-Entwickler mitmachen können. Wird für Anfragen zu „Connector“, „MCP-Setup“, „Jira-Verbindung“, „Confluence-Verbindung“, „Slack-Verbindung“, „Biskit-Verbindung“, „Apidocs-Verbindung“ oder „API-Docs-Verbindung“ verwendet.
