Пошаговое руководство по настройке подключения рабочих инструментов Smilegate (Slack, Jira, Confluence, BISKIT, API Docs) к Claude Code. Разработано так, что даже не разработчики могут следовать инструкциям. Используйте для запросов о «соединителе», «настройке MCP», «подключении jira», «подключении confluence», «слабом соединении», «подключении biskit», «подключении apidocs» или «подключении API docs».
