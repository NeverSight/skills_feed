Un guide de configuration étape par étape pour connecter les outils de travail Smilegate (Slack, Jira, Confluence, BISKIT, API Docs) à Claude Code. Conçu pour que même les non-développeurs puissent suivre. À utiliser pour les requêtes concernant le « connecteur », la « configuration MCP », la « connexion jira », la « connexion confluence », la « connexion slack », la « connexion biskit », la « connexion apidocs » ou la « connexion api docs ».
