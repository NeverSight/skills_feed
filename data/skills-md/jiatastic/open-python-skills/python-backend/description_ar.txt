خبرة في تطوير الواجهة الخلفية لـ Python لـ FastAPI وأنماط الأمان وعمليات قاعدة البيانات وتكاملات Upstash وجودة التعليمات البرمجية. يُستخدم عندما: (1) إنشاء واجهات برمجة تطبيقات REST باستخدام FastAPI، (2) تنفيذ مصادقة JWT/OAuth2، (3) إعداد قواعد بيانات SQLAlchemy/غير المتزامنة، (4) دمج التخزين المؤقت لـ Redis/Upstash، (5) إعادة هيكلة كود Python الذي تم إنشاؤه بواسطة الذكاء الاصطناعي (إزالة الانحدار)، (6) تصميم أنماط واجهة برمجة التطبيقات، أو (7) تحسين أداء الواجهة الخلفية.
