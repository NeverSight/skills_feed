Опыт разработки серверной части Python для FastAPI, шаблонов безопасности, операций с базами данных, интеграции Upstash и качества кода. Используйте при: (1) создании REST API с помощью FastAPI, (2) реализации аутентификации JWT/OAuth2, (3) настройке баз данных SQLAlchemy/async, (4) интеграции кэширования Redis/Upstash, (5) рефакторинге кода Python, сгенерированного искусственным интеллектом (деслопификация), (6) разработке шаблонов API или (7) оптимизации производительности серверной части.
