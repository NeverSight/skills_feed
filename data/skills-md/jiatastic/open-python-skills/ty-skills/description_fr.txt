Expertise en vérification de type Python utilisant ty - le vérificateur de type extrêmement rapide d'Astral. À utiliser lorsque : (1) Ajout d'annotations de type au code Python, (2) Correction des erreurs de type signalées par ty, (3) Migration de mypy/pyright vers ty, (4) Configuration de ty pour les projets, (5) Compréhension des modèles de type avancés (génériques, protocoles, types d'intersection), (6) Configuration de ty dans les éditeurs (VS Code, Cursor, Neovim, PyCharm).
