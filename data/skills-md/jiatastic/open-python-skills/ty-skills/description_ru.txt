Опыт проверки типов Python с использованием ty — чрезвычайно быстрой программы проверки типов от Astral. Используйте, когда: (1) добавление аннотаций типов в код Python, (2) исправление ошибок типов, о которых сообщил ty, (3) переход с mypy/pyright на ty, (4) настройка ty для проектов, (5) понимание расширенных шаблонов типов (обобщенные шаблоны, протоколы, типы пересечений), (6) настройка ty в редакторах (VS Code, Cursor, Neovim, PyCharm).
