Observabilité structurée avec Pydantic Logfire et OpenTelemetry. À utiliser lorsque : (1) l'ajout de traces/journaux aux API Python, (2) l'instrumentation de FastAPI, HTTPX, SQLAlchemy ou LLM, (3) la configuration des métadonnées du service, (4) la configuration de l'échantillonnage ou du nettoyage des données sensibles, (5) le test du code d'observabilité.
