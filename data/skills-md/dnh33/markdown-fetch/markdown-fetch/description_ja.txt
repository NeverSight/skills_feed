クロードが Web URL からコンテンツをフェッチ、読み取り、抽出、または分析する必要がある場合は常に、このスキルを使用します。生の HTML をフェッチする代わりに、markdown.new サービスを使用して、Web ページをクリーンでトークン効率の高いマークダウンに変換します。ユーザーが URL を提供し、そのコンテンツの要約、引用、分析、比較、抽出、または処理を希望するときにトリガーされます。また、Claude がドキュメント、ブログ投稿、記事、Wiki、リリース ノート、変更ログ、または Web でホストされているテキスト コンテンツを読む必要があるときにもトリガーされます。ユーザーが何も指示せずに URL を貼り付けるだけの場合でも、このスキルを使用してください。バイナリ ファイル、認証されたページ、または JSON/XML を返す API エンドポイントには使用しないでください。
