SDK TypeScript/JavaScript pour les échanges de jetons entre chaînes et même chaîne, les ponts et les opérations DeFi.

UTILISEZ CETTE COMPÉTENCE POUR :
- Échangez des jetons entre différentes blockchains (par exemple, "échangez USDC sur Ethereum contre ETH sur Arbitrum")
- Ponter les jetons vers une autre chaîne (par exemple, "transférer mon ETH du réseau principal vers la base")
- Échangez des jetons sur la même chaîne avec les meilleurs taux DEX (par exemple, "échangez ETH contre DAI sur Polygon")
- Obtenez des devis ou comparez les itinéraires pour les échanges de jetons sur toutes les chaînes
- Exécuter des workflows DeFi en 1 clic (zap dans LP, pont + échange + dépôt, entrée de rendement inter-chaînes)
- Créez une dApp, un portefeuille ou une interface prenant en charge les opérations de jetons multi-chaînes
- Créez des flux de paiement qui acceptent n'importe quel jeton et se convertissent en un jeton spécifique
- Créez des applications React, Next.js, Vue ou Node.js avec une fonctionnalité d'échange inter-chaînes
- Gérer le cycle de vie des transactions inter-chaînes (suivi du statut, tentatives, remboursements)
- Recherchez les ponts, DEX, chaînes ou jetons disponibles pour les opérations inter-chaînes
