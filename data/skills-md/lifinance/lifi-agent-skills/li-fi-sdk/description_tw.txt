TypeScript/JavaScript SDK，用於跨鍊和同鏈代幣交換、橋接和 DeFi 操作。

使用此技能可以：
- 在不同區塊鏈之間交換代幣（例如，“將以太坊上的 USDC 交換為 Arbitrum 上的 ETH”）
- 將代幣橋接到另一個鏈（例如，“將我的 ETH 從主網轉移到 Base”）
- 在同一條鏈上以最佳 DEX 匯率交換代幣（例如，“在 Polygon 上將 ETH 交換為 DAI”）
- 獲取報價或比較跨任何鏈的代幣交換路線
- 執行一鍵式 DeFi 工作流程（轉換為 LP、橋接+交換+存款、跨鏈收益入口）
- 構建支持多鏈代幣操作的 dApp、錢包或前端
- 創建接受任何代幣並轉換為特定代幣的支付流程
- 構建具有跨鏈交換功能的 React、Next.js、Vue 或 Node.js 應用程序
- 處理跨鏈交易生命週期（狀態跟踪、重試、退款）
- 查找可用的橋樑、DEX、鍊或代幣以進行跨鏈操作
