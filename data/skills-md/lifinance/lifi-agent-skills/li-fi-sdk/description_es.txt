SDK de TypeScript/JavaScript para intercambios de tokens, puentes y operaciones DeFi entre cadenas y en la misma cadena.

UTILIZA ESTA HABILIDAD PARA:
- Intercambiar tokens entre diferentes blockchains (por ejemplo, "intercambiar USDC en Ethereum por ETH en Arbitrum")
- Puentear tokens a otra cadena (por ejemplo, "transferir mi ETH de la red principal a la Base")
- Intercambie tokens en la misma cadena con las mejores tarifas DEX (por ejemplo, "intercambie ETH por DAI en Polygon")
- Obtenga cotizaciones o compare rutas para intercambios de tokens en cualquier cadena
- Ejecute flujos de trabajo DeFi con 1 clic (zap a LP, puente + intercambio + depósito, entrada de rendimiento entre cadenas)
- Cree una dApp, una billetera o una interfaz que admita operaciones de tokens de múltiples cadenas
- Cree flujos de pago que acepten cualquier token y lo conviertan a un token específico
- Cree aplicaciones React, Next.js, Vue o Node.js con funcionalidad de intercambio entre cadenas
- Manejar el ciclo de vida de las transacciones entre cadenas (seguimiento de estado, reintentos, reembolsos)
- Encuentre puentes, DEX, cadenas o tokens disponibles para operaciones entre cadenas
