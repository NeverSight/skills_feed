TypeScript/JavaScript SDK für kettenübergreifende und gleichketteige Token-Swaps, Bridging und DeFi-Operationen.

NUTZEN SIE DIESE FÄHIGKEIT, UM:
- Tauschen Sie Token zwischen verschiedenen Blockchains aus (z. B. „Swap USDC on Ethereum to ETH on Arbitrum“)
- Bridge-Tokens zu einer anderen Kette (z. B. „meine ETH vom Mainnet zur Base übertragen“)
- Tauschen Sie Token in derselben Kette mit den besten DEX-Kursen aus (z. B. „ETH gegen DAI auf Polygon tauschen“).
- Erhalten Sie Angebote oder vergleichen Sie Routen für Token-Swaps über beliebige Ketten hinweg
- Führen Sie 1-Klick-DeFi-Workflows aus (Zap in LP, Bridge+Swap+Deposit, Cross-Chain-Ertragseintrag)
- Erstellen Sie eine dApp, ein Wallet oder ein Frontend, das Multi-Chain-Token-Operationen unterstützt
- Erstellen Sie Zahlungsflüsse, die jeden Token akzeptieren und in einen bestimmten Token umwandeln
- Erstellen Sie React-, Next.js-, Vue- oder Node.js-Apps mit Cross-Chain-Swap-Funktionalität
- Abwicklung des kettenübergreifenden Transaktionslebenszyklus (Statusverfolgung, Wiederholungsversuche, Rückerstattungen)
- Finden Sie verfügbare Brücken, DEXs, Ketten oder Token für kettenübergreifende Operationen
