クロスチェーンおよび同一チェーンのトークン スワップ、ブリッジング、および DeFi 操作のための REST API。

ユーザーが次のことをしたい場合にこのスキルを使用してください。
- 異なるブロックチェーン間でトークンを交換します (例: 「イーサリアム上の USDC をアービトラム上の ETH に交換する」)
- トークンを別のチェーンにブリッジします (例: 「ETH をメインネットからオプティミズムに移動する」)
- 同じチェーン上で最良のレートでトークンを交換します (例: 「Polygon で ETH を USDC に交換」)
- チェーン間でのトークン交換の最適なルートまたは見積りを見つける
- チェーン全体で DeFi オペレーションを実行 (ザップ、ブリッジ + スワップ + デポジット、イールド ファーミング エントリー)
- マルチチェーンの支払いフローを構築 (任意のトークンを受け入れ、特定のトークンで決済)
- クロスチェーン転送用にサポートされているチェーン、トークン、またはブリッジを確認します。
- クロスチェーントランザクションのステータスを追跡する
- クロスチェーンスワップが必要なバックエンドサービス (Python、Go、Rust など) を構築する
- HTTP/REST (JavaScript SDK ではない) を介してクロスチェーン機能を統合します。
