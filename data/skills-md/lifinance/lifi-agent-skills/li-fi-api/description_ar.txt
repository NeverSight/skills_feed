REST API لمبادلات الرموز المميزة عبر السلسلة ونفس السلسلة وعمليات التجسير وعمليات التمويل اللامركزي.

استخدم هذه المهارة عندما يريد المستخدم:
- تبديل الرموز بين سلاسل الكتل المختلفة (على سبيل المثال، "مبادلة USDC على Ethereum بـ ETH على Arbitrum")
- نقل الرموز المميزة إلى سلسلة أخرى (على سبيل المثال، "انقل ETH الخاص بي من الشبكة الرئيسية إلى Optimism")
- مبادلة الرموز المميزة على نفس السلسلة بأفضل الأسعار (على سبيل المثال، "مبادلة ETH إلى USDC على Polygon")
- ابحث عن أفضل طريق أو عرض أسعار لمبادلة الرمز المميز عبر السلاسل
- تنفيذ عمليات DeFi عبر السلاسل (zap، Bridge+swap+deposit، دخول زراعة العائد)
- إنشاء تدفقات دفع متعددة السلاسل (قبول أي رمز مميز، والتسوية في رمز مميز محدد)
- التحقق من السلاسل أو الرموز المميزة أو الجسور المدعومة لعمليات النقل عبر السلسلة
- تتبع حالة المعاملة عبر السلسلة
- إنشاء خدمات الواجهة الخلفية (Python، Go، Rust، وما إلى ذلك) التي تحتاج إلى مقايضات عبر السلسلة
- دمج الوظائف عبر السلسلة عبر HTTP/REST (وليس JavaScript SDK)
