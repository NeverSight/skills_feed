크로스체인 및 동일 체인 토큰 스왑, 브리징 및 DeFi 작업을 위한 REST API입니다.

사용자가 원할 때 이 기술을 사용하십시오:
- 서로 다른 블록체인 간 토큰 교환(예: "이더리움의 USDC를 Arbitrum의 ETH로 교환")
- 토큰을 다른 체인에 연결합니다(예: "내 ETH를 메인넷에서 Optimism으로 이동")
- 동일한 체인에서 가장 좋은 가격으로 토큰을 교환합니다(예: "Polygon에서 ETH를 USDC로 교환")
- 체인 간 토큰 스왑을 위한 최적의 경로 또는 견적 찾기
- 체인 전반에 걸쳐 DeFi 작업 실행(Zap, 브리지+스왑+예금, 이자 농사 항목)
- 다중 체인 결제 흐름 구축(모든 토큰 수락, 특정 토큰 결제)
- 크로스체인 전송이 지원되는 체인, 토큰 또는 브리지를 확인하세요.
- 크로스체인 거래 상태 추적
- 크로스체인 스왑이 필요한 백엔드 서비스(Python, Go, Rust 등) 구축
- HTTP/REST(JavaScript SDK 아님)를 통해 크로스체인 기능 통합
