Hyprland - مركب Wayland ذو بلاط ديناميكي مع رسوم متحركة مذهلة وتأثيرات متسارعة بواسطة GPU.
يُستخدم عند تكوين مدير نوافذ Hyprland، أو روابط المفاتيح، أو الشاشات، أو مساحات العمل، أو الرسوم المتحركة، أو قواعد النوافذ، أو العمل باستخدام أدوات النظام البيئي Hypr.
