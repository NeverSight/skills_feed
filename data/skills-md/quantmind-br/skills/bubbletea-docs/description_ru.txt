Bubble Tea — это платформа Go для создания элегантных пользовательских интерфейсов терминала (TUI).
Используйте при создании приложений CLI с интерактивными меню, формами, списками, таблицами или любым пользовательским интерфейсом терминала.
На основе архитектуры Elm (модель-представление-обновление). Ключевые особенности: ввод с клавиатуры/мыши, адаптивная раскладка, 
асинхронные команды, компоненты Bubbles (спиннер, список, таблица, область просмотра, текстовый ввод, прогресс).
Включает блеск для губ для укладки и Huh для формы.
