Bubble Tea は、エレガントなターミナル ユーザー インターフェイス (TUI) を構築するための Go フレームワークです。
インタラクティブなメニュー、フォーム、リスト、テーブル、または任意のターミナル UI を備えた CLI アプリケーションを構築する場合に使用します。
Elm アーキテクチャ (モデル-ビュー-更新) に基づいています。主な機能: キーボード/マウス入力、応答性の高いレイアウト、 
async コマンド、Bubbles コンポーネント (スピナー、リスト、テーブル、ビューポート、テキスト入力、進行状況)。
スタイリング用のリップグロスとフォーム用のHuhが含まれています。
