Bubble Tea es un marco de Go para crear elegantes interfaces de usuario de terminales (TUI).
Úselo al crear aplicaciones CLI con menús interactivos, formularios, listas, tablas o cualquier interfaz de usuario de terminal.
Basado en The Elm Architecture (Modelo-Vista-Actualización). Características clave: entrada de teclado/ratón, diseños responsivos, 
Comandos asíncronos, componentes de Bubbles (spinner, lista, tabla, ventana gráfica, entrada de texto, progreso).
Incluye Lip Gloss para estilizar y Huh para formas.
