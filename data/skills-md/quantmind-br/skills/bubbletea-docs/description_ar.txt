Bubble Tea هو إطار عمل Go لبناء واجهات مستخدم طرفية أنيقة (TUIs).
يُستخدم عند إنشاء تطبيقات CLI باستخدام القوائم التفاعلية، أو النماذج، أو القوائم، أو الجداول، أو أي واجهة مستخدم طرفية.
بناءً على بنية Elm (نموذج-عرض-تحديث). الميزات الرئيسية: إدخال لوحة المفاتيح/الماوس، والتخطيطات سريعة الاستجابة، 
أوامر غير متزامنة، مكونات الفقاعات (الدوار، القائمة، الجدول، إطار العرض، إدخال النص، التقدم).
يشتمل على ملمع الشفاه للتصفيف وهوه للأشكال.
