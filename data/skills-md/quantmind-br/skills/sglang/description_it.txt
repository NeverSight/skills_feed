SGLang - Framework di servizio ad alte prestazioni per modelli linguistici e multimodali di grandi dimensioni.
Da utilizzare durante la distribuzione di LLM su larga scala, la creazione di server di inferenza, l'utilizzo di API compatibili con OpenAI, l'ottimizzazione delle prestazioni del modello o l'integrazione di output strutturati.
Parole chiave: sglang, llm-serving, motore di inferenza, openai-api, multimodale, inferenza batch, output strutturati, quantizzazione, decodifica speculativa, kv-cache, parallelismo tensore, alternativa vllm.
