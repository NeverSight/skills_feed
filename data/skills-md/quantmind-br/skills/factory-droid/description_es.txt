Factory Droid: plataforma de desarrollo de software nativa de IA con CLI y aplicación de escritorio.
Úselo cuando trabaje con la CLI Droid de Factory, configure agentes de IA, configure AGENTS.md, 
personalizar modelos (BYOK), crear ganchos, habilidades o comandos de barra personalizados.
Palabras clave: droide, factory-ai, cli, ai-agent, AGENTS.md, byok, modelos personalizados, ganchos, 
habilidades, modo de especificación, ejecución automática, empresa, mcp, integración de github, revisión de código.
