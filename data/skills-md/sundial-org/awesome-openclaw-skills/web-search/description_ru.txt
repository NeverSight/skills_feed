Этот навык следует использовать, когда пользователям необходимо искать информацию в Интернете, находить текущий контент, просматривать новостные статьи, искать изображения или видео. Он использует API поиска DuckDuckGo для возврата результатов в чистом, отформатированном виде (текст, уценка или JSON). Используйте для исследований, проверки фактов, поиска свежей информации или сбора веб-ресурсов.
