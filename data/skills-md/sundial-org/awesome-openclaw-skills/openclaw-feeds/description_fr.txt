Agrégateur de nouvelles RSS. Récupère les titres des flux organisés dans trois catégories : actualités, jeux et finance. À utiliser lorsque l'utilisateur pose des questions sur l'actualité, les gros titres, ce qui se passe, ce qui se passe, ou dit « quoi de neuf dans l'actualité », « quoi de neuf dans la finance », « quoi de neuf dans les jeux » ou les équivalents allemands « was geht mit nachrichten », « was geht mit money », « was geht mit gaming ». S'active également pour les requêtes telles que « donnez-moi un aperçu de l'actualité », « les derniers titres », « l'actualité du marché », « l'actualité des jeux », « l'actualité technologique », « le tour d'horizon financier » ou « briefing ». Renvoie le JSON structuré à partir de flux RSS publics – aucune clé API, aucune recherche sur le Web nécessaire.
