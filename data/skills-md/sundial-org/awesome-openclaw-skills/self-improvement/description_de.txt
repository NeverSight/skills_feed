Erfasst Erkenntnisse, Fehler und Korrekturen, um eine kontinuierliche Verbesserung zu ermöglichen. Wird verwendet, wenn: (1) ein Befehl oder eine Operation unerwartet fehlschlägt, (2) der Benutzer Claude korrigiert („Nein, das ist falsch...“, „Eigentlich...“), (3) der Benutzer eine Funktion anfordert, die nicht existiert, (4) eine externe API oder ein externes Tool fehlschlägt, (5) Claude erkennt, dass sein Wissen veraltet oder falsch ist, (6) ein besserer Ansatz für eine wiederkehrende Aufgabe gefunden wird. Überprüfen Sie auch die Erkenntnisse vor größeren Aufgaben.
