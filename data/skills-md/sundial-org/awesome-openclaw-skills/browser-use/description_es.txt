Utilice el navegador Utilice la API de la nube para activar navegadores en la nube para Clawdbot y ejecutar tareas de navegador autónomas. El uso principal es crear sesiones de navegador con perfiles (inicios de sesión persistentes/cookies) que Clawdbot puede controlar. El uso secundario es ejecutar subagentes de tareas para una automatización rápida y autónoma del navegador. Documentos en docs.browser-use.com y docs.cloud.browser-use.com.
