Búsqueda avanzada de Twitter y análisis de datos de redes sociales. Obtiene tweets por palabras clave utilizando la API de Twitter, procesa hasta 1000 resultados y genera informes de análisis de datos profesionales con información valiosa y recomendaciones prácticas. Úselo cuando el usuario solicite búsqueda de redes sociales Twitter/X, análisis de tendencias de redes sociales, extracción de datos de tweets, escucha social, identificación de personas influyentes, análisis de sentimientos sobre temas de tweets o cualquier tarea que implique recopilar y analizar datos de Twitter para obtener información.
