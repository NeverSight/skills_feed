Библиотека проверенных шаблонов запросов безопасности для безопасной реализации функций. Используйте при реализации форм, конечных точек, аутентификации, авторизации, загрузки файлов или проведения проверок безопасности. Триггеры включают «запрос безопасности», «безопасную форму», «RBAC», «модель угроз», «STRIDE», «конечную точку администратора», «загрузку файла», «тестирование безопасности», «проверку кода», «OWASP».
