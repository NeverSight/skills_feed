قم بتكوين رؤوس الأمان للدفاع ضد هجمات النقر، وXSS، وارتباك MIME، وهجمات تجريد SSL. استخدم هذه المهارة عندما تحتاج إلى إعداد سياسة أمان المحتوى أو خيارات الإطار X أو HSTS أو تكوين رؤوس البرامج الوسيطة أو فهم ميزات أمان المتصفح. تتضمن المشغلات "رؤوس الأمان"، و"CSP"، و"سياسة أمان المحتوى"، و"خيارات الإطار X"، و"HSTS"، و"clickjacking"، و"ارتباك MIME"، و"رؤوس البرامج الوسيطة".
