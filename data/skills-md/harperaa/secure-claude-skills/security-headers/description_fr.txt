Configurez les en-têtes de sécurité pour vous défendre contre les attaques de détournement de clic, de confusion XSS, MIME et de suppression SSL. Utilisez cette compétence lorsque vous devez configurer Content-Security-Policy, X-Frame-Options, HSTS, configurer les en-têtes middleware ou comprendre les fonctionnalités de sécurité du navigateur. Les déclencheurs incluent les « en-têtes de sécurité », « CSP », « politique de sécurité du contenu », « X-Frame-Options », « HSTS », « clickjacking », « confusion MIME », « en-têtes middleware ».
