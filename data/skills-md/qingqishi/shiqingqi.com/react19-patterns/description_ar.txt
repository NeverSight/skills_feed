React 19 نمطًا وسلوك React Compiler مع بناء جملة مختصر للسياق وخطاف use(). يُستخدم عند العمل مع السياق، أو useContext، أو خطاف use()، أو مكونات الموفر، أو أنماط التحسين مثل useMemo، أو useCallback، أو memo، أو memoization، أو عندما يذكر المستخدم React 19، أو React Compiler، أو context.Provider، أو التحسين اليدوي.
