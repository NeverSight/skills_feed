سير عمل إنشاء كود TMDB API مع مخططات Zod انتقائية باستخدام pnpm codegen:tmdb. يُستخدم عند العمل مع نقاط نهاية TMDB، أو إعادة إنشاء الأنواع، أو إضافة وظائف TMDB API، أو تعديل endpoints-config.js، أو tmdb-server-functions.ts، أو مخططات Zod، أو عندما يذكر المستخدم كود TMDB، أو endpoints-config، أو pnpm codegen:tmdb، أو needZodSchema، أو ملفات TMDB التي يتم إنشاؤها تلقائيًا.
