Workflow de génération de code API TMDB avec schémas Zod sélectifs utilisant pnpm codegen:tmdb. À utiliser lorsque vous travaillez avec des points de terminaison TMDB, régénérez des types, ajoutez des fonctionnalités de l'API TMDB, modifiez endpoints-config.js, tmdb-server-functions.ts, les schémas Zod ou lorsque l'utilisateur mentionne TMDB codegen, endpoints-config, pnpm codegen:tmdb, a besoin de ZodSchema ou de fichiers TMDB générés automatiquement.
