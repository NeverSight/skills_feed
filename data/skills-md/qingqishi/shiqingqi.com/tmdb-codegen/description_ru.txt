Рабочий процесс генерации кода API TMDB с выборочными схемами Zod с использованием pnpm codegen:tmdb. Используйте при работе с конечными точками TMDB, регенерации типов, добавлении функций API TMDB, изменении endpoints-config.js, tmdb-server-functions.ts, схем Zod или когда пользователь упоминает TMDB codegen, endpoints-config, pnpm codegen:tmdb, NeedZodSchema или автоматически создаваемые файлы TMDB.
