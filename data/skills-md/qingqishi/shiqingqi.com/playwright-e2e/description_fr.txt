Meilleures pratiques de test de bout en bout (e2e) de Playwright pour les tests centrés sur l'utilisateur à l'aide de localisateurs sémantiques. À utiliser lors de l'écriture de tests E2E, de tests d'intégration, de tests de flux utilisateur, de tests Playwright, de spécifications de test ou lorsque l'utilisateur mentionne Playwright, les tests e2e, getByRole, les flux de tests ou les tests utilisateur.
