Modelli di internazionalizzazione (i18n) per componenti server e client che utilizzano getTranslations e useTranslations. Da utilizzare quando si lavora con traduzioni, impostazioni locali, contenuto multilingue, file di traduzione, TranslationContextProvider, cambio di impostazioni locali o quando l'utente menziona i18n, traduzioni, getTranslations, useTranslations o file translation.json.
