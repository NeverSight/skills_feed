Patrones de internacionalización (i18n) para componentes de servidor y cliente utilizando getTranslations y useTranslations. Úselo cuando trabaje con traducciones, configuraciones regionales, contenido multilingüe, archivos de traducción, TranslationContextProvider, cambio de configuración regional o cuando el usuario mencione archivos i18n, traducciones, getTranslations, useTranslations o Translation.json.
