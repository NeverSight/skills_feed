Шаблоны выборки данных для серверных/клиентских компонентов с использованием API выборки, TanStack Query, useSuspenseQuery, apiRouteWrapper и apiRequestWrapper. Используйте при реализации загрузки данных, вызовов API, функций сервера, запросов, мутаций, маршрутов API или когда пользователь упоминает TanStack Query, useSuspenseQuery, apiRouteWrapper, apiRequestWrapper, tmdb-server-functions или выборку данных.
