Ejecute una auditoría de código multidisciplinaria de dos pasos dirigida por un líder de desempate, combinando seguridad, rendimiento, UX, DX y análisis de casos extremos en un informe priorizado con correcciones concretas. Úselo cuando el usuario solicite auditar el código, realizar una revisión profunda, probar una base de código o producir un plan de remediación clasificado por riesgo en backend, frontend, API, scripts de infraestructura y flujos de productos. Primero mantenga la pila de auditoría independiente y luego agregue comprobaciones específicas del tiempo de ejecución como una superposición.
