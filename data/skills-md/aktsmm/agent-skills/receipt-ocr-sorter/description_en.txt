OCR Receipt Auto-Sorting Tool. Reads receipts from images (JPEG/PNG)/PDF/ZIP/videos (MP4/MOV) via OCR, extracts date, amount, country, card, and purpose, renames to standard filenames, and sorts by project. Includes Markdown summary report. Supports D365 Expense category mapping and receipt attachment integration. Use when: (1) organizing receipts, (2) receipt sorting, (3) categorizing receipts, (4) organizing expense receipts into folders, (5) renaming via OCR, (6) D365 expense category mapping. Compatible with Surya OCR + GPU (CUDA).
