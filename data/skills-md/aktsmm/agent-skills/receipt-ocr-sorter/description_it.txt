Strumento di ordinamento automatico delle ricevute OCR. Legge l'OCR delle ricevute nei formati immagine (JPEG/PNG)/PDF/ZIP/video (MP4/MOV), estrae data, importo, paese, carta e scopo, le rinomina con un nome file standard e le ordina per progetto. Include un rapporto di riepilogo Markdown. Supporta la mappatura delle categorie di spesa D365 e l'integrazione degli allegati alle ricevute. Da utilizzare quando: (1) organizzare le ricevute, (2) ordinare le ricevute, (3) classificare le ricevute, (4) organizzare le ricevute di spesa in cartelle, (5) rinominare tramite OCR, (6) mappare le categorie di spesa D365. Compatibile con Surya OCR + GPU (CUDA).
