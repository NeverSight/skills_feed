⚠️ Alpha (in fase di sviluppo) – Competenza di provisioning dell'ambiente Enterprise Azure. Supporta l'utilizzo di AVM, l'inizializzazione delle app VM, la distribuzione delle app nel servizio app/AKS/app contenitore, l'integrazione della configurazione tra servizi e la generazione di CI/CD. Compatibile con architetture complesse come Hub-Spoke, AKS, AI Foundry, ecc. Recupera gli schemi più recenti tramite Bicep MCP e Microsoft Learn Docs MCP.
