Автоматизация браузера с использованием Playwright MCP для веб-тестирования, проверки пользовательского интерфейса и автоматизации форм. Используйте при навигации по веб-сайтам, нажатии элементов, заполнении форм, создании снимков экрана или тестировании веб-приложений. Поддерживает операции iframe и сложное выполнение JavaScript.
