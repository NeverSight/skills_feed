Оптимизированное для графического процессора распознавание текста с использованием Surya. Используйте, когда: (1) Извлечение текста из изображений/снимков экрана, (2) Обработка PDF-файлов со встроенными изображениями, (3) Оптическое распознавание символов многоязычных документов, (4) Анализ макета и обнаружение таблиц. Поддерживает более 90 языков с двукратной точностью по сравнению с Tesseract.
