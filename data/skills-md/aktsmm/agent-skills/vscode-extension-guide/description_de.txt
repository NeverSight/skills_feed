Leitfaden zum Erstellen von VS-Code-Erweiterungen von Grund auf bis zur Marketplace-Veröffentlichung. Wird verwendet, wenn: (1) eine neue VS Code-Erweiterung erstellt wird, (2) Befehle, Tastenkombinationen oder Einstellungen zu einer Erweiterung hinzugefügt werden, (3) im VS Code Marketplace veröffentlicht wird, (4) Probleme bei der Aktivierung oder Verpackung von Erweiterungen behoben werden, (5) TreeView- oder Webview-Benutzeroberfläche erstellt wird, (6) Erweiterungstests eingerichtet werden.
