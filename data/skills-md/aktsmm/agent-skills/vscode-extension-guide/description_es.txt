Guía para crear extensiones de VS Code desde cero para publicación en Marketplace. Utilícelo cuando: (1) cree una nueva extensión de VS Code, (2) agregue comandos, combinaciones de teclas o configuraciones a una extensión, (3) publique en VS Code Marketplace, (4) solucione problemas de activación o empaquetado de extensiones, (5) cree una interfaz de usuario de TreeView o Webview, (6) configure pruebas de extensión.
