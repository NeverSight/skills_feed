Guida per creare estensioni VS Code da zero fino alla pubblicazione sul Marketplace. Da utilizzare quando: (1) si crea una nuova estensione VS Code, (2) si aggiungono comandi, combinazioni di tasti o impostazioni a un'estensione, (3) si pubblica su VS Code Marketplace, (4) si risolve l'attivazione dell'estensione o i problemi di confezionamento, (5) si crea l'interfaccia utente TreeView o Webview, (6) si configurano i test dell'estensione.
