Руководство по созданию расширений VS Code с нуля до публикации на Marketplace. Используйте при: (1) создании нового расширения VS Code, (2) добавлении команд, сочетаний клавиш или настроек в расширение, (3) публикации в VS Code Marketplace, (4) устранении проблем с активацией или упаковкой расширения, (5) создании пользовательского интерфейса TreeView или Webview, (6) настройке тестов расширения.
