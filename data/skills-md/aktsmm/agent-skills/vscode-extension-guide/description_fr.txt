Guide pour créer des extensions VS Code à partir de zéro jusqu'à la publication Marketplace. À utiliser lorsque : (1) Création d'une nouvelle extension VS Code, (2) Ajout de commandes, de raccourcis clavier ou de paramètres à une extension, (3) Publication sur VS Code Marketplace, (4) Dépannage des problèmes d'activation ou d'empaquetage d'extension, (5) Création de l'interface utilisateur TreeView ou Webview, (6) Configuration de tests d'extension.
