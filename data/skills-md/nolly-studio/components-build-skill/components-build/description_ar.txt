أنشئ مكونات React UI حديثة وقابلة للتركيب ويمكن الوصول إليها باتباع مواصفات Components.build. يُستخدم عند إنشاء أو مراجعة أو إعادة بناء مكتبات المكونات أو أنظمة التصميم أو أي مكونات واجهة مستخدم قابلة لإعادة الاستخدام. يتم تشغيلها على المهام التي تتضمن واجهات برمجة تطبيقات المكونات، أو أنماط التركيب، أو إمكانية الوصول، أو أنظمة التصميم، أو دعائم TypeScript.
