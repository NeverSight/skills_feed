Полное руководство по использованию библиотеки Kor UI (@korsolutions/ui) в приложениях React Native и Expo. Используйте этот навык при создании пользовательских интерфейсов с универсальным пользовательским интерфейсом, настройке тем, настройке библиотеки, работе с любым из более чем 24 компонентов (кнопка, ввод, выбор, оповещение, карточка, вкладки, меню, всплывающее окно, календарь, тост и т. д.), стилизации и темировании, реализации составных компонентов, отладке проблем с компонентами или когда пользователь упоминает «@korsolutions/ui», «Universal UI», «UIProvider» или спрашивает о неоформленных примитивах, настройке темы или Компоненты React Native UI. Этот навык охватывает установку, настройку поставщика, шаблоны использования компонентов, настройку тем, систему вариантов, перехватчики, адаптивный дизайн и устранение неполадок.
