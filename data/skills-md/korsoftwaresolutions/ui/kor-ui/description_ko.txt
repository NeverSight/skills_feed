React Native 및 Expo 애플리케이션에서 Kor UI 라이브러리(@korsolutions/ui)를 사용하기 위한 전체 가이드입니다. Universal UI로 사용자 인터페이스 구축, 테마 사용자 정의, 라이브러리 설정, 24개 이상의 구성 요소(버튼, 입력, 선택, 경고, 카드, 탭, 메뉴, 팝오버, 달력, 토스트 등) 작업, 스타일 지정 및 테마 지정, 복합 구성 요소 구현, 구성 요소 문제 디버깅 또는 사용자가 "@korsolutions/ui", "Universal UI", "UIProvider"를 언급하거나 스타일이 지정되지 않은 기본 요소, 테마 사용자 정의 또는 React Native에 대해 문의할 때 이 기술을 사용하세요. UI 구성 요소. 이 기술은 설치, 공급자 설정, 구성 요소 사용 패턴, 테마 사용자 정의, 변형 시스템, 후크, 반응형 디자인 및 문제 해결을 다룹니다.
