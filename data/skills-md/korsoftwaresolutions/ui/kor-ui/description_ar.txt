الدليل الكامل لاستخدام مكتبة Kor UI (@korsolutions/ui) في تطبيقات React Native وExpo. استخدم هذه المهارة عند إنشاء واجهات المستخدم باستخدام Universal UI، أو تخصيص السمات، أو إعداد المكتبة، أو العمل مع أي من المكونات التي يزيد عددها عن 24 (زر، إدخال، تحديد، تنبيه، بطاقة، علامات تبويب، قائمة، Popover، تقويم، Toast، إلخ.)، التصميم والتخصيص، تنفيذ المكونات المركبة، تصحيح مشكلات المكونات، أو عندما يذكر المستخدم "@korsolutions/ui" أو "واجهة المستخدم العالمية" أو "UIProvider" أو يسأل عن العناصر الأولية غير ذات السمات، التخصيص، أو مكونات واجهة المستخدم الأصلية React. تغطي هذه المهارة التثبيت وإعداد الموفر وأنماط استخدام المكونات وتخصيص السمة والنظام المتغير والخطافات والتصميم سريع الاستجابة واستكشاف الأخطاء وإصلاحها.
