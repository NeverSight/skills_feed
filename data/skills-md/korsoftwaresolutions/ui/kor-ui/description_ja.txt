React Native および Expo アプリケーションで Kor UI ライブラリ (@korsolutions/ui) を使用するための完全なガイド。このスキルは、ユニバーサル UI によるユーザー インターフェイスの構築、テーマのカスタマイズ、ライブラリのセットアップ、24 以上のコンポーネント (ボタン、入力、選択、アラート、カード、タブ、メニュー、ポップオーバー、カレンダー、トーストなど) の操作、スタイルとテーマ設定、複合コンポーネントの実装、コンポーネントの問題のデバッグを行うとき、またはユーザーが「@korsolutions/ui」、「ユニバーサル UI」、「UIProvider」について言及したとき、またはスタイルなしについて質問したときに使用します。プリミティブ、テーマのカスタマイズ、または React Native UI コンポーネント。このスキルには、インストール、プロバイダーのセットアップ、コンポーネントの使用パターン、テーマのカスタマイズ、バリアント システム、フック、レスポンシブ デザイン、トラブルシューティングが含まれます。
