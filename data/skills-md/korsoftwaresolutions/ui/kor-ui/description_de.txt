Vollständige Anleitung zur Verwendung der Kor-UI-Bibliothek (@korsolutions/ui) in React Native- und Expo-Anwendungen. Verwenden Sie diese Fertigkeit beim Erstellen von Benutzeroberflächen mit Universal UI, beim Anpassen von Designs, beim Einrichten der Bibliothek, beim Arbeiten mit einer der über 24 Komponenten (Schaltfläche, Eingabe, Auswahl, Warnung, Karte, Registerkarten, Menü, Popover, Kalender, Toast usw.), bei der Gestaltung und Gestaltung von Themen, beim Implementieren zusammengesetzter Komponenten, beim Debuggen von Komponentenproblemen oder wenn der Benutzer „@korsolutions/ui“, „Universal UI“, „UIProvider“ erwähnt oder nach unstilisierten Grundelementen, Designanpassungen oder React Native UI-Komponenten fragt. Diese Fertigkeit umfasst Installation, Anbietereinrichtung, Komponentennutzungsmuster, Theme-Anpassung, Variantensystem, Hooks, responsives Design und Fehlerbehebung.
