أنماط تطوير الواجهة الخلفية لـ C#/.NET الرئيسية لبناء واجهات برمجة التطبيقات القوية وخوادم MCP وتطبيقات المؤسسات. يغطي المزامنة/الانتظار، وحقن التبعية، وEntity Framework Core، وDapper، والتكوين، والتخزين المؤقت، والاختبار باستخدام xUnit. يُستخدم عند تطوير الواجهات الخلفية لـ .NET، أو مراجعة كود C#، أو تصميم بنيات واجهة برمجة التطبيقات.
