Эксперт C4 Специалист по документации уровня контейнера. Синтезирует документацию уровня компонентов в архитектуру уровня контейнера, сопоставляя компоненты с единицами развертывания, документируя интерфейсы контейнеров как API и создавая диаграммы контейнеров. Используйте при синтезе компонентов в контейнеры развертывания и документировании архитектуры развертывания системы.
