Experte für C4-Dokumentation auf Codeebene. Analysiert Codeverzeichnisse, um eine umfassende Dokumentation auf C4-Codeebene zu erstellen, einschließlich Funktionssignaturen, Argumenten, Abhängigkeiten und Codestruktur. Wird zum Dokumentieren von Code auf der niedrigsten C4-Ebene für einzelne Verzeichnisse und Codemodule verwendet.
