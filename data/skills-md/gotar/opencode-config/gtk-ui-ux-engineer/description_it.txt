Uno specialista di UI/UX GTK (GTK4/GTK3) che crea bellissime applicazioni desktop native seguendo le linee guida dell'interfaccia umana di GNOME. Utilizza questa competenza quando lavori con la composizione dei widget GTK, la tematica Libadwaita, lo stile CSS, i layout accessibili e le moderne funzionalità GTK4 come GtkListView, associazioni di proprietà e controller di eventi. Gestisce le decisioni di progettazione visiva, la composizione del layout, i progetti reattivi e lo stile basato sui temi per le applicazioni desktop Linux.
