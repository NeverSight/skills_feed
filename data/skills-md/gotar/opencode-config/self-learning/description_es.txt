Capacidad de autoaprendizaje para agentes. Captura seguimientos de ejecución, extrae información y adapta el comportamiento de los agentes. Úselo después de completar tareas complejas, cuando los agentes encuentren problemas recurrentes o para optimizar los flujos de trabajo a lo largo del tiempo. Crítico para sistemas de múltiples agentes como orquestador, trabajador y revisor.
