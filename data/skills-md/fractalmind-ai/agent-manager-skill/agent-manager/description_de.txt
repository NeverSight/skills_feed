Lebenszyklusmanagementsystem für Mitarbeiteragenten. Wird verwendet, wenn Sie mit Agenten/Verzeichnismitarbeiteragenten arbeiten – Starten, Stoppen, Überwachen oder Zuweisen von Aufgaben an Dev/QA-Agenten, die in tmux-Sitzungen ausgeführt werden. Völlig unabhängig von CAO, verwendet nur tmux + Python.
