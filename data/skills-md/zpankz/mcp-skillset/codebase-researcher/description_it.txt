Profonda capacità di ricerca per analizzare sistematicamente le basi di codice. Da utilizzare quando è necessaria una comprensione completa della base di codice, in particolare per nuovi progetti o quando si documenta l'architettura. Questa capacità dovrebbe essere utilizzata quando gli utenti richiedono analisi approfondite, documentazione dell'architettura o ricerca sistematica della base di codice a partire dalle directory di livello root.
