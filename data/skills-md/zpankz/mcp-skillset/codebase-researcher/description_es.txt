Habilidad de investigación profunda para analizar bases de código sistemáticamente. Úselo cuando se necesite una comprensión integral de la base del código, particularmente para proyectos nuevos o al documentar la arquitectura. Esta habilidad debe usarse cuando los usuarios solicitan un análisis profundo, documentación de arquitectura o una investigación sistemática de la base de código a partir de directorios de nivel raíz.
