CLI pour Limitless.ai Pendant avec gestion du journal de vie, graphe sémantique FalkorDBLite, intégrations vectorielles et pipelines DAG. À utiliser pour les requêtes de mémoire personnelle, la recherche sémantique dans les journaux de vie/chats/personnes/sujets, l'extraction d'entités et les opérations de graphe de connaissances. Les déclencheurs incluent "lifelog", "pendentif", "illimité", "mémoire personnelle", "recherche sémantique", "requête graphique", "extraction".
