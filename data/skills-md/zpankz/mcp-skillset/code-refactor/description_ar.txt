تنفيذ عمليات إعادة بناء التعليمات البرمجية المجمعة مثل إعادة تسمية المتغيرات/الوظائف عبر الملفات، واستبدال الأنماط، وتحديث استدعاءات واجهة برمجة التطبيقات. يُستخدم عندما يطلب المستخدمون إعادة تسمية المعرفات، أو استبدال أنماط التعليمات البرمجية المهملة، أو تحديث استدعاءات الطريقة، أو إجراء تغييرات متسقة عبر مواقع متعددة. يتم التنشيط على عبارات مثل "إعادة تسمية جميع المثيلات"، أو "استبدال X بـ Y في كل مكان"، أو "إعادة البناء للاستخدام"، أو "تحديث كافة الاستدعاءات إلى".
