Führen Sie Massencode-Refactoring-Vorgänge durch, z. B. das Umbenennen von Variablen/Funktionen in mehreren Dateien, das Ersetzen von Mustern und das Aktualisieren von API-Aufrufen. Verwenden Sie diese Option, wenn Benutzer das Umbenennen von Bezeichnern, das Ersetzen veralteter Codemuster, das Aktualisieren von Methodenaufrufen oder das Vornehmen konsistenter Änderungen über mehrere Standorte hinweg anfordern. Wird bei Phrasen wie „Alle Instanzen umbenennen“, „X überall durch Y ersetzen“, „zu verwendendes Refactoring“ oder „Alle Aufrufe aktualisieren“ aktiviert.
