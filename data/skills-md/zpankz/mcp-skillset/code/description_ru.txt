Управление исполняемой документацией с помощью сложного проектирования и абдуктивного обучения. Обеспечивает соблюдение семи законов посредством компиляции типов, проверки схемы и принудительного применения на основе перехвата. Реализует программную составную разработку, где K' = K ∪ Crystallize(assess(τ)) для монотонного роста знаний. Интегрирует абстрактное абдуктивное обучение (протокол OHPT) для систематической отладки и извлечения шаблонов. Триггер при написании кода, отладке, установлении управления или при упоминании вибекода, составной, абдуктивной или исполняемой документации. Самоутверждающийся и гомоиконичный.
