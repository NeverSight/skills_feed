Integriert abstrahiertes abduktives Lernen (OHPT-Protokoll) für systematisches Debugging und Musterextraktion. Wird beim Schreiben von Code, beim Debuggen, beim Einrichten der Governance oder bei Erwähnung von Vibecode, zusammengesetzter, abduktiver oder ausführbarer Dokumentation ausgelöst.
