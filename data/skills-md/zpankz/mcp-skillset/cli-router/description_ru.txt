Направляет задачи локально установленным инструментам CLI, используя семантическое сопоставление. Запускается при выполнении задач, требующих команд оболочки, операций с файлами, поиска кода, обработки данных, визуализации или вызова внешних инструментов. Использует cli-index для семантической маршрутизации.
