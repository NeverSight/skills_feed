Систематическая методология проектирования API с 6 проверенными шаблонами, охватывающими категоризацию параметров, безопасный рефакторинг, подход, ориентированный на аудит, автоматическую проверку, контроль качества и документацию на основе примеров. Используйте при разработке новых API, улучшении согласованности API, реализации политик критических изменений или обеспечении контроля качества API. Предоставляет детерминированные деревья решений (5-уровневая система параметров), архитектуру инструментов проверки, шаблоны перехватчиков перед фиксацией. Подтверждено междоменное переносимость на 82,5 %, повышение эффективности на 37,5 % за счет рефакторинга сперва аудита.
