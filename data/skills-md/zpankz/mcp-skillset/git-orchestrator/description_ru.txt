Мета-оркестратор для управления ~/.claude как частным репозиторием git с автоматическим
управление версиями, оркестровка ветвей и отслеживание проблем на основе DAG. Используйте при инициализации
репозиторий конфигурации, фиксация изменений конфигурации, управление ветвями функций, создание
рабочие деревья для параллельных экспериментов или изучения истории конфигурации.
