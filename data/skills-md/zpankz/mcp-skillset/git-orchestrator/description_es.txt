Metaorquestador para administrar ~/.claude como un repositorio git privado con función automática
control de versiones, orquestación de ramas y seguimiento de problemas basado en DAG. Usar al inicializar
el repositorio de configuración, realizar cambios de configuración, administrar ramas de funciones, crear
árboles de trabajo para experimentación paralela o investigación del historial de configuración.
