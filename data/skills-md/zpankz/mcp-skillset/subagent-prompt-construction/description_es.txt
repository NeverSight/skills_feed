Metodología sistemática para construir indicaciones de subagente compactas (<150 líneas), expresivas e integradas en Claude Code utilizando contratos lambda y lógica simbólica. Úselo al crear nuevos subagentes especializados para Claude Code con composición de agentes, integración de herramientas MCP o referencias de habilidades. Validado con fase-planificador-ejecutor (V_instance=0.895).
