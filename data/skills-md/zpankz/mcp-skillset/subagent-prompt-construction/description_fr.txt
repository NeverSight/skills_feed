Méthodologie systématique pour construire des invites de sous-agent compactes (<150 lignes), expressives et intégrées à Claude Code à l'aide de contrats lambda et d'une logique symbolique. À utiliser lors de la création de nouveaux sous-agents spécialisés pour Claude Code avec composition d'agents, intégration d'outils MCP ou références de compétences. Validé avec phase-planner-executor (V_instance=0.895).
