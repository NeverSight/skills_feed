Систематическая методология построения компактных (<150 строк) и выразительных подсказок субагента, интегрированных в Claude Code, с использованием лямбда-контрактов и символической логики. Используйте при создании новых специализированных субагентов для Claude Code с составом агентов, интеграцией инструментов MCP или ссылками на навыки. Проверено с помощью Phase-planner-executor (V_instance=0,895).
