Metodologia sistematica per la costruzione di prompt subagente compatti (<150 righe), espressivi e integrati nel codice Claude utilizzando contratti lambda e logica simbolica. Da utilizzare durante la creazione di nuovi agenti secondari specializzati per Claude Code con composizione dell'agente, integrazione dello strumento MCP o riferimenti alle competenze. Convalidato con fase-pianificatore-esecutore (V_instance=0.895).
