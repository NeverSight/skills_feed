多代理 Claude Code 會話的實時可觀察性儀表板。

通過 Web 儀表板實時可視化代理交互、工具使用情況和會話流。通過泳道可視化、事件過濾和實時圖表跟踪並行運行的多個代理。

**主要特點：**
- 🔴 通過 WebSocket 進行實時事件流
- 📊 代理泳道顯示並行執行
- 🔍 按代理、會話、事件類型進行事件過濾
- 📈 工具使用模式的實時圖表
- 💾 基於文件系統（不需要數據庫）

**受到 [@indydevdan](https://github.com/indydevdan)** 關於多代理可觀察性的工作的啟發。

**我們的方法：** 文件系統 + 內存流與 indydevdan 的 SQLite 數據庫方法。
