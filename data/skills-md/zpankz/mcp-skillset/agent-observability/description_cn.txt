多代理 Claude Code 会话的实时可观察性仪表板。

通过 Web 仪表板实时可视化代理交互、工具使用情况和会话流。通过泳道可视化、事件过滤和实时图表跟踪并行运行的多个代理。

**主要特点：**
- 🔴 通过 WebSocket 进行实时事件流
- 📊 代理泳道显示并行执行
- 🔍 按代理、会话、事件类型进行事件过滤
- 📈 工具使用模式的实时图表
- 💾 基于文件系统（不需要数据库）

**受到 [@indydevdan](https://github.com/indydevdan)** 关于多代理可观察性的工作的启发。

**我们的方法：** 文件系统 + 内存流与 indydevdan 的 SQLite 数据库方法。
