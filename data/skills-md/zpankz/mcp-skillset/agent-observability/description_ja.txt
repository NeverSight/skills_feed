マルチエージェントの Claude Code セッション用のリアルタイム可観測性ダッシュボード。

Web ダッシュボードを通じて、エージェントの対話、ツールの使用状況、セッション フローをリアルタイムで視覚化します。スイム レーンの視覚化、イベント フィルタリング、ライブ チャートを使用して、並行して実行されている複数のエージェントを追跡します。

**主な機能:**
- 🔴 WebSocket 経由のリアルタイム イベント ストリーミング
- 📊 並列実行を示すエージェントのスイムレーン
- 🔍 エージェント、セッション、イベントタイプによるイベントフィルタリング
- 📈 ツールの使用パターンのライブチャート
- 💾 ファイルシステムベース (データベースは不要)

**[@indydevdan](https://github.com/indydevdan)** のマルチエージェント可観測性に関する研究からインスピレーションを受けています。

**私たちのアプローチ:** ファイルシステム + インメモリ ストリーミングと indydevdan の SQLite データベース アプローチ。
