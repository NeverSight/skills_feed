다중 에이전트 Claude Code 세션에 대한 실시간 관찰 대시보드입니다.

웹 대시보드를 통해 에이전트 상호 작용, 도구 사용 및 세션 흐름을 실시간으로 시각화합니다. 수영 레인 시각화, 이벤트 필터링 및 라이브 차트를 통해 병렬로 실행되는 여러 에이전트를 추적합니다.

**주요 기능:**
- 🔴 WebSocket을 통한 실시간 이벤트 스트리밍
- 📊 병렬 실행을 보여주는 에이전트 수영 레인
- 🔍 에이전트, 세션, 이벤트 유형별 이벤트 필터링
- 📈 도구 사용 패턴에 대한 실시간 차트
- 💾 파일 시스템 기반(데이터베이스 필요 없음)

**[@indydevdan](https://github.com/indydevdan)**의 다중 에이전트 관찰 가능성 연구에서 영감을 얻었습니다.

**저희 접근 방식:** 파일 시스템 + 인메모리 스트리밍과 indydevdan의 SQLite 데이터베이스 접근 방식.
