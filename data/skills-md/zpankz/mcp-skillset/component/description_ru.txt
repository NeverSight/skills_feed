Создавайте синтаксически правильные компоненты конфигурации Claude Code (команды, субагенты, навыки, стили вывода) с оптимальной межкомпонентной интеграцией. Запускается по запросам на создание косых команд, агентов, субагентов, навыков, стилей вывода, плагинов Claude Code, агентских рабочих процессов или многоагентных конвейеров. Обеспечивает достоверность параметров для каждого типа компонента и семантическую согласованность ссылок на компоненты.
