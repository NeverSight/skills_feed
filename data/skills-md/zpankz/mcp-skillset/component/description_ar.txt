قم بإنشاء مكونات تكوين Claude Code الصحيحة من الناحية النحوية (الأوامر والوكلاء الفرعيين والمهارات وأنماط الإخراج) مع التكامل الأمثل بين المكونات. يتم تشغيلها عند طلبات إنشاء أوامر الشرطة المائلة، أو الوكلاء، أو الوكلاء الفرعيين، أو المهارات، أو أنماط الإخراج، أو مكونات Claude Code الإضافية، أو مسارات عمل الوكيل، أو خطوط الأنابيب متعددة الوكلاء. يضمن صحة المعلمة لكل نوع مكون والتماسك الدلالي عبر مراجع المكونات.
