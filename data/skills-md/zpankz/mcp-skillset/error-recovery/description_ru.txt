Комплексная методология обработки ошибок с таксономией по 13 категориям, рабочими процессами диагностики, шаблонами восстановления и рекомендациями по предотвращению. Используйте, когда уровень ошибок >5 %, MTTD/MTTR слишком высоки, ошибки повторяются, требуется систематическое предотвращение ошибок или создание инфраструктуры обработки ошибок. Обеспечивает таксономию ошибок (операции с файлами, вызовы API, проверка данных, управление ресурсами, параллелизм, конфигурация, зависимости, сеть, синтаксический анализ, управление состоянием, аутентификация, тайм-аут, крайние случаи - охват 95,4%), 8 рабочих процессов диагностики, 5 шаблонов восстановления, 8 рекомендаций по предотвращению, 3 инструмента автоматизации (проверка пути к файлу, проверка чтения перед записью, проверка размера файла - предотвращение ошибок 23,7%). Проверено с 1336 историческими ошибками, переносимость между языками и платформами 85–90 %, ретроспективная проверка с достоверностью 0,79.
