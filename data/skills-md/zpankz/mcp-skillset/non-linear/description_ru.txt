Нелинейная система рассуждений с учетом неопределенности и рекурсивной оркестровкой субагентов. Триггеры для сложных рассуждений, исследований, многодоменного синтеза или использования явных команд `/nlr`, `/reason`, `/think-deep`. Интегрирует мыслительные навыки (рассуждение), основные навыки агента (действие) и инструменты MCP (инфранодус, экза, ученый-шлюз) в рекурсивных циклах мышление → действие → наблюдение. Использует изолированную среду кодирования для проверки выполнения и поддерживает преднамеренную зашумленность с помощью каркаса NoisyGraph. Поддерживает режим `/compact` для сокращенных выходных данных и режим `/semantic` для расширенного исследования.
