Applique la syntaxe Markdown à saveur Obsidian lors de la création d'un fichier .md, en fournissant des modèles, une validation et une application de modèles pour le contenu YAML, les liens wiki, les légendes, les requêtes de vue de données, les tableaux, les diagrammes de sirène, la syntaxe du modèle, le canevas/JSONCanvas, les bases et les notes de bas de page. S'active automatiquement, sauf indication contraire explicite.
