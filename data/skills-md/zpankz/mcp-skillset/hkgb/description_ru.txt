Этот навык следует использовать при построении гибридных диаграмм знаний, которые интегрируют структурированные данные (CSV, базы данных) с автоматически извлекаемыми объектами из неструктурированных документов (PDF-файлы, текст). Шаблон устанавливает надежный ключ соединения между графами предметной области и лексическими графами, позволяя использовать GraphRAG, прием документов с обогащением метаданных и построение диаграммы знаний из разнородных источников с использованием neo4j-graphrag SimpleKGPipeline.
