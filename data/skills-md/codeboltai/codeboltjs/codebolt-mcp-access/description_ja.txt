Codebolt エージェントが Codebolt MCP (Model Context Protocol) API と対話するためのコア スキル。エージェントがファイル操作、Git、ブラウザ自動化、ターミナル、メモリ、オーケストレーション、計画、テスト、コラボレーション、または Codebolt プラットフォーム機能用のツールを実行する必要がある場合に使用します。 codebolt.tools.executeTool()、ファイル システム操作、git コマンド、ブラウザ制御、エージェント オーケストレーション、タスク管理、または任意の MCP 名前空間操作を含むリクエストでトリガーされます。
