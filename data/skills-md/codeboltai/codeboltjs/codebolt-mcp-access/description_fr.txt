Compétence de base permettant aux agents Codebolt d'interagir avec l'API Codebolt MCP (Model Context Protocol). À utiliser lorsque les agents doivent exécuter des outils pour les opérations sur les fichiers, Git, l'automatisation du navigateur, le terminal, la mémoire, l'orchestration, la planification, les tests, la collaboration ou toute autre fonctionnalité de la plateforme Codebolt. Se déclenche sur des requêtes impliquant codebolt.tools.executeTool(), des opérations du système de fichiers, des commandes git, le contrôle du navigateur, l'orchestration d'agent, la gestion des tâches ou toute opération d'espace de noms MCP.
