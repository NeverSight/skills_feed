Créez des agents IA pour la plate-forme Codebolt à l'aide de @codebolt/agent. Lors de la création d'un agent, décidez d'abord du type dont vous avez besoin : (1) wrapper CodeboltAgent pour des agents simples basés sur des invites sans boucle personnalisée, (2) composants de base pour personnaliser la boucle agent ou (3) API principale pour un contrôle avancé complet. Couvre également les agents Remix (sans code), les ActionBlocks, les outils, les flux de travail et les processeurs. Les autorisations sont gérées par l'application et non par l'agent.
