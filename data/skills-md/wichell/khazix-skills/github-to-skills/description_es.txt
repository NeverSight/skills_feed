Fábrica automatizada para convertir repositorios de GitHub en habilidades especializadas de IA. Utilice esta habilidad cuando el usuario proporcione una URL de GitHub y quiera "empaquetar", "envolver" o "crear una habilidad" a partir de ella. Obtiene automáticamente los detalles del repositorio, los hashes de confirmación más recientes y genera una estructura de habilidades estandarizada con metadatos mejorados adecuados para la gestión del ciclo de vida.
