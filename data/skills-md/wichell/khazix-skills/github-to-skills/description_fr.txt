Usine automatisée pour convertir les référentiels GitHub en compétences spécialisées en IA. Utilisez cette compétence lorsque l'utilisateur fournit une URL GitHub et souhaite « empaqueter », « envelopper » ou « créer une compétence » à partir de celle-ci. Il récupère automatiquement les détails du référentiel, les derniers hachages de validation et génère une structure de compétences standardisée avec des métadonnées améliorées adaptées à la gestion du cycle de vie.
