Strukturierter Workflow für das Upgrade von Next.js-Anwendungen über Hauptversionen hinweg. Verwenden Sie diese Option, wenn Sie ein Next.js-Projekt von einer Hauptversion auf eine andere migrieren (z. B. 13 auf 14, 14 auf 15, 15 auf 16). Behandelt die Codemod-Automatisierung, die Erkennung von Breaking Changes, inkrementelle Migrationspfade und die Validierung nach dem Upgrade.
