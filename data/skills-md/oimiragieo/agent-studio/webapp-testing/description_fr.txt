Testez les applications Web locales à l'aide de Playwright avec Python. Vérifiez la fonctionnalité frontale, déboguez le comportement de l'interface utilisateur, capturez des captures d'écran et affichez les journaux de la console du navigateur. Prend en charge les fichiers HTML statiques, les applications Web dynamiques avec des serveurs en cours d'exécution et la génération de tests automatisés.
