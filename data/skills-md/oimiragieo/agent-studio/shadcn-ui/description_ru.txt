Глубокий опыт работы с библиотекой компонентов shadcn/ui, включая установку, настройку, шаблоны тем и специальных возможностей. Используйте при создании пользовательского интерфейса с помощью компонентов shadcn/ui, интеграции примитивов пользовательского интерфейса Radix, настройке тем Tailwind CSS v4 или реализации темного режима. Охватывает настройки Next.js App Router, Pages Router и Vite.
