Комплексный набор инструментов для научных исследований, включающий 139 специализированных навыков в области биологии, химии, медицины, обработки данных и вычислительных исследований. Превращает Клода в помощника по исследованию искусственного интеллекта с доступом к научным базам данных, инструментам анализа и рабочим процессам, специфичным для конкретной предметной области.
