Análisis de vulnerabilidad de código impulsado por IA y escaneo de dependencias utilizando patrones de extensión de seguridad Gemini CLI. Detecta secretos codificados, ataques de inyección, criptografía débil, fallas de autenticación e inyección rápida de LLM. También analiza las dependencias con la base de datos de vulnerabilidades OSV.dev.
