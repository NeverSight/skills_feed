Analisi della vulnerabilità del codice basata sull'intelligenza artificiale e scansione delle dipendenze utilizzando i modelli di estensione di sicurezza della CLI Gemini. Rileva segreti hardcoded, attacchi injection, crittografia debole, difetti di autenticazione e prompt injection LLM. Esegue inoltre la scansione delle dipendenze rispetto al database delle vulnerabilità OSV.dev.
