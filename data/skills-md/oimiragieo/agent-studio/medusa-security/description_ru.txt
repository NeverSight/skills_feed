Сканирование безопасности на основе искусственного интеллекта с помощью Medusa. Более 3000 шаблонов обнаружения, охватывающих уязвимости AI/ML, агентов, MCP, RAG, оперативного внедрения и традиционных уязвимостей SAST. Объединяет Medusa CLI с анализом SARIF/JSON, структурированным выводом результатов, сопоставлением OWASP и рекомендациями по исправлению.
