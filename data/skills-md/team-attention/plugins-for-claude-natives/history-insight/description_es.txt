Esta habilidad debe usarse cuando el usuario desea acceder, capturar o hacer referencia al historial de sesiones de Claude Code. Se activa cuando el usuario dice "capturar sesión", "guardar historial de sesión" o hace referencia a una conversación pasada/actual como fuente, ya sea para guardar, extraer, resumir o revisar. Esto incluye cualquier mención de "lo que discutimos", "el trabajo de hoy", "historial de sesiones" o cuando el usuario trata la conversación en sí como material fuente (por ejemplo, "de nuestra conversación").
