يجب استخدام هذه المهارة عندما يريد المستخدم الوصول إلى سجل جلسة Claude Code أو التقاطه أو الرجوع إليه. يتم تشغيله عندما يقول المستخدم "التقاط الجلسة"، أو "حفظ سجل الجلسة"، أو الإشارة إلى المحادثة السابقة/الحالية كمصدر - سواء للحفظ أو الاستخراج أو التلخيص أو المراجعة. يتضمن ذلك أي ذكر لـ "ما ناقشناه"، أو "عمل اليوم"، أو "سجل الجلسة"، أو عندما يتعامل المستخدم مع المحادثة نفسها كمواد مصدر (على سبيل المثال، "من محادثتنا").
