Diese Fähigkeit sollte verwendet werden, wenn der Benutzer auf den Sitzungsverlauf von Claude Code zugreifen, ihn erfassen oder darauf verweisen möchte. Wird ausgelöst, wenn der Benutzer „Sitzung erfassen“, „Sitzungsverlauf speichern“ sagt oder auf vergangene/aktuelle Konversationen als Quelle verweist – sei es zum Speichern, Extrahieren, Zusammenfassen oder Überprüfen. Dazu gehört jede Erwähnung von „was wir besprochen haben“, „heutige Arbeit“, „Sitzungsverlauf“ oder wenn der Benutzer die Konversation selbst als Quellmaterial behandelt (z. B. „aus unserer Konversation“).
