inference.sh 用の Python SDK - AI アプリを実行し、エージェントを構築し、150 以上のモデルと統合します。
パッケージ: inferencesh (pip install inferencesh)。同期/非同期、ストリーミング、ファイルのアップロードをサポートします。
テンプレートまたはアドホック パターン、ツール ビルダー API、スキル、人間の承認を使用してエージェントを構築します。
用途: Python 統合、AI アプリ、エージェント開発、RAG パイプライン、自動化。
