Компоненты пользовательского интерфейса жизненного цикла инструмента для React/Next.js из ui.inference.sh.
Отображать вызовы инструментов: ожидающие, ход выполнения, требуется одобрение, результаты.
Возможности: статус инструмента, индикаторы прогресса, потоки утверждения, отображение результатов.
Используется для: показа вызовов инструментов агента, одобрения человеком в процессе работы, выходных данных инструмента.
Триггеры: пользовательский интерфейс инструмента, вызовы инструментов, состояние инструмента, утверждение инструмента, результаты инструмента,
инструменты агента, пользовательский интерфейс инструментов mcp, пользовательский интерфейс вызова функций, жизненный цикл инструмента, ожидание инструмента
