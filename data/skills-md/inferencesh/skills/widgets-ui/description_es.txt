Widgets de interfaz de usuario declarativos de JSON para React/Next.js de ui.inference.sh.
Represente interfaces de usuario interactivas enriquecidas a partir de respuestas estructuradas de los agentes.
Capacidades: formularios, botones, tarjetas, diseños, entradas, selecciones, casillas de verificación.
Úselo para: UI generadas por agentes, formularios dinámicos, visualización de datos y tarjetas interactivas.
Activadores: widgets, interfaz de usuario declarativa, interfaz de usuario json, renderizador de widgets, widgets de agente,
interfaz de usuario dinámica, widgets de formulario, widgets de tarjeta, widgets shadcn, interfaz de usuario de salida estructurada
