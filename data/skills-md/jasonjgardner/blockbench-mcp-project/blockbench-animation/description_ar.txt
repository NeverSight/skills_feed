قم بإنشاء الرسوم المتحركة وإدارتها في Blockbench باستخدام أدوات MCP. يُستخدم عند تحريك النماذج ثلاثية الأبعاد، أو إنشاء الإطارات الرئيسية، أو إدارة منصات العظام، أو تحرير منحنيات الرسوم المتحركة، أو العمل مع المخططات الزمنية للرسوم المتحركة. يغطي دورات المشي، والرسوم المتحركة الخاملة، والرسوم المتحركة القتالية، والرسوم المتحركة المعقدة متعددة العظام.
