Sviluppo di plugin/estensioni Blockbench per lo strumento di modellazione 3D. Da utilizzare durante la creazione, la modifica o il debug di plug-in JavaScript per Blockbench, tra cui azioni, finestre di dialogo, pannelli, menu, barre degli strumenti, manipolazione di modelli, API di animazione e formati/codec personalizzati. Trigger sul plug-in Blockbench, sull'estensione Blockbench, sull'API Blockbench, sul BBPlugin, sul plug-in dell'editor di modelli o sull'estensione dello strumento di modellazione 3D.
