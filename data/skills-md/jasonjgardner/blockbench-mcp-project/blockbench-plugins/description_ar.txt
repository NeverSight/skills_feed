تطوير البرنامج المساعد/امتداد Blockbench لأداة النمذجة ثلاثية الأبعاد. يُستخدم عند إنشاء أو تعديل أو تصحيح أخطاء مكونات JavaScript الإضافية لـ Blockbench، بما في ذلك الإجراءات ومربعات الحوار واللوحات والقوائم وأشرطة الأدوات ومعالجة النماذج وواجهات برمجة تطبيقات الرسوم المتحركة والتنسيقات/برامج الترميز المخصصة. المشغلات على البرنامج الإضافي Blockbench، أو ملحق Blockbench، أو Blockbench API، أو BBPlugin، أو البرنامج الإضافي لمحرر النماذج، أو ملحق أداة النمذجة ثلاثية الأبعاد.
