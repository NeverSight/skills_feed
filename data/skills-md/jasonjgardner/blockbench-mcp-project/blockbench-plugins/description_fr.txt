Développement du plugin/extension Blockbench pour l'outil de modélisation 3D. À utiliser lors de la création, de la modification ou du débogage de plugins JavaScript pour Blockbench, y compris les actions, les boîtes de dialogue, les panneaux, les menus, les barres d'outils, la manipulation de modèles, les API d'animation et les formats/codecs personnalisés. Déclencheurs sur le plugin Blockbench, l'extension Blockbench, l'API Blockbench, BBPlugin, le plugin d'éditeur de modèle ou l'extension de l'outil de modélisation 3D.
