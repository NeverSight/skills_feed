Desarrollo de complemento/extensión Blockbench para la herramienta de modelado 3D. Úselo al crear, modificar o depurar complementos de JavaScript para Blockbench, incluidas acciones, cuadros de diálogo, paneles, menús, barras de herramientas, manipulación de modelos, API de animación y formatos/códecs personalizados. Activadores en el complemento Blockbench, la extensión Blockbench, la API Blockbench, BBPlugin, el complemento del editor de modelos o la extensión de la herramienta de modelado 3D.
