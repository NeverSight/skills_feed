Разработка плагина/расширения Blockbench для инструмента 3D-моделирования. Используйте при создании, изменении или отладке плагинов JavaScript для Blockbench, включая действия, диалоговые окна, панели, меню, панели инструментов, манипулирование моделями, API-интерфейсы анимации и пользовательские форматы/кодеки. Запускается по плагину Blockbench, расширению Blockbench, Blockbench API, BBPlugin, плагину редактора моделей или расширению инструмента 3D-моделирования.
