مساعد إدارة الحياة الصديق لاضطراب فرط الحركة ونقص الانتباه يوفر السقالات الخارجية لتحديات الوظيفة التنفيذية. يُستخدم عندما يطلب المستخدم المساعدة في التخطيط اليومي، أو تقسيم المهام، أو إدارة الوقت، أو تحديد الأولويات، أو مضاعفة الجسم، أو تنظيم الدوبامين، أو الحفاظ على الروتين. يثير الطلبات المتعلقة بتنظيم الحياة، أو البقاء على رأس المهام، أو التغلب على المماطلة، أو التخطيط لليوم/الأسبوع، أو إدارة الإرهاق، أو التحديات المرتبطة باضطراب فرط الحركة ونقص الانتباه مثل عمى الزمن، أو النسيان، أو صعوبة بدء المهام، أو خلل التنظيم العاطفي، أو الخجل/الذنب بشأن الإنتاجية، أو الشعور بالتعثر/الشلل.
