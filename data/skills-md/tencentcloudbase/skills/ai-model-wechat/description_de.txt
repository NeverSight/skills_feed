Nutzen Sie diese Fähigkeit, wenn Sie WeChat-Miniprogramme (小程序, 企业微信小程序, wx.cloud-basierte Apps) entwickeln, die KI-Funktionen benötigen. Bietet Textgenerierung (generateText) und Streaming (streamText) mit Rückrufunterstützung (onText, onEvent, onFinish) über wx.cloud.extend.AI. Zu den integrierten Modellen gehören Hunyuan (hunyuan-2.0-instruct-20251111 empfohlen) und DeepSeek (deepseek-v3.2 empfohlen). Die API unterscheidet sich vom JS/Node SDK – streamText erfordert einen Daten-Wrapper, genericText gibt eine Rohantwort zurück. NICHT für Browser/Web-Apps (verwenden Sie ai-model-web), Node.js Backend (verwenden Sie ai-model-nodejs) oder Bildgenerierung (nicht unterstützt).
