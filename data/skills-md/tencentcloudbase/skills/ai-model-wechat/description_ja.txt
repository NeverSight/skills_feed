AI 機能を必要とする WeChat ミニ プログラム (小程序、計画业微信小程序、wx.cloud ベースのアプリ) を開発する場合は、このスキルを使用します。 wx.cloud.extend.AI によるコールバック サポート (onText、onEvent、onFinish) を備えたテキスト生成 (generateText) とストリーミング (streamText) が特徴です。組み込みモデルには、Hunyuan (hunyuan-2.0-instruct-20251111 を推奨) と DeepSeek (deepseek-v3.2 を推奨) が含まれます。 API は JS/Node SDK とは異なります。streamText はデータ ラッパーを必要とし、generateText は生の応答を返します。ブラウザ/Web アプリ (ai-model-web を使用)、Node.js バックエンド (ai-model-nodejs を使用)、または画像生成 (サポートされていません) には使用できません。
