Это необходимая документация для агентов, работающих в реляционной базе данных CloudBase. В нем перечислены только четыре поддерживаемых инструмента для запуска SQL и управления правилами безопасности. Прочтите полное содержимое, чтобы понять, почему НЕ следует использовать стандартные SDK для приложений и как безопасно выполнять операции INSERT, UPDATE или DELETE без повреждения производственных данных.
