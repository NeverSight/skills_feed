Questa è la documentazione richiesta per gli agenti che operano sul database relazionale CloudBase. Elenca gli unici quattro strumenti supportati per l'esecuzione di SQL e la gestione delle regole di sicurezza. Leggi il contenuto completo per comprendere perché NON è necessario utilizzare gli SDK dell'applicazione standard e come eseguire in modo sicuro le operazioni INSERT, UPDATE o DELETE senza danneggiare i dati di produzione.
