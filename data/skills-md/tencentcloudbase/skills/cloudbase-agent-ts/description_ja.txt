AG-UI プロトコルを実装する TypeScript SDK である Cloudbase Agent (TypeScript) を使用して AI エージェントを構築およびデプロイします。次の場合に使用します: (1) @cloudbase/agent-server を使用してエージェント サーバーを展開する、(2) ClientStateAnnotation を使用して LangGraph アダプターを使用する、(3) clientTools() を使用して LangChain アダプターを使用する、(4) AbstractAgent を実装するカスタム アダプターを構築する、(5) AG-UI プロトコル イベントを理解する、(6) @ag-ui/client を使用して Web UI クライアントを構築する、(7) @ag-ui/client を使用して WeChat ミニ プログラム UI を構築する@cloudbase/エージェント-ui-ミニプログラム。
