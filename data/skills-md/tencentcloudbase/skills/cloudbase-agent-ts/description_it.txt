Crea e distribuisci agenti AI con Cloudbase Agent (TypeScript), un SDK TypeScript che implementa il protocollo AG-UI. Utilizzare quando: (1) distribuire server agente con @cloudbase/agent-server, (2) utilizzare l'adattatore LangGraph con ClientStateAnnotation, (3) utilizzare l'adattatore LangChain con clientTools(), (4) creare adattatori personalizzati che implementano AbstractAgent, (5) comprendere gli eventi del protocollo AG-UI, (6) creare client dell'interfaccia utente Web con @ag-ui/client, (7) creare interfacce utente del Mini Program WeChat con @cloudbase/agent-ui-miniprogram.
