Создавайте и развертывайте агенты искусственного интеллекта с помощью Cloudbase Agent (TypeScript), пакета SDK TypeScript, реализующего протокол AG-UI. Используйте при: (1) развертывании серверов агентов с помощью @cloudbase/agent-server, (2) использовании адаптера LangGraph с ClientStateAnnotation, (3) использовании адаптера LangChain с clientTools(), (4) создании пользовательских адаптеров, реализующих AbstractAgent, (5) понимании событий протокола AG-UI, (6) создании клиентов веб-интерфейса с помощью @ag-ui/client, (7) создании пользовательских интерфейсов мини-программ WeChat с помощью @cloudbase/agent-ui-miniprogram.
