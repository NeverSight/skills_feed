يمكنك إنشاء وكلاء الذكاء الاصطناعي ونشرهم باستخدام Cloudbase Agent (TypeScript)، وهو TypeScript SDK الذي ينفذ بروتوكول AG-UI. يُستخدم عندما: (1) نشر خوادم الوكيل مع @cloudbase/agent-server، (2) استخدام محول LangGraph مع ClientStateAnnotation، (3) استخدام محول LangChain مع clientTools()، (4) إنشاء محولات مخصصة تنفذ AbstractAgent، (5) فهم أحداث بروتوكول AG-UI، (6) إنشاء عملاء واجهة مستخدم الويب باستخدام @ag-ui/client، (7) إنشاء واجهات مستخدم برنامج WeChat Mini مع @cloudbase/agent-ui-miniprogram.
