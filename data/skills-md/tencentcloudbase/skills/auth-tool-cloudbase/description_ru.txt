Используйте инструмент CloudBase Auth для настройки и управления поставщиками аутентификации для веб-приложений: включите/отключите методы входа в систему (SMS, электронная почта, открытая платформа WeChat, Google, Anonymous, имя пользователя/пароль, OAuth, SAML, CAS, Dingding и т. д.) и настройте параметры поставщика с помощью инструментов MCP `callCloudApi`.
