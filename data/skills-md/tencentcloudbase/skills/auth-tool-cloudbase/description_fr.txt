Utilisez l'outil CloudBase Auth pour configurer et gérer les fournisseurs d'authentification pour les applications Web - activer/désactiver les méthodes de connexion (SMS, e-mail, WeChat Open Platform, Google, Anonyme, nom d'utilisateur/mot de passe, OAuth, SAML, CAS, Dingding, etc.) et configurer les paramètres du fournisseur via les outils MCP `callCloudApi`.
