Utilice la herramienta CloudBase Auth para configurar y administrar proveedores de autenticación para aplicaciones web: habilite/deshabilite los métodos de inicio de sesión (SMS, correo electrónico, plataforma abierta WeChat, Google, anónimo, nombre de usuario/contraseña, OAuth, SAML, CAS, Dingding, etc.) y configure los ajustes del proveedor a través de las herramientas MCP `callCloudApi`.
