Utilizza questa competenza quando sviluppi servizi backend Node.js o funzioni cloud CloudBase (Express/Koa/NestJS, serverless, API backend) che necessitano di funzionalità IA. Dispone di generazione di testo (generateText), streaming (streamText) E generazione di immagini (generateImage) tramite @cloudbase/node-sdk ≥3.16.0. I modelli integrati includono Hunyuan (consigliato hunyuan-2.0-instruct-20251111), DeepSeek (consigliato deepseek-v3.2) e hunyuan-image per le immagini. Questo è l'UNICO SDK che supporta la generazione di immagini. NON per browser/app Web (usa ai-model-web) o WeChat Mini Program (usa ai-model-wechat).
