Utilice esta habilidad cuando desarrolle servicios backend de Node.js o funciones en la nube CloudBase (Express/Koa/NestJS, sin servidor, API backend) que necesiten capacidades de IA. Incluye generación de texto (generateText), transmisión (streamText) Y generación de imágenes (generateImage) a través de @cloudbase/node-sdk ≥3.16.0. Los modelos integrados incluyen Hunyuan (se recomienda hunyuan-2.0-instruct-20251111), DeepSeek (se recomienda deepseek-v3.2) y hunyuan-image para imágenes. Este es el ÚNICO SDK que admite la generación de imágenes. NO para navegador/aplicaciones web (use ai-model-web) o WeChat Mini Program (use ai-model-wechat).
