Utilice esta habilidad cuando desarrolle navegadores/aplicaciones web (React/Vue/Angular, sitios web estáticos, SPA) que necesiten capacidades de IA. Incluye generación de texto (generateText) y transmisión (streamText) a través de @cloudbase/js-sdk. Los modelos integrados incluyen Hunyuan (se recomienda hunyuan-2.0-instruct-20251111) y DeepSeek (se recomienda deepseek-v3.2). NO para el backend de Node.js (use ai-model-nodejs), el Mini Programa WeChat (use ai-model-wechat) o la generación de imágenes (solo Node SDK).
