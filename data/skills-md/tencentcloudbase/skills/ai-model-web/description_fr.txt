Utilisez cette compétence lors du développement d'applications de navigateur/Web (React/Vue/Angular, sites Web statiques, SPA) nécessitant des capacités d'IA. Comprend la génération de texte (generateText) et le streaming (streamText) via @cloudbase/js-sdk. Les modèles intégrés incluent Hunyuan (hunyuan-2.0-instruct-20251111 recommandé) et DeepSeek (deepseek-v3.2 recommandé). PAS pour le backend Node.js (utilisez ai-model-nodejs), le mini-programme WeChat (utilisez ai-model-wechat) ou la génération d'images (SDK Node uniquement).
