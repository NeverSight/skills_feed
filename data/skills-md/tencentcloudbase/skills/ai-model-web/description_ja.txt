AI 機能を必要とするブラウザ/Web アプリケーション (React/Vue/Angular、静的 Web サイト、SPA) を開発する場合は、このスキルを使用します。 @cloudbase/js-sdk を介したテキスト生成 (generateText) とストリーミング (streamText) が特徴です。組み込みモデルには、Hunyuan (hunyuan-2.0-instruct-20251111 を推奨) と DeepSeek (deepseek-v3.2 を推奨) が含まれます。 Node.js バックエンド (ai-model-nodejs を使用)、WeChat ミニ プログラム (ai-model-wechat を使用)、またはイメージ生成 (Node SDK のみ) には使用できません。
