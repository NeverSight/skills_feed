استخدم هذه المهارة عند تطوير تطبيقات المتصفح/الويب (React/Vue/Angular، مواقع الويب الثابتة، SPA) التي تحتاج إلى قدرات الذكاء الاصطناعي. يتميز بإنشاء النص (generateText) والبث (streamText) عبر @cloudbase/js-sdk. تتضمن النماذج المضمنة Hunyuan (يوصى بـ hunyuan-2.0-instruct-20251111) وDeepSeek (يوصى بـdeepseek-v3.2). ليس للواجهة الخلفية لـ Node.js (استخدم ai-model-nodejs)، أو برنامج WeChat Mini (استخدم ai-model-wechat)، أو إنشاء الصور (Node SDK فقط).
