Используйте этот навык при разработке браузерных/веб-приложений (React/Vue/Angular, статических веб-сайтов, SPA), которым необходимы возможности искусственного интеллекта. Функции генерации текста (generateText) и потоковой передачи (streamText) через @cloudbase/js-sdk. Встроенные модели включают Hunyuan (рекомендуется hunyuan-2.0-instruct-20251111) и DeepSeek (рекомендуется deepseek-v3.2). НЕ для серверной части Node.js (используйте ai-model-nodejs), мини-программы WeChat (используйте ai-model-wechat) или генерации изображений (только Node SDK).
