Полная интеграция DataForSEO API для данных и анализа SEO. Используйте, когда пользователь запрашивает исследование ключевых слов, объем поиска, анализ поисковой выдачи, аудит обратных ссылок, анализ конкурентов, отслеживание рейтинга, авторитетность домена, технический SEO-аудит, мониторинг контента, Google Trends или любые запросы данных, связанные с SEO. Охватывает все API-интерфейсы DataForSEO, включая поисковую выдачу, данные ключевых слов, DataForSEO Labs, обратные ссылки, OnPage, аналитику домена, анализ контента, бизнес-данные, данные о продавцах, данные приложений и API-интерфейсы оптимизации AI. Выводит файлы CSV.
