Обеспечивает правильное выравнивание ведущих значков в списках по первой строке текста при переносе текста, предотвращая вертикальное центрирование значков по нескольким строкам. Используйте при реализации элементов списка с ведущими значками, значками SVG в гибких контейнерах или при устранении проблем с выравниванием значков в обернутом тексте.
