أنشئ تطبيقات الذكاء الاصطناعي باستخدام OpenAI Agents SDK - وكلاء النص، ووكلاء الصوت (في الوقت الفعلي)، وسير العمل متعدد الوكلاء مع عمليات التسليم، والأدوات مع مخططات Zod، وحواجز حماية الإدخال/الإخراج، والمخرجات المنظمة، والبث. قم بالنشر إلى Cloudflare Workers، أو Next.js، أو التفاعل مع أنماط الإنسان في الحلقة.

يُستخدم عندما: إنشاء وكلاء مستندين إلى النص باستخدام الأدوات ومخططات Zod، أو إنشاء وكلاء صوتيين في الوقت الفعلي باستخدام WebRTC/WebSocket، أو تنفيذ سير عمل متعدد الوكلاء مع عمليات التسليم بين المتخصصين، أو إعداد حواجز حماية الإدخال/الإخراج للسلامة، أو طلب موافقة بشرية على الإجراءات الحاسمة، أو استجابات وكيل التدفق، أو نشر الوكلاء إلى Cloudflare Workers أو Next.js، أو استكشاف أخطاء نوع مخطط Zod وإصلاحها، أو فشل تتبع MCP، أو الحلقات اللانهائية (MaxTurnsExceededError)، أو فشل استدعاء الأداة، أو عدم تطابق المخطط، أو قيود تسليم الوكيل الصوتي.
