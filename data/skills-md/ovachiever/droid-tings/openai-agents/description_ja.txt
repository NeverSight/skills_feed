OpenAI Agents SDK を使用して AI アプリケーションを構築します - テキスト エージェント、音声エージェント (リアルタイム)、ハンドオフを備えたマルチエージェント ワークフロー、Zod スキーマを備えたツール、入出力ガードレール、構造化出力、ストリーミング。 Cloudflare Workers、Next.js、または人間参加型パターンを使用した React にデプロイします。

次の場合に使用します: ツールと Zod スキーマを使用したテキストベースのエージェントの構築、WebRTC/WebSocket を使用したリアルタイム音声エージェントの作成、スペシャリスト間のハンドオフを伴うマルチエージェント ワークフローの実装、安全のための入力/出力ガードレールの設定、重要なアクションに対する人間の承認の要求、エージェントの応答のストリーミング、Cloudflare Workers または Next.js へのエージェントのデプロイ、または Zod スキーマ タイプ エラー、MCP トレースの失敗、無限ループのトラブルシューティング(MaxTurnsExceededError)、ツール呼び出しの失敗、スキーマの不一致、または音声エージェントのハンドオフ制約。
