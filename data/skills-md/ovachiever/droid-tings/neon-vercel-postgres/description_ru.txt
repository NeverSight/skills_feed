Настройте бессерверную Postgres с помощью Neon или Vercel Postgres для Cloudflare Workers/Edge. Включает пул соединений, ветвление в стиле git для сред предварительной версии и интеграцию Drizzle/Prisma.

Используйте при настройке Edge Postgres, настройке ветвления базы данных или устранении неполадок «TCP не поддерживается», исчерпании пула соединений, настройке SSL (sslmode=require) или совместимости с Prisma Edge.
