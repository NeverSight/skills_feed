قم بتخزين الكائنات باستخدام وحدة تخزين R2 المتوافقة مع S3 على حافة Cloudflare. يُستخدم عندما: تحميل/تنزيل الملفات، أو تكوين CORS، أو إنشاء عناوين URL الموقعة مسبقًا، أو التحميلات متعددة الأجزاء، أو إدارة بيانات التعريف، أو استكشاف أخطاء R2_ERROR وإصلاحها، أو فشل CORS، أو مشكلات URL الموقعة مسبقًا، أو أخطاء الحصص.
