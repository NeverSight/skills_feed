Создавайте полнофункциональные приложения React с помощью TanStack Start на Cloudflare Workers. Типобезопасная маршрутизация, функции сервера, SSR/потоковая передача и бесшовная интеграция D1/KV/R2.

Используйте, когда: создаете полнофункциональные приложения React, нуждаетесь в SSR с Cloudflare Worker, хотите использовать типобезопасные серверные функции или переходите с Next.js.

Статус RC — ожидается стабильная версия v1.0. Перед использованием в рабочей среде проверьте tanstack/router#5734 (утечка памяти).
