Inferir redes reguladoras de genes (GRN) a partir de datos de expresión genética utilizando algoritmos escalables (GRNBoost2, GENIE3). Úselo al analizar datos de transcriptómica (RNA-seq masivo, RNA-seq unicelular) para identificar relaciones entre el factor de transcripción y el gen objetivo y las interacciones regulatorias. Admite computación distribuida para conjuntos de datos a gran escala.
