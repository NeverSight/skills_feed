Создавайте типобезопасные API с помощью Hono — быстрой и легкой маршрутизации для Cloudflare Workers, Deno, Bun и Node.js. Настройте шаблоны маршрутизации, состав промежуточного программного обеспечения, проверку запросов (Zod/Valibot/Typia/ArkType), клиент/сервер RPC с полным выводом типов и обработку ошибок с помощью HTTPException.

Используйте при: создании API с помощью Hono, настройке проверки запросов с помощью библиотек схем, создании типобезопасной связи между клиентом и сервером RPC, реализации пользовательских цепочек промежуточного программного обеспечения, обработке ошибок с помощью HTTPException, расширении контекста с помощью пользовательских переменных или устранении неполадок вывода типа промежуточного программного обеспечения, путаницы при проверке, проблемах с производительностью RPC или ошибках ввода ответа промежуточного программного обеспечения.
