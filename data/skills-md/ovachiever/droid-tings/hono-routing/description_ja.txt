Hono を使用してタイプセーフな API を構築します。Cloudflare Workers、Deno、Bun、Node.js 向けの高速で軽量なルーティングです。ルーティング パターン、ミドルウェア構成、リクエスト検証 (Zod/Valibot/Typia/ArkType)、完全な型推論を備えた RPC クライアント/サーバー、および HTTPException によるエラー処理をセットアップします。

以下の場合に使用します: Hono を使用した API の構築、スキーマ ライブラリを使用したリクエスト検証のセットアップ、タイプセーフな RPC クライアント/サーバー通信の作成、カスタム ミドルウェア チェーンの実装、HTTPException によるエラーの処理、カスタム変数によるコンテキストの拡張、またはミドルウェアの型推論の問題、検証フックの混乱、RPC パフォーマンスの問題、またはミドルウェア応答の入力エラーのトラブルシューティング。
