使用 Hono 構建類型安全的 API - 適用於 Cloudflare Workers、Deno、Bun 和 Node.js 的快速、輕量級路由。設置路由模式、中間件組合、請求驗證（Zod/Valibot/Typia/ArkType）、具有完整類型推斷的 RPC 客戶端/服務器以及 HTTPException 的錯誤處理。

使用場合：使用 Hono 構建 API、使用模式庫設置請求驗證、創建類型安全的 RPC 客戶端/服務器通信、實現自定義中間件鏈、使用 HTTPException 處理錯誤、使用自定義變量擴展上下文，或者解決中間件類型推斷問題、驗證掛鉤混淆、RPC 性能問題或中間件響應鍵入錯誤。
