يمكنك البناء باستخدام واجهة برمجة تطبيقات Claude messages باستخدام المخرجات المنظمة (الإصدار 0.69.0+، نوفمبر 2025) لضمان التحقق من صحة مخطط JSON. يغطي التخزين المؤقت الفوري (توفير بنسبة 90%)، وتدفق SSE، واستخدام الأداة، وإيقاف النماذج (تم إيقاف الإصدار 3.5/3.7 في أكتوبر 2025). يُستخدم عندما: إنشاء روبوتات الدردشة/الوكلاء باستخدام استجابات JSON التي تم التحقق من صحتها، أو استكشاف أخطاء Rate_limit_error وإصلاحها، أو التحقق من صحة المخرجات المنظمة، أو عدم تنشيط التخزين المؤقت الفوري، أو تحليل تدفق SSE.
