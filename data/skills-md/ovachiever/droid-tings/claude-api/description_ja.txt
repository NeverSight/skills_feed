JSON スキーマ検証を保証するために、構造化出力 (v0.69.0 以降、2025 年 11 月) を使用して Claude Messages API を使用してビルドします。プロンプト キャッシュ (90% 節約)、ストリーミング SSE、ツールの使用、モデルの非推奨 (3.5/3.7 は 2025 年 10 月に廃止) をカバーします。次の場合に使用します: 検証済みの JSON 応答を使用してチャットボット/エージェントを構築する場合、または rate_limit_error、構造化出力検証、プロンプト キャッシュがアクティブ化されない場合、ストリーミング SSE 解析のトラブルシューティングを行う場合。
