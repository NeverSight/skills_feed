Workers + Durable Objects で Cloudflare Agents SDK を使用して AI エージェントを構築します。 Agent SDK (インフラストラクチャ/状態) と AI SDK (よりシンプルなフロー) のどちらを選択するかについての重要なガイダンスが含まれています。

SDK の選択を決定する場合、状態を使用して WebSocket エージェントを構築する場合、Vectorize を使用して RAG、MCP サーバー、マルチエージェント オーケストレーションを使用する場合、または「エージェント クラスは拡張する必要がある」、「new_sqlite_classes」、バインディング エラーのトラブルシューティングを行う場合に使用します。
