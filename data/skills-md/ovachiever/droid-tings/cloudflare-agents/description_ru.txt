Создавайте агенты искусственного интеллекта с помощью Cloudflare Agents SDK на основе Workers + Durable Objects. Включает важные рекомендации по выбору между Agents SDK (инфраструктура/состояние) и AI SDK (более простые процессы).

Используйте при принятии решения о выборе SDK, создании агентов WebSocket с состоянием, RAG с Vectorize, серверов MCP, многоагентной оркестрации или устранении неполадок «Класс агента должен расширяться», «new_sqlite_classes», ошибок привязки.
