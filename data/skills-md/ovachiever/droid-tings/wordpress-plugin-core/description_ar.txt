أنشئ مكونات WordPress الإضافية الآمنة باستخدام الأنماط الأساسية للخطافات وتفاعلات قاعدة البيانات وواجهة برمجة التطبيقات للإعدادات وأنواع المنشورات المخصصة وREST API وAJAX. يغطي ثلاثة أنماط معمارية (Simple، OOP، PSR-4) وSecurity Trinity.

يُستخدم عند إنشاء المكونات الإضافية، أو تنفيذ noces/sanitization/escaping، أو العمل مع عبارات $wpdb المعدة، أو استكشاف أخطاء حقن SQL وإصلاحها، أو ثغرات XSS، أو CSRF، أو أخطاء تنشيط البرنامج المساعد.
