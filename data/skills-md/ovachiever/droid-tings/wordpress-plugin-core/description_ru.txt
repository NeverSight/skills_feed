Создавайте безопасные плагины WordPress с основными шаблонами для перехватов, взаимодействия с базой данных, API настроек, настраиваемых типов сообщений, REST API и AJAX. Охватывает три шаблона архитектуры (Простой, ООП, PSR-4) и Security Trinity.

Используйте при создании плагинов, реализации nonce/санации/экранирования, работе с подготовленными операторами $wpdb или устранении неполадок SQL-инъекций, уязвимостей XSS, CSRF или ошибок активации плагинов.
