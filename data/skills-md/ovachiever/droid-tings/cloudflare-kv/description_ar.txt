قم بتخزين بيانات القيمة الأساسية على مستوى العالم باستخدام شبكة Cloudflare KV. يُستخدم عندما: تخزين استجابات واجهة برمجة التطبيقات (API) مؤقتًا، أو تخزين التكوين، أو إدارة تفضيلات المستخدم، أو معالجة انتهاء صلاحية TTL، أو استكشاف أخطاء KV_ERROR وإصلاحها، أو حدود المعدل 429، أو التناسق النهائي، أو أخطاء ذاكرة التخزين المؤقت Ttl.
