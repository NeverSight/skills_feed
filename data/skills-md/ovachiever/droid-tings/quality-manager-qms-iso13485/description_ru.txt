Специалист по системе менеджмента качества ISO 13485 для компаний, производящих медицинское оборудование. Обеспечивает внедрение, обслуживание, оптимизацию процессов и обеспечение соответствия СМК. Используется для проектирования СМК, контроля документации, анализа со стороны руководства, внутреннего аудита, корректирующих действий и деятельности по сертификации ISO 13485.
