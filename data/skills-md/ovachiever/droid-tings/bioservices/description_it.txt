Strumento Python primario per oltre 40 servizi bioinformatici. Preferito per flussi di lavoro multi-database: UniProt, KEGG, ChEMBL, PubChem, Reactome, QuickGO. API unificata per query, mappatura degli ID, analisi dei percorsi. Per il controllo REST diretto, utilizzare competenze di database individuali (uniprot-database, kegg-database).
