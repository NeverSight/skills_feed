Набор инструментов CLI/Python для быстрых биоинформатических запросов. Предпочтительно для быстрого поиска BLAST. Доступ к более чем 20 базам данных: информация о генах (Ensembl/UniProt), AlphaFold, ARCHS4, Enrichr, OpenTargets, COSMIC, загрузка геномов. Для расширенной BLAST/пакетной обработки используйте biopython. Для интеграции нескольких баз данных используйте биосервисы.
