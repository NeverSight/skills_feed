Configure Tailwind v4 con shadcn/ui usando el patrón en línea @theme y la arquitectura variable CSS. Patrón obligatorio de cuatro pasos: definir variables CSS en la raíz, asignar a las utilidades Tailwind, aplicar estilos base, obtener el modo oscuro automático.

Úselo cuando: inicialice proyectos de React con Tailwind v4, configure el modo oscuro shadcn/ui o arregle colores que no funcionan, tema que no se aplica, variables CSS rotas, errores de tw-animate-css o migrando desde v3.
