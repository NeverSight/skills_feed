Configura Tailwind v4 con shadcn/ui utilizzando il pattern in linea @theme e l'architettura delle variabili CSS. Modello obbligatorio in quattro passaggi: definizione delle variabili CSS alla radice, mappatura alle utilità Tailwind, applicazione degli stili di base, attivazione della modalità oscura automatica.

Da utilizzare quando: si inizializzano progetti React con Tailwind v4, si imposta la modalità oscura shadcn/ui o si correggono colori che non funzionano, temi non applicati, variabili CSS interrotte, errori tw-animate-css o migrazione da v3.
