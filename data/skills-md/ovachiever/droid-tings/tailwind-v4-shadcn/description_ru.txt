Настройте Tailwind v4 с помощью shadcn/ui, используя встроенный шаблон @theme и переменную архитектуру CSS. Четырехэтапный обязательный шаблон: определить переменные CSS в корне, сопоставить их с утилитами Tailwind, применить базовые стили, получить автоматический темный режим.

Используйте при: инициализации проектов React с помощью Tailwind v4, настройке темного режима shadcn/ui или исправлении неработающих цветов, неприменении темы, нарушении переменных CSS, ошибках tw-animate-css или переходе с v3.
