قم بإعداد Tailwind v4 مع shadcn/ui باستخدام النمط المضمن @theme وبنية CSS المتغيرة. نمط إلزامي من أربع خطوات: تحديد متغيرات CSS في الجذر، وتعيين أدوات Tailwind المساعدة، وتطبيق الأنماط الأساسية، والحصول على الوضع المظلم التلقائي.

يُستخدم عندما: تهيئة مشاريع React باستخدام Tailwind v4، أو إعداد الوضع الداكن shadcn/ui، أو إصلاح الألوان التي لا تعمل، أو عدم تطبيق السمة، أو تعطل متغيرات CSS، أو أخطاء tw-animate-css، أو الترحيل من الإصدار 3.
