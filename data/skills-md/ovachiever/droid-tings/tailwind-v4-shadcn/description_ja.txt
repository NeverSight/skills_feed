@theme インライン パターンと CSS 変数アーキテクチャを使用して、shadcn/ui で Tailwind v4 をセットアップします。 4 つのステップの必須パターン: ルートでの CSS 変数の定義、Tailwind ユーティリティへのマッピング、基本スタイルの適用、自動ダーク モードの取得。

以下の場合に使用します: Tailwind v4 で React プロジェクトを初期化する場合、shadcn/ui ダーク モードをセットアップする場合、または機能しない色を修正する場合、テーマが適用されない場合、CSS 変数が壊れている場合、tw-animate-css エラーがある場合、または v3 から移行する場合。
