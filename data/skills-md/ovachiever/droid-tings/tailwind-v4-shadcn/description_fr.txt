Configurez Tailwind v4 avec shadcn/ui en utilisant le modèle en ligne @theme et l'architecture de variables CSS. Modèle obligatoire en quatre étapes : définissez les variables CSS à la racine, mappez-les aux utilitaires Tailwind, appliquez les styles de base, obtenez le mode sombre automatique.

À utiliser lors de : l'initialisation de projets React avec Tailwind v4, la configuration du mode sombre shadcn/ui ou la correction des couleurs qui ne fonctionnent pas, le thème ne s'applique pas, les variables CSS cassées, les erreurs tw-animate-css ou la migration depuis la v3.
