Créez des agents IA autonomes avec le SDK Claude Agent. Les sorties structurées (v0.1.45, novembre 2025) garantissent la validation du schéma JSON, le système de plugins et les hooks pour les flux de travail pilotés par événements. À utiliser dans les cas suivants : création d'agents de codage avec des réponses JSON validées, systèmes SRE, auditeurs de sécurité ou CLI de dépannage introuvable, validation de sortie structurée, erreurs de fork de session.
