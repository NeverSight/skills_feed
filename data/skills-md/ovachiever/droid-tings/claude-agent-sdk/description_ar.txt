قم ببناء وكلاء الذكاء الاصطناعي المستقلين باستخدام Claude Agent SDK. تضمن المخرجات المنظمة (الإصدار 0.1.45، نوفمبر 2025) التحقق من صحة مخطط JSON ونظام المكونات الإضافية والخطافات لسير العمل القائم على الأحداث. يُستخدم عندما: إنشاء وكلاء ترميز مع استجابات JSON تم التحقق من صحتها، أو أنظمة SRE، أو مدققي الأمان، أو استكشاف أخطاء واجهة سطر الأوامر (CLI) وإصلاحها، أو التحقق من صحة المخرجات المنظمة، أو أخطاء تفرع الجلسة.
