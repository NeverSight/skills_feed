Costruisci agenti IA autonomi con Claude Agent SDK. Gli output strutturati (v0.1.45, novembre 2025) garantiscono la convalida dello schema JSON, un sistema di plug-in e hook per flussi di lavoro basati sugli eventi. Utilizzare quando: creazione di agenti di codifica con risposte JSON convalidate, sistemi SRE, auditor di sicurezza o risoluzione dei problemi CLI non trovata, convalida dell'output strutturato, errori di fork della sessione.
