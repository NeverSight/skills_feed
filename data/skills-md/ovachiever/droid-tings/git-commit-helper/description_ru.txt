Автоматически генерировать обычные сообщения о фиксации. Используйте, когда пользователь запускает git commit, вносит изменения или запрашивает справку по сообщению о фиксации. Анализирует git diff для создания четких, описательных сообщений о коммитах. Запускает git commit, проиндексированные изменения, запросы сообщений о фиксации.
