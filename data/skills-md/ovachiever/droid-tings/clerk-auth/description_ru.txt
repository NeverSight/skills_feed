Аутентификация клерка с критическими изменениями API версии 2025-11-10 (конечные точки выставления счетов, платежный_источник→платежный_метод), Next.js v6 async auth(), PKCE для пользовательского OAuth, защита от вставки учетных данных. Используйте при устранении неполадок «Отсутствует секретный ключ клерка», ошибок JWKS, CSRF авторизованных сторон, ограничений размера JWT (1,2 КБ), ошибок заголовка 431 (режим разработки Vite) или тестирования с 424242 OTP.
