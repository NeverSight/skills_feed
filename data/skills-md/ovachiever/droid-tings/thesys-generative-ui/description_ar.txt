قم بدمج واجهة برمجة تطبيقات TheSys C1 Geneative UI لدفق مكونات React التفاعلية (النماذج والمخططات والجداول) من استجابات LLM. يدعم Vite+React وNext.js وCloudflare Workers مع OpenAI وAnthropic Claude وWorkers AI.

يُستخدم عند إنشاء واجهات مستخدم للمحادثة، أو مساعدي الذكاء الاصطناعي ذوي التفاعلات الغنية، أو استكشاف أخطاء الاستجابات الفارغة وإصلاحها، أو فشل تطبيق السمة، أو مشكلات الدفق، أو أخطاء استدعاء الأداة.
