Esegui analisi esplorative complete dei dati su file di dati scientifici in oltre 200 formati di file. Questa abilità dovrebbe essere utilizzata quando si analizza qualsiasi file di dati scientifici per comprenderne la struttura, il contenuto, la qualità e le caratteristiche. Rileva automaticamente il tipo di file e genera report dettagliati sui ribassi con analisi specifiche del formato, metriche di qualità e consigli di analisi downstream. Copre chimica, bioinformatica, microscopia, spettroscopia, proteomica, metabolomica e formati di dati scientifici generali.
