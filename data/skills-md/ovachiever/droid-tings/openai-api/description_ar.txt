يمكنك البناء باستخدام واجهات برمجة التطبيقات عديمة الحالة الخاصة بـ OpenAI - إكمالات الدردشة (GPT-5 وGPT-4o) والتضمينات والصور (DALL-E 3) والصوت (Whisper + TTS) والاعتدال. يتضمن Node.js SDK والأساليب القائمة على الجلب لعمال Cloudflare.

يُستخدم عندما: تنفيذ عمليات إكمال الدردشة باستخدام GPT-5/GPT-4o، وبث الاستجابات باستخدام SSE، واستخدام استدعاءات الوظائف/الأدوات، وإنشاء مخرجات منظمة باستخدام مخططات JSON، وإنشاء تضمينات لـ RAG (دمج النص-3-صغير/كبير)، وإنشاء صور باستخدام DALL-E 3، وتحرير الصور باستخدام GPT-Image-1، ونسخ الصوت باستخدام Whisper، وتوليف الكلام باستخدام TTS (11 صوتًا)، والإشراف على المحتوى. (11 فئة أمان)، أو حدود معدل استكشاف الأخطاء وإصلاحها (429)، أو مفاتيح API غير صالحة (401)، أو فشل استدعاء الوظائف، أو أخطاء تحليل الدفق، أو عدم تطابق أبعاد التضمين، أو تم تجاوز حد الرمز المميز.
