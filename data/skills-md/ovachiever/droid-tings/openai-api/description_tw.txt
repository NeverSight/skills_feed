使用 OpenAI 的無狀態 API 進行構建 - 聊天完成（GPT-5、GPT-4o）、嵌入、圖像（DALL-E 3）、音頻（Whisper + TTS）和審核。包括適用於 Cloudflare Workers 的 Node.js SDK 和基於 fetch 的方法。

使用場合：使用 GPT-5/GPT-4o 實現聊天完成、使用 SSE 流式傳輸響應、使用函數調用/工具、使用 JSON 模式創建結構化輸出、為 RAG 生成嵌入（text-embedding-3-small/large）、使用 DALL-E 3 生成圖像、使用 GPT-Image-1 編輯圖像、使用 Whisper 轉錄音頻、使用 TTS 合成語音（11 個語音）、審核內容（11 個安全類別）、或對速率限制 (429)、無效 API 密鑰 (401)、函數調用失敗、流解析錯誤、嵌入維度不匹配或超出令牌限制進行故障排除。
