Создавайте с использованием API-интерфейсов OpenAI без сохранения состояния: завершение чата (GPT-5, GPT-4o), встраивание, изображения (DALL-E 3), аудио (шепот + TTS) и модерация. Включает Node.js SDK и подходы на основе выборки для Cloudflare Workers.

Используйте, когда: реализация завершения чата с помощью GPT-5/GPT-4o, потоковая передача ответов с помощью SSE, использование вызова функций/инструментов, создание структурированных выходных данных с помощью схем JSON, создание вложений для RAG (встраивание текста-3-маленький/большой), создание изображений с помощью DALL-E 3, редактирование изображений с помощью GPT-Image-1, расшифровка звука с помощью Whisper, синтез речи с помощью TTS (11 голосов), модерация контента (11 категорий безопасности) или ограничения скорости устранения неполадок (429), недопустимые ключи API (401), сбои вызова функций, ошибки анализа потоковой передачи, несоответствия размеров внедрений или превышение лимита токенов.
