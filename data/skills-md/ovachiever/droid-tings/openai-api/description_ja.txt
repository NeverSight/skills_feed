OpenAI のステートレス API - チャット補完 (GPT-5、GPT-4o)、埋め込み、画像 (DALL-E 3)、音声 (Whisper + TTS)、およびモデレーションを使用して構築します。 Node.js SDKとCloudflareワーカー向けのフェッチベースのアプローチが含まれています。

次の場合に使用します: GPT-5/GPT-4o によるチャット補完の実装、SSE による応答のストリーミング、関数呼び出し/ツールの使用、JSON スキーマによる構造化出力の作成、RAG の埋め込みの生成 (text-embedding-3-small/large)、DALL-E 3 による画像の生成、GPT-Image-1 による画像の編集、Whisper による音声の文字起こし、TTS (11 音声) による音声の合成、コンテンツのモデレート(11 の安全カテゴリ)、またはレート制限 (429)、無効な API キー (401)、関数呼び出しの失敗、ストリーミング解析エラー、埋め込みディメンションの不一致、またはトークン制限の超過のトラブルシューティング。
