OpenAI의 상태 비저장 API(채팅 완료(GPT-5, GPT-4o), 임베딩, 이미지(DALL-E 3), 오디오(속삭임 + TTS) 및 조정)로 구축하세요. Cloudflare 작업자를 위한 Node.js SDK 및 가져오기 기반 접근 방식이 포함되어 있습니다.

사용 시기: GPT-5/GPT-4o로 채팅 완료 구현, SSE로 응답 스트리밍, 함수 호출/도구 사용, JSON 스키마로 구조화된 출력 생성, RAG용 임베딩 생성(텍스트 임베딩-3-소형/대형), DALL-E 3으로 이미지 생성, GPT-Image-1로 이미지 편집, Whisper로 오디오 전사, TTS로 음성 합성(11개 음성), 콘텐츠 조정(11개 안전 카테고리) 또는 문제 해결 속도 제한(429), 잘못된 API 키(401), 함수 호출 실패, 스트리밍 구문 분석 오류, 임베딩 차원 불일치 또는 토큰 제한 초과.
