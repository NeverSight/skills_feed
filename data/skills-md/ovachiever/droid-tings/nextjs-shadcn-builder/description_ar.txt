أنشئ تطبيقات Next.js جديدة أو قم بترحيل الواجهات الأمامية الحالية (React وVue وAngular وvanilla JS وما إلى ذلك) إلى Next.js + shadcn/ui مع التحليل والتحويل المنهجي. يفرض مبادئ تصميم shadcn - متغيرات CSS للموضوعات، ومكونات واجهة المستخدم القياسية، ولا توجد قيم مضمنة، وطباعة/ألوان متسقة. يُستخدم لإنشاء تطبيقات Next.js، أو ترحيل الواجهات الأمامية، أو اعتماد shadcn/ui، أو توحيد مكتبات المكونات. يتضمن تكامل MCP لتوثيق shadcn والتحليل الآلي لقاعدة التعليمات البرمجية.
