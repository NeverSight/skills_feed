新しい Next.js アプリケーションを構築するか、体系的な分析と変換を使用して既存のフロントエンド (React、Vue、Angular、バニラ JS など) を Next.js + shadcn/ui に移行します。 shadcn デザイン原則を適用します - テーマ用の CSS 変数、標準 UI コンポーネント、ハードコードされた値なし、一貫したタイポグラフィ/カラー。 Next.js アプリの作成、フロントエンドの移行、shadcn/ui の採用、またはコンポーネント ライブラリの標準化に使用します。 shadcn ドキュメントと自動コードベース分析のための MCP 統合が含まれています。
