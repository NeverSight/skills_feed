Gestisci lo stato del server in React con TanStack Query v5. Configura query con useQuery, mutazioni con useMutation, configura strategie di memorizzazione nella cache QueryClient, implementa aggiornamenti ottimistici e gestisci lo scorrimento infinito con useInfiniteQuery.
