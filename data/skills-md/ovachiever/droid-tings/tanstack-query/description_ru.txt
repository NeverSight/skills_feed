Управляйте состоянием сервера в React с помощью TanStack Query v5. Настраивайте запросы с помощью useQuery, мутации с помощью useMutation, настраивайте стратегии кэширования QueryClient, реализуйте оптимистичные обновления и обрабатывайте бесконечную прокрутку с помощью useInfiniteQuery.

Используйте, когда: настройка выборки данных в проектах React, переход с версии 4 на версию 5 или исправление обязательных ошибок синтаксиса объекта, устранение проблем с помощью обратных вызовов запросов, переименование кэша в gcTime, путаница с isPending и isLoading, устранение проблем с помощью KeepPreviousData.
