Questa competenza deve essere utilizzata per attività di machine learning di serie temporali, tra cui classificazione, regressione, clustering, previsione, rilevamento di anomalie, segmentazione e ricerca di similarità. Da utilizzare quando si lavora con dati temporali, modelli sequenziali o osservazioni indicizzate nel tempo che richiedono algoritmi specializzati oltre gli approcci ML standard. Particolarmente adatto per l'analisi di serie temporali univariate e multivariate con API compatibili con scikit-learn.
