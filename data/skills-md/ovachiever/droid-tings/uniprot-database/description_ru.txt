Прямой доступ через REST API к UniProt. Поиск белков, поиск FASTA, ID-картирование, Swiss-Prot/TrEMBL. Для рабочих процессов Python с несколькими базами данных отдавайте предпочтение биосервисам (унифицированный интерфейс для более чем 40 сервисов). Используйте это для прямой работы HTTP/REST или управления, специфичного для UniProt.
