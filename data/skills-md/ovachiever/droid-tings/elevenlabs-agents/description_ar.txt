أنشئ وكلاء صوتيين للذكاء الاصطناعي للمحادثة باستخدام ElevenLabs Platform باستخدام React أو JavaScript أو React Native أو Swift SDKs.
قم بتكوين الوكلاء والأدوات (العميل/الخادم/MCP) وقواعد معرفة RAG والأصوات المتعددة وScribe في الوقت الفعلي STT.

يُستخدم عندما: إنشاء واجهات الدردشة الصوتية، وتنفيذ وكلاء هاتف الذكاء الاصطناعي باستخدام Twilio، وتكوين سير عمل الوكيل
أو الأدوات، أو إضافة قواعد معرفة RAG، أو الاختبار باستخدام "وكلاء CLI كرمز"، أو استكشاف الأخطاء وإصلاحها التي تم إهمالها @11labs
الحزم، أو قطع صوت Android، أو انتهاكات CSP، أو المتغيرات الديناميكية، أو تكوين WebRTC.
