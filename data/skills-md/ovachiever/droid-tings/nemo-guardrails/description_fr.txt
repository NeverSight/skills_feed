Le cadre de sécurité d'exécution de NVIDIA pour les applications LLM. Comprend la détection de jailbreak, la validation d'entrée/sortie, la vérification des faits, la détection d'hallucinations, le filtrage PII, la détection de toxicité. Utilise Colang 2.0 DSL pour les rails programmables. Prêt pour la production, fonctionne sur GPU T4.
