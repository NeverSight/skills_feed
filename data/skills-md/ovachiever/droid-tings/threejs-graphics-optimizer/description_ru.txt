Правила оптимизации производительности для THREE.js и графического программирования. Охватывает оптимизацию для мобильных устройств, резервные шаблоны, управление памятью, эффективность цикла рендеринга и общие рекомендации по работе с графикой для плавной работы со скоростью 60 кадров в секунду на всех устройствах.
