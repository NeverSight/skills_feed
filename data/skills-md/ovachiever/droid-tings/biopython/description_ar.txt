مجموعة أدوات بايثون الأساسية للبيولوجيا الجزيئية. يُفضل لاستعلامات PubMed/NCBI المستندة إلى Python (Bio.Entrez)، ومعالجة التسلسل، وتحليل الملفات (FASTA، وGenBank، وFASTQ، وPDB)، وسير عمل BLAST المتقدم، والهياكل، وعلم الوراثة. للحصول على انفجار سريع، استخدم gget. للحصول على REST API المباشر، استخدم قاعدة بيانات pubmed.
