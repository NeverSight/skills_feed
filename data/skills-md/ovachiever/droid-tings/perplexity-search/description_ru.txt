Выполняйте поиск в Интернете с помощью искусственного интеллекта и информации в реальном времени, используя модели Perplexity через LiteLLM и OpenRouter. Этот навык следует использовать при поиске текущей информации в Интернете, поиске последней научной литературы, получении обоснованных ответов со ссылками на источники или доступе к информации, выходящей за рамки знаний модели. Предоставляет доступ к нескольким моделям Perplexity, включая Sonar Pro, Sonar Pro Search (расширенный агентный поиск) и Sonar Reasoning Pro, через один ключ API OpenRouter.
