Esegui ricerche web basate sull'intelligenza artificiale con informazioni in tempo reale utilizzando i modelli Perplexity tramite LiteLLM e OpenRouter. Questa abilità dovrebbe essere utilizzata quando si conducono ricerche sul web per informazioni attuali, si trova letteratura scientifica recente, si ottengono risposte fondate con citazioni di fonti o si accede a informazioni oltre il limite di conoscenza del modello. Fornisce l'accesso a più modelli Perplexity tra cui Sonar Pro, Sonar Pro Search (ricerca agenti avanzata) e Sonar Reasoning Pro tramite un'unica chiave API OpenRouter.
