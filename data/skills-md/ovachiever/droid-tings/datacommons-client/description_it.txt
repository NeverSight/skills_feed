Collabora con Data Commons, una piattaforma che fornisce accesso programmatico a dati statistici pubblici provenienti da fonti globali. Utilizza questa competenza quando lavori con dati demografici, indicatori economici, statistiche sanitarie, dati ambientali o qualsiasi set di dati pubblici disponibile tramite Data Commons. Applicabile per interrogare statistiche sulla popolazione, dati sul PIL, tassi di disoccupazione, prevalenza di malattie, risoluzione di entità geografiche ed esplorare le relazioni tra entità statistiche.
