Erstellen Sie Model Context Protocol (MCP)-Server auf Cloudflare Workers – der einzigen Plattform mit offizieller Remote-MCP-Unterstützung. TypeScript-basiert mit OAuth, dauerhaften Objekten und WebSocket-Ruhezustand.

Verwenden Sie diese Option, wenn Sie Remote-MCP-Server bereitstellen, OAuth (GitHub/Google) implementieren, duale Transporte (SSE/HTTP) verwenden oder URL-Pfadkonflikte, McpAgent-Exporte, OAuth-Weiterleitungen oder CORS-Probleme beheben.
