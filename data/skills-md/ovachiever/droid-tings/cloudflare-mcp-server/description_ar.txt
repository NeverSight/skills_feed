قم ببناء خوادم بروتوكول السياق النموذجي (MCP) على Cloudflare Workers - النظام الأساسي الوحيد الذي يتمتع بدعم رسمي لـ MCP عن بعد. يعتمد على TypeScript مع OAuth، والكائنات الدائمة، وإسبات WebSocket.

يُستخدم عندما: نشر خوادم MCP عن بعد، أو تنفيذ OAuth (GitHub/Google)، أو استخدام وسائل النقل المزدوجة (SSE/HTTP)، أو استكشاف أخطاء عدم تطابق مسار URL وإصلاحها، أو تصدير McpAgent، أو عمليات إعادة توجيه OAuth، أو مشكلات CORS.
