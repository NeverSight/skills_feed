公式リモート MCP サポートを備えた唯一のプラットフォームである Cloudflare Workers 上にモデル コンテキスト プロトコル (MCP) サーバーを構築します。 OAuth、Durable Objects、WebSocket ハイバネーションを備えた TypeScript ベース。

次の場合に使用します: リモート MCP サーバーの展開、OAuth (GitHub/Google) の実装、デュアル トランスポート (SSE/HTTP) の使用、または URL パスの不一致、McpAgent エクスポート、OAuth リダイレクト、CORS の問題のトラブルシューティング。
