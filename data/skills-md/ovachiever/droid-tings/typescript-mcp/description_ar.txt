أنشئ خوادم MCP عديمة الحالة باستخدام TypeScript على Cloudflare Workers باستخدام @modelcontextprotocol/sdk. يوفر أنماطًا للأدوات والموارد والمطالبات والمصادقة (مفاتيح API وOAuth وZero Trust).

يُستخدم عند تعريض واجهات برمجة التطبيقات إلى LLMs، أو دمج خدمات Cloudflare (D1، KV، R2، Vectorize)، أو استكشاف أخطاء بناء جملة التصدير وإصلاحها، أو تسرب النقل غير المغلق، أو تكوينات CORS الخاطئة.
