Создавайте серверы MCP без сохранения состояния с помощью TypeScript на Cloudflare Workers, используя @modelcontextprotocol/sdk. Предоставляет шаблоны для инструментов, ресурсов, запросов и аутентификации (ключи API, OAuth, нулевое доверие).

Используйте при предоставлении API для LLM, интеграции сервисов Cloudflare (D1, KV, R2, Vectorize) или устранении синтаксических ошибок экспорта, незакрытых транспортных утечек или неправильных конфигураций CORS.
