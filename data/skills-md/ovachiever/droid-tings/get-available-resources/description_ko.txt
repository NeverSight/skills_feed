이 기술은 사용 가능한 시스템 리소스(CPU 코어, GPU, 메모리, 디스크 공간)를 감지하고 보고하기 위해 계산 집약적인 과학 작업을 시작할 때 사용해야 합니다. 병렬 처리(joblib, 멀티프로세싱), 코어 외부 컴퓨팅(Dask, Zarr), GPU 가속(PyTorch, JAX) 또는 메모리 효율적인 전략을 사용할지 여부와 같은 컴퓨팅 접근 방식 결정을 알리는 리소스 정보와 전략적 권장 사항이 포함된 JSON 파일을 생성합니다. 분석 실행, 모델 교육, 대규모 데이터 세트 처리 또는 리소스 제약이 중요한 작업을 실행하기 전에 이 기술을 사용하세요.
