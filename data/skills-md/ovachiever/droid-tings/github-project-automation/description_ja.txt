CI/CD ワークフロー、発行テンプレート、Dependabot、CodeQL セキュリティ スキャンを使用して GitHub リポジトリのセットアップを自動化します。
実稼働環境でテストされた 12 のワークフローが含まれており、YAML 構文、アクションの固定、構成などの 18 のエラーを防止します。

次の場合に使用します: GitHub Actions CI/CD のセットアップ、Issue/PR テンプレートの作成、Dependabot または CodeQL スキャンの有効化、
Cloudflare Workersへのデプロイ、マトリックステストの実装、またはYAMLインデントのトラブルシューティング、アクションバージョン
ピン留め、シークレット構文、ランナーのバージョン、または CodeQL 設定。

キーワード: github アクション、github ワークフロー、ci/cd、課題テンプレート、プル リクエスト テンプレート、
dependabot、codeql、セキュリティ スキャン、yaml 構文、github オートメーション、リポジトリ セットアップ、
