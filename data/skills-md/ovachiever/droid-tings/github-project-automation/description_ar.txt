أتمتة إعداد مستودع GitHub باستخدام سير عمل CI/CD، وقوالب المشكلات، وDependabot، والفحص الأمني ​​CodeQL.
يتضمن 12 سير عمل تم اختباره في الإنتاج ويمنع 18 خطأ: بناء جملة YAML، وتثبيت الإجراء، والتكوين.

يُستخدم عندما: إعداد GitHub Actions CI/CD، وإنشاء قوالب المشكلات/العلاقات العامة، وتمكين مسح Dependabot أو CodeQL،
النشر إلى Cloudflare Workers، أو تنفيذ اختبار المصفوفة، أو استكشاف أخطاء المسافة البادئة YAML وإصلاحها، إصدار الإجراء
التثبيت أو بناء جملة الأسرار أو إصدارات التشغيل أو تكوين CodeQL.

الكلمات الرئيسية: إجراءات جيثب، سير عمل جيثب، CI/CD، قوالب المشكلة، قوالب طلب السحب،
يعتمد على، Codeql، المسح الأمني، بناء جملة yaml، أتمتة جيثب، إعداد المستودع،
قوالب سير العمل، مصفوفة إجراءات جيثب، إدارة الأسرار، حماية الفروع، أصحاب الأكواد،
مشاريع جيثب، التكامل المستمر، النشر المستمر، خطأ في بناء جملة سير العمل،
