在 Cloudflare Workers 上构建持久、长期运行的工作流程，具有自动重试、状态持久性和
多步骤编排。支持step.do、step.sleep、step.waitForEvent，并运行数小时到数天。

使用时机：创建长时间运行的工作流程、实施重试逻辑、构建事件驱动流程、
协调 API 调用、安排多步骤任务或对 NonRetryableError、I/O 上下文进行故障排除
序列化错误或工作流执行失败。

关键词：cloudflare 工作流程、工作流程工作人员、持久执行、工作流程步骤、
WorkflowEntrypoint、step.do、step.sleep、工作流重试、NonRetryableError、
工作流状态、wrangler 工作流、工作流事件、长时间运行的任务、step.sleepUntil、
step.waitForEvent，工作流绑定
