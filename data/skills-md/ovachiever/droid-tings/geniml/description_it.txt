Questa competenza dovrebbe essere utilizzata quando si lavora con dati di intervalli genomici (file BED) per attività di machine learning. Utilizzare per l'addestramento di incorporamenti di regioni (Region2Vec, BEDspace), analisi ATAC-seq di singole cellule (scEmbed), creazione di picchi di consenso (universi) o qualsiasi analisi basata su ML di regioni genomiche. Si applica alle raccolte di file BED, ai dati scATAC-seq, ai set di dati sull'accessibilità della cromatina e all'apprendimento delle caratteristiche genomiche basato sulla regione.
