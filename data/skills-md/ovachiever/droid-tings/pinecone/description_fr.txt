Base de données vectorielles gérée pour les applications d'IA de production. Entièrement géré, mise à l'échelle automatique, avec recherche hybride (dense + clairsemée), filtrage des métadonnées et espaces de noms. Faible latence (<100 ms p95). Utilisation pour le RAG de production, les systèmes de recommandation ou la recherche sémantique à grande échelle. Idéal pour les infrastructures gérées sans serveur.
