Libreria Python completa per astronomia e astrofisica. Questa competenza dovrebbe essere utilizzata quando si lavora con dati astronomici tra cui coordinate celesti, unità fisiche, file FITS, calcoli cosmologici, sistemi temporali, tabelle, sistemi di coordinate mondiali (WCS) e analisi di dati astronomici. Da utilizzare quando le attività implicano trasformazioni di coordinate, conversioni di unità, manipolazione di file FITS, calcoli di distanze cosmologiche, conversioni di scale temporali o elaborazione di dati astronomici.
