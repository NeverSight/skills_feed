Chat Completions のステートフル後継である OpenAI の Responses API を使用してエージェント AI アプリケーションを構築します。ターン全体で推論を保持することで、マルチターンのパフォーマンスが 5% 向上し、キャッシュ使用率が 40 ～ 80% 向上します。

次の場合に使用します: 永続的な推論を備えた AI エージェントの構築、外部ツール用の MCP サーバーの統合、組み込みのコード インタープリター/ファイル検索/Web 検索の使用、ステートフルな会話の管理、長いタスクのバックグラウンド処理の実装、またはポリモーフィックな出力とサーバー側ツールを取得するためのチャット コンプリーションからの移行。
