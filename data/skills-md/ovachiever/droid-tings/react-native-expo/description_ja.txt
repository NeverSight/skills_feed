Expo SDK 52 を使用して React Native 0.76 以降のアプリを構築します。必須の新しいアーキテクチャ (0.82 以降)、React 19 の変更 (propTypes/forwardRef の削除)、新しい CSS (表示: コンテンツ、mixBlendMode、アウトライン)、Swift iOS テンプレート、および DevTools の移行をカバーします。

Expo アプリの構築、新しいアーキテクチャへの移行、または「ファブリック コンポーネントが見つかりません」、「propTypes が関数ではありません」、「TurboModule が登録されていません」、または Swift AppDelegate エラーのトラブルシューティングを行う場合に使用します。
