Создавайте приложения React Native 0.76+ с помощью Expo SDK 52. Охватывает обязательную новую архитектуру (0.82+), изменения React 19 (удаление propTypes/forwardRef), новый CSS (отображение: содержимое, mixBlendMode, структура), шаблон Swift iOS и миграцию DevTools.

Используйте при создании приложений Expo, переходе на новую архитектуру или устранении неполадок «Компонент Fabric не найден», «propTypes не является функцией», «TurboModule не зарегистрирован» или ошибок Swift AppDelegate.
