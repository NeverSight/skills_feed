Crea app ChatGPT con server MCP su Cloudflare Workers. Estendi ChatGPT con strumenti personalizzati e widget interattivi (interfaccia utente HTML/JS).

Da utilizzare quando: sviluppo di estensioni ChatGPT, implementazione di server MCP o risoluzione dei problemi di blocco CORS (consenti chatgpt.com), widget 404 (ui://widget/ mancante), tipo MIME errato (text/html+skybridge) o associazione ASSETS non definita.
