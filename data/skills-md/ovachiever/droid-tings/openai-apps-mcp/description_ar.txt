يُستخدم عندما: تطوير ملحقات ChatGPT، أو تنفيذ خوادم MCP، أو استكشاف أخطاء حظر CORS وإصلاحها (السماح بـ chatgpt.com)، أو عناصر واجهة المستخدم 404 (واجهة مستخدم مفقودة://widget/)، أو نوع MIME خاطئ (نص/html+skybridge)، أو ربط ASSETS غير محدد.
