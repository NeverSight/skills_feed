اكتب HTML دلاليًا وسهل الوصول إليه وفعالاً باستخدام أفضل الممارسات الحديثة. يُستخدم عندما يُطلب منك (1) إنشاء صفحات أو مستندات HTML، (2) كتابة علامات دلالية، (3) تحسين إمكانية الوصول، (4) تحسين بنية HTML والأداء، (5) تنفيذ النماذج أو الجداول أو التخطيطات المعقدة، أو عند ظهور عبارات مثل "صفحة HTML"، أو "صفحة ويب"، أو "توصيف"، أو "HTML الدلالي"، أو "إمكانية الوصول".
