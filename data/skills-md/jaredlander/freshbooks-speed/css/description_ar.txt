تطوير CSS على مستوى الخبراء مع الميزات الحديثة وأفضل الممارسات. يُستخدم عندما يُطلب منك (1) كتابة CSS أو تصحيح أخطاءه، (2) تنفيذ التخطيطات باستخدام استعلامات flexbox أو الشبكة أو الحاوية، (3) إنشاء رسوم متحركة وانتقالات، (4) تحسين أداء CSS، (5) العمل مع معالجات CSS الأولية أو CSS-in-JS، (6) تنفيذ تصميم سريع الاستجابة، أو عندما تظهر عبارات مثل "style" أو "CSS" أو "ورقة الأنماط" أو "design" أو "layout" أو "animation".
