تطوير JavaScript وReact على مستوى الخبراء. يُستخدم عندما يُطلب منك (1) كتابة تعليمات برمجية JavaScript تتطلب أنماطًا متقدمة مثل عمليات الإغلاق أو الوكلاء أو المولدات أو التكرارات غير المتزامنة، (2) إنشاء تطبيقات React باستخدام الخطافات أو السياق أو التشويق أو مكونات الخادم، (3) تحسين أداء JavaScript/React، (4) تنفيذ إدارة الحالة المعقدة، (5) كتابة TypeScript باستخدام أنماط الكتابة المتقدمة، أو عند استخدام عبارات مثل "مكون React" أو "وظيفة JavaScript" أو "TypeScript" أو "الخطافات" أو "إدارة الحالة" تظهر "الواجهة الأمامية"، و"تطبيق الويب".
