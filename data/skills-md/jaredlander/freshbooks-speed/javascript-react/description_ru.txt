Разработка JavaScript и React на экспертном уровне. Используйте, когда вас просят (1) написать код JavaScript, требующий расширенных шаблонов, таких как замыкания, прокси, генераторы или асинхронные итераторы, (2) создать приложения React с перехватчиками, контекстом, приостановкой или серверными компонентами, (3) оптимизировать производительность JavaScript/React, (4) реализовать сложное управление состоянием, (5) написать TypeScript с расширенными шаблонами типов или когда такие фразы, как «компонент React», «функция JavaScript», «TypeScript», «перехватчики», «управление состоянием», «интерфейс», «веб» приложение».
