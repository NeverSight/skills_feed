Scrivere unit test completi per il codice. Da utilizzare quando viene richiesto di (1) scrivere test per codice nuovo o esistente, (2) aggiungere test unitari, (3) testare una funzione/modulo/classe, (4) verificare che il codice funzioni ancora dopo le modifiche, (5) creare copertura del test o quando vengono visualizzate frasi come "scrivi test", "aggiungi test", "prova questo", "assicurati che funzioni". Supporta Python, JavaScript, R, Rust, Go, C++, SQL, Bash, Ansible, Kubernetes/Kustomize, Docker e Docker Compose.
