DBOS Python SDK для создания надежных, отказоустойчивых приложений с устойчивыми рабочими процессами. Используйте этот навык при написании кода Python с помощью DBOS, создании рабочих процессов и шагов, использовании очередей, использовании DBOSClient из внешних приложений или создании приложений, которые должны быть устойчивыми к сбоям.
