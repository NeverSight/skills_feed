DBOS Go SDK для создания надежных, отказоустойчивых приложений с устойчивыми рабочими процессами. Используйте этот навык при написании кода Go с DBOS, создании рабочих процессов и шагов, использовании очередей, использовании клиента DBOS из внешних приложений или создании приложений Go, которые должны быть устойчивыми к сбоям.
