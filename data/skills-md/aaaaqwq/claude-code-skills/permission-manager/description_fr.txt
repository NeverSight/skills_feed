Gérez la configuration globale des autorisations des outils pour Claude Code, en ajoutant automatiquement des commandes MCP ou d'autres outils à AllowTools pour éviter une approbation manuelle à chaque fois qu'ils sont utilisés. Workflow : confirmez la commande que l'utilisateur doit ajouter -> confirmez le niveau d'ajout (global par défaut ~/.claude.json) -> effectuez l'ajout -> vérifiez et invitez à redémarrer.
