Управляйте глобальной конфигурацией разрешений инструментов для Claude Code, автоматически добавляя команды MCP или другие инструменты в список разрешенных инструментов, чтобы избежать ручного утверждения при каждом их использовании. Рабочий процесс: подтвердите команду, которую пользователь должен добавить -> подтвердите уровень добавления (по умолчанию глобальный ~/.claude.json) -> выполните добавление -> проверьте и предложите перезапустить.
