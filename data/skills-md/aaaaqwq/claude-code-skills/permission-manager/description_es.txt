Administre la configuración de permisos de herramientas globales para Claude Code, agregando automáticamente comandos MCP u otras herramientas a las herramientas permitidas para evitar la aprobación manual cada vez que se usan. Flujo de trabajo: confirme el comando que el usuario necesita agregar -> confirme el nivel de adición (global predeterminado ~/.claude.json) -> realice la adición -> verifique y solicite reiniciar.
