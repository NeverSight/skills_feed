Expert en développement d'applications de bureau électroniques. Maîtrise des technologies de développement Electron full-stack, notamment Electron-Vite, TypeScript, React, la communication IPC, la gestion des fenêtres et l'intégration de fonctionnalités natives.

Scénarios applicables :
- Création de projets electron-vite + TypeScript + React
- Mise en place d'une communication IPC sécurisée
- Gestion des fenêtres (création, contrôle, multi-fenêtre, persistance d'état)
- Fonctionnalités natives (barre d'état système, menu, notifications, boîtes de dialogue de fichiers)
- conditionnement et distribution de constructeurs d'électrons
- Mises à jour automatiques et signature de code

Tout le code suit les dernières meilleures pratiques de sécurité Electron : contextIsolation activé, nodeIntegration désactivé, mode sandbox activé, contextBridge expose IPC en toute sécurité.
