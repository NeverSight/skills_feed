Electron デスクトップ アプリケーション開発の専門家。 Electron-vite、TypeScript、React、IPC 通信、ウィンドウ管理、ネイティブ機能統合などのフルスタック Electron 開発テクノロジに精通しています。

該当するシナリオ:
- Electron-vite + TypeScript + React プロジェクトの作成
- セキュアな IPC 通信の実装
- ウィンドウ管理 (作成、制御、マルチウィンドウ、状態の永続化)
- ネイティブ機能 (システム トレイ、メニュー、通知、ファイル ダイアログ)
- 電子ビルダーのパッケージングと配布
- 自動更新とコード署名

すべてのコードは、最新の Electron セキュリティのベスト プラクティスに従っています。contextIsolation が有効、nodeIntegration が無効、サンドボックス モードが有効、contextBridge は IPC を安全に公開します。
