Outil de nettoyage des fichiers système. Analyse et identifie les fichiers volumineux, les fichiers indésirables (fichiers temporaires, caches, journaux, sauvegardes, etc.) et fournit une interface de nettoyage interactive permettant aux utilisateurs de sélectionner les suppressions. Utilisez cette compétence lorsque les utilisateurs ont besoin de libérer de l'espace disque, d'organiser des fichiers système, de rechercher des fichiers volumineux, de supprimer des fichiers inutiles ou de libérer de l'espace de stockage.
