Herramienta de limpieza de archivos del sistema. Escanea e identifica archivos grandes, archivos basura (archivos temporales, cachés, registros, copias de seguridad, etc.) y proporciona una interfaz de limpieza interactiva para que los usuarios seleccionen eliminaciones. Utilice esta habilidad cuando los usuarios necesiten liberar espacio en el disco, organizar archivos del sistema, buscar archivos grandes, eliminar archivos basura o liberar espacio de almacenamiento.
