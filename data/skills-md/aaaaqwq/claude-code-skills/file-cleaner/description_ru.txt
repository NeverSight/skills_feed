Инструмент очистки системных файлов. Сканирует и идентифицирует большие файлы, ненужные файлы (временные файлы, кэши, журналы, резервные копии и т. д.) и предоставляет пользователям интерактивный интерфейс очистки для выбора удаления. Используйте этот навык, когда пользователям нужно освободить место на диске, упорядочить системные файлы, найти большие файлы, удалить ненужные файлы или освободить место для хранения.
