Tool zur Bereinigung von Systemdateien. Scannt und identifiziert große Dateien und Junk-Dateien (temporäre Dateien, Caches, Protokolle, Backups usw.) und bietet Benutzern eine interaktive Bereinigungsoberfläche zur Auswahl von Löschungen. Verwenden Sie diese Fähigkeit, wenn Benutzer Speicherplatz freigeben, Systemdateien organisieren, große Dateien finden, Junk-Dateien löschen oder Speicherplatz freigeben müssen.
