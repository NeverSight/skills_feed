A bidirectional integration channel between Feishu (Lark/Feishu) and OpenClaw. Receive and send messages via a Feishu bot, supporting private chats, group chats, @mention detection, card messages, and file transfers. Use this skill when you need to interact with an AI assistant through Feishu, have Feishu messages trigger AI responses, or send messages from OpenClaw to Feishu. Difference from feishu-automation: this skill focuses on message-channel integration, while feishu-automation focuses on automating operations on the Feishu platform (such as multi-dimensional tables, documents, etc.).
