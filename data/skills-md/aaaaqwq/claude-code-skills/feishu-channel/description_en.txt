A bidirectional integration channel between Feishu/Lark and OpenClaw. Use a Feishu bot to receive and send messages, supporting private and group chats, @ mention detection, card messages, and file transfer. Use this skill when you need to interact with an AI assistant through Feishu, have Feishu messages trigger AI responses, or send messages from OpenClaw to Feishu. Difference from feishu-automation: this skill focuses on message-channel integration, while feishu-automation focuses on automating Feishu platform operations (multi-dimensional tables, documents, etc.).
