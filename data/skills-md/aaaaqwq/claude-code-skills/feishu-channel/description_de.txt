Ein bidirektionaler Integrationskanal zwischen Feishu (Lark/Feishu) und OpenClaw. Empfangen und senden Sie Nachrichten über einen Feishu-Bot, der private Chats, Gruppenchats, @mention-Erkennung, Kartennachrichten und Dateiübertragungen unterstützt. Verwenden Sie diese Fähigkeit, wenn Sie über Feishu mit einem KI-Assistenten interagieren, Feishu-Nachrichten KI-Antworten auslösen lassen oder Nachrichten von OpenClaw an Feishu senden müssen. Unterschied zur Feishu-Automatisierung: Diese Fähigkeit konzentriert sich auf die Integration von Nachrichtenkanälen, während sich die Feishu-Automatisierung auf die Automatisierung von Vorgängen auf der Feishu-Plattform (z. B. mehrdimensionale Tabellen, Dokumente usw.) konzentriert.
