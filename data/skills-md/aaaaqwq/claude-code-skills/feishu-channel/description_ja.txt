Feishu (Lark/Feishu) と OpenClaw 間の双方向統合チャネル。 Feishu ボット経由でメッセージを送受信し、プライベート チャット、グループ チャット、@メンション検出、カード メッセージ、ファイル転送をサポートします。 Feishu を通じて AI アシスタントと対話する必要がある場合、Feishu メッセージで AI 応答をトリガーする必要がある場合、または OpenClaw から Feishu にメッセージを送信する必要がある場合に、このスキルを使用します。 feishu-automation との違い: このスキルはメッセージ チャネルの統合に重点を置いているのに対し、feishu-automation は Feishu プラットフォーム (多次元テーブル、ドキュメントなど) での操作の自動化に重点を置いています。
