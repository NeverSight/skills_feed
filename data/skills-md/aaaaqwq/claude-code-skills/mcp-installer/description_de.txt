Durchsuchen Sie GitHub und installieren und konfigurieren Sie MCP-Servertools (Model Context Protocol) automatisch in der Claude-Konfigurationsdatei. Lösen Sie diesen Skill aus, wenn der Benutzer MCP-Tools installieren muss. Arbeitsablauf: Auf GitHub nach MCP-Projekten suchen → Npx-Konfiguration extrahieren → zu ~/.claude.json hinzufügen → API-Schlüssel verarbeiten (falls vorhanden).
