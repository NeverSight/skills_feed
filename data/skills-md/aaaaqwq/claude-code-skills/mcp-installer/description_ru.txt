Найдите GitHub и автоматически установите и настройте серверные инструменты MCP (Model Context Protocol) в файл конфигурации Claude. Активируйте этот навык, когда пользователю необходимо установить инструменты MCP. Рабочий процесс: поиск проектов MCP на GitHub → извлечение конфигурации npx → добавление в ~/.claude.json → обработка ключей API (если есть).
