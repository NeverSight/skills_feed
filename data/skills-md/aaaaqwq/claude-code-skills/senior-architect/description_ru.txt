Этот навык следует использовать, когда пользователь просит «спроектировать архитектуру системы», «оценить микросервисы по сравнению с монолитом», «создать архитектурные диаграммы», «проанализировать зависимости», «выбрать базу данных», «планировать масштабируемость», «принять технические решения» или «пересмотреть проект системы». Используйте для записей архитектурных решений (ADR), оценки технического стека, обзоров проектирования систем, анализа зависимостей и создания архитектурных диаграмм в формате Mermaid, PlantUML или ASCII.
