يجب استخدام هذه المهارة عندما يطلب المستخدم "تصميم بنية النظام"، أو "تقييم الخدمات الصغيرة مقابل المتراصة"، أو "إنشاء مخططات معمارية"، أو "تحليل التبعيات"، أو "اختيار قاعدة بيانات"، أو "التخطيط لقابلية التوسع"، أو "اتخاذ قرارات فنية"، أو "مراجعة تصميم النظام". يُستخدم لسجلات قرارات البنية (ADRs)، وتقييم المكدس الفني، ومراجعات تصميم النظام، وتحليل التبعية، وإنشاء مخططات معمارية بتنسيق Mermaid أو PlantUML أو ASCII.
