نظام آلي للتخطيط والتفكير اليومي مع ملخصات صباحية، وإرشادات للاسترخاء، وتحفيزات للنوم، ومراجعات أسبوعية. يُستخدم عندما يريد المستخدم إعداد روتين يومي منظم، أو ملخصات صباحية، أو مطالبات بالتأمل المسائي، أو جلسات تخطيط أسبوعية. تتضمن المشغلات طلبات الجداول اليومية، أو الملخصات الصباحية، أو إجراءات الاسترخاء، أو تذكيرات النوم، أو المراجعات الأسبوعية، أو أنظمة الإنتاجية، أو أتمتة التخطيط اليومي.
