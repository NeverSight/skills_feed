Sistema automatizado de planificación y reflexión diaria con resúmenes matutinos, indicaciones para relajarse, estímulos para dormir y revisiones semanales. Úselo cuando el usuario desee establecer una rutina diaria estructurada, sesiones informativas matutinas, indicaciones de reflexión vespertinas o sesiones de planificación semanales. Los desencadenantes incluyen solicitudes de horarios diarios, resúmenes matutinos, rutinas de relajación, recordatorios de sueño, revisiones semanales, sistemas de productividad o automatización de la planificación diaria.
