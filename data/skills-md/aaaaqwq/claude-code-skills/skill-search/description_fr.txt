Recherchez des compétences de haute qualité qui correspondent à la description de l'utilisateur sur des sites officiels comme GitHub et SkillsMP, présentez-les pour sélection, puis clonez et installez automatiquement la compétence choisie dans le répertoire global ~/.claude/skills/. Déclenchez cette compétence chaque fois que l'utilisateur a besoin de rechercher ou d'installer une nouvelle compétence.
