Найдите качественные навыки, соответствующие описанию пользователя, на официальных сайтах, таких как GitHub и SkillsMP, представьте их для выбора, затем автоматически клонируйте и установите выбранный навык в глобальный каталог ~/.claude/skills/. Активируйте этот навык всякий раз, когда пользователю нужно найти или установить новый навык.
