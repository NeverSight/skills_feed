Suchen Sie auf offiziellen Websites wie GitHub und SkillsMP nach hochwertigen Fähigkeiten, die der Beschreibung des Benutzers entsprechen, präsentieren Sie sie zur Auswahl und klonen Sie dann automatisch die ausgewählte Fähigkeit und installieren Sie sie im globalen Verzeichnis ~/.claude/skills/. Lösen Sie diesen Skill immer dann aus, wenn der Benutzer nach einem neuen Skill suchen oder ihn installieren muss.
