Отправляйте в чат краткие описания статусов для долгосрочных задач. Используйте, когда вам необходимо предоставлять периодические обновления во время многоэтапных операций, подтверждать завершение задачи или уведомлять о сбоях. Включает автоматический периодический мониторинг, который отправляет обновления каждые 5 секунд, шаблоны сообщений о состоянии и вспомогательную функцию для согласованного отчета о состоянии.
