Envoyez de courtes descriptions de statut dans le chat pour les tâches de longue durée. À utiliser lorsque vous devez fournir des mises à jour périodiques lors d'opérations en plusieurs étapes, confirmer l'achèvement d'une tâche ou notifier des échecs. Comprend une surveillance périodique automatisée qui envoie des mises à jour toutes les 5 secondes, des modèles de messages d'état et une fonction d'assistance pour des rapports d'état cohérents.
