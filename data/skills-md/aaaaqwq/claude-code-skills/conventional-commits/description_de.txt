Formatieren Sie Commit-Nachrichten mit der Spezifikation „Conventional Commits“. Wird verwendet, wenn Sie Commits erstellen, Commit-Nachrichten schreiben oder wenn der Benutzer Commits, Git-Commits oder Commit-Nachrichten erwähnt. Stellt sicher, dass Commits dem Standardformat für automatisierte Tools, Änderungsprotokollgenerierung und semantische Versionierung folgen.
