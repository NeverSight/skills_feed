Корпоративная система автоматизации обслуживания клиентов WeChat. Автоматически утверждает запросы на добавление в друзья, предоставляет интеллектуальные вопросы и ответы на основе базы знаний и отправляет оповещения о вмешательстве человека по неизвестным вопросам. Чат-помощник с искусственным интеллектом, разработанный для сценариев обслуживания клиентов Enterprise WeChat.
