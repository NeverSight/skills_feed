Bewerten, optimieren und verbessern Sie Eingabeaufforderungen mithilfe von 58 bewährten Eingabeaufforderungstechniken. Wird verwendet, wenn der Benutzer eine Eingabeaufforderung verbessern, optimieren oder analysieren möchte. wenn eine Eingabeaufforderung mehr Klarheit, Spezifität oder Struktur erfordert; oder beim Generieren von Eingabeaufforderungsvarianten für verschiedene Anwendungsfälle. Umfasst Qualitätsbewertung, gezielte Verbesserungen und automatische Optimierung für Techniken wie CoT, Few-Shot-Learning, Rollenspiele und mehr als 50 mehr.
