قم بتقييم المطالبات وتحسينها وتعزيزها باستخدام 58 تقنية مطالبة مثبتة. يُستخدم عندما يطلب المستخدم تحسين المطالبة أو تحسينها أو تحليلها؛ عندما تحتاج المطالبة إلى وضوح أو خصوصية أو بنية أفضل؛ أو عند إنشاء اختلافات سريعة لحالات استخدام مختلفة. يغطي تقييم الجودة، والتحسينات المستهدفة، والتحسين التلقائي عبر تقنيات مثل CoT، والتعلم باللقطات القليلة، ولعب الأدوار، وأكثر من 50 تقنية أخرى.
