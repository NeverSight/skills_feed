تتبع جلسات العمل العميقة محليًا (البدء/الإيقاف/الحالة) وقم بإنشاء خريطة حرارية لدقائق يوميًا بنمط رسم بياني لمساهمة GitHub للمشاركة (على سبيل المثال، عبر Telegram). يُستخدم عندما يقول المستخدم أشياء مثل "بدء العمل العميق"، أو "إيقاف العمل العميق"، أو "هل أنا في جلسة؟"، أو "إظهار الرسم البياني للعمل العميق"، أو يطلب مراجعة سجل العمل العميق.
