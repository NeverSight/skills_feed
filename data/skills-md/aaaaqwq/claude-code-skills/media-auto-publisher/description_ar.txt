أداة عالمية للنشر التلقائي لمقالات الوسائط الذاتية. يدعم سير عمل النشر الآلي لثلاث منصات: Baijiahao وSohu وZhihu. يستخدم أدوات Playwright MCP للتنقل بين المنصات والوصول إلى صفحات النشر، مع تبديل الحساب عبر إدارة ملفات تعريف الارتباط StorageState. استخدم هذه المهارة عندما يحتاج المستخدمون إلى نشر المقالات على منصات الوسائط الذاتية، أو فتح صفحات نشر النظام الأساسي، أو تبديل الحسابات. سير العمل: 1) التحقق من النظام الأساسي المحدد وفتحه 2) التحقق من حالة تسجيل الدخول 3) الإغلاق التلقائي للنوافذ المنبثقة للإعلانات 4) انتقل إلى صفحة نشر المقالة 5) جمع معلومات النشر 6) الملء التلقائي والنشر.
