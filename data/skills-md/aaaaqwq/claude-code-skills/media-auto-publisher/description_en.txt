Universal self-media article auto-publishing tool. Supports automated publishing workflows for three platforms: Baijiahao, Sohu, and Zhihu. Uses Playwright MCP tools to navigate platforms and access publishing pages, with account switching via storageState cookie management. Use this skill when users need to publish articles to self-media platforms, open platform publishing pages, or switch accounts. Workflow: 1) Check and open specified platform 2) Verify login status 3) Auto-close ad pop-ups 4) Navigate to article publishing page 5) Collect publishing info 6) Auto-fill and publish.
