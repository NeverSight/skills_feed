Universal self-media article auto-publishing tool. Supports automated publishing workflows for Baijiahao, Sohu and Zhihu. Uses the Playwright MCP tool for platform navigation and entry into publishing pages, and supports account switching via storageState cookie management. Use this skill when users need to publish articles to self-media platforms, open platform publishing pages, or switch accounts. Workflow: 1) Check and open the specified platform 2) Verify login status 3) Automatically close ad pop-ups 4) Navigate to the article publishing page 5) Collect publishing information 6) Auto-fill and publish.
