Генерация мультимодального контента (изображения, видео). Используйте этот навык, когда пользователям необходимо генерировать изображения, создавать изображения, создавать видео, выполнять рисование с помощью ИИ, создавать графику, рисовать изображения, создавать видео, преобразование текста в изображение или текст в видео. Автоматически вызывать мультимодальный агент для генерации.
