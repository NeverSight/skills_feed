Optimierung und Formatierungsverbesserung von Feishu-Dokumentinhalten. Nutzen Sie diese Fähigkeit, wenn Benutzer das Layout, die Struktur, das Format oder die Ästhetik von Feishu-Dokumenten verbessern müssen. Unterstützt: (1) Lesen von Feishu-Dokumentinhalten, (2) Optimieren von Dokumentstruktur und -hierarchie, (3) Löschen und Ersetzen von Dokumentinhalten, (4) Bearbeiten von Dokumenten über Browser-Automatisierung. Auslöserwörter: Feishu-Dokument optimieren, Dokument verschönern, Dokumentformat organisieren, Dokumentlayout.
