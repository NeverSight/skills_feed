Herramienta de sincronización con un solo clic del entorno Claude Code. Sincronice todas las configuraciones desde un repositorio de GitHub a local: estilos de salida, mensajes globales CLAUDE.md, configuraciones del servidor MCP. Ideal para unificar entornos en múltiples dispositivos, restaurar después de cambiar de computadora y compartir la configuración del equipo. Utilice esta habilidad cuando los usuarios necesiten sincronizar las configuraciones del entorno de Claude Code desde un repositorio de GitHub.
