Outil de synchronisation en un clic de l'environnement Claude Code. Synchronisez toutes les configurations d'un référentiel GitHub vers local : styles de sortie, invites globales CLAUDE.md, configurations du serveur MCP. Idéal pour unifier les environnements sur plusieurs appareils, restaurer après un changement d'ordinateur et partager la configuration d'une équipe. Utilisez cette compétence lorsque les utilisateurs doivent synchroniser les configurations de l'environnement Claude Code à partir d'un référentiel GitHub.
