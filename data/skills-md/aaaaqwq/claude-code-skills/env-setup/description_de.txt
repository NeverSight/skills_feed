Ein-Klick-Synchronisierungstool für die Claude Code-Umgebung. Synchronisieren Sie alle Konfigurationen aus einem GitHub-Repository mit lokalen: Ausgabestilen, globalen CLAUDE.md-Eingabeaufforderungen und MCP-Serverkonfigurationen. Ideal für die Vereinheitlichung von Umgebungen auf mehreren Geräten, die Wiederherstellung nach einem Computerwechsel und die gemeinsame Nutzung von Teamkonfigurationen. Verwenden Sie diesen Skill, wenn Benutzer Claude Code-Umgebungskonfigurationen aus einem GitHub-Repository synchronisieren müssen.
