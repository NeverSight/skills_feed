Strumento di sincronizzazione con un clic dell'ambiente Claude Code. Sincronizza tutte le configurazioni da un repository GitHub in locale: stili di output, prompt globali CLAUDE.md, configurazioni del server MCP. Ideale per unificare ambienti su più dispositivi, ripristinare dopo aver cambiato computer e condividere la configurazione del team. Utilizza questa competenza quando gli utenti devono sincronizzare le configurazioni dell'ambiente Claude Code da un repository GitHub.
