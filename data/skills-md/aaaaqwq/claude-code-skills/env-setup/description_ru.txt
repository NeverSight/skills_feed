Инструмент синхронизации среды Claude Code одним щелчком мыши. Синхронизируйте все конфигурации из репозитория GitHub с локальным: стили вывода, глобальные запросы CLAUDE.md, конфигурации сервера MCP. Идеально подходит для объединения сред на нескольких устройствах, восстановления после замены компьютеров и совместного использования конфигурации команды. Используйте этот навык, когда пользователям необходимо синхронизировать конфигурации среды Claude Code из репозитория GitHub.
