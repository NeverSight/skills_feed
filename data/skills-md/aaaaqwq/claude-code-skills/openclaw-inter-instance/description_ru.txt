Межэкземплярное взаимодействие OpenClaw. Используйте этот навык, когда необходимо передавать сообщения, синхронизировать данные или удаленно выполнять команды между несколькими экземплярами OpenClaw. Охватывает обмен сообщениями между агентами, удаленное выполнение nodes.run, связь на уровне файлов и многое другое.
