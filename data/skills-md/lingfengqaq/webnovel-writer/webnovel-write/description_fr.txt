Rédige des chapitres de romans Web (3 000 à 5 000 mots) en utilisant l'architecture v5.2. L'agent de contexte produit un brief créatif, le rédacteur produit du texte pur, révise les agents signalent les problèmes, le peaufinage du roman Web corrige les problèmes, l'agent de données extrait les entités et enregistre les crochets/modèles.
