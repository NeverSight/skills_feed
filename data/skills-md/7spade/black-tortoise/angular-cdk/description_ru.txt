Опыт Angular Component Dev Kit (CDK) для создания пользовательских компонентов со специальными возможностями, наложениями, перетаскиванием, виртуальной прокруткой и утилитами макетирования. Используйте при создании пользовательских компонентов пользовательского интерфейса, реализации функций специальных возможностей, создании наложений/диалогов, виртуальной прокрутки для больших списков или использования утилит CDK для адаптивных макетов.
