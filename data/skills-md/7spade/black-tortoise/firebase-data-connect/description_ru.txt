Интеграция Firebase Data Connect для доступа к данным на основе GraphQL с PostgreSQL. Используйте при построении схем, запросов, мутаций GraphQL или интеграции Firebase Data Connect с приложениями Angular. Поддерживает типобезопасное создание пакетов SDK, подписки в реальном времени и проверку данных на стороне сервера.
