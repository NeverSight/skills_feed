Rendu des vidéos à partir du code du composant React/Remotion via inference.sh. Passez le code TSX, obtenez MP4. Prend en charge toutes les API Remotion : useCurrentFrame, useVideoConfig, spring, interpolate, AbsoluteFill, Sequence. Résolution configurable, FPS, durée, codec. À utiliser pour : la génération de vidéos programmatiques, les graphiques animés, le motion design, les vidéos basées sur les données, les animations React en vidéo. Déclencheurs : remotion, rendu vidéo à partir du code, tsx vers vidéo, vidéo de réaction, vidéo programmatique, rendu par remotion, code en vidéo, vidéo d'animation, code graphique animé, vidéo d'animation de réaction
