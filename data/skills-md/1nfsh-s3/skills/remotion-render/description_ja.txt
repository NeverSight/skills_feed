React/Remotion コンポーネント コードから inference.sh 経由でビデオをレンダリングします。 TSXコードを渡してMP4を取得します。すべての Remotion API をサポートします: useCurrentFrame、useVideoConfig、spring、interpolate、AbsoluteFill、Sequence。設定可能な解像度、FPS、期間、コーデック。用途: プログラムによるビデオ生成、アニメーション グラフィックス、モーション デザイン、データ駆動型ビデオ、ビデオへのアニメーションの反応。トリガー: リモート、コードからのビデオのレンダリング、tsx からビデオへの反応、ビデオの反応、プログラムによるビデオ、リモート レンダリング、コードからビデオへの変換、アニメーション ビデオ、モーション グラフィックス コード、アニメーション ビデオの反応
