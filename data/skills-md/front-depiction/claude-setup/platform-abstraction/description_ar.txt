استخدم تجريدات @effect/platform للإدخال/الإخراج للملفات عبر الأنظمة الأساسية، وتوليد العمليات، وعملاء HTTP، والعمليات الطرفية. قم بتطبيق هذه المهارة عند كتابة تعليمات برمجية تتفاعل مع نظام الملفات، أو تولد العمليات، أو تقدم طلبات HTTP، أو تنفذ إدخال/إخراج وحدة التحكم لضمان إمكانية النقل عبر بيئات Node.js وBun والمتصفح.
