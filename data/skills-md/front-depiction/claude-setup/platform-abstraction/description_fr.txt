Utilisez les abstractions @effect/platform pour les E/S de fichiers multiplateformes, la génération de processus, les clients HTTP et les opérations de terminal. Appliquez cette compétence lors de l'écriture de code qui interagit avec le système de fichiers, génère des processus, effectue des requêtes HTTP ou effectue des E/S de console pour garantir la portabilité dans les environnements Node.js, Bun et de navigateur.
