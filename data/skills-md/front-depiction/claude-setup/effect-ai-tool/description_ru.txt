Определите и внедрите инструменты искусственного интеллекта с помощью API-интерфейсов Tool и Toolkit @effect/ai. Используйте при построении интеграции LLM с определениями безопасных по типам инструментов, проверкой параметров и реализациями обработчиков. Охватывает определяемые пользователем инструменты, инструменты, определяемые поставщиком, а также состав набора инструментов.
