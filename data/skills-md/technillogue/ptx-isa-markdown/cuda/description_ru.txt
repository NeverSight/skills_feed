Разработка ядра CUDA, отладка и оптимизация производительности для Claude Code. Используйте при написании, отладке или оптимизации кода CUDA, ядер графического процессора или параллельных алгоритмов. Охватывает неинтерактивное профилирование с помощью nsys/ncu, отладку с помощью cuda-gdb/compute-sanitizer, бинарную проверку с помощью cuobjdump и рабочие процессы анализа производительности. Триггеры по CUDA, программированию графического процессора, оптимизации ядра, nsys, ncu, cuda-gdb, вычислительному дезинфицирующему средству, PTX, профилированию графического процессора, параллельной производительности.
