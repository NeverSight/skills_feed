تطوير نواة CUDA وتصحيح الأخطاء وتحسين الأداء لـ Claude Code. يُستخدم عند الكتابة أو تصحيح الأخطاء أو تحسين كود CUDA أو نواة وحدة معالجة الرسومات أو الخوارزميات المتوازية. يغطي ملفات التعريف غير التفاعلية باستخدام nsys/ncu، وتصحيح الأخطاء باستخدام cuda-gdb/compute-sanitizer، والفحص الثنائي باستخدام cuobjdump، وسير عمل تحليل الأداء. المشغلات على CUDA، برمجة GPU، تحسين kernel، nsys، ncu، cuda-gdb، المطهر الحسابي، PTX، ملفات تعريف GPU، الأداء المتوازي.
