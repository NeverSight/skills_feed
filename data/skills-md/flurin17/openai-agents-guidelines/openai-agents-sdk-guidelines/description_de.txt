Implementierungsrichtlinien für das OpenAI Agents SDK (Python). Wird zum Erstellen, Überprüfen oder Beheben von Agent-Workflows verwendet, einschließlich Tools, Übergaben, Leitplanken, Kontextverwaltung, Streaming, MCP-Integration und Runner-Konfiguration. Wird bei Anfragen wie „Agent erstellen“, „Übergaben hinzufügen“, „Leitlinien einrichten“, „Tools konfigurieren“ oder „Streaming implementieren“ ausgelöst.
