Подробное руководство по шлюзу событий Hookdeck для получения, маршрутизации и доставки веб-перехватчиков и событий. Охватывает настройку, подключения, аутентификацию, локальную разработку, мониторинг и автоматизацию API. Используйте при получении веб-перехватчиков, настройке конечных точек веб-перехватчиков, локальном тестировании веб-перехватчиков, настройке ретрансляции веб-перехватчиков или очереди событий, маршрутизации событий, повторной попытке веб-перехватчика, мониторинге веб-перехватчиков, сторонней маршрутизации, асинхронных API или разработке локальных веб-перехватчиков. Для веб-хуков провайдеров (Stripe, Shopify, Chargebee, GitHub и т. д.) используйте вместе с соответствующим навыком из Hookdeck/webhook-skills; не анализируйте только JSON — используйте проверку SDK поставщика и построение событий (например, StripestructEvent).
