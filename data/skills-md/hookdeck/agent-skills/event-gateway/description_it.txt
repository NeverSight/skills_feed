Guida completa all'Hookdeck Event Gateway per ricevere, instradare e distribuire webhook ed eventi. Copre configurazione, connessioni, autenticazione, sviluppo locale, monitoraggio e automazione delle API. Da utilizzare quando si ricevono webhook, si configurano endpoint webhook, si testano webhook localmente, si configura l'inoltro webhook o la coda di eventi, il routing di eventi, i tentativi di webhook, il monitoraggio dei webhook, il routing di terze parti, le API asincrone o lo sviluppo di webhook locali. Per i webhook dei provider (Stripe, Shopify, Chargebee, GitHub, ecc.), utilizzare insieme alla competenza corrispondente da hookdeck/webhook-skills; non analizzare solo JSON: utilizza la verifica dell'SDK del provider e la costruzione di eventi (ad esempio Stripe buildEvent).
