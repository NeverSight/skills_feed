دليل شامل لبوابة أحداث Hookdeck لتلقي خطافات الويب والأحداث وتوجيهها وتسليمها. يغطي الإعداد والاتصالات والمصادقة والتطوير المحلي والمراقبة وأتمتة واجهة برمجة التطبيقات. يُستخدم عند تلقي خطافات الويب، أو إعداد نقاط نهاية خطاف الويب، أو اختبار خطافات الويب محليًا، أو تكوين ترحيل خطاف الويب أو قائمة انتظار الأحداث، أو توجيه الأحداث، أو إعادة محاولة خطاف الويب، أو مراقبة خطاف الويب، أو توجيه الطرف الثالث، أو واجهات برمجة التطبيقات غير المتزامنة، أو تطوير خطاف الويب المحلي. بالنسبة إلى خطافات الويب للموفر (Stripe وShopify وCharbee وGitHub وما إلى ذلك)، استخدمها مع مهارة المطابقة من مهارات الخطاف/الخطاف الإلكتروني؛ لا تقم بتحليل JSON فقط - استخدم التحقق من SDK للموفر وإنشاء الحدث (على سبيل المثال، Stripe buildEvent).
