Guía completa de Hookdeck Event Gateway para recibir, enrutar y entregar webhooks y eventos. Cubre configuración, conexiones, autenticación, desarrollo local, monitoreo y automatización de API. Úselo al recibir webhooks, configurar puntos finales de webhook, probar webhooks localmente, configurar la retransmisión de webhook o la cola de eventos, el enrutamiento de eventos, el reintento de webhook, la supervisión de webhook, el enrutamiento de terceros, las API asincrónicas o el desarrollo de webhook local. Para webhooks de proveedores (Stripe, Shopify, Chargebee, GitHub, etc.), utilícelos junto con la habilidad de coincidencia de hookdeck/webhook-skills; no solo analice JSON: utilice la verificación del SDK del proveedor y la construcción de eventos (por ejemplo, Stripe constructEvent).
