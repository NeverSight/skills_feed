Webhook とイベントを受信、ルーティング、配信するための Hookdeck Event Gateway の包括的なガイド。セットアップ、接続、認証、ローカル開発、モニタリング、API 自動化について説明します。 Webhook の受信、Webhook エンドポイントのセットアップ、Webhook のローカルでのテスト、Webhook リレーまたはイベント キューの構成、イベント ルーティング、Webhook の再試行、Webhook モニタリング、サードパーティのルーティング、非同期 API、またはローカル Webhook 開発の場合に使用します。プロバイダー Webhook (Stripe、Shopify、Chargebee、GitHub など) の場合は、hookdeck/webhook-skills の一致するスキルと組み合わせて使用​​します。 JSON を解析するだけではなく、プロバイダー SDK 検証とイベント構築 (StripeconstructEvent など) を使用します。
