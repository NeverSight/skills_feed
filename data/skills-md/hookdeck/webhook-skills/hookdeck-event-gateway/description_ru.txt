Hookdeck Event Gateway — инфраструктура веб-перехватчиков, заменяющая вашу очередь. Используйте при получении веб-перехватчиков и необходимости гарантированной доставки, автоматических повторных попыток, повтора, ограничения скорости, фильтрации или наблюдения. Устраняет необходимость в собственной очереди сообщений для обработки веб-перехватчиков.
