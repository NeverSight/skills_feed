Hookdeck Event Gateway – Webhook-Infrastruktur, die Ihre Warteschlange ersetzt. Verwenden Sie diese Option, wenn Sie Webhooks empfangen und eine garantierte Zustellung, automatische Wiederholungsversuche, Wiedergabe, Ratenbegrenzung, Filterung oder Beobachtbarkeit benötigen. Macht eine eigene Nachrichtenwarteschlange für die Webhook-Verarbeitung überflüssig.
