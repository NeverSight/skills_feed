Helfen Sie Benutzern, hochwertige Fähigkeiten zu entwickeln, indem Sie bewährte Methoden von Fachexperten entdecken und integrieren. Verwenden Sie diesen Skill VOR dem Skill-Creator, wenn Benutzer einen neuen Skill erstellen möchten. Er verbessert den Skill-Creator, indem zunächst Experten-Frameworks und Best Practices identifiziert werden, die integriert werden sollen. Wird bei Anfragen wie „Helfen Sie mir, einen Skill für X zu erstellen“ oder „Ich möchte einen Skill erstellen, der Y unterstützt“ ausgelöst. Diese Fertigkeit leitet die Auswahl der Methodik und übergibt sie dann an den Fertigkeitsersteller für die eigentliche Fertigkeitsgenerierung.
