Идиомы управления потоком управления Go из книги «Эффективное Go». Охватывает if с инициализацией, опуская else для ранних возвратов, для форм цикла, диапазона, переключения без провала, переключения типа и шаблонов пустых идентификаторов. Используйте при написании условий, циклов или операторов переключения в Go.
