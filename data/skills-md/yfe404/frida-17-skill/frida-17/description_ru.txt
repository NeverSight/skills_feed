Средство проверки и исправления совместимости API JavaScript с Frida 17. Используйте при написании, проверке или исправлении сценариев Frida, особенно при переходе со старых версий Frida. Обнаруживает устаревшие API, удаленные в Frida 17 (май 2025 г.), и обеспечивает правильные замены. Охватывает API модулей, памяти, процессов и распространенные конфликты имен.
