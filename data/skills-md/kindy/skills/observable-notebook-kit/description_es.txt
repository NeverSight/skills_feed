Cree, edite y cree cuadernos observables utilizando Notebook Kit. Úselo cuando trabaje con archivos de cuaderno .html, genere sitios estáticos a partir de cuadernos, consulte bases de datos desde cuadernos o utilice cargadores de datos (Node.js/Python/R) en cuadernos. Cubre el formato de archivo del cuaderno, los tipos de celdas, los comandos CLI, los conectores de bases de datos y la API de JavaScript.
