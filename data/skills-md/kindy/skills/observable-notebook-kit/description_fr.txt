Créez, modifiez et créez des blocs-notes observables à l'aide de Notebook Kit. À utiliser lorsque vous travaillez avec des fichiers de notebook .html, générez des sites statiques à partir de notebooks, interrogez des bases de données à partir de notebooks ou utilisez des chargeurs de données (Node.js/Python/R) dans des notebooks. Couvre le format de fichier de bloc-notes, les types de cellules, les commandes CLI, les connecteurs de base de données et l'API JavaScript.
