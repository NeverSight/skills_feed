Создавайте, редактируйте и создавайте Observable Notebooks с помощью Notebook Kit. Используйте при работе с файлами блокнотов .html, создании статических сайтов из блокнотов, запросе баз данных из блокнотов или использовании загрузчиков данных (Node.js/Python/R) в блокнотах. Охватывает формат файлов записной книжки, типы ячеек, команды CLI, соединители баз данных и API JavaScript.
