モデル駆動型アプリの Dataverse Web リソースを作成、デプロイ、または管理するときに使用します。 JavaScript フォーム スクリプト (OnLoad、OnSave、OnChange イベント)、HTML ダッシュボード ページ、CSS スタイル、画像リソース、ナビゲーション/サイド ペイン、リボン/コマンド バーのカスタマイズ、ビジネス プロセス フロー クライアント API、および Web API を介したデプロイメントをカバーします。トリガー対象: 「Web リソース」、「JavaScript フォーム」、「フォーム スクリプト」、「HTML ダッシュボード」、「リボン コマンド」、「onload イベント」、「onchange イベント」、「onsave イベント」、「formContext」、「Xrm.WebApi」、「Web リソース デプロイメント」、「ダッシュボード ページ」、「フォーム イベント ハンドラー」、「サイド ペイン」、「コマンド バー」、「リボン」、「モーダル ダイアログ」、「ナビゲーション」、 「Xrm.App」、「Xrm.Navigation」、「業務プロセスフロー」、「bpf」、「ステージ変更」。
