Válvula de liberación de presión cognitiva para la ejecución del agente. Úselo cuando el estado de razonamiento interno no se puede llevar adelante de manera productiva: suposiciones contradictorias, alta presión de razonamiento o riesgo de errores compuestos. Invocar para abandonar explícitamente el estado interno improductivo. Se trata de un fregadero unidireccional, que no bloquea y deliberadamente poco fiable. El agente debe comportarse de manera idéntica ya sea que la descarga se realice correctamente o falle.
