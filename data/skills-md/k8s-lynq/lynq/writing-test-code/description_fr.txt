Lignes directrices pour l'écriture de code de test de style BDD à l'aide du framework Ginkgo/Gomega dans Go. À utiliser lors de l'écriture de tests pour les opérateurs, les contrôleurs ou les services Kubernetes. Se concentre sur le développement axé sur le comportement avec des modèles Given-When-Then et des tests basés sur des tables.
