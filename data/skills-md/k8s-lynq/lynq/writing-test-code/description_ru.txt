Рекомендации по написанию тестового кода в стиле BDD с использованием инфраструктуры Ginkgo/Gomega на Go. Используйте при написании тестов для операторов Kubernetes, контроллеров или сервисов Go. Основное внимание уделяется разработке на основе поведения с использованием шаблонов «Дано-Когда-То» и тестов на основе таблиц.
