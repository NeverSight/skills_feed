À utiliser lorsque l'utilisateur demande explicitement une COMPÉTENCE pour créer un arbre de travail. Si l'utilisateur ne mentionne pas la « compétence » ou ne demande pas explicitement l'invocation de la compétence, ne déclenchez PAS cette opération. À utiliser uniquement lorsque l'utilisateur dit des choses comme "utiliser une compétence pour créer un arbre de travail" ou "invoquer la compétence d'arbre de travail". Crée des arbres de travail Git isolés avec une configuration exécutée en parallèle.
