قم بإنشاء وإعادة تصميم تصميمات ويب HTML5/JavaScript من المواصفات أو الأوصاف. يُنشئ تصميمات ويب كاملة وسهلة الوصول وسريعة الاستجابة باستخدام أطر عمل حديثة. يتحقق تلقائيًا من التصميمات باستخدام Playwright MCP لاختبار إمكانية الوصول والوظائف. استخدم هذه المهارة عندما يطلب المستخدمون إنشاء تصميمات ويب أو نماذج بالحجم الطبيعي أو صفحات مقصودة أو تطبيقات ويب أو إعادة تصميم تصميمات HTML/CSS/JS الحالية.
