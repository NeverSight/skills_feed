Помогите пользователям создавать расширения Sindri. Используйте при создании новых расширений, понимании структуры файла Extension.yaml, проверке расширений на соответствие схемам или изучении методов установки расширений (mise, apt, двоичный, npm, скрипт, гибрид). Включает НОВУЮ систему возможностей для инициализации проекта, аутентификации (многометодный ключ API + аутентификация CLI), перехватчиков жизненного цикла и интеграции MCP. Помогает в разработке расширений, обновлении реестра и назначении категорий.
