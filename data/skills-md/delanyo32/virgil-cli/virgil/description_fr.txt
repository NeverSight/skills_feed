Analysez, explorez et comprenez des bases de code inconnues à l'aide de virgil-cli. Couvre l'intégralité du flux de travail : analyse de la source dans Parquet, interrogation avec les commandes intégrées, navigation dans les dépendances et les appelants et exécution de DuckDB SQL brut. À utiliser lorsqu'on vous le demande pour analyser une base de code, comprendre l'architecture, rechercher des symboles, tracer des dépendances, intégrer un projet, enquêter sur des bogues ou mapper la surface de l'API de n'importe quelle base de code TypeScript/JavaScript/C/C++/C#/Rust/Python/Go/Java/PHP.
