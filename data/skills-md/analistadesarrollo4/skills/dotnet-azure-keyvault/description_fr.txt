Intégration d'Azure Key Vault avec les applications Managed Identity pour .NET. À utiliser lorsque : (1) implémentation d'Azure Key Vault comme source de configuration, (2) configuration de l'authentification d'identité managée avec DefaultAzureCredential, (3) création de la méthode d'extension AddAzureKeyVault dans l'infrastructure, (4) configuration des secrets et des conventions de dénomination Key Vault, (5) questions sur l'authentification de développement local, (6) implémentation de KeyVaultSecretManager personnalisé ou (7) dépannage de l'accès et des autorisations Key Vault.
