Integración de Azure Key Vault con Managed Identity para aplicaciones .NET. Úselo cuando: (1) Implemente Azure Key Vault como origen de configuración, (2) Configure la autenticación de identidad administrada con DefaultAzureCredential, (3) Cree el método de extensión AddAzureKeyVault en infraestructura, (4) Configure secretos de Key Vault y convenciones de nomenclatura, (5) Preguntas sobre la autenticación de desarrollo local, (6) Implemente KeyVaultSecretManager personalizado o (7) Solucione problemas de acceso y permisos de Key Vault.
