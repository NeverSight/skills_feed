Integrazione di Azure Key Vault con identità gestita per applicazioni .NET. Utilizzare quando: (1) implementare Azure Key Vault come origine di configurazione, (2) configurare l'autenticazione dell'identità gestita con DefaultAzureCredential, (3) creare il metodo di estensione AddAzureKeyVault nell'infrastruttura, (4) configurare i segreti di Key Vault e le convenzioni di denominazione, (5) domande sull'autenticazione di sviluppo locale, (6) implementare KeyVaultSecretManager personalizzato o (7) risolvere i problemi di accesso e autorizzazioni di Key Vault.
