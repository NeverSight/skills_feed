Интеграция Azure Key Vault с приложениями Managed Identity для .NET. Используйте в следующих случаях: (1) реализация Azure Key Vault в качестве источника конфигурации, (2) настройка проверки подлинности управляемых удостоверений с помощью DefaultAzureCredential, (3) создание метода расширения AddAzureKeyVault в инфраструктуре, (4) настройка секретов Key Vault и соглашений об именах, (5) вопросы об аутентификации локальной разработки, (6) реализация пользовательского KeyVaultSecretManager или (7) устранение неполадок доступа и разрешений Key Vault.
