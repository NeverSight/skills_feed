Azure Key Vault-Integration mit Managed Identity für .NET-Anwendungen. Verwenden Sie Folgendes: (1) Implementieren von Azure Key Vault als Konfigurationsquelle, (2) Einrichten der verwalteten Identitätsauthentifizierung mit DefaultAzureCredential, (3) Erstellen der AddAzureKeyVault-Erweiterungsmethode in der Infrastruktur, (4) Konfigurieren von Key Vault-Geheimnissen und Namenskonventionen, (5) Fragen zur lokalen Entwicklungsauthentifizierung, (6) Implementieren von benutzerdefiniertem KeyVaultSecretManager oder (7) Fehlerbehebung bei Key Vault-Zugriff und -Berechtigungen.
