تكامل Azure Key Vault مع الهوية المُدارة لتطبيقات .NET. يُستخدم عندما: (1) تنفيذ Azure Key Vault كمصدر تكوين، (2) إعداد مصادقة الهوية المُدارة باستخدام DefaultAzureCredential، (3) إنشاء طريقة ملحق AddAzureKeyVault في البنية التحتية، (4) تكوين أسرار Key Vault واصطلاحات التسمية، (5) أسئلة حول مصادقة التطوير المحلي، (6) تنفيذ KeyVaultSecretManager المخصص، أو (7) استكشاف أخطاء الوصول إلى Key Vault وإصلاحها والأذونات.
