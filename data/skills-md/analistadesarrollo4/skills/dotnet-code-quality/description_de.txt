Codequalitätsstandards, Optimierung und SonarQube-Konformität für .NET-Anwendungen. Verwenden Sie, wenn: (1) Code auf Komplexität überprüft oder umgestaltet wird (zyklomatische Komplexität > 10–15), (2) EF Core-Abfragen mit AsNoTracking für schreibgeschützte Vorgänge optimieren, (3) Caching-Strategien implementieren (Speichercache, Redis), (4) SonarQube/SonarCloud-Regelkonformität sicherstellen, (5) SOLID-Prinzipien und Clean-Code-Praktiken anwenden, (6) Fragen zur Leistungsoptimierung stellen oder (7) Code-Smells und Wartbarkeit bewerten Probleme.
