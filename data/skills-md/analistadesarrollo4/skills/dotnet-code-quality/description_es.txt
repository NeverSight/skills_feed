Estándares de calidad de código, optimización y cumplimiento de SonarQube para aplicaciones .NET. Úselo cuando: (1) Revisar o refactorizar código para determinar su complejidad (complejidad ciclomática > 10-15), (2) Optimizar consultas de EF Core con AsNoTracking para operaciones de solo lectura, (3) Implementar estrategias de almacenamiento en caché (memoria caché, Redis), (4) Garantizar el cumplimiento de las reglas de SonarQube/SonarCloud, (5) Aplicar principios SOLID y prácticas de código limpio, (6) Preguntas sobre optimización del rendimiento o (7) Evaluar olores de código y problemas de mantenimiento.
