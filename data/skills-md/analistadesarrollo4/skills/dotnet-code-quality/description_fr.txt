Normes de qualité du code, optimisation et conformité SonarQube pour les applications .NET. À utiliser lorsque : (1) révision ou refactorisation du code pour la complexité (complexité cyclomatique > 10-15), (2) optimisation des requêtes EF Core avec AsNoTracking pour les opérations en lecture seule, (3) mise en œuvre de stratégies de mise en cache (cache mémoire, Redis), (4) garantie de la conformité aux règles SonarQube/SonarCloud, (5) application des principes SOLID et des pratiques de code propre, (6) questions d'optimisation des performances, ou (7) évaluation des odeurs et de la maintenabilité du code problèmes.
