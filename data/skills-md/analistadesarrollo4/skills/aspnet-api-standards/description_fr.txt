Implémentation de l'API Web ASP.NET Core : contrôleurs propres avec CQRS, gestion globale des erreurs, validation du modèle, Swagger/OpenAPI, gestion des versions de l'API, sécurité (CORS, authentification), pipeline middleware et modèles de performances. À utiliser lors de la création ou de la modification de contrôleurs, de filtres, de middleware, de Program.cs ou de points de terminaison d'API.
