قم بتصميم وبناء واجهات سطر الأوامر للوكيل أولاً باستخدام استجابات HATEOAS JSON ومخرجات حماية السياق وأشجار الأوامر ذاتية التوثيق. يُستخدم عند إنشاء أدوات واجهة سطر الأوامر الجديدة، أو إضافة أوامر إلى واجهات سطر الأوامر الموجودة (joelclaw، أو slog، أو igs)، أو مراجعة تصميم واجهة سطر الأوامر لسهولة التعامل مع الوكيل. يتم تشغيلها عند "إنشاء واجهة سطر الأوامر" أو "إضافة أمر" أو "تصميم واجهة سطر الأوامر" أو "الإخراج المناسب للوكيل" أو أي مهمة تتضمن إنشاء أداة سطر أوامر.
