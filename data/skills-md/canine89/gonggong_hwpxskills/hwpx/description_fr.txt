Compétence pour créer, lire, modifier et remplacer des documents HWPX (fichiers .hwpx). Chaque fois que des mots-clés tels que « document Hangul », « hwpx », « HWPX », « fichier Hangul », « créer un fichier .hwpx », « créer un document HWP », « rapport », « lettre officielle », « brouillon » ou « écrire en Hangul » apparaissent, cette compétence doit être utilisée. Il gère le format HWPX de Hancom (basé sur KS X 6101/OWPML, structure ZIP+XML) via la bibliothèque python-hwpx. Lorsqu'un modèle de rapport est nécessaire, utilisez les modèles de référence dans le dossier assets/. Pour les documents Word (.docx) classiques, utilisez la compétence docx.
