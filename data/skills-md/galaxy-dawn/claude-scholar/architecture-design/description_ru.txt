Используйте ТОЛЬКО при создании НОВЫХ регистрируемых компонентов в проектах ML, для которых требуются шаблоны Factory/Registry.

✅ ИСПОЛЬЗУЙТЕ, когда:
- Создание нового класса набора данных (требуется @register_dataset)
- Создание нового класса модели (требуется @register_model)
— Создание нового каталога модулей с помощью фабрики __init__.py.
- Инициализация новой структуры проекта ML с нуля.
- Добавление новых типов компонентов (Augmentation, CollateFunction, Metrics)

❌ НЕ ИСПОЛЬЗУЙТЕ, если:
- Изменение существующих функций или методов.
- Исправление ошибок в существующем коде
- Добавление вспомогательных функций или утилит.
- Рефакторинг без добавления новых регистрируемых компонентов.
- Простые изменения кода в одном файле
- Изменение файлов конфигурации.
- Чтение или понимание существующего кода.

Ключевой индикатор: требуется ли для задачи декоратор @register_* или шаблон Factory? Если нет, пропустите этот навык.
