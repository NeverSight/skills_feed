Этот навык следует использовать, когда пользователь просит «применить улучшения навыков», «обновить навык из плана», «выполнить план улучшения», «исправить проблемы с навыками», «внедрить рекомендации по навыкам» или упоминает о применении улучшений из отчетов о проверке качества. Считывает файлы Improve-plan-{name}.md, созданные с помощью Skill-Quality-Reviewer, и интеллектуально объединяет и выполняет предложенные изменения для улучшения качества Claude Skills.
