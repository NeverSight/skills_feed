مهارات المعلم الفردي لحل المشكلات الرياضية وإنشاء مستندات شرح بتنسيق HTML ومقاطع فيديو الرسوم المتحركة المنيم.
سير العمل الأساسي: تحليل الرياضيات ← تصور HTML ← نص القصة المصورة ← صوت تحويل النص إلى كلام ← تحديث التحقق من الصحة ← السقالات ← كود مانيم ← التحقق من صحة العرض
شروط التنفيذ: يقوم الطالب بلصق صورة المسألة الرياضية، ويحتاج إلى فيديو تعليمي، ويحتاج إلى مواد شرح بتنسيق HTML
