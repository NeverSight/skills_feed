Навыки индивидуального обучения для решения математических задач, создания пояснительных документов в формате HTML и анимационных видеороликов Маним с озвучкой.
Основной рабочий процесс: математический анализ → визуализация HTML → сценарий раскадровки → аудио TTS → обновление проверки → строительные леса → код Manim → проверка рендеринга.
Условия срабатывания: учащийся вставляет изображение математической задачи, нуждается в обучающем видео, нуждается в пояснительных материалах в формате HTML.
