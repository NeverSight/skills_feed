API de la plateforme fal.ai pour la gestion des modèles, la tarification, le suivi de l'utilisation et l'estimation des coûts. À utiliser lorsque l'utilisateur demande « afficher les prix », « vérifier l'utilisation », « estimer le coût », « configuration fal », « ajouter une clé API » ou des tâches de gestion de plate-forme.
