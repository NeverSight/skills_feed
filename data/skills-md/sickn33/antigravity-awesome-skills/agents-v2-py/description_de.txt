Erstellen Sie containerbasierte Foundry-Agents mithilfe des Azure AI Projects SDK mit ImageBasedHostedAgentDefinition.
Verwenden Sie diese Option, wenn Sie gehostete Agents erstellen, die benutzerdefinierten Code in Azure AI Foundry mit Ihren eigenen Container-Images ausführen.
Auslöser: „ImageBasedHostedAgentDefinition“, „Hosted Agent“, „Container Agent“, „Foundry Agent“,
„create_version“, „ProtocolVersionRecord“, „AgentProtocol.RESPONSES“, „Custom Agent Image“.
