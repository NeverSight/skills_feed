ImageBasedHostedAgentDefinition を備えた Azure AI Projects SDK を使用して、コンテナー ベースの Foundry Agent を構築します。
独自のコンテナー イメージを使用して Azure AI Foundry でカスタム コードを実行するホスト型エージェントを作成する場合に使用します。
