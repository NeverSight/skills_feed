Создавайте агенты Foundry на основе контейнеров с помощью пакета SDK Azure AI Projects с ImageBasedHostedAgentDefinition.
Используйте при создании размещенных агентов, выполняющих собственный код в Azure AI Foundry с вашими собственными образами контейнеров.
Триггеры: «ImageBasedHostedAgentDefinition», «размещенный агент», «контейнерный агент», «Foundry Agent»,
«create_version», «ProtocolVersionRecord», «AgentProtocol.RESPONSES», «пользовательский образ агента».
