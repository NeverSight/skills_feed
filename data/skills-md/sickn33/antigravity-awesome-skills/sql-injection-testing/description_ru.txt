Этот навык следует использовать, когда пользователь просит «проверить уязвимости внедрения SQL», «выполнить SQLi-атаки», «обойти аутентификацию с помощью внедрения SQL», «извлечь информацию из базы данных посредством внедрения», «обнаружить недостатки внедрения SQL» или «использовать уязвимости запросов к базе данных». Он предоставляет комплексные методы для выявления, использования и понимания векторов атак с помощью SQL-инъекций в различных системах баз данных.
