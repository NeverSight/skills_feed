Руководства и лучшие практики по работе с Neon Serverless Postgres. Охватывает начало работы, локальную разработку с помощью Neon, выбор метода подключения, функции Neon, аутентификацию (@neondatabase/auth), API данных в стиле PostgREST (@neondatabase/neon-js), Neon CLI и API/SDK платформы Neon. Используйте для любых вопросов, связанных с неоном.
