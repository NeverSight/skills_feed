Créez des agents Azure AI Foundry à l’aide du SDK Python Microsoft Agent Framework (agent-framework-azure-ai). À utiliser lors de la création d'agents persistants avec AzureAIAgentsProvider, à l'aide d'outils hébergés (interpréteur de code, recherche de fichiers, recherche Web), à l'intégration de serveurs MCP, à la gestion des fils de conversation ou à la mise en œuvre de réponses en continu. Couvre les outils fonctionnels, les sorties structurées et les agents multi-outils.
