運用グレードのパターンに従って、Python/FastAPI を使用して Azure Cosmos DB NoSQL サービスを構築します。デュアル認証 (DefaultAzureCredential + エミュレーター) を使用したデータベース クライアント セットアップ、CRUD 操作を使用したサービス レイヤー クラス、パーティション キー戦略、パラメーター化されたクエリ、または Cosmos の TDD パターンを実装する場合に使用します。 「Cosmos DB」、「NoSQL データベース」、「ドキュメント ストア」、「永続性の追加」、「データベース サービス レイヤー」、「Python Cosmos SDK」などのフレーズでトリガーされます。
