Experte für die Gestaltung effektiver Eingabeaufforderungen für LLM-basierte Anwendungen. Beherrscht die Struktur von Eingabeaufforderungen, das Kontextmanagement, die Ausgabeformatierung und die Auswertung von Eingabeaufforderungen. Verwendung bei: Prompt Engineering, System Prompt, Fow-Shot, Chain of Thought, Prompt Design.
