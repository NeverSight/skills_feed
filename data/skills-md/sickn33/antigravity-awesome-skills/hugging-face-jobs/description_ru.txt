Этот навык следует использовать, когда пользователи хотят запускать любую рабочую нагрузку в инфраструктуре Hugging Face Jobs. Охватывает UV-скрипты, задания на основе Docker, выбор оборудования, оценку стоимости, аутентификацию с помощью токенов, управление секретами, настройку тайм-аута и сохранение результатов. Разработан для вычислительных задач общего назначения, включая обработку данных, логические выводы, эксперименты, пакетные задания и любые задачи на основе Python. Следует вызывать для задач, связанных с облачными вычислениями, рабочими нагрузками графического процессора или когда пользователи упоминают выполнение заданий в инфраструктуре Hugging Face без локальной настройки.
