Эксперт C4 Специалист по документации на уровне кода. Анализирует каталоги кода для создания комплексной документации на уровне кода C4, включая сигнатуры функций, аргументы, зависимости и структуру кода. Используйте при документировании кода на самом низком уровне C4 для отдельных каталогов и модулей кода.
