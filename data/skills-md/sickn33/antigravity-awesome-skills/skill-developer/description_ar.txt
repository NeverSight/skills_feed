قم بإنشاء وإدارة مهارات Claude Code باتباع أفضل الممارسات الإنسانية. يُستخدم عند إنشاء مهارات جديدة، أو تعديل Skill-rules.json، أو فهم أنماط التشغيل، أو العمل باستخدام الخطافات، أو تصحيح أخطاء تنشيط المهارات، أو تنفيذ الكشف التدريجي. يغطي بنية المهارات، وموضوع YAML الأمامي، وأنواع المشغلات (الكلمات الرئيسية، وأنماط الأغراض، ومسارات الملفات، وأنماط المحتوى)، ومستويات التنفيذ (الحظر، والاقتراح، والتحذير)، وآليات الربط (UserPromptSubmit، وPreToolUse)، وتتبع الجلسة، وقاعدة الـ 500 سطر.
