انشر التطبيقات الموجودة في حاويات إلى Azure Container Apps باستخدام Azure Developer CLI (azd). يُستخدم عند إعداد مشاريع azd، أو كتابة تكوين azure.yaml، أو إنشاء بنية أساسية لـ Bicep لتطبيقات الحاوية، أو تكوين عمليات إنشاء عن بعد باستخدام ACR، أو تنفيذ عمليات نشر مستقلة، أو إدارة متغيرات البيئة عبر local/.azure/Bicep، أو استكشاف أخطاء فشل azd up وإصلاحها. تشغيل طلبات تكوين azd، ونشر تطبيقات الحاوية، وعمليات نشر الخدمات المتعددة، والبنية التحتية كرمز مع Bicep.
