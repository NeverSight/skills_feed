Déployez des applications conteneurisées sur Azure Container Apps à l’aide d’Azure Developer CLI (azd). À utiliser lors de la configuration de projets azd, de l'écriture de la configuration azure.yaml, de la création d'une infrastructure Bicep pour les applications de conteneurs, de la configuration de builds distants avec ACR, de la mise en œuvre de déploiements idempotents, de la gestion des variables d'environnement sur local/.azure/Bicep ou du dépannage des échecs azd up. Se déclenche sur les demandes de configuration azd, de déploiement de Container Apps, de déploiements multiservices et d'infrastructure en tant que code avec Bicep.
