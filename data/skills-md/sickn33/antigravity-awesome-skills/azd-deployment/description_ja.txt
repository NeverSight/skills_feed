Azure Developer CLI (azd) を使用して、コンテナー化されたアプリケーションを Azure Container Apps にデプロイします。 azd プロジェクトのセットアップ、azure.yaml 構成の作成、コンテナ アプリ用の Bicep インフラストラクチャの作成、ACR を使用したリモート ビルドの構成、冪等デプロイメントの実装、local/.azure/Bicep 全体にわたる環境変数の管理、または azd up エラーのトラブルシューティングを行うときに使用します。 azd 構成、コンテナー アプリのデプロイ、マルチサービスのデプロイ、Bicep を使用したコードとしてのインフラストラクチャのリクエストをトリガーします。
