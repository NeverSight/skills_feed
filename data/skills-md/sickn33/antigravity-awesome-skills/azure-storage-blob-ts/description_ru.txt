Пакет SDK JavaScript/TypeScript для хранилища BLOB-объектов Azure (@azure/storage-blob) для операций с большими двоичными объектами. Используйте для отправки, скачивания, перечисления и управления большими двоичными объектами и контейнерами. Поддерживает блочные BLOB-объекты, дополнительные BLOB-объекты, страничные BLOB-объекты, токены SAS и потоковую передачу. Триггеры: «хранилище BLOB-объектов», «@azure/storage-blob», «BlobServiceClient», «ContainerClient», «загрузить BLOB-объект», «загрузить BLOB-объект», «токен SAS», «блокировать BLOB-объект».
