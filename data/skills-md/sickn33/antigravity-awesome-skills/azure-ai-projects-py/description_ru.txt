Создавайте приложения искусственного интеллекта с помощью пакета SDK Azure AI Projects Python (azure-ai-projects). Используйте при работе с клиентами проекта Foundry, создании агентов с поддержкой версий с помощью PromptAgentDefinition, выполнении оценок, управлении подключениями/развертываниями/наборами данных/индексами или использовании клиентов, совместимых с OpenAI. Это пакет Foundry SDK высокого уровня — для операций агента низкого уровня используйте навык azure-ai-agents-python.
