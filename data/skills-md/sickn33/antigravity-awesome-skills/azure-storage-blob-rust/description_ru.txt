Пакет SDK хранилища BLOB-объектов Azure для Rust. Используйте для отправки, скачивания и управления большими двоичными объектами и контейнерами.
Триггеры: «ржавеет хранилище BLOB-объектов», «ржавеет BlobClient», «растрастает загрузку BLOB-объектов», «растрастает загрузку BLOB-объектов», «ржавеет контейнер».
