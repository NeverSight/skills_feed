Создавайте приложения потоковой передачи событий с помощью пакета SDK Центров событий Azure для JavaScript (@azure/event-hubs). Используйте при реализации высокопроизводительного приема событий, аналитики в реальном времени, телеметрии Интернета вещей или архитектур, управляемых событиями, с разделенными потребителями.
