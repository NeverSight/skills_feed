SDK persistente de los agentes Azure AI para .NET. SDK de bajo nivel para crear y administrar agentes de IA con subprocesos, mensajes, ejecuciones y herramientas. Úselo para agente CRUD, hilos de conversación, transmisión de respuestas, llamada de funciones, búsqueda de archivos e intérprete de código. Activadores: "PersistentAgentsClient", "agentes persistentes", "hilos de agentes", "ejecuciones de agentes", "agentes de transmisión", "agentes de llamada de funciones .NET".
