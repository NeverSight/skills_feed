Постоянный пакет SDK для агентов Azure AI для .NET. Низкоуровневый SDK для создания и управления агентами ИИ с помощью потоков, сообщений, запусков и инструментов. Используйте для агента CRUD, потоков разговоров, потоковой передачи ответов, вызова функций, поиска файлов и интерпретатора кода. Триггеры: «PersistentAgentsClient», «постоянные агенты», «потоки агентов», «запуск агентов», «потоковые агенты», «агенты вызова функций .NET».
