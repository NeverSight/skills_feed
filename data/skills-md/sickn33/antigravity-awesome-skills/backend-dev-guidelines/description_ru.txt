Убежденные стандарты разработки серверной части для микросервисов Node.js + Express + TypeScript. Охватывает многоуровневую архитектуру, шаблон BaseController, внедрение зависимостей, репозитории Prisma, проверку Zod, unifiedConfig, отслеживание ошибок Sentry, асинхронную безопасность и дисциплину тестирования.
