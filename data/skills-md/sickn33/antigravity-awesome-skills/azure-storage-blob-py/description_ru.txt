Пакет SDK хранилища BLOB-объектов Azure для Python. Используйте для отправки, скачивания, вывода списка больших двоичных объектов, управления контейнерами и жизненным циклом больших двоичных объектов.
Триггеры: «хранилище BLOB-объектов», «BlobServiceClient», «ContainerClient», «BlobClient», «загрузить BLOB-объект», «загрузить BLOB-объект».
