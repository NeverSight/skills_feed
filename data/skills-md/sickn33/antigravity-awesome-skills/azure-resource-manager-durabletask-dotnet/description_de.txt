Azure Resource Manager SDK für den dauerhaften Taskplaner in .NET. Verwendung für MANAGEMENT PLANE-Vorgänge: Erstellen/Verwalten dauerhafter Taskplaner, Task Hubs und Aufbewahrungsrichtlinien über Azure Resource Manager. Auslöser: „Durable Task Scheduler“, „Scheduler erstellen“, „Task Hub“, „DurableTaskSchedulerResource“, „Durable Task bereitstellen“, „Orchestrierungsplaner“.
