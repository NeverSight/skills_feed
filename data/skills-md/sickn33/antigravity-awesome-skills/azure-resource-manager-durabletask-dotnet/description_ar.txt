Azure Resource Manager SDK لجدولة المهام الدائمة في .NET. يُستخدم لعمليات خطة الإدارة: إنشاء/إدارة برامج جدولة المهام الدائمة ومراكز المهام وسياسات الاحتفاظ عبر Azure Resource Manager. المشغلات: "جدولة المهام الدائمة"، "إنشاء جدولة"، "مركز المهام"، "DurableTaskSchedulerResource"، "توفير مهمة دائمة"، "جدولة التنسيق".
