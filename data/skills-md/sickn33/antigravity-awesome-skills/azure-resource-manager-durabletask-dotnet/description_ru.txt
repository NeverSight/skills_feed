Пакет SDK Azure Resource Manager для устойчивого планировщика задач в .NET. Используйте для операций НА ПЛАНЕ УПРАВЛЕНИЯ: создание и управление устойчивыми планировщиками задач, концентраторами задач и политиками хранения с помощью Azure Resource Manager. Триггеры: «Планировщик устойчивых задач», «Создать планировщик», «концентратор задач», «DurableTaskSchedulerResource», «предоставление устойчивых задач», «планировщик оркестрации».
