Пакет SDK Azure Cosmos DB для JavaScript/TypeScript (@azure/cosmos) для операций плоскости данных. Используйте для операций CRUD над документами, запросами, массовыми операциями и управлением контейнерами. Триггеры: «Cosmos DB», «@azure/cosmos», «CosmosClient», «документ CRUD», «запросы NoSQL», «массовые операции», «ключ раздела», «container.items».
