Diese Fähigkeit sollte verwendet werden, wenn der Benutzer „Cloud-Penetrationstests durchführen“, „die Sicherheit von Azure, AWS oder GCP bewerten“, „Cloud-Ressourcen auflisten“, „Cloud-Fehlkonfigurationen ausnutzen“, „O365-Sicherheit testen“, „Geheimnisse aus Cloud-Umgebungen extrahieren“ oder „Cloud-Infrastruktur prüfen“ möchte. Es bietet umfassende Techniken zur Sicherheitsbewertung für alle wichtigen Cloud-Plattformen.
