Пакет SDK Azure OpenAI для .NET. Клиентская библиотека для служб Azure OpenAI и OpenAI. Используйте для завершения чата, встраивания, создания изображений, транскрипции звука и помощников. Триггеры: «Azure OpenAI», «AzureOpenAIClient», «ChatClient», «завершения чата .NET», «GPT-4», «внедрения», «DALL-E», «Whisper», «OpenAI .NET».
