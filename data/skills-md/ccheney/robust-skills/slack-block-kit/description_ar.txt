يتم التطبيق بشكل استباقي عند إنشاء حمولات Slack API مع الكتل، أو مكالمات chat.postMessage ذات المحتوى المنظم، أو مكالمات view.open/views.publish. المشغلات في Block Kit، كتل Slack، كتلة القسم، كتلة الإجراءات، كتلة الرأس، كتلة المقسم، كتلة السياق، كتلة الجدول، كتلة تخفيض السعر، كتلة النص المنسق، كتلة الصورة، كتلة الإدخال، كتلة الفيديو، كتلة context_actions، كتلة الخطة، كتلة Task_card، Slack modal، Slack App Home، أسطح Slack، عناصر Slack التفاعلية، زر Slack، قائمة تحديد Slack، تجاوز سعة Slack، منتقي البيانات Slack، مربعات اختيار Slack، أزرار الاختيار Slack، وكائنات العمل، وفك رابط Slack، وكتل chat.postMessage، وviews.open، وviews.update، وviews.push، وviews.publish، وكائنات تكوين Slack. يُستخدم عند إنشاء حمولات Block Kit، أو إنشاء مصفوفات الكتل، أو إنشاء نماذج أو طرق عرض الصفحة الرئيسية للتطبيق، أو إضافة عناصر تفاعلية، أو تنفيذ الارتباط مع كائنات العمل، أو تصميم تخطيطات رسائل غنية. إطار عمل Slack Block Kit UI لإنشاء تخطيطات رسائل غنية ونماذج وطرق عرض الصفحة الرئيسية للتطبيق.
