すべてのページをクロールし、フロントマターの概要を含む検索可能なマークダウンとしてパッケージ化することで、あらゆる Web サイトをインストール可能な AI スキル フォルダーに変換します。 Web サイト検索スキルの作成、AI エージェント用の Web サイトのインデックス作成、サイトを検索可能なナレッジ ベースにスクレイピング、Web サイトをオフラインで検索可能なスキルに変換、またはライブ ブラウジングなしで AI エージェントが Web サイトのコンテンツを利用できるようにする場合に使用します。インストール可能なスキル フォルダーと、AI エージェントに追加するための正確な npx コマンドを出力します。
