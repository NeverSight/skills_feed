Интегрирует и расширяет платформу криптовалютных платежей ICPay. Используйте при работе с icpay-widget, icpay-sdk, платежными ссылками, торговыми счетами, ретрансляционными платежами (получатель EVM/IC/Solana), X402 v2, возвратами, разделенными платежами, уведомлениями по электронной почте, веб-перехватчиками, demo.icpay.org, песочницей Betterstripe.com (тестовые сети), фильтрующими токенами/цепочками, WalletConnect QR и глубокими ссылками, адаптерами кошелька, валютой для платежных ссылок и профиля, плагинами WordPress (мгновенный). Crypto Payments, WooCommerce), регистрация на icpay.org, создание учетной записи, ключи API (публикуемые и секретные), .env для ключей, события SDK (icpay-sdk-transaction-completed для успеха, жизненный цикл транзакции, запуск метода/успех/ошибка) или любой код, связанный с ICPay, в монорепозитории icpay.
