Verwenden Sie es beim Erstellen von Cloud-Sandboxen (microVMs), um Code auszuführen, Entwicklungsserver zu starten und Live-Vorschau-URLs zu generieren. Behandelt auch die Bereitstellung von KI-Agenten, MCP-Servern und Batch-Jobs auf der serverlosen Infrastruktur von Blaxel. Greifen Sie auf diese Fähigkeit zu, wenn Sie isolierte Rechenumgebungen, App-Vorschauen in Echtzeit oder die Bereitstellung von Agenten-Workloads benötigen.
