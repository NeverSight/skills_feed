Разрабатывает приложения для iOS с помощью XcodeGen, SwiftUI и SPM. Запускается по конфигурации XcodeGen project.yml, проблемам зависимостей SPM, проблемам с развертыванием устройств, ошибкам подписи кода, отладке камеры/AVFoundation, совместимости версий iOS или ошибкам платформы «Библиотека не загружена @rpath». Используйте при создании приложений iOS, исправлении сбоев сборки Xcode или развертывании на реальных устройствах.
