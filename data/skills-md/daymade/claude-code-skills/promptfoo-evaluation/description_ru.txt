Настраивает и запускает оценку LLM с использованием платформы Promptfoo. Используйте при настройке быстрого тестирования, создании конфигураций оценки (promptfooconfig.yaml), написании пользовательских утверждений Python, реализации llm-rubric для LLM-as-judge или управлении примерами с несколькими кадрами в подсказках. Запускается по таким ключевым словам, как «promptfoo», «eval», «оценка LLM», «быстрое тестирование» или «сравнение моделей».
