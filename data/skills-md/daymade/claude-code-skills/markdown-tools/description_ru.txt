Преобразует документы в уценку с помощью нескольких инструментов для достижения наилучшего качества. Поддерживает быстрый режим (быстрый, один инструмент) и тяжелый режим (наилучшее качество, объединение нескольких инструментов). Используйте при преобразовании файлов PDF/DOCX/PPTX в уценку, извлечении изображений из документов, проверке качества преобразования или необходимости вывода документов, оптимизированных для LLM.
