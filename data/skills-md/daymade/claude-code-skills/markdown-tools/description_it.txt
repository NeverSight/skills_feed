Converte i documenti in markdown con orchestrazione multi-strumento per la migliore qualità. Supporta la modalità rapida (veloce, strumento singolo) e la modalità pesante (migliore qualità, unione di più strumenti). Da utilizzare quando si convertono file PDF/DOCX/PPTX in markdown, si estraggono immagini da documenti, si convalida la qualità della conversione o si necessita di output di documenti ottimizzati per LLM.
