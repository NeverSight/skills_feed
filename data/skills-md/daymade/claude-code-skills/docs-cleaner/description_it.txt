Consolida la documentazione ridondante preservando tutti i contenuti preziosi. Questa competenza dovrebbe essere utilizzata quando gli utenti desiderano ripulire la documentazione in eccesso, unire documenti ridondanti, ridurre l'espansione della documentazione o consolidare più file che trattano lo stesso argomento. I trigger includono "pulisci documenti", "consolida documentazione", "troppi file di documento", "unisci questi documenti" o quando la documentazione supera le 500 righe su più file che trattano argomenti simili.
