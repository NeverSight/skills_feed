Consolide la documentation redondante tout en préservant tout le contenu précieux. Cette compétence doit être utilisée lorsque les utilisateurs souhaitent nettoyer la surcharge de documentation, fusionner des documents redondants, réduire la prolifération de la documentation ou consolider plusieurs fichiers couvrant le même sujet. Les déclencheurs incluent « nettoyer les documents », « consolider la documentation », « trop de fichiers de documents », « fusionner ces documents » ou lorsque la documentation dépasse 500 lignes dans plusieurs fichiers couvrant des sujets similaires.
