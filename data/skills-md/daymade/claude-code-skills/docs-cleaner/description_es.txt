Consolida la documentación redundante mientras preserva todo el contenido valioso. Esta habilidad debe usarse cuando los usuarios desean limpiar la documentación excesiva, fusionar documentos redundantes, reducir la expansión de la documentación o consolidar varios archivos que cubren el mismo tema. Los desencadenantes incluyen "limpiar documentos", "consolidar documentación", "demasiados archivos de documentos", "fusionar estos documentos" o cuando la documentación supera las 500 líneas en varios archivos que cubren temas similares.
