Diagnostica i problemi di qualità della connessione dell'app Windows (desktop remoto Microsoft/desktop virtuale di Azure/W365) su macOS. Analizza la selezione del protocollo di trasporto (UDP Shortpath vs WebSocket), rileva l'interferenza VPN/proxy con la negoziazione STUN/TURN e analizza i log dell'app Windows per individuare eventuali errori Shortpath. Questa capacità deve essere utilizzata quando le connessioni VDI sono lente, quando il trasporto mostra WebSocket anziché UDP, quando RDP Shortpath non riesce a stabilire o quando RTT è inaspettatamente alto.
