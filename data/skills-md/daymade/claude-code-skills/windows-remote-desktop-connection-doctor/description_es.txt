Diagnosticar problemas de calidad de conexión de aplicaciones de Windows (Microsoft Remote Desktop/Azure Virtual Desktop/W365) en macOS. Analice la selección del protocolo de transporte (UDP Shortpath vs WebSocket), detecte interferencias de VPN/proxy con la negociación STUN/TURN y analice los registros de aplicaciones de Windows para detectar fallas en Shortpath. Esta habilidad debe usarse cuando las conexiones VDI son lentas, cuando el transporte muestra WebSocket en lugar de UDP, cuando RDP Shortpath no se establece o cuando RTT es inesperadamente alto.
