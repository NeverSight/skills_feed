Диагностика проблем с качеством подключения приложения Windows (Microsoft Remote Desktop/Виртуальный рабочий стол Azure/W365) в macOS. Анализируйте выбор транспортного протокола (UDP Shortpath или WebSocket), обнаруживайте помехи VPN/прокси-сервера при согласовании STUN/TURN и анализируйте журналы приложений Windows на предмет сбоев Shortpath. Этот навык следует использовать, когда соединения VDI медленные, когда транспорт показывает WebSocket вместо UDP, когда не удается установить RDP Shortpath или когда RTT неожиданно высокий.
