Diagnostiquer les problèmes de qualité de connexion de l'application Windows (Microsoft Remote Desktop / Azure Virtual Desktop / W365) sur macOS. Analysez la sélection du protocole de transport (UDP Shortpath vs WebSocket), détectez les interférences VPN/proxy avec la négociation STUN/TURN et analysez les journaux des applications Windows pour détecter les échecs Shortpath. Cette compétence doit être utilisée lorsque les connexions VDI sont lentes, lorsque le transport affiche WebSocket au lieu d'UDP, lorsque RDP Shortpath ne parvient pas à s'établir ou lorsque RTT est étonnamment élevé.
