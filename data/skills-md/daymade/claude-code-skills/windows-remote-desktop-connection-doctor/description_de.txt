Diagnostizieren Sie Probleme mit der Verbindungsqualität von Windows-Apps (Microsoft Remote Desktop/Azure Virtual Desktop/W365) unter macOS. Analysieren Sie die Auswahl des Transportprotokolls (UDP Shortpath vs. WebSocket), erkennen Sie VPN-/Proxy-Interferenzen bei der STUN/TURN-Aushandlung und analysieren Sie Windows-App-Protokolle auf Shortpath-Fehler. Diese Fähigkeit sollte verwendet werden, wenn VDI-Verbindungen langsam sind, wenn der Transport WebSocket anstelle von UDP anzeigt, wenn RDP Shortpath nicht hergestellt werden kann oder wenn RTT unerwartet hoch ist.
