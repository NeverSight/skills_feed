Создает обучающие публикации на канале Teams для внутреннего обмена знаниями о функциях, инструментах и ​​передовых практиках Claude Code. Применяется при написании сообщений, объявлений или документации для обучения коллег эффективному использованию Claude Code, анонсирования новых функций, обмена советами по продуктивности или документирования извлеченных уроков. Предоставляет шаблоны, рекомендации по написанию и структурированные подходы с акцентом на конкретные примеры, основные принципы и связи с лучшими практиками, такими как контекстная инженерия. Активируется для контента, включающего публикации Teams, объявления каналов, документацию по функциям или обмен советами.
