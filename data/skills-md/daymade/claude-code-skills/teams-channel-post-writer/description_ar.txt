يقوم بإنشاء منشورات قناة Teams التعليمية لمشاركة المعرفة الداخلية حول ميزات Claude Code وأدواته وأفضل الممارسات. ينطبق عند كتابة منشورات أو إعلانات أو وثائق لتعليم الزملاء الاستخدام الفعال لـ Claude Code، أو الإعلان عن ميزات جديدة، أو مشاركة نصائح الإنتاجية، أو توثيق الدروس المستفادة. يوفر قوالب وإرشادات للكتابة وأساليب منظمة تؤكد على الأمثلة الملموسة والمبادئ الأساسية والارتباطات بأفضل الممارسات مثل هندسة السياق. يتم التنشيط للمحتوى الذي يتضمن منشورات Teams أو إعلانات القنوات أو وثائق الميزات أو مشاركة النصائح.
