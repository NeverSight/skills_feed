Claude Code の機能、ツール、ベスト プラクティスに関する社内の知識を共有するための教育用 Teams チャネル投稿を作成します。クロード コードの効果的な使用法を同僚に教えたり、新機能を発表したり、生産性のヒントを共有したり、学んだ教訓を文書化したりするために、投稿、お知らせ、またはドキュメントを作成するときに適用されます。テンプレート、書き方のガイドライン、具体的な例、基礎となる原則、コンテキスト エンジニアリングなどのベスト プラクティスとのつながりを強調する構造化されたアプローチを提供します。 Teams の投稿、チャネルのお知らせ、機能のドキュメント、またはヒントの共有を含むコンテンツに対してアクティブ化されます。
