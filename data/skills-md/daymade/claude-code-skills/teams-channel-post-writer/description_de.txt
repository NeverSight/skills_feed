Erstellt pädagogische Teams-Kanalbeiträge für den internen Wissensaustausch über Funktionen, Tools und Best Practices von Claude Code. Gilt beim Verfassen von Beiträgen, Ankündigungen oder Dokumentationen, um Kollegen die effektive Nutzung von Claude Code beizubringen, neue Funktionen anzukündigen, Produktivitätstipps auszutauschen oder gewonnene Erkenntnisse zu dokumentieren. Bietet Vorlagen, Schreibrichtlinien und strukturierte Ansätze, die konkrete Beispiele, zugrunde liegende Prinzipien und Verbindungen zu Best Practices wie Kontext-Engineering hervorheben. Wird für Inhalte aktiviert, die Teams-Beiträge, Kanalankündigungen, Funktionsdokumentationen oder das Teilen von Tipps umfassen.
