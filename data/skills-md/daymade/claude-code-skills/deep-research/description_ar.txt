قم بإنشاء تقارير بحثية يتم التحكم في تنسيقها مع تتبع الأدلة والاستشهادات والمراجعة المتكررة. يجب استخدام هذه المهارة عندما يطلب المستخدمون تقريرًا بحثيًا، أو مراجعة الأدبيات، أو تحليل السوق أو الصناعة، أو المشهد التنافسي، أو ملخص السياسة أو التقنية، أو عندما يطلبون قالب تقرير صارمًا وتنسيق القسم الذي لا يمكن لتمرير بحث عميق واحد تنفيذه بشكل موثوق.
