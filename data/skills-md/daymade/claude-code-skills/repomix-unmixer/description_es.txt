Extrae archivos de repositorios empaquetados con repomix, restaurando estructuras de directorios originales desde formatos XML/Markdown/JSON. Se activa cuando los usuarios necesitan desmezclar archivos repomix, extraer repositorios empaquetados, restaurar estructuras de archivos a partir de la salida de repomix o invertir el proceso de empaquetado de repomix.
