Обеспечивает комплексные операции GitHub с использованием gh CLI и GitHub API. Активируется при работе с запросами на включение, проблемами, репозиториями, рабочими процессами или операциями API GitHub, включая создание/просмотр/объединение PR, управление проблемами, запросы к конечным точкам API и обработку рабочих процессов GitHub в корпоративных или общедоступных средах GitHub.
