Bietet umfassende GitHub-Vorgänge mithilfe von gh CLI und GitHub API. Wird aktiviert, wenn mit Pull Requests, Problemen, Repositorys, Workflows oder GitHub-API-Vorgängen gearbeitet wird, einschließlich Erstellen/Anzeigen/Zusammenführen von PRs, Verwalten von Problemen, Abfragen von API-Endpunkten und Bearbeiten von GitHub-Workflows in Unternehmens- oder öffentlichen GitHub-Umgebungen.
