Fournit des opérations GitHub complètes à l’aide de gh CLI et de l’API GitHub. S'active lorsque vous travaillez avec des demandes d'extraction, des problèmes, des référentiels, des flux de travail ou des opérations de l'API GitHub, notamment la création/affichage/fusion de PR, la gestion des problèmes, l'interrogation des points de terminaison de l'API et la gestion des flux de travail GitHub dans des environnements GitHub d'entreprise ou publics.
