Diagnostiquer et résoudre les problèmes de plugin et de compétences Claude Code. Cette compétence doit être utilisée lorsque les plugins sont installés mais n'apparaissent pas dans la liste des compétences disponibles, que les compétences ne s'activent pas comme prévu ou lors du dépannage de la configuration des plugins activés dans settings.json. Les déclencheurs incluent les problèmes « plugin ne fonctionne pas », « compétence non affichée », « installé mais désactivé » ou « enabledPlugins ».
