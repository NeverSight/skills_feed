Diagnosticar y resolver problemas de habilidades y complementos de Claude Code. Esta habilidad debe usarse cuando los complementos están instalados pero no se muestran en la lista de habilidades disponibles, las habilidades no se activan como se esperaba o cuando se solucionan problemas de configuración de complementos habilitados en settings.json. Los desencadenantes incluyen problemas de "complemento que no funciona", "habilidad que no se muestra", "instalado pero deshabilitado" o "complementos habilitados".
