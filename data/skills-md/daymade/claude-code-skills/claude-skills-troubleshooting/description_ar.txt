تشخيص وحل مشكلات المهارات والمكونات الإضافية لـ Claude Code. يجب استخدام هذه المهارة عند تثبيت المكونات الإضافية ولكن لا تظهر في قائمة المهارات المتاحة، أو عند عدم تنشيط المهارات كما هو متوقع، أو عند استكشاف أخطاء تكوين الإضافات الممكنة في settings.json. تتضمن المشغلات مشكلات "المكونات الإضافية لا تعمل"، أو "لا تظهر المهارة"، أو "مثبتة ولكن معطلة"، أو "المكونات الإضافية الممكنة".
