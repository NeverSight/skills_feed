Cette compétence doit être utilisée lorsque les utilisateurs souhaitent rechercher, découvrir, installer ou gérer les compétences Claude Code à partir du registre CCPM. Les déclencheurs incluent des requêtes telles que « trouver des compétences pour PDF », « rechercher des compétences en révision de code », « installer le dépannage cloudflare », « lister mes compétences installées », « que fait le créateur de compétences » ou toute mention de recherche/installation/gestion des compétences ou des plugins Claude Code.
