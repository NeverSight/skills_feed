Dieser Skill sollte verwendet werden, wenn Benutzer Claude Code-Skills in der CCPM-Registrierung suchen, entdecken, installieren oder verwalten möchten. Zu den Auslösern gehören Anfragen wie „Skills für PDF finden“, „Nach Code-Review-Skills suchen“, „Cloudflare-Fehlerbehebung installieren“, „Meine installierten Skills auflisten“, „Was macht Skill-Creator“ oder jede Erwähnung des Suchens/Installierens/Verwaltens von Claude Code-Skills oder Plugins.
