Этот навык следует использовать, когда пользователи хотят искать, обнаруживать, устанавливать навыки Claude Code или управлять ими из реестра CCPM. Триггеры включают такие запросы, как «найти навыки для PDF», «поиск навыков проверки кода», «установить Cloudflare-Troubleshooting», «перечислить мои установленные навыки», «что делает создатель навыков» или любое упоминание о поиске/установке/управлении навыками или плагинами Claude Code.
