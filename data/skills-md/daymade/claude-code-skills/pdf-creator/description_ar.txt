قم بإنشاء مستندات PDF من تخفيض السعر مع دعم الخط الصيني المناسب باستخدام weasyprint. يجب استخدام هذه المهارة عند تحويل تخفيض السعر إلى PDF، أو إنشاء مستندات رسمية (قانونية، أو ملفات العلامات التجارية، أو التقارير)، أو عندما تكون الطباعة الصينية مطلوبة. تتضمن المشغلات "التحويل إلى PDF" أو "إنشاء PDF" أو "تخفيض السعر إلى PDF" أو أي طلب لإنشاء مستندات قابلة للطباعة.
