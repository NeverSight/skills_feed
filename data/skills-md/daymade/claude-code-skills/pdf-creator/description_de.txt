Erstellen Sie mit weasyprint PDF-Dokumente aus Markdown mit ordnungsgemäßer Unterstützung für chinesische Schriftarten. Diese Fähigkeit sollte verwendet werden, wenn Markdown in PDF konvertiert wird, formelle Dokumente (Rechtsdokumente, Markenanmeldungen, Berichte) erstellt werden oder wenn chinesische Typografie erforderlich ist. Zu den Auslösern gehören „In PDF konvertieren“, „PDF generieren“, „Markdown in PDF“ oder jede Anfrage zum Erstellen druckbarer Dokumente.
