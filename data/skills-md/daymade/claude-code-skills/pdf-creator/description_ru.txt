Создавайте PDF-документы из уценки с правильной поддержкой китайских шрифтов с помощью weasyprint. Этот навык следует использовать при преобразовании уценки в PDF, создании официальных документов (юридических, заявок на товарные знаки, отчетов) или когда требуется китайская типографика. Триггеры включают «конвертировать в PDF», «создать PDF», «уценку в PDF» или любой запрос на создание документов для печати.
