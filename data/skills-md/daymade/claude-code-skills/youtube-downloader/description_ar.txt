قم بتنزيل مقاطع فيديو YouTube وتدفقات HLS (m3u8) من منصات مثل Mux وVimeo وما إلى ذلك باستخدام yt-dlp وffmpeg. استخدم هذه المهارة عندما يطلب المستخدمون تنزيل مقاطع فيديو، أو استخراج الصوت، أو التعامل مع التدفقات المحمية باستخدام رؤوس المصادقة، أو استكشاف مشكلات التنزيل وإصلاحها مثل فشل استخراج nsig، أو أخطاء 403، أو مشكلات استخراج ملفات تعريف الارتباط.
