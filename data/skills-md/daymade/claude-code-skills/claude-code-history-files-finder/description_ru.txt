Находит и восстанавливает содержимое из файлов истории сеансов Claude Code. Этот навык следует использовать при поиске удаленных файлов, отслеживании изменений между сеансами, анализе истории разговоров или восстановлении кода из предыдущих взаимодействий с Клодом. Триггеры включают упоминания «истории сеансов», «восстановления удаленных», «найти в истории», «предыдущего разговора» или «.claude/projects».
