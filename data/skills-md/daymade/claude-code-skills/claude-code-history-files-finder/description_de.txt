Findet und stellt Inhalte aus Claude Code-Sitzungsverlaufsdateien wieder her. Diese Fähigkeit sollte verwendet werden, wenn Sie nach gelöschten Dateien suchen, Änderungen über Sitzungen hinweg verfolgen, den Gesprächsverlauf analysieren oder Code aus früheren Claude-Interaktionen wiederherstellen. Zu den Auslösern gehören Erwähnungen von „Sitzungsverlauf“, „Gelöschte wiederherstellen“, „Im Verlauf finden“, „Vorherige Konversation“ oder „.claude/projects“.
