Получите содержимое публикации Twitter/X по URL-адресу, используя API jina.ai, чтобы обойти ограничения JavaScript. Используйте, когда Клоду необходимо получить содержимое твита, включая автора, временную метку, текст сообщения, изображения и ответы в цепочках. Поддерживает отдельные публикации или пакетную загрузку с URL-адресов x.com или twitter.com.
