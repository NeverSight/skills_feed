Многопутевой параллельный анализ продуктов с масштабированием вычислений во время тестирования между моделями. Создает параллельные агенты (группы агентов Claude Code + интерфейс командной строки Codex) для изучения продукта с разных точек зрения, а затем синтезирует результаты в действенные планы оптимизации. Может использовать анализ конкурентов для сравнительного анализа конкурентов. Используйте, когда «аудит продукта», «самопроверка», «发布前审查», «产品分析», «проанализируйте наш продукт», «UX-аудит» или «信息架构审计».
