Esta habilidad debe usarse cuando los usuarios quieran crear demostraciones CLI animadas, grabaciones de terminal o GIF de demostración de línea de comandos. Admite tanto la creación manual de archivos de cinta como la generación automatizada de demostraciones a partir de descripciones de comandos. Úselo cuando los usuarios mencionen la creación de demostraciones, la grabación de sesiones de terminal o la generación de GIF animados de flujos de trabajo CLI.
