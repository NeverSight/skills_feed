Examine et améliore les compétences de Claude Code par rapport aux meilleures pratiques officielles. Prend en charge trois modes : auto-évaluation (validez vos propres compétences), évaluation externe (évaluez les compétences des autres) et auto-PR (fork, améliorez, soumettez). À utiliser pour vérifier la qualité des compétences, examiner les référentiels de compétences ou contribuer à l'amélioration des compétences open source.
