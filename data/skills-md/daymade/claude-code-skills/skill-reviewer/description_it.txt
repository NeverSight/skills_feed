Rivede e migliora le competenze di Claude Code rispetto alle migliori pratiche ufficiali. Supporta tre modalità: auto-revisione (convalida le tue capacità), revisione esterna (valuta le capacità degli altri) e auto-PR (diffusione, miglioramento, invio). Da utilizzare per verificare la qualità delle competenze, rivedere i repository di competenze o contribuire a migliorare le competenze open source.
