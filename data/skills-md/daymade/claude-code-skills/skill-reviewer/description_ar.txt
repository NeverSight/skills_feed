يراجع مهارات Claude Code ويحسنها وفقًا لأفضل الممارسات الرسمية. يدعم ثلاثة أوضاع - المراجعة الذاتية (التحقق من مهاراتك الخاصة)، والمراجعة الخارجية (تقييم مهارات الآخرين)، والعلاقات العامة التلقائية (الشوكة، والتحسين، والإرسال). يُستخدم عند التحقق من جودة المهارات، أو مراجعة مستودعات المهارات، أو المساهمة في تحسين المهارات مفتوحة المصدر.
