Überprüft und verbessert die Claude-Code-Fähigkeiten anhand offizieller Best Practices. Unterstützt drei Modi: Selbstbewertung (Bestätigung der eigenen Fähigkeiten), externe Überprüfung (Bewertung der Fähigkeiten anderer) und Auto-PR (Fork, Verbesserung, Einreichung). Verwenden Sie diese Funktion, wenn Sie die Qualität von Fertigkeiten überprüfen, Fertigkeitsrepositorys überprüfen oder Verbesserungen an Open-Source-Fertigkeiten beisteuern möchten.
