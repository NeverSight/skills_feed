Guía estratégica para convertirse en un colaborador eficaz de GitHub. Cubre el descubrimiento de oportunidades, la selección de proyectos, la creación de relaciones públicas de alta calidad y la creación de reputación. Úselo cuando busque contribuir a proyectos de código abierto, crear presencia en GitHub o aprender las mejores prácticas de contribución.
