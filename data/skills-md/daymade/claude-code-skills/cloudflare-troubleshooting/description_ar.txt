التحقيق في مشكلات تكوين Cloudflare وحلها باستخدام جمع الأدلة المستندة إلى واجهة برمجة التطبيقات. يُستخدم عند استكشاف أخطاء ERR_TOO_MANY_REDIRECTS أو أخطاء SSL أو مشكلات DNS أو أي مشكلات متعلقة بـ Cloudflare وإصلاحها. ركز على التحقيق المنهجي باستخدام Cloudflare API لفحص التكوين الفعلي بدلاً من وضع الافتراضات.
