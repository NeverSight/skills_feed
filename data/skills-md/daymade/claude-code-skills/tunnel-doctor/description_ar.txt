تشخيص التعارضات وإصلاحها بين Tailscale وأدوات الوكيل/VPN (Shadowrocket وClash وSurge) على نظام التشغيل macOS. يغطي نوعين من التعارض - اختطاف المسار (يتجاوز الوكيل TUN مسارات Tailscale) واعتراض var لوكيل HTTP (تكوين http_proxy/NO_PROXY الخاطئ). يُستخدم عندما يعمل اختبار Tailscale ولكن تنتهي مهلة SSH/HTTP، أو عندما يعود curl to Tailscale IPs فارغًا/مهلة، أو عند إعداد مثيلات Tailscale SSH إلى WSL.
