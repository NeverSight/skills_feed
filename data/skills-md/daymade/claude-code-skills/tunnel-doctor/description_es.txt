Diagnosticar y solucionar conflictos entre Tailscale y las herramientas proxy/VPN (Shadowrocket, Clash, Surge) en macOS. Cubre dos tipos de conflictos: secuestro de ruta (el proxy TUN anula las rutas de Tailscale) e intercepción de var de entorno del proxy HTTP (configuración incorrecta de http_proxy/NO_PROXY). Úselo cuando el ping de Tailscale funcione pero SSH/HTTP agote el tiempo de espera, cuando el curl a las IP de Tailscale devuelva vacío/tiempo de espera de espera, o cuando configure instancias de Tailscale SSH a WSL.
