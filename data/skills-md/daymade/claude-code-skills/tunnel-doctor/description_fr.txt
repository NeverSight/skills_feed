Diagnostiquez et corrigez les conflits entre Tailscale et les outils proxy/VPN (Shadowrocket, Clash, Surge) sur macOS. Couvre deux types de conflits : le détournement de route (le proxy TUN remplace les routes Tailscale) et l'interception de la variable d'environnement du proxy HTTP (mauvaise configuration http_proxy/NO_PROXY). À utiliser lorsque le ping Tailscale fonctionne mais que SSH/HTTP expire, lorsque curl vers les adresses IP Tailscale renvoie un délai vide/expiré, ou lors de la configuration de Tailscale SSH sur des instances WSL.
