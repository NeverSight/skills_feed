Настраивает и настраивает строки состояния Claude Code с многострочными макетами, отслеживанием затрат с помощью ccusage, индикаторами состояния git и настраиваемыми цветами. Активируется для настройки, установки, настройки, изменения цвета, отображения стоимости, интеграции статуса git или устранения неполадок в строке состояния.
