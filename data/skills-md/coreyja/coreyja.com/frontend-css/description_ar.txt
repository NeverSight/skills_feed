اكتب CSS قابلاً للصيانة باتباع المنهجية المحددة للمشروع، سواء باستخدام فئات الأداة المساعدة Tailwind أو تسمية BEM أو وحدات CSS أو المكونات المصممة. استخدم هذه المهارة عند تصميم مكونات واجهة المستخدم أو الصفحات باستخدام CSS، أو تطبيق الرموز المميزة لنظام التصميم للألوان والمسافات، أو العمل مع أطر عمل CSS. عند كتابة فئات المرافق أو الأنماط المخصصة أو قواعد التصميم سريعة الاستجابة. عند العمل مع ملفات .css، .scss، .module.css، أو الأنماط المضمنة في JSX/templates. عند إنشاء أو الحفاظ على رموز التصميم للتصميم المتسق. عند تحسين CSS للإنتاج، يتم إنشاء الأنماط غير المستخدمة بالتطهير أو هز الأشجار.
