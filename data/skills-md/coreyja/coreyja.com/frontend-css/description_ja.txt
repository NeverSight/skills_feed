Tailwind ユーティリティ クラス、BEM 命名、CSS モジュール、またはスタイル付きコンポーネントを使用するかどうかにかかわらず、プロジェクトの確立された方法論に従って保守可能な CSS を作成します。このスキルは、CSS を使用して UI コンポーネントまたはページのスタイルを設定するとき、色と間隔のデザイン システム トークンを適用するとき、または CSS フレームワークを操作するときに使用します。ユーティリティ クラス、カスタム スタイル、またはレスポンシブ デザイン ルールを作成するとき。 .css、.scss、.module.css ファイル、または JSX/テンプレートのインライン スタイルを操作する場合。一貫したスタイルのためのデザイン トークンを確立または維持する場合。未使用のスタイルを削除またはツリーシェイクして実稼働ビルド用に CSS を最適化する場合。
