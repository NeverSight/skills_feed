يجب استخدام هذه المهارة عند تخزين التطبيقات في حاويات باستخدام Docker، أو إنشاء ملفات Dockerfiles، أو تكوينات إنشاء عامل الإرساء، أو نشر الحاويات على منصات مختلفة. مثالي لتطبيقات Next.js وReact وNode.js التي تتطلب النقل بالحاويات للتطوير أو الإنتاج أو خطوط أنابيب CI/CD. استخدم هذه المهارة عندما يحتاج المستخدمون إلى تكوينات Docker، أو عمليات إنشاء متعددة المراحل، أو تنسيق الحاويات، أو النشر إلى Kubernetes، أو ECS، أو Cloud Run، وما إلى ذلك.
