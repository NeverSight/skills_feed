Этот навык следует использовать при написании тестовых примеров, исправлении ошибок, анализе кода на наличие потенциальных проблем или улучшении тестового покрытия для приложений JavaScript/TypeScript. Используйте это для модульных тестов, интеграционных тестов, сквозных тестов, отладки ошибок времени выполнения, логических ошибок, проблем с производительностью, уязвимостей безопасности и систематического анализа кода.
