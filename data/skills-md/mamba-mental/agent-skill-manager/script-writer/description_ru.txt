Этот навык следует использовать всякий раз, когда пользователям нужно написать сценарии видео для YouTube. При первом использовании собирает всесторонние предпочтения, включая тип сценария, тон, целевую аудиторию, стиль, продолжительность видео, стиль зацепки, использование юмора, индивидуальность и подход к повествованию. Создает полные, готовые к использованию сценарии YouTube, адаптированные к требованиям пользователя для любой темы. Поддерживает базу данных предпочтений и прошлых сценариев для обеспечения единообразия стиля.
