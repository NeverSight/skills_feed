Guía de expertos para utilizar el servidor Pal MCP (zen-pal-nas). Esta habilidad debe usarse cuando se trabaja con orquestación de IA multimodelo, flujos de trabajo de herramientas (chat, thinkdeep, planificador, consenso, depuración, revisión de código, compromiso previo, clink), resolución de problemas de configuración u optimización de estrategias de selección de modelos. Se activa automáticamente cuando el usuario menciona Pal MCP, zen-pal-nas o nombres de herramientas específicas.
