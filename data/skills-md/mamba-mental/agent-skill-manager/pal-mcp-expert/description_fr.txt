Conseils d'experts pour l'utilisation du serveur Pal MCP (zen-pal-nas). Cette compétence doit être utilisée lorsque vous travaillez avec une orchestration d'IA multimodèle, des flux de travail d'outils (chat, thinkdeep, planificateur, consensus, débogage, révision de code, pré-engagement, clink), le dépannage de configuration ou l'optimisation des stratégies de sélection de modèles. S'active automatiquement lorsque l'utilisateur mentionne Pal MCP, zen-pal-nas ou des noms d'outils spécifiques.
