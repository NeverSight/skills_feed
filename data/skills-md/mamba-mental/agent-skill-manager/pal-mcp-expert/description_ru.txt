Экспертное руководство по использованию сервера Pal MCP (zen-pal-nas). Этот навык следует использовать при работе с многомодельной оркестровкой ИИ, рабочими процессами инструментов (чат, thinkdeep, планировщик, консенсус, отладка, проверка кода, предварительная фиксация, клик), устранением неполадок конфигурации или оптимизации стратегий выбора модели. Активируется автоматически, когда пользователь упоминает Pal MCP, zen-pal-nas или названия конкретных инструментов.
