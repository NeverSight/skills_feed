Этот навык следует использовать при создании или настройке файлов конвейера CI/CD для автоматического тестирования, сборки и развертывания. Используйте это для создания рабочих процессов GitHub Actions, конфигураций GitLab CI, конфигураций CircleCI или других конфигураций платформы CI/CD. Идеально подходит для настройки автоматизированных конвейеров для приложений Node.js/Next.js, включая анализ, тестирование, сборку и развертывание на таких платформах, как Vercel, Netlify или AWS.
