Cette compétence doit être utilisée lorsque vous travaillez avec des fichiers CSV pour créer des visualisations de données interactives, générer des graphiques statistiques, analyser les distributions de données, créer des tableaux de bord ou effectuer un profilage automatique des données. Il fournit des outils complets pour l'analyse exploratoire des données à l'aide de Plotly pour des visualisations interactives.
