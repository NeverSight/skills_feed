Этот навык следует использовать при написании документации для баз кода, включая файлы README, документацию по архитектуре, комментарии к коду и документацию по API. Используйте этот навык, когда пользователи запрашивают помощь в документировании своего кода, создании руководств по началу работы, объяснении структуры проекта или повышении доступности кодовых баз для новых разработчиков. Этот навык предоставляет шаблоны, лучшие практики и структурированные подходы для создания понятной, удобной для начинающих документации.
