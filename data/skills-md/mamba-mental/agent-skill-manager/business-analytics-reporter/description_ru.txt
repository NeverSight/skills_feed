Этот навык следует использовать при анализе данных о продажах и доходах бизнеса из файлов CSV, чтобы выявить слабые места, получить статистическую информацию и предоставить рекомендации по стратегическому улучшению. Используйте, когда пользователь запрашивает отчет об эффективности бизнеса, просит проанализировать данные о продажах, хочет выявить слабые места или нуждается в рекомендациях по стратегиям улучшения бизнеса.
