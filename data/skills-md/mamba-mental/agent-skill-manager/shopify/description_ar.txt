أنشئ تطبيقات وإضافات وسمات Shopify باستخدام واجهات برمجة تطبيقات GraphQL/REST وShopify CLI ومكونات Polaris UI وقوالب Liquid. تشمل الإمكانات تطوير التطبيق باستخدام مصادقة OAuth، وملحقات واجهة المستخدم الخاصة بالخروج لتخصيص تدفق الخروج، وملحقات واجهة المستخدم الإدارية لتكامل لوحة المعلومات، وملحقات نقاط البيع للبيع بالتجزئة، وتطوير السمات باستخدام Liquid، وإدارة الخطاف على الويب، وتكامل واجهة برمجة تطبيقات الفوترة، وإدارة المنتج/الطلب/العميل. يُستخدم عند إنشاء تطبيقات Shopify، أو تنفيذ تخصيصات الخروج، أو إنشاء واجهات الإدارة، أو تطوير السمات، أو دمج معالجة الدفع، أو إدارة بيانات المتجر عبر واجهات برمجة التطبيقات، أو توسيع وظائف Shopify.
