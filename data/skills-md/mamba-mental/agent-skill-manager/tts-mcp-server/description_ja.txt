このスキルは、官能的、魅惑的、不吉で感情表現豊かな音声出力を実現するために、TTS-MCP-SERVER を イレブンラボの eleven_v3 モデルで使用するための包括的なガイダンスを提供します。このスキルは、音声アナウンスの生成、ダーク/セクシー/いたずらな感情表現のためのオーディオ タグの適用、「いたずらな専門責任パートナー」スクリプトの作成、または TTS 統合のトラブルシューティングを行うときに使用します。 speech_text 通話、音声出力リクエスト、イレブンラボ TTS 操作、オーディオ タグの使用、または自動化されたワークフロー中の音声アナウンスでアクティブになります。
