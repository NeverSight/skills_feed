Questa competenza fornisce una guida completa per l'utilizzo di TTS-MCP-SERVER con il modello ElevenLabs Eleven_v3 per un output vocale sensuale, seducente, minaccioso ed emotivamente espressivo. Usa questa abilità quando generi annunci vocali, applichi tag audio per espressioni emotive oscure/sexy/maliziose, crei script di "partner di responsabilità professionale cattivo" o risolvi i problemi di integrazione TTS. Si attiva su chiamate speak_text, richieste di output vocale, operazioni TTS di ElevenLabs, utilizzo di tag audio o annunci vocali durante i flussi di lavoro automatizzati.
