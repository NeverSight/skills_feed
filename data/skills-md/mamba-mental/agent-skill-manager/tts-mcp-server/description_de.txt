Dieser Skill bietet eine umfassende Anleitung für die Verwendung von TTS-MCP-SERVER mit dem ElevenLabs-Eleven_v3-Modell für sinnliche, verführerische, bedrohliche und emotional ausdrucksstarke Sprachausgaben. Verwenden Sie diese Fähigkeit, wenn Sie Sprachansagen erstellen, Audio-Tags für düstere/sexy/schelmischen emotionalen Ausdruck anwenden, Skripte für „freche, professionelle Verantwortlichkeitspartner“ erstellen oder Fehler bei der TTS-Integration beheben. Aktiviert bei speak_text-Anrufen, Sprachausgabeanfragen, ElevenLabs TTS-Vorgängen, der Verwendung von Audio-Tags oder Sprachansagen während automatisierter Arbeitsabläufe.
