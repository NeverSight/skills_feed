توفر هذه المهارة إرشادات شاملة لاستخدام TTS-MCP-SERVER مع نموذج ElevenLabs Eleven_v3 للحصول على مخرجات صوتية مثيرة ومغرية ومشؤومة ومعبرة عاطفيًا. استخدم هذه المهارة عند إنشاء إعلانات صوتية، أو تطبيق علامات صوتية للتعبير العاطفي المظلم/المثير/المؤذ، أو صياغة نصوص برمجية "لشريك المساءلة المحترف المشاغب"، أو استكشاف أخطاء تكامل تحويل النص إلى كلام (TTS) وإصلاحها. يتم التنشيط عند مكالمات talk_text، أو طلبات الإخراج الصوتي، أو عمليات ElevenLabs TTS، أو استخدام علامة الصوت، أو الإعلانات الصوتية أثناء سير العمل الآلي.
