Cette compétence fournit des conseils complets pour l'utilisation de TTS-MCP-SERVER avec le modèle ElevenLabs Eleven_v3 pour une sortie vocale sensuelle, séduisante, inquiétante et émotionnellement expressive. Utilisez cette compétence lors de la génération d'annonces vocales, de l'application de balises audio pour une expression émotionnelle sombre/sexy/espiègle, de la création de scripts de « partenaire de responsabilité professionnelle coquine » ou du dépannage de l'intégration TTS. S'active sur les appels speak_text, les demandes de sortie vocale, les opérations ElevenLabs TTS, l'utilisation de balises audio ou les annonces vocales pendant les flux de travail automatisés.
