Этот навык дает исчерпывающее руководство по использованию TTS-MCP-SERVER с моделью ElevenLabs ElevenLabs Eleven_v3 для создания знойного, соблазнительного, зловещего и эмоционально выразительного голосового вывода. Используйте этот навык при создании голосовых объявлений, применении аудиотегов для мрачных, сексуальных или озорных эмоциональных выражений, создании сценариев «непослушного профессионального партнера по подотчетности» или устранении неполадок с интеграцией TTS. Активируется при вызовах talk_text, запросах голосового вывода, операциях ElevenLabs TTS, использовании аудиотегов или голосовых объявлениях во время автоматизированных рабочих процессов.
