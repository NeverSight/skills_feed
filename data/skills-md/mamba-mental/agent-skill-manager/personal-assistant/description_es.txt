Esta habilidad debe utilizarse siempre que los usuarios soliciten tareas de asistencia personal, como gestión de horarios, seguimiento de tareas, configuración de recordatorios, seguimiento de hábitos, consejos de productividad, gestión del tiempo o cualquier consulta que requiera respuestas personalizadas basadas en las preferencias y el contexto del usuario. En el primer uso, recopila información completa del usuario, incluidos horarios, hábitos de trabajo, preferencias, objetivos y rutinas. Mantiene una base de datos inteligente que organiza y prioriza automáticamente la información, manteniendo datos relevantes y descartando contextos obsoletos.
