يجب استخدام هذه المهارة عندما يطلب المستخدمون مهام مساعدة شخصية مثل إدارة الجدول الزمني، أو تتبع المهام، أو إعداد التذكير، أو مراقبة العادات، أو نصائح الإنتاجية، أو إدارة الوقت، أو أي استفسار يتطلب استجابات شخصية بناءً على تفضيلات المستخدم والسياق. عند الاستخدام لأول مرة، يجمع معلومات شاملة للمستخدم بما في ذلك الجدول الزمني وعادات العمل والتفضيلات والأهداف والروتينات. يحتفظ بقاعدة بيانات ذكية تقوم تلقائيًا بتنظيم المعلومات وترتيب أولوياتها، مع الاحتفاظ بالبيانات ذات الصلة والتخلص من السياق القديم.
