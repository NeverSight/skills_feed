Elabora file multimediali con FFmpeg (codifica video/audio, conversione, streaming, filtraggio, accelerazione hardware) e ImageMagick (manipolazione delle immagini, conversione del formato, elaborazione batch, effetti, composizione). Utilizzare per convertire formati multimediali, codificare video con codec specifici (H.264, H.265, VP9), ridimensionare/ritagliare immagini, estrarre audio da video, applicare filtri ed effetti, ottimizzare le dimensioni dei file, creare manifest di streaming (HLS/DASH), generare miniature, elaborare immagini in batch, creare immagini composite o implementare pipeline di elaborazione multimediale. Supporta oltre 100 formati, accelerazione hardware (NVENC, QSV) e grafici filtro complessi.
