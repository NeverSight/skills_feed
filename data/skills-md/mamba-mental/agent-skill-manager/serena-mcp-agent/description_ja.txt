Serena MCP サーバーの専門家による統合 - LLM に IDE のようなセマンティック コードの理解を提供する強力なコーディング エージェント ツールキット。このスキルは、Serena ツールを使用したコードベースの操作、Serena プロジェクトのセットアップ、セマンティック コード ナビゲーションと編集の実行、プロジェクト メモリの管理、複雑な自動化ワークフローのデバッグ、または Serena と Claude Desktop、Claude Code、Codex、ChatGPT、またはカスタム エージェントとの統合を行うときに使用する必要があります。 Serena ツールの使用、プロジェクトのアクティブ化/オンボーディング、シンボリック コード操作 (find_symbol、replace_symbol_body など)、メモリ管理 (write_memory、read_memory)、および MCP サーバー構成をトリガーします。構造の理解、タスクのリファクタリング、トークン効率の高いコード操作が必要な大規模/複雑なコードベースに使用します。
