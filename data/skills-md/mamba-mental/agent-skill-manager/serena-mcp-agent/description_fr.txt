Intégration experte pour le serveur Serena MCP - une puissante boîte à outils d'agent de codage fournissant aux LLM une compréhension du code sémantique de type IDE. Cette compétence doit être utilisée lors de l'utilisation de bases de code via les outils Serena, de la configuration de projets Serena, de la navigation et de l'édition de codes sémantiques, de la gestion des mémoires de projets, du débogage de flux de travail d'automatisation complexes ou de l'intégration de Serena avec Claude Desktop, Claude Code, Codex, ChatGPT ou des agents personnalisés. Déclencheurs sur l'utilisation de l'outil Serena, l'activation/l'intégration du projet, les opérations de code symbolique (find_symbol, replace_symbol_body, etc.), la gestion de la mémoire (write_memory, read_memory) et la configuration du serveur MCP. À utiliser pour les bases de code volumineuses/complexes nécessitant une compréhension structurelle, des tâches de refactorisation et des opérations de code efficaces en termes de jetons.
