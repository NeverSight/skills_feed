Integración experta para Serena MCP Server: un potente conjunto de herramientas de agente de codificación que proporciona comprensión del código semántico similar a IDE a los LLM. Esta habilidad debe usarse al trabajar con bases de código a través de herramientas de Serena, configurar proyectos de Serena, realizar navegación y edición de código semántico, administrar memorias de proyectos, depurar flujos de trabajo de automatización complejos o integrar Serena con Claude Desktop, Claude Code, Codex, ChatGPT o agentes personalizados. Activadores del uso de la herramienta Serena, activación/incorporación del proyecto, operaciones de código simbólico (find_symbol, replace_symbol_body, etc.), administración de memoria (write_memory, read_memory) y configuración del servidor MCP. Úselo para bases de código grandes/complejas que requieran comprensión estructural, tareas de refactorización y operaciones de código con token eficiente.
