Экспертная интеграция с сервером Serena MCP — мощным набором инструментов агента кодирования, обеспечивающим понимание семантического кода на уровне IDE для LLM. Этот навык следует использовать при работе с базами кода с помощью инструментов Serena, настройке проектов Serena, выполнении семантической навигации и редактирования кода, управлении памятью проектов, отладке сложных рабочих процессов автоматизации или интеграции Serena с Claude Desktop, Claude Code, Codex, ChatGPT или пользовательскими агентами. Запускает использование инструмента Serena, активацию/адаптацию проекта, операции с символьным кодом (find_symbol, replace_symbol_body и т. д.), управление памятью (write_memory, read_memory) и конфигурацию сервера MCP. Используйте для больших и сложных баз кода, требующих структурного понимания, задач рефакторинга и операций кода с эффективным использованием токенов.
