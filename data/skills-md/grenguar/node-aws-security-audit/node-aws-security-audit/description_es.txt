Realice auditorías de seguridad integrales en las bases de código de Node.js, JavaScript y TypeScript. Escanea el código fuente en busca de las 10 principales vulnerabilidades, patrones inseguros y riesgos de dependencia de OWASP y genera un informe de auditoría de Markdown priorizado con niveles de gravedad (crítico, alto, medio, bajo), citas de código y orientación sobre remediación. Úselo cuando se le solicite auditar, revisar, escanear o verificar la seguridad de un proyecto Node.js/JS/TS, encontrar vulnerabilidades, verificar el cumplimiento de OWASP o generar un informe de seguridad. También se activa en: revisión de seguridad, pentest, escaneo de vulnerabilidades, mi código es seguro, verificar si hay inyecciones, verificar dependencias. NO active para: revisiones generales de código, asistencia para corrección de errores, desarrollo de funciones, refactorización, optimización del rendimiento o tareas que no sean de seguridad.
