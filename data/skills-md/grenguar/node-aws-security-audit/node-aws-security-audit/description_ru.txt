Выполняйте комплексные проверки безопасности кодовых баз Node.js, JavaScript и TypeScript. Сканирует исходный код на предмет 10 наиболее уязвимых мест OWASP, небезопасных шаблонов, рисков зависимостей и создает приоритетный отчет аудита Markdown с уровнями серьезности (критический, высокий, средний, низкий), ссылками на код и рекомендациями по исправлению. Используйте, когда вас просят провести аудит, просмотр, сканирование или проверку безопасности проекта Node.js/JS/TS, найти уязвимости, проверить соответствие OWASP или создать отчет о безопасности. Также срабатывает при проверке безопасности, пентесте, сканировании уязвимостей, безопасности моего кода, проверке на внедрение, проверке зависимостей. НЕ активируйте для: общих проверок кода, помощи в исправлении ошибок, разработки функций, рефакторинга, оптимизации производительности или задач, не связанных с безопасностью.
