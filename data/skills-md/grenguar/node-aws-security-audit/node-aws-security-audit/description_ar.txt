قم بإجراء عمليات تدقيق أمنية شاملة على قواعد تعليمات Node.js وJavaScript وTypeScript. يقوم بمسح الكود المصدري لأهم 10 نقاط ضعف في OWASP، والأنماط غير الآمنة، ومخاطر التبعية، وينشئ تقرير تدقيق Markdown ذو أولوية مع مستويات الخطورة (حرجة، عالية، متوسطة، منخفضة)، واستشهادات التعليمات البرمجية، وإرشادات العلاج. يُستخدم عند مطالبتك بتدقيق أو مراجعة أو مسح أو التحقق من أمان مشروع Node.js/JS/TS، أو العثور على نقاط الضعف، أو التحقق من امتثال OWASP، أو إنشاء تقرير أمان. يتم تشغيله أيضًا على: مراجعة الأمان، اختبار الاختراق، فحص الثغرات الأمنية، هل الكود الخاص بي آمن، التحقق من الحقن، التحقق من التبعيات. لا تقم بتشغيل ما يلي: مراجعات التعليمات البرمجية العامة، أو المساعدة في إصلاح الأخطاء، أو تطوير الميزات، أو إعادة البناء، أو تحسين الأداء، أو المهام غير المتعلقة بالأمان.
