Effectuez des audits de sécurité complets sur les bases de code Node.js, JavaScript et TypeScript. Analyse le code source à la recherche des 10 principales vulnérabilités de l'OWASP, des modèles non sécurisés, des risques de dépendance et génère un rapport d'audit Markdown priorisé avec des niveaux de gravité (Critique, Élevé, Moyen, Faible), des citations de code et des conseils de remédiation. À utiliser lorsqu'on vous demande d'auditer, d'examiner, d'analyser ou de vérifier la sécurité d'un projet Node.js/JS/TS, de rechercher des vulnérabilités, de vérifier la conformité OWASP ou de générer un rapport de sécurité. Se déclenche également sur : examen de sécurité, pentest, analyse de vulnérabilités, mon code est-il sécurisé, vérification de l'injection, vérification des dépendances. Ne déclenchez PAS pour : les révisions générales du code, l'assistance à la correction de bogues, le développement de fonctionnalités, la refactorisation, l'optimisation des performances ou les tâches non liées à la sécurité.
