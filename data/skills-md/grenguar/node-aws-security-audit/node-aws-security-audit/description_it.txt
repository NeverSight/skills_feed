Esegui controlli di sicurezza completi sulle codebase Node.js, JavaScript e TypeScript. Esegue la scansione del codice sorgente per individuare le 10 principali vulnerabilità OWASP, modelli non sicuri, rischi di dipendenza e genera un rapporto di controllo Markdown con priorità con livelli di gravità (critico, alto, medio, basso), citazioni di codice e indicazioni per la risoluzione. Da utilizzare quando viene richiesto di verificare, rivedere, scansionare o verificare la sicurezza di un progetto Node.js/JS/TS, individuare vulnerabilità, verificare la conformità OWASP o generare un rapporto sulla sicurezza. Si attiva anche su: revisione della sicurezza, pentest, scansione delle vulnerabilità, il mio codice è sicuro, controllo dell'iniezione, controllo delle dipendenze. NON attivare per: revisioni generali del codice, assistenza per la correzione di bug, sviluppo di funzionalità, refactoring, ottimizzazione delle prestazioni o attività non di sicurezza.
