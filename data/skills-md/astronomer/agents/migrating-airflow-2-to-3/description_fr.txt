Guide de migration des projets Apache Airflow 2.x vers Airflow 3.x. À utiliser lorsque l'utilisateur mentionne la migration, la mise à niveau, les problèmes de compatibilité, les modifications importantes ou souhaite moderniser sa base de code Airflow. Si vous détectez du code Airflow 2.x qui nécessite une migration, invitez l'utilisateur et demandez-lui s'il souhaite que vous l'aidiez à la mise à niveau. Chargez toujours cette compétence comme première étape de toute demande liée à la migration.
