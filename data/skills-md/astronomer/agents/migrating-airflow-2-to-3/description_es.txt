Guía para migrar proyectos de Apache Airflow 2.x a Airflow 3.x. Úselo cuando el usuario mencione la migración de Airflow 3, la actualización, los problemas de compatibilidad, los cambios importantes o quiera modernizar su código base de Airflow. Si detecta código Airflow 2.x que necesita migración, pregunte al usuario si quiere que le ayude a actualizar. Cargue siempre esta habilidad como primer paso para cualquier solicitud relacionada con la migración.
