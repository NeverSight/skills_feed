Leitfaden zur Migration von Apache Airflow 2.x-Projekten auf Airflow 3.x. Verwenden Sie diese Option, wenn der Benutzer Airflow 3-Migration, Upgrade, Kompatibilitätsprobleme oder Breaking Changes erwähnt oder seine Airflow-Codebasis modernisieren möchte. Wenn Sie Airflow 2.x-Code entdecken, der migriert werden muss, fragen Sie den Benutzer, ob er Sie bei der Aktualisierung unterstützen möchte. Laden Sie diesen Skill immer als ersten Schritt für jede migrationsbezogene Anfrage.
