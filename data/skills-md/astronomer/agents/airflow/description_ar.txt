يدير عمليات Apache Airflow بما في ذلك إدراج DAGs واختبارها وتشغيلها وتصحيح أخطاءها، وعرض سجلات المهام، والتحقق من الاتصالات والمتغيرات، ومراقبة صحة النظام. يُستخدم عند العمل مع Airflow DAGs، أو خطوط الأنابيب، أو مسارات العمل، أو المهام، أو عندما يذكر المستخدم علامات الاختبار، أو تشغيل خطوط الأنابيب، أو تصحيح أخطاء سير العمل، أو فشل dag، أو أخطاء المهمة، أو حالة dag، أو حالة خط الأنابيب، أو قائمة dags، أو إظهار الاتصالات، أو التحقق من المتغيرات، أو صحة تدفق الهواء.
