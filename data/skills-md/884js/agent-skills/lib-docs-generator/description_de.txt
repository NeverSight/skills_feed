Generiert automatisch Fertigkeiten aus der Bibliotheksdokumentation.
Verwenden Sie diese Option, wenn Sie einen Skill aus externer Dokumentation erstellen, Dokumente in den lokalen Speicher abrufen oder Dokumentation mit Curl crawlen.
Kann auch direkt mit „ライブラリドキュメント“, „/lib-docs-generator“ aufgerufen werden.
