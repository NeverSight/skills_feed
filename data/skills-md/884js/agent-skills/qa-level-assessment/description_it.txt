Capacità di determinare il livello del test QA. Combina descrizioni di storie, differenze di codice e dialoghi per valutare in modo completo il rischio e proporre il livello di QA appropriato.
Utilizzare nelle seguenti situazioni:
- Quando vuoi determinare il livello di QA per un PR
- Quando è necessario decidere se è necessario eseguire un test o un doppio controllo
- Quando si imposta il livello per una revisione prospettica del test
Può essere richiamato anche con "giudizio QA", "livello di test", "valutazione del rischio", ecc.
