Compétence pour déterminer le niveau de test QA. Combine des descriptions d'histoires, des différences de code et des dialogues pour évaluer de manière exhaustive les risques et proposer le niveau d'assurance qualité approprié.
Utiliser dans les situations suivantes :
- Lorsque vous souhaitez déterminer le niveau d'assurance qualité d'un PR
- Lorsque vous devez décider si des tests ou une double vérification sont nécessaires
- Lors de la définition du niveau pour une révision en perspective de test
Peut également être invoqué avec « jugement AQ », « niveau de test », « évaluation des risques », etc.
