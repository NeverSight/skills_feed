Habilidad para determinar el nivel de la prueba de control de calidad. Combina descripciones de historias, diferencias de códigos y diálogos para evaluar de manera integral el riesgo y proponer el nivel de control de calidad apropiado.
Úselo en las siguientes situaciones:
- Cuando desea determinar el nivel de control de calidad para un PR
- Cuando necesite decidir si es necesario realizar una prueba o una doble verificación
- Al establecer el nivel para una revisión de perspectiva de prueba
También se puede invocar con "juicio de control de calidad", "nivel de prueba", "evaluación de riesgos", etc.
