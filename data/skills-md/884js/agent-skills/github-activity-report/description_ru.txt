Создайте портфолио отчета о вашей деятельности на GitHub (PR) для заявок на работу.
Используйте, когда пользователь спрашивает об их вкладе в GitHub, истории активности или хочет увидеть, что они создали.
Выводит файл Markdown со всеми коммитами, отсортированными по дате.
Также может быть вызван с помощью «GitHub活動», «コミット履歴», «作ったもの一覧», «ポートフォリオ».
