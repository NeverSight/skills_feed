Erstellen Sie einen Portfoliobericht Ihrer GitHub-Aktivitäten (PRs) für Bewerbungen.
Verwenden Sie diese Option, wenn der Benutzer nach seinen GitHub-Beiträgen oder seinem Aktivitätsverlauf fragt oder sehen möchte, was er erstellt hat.
Gibt eine Markdown-Datei mit allen Commits sortiert nach Datum aus.
Kann auch mit „GitHub活動“, „コミット履歴“, „作ったもの一覧“, „ポートフォリオ“ aufgerufen werden.
