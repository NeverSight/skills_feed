(Projekt – Fähigkeit) Generieren Sie KI-Bilder mit der Volcengine Jimeng API 4.0. Verwenden Sie diese Option, wenn Benutzer die Bildgenerierung über Texteingabeaufforderungen, die Bild-zu-Bild-Bearbeitung oder die Stapelbilderstellung anfordern. Zu den Auslösern gehören „Bild generieren“, „Bild erstellen“, „KI-Bild“, „Jimeng“, „Seedream“ oder jede Anfrage, die eine KI-gestützte Bilderstellung aus Beschreibungen beinhaltet.
