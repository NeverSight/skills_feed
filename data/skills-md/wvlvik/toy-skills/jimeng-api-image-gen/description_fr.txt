(projet - Compétence) Générez des images IA à l'aide de Volcengine Jimeng API 4.0. À utiliser lorsque les utilisateurs demandent la génération d'images à partir d'invites de texte, d'édition d'image à image ou de création d'images par lots. Les déclencheurs incluent « générer une image », « créer une image », « image IA », « Jimeng », « Seedream » ou toute demande impliquant la création d'image alimentée par l'IA à partir de descriptions.
