Cree interfaces frontales distintivas y de calidad de producción con alta calidad de diseño. Utilice esta habilidad cuando el usuario solicite crear componentes, páginas o aplicaciones web, O cuando proporcione capturas de pantalla/imágenes/diseños para replicar o inspirarse. Para las entradas de capturas de pantalla, primero extrae las pautas de diseño utilizando el análisis multimodal de IA y luego implementa el código siguiendo esas pautas. Genera código creativo y pulido que evita la estética genérica de la IA.
