Créez des expériences Web 3D immersives avec Three.js - bibliothèque WebGL/WebGPU pour les scènes, les caméras, les géométries, les matériaux, les lumières, les animations, les chargeurs, le post-traitement, les shaders (y compris TSL basé sur des nœuds), le calcul, la physique, la VR/XR et le rendu avancé. À utiliser lors de la création de visualisations 3D, de jeux, de graphiques interactifs, de visualisations de données, de configurateurs de produits, de présentations architecturales ou d'applications WebGL/WebGPU. Couvre OrbitControls, le chargement GLTF/FBX, les matériaux PBR, le mappage d'ombres, les effets de post-traitement (bloom, SSAO, SSR), les shaders personnalisés, l'instanciation, le LOD, les systèmes d'animation et WebXR.
