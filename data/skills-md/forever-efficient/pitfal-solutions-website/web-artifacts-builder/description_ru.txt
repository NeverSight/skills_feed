Набор инструментов для создания сложных многокомпонентных HTML-артефактов claude.ai с использованием современных веб-технологий внешнего интерфейса (React, Tailwind CSS, shadcn/ui). Используйте для сложных артефактов, требующих управления состоянием, маршрутизации или компонентов shadcn/ui, а не для простых однофайловых артефактов HTML/JSX.
