Analyse les modifications de Git et aide à créer des demandes d'extraction complètes. À utiliser lorsque l'utilisateur souhaite créer un PR, examiner les modifications avant le PR ou a besoin d'aide pour rédiger des descriptions de PR. Se déclenche sur des expressions telles que « créer un PR », « faire une pull request », « rédiger une description du PR », « ce qui a changé dans cette branche », « préparer un PR ».
