Анализирует изменения git и помогает создавать комплексные запросы на включение. Используйте, когда пользователь хочет создать PR, просмотреть изменения перед PR или ему нужна помощь в составлении описаний PR. Срабатывает на такие фразы, как «создать PR», «сделать запрос на включение», «проект описания PR», «что изменилось в этой ветке», «подготовить PR».
