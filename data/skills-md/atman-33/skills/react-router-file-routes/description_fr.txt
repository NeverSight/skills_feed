Guide de référence pour les conventions de routage basées sur les fichiers de React Router v7. À utiliser lors de la création ou de l'organisation de routes dans des projets React Router v7, pour nommer des fichiers de route ou pour travailler avec des segments dynamiques, des routes imbriquées, des mises en page sans chemin, des segments facultatifs, des routes splat et des caractères spéciaux dans le routage basé sur des fichiers.
