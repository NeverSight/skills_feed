الدليل المرجعي لاتفاقيات التوجيه المستندة إلى الملفات React Router v7. يُستخدم عند إنشاء المسارات أو تنظيمها في مشاريع React Router v7، أو تسمية ملفات المسار، أو العمل مع المقاطع الديناميكية، والمسارات المتداخلة، والتخطيطات غير المسارية، والمقاطع الاختيارية، والمسارات المسطحة، والأحرف الخاصة في التوجيه المستند إلى الملف.
