Guida di riferimento per le convenzioni di routing basato su file di React Router v7. Da utilizzare durante la creazione o l'organizzazione di percorsi in progetti React Router v7, la denominazione di file di percorso o l'utilizzo di segmenti dinamici, percorsi nidificati, layout senza percorso, segmenti opzionali, percorsi splat e caratteri speciali nel routing basato su file.
