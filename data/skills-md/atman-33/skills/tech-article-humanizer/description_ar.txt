قم بتحويل مسودات المقالات الفنية أو المواد المصدرية إلى مقالات تقنية يابانية عالية الجودة شبيهة بالبشر. استخدم هذه المهارة عندما يريد المستخدم إنشاء مقالات فنية أو إعادة كتابتها أو إضفاء الطابع الإنساني عليها (خاصة حول TypeScript أو JavaScript أو React أو موضوعات الواجهة الأمامية) باتباع أنماط محددة من الكتابة البشرية وإرشادات الأسلوب. تتضمن المشغلات طلبات مثل "記事を人間風に" أو "مقالة تقنية を生成" أو "إضفاء طابع إنساني على هذه المقالة" أو تقديم مواد مصدر المقالة.
