Синхронизируйте агенты GitHub Copilot для рабочего процесса OpenSpec на основе FF15 (версия, отличная от MCP). В команду входят Noctis (оркестратор + создатель OpenSpec), Iris (управление проблемами), Gladiolus (реализация), Prompto (качество кода), Ignis (документация) и Lunafreya (создание связей с общественностью). Поддерживает несколько рабочих процессов OpenSpec (создать, продолжить, ff, изучить, проверить).
