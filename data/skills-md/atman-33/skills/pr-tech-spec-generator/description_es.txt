Genere especificaciones técnicas completas a partir de solicitudes de extracción de GitHub. Úselo cuando los usuarios proporcionen una URL de relaciones públicas de GitHub y soliciten documentación, especificaciones técnicas o análisis de cambios. Analiza las confirmaciones y diferencias de relaciones públicas utilizando comandos gh CLI y git para producir documentación estructurada que cubra la estructura de carpetas, cambios en el esquema de base de datos, definiciones de API, cambios de prueba y actualizaciones de dependencia.
