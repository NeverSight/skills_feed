Genera specifiche tecniche complete dalle richieste pull di GitHub. Da utilizzare quando gli utenti forniscono un URL PR GitHub e richiedono documentazione, specifiche tecniche o analisi delle modifiche. Analizza i commit e le differenze PR utilizzando la CLI gh e i comandi git per produrre documentazione strutturata che copre la struttura delle cartelle, le modifiche allo schema DB, le definizioni API, le modifiche ai test e gli aggiornamenti delle dipendenze.
