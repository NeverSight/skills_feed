Utilisez cette compétence lorsque l'utilisateur demande à sauvegarder, mémoriser, rappeler ou organiser des souvenirs. Se déclenche sur des expressions telles que « souviens-toi de ceci », « enregistre ceci », « note ceci », « de quoi avons-nous discuté de... », « vérifie tes notes », « nettoie les souvenirs ». Utilisez également de manière proactive lorsque vous découvrez des découvertes précieuses qui méritent d'être préservées.
