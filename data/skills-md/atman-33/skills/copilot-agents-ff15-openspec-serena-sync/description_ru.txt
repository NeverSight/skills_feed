Синхронизируйте агенты GitHub Copilot для рабочего процесса OpenSpec, основанного на FF15. В команду входят Noctis (оркестратор + создатель OpenSpec), Iris (управление проблемами), Gladiolus (реализация), Prompto (качество кода), Ignis (документация) и Lunafreya (создание связей с общественностью). Поддерживает несколько рабочих процессов OpenSpec (создать, продолжить, ff, изучить, проверить).
