Automatiza las interacciones del navegador para pruebas web, llenado de formularios, capturas de pantalla y extracción de datos mediante playwright-cli. Úselo cuando el usuario necesite navegar por sitios web, interactuar con páginas web, completar formularios, tomar capturas de pantalla, probar aplicaciones web, extraer información de páginas web, simular solicitudes de red, administrar sesiones de navegador o generar código de prueba.
