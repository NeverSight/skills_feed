Automatizza le interazioni del browser per test web, compilazione di moduli, screenshot ed estrazione di dati utilizzando playwright-cli. Utilizzare quando l'utente deve navigare in siti Web, interagire con pagine Web, compilare moduli, acquisire schermate, testare applicazioni Web, estrarre informazioni da pagine Web, simulare richieste di rete, gestire sessioni del browser o generare codice di test.
