Règles UX frontend pour les dApps Ethereum qui empêchent les bugs d'interface utilisateur des agents IA les plus courants. Modèles obligatoires pour les boutons en chaîne, les flux d'approbation des jetons, l'affichage des adresses, les valeurs USD, la configuration RPC et les métadonnées de pré-publication. Construit autour de Scaffold-ETH 2, mais les modèles s'appliquent à n'importe quelle interface Ethereum. À utiliser lors de la création d’une interface dApp.
