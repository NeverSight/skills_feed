Habilidad integral de investigación en Wikipedia para extraer información verificable respaldada por citas en formato JSON estructurado optimizado para el consumo de IA y canales de verificación.
DESENCADENADORES OBLIGATORIOS: investigación de Wikipedia, citas de Wikipedia, verificar afirmaciones de Wikipedia, extraer referencias de Wikipedia, verificar datos con fuentes, investigar temas con citas, recopilar información verificable, API de Wikipedia, recopilar datos de investigación, crear gráficos de conocimiento, extracción de entidades, línea de tiempo de investigación.
Úselo cuando: (1) Investigar temas que necesitan verificación de fuente, (2) Extraer contenido de Wikipedia CON sus citas subyacentes, (3) Construir conjuntos de datos de investigación para la verificación de IA, (4) Recopilar datos bibliográficos estructurados, (5) Rastrear afirmaciones hasta fuentes originales, (6) Construir gráficos de conocimiento de entidades y relaciones, (7) Construir líneas de tiempo a partir de datos biográficos/históricos, (8) Validación cruzada de hechos en múltiples fuentes
