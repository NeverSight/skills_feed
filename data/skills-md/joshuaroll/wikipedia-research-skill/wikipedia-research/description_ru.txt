Комплексные навыки исследования Википедии для извлечения поддающейся проверке, подтвержденной цитатами информации в структурированном формате JSON, оптимизированном для использования ИИ и конвейеров проверки.
ОБЯЗАТЕЛЬНЫЕ ТРИГГЕРЫ: исследование Википедии, цитаты из Википедии, проверка утверждений Википедии, извлечение ссылок из Википедии, проверка фактов с источниками, тема исследования с цитатами, сбор проверяемой информации, API Википедии, сбор исследовательских данных, построение графика знаний, извлечение сущностей, график исследования.
Используйте, когда: (1) исследуете темы, требующие проверки источника, (2) извлекаете контент Википедии вместе с лежащими в его основе цитатами, (3) создаете наборы исследовательских данных для проверки ИИ, (4) собираете структурированные библиографические данные, (5) отслеживаете претензии к первоисточникам, (6) строите графики знаний сущностей и отношений, (7) строите временные рамки на основе биографических/исторических данных, (8) перекрестную проверку фактов из нескольких источников
