Bloomerang CRM API-Integrationsreferenz zum Erstellen von Spenderverwaltungsfunktionen. Wird zum Schreiben von Code verwendet, der mit der Bloomerang-API interagiert, einschließlich Abruffunktionen, Normalisierern oder einer Backend-Integration mit Bloomerang-Spenderdaten. Deckt Bestandteile, Transaktionen, Zusagen, Kampagnen, Einsprüche, Interaktionen, Aufgaben und Beziehungen ab.
