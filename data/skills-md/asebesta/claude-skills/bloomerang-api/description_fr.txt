Référence d'intégration de l'API Bloomerang CRM pour créer des fonctionnalités de gestion des donateurs. À utiliser lors de l'écriture de code qui interagit avec l'API Bloomerang, y compris les récupérateurs, les normalisateurs ou toute intégration back-end avec les données des donateurs Bloomerang. Couvre les constituants, les transactions, les engagements, les campagnes, les appels, les interactions, les tâches et les relations.
