مرجع تكامل Bloomerang CRM API لبناء ميزات إدارة الجهات المانحة. يُستخدم عند كتابة تعليمات برمجية تتفاعل مع واجهة برمجة تطبيقات Bloomerang، بما في ذلك أدوات الجلب أو التسوية أو أي تكامل خلفي مع بيانات الجهات المانحة لـ Bloomerang. يغطي المكونات والمعاملات والتعهدات والحملات والاستئنافات والتفاعلات والمهام والعلاقات.
