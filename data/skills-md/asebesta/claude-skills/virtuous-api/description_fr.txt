Référence d'intégration de l'API Virtuous CRM pour créer des fonctionnalités de gestion des donateurs. À utiliser lors de l'écriture de code qui interagit avec l'API Virtuous, y compris les récupérateurs, les normalisateurs ou toute intégration back-end avec les données des donateurs Virtuous. Couvre les contacts, les individus, les cadeaux, les projets, les campagnes, les événements, les subventions, les tâches et les webhooks.
