Virtuous CRM API-Integrationsreferenz zum Erstellen von Spenderverwaltungsfunktionen. Wird zum Schreiben von Code verwendet, der mit der Virtuous-API interagiert, einschließlich Fetchern, Normalisierern oder einer Backend-Integration mit Virtuous-Spenderdaten. Deckt Kontakte, Einzelpersonen, Geschenke, Projekte, Kampagnen, Veranstaltungen, Zuschüsse, Aufgaben und Webhooks ab.
