مرجع تكامل CRM API الفاضل لبناء ميزات إدارة الجهات المانحة. يُستخدم عند كتابة تعليمات برمجية تتفاعل مع واجهة برمجة تطبيقات Virtious، بما في ذلك أدوات الجلب أو أدوات التسوية أو أي تكامل خلفي مع بيانات الجهات المانحة الفاضلة. يغطي جهات الاتصال والأفراد والهدايا والمشاريع والحملات والأحداث والمنح والمهام وخطافات الويب.
