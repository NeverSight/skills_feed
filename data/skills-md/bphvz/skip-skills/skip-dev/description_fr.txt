Conseils d'experts sur la création, le débogage et le test d'applications et de frameworks iOS/Android multiplateformes avec Skip (skip.dev). À utiliser lorsque les développeurs mentionnent : (1) Skip, Skip.dev, skip-tools ou SkipStack, (2) créer une application multiplateforme iOS+Android à partir de Swift/SwiftUI, (3) les modes Skip Fuse (natif) ou Skip Lite (transpilé), (4) transpiler Swift vers Kotlin, (5) ponter SwiftUI vers Jetpack Compose, (6) configuration skip.yml, (7) débogage des builds Android à partir de Xcode, (8) Ignorer les commandes CLI (ignorer create/init/test/export), (9) compilation conditionnelle avec #if SKIP ou #if os(Android), (10) Ignorer les commentaires (SKIP INSERT/REPLACE/DECLARE/NOWARN), (11) relier le code Swift et Kotlin, (12) Ignorer les dépendances du module ou la configuration Android Gradle.
