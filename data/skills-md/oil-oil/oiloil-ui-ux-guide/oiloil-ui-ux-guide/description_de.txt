Moderne, saubere UI/UX-Anleitung + Überprüfungskompetenz. Verwenden Sie es, wenn Sie umsetzbare UX/UI-Empfehlungen, Designprinzipien oder eine Checkliste zur Designüberprüfung für neue Funktionen oder bestehende Systeme (Web/App) benötigen. Konzentrieren Sie sich auf CRAP (Kontrast/Wiederholung/Ausrichtung/Nähe) plus aufgabenorientiertes UX, Informationsarchitektur, Feedback und Systemstatus, Konsistenz, Erschwinglichkeit, Fehlervermeidung/-behebung und kognitive Belastung. Erzwingen Sie einen modernen, minimalistischen Stil (sauber, geräumig, typografieorientiert), reduzieren Sie unnötige Kopien, verbieten Sie Emojis als Symbole und empfehlen Sie intuitive, raffinierte Symbole aus einem konsistenten Symbolsatz.
