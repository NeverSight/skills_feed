À utiliser pour l'installation, la mise à niveau, l'authentification et l'utilisation de l'interface de ligne de commande Atlassian (ACLI, « acli ») pour Atlassian Cloud. Les déclencheurs incluent les tâches Jira CLI (éléments de travail, projets, tableaux, sprints), les tâches Atlassian Admin CLI (administrateur utilisateur/organisation via la clé API), l'utilisation de Rovo Dev CLI, l'activation de l'achèvement du shell, l'utilisation des scripts/CI et le dépannage des erreurs « acli » ou des ID de trace.
