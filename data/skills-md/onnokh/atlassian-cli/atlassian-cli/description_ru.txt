Используйте для установки, обновления, аутентификации и использования интерфейса командной строки Atlassian (ACLI, `acli`) для Atlassian Cloud. Триггеры включают в себя задачи CLI Jira (рабочие элементы, проекты, доски, спринты), задачи Atlassian Admin CLI (администратор пользователя/организации через ключ API), использование Rovo Dev CLI, включение завершения оболочки, использование сценариев/CI и устранение ошибок acli или идентификаторов трассировки.
