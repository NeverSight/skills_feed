Da utilizzare per l'installazione, l'aggiornamento, l'autenticazione e l'utilizzo dell'Atlassian Command Line Interface (ACLI, `acli`) per Atlassian Cloud. I trigger includono attività CLI di Jira (elementi di lavoro, progetti, schede, sprint), attività CLI di Atlassian Admin (amministratore utente/organizzazione tramite chiave API), utilizzo della CLI di Rovo Dev, abilitazione del completamento della shell, utilizzo di script/CI e risoluzione dei problemi di errori "acli" o ID di traccia.
