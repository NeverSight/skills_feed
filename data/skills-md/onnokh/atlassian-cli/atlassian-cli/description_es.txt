Úselo para instalar, actualizar, autenticar y usar la interfaz de línea de comandos de Atlassian (ACLI, `acli`) para Atlassian Cloud. Los activadores incluyen tareas de Jira CLI (elementos de trabajo, proyectos, tableros, sprints), tareas de CLI de administración de Atlassian (administrador de usuario/organización mediante clave API), uso de Rovo Dev CLI, habilitación de finalización de shell, uso de secuencias de comandos/CI y resolución de problemas de errores de "acli" o ID de seguimiento.
