Zur Installation, Aktualisierung, Authentifizierung und Verwendung des Atlassian Command Line Interface (ACLI, „acli“) für Atlassian Cloud. Zu den Auslösern gehören Jira CLI-Aufgaben (Arbeitselemente, Projekte, Boards, Sprints), Atlassian Admin CLI-Aufgaben (Benutzer-/Organisationsadministrator über API-Schlüssel), Rovo Dev CLI-Nutzung, Aktivierung der Shell-Vervollständigung, Skripting/CI-Nutzung und Fehlerbehebung bei „Acli“-Fehlern oder Trace-IDs.
