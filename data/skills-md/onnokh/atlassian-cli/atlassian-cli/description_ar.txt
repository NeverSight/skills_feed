يُستخدم للتثبيت والترقية والمصادقة واستخدام Atlassian Command Line Interface (ACLI، `acli`) لـ Atlassian Cloud. تتضمن المشغلات مهام Jira CLI (عناصر العمل، والمشاريع، واللوحات، والسباقات السريعة)، ومهام Atlassian Admin CLI (مسؤول المستخدم/المؤسسة عبر مفتاح API)، واستخدام Rovo Dev CLI، وتمكين إكمال الصدفة، والبرمجة النصية/استخدام CI، واستكشاف أخطاء `acli` أو معرفات التتبع وإصلاحها.
