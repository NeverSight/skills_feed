Обрабатывайте документы с помощью API Nutrient DWS. Используйте этот навык, когда пользователь хочет конвертировать документы (PDF, DOCX, XLSX, PPTX, HTML, изображения), извлекать текст или таблицы из PDF-файлов, сканировать документы OCR, редактировать конфиденциальную информацию (PII, SSN, электронные письма, кредитные карты), добавлять водяные знаки, подписывать PDF-файлы цифровой подписью, заполнять PDF-формы или проверять использование кредитов API. Активируется по ключевым словам: PDF, документ, преобразование, извлечение, распознавание текста, редактирование, водяной знак, подпись, объединение, сжатие, заполнение формы, обработка документа.
