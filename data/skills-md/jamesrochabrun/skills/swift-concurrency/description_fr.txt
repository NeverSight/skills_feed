Guide pour la création, l'audit et la refactorisation du code Swift à l'aide de modèles de concurrence modernes (Swift 6+). Cette compétence doit être utilisée lorsque vous travaillez avec async/wait, des tâches, des acteurs, MainActor, des types Sendable, des domaines d'isolation ou lors de la migration du code de rappel/combinaison existant vers une concurrence structurée. Couvre les paramètres de concurrence accessible, les paramètres isolés et les pièges courants.
