Этот навык следует использовать, когда пользователи хотят направлять запросы LLM различным поставщикам ИИ (OpenAI, Grok/xAI, Groq, DeepSeek, OpenRouter) с помощью SwiftOpenAI-CLI. Используйте этот навык, когда пользователи просят «использовать Grok», «спросить Grok», «использовать Groq», «спросить DeepSeek» или любой аналогичный запрос для запроса конкретного поставщика LLM в режиме агента.
