Cette compétence doit être utilisée lorsque les utilisateurs souhaitent acheminer les requêtes LLM vers différents fournisseurs d'IA (OpenAI, Grok/xAI, Groq, DeepSeek, OpenRouter) à l'aide de SwiftOpenAI-CLI. Utilisez cette compétence lorsque les utilisateurs demandent « utiliser grok », « demander à grok », « utiliser groq », « demander à deepseek » ou toute demande similaire pour interroger un fournisseur LLM spécifique en mode agent.
