Шаблоны игрового аудиодизайна для создания звуковых эффектов и звука пользовательского интерфейса. Используйте при разработке звуков для игр, написании звуковых подсказок ИИ (ElevenLabs и т. д.), создании звуков обратной связи или указании звука для способностей/интерфейса. Включает психологические принципы из Kind Games, иерархию громкости, предотвращение маскировки частоты и быструю разработку для генерации звука с помощью ИИ.
