Animations Anime.js 4.0 pour les composants Web : glisser-déposer, retour de clic, échanges, mouvement annulable. À utiliser lors de l'ajout d'animations, d'interactions de glissement, de commentaires visuels ou de mouvements sur des éléments personnalisés. Se combine avec l'architecture de composants Web pour le nettoyage du cycle de vie.
