Modelli testati in produzione per audio browser con tolleranza agli errori con supporto rapido a ritardo zero. Da utilizzare quando si implementano effetti sonori, musica di sottofondo, feedback vocale o qualsiasi riproduzione audio nelle applicazioni web. Copre il singleton AudioContext, il precaricamento, cloneNode per il fuoco rapido, la gestione della riproduzione automatica e gli effetti API Web Audio.
