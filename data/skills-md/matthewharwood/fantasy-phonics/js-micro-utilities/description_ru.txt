Утилиты JavaScript с нулевой зависимостью, использующие собственные API и микропакеты just-*. Используйте при манипулировании объектами, массивами, строками, числами или функциями. Сначала сканируйте таблицы на наличие собственного решения (обратные кавычки), возвращайтесь к пакету just-* только при необходимости. Предпочитайте нативный код зависимостям.
