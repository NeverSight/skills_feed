اتفاقيات تطوير واجهة برمجة تطبيقات الويب لمشاريع Adynato. يغطي مسارات واجهة برمجة التطبيقات والبرامج الوسيطة والمصادقة ومعالجة الأخطاء والتحقق من الصحة وتنسيقات الاستجابة للواجهات الخلفية لـ Next.js وNode.js. يُستخدم عند إنشاء أو تعديل نقاط نهاية واجهة برمجة التطبيقات أو إجراءات الخادم أو منطق الواجهة الخلفية.
