Verwenden Sie diese Fähigkeit, wenn Sie mit arbeitsspezifischen Punktdateien arbeiten, die von yadm-work verwaltet werden. Dazu gehören das Abrufen von Remote-Änderungen, das Festschreiben und Übertragen von Work-Dotfile-Änderungen, das Ändern von Arbeitskonfigurationsdateien, das Anzeigen nachverfolgter Arbeitsdateien, das Lösen von Zusammenführungskonflikten und die Pflege des Work-Dotfiles-Repositorys bei Spotify GHE. Löst auf „work dotfiles“, „yadm-work“, „spotify dotfiles“ oder arbeitsbezogener Konfigurationsverwaltung aus.
