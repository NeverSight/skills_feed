Utilice esta habilidad cuando trabaje con archivos de puntos personales administrados por yadm. Esto incluye realizar cambios remotos, confirmar y enviar cambios en archivos de puntos, modificar archivos de configuración (shell, editor, terminal, git, etc.), ver archivos rastreados, resolver conflictos de fusión y mantener el repositorio de archivos de puntos. Para archivos de puntos de trabajo, utilice la habilidad de administrar archivos de puntos de trabajo.
