Используйте этот навык при работе с личными точечными файлами, управляемыми yadm. Сюда входит получение удаленных изменений, фиксация и отправка изменений точечных файлов, изменение файлов конфигурации (оболочка, редактор, терминал, git и т. д.), просмотр отслеживаемых файлов, разрешение конфликтов слияния и поддержка репозитория точечных файлов. Для рабочих точечных файлов вместо этого используйте навык управления рабочими точечными файлами.
