تنفيذ الميزات وكتابة التعليمات البرمجية وإعداد مشاريع الويب CE.SDK.
يغطي React وVue.js وSvelte وAngular وElectron وVanilla JavaScript وNode.js وNuxt.js وNext.js وSvelteKit.

يُستخدم عندما يطلب المستخدم التنفيذ أو الإنشاء أو الإضافة أو البناء أو الإعداد أو التكامل
شيء مع CE.SDK للويب. يتم تشغيله بواسطة "ساعدني في الإضافة" و"الإعداد" و"إنشاء ملف".
محرر الصور"، أو "إنشاء أداة تصميم". يغطي محرري الصور والفيديو والتصميم.

ليس للبحث عن المستندات الموجودة (استخدم docs-{framework}) أو شرح المفاهيم
(استخدم الشرح).

<مثال>
السياق: يريد المستخدم إنشاء تطبيق ويب باستخدام CE.SDK
المستخدم: "ساعدني في إعداد CE.SDK في مشروعي"
المساعد: "سأستخدم /cesdk:build للمساعدة في إعداد هذا."
</مثال>

<مثال>
السياق: يريد المستخدم إضافة ميزة محددة
المستخدم: "أضف تراكبات نصية إلى محرر الصور الخاص بي"
المساعد: "اسمح لي باستخدام /cesdk:build لتنفيذ تراكبات النص."
</مثال>
