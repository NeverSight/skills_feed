기능을 구현하고, 코드를 작성하고, CE.SDK 웹 프로젝트를 설정하세요.
React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit을 다룹니다.

사용자가 구현, 생성, 추가, 빌드, 설정 또는 통합을 요청할 때 사용합니다.
웹용 CE.SDK를 사용하면 됩니다. "추가 도움말", "설정", "빌드"에 의해 트리거됨
사진 편집기" 또는 "디자인 도구 만들기". 사진, 비디오, 디자인 편집자를 다룹니다.

기존 문서(docs-{framework} 사용) 또는 개념 설명을 검색하기 위한 것이 아닙니다.
(설명을 사용하세요).

<예>
컨텍스트: 사용자가 CE.SDK를 사용하여 웹 앱을 구축하려고 합니다.
사용자: "내 프로젝트에서 CE.SDK를 설정하는 데 도움을 주세요."
조수: "이 설정을 돕기 위해 /cesdk:build를 사용하겠습니다."
</예>

<예>
컨텍스트: 사용자가 특정 기능을 추가하려고 합니다.
사용자: "내 이미지 편집기에 텍스트 오버레이 추가"
조수: "/cesdk:build를 사용하여 텍스트 오버레이를 구현하겠습니다."
</예>
