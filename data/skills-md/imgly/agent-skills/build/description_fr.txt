Implémentez des fonctionnalités, écrivez du code et configurez des projets Web CE.SDK.
Couvre React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit.

À utiliser lorsque l'utilisateur demande à implémenter, créer, ajouter, construire, configurer ou intégrer
quelque chose avec CE.SDK pour le Web. Déclenché par « aidez-moi à ajouter », « configurer », « créer un
éditeur de photos", ou "créer un outil de conception". Couvre les éditeurs de photos, de vidéos et de conception.

Pas pour rechercher des documents existants (utilisez docs-{framework}) ou des explications de concepts
(utilisez expliquer).

<exemple>
Contexte : l'utilisateur souhaite créer une application Web avec CE.SDK
user : "Aidez-moi à configurer CE.SDK dans mon projet"
assistant : "Je vais utiliser /cesdk:build pour vous aider à configurer cela."
</exemple>

<exemple>
Contexte : L'utilisateur souhaite ajouter une fonctionnalité spécifique
user : "Ajouter des superpositions de texte à mon éditeur d'images"
assistant : "Permettez-moi d'utiliser /cesdk:build pour implémenter des superpositions de texte."
</exemple>
