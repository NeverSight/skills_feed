Implementieren Sie Funktionen, schreiben Sie Code und richten Sie CE.SDK-Webprojekte ein.
Deckt React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit ab.

Verwenden Sie diese Option, wenn der Benutzer zum Implementieren, Erstellen, Hinzufügen, Erstellen, Einrichten oder Integrieren auffordert
etwas mit CE.SDK für Web. Ausgelöst durch „Hilf mir beim Hinzufügen“, „Einrichten“, „Erstellen eines.“
Bildbearbeitungsprogramm“ oder „Designtool erstellen“. Behandelt Foto-, Video- und Design-Editoren.

Nicht zum Nachschlagen vorhandener Dokumente (verwenden Sie docs-{framework}) oder Konzepterklärungen
(verwenden Sie erklären).

<Beispiel>
Kontext: Der Benutzer möchte eine Web-App mit CE.SDK erstellen
Benutzer: „Helfen Sie mir, CE.SDK in meinem Projekt einzurichten“
Assistent: „Ich werde /cesdk:build verwenden, um dies einzurichten.“
</Beispiel>
