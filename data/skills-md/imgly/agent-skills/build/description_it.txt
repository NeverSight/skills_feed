Implementa funzionalità, scrivi codice e configura progetti Web CE.SDK.
Copre React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit.

Utilizzare quando l'utente chiede di implementare, creare, aggiungere, creare, configurare o integrare
qualcosa con CE.SDK per il Web. Attivato da "aiutami ad aggiungere", "imposta", "crea a
editor di foto" o "crea uno strumento di progettazione". Copre editor di foto, video e design.

Non per cercare documenti esistenti (usare docs-{framework}) o spiegazioni di concetti
(usa spiegare).

<esempio>
Contesto: l'utente desidera creare un'app Web con CE.SDK
utente: "Aiutami a configurare CE.SDK nel mio progetto"
assistente: "Utilizzerò /cesdk:build per aiutarmi a configurarlo."
</esempio>

<esempio>
Contesto: l'utente desidera aggiungere una funzionalità specifica
utente: "Aggiungi sovrapposizioni di testo al mio editor di immagini"
assistente: "Permettimi di usare /cesdk:build per implementare le sovrapposizioni di testo."
</esempio>
