Implemente funciones, escriba código y configure proyectos web CE.SDK.
Cubre React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit.

Úselo cuando el usuario solicite implementar, crear, agregar, construir, configurar o integrar.
algo con CE.SDK para Web. Activado por "ayúdame a agregar", "configurar", "crear un
editor de fotos" o "crear una herramienta de diseño". Cubre editores de fotografía, vídeo y diseño.

No para buscar documentos existentes (use docs-{framework}) o explicaciones de conceptos
(use explicar).

<ejemplo>
Contexto: el usuario quiere crear una aplicación web con CE.SDK
usuario: "Ayúdame a configurar CE.SDK en mi proyecto"
asistente: "Usaré /cesdk:build para ayudar a configurar esto".
</ejemplo>
