實作功能、編寫程式碼並設定 CE.SDK Web 專案。
涵蓋 React、Vue.js、Svelte、Angular、Electron、Vanilla JavaScript、Node.js、Nuxt.js、Next.js、SvelteKit。

當使用者要求實施、建立、新增、建置、設定或整合時使用
CE.SDK for Web 的一些東西。由「幫我添加」、「設定」、「建構一個
照片編輯器”或“創建設計工具”。涵蓋照片、影片和設計編輯。

不適用於尋找現有文件（使用 docs-{framework}）或概念解釋
（使用解釋）。

<範例>
上下文：使用者想要使用 CE.SDK 建立 Web 應用程式
使用者：“幫我在我的專案中設定 CE.SDK”
助理：“我將使用 /cesdk:build 來幫助設定。”
</範例>

<範例>
上下文：使用者想要新增特定功能
使用者：“將文字疊加添加到我的圖像編輯器”
助理：“讓我使用 /cesdk:build 來實現文字疊加。”
</範例>
