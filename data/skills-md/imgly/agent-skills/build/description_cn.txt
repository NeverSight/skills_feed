实现功能、编写代码并设置 CE.SDK Web 项目。
涵盖 React、Vue.js、Svelte、Angular、Electron、Vanilla JavaScript、Node.js、Nuxt.js、Next.js、SvelteKit。

当用户要求实施、创建、添加、构建、设置或集成时使用
CE.SDK for Web 的一些东西。由“帮我添加”、“设置”、“构建一个
照片编辑器”或“创建设计工具”。涵盖照片、视频和设计编辑。

不适用于查找现有文档（使用 docs-{framework}）或概念解释
(use explain).

<示例>
上下文：用户想要使用 CE.SDK 构建 Web 应用程序
用户：“帮我在我的项目中设置 CE.SDK”
助理：“我将使用 /cesdk:build 来帮助设置。”
</示例>

上下文：用户想要添加特定功能
用户：“将文本叠加添加到我的图像编辑器”
助理：“让我使用 /cesdk:build 来实现文本叠加。”
</示例>
