Внедряйте функции, пишите код и настраивайте веб-проекты CE.SDK.
Охватывает React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit.

Используйте, когда пользователь просит внедрить, создать, добавить, построить, настроить или интегрировать.
что-то с CE.SDK для Интернета. Запускается по фразам «помогите добавить», «настроить», «создать
фоторедактор» или «создать инструмент для дизайна». Охватывает фото-, видео- и дизайнерские редакторы.

Не для поиска существующей документации (используйте docs-{framework}) или пояснений концепций.
(используйте объяснение).

<пример>
Контекст: пользователь хочет создать веб-приложение с помощью CE.SDK.
пользователь: "Помогите мне настроить CE.SDK в моем проекте"
помощник: «Я воспользуюсь /cesdk:build, чтобы настроить это».
</пример>

<пример>
Контекст: пользователь хочет добавить определенную функцию.
пользователь: "Добавьте наложение текста в мой графический редактор"
помощник: «Позвольте мне использовать /cesdk:build для реализации наложения текста».
</пример>
