اشرح كيفية عمل ميزات الويب CE.SDK - المفاهيم والهندسة المعمارية وسير العمل.
يغطي React وVue.js وSvelte وAngular وElectron وVanilla JavaScript وNode.js وNuxt.js وNext.js وSvelteKit.

يُستخدم عندما يقول المستخدم "اشرح"، "كيف يعمل X"، "أرشدني عبر"، "ما هو"،
أو "الوصف" ويريد فهم مفهوم CE.SDK لتطوير الويب.
يُنشئ تفسيرات تخفيض السعر المخصصة باستخدام الرسوم البيانية وأمثلة التعليمات البرمجية.

ليس للبحث عن المستندات الموجودة (استخدم docs-{framework}) أو لكتابة التعليمات البرمجية (استخدم build).

<مثال>
السياق: يريد المستخدم فهم كيفية عمل طبقات النص
المستخدم: "اشرح كيفية عمل طبقات النص في CE.SDK"
المساعد: "سأستخدم /cesdk:explain لإنشاء شرح مفصل."
</مثال>

<مثال>
السياق: يحتاج المستخدم إلى شرح المفهوم
المستخدم: "كيف يعمل التسلسل الهرمي للكتل لتحرير الفيديو؟"
المساعد: "اسمح لي باستخدام /cesdk:explain لشرح التسلسل الهرمي لكتلة الفيديو."
</مثال>
