Объясните, как работают веб-функции CE.SDK — концепции, архитектура и рабочие процессы.
Охватывает React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit.

Используйте, когда пользователь говорит «объясни», «как работает X», «расскажи мне», «что такое»,
или «описать» и хочет понять концепцию CE.SDK для веб-разработки.
Создает пользовательские объяснения уценки с диаграммами и примерами кода.

Не для поиска существующей документации (используйте docs-{framework}) или написания кода (используйте сборку).

<пример>
Контекст: пользователь хочет понять, как работают текстовые слои.
пользователь: «Объясните, как работают текстовые слои в CE.SDK»
помощник: «Я использую /cesdk:explain, чтобы создать подробное объяснение».
</пример>

<пример>
Контекст: пользователю нужно объяснить концепцию.
пользователь: «Как иерархия блоков работает при редактировании видео?»
помощник: «Позвольте мне использовать /cesdk:explain, чтобы объяснить иерархию видеоблоков».
</пример>
