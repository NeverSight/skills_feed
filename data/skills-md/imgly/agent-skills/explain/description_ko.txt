CE.SDK 웹 기능의 작동 방식(개념, 아키텍처 및 워크플로)을 설명합니다.
React, Vue.js, Svelte, Angular, Electron, Vanilla JavaScript, Node.js, Nuxt.js, Next.js, SvelteKit을 다룹니다.

사용자가 "설명", "X는 어떻게 작동하나요?", "안내해 주세요", "무엇인가요?"라고 말할 때 사용하세요.
또는 "설명"하고 웹 개발을 위한 CE.SDK 개념을 이해하고 싶습니다.
다이어그램 및 코드 예제를 사용하여 사용자 정의 마크다운 설명을 생성합니다.

기존 문서를 검색하거나(docs-{framework} 사용) 코드를 작성(빌드 사용)할 필요가 없습니다.

<예>
맥락: 사용자는 텍스트 레이어의 작동 방식을 이해하고 싶어합니다.
사용자: "CE.SDK에서 텍스트 레이어가 작동하는 방식을 설명하세요."
조수: "자세한 설명을 생성하려면 /cesdk:explain을 사용하겠습니다."
</예>

<예>
맥락: 사용자는 개념 설명이 필요합니다.
사용자: "비디오 편집 시 블록 계층 구조는 어떻게 작동하나요?"
어시스턴트: "비디오 블록 계층 구조를 설명하기 위해 /cesdk:explain을 사용하겠습니다."
</예>
