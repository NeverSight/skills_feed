ابحث عن المستندات والأدلة وصفحات التكوين المرجعية لـ CE.SDK React.

يُستخدم عندما يحتاج المستخدم إلى مستندات CE.SDK لتكوين React، وتخصيص واجهة المستخدم،
خيارات التصدير أو أدلة الميزات أو تعليمات البدء. تم تشغيله أيضًا بواسطة "IMG.LY"، "CreativeEditor"،
"CE.SDK" أو "cesdk" عندما يحتاج المستخدم إلى صفحة مستند React موجودة.

ليس لكتابة التعليمات البرمجية (استخدم البناء) أو شرح المفاهيم (استخدم الشرح).

<مثال>
السياق: يسأل المستخدم عن تكوين React
المستخدم: "كيف أقوم بتكوين المحرر في React؟"
المساعد: "سأستخدم /cesdk:docs-react للبحث عن خيارات التكوين."
</مثال>

<مثال>
السياق: يحتاج المستخدم إلى إعداد مكون React
المستخدم: "كيف يمكنني تضمين CE.SDK في مكون React؟"
المساعد: "اسمح لي باستخدام /cesdk:docs-react للعثور على الوثائق ذات الصلة."
</مثال>
