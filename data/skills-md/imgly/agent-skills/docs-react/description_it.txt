Da utilizzare quando l'utente necessita di documenti CE.SDK per React: configurazione, personalizzazione dell'interfaccia utente,
opzioni di esportazione, guide alle funzionalità o istruzioni introduttive. Attivato anche da "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" quando l'utente necessita di una pagina di documenti React esistente.

Non per scrivere codice (usa build) o spiegazioni di concetti (usa spiega).
