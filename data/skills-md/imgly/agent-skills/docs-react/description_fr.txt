Recherchez les documents de référence, les guides et les pages de configuration CE.SDK React.

À utiliser lorsque l'utilisateur a besoin de documents CE.SDK pour React : configuration, personnalisation de l'interface utilisateur,
options d'exportation, guides de fonctionnalités ou instructions de démarrage. Également déclenché par "IMG.LY", "CreativeEditor",
"CE.SDK" ou "cesdk" lorsque l'utilisateur a besoin d'une page de documentation React existante.

Pas pour écrire du code (utiliser build) ou des explications de concepts (utiliser expliquer).

<exemple>
Contexte : l'utilisateur pose des questions sur la configuration de React
user : "Comment configurer l'éditeur dans React ?"
assistant : "J'utiliserai /cesdk:docs-react pour rechercher les options de configuration."
</exemple>

<exemple>
Contexte : l'utilisateur a besoin de la configuration du composant React
user : "Comment intégrer CE.SDK dans un composant React ?"
assistant : "Permettez-moi d'utiliser /cesdk:docs-react pour trouver la documentation pertinente."
</exemple>
