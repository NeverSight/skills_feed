Suchen Sie nach CE.SDK React-Referenzdokumenten, Anleitungen und Konfigurationsseiten.

Verwenden Sie diese Option, wenn der Benutzer CE.SDK-Dokumente für React benötigt – Konfiguration, UI-Anpassung,
Exportoptionen, Funktionshandbücher oder Anleitungen für den Einstieg. Wird auch durch „IMG.LY“, „CreativeEditor“ ausgelöst.
„CE.SDK“ oder „cesdk“, wenn der Benutzer eine vorhandene React-Dokumentseite benötigt.

Nicht zum Schreiben von Code (Build verwenden) oder Konzepterklärungen (Explain verwenden).

<Beispiel>
Kontext: Benutzer fragt nach der React-Konfiguration
Benutzer: „Wie konfiguriere ich den Editor in React?“
Assistent: „Ich verwende /cesdk:docs-react, um nach Konfigurationsoptionen zu suchen.“
</Beispiel>

<Beispiel>
Kontext: Der Benutzer benötigt die Einrichtung der React-Komponente
Benutzer: „Wie bette ich CE.SDK in eine React-Komponente ein?“
Assistent: „Lassen Sie mich /cesdk:docs-react verwenden, um die relevante Dokumentation zu finden.“
</Beispiel>
