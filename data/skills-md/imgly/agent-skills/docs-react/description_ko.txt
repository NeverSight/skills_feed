CE.SDK React 참조 문서, 가이드 및 구성 페이지를 찾아보세요.

사용자가 React용 CE.SDK 문서가 필요할 때 사용 — 구성, UI 사용자 정의,
내보내기 옵션, 기능 가이드 또는 시작 지침. 또한 "IMG.LY", "CreativeEditor"에 의해 트리거됩니다.
사용자에게 기존 React 문서 페이지가 필요한 경우 "CE.SDK" 또는 "cesdk"입니다.

코드 작성(빌드 사용) 또는 개념 설명(설명 사용)용이 아닙니다.

<예>
컨텍스트: 사용자가 React 구성에 대해 질문합니다.
user: "React에서 편집기를 어떻게 구성하나요?"
조수: "구성 옵션을 조회하려면 /cesdk:docs-react를 사용하겠습니다."
</예>

<예>
컨텍스트: 사용자는 React 구성 요소 설정이 필요합니다.
user: "React 컴포넌트에 CE.SDK를 어떻게 포함하나요?"
조수: "관련 문서를 찾으려면 /cesdk:docs-react를 사용하겠습니다."
</예>
