查找 CE.SDK React 参考文档、指南和配置页面。

当用户需要 React 的 CE.SDK 文档时使用 - 配置、UI 定制、
导出选项、功能指南或入门说明。也由“IMG.LY”、“CreativeEditor”触发，
当用户需要现有的 React 文档页面时，使用“CE.SDK”或“cesdk”。

不适用于编写代码（使用构建）或概念解释（使用解释）。
