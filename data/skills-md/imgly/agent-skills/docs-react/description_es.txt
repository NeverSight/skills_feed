Busque documentos de referencia, guías y páginas de configuración de CE.SDK React.

Úselo cuando el usuario necesite documentos CE.SDK para React: configuración, personalización de la interfaz de usuario,
opciones de exportación, guías de funciones o instrucciones de introducción. También activado por "IMG.LY", "CreativeEditor",
"CE.SDK" o "cesdk" cuando el usuario necesita una página de documento de React existente.

No para escribir código (use compilación) o explicaciones de conceptos (use explicación).

<ejemplo>
Contexto: el usuario pregunta sobre la configuración de React
usuario: "¿Cómo configuro el editor en React?"
asistente: "Usaré /cesdk:docs-react para buscar opciones de configuración".
</ejemplo>

<ejemplo>
Contexto: el usuario necesita la configuración del componente React
usuario: "¿Cómo incrusto CE.SDK en un componente de React?"
