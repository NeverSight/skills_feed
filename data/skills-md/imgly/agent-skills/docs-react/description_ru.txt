Найдите справочную документацию, руководства и страницы конфигурации CE.SDK React.

Используйте, когда пользователю нужна документация CE.SDK для React — настройка, настройка пользовательского интерфейса,
параметры экспорта, руководства по функциям или инструкции по началу работы. Также запускается "IMG.LY", "CreativeEditor",
«CE.SDK» или «cesdk», если пользователю нужна существующая страница документа React.

Не для написания кода (используйте сборку) или объяснения концепций (используйте объяснение).

<пример>
Контекст: пользователь спрашивает о конфигурации React.
пользователь: «Как мне настроить редактор в React?»
помощник: «Я буду использовать /cesdk:docs-react для поиска параметров конфигурации».
</пример>

<пример>
Контекст: пользователю нужна настройка компонента React.
пользователь: «Как мне встроить CE.SDK в компонент React?»
помощник: «Позвольте мне использовать /cesdk:docs-react, чтобы найти соответствующую документацию».
</пример>
