CE.SDK React のリファレンス ドキュメント、ガイド、構成ページを検索してください。

ユーザーが React 用の CE.SDK ドキュメントを必要とする場合 (構成、UI カスタマイズ、
エクスポート オプション、機能ガイド、または開始手順。 「IMG.LY」、「CreativeEditor」によってもトリガーされます。
「CE.SDK」、またはユーザーが既存の React ドキュメント ページを必要とする場合は「cesdk」。

コードの作成 (build を使用) や概念の説明 (explain を使用) のためではありません。

<例>
コンテキスト: ユーザーが React 構成について尋ねる
ユーザー: 「React でエディターを設定するにはどうすればよいですか?」
アシスタント: 「/cesdk:docs-react を使用して構成オプションを調べます。」
</例>

<例>
コンテキスト: ユーザーは React コンポーネントのセットアップを必要としています
ユーザー: 「React コンポーネントに CE.SDK を埋め込むにはどうすればよいですか?」
アシスタント: 「関連するドキュメントを見つけるには /cesdk:docs-react を使用させてください。」
</例>
