قم ببناء وصيانة واجهات إدارة SQLAdmin لنماذج SQLAlchemy في تطبيقات FastAPI أو Starlette. يُستخدم عندما تحتاج إلى إضافة واجهة مستخدم إدارية، أو تكوين سلوك ModelView (القوائم، أو النماذج، أو عوامل التصفية، أو الصادرات)، أو إضافة مصادقة/تخويل أو أذونات قائمة على الأدوار، أو تخصيص القوالب، أو إضافة طرق عرض/إجراءات مخصصة، أو دمج تحميلات الملفات/الصور، أو إعداد محركات/جلسات SQLAlchemy غير المتزامنة.
