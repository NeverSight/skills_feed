Schritt-für-Schritt-Anleitung zum Erstellen neuer interner MCP-Serverintegrationen in Dust, die eine Verbindung zu Remote-Plattformen (Jira, HubSpot, Salesforce usw.) herstellen. Verwenden Sie diese Option, wenn Sie einen neuen MCP-Server hinzufügen, eine Plattformintegration implementieren oder Dust mit einem neuen externen Dienst verbinden.
