Utilizza questa abilità quando crei nuove abilità di Claude Code da zero, modifichi le abilità esistenti per migliorarne le descrizioni o la struttura o converti i sub-agenti di Claude Code in abilità. Ciò include la progettazione di flussi di lavoro delle competenze, la scrittura di file SKILL.md, l'organizzazione di file di supporto con nomi che ne rivelino le intenzioni e l'utilizzo degli strumenti CLI e degli script Node.js.
