استخدم هذه المهارة عند إنشاء مهارات Claude Code جديدة من البداية، أو تحرير المهارات الحالية لتحسين أوصافها أو بنيتها، أو تحويل وكلاء Claude Code الفرعيين إلى مهارات. يتضمن ذلك تصميم سير عمل المهارات، وكتابة ملفات SKILL.md، وتنظيم الملفات الداعمة بأسماء تكشف النية، والاستفادة من أدوات CLI والبرمجة النصية Node.js.
