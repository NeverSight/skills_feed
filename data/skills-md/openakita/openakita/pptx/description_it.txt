Utilizza questa capacità ogni volta che un file .pptx è coinvolto in qualsiasi modo: come input, output o entrambi. Ciò include: creazione di slide deck, pitch deck o presentazioni; leggere, analizzare o estrarre testo da qualsiasi file .pptx (anche se il contenuto estratto verrà utilizzato altrove, come in un'e-mail o in un riepilogo); modificare, modificare o aggiornare presentazioni esistenti; combinare o dividere file di diapositive; lavorare con modelli, layout, note del relatore o commenti. Si attiva ogni volta che l'utente menziona "composizione", "diapositive", "presentazione" o fa riferimento a un nome file .pptx, indipendentemente da cosa intende fare successivamente con il contenuto. Se è necessario aprire, creare o toccare un file .pptx, utilizza questa abilità.
