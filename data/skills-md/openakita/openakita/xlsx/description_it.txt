Utilizza questa competenza ogni volta che un file di foglio di calcolo costituisce l'input o l'output principale. Ciò significa qualsiasi attività in cui l'utente desidera: aprire, leggere, modificare o correggere un file .xlsx, .xlsm, .csv o .tsv esistente (ad esempio, aggiungere colonne, calcolare formule, formattare, creare grafici, pulire dati disordinati); creare un nuovo foglio di calcolo da zero o da altre fonti dati; o convertire tra formati di file tabulari. Si attiva soprattutto quando l'utente fa riferimento a un file di foglio di calcolo per nome o percorso, anche casualmente (come "xlsx nei miei download"), e desidera che venga fatto o prodotto qualcosa da esso. Attiva anche la pulizia o la ristrutturazione di file di dati tabulari disordinati (righe malformate, intestazioni fuori posto, dati spazzatura) in fogli di calcolo adeguati. Il risultato finale deve essere un file di foglio di calcolo. NON attivare quando il risultato finale principale è un documento Word, un report HTML, uno script Python autonomo, una pipeline di database o un'integrazione API di Fogli Google, anche se sono coinvolti dati tabulari.
