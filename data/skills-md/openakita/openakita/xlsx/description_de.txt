Verwenden Sie diese Fähigkeit immer dann, wenn eine Tabellenkalkulationsdatei die primäre Eingabe oder Ausgabe ist. Damit ist jede Aufgabe gemeint, bei der der Benutzer Folgendes tun möchte: eine vorhandene XLSX-, XLSM-, CSV- oder TSV-Datei öffnen, lesen, bearbeiten oder reparieren (z. B. Spalten hinzufügen, Formeln berechnen, formatieren, Diagramme erstellen, unordentliche Daten bereinigen); Erstellen Sie eine neue Tabelle von Grund auf oder aus anderen Datenquellen. oder zwischen tabellarischen Dateiformaten konvertieren. Wird insbesondere dann ausgelöst, wenn der Benutzer mit Namen oder Pfad auf eine Tabellenkalkulationsdatei verweist – auch nur beiläufig (z. B. „die XLSX-Datei in meinen Downloads“) – und möchte, dass etwas damit gemacht oder daraus erstellt wird. Lösen Sie außerdem die Bereinigung oder Umstrukturierung unordentlicher tabellarischer Datendateien (fehlerhafte Zeilen, falsch platzierte Kopfzeilen, Junk-Daten) in ordnungsgemäße Tabellenkalkulationen aus. Bei dem Liefergegenstand muss es sich um eine Tabellenkalkulationsdatei handeln. NICHT auslösen, wenn das primäre Ergebnis ein Word-Dokument, ein HTML-Bericht, ein eigenständiges Python-Skript, eine Datenbankpipeline oder eine Google Sheets-API-Integration ist, selbst wenn es sich um tabellarische Daten handelt.
