استخدم هذه المهارة عندما يريد المستخدم إنشاء مستندات Word (ملفات .docx) أو قراءتها أو تحريرها أو معالجتها. تتضمن المشغلات: أي ذكر لـ "Word doc" أو "word document" أو ".docx" أو طلبات إنتاج مستندات احترافية بتنسيق مثل جداول المحتويات أو العناوين أو أرقام الصفحات أو الترويسات. يُستخدم أيضًا عند استخراج المحتوى أو إعادة تنظيمه من ملفات .docx، أو إدراج الصور أو استبدالها في المستندات، أو إجراء البحث والاستبدال في ملفات Word، أو العمل مع التغييرات أو التعليقات المتعقبة، أو تحويل المحتوى إلى مستند Word مصقول. إذا طلب المستخدم "تقريرًا" أو "مذكرة" أو "خطابًا" أو "قالبًا" أو ما شابه ذلك كملف Word أو .docx، فاستخدم هذه المهارة. لا تستخدمه مع ملفات PDF أو جداول البيانات أو مستندات Google أو مهام الترميز العامة التي لا علاقة لها بإنشاء المستندات.
