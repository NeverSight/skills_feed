Создает масштабируемые системы дизайна с помощью Tailwind CSS 4, используя токены на основе CSS, шаблоны компонентов и адаптивные рекомендации с проверкой доступности. Используйте при создании библиотек компонентов, реализации систем дизайна или рефакторинге и стандартизации шаблонов пользовательского интерфейса.
