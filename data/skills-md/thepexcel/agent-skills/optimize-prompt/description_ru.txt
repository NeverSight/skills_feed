Консультант по оперативной оптимизации для подсказок системы искусственного интеллекта и индивидуальных инструкций. Поддерживает Claude API, OpenAI API, Gemini API, CLAUDE.md, ChatGPT и n8n. Используйте, когда пользователь хочет улучшить, просмотреть или создать подсказки, системные или специальные инструкции для любой платформы искусственного интеллекта.
