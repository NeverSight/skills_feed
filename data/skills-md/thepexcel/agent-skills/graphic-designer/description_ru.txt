Создает графику для миниатюр, социальных сетей, баннеров и презентаций. Применяет принципы дизайна (CRAP, гештальт, визуальная иерархия) с научно обоснованными методами. Интегрируется с /geometric-elements для декоративных ресурсов. Используйте при создании макетов, выборе типографики/цветов или разработке любых графических ресурсов. Для подсказок по фотографии/кинематографии используйте вместо этого /art-director.
