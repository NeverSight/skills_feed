Преобразует Markdown в стилизованный HTML с помощью тем, совместимых с WeChat. Поддерживает выделение кода, математические вычисления, PlantUML, сноски, оповещения и инфографику. Используйте, когда пользователь запрашивает «уценку в html», «конвертировать md в html», «md转html» или нуждается в стилизованном выводе HTML из уценки.
