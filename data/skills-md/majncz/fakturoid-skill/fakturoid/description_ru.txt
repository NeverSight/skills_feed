Интеграция чешского API для выставления счетов для Fakturoid. Используйте, когда пользователю необходимо создавать, управлять или получать счета, контакты/темы или расходы из Fakturoid. Поддерживает аутентификацию OAuth 2.0, создание счетов с соблюдением чешского налогового законодательства (IČO, DIČ, НДС), загрузку PDF-файлов, отслеживание платежей, управление расходами, инвентаризацию, повторяющиеся счета, веб-перехватчики и шаблоны счетов (генераторы). Запускается при упоминании Fakturoid, чешского выставления счетов, IČO/DIČ или задач управления счетами.
