Integración API de facturación checa para Fakturoid. Úselo cuando el usuario necesite crear, administrar o recuperar facturas, contactos/asuntos o gastos de Fakturoid. Admite autenticación OAuth 2.0, creación de facturas con cumplimiento tributario checo (IČO, DIČ, IVA), descargas de PDF, seguimiento de pagos, gestión de gastos, inventario, facturas recurrentes, webhooks y plantillas de facturas (generadores). Se activa ante menciones de Fakturoid, facturación checa, IČO/DIČ o tareas de gestión de facturas.
