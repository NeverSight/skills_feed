Guía de integración de pagos de Teya que cubre POSLink (integración de terminal basada en la nube), All-In-One (Android de un solo dispositivo), API de comercio electrónico (pago alojado, enlaces de pago) y pasarela de pagos. Incluye árboles de decisión, patrones de autenticación, tarjetas de prueba y orientación para la certificación.
