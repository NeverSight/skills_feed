Integración API de firma electrónica checa para DigiSign. Úselo cuando el usuario necesite crear, enviar y administrar sobres de firma digital, cargar documentos para firmar, realizar un seguimiento del estado de la firma o integrar flujos de firma integrados. Admite múltiples tipos de firma, webhooks para seguimiento de estado y verificación de identificación del banco checo. Se activa ante menciones de DigiSign, podpis electrónicos, firmas digitales, firma de sobres o flujos de trabajo de firma de documentos.
