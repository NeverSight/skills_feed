Normes de qualité du code : avertissements en tant qu'erreurs, pas de préfixes de trait de soulignement pour les variables inutilisées, couverture de test obligatoire. Appliquez-le à n’importe quel projet TypeScript, JavaScript, Rust ou Python. Déclencheurs : revue de code, peluchage, écriture de nouvelles fonctions, refactoring.
