Die Erstellung der API-Dokumentation basiert auf OpenAPI/Swagger und dem PHP/Laravel-Code. Zur Dokumentation von Endpunkten, zur Erstellung von OpenAPI 3.0-Spezifikationen, zur Erstellung interaktiver Dokumentation, zur Dokumentation von Authentifizierungs-, Anforderungs-/Antwortschemata und zur Integration von Komponenten wie Swagger UI, Redoc oder Stoplight.
