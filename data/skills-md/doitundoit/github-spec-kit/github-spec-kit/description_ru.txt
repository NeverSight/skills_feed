Экспертное руководство по разработке на основе спецификаций с использованием GitHub Spec Kit. Используйте, когда пользователь хочет инициализировать проект, создать спецификации, спланировать реализацию или выполнить задачи на основе рабочего процесса Spec Kit. Запускается по «комплекту спецификаций», «указанию», «созданию спецификации», «разработке на основе спецификации».
