Создает новый репозиторий стартового комплекта Next.js со структурой документации PRD. Настраивает минимальное приложение Next.js с помощью shadcn/ui и Tailwind CSS, затем создает каталог документации с шаблонами PRD. Используйте, когда пользователь хочет создать новый репозиторий стартового комплекта для документирования нового продукта или системы.
