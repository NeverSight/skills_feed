Erstellt ein neues Next.js-Starterkit-Repository mit PRD-Dokumentationsstruktur. Richtet eine minimale Next.js-App mit shadcn/ui und Tailwind CSS ein und erstellt dann ein Gerüst für ein Dokumentenverzeichnis mit PRD-Vorlagen. Verwenden Sie diese Option, wenn der Benutzer ein neues Starterkit-Repository zur Dokumentation eines neuen Produkts oder Systems erstellen möchte.
