عندما يريد المستخدم تحسين تدفقات الاشتراك أو التسجيل أو إنشاء الحساب أو التنشيط التجريبي. يُستخدم أيضًا عندما يذكر المستخدم "تحويلات الاشتراك"، أو "احتكاك التسجيل"، أو "تحسين نموذج الاشتراك"، أو "اشتراك تجريبي مجاني"، أو "تقليل الانقطاع عن الاشتراك"، أو "تدفق إنشاء الحساب". للتعرف على الإعداد بعد الاشتراك، راجع onboarding-cro. للحصول على نماذج التقاط العملاء المحتملين (وليس إنشاء الحساب)، راجع form-cro.
