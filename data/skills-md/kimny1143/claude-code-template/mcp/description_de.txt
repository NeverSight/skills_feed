Handbuch zur Erstellung und Verwaltung von Model Context Protocol (MCP)-Servern. So fügen Sie benutzerdefinierte Tools hinzu. Verwendungszweck: (1) Wenn Sie benutzerdefinierte Tools zu Claude Code hinzufügen möchten (2) Bei der Integration mit externen Diensten (3) Wenn Sie projektspezifische Automatisierungstools erstellen möchten (4) Wenn Sie wissen möchten, wie ein MCP-Server konfiguriert wird. Trigger-Beispiele: „Einen MCP-Server erstellen“ „Ein benutzerdefiniertes Tool hinzufügen“ „MCP-Einstellungen“ „Eine externe API in ein Tool umwandeln“ „Test Runner MCP“
