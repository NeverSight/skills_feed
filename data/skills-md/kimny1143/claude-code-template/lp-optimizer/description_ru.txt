Навыки стороннего анализа и оптимизации маркетинговых страниц. Цели: целевые страницы, домашние страницы, страницы с ценами, тематические страницы, блоги. Интегрирует структуры CRO с принципами прямого ответа (Шварц/Хопкинс/Огилви). Варианты использования: (1) запросы на аудит/анализ страниц (2) консультации по оптимизации конверсии (3) улучшения копирайтинга (4) разработка A/B-тестов (5) определение приоритетов реализации. Примеры триггеров: «проанализируйте эту LP», «повысьте коэффициент конверсии», «проверьте страницу с ценами», «оптимизируйте главный раздел», «оптимизируйте призыв к действию», «идеи по улучшению домашней страницы».
