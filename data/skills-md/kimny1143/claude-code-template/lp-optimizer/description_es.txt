Habilidades de análisis y optimización de terceros para páginas de marketing. Objetivos: páginas de destino, páginas de inicio, páginas de precios, páginas de funciones, blogs. Integra marcos de CRO con principios de respuesta directa (Schwartz/Hopkins/Ogilvy). Casos de uso: (1) solicitudes de análisis/auditoría de página (2) consultas de optimización de conversión (3) mejoras de redacción (4) diseño de pruebas A/B (5) priorización de implementación. Ejemplos de desencadenantes: "analizar este LP", "aumentar la tasa de conversión", "revisar la página de precios", "optimizar la sección principal", "optimizar la CTA", "ideas para mejorar la página de inicio".
