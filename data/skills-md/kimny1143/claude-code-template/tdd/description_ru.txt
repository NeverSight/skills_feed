Навыки выполнения рабочего процесса разработки через тестирование (TDD). Используйте, когда: (1) начинаете реализовывать новую функцию, (2) исправляете ошибку, (3) перед рефакторингом, (4) вам говорят «сначала написать тест», (5) необходима реализация, ориентированная на качество. Примеры триггеров: «реализовать с помощью TDD», «на основе тестирования», «сначала написать тест», «КРАСНЫЙ-ЗЕЛЕНЫЙ-РЕФАКТОР», «написать тесты для UserService».
