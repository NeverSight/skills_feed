Руководство по созданию и управлению Claude Code Hook. Реализация таких хуков, как PreToolUse, PostToolUse, Stop и т. д.  
Срок использования:  
(1) Если вы хотите заблокировать опасные операции  
(2) Если вы хотите внести предложения после выполнения инструмента  
(3) Если вы хотите выполнить очистку в конце сеанса  
(4) Если вы хотите настроить Claude Code  

Примеры триггеров:  
«Создать крючок»  
«Хук PreToolUse»  
«Блокировать опасные операции»  
«Предложить после фиксации»  
«Хуки Клода Кода»
