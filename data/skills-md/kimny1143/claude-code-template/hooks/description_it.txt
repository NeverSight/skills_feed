Guida alla creazione e gestione di Claude Code Hook. Implementazione di hook come PreToolUse, PostToolUse, Stop, ecc.  
Tempi di utilizzo:  
(1) Quando si desidera bloccare operazioni pericolose  
(2) Quando si desidera fornire suggerimenti dopo l'esecuzione dello strumento  
(3) Quando si desidera eseguire la pulizia al termine della sessione  
(4) Quando desideri personalizzare Claude Code  

Esempi di trigger:  
"Crea un gancio"  
"Gancio PreToolUse"  
"Blocca operazioni pericolose"  
"Suggerisci dopo il commit"  
"Ganci Claude Code"
