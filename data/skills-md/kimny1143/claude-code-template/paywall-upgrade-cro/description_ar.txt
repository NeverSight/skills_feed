عندما يريد المستخدم إنشاء أو تحسين نظام حظر الاشتراك غير المدفوع داخل التطبيق، أو ترقية الشاشات، أو زيادة المبيعات، أو بوابات الميزات. استخدمه أيضًا عندما يذكر المستخدم "نظام حظر الاشتراك غير المدفوع"، أو "شاشة الترقية"، أو "ترقية مشروطة"، أو "زيادة المبيعات"، أو "بوابة الميزات"، أو "التحويل مجانًا إلى مدفوع"، أو "تحويل freemium"، أو "شاشة انتهاء صلاحية النسخة التجريبية"، أو "شاشة الحد الذي تم الوصول إليه"، أو "المطالبة بترقية الخطة"، أو "التسعير داخل التطبيق". تختلف عن صفحات التسعير العامة (راجع lp-optimizer) — تركز هذه المهارة على لحظات الترقية داخل المنتج حيث يكون المستخدم قد اختبر القيمة بالفعل.
