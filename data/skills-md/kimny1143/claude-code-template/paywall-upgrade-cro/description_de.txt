Wenn der Benutzer In-App-Paywalls, Upgrade-Bildschirme, Upselling-Modi oder Feature-Gates erstellen oder optimieren möchte. Verwenden Sie es auch, wenn der Benutzer „Paywall“, „Upgrade-Bildschirm“, „Upgrade-Modal“, „Upsell“, „Feature Gate“, „Kostenlos in kostenpflichtig umwandeln“, „Freemium-Konvertierung“, „Testablaufbildschirm“, „Bildschirm „Limit erreicht“, „Aufforderung zum Upgrade des Plans“ oder „In-App-Preise“ erwähnt. Im Gegensatz zu öffentlichen Preisseiten (siehe lp-optimizer) – dieser Skill konzentriert sich auf Momente des Produkt-Upgrades, in denen der Benutzer bereits einen Mehrwert erlebt hat.
