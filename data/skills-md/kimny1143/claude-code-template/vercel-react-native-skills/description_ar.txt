React Native and Expo أفضل الممارسات لبناء تطبيقات الهاتف المحمول عالية الأداء. يُستخدم عند إنشاء مكونات React Native، أو تحسين أداء القائمة، أو تنفيذ الرسوم المتحركة، أو العمل مع الوحدات النمطية الأصلية. يتم تشغيل المهام التي تتضمن React Native، أو Expo، أو أداء الهاتف المحمول، أو واجهات برمجة تطبيقات النظام الأساسي الأصلي.
