عندما يريد المستخدم كتابة أو إعادة كتابة أو تحسين نسخة تسويقية لأي صفحة - بما في ذلك الصفحة الرئيسية أو الصفحات المقصودة أو صفحات التسعير أو صفحات الميزات أو الصفحات الخاصة أو صفحات المنتج. يُستخدم أيضًا عندما يقول المستخدم "اكتب نسخة لـ" أو "تحسين هذه النسخة" أو "أعد كتابة هذه الصفحة" أو "نسخة تسويقية" أو "مساعدة العنوان" أو "نسخة CTA". للحصول على نسخة البريد الإلكتروني، راجع تسلسل البريد الإلكتروني.
