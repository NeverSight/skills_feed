Wenn der Benutzer eine iOS-App an den App Store senden möchte. Wird verwendet, wenn der Benutzer „App Store“, „App Store Connect“, „TestFlight“, „iOS-Einreichung“, „App-Überprüfung“, „EAS-Build“, „eas-Einreichung“, „Apple-Überprüfung“, „App-Ablehnung“ oder „iOS-Release“ erwähnt. Diese Fähigkeit deckt den gesamten Prozess von der Einrichtung bis zur Genehmigung der App Store-Überprüfung ab.
