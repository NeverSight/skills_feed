Managing multiple branches simultaneously using Git worktree.  
When to use:  
(1) When you want to develop multiple features in parallel  
(2) When you want to work on something else while reviewing a PR  
(3) When you need to handle a production hotfix and ongoing development at the same time  
(4) When you want to learn how to use worktree  

Trigger examples:  
“with worktree,” “simultaneously on another branch,” “want to develop in parallel,” “develop while reviewing a PR,” “worktree for hotfix”
