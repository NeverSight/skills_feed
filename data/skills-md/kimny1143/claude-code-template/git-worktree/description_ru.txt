Одновременное управление несколькими ветками с помощью рабочего дерева Git.  
Когда использовать:  
(1) Если вы хотите параллельно разрабатывать несколько функций.  
(2) Когда вы хотите поработать над чем-то другим, просматривая PR.  
(3) Когда вам нужно одновременно обрабатывать производственное исправление и текущую разработку.  
(4) Если вы хотите научиться использовать рабочее дерево  

Примеры триггеров:  
«с рабочим деревом», «одновременно в другой ветке», «хочется разрабатывать параллельно», «разрабатывать во время рассмотрения заявки на поставку», «рабочее дерево для исправления»
