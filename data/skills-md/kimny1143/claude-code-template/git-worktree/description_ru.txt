Одновременное управление несколькими ветками с помощью рабочего дерева Git. Когда использовать: (1) Когда вы хотите параллельно разрабатывать несколько функций (2) Когда вы хотите работать над чем-то еще, просматривая PR (3) Когда вам нужно обработать производственное исправление, продолжая разработку (4) Когда вы хотите научиться использовать рабочее дерево. Примеры триггеров: «с рабочим деревом», «одновременно в другой ветке», «хочу разрабатывать параллельно», «разрабатывать во время проверки PR», «рабочее дерево для исправления».
