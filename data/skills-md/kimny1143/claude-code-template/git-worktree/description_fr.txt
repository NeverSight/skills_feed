Gérer plusieurs branches simultanément à l'aide de l'arbre de travail Git. Quand l'utiliser : (1) Lorsque vous souhaitez développer plusieurs fonctionnalités en parallèle (2) Lorsque vous souhaitez travailler sur autre chose tout en examinant un PR (3) Lorsque vous devez gérer un correctif de production tout en poursuivant le développement (4) Lorsque vous souhaitez apprendre à utiliser l'arbre de travail. Exemples de déclencheurs : "avec worktree", "simultanément sur une autre branche", "vous souhaitez développer en parallèle", "développer tout en examinant les relations publiques", "worktree pour hotfix"
