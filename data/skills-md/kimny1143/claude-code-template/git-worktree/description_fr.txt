Gérer plusieurs branches simultanément à l'aide de l'arbre de travail Git.  
Quand utiliser :  
(1) Lorsque vous souhaitez développer plusieurs fonctionnalités en parallèle  
(2) Lorsque vous souhaitez travailler sur autre chose tout en révisant un PR  
(3) Lorsque vous devez gérer simultanément un correctif de production et un développement en cours  
(4) Quand vous voulez apprendre à utiliser worktree  

Exemples de déclencheurs :  
« avec worktree », « simultanément sur une autre branche », « vouloir développer en parallèle », « développer en examinant un PR », « worktree pour hotfix »
