Administrar múltiples sucursales simultáneamente usando el árbol de trabajo de Git.  
Cuando usar:  
(1) Cuando desee desarrollar varias funciones en paralelo  
(2) Cuando quieres trabajar en otra cosa mientras revisas un PR  
(3) Cuando necesita manejar una revisión de producción y un desarrollo continuo al mismo tiempo  
(4) Cuando quieras aprender a usar worktree  

Ejemplos de desencadenantes:  
“con árbol de trabajo”, “simultáneamente en otra rama”, “quiero desarrollar en paralelo”, “desarrollar mientras revisa un PR”, “árbol de trabajo para revisión”
