Administrar múltiples sucursales simultáneamente usando el árbol de trabajo de Git. Cuándo usarlo: (1) Cuando desee desarrollar múltiples funciones en paralelo (2) Cuando desee trabajar en otra cosa mientras revisa un PR (3) Cuando necesite manejar una revisión de producción mientras continúa el desarrollo (4) Cuando desee aprender a usar el árbol de trabajo. Ejemplos de activación: "con árbol de trabajo", "simultáneamente en otra rama", "quiero desarrollar en paralelo", "desarrollar mientras se revisan las relaciones públicas", "árbol de trabajo para revisión"
