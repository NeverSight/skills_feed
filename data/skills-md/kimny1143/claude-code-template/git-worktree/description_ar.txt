إدارة فروع متعددة في وقت واحد باستخدام Git Worktree.  
متى تستخدم:  
(1) عندما تريد تطوير ميزات متعددة بالتوازي  
(2) عندما تريد العمل على شيء آخر أثناء مراجعة العلاقات العامة  
(3) عندما تحتاج إلى التعامل مع إصلاح عاجل للإنتاج والتطوير المستمر في نفس الوقت  
(4) عندما تريد معرفة كيفية استخدام Worktree  

أمثلة الزناد:  
"مع شجرة العمل"، "في وقت واحد على فرع آخر"، "أريد التطوير بالتوازي"، "التطوير أثناء مراجعة العلاقات العامة"، "شجرة العمل للإصلاح العاجل"
