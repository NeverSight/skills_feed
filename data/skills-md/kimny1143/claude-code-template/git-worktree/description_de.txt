Mehrere Zweige gleichzeitig mit Git Worktree verwalten.  
Wann zu verwenden:  
(1) Wenn Sie mehrere Funktionen parallel entwickeln möchten  
(2) Wenn Sie während der Überprüfung einer PR an etwas anderem arbeiten möchten  
(3) Wenn Sie gleichzeitig einen Produktions-Hotfix und eine laufende Entwicklung durchführen müssen  
(4) Wenn Sie lernen möchten, wie man Worktree verwendet  

Trigger-Beispiele:  
„mit Arbeitsbaum“, „gleichzeitig auf einem anderen Zweig“, „parallel entwickeln wollen“, „entwickeln während der Prüfung einer PR“, „Arbeitsbaum für Hotfix“
