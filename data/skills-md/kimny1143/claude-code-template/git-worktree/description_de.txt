Mehrere Zweige gleichzeitig mit Git Worktree verwalten. Verwendungszweck: (1) Wenn Sie mehrere Funktionen parallel entwickeln möchten (2) Wenn Sie während der Überprüfung einer PR an etwas anderem arbeiten möchten (3) Wenn Sie einen Produktions-Hotfix bearbeiten müssen, während Sie die Entwicklung fortsetzen (4) Wenn Sie lernen möchten, wie man Worktree verwendet. Trigger-Beispiele: „mit Arbeitsbaum“, „gleichzeitig auf einem anderen Zweig“, „parallel entwickeln wollen“, „entwickeln während der PR-Prüfung“, „Arbeitsbaum für Hotfix“
