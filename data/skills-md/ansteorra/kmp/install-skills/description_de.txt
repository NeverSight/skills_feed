Installieren und verwalten Sie Agent Skills automatisch aus GitHub-Repositorys. Verwenden Sie diese Option, wenn Sie aufgefordert werden, „einen Skill zu installieren“, „einen Skill hinzuzufügen“, „Skills zu finden“, „Skills zu durchsuchen“, „Skills von GitHub abzurufen“ oder wenn der Benutzer eine bestimmte Fähigkeit benötigt, die möglicherweise als Community-Skill vorhanden ist. Unterstützt Anthropics/Skills, Github/Awesome-Copilot und benutzerdefinierte GitHub-Repositorys.
