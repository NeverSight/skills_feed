Installez et gérez automatiquement les compétences d'agent à partir des référentiels GitHub. À utiliser lorsqu'on lui demande « installer une compétence », « ajouter une compétence », « rechercher des compétences », « parcourir les compétences », « obtenir des compétences depuis GitHub » ou lorsque l'utilisateur a besoin d'une fonctionnalité spécifique qui pourrait exister en tant que compétence communautaire. Prend en charge les référentiels anthropiques/compétences, github/awesome-copilot et GitHub personnalisés.
