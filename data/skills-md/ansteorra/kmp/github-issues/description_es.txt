Cree, actualice y administre problemas de GitHub utilizando herramientas MCP. Utilice esta habilidad cuando los usuarios quieran crear informes de errores, solicitudes de funciones o problemas de tareas, actualizar problemas existentes, agregar etiquetas/asignados/hitos o administrar flujos de trabajo de problemas. Activadores de solicitudes como "crear un problema", "presentar un error", "solicitar una función", "actualizar el problema X" o cualquier tarea de administración de problemas de GitHub.
