Создавайте, обновляйте и управляйте проблемами GitHub с помощью инструментов MCP. Используйте этот навык, когда пользователи хотят создавать отчеты об ошибках, запросы функций или проблемы с задачами, обновлять существующие проблемы, добавлять метки/назначенных/этапы или управлять рабочими процессами решения проблем. Запускается по таким запросам, как «создать проблему», «сообщить об ошибке», «запросить функцию», «обновить проблему X» или любой задаче управления проблемами GitHub.
