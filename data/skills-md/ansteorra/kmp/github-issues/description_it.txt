Crea, aggiorna e gestisci i problemi di GitHub utilizzando gli strumenti MCP. Utilizza questa competenza quando gli utenti desiderano creare segnalazioni di bug, richieste di funzionalità o problemi relativi alle attività, aggiornare problemi esistenti, aggiungere etichette/assegnatari/traguardi o gestire flussi di lavoro relativi ai problemi. Si attiva su richieste come "crea un problema", "segnala un bug", "richiedi una funzionalità", "aggiorna il problema X" o qualsiasi attività di gestione dei problemi di GitHub.
