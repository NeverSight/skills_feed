Выполните git commit с помощью обычного анализа сообщений о фиксации, интеллектуальной подготовки и генерации сообщений. Используйте, когда пользователь просит зафиксировать изменения, создать коммит git или упоминает «/commit». Поддерживается: (1) автоматическое определение типа и области действия изменений, (2) генерация обычных сообщений о фиксации на основе различий, (3) интерактивная фиксация с дополнительными переопределениями типа/области/описания, (4) интеллектуальная подготовка файлов для логической группировки.
