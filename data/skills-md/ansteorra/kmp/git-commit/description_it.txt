Esegui git commit con l'analisi convenzionale dei messaggi di commit, gestione temporanea intelligente e generazione di messaggi. Da utilizzare quando l'utente chiede di confermare modifiche, creare un commit git o menziona "/commit". Supporta: (1) rilevamento automatico del tipo e dell'ambito dalle modifiche, (2) generazione di messaggi di commit convenzionali da diff, (3) commit interattivo con sostituzioni facoltative di tipo/ambito/descrizione, (4) gestione temporanea intelligente dei file per il raggruppamento logico
