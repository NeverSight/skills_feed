Exécutez git commit avec une analyse conventionnelle des messages de validation, une mise en scène intelligente et une génération de messages. À utiliser lorsque l'utilisateur demande de valider les modifications, crée un commit git ou mentionne "/commit". Prend en charge : (1) Détection automatique du type et de la portée des modifications, (2) Génération de messages de validation conventionnels à partir des différences, (3) Validation interactive avec remplacements facultatifs de type/portée/description, (4) Mise en scène intelligente des fichiers pour le regroupement logique
