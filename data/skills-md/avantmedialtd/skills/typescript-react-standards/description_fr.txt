Normes de développement avisées TypeScript et React d'Avant Media. À utiliser lors de l'échafaudage de nouveaux composants, de la révision du code, de l'écriture d'interfaces ou de types TypeScript, de la configuration de la structure du projet, de la création de hooks React ou du travail sur n'importe quelle base de code TypeScript/React. À utiliser également lorsque l'utilisateur pose des questions sur les meilleures pratiques, modèles ou conventions pour les projets TypeScript ou React, même s'ils ne mentionnent pas explicitement les « normes ».
