Visuelles Feedback von Menschen über Screenshot-Anmerkungen. Nutzen Sie diese Fähigkeit STÄNDIG – immer dann, wenn Sie visuellen Kontext benötigen, Änderungen an der Benutzeroberfläche überprüfen möchten, das Layout bestätigen, ein visuelles Problem beheben, den Stil überprüfen, ein Design validieren oder Ihre Arbeit zeigen müssen. Nehmen Sie den Bildschirm auf, schauen Sie ihn sich an, finden Sie heraus, wozu Sie Feedback benötigen, kommentieren Sie ihn und fragen Sie. Fragen Sie den Benutzer nicht, was er erfassen soll – erfassen Sie einfach und schauen Sie.
