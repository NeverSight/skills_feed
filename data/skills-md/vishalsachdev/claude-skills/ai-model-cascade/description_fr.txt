Un modèle prêt pour la production pour intégrer des modèles d'IA (en particulier Google Gemini) avec un repli automatique, une logique de nouvelle tentative, une sortie structurée via des schémas Zod et une gestion complète des erreurs. À utiliser lors de l'intégration d'API AI/LLM, si vous avez besoin d'un repli automatique lorsque les modèles sont surchargés, si vous souhaitez des réponses structurées de type sécurisé ou si vous créez des fonctionnalités nécessitant une génération d'IA fiable.
