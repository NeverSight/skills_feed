Un modello pronto per la produzione per l'integrazione di modelli IA (in particolare Google Gemini) con fallback automatico, logica di ripetizione, output strutturato tramite schemi Zod e gestione completa degli errori. Da utilizzare quando si integrano API AI/LLM, è necessario il fallback automatico quando i modelli sono sovraccarichi, si desiderano risposte strutturate indipendenti dai tipi o si creano funzionalità che richiedono una generazione di IA affidabile.
