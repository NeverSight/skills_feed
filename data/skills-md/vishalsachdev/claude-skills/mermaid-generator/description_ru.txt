Этот навык создает интерактивные диаграммы рабочего процесса с использованием библиотеки JavaScript Mermaid для размещения в узкой области учебника с помощью iframe. Используйте этот навык, когда пользователи запрашивают создание блок-схем, диаграмм процессов, визуализаций рабочих процессов или деревьев решений для учебных учебников. Этот навык создает пакеты MicroSim с отдельными HTML-файлами с ярким фоном, шрифтами размером 16 пунктов и рендерингом сверху вниз по умолчанию, сохраняемыми в /docs/sims/ в соответствии с шаблоном MicroSim.
