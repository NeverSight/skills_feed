Questa competenza genera diagrammi di flusso di lavoro interattivi utilizzando la libreria JavaScript Sirena per il posizionamento in un'area ristretta di un libro di testo utilizzando un iframe. Utilizza questa competenza quando gli utenti richiedono la creazione di diagrammi di flusso, diagrammi di processo, visualizzazioni di flussi di lavoro o alberi decisionali per libri di testo didattici. La competenza crea pacchetti MicroSim con file HTML autonomi con sfondi colorati, caratteri a 16 punti e rendering dall'alto verso il basso per impostazione predefinita, salvati in /docs/sims/ seguendo il modello MicroSim.
