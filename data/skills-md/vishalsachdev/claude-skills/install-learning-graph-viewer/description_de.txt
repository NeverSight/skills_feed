Dieser Skill installiert eine interaktive Lerngraph-Viewer-Anwendung in ein intelligentes Lehrbuchprojekt. Verwenden Sie diese Fähigkeit, wenn Sie mit einem Lehrbuch arbeiten, das über eine Datei „learning-graph.json“ verfügt und ein visuelles, interaktives Tool zur Diagrammerkundung mit Such-, Filter- und Statistikfunktionen benötigt.
