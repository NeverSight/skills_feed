Этот навык позволяет установить приложение для просмотра интерактивных обучающих графиков в интеллектуальный проект учебника. Используйте этот навык при работе с учебником, в котором есть файл Learning-graph.json и которому необходим визуальный интерактивный инструмент исследования графиков с возможностями поиска, фильтрации и статистики.
