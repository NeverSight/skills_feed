Этот навык создает тщательно подобранные, проверенные списки литературы для учебников с ресурсами, соответствующими уровню (10 для младших классов средней школы, 20 для старших классов, 30 для колледжей, 40 для аспирантов). Ссылки форматируются с использованием ссылок, сведений о публикации и релевантных описаний. Используйте этот навык при работе с интеллектуальными учебниками, требующими академических ссылок, либо на уровне книги, либо на уровне главы.
