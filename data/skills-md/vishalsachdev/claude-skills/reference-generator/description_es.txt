Esta habilidad genera listas de referencias seleccionadas y verificadas para libros de texto con recursos apropiados para el nivel (10 para secundaria, 20 para secundaria, 30 para universidad, 40 para posgrado). Las referencias están formateadas con enlaces, detalles de publicación y descripciones de relevancia. Utilice esta habilidad cuando trabaje con libros de texto inteligentes que necesiten referencias académicas, ya sea a nivel de libro o de capítulo.
