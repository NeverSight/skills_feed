Un système middleware de sécurité complet pour les routes API Next.js 13+ App Router qui fournit l'authentification, la limitation de débit, la protection CSRF, la journalisation d'audit et les en-têtes de sécurité dans un modèle composable et prêt pour la production. À utiliser lors de la création d'API Next.js sécurisées qui nécessitent une protection contre les vulnérabilités Web courantes.
