تنشئ هذه المهارة تصورات تفاعلية لمخطط Venn باستخدام مكتبة JavaScript venn.js. استخدم هذه المهارة عندما يطلب المستخدمون إنشاء مخططات Venn، أو تعيين تصورات، أو مخططات متداخلة، أو مخططات مقارنة للكتب المدرسية التعليمية. تنشئ المهارة حزم MicroSim كاملة مع ملفات HTML مستقلة تتميز بدوائر ملونة وتسميات واضحة وتلميحات أدوات تفاعلية، ويتم حفظها في /docs/sims/ باتباع نمط MicroSim.
