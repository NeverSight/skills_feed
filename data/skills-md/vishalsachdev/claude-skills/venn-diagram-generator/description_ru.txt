Этот навык генерирует интерактивные визуализации диаграмм Венна с использованием библиотеки JavaScript venn.js. Используйте этот навык, когда пользователи запрашивают создание диаграмм Венна, настройку визуализаций, диаграмм перекрытия или сравнительных диаграмм для учебных учебников. Этот навык создает полные пакеты MicroSim с отдельными HTML-файлами с цветными кружками, четкими метками и интерактивными подсказками, сохраняемыми в /docs/sims/ в соответствии с шаблоном MicroSim.
