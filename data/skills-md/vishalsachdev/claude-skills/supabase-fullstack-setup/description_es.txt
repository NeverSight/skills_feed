Guía completa para integrar Supabase con Next.js 13+ App Router, que incluye autenticación, configuración de bases de datos, seguridad de nivel de fila (RLS) y patrones comunes para aplicaciones de producción. Úselo cuando inicie un nuevo proyecto Next.js con Supabase, necesite autenticación con proveedores sociales o cree aplicaciones con datos específicos del usuario.
