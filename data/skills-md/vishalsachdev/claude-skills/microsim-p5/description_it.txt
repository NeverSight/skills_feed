Crea un MicroSim educativo interattivo utilizzando la libreria JavaScript p5.js con regioni distinte per il disegno e controlli interattivi.  Ogni MicroSim è una directory situata nella cartella /docs/sims.  Ha un file main.html che fa riferimento al codice javascript e main.html può essere referenziato come iframe da index.md.  metadata.json contiene i metadati Dublin Core relativi a MicroSim.
