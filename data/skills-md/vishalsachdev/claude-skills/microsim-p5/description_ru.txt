Создайте интерактивный образовательный MicroSim, используя библиотеку JavaScript p5.js с отдельными областями для рисования и интерактивными элементами управления.  Каждая MicroSim представляет собой каталог, расположенный в папке /docs/sims.  У него есть файл main.html, который ссылается на код JavaScript, а на main.html можно ссылаться как на iframe из index.md.  Metadata.json содержит основные метаданные Dublin о MicroSim.
