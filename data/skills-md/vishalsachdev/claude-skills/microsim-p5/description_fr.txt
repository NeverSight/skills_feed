Créez un MicroSim éducatif interactif à l'aide de la bibliothèque JavaScript p5.js avec des régions distinctes pour le dessin et les contrôles interactifs.  Chaque MicroSim est un répertoire situé dans le dossier /docs/sims.  Il contient un fichier main.html qui fait référence au code javascript et le main.html peut être référencé en tant qu'iframe à partir de index.md.  Le fichier metadata.json contient des métadonnées Dublin Core sur MicroSim.
