Этот навык генерирует подробные отчеты о показателях для интеллектуальных учебников, созданных с помощью MkDocs Material, анализируя главы, концепции, термины глоссария, часто задаваемые вопросы, вопросы викторины, диаграммы, уравнения, MicroSims, количество слов и ссылки. Используйте этот навык при работе над интеллектуальным проектом учебника, который требует количественного анализа его содержания, обычно после значительной разработки контента или для отчетности о состоянии проекта. Навык создает два файла уценки — book-metrics.md с общей статистикой и Chapter-metrics.md с разбивкой по главам — в каталоге docs/learning-graph/.
