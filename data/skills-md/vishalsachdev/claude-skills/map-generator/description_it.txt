Questa competenza genera mappe interattive utilizzando la libreria JavaScript Leaflet. Utilizza questa competenza quando gli utenti devono creare visualizzazioni geografiche, visualizzazioni di dati basati sulla posizione o mappe interattive per libri di testo didattici. La competenza crea pacchetti MicroSim completi con HTML, CSS e documentazione, ottimizzati per l'incorporamento di iframe in pagine MkDocs strette con barra di navigazione e sommario.
