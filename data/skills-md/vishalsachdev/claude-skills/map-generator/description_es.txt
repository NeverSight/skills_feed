Esta habilidad genera mapas interactivos utilizando la biblioteca JavaScript Leaflet. Utilice esta habilidad cuando los usuarios necesiten crear visualizaciones geográficas, visualizaciones de datos basados ​​en la ubicación o mapas interactivos para libros de texto educativos. La habilidad crea paquetes completos de MicroSim con HTML, CSS y documentación, optimizados para la incrustación de iframe en páginas estrechas de MkDocs con barra de navegación y TOC.
