Этот навык генерирует интерактивные карты с использованием библиотеки JavaScript Leaflet. Используйте этот навык, когда пользователям необходимо создавать географические визуализации, отображения данных с привязкой к местоположению или интерактивные карты для учебных учебников. Этот навык позволяет создавать полные пакеты MicroSim с HTML, CSS и документацией, оптимизированные для встраивания iframe в узкие страницы MkDocs с навигационной панелью и оглавлением.
