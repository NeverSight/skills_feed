Esta habilidad genera visualizaciones interactivas de Chart.js para usar en iframes utilizando cualquier tipo de gráfico admitido por la biblioteca (línea, barra, circular, donut, radar, área polar, burbuja, dispersión). Utilice esta habilidad cuando los usuarios necesiten crear visualizaciones de datos para contenido educativo, informes o paneles. La habilidad crea paquetes completos de MicroSim con HTML, CSS y documentación.
