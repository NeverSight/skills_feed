تنشئ هذه المهارة تصورات تفاعلية للخط الزمني باستخدام مكتبة JavaScript الخاصة بالخط الزمني. استخدم هذه المهارة عندما يحتاج المستخدمون إلى إنشاء جداول زمنية تاريخية، أو جداول زمنية للمشروع، أو تسلسلات للأحداث، أو أي تصور للبيانات بتسلسل زمني مع تصفية اختيارية للفئة. تقوم المهارة بإنشاء حزمة MicroSim كاملة مع بيانات HTML وCSS وJSON والوثائق.
