Этот навык генерирует интерактивные визуализации временной шкалы с использованием библиотеки JavaScript vis-timeline. Используйте этот навык, когда пользователям необходимо создать исторические временные шкалы, временные шкалы проекта, последовательности событий или любую визуализацию хронологических данных с дополнительной фильтрацией по категориям. Этот навык создает полный пакет MicroSim с данными HTML, CSS, JSON и документацией.
