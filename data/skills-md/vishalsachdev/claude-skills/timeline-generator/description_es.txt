Esta habilidad genera visualizaciones de línea de tiempo interactivas utilizando la biblioteca JavaScript vis-timeline. Utilice esta habilidad cuando los usuarios necesiten crear cronogramas históricos, cronogramas de proyectos, secuencias de eventos o cualquier visualización de datos cronológicos con filtrado de categorías opcional. La habilidad crea un paquete MicroSim completo con HTML, CSS, datos JSON y documentación.
