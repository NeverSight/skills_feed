Questa competenza genera visualizzazioni interattive della sequenza temporale utilizzando la libreria JavaScript vis-timeline. Utilizza questa competenza quando gli utenti devono creare sequenze temporali storiche, sequenze temporali di progetti, sequenze di eventi o qualsiasi visualizzazione di dati cronologici con filtro di categoria opzionale. La competenza crea un pacchetto MicroSim completo con dati HTML, CSS, JSON e documentazione.
