تقوم هذه المهارة بتثبيت نظام كامل لتتبع مهارات Claude Code باستخدام الخطافات لتسجيل استخدام المهارة ومدة التنفيذ واستخدام الرمز المميز ومطالبات المستخدم للتحليل لاحقًا تلقائيًا. استخدم هذه المهارة عند إعداد تتبع النشاط في مشروع Claude Code لتحديد الأنماط ومراقبة التكاليف واكتشاف الفرص لمهارات جديدة.
