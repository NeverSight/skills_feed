Этот навык устанавливает полную систему отслеживания навыков Claude Code с использованием перехватчиков для автоматической регистрации использования навыков, продолжительности выполнения, использования токенов и запросов пользователя для последующего анализа. Используйте этот навык при настройке отслеживания активности в проекте Claude Code, чтобы выявлять закономерности, отслеживать затраты и открывать возможности для новых навыков.
