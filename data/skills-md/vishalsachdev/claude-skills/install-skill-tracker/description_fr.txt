Cette compétence installe un système complet de suivi des compétences Claude Code à l'aide de hooks pour enregistrer automatiquement l'utilisation des compétences, la durée d'exécution, l'utilisation des jetons et les invites de l'utilisateur pour une analyse ultérieure. Utilisez cette compétence lors de la configuration du suivi des activités dans un projet Claude Code pour identifier les modèles, surveiller les coûts et découvrir des opportunités pour de nouvelles compétences.
