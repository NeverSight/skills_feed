Questa competenza installa un sistema completo di monitoraggio delle competenze di Claude Code utilizzando hook per registrare automaticamente l'utilizzo delle competenze, la durata dell'esecuzione, l'utilizzo dei token e le richieste dell'utente per un'analisi successiva. Utilizza questa competenza quando imposti il ​​monitoraggio delle attività in un progetto Claude Code per identificare modelli, monitorare i costi e scoprire opportunità per nuove competenze.
