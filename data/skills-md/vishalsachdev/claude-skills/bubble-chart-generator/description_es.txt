Esta habilidad genera visualizaciones interactivas de gráficos de burbujas Chart.js para matrices de prioridad y análisis de datos multidimensionales. Utilice esta habilidad cuando los usuarios necesiten crear diagramas de dispersión con tamaños de burbujas variables, particularmente para matrices de prioridad de 2x2 (Impacto vs Esfuerzo, Riesgo vs Valor, etc.), análisis de cartera o cualquier visualización que compare elementos en dos dimensiones con una tercera dimensión representada por tamaño. La habilidad crea un paquete MicroSim completo con HTML, CSS y documentación.
