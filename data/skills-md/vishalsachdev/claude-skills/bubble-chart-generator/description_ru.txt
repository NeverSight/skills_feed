Этот навык генерирует интерактивные визуализации пузырьковых диаграмм Chart.js для матриц приоритетов и многомерного анализа данных. Используйте этот навык, когда пользователям необходимо создать точечные диаграммы с пузырьками переменного размера, особенно для матриц приоритетов 2x2 (Воздействие против Усилий, Риск против Ценности и т. д.), анализа портфеля или любой визуализации, сравнивающей элементы по двум измерениям с третьим измерением, представленным размером. Этот навык создает полный пакет MicroSim с HTML, CSS и документацией.
