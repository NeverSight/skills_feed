Создавайте научно обоснованные шаблоны обратной связи для заданий Canvas. Используйте его, когда преподавателям нужны шаблоны обратной связи для выставления оценок, они хотят создавать комментарии к рубрикам, нуждаются в языке поощряющей обратной связи, соответствующем принципам компонента 3 (целевой, поощряющий, немедленный) или хотят настроить библиотеки комментариев SpeedGrader. Работает с Canvas-MCP для массовой цветокоррекции.
