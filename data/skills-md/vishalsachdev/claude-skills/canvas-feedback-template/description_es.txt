Genere plantillas de comentarios de aprendizaje respaldadas por la ciencia para las tareas de Canvas. Úselo cuando los educadores necesiten plantillas de comentarios para calificar, quieran crear comentarios de rúbricas, necesiten un lenguaje de comentarios alentador alineado con los principios del Pilar 3 (dirigido, alentador, inmediato) o quieran configurar bibliotecas de comentarios de SpeedGrader. Funciona con canvas-mcp para clasificación masiva.
