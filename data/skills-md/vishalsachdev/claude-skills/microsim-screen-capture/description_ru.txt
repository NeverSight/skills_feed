Этот навык автоматизирует создание высококачественных снимков экрана для визуализаций MicroSim с использованием безголового режима Chrome. Используйте этот навык при работе с MicroSims, которым нужны изображения предварительного просмотра для публикации в социальных сетях, документирования или оценки качества. Навык обрабатывает визуализации с большим количеством JavaScript, требующие надлежащего времени рендеринга и внешних ресурсов CDN.
