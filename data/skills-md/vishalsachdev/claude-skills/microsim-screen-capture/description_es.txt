Esta habilidad automatiza la captura de capturas de pantalla de alta calidad para visualizaciones de MicroSim utilizando el modo sin cabeza de Chrome. Utilice esta habilidad cuando trabaje con MicroSims que necesiten imágenes de vista previa para compartir en redes sociales, documentación o evaluación de calidad. La habilidad maneja visualizaciones con mucho JavaScript que requieren un tiempo de renderizado adecuado y recursos CDN externos.
