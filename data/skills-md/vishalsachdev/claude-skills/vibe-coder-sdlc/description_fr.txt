Cette compétence doit être utilisée lorsque les utilisateurs souhaitent des pratiques SDLC professionnelles avec des flux de travail Git automatisés, une révision intelligente du code et une gestion de projet sans processus rigides. Orchestre des agents spécialisés pour le développement de fonctionnalités, la gestion des branches, la révision du code spécifique au langage et l'intégration de GitHub. Idéal pour les développeurs qui souhaitent rester fluides tout en conservant la qualité du code.
