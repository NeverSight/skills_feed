Этот навык анализирует или создает описания курсов для интеллектуальных учебников, проверяя полноту необходимых элементов (название, аудитория, предварительные условия, темы, результаты Таксономии Блума) и предоставляя оценки качества с предложениями по улучшению. Используйте этот навык при работе с описаниями курсов в /docs/course-description.md, которые требуют проверки или создания для создания обучающих графиков.
