Этот навык генерирует интерактивные графики математических функций с использованием Plotly.js для встраивания iframe в интеллектуальные учебники. Создает визуализации с всплывающими подсказками при наведении, интерактивными ползунками для перемещения точек вдоль кривых, адаптивным дизайном, оптимизированным для узких макетов, и подробной учебной документацией. Используйте это, когда пользователи запрашивают построение математических функций, графическое отображение уравнений, визуализацию f(x) или создание интерактивных обозревателей функций для курсов по исчислению, предварительным расчетам, физике или инженерному делу.
