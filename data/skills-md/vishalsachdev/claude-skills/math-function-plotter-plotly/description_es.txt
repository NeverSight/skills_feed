Esta habilidad genera gráficos de funciones matemáticas interactivas utilizando Plotly.js para incrustar iframe en libros de texto inteligentes. Crea visualizaciones con información sobre herramientas al pasar el mouse, controles deslizantes interactivos para mover puntos a lo largo de curvas, diseño responsivo optimizado para diseños estrechos y documentación educativa completa. Utilícelo cuando los usuarios soliciten trazar funciones matemáticas, graficar ecuaciones, visualizar f(x) o crear exploradores de funciones interactivos para cursos de cálculo, precálculo, física o ingeniería.
