يقوم بإجراء مراجعات شاملة للتعليمات البرمجية باتباع أفضل ممارسات الصناعة. يُستخدم عند مراجعة طلبات السحب أو تغييرات التعليمات البرمجية أو عندما يُطلب منك تحليل جودة التعليمات البرمجية أو الأمان أو الأداء أو إمكانية الصيانة. التحقق من الأخطاء الشائعة والثغرات الأمنية وروائح التعليمات البرمجية والالتزام بمعايير الترميز.
