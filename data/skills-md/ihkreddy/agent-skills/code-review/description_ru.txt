Выполняет комплексные проверки кода в соответствии с лучшими отраслевыми практиками. Используйте при проверке запросов на включение, изменений кода или когда вас просят проанализировать качество, безопасность, производительность или удобство обслуживания кода. Проверяет наличие распространенных ошибок, уязвимостей безопасности, запахов кода и соблюдение стандартов кодирования.
