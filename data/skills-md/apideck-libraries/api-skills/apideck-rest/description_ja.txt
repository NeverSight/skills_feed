あらゆる言語の Apideck Unified REST API リファレンス。会計ソフトウェア (QuickBooks、Xero、NetSuite)、CRM (Salesforce、HubSpot、Pipedrive)、HRIS プラットフォーム (Workday、BambooHR)、ファイル ストレージ (Google Drive、Dropbox、Box)、ATS システム (Greenhouse、Lever)、電子商取引、または直接 HTTP 呼び出しを使用する Apideck の 200 以上のコネクタとの統合を構築する場合に使用します。認証ヘッダー、CRUD 操作、カーソルベースのページネーション、フィルタリング、並べ替え、エラー処理、レート制限、パススルー パラメーター、Webhook について説明します。言語に依存しない —curl、fetch、axios、httpx、または任意の HTTP クライアントで動作します。
