适用于任何语言的 Apideck 统一 REST API 参考。在使用直接 HTTP 调用与会计软件（QuickBooks、Xero、NetSuite）、CRM（Salesforce、HubSpot、Pipedrive）、HRIS 平台（Workday、BambooHR）、文件存储（Google Drive、Dropbox、Box）、ATS 系统（Greenhouse、Lever）、电子商务或任何 Apideck 的 200 多个连接器构建集成时使用。涵盖身份验证标头、CRUD 操作、基于游标的分页、过滤、排序、错误处理、速率限制、传递参数和 Webhooks。与语言无关——适用于curl、fetch、axios、httpx或任何HTTP客户端。
