Lorsque l'utilisateur souhaite définir son profil client idéal, positionner un produit d'IA, créer une architecture de messagerie ou valider l'adéquation produit-marché. À utiliser également lorsque l'utilisateur mentionne « ICP », « profil client idéal », « positionnement », « PMF », « adéquation produit-marché », « messagerie », « personnalité de l'acheteur », « signaux d'enrichissement », « positionnement sur le marché » ou « positionnement concurrentiel ». Cette compétence couvre le positionnement sur le marché, la définition ICP, l'architecture de messagerie et la validation PMF pour les produits natifs d'IA.
