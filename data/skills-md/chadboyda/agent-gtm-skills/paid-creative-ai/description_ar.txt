عندما يريد المستخدم إنشاء تصميم إعلاني تم إنشاؤه بواسطة الذكاء الاصطناعي، أو اختبار تصميم الأداء، أو إدارة التعب الإبداعي، أو تحسين الوسائط المدفوعة باستخدام أدوات الذكاء الاصطناعي. يُستخدم أيضًا عندما يذكر المستخدم "تصميم الإعلان" أو "تصميم الأداء" أو "الاختبار الإبداعي" أو "التعب الإبداعي" أو "إعلانات التعريف" أو "إعلانات Google" أو "إعلانات TikTok" أو "إعلانات الذكاء الاصطناعي" أو "ميزانية الإعلان" أو "عائد النفقات الإعلانية" أو "المميزات+" أو "الأداء الأقصى". تغطي هذه المهارة التصميم الإبداعي المدفوع المدعوم بالذكاء الاصطناعي منذ الأجيال وحتى تحسين الأداء.
