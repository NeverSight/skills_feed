Этот навык предоставляет подробные инструкции по взаимодействию с Notion API посредством вызовов REST. Этот навык следует использовать всякий раз, когда пользователь просит взаимодействовать с Notion, включая чтение, создание, обновление или удаление страниц, баз данных, блоков, комментариев или любого другого контента Notion. Навыки охватывают аутентификацию, все доступные конечные точки, нумерацию страниц, обработку ошибок и лучшие практики.
