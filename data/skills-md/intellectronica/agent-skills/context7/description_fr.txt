Récupérez une documentation à jour pour les bibliothèques logicielles, les frameworks et les composants via l'API Context7. Cette compétence doit être utilisée lors de la recherche de documentation sur une bibliothèque ou un framework de programmation, la recherche d'exemples de code pour des API ou des fonctionnalités spécifiques, la vérification de l'utilisation correcte des fonctions de bibliothèque ou l'obtention d'informations actuelles sur les API de bibliothèque qui peuvent avoir changé depuis la formation.
