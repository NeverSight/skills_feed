Конвертируйте документы и файлы в Markdown с помощью markitdown. Используйте при преобразовании PDF, Word (.docx), PowerPoint (.pptx), Excel (.xlsx, .xls), HTML, CSV, JSON, XML, изображений (с EXIF/OCR), аудио (с транскрипцией), ZIP-архивов, URL-адресов YouTube или EPubs в формат Markdown для обработки LLM или анализа текста.
