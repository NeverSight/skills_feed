Converti documenti e file in Markdown utilizzando markitdown. Da utilizzare durante la conversione di PDF, Word (.docx), PowerPoint (.pptx), Excel (.xlsx, .xls), HTML, CSV, JSON, XML, immagini (con EXIF/OCR), audio (con trascrizione), archivi ZIP, URL di YouTube o EPub nel formato Markdown per l'elaborazione LLM o l'analisi del testo.
