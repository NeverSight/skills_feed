تحويل المستندات والملفات إلى Markdown باستخدام markitdown. يُستخدم عند تحويل PDF، أو Word (.docx)، أو PowerPoint (.pptx)، أو Excel (.xlsx، .xls)، أو HTML، أو CSV، أو JSON، أو XML، أو الصور (باستخدام EXIF/OCR)، أو الصوت (مع النسخ)، أو أرشيفات ZIP، أو عناوين URL على YouTube، أو EPubs إلى تنسيق Markdown لمعالجة LLM أو تحليل النص.
