Cette compétence fournit des instructions pour interagir avec l'API REST v2 de Todoist à l'aide de curl et jq. Il couvre l'authentification, les opérations CRUD pour les tâches/projets/sections/étiquettes/commentaires, la gestion de la pagination et nécessite une confirmation avant les actions destructrices. Utilisez cette compétence lorsque l'utilisateur souhaite lire, créer, mettre à jour ou supprimer des données Todoist via l'API.
