Esta habilidad proporciona instrucciones para interactuar con Todoist REST API v2 usando curl y jq. Cubre autenticación, operaciones CRUD para tareas/proyectos/secciones/etiquetas/comentarios, manejo de paginación y requiere confirmación antes de acciones destructivas. Usa esta habilidad cuando el usuario quiera leer, crear, actualizar o eliminar datos de Todoist a través de la API.
