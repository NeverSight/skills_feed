Этот навык содержит инструкции по взаимодействию с Todoist REST API v2 с использованием Curl и jq. Он охватывает аутентификацию, операции CRUD для задач/проектов/разделов/меток/комментариев, обработку страниц и требует подтверждения перед деструктивными действиями. Используйте этот навык, когда пользователь хочет прочитать, создать, обновить или удалить данные Todoist через API.
