Questa Skill fornisce istruzioni per interagire con l'API REST di Todoist v2 utilizzando curl e jq. Copre l'autenticazione, le operazioni CRUD per attività/progetti/sezioni/etichette/commenti, la gestione dell'impaginazione e richiede la conferma prima di azioni distruttive. Utilizza questa abilità quando l'utente desidera leggere, creare, aggiornare o eliminare i dati di Todoist tramite l'API.
