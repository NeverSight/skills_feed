Dieser Skill bietet Anweisungen für die Interaktion mit der Todoist REST API v2 unter Verwendung von Curl und JQ. Es umfasst Authentifizierung, CRUD-Operationen für Aufgaben/Projekte/Abschnitte/Beschriftungen/Kommentare, Paginierungsbehandlung und erfordert eine Bestätigung vor destruktiven Aktionen. Verwenden Sie diesen Skill, wenn der Benutzer Todoist-Daten über die API lesen, erstellen, aktualisieren oder löschen möchte.
