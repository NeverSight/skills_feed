Этот навык предоставляет подробные инструкции по взаимодействию со службой закладок Raindrop.io через REST API с использованием Curl и jq. Он охватывает аутентификацию, операции CRUD для коллекций, капель дождя (закладок), тегов, выделения, фильтров, импорта/экспорта и резервных копий. Используйте этот навык всякий раз, когда пользователь просит поработать со своими закладками из Raindrop.io, включая чтение, создание, обновление, удаление, поиск или организацию закладок и коллекций.
