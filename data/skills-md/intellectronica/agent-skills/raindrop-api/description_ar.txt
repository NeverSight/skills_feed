توفر هذه المهارة إرشادات شاملة للتفاعل مع خدمة الإشارات المرجعية Raindrop.io عبر REST API الخاصة بها باستخدام curl وjq. ويغطي المصادقة وعمليات CRUD للمجموعات وقطرات المطر (الإشارات المرجعية) والعلامات والإبرازات والمرشحات والاستيراد/التصدير والنسخ الاحتياطية. استخدم هذه المهارة عندما يطلب المستخدم العمل مع الإشارات المرجعية الخاصة به من Raindrop.io، بما في ذلك القراءة أو الإنشاء أو التحديث أو الحذف أو البحث أو تنظيم الإشارات المرجعية والمجموعات.
