Esta habilidad proporciona instrucciones completas para interactuar con el servicio de marcadores Raindrop.io a través de su API REST usando curl y jq. Cubre autenticación, operaciones CRUD para colecciones, gotas de lluvia (marcadores), etiquetas, resaltados, filtros, importación/exportación y copias de seguridad. Utilice esta habilidad siempre que el usuario solicite trabajar con sus marcadores de Raindrop.io, lo que incluye leer, crear, actualizar, eliminar, buscar u organizar marcadores y colecciones.
