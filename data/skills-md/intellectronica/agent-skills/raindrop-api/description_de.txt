Dieser Skill bietet umfassende Anweisungen für die Interaktion mit dem Lesezeichendienst Raindrop.io über seine REST-API unter Verwendung von Curl und JQ. Es umfasst Authentifizierung, CRUD-Vorgänge für Sammlungen, Regentropfen (Lesezeichen), Tags, Hervorhebungen, Filter, Import/Export und Backups. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer mit seinen Lesezeichen von Raindrop.io arbeiten möchte, einschließlich Lesen, Erstellen, Aktualisieren, Löschen, Suchen oder Organisieren von Lesezeichen und Sammlungen.
