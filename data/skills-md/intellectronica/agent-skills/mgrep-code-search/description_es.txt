Búsqueda de código semántico utilizando mgrep para una exploración eficiente de la base de código. Esta habilidad se debe utilizar al buscar o explorar bases de código con más de 30 archivos no ignorados por Git y/o estructuras de directorios anidados. Proporciona búsqueda semántica en lenguaje natural que complementa el grep/ripgrep tradicional para encontrar características, comprender la intención y explorar código desconocido.
