Semantische Codesuche mit mgrep für eine effiziente Codebasis-Erkundung. Diese Fähigkeit sollte beim Durchsuchen oder Erkunden von Codebasen mit mehr als 30 nicht gitignorierten Dateien und/oder verschachtelten Verzeichnisstrukturen verwendet werden. Es bietet eine semantische Suche in natürlicher Sprache, die das herkömmliche grep/ripgrep ergänzt, um Funktionen zu finden, Absichten zu verstehen und unbekannten Code zu erkunden.
