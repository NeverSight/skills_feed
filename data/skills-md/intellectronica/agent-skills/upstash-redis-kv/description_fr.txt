Lisez et écrivez dans un magasin de valeurs-clés compatible Upstash Redis via l'API REST. À utiliser lorsqu'il est nécessaire de sauvegarder ou de récupérer des données clé-valeur, d'utiliser les fonctionnalités Redis (mise en cache, compteurs, listes, ensembles, hachages, ensembles triés, etc.) pour l'interaction en cours, ou lorsque l'utilisateur demande explicitement d'utiliser Upstash ou Redis.
