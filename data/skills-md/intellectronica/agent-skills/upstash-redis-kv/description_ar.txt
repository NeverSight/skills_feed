القراءة والكتابة إلى متجر القيمة الرئيسية المتوافق مع Upstash Redis عبر REST API. يُستخدم عندما تكون هناك حاجة لحفظ أو استرداد بيانات القيمة الرئيسية، أو استخدم ميزات Redis (التخزين المؤقت، والعدادات، والقوائم، والمجموعات، والتجزئة، والمجموعات المصنفة، وما إلى ذلك) للتفاعل الحالي، أو عندما يطلب المستخدم صراحةً استخدام Upstash أو Redis.
