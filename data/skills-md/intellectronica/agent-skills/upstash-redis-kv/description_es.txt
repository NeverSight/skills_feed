Lea y escriba en el almacén de valores clave compatible con Upstash Redis a través de la API REST. Úselo cuando sea necesario guardar o recuperar datos clave-valor, usar funciones de Redis (almacenamiento en caché, contadores, listas, conjuntos, hashes, conjuntos ordenados, etc.) para la interacción actual, o cuando el usuario solicite explícitamente usar Upstash o Redis.
