Google の Nano Banana Pro (Gemini 3 Pro Image) API を使用して画像を生成および編集します。ユーザーが画像の生成、作成、編集、変更、変更、変更、または更新を要求する場合に使用します。ユーザーが既存の画像ファイルを参照し、何らかの方法でそれを変更するように要求する場合にも使用します (例: 「この画像を変更する」、「背景を変更する」、「X を Y に置き換える」)。テキストから画像への生成と、構成可能な解像度 (デフォルトでは 1K、高解像度の場合は 2K、または 4K) による画像から画像への編集の両方をサポートします。最初にイメージ ファイルを読み取らないでください。このスキルを --input-image パラメータとともに直接使用してください。
