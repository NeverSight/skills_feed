Genere y edite imágenes utilizando la API Nano Banana Pro (Gemini 3 Pro Image) de Google. Se utiliza cuando el usuario solicita generar, crear, editar, modificar, cambiar, alterar o actualizar imágenes. También se utiliza cuando el usuario hace referencia a un archivo de imagen existente y solicita modificarlo de alguna manera (por ejemplo, "modificar esta imagen", "cambiar el fondo", "reemplazar X con Y"). Admite tanto la generación de texto a imagen como la edición de imagen a imagen con resolución configurable (1K por defecto, 2K o 4K para alta resolución). NO lea el archivo de imagen primero; use esta habilidad directamente con el parámetro --input-image.
