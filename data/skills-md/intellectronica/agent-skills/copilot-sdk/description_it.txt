Questa competenza fornisce indicazioni per la creazione di agenti e applicazioni con GitHub Copilot SDK. Dovrebbe essere utilizzato quando l'utente desidera creare, modificare o lavorare su software che utilizza GitHub Copilot SDK in TypeScript, Python, Go o .NET. La competenza copre i modelli di utilizzo dell'SDK, la configurazione della CLI, gli strumenti personalizzati, i server MCP e gli agenti personalizzati.
