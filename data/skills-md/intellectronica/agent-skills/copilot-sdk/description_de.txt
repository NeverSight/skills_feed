Dieser Skill bietet Anleitungen zum Erstellen von Agenten und Anwendungen mit dem GitHub Copilot SDK. Es sollte verwendet werden, wenn der Benutzer Software erstellen, ändern oder daran arbeiten möchte, die das GitHub Copilot SDK in TypeScript, Python, Go oder .NET verwendet. Die Fertigkeit umfasst SDK-Nutzungsmuster, CLI-Konfiguration, benutzerdefinierte Tools, MCP-Server und benutzerdefinierte Agenten.
