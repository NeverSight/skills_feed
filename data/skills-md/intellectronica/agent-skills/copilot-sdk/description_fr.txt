Cette compétence fournit des conseils pour créer des agents et des applications avec le SDK GitHub Copilot. Il doit être utilisé lorsque l'utilisateur souhaite créer, modifier ou travailler sur un logiciel qui utilise le SDK GitHub Copilot dans TypeScript, Python, Go ou .NET. La compétence couvre les modèles d'utilisation du SDK, la configuration CLI, les outils personnalisés, les serveurs MCP et les agents personnalisés.
