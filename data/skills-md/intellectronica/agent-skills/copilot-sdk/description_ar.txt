توفر هذه المهارة إرشادات لإنشاء الوكلاء والتطبيقات باستخدام GitHub Copilot SDK. يجب استخدامه عندما يريد المستخدم إنشاء أو تعديل أو العمل على برنامج يستخدم GitHub Copilot SDK في TypeScript أو Python أو Go أو .NET. تغطي المهارة أنماط استخدام SDK، وتكوين واجهة سطر الأوامر (CLI)، والأدوات المخصصة، وخوادم MCP، والوكلاء المخصصين.
