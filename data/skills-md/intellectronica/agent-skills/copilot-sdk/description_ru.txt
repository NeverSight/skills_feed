Этот навык дает рекомендации по созданию агентов и приложений с помощью GitHub Copilot SDK. Его следует использовать, когда пользователь хочет создавать, изменять или работать с программным обеспечением, использующим GitHub Copilot SDK в TypeScript, Python, Go или .NET. Навыки охватывают шаблоны использования SDK, настройку CLI, специальные инструменты, серверы MCP и специальные агенты.
