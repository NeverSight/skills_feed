Esta habilidad proporciona orientación para crear agentes y aplicaciones con el SDK de GitHub Copilot. Debe usarse cuando el usuario desea crear, modificar o trabajar en software que utiliza el SDK de GitHub Copilot en TypeScript, Python, Go o .NET. La habilidad cubre patrones de uso de SDK, configuración de CLI, herramientas personalizadas, servidores MCP y agentes personalizados.
