Guía para implementar acciones del servidor Next.js usando ZSA (Zod Server Actions) con autenticación, validación e integración de React Query. Úselo al crear puntos finales de API, controladores de formularios, mutaciones o cualquier operación de datos del lado del servidor. Mejores prácticas para crear acciones de servidor Next.js con seguridad de tipos con ZSA (Zod Server Actions). Utilice esta habilidad al crear acciones de servidor validadas, implementar procedimientos de autenticación/autorización, manejar envíos de formularios, integrar con React Query, administrar errores o crear puntos finales seguros similares a API en Next.js App Router. Cubre validación de entrada/salida, procedimientos (middleware), devoluciones de llamadas, actualizaciones optimistas, reintentos, tiempos de espera y enlaces del lado del cliente.
