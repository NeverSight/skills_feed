Guida per l'implementazione delle azioni del server Next.js utilizzando ZSA (Zod Server Actions) con autenticazione, convalida e integrazione di React Query. Da utilizzare durante la creazione di endpoint API, gestori di moduli, mutazioni o qualsiasi operazione sui dati lato server. Procedure consigliate per la creazione di azioni server Next.js indipendenti dai tipi con ZSA (Zod Server Actions). Utilizza questa competenza durante la creazione di azioni server convalidate, l'implementazione di procedure di autenticazione/autorizzazione, la gestione dell'invio di moduli, l'integrazione con React Query, la gestione degli errori o la creazione di endpoint sicuri simili ad API in Next.js App Router. Copre la convalida di input/output, procedure (middleware), callback, aggiornamenti ottimistici, tentativi, timeout e hook lato client.
