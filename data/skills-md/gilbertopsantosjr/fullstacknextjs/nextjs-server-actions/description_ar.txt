دليل لتنفيذ إجراءات خادم Next.js باستخدام ZSA (إجراءات خادم Zod) مع المصادقة والتحقق وتكامل React Query. يُستخدم عند إنشاء نقاط نهاية API أو معالجات النماذج أو الطفرات أو أي عمليات بيانات من جانب الخادم. أفضل الممارسات لإنشاء إجراءات خادم Next.js الآمنة من النوع باستخدام ZSA (Zod Server Actions). استخدم هذه المهارة عند إنشاء إجراءات خادم تم التحقق من صحتها، أو تنفيذ إجراءات المصادقة/الترخيص، أو التعامل مع عمليات إرسال النماذج، أو التكامل مع React Query، أو إدارة الأخطاء، أو إنشاء نقاط نهاية آمنة تشبه واجهة برمجة التطبيقات في Next.js App Router. يغطي التحقق من صحة الإدخال/الإخراج، والإجراءات (البرامج الوسيطة)، وعمليات الاسترجاعات، والتحديثات المتفائلة، وإعادة المحاولة، والمهلات، والخطافات من جانب العميل.
