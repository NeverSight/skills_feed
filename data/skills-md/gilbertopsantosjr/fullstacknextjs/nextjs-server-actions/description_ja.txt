認証、検証、および React Query の統合を備えた ZSA (Zod サーバー アクション) を使用して Next.js サーバー アクションを実装するためのガイド。 API エンドポイント、フォーム ハンドラー、ミューテーション、またはサーバー側のデータ操作を作成するときに使用します。 ZSA (Zod サーバー アクション) を使用してタイプ セーフな Next.js サーバー アクションを構築するためのベスト プラクティス。このスキルは、検証済みのサーバー アクションの作成、認証/認可手順の実装、フォーム送信の処理、React Query との統合、エラーの管理、または Next.js App Router で安全な API のようなエンドポイントの構築を行うときに使用します。入出力検証、プロシージャ (ミドルウェア)、コールバック、オプティミスティック更新、再試行、タイムアウト、クライアント側フックをカバーします。
