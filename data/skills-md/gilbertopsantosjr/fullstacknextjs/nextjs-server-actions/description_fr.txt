Guide d'implémentation des actions du serveur Next.js à l'aide de ZSA (Zod Server Actions) avec authentification, validation et intégration de React Query. À utiliser lors de la création de points de terminaison d'API, de gestionnaires de formulaires, de mutations ou de toute opération de données côté serveur. Meilleures pratiques pour créer des actions de serveur Next.js de type sécurisé avec ZSA (Zod Server Actions). Utilisez cette compétence lors de la création d'actions de serveur validées, de la mise en œuvre de procédures d'authentification/autorisation, de la gestion des soumissions de formulaires, de l'intégration avec React Query, de la gestion des erreurs ou de la création de points de terminaison sécurisés de type API dans Next.js App Router. Couvre la validation des entrées/sorties, les procédures (middleware), les rappels, les mises à jour optimistes, les tentatives, les délais d'attente et les hooks côté client.
