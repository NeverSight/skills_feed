Эксперт Bun AWS Lambda по созданию, развертыванию и оптимизации функций Lambda с использованием среды выполнения Bun. Используйте при создании обработчиков Lambda с помощью Bun, настройке пользовательских сред выполнения, создании образов контейнеров для Lambda, настройке артефактов развертывания или преобразовании обработчиков Node.js Lambda в Bun. Запускается по запросам, включающим Bun + Lambda, бессерверным функциям с Bun, обработке событий Lambda (API Gateway, SQS, SNS, EventBridge, S3, DynamoDB Streams), оптимизации холодного запуска или шаблонам развертывания Lambda.
