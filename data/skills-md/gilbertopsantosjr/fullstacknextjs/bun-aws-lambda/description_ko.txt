Bun 런타임을 사용하여 Lambda 함수를 생성, 배포 및 최적화하는 AWS Lambda 전문가입니다. Bun을 사용하여 Lambda 핸들러를 구축하거나, 사용자 지정 런타임을 설정하거나, Lambda용 컨테이너 이미지를 생성하거나, 배포 아티팩트를 구성하거나, Node.js Lambda 핸들러를 Bun으로 변환할 때 사용합니다. Bun + Lambda, Bun이 포함된 서버리스 기능, Lambda 이벤트 처리(API Gateway, SQS, SNS, EventBridge, S3, DynamoDB Streams), 콜드 스타트 ​​최적화 또는 Lambda 배포 패턴과 관련된 요청에 대해 트리거됩니다.
