خبير Bun AWS Lambda في إنشاء وظائف Lambda ونشرها وتحسينها باستخدام وقت تشغيل Bun. يُستخدم عند إنشاء معالجات Lambda باستخدام Bun، أو إعداد أوقات تشغيل مخصصة، أو إنشاء صور حاوية لـ Lambda، أو تكوين عناصر النشر، أو تحويل معالجات Node.js Lambda إلى Bun. تشغيل الطلبات التي تتضمن Bun + Lambda، أو الوظائف بدون خادم مع Bun، أو معالجة أحداث Lambda (بوابة API، أو SQS، أو SNS، أو EventBridge، أو S3، أو DynamoDB Streams)، أو تحسين البدء البارد، أو أنماط نشر Lambda.
