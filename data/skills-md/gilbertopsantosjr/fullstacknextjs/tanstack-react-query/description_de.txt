TanStack React Query-Experte für Datenabruf und Mutationen in React-Anwendungen. Wird verwendet, wenn Sie mit useQuery, useMutation, Cache-Invalidierung, optimistischen Updates, Abfrageschlüsseln arbeiten oder Serveraktionen mit React Query über @saas4dev/core-Hooks integrieren (useServerActionQuery, useServerActionMutation, useServerActionInfiniteQuery). Wird bei Anfragen ausgelöst, die das Abrufen von API-Daten, die Verwaltung des Serverstatus, Cache-Strategien oder die Konvertierung von Fetch/UseEffect-Mustern in React Query betreffen.
