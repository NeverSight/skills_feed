Esperto di TanStack React Query per il recupero dei dati e le mutazioni nelle applicazioni React. Utilizzare quando si lavora con useQuery, useMutation, invalidazione della cache, aggiornamenti ottimistici, chiavi di query o integrazione di azioni del server con React Query tramite hook @saas4dev/core (useServerActionQuery, useServerActionMutation, useServerActionInfiniteQuery). Trigger su richieste che coinvolgono il recupero dei dati API, la gestione dello stato del server, le strategie della cache o la conversione di modelli fetch/useEffect in React Query.
