Expert TanStack React Query pour la récupération de données et les mutations dans les applications React. À utiliser lorsque vous travaillez avec useQuery, useMutation, l'invalidation du cache, les mises à jour optimistes, les clés de requête ou l'intégration d'actions de serveur avec React Query via des hooks @saas4dev/core (useServerActionQuery, useServerActionMutation, useServerActionInfiniteQuery). Se déclenche sur des requêtes impliquant la récupération de données API, la gestion de l'état du serveur, les stratégies de cache ou la conversion de modèles fetch/useEffect en React Query.
