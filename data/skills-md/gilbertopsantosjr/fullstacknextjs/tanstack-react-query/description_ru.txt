TanStack React — эксперт по запросам для извлечения данных и изменения в приложениях React. Используйте при работе с useQuery, useMutation, аннулированием кэша, оптимистическими обновлениями, ключами запроса или интеграцией действий сервера с React Query через перехватчики @saas4dev/core (useServerActionQuery, useServerActionMutation, useServerActionInfiniteQuery). Запускается по запросам, включающим получение данных API, управление состоянием сервера, стратегии кэширования или преобразование шаблонов fetch/useEffect в React Query.
