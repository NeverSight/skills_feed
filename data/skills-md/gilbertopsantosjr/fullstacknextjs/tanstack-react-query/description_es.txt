Experto en TanStack React Query para recuperación de datos y mutaciones en aplicaciones React. Úselo cuando trabaje con useQuery, useMutation, invalidación de caché, actualizaciones optimistas, claves de consulta o integre acciones del servidor con React Query a través de enlaces @saas4dev/core (useServerActionQuery, useServerActionMutation, useServerActionInfiniteQuery). Se activan solicitudes que involucran la recuperación de datos de API, la administración del estado del servidor, las estrategias de caché o la conversión de patrones de recuperación/uso de efectos a React Query.
