خبير TanStack React Query لجلب البيانات والطفرات في تطبيقات React. يُستخدم عند العمل مع useQuery أو useMutation أو إبطال ذاكرة التخزين المؤقت أو التحديثات المتفائلة أو مفاتيح الاستعلام أو دمج إجراءات الخادم مع React Query عبر الخطافات @saas4dev/core (useServerActionQuery أو useServerActionMutation أو useServerActionInfiniteQuery). يتم تشغيلها عند الطلبات التي تتضمن جلب بيانات واجهة برمجة التطبيقات (API)، أو إدارة حالة الخادم، أو إستراتيجيات ذاكرة التخزين المؤقت، أو تحويل أنماط الجلب/استخدامEffect إلى React Query.
