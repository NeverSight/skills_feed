React 애플리케이션의 데이터 가져오기 및 변형에 대한 TanStack React Query 전문가입니다. useQuery, useMutation, 캐시 무효화, 낙관적 업데이트, 쿼리 키로 작업하거나 @saas4dev/core 후크(useServerActionQuery, useServerActionMutation, useServerActionInfiniteQuery)를 통해 React Query와 서버 작업을 통합할 때 사용합니다. API 데이터 가져오기, 서버 상태 관리, 캐시 전략 또는 fetch/useEffect 패턴을 React 쿼리로 변환하는 것과 관련된 요청에 대해 트리거합니다.
