TanStack React React アプリケーションでのデータのフェッチとミューテーションのクエリ エキスパート。 useQuery、useMutation、キャッシュの無効化、オプティミスティック更新、クエリ キーを操作する場合、または @saas4dev/core フック (useServerActionQuery、useServerActionMutation、useServerActionInfiniteQuery) を介してサーバー アクションを React Query と統合する場合に使用します。 API データの取得、サーバー状態管理、キャッシュ戦略、または fetch/useEffect パターンの React Query への変換を含むリクエストをトリガーします。
