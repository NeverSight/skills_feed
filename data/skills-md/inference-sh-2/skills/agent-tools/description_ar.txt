قم بتشغيل أكثر من 150 تطبيقًا للذكاء الاصطناعي عبر inference.sh CLI - إنشاء الصور وإنشاء الفيديو وLLMs والبحث والأبعاد الثلاثية وأتمتة Twitter. النماذج: FLUX، Veo، Gemini، Grok، Claude، Seedance، OmniHuman، Tavily، Exa، OpenRouter، وغيرها الكثير. يُستخدم عند تشغيل تطبيقات الذكاء الاصطناعي، أو إنشاء صور/مقاطع فيديو، أو الاتصال بطلاب LLM، أو البحث على الويب، أو تشغيل Twitter تلقائيًا. المشغلات: inference.sh، infsh، ai model، run ai، serverless ai، ai api، Flux، veo، claude api، توليد الصور، إنشاء الفيديو، openrouter، tavily، exa search، twitter api، grok
