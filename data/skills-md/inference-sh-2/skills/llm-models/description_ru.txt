Получите доступ к Claude, Gemini, Kimi, GLM и более чем к 100 LLM через интерфейс командной строки inference.sh с помощью OpenRouter. Модели: Claude Opus 4.5, Claude Sonnet 4.5, Claude Haiku 4.5, Gemini 3 Pro, Kimi K2, GLM-4.6, Intellect 3. Один API для всех моделей с автоматическим откатом и оптимизацией затрат. Использование для: помощников искусственного интеллекта, генерации кода, рассуждений, агентов, чата, создания контента. Триггеры: Клод API, openrouter, llm API, Клод Сонет, Клод Опус, Gemini API, Кими, языковая модель, альтернатива gpt, антропный API, API модели ИИ, доступ к llm, API чата, альтернатива Клода, альтернатива OpenAI
