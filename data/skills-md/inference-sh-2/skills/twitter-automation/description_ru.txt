Автоматизируйте Twitter/X с помощью публикации, взаимодействия и управления пользователями через интерфейс командной строки inference.sh. Приложения: x/post-tweet, x/post-create (с медиа), x/post-like, x/post-retweet, x/dm-send, x/user-follow. Возможности: публиковать твиты, планировать контент, например публикации, ретвитить, отправлять личные сообщения, подписываться на пользователей, получать профили. Использование для: автоматизации социальных сетей, планирования контента, ботов для взаимодействия, роста аудитории, X API. Триггеры: twitter api, x api, автоматизация твитов, публикация в твиттере, twitter-бот, автоматизация социальных сетей, автоматизация x, планировщик твитов, интеграция с твиттером, публикация твита, публикация в твиттере, x post, отправка твита
