Client FileMaker OData de type sécurisé avec génération de code ORM et TypeScript inspiré de Drizzle. À utiliser lorsque vous travaillez avec des bases de données FileMaker dans des projets TypeScript, interrogez des données FM, définissez des schémas typés, générez des types à partir de mises en page FM ou résolvez des problèmes de fmodata/typegen. Déclencheurs sur les tâches d'intégration FileMaker + TypeScript.
