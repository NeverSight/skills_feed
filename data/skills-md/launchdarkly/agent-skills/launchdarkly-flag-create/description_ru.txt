Создайте и настройте флаги функций LaunchDarkly таким образом, чтобы они соответствовали существующей кодовой базе. Используйте, когда пользователь хочет создать новый флаг, поместить в него код, добавить переключатель функции или настроить эксперимент. Направляет исследование существующих шаблонов перед созданием.
