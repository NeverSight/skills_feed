فرض اصطلاحات التعليمات البرمجية التي تجعل قواعد التعليمات البرمجية صديقة للوكيل وقابلة للإصلاح الذاتي. مستوحاة من فلسفة "تطوير الوكيل الأصلي" الخاصة بـ Factory AI حيث تعمل قواعد واتفاقيات الوبر كحواجز حماية لوكلاء ترميز الذكاء الاصطناعي. استخدم هذه المهارة عند إنشاء التعليمات البرمجية أو مراجعتها أو إعادة هيكلتها بأي لغة أو إطار عمل. يتم تشغيلها عند: إنشاء التعليمات البرمجية، أو إنشاء الملفات، أو إعادة البناء، أو مراجعة التعليمات البرمجية، أو الميزات الجديدة، أو إصلاحات الأخطاء، أو عمليات الترحيل، أو أي مهمة تنتج التعليمات البرمجية المصدر. يغطي تنظيم الملفات، والتسمية، وإمكانية البحث، والاختبار، ومعالجة الأخطاء، والتسجيل، والأمن، والحدود المعمارية. المراجع الخاصة بالمكدس متاحة لـ React وVue وAngular وLaravel و.NET وPython وGo وRust والمزيد.
