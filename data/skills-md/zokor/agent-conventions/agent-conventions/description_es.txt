Aplique convenciones de código que hagan que las bases de código sean fáciles de usar para los agentes y con capacidad de recuperación automática. Inspirado en la filosofía de "desarrollo de agentes nativos" de Factory AI, donde las reglas y convenciones de pelusa actúan como barreras de seguridad para los agentes de codificación de IA. Utilice esta habilidad siempre que genere, revise o refactorice código en cualquier lenguaje o marco. Activadores de: generación de código, creación de archivos, refactorización, revisión de código, nuevas funciones, corrección de errores, migraciones o cualquier tarea que produzca código fuente. Cubre la organización de archivos, los nombres, la capacidad de búsqueda, las pruebas, el manejo de errores, el registro, la seguridad y los límites arquitectónicos. Referencias específicas de pila disponibles para React, Vue, Angular, Laravel, .NET, Python, Go, Rust y más.
