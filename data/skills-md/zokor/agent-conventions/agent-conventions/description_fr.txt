Appliquez des conventions de code qui rendent les bases de code conviviales et auto-réparatrices. Inspiré par la philosophie de « développement natif d'agents » de Factory AI, où les règles et conventions de charpie agissent comme des garde-fous pour les agents de codage de l'IA. Utilisez cette compétence chaque fois que vous générez, révisez ou refactorisez du code dans n’importe quel langage ou framework. Déclencheurs sur : la génération de code, la création de fichiers, la refactorisation, la révision du code, les nouvelles fonctionnalités, les corrections de bogues, les migrations ou toute tâche produisant du code source. Couvre l'organisation des fichiers, la dénomination, la possibilité de recherche, les tests, la gestion des erreurs, la journalisation, la sécurité et les limites architecturales. Références spécifiques à la pile disponibles pour React, Vue, Angular, Laravel, .NET, Python, Go, Rust, etc.
