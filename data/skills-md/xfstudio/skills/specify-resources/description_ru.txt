Общие ресурсы (скрипты, шаблоны) для рабочего процесса разработки на основе спецификаций. Этот навык является зависимостью для всех навыков spec-* и НЕ должен вызываться пользователем напрямую. Он предоставляет сценарии bash для управления ветвями функций, проверки предварительных требований и шаблоны для спецификаций, планов, задач и контрольных списков.
