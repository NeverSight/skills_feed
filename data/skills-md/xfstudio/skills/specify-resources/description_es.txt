Recursos compartidos (scripts, plantillas) para el flujo de trabajo de desarrollo basado en especificaciones. Esta habilidad es una dependencia de todas las habilidades spec-* y los usuarios NO deben invocarla directamente. Proporciona scripts bash para la gestión de sucursales de funciones, verificación de requisitos previos y plantillas para especificaciones, planes, tareas y listas de verificación.
