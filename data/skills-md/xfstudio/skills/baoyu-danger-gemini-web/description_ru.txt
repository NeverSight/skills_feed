Генерирует изображения и текст с помощью обратного проектирования Gemini Web API. Поддерживает генерацию текста, генерацию изображений из подсказок, эталонные изображения для визуального ввода и многоходовые разговоры. Используйте, когда для других навыков требуется серверная часть создания изображений, или когда пользователь запрашивает «создать изображение с помощью Gemini», «создать текст Gemini» или требуется создание искусственного интеллекта с возможностями машинного зрения.
