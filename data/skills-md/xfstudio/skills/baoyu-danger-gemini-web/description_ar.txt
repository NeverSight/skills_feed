يُنشئ الصور والنصوص عبر واجهة برمجة تطبيقات Gemini Web API ذات الهندسة العكسية. يدعم إنشاء النص، وإنشاء الصور من المطالبات، والصور المرجعية لإدخال الرؤية، والمحادثات متعددة المنعطفات. يُستخدم عندما تحتاج مهارات أخرى إلى إنشاء صور خلفية، أو عندما يطلب المستخدم "إنشاء صورة باستخدام Gemini"، أو "إنشاء نص Gemini"، أو يحتاج إلى إنشاء ذكاء اصطناعي قادر على الرؤية.
