Преобразует твиты и статьи X (Twitter) в уценку с помощью вступительной части YAML. Использует API с обратной разработкой, требующий согласия пользователя. Используйте, когда пользователь упоминает «X в уценку», «твит в уценку», «сохранить твит» или предоставляет URL-адреса x.com/twitter.com для конвертации.
