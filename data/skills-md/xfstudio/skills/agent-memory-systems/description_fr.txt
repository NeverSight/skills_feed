La mémoire est la pierre angulaire des agents intelligents. Sans cela, chaque interaction repart de zéro. Cette compétence couvre l'architecture de la mémoire des agents : court terme (fenêtre contextuelle), long terme (magasins de vecteurs), et les architectures cognitives qui les organisent.  Aperçu clé : la mémoire n'est pas seulement un stockage : c'est une récupération. Un million de données stockées ne signifient rien si vous ne parvenez pas à trouver la bonne. Les stratégies de regroupement, d'intégration et de récupération déterminent si votre agent se souvient ou oublie.  Le champ est fragmenté
