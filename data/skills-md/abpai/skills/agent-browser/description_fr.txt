CLI d’automatisation du navigateur pour les agents IA. À utiliser lorsque l'utilisateur a besoin d'interagir avec des sites Web, notamment pour naviguer dans des pages, remplir des formulaires, cliquer sur des boutons, prendre des captures d'écran, extraire des données, tester des applications Web ou automatiser n'importe quelle tâche du navigateur. Les déclencheurs incluent les demandes « ouvrir un site Web », « remplir un formulaire », « cliquer sur un bouton », « prendre une capture d'écran », « extraire les données d'une page », « tester cette application Web », « se connecter à un site », « automatiser les actions du navigateur » ou toute tâche nécessitant une interaction Web programmatique.
