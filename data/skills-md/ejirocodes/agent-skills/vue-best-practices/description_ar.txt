أفضل ممارسات Vue 3 وVue.js لأنماط TypeScript وvue-tsc وVolar والمكونات. يُستخدم عند كتابة مكونات Vue 3 أو مراجعتها أو إعادة هيكلتها باستخدام TypeScript، أو تكوين Volar/vueCompilerOptions، أو استخراج أنواع المكونات، أو العمل مع DefineModel/withDefaults، أو إعداد اختبارات متجر Pinia، أو تصحيح أخطاء أدوات Vue. المشغلات على مكونات Vue، واستخراج الدعائم، ومكونات الغلاف، والتحقق من نوع القالب، وstrictTemplates، وvueCompilerOptions، وVolar 3، ووحدات CSS، وسمات السقوط، وdefineModel، وwithDefaults، والمراقبة العميقة، والمعاملات المكتوبة بجهاز توجيه vue، وسخرية Pinia، وHMR SSR، ومجمع ModuleResolution، وuseTemplateRef، وonWatcherCleanup، وuseId، والمكونات العامة، والدعائم التفاعلية تدمير.
