أفضل ممارسات NestJS 11+ لتطبيقات Node.js الخاصة بالمؤسسات باستخدام TypeScript. يُستخدم عند كتابة أو مراجعة أو إعادة بناء وحدات التحكم أو الخدمات أو الوحدات النمطية أو واجهات برمجة التطبيقات NestJS. المشغلات على: وحدات NestJS، وحدات التحكم، المزودين، حقن التبعية، @Injectable، @Controller، @Module، البرامج الوسيطة، الحراس، المعترضات، الأنابيب، مرشحات الاستثناء، ValidationPipe، مدقق الفئة، محول الفئة، DTOs، مصادقة JWT، استراتيجيات Passport، @nestjs/passport، كيانات TypeORM، عميل Prisma، Drizzle ORM، نمط المستودع، دائري التبعيات، ForwardRef، @nestjs/swagger، ديكورات OpenAPI، محللات GraphQL، @nestjs/graphql، الخدمات الصغيرة، نقل TCP، نقل Redis، NATS، Kafka، NestJS 11 التغييرات العاجلة، ترحيل Express v5، أدوات تزيين مخصصة، ConfigService، @nestjs/config، فحوصات السلامة، أو أنماط اختبار NestJS.
