Создавайте полнофункциональные приложения с помощью Bun.js и Elysia.js. Используйте этот навык при работе со средой выполнения Bun, собственными API Bun (базы данных SQL, Redis, S3), рабочими пространствами/монорепозиториями Bun, веб-фреймворком Elysia.js или при переходе с Fastify на Elysia. Охватывает настройку, маршрутизацию, проверку, аутентификацию, WebSocket и сквозную безопасность типов с помощью Eden.
