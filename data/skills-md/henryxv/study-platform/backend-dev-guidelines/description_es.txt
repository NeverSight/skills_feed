Guía completa de desarrollo de backend para Next.js 14/tRPC/Express/TypeScript monorepo de Langfuse. Úselo al crear enrutadores tRPC, puntos finales API públicos, procesadores de colas BullMQ, servicios o al trabajar con procedimientos tRPC, rutas API Next.js, acceso a bases de datos Prisma, consultas analíticas de ClickHouse, colas Redis, instrumentación OpenTelemetry, validación Zod v4, configuración env.mjs, patrones de aislamiento de inquilinos o patrones asíncronos. Cubre arquitectura en capas (procedimientos tRPC → servicios, procesadores de colas → servicios), sistema de base de datos dual (PostgreSQL + ClickHouse), filtrado de projectId para aislamiento de múltiples inquilinos, manejo de errores traceException, patrones de observabilidad y estrategias de prueba (Jest para web, vitest para trabajadores).
