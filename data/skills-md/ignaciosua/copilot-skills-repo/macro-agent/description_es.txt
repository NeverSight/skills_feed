Control de macros de escritorio con reconocimiento de imágenes. Comandos: buscar, buscar, hacer clic, hacer clic, moverse a, escribir, presionar, tecla de acceso rápido, desplazarse, arrastrar, captura de pantalla, captura de región, crear secuencias/agregar/ejecutar/listar/eliminar. SIEMPRE utiliza coincidencia de plantillas (búsqueda de imágenes), NUNCA coordenadas fijas. Utilice 'captura de región' para capturar nuevos elementos.
