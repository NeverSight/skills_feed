Управление макросами рабочего стола с распознаванием изображений. Команды: найти, найти, нажать, щелкнуть, перейти, написать, нажать, горячая клавиша, прокрутка, перетаскивание, снимок экрана, захват региона, последовательное создание/добавление/запуск/список/удаление. ВСЕГДА используется сопоставление шаблонов (поиск изображений), НИКОГДА не фиксированные координаты. Используйте «захват региона» для захвата новых элементов.
