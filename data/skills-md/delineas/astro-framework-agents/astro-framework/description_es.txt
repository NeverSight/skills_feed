Guía completa de desarrollo del marco Astro para crear sitios web rápidos basados ​​en contenido utilizando la arquitectura de islas. Utilice esta habilidad al crear componentes Astro, implementar islas con hidratación selectiva, trabajar con colecciones de contenido, configurar adaptadores SSR, crear puntos finales API, implementar transiciones de vista o integrar marcos de interfaz de usuario (React, Vue, Svelte, Solid). Activadores en Astro, arquitectura de islas, colecciones de contenido, directivas de cliente, transiciones de vista, Astro SSR, renderizado híbrido, generación de sitios estáticos, astro.config.
