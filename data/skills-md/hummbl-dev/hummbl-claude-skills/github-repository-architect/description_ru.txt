Полная настройка репозитория GitHub со стандартами производственного уровня, включая файлы состояния сообщества, рабочие процессы CI/CD, шаблоны проблем, сайт документации, значки, CODEOWNERS и управление выпусками. Выполняет инициализацию, настройку, развертывание страниц GitHub и автоматизированные проверки качества для профессиональных проектов с открытым исходным кодом или корпоративных проектов.
