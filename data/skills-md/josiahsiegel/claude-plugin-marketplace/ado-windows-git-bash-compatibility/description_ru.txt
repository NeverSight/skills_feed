Руководство по совместимости Windows и Git Bash для Azure Pipelines. Охватывает проблемы преобразования путей, обнаружение оболочки в сценариях конвейера, обработку путей MINGW/MSYS, настройку агента Windows, шаблоны межплатформенных сценариев и устранение распространенных сбоев конвейера, характерных для Windows.
