تقنيات تحسين استعلام T-SQL لـ SQL Server وقاعدة بيانات Azure SQL. استخدم هذه المهارة عندما: (1) يحتاج المستخدم إلى تحسين الاستعلامات البطيئة، (2) يسأل المستخدم عن قابلية SARG أو البحث عن الفهرس، (3) يحتاج المستخدم إلى مساعدة في تلميحات الاستعلام، (4) يواجه المستخدم مشكلات في التعرف على المعلمات، (5) يحتاج المستخدم إلى فهم خطط التنفيذ، (6) يسأل المستخدم عن الإحصائيات وتقدير الأصل.
