Techniques d'optimisation des requêtes T-SQL pour SQL Server et Azure SQL Database. Utilisez cette compétence lorsque : (1) l'utilisateur a besoin d'optimiser les requêtes lentes, (2) l'utilisateur pose des questions sur la SARGabilité ou les recherches d'index, (3) l'utilisateur a besoin d'aide avec des conseils de requête, (4) l'utilisateur a des problèmes de détection de paramètres, (5) l'utilisateur doit comprendre les plans d'exécution, (6) l'utilisateur pose des questions sur les statistiques et l'estimation de la cardinalité.
