Técnicas de optimización de consultas T-SQL para SQL Server y Azure SQL Database. Utilice esta habilidad cuando: (1) el usuario necesite optimizar consultas lentas, (2) el usuario pregunte sobre SARGability o búsquedas de índice, (3) el usuario necesite ayuda con sugerencias de consulta, (4) el usuario tenga problemas de rastreo de parámetros, (5) el usuario necesite comprender los planes de ejecución, (6) el usuario pregunte sobre estadísticas y estimación de cardinalidad.
