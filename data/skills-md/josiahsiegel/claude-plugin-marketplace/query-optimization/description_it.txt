Tecniche di ottimizzazione delle query T-SQL per SQL Server e il database SQL di Azure. Utilizzare questa competenza quando: (1) l'utente deve ottimizzare le query lente, (2) l'utente chiede informazioni sulla SARGabilità o sulle ricerche di indici, (3) l'utente ha bisogno di aiuto con i suggerimenti per le query, (4) l'utente ha problemi di sniffing dei parametri, (5) l'utente deve comprendere i piani di esecuzione, (6) l'utente chiede informazioni su statistiche e stima della cardinalità.
