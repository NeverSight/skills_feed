Методы оптимизации запросов T-SQL для SQL Server и базы данных SQL Azure. Используйте этот навык, когда: (1) пользователю необходимо оптимизировать медленные запросы, (2) пользователь спрашивает о SARGability или поиске по индексу, (3) пользователю нужна помощь с подсказками по запросу, (4) у пользователя есть проблемы с анализом параметров, (5) пользователю необходимо понять планы выполнения, (6) пользователь спрашивает о статистике и оценке кратности.
