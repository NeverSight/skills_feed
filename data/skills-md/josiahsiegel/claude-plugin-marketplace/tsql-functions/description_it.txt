Riferimento completo alle funzioni T-SQL per SQL Server e il database SQL di Azure. Utilizzare questa competenza quando: (1) l'utente chiede informazioni su stringhe T-SQL, data, matematica o funzioni di conversione, (2) l'utente ha bisogno di aiuto con funzioni di finestra/classificazione, (3) l'utente lavora con JSON o XML in T-SQL, (4) l'utente chiede informazioni su funzioni di aggregazione o GRUPPO PER, (5) l'utente ha bisogno di funzioni di sistema o di metadati.
