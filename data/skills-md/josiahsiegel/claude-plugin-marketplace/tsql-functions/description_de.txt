Vollständige T-SQL-Funktionsreferenz für SQL Server und Azure SQL-Datenbank. Verwenden Sie diese Fähigkeit, wenn: (1) der Benutzer nach T-SQL-Zeichenfolge-, Datums-, Mathematik- oder Konvertierungsfunktionen fragt, (2) der Benutzer Hilfe bei Fenster-/Rankingfunktionen benötigt, (3) der Benutzer mit JSON oder XML in T-SQL arbeitet, (4) der Benutzer nach Aggregatfunktionen oder GROUP BY fragt, (5) der Benutzer System- oder Metadatenfunktionen benötigt.
