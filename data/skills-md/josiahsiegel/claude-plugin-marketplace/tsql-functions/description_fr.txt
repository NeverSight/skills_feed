Référence complète des fonctions T-SQL pour SQL Server et Azure SQL Database. Utilisez cette compétence lorsque : (1) l'utilisateur pose des questions sur les fonctions de chaîne, de date, mathématiques ou de conversion T-SQL, (2) l'utilisateur a besoin d'aide avec les fonctions de fenêtre/classement, (3) l'utilisateur travaille avec JSON ou XML dans T-SQL, (4) l'utilisateur pose des questions sur les fonctions d'agrégation ou GROUP BY, (5) l'utilisateur a besoin de fonctions système ou de métadonnées.
