مرجع وظيفة T-SQL الكامل لـ SQL Server وقاعدة بيانات Azure SQL. استخدم هذه المهارة عندما: (1) يسأل المستخدم عن سلسلة T-SQL أو التاريخ أو الرياضيات أو وظائف التحويل، (2) يحتاج المستخدم إلى مساعدة في وظائف النافذة/التصنيف، (3) يعمل المستخدم مع JSON أو XML في T-SQL، (4) يسأل المستخدم عن الوظائف المجمعة أو GROUP BY، (5) يحتاج المستخدم إلى وظائف النظام أو بيانات التعريف.
