Полное руководство по функциям T-SQL для SQL Server и базы данных SQL Azure. Используйте этот навык, когда: (1) Пользователь спрашивает о строке T-SQL, дате, математических функциях или функциях преобразования, (2) Пользователю нужна помощь с функциями окна/ранжирования, (3) Пользователь работает с JSON или XML в T-SQL, (4) Пользователь спрашивает об агрегатных функциях или GROUP BY, (5) Пользователю нужны системные функции или функции метаданных.
