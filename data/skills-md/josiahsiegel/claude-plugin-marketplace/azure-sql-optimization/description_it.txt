Ottimizzazione del database SQL di Azure e funzionalità specifiche della piattaforma. Utilizzare questa competenza quando: (1) l'utente chiede informazioni sull'ottimizzazione del database SQL di Azure, (2) l'utente necessita di indicazioni DTU/vCore, (3) l'utente desidera utilizzare l'ottimizzazione automatica, (4) l'utente richiede informazioni su iperscalabilità o serverless, (5) l'utente necessita del monitoraggio delle prestazioni SQL di Azure.
