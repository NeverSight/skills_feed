Оптимизация базы данных SQL Azure и функции, специфичные для платформы. Используйте этот навык, когда: (1) пользователь спрашивает об оптимизации базы данных SQL Azure, (2) пользователю требуется руководство по DTU/виртуальному ядру, (3) пользователь хочет использовать автоматическую настройку, (4) пользователь спрашивает о гипермасштабировании или бессерверной работе, (5) пользователю нужен мониторинг производительности SQL Azure.
