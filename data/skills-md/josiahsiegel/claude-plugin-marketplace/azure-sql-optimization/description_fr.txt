Optimisation d'Azure SQL Database et fonctionnalités spécifiques à la plateforme. Utilisez cette compétence lorsque : (1) l'utilisateur pose des questions sur l'optimisation de la base de données Azure SQL, (2) l'utilisateur a besoin de conseils DTU/vCore, (3) l'utilisateur souhaite utiliser le réglage automatique, (4) l'utilisateur pose des questions sur Hyperscale ou sans serveur, (5) l'utilisateur a besoin d'une surveillance des performances Azure SQL.
