تحسين قاعدة بيانات Azure SQL والميزات الخاصة بالنظام الأساسي. استخدم هذه المهارة عندما: (1) يسأل المستخدم عن تحسين قاعدة بيانات Azure SQL، (2) يحتاج المستخدم إلى إرشادات DTU/vCore، (3) يريد المستخدم استخدام الضبط التلقائي، (4) يسأل المستخدم عن Hyperscale أو بدون خادم، (5) يحتاج المستخدم إلى مراقبة أداء Azure SQL.
