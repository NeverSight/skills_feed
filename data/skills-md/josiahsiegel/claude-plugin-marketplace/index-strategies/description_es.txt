Estrategias de optimización y diseño de índices de SQL Server. Utilice esta habilidad cuando: (1) el usuario necesite ayuda para diseñar índices, (2) el usuario pregunte sobre índices agrupados y no agrupados, (3) el usuario quiera optimizar los índices del almacén de columnas, (4) el usuario necesite índices filtrados o que cubran, (5) el usuario pregunte sobre el mantenimiento y la fragmentación del índice.
