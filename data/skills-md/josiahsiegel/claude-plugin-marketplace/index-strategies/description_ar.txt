تصميم فهرس SQL Server واستراتيجيات التحسين. استخدم هذه المهارة عندما: (1) يحتاج المستخدم إلى مساعدة في تصميم الفهارس، (2) يسأل المستخدم عن الفهارس المجمعة مقابل الفهارس غير المجمعة، (3) يريد المستخدم تحسين فهارس مخزن الأعمدة، (4) يحتاج المستخدم إلى تصفية الفهارس أو تغطيتها، (5) يسأل المستخدم عن صيانة الفهرس وتجزئته.
