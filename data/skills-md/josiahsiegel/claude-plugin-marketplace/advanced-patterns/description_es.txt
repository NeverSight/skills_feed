Patrones y técnicas avanzadas de T-SQL para SQL Server. Utilice esta habilidad cuando: (1) el usuario necesite ayuda con CTE o consultas recursivas, (2) el usuario pregunte sobre el operador APPLY, (3) el usuario quiera ayuda con la cláusula MERGE o OUTPUT, (4) el usuario trabaje con tablas temporales, (5) el usuario necesite guía OLTP en memoria, (6) el usuario pregunte sobre agrupación avanzada (ROLLUP, CUBE, GROUPING SETS).
