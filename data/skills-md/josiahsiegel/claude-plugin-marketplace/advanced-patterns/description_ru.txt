Расширенные шаблоны и методы T-SQL для SQL Server. Используйте этот навык, когда: (1) пользователю нужна помощь с CTE или рекурсивными запросами, (2) пользователь спрашивает об операторе APPLY, (3) пользователю нужна помощь по предложениям MERGE или OUTPUT, (4) пользователь работает с временными таблицами, (5) пользователю нужны инструкции по OLTP в памяти, (6) пользователь спрашивает о расширенной группировке (ROLLUP, CUBE, GROUPING SETS).
