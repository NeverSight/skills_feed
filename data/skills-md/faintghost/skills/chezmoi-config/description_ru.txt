Комплексные навыки управления точечными файлами chezmoi для создания, изменения и поддержки шаблонов точечных файлов, файлов конфигурации, сценариев и внешних ресурсов. Используйте при работе с chezmoi для создания новых шаблонов точечных файлов (файлов .tmpl) с синтаксисом шаблонов Go, управления конфигурацией .chezmoi.toml.tmpl и .chezmoidata, настройки файлов .chezmoieexternal для внешних ресурсов, написания сценариев run_once/run_onchange/run_before/run_after, интеграции менеджеров паролей для секретов, управления кросс-платформенными конфигурациями и обработки условной логики на основе ОС/архитектура/имя хоста. Этот навык включает в себя подробную справочную документацию по всем функциям chezmoi, а также возможность ознакомиться с последней версией документации chezmoi через Context7.
