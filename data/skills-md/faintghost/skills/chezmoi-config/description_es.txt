Habilidad integral de administración de archivos de puntos chezmoi para crear, modificar y mantener plantillas de archivos de puntos, archivos de configuración, scripts y recursos externos. Úselo cuando trabaje con chezmoi para crear nuevas plantillas de archivos de puntos (archivos .tmpl) con sintaxis de plantilla Go, administrar la configuración .chezmoi.toml.tmpl y .chezmoidata, configurar archivos .chezmoie externos para recursos externos, escribir scripts run_once/run_onchange/run_before/run_after, integrar administradores de contraseñas para secretos, administrar configuraciones multiplataforma y manejar lógica condicional basada en SO/arquitectura/nombre de host. Esta habilidad incluye documentación de referencia completa para todas las funciones de chezmoi y puede consultar la documentación de chezmoi más reciente a través de Context7.
