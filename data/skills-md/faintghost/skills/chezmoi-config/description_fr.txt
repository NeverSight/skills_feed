Compétence complète de gestion de fichiers dot chezmoi pour créer, modifier et maintenir des modèles de fichiers dot, des fichiers de configuration, des scripts et des ressources externes. À utiliser lorsque vous travaillez avec chezmoi pour créer de nouveaux modèles de fichiers dotfile (fichiers .tmpl) avec la syntaxe de modèle Go, gérer la configuration .chezmoi.toml.tmpl et .chezmoidata, configurer les fichiers .chezmoiexternal pour les ressources externes, écrire des scripts run_once/run_onchange/run_before/run_after, intégrer des gestionnaires de mots de passe pour les secrets, gérer les configurations multiplateformes et gérer la logique conditionnelle basée sur le système d'exploitation/l'architecture/le nom d'hôte. Cette compétence comprend une documentation de référence complète pour toutes les fonctionnalités chezmoi et permet de consulter la dernière documentation chezmoi via Context7.
