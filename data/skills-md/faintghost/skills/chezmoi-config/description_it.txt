Competenze complete di gestione dei dotfile di Chezmoi per la creazione, la modifica e la manutenzione di modelli di dotfile, file di configurazione, script e risorse esterne. Utilizzare quando si lavora con chezmoi per creare nuovi modelli dotfile (file .tmpl) con la sintassi del modello Go, gestire la configurazione .chezmoi.toml.tmpl e .chezmoidata, configurare file .chezmoiexternal per risorse esterne, scrivere script run_once/run_onchange/run_before/run_after, integrare gestori di password per segreti, gestire configurazioni multipiattaforma e gestire la logica condizionale basata su Sistema operativo/architettura/nome host. Questa competenza include una documentazione di riferimento completa per tutte le funzionalità di chezmoi e può consultare la documentazione di chezmoi più recente tramite Context7.
