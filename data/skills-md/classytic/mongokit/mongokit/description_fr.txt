@classytic/mongokit — Modèle de référentiel MongoDB de qualité production pour Node.js/TypeScript.
À utiliser lors de la création de MongoDB CRUD, des API REST avec Mongoose 9, du modèle de référentiel,
pagination, mise en cache, suppression logicielle, piste d'audit, multi-tenant, génération d'ID personnalisé ou analyse de requêtes.
Déclencheurs : modèle mangouste, modèle de référentiel, mongokit, mongo crud, pagination,
suppression logicielle, piste d'audit, multi-tenant, identifiant personnalisé, analyseur de requêtes, plugin de cache, BaseController.
