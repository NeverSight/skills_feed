@classytic/mongokit — Node.js/TypeScript용 프로덕션 등급 MongoDB 리포지토리 패턴입니다.
MongoDB CRUD, Mongoose 9를 사용한 REST API, 리포지토리 패턴을 구축할 때 사용합니다.
페이지 매김, 캐싱, 일시 삭제, 감사 추적, 다중 테넌트, 사용자 정의 ID 생성 또는 쿼리 구문 분석.
트리거: 몽구스 모델, 저장소 패턴, mongokit, mongo crud, 페이지 매김,
일시 삭제, 감사 추적, 다중 테넌트, 사용자 정의 ID, 쿼리 파서, 캐시 플러그인, BaseController.
