@classytic/mongokit — шаблон репозитория MongoDB производственного уровня для Node.js/TypeScript.
Используйте при создании MongoDB CRUD, REST API с Mongoose 9, шаблона репозитория,
разбиение на страницы, кэширование, обратимое удаление, контрольный журнал, мультитенантность, создание собственных идентификаторов или анализ запросов.
Триггеры: модель mongoose, шаблон репозитория, mongokit, mongo crud, нумерация страниц,
обратимое удаление, контрольный журнал, мультитенант, собственный идентификатор, анализатор запросов, плагин кэша, BaseController.
