Fachkundige Anleitung zum Erstellen von MCP-Servern (Model Context Protocol) mit dem TypeScript SDK. Verwenden Sie diese Option, wenn Sie MCP-Server entwickeln, Tools/Ressourcen/Eingabeaufforderungen implementieren oder mit dem @modelcontextprotocol/sdk-Paket arbeiten. Behandelt Serverinitialisierung, Anforderungshandler, Zod-Schemas, Fehlerbehandlung und JSON-RPC-Muster.
