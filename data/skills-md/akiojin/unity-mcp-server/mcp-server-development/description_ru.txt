Экспертное руководство по созданию серверов MCP (Model Context Protocol) с использованием TypeScript SDK. Используйте при разработке серверов MCP, реализации инструментов/ресурсов/подсказок или работе с пакетом @modelcontextprotocol/sdk. Охватывает инициализацию сервера, обработчики запросов, схемы Zod, обработку ошибок и шаблоны JSON-RPC.
