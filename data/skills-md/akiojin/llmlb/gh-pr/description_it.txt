Crea o aggiorna le richieste pull di GitHub con la CLI gh, inclusa la decisione se creare un nuovo PR o eseguire il push solo in base allo stato di unione PR esistente. Da utilizzare quando l'utente chiede di aprire/creare/modificare un PR, generare un corpo/modello PR o dice "PRを出して/PR作成/gh pr". Impostazioni predefinite: base=sviluppo, head=ramo corrente (solo sullo stesso ramo; non creare/cambiare mai rami).
