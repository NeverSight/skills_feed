قم بإنشاء أو تحديث طلبات سحب GitHub باستخدام واجهة سطر الأوامر gh، بما في ذلك تحديد ما إذا كنت تريد إنشاء علاقات عامة جديدة أو الدفع فقط بناءً على حالة دمج العلاقات العامة الحالية. يُستخدم عندما يطلب المستخدم فتح/إنشاء/تحرير PR، أو إنشاء نص/قالب PR، أو يقول "PRを出して/PR作成/gh pr". الإعدادات الافتراضية: القاعدة = تطوير، الرأس = الفرع الحالي (نفس الفرع فقط؛ لا تقم أبدًا بإنشاء/تبديل الفروع).
