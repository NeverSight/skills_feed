Cree o actualice solicitudes de extracción de GitHub con la CLI de gh, incluida la decisión de crear un nuevo PR o solo enviarlo según el estado de fusión del PR existente. Úselo cuando el usuario solicite abrir/crear/editar un PR, generar un cuerpo/plantilla de PR, o diga 'PRを出して/PR作成/gh pr'. Valores predeterminados: base=desarrollar, head=rama actual (solo la misma rama; nunca crear/cambiar ramas).
