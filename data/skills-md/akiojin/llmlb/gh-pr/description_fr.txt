Créez ou mettez à jour des requêtes Pull GitHub avec la CLI gh, notamment en décidant de créer un nouveau PR ou uniquement de le pousser en fonction de l'état de fusion du PR existant. À utiliser lorsque l'utilisateur demande d'ouvrir/créer/modifier un PR, de générer un corps/un modèle de PR ou de dire « PRを出して/PR作成/gh pr ». Valeurs par défaut : base=develop, head=branche actuelle (même branche uniquement ; ne jamais créer/changer de branche).
