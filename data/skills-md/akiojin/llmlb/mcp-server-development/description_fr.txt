Conseils d'experts pour la création de serveurs MCP (Model Context Protocol) à l'aide du SDK TypeScript. À utiliser lors du développement de serveurs MCP, de la mise en œuvre d'outils/ressources/invites ou de l'utilisation du package @modelcontextprotocol/sdk. Couvre l'initialisation du serveur, les gestionnaires de requêtes, les schémas Zod, la gestion des erreurs et les modèles JSON-RPC.
