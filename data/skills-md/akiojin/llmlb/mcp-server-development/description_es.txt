Guía de expertos para crear servidores MCP (Protocolo de contexto modelo) utilizando el SDK de TypeScript. Úselo al desarrollar servidores MCP, implementar herramientas/recursos/indicaciones o trabajar con el paquete @modelcontextprotocol/sdk. Cubre la inicialización del servidor, controladores de solicitudes, esquemas Zod, manejo de errores y patrones JSON-RPC.
