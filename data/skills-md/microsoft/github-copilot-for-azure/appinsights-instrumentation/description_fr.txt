Conseils pour instrumenter des applications Web avec Azure Application Insights. Fournit des modèles de télémétrie, la configuration du SDK et des références de configuration.
UTILISER POUR : comment instrumenter une application, le SDK App Insights, les modèles de télémétrie, qu'est-ce qu'App Insights, les conseils d'Application Insights, les exemples d'instrumentation, les meilleures pratiques APM.
NE PAS UTILISER POUR : ajouter App Insights à mon application (utiliser Azure-prepare), ajouter de la télémétrie à mon projet (utiliser Azure-prepare), ajouter une surveillance (utiliser Azure-prepare). Cette compétence fournit des conseils : Azure-prepare orchestre les modifications des composants.
