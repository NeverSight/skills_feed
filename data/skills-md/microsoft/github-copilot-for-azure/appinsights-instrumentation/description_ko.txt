Azure Application Insights를 사용하여 웹앱 계측에 대한 지침입니다. 원격 분석 패턴, SDK 설정 및 구성 참조를 제공합니다.
사용 대상: 앱 계측 방법, App Insights SDK, 원격 분석 패턴, App Insights란 무엇입니까, Application Insights 지침, 계측 예제, APM 모범 사례.
사용하지 마십시오: 내 앱에 App Insights 추가(azure-prepare 사용), 내 프로젝트에 원격 분석 추가(azure-prepare 사용), 모니터링 추가(azure-prepare 사용). 이 기술은 지침을 제공합니다. 즉, azure-prepare가 구성 요소 변경 사항을 조정합니다.
