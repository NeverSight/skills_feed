إرشادات لاستخدام تطبيقات الويب باستخدام Azure Application Insights. يوفر أنماط القياس عن بعد وإعداد SDK ومراجع التكوين.
الاستخدام من أجل: كيفية استخدام التطبيق، وApp Insights SDK، وأنماط القياس عن بعد، وما هو App Insights، وإرشادات Application Insights، وأمثلة للأجهزة، وأفضل ممارسات APM.
لا تستخدمه من أجل: إضافة App Insights إلى تطبيقي (استخدم azure-prepare)، وإضافة القياس عن بعد إلى مشروعي (استخدم azure-prepare)، وإضافة المراقبة (استخدم azure-prepare). توفر هذه المهارة التوجيه — يقوم azure-prepare بتنسيق تغييرات المكونات.
