Guía para instrumentar aplicaciones web con Azure Application Insights. Proporciona patrones de telemetría, instalación de SDK y referencias de configuración.
UTILIZAR PARA: cómo instrumentar una aplicación, SDK de App Insights, patrones de telemetría, qué es App Insights, orientación de Application Insights, ejemplos de instrumentación, mejores prácticas de APM.
NO UTILIZAR PARA: agregar App Insights a mi aplicación (use azure-prepare), agregar telemetría a mi proyecto (use azure-prepare), agregar monitoreo (use azure-prepare). Esta habilidad proporciona orientación: Azure-Prepare organiza los cambios de componentes.
