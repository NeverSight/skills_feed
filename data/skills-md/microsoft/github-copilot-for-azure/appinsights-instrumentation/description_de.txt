VERWENDEN FÜR: Instrumentierung einer App, App Insights SDK, Telemetriemuster, Was ist App Insights, Anleitung zu Application Insights, Instrumentierungsbeispiele, Best Practices für APM.
NICHT VERWENDEN FÜR: Hinzufügen von App Insights zu meiner App (verwenden Sie Azure-Prepare), Hinzufügen von Telemetrie zu meinem Projekt (Verwenden Sie Azure-Prepare), Hinzufügen von Überwachung (Verwenden Sie Azure-Prepare).
