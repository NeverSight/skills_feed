Руководство по оснащению веб-приложений с помощью Azure Application Insights. Предоставляет шаблоны телеметрии, настройку SDK и ссылки на конфигурацию.
ИСПОЛЬЗОВАНИЕ: как инструментировать приложение, App Insights SDK, шаблоны телеметрии, что такое App Insights, руководство по Application Insights, примеры инструментирования, лучшие практики APM.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: добавления App Insights в мое приложение (используйте azure-prepare), добавления телеметрии в мой проект (используйте azure-prepare), добавления мониторинга (используйте azure-prepare). Этот навык обеспечивает руководство — Azure-Prepare организует изменения компонентов.
