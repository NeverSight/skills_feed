実際のコストを分析することで、Azure サブスクリプション全体のコスト削減を特定して定量化します。
使用率メトリクスを作成し、実用的な最適化の推奨事項を生成します。用途:
Azure コストの最適化、Azure 支出の削減、Azure コストの削減、Azure コストの分析、
コスト削減の発見、コスト最適化レポートの生成、孤立したリソースの発見、VM の適切なサイズ、
コスト分析、無駄の削減、Azure 支出分析、未使用リソースの検出、Redis の最適化
費用がかかります。次の用途には使用しないでください: リソースのデプロイ (azure-deploy を使用)、一般的な Azure 診断
(azure-diagnostics を使用)、セキュリティの問題 (azure-security を使用)
