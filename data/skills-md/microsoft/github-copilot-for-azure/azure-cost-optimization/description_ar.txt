تحديد وقياس وفورات التكلفة عبر اشتراكات Azure من خلال تحليل التكاليف الفعلية،
مقاييس الاستخدام، وإنشاء توصيات تحسين قابلة للتنفيذ. استخدام ل:
تحسين تكاليف Azure، وتقليل إنفاق Azure، وتقليل نفقات Azure، وتحليل تكاليف Azure،
العثور على وفورات في التكاليف، وإنشاء تقرير تحسين التكلفة، والعثور على الموارد المعزولة، وحجم الأجهزة الافتراضية،
تحليل التكلفة، وتقليل الهدر، وتحليل إنفاق Azure، والعثور على الموارد غير المستخدمة، وتحسين Redis
التكاليف. لا تستخدم من أجل: نشر الموارد (استخدم نشر Azure)، وتشخيصات Azure العامة
(استخدم azure-diagnostics)، ومشكلات الأمان (استخدم azure-security)
