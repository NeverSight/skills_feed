Crea nuove istanze del database di Azure per il server flessibile PostgreSQL e configura l'autenticazione senza password con l'ID Microsoft Entra. Configura l'accesso per sviluppatori, identità gestite per le app, autorizzazioni basate su gruppi ed esegui la migrazione dall'autenticazione basata su password a quella con ID Entra. Le frasi trigger includono "passwordless for postgres", "entra id postgres", "autenticazione di Azure ad Postgres", "identità gestita postgres", "migra postgres a passwordless".
