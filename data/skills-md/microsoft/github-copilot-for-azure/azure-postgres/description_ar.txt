أنشئ قاعدة بيانات Azure جديدة لمثيلات PostgreSQL Elastic Server وقم بتكوين المصادقة بدون كلمة مرور باستخدام معرف Microsoft Entra. قم بإعداد وصول المطورين، والهويات المُدارة للتطبيقات، والأذونات المستندة إلى المجموعة، والانتقال من مصادقة Entra ID المستندة إلى كلمة المرور. تتضمن عبارات التشغيل "بدون كلمة مرور لـ postgres"، و"entra id postgres"، و"مصادقة azure ad postgres"، و"الهوية المُدارة لـ postgres"، و"ترحيل postgres إلى بدون كلمة مرور".
