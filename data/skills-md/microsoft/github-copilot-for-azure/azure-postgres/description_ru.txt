Создайте новую базу данных Azure для экземпляров гибкого сервера PostgreSQL и настройте проверку подлинности без пароля с помощью Microsoft Entra ID. Настройте доступ разработчика, управляемые удостоверения для приложений, разрешения на основе групп и перейдите от аутентификации на основе пароля к аутентификации Entra ID. Триггерные фразы включают «без пароля для postgres», «entra id postgres», «аутентификация Azure Ad Postgres», «управляемая идентификация Postgres», «переход Postgres в режим без пароля».
