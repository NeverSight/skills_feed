Erstellen Sie neue Azure Database for PostgreSQL Flexible Server-Instanzen und konfigurieren Sie die kennwortlose Authentifizierung mit Microsoft Entra ID. Richten Sie Entwicklerzugriff, verwaltete Identitäten für Apps und gruppenbasierte Berechtigungen ein und migrieren Sie von der passwortbasierten zur Entra-ID-Authentifizierung. Zu den Triggerphrasen gehören „Passwortlos für Postgres“, „Entra-ID-Postgres“, „Azure Ad Postgres-Authentifizierung“, „Postgres-verwaltete Identität“ und „Postgres auf Passwortlos migrieren“.
