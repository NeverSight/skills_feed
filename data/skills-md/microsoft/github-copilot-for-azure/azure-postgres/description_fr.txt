Créez de nouvelles instances Azure Database pour PostgreSQL Flexible Server et configurez l’authentification sans mot de passe avec Microsoft Entra ID. Configurez l'accès des développeurs, les identités gérées pour les applications, les autorisations basées sur les groupes et migrez de l'authentification par mot de passe vers l'authentification Entra ID. Les expressions de déclenchement incluent « sans mot de passe pour postgres », « entra id postgres », « authentification Azure ad postgres », « identité gérée postgres », « migrer postgres vers sans mot de passe ».
