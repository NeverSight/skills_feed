Guía el registro de la aplicación Microsoft Entra ID, la autenticación OAuth 2.0 y la integración MSAL.
UTILIZAR PARA: crear un registro de aplicación, registrar una aplicación de Azure AD, configurar OAuth, configurar la autenticación, agregar permisos de API, generar una entidad de servicio, ejemplo de MSAL, autenticación de la aplicación de consola, configuración de Entra ID, autenticación de Azure AD.
NO UTILIZAR PARA: Azure RBAC o asignaciones de roles (use azure-rbac), secretos de Key Vault (use azure-keyvault-expiration-audit), seguridad de recursos de Azure (use azure-security).
