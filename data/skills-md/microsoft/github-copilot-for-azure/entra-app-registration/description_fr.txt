Guide l’enregistrement de l’application Microsoft Entra ID, l’authentification OAuth 2.0 et l’intégration MSAL.
UTILISER POUR : créer l'enregistrement d'une application, enregistrer l'application Azure AD, configurer OAuth, configurer l'authentification, ajouter des autorisations API, générer un principal de service, un exemple MSAL, l'authentification de l'application console, la configuration d'Entra ID, l'authentification Azure AD.
NE PAS UTILISER POUR : Azure RBAC ou attributions de rôles (utilisez azure-rbac), les secrets Key Vault (utilisez azure-keyvault-expiration-audit), la sécurité des ressources Azure (utilisez azure-security).
