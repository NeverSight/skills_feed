يرشد تسجيل تطبيق Microsoft Entra ID ومصادقة OAuth 2.0 وتكامل MSAL.
الاستخدام من أجل: إنشاء تسجيل التطبيق، وتسجيل تطبيق Azure AD، وتكوين OAuth، وإعداد المصادقة، وإضافة أذونات API، وإنشاء الخدمة الأساسية، ومثال MSAL، ومصادقة تطبيق وحدة التحكم، وإعداد Entra ID، ومصادقة Azure AD.
لا تستخدم من أجل: Azure RBAC أو تعيينات الأدوار (استخدم azure-rbac)، وأسرار Key Vault (استخدم azure-keyvault-expiration-audit)، وأمان موارد Azure (استخدم azure-security).
