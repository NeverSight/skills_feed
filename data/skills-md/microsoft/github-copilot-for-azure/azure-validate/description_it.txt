Checkpoint di convalida pre-distribuzione. Esegui controlli approfonditi per assicurarti che la tua applicazione sia pronta per la distribuzione di Azure. Convalida configurazione, infrastruttura, autorizzazioni e prerequisiti.
UTILIZZARE PER: convalidare l'app, verificare la disponibilità della distribuzione, eseguire controlli preliminari, verificare la configurazione, verificare se è pronta per la distribuzione, convalidare azure.yaml, convalidare Bicep, testare prima della distribuzione, risolvere gli errori di distribuzione.
NON UTILIZZARE PER: creazione o creazione di app (usare azure-prepare), esecuzione di distribuzioni (usare azure-deploy).
