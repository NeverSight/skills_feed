Контрольная точка проверки перед развертыванием. Выполните тщательную проверку, чтобы убедиться, что ваше приложение готово к развертыванию в Azure. Проверяет конфигурацию, инфраструктуру, разрешения и предварительные условия.
ИСПОЛЬЗОВАНИЕ ДЛЯ: проверки моего приложения, проверки готовности к развертыванию, запуска предварительных проверок, проверки конфигурации, проверки готовности к развертыванию, проверки azure.yaml, проверки Bicep, тестирования перед развертыванием, устранения ошибок развертывания.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: создания или разработки приложений (используйте azure-prepare), выполнения развертываний (используйте azure-deploy).
