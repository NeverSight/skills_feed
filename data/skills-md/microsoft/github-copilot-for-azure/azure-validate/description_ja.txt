導入前の検証チェックポイント。詳細なチェックを実行して、アプリケーションが Azure にデプロイできる準備ができていることを確認します。構成、インフラストラクチャ、権限、前提条件を検証します。
用途: アプリの検証、デプロイメントの準備状況の確認、プリフライト チェックの実行、構成の検証、デプロイの準備ができているかどうかの確認、azure.yaml の検証、Bicep の検証、デプロイ前のテスト、デプロイメント エラーのトラブルシューティング。
アプリの作成または構築 (azure-prepare を使用)、デプロイメントの実行 (azure-deploy を使用) には使用しないでください。
