列出、尋找和顯示 Azure 資源。回答「列出我的虛擬機器」、「顯示我的儲存帳戶」、「列出網站」、
「尋找容器應用程式」、「我有哪些資源」以及針對任何 Azure 資源類型的類似查詢。
用途：列出資源、列出虛擬機器、列出 VM、列出儲存帳戶、列出網站、列出 Web 應用程式、
列出容器應用程式、顯示資源、尋找資源、我有哪些資源、列出資源群組中的資源、
列出訂閱中的資源、按標籤尋找資源、尋找孤立資源、資源庫存、
按類型統計資源、跨訂閱資源查詢、Azure Resource Graph、資源發現、
列出容器註冊表、列出 SQL 伺服器、列出 Key Vault、顯示資源群組、列出應用程式服務、
跨訂閱尋找資源、尋找未連線的磁碟、標籤分析。
請勿用於：部署資源（使用 azure-deploy）、建立或修改資源、
成本優化（使用 azure-cost-optimization）、編寫應用程式程式碼、非 Azure 雲端。
