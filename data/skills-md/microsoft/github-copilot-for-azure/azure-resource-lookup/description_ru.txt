Перечисляйте, находите и показывайте ресурсы Azure. Ответы «перечислить мои виртуальные машины», «показать мои учетные записи хранения», «перечислить веб-сайты»,
«найти приложения-контейнеры», «какие ресурсы у меня есть» и подобные запросы для любого типа ресурсов Azure.
ИСПОЛЬЗОВАНИЕ ДЛЯ: списка ресурсов, списка виртуальных машин, списка виртуальных машин, списка учетных записей хранения, списка веб-сайтов, списка веб-приложений,
список приложений-контейнеров, показать ресурсы, найти ресурсы, какие ресурсы у меня есть, перечислить ресурсы в группе ресурсов,
список ресурсов в подписке, поиск ресурсов по тегу, поиск потерянных ресурсов, инвентаризация ресурсов,
подсчет ресурсов по типу, запрос ресурсов с перекрестной подпиской, Azure Resource Graph, обнаружение ресурсов,
список реестров контейнеров, список серверов SQL, список хранилищ ключей, отображение групп ресурсов, список служб приложений,
поиск ресурсов в подписках, поиск неприсоединенных дисков, анализ тегов.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: развертывания ресурсов (используйте azure-deploy), создания или изменения ресурсов,
оптимизация затрат (используйте azure-cost-optimization), написание кода приложения, облака, отличные от Azure.
