列出、查找和显示 Azure 资源。回答“列出我的虚拟机”、“显示我的存储帐户”、“列出网站”、
“查找容器应用程序”、“我有哪些资源”以及针对任何 Azure 资源类型的类似查询。
用途：列出资源、列出虚拟机、列出 VM、列出存储帐户、列出网站、列出 Web 应用程序、
列出容器应用程序、显示资源、查找资源、我有哪些资源、列出资源组中的资源、
列出订阅中的资源、按标签查找资源、查找孤立资源、资源库存、
按类型统计资源、跨订阅资源查询、Azure Resource Graph、资源发现、
列出容器注册表、列出 SQL 服务器、列出 Key Vault、显示资源组、列出应用程序服务、
跨订阅查找资源、查找未连接的磁盘、标签分析。
请勿用于：部署资源（使用 azure-deploy）、创建或修改资源、
成本优化（使用 azure-cost-optimization）、编写应用程序代码、非 Azure 云。
