Répertoriez, recherchez et affichez les ressources Azure. Réponses "lister mes VM", "afficher mes comptes de stockage", "lister les sites Web",
« Trouver des applications de conteneur », « de quelles ressources ai-je » et requêtes similaires pour tout type de ressource Azure.
UTILISER POUR : répertorier les ressources, répertorier les machines virtuelles, répertorier les machines virtuelles, répertorier les comptes de stockage, répertorier les sites Web, répertorier les applications Web,
répertorier les applications conteneurs, afficher les ressources, rechercher des ressources, de quelles ressources ai-je, répertorier les ressources dans le groupe de ressources,
répertorier les ressources dans l'abonnement, rechercher des ressources par balise, rechercher des ressources orphelines, un inventaire des ressources,
compter les ressources par type, requête de ressources par abonnement croisé, Azure Resource Graph, découverte de ressources,
répertorier les registres de conteneurs, répertorier les serveurs SQL, répertorier les Key Vaults, afficher les groupes de ressources, répertorier les services d'application,
recherchez des ressources parmi les abonnements, recherchez des disques non connectés, analysez les balises.
NE PAS UTILISER POUR : déployer des ressources (utiliser azure-deploy), créer ou modifier des ressources,
optimisation des coûts (utiliser Azure-cost-optimisation), écriture de code d'application, cloud non Azure.
