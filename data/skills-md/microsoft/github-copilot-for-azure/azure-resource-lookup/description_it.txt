Elenca, trova e mostra le risorse di Azure. Risposte "elenca le mie VM", "mostra i miei account di archiviazione", "elenca siti web",
"trova app contenitore", "quali risorse ho" e query simili per qualsiasi tipo di risorsa di Azure.
UTILIZZARE PER: elencare risorse, elencare macchine virtuali, elencare VM, elencare account di archiviazione, elencare siti Web, elencare app Web,
elenca le app contenitore, mostra risorse, trova risorse, quali risorse ho, elenca le risorse nel gruppo di risorse,
elencare le risorse nell'abbonamento, trovare risorse per tag, trovare risorse orfane, inventario delle risorse,
conteggio delle risorse per tipo, query di risorse tra sottoscrizioni, Azure Resource Graph, individuazione delle risorse,
elencare registri contenitori, elencare server SQL, elencare Key Vault, mostrare gruppi di risorse, elencare servizi app,
trovare risorse tra abbonamenti, trovare dischi non collegati, analisi dei tag.
NON UTILIZZARE PER: distribuire risorse (usare azure-deploy), creare o modificare risorse,
ottimizzazione dei costi (usare l'ottimizzazione dei costi di Azure), scrittura del codice dell'applicazione, cloud non Azure.
