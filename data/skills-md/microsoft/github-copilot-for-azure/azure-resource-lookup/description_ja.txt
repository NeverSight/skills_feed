Azure リソースを一覧表示、検索、表示します。回答「VM をリストする」、「ストレージ アカウントを表示する」、「Web サイトをリストする」、
「コンテナー アプリの検索」、「所有しているリソースは何ですか」、および任意の Azure リソース タイプに対する同様のクエリ。
用途: リソースの一覧表示、仮想マシンの一覧表示、VM の一覧表示、ストレージ アカウントの一覧表示、Web サイトの一覧表示、Web アプリの一覧表示、
コンテナー アプリの一覧表示、リソースの表示、リソースの検索、所有するリソースの内容、リソース グループ内のリソースの一覧表示、
サブスクリプション内のリソースの一覧表示、タグによるリソースの検索、孤立したリソースの検索、リソース インベントリ、
タイプ別のリソースのカウント、クロスサブスクリプション リソース クエリ、Azure Resource Graph、リソースの検出、
コンテナー レジストリの一覧表示、SQL サーバーの一覧表示、Key Vault の一覧表示、リソース グループの表示、アプリ サービスの一覧表示、
サブスクリプション全体のリソースの検索、接続されていないディスクの検索、タグ分析。
次の用途には使用しないでください: リソースのデプロイ (azure-deploy を使用)、リソースの作成または変更、
コストの最適化 (azure-cost-optimization を使用)、アプリケーション コードの作成、Azure 以外のクラウド。
