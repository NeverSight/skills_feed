Анализируйте группы ресурсов Azure и создавайте подробные схемы архитектуры Mermaid, показывающие связи между отдельными ресурсами.
ИСПОЛЬЗОВАНИЕ ДЛЯ: создания диаграммы архитектуры, визуализации ресурсов Azure, отображения взаимосвязей ресурсов, создания диаграммы Mermaid, анализа группы ресурсов, диаграммы моих ресурсов, визуализации архитектуры, топологии ресурсов, отображения инфраструктуры Azure.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: создания/изменения ресурсов (используйте azure-deploy), сканирования безопасности (используйте azure-security), устранения неполадок производительности (используйте azure-diagnostics), генерации кода (используйте соответствующие навыки обслуживания).
