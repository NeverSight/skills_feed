Azure リソース グループを分析し、個々のリソース間の関係を示す詳細な Mermaid アーキテクチャ図を生成します。
用途: アーキテクチャ図の作成、Azure リソースの視覚化、リソース関係の表示、マーメイド図の生成、リソース グループの分析、リソースの図表、アーキテクチャの視覚化、リソース トポロジ、Azure インフラストラクチャのマップ
次の用途には使用しないでください: リソースの作成/変更 (azure-deploy を使用)、セキュリティ スキャン (azure-security を使用)、パフォーマンスのトラブルシューティング (azure-diagnostics を使用)、コード生成 (関連するサービス スキルを使用)
