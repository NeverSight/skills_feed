Analysez les groupes de ressources Azure et générez des diagrammes d’architecture Mermaid détaillés montrant les relations entre les ressources individuelles.
UTILISER POUR : créer un diagramme d'architecture, visualiser les ressources Azure, afficher les relations entre les ressources, générer un diagramme Mermaid, analyser un groupe de ressources, schématiser mes ressources, visualisation de l'architecture, topologie des ressources, cartographier l'infrastructure Azure
NE PAS UTILISER POUR : la création/modification de ressources (utiliser Azure-Deploy), l'analyse de sécurité (utiliser Azure-Security), le dépannage des performances (utiliser Azure-Diagnostics), la génération de code (utiliser les compétences de service appropriées)
