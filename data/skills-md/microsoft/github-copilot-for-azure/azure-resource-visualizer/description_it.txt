Analizza i gruppi di risorse di Azure e genera diagrammi dettagliati dell'architettura della sirena che mostrano le relazioni tra le singole risorse.
UTILIZZO PER: creare un diagramma dell'architettura, visualizzare le risorse di Azure, mostrare le relazioni tra le risorse, generare un diagramma della sirena, analizzare un gruppo di risorse, creare un diagramma delle mie risorse, visualizzazione dell'architettura, topologia delle risorse, mappare l'infrastruttura di Azure
NON UTILIZZARE PER: creazione/modifica di risorse (usare azure-deploy), scansione di sicurezza (usare azure-security), risoluzione dei problemi relativi alle prestazioni (usare azure-diagnostics), generazione di codice (usare competenze di servizio pertinenti)
