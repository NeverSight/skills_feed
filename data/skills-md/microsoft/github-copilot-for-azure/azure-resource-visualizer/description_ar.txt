قم بتحليل مجموعات موارد Azure وإنشاء مخططات تفصيلية لبنية حورية البحر توضح العلاقات بين الموارد الفردية.
الاستخدام من أجل: إنشاء رسم تخطيطي للهندسة، وتصور موارد Azure، وإظهار علاقات الموارد، وإنشاء مخطط حورية البحر، وتحليل مجموعة الموارد، ورسم تخطيطي لمواردي، وتصور البنية، وطوبولوجيا الموارد، وخريطة البنية التحتية لـ Azure
لا تستخدم من أجل: إنشاء/تعديل الموارد (استخدم azure-deploy)، والفحص الأمني (استخدم azure-security)، واستكشاف أخطاء الأداء وإصلاحها (استخدم azure-diagnostics)، وإنشاء التعليمات البرمجية (استخدم مهارات الخدمة ذات الصلة)
