이 기술을 사용하여 Microsoft Foundry(Azure AI Foundry)와 작업할 수 있습니다. 카탈로그에서 AI 모델 배포, 지식 인덱스를 사용하여 RAG 애플리케이션 구축, AI 에이전트 생성 및 평가, RBAC 권한 및 역할 할당 관리, 할당량 및 용량 관리, Foundry 리소스 생성.
사용 대상: Microsoft Foundry, AI Foundry, 모델 배포, 모델 카탈로그, RAG, 지식 인덱스, 에이전트 생성, 에이전트 평가, 에이전트 모니터링, Foundry 프로젝트 생성, 새 Foundry 프로젝트, Foundry 설정, Foundry 온보딩, Foundry 인프라 프로비저닝, Foundry 리소스 생성, AI 서비스 생성, 다중 서비스 리소스, AIServices 종류, 리소스 공급자 등록, Cognitive Services 활성화, AI 서비스 계정 설정, Foundry용 리소스 그룹 생성, RBAC, 역할 할당, 관리 ID, 서비스 주체, 권한, 할당량, 용량, TPM, 배포 실패, QuotaExceeded.
사용하지 마십시오: Azure Functions(azure-functions 사용), App Service(azure-create-app 사용), 일반 Azure 리소스 생성(azure-create-app 사용).
