このスキルを使用して Microsoft Foundry (Azure AI Foundry) を操作します。カタログからの AI モデルのデプロイ、ナレッジ インデックスを使用した RAG アプリケーションの構築、AI エージェントの作成と評価、RBAC のアクセス許可とロールの割り当ての管理、クォータと容量の管理、Foundry リソースの作成を行います。
用途: Microsoft Foundry、AI Foundry、モデルの展開、モデル カタログ、RAG、ナレッジ インデックス、エージェントの作成、エージェントの評価、エージェントの監視、Foundry プロジェクトの作成、新しい Foundry プロジェクト、Foundry のセットアップ、Foundry へのオンボード、Foundry インフラストラクチャのプロビジョニング、Foundry リソースの作成、AI サービスの作成、マルチサービス リソース、AIService の種類、リソース プロバイダーの登録、Cognitive Services の有効化、AI サービス アカウントのセットアップ、Foundry のリソース グループの作成、RBAC、ロールの割り当て、マネージド ID、サービス プリンシパル、権限、クォータ、容量、TPM、展開の失敗、QuotaExceeded。
次の用途には使用しないでください: Azure Functions (azure-functions を使用)、App Service (azure-create-app を使用)、汎用 Azure リソースの作成 (azure-create-app を使用)。
