Utilizza questa competenza per lavorare con Microsoft Foundry (Azure AI Foundry): distribuire modelli IA dal catalogo, creare applicazioni RAG con indici di conoscenza, creare e valutare agenti IA, gestire autorizzazioni RBAC e assegnazioni di ruoli, gestire quote e capacità, creare risorse Foundry.
UTILIZZARE PER: Microsoft Foundry, AI Foundry, distribuire modello, catalogo modelli, RAG, indice di conoscenza, creare agente, valutare agente, monitoraggio agente, creare progetto Foundry, nuovo progetto Foundry, impostare Foundry, eseguire l'onboarding in Foundry, eseguire il provisioning dell'infrastruttura Foundry, creare risorsa Foundry, creare servizi AI, risorsa multiservizio, tipo AIServices, registrare il provider di risorse, abilitare Servizi cognitivi, impostare un account AI Services, creare un gruppo di risorse per Foundry, RBAC, assegnazione di ruoli, identità gestita, entità servizio, autorizzazioni, quota, capacità, TPM, errore di distribuzione, quota superata.
NON UTILIZZARE PER: Funzioni di Azure (usare azure-functions), Servizio app (usare azure-create-app), creazione di risorse generiche di Azure (usare azure-create-app).
