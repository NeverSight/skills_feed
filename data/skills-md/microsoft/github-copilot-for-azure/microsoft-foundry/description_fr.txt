Utilisez cette compétence pour travailler avec Microsoft Foundry (Azure AI Foundry) : déployer des modèles d'IA à partir du catalogue, créer des applications RAG avec des index de connaissances, créer et évaluer des agents d'IA, gérer les autorisations RBAC et les attributions de rôles, gérer les quotas et la capacité, créer des ressources Foundry.
UTILISER POUR : Microsoft Foundry, AI Foundry, déployer un modèle, un catalogue de modèles, RAG, un index de connaissances, créer un agent, évaluer un agent, surveiller un agent, créer un projet Foundry, un nouveau projet Foundry, configurer Foundry, intégrer à Foundry, fournir l'infrastructure Foundry, créer une ressource Foundry, créer des services AI, une ressource multiservice, un type AIServices, enregistrer un fournisseur de ressources, activer Cognitive Services, configurer un compte AI Services, créer un groupe de ressources pour Foundry, RBAC, attribution de rôle, identité gérée, principal de service, autorisations, quota, capacité, TPM, échec de déploiement, QuotaExceeded.
NE PAS UTILISER POUR : Azure Functions (utilisez azure-functions), App Service (utilisez azure-create-app), création de ressources Azure génériques (utilisez azure-create-app).
