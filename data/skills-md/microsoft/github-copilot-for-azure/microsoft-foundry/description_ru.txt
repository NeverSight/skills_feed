Используйте этот навык для работы с Microsoft Foundry (Azure AI Foundry): развертывайте модели ИИ из каталога, создавайте приложения RAG с индексами знаний, создавайте и оценивайте агенты ИИ, управляйте разрешениями RBAC и назначениями ролей, управляйте квотами и емкостью, создавайте ресурсы Foundry.
ИСПОЛЬЗОВАНИЕ ДЛЯ: Microsoft Foundry, AI Foundry, развертывание модели, каталог моделей, RAG, индекс знаний, создание агента, оценка агента, мониторинг агента, создание проекта Foundry, новый проект Foundry, настройка Foundry, подключение к Foundry, предоставление инфраструктуры Foundry, создание ресурса Foundry, создание AI Services, мультисервисный ресурс, тип AIServices, регистрация поставщика ресурсов, включение Cognitive Services, настройка учетной записи AI Services, создание группы ресурсов для Foundry, RBAC, назначение ролей, управляемое удостоверение, субъект службы, разрешения, квота, емкость, TPM, сбой развертывания, QuotaExceeded.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: функций Azure (используйте azure-functions), службы приложений (используйте azure-create-app), создания общих ресурсов Azure (используйте azure-create-app).
