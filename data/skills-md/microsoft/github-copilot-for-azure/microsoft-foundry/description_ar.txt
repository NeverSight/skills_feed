استخدم هذه المهارة للعمل مع Microsoft Foundry (Azure AI Foundry): نشر نماذج الذكاء الاصطناعي من الكتالوج، وإنشاء تطبيقات RAG باستخدام فهارس المعرفة، وإنشاء وكلاء الذكاء الاصطناعي وتقييمهم، وإدارة أذونات RBAC وتعيينات الأدوار، وإدارة الحصص والقدرات، وإنشاء موارد Foundry.
الاستخدام لـ: Microsoft Foundry، AI Foundry، نشر النموذج، كتالوج النموذج، RAG، فهرس المعرفة، إنشاء وكيل، تقييم الوكيل، مراقبة الوكيل، إنشاء مشروع مسبك، مشروع مسبك جديد، إعداد المسبك، على متن المسبك، توفير البنية التحتية للمسبك، إنشاء مورد المسبك، إنشاء خدمات الذكاء الاصطناعي، مورد متعدد الخدمات، نوع AIServices، تسجيل موفر الموارد، تمكين الخدمات المعرفية، إعداد حساب خدمات الذكاء الاصطناعي، إنشاء مجموعة موارد للمسبك، RBAC، تعيين الدور، الهوية المُدارة، مدير الخدمة، الأذونات، الحصة، السعة، TPM، فشل النشر، تجاوز الحصة.
لا تستخدمه من أجل: وظائف Azure (استخدم وظائف azure)، وخدمة التطبيقات (استخدم azure-create-app)، وإنشاء موارد Azure العامة (استخدم azure-create-app).
