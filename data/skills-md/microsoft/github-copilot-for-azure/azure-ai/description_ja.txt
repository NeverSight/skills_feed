Azure AI に使用: 検索、音声、OpenAI、Document Intelligence。検索、ベクトル/ハイブリッド検索、音声合成、テキスト読み上げ、文字起こし、OCR に役立ちます。用途: AI 検索、クエリ検索、ベクトル検索、ハイブリッド検索、セマンティック検索、音声からテキストへの変換、テキストから音声への変換、文字起こし、OCR、テキストから音声への変換。次の用途には使用しないでください: Function apps/Functions (azure-functions を使用)、データベース (azure-postgres/azure-kusto)、一般的な Azure リソース。
