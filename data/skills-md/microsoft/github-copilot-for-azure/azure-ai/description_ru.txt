ИСПОЛЬЗОВАНИЕ ДЛЯ: Поиск AI, поиск по запросу, векторный поиск, гибридный поиск, семантический поиск, преобразование речи в текст, преобразование текста в речь, расшифровка, распознавание текста, преобразование текста в речь.
