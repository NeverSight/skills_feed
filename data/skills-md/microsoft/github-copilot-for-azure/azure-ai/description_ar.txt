الاستخدام لـ Azure AI: البحث والكلام وOpenAI وذكاء المستندات. يساعد في البحث، والبحث المتجه/الهجين، وتحويل الكلام إلى نص، وتحويل النص إلى كلام، والنسخ، والتعرف الضوئي على الحروف. الاستخدام من أجل: بحث الذكاء الاصطناعي، بحث الاستعلام، بحث المتجهات، البحث المختلط، البحث الدلالي، تحويل الكلام إلى نص، تحويل النص إلى كلام، النسخ، التعرف الضوئي على الحروف، تحويل النص إلى كلام. لا تستخدمه من أجل: تطبيقات/وظائف الوظائف (استخدم وظائف azure)، وقواعد البيانات (azure-postgres/azure-kusto)، وموارد Azure العامة.
