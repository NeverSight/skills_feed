Помогает пользователям найти подходящую роль Azure RBAC для удостоверения с наименьшими привилегиями доступа, а затем генерировать команды CLI и код Bicep для ее назначения.
ИСПОЛЬЗОВАНИЕ ДЛЯ: «какую роль мне следует назначить», «роль с наименьшими привилегиями», «роль RBAC для», «роль для чтения больших двоичных объектов», «роль для управляемого удостоверения», «определение пользовательской роли», «назначить роль удостоверению».
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: создания или настройки управляемых удостоверений или общего усиления безопасности Azure; они выходят за рамки этого навыка выбора роли.
