UTILIZZARE PER: "quale ruolo devo assegnare", "ruolo con privilegi minimi", "ruolo RBAC per", "ruolo per leggere i BLOB", "ruolo per identità gestita", "definizione di ruolo personalizzato", "assegna ruolo all'identità".
NON UTILIZZARE PER: creazione o configurazione di identità gestite o rafforzamento generale della sicurezza di Azure; questi non rientrano nell'ambito di questa abilità di selezione del ruolo.
