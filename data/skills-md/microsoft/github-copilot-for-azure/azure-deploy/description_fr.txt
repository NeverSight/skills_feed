Exécutez le déploiement sur Azure. Dernière étape après préparation et validation. Exécute les commandes azd up, azd déployer ou de provisionnement d’infrastructure.
UTILISER POUR : exécuter azd, exécuter azd déployer, exécuter le déploiement, provisionner l'infrastructure, pousser en production, mettre en ligne, expédier, déployer une application Web, déployer une application conteneur, déployer un site statique, déployer Azure Functions, déployer des biceps, appliquer terraform.
NE PAS UTILISER POUR : créer ou créer des applications (utilisez azure-prepare), valider avant le déploiement (utiliser azure-validate).
