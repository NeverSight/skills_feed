تنفيذ النشر إلى Azure. الخطوة النهائية بعد الإعداد والتحقق من الصحة. يقوم بتشغيل أوامر azd up، أو نشر azd، أو توفير البنية التحتية.
الاستخدام من أجل: تشغيل azd up، وتشغيل azd Publish، وتنفيذ النشر، وتوفير البنية التحتية، والدفع إلى الإنتاج، والبدء المباشر، وشحنه، ونشر تطبيق الويب، ونشر تطبيق الحاوية، ونشر موقع ثابت، ونشر وظائف Azure، ونشر العضلة ذات الرأسين، وتطبيق terraform.
لا تستخدم من أجل: إنشاء التطبيقات أو إنشائها (استخدم azure-prepare)، والتحقق من الصحة قبل النشر (استخدم azure-validate).
