Azureへのデプロイを実行します。準備と検証後の最終ステップ。 azd up、azddeploy、またはインフラストラクチャ プロビジョニング コマンドを実行します。
用途: azd up の実行、azddeploy の実行、デプロイの実行、インフラストラクチャのプロビジョニング、運用環境へのプッシュ、ライブへの移行、出荷、Web アプリのデプロイ、コンテナー アプリのデプロイ、静的サイトのデプロイ、Azure Functions のデプロイ、二頭筋デプロイ、Terraform 適用。
次の目的には使用しないでください: アプリの作成または構築 (azure-prepare を使用)、デプロイ前の検証 (azure-validate を使用)。
