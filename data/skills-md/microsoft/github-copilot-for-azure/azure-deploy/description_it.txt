Eseguire la distribuzione in Azure. Passaggio finale dopo la preparazione e la convalida. Esegue i comandi azd up, azd deploy o il provisioning dell'infrastruttura.
UTILIZZARE PER: eseguire azd up, eseguire azd deploy, eseguire la distribuzione, eseguire il provisioning dell'infrastruttura, eseguire il push in produzione, passare allo stato Live, spedirlo, distribuire l'app Web, distribuire l'app contenitore, distribuire il sito statico, distribuire Funzioni di Azure, distribuire bicep, applicare terraform.
NON UTILIZZARE PER: creazione o creazione di app (usare azure-prepare), convalida prima della distribuzione (usare azure-validate).
