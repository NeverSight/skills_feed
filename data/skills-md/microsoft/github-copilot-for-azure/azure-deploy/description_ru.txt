Выполните развертывание в Azure. Последний шаг после подготовки и проверки. Запускает команды azd up, azd Deploy или подготовки инфраструктуры.
ИСПОЛЬЗОВАНИЕ ДЛЯ: запустить azd up, запустить azd Deploy, выполнить развертывание, подготовить инфраструктуру, отправить в рабочую среду, запустить в эксплуатацию, отправить, развернуть веб-приложение, развернуть контейнерное приложение, развернуть статический сайт, развернуть функции Azure, развернуть бицепс, применить terraform.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: создания или сборки приложений (используйте azure-prepare), проверки перед развертыванием (используйте azure-validate).
