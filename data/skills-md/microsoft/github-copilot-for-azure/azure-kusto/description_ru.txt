Запрашивайте и анализируйте данные в Azure Data Explorer (Kusto/ADX) с помощью KQL для анализа журналов, телеметрии и анализа временных рядов.
ИСПОЛЬЗОВАНИЕ ДЛЯ: запросов KQL, запросов к базе данных Kusto, Azure Data Explorer, кластеров ADX, анализа журналов, данных временных рядов, телеметрии IoT, обнаружения аномалий.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: баз данных SQL (используйте azure-postgres), запросов NoSQL (используйте azure-storage), Elasticsearch, инструментов аналитики AWS.
