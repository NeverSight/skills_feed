Esegui query e analizza i dati in Azure Esplora dati (Kusto/ADX) usando KQL per l'analisi dei log, la telemetria e l'analisi delle serie temporali.
UTILIZZARE PER: query KQL, query di database Kusto, Esplora dati di Azure, cluster ADX, analisi dei log, dati di serie temporali, telemetria IoT, rilevamento di anomalie
NON UTILIZZARE PER: database SQL (usare azure-postgres), query NoSQL (usare azure-storage), Elasticsearch, strumenti di analisi AWS
