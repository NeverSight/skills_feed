Consulte y analice datos en Azure Data Explorer (Kusto/ADX) usando KQL para análisis de registros, telemetría y análisis de series temporales.
USO PARA: consultas KQL, consultas de bases de datos Kusto, Azure Data Explorer, clústeres ADX, análisis de registros, datos de series temporales, telemetría de IoT, detección de anomalías
NO UTILIZAR PARA: bases de datos SQL (use azure-postgres), consultas NoSQL (use azure-storage), Elasticsearch, herramientas de análisis de AWS
