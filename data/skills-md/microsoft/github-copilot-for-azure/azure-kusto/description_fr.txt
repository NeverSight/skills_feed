Interrogez et analysez des données dans Azure Data Explorer (Kusto/ADX) à l'aide de KQL pour l'analyse des journaux, la télémétrie et l'analyse des séries chronologiques.
UTILISER POUR : requêtes KQL, requêtes de base de données Kusto, Azure Data Explorer, clusters ADX, analyses de journaux, données de séries chronologiques, télémétrie IoT, détection d'anomalies
NE PAS UTILISER POUR : bases de données SQL (utilisez Azure-postgres), requêtes NoSQL (utilisez Azure-Storage), Elasticsearch, outils d'analyse AWS
