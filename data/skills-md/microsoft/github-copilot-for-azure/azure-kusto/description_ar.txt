الاستعلام عن البيانات وتحليلها في Azure Data Explorer (Kusto/ADX) باستخدام KQL لتحليلات السجل والقياس عن بعد وتحليل السلاسل الزمنية.
الاستخدام من أجل: استعلامات KQL، واستعلامات قاعدة بيانات Kusto، وAzure Data Explorer، ومجموعات ADX، وتحليلات السجل، وبيانات السلاسل الزمنية، والقياس عن بعد لإنترنت الأشياء، والكشف عن الحالات الشاذة
لا تستخدمه من أجل: قواعد بيانات SQL (استخدم azure-postgres)، واستعلامات NoSQL (استخدم azure-storage)، وElasticsearch، وأدوات تحليلات AWS
