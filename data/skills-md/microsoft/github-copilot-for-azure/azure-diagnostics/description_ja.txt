Azure での運用上の問題のデバッグとトラブルシューティングを行います。コンテナー アプリの診断、KQL によるログ分析、ヘルス チェック、イメージ プル、コールド スタート、ヘルス プローブに関する一般的な問題の解決について説明します。
用途: 運用上の問題のデバッグ、コンテナ アプリのトラブルシューティング、KQL によるログの分析、イメージ プルの失敗の修正、コールド スタートの問題の解決、正常性プローブの失敗の調査、リソースの正常性の確認、アプリケーション ログの表示、エラーの根本原因の検索
次の用途には使用しないでください: アプリケーションのデプロイ (azure-deploy を使用)、新しいリソースの作成 (azure-prepare を使用)、監視の設定 (azure-observability を使用)、コストの最適化 (azure-cost-optimization を使用)
