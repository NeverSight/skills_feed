Déboguer et résoudre les problèmes de production sur Azure. Couvre les diagnostics des applications conteneurisées, l'analyse des journaux avec KQL, les vérifications de l'état et la résolution des problèmes courants pour les extractions d'images, les démarrages à froid et les sondes d'état.
UTILISER POUR : déboguer les problèmes de production, dépanner les applications de conteneur, analyser les journaux avec KQL, corriger les échecs d'extraction d'image, résoudre les problèmes de démarrage à froid, enquêter sur les échecs de la sonde d'intégrité, vérifier l'état des ressources, afficher les journaux d'application, rechercher la cause première des erreurs.
NE PAS UTILISER POUR : déployer des applications (utiliser azure-deploy), créer de nouvelles ressources (utiliser azure-prepare), configurer la surveillance (utiliser azure-observability), optimiser les coûts (utiliser azure-cost-optimization)
