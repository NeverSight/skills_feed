Отладка и устранение производственных проблем в Azure. Охватывает диагностику приложений-контейнеров, анализ журналов с помощью KQL, проверки работоспособности и решение распространенных проблем, связанных с получением изображений, холодным запуском и проверками работоспособности.
ИСПОЛЬЗОВАНИЕ ДЛЯ: отладки производственных проблем, устранения неполадок в приложениях-контейнерах, анализа журналов с помощью KQL, исправления сбоев извлечения образа, решения проблем с холодным запуском, исследования сбоев проверки работоспособности, проверки работоспособности ресурсов, просмотра журналов приложений, поиска основной причины ошибок.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: развертывания приложений (используйте azure-deploy), создания новых ресурсов (используйте azure-prepare), настройки мониторинга (используйте azure-observability), оптимизации затрат (используйте azure-cost-optimization).
