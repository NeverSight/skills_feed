Depurar y solucionar problemas de producción en Azure. Cubre diagnósticos de Container Apps, análisis de registros con KQL, comprobaciones de estado y resolución de problemas comunes para extracción de imágenes, arranques en frío y sondeos de estado.
UTILIZAR PARA: depurar problemas de producción, solucionar problemas de aplicaciones de contenedor, analizar registros con KQL, corregir fallas de extracción de imágenes, resolver problemas de arranque en frío, investigar fallas de sondas de estado, verificar el estado de los recursos, ver registros de aplicaciones, encontrar la causa raíz de los errores
NO UTILIZAR PARA: implementar aplicaciones (use azure-deploy), crear nuevos recursos (use azure-prepare), configurar el monitoreo (use azure-observability), optimización de costos (use azure-cost-optimization)
