Debuggen und beheben Sie Produktionsprobleme in Azure. Behandelt Container-Apps-Diagnose, Protokollanalyse mit KQL, Integritätsprüfungen und allgemeine Problemlösung für Image-Pulls, Kaltstarts und Integritätsprüfungen.
VERWENDUNG FÜR: Debuggen von Produktionsproblemen, Beheben von Fehlern bei Container-Apps, Analysieren von Protokollen mit KQL, Beheben von Image-Pull-Fehlern, Beheben von Kaltstartproblemen, Untersuchen von Integritätsprüfungsfehlern, Überprüfen des Ressourcenzustands, Anzeigen von Anwendungsprotokollen, Ermitteln der Grundursache von Fehlern
NICHT VERWENDEN FÜR: Anwendungen bereitstellen (Azure-Deploy verwenden), neue Ressourcen erstellen (Azure-Prepare verwenden), Überwachung einrichten (Azure-Observability verwenden), Kostenoptimierung (Azure-Kostenoptimierung verwenden)
