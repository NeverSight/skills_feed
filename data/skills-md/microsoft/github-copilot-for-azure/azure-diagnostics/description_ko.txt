Azure에서 프로덕션 문제를 디버그하고 해결합니다. 컨테이너 앱 진단, KQL을 사용한 로그 분석, 상태 확인, 이미지 가져오기, 콜드 스타트 ​​및 상태 프로브에 대한 일반적인 문제 해결을 다룹니다.
사용 대상: 프로덕션 문제 디버그, 컨테이너 앱 문제 해결, KQL을 사용하여 로그 분석, 이미지 가져오기 오류 수정, 콜드 스타트 문제 해결, 상태 프로브 오류 조사, 리소스 상태 확인, 애플리케이션 로그 보기, 오류의 근본 원인 찾기
사용하지 마십시오: 애플리케이션 배포(azure-deploy 사용), 새 리소스 생성(azure-prepare 사용), 모니터링 설정(azure-observability 사용), 비용 최적화(azure-cost-optimization 사용)
