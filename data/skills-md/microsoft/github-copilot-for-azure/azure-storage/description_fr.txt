Services de stockage Azure, notamment le stockage Blob, les partages de fichiers, le stockage de files d'attente, le stockage de tables et Data Lake. Fournit des capacités de stockage d'objets, de partages de fichiers SMB, de messagerie asynchrone, de valeur-clé NoSQL et d'analyse Big Data. Inclut les niveaux d'accès (hot, cool, archive) et la gestion du cycle de vie.
