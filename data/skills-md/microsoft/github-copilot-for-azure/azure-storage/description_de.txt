Azure Storage Services einschließlich Blob Storage, Dateifreigaben, Warteschlangenspeicher, Tabellenspeicher und Data Lake. Bietet Objektspeicher, SMB-Dateifreigaben, asynchrones Messaging, NoSQL-Schlüsselwert- und Big-Data-Analysefunktionen. Beinhaltet Zugriffsebenen (Hot, Cool, Archiv) und Lebenszyklusverwaltung.
