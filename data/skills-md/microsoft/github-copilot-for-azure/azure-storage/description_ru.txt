Службы хранения Azure, включая хранилище BLOB-объектов, общие файловые ресурсы, хранилище очередей, хранилище таблиц и озеро данных. Предоставляет объектное хранилище, общие файловые ресурсы SMB, асинхронный обмен сообщениями, ключ-значение NoSQL и возможности анализа больших данных. Включает уровни доступа (горячий, прохладный, архивный) и управление жизненным циклом.
