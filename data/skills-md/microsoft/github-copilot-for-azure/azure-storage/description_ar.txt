خدمات تخزين Azure بما في ذلك تخزين البيانات الثنائية كبيرة الحجم ومشاركات الملفات وتخزين قائمة الانتظار وتخزين الجداول وبحيرة البيانات. يوفر تخزين الكائنات ومشاركة ملفات SMB والمراسلة غير المتزامنة وقيمة مفتاح NoSQL وإمكانات تحليل البيانات الضخمة. يتضمن مستويات الوصول (السريعة، الرائعة، الأرشيفية) وإدارة دورة الحياة.
