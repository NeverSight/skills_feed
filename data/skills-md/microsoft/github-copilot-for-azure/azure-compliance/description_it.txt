Funzionalità complete di conformità e controllo della sicurezza di Azure, inclusa la valutazione delle best practice,
Monitoraggio della scadenza di Key Vault e convalida della configurazione delle risorse.
UTILIZZARE PER: scansione di conformità, controllo di sicurezza, azqr, procedure consigliate di Azure, controllo della scadenza di Key Vault,
valutazione della conformità, revisione delle risorse, convalida della configurazione, certificati scaduti, segreti in scadenza,
risorse orfane, conformità alle policy, valutazione del livello di sicurezza.
NON UTILIZZARE PER: distribuzione di risorse (usare azure-deploy), sola analisi dei costi (usare azure-cost-optimization),
rafforzamento della sicurezza attivo (usare azure-security-hardening), query generali di Azure Advisor (usare azure-observability).
