ベスト プラクティスの評価を含む包括的な Azure コンプライアンスおよびセキュリティ監査機能、
Key Vault の有効期限の監視とリソース構成の検証。
用途: コンプライアンス スキャン、セキュリティ監査、azqr、Azure のベスト プラクティス、Key Vault の有効期限チェック、
コンプライアンス評価、リソースレビュー、構成検証、期限切れの証明書、期限切れのシークレット、
孤立したリソース、ポリシーの遵守、セキュリティ体制の評価。
次の目的には使用しないでください: リソースのデプロイ (azure-deploy を使用)、コスト分析のみ (azure-cost-optimization を使用)、
アクティブなセキュリティ強化 (azure-security-hardening を使用)、一般的な Azure Advisor クエリ (azure-observability を使用)。
