Комплексные возможности аудита соответствия требованиям и безопасности Azure, включая оценку лучших практик,
Мониторинг срока действия Key Vault и проверка конфигурации ресурсов.
ИСПОЛЬЗОВАНИЕ ДЛЯ: сканирования соответствия, аудита безопасности, azqr, рекомендаций Azure, проверки срока действия Key Vault,
оценка соответствия, проверка ресурсов, проверка конфигурации, сертификаты с истекшим сроком действия, секреты с истекающим сроком действия,
потерянные ресурсы, соответствие политике, оценка состояния безопасности.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: развертывания ресурсов (используйте azure-deploy), только анализа затрат (используйте azure-cost-optimization),
усиление активной безопасности (используйте azure-security-hardening), общие запросы Azure Advisor (используйте azure-observability).
