الامتثال الشامل لـ Azure وقدرات التدقيق الأمني بما في ذلك تقييم أفضل الممارسات،
مراقبة انتهاء صلاحية Key Vault والتحقق من صحة تكوين الموارد.
الاستخدام من أجل: فحص الامتثال، والتدقيق الأمني، وazqr، وأفضل ممارسات Azure، والتحقق من انتهاء صلاحية Key Vault،
تقييم الامتثال، ومراجعة الموارد، والتحقق من صحة التكوين، والشهادات منتهية الصلاحية، والأسرار منتهية الصلاحية،
الموارد اليتيمة، والامتثال للسياسة، وتقييم الوضع الأمني.
لا تستخدم من أجل: نشر الموارد (استخدم نشر azure)، وتحليل التكلفة فقط (استخدم تحسين تكلفة azure)،
تقوية الأمان النشطة (استخدم تقوية أمان azure)، واستعلامات Azure Advisor العامة (استخدم إمكانية ملاحظة azure).
