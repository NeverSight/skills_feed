모범 사례 평가를 포함한 포괄적인 Azure 규정 준수 및 보안 감사 기능
Key Vault 만료 모니터링 및 리소스 구성 유효성 검사.
사용 대상: 규정 준수 검사, 보안 감사, azqr, Azure 모범 사례, Key Vault 만료 확인,
규정 준수 평가, 리소스 검토, 구성 검증, 만료된 인증서, 만료되는 비밀,
고아 리소스, 정책 준수, 보안 상태 평가.
사용하지 마십시오: 리소스 배포(azure-deploy 사용), 비용 분석 단독(azure-cost-optimization 사용),
활성 보안 강화(azure-security-hardening 사용), 일반 Azure Advisor 쿼리(azure-observability 사용).
