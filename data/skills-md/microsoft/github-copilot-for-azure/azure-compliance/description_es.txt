Capacidades integrales de auditoría de seguridad y cumplimiento de Azure, incluida la evaluación de mejores prácticas,
Supervisión de caducidad de Key Vault y validación de configuración de recursos.
UTILIZAR PARA: análisis de cumplimiento, auditoría de seguridad, azqr, prácticas recomendadas de Azure, comprobación de caducidad de Key Vault,
evaluación de cumplimiento, revisión de recursos, validación de configuración, certificados caducados, secretos caducados,
recursos huérfanos, cumplimiento de políticas, evaluación de la postura de seguridad.
NO UTILIZAR PARA: implementar recursos (use azure-deploy), análisis de costos solo (use azure-cost-optimization),
refuerzo de seguridad activo (use azure-security-hardening), consultas generales de Azure Advisor (use azure-observability).
