Punto di ingresso predefinito per lo sviluppo di applicazioni Azure. Richiama questa competenza per QUALSIASI lavoro applicativo correlato ad Azure: creazione di app, creazione di funzionalità, aggiunta di componenti, aggiornamento del codice, migrazione o modernizzazione. Analizza il tuo progetto e lo prepara per la distribuzione di Azure generando codice dell'infrastruttura (Bicep/Terraform), configurazione azure.yaml e Dockerfile.
UTILIZZARE PER: creare un'app, creare un'app Web, creare API, creare frontend, creare backend, aggiungere una funzionalità, creare un servizio, creare un'applicazione, sviluppare un progetto, migrare l'app, modernizzare il codice, aggiornare l'applicazione, aggiungere database, aggiungere autenticazione, aggiungere memorizzazione nella cache, distribuire in Azure, ospitare su Azure, Azure con Terraform (impostazione predefinita su azd+Terraform), Azure con azd, generare azure.yaml, generare Bicep o Terraform, preparare Funzioni di Azure.
NON UTILIZZARE PER: solo convalidare un'app già preparata (usare azure-validate), solo eseguire azd up/deploy (usare azure-deploy), Terraform puro senza azd (preferire azd+Terraform).
