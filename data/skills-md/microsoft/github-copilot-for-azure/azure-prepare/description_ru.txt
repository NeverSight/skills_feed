Точка входа по умолчанию для разработки приложений Azure. Используйте этот навык для ЛЮБОЙ работы с приложениями, связанной с Azure: создания приложений, создания функций, добавления компонентов, обновления кода, миграции или модернизации. Анализирует ваш проект и подготавливает его к развертыванию в Azure, создавая код инфраструктуры (Bicep/Terraform), конфигурацию azure.yaml и файлы Dockerfiles.
ИСПОЛЬЗОВАНИЕ ДЛЯ: создания приложения, создания веб-приложения, создания API, создания внешнего интерфейса, создания серверной части, добавления функции, создания службы, создания приложения, разработки проекта, переноса моего приложения, модернизации моего кода, обновления моего приложения, добавления базы данных, добавления аутентификации, добавления кэширования, развертывания в Azure, размещения в Azure, Azure с Terraform (по умолчанию azd+Terraform), Azure с azd, создания azure.yaml, создания Bicep или Terraform, подготовки Azure Функции.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: только проверки уже подготовленного приложения (используйте azure-validate), только запуска azd up/deploy (используйте azure-deploy), чистого Terraform без azd (предпочитайте azd+Terraform).
