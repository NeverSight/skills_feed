نقطة الدخول الافتراضية لتطوير تطبيقات Azure. قم باستدعاء هذه المهارة لأي عمل تطبيقي متعلق بـ Azure: إنشاء التطبيقات، أو بناء الميزات، أو إضافة المكونات، أو تحديث التعليمات البرمجية، أو الترحيل، أو التحديث. يقوم بتحليل مشروعك وإعداده لنشر Azure عن طريق إنشاء كود البنية التحتية (Bicep/Terraform)، وتكوين azure.yaml، وDockerfiles.
الاستخدام من أجل: إنشاء تطبيق، إنشاء تطبيق ويب، إنشاء واجهة برمجة التطبيقات، إنشاء واجهة أمامية، إنشاء واجهة خلفية، إضافة ميزة، بناء خدمة، إنشاء تطبيق، تطوير مشروع، ترحيل تطبيقي، تحديث الكود الخاص بي، تحديث تطبيقي، إضافة قاعدة بيانات، إضافة مصادقة، إضافة تخزين مؤقت، نشر إلى Azure، استضافة على Azure، Azure مع Terraform (افتراضيات azd+Terraform)، Azure مع azd، إنشاء azure.yaml، إنشاء Bicep أو Terraform، قم بإعداد وظائف Azure.
لا تستخدمه من أجل: التحقق فقط من صحة تطبيق تم إعداده بالفعل (استخدم azure-validate)، وتشغيل azd up/deploy فقط (استخدم azure-deploy)، وTerraform خالص بدون azd (يفضل azd+Terraform).
