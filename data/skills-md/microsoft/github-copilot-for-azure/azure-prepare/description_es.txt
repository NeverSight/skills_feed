Punto de entrada predeterminado para el desarrollo de aplicaciones de Azure. Invoque esta habilidad para CUALQUIER trabajo de aplicación relacionado con Azure: crear aplicaciones, crear características, agregar componentes, actualizar código, migrar o modernizar. Analiza su proyecto y lo prepara para la implementación de Azure generando código de infraestructura (Bicep/Terraform), configuración azure.yaml y Dockerfiles.
USO PARA: crear una aplicación, crear una aplicación web, crear API, crear frontend, crear backend, agregar una característica, crear un servicio, crear una aplicación, desarrollar un proyecto, migrar mi aplicación, modernizar mi código, actualizar mi aplicación, agregar una base de datos, agregar autenticación, agregar almacenamiento en caché, implementar en Azure, alojar en Azure, Azure con Terraform (el valor predeterminado es azd+Terraform), Azure con azd, generar azure.yaml, generar Bicep o Terraform, preparar funciones de Azure.
NO UTILIZAR PARA: solo validar una aplicación ya preparada (use azure-validate), solo ejecutar azd up/deploy (use azure-deploy), Terraform puro sin azd (prefiera azd+Terraform).
