Standardeinstiegspunkt für die Azure-Anwendungsentwicklung. Nutzen Sie diese Fähigkeit für JEDE Anwendungsarbeit im Zusammenhang mit Azure: Erstellen von Apps, Erstellen von Features, Hinzufügen von Komponenten, Aktualisieren von Code, Migration oder Modernisierung. Analysiert Ihr Projekt und bereitet es für die Azure-Bereitstellung vor, indem es Infrastrukturcode (Bicep/Terraform), azure.yaml-Konfiguration und Dockerfiles generiert.
VERWENDUNG FÜR: App erstellen, Web-App erstellen, API erstellen, Frontend erstellen, Backend erstellen, Funktion hinzufügen, Dienst erstellen, Anwendung erstellen, Projekt entwickeln, App migrieren, Code modernisieren, Anwendung aktualisieren, Datenbank hinzufügen, Authentifizierung hinzufügen, Caching hinzufügen, in Azure bereitstellen, auf Azure hosten, Azure mit Terraform (standardmäßig azd+Terraform), Azure mit azd, azure.yaml generieren, Bicep oder Terraform generieren, Azure vorbereiten Funktionen.
NICHT VERWENDEN FÜR: Nur die Validierung einer bereits vorbereiteten App (verwenden Sie azure-validate), nur das Ausführen von azd up/deploy (verwenden Sie azure-deploy), reines Terraform ohne azd (bevorzugen Sie azd+Terraform).
