AI 모델, MCP 서버, 에이전트를 보호, 관찰, 제어하기 위해 APIM(Azure API Management)을 AI 게이트웨이로 구성합니다. 속도 제한, 의미론적 캐싱, 콘텐츠 안전, 로드 밸런싱에 도움이 됩니다.
사용 대상: AI 게이트웨이, APIM, 게이트웨이 설정, 게이트웨이 구성, 게이트웨이 추가, 모델 게이트웨이, MCP 서버, 속도 제한, 토큰 제한, 의미 체계 캐시, 콘텐츠 안전, 로드 밸런싱, OpenAPI 가져오기, API를 MCP로 변환.
사용하지 마십시오: 모델 배포(microsoft-foundry 사용), Azure Functions(azure-functions 사용), 데이터베이스(azure-postgres 사용).
