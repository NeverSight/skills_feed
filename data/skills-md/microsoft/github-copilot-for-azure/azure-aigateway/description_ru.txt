Настройте управление API Azure (APIM) в качестве шлюза AI для защиты, наблюдения и управления моделями AI, серверами MCP и агентами. Помогает с ограничением скорости, семантическим кэшированием, безопасностью контента, балансировкой нагрузки.
ИСПОЛЬЗОВАНИЕ ДЛЯ: AI Gateway, APIM, настройки шлюза, настройки шлюза, добавления шлюза, модели шлюза, сервера MCP, ограничения скорости, ограничения токена, семантического кэша, безопасности контента, балансировки нагрузки, импорта OpenAPI, преобразования API в MCP.
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: моделей развертывания (используйте microsoft-foundry), функций Azure (используйте функции azure), баз данных (используйте azure-postgres).
