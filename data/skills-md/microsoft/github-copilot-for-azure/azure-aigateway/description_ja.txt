Azure API Management (APIM) を AI ゲートウェイとして構成し、AI モデル、MCP サーバー、エージェントを保護、監視、制御します。レート制限、セマンティック キャッシュ、コンテンツの安全性、負荷分散に役立ちます。
用途: AI ゲートウェイ、APIM、ゲートウェイのセットアップ、ゲートウェイの構成、ゲートウェイの追加、モデル ゲートウェイ、MCP サーバー、レート制限、トークン制限、セマンティック キャッシュ、コンテンツの安全性、ロード バランス、OpenAPI インポート、API から MCP への変換。
以下には使用しないでください: モデルのデプロイ (microsoft-foundry を使用)、Azure Functions (azure-functions を使用)、データベース (azure-postgres を使用)。
