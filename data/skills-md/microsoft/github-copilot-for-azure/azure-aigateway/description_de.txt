Konfigurieren Sie Azure API Management (APIM) als AI Gateway, um AI-Modelle, MCP-Server und Agenten zu sichern, zu beobachten und zu steuern. Hilft bei Ratenbegrenzung, semantischem Caching, Inhaltssicherheit und Lastausgleich.
VERWENDUNG FÜR: AI Gateway, APIM, Gateway einrichten, Gateway konfigurieren, Gateway hinzufügen, Modell-Gateway, MCP-Server, Ratenlimit, Token-Limit, semantischer Cache, Inhaltssicherheit, Lastausgleich, OpenAPI-Import, API in MCP konvertieren.
NICHT VERWENDEN FÜR: Modelle bereitstellen (Microsoft-Foundry verwenden), Azure-Funktionen (Azure-Funktionen verwenden), Datenbanken (Azure-Postgres verwenden).
