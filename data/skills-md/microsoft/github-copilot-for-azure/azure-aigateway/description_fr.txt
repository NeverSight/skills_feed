Configurez Azure API Management (APIM) en tant que AI Gateway pour sécuriser, observer et contrôler les modèles d'IA, les serveurs MCP et les agents. Aide à la limitation du débit, à la mise en cache sémantique, à la sécurité du contenu et à l'équilibrage de charge.
UTILISER POUR : AI Gateway, APIM, configuration de la passerelle, configuration de la passerelle, ajout d'une passerelle, modèle de passerelle, serveur MCP, limite de débit, limite de jetons, cache sémantique, sécurité du contenu, équilibre de charge, importation OpenAPI, conversion de l'API en MCP.
NE PAS UTILISER POUR : déployer des modèles (utiliser Microsoft-foundry), Azure Functions (utiliser Azure-Functions), des bases de données (utiliser Azure-postgres).
