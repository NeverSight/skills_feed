Configura Gestione API di Azure (APIM) come gateway AI per proteggere, osservare e controllare modelli AI, server MCP e agenti. Aiuta con limitazione della velocità, memorizzazione nella cache semantica, sicurezza dei contenuti, bilanciamento del carico.
UTILIZZARE PER: AI Gateway, APIM, impostazione gateway, configurazione gateway, aggiunta gateway, gateway modello, server MCP, limite di velocità, limite token, cache semantica, sicurezza dei contenuti, bilanciamento del carico, importazione OpenAPI, conversione API in MCP.
NON UTILIZZARE PER: distribuire modelli (usare microsoft-foundry), Funzioni di Azure (usare funzioni di azure), database (usare azure-postgres).
