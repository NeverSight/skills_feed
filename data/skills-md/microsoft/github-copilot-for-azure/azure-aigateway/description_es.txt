Configure Azure API Management (APIM) como AI Gateway para proteger, observar y controlar modelos de IA, servidores MCP y agentes. Ayuda con la limitación de velocidad, el almacenamiento en caché semántico, la seguridad del contenido y el equilibrio de carga.
USO PARA: AI Gateway, APIM, configurar gateway, configurar gateway, agregar gateway, modelo de gateway, servidor MCP, límite de velocidad, límite de token, caché semántica, seguridad de contenido, equilibrio de carga, importación de OpenAPI, convertir API a MCP.
NO UTILIZAR PARA: implementar modelos (use microsoft-foundry), Azure Functions (use azure-functions), bases de datos (use azure-postgres).
