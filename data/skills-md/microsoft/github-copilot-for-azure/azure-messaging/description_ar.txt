استكشاف الأخطاء وإصلاحها وحل المشكلات باستخدام Azure Messaging SDK لمراكز الأحداث وحافلة الخدمة. يغطي حالات فشل الاتصال، وأخطاء المصادقة، ومشكلات معالجة الرسائل، ومشكلات تكوين SDK. الاستخدام لـ: خطأ SDK لمركز الأحداث، مشكلة SDK لحافلة الخدمة، فشل اتصال الرسائل، خطأ AMQP، مشكلة مضيف معالج الأحداث، فقدان قفل الرسائل، مهلة الإرسال، قطع اتصال جهاز الاستقبال، استكشاف أخطاء SDK وإصلاحها، SDK للمراسلة اللازوردية، مستهلك مركز الأحداث، مشكلة قائمة انتظار ناقل الخدمة، خطأ في الاشتراك في الموضوع، تمكين مركز أحداث التسجيل، تسجيل ناقل الخدمة، إيفينثوب بيثون، سيرفيسبوس جافا، إيفينثوب جافا سكريبت، سيرفيسبوس دوت نت، نقطة تفتيش محور الأحداث، مركز الأحداث لا يتلقى الرسائل، لا تستخدم الرسالة الميتة لحافلة الخدمة من أجل: إنشاء موارد مركز الأحداث أو ناقل الخدمة (استخدم إعداد azure)، ومقاييس المراقبة (استخدم إمكانية ملاحظة azure)، وتحليل التكلفة (استخدم تحسين تكلفة azure)
