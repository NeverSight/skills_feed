قم بتحسين توافق المهارات الأمامية بشكل متكرر باستخدام نمط حلقة رالف. الاستخدام من أجل: تشغيل المدرب، مساعدة المدرب، تحسين المهارات، إصلاح المادة الأمامية، الامتثال للمهارة، تدقيق المادة الأمامية، تحسين المشغلات، إضافة مضادات المشغلات، تحسين مهارات الدفعة، التحقق من رموز المهارة. لا تستخدم من أجل: إنشاء مهارات جديدة (استخدم تأليف المهارات)، أو كتابة محتوى المهارات، أو تحسين الرمز المميز فقط (استخدم مُحسِّن تخفيض السعر المميز)، أو التغييرات غير المتعلقة بالواجهة الأمامية.
