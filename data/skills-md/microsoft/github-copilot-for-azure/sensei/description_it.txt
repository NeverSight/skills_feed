Migliora in modo iterativo la conformità alle priorità delle competenze utilizzando il modello Ralph loop. UTILIZZARE PER: eseguire il sensei, aiutare il sensei, migliorare l'abilità, correggere il frontmatter, la conformità delle abilità, il controllo del frontmatter, migliorare i trigger, aggiungere anti-trigger, miglioramento batch delle abilità, controllare i token delle abilità. NON UTILIZZARE PER: creazione di nuove competenze (utilizzare skill-authoring), scrittura di contenuti di competenze, solo ottimizzazione dei token (utilizzare markdown-token-optimizer) o modifiche non legate al frontmatter.
