Cree aplicaciones de IA mediante el SDK de Python de Azure AI Projects (azure-ai-projects). Úselo cuando trabaje con clientes de proyectos Foundry, cree agentes versionados con PromptAgentDefinition, ejecute evaluaciones, administre conexiones/implementaciones/conjuntos de datos/índices o utilice clientes compatibles con OpenAI. Este es el SDK de Foundry de alto nivel: para operaciones de agentes de bajo nivel, utilice la habilidad azure-ai-agents-python.
