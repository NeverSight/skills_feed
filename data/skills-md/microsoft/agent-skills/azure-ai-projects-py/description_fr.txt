Créez des applications IA à l’aide du SDK Python Azure AI Projects (azure-ai-projects). À utiliser lorsque vous travaillez avec des clients de projet Foundry, créez des agents versionnés avec PromptAgentDefinition, exécutez des évaluations, gérez les connexions/déploiements/ensembles de données/index ou utilisez des clients compatibles OpenAI. Il s'agit du SDK Foundry de haut niveau : pour les opérations d'agent de bas niveau, utilisez la compétence azure-ai-agents-python.
