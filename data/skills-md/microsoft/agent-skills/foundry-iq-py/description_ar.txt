أنشئ حلول استرجاع وكيل باستخدام قواعد معارف Azure AI Search وخدمة Foundry Agent Service باستخدام Python SDK. يُستخدم عند إنشاء مصادر/قواعد المعرفة، أو ربط الوكلاء عبر MCP لـ RAG، أو تنفيذ البحث المختلط مع إعادة الترتيب الدلالي، أو إنشاء تطبيقات المحادثة مع دعم الاقتباس. يغطي SearchIndexClient وKnowledgeBaseRetrievalClient وAIProjectClient.
