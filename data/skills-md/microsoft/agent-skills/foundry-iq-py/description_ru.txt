Создавайте решения агентного поиска с помощью баз знаний Azure AI Search и службы агента Foundry с помощью пакета SDK Python. Используйте при создании источников/баз знаний, подключении агентов через MCP для RAG, реализации гибридного поиска с семантическим реранжированием или создании диалоговых приложений с поддержкой цитирования. Охватывает SearchIndexClient, KnowledgeBaseRetrivalClient и AIProjectClient.
