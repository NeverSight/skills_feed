Kit SDK Azure Resource Manager pour Fabric dans .NET. À utiliser pour les opérations du PLAN DE GESTION : provisionnement, mise à l'échelle, suspension/reprise des capacités Microsoft Fabric, vérification de la disponibilité des noms et liste des SKU via Azure Resource Manager. Déclencheurs : "Capacité de structure", "créer de la capacité", "suspendre la capacité", "reprendre la capacité", "Fabric SKU", "provision Fabric", "ARM Fabric", "FabricCapacityResource".
