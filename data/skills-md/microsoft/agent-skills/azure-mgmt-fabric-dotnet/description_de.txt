Azure Resource Manager SDK für Fabric in .NET. Verwendung für MANAGEMENT PLANE-Vorgänge: Bereitstellung, Skalierung, Anhalten/Wiederaufnahme von Microsoft Fabric-Kapazitäten, Überprüfen der Namensverfügbarkeit und Auflisten von SKUs über Azure Resource Manager. Auslöser: „Fabric-Kapazität“, „Kapazität erstellen“, „Kapazität aussetzen“, „Kapazität wiederherstellen“, „Fabric-SKU“, „Fabric bereitstellen“, „ARM-Fabric“, „FabricCapacityResource“.
