SDK de Azure Resource Manager para Fabric en .NET. Úselo para operaciones del PLANO DE ADMINISTRACIÓN: aprovisionamiento, escalado, suspensión/reanudación de capacidades de Microsoft Fabric, verificación de la disponibilidad de nombres y listado de SKU a través de Azure Resource Manager. Activadores: "Capacidad de Fabric", "crear capacidad", "suspender capacidad", "reanudar capacidad", "SKU de Fabric", "aprovisionar Fabric", "ARM Fabric", "FabricCapacityResource".
