SDK de proyectos de IA de Azure para .NET. Cliente de alto nivel para proyectos de Azure AI Foundry, incluidos agentes, conexiones, conjuntos de datos, implementaciones, evaluaciones e índices. Úselo para la gestión de proyectos de AI Foundry, agentes versionados y orquestación.
