Kit de développement logiciel (SDK) Azure Service Bus pour .NET. Messagerie d'entreprise avec files d'attente, sujets, abonnements et sessions. À utiliser pour la livraison fiable des messages, les modèles de publication/sous-traitance, la gestion des lettres mortes et le traitement en arrière-plan. Déclencheurs : "Service Bus", "ServiceBusClient", "ServiceBusSender", "ServiceBusReceiver", "ServiceBusProcessor", "file d'attente de messages", "pub/sub .NET", "file d'attente de lettres mortes".
