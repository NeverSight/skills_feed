Пакет SDK конфигурации приложений Azure для Java. Централизованное управление конфигурацией приложений с помощью настроек «ключ-значение», флагов функций и снимков.
Триггеры: «ConfigurationClient Java», «Java-конфигурация приложения», «Java-флаг функции», «Java-настройка конфигурации», «Java-конфигурация Azure».
