SDK de Azure Resource Manager para pruebas de dramaturgos de Microsoft en .NET. Úselo para operaciones del PLANO DE ADMINISTRACIÓN: crear/administrar áreas de trabajo de Playwright Testing, verificar la disponibilidad de nombres y administrar cuotas de áreas de trabajo a través de Azure Resource Manager. NO para ejecutar pruebas de Playwright; use Azure.Developer.MicrosoftPlaywrightTesting.NUnit para eso. Activadores: "Espacio de trabajo de Dramaturgo", "crear espacio de trabajo de Pruebas de Dramaturgo", "administrar recursos de Dramaturgo", "ARM Dramaturgo", "PlaywrightWorkspaceResource", "aprovisionar Pruebas de Dramaturgo".
