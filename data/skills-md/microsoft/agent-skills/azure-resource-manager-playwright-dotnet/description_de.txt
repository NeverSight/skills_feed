Azure Resource Manager SDK für Microsoft Playwright-Tests in .NET. Verwendung für MANAGEMENT PLANE-Vorgänge: Erstellen/Verwalten von Playwright Testing-Arbeitsbereichen, Überprüfen der Namensverfügbarkeit und Verwalten von Arbeitsbereichskontingenten über Azure Resource Manager. NICHT zum Ausführen von Playwright-Tests – verwenden Sie dafür Azure.Developer.MicrosoftPlaywrightTesting.NUnit. Auslöser: „Playwright-Arbeitsbereich“, „Playwright-Testarbeitsbereich erstellen“, „Playwright-Ressourcen verwalten“, „ARM Playwright“, „PlaywrightWorkspaceResource“, „Playwright-Testing bereitstellen“.
