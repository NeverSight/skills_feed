用於 .NET 中的 Microsoft Playwright 測試的 Azure 資源管理器 SDK。用於管理平面操作：創建/管理 Playwright 測試工作區、檢查名稱可用性以及通過 Azure 資源管理器管理工作區配額。不適用於運行 Playwright 測試 - 請使用 Azure.Developer.MicrosoftPlaywrightTesting.NUnit。觸發器：“Playwright 工作區”、“創建 Playwright 測試工作區”、“管理 Playwright 資源”、“ARM Playwright”、“PlaywrightWorkspaceResource”、“提供 Playwright 測試”。
