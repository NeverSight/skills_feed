Azure Resource Manager SDK für Redis in .NET. Verwendung für MANAGEMENT PLANE-Vorgänge: Erstellen/Verwalten von Azure Cache for Redis-Instanzen, Firewallregeln, Zugriffsschlüsseln, Patchzeitplänen, Verbindungsservern (Georeplikation) und privaten Endpunkten über Azure Resource Manager. NICHT für Operationen auf Datenebene (Schlüssel abrufen/setzen, Pub/Sub) – verwenden Sie dafür StackExchange.Redis. Auslöser: „Redis-Cache“, „Redis erstellen“, „Redis verwalten“, „ARM Redis“, „RedisResource“, „Redis bereitstellen“, „Azure Cache für Redis“.
