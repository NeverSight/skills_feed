Erstellen Sie Azure AI Foundry-Agenten mit dem Microsoft Agent Framework Python SDK (agent-framework-azure-ai). Verwenden Sie diese Option, wenn Sie persistente Agents mit AzureAIAgentsProvider erstellen, gehostete Tools (Codeinterpreter, Dateisuche, Websuche) verwenden, MCP-Server integrieren, Konversationsthreads verwalten oder Streaming-Antworten implementieren. Behandelt Funktionstools, strukturierte Ausgaben und Multitool-Agenten.
