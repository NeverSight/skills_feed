قم ببناء وكلاء Azure AI Foundry باستخدام Microsoft Agent Framework Python SDK (agent-framework-azure-ai). يُستخدم عند إنشاء وكلاء دائمين باستخدام AzureAIAgentsProvider، أو استخدام الأدوات المستضافة (مترجم التعليمات البرمجية، أو البحث عن الملفات، أو بحث الويب)، أو دمج خوادم MCP، أو إدارة سلاسل المحادثات، أو تنفيذ استجابات التدفق. يغطي الأدوات الوظيفية والمخرجات المنظمة والوكلاء متعددي الأدوات.
