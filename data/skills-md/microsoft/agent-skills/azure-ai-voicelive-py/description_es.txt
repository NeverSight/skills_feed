Cree aplicaciones de IA de voz en tiempo real mediante el SDK de Azure AI Voice Live (azure-ai-voicelive). Utilice esta habilidad al crear aplicaciones Python que necesiten comunicación de audio bidireccional en tiempo real con Azure AI, incluidos asistentes de voz, chatbots habilitados para voz, traducción de voz a voz en tiempo real, avatares controlados por voz o cualquier transmisión de audio basada en WebSocket con modelos de IA. Admite Server VAD (detección de actividad de voz), conversación por turnos, llamadas de funciones, herramientas MCP, integración de avatar y transcripción.
