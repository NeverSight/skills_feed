Пакет SDK Azure AI Projects для Java. Высокоуровневый пакет SDK для управления проектами Azure AI Foundry, включая подключения, наборы данных, индексы и оценки.
Триггеры: «AIProjectClient java», «Java проектов Azure AI», «Java проекта Foundry», «ConnectionsClient», «DatasetsClient», «IndexesClient».
