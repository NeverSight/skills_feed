Kit de développement logiciel (SDK) des agents Microsoft 365 pour Python. Créez des agents multicanaux pour Teams/M365/Copilot Studio avec l'hébergement aiohttp, le routage AgentApplication, les réponses en continu et l'authentification basée sur MSAL. Déclencheurs : « SDK Microsoft 365 Agents », « microsoft_agents », « AgentApplication », « start_agent_process », « TurnContext », « Client Copilot Studio », « CloudAdapter ».
