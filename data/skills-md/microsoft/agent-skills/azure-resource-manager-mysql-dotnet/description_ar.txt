Azure MySQL Flex Server SDK لـ .NET. إدارة قواعد البيانات لعمليات نشر MySQL Elastic Server. يُستخدم لإنشاء الخوادم وقواعد البيانات وقواعد جدار الحماية والتكوينات والنسخ الاحتياطية والتوفر العالي. المشغلات: "MySQL"، "MySqlFlexibleServer"، "MySQL Elastic Server"، "Azure Database for MySQL"، "إدارة قاعدة بيانات MySQL"، "جدار الحماية MySQL"، "النسخ الاحتياطي لـ MySQL".
