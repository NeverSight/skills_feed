Пакет SDK Azure Resource Manager для службы ботов в .NET. Операции плоскости управления для создания ресурсов Azure Bot, каналов (Teams, DirectLine, Slack) и управления ими, а также параметров подключения. Триггеры: «Служба бота», «BotResource», «Бот Azure», «Канал DirectLine», «Канал Teams», «Управление ботом .NET», «Создать бота».
