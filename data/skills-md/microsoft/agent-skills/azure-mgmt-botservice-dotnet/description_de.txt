Azure Resource Manager SDK für Bot Service in .NET. Verwaltungsebenenvorgänge zum Erstellen und Verwalten von Azure Bot-Ressourcen, Kanälen (Teams, DirectLine, Slack) und Verbindungseinstellungen. Auslöser: „Bot Service“, „BotResource“, „Azure Bot“, „DirectLine-Kanal“, „Teams-Kanal“, „Bot-Management .NET“, „Bot erstellen“.
