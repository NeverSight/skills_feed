SDK de Azure Resource Manager para el Programador de tareas duradero en .NET. Úselo para operaciones del PLANO DE ADMINISTRACIÓN: creación/administración de programadores de tareas duraderos, centros de tareas y políticas de retención a través de Azure Resource Manager. Activadores: "Programador de tareas duradero", "crear programador", "centro de tareas", "DurableTaskSchedulerResource", "aprovisionar tareas duraderas", "programador de orquestación".
