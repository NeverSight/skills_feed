SDK de Azure AI Text Analytics para análisis de sentimientos, reconocimiento de entidades, frases clave, detección de idioma, PII y PNL para atención médica. Úselo para el procesamiento del lenguaje natural en texto.
Activadores: "análisis de texto", "análisis de sentimiento", "reconocimiento de entidad", "frase clave", "detección de PII", "TextAnalyticsClient".
