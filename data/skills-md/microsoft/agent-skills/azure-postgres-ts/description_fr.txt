Connectez-vous à Azure Database pour PostgreSQL Flexible Server à partir de Node.js/TypeScript à l'aide du package pg (node-postgres). À utiliser pour les requêtes PostgreSQL, le regroupement de connexions, les transactions et l'authentification Microsoft Entra ID (sans mot de passe).
