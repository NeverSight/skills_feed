REST API преобразования речи в текст Azure для коротких аудио (Python). Используйте для простого распознавания речи аудиофайлов продолжительностью до 60 секунд без Speech SDK.
Триггеры: «REST преобразования речи в текст», «короткая аудиотранскрипция», «REST API распознавания речи», «STT REST», «распознавание речи REST».
НЕ ИСПОЛЬЗУЙТЕ ДЛЯ: длинного звука (>60 секунд), потоковой передачи в реальном времени, пакетной транскрипции, пользовательских моделей речи, перевода речи. Вместо этого используйте Speech SDK или API пакетной транскрипции.
