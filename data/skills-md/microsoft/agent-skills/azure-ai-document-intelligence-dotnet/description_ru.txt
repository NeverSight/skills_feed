Пакет SDK Azure AI Document Intelligence для .NET. Извлекайте текст, таблицы и структурированные данные из документов, используя готовые и пользовательские модели. Используйте для обработки счетов, извлечения квитанций, анализа идентификационных документов и создания пользовательских моделей документов. Триггеры: «Разведка документов», «DocumentIntelligenceClient», «распознаватель форм», «извлечение счетов», «распознавание текста квитанций», «анализ документов .NET».
