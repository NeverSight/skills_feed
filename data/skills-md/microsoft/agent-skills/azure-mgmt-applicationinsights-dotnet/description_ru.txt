Пакет SDK Azure Application Insights для .NET. Мониторинг производительности приложений и управление ресурсами наблюдения. Используйте для создания компонентов Application Insights, веб-тестов, книг, элементов аналитики и ключей API. Триггеры: «Application Insights», «ApplicationInsights», «App Insights», «APM», «мониторинг приложений», «веб-тесты», «тесты доступности», «книги».
