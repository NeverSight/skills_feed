Azure PostgreSQL خادم SDK المرن لـ .NET. إدارة قواعد البيانات لعمليات نشر PostgreSQL Elastic Server. يُستخدم لإنشاء الخوادم وقواعد البيانات وقواعد جدار الحماية والتكوينات والنسخ الاحتياطية والتوفر العالي. المشغلات: "PostgreSQL"، و"PostgreSqlFlexibleServer"، و"PostgreSQL Elastic Server"، و"Azure Database for PostgreSQL"، و"إدارة قاعدة بيانات PostgreSQL"، و"جدار حماية PostgreSQL"، و"PostgreSQL Backup"، و"Postgres".
