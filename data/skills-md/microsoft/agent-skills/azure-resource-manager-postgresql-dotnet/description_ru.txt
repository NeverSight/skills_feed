Пакет SDK гибкого сервера Azure PostgreSQL для .NET. Управление базами данных для развертываний PostgreSQL Flexible Server. Используйте для создания серверов, баз данных, правил брандмауэра, конфигураций, резервного копирования и обеспечения высокой доступности. Триггеры: «PostgreSQL», «PostgreSqlFlexibleServer», «Гибкий сервер PostgreSQL», «База данных Azure для PostgreSQL», «Управление базой данных PostgreSQL», «Брандмауэр PostgreSQL», «Резервное копирование PostgreSQL», «Postgres».
