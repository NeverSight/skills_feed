قم بإنشاء روايات صوتية بنمط البودكاست مدعومة بالذكاء الاصطناعي باستخدام نموذج GPT Realtime Mini الخاص بـ Azure OpenAI عبر WebSocket. يُستخدم عند إنشاء ميزات تحويل النص إلى كلام، أو إنشاء السرد الصوتي، أو إنشاء البودكاست من المحتوى، أو التكامل مع Azure OpenAI Realtime API للحصول على إخراج صوتي حقيقي. يغطي التنفيذ الكامل للمكدس من واجهة React الأمامية إلى واجهة Python FastAPI الخلفية مع تدفق WebSocket.
