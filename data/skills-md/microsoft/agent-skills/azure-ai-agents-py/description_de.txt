Erstellen Sie KI-Agenten mit dem Azure AI Agents Python SDK (azure-ai-agents). Verwenden Sie diese Option, wenn Sie auf Azure AI Foundry gehostete Agents mit Tools (Dateisuche, Code-Interpreter, Bing Grounding, Azure AI Search, Function Calling, OpenAPI, MCP) erstellen, Threads und Nachrichten verwalten, Streaming-Antworten implementieren oder mit Vektorspeichern arbeiten. Dies ist das Low-Level-SDK – für Abstraktionen auf höherer Ebene verwenden Sie stattdessen den Agent-Framework-Skill.
