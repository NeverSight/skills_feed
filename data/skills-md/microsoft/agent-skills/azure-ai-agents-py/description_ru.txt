Создавайте агенты ИИ с помощью пакета SDK Python для агентов ИИ Azure (azure-ai-agents). Используйте при создании агентов, размещенных в Azure AI Foundry, с помощью инструментов (поиск файлов, интерпретатор кода, заземление Bing, поиск Azure AI, вызов функций, OpenAPI, MCP), управления потоками и сообщениями, реализации потоковых ответов или работы с векторными хранилищами. Это SDK низкого уровня — для абстракций более высокого уровня вместо этого используйте навык агента-фреймворка.
