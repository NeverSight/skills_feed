Azure AI エージェント Python SDK (azure-ai-agents) を使用して AI エージェントを構築します。ツール (ファイル検索、コード インタープリター、Bing Grounding、Azure AI Search、関数呼び出し、OpenAPI、MCP) を使用して Azure AI Foundry でホストされるエージェントを作成する場合、スレッドとメッセージを管理する場合、ストリーミング応答を実装する場合、またはベクター ストアを操作する場合に使用します。これは低レベルの SDK です。より高いレベルの抽象化の場合は、代わりにエージェント フレームワーク スキルを使用してください。
