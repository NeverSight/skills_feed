Modèles de code propres pour le SDK Python Azure AI Search (azure-search-documents). À utiliser lors de la création d'applications de recherche, de la création/gestion d'index, de la mise en œuvre de la récupération agentique avec des bases de connaissances ou de l'utilisation de la recherche vectorielle/hybride. Couvre SearchClient, SearchIndexClient, SearchIndexerClient et KnowledgeBaseRetrievalClient.
