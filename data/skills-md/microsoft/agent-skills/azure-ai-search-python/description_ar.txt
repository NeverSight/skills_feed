أنماط التعليمات البرمجية النظيفة لـ Azure AI Search Python SDK (مستندات البحث azure). يُستخدم عند إنشاء تطبيقات البحث، أو إنشاء/إدارة الفهارس، أو تنفيذ استرجاع الوكيل باستخدام قواعد المعرفة، أو العمل مع البحث المتجه/المختلط. يغطي SearchClient وSearchIndexClient وSearchIndexerClient وKnowledgeBaseRetrievalClient.
