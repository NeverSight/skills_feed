Kit SDK Azure Resource Manager pour Azure SQL dans .NET. À utiliser pour les opérations MANAGEMENT PLANE : création/gestion de serveurs SQL, de bases de données, de pools élastiques, de règles de pare-feu et de groupes de basculement via Azure Resource Manager. PAS pour les opérations sur le plan de données (exécution de requêtes) - utilisez Microsoft.Data.SqlClient pour cela. Déclencheurs : "Serveur SQL", "créer une base de données SQL", "gérer les ressources SQL", "ARM SQL", "SqlServerResource", "provisionner Azure SQL", "pool élastique", "règle de pare-feu".
