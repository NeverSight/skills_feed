SDK de Azure Resource Manager para Azure SQL en .NET. Úselo para operaciones del PLANO DE ADMINISTRACIÓN: creación/administración de servidores SQL, bases de datos, grupos elásticos, reglas de firewall y grupos de conmutación por error a través de Azure Resource Manager. NO para operaciones del plano de datos (ejecución de consultas); utilice Microsoft.Data.SqlClient para eso. Activadores: "servidor SQL", "crear base de datos SQL", "administrar recursos SQL", "ARM SQL", "SqlServerResource", "aprovisionar Azure SQL", "grupo elástico", "regla de firewall".
