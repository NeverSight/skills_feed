Azure Resource Manager SDK لـ Azure SQL في .NET. يُستخدم لعمليات خطة الإدارة: إنشاء/إدارة خوادم SQL وقواعد البيانات والتجمعات المرنة وقواعد جدار الحماية ومجموعات تجاوز الفشل عبر Azure Resource Manager. ليس لعمليات مستوى البيانات (تنفيذ الاستعلامات) - استخدم Microsoft.Data.SqlClient لذلك. المشغلات: "خادم SQL"، "إنشاء قاعدة بيانات SQL"، "إدارة موارد SQL"، "ARM SQL"، "SqlServerResource"، "توفير Azure SQL"، "التجمع المرن"، "قاعدة جدار الحماية".
