Azure Resource Manager SDK für Azure SQL in .NET. Verwendung für MANAGEMENT PLANE-Vorgänge: Erstellen/Verwalten von SQL-Servern, Datenbanken, elastischen Pools, Firewallregeln und Failovergruppen über Azure Resource Manager. NICHT für Operationen auf Datenebene (Ausführen von Abfragen) – verwenden Sie dafür Microsoft.Data.SqlClient. Auslöser: „SQL Server“, „SQL-Datenbank erstellen“, „SQL-Ressourcen verwalten“, „ARM SQL“, „SqlServerResource“, „Azure SQL bereitstellen“, „elastischer Pool“, „Firewallregel“.
