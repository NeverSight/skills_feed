Пакет SDK Azure Resource Manager для Azure SQL в .NET. Используйте для операций на плоскости управления: создание и управление SQL-серверами, базами данных, эластичными пулами, правилами брандмауэра и группами отработки отказа с помощью Azure Resource Manager. НЕ для операций с плоскостью данных (выполнение запросов) - используйте для этого Microsoft.Data.SqlClient. Триггеры: «SQL-сервер», «создание базы данных SQL», «управление ресурсами SQL», «ARM SQL», «SqlServerResource», «предоставление SQL Azure», «эластичный пул», «правило брандмауэра».
