SDK di Azure Resource Manager per Azure SQL in .NET. Utilizzare per operazioni MANAGEMENT PLANE: creazione/gestione di server SQL, database, pool elastici, regole firewall e gruppi di failover tramite Azure Resource Manager. NON per operazioni sul piano dati (esecuzione di query): utilizzare Microsoft.Data.SqlClient per questo. Trigger: "server SQL", "crea database SQL", "gestisci risorse SQL", "ARM SQL", "SqlServerResource", "provisioning Azure SQL", "pool elastico", "regola firewall".
