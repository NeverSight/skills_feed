أنشئ تطبيقات الذكاء الاصطناعي باستخدام Azure AI Projects Python SDK (azure-ai-projects). يُستخدم عند العمل مع عملاء مشروع Foundry، أو إنشاء وكلاء بإصدارات باستخدام PromptAgentDefinition، أو تشغيل التقييمات، أو إدارة الاتصالات/عمليات النشر/مجموعات البيانات/الفهارس، أو استخدام العملاء المتوافقين مع OpenAI. هذه هي Foundry SDK عالية المستوى - بالنسبة لعمليات الوكيل ذات المستوى المنخفض، استخدم مهارة azure-ai-agents-python.
