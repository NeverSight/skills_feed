Azure AI プロジェクト Python SDK (azure-ai-projects) を使用して AI アプリケーションを構築します。 Foundry プロジェクト クライアントを操作する場合、PromptAgentDefinition を使用してバージョン管理されたエージェントを作成する場合、評価を実行する場合、接続/デプロイメント/データセット/インデックスを管理する場合、または OpenAI 互換クライアントを使用する場合に使用します。これは高レベルの Foundry SDK です。低レベルのエージェント操作には、azure-ai-agents-python スキルを使用します。
