Пакет SDK Azure Resource Manager для Cosmos DB в .NET. Используйте для операций на плоскости управления: создание и управление учетными записями Cosmos DB, базами данных, контейнерами, параметрами пропускной способности и RBAC через Azure Resource Manager. НЕ для операций плоскости данных (CRUD в документах) — используйте для этого Microsoft.Azure.Cosmos. Триггеры: «Учетная запись Cosmos DB», «создать учетную запись Cosmos», «управление ресурсами Cosmos», «ARM Cosmos», «CosmosDBAccountResource», «предоставление Cosmos DB».
