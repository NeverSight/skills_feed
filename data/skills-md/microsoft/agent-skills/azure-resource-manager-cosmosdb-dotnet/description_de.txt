Azure Resource Manager SDK für Cosmos DB in .NET. Verwendung für MANAGEMENT PLANE-Vorgänge: Erstellen/Verwalten von Cosmos DB-Konten, Datenbanken, Containern, Durchsatzeinstellungen und RBAC über Azure Resource Manager. NICHT für Operationen auf Datenebene (CRUD für Dokumente) – verwenden Sie dafür Microsoft.Azure.Cosmos. Auslöser: „Cosmos DB-Konto“, „Cosmos-Konto erstellen“, „Cosmos-Ressourcen verwalten“, „ARM Cosmos“, „CosmosDBAccountResource“, „Cosmos DB bereitstellen“.
