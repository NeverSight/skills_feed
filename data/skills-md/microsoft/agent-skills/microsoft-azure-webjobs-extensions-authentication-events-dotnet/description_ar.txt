Microsoft Entra Authentication Events SDK لـ .NET. تعمل وظائف Azure على تشغيل ملحقات المصادقة المخصصة. يُستخدم لإثراء الرمز المميز، والمطالبات المخصصة، وجمع السمات، وتخصيص OTP في معرف Entra. المشغلات: "أحداث المصادقة"، و"WebJobsAuthenticationEventsTrigger"، و"OnTokenIssuanceStart"، و"OnAttributeCollectionStart"، و"المطالبات المخصصة"، و"إثراء الرمز المميز"، و"ملحق Entra المخصص"، و"ملحق المصادقة".
