Пакет SDK Microsoft Entra Authentication Events для .NET. Триггеры функций Azure для пользовательских расширений проверки подлинности. Используйте для расширения токенов, пользовательских утверждений, сбора атрибутов и настройки OTP в Entra ID. Триггеры: «События аутентификации», «WebJobsAuthenticationEventsTrigger», «OnTokenIssuanceStart», «OnAttributeCollectionStart», «пользовательские утверждения», «обогащение токена», «Пользовательское расширение Entra», «расширение аутентификации».
