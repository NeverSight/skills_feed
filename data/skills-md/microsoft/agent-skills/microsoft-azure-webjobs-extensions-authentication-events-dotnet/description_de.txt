Microsoft Entra Authentication Events SDK für .NET. Azure Functions-Trigger für benutzerdefinierte Authentifizierungserweiterungen. Verwendung für Token-Anreicherung, benutzerdefinierte Ansprüche, Attributsammlung und OTP-Anpassung in Entra ID. Auslöser: „Authentifizierungsereignisse“, „WebJobsAuthenticationEventsTrigger“, „OnTokenIssuanceStart“, „OnAttributeCollectionStart“, „benutzerdefinierte Ansprüche“, „Token-Anreicherung“, „benutzerdefinierte Entra-Erweiterung“, „Authentifizierungserweiterung“.
