.NET 用の Microsoft Entra Authentication Events SDK。カスタム認証拡張機能の Azure Functions トリガー。 Entra ID でのトークン エンリッチメント、カスタム クレーム、属性収集、OTP カスタマイズに使用します。トリガー: 「認証イベント」、「WebJobsAuthenticationEventsTrigger」、「OnTokenIssuanceStart」、「OnAttributeCollectionStart」、「カスタム クレーム」、「トークン エンリッチメント」、「Entra カスタム拡張機能」、「認証拡張機能」。
