SDK degli eventi di autenticazione di Microsoft Entra per .NET. Trigger di Funzioni di Azure per le estensioni di autenticazione personalizzate. Utilizzare per l'arricchimento dei token, le attestazioni personalizzate, la raccolta di attributi e la personalizzazione OTP nell'ID Entra. Trigger: "Eventi di autenticazione", "WebJobsAuthenticationEventsTrigger", "OnTokenIssuanceStart", "OnAttributeCollectionStart", "attestazioni personalizzate", "arricchimento token", "estensione personalizzata Entra", "estensione di autenticazione".
