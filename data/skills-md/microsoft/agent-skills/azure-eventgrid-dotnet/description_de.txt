Azure Event Grid SDK für .NET. Clientbibliothek zum Veröffentlichen und Nutzen von Ereignissen mit Azure Event Grid. Verwendung für ereignisgesteuerte Architekturen, Pub/Sub-Messaging, CloudEvents und EventGridEvents. Auslöser: „Event Grid“, „EventGridPublisherClient“, „CloudEvent“, „EventGridEvent“, „Ereignisse veröffentlichen .NET“, „ereignisgesteuert“, „pub/sub“.
