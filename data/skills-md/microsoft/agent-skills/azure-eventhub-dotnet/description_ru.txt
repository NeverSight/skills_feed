Пакет SDK Центров событий Azure для .NET. Используйте для потоковой передачи событий с высокой пропускной способностью: отправка событий (EventHubProducerClient, EventHubBufferedProducerClient), получение событий (EventProcessorClient с контрольными точками), управление разделами и прием данных в реальном времени. Триггеры: «Концентраторы событий», «потоковая передача событий», «EventHubProducerClient», «EventProcessorClient», «отправка событий», «получение событий», «контрольная точка», «раздел».
