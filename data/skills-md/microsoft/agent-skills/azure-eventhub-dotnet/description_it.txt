SDK di Hub eventi di Azure per .NET. Utilizzare per lo streaming di eventi a throughput elevato: invio di eventi (EventHubProducerClient, EventHubBufferedProducerClient), ricezione di eventi (EventProcessorClient con checkpoint), gestione delle partizioni e inserimento di dati in tempo reale.
