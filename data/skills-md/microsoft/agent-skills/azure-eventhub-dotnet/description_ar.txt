Azure Event Hubs SDK لـ .NET. يستخدم لتدفق الأحداث عالية الإنتاجية: إرسال الأحداث (EventHubProducerClient، EventHubBufferedProducerClient)، تلقي الأحداث (EventProcessorClient مع نقطة التفتيش)، وإدارة الأقسام، واستيعاب البيانات في الوقت الفعلي. المشغلات: "مراكز الأحداث"، و"تدفق الأحداث"، و"EventHubProducerClient"، و"EventProcessorClient"، و"إرسال الأحداث"، و"استقبال الأحداث"، و"checkpointing"، و"partition".
