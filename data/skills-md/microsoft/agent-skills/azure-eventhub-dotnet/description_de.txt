Azure Event Hubs SDK für .NET. Verwendung für Ereignis-Streaming mit hohem Durchsatz: Senden von Ereignissen (EventHubProducerClient, EventHubBufferedProducerClient), Empfangen von Ereignissen (EventProcessorClient mit Checkpointing), Partitionsverwaltung und Datenaufnahme in Echtzeit. Auslöser: „Event Hubs“, „Event-Streaming“, „EventHubProducerClient“, „EventProcessorClient“, „Ereignisse senden“, „Ereignisse empfangen“, „Checkpointing“, „Partition“.
