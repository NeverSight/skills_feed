.NET 用の Azure Event Hubs SDK。高スループットのイベント ストリーミングに使用します: イベントの送信 (EventHubProducerClient、EventHubBufferedProducerClient)、イベントの受信 (チェックポイント機能を備えた EventProcessorClient)、パーティション管理、およびリアルタイム データの取り込み。トリガー: 「Event Hubs」、「イベント ストリーミング」、「EventHubProducerClient」、「EventProcessorClient」、「イベントの送信」、「イベントの受信」、「チェックポイント」、「パーティション」。
