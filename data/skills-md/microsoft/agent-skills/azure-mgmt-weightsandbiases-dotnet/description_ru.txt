Пакет SDK Azure Weights & Biases для .NET. Отслеживание экспериментов машинного обучения и управление моделями через Azure Marketplace. Используйте для создания экземпляров W&B, управления единым входом, интеграции с рынком и наблюдения за машинным обучением. Триггеры: «Вес и предвзятость», «W&B», «WeightsAndBiases», «Отслеживание экспериментов ML», «Реестр моделей», «Управление экспериментами», «wandb».
