Kit de développement logiciel (SDK) d’ingestion Azure Monitor pour Java. Envoyez des journaux personnalisés à Azure Monitor via les règles de collecte de données (DCR) et les points de terminaison de collecte de données (DCE).
Déclencheurs : « LogsIngestionClient java », « Azure Monitor ingestion java », « logs personnalisés java », « DCR java », « règle de collecte de données java ».
