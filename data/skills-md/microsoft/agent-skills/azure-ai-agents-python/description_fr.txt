Créez des agents IA à l’aide du SDK Python Azure AI Agents (azure-ai-agents). À utiliser lors de la création d'agents hébergés sur Azure AI Foundry avec des outils (Recherche de fichiers, Interpréteur de code, Bing Grounding, Azure AI Search, Function Calling, OpenAPI, MCP), de la gestion des threads et des messages, de la mise en œuvre de réponses en continu ou de l'utilisation de magasins de vecteurs. Il s'agit du SDK de bas niveau. Pour les abstractions de niveau supérieur, utilisez plutôt la compétence agent-framework.
