أنشئ وكلاء الذكاء الاصطناعي باستخدام Azure AI Agents Python SDK (azure-ai-agents). استخدمه عند إنشاء وكلاء مستضافين على Azure AI Foundry باستخدام الأدوات (البحث عن الملفات، أو Code Interpreter، أو Bing Grounding، أو Azure AI Search، أو Function Calling، أو OpenAPI، أو MCP)، أو إدارة المواضيع والرسائل، أو تنفيذ استجابات التدفق، أو العمل مع مخازن المتجهات. هذا هو SDK منخفض المستوى - بالنسبة للتجريدات ذات المستوى الأعلى، استخدم مهارة إطار عمل الوكيل بدلاً من ذلك.
