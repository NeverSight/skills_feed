Kit SDK d'inférence Azure AI pour .NET. Complétions de chat, intégrations et inférence de modèle pour les modèles d'IA déployés via Azure AI Foundry et Azure Machine Learning Studio. À utiliser pour les points de terminaison d'API sans serveur et les points de terminaison de calcul gérés. Déclencheurs : "Azure.AI.Inference", "ChatCompletionsClient", "EmbeddingsClient", "inférence de modèle", "achèvements de chat .NET", "incorporations de texte .NET", "inférence Azure AI Foundry".
