SDK de inferencia de IA de Azure para .NET. Finalizaciones de chat, incorporaciones e inferencia de modelos para modelos de IA implementados a través de Azure AI Foundry y Azure Machine Learning Studio. Úselo para puntos finales de API sin servidor y puntos finales de computación administrada. Desencadenadores: "Azure.AI.Inference", "ChatCompletionsClient", "EmbeddingsClient", "inferencia de modelo", "finalizaciones de chat .NET", "incrustaciones de texto .NET", "inferencia de Azure AI Foundry".
