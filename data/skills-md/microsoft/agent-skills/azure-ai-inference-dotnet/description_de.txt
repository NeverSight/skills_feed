Azure AI Inference SDK für .NET. Chat-Vervollständigungen, Einbettungen und Modellinferenz für KI-Modelle, die über Azure AI Foundry und Azure Machine Learning Studio bereitgestellt werden. Verwendung für serverlose API-Endpunkte und verwaltete Compute-Endpunkte. Auslöser: „Azure.AI.Inference“, „ChatCompletionsClient“, „EmbeddingsClient“, „Model Inference“, „Chat Completions .NET“, „Text Embeddings .NET“, „Azure AI Foundry Inference“.
