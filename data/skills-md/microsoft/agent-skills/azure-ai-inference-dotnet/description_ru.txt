Пакет SDK Azure AI Inference для .NET. Завершения чата, внедрения и вывод моделей для моделей ИИ, развернутых с помощью Azure AI Foundry и Azure Machine Learning Studio. Используйте для бессерверных конечных точек API и конечных точек управляемых вычислений. Триггеры: «Azure.AI.Inference», «ChatCompletionsClient», «EmbeddingsClient», «вывод модели», «завершения чата .NET», «внедрения текста .NET», «вывод Azure AI Foundry».
