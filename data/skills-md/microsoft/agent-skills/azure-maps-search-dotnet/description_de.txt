Azure Maps SDK für .NET. Standortbasierte Dienste einschließlich Geokodierung, Routing, Rendering, Geolokalisierung und Wetter. Zur Adresssuche, Wegbeschreibungen, Kartenkacheln, IP-Geolokalisierung und Wetterdaten. Auslöser: „Azure Maps“, „MapsSearchClient“, „MapsRoutingClient“, „MapsRenderingClient“, „Geocoding .NET“, „Routenrichtungen“, „Kartenkacheln“, „Geolocation“.
