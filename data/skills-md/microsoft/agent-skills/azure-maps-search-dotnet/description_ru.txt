Пакет SDK Azure Maps для .NET. Сервисы на основе местоположения, включая геокодирование, маршрутизацию, рендеринг, геолокацию и погоду. Используйте для поиска адресов, направлений, фрагментов карты, геолокации IP и данных о погоде. Триггеры: «Azure Maps», «MapsSearchClient», «MapsRoutingClient», «MapsRenderingClient», «геокодирование .NET», «направления маршрута», «фрагменты карты», «геолокация».
