Kit de développement logiciel (SDK) Azure Maps pour .NET. Services basés sur la localisation, notamment le géocodage, le routage, le rendu, la géolocalisation et la météo. À utiliser pour la recherche d'adresses, les itinéraires, les vignettes cartographiques, la géolocalisation IP et les données météorologiques. Déclencheurs : "Azure Maps", "MapsSearchClient", "MapsRoutingClient", "MapsRenderingClient", "géocodage .NET", "itinéraires", "tuiles de carte", "géolocalisation".
