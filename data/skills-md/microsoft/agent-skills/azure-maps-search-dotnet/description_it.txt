SDK di Mappe di Azure per .NET. Servizi basati sulla posizione tra cui geocodifica, routing, rendering, geolocalizzazione e meteo. Utilizzare per la ricerca di indirizzi, indicazioni stradali, riquadri della mappa, geolocalizzazione IP e dati meteorologici. Trigger: "Azure Maps", "MapsSearchClient", "MapsRoutingClient", "MapsRenderingClient", "geocodifica .NET", "indicazioni stradali", "riquadri mappa", "geolocalizzazione".
