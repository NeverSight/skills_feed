Kit de développement logiciel Azure API Center pour .NET. Gestion centralisée de l'inventaire des API avec gouvernance, gestion des versions et découverte. À utiliser pour créer des services API, des espaces de travail, des API, des versions, des définitions, des environnements, des déploiements et des schémas de métadonnées.
