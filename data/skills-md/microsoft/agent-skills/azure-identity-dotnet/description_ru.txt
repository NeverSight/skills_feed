Пакет SDK Azure Identity для .NET. Библиотека аутентификации для клиентов Azure SDK с использованием Microsoft Entra ID. Используйте для DefaultAzureCredential, управляемого удостоверения, субъектов-служб и учетных данных разработчика. Триггеры: «Azure Identity», «DefaultAzureCredential», «ManagedIdentityCredential», «ClientSecretCredential», «аутентификация .NET», «Azure auth», «цепочка учетных данных».
