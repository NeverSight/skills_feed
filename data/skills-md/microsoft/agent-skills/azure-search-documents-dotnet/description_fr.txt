Kit de développement logiciel (SDK) Azure AI Search pour .NET (Azure.Search.Documents). À utiliser pour créer des applications de recherche avec recherche en texte intégral, vectorielle, sémantique et hybride. Couvre SearchClient (requêtes, document CRUD), SearchIndexClient (gestion des index) et SearchIndexerClient (indexeurs, compétences). Déclencheurs : « Azure Search .NET », « SearchClient », « SearchIndexClient », « recherche vectorielle C# », « recherche sémantique .NET », « recherche hybride », « Azure.Search.Documents ».
