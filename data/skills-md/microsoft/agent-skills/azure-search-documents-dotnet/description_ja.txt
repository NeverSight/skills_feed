.NET 用の Azure AI Search SDK (Azure.Search.Documents)。全文検索、ベクター検索、セマンティック検索、およびハイブリッド検索を備えた検索アプリケーションを構築するために使用します。 SearchClient (クエリ、ドキュメント CRUD)、SearchIndexClient (インデックス管理)、および SearchIndexerClient (インデクサー、スキルセット) をカバーします。トリガー: "Azure Search .NET"、"SearchClient"、"SearchIndexClient"、"ベクター検索 C#"、"セマンティック検索 .NET"、"ハイブリッド検索"、"Azure.Search.Documents"。
