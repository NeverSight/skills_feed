SDK de Azure AI Search para .NET (Azure.Search.Documents). Úselo para crear aplicaciones de búsqueda con búsqueda de texto completo, vectorial, semántica e híbrida. Cubre SearchClient (consultas, documentos CRUD), SearchIndexClient (administración de índices) y SearchIndexerClient (indexadores, conjuntos de habilidades). Desencadenadores: "Azure Search .NET", "SearchClient", "SearchIndexClient", "búsqueda vectorial C#", "búsqueda semántica .NET", "búsqueda híbrida", "Azure.Search.Documents".
