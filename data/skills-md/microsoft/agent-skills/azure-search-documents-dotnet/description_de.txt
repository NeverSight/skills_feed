Azure AI Search SDK für .NET (Azure.Search.Documents). Zum Erstellen von Suchanwendungen mit Volltext-, Vektor-, semantischer und Hybridsuche. Deckt SearchClient (Abfragen, Dokument-CRUD), SearchIndexClient (Indexverwaltung) und SearchIndexerClient (Indexer, Skillsets) ab. Auslöser: „Azure Search .NET“, „SearchClient“, „SearchIndexClient“, „Vektorsuche C#“, „semantische Suche .NET“, „Hybridsuche“, „Azure.Search.Documents“.
