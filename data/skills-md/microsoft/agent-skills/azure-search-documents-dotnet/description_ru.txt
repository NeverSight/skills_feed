Пакет SDK Azure AI Search для .NET (Azure.Search.Documents). Используйте для создания поисковых приложений с полнотекстовым, векторным, семантическим и гибридным поиском. Охватывает SearchClient (запросы, документ CRUD), SearchIndexClient (управление индексами) и SearchIndexerClient (индексаторы, наборы навыков). Триггеры: «Azure Search .NET», «SearchClient», «SearchIndexClient», «векторный поиск C#», «семантический поиск .NET», «гибридный поиск», «Azure.Search.Documents».
