Verwalten Sie MongoDB Atlas-Organisationen als Azure ARM-Ressourcen mit dem Azure.ResourceManager.MongoDBAtlas SDK. Wird zum Erstellen, Aktualisieren, Auflisten oder Löschen von MongoDB Atlas-Organisationen über die Azure Marketplace-Integration verwendet. Dieses SDK verwaltet die Azure-seitige Organisationsressource, nicht direkt Atlas-Cluster/Datenbanken.
