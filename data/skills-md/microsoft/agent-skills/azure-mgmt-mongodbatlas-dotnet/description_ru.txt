Управляйте организациями MongoDB Atlas как ресурсами Azure ARM с помощью Azure.ResourceManager.MongoDBAtlas SDK. Используйте при создании, обновлении, перечислении или удалении организаций MongoDB Atlas посредством интеграции с Azure Marketplace. Этот SDK управляет ресурсом организации на стороне Azure, а не напрямую кластерами/базами данных Atlas.
