Gestisci le organizzazioni MongoDB Atlas come risorse ARM di Azure usando Azure.ResourceManager.MongoDBAtlas SDK. Da utilizzare durante la creazione, l'aggiornamento, l'elenco o l'eliminazione di organizzazioni MongoDB Atlas tramite l'integrazione di Azure Marketplace. Questo SDK gestisce la risorsa dell'organizzazione lato Azure, non direttamente i cluster/database Atlas.
