Boîte à outils pour générer des notes de version PowerToys à partir des PR de jalons GitHub ou des plages de validation. À utiliser lorsqu'on vous le demande pour créer des notes de version, résumer les PR d'étape, générer un journal des modifications, préparer la documentation de version, demander des révisions Copilot pour les PR, mettre à jour README pour une nouvelle version, gérer les jalons de PR ou collecter les PR entre les validations/balises. Prend en charge la collecte de relations publiques par jalon ou plage de validation, l'attribution de jalons, le regroupement par étiquette, le résumé avec attribution de contributeur externe et le remplacement de version README.
