مجموعة أدوات لإنشاء ملاحظات إصدار PowerToys من العلاقات العامة المهمة في GitHub أو نطاقات الالتزام. استخدمه عندما يُطلب منك إنشاء ملاحظات الإصدار، أو تلخيص نتائج العلاقات العامة، أو إنشاء سجل التغيير، أو إعداد وثائق الإصدار، أو طلب مراجعات Copilot للعلاقات العامة، أو تحديث الملف README لإصدار جديد، أو إدارة معالم العلاقات العامة، أو جمع العلاقات العامة بين الالتزامات/العلامات. يدعم جمع العلاقات العامة حسب المعالم الرئيسية أو نطاق الالتزام، وتعيين المعالم الرئيسية، والتجميع حسب التسمية، والتلخيص بإسناد المساهم الخارجي، ورفع إصدار README.
