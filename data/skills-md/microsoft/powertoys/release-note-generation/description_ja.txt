GitHub マイルストーン PR またはコミット範囲から PowerToys リリース ノートを生成するためのツールキット。リリース ノートの作成、マイルストーン PR の要約、変更ログの生成、リリース ドキュメントの準備、PR の Copilot レビューのリクエスト、新しいリリースの README の更新、PR マイルストーンの管理、またはコミット/タグ間の PR の収集を求められた場合に使用します。マイルストーンまたはコミット範囲による PR 収集、マイルストーンの割り当て、ラベルによるグループ化、外部貢献者の帰属による要約、および README のバージョン バンピングをサポートします。
