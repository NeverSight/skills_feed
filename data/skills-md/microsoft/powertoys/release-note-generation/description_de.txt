Toolkit zum Generieren von PowerToys-Versionshinweisen aus GitHub-Meilenstein-PRs oder Commit-Bereichen. Verwenden Sie diese Option, wenn Sie dazu aufgefordert werden, Versionshinweise zu erstellen, Meilenstein-PRs zusammenzufassen, Änderungsprotokolle zu erstellen, Versionsdokumentation vorzubereiten, Copilot-Bewertungen für PRs anzufordern, README für eine neue Version zu aktualisieren, PR-Meilensteine ​​zu verwalten oder PRs zwischen Commits/Tags zu sammeln. Unterstützt die PR-Erfassung nach Meilenstein oder Commit-Bereich, Meilensteinzuweisung, Gruppierung nach Label, Zusammenfassung mit externer Mitwirkendenzuordnung und README-Versions-Bumping.
