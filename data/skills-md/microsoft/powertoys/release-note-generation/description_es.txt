Kit de herramientas para generar notas de la versión de PowerToys a partir de relaciones públicas de hitos de GitHub o rangos de confirmación. Úselo cuando se le solicite crear notas de la versión, resumir PR de hitos, generar registros de cambios, preparar documentación de lanzamiento, solicitar revisiones de Copilot para PR, actualizar README para una nueva versión, administrar hitos de PR o recopilar PR entre confirmaciones/etiquetas. Admite la recopilación de relaciones públicas por hito o rango de compromiso, asignación de hitos, agrupación por etiqueta, resumen con atribución de colaborador externo y modificación de versiones README.
