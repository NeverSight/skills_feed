Набор инструментов для создания примечаний к выпуску PowerToys на основе PR-запросов или диапазонов коммитов GitHub. Используйте, когда вас просят создать примечания к выпуску, обобщить важные PR-запросы, создать журнал изменений, подготовить документацию к выпуску, запросить обзоры PR для Copilot, обновить README для нового выпуска, управлять контрольными точками PR или собирать PR-запросы между коммитами/тегами. Поддерживает сбор PR по вехам или диапазону фиксации, назначение вех, группировку по меткам, обобщение с указанием внешних участников и обновление версий README.
