Toolkit per generare note di rilascio di PowerToys da PR cardine di GitHub o intervalli di commit. Da utilizzare quando richiesto per creare note di rilascio, riepilogare le pietre miliari delle PR, generare un registro delle modifiche, preparare la documentazione di rilascio, richiedere revisioni del copilota per le PR, aggiornare il file README per una nuova versione, gestire le pietre miliari delle PR o raccogliere PR tra commit/tag. Supporta la raccolta PR per traguardo o intervallo di impegno, assegnazione di traguardo, raggruppamento per etichetta, riepilogo con attribuzione di contributore esterno e bumping della versione README.
