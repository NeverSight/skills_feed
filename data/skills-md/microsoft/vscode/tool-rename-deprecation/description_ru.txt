Обеспечьте сохранение обратной совместимости переименованных встроенных ссылок на инструменты. Используйте при переименовании ToolReferenceName, ReferenceName набора инструментов или любого идентификатора инструмента. Запускать при ЛЮБОМ изменении регистрационного кода инструмента. Охватывает LegacyToolReferenceFullNames для инструментов и LegacyFullNames для наборов инструментов.
