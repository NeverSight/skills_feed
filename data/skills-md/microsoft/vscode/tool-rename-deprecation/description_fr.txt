Assurez-vous que les références d’outils intégrés renommés préservent la compatibilité ascendante. À utiliser pour renommer un nom de référence d'outil, un nom de référence d'ensemble d'outils ou tout identifiant d'outil. Exécuter sur TOUT changement apporté au code d’enregistrement de l’outil. Couvre les LegacyToolReferenceFullNames pour les outils et les LegacyFullNames pour les ensembles d’outils.
