Wird zur Validierung von Azure DevOps-Pipeline-Änderungen für den VS Code-Build verwendet. Behandelt das Einreihen von Builds in die Warteschlange, das Überprüfen des Build-Status, das Anzeigen von Protokollen und das Iterieren von Pipeline-YAML-Änderungen, ohne auf vollständige CI-Ausführungen warten zu müssen.
