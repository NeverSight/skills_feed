Рекомендации по обеспечению специальных возможностей для функций VS Code — охватывают диалоговые окна помощи по специальным возможностям, доступные представления, настройки подробностей, сигналы доступности, оповещения/объявления о состоянии ARIA, навигацию с помощью клавиатуры и метки/роли ARIA. Применяется как к новым интерактивным поверхностям пользовательского интерфейса, так и к обновлениям существующих функций. Используйте при создании нового пользовательского интерфейса или обновлении существующих функций пользовательского интерфейса.
