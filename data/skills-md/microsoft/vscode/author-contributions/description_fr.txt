Identifiez tous les fichiers auxquels un auteur spécifique a contribué sur une branche par rapport à son amont, en traçant le code via les renommages. À utiliser lorsqu'on lui demande qui a édité quoi, quel code un auteur a contribué ou pour vérifier la paternité avant une fusion. Cette compétence doit être exécutée en tant que sous-agent : elle effectue de nombreuses opérations git et renvoie un tableau concis.
