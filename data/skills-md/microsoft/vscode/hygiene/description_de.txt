Verwenden Sie diese Option, wenn Sie Codeänderungen vornehmen, um sicherzustellen, dass sie die Hygieneprüfungen von VS Code bestehen. Behandelt den Pre-Commit-Hook, Unicode-Einschränkungen, String-Anführungszeichenregeln, Copyright-Header, Einrückung, Formatierung, ESLint und Stylelint. Führen Sie den Hygienecheck durch, bevor Sie die Arbeiten für abgeschlossen erklären.
