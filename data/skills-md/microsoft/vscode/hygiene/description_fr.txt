À utiliser lors de la modification du code pour garantir qu'ils réussissent les contrôles d'hygiène de VS Code. Couvre le hook de pré-validation, les restrictions Unicode, les règles de citation de chaînes, les en-têtes de droits d'auteur, l'indentation, le formatage, ESLint et stylelint. Effectuez le contrôle d’hygiène avant de déclarer les travaux terminés.
