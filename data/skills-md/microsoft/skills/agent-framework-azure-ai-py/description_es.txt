Cree agentes de Azure AI Foundry mediante el SDK de Python de Microsoft Agent Framework (agent-framework-azure-ai). Úselo al crear agentes persistentes con AzureAIAgentsProvider, usar herramientas alojadas (intérprete de código, búsqueda de archivos, búsqueda web), integrar servidores MCP, administrar subprocesos de conversación o implementar respuestas de transmisión. Cubre herramientas funcionales, salidas estructuradas y agentes multiherramienta.
