Azure Event Grid SDK لـ .NET. مكتبة العميل لنشر الأحداث واستهلاكها باستخدام Azure Event Grid. يُستخدم للبنيات المستندة إلى الأحداث، ورسائل النشر/الفرعية، وCloudEvents، وEventGridEvents. المشغلات: "Event Grid"، و"EventGridPublisherClient"، و"CloudEvent"، و"EventGridEvent"، و"publish events .NET"، و"event-driven"، و"pub/sub".
