Пакет SDK службы "Сетка событий Azure" для .NET. Клиентская библиотека для публикации и использования событий с помощью Azure Event Grid. Используйте для архитектур, управляемых событиями, обмена сообщениями издателя и подписки, CloudEvents и EventGridEvents. Триггеры: «Сетка событий», «EventGridPublisherClient», «CloudEvent», «EventGridEvent», «публикация событий .NET», «управляемый событиями», «pub/sub».
