Kit de développement logiciel (SDK) Azure Event Grid pour .NET. Bibliothèque cliente pour publier et consommer des événements avec Azure Event Grid. À utiliser pour les architectures basées sur les événements, la messagerie pub/sub, CloudEvents et EventGridEvents. Déclencheurs : "Event Grid", "EventGridPublisherClient", "CloudEvent", "EventGridEvent", "publish events .NET", "event-driven", "pub/sub".
