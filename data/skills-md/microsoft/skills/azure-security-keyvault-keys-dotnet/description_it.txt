SDK delle chiavi di Azure Key Vault per .NET. Libreria client per la gestione delle chiavi crittografiche in Azure Key Vault e HSM gestito. Utilizzare per la creazione, la rotazione, la crittografia, la decrittografia, la firma e la verifica delle chiavi.
