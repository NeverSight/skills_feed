Kit de développement logiciel (SDK) de clés Azure Key Vault pour .NET. Bibliothèque client pour la gestion des clés cryptographiques dans Azure Key Vault et Managed HSM. À utiliser pour la création, la rotation, le chiffrement, le déchiffrement, la signature et la vérification de clés. Déclencheurs : « Clés Key Vault », « KeyClient », « CryptographyClient », « Clé RSA », « Clé EC », « Chiffrer déchiffrer .NET », « Rotation des clés », « HSM ».
