SDK de claves de Azure Key Vault para .NET. Biblioteca cliente para administrar claves criptográficas en Azure Key Vault y Managed HSM. Úselo para la creación, rotación, cifrado, descifrado, firma y verificación de claves. Activadores: "Claves de Key Vault", "KeyClient", "CryptographyClient", "Clave RSA", "Clave EC", "cifrar y descifrar .NET", "rotación de claves", "HSM".
