Azure Key Vault Keys SDK für .NET. Clientbibliothek zum Verwalten kryptografischer Schlüssel in Azure Key Vault und Managed HSM. Zur Schlüsselerstellung, -rotation, -verschlüsselung, -entschlüsselung, -signierung und -verifizierung. Auslöser: „Key Vault-Schlüssel“, „KeyClient“, „CryptographyClient“, „RSA-Schlüssel“, „EC-Schlüssel“, „verschlüsseln, entschlüsseln .NET“, „Schlüsselrotation“, „HSM“.
