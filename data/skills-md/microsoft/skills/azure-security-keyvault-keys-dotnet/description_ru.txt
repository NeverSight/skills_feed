Пакет SDK ключей Azure Key Vault для .NET. Клиентская библиотека для управления криптографическими ключами в Azure Key Vault и Managed HSM. Используйте для создания, ротации, шифрования, дешифрования, подписания и проверки ключей. Триггеры: «Ключи хранилища ключей», «Клиент ключей», «Клиент криптографии», «Ключ RSA», «Ключ EC», «шифровать дешифровать .NET», «ротацию ключей», «HSM».
