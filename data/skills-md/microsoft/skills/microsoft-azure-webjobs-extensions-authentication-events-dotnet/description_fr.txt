SDK d’événements d’authentification Microsoft Entra pour .NET. Déclencheurs Azure Functions pour les extensions d’authentification personnalisées. À utiliser pour l'enrichissement des jetons, les revendications personnalisées, la collecte d'attributs et la personnalisation OTP dans Entra ID. Déclencheurs : "Événements d'authentification", "WebJobsAuthenticationEventsTrigger", "OnTokenIssuanceStart", "OnAttributeCollectionStart", "revendications personnalisées", "enrichissement de jetons", "extension personnalisée Entra", "extension d'authentification".
