SDK de eventos de autenticación de Microsoft Entra para .NET. Desencadenadores de Azure Functions para extensiones de autenticación personalizadas. Úselo para enriquecimiento de tokens, reclamos personalizados, recopilación de atributos y personalización de OTP en Entra ID. Activadores: "Eventos de autenticación", "WebJobsAuthenticationEventsTrigger", "OnTokenIssuanceStart", "OnAttributeCollectionStart", "reclamaciones personalizadas", "enriquecimiento de token", "Extensión personalizada de entrada", "extensión de autenticación".
