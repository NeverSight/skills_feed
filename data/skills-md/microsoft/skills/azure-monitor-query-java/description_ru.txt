Пакет SDK запросов Azure Monitor для Java. Выполняйте запросы Kusto к рабочим областям Log Analytics и запрашивайте метрики из ресурсов Azure.
Триггеры: «LogsQueryClient Java», «MetricsQueryClient Java», «Java-запрос Kusto», «Java-аналитика журналов», «Java-запрос монитора Azure».
Примечание. Этот пакет устарел. Перейдите на azure-monitor-query-logs и azure-monitor-query-metrics.
