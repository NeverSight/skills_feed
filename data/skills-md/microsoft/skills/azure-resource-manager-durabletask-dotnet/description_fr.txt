SDK Azure Resource Manager pour le planificateur de tâches durable dans .NET. À utiliser pour les opérations MANAGEMENT PLANE : création/gestion de planificateurs de tâches durables, de hubs de tâches et de politiques de rétention via Azure Resource Manager. Déclencheurs : « Durable Task Scheduler », « créer un planificateur », « hub de tâches », « DurableTaskSchedulerResource », « provisionner une tâche durable », « planificateur d'orchestration ».
