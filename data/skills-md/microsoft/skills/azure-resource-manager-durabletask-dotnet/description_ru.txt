Пакет SDK Azure Resource Manager для устойчивого планировщика задач в .NET. Используйте для операций на плоскости управления: создание устойчивых планировщиков задач, концентраторов задач и политик хранения и управление ими с помощью Azure Resource Manager. Триггеры: «Планировщик устойчивых задач», «Создать планировщик», «концентратор задач», «DurableTaskSchedulerResource», «предоставление устойчивых задач», «планировщик оркестрации».
