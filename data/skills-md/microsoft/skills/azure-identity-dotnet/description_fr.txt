Kit de développement logiciel (SDK) Azure Identity pour .NET. Bibliothèque d'authentification pour les clients Azure SDK utilisant Microsoft Entra ID. À utiliser pour DefaultAzureCredential, l’identité managée, les principaux de service et les informations d’identification du développeur. Déclencheurs : « Azure Identity », « DefaultAzureCredential », « ManagedIdentityCredential », « ClientSecretCredential », « authentification .NET », « Azure auth », « chaîne d'informations d'identification ».
