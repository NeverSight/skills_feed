Kit de développement logiciel (SDK) de serveur flexible Azure MySQL pour .NET. Gestion de base de données pour les déploiements de MySQL Flexible Server. À utiliser pour créer des serveurs, des bases de données, des règles de pare-feu, des configurations, des sauvegardes et une haute disponibilité. Déclencheurs : "MySQL", "MySqlFlexibleServer", "MySQL Flexible Server", "Azure Database pour MySQL", "Gestion de base de données MySQL", "Pare-feu MySQL", "Sauvegarde MySQL".
