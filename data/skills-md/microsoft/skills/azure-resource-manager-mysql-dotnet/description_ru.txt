Пакет SDK гибкого сервера Azure MySQL для .NET. Управление базами данных для развертываний MySQL Flexible Server. Используйте для создания серверов, баз данных, правил брандмауэра, конфигураций, резервного копирования и обеспечения высокой доступности. Триггеры: «MySQL», «MySqlFlexibleServer», «Гибкий сервер MySQL», «База данных Azure для MySQL», «Управление базой данных MySQL», «Брандмауэр MySQL», «Резервное копирование MySQL».
