Kit SDK de serveur flexible Azure PostgreSQL pour .NET. Gestion de base de données pour les déploiements de PostgreSQL Flexible Server. À utiliser pour créer des serveurs, des bases de données, des règles de pare-feu, des configurations, des sauvegardes et une haute disponibilité.
