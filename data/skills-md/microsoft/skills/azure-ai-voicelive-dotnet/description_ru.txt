Пакет SDK Azure AI Voice Live для .NET. Создавайте приложения голосового искусственного интеллекта в реальном времени с помощью двунаправленной связи через WebSocket. Используйте для голосовых помощников, диалогового искусственного интеллекта, преобразования речи в речь в реальном времени и чат-ботов с голосовой поддержкой. Триггеры: «голос в реальном времени», «голос в реальном времени», «VoiceLiveClient», «VoiceLiveSession», «голосовой помощник .NET», «двунаправленный звук», «речь в речь».
