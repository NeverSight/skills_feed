Пакет SDK Azure AI Projects для .NET. Клиент высокого уровня для проектов Azure AI Foundry, включая агенты, соединения, наборы данных, развертывания, оценки и индексы. Используйте для управления проектами AI Foundry, агентов с поддержкой версий и оркестрации. Триггеры: «Проекты AI», «AIProjectClient», «Проект Foundry», «Агенты с версиями», «Оценки», «Наборы данных», «Соединения», «Развертывания .NET».
