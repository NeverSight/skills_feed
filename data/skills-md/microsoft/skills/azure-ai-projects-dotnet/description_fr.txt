Kit de développement logiciel (SDK) de projets Azure AI pour .NET. Client de haut niveau pour les projets Azure AI Foundry, notamment les agents, les connexions, les ensembles de données, les déploiements, les évaluations et les index. À utiliser pour la gestion de projet AI Foundry, les agents versionnés et l'orchestration. Déclencheurs : "Projets AI", "AIProjectClient", "Projet Foundry", "Agents versionnés", "évaluations", "ensembles de données", "connexions", "déploiements .NET".
