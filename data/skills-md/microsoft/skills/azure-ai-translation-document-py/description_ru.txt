Пакет SDK для перевода документов Azure AI для пакетного перевода документов с сохранением формата. Используйте для перевода документов Word, PDF, Excel, PowerPoint и других форматов в любом масштабе.
Триггеры: «перевод документов», «пакетный перевод», «перевести документы», «DocumentTranslationClient».
