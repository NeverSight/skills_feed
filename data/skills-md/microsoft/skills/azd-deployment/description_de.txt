Stellen Sie Containeranwendungen in Azure Container Apps mithilfe der Azure Developer CLI (AZD) bereit. Verwenden Sie diese Option, wenn Sie Azd-Projekte einrichten, die azure.yaml-Konfiguration schreiben, eine Bicep-Infrastruktur für Container-Apps erstellen, Remote-Builds mit ACR konfigurieren, idempotente Bereitstellungen implementieren, Umgebungsvariablen über lokal/.azure/Bicep hinweg verwalten oder Azd-Up-Fehler beheben. Wird bei Anfragen zur Azd-Konfiguration, Container-Apps-Bereitstellung, Multi-Service-Bereitstellungen und Infrastructure-as-Code mit Bicep ausgelöst.
