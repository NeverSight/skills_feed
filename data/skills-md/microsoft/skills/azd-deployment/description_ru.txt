Развертывайте контейнерные приложения в приложениях-контейнерах Azure с помощью интерфейса командной строки Azure Developer (azd). Используйте при настройке проектов azd, написании конфигурации azure.yaml, создании инфраструктуры Bicep для контейнерных приложений, настройке удаленных сборок с помощью ACR, реализации идемпотентных развертываний, управлении переменными среды в local/.azure/Bicep или устранении сбоев azd up. Запускается по запросам на настройку azd, развертывание контейнерных приложений, развертывание нескольких сервисов и инфраструктуру как код с помощью Bicep.
