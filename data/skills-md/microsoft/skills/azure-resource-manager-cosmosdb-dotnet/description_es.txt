SDK de Azure Resource Manager para Cosmos DB en .NET. Úselo para operaciones del PLANO DE ADMINISTRACIÓN: creación/administración de cuentas, bases de datos, contenedores, configuraciones de rendimiento y RBAC de Cosmos DB a través de Azure Resource Manager. NO para operaciones del plano de datos (CRUD en documentos): use Microsoft.Azure.Cosmos para eso. Activadores: "cuenta de Cosmos DB", "crear cuenta de Cosmos", "administrar recursos de Cosmos", "ARM Cosmos", "CosmosDBAccountResource", "aprovisionar Cosmos DB".
