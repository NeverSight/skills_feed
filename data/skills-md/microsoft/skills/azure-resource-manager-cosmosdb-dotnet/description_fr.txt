Kit SDK Azure Resource Manager pour Cosmos DB dans .NET. À utiliser pour les opérations MANAGEMENT PLANE : création/gestion de comptes Cosmos DB, de bases de données, de conteneurs, de paramètres de débit et de RBAC via Azure Resource Manager. PAS pour les opérations sur le plan de données (CRUD sur les documents) - utilisez Microsoft.Azure.Cosmos pour cela. Déclencheurs : "Compte Cosmos DB", "créer un compte Cosmos", "gérer les ressources Cosmos", "ARM Cosmos", "CosmosDBAccountResource", "provisionner Cosmos DB".
