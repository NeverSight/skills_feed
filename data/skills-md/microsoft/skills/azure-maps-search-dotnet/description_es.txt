SDK de Azure Maps para .NET. Servicios basados ​​en la ubicación que incluyen codificación geográfica, enrutamiento, renderizado, geolocalización y clima. Úselo para búsqueda de direcciones, direcciones, mosaicos de mapas, geolocalización de IP y datos meteorológicos. Activadores: "Azure Maps", "MapsSearchClient", "MapsRoutingClient", "MapsRenderingClient", "geocodificación .NET", "indicaciones de ruta", "mosaicos de mapa", "geolocalización".
