Kit SDK Azure Resource Manager pour la gestion des API dans .NET. À utiliser pour les opérations MANAGEMENT PLANE : création/gestion de services APIM, d'API, de produits, d'abonnements, de stratégies, d'utilisateurs, de groupes, de passerelles et de backends via Azure Resource Manager. Déclencheurs : « Gestion des API », « Service APIM », « Créer APIM », « Gérer les API », « ApiManagementServiceResource », « Politiques API », « Produits APIM », « Abonnements APIM ».
