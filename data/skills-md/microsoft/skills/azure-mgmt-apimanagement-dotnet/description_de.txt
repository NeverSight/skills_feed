Azure Resource Manager SDK für API Management in .NET. Verwendung für MANAGEMENT PLANE-Vorgänge: Erstellen/Verwalten von APIM-Diensten, APIs, Produkten, Abonnements, Richtlinien, Benutzern, Gruppen, Gateways und Backends über Azure Resource Manager. Auslöser: „API-Management“, „APIM-Dienst“, „APIM erstellen“, „APIs verwalten“, „ApiManagementServiceResource“, „API-Richtlinien“, „APIM-Produkte“, „APIM-Abonnements“.
