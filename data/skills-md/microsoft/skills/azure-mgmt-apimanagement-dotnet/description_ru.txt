Пакет SDK Azure Resource Manager для управления API в .NET. Используйте для операций на плоскости управления: создание и управление службами APIM, API, продуктами, подписками, политиками, пользователями, группами, шлюзами и серверными модулями через Azure Resource Manager. Триггеры: «Управление API», «Служба APIM», «Создание APIM», «Управление API», «ApiManagementServiceResource», «Политики API», «Продукты APIM», «Подписки APIM».
