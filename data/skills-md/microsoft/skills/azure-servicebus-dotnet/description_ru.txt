Пакет SDK служебной шины Azure для .NET. Корпоративный обмен сообщениями с очередями, темами, подписками и сеансами. Используйте для надежной доставки сообщений, шаблонов публикации/подписки, обработки недоставленных писем и фоновой обработки. Триггеры: «Служебная шина», «ServiceBusClient», «ServiceBusSender», «ServiceBusReceiver», «ServiceBusProcessor», «очередь сообщений», «pub/sub .NET», «очередь недоставленных писем».
