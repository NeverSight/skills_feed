Пакет SDK Azure Resource Manager для тестирования драматургов Microsoft в .NET. Используйте для операций на плоскости управления: создание рабочих пространств драматургического тестирования и управление ими, проверка доступности имен и управление квотами рабочих пространств с помощью Azure Resource Manager. НЕ для запуска тестов Playwright — используйте для этого Azure.Developer.MicrosoftPlaywrightTesting.NUnit. Триггеры: «Рабочая область драматурга», «создать рабочую область тестирования драматурга», «управлять ресурсами драматурга», «ARM драматург», «PlaywrightWorkspaceResource», «обеспечить тестирование драматурга».
