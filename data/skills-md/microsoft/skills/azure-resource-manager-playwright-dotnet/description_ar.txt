Azure Resource Manager SDK لاختبار Microsoft Playwright في .NET. يُستخدم لعمليات خطة الإدارة: إنشاء/إدارة مساحات عمل اختبار الكاتب المسرحي، والتحقق من توفر الاسم، وإدارة حصص مساحة العمل عبر Azure Resource Manager. ليس لإجراء اختبارات Playwright - استخدم Azure.Developer.MicrosoftPlaywrightTesting.NUnit لذلك. المشغلات: "مساحة عمل الكاتب المسرحي"، "إنشاء مساحة عمل لاختبار الكاتب المسرحي"، "إدارة موارد الكاتب المسرحي"، "ARM Playwright"، "PlaywrightWorkspaceResource"، "توفير اختبار الكاتب المسرحي".
