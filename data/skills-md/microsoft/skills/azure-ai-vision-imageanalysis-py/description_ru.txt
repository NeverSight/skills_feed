Пакет SDK для анализа изображений Azure AI Vision для подписей, тегов, объектов, оптического распознавания символов, обнаружения людей и умной обрезки. Используйте для задач компьютерного зрения и понимания изображений.
Триггеры: «анализ изображения», «компьютерное зрение», «OCR», «обнаружение объекта», «ImageAnalysisClient», «подпись к изображению».
