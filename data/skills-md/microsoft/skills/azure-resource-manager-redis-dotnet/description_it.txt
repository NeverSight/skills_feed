SDK di Azure Resource Manager per Redis in .NET. Utilizzo per operazioni MANAGEMENT PLANE: creazione/gestione di istanze di Cache Redis di Azure, regole firewall, chiavi di accesso, pianificazioni di patch, server collegati (replica geografica) ed endpoint privati ​​tramite Azure Resource Manager. NON per operazioni sul piano dati (chiavi get/set, pub/sub): utilizzare StackExchange.Redis per questo. Trigger: "cache Redis", "crea Redis", "gestisci Redis", "ARM Redis", "RedisResource", "provisioning Redis", "Cache di Azure per Redis".
