Azure Resource Manager SDK لـ Redis في .NET. يُستخدم لعمليات خطة الإدارة: إنشاء/إدارة Azure Cache لمثيلات Redis وقواعد جدار الحماية ومفاتيح الوصول وجداول التصحيح والخوادم المرتبطة (النسخ المتماثل الجغرافي) ونقاط النهاية الخاصة عبر Azure Resource Manager. ليس لعمليات مستوى البيانات (الحصول على/تعيين المفاتيح، pub/sub) - استخدم StackExchange.Redis لذلك. المشغلات: "ذاكرة التخزين المؤقت لـ Redis"، و"إنشاء Redis"، و"إدارة Redis"، و"ARM Redis"، و"RedisResource"، و"توفير Redis"، و"Azure Cache for Redis".
