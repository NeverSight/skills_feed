Пакет SDK Azure Resource Manager для Redis в .NET. Используйте для операций НА ПЛАНЕ УПРАВЛЕНИЯ: создание или управление кэшем Azure для экземпляров Redis, правил брандмауэра, ключей доступа, расписаний исправлений, связанных серверов (георепликация) и частных конечных точек через Azure Resource Manager. НЕ для операций плоскости данных (получить/установить ключи, публикацию/подписку) - используйте для этого StackExchange.Redis. Триггеры: «Кэш Redis», «создание Redis», «управление Redis», «ARM Redis», «RedisResource», «предоставление Redis», «Кэш Azure для Redis».
