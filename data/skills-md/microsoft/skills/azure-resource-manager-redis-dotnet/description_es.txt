SDK de Azure Resource Manager para Redis en .NET. Úselo para operaciones del PLANO DE ADMINISTRACIÓN: creación/administración de instancias de Azure Cache para Redis, reglas de firewall, claves de acceso, programaciones de parches, servidores vinculados (replicación geográfica) y puntos de conexión privados a través de Azure Resource Manager. NO para operaciones del plano de datos (obtener/establecer claves, pub/sub): use StackExchange.Redis para eso. Desencadenadores: "Caché de Redis", "crear Redis", "administrar Redis", "ARM Redis", "RedisResource", "aprovisionar Redis", "Azure Cache para Redis".
