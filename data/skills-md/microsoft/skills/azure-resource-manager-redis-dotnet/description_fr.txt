Kit SDK Azure Resource Manager pour Redis dans .NET. À utiliser pour les opérations du PLAN DE GESTION : création/gestion des instances Azure Cache pour Redis, des règles de pare-feu, des clés d'accès, des planifications de correctifs, des serveurs liés (géo-réplication) et des points de terminaison privés via Azure Resource Manager. PAS pour les opérations sur le plan de données (obtenir/définir des clés, pub/sub) - utilisez StackExchange.Redis pour cela.
