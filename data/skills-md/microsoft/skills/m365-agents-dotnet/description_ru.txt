Пакет SDK агентов Microsoft 365 для .NET. Создавайте многоканальные агенты для Teams/M365/Copilot Studio с помощью хостинга ASP.NET Core, маршрутизации AgentApplication и проверки подлинности на основе MSAL. Триггеры: «SDK агентов Microsoft 365», «Microsoft.Agents», «AddAgentApplicationOptions», «AgentApplication», «AddAgentAspNetAuthentication», «Клиент Copilot Studio», «IAgentHttpAdapter».
