Kit de développement logiciel (SDK) des agents Microsoft 365 pour .NET. Créez des agents multicanaux pour Teams/M365/Copilot Studio avec l'hébergement ASP.NET Core, le routage AgentApplication et l'authentification basée sur MSAL. Déclencheurs : « SDK Microsoft 365 Agents », « Microsoft.Agents », « AddAgentApplicationOptions », « AgentApplication », « AddAgentAspNetAuthentication », « Client Copilot Studio », « IAgentHttpAdapter ».
