SDK de Azure Event Hubs para .NET. Úselo para transmisión de eventos de alto rendimiento: envío de eventos (EventHubProducerClient, EventHubBufferedProducerClient), recepción de eventos (EventProcessorClient con puntos de control), administración de particiones e ingesta de datos en tiempo real. Activadores: "Event Hubs", "transmisión de eventos", "EventHubProducerClient", "EventProcessorClient", "enviar eventos", "recibir eventos", "puntos de control", "partición".
