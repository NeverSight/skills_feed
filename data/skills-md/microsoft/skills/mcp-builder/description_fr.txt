Guide pour créer des serveurs MCP (Model Context Protocol) de haute qualité qui permettent aux LLM d'interagir avec des services externes grâce à des outils bien conçus. À utiliser lors de la création de serveurs MCP pour intégrer des API ou des services externes, que ce soit en Python (FastMCP), Node/TypeScript (MCP SDK) ou C#/.NET (Microsoft MCP SDK).
