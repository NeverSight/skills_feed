Guía para crear servidores MCP (Protocolo de contexto modelo) de alta calidad que permitan a los LLM interactuar con servicios externos a través de herramientas bien diseñadas. Úselo al crear servidores MCP para integrar API o servicios externos, ya sea en Python (FastMCP), Node/TypeScript (MCP SDK) o C#/.NET (Microsoft MCP SDK).
