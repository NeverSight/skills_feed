Пакет SDK агентов Microsoft 365 для Python. Создавайте многоканальные агенты для Teams/M365/Copilot Studio с помощью хостинга aiohttp, маршрутизации AgentApplication, потоковой передачи ответов и проверки подлинности на основе MSAL. Триггеры: «Microsoft 365 Agents SDK», «microsoft_agents», «AgentApplication», «start_agent_process», «TurnContext», «Клиент Copilot Studio», «CloudAdapter».
