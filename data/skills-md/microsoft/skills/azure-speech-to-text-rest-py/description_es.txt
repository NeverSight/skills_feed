API REST de voz a texto de Azure para audio breve (Python). Úselo para reconocimiento de voz simple de archivos de audio de hasta 60 segundos sin el SDK de voz.
Activadores: "REST de voz a texto", "transcripción de audio breve", "API REST de reconocimiento de voz", "REST STT", "reconocer REST de voz".
NO UTILIZAR PARA: Audio largo (>60 segundos), transmisión en tiempo real, transcripción por lotes, modelos de voz personalizados, traducción de voz. Utilice el SDK de voz o la API de transcripción por lotes en su lugar.
