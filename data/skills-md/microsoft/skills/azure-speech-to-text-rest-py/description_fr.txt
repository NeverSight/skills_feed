API REST Azure Speech to Text pour l'audio court (Python). À utiliser pour une reconnaissance vocale simple de fichiers audio jusqu'à 60 secondes sans le SDK Speech.
Déclencheurs : "parole en texte REST", "transcription audio courte", "API REST de reconnaissance vocale", "STT REST", "reconnaître la parole REST".
NE PAS UTILISER POUR : Audio long (> 60 secondes), streaming en temps réel, transcription par lots, modèles vocaux personnalisés, traduction vocale. Utilisez plutôt le SDK Speech ou l’API de transcription par lots.
