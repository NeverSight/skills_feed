Azure Speech to Text REST API للصوت القصير (Python). يُستخدم للتعرف البسيط على الكلام للملفات الصوتية لمدة تصل إلى 60 ثانية بدون Speech SDK.
المشغلات: "الكلام إلى النص REST"، "النسخ الصوتي القصير"، "REST API للتعرف على الكلام"، "STT REST"، "التعرف على الكلام REST".
لا تستخدمه في: الصوت الطويل (> 60 ثانية)، والبث في الوقت الفعلي، والنسخ المجمع، ونماذج الكلام المخصصة، وترجمة الكلام. استخدم Speech SDK أو Batch Transcription API بدلاً من ذلك.
