SDK di Azure Resource Manager per Fabric in .NET. Utilizzare per operazioni MANAGEMENT PLANE: provisioning, dimensionamento, sospensione/ripresa delle capacità di Microsoft Fabric, verifica della disponibilità dei nomi ed elenco degli SKU tramite Azure Resource Manager. Trigger: "capacità Fabric", "crea capacità", "sospendi capacità", "riprendi capacità", "SKU Fabric", "provisioning Fabric", "ARM Fabric", "FabricCapacityResource".
