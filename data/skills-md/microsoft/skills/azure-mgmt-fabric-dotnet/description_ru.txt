Пакет SDK Azure Resource Manager для Fabric в .NET. Используйте для операций НА ПЛАНЕ УПРАВЛЕНИЯ: подготовка, масштабирование, приостановка и возобновление работы ресурсов Microsoft Fabric, проверка доступности имени и составление списка номеров SKU через Azure Resource Manager. Триггеры: «Емкость структуры», «Создать емкость», «Приостановить емкость», «Возобновить емкость», «SKU Fabric», «Provision Fabric», «ARM Fabric», «FabricCapacityResource».
