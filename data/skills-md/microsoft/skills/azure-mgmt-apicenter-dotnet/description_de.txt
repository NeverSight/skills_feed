Azure API Center SDK für .NET. Zentralisierte API-Bestandsverwaltung mit Governance, Versionierung und Erkennung. Zum Erstellen von API-Diensten, Arbeitsbereichen, APIs, Versionen, Definitionen, Umgebungen, Bereitstellungen und Metadatenschemata verwenden. Auslöser: „API Center“, „ApiCenterService“, „ApiCenterWorkspace“, „ApiCenterApi“, „API-Inventar“, „API-Governance“, „API-Versionierung“, „API-Katalog“, „API-Erkennung“.
