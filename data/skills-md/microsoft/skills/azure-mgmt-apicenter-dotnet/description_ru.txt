Пакет SDK Azure API Center для .NET. Централизованное управление запасами API с управлением, управлением версиями и обнаружением. Используйте для создания служб API, рабочих областей, API, версий, определений, сред, развертываний и схем метаданных. Триггеры: «Центр API», «ApiCenterService», «ApiCenterWorkspace», «ApiCenterApi», «Инвентаризация API», «Управление API», «Управление версиями API», «Каталог API», «Обнаружение API».
