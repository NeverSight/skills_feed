Azure API Center SDK لـ .NET. إدارة مخزون واجهة برمجة التطبيقات (API) المركزية مع الإدارة وإصدار الإصدارات والاكتشاف. يُستخدم لإنشاء خدمات واجهة برمجة التطبيقات ومساحات العمل وواجهات برمجة التطبيقات والإصدارات والتعريفات والبيئات وعمليات النشر ومخططات بيانات التعريف. المشغلات: "مركز API"، و"ApiCenterService"، و"ApiCenterWorkspace"، و"ApiCenterApi"، و"مخزون API"، و"إدارة API"، و"إصدارات API"، و"كتالوج API"، و"اكتشاف API".
