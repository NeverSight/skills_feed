Создавайте приложения голосового искусственного интеллекта в реальном времени с помощью Azure AI Voice Live SDK (azure-ai-voicelive). Используйте этот навык при создании приложений Python, которым требуется двунаправленная аудиосвязь в реальном времени с Azure AI, включая голосовых помощников, голосовых чат-ботов, перевод речи в речь в реальном времени, голосовые аватары или любую потоковую передачу звука на основе WebSocket с моделями искусственного интеллекта. Поддерживает сервер VAD (обнаружение голосовой активности), пошаговый разговор, вызов функций, инструменты MCP, интеграцию аватаров и транскрипцию.
