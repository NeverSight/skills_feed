Erstellen Sie Sprach-KI-Anwendungen in Echtzeit mit dem Azure AI Voice Live SDK (azure-ai-voicelive). Nutzen Sie diese Fähigkeit, wenn Sie Python-Anwendungen erstellen, die eine bidirektionale Audiokommunikation in Echtzeit mit Azure AI benötigen, einschließlich Sprachassistenten, sprachaktivierten Chatbots, Echtzeit-Sprachübersetzung, sprachgesteuerten Avataren oder jedem WebSocket-basierten Audiostreaming mit KI-Modellen. Unterstützt Server VAD (Voice Activity Detection), rundenbasierte Konversation, Funktionsaufrufe, MCP-Tools, Avatar-Integration und Transkription.
