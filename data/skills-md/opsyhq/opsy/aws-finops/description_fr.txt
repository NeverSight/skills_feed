Analyse d'optimisation des coûts AWS FinOps pour Opsy. À utiliser lorsque les utilisateurs demandent à analyser les coûts AWS, à détecter les gaspillages, à optimiser les dépenses, à identifier les ressources inutilisées/sous-utilisées, à examiner la couverture des instances réservées ou des plans d'économies ou à générer des rapports d'optimisation des coûts. Déclencheurs sur « réduire les coûts AWS », « rechercher les ressources inutilisées », « l'analyse des coûts », « optimiser les dépenses », « ressources inutilisées », « redimensionner », « opportunités d'économies », « pourquoi ma facture AWS est-elle élevée ».
