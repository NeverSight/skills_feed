Enseña a los alumnos a extraer lecciones de diseño transferibles de bases de código del mundo real mediante evaluación crítica y exploración sistemática. Úselo cuando un alumno quiera estudiar el código existente para aprender patrones, arquitectura o decisiones de diseño, no solo comprender lo que hace. Guía a través de la navegación, el reconocimiento de patrones, la evaluación crítica (elección deliberada versus compromiso) y la extracción de lecciones. Se activan frases como "aprender de este código base", "estudiar cómo se implementa X", "comprender los patrones de diseño en Y" o cuando un alumno quiere mejorar leyendo código real.
