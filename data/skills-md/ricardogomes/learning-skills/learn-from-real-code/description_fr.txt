Apprenez aux apprenants à extraire des leçons de conception transférables à partir de bases de code du monde réel grâce à une évaluation critique et une exploration systématique. À utiliser lorsqu'un apprenant souhaite étudier le code existant pour apprendre des modèles, une architecture ou des décisions de conception, et pas seulement comprendre ce qu'il fait. Guide à travers la navigation, la reconnaissance de formes, l'évaluation critique (choix délibéré ou compromis) et l'extraction de leçons. Se déclenche sur des expressions telles que « apprendre à partir de cette base de code », « étudier comment X est implémenté », « comprendre les modèles de conception dans Y » ou lorsqu'un apprenant souhaite s'améliorer en lisant du code réel.
