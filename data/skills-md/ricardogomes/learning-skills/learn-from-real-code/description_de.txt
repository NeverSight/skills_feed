Bringt den Lernenden bei, durch kritische Bewertung und systematische Erkundung übertragbare Design-Lektionen aus realen Codebasen zu extrahieren. Verwenden Sie es, wenn ein Lernender vorhandenen Code studieren möchte, um Muster, Architektur oder Designentscheidungen zu lernen – und nicht nur verstehen möchte, was er tut. Leitet durch Navigation, Mustererkennung, kritische Bewertung (bewusste Entscheidung vs. Kompromiss) und Lektionsextraktion. Wird bei Formulierungen wie „aus dieser Codebasis lernen“, „untersuchen, wie X implementiert wird“, „Entwurfsmuster in Y verstehen“ ausgelöst oder wenn ein Lernender sich durch das Lesen von echtem Code verbessern möchte.
