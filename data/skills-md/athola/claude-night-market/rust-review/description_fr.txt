Utilisez cette compétence pour les audits de code spécifiques à Rust. À utiliser lors de la révision du code Rust, de l'audit des blocs non sécurisés, de l'analyse des modèles de propriété, de l'analyse des dépendances Cargo pour la sécurité. Ne pas utiliser lors de la révision générale du code sans Rust - utilisez la révision unifiée. NE PAS utiliser lorsque : profilage des performances - utilisez le modèle fourchelangue : python-performance.
