Используйте этот навык для аудита кода, специфичного для Rust. Используйте при проверке кода Rust, аудите небезопасных блоков, анализе моделей владения, сканировании зависимостей Cargo на предмет безопасности. Не используйте при общем обзоре кода без Rust — используйте unified-review. НЕ используйте, когда: профилирование производительности — используйте шаблон parseltongue:python- Performance.
