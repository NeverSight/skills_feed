Анализируйте и улучшайте качество живого кода: дублирование, эффективность алгоритмов, принципы чистого кода, архитектурное соответствие, шаблоны предотвращения ошибок и надежность обработки ошибок. Используйте при улучшении качества кода, уменьшении ошибок ИИ, рефакторинге для ясности, оптимизации алгоритмов, применении принципов чистого кода. Не используйте при удалении мертвого/неиспользуемого кода (используйте conserve:bloat-detector). проверка ошибок (используйте pensive:bug-review). выбор архитектурных парадигм (использовать навыки работы с архетипами). Этот навык активно улучшает «живой» код, дополняя обнаружение раздувания (удаление мертвого кода) улучшением качества (улучшение «живого» кода).
