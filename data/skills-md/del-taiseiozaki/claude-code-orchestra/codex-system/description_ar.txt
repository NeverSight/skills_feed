يتعامل Codex CLI مع التخطيط والتصميم وتنفيذ التعليمات البرمجية المعقدة.
يستخدم لـ: التصميم المعماري، وتخطيط التنفيذ، والخوارزميات المعقدة،
تصحيح الأخطاء (تحليل السبب الجذري)، وتقييم المفاضلة، ومراجعة الكود.
البحث الخارجي ليس من مهام Codex — استخدم Gemini CLI (أساس بحث Google) بدلاً من ذلك.
المحفزات الصريحة: "التخطيط"، "التصميم"، "الهندسة المعمارية"، "التفكير بشكل أعمق"،
"تحليل"، "تصحيح"، "معقد"، "تحسين".
