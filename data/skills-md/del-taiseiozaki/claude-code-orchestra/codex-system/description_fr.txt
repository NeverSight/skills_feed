Codex CLI gère la planification, la conception et la mise en œuvre de codes complexes.
Utiliser pour : la conception d'architecture, la planification de la mise en œuvre, les algorithmes complexes,
débogage (analyse des causes profondes), évaluation des compromis, révision du code.
La recherche externe n'est PAS le travail du Codex - utilisez plutôt Gemini CLI (Google Search grounding).
Déclencheurs explicites : "plan", "conception", "architecture", "réfléchir plus profondément",
"analyser", "déboguer", "complexe", "optimiser".
