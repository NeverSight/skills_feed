Codex CLI занимается планированием, проектированием и сложной реализацией кода.
Использование для: проектирования архитектуры, планирования реализации, сложных алгоритмов,
отладка (анализ первопричин), оценка компромиссов, проверка кода.
Внешние исследования НЕ являются работой Кодекса — вместо этого используйте Gemini CLI (обоснование поиска Google).
Явные триггеры: «план», «проектирование», «архитектура», «думать глубже»,
«анализировать», «отлаживать», «комплексовать», «оптимизировать».
