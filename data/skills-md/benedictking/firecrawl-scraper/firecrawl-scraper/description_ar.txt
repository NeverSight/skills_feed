مهارة تجريف الويب باستخدام Firecrawl API لاستخراج المحتوى العميق وتحويل التنسيق والتفاعل مع الصفحة. استخدمه عندما تحتاج إلى استخراج صفحات الويب، أو استخراج البيانات المنظمة، أو التقاط لقطات شاشة، أو تحليل ملفات PDF، أو الزحف إلى مواقع الويب بأكملها. المشغلات: الزحف الناري، الكشط، استخراج المحتوى، لقطة الشاشة، تحليل pdf، الزحف إلى موقع الويب، 抓取网页، 提取内容، 网页截图
