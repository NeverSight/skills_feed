Инструмент X/Twitter CLI для чтения твитов, тредов, ответов, поиска, управления закладками и получения новостей/популярных тем. Используйте, когда пользователю нужно читать твиты, выполнять поиск по контенту X/Twitter, просматривать хронику пользователей, получать закладки или получать актуальные новости из командной строки. Поддерживает аутентификацию на основе файлов cookie из Safari или Chrome.
