X/Twitter CLI-Tool zum Lesen von Tweets, Threads, Antworten, Suchen, Verwalten von Lesezeichen und Abrufen von Nachrichten/Trendthemen. Verwenden Sie diese Option, wenn der Benutzer Tweets lesen, X/Twitter-Inhalte durchsuchen, Benutzerzeitleisten abrufen, Lesezeichen abrufen oder Trendnachrichten über die Befehlszeile abrufen muss. Unterstützt die Cookie-basierte Authentifizierung von Safari oder Chrome.
