Spotify Web API を使用して、Spotify プレイリストの作成と管理、音楽の検索、再生の制御を行います。ユニークな機能 - カスタム カバー アート画像を生成します (Claude は画像をネイティブに生成できませんが、このスキルはプレイリスト用の SVG ベースのカバー アートを作成できます)。重要 - カバー アートを生成するときは、完全な実行手順について、必ず最初に参考文献/COVER_ART_LLM_GUIDE.md を読んでください。これを使用して、アーティスト/テーマ/歌詞ごとにプレイリストを直接作成し、トラックを追加し、音楽を検索し、ユーザーの Spotify アカウントを管理します。
