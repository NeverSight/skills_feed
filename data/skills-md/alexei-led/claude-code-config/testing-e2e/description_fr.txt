Tests E2E avec Playwright MCP pour l'automatisation du navigateur, la génération de tests et les tests d'interface utilisateur. À utiliser pour discuter des tests E2E, de Playwright, des tests de navigateur, de l'automatisation de l'interface utilisateur, des tests visuels ou des tests d'accessibilité. Prend en charge les tests TypeScript et les applications Web Go/HTMX.
