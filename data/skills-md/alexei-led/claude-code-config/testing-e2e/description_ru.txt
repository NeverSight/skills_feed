E2E-тестирование с помощью Playwright MCP для автоматизации браузера, создания тестов и тестирования пользовательского интерфейса. Используйте при обсуждении тестов E2E, драматурга, тестирования браузеров, автоматизации пользовательского интерфейса, визуального тестирования или тестирования доступности. Поддерживает тесты TypeScript и веб-приложения Go/HTMX.
