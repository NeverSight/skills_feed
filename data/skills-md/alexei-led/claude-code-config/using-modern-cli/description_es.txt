Prefiera herramientas CLI modernas para un mejor rendimiento: rg (ripgrep) en lugar de grep para búsqueda de texto, fd en lugar de buscar para descubrimiento de archivos, bat en lugar de cat para ver archivos, sd en lugar de sed para reemplazo de texto, eza en lugar de ls para listado de directorios, polvo en lugar de du, procs en lugar de ps. Úselo al escribir scripts bash, optimizar cadenas de comandos, trabajar con búsquedas de archivos o reemplazar grep/find/cat en flujos de trabajo. Estas herramientas son más rápidas, respetan .gitignore y tienen un mejor formato de salida.
