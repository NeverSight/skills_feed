パフォーマンスを向上させるために最新の CLI ツールを優先します。テキスト検索には grep の代わりに rg (ripgrep)、ファイル検出には find の代わりに fd、ファイルの表示には cat の代わりに Bat、テキスト置換には sed の代わりに sd、ディレクトリのリストには ls の代わりに eza、du の代わりにdust、ps の代わりに procs を使用します。 bash スクリプトの作成、コマンド チェーンの最適化、ファイル検索の操作、またはワークフローで grep/find/cat を置き換える場合に使用します。これらのツールは高速で、.gitignore を尊重し、出力フォーマットが向上しています。
