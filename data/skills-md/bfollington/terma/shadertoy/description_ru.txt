Этот навык следует использовать при работе с шейдерами Shadertoy, фрагментными шейдерами GLSL или при создании процедурной графики для Интернета. Используйте при написании файлов .glsl, реализации визуальных эффектов, создании генеративного искусства или работе с кодом шейдера WebGL. Этот навык содержит справочник по синтаксису GLSL ES, общие шаблоны шейдеров и соглашения, специфичные для Shadertoy.
