Cette compétence doit être utilisée lorsque vous travaillez avec Strudel.cc, un environnement musical de codage en direct. À utiliser lors de la création de motifs musicaux, de séquences de batterie, de mélodies, de lignes de basse ou de compositions génératives. L'utilisateur souhaitera toujours exécuter le code Strudel dans le navigateur, soit par copier-coller, soit en fournissant une URL cliquable avec le code encodé en base64.
