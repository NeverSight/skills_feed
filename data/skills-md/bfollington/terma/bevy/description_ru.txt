Этот навык следует использовать при работе над проектами игрового движка Bevy. Он предоставляет специализированные знания по архитектуре Entity Component System (ECS) Bevy, шаблонам проектирования на основе компонентов, упорядочению системы, разработке пользовательского интерфейса, стратегиям сборки и типичным ошибкам. Используйте этот навык при реализации функций игры, отладке кода Bevy, разработке архитектуры компонентов или работе с системой пользовательского интерфейса Bevy.
