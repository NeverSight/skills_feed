Скрипты IDA Pro Python для обратного проектирования. Используйте при написании сценариев IDAPython, анализе двоичных файлов, работе с API IDA для дизассемблирования, декомпиляции (Hex-Rays), системах типов, перекрестных ссылках, функциях, сегментах или любых манипуляциях с базой данных IDA. Охватывает модули ida_* (более 50), итераторы idautils и распространенные шаблоны.
