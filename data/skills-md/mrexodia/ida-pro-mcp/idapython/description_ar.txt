IDA Pro Python البرمجة النصية للهندسة العكسية. يُستخدم عند كتابة البرامج النصية لـ IDAPython، أو تحليل الثنائيات، أو العمل مع واجهة برمجة تطبيقات IDA للتفكيك، أو إلغاء الترجمة (الأشعة السداسية)، أو أنظمة الكتابة، أو الإسنادات الترافقية، أو الوظائف، أو المقاطع، أو أي معالجة لقاعدة بيانات IDA. يغطي وحدات ida_* (50+)، ومكررات idautils، والأنماط الشائعة.
