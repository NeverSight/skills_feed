Intégration OpenProject API v3 pour la gestion de projet. Gérez les projets, les lots de travaux, les entrées de temps, les documents, les utilisateurs, les notifications, les requêtes. À utiliser lorsque l'utilisateur doit interagir avec l'instance OpenProject - créer/mettre à jour des packages de travail, suivre le temps, gérer des projets, interroger des données, gérer des documents et des pièces jointes.
