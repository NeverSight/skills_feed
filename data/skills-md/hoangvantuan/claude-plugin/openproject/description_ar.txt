تكامل OpenProject API v3 لإدارة المشاريع. إدارة المشاريع وحزم العمل وإدخالات الوقت والمستندات والمستخدمين والإشعارات والاستعلامات. يُستخدم عندما يحتاج المستخدم إلى التفاعل مع مثيل OpenProject - إنشاء/تحديث حزم العمل، وتتبع الوقت، وإدارة المشاريع، والاستعلام عن البيانات، والتعامل مع المستندات والمرفقات.
