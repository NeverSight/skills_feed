Интеграция OpenProject API v3 для управления проектами. Управляйте проектами, рабочими пакетами, записями времени, документами, пользователями, уведомлениями, запросами. Используйте, когда пользователю необходимо взаимодействовать с экземпляром OpenProject — создавать/обновлять рабочие пакеты, отслеживать время, управлять проектами, запрашивать данные, обрабатывать документы и вложения.
