Integración de OpenProject API v3 para la gestión de proyectos. Gestionar proyectos, paquetes de trabajo, entradas de tiempos, documentos, usuarios, notificaciones, consultas. Úselo cuando el usuario necesite interactuar con la instancia de OpenProject: crear/actualizar paquetes de trabajo, realizar un seguimiento del tiempo, administrar proyectos, consultar datos, manejar documentos y archivos adjuntos.
