Abilità di meta-suggerimento che crea suggerimenti ben strutturati, verificabili e con poche allucinazioni per qualsiasi caso d'uso. Da utilizzare quando l'utente desidera creare, perfezionare o migliorare un prompt, inclusi prompt di sistema, prompt di ruolo, prompt di attività o qualsiasi set di istruzioni AI. Si attiva su richieste come "crea un prompt per...", "aiutami a scrivere un prompt", "perfeziona questo prompt", "crea un prompt migliore per..." o "genera un prompt che...".
