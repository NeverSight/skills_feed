Meta-Prompting-Fähigkeit, die gut strukturierte, überprüfbare und halluzinationsarme Eingabeaufforderungen für jeden Anwendungsfall erstellt. Verwenden Sie diese Option, wenn der Benutzer eine Eingabeaufforderung erstellen, verfeinern oder verbessern möchte – einschließlich Systemeingabeaufforderungen, Rolleneingabeaufforderungen, Aufgabeneingabeaufforderungen oder eines beliebigen KI-Anweisungssatzes. Wird bei Anfragen wie „Erstellen Sie eine Eingabeaufforderung für …“, „Hilf mir, eine Eingabeaufforderung zu schreiben“, „Verfeinern Sie diese Eingabeaufforderung“, „Erstellen Sie eine bessere Eingabeaufforderung für …“ oder „Generieren Sie eine Eingabeaufforderung, die ...“ ausgelöst.
