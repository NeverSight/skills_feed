Habilidad de metaincitaciones que crea indicaciones bien estructuradas, verificables y con bajas alucinaciones para cualquier caso de uso. Úselo cuando el usuario quiera crear, refinar o mejorar un mensaje, incluidos mensajes del sistema, mensajes de roles, mensajes de tareas o cualquier conjunto de instrucciones de IA. Activadores de solicitudes como "crear un mensaje para...", "ayúdame a escribir un mensaje", "refinar este mensaje", "crear un mensaje mejor para..." o "generar un mensaje que...".
