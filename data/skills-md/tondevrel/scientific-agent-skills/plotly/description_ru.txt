Высокоуровневая интерактивная графическая библиотека для Python. Идеально подходит для веб-визуализации, 3D-графиков и сложных интерактивных информационных панелей. Созданный на основеplotly.js, он позволяет пользователям масштабировать, панорамировать и наводить курсор на точки данных в среде браузера. Используйте для интерактивных диаграмм, веб-приложений, блокнотов Jupyter, 3D-визуализации данных, географических карт, финансовых диаграмм, анимации, анализа временных рядов и создания готовых к использованию информационных панелей с помощью Dash.
