Проект с открытым исходным кодом, упрощающий работу с геопространственными данными в Python. Расширяет типы данных, используемые пандами, позволяя выполнять пространственные операции с геометрическими типами. Построен на основе Shapely, Fiona и Pyproj. Используйте для чтения и записи пространственных форматов (Shapefile, GeoJSON, GeoPackage, KML), выполнения пространственных соединений, преобразований системы координат (перепроецирования), геометрического анализа (буферы, центроиды, выпуклые оболочки), тематического картографирования (картограммы), расчета пространственных отношений (содержит, перекрывается, касается, внутри), работы с данными OpenStreetMap или векторными данными, полученными со спутников.
