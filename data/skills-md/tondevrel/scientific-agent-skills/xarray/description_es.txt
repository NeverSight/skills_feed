Conjuntos de datos y matrices etiquetados de N dimensiones en Python. Construido sobre NumPy y Dask. Introduce etiquetas en forma de dimensiones, coordenadas y atributos sobre matrices sin formato tipo NumPy, lo que hace que el análisis de datos en ciencias físicas sea más intuitivo y menos propenso a errores. Úselo para trabajar con datos científicos multidimensionales, archivos NetCDF/GRIB/Zarr, conjuntos de datos climáticos/meteorológicos/oceanográficos, sensores remotos, imágenes geoespaciales, grandes conjuntos de datos sin memoria con Dask y operaciones de matrices etiquetadas.
