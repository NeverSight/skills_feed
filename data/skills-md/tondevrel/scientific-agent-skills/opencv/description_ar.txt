مكتبة الرؤية الحاسوبية مفتوحة المصدر (OpenCV) لمعالجة الصور في الوقت الفعلي، وتحليل الفيديو، واكتشاف الكائنات، والتعرف على الوجوه، ومعايرة الكاميرا. يُستخدم عند العمل مع الصور أو مقاطع الفيديو أو الكاميرات أو اكتشاف الحواف أو الخطوط الكنتورية أو اكتشاف الميزات أو تحويلات الصور أو تتبع الكائنات أو التدفق البصري أو أي مهمة رؤية للكمبيوتر.
