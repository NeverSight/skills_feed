Crea o migliora i file AGENTS.md per gli agenti di codifica AI. Da utilizzare quando l'utente richiede di creare, migliorare, rivedere o ottimizzare un file AGENTS.md (o CLAUDE.md). Da utilizzare anche quando si esegue /init su un nuovo progetto, quando vengono richieste informazioni sulle migliori pratiche per i file di configurazione dell'agente o durante la risoluzione dei problemi relativi al comportamento dell'agente relativo al contesto del progetto.
