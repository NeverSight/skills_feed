Aplique el desarrollo basado en pruebas con un estricto ciclo Red-Green-Refactor utilizando pruebas de integración. Se activa automáticamente al implementar nuevas características o funcionalidades. Las frases desencadenantes incluyen "implementar", "agregar función", "construir", "crear funcionalidad" o cualquier solicitud para agregar un nuevo comportamiento. NO activa correcciones de errores, documentación o cambios de configuración.
