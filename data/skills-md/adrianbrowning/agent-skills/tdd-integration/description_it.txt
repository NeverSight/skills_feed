Applica lo sviluppo basato sui test con un rigoroso ciclo Red-Green-Refactor utilizzando test di integrazione. Si attiva automaticamente quando si implementano nuove caratteristiche o funzionalità. Le frasi trigger includono "implement", "aggiungi funzionalità", "crea", "crea funzionalità" o qualsiasi richiesta di aggiungere un nuovo comportamento. NON si attiva per correzioni di bug, documentazione o modifiche alla configurazione.
