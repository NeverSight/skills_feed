Комплексные навыки проверки кода для TypeScript, JavaScript, Python, Swift, Kotlin, Go. Включает автоматический анализ кода, проверку передового опыта, сканирование безопасности и создание контрольного списка проверки. Используйте при проверке запросов на включение, предоставлении обратной связи по коду, выявлении проблем или обеспечении стандартов качества кода.
