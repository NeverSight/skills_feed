Extrayez du texte, des tableaux, des métadonnées et des images à partir de plus de 75 formats de documents (PDF, Office, images, HTML, courrier électronique, archives, universitaires) à l'aide de Kreuzberg. À utiliser lors de l'écriture de code qui appelle les API Kreuzberg en Python, Node.js/TypeScript, Rust ou CLI. Couvre l'installation, l'extraction (sync/async), la configuration (OCR, chunking, format de sortie), le traitement par lots, la gestion des erreurs et les plugins.
