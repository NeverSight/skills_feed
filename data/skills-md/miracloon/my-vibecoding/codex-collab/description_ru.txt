【КОГДА】Активируется, когда команда пользователя явно включает такие фразы, как «сотрудничество Кодекса», «синергия Кодекса» или «сотрудничать с Кодексом»; 【КОГДА】Активируется, когда пользователь запрашивает использование инструментов Codex MCP для анализа кода, создания прототипов, проверки или других сценариев совместной разработки. 【КОГДА НЕТ】Не используется для задач реализации автономного кода, для написания чистой документации, а также для обычных рабочих процессов разработки, не связанных с кодексом.
