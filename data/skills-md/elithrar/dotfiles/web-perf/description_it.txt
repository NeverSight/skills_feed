Analizza le prestazioni web utilizzando Chrome DevTools MCP. Misura i Core Web Vitals (FCP, LCP, TBT, CLS, Speed ​​Index), identifica le risorse che bloccano il rendering, le catene di dipendenze di rete, i cambiamenti di layout, i problemi di memorizzazione nella cache e le lacune di accessibilità. Da utilizzare quando viene richiesto di controllare, profilare, eseguire il debug o ottimizzare le prestazioni di caricamento della pagina, i punteggi Lighthouse o la velocità del sito.
