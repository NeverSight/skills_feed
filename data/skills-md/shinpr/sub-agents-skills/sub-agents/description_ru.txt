Выполняйте внешние искусственные интеллекты CLI в качестве изолированных субагентов для делегирования задач, параллельной обработки и разделения контекста. Используйте при делегировании сложных многоэтапных задач, проведении параллельных расследований, необходимости свежего контекста без текущей истории разговоров или использовании специализированных определений агентов. Возвращает структурированный JSON с выводом агента, кодом завершения и статусом выполнения.
