Ejecute CLI AI externas como subagentes aislados para la delegación de tareas, el procesamiento paralelo y la separación de contextos. Úselo al delegar tareas complejas de varios pasos, ejecutar investigaciones paralelas, necesitar un contexto nuevo sin un historial de conversaciones actual o aprovechar definiciones de agentes especializados. Devuelve JSON estructurado con salida del agente, código de salida y estado de ejecución.
