Рекомендации по оптимизации производительности Vue 3 и Nuxt. Этот навык следует использовать при написании, проверке или рефакторинге кода Vue 3/Nuxt, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, связанных с компонентами Vue, страницами Nuxt, получением данных, оптимизацией пакетов или улучшением производительности.
