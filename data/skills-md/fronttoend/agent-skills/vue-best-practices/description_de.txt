Richtlinien zur Leistungsoptimierung von Vue 3 und Nuxt. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von Vue 3/Nuxt-Code verwendet werden, um optimale Leistungsmuster sicherzustellen. Wird bei Aufgaben ausgelöst, die Vue-Komponenten, Nuxt-Seiten, Datenabruf, Bundle-Optimierung oder Leistungsverbesserungen betreffen.
