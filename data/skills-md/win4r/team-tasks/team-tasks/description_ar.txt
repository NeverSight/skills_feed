تنسيق مسارات التطوير متعددة الوكلاء باستخدام ملفات مهام JSON المشتركة. يُستخدم عند إرسال العمل عبر وكلاء فريق التطوير (وكيل التعليمات البرمجية، أو وكيل الاختبار، أو وكيل المستندات، أو برنامج المراقبة الآلي)، أو تتبع تقدم المسار، أو تشغيل مسارات عمل تسلسلية/متوازية. يغطي بدء المشروع وتعيين المهام وتتبع الحالة وإرسال الوكيل عبر session_send وجمع النتائج. يدعم وضعين: خطي (خط أنابيب متسلسل) وداغ (رسم بياني للتبعية مع تنفيذ متوازي).
