دليل شامل لتثبيت OpenClaw وتكوينه وتشغيله واستكشاف الأخطاء وإصلاحها - وهي عبارة عن بوابة وكيل AI متعددة القنوات ومستضافة ذاتيًا. يُستخدم عندما يسأل المستخدم عن إعداد OpenClaw، أو التكوين، أو إدارة القناة (WhatsApp/Telegram/Discord/Slack/iMessage/إلخ)، أو إعداد موفر النموذج، أو عمليات البوابة، أو التوجيه متعدد الوكلاء، أو تعزيز الأمان، أو استكشاف الأخطاء وإصلاحها، أو أي مهمة صيانة تتعلق بتثبيت OpenClaw المحلي. استخدمه أيضًا عند مواجهة أخطاء من أوامر `openclaw` CLI أو برنامج Gateway الخفي.
