Applique les normes de documentation JSDoc pour ce projet TypeScript. Cette compétence doit être utilisée lors de l'écriture ou de la révision de code TypeScript pour garantir une documentation appropriée avec les préambules de fichiers, les documents de fonction, les documents d'interface et la distinction critique entre documenter « quoi » et « pourquoi ». Utilisez cette compétence pour comprendre les règles JSDoc ESLint et les modèles établis du projet.
