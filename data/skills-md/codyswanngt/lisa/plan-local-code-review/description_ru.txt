Этот навык следует использовать при проверке кода на предмет локальных изменений в текущей ветке по сравнению с основной веткой. Он использует несколько параллельных агентов для проверки ошибок, соответствия CLAUDE.md, контекста истории git, предыдущих комментариев к PR и соблюдения комментариев к коду, а затем оценивает и фильтрует результаты по уровню достоверности.
