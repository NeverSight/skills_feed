API-Antwort-Wrapper-Muster für konsistente, vorhersehbare REST-APIs im ABP Framework. Wird verwendet, wenn: (1) einheitliche API-Antwortverträge entworfen werden, (2) Erfolgs-/Fehlerantwort-Wrapper implementiert werden, (3) Paginierung und Metadaten verarbeitet werden, (4) Fehlerantworten standardisiert werden.
