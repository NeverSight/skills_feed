Modèles de wrapper de réponse API pour des API REST cohérentes et prévisibles dans ABP Framework. À utiliser lorsque : (1) la conception de contrats de réponse API uniformes, (2) la mise en œuvre de wrappers de réponse de réussite/erreur, (3) la gestion de la pagination et des métadonnées, (4) la standardisation des réponses d'erreur.
