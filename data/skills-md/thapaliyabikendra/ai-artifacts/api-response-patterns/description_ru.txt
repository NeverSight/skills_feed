Шаблоны оболочек ответов API для согласованных и предсказуемых API REST в ABP Framework. Используйте при: (1) разработке унифицированных контрактов ответов API, (2) реализации оболочек ответов об успехе/ошибках, (3) обработке разбиения на страницы и метаданных, (4) стандартизации ответов об ошибках.
