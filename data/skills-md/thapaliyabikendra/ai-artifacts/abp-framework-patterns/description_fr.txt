Maîtrisez les modèles de framework ABP, notamment le modèle de référentiel, l'unité de travail, les services de domaine, les services d'application, l'autorisation, la multilocation, les tâches en arrière-plan et les événements distribués. À utiliser lorsque : (1) la création d'applications basées sur ABP avec une architecture DDD, (2) la création de services CRUD avec Entity, AppService, DTO, validateurs, (3) la gestion des autorisations/permissions, (4) la génération du code du module ABP.
