Modèles de test d'intégration pour les API ABP Framework à l'aide de xUnit et WebApplicationFactory. À utiliser lorsque : (1) tester les points de terminaison de l'API de bout en bout, (2) vérifier les codes d'état et les réponses HTTP, (3) tester l'autorisation, (4) tester l'intégration de la base de données.
