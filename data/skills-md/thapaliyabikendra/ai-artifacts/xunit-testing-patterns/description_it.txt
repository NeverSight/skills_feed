Padroneggia i modelli di test xUnit per le applicazioni ABP Framework, inclusi test unitari, test di integrazione, seeders di dati di test e strategie di mocking. Da utilizzare quando: (1) si scrivono test xUnit per i servizi ABP, (2) si creano seeders di dati di test, (3) si implementano test di integrazione, (4) si configura l'infrastruttura di test.
