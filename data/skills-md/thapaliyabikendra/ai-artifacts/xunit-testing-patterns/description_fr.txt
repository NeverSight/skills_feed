Maîtrisez les modèles de test xUnit pour les applications ABP Framework, y compris les tests unitaires, les tests d'intégration, les semoirs de données de test et les stratégies de simulation. À utiliser lorsque : (1) l'écriture de tests xUnit pour les services ABP, (2) la création de semoirs de données de test, (3) la mise en œuvre de tests d'intégration, (4) la configuration d'une infrastructure de test.
