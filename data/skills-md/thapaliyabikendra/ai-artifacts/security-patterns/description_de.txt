Sicherheitsmuster für Webanwendungen, einschließlich STRIDE-Bedrohungsmodellierung, OWASP Top 10-Compliance, ABP-Autorisierung und Sicherheitsüberprüfungsverfahren. Wird verwendet, wenn: (1) Sicherheitsüberprüfungen durchgeführt werden, (2) Authentifizierung/Autorisierung implementiert wird, (3) Bedrohungsmodelle erstellt werden, (4) Code auf Schwachstellen überprüft wird.
