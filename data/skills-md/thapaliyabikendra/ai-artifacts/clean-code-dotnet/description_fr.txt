Principes de Clean Code adaptés pour C#/.NET, y compris la dénomination, les variables, les fonctions, SOLID, la gestion des erreurs et les modèles asynchrones. À utiliser lorsque : (1) la révision du code C#, (2) la refactorisation pour plus de clarté, (3) l'écriture d'un nouveau code, (4) les commentaires sur la révision du code.
