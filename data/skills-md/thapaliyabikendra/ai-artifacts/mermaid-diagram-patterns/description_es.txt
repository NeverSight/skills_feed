Cree diagramas de sirena para documentación técnica, incluidos ERD, diagramas de secuencia, diagramas de flujo y diagramas de arquitectura. Úselo cuando: (1) diseñe esquemas de bases de datos (ERD), (2) documente interacciones API (secuencia), (3) ilustre flujos de procesos (diagrama de flujo), (4) visualice la arquitectura del sistema.
