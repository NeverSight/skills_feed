Экспертное руководство по написанию понятных, простых и эффективных тестов модулей, интеграции, компонентов, микросервисов и API. Используйте этот навык при проверке существующих тестов на наличие нарушений, написании новых тестов или рефакторинге тестов. НЕ для сквозных тестов, охватывающих несколько процессов — вместо этого используйте навыки тестирования e2e. Охватывает шаблон AAA, фабрики данных, стратегии макетирования, тестирование DOM, тестирование баз данных и лучшие практики утверждения.
