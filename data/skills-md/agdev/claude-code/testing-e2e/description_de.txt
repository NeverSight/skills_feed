Fachkundige Anleitung zum Schreiben belastbarer End-to-End-Tests, die mehrere Prozesse und Komponenten umfassen. Nutzen Sie diese Fähigkeit, wenn Sie systemweite E2E-Tests überprüfen, schreiben oder umgestalten. Behandelt benutzerorientierte Szenarien, Hilfsfunktionen, Datenfabriken, ARIA-basierte Selektoren und automatisch wiederholbare Behauptungen. NICHT für Unit-/Integrationstests – verwenden Sie stattdessen die Fähigkeit „testing-unit-integration“. Ausführliche Auswahlmuster finden Sie unter references/playwright-resilience.md.
