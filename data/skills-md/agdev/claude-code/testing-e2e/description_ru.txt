Экспертное руководство по написанию отказоустойчивых комплексных тестов, охватывающих несколько процессов и компонентов. Используйте этот навык при просмотре, написании или рефакторинге общесистемных тестов e2e. Охватывает сценарии взаимодействия с пользователем, вспомогательные функции, фабрики данных, селекторы на основе ARIA и автоматически повторяемые утверждения. НЕ для модульных/интеграционных тестов — вместо этого используйте навыки модульной интеграции тестирования. См. ссылки/playwright-resilience.md для получения подробных шаблонов селектора.
