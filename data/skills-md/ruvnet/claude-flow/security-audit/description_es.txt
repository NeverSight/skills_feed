Escaneo de seguridad integral y detección de vulnerabilidades. Incluye validación de entradas, prevención de cruce de rutas, detección de CVE y aplicación de patrones de codificación segura. Úselo cuando: implementación de autenticación, lógica de autorización, procesamiento de pagos, manejo de datos de usuario, creación de puntos finales de API, manejo de carga de archivos, consultas de bases de datos, integración de API externa. Omitir cuando: operaciones de solo lectura en datos públicos, herramientas de desarrollo interno, documentación estática, cambios de estilo.
