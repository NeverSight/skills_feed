Комплексное сканирование безопасности и обнаружение уязвимостей. Включает проверку входных данных, предотвращение обхода путей, обнаружение CVE и применение шаблонов безопасного кодирования. Используйте, когда: реализация аутентификации, логика авторизации, обработка платежей, обработка пользовательских данных, создание конечной точки API, обработка загрузки файлов, запросы к базе данных, интеграция внешнего API. Пропускать, когда: операции только для чтения с общедоступными данными, внутренние инструменты разработки, статическая документация, изменения стиля.
