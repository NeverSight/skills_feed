Automazione del flusso di lavoro GitHub, gestione delle PR, monitoraggio dei problemi e coordinamento della revisione del codice. Si integra con GitHub Actions e la gestione del repository. Utilizzare quando: creazione di PR, revisione del codice, gestione dei problemi, automazione del rilascio, impostazione del flusso di lavoro. Salta quando: modifiche solo locali, repository non GitHub.
