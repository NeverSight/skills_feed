Automatización del flujo de trabajo de GitHub, gestión de relaciones públicas, seguimiento de problemas y coordinación de revisión de código. Se integra con GitHub Actions y la gestión de repositorios. Úselo cuando: creación de relaciones públicas, revisión de código, gestión de problemas, automatización de lanzamientos, configuración del flujo de trabajo. Omitir cuando: cambios solo locales, repositorios que no sean de GitHub.
