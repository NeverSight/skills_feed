Система памяти AgentDB с векторным поиском HNSW. Обеспечивает в 150–12 500 раз более быстрый поиск шаблонов, постоянное хранение и возможности семантического поиска для обучения и управления знаниями. Используйте, когда: необходимо хранить успешные шаблоны, искать похожие решения, семантический поиск прошлой работы, учиться на предыдущих задачах, обмениваться знаниями между агентами, создавать базу знаний. Пропустить, когда: обучение не требуется, эфемерные разовые задачи, доступны внешние источники данных, исследование доступно только для чтения.
