Flujo de trabajo integral para administrar los comentarios de revisión de relaciones públicas de GitHub utilizando gh CLI y GraphQL API. Úselo cuando se le solicite abordar comentarios de revisión, buscar comentarios no respondidos, responder a hilos de revisión o resolver o cancelar conversaciones de revisión. Admite encontrar TODOS los comentarios a través de los límites de la paginación, responder a hilos y resolver conversaciones.
