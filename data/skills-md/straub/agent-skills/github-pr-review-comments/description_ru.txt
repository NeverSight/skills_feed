Комплексный рабочий процесс для управления комментариями к обзору PR на GitHub с использованием gh CLI и GraphQL API. Используйте, когда вас просят ответить на комментарии к отзывам, найти неотвеченные комментарии, ответить на ветки отзывов или разрешить/отменить разрешение разговоров об отзывах. Поддерживает поиск ВСЕХ комментариев за пределами нумерации страниц, ответы на темы и разрешение разговоров.
