Flusso di lavoro completo per la gestione dei commenti di revisione PR di GitHub utilizzando gh CLI e API GraphQL. Da utilizzare quando viene richiesto di rispondere ai commenti di revisione, trovare commenti senza risposta, rispondere a thread di revisione o risolvere/non risolvere conversazioni di revisione. Supporta la ricerca di TUTTI i commenti oltre i limiti dell'impaginazione, la risposta ai thread e la risoluzione delle conversazioni.
