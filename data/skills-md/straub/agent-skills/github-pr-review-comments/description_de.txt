Umfassender Workflow für die Verwaltung von GitHub PR-Bewertungskommentaren mithilfe der gh CLI und der GraphQL-API. Verwenden Sie diese Option, wenn Sie dazu aufgefordert werden, auf Bewertungskommentare einzugehen, unbeantwortete Kommentare zu finden, auf Bewertungsthreads zu antworten oder Bewertungsgespräche zu lösen bzw. aufzulösen. Unterstützt das Finden ALLER Kommentare über Seitengrenzen hinweg, das Beantworten von Threads und das Auflösen von Konversationen.
