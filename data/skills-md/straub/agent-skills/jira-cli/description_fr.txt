Interagissez avec Jira à l'aide de l'outil de ligne de commande jira-cli pour gérer les tickets, les sprints, les epics et les tableaux. À utiliser lorsque vous travaillez avec des tickets Jira, la planification de sprints, le suivi des problèmes ou lorsque l'utilisateur mentionne Jira, les tickets, les sprints, les épopées ou la gestion des problèmes.
