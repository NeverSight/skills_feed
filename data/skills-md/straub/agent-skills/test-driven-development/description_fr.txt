Appliquez le développement piloté par les tests (TDD) comme flux de travail par défaut pour les modifications de code. Utilisez cette compétence pour la mise en œuvre de fonctionnalités, les corrections de bogues, la refactorisation et toute modification nécessitant une vérification de l'exactitude. Guide les agents à travers les cycles Rouge-Vert-Refactor avec une discipline appropriée axée sur les tests. Se déclenche sur des requêtes telles que « implémenter une fonctionnalité », « corriger un bug », « refactoriser le code » ou toute tâche mentionnant des tests, l'exactitude ou la vérification du comportement.
