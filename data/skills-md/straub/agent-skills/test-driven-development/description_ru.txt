Включите разработку через тестирование (TDD) в качестве рабочего процесса по умолчанию для изменений кода. Используйте этот навык для реализации функций, исправления ошибок, рефакторинга и любых изменений, требующих проверки правильности. Проводит агентов через циклы «красный-зеленый-рефакторинг», соблюдая надлежащую дисциплину «сначала тестирование». Запускается по таким запросам, как «внедрить функцию», «исправить ошибку», «рефакторинг кода» или любой задаче, в которой упоминаются тесты, правильность или проверка поведения.
