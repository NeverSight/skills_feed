Pflegen Sie JavaScript/Node.js-Pakete durch Sicherheitsüberprüfungen oder Abhängigkeitsaktualisierungen in einem isolierten Git-Arbeitsbaum. Unterstützt npm, Yarn, pnpm und bun. Verwendung für: (1) Sicherheitsanfragen – Prüfung, CVE, Schwachstellen, Behebung von Sicherheitsproblemen, Prüfung auf anfällige Abhängigkeiten; (2) Update-Anfragen – Abhängigkeiten aktualisieren, Pakete aktualisieren, neueste Versionen abrufen, Abhängigkeiten modernisieren.
