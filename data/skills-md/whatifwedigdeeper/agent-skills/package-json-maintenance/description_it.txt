Mantieni i pacchetti JavaScript/Node.js tramite controlli di sicurezza o aggiornamenti delle dipendenze in un albero di lavoro git isolato. Supporta npm, filato, pnpm e panino. Utilizzare per: (1) Richieste di sicurezza: controllo, CVE, vulnerabilità, risoluzione di problemi di sicurezza, verifica delle dipendenze vulnerabili; (2) Richieste di aggiornamento: aggiorna le dipendenze, aggiorna i pacchetti, ottieni le versioni più recenti, modernizza le dipendenze.
