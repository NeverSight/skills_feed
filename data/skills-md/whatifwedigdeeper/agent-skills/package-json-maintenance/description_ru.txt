Поддерживайте пакеты JavaScript/Node.js посредством аудита безопасности или обновлений зависимостей в изолированном рабочем дереве git. Поддерживает npm, Yarn, pnpm и Bun. Используется для: (1) Запросов безопасности — аудит, CVE, уязвимости, исправление проблем безопасности, проверка уязвимых зависимостей; (2) Запросы на обновление — обновить зависимости, обновить пакеты, получить последние версии, модернизировать зависимости.
