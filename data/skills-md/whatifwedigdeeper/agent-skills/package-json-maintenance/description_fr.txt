Maintenez les packages JavaScript/Node.js via des audits de sécurité ou des mises à jour de dépendances dans un arbre de travail git isolé. Prend en charge npm, fil, pnpm et chignon. À utiliser pour : (1) Demandes de sécurité - audit, CVE, vulnérabilités, résolution de problèmes de sécurité, vérification des dépendances vulnérables ; (2) Demandes de mise à jour : mettre à jour les dépendances, mettre à niveau les packages, obtenir les dernières versions, moderniser les dépendances.
