Erstellen Sie einen Branch, schreiben Sie ihn fest, pushen Sie ihn und öffnen Sie einen Pull-Request. Wird verwendet, wenn: Der Benutzer sagt „Versenden“, „Versenden“, „Erstellen Sie eine PR“, „Öffnen Sie eine PR“, „Push und PR“ oder möchte von nicht festgeschriebenen Änderungen zu einer offenen Pull-Anfrage wechseln.
