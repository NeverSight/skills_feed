Maintenez les packages JavaScript/Node.js via des audits de sécurité ou des mises à jour de dépendances sur une branche dédiée. Prend en charge npm, fil, pnpm et chignon. À utiliser pour : les audits de sécurité, les correctifs CVE, les vérifications de vulnérabilités, les mises à jour de dépendances, les mises à niveau de packages, les packages obsolètes, les versions de remplacement, la correction des vulnérabilités npm, la modernisation des node_modules ou lorsque l'utilisateur tape "/js-deps" avec ou sans noms de package spécifiques ou modèles globaux. Utilisez "help" ou "--help" pour afficher les options.
