Возможность веб-поиска с использованием API DuckDuckGo по умолчанию и резервным API Tavily, если предоставляется TRAVILY_TOKEN. Используйте, когда вам нужно найти в Интернете актуальную информацию, темы исследования, найти URL-адреса или собрать данные из онлайн-источников. Поддерживает запросы новостей, технической информации, исследовательских работ, данных компании и общего веб-контента.
