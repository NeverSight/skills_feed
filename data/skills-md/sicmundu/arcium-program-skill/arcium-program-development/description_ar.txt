قم ببناء كود برنامج Arcium MXE وإعادة تصميمه وتصحيح أخطائه عبر دوائر Arcis (`encrypted-ixs`) وبرامج Anchor (`programs/*`). يُستخدم عند تنفيذ تعليمات سرية، أو توصيل `init_comp_def`/`queue_computation`/callbacks، أو اشتقاق أجهزة المساعد الرقمي الشخصي وحسابات رد الاتصال من Arcium، أو التعامل مع الإدخال/الإخراج المشفر والإزاحات، أو تحليل بنيات إخراج رد الاتصال التي تم إنشاؤها، أو إنشاء تدفقات ذات بوابات إذن، أو دمج مصادر الدوائر خارج السلسلة، أو الاختبار باستخدام `@arcium-hq/client`، أو نشر MXEs، أو ترحيل إصدارات Arcium.
