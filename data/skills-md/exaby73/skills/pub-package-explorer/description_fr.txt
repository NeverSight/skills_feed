Recherchez et lisez le code source des dépendances du package Dart ou Flutter installées dans un projet. À utiliser lorsqu'on vous demande d'inspecter le code d'un package sur pub.dev, de suivre les détails d'implémentation dans les dépendances ou de localiser les fichiers de package à partir des noms de package en résolvant `.dart_tool/package_config.json`.
