Suchen und lesen Sie den Quellcode für in einem Projekt installierte Dart- oder Flutter-Paketabhängigkeiten. Verwenden Sie diese Option, wenn Sie aufgefordert werden, Code aus einem Paket auf pub.dev zu überprüfen, Implementierungsdetails in Abhängigkeiten zu verfolgen oder Paketdateien anhand von Paketnamen zu finden, indem Sie „.dart_tool/package_config.json“ auflösen.
