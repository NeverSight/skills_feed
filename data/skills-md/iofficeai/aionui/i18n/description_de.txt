Internationalisierungs-Workflow (i18n) und Standards für die Verwaltung von Übersetzungen.
Wird verwendet, wenn: (1) neuer benutzerorientierter Text hinzugefügt wird, (2) neue Komponenten mit Text erstellt werden, (3) Code auf i18n-Konformität überprüft wird.
Funktionen: Schlüsselbenennungskonventionen, Synchronisierungsprüfung, Erkennung hartcodierter Zeichenfolgen, Übersetzungsworkflow.
