Bewährte Sicherheitspraktiken für konvexe Funktionen, einschließlich ConvexError-Behandlung, Argument-/Rückgabevalidierung, Authentifizierungshilfen, Zugriffskontrolle, Ratenbegrenzung und interne Funktionen. Wird zum Schreiben öffentlicher Abfragen/Mutationen/Aktionen, zum Implementieren der Authentifizierung, zum Hinzufügen von Autorisierungsprüfungen, zum Behandeln von Fehlern oder zum Überprüfen konvexer Funktionen auf Sicherheit verwendet.
