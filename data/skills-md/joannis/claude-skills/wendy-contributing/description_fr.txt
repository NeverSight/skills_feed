Conseils d'experts pour contribuer à WendyOS : builds Yocto, composants internes des agents, tests E2E et architecture système. À utiliser lorsque les développeurs mentionnent : (1) la construction d'images WendyOS, (2) les couches méta-wendyos ou bitbake, (3) le développement ou les composants internes de wendy-agent, (4) containersd ou nerdctl sur WendyOS, (5) les tests E2E pour wendy-agent, (6) les recettes Yocto ou les fichiers bbappend, (7) la configuration du service mDNS/Avahi, (8) l'identité de l'appareil ou la génération d'UUID.
