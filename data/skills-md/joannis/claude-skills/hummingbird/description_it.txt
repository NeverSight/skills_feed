Guida di esperti sul framework web Hummingbird 2. Da utilizzare quando gli sviluppatori menzionano: (1) Hummingbird, HB o Hummingbird 2, (2) server Web Swift o server HTTP, (3) routing o middleware Swift lato server, (4) creazione di API REST in Swift, (5) RequestContext o ChildRequestContext, (6) HummingbirdAuth o middleware di autenticazione, (7) HummingbirdWebSocket, (8) HummingbirdFluent o integrazione del database, (9) ResponseGenerator o Risposta modificata.
