Conseils d'experts sur la création de bibliothèques clientes de bases de données Swift. À utiliser lorsque les développeurs mentionnent : (1) la création d'un pilote de base de données, (2) la mise en œuvre d'un protocole filaire, (3) la conception de regroupement de connexions, (4) les machines à états pour la gestion du protocole, (5) les gestionnaires de canaux NIO pour les bases de données, (6) la contre-pression dans la diffusion des résultats, (7) l'alignement de l'exécuteur des acteurs avec NIO.
