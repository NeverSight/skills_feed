Экспертное руководство по созданию клиентских библиотек баз данных Swift. Используйте, когда разработчики упоминают: (1) создание драйвера базы данных, (2) реализацию проводного протокола, (3) проектирование пула соединений, (4) конечные автоматы для обработки протоколов, (5) обработчики каналов NIO для баз данных, (6) противодавление при потоковой передаче результатов, (7) согласование исполнителя актера с NIO.
