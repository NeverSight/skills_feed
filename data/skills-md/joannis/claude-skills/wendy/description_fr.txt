Conseils d'experts sur la création et le déploiement d'applications sur les appareils Edge WendyOS. À utiliser lorsque les développeurs mentionnent : (1) Wendy ou WendyOS, (2) les commandes Wendy CLI, (3) wendy.json ou droits, (4) le déploiement d'applications sur des appareils périphériques, (5) le débogage à distance Swift sur ARM64, (6) les applications NVIDIA Jetson ou Raspberry Pi, (7) la compilation croisée Swift pour ARM64.
