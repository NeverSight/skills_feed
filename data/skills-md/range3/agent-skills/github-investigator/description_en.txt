The ability to investigate discussions on Issues, PRs, and code related to a specific topic on GitHub and create a summary report. Uses the gh command. Employed when a user asks for investigation of discussions in a GitHub repository, cross-repository analysis of Issues/PRs, understanding the history behind a specific feature/bug/component, or tracking decisions made on GitHub. Handles requests like “What discussions exist about ○○ in this repository?”, “Look into the history of ○○”, “Compile related Issues and PRs”, or “Why was ○○ designed this way?”. Use this skill even if “GitHub” isn’t explicitly mentioned, as long as the task involves investigating OSS feature discussions or design history.
