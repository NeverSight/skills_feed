Analice si los usuarios finales descubrirán un valor claro en las ideas de productos. Utilícelo cuando: se discutan conceptos de productos, se evalúen características, se planifiquen estrategias de marketing, se analicen los problemas de adopción de los usuarios o cuando el usuario exprese incertidumbre sobre la dirección del producto (por ejemplo, "¿es buena esta idea?", "¿qué opinas de esto?", "¿querrán esto los usuarios?", "¿por qué no se quedan los usuarios?", "¿cómo deberíamos posicionar esto?").
