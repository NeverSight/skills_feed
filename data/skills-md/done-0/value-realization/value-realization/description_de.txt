Analysieren Sie, ob Endbenutzer einen klaren Wert in Produktideen erkennen. Wird verwendet, wenn: Produktkonzepte besprochen, Funktionen bewertet, Marketingstrategien geplant, Probleme bei der Benutzerakzeptanz analysiert werden oder wenn der Benutzer Unsicherheit über die Produktrichtung zum Ausdruck bringt (z. B. „Ist diese Idee gut?“, „Was halten Sie davon?“, „Werden Benutzer das wollen?“, „Warum bleiben Benutzer nicht?“, „Wie sollten wir das positionieren?“).
