Настраивает ключ API ElevenLabs для использования с инструментами ElevenLabs MCP. Используйте, когда пользователю необходимо настроить ключ API ElevenLabs, когда инструменты ElevenLabs выходят из строя из-за отсутствия ключа API или когда пользователь упоминает о необходимости доступа к ElevenLabs.
