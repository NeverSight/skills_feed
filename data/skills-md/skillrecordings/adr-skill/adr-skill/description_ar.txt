قم بإنشاء وصيانة سجلات قرارات الهندسة المعمارية (ADRs) المُحسّنة لسير عمل الترميز الوكيل. يُستخدم عندما تحتاج إلى اقتراح أو كتابة أو تحديث أو قبول/رفض أو إهمال أو استبدال ADR؛ تمهيد مجلد وفهرس adr ؛ استشارة التفاعلات الدوائية الحالية قبل تنفيذ التغييرات؛ أو فرض اتفاقيات ADR. تستخدم هذه المهارة الأسئلة السقراطية لالتقاط النية قبل الصياغة، والتحقق من صحة المخرجات مقابل قائمة مراجعة جاهزية الوكيل.
