Guide complet pour créer des applications Web progressives (PWA) avec Next.js en utilisant les approches modernes 2025. À utiliser lorsque les utilisateurs doivent implémenter des fonctionnalités PWA dans les applications Next.js, notamment l'installabilité, la prise en charge hors ligne, les techniciens de service, les manifestes d'applications Web, les stratégies de mise en cache, les notifications push ou la conversion d'applications Next.js existantes en PWA. Couvre à la fois la prise en charge native de Next.js PWA (zéro dépendances, App Router) et le package Serwist (capacités hors ligne avancées, gestion des techniciens de service). À utiliser également pour résoudre les problèmes de PWA, optimiser les performances de PWA ou migrer à partir de packages obsolètes comme next-pwa.
