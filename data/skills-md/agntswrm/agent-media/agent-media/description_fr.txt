Boîte à outils multimédia axée sur les agents pour le traitement des images, des vidéos et de l'audio. À utiliser lorsque vous devez redimensionner, convertir, générer des images, supprimer des arrière-plans, extraire de l'audio, transcrire un discours ou générer des vidéos. Toutes les commandes renvoient une sortie JSON déterministe.
