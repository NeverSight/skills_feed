Agent-First-Medien-Toolkit für die Bild-, Video- und Audioverarbeitung. Verwenden Sie es, wenn Sie die Größe ändern, konvertieren, Bilder generieren, Hintergründe entfernen, Audio extrahieren, Sprache transkribieren oder Videos erstellen müssen. Alle Befehle geben eine deterministische JSON-Ausgabe zurück.
