Umfassender Unit-Test-Leitfaden für Vue 3 + Vite-Projekte mit Vitest und Vue Test Utils. Wird zum Schreiben oder Überprüfen von Komponententests für Vue-Komponenten, Composables, Pinia-Stores oder TypeScript/JavaScript-Dienstprogramme in Vite-basierten Projekten verwendet. Behandelt Teststruktur, Best Practices, Mocking-Strategien und Vue-spezifische Testmuster.
