Guide complet de tests unitaires pour les projets Vue 3 + Vite utilisant Vitest et Vue Test Utils. À utiliser lors de l'écriture ou de la révision de tests unitaires pour les composants Vue, les composables, les magasins Pinia ou les utilitaires TypeScript/JavaScript dans des projets basés sur Vite. Couvre la structure des tests, les meilleures pratiques, les stratégies de simulation et les modèles de test spécifiques à Vue.
