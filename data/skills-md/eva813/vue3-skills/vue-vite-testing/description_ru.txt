Комплексное руководство по модульному тестированию проектов Vue 3 + Vite с использованием Vitest и Vue Test Utils. Используйте при написании или проверке модульных тестов для компонентов Vue, компонуемых объектов, хранилищ Pinia или утилит TypeScript/JavaScript в проектах на основе Vite. Охватывает структуру тестирования, лучшие практики, стратегии макетирования и шаблоны тестирования, специфичные для Vue.
