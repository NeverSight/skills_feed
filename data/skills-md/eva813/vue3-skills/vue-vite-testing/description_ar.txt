دليل اختبار الوحدة الشامل لمشاريع Vue 3 + Vite باستخدام Vitest وVue Test Utils. يُستخدم عند كتابة أو مراجعة اختبارات الوحدة لمكونات Vue أو المواد المركبة أو مخازن Pinia أو أدوات TypeScript/JavaScript المساعدة في المشروعات المستندة إلى Vite. يغطي بنية الاختبار وأفضل الممارسات واستراتيجيات السخرية وأنماط الاختبار الخاصة بـ Vue.
