Automatisation du navigateur avec état de page persistant. À utiliser lorsque les utilisateurs demandent à naviguer sur des sites Web, à remplir des formulaires, à prendre des captures d'écran, à extraire des données Web, à tester des applications Web ou à automatiser les flux de travail du navigateur. Les expressions déclencheurs incluent « aller sur [url] », « cliquer sur », « remplir le formulaire », « prendre une capture d'écran », « gratter », « automatiser », « tester le site Web », « se connecter » ou toute demande d'interaction du navigateur.
