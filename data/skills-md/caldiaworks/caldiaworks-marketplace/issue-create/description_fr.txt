Créez des problèmes GitHub à partir des entrées utilisateur ou des documents d'exigences USDM. Le mode autonome vous guide à travers le titre, le corps et les étiquettes. Le mode USDM crée une hiérarchie de problèmes à partir des exigences avec des spécifications dans le corps du problème. À utiliser lorsque : créer un problème, un nouveau problème, un problème GitHub, un problème ouvert, un problème de fichier, créer des problèmes à partir des exigences, USDM vers des problèmes, créer un problème, /issue-create.
