Преобразуйте неоднозначные запросы пользователей в структурированные документы с требованиями USDM. Декомпозирует требования на Требование → Причина → Описание → Иерархия спецификаций. Интегрируется с билетами GitHub Issues, Asana и Jira в качестве источников входных данных. Используйте, когда: создание требований, запись документа требований, USDM, декомпозиция требований, определение требований, 要件定義, 要件を整理, 要件分解.
