Генерация изображений ИИ с помощью API OpenAI, Google и DashScope. Поддерживает преобразование текста в изображение, эталонные изображения, соотношения сторон. Последовательный по умолчанию; параллельная генерация доступна по запросу. Используйте, когда пользователь просит сгенерировать, создать или нарисовать изображения.
