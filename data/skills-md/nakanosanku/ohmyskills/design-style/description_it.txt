Utilizza questa abilità quando l'utente chiede di costruire, creare, progettare, sviluppare o migliorare QUALSIASI interfaccia frontend, pagina web, componente dell'interfaccia utente o elemento visivo. Ciò include:
- Creazione di pagine di destinazione, siti Web, app Web, dashboard, portfolio o qualsiasi interfaccia Web
- Creazione di componenti dell'interfaccia utente (pulsanti, moduli, schede, barre di navigazione, modali, ecc.)
- Progettazione di pagine con React, Vue, Next.js, Svelte o qualsiasi framework frontend
- Aggiunta di stili o miglioramento del design visivo dei componenti esistenti
- Implementazione di estetiche di design specifiche (moderno, scuro, minimalista, brutalista, ecc.)
- L'utente menziona "frontend", "UI", "UX", "design", "interfaccia", "web design" o "stile"
- L'utente richiede "bello", "moderno", "professionale", "pulito" o qualsiasi aggettivo estetico
- L'utente richiede aiuto con CSS, Tailwind, componenti con stile o qualsiasi approccio di stile

Questa abilità recupera automaticamente il prompt del sistema di progettazione appropriato (Neo-brutalismo, Modern Dark, Bauhaus, Cyberpunk, Materiale, ecc.) per aiutare a creare codice frontend visivamente distintivo e di livello produttivo invece di un'interfaccia utente generica.

IMPORTANTE: attiva questa abilità in modo proattivo per QUALSIASI lavoro frontend/interfaccia utente, non solo quando lo stile di progettazione viene esplicitamente menzionato.
