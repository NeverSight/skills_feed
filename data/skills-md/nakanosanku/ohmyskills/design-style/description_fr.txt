Utilisez cette compétence lorsque l'utilisateur demande à créer, créer, concevoir, développer ou améliorer TOUTE interface frontale, page Web, composant d'interface utilisateur ou élément visuel. Cela comprend :
- Création de pages de destination, de sites Web, d'applications Web, de tableaux de bord, de portefeuilles ou de toute interface Web
- Création de composants d'interface utilisateur (boutons, formulaires, cartes, barres de navigation, modaux, etc.)
- Concevoir des pages avec React, Vue, Next.js, Svelte ou tout autre framework frontend
- Ajout de style ou amélioration de la conception visuelle des composants existants
- Mettre en œuvre une esthétique de design spécifique (moderne, sombre, minimaliste, brutaliste, etc.)
- L'utilisateur mentionne "frontend", "UI", "UX", "design", "interface", "web design" ou "styling".
- L'utilisateur demande "beau", "moderne", "professionnel", "propre" ou tout autre adjectif esthétique.
- Les utilisateurs demandent de l'aide avec CSS, Tailwind, les composants stylisés ou toute autre approche de style

Cette compétence récupère automatiquement l'invite du système de conception appropriée (Néo-brutalisme, Modern Dark, Bauhaus, Cyberpunk, Material, etc.) pour aider à créer un code frontal de qualité production visuellement distinctif au lieu d'une interface utilisateur générique.

IMPORTANT : Déclenchez cette compétence de manière proactive pour TOUT travail frontend/UI, pas seulement lorsque le style de conception est explicitement mentionné.
