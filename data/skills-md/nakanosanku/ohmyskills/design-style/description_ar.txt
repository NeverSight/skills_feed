استخدم هذه المهارة عندما يطلب المستخدم إنشاء أو إنشاء أو تصميم أو تطوير أو تحسين أي واجهة أمامية أو صفحة ويب أو مكون واجهة مستخدم أو عنصر مرئي. وهذا يشمل:
- إنشاء الصفحات المقصودة أو مواقع الويب أو تطبيقات الويب أو لوحات المعلومات أو المحافظ أو أي واجهة ويب
- إنشاء مكونات واجهة المستخدم (الأزرار، النماذج، البطاقات، أشرطة التنقل، الوسائط، وما إلى ذلك)
- تصميم الصفحات باستخدام React أو Vue أو Next.js أو Svelte أو أي إطار عمل أمامي
- إضافة التصميم أو تحسين التصميم المرئي للمكونات الموجودة
- تنفيذ جماليات تصميمية محددة (حديثة، داكنة، بسيطة، وحشية، وما إلى ذلك)
- يذكر المستخدم "الواجهة الأمامية" أو "UI" أو "UX" أو "التصميم" أو "الواجهة" أو "تصميم الويب" أو "التصميم"
- يسأل المستخدم عن "جميلة"، "حديثة"، "احترافية"، "نظيفة"، أو أي صفة جمالية
- يطلب المستخدم المساعدة في CSS أو Tailwind أو المكونات المصممة أو أي أسلوب تصميم

تسترد هذه المهارة تلقائيًا موجه نظام التصميم المناسب (الوحشية الجديدة، وModern Dark، وBauhaus، وCyberpunk، وMaterial، وما إلى ذلك) للمساعدة في إنشاء كود أمامي مميز بصريًا على مستوى الإنتاج بدلاً من واجهة المستخدم العامة.

هام: قم بتشغيل هذه المهارة بشكل استباقي لأي عمل للواجهة الأمامية/واجهة المستخدم، وليس فقط عند ذكر نمط التصميم صراحةً.
