Utilice esta habilidad cuando el usuario solicite construir, crear, diseñar, desarrollar o mejorar CUALQUIER interfaz frontal, página web, componente de interfaz de usuario o elemento visual. Esto incluye:
- Creación de páginas de destino, sitios web, aplicaciones web, paneles, portafolios o cualquier interfaz web.
- Creación de componentes de UI (botones, formularios, tarjetas, barras de navegación, modales, etc.)
- Diseño de páginas con React, Vue, Next.js, Svelte o cualquier framework frontend.
- Agregar estilo o mejorar el diseño visual de los componentes existentes.
- Implementar estéticas de diseño específicas (moderna, oscura, minimalista, brutalista, etc.)
- El usuario menciona "frontend", "UI", "UX", "diseño", "interfaz", "diseño web" o "estilo".
- El usuario pide "hermoso", "moderno", "profesional", "limpio" o cualquier adjetivo estético.
- El usuario solicita ayuda con CSS, Tailwind, componentes con estilo o cualquier enfoque de estilo

Esta habilidad recupera automáticamente el mensaje del sistema de diseño apropiado (Neobrutalismo, Modern Dark, Bauhaus, Cyberpunk, Material, etc.) para ayudar a crear un código de interfaz visualmente distintivo y de grado de producción en lugar de una interfaz de usuario genérica.

IMPORTANTE: active esta habilidad de forma proactiva para CUALQUIER trabajo de interfaz/UI, no solo cuando se mencione explícitamente el estilo de diseño.
