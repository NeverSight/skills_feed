قم بإنشاء وتحديث تقارير حالة المشروع المنظمة من الاجتماعات أو الرسائل أو المستندات أو الملاحظات. يُستخدم عندما يحتاج المستخدمون إلى استخراج تحديثات الحالة في تقارير تخفيض السعر مع أقسام الملخص والمشكلات والمخاطر والإجراءات، أو دمج تقارير الفريق المتعددة، أو تحديث التقارير الموجودة مع الحفاظ على الترتيب الزمني العكسي، أو حفظ/تعديل التقارير في Obsidian vault، أو ذكر مصطلحات مثل "تقرير الحالة"، أو "تحديث المشروع"، أو "المزامنة"، أو "غرفة الحرب"، أو "تقرير الفريق"، أو "التحديث الأسبوعي".
