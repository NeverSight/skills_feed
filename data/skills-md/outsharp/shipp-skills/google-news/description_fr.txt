Google News est un agrégateur d'actualités gratuit de Google qui présente les titres de milliers d'éditeurs dans le monde entier. Utilisez cette compétence pour récupérer les actualités les plus importantes, les flux thématiques, les actualités spécifiques à un emplacement et les recherches par mots clés via les flux RSS publics de Google Actualités. Aucune clé API ou authentification n’est requise.
