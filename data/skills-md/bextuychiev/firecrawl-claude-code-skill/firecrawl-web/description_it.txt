Recupera contenuti Web, acquisisci screenshot, estrai dati strutturati, esegui ricerche sul Web ed esegui la scansione dei siti di documentazione. Utilizzare quando l'utente ha bisogno di informazioni Web attuali, chiede di recuperare un URL, desidera uno screenshot, deve estrarre dati specifici da una pagina o desidera conoscere un framework o una libreria.
