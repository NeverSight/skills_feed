Modelli esperti per la fisica 2D di Godot, inclusi livelli/maschere di collisione, trigger Area2D, raycasting e query PhysicsDirectSpaceState2D. Da utilizzare quando si implementa il rilevamento delle collisioni, le zone di attivazione, i sistemi di linea di vista o le query fisiche manuali. Parole chiave di attivazione: CollisionShape2D, CollisionPolygon2D, collision_layer, collision_mask, set_collision_layer_value, set_collision_mask_value, Area2D, body_entered, body_exited, RayCast2D, force_raycast_update, PhysicsPointQueryParameters2D, PhysicsShapeQueryParameters2D, direct_space_state, sposta_e_collide, sposta_e_scorri.
