Экспертный план низкоуровневого доступа к серверу (RenderingServer, PhysicsServer2D/3D, NavigationServer) с использованием RID для максимальной производительности. Обходит накладные расходы дерева сцены для процедурной генерации, систем частиц и воксельных движков. Используйте, когда узлы работают слишком медленно ИЛИ управляют тысячами объектов. Ключевые слова RenderingServer, PhysicsServer, NavigationServer, RID, Canvas_item, body_create, низкий уровень, производительность.
