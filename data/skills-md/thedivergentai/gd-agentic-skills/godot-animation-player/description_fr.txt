Modèles experts pour AnimationPlayer, y compris les types de pistes (Valeur, Méthode, Audio, Bézier), l'extraction du mouvement racine, les rappels d'animation, la génération d'animation procédurale, l'optimisation du mode d'appel et les pistes RESET. À utiliser pour les animations basées sur la chronologie, les cinématiques ou les transitions de l'interface utilisateur. Mots-clés déclencheurs : AnimationPlayer, Animation, track_insert_key, root_motion, animation_finished, RESET_track, call_mode, animation_set_next, queue, blend_times.
