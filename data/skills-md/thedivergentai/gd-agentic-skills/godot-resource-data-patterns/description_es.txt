Plano experto para diseño orientado a datos utilizando clases Resource/RefCounted (bases de datos de elementos, estadísticas de personajes, estructuras de datos reutilizables). Cubre matrices escritas, serialización, recursos anidados y almacenamiento en caché de recursos. Úselo al implementar sistemas de datos O bases de datos de inventario/estadísticas/diálogo. Palabras clave Resource, RefCounted, ItemData, CharacterStats, base de datos, serialización, @export, matrices escritas.
