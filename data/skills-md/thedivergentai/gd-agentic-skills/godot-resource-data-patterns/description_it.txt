Progetto esperto per la progettazione orientata ai dati utilizzando classi Resource/RefCounted (database di elementi, statistiche dei personaggi, strutture di dati riutilizzabili). Copre array tipizzati, serializzazione, risorse nidificate e memorizzazione nella cache delle risorse. Da utilizzare quando si implementano sistemi di dati O database di inventario/statistiche/dialogo. Parole chiave Risorsa, RefCounted, ItemData, CharacterStats, database, serializzazione, @export, array tipizzati.
