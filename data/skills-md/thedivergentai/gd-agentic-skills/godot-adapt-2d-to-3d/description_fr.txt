Modèles experts pour la migration de jeux 2D vers la 3D, y compris les conversions de types de nœuds, les systèmes de caméra (troisième personne, première personne, orbite), la migration des couches physiques, le pipeline artistique sprite-modèle et les adaptations du schéma de contrôle. À utiliser lors du portage de projets 2D en 3D ou de l'ajout d'éléments 3D. Mots-clés déclencheurs : CharacterBody2D vers CharacterBody3D, Area2D vers Area3D, Camera2D vers Camera3D, Vector2 vers Vector3, migration collision_layer, sprite vers MeshInstance3D, conversion 2D vers 3D.
