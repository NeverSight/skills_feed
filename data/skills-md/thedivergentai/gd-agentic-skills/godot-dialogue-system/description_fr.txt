Modèles experts pour les systèmes de dialogue de branchement, y compris les graphiques de dialogue (basés sur les ressources), les portraits de personnages, les choix des joueurs, le dialogue conditionnel (drapeaux/quêtes), les effets de machine à écrire, la prise en charge de la localisation et l'intégration du doublage. À utiliser pour les jeux narratifs, les RPG ou les romans visuels. Mots-clés déclencheurs : DialogueLine, DialogueChoice, DialogueGraph, dialogue_manager, typewriter_effect, branching_dialogue, dialogue_flags, localisation, voice_acting.
