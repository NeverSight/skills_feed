Expertenentwurf für responsive UI-Layouts mithilfe von Containerknoten (HBoxContainer, VBoxContainer, GridContainer, MarginContainer, ScrollContainer). Behandelt Größenflaggen, Anker, geteilte Container und dynamische Layouts. Verwenden Sie es beim Erstellen adaptiver Schnittstellen ODER beim Implementieren responsiver Menüs. Schlüsselwörter Container, HBoxContainer, VBoxContainer, GridContainer, size_flags, EXPAND_FILL, Anker, responsiv.
