[MCP WRAPPER] Программное создание/изменение сцен Godot с помощью инструментов Godot MCP. Организует mcp_godot_create_scene, mcp_godot_add_node, mcp_godot_load_sprite в агентские рабочие процессы. Используйте, когда пользователь запрашивает создание/автоматизацию сцены через MCP. Ключевые слова MCP, автоматизация сцены, программное построение сцены, иерархия узлов.
