Modèle d'expert pour les machines à états finis hiérarchiques (HSM) et les automates pushdown pour les comportements complexes d'IA/de personnages. Couvre les piles d'états, les sous-états, la validation de transition et la transmission du contexte d'état. À utiliser lorsque les FSM de base sont insuffisants OU lorsque l’IA en couches est mise en œuvre. Mots-clés machine à états, HSM, hiérarchique, automates pushdown, pile d'états, FSM, comportement de l'IA.
