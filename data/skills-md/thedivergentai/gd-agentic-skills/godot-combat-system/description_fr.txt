Modèles experts pour les systèmes de combat, y compris l'architecture hitbox/hurtbox, le calcul des dégâts (classe DamageData), les composants de santé, les machines d'état de combat, les systèmes combo, les temps de recharge des capacités et les fenêtres contextuelles de dégâts. À utiliser pour les jeux d'action, les RPG ou les jeux de combat. Mots-clés déclencheurs : Hitbox, Hurtbox, DamageData, HealthComponent, combat_state, combo_system,ability_cooldown, invincibility_frames, Damage_popup.
