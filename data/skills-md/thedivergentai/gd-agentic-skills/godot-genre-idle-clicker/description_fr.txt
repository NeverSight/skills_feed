Plan expert pour les jeux inactifs/clicker comprenant la gestion des gros nombres (système de mantisse + exposant), les courbes de croissance exponentielle (cost_growth_factor 1,15x), les systèmes générateurs (autoproducteurs), le calcul de la progression hors ligne, les systèmes de prestige (réinitialisation pour les multiplicateurs permanents), le formatage des nombres (suffixes K/M/B, notation scientifique). À utiliser pour les jeux incrémentiels, les jeux inactifs ou les dérivés de cookie clicker. Mots-clés déclencheurs : ralenti_jeu, grand_numéro, croissance_exponentielle, système_générateur, progrès_hors ligne, système_prestige, formatage_numéro.
