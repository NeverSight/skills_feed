Экспертные шаблоны для AnimationTree, включая переходы StateMachine, BlendSpace2D для направленного движения, BlendTree для многоуровневой анимации, корневого движения, условий перехода, дополнительных выражений и подсостояний конечного автомата. Используется для сложных систем анимации персонажей со смешиванием движений и управлением состояниями. Ключевые слова триггера: AnimationTree, AnimationNodeStateMachine, BlendSpace2D, BlendSpace1D, BlendTree, переход_запрос, blend_position, заранее_выражение, AnimationNodeAdd2, AnimationNodeBlend2, root_motion.
