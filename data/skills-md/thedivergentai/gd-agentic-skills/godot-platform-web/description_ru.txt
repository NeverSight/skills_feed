Экспертный план для веб-платформ/браузерных платформ (экспорт в HTML5), охватывающий рендеринг WebGL/WebGPU, настраиваемые экраны загрузки, интеграцию JavaScriptBridge, сохранение LocalStorage и оптимизацию размера. Используйте при экспорте в Интернет или реализации функций, специфичных для браузера. Ключевые слова веб, HTML5, WebGL, WebGPU, JavaScriptBridge, localStorage, холст, API браузера.
