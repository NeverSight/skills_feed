Guide pour créer des compétences Claude Code suivant les meilleures pratiques officielles d'Anthropic. À utiliser lorsque l'utilisateur souhaite créer une nouvelle compétence, développer une compétence, écrire SKILL.md ou a besoin de directives de création de compétences. Fournit une structure, des conventions de dénomination, une rédaction de description et une liste de contrôle de qualité.
