Руководство по созданию навыков Claude Code в соответствии с официальными рекомендациями Anthropic. Используйте, когда пользователь хочет создать новый навык, развить навык, написать SKILL.md или ему нужны рекомендации по созданию навыков. Содержит структуру, соглашения об именах, написание описаний и контрольный список качества.
