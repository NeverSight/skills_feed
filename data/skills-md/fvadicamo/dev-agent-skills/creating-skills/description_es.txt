Guía para crear habilidades de Claude Code siguiendo las mejores prácticas oficiales de Anthropic. Úselo cuando el usuario quiera crear una nueva habilidad, desarrollar una habilidad, escribir SKILL.md o necesite pautas de creación de habilidades. Proporciona estructura, convenciones de nomenclatura, redacción de descripciones y lista de verificación de calidad.
