Fusionne les requêtes Pull GitHub après avoir validé la liste de contrôle de pré-fusion. À utiliser lorsque l'utilisateur souhaite fusionner le PR, fermer le PR, finaliser le PR, terminer la fusion, approuver et fusionner, ou exécuter la fusion. Exécute la validation avant la fusion (tests, peluches, CI, commentaires), confirme avec l'utilisateur, fusionne avec le format approprié, gère le nettoyage après la fusion.
