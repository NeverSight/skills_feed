Объединяет запросы на извлечение GitHub после проверки контрольного списка перед слиянием. Используйте, когда пользователь хочет объединить PR, закрыть PR, завершить PR, завершить объединение, утвердить и объединить или выполнить объединение. Запускает проверку перед слиянием (тесты, анализ, CI, комментарии), подтверждает пользователя, объединяет с правильным форматом, выполняет очистку после слияния.
