يدمج طلبات سحب GitHub بعد التحقق من قائمة مراجعة ما قبل الدمج. يُستخدم عندما يريد المستخدم دمج العلاقات العامة، أو إغلاق العلاقات العامة، أو إنهاء العلاقات العامة، أو إكمال الدمج، أو الموافقة والدمج، أو تنفيذ الدمج. تشغيل التحقق من صحة ما قبل الدمج (الاختبارات، والفحص، وCI، والتعليقات)، والتأكيد مع المستخدم، والدمج بالتنسيق المناسب، والتعامل مع التنظيف بعد الدمج.
