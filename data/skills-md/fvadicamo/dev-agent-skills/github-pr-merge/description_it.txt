Unisce le richieste pull GitHub dopo aver convalidato l'elenco di controllo pre-unione. Da utilizzare quando l'utente desidera unire PR, chiudere PR, finalizzare PR, completare l'unione, approvare e unire o eseguire l'unione. Esegue la convalida pre-unione (test, lint, CI, commenti), conferma con l'utente, unisce con il formato corretto, gestisce la pulizia post-unione.
