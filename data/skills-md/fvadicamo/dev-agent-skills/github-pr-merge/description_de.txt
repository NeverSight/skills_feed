Führt GitHub-Pull-Requests zusammen, nachdem die Checkliste vor dem Zusammenführen validiert wurde. Verwenden Sie diese Option, wenn der Benutzer PR zusammenführen, PR schließen, PR abschließen, Zusammenführung abschließen, genehmigen und zusammenführen oder Zusammenführung ausführen möchte. Führt die Validierung vor dem Zusammenführen durch (Tests, Lint, CI, Kommentare), bestätigt mit dem Benutzer, führt Zusammenführungen mit dem richtigen Format durch und übernimmt die Bereinigung nach dem Zusammenführen.
