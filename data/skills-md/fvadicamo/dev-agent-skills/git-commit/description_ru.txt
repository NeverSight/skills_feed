Создает коммиты git в формате обычных коммитов с типом/областью действия/темой. Используйте, когда пользователь хочет зафиксировать изменения, создать фиксацию, сохранить работу или подготовить и зафиксировать. Обрабатывает обычные коммиты ветвей (разработка) и коммиты слияний (закрытие PR). Обеспечивает соблюдение соглашений CLAUDE.md, специфичных для проекта.
