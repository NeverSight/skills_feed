Обрабатывает комментарии по PR-обзорам и принимает решения по отзывам. Используйте, когда пользователь хочет разрешить комментарии к PR, обработать отзывы о обзоре, исправить комментарии к обзору, обратиться к обзору PR, проверить статус обзора, ответить рецензенту или проверить готовность PR. Получает комментарии через GitHub CLI, классифицирует по серьезности, применяет исправления с подтверждения пользователя, фиксирует в правильном формате, отвечает в обсуждениях.
