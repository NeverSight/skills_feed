Gère les commentaires d’examen des relations publiques et la résolution des commentaires. À utiliser lorsque l'utilisateur souhaite résoudre les commentaires de PR, gérer les commentaires de révision, corriger les commentaires de révision, traiter la révision de PR, vérifier l'état de la révision, répondre au réviseur ou vérifier l'état de préparation des PR. Récupère les commentaires via GitHub CLI, les classe par gravité, applique les correctifs avec confirmation de l'utilisateur, valide avec le format approprié, répond aux fils de discussion.
