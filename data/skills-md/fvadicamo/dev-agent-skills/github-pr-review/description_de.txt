Behandelt Kommentare zur PR-Überprüfung und die Lösung von Rückmeldungen. Verwenden Sie diese Option, wenn der Benutzer PR-Kommentare auflösen, Bewertungsfeedback bearbeiten, Bewertungskommentare korrigieren, PR-Bewertungen ansprechen, den Bewertungsstatus prüfen, dem Prüfer antworten oder die PR-Bereitschaft überprüfen möchte. Ruft Kommentare über die GitHub-CLI ab, klassifiziert sie nach Schweregrad, wendet Korrekturen mit Benutzerbestätigung an, übermittelt sie im richtigen Format und antwortet auf Threads.
