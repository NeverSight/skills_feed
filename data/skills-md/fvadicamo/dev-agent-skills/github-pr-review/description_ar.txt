يتعامل مع تعليقات مراجعة العلاقات العامة وحل الملاحظات. يُستخدم عندما يريد المستخدم حل تعليقات العلاقات العامة، أو التعامل مع تعليقات المراجعة، أو إصلاح تعليقات المراجعة، أو معالجة مراجعة العلاقات العامة، أو التحقق من حالة المراجعة، أو الرد على المراجع، أو التحقق من جاهزية العلاقات العامة. جلب التعليقات عبر GitHub CLI، والتصنيف حسب الخطورة، وتطبيق الإصلاحات مع تأكيد المستخدم، والالتزام بالتنسيق المناسب، والرد على سلاسل الرسائل.
