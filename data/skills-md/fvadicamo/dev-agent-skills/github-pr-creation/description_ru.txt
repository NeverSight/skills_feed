Создает запросы на извлечение GitHub с автоматической проверкой и отслеживанием задач. Используйте, когда пользователь хочет создать PR, открыть запрос на включение, отправить на рассмотрение или проверить готовность к PR. Анализирует фиксации, проверяет выполнение задач, генерирует заголовок и описание обычных коммитов, предлагает метки. ПРИМЕЧАНИЕ. Для объединения существующих PR используйте вместо этого github-pr-merge.
