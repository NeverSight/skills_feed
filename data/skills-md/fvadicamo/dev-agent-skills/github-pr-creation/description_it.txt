Crea richieste pull GitHub con convalida automatizzata e monitoraggio delle attività. Da utilizzare quando l'utente desidera creare PR, aprire richieste pull, inviare per la revisione o verificare se è pronto per PR. Analizza i commit, convalida il completamento delle attività, genera il titolo e la descrizione dei commit convenzionali, suggerisce etichette. NOTA: per unire PR esistenti, utilizzare invece github-pr-merge.
