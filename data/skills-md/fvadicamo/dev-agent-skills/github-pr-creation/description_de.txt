Erstellt GitHub Pull Requests mit automatisierter Validierung und Aufgabenverfolgung. Verwenden Sie diese Option, wenn der Benutzer eine PR erstellen, eine Pull-Anfrage öffnen, zur Überprüfung einreichen oder prüfen möchte, ob er für die PR bereit ist. Analysiert Commits, validiert den Abschluss von Aufgaben, generiert Titel und Beschreibung für konventionelle Commits und schlägt Beschriftungen vor. HINWEIS: Zum Zusammenführen vorhandener PRs verwenden Sie stattdessen github-pr-merge.
