Crea solicitudes de extracción de GitHub con validación automatizada y seguimiento de tareas. Úselo cuando el usuario quiera crear PR, abrir una solicitud de extracción, enviarla para revisión o verificar si está listo para PR. Analiza las confirmaciones, valida la finalización de la tarea, genera el título y la descripción de las confirmaciones convencionales y sugiere etiquetas. NOTA: para fusionar PR existentes, use github-pr-merge en su lugar.
