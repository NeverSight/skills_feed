Crée des requêtes Pull GitHub avec validation automatisée et suivi des tâches. À utiliser lorsque l'utilisateur souhaite créer une PR, ouvrir une demande d'extraction, la soumettre pour examen ou vérifier s'il est prêt pour une PR. Analyse les commits, valide l'achèvement des tâches, génère le titre et la description des commits conventionnels, suggère des étiquettes. REMARQUE - pour fusionner des PR existants, utilisez plutôt github-pr-merge.
