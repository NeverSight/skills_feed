Cuando el usuario desea optimizar, mejorar o aumentar las conversiones en cualquier página de marketing, incluida la página de inicio, las páginas de destino, las páginas de precios, las páginas de funciones o las publicaciones de blog. Úselo también cuando el usuario diga "CRO", "optimización de la tasa de conversión", "esta página no está convirtiendo", "mejorar las conversiones" o "por qué esta página no funciona". Para flujos de registro/inscripción, consulte signup-flow-cro. Para la activación posterior al registro, consulte onboarding-cro. Para formularios fuera del registro, consulte form-cro. Para ventanas emergentes/modales, consulte popup-cro.
