Cette compétence doit être utilisée lorsque l'utilisateur demande « d'augmenter les privilèges sous Linux », « de trouver des vecteurs privesc sur les systèmes Linux », « d'exploiter les mauvaises configurations sudo », « d'abuser des binaires SUID », « d'exploiter les tâches cron pour un accès root », « d'énumérer les systèmes Linux pour une élévation de privilèges » ou « d'obtenir un accès root à partir d'un shell à faible privilège ». Il fournit des techniques complètes pour identifier et exploiter les chemins d'élévation de privilèges sur les systèmes Linux.
