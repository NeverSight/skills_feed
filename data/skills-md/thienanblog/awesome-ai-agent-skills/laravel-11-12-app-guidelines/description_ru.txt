Рекомендации и рабочий процесс для работы с приложениями Laravel 11 или Laravel 12 в общих стеках (только API или полный стек), включая дополнительные инструменты Docker Compose/Sail, Inertia + React, Livewire, Vue, Blade, Tailwind v4, Fortify, Wayfinder, PHPUnit, Pint и Laravel Boost MCP. Используйте при реализации функций, исправлении ошибок или внесении изменений в пользовательский интерфейс/серверную часть, следуя инструкциям для конкретного проекта (AGENTS.md, docs/).
