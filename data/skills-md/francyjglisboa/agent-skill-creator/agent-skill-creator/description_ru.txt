Этот расширенный навык следует использовать, когда пользователь просит создать агента, автоматизировать повторяющийся рабочий процесс, создать собственный навык или ему необходимы расширенные возможности создания агента. Активируется фразами типа каждый день, ежедневно мне нужно, мне нужно повторить, создать агента для, автоматизировать рабочий процесс, создать навык для, нужно автоматизировать, превратить процесс в агента. Поддерживает одиночные агенты, многоагентные пакеты, обработку расшифровок, создание на основе шаблонов и интерактивную настройку. Клод будет использовать расширенный протокол для исследования API, определения анализа, структурирования всего, реализации функционального кода и автономного создания полных навыков с дополнительным руководством пользователя.
