Dieser Skill sollte beim Erstellen eines Skills für ein CLI-Tool verwendet werden. Verwenden Sie diese Option, wenn Benutzer ein Befehlszeilentool dokumentieren, CLI-Anleitungen erstellen oder eine Fertigkeit für Terminalbefehle erstellen möchten. Unverzichtbar für die systematische Untersuchung von CLI-Tools anhand von Hilfetexten, Manpages, GitHub-Repos und Online-Recherchen sowie für die anschließende Organisation der Ergebnisse in einer effektiven Dokumentation der Fähigkeiten.
