Cette compétence doit être utilisée lors de la création d’une compétence pour un outil CLI. À utiliser lorsque les utilisateurs demandent à documenter un outil de ligne de commande, à créer des conseils CLI ou à développer une compétence pour les commandes de terminal. Indispensable pour introspecter systématiquement les outils CLI via le texte d'aide, les pages de manuel, les dépôts GitHub et la recherche en ligne, puis organiser les résultats dans une documentation efficace des compétences.
