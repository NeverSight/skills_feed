Этот навык следует использовать при создании навыка для инструмента CLI. Используйте, когда пользователи просят документировать инструмент командной строки, создать руководство по интерфейсу командной строки или развить навыки работы с командами терминала. Необходим для систематического изучения инструментов CLI с помощью текста справки, страниц руководства, репозиториев GitHub и онлайн-исследований, а затем организации результатов в эффективную документацию по навыкам.
