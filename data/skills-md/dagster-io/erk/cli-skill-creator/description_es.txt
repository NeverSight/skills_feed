Esta habilidad se debe utilizar al crear una habilidad para una herramienta CLI. Úselo cuando los usuarios soliciten documentar una herramienta de línea de comandos, crear una guía CLI o desarrollar una habilidad para comandos de terminal. Esencial para realizar una introspección sistemática de las herramientas CLI a través de textos de ayuda, páginas de manual, repositorios de GitHub e investigaciones en línea, y luego organizar los hallazgos en documentación de habilidades efectiva.
