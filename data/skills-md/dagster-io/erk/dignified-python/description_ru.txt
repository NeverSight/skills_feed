Стандарты кодирования Python с автоматическим определением версии. Используйте при написании, проверке или рефакторинге Python, чтобы обеспечить соблюдение шаблонов обработки исключений LBYL, современного синтаксиса типов (list[str], str | None), операций pathlib, интерфейсов на основе ABC, абсолютного импорта и явных границ ошибок на уровне CLI. Также предоставляет проверенные на производстве шаблоны запахов кода от Dagster Labs для проектирования API, сложности параметров и организации кода. Необходим для поддержания достойных стандартов Python erk.
