Cette compétence doit être utilisée lorsque vous travaillez avec GitHub CLI (gh) pour les demandes d'extraction, les problèmes, les versions et l'automatisation de GitHub. À utiliser lorsque les utilisateurs mentionnent les commandes gh, les workflows GitHub, les opérations de relations publiques, la gestion des problèmes ou l'accès à l'API GitHub. Indispensable pour comprendre le modèle mental de gh, la structure des commandes et l'intégration avec les workflows git.
