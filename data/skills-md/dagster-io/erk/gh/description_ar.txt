يجب استخدام هذه المهارة عند العمل مع GitHub CLI (gh) لطلبات السحب والمشكلات والإصدارات وأتمتة GitHub. يُستخدم عندما يذكر المستخدمون أوامر gh، أو سير عمل GitHub، أو عمليات العلاقات العامة، أو إدارة المشكلات، أو الوصول إلى GitHub API. ضروري لفهم النموذج العقلي لـ gh وبنية القيادة والتكامل مع سير عمل git.
