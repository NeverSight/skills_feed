Этот навык следует использовать при работе с GitHub CLI (gh) для запросов на включение, проблем, выпусков и автоматизации GitHub. Используйте, когда пользователи упоминают команды gh, рабочие процессы GitHub, операции по связям с общественностью, управление проблемами или доступ к API GitHub. Необходим для понимания ментальной модели gh, структуры команд и интеграции с рабочими процессами git.
