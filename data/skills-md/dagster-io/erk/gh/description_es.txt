Esta habilidad se debe utilizar cuando se trabaja con GitHub CLI (gh) para solicitudes de extracción, problemas, lanzamientos y automatización de GitHub. Úselo cuando los usuarios mencionen comandos gh, flujos de trabajo de GitHub, operaciones de relaciones públicas, gestión de problemas o acceso a la API de GitHub. Esencial para comprender el modelo mental, la estructura de comandos y la integración de gh con los flujos de trabajo de git.
