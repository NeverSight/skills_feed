Questa competenza dovrebbe essere utilizzata quando si lavora con GitHub CLI (gh) per richieste pull, problemi, versioni e automazione GitHub. Da utilizzare quando gli utenti menzionano comandi gh, flussi di lavoro GitHub, operazioni PR, gestione dei problemi o accesso API GitHub. Essenziale per comprendere il modello mentale, la struttura dei comandi e l'integrazione con i flussi di lavoro git di gh.
