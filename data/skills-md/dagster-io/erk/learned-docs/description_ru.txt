Этот навык следует использовать при написании, изменении или реорганизации документации в docs/learned/. Используйте при создании новых документов, обновлении заголовка, выборе категорий, создании индексных файлов, обновлении таблиц маршрутизации или перемещении файлов между категориями. Необходим для поддержания единообразной структуры документации.
