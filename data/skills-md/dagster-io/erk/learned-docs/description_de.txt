Verwenden Sie diese Option, wenn Sie neue Dokumente erstellen, Frontmatter aktualisieren, Kategorien auswählen, Indexdateien erstellen, Routing-Tabellen aktualisieren oder Dateien zwischen Kategorien verschieben.
