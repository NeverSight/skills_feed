Verwenden Sie Shopify Polaris Web Components (s-* benutzerdefinierte Elemente) für die App Home-Benutzeroberfläche. Wird zum Erstellen von App Home-Oberflächen (nicht eingebetteten Apps) und zum Entwerfen einer Benutzeroberfläche mit S-Page, S-Section, S-Stack, S-Box, S-Button und anderen S-*-Komponenten verwendet. Verwenden Sie nicht @shopify/polaris React – App Home erfordert Webkomponenten.
