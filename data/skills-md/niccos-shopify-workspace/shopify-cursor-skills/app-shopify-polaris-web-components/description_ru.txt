Используйте веб-компоненты Shopify Polaris (настраиваемые элементы s-*) для пользовательского интерфейса App Home. Используйте при создании поверхностей App Home (не встроенных приложений), разработке пользовательского интерфейса с s-page, s-section, s-stack, s-box, s-button и другими компонентами s-*. Не используйте @shopify/polaris React — для App Home требуются веб-компоненты.
