Utilisez les composants Web Shopify Polaris (éléments personnalisés s-*) pour l'interface utilisateur de l'application Home. À utiliser lors de la création de surfaces App Home (pas d'applications intégrées), de la conception d'une interface utilisateur avec s-page, s-section, s-stack, s-box, s-button et d'autres composants s-*. N'utilisez pas @shopify/polaris React - App Home nécessite des composants Web.
