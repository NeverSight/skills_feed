Utilizza i componenti Web Shopify Polaris (s-* elementi personalizzati) per l'interfaccia utente Home app. Da utilizzare durante la creazione di superfici App Home (non app incorporate), la progettazione dell'interfaccia utente con s-page, s-section, s-stack, s-box, s-button e altri componenti s-*. Non utilizzare @shopify/polaris React - App Home richiede componenti Web.
