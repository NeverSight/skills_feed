أنشئ خطافات React مخصصة جاهزة للإنتاج لأنماط واجهة المستخدم الشائعة. تُستخدم عندما يحتاج المستخدم إلى خطافات لإدارة الحالة أو تفاعلات DOM أو العمليات غير المتزامنة أو التصميم سريع الاستجابة في تطبيقات React. يغطي: useToggle وuseDebounce وuseThrottle وuseFetch وuseAsync وuseLocalStorage وuseInterval وuseKeyPress وuseHover وuseMediaQuery وuseOnClickOutside وuseOnScreen وuseScrollToEnd وuseScrolledPastThreshold وuseWindowSize.
