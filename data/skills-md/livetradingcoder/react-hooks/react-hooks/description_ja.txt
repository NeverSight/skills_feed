一般的な UI パターン用に本番環境に対応したカスタム React フックを生成します。ユーザーが React アプリケーションの状態管理、DOM インタラクション、非同期操作、またはレスポンシブ デザイン用のフックを必要とする場合に使用します。対象: useToggle、useDebounce、useThrottle、useFetch、useAsync、useLocalStorage、useInterval、useKeyPress、useHover、useMediaQuery、useOnClickOutside、useOnScreen、useScrollToEnd、useScrolledPastThreshold、useWindowSize。
