React Native アプリ用の React Native Reanimated アニメーション コードを生成します。ユーザーが React Native でアニメーションの作成、実装、追加を要求するときに使用します。これには、トランジション、ジェスチャー、スクロールリンク効果、レイアウト アニメーション、レイアウト トランジション、アニメーションの開始/終了、CSS アニメーション、CSS トランジション、共有要素トランジション、カラー アニメーション、視差、フェード、スライド、スケール、スプリング、タイミング、減衰、キーフレーム、ワークレット、アコーディオン、ボトム シート、フリップ カード、折りたたみヘッダー、またはを使用したモーション エフェクトが含まれます。反応ネイティブ蘇生。また、react-native-gesture-handler と Reanimated の統合、ワークレットの理解、画面間のアニメーション化、Jest でのアニメーションのテスト、またはアニメーション化可能なプロパティの確認にも使用します。
