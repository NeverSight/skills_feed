قم بإنشاء رمز الرسوم المتحركة React Native Reanimated لتطبيقات React Native. يُستخدم عندما يطلب المستخدم إنشاء رسوم متحركة أو تنفيذها أو إضافتها في React Native - بما في ذلك الانتقالات، والإيماءات، والتأثيرات المرتبطة بالتمرير، ورسوم التخطيط المتحركة، وانتقالات التخطيط، ورسوم الدخول/الخروج، ورسوم CSS المتحركة، وانتقالات CSS، وانتقالات العناصر المشتركة، ورسوم الألوان المتحركة، واختلاف المنظر، والتلاشي، والانزلاق، والمقياس، والربيع، والتوقيت، والانحلال، والإطارات الرئيسية، والعمليات، والأكورديون، والورقة السفلية، والبطاقة القابلة للطي، والرأس المطوي، أو أي تأثير حركة يستخدم رد الفعل الأصلي إحياء. استخدم أيضًا لدمج معالج الإيماءات التفاعلية الأصلية مع Reanimated، أو فهم الأعمال، أو الرسوم المتحركة بين الشاشات، أو اختبار الرسوم المتحركة باستخدام Jest، أو التحقق من الخصائص القابلة للتحريك.
