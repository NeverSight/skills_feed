Generieren Sie reanimierten React Native-Animationscode für React Native-Apps. Verwenden Sie diese Option, wenn der Benutzer Animationen in React Native erstellen, implementieren oder hinzufügen möchte – einschließlich Übergängen, Gesten, Scroll-Linked-Effekten, Layout-Animationen, Layout-Übergängen, Eingabe-/Beendigungsanimationen, CSS-Animationen, CSS-Übergängen, gemeinsamen Elementübergängen, Farbanimationen, Parallaxe, Überblendung, Folie, Skalierung, Sprung, Timing, Verfall, Keyframes, Worklets, Akkordeon, unterem Blatt, Flip-Card, kollabierendem Header oder einem beliebigen Bewegungseffekt React-Native-Reanimation. Wird auch zur Integration von React-Native-Gesture-Handler mit Reanimated, zum Verstehen von Worklets, zum Animieren zwischen Bildschirmen, zum Testen von Animationen mit Jest oder zum Überprüfen, welche Eigenschaften animierbar sind, verwendet.
