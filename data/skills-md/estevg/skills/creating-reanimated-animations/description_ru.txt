Создайте React Native реанимированный код анимации для приложений React Native. Используйте, когда пользователь запрашивает создание, реализацию или добавление анимации в React Native, включая переходы, жесты, эффекты прокрутки, анимацию макета, переходы макета, анимацию входа/выхода, анимацию CSS, переходы CSS, переходы общих элементов, цветовую анимацию, параллакс, затухание, слайд, масштаб, пружину, синхронизацию, затухание, ключевые кадры, рабочие листы, аккордеон, нижний лист, флип-карту, сворачивающийся заголовок или любой эффект движения с использованием реагировать-родной-реанимированный. Также используйте для интеграции обработчика Reanimated-native-gesture-handler, понимания рабочих задач, анимации между экранами, тестирования анимации с помощью Jest или проверки того, какие свойства являются анимируемыми.
