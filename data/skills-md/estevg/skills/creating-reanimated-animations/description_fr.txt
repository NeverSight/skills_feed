Générez du code d'animation React Native Reanimated pour les applications React Native. À utiliser lorsque l'utilisateur demande de créer, d'implémenter ou d'ajouter des animations dans React Native - y compris des transitions, des gestes, des effets liés au défilement, des animations de mise en page, des transitions de mise en page, des animations d'entrée/sortie, des animations CSS, des transitions CSS, des transitions d'éléments partagés, des animations de couleur, parallaxe, fondu, diapositive, mise à l'échelle, ressort, timing, décroissance, images clés, worklets, accordéon, feuille inférieure, carte à retourner, en-tête réduit ou tout effet de mouvement utilisant réagir-natif-réanimé. Utilisez également pour intégrer React-native-gesture-handler avec Reanimated, comprendre les worklets, animer entre les écrans, tester les animations avec Jest ou vérifier quelles propriétés peuvent être animées.
