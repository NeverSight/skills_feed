Mejores prácticas de comandos de Git moderno para agentes de IA. Utilice comandos modernos y útiles introducidos en Git 2.23+ en lugar de comandos multipropósito heredados. Enseña cuándo usar `git switch` (operaciones de rama), `git restaurar` (operaciones de archivos) y otras alternativas más seguras para mejorar la claridad y reducir errores.
