Bonnes pratiques de commande Git moderne pour les agents IA. Utilisez des commandes modernes et utiles introduites dans Git 2.23+ au lieu des commandes polyvalentes héritées. Apprend quand utiliser « git switch » (opérations de branche), « git restaurer » (opérations de fichiers) et d'autres alternatives plus sûres pour améliorer la clarté et réduire les erreurs.
