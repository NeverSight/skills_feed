Testa le applicazioni utilizzando il framework PHP Pest 3. Si attiva durante la scrittura di test, la creazione di test di unità o di funzionalità, l'aggiunta di asserzioni, il test di componenti Livewire, il test dell'architettura, il debug degli errori di test, l'utilizzo di set di dati o il mocking; o quando l'utente menziona test, specifiche, TDD, si aspetta, asserzione, copertura o deve verificare che la funzionalità funzioni.
