`excalidraw` CLI を介してライブ Excalidraw キャンバスを制御します。エージェントが、(1) ライブ キャンバス上で図を描画またはレイアウトする、(2) 個々の要素を作成/更新/削除する、(3) JSON から複雑な図をバッチ作成する、(4) 記述またはスクリーンショットを使用してキャンバスを検査する、(5) .excalidraw ファイル、PNG、または SVG をエクスポート/インポートする、(6) キャンバス スナップショットを保存/復元する、(7) 要素を整列、配布、グループ化、ロック、または複製する、(8) マーメイドを変換する必要がある場合に使用します。図を Excalidraw 要素に変換し、(9) 図を暗号化された excalidraw.com URL として共有します。実行中のキャンバス サーバーが必要です (デフォルトは http://localhost:3000)。
