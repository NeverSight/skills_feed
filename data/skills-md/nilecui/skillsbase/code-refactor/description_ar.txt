تنفيذ عمليات إعادة هيكلة التعليمات البرمجية المجمعة مثل إعادة تسمية المتغيرات/الوظائف عبر الملفات، واستبدال الأنماط، وتحديث استدعاءات واجهة برمجة التطبيقات. يُستخدم عندما يطلب المستخدمون إعادة تسمية المعرفات، أو استبدال أنماط التعليمات البرمجية المهملة، أو تحديث استدعاءات الطريقة، أو إجراء تغييرات متسقة عبر مواقع متعددة.
