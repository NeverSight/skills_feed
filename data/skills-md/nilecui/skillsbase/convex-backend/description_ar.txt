أنشئ تطبيقات خلفية تفاعلية في الوقت الفعلي باستخدام Convex باستخدام استعلامات TypeScript والطفرات والإجراءات مع التفاعل التلقائي والتحديثات المتفائلة. يُستخدم عند إنشاء تطبيقات تعاونية في الوقت الفعلي، أو تنفيذ مزامنة البيانات التفاعلية، أو كتابة وظائف خلفية بدون خادم، أو إنشاء استعلامات يتم تحديثها تلقائيًا، أو تنفيذ التحولات مع ضمانات المعاملات، أو التعامل مع تحميلات الملفات باستخدام تخزين Convex، أو تنفيذ المصادقة باستخدام Convex Auth، أو تصميم مخططات قاعدة بيانات تفاعلية، أو إنشاء تطبيقات تتطلب مزامنة فورية للبيانات.
