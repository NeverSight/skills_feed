Руководство по использованию рабочих деревьев git для распараллеливания разработки с помощью агентов кодирования. Используйте этот навык, когда пользователь запрашивает работу в новом рабочем дереве или хочет работать над отдельной функцией изолированно (например, «Работать в новом рабочем дереве», «Создать рабочее дерево для функции X»).
