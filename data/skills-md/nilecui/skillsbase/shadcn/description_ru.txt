Шаблоны библиотеки компонентов shadcn/ui с примитивами пользовательского интерфейса Radix и CSS Tailwind. Используйте при создании таблиц, форм, диалоговых окон, карточек, кнопок или любого компонента пользовательского интерфейса с помощью shadcn/ui, установки компонентов shadcn или стилизации с помощью шаблонов shadcn.
