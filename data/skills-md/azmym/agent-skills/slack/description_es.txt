Interactúe con Slack a través de la API web. Lea, resuma, busque, publique mensajes, reaccione, fije y administre canales. Úselo cuando el usuario (1) comparte una URL de Slack, (2) solicita leer o resumir un canal, (3) busca mensajes de Slack, (4) solicita enviar/publicar un mensaje, (5) solicita reaccionar o fijar un mensaje, (6) busca un usuario o (7) menciona un canal de Slack por su nombre (por ejemplo, "#nombre-del-canal"). También activadores de hilos de Slack, reuniones diarias, resúmenes de conversaciones o cualquier interacción de Slack.
