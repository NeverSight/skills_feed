Interagieren Sie mit Slack über die Web-API. Nachrichten lesen, zusammenfassen, suchen, posten, reagieren, pinnen und Kanäle verwalten. Wird verwendet, wenn der Benutzer (1) eine Slack-URL teilt, (2) darum bittet, einen Kanal zu lesen oder zusammenzufassen, (3) Slack-Nachrichten durchsucht, (4) um das Senden/Posten einer Nachricht bittet, (5) um eine Reaktion oder das Anpinnen einer Nachricht bittet, (6) einen Benutzer sucht oder (7) einen Slack-Kanal namentlich erwähnt (z. B. „#Kanalname“). Auch Auslöser für Slack-Threads, tägliche Standups, Konversationszusammenfassungen oder jede Slack-Interaktion.
