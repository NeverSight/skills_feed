Ejecute Opengrep para búsqueda de código basado en patrones y escaneo de seguridad. Úselo cuando grep sea insuficiente para encontrar patrones de código que requieran comprensión estructural (llamadas a funciones, flujo de datos, estructuras anidadas). Úselo también para la detección de vulnerabilidades de seguridad con reglas YAML personalizadas.
