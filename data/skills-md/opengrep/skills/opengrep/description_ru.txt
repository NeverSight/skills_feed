Запустите Opengrep для поиска кода на основе шаблонов и сканирования безопасности. Используйте, когда grep недостаточно для поиска шаблонов кода, требующих структурного понимания (вызовы функций, поток данных, вложенные структуры). Также используйте для обнаружения уязвимостей безопасности с помощью пользовательских правил YAML.
