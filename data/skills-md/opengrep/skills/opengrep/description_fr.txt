Exécutez Opengrep pour la recherche de code basée sur des modèles et l'analyse de sécurité. À utiliser lorsque grep est insuffisant pour trouver des modèles de code qui nécessitent une compréhension structurelle (appels de fonction, flux de données, structures imbriquées). Utilisez également pour la détection des vulnérabilités de sécurité avec des règles YAML personnalisées.
