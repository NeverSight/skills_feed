Implemente, configure y personalice Streamdown: un renderizador React Markdown optimizado para streaming con resaltado de sintaxis, diagramas de sirena, renderizado matemático y compatibilidad con CJK. Úselo cuando trabaje con la instalación, configuración, complementos, estilo, seguridad o integración de Streamdown con transmisión de IA (por ejemplo, Vercel AI SDK). Se activa en: (1) Instalar o configurar Streamdown, (2) Configurar complementos (código, sirena, matemáticas, cjk), (3) Aplicar estilo o tematizar la salida de Streamdown, (4) Integrar con chat/streaming de IA, (5) Configurar seguridad, seguridad de enlaces o etiquetas HTML personalizadas, (6) Usar símbolos de intercalación, modo estático o componentes personalizados, (7) Solucionar problemas de Tailwind, Shiki o Vite.
