قم بتنفيذ Streamdown وتكوينه وتخصيصه — عارض React Markdown المحسّن للتدفق مع تمييز بناء الجملة ومخططات حورية البحر وعرض الرياضيات ودعم CJK. يُستخدم عند العمل مع إعداد Streamdown أو التكوين أو المكونات الإضافية أو التصميم أو الأمان أو التكامل مع تدفق الذكاء الاصطناعي (على سبيل المثال، Vercel AI SDK). المشغلات على: (1) تثبيت Streamdown أو إعداده، (2) تكوين المكونات الإضافية (الكود، mermaid، math، cjk)، (3) تصميم مخرجات Streamdown أو تصميمها، (4) التكامل مع الدردشة/البث عبر الذكاء الاصطناعي، (5) تكوين الأمان أو أمان الارتباط أو علامات HTML المخصصة، (6) استخدام علامات الإقحام أو الوضع الثابت أو المكونات المخصصة، (7) استكشاف مشكلات Tailwind أو Shiki أو Vite وإصلاحها.
