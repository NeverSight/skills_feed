Configurez tRPC avec TanStack Query dans les projets Next.js App Router. À utiliser lorsque : (1) Configuration de tRPC à partir de zéro dans un projet Next.js, (2) Ajout de nouveaux routeurs ou procédures, (3) Intégration de l'authentification avec le contexte tRPC, (4) Utilisation de tRPC avec les composants du serveur React, (5) Configuration des modèles de prélecture et d'hydratation.
