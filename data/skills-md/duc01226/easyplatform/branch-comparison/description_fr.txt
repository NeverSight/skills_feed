À utiliser lorsque l'utilisateur demande à comparer des branches, à analyser des différences git, à examiner les modifications, à mettre à jour les spécifications en fonction des modifications de code ou à synchroniser les spécifications avec l'implémentation. Déclencheurs sur des mots-clés tels que « comparer les branches », « git diff », « ce qui a changé », « comparaison de branches », « mise à jour des spécifications », « spécifications de synchronisation ».
