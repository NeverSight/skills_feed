Система систематической отладки для исследования первопричин перед исправлением. Используйте для ошибок, неудачных тестов, неожиданного поведения, проблем с производительностью. Используйте флаг --autonomous для структурированной автономной отладки с шлюзами утверждения. Триггеры: отладка, ошибка, ошибка, исправление, диагностика, основная причина, трассировка стека, исследование проблемы. НЕ для: проверки кода (используйте проверку кода), упрощения (используйте упроститель кода).
