Revise el código de la interfaz de usuario para comprobar el cumplimiento del diseño de la interfaz web, que cubre la accesibilidad WCAG 2.2/3.0, el diseño responsivo, el rendimiento, la usabilidad y las mejores prácticas modernas. Úselo cuando se le solicite "revisar mi interfaz de usuario", "verificar accesibilidad", "auditar diseño", "revisar UX", "verificar diseño responsivo" o "comprobar mi sitio con las mejores prácticas". Acciones: revisar, comprobar, auditar, analizar, validar. Temas: accesibilidad, WCAG, ARIA, HTML semántico, estados de enfoque, formularios, animación, tipografía, manejo de contenido, imágenes, rendimiento, Core Web Vitals, navegación, interacción táctil, diseño responsivo, dispositivos móviles primero, modo oscuro, internacionalización, hidratación.
