Упакуйте целые репозитории кода в отдельные файлы, удобные для искусственного интеллекта, с помощью Repomix. Возможности включают в себя упаковку кодовых баз с настраиваемыми шаблонами включения/исключения, создание нескольких выходных форматов (XML, Markdown, обычный текст), сохранение структуры и контекста файла, оптимизацию для использования ИИ с подсчетом токенов, фильтрацию по типам файлов и каталогам, добавление пользовательских заголовков и сводок. Используйте при упаковке баз кода для анализа ИИ, создании снимков репозитория для контекста LLM, анализе сторонних библиотек, подготовке к аудиту безопасности, создании контекста документации или оценке незнакомых баз кода.
