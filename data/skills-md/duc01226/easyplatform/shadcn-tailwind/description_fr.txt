Compétence de bibliothèque de composants React pour shadcn/ui + Tailwind CSS. À utiliser pour les projets React/Next.js/Vite nécessitant des composants d'interface utilisateur accessibles (boîtes de dialogue, formulaires, tableaux, navigation), des primitives d'interface utilisateur Radix, le style de l'utilitaire Tailwind, les thèmes et le mode sombre. PAS pour les projets angulaires - voir les compétences frontend-angular-*. Déclencheurs sur « shadcn », « radix ui », « React Components », « Tailwind Components ». Pour les décisions de conception (styles, palettes, polices), utilisez plutôt ui-ux-pro-max.
