shadcn/ui + Tailwind CSS 用の React コンポーネント ライブラリ スキル。アクセス可能な UI コンポーネント (ダイアログ、フォーム、テーブル、ナビゲーション)、Radix UI プリミティブ、Tailwind ユーティリティ スタイル、テーマ、ダーク モードを必要とする React/Next.js/Vite プロジェクトに使用します。 Angular プロジェクト用ではありません。frontend-angular-* スキルを参照してください。 「shadcn」、「radix ui」、「react コンポーネント」、「tailwind コンポーネント」でトリガーされます。デザイン (スタイル、パレット、フォント) を決定するには、代わりに ui-ux-pro-max を使用してください。
