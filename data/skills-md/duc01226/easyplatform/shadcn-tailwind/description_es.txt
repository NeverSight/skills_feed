Habilidad de biblioteca de componentes React para shadcn/ui + Tailwind CSS. Úselo para proyectos React/Next.js/Vite que necesitan componentes de interfaz de usuario accesibles (diálogos, formularios, tablas, navegación), primitivas de interfaz de usuario de Radix, estilo de utilidad Tailwind, temas y modo oscuro. NO para proyectos Angular: consulte las habilidades frontend-angular-*. Activadores en "shadcn", "radix ui", "react componentes", "tailwind componentes". Para decisiones de diseño (estilos, paletas, fuentes), utilice ui-ux-pro-max en su lugar.
