Аналитическая база данных проектирования для принятия решений UI/UX (техно-агностика). Используйте FIRST при планировании красивых интерфейсов — предоставляйте рекомендации по стилю, цветовым палитрам, сочетаниям шрифтов и рекомендации по UX ДО реализации. Доступен для поиска: 50 стилей (глассморфизм, минимализм, брутализм), 21 цветовая палитра, 50 пар шрифтов, рекомендации по диаграммам, структуры целевых страниц, шаблоны специальных возможностей. Работает с любым стеком (React, Vue, Angular, Flutter). Для реализации компонента React вместо этого используйте shadcn-tailwind. Запускается по: красивому пользовательскому интерфейсу, системе дизайна, цветовой палитре, сочетанию шрифтов, обзору UX, руководству по стилю.
