Flux est la bibliothèque officielle de composants Livewire construite avec Tailwind CSS. Fournit des composants d'interface utilisateur entièrement flexibles, fonctionnels et accessibles pour les applications Livewire. Cette compétence comprend tous les composants (gratuits et Pro). À utiliser lors de la création d'une interface utilisateur avec les composants Fluxui Docs.
