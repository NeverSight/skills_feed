Интеграция Zoho Books и Zoho Inventory API для клиентской консоли TSH. Используйте, когда:
(1) Создание новых маршрутов API, которые вызывают конечные точки Zoho.
(2) Отладка ошибок API, проблем с токенами или ограничений скорости.
(3) Добавление новых функций получения данных Zoho.
(4) Понимание кэширования токенов OAuth с помощью Upstash Redis
(5) Работа с продуктами, заказами, счетами, платежами или кредит-нотами.
(6) Устранение неполадок «Свяжитесь по поводу цены» или проблем с отображением запасов.
