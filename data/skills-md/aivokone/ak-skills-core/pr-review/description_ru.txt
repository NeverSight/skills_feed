Используйте FIRST для любой работы по PR/проверке кода — проверки отзывов, чтения комментариев CR, ответа рецензентам, обработки комментариев ботов или людей, а также подготовки коммитов по PR. Собирает отзывы из ВСЕХ источников (обсуждения, встроенные сообщения, обзоры), чтобы предотвратить распространенную ошибку отсутствия встроенных отзывов. Начните с check-pr-feedback.sh, затем при необходимости ответьте онлайн и подведите итоги в одном отчете об исправлениях.
