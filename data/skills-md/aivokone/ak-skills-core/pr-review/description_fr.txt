Utilisez FIRST pour tout travail de révision de PR/code : vérifier les commentaires, lire les commentaires CR, répondre aux réviseurs, traiter les commentaires du robot de révision ou des humains, ou préparer les validations sur un PR. Recueille les commentaires de TOUTES les sources (conversation, en ligne, avis) pour éviter l'échec courant des commentaires en ligne manquants. Commencez par check-pr-feedback.sh, puis répondez en ligne si nécessaire et résumez avec un rapport de correctif.
