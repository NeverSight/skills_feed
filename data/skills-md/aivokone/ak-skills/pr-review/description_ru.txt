Используйте FIRST для любой работы по PR/проверке кода — проверки отзывов, чтения комментариев CR, ответа рецензентам, обработки комментариев ботов или людей, а также подготовки коммитов по PR. Собирает отзывы из ВСЕХ источников (обсуждения, встроенные сообщения, обзоры), чтобы предотвратить распространенную ошибку отсутствия встроенных отзывов. Начните с get-context.sh для определения состояния, затем следуйте дереву решений. Все операции выполняются с помощью предоставленных скриптов, а не необработанных команд git/gh.
