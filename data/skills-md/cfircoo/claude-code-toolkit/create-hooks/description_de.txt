Fachkundige Anleitung zum Erstellen, Konfigurieren und Verwenden von Claude Code-Hooks. Verwenden Sie diese Option, wenn Sie mit Hooks arbeiten, Ereignis-Listener einrichten, Befehle validieren, Arbeitsabläufe automatisieren, Benachrichtigungen hinzufügen oder Hook-Typen verstehen (PreToolUse, PostToolUse, Stop, SessionStart, UserPromptSubmit usw.).
