Преобразует документы с требованиями к продукту в формат Prd.json для системы автономных агентов Ralph. Используйте при подготовке PRD для выполнения Ralph, разбиении функций на атомарные пользовательские истории или когда пользователь упоминает Ralph, Prd.json или рабочие процессы автономного агента.
