Installa, configura e gestisci gli hook di sicurezza di Claude Code che bloccano comandi pericolosi e proteggono i file sensibili. Da utilizzare durante l'impostazione della protezione di sicurezza, il blocco di comandi distruttivi (rm -rf, git reset --hard), la protezione di percorsi sensibili (.env, credenziali) o la gestione degli hook PreToolUse.
