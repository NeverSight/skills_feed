Analizza file Excel disordinati e non strutturati per identificare problemi di qualità dei dati, rilevare incoerenze di formato, trovare valori mancanti e generare report di analisi completi. Da utilizzare quando Claude deve lavorare con file Excel (.xlsx, .xls) per la valutazione della qualità dei dati, l'analisi della struttura o quando gli utenti richiedono il controllo dei dati, consigli sulla pulizia o riepiloghi statistici dei dati del foglio di calcolo.
