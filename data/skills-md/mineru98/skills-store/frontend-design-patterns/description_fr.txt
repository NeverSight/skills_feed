Architecture Enterprise React combinant les modèles DDD et FSD. À utiliser lors de (1) la conception ou la structuration d'applications React, (2) l'implémentation du modèle de composant Index/Types/Styles, (3) la configuration du mappage Service/Hook 1:1 avec React Query et Axios, (4) la configuration de la gestion de l'état Zustand, (5) l'application des conventions TypeScript pour les bases de code maintenables. Les déclencheurs incluent les décisions architecturales, la planification de la structure des dossiers, la conception de la couche de données et les tâches d'organisation du code.
