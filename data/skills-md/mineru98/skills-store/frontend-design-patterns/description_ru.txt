Архитектура Enterprise React, сочетающая шаблоны DDD и FSD. Используйте при (1) проектировании или структурировании приложений React, (2) реализации шаблона компонента Index/Types/Styles, (3) настройке сопоставления Service/Hook 1:1 с помощью React Query и Axios, (4) настройке управления состоянием Zustand, (5) применении соглашений TypeScript для поддерживаемых кодовых баз. Триггеры включают архитектурные решения, планирование структуры папок, проектирование уровня данных и задачи организации кода.
