Arquitectura Enterprise React que combina patrones DDD y FSD. Úselo al (1) diseñar o estructurar aplicaciones React, (2) implementar el patrón de componentes Índice/Tipos/Estilos, (3) configurar el mapeo Servicio/Hook 1:1 con React Query y Axios, (4) configurar la administración de estado de Zustand, (5) aplicar convenciones de TypeScript para bases de código mantenibles. Los desencadenantes incluyen decisiones arquitectónicas, planificación de la estructura de carpetas, diseño de capas de datos y tareas de organización de código.
