Optimización lógica y tareas de código basado en imágenes con Codex CLI en modo --yolo. Úselo cuando (1) optimice la lógica del código o los algoritmos, (2) procese entradas de imágenes (capturas de pantalla, diagramas, maquetas) para generar o modificar código, (3) refactorice la lógica empresarial, (4) depure problemas lógicos complejos, (5) genere código a partir de referencias visuales o (6) cualquier tarea que requiera adjuntar imágenes para contexto.
