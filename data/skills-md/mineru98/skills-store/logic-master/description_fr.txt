Optimisation logique et tâches de code basées sur des images avec Codex CLI en mode --yolo. À utiliser lors de (1) l'optimisation de la logique du code ou des algorithmes, (2) le traitement des entrées d'image (captures d'écran, diagrammes, maquettes) pour générer ou modifier du code, (3) la refactorisation de la logique métier, (4) le débogage de problèmes de logique complexes, (5) la génération de code à partir de références visuelles ou (6) toute tâche nécessitant une pièce jointe d'image pour le contexte.
