Set di strumenti completo per interagire con le API LeadGenius Pro Agent. Da utilizzare per la gestione delle campagne ABM, del ciclo di vita dei lead, dell'arricchimento dell'intelligenza artificiale, degli account target, delle sequenze di sensibilizzazione e delle integrazioni di sistema. Supporta l'elenco, la creazione e l'aggiornamento delle risorse tramite l'API RESTful LeadGenius.
