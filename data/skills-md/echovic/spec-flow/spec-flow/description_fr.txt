Flux de travail de développement interactif basé sur les spécifications avec confirmation étape par phase. Chaque phase attend la confirmation de l'utilisateur avant de continuer. Les expressions déclencheurs incluent « flux de spécifications », « mode de spécification », « besoin d'un plan » ou « développement structuré ». Crée le répertoire .spec-flow/ avec les documents de proposition, d'exigences, de conception et de tâches.
