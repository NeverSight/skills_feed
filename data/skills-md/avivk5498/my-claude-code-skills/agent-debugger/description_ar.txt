مجموعة أدوات تصحيح الأخطاء المنهجية لسير عمل وكيل الذكاء الاصطناعي في دعم العملاء. يُستخدم عند تشخيص المشكلات مع وكلاء الذكاء الاصطناعي بما في ذلك الاستجابات الخاطئة، أو مشكلات استدعاء الأداة/الوظيفة، أو حلقات المحادثة، أو الحالات المتوقفة، أو مشكلات الأداء/زمن الاستجابة. يعمل مع أي إطار عمل (LangChain، والوكلاء المخصصون، وClaude API) ويقبل سجلات المحادثات، وسجلات واجهة برمجة التطبيقات، وسجلات تنفيذ الأداة، وتكوينات الوكيل.
