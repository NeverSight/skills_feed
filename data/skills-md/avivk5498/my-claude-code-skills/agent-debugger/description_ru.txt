Набор инструментов систематической отладки для рабочих процессов агентов искусственного интеллекта в службе поддержки клиентов. Используйте при диагностике проблем с агентами ИИ, включая неправильные ответы, проблемы с вызовом инструментов/функций, зацикливания диалога, зависания или проблемы с производительностью/задержкой. Работает с любой платформой (LangChain, пользовательскими агентами, Claude API) и принимает журналы разговоров, журналы API, журналы выполнения инструментов и конфигурации агентов.
