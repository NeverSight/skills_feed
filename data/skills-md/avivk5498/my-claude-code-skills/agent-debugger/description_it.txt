Kit di strumenti di debug sistematico per flussi di lavoro degli agenti IA nell'assistenza clienti. Da utilizzare durante la diagnosi di problemi con gli agenti IA, tra cui risposte errate, problemi di chiamata di strumenti/funzioni, cicli di conversazione, stati bloccati o problemi di prestazioni/latenza. Funziona con qualsiasi framework (LangChain, agenti personalizzati, API Claude) e accetta registri di conversazioni, registri API, registri di esecuzione degli strumenti e configurazioni di agenti.
