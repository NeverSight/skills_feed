Boîte à outils de débogage systématique pour les flux de travail agents IA dans le support client. À utiliser lors du diagnostic des problèmes avec les agents IA, notamment les mauvaises réponses, les problèmes d'appel d'outils/de fonctions, les boucles de conversation, les états bloqués ou les problèmes de performances/latence. Fonctionne avec n'importe quel framework (LangChain, agents personnalisés, API Claude) et accepte les journaux de conversation, les journaux d'API, les journaux d'exécution d'outils et les configurations d'agent.
