Systematisches Debugging-Toolkit für KI-Agenten-Workflows im Kundensupport. Wird zur Diagnose von Problemen mit KI-Agenten verwendet, einschließlich falscher Antworten, Problemen beim Aufrufen von Tools/Funktionen, Konversationsschleifen, hängengebliebenen Zuständen oder Leistungs-/Latenzproblemen. Funktioniert mit jedem Framework (LangChain, benutzerdefinierte Agenten, Claude API) und akzeptiert Konversationsprotokolle, API-Protokolle, Tool-Ausführungsprotokolle und Agentenkonfigurationen.
