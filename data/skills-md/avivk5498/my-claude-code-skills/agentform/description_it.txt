Crea ed esegui il debug delle configurazioni dell'agente AI Agentform (file .af). Utilizzare quando:
(1) Creazione di nuovi progetti o flussi di lavoro agentform
(2) Debug degli errori di sintassi del modulo agente
(3) Aggiunta di integrazioni del server MCP
(4) Configurazione di agenti, modelli, policy o funzionalità
(5) Scrivere le fasi del flusso di lavoro con instradamento e approvazione umana
Agentform è "Infrastruttura come codice per agenti AI": i file .af dichiarativi definiscono agenti, flussi di lavoro e policy.
