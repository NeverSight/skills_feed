Erstellen und debuggen Sie Agentform AI-Agentenkonfigurationen (.af-Dateien). Verwenden Sie, wenn:
(1) Erstellen neuer Agentform-Projekte oder Workflows
(2) Debuggen von Agentform-Syntaxfehlern
(3) Hinzufügen von MCP-Serverintegrationen
(4) Konfigurieren von Agenten, Modellen, Richtlinien oder Funktionen
(5) Schreiben von Workflow-Schritten mit Routing und menschlicher Genehmigung
Agentform ist „Infrastruktur als Code für KI-Agenten“ – deklarative .af-Dateien definieren Agenten, Arbeitsabläufe und Richtlinien.
