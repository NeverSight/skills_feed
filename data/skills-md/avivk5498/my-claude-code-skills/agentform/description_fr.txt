Créez et déboguez les configurations d'agent Agentform AI (fichiers .af). À utiliser lorsque :
(1) Création de nouveaux projets ou flux de travail de formulaire d'agent
(2) Débogage des erreurs de syntaxe du formulaire d'agent
(3) Ajout d'intégrations de serveur MCP
(4) Configuration d'agents, de modèles, de politiques ou de capacités
(5) Rédaction des étapes du flux de travail avec routage et approbation humaine
Agentform est « Infrastructure as Code for AI agents » : les fichiers déclaratifs .af définissent les agents, les flux de travail et les politiques.
