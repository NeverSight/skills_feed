إنشاء وتصحيح تكوينات وكيل Agentform AI (ملفات .af). استخدم عندما:
(1) إنشاء مشاريع أو مسارات عمل جديدة لنموذج الوكيل
(2) تصحيح أخطاء بناء جملة agentform
(3) إضافة عمليات تكامل خادم MCP
(4) تكوين الوكلاء أو النماذج أو السياسات أو القدرات
(5) كتابة خطوات سير العمل مع التوجيه والموافقة البشرية
Agentform هو "البنية الأساسية كرمز لوكلاء الذكاء الاصطناعي" - تحدد ملفات .af التعريفية الوكلاء ومسارات العمل والسياسات.
