Cree terminales sin servidor RunPod listos para producción con tiempos de inicio en frío optimizados. Úselo al crear o modificar trabajadores sin servidor RunPod para (1) inferencia LLM basada en vLLM, (2) generación de imágenes/videos ComfyUI o (3) inferencia Python personalizada. Admite tanto modelos horneados (arranques en frío más rápidos) como carga dinámica (modelos compartidos). Genera proyectos completos que incluyen Dockerfiles, controladores de trabajadores, scripts de inicio y configuración optimizada para una latencia mínima de inicio en frío.
