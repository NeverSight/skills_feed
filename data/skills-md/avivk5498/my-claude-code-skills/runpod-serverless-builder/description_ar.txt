أنشئ نقاط نهاية RunPod بدون خادم جاهزة للإنتاج مع أوقات بدء باردة محسنة. يُستخدم عند إنشاء أو تعديل عمال RunPod بدون خادم لـ (1) استدلال LLM المستند إلى vLLM، أو (2) إنشاء صور/فيديو ComfyUI، أو (3) استدلال Python المخصص. يدعم كلا من النماذج المخبوزة (أسرع عمليات التشغيل الباردة) والتحميل الديناميكي (النماذج المشتركة). يُنشئ مشاريع كاملة بما في ذلك ملفات Dockerfiles، ومعالجات العمال، والبرامج النصية لبدء التشغيل، والتكوين المحسّن للحد الأدنى من زمن الوصول لبدء التشغيل البارد.
