Utilice la CLI de Memberstack para administrar cuentas de Memberstack desde la terminal. Cubre autenticación, aplicaciones, miembros, planes, campos personalizados, tablas de datos y registros. Active esta habilidad siempre que el usuario quiera interactuar con Memberstack, incluida la gestión de miembros, planes, campos personalizados, tablas/registros de datos o la autenticación con Memberstack. También se activa cuando el usuario menciona "memberstack", "memberstack-cli", administración de membresías u operaciones de datos de miembros a través de CLI.
