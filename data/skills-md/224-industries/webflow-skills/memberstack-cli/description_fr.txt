Utilisez la CLI Memberstack pour gérer les comptes Memberstack à partir du terminal. Couvre l'authentification, les applications, les membres, les plans, les champs personnalisés, les tables de données et les enregistrements. Déclenchez cette compétence chaque fois que l'utilisateur souhaite interagir avec Memberstack, notamment en gérant les membres, les plans, les champs personnalisés, les tables/enregistrements de données ou en s'authentifiant auprès de Memberstack. Se déclenche également lorsque l'utilisateur mentionne "memberstack", "memberstack-cli", la gestion des membres ou les opérations sur les données des membres via CLI.
