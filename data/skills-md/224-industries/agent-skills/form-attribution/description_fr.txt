Implémentez la bibliothèque d'attribution de formulaires sur les sites Web pour capturer automatiquement les paramètres UTM, les identifiants de clic sur les annonces, les données de référence et autres attributions marketing. À utiliser lorsque l'utilisateur doit (1) ajouter un suivi d'attribution marketing à un site Web, (2) configurer l'attribution de formulaire pour des cas d'utilisation spécifiques tels que le suivi inter-sous-domaines ou l'intégration CRM, (3) résoudre les problèmes d'attribution de formulaire ou (4) implémenter des modèles spécifiques à la plate-forme pour Webflow, HubSpot, WordPress ou d'autres plates-formes sélectionnées.
