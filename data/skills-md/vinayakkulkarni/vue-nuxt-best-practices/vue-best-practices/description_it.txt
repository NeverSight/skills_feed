Linee guida per l'ottimizzazione delle prestazioni di Vue.js per la creazione di applicazioni veloci e manutenibili. Questa competenza dovrebbe essere utilizzata durante la scrittura, la revisione o il refactoring del codice Vue.js per garantire modelli di prestazioni ottimali. Trigger su attività che coinvolgono componenti Vue, reattività, API di composizione, gestione dello stato o miglioramenti delle prestazioni.
