Richtlinien zur Leistungsoptimierung von Vue.js zum Erstellen schneller, wartbarer Anwendungen. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von Vue.js-Code verwendet werden, um optimale Leistungsmuster sicherzustellen. Wird bei Aufgaben ausgelöst, die Vue-Komponenten, Reaktivität, Composition API, Statusverwaltung oder Leistungsverbesserungen betreffen.
