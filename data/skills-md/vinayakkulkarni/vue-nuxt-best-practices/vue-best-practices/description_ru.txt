Рекомендации по оптимизации производительности Vue.js для создания быстрых и удобных в обслуживании приложений. Этот навык следует использовать при написании, проверке или рефакторинге кода Vue.js, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, связанных с компонентами Vue, реактивностью, Composition API, управлением состоянием или улучшением производительности.
