Рекомендации по оптимизации производительности и архитектуре Nuxt 3/4 для создания быстрых и удобных в обслуживании полнофункциональных приложений. Этот навык следует использовать при написании, проверке или рефакторинге кода Nuxt для обеспечения оптимальных шаблонов. Запускается при выполнении задач, связанных с получением данных, маршрутами сервера, автоматическим импортом, режимами рендеринга или специфичными для Nuxt функциями.
