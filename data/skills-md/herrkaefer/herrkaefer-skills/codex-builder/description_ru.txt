Выполняйте планы реализации, делегируя CLI Codex CLI автономную реализацию кода. Используйте, когда: (1) у пользователя есть файл плана, созданный в режиме плана Claude Code, и он хочет его реализовать, (2) пользователь явно запрашивает Кодекс реализовать план, (3) после того, как проверка плана завершена и готова к реализации, (4) пользователь предоставляет путь к файлу уценки плана и желает автоматической реализации. Навык считывает файл плана и делегирует полную реализацию Кодексу.
