Ejecute planes de implementación delegando en Codex CLI la implementación autónoma del código. Úselo cuando: (1) El usuario tiene un archivo de plan generado por el modo de plan de Claude Code y desea implementarlo, (2) El usuario solicita explícitamente a Codex que implemente un plan, (3) Después de que la revisión del plan esté completa y lista para su implementación, (4) El usuario proporciona una ruta a un archivo de rebajas del plan y desea una implementación automática. La habilidad lee el archivo del plan y delega la implementación completa al Codex.
