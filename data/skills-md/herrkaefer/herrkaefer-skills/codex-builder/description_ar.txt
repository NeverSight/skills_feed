تنفيذ خطط التنفيذ من خلال التفويض إلى Codex CLI لتنفيذ التعليمات البرمجية المستقلة. يُستخدم عندما: (1) يكون لدى المستخدم ملف خطة تم إنشاؤه بواسطة وضع خطة Claude Code ويريد تنفيذه، (2) يطلب المستخدم صراحةً من Codex تنفيذ خطة، (3) بعد اكتمال مراجعة الخطة وجاهزيتها للتنفيذ، (4) يوفر المستخدم مسارًا إلى ملف تخفيض الخطة ويريد التنفيذ التلقائي. تقوم المهارة بقراءة ملف الخطة وتفويض التنفيذ الكامل إلى الدستور الغذائي.
