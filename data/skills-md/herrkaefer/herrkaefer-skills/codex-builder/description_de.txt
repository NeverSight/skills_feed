Führen Sie Implementierungspläne aus, indem Sie sie zur autonomen Codeimplementierung an Codex CLI delegieren. Verwenden Sie dies, wenn: (1) der Benutzer über eine Plandatei verfügt, die im Planmodus von Claude Code generiert wurde, und diese implementieren möchte, (2) der Benutzer Codex explizit auffordert, einen Plan zu implementieren, (3) nachdem die Planüberprüfung abgeschlossen und zur Implementierung bereit ist, (4) der Benutzer einen Pfad zu einer Plan-Markdown-Datei angibt und eine automatische Implementierung wünscht. Der Skill liest die Plandatei und delegiert die vollständige Implementierung an Codex.
