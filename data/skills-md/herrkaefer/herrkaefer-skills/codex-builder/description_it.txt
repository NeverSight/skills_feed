Esegui piani di implementazione delegando alla CLI Codex per l'implementazione autonoma del codice. Utilizzare quando: (1) l'utente ha un file di piano generato dalla modalità piano di Claude Code e desidera implementarlo, (2) l'utente richiede esplicitamente a Codex di implementare un piano, (3) dopo che la revisione del piano è completa e pronta per l'implementazione, (4) l'utente fornisce un percorso a un file di ribasso del piano e desidera l'implementazione automatica. La competenza legge il file del piano e delega la piena attuazione al Codex.
