Analisi descrittiva completa di set di dati tabulari con statistiche raggruppate, estrazione di entità e riepiloghi pronti per la pubblicazione. Da utilizzare quando si analizzano set di dati Excel/CSV che richiedono: (1) statistiche descrittive generali e raggruppate, (2) distribuzioni di frequenza per variabili categoriali, (3) estrazione di entità da campi di testo, (4) analisi delle dinamiche temporali, (5) tabelle di output CSV con report di riepilogo MD. Supporta punteggi SDT, metriche di coinvolgimento o qualsiasi variabile continua/categoriale.
