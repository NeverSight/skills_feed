Analice la calidad del código basándose en los principios de "Código limpio". Identifique problemas de nombres, tamaño de funciones, duplicación, ingeniería excesiva y números mágicos con clasificaciones de gravedad y sugerencias de refactorización. Úselo cuando el usuario solicite revisión de código, control de calidad, consejos de refactorización, análisis de código limpio, detección de olores de código o mencione términos como 代码体检, 代码质量, 重构检查.
