أفضل ممارسات Git لرسائل الالتزام وسير عمل الفرع. استخدم عندما:
- الكتابة أو المراجعة أو تقديم المشورة بشأن رسالة التزام git
- تقديم المشورة بشأن استراتيجية التفرع أو الدمج أو إعادة التأسيس
- إعداد أو شرح سير عمل Git للفريق
- إعداد أو مراجعة طلب السحب
- حل أو تقديم المشورة بشأن كيفية تجنب تعارضات الدمج
- أي مهمة تتضمن تاريخ git، أو التاريخ الخطي، أو نظافة العلاقات العامة
