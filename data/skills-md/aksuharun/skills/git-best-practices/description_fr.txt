Meilleures pratiques Git pour les messages de validation et le workflow de branche. À utiliser lorsque :
- Rédaction, révision ou conseil sur un message de commit git
- Conseil sur la stratégie de branchement, de fusion ou de rebasage
- Mise en place ou explication d'un workflow Git en équipe
- Préparer ou examiner une pull request
- Résoudre ou conseiller sur la manière d'éviter les conflits de fusion
- Toute tâche impliquant l'historique git, l'historique linéaire ou l'hygiène des relations publiques
