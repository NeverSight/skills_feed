Utilizza le icone Lucide correttamente in qualsiasi framework o ambiente. Lucide è una libreria di icone SVG agitabile con oltre 1000 icone. Usa questa abilità quando lavori con le icone Lucide in React, Vue, Svelte, Solid, Preact, Angular, Astro, React Native, Vanilla JS o ambienti statici/lato server. I trigger includono l'installazione di icone lucide, l'importazione di componenti lucide, la personalizzazione dell'aspetto delle icone, l'uso di icone dinamiche o l'uso di icone personalizzate di Lucide lab.
