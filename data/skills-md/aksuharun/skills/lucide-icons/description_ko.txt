모든 프레임워크나 환경에서 Lucide 아이콘을 올바르게 사용하세요. Lucide는 1000개 이상의 아이콘이 포함된 트리 셰이크 가능한 SVG 아이콘 라이브러리입니다. React, Vue, Svelte, Solid, Preact, Angular, Astro, React Native, Vanilla JS 또는 정적/서버 측 환경에서 Lucide 아이콘으로 작업할 때 이 기술을 사용하세요. 트리거에는 Lucide 아이콘 설치, Lucide 구성 요소 가져오기, 아이콘 모양 사용자 정의, 동적 아이콘 사용 또는 Lucide 랩 사용자 정의 아이콘 사용이 포함됩니다.
