X API e abilità di scraper Twitter per agenti di codifica AI. Crea integrazioni con l'API REST Xquik, server MCP e webhook: ricerca di tweet, ricerca di utenti, estrazione di follower, metriche di coinvolgimento, estrazioni di concorsi omaggio, argomenti di tendenza, monitoraggio dell'account, estrazione di risposte/retweet/quote, dati di community e spazio, controlli di follow reciproci. Funziona con Claude Code, Cursor, Codex, Copilot, Windsurf e oltre 40 agenti.
