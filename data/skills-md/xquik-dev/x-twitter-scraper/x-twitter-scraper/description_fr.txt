Compétence X API et Twitter Scraper pour les agents de codage IA. Crée des intégrations avec l'API Xquik REST, le serveur MCP et les webhooks : recherche de tweets, recherche d'utilisateurs, extraction de followers, mesures d'engagement, tirages au sort de concours, sujets tendances, surveillance de compte, extraction de réponses/retweets/citations, données de communauté et d'espace, contrôles de suivi mutuels. Fonctionne avec Claude Code, Cursor, Codex, Copilot, Windsurf et plus de 40 agents.
