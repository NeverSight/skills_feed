Навыки парсера X API и Twitter для агентов ИИ-кодирования. Обеспечивает интеграцию с API Xquik REST, сервером MCP и веб-перехватчиками: поиск твитов, поиск пользователей, извлечение подписчиков, показатели вовлеченности, розыгрыши бесплатных подарков, актуальные темы, мониторинг учетных записей, извлечение ответов/ретвитов/цитат, данные сообщества и пространства, взаимные проверки подписок. Работает с Claude Code, Cursor, Codex, Copilot, Windsurf и более чем 40 агентами.
