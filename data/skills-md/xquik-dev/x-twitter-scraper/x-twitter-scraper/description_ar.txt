X API ومهارة مكشطة Twitter لوكلاء ترميز الذكاء الاصطناعي. يبني عمليات تكامل مع Xquik REST API وخادم MCP وخطافات الويب: البحث عن التغريدات والبحث عن المستخدم واستخراج المتابعين ومقاييس المشاركة وسحوبات مسابقة الهدايا والموضوعات الشائعة ومراقبة الحساب واستخراج الرد/إعادة التغريد/الاقتباس وبيانات المجتمع والمساحة وعمليات التحقق من المتابعة المتبادلة. يعمل مع Claude Code وCursor وCodex وCopilot وWindsurf وأكثر من 40 وكيلًا.
