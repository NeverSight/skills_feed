أفضل الممارسات لبناء التطبيقات باستخدام واجهة برمجة تطبيقات بيانات سوق العملات المشفرة في الوقت الفعلي لـ MMT. يُستخدم عند إنشاء روبوتات تداول أو لوحات معلومات أو أدوات تحليلية أو أي تطبيق يستهلك MMT REST أو واجهات برمجة تطبيقات WebSocket. يغطي إدارة الاتصال، وتحديد المعدل، ومعالجة البيانات، وتجميع التبادلات المتعددة، وتحسين الأداء.
