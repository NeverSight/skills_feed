قائمة مراجعة مراجعة العلاقات العامة Security-first لهذا الريبو. يُستخدم عند مراجعة الاختلافات/PRs، خاصة التغييرات التي تتضمن المصادقة أو الشبكات أو البيانات الحساسة أو تحديثات التبعية/ملف القفل. التركيز على مخاطر تسرب المعلومات السرية/معلومات تحديد الهوية الشخصية (PII)، ومخاطر سلسلة التوريد (npm + فحص وحدات_العقدة)، والهندسة المعمارية عبر الأنظمة الأساسية (الامتداد/الجوال/سطح المكتب/الويب)، وأداء React (الخطافات + نقاط الاتصال لإعادة العرض). تجنب الأخطاء في أسلوب واجهة المستخدم. مراجعة العلاقات العامة.
