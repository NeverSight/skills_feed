Elenco di controllo per la revisione delle PR incentrata sulla sicurezza per questo repository. Da utilizzare durante la revisione di differenze/PR, in particolare modifiche che coinvolgono autenticazione, rete, dati sensibili o aggiornamenti di dipendenze/file di blocco. Focus sul rischio di perdita di segreti/PII, rischio della catena di fornitura (ispezione npm + node_modules), architettura multipiattaforma (estensione/mobile/desktop/web) e prestazioni di React (hook + hotspot di re-rendering). Evita i pignoli stile dell'interfaccia utente. Revisione delle pubbliche relazioni.
