Normes de qualité du code pour OneKey. À utiliser pour corriger les avertissements de charpie, exécuter des tâches de pré-validation, gérer les variables inutilisées, écrire des commentaires ou garantir la qualité du code. Tous les commentaires doivent être en anglais. Déclencheurs sur lint, lint, eslint, oxlint, tsc, vérification de type, variable inutilisée, commentaire, documentation, vérification orthographique, qualité du code, pré-validation, fil lint.
