Стандарты качества кода для OneKey. Используйте при исправлении предупреждений lint, выполнении задач перед фиксацией, обработке неиспользуемых переменных, написании комментариев или обеспечении качества кода. Все комментарии должны быть на английском языке. Триггеры: lint, linting, eslint, oxlint, tsc, проверка типа, неиспользуемая переменная, комментарий, документация, проверка орфографии, качество кода, предварительная фиксация, Yarn lint.
