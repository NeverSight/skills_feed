Опыт статического анализа на основе LLVM, включая анализ потоков данных, анализ указателей, отслеживание вредоносных данных и проверку программ. Используйте этот навык при внедрении сканеров безопасности, средств поиска ошибок, инструментов контроля качества кода или выполнения исследований по анализу программ.
