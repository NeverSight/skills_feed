Создает интерфейсы командной строки для библиотек Python с помощью Click или Typer. Включает группы команд, обработку аргументов, индикаторы выполнения, завершение оболочки и тестирование CLI с помощью CliRunner. Используйте при добавлении функций CLI в библиотеку или создании автономных инструментов командной строки.
