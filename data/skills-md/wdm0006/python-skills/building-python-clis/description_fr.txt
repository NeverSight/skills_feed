Construit des interfaces de ligne de commande pour les bibliothèques Python à l'aide de Click ou Typer. Comprend les groupes de commandes, la gestion des arguments, les barres de progression, l'achèvement du shell et les tests CLI avec CliRunner. À utiliser lors de l'ajout de fonctionnalités CLI à une bibliothèque ou de la création d'outils de ligne de commande autonomes.
