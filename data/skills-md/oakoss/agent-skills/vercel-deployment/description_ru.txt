Конфигурация развертывания Vercel, среды предварительной версии и пограничные функции. Охватывает конфигурацию vercel.json и vercel.ts, перезапись, перенаправление, заголовки, переменные среды в рабочей/предварительной версии/разработке, пользовательские домены, настройку DNS, пограничные и бессерверные функции, Vercel CLI, GitHub Actions CI/CD и развертывания монорепозитория.

Используйте при развертывании в Vercel, настройке сборок, настройке ветвей предварительного просмотра, управлении переменными среды, настройке пользовательских доменов или использовании пограничных/бессерверных функций. Используйте для vercel, развертывания, предварительного просмотра, Edge-функции, бессерверного использования, vercel-json, переменных среды, доменов, монорепозитория.
