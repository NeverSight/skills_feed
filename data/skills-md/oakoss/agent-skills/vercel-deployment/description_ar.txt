تكوين نشر Vercel وبيئات المعاينة ووظائف الحافة. يغطي تكوين vercel.json وvercel.ts، وإعادة الكتابة، وإعادة التوجيه، والرؤوس، ومتغيرات البيئة عبر الإنتاج/المعاينة/التطوير، والمجالات المخصصة، وإعداد DNS، ووظائف الحافة والوظائف بدون خادم، وVercel CLI، وGitHub Actions CI/CD، وعمليات نشر monorepo.

يُستخدم عند النشر إلى Vercel، أو تكوين الإصدارات، أو إعداد فروع المعاينة، أو إدارة متغيرات البيئة، أو تكوين المجالات المخصصة، أو استخدام وظائف الحافة/بدون خادم. يُستخدم لـ vercel، والنشر، والمعاينة، ووظيفة الحافة، وبدون خادم، وvercel-json، ومتغيرات البيئة، والمجالات، ومونوريبو.
