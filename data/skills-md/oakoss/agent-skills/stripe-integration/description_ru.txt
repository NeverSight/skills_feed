Интеграция платежей Stripe с Node.js SDK. Охватывает сеансы оформления заказа, платежные намерения, подписки, клиентский портал, веб-перехватчики и элементы Stripe.

Используйте при реализации потоков оформления заказа, обработки платежей, выставления счетов по подписке, обработчиков веб-перехватчиков или форм оплаты с помощью элементов Stripe.

Используйте для полос, платежей, оформления заказа, подписок, выставления счетов, веб-перехватчиков, платежных намерений, элементов полос.
