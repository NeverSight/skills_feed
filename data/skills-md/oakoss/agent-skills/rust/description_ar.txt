أنماط اللغة الصدأ والنظام البيئي الذي يغطي الملكية، والاقتراض، ومدة الحياة، ومعالجة الأخطاء (النتيجة/الخيار، هذا الخطأ، على أية حال)، والمزامنة مع Tokio، والسمات، والأدوية العامة، وتسلسل serde، ومساحات عمل Cargo، والاختبار. يتضمن إطار ويب أكسوم، وواجهة مستخدم محطة راتاتوي، وتحليل Clap CLI، وقياس المعايير، وأمن سلسلة التوريد لرفض البضائع، والتوزيع الثنائي لتقسيم البضائع، وأتمتة سجل التغيير لإصدار plz.

يُستخدم عند كتابة كود Rust، أو تصميم نماذج الملكية، أو إنشاء خدمات غير متزامنة باستخدام Tokio/axum، أو إنشاء واجهات مستخدم طرفية باستخدامratatui، أو إعداد تطبيقات CLI باستخدام clap، أو تكوين خطوط أنابيب CI/CD لـ Rust، أو قياس الأداء باستخدام المعيار، أو تدقيق التبعيات باستخدام رفض الشحن، أو توزيع الثنائيات باستخدام Cargo-dist.
