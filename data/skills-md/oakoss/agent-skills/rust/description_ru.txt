Шаблоны языка и экосистема Rust, охватывающие владение, заимствование, время жизни, обработку ошибок (результат/опция, эта ошибка и так далее), асинхронность с Tokio, особенности, дженерики, сериализацию Serde, рабочие пространства Cargo и тестирование. Включает в себя веб-фреймворк axum, пользовательский интерфейс терминала ratatui, анализ CLI clap, сравнительный анализ критериев, безопасность цепочки поставок с запретом груза, двоичное распределение груза-dist и автоматизацию журнала изменений Release-plz.

Используйте при написании кода Rust, разработке моделей владения, создании асинхронных сервисов с помощью Tokio/axum, создании пользовательских интерфейсов терминала с помощью ratatui, настройке приложений CLI с помощью clap, настройке конвейеров CI/CD для Rust, сравнительном тестировании с использованием критерия, аудите зависимостей с помощью Cargo-deny или распространении двоичных файлов с помощью Cargo-dist.
