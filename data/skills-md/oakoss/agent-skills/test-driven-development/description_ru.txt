Дисциплина «красно-зеленый рефакторинг» для написания кода, управляемого тестами. Охватывает цикл TDD, качество тестирования, макетирование границ и проектирование для тестируемости.

Используйте при реализации функций, исправлении ошибок, рефакторинге существующего кода, разработке интерфейсов модулей или работе с вертикальными срезами рабочего процесса трассировки.
