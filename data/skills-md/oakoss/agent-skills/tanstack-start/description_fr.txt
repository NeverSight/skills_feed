Framework React full-stack construit sur TanStack Router. Routage de type sécurisé, fonctions de serveur, SSR/streaming, middleware et déploiement multiplateforme. À utiliser lors de la création d'applications React full-stack avec SSR, fonctions de serveur, middleware ou authentification. À utiliser pour le déploiement createServerFn, createMiddleware, beforeLoad, API, Cloudflare Workers, Vercel, Netlify.
