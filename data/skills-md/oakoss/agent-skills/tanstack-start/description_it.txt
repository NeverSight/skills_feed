Framework React full-stack basato su TanStack Router. Routing indipendente dai tipi, funzioni server, SSR/streaming, middleware e distribuzione multipiattaforma. Da utilizzare durante la creazione di app React full-stack con SSR, funzioni server, middleware o autenticazione. Utilizzare per createServerFn, createMiddleware, beforeLoad, percorsi API, Cloudflare Workers, Vercel, distribuzione Netlify.
