Полнофункциональная платформа React, построенная на базе TanStack Router. Типобезопасная маршрутизация, функции сервера, SSR/потоковая передача, промежуточное программное обеспечение и многоплатформенное развертывание. Используйте при создании полнофункциональных приложений React с SSR, серверными функциями, промежуточным программным обеспечением или аутентификацией. Используйте для createServerFn, createMiddleware, beforeLoad, маршрутов API, Cloudflare Workers, Vercel, развертывания Netlify.
