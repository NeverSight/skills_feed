إطار عمل React كامل المكدس مبني على TanStack Router. التوجيه الآمن من النوع، ووظائف الخادم، وSSR/البث، والبرمجيات الوسيطة، والنشر متعدد المنصات. يُستخدم عند إنشاء تطبيقات React متكاملة مع SSR أو وظائف الخادم أو البرامج الوسيطة أو المصادقة. استخدم لـ createServerFn، و createMiddleware، و beforeLoad، ومسارات API، و Cloudflare Workers، و Vercel، و Netlify.
