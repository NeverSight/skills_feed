Automatisation du navigateur via l'extension Chrome et DevTools MCP. À utiliser pour contrôler un navigateur Chrome connecté, automatiser les flux de travail, remplir des formulaires, extraire des données, déboguer des applications Web, lire les journaux de la console ou exécuter des tâches de navigateur planifiées. Utilisation pour les tests de navigateur, l'automatisation de formulaires, l'extraction de données, le web scraping et la navigation authentifiée.
