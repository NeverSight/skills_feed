أتمتة المتصفح عبر امتداد Chrome وDevTools MCP. يُستخدم عند التحكم في متصفح Chrome الذي تم تسجيل الدخول إليه، أو أتمتة سير العمل، أو ملء النماذج، أو استخراج البيانات، أو تصحيح أخطاء تطبيقات الويب، أو قراءة سجلات وحدة التحكم، أو تشغيل مهام المتصفح المجدولة. يستخدم لاختبار المتصفح، وأتمتة النماذج، واستخراج البيانات، وكشط الويب، والتصفح المعتمد.
