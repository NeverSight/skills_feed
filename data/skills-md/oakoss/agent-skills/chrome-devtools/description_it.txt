Automazione del browser tramite estensione Chrome e DevTools MCP. Da utilizzare per controllare un browser Chrome a cui è stato effettuato l'accesso, automatizzare i flussi di lavoro, compilare moduli, estrarre dati, eseguire il debug di app Web, leggere registri della console o eseguire attività del browser pianificate. Utilizzare per test del browser, automazione dei moduli, estrazione dati, web scraping, navigazione autenticata.
