Автоматизация браузера с помощью расширения Chrome и DevTools MCP. Используйте при управлении браузером Chrome, вошедшим в систему, автоматизации рабочих процессов, заполнении форм, извлечении данных, отладке веб-приложений, чтении журналов консоли или выполнении запланированных задач браузера. Используйте для тестирования браузера, автоматизации форм, извлечения данных, очистки веб-страниц, просмотра с аутентификацией.
