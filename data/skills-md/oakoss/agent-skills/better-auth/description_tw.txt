自託管 TypeScript 驗證框架，具有社交身份驗證、2FA、金鑰、組織、RBAC 和 15 多個插件。支援 Drizzle/Prisma/Kysely 適配器。 Clerk/Auth.js 的自架替代方案。

在以下情況下使用：配置身份驗證、新增外掛程式、社交 OAuth、多租戶 SaaS、具有團隊和 RBAC 的組織、雙重認證（TOTP/OTP/備份代碼）、電子郵件驗證、密碼重設流程、會話管理、速率限制、CSRF 和 cookie 安全性、Expo/mobile、D1 適配器錯誤、TanStack Start 外掛程式遷移、從 Supabase Auth 遷移或故障排除身份驗證問題。
