使用 uv（快速 Python 包和项目管理器）进行现代 Python 开发。涵盖项目管理（uv init、uv add、uvsync、uv lock）、虚拟环境、Python 版本管理（uv python install/pin）、脚本运行器（uv run）、工具管理（uvx）、monorepos 的工作区支持以及发布到 PyPI。包括 FastAPI、Pydantic、async/await、类型检查、pytest、structlog 和 CLI 工具的 Python 模式。

在初始化 Python 项目、使用 uv 管理依赖项、配置 pyproject.toml、设置虚拟环境、运行脚本、管理 Python 版本、使用工作区构建 monorepos、容器化 Python 应用程序或使用类型提示编写现代 Python 时使用。
