Sviluppo Python moderno con uv, il pacchetto Python veloce e project manager. Copre la gestione dei progetti (uv init, uv add, uv sync, uv lock), ambienti virtuali, gestione delle versioni Python (uv python install/pin), script runner (uv run), gestione degli strumenti (uvx), supporto dell'area di lavoro per monorepos e pubblicazione su PyPI. Include modelli Python per FastAPI, Pydantic, async/await, controllo del tipo, pytest, structlog e strumenti CLI.

Da utilizzare durante l'inizializzazione di progetti Python, la gestione delle dipendenze con uv, la configurazione di pyproject.toml, la configurazione di ambienti virtuali, l'esecuzione di script, la gestione di versioni Python, la creazione di monorepos con spazi di lavoro, la creazione di contenitori di app Python o la scrittura di Python moderno con suggerimenti sul tipo.
