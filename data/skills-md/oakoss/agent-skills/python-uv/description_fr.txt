Développement Python moderne avec uv, le package Python rapide et le chef de projet. Couvre la gestion de projet (uv init, uv add, uv sync, uv lock), les environnements virtuels, la gestion des versions Python (uv python install/pin), les exécuteurs de script (uv run), la gestion des outils (uvx), la prise en charge de l'espace de travail pour monorepos et la publication sur PyPI. Inclut des modèles Python pour les outils FastAPI, Pydantic, async/await, vérification de type, pytest, structlog et CLI.

À utiliser lors de l'initialisation de projets Python, de la gestion des dépendances avec uv, de la configuration de pyproject.toml, de la configuration d'environnements virtuels, de l'exécution de scripts, de la gestion des versions Python, de la création de monorepos avec des espaces de travail, de la conteneurisation d'applications Python ou de l'écriture de Python moderne avec des astuces de type.
