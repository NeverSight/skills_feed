Современная разработка на Python с помощью uv, быстрого пакета Python и менеджера проектов. Охватывает управление проектами (uv init, uv add, uv sync, uv lock), виртуальные среды, управление версиями Python (uv python install/pin), запуск скриптов (uv run), управление инструментами (uvx), поддержку рабочего пространства для монорепозиториев и публикацию в PyPI. Включает шаблоны Python для инструментов FastAPI, Pydantic, async/await, проверки типов, pytest, structlog и CLI.

Используйте при инициализации проектов Python, управлении зависимостями с помощью uv, настройке pyproject.toml, настройке виртуальных сред, запуске сценариев, управлении версиями Python, создании монорепозиториев с рабочими областями, контейнеризации приложений Python или написании современного Python с подсказками типов.
