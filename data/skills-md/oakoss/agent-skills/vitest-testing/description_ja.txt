Vitest 単体テスト、コンポーネント テスト、および統合テスト用のテスト フレームワーク。テスト構造 (describe/it/test.each)、expect を使用したアサーション、ライフサイクル フック、モック (vi.fn/vi.mock/vi.spyOn)、React テスト ライブラリ パターン、renderHook を使用したフック テスト、ユーザー イベント インタラクション、非同期ユーティリティ、構成、ワークスペース セットアップ、カバレッジ、およびソース内テストや同時実行などの高度なパターンをカバーします。 CLI コマンド、テスト フィルタリング、タグ、シャーディング、test.extend を使用したフィクスチャ、およびフィクスチャのスコープについても説明します。

単体テスト、コンポーネント テスト、統合テスト、または React フックをテストするときに使用します。テスト構成、モジュールのモック化、非同期動作のテスト、スナップショット テスト、またはテスト カバレッジの設定に使用します。
