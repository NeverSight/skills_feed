Cadre de test Vitest pour les tests unitaires, de composants et d'intégration. Couvre la structure de test (describe/it/test.each), les assertions avec expect, les hooks de cycle de vie, les moqueries (vi.fn/vi.mock/vi.spyOn), les modèles de bibliothèque de tests React, les tests de hook avec renderHook, les interactions utilisateur-événement, les utilitaires asynchrones, la configuration, la configuration de l'espace de travail, la couverture et les modèles avancés tels que les tests dans la source et l'exécution simultanée. Couvre également les commandes CLI, le filtrage des tests, les balises, le partitionnement, les appareils avec test.extend et la portée des appareils.

À utiliser lors de l'écriture de tests unitaires, de tests de composants, de tests d'intégration ou de tests de hooks React. À utiliser pour la configuration des tests, les modules moqueurs, le test du comportement asynchrone, les tests d'instantanés ou la configuration de la couverture des tests.
