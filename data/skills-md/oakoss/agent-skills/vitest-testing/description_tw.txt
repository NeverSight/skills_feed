用於單元、元件和整合測試的 Vitest 測試框架。涵蓋測試結構 (describe/it/test.each)、expect 斷言、生命週期鉤子、模擬 (vi.fn/vi.mock/vi.spyOn)、React 測試庫模式、renderHook 鉤子測試、用戶事件交互、非同步實用程式、配置、工作區設定、覆蓋率以及高級模式（如來源內測試和並行執行）。還涵蓋 CLI 命令、測試過濾、標籤、分片、帶有 test.extend 的固定裝置以及固定裝置作用域。

在編寫單元測試、元件測試、整合測試或測試 React hook 時使用。用於測試配置、模擬模組、測試異步行為、快照測試或設定測試覆蓋率。
