React Native 与 Expo 框架一起使用单个 TypeScript 代码库在 Android、iOS 和 Web 上构建本机移动应用程序。涵盖基于 Expo Router 文件的导航（堆栈、选项卡、抽屉）、用于 CI/CD 和 OTA 更新的 EAS 构建/提交/更新、用于自定义开发构建的 expo-dev-client、用于本机 Swift/Kotlin 模块的 Expo Modules API、核心 RN 模式（StyleSheet、FlatList、特定于平台的代码）、本机设备 API（相机、通知、触觉）以及用于 Tailwind CSS 样式的 NativeWind。

在使用 Expo 构建 React Native 应用程序、配置 Expo Router 导航、设置 EAS Build 管道、实施 OTA 更新、创建本机模块或集成相机和推送通知等设备 API 时使用。
