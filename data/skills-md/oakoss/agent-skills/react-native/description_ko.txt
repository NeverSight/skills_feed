단일 TypeScript 코드베이스로 Android, iOS 및 웹에서 기본 모바일 앱을 구축하기 위한 Expo 프레임워크를 사용하여 React Native를 사용하세요. Expo Router 파일 기반 탐색(스택, 탭, 서랍), CI/CD 및 OTA 업데이트를 위한 EAS 빌드/제출/업데이트, 맞춤형 개발 빌드를 위한 expo-dev-client, 기본 Swift/Kotlin 모듈을 위한 Expo 모듈 API, 핵심 RN 패턴(StyleSheet, FlatList, 플랫폼별 코드), 기본 장치 API(카메라, 알림, 햅틱) 및 Tailwind CSS 스타일링을 위한 NativeWind를 다룹니다.

Expo를 사용하여 React Native 앱을 구축하고, Expo Router 탐색을 구성하고, EAS 빌드 파이프라인을 설정하고, OTA 업데이트를 구현하고, 네이티브 모듈을 생성하거나, 카메라 및 푸시 알림과 같은 장치 API를 통합할 때 사용합니다.
