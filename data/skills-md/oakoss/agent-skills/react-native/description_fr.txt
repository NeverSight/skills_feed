React Native avec le framework Expo pour créer des applications mobiles natives sur Android, iOS et Web à partir d'une seule base de code TypeScript. Couvre la navigation basée sur les fichiers Expo Router (pile, onglets, tiroir), EAS Build/Submit/Update pour les mises à jour CI/CD et OTA, expo-dev-client pour les versions de développement personnalisées, l'API Expo Modules pour les modules natifs Swift/Kotlin, les modèles RN de base (StyleSheet, FlatList, code spécifique à la plate-forme), les API de périphérique natives (caméra, notifications, haptique) et NativeWind pour le style CSS Tailwind.

À utiliser lors de la création d'applications React Native avec Expo, de la configuration de la navigation Expo Router, de la configuration des pipelines EAS Build, de la mise en œuvre de mises à jour OTA, de la création de modules natifs ou de l'intégration d'API d'appareil telles que la caméra et les notifications push.
