React Native 與 Expo 框架一起使用單一 TypeScript 程式碼庫在 Android、iOS 和 Web 上建立本機行動應用程式。涵蓋基於 Expo Router 檔案的導航（堆疊、選項卡、抽屜）、用於 CI/CD 和 OTA 更新的 EAS 建置/提交/更新、用於自訂開發建置的 expo-dev-client、用於本機 Swift/Kotlin 模組的 Expo Modules API、電腦 RN 模式（Styleet、FlatList NativeWind。

在使用 Expo 建立 React Native 應用程式、設定 Expo Router 導航、設定 EAS Build 管道、實作 OTA 更新、建立本機模組或整合相機和推播通知等裝置 API 時使用。
