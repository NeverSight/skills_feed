React Native con il framework Expo per creare app mobili native su Android, iOS e Web da un'unica codebase TypeScript. Copre la navigazione basata su file di Expo Router (stack, schede, drawer), EAS Build/Submit/Update per aggiornamenti CI/CD e OTA, expo-dev-client per build di sviluppo personalizzate, API Expo Modules per moduli Swift/Kotlin nativi, modelli RN core (StyleSheet, FlatList, codice specifico della piattaforma), API native del dispositivo (fotocamera, notifiche, aptica) e NativeWind per lo stile CSS Tailwind.

Da utilizzare durante la creazione di app React Native con Expo, la configurazione della navigazione Expo Router, l'impostazione di pipeline EAS Build, l'implementazione di aggiornamenti OTA, la creazione di moduli nativi o l'integrazione di API del dispositivo come fotocamera e notifiche push.
