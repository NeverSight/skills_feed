React Native с платформой Expo для создания собственных мобильных приложений для Android, iOS и Интернета из единой базы кода TypeScript. Охватывает навигацию на основе файлов Expo Router (стек, вкладки, ящик), сборку/отправку/обновление EAS для обновлений CI/CD и OTA, expo-dev-client для пользовательских сборок разработки, API модулей Expo для собственных модулей Swift/Kotlin, основные шаблоны RN (StyleSheet, FlatList, код, специфичный для платформы), собственные API-интерфейсы устройств (камера, уведомления, тактильные ощущения) и NativeWind для стилей CSS Tailwind.

Используйте при создании приложений React Native с помощью Expo, настройке навигации Expo Router, настройке конвейеров EAS Build, внедрении обновлений OTA, создании собственных модулей или интеграции API-интерфейсов устройств, таких как камера и push-уведомления.
