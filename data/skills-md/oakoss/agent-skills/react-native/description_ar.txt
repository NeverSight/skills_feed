إطار عمل React Native with Expo لإنشاء تطبيقات الهاتف المحمول الأصلية على Android وiOS والويب من قاعدة تعليمات برمجية واحدة لـ TypeScript. يغطي التنقل المستند إلى ملف Expo Router (المكدس، وعلامات التبويب، والدرج)، وإنشاء/إرسال/تحديث EAS لتحديثات CI/CD وOTA، وعميل expo-dev لبنيات التطوير المخصصة، وواجهة برمجة تطبيقات Expo Modules لوحدات Swift/Kotlin الأصلية، وأنماط RN الأساسية (StyleSheet، وFlatList، والتعليمات البرمجية الخاصة بالنظام الأساسي)، وواجهات برمجة التطبيقات الأصلية للجهاز (الكاميرا، والإشعارات، واللمسات)، وNativeWind لتصميم Tailwind CSS.

يُستخدم عند إنشاء تطبيقات React Native باستخدام Expo، أو تكوين التنقل في Expo Router، أو إعداد خطوط أنابيب EAS Build، أو تنفيذ تحديثات OTA، أو إنشاء وحدات أصلية، أو دمج واجهات برمجة تطبيقات الجهاز مثل الكاميرا وإشعارات الدفع.
