Разрабатывает рабочие процессы git, охватывающие стратегии ветвления, магистральную разработку, многоуровневые изменения, обычные фиксации, конвейеры CI/CD и гигиену репозитория. Используйте при настройке моделей ветвления, написании сообщений о фиксации, настройке действий GitHub, управлении сложенными PR, очистке устаревших ветвей, создании шаблонов задач или восстановлении потерянных коммитов.
