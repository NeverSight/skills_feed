تصميمات سير عمل git التي تغطي إستراتيجيات التفرع، والتطوير القائم على الجذع، والتغييرات المكدسة، والالتزامات التقليدية، وخطوط أنابيب CI/CD، ونظافة المستودع. يُستخدم عند إعداد نماذج التفريع، أو كتابة رسائل الالتزام، أو تكوين إجراءات GitHub، أو إدارة العلاقات العامة المكدسة، أو تنظيف الفروع القديمة، أو إنشاء قوالب المشكلات، أو استعادة الالتزامات المفقودة.
