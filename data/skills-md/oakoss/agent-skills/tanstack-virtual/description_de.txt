TanStack Virtual zur Virtualisierung großer Listen, Raster und Tabellen. Verwenden Sie diese Option, wenn Sie Tausende von Zeilen rendern, einen unendlichen Bildlauf implementieren oder die Anzeige großer Datenmengen optimieren. Verwendung für Virtual, Virtualize, Virtual-List, Virtual-Scroll, UseVirtualizer, Infinite-Scroll, Windowing.
