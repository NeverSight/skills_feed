Erkennt und installiert Agentenfähigkeiten aus dem offenen Ökosystem. Bereichert Suchergebnisse mit Beschreibungen, die von skills.sh abgerufen werden.

Verwenden Sie diese Option, wenn ein Benutzer fragt: „Wie mache ich X“, „Finde einen Skill für X“, „Gibt es einen Skill, der …“ oder die Agentenfunktionen erweitern möchte. Zum Erkennen von Fertigkeiten, zur Fertigkeitssuche, zur Fertigkeitsinstallation und zum Auffinden von Werkzeugen.
