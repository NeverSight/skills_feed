React 用の TanStack Table v8 ヘッドレス データ テーブル。列の定義、並べ替え、フィルタリング (ファジー/ファセット)、TanStack Query によるサーバー側のページネーション、無限スクロール、仮想化 (TanStack Virtual)、列/行の固定、行の展開/グループ化、列のサイズ変更、再利用可能な Shadcn スタイルのコンポーネントについて説明します。無限の再レンダリング、React コンパイラーの非互換性、サーバー側の状態の不一致など、文書化された 15 件のエラーを防止します。

データ テーブルの構築、テーブルのパフォーマンスの修正、サーバー側のページネーションの実装、フィルタリング/並べ替えの追加、またはテーブル状態の問題のデバッグを行うときに使用します。
