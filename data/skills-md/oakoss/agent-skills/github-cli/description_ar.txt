سير عمل GitHub CLI للمستودعات والمشكلات وطلبات السحب والإجراءات والإصدارات والمشاريع واستدعاءات واجهة برمجة التطبيقات. يُستخدم عند إنشاء العلاقات العامة، أو مراجعة التعليمات البرمجية، أو فرز المشكلات، أو تشغيل سير العمل، أو نشر الإصدارات، أو إدارة المشاريع، أو التحقق من الشهادات، أو الاستعلام عن واجهة برمجة تطبيقات GitHub. يستخدم لـ gh cli، وأتمتة github، ومراجعة التعليمات البرمجية، وإدارة الإصدار.
