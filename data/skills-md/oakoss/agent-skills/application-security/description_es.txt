Seguridad integral de aplicaciones que cubre modelado de amenazas (STRIDE), OWASP Top 10 (2025), OWASP API Security Top 10 (2023), revisión de codificación segura, patrones de autenticación/autorización, validación de entradas, cifrado, encabezados de seguridad, seguridad de la cadena de suministro, cumplimiento (GDPR/HIPAA/SOC2/PCI-DSS) y monitoreo de seguridad. Úselo al revisar código en busca de vulnerabilidades, implementar patrones de autenticación, proteger API, configurar encabezados de seguridad, fortalecer la cadena de suministro, prevenir ataques de inyección o prepararse para auditorías de cumplimiento.
