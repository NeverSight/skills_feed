Sicurezza applicativa completa che copre la modellazione delle minacce (STRIDE), OWASP Top 10 (2025), OWASP API Security Top 10 (2023), revisione della codifica sicura, modelli di autenticazione/autorizzazione, convalida dell'input, crittografia, intestazioni di sicurezza, sicurezza della catena di fornitura, conformità (GDPR/HIPAA/SOC2/PCI-DSS) e monitoraggio della sicurezza. Da utilizzare durante la revisione del codice per individuare eventuali vulnerabilità, l'implementazione di modelli di autenticazione, la protezione delle API, la configurazione di intestazioni di sicurezza, il rafforzamento della catena di fornitura, la prevenzione di attacchi di injection o la preparazione per gli audit di conformità.
