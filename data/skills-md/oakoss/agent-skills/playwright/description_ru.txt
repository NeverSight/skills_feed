API автоматизации браузера драматурга, парсинг веб-страниц и инструменты. Охватывает стратегии локатора, утверждения, тестирование API, скрытый режим, обход защиты от ботов, сеансы с аутентификацией, снимки экрана и PDF-файлы, развертывание Docker, настройку, отладку и интеграцию MCP с агентами ИИ. Предотвращает документированные ошибки, включая зависания тайм-аута CI, сбои тестирования расширений и проблемы с навигацией.

Используйте при автоматизации браузеров, очистке защищенных сайтов, обходе обнаружения ботов, создании снимков экрана или PDF-файлов, настройке теста Playwright, устранении ошибок Playwright или изучении шаблонов API Playwright. Для архитектуры тестирования E2E, объектных моделей страниц, стратегий сегментирования CI или шаблонов организации тестирования вместо этого используйте навыки e2e-тестирования.
