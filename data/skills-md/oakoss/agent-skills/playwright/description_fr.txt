API d'automatisation du navigateur Playwright, web scraping et outils. Couvre les stratégies de localisation, les assertions, les tests d'API, le mode furtif, le contournement anti-bot, les sessions authentifiées, les captures d'écran/PDF, le déploiement Docker, la configuration, le débogage et l'intégration MCP avec les agents IA. Empêche les erreurs documentées, notamment les blocages du délai d'expiration du CI, les échecs des tests d'extension et les problèmes de navigation.

À utiliser pour automatiser les navigateurs, supprimer des sites protégés, contourner la détection des robots, générer des captures d'écran/PDF, configurer Playwright Test, dépanner les erreurs Playwright ou apprendre les modèles d'API Playwright. Pour l'architecture de test E2E, les modèles d'objet de page, les stratégies de partitionnement CI ou les modèles d'organisation de test, utilisez plutôt la compétence e2e-testing.
