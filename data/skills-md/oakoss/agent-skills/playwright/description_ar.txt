واجهة برمجة تطبيقات أتمتة متصفح الكاتب المسرحي، وتجريد الويب، والأدوات. يغطي استراتيجيات تحديد المواقع، والتأكيدات، واختبار واجهة برمجة التطبيقات، ووضع التخفي، وتجاوز برامج مكافحة الروبوتات، والجلسات المعتمدة، ولقطات الشاشة/ملفات PDF، ونشر Docker، والتكوين، وتصحيح الأخطاء، وتكامل MCP مع وكلاء الذكاء الاصطناعي. يمنع الأخطاء الموثقة بما في ذلك توقف مهلة CI، وفشل اختبار الامتداد، ومشكلات التنقل.

يُستخدم عند أتمتة المتصفحات، أو استخراج المواقع المحمية، أو تجاوز اكتشاف الروبوتات، أو إنشاء لقطات شاشة/ملفات PDF، أو تكوين اختبار Playwright، أو استكشاف أخطاء Playwright وإصلاحها، أو تعلم أنماط Playwright API. بالنسبة لبنية اختبار E2E، أو نماذج كائنات الصفحة، أو استراتيجيات تقسيم CI، أو أنماط تنظيم الاختبار، استخدم مهارة اختبار e2e بدلاً من ذلك.
