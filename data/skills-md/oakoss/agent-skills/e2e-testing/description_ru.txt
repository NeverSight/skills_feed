Архитектура и шаблоны E2E-тестирования с Playwright. Используйте при разработке наборов тестов, структурировании объектных моделей страниц, планировании стратегий сегментирования CI, настройке потоков аутентификации или организации тестов с помощью тегов и аннотаций. Используйте для тестовой архитектуры, аудита доступности с помощью Axe-Core, стратегий сетевого макетирования, рабочих процессов визуальной регрессии, воспроизведения HAR и шаблонов аутентификации StorageState. Для получения сведений об API драматурга, автоматизации браузера или веб-скрапинга используйте вместо этого навык драматурга.
