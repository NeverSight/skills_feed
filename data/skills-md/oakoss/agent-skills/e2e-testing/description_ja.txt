Playwright を使用した E2E テスト アーキテクチャとパターン。テスト スイートの設計、ページ オブジェクト モデルの構築、CI シャーディング戦略の計画、認証フローの設定、またはタグと注釈を使用したテストの整理に使用します。テスト アーキテクチャ、axe-core によるアクセシビリティ監査、ネットワーク モッキング戦略、視覚的回帰ワークフロー、HAR リプレイ、および storageState 認証パターンに使用します。 Playwright API の詳細、ブラウザーの自動化、または Web スクレイピングについては、代わりに Playwright スキルを使用してください。
