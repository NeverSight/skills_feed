Создает трехмерные веб-интерфейсы с помощью Three.js, рендеринга WebGPU, шейдеров TSL и React Three Fiber. Охватывает настройку сцены, оптимизацию ресурсов, настройку производительности и интеграцию React 19. Используйте при создании 3D-сцен, настройке рендеринга WebGPU, написании шейдеров TSL, оптимизации вызовов отрисовки или интеграции Three.js с React и Next.js.
