TanStack Query v5 Serverstatusverwaltung für React. Behandelt Abfrage-/Mutationsmuster, v4-zu-v5-Migration (Objektsyntax, gcTime, isPending, keepPreviousData), optimistische Updates über useMutationState, SSR/Hydration mit HydrationBoundary, unendliche Abfragen, Offline-/PWA-Unterstützung, Fehlergrenzen mit throwOnError und React 19 Suspense-Integration.

Wird zum Erstellen von Datenabrufen, zum Beheben von Migrationsfehlern, zum Debuggen von Hydratationsinkongruenzen, zum Implementieren von Caching-Strategien oder zum Konfigurieren von Mutationen verwendet.
