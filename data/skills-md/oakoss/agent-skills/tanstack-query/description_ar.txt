إدارة حالة خادم TanStack Query v5 لـ React. يغطي أنماط الاستعلام/الطفرة، والترحيل من الإصدار 4 إلى الإصدار 5 (بناء جملة الكائن، وgcTime، وisPending، وkeepPreviousData)، والتحديثات المتفائلة عبر useMutationState، وSSR/hydration مع HydrationBoundary، والاستعلامات اللانهائية، ودعم دون اتصال/PWA، وحدود الأخطاء مع throwOnError، وتكامل React 19 Suspense.

يُستخدم عند إنشاء جلب البيانات، أو إصلاح أخطاء الترحيل، أو تصحيح عدم تطابق الماء، أو تنفيذ إستراتيجيات التخزين المؤقت، أو تكوين الطفرات.
