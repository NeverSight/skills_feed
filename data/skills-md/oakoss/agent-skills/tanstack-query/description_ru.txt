Управление состоянием сервера TanStack Query v5 для React. Охватывает шаблоны запросов/мутаций, миграцию с версии 4 на версию 5 (синтаксис объекта, gcTime, isPending, KeepPreviousData), оптимистичные обновления с помощью useMutationState, SSR/гидратацию с помощью HydrationBoundary, бесконечные запросы, поддержку автономного режима/PWA, границы ошибок с помощью throwOnError и интеграцию React 19 Suspense.

Используйте при построении выборки данных, исправлении ошибок миграции, отладке несоответствий гидратации, реализации стратегий кэширования или настройке мутаций.
