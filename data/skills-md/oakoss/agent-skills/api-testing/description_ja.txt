supertest、MSW、Vitest を使用した API テスト パターン。 REST API の統合テスト、Mock Service Worker v2 による HTTP リクエスト モック、レスポンス アサーション、スキーマ検証、テスト組織、Express、Fastify、および Hono のフレームワーク固有のパターンをカバーします。

REST API の統合テストを作成する場合、HTTP リクエストをモックする場合、または API エンドポイントをテストする場合に使用します。 api-test、supertest、msw、mock-service-worker、integration-test、http-mock、endpoint-test、request-test に使用します。
