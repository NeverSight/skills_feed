Создает доступные, настраиваемые библиотеки компонентов с помощью shadcn/ui с пользовательским интерфейсом Radix или базовым пользовательским интерфейсом, Tailwind CSS 4 и React 19. Охватывает владение компонентами, темы CSS oklch, типобезопасные формы с Field и Zod, рабочие процессы CLI и шаблоны реестра. Используйте при добавлении компонентов shadcn/ui, настройке тем, создании форм с помощью Zod, создании пользовательских реестров или составлении доступных вариантов компонентов. Используйте для интерфейса командной строки Shadcn, темного режима, вариантов компонентов, проверки формы.
