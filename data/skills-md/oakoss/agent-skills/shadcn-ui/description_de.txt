Erstellt zugängliche, anpassbare Komponentenbibliotheken mithilfe von shadcn/ui mit Radix UI oder Base UI, Tailwind CSS 4 und React 19. Behandelt Komponentenbesitz, oklch CSS-Design, typsichere Formulare mit Field und Zod, CLI-Workflows und Registrierungsmuster. Wird zum Hinzufügen von Shadcn/UI-Komponenten, zum Konfigurieren von Themen, zum Erstellen von Formularen mit Zod, zum Erstellen benutzerdefinierter Registrierungen oder zum Erstellen barrierefreier Komponentenvarianten verwendet. Verwendung für Shadcn-CLI, Dunkelmodus, Komponentenvarianten, Formularvalidierung.
