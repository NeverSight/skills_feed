ينشئ مكتبات مكونات يمكن الوصول إليها وقابلة للتخصيص باستخدام shadcn/ui مع Radix UI أو Base UI وTailwind CSS 4 وReact 19. ويغطي ملكية المكونات وموضوعات oklch CSS والنماذج الآمنة من النوع مع Field وZod وسير عمل CLI وأنماط التسجيل. يُستخدم عند إضافة مكونات shadcn/ui، أو تكوين السمات، أو إنشاء النماذج باستخدام Zod، أو إنشاء سجلات مخصصة، أو إنشاء متغيرات مكونات يمكن الوصول إليها. يستخدم لـ shadcn CLI، والوضع المظلم، ومتغيرات المكونات، والتحقق من صحة النموذج.
