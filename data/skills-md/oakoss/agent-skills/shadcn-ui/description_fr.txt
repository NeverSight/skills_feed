Crée des bibliothèques de composants accessibles et personnalisables à l'aide de shadcn/ui avec Radix UI ou Base UI, Tailwind CSS 4 et React 19. Couvre la propriété des composants, la thématisation CSS oklch, les formulaires de type sécurisé avec Field et Zod, les flux de travail CLI et les modèles de registre. À utiliser lors de l'ajout de composants shadcn/ui, de la configuration de thèmes, de la création de formulaires avec Zod, de la création de registres personnalisés ou de la composition de variantes de composants accessibles. À utiliser pour la CLI shadcn, le mode sombre, les variantes de composants, la validation de formulaire.
