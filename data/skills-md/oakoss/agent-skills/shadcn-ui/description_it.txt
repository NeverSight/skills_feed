Crea librerie di componenti accessibili e personalizzabili utilizzando shadcn/ui con Radix UI o Base UI, Tailwind CSS 4 e React 19. Copre la proprietà dei componenti, temi CSS oklch, moduli indipendenti dai tipi con Field e Zod, flussi di lavoro CLI e modelli di registro. Da utilizzare quando si aggiungono componenti shadcn/ui, si configurano temi, si creano moduli con Zod, si creano registri personalizzati o si compongono varianti di componenti accessibili. Utilizzare per CLI shadcn, modalità oscura, varianti di componenti, convalida del modulo.
