Хранилище данных Valkey (совместимое с Redis) в памяти для кэширования, хранения сеансов, публикации/подписки, очередей и распределенных блокировок. Используйте при настройке кэширования, совместимого с Valkey или Redis, выборе политик вытеснения, реализации ограничения скорости, настройке публикации/подписки или потоков, написании служб Docker Compose, переходе с Redis или настройке производительности. Используйте для valkey, redis, кэша, сеанса, публикации/подписки, потоков, распределенной блокировки, ограничения скорости, отсортированного набора.
