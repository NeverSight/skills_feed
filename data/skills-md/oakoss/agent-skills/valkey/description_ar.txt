مخزن بيانات في الذاكرة Valkey (متوافق مع Redis) للتخزين المؤقت وتخزين الجلسة والنشر/الفرعي وقوائم الانتظار والأقفال الموزعة. يُستخدم عند تكوين التخزين المؤقت المتوافق مع Valkey أو Redis، أو اختيار سياسات الإخلاء، أو تنفيذ تحديد المعدل، أو إعداد pub/sub أو التدفقات، أو كتابة خدمات Docker Compose، أو الترحيل من Redis، أو ضبط الأداء. يستخدم لـ valkey، وredis، وذاكرة التخزين المؤقت، والجلسة، والنشر/الفرعي، والتدفقات، والقفل الموزع، وحد المعدل، والمجموعة المصنفة.
