Reagieren Sie auf Fehlergrenzen und Fallback-Benutzeroberflächen, um Renderingfehler zu erkennen. Verwenden Sie diese Option, wenn Sie Komponentenabstürze behandeln, Fehlerzustände anzeigen, eine Fehlerbehebung implementieren oder ganzseitige Abstürze verhindern. Verwendung für ErrorBoundary, ComponentDidCatch, getDerivedStateFromError, Fehler-Fallback, Fehlerwiederherstellung, Absturzbehandlung, React-Error-Boundary-Bibliothek.
