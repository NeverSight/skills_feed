Modèles TypeScript avancés pour le développement en mode strict. Couvre les utilitaires de type (Pick, Omit, Partial, Record, Awaited), les génériques avec contraintes et inférences, les protections de type et le rétrécissement, les unions discriminées, les types conditionnels et mappés, les types littéraux de modèles, les assertions const, l'opérateur satisfait, les modèles de module et les idiomes JavaScript modernes (eslint-plugin-unicorn). À utiliser lors de la création d'API de type sécurisé, en évitant les erreurs d'exécution via les types, en travaillant avec une configuration TypeScript stricte, en débogant des erreurs de type complexes ou en appliquant des idiomes JS modernes. À utiliser pour les génériques, les gardes de types, les types utilitaires, le mode strict, l'inférence de type, le rétrécissement, la sécurité des types, les assertions const, les satisfactions, l'augmentation de module, la licorne, for-of, modern-js.
