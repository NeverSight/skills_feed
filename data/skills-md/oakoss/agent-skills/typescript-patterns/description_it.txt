Pattern TypeScript avanzati per lo sviluppo in modalità rigorosa. Copre le utilità di tipo (Pick, Omit, Partial, Record, Awaited), generici con vincoli e inferenza, protezioni e restringimenti dei tipi, unioni discriminate, tipi condizionali e mappati, tipi letterali modello, asserzioni const, operatore soddisfa, modelli di moduli e idiomi JavaScript moderni (eslint-plugin-unicorn). Da utilizzare durante la creazione di API indipendenti dai tipi, la prevenzione di errori di runtime attraverso i tipi, l'utilizzo di una configurazione TypeScript rigorosa, il debug di errori di tipo complessi o l'applicazione di idiomi JS moderni. Utilizzare per generici, guardie di tipo, tipi di utilità, modalità rigorosa, inferenza di tipo, restringimento, sicurezza del tipo, asserzioni const, soddisfa, aumento del modulo, unicorn, for-of, modern-js.
