أنماط TypeScript المتقدمة لتطوير الوضع الصارم. يغطي أدوات الكتابة المساعدة (اختيار، حذف، جزئي، تسجيل، انتظار)، الأدوية العامة ذات القيود والاستدلال، حراس النوع والتضييق، الاتحادات التمييزية، الأنواع الشرطية والمعينة، الأنواع الحرفية للقالب، تأكيدات const، المشغل المُرضي، أنماط الوحدة النمطية، ومصطلحات JavaScript الحديثة (eslint-plugin-unicorn). يُستخدم عند إنشاء واجهات برمجة تطبيقات آمنة للنوع، أو منع أخطاء وقت التشغيل من خلال الأنواع، أو العمل مع تكوين TypeScript الصارم، أو تصحيح أخطاء الكتابة المعقدة، أو فرض مصطلحات JS الحديثة. يُستخدم للأدوية العامة، ونوع الحراس، وأنواع المرافق، والوضع الصارم، واستدلال النوع، والتضييق، وسلامة النوع، والتأكيدات الثابتة، والرضا، وزيادة الوحدة النمطية، ويونيكورن، و for-of، و js الحديثة.
