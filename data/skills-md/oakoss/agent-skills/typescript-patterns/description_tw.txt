用於嚴格模式開發的高階 TypeScript 模式。涵蓋類型實用程式（Pick、Omit、Partial、Record、Awaited）、具有約束和推理的泛型、類型保護和縮小、可區分聯合、條件和映射類型、模板文字類型、const 斷言、滿足運算符、模組模式和現代 JavaScript 習慣用法 (eslint-plugin-unicorn)。在建置類型安全的 API、透過類型防止執行階段錯誤、使用嚴格的 TypeScript 配置、偵錯複雜類型錯誤或強制執行現代 JS 習慣用法時使用。用於泛型、類型保護、實用程式類型、嚴格模式、類型推斷、縮小、類型安全、const 斷言、滿足、模組增強、unicorn、for-of、modern-js。
