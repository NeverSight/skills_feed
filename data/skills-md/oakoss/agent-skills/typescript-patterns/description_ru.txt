Расширенные шаблоны TypeScript для разработки в строгом режиме. Охватывает утилиты типов (Pick, Omit, Partial, Record, Awaited), универсальные шаблоны с ограничениями и выводом, защиту типов и сужение, распознаваемые объединения, условные и отображаемые типы, литеральные типы шаблонов, константные утверждения, оператор удовлетворяет, шаблоны модулей и современные идиомы JavaScript (eslint-plugin-unicorn). Используйте при создании типобезопасных API, предотвращении ошибок во время выполнения с помощью типов, работе со строгой конфигурацией TypeScript, отладке сложных ошибок типов или применении современных идиом JS. Используется для дженериков, защиты типов, служебных типов, строгого режима, вывода типа, сужения, безопасности типов, константных утверждений, удовлетворений, расширения модулей, единорога, for-of, Modern-js.
