Шаблоны мультиагентного проектирования для Claude Code и систем разработки искусственного интеллекта. Охватывает пользовательские субагенты, встроенные субагенты (исследования/планирования/общего назначения), взаимозаменяемые и специализированные агенты, делегирование, оркестрацию, пакетные рабочие процессы, гигиену контекста и восстановление после сбоев.

Используйте при проектировании многоагентных систем, создании пользовательских субагентов, делегировании массовых операций, организации параллельной работы, выборе между взаимозаменяемыми и специализированными агентами, настройке внешнего вида агента или отладке проблем координации агентов.
