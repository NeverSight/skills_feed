Веб-анимация с помощью Motion (ранее Framer Motion) для React — жесты, эффекты прокрутки, пружинная физика, анимация макета, SVG, микровзаимодействия, состояния загрузки.

Используйте, когда: перетаскивание, анимация прокрутки, модальные окна, карусели, параллакс, переходы страниц, эффекты наведения, шахматные списки, загрузка счетчиков, числовые счетчики.
Устранение неполадок: выход из AnimatePresence, производительность списка, конфликты Tailwind, использование клиента Next.js, макет в масштабируемых контейнерах.
