Animations Web avec Motion (anciennement Framer Motion) pour React -- gestes, effets de défilement, physique des ressorts, animations de mise en page, SVG, micro-interactions, états de chargement.

À utiliser dans les cas suivants : glisser-déposer, animations de défilement, modaux, carrousels, parallaxe, transitions de page, effets de survol, listes décalées, chargement de spinners, compteurs numériques.
Dépannage : sortie d'AnimatePresence, performances de la liste, conflits Tailwind, "utilisation du client" Next.js, disposition dans des conteneurs mis à l'échelle.
