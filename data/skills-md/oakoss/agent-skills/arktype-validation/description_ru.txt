Проверка времени выполнения ArkType с использованием собственного синтаксиса TypeScript. Типобезопасные схемы с использованием строковых выражений, морфов, областей действия и дженериков. Включает ArkEnv для типобезопасной проверки переменных среды с автоматическим приведением и плагином Vite. Используйте при определении схем, проверке данных, преобразовании входных данных, создании типобезопасных API с помощью ArkType или проверке переменных среды с помощью ArkEnv.
