Configuration et écosystème de l'outil de build Vite. Couvre la configuration de vite.config.ts, la création de plugins et les plugins populaires, la configuration du serveur de développement (proxy, HMR, HTTPS), l'optimisation de la construction (chunking, tree-shaking, morceaux manuels, rollupOptions), le mode bibliothèque pour la publication de packages, la configuration SSR, les variables d'environnement (gestion .env), les applications multipages, la gestion CSS (PostCSS, modules CSS, préprocesseurs) et la gestion des actifs.

À utiliser lors de la configuration de projets Vite, de la création de plugins Vite, de l'optimisation des builds, de la configuration de proxys de serveur de développement, de la configuration de SSR, de la gestion des variables d'environnement ou du dépannage des problèmes Vite.
