Configurazione ed ecosistema dello strumento di creazione Vite. Copre la configurazione di vite.config.ts, creazione di plugin e plugin popolari, configurazione del server di sviluppo (proxy, HMR, HTTPS), ottimizzazione della build (chunking, tree-shaking, blocchi manuali, rollupOptions), modalità libreria per la pubblicazione di pacchetti, configurazione SSR, variabili di ambiente (gestione di .env), app multipagina, gestione CSS (PostCSS, moduli CSS, preprocessori) e gestione delle risorse.

Da utilizzare durante la configurazione di progetti Vite, la creazione di plug-in Vite, l'ottimizzazione delle build, l'impostazione dei proxy del server di sviluppo, la configurazione di SSR, la gestione delle variabili di ambiente o la risoluzione dei problemi di Vite.
