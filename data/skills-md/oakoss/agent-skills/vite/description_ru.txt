Конфигурация и экосистема инструмента сборки Vite. Охватывает настройку vite.config.ts, создание плагинов и популярные плагины, конфигурацию сервера разработки (прокси, HMR, HTTPS), оптимизацию сборки (разбиение на части, встряхивание деревьев, ручные фрагменты, RollupOptions), библиотечный режим для публикации пакетов, конфигурацию SSR, переменные среды (обработка .env), многостраничные приложения, обработку CSS (PostCSS, модули CSS, препроцессоры) и обработку ресурсов.

Используйте при настройке проектов Vite, создании плагинов Vite, оптимизации сборок, настройке прокси-серверов разработки, настройке SSR, обработке переменных среды или устранении проблем Vite.
