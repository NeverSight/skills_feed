Docker-контейнеризация для разработки и производства. Охватывает файлы Dockerfile, многоэтапные сборки, кэширование слоев, службы Compose, сеть, тома, проверки работоспособности, усиление безопасности и шаблоны производственного развертывания.

Используйте при написании Dockerfiles, оптимизации размера образа, настройке служб Compose, отладке сети контейнеров, настройке проверок работоспособности, усилении защиты контейнеров для производства или устранении проблем с кэшем сборки.
