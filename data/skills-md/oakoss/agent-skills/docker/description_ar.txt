حاويات دوكر للتطوير والإنتاج. يغطي ملفات Dockerfiles والإنشاءات متعددة المراحل والتخزين المؤقت للطبقات وخدمات الإنشاء والشبكات ووحدات التخزين والفحوصات الصحية وتشديد الأمان وأنماط نشر الإنتاج.

يُستخدم عند كتابة ملفات Dockerfiles، أو تحسين حجم الصورة، أو تكوين خدمات الإنشاء، أو تصحيح أخطاء شبكة الحاويات، أو إعداد فحوصات السلامة، أو تقوية الحاويات للإنتاج، أو استكشاف مشكلات ذاكرة التخزين المؤقت للإنشاء وإصلاحها.
