Сквозные типобезопасные API tRPC v11 для TypeScript. Охватывает определения маршрутизатора и процедур, проверку входных данных с помощью Zod, промежуточное программное обеспечение и контекст, подписки, обработку ошибок с помощью TRPCError, клиентские перехваты React через @trpc/react-query, серверные вызовы с помощью createCallerFactory, настройку адаптера (автономный, Express, Fastify, Hono) и шаблоны тестирования.

Используйте при создании маршрутизаторов tRPC, определении процедур, настройке промежуточного программного обеспечения, настройке адаптеров, интеграции tRPC с React, обработке ошибок, реализации подписок или тестировании процедур tRPC.
