tRPC v11 TypeScript 用のエンドツーエンドのタイプセーフ API。ルーターとプロシージャの定義、Zod による入力検証、ミドルウェアとコンテキスト、サブスクリプション、TRPCError によるエラー処理、@trpc/react-query による React クライアント フック、createCallerFactory によるサーバー側呼び出し元、アダプターのセットアップ (スタンドアロン、Express、Fastify、Hono)、およびテスト パターンについて説明します。

tRPC ルーターの構築、プロシージャの定義、ミドルウェアのセットアップ、アダプターの構成、tRPC と React の統合、エラーの処理、サブスクリプションの実装、または tRPC プロシージャのテストを行うときに使用します。
