Обеспечивает целостность нулевого несоответствия между HTML, отображаемым на сервере, и деревьями React на стороне клиента. Используйте при отладке ошибок гидратации, исправлении несоответствий текстового содержимого, обработке загрязнения DOM расширения браузера, реализации выборочной гидратации с помощью Suspense, использовании перехватчика use() React 19 для детерминированных мостов данных между сервером и клиентом или применении кэша использования Next.js для предотвращения дрейфа данных. Используйте для несоответствия гидратации, SSR, гидратРут, подавлениеГидратионВарнинг, onRecoverableError, двухпроходный рендеринг.
