يضمن عدم التطابق التام بين أشجار HTML المقدمة من الخادم وأشجار React من جانب العميل. يُستخدم عند تصحيح أخطاء الترطيب، أو إصلاح عدم تطابق محتوى النص، أو التعامل مع تلوث DOM لامتداد المتصفح، أو تنفيذ الترطيب الانتقائي باستخدام Suspense، أو استخدام خطاف React 19 use() لجسور البيانات الحتمية من الخادم إلى العميل، أو تطبيق Next.js استخدم ذاكرة التخزين المؤقت لمنع انجراف البيانات. يُستخدم في حالات عدم تطابق الترطيب، وSSR، وhydrotRoot، وقمع HydrationWarning، وonRecoverableError، والعرض ثنائي المسار.
