Garantiza una integridad sin coincidencias entre el HTML renderizado por el servidor y los árboles React del lado del cliente. Úselo al depurar errores de hidratación, corregir discrepancias en el contenido del texto, manejar la contaminación DOM de la extensión del navegador, implementar hidratación selectiva con Suspense, usar el gancho use() de React 19 para puentes de datos deterministas de servidor a cliente o aplicar el uso de caché de Next.js para prevenir la deriva de datos. Úselo para falta de coincidencia de hidratación, SSR, hidratoRoot, suprimirHydrationWarning, onRecoverableError, renderizado de dos pasadas.
