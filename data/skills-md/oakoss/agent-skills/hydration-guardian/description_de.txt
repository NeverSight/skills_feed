Gewährleistet eine konfliktfreie Integrität zwischen vom Server gerendertem HTML und clientseitigen React-Bäumen. Verwenden Sie diese Option, wenn Sie Hydratationsfehler debuggen, nicht übereinstimmende Textinhalte beheben, die DOM-Verschmutzung von Browsererweiterungen handhaben, selektive Hydratation mit Suspense implementieren, den React 19 use()-Hook für deterministische Server-zu-Client-Datenbrücken verwenden oder den Next.js-Use-Cache zur Verhinderung von Datendrift anwenden. Verwendung für Hydratationsinkongruenz, SSR, hydrateRoot, suppressHydrationWarning, onRecoverableError, Zwei-Durchgangs-Rendering.
