Управление состоянием Zustand v5 для React. Охватывает хранилища TypeScript с синтаксисом двойных скобок, постоянное промежуточное программное обеспечение с обработкой гидратации, инструменты разработчика, subscribeWithSelector, шаблон срезов для больших приложений, шаблон поставщика SSR/Next.js, промежуточное ПО для погружения, автоматическое создание селекторов, тестирование и ванильные хранилища.

Используйте при настройке глобального состояния, настройке постоянного промежуточного программного обеспечения, реализации шаблона срезов, миграции из Redux или Context, тестировании хранилищ или устранении неполадок гидратации и TypeScript.
