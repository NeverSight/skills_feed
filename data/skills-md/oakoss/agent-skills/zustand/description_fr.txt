Gestion de l'état Zustand v5 pour React. Couvre les magasins TypeScript avec une syntaxe à double parenthèses, le middleware persistant avec gestion de l'hydratation, les outils de développement, SubscribeWithSelector, le modèle de tranches pour les grandes applications, le modèle de fournisseur SSR/Next.js, le middleware immer, les sélecteurs de génération automatique, les tests et les magasins Vanilla.

À utiliser lors de la configuration de l'état global, de la configuration du middleware persistant, de la mise en œuvre d'un modèle de tranches, de la migration depuis Redux ou Context, du test des magasins ou du dépannage des problèmes d'hydratation et de TypeScript.
