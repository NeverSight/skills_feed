Supprime les artefacts d’écriture d’IA de la documentation et du code. À utiliser pour éditer de la prose générée par LLM, réviser des README, peaufiner des documents avant de les publier ou nettoyer le code généré par l'IA. À utiliser pour le nettoyage d'emdash, la suppression de phrases formulées, l'étalonnage des tonalités, le code sur-commenté, la dénomination verbeuse et la détection d'odeur de code AI.
