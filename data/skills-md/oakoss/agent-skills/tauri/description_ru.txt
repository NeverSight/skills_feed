Платформа Tauri v2 для создания настольных и мобильных приложений с веб-интерфейсом и серверной частью Rust. Охватывает настройку проекта, команды и события IPC, управление окнами, панель задач, доступ к файловой системе, систему плагинов (буфер обмена, диалоговое окно, уведомление, средство обновления, глубокая ссылка), многооконные приложения, дополнительные двоичные файлы, пользовательские протоколы, подписание приложений, распространение, автоматические обновления и модель безопасности на основе возможностей.

Используйте при создании проекта Tauri, определении команд Rust, вызове внутренней логики из внешнего интерфейса, настройке разрешений и возможностей, настройке плагинов, управлении состоянием приложения, построении меню на панели задач, создании многооконных макетов, объединении дополнительных двоичных файлов, реализации автоматических обновлений или подписании приложений для распространения.
