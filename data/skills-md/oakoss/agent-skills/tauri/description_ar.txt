إطار عمل Tauri v2 لبناء تطبيقات سطح المكتب والهاتف المحمول مع واجهات الويب الأمامية والواجهات الخلفية لـ Rust. يغطي إعداد المشروع، وأوامر وأحداث IPC، وإدارة النوافذ، وعلبة النظام، والوصول إلى نظام الملفات، ونظام المكونات الإضافية (الحافظة، ومربع الحوار، والإشعارات، والمحدث، والارتباط العميق)، وتطبيقات النوافذ المتعددة، والثنائيات الجانبية، والبروتوكولات المخصصة، وتوقيع التطبيق، والتوزيع، والتحديثات التلقائية، ونموذج الأمان القائم على القدرات.

يُستخدم عند دعم مشروع Tauri، أو تعريف أوامر Rust، أو استدعاء منطق الواجهة الخلفية من الواجهة الأمامية، أو تكوين الأذونات والإمكانيات، أو إعداد المكونات الإضافية، أو إدارة حالة التطبيق، أو إنشاء قوائم علبة النظام، أو إنشاء تخطيطات متعددة النوافذ، أو تجميع الثنائيات الجانبية، أو تنفيذ التحديثات التلقائية، أو توقيع التطبيقات للتوزيع.
