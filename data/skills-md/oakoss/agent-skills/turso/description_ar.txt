منصة SQLite المستضافة على حافة Turso والمبنية على libSQL (شوكة SQLite مفتوحة المصدر). يغطي @libsql/client SDK، والنسخ المتماثلة المضمنة مع المزامنة المحلية، وبنية قاعدة بيانات متعددة لكل مستأجر، وواجهة برمجة تطبيقات النظام الأساسي لتوفير قاعدة البيانات، وترحيل المخطط، والبحث المتجه باستخدام F32_BLOB، والعمليات المجمعة، والمعاملات التفاعلية، والتشفير أثناء الراحة.

يُستخدم عند الاتصال بقواعد بيانات Turso، أو تكوين النسخ المتماثلة المضمنة، أو توفير قواعد البيانات عبر واجهة برمجة تطبيقات النظام الأساسي، أو تنفيذ عزل قاعدة البيانات لكل مستأجر، أو إجراء بحث تشابه المتجهات، أو دمج libSQL مع Drizzle ORM.
