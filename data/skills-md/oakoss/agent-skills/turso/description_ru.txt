Платформа SQLite с периферийным размещением Turso, построенная на libSQL (вилка SQLite с открытым исходным кодом). Охватывает @libsql/client SDK, встроенные реплики с локальной синхронизацией, архитектуру с несколькими базами данных на одного клиента, API платформы для подготовки баз данных, миграцию схем, векторный поиск с помощью F32_BLOB, пакетные операции, интерактивные транзакции и шифрование неактивных данных.

Используйте при подключении к базам данных Turso, настройке встроенных реплик, предоставлении баз данных через API платформы, реализации изоляции базы данных для каждого арендатора, выполнении поиска по сходству векторов или интеграции libSQL с Drizzle ORM.
