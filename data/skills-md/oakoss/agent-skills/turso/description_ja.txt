Turso エッジホスト型 SQLite プラットフォームは、libSQL (オープンソース SQLite フォーク) 上に構築されています。 @libsql/client SDK、ローカル同期を備えた組み込みレプリカ、テナントごとのマルチデータベース アーキテクチャ、データベース プロビジョニング用のプラットフォーム API、スキーマ移行、F32_BLOB によるベクトル検索、バッチ操作、対話型トランザクション、保存時の暗号化をカバーします。

Turso データベースへの接続、埋め込みレプリカの構成、プラットフォーム API を介したデータベースのプロビジョニング、テナントごとのデータベース分離の実装、ベクトル類似性検索の実行、または libSQL と Drizzle ORM の統合を行う場合に使用します。
