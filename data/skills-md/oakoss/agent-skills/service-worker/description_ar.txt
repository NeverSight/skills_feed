دورة حياة عامل الخدمة، واستراتيجيات التخزين المؤقت، ومزامنة الخلفية، ودفع الإشعارات، والتوجيه الثابت لتطبيقات الويب التقدمية. يغطي استراتيجيات Workbox v7.4 (CacheFirst، وNetworkFirst، وStaleWhileRevalidate)، والتخزين المؤقت المسبق لمربع العمل لأغلفة التطبيقات، وواجهة برمجة تطبيقات مزامنة الخلفية مع عمليات احتياطية عبر المستعرضات، وواجهة برمجة تطبيقات Push مع مفاتيح VAPID، وواجهة برمجة تطبيقات التوجيه الثابت لعامل الخدمة (Chrome 123+).

يُستخدم عند تسجيل العاملين في الخدمة، أو تنفيذ التخزين المؤقت دون اتصال، أو تكوين مزامنة الخلفية، أو إضافة إشعارات الدفع، أو تحسين معالجة الجلب باستخدام المسارات الثابتة، أو دمج العاملين في الخدمة مع محركات المزامنة المحلية الأولى.
