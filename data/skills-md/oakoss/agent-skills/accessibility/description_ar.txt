أنشئ واجهات متوافقة مع WCAG 2.2 AA مع HTML الدلالي، وARIA، والتنقل عبر لوحة المفاتيح، وإدارة التركيز، وتباين الألوان، ودعم قارئ الشاشة. يغطي النماذج ومربعات الحوار وعلامات التبويب والمناطق المباشرة وتخطي الارتباطات والنص البديل وجداول البيانات.

يُستخدم عند تنفيذ واجهات مستخدم يمكن الوصول إليها، أو تدقيق امتثال WCAG، أو إصلاح مشكلات قارئ الشاشة، أو التنقل في لوحة المفاتيح، أو مصائد التركيز، أو استكشاف الأخطاء وإصلاحها "مخطط التركيز المفقود"، "يتطلب تسمية aria"، "التباين غير الكافي"، "نص بديل مفقود"، "التسلسل الهرمي للعناوين".
