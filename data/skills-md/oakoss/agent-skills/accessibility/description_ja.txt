セマンティック HTML、ARIA、キーボード ナビゲーション、フォーカス管理、カラー コントラスト、スクリーン リーダーのサポートを備えた WCAG 2.2 AA 準拠のインターフェイスを構築します。フォーム、ダイアログ、タブ、ライブ領域、スキップリンク、代替テキスト、データテーブルをカバーします。

アクセシブルな UI の実装、WCAG コンプライアンスの監査、スクリーン リーダーの問題の修正、キーボード ナビゲーション、フォーカス トラップ、または「フォーカス アウトラインの欠落」、「aria-label が必要」、「コントラスト不足」、「代替テキストの欠落」、「見出し階層」のトラブルシューティングを行う場合に使用します。
