Cree interfaces compatibles con WCAG 2.2 AA con HTML semántico, ARIA, navegación por teclado, gestión de enfoque, contraste de color y compatibilidad con lectores de pantalla. Cubre formularios, cuadros de diálogo, pestañas, regiones activas, enlaces para saltar, texto alternativo y tablas de datos.

Úselo al implementar interfaces de usuario accesibles, auditar el cumplimiento de las WCAG, solucionar problemas del lector de pantalla, navegación con el teclado, trampas de enfoque o solucionar problemas de "falta un esquema de enfoque", "se requiere etiqueta aria", "contraste insuficiente", "falta texto alternativo", "jerarquía de encabezados".
