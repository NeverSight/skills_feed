Создает редакторы форматированного текста с помощью Tiptap, платформы безголового редактора на основе ProseMirror для React с поддержкой Tailwind v4. Охватывает настройку SSR, загрузку изображений, оформление текста, совместное редактирование и поддержку уценки. Используйте при добавлении редактора форматированного текста, настройке расширений Tiptap, обработке загрузки изображений в редакторах или настройке совместного редактирования с помощью Y.js. Используйте для подсказок, форматированного текста, редактора, полузеркала, реакции, попутного ветра.
