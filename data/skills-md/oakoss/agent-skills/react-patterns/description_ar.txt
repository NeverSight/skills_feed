React أكثر من 19 نمطًا وتحسين الأداء وبنية المكونات. يغطي الخطافات وأشجار قرارات إدارة الحالة وجلب البيانات باستخدام واجهة برمجة تطبيقات use() ومكونات الخادم ومترجم React وتحسين الحزمة وإزالة إعادة العرض.

يُستخدم عند إنشاء المكونات، أو تحسين عمليات إعادة العرض، أو جلب البيانات، أو إدارة الحالة، أو التعامل مع النماذج، أو هيكلة الواجهات الأمامية، أو مراجعة كود React.
