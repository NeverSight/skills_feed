Liest Code, den Sie gerade geschrieben haben, aus einer neuen Perspektive erneut, um Bugs, Irrtümer und Probleme zu erkennen. Verwenden Sie es, nachdem Sie eine Funktion abgeschlossen, einen Fehler behoben oder Codeänderungen vorgenommen haben. Löst aus bei „Meinen Code überprüfen“, „Neue Augen“, „Auf Fehler prüfen“, „Habe ich etwas übersehen“ oder „Gesundheitsprüfung“.
