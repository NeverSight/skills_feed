يراجع تعليقات العلاقات العامة من GitHub (مساعد الطيار، المراجعون)، ويقيمها مقابل التعليمات البرمجية الفعلية، ويرد عليها بالاستدلال، ويحل سلاسل الرسائل. مشغلات على "مراجعة تعليقات العلاقات العامة"، أو "تناول تعليقات العلاقات العامة"، أو "إصلاح تعليقات العلاقات العامة"، أو "مراجعة اقتراحات مساعد الطيار".
