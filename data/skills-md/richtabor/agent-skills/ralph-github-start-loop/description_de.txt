Führt eine autonome Schleife zum Abrufen von Storys aus GitHub-Problemen aus. Implementiert und schließt Probleme wie erledigt. Wird ausgelöst bei „Meine PRDs durchgehen“, „An meinen Problemen arbeiten“, „Autonome Schleife starten“, „Meine PRDs implementieren“ oder Anfragen, GitHub-Probleme autonom zu bearbeiten.
