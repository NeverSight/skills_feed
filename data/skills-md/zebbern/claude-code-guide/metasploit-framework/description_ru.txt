Этот навык следует использовать, когда пользователь просит «использовать Metasploit для тестирования на проникновение», «эксплуатировать уязвимости с помощью msfconsole», «создавать полезные нагрузки с помощью msfvenom», «выполнять постэксплуатацию», «использовать вспомогательные модули для сканирования» или «разрабатывать собственные эксплойты». Он предоставляет исчерпывающее руководство по использованию Metasploit Framework в оценках безопасности.
