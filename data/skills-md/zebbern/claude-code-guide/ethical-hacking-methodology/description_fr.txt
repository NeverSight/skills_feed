Cette compétence doit être utilisée lorsque l'utilisateur demande « apprendre le piratage éthique », « comprendre le cycle de vie des tests d'intrusion », « effectuer une reconnaissance », « effectuer une analyse de sécurité », « exploiter les vulnérabilités » ou « rédiger des rapports de tests d'intrusion ». Il fournit une méthodologie et des techniques complètes de piratage éthique.
