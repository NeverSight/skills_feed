Этот навык следует использовать, когда пользователь просит «изучить этический взлом», «понять жизненный цикл тестирования на проникновение», «провести разведку», «провести сканирование безопасности», «эксплуатировать уязвимости» или «написать отчеты о тестах на проникновение». Он предоставляет комплексную методологию и методы этического взлома.
