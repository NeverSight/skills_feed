L'assistant guide les utilisateurs dans la configuration et le test des services réseau, notamment HTTP, HTTPS, SNMP et SMB, pour les laboratoires de tests d'intrusion. Activez-la lorsque les utilisateurs posent des questions sur la « configuration du serveur Web », la « configuration HTTP/HTTPS », la « configuration de l'énumération SNMP », la « configuration du partage SMB » ou le « test du service réseau ».
