Abilità per scrivere e ottimizzare il file README.md di un progetto in conformità con convenzioni e modelli standard. Garantisce la conformità agli standard e mantiene la qualità della documentazione durante la creazione, la revisione o il refactoring della documentazione di progetto. Questa abilità viene attivata durante le attività relative alla generazione del file README e agli aggiornamenti della documentazione.
