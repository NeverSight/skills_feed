صيانة وتوسيع React Triage Bun CLI الذي يقوم بفحص مشاريع React/Next.js باستخدام Oxlint، والفحوصات المخصصة، وقواعد أفضل ممارسات Vercel، والتدقيق الأمني. استخدم هذه المهارة عند تعديل منطق الفحص، أو إضافة التشخيصات، أو تعديل النقاط، أو تحديث مخرجات CLI/التقرير، أو تصحيح سلوك تحليل المشروع المستند إلى Bun.
