À utiliser pour enquêter ou améliorer les performances de WordPress (agent backend uniquement) : profilage et mesure (profil/médecin WP-CLI, synchronisation du serveur, moniteur de requêtes via les en-têtes REST), optimisation de la base de données/requêtes, options chargées automatiquement, mise en cache des objets, cron, appels d'API HTTP et vérification sécurisée.
