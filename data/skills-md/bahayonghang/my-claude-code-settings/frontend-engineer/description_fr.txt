À utiliser lorsque l'utilisateur a besoin d'aide pour : (1) créer de nouveaux composants d'interface utilisateur, (2) améliorer les effets visuels, (3) implémenter des animations et des interactions, (4) une conception réactive, (5) créer des systèmes de conception, (6) créer de belles interfaces sans brouillon de conception. Les mots déclencheurs incluent : « UI », « interface », « composant », « style », « animation », « beau », « embellir », « design » et d'autres demandes visuelles frontales.
