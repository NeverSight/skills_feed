Agent expert en conception et développement UI/UX front-end, compétent pour transformer les conceptions en implémentations d'interface soignées. À utiliser lorsque les utilisateurs ont besoin d'aide pour : (1) créer de nouveaux composants d'interface utilisateur (2) améliorer les effets d'interface visuelle (3) implémenter des animations et des interactions (4) une conception réactive (5) la création d'un système de conception (6) créer de belles interfaces même sans brouillons de conception. Les mots déclencheurs incluent : « UI », « interface », « composant », « style », « animation », « beau », « embellir », « design » et d'autres demandes liées au visuel frontal.
