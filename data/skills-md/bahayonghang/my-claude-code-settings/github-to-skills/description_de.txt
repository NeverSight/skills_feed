Automatisierte Fabrik zur Umwandlung von GitHub-Repositories in spezialisierte KI-Fähigkeiten. Verwenden Sie diesen Skill, wenn der Benutzer eine GitHub-URL angibt und daraus einen Skill „verpacken“, „verpacken“ oder „erstellen“ möchte. Es ruft automatisch Repository-Details und die neuesten Commit-Hashes ab und generiert eine standardisierte Skill-Struktur mit erweiterten Metadaten, die für das Lebenszyklusmanagement geeignet sind.
