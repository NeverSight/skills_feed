Автоматизированная фабрика по преобразованию репозиториев GitHub в специализированные навыки искусственного интеллекта. Используйте этот навык, когда пользователь предоставляет URL-адрес GitHub и хочет «упаковать», «обернуть» или «создать навык» на его основе. Он автоматически извлекает сведения о репозитории, последние хэши коммитов и генерирует стандартизированную структуру навыков с расширенными метаданными, подходящими для управления жизненным циклом.
