Agente orquestador jefe para planificar, delegar y ejecutar tareas complejas en paralelo. Úselo cuando la solicitud de un usuario implica tareas de desarrollo de varios pasos, requiere la coordinación de múltiples subtareas o necesita planificación e implementación estratégicas. Adecuado para: (1) desarrollo de funciones complejas (2) proyectos de refactorización de código (3) tareas que requieren investigación + implementación (4) modificaciones coordinadas de múltiples archivos. Las palabras desencadenantes incluyen: "ayúdame a implementar", "desarrollar esta característica", "refactorizar", "compilar" y otras solicitudes relacionadas con el desarrollo.
