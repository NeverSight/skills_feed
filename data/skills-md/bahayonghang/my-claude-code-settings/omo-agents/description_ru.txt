Многоагентная система оркестровки в стиле «о, мой открытый код». Он предоставляет семь специализированных агентов, работающих в тандеме: Sisyphus (главный оркестратор), Oracle (консультант по архитектуре), Библиотекарь (исследование документации), Explore (поиск кода), Frontend Engineer (UI/UX), Document Writer (составление документации) и Multimodal Looker (мультимодальный анализ). Используйте его для сложных задач разработки, решения многосторонних проблем или помощи в конкретных профессиональных областях.
