Многоагентная система оркестровки в стиле «oh-my-opencode». Он предоставляет семь специализированных агентов, работающих вместе: Sisyphus (главный оркестратор), Oracle (консультант по архитектуре), Библиотекарь (исследование документации), Explore (поиск кода), Frontend Engineer (UI/UX), Document Writer (написание документации), Multimodal Looker (мультимодальный анализ). Используйте, когда необходимы сложные задачи разработки, многостороннее решение проблем или помощь в конкретных профессиональных областях.
