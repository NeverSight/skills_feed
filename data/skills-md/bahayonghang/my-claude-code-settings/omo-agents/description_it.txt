Un sistema di orchestrazione multi-agente in stile oh-my-opencode. Fornisce sette agenti specializzati che lavorano in tandem: Sisyphus (capo orchestratore), Oracle (consulente di architettura), Librarian (ricerca di documentazione), Explore (ricerca di codice), Frontend Engineer (UI/UX), Document Writer (redazione di documentazione) e Multimodal Looker (analisi multimodale). Usalo per attività di sviluppo complesse, risoluzione di problemi multi-angolo o aiuto in ambiti professionali specifici.
