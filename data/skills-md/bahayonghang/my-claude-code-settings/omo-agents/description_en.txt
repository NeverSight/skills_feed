A multi-agent orchestration system in oh-my-opencode style. It provides seven specialized agents working together: Sisyphus (chief orchestrator), Oracle (architecture advisor), Librarian (documentation research), Explore (code search), Frontend Engineer (UI/UX), Document Writer (documentation writing), Multimodal Looker (multimodal analysis). Use when complex development tasks, multi-angle problem solving, or help in specific professional domains are needed.
