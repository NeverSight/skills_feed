An oh-my-opencode-style multi-agent orchestration system. It provides seven specialized agents working in tandem: Sisyphus (chief orchestrator), Oracle (architecture consultant), Librarian (documentation research), Explore (code search), Frontend Engineer (UI/UX), Document Writer (documentation drafting), and Multimodal Looker (multimodal analysis). Use it for complex development tasks, multi-angle problem solving, or help in specific professional domains.
