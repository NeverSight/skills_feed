Ein Multi-Agenten-Orchestrierungssystem im Oh-my-Opencode-Stil. Es stellt sieben spezialisierte Agenten bereit, die zusammenarbeiten: Sisyphus (Chef-Orchestrator), Oracle (Architekturberater), Librarian (Dokumentationsrecherche), Explore (Codesuche), Frontend Engineer (UI/UX), Document Writer (Dokumentationserstellung), Multimodal Looker (multimodale Analyse). Verwenden Sie es, wenn komplexe Entwicklungsaufgaben, Problemlösungen aus mehreren Blickwinkeln oder Hilfe in bestimmten beruflichen Bereichen benötigt werden.
