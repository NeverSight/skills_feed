Ein Multi-Agenten-Orchestrierungssystem im Opencode-Stil. Es stellt sieben spezialisierte Agenten bereit, die zusammenarbeiten: Sisyphus (Chef-Orchestrator), Oracle (Architekturberater), Librarian (Dokumentationsrecherche), Explore (Codesuche), Frontend Engineer (UI/UX), Document Writer (Dokumentationsentwurf) und Multimodal Looker (multimodale Analyse). Verwenden Sie es für komplexe Entwicklungsaufgaben, die Lösung von Problemen aus mehreren Blickwinkeln oder zur Unterstützung in bestimmten beruflichen Bereichen.
