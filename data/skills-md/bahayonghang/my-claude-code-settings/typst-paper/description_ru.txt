Typst Academic Paper Assistant (поддерживает статьи на китайском и английском языках, материалы для конференций/журналов).
Области: глубокое обучение, временные ряды, промышленный контроль, информатика.

Триггерные слова (любой модуль может быть вызван независимо):
- «скомпилировать», «типст-компилировать» → скомпилировать модуль
- «формат», «линт» → модуль проверки формата
- «грамматика», «корректура» → модуль грамматики
- «длинное предложение», «упростить» → модуль сложного предложения
- «академический тон», «улучшить письмо» → модуль академического стиля
- «перевести», «С китайского на английский» → модуль перевода
- «нагрудник», «библиография» → справочный модуль
- «деаи», «гуманизировать» → модуль де-ИИ
- «шаблон», «IEEE», «ACM» → модуль конфигурации шаблона
