Typst Academic Paper Assistant (中国語と英語の論文、会議/雑誌への投稿をサポート)。
分野: ディープラーニング、時系列、産業制御、コンピュータサイエンス。

トリガーワード (任意のモジュールを独立して呼び出すことができます):
- 「コンパイル」、「typst コンパイル」 → モジュールをコンパイルします
- 「format」、「lint」 → フォーマットチェックモジュール
- 「文法」、「校正」 → 文法モジュール
- 「長文」「簡略化」 → 複文モジュール
- 「アカデミックトーン」「ライティング改善」 → アカデミックスタイルモジュール
- 「翻訳」、「中国語から英語」 → 翻訳モジュール
- 「bib」、「参考文献」 → 参照モジュール
- 「deai」、「humanize」 → 脱AIモジュール
- 「テンプレート」、「IEEE」、「ACM」 → テンプレート設定モジュール
