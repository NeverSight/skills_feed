Typst Academic Paper Assistant (中国語/英語の論文、会議/雑誌の投稿をサポート)。
分野: ディープラーニング、時系列、産業制御、コンピュータサイエンス。

トリガーワード (任意のモジュールを独立して呼び出すことができます):
- 「compile」、「typst コンパイル」 → モジュールをコンパイルします
- 「format」、「lint」 → フォーマットチェックモジュール
- 「文法」、「校正」 → 文法解析モジュール
・「長文」「簡略化」 → 複文解析モジュール
・「アカデミックトーン」「ライティング向上」 → アカデミック表現モジュール
- "translate"、"中译英" → 翻訳モジュール
- 「bib」、「参考文献」 → 参考文献モジュール
- 「deai」、「humanize」 → De-AI編集モジュール
- 「テンプレート」、「IEEE」、「ACM」 → テンプレート設定モジュール
