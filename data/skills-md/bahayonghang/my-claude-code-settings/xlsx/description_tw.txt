當電子表格檔案是主要輸入或輸出時，請使用此技能。這表示使用者想要執行的任何任務： 開啟、讀取、編輯或修復現有 .xlsx、.xlsm、.csv 或 .tsv 檔案（例如，新增列、計算公式、格式化、圖表、清理混亂資料）；從頭開始或從其他資料來源建立新的電子表格；或在表格檔案格式之間進行轉換。特別是當使用者透過名稱或路徑引用電子表格檔案時（甚至是隨意地（例如「我下載的 xlsx」））並希望對其執行某些操作或從中產生某些內容時，尤其會觸發。還可以觸發清理或重組混亂的表格資料檔案（格式錯誤的行、錯誤的標題、垃圾資料）到正確的電子表格中。可交付成果必須是電子表格檔案。當主要交付成果是 Word 文件、HTML 報告、獨立 Python 腳本、資料庫管道或 Google Sheets API 整合時，即使涉及表格數據，也不要觸發。
