استخدم هذه المهارة في أي وقت يكون فيه ملف جدول البيانات هو الإدخال أو الإخراج الأساسي. وهذا يعني أي مهمة يريد المستخدم القيام بها: فتح أو قراءة أو تحرير أو إصلاح ملف موجود بتنسيق .xlsx أو .xlsm أو .csv أو .tsv (على سبيل المثال، إضافة أعمدة أو صيغ حسابية أو تنسيق أو رسم بياني أو تنظيف البيانات الفوضوية)؛ إنشاء جدول بيانات جديد من البداية أو من مصادر بيانات أخرى؛ أو التحويل بين تنسيقات الملفات الجدولية. يتم تشغيله بشكل خاص عندما يشير المستخدم إلى ملف جدول بيانات بالاسم أو المسار - حتى بشكل عرضي (مثل "xlsx في التنزيلات الخاصة بي") - ويريد القيام بشيء ما به أو إنتاجه منه. قم أيضًا بتشغيل تنظيف أو إعادة هيكلة ملفات البيانات الجدولية الفوضوية (الصفوف المشوهة، الرؤوس في غير مكانها، البيانات غير المرغوب فيها) في جداول بيانات مناسبة. يجب أن يكون التسليم ملف جدول بيانات. لا يتم التشغيل عندما يكون التسليم الأساسي عبارة عن مستند Word، أو تقرير HTML، أو برنامج Python النصي المستقل، أو خط أنابيب قاعدة البيانات، أو تكامل Google Sheets API، حتى لو كانت البيانات الجدولية متضمنة.
