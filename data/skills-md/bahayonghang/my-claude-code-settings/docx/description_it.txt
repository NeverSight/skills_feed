Utilizza questa capacità ogni volta che l'utente desidera creare, leggere, modificare o manipolare documenti Word (file .docx). I trigger includono: qualsiasi menzione di "Word doc", "documento word", ".docx" o richieste di produrre documenti professionali con formattazione come sommari, intestazioni, numeri di pagina o carta intestata. Da utilizzare anche per estrarre o riorganizzare contenuto da file .docx, inserire o sostituire immagini in documenti, eseguire operazioni di ricerca e sostituzione in file Word, lavorare con modifiche o commenti rilevati o convertire contenuto in un documento Word ottimizzato. Se l'utente richiede un "rapporto", un "memo", una "lettera", un "modello" o un documento simile come file Word o .docx, utilizza questa abilità. NON utilizzare per PDF, fogli di calcolo, documenti Google o attività di codifica generali non correlate alla generazione di documenti.
