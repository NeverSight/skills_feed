Преобразование серверов MCP (Model Context Protocol) в Claude Code Skills. Используйте, когда:
(1) Пользователь хочет преобразовать проект сервера MCP в навык.
(2) Пользователь упоминает «MCP в навык», «конвертировать MCP» или «MCP 转 навык».
(3) У пользователя есть кодовая база сервера MCP, и он хочет сделать ее навыком многократного использования.
(4) Пользователь хочет проанализировать структуру сервера MCP для создания навыков.
Поддерживает серверы TypeScript/JavaScript и Python MCP. Создает полный пакет навыков с помощью SKILL.md, сценариев и ссылок.
