MCP (Model Context Protocol) サーバーをクロード コード スキルに変換します。次の場合に使用します。
(1) ユーザーは MCP サーバー プロジェクトをスキルに変換したい
(2) ユーザーが「MCP をスキルに変換」、「MCP を変換」、または「MCP 转スキル」に言及
(3) ユーザーは MCP サーバーのコードベースを持っており、それを再利用可能なスキルにしたい
(4) ユーザーはスキル作成のために MCP サーバーの構造を解析したい
TypeScript/JavaScript および Python MCP サーバーをサポートします。 SKILL.md、スクリプト、リファレンスを含む完全なスキル パッケージを生成します。
