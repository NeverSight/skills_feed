MCP(Model Context Protocol) 서버를 Claude Code Skills로 변환합니다. 다음과 같은 경우에 사용하세요:
(1) MCP 서버 프로젝트를 스킬로 변환하려는 사용자
(2) 사용자가 "MCP를 스킬로", "MCP를 변환" 또는 "MCP 转 스킬"을 언급합니다.
(3) 사용자는 MCP 서버 코드베이스를 가지고 있으며 이를 재사용 가능한 스킬로 만들고 싶어합니다.
(4) 스킬 생성을 위해 MCP 서버 구조를 분석하고 싶은 사용자
TypeScript/JavaScript 및 Python MCP 서버를 지원합니다. SKILL.md, 스크립트 및 참조를 사용하여 완전한 기술 패키지를 생성합니다.
