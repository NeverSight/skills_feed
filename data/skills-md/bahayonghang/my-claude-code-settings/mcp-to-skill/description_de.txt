Konvertieren Sie MCP-Server (Model Context Protocol) in Claude Code Skills. Verwenden Sie, wenn:
(1) Der Benutzer möchte ein MCP-Serverprojekt in einen Skill umwandeln
(2) Der Benutzer erwähnt „MCP in Skill“, „MCP konvertieren“ oder „MCP in Skill“
(3) Der Benutzer verfügt über eine MCP-Server-Codebasis und möchte daraus einen wiederverwendbaren Skill machen
(4) Der Benutzer möchte die MCP-Serverstruktur für die Skill-Erstellung analysieren
Unterstützt TypeScript/JavaScript- und Python MCP-Server. Erzeugt ein komplettes Skill-Paket mit SKILL.md, Skripten und Referenzen.
