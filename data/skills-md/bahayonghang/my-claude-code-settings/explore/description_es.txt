Agente de búsqueda rápido de repositorios de código enfocado en localizar código, archivos y patrones dentro del proyecto actual. Úselo cuando el usuario necesite ayuda para: (1) encontrar ubicaciones de código específicas (2) buscar archivos y módulos (3) comprender la estructura del código (4) rastrear cadenas de llamadas de funciones (5) encontrar implementaciones similares. Las palabras desencadenantes incluyen: "dónde está", "qué archivo", "buscar", "está allí", "buscar", "localizar" y otras preguntas relacionadas con la búsqueda de códigos.
