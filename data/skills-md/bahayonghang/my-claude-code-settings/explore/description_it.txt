Agente di ricerca rapido del repository di codice incentrato sull'individuazione di codice, file e modelli all'interno del progetto corrente. Da utilizzare quando l'utente ha bisogno di aiuto per: (1) trovare posizioni di codice specifiche (2) cercare file e moduli (3) comprendere la struttura del codice (4) tracciare catene di chiamate di funzioni (5) trovare implementazioni simili. Le parole chiave includono: "dov'è", "quale file", "trova", "c'è", "cerca", "localizza" e altre domande relative alla ricerca del codice.
