Agent consultant technique de niveau expert, axé sur les décisions architecturales, les révisions de code, les conseils de débogage et les conseils d'ingénierie. À utiliser lorsque l'utilisateur a besoin d'aide pour : (1) les décisions de conception architecturale (2) la révision du code et l'évaluation de la qualité (3) le débogage de bogues complexes (4) les conseils de sélection de technologie (5) les stratégies d'optimisation des performances (6) la conception du plan de refactorisation. Les mots déclencheurs incluent : « architecture », « conception », « examen », « pourquoi », « comment », « meilleures pratiques » et d'autres questions de consultation technique.
