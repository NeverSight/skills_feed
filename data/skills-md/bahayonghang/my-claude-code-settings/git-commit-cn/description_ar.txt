قم بإنشاء رسائل التزام Git الصينية التي تتوافق مع مواصفات الالتزامات التقليدية. استخدم هذا عندما يطلب المستخدم إنشاء التزام Git، أو إنشاء رسالة التزام، أو يذكر "رمز الالتزام"، أو "التزام git"، أو "الالتزام الصيني". قم بتحليل تغييرات التعليمات البرمجية تلقائيًا وإنشاء رسائل التزام صينية موحدة، بما في ذلك بادئة النوع الصحيحة (feat/fix/docs، وما إلى ذلك) ووصف واضح.
