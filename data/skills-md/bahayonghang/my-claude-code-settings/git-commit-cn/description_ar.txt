قم بإنشاء رسائل التزام Git الصينية التي تتوافق مع مواصفات الالتزام التقليدية. يُستخدم عندما يطلب المستخدم إنشاء التزام Git، أو إنشاء رسالة التزام، أو الإشارة إلى "提交代码"، أو "git Commit"، أو "中文提交". قم بتحليل تغييرات التعليمات البرمجية تلقائيًا وإنشاء رسائل التزام صينية موحدة، بما في ذلك بادئة النوع الصحيحة (feat/fix/docs، وما إلى ذلك) ووصف واضح.
