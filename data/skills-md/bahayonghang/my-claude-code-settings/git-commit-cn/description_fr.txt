Générez des messages de validation Git chinois conformes à la spécification Conventional Commits. Utilisez-le lorsque l'utilisateur demande à créer un commit Git, à générer un message de commit ou à mentionner « code de commit », « commit git » ou « commit chinois ». Analysez automatiquement les modifications de code et générez des messages de validation en chinois standardisés, comprenant le préfixe de type correct (feat/fix/docs, etc.) et une description claire.
