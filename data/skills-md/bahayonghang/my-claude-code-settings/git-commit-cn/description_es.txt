Genere mensajes de confirmación de Git chinos que cumplan con la especificación de confirmaciones convencionales. Úselo cuando el usuario solicite crear una confirmación de Git, generar un mensaje de confirmación o mencione "código de confirmación", "compromiso de git" o "compromiso chino". Analice automáticamente los cambios de código y genere mensajes de confirmación en chino estandarizados, incluido el prefijo de tipo correcto (feat/fix/docs, etc.) y una descripción clara.
