Genere mensajes de confirmación de Git chinos que se ajusten a la especificación de confirmación convencional. Úselo cuando el usuario solicite crear una confirmación de Git, genere un mensaje de confirmación o mencione "提交代码", "git commit" o "中文提交". Analice automáticamente los cambios de código y genere mensajes de confirmación en chino estandarizados, incluido el prefijo de tipo correcto (feat/fix/docs, etc.) y una descripción clara.
