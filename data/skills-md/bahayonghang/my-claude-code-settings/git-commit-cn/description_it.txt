Genera messaggi di commit Git cinese conformi alla specifica di commit convenzionale. Da utilizzare quando l'utente richiede di creare un commit Git, generare un messaggio di commit o menziona "提交代码", "git commit" o "中文提交". Analizza automaticamente le modifiche al codice e genera messaggi di commit cinesi standardizzati, incluso il prefisso del tipo corretto (feat/fix/docs, ecc.) e una descrizione chiara.
