Genera messaggi di commit Git cinese conformi alla specifica Commit convenzionali. Utilizzalo quando l'utente richiede di creare un commit Git, generare un messaggio di commit o menziona "codice di commit", "git commit" o "commit cinese". Analizza automaticamente le modifiche al codice e genera messaggi di commit cinesi standardizzati, incluso il prefisso del tipo corretto (feat/fix/docs, ecc.) e una descrizione chiara.
