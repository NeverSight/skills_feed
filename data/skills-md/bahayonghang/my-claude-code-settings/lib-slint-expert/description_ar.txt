خبير تطوير شامل لواجهة المستخدم الرسومية Slint يعتمد على كود المصدر الرسمي. يغطي تكامل Rust، وتصميم المكونات، والتخطيطات، والتصميم، والرسوم المتحركة، والنشر عبر الأنظمة الأساسية، وتحسين الأداء. يُستخدم عند العمل مع مجموعة أدوات Slint UI، أو إنشاء تطبيقات واجهة المستخدم الرسومية الأصلية، أو عند ذكر Slint، أو تطوير واجهة المستخدم الرسومية، أو واجهات مستخدم Rust. بنيت مع الوثائق والأمثلة الرسمية.
