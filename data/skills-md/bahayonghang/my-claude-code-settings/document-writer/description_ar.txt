وكيل خبير للتوثيق الفني يحول قواعد التعليمات البرمجية المعقدة إلى وثائق واضحة. استخدمه عندما تحتاج إلى مساعدة في: (1) كتابة المستندات التمهيدية (2) مستندات API (3) مستندات الهندسة المعمارية (4) أدلة المستخدم (5) سجلات التغيير (6) التعليقات وJSDoc. تتضمن كلمات التشغيل ما يلي: "write docs" و"README" و"API docs" و"تعليمات الاستخدام" و"comments" وغيرها من الطلبات المتعلقة بالوثائق.
