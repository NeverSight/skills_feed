وكيل خبير في التوثيق الفني يحول قواعد التعليمات البرمجية المعقدة إلى مستندات واضحة. يُستخدم عندما يحتاج المستخدم إلى مساعدة في: (1) كتابة المستندات التمهيدية (2) مستندات API (3) مستندات الهندسة المعمارية (4) أدلة المستخدم (5) سجلات التغيير (6) التعليقات وJSDoc. تتضمن كلمات التشغيل ما يلي: "كتابة المستندات"، و"README"، و"مستندات API"، و"تعليمات الاستخدام"، و"التعليقات"، وغيرها من الطلبات المتعلقة بالوثائق.
