Экспертный агент технической документации, который превращает сложные базы кода в понятную документацию. Используйте, когда вам нужна помощь в: (1) написании файлов README (2) документации API (3) документации по архитектуре (4) руководствах пользователя (5) журналах изменений (6) комментариях и JSDoc. К триггерным словам относятся: «написать документы», «README», «документы API», «инструкции по использованию», «комментарии» и другие запросы, связанные с документацией.
