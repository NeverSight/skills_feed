Агент эксперта по технической документации, который превращает сложные базы кода в понятные документы. Используйте, когда пользователю нужна помощь в: (1) написании файлов README (2) документации API (3) документации по архитектуре (4) руководств пользователя (5) журналов изменений (6) комментариев и JSDoc. К триггерным словам относятся: «написать документы», «README», «документы API», «инструкции по использованию», «комментарии» и другие запросы, связанные с документацией.
