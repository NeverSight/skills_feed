دليل مكتبة مكونات shadcn/ui الكامل، بما في ذلك تثبيت مكونات React التي يمكن الوصول إليها وتكوينها وتنفيذها. يُستخدم عند إعداد shadcn/ui، أو تثبيت المكونات، أو إنشاء النماذج باستخدام React Hook Form وZod، أو تخصيص السمات باستخدام Tailwind CSS، أو تنفيذ أنماط واجهة المستخدم مثل الأزرار، ومربعات الحوار، والقوائم المنسدلة، والجداول، وتخطيطات النماذج المعقدة.
