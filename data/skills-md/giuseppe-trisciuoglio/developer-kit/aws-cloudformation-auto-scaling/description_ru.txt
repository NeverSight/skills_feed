Предоставляет шаблоны AWS CloudFormation для автоматического масштабирования, включая EC2, ECS и Lambda. Используйте при создании групп автоматического масштабирования, конфигураций запуска, шаблонов запуска, политик масштабирования, перехватчиков жизненного цикла и прогнозного масштабирования. Охватывает структуру шаблона с параметрами, выходами, сопоставлениями, условиями, перекрестными ссылками и передовыми практиками для обеспечения высокой доступности и оптимизации затрат.
