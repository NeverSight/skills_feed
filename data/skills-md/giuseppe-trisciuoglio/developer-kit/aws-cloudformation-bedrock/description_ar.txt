يوفر أنماط AWS CloudFormation لموارد Amazon Bedrock بما في ذلك الوكلاء وقواعد المعرفة ومصادر البيانات وحواجز الحماية والمطالبات والتدفقات وملفات تعريف الاستدلال. يُستخدم عند إنشاء وكلاء Bedrock مع مجموعات العمل، وتنفيذ RAG مع قواعد المعرفة، وتكوين مخازن المتجهات، وإعداد حواجز حماية الإشراف على المحتوى، وإدارة المطالبات، وتنسيق سير العمل مع التدفقات، وتكوين ملفات تعريف الاستدلال لتحسين النموذج.
