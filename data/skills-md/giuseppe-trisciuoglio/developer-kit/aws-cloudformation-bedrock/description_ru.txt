Предоставляет шаблоны AWS CloudFormation для ресурсов Amazon Bedrock, включая агенты, базы знаний, источники данных, ограждения, подсказки, потоки и профили вывода. Используйте при создании агентов Bedrock с группами действий, реализации RAG с базами знаний, настройке векторных хранилищ, настройке ограничений модерации контента, управлении подсказками, организации рабочих процессов с помощью потоков и настройке профилей вывода для оптимизации модели.
