Предоставляет шаблоны AWS CloudFormation для баз данных Amazon RDS. Используйте при создании экземпляров RDS (MySQL, PostgreSQL, Aurora), кластеров БД, развертываний в нескольких зонах доступности, групп параметров, групп подсетей и реализации структуры шаблонов с параметрами, выходными данными, сопоставлениями, условиями и межстековыми ссылками.
