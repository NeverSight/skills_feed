Fournit des modèles AWS Key Management Service (KMS) à l'aide du kit AWS SDK pour Java 2.x. À utiliser lors de la création/gestion de clés de chiffrement, du chiffrement/déchiffrement de données, de la génération de clés de données, de la signature numérique, de la rotation des clés ou de l'intégration du chiffrement dans les applications Spring Boot.
