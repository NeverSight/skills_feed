Stellt AWS Key Management Service (KMS)-Muster mithilfe des AWS SDK für Java 2.x bereit. Wird zum Erstellen/Verwalten von Verschlüsselungsschlüsseln, zum Verschlüsseln/Entschlüsseln von Daten, zum Generieren von Datenschlüsseln, zum digitalen Signieren, zur Schlüsselrotation oder zum Integrieren der Verschlüsselung in Spring Boot-Anwendungen verwendet.
