Предоставляет шаблоны AWS CloudFormation для экземпляров EC2, групп безопасности, ролей IAM и балансировщиков нагрузки. Используйте при создании экземпляров EC2, экземпляров SPOT, групп безопасности, ролей IAM для EC2, балансировщиков нагрузки приложений (ALB), целевых групп и реализации структуры шаблона с параметрами, выходными данными, сопоставлениями, условиями и межстековыми ссылками.
