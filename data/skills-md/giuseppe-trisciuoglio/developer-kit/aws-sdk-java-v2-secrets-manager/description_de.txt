Stellt AWS Secrets Manager-Muster mit AWS SDK für Java 2.x bereit. Wird zum Speichern/Abrufen von Geheimnissen (Passwörter, API-Schlüssel, Token), zum automatischen Rotieren von Geheimnissen, zum Verwalten von Datenbankanmeldeinformationen oder zum Integrieren der Geheimnisverwaltung in Spring Boot-Anwendungen verwendet.
