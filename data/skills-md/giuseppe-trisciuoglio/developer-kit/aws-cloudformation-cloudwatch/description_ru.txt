Предоставляет шаблоны AWS CloudFormation для мониторинга CloudWatch, показателей, сигналов тревоги, информационных панелей, журналов и наблюдения. Используйте при создании метрик CloudWatch, сигналов тревоги, информационных панелей, групп журналов, подписок на журналы, обнаружения аномалий, синтезированных канареек, сигналов приложений, а также реализации структуры шаблонов с параметрами, выходными данными, сопоставлениями, условиями, межстековыми ссылками и передовыми практиками CloudWatch для мониторинга производственной инфраструктуры.
