Предоставляет шаблоны для реализации слоев персистентности с помощью Spring Data JPA. Используйте при создании репозиториев, настройке связей сущностей, написании запросов (производных и @Query), настройке разбиения на страницы, аудите базы данных, транзакциях, первичных ключах UUID, нескольких базах данных и индексировании баз данных.
