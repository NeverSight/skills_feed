Предоставляет шаблоны AWS CloudFormation для дистрибутивов CloudFront, источников (ALB, S3, Lambda@Edge, VPC Origins), CacheBehaviors, функций, SecurityHeaders, параметров, выходных данных и межстековых ссылок. Используйте при создании дистрибутивов CloudFront с помощью CloudFormation, настройке нескольких источников, реализации стратегий кэширования, управлении личными доменами с помощью ACM, настройке WAF и оптимизации производительности.
