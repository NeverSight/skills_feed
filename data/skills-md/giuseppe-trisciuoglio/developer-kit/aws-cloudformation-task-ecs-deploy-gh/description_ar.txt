يوفر أنماطًا لنشر مهام وخدمات ECS باستخدام GitHub Actions CI/CD. يُستخدم عند إنشاء صور Docker، والدفع إلى ECR، وتحديث تعريفات مهام ECS، ونشر خدمات ECS، والتكامل مع مجموعات CloudFormation، وتكوين مصادقة AWS OIDC لإجراءات GitHub، وتنفيذ مسارات نشر الحاويات الجاهزة للإنتاج. يدعم عمليات نشر ECS مع الأمان المناسب (مفاتيح OIDC أو IAM)، ودعم البيئات المتعددة، وعمليات النشر باللون الأزرق/الأخضر، ومستودعات ECR الخاصة مع مسح الصور، وتحديثات البنية التحتية لـ CloudFormation.
