Fornisce modelli per distribuire attività e servizi ECS con GitHub Actions CI/CD. Da utilizzare durante la creazione di immagini Docker, il push su ECR, l'aggiornamento delle definizioni delle attività ECS, la distribuzione di servizi ECS, l'integrazione con gli stack CloudFormation, la configurazione dell'autenticazione OIDC AWS per le azioni GitHub e l'implementazione di pipeline di distribuzione di contenitori pronti per la produzione. Supporta distribuzioni ECS con sicurezza adeguata (chiavi OIDC o IAM), supporto multiambiente, distribuzioni blu/verdi, repository privati ​​ECR con scansione di immagini e aggiornamenti dell'infrastruttura CloudFormation.
