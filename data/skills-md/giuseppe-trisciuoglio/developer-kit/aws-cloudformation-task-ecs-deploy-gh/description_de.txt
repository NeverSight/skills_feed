Stellt Muster zum Bereitstellen von ECS-Aufgaben und -Diensten mit GitHub Actions CI/CD bereit. Verwendung beim Erstellen von Docker-Images, beim Pushen an ECR, beim Aktualisieren von ECS-Aufgabendefinitionen, beim Bereitstellen von ECS-Diensten, beim Integrieren mit CloudFormation-Stacks, beim Konfigurieren der AWS-OIDC-Authentifizierung für GitHub-Aktionen und beim Implementieren produktionsbereiter Container-Bereitstellungspipelines. Unterstützt ECS-Bereitstellungen mit angemessener Sicherheit (OIDC- oder IAM-Schlüssel), Unterstützung für mehrere Umgebungen, blaue/grüne Bereitstellungen, private ECR-Repositorys mit Image-Scanning und CloudFormation-Infrastrukturaktualisierungen.
