GitHub Actions CI/CD を使用して ECS タスクとサービスをデプロイするためのパターンを提供します。 Docker イメージの構築、ECR へのプッシュ、ECS タスク定義の更新、ECS サービスのデプロイ、CloudFormation スタックとの統合、GitHub Actions の AWS OIDC 認証の設定、本番対応のコンテナ デプロイメント パイプラインの実装を行うときに使用します。適切なセキュリティ (OIDC または IAM キー)、マルチ環境のサポート、ブルー/グリーン デプロイメント、イメージ スキャンを備えた ECR プライベート リポジトリ、および CloudFormation インフラストラクチャの更新を備えた ECS デプロイメントをサポートします。
