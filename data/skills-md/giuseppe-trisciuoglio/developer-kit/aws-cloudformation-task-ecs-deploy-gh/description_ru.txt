Предоставляет шаблоны для развертывания задач и сервисов ECS с помощью GitHub Actions CI/CD. Используйте при создании образов Docker, отправке в ECR, обновлении определений задач ECS, развертывании сервисов ECS, интеграции со стеками CloudFormation, настройке аутентификации AWS OIDC для действий GitHub и реализации готовых к работе конвейеров развертывания контейнеров. Поддерживает развертывания ECS с надлежащей безопасностью (ключи OIDC или IAM), поддержку нескольких сред, синие/зеленые развертывания, частные репозитории ECR со сканированием изображений и обновления инфраструктуры CloudFormation.
