Proporciona patrones para implementar tareas y servicios de ECS con GitHub Actions CI/CD. Úselo al crear imágenes de Docker, enviar a ECR, actualizar definiciones de tareas de ECS, implementar servicios de ECS, integrarse con pilas de CloudFormation, configurar la autenticación OIDC de AWS para GitHub Actions e implementar canales de implementación de contenedores listos para producción. Admite implementaciones de ECS con seguridad adecuada (claves OIDC o IAM), compatibilidad con múltiples entornos, implementaciones azules/verdes, repositorios privados de ECR con escaneo de imágenes y actualizaciones de infraestructura de CloudFormation.
