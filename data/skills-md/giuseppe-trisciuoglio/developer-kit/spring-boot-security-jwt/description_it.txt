Fornisce modelli di autenticazione e autorizzazione JWT per Spring Boot 3.5.x che coprono la generazione di token con JJWT, autenticazione Bearer/cookie, integrazione database/OAuth2 e controllo degli accessi basato su RBAC/autorizzazione utilizzando Spring Security 6.x. Da utilizzare quando si implementa l'autenticazione o l'autorizzazione nelle applicazioni Spring Boot.
