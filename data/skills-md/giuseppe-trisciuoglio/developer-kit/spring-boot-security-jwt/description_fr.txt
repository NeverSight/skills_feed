Fournit des modèles d'authentification et d'autorisation JWT pour Spring Boot 3.5.x couvrant la génération de jetons avec JJWT, l'authentification Bearer/cookie, l'intégration base de données/OAuth2 et le contrôle d'accès RBAC/basé sur les autorisations à l'aide de Spring Security 6.x. À utiliser lors de la mise en œuvre de l'authentification ou de l'autorisation dans les applications Spring Boot.
