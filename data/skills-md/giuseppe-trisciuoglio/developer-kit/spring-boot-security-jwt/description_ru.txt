Предоставляет шаблоны аутентификации и авторизации JWT для Spring Boot 3.5.x, охватывающие генерацию токенов с помощью JJWT, аутентификацию носителя/cookie, интеграцию базы данных/OAuth2 и управление доступом на основе RBAC/разрешений с использованием Spring Security 6.x. Используйте при реализации аутентификации или авторизации в приложениях Spring Boot.
