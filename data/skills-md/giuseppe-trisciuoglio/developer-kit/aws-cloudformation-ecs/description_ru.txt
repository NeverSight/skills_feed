Предоставляет шаблоны AWS CloudFormation для кластеров ECS, сервисов и определений задач. Используйте при создании инфраструктуры ECS с помощью CloudFormation, настройке определений контейнеров, политик масштабирования, обнаружения сервисов, интеграции балансировки нагрузки и реализации структуры шаблонов с параметрами, выходными данными, сопоставлениями, условиями, межстековыми ссылками и сине-зелеными развертываниями с помощью CodeDeploy.
