Предоставляет шаблоны для создания систем дополнительной генерации (RAG) для приложений ИИ с векторными базами данных и семантическим поиском. Используйте при внедрении искусственного интеллекта, основанного на знаниях, создании систем вопросов и ответов для документов или интеграции LLM с внешними базами знаний.
