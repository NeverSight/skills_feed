Предоставляет шаблоны AWS CloudFormation для функций Lambda, слоев, источников событий и интеграций. Используйте при создании функций Lambda с CloudFormation, настройке API-шлюза, пошаговых функций, EventBridge, SQS, триггеров SNS и реализации структуры шаблона с параметрами, выходными данными, сопоставлениями, условиями, межстековыми ссылками и рекомендациями по оптимизации холодного запуска.
