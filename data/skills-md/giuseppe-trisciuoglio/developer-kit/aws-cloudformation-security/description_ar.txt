يوفر أنماط AWS CloudFormation لأمان البنية التحتية وإدارة الأسرار والتشفير والتعامل الآمن مع البيانات. استخدمه عند إنشاء قوالب CloudFormation آمنة باستخدام AWS Secrets Manager وتشفير KMS والمعلمات الآمنة وسياسات IAM ومجموعات أمان VPC وشهادات TLS/SSL وتكوينات حركة المرور المشفرة. يغطي بنية القالب وأفضل ممارسات المعلمات والمراجع المتداخلة واستراتيجيات الدفاع المتعمق.
