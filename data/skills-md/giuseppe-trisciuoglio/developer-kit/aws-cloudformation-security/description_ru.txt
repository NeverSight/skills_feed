Предоставляет шаблоны AWS CloudFormation для обеспечения безопасности инфраструктуры, управления секретами, шифрования и безопасной обработки данных. Используйте при создании безопасных шаблонов CloudFormation с помощью AWS Secrets Manager, шифрования KMS, параметров безопасности, политик IAM, групп безопасности VPC, сертификатов TLS/SSL и конфигураций зашифрованного трафика. Охватывает структуру шаблона, передовые методы работы с параметрами, межстековые ссылки и стратегии глубокоэшелонированной защиты.
