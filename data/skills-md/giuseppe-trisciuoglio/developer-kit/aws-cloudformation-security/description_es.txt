Proporciona patrones de AWS CloudFormation para seguridad de infraestructura, administración de secretos, cifrado y manejo seguro de datos. Úselo al crear plantillas seguras de CloudFormation con AWS Secrets Manager, cifrado KMS, parámetros seguros, políticas de IAM, grupos de seguridad de VPC, certificados TLS/SSL y configuraciones de tráfico cifrado. Cubre la estructura de la plantilla, las mejores prácticas de parámetros, referencias entre pilas y estrategias de defensa en profundidad.
