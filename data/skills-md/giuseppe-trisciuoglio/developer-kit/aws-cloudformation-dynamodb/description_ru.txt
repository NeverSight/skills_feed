Предоставляет шаблоны AWS CloudFormation для таблиц DynamoDB, GSI, LSI, автоматического масштабирования и потоков. Используйте при создании таблиц DynamoDB с помощью CloudFormation, настройке первичных ключей, локальных/глобальных вторичных индексов, режимов емкости (по требованию/предоставления), восстановления на момент времени, шифрования, TTL и реализации структуры шаблона с параметрами, выходными данными, сопоставлениями, условиями, межстековыми ссылками.
