Fournit des modèles de messagerie AWS à l'aide du kit AWS SDK for Java 2.x pour les files d'attente SQS et les sujets SNS. Gère l'envoi/la réception de messages, les files d'attente FIFO, le DLQ, les abonnements et les modèles de pub/sub. À utiliser lors de la mise en œuvre de la messagerie avec SQS ou SNS.
