Предоставляет шаблоны AWS CloudFormation для Amazon ElastiCache. Используйте при создании кластеров ElastiCache (Redis, Memcached), групп репликации, групп параметров, групп подсетей и реализации структуры шаблона с параметрами, выходными данными, сопоставлениями, условиями и межстековыми ссылками для инфраструктуры распределенного кэширования.
