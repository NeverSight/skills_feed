Proporciona patrones de AWS CloudFormation para usuarios, roles, políticas y políticas administradas de IAM. Úselo al crear recursos de IAM con CloudFormation, implementar acceso con privilegios mínimos, configurar el acceso entre cuentas, configurar centros de identidad, administrar límites de permisos y organizar la estructura de la plantilla con parámetros, resultados, asignaciones y condiciones para implementaciones de infraestructura segura.
