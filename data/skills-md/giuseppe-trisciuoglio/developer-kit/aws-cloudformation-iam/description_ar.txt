يوفر أنماط AWS CloudFormation لمستخدمي IAM والأدوار والسياسات والسياسات المُدارة. يُستخدم عند إنشاء موارد IAM باستخدام CloudFormation، وتنفيذ الوصول الأقل امتيازًا، وتكوين الوصول عبر الحسابات، وإعداد مراكز الهوية، وإدارة حدود الأذونات، وتنظيم بنية القالب باستخدام المعلمات والمخرجات والتعيينات والشروط لعمليات نشر البنية التحتية الآمنة.
