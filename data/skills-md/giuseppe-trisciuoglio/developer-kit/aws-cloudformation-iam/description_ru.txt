Предоставляет шаблоны AWS CloudFormation для пользователей, ролей, политик и управляемых политик IAM. Используйте при создании ресурсов IAM с помощью CloudFormation, реализации доступа с минимальными привилегиями, настройке доступа между учетными записями, настройке центров идентификации, управлении границами разрешений и организации структуры шаблонов с параметрами, выходами, сопоставлениями и условиями для развертывания безопасной инфраструктуры.
