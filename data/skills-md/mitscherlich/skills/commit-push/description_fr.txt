Lorsque l'utilisateur dit « soumettre le code », « valider », « pousser le code », « pousser », « valider et pousser », « soumettre les modifications », « pousser à distance », « aidez-moi à valider » ou « pousser le code vers le haut », cette compétence doit être utilisée. Il fournit un workflow git commit et push standardisé, analyse automatiquement les modifications, génère des messages de validation conformes et exécute les opérations correspondantes.
