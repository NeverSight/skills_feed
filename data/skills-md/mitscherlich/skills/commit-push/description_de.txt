Wenn der Benutzer „Code senden“, „festschreiben“, „Code pushen“, „pushen“, „festschreiben und pushen“, „Änderungen senden“, „an Remote senden“, „hilf mir beim Festschreiben“ oder „den Code hochschieben“ sagt, sollte diese Fähigkeit verwendet werden. Es bietet einen standardisierten Git-Commit- und Push-Workflow, analysiert automatisch die Änderungen, generiert konforme Commit-Nachrichten und führt die entsprechenden Vorgänge aus.
