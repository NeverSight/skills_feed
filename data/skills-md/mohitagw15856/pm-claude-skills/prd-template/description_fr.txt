Création de documents sur les exigences du produit selon une structure de modèle PM éprouvée. À utiliser lorsque l'utilisateur demande à créer, rédiger, rédiger ou aider avec un PRD, un document sur les exigences du produit, une spécification de produit, une spécification de fonctionnalité ou une documentation produit pour une nouvelle fonctionnalité ou un nouveau produit.
