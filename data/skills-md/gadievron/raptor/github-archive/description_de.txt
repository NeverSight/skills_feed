Untersuchen Sie GitHub-Sicherheitsvorfälle mithilfe manipulationssicherer GitHub-Archivdaten über BigQuery. Wird zum Überprüfen von Repository-Aktivitätsansprüchen, zum Wiederherstellen gelöschter PRs/Branches/Tags/Repos, zum Zuordnen von Aktionen zu Akteuren oder zum Rekonstruieren von Angriffszeitplänen verwendet. Bietet unveränderliche forensische Beweise für alle öffentlichen GitHub-Ereignisse seit 2011.
