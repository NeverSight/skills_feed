تحقق من حوادث أمان GitHub باستخدام بيانات أرشيف GitHub المقاومة للتلاعب عبر BigQuery. يُستخدم عند التحقق من مطالبات نشاط المستودع، أو استعادة العلاقات العامة/الفروع/العلامات/الريبو المحذوفة، أو إسناد الإجراءات إلى الجهات الفاعلة، أو إعادة بناء الجداول الزمنية للهجوم. يوفر أدلة جنائية ثابتة لجميع أحداث GitHub العامة منذ عام 2011.
