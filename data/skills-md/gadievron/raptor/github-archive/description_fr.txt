Enquêtez sur les incidents de sécurité GitHub à l'aide des données inviolables des archives GitHub via BigQuery. À utiliser lors de la vérification des déclarations d'activité du référentiel, de la récupération des PR/branches/tags/dépôts supprimés, de l'attribution d'actions aux acteurs ou de la reconstruction des chronologies des attaques. Fournit des preuves médico-légales immuables de tous les événements publics GitHub depuis 2011.
