Investigue incidentes de seguridad de GitHub utilizando datos de GitHub Archive a prueba de manipulaciones a través de BigQuery. Úselo al verificar reclamos de actividad del repositorio, recuperar PR/ramas/etiquetas/repositorios eliminados, atribuir acciones a los actores o reconstruir cronogramas de ataques. Proporciona evidencia forense inmutable de todos los eventos públicos de GitHub desde 2011.
