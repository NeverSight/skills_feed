Расследуйте инциденты безопасности GitHub, используя защищенные от несанкционированного доступа данные архива GitHub через BigQuery. Используйте при проверке заявлений об активности репозитория, восстановлении удаленных запросов/веток/тегов/репозиториев, приписывании действий злоумышленникам или восстановлении сроков атак. Предоставляет неизменяемые судебно-медицинские доказательства всех публичных событий GitHub с 2011 года.
