Stellen Sie gelöschte Commits von GitHub mithilfe der REST-API, der Webschnittstelle und Git Fetch wieder her. Verwenden Sie diese Option, wenn Sie Commit-SHAs haben und tatsächliche Commit-Inhalte, Unterschiede oder Patches abrufen müssen. Enthält Techniken für den Zugriff auf „gelöschte“ Commits, die auf GitHub-Servern verbleiben.
