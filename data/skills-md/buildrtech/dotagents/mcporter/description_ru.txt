Взаимодействуйте с серверами MCP (Model Context Protocol) с помощью интерфейса командной строки mcporter. Используйте этот навык, когда вам нужно составить список доступных серверов MCP, просмотреть схемы инструментов, вызвать инструменты MCP, управлять аутентификацией OAuth или создать оболочки CLI и типы TypeScript для серверов MCP.
