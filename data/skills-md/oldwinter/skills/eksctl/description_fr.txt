Cette compétence doit être utilisée lorsque les utilisateurs doivent gérer des clusters AWS EKS via eksctl CLI. Il couvre la création de cluster, la gestion des groupes de nœuds, les opérations complémentaires, l'intégration IAM et les mises à niveau du cluster. Complète kubectl pour les opérations au niveau du cluster. Se déclenche sur les requêtes mentionnant eksctl, la gestion de cluster EKS, les groupes de nœuds, les modules complémentaires EKS ou l'infrastructure de cluster Kubernetes sur AWS.
