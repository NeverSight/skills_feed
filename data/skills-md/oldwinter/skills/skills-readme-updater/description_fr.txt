Cette compétence doit être utilisée après avoir créé ou modifié des compétences pour mettre à jour le fichier README.md principal. Il analyse toutes les compétences dans ~/.claude/skills/, extrait les métadonnées des fichiers SKILL.md et régénère le README avec des listes de compétences catégorisées. Se déclenche sur les requêtes mentionnant « mettre à jour le fichier readme des compétences », « actualiser la liste des compétences » ou après l'ajout de nouvelles compétences.
