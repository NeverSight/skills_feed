Questa competenza deve essere utilizzata dopo aver creato o modificato le competenze per aggiornare il file README.md principale. Esegue la scansione di tutte le competenze in ~/.claude/skills/, estrae i metadati dai file SKILL.md e rigenera il file README con elenchi di competenze categorizzati. Si attiva su richieste che menzionano "aggiornamento leggimi competenze", "aggiornamento elenco competenze" o dopo l'aggiunta di nuove competenze.
