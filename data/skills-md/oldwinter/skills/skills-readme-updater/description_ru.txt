Этот навык следует использовать после создания или изменения навыков для обновления основного файла README.md. Он сканирует все навыки в ~/.claude/skills/, извлекает метаданные из файлов SKILL.md и заново создает README со списками навыков по категориям. Запускается по запросам с упоминанием «обновить файл сведений о навыках», «обновить список навыков» или после добавления новых навыков.
