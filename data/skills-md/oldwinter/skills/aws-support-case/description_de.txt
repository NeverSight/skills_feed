Diese Fähigkeit sollte verwendet werden, wenn Benutzer AWS Support-Fälle über CLI verwalten müssen. Es verwaltet das Auflisten von Fällen (letzte 2 Wochen, ungelöst), das Anzeigen von Falldetails mit zweisprachiger Anzeige (Englisch-Chinesisch Zeile für Zeile), das Erstellen neuer Fälle mit automatisch erkanntem Dienst/Kategorie, das Beantworten von Fällen und die Bearbeitung von Anhängen. Wird bei Anfragen ausgelöst, die „AWS-Support“, „Support-Fall“, „Hilfe“, „Support-Ticket“ oder Anfragen zum technischen Support von AWS erwähnen.
