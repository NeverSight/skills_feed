Этот навык следует использовать, когда пользователям необходимо взаимодействовать с GitHub через интерфейс командной строки gh. Он охватывает управление репозиторием (создание, удаление, клонирование, разветвление), рабочие процессы CI/CD (действия GitHub), проблемы, запросы на извлечение, выпуски и другие операции GitHub. Запускается по запросам, в которых упоминается GitHub, репозитории, PR, проблемы, действия или рабочие процессы.
