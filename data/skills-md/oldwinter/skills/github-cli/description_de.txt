Diese Fähigkeit sollte verwendet werden, wenn Benutzer über die gh-CLI mit GitHub interagieren müssen. Es umfasst die Repository-Verwaltung (Erstellen, Löschen, Klonen, Fork), CI/CD-Workflows (GitHub-Aktionen), Issues, Pull Requests, Releases und andere GitHub-Vorgänge. Wird bei Anfragen ausgelöst, die GitHub, Repos, PRs, Probleme, Aktionen oder Workflows erwähnen.
