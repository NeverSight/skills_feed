Cette compétence doit être utilisée lorsque les utilisateurs doivent interagir avec GitHub via la CLI gh. Il couvre la gestion du référentiel (créer, supprimer, cloner, fork), les workflows CI/CD (actions GitHub), les problèmes, les demandes d'extraction, les versions et d'autres opérations GitHub. Se déclenche sur les requêtes mentionnant GitHub, les dépôts, les PR, les problèmes, les actions ou les workflows.
