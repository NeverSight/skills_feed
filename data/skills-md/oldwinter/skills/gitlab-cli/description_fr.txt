Cette compétence doit être utilisée lorsque les utilisateurs doivent interagir avec GitLab via la CLI glab. Il couvre la gestion des référentiels (créer, supprimer, cloner, fork), les workflows CI/CD (pipelines, tâches, planifications), les demandes de fusion, les problèmes, les versions et d'autres opérations GitLab. Se déclenche sur les requêtes mentionnant GitLab, les dépôts, les MR, les problèmes, les pipelines ou les workflows CI/CD.
