Esta habilidad debe usarse cuando los usuarios necesitan consultar los detalles de uso y costos de AWS para una fecha específica. Admite consultas de costos a nivel de servicio (por ejemplo, EC2, S3, RDS) y profundización hasta el nivel de tipo de uso (por ejemplo, tipos de instancia, clases de almacenamiento, transferencia de datos).
