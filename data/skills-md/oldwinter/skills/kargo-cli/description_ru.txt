Этот навык следует использовать, когда пользователям необходимо управлять прогрессивной доставкой через Kargo CLI. Он охватывает управление грузоперевозками, продвижение по этапам, состояние склада и операции по развертыванию конвейера. Интегрируется с ArgoCD для синхронизации GitOps. Запускается по запросам, в которых упоминаются карго, фрахт, продвижение по этапам, поэтапная доставка или конвейеры развертывания.
