يجب استخدام هذه المهارة عندما يحتاج المستخدمون إلى التفاعل مع خدمات AWS عبر واجهة سطر الأوامر (CLI). ويغطي جميع خدمات AWS بما في ذلك EC2 وECS وEKS وLambda وS3 وRDS وDynamoDB وVPC وRoute53 وCloudFront وBedrock والدعم والفوترة والمزيد. يدعم الاستعلام وإنشاء وتعديل وحذف الموارد والمراقبة وتصحيح الأخطاء وتحليل التكلفة. يتم تشغيل الطلبات التي تشير إلى AWS، أو الموارد السحابية، أو أسماء خدمات AWS محددة.
