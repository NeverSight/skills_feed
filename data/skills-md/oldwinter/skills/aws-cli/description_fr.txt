Cette compétence doit être utilisée lorsque les utilisateurs doivent interagir avec les services AWS via CLI. Il couvre tous les services AWS, notamment EC2, ECS, EKS, Lambda, S3, RDS, DynamoDB, VPC, Route53, CloudFront, Bedrock, Support, Billing, etc. Prend en charge les requêtes, la création, la modification, la suppression de ressources, la surveillance, le débogage et l'analyse des coûts. Se déclenche sur les requêtes mentionnant AWS, des ressources cloud ou des noms de services AWS spécifiques.
