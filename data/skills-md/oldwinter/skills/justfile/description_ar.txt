يجب استخدام هذه المهارة عندما يريد المستخدمون إنشاء ملفات Justfiles أو تحويلها أو إدارتها لأتمتة الأوامر. فهو يتعامل مع تحويل Makefile أو البرامج النصية npm أو أوامر shell إلى تنسيق Justfile، وإنشاء قوالب خاصة بالمشروع، وتوفير إرشادات بناء جملة Justfile. يتم تشغيل الطلبات التي تشير إلى justfile أو الأمر فقط أو مشغل الأوامر أو تحويل Makefile أو أتمتة المهام.
