يجب استخدام هذه المهارة عندما يطلب المستخدم إضافة خدمة API جديدة لجهة خارجية إلى نظام مراقبة الفوترة/الحصص في AWS. فهو يتعامل مع عملية الإعداد الكاملة بما في ذلك إنشاء المحول ونشر Lambda وإنذارات CloudWatch وتحديثات لوحة المعلومات والتحقق. تشغيل الطلبات التي تشير إلى "إضافة مراقبة خدمة"، أو "مراقبة رصيد واجهة برمجة التطبيقات"، أو "تنبيهات حصة الإعداد"، أو "الإضافة إلى لوحة تحكم الفوترة"، أو طلبات تكامل الخدمة المشابهة.
