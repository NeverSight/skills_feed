Questa competenza deve essere utilizzata quando l'utente richiede di aggiungere un nuovo servizio API di terze parti al sistema di fatturazione/monitoraggio delle quote AWS. Gestisce l'intero processo di onboarding, inclusa la creazione dell'adattatore, la distribuzione Lambda, gli allarmi CloudWatch, gli aggiornamenti del dashboard e la verifica. Si attiva su richieste che menzionano "aggiunta del monitoraggio del servizio", "monitoraggio del saldo API", "configurazione degli avvisi di quota", "aggiunta al dashboard di fatturazione" o richieste simili di integrazione del servizio.
