Esta habilidad debe usarse cuando el usuario solicita agregar un nuevo servicio API de terceros al sistema de monitoreo de cuotas/facturación de AWS. Maneja el proceso de incorporación completo, incluida la creación de adaptadores, la implementación de Lambda, las alarmas de CloudWatch, las actualizaciones del panel y la verificación. Se activan solicitudes que mencionan "agregar monitoreo de servicios", "monitorear el saldo de API", "configurar alertas de cuota", "agregar al panel de facturación" o solicitudes de integración de servicios similares.
