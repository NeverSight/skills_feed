Esta habilidad proporciona un enfoque sistemático para investigar y corregir errores y problemas en el código base. Utilice esta habilidad cuando los usuarios informen errores, comportamientos inesperados, fallas en las pruebas o soliciten soluciones para problemas específicos. La habilidad guía a través del registro de problemas, análisis de causa raíz, evaluación de impacto, cambios mínimos de código, validación de calidad y pruebas E2E.
