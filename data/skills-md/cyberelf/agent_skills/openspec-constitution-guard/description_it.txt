Questa competenza compone i file di costituzione del progetto AGENTS.md in openspec/config.yaml per inserire porte di convalida della qualità nei flussi di lavoro OpenSpec. Utilizza questa capacità quando inizializzi openspec per la prima volta in un progetto o quando i file AGENTS.md vengono aggiornati. La competenza garantisce che gli artefatti OpenSpec siano convalidati rispetto ai criteri di qualità specifici del progetto stabiliti dalle costituzioni.
