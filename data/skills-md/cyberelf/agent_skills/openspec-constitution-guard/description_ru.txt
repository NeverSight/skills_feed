Этот навык объединяет файлы конституции проекта AGENTS.md в openspec/config.yaml для внедрения шлюзов проверки качества в рабочие процессы OpenSpec. Используйте этот навык при первой инициализации openspec в проекте или при обновлении файлов AGENTS.md. Этот навык обеспечивает проверку артефактов openspec на соответствие критериям качества, специфичным для проекта, из конституций.
