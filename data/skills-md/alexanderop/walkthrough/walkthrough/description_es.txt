Genera un archivo HTML autónomo con un diagrama de sirena interactivo en el que se puede hacer clic (diagrama de flujo o diagrama ER) que explica cómo funciona una característica, flujo, arquitectura o esquema de base de datos de código base. Diseñado para una incorporación rápida: cada recorrido es un modelo mental visual que se puede leer en menos de 2 minutos. Úselo cuando se le solicite realizar un recorrido, explicar un flujo, rastrear una ruta de código, mostrar cómo funciona algo, explicar la arquitectura, visualizar un esquema de base de datos o explorar un modelo de datos.
