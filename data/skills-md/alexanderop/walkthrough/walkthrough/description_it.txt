Genera un file HTML autonomo con un diagramma Sirena interattivo e cliccabile (diagramma di flusso o diagramma ER) che spiega come funziona una funzionalità di codebase, un flusso, un'architettura o uno schema di database. Progettato per un onboarding rapido: ogni procedura dettagliata è un modello mentale visivo leggibile in meno di 2 minuti. Da utilizzare quando viene richiesto di eseguire una procedura dettagliata, spiegare un flusso, tracciare un percorso di codice, mostrare come funziona qualcosa, spiegare l'architettura, visualizzare uno schema di database o esplorare un modello di dati.
