Génère un fichier HTML autonome avec un diagramme Sirène interactif et cliquable (organigramme ou diagramme ER) qui explique le fonctionnement d'une fonctionnalité de base de code, d'un flux, d'une architecture ou d'un schéma de base de données. Conçu pour une intégration rapide : chaque procédure pas à pas est un modèle mental visuel lisible en moins de 2 minutes. À utiliser lorsqu'on vous demande de vous présenter, d'expliquer un flux, de tracer un chemin de code, de montrer comment quelque chose fonctionne, d'expliquer l'architecture, de visualiser un schéma de base de données ou d'explorer un modèle de données.
