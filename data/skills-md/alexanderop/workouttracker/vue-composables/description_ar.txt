اكتب مكونات Vue 3 عالية الجودة باتباع الأنماط وأفضل الممارسات المعمول بها. يُستخدم عند إنشاء مواد قابلة للتركيب جديدة، أو إعادة هيكلة العناصر الموجودة، أو مراجعة التعليمات البرمجية القابلة للتركيب. تتضمن المشغلات طلبات "إنشاء ملف قابل للتركيب"، أو "كتابة دالة استخدام*"، أو "استخراج المنطق إلى ملف قابل للتركيب"، أو أي مهمة منطقية قابلة لإعادة الاستخدام لـ Vue Composition API.
