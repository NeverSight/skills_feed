اكتب اختبارات تكامل Vue 3 باستخدام وضع متصفح Vitest وكائنات الصفحة. استخدم بشكل استباقي عند إضافة اختبارات لطرق العرض أو المكونات ذات التوجيه أو تدفقات المستخدم أو الميزات التي تشمل مكونات متعددة. تتضمن المشغلات "إضافة اختبارات"، أو "اختبارات كتابة"، أو "اختبار هذا"، أو "اختبارات الاحتياجات"، أو "اختبارات التكامل"، أو "اختبار الميزة"، أو "اختبار تدفق المستخدم"، أو أي مهمة تتضمن اختبار تفاعلات واجهة المستخدم، أو التنقل، أو مربعات الحوار، أو النماذج، أو سير العمل متعدد الخطوات.
