Créez et gérez des référentiels Dexie/IndexedDB avec des interfaces de type sécurisé, des convertisseurs et des opérations CRUD standardisées. À utiliser lors de (1) l'ajout de stockage d'entités, (2) la mise en œuvre d'opérations de sauvegarde/chargement/suppression, (3) la conception de schémas et d'index de base de données, (4) la conversion entre les types de base de données (Db*) et de domaine, (5) la gestion des erreurs ou des migrations de base de données, (6) l'utilisation de référentiels existants (SettingsRepository, WorkoutsRepository, TemplatesRepository, CustomExercisesRepository, BenchmarksRepository, ActiveWorkoutRepository). Les déclencheurs incluent « base de données », « référentiel », « enregistrer les données », « extraire de la base de données », « supprimer du stockage », « schéma de base de données », « table de base de données », « index », « migration », « persister », « convertir l'entraînement », « convertisseur », « buildPartialUpdate », « référentiel fictif », « erreur de base de données », « opérations groupées », « importation/exportation » ou des noms de référentiel spécifiques.
