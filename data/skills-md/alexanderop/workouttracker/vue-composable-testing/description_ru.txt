Протестируйте компонуемые элементы Vue 3 с помощью Vitest, следуя лучшим практикам. Используйте при написании тестов для составных объектов, создании помощников тестирования для реактивности Vue или когда вас просят «проверить составной объект», «написать тесты для использования*» или «помочь протестировать функции композиции Vue». Охватывает независимые компонуемые объекты (прямое тестирование) и зависимые компонуемые объекты (тестирование жизненного цикла/внедрения с помощью помощников).
