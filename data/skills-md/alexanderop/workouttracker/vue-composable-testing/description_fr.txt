Testez les composables Vue 3 avec Vitest en suivant les meilleures pratiques. À utiliser lors de l'écriture de tests pour les composables, de la création d'assistants de test pour la réactivité de Vue, ou lorsqu'il est demandé de « tester un composable », « d'écrire des tests pour l'utiliser* » ou « d'aider à tester les fonctions de composition de Vue ». Couvre les composables indépendants (tests directs) et les composables dépendants (tests de cycle de vie/injection avec assistants).
