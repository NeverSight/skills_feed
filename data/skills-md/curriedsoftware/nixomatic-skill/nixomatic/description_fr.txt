Maintient les instructions de l'environnement de développement dans le projet README.md à l'aide de nixomatic. Lorsque l'utilisateur demande à créer, compiler, tester, lint, formater, vérifier le type, exécuter ou configurer un projet, ou lorsqu'une commande échoue parce qu'un outil ou un runtime n'est pas installé, analysez les fichiers du projet pour déterminer les packages requis et utilisez nix develop avec une URL nixomatic.com. Assurez-vous que le README.md documente comment reproduire l'environnement avec nix develop et docker run nixos/nix.
