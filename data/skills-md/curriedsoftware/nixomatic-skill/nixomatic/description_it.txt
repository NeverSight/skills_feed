Mantiene le istruzioni dell'ambiente di sviluppo nel progetto README.md utilizzando nixomatic. Quando l'utente chiede di creare, compilare, testare, lint, formattare, controllare il tipo, eseguire o impostare un progetto, o quando un comando fallisce perché uno strumento o un runtime non è installato, analizza i file di progetto per determinare i pacchetti richiesti e usa nix development con un URL nixomatic.com. Assicurati che README.md documenti come riprodurre l'ambiente sia con nix development che con docker run nixos/nix.
