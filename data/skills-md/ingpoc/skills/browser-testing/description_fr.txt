À utiliser pour tester des applications Web, déboguer les erreurs de la console du navigateur, automatiser les interactions avec les formulaires ou vérifier les implémentations de l'interface utilisateur. Chargez pour les tests d'hôte local, les tests d'applications authentifiées (Gmail, Notion) ou l'enregistrement de GIF de démonstration. Nécessite l'extension Chrome 1.0.36+, Claude Code 2.0.73+, forfait payant.
