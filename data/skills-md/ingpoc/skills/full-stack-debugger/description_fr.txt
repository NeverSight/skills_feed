Cette compétence doit être utilisée lors du débogage de problèmes de pile complète qui couvrent les couches de l'interface utilisateur, du backend et de la base de données. Il fournit un flux de travail systématique pour détecter les erreurs, analyser les causes profondes, appliquer des correctifs de manière itérative et vérifier les solutions via des redémarrages automatisés du serveur et des tests basés sur le navigateur. Idéal pour les scénarios tels que des planificateurs défaillants, des erreurs d'importation, des problèmes de base de données ou des problèmes de charge utile d'API où les problèmes proviennent du code backend mais se manifestent dans l'interface utilisateur.
