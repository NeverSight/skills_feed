Используйте при реализации перехватчиков, БЛОКИРУЮЩИХ недопустимые действия, создающих шлюзы качества для переходов между состояниями или обеспечивающих проверку «проверено: истина». Нагружайте при проектировании механизмов обеспечения соблюдения. Для блокировки используется код выхода 2, модификация JSON PermissionDecision:deny или UpdateInput. Правила — это инструкции; крючки являются принудительными.
