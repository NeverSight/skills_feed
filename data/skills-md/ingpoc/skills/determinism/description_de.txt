Verwenden Sie diese Option, wenn Sie Ergebnisse mit Code anstelle einer LLM-Beurteilung überprüfen, Eingabeaufforderungen mit Hashes versionieren oder reproduzierbares Agentenverhalten sicherstellen. Laden Sie es für jede kritische Überprüfung. Skripte geben boolesche Exit-Codes zurück, keine subjektiven Bewertungen. Eingabeaufforderungen verwenden semantische Versionierung mit SHA256-Validierung.
