Используйте при настройке крючков для конкретного проекта для принудительного применения кода Клода. Загрузка во время состояния INIT, когда .claude/hooks/ отсутствует или при создании нового проекта. Устанавливает 5 перехватчиков, которые читают файл .claude/config/project.json (тесты, работоспособность, зависимости).
