Intégrez les fragments Fragno existants dans les applications : installez des packages de fragments, configurez les instances de serveur (et les adaptateurs de base de données si nécessaire), montez les gestionnaires de serveur pour chaque framework, créez des intégrations côté client et utilisez des hooks/composables. À utiliser lorsqu'il est demandé de connecter un fragment à l'application de l'utilisateur.
