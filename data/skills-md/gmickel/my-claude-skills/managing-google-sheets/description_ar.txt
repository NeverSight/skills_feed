قراءة بيانات جداول بيانات Google وكتابتها وتحديثها عبر واجهة سطر الأوامر (CLI). يُستخدم عندما يطلب المستخدم قراءة بيانات جدول البيانات، أو تحديث الخلايا، أو إلحاق صفوف، أو العمل مع جداول بيانات Google. يتم تشغيله عند الإشارة إلى جداول البيانات أو الأوراق أو جداول بيانات Google أو البيانات الجدولية في السحابة أو أسماء أوراق محددة مثل "المشاريع" أو "المهام".
