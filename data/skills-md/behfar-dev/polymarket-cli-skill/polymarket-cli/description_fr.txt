Guide d'utilisation de Polymarket CLI — un outil de ligne de commande basé sur Rust pour parcourir les marchés de prédiction, effectuer des transactions, gérer les portefeuilles et interagir avec les contrats en chaîne sur Polymarket. Utilisez cette compétence chaque fois que l'utilisateur mentionne Polymarket, les marchés de prédiction CLI, polymarket-cli, le trading à partir du terminal, les cotes du marché, le carnet d'ordres CLOB, les jetons conditionnels (CTF) ou souhaite parcourir/échanger les marchés de prédiction par programme. Se déclenche également lorsque l'utilisateur souhaite créer un script sur les données de Polymarket, vérifier les prix du marché, passer des ordres à cours limité ou au marché, gérer des positions ou interagir avec l'API de Polymarket à partir d'un terminal ou d'un flux de travail automatisé.
