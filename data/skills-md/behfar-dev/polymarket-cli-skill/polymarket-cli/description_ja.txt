Polymarket CLI の使用ガイド。予測市場の閲覧、取引の実行、ウォレットの管理、Polymarket 上のオンチェーン コントラクトの操作を行うための Rust ベースのコマンドライン ツールです。ユーザーが Polymarket、予測市場 CLI、polymarket-cli、ターミナルからの取引、市場オッズ、CLOB オーダーブック、条件付きトークン (CTF) に言及した場合、または予測市場をプログラムで参照/取引したい場合は常に、このスキルを使用します。また、ユーザーが Polymarket データに対してスクリプトを実行したり、市場価格を確認したり、指値注文や成行注文を出したり、ポジションを管理したり、端末や自動ワークフローから Polymarket の API を操作したりする場合にもトリガーします。
