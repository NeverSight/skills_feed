Google Agent Development Kit (ADK) を使用して、本番環境に対応した AI エージェントを TypeScript で構築します。 LlmAgent、ワークフロー エージェント (SequentialAgent、ParallelAgent、LoopAgent)、カスタム エージェント (runAsyncImpl を備えた BaseAgent)、Zod スキーマを備えた FunctionTool、MCP および OpenAPI ツール、マルチエージェント オーケストレーション、セッションおよび状態管理 (outputKey、アプリ/ユーザー/一時プレフィックス、CallbackContext)、コールバックとガードレール、テストと評価、バックエンド フレームワークを使用してエージェントの TypeScript/JavaScript アプリケーションを作成する場合に使用します。 （Express、Hono）、または Cloud Run またはコンテナにデプロイします。 「エージェント TypeScript の構築」、「ADK TypeScript」、「ADK TS」、「@google/adk」、「マルチエージェント システム TypeScript」、「エージェント アプリケーション TypeScript」、「Gemini エージェント TypeScript」でトリガーされます。
