أنشئ وكلاء الذكاء الاصطناعي الجاهزين للإنتاج في TypeScript باستخدام Google Agent Development Kit (ADK). يُستخدم عند إنشاء تطبيقات TypeScript/JavaScript وكيلة باستخدام LlmAgent، ووكلاء سير العمل (SequentialAgent، ParallelAgent، LoopAgent)، والوكلاء المخصصين (BaseAgent مع runAsyncImpl)، وFunctionTool مع مخططات Zod، وأدوات MCP وOpenAPI، والتنسيق متعدد الوكلاء، وإدارة الجلسة والحالة (مفتاح الإخراج، وبادئات التطبيق/المستخدم/درجة الحرارة، CallbackContext)، وعمليات الاسترجاعات وحواجز الحماية، والاختبار والتقييم، أطر عمل الواجهة الخلفية (Express، Hono)، أو النشر إلى Cloud Run أو الحاويات. مشغلات على "إنشاء وكيل TypeScript"، و"ADK TypeScript"، و"ADK TS"، و"@google/adk"، و"TypeScript لنظام متعدد الوكلاء"، و"Agent application TypeScript"، و"Gemini Agent TypeScript".
