Wird zum Erstellen oder Debuggen von Elementen verwendet, die in einem Browser ausgeführt werden. Verwenden Sie es, wenn Sie das DOM überprüfen, Konsolenfehler erfassen, Netzwerkanforderungen analysieren, ein Leistungsprofil erstellen oder die visuelle Ausgabe mit echten Laufzeitdaten über Chrome DevTools MCP überprüfen müssen.
