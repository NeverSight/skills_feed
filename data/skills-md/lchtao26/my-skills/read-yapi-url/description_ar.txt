جلب وثائق API من منصة YApi عن طريق URL. يُستخدم عندما يطلب المستخدم قراءة وثائق YApi API أو جلبها أو الحصول عليها، أو يقدم عناوين URL لـ YApi مثل "https://yapi.nocode-tech.com/project/{id}/interface/api/{api_id}" أو "https://yapi.nocode-tech.com/project/{id}/interface/api/cat_{cat_id}". يدعم كلاً من عناوين URL لواجهة برمجة التطبيقات الفردية وعناوين URL للفئة. إرجاع بيانات وثائق API الأولية.
