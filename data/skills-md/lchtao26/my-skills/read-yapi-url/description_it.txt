Recupera la documentazione API dalla piattaforma YApi tramite URL. Da utilizzare quando l'utente chiede di leggere, recuperare o ottenere la documentazione dell'API YApi o fornisce URL YApi come "https://yapi.nocode-tech.com/project/{id}/interface/api/{api_id}" o "https://yapi.nocode-tech.com/project/{id}/interface/api/cat_{cat_id}". Supporta sia URL API singoli che URL di categoria. Restituisce i dati grezzi della documentazione API.
