Получите документацию API с платформы YApi по URL-адресу. Используйте, когда пользователь запрашивает чтение, получение или получение документации по API YApi или предоставляет URL-адреса YApi, например «https://yapi.nocode-tech.com/project/{id}/interface/api/{api_id}» или «https://yapi.nocode-tech.com/project/{id}/interface/api/cat_{cat_id}». Поддерживает как отдельные URL-адреса API, так и URL-адреса категорий. Возвращает необработанные данные документации API.
