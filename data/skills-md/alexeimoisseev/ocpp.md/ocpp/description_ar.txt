مرجع بروتوكول OCPP لتطوير البنية التحتية لشحن المركبات الكهربائية. يغطي OCPP 2.0.1 وOCPP 1.6J. يُستخدم عند العمل مع رسائل OCPP، أو رمز محطة الشحن، أو الواجهات الخلفية للنظام CSMS/النظام المركزي، أو الشحن الذكي، أو التعامل مع المعاملات، أو بروتوكولات شحن المركبات الكهربائية. يتم التنشيط بناءً على الكلمات الرئيسية: OCPP، أو محطة الشحن، أو نقطة الشحن، أو CSMS، أو النظام المركزي، أو EVSE، أو ملف تعريف الشحن، أو BootNotification، أو TransactionEvent، أو StartTransaction، أو StopTransaction، أو SetChargingProfile، أو أي اسم لرسالة OCPP.
