Ranger Finance SDK для создания приложений для бессрочной торговли фьючерсами на Solana. Первый агрегатор Solana Perps — агрегирует ликвидность по нескольким протоколам преступников (Drift, Flash, Adrena, Jupiter). Используйте при интеграции индивидуальной торговли, интеллектуальной маршрутизации ордеров, управления позициями или создания торговых агентов с искусственным интеллектом.
