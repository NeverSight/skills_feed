Полное руководство по @solana/kit — современному, поддерживающему древовидную структуру JavaScript SDK с нулевой зависимостью от Anza. Охватывает RPC-соединения, подписывающих лиц, построение транзакций с помощью канала, подписание, отправку и получение учетной записи с полной поддержкой TypeScript.
