Полное руководство по протоколу Light на Solana — включает сжатие ZK для бесплатных сжатых токенов и КПК с доказательствами с нулевым разглашением, а также программу Light Token для высокопроизводительного стандарта токенов (в 200 раз дешевле, чем SPL). Охватывает TypeScript SDK, методы JSON RPC и полные шаблоны интеграции.
