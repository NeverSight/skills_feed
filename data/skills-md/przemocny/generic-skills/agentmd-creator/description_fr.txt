À utiliser lorsque l'utilisateur souhaite créer un fichier de configuration d'agent, configurer l'assistant AI pour un rôle ou un domaine spécifique, configurer l'agent pour les flux de travail d'entreprise, générer AGENTS.md ou CLAUDE.md, personnaliser le comportement de l'IA pour l'organisation ou définir les limites et les directives de l'agent. Déclenchez des expressions telles que créer une configuration d'agent, configurer l'assistant IA, créer AGENTS.md, configurer l'agent pour le rôle, l'agent IA pour le domaine professionnel ou aidez-moi à configurer Claude/Curseur/Windsurf.
