Этот навык следует использовать, когда пользователь просит «создать подсказку Suno», «написать песню Suno», «создать музыку с помощью Suno», «помогите мне с Suno», «создать подсказку к песне», «создать текст песни для Suno», «создать музыкальную подсказку» или упоминает создание музыки Suno AI. Предоставляет комплексное руководство по созданию профессиональных подсказок Suno с использованием передовых стратегий подсказок, структурированного форматирования в пределах 1000 символов (БЕЗ пустых строк между разделами), оптимизации параметров, методов, специфичных для жанра, интерактивных вопросов с эффективным сбором названий проектов, автоматического исследования исполнителей/песен с помощью субагента (загрузка из Интернета + извлечение шаблонов), автоматического экспорта файлов в организованные каталоги проектов, избежание ошибок искусственного интеллекта для аутентичных, ориентированных на человека текстов, безопасных для авторских прав описаний стилей, в которых не используются имена исполнителей/альбомов/песен, подсчет символов. утилиты для точной проверки и опциональная независимая проверка качества через субагента для профессиональной оценки.
