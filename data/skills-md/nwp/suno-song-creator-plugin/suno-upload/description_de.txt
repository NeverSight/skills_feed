Laden Sie mithilfe der Chrome-Automatisierung eine Suno prompt.md-Datei auf suno.com hoch, ohne dass ein menschliches Eingreifen erforderlich ist. Analysiert die Eingabeaufforderungsdatei, navigiert zur Erstellungsoberfläche von Suno, füllt alle Formularfelder einschließlich Schiebereglern aus (Text, Stil, Titel, Verrücktheit, Stileinfluss, Stimmgeschlecht, Stile ausschließen) und sendet sie zur Songgenerierung. Verwendet bewährte koordinatenbasierte Schiebereglermanipulation für zuverlässige Automatisierung. Verwenden Sie diesen Skill, wenn der Benutzer „Auf Suno hochladen“, „Auf Suno erstellen“, „Mit Suno generieren“, „An Suno übermitteln“ auffordert oder eine generierte Eingabeaufforderung automatisch auf die Suno-Website hochladen möchte.
