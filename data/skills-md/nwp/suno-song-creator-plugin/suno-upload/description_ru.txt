Загрузите файл Suno Prompt.md на suno.com с помощью автоматизации Chrome, БЕЗ вмешательства человека. Анализирует файл подсказки, переходит к интерфейсу создания Suno, заполняет все поля формы, включая ползунки (текст, стиль, название, странность, влияние стиля, пол вокала, исключение стилей) и отправляет на генерацию песни. Использует проверенную манипуляцию ползунком на основе координат для надежной автоматизации. Используйте этот навык, когда пользователь просит «загрузить в Suno», «создать в Suno», «создать с помощью Suno», «отправить в Suno» или хочет автоматически загрузить сгенерированное приглашение на веб-сайт Suno.
