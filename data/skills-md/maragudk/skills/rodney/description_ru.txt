Руководство по автоматизации взаимодействия с браузером Chrome с помощью интерфейса командной строки Rodney. Этот навык следует использовать при выполнении задач веб-автоматизации, таких как переход по страницам, создание снимков экрана, нажатие элементов, заполнение форм, извлечение содержимого страницы или любое другое взаимодействие в браузере.
