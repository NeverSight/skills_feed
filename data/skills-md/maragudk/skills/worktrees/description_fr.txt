Guide d'utilisation des arbres de travail git pour paralléliser le développement avec des agents de codage. Utilisez cette compétence lorsque l'utilisateur demande à travailler dans un nouvel arbre de travail ou souhaite travailler sur une fonctionnalité distincte de manière isolée (par exemple, « Travailler dans un nouvel arbre de travail », « Créer un arbre de travail pour la fonctionnalité X »).
