Base de connaissances de l'API Polymarket CLOB pour la gestion des commandes, les événements WebSocket et les opérations de trading. À utiliser lorsque vous travaillez avec le cycle de vie des commandes (passer, remplir, annuler), déboguer les événements WebSocket USER_TRADE/USER_ORDER, comprendre les mappages de champs d'API ou implémenter la logique de trading.
