Exécute des requêtes de métriques sur Axiom MetricsDB via des scripts. Découvre les métriques, les balises et les valeurs de balises disponibles. À utiliser lorsqu'on vous demande d'interroger des métriques, d'explorer des ensembles de données de métriques, de vérifier les valeurs de métriques ou d'examiner les données de métriques OTel.
