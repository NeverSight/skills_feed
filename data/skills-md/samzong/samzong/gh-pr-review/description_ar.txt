قم بمراجعة طلب سحب GitHub بصفتك مالك مشروع مسؤول باستخدام `gh` CLI. يُستخدم عندما يقدم المستخدم عنوان URL للعلاقات العامة (على سبيل المثال https://github.com/ORG/REPO/pull/N) أو رقم العلاقات العامة لـ git repo الحالي (يفضل المنبع، وإلا الأصل) ويريد مستند مراجعة على مستوى المالك `review-N.md` مكتوبًا باللغة الصينية مع تعليقات GitHub القابلة للنسخ واللصق باللغة الإنجليزية. قم بنطاق المراجعة لتشمل الأسطر التي تم تغييرها بواسطة العلاقات العامة (لا تدقق في التعليمات البرمجية الموجودة مسبقًا غير ذات الصلة)، ولكن قم بتطبيق أفضل الممارسات وقم بوضع علامة على أي أخطاء واضحة أو مشكلات أمنية أو حالات فشل CI الناتجة عن التغيير.
