`gh` CLI를 사용하여 담당 프로젝트 소유자로서 GitHub 끌어오기 요청을 검토하세요. 사용자가 PR URL(예: https://github.com/ORG/REPO/pull/N) 또는 현재 git repo에 대한 PR 번호(업스트림 선호, 다른 원본 선호)를 제공하고 영어로 복사하여 붙여넣을 수 있는 GitHub 주석과 함께 중국어로 작성된 소유자 등급 검토 문서 `review-N.md`를 원할 때 사용합니다. PR에 의해 변경된 행으로 검토 범위를 지정하되(관련되지 않은 기존 코드를 선별하지 않음) 모범 사례를 적용하고 변경으로 인해 발생하는 명확한 버그, 보안 문제 또는 CI 오류에 플래그를 지정합니다.
