Genere un wiki de proyecto profundo y centrado en el código y un sitio de documentación de VitePress (estilo DeepWiki) para un repositorio. Úselo cuando se le solicite crear un wiki/manual completo para una nueva base de código, crear un sitio de documentación de código primero o generar una carpeta `codewiki/` con Markdown estructurado, diagramas y navegación basados ​​en el código fuente (documentos no existentes).
