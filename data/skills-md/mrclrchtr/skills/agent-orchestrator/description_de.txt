Koordinieren Sie komplexe Arbeiten mithilfe einer phasengesteuerten Multi-Agenten-Engineering-Schleife (Prüfung → Entwurf → Implementierung → Überprüfung → Validierung → Bereitstellung). Verwenden Sie diese Funktion, wenn Sie eine Aufgabe in Subsysteme aufteilen, zwei unabhängige Prüfungen durchführen, Ergebnisse in einer Liste bestätigter Probleme abgleichen, Korrekturen in Clustern delegieren, PASS-Gates für doppelte Überprüfungen erzwingen und eine End-to-End-Bereitstellung vorantreiben müssen. Entdecken und nutzen Sie lieber andere Spezialfähigkeiten, wenn Sie einen Teil der Arbeit schneller oder zuverlässiger ausführen können.
