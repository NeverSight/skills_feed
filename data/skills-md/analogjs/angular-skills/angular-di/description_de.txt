Implementieren Sie die Abhängigkeitsinjektion in Angular v20+ mithilfe von inject(), Injektionstokens und Anbieterkonfiguration. Verwendung für die Servicearchitektur, Bereitstellung von Abhängigkeiten auf verschiedenen Ebenen, Erstellung injizierbarer Token und Verwaltung von Singleton- und Scope-Services. Löst die Diensterstellung, die Konfiguration von Anbietern, die Verwendung von Injektionstokens oder das Verständnis der DI-Hierarchie aus.
