Cree directivas personalizadas en Angular v20+ para manipulación DOM y extensión de comportamiento. Úselo para directivas de atributos que modifican el comportamiento/apariencia del elemento, directivas estructurales para portales/superposiciones y directivas de host para composición. Activadores para crear comportamientos DOM reutilizables, ampliar la funcionalidad de elementos o componer comportamientos entre componentes. Nota: utilice @if/@for/@switch nativo para el flujo de control, no directivas estructurales personalizadas.
