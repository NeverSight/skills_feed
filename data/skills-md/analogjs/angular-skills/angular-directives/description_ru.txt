Создавайте собственные директивы в Angular v20+ для манипулирования DOM и расширения поведения. Используйте для директив атрибутов, которые изменяют поведение/внешний вид элемента, структурных директив для порталов/наложений и директив хоста для композиции. Запускается при создании многократно используемого поведения DOM, расширении функциональности элемента или составлении поведения между компонентами. Примечание. Для потока управления используйте собственный @if/@for/@switch, а не пользовательские структурные директивы.
