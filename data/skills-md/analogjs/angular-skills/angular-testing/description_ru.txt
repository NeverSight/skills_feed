Напишите модульные и интеграционные тесты для приложений Angular v21+, используя Vitest или Jasmine с TestBed, наборами компонентов и современными шаблонами тестирования. Используйте для тестирования компонентов с помощью сигналов, обнаружения изменений OnPush, сервисов с помощью inject() и HTTP-взаимодействий. Запускается при создании теста, тестировании компонентов на основе сигналов, имитации зависимостей или настройке тестовой инфраструктуры.
