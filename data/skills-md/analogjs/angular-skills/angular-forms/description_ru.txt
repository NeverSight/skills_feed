Создавайте формы на основе сигналов в Angular v21+, используя новый API Signal Forms. Используйте для создания форм с автоматической двусторонней привязкой, проверкой на основе схемы, управлением состоянием полей и динамическими формами. Запускается при реализации формы, добавлении проверки, создании многошаговых форм или создании форм с условными полями. Формы сигналов являются экспериментальными, но рекомендуются для новых проектов Angular.
