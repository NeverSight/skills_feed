Implémentez le rendu et l'hydratation côté serveur dans Angular v20+ à l'aide de @angular/ssr. À utiliser pour la configuration SSR, les stratégies d'hydratation, le pré-rendu des pages statiques et la gestion des API du navigateur uniquement. Se déclenche sur la configuration SSR, corrige les inadéquations d'hydratation, pré-rendu des itinéraires ou rend le code compatible SSR.
