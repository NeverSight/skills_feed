Visionneuse de flux RSS/Atom CLI conviviale pour les agents IA avec découverte, recherche et importation/exportation OPML. À utiliser lorsqu'un agent doit gérer des flux RSS, lire des articles ou exécuter la découverte de flux à partir du terminal. Déclenché par : "lire les flux RSS", "afficher le RSS", "lecteur RSS CLI", "rss-viewer", "découvrir les flux RSS", "importer OPML", "rechercher les flux" ou "découvrir les flux de recherche".
