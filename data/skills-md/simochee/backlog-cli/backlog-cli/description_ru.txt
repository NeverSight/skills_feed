CLI для управления проектами Backlog (от Nulab). Используйте этот навык, когда:
(1) Перечисление, создание, редактирование, закрытие или комментирование проблем.
(2) Создание, перечисление, объединение или комментирование запросов на включение
(3) Просмотр, создание или редактирование страниц Wiki.
(4) Запрос настроек проекта (типы проблем, статусы, категории, этапы, участники)
(5) Проверка уведомлений, звездочек или часов
(6) Выполнение необработанных запросов API через `backlog api`
