CLI para la gestión de proyectos Backlog (por Nulab). Utilice esta habilidad cuando:
(1) Listar, crear, editar, cerrar o comentar problemas
(2) Crear, enumerar, fusionar o comentar solicitudes de extracción
(3) Ver, crear o editar páginas Wiki
(4) Consultar la configuración del proyecto (tipos de problemas, estados, categorías, hitos, miembros)
(5) Verificar notificaciones, estrellas o relojes
(6) Realizar solicitudes API sin procesar a través de `api backlog`
