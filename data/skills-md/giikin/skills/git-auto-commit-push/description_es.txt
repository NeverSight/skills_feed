Git prepara, confirma y empuja automáticamente. Utilice esta habilidad cuando el usuario solicite confirmar código, enviar código, confirmar, enviar, confirmar y enviar, auto-commit, stage-commit-push o cualquier operación que involucre git commit/push. También se aplica cuando el usuario dice "confirmarlo", "enviar el código", "ayúdame a confirmar", "enviarlo hacia arriba", "solo confirmar" o "simplemente confirmar".
