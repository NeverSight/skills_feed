يقوم Git تلقائيًا بالمراحل والالتزامات والدفعات. استخدم هذه المهارة عندما يطلب المستخدم تنفيذ التعليمات البرمجية، أو دفع التعليمات البرمجية، أو الالتزام، أو الدفع، أو الالتزام والدفع، أو الالتزام التلقائي، أو الالتزام بالمرحلة، أو أي عملية التزام/دفع في git. ينطبق أيضًا على التعبيرات العامية مثل "ارتكبه" أو "ادفع الكود" أو "ساعدني في الالتزام" أو "ادفعه لأعلى" أو "التزم فقط" أو "التزم فقط".
