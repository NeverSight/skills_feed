Stage, commit e push in Git automaticamente. Usa questa abilità quando l'utente chiede di eseguire il commit del codice, inviare il codice, eseguire il commit, inviare, eseguire e inviare, eseguire l'auto-commit, stage-commit-push o qualsiasi frase che coinvolga git commit/push. Si applica anche quando l'utente dice cose come "commit it", "push the code", "help me commit", "push it up", "commit only" o "just commit".
