Git führt automatisch Staging, Commits und Pushs durch. Verwenden Sie diese Fähigkeit, wenn der Benutzer Code festschreiben, Code pushen, festschreiben, pushen, festschreiben und pushen, automatisch festschreiben, Stage-Commit-Push oder eine andere Operation mit Git-Commit/Push ausführen möchte. Dies gilt auch, wenn der Benutzer „Commit“, „Den Code übertragen“, „Helfen Sie mir beim Commit“, „Push it up“, „Nur Commit“ oder „Einfach Commit“ und andere umgangssprachliche Ausdrücke sagt.
