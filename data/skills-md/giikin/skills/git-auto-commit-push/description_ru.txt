Git автоматически создает, фиксирует и отправляет данные. Используйте этот навык, когда пользователь просит зафиксировать код, отправить код, зафиксировать, отправить, зафиксировать и отправить, автоматическую фиксацию, этапную фиксацию-отправить или любую формулировку, включающую операции git commit/push. Также применимо, когда пользователь говорит такие вещи, как «зафиксировать», «отправить код», «помогите мне зафиксировать», «отправить», «только зафиксировать» или «просто зафиксировать».
