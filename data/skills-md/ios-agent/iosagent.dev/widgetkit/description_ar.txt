أنشئ أدوات iOS/macOS/watchOS/visionOS والأنشطة المباشرة ومشاهدة المضاعفات وعناصر التحكم باستخدام إطار عمل Apple WidgetKit. استخدمه عند إنشاء ملحقات عناصر واجهة المستخدم، أو موفري المخطط الزمني، أو عناصر واجهة المستخدم القابلة للتكوين، أو عناصر واجهة مستخدم قفل الشاشة، أو عناصر واجهة المستخدم Smart Stack، أو الأنشطة المباشرة مع ActivityKit، أو عناصر واجهة المستخدم التفاعلية مع الأزرار/المفاتيح، أو مشاهدة المضاعفات. يغطي جميع عائلات عناصر واجهة المستخدم (نظام صغير/متوسط/كبير/كبير جدًا، ملحق دائري/مستطيل/مضمن/زاوية) وأوضاع العرض.
