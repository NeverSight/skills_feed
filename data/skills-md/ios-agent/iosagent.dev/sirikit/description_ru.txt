Интегрируйте голосовое взаимодействие Siri, ярлыки и интеллектуальные предложения в приложения iOS/watchOS с помощью платформы Apple SiriKit. Используйте при реализации расширений Intents, пользовательских намерений, ярлыков Siri, обработки голосовых фраз, разрешения/подтверждения/обработки намерений, IntentsUI для пользовательских интерфейсов Siri, ярлыков для пожертвований или миграции намерений приложения. Охватывает домены системных намерений (обмен сообщениями, платежи, бронирование поездок, тренировки, мультимедиа) и пользовательское определение намерений.
