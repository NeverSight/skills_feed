Apple の ProximityReader フレームワークのための包括的な iOS 開発スキル。 iPhone での Tap to Pay (支払いカード読み取り)、ポイント カード (VAS) 統合、ストア アンド フォワード モード、Verifier API (モバイル ドキュメント / ID 読み取り)、および販売者検出 UI をカバーします。ユーザーが ProximityReader、iPhone での Tap to Pay、iPhone での非接触型決済、NFC 支払い読み取り、ウォレットからのポイントカード読み取り、VAS リクエスト、モバイル運転免許証の確認、iPhone での ID 確認、MobileDocumentReader、PaymentCardReader、PaymentCardReaderSession、またはその他の関連トピックについて言及するたびに、このスキルを使用します。また、ユーザーが販売時点情報管理 (POS) アプリを構築したり、ハードウェアを使用せずに非接触型支払いを受け入れたり、iPhone でデジタル ウォレット パスを読み取ったりしたいときにもトリガーされます。
