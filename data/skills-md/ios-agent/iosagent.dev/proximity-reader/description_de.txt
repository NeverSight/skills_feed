Umfassende iOS-Entwicklungsfähigkeiten für das ProximityReader-Framework von Apple. Deckt „Tap to Pay“ auf dem iPhone (Lesen von Zahlungskarten), die Integration von Treuekarten (VAS), den Store-and-Forward-Modus, die Verifier API (Lesen mobiler Dokumente/IDs) und die Benutzeroberfläche zur Händlersuche ab. Verwenden Sie diesen Skill immer dann, wenn der Benutzer ProximityReader, Tap to Pay auf dem iPhone, kontaktloses Bezahlen auf dem iPhone, Lesen von NFC-Zahlungen, Lesen von Treuekarten aus Wallet, VAS-Anfragen, mobile Führerscheinüberprüfung, ID-Überprüfung auf dem iPhone, MobileDocumentReader, PaymentCardReader, PaymentCardReaderSession oder ein anderes verwandtes Thema erwähnt. Wird auch ausgelöst, wenn der Benutzer eine Point-of-Sale-App (POS) erstellen, kontaktlose Zahlungen ohne Hardware akzeptieren oder digitale Geldbörsenkarten auf dem iPhone lesen möchte.
