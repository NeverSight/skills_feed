Cree documentación del sistema y README de alta calidad para proyectos de software. Úselo cuando necesite: (1) escribir o actualizar la documentación README del proyecto, (2) agregar diagramas de flujo de principios de operación del sistema usando Mermaid, (3) escribir instrucciones de uso detalladas paso a paso, (4) organizar y optimizar descripciones de configuración (variables de entorno, archivos de configuración), (5) hacer que la documentación sea más fácil de usar y de entender. Aplicable a proyectos de código abierto, herramientas internas, servicios API y cualquier proyecto que requiera documentación mejorada.
