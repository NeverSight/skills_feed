Créez un fichier README et une documentation système de haute qualité pour les projets logiciels. À utiliser lorsque vous avez besoin de : (1) rédiger ou mettre à jour la documentation README du projet, (2) ajouter des organigrammes de principe de fonctionnement du système à l'aide de Mermaid, (3) rédiger des instructions d'utilisation détaillées étape par étape, (4) organiser et optimiser les descriptions de configuration (variables d'environnement, fichiers de configuration), (5) rendre la documentation plus conviviale et plus facile à comprendre. Applicable aux projets open source, aux outils internes, aux services API et à tout projet nécessitant une documentation améliorée.
