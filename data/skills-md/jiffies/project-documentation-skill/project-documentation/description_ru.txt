Создавайте высококачественные README и системную документацию для программных проектов. Используйте, когда вам нужно: (1) написать или обновить документацию README проекта, (2) добавить блок-схемы принципов работы системы с помощью Mermaid, (3) написать подробные пошаговые инструкции по использованию, (4) организовать и оптимизировать описания конфигурации (переменные среды, файлы конфигурации), (5) сделать документацию более удобной и понятной. Применимо к проектам с открытым исходным кодом, внутренним инструментам, службам API и любому проекту, требующему улучшенной документации.
