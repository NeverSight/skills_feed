Knative бессерверная платформа для Kubernetes. Используйте при развертывании бессерверных рабочих нагрузок, настройке автоматического масштабирования (масштабирование до нуля), архитектур, управляемых событиями, управления трафиком (сине-зеленый, канареечный), маршрутизации CloudEvents, брокеров/триггеров/источников или работы с Knative Serving/Eventing/Functions. Охватывает установку, работу в сети (Kourier/Istio/Contour) и устранение неполадок.
