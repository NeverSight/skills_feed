منصة Knative بدون خادم لـ Kubernetes. يُستخدم عند نشر أحمال العمل بدون خادم، أو تكوين التوسع التلقائي (التوسع إلى الصفر)، أو البنيات المستندة إلى الأحداث، أو إدارة حركة المرور (الأزرق والأخضر، الكناري)، أو توجيه CloudEvents، أو الوسطاء/المشغلات/المصادر، أو العمل مع Knative Serving/Eventing/Functions. يغطي التثبيت والشبكات (Kourier/Istio/Contour) واستكشاف الأخطاء وإصلاحها.
