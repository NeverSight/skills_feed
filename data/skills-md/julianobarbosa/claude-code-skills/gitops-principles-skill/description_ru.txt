Комплексная методология и принципы GitOps для облачных операций. Используйте, когда (1) проектируете архитектуру GitOps для развертываний Kubernetes, (2) реализуете декларативную инфраструктуру с Git как единый источник достоверной информации, (3) устанавливаете конвейеры непрерывного развертывания с помощью ArgoCD/Flux/Kargo, (4) устанавливаете стратегии ветвления и структуры репозитория, (5) устраняете неполадки, сбои синхронизации или проблемы согласования, (6) оцениваете решения по инструментам GitOps, (7) обучение или объяснение концепций и лучших практик GitOps, (8) Развертывание ArgoCD в Kubernetes или AKS с поддержкой Azure Arc с идентификацией рабочей нагрузки. Охватывает 4 столпа GitOps (OpenGitOps), шаблоны, антишаблоны, экосистему инструментов, интеграцию Azure Arc и руководство по эксплуатации.
