منهجية GitOps الشاملة ومهارة المبادئ للعمليات السحابية الأصلية. يُستخدم عند (1) تصميم بنية GitOps لعمليات نشر Kubernetes، (2) تنفيذ البنية التحتية التصريحية باستخدام Git كمصدر وحيد للحقيقة، (3) إعداد خطوط أنابيب النشر المستمر باستخدام ArgoCD/Flux/Kargo، (4) إنشاء استراتيجيات التفرع وهياكل المستودعات، (5) استكشاف أخطاء الانجراف أو فشل المزامنة أو مشكلات التسوية، (6) تقييم قرارات أدوات GitOps، (7) التدريس أو الشرح مفاهيم GitOps وأفضل الممارسات، (8) نشر ArgoCD على Kubernetes أو AKS الممكّن لـ Azure Arc مع هوية عبء العمل. يغطي الركائز الأربع لـ GitOps (OpenGitOps)، والأنماط، والأنماط المضادة، والنظام البيئي للأدوات، وتكامل Azure Arc، والتوجيه التشغيلي.
