دليل تنفيذ HolmesGPT - وكيل الذكاء الاصطناعي لاستكشاف أخطاء البيئات السحابية الأصلية وإصلاحها. يُستخدم عند التحقيق في مشكلات Kubernetes، أو تحليل التنبيهات من Prometheus/AlertManager/PagerDuty، أو إجراء تحليل السبب الجذري، أو تكوين عمليات تثبيت HolmesGPT (CLI/Helm/Docker)، أو إعداد موفري الذكاء الاصطناعي (OpenAI/Anthropic/Azure)، أو إنشاء مجموعات أدوات مخصصة، أو التكامل مع الأنظمة الأساسية لقابلية المراقبة (Grafana، وLoki، وTempo، وDataDog).
