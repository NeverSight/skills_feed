Руководство по внедрению HolmesGPT — агента искусственного интеллекта для устранения неполадок в облачных средах. Используйте при исследовании проблем Kubernetes, анализе оповещений от Prometheus/AlertManager/PagerDuty, выполнении анализа первопричин, настройке установок HolmesGPT (CLI/Helm/Docker), настройке поставщиков ИИ (OpenAI/Anthropic/Azure), создании пользовательских наборов инструментов или интеграции с платформами наблюдения (Grafana, Loki, Tempo, DataDog).
