Guide de mise en œuvre de HolmesGPT - un agent d'IA pour le dépannage des environnements cloud natifs. À utiliser pour enquêter sur les problèmes Kubernetes, analyser les alertes de Prometheus/AlertManager/PagerDuty, effectuer une analyse des causes profondes, configurer les installations HolmesGPT (CLI/Helm/Docker), configurer les fournisseurs d'IA (OpenAI/Anthropic/Azure), créer des ensembles d'outils personnalisés ou intégrer des plateformes d'observabilité (Grafana, Loki, Tempo, DataDog).
