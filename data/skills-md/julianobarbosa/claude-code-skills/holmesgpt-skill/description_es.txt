Guía para implementar HolmesGPT, un agente de IA para solucionar problemas de entornos nativos de la nube. Úselo al investigar problemas de Kubernetes, analizar alertas de Prometheus/AlertManager/PagerDuty, realizar análisis de causa raíz, configurar instalaciones de HolmesGPT (CLI/Helm/Docker), configurar proveedores de IA (OpenAI/Anthropic/Azure), crear conjuntos de herramientas personalizados o integrarse con plataformas de observabilidad (Grafana, Loki, Tempo, DataDog).
