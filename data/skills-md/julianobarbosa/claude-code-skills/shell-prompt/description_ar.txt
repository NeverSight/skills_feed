التكوين الفوري للقشرة الحديثة مع Powerlevel10k وZsh Vi Mode. يُستخدم عند تكوين مطالبات Shell، أو إعداد روابط المفاتيح vi/vim في zsh، أو تخصيص أنماط المؤشر لكل وضع، أو إضافة مؤشرات الوضع، أو تحسين الأداء الفوري، أو استكشاف أخطاء المطالبات البطيئة وإصلاحها. يغطي المطالبة الفورية P10k، والمكونات الإضافية لوضع vi، وتخصيص المؤشر.
