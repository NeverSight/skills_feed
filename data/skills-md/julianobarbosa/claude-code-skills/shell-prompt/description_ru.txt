Современная конфигурация командной оболочки с Powerlevel10k и режимом Zsh Vi. Используйте при настройке подсказок оболочки, настройке сочетаний клавиш vi/vim в zsh, настройке стилей курсора для каждого режима, добавлении индикаторов режима, оптимизации производительности подсказок или устранении неполадок с медленными подсказками. Охватывает мгновенную подсказку P10k, плагины режима vi и настройку курсора.
