Руководство по внедрению Grafana Loki — горизонтально масштабируемой высокодоступной системы агрегирования журналов. Используйте при настройке развертываний Loki, настройке серверных частей хранилища (S3, Azure Blob, GCS), написании запросов LogQL, настройке хранения и сжатия, развертывании через Helm, интеграции с OpenTelemetry или устранении неполадок Loki в Kubernetes.
