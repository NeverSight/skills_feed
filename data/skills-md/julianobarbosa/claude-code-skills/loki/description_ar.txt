دليل لتنفيذ Grafana Loki - نظام تجميع سجلات قابل للتطوير أفقيًا ومتاح للغاية. يُستخدم عند تكوين عمليات نشر Loki، أو إعداد واجهات التخزين الخلفية (S3، أو Azure Blob، أو GCS)، أو كتابة استعلامات LogQL، أو تكوين الاحتفاظ والضغط، أو النشر عبر Helm، أو التكامل مع OpenTelemetry، أو استكشاف مشكلات Loki وإصلاحها على Kubernetes.
