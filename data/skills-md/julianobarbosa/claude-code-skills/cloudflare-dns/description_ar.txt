دليل شامل لإدارة Cloudflare DNS مع تكامل Azure. يُستخدم عند تكوين Cloudflare كموفر DNS معتمد للتطبيقات المستضافة على Azure، أو إدارة سجلات DNS عبر واجهة برمجة التطبيقات، أو إعداد الرموز المميزة لواجهة برمجة التطبيقات، أو تكوين إعدادات الوكيل، أو استكشاف مشكلات DNS وإصلاحها، أو تنفيذ أفضل ممارسات أمان DNS، أو دمج DNS الخارجي مع أحمال عمل Cloudflare for Kubernetes.
