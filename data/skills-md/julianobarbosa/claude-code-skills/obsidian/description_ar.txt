دليل لتنفيذ إدارة المعرفة Obsidian - عمليات القبو، وتطوير المكونات الإضافية، وأتمتة مخطط URI، وامتدادات تخفيض السعر، وتكامل REST API المحلي. يُستخدم عند العمل مع خزائن Obsidian، أو إنشاء المكونات الإضافية، أو أتمتة سير عمل الملاحظات، أو الاستعلام عن الملاحظات عبر واجهة برمجة التطبيقات، أو تنفيذ ميزات الرسم البياني المعرفي.
