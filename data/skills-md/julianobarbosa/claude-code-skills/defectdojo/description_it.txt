Guida per l'implementazione di DefectDojo, una piattaforma DevSecOps, ASPM e di gestione delle vulnerabilità open source. Da utilizzare quando si eseguono query su vulnerabilità, si gestiscono risultati, si configurano importazioni di pipeline CI/CD o si lavora con dati di scansione di sicurezza. Include strumenti MCP per l'interazione diretta dell'API.
