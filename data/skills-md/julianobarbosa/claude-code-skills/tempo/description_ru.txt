Руководство по реализации Grafana Tempo — высокомасштабного распределенного бэкэнда трассировки OpenTelemetry. Используйте при настройке развертываний Tempo, настройке серверных частей хранилища (S3, Azure Blob, GCS), написании запросов TraceQL, развертывании через Helm, понимании структуры трассировки или устранении неполадок Tempo в Kubernetes.
