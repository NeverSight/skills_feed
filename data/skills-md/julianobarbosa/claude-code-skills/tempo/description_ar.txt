دليل تنفيذ Grafana Tempo - واجهة خلفية للتتبع الموزعة واسعة النطاق لتتبعات OpenTelemetry. يُستخدم عند تكوين عمليات نشر Tempo، أو إعداد واجهات التخزين الخلفية (S3، أو Azure Blob، أو GCS)، أو كتابة استعلامات TraceQL، أو النشر عبر Helm، أو فهم بنية التتبع، أو استكشاف مشكلات Tempo وإصلاحها على Kubernetes.
