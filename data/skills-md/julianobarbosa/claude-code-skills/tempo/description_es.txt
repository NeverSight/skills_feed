Guía para implementar Grafana Tempo: un backend de seguimiento distribuido de alta escala para seguimientos de OpenTelemetry. Úselo al configurar implementaciones de Tempo, configurar backends de almacenamiento (S3, Azure Blob, GCS), escribir consultas TraceQL, implementar a través de Helm, comprender la estructura de seguimiento o solucionar problemas de Tempo en Kubernetes.
