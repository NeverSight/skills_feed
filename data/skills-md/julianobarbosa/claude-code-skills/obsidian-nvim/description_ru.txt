Руководство по реализации obsidian.nvim — плагина Neovim для управления хранилищем Obsidian. Используйте при настройке, устранении неполадок или расширении функций obsidian.nvim, включая настройку рабочего пространства, ежедневные заметки, шаблоны, завершение, средства выбора и настройку пользовательского интерфейса.
