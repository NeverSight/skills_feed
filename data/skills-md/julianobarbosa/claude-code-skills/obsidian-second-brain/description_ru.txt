Подробное руководство по интеграции хранилищ Obsidian в качестве второго мозга на базе искусственного интеллекта с Claude Code. Охватывает интеграцию MCP, манифесты хранилища, саморазвивающиеся шаблоны, автоматическое связывание и автоматизацию графов знаний. Используйте при настройке рабочих процессов Claude Code + Obsidian, реализации двунаправленной синхронизации, создании манифестов CLAUDE.md или построении самовосстанавливающихся систем знаний.
