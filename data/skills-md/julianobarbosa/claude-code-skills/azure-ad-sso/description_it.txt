Integrazione SSO OAuth2/OIDC di Azure AD per le applicazioni Kubernetes. Da utilizzare durante l'implementazione del Single Sign-On, la configurazione delle registrazioni delle app Azure AD, la limitazione dell'accesso per gruppi o l'integrazione di strumenti (DefectDojo, Grafana, ArgoCD, Harbour, SonarQube) con l'autenticazione Azure AD.
