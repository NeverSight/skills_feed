Intégration Azure AD OAuth2/OIDC SSO pour les applications Kubernetes. À utiliser lors de la mise en œuvre de l'authentification unique, de la configuration des inscriptions d'applications Azure AD, de la restriction de l'accès par groupes ou de l'intégration d'outils (DefectDojo, Grafana, ArgoCD, Harbor, SonarQube) avec l'authentification Azure AD.
