Azure AD OAuth2/OIDC SSO-Integration für Kubernetes-Anwendungen. Verwenden Sie diese Option, wenn Sie Single Sign-On implementieren, Azure AD-App-Registrierungen konfigurieren, den Zugriff durch Gruppen einschränken oder Tools (DefectDojo, Grafana, ArgoCD, Harbor, SonarQube) mit der Azure AD-Authentifizierung integrieren.
