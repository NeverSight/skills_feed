دليل لاستخدام Microsoft MarkItDown - أداة مساعدة Python لتحويل الملفات إلى Markdown. يُستخدم عند تحويل ملفات PDF أو Word أو PowerPoint أو Excel أو الصور أو الصوت أو HTML أو CSV أو JSON أو XML أو ZIP أو عناوين URL على YouTube أو EPubs أو دفاتر Jupyter أو خلاصات RSS أو صفحات Wikipedia إلى تنسيق Markdown. يُستخدم أيضًا في خطوط أنابيب معالجة المستندات أو المعالجة المسبقة لـ LLM أو مهام استخراج النص.
