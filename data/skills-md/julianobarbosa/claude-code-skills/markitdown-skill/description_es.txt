Guía para usar Microsoft MarkItDown: una utilidad de Python para convertir archivos a Markdown. Úselo al convertir PDF, Word, PowerPoint, Excel, imágenes, audio, HTML, CSV, JSON, XML, ZIP, URL de YouTube, EPubs, cuadernos de Jupyter, fuentes RSS o páginas de Wikipedia al formato Markdown. También se utiliza para procesos de procesamiento de documentos, preprocesamiento de LLM o tareas de extracción de texto.
