Guide d'utilisation de Microsoft MarkItDown - un utilitaire Python pour convertir des fichiers en Markdown. À utiliser lors de la conversion de PDF, Word, PowerPoint, Excel, images, audio, HTML, CSV, JSON, XML, ZIP, URL YouTube, EPubs, blocs-notes Jupyter, flux RSS ou pages Wikipédia au format Markdown. Utilisez également pour les pipelines de traitement de documents, le prétraitement LLM ou les tâches d'extraction de texte.
