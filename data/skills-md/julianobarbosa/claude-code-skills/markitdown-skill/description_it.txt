Guida per l'utilizzo di Microsoft MarkItDown: un'utilità Python per convertire file in Markdown. Da utilizzare per convertire PDF, Word, PowerPoint, Excel, immagini, audio, HTML, CSV, JSON, XML, ZIP, URL di YouTube, EPub, quaderni Jupyter, feed RSS o pagine Wikipedia nel formato Markdown. Utilizzabile anche per pipeline di elaborazione di documenti, preelaborazione LLM o attività di estrazione di testo.
