Руководство по использованию Microsoft MarkItDown — утилиты Python для преобразования файлов в Markdown. Используйте при преобразовании PDF, Word, PowerPoint, Excel, изображений, аудио, HTML, CSV, JSON, XML, ZIP, URL-адресов YouTube, EPub, блокнотов Jupyter, RSS-каналов или страниц Википедии в формат Markdown. Также используйте для конвейеров обработки документов, предварительной обработки LLM или задач извлечения текста.
