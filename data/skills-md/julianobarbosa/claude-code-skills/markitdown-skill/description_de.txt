Anleitung zur Verwendung von Microsoft MarkItDown – einem Python-Dienstprogramm zum Konvertieren von Dateien in Markdown. Zum Konvertieren von PDF, Word, PowerPoint, Excel, Bildern, Audio, HTML, CSV, JSON, XML, ZIP, YouTube-URLs, EPubs, Jupyter-Notizbüchern, RSS-Feeds oder Wikipedia-Seiten in das Markdown-Format. Kann auch für Dokumentverarbeitungspipelines, LLM-Vorverarbeitung oder Textextraktionsaufgaben verwendet werden.
