CLI Agentic de Azure AKS: herramienta de información y solución de problemas basada en IA para el servicio Azure Kubernetes. Úselo para diagnosticar problemas del clúster de AKS, obtener información sobre el estado del clúster, solucionar problemas de redes/almacenamiento/seguridad o analizar la configuración del clúster con consultas en lenguaje natural.
