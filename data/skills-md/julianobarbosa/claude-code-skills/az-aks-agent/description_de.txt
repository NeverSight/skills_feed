Azure AKS Agentic CLI – KI-gestütztes Fehlerbehebungs- und Erkenntnistool für Azure Kubernetes Service. Verwenden Sie diese Option, um AKS-Clusterprobleme zu diagnostizieren, Einblicke in den Clusterzustand zu erhalten, Netzwerk-/Speicher-/Sicherheitsprobleme zu beheben oder die Clusterkonfiguration mit Abfragen in natürlicher Sprache zu analysieren.
