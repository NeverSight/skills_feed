Azure AKS Agentic CLI - Outil de dépannage et d'informations basé sur l'IA pour Azure Kubernetes Service. À utiliser pour diagnostiquer les problèmes du cluster AKS, obtenir des informations sur l'état du cluster, résoudre les problèmes de réseau/stockage/sécurité ou analyser la configuration du cluster avec des requêtes en langage naturel.
