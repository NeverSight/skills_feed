Azure AKS Agentic CLI — инструмент устранения неполадок и анализа на базе искусственного интеллекта для службы Azure Kubernetes. Используйте при диагностике проблем кластера AKS, получении информации о работоспособности кластера, устранении неполадок сети, хранилища или безопасности или анализе конфигурации кластера с помощью запросов на естественном языке.
