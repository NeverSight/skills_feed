أتمتة تحديثات صور الحاوية لأحمال عمل Kubernetes التي تتم إدارتها بواسطة Argo CD. يُستخدم عند تكوين ArgoCD Image Updater، أو إعداد تحديثات الصور التلقائية، أو تكوين إستراتيجيات التحديث (semver، أو Digest، أو أحدث إصدار، أو أبجدي)، أو تنفيذ إعادة كتابة git، أو استكشاف مشكلات تحديث الصورة وإصلاحها، أو العمل مع ImageUpdater CRDs. يغطي التثبيت والتكوين والمصادقة وأفضل الممارسات.
