Automatizza gli aggiornamenti delle immagini del contenitore per i carichi di lavoro Kubernetes gestiti da Argo CD. DA USARE QUANDO si configura ArgoCD Image Updater, si impostano gli aggiornamenti automatici delle immagini, si configurano le strategie di aggiornamento (semver, digest, newest-build, alfabetico), si implementa il writeback git, si risolvono i problemi di aggiornamento delle immagini o si lavora con i CRD di ImageUpdater. Copre installazione, configurazione, autenticazione e procedure consigliate.
