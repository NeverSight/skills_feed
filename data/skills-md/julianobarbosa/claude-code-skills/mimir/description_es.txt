Guía para implementar Grafana Mimir: una TSDB multiinquilino, de alta disponibilidad y escalable horizontalmente para el almacenamiento a largo plazo de métricas de Prometheus. Úselo al configurar Mimir en Kubernetes, configurar backends de almacenamiento de Azure/S3/GCS, solucionar problemas de autenticación u optimizar el rendimiento.
