Git-Arbeitsbaumverwaltung mit tmux- und iTerm2-Integration. Verwenden Sie diese Option, wenn Sie isolierte Entwicklungsumgebungen erstellen, parallele Feature-Branches verwalten, Kontexte ohne Stashing wechseln oder mehrere Claude-Instanzen ausführen. Behandelt die Erstellung von Arbeitsbäumen, die tmux-Fensterverwaltung, iTerm2-Registerkarten und Bereinigungsworkflows.
