Gestión del árbol de trabajo de Git con integración de tmux e iTerm2. Úselo al crear entornos de desarrollo aislados, administrar ramas de funciones paralelas, cambiar contextos sin almacenamiento oculto o ejecutar múltiples instancias de Claude. Cubre la creación de árboles de trabajo, administración de ventanas tmux, pestañas de iTerm2 y flujos de trabajo de limpieza.
