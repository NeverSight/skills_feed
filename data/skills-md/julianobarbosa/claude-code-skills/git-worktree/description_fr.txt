Gestion de l'arbre de travail Git avec intégration tmux et iTerm2. À utiliser lors de la création d'environnements de développement isolés, de la gestion de branches de fonctionnalités parallèles, du changement de contexte sans stockage ou de l'exécution de plusieurs instances Claude. Couvre la création d'arbres de travail, la gestion des fenêtres tmux, les onglets iTerm2 et les flux de travail de nettoyage.
