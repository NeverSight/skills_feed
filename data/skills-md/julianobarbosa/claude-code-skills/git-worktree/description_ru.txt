Управление рабочим деревом Git с интеграцией tmux и iTerm2. Используйте при создании изолированных сред разработки, управлении параллельными ветвями функций, переключении контекстов без сохранения или запуске нескольких экземпляров Claude. Охватывает создание рабочего дерева, управление окнами tmux, вкладки iTerm2 и рабочие процессы очистки.
