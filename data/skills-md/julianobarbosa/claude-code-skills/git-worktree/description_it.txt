Gestione Git Worktree con integrazione tmux e iTerm2. Da utilizzare durante la creazione di ambienti di sviluppo isolati, la gestione di rami di funzionalità parallele, il passaggio da un contesto all'altro senza memorizzazione o l'esecuzione di più istanze di Claude. Copre la creazione di alberi di lavoro, la gestione delle finestre tmux, le schede iTerm2 e i flussi di lavoro di pulizia.
