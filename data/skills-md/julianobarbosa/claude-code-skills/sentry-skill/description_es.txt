Habilidad integral para el monitoreo de errores y seguimiento del desempeño de Sentry. Úselo cuando Claude necesite (1) configurar los SDK de Sentry para el seguimiento de errores y el monitoreo del rendimiento, (2) administrar versiones, mapas de origen y símbolos de depuración a través de CLI, (3) consultar problemas, eventos y métricas a través de API, (4) configurar reglas de alertas y notificaciones, (5) configurar estrategias de muestreo y administración de cuotas, (6) implementar instancias Sentry autohospedadas, (7) integrar con OpenTelemetry para seguimiento distribuido o cualquier otra tarea de automatización Sentry.
