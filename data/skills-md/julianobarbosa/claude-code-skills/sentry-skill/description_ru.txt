Комплексные навыки мониторинга ошибок Sentry и отслеживания производительности. Используйте, когда Клоду необходимо (1) настроить Sentry SDK для отслеживания ошибок и мониторинга производительности, (2) управлять выпусками, картами исходного кода и символами отладки через CLI, (3) запрашивать проблемы, события и метрики через API, (4) настраивать правила оповещений и уведомлений, (5) настраивать стратегии выборки и управление квотами, (6) развертывать автономные экземпляры Sentry, (7) интегрироваться с OpenTelemetry для распределенной трассировки или любая другая задача автоматизации Sentry.
