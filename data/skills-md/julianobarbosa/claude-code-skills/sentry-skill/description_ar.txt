مهارة شاملة لمراقبة أخطاء الحراسة وتتبع الأداء. يُستخدم عندما يحتاج Claude إلى (1) تكوين Sentry SDK لتتبع الأخطاء ومراقبة الأداء، (2) إدارة الإصدارات وخرائط المصدر ورموز تصحيح الأخطاء عبر واجهة سطر الأوامر (CLI)، (3) مشكلات الاستعلام والأحداث والمقاييس عبر واجهة برمجة التطبيقات (API)، (4) إعداد قواعد التنبيه والإخطار، (5) تكوين إستراتيجيات أخذ العينات وإدارة الحصص، (6) نشر مثيلات Sentry المستضافة ذاتيًا، (7) التكامل مع OpenTelemetry الموزعة التتبع، أو أي مهمة أخرى لأتمتة Sentry.
