Grafana の HTTP API を操作して、ダッシュボード、データ ソース、フォルダー、アラート、注釈、ユーザー、チーム、組織を管理するための包括的なスキル。クロードが (1) Grafana ダッシュボードの作成、読み取り、更新、または削除、(2) データ ソースと接続の管理、(3) アラート ルール、連絡先ポイント、および通知ポリシーの構成、(4) フォルダーと権限の操作、(5) ユーザー、チーム、およびサービス アカウントの管理、(6) 注釈の作成またはクエリ、(7) データ ソースに対するクエリの実行、または API を介したその他の Grafana 自動化タスクを実行する必要がある場合に使用します。
