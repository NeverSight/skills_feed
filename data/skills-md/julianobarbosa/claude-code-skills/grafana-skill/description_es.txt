Habilidad integral para interactuar con la API HTTP de Grafana para administrar paneles, fuentes de datos, carpetas, alertas, anotaciones, usuarios, equipos y organizaciones. Úselo cuando Claude necesite (1) crear, leer, actualizar o eliminar paneles de Grafana, (2) administrar conexiones y fuentes de datos, (3) configurar reglas de alerta, puntos de contacto y políticas de notificación, (4) trabajar con carpetas y permisos, (5) administrar usuarios, equipos y cuentas de servicio, (6) crear o consultar anotaciones, (7) ejecutar consultas en fuentes de datos o cualquier otra tarea de automatización de Grafana a través de API.
