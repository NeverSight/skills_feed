Всесторонние навыки взаимодействия с HTTP API Grafana для управления панелями мониторинга, источниками данных, папками, оповещениями, аннотациями, пользователями, командами и организациями. Используйте, когда Клоду необходимо (1) создавать, читать, обновлять или удалять информационные панели Grafana, (2) управлять источниками данных и соединениями, (3) настраивать правила оповещений, точки контакта и политики уведомлений, (4) работать с папками и разрешениями, (5) управлять пользователями, командами и учетными записями служб, (6) создавать или запрашивать аннотации, (7) выполнять запросы к источникам данных или любые другие задачи автоматизации Grafana через API.
