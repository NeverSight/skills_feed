مهارة شاملة للتفاعل مع HTTP API الخاص بـ Grafana لإدارة لوحات المعلومات ومصادر البيانات والمجلدات والتنبيهات والتعليقات التوضيحية والمستخدمين والفرق والمؤسسات. يُستخدم عندما يحتاج Claude إلى (1) إنشاء لوحات معلومات Grafana أو قراءتها أو تحديثها أو حذفها، (2) إدارة مصادر البيانات والاتصالات، (3) تكوين قواعد التنبيه ونقاط الاتصال وسياسات الإعلام، (4) العمل مع المجلدات والأذونات، (5) إدارة المستخدمين والفرق وحسابات الخدمة، (6) إنشاء التعليقات التوضيحية أو الاستعلام عنها، (7) تنفيذ الاستعلامات ضد مصادر البيانات، أو أي مهمة أخرى لأتمتة Grafana عبر واجهة برمجة التطبيقات.
