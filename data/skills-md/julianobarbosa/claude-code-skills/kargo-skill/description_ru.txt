Используйте при реализации конвейеров постепенной доставки, рабочих процессов продвижения, управления грузоперевозками, интеграции ArgoCD, конфигурации склада, поэтапных конвейеров, шаблонов проверки или любых задач, связанных с Kargo. Охватывает установку, основные понятия, шаблоны, безопасность и полные примеры YAML.
