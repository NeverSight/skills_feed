مهارة منصة الترويج المستمر الشاملة لـ Kargo GitOps. يُستخدم عند تنفيذ مسارات التسليم التقدمية، أو سير عمل الترويج، أو إدارة الشحن، أو تكامل ArgoCD، أو تكوين المستودع، أو خطوط الأنابيب المرحلية، أو قوالب التحقق، أو أي مهام متعلقة بـ Kargo. يغطي التثبيت والمفاهيم الأساسية والأنماط والأمان وأمثلة YAML الكاملة.
