دليل لاستخدام direnv - امتداد Shell لتحميل متغيرات البيئة الخاصة بالدليل. يُستخدم عند إعداد بيئات المشروع، أو إنشاء ملفات .envrc، أو تكوين متغيرات البيئة لكل مشروع، أو التكامل مع تخطيطات Python/Node/Ruby/Go، أو العمل مع رقائق Nix، أو استكشاف مشكلات تحميل البيئة وإصلاحها على نظامي التشغيل macOS وLinux.
