Guida per l'utilizzo di direnv: un'estensione della shell per caricare variabili di ambiente specifiche della directory. Da utilizzare durante la configurazione di ambienti di progetto, la creazione di file .envrc, la configurazione di variabili di ambiente per progetto, l'integrazione con layout Python/Node/Ruby/Go, l'utilizzo di fiocchi Nix o la risoluzione dei problemi di caricamento dell'ambiente su macOS e Linux.
