Zabbix أتمتة نظام المراقبة عبر API وPython. يُستخدم عندما: (1) إدارة المضيفين أو القوالب أو العناصر أو المشغلات أو المجموعات المضيفة، (2) أتمتة تكوين المراقبة، (3) إرسال البيانات عبر مرسل/مصيد Zabbix، (4) الاستعلام عن البيانات أو الأحداث التاريخية، (5) العمليات المجمعة على كائنات Zabbix، (6) إدارة نوافذ الصيانة، (7) إدارة المستخدم/الأذونات
