Automazione del sistema di monitoraggio Zabbix tramite API e Python. Utilizzare quando: (1) gestire host, modelli, elementi, trigger o gruppi host, (2) automatizzare la configurazione del monitoraggio, (3) inviare dati tramite trapper/mittente Zabbix, (4) interrogare dati storici o eventi, (5) operazioni di massa su oggetti Zabbix, (6) gestione delle finestre di manutenzione, (7) gestione di utenti/autorizzazioni
