Automatisation du système de surveillance Zabbix via API et Python. À utiliser lorsque : (1) Gestion d'hôtes, de modèles, d'éléments, de déclencheurs ou de groupes d'hôtes, (2) Automatisation de la configuration de la surveillance, (3) Envoi de données via le trappeur/expéditeur Zabbix, (4) Interrogation de données ou d'événements historiques, (5) Opérations groupées sur des objets Zabbix, (6) Gestion des fenêtres de maintenance, (7) Gestion des utilisateurs/autorisations
