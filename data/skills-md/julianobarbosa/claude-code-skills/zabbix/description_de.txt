Automatisierung des Zabbix-Überwachungssystems über API und Python. Verwendung bei: (1) Verwalten von Hosts, Vorlagen, Elementen, Triggern oder Hostgruppen, (2) Automatisieren der Überwachungskonfiguration, (3) Senden von Daten über Zabbix Trapper/Sender, (4) Abfragen historischer Daten oder Ereignisse, (5) Massenvorgänge für Zabbix-Objekte, (6) Wartungsfensterverwaltung, (7) Benutzer-/Berechtigungsverwaltung
