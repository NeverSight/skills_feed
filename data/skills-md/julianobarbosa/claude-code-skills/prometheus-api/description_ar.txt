الاستعلام والتفاعل مع Prometheus HTTP API لمراقبة البيانات. يُستخدم عندما يحتاج Claude إلى الاستعلام عن مقاييس Prometheus، أو تنفيذ استعلامات PromQL، أو استرداد حالة الأهداف/التنبيهات/القواعد، أو الوصول إلى بيانات التعريف حول السلسلة/التسميات، أو إدارة عمليات TSDB، أو استكشاف أخطاء البنية التحتية للمراقبة وإصلاحها. يدعم الاستعلامات الفورية واستعلامات النطاق ونقاط نهاية البيانات التعريفية وواجهات برمجة التطبيقات الإدارية ومعلومات التنبيه.
