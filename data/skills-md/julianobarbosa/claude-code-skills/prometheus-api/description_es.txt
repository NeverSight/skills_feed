Consulta e interactúa con la API HTTP de Prometheus para monitorear datos. Úselo cuando Claude necesite consultar métricas de Prometheus, ejecutar consultas PromQL, recuperar objetivos/alertas/estado de reglas, acceder a metadatos sobre series/etiquetas, administrar operaciones TSDB o solucionar problemas de infraestructura de monitoreo. Admite consultas instantáneas, consultas de rango, puntos finales de metadatos, API de administración e información de alertas.
