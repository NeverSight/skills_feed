Interrogez et interagissez avec l'API HTTP Prometheus pour surveiller les données. À utiliser lorsque Claude doit interroger les métriques Prometheus, exécuter des requêtes PromQL, récupérer l'état des cibles/alertes/règles, accéder aux métadonnées sur les séries/étiquettes, gérer les opérations TSDB ou dépanner l'infrastructure de surveillance. Prend en charge les requêtes instantanées, les requêtes par plage, les points de terminaison de métadonnées, les API d'administration et les informations d'alerte.
