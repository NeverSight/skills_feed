Fragen Sie die Prometheus-HTTP-API ab und interagieren Sie mit ihr, um Daten zu überwachen. Verwenden Sie diese Option, wenn Claude Prometheus-Metriken abfragen, PromQL-Abfragen ausführen, den Status von Zielen/Warnungen/Regeln abrufen, auf Metadaten zu Serien/Bezeichnungen zugreifen, TSDB-Vorgänge verwalten oder Probleme bei der Überwachungsinfrastruktur beheben muss. Unterstützt Sofortabfragen, Bereichsabfragen, Metadatenendpunkte, Admin-APIs und Warninformationen.
