Запрашивайте и взаимодействуйте с HTTP API Prometheus для мониторинга данных. Используйте, когда Клоду необходимо запросить метрики Prometheus, выполнить запросы PromQL, получить статус целей/предупреждений/правил, получить доступ к метаданным о сериях/метках, управлять операциями TSDB или устранять неполадки в инфраструктуре мониторинга. Поддерживает мгновенные запросы, запросы диапазона, конечные точки метаданных, API-интерфейсы администратора и информацию для оповещений.
