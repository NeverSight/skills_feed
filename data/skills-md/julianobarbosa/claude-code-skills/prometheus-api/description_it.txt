Interroga e interagisci con l'API HTTP Prometheus per il monitoraggio dei dati. Da utilizzare quando Claude deve eseguire query sulle metriche Prometheus, eseguire query PromQL, recuperare lo stato di destinazioni/avvisi/regole, accedere ai metadati su serie/etichette, gestire operazioni TSDB o risolvere problemi dell'infrastruttura di monitoraggio. Supporta query istantanee, query di intervallo, endpoint di metadati, API di amministrazione e informazioni di avviso.
