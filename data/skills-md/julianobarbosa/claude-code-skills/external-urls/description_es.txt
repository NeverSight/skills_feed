Referencia de puntos finales y URL de infraestructura de Hypera. Úselo cuando el usuario pregunte sobre URL, dominios, puntos finales de ingreso, servidores API de clúster, URL de aplicaciones, repositorios de Helm, repositorios de Git o necesite verificar la conectividad a los servicios. Proporciona una búsqueda rápida de todas las URL externas en todos los entornos (hub, dev, prd) y clústeres (cafehyna, lealtad, Painelclientes, sonora).
