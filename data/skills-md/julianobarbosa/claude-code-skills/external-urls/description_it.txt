Riferimento agli URL e agli endpoint dell'infrastruttura Hypera. Da utilizzare quando l'utente richiede informazioni su URL, domini, endpoint di ingresso, server API cluster, URL dell'applicazione, repository Helm, repository Git o deve verificare la connettività ai servizi. Fornisce la ricerca rapida di tutti gli URL esterni in tutti gli ambienti (hub, dev, prd) e cluster (cafehyna, lealtà, paintelclientes, sonora).
