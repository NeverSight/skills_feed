Référence des URL et des points de terminaison de l’infrastructure Hypera. À utiliser lorsque l'utilisateur pose des questions sur les URL, les domaines, les points de terminaison d'entrée, les serveurs d'API de cluster, les URL d'application, les référentiels Helm, les référentiels Git ou doit vérifier la connectivité aux services. Fournit une recherche rapide de toutes les URL externes dans tous les environnements (hub, dev, prd) et clusters (cafehyna, Loyalty, Painelclientes, Sonora).
