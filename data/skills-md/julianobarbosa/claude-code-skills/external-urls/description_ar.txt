عناوين URL للبنية التحتية لـ Hypera ومرجع نقاط النهاية. يُستخدم عندما يسأل المستخدم عن عناوين URL، أو المجالات، أو نقاط نهاية الدخول، أو خوادم واجهة برمجة التطبيقات للمجموعة، أو عناوين URL للتطبيق، أو مستودعات Helm، أو مستودعات Git، أو يحتاج إلى التحقق من الاتصال بالخدمات. يوفر بحثًا سريعًا عن جميع عناوين URL الخارجية عبر جميع البيئات (hub، وdev، وprd) والمجموعات (cafehyna، وloyal، وpainelclientes، وsonora).
