Guía para usar uv: un paquete Python y administrador de proyectos extremadamente rápido escrito en Rust. Úselo al instalar Python, administrar entornos virtuales, agregar dependencias, ejecutar scripts, crear paquetes o trabajar con pyproject.toml. Reemplaza a pip, pip-tools, pipx, poesía, pyenv, twine y virtualenv.
