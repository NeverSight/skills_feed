Guida per l'utilizzo di uv: un pacchetto Python estremamente veloce e un project manager scritto in Rust. Da utilizzare durante l'installazione di Python, la gestione di ambienti virtuali, l'aggiunta di dipendenze, l'esecuzione di script, la creazione di pacchetti o l'utilizzo di pyproject.toml. Sostituisce pip, pip-tools, pipx, poetry, pyenv, twine e virtualenv.
