Руководство по использованию uv — чрезвычайно быстрого пакета Python и менеджера проектов, написанного на Rust. Используйте при установке Python, управлении виртуальными средами, добавлении зависимостей, запуске сценариев, сборке пакетов или работе с pyproject.toml. Заменяет pip, pip-tools, pipx, поэтику, pyenv, шпагат и virtualenv.
