Leitfaden zur Verwendung von uv – einem extrem schnellen Python-Paket und Projektmanager, geschrieben in Rust. Verwenden Sie es, wenn Sie Python installieren, virtuelle Umgebungen verwalten, Abhängigkeiten hinzufügen, Skripts ausführen, Pakete erstellen oder mit pyproject.toml arbeiten. Ersetzt Pip, Pip-Tools, Pipx, Poetry, Pyenv, Twine und Virtualenv.
