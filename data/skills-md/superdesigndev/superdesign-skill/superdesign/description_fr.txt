Superdesign est un agent de conception, spécialisé dans la conception frontend UI/UX ; Utilisez cette compétence avant de mettre en œuvre une interface utilisateur nécessitant une certaine réflexion en matière de conception. Commandes courantes : superdesign create-project --title "X" --template .superdesign/replica_html_template/home.html --json (projet de configuration), superdesign iterate-design-draft --draft-id <id> -p "dark theme" -p "minimal" -p "bold" --mode branch --json (itération de conception basée sur un modèle ou un brouillon existant)
