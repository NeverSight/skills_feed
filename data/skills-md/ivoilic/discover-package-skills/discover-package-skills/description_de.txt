Entdecken Sie proaktiv KI-Regeln, Eingabeaufforderungen, Agenten, Fertigkeiten und Fertigkeitspakete, die von npm-Paketen in einem Projekt bereitgestellt werden, integrieren Sie sie in editorspezifische Regel-/Konfigurationsdateien (AGENTS.md, .cursor/rules, .rules usw.) und installieren Sie gegebenenfalls wiederverwendbare Fertigkeiten über das Fertigkeitspaket (skills.sh). Verwenden Sie diese Fähigkeit, wenn Sie ein Projekt öffnen, nachdem sich Abhängigkeiten geändert haben oder bevor Sie sich auf ein Paket verlassen, damit Sie vorhandene Regeln/Fähigkeiten wiederverwenden können, anstatt sie von Grund auf neu zu erstellen.
