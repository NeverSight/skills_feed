Découvrez de manière proactive les règles, invites, agents, compétences et ensembles de compétences d'IA exposés par les packages npm dans un projet, intégrez-les dans des fichiers de règles/configuration spécifiques à l'éditeur (AGENTS.md, .cursor/rules, .rules, etc.) et, le cas échéant, installez des compétences réutilisables via le package de compétences (skills.sh). Utilisez cette compétence lors de l'ouverture d'un projet, après la modification des dépendances ou avant de vous appuyer sur un package afin de pouvoir réutiliser les règles/compétences existantes au lieu de les recréer à partir de zéro.
