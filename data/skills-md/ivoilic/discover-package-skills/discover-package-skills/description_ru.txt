Заблаговременно обнаруживайте правила ИИ, подсказки, агенты, навыки и наборы навыков, предоставляемые пакетами npm в проекте, интегрируйте их в файлы правил/конфигурации для конкретного редактора (AGENTS.md, .cursor/rules, .rules и т. д.) и, при необходимости, устанавливайте повторно используемые навыки через пакет навыков (skills.sh). Используйте этот навык при открытии проекта, после изменения зависимостей или перед использованием пакета, чтобы вы могли повторно использовать существующие правила/навыки, а не воссоздавать их с нуля.
