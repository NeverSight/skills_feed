Глубокий анализ проектов GitHub. Используйте, когда пользователь упоминает имя репозитория/проекта GitHub и хочет его понять — это вызывается такими фразами, как «帮我看看这个项目», «了解一下 XXX», «这个项目怎么样», «分析一下 репо» или любым запросом на изучить/оценить проект GitHub. Охватывает архитектуру, здравоохранение, конкурентную среду и кроссплатформенные источники знаний.
