Robusta extracción de URL a Markdown para flujos de trabajo de OpenClaw. Úselo cuando el usuario quiera "extraer/resumir/convertir una página web en rebajas" (especialmente WeChat mp.weixin.qq.com) y web_fetch/browser está bloqueado o desordenado. Primero utiliza una sonda barata a través de web_fetch, luego recurre a la API oficial de MinerU (a través de la habilidad local de extracción de mineru) y devuelve un contrato de resultados rastreable con enlaces de origen.
