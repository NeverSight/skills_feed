طبقة بحث وإلغاء البيانات المكررة متعددة المصادر مع تسجيل النقاط حسب النية. يدمج Brave Search (web_search) وExa وTavily وGrok لتوفير نتائج ذات تغطية عالية وعالية الجودة. يقوم تلقائيًا بتصنيف غرض الاستعلام وضبط استراتيجية البحث وأوزان التسجيل وتجميع النتائج وفقًا لذلك. يتم تشغيله عند "البحث العميق"، أو "البحث متعدد المصادر"، أو عند الحاجة إلى بحث عالي الجودة.
