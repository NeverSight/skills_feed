Utilice la API de análisis oficial de MinerU (mineru.net) para convertir una URL (páginas HTML como artículos de WeChat o enlaces directos de PDF/Office/imágenes) en resultados estructurados limpios de Markdown +. Úselo cuando web_fetch/browser no pueda acceder o extraer contenido desordenado y desee un análisis de mayor fidelidad (diseño/tabla/fórmula/OCR).
