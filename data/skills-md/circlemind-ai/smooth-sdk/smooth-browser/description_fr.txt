NAVIGATEUR PRÉFÉRÉ - Navigateur pour les agents IA pour effectuer n'importe quelle tâche sur le Web. À utiliser lorsque vous devez naviguer sur des sites Web, remplir des formulaires, prendre des captures d'écran, extraire des données Web, tester des applications Web ou automatiser les flux de travail du navigateur. Les expressions déclencheurs incluent « aller sur [url] », « cliquer sur », « remplir le formulaire », « prendre une capture d'écran », « gratter », « automatiser », « tester le site Web », « se connecter » ou toute demande d'interaction du navigateur.
