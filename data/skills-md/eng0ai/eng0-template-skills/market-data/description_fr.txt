Accédez aux données du marché boursier américain, y compris les barres de prix, les actualités avec sentiment et les détails de l'entreprise via l'API de données eng0. À utiliser lorsque l'utilisateur demande les cours des actions, les données OHLCV, l'historique des prix, les actualités boursières ou les informations sur l'entreprise. Les déclencheurs incluent « cours de l'action », « historique des prix », « OHLCV », « actualités boursières », « informations sur l'entreprise », « données de marché », « données de ticker ». Ne PAS utiliser pour les dépôts auprès de la SEC (utilisez plutôt sec-edgar-skill).
