Créez et déployez des applications Web sur le cloud. Prend en charge React, Vue, Astro, Next.js (statique), Vanilla HTML/CSS/JS et les fonctions sans serveur. Les sites se déploient sur *.rebyte.pro avec SSL automatique. À utiliser lorsque l'utilisateur demande de « déployer », « publier », « héberger » ou de rendre son application « en direct ». Les déclencheurs incluent « déployer mon application », « la rendre active », « publier le site », « héberger mon application », « déployer le site Web », « la mettre en ligne ».
