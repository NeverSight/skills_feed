Analyse des dossiers SEC EDGAR à l'aide d'EdgarTools. À utiliser lorsque l'utilisateur pose des questions sur les documents déposés auprès de la SEC, les données financières de l'entreprise, l'analyse 10-K/10-Q, les délits d'initiés, les tendances des revenus ou les comparaisons financières. Les déclencheurs incluent « dépôt auprès de la SEC », « 10-K », « 10-Q », « 8-K », « EDGAR », « données financières de l'entreprise », « revenus », « bénéfices », « délits d'initiés », « états financiers ». Ne PAS utiliser pour les cours boursiers ou les données de marché en temps réel (utilisez plutôt les compétences en matière de données de marché).
