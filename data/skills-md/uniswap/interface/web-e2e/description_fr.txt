Exécutez, créez et déboguez les tests Playwright e2e pour l'application Web. Invoquez TOUJOURS cette compétence à l'aide de l'outil SlashCommand (c'est-à-dire `/web-e2e`) AVANT de tenter d'exécuter des tests e2e, des tests de dramaturge, des tests d'enclume ou des échecs de tests de débogage. N'exécutez PAS `bun playwright test` ou d'autres commandes e2e directement - vous devez d'abord invoquer cette compétence pour apprendre les commandes correctes et tester l'architecture.
