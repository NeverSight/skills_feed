Esegui, crea ed esegui il debug dei test Playwright e2e per l'app Web. Richiama SEMPRE questa abilità utilizzando lo strumento SlashCommand (ad esempio, `/web-e2e`) PRIMA di tentare di eseguire test e2e, test di drammaturgo, test di incudine o test di debug falliti. NON eseguire direttamente `bun playwright test` o altri comandi e2e: è necessario prima invocare questa abilità per apprendere i comandi corretti e testare l'architettura.
