Recherchez des exemples de code réels dans des millions de référentiels GitHub à l'aide de grep.app. Cette compétence doit être utilisée lors de la recherche de modèles d'implémentation, d'exemples d'utilisation d'API, de modèles d'intégration de bibliothèque ou de références de code de production. Prend en charge la recherche de code littéral, les modèles d'expression régulière et le filtrage par langue/dépôt/chemin.
