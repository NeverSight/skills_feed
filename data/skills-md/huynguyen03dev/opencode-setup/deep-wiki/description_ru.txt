Получите доступ к документации и аналитическим данным, созданным искусственным интеллектом, для репозиториев GitHub через DeepWiki. Этот навык следует использовать при изучении незнакомых кодовых баз, понимании архитектуры репозитория, поиске шаблонов реализации или задавании вопросов о том, как работает проект GitHub. Поддерживает любой общедоступный репозиторий GitHub.
