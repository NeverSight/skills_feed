Accedi alla documentazione e agli approfondimenti generati dall'intelligenza artificiale per i repository GitHub tramite DeepWiki. Questa competenza dovrebbe essere utilizzata quando si esplorano basi di codice non familiari, si comprende l'architettura del repository, si trovano modelli di implementazione o si pongono domande su come funziona un progetto GitHub. Supporta qualsiasi repository GitHub pubblico.
