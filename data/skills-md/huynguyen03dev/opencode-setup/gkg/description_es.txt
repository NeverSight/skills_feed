Gráfico de conocimiento global para análisis de base de código. Esta habilidad debe usarse al buscar definiciones de código (funciones, clases, métodos), encontrar referencias a símbolos, comprender la estructura del código, analizar el uso de importaciones, generar mapas de repositorio o realizar análisis de impacto antes de la refactorización. Admite TypeScript, JavaScript, Python, Java y más.
