Глобальная сеть знаний для анализа кодовой базы. Этот навык следует использовать при поиске определений кода (функций, классов, методов), поиске ссылок на символы, понимании структуры кода, анализе использования импорта, создании карт репозитория или выполнении анализа воздействия перед рефакторингом. Поддерживает TypeScript, JavaScript, Python, Java и другие.
