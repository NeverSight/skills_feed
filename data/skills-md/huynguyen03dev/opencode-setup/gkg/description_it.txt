Global Knowledge Graph per l'analisi della base di codice. Questa competenza dovrebbe essere utilizzata durante la ricerca di definizioni di codice (funzioni, classi, metodi), la ricerca di riferimenti a simboli, la comprensione della struttura del codice, l'analisi dell'utilizzo dell'importazione, la generazione di mappe di repository o l'esecuzione di analisi di impatto prima del refactoring. Supporta TypeScript, JavaScript, Python, Java e altro.
