Globaler Wissensgraph für die Codebasisanalyse. Diese Fertigkeit sollte verwendet werden, wenn Sie nach Codedefinitionen (Funktionen, Klassen, Methoden) suchen, Verweise auf Symbole finden, die Codestruktur verstehen, die Importnutzung analysieren, Repository-Zuordnungen erstellen oder eine Auswirkungsanalyse vor dem Refactoring durchführen. Unterstützt TypeScript, JavaScript, Python, Java und mehr.
