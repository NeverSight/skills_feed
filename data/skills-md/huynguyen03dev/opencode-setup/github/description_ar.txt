تفاعل مع مستودعات GitHub والمشكلات وطلبات السحب والتعليمات البرمجية عبر خادم GitHub MCP. يجب استخدام هذه المهارة عند إدارة المستودعات، وإنشاء/تحديث الملفات، والعمل مع المشكلات والعلاقات العامة، والبحث عن التعليمات البرمجية/المستودعات/المستخدمين، وإنشاء الفروع، وإجراء مراجعات التعليمات البرمجية. يدعم جميع عمليات GitHub API الرئيسية.
