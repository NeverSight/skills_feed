Interactúe con repositorios, problemas, solicitudes de extracción y código de GitHub a través del servidor MCP de GitHub. Esta habilidad debe usarse al administrar repositorios, crear/actualizar archivos, trabajar con problemas y relaciones públicas, buscar código/repositorios/usuarios, crear sucursales y realizar revisiones de código. Admite todas las operaciones principales de la API de GitHub.
