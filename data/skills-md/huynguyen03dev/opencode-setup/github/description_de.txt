Interagieren Sie über den GitHub MCP-Server mit GitHub-Repositorys, Issues, Pull Requests und Code. Diese Fähigkeit sollte beim Verwalten von Repositorys, beim Erstellen/Aktualisieren von Dateien, beim Arbeiten mit Problemen und PRs, beim Durchsuchen von Code/Repos/Benutzern, beim Erstellen von Zweigen und beim Durchführen von Codeüberprüfungen verwendet werden. Unterstützt alle wichtigen GitHub-API-Vorgänge.
