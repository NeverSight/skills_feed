Взаимодействуйте с репозиториями GitHub, проблемами, запросами на включение и кодом через сервер GitHub MCP. Этот навык следует использовать при управлении репозиториями, создании/обновлении файлов, работе с проблемами и запросами на запросы, поиске кода/репозиториев/пользователей, создании веток и выполнении проверок кода. Поддерживает все основные операции API GitHub.
