Inspectez les assemblys .NET et les packages NuGet. À utiliser lorsque vous avez besoin de comprendre le contenu d'un package, d'afficher les surfaces d'API publiques, de comparer les API entre les versions ou d'auditer les assemblys pour SourceLink/déterminisme. Indispensable pour les tâches de développement .NET impliquant l'exploration de packages ou la découverte d'API.
