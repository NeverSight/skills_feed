Проверьте сборки .NET и пакеты NuGet. Используйте, когда вам нужно понять содержимое пакета, просмотреть общедоступные поверхности API, сравнить API разных версий или проверить сборки на предмет SourceLink/детерминизма. Необходим для задач разработки .NET, включающих исследование пакетов или обнаружение API.
