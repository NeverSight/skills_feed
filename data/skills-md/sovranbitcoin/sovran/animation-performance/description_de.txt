Bietet Richtlinien zur Leistungsoptimierung von React Native Reanimated, React Native Worklets und React Native Gesture Handler für reibungslose Animationen. Gilt für Aufgaben, die gemeinsame Werte, Worklets, Frame-Callbacks, Gesten-Handler, animierte Stile, erneutes Rendern oder das Debuggen von Animationsfehlern und Frame-Drops umfassen.
