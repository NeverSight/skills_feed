Proporciona pautas de optimización del rendimiento de React Native Reanimated, React Native Worklets y React Native Gesture Handler para animaciones fluidas. Se aplica a tareas que involucran valores compartidos, worklets, devoluciones de llamadas de fotogramas, controladores de gestos, estilos animados, re-renderizaciones o depuración de animaciones y caídas de fotogramas.
