Содержит рекомендации по оптимизации производительности React Native Reanimated, React Native Worklets, React Native Gesture Handler для плавной анимации. Применяется к задачам, связанным с общими значениями, рабочими модулями, обратными вызовами кадров, обработчиками жестов, анимированными стилями, повторной визуализацией или отладкой прерываний анимации и выпадения кадров.
