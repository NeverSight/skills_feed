يوفر React Native Reanimated وReact Native Worklets وإرشادات تحسين أداء React Native Gesture Handler للحصول على رسوم متحركة سلسة. ينطبق على المهام التي تتضمن قيمًا مشتركة، أو أوراق عمل، أو عمليات رد اتصال الإطار، أو معالجات الإيماءات، أو الأنماط المتحركة، أو عمليات إعادة العرض، أو تصحيح أخطاء الرسوم المتحركة غير المرغوب فيها وإسقاط الإطارات.
