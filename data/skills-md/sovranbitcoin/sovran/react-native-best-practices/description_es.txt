Proporciona pautas de optimización del rendimiento de React Native para FPS, TTI, tamaño de paquete, pérdidas de memoria, rerenderizaciones y animaciones. Se aplica a tareas que involucran optimización de Hermes, bloqueo de subprocesos JS, sobrecarga de puentes, FlashList, módulos nativos o depuración de bloqueos y caídas de fotogramas.
