Интегрируйте AI-компоненты CopilotKit во внешний интерфейс Next.js для создания агентных пользовательских интерфейсов. Включает контекстно-зависимые агенты ИИ, которые могут считывать состояние приложения и запускать инструменты/действия. Поддерживает пользовательские адаптеры для автономных LLM и интеграцию с несколькими поставщиками.
