Создавайте красивые интерфейсы терминала с помощью Charmbracelet. Для скриптов SHELL/BASH: используйте Gum (подсказки, счетчики, выбор), VHS (запись), Mods (AI), Freeze (скриншоты), Glow (уценку). Для приложений GO: используйте Bubble Tea (фреймворк), Bubbles (компоненты), Lip Gloss (стиль), Huh (формы), Glamour (уценка), Harmonica (анимация). Для ИНФРАСТРУКТУРЫ: Wish (SSH-приложения), Soft Serve (Git), teatest (тестирование). Запуск по: сценариям оболочки, требующим пользовательского интерфейса, инструментов Go CLI, панелей терминала, приложений SSH или «сделать этот CLI красивее».
