Créez de superbes interfaces utilisateur de terminal avec Charmbracelet. Pour les scripts SHELL/BASH : utilisez Gum (invites, spinners, sélection), VHS (enregistrement), Mods (AI), Freeze (captures d'écran), Glow (markdown). Pour les applications GO : utilisez Bubble Tea (framework), Bubbles (composants), Lip Gloss (style), Huh (formes), Glamour (markdown), Harmonica (animation). Pour INFRASTRUCTURE : Wish (applications SSH), Soft Serve (Git), teatest (test). Déclenchez sur : des scripts shell nécessitant une interface utilisateur, des outils Go CLI, des tableaux de bord de terminal, des applications SSH ou "rendre cette CLI plus jolie".
