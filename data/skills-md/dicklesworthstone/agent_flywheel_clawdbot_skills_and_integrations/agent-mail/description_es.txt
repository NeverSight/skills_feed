MCP Agent Mail: capa de coordinación similar a un correo para flujos de trabajo de múltiples agentes. Identidades, bandeja de entrada/salida, reservas de archivos, políticas de contacto, mensajería encadenada, guardia previa a la confirmación, supervisor humano, exportaciones estáticas, recuperación ante desastres.
