يزود تامبو بالبيانات والإمكانيات عبر الأدوات المخصصة وخوادم MCP ومساعدي السياق والموارد. استخدمها عند تسجيل الأدوات التي يمكن لـ Tambo الاتصال بها، أو توصيل خوادم MCP، أو إضافة سياق إلى الرسائل، أو تنفيذ @mentions، أو توفير مصادر بيانات إضافية باستخدام DefineTool، أو mcpServers، أو contextHelpers، أو useTamboContextAttachment.
