Регистрирует существующие компоненты React в Tambo, чтобы ИИ мог их визуализировать. Используйте, когда пользователь хочет сделать свои существующие компоненты доступными для ИИ, зарегистрировать компоненты для генеративного пользовательского интерфейса, преобразовать компоненты React в компоненты Tambo или упоминает /add-comComponents-to-registry. Для создания НОВЫХ компонентов см. навык «Компоненты». Для настройки проекта используйте навыки «добавить к существующему проекту» или «начать с нуля».
