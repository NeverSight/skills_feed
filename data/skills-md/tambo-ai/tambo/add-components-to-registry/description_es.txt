Registra los componentes de React existentes con Tambo para que la IA pueda renderizarlos. Úselo cuando el usuario quiera que sus componentes existentes estén disponibles para la IA, registrar componentes para la interfaz de usuario generativa, convertir componentes de React en componentes de Tambo o mencionar /add-components-to-registry. Para crear componentes NUEVOS, consulte la habilidad de componentes. Para la configuración del proyecto, utilice las habilidades para agregar a un proyecto existente o comenzar desde cero.
