Intègre Tambo dans les projets React EXISTANTS en détectant la pile technologique et en adaptant l'installation. À utiliser lors de l'ajout de Tambo à une application existante, de l'intégration aux frameworks actuels ou lorsque l'utilisateur dispose d'une base de code existante à laquelle il souhaite ajouter une IA/une interface utilisateur générative. Pour démarrer un NOUVEAU projet à partir de zéro, utilisez plutôt la compétence repartir de zéro. Pour enregistrer des composants existants, utilisez la compétence d'ajout de composants au registre.
