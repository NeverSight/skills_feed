Integra Tambo en proyectos EXISTENTES de React detectando la pila tecnológica y adaptando la instalación. Úselo al agregar Tambo a una aplicación existente, al integrarla con marcos actuales o cuando el usuario tiene una base de código existente a la que desea agregar IA/IU generativa. Para comenzar un proyecto NUEVO desde cero, utilice la habilidad de empezar desde cero. Para registrar componentes existentes, utilice la habilidad Agregar componentes al registro.
