Integriert Tambo in BESTEHENDE React-Projekte, indem es den Tech-Stack erkennt und die Installation anpasst. Verwenden Sie diese Option, wenn Sie Tambo zu einer vorhandenen App hinzufügen, in aktuelle Frameworks integrieren oder wenn der Benutzer über eine vorhandene Codebasis verfügt, zu der er KI/generative Benutzeroberfläche hinzufügen möchte. Um ein NEUES Projekt von Grund auf neu zu starten, verwenden Sie stattdessen die Fähigkeit, von Grund auf neu zu beginnen. Verwenden Sie zum Registrieren vorhandener Komponenten den Skill „Komponenten zur Registrierung hinzufügen“.
