Guía la creación de un nuevo proyecto Tambo desde cero con recomendaciones tecnológicas. Úselo cuando el usuario quiera comenzar un NUEVO proyecto, crear una nueva aplicación con Tambo, crear algo de la nada o necesite ayuda para elegir tecnologías para una nueva aplicación de interfaz de usuario generativa. Para agregar Tambo a proyectos EXISTENTES, utilice la habilidad Agregar a proyectos existentes.
