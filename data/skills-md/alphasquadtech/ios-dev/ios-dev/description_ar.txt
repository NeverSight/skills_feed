سير عمل تطوير iOS لإنشاء تطبيقات iOS وتشغيلها واختبارها وتصحيح الأخطاء وإصلاحها باستخدام Xcode وSimulator ولقطات الشاشة وأتمتة Maestro UI. يُستخدم عندما: إنشاء تطبيق iOS، أو التشغيل على جهاز محاكاة، أو التقاط لقطات شاشة للمحاكاة، أو التنقل في واجهة مستخدم التطبيق، أو تصحيح مشكلات iOS، أو إنشاء مشاريع Xcode، أو كتابة تعليمات برمجية SwiftUI أو UIKit، أو تشغيل xcodebuild، أو استخدام xcrun simctl، أو أي مهمة تطوير iOS/iPhone/iPad.
