قواعد اللعبة الشاملة لتطوير النجوم. يغطي عقود Soroban الذكية (Rust SDK)، وStellar CLI، وJavaScript/Python/Go SDKs لتطبيقات العميل، وStellar RPC (المفضل) وHorizon API (القديمة)، وStellar Assets مقابل Soroban tokens (SAC Bridge)، وتكامل المحفظة (Freighter، Stellar Wallets Kit)، والحسابات الذكية مع مفاتيح المرور، وأنماط إثبات المعرفة الصفرية الحساسة للحالة، واستراتيجيات الاختبار، وأنماط الأمان، والمزالق الشائعة. مُحسّن للمدفوعات، وترميز الأصول، وDeFi، والتطبيقات التي تهتم بالخصوصية، والتطبيقات المالية. يُستخدم عند البناء على Stellar، أو Soroban، أو العمل مع XLM، أو Stellar Assets، أو خطوط الثقة، أو المراسي، أو SEPs، أو إثباتات ZK، أو Stellar RPC/Horizon APIs.
