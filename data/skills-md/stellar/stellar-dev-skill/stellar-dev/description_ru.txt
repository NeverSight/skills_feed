Комплексное руководство по разработке Stellar. Охватывает смарт-контракты Soroban (Rust SDK), Stellar CLI, SDK JavaScript/Python/Go для клиентских приложений, Stellar RPC (предпочтительно) и Horizon API (устаревший), Stellar Assets и токены Soroban (мост SAC), интеграцию кошелька (Freighter, Stellar Wallets Kit), смарт-аккаунты с ключами доступа, чувствительные к статусу шаблоны доказательства с нулевым разглашением, стратегии тестирования, шаблоны безопасности и распространенные ошибки. Оптимизирован для платежей, токенизации активов, DeFi, приложений с учетом конфиденциальности и финансовых приложений. Используйте при создании Stellar, Soroban или работе с XLM, Stellar Assets, линиями доверия, якорями, SEP, доказательствами ZK или API Stellar RPC/Horizon.
