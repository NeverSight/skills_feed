Manuel de développement Stellar de bout en bout. Couvre les contrats intelligents Soroban (SDK Rust), Stellar CLI, les SDK JavaScript/Python/Go pour les applications client, Stellar RPC (préféré) et Horizon API (héritée), les jetons Stellar Assets vs Soroban (pont SAC), l'intégration de portefeuille (Freighter, Stellar Wallets Kit), les comptes intelligents avec clés d'accès, les modèles de preuve de connaissance nulle sensibles à l'état, les stratégies de test, les modèles de sécurité et les pièges courants. Optimisé pour les paiements, la tokenisation des actifs, la DeFi, les applications respectueuses de la confidentialité et les applications financières. À utiliser lors de la construction sur Stellar, Soroban ou lorsque vous travaillez avec XLM, Stellar Assets, des lignes de confiance, des ancres, des SEP, des preuves ZK ou les API Stellar RPC/Horizon.
