Этот навык следует использовать при работе с DSPy.rb, средой Ruby для создания типобезопасных компонуемых LLM-приложений. Используйте это при реализации предсказуемых функций искусственного интеллекта, создании подписей и модулей LLM, настройке поставщиков языковых моделей (OpenAI, Anthropic, Gemini, Ollama), создании систем агентов с помощью инструментов, оптимизации подсказок или тестировании функций LLM в приложениях Ruby.
