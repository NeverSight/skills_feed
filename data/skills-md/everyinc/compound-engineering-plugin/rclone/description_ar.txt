تحميل الملفات ومزامنتها وإدارتها عبر موفري التخزين السحابي باستخدام rclone. يُستخدم عند تحميل الملفات (الصور ومقاطع الفيديو والمستندات) إلى S3 أو Cloudflare R2 أو Backblaze B2 أو Google Drive أو Dropbox أو أي وحدة تخزين متوافقة مع S3. يتم تشغيلها عند "التحميل إلى S3"، أو "المزامنة إلى السحابة"، أو "rclone"، أو "النسخ الاحتياطي للملفات"، أو "تحميل الفيديو/الصورة إلى المجموعة"، أو طلبات نقل الملفات إلى وحدة التخزين البعيدة.
