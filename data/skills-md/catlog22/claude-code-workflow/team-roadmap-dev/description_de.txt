Einheitliche Teamkompetenz für einen Roadmap-gesteuerten Entwicklungsworkflow. Der Koordinator bespricht die Roadmap mit dem Benutzer und verteilt dann die Pipeline für die phasenweise Ausführung (Plan -> Ausführen -> Überprüfen). Alle Rollen rufen diesen Skill mit --role arg auf. Wird bei „team roadmap-dev“ ausgelöst.
