قم بإنشاء مستندات مواصفات تصميم حقوق الطبع والنشر للبرامج المتوافقة مع معايير مركز حماية حقوق الطبع والنشر الصيني (CPCC). ينشئ مستندات تصميم كاملة باستخدام مخططات حورية البحر بناءً على تحليل كود المصدر. يُستخدم لتسجيل حقوق الطبع والنشر للبرامج، أو إنشاء مواصفات التصميم، أو إنشاء مستندات متوافقة مع CPCC، أو توثيق البرامج لحماية الملكية الفكرية. المشغلات على "软件著作权"، "设计说明书"، "版权登记"، "CPCC"، "软著申请".
