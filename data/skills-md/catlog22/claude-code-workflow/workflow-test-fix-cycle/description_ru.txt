Сквозной рабочий процесс тестирования и исправления генерирует сеансы тестирования с прогрессивными уровнями (L0–L3), а затем выполняет итеративные циклы исправлений до тех пор, пока процент прохождения не >= 95%. Объединяет генерацию тестов и выполнение цикла тестирования в единый конвейер. Запускается по «рабочему процессу: цикл тестирования-исправления».
