Унифицированный конвейер тестирования и исправления, сочетающий генерацию тестов (сеанс, контекст, анализ, создание задач) с итеративным выполнением цикла тестирования (адаптивная стратегия, прогрессивное тестирование, откат CLI). Запускается по «рабочему процессу: test-fix-gen», «workflow: test-cycle-execute», «рабочему процессу тестового исправления».
