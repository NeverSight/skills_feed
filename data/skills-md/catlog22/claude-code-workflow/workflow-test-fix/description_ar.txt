خط أنابيب موحد للاختبار والإصلاح يجمع بين إنشاء الاختبار (الجلسة، والسياق، والتحليل، وتكوين المهمة) مع تنفيذ دورة الاختبار التكرارية (استراتيجية التكيف، والاختبار التدريجي، وتراجع واجهة سطر الأوامر). مشغلات على "سير العمل: اختبار الإصلاح-جنرال"، "سير العمل: تنفيذ دورة الاختبار"، "سير عمل إصلاح الاختبار".
