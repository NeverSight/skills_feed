Прогрессивное планирование дорожной карты на уровне требований с созданием проблем. Разлагает требования на конвергентные уровни или последовательности задач, создает проблемы посредством создания проблем ccw и генерирует roadmap.md для проверки человеком. Проблемы хранятся в файле .workflow/issues/issues.jsonl (единый источник достоверной информации).
