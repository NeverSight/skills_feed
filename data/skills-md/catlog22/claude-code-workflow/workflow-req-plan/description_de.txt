Progressive Roadmap-Planung auf Anforderungsebene mit Problemerstellung. Zerlegt Anforderungen in konvergente Ebenen oder Aufgabensequenzen, erstellt Probleme über CCW Issue Create und generiert roadmap.md zur menschlichen Überprüfung. In .workflow/issues/issues.jsonl (Single Source of Truth) gespeicherte Probleme.
