Универсальный инструмент диагностики и оптимизации навыков. Обнаруживайте и устраняйте проблемы с выполнением навыков, включая взрыв контекста, «длинное забывание», нарушение потока данных и сбои координации агентов. Поддерживает Gemini CLI для глубокого анализа. Запускает «настройку навыков», «настройку навыков», «диагностику навыков», «оптимизацию навыков», «отладку навыков».
