Interaktives Issue-Management mit menügesteuerten CRUD-Operationen. Zum Verwalten von Problemen, Anzeigen des Problemstatus, Bearbeiten von Problemfeldern, Durchführen von Massenvorgängen oder Anzeigen des Problemverlaufs. Löst „Problem verwalten“, „Problem auflisten“, „Problem bearbeiten“, „Problem löschen“, „Massenaktualisierung“, „Problem-Dashboard“, „Problemverlauf“ und „abgeschlossene Probleme“ aus.
