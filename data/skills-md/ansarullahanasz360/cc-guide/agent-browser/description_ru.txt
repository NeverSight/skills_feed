Используйте интерфейс командной строки Agent Browser для автономной автоматизации браузера и тестирования E2E. Вызывается, когда пользователю необходимо автоматизировать задачи браузера, запускать E2E-тесты, делать снимки экрана, взаимодействовать с веб-страницами или выполнять задачи по очистке веб-страниц.
