Создавайте самопроверяющиеся PRD для автономного выполнения. Опрашивает пользователей, чтобы собрать требования, затем генерирует структурированный файл prd.json с поэтапной реализацией и соответствующими стратегиями тестирования. Поддерживает 7 категорий задач с рабочими процессами для конкретного типа. Используйте, когда пользователь говорит «создать PRD», «PRD для», «Спланировать функцию», «Спланировать это», «Написать PRD» или хочет спланировать любую многоэтапную работу по реализации.
