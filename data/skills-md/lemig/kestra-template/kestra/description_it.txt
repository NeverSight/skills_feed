Genera, esegui il debug e ottimizza i file YAML di orchestrazione del flusso di lavoro di Kestra. Kestra è una piattaforma di orchestrazione dei dati dichiarativa open source che utilizza la sintassi YAML.

Usa questa abilità quando:
- Creazione di nuovi flussi Kestra (flussi di lavoro)
- Debug o correzione di Kestra YAML esistente
- Aggiunta di trigger (programmazione, webhook, basati sul flusso)
- Implementazione della gestione degli errori, dei nuovi tentativi o dei timeout
- Scrittura di attività di script (Python, Shell, Node.js, ecc.)
- Costruire flussi di lavoro paralleli o condizionali
- Conversione da Airflow/Prefetto a Kestra
- Qualsiasi domanda sulla sintassi, sui plugin o sulle migliori pratiche di Kestra
