Conéctese directamente a la API del servicio web AMap mediante un script para realizar geocodificación, geocodificación inversa, ubicación de IP, clima, planificación de rutas, medición de distancias y búsqueda de puntos de interés. Utilícelo cuando el usuario solicite "consulta AMap", "planificación de rutas", "codificación geográfica", "búsqueda de puntos de interés" o necesite llamar a la API AMap desde un script de línea de comandos.
