اتصل مباشرة بـ AMap Web Service API عبر البرنامج النصي لإجراء الترميز الجغرافي والترميز الجغرافي العكسي وموقع IP والطقس وتخطيط المسار وقياس المسافة والبحث عن النقاط المهمة. استخدم هذا عندما يطلب المستخدم "استعلام AMap"، أو "تخطيط الطريق"، أو "الترميز الجغرافي"، أو "بحث النقاط المهمة"، أو يحتاج إلى استدعاء AMap API من برنامج نصي لسطر الأوامر.
