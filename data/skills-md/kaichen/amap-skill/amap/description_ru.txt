Подключайтесь напрямую к API веб-службы AMap с помощью сценария для выполнения геокодирования, обратного геокодирования, определения IP-координаты, прогноза погоды, планирования маршрута, измерения расстояния и поиска POI. Используйте это, когда пользователь запрашивает «запрос AMap», «планирование маршрута», «геокодирование», «поиск POI» или ему необходимо вызвать API AMap из сценария командной строки.
