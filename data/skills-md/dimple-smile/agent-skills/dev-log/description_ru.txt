Решение для совместной отладки искусственного интеллекта. Преобразование console.log в HTTP-запросы для сбора логов; после того, как пользователь завершит работу, ИИ сможет просматривать и анализировать их самостоятельно — скриншоты или копирование с консоли не требуются. Поддерживает Claude Code, OpenCode, Cursor.
