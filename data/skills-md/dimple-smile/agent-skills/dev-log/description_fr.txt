Solution de collaboration pour le débogage de l'IA. Convertissez console.log en requêtes HTTP pour collecter les journaux ; une fois que l’utilisateur a fini de fonctionner, l’IA peut les visualiser et les analyser elle-même – aucune capture d’écran ni copie de la console n’est nécessaire. Prend en charge Claude Code, OpenCode, Curseur.
