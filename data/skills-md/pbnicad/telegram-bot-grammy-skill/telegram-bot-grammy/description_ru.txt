Навыки разработки ботов Telegram с использованием фреймворка grammY, TypeScript, Prisma ORM, тестирования Vitest, анализа Biome, развертывания в Cloudflare Workers с базой данных D1.
Варианты использования:
(1) Создание новых проектов Telegram Bot
(2) Добавление команд или функций к существующим ботам.
(3) Настройка развертывания Cloudflare Workers
(4) Настройка интеграции базы данных Prisma + D1.
(5) Написание тестовых примеров для ботов
(6) Настройка перехватчиков Git и действий GitHub CI/CD
