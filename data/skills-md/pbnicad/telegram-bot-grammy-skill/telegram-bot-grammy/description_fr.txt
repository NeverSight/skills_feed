Compétence en développement de Telegram Bot utilisant le framework GrammY, TypeScript, Prisma ORM, les tests Vitest, Biome Linting, déployé sur Cloudflare Workers avec la base de données D1.
Cas d'utilisation :
(1) Création de nouveaux projets Telegram Bot
(2) Ajout de commandes ou de fonctionnalités aux robots existants
(3) Configuration du déploiement de Cloudflare Workers
(4) Mise en place de l'intégration de la base de données Prisma + D1
(5) Rédaction de cas de test Bot
(6) Configuration des hooks Git et des actions GitHub CI/CD
