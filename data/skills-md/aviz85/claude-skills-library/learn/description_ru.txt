Преподавайте Клоду ЛЮБУЮ тему — библиотеки кода, API, концепции, инструменты, методологии или предметные области. Исследует через Интернет и документацию, а затем сохраняет знания как постоянный навык. Используйте, когда пользователь говорит «/learn <тема>», «узнайте об X», «научитесь Y», «станьте экспертом по Z». Примеры: «/learnstripe» для платежей, «/learn GTD» для производительности, «/learn израильское налоговое право» для знаний предметной области.
