Развертывайте статический или интерактивный контент внешнего интерфейса на страницах GitHub с помощью gh CLI. Используйте, когда пользователь хочет опубликовать, поделиться или сделать доступным любой контент HTML/CSS/JS, включая демонстрации, прототипы, визуализации, целевые страницы, портфолио, документацию, интерактивные инструменты, игры или любой браузерный проект. Активируйте всякий раз, когда контент должен быть доступен для публичного просмотра через URL-адрес, а не только тогда, когда явно упоминается «веб-сайт».
