Implemente contenido de interfaz estático o interactivo en páginas de GitHub utilizando gh CLI. Úselo cuando el usuario quiera publicar, compartir o hacer accesible cualquier contenido HTML/CSS/JS, incluidas demostraciones, prototipos, visualizaciones, páginas de destino, portafolios, documentación, herramientas interactivas, juegos o cualquier proyecto basado en navegador. Actívelo siempre que el contenido deba ser visible públicamente a través de una URL, no solo cuando se mencione explícitamente "sitio web".
