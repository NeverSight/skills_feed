基于 PostgreSQL 的语义和混合搜索，使用 pgvector 和 ParadeDB。
在实现矢量搜索、语义搜索、混合搜索时使用
或 PostgreSQL 中的全文搜索。涵盖 pgvector 设置、索引
（HNSW、IVFFlat）、混合搜索（FTS + BM25 + RRF）、ParadeDB as
Elasticsearch 替代方案，并使用 Cohere/交叉编码器重新排名。
支持 OpenAI 嵌入的向量（1536）和 halfvec（3072）类型。

触发器：pgvector、向量搜索、语义搜索、混合搜索、
嵌入搜索、PostgreSQL RAG、BM25、RRF、HNSW 索引、相似性搜索、
ParadeDB、pg_search、重新排名、Cohere 重新排名
