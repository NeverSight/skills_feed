pgvector と ParadeDB を使用した PostgreSQL ベースのセマンティック検索とハイブリッド検索。
ベクトル検索、セマンティック検索、ハイブリッド検索、
または PostgreSQL の全文検索。 pgvector のセットアップ、インデックス作成をカバーします。
(HNSW、IVFFlat)、ハイブリッド検索 (FTS + BM25 + RRF)、ParadeDB として
Elasticsearch の代替手段、および Cohere/クロスエンコーダーを使用した再ランキング。
OpenAI 埋め込みの Vector(1536) および Halfvec(3072) タイプをサポートします。
