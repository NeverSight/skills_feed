Семантический и гибридный поиск на основе PostgreSQL с использованием pgvector и ParadeDB.
Используйте при реализации векторного поиска, семантического поиска, гибридного поиска,
или полнотекстовый поиск в PostgreSQL. Охватывает настройку pgvector, индексирование.
(HNSW, IVFFlat), гибридный поиск (FTS + BM25 + RRF), ParadeDB как
Альтернатива Elasticsearch и изменение рейтинга с помощью Cohere/кросс-кодировщиков.
Поддерживает типы векторов (1536) и Halfvec (3072) для встраивания OpenAI.

Триггеры: pgvector, векторный поиск, семантический поиск, гибридный поиск,
поиск по встраиванию, PostgreSQL RAG, BM25, RRF, индекс HNSW, поиск по сходству,
ParadeDB, pg_search, изменение рейтинга, изменение рейтинга Cohere
