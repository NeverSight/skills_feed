Intégrez le SDK Outlit pour le suivi et l’analyse du parcours client. Se déclenche lorsque les utilisateurs doivent ajouter Outlit aux applications React, Next.js, Vue, Svelte, Angular, Astro, Node.js, Express ou Fastify. Couvre l'installation du SDK (@outlit/browser, @outlit/node), l'intégration du fournisseur d'authentification (Clerk, NextAuth, Supabase, Auth0, Firebase), la gestion des consentements RGPD, le suivi des événements, les événements du cycle de vie des utilisateurs (activer, engagé, inactif), le statut de facturation du client (en test, payé, désabonné) et l'intégration du webhook Stripe.
