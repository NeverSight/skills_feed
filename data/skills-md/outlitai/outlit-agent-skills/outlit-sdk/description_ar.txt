دمج Outlit SDK لتتبع رحلة العميل والتحليلات. يتم تشغيله عندما يحتاج المستخدمون إلى إضافة Outlit إلى تطبيقات React أو Next.js أو Vue أو Svelte أو Angular أو Astro أو Node.js أو Express أو Fastify. يغطي تثبيت SDK (@outlit/browser، @outlit/node)، وتكامل موفر المصادقة (Clerk، وNextAuth، وSupabase، وAuth0، وFirebase)، وإدارة موافقة القانون العام لحماية البيانات (GDPR)، وتتبع الأحداث، وأحداث دورة حياة المستخدم (تنشيط، مشارك، غير نشط)، وحالة فوترة العميل (التجربة، المدفوعة، المخفضة)، وتكامل خطاف الويب Stripe.
