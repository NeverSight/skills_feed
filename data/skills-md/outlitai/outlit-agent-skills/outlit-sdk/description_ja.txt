カスタマージャーニーの追跡と分析のために Outlit SDK を統合します。ユーザーが Outlit を React、Next.js、Vue、Svelte、Angular、Astro、Node.js、Express、または Fastify アプリに追加する必要があるときにトリガーされます。 SDK のインストール (@outlit/browser、@outlit/node)、認証プロバイダーの統合 (Clerk、NextAuth、Supabase、Auth0、Firebase)、GDPR 同意管理、イベント追跡、ユーザー ライフサイクル イベント (アクティブ化、エンゲージメント、非アクティブ)、顧客の請求ステータス (試用中、有料、チャーン)、および Stripe Webhook の統合をカバーします。
