Интегрируйте Outlit SDK для отслеживания пути клиента и аналитики. Срабатывает, когда пользователям необходимо добавить Outlit в приложения React, Next.js, Vue, Svelte, Angular, Astro, Node.js, Express или Fastify. Охватывает установку SDK (@outlit/browser, @outlit/node), интеграцию поставщика аутентификации (Clerk, NextAuth, Supabase, Auth0, Firebase), управление согласием GDPR, отслеживание событий, события жизненного цикла пользователя (активация, взаимодействие, неактивность), статус выставления счетов клиентам (пробная версия, оплачено, отработано) и интеграцию веб-перехватчика Stripe.
