Integre Outlit SDK para el seguimiento y análisis del recorrido del cliente. Se activa cuando los usuarios necesitan agregar Outlit a las aplicaciones React, Next.js, Vue, Svelte, Angular, Astro, Node.js, Express o Fastify. Cubre la instalación del SDK (@outlit/browser, @outlit/node), la integración del proveedor de autenticación (Clerk, NextAuth, Supabase, Auth0, Firebase), la gestión del consentimiento del RGPD, el seguimiento de eventos, los eventos del ciclo de vida del usuario (activado, comprometido, inactivo), el estado de facturación del cliente (en prueba, pagado, abandonado) y la integración del webhook de Stripe.
