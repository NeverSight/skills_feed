Da utilizzare quando si eseguono query sui dati dei clienti Outlit tramite gli strumenti MCP (outlit_*). Trigger su analisi dei clienti, metriche delle entrate, tempistiche delle attività, analisi di coorte, valutazione del rischio di abbandono, query SQL su dati di analisi o qualsiasi attività di esplorazione dei dati Outlit.
