Automatise les applications TUI du terminal (vim, htop, lazygit, dialog) via des sessions PTY gérées. À utiliser lorsque l'utilisateur doit interagir avec des applications de terminal, modifier des fichiers dans vim/nano, naviguer dans les menus TUI, cliquer sur les boutons/cases à cocher du terminal ou automatiser les flux de travail CLI avec des invites interactives.
