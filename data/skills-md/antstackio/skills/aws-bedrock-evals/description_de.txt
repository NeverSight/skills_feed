Erstellen und führen Sie LLM-as-Judge-Bewertungspipelines mithilfe von Amazon Bedrock Evaluation Jobs mit vorberechneten Inferenzdatensätzen aus. Verwenden Sie es, wenn Sie eine automatisierte Modellbewertung einrichten, Testszenarien entwerfen, vorberechnete Antworten sammeln, benutzerdefinierte Metriken konfigurieren, eine AWS-Infrastruktur erstellen, Bewertungsjobs ausführen, Ergebnisse analysieren und Ergebnisse iterieren.
