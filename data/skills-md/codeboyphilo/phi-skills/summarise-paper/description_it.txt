Da utilizzare quando le attività prevedono la lettura e il riepilogo di un documento accademico da un file PDF o da un URL arXiv. Se l'input è un PDF, convertilo in immagini per una lettura accurata (equazioni/figure). Se l'input è un URL arXiv, scarica il sorgente LaTeX da leggere. Genera il riepilogo come file LaTeX autonomo.
