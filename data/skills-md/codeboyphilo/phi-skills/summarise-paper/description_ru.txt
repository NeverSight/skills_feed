Используйте, когда задачи включают чтение и обобщение научной статьи из файла PDF или URL-адреса arXiv. Если входные данные представляют собой PDF-файл, преобразуйте его в изображения для точного чтения (уравнения/рисунки). Если входные данные представляют собой URL-адрес arXiv, загрузите исходный код LaTeX для чтения. Выведите сводку в виде отдельного файла LaTeX.
