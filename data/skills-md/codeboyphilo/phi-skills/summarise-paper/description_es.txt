Úselo cuando las tareas impliquen leer y resumir un artículo académico desde un archivo PDF o una URL arXiv. Si la entrada es un PDF, conviértalo a imágenes para una lectura precisa (ecuaciones/cifras). Si la entrada es una URL arXiv, descargue la fuente LaTeX para leerla. Genere el resumen como un archivo LaTeX independiente.
