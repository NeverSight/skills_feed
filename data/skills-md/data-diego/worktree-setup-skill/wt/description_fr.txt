Configurez des arbres de travail git avec la copie automatique des fichiers d'environnement, la liaison de services partagés, l'application de correctifs d'environnement, la vérification des tests de base et l'installation des dépendances. À utiliser lorsque l'utilisateur souhaite créer une configuration .wtsetup pour son projet, installer la fonction wt shell ou gérer les flux de travail de l'arbre de travail. Se déclenche sur les mentions d'arbres de travail, .wtsetup ou sur les demandes de configuration de la configuration de l'arbre de travail pour un dépôt.
