Konvertieren Sie NanoClaw von Apple Container in Docker für plattformübergreifende Unterstützung. Verwenden Sie diese Option, wenn der Benutzer unter Linux ausführen, zu Docker wechseln, eine plattformübergreifende Bereitstellung aktivieren oder von Apple Container weg migrieren möchte. Wird bei „Docker“, „Linux-Unterstützung“, „In Docker konvertieren“, „Plattformübergreifend“ oder „Apple-Container ersetzen“ ausgelöst.
