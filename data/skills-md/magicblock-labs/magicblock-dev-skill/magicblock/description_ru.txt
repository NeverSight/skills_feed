Шаблоны разработки MagicBlock Ephemeral Rollups для Solana. Охватывает потоки делегирования/отмены делегирования, архитектуру с двойным соединением (базовый уровень + ER), запуск запланированных задач, VRF для проверяемой случайности и интеграцию TypeScript/Anchor. Используйте для высокопроизводительных игр, приложений реального времени и высокой пропускной способности транзакций на Solana.
