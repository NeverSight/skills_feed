Modèles de développement de MagicBlock Ephemeral Rollups pour Solana. Couvre les flux de délégation/non-délégation, l'architecture à double connexion (couche de base + ER), les manivelles pour les tâches planifiées, le VRF pour le caractère aléatoire vérifiable et l'intégration TypeScript/Anchor. À utiliser pour des jeux hautes performances, des applications en temps réel et un débit de transaction rapide sur Solana.
