Referenz zur Arco Design React UI-Komponentenbibliothek (@arco-design/web-react). Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine Seite erstellen, eine Benutzeroberfläche erstellen, Frontend-Code schreiben, eine Webanwendung entwickeln, ein Dashboard entwerfen oder eine beliebige React-Schnittstelle implementieren möchte – insbesondere, wenn er Arco, arco-design, @arco-design/web-react oder einen beliebigen Namen einer Arco-Komponente (Schaltfläche, Tabelle, Formular, Modal, Auswählen, Menü usw.) erwähnt. Deckt alle 70 Komponenten mit vollständiger API, Codebeispielen, Importmustern, Themen, i18n, Layout, Formularen, Tabellen, Modalitäten, Navigation, Dateneingabe, Datenanzeige, Feedback, responsivem Design und Best Practices ab.
