Справочник по библиотеке компонентов пользовательского интерфейса Arco Design React (@arco-design/web-react). Используйте этот навык всякий раз, когда пользователь просит создать страницу, создать пользовательский интерфейс, написать код внешнего интерфейса, разработать веб-приложение, спроектировать панель мониторинга или реализовать любой интерфейс React — особенно когда он упоминает Arco,arco-design, @arco-design/web-react или имя любого компонента Arco (кнопка, таблица, форма, модальное окно, выбор, меню и т. д.). Охватывает все 70 компонентов с полным API, примерами кода, шаблонами импорта, темами, i18n, макетом, формами, таблицами, модальными окнами, навигацией, вводом данных, отображением данных, обратной связью, адаптивным дизайном и лучшими практиками.
