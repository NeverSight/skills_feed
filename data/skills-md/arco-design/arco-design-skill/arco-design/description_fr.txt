Référence de la bibliothèque de composants de l'interface utilisateur Arco Design React (@arco-design/web-react). Utilisez cette compétence chaque fois que l'utilisateur demande à créer une page, créer une interface utilisateur, écrire du code frontend, développer une application Web, concevoir un tableau de bord ou implémenter une interface React, en particulier lorsqu'il mentionne Arco, arco-design, @arco-design/web-react ou tout nom de composant Arco (Bouton, Table, Formulaire, Modal, Sélection, Menu, etc.). Couvre les 70 composants avec API complète, exemples de code, modèles d'importation, thèmes, i18n, mise en page, formulaires, tableaux, modaux, navigation, saisie de données, affichage des données, commentaires, conception réactive et meilleures pratiques.
