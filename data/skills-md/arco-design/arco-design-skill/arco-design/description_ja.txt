Arco Design React UI コンポーネント ライブラリ リファレンス (@arco-design/web-react)。ユーザーがページの構築、UI の作成、フロントエンド コードの作成、Web アプリケーションの開発、ダッシュボードの設計、または React インターフェイスの実装を要求するときは常に、このスキルを使用します。特に、Arco、arco-design、@arco-design/web-react、または Arco コンポーネント名 (ボタン、テーブル、フォーム、モーダル、選択、メニューなど) について言及する場合に使用します。完全な API、コード例、インポート パターン、テーマ、i18n、レイアウト、フォーム、テーブル、モーダル、ナビゲーション、データ入力、データ表示、フィードバック、レスポンシブ デザイン、ベスト プラクティスを備えた 70 個のコンポーネントすべてをカバーします。
