モバイル アプリでのユーザー エンゲージメントを高めるために、ローカル プッシュ通知キャンペーンを設計および実装します。ユーザーがローカル プッシュ通知、アプリ内通知、ユーザー エンゲージメント キャンペーン、保持メッセージング、再エンゲージメント ナッジについて言及したり、スケジュールされた通知をアプリに追加したい場合には、このスキルを使用します。また、ユーザーが通知タイミング戦略、オンボーディング通知フロー、またはワークアウト/習慣/目標リマインダー システムについて尋ねたときにもトリガーされます。 iOS (Swift/UNUserNotificationCenter)、Android (Kotlin/WorkManager+NotificationManager)、Flutter (flutter_local_notifications)、および React Native (notifee/expo-notifications) をカバーします。
