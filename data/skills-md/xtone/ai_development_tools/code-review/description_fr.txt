Nous effectuons des revues de code génériques pour les PR et les modifications de code. Nous évaluons la qualité, la sécurité et la conception du code par rapport à des normes communes et décidons de l'approuver ou de le rejeter. Utilisez-le lorsque vous recevez des demandes telles que « Vérifier ce PR », « Effectuer une révision du code » ou « Vérification préalable à la fusion » ou lors de l'exécution de la commande gh pr view.
