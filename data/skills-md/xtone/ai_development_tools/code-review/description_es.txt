Realizamos revisiones de código genérico para relaciones públicas y cambios de código. Evaluamos la calidad, la seguridad y el diseño del código según estándares comunes y decidimos si lo aprobamos o lo rechazamos. Utilícelo cuando reciba solicitudes como "Revisar este PR", "Hacer una revisión del código" o "Verificación previa a la fusión" o cuando ejecute el comando gh pr view.
