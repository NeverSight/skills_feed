Leitfaden zum Erstellen von Claude Code-Plugins mit Befehlen, Agenten, Skills, Hooks und MCP-Servern. Verwenden Sie diese Option, wenn der Benutzer ein neues Plugin erstellen, Plugin-Komponenten (Befehle, Agenten, Hooks) hinzufügen, einen Plugin-Marktplatz einrichten möchte oder Hilfe bei der Konfiguration von Plugin.json benötigt.
