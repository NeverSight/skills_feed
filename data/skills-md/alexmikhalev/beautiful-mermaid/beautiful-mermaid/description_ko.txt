아름다운 인어 라이브러리를 사용하여 인어 다이어그램을 SVG 또는 ASCII 아트로 렌더링합니다.
15가지 내장 테마와 Shiki VS Code 테마 지원을 통한 초고속 렌더링.
다음 경우에 사용하십시오: (1) Mermaid 구문에서 다이어그램 생성, (2) 순서도 렌더링,
상태, 시퀀스, 클래스 또는 ER 다이어그램, (3) 다음에 대한 ASCII 다이어그램 생성
터미널/마크다운 출력, (4) VS Code 색 구성표와 일치하는 테마 다이어그램,
(5) 문서나 웹 UI에 다이어그램을 삽입합니다. 다음을 사용하여 SVG 출력을 지원합니다.
일반 텍스트 컨텍스트에 대한 라이브 테마 전환 및 ASCII/유니코드 출력.
