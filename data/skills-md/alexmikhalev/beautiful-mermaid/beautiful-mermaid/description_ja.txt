beautiful-mermaid ライブラリを使用して、人魚の図を SVG または ASCII アートとしてレンダリングします。
15 の組み込みテーマと Shiki VS Code テーマのサポートによる超高速レンダリング。
次の場合に使用します: (1) Mermaid 構文から図を作成する、(2) フローチャートをレンダリングする、
状態、シーケンス、クラス、または ER 図、(3) ASCII 図の生成
ターミナル/マークダウン出力、(4) VS コードのカラースキームに一致するテーマ図、
(5) ドキュメントまたは Web UI への図の埋め込み。 SVG出力をサポート
ライブテーマの切り替えとプレーンテキストコンテキストの ASCII/Unicode 出力。
