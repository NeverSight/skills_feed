Guía las revisiones de código de Android con listas de verificación específicas del tipo y validación de patrones MVVM/Compose. Úselo al revisar relaciones públicas de Android, solicitudes de extracción, diferencias o cambios locales que involucren archivos Kotlin, ViewModel, Composable, Repository o Gradle. Activado por "revisar relaciones públicas", "revisar cambios", "verificar este código", "revisión de Android" o solicitudes de revisión de código que mencionen bitwarden/android. Carga listas de verificación especializadas para agregar funciones, corregir errores, mejorar la interfaz de usuario, refactorizar, actualizar dependencias y cambiar la infraestructura.
