Проводит проверку кода Android с помощью контрольных списков для конкретных типов и проверки шаблонов MVVM/Compose. Используйте при просмотре PR-запросов Android, запросов на включение, различий или локальных изменений, связанных с файлами Kotlin, ViewModel, Composable, Repository или Gradle. Запускается по запросам «проверка PR», «проверка изменений», «проверка этого кода», «проверка Android» или запросов на проверку кода с упоминанием bitwarden/android. Загружает специализированные контрольные списки для добавления функций, исправления ошибок, усовершенствований пользовательского интерфейса, рефакторинга, обновлений зависимостей и изменений инфраструктуры.
