Координируйте агентов через интерфейс командной строки AMQ для обмена межагентскими сообщениями на основе файлов. Используйте, когда вам нужно отправлять сообщения другому агенту (Claude/Codex), получать сообщения от агентов-партнеров, настраивать режим совместной работы между Claude Code и Codex CLI или управлять связью между агентами в любом многоагентном рабочем процессе. Триггеры включают «код сообщения», «поговорить с Клодом», «сотрудничать с агентом-партнером», «AMQ», «межагентный обмен сообщениями» или «координацию агента».
