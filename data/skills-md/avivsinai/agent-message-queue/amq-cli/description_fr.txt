Coordonnez les agents via l'AMQ CLI pour la messagerie inter-agents basée sur des fichiers. À utiliser lorsque vous devez envoyer des messages à un autre agent (Claude/Codex), recevoir des messages d'agents partenaires, configurer le mode coopératif entre Claude Code et Codex CLI ou gérer la communication d'agent à agent dans n'importe quel flux de travail multi-agent. Les déclencheurs incluent « codex de message », « parler à Claude », « collaborer avec un agent partenaire », « AMQ », « messagerie inter-agents » ou « coordination d'agents ».
