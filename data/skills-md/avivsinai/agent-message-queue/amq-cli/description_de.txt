Koordinieren Sie Agenten über die AMQ-CLI für dateibasierte Nachrichten zwischen Agenten. Verwenden Sie diese Option, wenn Sie Nachrichten an einen anderen Agenten (Claude/Codex) senden, Nachrichten von Partneragenten empfangen, den Koop-Modus zwischen Claude Code und Codex CLI einrichten oder die Agent-zu-Agent-Kommunikation in einem beliebigen Multi-Agent-Workflow verwalten müssen. Zu den Auslösern gehören „Nachrichtencodex“, „Mit Claude sprechen“, „Mit Partneragent zusammenarbeiten“, „AMQ“, „Mitteilung zwischen Agenten“ oder „Agentenkoordination“.
