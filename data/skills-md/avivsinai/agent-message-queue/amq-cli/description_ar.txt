تنسيق الوكلاء عبر AMQ CLI للمراسلة بين الوكلاء المستندة إلى الملفات. يُستخدم عندما تحتاج إلى إرسال رسائل إلى وكيل آخر (Claude/Codex)، أو تلقي رسائل من الوكلاء الشركاء، أو إعداد وضع التعاون بين Claude Code وCodex CLI، أو إدارة الاتصال من وكيل إلى وكيل في أي سير عمل متعدد الوكلاء. تتضمن المشغلات "مخطوطة الرسالة"، أو "التحدث إلى كلود"، أو "التعاون مع الوكيل الشريك"، أو "AMQ"، أو "المراسلة بين الوكلاء"، أو "تنسيق الوكيل".
