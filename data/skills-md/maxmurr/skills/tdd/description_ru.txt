Проводит агент через разработку через тестирование с использованием red-green-refactor. Используйте, когда пользователь упоминает TDD, красно-зеленый рефакторинг, разработку с упором на тестирование, TDD снаружи внутри, TDD-макист, TDD лондонской школы, TDD принятия или TDD с двойным циклом. Не используйте для написания тестов E2E/Playwright, настройки средств запуска тестов или фреймворков, добавления тестов без методологии TDD или общих рекомендаций по тестированию.
