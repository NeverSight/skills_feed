Recherchez, installez, configurez et intégrez les composants de l'interface utilisateur de l'outil dans les applications React à l'aide des entrées de registre shadcn, des vérifications de compatibilité, du câblage d'exécution échafaudé et des flux de travail de dépannage. À utiliser lorsque les développeurs demandent à ajouter un ou plusieurs composants de Tool UI, à choisir des composants pour un cas d'utilisation, à vérifier la compatibilité ou à câbler les charges utiles de Tool UI dans l'interface utilisateur de l'assistant ou dans une pile de discussion/d'exécution existante.
