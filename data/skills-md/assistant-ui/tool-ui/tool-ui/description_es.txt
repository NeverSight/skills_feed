Encuentre, instale, configure e integre componentes de Tool UI en aplicaciones React utilizando entradas de registro shadcn, comprobaciones de compatibilidad, cableado de tiempo de ejecución estructurado y flujos de trabajo de solución de problemas. Úselo cuando los desarrolladores soliciten agregar uno o más componentes de la interfaz de usuario de la herramienta, elijan componentes para un caso de uso, verifiquen la compatibilidad o conecten cargas útiles de la interfaz de usuario de la herramienta a la interfaz de usuario del asistente o a una pila de chat/tiempo de ejecución existente.
