Trova, installa, configura e integra i componenti dell'interfaccia utente dello strumento nelle app React utilizzando voci di registro shadcn, controlli di compatibilità, cablaggio di runtime con impalcature e flussi di lavoro per la risoluzione dei problemi. Da utilizzare quando gli sviluppatori chiedono di aggiungere uno o più componenti dell'interfaccia utente dello strumento, scegliere componenti per un caso d'uso, verificare la compatibilità o collegare i payload dell'interfaccia utente dello strumento all'interfaccia utente dell'assistente o a uno stack di chat/runtime esistente.
