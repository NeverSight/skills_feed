차트, 대시보드, 인포그래픽을 포함한 프로덕션 품질의 데이터 시각화를 만듭니다. 사용자가 데이터 시각화, 차트 생성, 대시보드 구축, 인포그래픽 생성, 통계 플롯 또는 데이터 세트를 시각적 표현으로 변환하도록 요청할 때 사용합니다. React/Recharts 아티팩트, 정적 이미지(Python을 통한 PNG/PDF) 및 대화형 HTML을 지원합니다. 트리거에는 "이 데이터 시각화", "차트 만들기", "대시보드 작성", "그래프 만들기", "이것을 플롯", "인포그래픽" 또는 데이터를 시각적으로 표현하기 위한 모든 요청이 포함됩니다.
