IBM の Carbon Design System を使用して UI を構築します。ユーザーが Carbon スタイルのインターフェイス、IBM スタイルのダッシュボード、Carbon の規則に従ったエンタープライズ UI を要求する場合、または Carbon、IBM デザイン、または @carbon/react に明示的に言及する場合に使用します。コンポーネントの使用法、デザイン トークン (色、タイポグラフィ、間隔、モーション)、テーマ (白、グレー 10、グレー 90、グレー 100)、グリッド レイアウト、アクセシビリティについて説明します。アーティファクト/HTML 出力 (CDN ベース) と完全な React プロジェクト出力 (@carbon/react) の両方をサポートします。トリガーには、「Carbon」、「IBM デザイン システム」、「エンタープライズ ダッシュボード」、「@carbon/react」、「カーボン コンポーネント」、または IBM スタイルのプロフェッショナル インターフェイスのリクエストが含まれます。
