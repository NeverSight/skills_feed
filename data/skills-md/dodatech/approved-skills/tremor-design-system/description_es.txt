Cree paneles, interfaces de análisis y UI ricas en datos utilizando el sistema de diseño Tremor (React + Tailwind CSS + Recharts). Úselo cuando el usuario solicite crear componentes de panel, tarjetas de KPI, gráficos, tablas de datos, páginas de análisis, interfaces de monitoreo o cualquier interfaz de usuario de visualización de datos que deba usar Tremor. Los desencadenantes incluyen menciones de "Tremor", "tremor.so", "@tremor/react", solicitudes de interfaces de usuario del panel con gráficos y tablas, o cuando el proyecto del usuario ya utiliza componentes Tremor. Admite las versiones Tremor Raw (copiar y pegar, tremor.so) y Tremor NPM (@tremor/react). NO lo use para trabajos frontend generales no relacionados con paneles o visualización de datos, o cuando el usuario solicite explícitamente una biblioteca de componentes diferente.
