使用 Tremor 设计系统（React + Tailwind CSS + Recharts）构建仪表板、分析界面和数据丰富的 UI。当用户要求创建仪表板组件、KPI 卡、图表、数据表、分析页面、监控界面或任何应使用 Tremor 的数据可视化 UI 时使用。触发器包括提及“Tremor”、“tremor.so”、“@tremor/react”、对带有图表和表格的仪表板 UI 的请求，或者当用户的项目已经使用 Tremor 组件时。支持 Tremor Raw（复制粘贴，tremor.so）和 Tremor NPM（@tremor/react）版本。不要用于与仪表板或数据可视化无关的一般前端工作，或者当用户明确请求不同的组件库时。
