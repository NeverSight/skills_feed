Tremor 디자인 시스템(React + Tailwind CSS + Recharts)을 사용하여 대시보드, 분석 인터페이스 및 데이터가 풍부한 UI를 구축하세요. 사용자가 Tremor를 사용해야 하는 대시보드 구성 요소, KPI 카드, 차트, 데이터 테이블, 분석 페이지, 모니터링 인터페이스 또는 데이터 시각화 UI를 생성하도록 요청할 때 사용합니다. 트리거에는 "Tremor", "tremor.so", "@tremor/react"에 대한 언급, 차트와 테이블이 포함된 대시보드 UI에 대한 요청 또는 사용자 프로젝트에서 이미 Tremor 구성 요소를 사용하는 경우가 포함됩니다. Tremor Raw(복사하여 붙여넣기, tremor.so) 및 Tremor NPM(@tremor/react) 버전을 모두 지원합니다. 대시보드나 데이터 시각화와 관련되지 않은 일반 프런트엔드 작업이나 사용자가 명시적으로 다른 구성 요소 라이브러리를 요청하는 경우에는 사용하지 마세요.
