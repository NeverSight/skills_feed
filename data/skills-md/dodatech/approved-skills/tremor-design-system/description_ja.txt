Tremor デザイン システム (React + Tailwind CSS + Recharts) を使用して、ダッシュボード、分析インターフェイス、データ豊富な UI を構築します。ユーザーがダッシュボード コンポーネント、KPI カード、グラフ、データ テーブル、分析ページ、監視インターフェイス、または Tremor を使用する必要があるデータ視覚化 UI の作成を要求する場合に使用します。トリガーには、「Tremor」、「tremor.so」、「@tremor/react」の言及、グラフや表を含むダッシュボード UI のリクエスト、またはユーザーのプロジェクトが既に Tremor コンポーネントを使用している場合が含まれます。 Tremor Raw (コピーアンドペースト、tremor.so) バージョンと Tremor NPM (@tremor/react) バージョンの両方をサポートします。ダッシュボードやデータ視覚化に関係のない一般的なフロントエンド作業、またはユーザーが別のコンポーネント ライブラリを明示的に要求する場合には使用しないでください。
