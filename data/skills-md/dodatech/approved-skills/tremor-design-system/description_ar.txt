أنشئ لوحات معلومات وواجهات تحليلية وواجهات مستخدم غنية بالبيانات باستخدام نظام تصميم Tremor (React + Tailwind CSS + Recharts). يُستخدم عندما يطلب المستخدم إنشاء مكونات لوحة المعلومات، أو بطاقات مؤشرات الأداء الرئيسية، أو المخططات، أو جداول البيانات، أو صفحات التحليلات، أو واجهات المراقبة، أو أي واجهة مستخدم لتصور البيانات يجب أن تستخدم Tremor. تتضمن المشغلات إشارات إلى "Tremor"، أو "tremor.so"، أو "@tremor/react"، أو طلبات واجهات مستخدم لوحة المعلومات مع المخططات والجداول، أو عندما يستخدم مشروع المستخدم مكونات Tremor بالفعل. يدعم كلاً من إصدارات Tremor Raw (النسخ واللصق، tremor.so) وإصدارات Tremor NPM (@tremor/react). لا تستخدمه في أعمال الواجهة الأمامية العامة التي لا تتعلق بلوحات المعلومات أو تصور البيانات، أو عندما يطلب المستخدم صراحةً مكتبة مكونات مختلفة.
