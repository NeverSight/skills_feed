Создавайте информационные панели, аналитические интерфейсы и насыщенные данными пользовательские интерфейсы с помощью системы дизайна Tremor (React + Tailwind CSS + Recharts). Используйте, когда пользователь запрашивает создание компонентов информационной панели, карточек KPI, диаграмм, таблиц данных, страниц аналитики, интерфейсов мониторинга или любого пользовательского интерфейса визуализации данных, который должен использовать Tremor. Триггеры включают упоминания «Tremor», «tremor.so», «@tremor/react», запросы на пользовательский интерфейс информационной панели с диаграммами и таблицами или когда проект пользователя уже использует компоненты Tremor. Поддерживает версии Tremor Raw (копирование и вставка, tremor.so) и Tremor NPM (@tremor/react). НЕ используйте для общей работы с внешним интерфейсом, не связанной с панелями мониторинга или визуализацией данных, или когда пользователь явно запрашивает другую библиотеку компонентов.
