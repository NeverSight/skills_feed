使用 Tremor 設計系統（React + Tailwind CSS + Recharts）建立儀表板、分析介面和資料豐富的 UI。當使用者要求建立儀表板元件、KPI 卡、圖表、資料表、分析頁面、監控介面或任何應使用 Tremor 的資料視覺化 UI 時使用。觸發器包括提及「Tremor」、「tremor.so」、「@tremor/react」、對帶有圖表和表格的儀表板 UI 的請求，或當使用者的項目已經使用 Tremor 元件時。支援 Tremor Raw（複製貼上，tremor.so）和 Tremor NPM（@tremor/react）版本。不要用於與儀表板或資料視覺化無關的一般前端工作，或當使用者明確要求不同的元件庫時。
