Référence complète du langage de création de modèles Liquid, y compris la syntaxe, les filtres, les objets, le flux de contrôle, les boucles et les conditions pour les thèmes Shopify. À utiliser lorsque vous travaillez avec des fichiers .liquid, créez des modèles de thème, implémentez du contenu dynamique, déboguez du code Liquid, travaillez avec des sections et des extraits de code ou restituez des données de produit/collection/panier dans les magasins Shopify.
