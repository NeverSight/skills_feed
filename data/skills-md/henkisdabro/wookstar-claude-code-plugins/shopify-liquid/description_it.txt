Riferimento completo al linguaggio dei modelli Liquid che include sintassi, filtri, oggetti, flusso di controllo, loop e condizionali per i temi Shopify. Da utilizzare quando si lavora con file .liquid, si creano modelli di temi, si implementa contenuto dinamico, si esegue il debug del codice Liquid, si lavora con sezioni e snippet o si esegue il rendering di dati di prodotto/collezione/carrello nei negozi Shopify.
