Einfügen von CSS oder JavaScript in Webseiten, Ändern des Website-Verhaltens, Erstellen von Browsererweiterungen,

Erkennen von URL-Änderungen in SPAs oder Speichern dauerhafter Benutzereinstellungen.

gängige Muster (DOM-Mutation, Erkennung von URL-Änderungen, Warten auf Elemente), Sicherheits-Sandboxing,
und browserübergreifende Kompatibilität (Chrome, Firefox, Edge).
