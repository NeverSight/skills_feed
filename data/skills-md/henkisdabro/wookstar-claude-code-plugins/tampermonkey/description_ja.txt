ブラウザの自動化、ページの変更、Web の機能強化のための Tampermonkey ユーザー スクリプトを作成します。
次の場合に使用します: ブラウザ スクリプトの作成、greasemonkey スクリプトの作成、ユーザー インタラクションの自動化、
Web ページへの CSS または JavaScript の挿入、Web サイトの動作の変更、ブラウザ拡張機能の構築、
不要なページ要素の非表示、フォーム自動入力の追加、Web サイトデータのスクレイピング、リクエストの傍受、
SPA での URL 変更の検出、または永続的なユーザー設定の保存。
ユーザースクリプトヘッダー (@match、@grant、@require)、同期および非同期 GM_* API 関数、
一般的なパターン (DOM の変更、URL 変更の検出、要素の待機)、セキュリティ サンドボックス、
クロスブラウザー互換性 (Chrome、Firefox、Edge)。
