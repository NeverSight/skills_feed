Напишите пользовательские сценарии Tampermonkey для автоматизации браузера, изменения страниц и улучшения веб-страниц.
Используйте при: создании скриптов браузера, написании скриптов Greasemonkey, автоматизации взаимодействия с пользователем,
внедрение CSS или JavaScript в веб-страницы, изменение поведения веб-сайта, создание расширений для браузера,
скрытие нежелательных элементов страницы, добавление автозаполнения форм, очистка данных сайта, перехват запросов,
обнаружение изменений URL-адресов в SPA или сохранение постоянных пользовательских настроек.
Охватывает заголовки пользовательских скриптов (@match, @grant, @require), синхронные и асинхронные функции API GM_*,
общие шаблоны (мутация DOM, обнаружение изменения URL-адреса, ожидание элемента), изолированная программная среда безопасности,
и кроссбраузерная совместимость (Chrome, Firefox, Edge).
