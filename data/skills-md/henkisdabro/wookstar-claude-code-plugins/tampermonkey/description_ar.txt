اكتب نصوص مستخدم Tampermonkey لأتمتة المتصفح وتعديل الصفحة وتحسين الويب.
يُستخدم عندما: إنشاء البرامج النصية للمتصفح، وكتابة البرامج النصية لـ Greasemonkey، وأتمتة تفاعلات المستخدم،
إدخال CSS أو JavaScript في صفحات الويب، وتعديل سلوك موقع الويب، وإنشاء ملحقات المتصفح،
إخفاء عناصر الصفحة غير المرغوب فيها، وإضافة الملء التلقائي للنماذج، واستخراج بيانات موقع الويب، واعتراض الطلبات،
اكتشاف تغييرات URL في SPA، أو تخزين تفضيلات المستخدم المستمرة.
يغطي رؤوس نص المستخدم (@match، @grant، @require)، ووظائف GM_* API المتزامنة وغير المتزامنة،
الأنماط الشائعة (طفرة DOM، اكتشاف تغيير عنوان URL، انتظار العناصر)، وضع الحماية للأمان،
والتوافق عبر المتصفحات (Chrome وFirefox وEdge).
