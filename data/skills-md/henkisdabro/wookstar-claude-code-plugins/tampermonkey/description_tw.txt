編寫 Tampermonkey 使用者腳本以實現瀏覽器自動化、頁面修改和 Web 增強。
使用場合：建立瀏覽器腳本、編寫greasemonkey腳本、自動化使用者互動、
將 CSS 或 JavaScript 注入網頁、修改網站行為、建立瀏覽器擴充功能、
隱藏不需要的頁面元素，添加表單自動填充，抓取網站數據，攔截請求，
偵測 SPA 中的 URL 更改，或儲存持久的使用者首選項。
涵蓋使用者腳本標頭（@match、@grant、@require）、同步和非同步 GM_* API 函數，
常見模式（DOM 突變、URL 變更偵測、元素等待）、安全沙箱、
和跨瀏覽器相容性（Chrome、Firefox、Edge）。
