Комплексное руководство по разработке сценариев Google Apps, охватывающее все встроенные службы (SpreadsheetApp, DocumentApp, GmailApp, DriveApp, CalendarApp, FormApp, SlidesApp), триггеры, авторизацию, обработку ошибок и оптимизацию производительности. Используйте при автоматизации операций Google Таблиц, создании документов Google, управлении Gmail/электронной почтой, работе с файлами Google Диска, автоматизации событий Календаря, реализации триггеров (на основе времени, на основе событий), создании пользовательских функций, создании надстроек, обработке областей OAuth, оптимизации производительности скриптов приложений, работе с UrlFetchApp для вызовов API, использовании PropertiesService для постоянного хранения или реализации CacheService для временных данных. Охватывает пакетные операции, восстановление ошибок и среду выполнения JavaScript ES6+.
