모든 내장 서비스(SpreadsheetApp, DocumentApp, GmailApp, DriveApp, CalendarApp, FormApp, SlidesApp), 트리거, 승인, 오류 처리 및 성능 최적화를 다루는 Google Apps Script 개발에 대한 종합 가이드입니다. Google Sheets 작업 자동화, Google Docs 생성, Gmail/이메일 관리, Google Drive 파일 작업, 캘린더 이벤트 자동화, 트리거 구현(시간 기반, 이벤트 기반), 맞춤 기능 구축, 추가 기능 생성, OAuth 범위 처리, Apps Script 성능 최적화, API 호출을 위한 UrlFetchApp 작업, 영구 저장을 위한 PropertiesService 사용, 임시 데이터를 위한 CacheService 구현 등을 수행할 때 사용합니다. 일괄 작업, 오류 복구 및 JavaScript ES6+ 런타임을 다룹니다.
