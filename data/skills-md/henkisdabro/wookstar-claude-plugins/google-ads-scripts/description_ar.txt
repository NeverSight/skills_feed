إرشادات الخبراء لتطوير البرنامج النصي لإعلانات Google، بما في ذلك AdsApp API وإدارة الحملات والمجموعات الإعلانية والكلمات الرئيسية وإستراتيجيات عروض الأسعار وإعداد تقارير الأداء وإدارة الميزانية والقواعد التلقائية وأنماط التحسين. يُستخدم عند أتمتة حملات إعلانات Google، أو إدارة الكلمات الرئيسية وعروض الأسعار، أو إنشاء تقارير الأداء، أو تنفيذ القواعد التلقائية، أو تحسين الإنفاق الإعلاني، أو العمل مع ميزانيات الحملات، أو مراقبة نقاط الجودة، أو تتبع التحويلات، أو إيقاف الكلمات الرئيسية منخفضة الأداء مؤقتًا، أو تعديل عروض الأسعار استنادًا إلى عائد النفقات الإعلانية، أو إنشاء نصوص برمجية تلقائية لـ إعلانات Google. يغطي عمليات الحملة، واستهداف الكلمات الرئيسية، وتحسين عروض الأسعار، وتتبع التحويل، ومعالجة الأخطاء، والأتمتة المستندة إلى JavaScript في محرر إعلانات Google.
