دليل تطوير السمات الكامل بما في ذلك بنية الملف وقوالب JSON والأقسام والمقتطفات ومخطط الإعدادات وبنية Online Store 2.0. يُستخدم عند إنشاء سمات Shopify، أو تنظيم ملفات السمات، أو إنشاء الأقسام والكتل، أو العمل مع ملفات قوالب .json، أو تكوين settings_schema.json، أو إنشاء مقتطفات، أو تنفيذ ميزات تخصيص السمات.
