Guide complet pour le développement de Google Apps Script couvrant tous les services intégrés (SpreadsheetApp, DocumentApp, GmailApp, DriveApp, CalendarApp, FormApp, SlidesApp), les déclencheurs, l'autorisation, la gestion des erreurs et l'optimisation des performances. À utiliser pour automatiser les opérations Google Sheets, créer Google Docs, gérer Gmail/e-mail, travailler avec des fichiers Google Drive, automatiser des événements de calendrier, implémenter des déclencheurs (basés sur le temps, basés sur des événements), créer des fonctions personnalisées, créer des modules complémentaires, gérer les étendues OAuth, optimiser les performances d'Apps Script, travailler avec UrlFetchApp pour les appels d'API, utiliser PropertiesService pour le stockage persistant ou implémenter CacheService pour les données temporaires. Couvre les opérations par lots, la récupération d’erreurs et le runtime JavaScript ES6+.
