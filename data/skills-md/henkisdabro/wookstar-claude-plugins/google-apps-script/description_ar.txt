دليل شامل لتطوير Google Apps Script يغطي جميع الخدمات المضمنة (SpreadsheetApp، DocumentApp، GmailApp، DriveApp، CalendarApp، FormApp، SlidesApp)، المشغلات، والترخيص، ومعالجة الأخطاء، وتحسين الأداء. يُستخدم عند أتمتة عمليات جداول بيانات Google، أو إنشاء مستندات Google، أو إدارة Gmail/البريد الإلكتروني، أو العمل مع ملفات Google Drive، أو أتمتة أحداث التقويم، أو تنفيذ المشغلات (المستندة إلى الوقت، أو المستندة إلى الحدث)، أو إنشاء وظائف مخصصة، أو إنشاء الوظائف الإضافية، أو التعامل مع نطاقات OAuth، أو تحسين أداء Apps Script، أو العمل مع UrlFetchApp لاستدعاءات واجهة برمجة التطبيقات، أو استخدام PropertiesService للتخزين المستمر، أو تنفيذ CacheService للبيانات المؤقتة. يغطي العمليات المجمعة، واسترداد الأخطاء، ووقت تشغيل JavaScript ES6+.
