Intégrez et utilisez argon-router dans les applications Web React avec Effector. À utiliser lorsque les tâches impliquent la création d'itinéraires et de routeurs, le câblage d'adaptateurs RouterControls/historique, la composition d'itinéraires avec chainRoute/group/createVirtualRoute, le rendu de vues avec RouterProvider/createRoutesView/Outlet, la création de liens avec Link/useLink et la gestion de l'état des requêtes d'URL avec trackQuery et @argon-router/paths.
