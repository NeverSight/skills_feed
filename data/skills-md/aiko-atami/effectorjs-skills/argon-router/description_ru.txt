Интегрируйте и используйте argon-router в веб-приложениях React с Effector. Используйте, когда задачи включают создание маршрутов и маршрутизаторов, подключение RouterControls/адаптеров истории, составление маршрутов с помощью ChainRoute/group/createVirtualRoute, рендеринг представлений с помощью RouterProvider/createRoutesView/Outlet, создание ссылок с помощью Link/useLink и управление состоянием URL-запроса с помощью trackQuery и @argon-router/paths.
