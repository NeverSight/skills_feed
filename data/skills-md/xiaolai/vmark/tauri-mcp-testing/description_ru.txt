Эксперт по E2E-тестированию приложений Tauri с использованием сервера Tauri MCP. Используйте при тестировании запуска приложений Tauri — управление сеансами, взаимодействие с веб-просмотром, проверка IPC, захват снимков экрана и отладка. ВСЕГДА используйте инструменты tauri_*, НИКОГДА не используйте Chrome DevTools MCP для приложений Tauri.
