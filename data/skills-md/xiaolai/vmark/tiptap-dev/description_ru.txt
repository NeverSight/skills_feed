Экспертное руководство по созданию редакторов форматированного текста с помощью Tiptap — безголового, независимого от платформы редактора, созданного на основе ProseMirror. Используйте при создании пользовательских узлов, меток или расширений для Tiptap, реализации правил ввода или правил вставки, работе с API команд Tiptap, построении интеграции React с помощью useEditor, расширении существующих расширений или создании пользовательских представлений узлов.
