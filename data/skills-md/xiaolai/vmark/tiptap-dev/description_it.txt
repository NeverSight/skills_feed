Guida esperta per la creazione di editor di testo RTF con Tiptap: un editor headless e indipendente dal framework basato su ProseMirror. Da utilizzare quando si creano nodi, contrassegni o estensioni personalizzati per Tiptap, si implementano regole di input o si incollano regole, si lavora con l'API dei comandi Tiptap, si creano integrazioni React con useEditor, si estendono estensioni esistenti o si creano visualizzazioni di nodi personalizzate.
