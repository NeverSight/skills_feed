Преобразуйте модели трансформеров HuggingFace в формат ONNX для вывода в браузере с помощью Transformers.js и WebGPU. Используйте при наличии ссылки на модель HuggingFace для преобразования в ONNX, при настройке оптимального-cli для экспорта ONNX, при квантовании моделей (fp16, q8, q4) для веб-развертывания, при настройке Transformers.js с ускорением WebGPU или при устранении ошибок преобразования ONNX. Запускается при упоминании преобразования ONNX, Transformers.js, вывода WebGPU, оптимального экспорта, квантования модели для браузера или запуска моделей машинного обучения в браузере.
