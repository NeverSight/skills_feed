Convierta los modelos de transformadores HuggingFace al formato ONNX para la inferencia del navegador con Transformers.js y WebGPU. Úselo cuando se le proporcione un enlace de modelo de HuggingFace para convertir a ONNX, al configurar Optimum-cli para la exportación de ONNX, al cuantificar modelos (fp16, q8, q4) para la implementación web, al configurar Transformers.js con aceleración WebGPU o al solucionar problemas de errores de conversión de ONNX. Se activa cuando se menciona la conversión ONNX, Transformers.js, inferencia WebGPU, exportación óptima, cuantificación de modelos para el navegador o ejecución de modelos ML en el navegador.
