Estrategia de búsqueda de verificación estándar de múltiples fuentes para investigaciones de complejidad moderada. Flujo de trabajo de 2 iteraciones con clasificación de fuentes, identificación de consenso y transparencia de citas. Úselo para comparaciones de funciones, temas de complejidad moderada y verificación de hechos. Palabras clave: comparar, diferencias, características, fact-check, verificar, qué son.
