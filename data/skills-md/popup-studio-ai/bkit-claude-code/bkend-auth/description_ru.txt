Навыки эксперта по аутентификации и безопасности bkend.ai.
Охватывает регистрацию/вход по электронной почте, вход в социальные сети (Google, GitHub), волшебную ссылку,
Токены JWT (доступ 1 час, обновление 7 дней), управление сеансами, RBAC (администратор/пользователь/собственный/гость),
Политики RLS, управление паролями и жизненный цикл учетной записи.
