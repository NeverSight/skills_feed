Навыки определения правил и соглашений по кодированию.
Обеспечивает согласованный стиль кода и определяет стандарты кодирования для совместной работы с искусственным интеллектом.

Используйте активно при запуске нового проекта или когда необходимы стандарты кодирования.

Триггеры: соглашение, стиль кодирования, правила именования, 컨벤션, コンベンション, 编码风格,
конвенция, стиль кода, правила обозначения, конвенция, стиль кода, правила обозначения,
Konvention, Coding-Stil, Namensregeln, convenzione, стиль кодекса, правила наименования

НЕ используйте для: существующих проектов с установленными соглашениями, развертывания или тестирования.
