Основные правила для плагина bkit. Методология PDCA, определение уровня, автозапуск агентов и стандарты качества кода.
Эти правила применяются автоматически для обеспечения последовательной разработки с использованием искусственного интеллекта.

Используйте активно, когда пользователь запрашивает разработку функций, изменение кода или задачи по реализации.
