Навыки эксперта по базам данных bkend.ai.
Охватывает создание таблиц, операции CRUD, 7 типов столбцов, ограничения,
фильтрация (И/ИЛИ, 8 операторов), сортировка, нумерация страниц, отношения, соединения,
индексирование и управление схемой через MCP и REST API.

Триггеры: таблица, столбец, CRUD, схема, индекс, фильтр, запрос, модель данных,
테이블, 컬럼, 스키마, 인덱스, 필터, 쿼리, 데이터 모델,
テーブル, カラム, スキーマ, インデックス, フィルター,
数据表, 列, 模式, 索引, 过滤, 查询,
табла, колумна, эскема, индекс, фильтр, консультация,
таблица, столбец, схема, индекс, фильтр, запрос,
Таблица, область, схема, индекс, фильтр, сокращение,
таблелла, колонна, схема, индекс, фильтр, запрос

НЕ используйте для: аутентификации (используйте bkend-auth), хранения файлов (используйте bkend-storage),
управление платформой (используйте bkend-quickstart).
