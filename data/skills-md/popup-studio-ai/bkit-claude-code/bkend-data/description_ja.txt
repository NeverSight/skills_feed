bkend.ai データベースのエキスパート スキル。
テーブルの作成、CRUD 操作、7 つの列タイプ、制約、
フィルタリング (AND/OR、8 演算子)、ソート、ページネーション、リレーション、結合、
インデックス作成、MCP および REST API を介したスキーマ管理。
