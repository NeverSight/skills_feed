bkend.ai مهارة خبير قاعدة البيانات.
يغطي إنشاء الجدول، وعمليات CRUD، وأنواع الأعمدة السبعة، والقيود،
التصفية (و/أو، 8 عوامل تشغيل)، والفرز، وترقيم الصفحات، والعلاقات، والصلات،
الفهرسة وإدارة المخطط عبر MCP وREST API.

테블، 컬럼، 스키마، 인덱스، 필터، 쿼리، 데이터 모델،
