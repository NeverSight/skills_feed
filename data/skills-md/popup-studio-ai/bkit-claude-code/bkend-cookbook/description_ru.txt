bkend.ai практические руководства по проектам и руководство по устранению неполадок.
Охватывает 10 руководств по одному проекту (todo to SaaS) и 4 полных проекта руководства.
(блог, приложение-рецепт, торговый центр, социальная сеть) с пошаговой реализацией.
Также включает в себя решения распространенных ошибок и часто задаваемые вопросы.

Триггеры: кулинарная книга, руководство, пример проекта, приложение todo, приложение для блога, торговый центр,
쿡북, 튜토리얼, 예제, 블로그, 쇼핑몰, 투두, 프로젝트 만들기,
クックブック, チュートリアル, 例題, ブログ, ショッピング,
食谱, 教程, 示例, 博客, 商城,
библиотека де кочина, учебник, ejemplo, блог, Tienda,
livre de Recettes, Tutoriel, пример, блог, бутик,
Kochbuch, Учебное пособие, Beispiel, Блог, Магазин,
рисеттарио, руководство, esempio, блог, негозио

НЕ используйте для: справочных сведений об API (используйте bkend-data/auth/storage),
управление платформой (используйте bkend-quickstart).
