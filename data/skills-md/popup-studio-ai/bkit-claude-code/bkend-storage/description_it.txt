Abilità dell'esperto di archiviazione di file bkend.ai.
Copre il caricamento di file singoli/multipli/in più parti tramite URL prefirmato,
download di file (CDN vs Prefirmato), 4 livelli di visibilità (pubblico/privato/protetto/condiviso),
gestione dei bucket e metadati dei file.

Trigger: caricamento file, download, prefirmato, bucket, archiviazione, CDN, immagine,
파일 업로드, 다운로드, 버킷, 스토리지, 이미지,
ファイルアップロード, ダウンロード, バケット, ストレージ,
文件上传, 下载, 存储桶, 存储,
carga de archivos, descarga, almacenamiento, cubo,
telechargement, televersement, stockage, seau,
Datei-Carica, Scarica, Speicher, Bucket,
caricamento file, scaricamento, archiviazione, bucket

NON utilizzare per: operazioni di database (utilizzare bkend-data), autenticazione (utilizzare bkend-auth).
