Клод Код обучения и педагогических навыков.

Работает в любом проекте и на любом языке.

Начните обучение/настройку с помощью «learn» или «setup».

Используйте активно, когда пользователь впервые работает с Claude Code, спрашивает о конфигурации,
или хочет улучшить настройку Claude Code.
