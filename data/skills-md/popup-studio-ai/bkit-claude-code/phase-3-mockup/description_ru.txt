Навык создания макетов с учетом тенденций UI/UX без дизайнера.
Разрабатывает прототипы HTML/CSS/JS, которые можно преобразовать в компоненты Next.js.

Используйте активно, когда пользователь хочет проверить UI/UX перед реализацией.
