Guía de conceptos básicos e incorporación de la plataforma bkend.ai.
Cubre la configuración de MCP, jerarquía de recursos (Organización->Proyecto->Entorno),
Modelo Tenant vs User y creación del primer proyecto.

Utilícelo de forma proactiva cuando el usuario sea nuevo en bkend o pregunte sobre la configuración inicial.

Activadores: configuración de bkend, primer proyecto, inicio de bkend, conexión MCP,
bkend 시작, 처음, 설정, MCP 연결, 프로젝트 생성,
bkend始め方, 初期設定, MCP接続, bkend入门, 初始设置, MCP连接,
configuracion bkend, primer proyecto, configuración bkend, premier proyecto,
bkend Einrichtung, primer proyecto, configuración bkend, primer proyecto

NO lo use para: operaciones específicas de base de datos/autenticación/almacenamiento (use habilidades específicas del dominio),
infraestructura empresarial (use infraestructura de arquitectura).
