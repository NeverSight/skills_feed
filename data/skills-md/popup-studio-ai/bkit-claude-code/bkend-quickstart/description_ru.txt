Руководство по адаптации платформы bkend.ai и основным концепциям.
Охватывает настройку MCP, иерархию ресурсов (Организация->Проект->Среда),
Модель арендатора и пользователя и создание первого проекта.

Используйте заранее, когда пользователь впервые работает с bkend или спрашивает о первоначальной настройке.

Триггеры: настройка bkend, первый проект, запуск bkend, подключение MCP,
bkend 시작, 처음, 설정, MCP 연결, 프로젝트 생성,
bkend始め方, 初期設定, MCP接続, bkend入门, 初始设置, MCP连接,
конфигурация bkend, начальный проект, конфигурация bkend, премьер-проект,
bkend Einrichtung, erstes Projekt, конфигурация bkend, первый проект

НЕ используйте для: определенных операций с базой данных/аутентификации/хранения (используйте навыки, специфичные для предметной области),
инфраструктура предприятия (используйте инфраархитектор).
