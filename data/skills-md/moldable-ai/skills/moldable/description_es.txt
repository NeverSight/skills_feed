Guía completa para crear aplicaciones moldeables. Utilice esta habilidad al crear nuevas aplicaciones con scaffoldApp, modificar aplicaciones existentes, implementar almacenamiento compatible con el espacio de trabajo, integrarse con el escritorio Moldable a través de las API postMessage (moldable:show-in-folder, moldable:set-chat-input, moldable:set-chat-instructions, moldable:save-file), configurar espacios de trabajo, administrar habilidades/MCP o solucionar problemas de aplicaciones. Esencial para cualquier tarea de desarrollo de aplicaciones Moldable.
