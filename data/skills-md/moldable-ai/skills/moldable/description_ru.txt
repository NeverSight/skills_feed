Полное руководство по созданию приложений Moldable. Используйте этот навык при создании новых приложений с помощью scaffoldApp, изменении существующих приложений, внедрении хранилища с поддержкой рабочего пространства, интеграции с рабочим столом Moldable через API-интерфейсы postMessage (moldable:show-in-folder, formable:set-chat-input, formable:set-chat-instructions, formable:save-file), настройке рабочих пространств, управлении навыками/MCP или устранении неполадок приложений. Незаменим для любой задачи разработки приложений Moldable.
