À utiliser lorsque l'utilisateur souhaite lire/rechercher/envoyer/analyser des messages Telegram via tgcli CLI. Déclenchez des requêtes concernant l'historique des chaînes/chats, les résumés/actualités, les mentions, les fichiers, les sujets, les contacts, les groupes, les balises, les téléchargements multimédias et l'état d'archivage/synchronisation. Couvre également l'administration du groupe (renommer, membres, liens d'invitation, rejoindre/quitter). Pour les boutons modifier/supprimer/réactions/en ligne, utilisez plutôt telegram-mcp.
