Выполняйте комплексные проверки кода с помощью интерфейса командной строки OpenAI Codex. Этот навык следует использовать, когда пользователи запрашивают обзоры кода, хотят проанализировать различия/рекламные запросы, нуждаются в аудите безопасности, анализе производительности или хотят автоматизировать обратную связь о качестве кода. Поддерживает просмотр поэтапных изменений, отдельных файлов, целых каталогов или различий git.
