Создавайте веб-интерфейсы для агентов с помощью аутентификации на основе ATXP, следуя шаблону ClawDirect. Используйте этот навык при создании веб-сайтов, с которыми агенты искусственного интеллекта взаимодействуют с помощью инструментов MCP, реализации аутентификации агента на основе файлов cookie или создания навыков агента для веб-приложений. Предоставляет шаблоны с использованием @longrun/turtle, Express, SQLite и ATXP.
