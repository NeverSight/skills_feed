Примените общий формат журнала изменений с расширением раздела «Невыпущенные». Проверяет журнал изменений коммитов, генерирует его из истории git и обеспечивает документирование всех объединенных запросов на добавление. Используйте при настройке рабочих процессов журнала изменений или модернизации существующих репозиториев.
