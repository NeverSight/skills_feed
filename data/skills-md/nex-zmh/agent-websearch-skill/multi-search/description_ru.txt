Интеллектуальный многодвижочный поиск, который автоматически определяет сетевое окружение и переключается по приоритету: DuckDuckGo → Tavily → Bing API → Парсер Bing. Поддерживает автоматическое управление квотами и сетевое кэширование. Вызов, когда пользователю нужен веб-поиск с автоматическим выбором механизма и адаптацией к сети.
