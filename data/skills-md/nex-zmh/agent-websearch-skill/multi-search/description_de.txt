Intelligente Multi-Engine-Suche, die die Netzwerkumgebung automatisch erkennt und nach Priorität wechselt: DuckDuckGo → Tavily → Bing API → Bing Scraper. Unterstützt automatische Kontingentverwaltung und Netzwerk-Caching. Wird aufgerufen, wenn der Benutzer eine Websuche mit automatischer Suchmaschinenauswahl und Netzwerkanpassung benötigt.
