Intégrez des évaluateurs LLM-as-a-Judge Scorables dans des applications avec des interactions LLM. À utiliser lorsque les utilisateurs souhaitent ajouter une évaluation, des garde-fous ou une surveillance de la qualité à leurs applications basées sur LLM. À utiliser également lorsque les utilisateurs mentionnent Scorable, les juges, l'évaluation LLM ou les applications de sauvegarde.
