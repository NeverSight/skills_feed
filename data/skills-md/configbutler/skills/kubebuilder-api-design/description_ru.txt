Используйте при создании или изменении типов API Kubebuilder Go и маркеров +kubebuilder, чтобы генерация (`makegenerate`/`makefiles`) выдавала предполагаемую схему CRD (проверка/значения по умолчанию, обязательная/необязательная, семантика списка, столбцы принтера, подресурсы) или при запросе, какой маркер/тип использовать.
