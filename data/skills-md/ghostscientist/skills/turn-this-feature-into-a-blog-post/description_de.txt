Generiert einen technischen Blog-Beitrag aus der Code-Implementierung. Verwenden Sie diese Option, wenn Sie gebeten werden, einen Blog-Beitrag über eine Funktion zu schreiben, eine Implementierung für einen Blog zu erklären, Code als Blog-Artikel zu dokumentieren oder technische Inhalte aus Quellcode zu erstellen. Wird durch Formulierungen wie „einen Blog-Beitrag darüber schreiben“, „in einen Blog verwandeln“, „einen technischen Artikel erstellen“ oder „erklären Sie dies für einen Blog“ ausgelöst.
