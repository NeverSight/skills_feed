Analyse les projets de plate-forme iOS/macOS/Apple existants pour créer un plan complet et par étapes pour créer un compagnon watchOS ou une application autonome. À utiliser lorsque les utilisateurs souhaitent ajouter la prise en charge de watchOS à une application de plate-forme Apple existante, créer une version d'application Watch de leur application iOS ou créer des fonctionnalités watchOS. La compétence digère l'architecture du projet, identifie les modèles, analyse la compatibilité des API, recherche la documentation watchOS actuelle et produit un plan de mise en œuvre détaillé avec des avertissements de disponibilité des API avant toute génération de code.
