既存の iOS/macOS/Apple プラットフォーム プロジェクトを分析して、watchOS コンパニオンまたはスタンドアロン アプリを構築するための包括的で段階的な計画を作成します。ユーザーが既存の Apple プラットフォーム アプリに watchOS サポートを追加する場合、iOS アプリの Watch アプリ バージョンを作成する場合、または watchOS 機能を構築する場合に使用します。このスキルは、プロジェクト アーキテクチャをダイジェストし、パターンを特定し、API 互換性を分析し、現在の watchOS ドキュメントを検索し、コード生成前に API の可用性に関する警告を含む詳細な実装計画を作成します。
