Анализирует существующие проекты платформ iOS/macOS/Apple для создания комплексного поэтапного плана создания сопутствующего или автономного приложения для watchOS. Используйте его, когда пользователи хотят добавить поддержку watchOS в существующее приложение платформы Apple, создать версию своего приложения iOS для Watch или создать функции watchOS. Навык анализирует архитектуру проекта, выявляет шаблоны, анализирует совместимость API, ищет текущую документацию watchOS и создает подробный план реализации с предупреждениями о доступности API перед генерацией кода.
