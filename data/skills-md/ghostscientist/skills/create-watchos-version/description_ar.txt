يقوم بتحليل مشاريع النظام الأساسي iOS/macOS/Apple الحالية لإنشاء خطة شاملة ومرحلية لإنشاء تطبيق مصاحب لـ watchOS أو تطبيق مستقل. يُستخدم عندما يرغب المستخدمون في إضافة دعم watchOS إلى تطبيق نظام Apple الأساسي الحالي، أو إنشاء إصدار تطبيق Watch من تطبيق iOS الخاص بهم، أو إنشاء ميزات watchOS. تستوعب المهارة بنية المشروع، وتحدد الأنماط، وتحلل توافق واجهة برمجة التطبيقات، وتبحث عن وثائق watchOS الحالية، وتنتج خطة تنفيذ مفصلة مع تحذيرات توفر واجهة برمجة التطبيقات قبل أي إنشاء تعليمات برمجية.
