Analiza proyectos existentes de plataformas iOS/macOS/Apple para crear un plan integral y por fases para crear una aplicación complementaria o independiente de watchOS. Úselo cuando los usuarios quieran agregar compatibilidad con watchOS a una aplicación de plataforma Apple existente, crear una versión de la aplicación Watch de su aplicación iOS o crear funciones de watchOS. La habilidad resume la arquitectura del proyecto, identifica patrones, analiza la compatibilidad de API, busca documentación actual de watchOS y produce un plan de implementación detallado con advertencias de disponibilidad de API antes de generar cualquier código.
