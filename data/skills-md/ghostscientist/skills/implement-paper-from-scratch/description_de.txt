Führt Sie Schritt für Schritt durch die Umsetzung einer Forschungsarbeit von Grund auf. Verwenden Sie es, wenn Sie aufgefordert werden, eine Arbeit umzusetzen, eine Arbeit zu kodieren, Forschungsergebnisse zu reproduzieren oder ein Modell aus einer Arbeit zu erstellen. Konzentriert sich auf den Aufbau von Verständnis durch Umsetzung mit Checkpoint-Fragen.
