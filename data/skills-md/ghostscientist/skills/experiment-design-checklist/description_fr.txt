Génère un plan d’expérience rigoureux à partir d’une hypothèse. À utiliser lorsqu'on vous demande de concevoir des expériences, de planifier des expériences, de créer une configuration expérimentale ou de découvrir comment tester une hypothèse de recherche. Couvre les contrôles, les lignes de base, les ablations, les mesures, les tests statistiques et les estimations de calcul.
