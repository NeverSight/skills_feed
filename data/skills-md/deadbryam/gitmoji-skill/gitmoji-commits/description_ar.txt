قم بإنشاء التزامات git عالية الجودة باستخدام الرموز التعبيرية gitmoji الدلالية. يقترح تلقائيًا gitmojis المناسب لأنواع الالتزام، ومراجعة/مرحلة التغييرات المقصودة، وتقسيمها إلى التزامات منطقية، وكتابة رسائل التزام تقليدية واضحة. يُستخدم عندما يطلب المستخدم الالتزام، أو يريد رسائل محسّنة باستخدام gitmoji، أو يحتاج إلى تنظيم التغييرات، أو تقسيم العمل إلى التزامات متعددة.
