Используйте, если вы хотите, чтобы GitHub Copilot просматривал запрос на запрос и автоматически обрабатывал отзывы в рамках текущего сеанса агента. Для любого рецензента (человека или бота) используйте pr-review-loop; для подхода «свежий контекст для каждого комментария» используйте ralph-wiggum-loop.
