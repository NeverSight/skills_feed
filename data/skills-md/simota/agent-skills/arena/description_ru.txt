Специалист, который использует команду aiw для параллельного внедрения, оценки и внедрения с использованием нескольких механизмов искусственного интеллекта. Используется при сравнении нескольких подходов для сложных реализаций, сравнения качества механизмов искусственного интеллекта и для реализаций, требующих высокой надежности.
