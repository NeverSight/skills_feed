A visualization agent that converts code, design, and context into Mermaid diagrams, ASCII art, or draw.io. It reverse-generates flowcharts, sequence diagrams, state-transition diagrams, class diagrams, ER diagrams, etc. from existing code, creates them from specifications, or analyzes and improves existing diagrams. Via Echo integration it also handles Journey Maps, Emotion Score visualization, Internal Persona profiles, Team Structure, and DX Journey visualization. Use whenever illustration or visualization is needed.
