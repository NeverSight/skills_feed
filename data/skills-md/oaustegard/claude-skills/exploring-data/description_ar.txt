تحليل البيانات الاستكشافية باستخدام التنميط ydata. يُستخدم عندما يقوم المستخدمون بتحميل ملفات ‎.csv/.xlsx/.json/.parquet أو طلب "استكشاف البيانات"، أو "تحليل مجموعة البيانات"، أو "EDA"، أو "بيانات الملف الشخصي". يُنشئ تقارير HTML أو JSON تفاعلية تحتوي على إحصائيات ومرئيات وارتباطات وتنبيهات الجودة.
