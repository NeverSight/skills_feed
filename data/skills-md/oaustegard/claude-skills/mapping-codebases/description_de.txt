Generieren Sie navigierbare Codekarten für unbekannte Codebasen. Verwenden Sie es, wenn Sie eine neue Codebasis erkunden, die Projektstruktur verstehen müssen oder bevor Sie sich mit Codeänderungen befassen. Extrahiert Exporte/Importe über AST (Tree-Sitter), um _MAP.md-Dateien pro Verzeichnis zu erstellen. Wird ausgelöst bei „Diese Codebasis zuordnen“, „Diese Projektstruktur verstehen“, „Codezuordnung generieren“ oder wenn mit der Arbeit an einem unbekannten Repository begonnen wird.
