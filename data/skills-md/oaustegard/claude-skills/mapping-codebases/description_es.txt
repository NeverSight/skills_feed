Genere mapas de código navegables para bases de código desconocidas. Úselo cuando explore una nueva base de código, necesite comprender la estructura del proyecto o antes de profundizar en las modificaciones del código. Extrae exportaciones/importaciones a través de AST (tree-sitter) para crear archivos _MAP.md por directorio. Activadores al "asignar esta base de código", "comprender la estructura de este proyecto", "generar mapa de código" o al comenzar a trabajar en un repositorio desconocido.
