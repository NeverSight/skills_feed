Générez des cartes de code navigables pour les bases de code inconnues. À utiliser lors de l'exploration d'une nouvelle base de code, lorsque vous avez besoin de comprendre la structure du projet ou avant de vous lancer dans des modifications de code. Extrait les exports/imports via AST (tree-sitter) pour créer des fichiers _MAP.md par répertoire. Déclencheurs sur « mapper cette base de code », « comprendre la structure de ce projet », « générer une carte de code » ou lors du démarrage de travaux sur un référentiel inconnu.
