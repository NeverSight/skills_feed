Compétences spécialisées en développement Preact pour les applications Web basées sur des normes avec une architecture native et une empreinte de dépendance minimale. À utiliser lors de la création de projets Preact, en particulier ceux impliquant la visualisation de données, les applications interactives, les applications monopage avec syntaxe HTM, l'intégration de composants Web, l'analyse de données CSV/JSON, les visualisations de shaders WebGL ou les solutions sans build avec importations CDN.
