Создавайте интерактивные визуализации данных, используя декларативную грамматику JSON Vega-Lite. Поддерживает более 20 типов диаграмм (столбчатые, линейные, точечные, гистограммы, коробчатые диаграммы, сгруппированные/сложные варианты и т. д.) с помощью шаблонов и программных конструкторов. Используйте, когда пользователи загружают данные для диаграмм, запрашивают определенные типы диаграмм или упоминают визуализации. Создает переносимые спецификации JSON со встроенными островками данных, которые работают с артефактами Claude и могут быть адаптированы для производства.
