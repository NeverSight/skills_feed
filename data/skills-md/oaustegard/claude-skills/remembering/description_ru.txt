Справочник по расширенным операциям с памятью. Основные шаблоны (загрузка профиля, простой вызов/запоминание) находятся в инструкциях проекта. Используйте этот навык для фоновой записи, управления версиями памяти, сложных запросов, крайних случаев, определения области сеанса, управления хранением, типобезопасных результатов, упреждающих подсказок по памяти и обнаружения доступа к GitHub.
