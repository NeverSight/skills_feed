Включает операции с репозиторием GitHub (чтение/запись/фиксация/PR) для чатов Claude.ai. Используйте, когда пользователи запрашивают фиксации GitHub, обновления репозитория, сохранение DEVLOG или управление состоянием между сеансами через ветки GitHub. Не требуется в Claude Code (имеет собственный доступ к git).
