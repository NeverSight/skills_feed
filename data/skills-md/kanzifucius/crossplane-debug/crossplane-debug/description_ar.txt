تصحيح تركيبات Crossplane ووظائفها (KCL، وقوالب Go، والتصحيح والتحويل، والجاهزية التلقائية)، والموارد المُدارة. يُستخدم عند استكشاف مشكلات عرض التركيبة وإصلاحها، أو أخطاء الوظائف، أو فشل إنشاء الموارد، أو مشكلات التبعية، أو مشكلات حالة المطالبة. يدعم تصحيح أخطاء المجموعة عن بعد عندما لا تكون ملفات التكوين متوفرة محليًا. يتم تشغيلها على الكلمات الرئيسية مثل "crossplane"، أو "composition"، أو "XR"، أو "claim"، أو "function-kcl"، أو "الموارد المُدارة"، أو عند تصحيح أخطاء موارد Kubernetes التي تم إنشاؤها بواسطة Crossplane.
