Создает навыки агента из текущего каталога проекта (локальные документы/источник). Используйте, когда пользователь хочет загрузить навыки для своего проекта, используя существующую документацию в рабочей области — без клона GitHub. Вывод осуществляется в разделе «skills/<name>/» в том же проекте; AGENTS.md обновлен.
