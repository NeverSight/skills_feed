Crée des compétences d'agent à partir d'un référentiel arbitraire lorsqu'aucune référence existante n'existe. Clone le dépôt dans .temp/repos, génère des compétences sous un nom défini par l'utilisateur conformément aux conventions du projet, itère jusqu'à ce que les principaux modules soient couverts, puis supprime le clone temporaire. À utiliser lorsque l'utilisateur fournit une URL de dépôt et un nom de compétences pour amorcer les compétences d'un framework ou d'un projet.
