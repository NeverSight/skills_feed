Создает навыки агента из произвольного репозитория, когда существующая ссылка не существует. Клонирует репозиторий в .temp/repos, генерирует навыки под определяемым пользователем именем в соответствии с соглашениями проекта, выполняет итерацию до тех пор, пока не будут покрыты основные модули, а затем удаляет временный клон. Используйте, когда пользователь предоставляет URL-адрес репозитория и имя навыков для загрузки навыков для платформы или проекта.
