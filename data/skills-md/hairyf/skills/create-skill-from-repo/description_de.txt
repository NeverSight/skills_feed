Erstellt Agentenfähigkeiten aus einem beliebigen Repository, wenn keine Referenz vorhanden ist. Klont das Repo nach .temp/repos, generiert Skills unter einem benutzerdefinierten Namen gemäß Projektkonventionen, iteriert, bis die Hauptmodule abgedeckt sind, und entfernt dann den temporären Klon. Wird verwendet, wenn der Benutzer eine Repo-URL und einen Kompetenznamen bereitstellt, um Kompetenzen für ein Framework oder Projekt zu booten.
