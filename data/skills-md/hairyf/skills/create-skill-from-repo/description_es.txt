Crea habilidades de agente a partir de un repositorio arbitrario cuando no existe ninguna referencia. Clona el repositorio en .temp/repos, genera habilidades con un nombre definido por el usuario siguiendo las convenciones del proyecto, itera hasta cubrir los módulos principales y luego elimina el clon temporal. Úselo cuando el usuario proporciona una URL de repositorio y un nombre de habilidades para iniciar habilidades para un marco o proyecto.
