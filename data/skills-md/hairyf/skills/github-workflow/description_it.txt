Flusso standard da qualsiasi origine dell'attività (collegamento o descrizione) alla creazione di un PR: risolvere l'attività, creare ramo e TODO.md, attendere correzioni, creare PR solo rispetto all'origine. Utilizzare le competenze di ricerca per scoprire metodi di query dell'origine dati; dopo la conferma, salva nella configurazione globale in modo che il passaggio di rilevamento possa essere saltato la prossima volta. Da utilizzare quando l'utente fornisce un collegamento/una descrizione dell'attività, chiede di "seguire il flusso di lavoro GitHub" o "creare PR dall'attività".
