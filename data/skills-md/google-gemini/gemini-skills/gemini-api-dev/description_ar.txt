استخدم هذه المهارة عند إنشاء تطبيقات باستخدام نماذج Gemini، أو Gemini API، أو العمل مع محتوى متعدد الوسائط (نص، صور، صوت، فيديو)، تنفيذ استدعاء الوظائف، استخدام المخرجات المنظمة، أو الحاجة إلى مواصفات النموذج الحالية. يغطي استخدام SDK (google-genai لـ Python، و@google/genai لـ JavaScript/TypeScript)، واختيار النموذج، وإمكانيات واجهة برمجة التطبيقات.
