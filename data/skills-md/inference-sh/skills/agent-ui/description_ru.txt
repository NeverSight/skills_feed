Компонент агента с батарейками для React/Next.js из ui.inference.sh.
Один компонент со встроенной средой выполнения, инструментами, потоковой передачей, утверждениями и виджетами.
Возможности: прямой агент, человек в процессе, инструменты на стороне клиента, заполнение форм.
Используется для: создания интерфейсов чата с искусственным интеллектом, агентских пользовательских интерфейсов, вторых пилотов SaaS, помощников.
Триггеры: компонент агента, пользовательский интерфейс агента, агент чата, агент shadcn, агент реагирования,
агентный пользовательский интерфейс, пользовательский интерфейс помощника ИИ, пользовательский интерфейс второго пилота, пользовательский интерфейс вывода, человек в цикле
