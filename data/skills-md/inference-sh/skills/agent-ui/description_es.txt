Componente de agente con baterías incluidas para React/Next.js de ui.inference.sh.
Un componente con tiempo de ejecución, herramientas, transmisión, aprobaciones y widgets integrados.
Capacidades: agente directo, humano involucrado, herramientas del lado del cliente, llenado de formularios.
Úselo para: crear interfaces de chat de IA, IU de agente, copilotos de SaaS y asistentes.
Activadores: componente del agente, interfaz de usuario del agente, agente de chat, agente shadcn, agente de reacción,
interfaz de usuario agente, interfaz de usuario asistente de ai, interfaz de usuario copiloto, interfaz de usuario de inferencia, humano en el bucle
