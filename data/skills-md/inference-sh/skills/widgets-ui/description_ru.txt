Декларативные виджеты пользовательского интерфейса из JSON для React/Next.js из ui.inference.sh.
Отрисовывайте насыщенные интерактивные пользовательские интерфейсы на основе структурированных ответов агентов.
Возможности: формы, кнопки, карточки, макеты, вводы, выборки, флажки.
Использование для: пользовательских интерфейсов, создаваемых агентами, динамических форм, отображения данных, интерактивных карточек.
Триггеры: виджеты, декларативный пользовательский интерфейс, json ui, средство рендеринга виджетов, виджеты агента,
динамический пользовательский интерфейс, виджеты форм, виджеты карточек, виджеты shadcn, пользовательский интерфейс структурированного вывода
