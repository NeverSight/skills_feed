Автоматизация браузера для агентов ИИ через inference.sh.
Перемещайтесь по веб-страницам, взаимодействуйте с элементами с помощью @e refs, делайте снимки экрана.
Возможности: парсинг веб-страниц, заполнение форм, нажатие, ввод текста, выполнение JavaScript.
Использование для: веб-автоматизации, извлечения данных, тестирования, просмотра агентов, исследований.
Триггеры: браузер, веб-автоматизация, очистка, навигация, щелчок, заполнение формы, снимок экрана,
просмотр веб-страниц, драматург, безголовый браузер, веб-агент, серфинг в Интернете
