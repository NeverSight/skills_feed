inference.sh CLI를 통해 게시, 참여, 사용자 관리를 통해 Twitter/X를 자동화합니다.
앱: x/post-tweet, x/post-create(미디어 포함), x/post-like, x/post-retweet, x/dm-send, x/user-follow.
기능: 트윗 게시, 게시물과 같은 콘텐츠 예약, 리트윗, DM 보내기, 사용자 팔로우, 프로필 가져오기.
용도: 소셜 미디어 자동화, 콘텐츠 일정 관리, 참여 봇, 청중 증가, X API.
트리거: twitter api, x api, 트윗 자동화, twitter에 게시, twitter bot,
