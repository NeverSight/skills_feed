Composants d'interface utilisateur du cycle de vie des outils pour React/Next.js à partir de ui.inference.sh.
Afficher les appels d'outils : en attente, progression, approbation requise, résultats.
Capacités : statut de l'outil, indicateurs de progression, flux d'approbation, affichage des résultats.
À utiliser pour : afficher les appels d'outils d'agent, les approbations humaines dans la boucle, la sortie de l'outil.
Déclencheurs : interface utilisateur de l'outil, appels d'outils, statut de l'outil, approbation de l'outil, résultats de l'outil,
outils d'agent, interface utilisateur des outils mcp, interface utilisateur d'appel de fonction, cycle de vie de l'outil, outil en attente
