Composant agent inclus dans les batteries pour React/Next.js à partir de ui.inference.sh.
Un composant avec runtime, outils, streaming, approbations et widgets intégrés.
Capacités : agent sans rendez-vous, humain dans la boucle, outils côté client, remplissage de formulaires.
Utiliser pour : créer des interfaces de chat IA, des interfaces utilisateur agents, des copilotes SaaS, des assistants.
Déclencheurs : composant d'agent, interface utilisateur de l'agent, agent de chat, agent shadcn, agent de réaction,
interface utilisateur agentique, interface utilisateur assistant IA, interface utilisateur copilote, interface utilisateur d'inférence, humain dans la boucle
