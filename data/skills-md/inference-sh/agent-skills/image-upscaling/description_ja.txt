inference.sh CLI 経由で Real-ESRGAN、Thera、Topaz、FLUX Upscaler を使用して画像をアップスケールおよび強化します。
モデル: Real-ESRGAN、Thera (任意のサイズ)、FLUX Dev Upscaler、Topaz Image Upscaler。
用途: 低解像度画像の強化、AI アートの高級化、古い写真の復元、解像度の向上。
トリガー: 画像のアップスケール、画像のアップスケーラー、画像の強化、解像度の向上、
リアルエスガン、AIアップスケール、超解像、イメージエンハンスメント、アップスケーリング、
画像拡大、高解像度、4Kアップスケール、HDアップスケール
