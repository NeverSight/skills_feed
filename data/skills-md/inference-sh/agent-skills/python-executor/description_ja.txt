[inference.sh](https://inference.sh) を介して、安全なサンドボックス環境で Python コードを実行します。
プリインストール: NumPy、Pandas、Matplotlib、リクエスト、BeautifulSoup、Selenium、
Playwright、MoviePy、Pillow、OpenCV、trimesh、および 100 以上のライブラリ。
用途: データ処理、Web スクレイピング、画像操作、ビデオ作成、
3D モデル処理、PDF 生成、API 呼び出し、自動化スクリプト。
トリガー: Python、コードの実行、スクリプトの実行、Web スクレイピング、データ分析、
画像処理、ビデオ編集、3D モデル、自動化、パンダ、matplotlib
