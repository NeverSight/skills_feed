Лучшие практики для Tone.js — программный синтез музыки и звука в браузере с использованием API веб-аудио. Охватывает синтезаторы, эффекты, секвенирование, тайминг, семплы, сигналы, шаблоны, MIDI, визуализацию, звук браузера, отображение музыкальных эмоций, построение ритма/бита, утилиты теории музыки, генеративную/алгоритмическую композицию, оптимизацию производительности, автономный рендеринг/запись и интеграцию React/Next.js.
