Лучшие практики совместного редактирования Velt CRDT (Yjs) для приложений реального времени. Этот навык следует использовать при реализации функций совместной работы с использованием хранилищ Velt CRDT, интеграции с такими редакторами, как Tiptap, BlockNote, CodeMirror или ReactFlow, а также при отладке проблем синхронизации. Запускается при выполнении задач, связанных с совместной работой в реальном времени, многопользовательским редактированием, магазинами CRDT или интеграцией Velt SDK.
