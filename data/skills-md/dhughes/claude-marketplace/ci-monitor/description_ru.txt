Этот навык следует использовать, когда пользователь просит «мониторить PR», «наблюдать за CI», «проверять, проходит ли CI», «дайте мне знать, когда CI завершится», «наблюдать за проверками», «отслеживать состояние CI», «сообщать мне, когда сборка завершится» или любой вариант, запрашивающий отслеживание статуса проверки PR GitHub до завершения. Также активно используйте этот навык после создания или обновления PR, когда пользователю будет полезно знать результат CI.
