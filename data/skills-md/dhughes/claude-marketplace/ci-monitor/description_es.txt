Esta habilidad debe usarse cuando el usuario solicita "monitorear el PR", "observar el CI", "verificar si el CI pasa", "avíseme cuando finalice el CI", "observar las comprobaciones", "monitorear el estado del CI", "avisarme cuando se complete la compilación" o cualquier variación que solicite realizar un seguimiento del estado de verificación de PR de GitHub hasta su finalización. Utilice también esta habilidad de forma proactiva después de crear o actualizar un PR cuando el usuario se beneficiaría de conocer el resultado del CI.
