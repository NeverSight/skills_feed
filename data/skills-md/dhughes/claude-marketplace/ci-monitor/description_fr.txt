Cette compétence doit être utilisée lorsque l'utilisateur demande de « surveiller le PR », « surveiller le CI », « vérifier si le CI réussit », « me faire savoir quand le CI se termine », « surveiller les vérifications », « surveiller l'état du CI », « me dire quand la construction est terminée », ou toute variante demandant de suivre l'état de la vérification des PR de GitHub jusqu'à la fin. Utilisez également cette compétence de manière proactive après la création ou la mise à jour d'un PR lorsque l'utilisateur aurait intérêt à connaître le résultat du CI.
