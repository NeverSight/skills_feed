Directives de développement backend FastAPI pour les applications asynchrones Python. Conception basée sur le domaine avec des routeurs FastAPI, SQLModel/SQLAlchemy ORM, modèle de référentiel, couche de service, modèles asynchrones/attente, validation Pydantic et gestion des erreurs. À utiliser lors de la création d'API, de routes, de services, de référentiels ou lors de l'utilisation de code backend.
