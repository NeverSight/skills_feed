مرجع PostgreSQL 16+ لكتابة الاستعلامات وتصميم المخططات وإدارة الفهارس وتحسين الأداء. يُستخدم عندما: (1) كتابة استعلامات SQL (SELECT، INSERT، UPDATE، DELETE، CTEs، وظائف النافذة، الاستعلامات الفرعية)، (2) تصميم مخططات الجدول أو تعديلها (CREATE TABLE، القيود، التسلسلات، أنواع البيانات)، (3) اختيار أو إنشاء الفهارس (B-tree، GIN، GiST، BRIN، Partial، Expression)، (4) إدارة المعاملات ومستويات العزل (BEGIN/COMMIT، نقاط الحفظ، MVCC)، (5) العمل باستخدام بيانات JSONB وjsonpath، (6) تحليل أداء الاستعلام باستخدام EXPLAIN ANALYZE، أو (7) باستخدام psql CLI.
