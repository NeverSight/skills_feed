Riferimento PostgreSQL 16+ per scrivere query, progettare schemi, gestire indici e ottimizzare le prestazioni. Utilizzare quando: (1) scrivere query SQL (SELECT, INSERT, UPDATE, DELETE, CTE, funzioni finestra, sottoquery), (2) progettare o modificare schemi di tabelle (CREATE TABLE, vincoli, sequenze, tipi di dati), (3) scegliere o creare indici (B-tree, GIN, GiST, BRIN, partial, espressione), (4) gestire transazioni e livelli di isolamento (BEGIN/COMMIT, punti di salvataggio, MVCC), (5) lavorare con Dati JSONB e jsonpath, (6) analisi delle prestazioni delle query con EXPLAIN ANALYZE o (7) utilizzo della CLI psql.
