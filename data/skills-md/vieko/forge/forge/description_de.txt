Überprüfungsgrenzen-CLI, die Aufgaben an autonome Agenten delegiert. Verwenden Sie diese Option, wenn der Benutzer Forge ausführen, Spezifikationen ausführen, Spezifikationen parallel ausführen, ausstehende Spezifikationen ausführen, Spezifikationen aus einer Beschreibung definieren, Spezifikationen auflösen, Code anhand von Spezifikationen prüfen, Änderungen überprüfen, Live-Protokolle ansehen, den Ausführungsstatus überprüfen, eine Sitzung fortsetzen oder komplexe mehrstufige Arbeiten an einen autonomen Agenten delegieren möchte. Zu den Auslösern gehören „Forge Run“, „Run This Spec“, „Run Specs Parallel“, „Run Pending“, „Forge Definieren“, „Define Specs“, „Audit the Codebase“, „Review Changes“, „Forge Watch“, „Forge Status“, „Rerun Failed“, „Resolve Spec“, „Delegate This to Forge“.
