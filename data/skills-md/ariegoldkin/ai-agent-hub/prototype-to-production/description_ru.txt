Преобразуйте прототипы дизайна (экспорт HTML, CSS, Figma) в готовые к производству компоненты. Анализирует структуру прототипа, извлекает токены проектирования, определяет повторно используемые шаблоны и генерирует типизированные компоненты React. Адаптируется к существующему технологическому стеку проекта с помощью React + TypeScript по умолчанию.
