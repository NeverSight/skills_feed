Converti prototipi di progettazione (esportazioni HTML, CSS, Figma) in componenti pronti per la produzione. Analizza la struttura del prototipo, estrae i token di progettazione, identifica modelli riutilizzabili e genera componenti React tipizzati. Si adatta allo stack tecnologico del progetto esistente con React + TypeScript come impostazione predefinita.
