Créez ou modifiez des applications qui utilisent le framework pytincture et le jeu de widgets dhxpyt. À utiliser lorsqu'on vous le demande pour créer des services pytincture, ajouter des classes/politiques backend-for-frontend (BFF), générer des mises en page d'interface utilisateur avec des widgets dhxpyt (grille, formulaire, barre d'outils, barre latérale, barre d'onglets, etc.) ou intégrer pytincture.js pour les scripts d'interface utilisateur Python uniquement pour navigateur.
