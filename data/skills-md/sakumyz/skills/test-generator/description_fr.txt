Générateur de cas de test frontend. Crée automatiquement des cas de test de base à partir de signatures de composants ou de fonctions, prenant en charge Jest/Vitest + React Testing Library/Vue Test Utils. Idéal pour : (1) ajouter des tests aux composants existants, (2) générer un échafaudage de tests pour les nouveaux composants, (3) produire des tests unitaires pour les fonctions utilitaires.
