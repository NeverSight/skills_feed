Front-End-Testfallgenerator. Generiert automatisch grundlegende Testfälle aus Komponenten- oder Funktionssignaturen und unterstützt Jest/Vitest + React Testing Library/Vue Test Utils. Geeignet für: (1) Ergänzung von Tests für bestehende Komponenten, (2) Generierung von Testgerüsten für neue Komponenten, (3) Erstellung von Unit-Tests für Utility-Funktionen.
