Генератор тестовых сценариев внешнего интерфейса. Автоматически создает базовые тестовые примеры на основе сигнатур компонентов или функций, поддерживая Jest/Vitest + React Testing Library/Vue Test Utils. Идеально подходит для: (1) добавления тестов к существующим компонентам, (2) создания скелетов тестов для новых компонентов, (3) создания модульных тестов для служебных функций.
