Assistent zur Überprüfung des Frontend-Codes. Führt eine eingehende Überprüfung von bereitgestelltem Code oder bestimmten Dateien durch und überprüft dabei Codestandards, potenzielle Probleme, Leistungsrisiken, Sicherheitslücken und TypeScript-Typsicherheit. Unterstützt React+MUI- und Vue3+AntDesign-Stacks. Geeignet für: (1) Selbstprüfung vor PR-Einreichung, (2) vollständige Überprüfung des Projektcodes, (3) eingehende Überprüfung bestimmter Dateien.
