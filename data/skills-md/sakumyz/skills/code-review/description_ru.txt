Помощник по проверке внешнего кода. Выполняет углубленные проверки промежуточного кода или указанных файлов, проверяя стандарты кода, потенциальные проблемы, риски производительности, уязвимости безопасности и безопасность типов TypeScript. Поддерживает технологические стеки React+MUI и Vue3+AntDesign. Подходит для: (1) самопроверки перед PR, (2) полного аудита кода проекта, (3) глубокой проверки конкретных файлов.
