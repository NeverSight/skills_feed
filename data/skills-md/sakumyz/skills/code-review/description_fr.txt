Assistant de révision de code front-end. Effectue des examens approfondis sur le code préparé ou les fichiers spécifiés, en vérifiant les normes de code, les problèmes potentiels, les risques de performances, les vulnérabilités de sécurité et la sécurité des types TypeScript. Prend en charge les piles React+MUI et Vue3+AntDesign. Convient pour : (1) l'auto-inspection avant la soumission du PR, (2) l'examen complet du code du projet, (3) l'examen approfondi de fichiers spécifiques.
