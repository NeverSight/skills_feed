Front-end intelligent debugging assistant. Quickly locate problematic code from error messages, console logs, and descriptions of abnormal behavior, and provide repair suggestions. Applicable to: (1) runtime errors (TypeError, ReferenceError, etc.), (2) console warning analysis, (3) white screen/stuttering troubleshooting, (4) network request anomalies, (5) component rendering anomalies. Difference from BugHunter: smart-debugger focuses on quick location, BugHunter provides complete closed-loop fixes.
