Configuración e implementación completa de documentación OpenAPI (Swagger) para aplicaciones NestJS. Úselo cuando trabaje con proyectos NestJS que necesiten documentación de API, configuración de UI de Swagger o generación de especificaciones de OpenAPI. Cubre la configuración inicial, el uso del decorador, la configuración del complemento CLI, las definiciones de modelos, la autenticación y funciones avanzadas como múltiples especificaciones y respuestas personalizadas.
