إعداد وثائق OpenAPI (Swagger) الشاملة وتنفيذها لتطبيقات NestJS. يُستخدم عند العمل مع مشاريع NestJS التي تحتاج إلى وثائق API أو إعداد Swagger UI أو إنشاء مواصفات OpenAPI. يغطي الإعداد الأولي، واستخدام الديكور، وتكوين مكون CLI الإضافي، وتعريفات النموذج، والمصادقة، والميزات المتقدمة مثل المواصفات المتعددة والاستجابات المخصصة.
