Configuration et mise en œuvre complète de la documentation OpenAPI (Swagger) pour les applications NestJS. À utiliser lorsque vous travaillez avec des projets NestJS nécessitant une documentation API, la configuration de l'interface utilisateur Swagger ou la génération de spécifications OpenAPI. Couvre la configuration initiale, l'utilisation du décorateur, la configuration du plug-in CLI, les définitions de modèles, l'authentification et les fonctionnalités avancées telles que les spécifications multiples et les réponses personnalisées.
