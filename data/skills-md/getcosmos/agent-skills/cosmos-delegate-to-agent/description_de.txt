Bringt Agenten bei, wie sie das DelegateToAgent-Tool effektiv zur Orchestrierung der Arbeit mehrerer Agenten nutzen können. Behandelt die Analyse von Abhängigkeitsdiagrammen, parallele und serielle Ausführungsmuster sowie die Fortschrittsverfolgung. Verwenden Sie diese Option, wenn ein Agent Aufgaben an spezialisierte Agenten delegieren muss, insbesondere bei der Arbeit mit Plänen, Aufgaben oder mehrstufigen Anfragen.
