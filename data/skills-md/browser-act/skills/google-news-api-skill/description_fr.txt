Cette compétence aide les utilisateurs à extraire automatiquement les données d'actualités structurées de Google Actualités via l'API BrowserAct. L'agent doit appliquer cette compétence de manière proactive lorsque les utilisateurs expriment des besoins tels que rechercher des actualités sur un sujet spécifique, suivre les tendances du secteur, surveiller les relations publiques ou les sentiments, collecter les mises à jour des concurrents, obtenir les derniers rapports sur des mots clés spécifiques, surveiller l'exposition de la marque dans les médias, rechercher des sujets d'actualité du marché, résumer les actualités quotidiennes de l'industrie, suivre les activités médiatiques d'individus spécifiques, récupérer les événements chauds des dernières 24 heures, extraire des données structurées pour les études de marché, surveiller les dernières nouvelles mondiales.
