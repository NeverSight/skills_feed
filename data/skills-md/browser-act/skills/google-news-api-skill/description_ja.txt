このスキルは、ユーザーが BrowserAct API を介して Google ニュースから構造化ニュース データを自動的に抽出するのに役立ちます。特定のトピックに関するニュースの検索、業界動向の追跡、広報やセンチメントの監視、競合他社の最新情報の収集、特定のキーワードに関する最新レポートの取得、メディアでのブランド露出の監視、市場で注目のトピックの調査、毎日の業界ニュースの要約、特定の個人のメディア活動の追跡、過去 24 時間の注目のイベントの取得、市場調査のための構造化データの抽出、世界的なニュース速報の監視などのニーズをユーザーが表明した場合、エージェントはこのスキルを積極的に適用する必要があります。
