قم بإنشاء ملاحظات الإصدار من التزامات git ومشكلات/علاقات العامة في GitHub. يُستخدم عندما يُطلب منك "إنشاء ملاحظات الإصدار"، أو "إنشاء سجل التغيير"، أو "التحضير للإصدار"، أو "ما تغير منذ الإصدار الأخير"، أو الحاجة إلى توثيق التغييرات لإصدار جديد. تحليلات الالتزام بالتاريخ، والعلاقات العامة المدمجة، والقضايا المغلقة لإنتاج ملاحظات منسقة لإصدارات GitHub.
