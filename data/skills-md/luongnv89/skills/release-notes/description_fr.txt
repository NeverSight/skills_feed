Générez des notes de version à partir des commits git et des PR/problèmes GitHub. À utiliser lorsqu'on vous demande de « créer des notes de version », « générer un journal des modifications », « préparer la version », « ce qui a changé depuis la dernière version » ou si vous avez besoin de documenter les modifications pour une nouvelle version. Analyse l'historique des validations, les PR fusionnés et les problèmes résolus pour produire des notes au format des versions GitHub.
