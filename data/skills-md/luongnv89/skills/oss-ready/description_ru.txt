Превратите проекты в профессиональные репозитории с открытым исходным кодом с помощью стандартных компонентов. Используйте, когда пользователи просят «сделать этот проект открытым исходным кодом», «добавить файлы с открытым исходным кодом», «настроить стандарты OSS», «создать руководство для участников», «добавить лицензию» или хотят подготовить проект к публичному выпуску с помощью шаблонов README, CONTRIBUTING, LICENSE и GitHub.
