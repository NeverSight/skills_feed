Genere tareas de desarrollo a partir de un archivo PRD con planificación basada en sprints. Úselo cuando los usuarios soliciten "crear tareas a partir de PRD", "desglosar el PRD", "generar tareas de sprint" o deseen convertir los requisitos del producto en tareas de desarrollo procesables. Crea task.md con enfoque POC → MVP → Funciones completas y análisis de dependencia.
