Audita y optimiza las bases de código de sitios web para SEO y escaneo de bots de IA. Cubre SEO técnico (metaetiquetas, mapas de sitios, robots.txt, URL canónicas, sugerencias de velocidad de página, datos estructurados), SEO de contenido (estructura de encabezados, texto alternativo, legibilidad) y accesibilidad de bots de IA (llms.txt, directivas GPTBot/ClaudeBot, ai-plugin.json, Schema.org/JSON-LD). Úselo cuando el usuario solicite "optimizar para SEO", "auditar SEO", "mejorar las clasificaciones de búsqueda", "hacer que el sitio sea compatible con la IA", "agregar datos estructurados", "corregir metaetiquetas" u "optimizar para los robots de IA". Funciona con cualquier marco web.
