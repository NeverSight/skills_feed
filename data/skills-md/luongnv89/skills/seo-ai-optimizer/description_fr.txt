Audite et optimise les bases de code des sites Web pour le référencement et l’analyse des robots IA. Couvre le référencement technique (balises méta, plans de site, robots.txt, URL canoniques, conseils de vitesse de page, données structurées), le référencement de contenu (structure des titres, texte alternatif, lisibilité) et l'accessibilité des robots IA (llms.txt, directives GPTBot/ClaudeBot, ai-plugin.json, Schema.org/JSON-LD). À utiliser lorsque l'utilisateur demande « optimiser le référencement », « auditer le référencement », « améliorer les classements de recherche », « rendre le site convivial pour l'IA », « ajouter des données structurées », « corriger les balises méta » ou « optimiser pour les robots IA ». Fonctionne avec n'importe quel framework Web.
