Wenden Sie OpenSpec OPSX in einer strikten Schleife für jeweils eine Aufgabe an. Verwenden Sie diese Option, wenn der Benutzer die Ausführung von Arbeiten als Einzelaufgabenänderungen anfordert, eine Spec-First-Implementierung pro Aufgabe wünscht oder sagt, dass er für jede Aufgabe aus einer Aufgabenliste die OpenSpec-Methode verwenden soll. Unterstützt sowohl native /opsx-Befehlsumgebungen als auch manuelles Fallback durch direktes Erstellen von OpenSpec-Artefaktdateien.
