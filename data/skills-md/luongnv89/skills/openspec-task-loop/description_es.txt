Aplique OpenSpec OPSX en un ciclo estricto de una tarea a la vez. Úselo cuando el usuario solicita ejecutar el trabajo como cambios de una sola tarea, desea una implementación de especificaciones por tarea primero o dice que se use el método OpenSpec para cada tarea de una lista de tareas. Admite entornos de comando nativos /opsx y respaldo manual mediante la creación directa de archivos de artefactos OpenSpec.
