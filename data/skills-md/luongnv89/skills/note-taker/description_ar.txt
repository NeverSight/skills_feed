التقط ملاحظات الدردشة (نص، صوت، صورة، فيديو، ملف) في مستودع الملاحظات المدعومة بـ git، وقم بتلخيصها وتنظيمها، واستخراج المهام إلى KANBAN.md، وتنفيذ/دفع التغييرات، والإبلاغ دائمًا عن روابط ويب GitHub إلى الملاحظات الجديدة/المحدثة. يُستخدم عندما يقول المستخدم أنه يريد تدوين ملاحظة، أو حفظ ملاحظة، أو التقاطها، أو إدارة ملاحظاته/سجلاته المتراكمة.
