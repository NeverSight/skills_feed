Effectuez des révisions de code en suivant les meilleures pratiques de Code Smells et The Pragmatic Programmer. À utiliser lorsqu'on vous demande de « réviser ce code », « vérifier les odeurs de code », « réviser mon PR », « auditer la base de code » ou avoir besoin d'un retour de qualité sur les modifications du code. Prend en charge à la fois les audits complets de la base de code et les examens PR/diff ciblés. Produit des rapports de démarques structurés regroupés par gravité.
