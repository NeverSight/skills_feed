قم بإجراء مراجعات التعليمات البرمجية باتباع أفضل الممارسات من Code Smells وThe Pragmatic Programmer. يُستخدم عندما يُطلب منك "مراجعة هذا الرمز"، أو "التحقق من روائح الكود"، أو "مراجعة العلاقات العامة الخاصة بي"، أو "تدقيق قاعدة التعليمات البرمجية"، أو الحاجة إلى تعليقات جيدة بشأن تغييرات التعليمات البرمجية. يدعم كلاً من عمليات تدقيق قاعدة التعليمات البرمجية الكاملة ومراجعات العلاقات العامة/الفرق المركزة. مخرجات تقارير تخفيض السعر المنظمة مجمعة حسب الخطورة.
