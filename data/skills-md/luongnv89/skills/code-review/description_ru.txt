Выполняйте проверки кода, следуя рекомендациям Code Smells и The Pragmatic Programmer. Используйте, когда вас просят «проверить этот код», «проверить наличие запахов в коде», «проверить мой PR», «аудит кодовой базы» или вам нужна качественная обратная связь по изменениям кода. Поддерживает как полный аудит кодовой базы, так и целевые PR/проверки различий. Выводит структурированные отчеты об уценках, сгруппированные по серьезности.
