Esegui revisioni del codice seguendo le best practice di Code Smells e The Pragmatic Programmer. Da utilizzare quando viene richiesto di "rivedere questo codice", "controllare gli odori del codice", "rivedere il mio PR", "verificare la base di codice" o se è necessario un feedback di qualità sulle modifiche al codice. Supporta sia controlli completi della base di codice che revisioni PR/diff mirate. Genera report strutturati sui ribassi raggruppati per gravità.
