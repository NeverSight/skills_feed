Создавайте кроссплатформенные сценарии установки для любого программного обеспечения, библиотеки или модуля. Используйте, когда пользователи просят «создать установщик», «создать сценарий установки», «автоматизировать установку», «скрипт установки для X», «установить X в любой ОС» или требуется автоматическое развертывание в Windows, Linux и macOS. Этот навык генерирует автономный скрипт install.sh (и, опционально, install.ps1), который можно запустить с помощью одной однострочной команды curl/wget из необработанного пользовательского контента GitHub. Он состоит из четырех этапов: (1) Исследование среды — обнаружение ОС, сбор информации о системе, проверка зависимостей; (2) Планирование установки – предложите шаги с проверкой; (3) Генерация сценария — создание автономного сценария установки с помощью однострочной команды; (4) Генерация документации.
