Generieren Sie plattformübergreifende Installationsskripte für jede Software, Bibliothek oder jedes Modul. Verwenden Sie diese Option, wenn Benutzer „einen Installer erstellen“, „Installationsskript generieren“, „Installation automatisieren“, „Setup-Skript für Der Skill generiert ein eigenständiges „install.sh“-Skript (und optional „install.ps1“), das über einen einzelnen Curl/Wget-Einzeiler aus GitHub-Rohbenutzerinhalten ausgeführt werden kann. Es folgt einem vierphasigen Ansatz: (1) Umgebungserkundung – Betriebssystem erkennen, Systeminformationen sammeln, Abhängigkeiten prüfen; (2) Installationsplanung – Schritte mit Überprüfung vorschlagen; (3) Skriptgenerierung – Erstellen Sie ein eigenständiges Installationsskript mit einem einzeiligen Befehl. (4) Dokumentationserstellung.
