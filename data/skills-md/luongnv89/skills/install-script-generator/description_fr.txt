Générez des scripts d'installation multiplateformes pour n'importe quel logiciel, bibliothèque ou module. À utiliser lorsque les utilisateurs demandent à « créer un programme d'installation », « générer un script d'installation », « automatiser l'installation », « script d'installation pour X », « installer X sur n'importe quel système d'exploitation » ou ont besoin d'un déploiement automatisé sur Windows, Linux et macOS. La compétence génère un script autonome « ​​install.sh » (et éventuellement « install.ps1 ») qui peut être exécuté via une seule ligne curl/wget à partir du contenu utilisateur brut de GitHub. Il suit une approche en quatre phases : (1) Exploration de l'environnement - détecter le système d'exploitation, collecter des informations sur le système, vérifier les dépendances ; (2) Planification de l'installation - proposer des étapes avec vérification ; (3) Génération de script - produit un script d'installation autonome avec une commande à une seule ligne ; (4) Génération de documentation.
