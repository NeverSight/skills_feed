Genere scripts de instalación multiplataforma para cualquier software, biblioteca o módulo. Úselo cuando los usuarios soliciten "crear un instalador", "generar un script de instalación", "automatizar la instalación", "configurar un script para X", "instalar X en cualquier sistema operativo" o necesitan una implementación automatizada en Windows, Linux y macOS. La habilidad genera un script independiente `install.sh` (y opcionalmente `install.ps1`) que se puede ejecutar a través de una sola línea curl/wget desde el contenido sin procesar del usuario de GitHub. Sigue un enfoque de cuatro fases: (1) Exploración del entorno: detectar el sistema operativo, recopilar información del sistema, comprobar las dependencias; (2) Planificación de la instalación: proponer pasos con verificación; (3) Generación de secuencias de comandos: produzca una secuencia de comandos de instalación independiente con un comando de una sola línea; (4) Generación de documentación.
