قم بإنشاء برامج نصية للتثبيت عبر الأنظمة الأساسية لأي برنامج أو مكتبة أو وحدة نمطية. يُستخدم عندما يطلب المستخدمون "إنشاء برنامج تثبيت"، أو "إنشاء برنامج نصي للتثبيت"، أو "التثبيت التلقائي"، أو "البرنامج النصي للإعداد لـ X"، أو "تثبيت X على أي نظام تشغيل"، أو عندما يحتاجون إلى النشر التلقائي عبر أنظمة التشغيل Windows، وLinux، وmacOS. تُنشئ المهارة نصًا برمجيًا مستقلاً "install.sh" (واختياريًا "install.ps1") يمكن تشغيله عبر سطر واحد من تجعيد/wget من محتوى مستخدم GitHub الأولي. ويتبع نهجًا من أربع مراحل: (1) استكشاف البيئة - اكتشاف نظام التشغيل، وجمع معلومات النظام، والتحقق من التبعيات؛ (2) تخطيط التثبيت - اقتراح الخطوات مع التحقق؛ (3) إنشاء البرنامج النصي - إنتاج برنامج نصي تثبيت قائم بذاته باستخدام أمر أحادي السطر؛ (4) توليد الوثائق.
