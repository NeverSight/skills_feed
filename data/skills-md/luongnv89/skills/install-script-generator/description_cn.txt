为任何软件、库或模块生成跨平台安装脚本。当用户要求“创建安装程序”、“生成安装脚本”、“自动安装”、“X 的安装脚本”、“在任何操作系统上安装 X”或需要跨 Windows、Linux 和 macOS 进行自动部署时使用。该技能会生成一个独立的“install.sh”（以及可选的“install.ps1”）脚本，该脚本可以通过来自 GitHub 原始用户内容的单个curl/wget 单行运行。它遵循四个阶段的方法：（1）环境探索 - 检测操作系统，收集系统信息，检查依赖关系； (2) 安装规划——提出步骤并进行验证； (3) 脚本生成——用一行命令生成一个独立的安装脚本； (4) 文档生成。
