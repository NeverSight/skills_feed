あらゆるソフトウェア、ライブラリ、モジュールのクロスプラットフォーム インストール スクリプトを生成します。ユーザーが「インストーラーの作成」、「インストール スクリプトの生成」、「インストールの自動化」、「X のセットアップ スクリプト」、「任意の OS に X をインストール」を要求する場合、または Windows、Linux、および macOS での自動展開が必要な場合に使用します。このスキルは、GitHub の生のユーザー コンテンツから 1 つのcurl/wget ワンライナーで実行できるスタンドアロンの「install.sh」（およびオプションで「install.ps1」）スクリプトを生成します。これは 4 段階のアプローチに従います。(1) 環境の調査 - OS を検出し、システム情報を収集し、依存関係を確認します。 (2) 設置計画 - 検証を伴う手順を提案します。 (3) スクリプト生成 - ワンライナー コマンドを使用して自己完結型のインストール スクリプトを生成します。 (4) ドキュメントの作成。
