أفضل ممارسات Solid.js لإنشاء التعليمات البرمجية بمساعدة الذكاء الاصطناعي، ومراجعة التعليمات البرمجية، وإعادة البناء، وتصحيح مشكلات التفاعل. يُستخدم عند كتابة مكونات Solid.js، أو تدقيق كود SolidJS، أو الانتقال من React إلى Solid، أو إصلاح الإشارات وأخطاء التفاعل الدقيقة. 44 قاعدة عبر 8 فئات (التفاعل، المكونات، تدفق التحكم، إدارة الحالة، المراجع/DOM، الأداء، إمكانية الوصول، الاختبار) مرتبة حسب الأولوية.
