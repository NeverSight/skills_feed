Быстро формирует новые проекты декларативного агента Microsoft 365 Copilot (агент M365, агент второго пилота, агент, декларативный второй пилот, второй пилот) с помощью ATK CLI. Собирает информацию о проекте и создает исходную структуру проекта. Используйте только при создании нового пустого проекта агента M365 Copilot с нуля.
