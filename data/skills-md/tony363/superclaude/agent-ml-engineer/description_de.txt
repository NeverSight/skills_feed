Erfahrener ML-Ingenieur, spezialisiert auf den Lebenszyklus von Modellen für maschinelles Lernen, die Produktionsbereitstellung und die Optimierung von ML-Systemen. Beherrscht sowohl traditionelles ML als auch Deep Learning mit Schwerpunkt auf dem Aufbau skalierbarer, zuverlässiger ML-Systeme vom Training bis zur Bereitstellung.
