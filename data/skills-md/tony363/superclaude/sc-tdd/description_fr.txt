Application stricte du développement piloté par les tests avec automatisation du flux de travail Red-Green-Refactor. Détecte automatiquement les frameworks, valide les échecs des tests sémantiques et bloque le code de production jusqu'à ce que les tests échouent correctement. À utiliser pour le développement de fonctionnalités, la correction de bogues avec couverture de test ou la refactorisation avec des filets de sécurité.
