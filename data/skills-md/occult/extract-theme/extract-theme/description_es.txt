Extraiga el tema visual completo de un sitio web (colores, tipografía, espaciado, degradados, sombras, radios de borde) usando Chrome DevTools y conviértalo en un tema CSS shadcn/ui compatible con tweakcn. Úselo cuando el usuario quiera clonar o replicar el sistema de diseño de un sitio web en su proyecto Tailwind CSS/shadcn.
