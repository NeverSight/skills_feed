Estrai il tema visivo completo di un sito web (colori, tipografia, spaziatura, sfumature, ombre, raggi del bordo) utilizzando Chrome DevTools e convertilo in un tema CSS shadcn/ui compatibile con tweakcn. Da utilizzare quando l'utente desidera clonare o replicare il sistema di progettazione di un sito Web nel proprio progetto CSS/shadcn Tailwind.
