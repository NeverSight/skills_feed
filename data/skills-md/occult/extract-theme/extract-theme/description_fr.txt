Extrayez le thème visuel complet d'un site Web (couleurs, typographie, espacement, dégradés, ombres, rayons de bordure) à l'aide de Chrome DevTools et convertissez-le en un thème CSS shadcn/ui compatible avec tweakcn. À utiliser lorsque l'utilisateur souhaite cloner ou répliquer le système de conception d'un site Web dans son projet Tailwind CSS/shadcn.
