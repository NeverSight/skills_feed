shadcn-vue для Vue/Nuxt с компонентами пользовательского интерфейса Reka и Tailwind. Используйте для доступного пользовательского интерфейса, автоматического формирования, таблиц данных, диаграмм, темного режима, настройки сервера MCP или обнаружения импорта компонентов, ошибок пользовательского интерфейса Reka.
