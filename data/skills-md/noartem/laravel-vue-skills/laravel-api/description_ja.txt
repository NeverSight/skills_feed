Laravel のベスト プラクティスに基づく独自のアーキテクチャ パターンを使用して、本番グレードの Laravel REST API を構築します。ステートレス設計、バージョン管理されたエンドポイント、呼び出し可能なコントローラー、フォームリクエスト DTO、アクションクラス、JWT 認証、および PSR-12 コード品質基準の仕様を備えた Laravel API を構築、スキャフォールディング、またはレビューするときに使用します。 「Laravel API の構築」、「Laravel エンドポイントの作成」、「API 認証の追加」、「Laravel API コードのレビュー」、「Laravel API のリファクタリング」、または「Laravel コードの品質の向上」をトリガーします。
