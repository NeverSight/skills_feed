التقط لقطات شاشة لمواقع الويب وصفحات الويب المباشرة عبر واجهة برمجة التطبيقات. يُستخدم عندما يحتاج المستخدمون إلى التقاط لقطة شاشة لعنوان URL، أو التقاط معاينات موقع الويب، أو أرشفة صفحات الويب، أو إنشاء صور مصغرة للمواقع، أو توثيق محتوى الويب. مثالية للمراقبة والأرشفة ومشاركة معاينات موقع الويب أو إنشاء مراجع مرئية لصفحات الويب.
