Комплексный эксперт по разработке 4D v21 со встроенной официальной документацией. Охватывает шаблоны ORDA, классы сущностей, классы модели данных, запросы, классические методы, типы данных, обработку ошибок, формы, события, веб-сервер и REST API. Используйте при работе с 4D-файлами (.4dm), языковыми вопросами 4D, структурой 4D-проекта, классами сущностей, запросами ORDA, операциями с базами данных, разработкой форм, конфигурацией веб-сервера, использованием REST API или любыми проблемами синтаксиса, специфичными для 4D. Включает полную официальную документацию 4D v21 для поиска по требованию. Запускается по: файлам .4dm, коду 4D, ORDA, выбору объекта, классу данных, запросу 4D, форме 4D, веб-серверу 4D, REST 4D, команде 4D, коллекции 4D, объекту 4D.
