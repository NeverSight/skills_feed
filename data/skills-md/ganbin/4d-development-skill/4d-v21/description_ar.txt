خبير تطوير شامل لـ 4D v21 مع الوثائق الرسمية المضمنة. يغطي أنماط ORDA، وفئات الكيانات، وفئات نماذج البيانات، والاستعلامات، والأساليب الكلاسيكية، وأنواع البيانات، ومعالجة الأخطاء، والنماذج، والأحداث، وخادم الويب، وREST API. يُستخدم عند العمل مع ملفات 4D (.4dm)، أو أسئلة اللغة 4D، أو بنية المشروع 4D، أو فئات الكيانات، أو استعلامات ORDA، أو عمليات قاعدة البيانات، أو تطوير النماذج، أو تكوين خادم الويب، أو استخدام REST API، أو أي مشكلة في بناء الجملة خاصة بـ 4D. يتضمن وثائق 4D v21 الرسمية الكاملة للاسترجاع عند الطلب. المشغلات على: ملفات .4dm، كود 4D، ORDA، تحديد الكيان، فئة البيانات، استعلام 4D، نموذج 4D، خادم الويب 4D، 4D REST، أمر 4D، مجموعة 4D، كائن 4D.
