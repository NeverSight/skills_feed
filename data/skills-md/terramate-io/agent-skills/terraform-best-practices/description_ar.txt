Terraform والبنية التحتية كإرشادات لتحسين التعليمات البرمجية من Terramate. يجب استخدام هذه المهارة عند كتابة كود Terraform/OpenTofu أو مراجعته أو إعادة هيكلته لضمان الأنماط المثالية للأمان وقابلية الصيانة والموثوقية. يتم تشغيل المهام التي تتضمن وحدات Terraform أو توفير البنية التحتية أو إدارة الحالة أو تحسين IaC.
