Linee guida per l'ottimizzazione di Terraform e Infrastructure as Code di Terramate. Questa competenza dovrebbe essere utilizzata durante la scrittura, la revisione o il refactoring del codice Terraform/OpenTofu per garantire modelli ottimali di sicurezza, manutenibilità e affidabilità. Trigger su attività che coinvolgono moduli Terraform, provisioning dell'infrastruttura, gestione dello stato o ottimizzazione IaC.
