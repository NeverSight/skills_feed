Modèles pour créer des données de test réutilisables : fonctions d'usine pour les objets de domaine, modèles de générateur pour les objets complexes, fabriques de luminaires, données de test paramétrées, héritage de données de test et générateurs de données de test de type sécurisé. À utiliser lors de la création répétée de données de test, de la création d'objets avec de nombreux champs facultatifs ou de la génération d'ensembles de données de test réalistes.

À utiliser lorsque : l'écriture de tests nécessitant des objets de domaine, la création de montages pour plusieurs tests, la génération de données de test avec des valeurs par défaut, la création d'objets complexes avec des champs facultatifs ou le paramétrage de tests avec diverses données.
