Enregistre les flux de travail du navigateur sous forme de GIF annotés pour la documentation et les didacticiels. À utiliser lors de la création d'une documentation visuelle des interactions du navigateur, de l'enregistrement de didacticiels étape par étape ou de la capture de flux de travail pour les tickets d'assistance. Se déclenche sur des expressions telles que « enregistrer ce flux de travail », « créer un didacticiel GIF », « documenter ce processus au format GIF », « montre-moi comment faire X au format GIF » ou lorsque l'utilisateur a besoin d'une documentation visuelle étape par étape. Fonctionne avec l'automatisation du navigateur Chrome via les outils MCP (tabs_context_mcp, gif_creator, ordinateur).
