Fournit des méthodes pour interagir avec l'API REST de Home Assistant pour la découverte d'entités, l'état
récupération et appels de service, y compris l'authentification, les modèles d'entité, les appels de service,
et les requêtes d'état. À utiliser lorsque vous travaillez avec les intégrations de Home Assistant, contrôlez des appareils,
interroger les données des capteurs ou automatiser les opérations de Home Assistant.
