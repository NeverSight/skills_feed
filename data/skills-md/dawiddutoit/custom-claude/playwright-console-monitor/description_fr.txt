Exécute les flux de travail du navigateur tout en surveillant la console JavaScript pour détecter les erreurs, les avertissements et les messages. À utiliser lorsque vous avez besoin d'une validation sans erreur, d'un contexte de débogage pendant l'automatisation ou d'une détection proactive des erreurs dans les applications Web. Déclencheurs sur « vérifier les erreurs de la console », « surveiller les erreurs JavaScript », « valider l'exécution sans erreur » ou « workflow de débogage du navigateur ». Fonctionne avec les outils d'automatisation du navigateur Playwright MCP.
