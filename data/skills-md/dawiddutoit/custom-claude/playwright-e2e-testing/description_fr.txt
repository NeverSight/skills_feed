Tests automatisés de bout en bout avec collecte complète de preuves à l’aide des outils Playwright MCP.
À utiliser lorsque vous devez tester les flux d'utilisateurs, valider les flux de travail, exécuter des tests E2E, vérifier l'application
comportement, ou collecter des preuves pour les rapports de bogues. Déclencheurs sur "tester le flux utilisateur", "exécuter le test E2E",
"valider le flux de travail", "tests automatisés", "tester cette fonctionnalité" ou "vérifier si cela fonctionne".
Fonctionne avec les applications Web, les formulaires, les SPA et toutes les interfaces basées sur un navigateur.
