Ouvrez plusieurs onglets de navigateur et comparez le contenu, le comportement ou l'apparence visuelle des pages. À utiliser pour comparer la préparation et la production, tester les variantes de page A/B, tester la cohérence entre les navigateurs, vérifier la parité du contenu ou analyser les différences entre plusieurs URL. Les expressions de déclenchement incluent « comparer ces pages », « ouvrir plusieurs onglets et comparer », « tester la cohérence entre », « test A/B », « préparation ou production » ou « comparer les versions ». Fonctionne avec les outils d'automatisation du navigateur Playwright MCP pour les flux de travail multi-onglets.
