Crée des serveurs MCP (Model Context Protocol) permettant aux LLM d'interagir avec des services externes
grâce à des outils, des ressources et des invites bien conçus. À utiliser lorsqu'on vous demande de "créer un serveur MCP",
"créer une intégration MCP", "implémenter des outils MCP", "se connecter à une API externe via MCP", ou
"développer le serveur MCP SDK". Guide à travers les phases de recherche, de conception, de mise en œuvre et de test
pour les serveurs Python (FastMCP) ou TypeScript (MCP SDK). Couvre la dénomination des outils, la conception de schémas,
gestion des erreurs, tests et optimisation du flux de travail. Fonctionne avec le protocole MCP, le framework FastMCP,
et le SDK MCP TypeScript.
