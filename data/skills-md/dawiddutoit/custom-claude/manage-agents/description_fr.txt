Crée, modifie et gère les sous-agents Claude Code en écrivant des fichiers d'agent avec le frontmatter YAML,
les invites du système et les configurations des outils. A utiliser lorsque vous devez « créer un agent », « modifier un agent »,
"mettre en place un spécialiste", "J'ai besoin d'un agent pour [tâche]", "un agent pour gérer [domaine]" ou "configurer
outils d'agents". Couvre le format de fichier d'agent, le thème YAML, les invites système, les restrictions d'outils,
Intégration MCP, sélection de modèles et tests.
