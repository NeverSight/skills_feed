Fournit des modèles complets pour tester le code Python asynchrone avec pytest : configuration pytest-asyncio, utilisation d'AsyncMock, appareils asynchrones, test de FastAPI avec AsyncClient, test des producteurs/consommateurs asynchrones Kafka, boucle d'événements et modèles de nettoyage.

À utiliser dans les cas suivants : test de fonctions asynchrones, de cas d'utilisation asynchrones, de points de terminaison FastAPI, d'opérations de base de données asynchrones, de clients asynchrones Kafka ou de tout modèle de code asynchrone/d'attente.
