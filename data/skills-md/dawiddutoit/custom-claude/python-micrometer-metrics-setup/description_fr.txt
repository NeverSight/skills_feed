Configure les métriques Micrometer dans un service Spring Boot à partir de zéro, y compris les dépendances, la configuration de l'actuateur et la personnalisation du registre. À utiliser lors du démarrage d'un nouveau microservice, de la configuration des backends d'exportation de métriques, de l'activation du scraping Prometheus ou des métriques JVM automatiquement configurées. Compétence fondamentale pour l'instrumentation de métriques dans les applications Java/Spring Boot.
