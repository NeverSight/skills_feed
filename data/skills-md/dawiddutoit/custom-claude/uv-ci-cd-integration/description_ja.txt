CI/CD パイプラインで uv (Rust ベースの Python パッケージ マネージャー) をセットアップします。
GitHub Actions ワークフロー、GitLab CI/CD、Docker ビルド、または Python バージョン間でのマトリックス テストを構成するときに使用します。
キャッシュの最適化、凍結されたロックファイル、マルチステージ ビルド、および信頼できる公開による PyPI 公開のパターンが含まれています。
GitHub Actions の setup-uv アクション、Docker のマルチステージの運用/開発ビルド、およびデプロイメント パターンについて説明します。
