Fournit l'installation et la configuration d'intégration pour les onduleurs solaires Sunsynk/Deye avec Home
Assistant utilisant le module complémentaire API cloud SolarSynkV3, y compris la configuration du module complémentaire, l'entité
modèles de dénomination, dépannage et configuration du tableau de bord. À utiliser lors de la configuration de sunsynk
intégration, configuration de l'onduleur Deye, dépannage des erreurs « aucune autorisation »,
accéder aux données du système solaire ou surveiller les capteurs de batterie/PV/réseau/charge.
