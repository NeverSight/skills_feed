Gère les versions Python avec uv. Découvrez comment découvrir les versions Python disponibles, installer des versions spécifiques, épingler des versions spécifiques au projet à l'aide de fichiers .python-version et exécuter des outils avec différentes versions de Python. À utiliser lors de la configuration de nouveaux projets, en garantissant que l'équipe utilise la même version de Python, en testant plusieurs versions ou en gérant des projets existants avec des exigences Python plus anciennes.
