Générez des cas de test JUnit 5 complets pour le code Java avec une simulation et une couverture appropriées.
À utiliser lors de l'écriture de tests, de la création de tests unitaires, de la génération de cas de test, de l'ajout de tests JUnit,
configurer des simulations Mockito, écrire des tests paramétrés ou atteindre une couverture de test.
Fonctionne avec les fichiers .java, génère des classes de test en suivant les conventions de dénomination,
inclut les cas extrêmes, les conditions aux limites et utilise le modèle Arrange-Act-Assert.
