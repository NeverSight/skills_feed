Construit des artefacts HTML claude.ai multi-composants élaborés à l'aide de React 18, TypeScript, Vite,
Tailwind CSS et shadcn/ui regroupés dans des livrables mono-fichiers. À utiliser lorsqu'on lui demande de "créer
artefact complexe", "construire un artefact React avec routage", "utiliser les composants shadcn/ui", "créer
"Artefact d'application Web avec état" ou "créer une interface multi-composants". Fournit l'initialisation
scripts, flux de travail de développement et regroupement de parcelles pour créer des artefacts HTML uniques sophistiqués
avec une gestion étatique et des composants modernes. Fonctionne avec la pile React/TypeScript pour claude.ai
artefacts nécessitant une complexité au-delà du simple HTML/JSX.
