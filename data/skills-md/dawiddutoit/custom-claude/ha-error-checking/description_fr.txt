Débogue et valide les tableaux de bord Home Assistant en vérifiant les journaux système, en vérifiant les ID d'entité,
valider les installations de cartes HACS et analyser les erreurs de configuration via l'API WebSocket.

À utiliser pour dépanner les erreurs du tableau de bord, valider les ID d'entité, vérifier les installations de cartes HACS,
enquêter sur les problèmes de lovelace/frontend, ou déboguer "Erreur de durée ApexCharts", "entité introuvable",
"La carte personnalisée ne se charge pas" ou "Le tableau de bord n'apparaît pas dans la barre latérale".
