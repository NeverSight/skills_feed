Refactorise le code Java existant vers des modèles modernes et des meilleures pratiques. À utiliser lors de la modernisation de l'ancien code Java, de la conversion vers les fonctionnalités Java 8+, de la refactorisation d'applications héritées, de l'application de modèles de conception, de l'amélioration de la gestion des erreurs, de l'extraction de méthodes/classes, de la conversion en flux/facultatif/enregistrements ou de la migration à partir d'anciennes versions Java. Fonctionne avec le code pré-Java 8, le Java procédural, les frameworks hérités et les modèles obsolètes.
