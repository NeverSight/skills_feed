自律型ワークフロー用の公式 Python SDK を使用して、本番環境に対応した AI エージェントを実装します。
エージェント ワークフロー、SDK 統合、ワンショット クエリ、インタラクティブを実装するときに使用します。
会話、カスタム MCP ツール、権限制御、CLAUDE.md コンテキストの読み込み、エラー
処理、フック、またはエージェントの委任。 .py ファイルおよびエージェント アーキテクチャで動作します。
トリガー用語: 「Agent SDK」、「query()」、「ClaudeSDKClient」、「@tooldecorator」、
「setting_sources」、「エージェント権限」、「SDK フック」。
