Automatise les flux de travail du navigateur Chrome pour tester les applications Web, déboguer avec les journaux de console/réseau,
extraire des données, remplir des formulaires et interagir avec des applications Web authentifiées (Google Docs,
Gmail, Notion). À utiliser pour tester des applications Web locales, déboguer des problèmes d'interface, automatiser la saisie de données,
récupérer du contenu Web ou travailler avec des services authentifiés. Déclencheurs sur "tester mon application web",
"vérifier la console", "remplir ce formulaire", "extraire les données de [URL]", "automatiser [tâche du navigateur]",
"ouvrir [application authentifiée]" ou déboguer des problèmes d'application Web. Fonctionne avec Chrome via Claude dans
Extension Chrome (outils MCP : naviguer, cliquer, form_input, read_console_messages, read_network_requests,
tabs_context_mcp, gif_creator).
