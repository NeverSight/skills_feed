Web アプリのテスト、コンソール/ネットワーク ログによるデバッグのための Chrome ブラウザ ワークフローを自動化します。
データの抽出、フォームへの入力、認証された Web アプリケーション (Google ドキュメント、
Gmail、Notion）。ローカル Web アプリのテスト、フロントエンドの問題のデバッグ、データ入力の自動化、
Web コンテンツのスクレイピング、または認証されたサービスの操作。 「Web アプリをテストする」でトリガーされます。
「コンソールを確認する」、「このフォームに記入する」、「[URL] からデータを抽出する」、「[ブラウザ タスク] を自動化する」、
「[認証されたアプリ] を開きます」、または Web アプリケーションの問題をデバッグします。クロード経由で Chrome で動作します。
Chrome 拡張機能 (MCP ツール: navigate、click、form_input、read_console_messages、read_network_requests、
tabs_context_mcp、gif_creator)。
