Automatise les tests de validation des formulaires en soumettant des données invalides et en capturant les états d'erreur
en utilisant l'automatisation du navigateur Playwright. À utiliser lorsque vous avez besoin de « tester la validation du formulaire », de « vérifier
erreurs de validation", "tester les entrées invalides", "vérifier les messages d'erreur" ou effectuer des
tests de forme. Déclencheurs sur les demandes de tests de validation, les flux de travail d'assurance qualité ou les contrôles de conformité des formulaires.
Fonctionne avec les formulaires Web, les champs de saisie et les cadres de validation dans n'importe quelle application Web.
