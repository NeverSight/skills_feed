Genera codice CRUD in base alle specifiche del framework RuoYi. Utilizzare questa competenza quando l'utente deve generare codice Java backend (entità, mappatore, servizio, controller), codice Vue frontend (pagina elenco, wrapper API), file di mappatura XML MyBatis o SQL di inizializzazione del menu per una tabella dati.  
Scenari applicabili: creazione di nuovi moduli, aggiunta di funzionalità aziendali, generazione di codice da strutture di tabelle, ecc.
