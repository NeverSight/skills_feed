Genera codice CRUD seguendo le convenzioni del framework RuoYi. Utilizzare questa competenza quando l'utente necessita di codice Java backend (entità, mappatore, servizio, controller), codice Vue frontend (pagina elenco, wrapper API), mappatori XML MyBatis o SQL di inizializzazione del menu per una tabella di database.  
Applicabile per: creazione di nuovi moduli, aggiunta di funzionalità aziendali, generazione di codice da strutture di tabelle, ecc.
