Сгенерируйте CRUD-код в соответствии с соглашениями платформы RuoYi. Используйте этот навык, когда пользователю нужен внутренний код Java (объект, Mapper, Сервис, Контроллер), внешний код Vue (страница списка, оболочка API), XML-сопоставители MyBatis или SQL инициализации меню для таблицы базы данных.  
Применимо для: создания новых модулей, добавления бизнес-функций, генерации кода из табличных структур и т. д.
