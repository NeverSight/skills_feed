قم بإنشاء كود CRUD بناءً على مواصفات إطار عمل RuoYi. استخدم هذه المهارة عندما يحتاج المستخدم إلى إنشاء كود Java للواجهة الخلفية (كيان، مخطط، خدمة، وحدة تحكم)، أو كود Vue للواجهة الأمامية (صفحة القائمة، غلاف واجهة برمجة التطبيقات)، أو ملفات تعيين MyBatis XML، أو تهيئة القائمة SQL لجدول البيانات.  
السيناريوهات القابلة للتطبيق: إنشاء وحدات نمطية جديدة، وإضافة ميزات الأعمال، وإنشاء التعليمات البرمجية من هياكل الجدول، وما إلى ذلك.
