قم بإنشاء كود CRUD باتباع اتفاقيات إطار عمل RuoYi. استخدم هذه المهارة عندما يحتاج المستخدم إلى كود Java الخلفي (كيان، مخطط، خدمة، وحدة تحكم)، كود Vue للواجهة الأمامية (صفحة القائمة، غلاف واجهة برمجة التطبيقات)، مخططات MyBatis XML، أو تهيئة القائمة SQL لجدول قاعدة البيانات.  
ينطبق على: إنشاء وحدات جديدة، وإضافة ميزات الأعمال، وإنشاء التعليمات البرمجية من هياكل الجدول، وما إلى ذلك.
