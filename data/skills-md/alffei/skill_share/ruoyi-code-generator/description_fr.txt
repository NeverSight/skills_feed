Générez du code CRUD basé sur les spécifications du framework RuoYi. Utilisez cette compétence lorsque l'utilisateur a besoin de générer du code Java back-end (entité, mappeur, service, contrôleur), du code Vue front-end (page de liste, wrapper API), des fichiers de mappage XML MyBatis ou un menu SQL d'initialisation pour une table de données.  
Scénarios applicables : création de nouveaux modules, ajout de fonctionnalités métiers, génération de code à partir de structures de tables, etc.
