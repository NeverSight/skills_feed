Générez du code CRUD en suivant les conventions du framework RuoYi. Utilisez cette compétence lorsque l'utilisateur a besoin de code Java backend (entité, mappeur, service, contrôleur), de code Vue frontend (page de liste, wrapper API), de mappeurs XML MyBatis ou d'initialisation de menu SQL pour une table de base de données.  
Applicable pour : créer de nouveaux modules, ajouter des fonctionnalités métier, générer du code à partir de structures de tables, etc.
