يدير مشكلات YouTrack والمشاريع والمستخدمين عبر yt CLI. يُستخدم عند البحث عن مشكلات YouTrack أو إنشائها أو تحديثها أو التعليق عليها، أو إدراج المشاريع، أو التحقق من معلومات المستخدم المصادق عليها. تشغيل عبارات مثل "مشكلة YouTrack"، و"إنشاء تذكرة"، و"بحث المشكلات"، و"تحديث المشكلة"، و"إضافة تعليق إلى"، و"قائمة المشاريع".
