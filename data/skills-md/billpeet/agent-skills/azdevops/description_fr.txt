Gère les projets Azure DevOps, les dépôts, les branches, les demandes d'extraction, les pipelines et les éléments de travail via la CLI azdevops. À utiliser lors de la recherche, de la création, de la mise à jour d'éléments de travail, de la gestion des PR, du déclenchement de pipelines ou de la liste des dépôts/branches. Déclencheurs sur des expressions telles que « Azure DevOps », « élément de travail », « pull request », « pipeline », « créer un bug », « list repos », « trigger build ».
