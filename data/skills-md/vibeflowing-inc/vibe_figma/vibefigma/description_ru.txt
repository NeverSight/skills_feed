Преобразуйте проекты Figma в готовые к использованию компоненты React с помощью Tailwind CSS. Используйте, когда пользователь предоставляет URL-адрес Figma, просит преобразовать проекты Figma в React/код, хочет извлечь компоненты из Figma или упоминает «vibefigma». Требуется токен доступа Figma (через флаг --token, переменную среды FigMA_TOKEN или файл .env).
