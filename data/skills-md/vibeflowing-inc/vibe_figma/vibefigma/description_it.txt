Converti i progetti Figma in componenti React pronti per la produzione con Tailwind CSS. Utilizzare quando l'utente fornisce un URL Figma, chiede di convertire i progetti Figma in React/codice, desidera estrarre componenti da Figma o menziona "vibefigma". Richiede un token di accesso Figma (tramite flag --token, FIGMA_TOKEN env var o file .env).
