Konvertieren Sie Figma-Designs mit Tailwind CSS in produktionsbereite React-Komponenten. Verwenden Sie diese Option, wenn der Benutzer eine Figma-URL angibt, Figma-Designs in React/Code konvertieren möchte, Komponenten aus Figma extrahieren möchte oder „vibefigma“ erwähnt. Erfordert ein Figma-Zugriffstoken (über das Flag --token, FIGMA_TOKEN env var oder .env-Datei).
