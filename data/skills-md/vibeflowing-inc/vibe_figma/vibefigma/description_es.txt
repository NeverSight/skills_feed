Convierta diseños de Figma en componentes React listos para producción con Tailwind CSS. Úselo cuando el usuario proporciona una URL de Figma, solicita convertir diseños de Figma a React/código, quiere extraer componentes de Figma o menciona "vibefigma". Requiere un token de acceso Figma (a través del indicador --token, FIGMA_TOKEN env var o archivo .env).
