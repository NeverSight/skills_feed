Создает, запускает и отслеживает тесты E2E через API ProxyUser. Используйте при реализации функций, требующих покрытия тестированием E2E, когда тесты терпят неудачу и требуется диагностика, при запуске тестов для предварительных развертываний или при настройке автоматического контроля качества для веб-приложения.
