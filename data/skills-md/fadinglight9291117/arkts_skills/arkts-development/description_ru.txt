Разработка приложений HarmonyOS ArkTS с использованием декларативной инфраструктуры пользовательского интерфейса ArkUI. Используйте при создании приложений HarmonyOS/OpenHarmony, создании компонентов ArkUI, реализации управления состоянием с помощью декораторов (@State, @Prop, @Link), переходе с TypeScript на ArkTS или работе с API-интерфейсами, специфичными для HarmonyOS (маршрутизатор, http, настройки). Охватывает жизненный цикл компонентов, шаблоны компоновки и ограничения языка ArkTS.
