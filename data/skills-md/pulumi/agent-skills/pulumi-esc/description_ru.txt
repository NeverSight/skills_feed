Руководство по работе с Pulumi ESC (Среда, секреты и конфигурация). Используйте, когда пользователи спрашивают об управлении секретами, конфигурацией, средами, краткосрочными учетными данными, настройке OIDC для AWS, Azure, GCP, интеграции с хранилищами секретов (AWS Secrets Manager, Azure Key Vault, HashiCorp Vault, 1Password) или использовании ESC со стеками Pulumi.
