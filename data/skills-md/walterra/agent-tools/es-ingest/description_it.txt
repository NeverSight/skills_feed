Inserisci e trasforma file di dati di grandi dimensioni (CSV/JSON) in indici Elasticsearch. Elaborazione basata sul flusso per file fino a 30 GB, migrazione tra versioni (ES 8.x ↔ 9.x), trasformazioni JavaScript personalizzate e reindicizzazione con trasformazioni. Da utilizzare quando è necessario caricare dati in Elasticsearch, migrare indici o trasformare dati durante l'acquisizione.
