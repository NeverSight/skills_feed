Ingérez et transformez des fichiers de données volumineux (CSV/JSON) en index Elasticsearch. Traitement basé sur le flux pour les fichiers jusqu'à 30 Go, migration entre versions (ES 8.x ↔ 9.x), transformations JavaScript personnalisées et réindexation avec transformations. À utiliser lorsque vous devez charger des données dans Elasticsearch, migrer des index ou transformer des données lors de l'ingestion.
