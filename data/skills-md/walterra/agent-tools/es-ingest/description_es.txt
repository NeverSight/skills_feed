Ingiere y transforma archivos de datos de gran tamaño (CSV/JSON) en índices de Elasticsearch. Procesamiento basado en secuencias para archivos de hasta 30 GB, migración entre versiones (ES 8.x ↔ 9.x), transformaciones JavaScript personalizadas y reindexación con transformaciones. Úselo cuando necesite cargar datos en Elasticsearch, migrar índices o transformar datos durante la ingesta.
