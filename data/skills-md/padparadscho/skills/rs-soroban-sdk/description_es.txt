Orientación de expertos para crear contratos inteligentes en Stellar utilizando el SDK de Soroban Rust. Utilice esta habilidad cuando trabaje con contratos inteligentes de Soroban para tareas que incluyen (1) crear nuevos contratos con atributos [contrato] y [contractimpl], (2) implementar almacenamiento con tipos de almacenamiento persistente, temporal o de instancia, (3) trabajar con contextos de autenticación y autorización, (4) manejar tokens y contratos de activos estelares, (5) escribir pruebas con testutils, (6) implementar contratos, (7) trabajar con eventos y registros, (8) usar funciones criptográficas, (9) depurar errores de contrato, (10) mejores prácticas de seguridad y prevención de vulnerabilidades, (11) evitar errores de seguridad comunes como falta de autorización, desbordamiento de enteros o ataques de reinicialización.
