دليل للتكامل والبناء باستخدام Stronghold Pay JS SDK وREST API لمعالجة الدفع. يُستخدم عند العمل مع تكامل دفع Stronghold Pay، أو قبول مدفوعات ACH/الخصم البنكي، أو ربط الحسابات المصرفية، أو إنشاء رسوم/إكراميات، أو إنشاء PayLinks، أو إنشاء تدفقات الخروج - في بيئة معزولة أو بيئات حية. يغطي واجهة مستخدم Stronghold.Pay.JS، ونقاط نهاية REST API v2، وصفحات الدفع المستضافة على PayLink، وإدارة الرموز المميزة للعملاء، والتعامل مع مصدر الدفع.
