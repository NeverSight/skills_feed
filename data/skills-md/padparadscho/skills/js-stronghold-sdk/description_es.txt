Guía para integrar y construir con Stronghold Pay JS SDK y REST API para el procesamiento de pagos. Úselo cuando trabaje con la integración de pagos de Stronghold Pay, acepte pagos de débito bancario/ACH, vincule cuentas bancarias, cree cargos/propinas, genere PayLinks o cree flujos de pago, en entornos sandbox o en vivo. Cubre la interfaz de usuario directa de Stronghold.Pay.JS, los puntos finales de REST API v2, las páginas de pago alojadas en PayLink, la administración de tokens de clientes y el manejo de fuentes de pago.
