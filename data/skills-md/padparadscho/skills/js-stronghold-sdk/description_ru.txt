Руководство по интеграции и созданию Stronghold Pay JS SDK и REST API для обработки платежей. Используйте при работе с интеграцией платежей Stronghold Pay, приеме дебетовых платежей ACH/банка, связывании банковских счетов, создании начислений/чаевых, создании ссылок PayLink или построении потоков оформления заказа — в изолированной или реальной среде. Охватывает всплывающий пользовательский интерфейс Stronghold.Pay.JS, конечные точки REST API v2, страницы оплаты, размещенные в PayLink, управление токенами клиентов и обработку источников платежей.
