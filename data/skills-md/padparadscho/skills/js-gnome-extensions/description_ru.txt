Создавайте, отлаживайте и обслуживайте расширения GNOME Shell с помощью GJS (GNOME JavaScript). Охватывает анатомию расширений (metadata.json, Extension.js, prefs.js, stylesheet.css), импорт ESModule, настройки GSettings, всплывающие меню, быстрые настройки, индикаторы панели, диалоговые окна, уведомления, поставщики поиска, переводы и режимы сеанса. Используйте, когда пользователь хочет: (1) Создать новое расширение GNOME Shell, (2) Добавить элементы пользовательского интерфейса, такие как кнопки панели, всплывающие меню, переключатели/ползунки быстрых настроек или модальные диалоги, (3) Реализовать настройки расширения с помощью GTK4/Adwaita, (4) Отладить или протестировать расширение, (5) Перенести расширение в более новую версию GNOME Shell (45–49+), (6) Подготовить расширение для отправки в Extensions.gnome.org, (7) Работа с внутренними API оболочки GNOME (Clutter, St, Meta, Shell, Main).
