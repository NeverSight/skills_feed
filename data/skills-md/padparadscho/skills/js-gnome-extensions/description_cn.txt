使用 GJS (GNOME JavaScript) 构建、调试和维护 GNOME Shell 扩展。涵盖扩展剖析（metadata.json、extension.js、prefs.js、stylesheet.css）、ESModule 导入、GSettings 首选项、弹出菜单、快速设置、面板指示器、对话框、通知、搜索提供程序、翻译和会话模式。当用户想要执行以下操作时使用：(1) 创建新的 GNOME Shell 扩展，(2) 添加 UI 元素，如面板按钮、弹出菜单、快速设置切换/滑块或模式对话框，(3) 使用 GTK4/Adwaita 实现扩展首选项，(4) 调试或测试扩展，(5) 将扩展移植到较新的 GNOME Shell 版本 (45-49+)，(6) 准备扩展以提交到 extensions.gnome.org， (7) 使用 GNOME Shell 内部 API（Clutter、St、Meta、Shell、Main）。
