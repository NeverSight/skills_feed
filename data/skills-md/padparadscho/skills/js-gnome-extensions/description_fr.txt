Créez, déboguez et maintenez les extensions GNOME Shell à l'aide de GJS (GNOME JavaScript). Couvre l'anatomie des extensions (metadata.json, extension.js, prefs.js, stylesheet.css), les importations ESModule, les préférences GSettings, les menus contextuels, les paramètres rapides, les indicateurs de panneau, les boîtes de dialogue, les notifications, les fournisseurs de recherche, les traductions et les modes de session. À utiliser lorsque l'utilisateur souhaite : (1) créer une nouvelle extension GNOME Shell, (2) ajouter des éléments d'interface utilisateur tels que des boutons de panneau, des menus contextuels, des bascules/curseurs de paramètres rapides ou des boîtes de dialogue modales, (3) implémenter des préférences d'extension avec GTK4/Adwaita, (4) déboguer ou tester une extension, (5) porter une extension vers une version plus récente de GNOME Shell (45-49+), (6) préparer une extension à soumettre à extensions.gnome.org, (7) Travaillez avec les API internes de GNOME Shell (Clutter, St, Meta, Shell, Main).
