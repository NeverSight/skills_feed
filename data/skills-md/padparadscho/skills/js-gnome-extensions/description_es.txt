Cree, depure y mantenga extensiones de GNOME Shell utilizando GJS (GNOME JavaScript). Cubre la anatomía de la extensión (metadata.json, extension.js, prefs.js, stylesheet.css), importaciones de ESModule, preferencias de GSettings, menús emergentes, configuraciones rápidas, indicadores de panel, cuadros de diálogo, notificaciones, proveedores de búsqueda, traducciones y modos de sesión. Úselo cuando el usuario quiera: (1) Crear una nueva extensión de GNOME Shell, (2) Agregar elementos de la interfaz de usuario como botones de panel, menús emergentes, controles deslizantes/configuraciones rápidas o cuadros de diálogo modales, (3) Implementar preferencias de extensión con GTK4/Adwaita, (4) Depurar o probar una extensión, (5) Portar una extensión a una versión más reciente de GNOME Shell (45-49+), (6) Preparar una extensión para enviarla a extensions.gnome.org, (7) Trabajar con las API internas de GNOME Shell (Clutter, St, Meta, Shell, Main).
