GJS (GNOME JavaScript) を使用して、GNOME Shell 拡張機能を構築、デバッグ、保守します。拡張機能の構造 (metadata.json、extension.js、prefs.js、stylesheet.css)、ESModule インポート、GSettings 設定、ポップアップ メニュー、クイック設定、パネル インジケーター、ダイアログ、通知、検索プロバイダー、翻訳、およびセッション モードについて説明します。ユーザーが次のことを行う場合に使用します: (1) 新しい GNOME Shell 拡張機能を作成する、(2) パネル ボタン、ポップアップ メニュー、クイック設定トグル/スライダー、モーダル ダイアログなどの UI 要素を追加する、(3) GTK4/Adwaita を使用して拡張機能設定を実装する、(4) 拡張機能をデバッグまたはテストする、(5) 拡張機能を新しい GNOME Shell バージョン (45 ～ 49 以降) に移植する、(6) 拡張機能を送信する準備をするextensions.gnome.org、(7) GNOME Shell 内部 API (Clutter、St、Meta、Shell、Main) を使用します。
