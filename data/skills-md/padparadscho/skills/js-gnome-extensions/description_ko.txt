GJS(GNOME JavaScript)를 사용하여 GNOME 셸 확장을 빌드, 디버그 및 유지 관리합니다. 확장 기능 구조(metadata.json, Extension.js, prefs.js, stylesheet.css), ESModule 가져오기, GSettings 환경 설정, 팝업 메뉴, 빠른 설정, 패널 표시기, 대화 상자, 알림, 검색 공급자, 번역 및 세션 모드를 다룹니다. 사용자가 원할 때 사용합니다: (1) 새로운 GNOME 셸 확장 생성, (2) 패널 버튼, 팝업 메뉴, 빠른 설정 토글/슬라이더 또는 모달 대화 상자와 같은 UI 요소 추가, (3) GTK4/Adwaita를 사용하여 확장 기본 설정 구현, (4) 확장 디버그 또는 테스트, (5) 최신 GNOME 셸 버전(45-49+)으로 확장 포팅, (6) Extensions.gnome.org에 제출할 확장 준비, (7) GNOME Shell 내부 API(Clutter, St, Meta, Shell, Main)로 작업합니다.
