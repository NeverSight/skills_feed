Crea, esegui il debug e mantieni le estensioni della shell GNOME utilizzando GJS (JavaScript GNOME). Copre l'anatomia delle estensioni (metadata.json, extension.js, prefs.js, stylesheet.css), importazioni ESModule, preferenze GSettings, menu popup, impostazioni rapide, indicatori di pannello, finestre di dialogo, notifiche, provider di ricerca, traduzioni e modalità di sessione. Da utilizzare quando l'utente desidera: (1) creare una nuova estensione della shell GNOME, (2) aggiungere elementi dell'interfaccia utente come pulsanti del pannello, menu a comparsa, cursori/attivatori delle impostazioni rapide o finestre di dialogo modali, (3) implementare le preferenze dell'estensione con GTK4/Adwaita, (4) eseguire il debug o testare un'estensione, (5) trasferire un'estensione a una versione più recente della shell GNOME (45-49+), (6) preparare un'estensione per l'invio a extensions.gnome.org, (7) Lavora con le API interne di GNOME Shell (Clutter, St, Meta, Shell, Main).
