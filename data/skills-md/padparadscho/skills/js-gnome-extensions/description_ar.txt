إنشاء ملحقات GNOME Shell وتصحيح أخطائها وصيانتها باستخدام GJS (GNOME JavaScript). يغطي تشريح الامتداد (metadata.json، وextension.js، وprefs.js، وstylesheet.css)، واستيرادات ESModule، وتفضيلات GSettings، والقوائم المنبثقة، والإعدادات السريعة، ومؤشرات اللوحة، ومربعات الحوار، والإشعارات، وموفري البحث، والترجمات، وأوضاع الجلسة. يُستخدم عندما يريد المستخدم: (1) إنشاء ملحق GNOME Shell جديد، (2) إضافة عناصر واجهة المستخدم مثل أزرار اللوحة، أو القوائم المنبثقة، أو مفاتيح التبديل/المنزلقات السريعة للإعدادات، أو مربعات الحوار المشروطة، (3) تنفيذ تفضيلات الامتداد باستخدام GTK4/Adwaita، (4) تصحيح الأخطاء أو اختبار الامتداد، (5) منفذ الامتداد إلى إصدار GNOME Shell أحدث (45-49+)، (6) إعداد ملحق لتقديمه إلى الامتدادات.gnome.org، (7) العمل مع واجهات برمجة التطبيقات الداخلية لـ GNOME Shell (Clutter، St، Meta، Shell، Main).
