使用 JavaScript (GJS) 與 GTK 4、Libadwaita 和 GNOME 平台建立本機 GNOME 桌面應用程式。當使用者想要建立、修改或偵錯用 JavaScript/GJS 編寫的 GNOME 應用程式時使用，包括使用 XML 或藍圖進行 UI 設計、GObject 子類化、Meson 建置設定、Flatpak 打包或涉及 GLib/GIO/GTK4/Adw 庫的 GJS 綁定的任何任務。在 GJS 專案上下文中處理「.ui」檔案、「meson.build」、GResource XML、GSettings 架構、「.desktop」檔案或 Flatpak 清單時也可使用。
