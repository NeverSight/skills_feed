使用 JavaScript (GJS) 与 GTK 4、Libadwaita 和 GNOME 平台构建本机 GNOME 桌面应用程序。当用户想要创建、修改或调试用 JavaScript/GJS 编写的 GNOME 应用程序时使用，包括使用 XML 或蓝图进行 UI 设计、GObject 子类化、Meson 构建设置、Flatpak 打包或涉及 GLib/GIO/GTK4/Adw 库的 GJS 绑定的任何任务。在 GJS 项目上下文中处理“.ui”文件、“meson.build”、GResource XML、GSettings 架构、“.desktop”文件或 Flatpak 清单时也可使用。
