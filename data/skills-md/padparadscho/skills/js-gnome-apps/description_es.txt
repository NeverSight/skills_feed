Cree aplicaciones de escritorio GNOME nativas utilizando JavaScript (GJS) con GTK 4, Libadwaita y la plataforma GNOME. Úselo cuando el usuario desee crear, modificar o depurar una aplicación GNOME escrita en JavaScript/GJS, incluido el diseño de interfaz de usuario con XML o Blueprint, subclases de GObject, configuración de compilación de Meson, empaquetado Flatpak o cualquier tarea que involucre enlaces GJS para bibliotecas GLib/GIO/GTK4/Adw. Utilícelo también cuando trabaje con archivos `.ui`, `meson.build`, GResource XML, esquemas GSettings, archivos `.desktop` o manifiestos Flatpak en el contexto de un proyecto GJS.
