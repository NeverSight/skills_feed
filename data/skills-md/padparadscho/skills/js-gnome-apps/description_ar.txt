أنشئ تطبيقات سطح مكتب GNOME أصلية باستخدام JavaScript (GJS) مع GTK 4 وLibadwaita ومنصة GNOME. يُستخدم عندما يريد المستخدم إنشاء أو تعديل أو تصحيح تطبيق GNOME مكتوب بلغة JavaScript/GJS، بما في ذلك تصميم واجهة المستخدم باستخدام XML أو Blueprint أو فئة GObject الفرعية أو إعداد Meson build أو تعبئة Flatpak أو أي مهمة تتضمن روابط GJS لمكتبات GLib/GIO/GTK4/Adw. استخدمه أيضًا عند العمل مع ملفات `.ui`، أو `meson.build`، أو GResource XML، أو مخططات GSettings، أو ملفات `.desktop`، أو بيانات Flatpak في سياق مشروع GJS.
