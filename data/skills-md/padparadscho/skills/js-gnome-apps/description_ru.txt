Создавайте собственные настольные приложения GNOME, используя JavaScript (GJS) с GTK 4, Libadwaita и платформой GNOME. Используйте, когда пользователь хочет создать, изменить или отладить приложение GNOME, написанное на JavaScript/GJS, включая дизайн пользовательского интерфейса с помощью XML или Blueprint, создание подклассов GObject, настройку сборки Meson, упаковку Flatpak или любую задачу, включающую привязки GJS для библиотек GLib/GIO/GTK4/Adw. Также используйте при работе с файлами .ui, meson.build, GResource XML, схемами GSettings, файлами .desktop или манифестами Flatpak в контексте проекта GJS.
