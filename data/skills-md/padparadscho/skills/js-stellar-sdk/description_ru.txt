Руководство по созданию приложений с помощью Stellar JS SDK (@stellar/stellar-sdk). Используйте при работе с блокчейном Stellar в JavaScript/TypeScript, включая отправку платежей, создание учетных записей, выпуск активов, управление линиями доверия, торговлю на DEX, запросы к Horizon, взаимодействие со Stellar RPC, потоковую передачу событий, создание и подписание транзакций, мультиподпись, заявленные балансы, спонсируемые резервы, аутентификацию SEP-10, федерацию, транзакции с дополнительными сборами и взаимодействие со смарт-контрактами Soroban через JS SDK. Охватывает все шаблоны использования @stellar/stellar-sdk (модуль Horizon, модуль rpc, модуль контракта, TransactionBuilder, пара ключей, операция, актив и т. д.).
