Leitfaden zum Erstellen von Anwendungen mit dem Stellar JS SDK (@stellar/stellar-sdk). Verwendung bei der Arbeit mit der Stellar-Blockchain in JavaScript/TypeScript – einschließlich Senden von Zahlungen, Erstellen von Konten, Ausgeben von Vermögenswerten, Verwalten von Trustlines, Handeln am DEX, Abfragen von Horizon, Interagieren mit Stellar RPC, Streaming-Ereignisse, Erstellen und Signieren von Transaktionen, Multisig, beanspruchbare Salden, gesponserte Reserven, SEP-10-Authentifizierung, Föderation, Gebührenerhöhungstransaktionen und Interagieren mit Soroban-Smart-Verträgen über das JS SDK. Deckt alle @stellar/stellar-sdk-Nutzungsmuster ab (Horizon-Modul, RPC-Modul, Vertragsmodul, TransactionBuilder, Keypair, Operation, Asset usw.).
