Guide pour créer des applications avec le SDK Stellar JS (@stellar/stellar-sdk). À utiliser lorsque vous travaillez avec la blockchain Stellar en JavaScript/TypeScript — y compris l'envoi de paiements, la création de comptes, l'émission d'actifs, la gestion des lignes de confiance, la négociation sur DEX, l'interrogation d'Horizon, l'interaction avec Stellar RPC, la diffusion d'événements, la création et la signature de transactions, le multisig, les soldes réclamables, les réserves sponsorisées, l'authentification SEP-10, la fédération, les transactions à frais majorés et l'interaction avec les contrats intelligents Soroban via le SDK JS. Couvre tous les modèles d'utilisation @stellar/stellar-sdk (module Horizon, module rpc, module de contrat, TransactionBuilder, Keypair, Operation, Asset, etc.).
