Используйте при подключении к событиям жизненного цикла Elementor, внедрении элементов управления, фильтрации вывода виджетов или использовании API JS. Охватывает elementor/init, elementor/element/before_section_end, elementor/element/after_section_end, elementor/widget/render_content filter, elementor/frontend/after_enqueue_styles, JS-хуки внешнего интерфейса (elementorFrontend.hooks, frontend/element_ready), JS-хуки редактора (elementor.hooks), $e.commands API ($e.run, $e.commands.register), $e.routes, $e.hooks (registerUIBefore, RegisterUIAfter), шаблоны внедрения элементов управления, перехватчики файлов CSS, перехватчики форм (Pro) и фильтры запросов.
