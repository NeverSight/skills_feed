Configura il file system iniziale per una nuova sandbox. Recupera le organizzazioni e gli artisti dell'account tramite la CLI di Recoup e struttura una struttura di cartelle supponente. Da utilizzare quando una sandbox è stata appena creata e non ha un file system esistente. Prima dell'esecuzione, controlla se la sandbox dispone già di una directory org/: in caso affermativo, questa competenza non è necessaria.
