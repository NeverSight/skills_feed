Управляйте контейнерами Docker PostgreSQL для локальной разработки с помощью команд docker pgpm. Используйте, когда вас просят «запустить postgres», «локально запустить базу данных», «настроить локальную базу данных», «остановить контейнер postgres» или когда вам нужна база данных PostgreSQL для разработки/тестирования.
