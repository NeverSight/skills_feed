Gérez les conteneurs PostgreSQL Docker pour le développement local à l'aide des commandes pgpm docker. À utiliser lorsqu'on vous demande de "démarrer Postgres", "exécuter la base de données localement", "configurer la base de données locale", "arrêter le conteneur Postgres" ou lorsque vous avez besoin d'une base de données PostgreSQL pour le développement/les tests.
