Gérez les extensions PostgreSQL et les modules pgpm dans un espace de travail pgpm. À utiliser lorsqu'on vous demande "ajouter une extension", "installer un module", "ajouter une dépendance", "utiliser pgcrypto/uuid-ossp", ou lorsque vous travaillez avec des fichiers .control et le répertoire extensions/. CRITIQUE — n'exécutez jamais CREATE EXTENSION directement ; pgpm gère la création d'extensions pendant le déploiement.
