تنفيذ استعلامات GraphQL مقابل واجهات برمجة التطبيقات البناءة باستخدام CLI cnc. يُستخدم عندما يُطلب منك "تشغيل استعلام" أو "تنفيذ GraphQL" أو "إعداد سياق واجهة برمجة التطبيقات" أو "تكوين الرمز المميز لواجهة برمجة التطبيقات" أو "إدارة نقاط نهاية واجهة برمجة التطبيقات" أو عند العمل مع واجهات برمجة تطبيقات GraphQL البناءة.
