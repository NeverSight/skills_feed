Exécutez des tests d'intégration PostgreSQL avec des bases de données isolées à l'aide de pgsql-test. À utiliser lorsqu'on vous demande « d'exécuter des tests de base de données », « de configurer une base de données de test », « d'écrire des tests d'intégration », de « tester des modules PGPM » ou lors de l'implémentation de tests nécessitant PostgreSQL.
