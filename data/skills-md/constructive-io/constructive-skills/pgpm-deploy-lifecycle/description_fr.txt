Cycle de vie complet des déploiements de bases de données pgpm : déploiement, vérification, restauration, marquage et vérification de l'état. À utiliser lorsqu'on vous demande de « déployer un module », « annuler une migration », « vérifier les déploiements », « marquer une version », « vérifier l'état du déploiement » ou lors de l'exécution de commandes pgpm déployer/annuler/vérifier.
