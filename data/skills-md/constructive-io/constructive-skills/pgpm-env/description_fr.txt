Gérez les variables d'environnement PostgreSQL avec prise en charge des profils à l'aide des commandes pgpm env. À utiliser lorsqu'on vous demande de « configurer l'environnement de base de données », « charger les variables d'environnement postgres », « exécuter la commande avec une connexion à la base de données », « utiliser supabase localement » ou lorsque vous devez configurer la connexion à la base de données pour les commandes.
