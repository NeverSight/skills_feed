Générez des hooks React Query de type sécurisé, un client ORM de type Prisma ou une CLI basée sur un demandeur à partir de points de terminaison GraphQL, de fichiers/répertoires de schéma, de bases de données ou de modules PGPM à l'aide de @constructive-io/graphql-codegen. Génère également de la documentation (README, AGENTS.md, skills/, mcp.json). À utiliser lorsqu'on vous demande de « générer des hooks GraphQL », « générer ORM », « générer CLI », « configurer codegen », « générer des documents », « générer des compétences », « exporter un schéma » ou lors de l'implémentation de la récupération de données pour un backend PostGraphile.
