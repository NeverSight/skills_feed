قم بتكوين وتشغيل خادم Constructive GraphQL (خادم CNC)، ومستكشف GraphiQL (مستكشف CNC)، وإنشاء التعليمات البرمجية (CNC Codegen). يُستخدم عندما يُطلب منك "بدء تشغيل الخادم"، أو "تشغيل خادم cnc"، أو "بدء تشغيل GraphQL API"، أو "تشغيل GraphiQL"، أو "تكوين توجيه واجهة برمجة التطبيقات"، أو "إنشاء الأنواع"، أو عند العمل مع Constructive CLI وPostGraphile.
