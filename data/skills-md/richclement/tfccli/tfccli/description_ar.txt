استخدم tfccli CLI لعمليات Terraform Cloud API: مساحات العمل، والتشغيل، والخطط، والتطبيقات، والمؤسسات، والمشاريع، وإصدارات التكوين، والمتغيرات وأي استعلامات أو أوامر أخرى تريد تنفيذها ضد Terraform Cloud. يتم التشغيل عندما يذكر المستخدم Terraform Cloud، أو TFC، أو HCP Terraform، أو يحتاج إلى إدارة موارد Terraform Cloud برمجيًا.
