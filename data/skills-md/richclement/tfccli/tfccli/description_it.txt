Utilizza la CLI tfccli per le operazioni API Terraform Cloud: spazi di lavoro, esecuzioni, piani, applicazioni, organizzazioni, progetti, versioni di configurazione, variabili e qualsiasi altra query o comando che desideri eseguire su Terraform Cloud. Si attiva quando l'utente menziona Terraform Cloud, TFC, HCP Terraform o deve gestire le risorse Terraform Cloud a livello di codice.
