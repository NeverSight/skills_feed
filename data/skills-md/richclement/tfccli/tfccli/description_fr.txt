Utilisez tfccli CLI pour les opérations de l'API Terraform Cloud : espaces de travail, exécutions, plans, applications, organisations, projets, versions de configuration, variables et toute autre requête ou commande que vous souhaitez exécuter sur Terraform Cloud. Déclenchez lorsque l'utilisateur mentionne Terraform Cloud, TFC, HCP Terraform ou doit gérer les ressources Terraform Cloud par programme.
