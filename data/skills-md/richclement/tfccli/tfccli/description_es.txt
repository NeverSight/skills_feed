Utilice tfccli CLI para las operaciones de la API de Terraform Cloud: espacios de trabajo, ejecuciones, planes, aplicaciones, organizaciones, proyectos, versiones de configuración, variables y cualquier otra consulta o comando que desee ejecutar en Terraform Cloud. Se activa cuando el usuario menciona Terraform Cloud, TFC, HCP Terraform o necesita administrar los recursos de Terraform Cloud mediante programación.
