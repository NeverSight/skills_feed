Mejores prácticas para automatizaciones, ayudantes, scripts y controles de dispositivos de Home Assistant.
DISPARE ESTA HABILIDAD CUANDO: - Cree o edite automatizaciones, scripts o escenas de HA - Escriba condiciones o activadores de automatización - Cree sensores de plantilla o sensores binarios - Utilice wait_template o wait_for_trigger - Elija entre activadores de dispositivo y activadores de estado - Seleccione modos de automatización (simple/reinicio/en cola/paralelo) - Configure el botón Zigbee/automatizaciones remotas (ZHA o Zigbee2MQTT) - Decida si usar un ayudante incorporado vs sensor de plantilla
SÍNTOMAS QUE ACTIVAN ESTA HABILIDAD: - El agente utiliza de manera predeterminada las plantillas Jinja2 donde existen construcciones nativas - El agente crea sensores de plantilla para una agregación simple (suma/mín/máx/media) - El agente usa condiciones de plantilla en lugar de condiciones de estado nativo/estado numérico - El agente usa plantilla de espera en lugar de espera_para_trigger - El agente usa id de dispositivo en lugar de id de entidad en activadores/acciones - El agente elige el modo de automatización incorrecto (por ejemplo, único para luces de movimiento)
