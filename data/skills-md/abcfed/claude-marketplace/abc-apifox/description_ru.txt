Инструмент запроса документации ABC Medical Cloud API. Считывает и запрашивает документы спецификации OpenAPI ABC API (более 5000 конечных точек), поддерживает поиск по модулю, пути и методу, а также автоматически разрешает ссылки $ref. Использует структуру кэширования с разделением модулей для быстрых запросов. Варианты использования: (1) Определения конечных точек API запросов (2) Поиск конечных точек конкретных функций (3) Определения схемы запроса (4) Просмотр статистики конечных точек
