Инструмент запроса документации ABC Medical Cloud API. Считывает и запрашивает документ спецификации ABC API OpenAPI (4209 интерфейсов), поддерживает поиск по модулю, пути и методу, а также автоматически разрешает ссылки $ref. Варианты использования: (1) Запрос определений интерфейса API (2) Поиск конкретных функциональных интерфейсов (3) Экспорт сводной документации интерфейса (4) Просмотр статистики интерфейса
