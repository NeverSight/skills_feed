Strumento di query sulla documentazione API di ABC Medical Cloud. Legge ed esegue query sul documento delle specifiche ABC API OpenAPI (4.209 interfacce), supporta la ricerca per modulo, percorso e metodo e risolve automaticamente i riferimenti $ref. Casi d'uso: (1) Query sulle definizioni dell'interfaccia API (2) Ricerca di interfacce funzionali specifiche (3) Esportazione di riepiloghi della documentazione dell'interfaccia (4) Visualizzazione delle statistiche dell'interfaccia
