Intégration de la plateforme de gestion de R&D agile TAPD. Utilisez des scripts pour appeler l'API TAPD afin de gérer des entités telles que les exigences, les défauts, les tâches, les itérations, les scénarios de test et le wiki. Les cas d'utilisation incluent : (1) interroger/créer/mettre à jour les exigences, les défauts, les tâches, les itérations (2) gérer les cas de test et le wiki (3) gérer les commentaires et les journaux de travail (4) lier les exigences et les défauts (5) obtenir les mots-clés de validation du code source
