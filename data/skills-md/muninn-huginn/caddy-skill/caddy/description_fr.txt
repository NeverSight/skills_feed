Gérez les proxys de développement locaux à l'aide de l'API d'administration du serveur Caddy. À utiliser lorsque les utilisateurs doivent allouer des ports localhost, créer des proxys inverses pour les serveurs de développement locaux, suivre les ports utilisés dans les sessions Claude Code ou accéder aux applications via des URL conviviales telles que myapp.localhost. Déclencheurs sur « configurer un proxy », « allouer un port », « ce qui s'exécute sur localhost », « enregistrer mon application » ou toute tâche de configuration Caddy.
