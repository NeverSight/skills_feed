Gestisci i proxy di sviluppo locale utilizzando l'API di amministrazione del server Caddy. Da utilizzare quando gli utenti devono allocare porte localhost, creare proxy inversi per server di sviluppo locali, tenere traccia delle porte in uso nelle sessioni di Claude Code o accedere alle app tramite URL semplici come myapp.localhost. Si attiva su "configura un proxy", "alloca una porta", "cosa è in esecuzione su localhost", "registra la mia app" o qualsiasi attività di configurazione di Caddy.
