Быстрый X CLI для твитов, ответов и чтения через X/Twitter GraphQL (аутентификация файлов cookie).
Поддержка: чтение твитов, поиск, темы/ответы, закладки, лайки, новости/тренды,
ленты пользователей, подписчики/подписчики, списки. Можно использовать как библиотеку.

Примечание. Рекомендуется только для чтения. Письмо быстро блокирует.
