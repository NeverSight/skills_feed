Validiert die Qualität der Testsuite durch Mutationstests. Erzeugt intelligente Codemutationen, führt Tests durch, um zu überprüfen, ob die Änderungen erkannt werden, und identifiziert Lücken in der Testabdeckung. Verwenden Sie es, wenn Sie die Testeffektivität bewerten, neu geschriebene Tests validieren oder die Testqualität für geschäftskritischen Code verbessern.
