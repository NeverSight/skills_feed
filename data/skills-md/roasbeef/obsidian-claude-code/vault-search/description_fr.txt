Recherche sémantique et requêtes de style Dataview dans le coffre-fort Obsidian. À utiliser pour rechercher des notes par signification, trouver du contenu associé, interroger des métadonnées de premier plan ou répondre à des questions sur le contenu du coffre-fort. Les expressions de déclenchement incluent « rechercher dans le coffre-fort », « trouver des notes sur », « qu'est-ce que j'ai sur », « notes associées », « lister les tâches », « afficher les positions ».
