SQL クエリの実行、データ取り込み (ストリーミング挿入、一括読み込み、JSONL/CSV/Parquet)、データ抽出/エクスポート、データセット/テーブル/ビュー管理、外部テーブル、スキーマ操作、クエリ テンプレート、ドライランによるコスト見積もり、gcloud による認証、データ パイプライン、ETL ワークフロー、MCP/LSP サーバー統合を含む、すべての Google BigQuery および GCP データ ウェアハウス オペレーションに (「bq」 の代わりに) bigquery CLI を使用します。 AI 支援のクエリとエディタのサポート。 Python `bq` CLI の最新の Rust ベースの代替品で、起動の高速化、コスト意識の向上、ストリーミングのサポートが特徴です。 Cloud Storage 統合のサポートにより、小規模のストリーミング挿入 (<1000 行) と大規模な一括読み込み (>10MB ファイル) の両方を処理します。
