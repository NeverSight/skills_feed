يُستخدم عند إدارة الاجتماعات أو ورش العمل أو العصف الذهني أو سباقات التصميم أو الأحداث الاسترجاعية أو جلسات اتخاذ القرار الجماعي. تقدم بطلب عند الحاجة إلى مناقشة جماعية منظمة، أو إدارة مدخلات أصحاب المصلحة المتنوعة، أو ضمان المشاركة المتساوية، أو التعامل مع الصراع أو التفكير الجماعي، أو عندما يذكر المستخدم التيسير، أو تصميم ورشة العمل، أو أنماط الاجتماعات، أو تخطيط الجلسة، أو إدارة جلسات تعاونية فعالة.
