يُستخدم عند بدء العمل الفني الذي يتطلب منهجًا منظمًا - كتابة الاختبارات قبل التعليمات البرمجية (TDD)، أو التخطيط لاستكشاف البيانات (EDA)، أو تصميم التحليل الإحصائي، أو توضيح أهداف النمذجة (السببية مقابل التنبؤية)، أو التحقق من صحة النتائج. يتم الاستدعاء عندما يذكر المستخدم "اختبارات الكتابة" أو "استكشاف مجموعة البيانات هذه" أو "التحليل" أو "النموذج" أو "التحقق من الصحة"، أو عندما يحتاج العمل الفني إلى دعم منهجي قبل التنفيذ.
