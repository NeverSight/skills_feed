Da utilizzare quando si inizia un lavoro tecnico che richiede un approccio strutturato: scrittura di test prima del codice (TDD), pianificazione dell'esplorazione dei dati (EDA), progettazione di analisi statistiche, chiarimento degli obiettivi di modellazione (causale o predittivo) o convalida dei risultati. Richiamare quando l'utente menziona "scrivi test per", "esplora questo set di dati", "analizza", "modello", "convalida" o quando il lavoro tecnico necessita di un'impalcatura sistematica prima dell'esecuzione.
