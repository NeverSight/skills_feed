يُستخدم عندما تنتج الموجهات مخرجات غير متسقة أو غير موثوقة، أو تحتاج إلى بنية وقيود واضحة، أو تتطلب حواجز أمان أو فحوصات للجودة، أو تتضمن تفكيرًا متعدد الخطوات يحتاج إلى تحليل، أو تحتاج إلى تشفير خبرة المجال، أو عندما يذكر المستخدم تحسين المطالبات، أو قوالب الموجهات، أو الموجهات المنظمة، أو التحسين الفوري، أو مخرجات الذكاء الاصطناعي الموثوقة، أو أنماط المطالبة.
