يُستخدم عند توثيق القرارات الفنية أو المعمارية المهمة التي تحتاج إلى تسجيل السياق والأساس المنطقي والعواقب. يتم الاستدعاء عند الاختيار بين خيارات التكنولوجيا، أو اتخاذ قرارات البنية التحتية، أو وضع المعايير، أو ترحيل الأنظمة، أو عندما يحتاج الفريق إلى فهم سبب اتخاذ القرار. يُستخدم عندما يذكر المستخدم ADR، أو قرار التصميم، أو سجل القرار الفني، أو وثائق القرار.
