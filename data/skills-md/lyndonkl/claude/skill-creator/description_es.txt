Úselo cuando el usuario tenga un documento (PDF, rebajas, notas de libros, trabajo de investigación, guía metodológica) que contenga conocimientos o marcos teóricos y quiera convertirlo en una habilidad reutilizable y procesable. Invocar cuando el usuario menciona "crear una habilidad a partir de este documento", "convertir esto en una habilidad", "extraer una habilidad de este archivo" o cuando analiza documentos con metodologías, marcos, procesos o enfoques sistemáticos que podrían ser procesables para uso futuro.
