Verwenden Sie diese Option, wenn der Benutzer über ein Dokument (PDF, Markdown, Buchnotizen, Forschungspapier, Methodenleitfaden) verfügt, das theoretisches Wissen oder Frameworks enthält, und dieses in eine umsetzbare, wiederverwendbare Fähigkeit umwandeln möchte. Wird aufgerufen, wenn der Benutzer „eine Fertigkeit aus diesem Dokument erstellen“, „dies in eine Fertigkeit umwandeln“, „eine Fertigkeit aus dieser Datei extrahieren“ erwähnt oder wenn er Dokumente mit Methoden, Frameworks, Prozessen oder systematischen Ansätzen analysiert, die für die zukünftige Verwendung umsetzbar gemacht werden könnten.
