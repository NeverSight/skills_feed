يُستخدم عندما تحتاج الفرق إلى توجيه مشترك ومواءمة عملية صنع القرار. الاستدعاء عند إنشاء فرق جديدة، أو توسيع نطاق المؤسسات، أو تحديد الثقافة، أو إنشاء رؤية للمنتج، أو حل الاختلال، أو إنشاء وضوح استراتيجي، أو وضع معايير سلوكية. يُستخدم عندما يذكر المستخدم North Star، أو قيم الفريق، أو المهمة، أو المبادئ، أو حواجز الحماية، أو إطار القرار، أو التوافق الثقافي.
