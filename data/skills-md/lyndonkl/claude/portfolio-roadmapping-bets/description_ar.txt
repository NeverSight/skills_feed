يُستخدم عند إدارة مبادرات متعددة عبر فترات زمنية (الآن/التالي/لاحقًا، H1/H2/H3)، وموازنة المخاطر مقابل العائد عبر المحفظة، وتحديد حجم الرهانات وتسلسلها مع التبعيات، وتحديد معايير الخروج/القياس للتجارب، وتخصيص الموارد عبر أنواع الابتكار (الأساسية/المجاورة/التحويلية)، أو عندما يذكر المستخدم تخطيط المحفظة، أو آفاق خارطة الطريق، أو إطار الرهان، أو تحديد أولويات المبادرة، أو محفظة الابتكار، أو تخصيص الموارد عبر الآفاق.
