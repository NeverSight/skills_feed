يُستخدم عند بدء التنبؤ لإنشاء خط أساس إحصائي (المعدل الأساسي) قبل تحليل التفاصيل. الاستدعاء عند الحاجة إلى تثبيت التنبؤات في الواقع التاريخي، وتجنب التحيز "هذه المرة مختلف"، أو إنشاء وجهة نظر خارجية قبل تحليل وجهة النظر الداخلية. يُستخدم عندما يذكر المستخدم الأسعار الأساسية، أو الفئات المرجعية، أو العرض الخارجي، أو بدء توقع جديد.
