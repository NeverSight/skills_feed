Richtlinien zur Leistungsoptimierung von React, React Native und Next.js von Vercel Engineering. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von React/React Native/Next.js-Code verwendet werden, um optimale Leistungsmuster sicherzustellen. Enthält Framework-spezifische Lösungen (Next.js, Vite, CRA) und Bibliotheksalternativen (SWR vs React Query, Jotai). Wird bei Aufgaben ausgelöst, die React-Komponenten, Next.js-Seiten, React Native-Apps, Datenabruf, Bundle-Optimierung oder Leistungsverbesserungen betreffen.
