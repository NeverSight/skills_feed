Linee guida per l'ottimizzazione delle prestazioni di React, React Native e Next.js di Vercel Engineering. Questa competenza dovrebbe essere utilizzata durante la scrittura, la revisione o il refactoring del codice React/React Native/Next.js per garantire modelli di prestazioni ottimali. Include soluzioni specifiche del framework (Next.js, Vite, CRA) e alternative alla libreria (SWR vs React Query, Jotai). Attiva attività che coinvolgono componenti React, pagine Next.js, app React Native, recupero di dati, ottimizzazione dei bundle o miglioramenti delle prestazioni.
