Vercel Engineering による React、React Native、Next.js のパフォーマンス最適化ガイドライン。最適なパフォーマンス パターンを確保するには、React/React Native/Next.js コードを作成、レビュー、またはリファクタリングするときにこのスキルを使用する必要があります。フレームワーク固有のソリューション (Next.js、Vite、CRA) および代替ライブラリ (SWR 対 React Query、Jotai) が含まれます。 React コンポーネント、Next.js ページ、React Native アプリ、データの取得、バンドルの最適化、またはパフォーマンスの向上に関連するタスクをトリガーします。
