Habilidad integral de desarrollo de Tauri v2 para crear aplicaciones de escritorio multiplataforma con backends y interfaces web de Rust. Esta habilidad debe usarse al crear nuevas aplicaciones Tauri, agregar comandos y comunicación IPC, desarrollar complementos, administrar el estado de la aplicación o integrar Rust con interfaces JavaScript/TypeScript. Activadores de tareas que involucran #[tauri::command], invoke(), complementos de Tauri, desarrollo de aplicaciones de escritorio o integración Rust-WebView.
