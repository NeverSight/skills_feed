مهارة تطوير Tauri v2 الشاملة لبناء تطبيقات سطح المكتب عبر الأنظمة الأساسية باستخدام واجهات Rust الخلفية وواجهات الويب الأمامية. يجب استخدام هذه المهارة عند إنشاء تطبيقات Tauri جديدة، أو إضافة أوامر واتصالات IPC، أو تطوير المكونات الإضافية، أو إدارة حالة التطبيق، أو دمج Rust مع واجهات JavaScript/TypeScript الأمامية. تشغيل المهام التي تتضمن #[tauri::command]، أو invoc()، أو مكونات Tauri الإضافية، أو تطوير تطبيقات سطح المكتب، أو تكامل Rust-WebView.
