Комплексные навыки разработки Tauri v2 для создания кроссплатформенных настольных приложений с использованием серверных и веб-интерфейсов Rust. Этот навык следует использовать при создании новых приложений Tauri, добавлении команд и связи IPC, разработке плагинов, управлении состоянием приложения или интеграции Rust с интерфейсами JavaScript/TypeScript. Запускается при выполнении задач, связанных с #[tauri::command], ignore(), плагинами Tauri, разработкой настольных приложений или интеграцией Rust-WebView.
