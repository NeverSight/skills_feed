قم بإنشاء مخططات Excalidraw (ملفات .excalidraw JSON) للوحة المعلومات والمخططات الانسيابية ومخططات الهندسة المعمارية ومخططات التسلسل والخرائط الذهنية والإطارات السلكية والمخططات التنظيمية. يُستخدم عندما يطلب المستخدم الرسوم التخطيطية أو الوثائق المرئية أو تصور بنية النظام أو تدفقات العمليات أو أي رسم تخطيطي للنمط مرسوم يدويًا. تشغيل الطلبات التي تشير إلى Excalidraw، أو إنشاء المخططات، أو المخططات الانسيابية، أو المخططات المعمارية، أو المخططات التسلسلية، أو الإطارات السلكية، أو الوثائق المرئية.
