Registro JSON estructurado de alto rendimiento para Node.js. Úselo al crear API de producción que necesitan registros estructurados y rápidos para plataformas de observabilidad (Datadog, ELK, CloudWatch). Proporciona middleware de registro de solicitudes, registradores secundarios para contexto y redacción de datos confidenciales. Elija Pino en lugar de console.log para cualquier backend de producción de TypeScript.
