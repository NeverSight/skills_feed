Высокопроизводительное структурированное ведение журналов JSON для Node.js. Используйте при создании производственных API, которым необходимы быстрые структурированные журналы для платформ наблюдения (Datadog, ELK, CloudWatch). Предоставляет промежуточное программное обеспечение для регистрации запросов, дочерние средства регистрации для контекста и редактирование конфиденциальных данных. Выберите Pino вместо console.log для любого производственного бэкэнда TypeScript.
