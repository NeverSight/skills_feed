Декоративная анимация JSON для обратной связи и доработки пользовательского интерфейса. Используйте для загрузки счетчиков, галочек успеха/ошибки, иллюстраций пустого состояния, анимированных значков. Просто проигрывает и зацикливает — никакой интерактивности. Для реактивной анимации/анимации с сохранением состояния используйте вместо этого Rive. Легкий и совместимый с SSR.
