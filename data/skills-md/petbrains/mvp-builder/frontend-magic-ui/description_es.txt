Componentes pulidos de la página de destino de SaaS. Úselo para animaciones de números/estadísticas, marquesinas de logotipos, rejillas bento, maquetas de dispositivos (iPhone, Safari), botones brillantes/arcoíris y efectos de escritura. Pulido de marketing profesional basado en Framer Motion + Tailwind. Para efectos dramáticos de héroe, use Aceternity, para una interfaz de usuario básica, use shadcn.
