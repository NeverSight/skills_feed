Efectos animados espectaculares para páginas de destino y sitios de marketing. Úselo cuando el usuario necesite "factor sorpresa", secciones de héroes con focos/auroras, tarjetas flotantes 3D, animaciones de revelación de texto, fondos animados. Construido sobre Framer Motion + Tailwind. NO para UI básica (use shadcn) o paneles de control SaaS (use Magic UI).
