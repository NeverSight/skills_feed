قم بإنشاء تصورات HTML تفاعلية لمجموعة المهارات أو بنية قاعدة التعليمات البرمجية أو الرسوم البيانية التبعية. يستخدم D3.js للتصور التفاعلي مع العقد القابلة للطي والفئات المرمزة بالألوان وتفاصيل التمرير. مشغلات على "تصور المهارات"، أو "إنشاء خريطة مهارات"، أو "تصور قاعدة التعليمات البرمجية"، أو "إظهار تبعيات المهارات".
