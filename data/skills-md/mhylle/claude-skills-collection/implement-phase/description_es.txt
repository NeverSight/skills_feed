Ejecutar una sola fase a partir de un plan de implementación con todas las puertas de calidad. Esta habilidad es la unidad de trabajo para implementar el plan, manejar la implementación, verificación, revisión del código, cumplimiento de ADR y sincronización del plan para UNA fase. Se activa cuando implementar-plan delega una fase, o manualmente con "/implement-phase" y una referencia de fase.
