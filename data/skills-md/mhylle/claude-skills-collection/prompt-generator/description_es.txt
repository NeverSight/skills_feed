Genere indicaciones de implementación para la ejecución de proyectos basados ​​en fases utilizando un patrón de orquestación. Úselo cuando los usuarios soliciten indicaciones para implementar fases del proyecto, necesiten indicaciones de orquestación para implementaciones de varios pasos, soliciten "/prompt", "generar solicitud", "crear solicitud de implementación" o deseen crear instrucciones de implementación estructuradas para la delegación de subagente. Genera el mensaje para chatear y lo guarda en la carpeta documentos/indicaciones.
