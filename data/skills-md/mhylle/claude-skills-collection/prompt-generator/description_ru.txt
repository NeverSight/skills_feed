Создавайте подсказки по реализации для поэтапного выполнения проекта, используя шаблон оркестрации. Используйте, когда пользователи запрашивают подсказки для реализации этапов проекта, нуждаются в подсказках по оркестрации для многоэтапных реализаций, запрашивают «/prompt», «создать подсказку», «создать подсказку о реализации» или хотят создать структурированные инструкции по реализации для делегирования субагента. Выводит приглашение в чат и сохраняет его в папке docs/prompts.
