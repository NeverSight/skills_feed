Systematische Codeüberprüfung für Implementierungsphasen zur Überprüfung von Architekturprinzipien, Framework-Standards, ADR-Konformität und Codequalität. Dieser Skill wird von der Implementierungsphase als Teil seiner Quality-Gate-Pipeline oder manuell beim Überprüfen von Codeänderungen aufgerufen. Wird bei „Code überprüfen“, „Codeüberprüfung“, „/code-review“ oder automatisch als Schritt 3 der Implementierungsphase ausgelöst.
