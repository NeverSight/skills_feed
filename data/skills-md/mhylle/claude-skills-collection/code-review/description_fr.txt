Examen systématique du code pour les phases de mise en œuvre vérifiant les principes architecturaux, les normes-cadres, la conformité ADR et la qualité du code. Cette compétence est invoquée lors de la phase de mise en œuvre dans le cadre de son pipeline de contrôle de qualité, ou manuellement lors de l'examen des modifications du code. Se déclenche lors de « révision du code », « révision du code », « /code-review » ou automatiquement à l'étape 3 de la phase de mise en œuvre.
