Erstellen Sie detaillierte Implementierungspläne durch interaktive Recherche und Iteration. Diese Fähigkeit sollte beim Erstellen neuer Implementierungspläne, beim Entwerfen von Funktionsspezifikationen, beim Planen technischer Arbeiten oder wenn der Benutzer die Planung einer Implementierung wünscht, eingesetzt werden. Wird bei Anfragen wie „Plan erstellen“, „Implementierung planen“, „Implementierung entwerfen“ ausgelöst oder wenn eine Funktion/Aufgabe gegeben wird, die vor der Implementierung eine strukturierte Planung erfordert.
