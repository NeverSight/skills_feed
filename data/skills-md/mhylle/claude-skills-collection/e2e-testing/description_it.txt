Competenze di test E2E complete utilizzando Playwright MCP per test sistematici di applicazioni web. Questa competenza dovrebbe essere utilizzata quando gli utenti hanno bisogno di testare sistemi basati sul web end-to-end, impostare regimi di test, eseguire test esplorativi o analizzare la cronologia dei test. Si attiva su richieste come "testare la mia webapp", "impostare test E2E", "eseguire i test", "cosa è stato instabile" o durante la convalida della funzionalità dell'applicazione web. L'abilità osserva e riporta soltanto: non risolve mai i problemi. Supporta tre modalità: configurazione (creazione di un regime di test), esecuzione (esecuzione di test) e report (analisi dei risultati).
