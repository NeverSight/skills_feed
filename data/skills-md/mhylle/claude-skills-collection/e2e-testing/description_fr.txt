Compétence complète en matière de tests E2E utilisant Playwright MCP pour des tests systématiques d'applications Web. Cette compétence doit être utilisée lorsque les utilisateurs doivent tester des systèmes Web de bout en bout, configurer des régimes de test, exécuter des tests exploratoires ou analyser l'historique des tests. Se déclenche sur des requêtes telles que « tester mon application Web », « configurer des tests E2E », « exécuter les tests », « ce qui n'a pas fonctionné » ou lors de la validation des fonctionnalités d'une application Web. La compétence observe et rapporte uniquement ; elle ne résout jamais les problèmes. Prend en charge trois modes : configuration (créer un régime de test), exécution (exécuter des tests) et rapport (analyser les résultats).
