体系的な Web アプリケーション テストに Playwright MCP を使用した包括的な E2E テスト スキル。このスキルは、ユーザーが Web ベースのシステムをエンドツーエンドでテストする必要がある場合、テスト体制を設定する場合、探索的テストを実行する場合、またはテスト履歴を分析する必要がある場合に使用する必要があります。 「Web アプリケーションをテストする」、「E2E テストをセットアップする」、「テストを実行する」、「不安定な内容」などのリクエスト、または Web アプリケーションの機能を検証するときにトリガーされます。このスキルは観察して報告するだけであり、問​​題を解決することはありません。セットアップ (テスト計画の作成)、実行 (テストの実行)、およびレポート (結果の分析) の 3 つのモードをサポートします。
