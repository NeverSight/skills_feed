Habilidad integral de prueba E2E utilizando Playwright MCP para pruebas sistemáticas de aplicaciones web. Esta habilidad debe usarse cuando los usuarios necesitan probar sistemas basados ​​en web de un extremo a otro, configurar regímenes de prueba, ejecutar pruebas exploratorias o analizar el historial de pruebas. Se activa en solicitudes como "probar mi aplicación web", "configurar pruebas E2E", "ejecutar las pruebas", "qué ha fallado" o al validar la funcionalidad de la aplicación web. La habilidad solo observa e informa; nunca soluciona problemas. Admite tres modos: configuración (crear un régimen de prueba), ejecutar (ejecutar pruebas) e informar (analizar resultados).
