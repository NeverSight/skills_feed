Навыки комплексного E2E-тестирования с использованием Playwright MCP для систематического тестирования веб-приложений. Этот навык следует использовать, когда пользователям необходимо комплексно тестировать веб-системы, настраивать режимы тестирования, запускать исследовательские тесты или анализировать историю тестов. Запускается по таким запросам, как «проверить мое веб-приложение», «настроить тесты E2E», «запустить тесты», «что пошло не так» или при проверке функциональности веб-приложения. Навык только наблюдает и сообщает, но никогда не устраняет проблемы. Поддерживает три режима — настройка (создание режима тестирования), запуск (выполнение тестов) и отчет (анализ результатов).
