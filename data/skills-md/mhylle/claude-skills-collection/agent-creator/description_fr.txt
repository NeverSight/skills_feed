Créez des systèmes d'agents IA composables dans les projets NestJS en suivant l'architecture « outils jusqu'en bas ». Utilisez cette compétence lorsque les utilisateurs souhaitent : (1) créer de nouveaux agents IA avec des composants orchestrateur/planificateur/exécuteur/évaluateur, (2) créer des systèmes agentiques qui peuvent appeler d'autres agents en tant qu'outils, (3) mettre en œuvre une planification basée sur DAG avec exécution parallèle, (4) ajouter une persistance d'état basée sur une base de données pour les exécutions d'agents, (5) créer des évaluateurs personnalisés pour l'assurance qualité. Déclencheurs sur les requêtes « créer un agent », « construire un agent », « un système d'agent », « un agent », « un orchestrateur », un « modèle de planificateur/exécuteur » ou un agent NestJS AI.
