Создавайте компонуемые системы агентов ИИ в проектах NestJS, следуя архитектуре «все инструменты». Используйте этот навык, когда пользователи хотят: (1) создавать новые агенты ИИ с компонентами оркестратора/планировщика/исполнителя/оценщика, (2) создавать агентные системы, которые могут вызывать другие агенты в качестве инструментов, (3) реализовывать планирование на основе DAG с параллельным выполнением, (4) добавлять сохранение состояния на базе базы данных для запуска агентов, (5) создавать собственные оценщики для обеспечения качества. Запускается по запросам «создание агента», «агент сборки», «система агентов», «агент», «оркестратор», «шаблон планировщика/исполнителя» или запросам агента NestJS AI.
