تدقيق أمني شامل لتغييرات التعليمات البرمجية. استخدم هذه المهارة عند تنفيذ المصادقة أو الترخيص أو معالجة إدخال المستخدم أو نقاط نهاية واجهة برمجة التطبيقات أو الأسرار/بيانات الاعتماد أو ميزات الدفع أو تحميل الملفات. يوفر قوائم التحقق من الأمان وأنماط الثغرات الأمنية وإرشادات العلاج. يتكامل مع مرحلة التنفيذ كبوابة جودة أمنية.
