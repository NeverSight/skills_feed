Auditoría de seguridad integral para cambios de código. Utilice esta habilidad al implementar autenticación, autorización, manejo de entradas de usuarios, puntos finales de API, secretos/credenciales, funciones de pago o carga de archivos. Proporciona listas de verificación de seguridad, patrones de vulnerabilidad y orientación para solucionarlos. Se integra con la fase de implementación como puerta de seguridad de calidad.
