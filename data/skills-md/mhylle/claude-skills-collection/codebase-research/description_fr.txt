Orchestre une recherche complète sur la base de code en décomposant les requêtes des utilisateurs en tâches de sous-agents parallèles et en synthétisant les résultats. Cette compétence doit être utilisée lorsque les utilisateurs posent des questions sur le fonctionnement du code, où les fonctionnalités existent, comment les composants interagissent ou ont besoin d'une documentation complète sur les implémentations existantes.
