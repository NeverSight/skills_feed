Esta habilidad debe usarse cuando el usuario solicita guardar el contexto, preservar el estado del trabajo, verificar el progreso o prepararse para la transferencia de la sesión. Se activa al "guardar contexto", "punto de control", "guardar progreso", "preservar estado" o cuando se le solicita explícitamente que cree un archivo de contexto para su posterior reanudación.
