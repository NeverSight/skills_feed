Generieren Sie ein Produktanforderungsdokument (PRD) für eine neue Funktion. Verwenden Sie diese Funktion, wenn Sie eine Funktion planen, ein neues Projekt starten oder wenn Sie aufgefordert werden, ein PRD zu erstellen. Auslöser bei: Erstellen eines PRD, Schreiben eines PRD für, Planen dieser Funktion, Anforderungen für, Spezifizieren.
