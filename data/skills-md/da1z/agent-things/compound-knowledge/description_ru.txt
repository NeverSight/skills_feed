Сохраняйте заметки о знаниях команды в ./knowledge/ с именами файлов с префиксом даты. Каждая решенная проблема, обнаруженная закономерность или ключевое решение становятся доступными для поиска знаниями, которые облегчают будущую работу. Используйте, когда пользователь хочет зафиксировать сбои, ошибки, проблемы сборки, решения, шаблоны или решения для дальнейшего использования.
