استخدم هذا عند إنشاء مشاريع جديدة، أو إنشاء الوثائق، أو تنظيف/تنظيم الريبو، أو اقتراح البنية، أو نشر الحاويات والخدمات، أو تسمية الملفات/المجلدات، أو عندما يشير المستخدم إلى "النظام البيئي" أو "الأنماط" أو "الحاويات". تحدد هذه المهارة اصطلاحات التسمية وتفضيلات المكدس وتنظيم المشروع (أشجار عمل iMi) وأنماط Docker وهياكل PRD من المحادثات السابقة.
