قم بإنشاء أدلة فنية شاملة وعالية الجودة مع سرد قصصي موضوعي باستخدام تنسيق متعدد الوكلاء. يُستخدم عندما يطلب المستخدم وثائق ذات موضوع معين، أو أدلة فنية سردية، أو مواد تدريبية لا تُنسى، أو يذكر موضوعات مثل العصر الفيكتوري، أو Steampunk، أو آرت ديكو، أو القرون الوسطى، أو البحرية مع موضوعات فنية. يتم تشغيل عبارات مثل "إنشاء دليل موضوعي" أو "توثيق مع سرد القصص" أو طلبات لجعل المستندات "أكثر جاذبية" أو "قابلة للتذكر".
