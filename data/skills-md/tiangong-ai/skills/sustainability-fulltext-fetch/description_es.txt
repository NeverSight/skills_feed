Obtenga y mantenga contenido para entradas RSS de sostenibilidad con clave DOI desde una base de datos SQLite de texto completo independiente, utilizando primero los metadatos de la API OpenAlex/Semantic Scholar y la extracción de texto completo de la página web como alternativa. Úselo al crear un enriquecimiento de contenido resistente con DOI primero después del etiquetado de relevancia.
