Crea e configura gli hook Claude Code per personalizzare il comportamento dell'agente. Da utilizzare quando l'utente desidera (1) creare un nuovo hook, (2) configurare la formattazione, la registrazione o le notifiche automatiche, (3) aggiungere la protezione dei file o autorizzazioni personalizzate, (4) impostare azioni pre/post esecuzione dello strumento o (5) richiedere eventi di hook come PreToolUse, PostToolUse, Notification, ecc.
