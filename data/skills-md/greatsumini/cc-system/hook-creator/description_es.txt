Cree y configure enlaces de Claude Code para personalizar el comportamiento del agente. Úselo cuando el usuario desee (1) crear un nuevo enlace, (2) configurar el formato, el registro o las notificaciones automáticos, (3) agregar protección de archivos o permisos personalizados, (4) configurar acciones previas y posteriores a la ejecución de la herramienta o (5) pregunta sobre eventos de enlace como PreToolUse, PostToolUse, Notificación, etc.
