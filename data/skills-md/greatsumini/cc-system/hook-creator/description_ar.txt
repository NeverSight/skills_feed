قم بإنشاء وتكوين خطافات Claude Code لتخصيص سلوك الوكيل. يُستخدم عندما يريد المستخدم (1) إنشاء رابط جديد، (2) تكوين التنسيق التلقائي أو التسجيل أو الإشعارات، (3) إضافة حماية للملفات أو أذونات مخصصة، (4) إعداد إجراءات تنفيذ الأداة قبل/بعد، أو (5) يسأل عن أحداث الربط مثل PreToolUse، وPostToolUse، وNotification، وما إلى ذلك.
