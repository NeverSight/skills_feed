Esta habilidad debe usarse cuando los usuarios desean restaurar, recargar o copiar sus archivos de configuración de Claude Code (comandos, habilidades, agentes) desde un directorio de respaldo a ~/.claude/. Activado por frases como "restaurar mi configuración de Claude", "recargar la configuración de Claude", "copiar la configuración de nuevo a ~/.claude/", "restaurar desde la copia de seguridad", "recuperar la configuración de Claude", "importar comandos y habilidades desde la copia de seguridad", "accidentalmente eliminé mis comandos", "restaurar mis antiguas habilidades", "copiar la configuración de la copia de seguridad" o "Necesito recuperar la configuración de mi Claude desde una copia de seguridad".
