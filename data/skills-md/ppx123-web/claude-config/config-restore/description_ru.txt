Этот навык следует использовать, когда пользователи хотят восстановить, перезагрузить или скопировать свои файлы конфигурации Claude Code (команды, навыки, агенты) из резервного каталога обратно в ~/.claude/. Запускается такими фразами, как «восстановить мою конфигурацию Клода», «перезагрузить настройки Клода», «скопировать конфигурацию обратно в ~/.claude/», «восстановить из резервной копии», «восстановить конфигурацию Клода», «импортировать команды и навыки из резервной копии», «Я случайно удалил свои команды», «восстановить мои старые навыки», «скопировать конфигурацию из резервной копии» или «Мне нужно восстановить настройки Клода из резервной копии».
