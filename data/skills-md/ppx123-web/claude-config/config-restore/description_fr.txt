Cette compétence doit être utilisée lorsque les utilisateurs souhaitent restaurer, recharger ou copier leurs fichiers de configuration Claude Code (commandes, compétences, agents) depuis un répertoire de sauvegarde vers ~/.claude/. Déclenché par des expressions telles que « restaurer ma configuration Claude », « recharger les paramètres Claude », « copier la configuration vers ~/.claude/ », « restaurer à partir d'une sauvegarde », « récupérer la configuration Claude », « importer des commandes et des compétences à partir d'une sauvegarde », « J'ai accidentellement supprimé mes commandes », « restaurer mes anciennes compétences », « copier la configuration à partir d'une sauvegarde » ou « Je dois récupérer mes paramètres Claude à partir d'une sauvegarde ».
