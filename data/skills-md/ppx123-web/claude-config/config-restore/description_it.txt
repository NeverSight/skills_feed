Questa abilità dovrebbe essere utilizzata quando gli utenti desiderano ripristinare, ricaricare o copiare i file di configurazione del codice Claude (comandi, abilità, agenti) da una directory di backup a ~/.claude/. Attivato da frasi come "ripristina la mia configurazione di Claude", "ricarica le impostazioni di Claude", "copia la configurazione in ~/.claude/", "ripristina dal backup", "ripristina la configurazione di Claude", "importa comandi e competenze dal backup", "Ho cancellato accidentalmente i miei comandi", "ripristina le mie vecchie competenze", "copia la configurazione dal backup" o "Ho bisogno di ripristinare le mie impostazioni di Claude da un backup".
