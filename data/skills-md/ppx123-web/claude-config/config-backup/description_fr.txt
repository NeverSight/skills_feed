À utiliser lorsque les utilisateurs souhaitent enregistrer, sauvegarder, copier, exporter ou contrôler la version de leurs fichiers de configuration Claude Code (commandes, compétences, agents) de ~/.claude/ vers un autre répertoire. Déclenché par des expressions telles que « enregistrer mes paramètres Claude », « sauvegarder ma configuration Claude », « synchroniser la configuration Claude », « copier ~/.claude vers quelque part », « exporter mes commandes et mes compétences » ou « archiver les paramètres Claude avant d'apporter des modifications ».
