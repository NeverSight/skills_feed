Используйте, когда пользователи хотят сохранить, сделать резервную копию, скопировать, экспортировать или контролировать версии своих файлов конфигурации Claude Code (команды, навыки, агенты) из ~/.claude/ в другой каталог. Запускается такими фразами, как «сохранить мои настройки Клода», «сохранить резервную копию моей конфигурации Клода», «синхронизировать конфигурацию Клода», «скопировать ~/.claude куда-нибудь», «экспортировать мои команды и навыки» или «архивировать настройки Клода перед внесением изменений».
