Этот навык следует использовать при создании и редактировании изображений с помощью Gemini API (Nano Banana Pro). Это применимо при создании изображений из текстовых подсказок, редактировании существующих изображений, применении переноса стилей, создании логотипов с текстом, создании наклеек, макетов продуктов или любых задачах создания/манипулирования изображениями. Поддерживает преобразование текста в изображение, редактирование изображений, многоэтапное уточнение и композицию из нескольких эталонных изображений.
