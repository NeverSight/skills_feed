Автоматизация браузера с помощью CLI агента-браузера Vercel. Используйте, когда вам нужно взаимодействовать с веб-страницами, заполнять формы, делать снимки экрана или собирать данные. Альтернатива Playwright MCP — использует команды Bash с выбором элемента на основе ссылки. Триггеры: «просмотреть веб-сайт», «заполнить форму», «нажать кнопку», «сделать снимок экрана», «очистить страницу», «веб-автоматизация».
