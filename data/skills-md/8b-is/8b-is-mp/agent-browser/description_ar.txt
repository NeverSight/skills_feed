أتمتة المتصفح باستخدام واجهة سطر الأوامر (CLI) لمتصفح وكيل Vercel. استخدمه عندما تحتاج إلى التفاعل مع صفحات الويب، أو ملء النماذج، أو التقاط لقطات شاشة، أو استخراج البيانات. بديل لـ Playwright MCP - يستخدم أوامر Bash مع اختيار العناصر المستندة إلى المرجع. مشغلات على "تصفح موقع الويب"، "ملء النموذج"، "زر النقر"، "التقاط لقطة شاشة"، "مسح الصفحة"، "أتمتة الويب".
