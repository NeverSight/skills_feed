Изучает обсуждения Claude Code в поисках лучших общих решений для кодирования с помощью поиска кода Claude (ccs). Использует сигналы качества — маркеры прорыва, индикаторы сложности и оценку ценности сообщества — для выявления запоминающегося контента из истории разговоров. Используйте, когда пользователь хочет «извлечь разговоры для обмена», «заполнить репозиторий общих ресурсов», «вспомнить мои лучшие работы», «найти мои лучшие решенные проблемы» или «решить проблему холодного запуска».
