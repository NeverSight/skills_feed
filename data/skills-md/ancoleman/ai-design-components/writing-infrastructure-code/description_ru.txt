Управление облачной инфраструктурой с помощью декларативных и императивных инструментов IaC. Используйте при предоставлении облачных ресурсов (Terraform/OpenTofu для мультиоблачной среды, Pulumi для рабочих процессов, ориентированных на разработчиков, AWS CDK для собственной инфраструктуры AWS), проектировании повторно используемых модулей, реализации шаблонов управления состоянием или организации рабочих процессов развертывания инфраструктуры.
