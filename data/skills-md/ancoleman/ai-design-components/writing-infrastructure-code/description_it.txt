Gestire l'infrastruttura cloud utilizzando strumenti IaC dichiarativi e imperativi. Da utilizzare durante il provisioning di risorse cloud (Terraform/OpenTofu per multi-cloud, Pulumi per flussi di lavoro incentrati sugli sviluppatori, AWS CDK per infrastrutture native AWS), la progettazione di moduli riutilizzabili, l'implementazione di modelli di gestione dello stato o la definizione di flussi di lavoro di distribuzione dell'infrastruttura.
