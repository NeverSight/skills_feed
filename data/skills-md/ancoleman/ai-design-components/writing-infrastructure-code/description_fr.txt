Gestion de l'infrastructure cloud à l'aide d'outils IaC déclaratifs et impératifs. À utiliser lors du provisionnement de ressources cloud (Terraform/OpenTofu pour le multi-cloud, Pulumi pour les workflows centrés sur les développeurs, AWS CDK pour l'infrastructure native AWS), la conception de modules réutilisables, la mise en œuvre de modèles de gestion d'état ou l'établissement de workflows de déploiement d'infrastructure.
