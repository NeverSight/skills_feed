REST、GraphQL、gRPC、tRPC パターンにわたる API の設計と実装。バックエンド サービス、パブリック API、またはサービス間通信を構築するときに使用します。 REST フレームワーク (FastAPI、Axum、Gin、Hono)、GraphQL ライブラリ (Strawberry、async-graphql、gqlgen、Pothos)、gRPC (Tonic、Connect-Go)、TypeScript 用の tRPC、ページ分割戦略 (カーソル ベース、オフセット ベース)、レート制限、キャッシュ、バージョン管理、OpenAPI ドキュメントの生成をカバーします。フォーム、テーブル、ダッシュボード、AI チャット スキルのフロントエンド統合パターンが含まれています。
