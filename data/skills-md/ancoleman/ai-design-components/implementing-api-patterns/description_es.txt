Diseño e implementación de API en patrones REST, GraphQL, gRPC y tRPC. Úselo al crear servicios backend, API públicas o comunicación entre servicios. Cubre marcos REST (FastAPI, Axum, Gin, Hono), bibliotecas GraphQL (Strawberry, async-graphql, gqlgen, Pothos), gRPC (Tonic, Connect-Go), tRPC para TypeScript, estrategias de paginación (basadas en cursor, basadas en desplazamiento), limitación de velocidad, almacenamiento en caché, control de versiones y generación de documentación OpenAPI. Incluye patrones de integración de frontend para formularios, tablas, paneles y habilidades de ai-chat.
