Проектирование и реализация API с использованием шаблонов REST, GraphQL, gRPC и tRPC. Используйте при создании серверных служб, общедоступных API или связи между службами. Охватывает платформы REST (FastAPI, Axum, Gin, Hono), библиотеки GraphQL (Strawberry, async-graphql, gqlgen, Pothos), gRPC (Tonic, Connect-Go), tRPC для TypeScript, стратегии нумерации страниц (на основе курсора, на основе смещения), ограничение скорости, кэширование, управление версиями и создание документации OpenAPI. Включает шаблоны интеграции внешнего интерфейса для форм, таблиц, информационных панелей и навыков чата.
