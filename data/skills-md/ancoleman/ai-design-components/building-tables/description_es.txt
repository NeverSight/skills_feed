Crea tablas y cuadrículas de datos para mostrar información tabular, desde simples tablas HTML hasta complejas cuadrículas de datos empresariales. Úselo al crear tablas, implementar clasificación/filtrado/paginación, manejar grandes conjuntos de datos (más de 10 a 1 millón de filas), crear interfaces similares a hojas de cálculo o diseñar componentes con muchos datos. Proporciona estrategias de optimización del rendimiento, patrones de accesibilidad (WCAG/ARIA), diseños responsivos y recomendaciones de bibliotecas (TanStack Table, AG Grid).
