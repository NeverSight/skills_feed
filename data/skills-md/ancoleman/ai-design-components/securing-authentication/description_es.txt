Implementación de autenticación, autorización y seguridad API. Úselo al crear sistemas de usuario, proteger API o implementar control de acceso. Cubre OAuth 2.1/OIDC, patrones JWT, sesiones, claves de acceso/WebAuthn, RBAC/ABAC/ReBAC, motores de políticas (OPA, Casbin, SpiceDB), autenticación administrada (Clerk, Auth0), autohospedado (Keycloak, Ory) y mejores prácticas de seguridad de API.
