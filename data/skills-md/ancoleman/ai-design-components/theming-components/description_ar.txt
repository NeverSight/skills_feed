يوفر نظام رمز التصميم وإطار عمل السمات لتصميم واجهة المستخدم المتسق والقابل للتخصيص عبر جميع المكونات. يغطي التصنيف الكامل للرموز المميزة (اللون، الطباعة، التباعد، الظلال، الحدود، الحركة، مؤشر z)، تبديل السمات (خصائص CSS المخصصة، موفري السمات)، دعم RTL/i18n (خصائص CSS المنطقية)، وإمكانية الوصول (تباين WCAG، سمات التباين العالي، الحركة المنخفضة). هذه هي طبقة التصميم الأساسية التي تشير إليها جميع مهارات المكونات. يُستخدم عند تخصيص المكونات، أو تنفيذ الوضع الفاتح/المظلم، أو إنشاء أنماط العلامة التجارية، أو تخصيص التصميم المرئي، أو ضمان اتساق التصميم، أو دعم لغات RTL.
