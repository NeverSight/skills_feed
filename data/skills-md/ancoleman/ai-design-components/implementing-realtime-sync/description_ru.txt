Шаблоны связи в реальном времени для обновлений в режиме реального времени, совместной работы и присутствия. Используйте при создании приложений чата, инструментов для совместной работы, интерактивных информационных панелей или потоковых интерфейсов (ответы LLM, метрики). Охватывает SSE (события, отправляемые сервером для односторонних потоков), WebSocket (двусторонняя связь), WebRTC (одноранговая видео/аудио), CRDT (Yjs, Automerge для бесконфликтной совместной работы), шаблоны присутствия, автономную синхронизацию и стратегии масштабирования. Поддерживает Python, Rust, Go и TypeScript.
