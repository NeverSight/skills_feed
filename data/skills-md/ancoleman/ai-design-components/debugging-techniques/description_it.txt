Flussi di lavoro di debug per Python (pdb, debugpy), Go (delve), Rust (lldb) e Node.js, incluso il debug dei contenitori (debug kubectl, contenitori temporanei) e tecniche di debug sicure per la produzione con tracciamento distribuito e ID di correlazione. Da utilizzare durante l'impostazione di punti di interruzione, il debug di contenitori/pod, il debug remoto o il debug di produzione.
