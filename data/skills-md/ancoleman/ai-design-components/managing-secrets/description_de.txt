Verwalten von Geheimnissen (API-Schlüssel, Datenbankanmeldeinformationen, Zertifikate) mit Vault, Cloud-Anbietern und Kubernetes. Verwenden Sie es zum Speichern vertraulicher Daten, zum Rotieren von Anmeldeinformationen, zum Synchronisieren von Geheimnissen mit Kubernetes, zum Implementieren dynamischer Geheimnisse oder zum Scannen von Code auf durchgesickerte Geheimnisse.
