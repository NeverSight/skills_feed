تنفيذ قاعدة البيانات العلائقية عبر Python وRust وGo وTypeScript. يُستخدم عند إنشاء تطبيقات CRUD أو أنظمة المعاملات أو تخزين البيانات المنظمة. يغطي PostgreSQL (الأساسي)، وMySQL، وSQLite، وORMs (SQLAlchemy، وPrisma، وSeaORM، وGORM)، ومنشئي الاستعلامات (Drizzle، وsqlc، وSQLx)، والترحيلات، وتجميع الاتصالات، وقواعد البيانات بدون خادم (Neon، وPlanetScale، وTurso).
