Implementazione di database relazionali su Python, Rust, Go e TypeScript. Da utilizzare durante la creazione di applicazioni CRUD, sistemi transazionali o archiviazione di dati strutturati. Copre PostgreSQL (primario), MySQL, SQLite, ORM (SQLAlchemy, Prisma, SeaORM, GORM), generatori di query (Drizzle, sqlc, SQLx), migrazioni, pool di connessioni e database serverless (Neon, PlanetScale, Turso).
