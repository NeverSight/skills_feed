Implémentation de bases de données relationnelles sur Python, Rust, Go et TypeScript. À utiliser lors de la création d'applications CRUD, de systèmes transactionnels ou de stockage de données structurées. Couvre PostgreSQL (primaire), MySQL, SQLite, les ORM (SQLAlchemy, Prisma, SeaORM, GORM), les générateurs de requêtes (Drizzle, sqlc, SQLx), les migrations, le pooling de connexions et les bases de données sans serveur (Neon, PlanetScale, Turso).
