Реализация базы данных документов для приложений с гибкой схемой. Используйте при создании управления контентом, профилей пользователей, каталогов или регистрации событий. Охватывает MongoDB (основной), DynamoDB, Firestore, шаблоны проектирования схем, стратегии индексирования и конвейеры агрегации.
