Implémentation d'une base de données documentaire pour des applications de schéma flexibles. À utiliser lors de la création de gestion de contenu, de profils utilisateur, de catalogues ou de journalisation d'événements. Couvre MongoDB (primaire), DynamoDB, Firestore, les modèles de conception de schéma, les stratégies d'indexation et les pipelines d'agrégation.
