Implémentation d'une base de données de séries chronologiques pour les backends de métriques, d'IoT, de données financières et d'observabilité. À utiliser lors de la création de tableaux de bord, de systèmes de surveillance, de plateformes IoT ou d'applications financières. Couvre TimescaleDB (PostgreSQL), InfluxDB, ClickHouse, QuestDB, les agrégats continus, le sous-échantillonnage (LTTB) et les politiques de rétention.
