قم بإنشاء وثائق فنية شاملة بما في ذلك مستندات API (OpenAPI/Swagger)، ووثائق التعليمات البرمجية (TypeDoc/Sphinx)، ومواقع التوثيق (Docusaurus/MkDocs)، وسجلات قرارات الهندسة المعمارية (ADRs)، والرسوم البيانية (Mermaid/PlantUML). يُستخدم عند توثيق واجهات برمجة التطبيقات أو المكتبات أو بنية الأنظمة أو إنشاء مواقع توثيق تواجه المطورين.
