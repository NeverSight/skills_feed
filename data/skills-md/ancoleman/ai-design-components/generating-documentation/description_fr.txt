Générez une documentation technique complète comprenant des documents API (OpenAPI/Swagger), de la documentation de code (TypeDoc/Sphinx), des sites de documentation (Docusaurus/MkDocs), des enregistrements de décisions d'architecture (ADR) et des diagrammes (Mermaid/PlantUML). À utiliser lors de la documentation des API, des bibliothèques, de l'architecture des systèmes ou de la création de sites de documentation destinés aux développeurs.
