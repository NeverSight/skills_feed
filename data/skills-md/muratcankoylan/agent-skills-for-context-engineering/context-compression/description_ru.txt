Этот навык следует использовать, когда пользователь просит «сжать контекст», «обобщить историю разговоров», «реализовать уплотнение», «уменьшить использование токенов» или упоминает сжатие контекста, структурированное суммирование, оптимизацию токенов для каждой задачи или длительные сеансы агента, превышающие ограничения контекста.
