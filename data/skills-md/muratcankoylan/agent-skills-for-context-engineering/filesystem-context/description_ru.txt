Этот навык следует использовать, когда пользователь просит «выгрузить контекст в файлы», «реализовать динамическое обнаружение контекста», «использовать файловую систему для памяти агента», «уменьшить раздувание контекстного окна» или упоминает управление контекстом на основе файлов, постоянство вывода инструмента, блокноты агента или своевременную загрузку контекста.
