Этот навык следует использовать, когда пользователь просит «спроектировать многоагентную систему», «реализовать шаблон супервизора», «создать роевую архитектуру», «координировать несколько агентов» или упоминает многоагентные шаблоны, изоляцию контекста, передачу обслуживания агентов, субагенты или параллельное выполнение агентов.
