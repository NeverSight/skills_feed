Этот навык следует использовать, когда пользователь просит «реализовать память агента», «сохранять состояние между сеансами», «построить граф знаний», «отслеживать объекты» или упоминает архитектуру памяти, временные графы знаний, векторные хранилища, память сущностей или постоянство между сеансами.
