Управляйте состоянием сервера в React с помощью TanStack Query v5. Охватывает useMutationState, упрощенные оптимистические обновления, throwOnError, сетевой режим (оффлайн/PWA) и InfiniteQueryOptions.

Используйте при настройке выборки данных, исправлении ошибок миграции v4 → v5 (синтаксис объекта, gcTime, isPending, KeepPreviousData) или отладке проблем SSR/гидратации с компонентами потокового сервера.
