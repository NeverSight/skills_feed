Справочник по библиотеке компонентов пользовательского интерфейса для Composable Svelte. Используйте при реализации проектов, выборе компонентов, стилизации макетов или работе с компонентами Shadcn-Svelte. Охватывает реквизиты компонентов, варианты, шаблоны доступности, визуальную композицию и когда какой компонент использовать. Для специализированных компонентов см. составную стройную графику (3D), составной стройный код (редакторы/медиа), составные стройные диаграммы (визуализация), составные стройные карты (геопространственные).
