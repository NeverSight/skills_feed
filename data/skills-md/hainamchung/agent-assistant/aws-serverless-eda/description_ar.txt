خبير في الهندسة المعمارية بدون خادم ومدفوع بالأحداث في AWS استنادًا إلى إطار عمل جيد التصميم. يُستخدم عند إنشاء واجهات برمجة التطبيقات بدون خادم، أو وظائف Lambda، أو واجهات برمجة تطبيقات REST، أو الخدمات الصغيرة، أو مسارات العمل غير المتزامنة. يغطي Lambda مع TypeScript/Python، وAPI Gateway (REST/HTTP)، وDynamoDB، وStep Functions، وEventBridge، وSQS، وSNS، والأنماط بدون خادم. ضروري عندما يذكر المستخدم بدون خادم، أو Lambda، أو API Gateway، أو معالجة غير متزامنة تعتمد على الأحداث، أو قوائم الانتظار، أو pub/sub، أو يريد إنشاء تطبيقات قابلة للتطوير بدون خادم باستخدام أفضل ممارسات AWS.
