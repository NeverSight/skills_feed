Обрабатывайте мультимедийные файлы с помощью FFmpeg (кодирование видео/аудио, преобразование, потоковая передача, фильтрация, аппаратное ускорение) и ImageMagick (манипулирование изображениями, преобразование формата, пакетная обработка, эффекты, композиция). Используйте при преобразовании медиаформатов, кодировании видео с помощью определенных кодеков (H.264, H.265, VP9), изменении размера/обрезки изображений, извлечении звука из видео, применении фильтров и эффектов, оптимизации размеров файлов, создании потоковых манифестов (HLS/DASH), создании миниатюр, пакетной обработке изображений, создании составных изображений или реализации конвейеров обработки мультимедиа. Поддерживает более 100 форматов, аппаратное ускорение (NVENC, QSV) и сложные графы фильтров.
