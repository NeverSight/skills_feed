Créez des interfaces utilisateur frontales élégantes en suivant le système de conception NextGen de Microsoft Foundry à l'aide de Vite + React + pnpm + Framer Motion. À utiliser lors de la création de tableaux de bord, de générateurs d'agents, de grilles de données, d'interfaces de gestion d'entités ou de toute application correspondant à l'esthétique raffinée du thème sombre de Foundry.
