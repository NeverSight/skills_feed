Чистые шаблоны кода для пакета SDK Python для поиска Azure AI (azure-search-documents). Используйте при создании поисковых приложений, создании индексов и управлении ими, реализации агентного поиска с помощью баз знаний или работе с векторным/гибридным поиском. Охватывает SearchClient, SearchIndexClient, SearchIndexerClient и KnowledgeBaseRetrivalClient.
