Cree agentes de IA mediante el SDK de Python de Azure AI Agents (azure-ai-agents). Úselo al crear agentes alojados en Azure AI Foundry con herramientas (búsqueda de archivos, intérprete de código, Bing Grounding, Azure AI Search, llamadas de funciones, OpenAPI, MCP), administrar subprocesos y mensajes, implementar respuestas de transmisión o trabajar con almacenes de vectores. Este es el SDK de bajo nivel; para abstracciones de nivel superior, utilice la habilidad de marco de agente.
