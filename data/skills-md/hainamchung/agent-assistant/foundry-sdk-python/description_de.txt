Erstellen Sie KI-Anwendungen mit dem Azure AI Projects Python SDK (azure-ai-projects). Verwenden Sie diese Option, wenn Sie mit Foundry-Projektclients arbeiten, versionierte Agenten mit PromptAgentDefinition erstellen, Auswertungen ausführen, Verbindungen/Bereitstellungen/Datensätze/Indizes verwalten oder OpenAI-kompatible Clients verwenden. Dies ist das High-Level-Foundry-SDK – für Low-Level-Agent-Vorgänge verwenden Sie den Skill azure-ai-agents-python.
