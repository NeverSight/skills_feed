Mithilfe von Tools interagieren KI-Agenten mit der Welt. Ein gut konzipiertes Tool macht den Unterschied zwischen einem Wirkstoff, der funktioniert, und einem Wirkstoff aus, der halluziniert, stillschweigend versagt oder zehnmal mehr Token kostet als nötig.  Diese Fähigkeit deckt das Tool-Design vom Schema bis zur Fehlerbehandlung ab. Best Practices für das JSON-Schema, das Verfassen von Beschreibungen, die dem LLM tatsächlich helfen, die Validierung und der neue MCP-Standard, der zur Verkehrssprache für KI-Tools wird.  Wichtige Erkenntnis: Werkzeugbeschreibungen sind wichtiger als Werkzeugimplementierungen
