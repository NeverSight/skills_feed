Construit des serveurs MCP (Model Context Protocol) distants sur Cloudflare Workers
avec des outils, l'authentification OAuth et le déploiement en production. Génère
code du serveur, configure les fournisseurs d'authentification et déploie sur Workers.

À utiliser lorsque : l'utilisateur souhaite "créer un serveur MCP", "créer des outils MCP", "à distance
MCP", "déployer MCP", ajouter "OAuth à MCP" ou mentionner le protocole de contexte de modèle
sur Cloudflare. Se déclenche également sur « authentification MCP » ou « déploiement MCP ».
