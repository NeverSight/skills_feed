Erstellt Remote-MCP-Server (Model Context Protocol) auf Cloudflare Workers
mit Tools, OAuth-Authentifizierung und Produktionsbereitstellung. Erzeugt
Servercode, konfiguriert Authentifizierungsanbieter und stellt sie für Worker bereit.

Verwenden Sie, wenn: Der Benutzer möchte „MCP-Server erstellen“, „MCP-Tools erstellen“, „remote
MCP“, „MCP bereitstellen“, „OAuth to MCP“ hinzufügen oder das Model Context Protocol erwähnen
auf Cloudflare. Wird auch bei „MCP-Authentifizierung“ oder „MCP-Bereitstellung“ ausgelöst.
