قم بتنظيم تغييرات git وتنفيذها ودفعها باستخدام رسائل الالتزام التقليدية. يُستخدم عندما يريد المستخدم الالتزام بالتغييرات ودفعها، أو يذكر الدفع إلى جهاز التحكم عن بُعد، أو يطلب حفظ عمله ودفعه. يتم تنشيطه أيضًا عندما يقول المستخدم "ادفع التغييرات"، أو "التزم وادفع"، أو "ادفع هذا"، أو "ادفع إلى github"، أو طلبات سير عمل git المماثلة.
