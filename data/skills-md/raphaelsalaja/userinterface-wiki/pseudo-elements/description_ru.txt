Современные псевдоэлементы CSS для стилизации без дополнительных узлов DOM. Используйте при реализации переходов видов, добавлении декоративных слоев или стилизации собственного пользовательского интерфейса браузера. Запускается при выполнении задач, включающих ::before, ::after, переход между видами, оформление фона или сокращение JavaScript для визуальных эффектов.
