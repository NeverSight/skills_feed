Des conseils complets pour le développement de mods Minecraft avec Fabric, y compris le portage à partir d'autres chargeurs de mods (Forge, NeoForge). Intègre les serveurs MCP pour la décompilation, l'accès à la documentation et la validation du mixin. À utiliser lors du développement de mods Fabric, du portage depuis Forge ou de l'utilisation du code source de Minecraft.
