Сценарий компиляции поддерживает pdflatex, xelatex, lualatex с автоматическим обнаружением, серверную часть latexmk, фильтрацию журналов texfot, вывод PDF/A и управление многословием (--verbose/--quiet). Эмпирически оптимизированное масштабирование: один агент 1–10 страниц, разделение 11–20, пакетный конвейер 7 21+.
