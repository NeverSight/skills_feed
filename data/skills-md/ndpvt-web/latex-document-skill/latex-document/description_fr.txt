Compétence documentaire universelle LaTeX : créez, compilez et convertissez n'importe quel document en PDF professionnel avec des aperçus PNG. Prend en charge les curriculum vitae, les rapports, les lettres de motivation, les factures, les articles académiques, les thèses/dissertations, les CV académiques, les présentations (Beamer), les affiches scientifiques, les lettres officielles, les examens/quiz, les livres, les aide-mémoire, les cartes de référence, les feuilles de formules d'examen, les formulaires PDF à remplir (champs de formulaire hyperref), le contenu conditionnel (basculements de la boîte à outils), le publipostage à partir de CSV/JSON (modèles Jinja2), les versions différentes (latexdiff), les graphiques. (pgfplots + matplotlib), tableaux (booktabs + import CSV), images (TikZ), diagrammes Mermaid, images générées par l'IA, filigranes, pages paysage, bibliographie/citations (BibTeX/biblatex), multi-langue/CJK (auto XeLaTeX), algorithmes/pseudocode, cases colorées (tcolorbox), unités SI (siunitx), conversion de format Pandoc (Markdown/DOCX/HTML ↔ LaTeX) et conversion PDF vers LaTeX de documents manuscrits ou imprimés (mathématiques, commerciaux, juridiques, généraux). Le script de compilation prend en charge pdflatex, xelatex, lualatex avec détection automatique, backend latexmk, filtrage des journaux texfot, sortie PDF/A et contrôle de la verbosité (--verbose/--quiet). Mise à l'échelle empiriquement optimisée : agent unique 1 à 10 pages, fractionné 11 à 20, pipeline batch-7 21+.
