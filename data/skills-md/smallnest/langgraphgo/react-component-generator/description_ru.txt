Создавайте компоненты React, следуя лучшим практикам использования TypeScript, Tailwind CSS и управления состоянием Zustand. Используйте этот навык, когда пользователь запрашивает создание компонентов React, элементов пользовательского интерфейса или упоминает создание компонентов. Поддерживает общие шаблоны компонентов, включая базовые компоненты, формы, списки, карточки, кнопки, модальные окна и компоненты с отслеживанием состояния, с интеграцией Zustand.
