Genera componenti React seguendo le best practice con TypeScript, Tailwind CSS e la gestione dello stato Zustand. Utilizza questa abilità quando l'utente richiede la creazione di componenti React, elementi dell'interfaccia utente o menziona la generazione di componenti. Supporta modelli di componenti comuni inclusi componenti di base, moduli, elenchi, schede, pulsanti, modali e componenti con stato con l'integrazione di Zustand.
