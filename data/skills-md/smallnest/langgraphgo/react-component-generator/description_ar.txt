قم بإنشاء مكونات React باتباع أفضل الممارسات مع إدارة حالة TypeScript وTailwind CSS وZustand. استخدم هذه المهارة عندما يطلب المستخدم إنشاء مكونات React أو عناصر واجهة المستخدم أو الإشارة إلى إنشاء المكونات. يدعم أنماط المكونات الشائعة بما في ذلك المكونات الأساسية والنماذج والقوائم والبطاقات والأزرار والنماذج والمكونات ذات الحالة مع تكامل Zustand.
