Directives complètes de développement de Laravel pour écrire du code propre, maintenable et idiomatique. Cette compétence fournit des règles et des exemples pour l'architecture des applications, l'optimisation des bases de données et le style de code. Utilisez-le lorsque : (1) vous écrivez ou refactorisez du code Laravel, (2) vous examinez les demandes d'extraction, (3) vous concevez l'architecture du système ou (4) vous optimisez les performances.
