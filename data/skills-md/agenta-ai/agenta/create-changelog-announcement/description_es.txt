Utilice esta habilidad para crear y publicar anuncios de registro de cambios para nuevas funciones, mejoras o correcciones de errores. Esta habilidad maneja el flujo de trabajo completo: crea páginas detalladas de documentación del registro de cambios, agrega tarjetas de anuncios en la barra lateral y garantiza que todo siga los estándares del proyecto. Úselo cuando el usuario mencione agregar entradas de registro de cambios, documentar nuevas funciones, crear notas de versión o anunciar actualizaciones de productos.
