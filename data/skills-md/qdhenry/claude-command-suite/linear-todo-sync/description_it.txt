Questa abilità recupera le attività aperte assegnate all'utente dall'API lineare e genera un file dell'elenco delle cose da fare con markdown nella radice del progetto. Questa abilità dovrebbe essere utilizzata quando l'utente chiede informazioni sui propri elementi di lavoro, vuole vedere su cosa deve lavorare o richiede di caricare/sincronizzare le proprie attività lineari. Richiede Python 3.7+, request, mdutils e pacchetti python-dotenv. Richiede LINEAR_API_KEY nel file .env.
