Cette compétence récupère les tâches ouvertes assignées à l'utilisateur à partir de l'API linéaire et génère un fichier de liste de tâches markdown à la racine du projet. Cette compétence doit être utilisée lorsque l'utilisateur pose des questions sur ses éléments de travail, souhaite voir sur quoi il doit travailler ou demande de charger/synchroniser ses tâches linéaires. Nécessite les packages Python 3.7+, requêtes, mdutils et python-dotenv. Nécessite LINEAR_API_KEY dans le fichier .env.
