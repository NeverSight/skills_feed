حواجز حماية HTML وCSS وأنماطها وأفضل الممارسات للتطوير بمساعدة الذكاء الاصطناعي.
يُستخدم عند العمل مع ملفات HTML/CSS (.html، و.css، و.scss، و.less)، أو عندما يذكر المستخدم HTML/CSS.
يوفر إرشادات HTML الدلالية، ومعايير إمكانية الوصول، وأنماط CSS الحديثة،
واتفاقيات التصميم سريعة الاستجابة الخاصة بمعايير الترميز الخاصة بهذا المشروع.
