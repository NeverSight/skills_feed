무언가가 작동하지 않을 때 Olakai 모니터링 문제를 해결하세요.
사용자가 다음을 언급할 때 자동 호출: 이벤트가 나타나지 않음, 잘못된 값을 표시하는 KPI, 숫자 대신 문자열을 표시하는 KPI, 사용자 정의 데이터 누락, Null KPI, 인증 오류, CLI가 작동하지 않음, 에이전트와 연결되지 않은 이벤트, 손상된 모니터링, SDK 오류 또는 Olakai 관련 문제.
트리거 키워드: olakai, 문제 해결, 디버그, 작동하지 않음, 이벤트 누락, KPI 잘못, KPI null, KPI 문자열, customData 누락, 인증 실패, CLI 오류, 이벤트 없음, 이벤트가 표시되지 않음, 진단, olakai 수정, 손상됨, SDK 오류, 모니터링 문제, API 키가 잘못됨, 이벤트가 추적되지 않음.
로드하지 마십시오: 초기 설정(olakai-create-agent 또는 olakai-add-monitoring 사용) 또는 보고서 생성(generate-analytics-reports 사용).
