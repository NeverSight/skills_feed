Solucionar problemas de monitoreo de Olakai cuando algo no funciona.
INVOCACIÓN AUTOMÁTICA cuando el usuario menciona: eventos que no aparecen, KPI que muestran valores incorrectos, KPI que muestran cadenas en lugar de números, faltan datos personalizados, KPI nulos, errores de autenticación, CLI no funciona, eventos no asociados con el agente, monitoreo roto, errores del SDK o cualquier problema relacionado con Olakai.
PALABRAS CLAVE DE ACTIVACIÓN: olakai, solución de problemas, depuración, no funciona, faltan eventos, KPI incorrecto, KPI nulo, cadena de KPI, faltan datos personalizados, error de autenticación, error de CLI, no hay eventos, los eventos no aparecen, diagnosticar, corregir olakai, roto, error de SDK, problema de monitoreo, clave API no válida, eventos no rastreados.
NO lo cargue para: configuración inicial (use olakai-create-agent o olakai-add-monitoring), ni para generar informes (use generate-analytics-reports).
