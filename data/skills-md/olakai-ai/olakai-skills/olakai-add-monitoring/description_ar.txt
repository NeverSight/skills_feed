أضف مراقبة Olakai إلى وكيل الذكاء الاصطناعي الحالي أو تكامل LLM.
الاستدعاء التلقائي عندما يريد المستخدم: إضافة مراقبة إلى التعليمات البرمجية الموجودة، أو دمج إمكانية المراقبة في وكيل عامل، أو استخدام نظام ذكاء اصطناعي موجود، أو إضافة تتبع لمكالمات LLM، أو تغليف عملاء OpenAI/Anthropic، أو إضافة تحليلات إلى كود الذكاء الاصطناعي العامل، أو تعديل المراقبة إلى إنتاج الذكاء الاصطناعي.
الكلمات الرئيسية للتشغيل: olakai، إضافة المراقبة، دمج المراقبة، الأداة، عميل التغليف، إمكانية الملاحظة، تكامل SDK، @olakai/sdk، olakai-sdk، الوكيل الحالي، التعليمات البرمجية الموجودة، إضافة التتبع، التحليلات، القياس عن بعد، المراقبة التحديثية، LLM الحالي، OpenAI الحالي، Anthropic الحالي.
لا تقم بالتحميل من أجل: إنشاء وكلاء جدد من البداية (استخدم olakai-create-agent)، أو استكشاف المشكلات وإصلاحها (استخدم olakai-Troubleshoot)، أو إنشاء التقارير (استخدم إنشاء تقارير التحليلات).
