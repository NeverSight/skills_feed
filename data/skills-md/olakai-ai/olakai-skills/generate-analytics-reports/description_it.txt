Genera report di analisi dai dati Olakai utilizzando i comandi CLI.
INVOCA AUTOMATICAMENTE quando l'utente desidera: riepiloghi sull'utilizzo, tendenze KPI, analisi dei rischi, report sul ROI, metriche di efficienza, confronti degli agenti, report sull'utilizzo dei token, analisi dei costi, report sulla conformità o qualsiasi analisi senza utilizzare la dashboard web.
PAROLE CHIAVE TRIGGER: Olakai, analisi, report, riepilogo utilizzo, tendenze KPI, analisi del rischio, ROI, efficienza, confronto agenti, utilizzo token, analisi dei costi, report parametri, dati dashboard, analisi CLI, report terminale, conformità, report utilizzo, riepilogo eventi, metriche prestazionali, statistiche utilizzo AI.
NON caricare per: impostazione del monitoraggio (usa olakai-add-monitoring), risoluzione dei problemi (usa olakai-troubleshoot) o creazione di nuovi agenti (usa olakai-create-agent).
