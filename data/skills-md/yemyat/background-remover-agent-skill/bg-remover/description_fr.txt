Supprimez les arrière-plans des images à l'aide du modèle BiRefNet de FAL.ai. À utiliser lorsque les utilisateurs demandent de supprimer l'arrière-plan, de créer un PNG transparent, d'extraire le sujet de l'image ou de créer des découpes. Les expressions déclencheurs incluent « supprimer l’arrière-plan », « arrière-plan transparent », « découper », « extraire le sujet » ou toute demande de suppression de l’arrière-plan.
