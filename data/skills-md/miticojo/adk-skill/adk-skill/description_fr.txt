Créez des systèmes mono-agent et multi-agents à l'aide du kit de développement d'agents (ADK) de Google en Python, Java, Go ou TypeScript. À utiliser lorsque l'utilisateur dit « créer un agent avec ADK », « créer un agent Gemini », « pipeline multi-agents », « orchestration d'agents avec Google » ou mentionne ADK, google-adk, le kit de développement d'agent Google, les agents séquentiels/parallèles/en boucle, les outils d'agent, les rappels, la gestion d'état, les tests d'agent ou le déploiement d'agent avec Gemini. Ne pas utiliser pour les frameworks d'agent LangChain, CrewAI, AutoGen ou non-ADK.
