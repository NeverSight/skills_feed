Govilo にファイルをアップロードし、Bot API 経由でロック解除リンクを生成します。次の場合に使用します: (1) ZIP、フォルダー、または個々のファイルから Govilo ロック解除リンクを作成する、(2) 署名付き URL を使用して Govilo R2 ストレージへのファイル アップロードを自動化する、(3) Govilo Bot API の対話を管理する (署名→アップロード→アイテムの作成)。 GOVILO_API_KEY および SELLER_ADDRESS 環境変数が必要です。見つからない場合は、https://govilo.xyz/ に登録するようにユーザーに案内します。
