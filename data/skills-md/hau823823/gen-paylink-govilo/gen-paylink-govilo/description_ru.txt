Загрузите файлы в Govilo и создайте ссылки для разблокировки через Bot API. Используйте в следующих случаях: (1) Создание ссылки для разблокировки Govilo из ZIP-архива, папки или отдельных файлов, (2) Автоматизация загрузки файлов в хранилище Govilo R2 с заранее заданными URL-адресами, (3) Управление взаимодействиями API Govilo Bot (предварительное оформление → загрузка → создание элемента). Требуются переменные среды GOVILO_API_KEY и SELLER_ADDRESS. Если он отсутствует, предлагает пользователю зарегистрироваться на https://govilo.xyz/.
