Carica file su Govilo e genera collegamenti di sblocco tramite Bot API. Da utilizzare quando: (1) si crea un collegamento di sblocco Govilo da un ZIP, una cartella o singoli file, (2) si automatizza il caricamento di file nell'archivio Govilo R2 con URL prefirmati, (3) si gestiscono le interazioni API Govilo Bot (presign → caricamento → crea elemento). Richiede GOVILO_API_KEY e SELLER_ADDRESS env vars. Se mancante, guida l'utente a registrarsi su https://govilo.xyz/.
