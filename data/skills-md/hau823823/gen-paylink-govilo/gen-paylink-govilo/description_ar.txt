قم بتحميل الملفات إلى Govilo وإنشاء روابط إلغاء القفل عبر Bot API. يُستخدم عندما: (1) إنشاء رابط فتح Govilo من ملف ZIP أو مجلد أو ملفات فردية، (2) أتمتة تحميل الملف إلى مساحة تخزين Govilo R2 باستخدام عناوين URL الموقعة مسبقًا، (3) إدارة تفاعلات Govilo Bot API (التوقيع المسبق → التحميل → إنشاء العنصر). يتطلب GOVILO_API_KEY وSELLER_ADDRESS env vars. إذا كان مفقودًا، قم بتوجيه المستخدم للتسجيل على https://govilo.xyz/.
