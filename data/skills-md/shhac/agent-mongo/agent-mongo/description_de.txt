Schreibgeschützte MongoDB-CLI für KI-Agenten. Verwenden Sie, wenn:
- Erkundung von MongoDB-Datenbanken, Sammlungen, Schemata oder Indizes
- Abfragen von Dokumenten (Suchen, Abrufen anhand der ID, Anzahl, Stichprobe, Unterscheidung, Aggregation)
- Verwalten von MongoDB-Verbindungen oder Anmeldeinformationen
- Überprüfung von Datenbank- oder Sammlungsstatistiken
Auslöser: „Mongodb“, „Mongo-Abfrage“, „Mongo-Suche“, „Mongo-Schema“, „Mongo-Sammlung“, „Mongo-Datenbank“, „Mongo-Verbindung“, „Mongo-Aggregat“, „Query Mongodb“, „Mongo-Statistik“
