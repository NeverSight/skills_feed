CLI MongoDB только для чтения для агентов ИИ. Используйте, когда:
- Исследование баз данных, коллекций, схем или индексов MongoDB.
- Запрос документов (найти, получить по идентификатору, подсчитать, выбрать, отличить, агрегировать)
- Управление подключениями или учетными данными MongoDB.
- Проверка базы данных или статистики коллекции
Триггеры: «mongodb», «запрос mongo», «поиск mongo», «схема mongo», «коллекция mongo», «база данных mongo», «соединение mongo», «агрегат mongo», «запрос mongodb», «статистика mongo».
