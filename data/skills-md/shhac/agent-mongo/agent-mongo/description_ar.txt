للقراءة فقط MongoDB CLI لوكلاء الذكاء الاصطناعي. استخدم عندما:
- استكشاف قواعد بيانات MongoDB أو المجموعات أو المخططات أو الفهارس
- الاستعلام عن المستندات (البحث، الحصول على المعرف، العد، العينة، التمييز، التجميع)
- إدارة اتصالات MongoDB أو بيانات الاعتماد
- التحقق من قاعدة البيانات أو إحصائيات المجموعة
