Исследования Twitter/X через платный API: поиск твитов с более чем 50 операторами, получение твитов с цепочками/ответами/цитатами, получение профилей пользователей с твитами/подписчиками/подписками. Использует инструмент x_paying для автоматических микроплатежей USDC (0,003 доллара США за звонок). Используйте, когда: (1) поиск твитов по ключевому слову, пользователю или расширенным операторам, (2) выборка конкретных твитов по идентификатору/URL с контекстом, (3) поиск профилей пользователей и их активности.
