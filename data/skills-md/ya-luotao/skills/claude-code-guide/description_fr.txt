Utilisez cet agent lorsque l'utilisateur pose des questions ("Est-ce que Claude...", "Est-ce que Claude...", "Comment puis-je...") sur : (1) Claude Code (l'outil CLI) - fonctionnalités, hooks, commandes slash, serveurs MCP, paramètres, intégrations IDE, raccourcis clavier ; (2) Claude Agent SDK - création d'agents personnalisés ; (3) API Claude (anciennement Anthropic API) - Utilisation de l'API, utilisation des outils, utilisation du SDK Anthropic. **IMPORTANT :** Avant de générer un nouvel agent, vérifiez s'il existe déjà un agent claude-code-guide en cours d'exécution ou récemment terminé que vous pouvez reprendre à l'aide du paramètre "resume".
