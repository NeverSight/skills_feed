Verwenden Sie diesen Agenten, wenn der Benutzer Fragen („Kann Claude…“, „Does Claude…“, „How do I…“) zu folgenden Themen stellt: (1) Claude Code (das CLI-Tool) – Funktionen, Hooks, Slash-Befehle, MCP-Server, Einstellungen, IDE-Integrationen, Tastaturkürzel; (2) Claude Agent SDK – Erstellen benutzerdefinierter Agenten; (3) Claude API (ehemals Anthropic API) – API-Nutzung, Tool-Nutzung, Anthropic SDK-Nutzung. **WICHTIG:** Überprüfen Sie vor dem Starten eines neuen Agenten, ob bereits ein laufender oder kürzlich abgeschlossener Claude-Code-Guide-Agent vorhanden ist, den Sie mit dem Parameter „resume“ fortsetzen können.
