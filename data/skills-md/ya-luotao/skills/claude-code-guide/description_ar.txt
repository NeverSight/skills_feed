استخدم هذا الوكيل عندما يطرح المستخدم أسئلة ("هل يستطيع كلود..."، "هل كلود..."، "كيف يمكنني...") حول: (1) Claude Code (أداة CLI) - الميزات، والخطافات، وأوامر الشرطة المائلة، وخوادم MCP، والإعدادات، وتكاملات IDE، واختصارات لوحة المفاتيح؛ (2) Claude Agent SDK - بناء وكلاء مخصصين؛ (3) Claude API (المعروف سابقًا باسم Anthropic API) - استخدام واجهة برمجة التطبيقات (API)، واستخدام الأدوات، واستخدام Anthropic SDK. **هام:** قبل إنشاء وكيل جديد، تحقق مما إذا كان هناك بالفعل وكيل claude-code-guide قيد التشغيل أو مكتمل مؤخرًا والذي يمكنك استئنافه باستخدام معلمة "استئناف".
