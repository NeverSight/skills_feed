Используйте этот агент, когда пользователь задает вопросы («Может ли Клод...», «Делает ли Клод...», «Как мне...») о: (1) Коде Клода (инструмент CLI) — функциях, перехватчиках, командах с косой чертой, серверах MCP, настройках, интеграции IDE, сочетаниях клавиш; (2) Claude Agent SDK — создание собственных агентов; (3) Claude API (ранее Anthropic API) — использование API, использование инструментов, использование Anthropic SDK. **ВАЖНО:** Прежде чем создавать новый агент, проверьте, существует ли уже работающий или недавно завершенный агент claude-code-guide, который можно возобновить с помощью параметра «resume».
