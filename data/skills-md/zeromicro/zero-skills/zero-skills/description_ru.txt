Обширная база знаний для нулевой инфраструктуры микросервисов. Используйте этот навык при работе с нулем, чтобы понять правильные шаблоны для REST API (архитектура обработчиков/логики/контекста), служб RPC (обнаружение служб, балансировка нагрузки), операций с базами данных (sqlx, MongoDB, кэширование), шаблонов устойчивости (автоматический выключатель, ограничение скорости) и устранения распространенных проблем. Необходим для создания готового к использованию нулевого кода, который следует соглашениям платформы.
