Base de connaissances complète pour le cadre de microservices go-zero. Utilisez cette compétence lorsque vous travaillez avec go-zero pour comprendre les modèles corrects pour les API REST (architecture gestionnaire/logique/contexte), les services RPC (découverte de services, équilibrage de charge), les opérations de base de données (sqlx, MongoDB, mise en cache), les modèles de résilience (disjoncteur, limitation de débit) et le dépannage des problèmes courants. Indispensable pour générer du code zéro prêt pour la production qui respecte les conventions-cadres.
