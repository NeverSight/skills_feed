Полная настройка SEO для приложений Next.js. Используйте, когда пользователь хочет реализовать или улучшить SEO в приложении Next.js, включая создание метаданных страницы, карты сайта (sitemap.xml и llms.txt), конфигурацию robots.txt, структурированные данные JSON-LD или SEO-аудит. Триггер для запросов о SEO-оптимизации Next.js, видимости в поисковых системах, управлении метаданными или когда пользователь упоминает о желании улучшить SEO для своего приложения Next.js.
