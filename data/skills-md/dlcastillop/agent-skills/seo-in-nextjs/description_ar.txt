أكمل إعداد SEO لتطبيقات Next.js. يُستخدم عندما يريد المستخدم تنفيذ تحسين محركات البحث أو تحسينه في تطبيق Next.js، بما في ذلك إنشاء البيانات التعريفية للصفحة، أو خرائط الموقع (sitemap.xml وllms.txt)، أو تكوين ملف robots.txt، أو البيانات المنظمة JSON-LD، أو تدقيق تحسين محركات البحث. مشغل للاستعلامات حول تحسين محركات البحث لـ Next.js، أو رؤية محرك البحث، أو إدارة البيانات الوصفية، أو عندما يذكر المستخدم رغبته في تحسين محركات البحث لـ Next.js لتطبيق Next.js الخاص به.
