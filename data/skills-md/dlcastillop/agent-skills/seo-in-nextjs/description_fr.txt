Configuration complète du référencement pour les applications Next.js. À utiliser lorsque l'utilisateur souhaite implémenter ou améliorer le référencement dans une application Next.js, y compris la génération de métadonnées de page, les plans de site (sitemap.xml et llms.txt), la configuration robots.txt, les données structurées JSON-LD ou l'audit SEO. Déclenchez des requêtes sur l'optimisation du référencement Next.js, la visibilité des moteurs de recherche, la gestion des métadonnées ou lorsque l'utilisateur mentionne vouloir un meilleur référencement pour son application Next.js.
