Este programa genera archivos de tareas de código estructurado a partir de descripciones aproximadas, ideas o planes de implementación de PDD. Detecta automáticamente el tipo de entrada y crea archivos de tareas de código con el formato adecuado siguiendo la especificación de formato de tarea de código de Amazon. Para los planes PDD, procesa los pasos de implementación uno a la vez para permitir el aprendizaje y la adaptación entre pasos.
