Cette solution génère des fichiers de tâches de code structuré à partir de descriptions approximatives, d'idées ou de plans de mise en œuvre de PDD. Il détecte automatiquement le type d'entrée et crée des fichiers de tâches de code correctement formatés conformément à la spécification de format de tâche de code d'Amazon. Pour les plans PDD, il traite les étapes de mise en œuvre une par une pour permettre l’apprentissage et l’adaptation entre les étapes.
