Эта процедура генерирует файлы задач структурированного кода на основе приблизительных описаний, идей или планов реализации PDD. Он автоматически определяет тип входных данных и создает файлы задач кода в правильном формате в соответствии со спецификацией формата задач кода Amazon. Для планов PDD он обрабатывает этапы реализации по одному, чтобы обеспечить возможность обучения и адаптации между этапами.
