Questo contentino genera file di attività in codice strutturato da descrizioni approssimative, idee o piani di implementazione PDD. Rileva automaticamente il tipo di input e crea file di attività di codice formattati correttamente seguendo le specifiche del formato dell'attività di codice di Amazon. Per i piani PDD, elabora le fasi di implementazione una alla volta per consentire l'apprendimento e l'adattamento tra le fasi.
