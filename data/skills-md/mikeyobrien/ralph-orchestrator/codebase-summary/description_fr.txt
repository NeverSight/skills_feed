Cette solution analyse une base de code et génère une documentation complète comprenant des fichiers de métadonnées structurées qui décrivent l'architecture du système, les composants, les interfaces et les flux de travail. Il peut créer des fichiers de documentation ciblés comme AGENTS.md (README pour les agents IA), README.md, CONTRIBUTING.md, ou générer un écosystème de documentation complet. La documentation est organisée pour permettre aux assistants IA de comprendre facilement le système et de les aider dans les tâches de développement.
