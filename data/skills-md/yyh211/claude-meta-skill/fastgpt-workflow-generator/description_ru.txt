Генерирует готовый к использованию рабочий процесс FastGPT в формате JSON на основе требований естественного языка. Использует семантическое сопоставление шаблонов на основе искусственного интеллекта из встроенных рабочих процессов (перевод документов, обучение продажам, проверка резюме, финансовые новости). Выполняет трехуровневую проверку (формат, связи, логическая полнота). Поддерживает дополнительные модификации для добавления/удаления/изменения узлов. Активируется, когда пользователь запрашивает «создать рабочий процесс FastGPT», «создать JSON рабочего процесса», «разработать приложение FastGPT» или упоминает автоматизацию рабочего процесса, многоагентные системы или шаблоны FastGPT.
