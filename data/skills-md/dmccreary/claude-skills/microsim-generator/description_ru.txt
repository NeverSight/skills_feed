Создает интерактивные образовательные MicroSims, используя наиболее подходящую библиотеку JavaScript (p5.js, Chart.js, Plotly, Mermaid, vis-network, vis-timeline, Leaflet, Venn.js). Анализирует требования пользователей для перехода к соответствующему типу визуализации и генерирует полные пакеты MicroSim с HTML, JavaScript, CSS, документацией и метаданными.
