Управляйте установленными навыками Claude Code: устанавливайте, обновляйте, переименовывайте, удаляйте и перечисляйте навыки по URL-адресам GitHub. Используйте, когда пользователь хочет установить навык, обновить навык, составить список установленных навыков, переименовать навык, удалить/удалить/удалить навык или предоставить URL-адрес GitHub для каталога навыков.
