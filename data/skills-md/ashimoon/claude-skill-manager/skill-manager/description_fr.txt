Gérez vos compétences Claude Code installées : installez, mettez à jour, renommez, désinstallez et répertoriez les compétences à partir des URL GitHub. À utiliser lorsque l'utilisateur souhaite installer une compétence, mettre à jour une compétence, répertorier les compétences installées, renommer une compétence, supprimer/supprimer/désinstaller une compétence ou fournir une URL GitHub vers un répertoire de compétences.
