Kenntnisse in den Bereichen GitHub Actions-Workflow-Erstellung, Sicherheit und Versionsverwaltung. Verwenden Sie es in den folgenden Situationen: (1) beim Schreiben einer neuen Workflow-Datei (.yml), (2) beim Ändern eines vorhandenen Workflows, (3) beim Überprüfen oder Aktualisieren von Aktionsversionen, (4) bei CI/CD-Sicherheitsüberprüfungen, (5) beim Arbeiten mit Schlüsselwörtern wie „actions/“, „uses:“, „workflow“, „.github/workflows“.
