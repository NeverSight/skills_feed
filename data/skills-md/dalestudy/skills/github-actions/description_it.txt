Competenze nella creazione di flussi di lavoro, nella sicurezza e nella gestione delle versioni di GitHub Actions. Utilizzare nelle seguenti situazioni: (1) quando si scrive un nuovo file di flusso di lavoro (.yml), (2) quando si modifica un flusso di lavoro esistente, (3) durante la revisione o l'aggiornamento delle versioni dell'azione, (4) durante i controlli di sicurezza CI/CD, (5) quando si lavora con parole chiave come 'actions/', 'uses:', 'workflow', '.github/workflows'
