Навыки создания рабочих процессов GitHub Actions, обеспечения безопасности и управления версиями. Используйте в следующих ситуациях: (1) при написании нового файла рабочего процесса (.yml), (2) при изменении существующего рабочего процесса, (3) при просмотре или обновлении версий действий, (4) во время аудита безопасности CI/CD, (5) при работе с такими ключевыми словами, как «actions/», «uses:», «workflow», «.github/workflows».
