Se utiliza al crear una nueva habilidad o modificar una existente en el repositorio de DaleStudy/skills. Se activa en las siguientes situaciones: (1) cuando se solicita la creación de una nueva habilidad, (2) cuando se escribe o modifica un archivo SKILL.md, (3) cuando se hacen preguntas relacionadas con la estructura de la habilidad o el frontmatter, (4) cuando se trabaja con palabras clave como 'skill', 'SKILL.md' o 'frontmatter'
