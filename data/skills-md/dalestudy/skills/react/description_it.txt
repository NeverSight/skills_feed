Ottimizzazione delle prestazioni di React e competenze relative alle migliori pratiche. Basato sulla guida Vercel Engineering, indipendente dal framework. Utilizzare nelle seguenti situazioni: (1) quando si scrivono o si modificano componenti React (.tsx, .jsx), (2) quando si lavora sulla gestione dello stato, sugli hook o sull'ottimizzazione del rerendering, (3) quando si gestisce il recupero di dati asincroni o modelli di suspense, (4) quando si ottimizzano le dimensioni del bundle o la suddivisione del codice, (5) quando si lavora con parole chiave come "react", "useState", "useEffect", 'useMemo', 'useCallback', 'memo', 'Suspense', 'pigro'
