Guide to interacting with the GitHub platform using GitHub CLI (gh). Use it in the following situations: (1) when creating, viewing, or modifying GitHub issues, (2) when creating, reviewing, or merging pull requests, (3) when creating and managing GitHub releases, (4) when managing projects such as labels and milestones, (5) when running GitHub Actions workflows and checking their results, (6) when working with keywords like 'gh', 'issue', 'pull request', 'PR', 'release', 'label', 'workflow', 'run'
