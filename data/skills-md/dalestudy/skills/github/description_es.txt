Guía para interactuar con la plataforma GitHub usando GitHub CLI (gh). Úselo en las siguientes situaciones: (1) al crear, ver o modificar problemas de GitHub, (2) al crear, revisar o fusionar solicitudes de extracción, (3) al crear y administrar lanzamientos de GitHub, (4) al administrar proyectos como etiquetas e hitos, (5) al ejecutar flujos de trabajo de GitHub Actions y verificar sus resultados, (6) al trabajar con palabras clave como 'gh', 'issue', 'pull request', 'PR', 'release', 'etiqueta', 'flujo de trabajo', 'ejecutar'
