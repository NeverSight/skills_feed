Guide GitHub CLI (gh) pour interagir avec la plateforme GitHub. À utiliser dans les situations suivantes : (1) lors de la création, de l'affichage ou de la modification de problèmes GitHub ; (2) lors de la création, de la révision ou de la fusion de demandes d'extraction ; (3) lors de la création et de la gestion des versions de GitHub ; (4) lors de la gestion de projets, y compris les étiquettes et les jalons ; (5) lors de l'exécution des workflows GitHub Actions et de la vérification des résultats ; (6) lorsque vous travaillez avec des mots-clés tels que « gh », « issue », « pull request », « PR », « release », « label », « workflow », « run »
