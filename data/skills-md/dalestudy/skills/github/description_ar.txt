دليل التفاعل مع منصة GitHub باستخدام GitHub CLI (gh). استخدمها في المواقف التالية: (1) عند إنشاء مشكلات GitHub أو عرضها أو تعديلها، (2) عند إنشاء طلبات السحب أو مراجعتها أو دمجها، (3) عند إنشاء إصدارات GitHub وإدارتها، (4) عند إدارة المشاريع مثل التسميات والمعالم الرئيسية، (5) عند تشغيل مهام سير عمل GitHub Actions والتحقق من نتائجها، (6) عند العمل باستخدام كلمات رئيسية مثل "gh"، و"issue"، و"pull request"، و"PR"، "الإصدار"، "التسمية"، "سير العمل"، "التشغيل"
