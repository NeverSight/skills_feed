دليل GitHub CLI (gh) للتفاعل مع منصة GitHub. يُستخدم في المواقف التالية: (1) عند إنشاء مشكلات GitHub أو عرضها أو تعديلها؛ (2) عند إنشاء طلبات السحب أو مراجعتها أو دمجها؛ (3) عند إنشاء إصدارات GitHub وإدارتها؛ (4) عند إدارة المشاريع، بما في ذلك التسميات والمعالم الرئيسية؛ (5) عند تشغيل سير عمل GitHub Actions والتحقق من النتائج؛ (6) عند العمل باستخدام كلمات رئيسية مثل "gh"، و"إصدار"، و"طلب سحب"، و"PR"، و"release"، و"label"، و"workflow"، و"run"
