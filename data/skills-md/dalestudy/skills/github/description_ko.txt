GitHub 플랫폼과 상호 작용하기 위한 GitHub CLI(gh) 가이드입니다. 다음 상황에서 사용하십시오: (1) GitHub 문제를 생성, 보기 또는 수정할 때; (2) 풀 요청을 생성, 검토 또는 병합할 때; (3) GitHub 릴리스를 생성하고 관리할 때; (4) 라벨 및 마일스톤을 포함한 프로젝트를 관리할 때; (5) GitHub Actions 워크플로를 실행하고 결과를 확인할 때; (6) 'gh', 'issue', 'pull request', 'PR', 'release', 'label', 'workflow', 'run'과 같은 키워드로 작업할 때
