GitHub CLI (gh) Guida per interagire con la piattaforma GitHub. Utilizzare nelle seguenti situazioni: (1) durante la creazione, la visualizzazione o la modifica di problemi di GitHub; (2) durante la creazione, la revisione o l'unione di richieste pull; (3) durante la creazione e la gestione delle versioni di GitHub; (4) durante la gestione dei progetti, comprese le etichette e le tappe fondamentali; (5) durante l'esecuzione dei flussi di lavoro GitHub Actions e il controllo dei risultati; (6) quando si lavora con parole chiave come "gh", "issue", "pull request", "PR", "release", "label", "workflow", "run"
