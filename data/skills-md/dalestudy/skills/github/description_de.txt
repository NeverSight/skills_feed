GitHub CLI (gh) Leitfaden für die Interaktion mit der GitHub-Plattform. Verwenden Sie es in den folgenden Situationen: (1) beim Erstellen, Anzeigen oder Ändern von GitHub-Problemen; (2) beim Erstellen, Überprüfen oder Zusammenführen von Pull-Anfragen; (3) beim Erstellen und Verwalten von GitHub-Releases; (4) bei der Verwaltung von Projekten, einschließlich Labels und Meilensteinen; (5) beim Ausführen von GitHub Actions-Workflows und beim Überprüfen der Ergebnisse; (6) bei der Arbeit mit Schlüsselwörtern wie „gh“, „issue“, „pull request“, „PR“, „release“, „label“, „workflow“, „run“
