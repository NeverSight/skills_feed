Руководство по взаимодействию с платформой GitHub с помощью GitHub CLI (gh). Используйте его в следующих ситуациях: (1) при создании, просмотре или изменении задач GitHub, (2) при создании, просмотре или объединении запросов на включение, (3) при создании выпусков GitHub и управлении ими, (4) при управлении такими проектами, как метки и контрольные точки, (5) при запуске рабочих процессов GitHub Actions и проверке их результатов, (6) при работе с такими ключевыми словами, как «gh», «issue», «pull request», «PR», «выпустить», «метка», «рабочий процесс», «запустить»
