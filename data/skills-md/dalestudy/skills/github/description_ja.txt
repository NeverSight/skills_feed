GitHub CLI (gh) を使用して GitHub プラットフォームと対話するためのガイド。次の状況で使用します: (1) GitHub の課題を作成、表示、または変更するとき、(2) プル リクエストを作成、レビュー、またはマージするとき、(3) GitHub リリースを作成および管理するとき、(4) ラベルやマイルストーンなどのプロジェクトを管理するとき、(5) GitHub Actions ワークフローを実行し、その結果を確認するとき、(6) 「gh」、「issue」、「pull request」、「PR」などのキーワードを操作するとき、 「リリース」、「ラベル」、「ワークフロー」、「実行」
