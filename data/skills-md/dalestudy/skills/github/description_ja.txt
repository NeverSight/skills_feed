GitHub プラットフォームと対話するための GitHub CLI (gh) ガイド。次の状況で使用します: (1) GitHub の問題を作成、表示、または変更するとき。 (2) プル リクエストを作成、レビュー、またはマージするとき。 (3) GitHub リリースを作成および管理するとき。 (4) ラベルやマイルストーンを含むプロジェクトを管理するとき。 (5) GitHub Actions ワークフローを実行し、結果を確認するとき。 (6) 「gh」、「issue」、「pull request」、「PR」、「release」、「label」、「workflow」、「run」などのキーワードを使用する場合
