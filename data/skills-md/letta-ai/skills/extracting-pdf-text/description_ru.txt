Извлекайте текст из PDF-файлов для использования в программе LLM. Используйте при обработке PDF-файлов для RAG, анализа документов или извлечения текста. Поддерживает сервисы API (Mistral OCR) и локальные инструменты (PyMuPDF, pdfplumber). Обрабатывает текстовые PDF-файлы, таблицы и отсканированные документы с помощью OCR.
