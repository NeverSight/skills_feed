Kenntnisse im Umgang mit PR-Code-Reviews. Wird verwendet, wenn dies durch einen PR-Überprüfungskommentar, eine Überprüfungsanfrage oder die Aufforderung zur Überprüfung von Codeänderungen ausgelöst wird. Bietet einen Workflow zum Lesen von Bewertungskommentaren, zum Verstehen von Feedback und zum Iterieren von Änderungen.
