Verwenden Sie diese Option, wenn der Benutzer eine Next.js-Migration planen, eine Migrations-Roadmap erstellen, die Migrationsreihenfolge festlegen, Migrationsphasen planen oder fragen möchte: „In welcher Reihenfolge soll ich Routen migrieren?“ Analysiert Routen über AST und generiert einen phasenweisen Migrationsplan (.migration/plan.md). Erfordert zunächst eine Migrationsbewertung.
