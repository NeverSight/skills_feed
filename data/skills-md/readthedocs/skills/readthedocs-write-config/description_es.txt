Cree o actualice los archivos de configuración Read the Docs `.readthedocs.yaml` v2 para compilaciones de Sphinx o MkDocs, incluidas imágenes/herramientas de compilación, instalaciones de dependencia, formatos, trabajos/comandos de compilación personalizados, entornos conda, submódulos y configuraciones de búsqueda. Úselo cuando un usuario solicita un archivo de configuración de lectura de documentos, cambios de YAML o actualizaciones de comportamiento de compilación.
