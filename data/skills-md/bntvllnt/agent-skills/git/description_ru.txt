Унифицированный рабочий процесс git для разработки с приоритетом ветвей: проверка статуса/разниц, коммиты с приоритетом безопасности,
рабочие деревья и создание/проверка PR через gh.
Автоматически активируется при: «фиксация», «нажатие», «ветвь», «рабочее дерево», «pr», «запрос на извлечение», «слияние», «перебазирование», «git».
