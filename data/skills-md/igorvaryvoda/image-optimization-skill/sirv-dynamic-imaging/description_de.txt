Sirv Dynamic Imaging URL API für die sofortige Bildtransformation. Verwenden Sie es, wenn Sie Bild-URLs mit Sirv CDN erstellen, die Größe von Bildern über URL-Parameter ändern, Wasserzeichen/Textüberlagerungen hinzufügen, zuschneiden, Filter anwenden, Formatkonvertierungen (WebP, AVIF) oder jede Sirv-URL-basierte Bildbearbeitung durchführen. Deckt über 100 URL-Parameter für Skalierung, Zuschnitt, Effekte, Text, Wasserzeichen, Rahmen und Optimierung ab.
