API URL di imaging dinamico Sirv per la trasformazione delle immagini al volo. Utilizzare quando si creano URL di immagini con Sirv CDN, ridimensionando le immagini tramite parametri URL, aggiungendo filigrane/sovrapposizioni di testo, ritagliando, applicando filtri, conversione di formato (WebP, AVIF) o qualsiasi manipolazione di immagini basata su URL Sirv. Copre oltre 100 parametri URL per ridimensionamento, ritaglio, effetti, testo, filigrane, cornici e ottimizzazione.
