API URL-адресов динамических изображений Sirv для оперативного преобразования изображений. Используйте при создании URL-адресов изображений с помощью Sirv CDN, изменении размера изображений с помощью параметров URL-адреса, добавлении водяных знаков/наложений текста, обрезке, применении фильтров, преобразовании формата (WebP, AVIF) или любых манипуляциях с изображениями на основе URL-адресов Sirv. Охватывает более 100 параметров URL-адресов для масштабирования, обрезки, эффектов, текста, водяных знаков, рамок и оптимизации.
