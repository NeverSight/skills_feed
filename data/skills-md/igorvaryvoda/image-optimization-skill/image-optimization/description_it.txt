Guida di esperti sull'ottimizzazione delle immagini per le prestazioni web. Da utilizzare quando si lavora con formati di immagine (WebP, AVIF, JPEG, PNG, GIF, SVG, HEIC, JPEG XL), impostazioni di compressione, immagini reattive, caricamento lento, CDN, Core Web Vitals o qualsiasi attività di sviluppo Web correlata alle immagini. Copre la selezione del formato, le impostazioni di qualità, srcset/dimensioni, elemento immagine, direzione artistica, fetchpriority, strategie segnaposto (LQIP, blur-up, blurhash), query contenitore, HDR/ampia gamma di colori, strumenti di immagine basati sull'intelligenza artificiale, elaborazione edge/serverless e ottimizzazione delle prestazioni.
