Integración de Sirv REST API para gestión de imágenes y archivos. Úselo cuando trabaje con Sirv CDN, cargue/descargue archivos a Sirv, administre metadatos de imágenes, busque archivos, cree giros de 360, convierta videos o cualquier operación de la API de Sirv. Cubre autenticación, operaciones de archivos, metadatos, consultas de búsqueda, trabajos asíncronos y administración de cuentas.
