Интеграция Sirv REST API для управления изображениями и файлами. Используйте при работе с Sirv CDN, загрузке/скачивании файлов в Sirv, управлении метаданными изображений, поиске файлов, создании 360-градусных вращений, конвертации видео или любых операциях Sirv API. Охватывает аутентификацию, файловые операции, метаданные, поисковые запросы, асинхронные задания и управление учетными записями.
