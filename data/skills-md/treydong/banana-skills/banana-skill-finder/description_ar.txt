اكتشاف مهارات Claude ذات الصلة والتوصية بها تلقائيًا عندما يواجه المستخدمون مهام يمكن أن تستفيد من القدرات المتخصصة. استخدم هذه المهارة بشكل استباقي عند اكتشاف أي من هذه الأنماط: (1) يذكر المستخدم العمل بتنسيقات ملفات محددة (PDF وDOCX وExcel والصور وما إلى ذلك)، (2) يصف المستخدم المهام المتكررة أو المتخصصة (تحليل البيانات، مراجعة التعليمات البرمجية، النشر، الاختبار، معالجة المستندات)، (3) يسأل المستخدم عما إذا كانت هناك أداة أو قدرة على شيء ما، (4) يواجه المستخدم صعوبة في العمل الخاص بمجال معين (تطوير React، استعلامات SQL، DevOps، كتابة المحتوى)، (5) يذكر المستخدم الحاجة أفضل الممارسات أو الأنماط للتكنولوجيا، (6) أي موقف حيث يمكن لمهارة متخصصة توفير الوقت أو تحسين الجودة. ابحث باستخدام SkillsMP API (إذا تم تكوينها)، أو لوحة المتصدرين Skills.sh، أو GitHub كبديل. يوصي بـ 1-3 من المهارات الأكثر صلة ويعرض التثبيت عبر إضافة مهارات npx.
