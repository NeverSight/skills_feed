Автоматически обнаруживайте и рекомендуйте соответствующие навыки Клода, когда пользователи сталкиваются с задачами, для которых могут быть полезны специальные возможности. Активно используйте этот навык при обнаружении любого из этих шаблонов: (1) Пользователь упоминает работу с определенными форматами файлов (PDF, DOCX, Excel, изображения и т. д.), (2) Пользователь описывает повторяющиеся или специализированные задачи (анализ данных, проверка кода, развертывание, тестирование, обработка документов), (3) Пользователь спрашивает, есть ли для чего-то инструмент или возможности, (4) Пользователь испытывает трудности с работой, специфичной для предметной области (разработка React, SQL-запросы, DevOps, написание контента), (5) Пользователь упоминает, что ему нужно лучшее практики или шаблоны для технологии. (6) Любая ситуация, в которой специальные навыки могут сэкономить время или улучшить качество. Выполняйте поиск с помощью SkillsMP API (если настроено), таблицы лидеровkills.sh или GitHub в качестве резервного варианта. Порекомендуйте 1-3 наиболее подходящих навыка и предложите установить их через добавление навыков npx.
