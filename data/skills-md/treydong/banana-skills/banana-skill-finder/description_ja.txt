ユーザーが特殊な機能から恩恵を受ける可能性のあるタスクに遭遇した場合、関連するクロード スキルを自動的に検出して推奨します。次のパターンのいずれかを検出した場合は、このスキルを積極的に使用してください: (1) ユーザーが特定のファイル形式の作業について言及 (PDF、DOCX、Excel、画像など)、(2) ユーザーが反復的または特殊なタスクについて説明 (データ分析、コード レビュー、展開、テスト、文書処理)、(3) ユーザーが何かのツールや機能があるかどうか尋ねる、(4) ユーザーがドメイン固有の作業に苦労している (React 開発、SQL クエリ、DevOps、コンテンツ作成)、(5) ユーザーが言及テクノロジーのベスト プラクティスやパターンが必要な場合、(6) 専門的なスキルによって時間の節約や品質の向上が期待できる状況。 SkillsMP API (構成されている場合)、skills.sh リーダーボード、またはフォールバックとして GitHub を使用して検索します。最も関連性の高いスキルを 1 ～ 3 つ推奨し、npx スキル追加経由でインストールすることを提案します。
