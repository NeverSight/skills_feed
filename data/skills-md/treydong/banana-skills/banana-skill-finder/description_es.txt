Descubra y recomiende automáticamente habilidades relevantes de Claude cuando los usuarios encuentren tareas que podrían beneficiarse de capacidades especializadas. Utilice esta habilidad de forma proactiva al detectar cualquiera de estos patrones: (1) El usuario menciona trabajar con formatos de archivos específicos (PDF, DOCX, Excel, imágenes, etc.), (2) El usuario describe tareas repetitivas o especializadas (análisis de datos, revisión de código, implementación, pruebas, procesamiento de documentos), (3) El usuario pregunta si hay una herramienta o capacidad para algo, (4) El usuario tiene dificultades con el trabajo de un dominio específico (desarrollo de React, consultas SQL, DevOps, redacción de contenido), (5) El usuario menciona necesitar mejores prácticas o patrones para una tecnología, (6) Cualquier situación en la que una habilidad especializada podría ahorrar tiempo o mejorar la calidad. Busque utilizando la API SkillsMP (si está configurada), la tabla de clasificación de skills.sh o GitHub como alternativa. Recomiende de 1 a 3 habilidades más relevantes y ofrezca instalarlas a través del complemento de habilidades npx.
