Entdecken und empfehlen Sie automatisch relevante Claude-Fähigkeiten, wenn Benutzer auf Aufgaben stoßen, die von speziellen Fähigkeiten profitieren könnten. Nutzen Sie diese Fähigkeit proaktiv, wenn Sie eines dieser Muster erkennen: (1) Benutzer erwähnt die Arbeit mit bestimmten Dateiformaten (PDF, DOCX, Excel, Bilder usw.), (2) Benutzer beschreibt sich wiederholende oder spezielle Aufgaben (Datenanalyse, Codeüberprüfung, Bereitstellung, Tests, Dokumentenverarbeitung), (3) Benutzer fragt, ob es für etwas ein Tool oder eine Funktion gibt, (4) Benutzer hat Probleme mit domänenspezifischer Arbeit (React-Entwicklung, SQL-Abfragen, DevOps, Inhaltserstellung), (5) Benutzer erwähnt, dass er das Beste braucht Praktiken oder Muster für eine Technologie, (6) Jede Situation, in der eine spezielle Fähigkeit Zeit sparen oder die Qualität verbessern könnte. Suchen Sie mit der SkillsMP-API (sofern konfiguriert), dem skills.sh-Leaderboard oder GitHub als Fallback. Empfehlen Sie 1-3 relevanteste Fähigkeiten und bieten Sie die Installation über npx skills add an.
