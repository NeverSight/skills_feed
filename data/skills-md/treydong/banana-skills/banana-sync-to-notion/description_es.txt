Sincronice archivos locales de Markdown con Notion con soporte completo de formato. Úselo cuando el usuario quiera hacer una copia de seguridad, sincronizar o migrar archivos a Notion, mencione cargarlos en Notion o diga sincronizar con Notion. Las características incluyen: (1) Preservación de la estructura de directorios recursiva, (2) Formato Markdown completo (negrita, cursiva, código, enlaces, listas, tablas, llamadas), (3) Iconos emoji automáticos basados ​​en nombres de archivos, (4) Detección de duplicados para sincronizaciones incrementales, (5) Fragmentación inteligente para archivos grandes, (6) Conversión de enlaces relativos a enlaces de páginas de Notion. Requiere el token API de Notion y el ID de la página de destino configurado en el archivo .env.
