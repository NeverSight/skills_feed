Sincronizza i file Markdown locali su Notion con il supporto completo della formattazione. Da utilizzare quando l'utente desidera eseguire il backup, sincronizzare o migrare file su Notion, menziona il caricamento su Notion o dice sincronizza su Notion. Le caratteristiche includono: (1) Conservazione ricorsiva della struttura delle directory, (2) Formattazione Markdown completa (grassetto, corsivo, codice, collegamenti, elenchi, tabelle, didascalie), (3) Icone emoji automatiche basate sui nomi dei file, (4) Rilevamento duplicati per sincronizzazioni incrementali, (5) Chunking intelligente per file di grandi dimensioni, (6) Conversione relativa dei collegamenti in collegamenti alle pagine di Notion. Richiede il token API Notion e l'ID della pagina di destinazione configurati nel file .env.
