Synchronisez les fichiers Markdown locaux avec Notion avec une prise en charge complète du formatage. À utiliser lorsque l'utilisateur souhaite sauvegarder, synchroniser ou migrer des fichiers vers Notion, mentionne le téléchargement vers Notion ou dit la synchronisation avec Notion. Les fonctionnalités incluent : (1) Préservation récursive de la structure des répertoires, (2) Formatage Markdown complet (gras, italique, code, liens, listes, tableaux, légendes), (3) Icônes emoji automatiques basées sur les noms de fichiers, (4) Détection des doublons pour les synchronisations incrémentielles, (5) Segmentation intelligente pour les fichiers volumineux, (6) Conversion de liens relatifs en liens de page Notion. Nécessite un jeton API Notion et un ID de page cible configurés dans le fichier .env.
