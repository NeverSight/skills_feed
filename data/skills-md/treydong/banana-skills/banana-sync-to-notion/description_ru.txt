Синхронизируйте локальные файлы Markdown с Notion с полной поддержкой форматирования. Используйте, когда пользователь хочет выполнить резервное копирование, синхронизировать или перенести файлы в Notion, упоминает о загрузке в Notion или говорит «Синхронизировать с Notion». Функции включают в себя: (1) Рекурсивное сохранение структуры каталогов, (2) Полное форматирование Markdown (жирный шрифт, курсив, код, ссылки, списки, таблицы, выноски), (3) Автоматические значки эмодзи на основе имен файлов, (4) Обнаружение дубликатов для инкрементальной синхронизации, (5) Интеллектуальное разбиение на блоки для больших файлов, (6) Преобразование относительных ссылок в ссылки на страницы Notion. Требуется токен Notion API и идентификатор целевой страницы, настроенные в файле .env.
