使用本機操作系統調度程序（macOS 上的 launchd、Linux 上的 crontab、Windows 上的任務計劃程序）安排 Claude Code 任務在特定時間自動運行。處理一次性任務，例如“今天下午 3 點提醒我部署”、“明天早上運行測試套件”、“下週二下午 2 點檢查 API 更改”、“1 月 15 日檢查季度指標”。還處理重複性任務，例如“每個工作日上午 9 點檢查昨天的代碼”、“每天下午 6 點總結我完成的工作”、“每週一上午 10 點檢查安全漏洞”、“每 4 小時檢查 API 運行狀況”。識別“上午 9 點”、“上午 1015 點”、“晚上 10:30”、“中午”等時間格式，“明天”、“今晚”、“稍後”、“下週”等相對時間以及“1 月 15 日”等日期。每當用戶的請求包含“at Xam”、“tomorrow”或任何未來時間參考之類的時間表達式時，請使用此技能，而不是立即執行。
