Úselo cuando necesite acceder a servidores MCP sin instalarlos directamente en Claude Code. MCPorter proporciona acceso CLI a cualquier servidor MCP a través de npx, lo que permite el descubrimiento de herramientas, llamadas directas y conexiones ad-hoc. Úselo para acceder a capacidades de MCP externas, como automatización del navegador, integraciones de API o cualquier herramienta basada en MCP.
