Verwenden Sie diese Option, wenn Sie auf MCP-Server zugreifen müssen, ohne diese direkt in Claude Code zu installieren. MCPorter bietet CLI-Zugriff auf jeden MCP-Server über npx und ermöglicht so die Tool-Erkennung, direkte Aufrufe und Ad-hoc-Verbindungen. Für den Zugriff auf externe MCP-Funktionen wie Browserautomatisierung, API-Integrationen oder andere MCP-basierte Tools verwenden.
