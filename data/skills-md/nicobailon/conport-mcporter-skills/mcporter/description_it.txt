Da utilizzare quando è necessario accedere ai server MCP senza installarli direttamente in Claude Code. MCPorter fornisce l'accesso CLI a qualsiasi server MCP tramite npx, consentendo il rilevamento degli strumenti, chiamate dirette e connessioni ad hoc. Da utilizzare per accedere a funzionalità MCP esterne come l'automazione del browser, le integrazioni API o qualsiasi strumento basato su MCP.
