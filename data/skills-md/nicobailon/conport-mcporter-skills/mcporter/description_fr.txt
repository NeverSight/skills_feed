À utiliser lorsque vous devez accéder aux serveurs MCP sans les installer directement dans Claude Code. MCPorter fournit un accès CLI à n'importe quel serveur MCP via npx, permettant la découverte d'outils, les appels directs et les connexions ad hoc. À utiliser pour accéder aux fonctionnalités MCP externes telles que l'automatisation du navigateur, les intégrations d'API ou tout outil basé sur MCP.
