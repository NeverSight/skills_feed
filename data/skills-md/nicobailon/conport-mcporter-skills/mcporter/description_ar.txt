استخدمه عندما تحتاج إلى الوصول إلى خوادم MCP دون تثبيتها مباشرة في Claude Code. يوفر MCPorter وصول CLI إلى أي خادم MCP عبر npx، مما يتيح اكتشاف الأدوات والمكالمات المباشرة والاتصالات المخصصة. يُستخدم للوصول إلى إمكانات MCP الخارجية مثل أتمتة المتصفح أو تكامل واجهة برمجة التطبيقات أو أي أدوات تعتمد على MCP.
