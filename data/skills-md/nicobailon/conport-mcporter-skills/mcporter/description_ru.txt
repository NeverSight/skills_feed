Используйте, когда вам нужно получить доступ к серверам MCP без их установки непосредственно в Claude Code. MCPorter обеспечивает доступ CLI к любому серверу MCP через npx, обеспечивая обнаружение инструментов, прямые вызовы и специальные соединения. Используйте для доступа к внешним возможностям MCP, таким как автоматизация браузера, интеграция API или любые инструменты на основе MCP.
