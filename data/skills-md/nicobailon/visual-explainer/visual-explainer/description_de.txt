Generieren Sie schöne, eigenständige HTML-Seiten, die Systeme, Codeänderungen, Pläne und Daten visuell erklären. Verwenden Sie es, wenn der Benutzer ein Diagramm, eine Architekturübersicht, eine Diff-Überprüfung, eine Planüberprüfung, eine Projektzusammenfassung, eine Vergleichstabelle oder eine visuelle Erklärung technischer Konzepte anfordert. Verwenden Sie es auch proaktiv, wenn Sie eine komplexe ASCII-Tabelle (mehr als vier Zeilen oder mehr als drei Spalten) rendern möchten – präsentieren Sie sie stattdessen als gestaltete HTML-Seite.
