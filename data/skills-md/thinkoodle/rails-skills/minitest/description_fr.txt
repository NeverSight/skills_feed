Conseils d'experts pour rédiger des tests Minitest rapides et maintenables dans les applications Rails. À utiliser lors de l'écriture de tests, de la conversion à partir de RSpec, du débogage des échecs de test, de l'amélioration des performances des tests ou du respect des meilleures pratiques de test. Couvre les tests de modèle, les tests de stratégie, les tests de requêtes, les tests système, les appareils et les flux de travail TDD.
