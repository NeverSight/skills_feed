Cree nuevos elementos de interfaz de usuario para el registro tryelements.dev. Úselo cuando: (1) Agregar nuevos componentes de la interfaz de usuario (botones, entradas, tarjetas), (2) Crear componentes de integración (Clerk, Stripe, Uploadthing), (3) Crear elementos relacionados con el tema, (4) Cualquier componente de registro estilo shadcn. IMPORTANTE: Para los componentes del logotipo con variantes (icono/marca denominativa/logotipo + oscuro/claro), utilice la habilidad de logotipo con variantes. Esta habilidad incluye andamiaje, esquema de registro y patrones de componentes. SIEMPRE utilice Context7 MCP para obtener los documentos de dependencia más recientes antes de implementarlos.
