Créez de nouveaux éléments d’interface utilisateur pour le registre tryelements.dev. À utiliser lorsque : (1) Ajout de nouveaux composants d'interface utilisateur (boutons, entrées, cartes), (2) Création de composants d'intégration (Clerk, Stripe, Uploadthing), (3) Création d'éléments liés au thème, (4) Tout composant de registre de style shadcn. IMPORTANT : pour les composants de logo avec des variantes (icône/mot-symbole/logo + sombre/clair), utilisez plutôt la compétence logo avec variantes. Cette compétence inclut l'échafaudage, le schéma de registre et les modèles de composants. Utilisez TOUJOURS Context7 MCP pour récupérer les derniers documents de dépendance avant l'implémentation.
