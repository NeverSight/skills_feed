API создания видео-аватара HeyGen AI. Используйте, когда: (1) создание видео-аватаров AI с помощью /v2/video/generate, (2) использование Video Agent для создания однократного запроса к видео, (3) работа с аватарами, голосами, фоном или подписями HeyGen, (4) создание прозрачных видеороликов WebM для компоновки, (5) опрос статуса видео или обработка веб-перехватчиков, (6) интеграция HeyGen с Remotion для программного видео, (7) перевод или дублирование существующих видео.
