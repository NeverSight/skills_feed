# 海报设计规范

## Prompt 模板
```
用于【应用场景/表达主题】的海报，采用【风格词/风格详细描述】风格，
以【表达手法】实现了【画面效果】
主标题是【主标题文案】，字体是【字体描述】，位于【画面位置】
画面核心是【画面意象描述】【画面整体描述】
其他文案是【文案内容】，字体是【字体描述】，位于【画面位置】
整体色调以【主色系】为主，搭配【辅助色】，光影为【光影描述】，质感【质感描述】
氛围呈现【氛围词】，整体风格【总结性氛围】
```

## 字体规范
- 避免黑体、无衬线体等过于常见的字体
- 选择具有艺术性或设计感的字体：手写体、行书、英文字体等
- 字体需与整体设计风格匹配

## 默认设置
- 数量：4张不同风格
- 比例：2:3

## 注意事项
- **文字是海报必要元素**，必须设计海报上的文字内容
- 灵活调整模板，非每个部分都必须填写
