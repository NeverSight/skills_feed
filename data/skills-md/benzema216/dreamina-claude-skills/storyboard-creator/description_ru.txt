Инструмент для создания романов/сценариев в раскадровке. Преобразует письменный контент в подробные раскадровки для управления созданием изображений и видео с помощью ИИ. Используйте этот навык, когда пользователю необходимо: (1) превратить главы романа в раскадровки, (2) создать сценарии раскадровки для генерации изображений ИИ, (3) спланировать раскадровки фильмов/телепередач, (4) создать раскадровки с подсказками по созданию изображений.
