# Prompt 撰写规范

## 构图描述要素
- 主体位置与比例
- 光影效果
- 材质质感
- 色彩搭配与视觉焦点
- 排版/图形元素
- 表现技法与创作意图

## 主体描述要素
- 人物：人种、性别、年龄、发型、发色、衣着、配饰
- 动物/角色：种类、颜色、特征、姿态

## 风格增强
- 使用艺术家/摄影师名字作为风格词（如 Alan Fearnley、Greg Rutkowski）
- 生成多张图时可为每张选择不同艺术家以增加多样性

## 发散生成（需求不明确时）
- 为每张图撰写详细完整描述
- 补充场景、动作、表情、主体细节
- 适当加入艺术风格

## 精确生成（用户给出详细Prompt时）
- 严格尊重用户提示词，不做修改
- 直接使用其描述生成
