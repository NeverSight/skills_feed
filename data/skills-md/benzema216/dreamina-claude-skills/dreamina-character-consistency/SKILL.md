# 角色一致性规范

## 方法
1. 先用 `gen_image_verbatim` 生成主角的**白底形象照**（正面）
2. 后续场景使用 `edit_image` 参考形象照生成

## 角色描述要求
- 必须清晰、详细描述特征
- 包括：种族、性别、发型发色、衣着样式与颜色
- 用户只指定部分特征时，需合理补充剩余特征
- **禁止**使用"特征同上"、"角色不变"等模糊表述（工具无法记忆上下文）

## 组图/套图生成
1. 首图独立生成
2. 从第二张开始，用 `edit_image` 以上一张图为参考生成
3. 形成**链式**传递，保证风格与元素连续

## 绘本/分镜生成
- 所有画面使用同一种艺术风格
- 每张图的 description 中**重复描述**该风格
- 每个场景详细描述**环境**（否则可能出现白底）
