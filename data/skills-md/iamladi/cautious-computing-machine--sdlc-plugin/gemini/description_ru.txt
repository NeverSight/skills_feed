Используйте, когда пользователь запрашивает запуск Gemini CLI для проверки кода, проверки плана или обработки большого контекста (>200 тыс.). Идеально подходит для комплексного анализа, требующего больших контекстных окон. По умолчанию использует Gemini 3 Pro для самых современных рассуждений и кодирования.
