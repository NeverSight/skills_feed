انتقل إلى قواعد تعليمات Python من خلال قراءة ملفات __init__.py قبل فتح ملفات أخرى في الحزمة. احتفظ بكل __init__.py كوثائق حية: سلسلة مستندية عالية المستوى تحتوي على وصف حزمة من سطر واحد، وشجرة ملفات لملفات .py والأدلة الفرعية، وغرض مختصر لكل إدخال. يُستخدم عندما: (1) استكشاف قاعدة بيانات Python أو التنقل فيها، (2) إنشاء حزم أو وحدات Python جديدة، (3) إضافة الملفات أو إزالتها أو إعادة تسميتها داخل حزمة Python، (4) الانضمام إلى مشروع Python غير مألوف.
