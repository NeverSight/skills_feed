Navegue por las bases de código de Python leyendo los archivos __init__.py antes de abrir otros archivos en un paquete. Mantenga cada __init__.py como documentación viva: una cadena de documentación de nivel superior que contiene una descripción del paquete de una línea, un árbol de archivos .py y subdirectorios, y un breve propósito para cada entrada. Úselo cuando: (1) explore o navegue por una base de código de Python, (2) cree nuevos paquetes o módulos de Python, (3) agregue, elimine o cambie el nombre de archivos dentro de un paquete de Python, (4) se incorpore a un proyecto de Python desconocido.
