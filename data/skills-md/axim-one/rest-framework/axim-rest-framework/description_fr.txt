Créez des API REST Spring Boot avec Axim REST Framework. À utiliser lors de la création d'entités, de référentiels, de services, de contrôleurs, de gestion des erreurs ou de pagination avec le framework axim-rest-framework (Spring Boot + MyBatis). Couvre @XEntity, @XRepository, IXRepository, la dérivation de requêtes, la sauvegarde/modification/upsert, XPagination, XPage, les codes d'erreur, les exceptions i18n et le client REST déclaratif.
