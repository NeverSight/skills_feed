Analice las configuraciones de malla de servicios de Istio, Consul y Linkerd para detectar vulnerabilidades de seguridad con asignaciones de control NIST 800-53. Úselo cuando los usuarios necesiten auditar la seguridad de la malla, identificar configuraciones incorrectas, verificar la configuración de mTLS, revisar las políticas de ACL o prepararse para las evaluaciones de FedRAMP. Se activa con palabras clave como "configuración de malla", "seguridad de istio", "consul ACL", "política de linkerd", "auditoría de malla de servicio" o "cumplimiento de NIST".
