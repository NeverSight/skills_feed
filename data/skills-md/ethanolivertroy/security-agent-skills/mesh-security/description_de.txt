Analysieren Sie Istio-, Consul- und Linkerd-Service-Mesh-Konfigurationen auf Sicherheitslücken mit NIST 800-53-Kontrollzuordnungen. Verwenden Sie es, wenn Benutzer die Mesh-Sicherheit prüfen, Fehlkonfigurationen identifizieren, mTLS-Einstellungen überprüfen, ACL-Richtlinien überprüfen oder sich auf FedRAMP-Bewertungen vorbereiten müssen. Löst auf Schlüsselwörtern wie „Mesh Config“, „Istio Security“, „Consul ACL“, „Linkerd Policy“, „Service Mesh Audit“ oder „NIST Compliance“ aus.
