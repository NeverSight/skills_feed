قم بتحليل تكوينات شبكة الخدمة Istio وConsul وLinkerd بحثًا عن الثغرات الأمنية باستخدام تعيينات التحكم NIST 800-53. يُستخدم عندما يحتاج المستخدمون إلى تدقيق أمان الشبكة، أو تحديد التكوينات الخاطئة، أو التحقق من إعدادات mTLS، أو مراجعة سياسات ACL، أو الاستعداد لتقييمات FedRAMP. مشغلات على الكلمات الرئيسية مثل "mesh config" أو "istio Security" أو "consul ACL" أو "سياسة الرابط" أو "تدقيق شبكة الخدمة" أو "الامتثال لـ NIST".
