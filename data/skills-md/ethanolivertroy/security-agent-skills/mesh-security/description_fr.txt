Analysez les configurations de maillage de services Istio, Consul et Linkerd pour détecter les vulnérabilités de sécurité avec les mappages de contrôle NIST 800-53. À utiliser lorsque les utilisateurs doivent auditer la sécurité du maillage, identifier les erreurs de configuration, vérifier les paramètres mTLS, examiner les politiques ACL ou se préparer aux évaluations FedRAMP. Se déclenche sur des mots-clés tels que « configuration du maillage », « sécurité istio », « consul ACL », « politique de linkerd », « audit du maillage de service » ou « conformité NIST ».
