Analizza le configurazioni mesh di servizi Istio, Consul e Linkerd per individuare eventuali vulnerabilità della sicurezza con le mappature di controllo NIST 800-53. Da utilizzare quando gli utenti devono verificare la sicurezza della mesh, identificare configurazioni errate, controllare le impostazioni mTLS, rivedere le policy ACL o prepararsi per le valutazioni FedRAMP. Si attiva su parole chiave come "mesh config", "istio security", "consul ACL", "linkerd policy", "service mesh audit" o "conformità NIST".
