Оптимизированная система визуализации JSON Canvas, разработанная специально для графиков знаний в учебных заметках. Предоставляет функции визуализации, такие как интеллектуальные карты, диаграммы взаимосвязей понятий и диаграммы путей обучения, которые помогают создавать интуитивные структуры знаний.
