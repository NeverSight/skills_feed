Eseguire query sui dati Developer Experience (DX) tramite il database PostgreSQL del server DX Data MCP. Utilizza questa competenza quando analizzi le metriche di produttività degli sviluppatori, le prestazioni del team, le metriche di revisione PR/codice, la frequenza di distribuzione, i dati sugli incidenti, l'adozione di strumenti AI, le risposte ai sondaggi, le metriche DORA o qualsiasi analisi tecnica. Attiva domande su punteggi DX, confronti tra team, tempi di ciclo, qualità del codice, sentiment degli sviluppatori, adozione dell'assistente di codifica AI, velocità di sprint o KPI di ingegneria.
