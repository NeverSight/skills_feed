الاستعلام عن بيانات تجربة المطور (DX) عبر قاعدة بيانات PostgreSQL لخادم DX Data MCP. استخدم هذه المهارة عند تحليل مقاييس إنتاجية المطورين، أو أداء الفريق، أو مقاييس مراجعة العلاقات العامة/الكود، أو تكرار النشر، أو بيانات الحوادث، أو اعتماد أدوات الذكاء الاصطناعي، أو استجابات الاستطلاع، أو مقاييس DORA، أو أي تحليلات هندسية. يتم تشغيل الأسئلة حول نتائج DX، أو مقارنات الفريق، أو أوقات الدورات، أو جودة التعليمات البرمجية، أو مشاعر المطور، أو اعتماد مساعد ترميز الذكاء الاصطناعي، أو سرعة الركض، أو مؤشرات الأداء الرئيسية الهندسية.
