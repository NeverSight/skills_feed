Комплексный набор инструментов для работы с PDF-файлами для извлечения текста и таблиц, создания новых PDF-файлов, объединения/разделения документов, обработки форм и оптического распознавания символов в отсканированных документах. Используйте при заполнении форм PDF, извлечении данных из PDF-файлов, объединении или разделении документов, программном создании PDF-файлов, масштабной обработке документов или выполнении оптического распознавания PDF-файлов на основе изображений.
