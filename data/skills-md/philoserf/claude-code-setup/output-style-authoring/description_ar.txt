دليل لتأليف أنماط الإخراج التي تغير سلوك كلود وشخصيته. يُستخدم عند الإنشاء أو الكتابة أو التصميم أو البناء أو المراجعة أو تحسين أنماط الإخراج أو أوضاع الشخصية أو تعديلات السلوك. يساعد في ملفات الأنماط، وقرارات تعليمات الاحتفاظ بالترميز، وتصميم الشخصية، والاختيار بين أنماط الإخراج، والوكلاء، والمهارات. خبير في أفضل ممارسات الأسلوب.
