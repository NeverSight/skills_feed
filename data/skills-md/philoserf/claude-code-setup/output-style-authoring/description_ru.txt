Руководство по созданию стилей вывода, которые меняют поведение и личность Клода. Используйте при создании, написании, проектировании, построении, проверке или улучшении стилей вывода, режимов личности или модификаций поведения. Помогает с файлами стилей, решениями по сохранению инструкций по кодированию, дизайном персонажей и выбором между стилями вывода, агентами и навыками. Эксперт в области лучших практик стиля.
