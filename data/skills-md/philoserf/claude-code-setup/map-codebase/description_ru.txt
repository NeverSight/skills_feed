Анализируйте кодовую базу с помощью параллельных агентов Explore для создания документов .planning/codebase/. Используйте для адаптации существующих проектов, обновления понимания кодовой базы после значительных изменений, перед крупным рефакторингом или при адаптации к незнакомым кодовым базам. Создает структурированную документацию по стеку, архитектуре, структуре, соглашениям, тестированию, интеграции и проблемам.
