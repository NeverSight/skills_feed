قم بتحليل قاعدة التعليمات البرمجية باستخدام وكلاء Explore المتوازيين لإنتاج مستندات .planning/codebase/. يُستخدم في الإعداد لمشروع Brownfield، أو تحديث فهم قاعدة التعليمات البرمجية بعد إجراء تغييرات كبيرة، أو قبل إعادة البناء الرئيسية، أو عند الإعداد إلى قواعد تعليمات برمجية غير مألوفة. ينشئ وثائق منظمة للمكدس والهندسة المعمارية والبنية والاتفاقيات والاختبار والتكامل والاهتمامات.
