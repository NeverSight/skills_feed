Эксперт в области uv, сверхбыстрого менеджера пакетов Python и инструмента проектов. Используйте при настройке проектов Python, управлении зависимостями, создании виртуальных сред, установке версий Python, работе с файлами блокировки, переходе с pip/poetry/pip-tools или оптимизации рабочих процессов Python с невероятно высокой производительностью uv.
