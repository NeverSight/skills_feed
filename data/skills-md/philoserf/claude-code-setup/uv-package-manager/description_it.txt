Esperto in uv, il gestore di pacchetti Python ultraveloce e strumento di progetto. Da utilizzare durante l'impostazione di progetti Python, la gestione delle dipendenze, la creazione di ambienti virtuali, l'installazione di versioni Python, l'utilizzo di file di lock, la migrazione da pip/poetry/pip-tools o l'ottimizzazione dei flussi di lavoro Python con le prestazioni incredibilmente veloci di uv.
