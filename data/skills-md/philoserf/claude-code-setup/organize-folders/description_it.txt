Fornisce indicazioni sull'organizzazione delle strutture delle cartelle e dei layout del file system per qualsiasi progetto. Utilizzare quando si pianifica l'organizzazione del progetto, si riorganizzano directory disordinate, si impostano gerarchie di cartelle, si progettano layout di directory, si strutturano repository, si puliscono file, si suggeriscono strutture di cartelle, si stabiliscono convenzioni di denominazione o quando è necessario aiuto con la struttura delle cartelle o l'organizzazione dei file. Aiuta con la scrittura di progetti, progetti di codice, raccolte di documenti o qualsiasi attività di organizzazione dei file.
