Содержит рекомендации по организации структур папок и макетов файловой системы для любого проекта. Используйте при планировании организации проекта, реорганизации беспорядочных каталогов, настройке иерархии папок, разработке макетов каталогов, структурировании репозиториев, очистке файлов, предложении структуры папок, установлении соглашений об именах или когда вам нужна помощь со структурой папок или организацией файлов. Помогает при написании проектов, проектов кода, коллекций документов или любых задач по организации файлов.
