قم بإنشاء تطبيقات Flutter باستخدام Clean Architecture مع بنية الميزة الأولى، وإدارة حالة Riverpod، وDio + Retrofit للتواصل، ومعالجة أخطاء fpdart. استخدم هذه المهارة عند إنشاء تطبيقات Flutter، أو تنفيذ الميزات بأنماط بنية نظيفة، أو إعداد موفري Riverpod، أو التعامل مع البيانات بأي من النوعين لمعالجة الأخطاء الوظيفية، أو تقديم طلبات HTTP مع عملاء واجهة برمجة التطبيقات الآمنة من النوع، أو هيكلة المشاريع باستخدام طبقات المجال/البيانات/العرض التقديمي. تتضمن المشغلات "تطبيق Flutter"، أو "البنية النظيفة"، أو "Riverpod"، أو "الميزة أولاً"، أو "إدارة الحالة"، أو "عميل API"، أو "Retrofit"، أو "Dio"، أو "REST API"، أو طلبات إنشاء ميزات Flutter مع فصل الاهتمامات.
