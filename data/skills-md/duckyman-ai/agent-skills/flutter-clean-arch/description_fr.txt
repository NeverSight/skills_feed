Utilisez cette compétence lors de la création d'applications Flutter, de la mise en œuvre de fonctionnalités avec des modèles d'architecture propres, de la configuration de fournisseurs Riverpod, de la gestion des données avec l'un ou l'autre type pour la gestion fonctionnelle des erreurs, de l'exécution de requêtes HTTP avec des clients API de type sécurisé ou de la structuration de projets avec des couches de domaine/données/présentation. Les déclencheurs incluent « Application Flutter », « Architecture propre », « Riverpod », « Fonctionnalité d'abord », « Gestion de l'état », « Client API », « Retrofit », « Dio », « API REST » ou des demandes de création de fonctionnalités Flutter avec séparation des préoccupations.
