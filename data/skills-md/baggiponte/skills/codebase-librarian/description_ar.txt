إنشاء جرد شامل لقاعدة التعليمات البرمجية. هيكل الخريطة ونقاط الدخول والخدمات والبنية التحتية ونماذج المجال وتدفقات البيانات. توثيق خالص - لا توجد آراء أو توصيات. يُستخدم عند الانضمام إلى قاعدة تعليمات برمجية غير مألوفة، أو توثيق البنية الحالية قبل التغييرات، أو التحضير لمراجعات البنية أو التخطيط للترحيل، أو إنشاء مرجع للفريق. مشغلات لطلبات مثل "تعيين قاعدة التعليمات البرمجية هذه" أو "توثيق البنية" أو "إنشاء مخزون" أو "ماذا تحتوي قاعدة التعليمات البرمجية هذه".
