Refactorisation systématique des composants de la base de code via un processus structuré en 3 phases. À utiliser lorsqu'on vous demande de refactoriser, de restructurer ou d'améliorer des composants, des modules ou des zones de code spécifiques. Produit de la documentation de recherche, des propositions de modification avec des exemples de code et des plans de test. Déclencheurs sur des requêtes telles que « refactoriser le module d'authentification », « restructurer la couche de données », « améliorer les gestionnaires d'API » ou « nettoyer le service de paiement ».
