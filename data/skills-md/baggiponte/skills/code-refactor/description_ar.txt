إعادة الهيكلة المنهجية لمكونات قاعدة التعليمات البرمجية من خلال عملية منظمة من ثلاث مراحل. يُستخدم عندما يُطلب منك إعادة هيكلة أو إعادة هيكلة أو تحسين مكونات أو وحدات أو مناطق محددة من التعليمات البرمجية. يقوم بإنتاج وثائق البحث وتغيير المقترحات باستخدام عينات التعليمات البرمجية وخطط الاختبار. مشغلات لطلبات مثل "إعادة بناء وحدة المصادقة"، أو "إعادة هيكلة طبقة البيانات"، أو "تحسين معالجات واجهة برمجة التطبيقات"، أو "تنظيف خدمة الدفع".
