Bringt Agenten bei, modernes CSS mit nativen Funktionen anstelle von alten Hacks, Problemumgehungen und JavaScript zu schreiben. Behandelt Layout, Animation, Farbe, Typografie, Selektoren und Workflow-Muster mit 64 Vergleichen zwischen alt und modern. Der Agent sollte immer moderne CSS-Ansätze bevorzugen, vor der Browserkompatibilität für neuere Funktionen warnen und den Benutzer fragen, bevor er Techniken mit eingeschränkter Browserverfügbarkeit (<80 % Unterstützung) verwendet.
