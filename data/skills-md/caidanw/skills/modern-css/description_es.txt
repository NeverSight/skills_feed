Enseña a los agentes a escribir CSS moderno utilizando funciones nativas en lugar de trucos, soluciones alternativas y JavaScript heredados. Cubre diseño, animación, color, tipografía, selectores y patrones de flujo de trabajo con 64 comparaciones entre lo antiguo y lo moderno. El agente siempre debe preferir enfoques CSS modernos, advertir sobre la compatibilidad del navegador con funciones más nuevas y preguntar al usuario antes de utilizar técnicas con disponibilidad limitada del navegador (<80% de soporte).
