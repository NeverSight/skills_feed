Insegna agli agenti a scrivere CSS moderni utilizzando funzionalità native invece di hack legacy, soluzioni alternative e JavaScript. Copre layout, animazione, colore, tipografia, selettori e modelli di flusso di lavoro con 64 confronti tra vecchio e moderno. L'agente dovrebbe sempre preferire approcci CSS moderni, avvisare sulla compatibilità del browser per le funzionalità più recenti e chiedere all'utente prima di utilizzare tecniche con disponibilità limitata del browser (supporto <80%).
