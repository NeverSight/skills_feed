يعلم الوكلاء كيفية كتابة CSS الحديثة باستخدام الميزات الأصلية بدلاً من الاختراقات القديمة والحلول البديلة وجافا سكريبت. يغطي التخطيط والرسوم المتحركة واللون والطباعة والمحددات وأنماط سير العمل مع 64 مقارنة بين القديم والحديث. يجب أن يفضل الوكيل دائمًا أساليب CSS الحديثة، ويحذر بشأن توافق المتصفح مع الميزات الأحدث، ويسأل المستخدم قبل استخدام التقنيات ذات توفر المستعرض المحدود (دعم <80٪).
