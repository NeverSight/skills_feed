Verwenden Sie die lokale Kostennutzung der CodexBar CLI, um die Nutzung pro Modell für Codex oder Claude zusammenzufassen, einschließlich des aktuellen (aktuellsten) Modells oder einer vollständigen Modellaufschlüsselung. Wird ausgelöst, wenn Sie von Codexbar nach Nutzungs-/Kostendaten auf Modellebene gefragt werden oder wenn Sie eine skriptfähige Zusammenfassung pro Modell aus Codexbar Cost JSON benötigen.
