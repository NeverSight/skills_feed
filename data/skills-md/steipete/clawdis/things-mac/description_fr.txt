Gérez Things 3 via la CLI `things` sur macOS (ajout/mise à jour de projets + tâches via un schéma d'URL ; lecture/recherche/liste à partir de la base de données Things locale). À utiliser lorsqu'un utilisateur demande à Moltbot d'ajouter une tâche à Things, de lister la boîte de réception/aujourd'hui/à venir, de rechercher des tâches ou d'inspecter des projets/zones/tags.
