Analyse des requêtes d'extraction GitHub en révision uniquement avec la CLI gh. À utiliser lorsqu'on vous demande d'examiner un PR, de fournir des commentaires structurés ou d'évaluer l'état de préparation à l'atterrissage. Ne fusionnez pas, ne poussez pas et n'apportez pas de modifications de code que vous avez l'intention de conserver.
