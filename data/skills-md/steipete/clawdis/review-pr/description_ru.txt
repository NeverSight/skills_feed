Анализ запросов на извлечение GitHub только для просмотра с помощью gh CLI. Используйте, когда вас просят просмотреть PR, предоставить структурированную обратную связь или оценить готовность к приземлению. Не объединяйте, не отправляйте и не вносите изменения в код, которые вы собираетесь сохранить.
