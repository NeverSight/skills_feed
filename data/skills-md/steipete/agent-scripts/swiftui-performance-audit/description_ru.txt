Аудит и улучшение производительности среды выполнения SwiftUI путем проверки кода и архитектуры. Используйте для запросов для диагностики медленного рендеринга, прерывистой прокрутки, высокой загрузки ЦП/памяти, чрезмерных обновлений представления или проблем с макетом в приложениях SwiftUI, а также для предоставления рекомендаций по профилированию инструментов, запускаемых пользователем, когда одной только проверки кода недостаточно.
