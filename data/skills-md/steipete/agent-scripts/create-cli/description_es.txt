Diseñe parámetros de interfaz de línea de comandos y UX: argumentos, indicadores, subcomandos, texto de ayuda, formatos de salida, mensajes de error, códigos de salida, indicaciones, precedencia de configuración/env y comportamiento seguro/ejecución en seco. Úselo cuando esté diseñando una especificación de CLI (antes de la implementación) o refactorizando el área de superficie de una CLI existente para lograr coherencia, componibilidad y capacidad de descubrimiento.
