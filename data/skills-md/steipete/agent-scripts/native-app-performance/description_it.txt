Profilazione nativa delle prestazioni delle app macOS/iOS tramite xctrace/Time Profiler e analisi solo CLI delle tracce degli strumenti. Utilizzare quando viene richiesto di profilare, allegare, registrare o analizzare file .trace di Instruments, trovare hotspot o ottimizzare le prestazioni dell'app nativa senza aprire l'interfaccia utente di Instruments.
