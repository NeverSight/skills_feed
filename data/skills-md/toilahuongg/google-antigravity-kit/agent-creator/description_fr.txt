Guide pour créer, configurer et affiner les agents AI. Utilisez cette compétence lorsque les utilisateurs souhaitent définir un nouveau personnage d'agent, générer une invite système ou assembler un ensemble spécifique de compétences/flux de travail pour un agent spécialisé (par exemple, « Créer un agent d'assurance qualité » ou « Concevoir un agent d'auditeur de sécurité »).
