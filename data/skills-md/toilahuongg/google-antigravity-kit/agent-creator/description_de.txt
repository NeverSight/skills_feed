Leitfaden zum Erstellen, Konfigurieren und Verfeinern von KI-Agenten. Verwenden Sie diese Fähigkeit, wenn Benutzer eine neue Agentenpersönlichkeit definieren, eine Systemeingabeaufforderung generieren oder einen bestimmten Satz von Fähigkeiten/Workflows für einen spezialisierten Agenten zusammenstellen möchten (z. B. „Erstellen eines QA-Agenten“ oder „Entwerfen eines Sicherheitsprüfer-Agenten“).
