Jujutsu (jj) — le système de contrôle de version compatible Git. Activez UNIQUEMENT lorsqu'un répertoire .jj/ est présent dans le projet ou lorsque jj/jujutsu est explicitement mentionné. Ne pas activer pour les dépôts git simples sans .jj/. À utiliser pour toutes les opérations VCS dans les projets gérés par jj : commit, push, pull, branch, bookmark, rebase, squash, merge, diff, log, status, working copy, change ID, revset, fileset, template, configuration, workspaces.
