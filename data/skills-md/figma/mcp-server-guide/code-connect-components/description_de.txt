Verbindet Figma-Designkomponenten mithilfe von Code Connect mit Codekomponenten. Wird verwendet, wenn der Benutzer „Code verbinden“, „Diese Komponente mit Code verbinden“, „Figma mit Code verbinden“, „Diese Komponente zuordnen“, „Komponente mit Code verknüpfen“, „Code-Verbindungszuordnung erstellen“, „Code-Verbindung hinzufügen“, „Design mit Code verbinden“ sagt oder Zuordnungen zwischen Figma-Designs und Code-Implementierungen herstellen möchte. Erfordert Figma MCP-Serververbindung.
