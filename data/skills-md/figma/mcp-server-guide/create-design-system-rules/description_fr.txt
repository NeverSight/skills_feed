Génère des règles de système de conception personnalisées pour la base de code de l'utilisateur. À utiliser lorsque l'utilisateur dit « créer des règles du système de conception », « générer des règles pour mon projet », « configurer des règles de conception », « personnaliser les directives du système de conception » ou souhaite établir des conventions spécifiques au projet pour les flux de travail Figma-to-code. Nécessite une connexion au serveur Figma MCP.
