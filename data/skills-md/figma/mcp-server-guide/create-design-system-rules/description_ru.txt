Создает пользовательские правила системы проектирования для базы кода пользователя. Используйте, когда пользователь говорит «создать правила системы дизайна», «создать правила для моего проекта», «настроить правила дизайна», «настроить рекомендации системы дизайна» или хочет установить специфичные для проекта соглашения для рабочих процессов преобразования Figma в код. Требуется подключение к серверу Figma MCP.
