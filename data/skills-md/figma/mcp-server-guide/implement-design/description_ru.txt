Переводит проекты Figma в готовый к использованию код с визуальной точностью 1:1. Используйте при реализации пользовательского интерфейса из файлов Figma, когда пользователь упоминает «реализовать дизайн», «сгенерировать код», «реализовать компонент», «создать дизайн Figma», предоставляет URL-адреса Figma или просит создать компоненты, соответствующие спецификациям Figma. Требуется подключение к серверу Figma MCP.
