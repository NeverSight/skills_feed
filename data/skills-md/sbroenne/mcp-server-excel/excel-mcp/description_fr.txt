Automatisez Microsoft Excel sur Windows via COM Interop. À utiliser lors de la création, de la lecture ou de la modification de classeurs Excel. Prend en charge Power Query (code M), le modèle de données (mesures DAX), les tableaux croisés dynamiques, les tableaux, les plages, les graphiques, le formatage, les macros VBA et les connexions. Déclencheurs : Excel, feuille de calcul, classeur, xlsx, Power Query, DAX, PivotTable, VBA.
