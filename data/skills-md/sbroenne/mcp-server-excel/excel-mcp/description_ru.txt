Автоматизируйте Microsoft Excel в Windows через COM-взаимодействие. Используйте при создании, чтении или изменении книг Excel. Поддерживает Power Query (код M), модель данных (меры DAX), сводные таблицы, таблицы, диапазоны, диаграммы, форматирование, макросы VBA и подключения. Триггеры: Excel, электронная таблица, книга, xlsx, Power Query, DAX, сводная таблица, VBA.
