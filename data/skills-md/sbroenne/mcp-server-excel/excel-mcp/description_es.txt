Automatice Microsoft Excel en Windows mediante interoperabilidad COM. Úselo al crear, leer o modificar libros de Excel. Admite Power Query (código M), modelo de datos (medidas DAX), tablas dinámicas, tablas, rangos, gráficos, formato, macros VBA y conexiones. Activadores: Excel, hoja de cálculo, libro de trabajo, xlsx, Power Query, DAX, tabla dinámica, VBA.
