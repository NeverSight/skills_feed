Automatisieren Sie Microsoft Excel unter Windows über COM-Interop. Zum Erstellen, Lesen oder Ändern von Excel-Arbeitsmappen verwenden. Unterstützt Power Query (M-Code), Datenmodell (DAX-Maßnahmen), PivotTables, Tabellen, Bereiche, Diagramme, Formatierung, VBA-Makros und Verbindungen. Auslöser: Excel, Tabellenkalkulation, Arbeitsmappe, XLSX, Power Query, DAX, PivotTable, VBA.
