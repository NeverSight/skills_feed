أتمتة Microsoft Excel على نظام التشغيل Windows عبر إمكانية التشغيل المتداخل لـ COM. يُستخدم عند إنشاء مصنفات Excel أو قراءتها أو تعديلها. يدعم Power Query (رمز M)، ونموذج البيانات (مقاييس DAX)، وPivotTables، والجداول، والنطاقات، والمخططات، والتنسيق، ووحدات ماكرو VBA، والاتصالات. المشغلات: Excel، وجداول البيانات، والمصنفات، وxlsx، وPower Query، وDAX، وPivotTable، وVBA.
