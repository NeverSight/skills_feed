Automatisieren Sie Microsoft Excel unter Windows über CLI. Zum Erstellen, Lesen oder Ändern von Excel-Arbeitsmappen aus Skripten, CI/CD oder Codierungsagenten. Unterstützt Power Query, DAX, PivotTables, Tabellen, Bereiche, Diagramme, VBA. Auslöser: Excel, Tabellenkalkulation, Arbeitsmappe, XLSX, Excelcli, CLI-Automatisierung.
