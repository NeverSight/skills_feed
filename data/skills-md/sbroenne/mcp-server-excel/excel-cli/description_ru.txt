Автоматизируйте Microsoft Excel в Windows через интерфейс командной строки. Используйте при создании, чтении или изменении книг Excel с помощью сценариев, CI/CD или агентов кодирования. Поддерживает Power Query, DAX, сводные таблицы, таблицы, диапазоны, диаграммы, VBA. Триггеры: Excel, электронная таблица, рабочая книга, xlsx, excelcli, автоматизация CLI.
