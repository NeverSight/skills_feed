Просматривает документацию по архитектуре C3 и изучает соответствующий код, чтобы ответить на вопросы по архитектуре.
Используйте, когда пользователь спрашивает «где находится X», «как работает X», «объясните X», «покажите мне архитектуру»,
«найти компонент», «что обрабатывает X», «какие ограничения применяются к X» или ссылается на идентификаторы C3 (c3-0, c3-1, adr-*).
Для существования требуется .c3/. Для внесения изменений вместо этого перейдите к c3-alter.
