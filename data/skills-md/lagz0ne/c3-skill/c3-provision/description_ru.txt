Архитектура документов БЕЗ реализации. Создает документы ADR и компонентов со статусом: подготовлено.

ДОЛЖЕН использовать этот навык, когда пользователь хочет ПРОЕКТИРОВАТЬ или ПЛАНИРОВАТЬ, не реализуя:
- «положение X», «предоставление компонента», «предоставление новой услуги»
- «спроектировать архитектуру для X», «спроектировать компонент».
- «план Х», «планировать новую услугу», «планировать архитектуру»
- «архитектура документа X», «архитектор X»
- «представить X», «набросать архитектуру X»

НЕ используйте этот навык, если подсказка содержит:
- "шаблон", "узоры" → путь к c3-ref
- "ref", "refs" → маршрут к c3-ref
- "конвенция", "стандарт" → путь к c3-ref

Ключевой индикатор: пользователю нужна документация/проект, а НЕ изменения кода.
Для существования требуется .c3/. Создает ADR со статусом: подготовлено.
Для запросов на реализацию направляйтесь к c3-alter.
