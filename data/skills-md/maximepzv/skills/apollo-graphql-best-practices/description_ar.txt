أفضل الممارسات لتطوير Apollo GraphQL بما في ذلك Apollo Client (خطافات React، والتخزين المؤقت، ومعالجة الأخطاء) وApollo Server (تصميم المخطط، والمحللات، والسياق، والمكونات الإضافية). يُستخدم عند كتابة كود GraphQL أو مراجعته أو إعادة هيكلته باستخدام Apollo، أو إعداد عميل/خادم Apollo، أو تنفيذ الاستعلامات/الطفرات، أو تكوين سياسات ذاكرة التخزين المؤقت، أو التعامل مع أخطاء GraphQL.
