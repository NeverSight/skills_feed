Mejores prácticas para el desarrollo de Apollo GraphQL, incluido Apollo Client (enganches de React, almacenamiento en caché, manejo de errores) y Apollo Server (diseño de esquemas, solucionadores, contexto, complementos). Úselo al escribir, revisar o refactorizar código GraphQL con Apollo, configurar el cliente/servidor Apollo, implementar consultas/mutaciones, configurar políticas de caché o manejar errores de GraphQL.
