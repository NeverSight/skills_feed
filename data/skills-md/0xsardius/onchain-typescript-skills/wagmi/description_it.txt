React hook per interazioni blockchain Ethereum ed EVM utilizzando Wagmi v3. Da utilizzare durante la creazione di app React o Next.js con connessioni wallet, letture/scritture di contratti o stato blockchain. Si attiva su useAccount, useConnect, useContractRead, useContractWrite, WagmiProvider, ConnectKit, RainbowKit o qualsiasi hook blockchain di React. NON utilizzare per script Node o codice non React (usa invece la skill viem).
