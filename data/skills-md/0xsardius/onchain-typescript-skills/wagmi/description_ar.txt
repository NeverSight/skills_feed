خطافات تفاعلية لتفاعلات blockchain الخاصة بـ Ethereum وEVM باستخدام Wagmi v3. يُستخدم عند إنشاء تطبيقات React أو Next.js باستخدام اتصالات المحفظة، أو عمليات قراءة/كتابة العقد، أو حالة blockchain. المشغلات على useAccount، أو useConnect، أو useContractRead، أو useContractWrite، أو WagmiProvider، أو ConnectKit، أو RainbowKit، أو أي خطافات React blockchain. لا تستخدمه مع نصوص Node أو التعليمات البرمجية غير React (استخدم مهارة viem بدلاً من ذلك).
