Ganchos de reacción para interacciones de blockchain Ethereum y EVM utilizando Wagmi v3. Úselo al crear aplicaciones React o Next.js con conexiones de billetera, lecturas/escrituras de contratos o estado de blockchain. Se activan useAccount, useConnect, useContractRead, useContractWrite, WagmiProvider, ConnectKit, RainbowKit o cualquier enlace de blockchain de React. NO lo use para scripts de Node o código que no sea de React (use la habilidad viem en su lugar).
