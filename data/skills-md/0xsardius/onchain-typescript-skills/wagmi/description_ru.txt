Перехватчики React для взаимодействия блокчейнов Ethereum и EVM с использованием Wagmi v3. Используйте при создании приложений React или Next.js с подключениями к кошельку, чтением/записью контрактов или состоянием блокчейна. Запускается по useAccount, useConnect, useContractRead, useContractWrite, WagmiProvider, ConnectKit, RainbowKit или любым перехватчикам блокчейна React. НЕ используйте для сценариев Node или код, отличный от React (вместо этого используйте навык viem).
