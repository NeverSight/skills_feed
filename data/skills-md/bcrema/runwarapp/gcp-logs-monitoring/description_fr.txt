À utiliser lorsque l'utilisateur a besoin d'inspecter les journaux, les métriques et les signaux de surveillance de Google Cloud (GCP) via gcloud pour le tri des incidents, le débogage ou l'analyse opérationnelle. Prend en charge les requêtes Cloud Logging, les lectures de séries temporelles Cloud Monitoring et les vérifications d'environnement pour un projet cible.
