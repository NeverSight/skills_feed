Générez de la documentation pour votre base de code : des documents de référence pour un fichier, des guides de fonctionnalités ou un site de documentation complet.
À utiliser lorsqu'on vous demande de "documenter ceci", "générer des documents", "écrire de la documentation", "créer une référence API",
ou lorsque vous avez besoin d'une documentation complète pour un module, une fonctionnalité ou un projet entier.
Compatible avec Framework : détecte Fumadocs, Nextra, Docusaurus, etc. et génère au bon format.
