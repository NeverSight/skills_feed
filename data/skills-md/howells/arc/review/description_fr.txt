À utiliser lorsqu'on vous demande de « réviser le plan », « obtenir des commentaires sur la conception », « vérifier cette approche »,
ou avant la mise en œuvre pour valider les décisions architecturales.

Argument facultatif : nom du réviseur (par exemple, `/arc:review daniel-product-engineer` pour utiliser un réviseur spécifique)
