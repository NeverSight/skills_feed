Génère des animations Lottie professionnelles à partir de SVG statiques. Remplace After Effects pour les graphiques animés.
À utiliser lorsque l'utilisateur demande : animer un logo, créer un lottie, une animation svg, des graphiques animés, une animation agitée,
effet de rebond, animation de rotation, effet d'impulsion, animation d'entrée, animation de chargement, animation en boucle,
animation d'icônes, animation de personnages, morphing, dessin de chemin, chemin de découpage, animation de marche, cycle d'exécution,
cycle de marche, animation image par image, animation de sprite.
Prend en charge les courbes de Bézier avancées, les modificateurs de forme, la parentalité, les caches, le morphing, les configurations de personnages,
et des techniques d'animation professionnelles image par image.
