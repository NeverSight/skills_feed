أنشئ مساعدين للذكاء الاصطناعي وروبوتات الدردشة وواجهات مستخدم وكيلة في React وNext.js باستخدام CopilotKit. استخدم هذه المهارة عندما يريد المستخدم إضافة مساعد الذكاء الاصطناعي، أو مساعد الطيار، أو واجهة الدردشة، أو منطقة نصية مدعومة بالذكاء الاصطناعي، أو واجهة مستخدم وكيلة إلى تطبيقه. يغطي الإعداد، والخطافات (useCopilotAction، وuseCopilotReadable، وuseCoAgent، وuseAgent)، ومكونات الدردشة (CopilotPopup، وCopilotSidebar، وCopilotChat)، وواجهة المستخدم التوليدية، والإنسان في الحلقة، والوكلاء المساعدون مع LangGraph، وبروتوكول AG-UI، وتطبيقات MCP، وتكامل Python SDK. المشغلات في CopilotKit، وcopilotkit، وuseCopilotAction، وuseCopilotReadable، وuseCoAgent، وuseAgent، وCopilotRuntime، وCopilotChat، وCopilotSidebar، وCopilotPopup، وCopilotTextarea، وAG-UI، والواجهة الأمامية الوكيلة، ومساعد الذكاء الاصطناعي داخل التطبيق، ومساعد الذكاء الاصطناعي React، وchatbot React، وuseFrontendTool، وuseRenderToolCall، useDefaultTool، useCoAgentStateRender، useLangGraphInterrupt، useCopilotChat، useCopilotAdditionalInstructions، useCopilotChatSuggestions، useHumanInTheLoop، CopilotTask، وقت تشغيل مساعد الطيار، LangGraphAgent، BasicAgent،builtInAgent، CopilotKitRemoteEndpoint، A2UI، MCP Apps، منطقة نصية AI، إكمال نموذج AI، إضافة تطبيق الذكاء الاصطناعي للرد.
