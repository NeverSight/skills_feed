Créez des copilotes d'IA, des chatbots et des interfaces utilisateur agents dans React et Next.js à l'aide de CopilotKit. Utilisez cette compétence lorsque l'utilisateur souhaite ajouter un assistant IA, un copilote, une interface de discussion, une zone de texte alimentée par l'IA ou une interface utilisateur agent à son application. Couvre la configuration, les hooks (useCopilotAction, useCopilotReadable, useCoAgent, useAgent), les composants de discussion (CopilotPopup, CopilotSidebar, CopilotChat), l'interface utilisateur générative, l'humain dans la boucle, les CoAgents avec LangGraph, le protocole AG-UI, les applications MCP et l'intégration du SDK Python.
