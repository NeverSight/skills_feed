Создавайте ИИ-пилотов, чат-ботов и агентские пользовательские интерфейсы в React и Next.js с помощью CopilotKit. Используйте этот навык, когда пользователь хочет добавить в свое приложение помощника искусственного интеллекта, второго пилота, интерфейс чата, текстовую область на основе искусственного интеллекта или агентный пользовательский интерфейс. Охватывает настройку, перехватчики (useCopilotAction, useCopilotReadable, useCoAgent, useAgent), компоненты чата (CopilotPopup, CopilotSidebar, CopilotChat), генеративный пользовательский интерфейс, управление человеком в цикле, CoAgents с LangGraph, протокол AG-UI, приложения MCP и интеграцию Python SDK. Триггеры на CopilotKit, copilotkit, useCopilotAction, useCopilotReadable, useCoAgent, useAgent, CopilotRuntime, CopilotChat, CopilotSidebar, CopilotPopup, CopilotTextarea, AG-UI, агентный интерфейс, AI-второй пилот в приложении, AI Assistant React, чат-бот React, useFrontendTool, useRenderToolCall, useDefaultTool, useCoAgentStateRender, useLangGraphInterrupt, useCopilotChat, useCopilotAdditionalInstructions, useCopilotChatSuggestions, useHumanInTheLoop, CopilotTask, среда выполнения второго пилота, LangGraphAgent, BasicAgent, ВстроенныйАгент, CopilotKitRemoteEndpoint, A2UI, MCP Приложения, текстовая область AI, заполнение форм AI, добавление AI в приложение React.
