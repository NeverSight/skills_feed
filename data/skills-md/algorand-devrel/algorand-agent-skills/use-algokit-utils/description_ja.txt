TypeScript または Python アプリケーションから Algorand ブロックチェーンと対話するための AlgoKit Utils ライブラリ。 Algorand ネットワーク (LocalNet、TestNet、MainNet) への接続、支払いの送金や資産の転送、アカウントの作成と管理、クライアント コードからのスマート コントラクトの展開や対話、またはトランザクション グループの作成時に使用します。スマート コントラクト コードを記述するためのものではありません (build-smart-contracts スキルを使用してください)。強力なトリガーには、「Algorand に接続するにはどうすればよいですか?」、「支払いトランザクションを送信する」、「アカウントを作成する」、「契約を展開する」、「AlgorandClient を取得する」、「AlgorandClient.fromEnvironment」などがあります。
