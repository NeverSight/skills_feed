Bibliothèque AlgoKit Utils pour interagir avec la blockchain Algorand à partir d'applications TypeScript ou Python. À utiliser lors de la connexion aux réseaux Algorand (LocalNet, TestNet, MainNet), pour l'envoi de paiements ou le transfert d'actifs, la création et la gestion de comptes, le déploiement ou l'interaction avec des contrats intelligents à partir du code client ou la composition de groupes de transactions. PAS pour écrire du code de contrat intelligent (utilisez la compétence build-smart-contracts). Les déclencheurs puissants incluent « Comment puis-je me connecter à Algorand ? », « envoyer une transaction de paiement », « créer un compte », « déployer mon contrat », « obtenir un AlgorandClient », « AlgorandClient.fromEnvironment ».
