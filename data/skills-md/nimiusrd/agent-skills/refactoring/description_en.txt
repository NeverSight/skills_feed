Skill to assist with code refactoring. Improves the internal structure of code without changing its external behavior (functionality). Used for the following requests:  
- Splitting and organizing functions/classes  
- Renaming variables and function names  
- Eliminating and consolidating duplicate code  
- Applying design patterns  
- Improving code readability and maintainability  
- Instructions like “refactor,” “clean up,” “organize,” or “make it DRY”  

Supported languages: language-agnostic.  
If tests are insufficient, refactoring will be halted.
