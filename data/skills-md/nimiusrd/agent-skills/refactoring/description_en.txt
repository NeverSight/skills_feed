A skill to assist with code refactoring. Improves the internal structure of code without changing its external behavior (functionality). Used for the following requests:  
- Splitting and organizing functions/classes  
- Renaming variables and functions  
- Removing duplicate code and making it reusable  
- Applying design patterns  
- Improving code readability and maintainability  
- Instructions like "refactor this," "clean it up," "organize it," or "make it DRY"  
Supported languages: Universal, regardless of language. If tests are insufficient, refactoring will be halted.
