Progetta e genera test basati sulle proprietà (PBT) per i file modificati nel ramo git corrente. Estrae specifiche, progetta proprietà (invarianti, andata e ritorno, idempotenza, metamorfico, monotonicità, modello di riferimento), crea strategie di generazione, implementa test e auto-punteggi rispetto a una rubrica (24/30+ richiesti). Supporta il controllo rapido (TS/JS), l'ipotesi (Python) e il proptest (Rust). Utilizzare quando: (1) "scrivere test di proprietà per le mie modifiche", (2) "aggiungere PBT", (3) "test basato su proprietà", (4) dopo aver implementato funzioni pure, validatori, parser o formattatori per verificare gli invarianti.
