Übernehmen Sie die Änderungen und erstellen Sie eine GitHub-Pull-Anfrage. Wird für Anfragen wie „Festschreiben und eine PR öffnen“, „Meine aktuelle Arbeit festschreiben und eine Pull-Anfrage erstellen“ oder „Änderungen übertragen und eine PR öffnen“ verwendet. Behandelt das Staging, das Generieren einer Commit-Nachricht, das Erstellen einer Verzweigung, das Pushen und Öffnen des PR als einen einzigen Workflow. Die Commit-Nachricht und der PR-Text sind auf Japanisch verfasst.
