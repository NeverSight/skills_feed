Confirme los cambios y cree una solicitud de extracción de GitHub. Se utiliza para solicitudes como "confirmar y abrir un PR", "confirmar mi trabajo actual y crear una solicitud de extracción" o "impulsar los cambios y abrir un PR". Maneja la preparación, la generación de un mensaje de confirmación, la creación de una rama, el envío y la apertura del PR como un único flujo de trabajo. El mensaje de confirmación y el cuerpo de relaciones públicas están escritos en japonés.
