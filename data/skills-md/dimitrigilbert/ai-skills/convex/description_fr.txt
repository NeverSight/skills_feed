Assistant de développement expert Convex. À utiliser lorsque les utilisateurs souhaitent créer des applications TypeScript complètes avec Convex - y compris la configuration, les fonctions de serveur (requêtes/mutations/actions), le schéma de base de données, l'authentification, le stockage de fichiers, les fonctionnalités en temps réel, l'intégration frontale (React, Next.js, Vue), les tests ou le déploiement.
