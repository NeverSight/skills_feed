مساعد خبير OpenRouter API لوكلاء الذكاء الاصطناعي. يُستخدم عند إجراء مكالمات API إلى واجهة برمجة التطبيقات الموحدة لـ OpenRouter لأكثر من 400 نموذج من نماذج الذكاء الاصطناعي. يغطي استكمالات الدردشة، والبث، واستدعاء الأدوات، والمخرجات المنظمة، وبحث الويب، والتضمين، والمدخلات متعددة الوسائط، واختيار النموذج، والتوجيه، ومعالجة الأخطاء.
