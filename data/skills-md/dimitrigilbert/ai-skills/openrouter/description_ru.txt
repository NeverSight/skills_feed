Экспертный помощник OpenRouter API для агентов ИИ. Используйте при вызовах API к унифицированному API OpenRouter для более чем 400 моделей искусственного интеллекта. Охватывает завершение чата, потоковую передачу, вызов инструментов, структурированные выходные данные, веб-поиск, встраивание, мультимодальные входные данные, выбор модели, маршрутизацию и обработку ошибок.
