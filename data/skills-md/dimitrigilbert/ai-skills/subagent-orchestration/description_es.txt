Organice flujos de trabajo de desarrollo de varias fases con una estricta separación de roles entre implementadores y validadores. Ejecuta planes automáticamente utilizando subagentes separados para implementación, validación y corrección con bucles de reintento automático. Úselo cuando construya sistemas complejos que requieran (1) fases de desarrollo secuenciales o paralelas de varios pasos, (2) validación automatizada con verificación de tipo/construcción/pruebas después de cada fase, (3) bucles de corrección de reintento automático hasta que pase la validación, (4) ejecución completa después de la aprobación de un solo usuario. Los desencadenantes incluyen "implementar este plan de múltiples fases", "construir un sistema con fases", "crear [un sistema complejo] siguiendo esta arquitectura", "automatizar el flujo de trabajo de desarrollo con validación" o cualquier solicitud de desarrollo orquestado con múltiples fases y controles de calidad. NO para tareas simples de un solo archivo o codificación exploratoria.
