Используйте при добавлении интерактивных 3D-сцен из Spline.design в веб-проекты. Охватывает методы внедрения (React, Next.js, vanilla JS), API @splinetool/runtime для программного управления (события, переменные, анимация, камера), оптимизации производительности и общие шаблоны интеграции, такие как главные разделы, средства просмотра продуктов и 3D-изображение с прокруткой.
