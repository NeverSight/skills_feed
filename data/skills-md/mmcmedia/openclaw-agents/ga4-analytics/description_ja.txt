Google Analytics 4、Search Console、および Indexing API ツールキット。 Web サイトのトラフィック、ページのパフォーマンス、ユーザー人口統計、リアルタイムの訪問者、検索クエリ、SEO 指標を分析します。ユーザーが次のことを要求する場合に使用します: サイトトラフィックの確認、ページビューの分析、トラフィックソースの確認、ユーザー属性の表示、リアルタイムの訪問者データの取得、Search Console のクエリの確認、SEO パフォーマンスの分析、URL の再インデックス作成のリクエスト、インデックスステータスの検査、期間の比較、直帰率の確認、コンバージョンデータの表示、または電子商取引収益の取得。 GA4 と Search Console にアクセスできる Google Cloud サービス アカウントが必要です。
