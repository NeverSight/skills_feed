Google Analytics 4, Search Console et la boîte à outils de l'API d'indexation. Analysez le trafic du site Web, les performances des pages, les données démographiques des utilisateurs, les visiteurs en temps réel, les requêtes de recherche et les mesures de référencement. À utiliser lorsque l'utilisateur demande : vérifier le trafic du site, analyser les pages vues, consulter les sources de trafic, afficher les données démographiques des utilisateurs, obtenir des données sur les visiteurs en temps réel, vérifier les requêtes de la console de recherche, analyser les performances de référencement, demander la réindexation des URL, inspecter l'état de l'index, comparer les plages de dates, vérifier les taux de rebond, afficher les données de conversion ou obtenir les revenus du commerce électronique. Nécessite un compte de service Google Cloud avec accès à GA4 et Search Console.
