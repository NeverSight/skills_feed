Google Analytics 4, Search Console e toolkit API di indicizzazione. Analizza il traffico del sito web, le prestazioni della pagina, i dati demografici degli utenti, i visitatori in tempo reale, le query di ricerca e le metriche SEO. Da utilizzare quando l'utente chiede di: controllare il traffico del sito, analizzare le visualizzazioni di pagina, vedere le sorgenti di traffico, visualizzare i dati demografici degli utenti, ottenere dati sui visitatori in tempo reale, controllare le query della console di ricerca, analizzare le prestazioni SEO, richiedere la reindicizzazione degli URL, ispezionare lo stato dell'indice, confrontare intervalli di date, controllare le frequenze di rimbalzo, visualizzare i dati di conversione o ottenere entrate dall'e-commerce. Richiede un account del servizio Google Cloud con accesso GA4 e Search Console.
