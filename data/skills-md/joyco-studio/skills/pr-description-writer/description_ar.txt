قم بإنشاء أوصاف طلب سحب عالية الجودة في Markdown. يدعم أنواع المشكلات والميزات وأنواع العلاقات العامة ذات الميزات الكبيرة بمخرجات منظمة وجاهزة للإنتاج. يقوم أيضًا بإنشاء عناوين العلاقات العامة ويدعم التسليم كمخرج دردشة أو ملف .md قابل للتنزيل. يكتشف بشكل استباقي العلاقات العامة المفتوحة لـ GitHub للفرع الحالي ويعرض تحديث عنوانها ووصفها مباشرة عبر gh CLI.
