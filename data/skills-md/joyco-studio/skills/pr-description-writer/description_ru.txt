Создавайте высококачественные описания запросов на включение в Markdown. Поддерживает типы PR-сообщений о проблемах, функциях и крупных функциях со структурированными, готовыми к производству результатами. Также генерирует PR-заголовки и поддерживает доставку в виде вывода чата или загружаемого файла .md. Заблаговременно обнаруживает открытые запросы GitHub для текущей ветки и предлагает обновить их заголовок и описание непосредственно через gh CLI.
