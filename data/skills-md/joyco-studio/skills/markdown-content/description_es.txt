Guía para crear páginas de contenido de rebajas (blogs, legales, registros de cambios) en Next.js usando fumadocs-mdx, rehype-pretty-code y shiki. Cubre colecciones MDX, tipografía en prosa independiente, bloques de código resaltados con sintaxis y asignaciones de componentes que mantienen visualmente consistentes los componentes de código de React personalizados y de rebajas renderizados. No se requiere ningún marco de interfaz de usuario de documentación.
