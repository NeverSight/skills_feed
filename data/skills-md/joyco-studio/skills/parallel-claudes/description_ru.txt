Руководство по запуску нескольких параллельных сеансов Claude Code с использованием cw (Claude Worktree Manager). Используйте, когда пользователь хочет распараллелить работу в нескольких экземплярах Claude, управлять рабочими деревьями git для параллельных задач по кодированию ИИ или снова объединить результаты. Запускается при выполнении задач, включающих параллельные сеансы Claude, управление рабочим деревом или разделение работы между несколькими агентами.
