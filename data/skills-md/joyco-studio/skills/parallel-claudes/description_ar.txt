دليل لتشغيل جلسات Claude Code المتوازية المتعددة باستخدام cw (Claude Worktree Manager). يُستخدم عندما يريد المستخدم موازنة العمل عبر مثيلات Claude المتعددة، أو إدارة أشجار عمل git لمهام ترميز الذكاء الاصطناعي المتزامنة، أو دمج النتائج معًا مرة أخرى. تشغيل المهام التي تتضمن جلسات Claude المتوازية، أو إدارة شجرة العمل، أو تقسيم العمل عبر وكلاء متعددين.
