Guide pour exécuter plusieurs sessions Claude Code parallèles à l'aide de cw (Claude Worktree Manager). À utiliser lorsque l'utilisateur souhaite paralléliser le travail sur plusieurs instances Claude, gérer les arbres de travail Git pour les tâches de codage d'IA simultanées ou fusionner les résultats. Se déclenche sur des tâches impliquant des sessions Claude parallèles, la gestion d'arbres de travail ou la répartition du travail entre plusieurs agents.
