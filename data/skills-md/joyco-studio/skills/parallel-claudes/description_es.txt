Guía para ejecutar múltiples sesiones paralelas de Claude Code usando cw (Claude Worktree Manager). Úselo cuando el usuario quiera paralelizar el trabajo en múltiples instancias de Claude, administrar árboles de trabajo de git para tareas simultáneas de codificación de IA o fusionar resultados nuevamente. Activadores de tareas que involucran sesiones de Claude paralelas, administración del árbol de trabajo o división del trabajo entre múltiples agentes.
