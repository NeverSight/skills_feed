使用 utoipa 和 Axum 框架的 Rust API 的 OpenAPI 文檔技能。
在通過自動 OpenAPI 規範生成和標量 UI 構建 REST API 時使用。
涵蓋：(1) 用於請求/響應類型的 ToSchema 派生，(2) 用於端點文檔的 #[utoipa::path]，
(3) 用於組合規範的 OpenApi 派生，(4) 安全方案（JWT、API 密鑰），(5) 用於查詢/路徑參數的 IntoParams，
(6) 標量 UI 集成，(7) 響應和錯誤處理模式。
觸發器：utoipa、openapi、scalar、axum api、ToSchema、IntoParams、rust api 文檔。
