مهارة توثيق OpenAPI لواجهات Rust API باستخدام uoipa مع إطار عمل Axum.
يُستخدم عند إنشاء واجهات برمجة تطبيقات REST مع إنشاء مواصفات OpenAPI تلقائيًا وScalar UI.
يغطي: (1) اشتقاق ToSchema لأنواع الطلب/الاستجابة، (2) #[utoipa::path] لتوثيق نقطة النهاية،
(3) مشتق OpenApi لدمج المواصفات، (4) أنظمة الأمان (JWT، مفاتيح API)، (5) IntoParams لمعلمات الاستعلام/المسار،
(6) تكامل واجهة المستخدم العددية، (7) أنماط الاستجابة ومعالجة الأخطاء.
المشغلات: utoipa، openapi، scalar، axum api، ToSchema، IntoParams، Rust api docs.
