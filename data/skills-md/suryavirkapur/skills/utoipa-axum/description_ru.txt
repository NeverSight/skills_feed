Навыки документации OpenAPI для API Rust с использованием utoipa с платформой Axum.
Используйте при создании REST API с автоматическим созданием спецификаций OpenAPI и скалярным пользовательским интерфейсом.
Охватывает: (1) производное от ToSchema для типов запросов/ответов, (2) #[utoipa::path] для документации по конечной точке,
(3) производное OpenApi для объединения спецификаций, (4) схемы безопасности (JWT, ключи API), (5) IntoParams для параметров запроса/пути,
(6) Интеграция скалярного пользовательского интерфейса, (7) Шаблоны ответов и обработки ошибок.
Триггеры: utoipa, openapi, скаляр, axum api, ToSchema, IntoParams, документы Rust API.
