Compétence en développement de framework SolidJS pour créer des applications Web réactives avec une réactivité fine.
À utiliser lorsque vous travaillez avec des projets SolidJS, notamment : (1) Création de composants avec des signaux, des magasins et des effets,
(2) Implémentation d'une gestion d'état réactive, (3) Utilisation de composants de flux de contrôle (Show, For, Switch/Match),
(4) Configuration du routage avec Solid Router, (5) Création d'applications full-stack avec SolidStart,
(6) Récupération de données avec createResource, (7) API de contexte pour l'état partagé, (8) configuration SSR/SSG.
Déclencheurs : solid, solidjs, solid-js, solid start, solidstart, createSignal, createStore, createEffect.
