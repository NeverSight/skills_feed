SolidJS 框架開發技能，用於構建具有細粒度反應性的反應式 Web 應用程序。
在處理 SolidJS 項目時使用，包括：(1) 創建帶有信號、存儲和效果的組件，
(2) 實現反應式狀態管理，(3) 使用控制流組件（Show、For、Switch/Match），
(4) 使用 Solid Router 設置路由，(5) 使用 SolidStart 構建全棧應用程序，
(6) 使用 createResource 獲取數據，(7) 用於共享狀態的上下文 API，(8) SSR/SSG 配置。
觸發器：solid、solidjs、solid-js、solid start、solidstart、createSignal、createStore、createEffect。
