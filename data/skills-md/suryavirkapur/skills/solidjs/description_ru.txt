Навыки разработки среды SolidJS для создания реактивных веб-приложений с детальной реактивностью.
Используйте при работе с проектами SolidJS, включая: (1) Создание компонентов с сигналами, хранилищами и эффектами.
(2) Реализация реактивного управления состоянием, (3) Использование компонентов потока управления (Показать, Для, Переключить/Сопоставить),
(4) Настройка маршрутизации с помощью Solid Router, (5) Создание полнофункциональных приложений с помощью SolidStart,
(6) Извлечение данных с помощью createResource, (7) Контекстный API для общего состояния, (8) Конфигурация SSR/SSG.
Триггеры: Solid, SolidJS, Solid-JS, Solid Start, SolidStart, createSignal, createStore, createEffect.
