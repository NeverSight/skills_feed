Erklärt Web-Rendering-Strategien, einschließlich Client-Side Rendering (CSR), Server-Side Rendering (SSR), Static Site Generation (SSG), Inkrementelle statische Regeneration (ISR) und Streaming. Verwenden Sie diese Option, wenn Sie eine Rendering-Strategie festlegen, die Leistung optimieren oder verstehen möchten, wann Inhalte generiert und an Benutzer gesendet werden.
