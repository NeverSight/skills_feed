Construit des applications ChatGPT avec le SDK OpenAI Apps. À utiliser lorsque : (1) Configuration de serveurs MCP pour l'intégration de ChatGPT, (2) Implémentation des points de terminaison du fournisseur OAuth (vous ÊTES le fournisseur, ChatGPT est le client), (3) Création de widgets avec le type MIME text/html+skybridge, (4) Utilisation de l'API window.openai pour la communication widget-ChatGPT, (5) Débogage d'OAuth, CORS, PKCE, CSP ou problèmes de rendu de widget, (6) Compréhension de l'architecture des applications ChatGPT (MCP + widgets + OAuth).
