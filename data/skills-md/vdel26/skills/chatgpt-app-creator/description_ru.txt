Создает приложения ChatGPT с помощью OpenAI Apps SDK. Используйте, когда: (1) Настройка серверов MCP для интеграции ChatGPT, (2) Реализация конечных точек поставщика OAuth (вы являетесь поставщиком, ChatGPT — клиент), (3) Создание виджетов с типом MIME text/html+skybridge, (4) Использование API window.openai для связи виджет-ChatGPT, (5) Отладка проблем OAuth, CORS, PKCE, CSP или виджетов, (6) Понимание Архитектура приложений ChatGPT (MCP + виджеты + OAuth).
