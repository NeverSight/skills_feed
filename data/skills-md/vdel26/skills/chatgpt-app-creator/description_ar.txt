إنشاء تطبيقات ChatGPT باستخدام OpenAI Apps SDK. يُستخدم عندما: (1) إعداد خوادم MCP لتكامل ChatGPT، (2) تنفيذ نقاط نهاية موفر OAuth (أنت الموفر، ChatGPT هو العميل)، (3) إنشاء عناصر واجهة مستخدم بنوع text/html+skybridge MIME، (4) استخدام window.openai API لاتصال القطعة-ChatGPT، (5) تصحيح أخطاء OAuth، أو CORS، أو PKCE، أو CSP، أو مشكلات عرض الأداة، (6) فهم بنية تطبيقات ChatGPT (MCP + عناصر واجهة المستخدم + OAuth).
