Создайте или расширите расширяемую структуру самопроверки пользовательского интерфейса для веб-интерфейсов (React/Vite или простой HTML) с помощью Playwright. Используйте, когда вас попросят добавить/обновить потоки ui-check или ui-watch, обеспечить обнаружение data-qa/data-testid, добавить проверки темы/контрастности, проверку анимации, проверку загрузки конечного автомата, проверку стабильности/перекрытия макета, правила модальной прокрутки или для документирования стандартов тегирования QA и команд тестирования пользовательского интерфейса.
