قم ببناء أو توسيع إطار عمل الفحص الذاتي لواجهة المستخدم الموسعة لواجهات مستخدم الويب (React/Vite أو HTML العادي) باستخدام Playwright. استخدمه عندما يُطلب منك إضافة/ترقية تدفقات فحص واجهة المستخدم أو مراقبة واجهة المستخدم، أو فرض اكتشاف data-qa/data-testid، أو إضافة عمليات فحص السمة/التباين، أو التحقق من الرسوم المتحركة، أو تحميل عمليات فحص حالة الجهاز، أو عمليات التحقق من ثبات التخطيط/التداخل، أو قواعد التمرير المشروطة، أو لتوثيق معايير وضع علامات ضمان الجودة وأوامر اختبار واجهة المستخدم.
