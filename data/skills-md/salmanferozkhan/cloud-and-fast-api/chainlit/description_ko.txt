Python에서 Chainlit 프레임워크를 사용하여 대화형 AI 애플리케이션을 구축하기 위한 전문가 지침입니다. (1) LLM 애플리케이션용 채팅 인터페이스 생성, (2) OpenAI, LangChain, LlamaIndex 또는 Mistral AI를 사용하여 앱 구축, (3) 스트리밍 응답 구현, (4) 이미지, 파일, 차트와 같은 UI 요소 추가, (5) 사용자 파일 업로드 처리, (6) 인증(OAuth, 비밀번호) 구현, (7) 눈에 보이는 단계가 있는 다단계 워크플로 생성, (8) 문서 업로드로 RAG 애플리케이션 구축 또는 (9) 채팅 앱을 웹, Slack에 배포할 때 사용합니다. 불일치 또는 팀.
