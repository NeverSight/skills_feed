إرشادات الخبراء لبناء تطبيقات الذكاء الاصطناعي للمحادثة باستخدام إطار عمل Chainlit في Python. يُستخدم عند (1) إنشاء واجهات دردشة لتطبيقات LLM، (2) إنشاء تطبيقات باستخدام OpenAI أو LangChain أو LlamaIndex أو Mistral AI، (3) تنفيذ استجابات البث، (4) إضافة عناصر واجهة المستخدم مثل الصور والملفات والمخططات، (5) التعامل مع تحميلات ملفات المستخدم، (6) تنفيذ المصادقة (OAuth، كلمة المرور)، (7) إنشاء سير عمل متعدد الخطوات بخطوات مرئية، (8) إنشاء تطبيقات RAG مع تحميل المستندات، أو (9) نشر تطبيقات الدردشة إلى الويب أو Slack أو Discord أو Teams.
