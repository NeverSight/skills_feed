إرشادات الخبراء لـ SQLModel - مكتبة Python التي تجمع بين SQLAlchemy وPydantic لنماذج قواعد البيانات. يُستخدم عند (1) إنشاء نماذج قاعدة بيانات تعمل كمخططات SQLAlchemy ORM وPydantic، (2) إنشاء تطبيقات FastAPI مع تكامل قاعدة البيانات، (3) تحديد علاقات النماذج (واحد إلى متعدد، متعدد إلى متعدد)، (4) تنفيذ عمليات CRUD مع أمان النوع، (5) إعداد جلسات قاعدة البيانات غير المتزامنة، (6) التكامل مع عمليات الترحيل Alembic، (7) التعامل مع وراثة النموذج والخلطات، أو (8) التحويل بين قاعدة البيانات النماذج ومخططات API.
