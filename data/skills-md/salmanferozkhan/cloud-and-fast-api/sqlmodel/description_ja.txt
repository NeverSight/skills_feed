SQLModel の専門家によるガイダンス - データベース モデル用に SQLAlchemy と Pydantic を組み合わせた Python ライブラリ。 (1) SQLAlchemy ORM と Pydantic スキーマの両方として機能するデータベース モデルの作成、(2) データベース統合を使用した FastAPI アプリの構築、(3) モデルの関係 (1 対多、多対多) の定義、(4) タイプ セーフティを使用した CRUD 操作の実行、(5) 非同期データベース セッションの設定、(6) Alembic 移行との統合、(7) モデルの継承とミックスインの処理、または (8) データベース モデルとAPI スキーマ。
