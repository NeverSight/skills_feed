Guía experta para SQLModel: la biblioteca de Python que combina SQLAlchemy y Pydantic para modelos de bases de datos. Úselo cuando (1) cree modelos de bases de datos que funcionen como SQLAlchemy ORM y esquemas Pydantic, (2) cree aplicaciones FastAPI con integración de bases de datos, (3) defina relaciones de modelos (uno a muchos, muchos a muchos), (4) realice operaciones CRUD con seguridad de tipos, (5) configure sesiones de bases de datos asíncronas, (6) se integre con migraciones de Alembic, (7) maneje la herencia de modelos y mixins, o (8) convierta entre modelos de bases de datos y esquemas API.
