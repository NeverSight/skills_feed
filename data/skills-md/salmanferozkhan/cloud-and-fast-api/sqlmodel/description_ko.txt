SQLModel에 대한 전문가 지침 - 데이터베이스 모델을 위해 SQLAlchemy와 Pydantic을 결합한 Python 라이브러리입니다. (1) SQLAlchemy ORM 및 Pydantic 스키마로 작동하는 데이터베이스 모델 생성, (2) 데이터베이스 통합으로 FastAPI 앱 구축, (3) 모델 관계(일대다, 다대다) 정의, (4) 유형 안전성을 갖춘 CRUD 작업 수행, (5) 비동기 데이터베이스 세션 설정, (6) Alembic 마이그레이션과 통합, (7) 모델 상속 및 믹스인 처리 또는 (8) 데이터베이스 모델과 API 스키마 간 변환 시 사용합니다.
