Guide du cadre de test PowerShell TDD pour Pester v5+. À utiliser lors de l'écriture, de la structuration ou du débogage de tests unitaires PowerShell ; applets de commande moqueuses, commandes natives (bash, git, curl) ou types .NET ; isoler les tests avec TestDrive/TestRegistry ; capturer les flux de sortie ; générer une couverture de code ou des rapports JUnit/NUnit pour CI/CD ; exécuter des tests paramétrés ou étiquetés ; ou dépanner les problèmes de phase Pester Discovery vs Run.
