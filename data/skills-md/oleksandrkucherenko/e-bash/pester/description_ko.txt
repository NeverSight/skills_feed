Pester v5+에 대한 PowerShell TDD 테스트 프레임워크 지침입니다. PowerShell 단위 테스트를 작성, 구조화 또는 디버깅할 때 사용합니다. cmdlet, 기본 명령(bash, git,curl) 또는 .NET 유형을 모의합니다. TestDrive/TestRegistry를 사용하여 테스트를 격리합니다. 출력 스트림 캡처; CI/CD에 대한 코드 적용 범위 또는 JUnit/NUnit 보고서 생성 매개변수화되거나 태그가 지정된 테스트를 실행합니다. 또는 Pester Discovery와 실행 단계 문제를 해결합니다.
