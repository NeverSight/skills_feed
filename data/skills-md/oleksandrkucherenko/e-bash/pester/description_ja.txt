Pester v5+ の PowerShell TDD テスト フレームワーク ガイダンス。 PowerShell 単体テストの作成、構造化、またはデバッグ時に使用します。コマンドレット、ネイティブ コマンド (bash、git、curl)、または .NET タイプのモック。 TestDrive/TestRegistry を使用してテストを分離する。出力ストリームをキャプチャする。 CI/CD のコード カバレッジまたは JUnit/NUnit レポートを生成します。パラメータ化されたテストまたはタグ付けされたテストを実行します。または、Pester Discovery と Run フェーズの問題のトラブルシューティングを行います。
