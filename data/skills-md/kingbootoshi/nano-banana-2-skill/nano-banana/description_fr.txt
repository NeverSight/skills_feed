Génère des images IA à l'aide de la CLI nano-banane (Gemini 3.1 Flash par défaut, Pro disponible). Gère la multi-résolution (512-4K), les formats d'image, les images de référence pour le transfert de style, le flux de travail sur écran vert pour les actifs transparents, le suivi des coûts et le contrôle exact des dimensions. À utiliser lorsqu'on vous demande de « générer une image », « créer un sprite », « créer un élément », « générer une illustration » ou toute tâche de génération d'image pour les maquettes d'interface utilisateur, les ressources de jeu, les vidéos ou les supports marketing.
