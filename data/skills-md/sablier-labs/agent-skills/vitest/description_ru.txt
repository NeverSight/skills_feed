Этот навык следует использовать, когда пользователь просит «написать тесты», «добавить тесты», «покрыть тестами», «запустить тесты», «отладить неудачные тесты», «имитировать функции» или упоминает Vitest, модульные тесты, компонентные тесты, разработку через тестирование или утилиты тестирования. Предоставляет подробное руководство по Vitest v4 для проектов TypeScript React/Next.js.
