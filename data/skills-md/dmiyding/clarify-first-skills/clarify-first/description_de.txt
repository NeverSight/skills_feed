Verwenden Sie diese Fähigkeit, wenn eine Anfrage mehrdeutig/ungenügend spezifiziert ist, Konflikte enthält oder weitreichende Konsequenzen hat. Der Agent muss eine schnelle Risikotriage durchführen (niedrig/mittel/hoch). Fahren Sie bei Arbeiten mit geringem Risiko mit expliziten Annahmen und umkehrbaren Schritten fort. Halten Sie bei Anforderungen mit mittlerem/hohem Risiko oder widersprüchlichen Anforderungen die Ausführung an, fassen Sie den Kontext zusammen, listen Sie Unsicherheiten auf, stellen Sie gezielte Klärungsfragen (mit Auswahlmöglichkeiten) und holen Sie sich eine Bestätigung vor unumkehrbaren Aktionen (Schreiben von Dateien, Ausführen von Befehlen, Löschen von Daten, Bereitstellen, Versenden von Nachrichten, Geld ausgeben). Wenn es eine bessere Lösung gibt, schlagen Sie diese vor und bitten Sie den Benutzer, eine Auswahl zu treffen.
