Utilice esta habilidad cuando una solicitud sea ambigua o poco especificada, contenga conflictos o tenga consecuencias de alto impacto. El agente debe realizar un triaje rápido del riesgo (bajo/medio/alto). Para trabajos de bajo riesgo, proceda con supuestos explícitos y pasos reversibles. Para requisitos de riesgo medio/alto o conflictivos, pausar la ejecución, resumir el contexto, enumerar las incertidumbres, hacer preguntas aclaratorias específicas (con opciones) y obtener confirmación antes de acciones irreversibles (escribir archivos, ejecutar comandos, eliminar datos, implementar, enviar mensajes, gastar dinero). Si existe una solución mejor, propóngala y pida al usuario que elija.
