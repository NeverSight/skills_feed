Реализуйте реактивное управление состоянием на основе сигналов в Angular v20+. Используйте для создания реактивного состояния с помощью signal(), производного состояния с помощью Computed(), зависимого состояния с помощью linkedSignal() и побочных эффектов с помощью effect(). Запускает вопросы управления состоянием, преобразует шаблоны BehaviorSubject/Observable в сигналы или реализует реактивные потоки данных.
