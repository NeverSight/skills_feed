Implémentez la récupération de données HTTP dans Angular v20+ à l'aide de resource(), httpResource() et HttpClient. À utiliser pour les appels API, le chargement de données avec des signaux, la gestion des requêtes/réponses et les intercepteurs. Déclencheurs sur la récupération de données, l'intégration d'API, les états de chargement, la gestion des erreurs ou la conversion du HTTP basé sur les observables en modèles basés sur le signal.
