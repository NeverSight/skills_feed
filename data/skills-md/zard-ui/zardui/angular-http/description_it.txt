Implementa il recupero dei dati HTTP in Angular v20+ utilizzando Resource(), httpResource() e HttpClient. Utilizzare per chiamate API, caricamento dati con segnali, gestione di richieste/risposte e intercettori. Trigger relativi al recupero dei dati, all'integrazione API, agli stati di caricamento, alla gestione degli errori o alla conversione di HTTP basato su osservabile in modelli basati su segnali.
