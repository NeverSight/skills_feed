Cree formularios basados ​​en señales en Angular v21+ utilizando la nueva API Signal Forms. Úselo para la creación de formularios con enlace bidireccional automático, validación basada en esquemas, administración del estado de los campos y formularios dinámicos. Activadores al implementar formularios, agregar validación, crear formularios de varios pasos o crear formularios con campos condicionales. Signal Forms son experimentales pero se recomiendan para nuevos proyectos de Angular.
