Utilizzare quando gli utenti chiedono di navigare nei siti Web, compilare moduli, acquisire screenshot, estrarre dati Web, testare app Web o automatizzare i flussi di lavoro del browser. Le frasi trigger includono "vai a [url]", "fai clic su", "compila il modulo", "fai uno screenshot", "raschia", "automatizza", "prova il sito web", "accedi" o qualsiasi richiesta di interazione con il browser. NON caricare per: condivisione di URL, incorporamento di collegamenti, file di immagini di screenshot.
