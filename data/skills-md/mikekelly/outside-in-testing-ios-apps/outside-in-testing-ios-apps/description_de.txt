Testen Sie iOS-Apps, die im Simulator ausgeführt werden, von außen nach innen. Verwenden Sie PROAKTIV, wenn Sie die Funktionsfähigkeit einer iOS-App überprüfen, Akzeptanztests durchführen, das UI-Verhalten überprüfen, Outside-in-Tests, End-to-End-Tests, Funktionstests, explorative Tests oder Smoke-Tests von iOS-Apps durchführen müssen. Behandelt das Starten von Testabläufen, das Überprüfen von Bildschirmen, die Interaktion mit UI-Elementen und das Diagnostizieren von Fehlern. Wird ausgelöst bei: Testen Sie meine App, überprüfen Sie die App, überprüfen Sie die Benutzeroberfläche, führt diese Arbeit aus, Abnahmetest, Outside-In-Test, E2E-Test, Funktionstest, Rauchtest, Erkundungstest.
