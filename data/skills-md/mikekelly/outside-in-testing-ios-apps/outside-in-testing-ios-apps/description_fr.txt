Testez les applications iOS exécutées dans Simulator de l'extérieur vers l'intérieur. Utilisez de manière PROACTIVE lorsque vous devez vérifier le fonctionnement d'une application iOS, exécuter des tests d'acceptation, vérifier le comportement de l'interface utilisateur, effectuer des tests de l'extérieur vers l'intérieur, des tests de bout en bout, des tests fonctionnels, des tests exploratoires ou des tests de fumée des applications iOS. Couvre le lancement de flux de test, la vérification des écrans, l'interaction avec les éléments de l'interface utilisateur et le diagnostic des échecs. Déclencheurs sur : tester mon application, vérifier l'application, vérifier l'interface utilisateur, faire ce travail, test d'acceptation, test extérieur-intérieur, test e2e, test fonctionnel, test de fumée, test exploratoire.
