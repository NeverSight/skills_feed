DOIT être utilisé lorsque vous devez naviguer sur le Web. Automatisation efficace du navigateur conçue pour les agents : permet une navigation Web intuitive, le remplissage de formulaires, des captures d'écran et la récupération de données via des flux de travail basés sur l'accessibilité. Déclencheurs sur : parcourir le site Web, visiter l'URL, ouvrir une page Web, remplir un formulaire, cliquer sur un bouton, prendre une capture d'écran, récupérer des données, automatisation Web, interagir avec le site Web.
