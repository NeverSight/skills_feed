DEVE essere utilizzato quando è necessario navigare sul Web. Automazione efficiente del browser progettata per gli agenti: consente la navigazione web intuitiva, la compilazione di moduli, schermate e raccolta di dati attraverso flussi di lavoro basati sull'accessibilità. Si attiva su: sfoglia il sito web, visita l'URL, apri la pagina web, compila il modulo, fai clic sul pulsante, acquisisci screenshot, acquisisci dati, automazione web, interagisci con il sito web.
