ДОЛЖЕН использоваться, когда вам нужно просматривать веб-страницы. Эффективная автоматизация браузера, разработанная для агентов, обеспечивает интуитивно понятную веб-навигацию, заполнение форм, создание снимков экрана и сбор данных с помощью рабочих процессов, основанных на специальных возможностях. Срабатывает при: просмотре веб-сайта, посещении URL-адреса, открытии веб-страницы, заполнении формы, нажатии кнопки, создании снимка экрана, очистке данных, веб-автоматизации, взаимодействии с веб-сайтом.
