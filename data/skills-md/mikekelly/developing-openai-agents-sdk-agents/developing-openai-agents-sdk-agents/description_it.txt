Costruisci, crea, esegui il debug, rivedi, implementa e ottimizza le applicazioni AI agenti utilizzando OpenAI Agents SDK per TypeScript. Da utilizzare durante la creazione di nuovi agenti, la definizione di strumenti, l'implementazione di trasferimenti tra agenti, l'aggiunta di guardrail, il debug del comportamento degli agenti, la revisione del codice dell'agente o l'orchestrazione di sistemi multi-agente con il pacchetto @openai/agents.
