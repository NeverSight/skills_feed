قم ببناء وإنشاء وتصحيح الأخطاء ومراجعة وتنفيذ وتحسين تطبيقات الذكاء الاصطناعي الوكيل باستخدام OpenAI Agents SDK لـ TypeScript. يُستخدم عند إنشاء وكلاء جدد، أو تحديد الأدوات، أو تنفيذ عمليات التسليم بين الوكلاء، أو إضافة حواجز الحماية، أو تصحيح سلوك الوكيل، أو مراجعة رمز الوكيل، أو تنسيق أنظمة متعددة الوكلاء باستخدام حزمة @openai/agents.
