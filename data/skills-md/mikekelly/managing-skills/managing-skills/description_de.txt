Agentenfähigkeiten installieren, finden, aktualisieren und verwalten. Verwenden Sie diese Option, wenn der Benutzer eine neue Fertigkeit hinzufügen, nach Fertigkeiten suchen möchte, die etwas bewirken, prüfen möchte, ob Fertigkeiten aktuell sind, oder vorhandene Fertigkeiten aktualisieren möchte. Wird ausgelöst bei: Skill installieren, Skill hinzufügen, Skill abrufen, Skill finden, Skill suchen, Skill aktualisieren, Skills prüfen, Skills auflisten.
