AI personal memory system's memory query function. Retrieves memory files from all levels and synthesizes multi-level information to answer user questions. Use cases: (1) When the user asks "In my memory about XXX"; (2) When the user inquires about their habits/preferences/values; (3) When providing recommendations based on user history. This skill automatically searches levels L1–L4, cites sources, and gives personalized answers based on memory.
