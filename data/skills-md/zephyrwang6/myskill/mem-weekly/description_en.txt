Weekly Review feature of the AI personal memory system. Analyzes this week’s L1 context-layer entries, identifies recurring patterns, and distills them into the L2 behavior layer.  
Use cases:  
(1) When the user says “weekly review”;  
(2) During scheduled weekly retrospectives;  
(3) When daily records need to be distilled into behavior patterns.  

This skill automatically tallies behaviors that repeat 3+ times, generates a candidate list, and, once the user confirms, updates the L2 behavior layer.
