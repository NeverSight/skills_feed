AI Personal Memory System's memory recording function. Automatically extracts key information from conversations and records it into the appropriate level.  
Use cases:  
(1) When the user says "record into the memory system," "remember this," or "save this conversation";  
(2) When important events, decisions, or preference expressions are detected;  
(3) When the user completes an important task or makes a decision.  

This skill automatically determines whether to record to L1 Context Layer, L2 Behavior Layer, L3 Cognitive Layer, or suggests updating the L4 Core Layer.
