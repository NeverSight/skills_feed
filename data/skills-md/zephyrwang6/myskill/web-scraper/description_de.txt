Rufen und extrahieren Sie Inhalte von Webseiten und konvertieren Sie HTML in sauberes Markdown. Verwenden Sie es, wenn Benutzer Webartikel lesen, Informationen aus URLs extrahieren, Webinhalte extrahieren möchten oder wenn das integrierte WebFetch-Tool aufgrund von Netzwerkbeschränkungen fehlschlägt. Wird ausgelöst, wenn der Benutzer URLs zum Lesen bereitstellt, den Abruf von Webinhalten anfordert oder Text von Websites extrahieren muss.
