Busque y extraiga contenido de páginas web, convirtiendo HTML en rebajas limpias. Utilícelo cuando los usuarios quieran leer artículos web, extraer información de URL, extraer contenido web o cuando la herramienta WebFetch integrada falle debido a restricciones de red. Se activa cuando el usuario proporciona URL para leer, solicita recuperar contenido web o necesita extraer texto de sitios web.
