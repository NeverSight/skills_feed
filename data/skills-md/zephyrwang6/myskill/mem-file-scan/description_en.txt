File-scan review feature of an AI personal memory system. Scans files modified this week in the Obsidian vault (excluding the AI_MEMORY folder) to identify potentially important events and decisions. Use cases: (1) when the user says "file scan," "check this week's files," or "scan file changes"; (2) before weekly review to supplement L1 records; (3) when the user wants to review this week's activity in Obsidian. This skill lists the modified files, asks the user which should be recorded in L1, and assists in recording them.
