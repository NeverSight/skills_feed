Monthly review feature of the AI personal memory system. Analyzes updates in the L2 behavior layer this month, distills thinking patterns into the L3 cognition layer, and checks whether the L4 core layer needs calibration. Use cases: (1) when the user says "monthly review"; (2) during regular end-of-month reviews; (3) when behavioral patterns need to be distilled into a cognitive framework. This skill will analyze whether multiple behaviors point to the same thinking principle, attempt to distill it into L3, and guide the user to reflect on the L4 core layer.
