Schließt jede PR für die Zusammenführung ab, indem die Implementierung der Titel-/Beschreibungsübereinstimmung überprüft wird UND eine Codeüberprüfung auf Best Practices durchgeführt wird. Verwenden Sie diese Option, wenn Sie aufgefordert werden, „PR fertigzustellen“, „PR-Beschreibung zu überprüfen“, „Commit-Nachricht zu überprüfen“, bevor Sie PR zusammenführen oder wenn sich die PR-Implementierung während der Überprüfung geändert hat. NICHT zum Extrahieren von Lektionen (verwenden Sie Learn-from-PR), zum Schreiben von Tests (verwenden Sie Write-Tests-Agent) oder zum Untersuchen von Build-Fehlern (verwenden Sie pr-build-status) verwenden.
