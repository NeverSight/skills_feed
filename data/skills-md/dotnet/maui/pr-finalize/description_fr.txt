Finalise tout PR à fusionner en vérifiant la mise en œuvre de la correspondance titre/description ET en effectuant une révision du code pour les meilleures pratiques. À utiliser lorsqu'il est demandé de « finaliser le PR », « vérifier la description du PR », « réviser le message de validation », avant de fusionner un PR, ou lorsque l'implémentation du PR a changé pendant la révision. Ne PAS utiliser pour extraire des leçons (utilisez learn-from-pr), écrire des tests (utilisez write-tests-agent) ou enquêter sur les échecs de construction (utilisez pr-build-status).
