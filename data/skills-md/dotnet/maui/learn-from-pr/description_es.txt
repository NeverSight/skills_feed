Analiza un PR completado para extraer lecciones aprendidas del comportamiento del agente. Úselo después de cualquier PR con participación de un agente, ya sea que el agente haya fallado, haya tenido éxito lentamente o haya tenido éxito rápidamente. Identifica patrones para reforzar o corregir y genera recomendaciones prácticas para archivos de instrucción, habilidades y documentación.
