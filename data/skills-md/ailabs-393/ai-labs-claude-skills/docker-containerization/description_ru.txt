Этот навык следует использовать при контейнеризации приложений с помощью Docker, создании файлов Dockerfile, конфигурациях Docker-Compose или развертывании контейнеров на различных платформах. Идеально подходит для приложений Next.js, React, Node.js, требующих контейнеризации для разработки, производства или конвейеров CI/CD. Используйте этот навык, когда пользователям нужны конфигурации Docker, многоэтапные сборки, оркестровка контейнеров или развертывание в Kubernetes, ECS, Cloud Run и т. д.
