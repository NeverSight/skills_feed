Этот навык следует использовать всякий раз, когда пользователям нужна помощь в создании резюме, обновлении профессиональных профилей, отслеживании карьерного опыта, управлении портфелем проектов или составлении индивидуальных резюме для заявлений о приеме на работу. При первом использовании извлекает данные из существующего резюме пользователя и поддерживает структурированную базу данных об опыте, проектах, образовании и навыках. Создает профессионально оформленные одностраничные резюме в формате PDF, адаптированные для конкретных должностей, выбирая из базы данных только самую актуальную информацию.
