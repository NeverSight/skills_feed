Questa competenza deve essere utilizzata durante la creazione o la configurazione di file della pipeline CI/CD per test, creazione e distribuzione automatizzati. Utilizzalo per generare flussi di lavoro GitHub Actions, configurazioni CI GitLab, configurazioni CircleCI o altre configurazioni della piattaforma CI/CD. Ideale per configurare pipeline automatizzate per applicazioni Node.js/Next.js, inclusi linting, test, creazione e distribuzione su piattaforme come Vercel, Netlify o AWS.
