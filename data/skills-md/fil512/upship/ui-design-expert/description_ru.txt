Экспертное руководство по дизайну пользовательского интерфейса для совершенствования игровых интерфейсов. Используйте при просмотре снимков экрана, рекомендации по улучшению CSS, разработке цветовых схем, улучшении типографики, визуальном доработке или придании игре профессионального вида. Специализируется на эстетике настольных игр с темой эпохи стимпанк/брасс.
