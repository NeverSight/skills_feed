Crea e modifica le competenze di Claude Code con frontmatter YAML, struttura delle cartelle e contenuti approfonditi. Da utilizzare quando si creano nuove competenze, si aggiornano competenze esistenti, si progettano metadati SKILL.md, si organizzano cartelle di competenze, si convalida la struttura delle competenze o si aggiungono script Python e TypeScript per operazioni deterministiche.
