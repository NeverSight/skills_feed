Создает и редактирует навыки Claude Code с помощью YAML-интерфейса, структуры папок и контента с масштабированием по глубине. Используйте при создании новых навыков, обновлении существующих навыков, разработке метаданных SKILL.md, организации папок навыков, проверке структуры навыков или добавлении сценариев Python и TypeScript для детерминированных операций.
