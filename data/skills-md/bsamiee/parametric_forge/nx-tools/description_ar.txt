البيانات التعريفية لمساحة عمل الاستعلامات Nx، وتكوينات المشروع، والكشف المتأثر، ومخططات المولد، والرسوم البيانية التبعية عبر Python CLI الموحد. يُستخدم عند تحليل بنية monorepo، أو فحص تكوينات project.json، أو تحديد المشاريع المتأثرة لتحسين CI، أو اكتشاف المولدات المتاحة، أو تصور تبعيات مساحة العمل.
