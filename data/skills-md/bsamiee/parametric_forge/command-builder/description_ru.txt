Создает и настраивает команды косой черты Claude Code с использованием заголовка YAML, обработки аргументов и разрешений инструмента. Используйте при создании новых команд, добавлении параметров $ARGUMENTS или $1-$N, настройке разрешенных инструментов, организации пространств имен команд или исправлении сбоев команд.
