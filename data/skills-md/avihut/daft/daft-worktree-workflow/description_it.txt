Guida il flusso di lavoro stupido dell'albero di lavoro per lo sviluppo Git compartimentato. Da utilizzare quando si lavora in repository gestiti daft (repos con una directory .git/ bare e alberi di lavoro ramificati come directory sorelle), quando si imposta l'isolamento dell'ambiente dell'albero di lavoro o quando gli utenti chiedono informazioni sui flussi di lavoro basati sull'albero di lavoro. Copre i comandi daft, l'automazione degli hook tramite daft.yml e gli strumenti ambientali come mise, direnv, nvm e pyenv.
