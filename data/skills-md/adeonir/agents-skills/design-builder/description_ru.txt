Конвейер проектирования в код: извлекайте копии из URL-адресов, извлекайте токены дизайна из изображений, затем создавайте компоненты React или варианты предварительного просмотра HTML. Используйте, когда: извлечение контента с веб-сайтов, извлечение систем дизайна, генерация кода внешнего интерфейса, предварительный просмотр вариантов дизайна, отправка в Figma через MCP. Также используйте, когда пользователь хочет создать страницу на основе URL-адреса или снимка экрана, изменить дизайн существующего сайта, создать визуальные прототипы или сгенерировать код на основе дизайна. Триггеры: «извлечь копию», «извлечь дизайн», «создать внешний интерфейс», «создать варианты», «экспортировать дизайн», «отправить в Figma», «создать на основе образца», «перепроектировать это», «создать прототип».
