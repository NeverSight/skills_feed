デザインからコードまでのパイプライン: URL からコピーを抽出し、画像からデザイン トークンを抽出して、React コンポーネントまたは HTML プレビュー バリアントを構築します。 Web サイトからコンテンツを抽出する、デザイン システムを抽出する、フロントエンド コードを生成する、デザイン バリアントをプレビューする、MCP 経由で Figma に送信する場合に使用します。ユーザーが参照 URL またはスクリーンショットからページを構築する場合、既存のサイトを再デザインする場合、ビジュアル プロトタイプを作成する場合、またはデザインからコードを生成する場合にも使用します。 「コピーの抽出」、「デザインの抽出」、「フロントエンドの構築」、「バリアントの生成」、「デザインのエクスポート」、「Figma に送信」、「参照から構築」、「これを再設計」、「プロトタイプの作成」をトリガーします。
