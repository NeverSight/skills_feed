Assistant de flux de travail Git pour les validations conventionnelles, la révision du code évaluée en toute confiance et la gestion des demandes d'extraction. À utiliser lorsque : valider des modifications, réviser le code, créer des PR, générer des descriptions de PR. À utiliser également lorsque l'utilisateur dit qu'il a fini de coder et qu'il est prêt à s'engager, souhaite des commentaires sur ses modifications, doit pousser et ouvrir un PR, ou demande de vérifier son code avant de fusionner. Déclencheurs sur « valider », « réviser », « pousser », « créer un PR », « description du PR », « résumer les modifications », « terminé », « prêt à pousser », « vérifier mon code ».
