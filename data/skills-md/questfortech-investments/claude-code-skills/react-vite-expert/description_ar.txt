خبرة React + Vite الكاملة لبناء تطبيقات محسنة وقابلة للتطوير. يغطي بنية المشروع، وبنية المجلد، وأنماط المكونات، وتحسين الأداء، وأفضل ممارسات TypeScript، والاختبار، وتكوين البناء. يُستخدم عند إنشاء تطبيقات React باستخدام Vite، أو تنظيم المشاريع، أو تحسين الأداء، أو تنفيذ أفضل الممارسات. يتضمن مولدات الأكواد ومحلل الحزم والقوالب الجاهزة للإنتاج.
