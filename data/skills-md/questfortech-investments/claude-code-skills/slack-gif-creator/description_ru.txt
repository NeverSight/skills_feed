Набор инструментов для создания анимированных GIF-файлов, оптимизированных для Slack, с валидаторами ограничений размера и составными примитивами анимации. Этот навык применяется, когда пользователи запрашивают анимированные GIF-файлы или эмодзи-анимацию для Slack из описаний типа «сделай мне GIF для Slack из X, делающего Y».
