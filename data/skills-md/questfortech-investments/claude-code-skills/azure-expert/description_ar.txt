خبرة سحابة Azure شاملة تغطي جميع الخدمات الرئيسية (خدمة التطبيقات، الوظائف، تطبيقات الحاويات، AKS، قواعد البيانات، التخزين، المراقبة). يُستخدم عند العمل مع البنية الأساسية لـ Azure، أو عمليات النشر، أو استكشاف الأخطاء وإصلاحها، أو تحسين التكلفة، أو IaC (Bicep/ARM)، أو خطوط أنابيب CI/CD، أو أي مهام تطوير متعلقة بـ Azure. يوفر البرامج النصية والقوالب وأفضل الممارسات لحلول Azure الجاهزة للإنتاج.
