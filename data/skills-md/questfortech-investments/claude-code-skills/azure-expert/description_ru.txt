Комплексный опыт работы в облаке Azure, охватывающий все основные службы (служба приложений, функции, приложения-контейнеры, AKS, базы данных, хранилище, мониторинг). Используйте при работе с инфраструктурой Azure, развертываниях, устранении неполадок, оптимизации затрат, IaC (Bicep/ARM), конвейерах CI/CD или любых задачах разработки, связанных с Azure. Предоставляет сценарии, шаблоны и рекомендации для готовых к работе решений Azure.
