Expertise complète du cloud Azure couvrant tous les services majeurs (App Service, Fonctions, Container Apps, AKS, bases de données, stockage, surveillance). À utiliser lorsque vous travaillez avec l'infrastructure Azure, les déploiements, le dépannage, l'optimisation des coûts, l'IaC (Bicep/ARM), les pipelines CI/CD ou toute tâche de développement liée à Azure. Fournit des scripts, des modèles et des bonnes pratiques pour les solutions Azure prêtes pour la production.
