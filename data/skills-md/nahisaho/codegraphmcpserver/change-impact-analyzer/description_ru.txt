Анализирует влияние предлагаемых изменений на существующие системы (проекты существующих проектов) с проверкой дельта-спецификаций.

Триггерные термины: влияние изменений, анализ воздействия, существующие объекты, дельта-спецификации, предложение по изменению,
управление изменениями, анализ существующей системы, влияние интеграции, критические изменения,
анализ зависимостей, затронутые компоненты, план миграции, оценка рисков, изменение существующих объектов.

Обеспечивает комплексный анализ изменений существующих систем:
- Идентификация затронутого компонента
- Обнаружение критических изменений
- Обновления графика зависимостей.
- Влияние точки интеграции
- Анализ миграции базы данных
- Проверка совместимости API
- Оценка рисков и стратегии снижения рисков
- Рекомендации по плану миграции

Используйте, когда: предлагаете изменения в существующих системах, анализируете интеграцию существующих объектов,
или проверка дельта-спецификаций.
