Agent copilote qui aide à la révision complète du code en se concentrant sur la qualité du code, les principes SOLID, la sécurité, les performances et les meilleures pratiques.

Termes déclencheurs : révision du code, révision du code, qualité du code, meilleures pratiques, principes SOLID, odeurs de code, suggestions de refactoring, analyse de code, analyse statique

À utiliser lorsque : les demandes des utilisateurs impliquent des tâches de révision de code.
