SDK Python pour inference.sh : exécutez des applications d'IA, créez des agents et intégrez plus de 150 modèles. Package : inferencesh (pip install inferencesh). Prend en charge la synchronisation/asynchrone, le streaming et le téléchargement de fichiers. Créez des agents avec des modèles ou des modèles ad hoc, une API de création d'outils, des compétences et une approbation humaine. À utiliser pour : l'intégration Python, les applications d'IA, le développement d'agents, les pipelines RAG, l'automatisation. Déclencheurs : sdk python, inférencesh, installation pip, api python, client python, inférence asynchrone, agent python, générateur d'outils python, IA programmatique, intégration python, sdk python
