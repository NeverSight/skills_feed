Automatisez Twitter/X avec la publication, l'engagement et la gestion des utilisateurs via la CLI inference.sh. Applications : x/post-tweet, x/post-create (avec média), x/post-like, x/post-retweet, x/dm-send, x/user-follow. Capacités : publier des tweets, planifier du contenu, aimer des publications, retweeter, envoyer des DM, suivre des utilisateurs, obtenir des profils. À utiliser pour : l'automatisation des médias sociaux, la planification de contenu, les robots d'engagement, la croissance de l'audience, l'API X. Déclencheurs : API Twitter, API X, automatisation des tweets, publication sur Twitter, bot Twitter, automatisation des médias sociaux, automatisation X, planificateur de tweets, intégration Twitter, publication de tweet, publication Twitter, publication x, envoi de tweet
