التحكم في متصفح Chrome باستخدام الذكاء الاصطناعي باستخدام بروتوكول MCP. يُستخدم عندما يريد المستخدمون أتمتة مهام المتصفح، أو التقاط لقطات شاشة، أو ملء النماذج، أو النقر فوق العناصر، أو التنقل بين الصفحات، أو البحث في سجل التصفح، أو إدارة الإشارات المرجعية، أو تنفيذ أي أتمتة تعتمد على المتصفح. يعمل مع متصفح Chrome الحالي وجلسات تسجيل الدخول.
