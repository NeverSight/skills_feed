Рекомендации по оптимизации производительности React от Vercel Engineering. Этот навык следует использовать при написании, проверке или рефакторинге кода React, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, включающих компоненты React, получение данных, оптимизацию пакетов или повышение производительности.
