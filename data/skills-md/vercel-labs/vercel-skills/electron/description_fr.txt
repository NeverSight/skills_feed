Automatisez les applications de bureau Electron (VS Code, Slack, Discord, Figma, Notion, Spotify, etc.) à l'aide du navigateur d'agent via le protocole Chrome DevTools. À utiliser lorsque l'utilisateur doit interagir avec une application Electron, automatiser une application de bureau, se connecter à une application en cours d'exécution, contrôler une application native ou tester une application Electron. Les déclencheurs incluent « automatiser l'application Slack », « contrôler VS Code », « interagir avec l'application Discord », « tester cette application Electron », « se connecter à l'application de bureau » ou toute tâche nécessitant l'automatisation d'une application Electron native.
