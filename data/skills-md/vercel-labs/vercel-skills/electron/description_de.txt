Automatisieren Sie Electron-Desktop-Apps (VS Code, Slack, Discord, Figma, Notion, Spotify usw.) mithilfe eines Agent-Browsers über das Chrome DevTools-Protokoll. Verwenden Sie diese Option, wenn der Benutzer mit einer Electron-App interagieren, eine Desktop-App automatisieren, eine Verbindung zu einer laufenden App herstellen, eine native App steuern oder eine Electron-Anwendung testen muss. Zu den Auslösern gehören „Slack-App automatisieren“, „VS-Code steuern“, „mit Discord-App interagieren“, „Diese Electron-App testen“, „Mit Desktop-App verbinden“ oder jede Aufgabe, die die Automatisierung einer nativen Electron-Anwendung erfordert.
