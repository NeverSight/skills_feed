رد فعل العارض الأصلي لـ json-render الذي يحول مواصفات JSON إلى واجهات مستخدم أصلية للهاتف المحمول. يُستخدم عند العمل مع @json-render/react-native، أو إنشاء واجهات مستخدم React Native من JSON، أو إنشاء كتالوجات مكونات الهاتف المحمول، أو عرض المواصفات التي تم إنشاؤها بواسطة الذكاء الاصطناعي على الهاتف المحمول.
