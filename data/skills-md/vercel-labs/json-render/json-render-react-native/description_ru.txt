Средство рендеринга React Native для json-рендеринга, которое превращает спецификации JSON в собственные мобильные пользовательские интерфейсы. Используйте при работе с @json-render/react-native, создании пользовательских интерфейсов React Native из JSON, создании каталогов мобильных компонентов или рендеринге спецификаций, созданных искусственным интеллектом, на мобильных устройствах.
