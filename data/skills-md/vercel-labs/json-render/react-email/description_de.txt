React Email-Renderer für JSON-Render, der JSON-Spezifikationen mithilfe von @react-email/components und @react-email/render in HTML- oder Nur-Text-E-Mails umwandelt. Verwenden Sie diese Option, wenn Sie mit @json-render/react-email arbeiten, Transaktions- oder Marketing-E-Mails aus JSON erstellen, E-Mail-Kataloge erstellen, KI-generierte E-Mail-Spezifikationen rendern oder wenn der Benutzer React-Email, HTML-E-Mail oder Transaktions-E-Mail erwähnt.
