Representador de correo electrónico React para json-render que convierte las especificaciones JSON en HTML o correos electrónicos de texto sin formato usando @react-email/components y @react-email/render. Úselo cuando trabaje con @json-render/react-email, cree correos electrónicos transaccionales o de marketing a partir de JSON, cree catálogos de correo electrónico, represente especificaciones de correo electrónico generadas por IA o cuando el usuario mencione reaccionar-email, correo electrónico HTML o correo electrónico transaccional.
