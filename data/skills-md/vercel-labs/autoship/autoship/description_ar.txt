أداة CLI للإصدارات الآلية المستندة إلى مجموعة التغييرات مع أوصاف تم إنشاؤها بواسطة الذكاء الاصطناعي. يُستخدم عندما يحتاج المستخدم إلى إصدار حزمة، أو إنشاء مجموعات تغييرات، أو زيادة الإصدارات، أو أتمتة سير عمل نشر npm. تتضمن المشغلات طلبات "لتحرير حزمة"، أو "إنشاء مجموعة تغييرات"، أو "النشر إلى npm"، أو "رفع الإصدار"، أو "أتمتة الإصدارات"، أو أي مهمة تتضمن إدارة الإصدار ونشر الحزمة للمستودعات باستخدام سير عمل مجموعات التغييرات.
