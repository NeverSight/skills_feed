Outil CLI pour les versions automatisées basées sur des ensembles de modifications avec des descriptions générées par l'IA. À utiliser lorsque l'utilisateur doit publier un package, créer des ensembles de modifications, modifier des versions ou automatiser les flux de publication npm. Les déclencheurs incluent les demandes de « publier un package », « créer un ensemble de modifications », « publier sur npm », « augmenter la version », « automatiser les versions » ou toute tâche impliquant la gestion des versions et la publication de packages pour les référentiels utilisant le workflow des ensembles de modifications.
