Interagieren Sie mithilfe der Browser-Automatisierung mit Slack-Arbeitsbereichen. Verwenden Sie es, wenn der Benutzer ungelesene Kanäle überprüfen, in Slack navigieren, Nachrichten senden, Daten extrahieren, Informationen finden, Konversationen durchsuchen oder Slack-Aufgaben automatisieren muss. Zu den Auslösern gehören „Mein Slack prüfen“, „Welche Kanäle haben ungelesene Inhalte“, „Eine Nachricht senden an“, „Slack durchsuchen nach“, „Aus Slack extrahieren“, „Finden, wer das gesagt hat“ oder jede andere Aufgabe, die eine programmatische Slack-Interaktion erfordert.
