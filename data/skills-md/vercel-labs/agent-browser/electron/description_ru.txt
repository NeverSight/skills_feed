Автоматизируйте настольные приложения Electron (VS Code, Slack, Discord, Figma, Notion, Spotify и т. д.) с помощью агента-браузера через протокол Chrome DevTools. Используйте, когда пользователю необходимо взаимодействовать с приложением Electron, автоматизировать настольное приложение, подключиться к работающему приложению, управлять собственным приложением или протестировать приложение Electron. Триггеры включают «автоматизировать приложение Slack», «управлять VS Code», «взаимодействовать с приложением Discord», «проверить это приложение Electron», «подключиться к настольному приложению» или любую задачу, требующую автоматизации собственного приложения Electron.
