Guida gli sviluppatori nella creazione di app MCP.
Copre l'intero ciclo di vita: brainstorming di idee rispetto alle linee guida UX, avvio di progetti, implementazione di strumenti/widget, debug, esecuzione di server di sviluppo, distribuzione e connessione di app a ChatGPT.
Utilizzare quando un utente desidera creare o aggiornare un'app MCP, un server MCP o utilizzare il framework Skybridge.
