Создавайте автономные агенты искусственного интеллекта с помощью OpenServ SDK (@openserv-labs/sdk). Используйте при создании агентов с возможностями, задачами, файловыми операциями или интеграцией. Запускается по таким ключевым словам, как OpenServ, агентский SDK, addCapability, рабочая область или многоагентные системы.
