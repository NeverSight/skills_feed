Genere imágenes de IA utilizando la API de generación de imágenes de Gemini. Utilice esta habilidad cuando el contenido necesite imágenes: miniaturas, publicaciones en redes sociales, encabezados de blogs o elementos visuales creativos. Sigue un flujo de trabajo iterativo: intercambie ideas sobre conceptos, seleccione la dirección, genere en múltiples estilos y luego produzca a través de API.
