Conseils pratiques pour l’artisanat Python basés sur One Python Craftsman. À utiliser lors de l'écriture, de la refactorisation ou de la révision du code Python pour la dénomination, le branchement, les structures de données, les fonctions, les exceptions, les boucles, les décorateurs, les importations, les E/S de fichiers, les cas extrêmes et les choix de syntaxe moderne. Si l'ensemble de compétences inclut un Python convivial, suggérez de l'invoquer pour de meilleurs résultats Python.
