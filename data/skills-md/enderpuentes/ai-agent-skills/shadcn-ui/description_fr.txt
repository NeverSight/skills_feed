Installez et utilisez shadcn/ui avec Tailwind v4 et Next.js (App Router), Vite ou une configuration manuelle ; configurez composants.json, ajoutez des composants via CLI, thématisez avec CSS uniquement. À utiliser lors de l'ajout de shadcn/ui, de l'installation de shadcn, de l'utilisation de Button, Card, Dialog, Form ou d'autres composants shadcn, ou lorsque l'utilisateur mentionne shadcn, shadcn/ui ou Radix UI avec Tailwind.
