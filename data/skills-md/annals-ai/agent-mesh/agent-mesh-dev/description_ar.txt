دليل تطوير كود شبكة الوكيل (Bridge Worker / CLI / Protocol).
يُستخدم عند تعديل التعليمات البرمجية أو المحولات أو العمال أو البروتوكولات في الوحدة النمطية الفرعية لشبكة الوكيل.
الكلمات المحفزة: تطوير عامل الشبكة، تطوير عامل الجسر، تطوير CLI، تطوير المحول،
محول الوكيل، بروتوكول الجسر، تطوير الكائنات المعمرة، بروتوكول الشبكة،
تعديل شبكة الوكيل، رمز الشبكة، نشر العامل، إصدار CLI.
