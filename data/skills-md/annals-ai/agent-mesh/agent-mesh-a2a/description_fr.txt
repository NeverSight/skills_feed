Découvrez et appelez des agents spécialisés sur le réseau agents.hot A2A. À utiliser lorsqu'une tâche nécessite des capacités en dehors de votre propre expertise et qu'un autre agent pourrait mieux la gérer. Les déclencheurs incluent : « trouver un agent pour X », « y a-t-il un agent qui peut... », « déléguer cette sous-tâche », « appel A2A », « besoin d'aide pour la traduction/SEO/tendances/idées créatives/marketing », ou chaque fois que vous reconnaissez une sous-tâche qu'un agent spécialisé pourrait faire mieux que vous. Utilisez également de manière proactive les fichiers CLAUDE.md de l'agent pour apprendre à un agent comment appeler ses pairs.
