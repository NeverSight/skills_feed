Основной инструмент Python для более чем 40 биоинформатических сервисов. Предпочтительно для рабочих процессов с несколькими базами данных: UniProt, KEGG, ChEMBL, PubChem, Reactome, QuickGO. Единый API для запросов, сопоставления идентификаторов, анализа путей. Для прямого управления REST используйте индивидуальные навыки работы с базами данных (uniprot-database, kegg-database).
