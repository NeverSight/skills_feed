Этот навык следует использовать, когда пользователь просит «добавить сервер MCP», «интегрировать MCP», «настроить MCP в плагине», «использовать .mcp.json», «настроить протокол контекста модели», «подключить внешний сервис», упоминает «${CLAUDE_PLUGIN_ROOT} с MCP» или обсуждает типы серверов MCP (SSE, stdio, HTTP, WebSocket). Содержит подробное руководство по интеграции серверов протокола контекста модели в плагины Claude Code для интеграции внешних инструментов и сервисов.
