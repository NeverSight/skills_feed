تطوير تطبيق Shopify مخصص باستخدام Shopify CLI، وبنية التطبيق، ومصادقة OAuth، وملحقات التطبيق، وواجهة المستخدم الإدارية، وأطر عمل Hydrogen/Remix، والنشر. يُستخدم عند إنشاء تطبيقات Shopify، أو إعداد Shopify CLI، أو إنشاء ملحقات التطبيق، أو تنفيذ تدفقات OAuth، أو إنشاء مكونات واجهة المستخدم الإدارية، أو العمل مع Hydrogen أو Remix، أو النشر إلى Cloudflare Workers، أو دمج خدمات الجهات الخارجية مع متاجر Shopify.
