Bibliothèque Python complète pour l'astronomie et l'astrophysique. Cette compétence doit être utilisée lorsque vous travaillez avec des données astronomiques, notamment les coordonnées célestes, les unités physiques, les fichiers FITS, les calculs cosmologiques, les systèmes temporels, les tableaux, les systèmes de coordonnées mondiales (WCS) et l'analyse des données astronomiques. À utiliser lorsque les tâches impliquent des transformations de coordonnées, des conversions d'unités, une manipulation de fichiers FITS, des calculs de distance cosmologique, des conversions d'échelle de temps ou un traitement de données astronomiques.
