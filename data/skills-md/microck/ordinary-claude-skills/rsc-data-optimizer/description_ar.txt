قم بتحسين جلب بيانات جهاز توجيه التطبيقات Next.js عن طريق تحويل جانب العميل البطيء 
الجلب إلى الجلب السريع من جانب الخادم باستخدام React Server Components (RSC).

استخدم عندما:
- أبلغ المستخدم عن بطء التحميل الأولي للصفحة أثناء تحميل المغازل
- تستخدم الصفحة useEffect + useState لجلب البيانات
- يؤدي نمط StoreContext/useStore إلى جلب الشلال
- الحاجة إلى تحسين مُحسنات محركات البحث (المحتوى ليس بتنسيق HTML الأولي)
- تحويل صفحات "استخدام العميل" إلى مكونات الخادم
