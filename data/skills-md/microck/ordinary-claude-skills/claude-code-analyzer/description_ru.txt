Анализирует шаблоны использования Claude Code и предоставляет исчерпывающие рекомендации. Выполняет анализ использования, обнаруживает ресурсы сообщества GitHub, предлагает улучшения CLAUDE.md и получает новейшие документы по запросу. Используйте, когда пользователь хочет оптимизировать рабочий процесс Claude Code, создать конфигурации (агенты/навыки/команды) или настроить проектную документацию.
