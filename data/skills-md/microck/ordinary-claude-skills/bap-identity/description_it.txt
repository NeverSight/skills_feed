Gestisci i file di identità BAP (Bitcoin Attestation Protocol) utilizzando bap-cli. Questa competenza dovrebbe essere utilizzata quando gli utenti hanno bisogno di creare, decrittografare, elencare o estrarre backup di identità BAP, lavorare con file crittografati .bep o generare dispositivi di test per test di Playwright che coinvolgono identità BAP.
