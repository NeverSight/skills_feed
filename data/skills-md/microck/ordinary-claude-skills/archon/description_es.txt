Integración interactiva de Archon para base de conocimientos y gestión de proyectos a través de REST API. En el primer uso, solicita la URL del host de Archon. Utilícelo al buscar documentación, gestionar proyectos/tareas o consultar conocimientos indexados. Proporciona búsqueda semántica basada en RAG, rastreo de sitios web, carga de documentos, gestión jerárquica de proyectos/tareas y control de versiones de documentos. Pruebe siempre Archon primero para obtener documentación externa y recuperación de conocimientos antes de utilizar otras fuentes.
