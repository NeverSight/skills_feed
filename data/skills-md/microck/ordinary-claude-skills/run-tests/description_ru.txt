Проверяйте изменения кода, разумно выбирая и запуская соответствующие наборы тестов. Используйте это при редактировании кода, чтобы проверить правильность работы изменений, запустить тесты, проверить функциональность или проверить наличие регрессий. Автоматически обнаруживает затронутые наборы тестов, выбирает минимальный набор venv, необходимый для проверки, и при необходимости обрабатывает выполнение тестов с помощью служб Docker.
