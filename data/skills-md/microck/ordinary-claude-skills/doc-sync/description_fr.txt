Maintient la documentation IdeaVim synchronisée avec les modifications de code. Utilisez cette compétence lorsque vous devez vérifier l'exactitude de la documentation après des modifications de code, ou lorsque vous vérifiez si la documentation (dans doc/, README.md, CONTRIBUTING.md) correspond à la base de code actuelle. La compétence peut fonctionner de manière bidirectionnelle : de la documentation à la vérification du code, ou des modifications du code aux mises à jour de la documentation.
