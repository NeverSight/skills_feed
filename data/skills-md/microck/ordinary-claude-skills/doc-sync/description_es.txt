Mantiene la documentación de IdeaVim sincronizada con los cambios de código. Utilice esta habilidad cuando necesite verificar la exactitud de la documentación después de cambios de código, o cuando verifique si la documentación (en doc/, README.md, CONTRIBUTING.md) coincide con el código base actual. La habilidad puede funcionar de forma bidireccional: desde documentos hasta verificación de código, o desde cambios de código hasta actualizaciones de documentación.
