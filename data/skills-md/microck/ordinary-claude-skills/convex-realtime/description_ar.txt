قم بتنفيذ ميزات الوقت الفعلي باستخدام الاستعلامات التفاعلية المحدبة التي يتم تحديثها تلقائيًا عند تغيير البيانات، مما يتيح التعاون المباشر والتحديثات الفورية وواجهات المستخدم التفاعلية دون الاستقصاء اليدوي. يُستخدم عند إنشاء لوحات معلومات مباشرة، أو تنفيذ التحرير التعاوني، أو إنشاء تطبيقات الدردشة، أو عرض الإشعارات في الوقت الفعلي، أو إنشاء خلاصات الأنشطة، أو تنفيذ مؤشرات التواجد، أو إنشاء بحث تفاعلي، أو أي ميزة تتطلب مزامنة فورية للبيانات عبر العملاء.
