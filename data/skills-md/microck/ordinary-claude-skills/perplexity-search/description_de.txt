Führen Sie KI-gestützte Websuchen mit Echtzeitinformationen mithilfe von Perplexity-Modellen über LiteLLM und OpenRouter durch. Diese Fähigkeit sollte verwendet werden, wenn Sie im Internet nach aktuellen Informationen suchen, aktuelle wissenschaftliche Literatur finden, fundierte Antworten mit Quellenangaben erhalten oder auf Informationen zugreifen, die über den Wissensgrenzwert des Modells hinausgehen. Bietet Zugriff auf mehrere Perplexity-Modelle, einschließlich Sonar Pro, Sonar Pro Search (erweiterte Agentensuche) und Sonar Reasoning Pro über einen einzigen OpenRouter-API-Schlüssel.
