LiteLLM と OpenRouter 経由で Perplexity モデルを使用し、リアルタイム情報を使用して AI を活用した Web 検索を実行します。このスキルは、最新情報の Web 検索を実行する場合、最近の科学文献を検索する場合、出典の引用を含む根拠のある回答を得る場合、またはモデルの知識限界を超えた情報にアクセスする場合に使用する必要があります。単一の OpenRouter API キーを介して、Sonar Pro、Sonar Pro Search (高度なエージェント検索)、Sonar Reasoning Pro を含む複数の Perplexity モデルへのアクセスを提供します。
