Основной набор инструментов Python для молекулярной биологии. Предпочтительно для запросов PubMed/NCBI на основе Python (Bio.Entrez), манипуляций с последовательностями, анализа файлов (FASTA, GenBank, FASTQ, PDB), расширенных рабочих процессов BLAST, структур и филогенетики. Для быстрого BLAST используйте gget. Для прямого REST API используйте pubmed-database.
