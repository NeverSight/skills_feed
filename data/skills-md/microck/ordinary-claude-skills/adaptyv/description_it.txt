Piattaforma di laboratorio cloud per analisi e validazione automatizzate delle proteine. Utilizzare quando si progettano proteine ​​e si necessita di validazione sperimentale, inclusi test di legame, test di espressione, misurazioni di termostabilità, test di attività enzimatica o ottimizzazione della sequenza proteica. Utilizzabile anche per inviare esperimenti tramite API, monitorare lo stato degli esperimenti, scaricare risultati, ottimizzare sequenze proteiche per una migliore espressione utilizzando strumenti computazionali (NetSolP, SoluProt, SolubleMPNN, ESM) o gestire flussi di lavoro di progettazione di proteine ​​con convalida wet-lab.
