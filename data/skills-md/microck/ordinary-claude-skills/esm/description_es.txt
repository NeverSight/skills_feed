Conjunto de herramientas completo para modelos de lenguaje de proteínas, incluido ESM3 (diseño generativo de proteínas multimodal en secuencia, estructura y función) y ESM C (incrustaciones y representaciones de proteínas eficientes). Utilice esta habilidad cuando trabaje con secuencias de proteínas, estructuras o predicción de funciones; diseñar nuevas proteínas; generar incorporaciones de proteínas; realizar plegado inverso; o realizar tareas de ingeniería de proteínas. Admite el uso de modelos locales y la API Forge basada en la nube para una inferencia escalable.
