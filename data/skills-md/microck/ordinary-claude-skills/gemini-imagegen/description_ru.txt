Создавайте и редактируйте изображения с помощью Gemini API (Nano Banana Pro). Используйте этот навык при создании изображений из текстовых подсказок, редактировании существующих изображений, применении переноса стилей, создании логотипов с текстом, создании наклеек, макетов продуктов или любых задачах по созданию/манипулированию изображениями. Поддерживает преобразование текста в изображение, редактирование изображений, многоэтапное уточнение и композицию из нескольких эталонных изображений.
