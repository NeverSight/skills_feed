Интеграция GitHub Spec-Kit для разработки на основе спецификаций. 7-этапный рабочий процесс (составление, уточнение, уточнение, планирование, задачи, анализ, реализация). Используйте при работе с интерфейсом командной строки spec-kit, каталогами .specify/ или при создании спецификаций с разработкой на основе конституции. Запускается «spec-kit», «speckit», «constitution», «specify», ссылками на каталог .specify/ или командами spec-kit.
