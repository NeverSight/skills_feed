قم بإنشاء وإدارة مهارات Claude Code في مستودع HASH باتباع أفضل الممارسات الإنسانية. يُستخدم عند إنشاء مهارات جديدة، أو تعديل Skill-rules.json، أو فهم أنماط التشغيل، أو العمل باستخدام الخطافات، أو تصحيح أخطاء تنشيط المهارات، أو تنفيذ الكشف التدريجي. يغطي بنية المهارات، والمادة الأمامية YAML، وأنواع المشغلات (الكلمات الرئيسية، وأنماط النوايا)، وخطاف UserPromptSubmit، وقاعدة الـ 500 سطر. يتضمن التحقق من الصحة وتصحيح الأخطاء باستخدام SKILL_DEBUG. تشمل الأمثلة كومة أخطاء الصدأ، وتبعيات البضائع، ومهارات توثيق الصدأ.
