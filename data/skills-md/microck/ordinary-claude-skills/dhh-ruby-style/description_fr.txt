Écrivez du code Ruby et Rails dans le style distinctif 37signaux de DHH. Utilisez cette compétence lors de l'écriture de code Ruby, d'applications Rails, de la création de modèles, de contrôleurs ou de tout fichier Ruby. Déclencheurs lors de la génération de code Ruby/Rails, des demandes de refactoring, de la révision du code ou lorsque l'utilisateur mentionne le style DHH, 37signals, Basecamp, HEY ou Campfire. Incarne la pureté REST, les gros modèles, les contrôleurs minces, les attributs actuels, les modèles Hotwire et la philosophie « la clarté plutôt que l'intelligence ».
