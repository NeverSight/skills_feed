Créez des applications React Native 0.76+ avec Expo SDK 52. Couvre la nouvelle architecture obligatoire (0.82+), les modifications de React 19 (suppression de propTypes/forwardRef), le nouveau CSS (affichage : contenu, mixBlendMode, contour), le modèle Swift iOS et la migration de DevTools.

À utiliser lors de : la création d'applications Expo, la migration vers une nouvelle architecture ou le dépannage de "Composant Fabric introuvable", "propTypes pas une fonction", "TurboModule non enregistré" ou d'erreurs Swift AppDelegate.
