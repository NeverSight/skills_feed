Defina modelos de bases de datos con nombres claros, tipos de datos apropiados, restricciones, relaciones y validación en múltiples capas. Utilice esta habilidad al crear o modificar archivos de modelos de bases de datos, clases ORM, definiciones de esquemas o relaciones de modelos de datos. Se aplica cuando se trabaja con archivos de modelo (por ejemplo, models.py, models/, clases ActiveRecord, esquema Prisma, modelos Sequelize), se definen estructuras de tablas, se configuran relaciones y claves externas, se configuran comportamientos en cascada, se implementan validaciones de modelos, se agregan marcas de tiempo o se trabaja con restricciones de bases de datos (NO NULAS, ÚNICAS, claves externas). Úselo para cualquier tarea que implique el cumplimiento de la integridad de los datos, definiciones de relaciones o validación de datos a nivel de modelo.
