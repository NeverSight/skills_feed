Definisci modelli di database con denominazioni chiare, tipi di dati, vincoli, relazioni e convalida appropriati a più livelli. Utilizzare questa competenza durante la creazione o la modifica di file di modello di database, classi ORM, definizioni di schema o relazioni di modelli di dati. Applicare quando si lavora con file di modello (ad esempio, models.py, models/, classi ActiveRecord, schema Prisma, modelli Sequelize), definendo strutture di tabelle, impostando chiavi esterne e relazioni, configurando comportamenti a cascata, implementando convalide di modelli, aggiungendo timestamp o lavorando con vincoli di database (NON NULL, UNIQUE, chiavi esterne). Utilizzare per qualsiasi attività che coinvolga l'applicazione dell'integrità dei dati, le definizioni delle relazioni o la convalida dei dati a livello di modello.
