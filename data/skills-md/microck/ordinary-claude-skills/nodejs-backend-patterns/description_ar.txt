أنشئ خدمات Node.js الخلفية جاهزة للإنتاج باستخدام Express/Fastify، وتنفيذ أنماط البرامج الوسيطة، ومعالجة الأخطاء، والمصادقة، وتكامل قاعدة البيانات، وأفضل ممارسات تصميم واجهة برمجة التطبيقات. يُستخدم عند إنشاء خوادم Node.js، أو واجهات برمجة تطبيقات REST، أو واجهات GraphQL الخلفية، أو بنيات الخدمات الصغيرة.
