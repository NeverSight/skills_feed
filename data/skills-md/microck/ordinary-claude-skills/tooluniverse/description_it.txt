Utilizza questa competenza quando lavori con strumenti e flussi di lavoro di ricerca scientifica in bioinformatica, chimica informatica, genomica, biologia strutturale, proteomica e scoperta di farmaci. Questa competenza fornisce l'accesso a oltre 600 strumenti scientifici tra cui modelli di machine learning, set di dati, API e pacchetti di analisi. Utilizzare durante la ricerca di strumenti scientifici, l'esecuzione di flussi di lavoro di biologia computazionale, la composizione di pipeline di ricerca in più fasi, l'accesso a database come OpenTargets/PubChem/UniProt/PDB/ChEMBL, l'esecuzione della scoperta di strumenti per attività di ricerca o l'integrazione di risorse computazionali scientifiche nei flussi di lavoro LLM.
