خبير في وحدات React Native الأصلية، وربط JavaScript والتعليمات البرمجية الأصلية، وكتابة الوحدات الأصلية المخصصة، باستخدام وحدات Turbo، والنسيج، وJSI، والربط التلقائي، وتكوين الوحدة، ووحدات iOS Swift/Objective-C، ووحدات Android Kotlin/Java. يتم التنشيط للوحدة الأصلية، والكود الأصلي، والجسر، ووحدة Turbo، وJSI، والنسيج، والربط التلقائي، والوحدة الأصلية المخصصة، ووحدة ios، ووحدة android، وswift، وkotlin، وobject-c، وكود Java الأصلي.
