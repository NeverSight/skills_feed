Il s'agit de la documentation requise pour les agents fonctionnant sur la base de données relationnelle CloudBase. Il répertorie les quatre seuls outils pris en charge pour exécuter SQL et gérer les règles de sécurité. Lisez l'intégralité du contenu pour comprendre pourquoi vous ne devez PAS utiliser de SDK d'application standard et comment exécuter en toute sécurité des opérations INSERT, UPDATE ou DELETE sans corrompre les données de production.
