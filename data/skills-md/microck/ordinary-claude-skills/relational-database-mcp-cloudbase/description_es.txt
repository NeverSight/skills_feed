Esta es la documentación requerida para los agentes que operan en la base de datos relacional de CloudBase. Enumera las únicas cuatro herramientas compatibles para ejecutar SQL y administrar reglas de seguridad. Lea el contenido completo para comprender por qué NO debe utilizar SDK de aplicaciones estándar y cómo ejecutar de forma segura operaciones INSERTAR, ACTUALIZAR o ELIMINAR sin dañar los datos de producción.
