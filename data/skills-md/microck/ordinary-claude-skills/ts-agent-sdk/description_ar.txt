قم بإنشاء حزم TypeScript SDK المكتوبة لوكلاء الذكاء الاصطناعي للتفاعل مع خوادم MCP. يحول أوامر تجعيد JSON-RPC المطولة إلى استدعاءات الوظائف النظيفة (docs.createDocument() vs cur). يكتشف أدوات MCP تلقائيًا من وحدات الخادم، وينشئ أنواع TypeScript وأساليب العميل، وينشئ نماذج نصية قابلة للتشغيل.

يُستخدم عندما: إنشاء تطبيقات تدعم MCP، أو تحتاج إلى وصول برمجي مكتوب إلى أدوات MCP، أو تريد أن يقوم Claude Code بإدارة التطبيقات عبر البرامج النصية، أو التخلص من أوامر تجعيد JSON-RPC اليدوية، أو التحقق من صحة مدخلات/مخرجات MCP، أو إنشاء أتمتة وكيل قابلة لإعادة الاستخدام.
