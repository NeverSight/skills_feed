AI エージェントが MCP サーバーと対話するための型付き TypeScript SDK を生成します。冗長な JSON-RPCcurl コマンドをクリーンな関数呼び出しに変換します (docs.createDocument() とcurl)。サーバー モジュールから MCP ツールを自動検出し、TypeScript タイプとクライアント メソッドを生成し、実行可能なサンプル スクリプトを作成します。

次の場合に使用します: MCP 対応アプリケーションを構築する場合、MCP ツールへの型指定されたプログラムによるアクセスが必要な場合、Claude Code でスクリプト経由でアプリを管理する場合、手動の JSON-RPCcurl コマンドを排除する場合、MCP 入出力を検証する場合、または再利用可能なエージェント自動化を作成する場合。
