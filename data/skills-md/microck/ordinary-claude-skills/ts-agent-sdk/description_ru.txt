Создавайте типизированные SDK TypeScript для агентов ИИ для взаимодействия с серверами MCP. Преобразует подробные команды Curl JSON-RPC в чистые вызовы функций (docs.createDocument() вместо Curl). Автоматически обнаруживает инструменты MCP из серверных модулей, генерирует типы TypeScript и клиентские методы, создает работоспособные примеры сценариев.

Используйте, когда: создание приложений с поддержкой MCP, требуется типизированный программный доступ к инструментам MCP, требуется, чтобы Claude Code управлял приложениями с помощью сценариев, устраняя ручные команды скручивания JSON-RPC, проверяя входные/выходные данные MCP или создавая многоразовую автоматизацию агента.
