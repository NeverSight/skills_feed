AI 에이전트가 MCP 서버와 상호 작용할 수 있도록 형식화된 TypeScript SDK를 생성합니다. 자세한 JSON-RPC 컬 명령을 정리 함수 호출(docs.createDocument() 대 컬)로 변환합니다. 서버 모듈에서 MCP 도구를 자동 감지하고, TypeScript 유형 및 클라이언트 메서드를 생성하고, 실행 가능한 예제 스크립트를 생성합니다.

사용 시기: MCP 지원 애플리케이션 구축, MCP 도구에 대한 입력된 프로그래밍 방식 액세스 필요, Claude Code가 스크립트를 통해 앱을 관리하기를 원함, 수동 JSON-RPC 컬 명령 제거, MCP 입력/출력 유효성 검사 또는 재사용 가능한 에이전트 자동화 생성.
