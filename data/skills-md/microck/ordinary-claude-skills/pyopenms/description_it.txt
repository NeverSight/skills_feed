Interfaccia Python per OpenMS per l'analisi dei dati di spettrometria di massa. Utilizzare per flussi di lavoro di proteomica e metabolomica LC-MS/MS, inclusa la gestione dei file (mzML, mzXML, mzTab, FASTA, pepXML, protXML, mzIdentML), elaborazione del segnale, rilevamento delle caratteristiche, identificazione dei peptidi e analisi quantitativa. Applicabile quando si lavora con dati di spettrometria di massa, si analizzano esperimenti di proteomica o si elaborano set di dati di metabolomica.
