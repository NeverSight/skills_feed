Base de datos integrada de código abierto para aplicaciones de IA. Almacene incrustaciones y metadatos, realice búsquedas vectoriales y de texto completo, filtre por metadatos. API simple de 4 funciones. Escala desde portátiles hasta grupos de producción. Úselo para búsqueda semántica, aplicaciones RAG o recuperación de documentos. Lo mejor para el desarrollo local y proyectos de código abierto.
