Набор инструментов для автоматизации лабораторий для управления устройствами для работы с жидкостями, планшетными считывателями, насосами, нагревательными шейкерами, инкубаторами, центрифугами и аналитическим оборудованием. Используйте этот навык при автоматизации лабораторных рабочих процессов, программировании роботов для работы с жидкостями (Hamilton STAR, Opentrons OT-2, Tecan EVO), интеграции лабораторного оборудования, управлении планировкой и ресурсами деки (планшеты, наконечники, контейнеры), считывании планшетов или создании воспроизводимых лабораторных протоколов. Применимо как для моделируемых протоколов, так и для физического управления оборудованием.
