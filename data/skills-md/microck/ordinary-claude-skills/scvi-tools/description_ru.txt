Этот навык следует использовать при работе с анализом данных омики отдельных клеток с использованием инструментов scvi, включая scRNA-seq, scATAC-seq, CITE-seq, пространственную транскриптомику и другие методы одноклеточных исследований. Используйте этот навык для вероятностного моделирования, пакетной коррекции, уменьшения размерности, дифференциального выражения, аннотации типов ячеек, мультимодальной интеграции и задач пространственного анализа.
