تخطيط وتنفيذ عمليات ترحيل قاعدة البيانات، وتحويلات البيانات، وعمليات ترحيل النظام بأمان باستخدام إستراتيجيات التراجع والتحقق من صحة تكامل البيانات. يُستخدم عند ترحيل قواعد البيانات، أو تحويل مخططات البيانات، أو التنقل بين أنظمة قواعد البيانات، أو تنفيذ عمليات ترحيل الإصدارات، أو التعامل مع تحويلات البيانات، أو ضمان تكامل البيانات، أو التخطيط لعمليات ترحيل بدون توقف.
