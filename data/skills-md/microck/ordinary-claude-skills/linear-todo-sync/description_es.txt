Esta habilidad recupera las tareas abiertas asignadas al usuario desde Linear API y genera un archivo de lista de tareas pendientes de rebajas en la raíz del proyecto. Esta habilidad debe usarse cuando el usuario pregunta sobre sus elementos de trabajo, quiere ver en qué necesita trabajar o solicita cargar/sincronizar sus tareas lineales. Requiere paquetes Python 3.7+, solicitudes, mdutils y python-dotenv. Requiere LINEAR_API_KEY en el archivo .env.
