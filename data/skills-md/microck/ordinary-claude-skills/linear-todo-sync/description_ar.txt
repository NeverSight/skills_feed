تجلب هذه المهارة المهام المفتوحة المعينة للمستخدم من Linear API وتقوم بإنشاء ملف قائمة مهام تخفيض السعر في جذر المشروع. يجب استخدام هذه المهارة عندما يسأل المستخدم عن عناصر عمله، أو يريد رؤية ما يحتاج إلى العمل عليه، أو يطلب تحميل/مزامنة مهامه الخطية. يتطلب Python 3.7+ والطلبات وmdutils وحزم python-dotenv. يتطلب LINEAR_API_KEY في ملف .env.
