Этот навык извлекает открытые задачи, назначенные пользователю, из Linear API, и создает файл списка дел с уценкой в ​​корне проекта. Этот навык следует использовать, когда пользователь спрашивает о своих рабочих элементах, хочет узнать, над чем ему нужно работать, или когда просит загрузить/синхронизировать свои линейные задачи. Требуются пакеты Python 3.7+, запросы, mdutils и python-dotenv. Требуется LINEAR_API_KEY в файле .env.
