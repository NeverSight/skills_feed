دليل تكامل API الكامل لـ Shopify بما في ذلك GraphQL Admin API وREST Admin API وStorefront API وAjax API ومصادقة OAuth وتحديد المعدل وخطافات الويب. يُستخدم عند إجراء مكالمات API إلى Shopify، أو مصادقة التطبيقات، أو جلب بيانات المنتج/الطلب/العميل برمجيًا، أو تنفيذ عمليات سلة التسوق، أو التعامل مع خطافات الويب، أو العمل مع إصدار API 2025-10. يتطلب جلب أو axios لتطبيقات JavaScript.
