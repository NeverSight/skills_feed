Langfuse の Next.js 14/tRPC/Express/TypeScript モノリポジトリの包括的なバックエンド開発ガイド。 tRPC ルーター、パブリック API エンドポイント、BullMQ キュー プロセッサ、サービスを作成する場合、または tRPC プロシージャ、Next.js API ルート、Prisma データベース アクセス、ClickHouse 分析クエリ、Redis キュー、OpenTelemetry インストルメンテーション、Zod v4 検証、env.mjs 構成、テナント分離パターン、または非同期パターンを操作する場合に使用します。階層化アーキテクチャ (tRPC プロシージャ → サービス、キュー プロセッサ → サービス)、デュアル データベース システム (PostgreSQL + ClickHouse)、マルチテナント分離のための projectId フィルタリング、traceException エラー処理、可観測性パターン、およびテスト戦略 (Web 用の Jest、ワーカー用の vitest) をカバーします。
