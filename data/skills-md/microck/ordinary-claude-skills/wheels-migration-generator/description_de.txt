Generieren Sie datenbankunabhängige Wheels-Migrationen zum Erstellen von Tabellen, zum Ändern von Schemata und zum Verwalten von Datenbankänderungen. Wird zum Erstellen oder Ändern eines Datenbankschemas sowie zum Hinzufügen von Tabellen, Spalten, Indizes oder Fremdschlüsseln verwendet. Verhindert datenbankspezifisches SQL und gewährleistet datenbankübergreifende Kompatibilität.
