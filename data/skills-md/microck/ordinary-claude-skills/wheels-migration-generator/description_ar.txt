قم بإنشاء عمليات ترحيل Wheels غير المعتمدة على قاعدة البيانات لإنشاء الجداول وتغيير المخططات وإدارة تغييرات قاعدة البيانات. يُستخدم عند إنشاء مخطط قاعدة البيانات أو تعديله، أو إضافة الجداول، أو الأعمدة، أو الفهارس، أو المفاتيح الخارجية. يمنع SQL الخاصة بقاعدة البيانات ويضمن التوافق بين قواعد البيانات.
