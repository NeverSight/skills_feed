Basandosi sull'esperienza di implementazione di successo di Hyperliquid, questa guida fornisce l'integrazione del framework Yuan per i nuovi fornitori di scambi. Utilizza questa competenza durante la creazione di implementazioni dei fornitori per nuovi scambi, inclusa la progettazione della struttura del progetto, l'integrazione delle API, l'implementazione dei servizi principali e le migliori pratiche. Applicabile all'integrazione delle API di scambio, allo sviluppo del sistema finanziario e alla progettazione dell'architettura dei microservizi.
