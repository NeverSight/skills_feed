Biblioteca interactiva de visualización de datos científicos y estadísticos para Python. Úselo al crear gráficos, diagramas o visualizaciones, incluidos diagramas de dispersión, gráficos de líneas, gráficos de barras, mapas de calor, diagramas 3D, mapas geográficos, distribuciones estadísticas, gráficos financieros y paneles. Admite visualizaciones rápidas (Plotly Express) y personalización detallada (objetos gráficos). Genera HTML interactivo o imágenes estáticas (PNG, PDF, SVG).
