Ejercicios de recuperación de desastres y listas de verificación de seguridad para proyectos de aplicaciones web (SPA, SSR, aplicaciones web full-stack). Enfocado a desarrolladores independientes/independientes que utilizan infraestructura de nivel gratuito (Vercel, Supabase, Cloudflare, Netlify, Railway, etc.). Une las mejores prácticas de las grandes tecnologías (NIST, Google SRE DiRT, ISO 22301) a la escala independiente. Úselo cuando el usuario mencione simulacros, recuperación ante desastres, auditoría de seguridad, simulación de incidentes, verificación del estado del proyecto, pruebas de resiliencia, estrategias de respaldo, rotación de secretos o respuesta a incidentes para proyectos web. No para aplicaciones móviles, software de escritorio, herramientas CLI o juegos.
