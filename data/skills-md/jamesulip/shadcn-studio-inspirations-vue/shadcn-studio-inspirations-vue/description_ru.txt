Огромный каталог, содержащий более 600 идей для компонентов Vue SFC, переведенных из кодовой базы React shadcn-studio. Используйте этот навык, когда пользователь просит вдохновения для дизайна, создания полностраничных макетов, сложных вложенных компонентов или вам нужно увидеть расширенные примеры того, как скомпоновать компоненты shadcn-vue вместе.
