يوجه التطوير باستخدام supastarter لـ Next.js فقط (وليس Vue/Nuxt): مكدس التكنولوجيا، الإعداد، التكوين، قاعدة البيانات (Prisma)، API (Hono/oRPC)، المصادقة (Better Auth)، المؤسسات، المدفوعات (Stripe)، الذكاء الاصطناعي، التخصيص، التخزين، البريد، i18n، SEO، النشر، المهام الخلفية، التحليلات، المراقبة، E2E. يُستخدم عند إنشاء تطبيقات supastarter Next.js أو تعديلها، أو إضافة ميزات، أو عندما يذكر المستخدم supastarter Next.js، أو Prisma، أو oRPC، أو Better Auth، أو موضوعات مكدس Next.js ذات الصلة.
