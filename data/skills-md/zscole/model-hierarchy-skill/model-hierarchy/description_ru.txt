Оптимизируйте затраты на операции агентов ИИ, перенаправляя задачи на соответствующие модели в зависимости от сложности. Используйте этот навык, когда: (1) решаете, какую модель использовать для задачи, (2) создаете субагентов, (3) рассматриваете экономическую эффективность, (4) текущая модель кажется излишней для этой задачи. Триггеры: «маршрутизация модели», «оптимизация затрат», «какая модель», «слишком дорого», «породить агент».
