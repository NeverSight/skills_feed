Optimieren Sie die Kosten für den Betrieb von KI-Agenten, indem Sie Aufgaben je nach Komplexität an geeignete Modelle weiterleiten. Verwenden Sie diese Fähigkeit, wenn: (1) Sie entscheiden, welches Modell für eine Aufgabe verwendet werden soll, (2) Unteragenten erzeugen, (3) Kosteneffizienz berücksichtigen, (4) das aktuelle Modell für die Aufgabe übertrieben erscheint. Auslöser: „Modellrouting“, „Kostenoptimierung“, „Welches Modell“, „zu teuer“, „Spawn-Agent“.
