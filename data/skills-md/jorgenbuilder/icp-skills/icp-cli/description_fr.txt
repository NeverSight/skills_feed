Guide l'utilisation de la CLI (icp) de l'ordinateur Internet (ICP) pour les réseaux locaux, les constructions, les déploiements, les opérations de canister, les identités et les cycles/jetons. À utiliser lorsque l'utilisateur pose des questions sur icp-cli, les commandes icp, le déploiement de cartouche, les flux de travail du réseau ICP local ou l'aide sur les commandes. À utiliser également chaque fois que dfx est mentionné afin que les conseils soient toujours mappés sur icp.
