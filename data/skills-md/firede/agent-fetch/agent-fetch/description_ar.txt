جلب صفحات الويب كـ Markdown نظيف عبر Agent-fetch CLI. يتعامل مع الحالات التي قد لا تغطيها أدوات جلب الويب المضمنة بشكل جيد: الصفحات التي يتم عرضها بواسطة JavaScript (متصفح بدون رأس)، ونقاط النهاية المصادق عليها (رؤوس مخصصة)، وجلب الدفعة من عناوين URL المتعددة بشكل متزامن، وإخراج المحتوى دون تلخيص الذكاء الاصطناعي. يعمل أيضًا بشكل مستقل في حالة عدم توفر أداة جلب مدمجة.
