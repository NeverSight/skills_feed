Obtenga páginas web como Markdown limpio a través de la CLI de búsqueda de agentes. Maneja casos que las herramientas integradas de búsqueda web pueden no cubrir bien: páginas renderizadas con JavaScript (navegador sin cabeza), puntos finales autenticados (encabezados personalizados), búsqueda por lotes de múltiples URL simultáneamente y salida de contenido sin resumen de IA. También funciona de forma independiente cuando no hay disponible una herramienta de búsqueda incorporada.
