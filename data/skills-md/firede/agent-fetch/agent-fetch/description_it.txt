Recupera le pagine Web come Markdown pulito tramite la CLI di recupero dell'agente. Gestisce i casi che gli strumenti di recupero web integrati potrebbero non coprire bene: pagine con rendering JavaScript (browser headless), endpoint autenticati (intestazioni personalizzate), recupero in batch di più URL contemporaneamente e output di contenuto senza riepilogo AI. Funziona anche in modalità autonoma quando non è disponibile alcuno strumento di recupero integrato.
