Получайте веб-страницы в чистом виде Markdown через интерфейс командной строки агента. Обрабатывает случаи, которые встроенные инструменты веб-извлечения могут плохо обрабатывать: страницы, отображаемые на JavaScript (безголовый браузер), аутентифицированные конечные точки (настраиваемые заголовки), одновременная пакетная выборка нескольких URL-адресов и вывод контента без суммирования AI. Также работает автономно, когда встроенный инструмент выборки недоступен.
