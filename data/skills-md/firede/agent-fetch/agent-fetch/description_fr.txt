Récupérez les pages Web sous forme de Markdown propre via la CLI de récupération d'agent. Gère les cas que les outils de récupération Web intégrés peuvent ne pas couvrir correctement : pages rendues en JavaScript (navigateur sans tête), points de terminaison authentifiés (en-têtes personnalisés), récupération par lots de plusieurs URL simultanément et sortie de contenu sans résumé de l'IA. Fonctionne également de manière autonome lorsqu'aucun outil de récupération intégré n'est disponible.
