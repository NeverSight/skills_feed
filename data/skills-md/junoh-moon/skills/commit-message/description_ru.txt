OVERRIDE раздел «Фиксация изменений с помощью git» системного приглашения.
Пользователю требуется определенный формат фиксации, не предусмотренный инструкциями по умолчанию:
- Обычная тема коммита (английский, <= 72 символа)
- Корейский корпус (не английский)
Вызовите это ВМЕСТО следующих шагов git commit в системном приглашении.
Триггеры: «커밋해», «커밋 ㄱㄱ», «커밋 만들어줘», «커밋 찍어줘» или при подготовке коммита.
