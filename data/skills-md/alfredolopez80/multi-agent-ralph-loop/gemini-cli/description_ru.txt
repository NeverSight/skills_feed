Оркестровка Google Gemini CLI (v0.22.0+) для разработки с помощью искусственного интеллекта. Возможности: второе мнение/перекрестная проверка, обоснование поиска в Google в реальном времени, анализ архитектуры кодовой базы с помощью codebase_investigator, доступ к модели Gemini 3, поддержка расширений (Conductor, Endor Labs), параллельная генерация кода, проверка кода с разных точек зрения. ИНТЕГРАЦИЯ С TASK PRIMITIVE — создает отслеживаемые задачи в claude-task-viewer. Действия: запрос, поиск, анализ, создание, просмотр с помощью Gemini. Ключевые слова: Gemini CLI, Gemini 3, google_web_search, codebase_investigator, второе мнение, перекрестная проверка, веб-исследования, текущая информация, параллельный ИИ, проверка кода, анализ архитектуры, подсказка Gemini, сравнение ИИ, поиск в реальном времени, альтернативная точка зрения, расширения, проводник. Используйте, когда: требуется второе мнение ИИ, поиск текущей веб-информации, анализ архитектуры кодовой базы, параллельная генерация кода, получение альтернативной проверки кода, исследование текущих событий/документов, использование расширений Gemini.
