Interaction et modèles UX des directives Apple Human Interface. Utilisez cette compétence lorsque l'utilisateur pose des questions sur le « flux d'intégration », « l'intégration de l'utilisateur », « lancement de l'application », « état de chargement », « glisser-déposer », « modèle de recherche », « conception des paramètres », « notifications », « modalité », « multitâche », « modèle de retour », « haptique », « annuler le rétablissement », « gestion de fichiers », saisie de données, partage, collaboration, plein écran, audio, vidéo, retour haptique, évaluations, impression, aide ou gestion de compte dans les applications Apple. À utiliser également lorsque l'utilisateur dit "comment l'intégration devrait-elle fonctionner", "mon application prend trop de temps à charger", "dois-je utiliser un modal ici", "comment gérer les erreurs", "quand dois-je demander des autorisations", "comment afficher la progression" ou "quelle est la bonne façon de confirmer une suppression". Références croisées : bases élevées pour les principes sous-jacents, plates-formes élevées pour les spécificités de la plate-forme, mise en page à composants élevés pour la navigation, contenu à composants élevés pour l'affichage des données.
