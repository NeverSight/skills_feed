Основы проектирования Рекомендаций Apple Human Interface Guidelines. Используйте этот навык, когда пользователь спрашивает о «цвете HIG», «типографике Apple», «символах научной фантастики», «рекомендациях по темному режиму», «доступном дизайне», «основах дизайна Apple», «значке приложения», «рекомендациях по макету», «материалах», «движении», «конфиденциальности», «справа налево», «RTL», «инклюзивном дизайне», брендинге, изображениях, пространственной компоновке или стиле письма. Также используйте, когда пользователь говорит: «Мои цвета выглядят неправильно в темном режиме», «Какой шрифт мне следует использовать», «Достаточно ли доступно мое приложение», «Как мне поддерживать динамический тип», «Какой коэффициент контрастности мне нужен», «Как мне выбрать системные цвета» или «Мои значки не соответствуют стилю системы». Перекрестные ссылки: hig-platforms для руководства по конкретной платформе, hig-patterns для шаблонов взаимодействия, hig-Components-Layout для структурных компонентов, hig-Components-Content для отображения.
