Directives Apple Human Interface pour la conception spécifique à la plate-forme. Utilisez cette compétence lorsque l'utilisateur pose des questions sur « conception pour iOS », « conception d'applications iPad », « conception macOS », « tvOS », « visionOS », « watchOS », « plate-forme Apple », « quelle plate-forme », les différences de plate-forme, les conventions spécifiques à la plate-forme ou la conception d'applications multiplateformes. À utiliser également lorsque l'utilisateur demande « dois-je concevoir différemment pour iPad et iPhone », « comment mon application fonctionne-t-elle sur visionOS », « qu'est-ce qui est différent avec les applications macOS », « porter mon application sur une autre plate-forme », « conception d'application universelle » ou « quelles méthodes de saisie cette plate-forme utilise-t-elle ». Références croisées : fondations élevées pour les fondations de conception partagées, modèles élevés pour les modèles d'interaction, mise en page à composants élevés pour les structures de navigation, contenu à composants élevés pour l'affichage du contenu.
