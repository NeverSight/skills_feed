Рекомендации Apple по пользовательскому интерфейсу для проектирования с учетом особенностей платформы. Используйте этот навык, когда пользователь спрашивает о «проектировании для iOS», «дизайне приложений для iPad», «дизайне macOS», «tvOS», «visionOS», «watchOS», «платформе Apple», «какой платформе», различиях платформ, соглашениях, специфичных для платформы, или дизайне многоплатформенного приложения. Также используйте, когда пользователь говорит: «Следует ли мне создавать разные дизайны для iPad и iPhone», «как мое приложение работает на VisionOS», «Чем отличаются приложения для MacOS», «Перенос моего приложения на другую платформу», «Универсальный дизайн приложения» или «Какие методы ввода использует эта платформа». Перекрестные ссылки: hig-foundations для общих основ дизайна, hig-patterns для шаблонов взаимодействия, hig-Components-Layout для навигационных структур, hig-Components-Content для отображения контента.
