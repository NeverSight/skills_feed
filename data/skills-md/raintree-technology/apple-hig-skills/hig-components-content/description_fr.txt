Directives Apple Human Interface pour les composants d’affichage de contenu. Utilisez cette compétence lorsque l'utilisateur pose des questions sur le « composant graphique », la « vue de collection », la « vue d'image », la « vue Web », le « puits de couleurs », le « puits d'image », la « vue d'activité », le « verrouillage », la « visualisation des données », « l'affichage du contenu », l'affichage d'images, le rendu de contenu Web, les sélecteurs de couleurs ou la présentation de collections d'éléments dans les applications Apple. À utiliser également lorsque l'utilisateur demande « comment afficher les graphiques », « quelle est la meilleure façon d'afficher les images », « dois-je utiliser une vue Web », « comment créer une grille d'éléments », « quel composant affiche les médias » ou « comment présenter une feuille de partage ». Références croisées : bases élevées pour la couleur/typographie/accessibilité, modèles élevés pour les modèles de visualisation de données, disposition de composants élevés pour les conteneurs structurels, plates-formes élevées pour le comportement des composants spécifiques à la plate-forme.
