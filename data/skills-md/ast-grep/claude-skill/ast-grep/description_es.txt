Guía para escribir reglas ast-grep para realizar búsqueda y análisis de código estructural. Úselo cuando los usuarios necesiten buscar bases de código utilizando patrones de árbol de sintaxis abstracta (AST), encontrar estructuras de código específicas o realizar consultas de código complejas que van más allá de una simple búsqueda de texto. Esta habilidad debe usarse cuando los usuarios solicitan buscar patrones de código, encontrar construcciones de lenguaje específicas o localizar código con características estructurales particulares.
