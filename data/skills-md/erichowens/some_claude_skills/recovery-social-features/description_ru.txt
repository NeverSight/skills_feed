Социальные функции, ориентированные на конфиденциальность, для приложений восстановления — спонсоры, группы, обмен сообщениями, связи с друзьями. Используйте для систем спонсор/спонсор, групп на основе встреч, поддержки коллег, безопасного обмена сообщениями. Активируйте «спонсор», «спонсор», «группа восстановления», «партнер по подотчетности», «трезвая сеть», «группа встреч», «поддержка коллег». НЕ для общих шаблонов социальных сетей (используйте стандартные социальные сети), функций знакомств или общедоступных профилей.
