Авторитетный мета-навык для создания, проверки и улучшения навыков агента. Сочетает опыт тренера по навыкам с рабочими процессами по созданию навыков. Используйте для создания, проверки, улучшения, отладки активации и разработки постепенного раскрытия навыков. НЕ для общих функций Claude Code, отладки во время выполнения или кодирования, не требующего навыков.
