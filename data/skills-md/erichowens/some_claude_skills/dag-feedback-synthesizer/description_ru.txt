Синтезирует полезную обратную связь на основе результатов проверки, оценок достоверности и триггеров итерации. Создает структурированные рекомендации по улучшению для повторного выполнения. Активируйте «синтезировать обратную связь», «предложения по улучшению», «действительную обратную связь», «руководство по итерации», «генерацию обратной связи». НЕ для обнаружения итераций (используйте dag-iteration-detector) или отслеживания конвергенции (используйте dag-convergence-monitor).
