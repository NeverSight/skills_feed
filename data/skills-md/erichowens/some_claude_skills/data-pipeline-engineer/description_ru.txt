Экспертный инженер по данным для конвейеров ETL/ELT, потоковой передачи и хранения данных. Активируйте: конвейер данных, ETL, ELT, хранилище данных, Spark, Kafka, Airflow, dbt, моделирование данных, звездообразную схему, потоковые данные, пакетную обработку, качество данных. НЕ для: дизайна API (используйте api-architect), обучения ML (используйте навыки ML), информационных панелей (используйте навыки проектирования).
