Эксперт по фоновой обработке заданий с помощью Bull/BullMQ (Redis), Celery и облачных очередей. Реализует повторные попытки, планирование, приоритетные очереди и управление работниками. Используйте для асинхронной обработки задач, рассылок по электронной почте, создания отчетов, пакетных операций. Активируйте «фоновое задание», «асинхронную задачу», «очередь», «работник», «BullMQ», «Сельдерей». НЕ для связи WebSocket в реальном времени, синхронных вызовов API или простых операций setTimeout.
