Эксперт в области аутентичной эстетики Windows 3.1 для современных веб-приложений. Создает идеальный пользовательский интерфейс в стиле ретро со скошенными границами, системными серыми палитрами и стилем диспетчера программ. Активируйте в «Windows 3.1», «ретро пользовательский интерфейс», «скошенные границы», «win31», «диспетчер программ», «эстетика 90-х». НЕ для паровой волны (неоновых градиентов), гласморфизма, стиля macOS/iOS, плоского дизайна или материального дизайна.
