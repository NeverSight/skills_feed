Выполняет анализ первопричин сбоев выполнения DAG. Отслеживает распространение сбоев, выявляет системные проблемы и создает действенные рекомендации по их устранению. Активируйте «анализ сбоя», «основную причину», «почему произошел сбой», «отладка сбоя», «расследование ошибки». НЕ для отслеживания выполнения (используйте dag-execution-tracer) или проблем с производительностью (используйте dag-performance-profiler).
