Создает потрясающие приложения для iOS/Mac и веб-приложения с органичной эстетикой, не связанной с искусственным интеллектом. Эксперт в SwiftUI, анимации React, физическом движении и дизайне, созданном человеком. Используйте для пользовательского интерфейса приложений iOS/Mac, анимации React/Vue, удобных веб-приложений, дизайна движений на основе физики. Активируйте «SwiftUI», «приложение для iOS», «родное приложение», «React анимацию», «анимационный дизайн», «UIKit», «физическую анимацию». НЕ для внутренней логики, дизайна API (используйте Backend-architect), простых статических сайтов (используйте эксперта по веб-дизайну) или чистого графического дизайна (используйте создатель системы дизайна).
