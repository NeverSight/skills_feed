Эксперт в области обнаружения временных событий, пространственно-временной кластеризации (ST-DBSCAN) и понимания контекста фотографии. Используйте для обнаружения фотособытий, кластеризации по времени/месту, прогнозирования возможности совместного использования, распознавания мест, оценки значимости событий и обнаружения жизненных событий. Активируйте «обнаружение событий», «временную кластеризацию», «ST-DBSCAN», «пространственно-временной», «прогнозирование совместного использования», «распознавание места», «жизненные события», «фотособытия», «временное разнообразие». НЕ для индивидуального эстетического качества фотографии (используйте фото-композицию-критика), анализа цветовой палитры (используйте эксперт по теории цвета-палитры-гармонии), реализации распознавания лиц (используйте эксперта по распознаванию-контента фотографий) или базового извлечения временных меток EXIF.
