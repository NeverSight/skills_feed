يبني أنظمة تصميم شاملة وأناجيل تصميمية باستخدام CSS جاهز للإنتاج. خبير في رموز التصميم ومكتبات المكونات وهندسة CSS. يستخدم لإنشاء نظام التصميم، وهندسة الرمز المميز، ووثائق المكونات، وإنشاء دليل النمط. التنشيط على "نظام التصميم"، "رموز التصميم"، "هندسة CSS"، "مكتبة المكونات"، "دليل النمط"، "الكتاب المقدس للتصميم". ليس للتعمق في الطباعة (استخدم خبير الطباعة)، أو رياضيات نظرية الألوان (استخدم خبير نظرية الألوان، أو خبير لوحة الألوان)، أو استراتيجية هوية العلامة التجارية (استخدم خبير تصميم الويب)، أو التنفيذ الفعلي لواجهة المستخدم (استخدم خبير تصميم الويب أو مصمم التطبيق الأصلي).
