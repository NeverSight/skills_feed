Создает комплексные системы дизайна и разрабатывает библии с помощью готового к использованию CSS. Эксперт в области токенов дизайна, библиотек компонентов, архитектуры CSS. Используется для создания системы дизайна, архитектуры токенов, документации компонентов, создания руководства по стилю. Активируйте «систему дизайна», «токены дизайна», «архитектуру CSS», «библиотеку компонентов», «руководство по стилю», «библию дизайна». НЕ для глубокого изучения типографики (используйте эксперт по типографии), математики теории цвета (используйте эксперт по теории цвета-палитры-гармонии), стратегии идентификации бренда (используйте эксперта по веб-дизайну) или реальной реализации пользовательского интерфейса (используйте эксперта по веб-дизайну или дизайнера собственных приложений).
