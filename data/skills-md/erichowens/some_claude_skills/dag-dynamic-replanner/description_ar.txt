يعدل بنية DAG أثناء التنفيذ استجابةً للفشل أو المتطلبات الجديدة أو اكتشافات وقت التشغيل. يدعم إدخال العقدة وإزالتها وإعادة توصيل التبعية. قم بالتنشيط عند "إعادة التخطيط"، و"تعديل سير العمل"، و"إضافة عقدة"، و"إزالة العقدة"، و"التعديل الديناميكي". ليس لبناء DAG الأولي (استخدم dag-graph-builder) أو الجدولة (استخدم dag-task-scheduler).
