Проверяет структуры DAG, выполняет топологическую сортировку, обнаруживает циклы и разрешает конфликты зависимостей. Использует алгоритм Кана для оптимального порядка выполнения. Активируйте «разрешение зависимостей», «топологическую сортировку», «обнаружение цикла», «порядок зависимостей», «проверку данных». НЕ для создания групп DAG (используйте dag-graph-builder) или планирования выполнения (используйте dag-task-scheduler).
