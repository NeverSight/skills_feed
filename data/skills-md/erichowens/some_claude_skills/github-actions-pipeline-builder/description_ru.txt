Создавайте рабочие конвейеры CI/CD с помощью GitHub Actions. Осуществляет построение матриц, кэширование, развертывание, тестирование и сканирование безопасности. Используйте для автоматического тестирования, развертывания и рабочих процессов выпуска. Активируйте «Действия GitHub», «CI/CD», «рабочий процесс», «конвейер развертывания», «автоматическое тестирование». НЕ для Jenkins/CircleCI, ручного развертывания или репозиториев, отличных от GitHub.
