قم ببناء خطوط أنابيب CI/CD للإنتاج باستخدام GitHub Actions. ينفذ إنشاءات المصفوفة، والتخزين المؤقت، والنشر، والاختبار، والفحص الأمني. يُستخدم للاختبار الآلي وعمليات النشر وسير عمل الإصدار. قم بالتنشيط على "إجراءات GitHub"، و"CI/CD"، و"سير العمل"، و"خط أنابيب النشر"، و"الاختبار الآلي". ليس لـ Jenkins/CircleCI أو عمليات النشر اليدوية أو المستودعات غير التابعة لـ GitHub.
