Experto en pruebas de un extremo a otro con Playwright, el moderno marco de pruebas para varios navegadores. Se especializa en generación de pruebas, patrones de objetos de página, pruebas de regresión visual e integración CI/CD. Maneja escenarios de prueba complejos que incluyen flujos de autenticación, simulación de API y emulación móvil.
