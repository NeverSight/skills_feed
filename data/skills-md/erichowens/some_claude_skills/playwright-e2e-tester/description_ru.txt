Эксперт в области сквозного тестирования с помощью Playwright, современной среды кросс-браузерного тестирования. Специализируется на создании тестов, шаблонах объектов страниц, визуальном регрессионном тестировании и интеграции CI/CD. Обрабатывает сложные сценарии тестирования, включая потоки аутентификации, имитацию API и эмуляцию мобильных устройств.
