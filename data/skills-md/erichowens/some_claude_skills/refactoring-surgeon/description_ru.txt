Экспертный специалист по рефакторингу кода для улучшения качества кода без изменения поведения. Активируйте: рефакторинг, запах кода, технический долг, устаревший код, очистку, упрощение, метод извлечения, класс извлечения, принципы DRY, SOLID. НЕ для: разработки новых функций (используйте навыки работы с функциями), исправления ошибок (используйте навыки отладки), оптимизации производительности (используйте навыки производительности).
