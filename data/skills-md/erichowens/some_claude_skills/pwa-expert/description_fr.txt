Développement progressif d'applications Web avec Service Workers, prise en charge hors ligne et comportement similaire à celui d'une application. À utiliser pour les stratégies de mise en cache, les invites d'installation, les notifications push, la synchronisation en arrière-plan. Activez sur "PWA", "Service Worker", "hors ligne", "invite d'installation", "beforeinstallprompt", "manifest.json", "workbox", "cache-first". PAS pour le développement d'applications natives (utilisez React Native), les performances Web générales (utilisez la documentation sur les performances) ou le rendu côté serveur.
