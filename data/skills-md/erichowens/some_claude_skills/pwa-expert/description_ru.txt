Прогрессивная разработка веб-приложений с использованием Service Workers, автономной поддержкой и поведением, подобным приложению. Используйте для стратегий кэширования, запросов на установку, push-уведомлений, фоновой синхронизации. Активируйте «PWA», «Service Worker», «офлайн», «приглашение к установке», «beforeinstallprompt», «manifest.json», «workbox», «сначала кэш». НЕ для разработки собственных приложений (используйте React Native), общей производительности в Интернете (используйте документацию по производительности) или рендеринга на стороне сервера.
