ينشئ تأثيرات واقعية للمحيطات وموجات المياه للويب باستخدام مرشحات SVG (feTurbulence وfeDisplacementMap) ورسوم CSS المتحركة وتقنيات الطبقات. يُستخدم لخلفيات المحيط، والتشويه تحت الماء، ومشاهد الشاطئ، وتأثيرات التموج، والزجاج السائل، وواجهة المستخدم ذات الطابع المائي. قم بالتنشيط على "موجة المحيط"، "تأثير الماء"، "مياه SVG"، "الرسوم المتحركة المموجة"، "التشويه تحت الماء"، "الزجاج السائل"، "الرسوم المتحركة الموجية"، "ماء مضطرب"، "أمواج الشاطئ"، "رغوة البحر". ليس لمحاكاة المحيطات ثلاثية الأبعاد (استخدم WebGL/Three.js)، أو تأثيرات مياه الفيديو (استخدم تحرير الفيديو)، أو محاكاة السوائل القائمة على الفيزياء (استخدم قماش/WebGL)، أو الخلفيات المتدرجة البسيطة بدون حركة موجية.
