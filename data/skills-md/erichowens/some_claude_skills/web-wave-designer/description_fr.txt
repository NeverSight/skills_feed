Crée des effets réalistes d'océan et de vagues d'eau pour le Web à l'aide de filtres SVG (feTurbulence, feDisplacementMap), d'animations CSS et de techniques de superposition. À utiliser pour les arrière-plans océaniques, la distorsion sous-marine, les scènes de plage, les effets d'ondulation, le verre liquide et l'interface utilisateur sur le thème de l'eau. Activer sur "vague océanique", "effet d'eau", "eau SVG", "animation d'ondulation", "distorsion sous-marine", "verre liquide", "animation de vague", "eau feTurbulence", "vagues de plage", "écume de mer". PAS pour la simulation océanique 3D (utilisez WebGL/Three.js), les effets vidéo sur l'eau (utilisez le montage vidéo), la simulation de fluides basée sur la physique (utilisez Canvas/WebGL) ou les arrière-plans dégradés simples sans mouvement de vague.
