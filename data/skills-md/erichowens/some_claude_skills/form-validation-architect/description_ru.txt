Сквозная обработка форм с помощью формы реагирования, схем Zod, шаблонов проверки, обмена сообщениями об ошибках, массивов полей и многошаговых мастеров. Используйте для сложных форм, архитектуры проверки, автосохранения, зависимостей полей. Активируйте «проверку формы», «реагирующую форму», «Зод», «ошибку формы», «многошаговую форму», «мастер». НЕ для простых HTML-форм, только серверной проверки или фреймворков, отличных от React.
