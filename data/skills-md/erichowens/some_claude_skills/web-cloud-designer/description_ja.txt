SVG フィルター (feTurbulence、feDisplacementMap)、CSS アニメーション、およびレイヤー化技術を使用して、Web 用のリアルな雲エフェクトを作成します。大気の背景、気象効果、スカイボックス、視差シーン、装飾的な雲の要素に使用します。 「雲効果」、「SVG雲」、「リアル雲」、「大気背景」、「空アニメーション」、「feTurbulence」、「天気効果」、「視差雲」で有効化します。 3D レンダリング (WebGL/Three.js スキルを使用)、写真操作 (画像編集ツールを使用)、気象データ API (データ統合スキルを使用)、または体積効果のない単純な CSS グラデーションには使用できません。
