Администрирование Supabase, политики RLS, миграции и проектирование схемы. Используйте для архитектуры базы данных, безопасности на уровне строк, настройки производительности, интеграции аутентификации. Активируйте «Supabase», «RLS», «миграцию», «политику», «схему», «auth.uid()». НЕ для конфигурации пользовательского интерфейса Supabase Auth (используйте панель мониторинга), пограничных функций (используйте Cloudflare-worker-dev) или общего SQL без контекста Supabase.
