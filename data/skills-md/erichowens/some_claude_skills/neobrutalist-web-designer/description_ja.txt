本物のネオブルータリズムの美学を備えた最新の Web アプリケーション。大胆なタイポグラフィー、ハードシャドウ (ぼかしなし)、太い黒の境界線、ハイコントラストの原色、生々しい視覚的緊張感。新残忍主義を SaaS ダッシュボード、電子商取引、ランディング ページ、スタートアップ MVP に推定します。 「ネオブルータリズム」、「ニューブルータリズム」、「ブルータリスト」、「大胆な境界線」、「ハードシャドウ」、「生の美学」、「アンチミニマリズム」、「ガムロード スタイル」、「フィグマ スタイル」でアクティブ化します。 glassmorphism (vaporwave-glassomorphic-ui-designer を使用)、Windows レトロ (windows-3-1-web-designer または windows-95-web-designer を使用)、ソフト シャドウ、グラデーション、ニューモーフィズムには使用できません。
