Des applications Web modernes avec une esthétique néobrutaliste authentique. Typographie audacieuse, ombres dures (pas de flou), bordures noires épaisses, couleurs primaires très contrastées, tension visuelle brute. Extrapole le néobrutalisme aux tableaux de bord SaaS, au commerce électronique, aux pages de destination et aux MVP de startup. Activez sur le « néobrutalisme », le « neubrutalisme », le « brutaliste », les « frontières audacieuses », les « ombres dures », « l'esthétique brute », « l'anti-minimalisme », le « style gumroad », le « style figma ». PAS pour le glassmorphisme (utilisez vaporwave-glassomorphic-ui-designer), Windows retro (utilisez windows-3-1-web-designer ou windows-95-web-designer), les ombres douces, les dégradés, le neumorphisme.
