Управляйте манифестами функций для отслеживания кода. Используйте при создании новых функций, обновлении существующих функций, проверке работоспособности функций или изучении взаимосвязи между функцией и кодом. Активируется для проверки манифеста, создания функций, обновлений журнала изменений и запросов на отслеживаемость.
