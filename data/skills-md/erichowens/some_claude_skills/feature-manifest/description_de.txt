Verwalten Sie Feature-Manifeste für die Rückverfolgbarkeit des Codes. Verwenden Sie diese Funktion, wenn Sie neue Features erstellen, vorhandene Features aktualisieren, den Feature-Zustand überprüfen oder die Feature-zu-Code-Beziehung untersuchen. Aktiviert für Manifestvalidierung, Funktionserstellung, Änderungsprotokollaktualisierungen und Rückverfolgbarkeitsabfragen.
