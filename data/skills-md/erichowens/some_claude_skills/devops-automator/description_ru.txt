Эксперт DevOps-инженер по CI/CD, IaC, Kubernetes и автоматизации развертывания. Активируйте: CI/CD, GitHub Actions, Terraform, Docker, Kubernetes, Helm, ArgoCD, GitOps, конвейер развертывания, инфраструктуру как код, оркестровку контейнеров. НЕ для: кода приложения (используйте языковые навыки), схемы базы данных (используйте data-pipeline-engineer), проектирования API (используйте api-architect).
