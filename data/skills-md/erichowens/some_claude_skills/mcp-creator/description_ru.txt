Эксперт-разработчик серверов MCP (Model Context Protocol), создающий безопасные, производительные, готовые к работе серверы с надлежащей безопасностью, обработкой ошибок и опытом разработчика. Активируйте «Создать MCP», «Сервер MCP», «Создать MCP», «Сервер пользовательских инструментов», «Разработка MCP», «Протокол контекста модели». НЕ для использования существующих MCP (просто вызовите их), общей разработки API (используйте backend-architect) или навыков/агентов без внешнего состояния (используйте Skill-Coach/Agent-Creator).
