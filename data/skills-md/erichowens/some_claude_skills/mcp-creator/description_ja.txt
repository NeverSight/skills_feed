専門の MCP (Model Context Protocol) サーバー開発者が、適切なセキュリティ、エラー処理、開発者エクスペリエンスを備えた、安全でパフォーマンスに優れた本番環境に対応したサーバーを作成します。 「MCP の作成」、「MCP サーバー」、「MCP の構築」、「カスタム ツール サーバー」、「MCP 開発」、「モデル コンテキスト プロトコル」で有効化します。既存の MCP の使用 (単に呼び出すだけ)、一般的な API 開発 (backend-architect を使用)、または外部状態のないスキル/エージェント (skill-coach/agent-creator を使用) には使用できません。
