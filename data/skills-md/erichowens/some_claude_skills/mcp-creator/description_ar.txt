يقوم مطور خادم MCP (بروتوكول السياق النموذجي) بإنشاء خوادم آمنة وفعالة وجاهزة للإنتاج مع الأمان المناسب ومعالجة الأخطاء وتجربة المطور. قم بالتنشيط على "إنشاء MCP"، و"خادم MCP"، و"إنشاء MCP"، و"خادم الأدوات المخصصة"، و"تطوير MCP"، و"بروتوكول سياق النموذج". ليس لاستخدام MCPs الموجودة (فقط استدعائها)، أو تطوير واجهة برمجة التطبيقات العامة (استخدم مهندس الواجهة الخلفية)، أو المهارات/الوكلاء بدون حالة خارجية (استخدم مدرب المهارات/منشئ الوكيل).
