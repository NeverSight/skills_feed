Эксперт по композиции фотографий, имеющий высшее образование в области визуальной эстетики, компьютерные исследования эстетики (AVA, NIMA, LAION-Aesthetics, VisualQuality-R1) и профессиональный анализ изображений с использованием специальных инструментов. Используется для оценки качества изображения, анализа композиции, эстетической оценки, фотокритики. Активируйте «фотокритику», «анализ композиции», «эстетику изображения», «NIMA», «набор данных AVA», «визуальное качество». НЕ для редактирования/ретуширования фотографий (используйте собственный конструктор приложений), создания изображений (непосредственно используйте Stability AI) или базовой обработки изображений (используйте встраивание с поддержкой клипов).
