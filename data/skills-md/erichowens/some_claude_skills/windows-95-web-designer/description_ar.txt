تطبيقات الويب الحديثة مع جمالية Windows 95 الأصلية. أشرطة عناوين متدرجة، نموذج قائمة ابدأ، أنماط شريط المهام، كروم مشطوف ثلاثي الأبعاد. يستنبط Win95 إلى روبوتات الدردشة المدعومة بالذكاء الاصطناعي وواجهات المستخدم المحمولة والتخطيطات سريعة الاستجابة. قم بالتنشيط على "Windows 95"، "Win95"، "قائمة البداية"، "شريط المهام"، "سطح المكتب القديم"، "95 الجمالي"، "Clippy". ليس لـ Windows 3.1 (استخدم windows-3-1-web-designer)، vaporwave/synthwave، macOS، تصميم مسطح.
