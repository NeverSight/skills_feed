Современные веб-приложения с аутентичной эстетикой Windows 95. Градиентные заголовки, парадигма меню «Пуск», узоры на панели задач, трехмерный скошенный хром. Экстраполирует Win95 на чат-ботов с искусственным интеллектом, мобильные пользовательские интерфейсы и адаптивные макеты. Активируйте в «Windows 95», «Win95», «Меню «Пуск», «Панель задач», «Рабочий стол в стиле ретро», «Эстетика 95», «Клиппи». НЕ для Windows 3.1 (используйте windows-3-1-web-designer), вапорвейв/синтвейв, macOS, плоский дизайн.
