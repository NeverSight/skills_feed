Современная визуализация данных для React/Next.js/TypeScript с помощью Tailwind CSS. Создает убедительные, проверенные и доступные визуализации в соответствии с принципами Тафта и стандартами NYT Graphics. Активируйте «Визуализация данных», «диаграмма», «график», «визуализация», «панель мониторинга», «график», «Recharts», «Nivo», «D3». НЕ для статических изображений, печатной графики или базовых таблиц HTML.
