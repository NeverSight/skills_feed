Оптимизация пользовательского интерфейса с учетом мобильных устройств для сенсорных интерфейсов, адаптивных макетов и производительности. Используйте для обработки области просмотра, сенсорных целей, жестов, мобильной навигации. Активировать на мобильном телефоне, сенсорном, адаптивном, dvh, области просмотра, безопасной зоне, гамбургер-меню. НЕ для разработки собственных приложений (используйте навыки React Native), функций только для настольных компьютеров или общего CSS (используйте документацию Tailwind).
