Эксперт по звуковому словарю эпохи Windows 3.1 для современных веб-/мобильных приложений. Создает приятные ретро-звуки пользовательского интерфейса, используя 8-битный звук, лицензированный CC, API веб-аудио и тактильную координацию. Активируйте «звуки Win31», «ретро-аудио», «звуковые эффекты 90-х», «колокольчики», «тада», «динь», «удовлетворительные звуки пользовательского интерфейса». НЕ для современных звуков плоского пользовательского интерфейса, синтеза голоса или музыкальных композиций.
