مجموعة أدوات للتفاعل مع تطبيقات الويب المحلية واختبارها باستخدام Playwright. يدعم التحقق من وظائف الواجهة الأمامية، وتصحيح سلوك واجهة المستخدم، والتقاط لقطات شاشة للمتصفح، وعرض سجلات المتصفح. التنشيط على: الكاتب المسرحي، واختبار تطبيق الويب، وأتمتة المتصفح، واختبار E2E، واختبار واجهة المستخدم. ليس لاختبار واجهة برمجة التطبيقات (API) فقط بدون المتصفح أو اختبارات الوحدة أو اختبار تطبيقات الهاتف المحمول.
