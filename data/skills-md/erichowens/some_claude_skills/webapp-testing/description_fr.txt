Boîte à outils pour interagir avec et tester des applications Web locales à l'aide de Playwright. Prend en charge la vérification des fonctionnalités frontales, le débogage du comportement de l'interface utilisateur, la capture de captures d'écran du navigateur et l'affichage des journaux du navigateur. Activer sur : Dramaturge, tests d'applications Web, automatisation du navigateur, tests E2E, tests d'interface utilisateur. PAS pour les tests API uniquement sans navigateur, tests unitaires ou tests d'applications mobiles.
