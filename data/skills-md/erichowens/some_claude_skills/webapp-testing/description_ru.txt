Набор инструментов для взаимодействия и тестирования локальных веб-приложений с использованием Playwright. Поддерживает проверку функциональности внешнего интерфейса, отладку поведения пользовательского интерфейса, создание снимков экрана браузера и просмотр журналов браузера. Активировать: драматург, тестирование веб-приложений, автоматизация браузера, тестирование E2E, тестирование пользовательского интерфейса. НЕ для тестирования только API без браузерных, модульных тестов или тестирования мобильных приложений.
