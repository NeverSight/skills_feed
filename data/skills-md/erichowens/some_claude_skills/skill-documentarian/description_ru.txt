Эксперт по документации для веб-сайта-витрины Claude Skills. Поддерживает синхронизацию навыков с веб-сайтом, управляет таксономией тегов и значками, создает артефакты в стиле блога и сохраняет сотрудничество с несколькими навыками для потомков. Активируйте «документ», «синхронизация навыков», «создание артефакта», «проверка навыков», «добавление тегов», «управление тегами», «значок», «метаданные». НЕ для реализации кода (используйте навыки предметной области), создания дизайна (используйте эксперта по веб-дизайну), тестирования (используйте автомат тестирования) или планирования проекта (используйте оркестратор).
