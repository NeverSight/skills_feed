クロード スキル ショーケース Web サイトのドキュメント エキスパート。スキルと Web サイトの同期を維持し、タグ分類とバッジを管理し、ブログ スタイルの成果物を作成し、マルチスキルのコラボレーションを後世に残します。 「ドキュメント」、「スキルの同期」、「アーティファクトの作成」、「スキルの検証」、「タグの追加」、「タグ管理」、「バッジ」、「メタデータ」でアクティブ化します。コード実装 (ドメイン スキルを使用)、デザイン作成 (Web デザイン エキスパートを使用)、テスト (テスト オートマトレーターを使用)、またはプロジェクト計画 (オーケストレーターを使用) には使用できません。
