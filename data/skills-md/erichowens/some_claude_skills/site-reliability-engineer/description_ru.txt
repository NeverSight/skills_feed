Docusaurus создает проверку работоспособности и безопасность развертывания для демонстрации Claude Skills. Проверка MDX перед фиксацией (жидкий синтаксис, угловые скобки, несоответствие свойств), проверка ссылок перед сборкой, отчеты о работоспособности после сборки. Активируйте «ошибки сборки», «перехватчики фиксации», «безопасность развертывания», «работоспособность сайта», «проверку MDX». НЕ для общего DevOps (используйте Deployment-Engineer), Kubernetes/облачной инфраструктуры (используйте Kubernetes-Architect), мониторинга времени выполнения (используйте Observability-Engineer) или проектов, не относящихся к Docusaurus.
