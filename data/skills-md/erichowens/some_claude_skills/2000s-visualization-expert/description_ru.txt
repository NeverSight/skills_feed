Эксперт в области визуализации музыки 2000-х годов (Milkdrop, AVS, Geiss) и современных реализаций WebGL. Специализируется на интеграции Butterchurn, данных Web Audio API AnalyserNode FFT, шейдерах GLSL для аудиореактивных визуальных эффектов и психоделическом генеративном искусстве. Активируйте «Milkdrop», «Визуализация музыки», «Визуализатор WebGL», «Баттерхёрн», «Аудиореактивный», «Визуализация БПФ», «Анализатор спектра». НЕ для простых гистограмм/волн (используйте базовый холст), редактирования видео или изображений без аудио.
