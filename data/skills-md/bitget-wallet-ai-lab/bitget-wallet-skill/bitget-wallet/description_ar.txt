تفاعل مع Bitget Wallet API للحصول على بيانات سوق التشفير ومعلومات الرمز المميز وعروض أسعار المبادلة وعمليات التدقيق الأمني. يُستخدم عندما يسأل المستخدم عن أسعار الرمز المميز، أو بيانات السوق، أو عروض أسعار المبادلة/التداول، أو فحوصات أمان الرمز المميز، أو مخططات K-line، أو تصنيفات الرمز المميز على السلاسل المدعومة (ETH، وSOL، وBSC، وBase، وما إلى ذلك).
