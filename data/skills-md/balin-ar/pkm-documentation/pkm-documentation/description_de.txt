Persönliches Wissensmanagement-Dokumentationssystem basierend auf Zettelkasten-Prinzipien. Verwenden Sie es, wenn Sie Gespräche, Entscheidungen, Entwicklungsarbeit, Erkenntnisse oder anderes Wissen dokumentieren, das erhalten bleiben soll. Wird bei neuen Projekten, technischen Entscheidungen, abgeschlossenen Aufgaben, gewonnenen Erkenntnissen, Besprechungsnotizen oder wenn der Benutzer „Dokumentieren“ sagt, ausgelöst. Wenden Sie diesen Workflow auf alle täglichen Gedächtnisnotizen und Langzeitgedächtnisaktualisierungen an.
