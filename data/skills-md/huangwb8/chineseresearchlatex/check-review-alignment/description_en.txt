Use only when the user explicitly requests to "check/optimize citations in the body of the review `{topic}_review.tex`" or to "run check-review-alignment".

Via the host AI’s semantic understanding, verify each citation against the source content; **only when a fatal citation error is found**, perform the minimal rewrite of the "sentence containing the citation", and reuse the `systematic-literature-review` rendering script to output PDF/Word (the script itself does not call an LLM API locally).

Core principle: **do not change for the sake of changing**. If uncertain whether it is a fatal error, leave it as-is and flag a warning in the report.

⚠️ Not applicable for:
- Users who simply want to generate systematic-review text (use systematic-literature-review)
- Users who only want to add/verify BibTeX entries (use the dedicated bib management workflow)
