Rendern Sie Meerjungfrau-Diagramme als lesbare ASCII/Unicode-Grafik im Terminal (aus .mmd/.mermaid-Dateien, stdin oder Markdown „Mermaid Fences“). Verwenden Sie diese Option, wenn Sie lukilabs/beautiful-mermaid installieren oder verwenden, einen CLI-Renderer für die Mermaid-zu-ASCII-Ausgabe erstellen, Mermaid-Diagramme im Terminal in der Vorschau anzeigen oder Mermaid-Blöcke aus Markdown-Dateien extrahieren/rendern.
