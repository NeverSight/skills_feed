Gérez les événements du serveur Hytale, notamment les actions des joueurs, les changements dans le monde, les interactions entre entités et les événements personnalisés. À utiliser lorsqu'on vous demande d'« écouter les événements », de « gérer les actions des joueurs », de « réagir aux modifications bloquées », de « créer des gestionnaires d'événements » ou de « mettre en œuvre des écouteurs d'événements ».
