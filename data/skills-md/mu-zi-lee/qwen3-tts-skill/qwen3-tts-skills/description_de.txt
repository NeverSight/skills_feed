Stellen Sie einen lokalen TTS-Workflow rund um Qwen3-TTS bereit. Funktionen: Einzelsatz-Sprachgenerierung (CustomVoice/VoiceDesign/VoiceClone), Batch-Voice-Over-Generierung für lange Skripte (Artikel → Voice-Over-JSON → Batch-TTS → Zusammenführen). Anwendungsfälle: Sprachgenerierung, Erzählung von Hörbüchern, Video-Voice-Over, Vorlesen von Dialogen mit mehreren Charakteren, Klonen von Stimmen.
