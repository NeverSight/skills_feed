Обеспечьте локальный рабочий процесс TTS вокруг Qwen3-TTS. Возможности: генерация голоса из одного предложения (CustomVoice/VoiceDesign/VoiceClone), пакетная генерация закадрового текста для длинных скриптов (статья → закадровый JSON → пакетный TTS → слияние). Варианты использования: генерация голоса, повествование в аудиокниге, озвучка видео, чтение многосимвольных диалогов, клонирование голоса.
