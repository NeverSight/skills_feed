Fournissez un flux de travail TTS local autour de Qwen3-TTS. Fonctionnalités : génération de voix d'une seule phrase (CustomVoice/VoiceDesign/VoiceClone), génération de voix off par lots pour les scripts longs (article → voix off JSON → batch TTS → fusion). Cas d'utilisation : génération de voix, narration de livres audio, voix off vidéo, lecture de dialogues multi-personnages, clonage de voix.
