Tests de bout en bout, API et réactifs pour les applications Web utilisant Playwright avec TypeScript. À utiliser lorsqu'on vous demande d'écrire, d'exécuter, de déboguer ou de maintenir des tests TypeScript Playwright (@playwright/test) pour le comportement de l'interface utilisateur, les soumissions de formulaires, les flux d'utilisateurs, la validation d'API, la conception réactive ou la régression visuelle. Couvre l'automatisation du navigateur, l'interception de réseau, la simulation, le modèle d'objet de page, les appareils et l'exécution parallèle.
