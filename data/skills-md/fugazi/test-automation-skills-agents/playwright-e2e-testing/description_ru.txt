Сквозное, API и адаптивное тестирование веб-приложений с использованием Playwright и TypeScript. Используйте, когда вас просят написать, запустить, отладить или поддерживать тесты TypeScript Playwright (@playwright/test) для поведения пользовательского интерфейса, отправки форм, пользовательских потоков, проверки API, адаптивного дизайна или визуальной регрессии. Охватывает автоматизацию браузера, сетевой перехват, макетирование, объектную модель страницы, фикстуры и параллельное выполнение.
