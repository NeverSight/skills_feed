Test end-to-end, API e reattivi per applicazioni Web utilizzando Playwright con TypeScript. Da utilizzare quando viene richiesto di scrivere, eseguire, eseguire il debug o gestire i test TypeScript di Playwright (@playwright/test) per il comportamento dell'interfaccia utente, l'invio di moduli, i flussi utente, la convalida dell'API, il design reattivo o la regressione visiva. Copre l'automazione del browser, l'intercettazione della rete, il mocking, il Page Object Model, le soluzioni e l'esecuzione parallela.
