Pruebas de extremo a extremo, API y responsivas para aplicaciones web que utilizan Playwright con TypeScript. Úselo cuando se le solicite escribir, ejecutar, depurar o mantener pruebas TypeScript de Playwright (@playwright/test) para el comportamiento de la interfaz de usuario, envíos de formularios, flujos de usuarios, validación de API, diseño responsivo o regresión visual. Cubre la automatización del navegador, interceptación de redes, burlas, modelo de objetos de página, accesorios y ejecución paralela.
