اختبار شامل وواجهة برمجة التطبيقات (API) واختبار الاستجابة لتطبيقات الويب باستخدام Playwright مع TypeScript. يُستخدم عندما يُطلب منك كتابة اختبارات Playwright (@playwright/test) أو تشغيلها أو تصحيحها أو صيانتها لسلوك واجهة المستخدم أو عمليات إرسال النماذج أو تدفقات المستخدم أو التحقق من صحة واجهة برمجة التطبيقات أو التصميم سريع الاستجابة أو الانحدار المرئي. يغطي أتمتة المتصفح، واعتراض الشبكة، والسخرية، ونموذج كائن الصفحة، والتركيبات، والتنفيذ المتوازي.
