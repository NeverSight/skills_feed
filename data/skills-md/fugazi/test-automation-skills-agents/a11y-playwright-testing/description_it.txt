Test di accessibilità per applicazioni web utilizzando Playwright (@playwright/test) con TypeScript e axe-core. Utilizzare quando viene richiesto di scrivere, eseguire o eseguire il debug di controlli di accessibilità automatizzati, test di navigazione da tastiera, gestione del focus, convalide ARIA/semantica, compatibilità con screen reader o test di conformità WCAG 2.1 livello AA. Copre l'integrazione axe-core, i principi POUR (percettibile, operabile, comprensibile, robusto), il contrasto dei colori, le etichette dei moduli, i punti di riferimento e i nomi accessibili.
