Pruebas de accesibilidad para aplicaciones web usando Playwright (@playwright/test) con TypeScript y axe-core. Úselo cuando se le solicite escribir, ejecutar o depurar comprobaciones de accesibilidad automatizadas, pruebas de navegación por teclado, gestión de enfoque, validaciones ARIA/semánticas, compatibilidad de lectores de pantalla o pruebas de cumplimiento de WCAG 2.1 Nivel AA. Cubre la integración del eje-núcleo, los principios POUR (perceptible, operable, comprensible, robusto), contraste de color, etiquetas de forma, puntos de referencia y nombres accesibles.
