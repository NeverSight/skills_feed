Tests d'accessibilité pour les applications Web utilisant Playwright (@playwright/test) avec TypeScript et axe-core. À utiliser lorsqu'on vous demande d'écrire, d'exécuter ou de déboguer des contrôles d'accessibilité automatisés, des tests de navigation au clavier, la gestion du focus, des validations ARIA/sémantiques, la compatibilité des lecteurs d'écran ou des tests de conformité WCAG 2.1 niveau AA. Couvre l'intégration axe-core, les principes POUR (perceptibles, exploitables, compréhensibles, robustes), le contraste des couleurs, les étiquettes de formulaire, les repères et les noms accessibles.
