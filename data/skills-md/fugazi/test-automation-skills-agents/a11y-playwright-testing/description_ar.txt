اختبار إمكانية الوصول لتطبيقات الويب باستخدام Playwright (@playwright/test) مع TypeScript وax-core. يُستخدم عند مطالبتك بكتابة أو تشغيل أو تصحيح فحوصات إمكانية الوصول التلقائية، أو اختبارات التنقل عبر لوحة المفاتيح، أو إدارة التركيز، أو التحقق من صحة ARIA/الدلالي، أو توافق قارئ الشاشة، أو اختبار التوافق مع WCAG 2.1 Level AA. يغطي التكامل المحوري الأساسي، ومبادئ POUR (يمكن إدراكها، وقابلية التشغيل، ومفهومة، وقوية)، وتباين الألوان، وتسميات النماذج، والمعالم، والأسماء التي يمكن الوصول إليها.
