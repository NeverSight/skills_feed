أنشئ وحدات مخصصة لـ [Harbor Boost](https://github.com/av/harbor/tree/main/boost)، وهو وكيل LLM مُحسّن. يُستخدم عند إنشاء وحدات Python التي تعترض/تحول عمليات استكمال دردشة LLM - سلاسل التفكير، أو الإدخال الفوري، أو المخرجات المنظمة، أو العناصر الاصطناعية، أو مسارات العمل المخصصة. يتم تشغيل الطلبات لإنشاء وحدات Boost، أو توسيع سلوك LLM عبر الوكيل، أو تنفيذ البرامج الوسيطة لإكمال الدردشة.
