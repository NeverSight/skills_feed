Erstellen Sie benutzerdefinierte Module für [Harbor Boost](https://github.com/av/harbor/tree/main/boost), einen optimierenden LLM-Proxy. Verwenden Sie es beim Erstellen von Python-Modulen, die LLM-Chat-Abschlüsse abfangen/transformieren – Argumentationsketten, Prompt-Injection, strukturierte Ausgaben, Artefakte oder benutzerdefinierte Workflows. Wird bei Anfragen zum Erstellen von Boost-Modulen, zum Erweitern des LLM-Verhaltens über Proxy oder zum Implementieren von Chat-Completion-Middleware ausgelöst.
