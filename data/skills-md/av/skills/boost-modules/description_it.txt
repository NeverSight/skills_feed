Crea moduli personalizzati per [Harbor Boost](https://github.com/av/harbor/tree/main/boost), un proxy LLM ottimizzante. Da utilizzare durante la creazione di moduli Python che intercettano/trasformano i completamenti delle chat LLM: catene di ragionamento, inserimento di prompt, output strutturati, artefatti o flussi di lavoro personalizzati. Attiva le richieste per creare moduli Boost, estendere il comportamento LLM tramite proxy o implementare middleware di completamento della chat.
