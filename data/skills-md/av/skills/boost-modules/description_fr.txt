Créez des modules personnalisés pour [Harbor Boost](https://github.com/av/harbor/tree/main/boost), un proxy LLM d'optimisation. À utiliser lors de la création de modules Python qui interceptent/transforment les complétions de chat LLM : chaînes de raisonnement, injection d'invites, sorties structurées, artefacts ou flux de travail personnalisés. Se déclenche sur les demandes pour créer des modules Boost, étendre le comportement LLM via un proxy ou implémenter un middleware de complétion de chat.
