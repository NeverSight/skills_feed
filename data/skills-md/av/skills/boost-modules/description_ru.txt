Создайте собственные модули для [Harbor Boost](https://github.com/av/harbor/tree/main/boost), оптимизирующего прокси-сервера LLM. Используйте при создании модулей Python, которые перехватывают/преобразовывают завершения чата LLM — цепочки рассуждений, внедрение подсказок, структурированные выходные данные, артефакты или пользовательские рабочие процессы. Запускается по запросам на создание модулей Boost, расширение поведения LLM через прокси-сервер или внедрение промежуточного программного обеспечения для завершения чата.
