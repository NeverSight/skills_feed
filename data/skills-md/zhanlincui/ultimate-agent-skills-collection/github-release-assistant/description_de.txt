Generieren Sie eine zweisprachige GitHub-Release-Dokumentation (README.md + README.zh.md) aus Repo-Metadaten und Benutzereingaben und leiten Sie die Release-Vorbereitung mit git add/commit/push. Verwenden Sie diese Option, wenn der Benutzer README-Dateien schreiben oder polieren, zweisprachige Dokumente erstellen, eine GitHub-Veröffentlichung vorbereiten oder den Release-Assistenten/die README-Generierung erwähnen möchte.
