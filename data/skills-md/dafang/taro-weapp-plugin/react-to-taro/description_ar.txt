نقل تطبيقات React Web (الإصدار 18+) إلى كود Taro 4.x المُحسّن لبرنامج WeChat Mini. استخدم هذه المهارة عند تحويل مكونات React أو صفحاتها أو أدواتها المساعدة إلى تعليمات برمجية متوافقة مع Taro باستخدام تعيين عناصر JSX المناسب، وتحويل معالج الأحداث، وتحويل التنقل/التوجيه، وطبقة واجهة برمجة التطبيقات (API shim).
