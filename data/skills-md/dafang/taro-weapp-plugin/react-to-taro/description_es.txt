Transpile aplicaciones web React (v18+) al código Taro 4.x optimizado para el mini programa WeChat. Utilice esta habilidad al convertir componentes, páginas o utilidades de React a código compatible con Taro con el mapeo de elementos JSX adecuado, la transformación del controlador de eventos, la conversión de navegación/enrutamiento y la capa de corrección de API.
