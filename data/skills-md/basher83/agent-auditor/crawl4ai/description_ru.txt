Этот навык следует использовать, когда пользователям необходимо парсить веб-сайты, извлекать структурированные данные, обрабатывать страницы с большим количеством JavaScript, сканировать несколько URL-адресов или создавать автоматизированные конвейеры веб-данных. Включает оптимизированные шаблоны извлечения с генерацией схемы для эффективного извлечения без LLM.
