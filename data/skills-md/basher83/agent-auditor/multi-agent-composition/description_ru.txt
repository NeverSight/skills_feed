Этот навык следует использовать, когда пользователь просит «выбрать между навыком и агентом», «составить многоагентную систему», «оркестрировать агентов», «управлять контекстом агента», «проектировать архитектуру компонентов», «следует ли мне использовать навык или агент», «когда использовать перехватчики против MCP», «построить рабочий процесс оркестратора», нуждается в структурах принятия решений для компонентов Claude Code (навыки, субагенты, перехватчики, серверы MCP, команды с косой чертой), шаблоны управления контекстом или хочет создать эффективные многокомпонентные агентные системы с правильной оркестровкой и управлением антипаттернами.
