Erstellen, listen und verwalten Sie macOS-Kalenderereignisse über AppleScript. Verwenden Sie diese Option, wenn der Benutzer eine Erinnerung hinzufügen, ein Ereignis planen, einen Kalendereintrag erstellen, eine Frist festlegen oder alles, was mit Apple Calendar auf macOS zu tun hat, wünscht. Wird bei Anfragen wie „Erinnere mich in 3 Tagen“, „Zu meinem Kalender hinzufügen“, „Plane eine Besprechung für nächsten Montag um 14 Uhr“ oder „Erstelle ein wiederkehrendes wöchentliches Ereignis“ ausgelöst. Nur macOS.
