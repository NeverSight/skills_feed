Создавайте, просматривайте и управляйте событиями календаря macOS с помощью AppleScript. Используйте, когда пользователь просит добавить напоминание, запланировать событие, создать запись в календаре, установить крайний срок или что-либо еще, связанное с Apple Calendar на macOS. Запускается по таким запросам, как «напомнить мне через 3 дня», «добавить в мой календарь», «запланировать встречу в следующий понедельник в 14:00», «создать повторяющееся еженедельное мероприятие». только для MacOS.
