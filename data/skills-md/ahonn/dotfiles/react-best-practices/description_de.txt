React Best Practices von React.dev und Vercel. Wird verwendet, wenn: (1) React-Code überprüft wird, (2) Leistungsprobleme behoben werden, (3) die Bundle-Größe optimiert wird, (4) Effekte oder Zustandslogik geschrieben werden. Auslöser für: Reaktionsleistung, erneutes Rendern, Bundle-Größe, Wasserfälle, Codeaufteilung, Memo, useCallback, useMemo, useEffect, SSR-Flimmern, langsames anfängliches Laden, React Compiler, Statusstruktur, unveränderliche Aktualisierung, setState-Array-Objekt.
