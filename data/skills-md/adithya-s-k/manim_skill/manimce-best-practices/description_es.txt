Se activa cuando: (1) El usuario menciona "manim" o "Comunidad Manim" o "ManimCE", (2) El código contiene `from manim import *`, (3) El usuario ejecuta comandos CLI `manim`, (4) Trabaja con clases específicas de Scene, MathTex, Create() o ManimCE.

Mejores prácticas para Manim Community Edition: el motor de animación Python mantenido por la comunidad. Cubre la estructura de la escena, animaciones, LaTeX/MathTex, 3D con ThreeDScene, control de cámara, estilo y uso de CLI.

NO para la versión ManimGL/3b1b (que usa importaciones `manimlib` y CLI `manimgl`).
