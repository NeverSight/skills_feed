Срабатывает, когда: (1) пользователь упоминает «manimgl» или «ManimGL» или «3b1b manim», (2) код содержит `from manimlib import *`, (3) пользователь запускает команды CLI `manimgl`, (4) работает с InteractiveScene, self.frame, self.embed(), ShowCreation() или шаблонами, специфичными для ManimGL.

Лучшие практики для ManimGL (версия 3Blue1Brown Гранта Сандерсона) — движка анимации на основе OpenGL с интерактивной разработкой. Охватывает InteractiveScene, Tex с t2c, управление кадром камеры, интерактивный режим (флаг -se), 3D-рендеринг и рабочий процесс checkpoint_paste().

НЕ для Manim Community Edition (который использует импорт manim и интерфейс командной строки manim).
