أفضل ممارسات وأنماط NestJS لإنشاء تطبيقات خلفية قابلة للتطوير وقابلة للصيانة. يجب استخدام هذه المهارة عند كتابة كود NestJS أو مراجعته أو إعادة هيكلته لضمان البنية المناسبة والأمان والأداء وجودة الكود. يتم تشغيل المهام التي تتضمن وحدات NestJS أو وحدات التحكم أو الخدمات أو الحراس أو الأنابيب أو البرامج الوسيطة أو عمليات قاعدة بيانات Prisma أو المصادقة أو أي أنماط خاصة بـ NestJS.
