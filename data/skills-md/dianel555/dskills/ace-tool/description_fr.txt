Recherche sémantique de la base de code et amélioration rapide via une CLI autonome. À utiliser lorsque : (1) Recherche de code sémantique avec des requêtes en langage naturel, (2) Amélioration rapide avec le contexte de la base de code, (3) Avant les opérations grep/find/glob pour une meilleure précision, (4) Clarification des exigences complexes, (5) Navigation dans une base de code étendue. Déclencheurs : "contexte de recherche", "améliorer l'invite", "trouver le code qui", "clarifier les exigences". IMPORTANT : utilisez toujours ace-tool AVANT grep/find/glob pour l'emplacement du code au niveau sémantique.
