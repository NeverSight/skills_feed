Семантический поиск по кодовой базе и быстрое улучшение с помощью автономного интерфейса командной строки. Используйте, когда: (1) Семантический поиск кода с запросами на естественном языке, (2) Быстрое улучшение контекста кодовой базы, (3) Перед операциями grep/find/glob для большей точности, (4) Уточнение сложных требований, (5) Навигация по большой кодовой базе. Триггеры: «контекст поиска», «усовершенствовать подсказку», «найти код», «уточнить требования». ВАЖНО: Всегда используйте ace-tool ПЕРЕД grep/find/glob для поиска кода на семантическом уровне.
