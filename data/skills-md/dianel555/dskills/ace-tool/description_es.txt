Búsqueda de base de código semántica y mejora rápida a través de CLI independiente. Úselo cuando: (1) Búsqueda de código semántico con consultas en lenguaje natural, (2) Mejora rápida con contexto de base de código, (3) Antes de las operaciones grep/find/glob para una mayor precisión, (4) Aclaración de requisitos complejos, (5) Navegación de base de código grande. Activadores: "contexto de búsqueda", "mejorar mensaje", "buscar código que", "aclarar requisitos". IMPORTANTE: Utilice siempre ace-tool ANTES de grep/find/glob para la ubicación del código de nivel semántico.
