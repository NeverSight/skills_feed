Recherche sémantique de haute précision et récupération de contenu via Exa API. À utiliser lorsque : (1) Recherche approfondie nécessitant une compréhension sémantique, (2) Recherche de documentation de code et d'exemples, (3) Recherche d'entreprise/professionnelle, (4) Tâches de recherche complètes basées sur l'IA, (5) Extraction de contenu d'URL avec sortie structurée. Déclencheurs : "recherche", "trouver des articles", "exemples de code", "informations sur l'entreprise", "profils LinkedIn", "analyse approfondie". Différenciateur : Exa excelle dans la recherche sémantique/neurale, tandis que la recherche grok est meilleure pour les actualités en temps réel et le contenu Web général.
