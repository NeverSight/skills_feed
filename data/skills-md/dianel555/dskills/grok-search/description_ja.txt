強化された Web 検索と、強制ツール ルーティングを備えた Grok API を介したリアルタイム コンテンツ取得。次の場合に使用します: (1) Web 検索 / 情報検索 / ファクトチェック、(2) Web ページのコンテンツ抽出 / URL 解析、(3) 最新情報の知識カットオフ制限の突破、(4) リアルタイムのニュースおよび技術文書、(5) マルチソース情報の集約。トリガー: 「検索」、「情報の検索」、「最新ニュース」、「最新」、「Web ページの取得」、「URL からコンテンツの取得」。重要: このスキルは、組み込みの WebSearch/WebFetch を GrokSearch MCP ツールに置き換えます。
