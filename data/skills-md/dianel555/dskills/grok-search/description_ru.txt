Улучшенный веб-поиск и получение контента в реальном времени через API Grok с принудительной маршрутизацией инструментов. Используйте, когда: (1) поиск в Интернете/поиск информации/проверка фактов, (2) извлечение содержимого веб-страницы/анализ URL-адресов, (3) превышение пределов знаний для текущей информации, (4) новости и техническая документация в режиме реального времени, (5) агрегирование информации из нескольких источников. Триггеры: «поиск», «найти информацию о», «последние новости», «текущие», «получить веб-страницу», «получить контент по URL». ВАЖНО: этот навык ЗАМЕНЯЕТ встроенный WebSearch/WebFetch инструментами GrokSearch MCP.
