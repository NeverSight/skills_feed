Diseñe componentes de interfaz de usuario reutilizables y componibles siguiendo el principio de responsabilidad única con interfaces claras, encapsulación y accesorios mínimos. Utilice esta habilidad al crear o modificar componentes frontend en cualquier marco (React, Vue, Svelte, componentes web) o archivos de componentes. Aplique cuando trabaje con archivos .jsx, .tsx, .vue, .svelte en directorios de componentes (components/, src/components/, ui/, lib/), defina interfaces y accesorios de componentes, implemente patrones de composición de componentes, administre el estado a nivel de componente, cree elementos de interfaz de usuario reutilizables (botones, formularios, tarjetas, modales), documente las API de componentes o refactorice componentes para una mejor reutilización y mantenibilidad.
