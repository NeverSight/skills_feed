Progetta componenti dell'interfaccia utente riutilizzabili e componibili seguendo il principio di responsabilità unica con interfacce chiare, incapsulamento e accessori minimi. Utilizza questa competenza durante la creazione o la modifica di componenti frontend in qualsiasi framework (React, Vue, Svelte, componenti Web) o file di componenti. Applicare quando si lavora con file .jsx, .tsx, .vue, .svelte nelle directory dei componenti (components/, src/components/, ui/, lib/), definendo proprietà e interfacce dei componenti, implementando modelli di composizione dei componenti, gestendo lo stato a livello di componente, creando elementi dell'interfaccia utente riutilizzabili (pulsanti, moduli, schede, modali), documentando le API dei componenti o refactoring dei componenti per una migliore riusabilità e manutenibilità.
