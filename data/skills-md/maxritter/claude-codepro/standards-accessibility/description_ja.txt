セマンティック HTML、適切な ARIA 属性、キーボード ナビゲーション、カラー コントラスト、スクリーン リーダーの互換性を使用して、アクセスしやすいユーザー インターフェイスを構築します。このスキルは、フロントエンド コンポーネント、HTML テンプレート、React/Vue/Svelte コンポーネント、フォーム、インタラクティブ要素、ナビゲーション メニュー、モーダル、または任意の UI 要素を作成または変更するときに使用します。 HTML ファイル、JSX/TSX コンポーネント、テンプレート ファイルの操作、キーボード アクセシビリティの確保、フォーカス管理の実装、画像への代替テキストの追加、フォーム ラベルの作成、スクリーン リーダーでのテスト、ARIA 属性の管理、カラー コントラスト比の維持、または見出し階層の構築を行う場合に適用します。 UI アクセシビリティ コンプライアンス、WCAG 標準、または包括的なデザイン パターンに関連するあらゆるタスクに使用します。
