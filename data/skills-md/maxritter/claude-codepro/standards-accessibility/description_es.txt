Cree interfaces de usuario accesibles utilizando HTML semántico, atributos ARIA adecuados, navegación por teclado, contraste de color y compatibilidad con lectores de pantalla. Utilice esta habilidad al crear o modificar componentes de interfaz, plantillas HTML, componentes de React/Vue/Svelte, formularios, elementos interactivos, menús de navegación, modales o cualquier elemento de la interfaz de usuario. Aplíquelo cuando trabaje con archivos HTML, componentes JSX/TSX, archivos de plantilla, garantice la accesibilidad del teclado, implemente la administración del enfoque, agregue texto alternativo a las imágenes, cree etiquetas de formulario, realice pruebas con lectores de pantalla, administre atributos ARIA, mantenga relaciones de contraste de color o cree jerarquías de encabezados. Úselo para cualquier tarea que implique el cumplimiento de la accesibilidad de la interfaz de usuario, los estándares WCAG o patrones de diseño inclusivos.
