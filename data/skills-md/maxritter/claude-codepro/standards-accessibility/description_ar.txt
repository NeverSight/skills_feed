أنشئ واجهات مستخدم يمكن الوصول إليها باستخدام لغة HTML الدلالية وسمات ARIA المناسبة والتنقل عبر لوحة المفاتيح وتباين الألوان وتوافق قارئ الشاشة. استخدم هذه المهارة عند إنشاء أو تعديل مكونات الواجهة الأمامية، أو قوالب HTML، أو مكونات React/Vue/Svelte، أو النماذج، أو العناصر التفاعلية، أو قوائم التنقل، أو الوسائط، أو أي عناصر واجهة المستخدم. يتم تطبيقه عند العمل مع ملفات HTML، أو مكونات JSX/TSX، أو ملفات القوالب، أو ضمان إمكانية الوصول إلى لوحة المفاتيح، أو تنفيذ إدارة التركيز، أو إضافة نص بديل إلى الصور، أو إنشاء تسميات النماذج، أو الاختبار باستخدام قارئات الشاشة، أو إدارة سمات ARIA، أو الحفاظ على نسب تباين الألوان، أو إنشاء تسلسلات هرمية للعناوين. يُستخدم في أي مهمة تتضمن الامتثال لإمكانية الوصول إلى واجهة المستخدم، أو معايير WCAG، أو أنماط التصميم الشاملة.
