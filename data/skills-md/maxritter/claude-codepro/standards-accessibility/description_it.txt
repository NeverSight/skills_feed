Crea interfacce utente accessibili utilizzando HTML semantico, attributi ARIA corretti, navigazione tramite tastiera, contrasto cromatico e compatibilità con lo screen reader. Utilizza questa competenza durante la creazione o la modifica di componenti frontend, modelli HTML, componenti React/Vue/Svelte, moduli, elementi interattivi, menu di navigazione, modali o qualsiasi elemento dell'interfaccia utente. Applicare quando si lavora con file HTML, componenti JSX/TSX, file modello, garantendo l'accessibilità da tastiera, implementando la gestione del focus, aggiungendo testo alternativo alle immagini, creando etichette di moduli, testando con lettori di schermo, gestendo attributi ARIA, mantenendo i rapporti di contrasto dei colori o costruendo gerarchie di intestazioni. Utilizzare per qualsiasi attività che coinvolga la conformità all'accessibilità dell'interfaccia utente, gli standard WCAG o modelli di progettazione inclusivi.
