Создавайте доступные пользовательские интерфейсы, используя семантический HTML, правильные атрибуты ARIA, навигацию с помощью клавиатуры, цветовой контраст и совместимость с программой чтения с экрана. Используйте этот навык при создании или изменении компонентов внешнего интерфейса, шаблонов HTML, компонентов React/Vue/Svelte, форм, интерактивных элементов, навигационных меню, модальных окон или любых элементов пользовательского интерфейса. Применяется при работе с файлами HTML, компонентами JSX/TSX, файлами шаблонов, обеспечении доступности клавиатуры, реализации управления фокусом, добавлении замещающего текста к изображениям, создании меток форм, тестировании с помощью программ чтения с экрана, управлении атрибутами ARIA, поддержании коэффициентов цветового контраста или построении иерархии заголовков. Используйте для любых задач, связанных с соблюдением специальных возможностей пользовательского интерфейса, стандартами WCAG или инклюзивными шаблонами проектирования.
