適切な HTTP メソッド、ステータス コード、リソースベースの URL を使用して、REST 原則に従って RESTful API エンドポイントを設計および実装します。このスキルは、API エンドポイント、ルート ハンドラー、コントローラー、または API 構成ファイルを作成または変更するときに使用します。 REST API 設計、エンドポイント実装、API バージョン管理、リクエスト/レスポンス処理、HTTP メソッド ルーティング (GET、POST、PUT、PATCH、DELETE)、クエリ パラメーター フィルタリング、API レート制限、または API ルートを定義するファイル (routes.py、api.js、controllers/、endpoints/、API ドキュメント ファイルなど) を作業するときに適用します。
