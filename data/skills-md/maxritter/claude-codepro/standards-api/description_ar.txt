قم بتصميم وتنفيذ نقاط نهاية RESTful API باتباع مبادئ REST باستخدام أساليب HTTP المناسبة ورموز الحالة وعناوين URL المستندة إلى الموارد. استخدم هذه المهارة عند إنشاء أو تعديل نقاط نهاية API أو معالجات المسار أو وحدات التحكم أو ملفات تكوين API. يتم تطبيقه عند العمل على تصميم REST API، أو تطبيقات نقطة النهاية، أو إصدار API، أو معالجة الطلب/الاستجابة، أو توجيه أسلوب HTTP (GET، أو POST، أو PUT، أو PATCH، أو DELETE)، أو تصفية معلمات الاستعلام، أو تحديد معدل API، أو أي ملف يحدد مسارات API مثل Routes.py، أو api.js، أو وحدات التحكم/، أو نقاط النهاية/، أو ملفات وثائق API.
