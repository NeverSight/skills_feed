Управление задачами для агентов ИИ в контекстных окнах. Используйте, когда агентам необходимо отслеживать работу, регистрировать ход выполнения, передавать состояние и поддерживать контекст между сеансами. Включает рабочие процессы для сосредоточения внимания на одной проблеме, рабочих сеансов по нескольким проблемам и структурированной передачи обслуживания. Необходим для разработки с помощью искусственного интеллекта, когда контекстные окна сбрасываются между сеансами.
