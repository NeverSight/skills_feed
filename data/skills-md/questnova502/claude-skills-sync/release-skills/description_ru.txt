Выпуск рабочего процесса для плагина baoyu-skills. Используйте, когда пользователь говорит «выпустить», «发布», «push», «推送», «новая версия», «新版本», «bump version», «更新版本» или хочет опубликовать изменения на удаленном компьютере. Анализирует изменения с момента последнего тега, обновляет CHANGELOG (EN/CN), обновляет версию marketplace.json, фиксирует и создает тег версии. ДОЛЖЕН использоваться перед любым нажатием git с незафиксированными изменениями навыков.
