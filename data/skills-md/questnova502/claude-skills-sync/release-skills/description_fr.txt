Libérer le workflow pour le plugin baoyu-skills. À utiliser lorsque l'utilisateur dit "release", "发布", "push", "推送", "new version", "新版本", "bump version", "更新版本" ou souhaite publier des modifications sur un site distant. Analyse les modifications depuis la dernière balise, met à jour CHANGELOG (EN/CN), modifie la version Marketplace.json, valide et crée une balise de version. DOIT être utilisé avant tout git push avec des changements de compétences non validés.
