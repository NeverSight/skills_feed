Старший менеджер по качеству документации для комплексного контроля документации и проверки нормативной документации. Обеспечивает проектирование системы документооборота, контроль изменений, управление конфигурациями и контроль нормативной документации. Используется для внедрения системы документооборота, проверки нормативной документации, управления изменениями и проверки соответствия документации.
