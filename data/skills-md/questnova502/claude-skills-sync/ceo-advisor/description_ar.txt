توجيهات القيادة التنفيذية لاتخاذ القرارات الاستراتيجية، والتطوير التنظيمي، وإدارة أصحاب المصلحة. يتضمن محلل الإستراتيجية، ونمذجة السيناريوهات المالية، وأطر حوكمة مجلس الإدارة، وكتب قواعد اللعبة المتعلقة بعلاقات المستثمرين. يُستخدم عند تخطيط الإستراتيجية، أو إعداد العروض التقديمية لمجلس الإدارة، أو إدارة المستثمرين، أو تطوير الثقافة التنظيمية، أو اتخاذ القرارات التنفيذية، أو عندما يذكر المستخدم الرئيس التنفيذي، أو التخطيط الاستراتيجي، أو اجتماعات مجلس الإدارة، أو تحديثات المستثمرين، أو القيادة التنظيمية، أو الإستراتيجية التنفيذية.
