مهارة تطوير الواجهة الخلفية الشاملة لبناء أنظمة خلفية قابلة للتطوير باستخدام NodeJS وExpress وGo وPython وPostgres وGraphQL وREST APIs. يتضمن سقالات واجهة برمجة التطبيقات (API) وتحسين قاعدة البيانات وتنفيذ الأمان وضبط الأداء. يُستخدم عند تصميم واجهات برمجة التطبيقات، أو تحسين استعلامات قاعدة البيانات، أو تنفيذ منطق الأعمال، أو التعامل مع المصادقة/الترخيص، أو مراجعة التعليمات البرمجية الخلفية.
