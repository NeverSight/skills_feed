Комплексный помощник для разработчиков Agora.io, создающих приложения для взаимодействия в реальном времени. Создает готовый к использованию код для видео/голосовых вызовов, потокового вещания в реальном времени, записи в облаке, push/pull мультимедиа и агентов искусственного интеллекта. Предоставляет устранение неполадок, проверку кода и лучшие практики для веб-платформ, iOS, Android, React Native и Flutter.
