Dieser Skill bietet Anleitungen zur Zig-Speicherverwaltung. Es gewährleistet die ordnungsgemäße Verwendung von Verzögerungs-/Fehlerverzögerungsmustern, Zuweisungen und Leckerkennung. Unverzichtbar für das Schreiben von Zig-Code mit dynamischer Zuordnung, das Beheben von Speicherlecks, die Implementierung einer Ressourcenbereinigung und die Arbeit mit Allokatoren.
