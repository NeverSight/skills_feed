توفر هذه المهارة إرشادات Zig 0.15.x API ويجب استخدامها عند كتابة كود Zig أو مراجعته. فهو يضمن الاستخدام الصحيح لواجهات برمجة التطبيقات Zig 0.15، مما يمنع الأخطاء الشائعة من استخدام أنماط 0.11/0.12/0.13/0.14 قديمة. ضروري لـ ArrayList وstd.Io.Writer/Reader (Writergate) وعميل HTTP وEd25519 وJSON وواجهات برمجة تطبيقات الاستبطان.
