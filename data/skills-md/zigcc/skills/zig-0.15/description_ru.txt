Этот навык предоставляет рекомендации по API Zig 0.15.x и должен использоваться при написании или проверке кода Zig. Это обеспечивает правильное использование API-интерфейсов Zig 0.15, предотвращая распространенные ошибки, связанные с использованием устаревших шаблонов 0.11/0.12/0.13/0.14. Необходим для ArrayList, std.Io.Writer/Reader (Writergate), HTTP-клиента, Ed25519, JSON и API-интерфейсов интроспекции типов.
