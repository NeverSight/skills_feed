Cette compétence fournit des conseils sur l'API Zig 0.15.x et doit être utilisée lors de l'écriture ou de la révision du code Zig. Il garantit une utilisation correcte des API Zig 0.15, évitant ainsi les erreurs courantes liées à l'utilisation de modèles 0.11/0.12/0.13/0.14 obsolètes. Essentiel pour ArrayList, std.Io.Writer/Reader (Writergate), le client HTTP, Ed25519, JSON et les API d'introspection de type.
