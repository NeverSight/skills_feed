Questa competenza fornisce indicazioni sull'API Zig 0.15.x e deve essere utilizzata durante la scrittura o la revisione del codice Zig. Garantisce l'utilizzo corretto delle API Zig 0.15, prevenendo errori comuni derivanti dall'utilizzo di modelli 0.11/0.12/0.13/0.14 obsoleti. Essenziale per ArrayList, std.Io.Writer/Reader (Writergate), client HTTP, Ed25519, JSON e API di introspezione dei tipi.
