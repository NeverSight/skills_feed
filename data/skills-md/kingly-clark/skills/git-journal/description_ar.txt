التقط "لماذا" وراء تغييرات التعليمات البرمجية أثناء التطوير بمساعدة الذكاء الاصطناعي. ينشئ مجلات تخفيض السعر على مستوى الفرع مع الحفاظ على المنطق والمقايضات والسياق الذي قد يتم فقدانه. يُستخدم قبل الالتزامات أو العلاقات العامة أو تغييرات الملفات المتعددة. يُستخدم عند حدوث تفكير مهم في المحادثة والذي يجب الحفاظ عليه. يُستخدم عندما يطلب المستخدم توثيق سبب اتخاذ القرار. يتم استخدامه عندما يقول المستخدم "تذكر دفتر اليومية" أو "أي شيء لتدوينه" أو مطالبات مماثلة.
