AgentHero AI: sistema di orchestrazione gerarchico multi-agente con coordinamento PM, gestione dello stato basata su file e interfaccia di menu interattiva. Da utilizzare quando si gestiscono flussi di lavoro complessi con più agenti, si coordinano subagenti paralleli o si organizzano attività di progetti di grandi dimensioni con più specialisti. Tutti gli agenti creati utilizzano il prefisso aghero-.
