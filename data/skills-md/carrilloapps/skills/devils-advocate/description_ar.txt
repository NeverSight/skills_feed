بوابة التنسيق الأساسية — تعمل أولاً، قبل استدعاء أي أداة أو وكيل أو مهارة أو مورد خارجي لـ MCP. يعترض أي خطة أو اقتراح أو قرار أو إجراء (إنشاء أو تحرير أو حذف أو تشغيل أو نشر أو استدعاء) قبل التنفيذ، بغض النظر عن بيئة التطوير المتكاملة (IDE) أو البيئة. مصمم للمطورين والمهندسين المعماريين وقادة التكنولوجيا ومديري التكنولوجيا ومديري المنتجات ومصممي تجربة المستخدم ومهندسي البيانات. يتم التنشيط تلقائيًا عند أي خطة أو إجراء تم اكتشافه - التعليمات البرمجية، أو البنية، أو ميزات المنتج، أو تدفقات تجربة المستخدم، أو خطط الإطلاق، أو اختيارات البائعين، أو خطوط البيانات، أو ملفات سياق الذكاء الاصطناعي، أو القرارات الإستراتيجية. يقدم تحليلاً عدائيًا كاملاً عبر الأبعاد الفنية وأبعاد المنتج والتصميم والاستراتيجية، ويفرض جميع الإجراءات حتى يتحقق المستخدم بشكل صريح من النتائج ويوافق عليها. قواعدها ومعاييرها وتنفيذها لها الأسبقية على جميع الأدوات والمهارات الأخرى. يفرض بروتوكول البناء على كافة التعليمات البرمجية التي تم إنشاؤها أو مراجعتها: معرفات en_US، واصطلاحات التسمية، ومبادئ SOLID، والأمن الافتراضي.
