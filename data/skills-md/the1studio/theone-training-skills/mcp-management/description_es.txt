Administre servidores Model Context Protocol (MCP): descubra, analice y ejecute herramientas/solicitudes/recursos desde servidores MCP configurados. Úselo cuando trabaje con integraciones de MCP, necesite descubrir capacidades de MCP disponibles, filtrar herramientas de MCP para tareas específicas, ejecutar herramientas de MCP mediante programación, acceder a solicitudes/recursos de MCP o implementar la funcionalidad de cliente de MCP. Admite la selección inteligente de herramientas, la administración de múltiples servidores y el descubrimiento de capacidades contextualmente eficiente.
