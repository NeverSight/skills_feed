إدارة خوادم بروتوكول السياق النموذجي (MCP) - اكتشاف الأدوات/المطالبات/الموارد وتحليلها وتنفيذها من خوادم MCP التي تم تكوينها. يُستخدم عند العمل مع عمليات تكامل MCP، ويحتاج إلى اكتشاف إمكانيات MCP المتاحة، أو تصفية أدوات MCP لمهام محددة، أو تنفيذ أدوات MCP برمجيًا، أو الوصول إلى مطالبات/موارد MCP، أو تنفيذ وظائف عميل MCP. يدعم اختيار الأدوات الذكية، وإدارة الخوادم المتعددة، واكتشاف القدرات بكفاءة في السياق.
