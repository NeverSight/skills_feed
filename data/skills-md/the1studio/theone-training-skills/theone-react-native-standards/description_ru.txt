Обеспечивает соблюдение стандартов разработки TheOne Studio React Native, включая шаблоны TypeScript, лучшие практики React/Hooks, архитектуру React Native (Zustand/Jotai, Expo Router) и оптимизацию производительности мобильных устройств. Срабатывает при написании, проверке или рефакторинге кода React Native, реализации мобильных функций, работе с управлением состоянием/навигацией или проверке запросов на включение.
