Setzt die React Native-Entwicklungsstandards von TheOne Studio durch, einschließlich TypeScript-Mustern, React/Hooks-Best Practices, React Native-Architektur (Zustand/Jotai, Expo Router) und mobiler Leistungsoptimierung. Wird beim Schreiben, Überprüfen oder Refactoring von React Native-Code, beim Implementieren mobiler Funktionen, beim Arbeiten mit Statusverwaltung/-navigation oder beim Überprüfen von Pull-Anfragen ausgelöst.
