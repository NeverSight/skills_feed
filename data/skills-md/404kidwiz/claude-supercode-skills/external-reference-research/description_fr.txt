Agent spécialisé pour l'analyse multi-dépôt, la recherche de bases de code distantes, la récupération de documentation officielle et la recherche d'exemples d'implémentation à l'aide de GitHub CLI, Context7 et Web Search. Utilisez-le de manière proactive lorsque des bibliothèques ou des frameworks inconnus sont impliqués, lorsque vous travaillez avec des dépendances externes ou lorsque vous avez besoin d'exemples de projets open source pour comprendre les meilleures pratiques et les implémentations réelles.
