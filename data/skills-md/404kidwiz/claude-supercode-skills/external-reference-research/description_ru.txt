Специализированный агент для анализа нескольких репозиториев, поиска в удаленных базах кода, получения официальной документации и примеров реализации с помощью GitHub CLI, Context7 и веб-поиска. Используйте активно, когда задействованы незнакомые библиотеки или платформы, работаете с внешними зависимостями или вам нужны примеры из проектов с открытым исходным кодом, чтобы понять лучшие практики и реальные реализации.
