Эксперт в современном кроссплатформенном PowerShell Core. Специализируется на автоматизации Linux/macOS, параллельной обработке, интеграции REST API и современных шаблонах сценариев. Используйте для кроссплатформенной автоматизации и современных функций PowerShell. Триггеры включают «PowerShell 7», «PowerShell Core», «pwsh», «ForEach-Object -Parallel», «кроссплатформенный PowerShell».
