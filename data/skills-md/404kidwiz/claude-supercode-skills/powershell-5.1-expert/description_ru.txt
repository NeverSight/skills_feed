Эксперт по устаревшей версии Windows PowerShell 5.1. Специализируется на автоматизации WMI, ADSI, COM и обеспечении обратной совместимости со средами Windows Server. Используйте для автоматизации, специфичной для Windows, в устаревших системах. Триггеры включают «PowerShell 5.1», «Windows PowerShell», «WMI», «ADSI», «COM-объект», «устаревшую версию PowerShell».
