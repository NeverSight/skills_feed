Эксперт по созданию графических и TUI-интерфейсов с помощью PowerShell с использованием платформ WinForms, WPF и Console/TUI. Используйте при создании инструментов PowerShell с графическим или терминальным интерфейсом. Триггеры включают «PowerShell GUI», «WinForms», «WPF PowerShell», «PowerShell TUI», «пользовательский интерфейс терминала», «интерфейс PowerShell».
