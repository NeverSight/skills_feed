ابحث في قوائم انتظار مشكلات Drupal.org وقم بتلخيص المشكلات الفردية عبر نقاط نهاية Drupal.org api-d7. يُستخدم لفرز الأخطاء أو تصحيح الأخطاء عن طريق التحقق من المشكلات الموجودة قبل التصحيح، أو التصفية حسب الحالة/الأولوية/الفئة/الإصدار/المكون/العلامة، أو تلخيص سلاسل المشكلات، أو استخراج التصحيحات/الملفات الحديثة إلى JSON أو Markdown مع احترام قيود Drupal.org API (خيط مفرد، مخزن مؤقتًا، محدود المعدل، للقراءة فقط).
