ОБЯЗАТЕЛЬНО, когда пользователь упоминает модуль Drupal + ошибку/ошибку/проблему - даже без трассировки стека. Запуск по: (1) «В модуле <имя_модуля> есть ошибка/ошибка/проблема», (2) «Acquia/Pantheon/Platform.sh» + проблема с модулем, (3) любое имя дополнительного модуля (метатег, веб-форма, mcp, абзацы и т. д.) + описание проблемы. Ищет drupal.org ПЕРЕД написанием любого патча. НЕ только для вкладов вышестоящих разработчиков — используйте для ВСЕХ локальных исправлений вклада/ядра.
