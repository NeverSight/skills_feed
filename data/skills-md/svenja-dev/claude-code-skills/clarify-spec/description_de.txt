Wird bei vagen Aufträgen automatisch aktiv. Lieber einmal zu oft nachfragen als falsch umsetzen. Erkennungsmerkmale: Auftrag unter 25 Wörtern, keine konkreten Dateinamen/Pfade, vage Verben (optimieren, fixen, verbessern …), unsichere Sprache (vielleicht, schnell …), fehlende Erfolgskriterien, relative Begriffe ohne Kontext. Ausgabe: strukturiertes JSON für prompt-architect. Escape: „mach einfach“, „keine Rückfragen“, „entscheide selbst“ überspringt Klärung.
