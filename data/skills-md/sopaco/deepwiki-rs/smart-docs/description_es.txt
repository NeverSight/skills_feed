Generador de documentación base de código integral impulsado por IA. Analiza la estructura del proyecto, identifica patrones arquitectónicos, crea diagramas de modelo C4 y genera documentación técnica profesional. Úselo cuando los usuarios necesiten documentar bases de código, comprender la arquitectura del software, crear especificaciones técnicas o generar guías para desarrolladores. Soporta todos los lenguajes de programación. Alternativa a Litho/deepwiki-rs que utiliza la suscripción a Claude Code sin costos de API externos.
