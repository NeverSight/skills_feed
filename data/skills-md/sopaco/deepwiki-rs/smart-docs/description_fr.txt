Générateur de documentation de base de code complet alimenté par l'IA. Analyse la structure du projet, identifie les modèles d'architecture, crée des diagrammes de modèle C4 et génère une documentation technique professionnelle. À utiliser lorsque les utilisateurs ont besoin de documenter les bases de code, de comprendre l'architecture logicielle, de créer des spécifications techniques ou de générer des guides de développement. Prend en charge tous les langages de programmation. Alternative à Litho/deepwiki-rs qui utilise l'abonnement Claude Code sans frais d'API externe.
