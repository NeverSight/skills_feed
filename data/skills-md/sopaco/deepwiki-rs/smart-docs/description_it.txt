Generatore di documentazione di base di codice completo basato sull'intelligenza artificiale. Analizza la struttura del progetto, identifica i modelli di architettura, crea diagrammi del modello C4 e genera documentazione tecnica professionale. Utilizzalo quando gli utenti hanno bisogno di documentare basi di codice, comprendere l'architettura software, creare specifiche tecniche o generare guide per gli sviluppatori. Supporta tutti i linguaggi di programmazione. Alternativa a Litho/deepwiki-rs che utilizza l'abbonamento a Claude Code senza costi API esterni.
