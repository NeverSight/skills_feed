Комплексный генератор документации по кодовой базе на базе искусственного интеллекта. Анализирует структуру проекта, определяет шаблоны архитектуры, создает схемы моделей C4 и создает профессиональную техническую документацию. Используйте, когда пользователям необходимо документировать базы кода, понимать архитектуру программного обеспечения, создавать технические спецификации или создавать руководства для разработчиков. Поддерживает все языки программирования. Альтернатива Litho/deepwiki-rs, использующая подписку на Claude Code без затрат на внешний API.
