shadcn/ui 아키텍처, CVA 변형, Radix UI 기본 요소 및 Tailwind CSS를 사용하는 30개 이상의 프로덕션 지원 구성 요소가 포함된 포괄적인 React 구성 요소 라이브러리입니다. 사용자가 (1) 최신 패턴으로 React UI 구성 요소 생성, (2) 일관된 디자인으로 완전한 구성 요소 시스템 구축, (3) 액세스 가능하고 반응성이 뛰어난 다크 모드 지원 구성 요소 구현, (4) React Hook Form 통합으로 양식 구성 요소 생성, (5) 테이블, 카드, 차트와 같은 데이터 표시 구성 요소 생성 또는 (6) 탐색, 레이아웃 또는 피드백 구성 요소 구축이 필요할 때 사용합니다. 수동으로 코딩하는 데 각각 20~45분이 소요되는 사용자 정의 가능한 구성 요소를 즉시 생성할 수 있습니다.
