إرشادات تطوير الواجهة الأمامية لتطبيقات React/TypeScript. الأنماط الحديثة بما في ذلك Suspense، والتحميل البطيء، وuseSuspenseQuery، وتنظيم الملفات مع دليل الميزات، وأفضل ممارسات التصميم، والتوجيه، وتحسين الأداء، وأفضل ممارسات TypeScript. يُستخدم عند إنشاء المكونات أو الصفحات أو الميزات أو جلب البيانات أو التصميم أو التوجيه أو العمل باستخدام كود الواجهة الأمامية.
