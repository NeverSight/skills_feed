Linee guida per lo sviluppo del frontend per applicazioni React/TypeScript. Modelli moderni tra cui Suspense, caricamento lento, useSuspenseQuery, organizzazione dei file con directory delle funzionalità, best practice per lo stile, routing, ottimizzazione delle prestazioni e best practice per TypeScript. Da utilizzare durante la creazione di componenti, pagine, funzionalità, recupero di dati, stili, routing o utilizzo del codice frontend.
