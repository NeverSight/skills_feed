Erstellen Sie eine umfassende Dokumentation im Stil einer wissenschaftlichen Forschung für abgeschlossene Codierungsprojekte. Verwenden Sie diese Option, wenn der Benutzer eine Projektdokumentation, eine technische Aufschlüsselung, eine Studienarbeit oder ein Vorlesungsdokument anfordert oder alles über ein gerade erstelltes Projekt erfahren möchte. Zu den Auslösern gehören Formulierungen wie „Dokumentieren Sie dieses Projekt“, „Erstellen Sie eine Studienarbeit“, „Erklären Sie alles, was wir getan haben“, „Schreiben Sie die vollständige Aufschlüsselung auf“, „Wissenschaftliche Dokumentation“ oder „Ich möchte aus diesem Projekt lernen“. Erstellt formelle Word-Dokumente (.docx) mit akademischer Struktur, anfängerfreundlichen Erklärungen und umfassender Codeanalyse.
