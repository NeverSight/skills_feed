Générez une documentation complète de type recherche scientifique pour les projets de codage terminés. À utiliser lorsque l'utilisateur demande une documentation de projet, une ventilation technique, un document d'étude, un document de cours ou souhaite tout comprendre sur un projet qu'il vient de construire. Les déclencheurs incluent des expressions telles que « documenter ce projet », « créer un document d'étude », « expliquer tout ce que nous avons fait », « rédiger le détail complet », « documentation scientifique » ou « Je veux apprendre de ce projet ». Produit des documents Word formels (.docx) avec une structure académique, des explications adaptées aux débutants et une analyse exhaustive du code.
