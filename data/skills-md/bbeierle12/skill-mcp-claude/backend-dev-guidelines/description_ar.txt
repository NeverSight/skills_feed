إرشادات تطوير الواجهة الخلفية لتطبيقات Node.js/Express/TypeScript. بنية الطبقات (المسارات → وحدات التحكم → الخدمات → المستودعات)، ومعالجة الأخطاء، والتحقق من الصحة، وأنماط البرامج الوسيطة، والوصول إلى قاعدة البيانات، والاختبار. يُستخدم عند إنشاء المسارات أو نقاط النهاية أو واجهات برمجة التطبيقات أو وحدات التحكم أو الخدمات أو المستودعات أو البرامج الوسيطة أو العمل مع التعليمات البرمجية الخلفية.
