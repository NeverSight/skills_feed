Directives de développement backend pour les applications Node.js/Express/TypeScript. Architecture en couches (Routes → Contrôleurs → Services → Référentiels), gestion des erreurs, validation, modèles middleware, accès à la base de données et tests. À utiliser lors de la création de routes, de points de terminaison, d'API, de contrôleurs, de services, de référentiels, de middleware ou lors de l'utilisation de code backend.
