Настройка и архитектура EffectComposer для конвейеров постобработки Three.js. Используйте при настройке многопроходного рендеринга, комбинировании эффектов, создании пользовательских проходов, управлении целями рендеринга или создании повторно используемых стеков эффектов. Базовые навыки для всех работ по постобработке.
