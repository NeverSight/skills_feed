Строительные системы для конкретных платформ для мобильных устройств, виртуальной реальности и доступности. Используйте при реализации сенсорного управления для создания игр, пространственного ввода в виртуальной реальности, обратной связи для дальтоников или кроссплатформенной строительной механики. Охватывает адаптацию ввода и инклюзивные шаблоны проектирования.
