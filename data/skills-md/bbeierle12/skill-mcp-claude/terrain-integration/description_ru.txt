Системы взаимодействия с местностью для строительных игр Three.js. Используйте при реализации обработки склонов, анкеровки фундамента (шаблон Вальхейма), модификации ландшафта, автоматического выравнивания или создания столбов/опор. Интегрируется со структурной физикой для обеспечения устойчивости на земле.
