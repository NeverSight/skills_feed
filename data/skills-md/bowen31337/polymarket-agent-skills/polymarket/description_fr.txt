SDK Python complet pour la plateforme de marchés de prédiction Polymarket. À utiliser lorsque :
(1) Trading sur Polymarket - passation, annulation, gestion des ordres
(2) Rechercher des marchés - découvrir des événements, analyser les prix, suivre les positions
(3) Création de robots de trading - Streaming WebSocket, flux de données en temps réel
(4) Gestion des fonds - dépôts, retraits via Bridge API
(5) Toute tâche impliquant des marchés de prédiction ou des API Polymarket
