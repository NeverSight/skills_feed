قم بإنشاء وتحرير الصور باستخدام Gemini API (Nano Banana). استخدم هذه المهارة عند إنشاء صور من المطالبات النصية، أو تحرير الصور الموجودة، أو تطبيق عمليات نقل الأنماط، أو إنشاء شعارات مع نص، أو إنشاء ملصقات، أو نماذج بالحجم الطبيعي للمنتج، أو أي مهمة لإنشاء/معالجة الصور. يدعم تحويل النص إلى صورة، وتحرير الصور، والتحسين متعدد المنعطفات، والتكوين من صور مرجعية متعددة.
