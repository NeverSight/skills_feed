Автоматически ищите, очищайте и переписывайте технический контент для создания научно-популярных статей на китайском языке, подходящих для официальных учетных записей WeChat. Охватывает крупные модели искусственного интеллекта, инструменты с открытым исходным кодом GitHub и технические темы. Используйте, когда пользователь говорит: «Напишите статью о XXX», «Изучите XXX для меня» или «Создайте статью в WeChat».
