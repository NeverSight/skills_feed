AWS CloudWatch осуществляет мониторинг журналов, показателей, сигналов тревоги и информационных панелей. Используйте при настройке мониторинга, создании сигналов тревоги, запросе журналов с помощью Insights, настройке фильтров метрик, создании информационных панелей или устранении неполадок приложений.
