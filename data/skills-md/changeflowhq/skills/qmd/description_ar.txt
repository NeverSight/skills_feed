محرك بحث دلالي محلي لقواعد المعرفة الخاصة بتخفيض السعر باستخدام qmd (tobi/qmd).
فهرسة ملفات تخفيض السعر باستخدام البحث عن الكلمات الرئيسية BM25، والتضمين المتجه للبحث الدلالي،
والاستعلامات الهجين المعاد ترتيبها. الفهرسة التلقائية لتعديلات الملف عبر خطافات Claude Code و
ينعش التضمينات بين عشية وضحاها عبر Launchd. استخدم عند البحث في مستندات المشروع،
قواعد المعرفة أو ملاحظات الاجتماعات أو أي مجموعة تخفيضات مفهرسة.
