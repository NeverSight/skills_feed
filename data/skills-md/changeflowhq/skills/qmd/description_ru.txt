Локальная семантическая поисковая система для баз знаний с уценкой с использованием qmd (tobi/qmd).
Индексирует файлы уценки с поиском по ключевым словам BM25, векторными вложениями для семантического поиска,
и гибридные запросы с переранжированием. Автоматическая индексация при редактировании файлов с помощью перехватчиков Claude Code и
обновляет встраивания в ночное время через launchd. Используйте при поиске документации проекта,
базы знаний, заметки о встречах или любая индексированная коллекция уценок.
