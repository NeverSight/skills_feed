Automatisation invisible de Chrome pour le web scraping via CDP. À utiliser lorsque WebFetch échoue ou est bloqué (403, 429, Cloudflare, protection contre les robots, pages rendues JS). Lance votre véritable installation Chrome complètement cachée, envoie des commandes via le protocole Chrome DevTools. Les sites voient un navigateur normal avec de vraies extensions – aucune automatisation détectable. Apprend quels domaines bloquent et passe directement en mode furtif pour les demandes futures. Gère également le remplissage de formulaires, les clics, les captures d’écran et la récupération de contenu dynamique.
