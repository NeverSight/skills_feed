WebFetch가 실패하거나 차단될 때 사용합니다(403, 429, Cloudflare, 봇 보호, JS 렌더링 페이지). 완전히 숨겨진 실제 Chrome 설치를 시작하고 Chrome DevTools 프로토콜을 통해 명령을 보냅니다. 사이트에는 실제 확장 기능이 포함된 일반 브라우저가 표시되며 감지할 수 있는 자동화는 없습니다. 어떤 도메인이 차단되는지 알아보고 향후 요청 시 곧바로 스텔스로 건너뜁니다. 또한 양식 채우기, 클릭, 스크린샷 및 동적 콘텐츠 스크랩을 처리합니다.
