CDP を介した Web スクレイピングのための非表示の Chrome 自動化。 WebFetch が失敗するかブロックされた場合に使用します (403、429、Cloudflare、ボット保護、JS レンダリングされたページ)。実際の Chrome インストールを完全に非表示にして起動し、Chrome DevTools プロトコル経由でコマンドを送信します。サイトでは、実際の拡張機能を備えた通常のブラウザが表示されます。検出可能な自動化はありません。どのドメインがブロックしているかを学習し、今後のリクエストをステルスに直接スキップします。フォームの入力、クリック、スクリーンショット、および動的コンテンツのスクレイピングも処理します。
