Analysieren Sie eine Codebasis und erstellen Sie eine umfassende Dokumentation einschließlich Architektur, Komponenten, Schnittstellen, Arbeitsabläufen und Abhängigkeiten. Erstellt eine KI-optimierte Wissensdatenbank (index.md) und kann in AGENTS.md, README.md oder CONTRIBUTING.md konsolidiert werden. Verwenden Sie diese Option, wenn der Benutzer eine Codebasis dokumentieren, AGENTS.md erstellen, die Systemarchitektur verstehen, Entwicklerdokumentation erstellen oder „die Codebasis zusammenfassen“ möchte.
