Analysez une base de code et générez une documentation complète comprenant l'architecture, les composants, les interfaces, les flux de travail et les dépendances. Crée une base de connaissances optimisée pour l'IA (index.md) et peut être consolidée dans AGENTS.md, README.md ou CONTRIBUTING.md. À utiliser lorsque l'utilisateur souhaite documenter une base de code, créer AGENTS.md, comprendre l'architecture du système, générer de la documentation pour les développeurs ou demande de « résumer la base de code ».
