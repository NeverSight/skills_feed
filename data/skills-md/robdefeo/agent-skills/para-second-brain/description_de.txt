Verwenden Sie diese Fähigkeit, wenn der Benutzer ein zweites Gehirn nach der PARA-Methode organisieren, klassifizieren oder verwalten möchte. Zu den Auslösern gehören die Frage, wo etwas abgelegt werden soll, das Unterscheiden von Projekten von Bereichen, das Verarbeiten eines Posteingangs, das Einrichten eines neuen Projekts, das Abschließen oder Archivieren eines Projekts, das Durchführen einer monatlichen Überprüfung, das Überprüfen der Systemstruktur oder das Auffinden veralteter/verwaister Inhalte.
