استخدم هذه المهارة عندما يريد المستخدم تنظيم أو تصنيف أو صيانة الدماغ الثاني بطريقة PARA. تتضمن المشغلات السؤال عن مكان تقديم شيء ما، أو التمييز بين المشاريع والمناطق، أو معالجة البريد الوارد، أو إعداد مشروع جديد، أو إكمال المشروع أو أرشفته، أو إجراء مراجعة شهرية، أو التحقق من صحة بنية النظام، أو البحث عن محتوى قديم/معزول.
