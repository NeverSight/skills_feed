Actualice los archivos de fórmulas de Homebrew con las últimas URL de versión de GitHub y sumas de comprobación SHA256. Esta habilidad debe usarse cuando el usuario desea actualizar una fórmula de tap Homebrew (archivo .rb) con una nueva versión de una versión de GitHub. Se activa con frases como "actualizar fórmula", "actualizar homebrew", "nueva versión", "versión mejorada" o cuando apunta a un archivo de fórmula .rb que necesita actualizaciones de versión.
