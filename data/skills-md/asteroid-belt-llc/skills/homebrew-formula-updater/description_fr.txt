Mettez à jour les fichiers de formule Homebrew avec les dernières URL de version GitHub et les sommes de contrôle SHA256. Cette compétence doit être utilisée lorsque l'utilisateur souhaite mettre à jour une formule de tap Homebrew (fichier .rb) avec une nouvelle version d'une version de GitHub. Se déclenche sur des expressions telles que « formule de mise à jour », « mise à jour homebrew », « nouvelle version », « version de remplacement » ou lorsque vous pointez vers un fichier de formule .rb qui nécessite des mises à jour de version.
