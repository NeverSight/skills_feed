Обновите файлы формул Homebrew, добавив URL-адреса последней версии GitHub и контрольные суммы SHA256. Этот навык следует использовать, когда пользователь хочет обновить формулу крана Homebrew (файл .rb) новой версией из выпуска GitHub. Запускается по таким фразам, как «формула обновления», «обновление доморощенного», «новый выпуск», «увеличенная версия» или при указании на файл формулы .rb, который требует обновления версии.
