Aggiorna i file delle formule Homebrew con gli URL di versione GitHub più recenti e i checksum SHA256. Questa abilità dovrebbe essere utilizzata quando l'utente desidera aggiornare una formula di tap Homebrew (file .rb) con una nuova versione da una versione di GitHub. Si attiva su frasi come "aggiorna formula", "aggiorna homebrew", "nuova versione", "versione bump" o quando si punta a un file formula .rb che necessita di aggiornamenti di versione.
