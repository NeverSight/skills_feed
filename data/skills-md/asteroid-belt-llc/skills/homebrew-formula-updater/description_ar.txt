قم بتحديث ملفات صيغة Homebrew بأحدث عناوين URL لإصدار GitHub والمجاميع الاختبارية SHA256. يجب استخدام هذه المهارة عندما يريد المستخدم تحديث صيغة Homebrew Tap (ملف .rb) بإصدار جديد من إصدار GitHub. يتم تشغيل عبارات مثل "صيغة التحديث"، أو "تحديث البيرة المنزلية"، أو "الإصدار الجديد"، أو "إصدار نتوء"، أو عند الإشارة إلى ملف صيغة ‎.rb الذي يحتاج إلى تحديثات الإصدار.
