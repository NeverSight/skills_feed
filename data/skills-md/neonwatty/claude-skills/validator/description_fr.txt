Exécute des opérations de validation locales pour n’importe quel projet. Détecte automatiquement le langage/le framework et exécute le peluchage, la vérification de type, les tests et la détection de code mort. Utilisez-le lorsque l'utilisateur dit « valider », « exécuter la validation », « vérifier le code », « pelucher et tester » ou « exécuter des vérifications ».
