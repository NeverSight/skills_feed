Создает тесты Playwright, которые обнаруживают анти-шаблоны iOS/мобильного UX в CI. Используйте это, когда пользователь хочет добавить автоматические проверки шаблонов UX для веб-приложений, ориентированных на мобильные устройства/iOS. Создает тесты, которые проваливаются при обнаружении гамбургер-меню, FAB, небольших сенсорных целей или других анти-шаблонов.
