Genera pruebas de Playwright que detectan antipatrones de UX de iOS/móvil en CI. Úselo cuando el usuario quiera agregar verificaciones automatizadas de patrones de UX para aplicaciones web orientadas a dispositivos móviles/iOS. Crea pruebas que FALLAN cuando se detectan menús de hamburguesas, FAB, pequeños objetivos táctiles u otros antipatrones.
