Génère, crée ou met à jour des fichiers de flux de travail iOS pour tester les applications Web dans Safari sur le simulateur iOS. Utilisez-le lorsque l'utilisateur dit "générer des workflows iOS", "créer des workflows iOS", "mettre à jour les workflows iOS" ou "itérer sur les workflows iOS". Explorez en profondeur la base de code de l'application Web pour découvrir toutes les fonctionnalités, pages et interactions destinées aux utilisateurs. Crée des flux de travail numérotés complets avec des sous-étapes qui couvrent toute l'expérience utilisateur lorsqu'ils sont affichés sur iOS Safari.
