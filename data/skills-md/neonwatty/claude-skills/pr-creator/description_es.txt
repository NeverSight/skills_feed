Crea una solicitud de extracción a partir de los cambios actuales, monitorea GitHub CI y depura cualquier falla hasta que se apruebe el CI. Utilice esto cuando el usuario diga "crear pr", "hacer un pr", "abrir solicitud de extracción", "enviar pr" o "pr para estos cambios". NO se fusiona: se detiene cuando CI pasa y proporciona el enlace PR.
