Практическое руководство по созданию безопасных, учитывающих синтаксис команд CLI srgn для поиска и преобразования исходного кода. Используйте, когда пользователи запрашивают команды srgn, рефакторинг области действия (комментарии/строки документации/импорты/функции), перезапись нескольких файлов с помощью --glob, использование специального запроса Tree-Sitter или проверки в стиле CI с --fail-any/--fail-none.
