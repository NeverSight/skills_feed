Guía práctica para crear comandos CLI srgn seguros y con sintaxis para la búsqueda y transformación de código fuente. Úselo cuando los usuarios soliciten comandos srgn, refactores de alcance (comentarios/cadenas de documentos/importaciones/funciones), reescrituras de múltiples archivos con --glob, uso personalizado de consultas de cuidado de árboles o comprobaciones de estilo CI con --fail-any/--fail-none.
