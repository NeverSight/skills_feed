دليل عملي لبناء أوامر srgn CLI الآمنة والمدركة لبناء الجملة للبحث عن التعليمات البرمجية المصدر وتحويلها. يُستخدم عندما يطلب المستخدمون أوامر srgn، أو إعادة بناء النطاق (التعليقات/المستندات/الواردات/الوظائف)، أو إعادة كتابة الملفات المتعددة باستخدام --glob، أو استخدام استعلام الشجرة المخصص، أو عمليات التحقق من نمط CI باستخدام --fail-any/-fail-none.
