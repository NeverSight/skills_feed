Créez et gérez des applications de chat React basées sur une interface utilisateur avec une configuration fiable, une sélection d'exécution, un câblage LangGraph, une intégration de l'interface utilisateur de l'outil et des flux de travail de mise à niveau. À utiliser lorsque les tâches impliquent explicitement des dépendances ou des API `assistant-ui`, y compris les commandes CLI `assistant-ui` (`create/init/add/update/upgrade/codemod`), les fournisseurs/environnements d'exécution `@assistant-ui/*`, `AssistantRuntimeProvider`, les primitives Thread/Composer, la persistance dans le cloud ou le comportement de rendu des outils. Ne pas utiliser pour le travail de chat React générique, les tâches LangGraph back-end uniquement ou le travail d'interface utilisateur sans assistant-ui. Si l'invite indique explicitement sans/non/pas d'assistant-ui, ne déclenchez pas cette compétence.
