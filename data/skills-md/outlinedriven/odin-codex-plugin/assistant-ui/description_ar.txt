قم ببناء وصيانة تطبيقات الدردشة React المستندة إلى واجهة المستخدم المساعدة من خلال الإعداد الموثوق به، واختيار وقت التشغيل، وأسلاك LangGraph، وتكامل واجهة مستخدم الأداة، وترقية سير العمل. يُستخدم عندما تتضمن المهام بشكل صريح تبعيات `assistant-ui` أو واجهات برمجة التطبيقات، بما في ذلك أوامر `assistant-ui` CLI (`create/init/add/update/upgrade/codemod`)، أو `@assistant-ui/*` مقدمي الخدمات/أوقات التشغيل، أو `AssistantRuntimeProvider`، أو أساسيات Thread/Composer، أو استمرارية السحابة، أو سلوك عرض الأداة. لا تستخدمه في أعمال محادثة React العامة، أو مهام LangGraph للواجهة الخلفية فقط، أو أعمال واجهة المستخدم غير المساعدة. إذا كانت المطالبة تقول صراحةً بدون/لا/ليست واجهة المستخدم المساعدة، فلا تقم بتشغيل هذه المهارة.
