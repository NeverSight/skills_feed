Cree y mantenga aplicaciones de chat React basadas en interfaz de usuario asistente con configuración confiable, selección de tiempo de ejecución, cableado LangGraph, integración de interfaz de usuario de herramientas y flujos de trabajo de actualización. Úselo cuando las tareas impliquen explícitamente dependencias o API de `assistant-ui`, incluidos los comandos CLI de `assistant-ui` (`create/init/add/update/upgrade/codemod`), proveedores/tiempos de ejecución `@assistant-ui/*`, `AssistantRuntimeProvider`, primitivas de Thread/Composer, persistencia en la nube o comportamiento de representación de herramientas. No lo use para trabajo de chat genérico de React, tareas LangGraph solo de backend o trabajo de interfaz de usuario sin asistente. Si el mensaje dice explícitamente sin/no/no asistente-ui, no active esta habilidad.
