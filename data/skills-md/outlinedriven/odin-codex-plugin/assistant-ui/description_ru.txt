Создавайте и обслуживайте чат-приложения React на основе пользовательского интерфейса Assistant с надежной настройкой, выбором среды выполнения, подключением LangGraph, интеграцией пользовательского интерфейса инструментов и рабочими процессами обновления. Используйте, когда задачи явно включают в себя зависимости или API-интерфейсы `assistant-ui`, включая команды CLI `assistant-ui` (`create/init/add/update/upgrade/codemod`), `@assistant-ui/*` провайдеры/среды выполнения, `AssistantRuntimeProvider`, примитивы Thread/Composer, постоянство облака или поведение отрисовки инструментов. Не используйте для общей работы в чате React, задач LangGraph только для серверной части или работы с пользовательским интерфейсом без помощника. Если в подсказке явно указано «без/нет/нет помощника-UI», не активируйте этот навык.
