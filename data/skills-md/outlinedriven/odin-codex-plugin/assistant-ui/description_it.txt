Crea e mantieni app di chat React basate sull'interfaccia utente dell'assistente con configurazione affidabile, selezione del runtime, cablaggio LangGraph, integrazione dell'interfaccia utente dello strumento e flussi di lavoro di aggiornamento. Da utilizzare quando le attività coinvolgono esplicitamente dipendenze o API `assistant-ui`, inclusi comandi CLI di `assistant-ui` (`create/init/add/update/upgrade/codemod`), provider/runtime di `@assistant-ui/*`, `AssistantRuntimeProvider`, primitive di thread/compositore, persistenza del cloud o comportamento di rendering dello strumento. Non utilizzare per lavori generici di chat React, attività LangGraph solo backend o lavori dell'interfaccia utente non dell'assistente. Se il prompt dice esplicitamente senza/no/non assistente-ui, non attivare questa abilità.
