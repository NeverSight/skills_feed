استخدم هذه المهارة عندما يريد المستخدم سرد نتائج أمان التطبيق وتصفيتها، واكتشاف التطبيقات والإصدارات، وإدارة التطبيقات باستخدام Fortify Software Security Center (SSC). تتضمن المشغلات: أي ذكر لـ "SSC"، و"قائمة الثغرات الأمنية"، و"قائمة التطبيقات"، وطلبات مماثلة تشير إلى التفاعل مع Fortify SSC لمهام أمان التطبيق. OpenText Application Security هو الاسم الجديد لـ Fortify Software Security Center.
