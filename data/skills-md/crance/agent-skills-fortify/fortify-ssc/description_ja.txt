ユーザーがアプリケーションのセキュリティ検出結果をリストおよびフィルタリングし、アプリケーションとバージョンを検出し、Fortify Software Security Center (SSC) を使用してアプリケーションを管理する場合は常にこのスキルを使用します。トリガーには、「SSC」、「脆弱性のリスト」、「アプリケーションのリスト」に関する言及、およびアプリケーション セキュリティ タスクのための Fortify SSC との対話を示す同様のリクエストが含まれます。 OpenText Application Security は、Fortify Software Security Center の新しい名前です。
