Устанавливайте, обнаруживайте, удаляйте и обновляйте навыки агентов с помощью интерфейса командной строки навыков npx. Используйте, когда вас попросят установить навык, добавить навык из репозитория, найти или выполнить поиск навыков, составить список установленных навыков, удалить или удалить навык, обновить навыки или проверить наличие обновлений. Срабатывает при: «установить навык X», «добавить навык Y», «найти навыки для Z», «какие навыки доступны», «удалить навык», «обновить мои навыки», «проверить наличие обновлений навыков», «поиск навыка, который делает X».
