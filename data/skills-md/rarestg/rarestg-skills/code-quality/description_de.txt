Richten Sie beim Gerüstbau oder beim Starten eines neuen Projekts Formatierung, Linting, Importsortierung, Typprüfung und Pre-Commit-Hooks ein. Verwenden Sie diese Fähigkeit immer dann, wenn Sie ein neues Projekt erstellen, ein Repo initialisieren, ein Gerüst für eine App erstellen oder wenn der Benutzer Linting/Formatierung zu einem vorhandenen Projekt hinzufügen möchte. Auslöser bei: „Neues Projekt“, „Gerüst“, „Init“, „Linting einrichten“, „Formatierer hinzufügen“, „Pre-Commit-Hooks hinzufügen“, „Biom konfigurieren“, „Ruff konfigurieren“. Das Ziel besteht darin, vom ersten Tag an Codequalitätstools zu etablieren, damit Probleme schrittweise erkannt werden und nicht später in einer mühsamen Massenbehebung.
