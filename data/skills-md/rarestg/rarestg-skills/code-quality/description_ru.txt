Настройте форматирование, проверку, сортировку импорта, проверку типов и перехватчики предварительной фиксации при создании шаблонов или запуске нового проекта. Используйте этот навык всякий раз, когда создаете новый проект, инициализируете репозиторий, создаете приложение или когда пользователь просит добавить проверку/форматирование в существующий проект. Запускается по: «новый проект», «эшафот», «инициализация», «настройка линтинга», «добавление форматтера», «добавление перехватчиков предварительной фиксации», «настройка биома», «настройка ерша». Цель состоит в том, чтобы с самого первого дня внедрить инструменты обеспечения качества кода, чтобы проблемы выявлялись постепенно, а не в результате мучительного массового исправления позже.
