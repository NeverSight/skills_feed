Configure formato, linting, clasificación de importaciones, verificación de tipos y ganchos de confirmación previa al realizar scaffolding o iniciar un nuevo proyecto. Utilice esta habilidad siempre que cree un nuevo proyecto, inicialice un repositorio, cree una aplicación o cuando el usuario solicite agregar linting/formato a un proyecto existente. Activadores en: "nuevo proyecto", "andamio", "init", "configurar linting", "agregar formateador", "agregar ganchos de confirmación previa", "configurar bioma", "configurar ruff". El objetivo es establecer herramientas de calidad del código desde el primer día para que los problemas se detecten de forma incremental, y no en una dolorosa solución masiva posterior.
