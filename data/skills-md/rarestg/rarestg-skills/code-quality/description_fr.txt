Configurez le formatage, le peluchage, le tri des importations, la vérification du type et les hooks de pré-validation lors de l'échafaudage ou du démarrage d'un nouveau projet. Utilisez cette compétence lors de la création d'un nouveau projet, de l'initialisation d'un dépôt, de l'échafaudage d'une application ou lorsque l'utilisateur demande d'ajouter du peluchage/formattage à un projet existant. Déclencheurs sur : "nouveau projet", "échafaudage", "init", "configurer le linting", "ajouter un formateur", "ajouter des hooks de pré-validation", "configurer le biome", "configurer ruff". L'objectif est d'établir des outils de qualité de code dès le premier jour afin que les problèmes soient détectés progressivement, et non dans le cadre d'une correction groupée douloureuse plus tard.
