قم بإنشاء مقتطفات وأنماط كود Arduino/C++ المضمنة حسب الطلب لـ UNO/ESP32/RP2040. يُستخدم عندما يطلب المستخدمون كود Arduino لأجهزة الاستشعار، أو المحركات، أو بروتوكولات الاتصال، أو أجهزة الحالة، أو أجهزة ضبط الوقت غير المحظورة، أو تسجيل البيانات، أو تجريد الأجهزة. يُنشئ تعليمات برمجية جاهزة للإنتاج مع إدارة مناسبة للذاكرة وأنماط التوقيت والتحسين الخاص باللوحة. يدعم DHT22، BME280، الأزرار، I2C/SPI، EEPROM، بطاقات SD، WiFi، والأجهزة الطرفية الشائعة.
