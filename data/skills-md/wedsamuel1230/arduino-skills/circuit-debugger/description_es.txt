Guía sistemática de depuración de hardware para circuitos Arduino/ESP32/RP2040. Úselo cuando el usuario informe: el circuito no funciona, los componentes se calientan, no hay energía, fallas intermitentes, comportamiento inesperado, el sensor no responde, el LED no se enciende, el motor no gira. Guía a través de comprobaciones de energía, pruebas de continuidad, rastreo de señales y aislamiento de componentes utilizando técnicas de multímetro.
