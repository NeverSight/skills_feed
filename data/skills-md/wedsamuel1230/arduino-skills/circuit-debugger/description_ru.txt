Систематическое руководство по отладке оборудования для схем Arduino/ESP32/RP2040. Используйте, когда пользователь сообщает: схема не работает, компоненты нагреваются, нет питания, периодические сбои, неожиданное поведение, датчик не отвечает, светодиод не горит, двигатель не вращается. Помогает выполнить проверку питания, проверку целостности, отслеживание сигнала и изоляцию компонентов с помощью мультиметра.
