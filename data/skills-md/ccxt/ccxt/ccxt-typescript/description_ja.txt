TypeScript および JavaScript 開発者 (Node.js およびブラウザ) のための CCXT 暗号通貨交換ライブラリ。 REST API (標準) と WebSocket API (リアルタイム) の両方をカバーします。 CCXT のインストール、取引所への接続、市場データの取得、注文、ライブ ティッカー/オーダーブックのストリーミング、認証の処理、エラーの管理を支援します。 TypeScript/JavaScript プロジェクト、取引ボット、アービトラージ システム、またはポートフォリオ管理ツールで暗号通貨取引所を操作する場合に使用します。 REST と WebSocket の両方の例が含まれています。
