Библиотека обмена криптовалютой CCXT для разработчиков TypeScript и JavaScript (Node.js и браузер). Охватывает как REST API (стандартный), так и WebSocket API (реального времени). Помогает установить CCXT, подключаться к биржам, получать рыночные данные, размещать заказы, транслировать тикеры/книги заказов в реальном времени, обрабатывать аутентификацию и управлять ошибками. Используйте при работе с криптобиржами в проектах TypeScript/JavaScript, торговых ботах, арбитражных системах или инструментах управления портфелем. Включает примеры REST и WebSocket.
