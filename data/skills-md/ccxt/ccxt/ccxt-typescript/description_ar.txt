مكتبة تبادل العملات المشفرة CCXT لمطوري TypeScript وJavaScript (Node.js والمتصفح). يغطي كلاً من REST API (قياسي) وWebSocket API (في الوقت الفعلي). يساعد في تثبيت CCXT، والاتصال بالبورصات، وجلب بيانات السوق، وتقديم الطلبات، وبث المؤشرات/سجلات الطلبات المباشرة، والتعامل مع المصادقة، وإدارة الأخطاء. يُستخدم عند العمل مع عمليات تبادل العملات المشفرة في مشاريع TypeScript/JavaScript، أو روبوتات التداول، أو أنظمة المراجحة، أو أدوات إدارة المحافظ. يتضمن أمثلة REST وWebSocket.
