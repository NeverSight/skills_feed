Biblioteca de intercambio de criptomonedas CCXT para desarrolladores de TypeScript y JavaScript (Node.js y navegador). Cubre tanto la API REST (estándar) como la API WebSocket (en tiempo real). Ayuda a instalar CCXT, conectarse a intercambios, obtener datos de mercado, realizar pedidos, transmitir tickers/libros de pedidos en vivo, manejar la autenticación y administrar errores. Úselo cuando trabaje con intercambios de cifrado en proyectos TypeScript/JavaScript, robots comerciales, sistemas de arbitraje o herramientas de administración de carteras. Incluye ejemplos de REST y WebSocket.
