Библиотека обмена криптовалютой CCXT для разработчиков Python. Охватывает как REST API (стандартный), так и WebSocket API (реального времени). Помогает установить CCXT, подключаться к биржам, получать рыночные данные, размещать заказы, транслировать тикеры/книги заказов в реальном времени, обрабатывать аутентификацию и управлять ошибками в Python. Используйте при работе с криптобиржами в проектах Python, торговых ботах, анализе данных или управлении портфелем. Поддерживает как синхронное, так и асинхронное (asyncio) использование.
