مكتبة تبادل العملات المشفرة CCXT لمطوري Python. يغطي كلاً من REST API (قياسي) وWebSocket API (في الوقت الفعلي). يساعد في تثبيت CCXT، والاتصال بالبورصات، وجلب بيانات السوق، وتقديم الطلبات، ودفق المؤشرات/سجلات الطلبات المباشرة، والتعامل مع المصادقة، وإدارة الأخطاء في Python. يُستخدم عند العمل مع بورصات العملات المشفرة في مشاريع Python أو روبوتات التداول أو تحليل البيانات أو إدارة المحافظ. يدعم استخدام المزامنة وغير المتزامنة (غير المتزامنة).
