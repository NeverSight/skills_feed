Biblioteca de intercambio de criptomonedas CCXT para desarrolladores de Python. Cubre tanto la API REST (estándar) como la API WebSocket (en tiempo real). Ayuda a instalar CCXT, conectarse a intercambios, obtener datos de mercado, realizar pedidos, transmitir tickers/libros de pedidos en vivo, manejar la autenticación y administrar errores en Python. Úselo cuando trabaje con intercambios de cifrado en proyectos de Python, robots comerciales, análisis de datos o gestión de carteras. Admite el uso tanto sincronizado como asíncrono (asyncio).
