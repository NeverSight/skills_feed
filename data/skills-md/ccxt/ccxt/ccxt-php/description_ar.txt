مكتبة تبادل العملات المشفرة CCXT لمطوري PHP. يغطي كلاً من REST API (قياسي) وWebSocket API (في الوقت الفعلي). يساعد في تثبيت CCXT، والاتصال بالبورصات، وجلب بيانات السوق، وتقديم الطلبات، وبث المؤشرات/دفاتر الطلبات المباشرة، والتعامل مع المصادقة، وإدارة الأخطاء في PHP 8.1+. يُستخدم عند العمل مع عمليات تبادل العملات المشفرة في مشاريع PHP أو روبوتات التداول أو تطبيقات الويب. يدعم استخدام المزامنة وغير المتزامنة (ReactPHP).
