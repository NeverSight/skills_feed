Biblioteca de intercambio de criptomonedas CCXT para desarrolladores de PHP. Cubre tanto la API REST (estándar) como la API WebSocket (en tiempo real). Ayuda a instalar CCXT, conectarse a intercambios, obtener datos de mercado, realizar pedidos, transmitir tickers/libros de pedidos en vivo, manejar la autenticación y administrar errores en PHP 8.1+. Úselo cuando trabaje con intercambios de cifrado en proyectos PHP, robots comerciales o aplicaciones web. Admite el uso tanto sincronizado como asíncrono (ReactPHP).
