Библиотека обмена криптовалютой CCXT для разработчиков PHP. Охватывает как REST API (стандартный), так и WebSocket API (реального времени). Помогает установить CCXT, подключаться к биржам, получать рыночные данные, размещать заказы, транслировать тикеры/книги заказов в реальном времени, обрабатывать аутентификацию и управлять ошибками в PHP 8.1+. Используйте при работе с криптобиржами в проектах PHP, торговых ботах или веб-приложениях. Поддерживает как синхронное, так и асинхронное использование (ReactPHP).
