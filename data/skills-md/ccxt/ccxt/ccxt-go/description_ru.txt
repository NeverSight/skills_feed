Библиотека обмена криптовалют CCXT для разработчиков Go. Охватывает как REST API (стандартный), так и WebSocket API (реального времени). Помогает установить CCXT, подключаться к биржам, получать рыночные данные, размещать заказы, транслировать тикеры/книги заказов в реальном времени, обрабатывать аутентификацию и управлять ошибками в проектах Go. Используйте при работе с криптобиржами в приложениях Go, микросервисах или торговых системах.
