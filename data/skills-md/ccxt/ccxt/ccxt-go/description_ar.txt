مكتبة تبادل العملات المشفرة CCXT لمطوري Go. يغطي كلاً من REST API (قياسي) وWebSocket API (في الوقت الفعلي). يساعد في تثبيت CCXT، والاتصال بالبورصات، وجلب بيانات السوق، وتقديم الطلبات، وبث المؤشرات/دفاتر الطلبات المباشرة، والتعامل مع المصادقة، وإدارة الأخطاء في مشاريع Go. يُستخدم عند العمل مع بورصات العملات المشفرة في تطبيقات Go أو الخدمات الصغيرة أو أنظمة التداول.
