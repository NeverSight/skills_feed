Biblioteca de intercambio de criptomonedas CCXT para desarrolladores de C# y .NET. Cubre tanto la API REST (estándar) como la API WebSocket (en tiempo real). Ayuda a instalar CCXT, conectarse a intercambios, obtener datos de mercado, realizar pedidos, transmitir tickers/libros de pedidos en vivo, manejar la autenticación y administrar errores en proyectos .NET. Úselo cuando trabaje con intercambios de cifrado en aplicaciones C#, sistemas comerciales o software financiero. Admite .NET Standard 2.0+.
