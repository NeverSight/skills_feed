Библиотека обмена криптовалютой CCXT для разработчиков C# и .NET. Охватывает как REST API (стандартный), так и WebSocket API (реального времени). Помогает установить CCXT, подключаться к биржам, получать рыночные данные, размещать заказы, транслировать тикеры/книги заказов в реальном времени, обрабатывать аутентификацию и управлять ошибками в проектах .NET. Используйте при работе с криптобиржами в приложениях C#, торговых системах или финансовом программном обеспечении. Поддерживает .NET Standard 2.0+.
