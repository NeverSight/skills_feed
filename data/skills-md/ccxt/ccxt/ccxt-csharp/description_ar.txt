مكتبة تبادل العملات المشفرة CCXT لمطوري C# و.NET. يغطي كلاً من REST API (قياسي) وWebSocket API (في الوقت الفعلي). يساعد في تثبيت CCXT، والاتصال بالبورصات، وجلب بيانات السوق، وتقديم الطلبات، ودفق المؤشرات/دفاتر الطلبات المباشرة، والتعامل مع المصادقة، وإدارة الأخطاء في مشاريع .NET. يُستخدم عند العمل مع عمليات تبادل العملات المشفرة في تطبيقات C# أو أنظمة التداول أو البرامج المالية. يدعم .NET Standard 2.0+.
