This skill provides mandatory guidelines for UI/UX design, state management, networking, and API integration that must be followed when developing apps with the Flutter framework. This skill does not offer optional information; it delivers essential instructions for Flutter app development that must be strictly adhered to. Whenever a developer makes requests related to design, UI, UX, design effects, state management, routing, networking, API integration, comic design, Comic-related requests, chat, FCM, push notifications, messages, or notifications, this skill must be used and the work must be carried out exactly as the skill specifies. At the top of each skill document there is a Workflow that must always be followed. That Workflow must be strictly followed.
