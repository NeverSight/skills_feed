À utiliser lors de la création d'une interface utilisateur stylisée avec des composants @nuxt/ui v4 (bouton, modal, formulaire, tableau, etc.) - fournit des composants prêts à l'emploi avec la thématique Tailwind Variants. Utilisez la compétence vue pour les modèles de composants bruts, reka-ui pour les primitives sans tête.
