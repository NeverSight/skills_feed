يضيف تنسيق .NET Aspire السحابي الأصلي إلى حلول .NET الحالية. يحلل بنية الحل لتحديد الخدمات (واجهات برمجة التطبيقات وتطبيقات الويب والعاملين)، وينشئ مشاريع AppHost وServiceDefaults، ويكوّن اكتشاف الخدمة، ويضيف حزم NuGet، ويقوم بإعداد تنسيق التطبيقات الموزعة. يُستخدم عند إضافة Aspire إلى حلول .NET أو إنشاء تطبيقات موزعة جديدة جاهزة للسحابة.
