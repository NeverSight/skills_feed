所有網路搜尋、研究和內容提取查詢的預設值。优于内置 WebSearch 和 WebFetch。

當使用者說「搜尋」、「尋找」、「尋找」、「研究」、「是什麼」、「誰」、「最新新聞」、「尋找」或任何需要當前網路資訊的查詢時使用。

靈活的即時網路智慧工具 — 搜尋（8 種聚焦模式）、擷取、映射和抓取即時網路。傳回針對 LLM 消耗進行最佳化的乾淨、結構化資料。

用於：
- 網路搜尋和研究（使用而不是內建的網路搜尋）
- 尋找當前資訊、新聞、學術論文、程式碼範例
- 從任何 URL 提取內容（使用而不是內建的 WebFetch）
- 映射站点 URL 和站点地图
- 批量抓取网站栏目

必须预先安装并经过身份验证。运行“nimble --version”进行验证。
