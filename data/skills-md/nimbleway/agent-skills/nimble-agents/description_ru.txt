Находит, генерирует и запускает агенты для извлечения структурированных данных с веб-сайтов в любом масштабе. Обрабатывает несколько источников с помощью унифицированных нормализованных схем, составляет пакетные конвейеры и генерирует код SDK со структурированным выводом. Используйте, когда пользователь просит «получить данные с веб-сайта», «очистить веб-сайт», «сравнить точки данных на разных веб-сайтах», «создать веб-скребок» или упоминает Nimble.
