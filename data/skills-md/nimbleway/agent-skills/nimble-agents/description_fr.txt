Recherche, génère et exécute des agents pour extraire des données structurées de sites Web à grande échelle. Gère le multi-source avec des schémas normalisés unifiés, la composition de pipelines par lots et la génération de leur code SDK avec une sortie structurée. À utiliser lorsque l'utilisateur demande « d'obtenir des données d'un site Web », de « gratter un site Web », de « comparer des points de données sur plusieurs sites Web », de « générer un grattoir Web » ou mentionne Nimble.
