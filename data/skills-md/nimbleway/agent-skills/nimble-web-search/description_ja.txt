Nimble Search API を活用したリアルタイム Web インテリジェンス。 8 つの特殊なフォーカス モード (一般、コーディング、ニュース、学術、ショッピング、ソーシャル、地域、位置) を使用してインテリジェントな Web 検索を実行します。このスキルは、Web の検索、最新情報の検索、URL の発見、トピックの調査、または最新データの収集が必要な場合に、リアルタイムの検索結果を提供します。情報の検索、最近のニュースの検索、学術論文の検索、コーディング例の検索、ショッピング結果の検索、ソーシャル メディア投稿の発見、トピックの調査、最新のリアルタイム データの取得などの場合に使用します。
