Этот навык анализирует код для улучшения качества проектирования по 8 измерениям: именование, художественная гимнастика объектов, связь и связность, неизменность, целостность домена, система типов, простота и производительность. Обеспечивает тщательный анализ, основанный на фактических данных, путем: (1) сначала понимания потока кода с помощью протокола анализа реализации, (2) систематической оценки каждого измерения с использованием определенных критериев, (3) предоставления практических результатов со ссылками на файлы и строки. Запускается, когда пользователи запрашивают: анализ кода, обзор дизайна, возможности рефакторинга, оценку качества кода, оценку архитектуры.
