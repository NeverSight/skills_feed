Convertissez les spécifications OpenAPI v3 en serveurs MCP à l'aide de HAPI CLI. Prend en charge le déploiement local, Docker et Cloudflare Workers. Code zéro requis. À utiliser lorsque vous devez exposer les fonctionnalités de l'API en tant qu'outils MCP ou déployer des services basés sur l'API sur plusieurs plates-formes d'hébergement.
