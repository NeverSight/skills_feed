Преобразуйте спецификации OpenAPI v3 в серверы MCP с помощью HAPI CLI. Поддерживает локальное развертывание рабочих процессов Docker и Cloudflare. Требуется нулевой код. Используйте, когда вам нужно предоставить функциональность API в виде инструментов MCP или развернуть службы на основе API на нескольких платформах хостинга.
