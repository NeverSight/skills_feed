Generiert Storytelling-fokussierte konventionelle Commits-Nachrichten mit Jira-Kontextintegration, schreibt dann Änderungen fest und pusht sie. Verwenden Sie diese Option, wenn der Benutzer „commit“, „git commit“ sagt oder nach einem Commit für Änderungen fragt, einen Commit erstellen möchte oder wenn die Arbeit abgeschlossen und zum Commit bereit ist.
