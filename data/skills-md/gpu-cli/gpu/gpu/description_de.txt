GPU-CLI-Assistent – ​​Führen Sie NVIDIA-GPU-Workloads von Ihrem Mac aus. Verwenden Sie es, wenn Benutzer:
- Sie möchten ML/KI auf Cloud-GPUs ausführen („Modell trainieren“, „Inferenz ausführen“, „Bilder generieren“).
- Benötigen Sie Hilfe beim Erstellen von GPU-Projekten (ComfyUI, vLLM, Training, Inferenz)
- Es liegen GPU-CLI-Fehler vor (OOM-, Synchronisierungs-, Verbindungs-, Modellladeprobleme)
- Sie möchten die GPU-Kosten optimieren oder die richtige GPU auswählen
- Benötigen Sie Hilfe bei der gpu.jsonc-Konfiguration
- Fragen Sie nach GPU-CLI-Befehlen (GPU-Ausführung, GPU-Nutzung, GPU-Status usw.)
