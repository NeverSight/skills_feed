Assistant GPU CLI - exécutez les charges de travail NVIDIA GPU depuis votre Mac. À utiliser lorsque les utilisateurs :
- Vous souhaitez exécuter du ML/AI sur des GPU cloud ("entraîner un modèle", "exécuter une inférence", "générer des images")
- Besoin d'aide pour créer des projets GPU (ComfyUI, vLLM, formation, inférence)
- Avoir des erreurs GPU CLI (MOO, synchronisation, connexion, problèmes de chargement de modèle)
- Vous souhaitez optimiser les coûts GPU ou choisir le bon GPU
- Besoin d'aide avec la configuration de gpu.jsonc
- Renseignez-vous sur les commandes GPU CLI (exécution du GPU, utilisation du GPU, état du GPU, etc.)
