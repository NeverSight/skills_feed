GPU CLI アシスタント - Mac から NVIDIA GPU ワークロードを実行します。ユーザーが次の場合に使用します。
- クラウド GPU で ML/AI を実行したい (「モデルをトレーニングする」、「推論を実行する」、「画像を生成する」)
- GPU プロジェクト (ComfyUI、vLLM、トレーニング、推論) の作成にサポートが必要
- GPU CLI エラーがある (OOM、同期、接続、モデル読み込みの問題)
- GPU コストを最適化したい、または適切な GPU を選択したい
- gpu.jsonc 設定についてサポートが必要です
- GPU CLI コマンド (GPU 実行、GPU 使用、GPU ステータスなど) について質問します。
