مساعد GPU CLI - قم بتشغيل أحمال عمل NVIDIA GPU من جهاز Mac الخاص بك. يُستخدم عندما يقوم المستخدمون بما يلي:
- ترغب في تشغيل ML/AI على وحدات معالجة الرسومات السحابية ("تدريب نموذج"، "تشغيل الاستدلال"، "إنشاء صور")
- بحاجة إلى مساعدة في إنشاء مشاريع GPU (ComfyUI، vLLM، التدريب، الاستدلال)
- وجود أخطاء GPU CLI (OOM، والمزامنة، والاتصال، ومشكلات تحميل النموذج)
- هل ترغب في تحسين تكاليف GPU أو اختيار GPU المناسب
- بحاجة إلى مساعدة في تكوين gpu.jsonc
- اسأل عن أوامر GPU CLI (تشغيل GPU، واستخدام GPU، وحالة GPU، وما إلى ذلك)
