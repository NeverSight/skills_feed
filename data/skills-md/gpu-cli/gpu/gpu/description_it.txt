Assistente CLI GPU: esegui carichi di lavoro GPU NVIDIA dal tuo Mac. Utilizzare quando gli utenti:
- Vuoi eseguire ML/AI su GPU cloud ("addestra un modello", "esegui inferenza", "genera immagini")
- Hai bisogno di aiuto per creare progetti GPU (ComfyUI, vLLM, formazione, inferenza)
- Presentano errori CLI della GPU (OOM, sincronizzazione, connessione, problemi di caricamento del modello)
- Vuoi ottimizzare i costi della GPU o scegliere la GPU giusta
- Hai bisogno di aiuto con la configurazione di gpu.jsonc
- Chiedi informazioni sui comandi CLI della GPU (esecuzione della GPU, utilizzo della GPU, stato della GPU, ecc.)
