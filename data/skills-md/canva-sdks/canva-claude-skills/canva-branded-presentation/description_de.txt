Erstellen Sie markengerechte Canva-Präsentationen anhand einer Gliederung oder eines Briefings. Verwenden Sie diese Option, wenn der Benutzer eine Markenpräsentation erstellen, eine markenbezogene Präsentation erstellen, eine Gliederung in Folien umwandeln oder aus einem Briefing eine Präsentation erstellen möchte. Die Eingabe kann Text direkt in der Nachricht, ein namentlich genannter Verweis auf ein Canva-Dokument oder ein Canva-Design-Link (z. B. https://www.canva.com/design/...) sein.
