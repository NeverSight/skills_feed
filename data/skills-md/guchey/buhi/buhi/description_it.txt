Notifiche sonore per le attività di Claude Code e conferme degli utenti. Utilizza questa abilità quando l'utente desidera (1) testare il suono di notifica eseguendo /buhi, (2) ascoltare il suono di notifica manualmente o (3) ottenere assistenza con la configurazione della riproduzione automatica del suono al completamento dell'attività e alle richieste di conferma dell'utente utilizzando il sistema di hook di Claude Code (Stop hook e PreToolUse con il matcher AskUserQuestion). La competenza rileva automaticamente se è installata a livello globale o locale e configura il file settings.json appropriato con percorsi assoluti.
