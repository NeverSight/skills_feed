Звуковые уведомления о задачах Claude Code и подтверждениях пользователей. Используйте этот навык, когда пользователь хочет (1) протестировать звук уведомления, запустив /buhi, (2) услышать звук уведомления вручную или (3) получить помощь в настройке автоматического воспроизведения звука при завершении задачи и запросах подтверждения пользователя с помощью системы перехватчиков Claude Code (перехват Stop и перехват PreToolUse с сопоставлением AskUserQuestion). Этот навык автоматически определяет, установлен ли он глобально или локально, и настраивает соответствующий файл settings.json с абсолютными путями.
