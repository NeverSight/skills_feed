Notificaciones audibles para tareas de Claude Code y confirmaciones de usuario. Utilice esta habilidad cuando el usuario desee (1) probar el sonido de notificación ejecutando /buhi, (2) escuchar el sonido de notificación manualmente u (3) obtener ayuda para configurar la reproducción automática de sonido al completar la tarea y las solicitudes de confirmación del usuario utilizando el sistema de enlace de Claude Code (enganche Stop y enlace PreToolUse con comparador AskUserQuestion). La habilidad detecta automáticamente si está instalada global o localmente y configura el archivo settings.json apropiado con rutas absolutas.
