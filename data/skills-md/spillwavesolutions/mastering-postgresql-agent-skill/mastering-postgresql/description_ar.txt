تطوير PostgreSQL لـ Python مع البحث عن النص الكامل (tsvector، tsquery، BM25 عبر pg_search)، وتشابه المتجهات (pgvector مع HNSW/IVFFlat)، وفهرسة JSONB والمصفوفات، ونشر الإنتاج. يُستخدم عند إنشاء ميزات البحث، أو تخزين عمليات تضمين الذكاء الاصطناعي، أو الاستعلام عن تشابه المتجهات، أو تحسين فهارس PostgreSQL، أو النشر إلى AWS RDS/Aurora، أو GCP Cloud SQL/AlloyDB، أو Azure. يغطي psycopg2، وpsycopg3، وasyncpg، وتكامل SQLAlchemy، وإعداد تطوير Docker، واستراتيجيات اختيار الفهرس. المشغلات هي استخدام "بحث PostgreSQL"، و"pgvector"، و"BM25 postgres"، و"JSONB Index"، و"psycopg"، و"asyncpg"، و"PostgreSQL Docker"، و"AlloyDB Vector". لا يغطي - إدارة DBA (النسخ الاحتياطي، النسخ المتماثل، المستخدمين)، MySQL/MongoDB/Redis، نظرية تصميم المخطط، الإجراءات المخزنة.
