全文検索 (pg_search を使用した tsvector、tsquery、BM25)、ベクトル類似性 (HNSW/IVFFlat を使用した pgvector)、JSONB および配列のインデックス付け、および運用環境のデプロイメントを備えた Python 用の PostgreSQL 開発。検索機能の作成、AI 埋め込みの保存、ベクトル類似性のクエリ、PostgreSQL インデックスの最適化、または AWS RDS/Aurora、GCP Cloud SQL/AlloyDB、または Azure へのデプロイ時に使用します。 psycopg2、psycopg3、asyncpg、SQLAlchemy 統合、Docker 開発セットアップ、インデックス選択戦略について説明します。トリガーは、「PostgreSQL 検索」、「pgvector」、「BM25 postgres」、「JSONB インデックス」、「psycopg」、「asyncpg」、「PostgreSQL Docker」、「AlloyDB ベクター」を使用します。 DBA 管理 (バックアップ、レプリケーション、ユーザー)、MySQL/MongoDB/Redis、スキーマ設計理論、ストアド プロシージャは対象外です。
