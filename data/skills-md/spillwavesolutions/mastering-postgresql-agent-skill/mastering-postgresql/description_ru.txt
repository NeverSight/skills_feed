Разработка PostgreSQL для Python с полнотекстовым поиском (tsvector, tsquery, BM25 через pg_search), сходством векторов (pgvector с HNSW/IVFFlat), JSONB и индексацией массивов, а также производственным развертыванием. Используйте при создании функций поиска, хранении внедрений ИИ, запросе сходства векторов, оптимизации индексов PostgreSQL или развертывании в AWS RDS/Aurora, GCP Cloud SQL/AlloyDB или Azure. Охватывает psycopg2, psycopg3, asyncpg, интеграцию SQLAlchemy, настройку разработки Docker и стратегии выбора индекса. Триггеры: используйте «Поиск PostgreSQL», «pgvector», «BM25 postgres», «Индекс JSONB», «psycopg», «asyncpg», «PostgreSQL Docker», «Вектор AlloyDB». НЕ охватывает - администрирование администратора баз данных (резервное копирование, репликация, пользователи), MySQL/MongoDB/Redis, теорию проектирования схем, хранимые процедуры.
