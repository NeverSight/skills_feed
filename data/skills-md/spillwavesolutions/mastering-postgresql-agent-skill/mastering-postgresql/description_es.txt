Desarrollo de PostgreSQL para Python con búsqueda de texto completo (tsvector, tsquery, BM25 vía pg_search), similitud de vectores (pgvector con HNSW/IVFFlat), indexación de matrices y JSONB, e implementación en producción. Úselo al crear funciones de búsqueda, almacenar incrustaciones de IA, consultar similitudes de vectores, optimizar índices de PostgreSQL o implementar en AWS RDS/Aurora, GCP Cloud SQL/AlloyDB o Azure. Cubre psycopg2, psycopg3, asyncpg, integración de SQLAlchemy, configuración de desarrollo de Docker y estrategias de selección de índices. Los activadores son el uso de "búsqueda PostgreSQL", "pgvector", "BM25 postgres", "índice JSONB", "psycopg", "asyncpg", "PostgreSQL Docker", "vector AlloyDB". NO cubre: administración de DBA (copia de seguridad, replicación, usuarios), MySQL/MongoDB/Redis, teoría del diseño de esquemas, procedimientos almacenados.
