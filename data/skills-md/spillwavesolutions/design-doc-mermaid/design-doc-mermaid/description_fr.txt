Créez des diagrammes Mermaid (activité, déploiement, séquence, architecture) à partir de descriptions textuelles ou de code source. À utiliser lorsqu'on vous demande de « créer un diagramme », « générer une sirène », « une architecture de document », « coder en diagramme », « créer un document de conception » ou « convertir du code en diagramme ». Prend en charge le chargement de guides hiérarchiques à la demande, les symboles sémantiques Unicode et les utilitaires Python pour l'extraction de diagrammes et la conversion d'images.
