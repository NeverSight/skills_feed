Используйте, когда вас просят «создать диаграмму», «создать русалку», «архитектуру документа», «код в диаграмму», «создать проектную документацию» или «преобразовать код в диаграмму». Поддерживает иерархическую загрузку руководств по требованию, семантические символы Unicode и утилиты Python для извлечения диаграмм и преобразования изображений.
