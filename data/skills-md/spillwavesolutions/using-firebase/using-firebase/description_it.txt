Guida completa allo sviluppo di Firebase per applicazioni ospitate su GCP. Copre le operazioni del database Firestore (CRUD, query, transazioni, modellazione dei dati), Cloud Functions (1a e 2a generazione, TypeScript e Python, tutti i tipi di trigger), operazioni CLI Firebase, configurazione dell'emulatore e persistenza dei dati, regole di sicurezza (Firestore e Storage), integrazione dell'autenticazione, configurazione dell'hosting e integrazione del servizio GCP. Da utilizzare quando si lavora con progetti Firebase, si distribuisce Cloud Functions, si eseguono query su Firestore, si configurano trigger (Firestore, Auth, Storage, HTTP, Callable, Scheduled, Pub/Sub), si gestiscono regole di sicurezza, si configurano riscritture/intestazioni di hosting, si gestiscono segreti o si integra con servizi GCP come BigQuery e Cloud Tasks. I trigger includono Firebase, Firestore, funzioni cloud, funzioni Firebase, hosting Firebase, autenticazione Firebase, archiviazione Firebase, emulatore Firebase, distribuzione Firebase, init Firebase, regole Firebase, funzione richiamabile, funzione pianificata, onDocumentCreated, onRequest, onCall, onSchedule.
