Guía completa de desarrollo de Firebase para aplicaciones alojadas en GCP. Cubre las operaciones de la base de datos de Firestore (CRUD, consultas, transacciones, modelado de datos), funciones de nube (primera y segunda generación, TypeScript y Python, todos los tipos de desencadenadores), operaciones de Firebase CLI, configuración del emulador y persistencia de datos, reglas de seguridad (Firestore y almacenamiento), integración de autenticación, configuración de alojamiento e integración de servicios GCP. Úselo cuando trabaje con proyectos de Firebase, implemente Cloud Functions, consulte Firestore, configure activadores (Firestore, Auth, Storage, HTTP, Callable, Scheduled, Pub/Sub), administre reglas de seguridad, configure reescrituras/encabezados de alojamiento, administre secretos o integre con servicios de GCP como BigQuery y Cloud Tasks. Los activadores incluyen firebase, firestore, funciones en la nube, funciones de firebase, alojamiento de firebase, autenticación de firebase, almacenamiento de firebase, emulador de firebase, implementación de firebase, init de firebase, reglas de firebase, función invocable, función programada, onDocumentCreated, onRequest, onCall, onSchedule.
