Комплексное руководство по разработке Firebase для приложений, размещенных на GCP. Охватывает операции с базой данных Firestore (CRUD, запросы, транзакции, моделирование данных), облачные функции (1-го и 2-го поколения, TypeScript и Python, все типы триггеров), операции CLI Firebase, настройку эмулятора и сохранение данных, правила безопасности (Firestore и хранилище), интеграцию аутентификации, конфигурацию хостинга и интеграцию службы GCP. Используйте при работе с проектами Firebase, развертывании облачных функций, запросе Firestore, настройке триггеров (Firestore, Auth, Storage, HTTP, Callable, Scheduled, Pub/Sub), управлении правилами безопасности, настройке перезаписи/заголовков хостинга, управлении секретами или интеграции со службами GCP, такими как BigQuery и Cloud Tasks. Триггеры включают Firebase, Firestore, облачные функции, функции Firebase, хостинг Firebase, аутентификацию Firebase, хранилище Firebase, эмулятор Firebase, развертывание Firebase, инициализацию Firebase, правила Firebase, вызываемую функцию, запланированную функцию, onDocumentCreated, onRequest, onCall, onSchedule.
