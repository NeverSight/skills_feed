GCP でホストされるアプリケーションのための包括的な Firebase 開発ガイダンス。 Firestore データベース操作 (CRUD、クエリ、トランザクション、データ モデリング)、Cloud Functions (第 1 世代と第 2 世代、TypeScript と Python、すべてのトリガー タイプ)、Firebase CLI 操作、エミュレータのセットアップとデータの永続化、セキュリティ ルール (Firestore とストレージ)、認証の統合、ホスティング構成、GCP サービスの統合をカバーします。 Firebase プロジェクトの操作、Cloud Functions のデプロイ、Firestore のクエリ、トリガーの設定 (Firestore、Auth、Storage、HTTP、Callable、Scheduled、Pub/Sub)、セキュリティ ルールの管理、ホスティング リライト/ヘッダーの構成、シークレットの管理、BigQuery や Cloud Tasks などの GCP サービスとの統合を行う場合に使用します。トリガーには、firebase、firestore、クラウド関数、firebase 関数、firebase ホスティング、firebase auth、firebase storage、firebase エミュレータ、firebasedeploy、firebase init、firebase ルール、呼び出し可能関数、スケジュールされた関数、onDocumentCreated、onRequest、onCall、onSchedule が含まれます。
