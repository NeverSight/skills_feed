Conseils complets de développement Firebase pour les applications hébergées par GCP. Couvre les opérations de base de données Firestore (CRUD, requêtes, transactions, modélisation de données), les fonctions Cloud (1re et 2e génération, TypeScript et Python, tous les types de déclencheurs), les opérations Firebase CLI, la configuration de l'émulateur et la persistance des données, les règles de sécurité (Firestore et stockage), l'intégration de l'authentification, la configuration de l'hébergement et l'intégration du service GCP. À utiliser lorsque vous travaillez avec des projets Firebase, déployez Cloud Functions, interrogez Firestore, configurez des déclencheurs (Firestore, Auth, Storage, HTTP, Callable, Scheduled, Pub/Sub), gérez les règles de sécurité, configurez les réécritures/en-têtes d'hébergement, gérez les secrets ou intégrez des services GCP comme BigQuery et Cloud Tasks. Les déclencheurs incluent Firebase, Firestore, les fonctions cloud, les fonctions Firebase, l'hébergement Firebase, l'authentification Firebase, le stockage Firebase, l'émulateur Firebase, le déploiement Firebase, l'initialisation Firebase, les règles Firebase, la fonction appelable, la fonction planifiée, onDocumentCreated, onRequest, onCall, onSchedule.
