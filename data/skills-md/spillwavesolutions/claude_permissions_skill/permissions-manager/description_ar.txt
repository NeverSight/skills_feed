تكوين أذونات Claude Code عبر اللغة الطبيعية لأدوات CLI (git، وgcloud، وaws، وkubectl، وmaven، وgradle، وnpm، وdocker)، وأنواع المشاريع (Rust، وJava، وTypeScript، وPython)، وأنماط الملفات. استخدمه عندما يُطلب منك "تكوين الأذونات" أو "تمكين git" أو "السماح بعامل الإرساء" أو "إعداد أذونات المشروع" أو "تطبيق ملف تعريف الأذونات" أو "جعل الملفات قابلة للتحرير". يكتشف تلقائيًا أنواع المشاريع ويبحث عن أدوات غير معروفة.
