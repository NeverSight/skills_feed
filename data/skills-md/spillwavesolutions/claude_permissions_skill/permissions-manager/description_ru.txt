Настраивает разрешения Claude Code через естественный язык для инструментов CLI (git, gcloud, aws, kubectl, maven, gradle, npm, docker), типов проектов (Rust, Java, TypeScript, Python) и шаблонов файлов. Используйте, когда вас просят «настроить разрешения», «включить git», «разрешить Docker», «настроить разрешения проекта», «применить профиль разрешений» или «сделать файлы редактируемыми». Автоматически определяет типы проектов и исследует неизвестные инструменты.
