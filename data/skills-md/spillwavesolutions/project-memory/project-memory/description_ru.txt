Используйте этот навык, когда вас просят «настроить память проекта», «отследить наши решения», «зафиксировать исправление ошибки», «обновить память проекта» или «инициализировать систему памяти». Настраивает CLAUDE.md и AGENTS.md для поддержки памяти при использовании различных инструментов кодирования AI.
