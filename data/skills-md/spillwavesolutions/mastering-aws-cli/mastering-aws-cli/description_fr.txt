Référence rapide AWS CLI v2 pour les développeurs expérimentés. Couvre le calcul (Lambda, ECS, EKS), le stockage (S3, DynamoDB, Aurora), la mise en réseau (VPC, tunneling SSM), la sécurité (IAM, Secrets Manager) et GitHub Actions CI/CD. À utiliser lorsqu'on vous demande d'« écrire des commandes AWS », « déboguer l'accès AWS », « configurer des rôles entre comptes », « configurer AWS cli », « assumer un rôle », « opérations du compartiment S3 » ou « déployer sur ECS ».
