Краткий справочник по AWS CLI v2 для опытных разработчиков. Охватывает вычисления (Lambda, ECS, EKS), хранилище (S3, DynamoDB, Aurora), сети (VPC, туннелирование SSM), безопасность (IAM, Secrets Manager) и GitHub Actions CI/CD. Используйте, когда вас просят «написать команды aws», «отладить доступ к aws», «настроить роли для нескольких учетных записей», «настроить aws cli», «присвоить роль», «операции с сегментом S3» или «развернуть в ECS».
