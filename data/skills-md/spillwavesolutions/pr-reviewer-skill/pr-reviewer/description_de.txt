Umfassende Fähigkeiten zur Überprüfung des GitHub Pull Request-Codes. Wird verwendet, wenn Sie aufgefordert werden, „diese PR zu überprüfen“, „Code zu überprüfen“, „Pull-Anfrage zu überprüfen“, „diese PR zu überprüfen“ oder wenn eine GitHub-PR-URL bereitgestellt wird. Ruft PR-Metadaten, Unterschiede, Kommentare, Commits und verwandte Probleme mithilfe der GH-CLI ab. Erstellt einen organisierten Überprüfungsarbeitsbereich, analysiert Code anhand branchenüblicher Kriterien und fügt optional Inline-Kommentare zum PR hinzu.
