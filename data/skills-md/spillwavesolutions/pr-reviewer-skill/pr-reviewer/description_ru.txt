Комплексные навыки проверки кода GitHub Pull Request. Используйте, когда вас просят «просмотреть этот PR», «проверку кода», «проверить запрос на включение», «проверить этот PR» или когда указан URL-адрес PR GitHub. Извлекает метаданные PR, различия, комментарии, фиксации и связанные с ними проблемы с помощью gh CLI. Создает организованное рабочее пространство проверки, анализирует код на соответствие стандартным отраслевым критериям и при необходимости добавляет встроенные комментарии к PR.
