Compétence complète de révision du code GitHub Pull Request. À utiliser lorsqu'on vous demande de « réviser ce PR », « révision du code », « réviser la demande d'extraction », « vérifier ce PR » ou lorsqu'une URL de PR GitHub est fournie. Récupère les métadonnées PR, les différences, les commentaires, les validations et les problèmes associés à l'aide de gh CLI. Crée un espace de travail de révision organisé, analyse le code par rapport aux critères standard de l'industrie et ajoute éventuellement des commentaires en ligne au PR.
