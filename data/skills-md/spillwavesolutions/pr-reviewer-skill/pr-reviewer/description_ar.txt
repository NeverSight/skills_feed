مهارة مراجعة كود طلب سحب GitHub الشاملة. يُستخدم عندما يُطلب منك "مراجعة العلاقات العامة" أو "مراجعة الكود" أو "مراجعة طلب السحب" أو "التحقق من العلاقات العامة" أو عند توفير عنوان URL للعلاقات العامة على GitHub. جلب البيانات الوصفية للعلاقات العامة، والاختلافات، والتعليقات، والالتزامات، والمشكلات ذات الصلة باستخدام gh CLI. ينشئ مساحة عمل مراجعة منظمة، ويحلل التعليمات البرمجية وفقًا لمعايير الصناعة، ويضيف بشكل اختياري التعليقات المضمنة إلى العلاقات العامة.
