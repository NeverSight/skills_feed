Habilidad completa de revisión de código de solicitud de extracción de GitHub. Úselo cuando se le solicite "revisar este PR", "revisión de código", "revisar solicitud de extracción", "verificar este PR" o cuando se proporcione una URL de PR de GitHub. Obtiene metadatos de relaciones públicas, diferencias, comentarios, confirmaciones y problemas relacionados mediante gh CLI. Crea un espacio de trabajo de revisión organizado, analiza el código según criterios estándar de la industria y, opcionalmente, agrega comentarios en línea al PR.
