أدلة المستخدم، والبرامج التعليمية، وكتيبات التشغيل) من القوالب (المجال الأخضر) أو وثائق الهندسة العكسية
من التعليمات البرمجية الموجودة مثل Spring Boot أو FastAPI (brownfield). يتعامل أيضًا مع عمليات تدقيق/مراجعات الوثائق،
تحويل التنسيق (Markdown، DOCX، PDF)، وإنشاء المخططات (C4، Mermaid، PlantUML، ER، التسلسل).
استخدم عندما يُطلب منك "إنشاء وثائق" أو "توثيق الكود الخاص بي" أو "كتابة SRS" أو "إنشاء PRD" أو "متخصص التوثيق".
