Automatise les applications macOS via Apple Events à l'aide d'AppleScript (découverte), JXA (hérité) et PyXA (Python moderne). À utiliser lorsqu'on vous demande « d'automatiser les applications Mac », « d'écrire AppleScript », « de script JXA », « d'automatiser osascript » ou « d'automatiser PyXA Python ». Compétence de base pour toutes les automatisations d’applications macOS.
