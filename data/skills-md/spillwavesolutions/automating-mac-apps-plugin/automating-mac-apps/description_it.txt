Automatizza le app macOS tramite Apple Events utilizzando AppleScript (discovery), JXA (legacy) e PyXA (Python moderno). Da utilizzare quando viene richiesto di "automatizzare le app Mac", "scrivere AppleScript", "scripting JXA", "automazione osascript" o "automazione Python PyXA". Competenza di base per tutta l'automazione delle app macOS.
