Automatise le calendrier macOS via JXA avec la découverte du dictionnaire AppleScript. À utiliser lorsqu'on vous demande de « créer des événements de calendrier », « automatiser le calendrier », « scripts de calendrier JXA », « automatisation d'EventKit » ou « automatisation de calendrier PyXA ». Couvre les événements, les calendriers, la récurrence, les fuseaux horaires, les opérations par lots et le pont EventKit ObjC.
