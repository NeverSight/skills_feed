Generador de sistemas universales de múltiples subagentes. Construya rápidamente sistemas multiagente para cualquier negocio basados ​​en un patrón de interacción maestro-sub. Analice procesos de negocio, recomiende tipos de agentes y genere plantillas de avisos. Casos de uso: (1) Se activa cuando el usuario dice "Crear un sistema de múltiples subagentes" o "Construir un sistema de múltiples agentes" (2) Flujos de automatización que requieren una arquitectura maestro-sub (3) Canalizaciones de procesamiento de datos, generación de contenido, revisión de código, etc. (4) Escenarios que necesitan orquestación paralela/en serie de múltiples subtareas
