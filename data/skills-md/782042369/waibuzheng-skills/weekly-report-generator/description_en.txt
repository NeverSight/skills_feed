Generate professional weekly work reports from Git commit history. Supports single-week/multi-week batch generation, multi-project rollup, custom templates (Markdown/Word). Automatically translates technical terms into business language and adds next-week plans, this-week summaries, and issues/risks sections. Use cases: (1) Create weekly status reports from Git logs (2) Roll up commit records from multiple projects into one report (3) Generate leader-required formats based on custom Word templates (4) Batch-create reports for a month or multiple weeks
