Generate professional weekly work reports from Git commit history. Supports single-week/multi-week batch generation, multi-project aggregation, and custom templates (Markdown/Word). Automatically translates technical terms into business language and adds sections for next-week plans/weekly summaries/issues and risks. Use cases: (1) Generate current-week work reports from Git logs (2) Combine commit records from multiple projects into one weekly report (3) Create leader-required formats based on custom Word templates (4) Batch-generate reports for a month or multiple weeks
