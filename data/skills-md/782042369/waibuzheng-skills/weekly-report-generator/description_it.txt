Genera report di lavoro settimanali professionali dalla cronologia dei commit Git. Supporta la generazione batch di una o più settimane, rollup multi-progetto, modelli personalizzati (Markdown/Word). Traduce automaticamente i termini tecnici nel linguaggio aziendale e aggiunge piani per la settimana successiva, riepiloghi di questa settimana e sezioni su problemi/rischi. Casi d'uso: (1) Creare report di stato settimanali dai log Git (2) Raggruppare record di commit da più progetti in un unico report (3) Generare formati richiesti dal leader in base a modelli Word personalizzati (4) Creare report in batch per un mese o più settimane
