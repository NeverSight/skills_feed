Предоставляет шаблон подсказки и рекомендации по созданию команд агентов Claude Code. Охватывает состав команды, приглашения на появление товарищей по команде, разбивку задач, владение файлами, протоколы связи и режим делегирования. Используйте при создании команды агентов, создании товарищей по команде, распараллеливании работы в нескольких экземплярах Claude Code или принятии решения о том, является ли команда агентов правильным подходом.
