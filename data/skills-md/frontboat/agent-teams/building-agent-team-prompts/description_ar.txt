يوفر نموذجًا سريعًا وأفضل الممارسات لبناء فرق وكلاء Claude Code. يغطي تكوين الفريق، ومطالبات ظهور أعضاء الفريق، وتقسيم المهام، وملكية الملفات، وبروتوكولات الاتصال، ووضع التفويض. يُستخدم عند إنشاء فريق وكيل، أو إنشاء زملاء في الفريق، أو موازنة العمل عبر مثيلات Claude Code المتعددة، أو تحديد ما إذا كان فريق الوكلاء هو النهج الصحيح.
