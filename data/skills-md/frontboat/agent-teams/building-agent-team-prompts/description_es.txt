Proporciona una plantilla rápida y mejores prácticas para crear equipos de agentes de Claude Code. Cubre la composición del equipo, las indicaciones de generación de compañeros de equipo, el desglose de tareas, la propiedad de archivos, los protocolos de comunicación y el modo de delegado. Úselo al crear un equipo de agentes, generar compañeros de equipo, paralelizar el trabajo en múltiples instancias de Claude Code o decidir si un equipo de agentes es el enfoque correcto.
