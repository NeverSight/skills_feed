Fournit un modèle d’invite et les meilleures pratiques pour créer des équipes d’agents Claude Code. Couvre la composition de l'équipe, les invites d'apparition des coéquipiers, la répartition des tâches, la propriété des fichiers, les protocoles de communication et le mode délégué. À utiliser pour créer une équipe d'agents, générer des coéquipiers, paralléliser le travail sur plusieurs instances de Claude Code ou décider si une équipe d'agents est la bonne approche.
