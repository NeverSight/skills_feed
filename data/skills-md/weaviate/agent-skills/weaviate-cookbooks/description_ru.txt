Используйте этот навык, когда пользователь хочет создавать приложения искусственного интеллекта с помощью Weaviate. Он содержит общий указатель архитектурных шаблонов, «одноразовые» схемы и лучшие практики для распространенных случаев использования. В настоящее время он включает ссылки на создание чат-бота Query Agent, Data Explorer, Multimodal PDF RAG (поиск документов), Basic RAG, Advanced RAG, Basic Agent, Agentic RAG, а также дополнительные инструкции по созданию внешнего интерфейса для каждого из них.
