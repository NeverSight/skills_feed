Cette compétence doit être utilisée lors de l'application des conventions Vue 3 avisées spécifiques au projet. Il couvre l'ordre des blocs SFC (modèle d'abord vs script d'abord), le typage en ligne pour définirProps et définirEmits, l'utilisation de reactive() pour la gestion de l'état du formulaire, les assertions de type ref avec as Ref<Type> pour les types complexes, les liaisons abrégées du même nom dans Vue 3.4+ et les modèles d'accès aux accessoires dans les modèles. Pertinent lorsqu'un utilisateur pose des questions sur l'ordre des modèles, comment taper des accessoires ou émettre en ligne, réactif ou ref pour les formulaires, les modèles d'assertion de type ref, les liaisons d'attributs abrégés, l'utilisation des accessoires dans les modèles ou le respect des conventions de codage Vue de Lucas Yang.
