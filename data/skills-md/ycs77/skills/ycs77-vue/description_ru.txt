Этот навык следует использовать при применении строгих соглашений Vue 3, специфичных для конкретного проекта. Он охватывает порядок блоков SFC (сначала шаблон или сначала скрипт), встроенную типизацию для defineProps и defineEmits, использование reactive() для управления состоянием формы, утверждения типа ссылки с помощью Ref<Type> для сложных типов, одноимённые сокращенные привязки в Vue 3.4+ и шаблоны доступа к реквизитам в шаблонах. Актуально, когда пользователь спрашивает о порядке шаблонов, о том, как вводить реквизиты или создавать встроенные, реактивные и реферальные формы для форм, шаблоны утверждения типов ссылок, сокращенные привязки атрибутов, использование реквизитов в шаблонах или следование соглашениям Лукаса Янга по кодированию Vue.
