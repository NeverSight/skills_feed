Converti le interazioni del browser in flussi di lavoro API deterministici con la CLI Grabbit. Da utilizzare quando richiesto per registrare il traffico del browser (HAR), generare flussi di lavoro da interazioni web/API, automatizzare l'estrazione dei dati o guidare gli utenti attraverso l'utilizzo della CLI Grabbit, incluso il controllo del browser, la gestione delle sessioni e la generazione di flussi di lavoro.
