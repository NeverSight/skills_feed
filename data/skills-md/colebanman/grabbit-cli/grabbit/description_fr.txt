Contrôlez la CLI Grabbit pour enregistrer les interactions du navigateur (HAR) et générer des flux de travail API. À utiliser lorsque l'utilisateur souhaite : (1) automatiser les actions du navigateur, (2) capturer le trafic Web pour l'analyse des API, (3) créer des flux de travail déterministes à partir de sessions de navigation ou (4) apprendre à utiliser les commandes CLI Grabbit.
