Controlla la CLI Grabbit per registrare le interazioni del browser (HAR) e generare flussi di lavoro API. Utilizzare quando l'utente desidera: (1) automatizzare le azioni del browser, (2) acquisire traffico Web per l'analisi API, (3) creare flussi di lavoro deterministici dalle sessioni di navigazione o (4) apprendere come utilizzare i comandi CLI di Grabbit.
