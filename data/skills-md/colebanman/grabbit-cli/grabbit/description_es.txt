Controle Grabbit CLI para registrar interacciones del navegador (HAR) y generar flujos de trabajo API. Úselo cuando el usuario desee: (1) Automatizar las acciones del navegador, (2) Capturar tráfico web para análisis de API, (3) Crear flujos de trabajo deterministas a partir de sesiones de navegación o (4) Aprender a usar los comandos de Grabbit CLI.
