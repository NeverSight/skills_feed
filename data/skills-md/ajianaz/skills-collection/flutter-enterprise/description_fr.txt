Compétence de développement d'entreprise Flutter léger axée sur une architecture propre basée sur des fonctionnalités. À utiliser lorsque l'utilisateur demande à : (1) créer des applications Flutter d'entreprise avec une architecture propre, (2) mettre en œuvre une structure modulaire basée sur les fonctionnalités, (3) configurer une organisation de projet Flutter évolutive, (4) créer une base de code d'entreprise maintenable. Déclencheurs : application d'entreprise Flutter, architecture propre Flutter, Flutter basé sur les fonctionnalités, structure Flutter d'entreprise, architecture Flutter modulaire
