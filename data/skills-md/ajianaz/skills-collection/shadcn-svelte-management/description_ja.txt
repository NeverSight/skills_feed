Svelte エコシステムに重点を置いた MCP ツールを使用して UI コンポーネントを管理します。 shadcn-svelte、Skeleton UI、Melt UI などの Svelte コンポーネント ライブラリに特化しています。ユーザーが次のことを行う必要がある場合に使用します。
(1) 新しい Svelte UI コンポーネントをプロジェクトに追加する
(2) 複数のコンポーネントを必要とする複雑な UI 機能を構築する
(3) 研究コンポーネントの実装と例
(4) コンポーネントインストールコマンドの取得
(5) 適切な Svelte コンポーネント ライブラリを選択する
