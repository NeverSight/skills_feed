Contrôlez un navigateur Chrome sans tête ou avec tête via l'API HTTP de Pinchtab. À utiliser pour l'automatisation Web, le scraping, le remplissage de formulaires, la navigation et les flux de travail multi-onglets. Pinchtab expose l'arborescence d'accessibilité sous forme de JSON plat avec des références stables - optimisées pour les agents IA (faible coût de jeton, rapide). À utiliser lorsque la tâche implique : parcourir des sites Web, remplir des formulaires, cliquer sur des boutons, extraire le texte d'une page, prendre des captures d'écran ou toute automatisation basée sur un navigateur. Nécessite une instance Pinchtab en cours d’exécution (Go binaire).
