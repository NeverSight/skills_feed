Pinchtab の HTTP API を介して、ヘッドレスまたはヘッド付きの Chrome ブラウザを制御します。 Web オートメーション、スクレイピング、フォーム入力、ナビゲーション、およびマルチタブ ワークフローに使用します。 Pinchtab は、AI エージェント向けに最適化された (低トークン コスト、高速) 安定した参照を持つフラットな JSON としてアクセシビリティ ツリーを公開します。 Web サイトの閲覧、フォームへの入力、ボタンのクリック、ページ テキストの抽出、スクリーンショットの撮影、またはブラウザ ベースの自動化などのタスクが含まれる場合に使用します。実行中の Pinchtab インスタンス (Go バイナリ) が必要です。
