Steuern Sie einen Headless- oder Headed-Chrome-Browser über die HTTP-API von Pinchtab. Verwendung für Webautomatisierung, Scraping, Formularausfüllung, Navigation und Workflows mit mehreren Registerkarten. Pinchtab stellt den Barrierefreiheitsbaum als flaches JSON mit stabilen Referenzen bereit – optimiert für KI-Agenten (niedrige Token-Kosten, schnell). Verwenden Sie es, wenn die Aufgabe Folgendes umfasst: Durchsuchen von Websites, Ausfüllen von Formularen, Klicken auf Schaltflächen, Extrahieren von Seitentext, Erstellen von Screenshots oder eine browserbasierte Automatisierung. Erfordert eine laufende Pinchtab-Instanz (Go Binary).
