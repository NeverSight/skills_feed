Steuern Sie einen Headless- oder Headed-Chrome-Browser über die HTTP-API von Pinchtab. Verwendung für Webautomatisierung, Scraping, Formularausfüllung, Navigation und Workflows mit mehreren Registerkarten. Pinchtab stellt den Barrierefreiheitsbaum als flaches JSON mit stabilen Refs bereit – optimiert für KI-Agenten (niedrige Token-Kosten, schnell). Verwenden Sie es, wenn die Aufgabe das Durchsuchen von Websites, das Ausfüllen von Formularen, das Klicken auf Schaltflächen, das Extrahieren von Seitentext, das Erstellen von Screenshots oder eine browserbasierte Automatisierung umfasst. Erfordert eine laufende Pinchtab-Instanz (Go Binary).
