Управляйте безголовым или управляемым браузером Chrome через HTTP API Pinchtab. Используйте для веб-автоматизации, очистки данных, заполнения форм, навигации и рабочих процессов с несколькими вкладками. Pinchtab представляет дерево доступности в виде плоского JSON со стабильными ссылками, оптимизированного для агентов ИИ (низкая стоимость токена, скорость). Используйте, когда задача включает в себя: просмотр веб-сайтов, заполнение форм, нажатие кнопок, извлечение текста страницы, создание снимков экрана или любую автоматизацию на основе браузера. Требуется работающий экземпляр Pinchtab (двоичный файл Go).
