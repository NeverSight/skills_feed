Sviluppa plugin nativi WebF personalizzati basati sui pacchetti Flutter. Crea plug-in riutilizzabili che racchiudono le funzionalità Flutter/della piattaforma come API JavaScript. Da utilizzare durante la creazione di plug-in per funzionalità native come fotocamera, pagamenti, sensori, accesso ai file o il confezionamento di pacchetti Flutter esistenti.
