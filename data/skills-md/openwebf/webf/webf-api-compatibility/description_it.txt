Controlla la compatibilità dell'API Web e delle funzionalità CSS in WebF: determina quali API JavaScript, metodi DOM, proprietà CSS e modalità di layout sono supportate. Da utilizzare durante la pianificazione delle funzionalità, il debug del motivo per cui le API non funzionano o la ricerca di alternative per funzionalità non supportate come IndexedDB, WebGL, layout mobile o griglia CSS.
