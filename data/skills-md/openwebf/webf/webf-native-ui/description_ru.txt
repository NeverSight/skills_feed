Настройте и используйте библиотеку пользовательского интерфейса WebF Cupertino для создания собственных пользовательских интерфейсов в стиле iOS с помощью готовых компонентов вместо того, чтобы создавать все с помощью HTML/CSS. Используйте при создании приложений для iOS, добавлении собственных компонентов пользовательского интерфейса или повышении производительности пользовательского интерфейса.
