Inicialice o mejore los archivos AGENTS.md que definen cómo operan los agentes de codificación en un repositorio. Úselo cuando se le solicite configurar o reemplazar un comando de inicio de agente (Codex, Claude), estandarizar el comportamiento de múltiples agentes o auditar un AGENTS.md existente para mayor claridad, comandos, límites y contexto específico del repositorio. Para Claude Code, cree también CLAUDE.md como un enlace simbólico a AGENTS.md.
