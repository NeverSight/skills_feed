Initialisez ou améliorez les fichiers AGENTS.md qui définissent le fonctionnement des agents de codage dans un dépôt. À utiliser lorsqu'on vous demande de configurer ou de remplacer une commande d'initialisation d'agent (Codex, Claude), de normaliser le comportement multi-agents ou d'auditer un AGENTS.md existant pour en vérifier la clarté, les commandes, les limites et le contexte spécifique au dépôt. Pour Claude Code, créez également CLAUDE.md comme lien symbolique vers AGENTS.md.
