Documentation complète de la plateforme Supabase. À utiliser lorsque vous travaillez avec Supabase - couvre l'authentification (e-mail, OAuth, liens magiques, téléphone, SSO), la base de données (PostgreSQL, RLS, migrations), le stockage (téléchargements de fichiers, CDN), les fonctions Edge, les abonnements en temps réel, l'IA/l'intégration, les tâches cron, les files d'attente et la gestion de la plateforme. Inclut des intégrations de framework (Next.js, React, SvelteKit, etc.).
