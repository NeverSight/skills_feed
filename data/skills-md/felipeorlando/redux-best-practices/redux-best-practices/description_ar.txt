أفضل الممارسات والأنماط والأنماط المضادة لـ Redux وReact-Redux استنادًا إلى دليل أسلوب Redux الرسمي. يُستخدم عند كتابة كود Redux أو مراجعته أو إعادة هيكلته، بما في ذلك إعداد المتجر أو الشرائح أو المخفضات أو المحددات أو المنطق غير المتزامن أو تسوية الحالة أو خطافات React-Redux. تشغيل المهام التي تتضمن createSlice، أوconfigurationStore، أوuseSelector، أوuseDispatch، أو thunks، أو RTK Query، أو قرارات هندسة إدارة الحالة.
