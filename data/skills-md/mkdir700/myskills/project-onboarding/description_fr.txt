Comprenez rapidement les bases de code inconnues grâce à une analyse structurée. À utiliser lorsque l'utilisateur a besoin de comprendre un nouveau projet, en particulier un code mal documenté ou non standard. Les déclencheurs incluent « m'aider à comprendre ce projet », « analyser cette base de code », « m'intégrer à [projet] », « que fait ce projet » ou lorsque l'utilisateur fournit un chemin de projet à explorer. Particulièrement efficace pour le code existant, les projets de piratage et les bases de code manquant de documentation.
