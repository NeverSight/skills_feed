Comprenda rápidamente bases de código desconocidas mediante análisis estructurados. Úselo cuando el usuario necesite comprender un nuevo proyecto, especialmente código mal documentado o no estándar. Los desencadenantes incluyen "ayúdame a comprender este proyecto", "analizar este código base", "incorporarme a [proyecto]", "qué hace este proyecto" o cuando el usuario proporciona una ruta del proyecto para su exploración. Particularmente eficaz para código heredado, proyectos de piratería y bases de código que carecen de documentación.
