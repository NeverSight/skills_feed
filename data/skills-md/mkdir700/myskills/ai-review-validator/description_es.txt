Valide y ejecute de forma autónoma sugerencias de revisión de IA a partir de comentarios de relaciones públicas. Úselo cuando los usuarios proporcionen comentarios de AI Review (de GitHub Copilot, CodeRabbit, etc.) que sugieran cambios de código, migraciones de API o correcciones. La habilidad verifica las sugerencias con documentos oficiales, prueba la compilación, calcula puntuaciones de confianza y aplica automáticamente los cambios cuando se verifica. Se activan frases como "verificar esta revisión de AI", "aplicar esta sugerencia", "validar el comentario de la revisión de AI" o cuando los usuarios pegan las URL o el contenido de la revisión de AI.
