Создавайте документацию по функциям в одном файле с проверенными тестовыми примерами. Используйте для быстрой документации по функциям, README функций, любого проекта (кроме корпоративных модулей). Выводит один файл уценки с доказательством кода. Запускается по «краткой документации по функциям», «файлу ознакомления с функциями», «документам в одном файле», «проверенной документации». Для иерархии корпоративных модулей вместо этого используйте business-feature-docs.
