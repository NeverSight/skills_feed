À utiliser lorsque l'utilisateur demande à déboguer, diagnostiquer, corriger un bogue, résoudre des erreurs, enquêter sur des problèmes ou coller des messages d'erreur/des traces de pile. Se déclenche sur des mots-clés tels que « bug », « erreur », « correctif », « ne fonctionne pas », « cassé », « débogage », « trace de pile », « exception », « crash », « problème ».
