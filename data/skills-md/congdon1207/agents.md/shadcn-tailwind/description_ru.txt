Навыки работы с библиотекой компонентов React для shadcn/ui + Tailwind CSS. Используйте для проектов React/Next.js/Vite, которым необходимы доступные компоненты пользовательского интерфейса (диалоги, формы, таблицы, навигация), примитивы пользовательского интерфейса Radix, стили утилиты Tailwind, темы и темный режим. НЕ для проектов Angular — см. навыки frontend-angular-*. Триггеры по «shadcn», «radix ui», «компонентам реакции», «компонентам попутного ветра». Для дизайнерских решений (стили, палитры, шрифты) используйте вместо этого ui-ux-pro-max.
