À utiliser lorsque l'utilisateur demande à comparer des branches, à analyser des différences git, à examiner les modifications entre les branches, à mettre à jour les spécifications en fonction des modifications de code ou à analyser ce qui a changé. Déclencheurs sur des mots-clés tels que « comparer les branches », « git diff », « ce qui a changé », « comparaison de branches », « modifications de code », « mise à jour des spécifications ».
