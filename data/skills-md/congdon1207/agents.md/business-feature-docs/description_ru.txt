Создавайте документацию корпоративного модуля со структурой из 26 разделов и иерархией папок. Используйте для документации модулей, корпоративных функций и подробных спецификаций в docs/business-features/{Module}/. Включает README, INDEX, API-REFERENCE, подробные характеристики/. Запускается по «документам модуля», «документам по функциям предприятия», «бизнес-модулю», «документам из 26 разделов», «подробным спецификациям функций». Для однофайловых быстрых документов используйте вместо этого Feature-Docs.
