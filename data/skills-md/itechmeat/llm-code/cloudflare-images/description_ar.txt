قم بتخزين الصور المحسنة وتحويلها وتقديمها باستخدام Cloudflare Images. يغطي تحميل الصور (API، Worker، Direct Creator Upload)، والمتغيرات، والتحويلات (URL وWorkers)، والروابط، والبولندية، وعناوين URL الموقعة، والتنسيقات (AVIF، WebP)، وتكامل R2، والعلامات المائية. الكلمات الرئيسية: صور Cloudflare، تحويلات الصور، المتغيرات، /cdn-cgi/image/، imagedelivery.net، البولندية، AVIF، WebP، عناوين URL الموقعة، التحميل المباشر للمنشئ، ربط الصور، R2، العلامة المائية.
