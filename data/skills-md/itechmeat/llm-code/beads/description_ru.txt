Beads (bd) — распределенное средство отслеживания проблем для агентов ИИ на базе git: идентификаторы на основе хеша, графики зависимостей, рабочие деревья, молекулы, синхронизация. Ключевые слова: bd, бусины, трекер проблем, поддержка git, зависимости, молекулы, рабочее дерево, синхронизация, агенты ИИ.
