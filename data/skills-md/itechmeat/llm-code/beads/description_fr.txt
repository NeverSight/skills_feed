Bead (bd) a distribué un système de suivi des problèmes basé sur git pour les agents d'IA : identifiants basés sur le hachage, graphiques de dépendances, arbres de travail, molécules, synchronisation. Mots-clés : bd, perles, suivi des problèmes, soutenu par git, dépendances, molécules, arbre de travail, synchronisation, agents IA.
