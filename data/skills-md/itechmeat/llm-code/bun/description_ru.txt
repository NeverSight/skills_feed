Среда выполнения JavaScript/TypeScript и универсальный инструментарий. Охватывает среду выполнения, менеджер пакетов, сборщик, средство запуска тестов, HTTP-сервер, WebSockets, SQLite, S3, Redis, файловый ввод-вывод, сценарии оболочки, FFI, анализатор Markdown. Ключевые слова: булочка, Bunx, установка плюшки, запуск плюшки, тест плюшки, сборка плюшки, Bun.serve, Bun.file, Bun:sqlite, Bun.markdown.
