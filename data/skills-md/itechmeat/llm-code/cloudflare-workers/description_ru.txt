Создавайте бессерверные приложения на Cloudflare Workers. Охватывает API среды выполнения, обработчики (извлечение, расписание, очередь, электронная почта), привязки (KV, R2, D1, устойчивые объекты, очереди, AI, векторизация), конфигурацию wrangler.toml, локальную разработку с помощью Miniflare, статические ресурсы, флаги совместимости, тестирование с помощью Vitest. Ключевые слова: Cloudflare Workers, бессерверные вычисления, периферийные вычисления, Wrangler, обработчик выборки, запланированный обработчик, привязки, KV, R2, D1, Durable Objects, Workers AI, Miniflare, wrangler.toml, Совместимость_дата.
