Создавайте и используйте промежуточное программное обеспечение Inngest для решения сквозных задач. Охватывает жизненный цикл промежуточного программного обеспечения, создание специального промежуточного программного обеспечения, внедрение зависимостей с помощью dependencyInjectionMiddleware, шифрование с помощью @inngest/middleware-encryption, отслеживание ошибок Sentry с помощью @inngest/middleware-sentry и пользовательские шаблоны промежуточного программного обеспечения.
