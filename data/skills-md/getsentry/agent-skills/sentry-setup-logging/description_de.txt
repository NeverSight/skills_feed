Richten Sie Sentry Logging in jedem Projekt ein. Verwenden Sie diese Option, wenn Sie aufgefordert werden, Sentry-Protokolle hinzuzufügen, strukturierte Protokollierung zu aktivieren, Konsolenprotokolle zu erfassen oder Protokollierungsbibliotheken (Pino, Winston, Loguru) in Sentry zu integrieren. Unterstützt JavaScript, Python und Ruby.
