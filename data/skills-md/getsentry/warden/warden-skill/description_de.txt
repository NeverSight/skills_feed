Leitfaden zur lokalen Verwendung von Warden CLI zur Analyse von Codeänderungen. Verwenden Sie diese Funktion, wenn Sie Warden-Befehle ausführen, warden.toml konfigurieren, benutzerdefinierte Fertigkeiten erstellen, Auslöser verstehen oder Analyseprobleme beheben. Wird bei „Warden ausführen“, „Warden-Konfiguration“, „warden.toml“, „Warden-Skill erstellen“, „Auslöser hinzufügen“ oder jeder Warden-bezogenen lokalen Entwicklungsaufgabe ausgelöst.
