Bietet umfassende Anleitung zur Codeüberprüfung für React 19, Vue 3, Rust, TypeScript, Java, Python und C/C++.
Hilft, Fehler zu erkennen, die Codequalität zu verbessern und konstruktives Feedback zu geben.
Verwendung bei: Überprüfung von Pull-Anfragen, Durchführung von PR-Überprüfungen, Codeüberprüfung, Überprüfung von Codeänderungen,
Festlegung von Überprüfungsstandards, Betreuung von Entwicklern, Architekturprüfungen, Sicherheitsaudits,
Überprüfen der Codequalität, Finden von Fehlern, Geben von Feedback zum Code.
