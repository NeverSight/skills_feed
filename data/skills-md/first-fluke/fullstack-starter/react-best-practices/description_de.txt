Richtlinien zur Leistungsoptimierung von React und Next.js, optimiert von Gracefullight, abgeleitet von Vercel Engineering. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von React/Next.js-Code verwendet werden, um optimale Leistungsmuster sicherzustellen. Wird bei Aufgaben ausgelöst, die React-Komponenten, Next.js-Seiten, Datenabruf, Bundle-Optimierung oder Leistungsverbesserungen betreffen.
