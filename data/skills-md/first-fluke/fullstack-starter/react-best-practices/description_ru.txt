Рекомендации по оптимизации производительности React и Next.js, настроенные с помощью Gracefullight, созданного Vercel Engineering. Этот навык следует использовать при написании, проверке или рефакторинге кода React/Next.js, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, включающих компоненты React, страницы Next.js, выборку данных, оптимизацию пакетов или повышение производительности.
