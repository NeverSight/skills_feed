Verwalten Sie Model Context Protocol (MCP)-Server – entdecken, analysieren und führen Sie Tools/Eingabeaufforderungen/Ressourcen von konfigurierten MCP-Servern aus. Wird verwendet, wenn Sie mit MCP-Integrationen arbeiten, verfügbare MCP-Funktionen ermitteln, MCP-Tools für bestimmte Aufgaben filtern, MCP-Tools programmgesteuert ausführen, auf MCP-Eingabeaufforderungen/-Ressourcen zugreifen oder MCP-Client-Funktionalität implementieren müssen. Unterstützt die intelligente Toolauswahl, die Verwaltung mehrerer Server und die kontexteffiziente Funktionserkennung.
