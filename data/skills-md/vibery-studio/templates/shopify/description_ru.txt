Создавайте приложения, расширения и темы Shopify, используя API-интерфейсы GraphQL/REST, интерфейс командной строки Shopify, компоненты пользовательского интерфейса Polaris и шаблоны Liquid. Возможности включают разработку приложений с аутентификацией OAuth, расширения пользовательского интерфейса для настройки процесса оформления заказа, расширения пользовательского интерфейса администратора для интеграции с информационной панелью, расширения POS для розничной торговли, разработку тем с помощью Liquid, управление веб-перехватчиками, интеграцию API для выставления счетов, управление продуктами/заказами/клиентами. Используйте при создании приложений Shopify, реализации настроек оформления заказа, создании интерфейсов администратора, разработке тем, интеграции обработки платежей, управлении данными магазина через API или расширении функциональности Shopify.
