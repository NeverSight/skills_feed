GraphQL/REST API、Shopify CLI、Polaris UI コンポーネント、Liquid テンプレートを使用して、Shopify アプリケーション、拡張機能、テーマを構築します。機能には、OAuth 認証を使用したアプリ開発、チェックアウト フローをカスタマイズするためのチェックアウト UI 拡張機能、ダッシュボード統合のための管理 UI 拡張機能、小売用 POS 拡張機能、Liquid を使用したテーマ開発、Webhook 管理、請求 API 統合、製品/注文/顧客管理が含まれます。 Shopify アプリの構築、チェックアウトのカスタマイズの実装、管理インターフェイスの作成、テーマの開発、支払い処理の統合、API を介したストア データの管理、または Shopify 機能の拡張を行うときに使用します。
