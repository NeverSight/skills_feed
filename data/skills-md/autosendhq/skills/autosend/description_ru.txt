Интеграция API электронной почты AutoSend. Используйте при отправке транзакционных электронных писем, управлении контактами или использовании шаблонов электронной почты с помощью AutoSend. Срабатывает при: отправке электронной почты с помощью автоотправки, автоотправке API, автоотправке контактов, автоотправке шаблона электронной почты.
