Integración de API de correo electrónico de AutoSend. Úselo al enviar correos electrónicos transaccionales, administrar contactos o usar plantillas de correo electrónico con AutoSend. Activadores: envío de correo electrónico con envío automático, API de envío automático, envío automático de contactos, envío automático de plantilla de correo electrónico.
