Verwenden Sie diese Option, wenn der Benutzer Fertigkeiten auflisten, suchen, installieren, entfernen, prüfen, validieren, prüfen oder aktualisieren möchte. Verwenden Sie diese Option, wenn Sie fragen: „Welche Fertigkeiten habe ich“, „Gibt es eine Fertigkeit für X“, „Überprüfen Sie meine Fertigkeiten auf Probleme“ oder „Installieren Sie eine Fertigkeit“. Wird auch verwendet, wenn der Skill-Zustand aller Agenten überprüft wird (Claude Code, Codex, Agents CLI).
