Flusso di lavoro di risoluzione dei problemi end-to-end di GitHub utilizzando gh, modifiche al codice locale, build/test e git push. Da utilizzare quando viene richiesto di prendere un numero di problema, ispezionare il problema tramite gh, implementare una correzione, eseguire build/test XcodeBuildMCP, eseguire il commit con un messaggio di chiusura ed eseguire il push.
