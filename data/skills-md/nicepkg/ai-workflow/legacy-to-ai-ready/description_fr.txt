Transformez les bases de code existantes en projets prêts pour l'IA avec les configurations Claude Code. À utiliser lors de (1) l'analyse d'anciens projets pour générer des configurations de codage d'IA, (2) la création de CLAUDE.md, de compétences, de sous-agents, de commandes slash, de hooks ou de règles pour des projets existants, (3) l'utilisateur souhaite activer le codage d'ambiance pour une base de code, (4) l'intégration de nouveaux membres de l'équipe avec un développement assisté par l'IA, (5) l'utilisateur mentionne « préparer le projet à l'IA », « générer une configuration Claude » ou « créer des normes de codage pour l'IA ».
