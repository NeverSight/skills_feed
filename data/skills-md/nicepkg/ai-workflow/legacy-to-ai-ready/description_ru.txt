Преобразуйте устаревшие кодовые базы в проекты, готовые к использованию ИИ, с помощью конфигураций Claude Code. Используйте, когда (1) анализируете старые проекты для создания конфигураций кодирования ИИ, (2) создаете CLAUDE.md, навыки, субагенты, команды с косой чертой, перехватчики или правила для существующих проектов, (3) пользователь хочет включить кодирование Vibe для базы кода, (4) привлекает новых членов команды к разработке с помощью ИИ, (5) пользователь упоминает «сделать проект готовым к ИИ», «создать конфигурацию Клода» или «создать стандарты кодирования для ИИ».
