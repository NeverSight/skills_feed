Créez des répertoires complets de flux de travail Claude Code avec des compétences sélectionnées. À utiliser lorsque l'utilisateur souhaite (1) créer un nouveau flux de travail pour un cas d'utilisation spécifique (créateur de médias, développeur, spécialiste du marketing, etc.), (2) mettre en place un projet Claude Code avec des compétences préconfigurées, (3) télécharger et organiser les compétences à partir des référentiels GitHub, ou (4) générer la documentation README.md et AGENTS.md pour les flux de travail. Se déclenche sur des expressions telles que « créer un workflow », « nouveau workflow », « configurer un workflow », « créer un workflow xxx ».
