Crea directory complete del flusso di lavoro di Claude Code con competenze selezionate. Da utilizzare quando l'utente desidera (1) creare un nuovo flusso di lavoro per un caso d'uso specifico (creatore di contenuti multimediali, sviluppatore, operatore di marketing, ecc.), (2) impostare un progetto Claude Code con competenze preconfigurate, (3) scaricare e organizzare competenze dai repository GitHub o (4) generare documentazione README.md e AGENTS.md per i flussi di lavoro. Si attiva su frasi come "crea flusso di lavoro", "nuovo flusso di lavoro", "imposta flusso di lavoro", "crea un flusso di lavoro xxx".
