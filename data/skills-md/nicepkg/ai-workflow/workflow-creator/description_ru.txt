Создавайте полные каталоги рабочих процессов Claude Code с тщательно подобранными навыками. Используйте, когда пользователь хочет (1) создать новый рабочий процесс для конкретного варианта использования (создатель мультимедиа, разработчик, маркетолог и т. д.), (2) настроить проект Claude Code с предварительно настроенными навыками, (3) загрузить и систематизировать навыки из репозиториев GitHub или (4) создать документацию README.md и AGENTS.md для рабочих процессов. Запускается по таким фразам, как «создать рабочий процесс», «новый рабочий процесс», «настроить рабочий процесс», «создать xxx-рабочий процесс».
