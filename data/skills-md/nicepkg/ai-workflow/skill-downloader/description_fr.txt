Téléchargez et installez les compétences Claude Code à partir de diverses sources. Prend en charge les référentiels GitHub, les archives compressées (.zip, .tar.gz, .skill) et les URL directes. À utiliser lorsque l'utilisateur souhaite télécharger, installer ou ajouter une compétence à partir de GitHub, d'une URL ou d'un fichier d'archive. Déclencheurs sur "télécharger la compétence", "installer la compétence", "ajouter une compétence à partir de", "obtenir une compétence".
