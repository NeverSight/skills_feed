Ищите и изучайте навыки Claude Code и серверы MCP на торговых площадках, в репозиториях GitHub и реестрах. Используйте, когда (1) пользователь просит найти навыки для конкретной задачи, (2) ищет серверы MCP для подключения внешних инструментов, (3) пользователь упоминает «найти навык», «поиск MCP», «найти инструменты» или «какие навыки существуют для X», (4) перед созданием специального навыка, чтобы проверить, существует ли он уже.
