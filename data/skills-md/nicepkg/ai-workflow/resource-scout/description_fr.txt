Recherchez et découvrez les compétences Claude Code et les serveurs MCP sur les places de marché, les référentiels GitHub et les registres. À utiliser lorsque (1) l'utilisateur demande à trouver des compétences pour une tâche spécifique, (2) recherche des serveurs MCP pour connecter des outils externes, (3) l'utilisateur mentionne "trouver une compétence", "rechercher MCP", "découvrir des outils" ou "quelles compétences existent pour X", (4) avant de créer une compétence personnalisée pour vérifier si elle existe déjà.
