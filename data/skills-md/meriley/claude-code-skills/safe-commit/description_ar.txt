⚠️ إلزامية - يجب عليك استدعاء هذه المهارة عند الالتزام. أكمل سير عمل الالتزام مع جميع فحوصات السلامة. يستدعي مهارات الفحص الأمني ​​وفحص الجودة واختبارات التشغيل. يُظهر الفرق، ويحصل على موافقة المستخدم، وينشئ التزامًا بالتنسيق التقليدي. لا إسناد منظمة العفو الدولية. موافقة المستخدم مطلوبة إلا أثناء إنشاء العلاقات العامة. لا تلتزم أبدًا يدويًا.
