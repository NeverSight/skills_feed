Шаблоны интеграции C++ и Qt для плагинов OBS Studio. Охватывает виджеты Qt6 для диалоговых окон настроек, CMAKE_AUTOMOC, API внешнего интерфейса OBS, дополнительные сборки Qt с резервными вариантами C и шаблоны модальных диалогов. Используйте при добавлении компонентов пользовательского интерфейса или функций C++ в плагины OBS.
