Modèles d'intégration C++ et Qt pour les plugins OBS Studio. Couvre les widgets Qt6 pour les boîtes de dialogue de paramètres, CMAKE_AUTOMOC, l'API frontale OBS, les versions Qt facultatives avec des solutions de repli C et les modèles de dialogue modaux. À utiliser lors de l'ajout de composants d'interface utilisateur ou de fonctionnalités C++ aux plugins OBS.
