Эффективно извлекает определенные поля из файлов YAML, используя yq вместо чтения целых файлов, сохраняя 80–95% контекста. Используйте этот навык при запросе файлов YAML, фильтрации/преобразовании данных конфигурации или получении определенных полей из больших файлов YAML, таких как docker-compose.yml или рабочих процессов GitHub Actions.
