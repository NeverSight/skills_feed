Эффективно извлекает структуру файла (функции, классы, экспорт) без чтения целых файлов, используя ast-grep, анализатор Dart/Flutter, ctags или другие специфичные для языка инструменты для получения структур и подписей. Используйте этот навык при перечислении всех методов, функций или классов в файле, изучении незнакомого кода, получении обзоров API или выборе выборочного чтения.
