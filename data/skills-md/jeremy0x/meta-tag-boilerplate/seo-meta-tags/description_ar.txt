تنفيذ شامل لتحسين محركات البحث وبيانات التعريف الاجتماعية لمشاريع HTML وNext.js App Router وVite (React & Vue). يُستخدم عندما: (1) إنشاء العلامات الوصفية أو تحريرها أو مراجعتها، أو الرسم البياني المفتوح، أو بطاقات Twitter، أو الروابط الأساسية/hreflang، أو البيانات المنظمة JSON-LD، أو لون السمة، أو تكوين الأيقونة المفضلة/البيان؛ (2) العمل مع html/index.html، أو nextjs/metadata.ts، أو nextjs/dynamic-metadata-example.tsx، أو vite/index.html، أو vite/SEO.tsx، أو vite/SEO.vue؛ (3) إضافة تحسين محركات البحث (SEO) إلى صفحة أو مسار جديد؛ (4) الترحيل من React-helmet أو vue-meta أو next-seo؛ (5) إنشاء بيانات وصفية ديناميكية للصفحات المبنية على المحتوى؛ (6) تدقيق البيانات الوصفية للتأكد من اكتمالها أو الامتثال لأفضل الممارسات.
