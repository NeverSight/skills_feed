Planen und (wenn möglich) Implementierung oder Durchführung von Benutzerakzeptanztests (UAT) /
End-to-End-Akzeptanzszenarien. Konvertiert Anforderungen oder User Stories in
Akzeptanzkriterien, Testfälle, Testdaten und eine Checkliste für die Freigabe; schlägt vor
Automatisierung (Playwright/Cypress für Web, Golden/Snapshot-Tests für CLIs/APIs).
Wird zur Validierung des für den Benutzer sichtbaren Verhaltens für ein Release oder Mapping verwendet
Anforderungen an die Abnahmedeckung.
