يُنشئ رمز Rust لمكونات واجهة مستخدم سطح المكتب GPUI التي تتبع أنماط محرر Zed. يُستخدم عند إنشاء تطبيقات سطح المكتب باستخدام صندوق gpui، أو إنشاء مكونات واجهة المستخدم ذات السمات، أو تنفيذ الإكمال التلقائي/الإكمال، أو إنشاء لوحات الأوامر، أو العمل مع مكتبة مكونات gpui. يغطي مكونات RenderOnce، وإدارة حالة الكيان، والتخصيص باستخدام ActiveTheme، وأنماط واجهة المستخدم بنمط Zed.
