Генерирует код Rust для компонентов настольного пользовательского интерфейса GPUI по шаблонам редактора Zed. Используйте при создании настольных приложений с помощью контейнера gpui, создании тематических компонентов пользовательского интерфейса, реализации автозаполнения/дополнения, создании палитр команд или работе с библиотекой компонентов gpui. Охватывает компоненты RenderOnce, управление состоянием объектов, оформление тем с помощью ActiveTheme и шаблоны пользовательского интерфейса в стиле Zed.
