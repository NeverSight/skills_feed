Erstellen oder prüfen Sie die Rückverfolgbarkeit von Anforderungen über Design, Code und Tests. Baut ein
Rückverfolgbarkeitsmatrix (REQ → Design/ADR → Implementierungsdateien → Tests →
Beweise) und weist auf Lücken hin (nicht umgesetzte Anforderungen, ungetestete Änderungen,
undokumentierte Entscheidungen). Verwenden Sie diese Option, wenn Sie eine Überprüfung der Rückverfolgbarkeit von Anforderungen benötigen
für eine PR/Veröffentlichung, Regulierungs-/Compliance-Arbeiten oder wenn die Anforderungen schwanken
aus der Umsetzung.
