Разработайте формальные спецификации для систем управления конфигурацией (CM) с поддержкой искусственного интеллекта, которые реализуют принципы контекстного проектирования. Используйте при выполнении следующих задач: (1) создание спецификаций CM для программ с поддержкой ИИ, (2) определение моделей семантического управления для артефактов ИИ, (3) разработка инфраструктур обнаружения отклонений для подсказок/моделей/доменов, (4) определение логики управления этапами с семантической проверкой, (5) построение моделей отслеживания на этапах жизненного цикла ИИ, (6) определение управления изменениями для подсказок и развития модели, (7) проектирование ролей агента ИИ для управления конфигурацией или (8) любое Задача управления конфигурацией, включающая вероятностное поведение ИИ, развивающиеся подсказки, изменение определений предметной области или распространение артефактов.
