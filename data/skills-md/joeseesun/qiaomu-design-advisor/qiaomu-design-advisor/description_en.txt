Paranoid Design Consultant — Jobs-style product intuition + Rams-style functional purism. Use when redesigning pages, reviewing UI proposals, or optimizing interaction experience. Trigger words: “redesign”, “optimize interface”, “optimize interaction”, “design proposal”, “UI review”, “this page is bad”, “the UI is ugly”, “help me check this design”, “design advice”, “/design-advisor”. Applicable to: (1) page/component redesign, (2) UI/UX proposal review, (3) interaction logic optimization, (4) visual system establishment, (5) design-decision consultation. Difference from design-taste-frontend: design-taste-frontend is a technical implementation spec (colors, spacing, animation values); this skill is a design-thinking methodology (how to think, how to decide, how to deliver a solution). The two are complementary and can be used together.
