YouTube视频搜索、视频下载、字幕下载工具。集成 YouTube Data API v3 以进行高级搜索，并使用 yt-dlp 下载视频/音频/字幕。
核心能力：全站关键词搜索、频道浏览、按时间/观看次数/相关性排序、视频下载、音频提取（MP3）、字幕下载（中/英文）、视频详情。
触发场景（将此技能用于任何与 YouTube 相关的操作）：
- 搜索：“搜索 YouTube”、“YouTube 搜索”、“查找 YouTube 视频”、“搜索 [频道] 最新视频”、“查找 [人] 的 YouTube”、“[人] 最近的 YouTube”、“关于 XXX 的 YouTube”
- 频道浏览：“浏览频道”、“查看[频道]最新视频”、“[人]最近上传的内容”、“[人] YouTube 更新”、“查看[频道]视频列表”
- 下载视频：“下载 YouTube”、“下载此视频”、“下载 YouTube 视频”、“保存此视频”、“保存视频”
- 下载音频：“提取音频”、“下载音频”、“YouTube 到 MP3”、“仅音频”
- 下载字幕：“下载字幕”、“提取字幕”、“获取字幕”、“下载 YouTube 字幕”、“中文字幕”、“英文字幕”、“提取文本”、“视频转文本”、“获取视频文本”
- 视频信息：“视频详细信息”、“视频信息”、“该视频时长”
关键词匹配：当用户消息中包含“YouTube”、“油管”、“YT”、“yt”以及搜索/下载/字幕/频道/视频等动作词时，触发此技能。
