Поиск видео на YouTube, загрузчик видео, загрузчик субтитров. Интегрирует API данных YouTube v3 для расширенного поиска и yt-dlp для получения видео/аудио/субтитров.
Основные возможности: поиск по ключевым словам по всему сайту, просмотр каналов, сортировка по времени/просмотрам/релевантности, загрузка видео, извлечение аудио (MP3), загрузка субтитров (китайский/английский), просмотр деталей видео.
Триггерные сценарии (используйте этот навык для любых действий, связанных с YouTube):
- Поиск: «поиск на YouTube», «поиск на YouTube», «найти видео на YouTube», «поиск последних видео на [канале]», «найти на YouTube [имя]», «недавние загрузки на YouTube [имя]», «видео на YouTube о XXX».
- Просмотр каналов: «просмотреть канал», «показать последние видео [канал]», «что [имя] недавно загружало», «[имя] обновления YouTube», «просмотреть список видео [канал]»
- Загрузить видео: «загрузить YouTube», «загрузить это видео», «загрузить видео с YouTube», «сохранить это видео».
- Загрузить аудио: «извлечь аудио», «загрузить аудио», «YouTube в MP3», «только аудио».
- Загрузка субтитров: «загрузить субтитры», «извлечь субтитры», «получить субтитры», «загрузить субтитры YouTube», «китайские субтитры», «английские субтитры», «извлечь текст», «видео в текст», «получить видеотекст».
- Информация о видео: «детали видео», «информация о видео», «сколько длится это видео».
Соответствие ключевому слову: активируется всякий раз, когда сообщение пользователя содержит «YouTube», «油管», «YT», «yt» плюс слова действия, такие как поиск/загрузка/субтитры/канал/видео.
