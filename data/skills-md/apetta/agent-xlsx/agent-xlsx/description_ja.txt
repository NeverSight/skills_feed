データ抽出、分析、書き込み、書式設定、ビジュアル キャプチャ、VBA 分析、シート管理のために、agent-xlsx CLI を使用して Excel ファイル (.xlsx、.xlsm、.xlsb、.xls、.ods) を操作します。ユーザーが次のことを要求する場合に使用します: (1) スプレッドシート内のデータの読み取り、分析、または検索、(2) セルへの値または数式の書き込み、(3) 書式設定、数式、グラフ、またはメタデータの検査、(4) シートのスクリーンショットまたはビジュアル キャプチャの取得、(5) CSV/JSON/Markdown へのシートのエクスポート、(6) シートの管理 (作成、名前変更、削除、コピー、非表示)、(7) VBA マクロの分析または実行、 (8) 埋め込みオブジェクト (チャート、図形、画像) のリスト/エクスポート、(9) 数式エラーのチェック、または (10) Excel ファイルの対話を伴うタスク。 openpyxl/pandas スクリプトよりも、AI 向けに最適化された高速で構造化された JSON を優先します。
