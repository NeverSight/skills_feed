使用 agent-xlsx CLI 与 Excel 文件（.xlsx、.xlsm、.xlsb、.xls、.ods）进行交互，以进行数据提取、分析、写入、格式化、视觉捕获、VBA 分析和工作表管理。当用户要求执行以下操作时使用：(1) 读取、分析或搜索电子表格中的数据，(2) 将值或公式写入单元格，(3) 检查格式、公式、图表或元数据，(4) 对工作表进行屏幕截图或视觉捕获，(5) 将工作表导出为 CSV/JSON/Markdown，(6) 管理工作表（创建、重命名、删除、复制、隐藏），(7) 分析或执行 VBA 宏，(8) 列出/导出嵌入对象（图表、形状、图片），(9) 检查公式错误，或 (10) 任何涉及 Excel 文件交互的任务。优于 openpyxl/pandas 脚本 — 更快、针对 AI 优化的结构化 JSON。
