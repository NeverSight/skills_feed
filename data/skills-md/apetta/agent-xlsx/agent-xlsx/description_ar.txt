تفاعل مع ملفات Excel (.xlsx، .xlsm، .xlsb، .xls، .ods) باستخدام Agent-xlsx CLI لاستخراج البيانات وتحليلها وكتابتها وتنسيقها والتقاط الصور المرئية وتحليل VBA وإدارة الأوراق. يُستخدم عندما يطلب المستخدم ما يلي: (1) قراءة البيانات أو تحليلها أو البحث عنها في جداول البيانات، (2) كتابة القيم أو الصيغ في الخلايا، (3) فحص التنسيق أو الصيغ أو المخططات أو بيانات التعريف، (4) التقاط لقطات شاشة أو لقطات مرئية للأوراق، (5) تصدير الأوراق إلى CSV/JSON/Markdown، (6) إدارة الأوراق (إنشاء أو إعادة تسمية أو حذف أو نسخ أو إخفاء)، (7) تحليل وحدات ماكرو VBA أو تنفيذها، (8) سرد/تصدير الكائنات المضمنة (المخططات والأشكال والصور)، (9) التحقق من وجود أخطاء في الصيغة، أو (10) أي مهمة تتضمن تفاعل ملف Excel. تفضل البرامج النصية openpyxl/pandas - JSON الأسرع والمنظم والمحسن للذكاء الاصطناعي.
