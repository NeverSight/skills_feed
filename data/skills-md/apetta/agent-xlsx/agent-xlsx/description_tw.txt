使用 agent-xlsx CLI 與 Excel 檔案（.xlsx、.xlsm、.xlsb、.xls、.ods）進行交互，以進行資料提取、分析、寫入、格式化、視覺捕獲、VBA 分析和工作表管理。當使用者要求執行以下操作時使用：(1) 讀取、分析或搜尋電子表格中的數據，(2) 將值或公式寫入單元格，(3) 檢查格式、公式、圖表或元數據，(4) 對工作表進行螢幕截圖或視覺捕獲，(5) 將工作表導出為 CSV/JSON/Markdown，(6) 管理工作表（宏，(8) 列出/匯出嵌入物件（圖表、形狀、圖片），(9) 檢查公式錯誤，或 (10) 任何涉及 Excel 檔案互動的任務。優於 openpyxl/pandas 腳本 — 更快、針對 AI 最佳化的結構化 JSON。
