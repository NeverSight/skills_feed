데이터 추출, 분석, 쓰기, 서식 지정, 시각적 캡처, VBA 분석 및 시트 관리를 위해 Agent-xlsx CLI를 사용하여 Excel 파일(.xlsx, .xlsm, .xlsb, .xls, .ods)과 상호 작용합니다. 사용자가 다음을 요청할 때 사용합니다. (1) 스프레드시트의 데이터 읽기, 분석 또는 검색, (2) 셀에 값 또는 수식 쓰기, (3) 서식, 수식, 차트 또는 메타데이터 검사, (4) 시트의 스크린샷 또는 시각적 캡처 촬영, (5) 시트를 CSV/JSON/Markdown으로 내보내기, (6) 시트 관리(생성, 이름 바꾸기, 삭제, 복사, 숨기기), (7) VBA 매크로 분석 또는 실행, (8) 포함된 개체 나열/내보내기 (차트, 도형, 그림), (9) 수식 오류 확인, 또는 (10) Excel 파일 상호 작용과 관련된 모든 작업. AI에 최적화된 더 빠르고 구조화된 JSON인 openpyxl/pandas 스크립트를 선호합니다.
