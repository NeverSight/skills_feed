Interagisci con file Excel (.xlsx, .xlsm, .xlsb, .xls, .ods) utilizzando la CLI agent-xlsx per l'estrazione, l'analisi, la scrittura, la formattazione, l'acquisizione visiva, l'analisi VBA e la gestione dei fogli dei dati. Da utilizzare quando l'utente richiede di: (1) leggere, analizzare o cercare dati nei fogli di calcolo, (2) scrivere valori o formule nelle celle, (3) ispezionare formattazione, formule, grafici o metadati, (4) acquisire schermate o acquisizioni visive di fogli, (5) esportare fogli in CSV/JSON/Markdown, (6) gestire fogli (creare, rinominare, eliminare, copiare, nascondere), (7) analizzare o eseguire macro VBA, (8) Elenca/esporta oggetti incorporati (grafici, forme, immagini), (9) verifica errori di formula o (10) qualsiasi attività che coinvolga l'interazione con file Excel. Preferisci gli script openpyxl/pandas: JSON più veloce e strutturato, ottimizzato per l'intelligenza artificiale.
