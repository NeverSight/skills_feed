Interactúe con archivos de Excel (.xlsx, .xlsm, .xlsb, .xls, .ods) utilizando la CLI de agent-xlsx para extracción de datos, análisis, escritura, formateo, captura visual, análisis de VBA y administración de hojas. Úselo cuando el usuario solicite: (1) Leer, analizar o buscar datos en hojas de cálculo, (2) Escribir valores o fórmulas en celdas, (3) Inspeccionar formato, fórmulas, gráficos o metadatos, (4) Tomar capturas de pantalla o capturas visuales de hojas, (5) Exportar hojas a CSV/JSON/Markdown, (6) Administrar hojas (crear, renombrar, eliminar, copiar, ocultar), (7) Analizar o ejecutar macros de VBA, (8) Enumerar/exportar objetos incrustados (gráficos, formas, imágenes), (9) Verificar errores de fórmula o (10) Cualquier tarea que implique interacción con archivos de Excel. Prefiere los scripts openpyxl/pandas: JSON estructurado, más rápido y optimizado para IA.
