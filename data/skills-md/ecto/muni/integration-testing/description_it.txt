Guida test, mocking e simulazione end-to-end per la codebase Muni. Da utilizzare durante la scrittura di test di integrazione, l'impostazione di ambienti di test, la creazione di bus CAN fittizi, il test di protocolli WebSocket, la convalida di dispositivi di database o il debug di errori di test. Copre modelli di test Rust (tokio::test, test di integrazione), test TypeScript (Vitest), infrastruttura simulata (Docker Compose per test), simulazione del bus CAN, client di test WebSocket, seeding del database e convalida della registrazione Rerun. Essenziale per garantire che i componenti funzionino insieme correttamente.
