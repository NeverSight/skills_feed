Управляет сквозным тестированием, макетированием и моделированием кодовой базы Muni. Используйте при написании интеграционных тестов, настройке тестовых сред, создании макета CAN-шины, тестировании протоколов WebSocket, проверке параметров базы данных или отладке сбоев тестов. Охватывает шаблоны тестирования Rust (tokio::test, интеграционные тесты), тестирование TypeScript (Vitest), макет инфраструктуры (Docker Compose для тестов), симуляцию шины CAN, тестовые клиенты WebSocket, заполнение базы данных и проверку записи повторного запуска. Необходим для обеспечения правильной совместной работы компонентов.
