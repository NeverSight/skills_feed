Guide les tests, les simulations et la simulation de bout en bout pour la base de code Muni. À utiliser lors de l'écriture de tests d'intégration, de la configuration d'environnements de test, de la création d'un bus CAN simulé, du test des protocoles WebSocket, de la validation des appareils de base de données ou du débogage des échecs de test. Couvre les modèles de test Rust (tokio::test, tests d'intégration), les tests TypeScript (Vitest), l'infrastructure fictive (Docker Compose pour les tests), la simulation de bus CAN, les clients de test WebSocket, l'amorçage de la base de données et la validation de l'enregistrement de réexécution. Indispensable pour garantir que les composants fonctionnent correctement ensemble.
