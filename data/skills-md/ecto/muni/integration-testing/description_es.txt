Guía de pruebas, simulacros y simulaciones de un extremo a otro para el código base de Muni. Úselo al escribir pruebas de integración, configurar entornos de prueba, crear simulacros de bus CAN, probar protocolos WebSocket, validar dispositivos de bases de datos o depurar fallas de pruebas. Cubre patrones de prueba de Rust (tokio::test, pruebas de integración), pruebas de TypeScript (Vitest), infraestructura simulada (Docker Compose para pruebas), simulación de bus CAN, clientes de prueba WebSocket, inicialización de bases de datos y validación de grabaciones de repetición. Esencial para garantizar que los componentes funcionen juntos correctamente.
