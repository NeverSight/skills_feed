Обзоры микросервисов Rust в каталоге depot/ с акцентом на веб-инфраструктуру Axum, шаблоны баз данных SQLx, развертывание Docker и связь через WebSocket. Используйте при просмотре изменений в службах депо (обнаружение, отправка, Map-API, GPS-статус, Mapper), добавлении новых конечных точек, изменении схем базы данных, реализации протоколов WebSocket или отладке проблем интеграции служб. Охватывает асинхронные шаблоны Tokio, проектирование API REST/WebSocket, миграцию PostgreSQL, обработку ошибок, проверки работоспособности и многоэтапные сборки Docker.
