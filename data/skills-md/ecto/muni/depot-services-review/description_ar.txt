يراجع خدمات Rust الصغيرة في المستودع/الدليل مع التركيز على إطار عمل ويب Axum وأنماط قاعدة بيانات SQLx ونشر Docker واتصالات WebSocket. يُستخدم عند مراجعة تغييرات خدمة المستودع (الاكتشاف، أو الإرسال، أو Map-api، أو gps-status، أو مخطط الخرائط)، أو إضافة نقاط نهاية جديدة، أو تعديل مخططات قاعدة البيانات، أو تنفيذ بروتوكولات WebSocket، أو تصحيح أخطاء مشكلات تكامل الخدمة. يغطي أنماط Tokio غير المتزامنة، وتصميم REST/WebSocket API، وعمليات ترحيل PostgreSQL، ومعالجة الأخطاء، والفحوصات الصحية، وإنشاءات Docker متعددة المراحل.
