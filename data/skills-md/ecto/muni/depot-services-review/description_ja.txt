Axum Web フレームワーク、SQLx データベース パターン、Docker デプロイメント、および WebSocket 通信に焦点を当てて、depot/ ディレクトリ内の Rust マイクロサービスをレビューします。デポ サービスの変更 (検出、ディスパッチ、マップ API、GPS ステータス、マッパー) の確認、新しいエンドポイントの追加、データベース スキーマの変更、WebSocket プロトコルの実装、またはサービス統合の問題のデバッグを行うときに使用します。 Tokio の非同期パターン、REST/WebSocket API 設計、PostgreSQL の移行、エラー処理、ヘルス チェック、Docker のマルチステージ ビルドについて説明します。
