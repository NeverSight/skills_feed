Обзор встроенной прошивки Rust для микроконтроллеров RP2350 (Raspberry Pi Pico 2 Вт) и ESP32-S3 (Heltec) с упором на асинхронное время выполнения Embassy, ​​ограничения памяти, управление светодиодами, протоколы подключения CAN и мостовое соединение SLCAN. Используйте при просмотре изменений встроенного ПО MCU, отладке проблем с контроллером светодиодов, оценке асинхронных шаблонов Embassy, ​​проверке распределения статической памяти, проверке реализаций SLCAN или оценке аппаратного кода для светодиодов WS2812, периферийных устройств CAN и насадок для инструментов. Охватывает среды no_std, конечные автоматы PIO, периферийные устройства RMT и последовательную связь USB CDC.
