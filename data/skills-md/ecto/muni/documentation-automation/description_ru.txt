Автоматически поддерживает проектную документацию, включая CHANGELOG.md, файлы README, встроенную документацию по коду и перекрестные ссылки. Используйте активно после реализации функций, исправления ошибок, внесения изменений в API или выполнения важной работы. Обновляет CHANGELOG.md в обычном формате фиксации, добавляет разделы README для новых функций, генерирует встроенную документацию для новых функций/структур/компонентов и обеспечивает актуальность перекрестных ссылок на документацию. Охватывает комментарии к документации Rust, TypeScript JSDoc, обычные коммиты (feat/fix/docs/refactor/test/chore) и многоуровневую организацию README.
