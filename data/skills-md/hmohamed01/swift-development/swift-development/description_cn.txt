用于构建、测试和部署 iOS/macOS 应用程序的全面 Swift 开发。当 Claude 需要执行以下操作时使用：(1) 从命令行构建 Swift 包或 Xcode 项目，(2) 使用 XCTest 或 Swift 测试框架运行测试，(3) 使用 simctl 管理 iOS 模拟器，(4) 处理代码签名、配置文件和应用程序分发，(5) 使用 SwiftFormat/SwiftLint 格式化或检查 Swift 代码，(6) 使用 Swift 包管理器 (SPM)，(7) 实施 Swift 6 并发模式（异步/等待、参与者、可发送），(8) 使用 MVVM 架构创建 SwiftUI 视图，(9) 设置 Core Data 或 SwiftData 持久性，或任何其他 Swift/iOS/macOS 开发任务。
