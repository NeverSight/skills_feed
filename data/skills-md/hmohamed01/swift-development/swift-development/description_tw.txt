用於構建、測試和部署 iOS/macOS 應用程序的全面 Swift 開發。當 Claude 需要執行以下操作時使用：(1) 從命令行構建 Swift 包或 Xcode 項目，(2) 使用 XCTest 或 Swift 測試框架運行測試，(3) 使用 simctl 管理 iOS 模擬器，(4) 處理代碼簽名、配置文件和應用程序分發，(5) 使用 SwiftFormat/SwiftLint 格式化或檢查 Swift 代碼，(6) 使用 Swift 包管理器 (SPM)，(7) 實施 Swift 6 並發模式（異步/等待、參與者、可發送），(8) 使用 MVVM 架構創建 SwiftUI 視圖，(9) 設置 Core Data 或 SwiftData 持久性，或任何其他 Swift/iOS/macOS 開發任務。
