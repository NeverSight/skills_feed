قم بتطوير البرامج النصية والأدوات والوحدات النمطية وواجهات المستخدم الرسومية لـ PowerShell باتباع أفضل ممارسات Microsoft. يُستخدم عند كتابة تعليمات PowerShell البرمجية، أو إنشاء واجهات Windows Forms/WPF، أو العمل مع وحدات معرض PowerShell، أو الحاجة إلى توصيات cmdlet/الوحدة النمطية. يغطي تطوير البرنامج النصي، وتصميم المعلمات، ومعالجة خطوط الأنابيب، وإدارة الأخطاء، وأنماط إنشاء واجهة المستخدم الرسومية. التحقق من توفر الوحدة النمطية وبناء جملة cmdlet مقابل التوثيق المباشر عندما تكون الدقة أمرًا بالغ الأهمية.
