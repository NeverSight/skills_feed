Assistant d'échafaudage de projet de niveau IDE pour créer de nouveaux projets avec une configuration complète. Prend en charge plus de 70 types de projets : sites Web HTML/CSS, React, Next.js, Vue, Astro, Remix, React Native, Flutter, Expo, FastAPI, Django, Express, NestJS, Go/Gin, Rust/Axum, Spring Boot, Hono, Elysia, Chrome Extensions, VS Code Extensions, applications de bureau Tauri, fonctions sans serveur, etc. Fournit la création de projets au niveau WebStorm/PyCharm avec sélection interactive de SDK, configuration du framework, configuration de la base de données et outils DevOps. À utiliser lors de : la création d'un nouveau projet, la configuration d'une application-cadre, l'initialisation d'une base de code, la création d'un passe-partout, la création d'extensions, la création d'applications mobiles/de bureau/Web, la configuration de monorepos ou la création de sites Web/pages de destination statiques.
