Projektgerüst-Assistent auf IDE-Niveau zum Erstellen neuer Projekte mit umfassender Konfiguration. Unterstützt über 70 Projekttypen: HTML/CSS-Websites, React, Next.js, Vue, Astro, Remix, React Native, Flutter, Expo, FastAPI, Django, Express, NestJS, Go/Gin, Rust/Axum, Spring Boot, Hono, Elysia, Chrome-Erweiterungen, VS-Code-Erweiterungen, Tauri-Desktop-Apps, serverlose Funktionen und mehr. Bietet Projekterstellung auf WebStorm-/PyCharm-Ebene mit interaktiver SDK-Auswahl, Framework-Konfiguration, Datenbank-Setup und DevOps-Tools. Verwendung bei: Erstellen eines neuen Projekts, Einrichten einer Framework-Anwendung, Initialisieren einer Codebasis, Gerüstbau, Erstellen von Erweiterungen, Erstellen von Mobil-/Desktop-/Web-Apps, Einrichten von Monorepos oder Erstellen statischer Websites/Landingpages.
