Создайте или реорганизуйте оболочку .NET 8 WPF для узла виджетов рабочего стола с помощью CommunityToolkit.Mvvm и пользовательской службы навигации, ориентированной на модель представления. Используйте при работе с App.xaml, MainWindow/ShellWindow, общими ResourceDictionaries, светлыми и темными темами среды выполнения или макетом оболочки для приложений в стиле виджетов. Избегайте Prism, Frame Navigation и тяжелых менеджеров регионов.
