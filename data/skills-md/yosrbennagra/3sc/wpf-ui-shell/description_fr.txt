Créez ou refactorisez un shell WPF .NET 8 pour un hôte de widget de bureau à l'aide de CommunityToolkit.Mvvm et d'un service de navigation personnalisé axé sur le modèle d'affichage. À utiliser lorsque vous travaillez sur App.xaml, MainWindow/ShellWindow, des ResourceDictionaries partagés, des thèmes d'exécution clair/sombre ou une disposition de shell pour les applications de style widget. Évitez Prism, la navigation Frame et les gestionnaires de régions lourds.
