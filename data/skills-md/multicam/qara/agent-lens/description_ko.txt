Agent Lens: 다중 에이전트 Claude Code 세션에 대한 실시간 관찰 대시보드입니다.

최신 이중 창 인터페이스를 통해 상담원 상호 작용, 도구 사용 및 세션 흐름을 시각화합니다. 계층적 타임라인 시각화, HITL 승인 및 포괄적인 지표를 통해 상위-하위 이벤트 관계를 추적하세요.

**주요 기능:**
- 👁️ 이중 창 레이아웃(프로세스 타임라인 + 결과/측정항목)
- 🌳 상위-하위 관계를 통한 범위 계층 시각화
- 🔴 WebSocket을 통한 실시간 이벤트 스트리밍
- 🤖 시간 초과 표시기와 3가지 작업 패턴이 있는 HITL 인터페이스
- 📊 토큰 사용량, 비용 추적 및 성능 지표
- 🎨 OLED에 최적화된 다크 테마 (2026 기준)
- 💾 파일 시스템 기반(데이터베이스 필요 없음)
