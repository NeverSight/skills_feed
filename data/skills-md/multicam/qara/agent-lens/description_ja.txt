Agent Lens: マルチエージェントの Claude Code セッション用のリアルタイム可観測性ダッシュボード。

最新のデュアルペイン インターフェイスを通じて、エージェントの対話、ツールの使用状況、セッション フローを視覚化します。階層的なタイムラインの視覚化、HITL 承認、包括的なメトリクスを使用して、親子イベントの関係を追跡します。

**主な機能:**
- 👁️ デュアルペイン レイアウト (プロセス タイムライン + 結果/指標)
- 🌳 親子関係によるスパン階層の視覚化
- 🔴 WebSocket 経由のリアルタイム イベント ストリーミング
- 🤖 タイムアウトインジケーターと 3 つのアクションパターンを備えた HITL インターフェイス
- 📊 トークンの使用状況、コスト追跡、パフォーマンス指標
- 🎨 OLED に最適化されたダークテーマ (2026 基準)
- 💾 ファイルシステムベース (データベースは不要)
