Принципы проектирования инструментов CLI, с которыми агенты LLM/AI могут безопасно и надежно работать.
Обеспечивает 7 основных принципов (машиночитаемый, неинтерактивный, идемпотентный, безопасный по умолчанию,
Наблюдаемый, контекстно-эффективный, интроспективный) с системой показателей, шаблонами и антишаблонами.
Используйте при: (1) разработке новых инструментов CLI для использования агентами, (2) улучшении существующих CLI для агентов.
совместимости, (3) проверяем дизайн CLI на предмет удобства для агентов или (4) нуждаемся в руководстве по
Форматы вывода JSON, коды выхода, потоки аутентификации или самоописывающиеся шаблоны CLI.
