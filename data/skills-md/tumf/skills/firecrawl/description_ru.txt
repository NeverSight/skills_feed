Комплексный набор инструментов для парсинга, сканирования и извлечения данных из Интернета на базе Firecrawl API.
Предоставляет сценарии для парсинга одной страницы (scrape.py), веб-поиска (search.py),
Обнаружение URL-адресов (map.py), многостраничное сканирование (crawl.py), извлечение структурированных данных (extract.py),
и автономный сбор данных (agent.py).
Используйте, когда вам нужно: (1) извлечь контент с веб-страниц, (2) выполнить поиск и очистить Интернет,
(3) обнаруживать URL-адреса на веб-сайтах, (4) сканировать несколько страниц, (5) извлекать структурированные данные с помощью схем JSON,
или (6) автономно собирать данные из любой точки Интернета.
Требуется переменная среды FIRECRAWL_API_KEY.
