Firecrawl API を利用した包括的な Web スクレイピング、クローリング、およびデータ抽出ツールキット。
単一ページのスクレイピング (scrape.py)、Web 検索 (search.py)、
URL ディスカバリ (map.py)、マルチページ クローリング (crawl.py)、構造化データ抽出 (extract.py)、
自律的なデータ収集 (agent.py)。
次の必要がある場合に使用します: (1) Web ページからコンテンツを抽出する、(2) Web を検索してスクレイピングする、
(3) Web サイト上の URL を検出する、(4) 複数のページをクロールする、(5) JSON スキーマを使用して構造化データを抽出する、
または (6) Web 上のどこからでもデータを自律的に収集します。
FIRECRAWL_API_KEY 環境変数が必要です。
