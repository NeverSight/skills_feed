Firecrawl API를 기반으로 하는 포괄적인 웹 스크래핑, 크롤링 및 데이터 추출 툴킷입니다.
단일 페이지 스크래핑(scrape.py), 웹 검색(search.py),
URL 검색(map.py), 다중 페이지 크롤링(crawl.py), 구조화된 데이터 추출(extract.py),
자율 데이터 수집(agent.py).
(1) 웹 페이지에서 콘텐츠 추출, (2) 웹 검색 및 스크랩,
(3) 웹사이트에서 URL 검색, (4) 여러 페이지 크롤링, (5) JSON 스키마로 구조화된 데이터 추출,
또는 (6) 웹 어디에서나 자동으로 데이터를 수집합니다.
FIRECRAWL_API_KEY 환경 변수가 필요합니다.
