「ドキュメントとしてのメモリ」レイアウトを使用して、OpenClaw エージェント ワークスペースを作成および維持します。
次の必要がある場合に使用します: (1) 新しい OpenClaw ワークスペース/プロファイルを作成する、(2) ブートストラップ ファイルを生成する
(AGENTS.md、SOUL.md、USER.md、IDENTITY.md、TOOLS.md、MEMORY.md、memory/YYYY-MM-DD.md)、
(3) 命令/ペルソナ/メモリを分離し、トークンを無駄にしないようにする、または (4) 安全な git バックアップを設定する
シークレットをコミットせずに OpenClaw ワークスペースを作成します。
