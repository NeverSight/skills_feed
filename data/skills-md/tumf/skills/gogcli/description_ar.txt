واجهة سطر أوامر (CLI) سريعة وسهلة الاستخدام للنصوص البرمجية لـ Google Workspace وGmail. يُستخدم عندما تحتاج إلى التشغيل التلقائي أو التفاعل مع Gmail (بحث/إرسال/إدارة رسائل البريد الإلكتروني)، أو تقويم Google (إنشاء/إدارة الأحداث)، أو Google Drive (تحميل/تنزيل/تنظيم الملفات)، أو جداول بيانات Google/المستندات/العروض التقديمية (قراءة/كتابة/تصدير)، أو جهات الاتصال، أو المهام، أو الدردشة، أو Classroom، أو المجموعات، أو Keep. يدعم إخراج JSON والحسابات المتعددة والعمليات المجمعة والمصادقة ذات الامتيازات الأقل. مثالي لأتمتة البريد الإلكتروني وإدارة التقويم وعمليات الملفات واستخراج البيانات وإدارة مساحة العمل والتكامل مع خدمات Google.
