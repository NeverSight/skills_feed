Modèles de schéma de base de données Drizzle ORM pour les jeux, y compris les profils de joueurs, les inventaires, les classements, les sessions de jeu et les réalisations. À utiliser lors de la conception de schémas de base de données, de la création de migrations, de l'optimisation de requêtes ou de la mise en œuvre de systèmes de sauvegarde. Se déclenche sur les demandes de conception de bases de données de jeu, de schémas Drizzle, de stockage de données de joueurs ou de systèmes de classement.
