Реагируйте на шаблоны игрового пользовательского интерфейса, используя shadcn/ui, Tailwind и Framer Motion для улучшения игровых интерфейсов. Используйте при создании HUD, панелей ресурсов, табло, модальных окон, всплывающих подсказок, компонентов карт или любого игрового пользовательского интерфейса. Включает микровзаимодействия, анимацию, адаптивные макеты и специальные возможности для игр. Запускается по запросам компонентов игрового интерфейса, анимации пользовательского интерфейса или игровых шаблонов shadcn/ui.
