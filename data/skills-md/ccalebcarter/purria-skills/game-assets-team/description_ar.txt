فريق كامل لتصميم أصول اللعبة وإنشائها وتنفيذها وتحسينها. يُستخدم عند إنشاء أصول مرئية، أو توجيه فني، أو نقوش متحركة، أو عناصر واجهة مستخدم، أو أيقونات، أو مواد، أو مواصفات رسوم متحركة، أو تصميم صوتي، أو أي عناصر مرئية للعبة. يغطي إنشاء صور الذكاء الاصطناعي (Gemini)، وخطوط أنابيب الأصول، وتحسين الويب/الهاتف المحمول، وأدلة الأنماط، وضبط الأداء. يتم تشغيلها بناءً على طلبات صور اللعبة أو الرموز أو الخلفيات أو تصميمات الشخصيات أو أصول واجهة المستخدم أو المواد الترويجية أو استكشاف أخطاء الأصول وإصلاحها.
