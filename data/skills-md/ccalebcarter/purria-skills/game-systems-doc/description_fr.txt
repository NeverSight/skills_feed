Création de documents de conception de jeux de qualité AAA pour les jeux hybrides de casino et d'agriculture. À utiliser lors de la création de GDD, de spécifications de mini-jeux, de documents de conception de système, de résumés de fonctionnalités ou de documents de conception technique. Se déclenche sur les demandes de documentation sur la conception de jeux, de spécifications mécaniques, de pannes de système ou de livrables de développement de jeux professionnels. Spécialisé dans les mécanismes de jeux de cartes, les systèmes de simulation agricole et la documentation de genre hybride.
