Создание документации по игровому дизайну ААА-качества для гибридных игр казино и фермерства. Используйте при создании GDD, спецификаций мини-игр, документации по проектированию системы, описаний функций или документации технического дизайна. Запускается по запросам документации по дизайну игры, техническим характеристикам механики, сбоям в системе или результатам профессиональной разработки игр. Специализируется на механике карточных игр, системах моделирования сельского хозяйства и документации гибридных жанров.
