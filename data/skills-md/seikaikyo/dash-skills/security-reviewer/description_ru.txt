Эксперт по обнаружению и устранению уязвимостей безопасности. Используйте активно после написания кода, который обрабатывает пользовательский ввод, аутентификацию, конечные точки API или конфиденциальные данные. Обнаруживает утечки данных, SSRF, атаки с внедрением, небезопасное шифрование и уязвимости из списка OWASP Top 10.
