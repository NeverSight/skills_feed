Разработайте компоненты @ant-design/agent-ui для интерфейсов чата с искусственным интеллектом. Используйте при создании визуализации цепочки мыслей, отображений вызовов инструментов, редакторов уценок, пузырьковых компонентов, панелей рабочего пространства или любой разработки агентного пользовательского интерфейса. Запускается по таким ключевым словам, как пузырь, цепочка мыслей, использование инструментов, редактор уценок, рабочее пространство, макет чата, агент.
