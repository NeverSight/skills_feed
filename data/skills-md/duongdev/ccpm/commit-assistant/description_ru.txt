Предоставляет традиционные рекомендации по фиксации и автоматически генерирует сообщения о фиксации на основе изменений git. Интегрируется с /ccpm:commit для автоматических коммитов git, связанных с линейными проблемами. Автоматически активируется, когда пользователи спрашивают о фиксации, создании коммитов git или обсуждении форматов сообщений о фиксации.
