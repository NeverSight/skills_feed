Testa gli endpoint API REST e GraphQL con asserzioni e report strutturati. Da utilizzare quando un utente chiede di testare un'API, raggiungere un endpoint, verificare se un'API funziona, convalidare una risposta, eseguire il debug di una chiamata API, testare i flussi di autenticazione o verificare i contratti API. Supporta GET, POST, PUT, PATCH, DELETE con intestazioni, corpo, autenticazione e convalida della risposta.
