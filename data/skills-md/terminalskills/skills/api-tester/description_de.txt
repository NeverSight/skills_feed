Testen Sie REST- und GraphQL-API-Endpunkte mit strukturierten Aussagen und Berichten. Verwenden Sie diese Option, wenn ein Benutzer eine API testen, einen Endpunkt erreichen, prüfen möchte, ob eine API funktioniert, eine Antwort validieren, einen API-Aufruf debuggen, Authentifizierungsabläufe testen oder API-Verträge überprüfen möchte. Unterstützt GET, POST, PUT, PATCH, DELETE mit Headern, Text, Authentifizierung und Antwortvalidierung.
