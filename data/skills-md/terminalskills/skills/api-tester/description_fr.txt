Testez les points de terminaison des API REST et GraphQL avec des assertions et des rapports structurés. À utiliser lorsqu'un utilisateur demande de tester une API, d'atteindre un point de terminaison, de vérifier si une API fonctionne, de valider une réponse, de déboguer un appel d'API, de tester des flux d'authentification ou de vérifier des contrats d'API. Prend en charge GET, POST, PUT, PATCH, DELETE avec les en-têtes, le corps, l'authentification et la validation des réponses.
