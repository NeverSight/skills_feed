اختبر نقاط نهاية REST وGraphQL API من خلال التأكيدات والتقارير المنظمة. يُستخدم عندما يطلب المستخدم اختبار واجهة برمجة التطبيقات، أو الوصول إلى نقطة نهاية، أو التحقق من عمل واجهة برمجة التطبيقات، أو التحقق من صحة الاستجابة، أو تصحيح استدعاء واجهة برمجة التطبيقات، أو اختبار تدفقات المصادقة، أو التحقق من عقود واجهة برمجة التطبيقات. يدعم GET، POST، PUT، PATCH، DELETE مع الرؤوس والنص والمصادقة والتحقق من الاستجابة.
