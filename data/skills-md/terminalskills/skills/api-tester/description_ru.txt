Тестируйте конечные точки API REST и GraphQL с помощью структурированных утверждений и отчетов. Используйте, когда пользователь запрашивает протестировать API, обратиться к конечной точке, проверить, работает ли API, проверить ответ, отладить вызов API, протестировать потоки аутентификации или проверить контракты API. Поддерживает GET, POST, PUT, PATCH, DELETE с заголовками, телом, аутентификацией и проверкой ответа.
