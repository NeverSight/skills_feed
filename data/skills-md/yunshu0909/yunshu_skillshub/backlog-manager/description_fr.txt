Gestion du pool de besoins. Les utilisateurs peuvent proposer des idées/des problèmes à tout moment ; l'IA les suit, les organise, les fusionne et les archive dans un fichier de pool d'exigences. Lorsque l'utilisateur est prêt à démarrer une nouvelle version, cela permet de filtrer le pool. Axé sur les points sensibles, sans planification préalable.
