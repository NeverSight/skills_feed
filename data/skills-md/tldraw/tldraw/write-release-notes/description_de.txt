Schreiben von Artikeln mit Versionshinweisen für tldraw SDK-Versionen. Verwenden Sie es, wenn Sie eine neue Versionsdokumentation erstellen, Versionshinweise von Grund auf entwerfen oder die Qualität von Versionshinweisen überprüfen. Bietet Anleitungen zu Struktur, Stimme und Inhalt für Release-Dateien in „apps/docs/content/releases/“.
