Комплексный помощник по разработке Unreal Engine C++ и Blueprint с глубоким пониманием структуры проекта. Используйте при помощи с проектами Unreal Engine, включая: программирование игрового процесса на C++, разработку чертежей, настройку системы ввода (расширенный ввод), систему возможностей игрового процесса (GAS), навигацию по структуре проекта, обнаружение и ссылки на ресурсы, интеграцию плагинов (экспериментальная/бета-версия), поиск API для недостаточно документированных функций и отладку. Запускается при любом вопросе разработки Unreal Engine, особенно при работе в каталоге .uproject.
