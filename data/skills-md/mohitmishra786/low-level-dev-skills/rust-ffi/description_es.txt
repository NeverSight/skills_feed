Habilidad Rust FFI para la interoperabilidad de C. Úselo al llamar a bibliotecas de C desde Rust, generar enlaces de Rust con bindgen, exportar funciones de Rust a C con cbindgen, escribir envoltorios seguros alrededor de FFI inseguros o vincular bibliotecas de sistemas y proveedores. Se activa en consultas sobre bindgen, cbindgen, "C" externo, FFI inseguro, enlaces de Rust C, vinculación de C desde Rust o cajas de sistema.
