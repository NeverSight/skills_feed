مهارة الصدأ FFI لقابلية التشغيل البيني C. يُستخدم عند استدعاء مكتبات C من Rust، أو إنشاء روابط Rust باستخدام bindgen، أو تصدير وظائف Rust إلى C باستخدام cbindgen، أو كتابة أغلفة آمنة حول FFI غير الآمن، أو ربط مكتبات النظام والبائعين. يتم التنشيط عند الاستعلامات حول bindgen، أو cbindgen، أو extern "C"، أو FFI غير الآمن، أو روابط Rust C، أو ربط C من Rust، أو صناديق sys.
