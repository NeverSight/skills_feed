Rust-Kenntnisse für unsicheren Code für die Systemprogrammierung. Verwenden Sie es, wenn Sie unsicheres Rust schreiben oder überprüfen, verstehen, welche Vorgänge unsicher sind, sichere Abstraktionen über unsicheren Code implementieren, unsichere Blöcke prüfen oder Rohzeiger, Transmute und Extern verstehen. Aktiviert bei Abfragen zu unsicherem Rust, Rohzeigern, Transmutierung, unsicheren Blöcken, dem Schreiben sicherer Wrapper, UnsafeCell, unsicheren Trait-Impl oder der Prüfung von unsicherem Code.
