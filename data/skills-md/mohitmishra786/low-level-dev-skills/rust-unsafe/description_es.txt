Habilidad de código inseguro de Rust para la programación de sistemas. Úselo al escribir o revisar Rust inseguro, comprender qué operaciones requieren inseguridad, implementar abstracciones seguras sobre código inseguro, auditar bloques inseguros o comprender punteros sin formato, transmutar y externos. Se activa ante consultas sobre Rust inseguro, punteros sin formato, transmutación, bloques inseguros, escritura de envoltorios seguros, UnsafeCell, impl. de rasgo inseguro o auditoría de código inseguro.
