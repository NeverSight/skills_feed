مهارة GNU Make لأنظمة بناء C/C++. يُستخدم عند كتابة ملفات Makefiles أو تصحيح أخطاءها، أو فهم قواعد النمط وإنشاء التبعية التلقائية، أو إدارة CFLAGS/LDFLAGS، أو تحويل أوامر الترجمة المخصصة إلى ملفات Makefiles قابلة للصيانة، أو تشخيص مشكلات البناء المتزايدة. يتم التنشيط عند الاستعلامات حول Makefiles، أو إنشاء أهداف، أو قواعد النمط، أو أهداف زائفة، أو تتبع التبعية، أو إنشاء تكراري، أو ارتكاب أخطاء.
