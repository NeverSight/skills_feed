Помогите пользователям совместно писать документы с помощью структурированного рабочего процесса. Используйте этот рабочий процесс, когда пользователи хотят создавать документы, предложения, технические спецификации, записи решений или аналогичный структурированный контент. Это помогает пользователям эффективно передавать контекст, уточнять контент посредством итерации и проверять, готов ли документ для целевой аудитории. Запускайте этот рабочий процесс всякий раз, когда пользователи упоминают о написании документа, создании предложения, составлении спецификации или подобных задачах документации.
