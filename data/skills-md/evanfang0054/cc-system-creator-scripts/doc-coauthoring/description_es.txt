Guíe a los usuarios en la redacción colaborativa de documentos a través de un flujo de trabajo estructurado. Utilice este flujo de trabajo cuando los usuarios quieran crear documentos, propuestas, especificaciones técnicas, registros de decisiones o contenido estructurado similar. Ayuda a los usuarios a transmitir contexto de manera eficiente, refinar el contenido mediante iteración y verificar que el documento esté listo para el público objetivo. Active este flujo de trabajo cada vez que los usuarios mencionen escribir un documento, crear una propuesta, redactar una especificación o tareas de documentación similares.
