Leiten Sie Benutzer beim gemeinsamen Schreiben von Dokumenten durch einen strukturierten Workflow. Verwenden Sie diesen Workflow, wenn Benutzer Dokumente, Vorschläge, technische Spezifikationen, Entscheidungsaufzeichnungen oder ähnliche strukturierte Inhalte erstellen möchten. Es hilft Benutzern, Kontext effizient zu vermitteln, Inhalte durch Iteration zu verfeinern und zu überprüfen, ob das Dokument für die beabsichtigte Zielgruppe bereit ist. Lösen Sie diesen Workflow aus, wenn Benutzer das Schreiben eines Dokuments, das Erstellen eines Angebots, das Verfassen einer Spezifikation oder ähnliche Dokumentationsaufgaben erwähnen.
