Interrogez le centre de données régional de Pennsylvanie occidentale (WPRDC) de Pittsburgh : plus de 363 ensembles de données couvrant les évaluations foncières, la qualité de l'air, 311 demandes, le recensement des prisons, les données sur les surdoses, les permis, les violations, etc. Exécutez des requêtes SQL sur des tables dynamiques sans téléchargement. À utiliser lorsque l'utilisateur pose des questions sur les données publiques du comté de Pittsburgh/Allegheny, les recherches de propriétés, les données civiques ou les statistiques régionales.
