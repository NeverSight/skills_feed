Figma MCP と Playwright を使用した Next.js/React の自律的なピクセルパーフェクト UI 実装ループ。反復的な視覚的検証により、Figma デザインを本番環境に対応したコンポーネントに変換します。 **自動トリガー** は、「この Figma を実装する」、「Figma からこのページ/コンポーネントを構築する」、「Figma デザインから作成」、「デザインを実装」、「このブロックを構築」、「デザインからコンポーネントを作成」など、Figma デザインを実装するためのあらゆるリクエストで実行されます。用途: (1) Figma からのページ/コンポーネントの構築、(2) ピクセルパーフェクトな精度、(3) レスポンシブ レイアウト、(4) デザイン トークンの変換。
