Boucle d'implémentation d'interface utilisateur autonome au pixel près pour Next.js/React à l'aide de Figma MCP et Playwright. Convertit les conceptions Figma en composants prêts pour la production avec validation visuelle itérative. ** DÉCLENCHEURS AUTOMATIQUES ** sur TOUTE demande d'implémentation de conceptions Figma, y ​​compris : "implémenter ce Figma", "construire cette page/composant à partir de Figma", "créer à partir du design Figma", "implémenter le design", "construire ce bloc", "créer un composant à partir du design". À utiliser pour : (1) Création de pages/composants à partir de Figma, (2) Précision parfaite au pixel près, (3) Mises en page réactives, (4) Conversion de jetons de conception.
