Используйте при организации параллельных фаз выполнения плана — создает изолированные рабочие деревья для одновременного выполнения задач, устанавливает зависимости, порождает параллельные субагенты, проверяет завершение, линейно укладывает ветви и очищает (обязательно для ВСЕХ параллельных фаз, включая N=1).
