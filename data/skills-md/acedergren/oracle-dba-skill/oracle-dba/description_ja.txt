OCI 上の自律型データベース (ADB) に関する Oracle DBA および DevOps の専門知識。このスキルは、Oracle Autonomous Databaseの管理、最適化されたSQL/PLSQLの作成、セキュリティの構成(TDE、Database Vault、Data Safe)、HA/DR(Data Guard、PITR)の実装、データベース操作でのOCI CLIの使用、またはAI支援データベース管理のためのOracle MCPサーバーとの統合を行うときに使用する必要があります。 Oracle Database バージョン 19c、21c、23ai、および 26ai をカバーします。
