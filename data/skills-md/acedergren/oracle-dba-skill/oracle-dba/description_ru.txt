Опыт Oracle DBA и DevOps для автономной базы данных (ADB) на OCI. Этот навык следует использовать при управлении автономными базами данных Oracle, написании оптимизированного SQL/PLSQL, настройке безопасности (TDE, Database Vault, Data Safe), реализации HA/DR (Data Guard, PITR), использовании OCI CLI для операций с базами данных или интеграции с серверами Oracle MCP для управления базами данных с помощью искусственного интеллекта. Охватывает версии Oracle Database 19c, 21c, 23ai и 26ai.
