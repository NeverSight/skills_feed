Experiencia en Oracle DBA y DevOps para bases de datos autónomas (ADB) en OCI. Esta habilidad debe usarse al administrar bases de datos autónomas de Oracle, escribir SQL/PLSQL optimizado, configurar la seguridad (TDE, Database Vault, Data Safe), implementar HA/DR (Data Guard, PITR), usar OCI CLI para operaciones de bases de datos o integrarse con servidores Oracle MCP para la administración de bases de datos asistida por IA. Cubre las versiones 19c, 21c, 23ai y 26ai de Oracle Database.
