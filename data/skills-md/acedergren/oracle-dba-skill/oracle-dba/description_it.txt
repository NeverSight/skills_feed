Competenze Oracle DBA e DevOps per Autonomous Database (ADB) su OCI. Questa competenza deve essere utilizzata durante la gestione di Oracle Autonomous Database, la scrittura di SQL/PLSQL ottimizzato, la configurazione della sicurezza (TDE, Database Vault, Data Safe), l'implementazione di HA/DR (Data Guard, PITR), l'utilizzo della CLI OCI per le operazioni del database o l'integrazione con i server Oracle MCP per la gestione del database assistita dall'intelligenza artificiale. Copre le versioni Oracle Database 19c, 21c, 23ai e 26ai.
