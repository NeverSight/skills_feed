Expertise Oracle DBA et DevOps pour Autonomous Database (ADB) sur OCI. Cette compétence doit être utilisée lors de la gestion de bases de données autonomes Oracle, de l'écriture de SQL/PLSQL optimisé, de la configuration de la sécurité (TDE, Database Vault, Data Safe), de la mise en œuvre de HA/DR (Data Guard, PITR), de l'utilisation d'OCI CLI pour les opérations de base de données ou de l'intégration avec des serveurs Oracle MCP pour la gestion de bases de données assistée par l'IA. Couvre les versions 19c, 21c, 23ai et 26ai d'Oracle Database.
