خبرة Oracle DBA وDevOps في قاعدة البيانات الذاتية (ADB) على OCI. يجب استخدام هذه المهارة عند إدارة قواعد بيانات Oracle المستقلة، وكتابة SQL/PLSQL محسنة، وتكوين الأمان (TDE، وDatabase Vault، وData Safe)، وتنفيذ HA/DR (Data Guard، PITR)، واستخدام OCI CLI لعمليات قاعدة البيانات، أو التكامل مع خوادم Oracle MCP لإدارة قواعد البيانات بمساعدة الذكاء الاصطناعي. يغطي إصدارات قاعدة بيانات Oracle 19c و21c و23ai و26ai.
