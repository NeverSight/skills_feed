À utiliser lors de la mise en œuvre de l'automatisation basée sur les événements, de la configuration des règles CloudEvents, du dépannage des échecs de diffusion d'événements ou de l'intégration avec Fonctions/Streaming/Notifications. Couvre les modèles de règles d'événement, la syntaxe de filtre, les types d'actions, la configuration de la file d'attente de lettres mortes et les anti-modèles d'architecture basés sur les événements.
