Полнофункциональный конвейер — предполетные проверки, цикл TDD, защита объема, подтверждение качества. Объединяет предполетную подготовку + tdd + проверку объема + подтверждение качества в один поток. Используйте при реализации функции, добавлении конечной точки или создании любого нетривиального изменения кода.
