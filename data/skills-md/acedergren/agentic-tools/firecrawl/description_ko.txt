모든 URL에서 깨끗한 마크다운을 반환하는 웹 스크래핑 및 검색 CLI(JS 렌더링 처리)
페이지, SPA). 사용자가 요청할 때 사용: (1) "웹에서 X 검색", (2) "스크래핑/가져오기"
URL 콘텐츠", (3) "웹사이트에서 콘텐츠 가져오기", (4) "X에 관한 최근 기사 찾기",
(5) 현재 웹 데이터가 필요한 연구 작업, (6) 페이지에서 구조화된 데이터를 추출합니다.

LLM 친화적인 Markdown 출력, Firecrawl 로그인을 통한 인증 처리, 지원
대량 작업을 위한 병렬 스크래핑. .firecrawl/ 디렉터리에 자동으로 씁니다.

트리거: 웹 스크래핑, 웹 검색, URL 가져오기, 콘텐츠 추출, Firecrawl, 스크랩
웹사이트, 페이지 콘텐츠 가져오기, 웹 조사, 사이트 맵, 사이트 크롤링.
