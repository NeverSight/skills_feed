Fügen Sie Testabdeckung zu vorhandenem Code mit korrekten Scheinmustern hinzu. Verwenden Sie diese Option, wenn Sie Tests zu ungetesteten Modulen hinzufügen, Regressionstests für Fehler schreiben oder wenn der Benutzer eine bestimmte Datei testen möchte. Behandelt die Dienstprogramme „mockReset:true“, „vi.hoisted()“, „Weiterleitungsmuster“ und „Test-App-Builder“.
