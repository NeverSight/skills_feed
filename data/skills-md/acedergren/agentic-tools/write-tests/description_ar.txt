أضف تغطية اختبارية إلى التعليمات البرمجية الموجودة باستخدام الأنماط الوهمية الصحيحة. يُستخدم عند إضافة اختبارات إلى وحدات لم يتم اختبارها، أو كتابة اختبارات الانحدار للأخطاء، أو عندما يطلب المستخدم اختبار ملف معين. يتعامل مع mockReset:true، وvi.hoisted()، ونمط إعادة التوجيه، واختبار الأدوات المساعدة لمنشئ التطبيق.
