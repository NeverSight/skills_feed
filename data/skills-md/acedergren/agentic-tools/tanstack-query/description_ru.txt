Руководство эксперта по TanStack Query v5 — ошибки миграции (критические изменения v4 → v5),
проблемы с производительностью (бесконечные циклы выборки, ловушки устаревания) и принятие решений
фреймворки (когда НЕ использовать запросы, компромиссы между SWR и React Query).

Используйте, когда: (1) отладка ошибок миграции v4 → v5 (gcTime, isPending, throwOnError),
(2) бесконечные циклы повторной выборки, (3) несоответствия гидратации SSR, (4) выбор между
React Query, SWR и выборка, (5) оптимистичные шаблоны обновления не работают.

НЕ для базовой настройки (см. официальную документацию). Фокусируется на неочевидных решениях и
шаблоны, которые вызывают производственные проблемы.

Триггеры: React Query, TanStack Query, миграция v5, цикл обновления, устаревшие данные,
Гидратация SSR, аннулирование запросов, отладка оптимистичных обновлений.
