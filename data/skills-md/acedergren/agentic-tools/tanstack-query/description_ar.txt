إرشادات الخبراء لـ TanStack Query v5 - مشاكل الترحيل (التغييرات العاجلة v4 → v5)،
مخاطر الأداء (حلقات الجلب اللانهائية، ومصائد الجمود)، والقرار
الأطر (عندما لا تستخدم الاستعلامات، مقايضات SWR مقابل React Query).

يُستخدم عندما: (1) تصحيح أخطاء الترحيل v4 → v5 (gcTime، isPending، throwOnError)،
(2) حلقات إعادة جلب لا نهائية، (3) عدم تطابق ترطيب SSR، (4) الاختيار بين
React Query vs SWR vs fetch، (5) أنماط التحديث المتفائلة لا تعمل.

ليس للإعداد الأساسي (انظر المستندات الرسمية). يركز على القرارات غير الواضحة و
الأنماط التي تسبب مشاكل الإنتاج.

المشغلات: React Query، وTanStack Query، وترحيل الإصدار 5، وحلقة الإعادة، والبيانات القديمة،
ترطيب SSR، وإبطال الاستعلام، وتصحيح التحديثات المتفائلة.
