Cree, valide y evolucione documentos de requisitos de productos con descubrimiento interactivo, arquitectura técnica, fases, protocolo TDD y análisis de dependencia. Úselo cuando: escriba un PRD, planifique una función, defina requisitos o actualice un PRD existente. Admite /prd (crear), /prd --update (actualización incremental), /prd --validate (ejecutar lista de verificación), /prd --audit-deps (solo análisis de dependencia), /prd --to-plan (generar un plan de tareas listo para orquestar).
