Управляет сеансами удаленного рабочего стола Windows для автоматизации, тестирования и удаленного администрирования. Используйте, когда пользователю необходимо подключиться к компьютерам Windows через RDP, сделать снимки экрана, щелкнуть мышью, ввести текст или взаимодействовать с удаленными рабочими столами Windows.
