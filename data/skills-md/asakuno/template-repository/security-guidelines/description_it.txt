Capacità di diagnostica e revisione della sicurezza conforme alle linee guida IPA per le applicazioni Laravel/React. Da utilizzare quando si eseguono controlli di sicurezza durante l'implementazione o la revisione del codice. Diagnostica 11 tipi di vulnerabilità (SQL injection, XSS, CSRF, ecc.), fornisce esempi di codice sicuro ed esegue la convalida basata su checklist. Funziona insieme a .claude/rules/security/ per fornire flussi di lavoro diagnostici concreti durante le fasi di implementazione. Si attiva quando: (1) revisione del codice per vulnerabilità di sicurezza, (2) implementazione di autenticazione/autorizzazione, (3) gestione di input/output dell'utente, (4) utilizzo di sessioni/cookie, (5) elaborazione di file o esecuzione di comandi, (6) creazione di moduli o API, (7) esecuzione di controlli di sicurezza.
