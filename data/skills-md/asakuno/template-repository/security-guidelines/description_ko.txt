Laravel/React 애플리케이션에 대한 IPA 지침 준수 보안 진단 및 검토 기술입니다. 코드 구현이나 검토 중에 보안 검사를 수행할 때 사용합니다. 11가지 취약점 유형(SQL 인젝션, XSS, CSRF 등)을 진단하고 안전한 코드 예시를 제공하며 체크리스트 기반 검증을 수행합니다. .claude/rules/security/와 함께 작동하여 구현 단계에서 구체적인 진단 워크플로우를 제공합니다. (1) 보안 취약성에 대한 코드 검토, (2) 인증/권한 부여 구현, (3) 사용자 입력/출력 처리, (4) 세션/쿠키 작업, (5) 파일 처리 또는 명령 실행, (6) 양식 또는 API 생성, (7) 보안 감사 수행 등의 경우에 트리거됩니다.
