IPAガイドラインに準拠したLaravel/Reactアプリケーションのセキュリティ診断・レビュースキル。コードの実装またはレビュー中にセキュリティ チェックを実行するときに使用します。 11 種類の脆弱性 (SQL インジェクション、XSS、CSRF など) を診断し、安全なコード例を提供し、チェックリスト ベースの検証を実行します。 .claude/rules/security/ と連携して、実装段階で具体的な診断ワークフローを提供します。トリガーされるタイミング: (1) セキュリティ脆弱性のコードのレビュー、(2) 認証/認可の実装、(3) ユーザー入力/出力の処理、(4) セッション/Cookie の操作、(5) ファイルの処理またはコマンドの実行、(6) フォームまたは API の作成、(7) セキュリティ監査の実行。
