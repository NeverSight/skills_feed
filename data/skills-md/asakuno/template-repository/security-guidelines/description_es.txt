Habilidad de revisión y diagnóstico de seguridad que cumple con las pautas de IPA para aplicaciones Laravel/React. Úselo al realizar controles de seguridad durante la implementación o revisión del código. Diagnostica 11 tipos de vulnerabilidad (inyección SQL, XSS, CSRF, etc.), proporciona ejemplos de código seguro y realiza una validación basada en listas de verificación. Funciona en conjunto con .claude/rules/security/ para proporcionar flujos de trabajo de diagnóstico concretos durante las fases de implementación. Se activa cuando: (1) revisa el código en busca de vulnerabilidades de seguridad, (2) implementa autenticación/autorización, (3) maneja la entrada/salida del usuario, (4) trabaja con sesiones/cookies, (5) procesa archivos o ejecuta comandos, (6) crea formularios o API, (7) realiza auditorías de seguridad.
