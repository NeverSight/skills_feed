Compétences de diagnostic et d'examen de sécurité conformes aux directives IPA pour les applications Laravel/React. À utiliser lors de l'exécution de contrôles de sécurité lors de l'implémentation ou de la révision du code. Diagnostique 11 types de vulnérabilités (injection SQL, XSS, CSRF, etc.), fournit des exemples de code sécurisés et effectue une validation basée sur une liste de contrôle. Fonctionne en conjonction avec .claude/rules/security/ pour fournir des flux de travail de diagnostic concrets pendant les phases de mise en œuvre. Se déclenche lorsque : (1) l'examen du code pour détecter les vulnérabilités de sécurité, (2) la mise en œuvre de l'authentification/autorisation, (3) la gestion des entrées/sorties de l'utilisateur, (4) l'utilisation de sessions/cookies, (5) le traitement de fichiers ou l'exécution de commandes, (6) la création de formulaires ou d'API, (7) la réalisation d'audits de sécurité.
