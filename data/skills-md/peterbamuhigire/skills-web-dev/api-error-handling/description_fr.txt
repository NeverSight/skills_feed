Système complet et standardisé de réponse aux erreurs pour les API PHP REST avec intégration SweetAlert2. À utiliser lors de la création d'API REST nécessitant un formatage d'erreur cohérent, une extraction de messages d'erreur spécifiques à partir d'exceptions de base de données, une gestion des erreurs de validation et une intégration transparente du frontend. Comprend l'analyse PDOException, l'extraction de règles métier et les modèles complets d'affichage des erreurs SweetAlert2.
