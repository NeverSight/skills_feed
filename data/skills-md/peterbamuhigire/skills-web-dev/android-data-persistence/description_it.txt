Standard di persistenza dei dati Android con Room come spazio di archiviazione locale principale e backend API personalizzati per la sincronizzazione cloud. Copre SharedPreferences, DataStore, Room (entità, DAO, relazioni, migrazioni), archiviazione di file, architettura offline-first e modello di repository per la sincronizzazione supportata da API. Da utilizzare quando si implementano funzionalità di archiviazione, memorizzazione nella cache o sincronizzazione dei dati nelle app Android.
