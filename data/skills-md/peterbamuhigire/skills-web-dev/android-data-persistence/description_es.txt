Estándares de persistencia de datos de Android con Room como almacenamiento local principal y backends API personalizados para sincronización en la nube. Cubre SharedPreferences, DataStore, Room (entidades, DAO, relaciones, migraciones), almacenamiento de archivos, arquitectura fuera de línea y patrón de repositorio para sincronización respaldada por API. Úselo al implementar cualquier función de almacenamiento, caché o sincronización de datos en aplicaciones de Android.
