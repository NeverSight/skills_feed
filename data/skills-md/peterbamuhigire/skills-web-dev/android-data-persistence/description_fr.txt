Normes de persistance des données Android avec Room comme stockage local principal et backends d'API personnalisés pour la synchronisation dans le cloud. Couvre SharedPreferences, DataStore, Room (entités, DAO, relations, migrations), le stockage de fichiers, l'architecture hors ligne et le modèle de référentiel pour la synchronisation basée sur l'API. À utiliser lors de la mise en œuvre d'une fonctionnalité de stockage de données, de mise en cache ou de synchronisation dans les applications Android.
