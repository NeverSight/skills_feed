Стандарты сохранения данных Android с Room в качестве основного локального хранилища и настраиваемыми серверными API для синхронизации с облаком. Охватывает SharedPreferences, DataStore, Room (сущности, DAO, отношения, миграции), хранилище файлов, автономную архитектуру и шаблон репозитория для синхронизации с помощью API. Используйте при реализации любых функций хранения, кэширования или синхронизации данных в приложениях Android.
