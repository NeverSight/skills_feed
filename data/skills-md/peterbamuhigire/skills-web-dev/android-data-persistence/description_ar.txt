معايير استمرارية بيانات Android مع الغرفة كتخزين محلي أساسي وواجهات خلفية API مخصصة للمزامنة السحابية. يغطي SharedPreferences، وDataStore، والغرفة (الكيانات، وDAOs، والعلاقات، وعمليات الترحيل)، وتخزين الملفات، والبنية غير المتصلة أولاً، ونمط المستودع للمزامنة المدعومة بواجهة برمجة التطبيقات (API). يُستخدم عند تنفيذ أي ميزة لتخزين البيانات أو التخزين المؤقت أو المزامنة في تطبيقات Android.
