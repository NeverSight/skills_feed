Modèle de pagination curseur/décalage pour les API PHP REST et Android (Jetpack Compose + MVVM). Couvre le format de réponse backend, les DTO Android, le référentiel, le cas d'utilisation, l'état ViewModel et LazyColumn à défilement infini. À utiliser lors de l’ajout d’une pagination à n’importe quel point de terminaison de liste.
