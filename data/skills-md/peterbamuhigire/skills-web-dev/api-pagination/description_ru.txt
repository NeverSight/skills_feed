Шаблон нумерации курсора/смещения для PHP REST API и Android (Jetpack Compose + MVVM). Охватывает формат ответа серверной части, DTO Android, репозиторий, вариант использования, состояние ViewModel и LazyColumn с бесконечной прокруткой. Используйте при добавлении нумерации страниц к любой конечной точке списка.
