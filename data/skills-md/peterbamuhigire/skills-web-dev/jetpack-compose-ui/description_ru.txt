Стандарты пользовательского интерфейса Jetpack Compose для красивых, изящных и минималистичных приложений для Android. Обеспечивает дизайн Material 3, однонаправленный поток данных, подъем состояния, согласованное оформление тем, плавную анимацию и шаблоны производительности. Используйте при создании или проверке кода Compose UI, чтобы обеспечить современные, удобные и быстро загружающиеся интерфейсы, которые являются стандартными для всех приложений.
