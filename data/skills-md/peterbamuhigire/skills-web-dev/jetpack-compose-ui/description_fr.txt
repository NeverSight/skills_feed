Normes d'interface utilisateur Jetpack Compose pour des applications Android belles, élégantes et minimalistes. Applique la conception Material 3, le flux de données unidirectionnel, le levage d’état, les thèmes cohérents, les animations fluides et les modèles de performances. À utiliser lors de la création ou de la révision du code de l'interface utilisateur de Compose pour garantir des interfaces modernes, conviviales et à chargement rapide, standard dans toutes les applications.
