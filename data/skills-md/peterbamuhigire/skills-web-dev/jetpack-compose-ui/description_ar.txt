معايير Jetpack Compose UI لتطبيقات Android الجميلة والأنيقة والبسيطة. يفرض تصميم المادة 3، وتدفق البيانات أحادي الاتجاه، ورفع الحالة، والسمات المتسقة، والرسوم المتحركة السلسة، وأنماط الأداء. استخدمه عند إنشاء رمز Compose UI أو مراجعته لضمان واجهات حديثة وسهلة الاستخدام وسريعة التحميل ومعيارية عبر جميع التطبيقات.
