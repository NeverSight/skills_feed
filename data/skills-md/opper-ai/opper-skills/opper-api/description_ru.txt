Используйте Opper REST API напрямую через HTTP для выполнения задач ИИ, структурированного вывода с помощью схемы JSON, потоковой передачи через SSE, семантического поиска в базе знаний, управления функциями и трассировки. Активируйте при вызове API Opper с помощью Curl, Fetch или любого HTTP-клиента без использования Python или Node SDK.
