Utilisez l'API REST Opper directement via HTTP pour l'exécution des tâches d'IA, la sortie structurée avec le schéma JSON, le streaming via SSE, la recherche sémantique dans la base de connaissances, la gestion des fonctions et le traçage. Activez-le lors de l'appel de l'API Opper avec curl, fetch ou tout autre client HTTP sans utiliser les SDK Python ou Node.
