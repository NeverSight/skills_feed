Utilice la API REST de Opper directamente a través de HTTP para completar tareas de IA, salida estructurada con esquema JSON, transmisión a través de SSE, búsqueda semántica en la base de conocimientos, administración de funciones y seguimiento. Actívelo al llamar a la API de Opper con curl, fetch o cualquier cliente HTTP sin usar los SDK de Python o Node.
