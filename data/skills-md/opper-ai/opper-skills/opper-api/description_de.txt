Verwenden Sie die Opper REST API direkt über HTTP für die Erledigung von KI-Aufgaben, strukturierte Ausgabe mit JSON-Schema, Streaming über SSE, semantische Suche in der Wissensdatenbank, Funktionsverwaltung und Ablaufverfolgung. Aktivieren Sie diese Option, wenn Sie die Opper-API mit Curl, Fetch oder einem beliebigen HTTP-Client aufrufen, ohne die Python- oder Node-SDKs zu verwenden.
