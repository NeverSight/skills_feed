استخدم Opper TypeScript Agent SDK (@opperai/agents) لإنشاء وكلاء الذكاء الاصطناعي باستخدام حلقات التفكير والفعل، واستخدام الأداة عبر createFunctionTool أو @tool Decorator، والتحقق من صحة مخطط Zod، والتكوين متعدد الوكلاء، وتكامل MCP، والتدفق، والخطافات لقابلية المراقبة. يمكنك التنشيط عند إنشاء وكلاء TypeScript AI أو أنظمة مستقلة أو حلقات استدلالية تستخدم الأدوات باستخدام منصة Opper.
