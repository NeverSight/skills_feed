Verwenden Sie das Opper TypeScript Agent SDK (@opperai/agents), um KI-Agenten mit Denk-Handlungs-Argumentationsschleifen, Tool-Nutzung über createFunctionTool oder @tool decorator, Zod-Schema-Validierung, Multi-Agent-Komposition, MCP-Integration, Streaming und Hooks für Beobachtbarkeit zu erstellen. Aktivieren Sie diese Funktion, wenn Sie mit der Opper-Plattform TypeScript-KI-Agenten, autonome Systeme oder Tool-verwendende Argumentationsschleifen erstellen.
