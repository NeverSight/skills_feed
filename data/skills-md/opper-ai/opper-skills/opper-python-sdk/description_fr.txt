Utilisez le SDK Opper Python (opperai) pour l'exécution des tâches d'IA, la sortie structurée avec le schéma JSON, la recherche sémantique dans la base de connaissances et le traçage/observabilité. Activez-le lors de la création d'applications Python nécessitant l'exécution de tâches basées sur LLM, des pipelines RAG, l'indexation de documents ou l'orchestration de fonctions IA avec la plateforme Opper.
