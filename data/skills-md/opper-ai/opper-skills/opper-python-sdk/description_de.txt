Verwenden Sie das Opper Python SDK (opperai) für die Erledigung von KI-Aufgaben, die strukturierte Ausgabe mit JSON-Schema, die semantische Suche in der Wissensdatenbank und die Nachverfolgung/Beobachtbarkeit. Aktivieren Sie diese Option, wenn Sie Python-Anwendungen erstellen, die LLM-gestützte Aufgabenerledigung, RAG-Pipelines, Dokumentindizierung oder KI-Funktionsorchestrierung mit der Opper-Plattform benötigen.
