استخدم Opper Python Agent SDK (وكلاء opper) لإنشاء وكلاء الذكاء الاصطناعي باستخدام حلقات التفكير والفعل، واستخدام الأداة عبر @tool Decorator، والتكوين متعدد الوكلاء، وتكامل MCP، والذاكرة المستمرة، وخطافات لقابلية الملاحظة، والتحقق من صحة مخطط Pydantic. يمكنك التنشيط عند إنشاء وكلاء Python AI أو أنظمة مستقلة أو حلقات استدلالية تستخدم الأدوات باستخدام منصة Opper.
