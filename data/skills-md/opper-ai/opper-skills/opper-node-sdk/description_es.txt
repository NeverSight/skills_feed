Utilice el SDK de Opper Node/TypeScript (opperai) para completar tareas de IA, resultados estructurados con JSON Schema o Zod, búsqueda semántica de la base de conocimientos, transmisión de respuestas y seguimiento. Actívelo cuando cree aplicaciones TypeScript o JavaScript que necesiten completar tareas impulsadas por LLM, canalizaciones RAG u orquestación de funciones de IA con la plataforma Opper.
