استخدم Opper Node/TypeScript SDK (opperai) لإكمال مهمة الذكاء الاصطناعي، والإخراج المنظم باستخدام JSON Schema أو Zod، والبحث الدلالي في قاعدة المعرفة، والاستجابات المتدفقة، والتتبع. قم بالتنشيط عند إنشاء تطبيقات TypeScript أو JavaScript التي تحتاج إلى إكمال المهام التي تدعم LLM، أو خطوط أنابيب RAG، أو تنسيق وظائف الذكاء الاصطناعي مع منصة Opper.
