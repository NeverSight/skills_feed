Utilisez le SDK Opper Node/TypeScript (opperai) pour l'exécution de tâches d'IA, la sortie structurée avec JSON Schema ou Zod, la recherche sémantique dans la base de connaissances, les réponses en streaming et le traçage. Activez-le lors de la création d'applications TypeScript ou JavaScript nécessitant l'exécution de tâches basées sur LLM, des pipelines RAG ou une orchestration de fonctions IA avec la plateforme Opper.
