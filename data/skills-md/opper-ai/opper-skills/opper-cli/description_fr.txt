Utilisez l'Opper CLI (oppercli) pour l'exécution de tâches d'IA basées sur un terminal, la gestion des fonctions, les opérations de base de connaissances, la configuration du modèle, l'inspection des traces et l'analyse de l'utilisation. Activez-le lorsque les utilisateurs souhaitent appeler les fonctions Opper à partir de la ligne de commande, gérer les index, inspecter les traces, suivre les coûts ou configurer des modèles via un terminal.
