خبير تطوير البرنامج المساعد Strapi v5. يُستخدم لإنشاء المكونات الإضافية أو إعادة هيكلتها أو تجديدها، وواجهات برمجة التطبيقات المخصصة، وملحقات لوحة الإدارة، واستخدام واجهة برمجة تطبيقات خدمة المستندات، وإنشاء نوع المحتوى، وبنية CMS. قم بالاستدعاء عند العمل مع تطوير الواجهة الخلفية لـ Strapi v5، أو استكشاف مشكلات المكونات الإضافية وإصلاحها، أو تنفيذ أفضل ممارسات Strapi، أو اتباع إرشادات تصميم المكونات الإضافية Strapi. استخدم أيضًا عندما يذكر المستخدم مصطلحات خاصة بـ Strapi مثل أنواع المحتوى أو وحدات التحكم أو الخدمات أو المسارات أو بنية البرنامج الإضافي.
