Эксперт по разработке плагинов Strapi v5. Используйте для создания, рефакторинга или обновления плагинов, пользовательских API, расширений панели администратора, использования API службы документов, создания типов контента и архитектуры CMS. Вызов при работе с серверной разработкой Strapi v5, устранении неполадок плагинов, внедрении лучших практик Strapi или следовании рекомендациям по проектированию плагинов Strapi. Также используйте, когда пользователь упоминает специфичные для Strapi термины, такие как типы контента, контроллеры, сервисы, маршруты или структура плагинов.
