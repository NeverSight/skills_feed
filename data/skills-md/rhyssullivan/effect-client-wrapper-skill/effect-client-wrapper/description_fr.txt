Modèle pour encapsuler les clients SDK tiers (Stripe, Resend, AWS, etc.) avec Effect. À utiliser lors de la création de services Effect qui encapsulent des bibliothèques externes avec des API basées sur Promise. Fournit une gestion des erreurs de type sécurisé, un traçage automatique et une injection de dépendance propre via le modèle « use ».
