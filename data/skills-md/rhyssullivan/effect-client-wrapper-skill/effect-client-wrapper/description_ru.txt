Шаблон для упаковки сторонних клиентов SDK (Stripe, Resend, AWS и т. д.) с помощью Effect. Используйте при создании служб эффектов, которые оборачивают внешние библиотеки API-интерфейсами на основе Promise. Обеспечивает типобезопасную обработку ошибок, автоматическую трассировку и чистое внедрение зависимостей с помощью шаблона «использование».
