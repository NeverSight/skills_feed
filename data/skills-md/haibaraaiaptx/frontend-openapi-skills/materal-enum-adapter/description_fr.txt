Récupère les données d'énumération de Materal Framework à partir de l'API pour la génération d'énumérations TypeScript. À utiliser lorsque : (1) vous travaillez avec les spécifications OpenAPI de Materal Framework, (2) vous avez besoin de valeurs d'énumération réelles de l'API (pas uniquement de schéma), (3) vous générez des énumérations TypeScript avec des traductions du chinois vers l'anglais via l'IA.
