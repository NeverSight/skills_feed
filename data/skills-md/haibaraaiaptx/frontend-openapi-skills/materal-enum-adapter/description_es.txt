Obtiene datos de enumeración de Materal Framework de la API para la generación de enumeraciones de TypeScript. Úselo cuando: (1) trabaje con especificaciones de Materal Framework OpenAPI, (2) necesite valores de enumeración reales de API (no solo de esquema), (3) genere enumeraciones de TypeScript con traducciones del chino al inglés a través de IA.
