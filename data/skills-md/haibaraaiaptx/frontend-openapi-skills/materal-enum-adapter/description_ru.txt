Извлекает данные перечисления Materal Framework из API для генерации перечисления TypeScript. Используйте, когда: (1) работает со спецификациями OpenAPI Materal Framework, (2) нужны реальные значения перечислений из API (а не только схемы), (3) создание перечислений TypeScript с переводом с китайского на английский с помощью AI.
