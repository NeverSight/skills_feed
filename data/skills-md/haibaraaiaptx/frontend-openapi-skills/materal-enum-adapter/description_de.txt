Ruft Materal Framework-Enum-Daten von der API für die TypeScript-Enum-Generierung ab. Wird verwendet, wenn: (1) mit Materal Framework OpenAPI-Spezifikationen gearbeitet wird, (2) echte Enum-Werte von der API benötigt werden (nicht nur Schema), (3) TypeScript-Enums mit Chinesisch-Englisch-Übersetzungen über KI generiert werden.
