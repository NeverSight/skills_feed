Генерирует запросы изображений Seedream 5.0/4.0 (即梦AI) и может вызывать API для создания
изображений и автоматическую загрузку в файл вывода/. Рабочий процесс: описать идею → подсказка о результатах работы агента
для проверки → пользователь подтверждает → агент запускает файлgener.py. Охватывает преобразование текста в изображение, изображение
редактирование, объединение нескольких изображений, согласованность персонажей, карточки знаний, плакаты,
Фоны PPT, электронная коммерция, аватары, группы/раскадровка. Используйте, когда пользователь упоминает
семя, 即梦, цзимен, AI生图, 文生图, 图生图, подсказка семени, 提示词, 一键生图,
карточка знаний, дизайн плаката, 海报, 电商图, 角色一致性 или создание изображений.
