Esta habilidad debe usarse al escribir casos de prueba, corregir errores, analizar código en busca de problemas potenciales o mejorar la cobertura de pruebas para aplicaciones JavaScript/TypeScript. Úselo para pruebas unitarias, pruebas de integración, pruebas de un extremo a otro, depuración de errores de tiempo de ejecución, errores lógicos, problemas de rendimiento, vulnerabilidades de seguridad y análisis de código sistemático.
