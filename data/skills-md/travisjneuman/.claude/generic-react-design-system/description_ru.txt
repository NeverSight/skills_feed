Полный справочник по системе проектирования для приложений React. Охватывает цвета, типографику, интервалы, шаблоны компонентов, эффекты гласморфизма, анимацию с ускорением на графическом процессоре и специальные возможности WCAG AA. Используйте при реализации пользовательского интерфейса, выборе цветов, применении интервалов, создании компонентов или обеспечении единообразия бренда.
