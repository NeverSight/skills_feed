Руководство по разработке функций для приложений React с акцентом на архитектуру. Охватывает шаблоны Zustand/Redux, использование IndexedDB, системы компонентов, стратегии отложенной загрузки и бесшовную интеграцию. Используйте при добавлении новых функций, рефакторинге существующего кода или планировании крупных изменений.
