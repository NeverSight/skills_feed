Шаблоны архитектуры для приложений пользовательского интерфейса Rust, включая шаблоны, специфичные для графического процессора, организацию кода, модульность и масштабируемость. Используйте, когда пользователю требуется руководство по архитектуре приложения, организации кода или масштабированию приложений пользовательского интерфейса.
