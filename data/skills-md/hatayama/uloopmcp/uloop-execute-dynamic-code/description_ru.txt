Динамическое выполнение кода C# в редакторе Unity через интерфейс командной строки uloop. Используйте для автоматизации редактора: (1) связывание префабов/материалов и операции AddComponent, (2) связывание ссылок с SerializedObject, (3) редактирование сцены/иерархии и пакетные операции. НЕ для файлового ввода-вывода или создания сценариев.
