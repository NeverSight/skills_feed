Esegui il codice C# in modo dinamico in Unity Editor tramite la CLI uloop. Utilizzare per l'automazione dell'editor: (1) Cablaggio prefabbricato/materiale e operazioni di aggiunta componenti, (2) Cablaggio di riferimento con SerializedObject, (3) Modifiche di scene/gerarchia e operazioni batch. NON per I/O di file o creazione di script.
