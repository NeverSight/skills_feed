Exécutez le code C# de manière dynamique dans Unity Editor via uloop CLI. Utilisation pour l'automatisation de l'éditeur : (1) Câblage de préfabriqués/matériaux et opérations AddComponent, (2) Câblage de référence avec SerializedObject, (3) Modifications de scène/hiérarchie et opérations par lots. PAS pour les E/S de fichiers ou la création de scripts.
