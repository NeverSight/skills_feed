Координируйте команды агентов ИИ с помощью доски задач Канбан с локальным хранилищем JSON. Обеспечивает рабочие процессы с несколькими агентами, в которых руководитель группы назначает работу, а рабочие агенты выполняют задачи посредством опроса пульса. Идеально подходит для создания командных центров агентов ИИ.
