تطوير لغة Python المتقدم مع خبرة في الهندسة المعمارية وإنفاذ جودة التعليمات البرمجية تلقائيًا. يُستخدم عند تطوير تطبيقات Python أو المكتبات أو البرامج النصية التي تتطلب جودة عالية للتعليمات البرمجية وأمان الكتابة وأفضل الممارسات وأنماط البنية الاحترافية. يتضمن الفحص الآلي باستخدام Ruff، والتحقق من النوع باستخدام MyPy، والفحص الأمني ​​باستخدام Bandit، والاختبار الشامل. تقدم بطلب للحصول على المهام التي تتضمن إنشاء كود Python، أو إعادة البناء، أو فحوصات الجودة، أو تهيئة المشروع، أو تنفيذ أنماط التصميم.
