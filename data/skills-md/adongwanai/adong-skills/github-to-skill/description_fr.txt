Recherchez les référentiels GitHub et générez des compétences Claude à partir d'eux. Utilisez cette compétence lorsque les utilisateurs souhaitent créer une nouvelle compétence basée sur un référentiel GitHub existant, doivent trouver des bibliothèques de haute qualité pour le développement des compétences ou souhaitent transformer un projet GitHub en une fonctionnalité Claude réutilisable. Prend en charge la recherche dans le référentiel, le filtrage de qualité et la génération de modèles de compétences.
