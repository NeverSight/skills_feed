Cerca nei repository GitHub e genera competenze Claude da essi. Utilizza questa competenza quando gli utenti desiderano creare una nuova competenza basata su un repository GitHub esistente, hanno bisogno di trovare librerie di alta qualità per lo sviluppo di competenze o desiderano trasformare un progetto GitHub in una funzionalità Claude riutilizzabile. Supporta la ricerca nel repository, il filtraggio della qualità e la generazione di modelli di competenze.
