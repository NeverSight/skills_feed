ابحث في مستودعات GitHub وقم بتوليد مهارات كلود منها. استخدم هذه المهارة عندما يريد المستخدمون إنشاء مهارة جديدة بناءً على مستودع GitHub موجود، أو عندما يحتاجون إلى العثور على مكتبات عالية الجودة لتنمية المهارات، أو يريدون تحويل مشروع GitHub إلى قدرة Claude قابلة لإعادة الاستخدام. يدعم البحث في المستودعات وتصفية الجودة وإنشاء قوالب المهارات.
