Найдите репозитории GitHub и создайте из них Claude Skills. Используйте этот навык, когда пользователи хотят создать новый навык на основе существующего репозитория GitHub, нуждаются в поиске высококачественных библиотек для развития навыков или хотят преобразовать проект GitHub в возможность многократного использования Claude. Поддерживает поиск по репозиторию, фильтрацию качества и создание шаблонов навыков.
