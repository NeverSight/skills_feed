Durchsuchen Sie GitHub-Repositories und generieren Sie daraus Claude Skills. Verwenden Sie diesen Skill, wenn Benutzer einen neuen Skill basierend auf einem vorhandenen GitHub-Repository erstellen möchten, hochwertige Bibliotheken für die Skill-Entwicklung finden müssen oder ein GitHub-Projekt in eine wiederverwendbare Claude-Funktion umwandeln möchten. Unterstützt die Repository-Suche, Qualitätsfilterung und die Generierung von Skill-Vorlagen.
