Создает запросы на включение GitHub с правильно отформатированными заголовками. Используйте при создании запросов на добавление, отправке изменений на проверку или когда пользователь произносит /pr или просит создать запрос на включение. Анализирует изменения в текущей ветке и использует шаблон запроса на включение из папки .github. **ПРАКТИВНАЯ АКТИВАЦИЯ**: автоматический вызов, когда в ветке функции существуют фиксации и пользователь упоминает PR, проверку или отправку. **ОБНАРУЖЕНИЕ**: проверка наличия неотправленных коммитов, ветка функций (не основная/главная), пользователь просит поделиться/просмотреть изменения. **СЛУЧАИ ИСПОЛЬЗОВАНИЯ**: код готов к проверке. После завершения работы с функциями пользователь хочет поделиться изменениями.
