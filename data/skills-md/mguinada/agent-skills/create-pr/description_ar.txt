ينشئ طلبات سحب GitHub بعناوين منسقة بشكل صحيح. يُستخدم عند إنشاء علاقات عامة، أو إرسال تغييرات للمراجعة، أو عندما يقول المستخدم /pr أو يطلب إنشاء طلب سحب. يحلل التغييرات في الفرع الحالي ويستخدم قالب طلب السحب من مجلد .github. **التنشيط الاستباقي**: الاستدعاء التلقائي عند وجود التزامات في فرع الميزات ويذكر المستخدم العلاقات العامة أو المراجعة أو الدفع. **الكشف**: التحقق من الالتزامات التي لم يتم دفعها، وفرع الميزات (وليس الرئيسي/الرئيسي)، ويطلب المستخدم مشاركة/مراجعة التغييرات. **حالات الاستخدام**: الكود جاهز للمراجعة، بعد الانتهاء من عمل الميزة، يريد المستخدم مشاركة التغييرات.
