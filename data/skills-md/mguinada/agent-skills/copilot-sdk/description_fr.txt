Créez des applications agentiques avec le SDK GitHub Copilot. À utiliser lors de l'intégration d'agents IA dans des applications, de la création d'outils personnalisés, de la mise en œuvre de réponses en streaming, de la gestion de sessions, de la connexion à des serveurs MCP ou de la création d'agents personnalisés. Déclencheurs sur Copilot SDK, GitHub SDK, application agentique, intégration Copilot, agent programmable, serveur MCP, agent personnalisé. **ACTIVATION PROACTIVE** : appel automatique lors de la création d'applications agentiques ou de l'intégration du SDK Copilot. **DÉTECTION** : vérifiez les importations @github/copilot-sdk et les dépendances copilot dans package.json/pyproject.toml/go.mod. **CAS D'UTILISATION** : Intégration d'agents dans des applications, création d'outils personnalisés, mise en œuvre du streaming, gestion des sessions, connexion aux serveurs MCP.
