Создавайте агентные приложения с помощью GitHub Copilot SDK. Используйте при встраивании агентов ИИ в приложения, создании пользовательских инструментов, реализации потоковых ответов, управлении сеансами, подключении к серверам MCP или создании пользовательских агентов. Запускается на Copilot SDK, GitHub SDK, агентском приложении, встроенном Copilot, программируемом агенте, сервере MCP, пользовательском агенте. **ПРАКТИВНАЯ АКТИВАЦИЯ**: автоматический вызов при создании агентских приложений или интеграции Copilot SDK. **ОБНАРУЖЕНИЕ**: проверьте импорт @github/copilot-sdk и зависимости второго пилота в package.json/pyproject.toml/go.mod. **СЛУЧАИ ИСПОЛЬЗОВАНИЯ**: встраивание агентов в приложения, создание пользовательских инструментов, реализация потоковой передачи, управление сеансами, подключение к серверам MCP.
