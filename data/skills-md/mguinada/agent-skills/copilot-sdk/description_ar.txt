أنشئ تطبيقات وكيلة باستخدام GitHub Copilot SDK. يُستخدم عند تضمين وكلاء الذكاء الاصطناعي في التطبيقات، أو إنشاء أدوات مخصصة، أو تنفيذ استجابات البث، أو إدارة الجلسات، أو الاتصال بخوادم MCP، أو إنشاء وكلاء مخصصين. المشغلات على Copilot SDK، وGitHub SDK، والتطبيق الوكيل، وتضمين Copilot، والوكيل القابل للبرمجة، وخادم MCP، والوكيل المخصص. **التنشيط الاستباقي**: الاستدعاء التلقائي عند إنشاء تطبيقات وكيلة أو دمج Copilot SDK. **الكشف**: تحقق من واردات @github/copilot-sdk وتبعيات مساعد الطيار في package.json/pyproject.toml/go.mod. **حالات الاستخدام**: تضمين الوكلاء في التطبيقات، وإنشاء أدوات مخصصة، وتنفيذ البث، وإدارة الجلسات، والاتصال بخوادم MCP.
