Crée des invites système, rédige des descriptions d'outils et structure les instructions d'agent pour les systèmes agents. À utiliser lorsque l'utilisateur demande à créer, générer ou concevoir des invites pour les agents IA, en particulier pour les agents utilisant des outils, les agents de planification ou les systèmes autonomes. **ACTIVATION PROACTIVE** : appel automatique lors de la conception d'invites pour les agents, les outils ou les flux de travail agentiques dans les projets d'IA. **DÉTECTION** : recherchez le code lié à l'agent/à l'outil, les fichiers d'invite ou les mentions utilisateur de "invite", "agent", "LLM". **CAS D'UTILISATION** : Conception d'invites système, de descriptions d'outils, d'instructions pour les agents, d'optimisation des invites, de réduction des hallucinations.
