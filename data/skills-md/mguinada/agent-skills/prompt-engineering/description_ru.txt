Создает системные подсказки, пишет описания инструментов и структурирует инструкции агентов для агентных систем. Используйте, когда пользователь запрашивает создание, генерацию или разработку подсказок для агентов ИИ, особенно для агентов, использующих инструменты, агентов планирования или автономных систем. **ПРАКТИВНАЯ АКТИВАЦИЯ**: автоматический вызов при разработке подсказок для агентов, инструментов или агентских рабочих процессов в проектах ИИ. **ОБНАРУЖЕНИЕ**: проверьте код, связанный с агентом/инструментом, файлы подсказок или упоминания пользователем слов «подсказка», «агент», «LLM». **СЛУЧАИ ИСПОЛЬЗОВАНИЯ**: Разработка системных подсказок, описаний инструментов, инструкций для агентов, оптимизация подсказок, уменьшение галлюцинаций.
