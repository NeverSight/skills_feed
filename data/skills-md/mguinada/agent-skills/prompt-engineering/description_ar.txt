ينشئ مطالبات النظام، ويكتب أوصاف الأدوات، ويبني تعليمات الوكيل للأنظمة الوكيلة. يتم استخدامه عندما يطلب المستخدم إنشاء أو إنشاء أو تصميم مطالبات لوكلاء الذكاء الاصطناعي، خاصة للوكلاء الذين يستخدمون الأدوات أو وكلاء التخطيط أو الأنظمة الذاتية. **التنشيط الاستباقي**: الاستدعاء التلقائي عند تصميم مطالبات للوكلاء أو الأدوات أو سير عمل الوكلاء في مشاريع الذكاء الاصطناعي. **الكشف**: تحقق من وجود تعليمات برمجية متعلقة بالوكيل/الأداة، أو ملفات المطالبة، أو إشارات المستخدم لـ "مطالبة"، و"agent"، و"LLM". **حالات الاستخدام**: تصميم مطالبات النظام، وأوصاف الأدوات، وتعليمات الوكيل، والتحسين الفوري، وتقليل الهلوسة.
