Cette compétence doit être utilisée lorsque l'utilisateur demande de « renforcer les compétences », « ajouter une carte de compétences », « mettre à jour la carte de compétences », « synchroniser les compétences avec CLAUDE.md », « conserver les compétences », « enregistrer les compétences dans le projet », « intégrer les compétences », « les compétences continuent d'être oubliées », « j'oublie toujours les compétences », ou lors de la mise en place d'un nouveau projet où les compétences installées doivent être conservées en tant que contexte. Génère une directive compressée de cartographie des compétences dans CLAUDE.md suivant le modèle de recherche Vercel AGENTS.md.
