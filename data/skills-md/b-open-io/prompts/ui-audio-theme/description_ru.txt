Создавайте связные аудиотемы пользовательского интерфейса с тонкими минимальными звуковыми эффектами для приложений. Этот навык следует использовать, когда пользователи хотят создать набор скоординированных звуков интерфейса для приложений кошелька, информационных панелей или веб-приложений — генерируя звуки, сопоставленные с константами взаимодействия пользовательского интерфейса, такими как нажатия кнопок, уведомления и переходы навигации, с помощью API ElevenLabs.
