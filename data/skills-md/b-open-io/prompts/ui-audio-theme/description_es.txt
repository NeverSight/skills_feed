Genere temas de audio de interfaz de usuario coherentes con efectos de sonido mínimos y sutiles para aplicaciones. Esta habilidad debe usarse cuando los usuarios desean crear un conjunto de sonidos de interfaz coordinados para aplicaciones de billetera, paneles o aplicaciones web, generando sonidos asignados a constantes de interacción de la interfaz de usuario, como clics en botones, notificaciones y transiciones de navegación utilizando la API de ElevenLabs.
