Этот навык следует использовать, когда пользователь хочет опубликовать пакет в npm, обновить версию, выпустить новую версию или упоминает «публикацию npm», «публикацию булочки», «выпуск версии» или «выпуск». Обрабатывает обновления журнала изменений, git push и публикацию npm с аутентификацией OTP.
