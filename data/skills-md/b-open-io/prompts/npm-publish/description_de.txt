Diese Fähigkeit sollte verwendet werden, wenn der Benutzer ein Paket auf npm veröffentlichen, eine Version erweitern, eine neue Version veröffentlichen möchte oder „npm Publish“, „Bun Publish“, „Version Bump“ oder „Release“ erwähnt. Verarbeitet Changelog-Updates, Git-Push und NPM-Veröffentlichung mit OTP-Authentifizierung.
