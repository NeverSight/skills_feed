Этот навык следует использовать, когда пользователь просит «создать новый проект», «подготовить приложение Next.js», «инициализировать новое приложение», «начать новый проект», «настроить новый проект Next.js» или упоминает «создать-следующий-проект». Обеспечивает управляемую и продуманную инициализацию полнофункционального проекта Next.js с развертыванием Biome, Tailwind v4, shadcn/ui, Better-auth и Vercel. Использует команды агентов для параллельного выполнения.
