Cette compétence doit être utilisée lorsque l'utilisateur demande de « créer un nouveau projet », « d'échafauder une application Next.js », « d'initialiser une nouvelle application », de « démarrer un nouveau projet », de « configurer un nouveau projet Next.js » ou mentionne « créer un projet suivant ». Fournit une initialisation guidée et avisée du projet Next.js full-stack avec le déploiement de Biome, Tailwind v4, shadcn/ui, better-auth et Vercel. Utilise des équipes d'agents pour une exécution parallèle.
