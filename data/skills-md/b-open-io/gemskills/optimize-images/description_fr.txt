Cette compétence doit être utilisée lorsque l'utilisateur demande « d'optimiser les images », « compresser les images », « réduire la taille du fichier image », « rendre les images plus petites », « optimiser les fichiers PNG », « optimiser les fichiers JPEG », « accélérer les images de sites Web », « réduire la taille des images du paquet » ou a besoin d'aide pour la compression d'images pour des projets Web. Fournit des flux de travail et des scripts pour l’optimisation des images par lots à l’aide de Sharp.
