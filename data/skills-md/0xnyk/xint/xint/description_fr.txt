X Intelligence CLI — recherchez, analysez et engagez-vous sur X/Twitter depuis le terminal. À utiliser lorsque : (1) l'utilisateur dit "x recherche", "recherche x pour", "recherche sur Twitter", "de quoi disent les gens", "que dit Twitter", "vérifie x pour", "x recherche", "recherche x", "trouver des tweets sur", "surveiller x pour", "suivre les abonnés", (2) l'utilisateur travaille sur quelque chose où le discours récent de X fournirait un contexte utile (nouvelles versions de bibliothèque, modifications d'API, lancements de produits, événements culturels, drame de l'industrie), (3) l'utilisateur veut trouver quoi les développeurs/experts/communauté réfléchissent à un sujet, (4) l'utilisateur a besoin d'une surveillance en temps réel (« regarder »), (5) l'utilisateur souhaite une analyse basée sur l'IA (« analyser », « sentiment », « rapport »). Prend également en charge : les signets, les likes, le suivi (lecture/écriture), les sujets tendances, l'analyse Grok AI et le suivi des coûts. Exportez au format JSON, JSONL (pipeable), CSV ou Markdown. Non-objectifs : pas pour publier des tweets, ni pour les DM, ni pour les fonctionnalités d'entreprise. Nécessite OAuth pour les opérations de contexte utilisateur (signets, likes, suivi, différence).
