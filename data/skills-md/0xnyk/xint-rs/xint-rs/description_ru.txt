Fast X Intelligence CLI (Rust) — поиск, анализ и взаимодействие с X/Twitter с терминала. Используйте, когда: (1) пользователь говорит «исследование x», «искать x», «искать в Твиттере», «о чем говорят люди», «о чем говорят в Твиттере», «проверить x», «поиск x», «найти x», (2) пользователь хочет осуществлять мониторинг в реальном времени с помощью «смотреть», (3) пользователю нужен анализ с помощью искусственного интеллекта с помощью Grok («анализировать», «настроения»), (4) пользователю нужны разведывательные отчеты («отчет»), (5) пользователь хочет отслеживать подписчиков («diff»), (6) пользователю нужны актуальные темы («тренды»). Также поддерживает: закладки, лайки, подписку (OAuth), x-поиск, коллекции, экспорт CSV/JSON/JSONL. Нецели: не для публикации твитов, не для личных сообщений, не для корпоративных функций.
