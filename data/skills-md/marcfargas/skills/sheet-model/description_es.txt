Motor de hoja de cálculo sin cabeza para modelado financiero, análisis de datos y comparación de escenarios. Úselo cuando: cree modelos financieros con proporciones y escenarios hipotéticos, calcule valores derivados de datos tabulares con fórmulas, produzca archivos .xlsx con fórmulas en vivo (no valores estáticos) para revisión humana, cualquier tarea en la que el agente escribiría código imperativo para manipular números que una hoja de cálculo hace de forma natural. Desencadenantes: modelo financiero, análisis de escenarios, cálculo de ratios, balance general, pérdidas y ganancias, análisis de sensibilidad, ratios bancarios, modelo de hoja de cálculo, construcción de un modelo, proyección, pronóstico. NO lo use para: lectura/escritura simple de CSV/Excel (use la habilidad xlsx), tareas de solo gráficos o volúmenes de datos que excedan ~5000 filas.
