Безголовый механизм электронных таблиц для финансового моделирования, анализа данных и сравнения сценариев. Используйте при: построении финансовых моделей с коэффициентами и сценариями «что, если», вычислении производных значений из табличных данных с помощью формул, создании файлов .xlsx с живыми формулами (не статическими значениями) для проверки человеком, любой задаче, где в противном случае агенту пришлось бы писать императивный код для манипулирования числами, которые электронные таблицы выполняют естественным образом. Триггеры: финансовая модель, анализ сценариев, расчет коэффициентов, баланс, прибыли и убытки, «что, если», анализ чувствительности, банковские коэффициенты, модель электронных таблиц, построение модели, прогноз, прогноз. НЕ используйте для: простого чтения/записи CSV/Excel (используйте навыки xlsx), задач только для диаграмм или объемов данных, превышающих ~ 5000 строк.
