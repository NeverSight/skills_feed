Azure CLI (Азбука). Используйте при управлении ресурсами Azure, развертывании в службе приложений, функциях, приложениях-контейнерах или AKS, работе с хранилищем, базой данных SQL, Cosmos DB, виртуальными машинами, виртуальными сетями, группами безопасности сети, Key Vault, Entra ID (Azure AD), RBAC, служебной шиной, концентраторами событий, реестром контейнеров, Azure Monitor, DNS или любой службой Azure. Также рассматриваются: аутентификация, управление подписками, интеграция CI/CD (GitHub Actions/Azure DevOps), шаблоны Bicep/ARM, управляемые удостоверения и автоматизация инфраструктуры.
