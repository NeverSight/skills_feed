Azure CLI (az). À utiliser dans les cas suivants : gestion des ressources Azure, déploiement sur App Service/Functions/Container Apps/AKS, utilisation du stockage, base de données SQL, Cosmos DB, machines virtuelles, réseaux virtuels, NSG, Key Vault, Entra ID (Azure AD), RBAC, Service Bus, Event Hubs, Container Registry, Azure Monitor, DNS ou tout autre service Azure. Couvre également : l'authentification, la gestion des abonnements, l'intégration CI/CD (GitHub Actions/Azure DevOps), les modèles Bicep/ARM, les identités managées et l'automatisation de l'infrastructure.
