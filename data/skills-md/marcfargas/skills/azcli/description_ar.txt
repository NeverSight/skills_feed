أزور سطر الأوامر (az). يُستخدم عندما: إدارة موارد Azure، أو النشر إلى خدمة التطبيقات/الوظائف/Container Apps/AKS، أو العمل مع التخزين، أو قاعدة بيانات SQL، أو Cosmos DB، أو VMs، أو VNets، أو NSGs، أو Key Vault، أو Entra ID (Azure AD)، أو RBAC، أو ناقل الخدمة، أو مراكز الأحداث، أو سجل الحاويات، أو Azure Monitor، أو DNS، أو أي خدمة Azure. يغطي أيضًا: المصادقة، وإدارة الاشتراك، وتكامل CI/CD (GitHub Actions/Azure DevOps)، وقوالب Bicep/ARM، والهويات المُدارة، وأتمتة البنية التحتية.
