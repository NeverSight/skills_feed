Azure CLI (az). Verwendung bei: Verwaltung von Azure-Ressourcen, Bereitstellung in App Service/Funktionen/Container Apps/AKS, Arbeit mit Speicher, SQL-Datenbank, Cosmos DB, VMs, VNets, NSGs, Key Vault, Entra ID (Azure AD), RBAC, Service Bus, Event Hubs, Container Registry, Azure Monitor, DNS oder einem beliebigen Azure-Dienst. Deckt außerdem ab: Authentifizierung, Abonnementverwaltung, CI/CD-Integration (GitHub Actions/Azure DevOps), Bicep/ARM-Vorlagen, verwaltete Identitäten und Infrastrukturautomatisierung.
