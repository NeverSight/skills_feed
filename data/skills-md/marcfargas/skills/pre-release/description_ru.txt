Предварительная проверка и создание набора изменений для пакетов npm с помощью @changesets/cli. Используйте при подготовке выпуска, создании журналов изменений, контрольном списке перед выпуском, обновлении версии, написании примечаний к выпуску, просмотре README перед публикацией. Триггеры: «подготовка выпуска», «предварительная версия», «создание журнала изменений», «запись наборов изменений», «контрольный список выпуска».
