Examen préliminaire et génération d'ensembles de modifications pour les packages npm à l'aide de @changesets/cli. À utiliser lors de : la préparation d'une version, la génération de journaux de modifications, la liste de contrôle de pré-version, le changement de version, la rédaction de notes de version, la révision du fichier README avant la publication. Déclencheurs : "préparer la version", "pré-version", "générer le journal des modifications", "écrire les ensembles de modifications", "liste de contrôle de publication".
