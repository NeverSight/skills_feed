Интерфейс командной строки Google Cloud Platform (gcloud, хранилище gcloud, bq). Используйте при: управлении ресурсами GCP, развертывании в Cloud Run/Cloud Functions/GKE/App Engine, работе с Cloud Storage, BigQuery, IAM, Compute Engine, Cloud SQL, Pub/Sub, Secret Manager, Registry Artifact, Cloud Build, Cloud Scheduler, Cloud Tasks, Vertex AI, VPC/сеть, DNS, ведение журналов/мониторинг или любая служба GCP. Также охватывает: аутентификацию, управление проектами/конфигурациями, интеграцию CI/CD, бессерверное развертывание, реестр контейнеров, отправку докера в GCP, управление секретами, федерацию идентификации рабочей нагрузки и автоматизацию инфраструктуры.
