Google Cloud Platform CLI (gcloud, stockage gcloud, bq). À utiliser dans les cas suivants : gestion des ressources GCP, déploiement sur Cloud Run/Cloud Functions/GKE/App Engine, utilisation avec Cloud Storage, BigQuery, IAM, Compute Engine, Cloud SQL, Pub/Sub, Secret Manager, Artifact Registry, Cloud Build, Cloud Scheduler, Cloud Tasks, Vertex AI, VPC/réseau, DNS, journalisation/surveillance ou tout service GCP. Couvre également : l'authentification, la gestion de projet/configuration, l'intégration CI/CD, les déploiements sans serveur, le registre de conteneurs, le push Docker vers GCP, la gestion des secrets, la fédération d'identité de charge de travail et l'automatisation de l'infrastructure.
