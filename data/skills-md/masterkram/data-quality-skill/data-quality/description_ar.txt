تشخيص وإصلاح مشاكل جودة البيانات في مجموعات البيانات. يُستخدم عند العمل مع البيانات القذرة، أو البحث عن التكرارات، أو التعامل مع القيم المفقودة، أو اكتشاف القيم المتطرفة/الحالات الشاذة، أو التحقق من صحة القيود (التبعيات الوظيفية، أو التكامل المرجعي)، أو تحديد ملفات تعريف مجموعات البيانات، أو تنظيف البيانات للتحليل أو التعلم الآلي. يغطي دورة حياة جودة البيانات الكاملة - التحديد والكشف والتنظيف والقياس.
