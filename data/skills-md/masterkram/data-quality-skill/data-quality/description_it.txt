Diagnosticare e risolvere i problemi di qualità dei dati nei set di dati. Da utilizzare quando si lavora con dati sporchi, si trovano duplicati, si gestiscono valori mancanti, si rilevano valori anomali/anomalie, si convalidano vincoli (dipendenze funzionali, integrità referenziale), si profilano set di dati o si puliscono dati per analisi o machine learning. Copre l'intero ciclo di vita della qualità dei dati: definizione, rilevamento, pulizia, misurazione.
