Project planning output tool. Through requirement clarification, design discussion, and plan writing, it produces an executable development plan.

**Usage scenarios:**
- Requirements are unclear and need multi-round discussion (Brainstorming mode)
- Requirements are clear, write detailed implementation plan directly (Writing Plans mode)
- Prepare executable plan documents for project-workflow

**Output:**
- Plan document: `docs/plans/001-feature-name.md` (includes design and implementation, detail level based on complexity)

**Trigger phrases:**
- "Help me plan XXX feature" / "Design XXX module"
- "I want to implement XXX but not sure how"
- "Write implementation plan for XXX feature"
