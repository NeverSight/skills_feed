高度な製品処理として、多くの深いリサーチを通じてニーズを把握し、妥当性を分析し、最適な実践の提案を提供し、完全な技術文書システムを最終的に生成します。

**使用タイムマシン：**
- 新项目启動、要求建立文档系（推荐第一步）
- 现有项目缺少规范文档，需要补充完善
- 已有码但缺少文档，要补代換文档
- 製品およびフレーム構造の分析が必要

**2 つの操作モードのサポート：**
- モード A：新项目文档创建（从必要要求到文档）
- モードB：代码回补文档（从代到文档）

** プロジェクト ワークフローとの関係：**
- 本スキル负责创建完整文档（PRD、SAD、开公開指南等）
- プロジェクトワークフロー负责に基づく文档行开発行任务
- 建议流程：project-docs-setup → project-workflow

**触発方法：**
- "创建项目文档" / "初化化项目文档" / "设置项目文档"
- "根元代码生成文档" / "回补项目文档" / "补充项目文档"
- 直接説明项目需要，例:「帮我创建一在在線上教育平台的文档」
