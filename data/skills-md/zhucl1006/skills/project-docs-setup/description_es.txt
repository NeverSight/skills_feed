Asistente para crear la estructura de documentación del proyecto. Como PM senior, profundiza en los requisitos por diálogo, analiza coherencia y sugiere buenas prácticas; genera el sistema de documentación técnica. Para arranque de proyecto, completar documentación o generar documentación a partir de código. Modos: nuevo proyecto (requisitos→documentos) o rellenar desde código. Activar con «crear documentación del proyecto», «inicializar documentación», «generar documentación a partir del código».
