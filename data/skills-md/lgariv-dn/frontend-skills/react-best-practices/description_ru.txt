Рекомендации по оптимизации производительности React от Vercel Engineering. Этот навык следует активно использовать при написании, проверке или рефакторинге кода React, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, связанных с компонентами React, оптимизацией пакета или улучшением производительности.
