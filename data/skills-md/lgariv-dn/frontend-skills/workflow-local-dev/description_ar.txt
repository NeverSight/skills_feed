دعم تطوير النظام الأساسي لسير العمل المحلي في مساحة عمل DAP عبر فرق الواجهة الأمامية والخلفية والبنية التحتية. يوفر الوصول إلى Kubernetes (Kind)، وإدارة خدمة Tilt، واستعلامات قاعدة البيانات، واستكشاف الأخطاء وإصلاحها. يُستخدم عند إنشاء ميزات الواجهة الخلفية/واجهة برمجة التطبيقات (API)، أو ضبط تكوينات الأشعة تحت الحمراء، أو فحص السجلات، أو تشغيل الاختبارات، أو تصحيح الأخطاء مقابل مكونات محرك سير العمل المنشورة محليًا.
