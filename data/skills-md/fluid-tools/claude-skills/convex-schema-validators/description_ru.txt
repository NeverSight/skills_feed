Руководство по проектированию выпуклой схемы, валидаторам и типам TypeScript. Используйте при определении схем базы данных, создании валидаторов для аргументов/возвратов функций, работе с типами документов или обеспечении безопасности типов. Активируется для создания файла Schema.ts, использования валидатора, обработки типов идентификаторов/документов или задач интеграции TypeScript.
