Guía para el diseño de esquemas convexos, validadores y tipos de TypeScript. Úselo al definir esquemas de bases de datos, crear validadores para argumentos/devoluciones de funciones, trabajar con tipos de documentos o garantizar la seguridad de tipos. Se activa para la creación de esquemas.ts, el uso del validador, el manejo de tipos de Id/Doc o tareas de integración de TypeScript.
