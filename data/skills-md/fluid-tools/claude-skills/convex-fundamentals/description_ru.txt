Руководство по основам разработки серверной части Convex, включая типы функций (запросы, мутации, действия), многоуровневую архитектуру, действия HTTP и базовую ментальную модель. Используйте при создании бэкэндов Convex, создании запросов/мутаций/действий, реализации веб-перехватчиков HTTP или понимании модели реактивных данных Convex. Активируется для настройки проекта Convex, определения функций, проектирования API или задач внутренней архитектуры.
