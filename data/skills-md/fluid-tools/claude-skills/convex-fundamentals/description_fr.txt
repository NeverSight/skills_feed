Guide sur les principes fondamentaux du développement backend Convex, y compris les types de fonctions (requêtes, mutations, actions), l'architecture en couches, les actions HTTP et le modèle mental de base. À utiliser lors de la création de backends Convex, de la création de requêtes/mutations/actions, de la mise en œuvre de webhooks HTTP ou de la compréhension du modèle de données réactif de Convex. S'active pour les tâches de configuration de projet Convex, de définition de fonctions, de conception d'API ou d'architecture backend.
