Guía para los fundamentos del desarrollo backend de Convex, incluidos los tipos de funciones (consultas, mutaciones, acciones), arquitectura en capas, acciones HTTP y el modelo mental central. Úselo al crear backends de Convex, crear consultas/mutaciones/acciones, implementar webhooks HTTP o comprender el modelo de datos reactivo de Convex. Se activa para tareas de configuración de proyectos convexos, definición de funciones, diseño de API o arquitectura backend.
