Guía para la optimización del rendimiento de Convex que incluye desnormalización, diseño de índices, evitar consultas N+1, OCC (control de concurrencia optimista) y manejo de puntos calientes. Úselo para optimizar el rendimiento de consultas, diseñar modelos de datos, manejar escrituras de alta contención o solucionar problemas de errores de OCC. Se activa para problemas de rendimiento, optimización de índices, patrones de desnormalización o tareas de control de concurrencia.
