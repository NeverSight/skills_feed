Guía para la creación de componentes Convex: módulos backend aislados y reutilizables con su propio esquema y funciones. Úselo al crear bibliotecas reutilizables, empaquetar la funcionalidad Convex para NPM, crear subsistemas aislados o integrar componentes de terceros. Se activa para tareas de creación de componentes, configuración de convex.config.ts, pruebas de componentes o publicación de NPM.
