Guía para los patrones de implementación de Vercel AI SDK v6, que incluyen generateText, streamText, ToolLoopAgent, salida estructurada con ayudas de salida, useChat Hook, llamadas de herramientas, incrustaciones, middleware e integración de MCP. Úselo al implementar interfaces de chat de IA, transmisión de respuestas, aplicaciones de agente, llamadas de herramientas/funciones, incrustaciones de texto, patrones de flujo de trabajo o al trabajar con convertToModelMessages y toUIMessageStreamResponse. Se activa para la integración del SDK de IA, el uso del enlace de chat, la transmisión de mensajes, el desarrollo de agentes o las tareas de llamada de herramientas.
