دليل لممارسات TypeScript الصارمة بما في ذلك تجنب أي منها، واستخدام التعليقات التوضيحية المناسبة للنوع، والاستفادة من نظام الكتابة في TypeScript بشكل فعال. يُستخدم عند العمل مع قواعد تعليمات TypeScript البرمجية التي تفرض فحصًا صارمًا للنوع، أو عندما تحتاج إلى إرشادات حول أنماط أمان الكتابة، أو عند مواجهة أخطاء في الكتابة. يتم التنشيط بسبب أخطاء نوع TypeScript، أو انتهاكات الوضع الصارمة، أو أفضل ممارسات TypeScript العامة.
