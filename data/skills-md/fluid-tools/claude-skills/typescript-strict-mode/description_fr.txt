Guide sur les pratiques strictes de TypeScript, notamment en évitant celles-ci, en utilisant des annotations de type appropriées et en exploitant efficacement le système de types de TypeScript. À utiliser lorsque vous travaillez avec des bases de code TypeScript qui appliquent une vérification de type stricte, lorsque vous avez besoin de conseils sur les modèles de sécurité de type ou lorsque vous rencontrez des erreurs de type. S'active pour les erreurs de type TypeScript, les violations du mode strict ou les meilleures pratiques générales de TypeScript.
