Guía para prácticas estrictas de TypeScript que incluyen evitarlas, usar anotaciones de tipo adecuadas y aprovechar el sistema de tipos de TypeScript de manera efectiva. Úselo cuando trabaje con bases de código TypeScript que imponen una verificación de tipos estricta, cuando necesite orientación sobre patrones de seguridad de tipos o cuando encuentre errores de tipo. Se activa para errores de tipo TypeScript, violaciones de modo estricto o mejores prácticas generales de TypeScript.
