Guía para patrones de biblioteca de ayudas convexas que incluyen activadores, seguridad a nivel de fila (RLS), ayudas de relaciones, funciones personalizadas, limitación de velocidad y grupo de trabajo. Úselo al implementar efectos secundarios automáticos, control de acceso, cruce de relaciones, contenedores de autenticación o administración de concurrencia. Se activa para la configuración de activadores, la implementación de RLS, envoltorios de funciones personalizadas o tareas de integración de ayudas convexas.
