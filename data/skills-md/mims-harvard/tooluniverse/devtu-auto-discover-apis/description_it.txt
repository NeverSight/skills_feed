Scopri automaticamente le API delle scienze della vita online, crea strumenti ToolUniverse, convalidali e prepara PR di integrazione. Esegue l'analisi delle lacune per identificare le categorie di strumenti mancanti, ricerche web per API, creazione automatizzata di strumenti utilizzando modelli devtu-create-tool, convalida con devtu-fix-tool e gestione del flusso di lavoro git. Da utilizzare quando si espande la copertura di ToolUniverse, si aggiungono nuove integrazioni API o si scoprono sistematicamente risorse scientifiche.
