Оптимизируйте описания инструментов в конфигурациях ToolUniverse JSON для ясности и удобства использования. Просматривает описания на предмет отсутствующих предварительных условий, нераскрытых сокращений, неясных параметров и отсутствующих указаний по использованию. Используйте при просмотре описаний инструментов, улучшении документации API или когда пользователь просит проверить, легко ли понять инструменты.
