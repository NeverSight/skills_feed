Erstellen Sie neue wissenschaftliche Tools für das ToolUniverse-Framework mit der richtigen Struktur, Validierung und Prüfung. Unterstützt das Erstellen von Toolklassen, JSON-Konfigurationen, Validierung, Fehlerbehandlung und Testbeispielen.
