Создайте новые научные инструменты для платформы ToolUniverse с правильной структурой, проверкой и тестированием. Используйте, когда пользователям необходимо добавить инструменты в ToolUniverse, реализовать новые интеграции API, создать оболочки инструментов для научных баз данных/сервисов, расширить возможности ToolUniverse или следовать рекомендациям по вкладу ToolUniverse. Поддерживает создание классов инструментов, конфигураций JSON, проверку, обработку ошибок и примеры тестирования.
