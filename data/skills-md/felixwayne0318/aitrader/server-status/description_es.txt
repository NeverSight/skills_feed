Verifique el estado del servidor y el estado del robot comercial. 检查服务器状态和交易机器人运行情况.

Utilice esta habilidad cuando:
- Comprobar si el robot comercial se está ejecutando (检查机器人是否运行)
- Ver registros de servicio (查看服务日志)
- Monitoreo del estado del sistema (监控系统健康)
- Comprobación del estado del proceso (检查进程状态)
- Solución de problemas de servicio (排查服务问题)

Palabras clave: estado, registros, salud, monitor, proceso, ejecución, servicio, 状态, 日志, 监控
