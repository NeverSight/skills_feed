Vincular paquetes de espacio de trabajo en monorepos (npm, hilo, pnpm, bun). USE CUANDO: (1) acaba de crear o generar nuevos paquetes y necesita conectar sus dependencias, (2) el usuario importa desde un paquete hermano y necesita agregarlo como una dependencia, (3) obtiene errores de resolución para los paquetes del espacio de trabajo (@org/*) como "no se puede encontrar el módulo", "no se pudo resolver la importación", "TS2307" o "no se puede resolver". NO parchee con rutas tsconfig o ediciones manuales de package.json; use los comandos del espacio de trabajo del administrador de paquetes para corregir los enlaces reales.
