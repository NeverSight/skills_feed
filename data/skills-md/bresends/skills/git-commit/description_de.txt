Erstellen Sie Git-Commits mit herkömmlichen Commits mit Bereichen. Wird verwendet, wenn der Benutzer einen Commit, ein Commit, das Speichern von Änderungen oder einen beliebigen Git-Commit-Vorgang durchführen möchte. Fügen Sie niemals „Co-Authored-By“-Zeilen, Erwähnungen von KI-Agenten oder Verweise auf Claude, KI oder automatisierte Tools in Commit-Nachrichten ein.
