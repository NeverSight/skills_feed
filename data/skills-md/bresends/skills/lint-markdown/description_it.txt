Lint e correggi i file Markdown utilizzando markdownlint-cli2. Utilizzare quando viene richiesto di: sfilacciare, correggere, formattare o pulire i file Markdown (.md); applicare le migliori pratiche o standard di stile Markdown; risolvere errori o avvisi di markdownlint. Si attiva su richieste che menzionano markdownlint, markdown linting o formattazione Markdown secondo i migliori standard.
