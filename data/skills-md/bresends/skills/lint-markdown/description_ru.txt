Проверьте и исправьте файлы Markdown с помощью markdownlint-cli2. Используйте, когда вас попросят: проверить, исправить, отформатировать или очистить файлы Markdown (.md); применять лучшие практики Markdown или стандарты стиля; устранить ошибки или предупреждения markdownlint. Запускается по запросам, в которых упоминается markdownlint, анализ уценки или форматирование Markdown в соответствии с лучшими стандартами.
