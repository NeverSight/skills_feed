Lint et corrigez les fichiers Markdown à l'aide de markdownlint-cli2. À utiliser lorsqu'on vous le demande pour : pelucher, réparer, formater ou nettoyer les fichiers Markdown (.md) ; appliquer les meilleures pratiques ou normes de style Markdown ; résoudre les erreurs ou les avertissements de markdownlint. Se déclenche sur les demandes mentionnant markdownlint, markdown lint ou formatage Markdown selon les meilleures normes.
