Отладка ошибок теста Playwright путем анализа файлов трассировки с помощью pwtrace CLI. Используйте этот навык после того, как тесты драматурга провалились и у вас есть файл трассировки.zip. Обучает систематическому рабочему процессу отладки от обзора до детальной проверки (показать → шаг → dom → консоль → снимок экрана → сеть).
