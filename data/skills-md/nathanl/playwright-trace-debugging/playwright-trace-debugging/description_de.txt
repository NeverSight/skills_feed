Debuggen Sie Playwright-Testfehler, indem Sie Trace-Dateien mithilfe der pwtrace-CLI analysieren. Verwenden Sie diese Fähigkeit, nachdem die Playwright-Tests fehlgeschlagen sind und Sie über eine Datei „trace.zip“ verfügen. Vermittelt den systematischen Debugging-Workflow von der Übersicht bis zur detaillierten Inspektion (Anzeigen → Schritt → Dom → Konsole → Screenshot → Netzwerk).
