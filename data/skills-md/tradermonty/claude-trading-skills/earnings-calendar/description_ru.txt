Этот навык позволяет получать объявления о предстоящих доходах по акциям США с помощью API подготовки к финансовому моделированию (FMP). Используйте это, когда пользователь запрашивает данные календаря доходов, хочет знать, какие компании сообщают о доходах на предстоящей неделе, или нуждается в еженедельном обзоре доходов. Навык ориентирован на компании со средней капитализацией и выше (рыночная капитализация более 2 миллиардов долларов), которые оказывают значительное влияние на рынок, упорядочивая данные по дате и времени в чистом формате таблицы уценок. Поддерживает несколько сред (CLI, рабочий стол, Интернет) с гибким управлением ключами API.
