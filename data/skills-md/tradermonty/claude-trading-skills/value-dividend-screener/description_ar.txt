قم بفحص الأسهم الأمريكية بحثًا عن فرص توزيعات أرباح عالية الجودة تجمع بين خصائص القيمة (نسبة السعر إلى الربحية أقل من 20، ونسبة السعر إلى القيمة الدفترية أقل من 2)، والعائدات الجذابة (3٪ أو أعلى)، والنمو المستمر (تتجه الأرباح / الإيرادات / ربحية السهم إلى الارتفاع على مدى 3 سنوات). يدعم الفحص على مرحلتين باستخدام FINVIZ Elite API للتصفية المسبقة الفعالة متبوعة بـ FMP API للتحليل التفصيلي. يُستخدم عندما يطلب المستخدم فحص أرباح الأسهم، أو أفكار محفظة الدخل، أو الأسهم ذات القيمة عالية الجودة ذات الأساسيات القوية.
