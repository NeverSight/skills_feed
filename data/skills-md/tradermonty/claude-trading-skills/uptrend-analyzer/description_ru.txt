Анализирует широту рынка с использованием данных панели мониторинга Monty's Uptrend Ratio для диагностики текущей рыночной среды. Формирует сводную оценку от 0 до 100 на основе 5 компонентов (ширина, участие сектора, ротация, динамика, исторический контекст). Используйте, когда спрашиваете об широте рынка, коэффициентах восходящего тренда или о том, поддерживает ли рыночная среда риск инвестирования в акции. Ключ API не требуется.
