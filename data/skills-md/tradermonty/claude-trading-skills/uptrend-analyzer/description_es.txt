Analiza la amplitud del mercado utilizando los datos del Uptrend Ratio Dashboard de Monty para diagnosticar el entorno actual del mercado. Genera una puntuación compuesta de 0 a 100 a partir de 5 componentes (amplitud, participación del sector, rotación, impulso, contexto histórico). Úselo cuando pregunte sobre la amplitud del mercado, los índices de tendencia alcista o si el entorno del mercado respalda la exposición a la renta variable. No se requiere clave API.
