تحليل شامل للمحفظة باستخدام تكامل Alpaca MCP Server لجلب المقتنيات والمواقف، ثم تحليل تخصيص الأصول ومقاييس المخاطر ومواقف الأسهم الفردية والتنويع وإنشاء توصيات إعادة التوازن. يُستخدم عندما يطلب المستخدم مراجعة المحفظة أو تحليل الموقف أو تقييم المخاطر أو تقييم الأداء أو اقتراحات إعادة التوازن لحساب الوساطة الخاص به.
