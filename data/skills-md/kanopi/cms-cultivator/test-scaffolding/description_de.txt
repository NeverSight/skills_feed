Generieren Sie automatisch ein Testgerüst, wenn der Benutzer neuen Code ohne Tests schreibt oder erwähnt, dass Tests erforderlich sind. Unterstützt Unit-, Integrations-, E2E- und Datentests für PHP und JavaScript. Wird aufgerufen, wenn der Benutzer „Tests“, „Testing“, „Coverage“, „Tests schreiben“ erwähnt oder neuen ungetesteten Code anzeigt.
