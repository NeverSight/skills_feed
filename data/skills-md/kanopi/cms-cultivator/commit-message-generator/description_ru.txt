Автоматически генерировать обычные сообщения о фиксации, когда пользователь вносит изменения и упоминает о фиксации. Анализирует различия и статус git для создания правильно отформатированных сообщений о коммитах в соответствии с обычной спецификацией коммитов. Вызывается, когда пользователь упоминает «фиксацию», «постановку», «фиксацию» или просит помощи с сообщениями о фиксации.
