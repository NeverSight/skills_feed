Outil complet de conception de bases de données créant des schémas complets avec des tables, des champs, des index et des diagrammes ER. À utiliser lorsque les utilisateurs demandent la conception d'une base de données pour n'importe quel système. Génère des documents Markdown, des scripts SQL et des fichiers JSON/DBML compatibles DrawDB. Prend en charge MySQL, PostgreSQL, SQL Server. Déclencheurs : conception de base de données, schéma, diagramme ER, SQL, modèle de données. | 全面的数据库设计工具，创建完整的数据库架构。触发词：数据库设计、数据库架构、ER图、SQL、数据模型、表设计。
