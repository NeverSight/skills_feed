Strumento completo di progettazione di database che crea schemi completi con tabelle, campi, indici e diagrammi ER. Da utilizzare quando gli utenti richiedono la progettazione del database per qualsiasi sistema. Genera documenti Markdown, script SQL e file JSON/DBML compatibili con DrawDB. Supporta MySQL, PostgreSQL, SQL Server. Trigger: progettazione del database, schema, diagramma ER, SQL, modello dati. | Vedi l'originale据库设计、数据库架构、ER图、SQL、数据模型、表设计.
