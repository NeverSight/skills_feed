Систематически устраняйте проблемы с линтингом в файлах уценки с помощью markdownlint-cli2. Этот навык следует использовать при необходимости сканирования, диагностики и устранения проблем форматирования уценки в проектах с одним или несколькими файлами .md, уделяя особое внимание ошибкам нумерации упорядоченного списка (MD029).
