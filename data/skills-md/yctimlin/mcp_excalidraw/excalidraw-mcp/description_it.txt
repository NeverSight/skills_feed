Crea, modifica ed esporta diagrammi Excalidraw in tempo reale utilizzando mcp-excalidraw-server (strumenti MCP + API REST canvas). Da utilizzare quando un agente deve disegnare/disporre diagrammi, convertire Sirena in Excalidraw, eseguire query/aggiornare/eliminare elementi o esportare/importare elementi da un server Canvas in esecuzione (EXPRESS_SERVER_URL, http://localhost:3000 predefinito).
