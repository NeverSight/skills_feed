Cree, edite y exporte diagramas de Excalidraw en vivo utilizando mcp-excalidraw-server (herramientas MCP + API REST de lienzo). Úselo cuando un agente necesita dibujar/diseñar diagramas, convertir Mermaid a Excalidraw, consultar/actualizar/eliminar elementos o exportar/importar elementos desde un servidor Canvas en ejecución (EXPRESS_SERVER_URL, http://localhost:3000 predeterminado).
