Это руководство основано на успешной реализации Hyperliquid и представляет собой руководство по интеграции платформы Yuan для новых поставщиков бирж. Используйте этот навык, когда вам нужно создать реализацию поставщика для новой биржи, включая проектирование структуры проекта, интеграцию API, реализацию основных сервисов и лучшие практики. Применимо для интеграции API обмена, разработки финансовых систем и проектирования архитектуры микросервисов.
