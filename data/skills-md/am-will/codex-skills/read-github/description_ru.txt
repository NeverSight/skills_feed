Прочтите и выполните поиск документации по репозиторию GitHub через службу gitmcp.io MCP.

**КОГДА ИСПОЛЬЗОВАТЬ:**
– Пользователь предоставляет URL-адрес GitHub.
- Пользователь упоминает конкретное репо в формате владелец/репо.
- Пользователь спрашивает: «Что делает этот репозиторий?», «Прочитайте документацию по репозиторию X» или что-то подобное.
- Пользователь хочет выполнить поиск кода или документов в репозитории.
