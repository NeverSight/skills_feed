Leggere e cercare la documentazione del repository GitHub tramite il servizio MCP gitmcp.io.

**QUANDO USARLO:**
- L'utente fornisce un URL GitHub
- L'utente menziona un repository specifico nel formato proprietario/repo
- L'utente chiede "cosa fa questo repository?", "leggi la documentazione per X repository" o simili
- L'utente desidera cercare codice o documenti all'interno di un repository
