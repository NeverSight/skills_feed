Lisez et recherchez la documentation du référentiel GitHub via le service gitmcp.io MCP.

**QUAND UTILISER :**
- L'utilisateur fournit une URL GitHub
- L'utilisateur mentionne un dépôt spécifique au format propriétaire/repo
- L'utilisateur demande "à quoi sert ce dépôt ?", "lire la documentation du dépôt X", ou similaire
- L'utilisateur souhaite rechercher du code ou des documents dans un dépôt
