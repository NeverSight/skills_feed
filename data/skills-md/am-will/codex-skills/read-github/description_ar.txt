اقرأ وثائق مستودع GitHub وابحث فيها عبر خدمة gitmcp.io MCP.

**متى يستخدم:**
- يقدم المستخدم عنوان URL لـ GitHub
- يذكر المستخدم مستودعًا محددًا بتنسيق المالك/الريبو
- يسأل المستخدم "ماذا يفعل هذا الريبو؟" أو "اقرأ المستندات الخاصة بـ X repo" أو ما شابه
- يريد المستخدم البحث عن الكود أو المستندات داخل الريبو
