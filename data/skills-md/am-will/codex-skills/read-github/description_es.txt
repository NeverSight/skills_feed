Lea y busque la documentación del repositorio de GitHub a través del servicio MCP gitmcp.io.

**CUÁNDO USAR:**
- El usuario proporciona una URL de GitHub
- El usuario menciona un repositorio específico en formato propietario/repositorio
- El usuario pregunta "¿qué hace este repositorio?", "leer los documentos del repositorio X" o algo similar.
- El usuario quiere buscar códigos o documentos dentro de un repositorio.
