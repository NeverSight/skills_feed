استعلم عن وثائق مطور OpenAI عبر خادم OpenAI Docs MCP باستخدام CLI (curl/jq). استخدمه عندما تتضمن مهمة واجهة برمجة تطبيقات OpenAI (الاستجابات، وإكمال الدردشة، والوقت الفعلي، وما إلى ذلك)، أو OpenAI SDK، أو ChatGPT Apps SDK، أو Codex، أو عمليات تكامل MCP، أو مخططات نقطة النهاية، أو المعلمات، أو الحدود، أو عمليات الترحيل، وتحتاج إلى إرشادات رسمية محدثة.
