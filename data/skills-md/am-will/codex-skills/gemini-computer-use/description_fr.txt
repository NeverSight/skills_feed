Créez et exécutez un ordinateur Gemini 2.5. Utilisez des agents de contrôle du navigateur avec Playwright. À utiliser lorsqu'un utilisateur souhaite automatiser les tâches du navigateur Web via le modèle Gemini Computer Use, a besoin d'une boucle d'agent (capture d'écran → function_call → action → function_response) ou demande d'intégrer une confirmation de sécurité pour les actions risquées de l'interface utilisateur.
