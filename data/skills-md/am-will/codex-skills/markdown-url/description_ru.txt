Направьте любой веб-сайт, который вам нужно посетить, через markdown.new, добавив URL-адрес в префикс.

**КОГДА ИСПОЛЬЗОВАТЬ:**
- Обычно вы открываете ссылку на веб-сайт, чтобы прочитать контент (документы, сообщения в блогах, журналы изменений, проблемы GitHub и т. д.).
- Вам нужен более чистый, удобный для Markdown вид для копирования заметок или подведения итогов.
