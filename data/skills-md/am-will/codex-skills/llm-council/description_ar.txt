قم بتنسيق مجلس تخطيط CLI متعدد الأعضاء وقابل للتكوين (Codex أو Claude Code أو Gemini أو OpenCode أو Custom) لإنتاج خطط تنفيذ مستقلة، وإخفاء هويتها وعشوائية، ثم الحكم عليها ودمجها في خطة نهائية واحدة. استخدمه عندما تحتاج إلى سير عمل تخطيط قوي ومقاوم للتحيز، ومخرجات JSON منظمة، وإعادة المحاولة، ومعالجة الفشل عبر وكلاء CLI المتعددين.
