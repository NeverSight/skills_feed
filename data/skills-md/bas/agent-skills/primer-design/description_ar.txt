قم بإنشاء واجهات أمامية مميزة على مستوى الإنتاج بجودة تصميم عالية باستخدام نظام التصميم التمهيدي وإرشادات العلامة التجارية. استخدم هذه المهارة عندما يطلب المستخدم إنشاء مكونات ويب أو صفحات أو منتجات أو ملصقات أو تطبيقات (تشمل الأمثلة مواقع الويب أو الصفحات المقصودة أو لوحات المعلومات أو مكونات React أو تخطيطات HTML/CSS أو عند تصميم/تجميل أي واجهة مستخدم ويب). يُنشئ كودًا مصقولًا وتصميمًا لواجهة المستخدم يتوافق تمامًا مع نظام التصميم التمهيدي.
