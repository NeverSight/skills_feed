Text and image generation via reverse-engineered Gemini Web API.

Core capabilities:
- Text generation: generate text responses using the Gemini model
- Image generation: create images from text prompts and save them locally
- Visual input: support reference images for image-to-image or image-to-text
- Multi-turn dialogue: maintain context continuity through sessionId

When to use:
- When you need to generate high-quality AI images (covers, illustrations, etc.)
- When you need text generation via the Gemini model
- When you want to create variants or descriptions based on a reference image
- As an image-generation backend for other skills (pw-cover-image, pw-redbook-image)

Not suitable for:
- Production environments requiring official API support and stability guarantees
- Critical business processes sensitive to API changes
- High-frequency batch calls (may trigger rate limits)

Important reminders:
- First-time use requires user consent to disclaimer
- Google account authentication required (browser login auto-opens)
- Proxy configuration needed in mainland China
- This is an unofficial API and may break at any time
