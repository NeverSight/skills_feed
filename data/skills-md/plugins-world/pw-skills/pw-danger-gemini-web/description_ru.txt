Генерация текста и изображений с помощью обратного проектирования Gemini Web API.

Основные возможности:
- Генерация текста: генерация текстовых ответов с использованием модели Gemini.
- Генерация изображений: создавайте изображения из текстовых подсказок и сохраняйте их локально.
- Визуальный ввод: поддержка эталонных изображений для преобразования изображения в изображение или изображения в текст.
- Многоходовой диалог: поддерживайте непрерывность контекста через sessionId.

Когда использовать:
- Когда вам нужно создать качественные AI-изображения (обложки, иллюстрации и т. д.)
- Когда вам нужна генерация текста по модели Gemini
- Когда вам нужно создать варианты или описания на основе эталонных изображений.
- В качестве бэкэнда для создания изображений для других навыков (pw-cover-image, pw-redbook-image)

Не подходит для:
- Производственные среды, требующие официальной поддержки API и гарантий стабильности.
- Критические бизнес-процессы, чувствительные к изменениям API.
- Высокочастотные пакетные вызовы (могут привести к ограничению скорости)

Важные напоминания:
- При первом использовании требуется согласие пользователя на отказ от ответственности.
- Требуется аутентификация учетной записи Google (браузер автоматически открывается для входа в систему)
- Для доступа из материкового Китая необходима конфигурация прокси-сервера.
- Это неофициальный API, который может сломаться в любой момент.
