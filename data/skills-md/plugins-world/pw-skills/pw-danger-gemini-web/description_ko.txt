일반적인 Gemini Web API는 进行文本과 图image生成입니다.

核心能력:
- 文本生成: 使用 쌍둥이자리 模型生成文本响应
- 图image生成: 从文本提示生成图image并保存到本地
- 视觉输入: 支持参考图image进行图生图或图生文
- 多轮对话: 通过 sessionId 保持上下文连续性

사용 활동:
- 需要生成高质weight AI 图image时(封face图、配图、插画等)
- 需要 Gemini 模型进行文本生成时
- 需要基于参考图image生成变体或描述时
- 作为其他技能 (pw-cover-image、pw-redbook-image) 的图image生成后端

사용하지 않는 방법:
- 需要官方 API는 사용자의 삶을 결정합니다.
- 对 API 变更敏感的关键业务流程
- 需要批weight高频调用场景 (可能触发限流)

심각한 내용:
- 首次使用需要用户同의미免责声明
- 需要 Google 账号认证(自动打开浏览器登录)
- 中國大陆需要配置代理访问
- 这是不官方 API, 可能随时失效
