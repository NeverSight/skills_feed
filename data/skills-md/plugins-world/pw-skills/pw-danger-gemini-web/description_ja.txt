テキストと画像の生成は、逆方向プロセスの Gemini Web API を通じて行われます。

核心能力：
- 文本生成: Gemini モデルを使用して文本を生成
- 画像生成：本文から画像生成を提案し、本地に保存
- ビデオ入力: 参照サポート图像行图生图または图生文
- 多轮对话: sessionId を介して上下文连续性を保持する

タイムマシンを使用する:
- 高质量 AI 画像の生成が必要 (封面図、配図、插画など)
- Gemini モデルの文本生成時に必要
- 参考画像生成体や記述時に必要
- その他のスキル (pw-cover-image、pw-redbook-image) としての画像生成最終端

不適な景色:
- 官方 API サポートと安定性保護の生成環境が必要
- API の感度が向上したアクセス フロー
- 要求量高周波数调用场景 (可能接触制限流)

重要注意:
- 初回使用が必要な使用户免同意责票
- Google 账号认证 (自動打开浏览器登录) が必要です
- 在中国大陆必要配置代理访问
- これは不可官方 API、場合によっては失われる可能性があります
