Встроенный помощник по стилю кодирования на языке C, основанный на стандартах кода для 51 проекта обучения микроконтроллерам. По умолчанию используется Snake_case с дополнительным CamelCase. Создает структуры проектов, соответствующие стандартам разработки встроенных систем, оптимизирует стиль кода и предоставляет шаблоны драйверов оборудования. Подходит для встроенных проектов C на 51 микроконтроллере, STM32 и подобных платформах.
