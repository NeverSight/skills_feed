Assistant de style de codage C intégré basé sur les normes de code pour 51 projets pédagogiques sur les microcontrôleurs. Utilise Snake_case par défaut avec camelCase en option. Crée des structures de projet conformes aux normes de développement intégrées, optimise le style de code et fournit des modèles de pilotes matériels. Convient aux projets C embarqués sur 51 microcontrôleurs, STM32 et plates-formes similaires.
