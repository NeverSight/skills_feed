An automated tool for posting content to WeChat Official Accounts.

Core features:
– Image-text post: multi-image with text, up to 9 images supported
– Article post: full Markdown formatting, preserving styles and layout
– Automated workflow: uses Chrome CDP to log in, fill forms, and publish
– Smart compression: auto-compresses title and content when limits exceeded
– Theme support: article posts support default, grace, and simple themes

When to use:
– User explicitly says “post to WeChat Official Account” or “send to Official Account”
– User provides a Markdown file or content that needs publishing
– User needs to batch-post image-text content

Not applicable:
– User only asks how to post (just give advice)
– User needs to edit an existing Official Account article (manual operation)
– User needs to post to other platforms (use the corresponding platform tools)
