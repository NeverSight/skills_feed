Automated tool for posting content to WeChat Official Accounts.

Core features:
- Image-text post: multi-image with text mode, up to 9 images supported
- Article post: full Markdown formatting, preserving style and layout
- Automated workflow: uses Chrome CDP to auto-login, fill, and publish
- Smart compression: auto-compresses title and content when limits exceeded
- Theme support: article posts support default, grace, and simple themes

When to use:
- User explicitly says “post to WeChat Official Account” or “send to Official Account”
- User provides a Markdown file or content needing publication
- User needs to batch-post image-text content

Not applicable:
- User only asks how to post (just give advice)
- User needs to edit existing Official Account articles (manual operation)
- User needs to post to other platforms (use respective platform tools)
