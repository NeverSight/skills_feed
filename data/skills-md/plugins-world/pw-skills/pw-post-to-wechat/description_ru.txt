Автоматизированный инструмент для публикации контента в официальных аккаунтах WeChat.

Основные возможности:
- Сообщение с изображением текста: несколько изображений в текстовом режиме, поддерживается до 9 изображений.
- Публикация статьи: полное форматирование Markdown, сохранение стиля и макета.
- Автоматизированный рабочий процесс: используется Chrome CDP для автоматического входа в систему, заполнения и публикации.
- Интеллектуальное сжатие: автоматически сжимает заголовок и контент при превышении ограничений.
- Поддержка тем: публикации статей поддерживают темы по умолчанию, изящество и простые темы.

Когда использовать:
- Пользователь прямо говорит «опубликовать в официальной учетной записи WeChat» или «отправить в официальную учетную запись».
- Пользователь предоставляет файл Markdown или контент, требующий публикации.
- Пользователю необходимо пакетно публиковать графический и текстовый контент.

Не применимо:
- Пользователь только спрашивает, как опубликовать (просто дайте совет)
- Пользователю необходимо редактировать существующие статьи официального аккаунта (ручное управление).
- Пользователю необходимо публиковать сообщения на других платформах (используйте соответствующие инструменты платформы).
