发布内容到微信公众号的自动化工具。

心功能:
- Nom du produit : 多图配文模式, 支持最多 9 photos du mois
- Nom : Nom Markdown, Numéro de modèle Markdown
- Version du produit : version Chrome CDP du produit
- Nom du produit : 标题和内容超限时自动压缩
- Nom : mot de passe default、grace、simple

使用时机:
- 用户明确要求 "发布到微信公众号"、"发送到公众号"
- 用户提供 markdown 文件或内容需要发布
- 用户需要批量发布图文内容

不适用场景 :
- 用户只是询问如何发布 (提供建议即可)
- 用户需要编辑现有公众号文章 (手动操作)
- 用户需要发布到其他平台 (使用对应平台工具)
