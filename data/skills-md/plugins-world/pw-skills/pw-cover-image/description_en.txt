A specialized tool for generating stunning cover images for article content.

Core features:
- Analyzes article content and automatically selects the most suitable visual style
- Generates high-quality cover images in a hand-drawn style
- Supports 19 preset styles (elegant, blueprint, bold-editorial, etc.)
- Supports multiple aspect ratios (2.35:1 cinematic, 16:9 widescreen, 1:1 square)
- Automatically extracts core themes and generates title text

When to use:
- User explicitly requests "generate cover image," "create article cover," or "make cover"
- User provides article content and needs accompanying images
- User needs covers for blogs, WeChat articles, or social media

Not applicable:
- User simply asks how to make a cover (just provide advice)
- User needs to edit existing images (use image editing tools)
- User needs to generate non-cover images (use general image generation tools)
