Una habilidad para generar metadatos (títulos, descripciones, hashtags) para videos explicativos técnicos de YouTube. Aprovecha la psicología de aversión a las pérdidas de la teoría prospectiva para los títulos, crea descripciones amigables para el espectador (con tabla de contenido, marcas de tiempo, enlaces sociales) y selecciona hashtags sistemáticos. Úselo mediante indicaciones como "Crear un título de YouTube" o "Generar una descripción de video".
