Eine Fähigkeit zum Generieren von Metadaten (Titel, Beschreibungen, Hashtags) für technische Erklärvideos auf YouTube. Es nutzt die Verlustaversionspsychologie der Prospect-Theorie für Titel, erstellt zuschauerfreundliche Beschreibungen (mit Inhaltsverzeichnis, Zeitstempeln, sozialen Links) und wählt systematische Hashtags aus. Verwenden Sie es über Eingabeaufforderungen wie „Erstellen Sie einen YouTube-Titel“ oder „Generieren Sie eine Videobeschreibung“.
