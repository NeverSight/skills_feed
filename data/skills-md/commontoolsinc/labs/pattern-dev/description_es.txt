Guía para desarrollar patrones CommonTools (módulos de TypeScript que definen transformaciones de datos reactivos con UI). Utilice esta habilidad al crear patrones, modificar patrones existentes o trabajar con el marco de patrones. Los desencadenantes incluyen solicitudes como "crear un patrón", "corregir este error de patrón", "implementar este acceso/parche" o preguntas sobre controladores y patrones reactivos.
