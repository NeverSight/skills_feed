Guía para usar el binario ct (CommonTools) para interactuar con amuletos, recetas y Common Fabric. Utilice esta habilidad al implementar recetas, administrar accesos, vincular datos entre accesos o depurar la ejecución de recetas. Los desencadenantes incluyen solicitudes para "implementar esta receta", "llamar a un controlador", "vincular estos accesos", "obtener datos del acceso" o "probar esta receta localmente".
