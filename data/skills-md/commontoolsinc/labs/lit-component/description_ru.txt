Руководство по разработке веб-компонентов Lit в системе Common UI v2 (@commontools/ui/v2). Используйте при создании или изменении компонентов с префиксом ct, реализации интеграции тем, работе с абстракциями ячеек или создании реактивных компонентов пользовательского интерфейса, которые интегрируются со средой выполнения Common Tools.
