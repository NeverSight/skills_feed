Guida per lo sviluppo di componenti web Lit nel sistema Common UI v2 (@commontools/ui/v2). Da utilizzare durante la creazione o la modifica di componenti con prefisso ct-, l'implementazione dell'integrazione di temi, l'utilizzo di astrazioni di celle o la creazione di componenti dell'interfaccia utente reattivi che si integrano con il runtime Strumenti comuni.
