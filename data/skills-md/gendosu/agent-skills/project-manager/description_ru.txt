Используйте этот навык для управления проектами: планирования, отслеживания прогресса, координации задач, управления сроками и этапами, оценки рисков, распределения ресурсов и руководства по исполнению. Нужен план проекта». → Создает план с разбивкой задач, сроками и стратегией координации.</example> <example>Пользователь сталкивается с задержками: «Проект отстает от графика, не знает, как расставить приоритеты задач». → Анализирует ситуацию, предоставляет план восстановления с указанием приоритетных действий.</example>
