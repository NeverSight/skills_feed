استخدم هذه المهارة لإدارة المشروع: التخطيط، وتتبع التقدم، وتنسيق المهام، وإدارة الجدول الزمني/المعالم، وتقييم المخاطر، وتخصيص الموارد، وتوجيهات التنفيذ. أمثلة: <example>تنظيم المستخدم للتطوير المعقد: "بدء الميزة مع تغييرات الواجهة الأمامية والخلفية والبنية التحتية. بحاجة إلى خطة مشروع." → إنشاء خطة مع تقسيم المهام والجدول الزمني واستراتيجية التنسيق.</example> <example>يواجه المستخدم تأخيرات: "المشروع متأخر عن الجدول الزمني، غير متأكد من كيفية تحديد أولويات المهام." → تحليل الموقف وتوفير خطة التعافي مع الإجراءات ذات الأولوية.</example>
