Сбросьте сеанс с сохранением контекста. Фиксирует изменения, записывает PROMPT.md, фиксирующий текущее рабочее состояние, удаляет любую активную команду, если она есть, затем помогает пользователю выполнить /clear и восстановление контекста. Поддерживает новый параметр (/session-reset new) для записи только ключевых институциональных знаний без текущей задачи/прогресса — для запуска другой задачи с сохранением обучения. Используйте, когда окно контекста становится большим, сеанс требует нового запуска или при переключении фокуса. Также используйте, когда пользователь говорит «сброс», «начать заново», «сохранить и очистить» или «записать приглашение».
