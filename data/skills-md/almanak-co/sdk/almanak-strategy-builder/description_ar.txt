قم ببناء واختبار ونشر استراتيجيات تداول DeFi باستخدام Almanak SDK. استخدم هذه المهارة دائمًا عندما يذكر المستخدم التقويم، أو استراتيجية DeFi، أو استراتيجية التداول، أو زراعة العائد، أو توفير السيولة، أو مبادلة الرموز، أو الاقتراض، أو الإقراض، أو الأبدية، أو التوقيع المساحي، أو إيداع القبو، أو الرموز المميزة، أو الاختبار الخلفي، أو تداول الورق، أو التنفيذ على السلسلة. يُستخدم لكتابة ملفات Strategy.py، أو إنشاء الأغراض (Swap، أو LP، أو Borrow، أو Supply، أو Perp، أو Bridge، أو Stake، أو Vault، أو Prediction)، أو العمل مع معلمات إستراتيجية config.json، أو تشغيل أوامر almanak strat أو almanak gate CLI، أو تصحيح أخطاء تنفيذ الإستراتيجية على شوكات Anvil. لا تستخدم لتطوير العقود الذكية العامة أو رمز Solidity أو أدوات SDK الداخلية غير الإستراتيجية.
