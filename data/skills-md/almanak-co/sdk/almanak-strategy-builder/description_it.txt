Costruisci, testa e implementa strategie di trading DeFi utilizzando Almanak SDK. Usa SEMPRE questa abilità quando l'utente menziona almanacco, strategia DeFi, strategia di trading, yield farming, fornitura di liquidità, scambio di token, prestiti, prestiti, perpetui, staking, deposito in caveau, token ponte, backtesting, trading cartaceo o esecuzione on-chain. Utilizzare per scrivere file Strategy.py, comporre intenti (Swap, LP, Borrow, Supply, Perp, Bridge, Stake, Vault, Prediction), lavorare con parametri di strategia config.json, eseguire comandi CLI almanak strat o almanak gateway o eseguire il debug dell'esecuzione della strategia sui fork Anvil. NON utilizzare per lo sviluppo generale di contratti intelligenti, codice Solidity o interni SDK non strategici.
