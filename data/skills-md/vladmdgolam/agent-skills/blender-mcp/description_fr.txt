Expert Blender MCP pour l'inspection de scènes, les scripts Python, l'exportation GLTF et l'extraction de matériaux/animations. Activer lorsque : (1) vous utilisez les outils Blender MCP (get_scene_info, perform_python, capture d'écran, etc.), (2) vous écrivez des scripts Blender Python pour l'extraction ou la manipulation, (3) vous exportez des scènes vers GLTF/GLB pour le Web (Three.js, R3F), (4) vous déboguez des pertes d'exportation de matériaux ou de textures, (5) vous optimisez des fichiers GLB avec gltf-transform, (6) vous utilisez des intégrations d'actifs (PolyHaven, Sketchfab, Hyper3D Rodin, Hunyuan3D). Couvre les pièges critiques de l'exportation, la survie du mappage des matériaux, le pipeline d'optimisation des textures, les modèles CLI sans tête et les modes de défaillance connus.
