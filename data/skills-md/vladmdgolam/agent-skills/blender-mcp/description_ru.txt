Эксперт Blender MCP по проверке сцен, написанию сценариев Python, экспорту GLTF и извлечению материалов/анимации. Активируйте, когда: (1) используете инструменты Blender MCP (get_scene_info, Execute_python, снимок экрана и т. д.), (2) пишете скрипты Blender Python для извлечения или манипулирования, (3) экспортируете сцены в GLTF/GLB для Интернета (Three.js, R3F), (4) отлаживаете потери материала или экспорта текстур, (5) оптимизируете файлы GLB с помощью gltf-transform, (6) используете интеграцию ресурсов (PolyHaven, Sketchfab, Hyper3D Rodin, Hunyuan3D). Охватывает критические ошибки экспорта, выживание сопоставления материалов, конвейер оптимизации текстур, шаблоны безголового интерфейса командной строки и известные режимы сбоев.
