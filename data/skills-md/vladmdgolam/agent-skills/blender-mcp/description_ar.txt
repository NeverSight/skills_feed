Blender MCP خبير في فحص المشهد والبرمجة النصية لـ Python وتصدير GLTF واستخراج المواد/الرسوم المتحركة. يتم التنشيط عندما: (1) استخدام أدوات Blender MCP (get_scene_info وexecute_python ولقطة الشاشة وما إلى ذلك)، (2) كتابة البرامج النصية لـ Blender Python للاستخراج أو المعالجة، (3) تصدير المشاهد إلى GLTF/GLB للويب (Three.js، R3F)، (4) تصحيح أخطاء فقدان تصدير المواد أو النسيج، (5) تحسين ملفات GLB باستخدام تحويل gltf، (6) استخدام تكامل الأصول (PolyHaven، Sketchfab، Hyper3D رودين، Hunyuan3D). يغطي مشاكل التصدير الهامة، وبقاء رسم خرائط المواد، وخط أنابيب تحسين النسيج، وأنماط CLI مقطوعة الرأس، وأنماط الفشل المعروفة.
