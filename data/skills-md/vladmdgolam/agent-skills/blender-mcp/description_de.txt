Blender MCP-Experte für Szeneninspektion, Python-Skripterstellung, GLTF-Export und Material-/Animationsextraktion. Aktivieren, wenn: (1) Blender-MCP-Tools (get_scene_info,execute_python, Screenshot usw.) verwendet werden, (2) Blender-Python-Skripte zum Extrahieren oder Bearbeiten geschrieben werden, (3) Szenen nach GLTF/GLB für das Web exportiert werden (Three.js, R3F), (4) Material- oder Texturexportverluste behoben werden, (5) GLB-Dateien mit gltf-transform optimiert werden, (6) Asset-Integrationen verwendet werden (PolyHaven, Sketchfab, Hyper3D Rodin, Hunyuan3D). Behandelt kritische Exportprobleme, das Überleben von Materialzuordnungen, die Pipeline zur Texturoptimierung, Headless-CLI-Muster und bekannte Fehlermodi.
