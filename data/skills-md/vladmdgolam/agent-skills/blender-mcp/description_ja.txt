シーン検査、Python スクリプト作成、GLTF エクスポート、マテリアル/アニメーション抽出の Blender MCP エキスパート。次の場合にアクティブ化します: (1) Blender MCP ツール (get_scene_info、execute_python、スクリーンショットなど) を使用する場合、(2) 抽出または操作用の Blender Python スクリプトを作成する場合、(3) Web 用の GLTF/GLB にシーンをエクスポートする場合 (Three.js、R3F)、(4) マテリアルまたはテクスチャのエクスポートの損失をデバッグする場合、(5) gltf-transform で GLB ファイルを最適化する場合、(6) アセット統合を使用する場合 (PolyHaven、 Sketchfab、Hyper3D Rodin、Hunyuan3D)。エクスポートの重要な問題、マテリアル マッピングの存続、テクスチャ最適化パイプライン、ヘッドレス CLI パターン、既知の障害モードについて説明します。
