Создавайте новые изображения из текстовых подсказок с помощью моделей искусственного интеллекта EachLabs. Поддерживает преобразование текста в изображение с несколькими семействами моделей, включая Flux, GPT Image, Gemini, Imagen, Seedream и другие. Используйте, когда пользователь хочет создать новые изображения из текста. Для редактирования существующих изображений см. раздел Eachlabs-image-edit.
