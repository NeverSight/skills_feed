Risolti i problemi di idratazione di React in cui l'input dell'utente digitato prima dell'idratazione veniva cancellato/cancellato quando React prende il sopravvento. Da utilizzare quando (1) gli utenti segnalano che i campi di input vengono cancellati al caricamento della pagina, (2) lavorano con app React SSR/SSG (Next.js, Remix, Astro) che hanno input controllati, (3) controllano i moduli per la sicurezza dell'idratazione o (4) creano nuovi moduli in app React renderizzate staticamente.
