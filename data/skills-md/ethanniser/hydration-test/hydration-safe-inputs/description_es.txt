Solucione los problemas de hidratación de React donde la entrada del usuario escrita antes de que la hidratación se borre/borre cuando React toma el control. Úselo cuando (1) los usuarios informan que los campos de entrada se borran al cargar la página, (2) trabajan con aplicaciones SSR/SSG React (Next.js, Remix, Astro) que tienen entradas controladas, (3) auditan formularios para la seguridad de la hidratación o (4) crean nuevos formularios en aplicaciones React renderizadas estáticamente.
