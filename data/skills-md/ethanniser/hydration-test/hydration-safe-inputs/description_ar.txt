أصلح مشكلات ترطيب React حيث تتم كتابة مدخلات المستخدم قبل مسح/مسح الترطيب عندما تتولى React المسؤولية. يُستخدم عندما (1) يبلغ المستخدمون عن مسح حقول الإدخال عند تحميل الصفحة، أو (2) العمل مع تطبيقات React SSR/SSG (Next.js، وRemix، وAstro) التي تحتوي على مدخلات متحكم فيها، أو (3) نماذج التدقيق لسلامة الترطيب، أو (4) إنشاء نماذج جديدة في تطبيقات React المقدمة بشكل ثابت.
