سير عمل خاص بـ GitHub لإعادة تكوين فرع مصدر كبير وجاهز للمراجعة في سلسلة مرتبة من مجموعات التغييرات الأصغر (الفروع المكدسة + GitHub PRs) التي تحافظ على السلوك وتحسن إمكانية المراجعة. استخدمه عند العمل مع مكدسات GitHub PR عبر gh CLI لتقسيم الفروع المتجانسة، وإنشاء علاقات عامة مكدسة، ودمج مجموعات التغييرات التي تمت مراجعتها، ونشر/تحديث قواعد العلاقات العامة النهائية بعد عمليات الدمج.
