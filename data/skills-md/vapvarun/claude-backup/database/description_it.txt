Progettazione, ottimizzazione e gestione di database per database SQL e NoSQL. Copre la progettazione dello schema, l'indicizzazione, l'ottimizzazione delle query, le migrazioni e le best practice del database. Da utilizzare durante la progettazione di schemi di database, l'ottimizzazione delle query, la risoluzione dei problemi relativi alle prestazioni del database o l'implementazione di modelli di dati.
