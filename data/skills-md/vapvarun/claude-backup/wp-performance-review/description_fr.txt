À utiliser lors de l'examen du code WordPress PHP pour les problèmes de performances, de l'audit des thèmes/plugins pour l'évolutivité, de l'optimisation de WP_Query, de l'analyse des stratégies de mise en cache ou de la détection des anti-modèles dans les requêtes de base de données, les hooks, la mise en cache d'objets, AJAX et le chargement de modèles.
