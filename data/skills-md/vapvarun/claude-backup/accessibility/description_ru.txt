Рекомендации по веб-доступности (a11y) для обеспечения соответствия WCAG 2.1 AA, атрибутов ARIA, навигации с помощью клавиатуры, поддержки чтения с экрана и инклюзивного дизайна. Используйте при аудите доступности, реализации доступных функций, устранении любых проблем или когда пользователь упоминает WCAG, специальные возможности, программы чтения с экрана или навигацию с помощью клавиатуры.
