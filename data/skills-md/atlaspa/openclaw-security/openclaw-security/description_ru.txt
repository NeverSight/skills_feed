Единый пакет безопасности для рабочих мест агентов. Устанавливает, настраивает и координирует все 11 инструментов безопасности OpenClaw с помощью одной команды — целостность, секреты, разрешения, сеть, журнал аудита, подписание, цепочку поставок, учетные данные, защиту от внедрения, соответствие требованиям и реагирование на инциденты.
