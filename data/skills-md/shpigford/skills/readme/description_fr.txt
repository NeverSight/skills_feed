Lorsque l'utilisateur souhaite créer ou mettre à jour un fichier README.md pour un projet. À utiliser également lorsque l'utilisateur dit « écrire le fichier readme », « créer le fichier readme », « documenter ce projet », « documentation du projet » ou demande de l'aide avec README.md. Cette compétence crée une documentation absurdement complète couvrant la configuration locale, l'architecture et le déploiement.
