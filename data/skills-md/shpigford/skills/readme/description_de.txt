Wenn der Benutzer eine README.md-Datei für ein Projekt erstellen oder aktualisieren möchte. Wird auch verwendet, wenn der Benutzer „Readme-Datei schreiben“, „Readme-Datei erstellen“, „Dieses Projekt dokumentieren“, „Projektdokumentation“ sagt oder um Hilfe zu README.md bittet. Diese Fähigkeit erstellt eine absurd gründliche Dokumentation, die die lokale Einrichtung, Architektur und Bereitstellung abdeckt.
