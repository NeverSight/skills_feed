Erstellen Sie neue Fähigkeiten, ändern und verbessern Sie bestehende Fähigkeiten und messen Sie die Fähigkeitsleistung. Verwenden Sie diese Funktion, wenn Benutzer eine Fertigkeit von Grund auf neu erstellen, eine vorhandene Fertigkeit aktualisieren oder optimieren, Evaluierungen durchführen möchten, um eine Fertigkeit zu testen, die Fertigkeitsleistung mit einer Varianzanalyse vergleichen oder die Beschreibung einer Fertigkeit für eine bessere Auslösegenauigkeit optimieren möchten.
