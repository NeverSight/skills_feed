أنشئ تطبيقات معزولة لتنفيذ التعليمات البرمجية بشكل آمن. قم بالتحميل عند إنشاء تنفيذ تعليمات برمجية للذكاء الاصطناعي، أو مترجمات التعليمات البرمجية، أو أنظمة CI/CD، أو بيئات التطوير التفاعلية، أو تنفيذ تعليمات برمجية غير موثوقة. يغطي دورة حياة Sandbox SDK والأوامر والملفات ومترجم التعليمات البرمجية وعناوين URL للمعاينة.
