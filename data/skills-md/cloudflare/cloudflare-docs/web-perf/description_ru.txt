Анализирует веб-производительность с помощью Chrome DevTools MCP. Измеряет основные веб-жизненные показатели (FCP, LCP, TBT, CLS, индекс скорости), определяет ресурсы, блокирующие рендеринг, цепочки сетевых зависимостей, изменения макета, проблемы с кэшированием и пробелы в доступности. Используйте, когда вас просят проверить, профилировать, отладить или оптимизировать производительность загрузки страниц, оценки Lighthouse или скорость сайта.
