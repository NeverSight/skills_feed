أنشئ وكلاء الذكاء الاصطناعي على Cloudflare Workers باستخدام Agents SDK. قم بالتحميل عند إنشاء وكلاء ذوي حالة، أو سير عمل دائم، أو تطبيقات WebSocket في الوقت الفعلي، أو مهام مجدولة، أو خوادم MCP، أو تطبيقات الدردشة. يغطي فئة الوكيل، وإدارة الحالة، وRPC القابل للاستدعاء، وتكامل سير العمل، وخطافات React.
