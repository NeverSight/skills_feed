Создавайте агенты искусственного интеллекта на Cloudflare Worker с помощью Agents SDK. Загружайте при создании агентов с отслеживанием состояния, устойчивых рабочих процессов, приложений WebSocket реального времени, запланированных задач, серверов MCP или приложений чата. Охватывает класс агента, управление состоянием, вызываемый RPC, интеграцию рабочих процессов и перехватчики React.
