Управляйте Chrome без управления через браузер Cloudflare. Рендеринг CDP WebSocket. Используйте для создания снимков экрана, навигации по страницам, очистки данных и захвата видео, когда в среде Cloudflare Workers требуется автоматизация браузера. Требуется переменная среды CDP_SECRET и cdpUrl, настроенные в браузере.profiles.
