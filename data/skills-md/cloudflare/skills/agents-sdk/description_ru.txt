Создавайте агенты искусственного интеллекта с отслеживанием состояния с помощью SDK Cloudflare Agents. Загружайте при создании агентов с постоянным состоянием, планированием, серверами RPC, MCP, обработкой электронной почты или потоковым чатом. Охватывает класс агента, AICchatAgent, управление состоянием и режим кода для сокращения использования токенов.
