Costruisce server MCP (Model Context Protocol) remoti su Cloudflare Workers
con strumenti, autenticazione OAuth e distribuzione in produzione. Genera
codice server, configura i provider di autenticazione e distribuisce ai lavoratori.

Utilizzare quando: l'utente desidera "costruire un server MCP", "creare strumenti MCP", "remoto
MCP", "distribuisci MCP", aggiungi "OAuth a MCP" o menziona Model Context Protocol
su Cloudflare. Si attiva anche su "Autenticazione MCP" o "Distribuzione MCP".
