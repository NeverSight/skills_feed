Analiza el rendimiento web utilizando Chrome DevTools MCP. Mide Core Web Vitals (FCP, LCP, TBT, CLS, índice de velocidad), identifica recursos que bloquean el procesamiento, cadenas de dependencia de la red, cambios de diseño, problemas de almacenamiento en caché y brechas de accesibilidad. Úselo cuando se le solicite auditar, perfilar, depurar u optimizar el rendimiento de carga de la página, las puntuaciones de Lighthouse o la velocidad del sitio.
