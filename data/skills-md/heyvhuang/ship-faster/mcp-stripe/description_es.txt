Habilidad de operaciones de transacciones de Stripe MCP. Ejecute operaciones de transacciones a través del servidor Stripe MCP (https://mcp.stripe.com): gestión de clientes, productos/precios, facturas, enlaces de pago, suscripciones, reembolsos, manejo de disputas, consultas de saldo. Activadores: el usuario solicita operaciones de Stripe que incluyen crear cliente, crear producto, crear factura, generar enlace de pago, consultar transacciones, procesar reembolsos, administrar suscripciones, ver disputas, consultar saldo, etc.
