Навыки транзакционных операций Stripe MCP. Выполнение транзакционных операций через сервер Stripe MCP (https://mcp.stripe.com): управление клиентами, продукты/цены, счета, ссылки для оплаты, подписки, возврат средств, обработка споров, запросы баланса. Триггеры: пользователь запрашивает операции Stripe, включая создание клиента, создание продукта, создание счета, создание ссылки для оплаты, запрос транзакций, обработку возвратов, управление подписками, просмотр споров, проверку баланса и т. д.
