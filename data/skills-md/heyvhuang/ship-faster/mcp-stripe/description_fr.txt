Compétence en opérations de transaction Stripe MCP. Exécuter les opérations de transaction via le serveur Stripe MCP (https://mcp.stripe.com) : gestion des clients, produits/prix, factures, liens de paiement, abonnements, remboursements, gestion des litiges, requêtes de solde. Déclencheurs : l'utilisateur demande des opérations Stripe, notamment créer un client, créer un produit, créer une facture, générer un lien de paiement, interroger des transactions, traiter des remboursements, gérer des abonnements, afficher des litiges, vérifier le solde, etc.
