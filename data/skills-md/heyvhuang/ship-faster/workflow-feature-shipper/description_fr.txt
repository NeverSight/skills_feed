À utiliser lorsque vous devez expédier de bout en bout une seule fonctionnalité de taille PR (planifier -> implémenter -> vérifier) ​​avec des artefacts. Expédiez rapidement les fonctionnalités principales du produit dans une base de code Next.js : transformez une idée de fonctionnalité en un plan exécutable, implémentez-la en tranches de la taille d'un PR et conservez les artefacts sous exécutions/ (ou modifications OpenSpec/ lorsqu'elles sont disponibles). Prend en charge le mode plan uniquement pour une cadrage précoce. Pour le travail de prototype d'interface utilisateur, incluez par défaut un moment wow prêt pour la démonstration (animation/micro-interaction), sauf si l'utilisateur se désengage.
