単一の PR サイズの機能をアーティファクトとともにエンドツーエンド (計画 -> 実装 -> 検証) で出荷する必要がある場合に使用します。 Next.js コードベースでコア製品機能を迅速に出荷します。機能アイデアを実行可能な計画に変換し、PR サイズのスライスで実装し、アーティファクトを実行/ (または利用可能な場合は OpenSpec の変更/) の下に保持します。早期スコーピングのためのプランのみモードをサポートします。プロトタイプ UI 作業の場合、ユーザーがオプトアウトしない限り、デモの準備ができた驚きの瞬間 (アニメーション/マイクロ インタラクション) をデフォルトで含めます。
