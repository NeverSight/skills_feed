Verwenden Sie diese Option, wenn Sie ein einzelnes Feature in PR-Größe durchgängig (Plan -> Implementieren -> Überprüfen) mit Artefakten ausliefern müssen. Liefern Sie Kernproduktfunktionen schnell in einer Next.js-Codebasis: Verwandeln Sie eine Feature-Idee in einen ausführbaren Plan, implementieren Sie sie in Slices in PR-Größe und behalten Sie Artefakte unter Runs/ (oder OpenSpec-Änderungen/, sofern verfügbar). Unterstützt den Nur-Plan-Modus für frühzeitiges Scoping. Fügen Sie für Prototyp-UI-Arbeiten standardmäßig einen demofertigen Wow-Moment (Animation/Mikrointeraktion) hinzu, es sei denn, der Benutzer entscheidet sich dagegen.
