Úselo cuando necesite enviar una única característica del tamaño de PR de extremo a extremo (planificar -> implementar -> verificar) con artefactos. Envíe las funciones principales del producto rápidamente en una base de código Next.js: convierta una idea de función en un plan ejecutable, impleméntela en porciones del tamaño de PR y mantenga los artefactos bajo ejecución (o cambios de OpenSpec cuando estén disponibles). Admite el modo de solo plan para una evaluación temprana del alcance. Para el trabajo de prototipo de interfaz de usuario, incluya un momento sorpresa listo para la demostración (animación/microinteracción) de forma predeterminada, a menos que el usuario opte por no participar.
