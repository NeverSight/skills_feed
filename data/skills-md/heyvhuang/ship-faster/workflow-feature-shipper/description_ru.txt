Используйте, когда вам нужно реализовать одну сквозную функцию размером с PR (планировать -> реализовать -> проверить) с артефактами. Быстро внедряйте основные функции продукта в кодовую базу Next.js: превращайте идею функции в исполняемый план, реализуйте фрагментами размером с PR и сохраняйте артефакты при запуске/(или изменениях OpenSpec/, если они доступны). Поддерживает режим «только план» для раннего определения области действия. Для работы с прототипом пользовательского интерфейса по умолчанию включите готовый к демонстрации демонстрационный момент (анимацию/микровзаимодействие), если пользователь не откажется от этого.
