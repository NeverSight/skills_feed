Schnellerer End-to-End-Workflow für kleine Web-Apps (Standard: Next.js 16.1.1): Idee/Prototyp → Foundation Gate → design-system.md → leichte Leitplanken + Dokumente → Feature-Iteration → optional Supabase + Stripe → optionale GitHub + Vercel-Bereitstellung → optionale KI-Ära-SEO (sitemap/robots/llms.txt). Fortsetzbar, artefaktorientiert unter „runs/ship-faster/“ (oder „OpenSpec changes/“). Auslöser: MVP ausliefern/starten/bereitstellen/produktionsbereit.
