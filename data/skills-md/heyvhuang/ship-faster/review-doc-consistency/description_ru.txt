Средство проверки согласованности документации, которое проверяет соответствие между реализацией кода и документацией. Используйте, когда пользователь запрашивает проверку документации и согласованность кода, проверку актуальности README/документации, проверку точности документации API. Применимо для (1) проверки согласованности README и реализации (2) проверки того, не устарело ли содержимое каталога docs/ (3) проверки точности документации API/конфигурации (4) создания отчетов о согласованности документации. Триггерные слова включают проверку документации, согласованность документации, проверку устаревших документов, проверку документов.
