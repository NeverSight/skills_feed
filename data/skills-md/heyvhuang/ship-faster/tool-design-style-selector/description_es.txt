Úselo cuando necesite definir o hacer converger la dirección visual de un proyecto. Escanee la documentación del proyecto para identificar la intención y luego produzca un design-system.md (conserve el estilo existente o elija entre 30 ajustes preestablecidos). Activadores: sistema de diseño, especificaciones de diseño, estilo de interfaz de usuario, estilo visual, tokens de diseño, paleta de colores, tipografía, diseño. Flujo: escanear → intención → (puerta) preservar vs preestablecer → implementar design-system.md después de la confirmación → (predeterminado) implementar UI/UX según design-system.md (planificar primero, luego ejecutar).
