Используйте, когда вам нужно определить или объединить визуальное направление проекта. Отсканируйте проектную документацию, чтобы определить намерения, а затем создайте файл design-system.md (либо сохраните существующий стиль, либо выберите один из 30 пресетов). Триггеры: система дизайна, спецификация дизайна, стиль пользовательского интерфейса, визуальный стиль, токены дизайна, цветовая палитра, типографика, макет. Последовательность действий: сканирование → намерение → (ворота) сохранение или предустановка → развертывание design-system.md после подтверждения → (по умолчанию) реализация UI/UX для каждого design-system.md (сначала планируйте, затем выполните).
