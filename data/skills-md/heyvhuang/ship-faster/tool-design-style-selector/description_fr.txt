À utiliser lorsque vous devez définir ou faire converger la direction visuelle d'un projet. Scannez la documentation du projet pour identifier l'intention, puis produisez un design-system.md (soit en préservant le style existant, soit en choisissant parmi 30 préréglages). Déclencheurs : système de conception, spécifications de conception, style d'interface utilisateur, style visuel, jetons de conception, palette de couleurs, typographie, mise en page. Flux : analyse → intention → (porte) préserver vs préréglage → déployer design-system.md après confirmation → (par défaut) implémenter l'UI/UX selon design-system.md (planifier d'abord, puis exécuter).
