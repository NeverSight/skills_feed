Opérations d'infrastructure pour Cloudflare : Workers, KV, R2, D1, Hyperdrive, observabilité, builds, journaux d'audit. Déclencheurs : travailleur/KV/R2/D1/logs/build/deploy/audit. Trois niveaux d'autorisation : diagnostiquer (lecture seule), modifier (l'écriture nécessite une confirmation), super administrateur (environnement isolé). Les opérations d'écriture suivent le modèle de lecture en premier, de confirmation, d'exécution et de vérification. MCP est facultatif – fonctionne également avec Wrangler CLI/Dashboard.
