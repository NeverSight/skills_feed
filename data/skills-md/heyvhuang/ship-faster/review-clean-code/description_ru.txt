Анализируйте качество кода на основе принципов «Чистого кода». Выявляйте проблемы с именованием, размером функций, дублированием, чрезмерным проектированием и магическими числами с помощью оценок серьезности и предложений по рефакторингу. Используйте, когда пользователь запрашивает проверку качества кода, рекомендации по рефакторингу, анализ чистого кода, обнаружение запаха кода или упоминает такие термины, как проверка кода, качество кода, проверка рефакторинга.
