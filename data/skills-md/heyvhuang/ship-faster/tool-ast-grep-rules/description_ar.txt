اكتب بحثًا عن التعليمات البرمجية استنادًا إلى AST وأعد كتابة القواعد باستخدام ast-grep YAML. أنشئ قواعد الفحص وتحديثات التعليمات البرمجية وعمليات ترحيل واجهة برمجة التطبيقات باستخدام الإصلاح التلقائي. يُستخدم عندما يذكر المستخدم أنماط ast-grep، أو أنماط تواجد الأشجار، أو قواعد البحث عن التعليمات البرمجية، أو قواعد الوبر مع YAML، أو مطابقة AST، أو أنماط إعادة هيكلة التعليمات البرمجية.
