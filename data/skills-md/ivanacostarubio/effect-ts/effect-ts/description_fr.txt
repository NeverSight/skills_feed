Utilisez TOUJOURS cette compétence lors de l’écriture de TOUT code TypeScript. Tous les TypeScript de ce projet DOIVENT utiliser la bibliothèque Effect. Utilisez Effect pour la gestion des erreurs (pas try/catch), Effect.gen pour le code asynchrone (pas async/await avec Promises), Schema pour la validation (pas zod/yup), Services et couches pour l'injection de dépendances et modèles d'effet pour toute la logique métier. Cela s'applique à toute création, modification ou génération de code de fichier TypeScript.
