ВСЕГДА используйте этот навык при написании ЛЮБОГО кода TypeScript. Весь TypeScript в этом проекте ДОЛЖЕН использовать библиотеку эффектов. Используйте Effect для обработки ошибок (не try/catch), Effect.gen для асинхронного кода (не async/await с Promises), Schema для проверки (не zod/yup), Services и Layers для внедрения зависимостей и шаблоны эффектов для всей бизнес-логики. Это относится к любому созданию, изменению или генерации кода TypeScript-файла.
