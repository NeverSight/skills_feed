Python プロジェクトの初期化時に、フォーマッタとリンターの設定を自動的に構成します。デフォルトで ruff をインストールし、構成に pyproject.toml を使用します。 venv/poetry/uv 環境を検出またはユーザーに質問し、開発依存関係としてインストールします。オプションで、プリコミットと ty を設定します。 VSCode を使用する場合の VSCode ワークスペース設定を提案します。このスキルは、Python プロジェクトを開始するときに使用します (「Python プロジェクトの開始」、「FastAPI プロジェクトの作成」、「データ分析 Python プロジェクトの作成」など)。
