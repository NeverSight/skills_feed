Автоматически настраивает параметры форматирования и линтера при инициализации проектов Python. По умолчанию устанавливает ruff и использует pyproject.toml для настройки. Обнаруживает или запрашивает пользователя о средах venv/poetry/uv и устанавливает как зависимости для разработчиков. При необходимости настраивает предварительную фиксацию и ty. Предлагает настройки рабочей области VSCode при использовании VSCode. Используйте этот навык при запуске проектов Python («Начать проект Python», «Создать проект FastAPI», «Создать проект Python для анализа данных» и т. д.).
