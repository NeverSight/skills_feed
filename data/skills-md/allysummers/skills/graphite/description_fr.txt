À utiliser pour le workflow de PR empilés Graphite CLI dans les dépôts avec .git/.graphite_repo_config.
Déclencheurs : graphite, PR empilés, PR dépendants, PR enchaînés, pile de PR, gt create,
gt modifier, gt soumettre, gt sync, gt restack, gt log, gt checkout, gt up, gt down,
rebaser ma pile, résoudre les conflits de pile, diviser les relations publiques, atterrir ma pile, fusionner la pile,
synchroniser avec principal/tronc, réorganiser les branches, plier les validations, modifier la pile, déplacer la branche
à un parent différent, pile obsolète, mettre à jour ma pile. Pour les repos SANS
.git/.graphite_repo_config, utilisez plutôt les commandes git standard.
