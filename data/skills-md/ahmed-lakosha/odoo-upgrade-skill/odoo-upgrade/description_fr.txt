Assistant complet de mise à niveau Odoo ERP pour la migration des modules entre les versions Odoo (14-19). Gère les vues XML, les modifications de l'API Python, les composants JavaScript/OWL, les variables SCSS de thème et les mises à jour du manifeste. À utiliser lorsque l'utilisateur demande de mettre à niveau les modules Odoo, de résoudre les problèmes de compatibilité des versions, de migrer les thèmes entre les versions ou de résoudre les erreurs de migration Odoo 17/18/19. Spécialisé dans les migrations de services RPC frontaux, les transformations XML d'affichage et la restructuration des variables de thème.
