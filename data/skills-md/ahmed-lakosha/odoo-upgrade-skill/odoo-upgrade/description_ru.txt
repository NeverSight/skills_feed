Комплексный помощник по обновлению Odoo ERP для миграции модулей между версиями Odoo (14–19). Обрабатывает представления XML, изменения API Python, компоненты JavaScript/OWL, переменные SCSS темы и обновления манифеста. Используйте, когда пользователь просит обновить модули Odoo, исправить проблемы совместимости версий, перенести темы между версиями или устранить ошибки миграции Odoo 17/18/19. Специализируется на миграции внешних служб RPC, преобразованиях XML-представлений и реструктуризации переменных темы.
