Asistente integral de actualización de Odoo ERP para migrar módulos entre versiones de Odoo (14-19). Maneja vistas XML, cambios en la API de Python, componentes JavaScript/OWL, variables SCSS de temas y actualizaciones de manifiestos. Úselo cuando el usuario solicite actualizar módulos de Odoo, solucionar problemas de compatibilidad de versiones, migrar temas entre versiones o resolver errores de migración de Odoo 17/18/19. Se especializa en migraciones de servicios frontend RPC, ver transformaciones XML y reestructuración de variables temáticas.
