Assistente completo all'aggiornamento ERP Odoo per la migrazione dei moduli tra le versioni Odoo (14-19). Gestisce visualizzazioni XML, modifiche API Python, componenti JavaScript/OWL, variabili SCSS del tema e aggiornamenti manifest. Da utilizzare quando l'utente chiede di aggiornare i moduli Odoo, risolvere problemi di compatibilità delle versioni, migrare temi tra versioni o risolvere errori di migrazione di Odoo 17/18/19. È specializzato in migrazioni di servizi RPC frontend, visualizzazioni di trasformazioni XML e ristrutturazione di variabili di temi.
