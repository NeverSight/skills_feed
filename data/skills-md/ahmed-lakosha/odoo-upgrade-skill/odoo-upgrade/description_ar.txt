مساعد ترقية Odoo ERP الشامل لترحيل الوحدات بين إصدارات Odoo (14-19). يتعامل مع طرق عرض XML، وتغييرات Python API، ومكونات JavaScript/OWL، ومتغيرات SCSS للموضوع، وتحديثات البيان. يُستخدم عندما يطلب المستخدم ترقية وحدات Odoo، أو إصلاح مشكلات توافق الإصدار، أو ترحيل السمات بين الإصدارات، أو حل أخطاء ترحيل Odoo 17/18/19. متخصص في عمليات ترحيل خدمة RPC للواجهة الأمامية، وعرض تحويلات XML، وإعادة هيكلة متغير السمة.
