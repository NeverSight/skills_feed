Configurez et étendez l'extension Records List Types pour TYPO3 v14. Modes d'affichage Grille, Compact, Teaser et personnalisé pour le module Backend Records avec vignettes, glisser-déposer, mode sombre, indicateurs d'espace de travail et extensibilité zéro PHP via les modèles TSconfig + Fluid. À utiliser lorsque vous travaillez avec une liste d'enregistrements back-end, créez des types de vue personnalisés ou configurez des champs d'affichage par table.
