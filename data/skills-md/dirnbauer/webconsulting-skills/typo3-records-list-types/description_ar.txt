قم بتكوين وتوسيع ملحق أنواع قائمة السجلات لـ TYPO3 v14. أوضاع الشبكة، والضغط، والتشويق، والمخصصة لوحدة السجلات الخلفية مع الصور المصغرة، والسحب والإفلات، والوضع المظلم، ومؤشرات مساحة العمل، وقابلية التوسعة الصفرية PHP عبر قوالب TSconfig + Fluid. يُستخدم عند العمل مع قائمة السجلات الخلفية، أو إنشاء أنواع طرق عرض مخصصة، أو تكوين حقول العرض لكل جدول.
