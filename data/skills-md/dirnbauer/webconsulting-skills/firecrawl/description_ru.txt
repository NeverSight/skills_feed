Официальный навык Firecrawl CLI для очистки веб-страниц, поиска, сканирования и автоматизации браузера. Возвращает чистую уценку, оптимизированную для LLM.

ИСПОЛЬЗОВАТЬ ДЛЯ:
- Поиск и исследование в Интернете.
- Парсинг страниц, документов и статей.
- Картирование сайта и массовое извлечение контента.
- Автоматизация браузера для интерактивных страниц.

Должен быть предварительно установлен и проверен. См. правила/install.md для настройки и правила/security.md для обработки вывода.
