Habilidad oficial de Firecrawl CLI para web scraping, búsqueda, rastreo y automatización del navegador. Devuelve una rebaja limpia optimizada para LLM.

UTILIZAR PARA:
- Búsqueda e investigación web.
- Scraping de páginas, documentos y artículos.
- Mapeo del sitio y extracción masiva de contenido.
- Automatización del navegador para páginas interactivas.

Debe estar preinstalado y autenticado. Consulte reglas/install.md para configuración, reglas/seguridad.md para manejo de salida.
