Создавайте агентные приложения с помощью GitHub Copilot SDK. Используйте при встраивании агентов ИИ в приложения, создании пользовательских инструментов, реализации потоковых ответов, управлении сеансами, подключении к серверам MCP или создании пользовательских агентов. Запускается на Copilot SDK, GitHub SDK, агентском приложении, встроенном Copilot, программируемом агенте, сервере MCP, пользовательском агенте.
