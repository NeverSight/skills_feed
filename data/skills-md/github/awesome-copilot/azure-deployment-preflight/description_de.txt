Führt eine umfassende Preflight-Validierung von Bicep-Bereitstellungen in Azure durch, einschließlich der Validierung der Vorlagensyntax, Was-wäre-wenn-Analysen und Berechtigungsprüfungen. Nutzen Sie diese Fähigkeit vor jeder Bereitstellung in Azure, um Änderungen in der Vorschau anzuzeigen, potenzielle Probleme zu identifizieren und sicherzustellen, dass die Bereitstellung erfolgreich ist. Aktivieren Sie diese Option, wenn Benutzer die Bereitstellung in Azure, die Validierung von Bicep-Dateien, die Überprüfung von Bereitstellungsberechtigungen, die Vorschau von Infrastrukturänderungen, die Ausführung von Was-wäre-wenn oder die Vorbereitung auf die Azd-Bereitstellung erwähnen.
