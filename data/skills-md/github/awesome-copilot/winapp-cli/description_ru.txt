CLI для разработки приложений Windows (winapp) для создания, упаковки и развертывания приложений Windows. Используйте, когда вас попросят инициализировать проекты приложений Windows, создать пакеты MSIX, создать AppxManifest.xml, управлять сертификатами разработки, добавить удостоверение пакета для отладки, подписать пакеты или получить доступ к инструментам сборки Windows SDK. Поддерживает .NET, C++, Electron, Rust, Tauri и кроссплатформенные платформы, ориентированные на Windows.
