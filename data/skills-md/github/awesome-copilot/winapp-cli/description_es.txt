CLI de desarrollo de aplicaciones de Windows (winapp) para crear, empaquetar e implementar aplicaciones de Windows. Úselo cuando se le solicite inicializar proyectos de aplicaciones de Windows, crear paquetes MSIX, generar AppxManifest.xml, administrar certificados de desarrollo, agregar identidad de paquete para depuración, firmar paquetes o acceder a herramientas de compilación de SDK de Windows. Admite .NET, C++, Electron, Rust, Tauri y marcos multiplataforma dirigidos a Windows.
