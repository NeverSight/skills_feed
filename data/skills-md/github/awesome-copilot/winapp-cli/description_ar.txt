Windows App Development CLI (winapp) لإنشاء تطبيقات Windows وتعبئتها ونشرها. يُستخدم عندما يُطلب منك تهيئة مشاريع تطبيقات Windows، أو إنشاء حزم MSIX، أو إنشاء AppxManifest.xml، أو إدارة شهادات التطوير، أو إضافة هوية الحزمة لتصحيح الأخطاء، أو توقيع الحزم، أو الوصول إلى أدوات إنشاء Windows SDK. يدعم .NET، وC++، وElectron، وRust، وTauri، وأطر العمل عبر الأنظمة الأساسية التي تستهدف Windows.
