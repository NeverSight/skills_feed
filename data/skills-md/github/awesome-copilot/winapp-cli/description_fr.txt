CLI de développement d'applications Windows (winapp) pour la création, l'empaquetage et le déploiement d'applications Windows. À utiliser lorsqu'on vous demande d'initialiser des projets d'application Windows, de créer des packages MSIX, de générer AppxManifest.xml, de gérer les certificats de développement, d'ajouter une identité de package pour le débogage, de signer des packages ou d'accéder aux outils de génération du SDK Windows. Prend en charge les frameworks .NET, C++, Electron, Rust, Tauri et multiplateformes ciblant Windows.
