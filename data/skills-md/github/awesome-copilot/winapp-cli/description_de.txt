Windows App Development CLI (winapp) zum Erstellen, Verpacken und Bereitstellen von Windows-Anwendungen. Wird verwendet, wenn Sie dazu aufgefordert werden, Windows-App-Projekte zu initialisieren, MSIX-Pakete zu erstellen, AppxManifest.xml zu generieren, Entwicklungszertifikate zu verwalten, Paketidentität zum Debuggen hinzuzufügen, Pakete zu signieren oder auf Windows SDK-Build-Tools zuzugreifen. Unterstützt .NET, C++, Electron, Rust, Tauri und plattformübergreifende Frameworks für Windows.
