「marked.js」、「pandoc」、「gomarkdown/markdown」、または同様のツールと同様に、Markdown ファイルを HTML に変換します。または、マークダウンを HTML に変換するカスタム スクリプトを作成したり、`jekyll/jekyll`、`gohugoio/hugo` などの Web テンプレート システム、またはマークダウン ドキュメントを利用して HTML に変換する同様の Web テンプレート システムを操作したりすることもできます。 「マークダウンを HTML に変換」、「MD を HTML に変換」、「マークダウンをレンダリング」、「マークダウンから HTML を生成」を要求された場合、または .md ファイルや Web でマークダウンを HTML 出力に変換するテンプレート システムを操作する場合に使用します。 GFM、CommonMark、および標準の Markdown フレーバーを使用した CLI および Node.js ワークフローをサポートします。
