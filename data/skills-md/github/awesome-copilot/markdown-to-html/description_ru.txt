Преобразование файлов Markdown в HTML, аналогичное «marked.js», «pandoc», «gomarkdown/markdown» или аналогичным инструментам; или написание собственного сценария для преобразования уценки в html и/или работы с системами веб-шаблонов, такими как «jekyll/jekyll», «gohugoio/hugo», или аналогичными системами веб-шаблонов, которые используют документы уценки, преобразуя их в html. Используйте, когда вас просят «конвертировать уценку в html», «преобразовать md в html», «отобразить уценку», «сгенерировать HTML из уценки» или при работе с файлами .md и/или веб-системой шаблонов, которая преобразует уценку в вывод HTML. Поддерживает рабочие процессы CLI и Node.js с GFM, CommonMark и стандартными вариантами Markdown.
