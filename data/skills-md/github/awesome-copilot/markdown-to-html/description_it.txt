Converti file Markdown in HTML simili a `marked.js`, `pandoc`, `gomarkdown/markdown` o strumenti simili; o scrivere script personalizzati per convertire markdown in html e/o lavorare su sistemi di modelli web come `jekyll/jekyll`, `gohugoio/hugo` o sistemi di modelli web simili che utilizzano documenti markdown, convertendoli in html. Da utilizzare quando viene richiesto di "convertire markdown in html", "trasformare md in html", "renderizzare markdown", "generare html da markdown" o quando si lavora con file .md e/o web un sistema di modelli che converte markdown in output HTML. Supporta flussi di lavoro CLI e Node.js con versioni GFM, CommonMark e Markdown standard.
