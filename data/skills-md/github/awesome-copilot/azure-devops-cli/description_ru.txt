Управляйте ресурсами Azure DevOps через интерфейс командной строки, включая проекты, репозитории, конвейеры, сборки, запросы на включение, рабочие элементы, артефакты и конечные точки служб. Используйте при работе с Azure DevOps, командами az, автоматизацией DevOps, CI/CD или когда пользователь упоминает Azure DevOps CLI.
