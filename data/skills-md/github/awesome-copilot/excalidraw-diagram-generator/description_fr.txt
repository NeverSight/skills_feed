Générez des diagrammes Excalidraw à partir de descriptions en langage naturel. À utiliser lorsqu'on vous demande de « créer un diagramme », « créer un organigramme », « visualiser un processus », « dessiner une architecture système », « créer une carte mentale » ou « générer un fichier Excalidraw ». Prend en charge les organigrammes, les diagrammes de relations, les cartes mentales et les diagrammes d'architecture système.
