مهارة استخدام أداة سطر الأوامر pdftk (خادم PDFtk) للعمل مع ملفات PDF. استخدمه عندما يُطلب منك دمج ملفات PDF، أو تقسيم ملفات PDF، أو تدوير الصفحات، أو تشفير ملفات PDF أو فك تشفيرها، أو ملء نماذج PDF، أو تطبيق علامات مائية، أو تراكبات الطوابع، أو استخراج البيانات التعريفية، أو تقسيم المستندات إلى صفحات، أو إصلاح ملفات PDF التالفة، أو إرفاق الملفات أو استخراجها، أو تنفيذ أي معالجة لملفات PDF من سطر الأوامر.
