Habilidad para utilizar la herramienta de línea de comandos pdftk (PDFtk Server) para trabajar con archivos PDF. Úselo cuando se le solicite fusionar archivos PDF, dividir archivos PDF, rotar páginas, cifrar o descifrar archivos PDF, completar formularios PDF, aplicar marcas de agua, sellar superposiciones, extraer metadatos, dividir documentos en páginas, reparar archivos PDF corruptos, adjuntar o extraer archivos, o realizar cualquier manipulación de PDF desde la línea de comandos.
