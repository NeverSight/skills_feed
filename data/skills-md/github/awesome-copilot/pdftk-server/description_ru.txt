Навыки использования инструмента командной строки pdftk (PDFtk Server) для работы с PDF-файлами. Используйте, когда вас просят объединить PDF-файлы, разделить PDF-файлы, повернуть страницы, зашифровать или расшифровать PDF-файлы, заполнить PDF-формы, применить водяные знаки, наложить штампы, извлечь метаданные, разбить документы на страницы, восстановить поврежденные PDF-файлы, прикрепить или извлечь файлы или выполнить любые манипуляции с PDF-файлами из командной строки.
