Utilisez cette compétence lorsque quelqu’un souhaite apprendre GitHub Copilot CLI à partir de zéro. Propose des didacticiels interactifs étape par étape avec des pistes distinctes pour les développeurs et les non-développeurs, ainsi que des questions et réponses à la demande. Dites simplement « démarrer le didacticiel » ou posez une question ! Remarque : Cette compétence cible spécifiquement GitHub Copilot CLI et utilise des outils spécifiques à la CLI (ask_user, sql, fetch_copilot_cli_documentation).
