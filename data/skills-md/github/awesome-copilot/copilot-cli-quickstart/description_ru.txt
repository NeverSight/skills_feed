Используйте этот навык, если кто-то хочет изучить интерфейс командной строки GitHub Copilot с нуля. Предлагает интерактивные пошаговые руководства с отдельными курсами для разработчиков и не для разработчиков, а также вопросы и ответы по запросу. Просто скажите «начать обучение» или задайте вопрос! Примечание. Этот навык предназначен специально для GitHub Copilot CLI и использует инструменты, специфичные для CLI (ask_user, sql, fetch_copilot_cli_documentation).
