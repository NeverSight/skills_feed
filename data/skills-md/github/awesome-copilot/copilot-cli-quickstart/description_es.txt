Utilice esta habilidad cuando alguien quiera aprender GitHub Copilot CLI desde cero. Ofrece tutoriales interactivos paso a paso con pistas separadas para desarrolladores y no desarrolladores, además de preguntas y respuestas bajo demanda. ¡Simplemente diga "iniciar tutorial" o haga una pregunta! Nota: Esta habilidad está dirigida específicamente a la CLI de GitHub Copilot y utiliza herramientas específicas de la CLI (ask_user, sql, fetch_copilot_cli_documentation).
