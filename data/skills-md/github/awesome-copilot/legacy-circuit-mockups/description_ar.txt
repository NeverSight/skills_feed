قم بإنشاء نماذج بالحجم الطبيعي لدوائر لوحة التجارب ومخططات مرئية باستخدام تقنيات الرسم القماشية بتنسيق HTML5. استخدمه عندما يُطلب منك إنشاء تخطيطات دوائر، أو تصور مواضع المكونات الإلكترونية، أو رسم مخططات لوحة التجارب، أو تصميمات نموذج بالحجم الطبيعي 6502، أو إنشاء مخططات كمبيوتر قديمة، أو تصميم مشاريع إلكترونية قديمة. يدعم 555 مؤقتًا، والمعالجات الدقيقة W65C02S، و28C256 EEPROMs، ورقائق W65C22 VIA، والبوابات المنطقية من سلسلة 7400، ومصابيح LED، والمقاومات، والمكثفات، والمفاتيح، والأزرار، والبلورات، والأسلاك.
