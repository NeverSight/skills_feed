Создавайте художественные диаграммы ASCII, используя текстовый режим PlantUML. Используйте, когда пользователь запрашивает создание диаграмм ASCII, текстовых диаграмм, диаграмм, удобных для терминала, или упоминает plantuml ascii, текстовую диаграмму, художественную диаграмму ascii. Поддерживает: преобразование диаграмм PlantUML в изображение ASCII, создание диаграмм последовательности, диаграмм классов, блок-схем в формате ASCII, создание изображений ASCII с расширенным Unicode с флагом -utxt.
