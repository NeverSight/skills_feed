قم بإنشاء مخططات فنية ASCII باستخدام وضع النص PlantUML. يُستخدم عندما يطلب المستخدم إنشاء مخططات ASCII، أو مخططات نصية، أو مخططات سهلة الاستخدام للمحطات الطرفية، أو يذكر plantuml ascii، أو مخطط نصي، أو مخطط فني ascii. يدعم: تحويل مخططات PlantUML إلى فن ASCII، وإنشاء مخططات تسلسلية، ورسوم بيانية للفصل، ومخططات انسيابية بتنسيق ASCII، وإنشاء فن ASCII مُحسّن بواسطة Unicode مع علامة -utxt
