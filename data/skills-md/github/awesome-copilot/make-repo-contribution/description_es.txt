Todos los cambios al código deben seguir las instrucciones documentadas en el repositorio. Antes de presentar cualquier problema, realizar una rama, generar confirmaciones o crear una solicitud de extracción (o PR), se debe realizar una búsqueda para garantizar que se sigan los pasos correctos. Siempre que se le solicite crear un problema, confirmar mensajes, enviar código o crear un PR, utilice esta habilidad para que todo se haga correctamente.
