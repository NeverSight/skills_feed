Alle Änderungen am Code müssen den im Repository dokumentierten Leitlinien folgen. Bevor ein Problem eingereicht, eine Verzweigung vorgenommen, Commits generiert oder eine Pull-Anfrage (oder PR) erstellt wird, muss eine Suche durchgeführt werden, um sicherzustellen, dass die richtigen Schritte befolgt werden. Wenn Sie aufgefordert werden, ein Problem zu erstellen, Nachrichten zu übermitteln, Code zu pushen oder eine PR zu erstellen, nutzen Sie diese Fähigkeit, damit alles richtig gemacht wird.
