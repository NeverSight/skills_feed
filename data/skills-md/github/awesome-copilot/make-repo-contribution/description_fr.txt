Toutes les modifications apportées au code doivent suivre les instructions documentées dans le référentiel. Avant qu'un problème ne soit déposé, qu'une branche soit créée, des commits générés ou une demande d'extraction (ou PR) créée, une recherche doit être effectuée pour garantir que les bonnes étapes sont suivies. Chaque fois qu'on vous demande de créer un problème, de valider des messages, de pousser du code ou de créer un PR, utilisez cette compétence pour que tout soit fait correctement.
