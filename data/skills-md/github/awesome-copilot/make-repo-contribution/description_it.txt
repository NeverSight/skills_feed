Tutte le modifiche al codice devono seguire le linee guida documentate nel repository. Prima di archiviare qualsiasi problema, creare un ramo, generare commit o creare una richiesta pull (o PR), è necessario eseguire una ricerca per garantire che vengano seguiti i passaggi corretti. Ogni volta che ti viene chiesto di creare un problema, inviare messaggi, inviare codice o creare un PR, usa questa abilità in modo che tutto venga fatto correttamente.
