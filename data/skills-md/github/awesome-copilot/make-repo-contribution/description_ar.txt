يجب أن تتبع جميع التغييرات في التعليمات البرمجية الإرشادات الموثقة في المستودع. قبل رفع أي مشكلة، أو إنشاء التفرع، أو إنشاء الالتزامات، أو إنشاء طلب سحب (أو PR)، يجب إجراء بحث لضمان اتباع الخطوات الصحيحة. عندما يُطلب منك إنشاء مشكلة، أو إرسال رسائل، أو إرسال رمز، أو إنشاء علاقات عامة، استخدم هذه المهارة حتى يتم كل شيء بشكل صحيح.
