تنفيذ التزام git من خلال تحليل رسائل الالتزام التقليدية والتدريج الذكي وإنشاء الرسائل. يُستخدم عندما يطلب المستخدم تنفيذ تغييرات، أو إنشاء التزام git، أو الإشارة إلى "/commit". يدعم: (1) الاكتشاف التلقائي للنوع والنطاق من التغييرات، (2) إنشاء رسائل الالتزام التقليدية من الفرق، (3) الالتزام التفاعلي مع تجاوزات النوع/النطاق/الوصف الاختيارية، (4) التدريج الذكي للملفات للتجميع المنطقي
