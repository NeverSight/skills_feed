Führen Sie Git Commit mit herkömmlicher Commit-Nachrichtenanalyse, intelligentem Staging und Nachrichtengenerierung aus. Verwenden Sie diese Option, wenn der Benutzer darum bittet, Änderungen zu übernehmen, einen Git-Commit zu erstellen oder „/commit“ erwähnt. Unterstützt: (1) Automatische Erkennung von Typ und Umfang anhand von Änderungen, (2) Generieren herkömmlicher Commit-Nachrichten aus Diff, (3) Interaktives Commit mit optionalen Überschreibungen von Typ/Bereich/Beschreibung, (4) Intelligentes Datei-Staging für logische Gruppierung
