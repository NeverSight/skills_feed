Развивайте навыки агентов по технологиям Microsoft с помощью инструментов Learn MCP. Используйте, когда пользователи хотят создать навык, который обучает агентов любой технологии, библиотеке, платформе или службе Microsoft (Azure, .NET, M365, VS Code, Bicep и т. д.). Глубоко исследует темы, а затем генерирует гибридный навык, позволяющий локально хранить необходимые знания и одновременно обеспечивать более глубокое динамическое исследование.
