Crea competenze di agente per le tecnologie Microsoft utilizzando gli strumenti Learn MCP. Da utilizzare quando gli utenti desiderano creare una competenza che insegni agli agenti qualsiasi tecnologia, libreria, framework o servizio Microsoft (Azure, .NET, M365, VS Code, Bicep e così via). Esamina gli argomenti in modo approfondito, quindi genera un'abilità ibrida che memorizza le conoscenze essenziali a livello locale consentendo al tempo stesso un'indagine dinamica più approfondita.
