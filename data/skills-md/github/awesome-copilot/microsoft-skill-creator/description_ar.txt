قم بإنشاء مهارات الوكيل لتقنيات Microsoft باستخدام أدوات Learn MCP. يُستخدم عندما يريد المستخدمون إنشاء مهارة تعلم الوكلاء أي تقنية أو مكتبة أو إطار عمل أو خدمة من Microsoft (Azure، .NET، M365، VS Code، Bicep، وما إلى ذلك). يبحث في المواضيع بعمق، ثم يولّد مهارة هجينة تخزن المعرفة الأساسية محليًا مع تمكين التحقيق الديناميكي الأعمق.
