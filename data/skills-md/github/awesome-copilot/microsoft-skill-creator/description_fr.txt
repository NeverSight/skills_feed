Créez des compétences d'agent pour les technologies Microsoft à l'aide des outils Learn MCP. À utiliser lorsque les utilisateurs souhaitent créer une compétence qui enseigne aux agents toute technologie, bibliothèque, framework ou service Microsoft (Azure, .NET, M365, VS Code, Bicep, etc.). Étudie les sujets en profondeur, puis génère une compétence hybride stockant les connaissances essentielles localement tout en permettant une enquête dynamique plus approfondie.
