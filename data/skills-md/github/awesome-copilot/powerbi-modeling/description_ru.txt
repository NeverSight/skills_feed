Помощник по семантическому моделированию Power BI для создания оптимизированных моделей данных. Используйте при работе с семантическими моделями Power BI, создании показателей, разработке звездообразных схем, настройке связей, реализации RLS или оптимизации производительности модели. Запускается по запросам о вычислениях DAX, связях таблиц, дизайне таблиц измерений/фактов, соглашениях об именах, документации модели, мощности, направлении перекрестной фильтрации, группах вычислений и передовом опыте модели данных. Всегда сначала подключается к активной модели, используя инструменты Power-Bi-моделирования MCP, чтобы понять структуру данных, прежде чем давать рекомендации.
