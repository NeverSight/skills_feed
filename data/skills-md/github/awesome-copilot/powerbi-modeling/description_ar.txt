مساعد النمذجة الدلالية Power BI لبناء نماذج بيانات محسنة. يُستخدم عند العمل مع النماذج الدلالية Power BI، أو إنشاء المقاييس، أو تصميم المخططات النجمية، أو تكوين العلاقات، أو تنفيذ RLS، أو تحسين أداء النموذج. مشغلات للاستعلامات حول حسابات DAX، وعلاقات الجدول، وتصميم جدول الأبعاد/الحقيقة، واصطلاحات التسمية، ووثائق النموذج، والأصل، واتجاه عامل التصفية المتقاطع، ومجموعات الحساب، وأفضل ممارسات نموذج البيانات. يتصل دائمًا بالنموذج النشط أولاً باستخدام أدوات MCP لنمذجة الطاقة الثنائية لفهم بنية البيانات قبل تقديم التوجيه.
