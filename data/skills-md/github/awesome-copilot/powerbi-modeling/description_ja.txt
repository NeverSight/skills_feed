最適化されたデータ モデルを構築するための Power BI セマンティック モデリング アシスタント。 Power BI セマンティック モデルの操作、メジャーの作成、スター スキーマの設計、関係の構成、RLS の実装、またはモデルのパフォーマンスの最適化を行うときに使用します。 DAX 計算、テーブルの関係、ディメンション/ファクト テーブルの設計、命名規則、モデルのドキュメント、カーディナリティ、クロスフィルターの方向、計算グループ、およびデータ モデルのベスト プラクティスに関するクエリをトリガーします。ガイダンスを提供する前に、常に最初にパワー バイ モデリング MCP ツールを使用してアクティブなモデルに接続し、データ構造を理解します。
