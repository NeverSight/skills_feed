Guide d'utilisation de la bibliothèque de composants Microsoft Fluent UI Blazor (package Microsoft.FluentUI.AspNetCore.Components NuGet) dans les applications Blazor. Utilisez-le lorsque l'utilisateur crée une application Blazor avec des composants Fluent UI, configure la bibliothèque, utilise des composants FluentUI tels que FluentButton, FluentDataGrid, FluentDialog, FluentToast, FluentNavMenu, FluentTextField, FluentSelect, FluentAutocomplete, FluentDesignTheme ou tout composant préfixé par "Fluent". À utiliser également lors du dépannage des fournisseurs manquants, des problèmes d'interopérabilité JS ou de la création de thèmes.
