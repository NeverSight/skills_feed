Руководство по использованию библиотеки компонентов Microsoft Fluent UI Blazor (пакет Microsoft.FluentUI.AspNetCore.Components NuGet) в приложениях Blazor. Используйте это, когда пользователь создает приложение Blazor с компонентами пользовательского интерфейса Fluent, настраивает библиотеку, использует компоненты FluentUI, такие как FluentButton, FluentDataGrid, FluentDialog, FluentToast, FluentNavMenu, FluentTextField, FluentSelect, FluentAutocomplete, FluentDesignTheme или любой компонент с префиксом «Fluent». Также используйте при устранении неполадок с отсутствующими поставщиками, проблемами взаимодействия JS или тем.
