有关在 Blazor 应用程序中使用 Microsoft Fluent UI Blazor 组件库（Microsoft.FluentUI.AspNetCore.Components NuGet 包）的指南。当用户使用 Fluent UI 组件构建 Blazor 应用程序、设置库、使用 FluentUI 组件（例如 FluentButton、FluentDataGrid、FluentDialog、FluentToast、FluentNavMenu、FluentTextField、FluentSelect、FluentAutocomplete、FluentDesignTheme 或任何带有“Fluent”前缀的组件）时，请使用此功能。还可以在对缺少的提供程序、JS 互操作问题或主题进行故障排除时使用。
