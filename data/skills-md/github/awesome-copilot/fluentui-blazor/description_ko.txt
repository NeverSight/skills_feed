Blazor 애플리케이션에서 Microsoft Fluent UI Blazor 구성 요소 라이브러리(Microsoft.FluentUI.AspNetCore.Components NuGet 패키지)를 사용하기 위한 가이드입니다. 사용자가 FluentUI 구성 요소로 Blazor 앱을 빌드하고, 라이브러리를 설정하고, FluentButton, FluentDataGrid, FluentDialog, FluentToast, FluentNavMenu, FluentTextField, FluentSelect, FluentAutocomplete, FluentDesignTheme 또는 "Fluent" 접두사가 붙은 모든 구성 요소와 같은 FluentUI 구성 요소를 사용하는 경우 이 기능을 사용하세요. 또한 누락된 공급자, JS 상호 운용성 문제 또는 테마 문제를 해결할 때 사용합니다.
