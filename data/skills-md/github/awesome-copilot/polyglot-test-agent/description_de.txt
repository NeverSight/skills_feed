Erzeugt mithilfe einer Multi-Agenten-Pipeline umfassende, praktikable Unit-Tests für jede Programmiersprache. Verwenden Sie diese Option, wenn Sie aufgefordert werden, Tests zu generieren, Komponententests zu schreiben, die Testabdeckung zu verbessern, Testabdeckung hinzuzufügen, Testdateien zu erstellen oder eine Codebasis zu testen. Unterstützt C#, TypeScript, JavaScript, Python, Go, Rust, Java und mehr. Orchestriert Forschungs-, Planungs- und Implementierungsphasen, um Tests zu erstellen, die Projektkonventionen kompilieren, bestehen und befolgen.
