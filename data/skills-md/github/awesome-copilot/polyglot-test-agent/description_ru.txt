Создает комплексные работоспособные модульные тесты для любого языка программирования с использованием многоагентного конвейера. Используйте, когда вас просят создать тесты, написать модульные тесты, улучшить тестовое покрытие, добавить тестовое покрытие, создать тестовые файлы или протестировать базу кода. Поддерживает C#, TypeScript, JavaScript, Python, Go, Rust, Java и другие. Организует этапы исследования, планирования и реализации для создания тестов, которые компилируются, проходят и соответствуют соглашениям проекта.
