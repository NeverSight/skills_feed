Génère des tests unitaires complets et réalisables pour n'importe quel langage de programmation à l'aide d'un pipeline multi-agents. À utiliser lorsqu'on vous le demande pour générer des tests, écrire des tests unitaires, améliorer la couverture des tests, ajouter une couverture de test, créer des fichiers de test ou tester une base de code. Prend en charge C#, TypeScript, JavaScript, Python, Go, Rust, Java et plus encore. Orchestre les phases de recherche, de planification et de mise en œuvre pour produire des tests qui compilent, réussissent et suivent les conventions du projet.
