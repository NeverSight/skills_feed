Хирургический рефакторинг кода для улучшения удобства сопровождения без изменения поведения. Охватывает извлечение функций, переименование переменных, разбивку божественных функций, повышение безопасности типов, устранение запахов кода и применение шаблонов проектирования. Менее радикальный, чем репо-восстановитель; использовать для постепенного улучшения.
