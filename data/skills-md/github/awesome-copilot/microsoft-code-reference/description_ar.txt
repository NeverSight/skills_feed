ابحث عن مراجع Microsoft API، وابحث عن نماذج التعليمات البرمجية العاملة، وتحقق من صحة كود SDK. استخدمه عند العمل مع Azure SDKs أو مكتبات .NET أو Microsoft APIs - للعثور على الطريقة الصحيحة أو التحقق من المعلمات أو الحصول على أمثلة عملية أو استكشاف الأخطاء وإصلاحها. يلتقط الأساليب المهلوسة والتوقيعات الخاطئة والأنماط المهملة من خلال الاستعلام عن المستندات الرسمية.
