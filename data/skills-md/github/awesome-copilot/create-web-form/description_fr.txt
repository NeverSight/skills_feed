Créez des formulaires Web robustes et accessibles avec les meilleures pratiques en matière de structure HTML, de style CSS, d'interactivité JavaScript, de validation de formulaire et de traitement côté serveur. À utiliser lorsqu'on vous demande de « créer un formulaire », « créer un formulaire Web », « ajouter un formulaire de contact », « créer un formulaire d'inscription » ou lors de la création d'un formulaire HTML avec gestion des données. Couvre les backends PHP et Python, l'intégration de bases de données MySQL, les API REST, l'échange de données XML, l'accessibilité (ARIA) et les applications Web progressives.
