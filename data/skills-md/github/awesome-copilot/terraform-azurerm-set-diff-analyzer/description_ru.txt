Анализируйте выходные данные JSON плана Terraform для поставщика AzureRM, чтобы различать ложноположительные различия (изменения только по порядку в атрибутах Set-type) и фактические изменения ресурсов. Используйте при просмотре выходных данных плана terraform для ресурсов Azure, таких как шлюз приложений, балансировщик нагрузки, брандмауэр, передняя дверь, NSG и другие ресурсы с атрибутами типа Set, которые вызывают ложные различия из-за внутренних изменений порядка.
