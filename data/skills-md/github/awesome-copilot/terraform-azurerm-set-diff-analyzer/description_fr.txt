Analysez la sortie JSON du plan Terraform pour le fournisseur AzureRM afin de faire la distinction entre les différences faussement positives (modifications de commande uniquement dans les attributs de type ensemble) et les modifications réelles des ressources. À utiliser lors de l'examen de la sortie du plan Terraform pour les ressources Azure telles qu'Application Gateway, Load Balancer, Firewall, Front Door, NSG et d'autres ressources avec des attributs de type Set qui provoquent des différences parasites en raison de modifications d'ordre internes.
