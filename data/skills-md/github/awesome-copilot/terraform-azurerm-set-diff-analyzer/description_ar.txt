قم بتحليل مخرجات JSON لخطة Terraform لموفر AzureRM للتمييز بين الاختلافات الإيجابية الخاطئة (تغييرات الترتيب فقط في سمات نوع المجموعة) وتغييرات الموارد الفعلية. استخدمه عند مراجعة مخرجات خطة Terraform لموارد Azure مثل بوابة التطبيق، وموازن التحميل، وجدار الحماية، والباب الأمامي، وموردي المواد النووية، وغيرها من الموارد ذات سمات نوع المجموعة التي تسبب اختلافات زائفة بسبب تغييرات الترتيب الداخلي.
