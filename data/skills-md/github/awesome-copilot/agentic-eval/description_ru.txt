Шаблоны и методы оценки и улучшения результатов работы агентов ИИ. Используйте этот навык, когда:
- Внедрение циклов самокритики и размышлений.
- Построение конвейеров оценщика-оптимизатора для генерации критически важных по качеству данных.
- Создание рабочих процессов доработки кода на основе тестирования.
- Разработка систем оценки на основе рубрик или LLM в качестве судьи.
- Добавление итеративного улучшения результатов работы агента (код, отчеты, анализ).
- Измерение и улучшение качества ответа агентов
