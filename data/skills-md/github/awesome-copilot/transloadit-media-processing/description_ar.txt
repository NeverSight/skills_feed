معالجة ملفات الوسائط (الفيديو والصوت والصور والمستندات) باستخدام Transloadit. استخدمه عندما يُطلب منك تشفير الفيديو إلى HLS/MP4، أو إنشاء صور مصغرة، أو تغيير حجم الصور أو وضع علامة مائية عليها، أو استخراج الصوت، أو تسلسل المقاطع، أو إضافة ترجمات، أو مستندات التعرف الضوئي على الحروف، أو تشغيل أي مسار لمعالجة الوسائط. يغطي أكثر من 86 روبوت معالجة لتحويل الملفات على نطاق واسع.
