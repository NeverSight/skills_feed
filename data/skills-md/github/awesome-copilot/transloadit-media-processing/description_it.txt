Elabora file multimediali (video, audio, immagini, documenti) utilizzando Transloadit. Da utilizzare quando viene richiesto di codificare video in HLS/MP4, generare miniature, ridimensionare o filigranare immagini, estrarre audio, concatenare clip, aggiungere sottotitoli, documenti OCR o eseguire qualsiasi pipeline di elaborazione multimediale. Copre oltre 86 robot di elaborazione per la trasformazione di file su larga scala.
