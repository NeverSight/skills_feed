Aiuta a creare, configurare e distribuire app Web statiche di Azure utilizzando l'interfaccia della riga di comando SWA. Da usare durante la distribuzione di siti statici in Azure, la configurazione dello sviluppo locale SWA, la configurazione di staticwebapp.config.json, l'aggiunta delle API di Funzioni di Azure a SWA o la configurazione di CI/CD di GitHub Actions per app Web statiche.
