Hilft beim Erstellen, Konfigurieren und Bereitstellen von Azure Static Web Apps mithilfe der SWA-CLI. Verwenden Sie diese Option, wenn Sie statische Websites in Azure bereitstellen, die lokale SWA-Entwicklung einrichten, staticwebapp.config.json konfigurieren, Azure Functions-APIs zu SWA hinzufügen oder GitHub Actions CI/CD für statische Web-Apps einrichten.
