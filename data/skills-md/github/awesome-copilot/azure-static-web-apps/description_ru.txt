Помогает создавать, настраивать и развертывать статические веб-приложения Azure с помощью интерфейса командной строки SWA. Используйте при развертывании статических сайтов в Azure, настройке локальной разработки SWA, настройке staticwebapp.config.json, добавлении API-интерфейсов функций Azure в SWA или настройке GitHub Actions CI/CD для статических веб-приложений.
