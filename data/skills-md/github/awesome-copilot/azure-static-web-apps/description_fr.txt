Aide à créer, configurer et déployer des applications Web statiques Azure à l'aide de la CLI SWA. À utiliser lors du déploiement de sites statiques sur Azure, de la configuration du développement local SWA, de la configuration de staticwebapp.config.json, de l'ajout d'API Azure Functions à SWA ou de la configuration de GitHub Actions CI/CD pour les applications Web statiques.
