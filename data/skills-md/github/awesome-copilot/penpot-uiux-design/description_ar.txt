دليل شامل لإنشاء تصميمات UI/UX احترافية في Penpot باستخدام أدوات MCP. استخدم هذه المهارة عندما: (1) إنشاء تصميمات UI/UX جديدة لتطبيقات الويب أو الهاتف المحمول أو سطح المكتب، (2) إنشاء أنظمة تصميم باستخدام المكونات والرموز المميزة، (3) تصميم لوحات المعلومات أو النماذج أو التنقل أو الصفحات المقصودة، (4) تطبيق معايير إمكانية الوصول وأفضل الممارسات، (5) اتباع إرشادات النظام الأساسي (iOS، Android، تصميم متعدد الأبعاد)، (6) مراجعة تصميمات Penpot الحالية أو تحسينها لسهولة الاستخدام. المشغلات: "تصميم واجهة مستخدم"، "إنشاء واجهة"، "إنشاء تخطيط"، "لوحة تحكم التصميم"، "إنشاء نموذج"، "صفحة مقصودة للتصميم"، "إتاحة الوصول إليها"، "نظام التصميم"، "مكتبة المكونات".
