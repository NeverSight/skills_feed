Esta habilidad permite la inspección visual de sitios web que se ejecutan de forma local o remota para identificar y solucionar problemas de diseño. Activadores de solicitudes como "revisar el diseño del sitio web", "verificar la interfaz de usuario", "arreglar el diseño", "encontrar problemas de diseño". Detecta problemas con el diseño responsivo, la accesibilidad, la coherencia visual y las roturas del diseño, y luego realiza correcciones a nivel del código fuente.
