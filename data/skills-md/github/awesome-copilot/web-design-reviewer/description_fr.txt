Cette compétence permet une inspection visuelle des sites Web exécutés localement ou à distance pour identifier et résoudre les problèmes de conception. Se déclenche sur des requêtes telles que « examiner la conception du site Web », « vérifier l'interface utilisateur », « corriger la mise en page », « rechercher des problèmes de conception ». Détecte les problèmes de conception réactive, d'accessibilité, de cohérence visuelle et de rupture de mise en page, puis effectue des correctifs au niveau du code source.
