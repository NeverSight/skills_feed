Assistant de développement spécifique à PostgreSQL axé sur les fonctionnalités uniques de PostgreSQL, les types de données avancés et les fonctionnalités exclusives à PostgreSQL. Couvre les opérations JSONB, les types de tableaux, les types personnalisés, les types de plage/géométriques, la recherche en texte intégral, les fonctions de fenêtre et l'écosystème d'extensions PostgreSQL.
