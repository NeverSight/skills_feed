このスキルは、ユーザーが「この Web サイトをテストする」、「探索的テストを実行する」、「アクセシビリティの問題をチェックする」、「ログイン フローが機能することを確認する」、「このページのバグを見つける」、または自動 QA テストを要求する場合に使用する必要があります。 ScoutQA CLI を使用したスモーク テスト、アクセシビリティ監査、電子商取引フロー、ユーザー フロー検証などの Web アプリケーション テスト シナリオをトリガーします。重要: Web アプリケーション機能を実装した後、このスキルを積極的に使用して、機能が正しく動作することを確認してください。ユーザーがテストを要求するまで待つ必要はありません。
