Cette compétence doit être utilisée lorsque l'utilisateur demande de « tester ce site Web », « d'exécuter des tests exploratoires », de « vérifier les problèmes d'accessibilité », de « vérifier que le flux de connexion fonctionne », de « trouver des bogues sur cette page » ou demande des tests d'assurance qualité automatisés. Déclencheurs sur des scénarios de test d'applications Web, notamment des tests de fumée, des audits d'accessibilité, des flux de commerce électronique et la validation des flux d'utilisateurs à l'aide de ScoutQA CLI. IMPORTANT : utilisez cette compétence de manière proactive après avoir implémenté les fonctionnalités de l'application Web pour vérifier qu'elles fonctionnent correctement - n'attendez pas que l'utilisateur demande des tests.
