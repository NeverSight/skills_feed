Verwalten Sie NuGet-Pakete in .NET-Projekten/-Lösungen. Verwenden Sie diese Fähigkeit, wenn Sie NuGet-Paketversionen hinzufügen, entfernen oder aktualisieren. Es erzwingt die Verwendung der „dotnet“-CLI für die Paketverwaltung und bietet strenge Verfahren für direkte Dateibearbeitungen nur beim Aktualisieren von Versionen.
