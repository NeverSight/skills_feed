اكتشف أيًا من تبعيات مستودع GitHub يمكن رعايتها عبر رعاة GitHub. يستخدم واجهة برمجة التطبيقات deps.dev لحل التبعية عبر npm وPyPI وCargo وGo وRubyGems وMaven وNuGet. يتحقق من البيانات التعريفية لتمويل npm وملفات FUNDING.yml وبحث الويب. يتحقق من كل رابط. يُظهر التبعيات المباشرة والانتقالية مع البيانات الصحية لبطاقة أداء OSSF. الاستدعاء من خلال توفير مالك/ريبو GitHub (على سبيل المثال، "ابحث عن التبعيات المدعومة في Expressjs/Express").
