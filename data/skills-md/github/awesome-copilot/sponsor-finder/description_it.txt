Scopri quali dipendenze di un repository GitHub sono sponsorizzabili tramite GitHub Sponsors. Utilizza l'API deps.dev per la risoluzione delle dipendenze tra npm, PyPI, Cargo, Go, RubyGems, Maven e NuGet. Controlla i metadati di finanziamento npm, i file FUNDING.yml e la ricerca sul Web. Verifica ogni collegamento. Mostra le dipendenze dirette e transitive con i dati sanitari della OSSF Scorecard. Invocare fornendo un proprietario/repo GitHub (ad esempio "trova dipendenze sponsorizzabili in expressjs/express").
