Recherchez quelles dépendances d'un référentiel GitHub sont sponsorisables via les sponsors GitHub. Utilise l'API deps.dev pour la résolution des dépendances sur npm, PyPI, Cargo, Go, RubyGems, Maven et NuGet. Vérifie les métadonnées de financement npm, les fichiers FUNDING.yml et la recherche sur le Web. Vérifie chaque lien. Affiche les dépendances directes et transitives avec les données de santé OSSF Scorecard. Invoquez en fournissant un propriétaire/dépôt GitHub (par exemple "trouver des dépendances sponsorisables dans expressjs/express").
