Encuentre cuáles de las dependencias de un repositorio de GitHub se pueden patrocinar a través de los patrocinadores de GitHub. Utiliza la API deps.dev para la resolución de dependencias en npm, PyPI, Cargo, Go, RubyGems, Maven y NuGet. Comprueba los metadatos de financiación de npm, los archivos FUNDING.yml y la búsqueda web. Verifica cada enlace. Muestra dependencias directas y transitivas con datos de salud del OSSF Scorecard. Invoque proporcionando un propietario/repositorio de GitHub (por ejemplo, "buscar dependencias patrocinables en expressjs/express").
