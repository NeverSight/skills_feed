مهارة دمج Zotero local API + REST API (Web API v3). - استيراد المراجع دفعة واحدة من DOI وإدارة المجموعات والبحث وعناصر القائمة. عبر REST API، يمكنك إنشاء مجموعات، وتنفيذ العنصر CRUD، وإدارة العلامات، وتحميل المرفقات، وتشغيل مكتبات المجموعة. يُستخدم عندما: (1) تسجيل المراجع في Zotero من قائمة DOI، (2) استرداد قوائم مجموعة/عناصر Zotero، (3) استيراد بيانات BibTeX/RIS إلى Zotero، (4) البحث في معرفات DOI عبر CrossRef API، (5) إنشاء مجموعات وإضافة عناصر وإدارة العلامات عبر REST API، (6) مكتبات مجموعة التشغيل. المشغلات: "zotero"، "文献登録"، "DOI インポート"، "コレクション"، "論文追加"، "文献管理"
