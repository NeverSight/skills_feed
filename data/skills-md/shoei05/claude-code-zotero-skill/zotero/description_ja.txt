Zotero ローカル API + REST API (Web API v3) を統合するスキル。 DOI から参照をバッチインポートし、コレクションを管理し、項目を検索およびリストします。 REST API を介して、コレクションの作成、アイテムの CRUD の実行、タグの管理、添付ファイルのアップロード、グループ ライブラリの操作を行うことができます。次の場合に使用します: (1) DOI リストから Zotero に参照を登録する、(2) Zotero コレクション/アイテム リストを取得する、(3) BibTeX/RIS データを Zotero にインポートする、(4) CrossRef API 経由で DOI を検索する、(5) REST API 経由でコレクションを作成、アイテムを追加、タグを管理する、(6) グループ ライブラリを操作する。トリガー: "zotero"、"文書登録"、"DOI インポート"、"コレクション"、"論文追加"、"文書管理"
