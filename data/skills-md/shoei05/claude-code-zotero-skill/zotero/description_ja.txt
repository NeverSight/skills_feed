Zotero ローカル API + REST API (Web API v3) 連携スキル。DOI からの一括インポート、管理、アイテム検索・一覧表示を行う。 REST API 経由でコレクション作成・アイテム CRUD・タグ管理・添付ファイルアップロード・グループライブラリ操作も可能。 使用する場合: (1) DOI リストから Zotero に登録する、(2) Zotero 文献文献一覧・アイテム一覧を取得する、(3) BibTeX/RIS データをZoteroにインポートする、(4) CrossRef APIでのDOIを検索する、(5) REST APIでコレクション作成・アイテム追加・タグ管理を行う、(6) グループライブラリを操作する。 トリガー: "zotero"、"文書登録"、"DOIインポート"、"コレクション"、"論文追加"、"文書文書管理"
