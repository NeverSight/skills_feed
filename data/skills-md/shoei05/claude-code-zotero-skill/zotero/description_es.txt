Habilidad para integrar API local de Zotero + API REST (Web API v3). Importe por lotes referencias desde DOI, administre colecciones, busque y enumere elementos. A través de la API REST puede crear colecciones, realizar CRUD de elementos, administrar etiquetas, cargar archivos adjuntos y operar bibliotecas de grupo. Úselo cuando: (1) registrar referencias en Zotero desde una lista de DOI, (2) recuperar colecciones/listas de elementos de Zotero, (3) importar datos BibTeX/RIS a Zotero, (4) buscar DOI a través de la API CrossRef, (5) crear colecciones, agregar elementos y administrar etiquetas a través de la API REST, (6) bibliotecas de grupos operativos.
