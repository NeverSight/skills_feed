Fähigkeit zur Integration der lokalen Zotero-API + REST-API (Web-API v3). Importieren Sie Referenzen stapelweise aus DOI, verwalten Sie Sammlungen, suchen und listen Sie Elemente auf. Über die REST-API können Sie Sammlungen erstellen, Element-CRUD durchführen, Tags verwalten, Anhänge hochladen und Gruppenbibliotheken betreiben. Wird verwendet, wenn: (1) Referenzen aus einer DOI-Liste in Zotero registriert werden, (2) Zotero-Sammlungs-/Elementlisten abgerufen werden, (3) BibTeX/RIS-Daten in Zotero importiert werden, (4) DOIs über die CrossRef-API durchsucht werden, (5) Sammlungen erstellt, Elemente hinzugefügt und Tags über die REST-API verwaltet werden, (6) Gruppenbibliotheken betrieben werden. Auslöser: „zotero“, „文献登録“, „DOI インポート“, „コレクション“, „論文追加“, „文献管理“
