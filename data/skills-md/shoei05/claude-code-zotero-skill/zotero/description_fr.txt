Compétence pour intégrer l'API locale Zotero + l'API REST (Web API v3). Importez par lots des références à partir de DOI, gérez les collections, recherchez et répertoriez les éléments. Via l'API REST, vous pouvez créer des collections, effectuer un CRUD d'éléments, gérer des balises, télécharger des pièces jointes et exploiter des bibliothèques de groupe. À utiliser lorsque : (1) l'enregistrement de références dans Zotero à partir d'une liste DOI, (2) la récupération de listes de collections/éléments Zotero, (3) l'importation de données BibTeX/RIS dans Zotero, (4) la recherche de DOI via l'API CrossRef, (5) la création de collections, l'ajout d'éléments et la gestion de balises via l'API REST, (6) l'exploitation de bibliothèques de groupes. Déclencheurs : "zotero", "文献登録", "DOI インポート", "コレクション", "論文追加", "文献管理"
