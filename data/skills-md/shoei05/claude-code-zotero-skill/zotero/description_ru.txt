Навыки интеграции локального API Zotero + REST API (Web API v3). Пакетный импорт ссылок из DOI, управление коллекциями, поиск и составление списков элементов. Через REST API вы можете создавать коллекции, выполнять CRUD элементов, управлять тегами, загружать вложения и управлять групповыми библиотеками. Используйте при: (1) регистрации ссылок в Zotero из списка DOI, (2) получении списков коллекций/элементов Zotero, (3) импорте данных BibTeX/RIS в Zotero, (4) поиске DOI через CrossRef API, (5) создании коллекций, добавлении элементов и управлении тегами через REST API, (6) библиотеках рабочих групп. Триггеры: «zotero», «文献登録», «DOI インポート», «コレクション», «論文追加», «文献管理»
