Füllen Sie fehlende WorkAdventure i18n-Übersetzungen aus, indem Sie das i18n-Diff in play/ ausführen und die Gebietsschemadateien unter play/src/i18n aktualisieren. Verwenden Sie diese Option, wenn Sie aufgefordert werden, npm run i18n:diff auszuführen, fehlende Übersetzungsschlüssel zu reparieren oder Gebietsschemamodule zu vervollständigen.
