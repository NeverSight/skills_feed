Remplissez les traductions WorkAdventure i18n manquantes en exécutant le diff i18n dans play/ et en mettant à jour les fichiers de paramètres régionaux sous play/src/i18n. À utiliser lorsqu'on vous demande d'exécuter npm run i18n:diff, de corriger les clés de traduction manquantes ou de compléter les modules de paramètres régionaux.
