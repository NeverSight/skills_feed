Также срабатывает, когда пользователь спрашивает о ценах на песочницу, ограничениях ресурсов, аутентификации (токены OIDC, токены доступа), спецификациях системы, командах CLI («песочница Vercel») или хочет обновить локальный кеш документации для этого навыка.
