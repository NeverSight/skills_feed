العمل مع Vercel Sandbox — وحدات MicroVM سريعة الزوال لنظام التشغيل Linux لتشغيل تعليمات برمجية غير موثوقة ومخرجات وكيل الذكاء الاصطناعي وتجربة المطورين على Vercel. استخدم هذه المهارة عندما يذكر المستخدم "Vercel Sandbox"، أو "@vercel/sandbox"، أو microVMs لصندوق الحماية، أو تشغيل التعليمات البرمجية في بيئات معزولة على Vercel، أو عندما يريد إنشاء/إدارة/صناديق حماية لقطة عبر TypeScript/Python SDK أو Vercel CLI. يتم تشغيله أيضًا عندما يسأل المستخدم عن تسعير وضع الحماية، أو حدود الموارد، أو المصادقة (رموز OIDC، رموز الوصول)، أو مواصفات النظام، أو أوامر CLI (`vercel sandbox`)، أو يريد تحديث ذاكرة التخزين المؤقت للوثائق المحلية لهذه المهارة.
