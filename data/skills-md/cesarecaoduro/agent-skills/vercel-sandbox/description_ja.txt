Vercel Sandbox を使用します。Vercel 上で信頼できないコード、AI エージェントの出力、開発者の実験を実行するための一時的な Linux microVM です。このスキルは、ユーザーが「Vercel Sandbox」、「@vercel/sandbox」、サンドボックス microVM、Vercel 上の分離環境でコードを実行していることに言及した場合、または TypeScript/Python SDK または Vercel CLI を介してサンドボックスを作成/管理/スナップショットしたい場合に使用します。また、ユーザーがサンドボックスの価格設定、リソース制限、認証 (OIDC トークン、アクセス トークン)、システム仕様、CLI コマンド (「vercel Sandbox」) について質問した場合、またはこのスキルのローカル ドキュメント キャッシュを更新したい場合にもトリガーされます。
