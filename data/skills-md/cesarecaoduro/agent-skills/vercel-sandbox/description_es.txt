Trabaje con Vercel Sandbox: microVM de Linux efímeras para ejecutar código que no es de confianza, salida de agentes de IA y experimentación de desarrolladores en Vercel. Utilice esta habilidad cuando el usuario mencione "Vercel Sandbox", "@vercel/sandbox", microVM de sandbox, ejecute código en entornos aislados en Vercel o quiera crear/administrar/instantáneas sandboxes mediante TypeScript/Python SDK o Vercel CLI. También se activa cuando el usuario pregunta sobre los precios de la zona de pruebas, los límites de recursos, la autenticación (tokens OIDC, tokens de acceso), las especificaciones del sistema, los comandos CLI (`vercel sandbox`) o desea actualizar el caché de documentación local para esta habilidad.
