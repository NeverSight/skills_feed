Travaillez avec Vercel Sandbox : des microVM Linux éphémères pour exécuter du code non fiable, la sortie d'un agent IA et l'expérimentation des développeurs sur Vercel. Utilisez cette compétence lorsque l'utilisateur mentionne « Vercel Sandbox », « @vercel/sandbox », des microVM sandbox, exécute du code dans des environnements isolés sur Vercel, ou souhaite créer/gérer/snapshot instantané via le SDK TypeScript/Python ou la CLI Vercel. Se déclenche également lorsque l'utilisateur pose des questions sur la tarification du sandbox, les limites de ressources, l'authentification (jetons OIDC, jetons d'accès), les spécifications du système, les commandes CLI (`vercel sandbox`) ou souhaite mettre à jour le cache de documentation local pour cette compétence.
