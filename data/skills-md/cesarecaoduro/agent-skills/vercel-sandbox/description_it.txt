Lavora con Vercel Sandbox: microVM Linux temporanee per l'esecuzione di codice non attendibile, output di agenti AI e sperimentazione degli sviluppatori su Vercel. Utilizza questa competenza quando l'utente menziona "Vercel Sandbox", "@vercel/sandbox", microVM sandbox, esegue codice in ambienti isolati su Vercel o desidera creare/gestire/istantanea sandbox tramite TypeScript/Python SDK o Vercel CLI. Si attiva anche quando l'utente chiede informazioni sui prezzi del sandbox, sui limiti delle risorse, sull'autenticazione (token OIDC, token di accesso), sulle specifiche del sistema, sui comandi CLI (`vercel sandbox`) o desidera aggiornare la cache della documentazione locale per questa competenza.
