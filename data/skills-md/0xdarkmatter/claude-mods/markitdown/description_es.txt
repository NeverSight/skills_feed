Convierta documentos locales a Markdown utilizando la CLI de Markitdown de Microsoft. Ideal para: PDF, Word, Excel, PowerPoint, imágenes (OCR), audio. Puede recuperar URL, pero Jina es más rápido para la web. Activadores de: convertir a rebajas, leer PDF, analizar documentos, extraer texto de, docx, xlsx, pptx, imagen OCR, archivo local.
