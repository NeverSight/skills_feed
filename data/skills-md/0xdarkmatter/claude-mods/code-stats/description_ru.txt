Анализируйте кодовую базу с помощью tokei (быстрый подсчет строк по языку) и difft (семантические различия с учетом AST). Получите быстрый обзор проекта без ручного подсчета. Запускается по следующим параметрам: насколько велика кодовая база, подсчет строк кода, какие языки, отображение семантических различий, сравнение файлов, статистика кода.
