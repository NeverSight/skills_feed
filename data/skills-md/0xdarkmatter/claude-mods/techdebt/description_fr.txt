Détection et remédiation des dettes techniques. Exécutez à la fin de la session pour rechercher le code dupliqué, les importations mortes, les problèmes de sécurité et les points chauds de complexité. Déclencheurs : « trouver une dette technologique », « rechercher des problèmes », « vérifier la qualité du code », « conclure la session », « prêt à s'engager », « avant la fusion », « préparer la révision du code ». Utilise toujours des sous-agents parallèles pour une analyse rapide.
