Context7 API を介して最新のライブラリ ドキュメントを取得します。次の場合には積極的に使用してください。
(1) 外部ライブラリ (React、Next.js、Supabase など) を操作する
(2) ユーザーがライブラリ API、パターン、またはベスト プラクティスについて質問する
(3) サードパーティパッケージに依存した機能の実装
(4) ライブラリ固有の問題のデバッグ
(5) トレーニング データのカットオフを超えた最新の文書が必要
(6) そして最も重要なことですが、依存関係、ライブラリ、またはフレームワークをインストールするときは、常にドキュメントをチェックして最新バージョンが何であるかを確認する必要があります。古い知識に頼らないでください。
ライブラリ API を推測したり、古い知識を使用したりするよりも、常にこれを優先します。
