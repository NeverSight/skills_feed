Прочтите и выполните поиск документации по репозиторию GitHub через службу gitmcp.io MCP. Используйте, когда вам предоставлен URL-адрес репозитория GitHub или путь владельца/репозитория и вас просят прочитать документацию, разобраться в кодовой базе, выполнить поиск кода или изучить содержимое репозитория. Преобразует URL-адреса github.com в gitmcp.io для доступа к LLM.
