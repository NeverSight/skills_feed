Lisez et recherchez la documentation du référentiel GitHub via le service gitmcp.io MCP. À utiliser lorsqu'on vous donne une URL de dépôt GitHub ou un chemin de propriétaire/dépôt et qu'on vous demande de lire des documents, de comprendre la base de code, de rechercher du code ou d'explorer le contenu du référentiel. Convertit les URL github.com en gitmcp.io pour un accès convivial LLM.
