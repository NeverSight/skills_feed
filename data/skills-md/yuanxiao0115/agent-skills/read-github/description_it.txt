Leggere e cercare la documentazione del repository GitHub tramite il servizio MCP gitmcp.io. Da utilizzare quando viene fornito un URL del repository GitHub o un proprietario/percorso del repository e viene chiesto di leggere documenti, comprendere la codebase, cercare codice o esplorare i contenuti del repository. Converte gli URL github.com in gitmcp.io per un accesso compatibile con LLM.
