اقرأ وثائق مستودع GitHub وابحث فيها عبر خدمة gitmcp.io MCP. يُستخدم عندما يتم إعطاؤك عنوان URL لـ GitHub repo أو مسار المالك/الريبو ويطلب منك قراءة المستندات أو فهم قاعدة التعليمات البرمجية أو البحث في التعليمات البرمجية أو استكشاف محتويات المستودع. يحول عناوين URL الخاصة بـ github.com إلى gitmcp.io للوصول إلى LLM بسهولة.
