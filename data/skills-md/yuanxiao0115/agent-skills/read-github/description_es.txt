Lea y busque la documentación del repositorio de GitHub a través del servicio MCP gitmcp.io. Úselo cuando se le proporcione una URL de repositorio de GitHub o una ruta de propietario/repositorio y se le solicite leer documentos, comprender el código base, buscar código o explorar el contenido del repositorio. Convierte las URL de github.com a gitmcp.io para un acceso compatible con LLM.
