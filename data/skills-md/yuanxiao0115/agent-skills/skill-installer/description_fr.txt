Installez les compétences Codex dans $CODEX_HOME/skills à partir d'une liste organisée ou d'un chemin de dépôt GitHub. À utiliser lorsqu'un utilisateur demande de répertorier les compétences installables, d'installer une compétence organisée ou d'installer une compétence à partir d'un autre dépôt (y compris les dépôts privés).
