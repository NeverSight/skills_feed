Создает хорошо структурированные задачи GitHub для проекта MCPSpy с помощью инструмента gh CLI. Используйте, когда вас просят создать проблемы, сообщить об ошибках или документировать функции. Следует традиционным именованиям с префиксами «подвиг/работа/исправление» и поддерживает соответствующий уровень детализации.
