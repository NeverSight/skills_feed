Cree un elemento componible de Vue de nivel de biblioteca que acepte entradas quizás reactivas (MaybeRef / MaybeRefOrGetter) para que las personas que llaman puedan pasar un valor simple, una referencia o un captador. Normalice las entradas con toValue()/toRef() dentro de los efectos reactivos (watch/watchEffect) para mantener el comportamiento predecible y reactivo. Utilice esta habilidad cuando el usuario solicite la creación de elementos componibles adaptables o reutilizables.
