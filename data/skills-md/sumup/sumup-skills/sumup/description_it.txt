Guida per la creazione di integrazioni di pagamento SumUp che coprano i flussi di pagamento sia tramite terminale (carta presente) che online (carta non presente) utilizzando SDK e API SumUp. Da utilizzare durante l'implementazione o il debug della creazione di checkout SumUp, dell'elaborazione dei pagamenti, dell'associazione dei lettori, delle integrazioni dei widget delle carte, dei checkout dei lettori API Cloud o della configurazione dell'autorizzazione con chiavi API/OAuth e chiavi di affiliazione.
