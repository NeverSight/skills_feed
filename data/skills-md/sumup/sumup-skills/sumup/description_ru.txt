Руководство по созданию интеграции платежей SumUp, охватывающей потоки оплаты как через терминал (при наличии карты), так и онлайн (без карты) с использованием SumUp SDK и API. Используйте при реализации или отладке создания чеков SumUp, обработки платежей, сопряжения считывателей, интеграции виджетов карт, считывателей Cloud API или настройки авторизации с помощью ключей API/OAuth и партнерских ключей.
