Guía para crear integraciones de pago de SumUp que cubran flujos de pago tanto en terminal (con tarjeta presente) como en línea (sin tarjeta presente) utilizando los SDK y API de SumUp. Úselo al implementar o depurar la creación de pagos de SumUp, el procesamiento de pagos, el emparejamiento de lectores, las integraciones de widgets de tarjetas, los pagos de lectores de API en la nube o la configuración de autorización con claves API/OAuth y claves de afiliados.
