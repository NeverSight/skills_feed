دليل لإنشاء عمليات تكامل دفع SumUp التي تغطي تدفقات الدفع الطرفية (البطاقة الموجودة) وعبر الإنترنت (البطاقة غير موجودة) باستخدام SumUp SDKs وواجهات برمجة التطبيقات. يُستخدم عند تنفيذ أو تصحيح أخطاء إنشاء تسجيل الخروج SumUp، أو معالجة الدفع، أو إقران القارئ، أو عمليات تكامل عناصر واجهة المستخدم للبطاقة، أو عمليات سحب قارئ Cloud API، أو إعداد التفويض باستخدام مفاتيح API/OAuth والمفاتيح التابعة.
