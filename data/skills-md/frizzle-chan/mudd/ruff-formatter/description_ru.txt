Исправьте проблемы с форматированием кода Python с помощью форматтера Ruff. Используйте, когда: (1) ошибки форматирования обнаружены с помощью ruff format --check, (2) файлы Python необходимо отформатировать в соответствии со стилем проекта, (3) перехватчики предварительной фиксации или CI не работают из-за проблем с форматированием.
