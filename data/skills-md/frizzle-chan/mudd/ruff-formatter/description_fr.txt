Résolvez les problèmes de formatage du code Python à l'aide du formateur Ruff. À utiliser lorsque : (1) des erreurs de formatage sont détectées par ruff format --check, (2) les fichiers Python doivent être formatés pour correspondre au style du projet, (3) les hooks de pré-validation ou CI échouent en raison de problèmes de formatage.
