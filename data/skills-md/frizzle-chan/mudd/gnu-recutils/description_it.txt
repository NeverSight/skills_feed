Lavora con recutils GNU per database di record in testo semplice. Da utilizzare durante la creazione, l'esecuzione di query o la convalida di file .rec, la definizione di schemi di record con descrittori %rec, la conversione tra formati rec/CSV o quando l'utente menziona recutils, recsel, recins, recfmt o file di dati orientati ai record.
