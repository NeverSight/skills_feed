العمل مع نسخ GNU لقواعد بيانات سجل النص العادي. يُستخدم عند إنشاء ملفات .rec أو الاستعلام عنها أو التحقق من صحتها، أو تحديد مخططات السجلات باستخدام واصفات %rec، أو التحويل بين تنسيقات recutil/CSV، أو عندما يذكر المستخدم ملفات البيانات recutils أو reccel أو recins أو recins أو ملفات البيانات الموجهة نحو السجلات.
