Maîtrisez les compétences pour une exécution parallèle pilotée par des sous-agents avec retour automatique au mode séquentiel mono-agent. À utiliser lors de la mise en œuvre de plans avec plusieurs sous-phases indépendantes (SP1, SP2...) pour répartir des sous-agents parallèles, ou lorsqu'il est nécessaire de revoir le code entre la mise en œuvre et les tests. Phrases de déclenchement : « paralléliser », « distribuer le sous-agent », « diviser en sous-phases », « sous-agent de révision du code », « tests parallèles ».
