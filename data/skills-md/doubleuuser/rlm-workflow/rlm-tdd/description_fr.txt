À utiliser lors de l'implémentation de n'importe quel code dans la phase 3 de RLM. Applique une discipline stricte ROUGE-VERT-REFACTOR avec la loi de fer - pas de code de production sans un test d'échec au préalable. Phrases de déclenchement : « implémenter ceci », « ajouter une fonctionnalité », « corriger un bug », « écrire un test ayant échoué », « TDD ».
