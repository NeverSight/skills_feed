Applica uno stile supponente all'HTML barebone. Da utilizzare quando l'utente dispone di codice HTML semplice/senza stile e desidera applicare uno stile visivo coerente. Trigger: applica uno stile a questo HTML, applica uno stile, rendilo bello, in stile /html o quando l'utente condivide HTML che necessita di CSS. Trasforma tabelle, elenchi, indicatori di stato, pulsanti e layout in un sistema di progettazione coerente.
