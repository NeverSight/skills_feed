Appliquez un style opiniâtre au HTML barebones. À utiliser lorsque l'utilisateur dispose d'un code HTML brut/sans style et souhaite appliquer un style visuel cohérent. Déclencheurs : stylisez ce HTML, appliquez un style, donnez-lui une belle apparence, dans le style /html, ou lorsque l'utilisateur partage du HTML nécessitant du CSS. Transforme les tableaux, listes, indicateurs d'état, boutons et mises en page en un système de conception cohérent.
