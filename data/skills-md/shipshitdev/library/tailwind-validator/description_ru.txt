Проверка конфигурации Tailwind CSS v4 и обнаружение/предотвращение шаблонов Tailwind v3. Используйте этот навык при настройке Tailwind, аудите конфигурации CSS или когда вы подозреваете, что используются устаревшие шаблоны Tailwind. Обеспечивает настройку CSS в первую очередь с помощью блоков @theme.
