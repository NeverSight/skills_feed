Organice, clone e implemente páginas de inicio de NextJS basadas en configuración que utilicen un paquete de componentes de interfaz de usuario compartidos. Utilice esta habilidad al crear una o varias páginas de inicio de inicio con captura de correo electrónico, análisis y diseño moderno. Admite la creación por lotes a partir de plantillas o archivos CSV/JSON y la implementación automática de Vercel con dominios personalizados. Cada aterrizaje es una aplicación NextJS independiente impulsada por un archivo de configuración app.json.
