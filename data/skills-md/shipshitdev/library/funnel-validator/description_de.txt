Verwenden Sie diese Fähigkeit, wenn Benutzer einen vorhandenen Verkaufstrichter, eine Zielseitensequenz oder eine Customer Journey validieren müssen. Wird aktiviert für „Meinen Trichter validieren“, „Ist mein Trichter gut“, „Warum konvertiert mein Trichter nicht“ oder wenn die Trichterqualität überprüft wird, bevor der Traffic gesteigert wird.
