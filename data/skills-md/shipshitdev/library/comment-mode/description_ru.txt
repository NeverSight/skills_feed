Подробная обратная связь по черновикам без переписывания. Создает выделенный HTML-код со встроенными комментариями, которые можно открыть по щелчку. Используйте, когда пользователь говорит «прокомментировать это», «оставить комментарий», «оставить отзыв» или просит оставить отзыв о черновике. Поддерживает несколько объективов — отзывы редактора, симуляцию точки зрения («как отреагировал бы Брайан») или сфокусированные ракурсы («только выбор слов», «слабые аргументы»). Детализированная альтернатива переписыванию, которая позволяет пользователям постепенно просматривать отзывы, не теряя при этом голоса.
