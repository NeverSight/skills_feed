Richten Sie einen .devcontainer für VS Code mit Docker, Claude Code CLI-Unterstützung und konfigurierbaren Optionen ein. Verwenden Sie diese Option, wenn der Benutzer „Devcontainer einrichten“, „Docker-Entwicklungsumgebung hinzufügen“ oder „Dev-Container konfigurieren“ fragt oder seinen Entwicklungsworkflow containerisieren muss.
