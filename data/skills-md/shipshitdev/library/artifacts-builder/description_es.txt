Conjunto de herramientas para crear artefactos HTML claude.ai elaborados y de múltiples componentes utilizando tecnologías web frontend modernas (React, Tailwind CSS, @agenticindiedev/ui). Úselo para artefactos complejos que requieren administración de estado o componentes de interfaz de usuario compartidos, no para artefactos HTML/JSX de un solo archivo simples.
