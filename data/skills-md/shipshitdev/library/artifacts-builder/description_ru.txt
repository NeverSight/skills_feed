Набор инструментов для создания сложных многокомпонентных HTML-артефактов claude.ai с использованием современных веб-технологий внешнего интерфейса (React, Tailwind CSS, @agenticindiedev/ui). Используйте для сложных артефактов, требующих управления состоянием или общих компонентов пользовательского интерфейса, а не для простых однофайловых артефактов HTML/JSX.
