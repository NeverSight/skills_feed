Организует комплексный аудит производительности в полнофункциональных монорепозиториях. Координирует навыки эксперта по производительности, аудитора согласованности дизайна, эксперта по доступности, эксперта по безопасности и специалиста по контролю качества для аудита внешнего интерфейса, серверной части, базы данных, расширений браузера и общих пакетов.
