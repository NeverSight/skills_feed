Configura o verifica gli hook git Husky per garantire che tutti i test vengano eseguiti e che la copertura rimanga superiore all'80% (configurabile) per i progetti Node.js/TypeScript. Questa competenza deve essere utilizzata quando gli utenti desiderano applicare la copertura del test tramite hook di pre-commit, verificare la configurazione Husky/test esistente o configurare le soglie di copertura per i test runner Jest, Vitest o Mocha.
