Configurez ou vérifiez les hooks git Husky pour garantir que tous les tests sont exécutés et que la couverture reste supérieure à 80 % (configurable) pour les projets Node.js/TypeScript. Cette compétence doit être utilisée lorsque les utilisateurs souhaitent appliquer la couverture de test via des hooks de pré-validation, vérifier la configuration Husky/test existante ou configurer des seuils de couverture pour les exécuteurs de test Jest, Vitest ou Mocha.
