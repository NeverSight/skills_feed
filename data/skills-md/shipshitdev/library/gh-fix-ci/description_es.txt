Inspeccione las comprobaciones de relaciones públicas de GitHub con gh, extraiga los registros de acciones de GitHub fallidas, resuma el contexto del error y luego cree un plan de reparación e impleméntelo después de la aprobación del usuario. Úselo cuando un usuario solicita depurar o corregir comprobaciones fallidas de PR CI/CD en GitHub Actions y desea un plan más cambios de código; para comprobaciones externas (por ejemplo, Buildkite), solo informe la URL de detalles y márquelas fuera de alcance.
