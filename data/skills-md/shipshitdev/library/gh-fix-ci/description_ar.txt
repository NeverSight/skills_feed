افحص عمليات فحص GitHub PR باستخدام gh، واسحب سجلات إجراءات GitHub الفاشلة، ولخص سياق الفشل، ثم أنشئ خطة إصلاح وقم بتنفيذها بعد موافقة المستخدم. يُستخدم عندما يطلب المستخدم تصحيح الأخطاء أو إصلاح عمليات فحص PR CI/CD الفاشلة في إجراءات GitHub ويريد تغيير الخطة بالإضافة إلى التعليمات البرمجية؛ بالنسبة لعمليات التحقق الخارجية (على سبيل المثال، Buildkite)، قم فقط بالإبلاغ عن عنوان URL الخاص بالتفاصيل ووضع علامة عليها خارج النطاق.
