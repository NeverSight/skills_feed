Utilizza questa competenza quando gli utenti richiedono nuove funzionalità, miglioramenti, correzioni di bug o qualsiasi lavoro che richieda pianificazione. Crea file di attività strutturati e PRD (documenti sui requisiti del prodotto) prima dell'implementazione. Si attiva per "Desidero aggiungere X", "implementare Y", "creare un'attività per Z", "pianificare questa funzionalità" o qualsiasi richiesta di funzionalità.
