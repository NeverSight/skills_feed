Используйте этот навык, когда пользователи запрашивают новые функции, улучшения, исправления ошибок или любую работу, требующую планирования. Перед внедрением создает структурированные файлы задач и PRD (документы с требованиями к продукту). Активируется для «Я хочу добавить X», «реализовать Y», «создать задачу для Z», «запланировать эту функцию» или любого запроса функции.
