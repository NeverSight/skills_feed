قم بإنشاء الحد الأدنى من صفحات HTML لمراجعة مخرجات Claude Code في المتصفح. يُستخدم عندما يكون من الصعب قراءة مخرجات الوحدة الطرفية، أو عند مراجعة القوائم/الجداول/المسودات، أو عندما يقول المستخدم "أرني"، أو "اجعل هذا قابلاً للمراجعة"، أو "عرض سريع"، أو "فتح كصفحة ويب". ينتج HTML الدلالي غير المنسق فقط. للحصول على تعليقات تفصيلية مع التعليقات المضمنة، راجع مهارة وضع التعليق.
