Создавайте минимальные HTML-страницы для просмотра вывода Claude Code в браузере. Используйте, когда вывод терминала трудно читать, при просмотре списков/таблиц/черновиков или когда пользователь говорит «покажи мне», «сделай это доступным для просмотра», «быстрый просмотр» или «открыть как веб-страницу». Создает только нестилизованный семантический HTML. Подробную обратную связь со встроенными комментариями см. в разделе «Режим комментариев».
