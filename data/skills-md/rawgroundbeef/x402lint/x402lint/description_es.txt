Valide y cree respuestas de punto final de pago x402 (se requiere pago HTTP 402). Úselo cuando el usuario solicite: (1) validar una configuración x402 o una respuesta 402, (2) crear/generar una configuración de pago x402, (3) crear un punto final HTTP 402 que devuelva los requisitos de pago, (4) depurar por qué una configuración x402 no es válida, (5) convertir entre los formatos x402 v1 y v2, (6) verificar las direcciones EVM/Solana para x402 o (7) trabajar con Identificadores de red CAIP-2 para configuraciones de pago. Activadores de palabras clave: x402, pago 402, encabezado de pago requerido, configuración de muro de pago, pago CAIP-2.
