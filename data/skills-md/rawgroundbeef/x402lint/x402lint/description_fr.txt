Validez et créez des réponses de point de terminaison de paiement x402 (HTTP 402 Payment Required). À utiliser lorsque l'utilisateur demande à : (1) valider une configuration x402 ou une réponse 402, (2) créer/générer une configuration de paiement x402, (3) créer un point de terminaison HTTP 402 qui renvoie les exigences de paiement, (4) déboguer pourquoi une configuration x402 n'est pas valide, (5) convertir entre les formats x402 v1 et v2, (6) vérifier les adresses EVM/Solana pour x402, ou (7) travailler avec Identifiants réseau CAIP-2 pour les configurations de paiement. Déclencheurs sur mots-clés : x402, paiement 402, en-tête de paiement requis, configuration du paywall, paiement CAIP-2.
