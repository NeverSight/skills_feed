قم بإنشاء وثائق شاملة لواجهة برمجة التطبيقات (API) بما في ذلك أوصاف نقطة النهاية وأمثلة الطلب/الاستجابة وأدلة المصادقة وأكواد الأخطاء ومجموعات SDK. ينشئ مواصفات OpenAPI/Swagger ومستندات REST API ومواد مرجعية سهلة الاستخدام للمطورين. يُستخدم عندما يحتاج المستخدمون إلى توثيق واجهات برمجة التطبيقات، أو إنشاء مراجع فنية، أو كتابة وثائق المطور.
