إنشاء تطبيقات الجيل المعزز للاسترجاع (RAG) التي تجمع بين قدرات LLM ومصادر المعرفة الخارجية. يغطي قواعد بيانات المتجهات والتضمينات واستراتيجيات الاسترجاع وتوليد الاستجابة. يُستخدم عند إنشاء أنظمة أسئلة وأجوبة للمستندات، أو تطبيقات قاعدة المعرفة، أو البحث في المؤسسات، أو دمج LLMs مع البيانات المخصصة.
