Создавайте приложения дополненной генерации (RAG), которые сочетают возможности LLM с внешними источниками знаний. Охватывает векторные базы данных, встраивания, стратегии поиска и генерацию ответов. Используйте при создании систем вопросов и ответов для документов, приложений баз знаний, корпоративного поиска или объединения LLM с пользовательскими данными.
