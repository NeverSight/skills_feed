Стратегические шаблоны навигации и руководства по выбору для тщательного изучения веб-страниц с помощью Playwright MCP. Предоставляет деревья решений, стратегии навигации и селекторы для конкретного сайта для систематического чтения нескольких страниц. Используйте при планировании навигации по веб-сайтам, определении глубины чтения или поиске правильных селекторов для команд Playwright MCP.
