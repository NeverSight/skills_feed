`flow_setting.py` (フロー バンドル) で flow.kit ベースの test.guide ワークフローを作成、デバッグ、最適化します。 「FlowBuilder」/「add_block_with」/「Assign」を使用してフローを設計または変更する場合、flow.kit ブロック (トリガー、test.guide、制御フロー、ユーザーコード) を選択/接続する場合、「main.py」を実行して検証/実行/視覚化する場合、または「.vscode/flow-kit-snippets.code-snippets」および「flow_kit/**/*.pyi」を参照する場合に使用します。 API の詳細については、「site-packages/**/*.pyi」を参照してください。
