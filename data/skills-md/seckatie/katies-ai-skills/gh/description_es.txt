GitHub CLI (gh) para gestión de repositorios, conjuntos de reglas, lanzamientos, relaciones públicas y problemas. Esta habilidad se activa cuando el usuario dice cosas como "crear un PR de GitHub", "enumerar problemas de GitHub", "configurar protección de rama", "crear un conjunto de reglas", "configurar conjuntos de reglas de GitHub", "crear una versión de GitHub", "clonar este repositorio" o "administrar la configuración del repositorio de GitHub".
