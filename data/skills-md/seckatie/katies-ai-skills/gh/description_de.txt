GitHub CLI (gh) für Repository-Verwaltung, Regelsätze, Releases, PRs und Probleme. Dieser Skill wird ausgelöst, wenn der Benutzer Dinge sagt wie „GitHub-PR erstellen“, „GitHub-Probleme auflisten“, „Zweigschutz einrichten“, „einen Regelsatz erstellen“, „GitHub-Regelsätze konfigurieren“, „eine GitHub-Version erstellen“, „dieses Repo klonen“ oder „GitHub-Repository-Einstellungen verwalten“.
