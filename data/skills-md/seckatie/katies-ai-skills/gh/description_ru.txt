GitHub CLI (gh) для управления репозиторием, наборами правил, выпусками, PR и проблемами. Этот навык срабатывает, когда пользователь говорит такие вещи, как «создать PR GitHub», «составить список проблем GitHub», «настроить защиту ветки», «создать набор правил», «настроить наборы правил GitHub», «создать выпуск GitHub», «клонировать этот репозиторий» или «управлять настройками репозитория GitHub».
