GitHub CLI (gh) pour la gestion des référentiels, les ensembles de règles, les versions, les PR et les problèmes. Cette compétence est déclenchée lorsque l'utilisateur dit des choses comme « créer un PR GitHub », « répertorier les problèmes GitHub », « configurer la protection des branches », « créer un ensemble de règles », « configurer les ensembles de règles GitHub », « créer une version GitHub », « cloner ce dépôt » ou « gérer les paramètres du référentiel GitHub ».
