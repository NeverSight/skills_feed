Accedi e interagisci con modelli linguistici di grandi dimensioni dalla riga di comando utilizzando lo strumento CLI llm di Simon Willison. Supporta OpenAI, Anthropic, Gemini, Llama e dozzine di altri modelli tramite plugin. Le funzionalità includono sessioni di chat, incorporamenti, estrazione strutturata di dati con schemi, modelli di prompt, registrazione delle conversazioni e utilizzo di strumenti. Questa abilità viene attivata quando l'utente dice cose come "esegui un prompt con llm", "usa il comando llm", "chiama un LLM dalla riga di comando", "imposta le chiavi API llm", "installa plugin llm", "crea incorporamenti" o "estrai dati strutturati dal testo".
