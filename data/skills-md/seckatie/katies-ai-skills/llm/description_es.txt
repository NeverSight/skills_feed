Acceda e interactúe con modelos de lenguaje grandes desde la línea de comandos utilizando la herramienta CLI llm de Simon Willison. Admite OpenAI, Anthropic, Gemini, Llama y docenas de otros modelos a través de complementos. Las características incluyen sesiones de chat, incrustaciones, extracción de datos estructurados con esquemas, plantillas de mensajes, registro de conversaciones y uso de herramientas. Esta habilidad se activa cuando el usuario dice cosas como "ejecutar un mensaje con llm", "usar el comando llm", "llamar a un LLM desde la línea de comando", "configurar claves API de llm", "instalar complementos de llm", "crear incrustaciones" o "extraer datos estructurados del texto".
