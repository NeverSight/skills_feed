Greifen Sie über die Befehlszeile mit dem llm CLI-Tool von Simon Willison auf große Sprachmodelle zu und interagieren Sie mit ihnen. Unterstützt OpenAI, Anthropic, Gemini, Llama und Dutzende anderer Modelle über Plugins. Zu den Funktionen gehören Chat-Sitzungen, Einbettungen, strukturierte Datenextraktion mit Schemata, Eingabeaufforderungsvorlagen, Konversationsprotokollierung und Tool-Nutzung. Dieser Skill wird ausgelöst, wenn der Benutzer Dinge sagt wie „Führen Sie eine Eingabeaufforderung mit llm aus“, „verwenden Sie den llm-Befehl“, „rufen Sie ein LLM über die Befehlszeile auf“, „richten Sie llm-API-Schlüssel ein“, „installieren Sie llm-Plugins“, „erstellen Sie Einbettungen“ oder „extrahieren Sie strukturierte Daten aus Text“.
