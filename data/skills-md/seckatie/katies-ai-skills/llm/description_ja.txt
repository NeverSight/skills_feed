Simon Willison の llm CLI ツールを使用して、コマンド ラインから大規模言語モデルにアクセスし、対話します。 OpenAI、Anthropic、Gemini、Llama、その他多数のモデルをプラグイン経由でサポートします。機能には、チャット セッション、埋め込み、スキーマを使用した構造化データ抽出、プロンプト テンプレート、会話ログ、ツールの使用が含まれます。このスキルは、ユーザーが「llm でプロンプトを実行する」、「llm コマンドを使用する」、「コマンドラインから LLM を呼び出す」、「llm API キーを設定する」、「llm プラグインをインストールする」、「埋め込みを作成する」、または「テキストから構造化データを抽出する」などの発言をしたときにトリガーされます。
