Доступ к большим языковым моделям и взаимодействие с ними из командной строки с помощью инструмента CLI llm Саймона Уиллисона. Поддерживает OpenAI, Anthropic, Gemini, Llama и десятки других моделей через плагины. Функции включают сеансы чата, внедрения, извлечение структурированных данных с помощью схем, шаблоны подсказок, ведение журнала разговоров и использование инструментов. Этот навык срабатывает, когда пользователь говорит такие вещи, как «запустить приглашение с помощью llm», «использовать команду llm», «вызвать LLM из командной строки», «настроить ключи API llm», «установить плагины llm», «создать встраивания» или «извлечь структурированные данные из текста».
