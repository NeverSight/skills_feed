Administre tickets de Jira desde la línea de comando usando jira-cli. Contiene instrucciones de configuración esenciales, patrones de comando no interactivos con indicadores requeridos (--plain, --raw, etc.), solución de problemas de autenticación y referencia completa de comandos. Esta habilidad se activa cuando el usuario dice cosas como "crear un ticket de Jira", "enumerar mis problemas de Jira", "actualizar el problema de Jira", "mover el ticket de Jira a listo", "registrar tiempo en Jira", "agregar comentario a Jira" o "buscar problemas de Jira". IMPORTANTE: lea esta habilidad antes de ejecutar cualquier comando de jira-cli para evitar el bloqueo en el modo interactivo.
