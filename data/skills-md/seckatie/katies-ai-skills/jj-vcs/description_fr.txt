Jujutsu (jj) est un puissant système de contrôle de version compatible Git avec des fonctionnalités innovantes telles que le rebasage automatique, la copie de travail en tant que validation, le journal des opérations avec annulation et le suivi des conflits de première classe. Cette compétence est déclenchée lorsque l'utilisateur dit des choses comme « utiliser jj », « exécuter les commandes jj », « contrôle de version jujutsu », « migrer de git vers jj », « jj rebase », « jj squash », « jj log » ou « aider avec le workflow jj ».
