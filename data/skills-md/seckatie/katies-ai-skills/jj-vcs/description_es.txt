Jujutsu (jj) es un potente sistema de control de versiones compatible con Git con funciones innovadoras como cambio de base automático, copia de trabajo como confirmación, registro de operaciones con deshacer y seguimiento de conflictos de primera clase. Esta habilidad se activa cuando el usuario dice cosas como "usar jj", "ejecutar comandos jj", "control de versiones jujutsu", "migrar de git a jj", "jj rebase", "jj squash", "jj log" o "ayuda con el flujo de trabajo jj".
