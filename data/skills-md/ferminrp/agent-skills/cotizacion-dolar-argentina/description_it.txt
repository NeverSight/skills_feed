Consulta le tariffe del dollaro in Argentina utilizzando DolarAPI. Usar cuando el usuario pida "dolar hoy", "dolar blue", "dolar oficial", "dolar bolsa", "MEP", "CCL", "dolar tarjeta", "dolar mayorista", "dolar cripto", o quiera comparar compra/venta y attualizacion de cotizaciones. Puoi anche usarlo quando l'utente ha una quotazione rapida di euro, real brasileno, peso cileno o peso uruguayo con endpoint di /v1/cotizaciones.
