Consulta cotizaciones de dólares en Argentina usando DolarAPI. Usar cuando el usuario pida "dolar hoy", "dolar blue", "dolar oficial", "dolar bolsa", "MEP", "CCL", "dolar tarjeta", "dolar mayorista", "dolar cripto", o quiera comparar compra/venta y actualización de cotizaciones. También usar cuando el usuario pide cotización de euro, real brasileño, peso chileno o peso uruguayo con endpoints de /v1/cotizaciones.
