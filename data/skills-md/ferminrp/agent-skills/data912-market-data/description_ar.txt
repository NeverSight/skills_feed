الاستعلام عن نقاط نهاية بيانات السوق Data912 لأدوات الأرجنتين والولايات المتحدة الأمريكية. يُستخدم عندما يطلب المستخدم عروض أسعار MEP/CCL، أو لوحات السوق الأرجنتينية المباشرة (الأسهم، أو الخيارات، أو الأوراق النقدية، أو السندات، أو ديون الشركات، أو السندات)، أو لوحات الولايات المتحدة الأمريكية (ADRs، الأسهم)، أو سلسلة OHLC التاريخية بواسطة شريط، أو سلاسل خيارات الولايات المتحدة الأمريكية، أو مقاييس التقلب/المخاطر. استخدمه أيضًا عندما يذكر المستخدم "Data912" أو "mep" أو "ccl" أو "cedears" أو "سلسلة الخيارات" أو "الأشرطة التاريخية" أو "OHLC" أو "التقلبات الضمنية" أو "التقلبات التاريخية" أو "النسب المئوية للتقلب" ويتوقع لقطات سوقية مدعومة بواجهة برمجة التطبيقات (API).
