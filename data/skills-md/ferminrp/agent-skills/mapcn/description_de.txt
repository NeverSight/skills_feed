Technischer Leitfaden zur Integration von Mapcn in React + Shadcn-Projekte. Wird verwendet, wenn der Benutzer Mapcn integrieren, Mapcn zu einer Shadcn-App hinzufügen, MapControls/Markers/Routes/Clusters von Mapcn verwenden, einen kontrollierten Viewport-Status implementieren, den useMap-Hook verwenden oder über MapLibre auf MapLibre zugreifen möchte.
