Guide technique pour intégrer mapcn dans les projets React + shadcn. À utiliser lorsque l'utilisateur demande à intégrer mapcn, à ajouter mapcn à une application shadcn, à utiliser mapcn MapControls/Markers/Routes/Clusters, à implémenter un état de fenêtre contrôlé, à utiliser le hook useMap ou à accéder à MapLibre via mapcn.
