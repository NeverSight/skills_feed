Техническое руководство по интеграции Mapcn в проекты React + Shadcn. Используйте, когда пользователь запрашивает интеграцию Mapcn, добавляет Mapcn в приложение Shadcn, использует MapCn MapControls/Markers/Routes/Clusters, реализует контролируемое состояние области просмотра, использует перехватчик useMap или получает доступ к MapLibre через Mapcn.
