Consulta la API Central de Deudores del BCRA (Banco Central de la República Argentina) para consultar el estado crediticio de personas o empresas del sistema financiero argentino. Se utiliza cuando el usuario solicita verificar la situación de deuda, informe crediticio, situación financiera, cheques rechazados o historial crediticio de alguien utilizando un número CUIT/CUIL/CDI. También se utiliza cuando el usuario menciona "central de deudores", "situación crediticia", "deudas BCRA", "cheques rechazados", "historial crediticio", "informe crediticio", o quiere saber si una persona o empresa tiene deudas reportadas en el sistema financiero argentino.
