Интегрирует конечные точки API Dub Links для создания, обновления, удаления, извлечения, составления списка, подсчета и выполнения массовых операций с короткими ссылками. Используйте, когда пользователь запрашивает «дублировать ссылки API», «создать дублирование ссылки», «вставить дублирование ссылки», «перечислить ссылки», «подсчитать ссылки», «массовые ссылки» или выполнить поиск по linkId/domain+key/externalId.
