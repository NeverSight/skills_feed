Intègre les points de terminaison de l'API Dub Links pour créer, mettre à jour, supprimer, récupérer, répertorier, compter et exécuter des opérations en masse sur des liens courts. À utiliser lorsque l'utilisateur demande « API de liens dub », « créer un dub de lien », « insérer un lien dub », « liste de liens », « compter les liens », « liens en masse » ou des recherches par linkId/domain+key/externalId.
