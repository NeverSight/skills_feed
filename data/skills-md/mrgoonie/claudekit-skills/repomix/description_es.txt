Empaquete repositorios de código completos en archivos únicos compatibles con IA usando Repomix. Las capacidades incluyen paquetes de bases de código con patrones de inclusión/exclusión personalizables, generar múltiples formatos de salida (XML, Markdown, texto sin formato), preservar la estructura y el contexto de los archivos, optimizar el consumo de IA con recuento de tokens, filtrar por tipos de archivos y directorios, agregar encabezados y resúmenes personalizados. Úselo al empaquetar bases de código para análisis de IA, crear instantáneas de repositorio para el contexto de LLM, analizar bibliotecas de terceros, prepararse para auditorías de seguridad, generar contexto de documentación o evaluar bases de código desconocidas.
