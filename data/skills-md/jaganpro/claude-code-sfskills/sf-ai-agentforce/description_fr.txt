Compétence standard de la plateforme Agentforce. Couvre la création d'agents basés sur l'interface utilisateur, la configuration de sujets/actions, les métadonnées PromptTemplate, l'API des modèles Einstein, la configuration GenAiFunction/GenAiPlugin et les types Lightning personnalisés. Pour le développement Agent Script DSL, utilisez plutôt sf-ai-agentscript.
