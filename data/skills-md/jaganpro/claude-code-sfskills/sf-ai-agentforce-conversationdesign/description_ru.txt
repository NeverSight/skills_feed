Навыки проектирования диалогов для Salesforce Agentforce. Создает персональные документы, архитектуры тем, наборы инструкций, библиотеки высказываний, матрицы эскалации и конфигурации ограждений. Проверяет существующих агентов на предмет соответствия лучшим практикам построения диалогов с помощью 120-балльной оценки.
