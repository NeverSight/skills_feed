Стандартные навыки работы с платформой Agentforce. Охватывает создание агентов на основе пользовательского интерфейса, настройку тем/действий, метаданные PromptTemplate, API моделей Einstein, настройку GenAiFunction/GenAiPlugin и пользовательские типы Lightning. Для разработки DSL на основе агентских сценариев используйте вместо этого sf-ai-agentscript.
