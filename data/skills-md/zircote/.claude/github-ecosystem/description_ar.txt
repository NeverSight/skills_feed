أنشئ تكوينًا شاملاً للنظام البيئي لـ GitHub، بما في ذلك سير عمل CI/CD، وقوالب المشكلات، وقوالب العلاقات العامة، وCODEOWNERS، وdependabot، وتعليمات Copilot. لا يعرف اللغة مع دعم Python وGo وTypeScript. يكتشف تلقائيًا لغة المشروع من pyproject.toml أو go.mod أو package.json. يُستخدم عند إعداد مستودعات GitHub، أو إضافة خطوط أنابيب CI/CD، أو إنشاء قوالب المشكلات/العلاقات العامة، أو تكوين التابع، أو إضافة CODEOWNERS، أو إعداد تعليمات Copilot، أو تحسين المشاريع الحالية باستخدام أدوات GitHub. يعمل على المستودعات الجديدة أو الحالية من خلال الدمج الذكي.
