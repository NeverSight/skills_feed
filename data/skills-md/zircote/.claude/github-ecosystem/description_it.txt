Genera una configurazione completa dell'ecosistema GitHub, inclusi flussi di lavoro CI/CD, modelli di problemi, modelli PR, CODEOWNERS, dependabot e istruzioni Copilot. Indipendente dal linguaggio con supporto Python, Go e TypeScript. Rileva automaticamente la lingua del progetto da pyproject.toml, go.mod o package.json. Da utilizzare durante la configurazione di repository GitHub, l'aggiunta di pipeline CI/CD, la creazione di modelli di problemi/PR, la configurazione di dependabot, l'aggiunta di CODEOWNERS, l'impostazione di istruzioni Copilot o il miglioramento di progetti esistenti con gli strumenti GitHub. Funziona su repository nuovi o esistenti con fusione intelligente.
