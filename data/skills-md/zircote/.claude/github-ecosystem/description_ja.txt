CI/CD ワークフロー、課題テンプレート、PR テンプレート、CODEOWNERS、dependabot、Copilot 命令などの包括的な GitHub エコシステム構成を生成します。言語に依存せず、Python、Go、TypeScript をサポートします。 pyproject.toml、go.mod、または package.json からプロジェクト言語を自動検出します。 GitHub リポジトリの設定、CI/CD パイプラインの追加、課題/PR テンプレートの作成、dependabot の構成、CODEOWNERS の追加、Copilot 命令の設定、または GitHub ツールを使用した既存のプロジェクトの強化を行うときに使用します。スマート マージにより、新規または既存のリポジトリで動作します。
