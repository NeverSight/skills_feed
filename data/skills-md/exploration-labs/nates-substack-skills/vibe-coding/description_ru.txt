Подробное руководство по кодированию вибрации с помощью искусственного интеллекта. Используйте, когда пользователь хочет создавать приложения с помощью подсказок на естественном языке, используя такие инструменты, как Lovable, Cursor, Replit или Bolt. Включает в себя лучшие практики, понимание ошибок, рекомендации по конкретным инструментам, поддержку архитектурных решений и определение объема MVP с уклоном в сторону агрессивного сокращения функций для ускорения выпуска.
