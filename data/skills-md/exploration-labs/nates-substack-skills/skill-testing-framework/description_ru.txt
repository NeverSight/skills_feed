Предоставляет тестовые примеры и инструменты проверки навыков. Используйте при создании тестов для нового навыка, добавлении регрессионных тестов после обновления навыков, запуске наборов тестов для проверки функциональности навыка или проверке соответствия выходных данных навыка ожидаемым результатам. Поддерживает модульные тесты, интеграционные тесты и регрессионные тесты с проверкой пары входных/выходных данных.
