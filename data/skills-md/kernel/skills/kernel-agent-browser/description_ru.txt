Рекомендации по использованию агента-браузера с облачными браузерами Kernel. Используйте при автоматизации веб-сайтов с ядром агента-браузера -p, при работе с обнаружением ботов, iframe, постоянством входа в систему или при необходимости найти идентификаторы сеансов браузера ядра и URL-адреса просмотра в реальном времени.
