Diese Fähigkeit generiert detaillierte virtuelle Charakterprofile basierend auf einer standardisierten Vorlage. Verwenden Sie diese Fähigkeit, wenn ein Benutzer einen Charakter für Geschichten, Rollenspiele oder virtuelle Assistenten erstellen oder verfeinern möchte. Es führt den Benutzer durch einen klärenden Dialog, um die Kernpersönlichkeit, das Verhalten und den Hintergrund zu ermitteln, bevor das endgültige Profil erstellt wird.
