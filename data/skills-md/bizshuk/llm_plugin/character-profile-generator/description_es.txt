Esta habilidad genera perfiles de personajes virtuales detallados basados ​​en una plantilla estandarizada. Utilice esta habilidad cuando un usuario quiera crear o perfeccionar un personaje para historias, juegos de rol o asistentes virtuales. Guía al usuario a través de un diálogo aclaratorio para establecer la personalidad, el comportamiento y los antecedentes básicos antes de generar el perfil final.
