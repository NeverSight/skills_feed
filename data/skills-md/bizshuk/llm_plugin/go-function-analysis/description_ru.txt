Анализируйте длину функций Go в рабочей области и генерируйте статистику (p50, p90, p99). Используйте, когда вам нужно проверить сложность функции, определить длинные функции или сгенерировать метрики кода для проектов Go. Анализирует только файлы, соответствующие **/*.go, в текущей рабочей области, исключая зависимости.
