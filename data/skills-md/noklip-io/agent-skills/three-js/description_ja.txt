このスキルは、ユーザーが「3D シーンの作成」、「メッシュの追加」、
"OrbitControls を実装する"、"GLTF モデルをロードする"、"ブルーム後処理を追加する"、
「カスタム シェーダを作成する」、「パーティクル エフェクトを作成する」、「Three.js のパフォーマンスを最適化する」、
「WebGPU を使用する」、「影を追加する」、「モデルをアニメーション化する」、または Three.js、threejs、WebGL、
WebGPU、GLTF、レイキャスター、シェーダー マテリアル、PBR マテリアル、または後処理エフェクト。

重要: このスキルは VANILLA Three.js (命令型 JavaScript) 用です。リアクト 3 の場合
Fiber (@react-three/fiber、R3F、drei) は、「r3f-best-practices」スキルを確認してください。ただし、R3F は Three.js の React レンダラーであるため、three-js スキルは R3F を操作するときに役に立ちます。

シーンのセットアップを含む 3D Web グラフィックスの完全な Three.js リファレンスを提供します。

インタラクション、シェーダー、後処理、パフォーマンスの最適化、TSL/ノードマテリアル、
