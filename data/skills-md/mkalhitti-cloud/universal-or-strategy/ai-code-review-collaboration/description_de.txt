Framework zum Austauschen von Codeüberprüfungen zwischen mehreren KI-Modellen (Claude, Gemini, DeepSeek, GPT usw.), um eine umfassende Analyse zu erhalten. Verwenden Sie diese Option, wenn der Benutzer mehrere KI-Perspektiven auf Code, Architekturentscheidungen oder technische Überprüfungen wünscht. Auslöser für „Andere KI-Meinungen einholen“, „Überprüfung mit mehreren KIs“, „Ideen von anderen KIs abgeben“, „Multi-AI-Überprüfung“ oder „Gemeinsame KI-Überprüfung“.
