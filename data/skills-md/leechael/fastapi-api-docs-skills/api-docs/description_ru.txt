Анализируйте файлы маршрутов FastAPI и полные метаданные документации API.
Отслеживает весь стек вызовов, включая Depends, отслеживает все возможные ошибки,
и генерирует комплексные элементы документации OpenAPI.
Используйте, когда: пользователь предоставляет путь к файлу с синтаксисом @filename и хочет, чтобы документация API была завершена.
