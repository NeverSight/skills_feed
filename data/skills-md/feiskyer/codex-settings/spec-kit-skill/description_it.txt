Integrazione di GitHub Spec-Kit per uno sviluppo basato sulle specifiche basato sulla costituzione. Flusso di lavoro in 7 fasi (costituzione, specificare, chiarire, pianificare, compiti, analizzare, implementare). Da utilizzare quando si lavora con la CLI spec-kit, le directory .specify/ o quando si creano specifiche con uno sviluppo basato sulla costituzione. Attivato da "spec-kit", "specit", "constitution", "specify", riferimenti alla directory .specify/ o comandi spec-kit.
