تكامل GitHub Spec-Kit للتطوير القائم على المواصفات. سير العمل مكون من 7 مراحل (الدستور، التحديد، التوضيح، التخطيط، المهام، التحليل، التنفيذ). يُستخدم عند العمل مع مجموعة المواصفات CLI، أو .specify/ الدلائل، أو إنشاء مواصفات مع التطوير المستند إلى الدستور. يتم تشغيله بواسطة "spec-kit"، أو "speckit"، أو "constitution"، أو "specify"، أو الإشارات إلى دليل .specify/، أو أوامر spec-kit.
