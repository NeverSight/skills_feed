À utiliser lorsque l'utilisateur souhaite exécuter des tâches de longue durée qui nécessitent plusieurs sessions. Cette compétence gère la décomposition des tâches, le suivi de la progression et l'exécution autonome à l'aide du mode sans tête Claude Code avec continuation automatique.
