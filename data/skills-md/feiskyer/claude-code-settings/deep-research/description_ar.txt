سير عمل التنسيق متعدد الوكلاء للأبحاث العميقة: تقسيم هدف البحث إلى أهداف فرعية متوازية، وتشغيل كل منها في عملية فرعية غير تفاعلية لـ Claude Code (`claude -p`)؛ تفضيل المهارات المثبتة لجمع الويب والبيانات، والرجوع إلى أدوات MCP؛ تجميع النتائج الفرعية من خلال البرنامج النصي، وصقل الفصل، وتقديم "مسار ملف التقرير النهائي + ملخص النتائج/التوصيات الرئيسية". يُستخدم من أجل: البحث المنهجي على الويب/البيانات، أو تحليل المنافسين/الصناعة، أو استرجاع أجزاء مجموعة الارتباط/مجموعة البيانات، أو الكتابة الطويلة مع تكامل الأدلة، أو عندما يقول المستخدم "بحث عميق / بحث واسع / بحث متوازي متعدد الوكلاء / بحث متعدد العمليات".
