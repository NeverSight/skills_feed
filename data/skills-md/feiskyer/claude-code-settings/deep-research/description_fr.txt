Flux de travail d'orchestration multi-agents de recherche approfondie : divisez un objectif de recherche en sous-objectifs parallèles, exécutez chacun dans un sous-processus Claude Code non interactif (`claude -p`) ; privilégier les compétences installées pour le web et la collecte de données, repli sur les outils MCP ; agréger les sous-résultats via un script, peaufiner les chapitres et fournir « le chemin du fichier du rapport final + le résumé des principales conclusions/recommandations ». À utiliser pour : la recherche systématique sur le Web/les données, l'analyse des concurrents/de l'industrie, la récupération de fragments de liens par lots/d'ensembles de données, l'écriture longue avec intégration de preuves, ou lorsque l'utilisateur dit « recherche approfondie / recherche étendue / recherche parallèle multi-agents / recherche multi-processus ».
