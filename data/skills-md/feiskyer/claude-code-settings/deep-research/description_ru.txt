Рабочий процесс многоагентной оркестровки глубоких исследований: разбейте цель исследования на параллельные подцели, запустите каждую в неинтерактивном подпроцессе Claude Code (`claude -p`); предпочитают уже имеющиеся навыки работы в Интернете и сбора данных, откат к инструментам MCP; агрегировать промежуточные результаты с помощью сценария, дорабатывать по главам и предоставлять «путь к файлу окончательного отчета + сводку основных выводов/рекомендаций». Используется для: систематического исследования сети/данных, анализа конкурентов/отрасли, пакетного извлечения ссылок/осколков набора данных, написания длинных форм с интеграцией доказательств или когда пользователь говорит «глубокое исследование / широкое исследование / многоагентное параллельное исследование / многопроцессное исследование».
