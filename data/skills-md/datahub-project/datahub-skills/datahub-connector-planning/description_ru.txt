Используйте этот навык, когда пользователь хочет спланировать новый соединитель DataHub, изучить исходную систему для разработки соединителя, создать документ планирования соединителя или спроектировать архитектуру соединителя. Срабатывает при: «запланировать соединитель», «новый соединитель для X», «исследовать X для DataHub», «спроектировать соединитель для X», «создать документ планирования» или любой запрос на планирование, исследование или проектирование источника приема DataHub.
