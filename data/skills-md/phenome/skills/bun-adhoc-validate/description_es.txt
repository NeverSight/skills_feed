Ejecute validaciones ad hoc rápidas o comprobaciones de prueba usando Bun escribiendo un script TypeScript temporal en el repositorio que importa código local, iterando el script y ejecutándolo nuevamente hasta que se valide (incluida la confirmación de una falla) y luego eliminando el script. Úselo cuando se necesita una validación inmediata para confirmar suposiciones, inspeccionar estados remotos o crear prototipos de comprobaciones pequeñas sin agregar archivos o pruebas permanentes.
