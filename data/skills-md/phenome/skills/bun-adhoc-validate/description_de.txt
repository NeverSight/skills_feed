Führen Sie schnelle Ad-hoc-Validierungen oder Testprüfungen mit Bun durch, indem Sie ein temporäres TypeScript-Skript im Repo schreiben, das lokalen Code importiert, das Skript iteriert und bis zur Validierung erneut ausführt (einschließlich der Bestätigung eines Fehlers) und dann das Skript löscht. Verwenden Sie es, wenn eine sofortige Validierung erforderlich ist, um Annahmen zu bestätigen, den Remote-Status zu überprüfen oder kleine Prototypenprüfungen durchzuführen, ohne permanente Dateien oder Tests hinzuzufügen.
