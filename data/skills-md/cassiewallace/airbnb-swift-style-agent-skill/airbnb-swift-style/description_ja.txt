Airbnb iOS Swift スタイルガイドを Swift コードに適用します。このスキルは、ユーザーが希望するときにいつでも使用できます: Swift コードのスタイル準拠のレビュー、iOS コードベースの lint または監査、新規または既存のコードへの Airbnb Swift 規約の適用、コードが Airbnb スタイル ルールに違反する理由の理解、または Swift スニペットの修正されたリライトの取得。 Swift コードの品質、iOS コード レビュー、Airbnb スタイル、Swift ファイルのリンティングに言及するリクエスト、またはユーザーが Swift コードを貼り付けてフィードバック、修正、レビューを求めたときにトリガーされます。ユーザーが Swift コードを「クリーンアップ」または「フォーマット」するように要求する場合、または「これは Swift で良いですか?」と尋ねる場合にも使用します。 —たとえAirbnbの名前を出さなかったとしても。
