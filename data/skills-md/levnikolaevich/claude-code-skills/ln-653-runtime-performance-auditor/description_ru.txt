Работник аудита производительности во время выполнения (L3). Проверяет блокировку ввода-вывода в асинхронном режиме, ненужные выделения, синхронный сон в асинхронном режиме, конкат строк в циклах, отсутствие to_thread для привязанных к ЦП, избыточных копий данных. Возвращает результаты с указанием серьезности, местоположения, усилий и рекомендаций.
