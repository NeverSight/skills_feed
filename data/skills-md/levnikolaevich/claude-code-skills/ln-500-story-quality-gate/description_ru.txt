Органайзер качества на уровне истории с 4-уровневыми воротами (ПРОГЛАСЕН/ПРОБЛЕМЫ/НЕ ПРОЙДЕН/ОТКАЗАН) и показателем качества. Шаг 1: качество кода -> регрессия -> ручное тестирование. Шаг 2: проверьте тесты/покрытие -> подсчитайте баллы NFR -> отметьте «История завершена». Используйте, когда пользователь запрашивает контроль качества для Story или когда ln-400 делегирует проверку качества.
