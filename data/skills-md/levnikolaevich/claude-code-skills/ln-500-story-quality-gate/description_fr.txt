Orchestrateur de qualité au niveau de l'histoire avec Gate à 4 niveaux (PASS/CONCERNS/FAIL/WAIVED) et Quality Score. Passe 1 : qualité du code -> régression -> tests manuels. Passe 2 : vérifier les tests/couverture -> calculer les scores NFR -> marquer l'histoire terminée. À utiliser lorsque l'utilisateur demande une porte de qualité pour Story ou lorsque ln-400 délègue un contrôle de qualité.
