Orquestador de calidad a nivel de historia con puerta de 4 niveles (APROBADO/INQUIERTAS/FALLO/RENUNCADO) y Nivel de calidad. Paso 1: calidad del código -> regresión -> prueba manual. Pase 2: verificar pruebas/cobertura -> calcular puntajes NFR -> marcar Historia como Terminada. Úselo cuando el usuario solicita un control de calidad para Story o cuando el ln-400 delega el control de calidad.
