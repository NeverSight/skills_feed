Специалист по аудиту качества кода (L3). Проверяет цикломатическую сложность, глубокую вложенность, длинные методы, классы богов, алгоритмы O(n²), запросы N+1, магические числа, децентрализованные константы, повторяющиеся константы. Возвращает результаты с указанием серьезности, местоположения, усилий и рекомендаций.
