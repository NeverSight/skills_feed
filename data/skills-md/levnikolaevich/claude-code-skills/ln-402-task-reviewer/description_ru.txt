Л3 Рабочий. Проверяет реализацию задач на качество, стандарты кода, тестовое покрытие. Создает задачи [BUG] для проблем с побочными эффектами, обнаруженных за пределами задачи. Устанавливает задачу «Выполнено» или «Переработать». Обычно вызывается ln-400 с изолированным контекстом, а также может просматривать конкретную задачу по запросу пользователя.
