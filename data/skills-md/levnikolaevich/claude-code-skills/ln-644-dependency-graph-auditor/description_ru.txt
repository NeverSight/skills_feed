Л3 Рабочий. Строит граф зависимостей модулей, обнаруживает транзитивные циклы (DFS), проверяет граничные правила (запрещено/разрешено/обязательно), рассчитывает метрики связи (Ca/Ce/I, CCD/NCCD). Обнаружение адаптивной архитектуры: пользовательские правила > документы > автоматическое обнаружение. Поддерживает гибридные архитектуры.
