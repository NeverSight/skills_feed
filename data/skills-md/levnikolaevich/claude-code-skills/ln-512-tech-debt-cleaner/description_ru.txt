Автоматизированный специалист по очистке технического долга (L3). Считывает результаты аудита кодовой базы, применяет безопасные автоматические исправления для проблем с низким уровнем риска (неиспользуемый импорт, мертвый код, закомментированный код, устаревшие псевдонимы). Уверенность >=90%. Создает одиночный коммит со сводкой.
