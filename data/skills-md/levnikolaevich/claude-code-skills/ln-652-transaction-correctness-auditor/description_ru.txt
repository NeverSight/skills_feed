Работник проверки правильности транзакций (L3). Проверяет отсутствие промежуточных коммитов, область транзакций (слишком широкая/узкая), отсутствие обработки отката, длительные транзакции, взаимодействие триггера/уведомления. Возвращает результаты с указанием серьезности, местоположения, усилий и рекомендаций.
