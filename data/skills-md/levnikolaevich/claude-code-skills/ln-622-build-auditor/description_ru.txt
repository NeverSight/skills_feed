Создайте работника проверки работоспособности (L3). Проверяет ошибки компилятора/линтера, предупреждения об устаревании, ошибки типа, неудачные тесты, проблемы с конфигурацией сборки. Возвращает результаты с указанием серьезности (критическая/высокая/средняя/низкая), местоположения, усилий и рекомендаций.
