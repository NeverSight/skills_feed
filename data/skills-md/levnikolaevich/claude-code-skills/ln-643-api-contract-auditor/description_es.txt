Trabajador de auditoría por contrato API (N3). Comprueba la fuga de capas en las firmas de métodos, los DTO faltantes, la fuga de entidades a la API, los contratos de error inconsistentes y las sobrecargas de métodos redundantes. Devuelve resultados con un modelo de 4 puntuaciones (cumplimiento, integridad, calidad, implementación).
