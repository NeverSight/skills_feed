Аудит архитектурных шаблонов на предмет соответствия лучшим практикам (MCP Ref, Context7, WebSearch). Поддерживает каталог шаблонов, вычисляет 4 балла, создает истории рефакторинга с помощью ln-220. Используйте, когда пользователь просит: (1) проверить работоспособность архитектуры, (2) проверить шаблоны перед рефакторингом, (3) найти недокументированные шаблоны в базе кода.
