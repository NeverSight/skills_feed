Utilisez la CLI Memberstack pour gérer les comptes Memberstack à partir du terminal. Couvre l'authentification, les applications, les membres, les forfaits, les prix, les champs personnalisés, les tables de données, les enregistrements, les autorisations, les fournisseurs d'authentification, le SSO et les utilisateurs. Déclenchez cette compétence chaque fois que l'utilisateur souhaite interagir avec Memberstack, notamment en gérant les membres, les forfaits, les prix, les champs personnalisés, les tables/enregistrements de données, les autorisations, les fournisseurs d'authentification, les applications SSO ou en s'authentifiant auprès de Memberstack. Se déclenche également lorsque l'utilisateur mentionne "memberstack", "memberstack-cli", la gestion des membres ou les opérations sur les données des membres via CLI.
