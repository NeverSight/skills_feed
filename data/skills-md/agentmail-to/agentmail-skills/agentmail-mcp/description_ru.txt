Сервер AgentMail MCP для инструментов электронной почты в помощниках AI. Используйте при настройке AgentMail с Claude Desktop, Cursor, VS Code, Windsurf или другими MCP-совместимыми клиентами. Предоставляет инструменты для управления входящими сообщениями, отправки/получения электронных писем и обработки потоков.
