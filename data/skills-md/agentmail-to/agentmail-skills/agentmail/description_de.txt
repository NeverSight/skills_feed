Geben Sie KI-Agenten mithilfe der AgentMail-API eigene E-Mail-Postfächer. Verwenden Sie sie beim Erstellen von E-Mail-Agenten, beim programmgesteuerten Senden/Empfangen von E-Mails, beim Verwalten von Posteingängen, beim Bearbeiten von Anhängen, beim Organisieren mit Labels, beim Erstellen von Entwürfen zur menschlichen Genehmigung oder beim Einrichten von Echtzeitbenachrichtigungen über Webhooks/Websockets. Unterstützt die mandantenfähige Isolation mit Pods.
