Fähigkeit zur Browserautomatisierung für UI-Tests über Chrome MCP-Tools. Verwenden Sie, wenn:
(1) Der QA-Agent muss die Benutzeroberfläche visuell überprüfen oder Interaktionen testen.
(2) UI/UX-Designer müssen responsive Design- oder Komponentenzustände überprüfen,
(3) Frontend Dev benötigt während der Entwicklung eine schnelle visuelle Überprüfung.
(4) Der Testautor muss Benutzerabläufe mit Screenshots/GIFs dokumentieren.
(5) Jeder Agent muss Webschnittstellen testen, Demos aufzeichnen oder UI-Probleme beheben.
Funktionen: Screenshots, Interaktionstests, Barrierefreiheitsprüfungen, GIF-Aufzeichnung,
Responsive Tests, Konsolen-/Netzwerk-Debugging.
