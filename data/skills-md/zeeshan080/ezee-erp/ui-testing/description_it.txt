Competenza di automazione del browser per test dell'interfaccia utente tramite strumenti MCP di Chrome. Utilizzare quando:
(1) L'agente QA deve verificare visivamente l'interfaccia utente o testare le interazioni,
(2) UI/UX Designer deve verificare il design reattivo o gli stati dei componenti,
(3) Frontend Dev necessita di una rapida verifica visiva durante lo sviluppo,
(4) Test Writer deve documentare i flussi utente con screenshot/GIF,
(5) Qualsiasi agente deve testare le interfacce web, registrare demo o eseguire il debug dei problemi dell'interfaccia utente.
Funzionalità: screenshot, test di interazione, controlli di accessibilità, registrazione GIF,
test reattivi, debug di console/rete.
