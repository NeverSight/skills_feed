Навыки автоматизации браузера для тестирования пользовательского интерфейса с помощью инструментов Chrome MCP. Используйте, когда:
(1) Агенту по обеспечению качества необходимо визуально проверить пользовательский интерфейс или протестировать взаимодействие,
(2) Дизайнеру пользовательского интерфейса/UX необходимо проверить адаптивный дизайн или состояние компонентов,
(3) Frontend Dev нуждается в быстрой визуальной проверке во время разработки,
(4) Автору тестов необходимо документировать потоки пользователей с помощью снимков экрана/GIF,
(5) Любому агенту необходимо тестировать веб-интерфейсы, записывать демонстрации или устранять проблемы пользовательского интерфейса.
Возможности: скриншоты, тестирование взаимодействия, проверки доступности, запись GIF,
адаптивное тестирование, отладка консоли/сети.
