Habilidad de automatización del navegador para pruebas de UI a través de herramientas Chrome MCP. Úselo cuando:
(1) El agente de control de calidad necesita verificar la interfaz de usuario visualmente o probar las interacciones,
(2) UI/UX Designer necesita verificar el diseño responsivo o los estados de los componentes,
(3) Frontend Dev necesita una verificación visual rápida durante el desarrollo,
(4) Test Writer necesita documentar los flujos de usuarios con capturas de pantalla/GIF,
(5) Cualquier agente necesita probar interfaces web, grabar demostraciones o depurar problemas de UI.
Capacidades: capturas de pantalla, pruebas de interacción, comprobaciones de accesibilidad, grabación de GIF,
pruebas de respuesta, depuración de consola/red.
