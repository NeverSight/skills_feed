Chrome MCP 도구를 통한 UI 테스트를 위한 브라우저 자동화 기술입니다. 다음과 같은 경우에 사용하세요:
(1) QA 상담원은 UI를 시각적으로 확인하거나 상호작용을 테스트해야 하며,
(2) UI/UX 디자이너는 반응형 디자인이나 컴포넌트 상태를 확인해야 하며,
(3) Frontend Dev는 개발 중에 빠른 시각적 검증이 필요하며,
(4) 테스트 작성자는 스크린샷/GIF로 사용자 흐름을 문서화해야 합니다.
(5) 모든 상담원은 웹 인터페이스를 테스트하고, 데모를 기록하거나, UI 문제를 디버깅해야 합니다.
기능: 스크린샷, 상호 작용 테스트, 접근성 확인, GIF 녹화,
반응형 테스트, 콘솔/네트워크 디버깅.
