Compétence en automatisation du navigateur pour les tests d'interface utilisateur via les outils Chrome MCP. À utiliser lorsque :
(1) L'agent d'assurance qualité doit vérifier visuellement l'interface utilisateur ou tester les interactions,
(2) UI/UX Designer doit vérifier la conception réactive ou les états des composants,
(3) Frontend Dev nécessite une vérification visuelle rapide pendant le développement,
(4) Test Writer doit documenter les flux d'utilisateurs avec des captures d'écran/GIF,
(5) Tout agent doit tester les interfaces Web, enregistrer des démos ou déboguer les problèmes d'interface utilisateur.
Capacités : captures d'écran, tests d'interaction, contrôles d'accessibilité, enregistrement GIF,
tests réactifs, débogage console/réseau.
