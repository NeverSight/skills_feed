Chrome MCP ツールを介した UI テストのためのブラウザ自動化スキル。次の場合に使用します。
(1) QA エージェントは UI を視覚的に検証するか、インタラクションをテストする必要があります。
(2) UI/UX デザイナーはレスポンシブ デザインまたはコンポーネントの状態を確認する必要があります。
(3) フロントエンド開発者は開発中に迅速な視覚的検証を必要とします。
(4) テスト ライターはスクリーンショット/GIF を使用してユーザー フローを文書化する必要があります。
(5) すべてのエージェントは、Web インターフェイスのテスト、デモの記録、または UI の問題のデバッグを行う必要があります。
機能: スクリーンショット、インタラクション テスト、アクセシビリティ チェック、GIF 記録、
応答性テスト、コンソール/ネットワークのデバッグ。
