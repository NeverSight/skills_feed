Automazione completa di Chrome DevTools per test delle prestazioni, misurazione dei Core Web Vitals (INP, LCP, CLS), monitoraggio della rete, convalida dell'accessibilità, test reattivi e automazione del browser. Utilizza il protocollo Chrome DevTools tramite MCP per fornire funzionalità di test, debug e analisi delle applicazioni Web di livello professionale, tra cui tracciamento delle prestazioni, esportazione HAR, emulazione di dispositivi e flussi di lavoro multipagina.
