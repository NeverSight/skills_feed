Комплексная автоматизация Chrome DevTools для тестирования производительности, измерения основных веб-показателей (INP, LCP, CLS), мониторинга сети, проверки доступности, адаптивного тестирования и автоматизации браузера. Использует протокол Chrome DevTools через MCP для обеспечения профессионального уровня тестирования, отладки и анализа веб-приложений, включая отслеживание производительности, экспорт HAR, эмуляцию устройств и многостраничные рабочие процессы.
