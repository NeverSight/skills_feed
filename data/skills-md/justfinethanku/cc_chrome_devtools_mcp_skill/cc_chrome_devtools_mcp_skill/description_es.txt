Automatización integral de Chrome DevTools para pruebas de rendimiento, medición de Core Web Vitals (INP, LCP, CLS), monitoreo de red, validación de accesibilidad, pruebas de respuesta y automatización del navegador. Utiliza el protocolo Chrome DevTools a través de MCP para proporcionar capacidades de prueba, depuración y análisis de aplicaciones web de nivel profesional, incluido el seguimiento del rendimiento, la exportación HAR, la emulación de dispositivos y los flujos de trabajo de varias páginas.
