Configurar y utilizar capacidades de texto a SQL para bases de datos SQL (SQLite, PostgreSQL, MySQL, MariaDB, etc.).
Úselo cuando: (1) El usuario desee consultar bases de datos utilizando lenguaje natural,
(2) El usuario solicita configurar el proyecto de texto a SQL, (3) El usuario menciona la extracción
datos de la base de datos, (4) el usuario tiene un archivo .sqlite/.db o credenciales de base de datos y
quiere trabajar con datos. Esta habilidad establece la estructura del proyecto, extrae
esquema de base de datos y permite la conversión de lenguaje natural a SQL.
