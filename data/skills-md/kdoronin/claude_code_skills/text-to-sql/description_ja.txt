SQL データベース (SQLite、PostgreSQL、MySQL、MariaDB など) の text-to-SQL 機能をセットアップして使用します。
次の場合に使用します: (1) ユーザーが自然言語を使用してデータベースにクエリを実行したい場合
(2) ユーザーが text-to-sql プロジェクトのセットアップを要求する、(3) ユーザーが抽出について言及する
データベースからのデータ、(4) ユーザーが .sqlite/.db ファイルまたはデータベースの資格情報を持っている、および
データを操作したいと考えています。このスキルはプロジェクト構造を設定し、抽出します。
データベース スキーマをサポートし、自然言語から SQL への変換を可能にします。
