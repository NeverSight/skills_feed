Настройка и использование возможностей преобразования текста в SQL для баз данных SQL (SQLite, PostgreSQL, MySQL, MariaDB и т. д.).
Используйте, когда: (1) Пользователь хочет выполнять запросы к базам данных, используя естественный язык;
(2) Пользователь запрашивает настройку проекта преобразования текста в SQL, (3) Пользователь упоминает извлечение
данные из базы данных, (4) Пользователь имеет файл .sqlite/.db или учетные данные базы данных и
хочет работать с данными. Этот навык устанавливает структуру проекта, извлекает
схему базы данных и обеспечивает преобразование естественного языка в SQL.
