SQL 데이터베이스(SQLite, PostgreSQL, MySQL, MariaDB 등)에 대한 텍스트-SQL 기능을 설정하고 사용합니다.
다음 경우에 사용하십시오: (1) 사용자가 자연어를 사용하여 데이터베이스를 쿼리하려고 합니다.
(2) 사용자가 text-to-sql 프로젝트 설정을 요청합니다. (3) 사용자가 추출을 언급합니다.
(4) 사용자가 .sqlite/.db 파일 또는 데이터베이스 자격 증명을 가지고 있으며
데이터 작업을 원합니다. 이 스킬은 프로젝트 구조를 설정하고,
데이터베이스 스키마를 사용하여 자연어를 SQL로 변환할 수 있습니다.
