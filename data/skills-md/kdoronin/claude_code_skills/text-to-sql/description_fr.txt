Configurer et utiliser les fonctionnalités texte vers SQL pour les bases de données SQL (SQLite, PostgreSQL, MySQL, MariaDB, etc.).
À utiliser lorsque : (1) L'utilisateur souhaite interroger des bases de données en utilisant le langage naturel,
(2) L'utilisateur demande de configurer le projet text-to-sql, (3) L'utilisateur mentionne l'extraction
données de la base de données, (4) l'utilisateur dispose des informations d'identification du fichier .sqlite/.db ou de la base de données et
veut travailler avec des données. Cette compétence met en place la structure du projet, extrait
schéma de base de données et permet la conversion du langage naturel vers SQL.
