قم بمراجعة أكواد TypeScript وReact وNext.js وفقًا للمعايير العالية للوضوح والبساطة وقابلية الصيانة. استخدم هذه المهارة عندما: (1) تمت كتابة التعليمات البرمجية أو تعديلها بواسطة Claude أو وكيل فرعي، (2) يطلب المستخدم صراحةً مراجعة التعليمات البرمجية، (3) اكتمال إعادة البناء، (4) تم تنفيذ ميزات أو مكونات جديدة. توفر هذه المهارة تعليقات صادقة جدًا ولكنها داعمة وفقًا لفلسفة كارلوس - الاحتفال بالعمل الجيد مع التراجع بشدة عن التعقيد أو الأنماط غير الواضحة أو انتهاكات أفضل الممارسات.
