Revise el código de TypeScript, React y Next.js con estándares elevados para mayor claridad, simplicidad y facilidad de mantenimiento. Utilice esta habilidad cuando: (1) Claude o un subagente acaba de escribir o modificar el código, (2) el usuario solicita explícitamente la revisión del código, (3) se ha completado la refactorización, (4) se han implementado nuevas características o componentes. La habilidad proporciona retroalimentación brutalmente honesta pero de apoyo siguiendo la filosofía de Carlos: celebrar el buen trabajo mientras se rechaza la complejidad, los patrones poco claros o las violaciones de las mejores prácticas.
