Überprüfen Sie TypeScript-, React- und Next.js-Code anhand hoher Standards auf Klarheit, Einfachheit und Wartbarkeit. Verwenden Sie diese Fähigkeit, wenn: (1) Code gerade von Claude oder einem Unteragenten geschrieben oder geändert wurde, (2) der Benutzer explizit eine Codeüberprüfung anfordert, (3) das Refactoring abgeschlossen ist, (4) neue Funktionen oder Komponenten implementiert wurden. Die Fähigkeit liefert brutal ehrliches, aber unterstützendes Feedback, das der Philosophie von Carlos folgt – gute Arbeit zu würdigen und gleichzeitig Komplexität, unklare Muster oder Verstöße gegen Best Practices energisch zu bekämpfen.
