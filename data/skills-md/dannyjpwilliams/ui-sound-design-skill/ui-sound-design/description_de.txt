Programmatisches UI-Sounddesign mit Web Audio API und Tone.js. Verwenden Sie diese Option, wenn Sie Klickgeräusche, Benachrichtigungssignale, Umschalt-Feedback, Hover-Geräusche, Erfolgs-/Fehler-Audio, Wusch-Effekte erstellen oder eine Soundbibliothek für UI-Interaktionen erstellen. Bietet einen iterativen Arbeitsablauf zum Beschreiben, Generieren, Hören und Verfeinern mit Audiotechnik-Wissen, das in einfaches Englisch übersetzt wird.
