تصميم صوت برمجي لواجهة المستخدم باستخدام Web Audio API وTone.js. يُستخدم عند إنشاء أصوات النقر، أو نغمات الإشعارات، أو تبديل التعليقات، أو أصوات التمرير، أو صوت النجاح/الخطأ، أو تأثيرات whoosh، أو إنشاء مكتبة صوتية لتفاعلات واجهة المستخدم. يوفر سير عمل متكررًا للوصف والتوليد والاستماع والتحسين مع معرفة الهندسة الصوتية المترجمة إلى اللغة الإنجليزية البسيطة.
