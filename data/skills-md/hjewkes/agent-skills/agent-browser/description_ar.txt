أتمتة المتصفح CLI لوكلاء الذكاء الاصطناعي. يُستخدم عندما يحتاج المستخدم إلى التفاعل مع مواقع الويب، بما في ذلك التنقل بين الصفحات، أو ملء النماذج، أو النقر على الأزرار، أو التقاط لقطات الشاشة، أو استخراج البيانات، أو اختبار تطبيقات الويب، أو أتمتة أي مهمة في المتصفح. تتضمن المشغلات طلبات "فتح موقع ويب"، أو "ملء نموذج"، أو "النقر فوق زر"، أو "التقاط لقطة شاشة"، أو "استخلاص البيانات من الصفحة"، أو "اختبار تطبيق الويب هذا"، أو "تسجيل الدخول إلى موقع"، أو "أتمتة إجراءات المتصفح"، أو أي مهمة تتطلب تفاعلًا برمجيًا على الويب.
