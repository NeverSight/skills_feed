قم بتوجيه إنشاء الوكلاء الفرعيين لـ Claude Code مع تعريف نطاق المهمة واختيار الأداة وتصميم موجه النظام وأنماط التفويض. يضمن تكوين الوكيل الفرعي المناسب مع الوصول المناسب للأداة وعزل السياق. يُستخدم عند إنشاء وكلاء فرعيين، أو تفويض المهام، أو إنشاء وكلاء متخصصين، أو عندما يحتاج المستخدمون إلى تنفيذ مهمة معزولة.
