قم بتوجيه إنشاء أوامر Claude Code من خلال سير العمل خطوة بخطوة، ومعالجة الوسيطات، وتكوين المادة الأمامية، والاختبار. يضمن أن الأوامر تتبع أفضل الممارسات وتصحح بنية YAML. يُستخدم عند إنشاء الأوامر، أو إنشاء أوامر الشرطة المائلة، أو إنشاء مهام سير عمل يتم تشغيلها بواسطة المستخدم، أو عندما يريد المستخدمون تنفيذ أمر صريح.
