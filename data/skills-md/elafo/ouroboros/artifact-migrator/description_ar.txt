ساعد في ترحيل وتحديث عناصر Claude Code الموجودة (المهارات والأوامر والوكلاء الفرعيين والخطافات) عندما تتغير المواصفات أو تتطور أفضل الممارسات. اكتشاف الأنماط القديمة واقتراح التحسينات وتوجيه عملية الترحيل. يُستخدم عندما تتوقف العناصر عن العمل، أو تحتاج إلى تحديثات، أو عندما تتغير مواصفات Claude Code.
