Una habilidad para apoyar la redacción de documentos. Se utiliza cuando un usuario desea crear, editar o actualizar documentos basados ​​en un PRD. Al hacer referencia a la especificación de requisitos del producto escrita en docs/prd.md, esta habilidad ayuda a escribir documentos técnicos, documentos de diseño, guías de usuario, documentación API y más. Se activa mediante tareas de creación de documentos como "escribir documentación", "crear especificaciones" o "hacer una guía".
