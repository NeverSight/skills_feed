Fähigkeiten zur Durchführung von Codeüberprüfungen. Wird verwendet, wenn ein Benutzer eine Codeüberprüfung unter den Gesichtspunkten Codequalität, Sicherheit, Tests, Leistung und Architektur anfordert. Ausgelöst durch Erwähnungen von Codeüberprüfungsaufgaben wie „Codeüberprüfung“, „Überprüfen Sie den Code“, „Überprüfen Sie die PR“ oder „Sehen Sie sich diesen Code an“. Priorisieren und weisen Sie auf Sicherheitslücken, Logikfehler, Testqualität und Leistungsprobleme hin und machen Sie konkrete Verbesserungsvorschläge.
