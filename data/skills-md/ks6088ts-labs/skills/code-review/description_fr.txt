Compétences pour effectuer des révisions de code. Utilisé lorsqu'un utilisateur demande une révision du code du point de vue de la qualité du code, de la sécurité, des tests, des performances et de l'architecture. Déclenché par des mentions de tâches de révision de code telles que « révision de code », « vérifier le code », « réviser le PR » ou « regarder ce code ». Priorise et souligne les vulnérabilités de sécurité, les erreurs logiques, la qualité des tests et les problèmes de performances, et fournit des suggestions d'amélioration concrètes.
