Habilidades para realizar revisiones de código. Se utiliza cuando un usuario solicita una revisión del código desde las perspectivas de calidad, seguridad, pruebas, rendimiento y arquitectura del código. Activado por menciones de tareas de revisión de código como "revisión de código", "verificar el código", "revisar el PR" o "mirar este código". Prioriza y señala vulnerabilidades de seguridad, errores lógicos, calidad de las pruebas y problemas de rendimiento, y proporciona sugerencias de mejora concretas.
