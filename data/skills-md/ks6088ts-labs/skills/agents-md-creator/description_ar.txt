مهارة إنشاء "AGENTS.md"، وهو مستند تعليمات لوكلاء تشفير الذكاء الاصطناعي. استخدمه عندما تريد إنشاء ملف يجمع السياق والتعليمات لوكلاء الذكاء الاصطناعي للعمل في مشروع. يتم تشغيله بطلبات مثل "إنشاء AGENTS.md" أو "إنشاء ورقة تعليمات لعملاء الذكاء الاصطناعي" أو "إنشاء ملف README موجه للوكيل". إنه تنسيق مفتوح ومعياري يمكن مشاركته عبر العديد من وكلاء الذكاء الاصطناعي مثل OpenAI Codex وClaude Code وGitHub Copilot وCursor.
