Habilidad para crear "AGENTS.md", un documento de instrucciones para agentes codificadores de IA. Úselo cuando desee crear un archivo que recopile contexto e instrucciones para que los agentes de IA trabajen en un proyecto. Activado por solicitudes como "Crear AGENTES.md", "Crear una hoja de instrucciones para agentes de IA" o "Crear un README orientado a agentes". Es un formato estándar abierto que se puede compartir entre múltiples agentes de IA, como OpenAI Codex, Claude Code, GitHub Copilot y Cursor.
