A skill for creating "AGENTS.md", an instruction manual for AI coding agents. Use it when you want to create a file that gathers context and instructions for AI agents to work on a project. Triggered by requests such as "Create AGENTS.md", "Make an instruction manual for AI agents", or "Create an agent-oriented README". It is an open, standard format that can be shared across multiple AI agents like OpenAI Codex, Claude Code, GitHub Copilot, and Cursor.
