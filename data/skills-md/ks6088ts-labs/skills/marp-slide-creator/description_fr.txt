La possibilité de créer des fichiers Markdown pour les diapositives de présentation à l'aide de Marp. Sur la base des demandes des utilisateurs, il génère des diapositives de haute qualité qui utilisent pleinement les fonctionnalités de Marp (directives, syntaxe d'image, listes de fragments, paramètres de thème, etc.). Utilisez-le pour des requêtes telles que « créer une diapositive », « faire une présentation », « écrire une diapositive dans Marp » ou « créer des supports de présentation dans Markdown ».
