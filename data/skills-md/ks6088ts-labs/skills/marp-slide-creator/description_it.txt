La possibilità di creare file Markdown per le diapositive di presentazione utilizzando Marp. In base alle richieste degli utenti, genera diapositive di alta qualità che sfruttano appieno le funzionalità di Marp (direttive, sintassi delle immagini, elenchi di frammenti, impostazioni dei temi, ecc.). Usalo per richieste come "crea una diapositiva", "crea una presentazione", "scrivi una diapositiva in Marp" o "crea materiali di presentazione in Markdown".
