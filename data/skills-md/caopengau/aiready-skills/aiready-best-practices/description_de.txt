Richtlinien zum Schreiben von KI-freundlichem Code. Erkennt semantische Duplikate, Kontextfragmentierung und Namensinkonsistenzen. Verwenden Sie es, wenn Sie neuen Code schreiben, PRs überprüfen, für die KI-Einführung umgestalten oder Verwirrung bei KI-Assistenten beheben. Hilft, Kontextverschwendung zu minimieren und das KI-Verständnis zu verbessern.
