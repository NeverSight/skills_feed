Erstellen Sie eine haloy.yaml-Konfigurationsdatei für die Bereitstellung von Anwendungen mit haloy. Wird verwendet, wenn der Benutzer „Haloy-Konfiguration erstellen“, „Haloy.yaml hinzufügen“, „Für Haloy konfigurieren“, „Haloy-Bereitstellung einrichten“ oder „Für Haloy vorbereiten“ sagt. Unterstützt Einzelziel- und Mehrzielbereitstellungen mit optionalen selbstgehosteten Datenbanken. Nicht zum Erstellen von Docker-Dateien (verwenden Sie den Dockerize-Skill), Bereitstellungen in mehreren Umgebungen oder Docker-Compose/Kubernetes-Setups.
