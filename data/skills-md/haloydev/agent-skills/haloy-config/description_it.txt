Crea un file di configurazione haloy.yaml per distribuire applicazioni con haloy. Da utilizzare quando l'utente dice "crea haloy config", "aggiungi haloy.yaml", "configure for haloy", "imposta la distribuzione di haloy" o "prepara per haloy". Supporta distribuzioni a destinazione singola e multi-target con database self-hosted opzionali. Non per la creazione di Dockerfile (utilizzare la competenza dockerize), distribuzioni multiambiente o configurazioni docker-compose/Kubernetes.
