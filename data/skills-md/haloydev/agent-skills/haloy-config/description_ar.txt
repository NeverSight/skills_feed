قم بإنشاء ملف تكوين haloy.yaml لنشر التطبيقات باستخدام haloy. يُستخدم عندما يقول المستخدم "إنشاء تكوين هالوي"، أو "إضافة هالوي.يامل"، أو "تكوين هالوي"، أو "إعداد نشر هالوي"، أو "التحضير لـ هالوي". يدعم عمليات النشر ذات الهدف الواحد والمتعددة الأهداف مع قواعد بيانات اختيارية ذاتية الاستضافة. ليس لإنشاء ملفات Dockerfiles (استخدم مهارة الإرساء)، أو عمليات النشر في بيئات متعددة، أو إعدادات إنشاء عامل الإرساء/Kubernetes.
