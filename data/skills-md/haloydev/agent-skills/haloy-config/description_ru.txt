Создайте файл конфигурации haloy.yaml для развертывания приложений с помощью haloy. Используйте, когда пользователь говорит «создать конфигурацию haloy», «добавить haloy.yaml», «настроить для haloy», «настроить развертывание haloy» или «подготовиться к haloy». Поддерживает одноцелевое и многоцелевое развертывание с дополнительными локальными базами данных. Не для создания файлов Dockerfile (используйте навык dockerize), развертываний в нескольких средах или настроек docker-compose/Kubernetes.
