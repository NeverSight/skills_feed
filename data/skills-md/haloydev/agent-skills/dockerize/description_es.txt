Cree un Dockerfile para que el proyecto actual se implemente con haloy. Úselo cuando el usuario diga "dockerizar", "crear un Dockerfile", "contener mi aplicación", "hacer que esto sea desplegable" o "prepararse para haloy". Esta habilidad crea Dockerfiles optimizados y listos para producción sin docker-compose. No apto para configuraciones de múltiples contenedores que requieren docker-compose, entornos de desarrollo local con múltiples servicios o manifiestos de Kubernetes.
