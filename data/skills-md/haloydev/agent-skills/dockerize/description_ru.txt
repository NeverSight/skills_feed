Создайте Dockerfile для текущего проекта для развертывания с помощью haloy. Используйте, когда пользователь говорит «докеризовать», «создать Dockerfile», «контейнеризировать мое приложение», «сделать его развертываемым» или «подготовиться к халою». Этот навык позволяет создавать оптимизированные, готовые к использованию файлы Dockerfile без использования docker-compose. Не для многоконтейнерных установок, требующих Docker-Compose, локальных сред разработки с несколькими сервисами или манифестов Kubernetes.
