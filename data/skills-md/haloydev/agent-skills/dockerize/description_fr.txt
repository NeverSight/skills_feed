Créez un Dockerfile pour le projet en cours à déployer avec haloy. À utiliser lorsque l'utilisateur dit « dockerize », « créer un fichier Docker », « conteneuriser mon application », « la rendre déployable » ou « préparer Haloy ». Cette compétence crée des fichiers Docker optimisés et prêts pour la production sans docker-compose. Pas pour les configurations multi-conteneurs nécessitant Docker-compose, des environnements de développement locaux avec plusieurs services ou des manifestes Kubernetes.
