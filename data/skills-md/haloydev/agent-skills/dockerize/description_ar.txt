أنشئ ملف Dockerfile للمشروع الحالي ليتم نشره باستخدام haloy. يُستخدم عندما يقول المستخدم "dockerize" أو "إنشاء ملف Dockerfile" أو "وضع تطبيقي في حاوية" أو "اجعل هذا قابلاً للنشر" أو "التحضير لـ haloy". تقوم هذه المهارة بإنشاء ملفات Dockerfiles محسّنة وجاهزة للإنتاج دون الحاجة إلى إنشاء عامل إرساء. ليس للإعدادات متعددة الحاويات التي تتطلب إنشاء عامل إرساء، أو بيئات التطوير المحلية مع خدمات متعددة، أو بيانات Kubernetes.
