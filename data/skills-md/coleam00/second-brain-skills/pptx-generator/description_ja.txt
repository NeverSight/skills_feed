プレゼンテーション スライドを PPTX ファイルとして生成および編集します。また、LinkedIn カルーセルを作成し、再利用可能なスライド レイアウトを管理します。

トリガー - ユーザーが次のように言うときにこのスキルを使用します。
- 「[ブランド] のスライドを作成」 / 「[ブランド] のプレゼンテーションを生成」 / 「[ブランド] のスライドを作成」
- 「[ブランド] のカルーセルを作成する」 / 「リンクされたカルーセル」 / 「[トピック] についてのカルーセルを作成する」
- 「この pptx を編集する」 / 「スライドを更新する」 / 「このプレゼンテーションを変更する」
- 「新しいレイアウトを作成する」 / 「クックブックにレイアウトを追加する」 / 「[タイプ] レイアウト テンプレートを作成する」
- 「[名前] レイアウトを編集する」 / 「クックブックを更新する」 / 「[名前] テンプレートを改善する」
- スライド、プレゼンテーション、カルーセル、PPTX、またはブランド名のレイアウトに関するリクエスト

PowerPoint、Google Slides、Keynote と互換性のある .pptx ファイルを作成します。
LinkedIn 用の PDF カルーセルを作成します (正方形 1:1 形式)。
