Универсальный MCP-клиент для подключения к любому MCP-серверу с прогрессивным раскрытием. Обертывает серверы MCP как навыки, чтобы избежать раздувания контекстных окон из-за определений инструментов. Используйте при взаимодействии с внешними серверами MCP (Zapier, Sequential Thinking, GitHub, файловой системой и т. д.), перечислении доступных инструментов или выполнении вызовов инструментов MCP. Запускается по таким запросам, как «подключиться к Zapier», «использовать сервер MCP», «перечислить инструменты MCP», «вызвать действие Zapier», «использовать последовательное мышление» или любому взаимодействию с сервером MCP.
