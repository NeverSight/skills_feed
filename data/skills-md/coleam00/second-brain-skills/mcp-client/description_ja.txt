漸進的開示を使用して任意の MCP サーバーに接続するためのユニバーサル MCP クライアント。 MCP サーバーをスキルとしてラップし、ツール定義によるコンテキスト ウィンドウの肥大化を回避します。外部 MCP サーバー (Zapier、Sequential Thinking、GitHub、ファイルシステムなど) と対話するとき、利用可能なツールをリストするとき、または MCP ツール呼び出しを実行するときに使用します。 「Zapier に接続する」、「MCP サーバーを使用する」、「MCP ツールをリストする」、「Zapier アクションを呼び出す」、「逐次的思考を使用する」、または MCP サーバーとの対話などのリクエストをトリガーします。
