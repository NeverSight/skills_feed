عميل MCP العالمي للاتصال بأي خادم MCP مع الكشف التدريجي. يتم تغليف خوادم MCP كمهارات لتجنب انتفاخ نافذة السياق من تعريفات الأداة. يُستخدم عند التفاعل مع خوادم MCP الخارجية (Zapier، وSequential Thinking، وGitHub، ونظام الملفات، وما إلى ذلك)، أو إدراج الأدوات المتاحة، أو تنفيذ استدعاءات أدوات MCP. تشغيل طلبات مثل "الاتصال بـ Zapier" أو "استخدام خادم MCP" أو "إدراج أدوات MCP" أو "استدعاء إجراء Zapier" أو "استخدام التفكير المتسلسل" أو أي تفاعل لخادم MCP.
