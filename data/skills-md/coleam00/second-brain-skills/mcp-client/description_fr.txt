Client MCP universel pour se connecter à n'importe quel serveur MCP avec divulgation progressive. Encapsule les serveurs MCP en tant que compétences pour éviter l'encombrement de la fenêtre contextuelle à partir des définitions d'outils. À utiliser lors de l'interaction avec des serveurs MCP externes (Zapier, Sequential Thinking, GitHub, système de fichiers, etc.), en répertoriant les outils disponibles ou en exécutant des appels d'outils MCP. Se déclenche sur des requêtes telles que « se connecter à Zapier », « utiliser le serveur MCP », « lister les outils MCP », « appeler l'action Zapier », « utiliser la pensée séquentielle » ou toute interaction du serveur MCP.
