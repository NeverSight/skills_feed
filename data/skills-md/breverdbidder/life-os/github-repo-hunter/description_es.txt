Agente autónomo de integración y descubrimiento de repositorios de GitHub que busca proyectos relevantes de código abierto para BidDeed.AI (subastas de ejecución hipotecaria) y Life OS (productividad/TDAH). Busca en GitHub repositorios que coincidan con palabras clave de dominio, evalúa la relevancia a través del análisis README/pila tecnológica, agrega automáticamente como submódulos o archivos a repositorios de destino y alerta a Ariel con resúmenes prácticos. Úselo cuando (1) el usuario solicite "encontrar repositorios de GitHub relevantes", (2) explore nuevas integraciones para los flujos de trabajo de BidDeed.AI, (3) descubra herramientas de productividad para Life OS, (4) mantenga los repositorios actualizados con las últimas innovaciones de código abierto.
