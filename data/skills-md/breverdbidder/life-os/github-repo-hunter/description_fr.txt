Agent autonome de découverte et d'intégration de référentiels GitHub qui recherche les projets open source pertinents pour BidDeed.AI (enchères de saisies immobilières) et Life OS (productivité/TDAH). Recherche sur GitHub les mots-clés de domaine correspondant aux dépôts, évalue la pertinence via l'analyse README/tech stack, s'ajoute automatiquement en tant que sous-modules ou archives aux dépôts cibles et alerte Ariel avec des résumés exploitables. À utiliser lorsque (1) l'utilisateur demande « trouver des dépôts GitHub pertinents », (2) explorer de nouvelles intégrations pour les flux de travail BidDeed.AI, (3) découvrir des outils de productivité pour Life OS, (4) maintenir les référentiels à jour avec les dernières innovations open source.
