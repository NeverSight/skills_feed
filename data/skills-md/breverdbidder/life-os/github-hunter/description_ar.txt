اكتشاف مستودعات GitHub ذات الصلة وتسجيلها وأرشفتها تلقائيًا لمشاريع BidDeed.AI وLife OS. استخدم هذه المهارة عندما يطلب المستخدم العثور على مستودعات GitHub لموضوعات أو تقنيات أو مجالات محددة يمكن دمجها في المشاريع الحالية. تتضمن المشغلات عبارات مثل "ابحث عن مستودعات GitHub لـ"، أو "اكتشف مستودعات حول"، أو "ابحث في GitHub عن مشاريع ذات صلة"، أو عند تحليل مقاطع الفيديو/المقالات التي تشير إلى مشاريع GitHub.
