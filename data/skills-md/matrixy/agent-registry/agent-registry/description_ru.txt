ОБЯЗАТЕЛЬНАЯ система обнаружения агентов для загрузки агентов с использованием токенов. Клод ДОЛЖЕН использовать этот навык 
вместо загрузки агентов непосредственно из ~/.claude/agents/ или .claude/agents/. Обеспечивает ленивый 
загрузка с помощью инструментов search_agents и get_agent. Используйте, когда: (1) пользовательская задача может выиграть от 
специализированная экспертиза агента, (2) пользователь спрашивает о доступных агентах, (3) запуск комплекса 
рабочие процессы, в которых исторически использовались агенты. Этот навык снижает использование контекстного окна примерно на 95 %. 
по сравнению с предварительной загрузкой всех агентов.
