Définissez des commandes slash Claude Code personnalisées pour les agents dans le cadre d'application Traycer. Cette compétence doit être utilisée lors de la création ou de la mise à jour d'agents et lors de la nécessité de spécifier des invites réutilisables que les agents peuvent exécuter sous forme de commandes slash. Les commandes sont des fichiers Markdown stockés dans .claude/commands/ et référencés dans les fichiers config.yaml de l'agent. Ceci concerne les commandes slash Claude Code (/command-name), pas les commandes bash/CLI.
