Créez des agents Claude Code complets pour le cadre d'application Traycer. Cette compétence doit être utilisée lors de la création de nouveaux agents ou de la mise à jour d'agents existants. Crée tous les composants de l'agent, y compris l'invite système, les hooks, les compétences, les commandes, les scripts et les documents de référence. Met également à jour les documents de coordination qui répertorient les agents disponibles. Basé sur les meilleures pratiques de développement d'agents officiels d'Anthropic.
