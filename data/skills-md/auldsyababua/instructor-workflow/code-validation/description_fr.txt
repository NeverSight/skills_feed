Validation automatisée du code pour l'examen des différences, l'hygiène des changements et la détection des signaux d'alarme. À utiliser lors de l'examen des différences git, des PR ou des fichiers modifiés pour tester les modèles de désactivation, les secrets, les problèmes de portabilité des chemins, les indicateurs de sécurité et les suppressions importantes. Prend en charge Python, TypeScript, JavaScript, HTML et CSS.
