Asistente de prepublicación para nuevas publicaciones de blog. Úselo cuando el usuario quiera clasificar una nueva publicación con categorías y etiquetas, generar metadatos de SEO (título, descripción, frase clave de enfoque) u obtener sugerencias inteligentes con fundamento. Funciona con borrador de contenido (ruta de archivo, URL o texto) y sugiere a partir de taxonomía existente para mantener una distribución equilibrada.
