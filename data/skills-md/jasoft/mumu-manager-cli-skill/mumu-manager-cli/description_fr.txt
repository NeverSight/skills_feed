Utilisez MuMuManager.exe pour gérer et exploiter de manière centralisée les instances de MuMu Player 12, y compris la configuration de l'instance, la création/clonage/suppression/renommage, l'importation/exportation, le démarrage/arrêt/redémarrage/contrôle de fenêtre, l'installation/désinstallation/démarrage/arrêt d'application, les informations sur le périphérique, les outils système, les commandes ADB, les pilotes et le tri. Idéal pour l'automatisation par lots de scripts de la gestion de MuMu Player.
