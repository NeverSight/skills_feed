Scrivi test che iniziano con criteri di accettazione, quindi aggiungi test di implementazione per la robustezza. Da utilizzare durante la scrittura di unit test (Vitest), test end-to-end (Playwright), test di regressione visiva o test di accessibilità. Enfatizza i test incentrati sull'utente, i localizzatori semantici, la convalida dell'accessibilità e l'equilibrio tra test di accettazione e implementazione.
