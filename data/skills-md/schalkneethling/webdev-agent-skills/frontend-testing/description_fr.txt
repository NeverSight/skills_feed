Écrivez des tests qui commencent par des critères d'acceptation, puis ajoutez des tests d'implémentation pour la robustesse. À utiliser lors de l'écriture de tests unitaires (Vitest), de tests de bout en bout (Playwright), de tests de régression visuelle ou de tests d'accessibilité. Met l'accent sur les tests centrés sur l'utilisateur, les localisateurs sémantiques, la validation de l'accessibilité et l'équilibre entre les tests d'acceptation et de mise en œuvre.
