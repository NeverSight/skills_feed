Руководство по созданию CSS с упором на веб-стандарты, доступность и производительность. Используйте при написании, просмотре или рефакторинге CSS. Предоставляет шаблоны, фрагменты и соглашения, которые отдают приоритет собственному CSS над платформами, семантической структурой и поддерживаемым кодом. Обратитесь к references/patterns.md для получения конкретных шаблонов и фрагментов.
