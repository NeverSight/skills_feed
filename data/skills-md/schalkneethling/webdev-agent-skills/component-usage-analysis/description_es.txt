Analice las dependencias de los componentes y los patrones de uso en una biblioteca de componentes Drupal/Twig. Úselo cuando el usuario solicite saber dónde se usa un componente, verificar si un componente se puede eliminar de manera segura, auditar las dependencias de los componentes, encontrar componentes que utilicen propiedades específicas o analizar el impacto de refactorizar un componente.
