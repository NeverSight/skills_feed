Analizza le dipendenze dei componenti e i modelli di utilizzo in una libreria di componenti Drupal/Twig. Da utilizzare quando l'utente chiede di trovare dove viene utilizzato un componente, verificare se un componente può essere rimosso in modo sicuro, controllare le dipendenze dei componenti, trovare componenti utilizzando proprietà specifiche o analizzare l'impatto del refactoring di un componente.
