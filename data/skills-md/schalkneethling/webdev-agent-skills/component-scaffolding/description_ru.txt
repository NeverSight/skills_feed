Создавайте скелеты компонентов Drupal/Twig с помощью веб-компонентов и проверки Miyagi. Используйте, когда пользователь запрашивает создание, создание или добавление нового компонента по определенному пути (например, «добавить скелет компонента в шаблоны/кнопку общего доступа») или при создании файлов компонентов, включая шаблоны Twig, CSS, веб-компоненты JavaScript, схемы JSON или файлы макетов данных.
