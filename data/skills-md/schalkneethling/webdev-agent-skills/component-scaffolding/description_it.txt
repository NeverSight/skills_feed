Genera scheletri di componenti Drupal/Twig con componenti web e convalida Miyagi. Da utilizzare quando l'utente richiede di creare, impalcare o aggiungere un nuovo componente in un percorso specifico (ad esempio, "aggiungi la struttura del componente ai modelli/pulsante di condivisione") o quando si creano file di componenti inclusi modelli Twig, CSS, componenti Web JavaScript, schemi JSON o file di dati fittizi.
