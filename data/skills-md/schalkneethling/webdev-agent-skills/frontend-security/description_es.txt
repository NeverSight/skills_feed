Audite las bases de código frontend para detectar vulnerabilidades de seguridad y malas prácticas. Úselo al realizar revisiones de seguridad, auditar código para vulnerabilidades XSS/CSRF/DOM, verificar configuraciones de políticas de seguridad de contenido, validar el manejo de entradas, revisar la seguridad de carga de archivos o examinar dependencias de Node.js/NPM. Los marcos de destino incluyen plataforma web (HTML/CSS/JS básico), React, Astro, plantillas Twig, Node.js y Bun. Basado en las pautas de seguridad de OWASP.
