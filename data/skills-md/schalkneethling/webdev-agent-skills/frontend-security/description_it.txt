Controlla le basi di codice frontend per individuare vulnerabilità della sicurezza e cattive pratiche. Da utilizzare durante l'esecuzione di revisioni della sicurezza, il controllo del codice per le vulnerabilità XSS/CSRF/DOM, il controllo delle configurazioni dei criteri di sicurezza dei contenuti, la convalida della gestione degli input, la revisione della sicurezza del caricamento di file o l'esame delle dipendenze Node.js/NPM. I framework di destinazione includono la piattaforma web (vanilla HTML/CSS/JS), React, Astro, modelli Twig, Node.js e Bun. Basato sulle linee guida di sicurezza OWASP.
