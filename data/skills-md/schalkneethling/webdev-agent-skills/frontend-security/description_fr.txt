Auditez les bases de code frontend pour détecter les vulnérabilités de sécurité et les mauvaises pratiques. À utiliser lors de l'exécution d'examens de sécurité, de l'audit du code pour les vulnérabilités XSS/CSRF/DOM, de la vérification des configurations de la politique de sécurité du contenu, de la validation de la gestion des entrées, de l'examen de la sécurité du téléchargement de fichiers ou de l'examen des dépendances Node.js/NPM. Les frameworks cibles incluent la plate-forme Web (vanilla HTML/CSS/JS), React, Astro, les modèles Twig, Node.js et Bun. Basé sur les directives de sécurité de l'OWASP.
