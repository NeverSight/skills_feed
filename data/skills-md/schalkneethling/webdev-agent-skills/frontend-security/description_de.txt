Überprüfen Sie Frontend-Codebasen auf Sicherheitslücken und schlechte Praktiken. Verwenden Sie es, wenn Sie Sicherheitsüberprüfungen durchführen, Code auf XSS/CSRF/DOM-Schwachstellen prüfen, Konfigurationen von Inhaltssicherheitsrichtlinien überprüfen, die Eingabeverarbeitung validieren, die Datei-Upload-Sicherheit überprüfen oder Node.js/NPM-Abhängigkeiten untersuchen. Zu den Ziel-Frameworks gehören Webplattformen (Vanilla HTML/CSS/JS), React, Astro, Twig-Vorlagen, Node.js und Bun. Basierend auf OWASP-Sicherheitsrichtlinien.
