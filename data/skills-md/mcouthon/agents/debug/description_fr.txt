Débogage systématique avec enquête basée sur des hypothèses. À utiliser lorsque quelque chose est cassé, que les tests échouent, qu'un comportement inattendu se produit ou que des erreurs nécessitent une enquête. Se déclenche sur : "ceci est cassé", "débogage", "pourquoi cela échoue-t-il", "erreur inattendue", "ne fonctionne pas", "bug", "réparer ce problème", "enquêter", "échec des tests", "tracer l'erreur", "utiliser le mode débogage". Mode d'accès complet : peut exécuter des commandes, ajouter une journalisation et résoudre des problèmes.
