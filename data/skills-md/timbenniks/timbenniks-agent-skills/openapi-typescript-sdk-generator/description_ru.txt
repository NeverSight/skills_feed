Создайте рабочий SDK TypeScript на основе спецификации OpenAPI 3.0/3.1 (URL-адрес или локальный файл) с понятными для человека именами методов, интерактивным подтверждением и детерминированным выводом. Используйте, когда пользователю нужен типобезопасный SDK на основе выборки или встроенный клиент. Не используйте для потоковой передачи/SSE/WebSockets или интерактивных потоков OAuth.
