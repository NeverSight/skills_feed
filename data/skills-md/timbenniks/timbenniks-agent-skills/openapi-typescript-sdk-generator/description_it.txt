Genera un SDK TypeScript di produzione da una specifica OpenAPI 3.0/3.1 (URL o file locale), con nomi di metodi intuitivi, conferma interattiva e output deterministico. Da utilizzare quando l'utente desidera un SDK indipendente dai tipi e basato sul recupero o un client drop-in. Non utilizzare per streaming/SSE/WebSocket o flussi OAuth interattivi.
