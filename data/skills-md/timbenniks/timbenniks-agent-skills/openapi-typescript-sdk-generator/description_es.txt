Genere un SDK de TypeScript de producción a partir de una especificación OpenAPI 3.0/3.1 (URL o archivo local), con nombres de métodos amigables para los humanos, confirmación interactiva y salida determinista. Úselo cuando el usuario desee un SDK basado en recuperación y con seguridad de tipos o un cliente directo. No lo utilice para streaming/SSE/WebSockets o flujos interactivos de OAuth.
