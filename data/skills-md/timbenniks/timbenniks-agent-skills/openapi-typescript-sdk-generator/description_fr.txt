Générez un SDK TypeScript de production à partir d'une spécification OpenAPI 3.0/3.1 (URL ou fichier local), avec des noms de méthodes conviviaux, une confirmation interactive et une sortie déterministe. À utiliser lorsque l'utilisateur souhaite un SDK de type sécurisé basé sur la récupération ou un client instantané. Ne pas utiliser pour le streaming/SSE/WebSockets ou les flux OAuth interactifs.
