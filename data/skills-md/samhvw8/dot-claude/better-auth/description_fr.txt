Cadre d'authentification TypeScript (indépendant du framework). Fonctionnalités : email/mot de passe, OAuth (Google, GitHub, Discord), 2FA (TOTP, SMS), mots de passe/WebAuthn, gestion de session, RBAC, limitation de débit, adaptateurs de base de données. Actions : mettre en œuvre, configurer, sécuriser les systèmes d'authentification. Mots clés : meilleure authentification, authentification, autorisation, OAuth, e-mail/mot de passe, 2FA, MFA, TOTP, mots de passe, WebAuthn, gestion de session, RBAC, limitation de débit, adaptateur de base de données, authentification TypeScript, connexion sociale, authentification Google, authentification GitHub, authentification Discord, vérification par e-mail, réinitialisation du mot de passe. À utiliser lors de : la mise en œuvre de l'authentification TypeScript, l'ajout de fournisseurs OAuth, la configuration de 2FA/MFA, la gestion de sessions, la configuration de RBAC, la création de systèmes d'authentification sécurisés.
