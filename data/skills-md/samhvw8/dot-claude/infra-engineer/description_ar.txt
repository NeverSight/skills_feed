هندسة بنية تحتية شاملة تغطي DevOps والأنظمة الأساسية السحابية وFinOps وDevSecOps. الأنظمة الأساسية: AWS (EC2، Lambda، S3، ECS، EKS، RDS، CloudFormation)، أساسيات Azure، Cloudflare (Workers، R2، D1، Pages)، GCP (GKE، Cloud Run، Cloud Storage)، Docker، Kubernetes. القدرات: خطوط أنابيب CI/CD (GitHub Actions، GitLab CI، Jenkins)، GitOps، البنية التحتية كرمز (Terraform، CloudFormation)، تنسيق الحاويات، تحسين التكلفة، المسح الأمني، إدارة الثغرات الأمنية، إدارة الأسرار، الامتثال (SOC2، HIPAA). الإجراءات: نشر البنية التحتية السحابية وتكوينها وإدارتها وتوسيع نطاقها ومراقبتها وتأمينها وتحسينها. الكلمات الرئيسية: AWS، EC2، Lambda، S3، ECS، EKS، RDS، CloudFormation، Azure، Kubernetes، k8s، Docker، Terraform، CI/CD، GitHub Actions، GitLab CI، Jenkins، ArgoCD، Flux، تحسين التكلفة، FinOps، المثيلات المحجوزة، المثيلات الموضعية، المسح الأمني، SAST، DAST، إدارة الثغرات الأمنية، إدارة الأسرار، Vault، الامتثال والرصد والملاحظة. يُستخدم عندما: النشر إلى AWS/Azure/GCP/Cloudflare، وإعداد خطوط أنابيب CI/CD، وتنفيذ سير عمل GitOps، وإدارة مجموعات Kubernetes، وتحسين تكاليف السحابة، وتنفيذ أفضل ممارسات الأمان، وإدارة البنية التحتية كرمز، وتنسيق الحاويات، ومتطلبات الامتثال، وتحليل التكلفة والتحسين.
