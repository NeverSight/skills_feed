Sistemi di componenti dell'interfaccia utente React con TailwindCSS + Radix + shadcn/ui. Stack: TailwindCSS (stile), Radix UI (primitive), shadcn/ui (componenti), React/Next.js. Funzionalità: architettura del sistema di progettazione, componenti accessibili, layout reattivi, temi, modalità oscura, composizione dei componenti. Azioni: revisione, progettazione, creazione, miglioramento, refactoring dei componenti dell'interfaccia utente. Parole chiave: TailwindCSS, Radix UI, shadcn/ui, sistema di progettazione, libreria di componenti, accessibilità, ARIA, responsivo, modalità oscura, temi, variabili CSS, architettura dei componenti, progettazione atomica, token di progettazione, variante, slot, composizione. Da utilizzare quando: creazione di librerie di componenti, implementazione di shadcn/ui, creazione di interfacce utente accessibili, impostazione di sistemi di progettazione, aggiunta di modalità oscura/temi, revisione dell'architettura dei componenti dell'interfaccia utente.
