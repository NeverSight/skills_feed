Системы компонентов пользовательского интерфейса React с TailwindCSS + Radix + shadcn/ui. Стек: TailwindCSS (стилизация), Radix UI (примитивы), shadcn/ui (компоненты), React/Next.js. Возможности: проектирование архитектуры системы, доступные компоненты, адаптивные макеты, темирование, темный режим, композиция компонентов. Действия: просмотр, проектирование, сборка, улучшение, рефакторинг компонентов пользовательского интерфейса. Ключевые слова: TailwindCSS, Radix UI, shadcn/ui, система дизайна, библиотека компонентов, доступность, ARIA, адаптивность, темный режим, темы, переменные CSS, архитектура компонентов, атомарный дизайн, токены дизайна, вариант, слот, композиция. Используйте при: создании библиотек компонентов, реализации shadcn/ui, создании доступных пользовательских интерфейсов, настройке систем дизайна, добавлении темного режима/тем, проверке архитектуры компонентов пользовательского интерфейса.
