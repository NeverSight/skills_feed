使用 TailwindCSS + Radix + shadcn/ui 来响应 UI 组件系统。堆栈：TailwindCSS（样式）、Radix UI（基元）、shadcn/ui（组件）、React/Next.js。功能：设计系统架构、可访问组件、响应式布局、主题、暗模式、组件组合。操作：审查、设计、构建、改进、重构 UI 组件。关键词：TailwindCSS、Radix UI、shadcn/ui、设计系统、组件库、可访问性、ARIA、响应式、暗模式、主题、CSS 变量、组件架构、原子设计、设计令牌、变体、插槽、组合。使用场合：构建组件库、实现 shadcn/ui、创建可访问的 UI、设置设计系统、添加深色模式/主题、审查 UI 组件架构。
