Systèmes de composants d'interface utilisateur React avec TailwindCSS + Radix + shadcn/ui. Pile : TailwindCSS (style), Radix UI (primitives), shadcn/ui (composants), React/Next.js. Capacités : conception de l'architecture du système, composants accessibles, mises en page réactives, thèmes, mode sombre, composition des composants. Actions : réviser, concevoir, construire, améliorer, refactoriser les composants de l'interface utilisateur. Mots clés : TailwindCSS, Radix UI, shadcn/ui, système de conception, bibliothèque de composants, accessibilité, ARIA, réactif, mode sombre, thématisation, variables CSS, architecture des composants, conception atomique, jetons de conception, variante, emplacement, composition. À utiliser dans les cas suivants : création de bibliothèques de composants, implémentation de shadcn/ui, création d'interfaces utilisateur accessibles, configuration de systèmes de conception, ajout d'un mode sombre/thème, révision de l'architecture des composants de l'interface utilisateur.
