3D-Webgrafiken mit Three.js (WebGL/WebGPU). Fähigkeiten: Szenen, Kameras, Geometrien, Materialien, Lichter, Animationen, Modellladen (GLTF/FBX), PBR-Materialien, Schatten, Nachbearbeitung (Bloom, SSAO, SSR), benutzerdefinierte Shader, Instanzierung, LOD, Physik, VR/XR. Aktionen: Erstellen, Erstellen, Animieren, Rendern von 3D-Szenen/Modellen. Schlüsselwörter: Three.js, WebGL, WebGPU, 3D-Grafik, Szene, Kamera, Geometrie, Material, Licht, Animation, GLTF, FBX, OrbitControls, PBR, Schattenmapping, Nachbearbeitung, Bloom, SSAO, Shader, Instanzierung, LOD, WebXR, VR, AR, Produktkonfigurator, Datenvisualisierung, architektonische Komplettlösung, interaktives 3D, Leinwand. Verwendung bei: Erstellen von 3D-Visualisierungen, Erstellen von WebGL/WebGPU-Apps, Laden von 3D-Modellen, Hinzufügen von Animationen, Implementieren von VR/XR, Erstellen interaktiver Grafiken, Erstellen von Produktkonfiguratoren.
