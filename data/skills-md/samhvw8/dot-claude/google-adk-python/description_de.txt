Google Agent Development Kit (ADK) für Python. Fähigkeiten: Erstellung von KI-Agenten, Multiagentensysteme, Workflow-Agenten (sequentiell/parallel/Schleife), Tool-Integration (Google-Suche, Codeausführung), Vertex-KI-Bereitstellung, Agentenbewertung, Human-in-the-Loop-Abläufe. Aktionen: Erstellen, Erstellen, Bereitstellen, Bewerten, Orchestrieren von KI-Agenten. Schlüsselwörter: Google ADK, Agent Development Kit, KI-Agent, Multi-Agenten-System, LlmAgent, SequentialAgent, ParallelAgent, LoopAgent, Tool-Integration, Google-Suche, Codeausführung, Vertex AI, Cloud Run, Agentenbewertung, Human-in-the-Loop, Agenten-Orchestrierung, Workflow-Agent, hierarchische Koordination. Verwendung bei: Erstellung von KI-Agenten, Erstellung von Systemen mit mehreren Agenten, Implementierung von Workflow-Pipelines, Integration von LLM-Agenten mit Tools, Bereitstellung auf Vertex AI, Bewertung der Agentenleistung, Implementierung von Genehmigungsabläufen.
