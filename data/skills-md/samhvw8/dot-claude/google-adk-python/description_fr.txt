Kit de développement d'agent Google (ADK) pour Python. Capacités : création d'agents IA, systèmes multi-agents, agents de workflow (séquentiels/parallèles/boucles), intégration d'outils (recherche Google, exécution de code), déploiement de Vertex AI, évaluation d'agents, flux humains dans la boucle. Actions : construire, créer, déployer, évaluer, orchestrer des agents IA. Mots clés : Google ADK, Agent Development Kit, agent IA, système multi-agent, LlmAgent, SequentialAgent, ParallelAgent, LoopAgent, intégration d'outils, recherche Google, exécution de code, Vertex AI, Cloud Run, évaluation d'agent, human-in-the-loop, orchestration d'agent, agent de workflow, coordination hiérarchique. À utiliser dans les cas suivants : création d'agents IA, création de systèmes multi-agents, mise en œuvre de pipelines de flux de travail, intégration d'agents LLM avec des outils, déploiement sur Vertex AI, évaluation des performances des agents, mise en œuvre de flux d'approbation.
