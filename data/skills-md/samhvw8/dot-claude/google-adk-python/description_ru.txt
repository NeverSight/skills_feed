Комплект разработки агента Google (ADK) для Python. Возможности: создание агентов искусственного интеллекта, многоагентные системы, агенты рабочих процессов (последовательные/параллельные/циклические), интеграция инструментов (поиск Google, выполнение кода), развертывание Vertex AI, оценка агентов, потоки управления человеком в цикле. Действия: сборка, создание, развертывание, оценка, оркестровка агентов ИИ. Ключевые слова: Google ADK, комплект разработки агентов, агент ИИ, многоагентная система, LlmAgent, SequentialAgent, ParallelAgent, LoopAgent, интеграция инструментов, поиск Google, выполнение кода, Vertex AI, Cloud Run, оценка агента, управление человеком в цикле, оркестровка агентов, агент рабочего процесса, иерархическая координация. Используйте при: создании агентов ИИ, создании многоагентных систем, реализации конвейеров рабочих процессов, интеграции агентов LLM с инструментами, развертывании в Vertex AI, оценке производительности агентов, реализации потоков утверждения.
