مجموعة أدوات تطوير وكيل Google (ADK) للغة Python. القدرات: بناء وكيل الذكاء الاصطناعي، وأنظمة متعددة الوكلاء، ووكلاء سير العمل (متسلسل/متوازي/حلقة)، وتكامل الأدوات (بحث Google، وتنفيذ التعليمات البرمجية)، ونشر Vertex AI، وتقييم الوكيل، وتدفقات الإنسان داخل الحلقة. الإجراءات: بناء وإنشاء ونشر وتقييم وتنسيق وكلاء الذكاء الاصطناعي. الكلمات الرئيسية: Google ADK، Agent Development Kit، وكيل الذكاء الاصطناعي، نظام متعدد الوكلاء، LlmAgent، SequentialAgent، ParallelAgent، LoopAgent، تكامل الأدوات، بحث Google، تنفيذ التعليمات البرمجية، Vertex AI، Cloud Run، تقييم الوكيل، الإنسان داخل الحلقة، تنسيق الوكيل، وكيل سير العمل، التنسيق الهرمي. يُستخدم عندما: إنشاء وكلاء الذكاء الاصطناعي، وإنشاء أنظمة متعددة الوكلاء، وتنفيذ مسارات سير العمل، ودمج وكلاء LLM مع الأدوات، والنشر إلى Vertex AI، وتقييم أداء الوكيل، وتنفيذ تدفقات الموافقة.
