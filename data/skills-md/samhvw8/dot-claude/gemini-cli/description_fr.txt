Orchestration Google Gemini CLI pour le développement assisté par l'IA. Capacités : deuxième avis/validation croisée, recherche Web en temps réel (recherche Google), analyse de l'architecture de la base de code, génération de code parallèle, révision du code sous différents angles. Actions : interroger, rechercher, analyser, générer, réviser avec Gemini. Mots clés : Gemini CLI, deuxième avis, validation croisée, recherche Google, recherche sur le Web, informations actuelles, IA parallèle, révision de code, analyse d'architecture, invite Gemini, comparaison d'IA, recherche en temps réel, perspective alternative, génération de code. À utiliser lorsque : besoin d'un deuxième avis sur l'IA, recherche d'informations Web actuelles, analyse de l'architecture de base de code, génération de code en parallèle, obtention d'une révision alternative du code, recherche d'événements/documents actuels.
