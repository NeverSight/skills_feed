Orchestrazione della CLI di Google Gemini per lo sviluppo assistito dall'intelligenza artificiale. Funzionalità: seconda opinione/convalida incrociata, ricerca web in tempo reale (Ricerca Google), analisi dell'architettura della base di codice, generazione di codice parallelo, revisione del codice da diverse prospettive. Azioni: interrogare, cercare, analizzare, generare, rivedere con Gemini. Parole chiave: Gemini CLI, seconda opinione, convalida incrociata, Ricerca Google, ricerca web, informazioni attuali, intelligenza artificiale parallela, revisione del codice, analisi dell'architettura, prompt Gemini, confronto AI, ricerca in tempo reale, prospettiva alternativa, generazione di codice. Da utilizzare quando: è necessaria una seconda opinione dell'intelligenza artificiale, si ricercano informazioni Web attuali, si analizza l'architettura della base di codice, si genera codice in parallelo, si ottiene una revisione del codice alternativa, si ricercano eventi/documenti attuali.
