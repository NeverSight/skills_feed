Оркестровка Google Gemini CLI для разработки с помощью искусственного интеллекта. Возможности: второе мнение/перекрестная проверка, веб-поиск в реальном времени (поиск Google), анализ архитектуры кодовой базы, параллельная генерация кода, проверка кода с разных точек зрения. Действия: запрос, поиск, анализ, создание, просмотр с помощью Gemini. Ключевые слова: Gemini CLI, второе мнение, перекрестная проверка, поиск Google, веб-исследования, текущая информация, параллельный ИИ, проверка кода, анализ архитектуры, подсказка Gemini, сравнение ИИ, поиск в реальном времени, альтернативная точка зрения, генерация кода. Используйте, когда: требуется второе мнение ИИ, поиск текущей веб-информации, анализ архитектуры кодовой базы, параллельная генерация кода, получение альтернативной проверки кода, исследование текущих событий/документов.
