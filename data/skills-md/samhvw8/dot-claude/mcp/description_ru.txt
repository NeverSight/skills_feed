Разработка сервера протокола контекста модели (MCP) и управление инструментами. Языки: Python, TypeScript. Возможности: создание серверов MCP, интеграция внешних API, обнаружение/запуск инструментов MCP, управление многосерверными конфигурациями, разработка инструментов, ориентированных на агентов. Действия: создание, сборка, интеграция, обнаружение, выполнение, настройка серверов/инструментов MCP. Ключевые слова: MCP, протокол контекста модели, сервер MCP, инструмент MCP, транспорт stdio, транспорт SSE, обнаружение инструмента, поставщик ресурсов, шаблон приглашения, интеграция внешнего API, Gemini CLI MCP, Claude MCP, инструменты агента, выполнение инструмента, конфигурация сервера. Используйте при: создании серверов MCP, интеграции внешних API в качестве инструментов MCP, обнаружении доступных инструментов MCP, реализации возможностей MCP, настройке многосерверных установок, разработке инструментов для агентов ИИ.
