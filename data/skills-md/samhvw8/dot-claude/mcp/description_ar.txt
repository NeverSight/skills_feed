تطوير خادم بروتوكول السياق النموذجي (MCP) وإدارة الأدوات. اللغات: بايثون، تايب سكريبت. القدرات: إنشاء خوادم MCP، ودمج واجهات برمجة التطبيقات الخارجية، واكتشاف/تنفيذ أدوات MCP، وإدارة تكوينات الخوادم المتعددة، وتصميم الأدوات التي تركز على الوكيل. الإجراءات: إنشاء وبناء ودمج واكتشاف وتنفيذ وتكوين خوادم/أدوات MCP. الكلمات الرئيسية: MCP، بروتوكول السياق النموذجي، خادم MCP، أداة MCP، نقل stdio، نقل SSE، اكتشاف الأدوات، موفر الموارد، القالب الفوري، تكامل واجهة برمجة التطبيقات الخارجية، Gemini CLI MCP، Claude MCP، أدوات الوكيل، تنفيذ الأداة، تكوين الخادم. يُستخدم عندما: إنشاء خوادم MCP، ودمج واجهات برمجة التطبيقات الخارجية كأدوات MCP، واكتشاف أدوات MCP المتاحة، وتنفيذ إمكانات MCP، وتكوين إعدادات الخوادم المتعددة، وتصميم الأدوات لوكلاء الذكاء الاصطناعي.
