Développement de serveur MCP (Model Context Protocol) et gestion d'outils. Langages : Python, TypeScript. Capacités : créer des serveurs MCP, intégrer des API externes, découvrir/exécuter des outils MCP, gérer des configurations multi-serveurs, concevoir des outils centrés sur les agents. Actions : créer, construire, intégrer, découvrir, exécuter, configurer des serveurs/outils MCP. Mots clés : MCP, Model Context Protocol, serveur MCP, outil MCP, transport stdio, transport SSE, découverte d'outils, fournisseur de ressources, modèle d'invite, intégration d'API externe, Gemini CLI MCP, Claude MCP, outils d'agent, exécution d'outils, configuration de serveur. À utiliser lors de : la création de serveurs MCP, l'intégration d'API externes en tant qu'outils MCP, la découverte des outils MCP disponibles, l'exécution de fonctionnalités MCP, la configuration de configurations multi-serveurs, la conception d'outils pour les agents IA.
