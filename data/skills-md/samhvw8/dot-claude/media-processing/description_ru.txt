Обработка видео/аудио/изображений с помощью FFmpeg и ImageMagick. Инструменты: FFmpeg (видео/аудио), ImageMagick (изображения). Возможности: преобразование формата, кодирование (H.264/H.265/VP9/AV1), потоковая передача (HLS/DASH), фильтры, эффекты, миниатюры, водяные знаки, пакетная обработка, аппаратное ускорение (NVENC/QSV). Действия: конвертировать, кодировать, изменять размер, обрезать, сжимать, извлекать, объединять, транслировать, перекодировать медиафайлы. Ключевые слова: FFmpeg, ImageMagick, кодирование видео, извлечение аудио, изменение размера изображения, миниатюра, водяной знак, HLS, DASH, H.264, H.265, VP9, ​​AV1, кодек, битрейт, частота кадров, разрешение, соотношение сторон, фильтр, наложение, объединение, обрезка, затухание, пакетная обработка. Используйте при: конвертировании форматов видео/аудио, кодировании с помощью определенных кодеков, создании миниатюр, создании манифестов потоковой передачи, извлечении аудио из видео, пакетной обработке изображений, добавлении водяных знаков, оптимизации размеров файлов.
