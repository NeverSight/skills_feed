Автоматизация браузера с помощью сценариев CLI Puppeteer (вывод JSON). Возможности: снимки экрана, создание PDF-файлов, очистка веб-страниц, автоматизация форм, мониторинг сети, профилирование производительности, отладка JavaScript, просмотр без монитора. Действия: снимок экрана, очистка, автоматизация, тестирование, профиль, мониторинг, отладка браузера. Ключевые слова: Puppeteer, Headless Chrome, скриншот, PDF, парсинг веб-страниц, заполнение формы, щелчок, навигация, сетевой трафик, аудит производительности, Lighthouse, журналы консоли, манипуляции с DOM, выбор элементов, ожидание, прокрутка, скрипт автоматизации. Используйте, когда: создание снимков экрана, создание PDF-файлов из Интернета, очистка веб-сайтов, автоматизация отправки форм, мониторинг сетевых запросов, профилирование производительности страниц, отладка JavaScript, тестирование веб-интерфейсов.
