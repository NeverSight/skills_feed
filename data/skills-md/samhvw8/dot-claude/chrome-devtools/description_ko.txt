Puppeteer CLI 스크립트(JSON 출력)를 통한 브라우저 자동화. 기능: 스크린샷, PDF 생성, 웹 스크래핑, 양식 자동화, 네트워크 모니터링, 성능 프로파일링, JavaScript 디버깅, 헤드리스 브라우징. 작업: 스크린샷, 스크랩, 자동화, 테스트, 프로파일링, 모니터링, 브라우저 디버그. 키워드: Puppeteer, 헤드리스 Chrome, 스크린샷, PDF, 웹 스크래핑, 양식 채우기, 클릭, 탐색, 네트워크 트래픽, 성능 감사, Lighthouse, 콘솔 로그, DOM 조작, 요소 선택기, 대기, 스크롤, 자동화 스크립트. 사용 시기: 스크린샷 찍기, 웹에서 PDF 생성, 웹사이트 스크랩, 양식 제출 자동화, 네트워크 요청 모니터링, 페이지 성능 프로파일링, JavaScript 디버깅, 웹 UI 테스트.
