Extensibilité du Code Claude : agents, compétences, styles de sortie. Capacités : créer/mettre à jour/supprimer des agents et des compétences, frontmatter YAML, invites système, sélection d'outils/modèles, agents pouvant être repris, agents définis par la CLI. Actions : créer, modifier, supprimer, optimiser, tester les extensions. Mots-clés : agent, compétence, style de sortie, SKILL.md, sous-agent, outil de tâche, divulgation progressive. À utiliser lors de : la création d'agents/compétences, la modification d'extensions, la configuration de l'accès aux outils, le choix de modèles, le test d'activation.
