Emballage de référentiel pour l'analyse AI/LLM. Capacités : regroupez les dépôts dans des fichiers uniques, générez un contexte adapté à l'IA, des instantanés de base de code, une préparation aux audits de sécurité, des modèles de filtrage/exclusion, un comptage de jetons, plusieurs formats de sortie. Actions : packer, générer, exporter, analyser les référentiels pour les LLM. Mots clés : Repomix, packaging de référentiel, contexte LLM, analyse IA, instantané de base de code, contexte Claude, contexte ChatGPT, contexte Gemini, packaging de code, nombre de jetons, filtrage de fichiers, audit de sécurité, analyse de bibliothèque tierce, fenêtre contextuelle, sortie de fichier unique. À utiliser lorsque : empaqueter des bases de code pour l'IA, générer un contexte LLM, créer des instantanés de base de code, analyser des bibliothèques tierces, préparer des audits de sécurité, alimenter des dépôts vers Claude/ChatGPT/Gemini.
