Gestionnaire d'environnement de développement Mise (asdf + direnv + make remplacement). Capacités : gestion des versions d'outils (node, python, go, ruby, rust), variables d'environnement, exécuteurs de tâches, configurations locales du projet. Actions : installer, gérer, configurer, exécuter des outils/tâches avec mise. Mots-clés : mise, mise.toml, version de l'outil, version d'exécution, nœud, python, go, ruby, rust, asdf, direnv, exécuteur de tâches, variables d'environnement, gestionnaire de versions, .tool-versions, mise install, mise use, mise run, mise tâches, configuration du projet, configuration globale. À utiliser lors de : l'installation de versions d'exécution, la gestion des versions d'outils, la configuration d'environnements de développement, la création d'exécuteurs de tâches, le remplacement d'asdf/direnv/make, la configuration d'outils locaux du projet.
