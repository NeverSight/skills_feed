Менеджер среды разработки Mise (asdf + direnv + make replace). Возможности: управление версиями инструментов (node, python, go, Ruby, Rust), переменные среды, средства запуска задач, локальные конфигурации проекта. Действия: установка, управление, настройка, запуск инструментов/задач с помощью Mise. Ключевые слова: Mise, mise.toml, версия инструмента, версия времени выполнения, узел, Python, go, Ruby, Rust, asdf, direnv, средство запуска задач, переменные среды, менеджер версий, .tool-версии, установка MISE, использование MISE, запуск MISE, задачи MISE, конфигурация проекта, глобальная конфигурация. Используйте при: установке версий среды выполнения, управлении версиями инструментов, настройке сред разработки, создании средств запуска задач, замене asdf/direnv/make, настройке локальных инструментов проекта.
