Administration de bases de données MongoDB et PostgreSQL. Bases de données : MongoDB (magasin de documents, agrégation, Atlas), PostgreSQL (relationnel, SQL, psql). Capacités : conception de schémas, optimisation des requêtes, indexation, migrations, réplication, partitionnement, sauvegarde/restauration, gestion des utilisateurs, analyse des performances. Actions : concevoir, interroger, optimiser, migrer, sauvegarder, restaurer, indexer, fragmenter des bases de données. Mots clés : MongoDB, PostgreSQL, SQL, NoSQL, BSON, pipeline d'agrégation, Atlas, psql, pgAdmin, conception de schéma, index, optimisation de requêtes, EXPLAIN, réplication, partitionnement, sauvegarde, restauration, migration, ORM, Prisma, Mongoose, pooling de connexions, transactions, ACID. À utiliser dans les cas suivants : conception de schémas de base de données, écriture de requêtes complexes, optimisation des performances des requêtes, création d'index, réalisation de migrations, configuration de la réplication, mise en œuvre de stratégies de sauvegarde, gestion des autorisations de base de données, dépannage de requêtes lentes.
