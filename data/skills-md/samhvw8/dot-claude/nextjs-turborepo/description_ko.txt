Next.js 및 Turborepo를 사용한 풀스택 웹 개발. 스택: Next.js 14+(앱 라우터, RSC, 서버 작업, PPR, SSR, SSG, ISR), Turborepo(모노레포, 파이프라인, 원격 캐싱), RemixIcon(3100개 이상의 아이콘). 기능: 서버 구성 요소, API 경로, 미들웨어, 캐싱 전략, 빌드 최적화, 모노레포 관리. 작업: Next.js 앱 생성, 빌드, 배포, 최적화, 단일 저장소 설정, 캐싱 구성. 키워드: Next.js, 앱 라우터, 서버 구성 요소, RSC, 서버 작업, SSR, SSG, ISR, PPR, Turborepo, monorepo, 원격 캐시, 빌드 파이프라인, 병렬 실행, 작업 공간, pnpm, 아이콘. 사용 시기: Next.js 앱 빌드, SSR/SSG 구현, 단일 저장소 설정, 빌드 성능 최적화, 캐싱 전략 구성, 공유 종속성 관리.
