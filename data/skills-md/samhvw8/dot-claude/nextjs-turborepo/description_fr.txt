Développement web full-stack avec Next.js et Turborepo. Pile : Next.js 14+ (Routeur d'applications, RSC, Actions de serveur, PPR, SSR, SSG, ISR), Turborepo (monorepo, pipelines, mise en cache à distance), RemixIcon (plus de 3 100 icônes). Capacités : composants serveur, routes API, middleware, stratégies de mise en cache, optimisation de build, gestion monorepo. Actions : créer, créer, déployer, optimiser les applications Next.js, configurer le monorepo, configurer la mise en cache. Mots-clés : Next.js, App Router, Server Components, RSC, Server Actions, SSR, SSG, ISR, PPR, Turborepo, monorepo, cache distant, pipeline de build, exécution parallèle, espace de travail, pnpm, icônes. À utiliser lors de : la création d'applications Next.js, la mise en œuvre de SSR/SSG, la configuration de monorepos, l'optimisation des performances de build, la configuration de stratégies de mise en cache, la gestion des dépendances partagées.
