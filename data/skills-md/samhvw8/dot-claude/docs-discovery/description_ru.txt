Поиск технической документации через context7 и веб-поиск. Возможности: поиск документов библиотеки/фреймворка, поиск по конкретной теме. Ключевые слова: llms.txt, context7, документация, документация библиотеки, документация API. Используйте при: поиске документации по библиотеке, поиске руководств по фреймворку, поиске ссылок на API.
