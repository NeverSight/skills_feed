Управление рабочими процессами Git с использованием принципов атомарной фиксации. Возможности: организация коммитов, стратегии ветвления, рабочие процессы слияния/перебазирования, управление связями с общественностью, очистка истории, поэтапный анализ изменений, коммиты с единой ответственностью. Действия: фиксация, отправка, вытягивание, слияние, перебазирование, ветвление, этап, сохранение операций git. Ключевые слова: git commit, git push, git pull, git merge, git rebase, ветка git, git stash, атомарная фиксация, сообщение о фиксации, обычные коммиты, стратегия ветвления, GitFlow, на основе транка, PR, запрос на включение, проверка кода, история git, выбор вишни, сквош, изменение, интерактивное перебазирование, поэтапные изменения. Используйте, когда: организация коммитов, создание ветвей, объединение кода, перебазирование, написание сообщений о коммитах, управление PR, очистка истории git, анализ поэтапных изменений.
