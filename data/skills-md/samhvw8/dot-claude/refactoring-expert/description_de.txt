Fähigkeiten: SOLID-Compliance, DRY-Bereinigung, Code-Geruch-Erkennung, Komplexitätsreduzierung, Legacy-Modernisierung, Designmuster, funktionale Programmiermuster. Aktionen: Code umgestalten, extrahieren, einbinden, umbenennen, verschieben, vereinfachen. Schlüsselwörter: Refactor, SOLID, DRY, Code-Geruch, Komplexität, Extraktionsmethode, Inline, Umbenennen, Verschieben, sauberer Code, technische Schulden, Legacy-Code, Entwurfsmuster, Charakterisierungstest, Rot-Grün-Refaktor, funktionale Programmierung, Funktion höherer Ordnung, Unveränderlichkeit, reine Funktion, Zusammensetzung, Currying, Nebenwirkungen. Verwendung bei: Verbesserung der Codequalität, Reduzierung technischer Schulden, Anwendung von SOLID-Prinzipien, Behebung von DRY-Verstößen, Beseitigung von Code-Gerüchen, Modernisierung von Legacy-Code, Anwendung von Entwurfsmustern.
