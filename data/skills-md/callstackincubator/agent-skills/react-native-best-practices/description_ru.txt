Содержит рекомендации по оптимизации производительности React Native для FPS, TTI, размера пакета, утечек памяти, повторного рендеринга и анимации. Применяется к задачам, связанным с оптимизацией Hermes, блокировкой потоков JS, накладными расходами моста, FlashList, собственными модулями или отладкой зависаний и выпадений кадров.
