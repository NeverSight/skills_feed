عندما يريد المستخدم كتابة نسخة لمواقع التجارة الإلكترونية أو المتاجر الإلكترونية أو المتاجر عبر الإنترنت. يُستخدم عند إنشاء صفحات المنتج، أو أوصاف المنتج، أو تدفقات الخروج، أو صفحات الفئات، أو رسائل البريد الإلكتروني الخاصة بالمعاملات، أو نسخة سلة التسوق، أو أي نسخة متجر عبر الإنترنت. استخدمه أيضًا عندما يذكر المستخدم وصف المنتج، أو نسخة الخروج، أو نسخة التجارة الإلكترونية، أو نسخة متجر الويب، أو البريد الإلكتروني لسلة التسوق المهجورة، أو تأكيد الطلب. يغطي أوصاف المنتجات الفردية والمجمّعة، ونسخة UX لعربة التسوق/الخروج، وإشارات الثقة، والنسخة الترويجية، والمحتوى التعريفي لتحسين محركات البحث، وتسلسلات البريد الإلكتروني.
