نشر وتشغيل خوادم وكيل الإنتاج باستخدام LangSmith Deployment. يُستخدم عندما يتضمن العمل اختيار Cloud vs Hybrid/Self-hosted-with-control-plane vs Standalone، أو إعداد/التحقق من صحة langgraph.json، أو إنشاء عمليات نشر أو مراجعات، أو التراجع عن المراجعات، أو توصيل CI/CD بواجهات برمجة تطبيقات مستوى التحكم، أو تكوين متغيرات البيئة والأسرار، أو إعداد المراقبة/التنبيهات/خطافات الويب، أو استكشاف مشكلات النشر/وقت التشغيل/القياس وإصلاحها لتطبيقات LangChain/LangGraph.
