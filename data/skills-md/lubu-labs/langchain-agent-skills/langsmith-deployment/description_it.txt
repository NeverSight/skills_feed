Distribuisci e gestisci server agenti di produzione con LangSmith Deployment. Da utilizzare quando il lavoro prevede la scelta del cloud o ibrido/self-hosted con piano di controllo o autonomo, la preparazione/convalida di langgraph.json, la creazione di distribuzioni o revisioni, il rollback delle revisioni, il collegamento di CI/CD alle API del piano di controllo, la configurazione di variabili e segreti di ambiente, l'impostazione di monitoraggio/avvisi/webhook o la risoluzione dei problemi di distribuzione/runtime/ridimensionamento per le applicazioni LangChain/LangGraph.
