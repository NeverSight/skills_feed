Initialisez et configurez les projets LangGraph avec une structure appropriée, une configuration langgraph.json, des variables d'environnement et une gestion des dépendances. À utiliser lorsque les utilisateurs souhaitent (1) créer un nouveau projet LangGraph, (2) configurer langgraph.json pour le déploiement, (3) configurer les variables d'environnement pour les fournisseurs LLM, (4) initialiser la structure du projet pour les agents, (5) configurer le développement local avec LangGraph Studio, (6) configurer les dépendances (pyproject.toml, conditions.txt, package.json) ou (7) résoudre les problèmes de configuration du projet.
