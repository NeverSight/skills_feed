Инициализируйте и настройте проекты LangGraph с правильной структурой, конфигурацией langgraph.json, переменными среды и управлением зависимостями. Используйте, когда пользователи хотят (1) создать новый проект LangGraph, (2) настроить langgraph.json для развертывания, (3) настроить переменные среды для поставщиков LLM, (4) инициализировать структуру проекта для агентов, (5) настроить локальную разработку с помощью LangGraph Studio, (6) настроить зависимости (pyproject.toml, require.txt, package.json) или (7) устранить проблемы с конфигурацией проекта.
