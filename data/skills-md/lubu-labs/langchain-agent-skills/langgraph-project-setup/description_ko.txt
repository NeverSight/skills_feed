적절한 구조, langgraph.json 구성, 환경 변수 및 종속성 관리를 사용하여 LangGraph 프로젝트를 초기화하고 구성합니다. 사용자가 (1) 새 LangGraph 프로젝트 생성, (2) 배포용 langgraph.json 설정, (3) LLM 제공자를 위한 환경 변수 구성, (4) 에이전트용 프로젝트 구조 초기화, (5) LangGraph Studio를 사용한 로컬 개발 설정, (6) 종속성 구성(pyproject.toml, 요구 사항.txt, package.json) 또는 (7) 프로젝트 구성 문제 해결을 원할 때 사용합니다.
