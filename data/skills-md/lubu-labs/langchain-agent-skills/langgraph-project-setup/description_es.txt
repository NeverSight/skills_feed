Inicialice y configure proyectos LangGraph con la estructura adecuada, configuración de langgraph.json, variables de entorno y gestión de dependencias. Úselo cuando los usuarios quieran (1) crear un nuevo proyecto LangGraph, (2) configurar langgraph.json para la implementación, (3) configurar variables de entorno para proveedores de LLM, (4) inicializar la estructura del proyecto para agentes, (5) configurar el desarrollo local con LangGraph Studio, (6) configurar dependencias (pyproject.toml, requisitos.txt, paquete.json) o (7) solucionar problemas de configuración del proyecto.
