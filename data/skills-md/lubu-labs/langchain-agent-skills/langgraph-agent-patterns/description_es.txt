Implemente patrones de coordinación de múltiples agentes (supervisor-subagente, enrutador, orquestador-trabajador, traspasos) para aplicaciones LangGraph. Úselo cuando los usuarios quieran (1) implementar sistemas de múltiples agentes, (2) coordinar múltiples agentes especializados, (3) elegir entre patrones de coordinación, (4) configurar flujos de trabajo de supervisor y subagente, (5) implementar la selección de agentes basada en enrutador, (6) crear patrones paralelos de orquestador-trabajador, (7) implementar transferencias de agentes, (8) diseñar esquemas de estado para sistemas de múltiples agentes o (9) depurar problemas de coordinación de múltiples agentes.
