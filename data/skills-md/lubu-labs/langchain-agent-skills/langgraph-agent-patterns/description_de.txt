Implementieren Sie Multi-Agent-Koordinationsmuster (Supervisor-Subagent, Router, Orchestrator-Worker, Übergaben) für LangGraph-Anwendungen. Wird verwendet, wenn Benutzer (1) Multiagentensysteme implementieren, (2) mehrere spezialisierte Agenten koordinieren, (3) zwischen Koordinationsmustern wählen, (4) Supervisor-Subagenten-Workflows einrichten, (5) eine routerbasierte Agentenauswahl implementieren, (6) parallele Orchestrator-Worker-Muster erstellen, (7) Agentenübergaben implementieren, (8) Zustandsschemata für Multiagentensysteme entwerfen oder (9) Probleme bei der Multiagentenkoordination beheben möchten.
