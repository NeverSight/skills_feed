Récupérez, organisez et analysez les traces LangSmith pour le débogage et l'évaluation. À utiliser lorsque vous devez : interroger des traces/exécutions par projet, métadonnées, statut ou fenêtre temporelle ; télécharger les traces vers JSON ; organiser les résultats en catégories de réussite/échec/erreur ; analyser les modèles de jeton/message/appel d'outil ; comparer les comportements réussis et échoués ; ou enquêter sur les échecs de référence et de production.
