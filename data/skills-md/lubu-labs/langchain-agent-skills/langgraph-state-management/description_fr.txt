Concevez des schémas d'état, implémentez des réducteurs, configurez la persistance et déboguez les problèmes d'état pour les applications LangGraph. À utiliser lorsque les utilisateurs souhaitent (1) concevoir ou définir des schémas d'état pour les graphiques LangGraph, (2) implémenter des fonctions de réduction pour l'accumulation d'état, (3) configurer la persistance avec des points de contrôle (InMemorySaver/MemorySaver, SqliteSaver, PostgresSaver), (4) déboguer des problèmes de mise à jour d'état ou un comportement d'état inattendu, (5) migrer des schémas d'état entre les versions, (6) valider la structure du schéma d'état, (7) choisir entre TypedDict et Les modèles MessagesState, (8) implémentent des réducteurs personnalisés pour les listes, les dicts ou les ensembles, (9) utilisent le type Overwrite pour contourner les réducteurs, (10) configurent la persistance basée sur les threads pour les conversations à plusieurs tours, ou (11) inspectent les points de contrôle pour le débogage.
