Diseñe esquemas de estado, implemente reductores, configure la persistencia y depure problemas de estado para aplicaciones LangGraph. Úselo cuando los usuarios quieran (1) diseñar o definir esquemas de estado para gráficos LangGraph, (2) implementar funciones reductoras para la acumulación de estado, (3) configurar la persistencia con punteros de control (InMemorySaver/MemorySaver, SqliteSaver, PostgresSaver), (4) depurar problemas de actualización de estado o comportamiento de estado inesperado, (5) migrar esquemas de estado entre versiones, (6) validar la estructura del esquema de estado, (7) elegir entre los patrones TypedDict y MessagesState, (8) implementar reductores personalizados para listas, dictados o conjuntos, (9) usar el tipo Sobrescribir para omitir reductores, (10) configurar la persistencia basada en subprocesos para conversaciones de varios turnos o (11) inspeccionar puntos de control para depuración.
