تنفيذ معالجة أخطاء LangGraph مع أنماط الإصدار 1 الحالية. يُستخدم عندما يحتاج المستخدمون إلى تصنيف حالات الفشل، أو إضافة RetryPolicy للمشكلات العابرة، أو إنشاء حلقات استرداد LLM باستخدام توجيه الأوامر، أو إضافة إنسان في الحلقة باستخدام المقاطعة ()/الاستئناف، أو التعامل مع أخطاء ToolNode، أو اختيار استراتيجية آمنة بين إعادة المحاولة والاسترداد والتصعيد.
