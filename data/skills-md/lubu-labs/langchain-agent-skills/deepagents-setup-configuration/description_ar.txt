قم بتهيئة مشاريع Deep Agents والتحقق من صحتها واستكشاف أخطائها وإصلاحها في Python أو JavaScript باستخدام الحزمة "deepagents". يُستخدم عندما يحتاج المستخدمون إلى إنشاء وكلاء باستخدام تخطيط/نظام ملفات/وكلاء فرعيين مدمجين، وتكوين البرامج الوسيطة/الواجهات الخلفية/checkpointing/HITL، والترحيل من `create_react_agent` أو `create_agent`، ومشاريع السقالة باستخدام البرامج النصية للريبو، والتحقق من صحة ملفات تكوين الوكيل، وتأكيد التوافق مع مستندات LangChain/LangGraph/LangSmith الحالية.
