「deepagents」パッケージを使用して、Python または JavaScript で Deep Agent プロジェクトを初期化、検証、トラブルシューティングします。ユーザーが、組み込みの計画/ファイルシステム/サブエージェントを使用してエージェントを作成し、ミドルウェア/バックエンド/チェックポイント/HITL を構成し、「create_react_agent」または「create_agent」から移行し、リポジトリ スクリプトを使用してプロジェクトをスキャフォールディングし、エージェント構成ファイルを検証し、現在の LangChain/LangGraph/LangSmith ドキュメントとの互換性を確認する必要がある場合に使用します。
