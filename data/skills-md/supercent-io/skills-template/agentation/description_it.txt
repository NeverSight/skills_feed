Strumento di annotazione dell'interfaccia utente visiva per agenti IA. Trascina la barra degli strumenti React in qualsiasi app: gli utenti fanno clic sugli elementi e lasciano feedback, gli agenti ricevono selettori CSS strutturati, riquadri di delimitazione e alberi dei componenti React per trovare il codice esatto. Supporta watch-loop MCP, hook specifici della piattaforma (Claude Code/Codex/Gemini CLI/OpenCode), distribuzione di webhook e critica di guida autonoma con agent-browser.
