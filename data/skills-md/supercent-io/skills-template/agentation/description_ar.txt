أداة التعليقات التوضيحية لواجهة المستخدم المرئية لوكلاء الذكاء الاصطناعي. أسقط شريط أدوات React في أي تطبيق — ينقر البشر على العناصر ويتركون تعليقات، ويتلقى الوكلاء محددات CSS منظمة ومربعات محيطة وأشجار مكونات React للعثور على التعليمات البرمجية الدقيقة. يدعم حلقة المراقبة MCP، والخطافات الخاصة بالمنصة (Claude Code / Codex / Gemini CLI / OpenCode)، وتسليم الخطاف عبر الويب، والنقد الذاتي القيادة باستخدام متصفح الوكيل.
