Herramienta de anotación de interfaz de usuario visual para agentes de IA. Coloque la barra de herramientas de React en cualquier aplicación: los humanos hacen clic en los elementos y dejan comentarios, los agentes reciben selectores de CSS estructurados, cuadros delimitadores y árboles de componentes de React para encontrar el código exacto. Admite bucle de vigilancia MCP, ganchos específicos de plataforma (Claude Code / Codex / Gemini CLI / OpenCode), entrega de webhooks y crítica autónoma con navegador de agente.
