Инструмент аннотаций визуального пользовательского интерфейса для агентов ИИ. Перетащите панель инструментов React в любое приложение — люди щелкают элементы и оставляют отзывы, агенты получают структурированные селекторы CSS, ограничивающие рамки и деревья компонентов React для поиска точного кода. Поддерживает цикл наблюдения MCP, перехватчики для конкретной платформы (Claude Code / Codex / Gemini CLI / OpenCode), доставку веб-перехватчиков и автономную самостоятельную критику с помощью агента-браузера.
