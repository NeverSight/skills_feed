Outil d'annotation visuelle de l'interface utilisateur pour les agents IA. Déposez la barre d'outils React dans n'importe quelle application : les humains cliquent sur les éléments et laissent des commentaires, les agents reçoivent des sélecteurs CSS structurés, des cadres de délimitation et des arborescences de composants React pour trouver le code exact. Prend en charge la boucle de surveillance MCP, les hooks spécifiques à la plate-forme (Claude Code / Codex / Gemini CLI / OpenCode), la livraison de webhooks et la critique de conduite autonome autonome avec un navigateur d'agent.
