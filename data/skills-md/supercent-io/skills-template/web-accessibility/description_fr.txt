Mettre en œuvre les normes d’accessibilité Web (a11y) conformément aux directives WCAG 2.1. À utiliser pour créer des interfaces utilisateur accessibles, résoudre des problèmes d'accessibilité ou garantir le respect des normes en matière de handicap. Gère les attributs ARIA, la navigation au clavier, les lecteurs d'écran, le HTML sémantique et les tests d'accessibilité.
