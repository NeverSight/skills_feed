Implementieren Sie Web-Barrierefreiheitsstandards (a11y) gemäß den WCAG 2.1-Richtlinien. Verwenden Sie es, wenn Sie barrierefreie Benutzeroberflächen erstellen, Barrierefreiheitsprobleme beheben oder die Einhaltung von Behindertenstandards sicherstellen. Verarbeitet ARIA-Attribute, Tastaturnavigation, Bildschirmleseprogramme, semantisches HTML und Barrierefreiheitstests.
