Внедрите стандарты веб-доступности (a11y) в соответствии с рекомендациями WCAG 2.1. Используйте при создании доступных пользовательских интерфейсов, устранении проблем с доступностью или обеспечении соответствия стандартам для людей с ограниченными возможностями. Обрабатывает атрибуты ARIA, навигацию с помощью клавиатуры, программы чтения с экрана, семантический HTML и тестирование доступности.
