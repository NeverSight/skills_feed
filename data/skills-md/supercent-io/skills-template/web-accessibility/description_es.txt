Implementar estándares de accesibilidad web (a11y) siguiendo las pautas WCAG 2.1. Úselo al crear interfaces de usuario accesibles, solucionar problemas de accesibilidad o garantizar el cumplimiento de los estándares de discapacidad. Maneja atributos ARIA, navegación con teclado, lectores de pantalla, HTML semántico y pruebas de accesibilidad.
