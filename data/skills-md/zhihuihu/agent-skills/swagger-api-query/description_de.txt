Fragen und analysieren Sie große Swagger/OpenAPI-JSON-Dokumente bei Bedarf, um Endpunktdetails und Schemata zu extrahieren, ohne die gesamte Spezifikation zu laden. Zur API-Erkennung, Suche nach Tag/Schlüsselwort, Verständnis von Anforderungs-/Antwortstrukturen und Nachverfolgung von Datenmodellen. Wird aufgerufen, wenn der Benutzer „einen Endpunkt finden“, „nach Anmelde-/Upload-/Benutzer-APIs suchen“ oder „ein Schema erklären“ fragt. Verwenden Sie --spec, um den Dokumentpfad anzugeben, wenn er sich nicht im aktuellen Verzeichnis befindet, und --format json für eine strukturierte Ausgabe.
