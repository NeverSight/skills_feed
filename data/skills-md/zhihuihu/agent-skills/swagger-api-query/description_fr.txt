Interrogez et analysez des documents Swagger/OpenAPI JSON volumineux à la demande pour extraire les détails et les schémas des points de terminaison sans charger l'intégralité des spécifications. À utiliser pour la découverte d'API, la recherche par balise/mot-clé, la compréhension des structures de requête/réponse et le traçage des modèles de données. À appeler lorsque l'utilisateur demande de « trouver un point de terminaison », de « rechercher des API de connexion/téléchargement/utilisateur » ou « d'expliquer un schéma ». Utilisez --spec pour spécifier le chemin du document s'il ne se trouve pas dans le répertoire actuel et --format json pour une sortie structurée.
