Запрашивайте и анализируйте большие документы Swagger/OpenAPI JSON по требованию, чтобы извлечь сведения и схемы конечных точек без загрузки всей спецификации. Используйте для обнаружения API, поиска по тегу/ключевому слову, понимания структур запроса/ответа и отслеживания моделей данных. Вызывается, когда пользователь просит «найти конечную точку», «найти API входа/загрузки/пользователя» или «объяснить схему». Используйте --spec, чтобы указать путь к документу, если он не находится в текущем каталоге, и --format json для структурированного вывода.
