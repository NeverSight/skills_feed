Esegui query e analizza documenti JSON Swagger/OpenAPI di grandi dimensioni on-demand per estrarre dettagli e schemi dell'endpoint senza caricare l'intera specifica. Utilizzare per l'individuazione delle API, la ricerca per tag/parola chiave, la comprensione delle strutture di richiesta/risposta e il tracciamento dei modelli di dati. Richiamare quando l'utente chiede di "trovare un endpoint", "cercare login/caricamento/API utente" o "spiegare uno schema". Utilizza --spec per specificare il percorso del documento se non si trova nella directory corrente e --format json per l'output strutturato.
