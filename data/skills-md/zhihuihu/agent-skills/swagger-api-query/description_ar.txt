يمكنك الاستعلام عن مستندات Swagger/OpenAPI JSON الكبيرة وتحليلها عند الطلب لاستخراج تفاصيل ومخططات نقطة النهاية دون تحميل المواصفات بأكملها. يُستخدم لاكتشاف واجهة برمجة التطبيقات، والبحث حسب العلامة/الكلمة الرئيسية، وفهم هياكل الطلب/الاستجابة، وتتبع نماذج البيانات. يتم الاستدعاء عندما يطلب المستخدم "العثور على نقطة نهاية"، أو "البحث عن واجهات برمجة تطبيقات تسجيل الدخول/التحميل/المستخدم"، أو "شرح المخطط". استخدم --spec لتحديد مسار المستند إذا لم يكن موجودًا في الدليل الحالي، و-format json للإخراج المنظم.
