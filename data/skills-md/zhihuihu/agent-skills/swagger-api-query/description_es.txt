Consulte y analice grandes documentos Swagger/OpenAPI JSON bajo demanda para extraer detalles y esquemas de endpoints sin cargar todas las especificaciones. Úselo para el descubrimiento de API, la búsqueda por etiqueta/palabra clave, la comprensión de las estructuras de solicitud/respuesta y el seguimiento de modelos de datos. Invocar cuando el usuario solicita "encontrar un punto final", "buscar API de inicio de sesión/carga/usuario" o "explicar un esquema". Utilice --spec para especificar la ruta del documento si no está en el directorio actual y --format json para una salida estructurada.
