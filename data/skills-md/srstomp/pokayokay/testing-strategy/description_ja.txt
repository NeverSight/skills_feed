API テストを超えた包括的なテスト アーキテクチャ、カバレッジ戦略、テスト設計。テスト ピラミッド設計、フロントエンド/コンポーネント テスト (React Testing Library、Vue Test Utils)、E2E テスト (Playwright、Cypress)、ビジュアル リグレッション、モッキング戦略 (MSW)、不安定なテストの防止をカバーします。このスキルは、テスト アーキテクチャの設計、使用するテスト タイプの決定、コンポーネントまたは E2E テストのセットアップ、不安定なテストのデバッグ、カバレッジ戦略のレビュー、またはテスト ファイルの整理に使用します。 「テスト戦略」、「何をテストするか」、「カバレッジ」、「e2e」、「プレイライト」、「サイプレス」、「コンポーネント テスト」、「不安定なテスト」、「テスト ピラミッド」に関するトリガー。
