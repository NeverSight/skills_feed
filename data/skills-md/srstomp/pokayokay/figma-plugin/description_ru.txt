Создавайте плагины Figma с помощью API плагинов. Охватывает архитектуру плагинов (песочница + пользовательский интерфейс), манипулирование узлами, стили, компоненты, разработку пользовательского интерфейса с использованием связи postMessage и публикацию. Используйте этот навык при создании плагинов Figma, работе с API плагинов Figma или создании инструментов автоматизации проектирования.
