Erstellen, extrahieren und veröffentlichen Sie TypeScript-SDKs. Behandelt die Analyse vorhandener Anwendungen, um wiederverwendbare Logik zu extrahieren, das Entwerfen sauberer SDK-APIs, die Implementierung typisierter Clients mit ordnungsgemäßer Fehlerbehandlung, die Bündelung für mehrere Ziele (ESM/CJS/Browser) und die Veröffentlichung in npm (öffentliche oder private Register). Nutzen Sie diese Fähigkeit, wenn Sie SDKs erstellen, gemeinsam genutzten Code in Pakete extrahieren oder Entwickler-Tooling-Bibliotheken erstellen.
