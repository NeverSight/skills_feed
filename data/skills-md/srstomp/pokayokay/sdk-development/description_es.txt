Cree, extraiga y publique SDK de TypeScript. Cubre el análisis de aplicaciones existentes para extraer lógica reutilizable, el diseño de API de SDK limpias, la implementación de clientes tipificados con un manejo adecuado de errores, la agrupación para múltiples objetivos (ESM/CJS/navegador) y la publicación en npm (registros públicos o privados). Utilice esta habilidad al crear SDK, extraer código compartido en paquetes o crear bibliotecas de herramientas para desarrolladores.
