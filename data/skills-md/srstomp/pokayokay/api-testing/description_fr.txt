Testez les API avec des tests d'intégration, des tests contractuels et une validation E2E. Couvre Jest, Vitest et Supertest pour les API Node.js/TypeScript. Comprend la gestion des données de test, les montages, les usines, la configuration de l'environnement, l'intégration CI/CD, les services externes simulés et les tests sous contrat avec validation OpenAPI. Utilisez cette compétence lors de la création de suites de tests pour les API REST, de la validation de contrats d'API ou de la configuration d'une infrastructure de test d'API.
