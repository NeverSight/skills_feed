Тестируйте API с помощью интеграционных тестов, контрактных тестов и проверки E2E. Охватывает Jest, Vitest и Supertest для API Node.js/TypeScript. Включает управление тестовыми данными, приспособления, фабрики, конфигурацию среды, интеграцию CI/CD, имитацию внешних сервисов и контрактное тестирование с проверкой OpenAPI. Используйте этот навык при создании наборов тестов для REST API, проверке контрактов API или настройке инфраструктуры тестирования API.
