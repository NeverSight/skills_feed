Создавайте API-интерфейсы RESTful с использованием согласованных шаблонов, четких соглашений и подробной документации. Охватывает проектирование конечных точек, методы HTTP, коды состояния, форматы запросов/ответов, нумерацию страниц, фильтрацию, управление версиями, аутентификацию и спецификации OpenAPI. Используйте этот навык при разработке новых API, анализе проектов API или установлении стандартов API для проекта или организации.
