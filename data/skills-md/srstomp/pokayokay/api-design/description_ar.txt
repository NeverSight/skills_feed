صمم واجهات برمجة تطبيقات RESTful بأنماط متسقة واصطلاحات واضحة ووثائق شاملة. يغطي تصميم نقطة النهاية وطرق HTTP ورموز الحالة وتنسيقات الطلب/الاستجابة وترقيم الصفحات والتصفية وتحديد الإصدارات والمصادقة ومواصفات OpenAPI. استخدم هذه المهارة عند تصميم واجهات برمجة التطبيقات الجديدة، أو مراجعة تصميمات واجهة برمجة التطبيقات، أو إنشاء معايير واجهة برمجة التطبيقات لمشروع أو مؤسسة.
