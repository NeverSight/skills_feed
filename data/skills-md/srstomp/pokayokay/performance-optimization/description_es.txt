Análisis de rendimiento, técnicas de optimización y pruebas de rendimiento para aplicaciones web. Cubre la creación de perfiles, la identificación de cuellos de botella, la optimización del frontend (tamaño del paquete, renderizado, Core Web Vitals), la optimización del backend (optimización de consultas, almacenamiento en caché, patrones asíncronos), pruebas de carga (k6, Artillery) y monitoreo (presupuestos de rendimiento, SLI). Utilice esta habilidad al analizar problemas de rendimiento, optimizar páginas o API lentas, configurar pruebas de carga, implementar almacenamiento en caché, reducir el tamaño de los paquetes o establecer presupuestos de rendimiento.
