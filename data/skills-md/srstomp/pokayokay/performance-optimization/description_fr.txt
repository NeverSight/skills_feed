Analyse des performances, techniques d'optimisation et tests de performances pour les applications Web. Couvre le profilage, l'identification des goulots d'étranglement, l'optimisation du front-end (taille du bundle, rendu, Core Web Vitals), l'optimisation du back-end (optimisation des requêtes, mise en cache, modèles asynchrones), les tests de charge (k6, Artillery) et la surveillance (budgets de performances, SLI). Utilisez cette compétence pour analyser les problèmes de performances, optimiser les pages ou les API lentes, configurer les tests de charge, mettre en œuvre la mise en cache, réduire la taille des bundles ou établir des budgets de performances. Déclencheurs sur « performances », « lent », « optimiser », « taille du bundle », « tests de charge », « cache », « goulot d'étranglement », « latence », « Core Web Vitals », « LCP », « FCP », « phare », « profilage ».
