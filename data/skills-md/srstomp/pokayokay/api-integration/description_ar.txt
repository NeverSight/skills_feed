استخدم واجهات برمجة التطبيقات الخارجية مع أمان النوع والتعامل القوي مع الأخطاء وأنماط جاهزة للإنتاج. يتعامل مع مواصفات OpenAPI/Swagger ومخططات GraphQL ووثائق REST وطلبات الأمثلة. يُنتج عملاء مكتوبين باستخدام المصادقة ومنطق إعادة المحاولة ومعالجة شاملة للأخطاء. التركيز الأساسي على TypeScript مع الأنماط المطبقة على اللغات الأخرى. استخدم هذه المهارة عند دمج واجهات برمجة التطبيقات التابعة لجهات خارجية، أو إنشاء عملاء واجهة برمجة التطبيقات، أو تنفيذ تدفقات المصادقة.
