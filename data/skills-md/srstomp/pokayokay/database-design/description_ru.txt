Проектируйте схемы баз данных, планируйте миграции, оптимизируйте запросы и управляйте моделями данных. Охватывает реляционные базы данных (PostgreSQL, MySQL, SQLite), базы данных документов (MongoDB) и интеграцию ORM (Prisma, Drizzle, TypeORM). Используйте этот навык при разработке схем, просмотре моделей данных, планировании миграции, оптимизации медленных запросов или создании шаблонов баз данных для проекта. Триггеры по «базе данных», «схеме», «миграции», «модели», «оптимизации запросов», «индексу», «нормализации/денормализации».
