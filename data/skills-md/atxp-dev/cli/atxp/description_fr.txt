Accédez aux outils API payants ATXP pour la recherche sur le Web, la génération d'images IA, la création musicale, la génération de vidéos et la recherche X/Twitter. À utiliser lorsque les utilisateurs ont besoin d'une recherche sur le Web en temps réel, de médias générés par l'IA (images, musique, vidéo) ou d'une recherche X/Twitter. Nécessite une authentification via `npx atxp login`.
