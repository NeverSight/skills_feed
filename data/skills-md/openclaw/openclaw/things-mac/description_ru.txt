Управляйте Things 3 через интерфейс командной строки «things» в macOS (добавление/обновление проектов и задач через схему URL-адресов; чтение/поиск/список из локальной базы данных Things). Используйте, когда пользователь просит OpenClaw добавить задачу в Things, перечислить входящие/сегодняшние/предстоящие, выполнить поиск задач или проверить проекты/области/теги.
