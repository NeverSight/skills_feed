قم بإحضار وثائق المكتبة المحدثة عبر context7 REST API. يُستخدم عند الحاجة إلى مستندات API الحالية أو أنماط إطار العمل أو أمثلة التعليمات البرمجية لأي مكتبة. يُستخدم عندما يسأل المستخدم عن React أو Next.js أو Prisma أو Express أو Vue أو Angular أو Svelte أو أي حزمة npm/PyPI. يُستخدم عندما يقول المستخدم "كيف يمكنني استخدام مكتبة X"، أو "ما هي واجهة برمجة التطبيقات لـ Y"، أو يحتاج إلى وثائق رسمية. بديل خفيف الوزن لـ context7 MCP بدون حمل سياق مستمر.
