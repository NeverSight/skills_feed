قم بإنشاء واجهات أمامية على مستوى الإنتاج باستخدام تجربة مستخدم قوية وحرفية مرئية. يُستخدم عند إنشاء مكونات الويب أو الصفحات أو لوحات المعلومات أو النماذج أو الصفحات المقصودة أو أي واجهة مستخدم. يُستخدم عندما يقول المستخدم "إنشاء نموذج"، أو "إنشاء لوحة معلومات"، أو "تصميم مكون"، أو "إنشاء صفحة مقصودة"، أو يطلب عمل واجهة المستخدم/تجربة المستخدم.
