Intelligence sulle dipendenze JVM tramite il server MCP Maven Tools. Da utilizzare quando l'utente richiede informazioni su dipendenze, versioni, aggiornamenti, CVE o licenze Java/Kotlin/Scala. Da utilizzare durante l'analisi di pom.xml, build.gradle o qualsiasi dipendenza di Maven Central. Da utilizzare quando l'utente dice "controlla le mie dipendenze", "devo aggiornare X", "questa versione è sicura" o "qual è l'ultima versione di Y".
