Intelligence des dépendances JVM via le serveur Maven Tools MCP. À utiliser lorsque l'utilisateur pose des questions sur les dépendances, les versions, les mises à niveau, les CVE ou les licences Java/Kotlin/Scala. À utiliser lors de l'analyse de pom.xml, build.gradle ou de toute dépendance de Maven Central. À utiliser lorsque l'utilisateur demande « vérifier mes dépendances », « dois-je mettre à niveau X », « cette version est-elle sûre » ou « quelle est la dernière version de Y ».
