Анализ зависимостей JVM через сервер Maven Tools MCP. Используйте, когда пользователь спрашивает о зависимостях, версиях, обновлениях, CVE или лицензиях Java/Kotlin/Scala. Используйте при анализе pom.xml, build.gradle или любой зависимости Maven Central. Используйте, когда пользователь говорит: «проверьте мои зависимости», «должен ли я обновить X», «безопасна ли эта версия» или «какая последняя версия Y».
