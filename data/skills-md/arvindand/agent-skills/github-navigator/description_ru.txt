Операции GitHub через gh CLI. ВАЖНО: Всегда используйте вместо WebFetch для ЛЮБОГО URL-адреса github.com. Используйте, когда пользователь предоставляет URL-адрес GitHub, говорит «facebook/react», «показать README», «составить список проблем», «проверить PR», «клонировать репозиторий», «проанализировать этот репозиторий», «понять архитектуру», «как структурирован X», «изучить кодовую базу». Для глубокого анализа внешних репозиториев клонируйте их локально.
