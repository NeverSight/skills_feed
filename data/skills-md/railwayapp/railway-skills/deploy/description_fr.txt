Cette compétence doit être utilisée lorsque l'utilisateur souhaite transmettre du code au chemin de fer, par exemple "chemin de fer vers le haut", "déploiement", "déploiement sur le chemin de fer", "navire" ou "pousser". Pour la configuration initiale ou la création de services, utilisez une nouvelle compétence. Pour les images Docker, utilisez la compétence d'environnement.
