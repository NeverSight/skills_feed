Esta habilidad debe usarse cuando el usuario pregunta sobre el estado del servicio, desea cambiar el nombre de un servicio, cambiar los íconos del servicio, vincular servicios o crear servicios con imágenes de Docker. Para crear servicios con código local, prefiera la habilidad "nueva". Para las fuentes del repositorio de GitHub, use la habilidad "nueva" para crear un servicio vacío y luego la habilidad "entorno" para configurar la fuente.
