يجب استخدام هذه المهارة عندما يسأل المستخدم عن حالة الخدمة، أو يريد إعادة تسمية خدمة، أو تغيير أيقونات الخدمة، أو ربط الخدمات، أو إنشاء خدمات باستخدام صور Docker. لإنشاء خدمات باستخدام كود محلي، تفضل المهارة "الجديدة". بالنسبة لمصادر GitHub repo، استخدم مهارة "جديدة" لإنشاء خدمة فارغة ثم مهارة "البيئة" لتكوين المصدر.
