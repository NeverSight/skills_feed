Cette compétence doit être utilisée lorsque l'utilisateur souhaite ajouter une base de données (Postgres, Redis, MySQL, MongoDB), dit "ajouter postgres", "ajouter redis", "ajouter une base de données", "se connecter à la base de données" ou "connecter la base de données". Pour les autres modèles (Ghost, Strapi, n8n, etc.), utilisez la compétence modèles.
