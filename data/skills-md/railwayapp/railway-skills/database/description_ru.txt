Этот навык следует использовать, когда пользователь хочет добавить базу данных (Postgres, Redis, MySQL, MongoDB), говорит «добавить postgres», «добавить redis», «добавить базу данных», «подключиться к базе данных» или «подключить базу данных». Для других шаблонов (Ghost, Strapi, n8n и т. д.) используйте навык шаблонов.
