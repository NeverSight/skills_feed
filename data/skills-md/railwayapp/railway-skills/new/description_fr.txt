Cette compétence doit être utilisée lorsque l'utilisateur dit « configuration », « déployer sur le chemin de fer », « initialiser », « créer un projet », « créer un service » ou souhaite déployer à partir de GitHub. Gère la configuration initiale ET l’ajout de services aux projets existants. Pour les bases de données, utilisez plutôt la compétence base de données.
