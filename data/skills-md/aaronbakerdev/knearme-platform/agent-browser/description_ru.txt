Быстрый интерфейс командной строки для автоматизации браузера для агентов ИИ с помощью инструмента агента-браузера Vercel. Используйте для автономного управления браузером, очистки веб-страниц, заполнения форм, автоматического тестирования пользовательского интерфейса или когда пользователь запрашивает агент-браузер/автоматизацию браузера/безголовый браузер/автоматизацию браузера; оптимизирован для платформы портфолио KnearMe (knearme-portfolio и knearme-cms).
