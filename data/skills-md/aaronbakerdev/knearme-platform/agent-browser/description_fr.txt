CLI d'automatisation rapide du navigateur pour les agents IA à l'aide de l'outil de navigateur d'agent Vercel. À utiliser pour le contrôle du navigateur sans tête, le scraping Web, le remplissage de formulaires, les tests automatisés de l'interface utilisateur ou lorsqu'un utilisateur demande un navigateur d'agent/une automatisation du navigateur/un navigateur sans tête/un navigateur automatisé ; optimisé pour la plateforme de portefeuille KnearMe (knearme-portfolio et knearme-cms).
