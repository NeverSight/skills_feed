CLI di automazione rapida del browser per agenti AI che utilizzano lo strumento browser-agente Vercel. Utilizzare per il controllo del browser headless, il web scraping, la compilazione di moduli, il test automatizzato dell'interfaccia utente o quando un utente richiede un browser agente/automazione del browser/browser headless/browser automatizzato; ottimizzato per la piattaforma di portfolio KnearMe (knearme-portfolio e knearme-cms).
