Resend Box est un bac à sable de messagerie local qui se moque de l'API Resend et capture les e-mails pour inspection. Utilisez cette compétence lorsque vous travaillez avec le SDK Resend ou SMTP pour envoyer des e-mails pendant le développement. Il capture tous les e-mails envoyés via le SDK Resend (lorsque RESEND_BASE_URL pointe vers le bac à sable) ou via SMTP (port 1025). Utilisez-le pour vérifier que les e-mails sont envoyés correctement, inspecter le contenu des e-mails ou tester les modèles d'e-mails sans envoyer de vrais e-mails.
