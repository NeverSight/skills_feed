Resend Box es un entorno limitado de correo electrónico local que se burla de la API de Resend y captura correos electrónicos para su inspección. Utilice esta habilidad cuando trabaje con Resend SDK o SMTP para enviar correos electrónicos durante el desarrollo. Captura todos los correos electrónicos enviados a través del SDK de reenvío (cuando RESEND_BASE_URL apunta al entorno limitado) o mediante SMTP (puerto 1025). Úselo para verificar que los correos electrónicos se envíen correctamente, inspeccionar el contenido del correo electrónico o probar plantillas de correo electrónico sin enviar correos electrónicos reales.
