Resend Box عبارة عن صندوق حماية محلي للبريد الإلكتروني يسخر من Resend API ويلتقط رسائل البريد الإلكتروني لفحصها. استخدم هذه المهارة عند العمل مع Resend SDK أو SMTP لإرسال رسائل البريد الإلكتروني أثناء التطوير. فهو يلتقط جميع رسائل البريد الإلكتروني المرسلة عبر Resend SDK (عندما يشير RESEND_BASE_URL إلى صندوق الحماية) أو عبر SMTP (المنفذ 1025). يُستخدم للتحقق من إرسال رسائل البريد الإلكتروني بشكل صحيح، أو فحص محتوى البريد الإلكتروني، أو اختبار قوالب البريد الإلكتروني دون إرسال رسائل بريد إلكتروني حقيقية.
