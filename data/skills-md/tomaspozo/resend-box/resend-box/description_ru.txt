Resend Box — это локальная песочница для электронной почты, которая имитирует Resend API и собирает электронные письма для проверки. Используйте этот навык при работе с Resend SDK или SMTP для отправки электронных писем во время разработки. Он фиксирует все электронные письма, отправленные через Resend SDK (когда RESEND_BASE_URL указывает на песочницу) или через SMTP (порт 1025). Используйте для проверки правильности отправки электронных писем, проверки содержимого электронных писем или тестирования шаблонов электронных писем без отправки реальных писем.
