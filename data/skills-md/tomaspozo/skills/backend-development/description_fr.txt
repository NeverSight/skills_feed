Flux de travail de développement backend Supabase. À utiliser pour TOUT travail backend dans les projets Supabase : modifications de schéma, points de terminaison d'API, fonctions de base de données, politiques RLS, fonctions Edge, authentification, stockage, logique métier ou accès aux données. Activez chaque fois que la tâche implique une logique côté serveur, une couche de données ou des fonctionnalités Supabase.
