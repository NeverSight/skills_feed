Python >=3.11 directives d'optimisation des performances. Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la refactorisation du code Python pour garantir des modèles de performances optimaux. Déclencheurs sur des tâches impliquant l'asyncio, les structures de données, la gestion de la mémoire, la sérialisation, la concurrence ou l'optimisation des performances Python.
