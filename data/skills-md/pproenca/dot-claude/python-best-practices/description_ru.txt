Рекомендации по оптимизации производительности Python >=3.11. Этот навык следует использовать при написании, проверке или рефакторинге кода Python, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, связанных с asyncio, структурами данных, управлением памятью, сериализацией, параллелизмом или оптимизацией производительности Python.
