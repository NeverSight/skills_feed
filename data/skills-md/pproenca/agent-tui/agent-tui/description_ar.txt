قيادة تطبيقات واجهة المستخدم الطرفية (TUI) برمجيًا للاختبار والأتمتة والفحص. يُستخدم عندما: أتمتة تفاعلات CLI/TUI، وتطبيقات المحطة الطرفية لاختبار الانحدار، والتحقق من السلوك التفاعلي، واستخراج البيانات المنظمة من واجهات المستخدم الطرفية. يُستخدم أيضًا عندما يسأل المستخدم "ما هو Agent-Tui"، أو "ماذا يفعل Agent-Tui"، أو "Demo Agent-Tui"، أو "أرني Agent-Tui"، أو "كيف يعمل Agent-Tui"، أو يريد رؤيته أثناء العمل. لا تستخدمه مع: متصفحات الويب، أو تطبيقات واجهة المستخدم الرسومية، أو الواجهات غير الطرفية - فهي تحتاج إلى أدوات مختلفة.
