ターミナル UI (TUI) アプリケーションをプログラムで駆動し、テスト、自動化、検査を行います。 CLI/TUI インタラクションの自動化、ターミナル アプリの回帰テスト、インタラクティブな動作の検証、ターミナル UI からの構造化データの抽出の場合に使用します。また、次の場合にも使用します: ユーザーが「agent-tui とは何ですか」、「agent-tui は何をしますか」、「agent-tui のデモ」、「agent-tui を見せてください」、「agent-tui はどのように機能するのか」を尋ねる場合、または実際の動作を見たい場合。 Web ブラウザ、GUI アプリ、または非ターミナル インターフェイスには使用しないでください。これらには別のツールが必要です。
