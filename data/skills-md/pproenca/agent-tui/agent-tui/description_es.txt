Impulse aplicaciones de interfaz de usuario de terminal (TUI) mediante programación para pruebas, automatización e inspección. Úselo cuando: automatizar interacciones CLI/TUI, realizar pruebas de regresión en aplicaciones de terminal, verificar el comportamiento interactivo, extraer datos estructurados de las IU del terminal. Úselo también cuando: el usuario pregunta "qué es agent-tui", "qué hace agent-tui", "demostración de agent-tui", "muéstreme agent-tui", "cómo funciona agent-tui" o quiere verlo en acción. NO lo use para: navegadores web, aplicaciones GUI o interfaces que no sean de terminal, ya que necesitan herramientas diferentes.
