Программно управляйте приложениями пользовательского интерфейса терминала (TUI) для тестирования, автоматизации и проверки. Используйте при: автоматизации взаимодействия CLI/TUI, регрессионном тестировании приложений терминала, проверке интерактивного поведения, извлечении структурированных данных из пользовательского интерфейса терминала. Также используйте, когда: пользователь спрашивает «что такое агент-tui», «что делает агент-tui», «демо-агент-tui», «покажи мне агент-tui», «как работает агент-tui» или хочет увидеть его в действии. НЕ используйте для: веб-браузеров, приложений с графическим интерфейсом или нетерминальных интерфейсов — для них нужны другие инструменты.
