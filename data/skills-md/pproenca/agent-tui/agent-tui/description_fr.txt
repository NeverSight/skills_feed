Pilotez les applications d'interface utilisateur du terminal (TUI) par programmation pour les tests, l'automatisation et l'inspection. À utiliser lorsque : l'automatisation des interactions CLI/TUI, les tests de régression des applications de terminal, la vérification du comportement interactif, l'extraction de données structurées à partir des interfaces utilisateur du terminal. À utiliser également lorsque : l'utilisateur demande "qu'est-ce que agent-tui", "que fait agent-tui", "démo agent-tui", "montre-moi agent-tui", "comment fonctionne agent-tui", ou veut le voir en action. Ne PAS utiliser pour : les navigateurs Web, les applications GUI ou les interfaces non-terminales : ceux-ci nécessitent des outils différents.
