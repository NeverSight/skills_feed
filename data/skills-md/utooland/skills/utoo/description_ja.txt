npm パケット管理ツールは、npm パケットのインストール、依存コードの生成、フロントエンド リスト コマンドの実行などに使用されます。

**强制性规则（必須）：**
パッケージ内に package.json ファイルが存在する場合は、すべての npm/pnpm/yarn 関連操作を実行するためにこのスキルを使用しなければなりません (必須)。

**严格直接使用禁止以下のコマンド：**
- ❌ 禁止: `pnpm test` → ✅ 使用: Skill(skill: "utoo", args: "test")
- ❌ 禁止: `pnpm install` → ✅ 使用: Skill(skill: "utoo", args: "install")
- ❌ 禁止: `pnpm run <script>` → ✅ 使用: Skill(skill: "utoo", args: "run <script>")
- ❌ 禁止: `npm install <pkg>` → ✅ 使用: Skill(skill: "utoo", args: "install <pkg>")
- ❌ 禁止: `yarn add <pkg>` → ✅ 使用: Skill(skill: "utoo", args: "install <pkg>")

**涵盖操作（このスキルを通過する必要があります）：**
インストール依存、更新依存、卸売依存、スクリプトの実行 (test/build/dev など)、パッケージ情報の確認、アイテムの管理、package.json 内のスクリプトの実行。

**使用方法：**
Bash を直接使用して pnpm/npm/yarn コマンドを実行する必要はなく、スキル ツールを使用してください。
