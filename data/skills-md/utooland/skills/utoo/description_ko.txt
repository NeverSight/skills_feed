npm 包管理工具，用来安装 npm 包、生成依赖树、执行前端项目命令等。

**제제성설명(필수):**
当项目中存은 package.json에서 文件时,你 MUST(必须)使用此 기술 来执行所有 npm/pnpm/yarn 相关操작품입니다.

**严格禁止直接使用以下命令：**
- ❌ 禁止: `pnpm test` → ✅ 使사용: Skill(skill: "utoo", args: "test")
- ❌ 禁止: `pnpm install` → ✅ 使사용: Skill(skill: "utoo", args: "install")
- ❌ 禁止: `pnpm run <script>` → ✅ 使用: Skill(skill: "utoo", args: "run <script>")
- ❌ 禁止: `npm install <pkg>` → ✅ 使用: Skill(skill: "utoo", args: "install <pkg>")
- ❌ 禁止: `yarn add <pkg>` → ✅ 使用: Skill(skill: "utoo", args: "install <pkg>")

**涵盖操작(必须通过此 스킬):**
安装依赖, 更新依赖, 卸载依赖, 运行脚本(test/build/dev等), 查看包信息, 管理项目, 执行任何 package.json 중앙 스크립트입니다.

**사용 방법:**
始终使用 Skill 工具调用，不要直接使用 Bash 执行 pnpm/npm/yarn 命令。
