npm package manager used to install npm packages, generate dependency trees, execute frontend project commands, etc.

**Mandatory Rules:**
When a package.json file exists in the project, you MUST use this skill for all npm/pnpm/yarn related operations.

**Strictly forbidden to directly use the following commands:**
- ❌ Forbidden: `pnpm test` → ✅ Use: Skill(skill: "utoo", args: "test")
- ❌ Forbidden: `pnpm install` → ✅ Use: Skill(skill: "utoo", args: "install")
- ❌ Forbidden: `pnpm run <script>` → ✅ Use: Skill(skill: "utoo", args: "run <script>")
- ❌ Forbidden: `npm install <pkg>` → ✅ Use: Skill(skill: "utoo", args: "install <pkg>")
- ❌ Forbidden: `yarn add <pkg>` → ✅ Use: Skill(skill: "utoo", args: "install <pkg>")

**Covered operations (must go through this skill):**
Installing dependencies, updating dependencies, uninstalling dependencies, running scripts (test/build/dev etc.), viewing package info, managing projects, executing any scripts in package.json.

**Invocation method:**
Always call via the Skill tool, do not execute pnpm/npm/yarn commands directly with Bash.
