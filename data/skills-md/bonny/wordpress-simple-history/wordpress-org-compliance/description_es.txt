Garantiza el cumplimiento de WordPress.org para complementos freemium (funciones gratuitas versus premium, claves de licencia, límites de prueba, ventas adicionales). Previene violaciones del software de prueba. Úselo al agregar funciones premium, implementar ventas adicionales, verificar claves de licencia, crear avances, revisar el código antes del envío de WordPress.org o cuando el usuario mencione "premium", "upsell" o "freemium".
