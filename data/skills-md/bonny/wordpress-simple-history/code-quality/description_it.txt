Fornisce standard di codifica WordPress, guida di stile PHP/CSS/JS, linting del codice e analisi statica (phpcs, phpstan, rector, phpcbf). Da utilizzare prima di scrivere o modificare codice PHP, CSS o JavaScript, quando si correggono errori di lanugine, si eseguono strumenti di qualità del codice o si verifica la conformità dello stile del codice.
