Fournit les normes de codage WordPress, un guide de style PHP/CSS/JS, le linting de code et l'analyse statique (phpcs, phpstan, rector, phpcbf). À utiliser avant d'écrire ou de modifier du code PHP, CSS ou JavaScript, lors de la correction d'erreurs de charpie, de l'exécution d'outils de qualité de code ou de la vérification de la conformité du style de code.
