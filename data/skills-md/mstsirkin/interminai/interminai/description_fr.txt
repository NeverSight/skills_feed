Contrôlez les applications de terminal interactif telles que vim, git rebase -i, git add -i, git add -p, apt, rclone config, sudo, w3m et les applications TUI. Peut également superviser un autre CLI LLM (curseur-agent, codex, etc.) - approuver ou rejeter ses actions en appuyant sur o/n aux invites de confirmation. À utiliser lorsque vous devez interagir avec des applications qui nécessitent une saisie au clavier, afficher des invites, des menus ou disposer d'interfaces plein écran. À utiliser également lorsque les commandes échouent ou se bloquent avec des erreurs telles que « L'entrée n'est pas un terminal » ou « La sortie n'est pas un terminal ». Mieux que les hacks spécifiques à une application tels que GIT_SEQUENCE_EDITOR ou le contournement de l'interactivité via l'utilisation de fichiers.
