Controle aplicaciones de terminal interactivas como vim, git rebase -i, git add -i, git add -p, apt, rclone config, sudo, w3m y aplicaciones TUI. También puede supervisar otro CLI LLM (cursor-agente, códice, etc.): apruebe o rechace sus acciones presionando y/n en las solicitudes de confirmación. Úselo cuando necesite interactuar con aplicaciones que requieren entrada de teclado, mostrar indicaciones, menús o tener interfaces de pantalla completa. Úselo también cuando los comandos fallan o se bloquean con errores como "La entrada no es un terminal" o "La salida no es un terminal". Mejor que los trucos específicos de aplicaciones como GIT_SEQUENCE_EDITOR o evitar la interactividad mediante el uso de archivos.
