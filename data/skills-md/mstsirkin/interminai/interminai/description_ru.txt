Управляйте приложениями интерактивного терминала, такими как vim, git rebase -i, git add -i, git add -p, apt, rclone config, sudo, w3m и приложения TUI. Также можно контролировать другой CLI LLM (курсор-агент, кодекс и т. д.) — одобряйте или отклоняйте его действия, нажимая Y/N при запросе подтверждения. Используйте, когда вам нужно взаимодействовать с приложениями, требующими ввода с клавиатуры, отображать подсказки, меню или иметь полноэкранный интерфейс. Также используйте, когда команды не выполняются или зависают с такими ошибками, как «Ввод не является терминалом» или «Выход не является терминалом». Лучше, чем хаки для конкретных приложений, такие как GIT_SEQUENCE_EDITOR или обход интерактивности посредством использования файлов.
