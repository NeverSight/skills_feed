Controlla le applicazioni terminali interattive come vim, git rebase -i, git add -i, git add -p, apt, rclone config, sudo, w3m e le app TUI. Può anche supervisionare un altro CLI LLM (agente cursore, codice, ecc.) - approvare o rifiutare le sue azioni premendo y/n alle richieste di conferma. Utilizzalo quando devi interagire con applicazioni che richiedono l'input da tastiera, mostrano istruzioni, menu o dispongono di interfacce a schermo intero. Da utilizzare anche quando i comandi falliscono o si bloccano con errori come "L'input non è un terminale" o "L'output non è un terminale". Meglio degli hack specifici dell'applicazione come GIT_SEQUENCE_EDITOR o bypassare l'interattività attraverso l'uso dei file.
