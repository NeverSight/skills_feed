Steuern Sie interaktive Terminalanwendungen wie vim, git rebase -i, git add -i, git add -p, apt, rclone config, sudo, w3m und TUI-Apps. Kann auch ein anderes CLI-LLM (Cursor-Agent, Codex usw.) überwachen – seine Aktionen genehmigen oder ablehnen, indem Sie bei Bestätigungsaufforderungen J/N drücken. Verwenden Sie diese Option, wenn Sie mit Anwendungen interagieren müssen, die Tastatureingaben erfordern, Eingabeaufforderungen und Menüs anzeigen oder über Vollbildschnittstellen verfügen. Wird auch verwendet, wenn Befehle fehlschlagen oder mit Fehlern wie „Eingabe ist kein Terminal“ oder „Ausgabe ist kein Terminal“ hängen bleiben. Besser als anwendungsspezifische Hacks wie GIT_SEQUENCE_EDITOR oder das Umgehen der Interaktivität durch Dateinutzung.
