اعمل مع خزائن Obsidian المستندة إلى Ideaverse باستخدام إطار عمل ACE (Atlas/Calendar/Efforts) ومنهجية LYT (ربط تفكيرك). يُستخدم عند إنشاء الملاحظات، أو تنظيم المعرفة، أو إنشاء خرائط المحتوى (MOCs)، أو الاحتفاظ بالسجلات اليومية، أو الإسناد الترافقي للأفكار، أو استيعاب المعلومات الجديدة، أو العمل مع أي قبو Obsidian يتبع اصطلاحات Ideaverse. يتم تشغيل طلبات إضافة المعرفة أو ربط الملاحظات أو إنشاء MOCs أو تحديث السجلات اليومية أو أرشفة المشاريع أو اكتشاف الروابط بين الأفكار.
