Integre API y SDK de terceros en proyectos. Úselo cuando el usuario quiera conectarse a servicios externos (Resend, Stripe, Zendesk, Ahrefs, Twilio, etc.) o solicite "integrar", "conectar" o "agregar" una API. Maneja la investigación de documentación, la selección de SDK, la coincidencia de patrones de proyectos y la implementación.
