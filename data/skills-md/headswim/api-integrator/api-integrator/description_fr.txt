Intégrez des API et des SDK tiers dans vos projets. À utiliser lorsque l'utilisateur souhaite se connecter à des services externes (Resend, Stripe, Zendesk, Ahrefs, Twilio, etc.) ou demande « d'intégrer », de « se connecter » ou « d'ajouter » une API. Gère la recherche de documentation, la sélection du SDK, la correspondance de modèles de projet et la mise en œuvre.
