Интегрируйте сторонние API и SDK в проекты. Используйте, когда пользователь хочет подключиться к внешним сервисам (Resend, Stripe, Zendesk, Ahrefs, Twilio и т. д.) или просит «интегрировать», «подключить» или «добавить» API. Занимается исследованием документации, выбором SDK, сопоставлением шаблонов проектов и реализацией.
