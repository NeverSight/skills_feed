دمج واجهات برمجة التطبيقات ومجموعات SDK التابعة لجهات خارجية في المشاريع. يُستخدم عندما يريد المستخدم الاتصال بخدمات خارجية (Resend، Stripe، Zendesk، Ahrefs، Twilio، وما إلى ذلك) أو يطلب "دمج" أو "اتصال" أو "إضافة" واجهة برمجة التطبيقات. يتعامل مع أبحاث التوثيق واختيار SDK ومطابقة نمط المشروع والتنفيذ.
