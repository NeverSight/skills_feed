Контейнеризация Docker промышленного уровня для приложений Python и Node.js.
Этот навык следует использовать, когда пользователи просят контейнеризировать приложения, создавать Dockerfiles,
dockerize проекты или настройте Docker Compose. Автоматически определяет структуру проекта,
анализирует .env на наличие секретов, проверяет безопасность и генерирует проверенные файлы Dockerfile.
