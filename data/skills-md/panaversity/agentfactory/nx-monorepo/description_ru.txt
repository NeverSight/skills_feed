Навыки управления монорепозиториями Nx для разработки с использованием искусственного интеллекта. Этот навык следует использовать при работе с рабочими пространствами Nx, графами проектов, обнаружением уязвимостей, генерацией кода и кэшированием. Используйте при анализе зависимостей, запуске затронутых команд, создании кода, настройке Nx Cloud или оптимизации производительности сборки. Вызов инструментов nx-mcp для запросов документации.
