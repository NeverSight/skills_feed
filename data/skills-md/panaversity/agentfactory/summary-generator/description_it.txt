Questa abilità dovrebbe essere utilizzata quando si generano riassunti delle lezioni per contenuti didattici. Estrae concetti chiave, modelli mentali, schemi ed errori comuni dai file di markdown delle lezioni utilizzando un processo di estrazione socratica. Utilizza questa abilità quando un file di lezione necessita di un file complementare `.summary.md` o quando rivedi/aggiorni i riepiloghi esistenti.
