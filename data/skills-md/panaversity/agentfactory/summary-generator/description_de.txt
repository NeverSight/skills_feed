Diese Fähigkeit sollte beim Erstellen von Unterrichtszusammenfassungen für Bildungsinhalte verwendet werden. Mithilfe eines sokratischen Extraktionsprozesses werden Schlüsselkonzepte, mentale Modelle, Muster und häufige Fehler aus Lektions-Markdown-Dateien extrahiert. Verwenden Sie diese Fähigkeit, wenn eine Unterrichtsdatei eine Begleitdatei „.summary.md“ benötigt oder wenn Sie vorhandene Zusammenfassungen überprüfen/aktualisieren.
