يجلب الوثائق الرسمية للمكتبات والأطر الخارجية (React، وNext.js، وPrisma، وFastAPI، وExpress، وTailwind، وMongoDB، وما إلى ذلك) مع توفير الرموز المميزة بنسبة 60-90% عبر تصفية نوع المحتوى. استخدم هذه المهارة عند تنفيذ الميزات باستخدام واجهات برمجة تطبيقات المكتبة، أو تصحيح الأخطاء الخاصة بالمكتبة، أو استكشاف مشكلات التكوين وإصلاحها، أو تثبيت أطر العمل أو إعدادها، أو دمج حزم الجهات الخارجية، أو الترقية بين إصدارات المكتبة، أو البحث عن أنماط واجهة برمجة التطبيقات الصحيحة وأفضل الممارسات. يتم التشغيل تلقائيًا أثناء عمل البرمجة - قم بإحضار المستندات قبل كتابة كود المكتبة للحصول على الأنماط الصحيحة، وليس بعد التخمين الخاطئ.
