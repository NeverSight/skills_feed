コンテンツ タイプ フィルタリングにより、トークンを 60 ～ 90% 節約して、外部ライブラリおよびフレームワーク (React、Next.js、Prisma、FastAPI、Express、Tailwind、MongoDB など) の公式ドキュメントを取得します。このスキルは、ライブラリ API を使用した機能の実装、ライブラリ固有のエラーのデバッグ、構成の問題のトラブルシューティング、フレームワークのインストールまたはセットアップ、サードパーティ パッケージの統合、ライブラリ バージョン間のアップグレード、または正しい API パターンとベスト プラクティスの検索を行うときに使用します。コーディング作業中に自動的にトリガーされます。間違った推測の後ではなく、ライブラリ コードを作成する前にドキュメントを取得して、正しいパターンを取得します。
