Получает официальную документацию для внешних библиотек и платформ (React, Next.js, Prisma, FastAPI, Express, Tailwind, MongoDB и т. д.) с экономией токенов 60–90 % за счет фильтрации по типам контента. Используйте этот навык при реализации функций с использованием API-интерфейсов библиотеки, отладке ошибок, связанных с библиотекой, устранении проблем с конфигурацией, установке или настройке платформ, интеграции сторонних пакетов, обновлении между версиями библиотеки или поиске правильных шаблонов API и рекомендаций. Запускается автоматически во время работы с кодом — извлекайте документы перед написанием кода библиотеки, чтобы получить правильные шаблоны, а не после догадки.
