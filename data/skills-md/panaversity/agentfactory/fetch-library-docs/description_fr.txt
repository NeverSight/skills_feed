Récupère la documentation officielle pour les bibliothèques et les frameworks externes (React, Next.js, Prisma, FastAPI, Express, Tailwind, MongoDB, etc.) avec 60 à 90 % d'économies de jetons via le filtrage par type de contenu. Utilisez cette compétence lors de la mise en œuvre de fonctionnalités à l'aide des API de bibliothèque, du débogage d'erreurs spécifiques à la bibliothèque, du dépannage des problèmes de configuration, de l'installation ou de la configuration de frameworks, de l'intégration de packages tiers, de la mise à niveau entre les versions de bibliothèque ou de la recherche de modèles d'API corrects et de bonnes pratiques. Se déclenche automatiquement pendant le travail de codage : récupérez les documents avant d'écrire le code de la bibliothèque pour obtenir les modèles corrects, et non après avoir deviné une erreur.
