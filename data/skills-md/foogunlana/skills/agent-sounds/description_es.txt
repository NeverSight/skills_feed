Agregue notificaciones audibles (campanas) a Claude Code y otros agentes de codificación. Proporciona un pequeño script que reproduce sonidos configurables en enlaces comunes del ciclo de vida, como finalización de planes, finalización de tareas, errores y finalización de comandos de larga duración. Los valores predeterminados funcionan de forma inmediata en macOS (sonidos del sistema) con respaldos multiplataforma.
