Добавьте звуковые уведомления (звоночки) в Claude Code и другие агенты кодирования. Предоставляет небольшой скрипт, который воспроизводит настраиваемые звуки при распространенных перехватчиках жизненного цикла, таких как завершение плана, завершение задачи, ошибки и завершение длительной команды. Значения по умолчанию работают в macOS «из коробки» (системные звуки) с кроссплатформенными резервными вариантами.
