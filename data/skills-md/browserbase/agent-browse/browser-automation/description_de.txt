Automatisieren Sie Webbrowser-Interaktionen mithilfe natürlicher Sprache über CLI-Befehle. Wird verwendet, wenn der Benutzer Websites durchsuchen, auf Webseiten navigieren, Daten von Websites extrahieren, Screenshots erstellen, Formulare ausfüllen, auf Schaltflächen klicken oder mit Webanwendungen interagieren möchte. Zu den Auslösern gehören „Durchsuchen“, „Navigieren zu“, „Zur Website gehen“, „Daten von Webseite extrahieren“, „Screenshot“, „Web Scraping“, „Formular ausfüllen“, „Anklicken“ und „Im Web suchen“. Seien Sie beim Ergreifen von Maßnahmen so konkret wie möglich.
