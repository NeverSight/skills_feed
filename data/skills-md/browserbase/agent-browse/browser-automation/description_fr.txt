Automatisez les interactions du navigateur Web en utilisant le langage naturel via les commandes CLI. À utiliser lorsque l'utilisateur demande à parcourir des sites Web, à naviguer dans des pages Web, à extraire des données de sites Web, à prendre des captures d'écran, à remplir des formulaires, à cliquer sur des boutons ou à interagir avec des applications Web. Les déclencheurs incluent « parcourir », « naviguer vers », « accéder au site Web », « extraire des données d'une page Web », « capture d'écran », « web scraping », « remplir un formulaire », « cliquer sur », « rechercher sur le Web ». Lorsque vous prenez des mesures, soyez aussi précis que possible.
