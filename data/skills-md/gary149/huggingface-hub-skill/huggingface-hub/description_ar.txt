معرفة الخبراء بالنظام البيئي Hugging Face Hub، بما في ذلك استخدام HTTP/curl API وPython/JavaScript SDKs وتطبيقات Spaces (Gradio وDocker) وتنسيقات النماذج (GGUF وPEFT/LoRA) ومجموعات البيانات (pandas وPolars وDuckDB) والأتمتة (webhooks والوظائف) وميزات الأمان. يُستخدم عند العمل مع Hub APIs، أو Huggingface_hub Python code، أو تطبيقات Spaces، أو موفري الاستدلال، أو مستودعات النماذج/مجموعة البيانات، أو أتمتة Hub.
