Appliquez des techniques systématiques d'optimisation des performances pour le code Python et Rust : estimation + profilage, conception API/bloc, victoires algorithmiques, disposition de la mémoire respectueuse du cache, moins d'allocations, chemins rapides, mise en cache et boucles chaudes conviviales pour le compilateur. À utiliser pour les révisions de code de performance, les refactorisations et les optimisations basées sur le profilage. Mots clés : performances, latence, débit, cache, allocation, disposition de la mémoire, PyO3, msgspec, tokio, async, pprof, py-spy, perf.
