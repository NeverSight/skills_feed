يدير مجالات طبقة السياق والمهام والوثائق. يُستخدم عندما يذكر المستخدم "طبقة السياق" (على سبيل المثال، "استخدام طبقة السياق"، "في طبقة السياق"، "في السياق"، "إلى طبقة السياق"، "في طبقة السياق") أوامر ctx أو مشاريع أو مهام طبقة السياق، أو يطلب منك كتابة أي شكل من أشكال الوثائق - مثل توثيق الميزات المنفذة أو نتائج الأبحاث أو رسم المخططات. استخدمه أيضًا عندما يطلب المستخدم إضافة مستودع إلى طبقة السياق، أو استنساخ الريبو كسياق طبقة السياق، أو الوصول إلى مجلد "البيانات" الخاص بطبقة السياق.
