Assistere gli sviluppatori nella scrittura di codice pulito e gestibile seguendo le migliori pratiche di ingegneria del software. Da utilizzare durante l'esecuzione di revisioni del codice, il refactoring del codice, l'applicazione di standard di codifica, la ricerca di indicazioni sui principi di pulizia del codice o l'integrazione di controlli di qualità automatizzati nei flussi di lavoro di sviluppo.
