قم بإنشاء كود Vue TypeScript حديث وقابل للصيانة مع أفضل الممارسات: Composition API، وإعداد البرنامج النصي مع TS، والأنماط القابلة للتركيب على المتاجر العالمية، وتدمير الدعائم، وتكامل VueUse، وTailwind CSS، وإمكانية الوصول، وتحسين الأداء، واختبار Vitest. يُستخدم عند إنشاء مكونات Vue 3 أو المواد القابلة للتركيب أو إعادة بناء كود Vue لاتباع الأنماط الحديثة.
