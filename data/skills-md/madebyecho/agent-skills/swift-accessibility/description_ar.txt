يطبق تلقائيًا أفضل ممارسات إمكانية الوصول على مشاريع Swift (SwiftUI وUIKit). يُستخدم عند العمل على مشروعات iOS/macOS التي تحتاج إلى دعم VoiceOver أو النوع الديناميكي أو امتثال WCAG أو عمليات تدقيق إمكانية الوصول. يتم تشغيلها عند تنفيذ مهام إمكانية الوصول إلى Swift أو إجراء تحسينات أو عندما يذكر المستخدم إمكانية الوصول أو VoiceOver أو النوع الديناميكي.
