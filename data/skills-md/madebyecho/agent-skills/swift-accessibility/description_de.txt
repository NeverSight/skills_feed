Wendet Best Practices zur Barrierefreiheit automatisch auf Swift-Projekte an (SwiftUI und UIKit). Verwenden Sie diese Option, wenn Sie an iOS-/macOS-Projekten arbeiten, die VoiceOver-Unterstützung, Dynamic Type, WCAG-Konformität oder Barrierefreiheitsprüfungen benötigen. Wird bei Swift-Eingabehilfen-Aufgaben, bevorstehenden Verbesserungen oder wenn der Benutzer Eingabehilfen, VoiceOver oder Dynamic Type erwähnt, ausgelöst.
