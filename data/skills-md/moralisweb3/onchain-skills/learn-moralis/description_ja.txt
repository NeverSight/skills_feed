Moralis と Web3 開発について学びます。質問なしで呼び出すと、何が利用可能か、どのようなデータを取得できるか、すべてがどのように組み合わされるかなど、わかりやすいプラットフォームのウォークスルーが表示されます。質問で呼び出され、それに直接答えます。 「Moralis とは何ですか」、「Moralis は X を行うことができますか」、「どのチェーンがサポートされていますか」、「どのように始めればよいですか」、「どの API を使用すべきか」、価格設定、機能の比較、またはその他の調査的な質問に使用します。回答後、正しい技術スキル (@moralis-data-api または @moralis-streams-api) にルーティングされます。
