Moralis API から Web3 ブロックチェーン データをクエリします。ユーザーがウォレット データ (残高、トークン、NFT、取引履歴、収益性、純資産)、トークン データ (価格、メタデータ、DEX ペア、分析、セキュリティ スコア)、NFT データ (メタデータ、転送、特性、希少性、下限価格)、DeFi ポジション、取引所と資金のエンティティ/ラベル データ、またはブロック データとトランザクション データについて質問するときに使用します。 EVM チェーン (Ethereum、Polygon、BSC、Arbitrum、Base、Optimism、Avalanche など) と Solana をサポートします。リアルタイム ストリーミングには使用できません。代わりに、moraris-streams-api を使用してください。
