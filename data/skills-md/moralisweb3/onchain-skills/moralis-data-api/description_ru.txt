Запросите данные блокчейна Web3 из Moralis API. Используйте, когда пользователь запрашивает данные кошелька (балансы, токены, NFT, историю транзакций, прибыльность, собственный капитал), данные токенов (цены, метаданные, пары DEX, аналитика, оценки безопасности), данные NFT (метаданные, переводы, характеристики, редкость, минимальные цены), позиции DeFi, данные об организациях/метках для бирж и фондов или данные о блоках и транзакциях. Поддерживает цепочки EVM (Ethereum, Polygon, BSC, Arbitrum, Base, Optimism, Avalanche и т. д.) и Solana. НЕ для потоковой передачи в реальном времени — вместо этого используйте Moralis-streams-api.
