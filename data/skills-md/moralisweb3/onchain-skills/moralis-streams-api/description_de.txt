Überwachung von Blockchain-Ereignissen in Echtzeit mit Webhooks. Verwenden Sie diese Option, wenn der Benutzer Fragen zum Einrichten von Webhooks, zum Streamen von Ereignissen in Echtzeit, zum Überwachen von Wallet-Adressen, zum Verfolgen von Tokenübertragungen in Echtzeit, zum Abhören aller Adressen in einer Kette, zum Erstellen/Aktualisieren/Löschen von Streams, zum Hinzufügen/Entfernen von Adressen zu Streams oder zum Empfangen von Blockchain-Ereignissen hat, sobald diese eintreten. Unterstützt alle EVM-Ketten. NICHT zum Abfragen des historischen oder aktuellen Blockchain-Status – verwenden Sie stattdessen moralis-data-api.
