Руководство по разработке и планированию курсов Educates, которые представляют собой структурированные наборы семинаров для интерактивной учебной платформы Educates. Используйте этот навык, когда пользователь просит разработать курс, спланировать курс, расширить существующий курс или организовать темы в учебную программу, состоящую из нескольких семинаров. НЕ используйте этот навык, если пользователь хочет создать только одну автономную мастерскую; вместо этого используйте для этого навык «обучение-семинар-авторство».
