Sistema completo de gestión de proyectos impulsado por el rastreador de tareas de Beads. Incluye creación de PRD, descomposición épica, gestión de tareas, ejecución de agentes paralelos, sincronización bidireccional, árboles de trabajo git y comandos de flujo de trabajo integrales. Úselo cuando el usuario necesite administrar proyectos de software con PRD, épicos, problemas y flujos de trabajo de sprint.
