Leitfaden zum Erstellen von Sandbox-Anwendungen mit ComputeSDK, einem einheitlichen TypeScript-SDK zum Ausführen von nicht vertrauenswürdigem Code in Sandbox-Umgebungen über mehrere Compute-Anbieter hinweg (E2B, Daytona, Vercel, Modal, Railway, Namespace, Render). Nutzen Sie diese Fähigkeit, wenn Sie die Sandbox-Codeausführung implementieren, isolierte Entwicklungsumgebungen erstellen, LLM-generierten Code sicher ausführen, App-Builder erstellen oder die dynamische Codeausführung in Anwendungen integrieren.
