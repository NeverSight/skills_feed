Guía para crear aplicaciones de espacio aislado con ComputeSDK, un SDK de TypeScript unificado para ejecutar código que no es de confianza en entornos de espacio aislado en múltiples proveedores de computación (E2B, Daytona, Vercel, Modal, Railway, Namespace, Render). Utilice esta habilidad al implementar la ejecución de código en espacio aislado, crear entornos de desarrollo aislados, ejecutar código generado por LLM de forma segura, crear creadores de aplicaciones o integrar la ejecución dinámica de código en aplicaciones.
