Guide pour créer des applications sandbox avec ComputeSDK, un SDK TypeScript unifié pour exécuter du code non fiable dans des environnements sandbox sur plusieurs fournisseurs de calcul (E2B, Daytona, Vercel, Modal, Railway, Namespace, Render). Utilisez cette compétence lors de la mise en œuvre de l'exécution de code en bac à sable, de la création d'environnements de développement isolés, de l'exécution en toute sécurité du code généré par LLM, de la création de générateurs d'applications ou de l'intégration de l'exécution de code dynamique dans des applications.
