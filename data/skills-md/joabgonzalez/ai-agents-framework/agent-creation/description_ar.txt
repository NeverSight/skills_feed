يرشد الوكلاء والمستخدمين لإنشاء تعريفات وكيل متوافقة مع المعايير باستخدام قوالب بتنسيق Frontmatter، مما يضمن جمع السياق والوضوح والبيانات الوصفية المناسبة. يجبر الوكلاء على طرح أسئلة توضيحية عندما يكون السياق غير كافٍ. مطلوب وثائق فعالة للرمز. يجب أن يكون كل المحتوى باللغة الإنجليزية. المشغل: عند إنشاء تعريف وكيل جديد، أو إعداد وكلاء المشروع، أو توثيق سير عمل الوكيل.
