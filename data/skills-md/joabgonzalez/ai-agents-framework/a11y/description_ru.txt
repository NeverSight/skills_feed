Лучшие практики и стандарты универсальной доступности для всех технологий (HTML, CSS, React, MUI, Astro, React Native). Соответствие WCAG 2.1/2.2 уровня AA, семантический HTML, ARIA, навигация с помощью клавиатуры, цветовой контраст. Триггер: при реализации компонентов пользовательского интерфейса, добавлении интерактивных элементов или обеспечении соответствия требованиям доступности.
