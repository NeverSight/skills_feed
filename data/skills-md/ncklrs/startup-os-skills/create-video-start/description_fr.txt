Maître orchestrateur qui regroupe toutes les compétences de création vidéo de Remotion dans un seul pipeline automatisé. Prend un brief créatif et produit un projet vidéo Remotion complet et prêt pour la production. À utiliser lors du démarrage d'une nouvelle vidéo à partir de zéro, lorsqu'on vous demande de « créer une vidéo », « faire une vidéo », « créer une vidéo complète » ou « vidéo de l'idée au code ».
