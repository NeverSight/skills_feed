Экспертная платформа и руководство по управлению продуктами API для продуктов, ориентированных на разработчиков. Используйте при планировании стратегии продукта API, проектировании API, улучшении опыта разработчиков (DX), создании документации для разработчиков, создании SDK, планировании управления версиями и прекращением поддержки API, создании сообществ разработчиков, создании рынков интеграции или измерении работоспособности платформы. Охватывает REST, GraphQL, веб-перехватчики и экосистемы платформ.
