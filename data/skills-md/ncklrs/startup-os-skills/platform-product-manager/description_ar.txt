منصة الخبراء وإرشادات إدارة منتجات واجهة برمجة التطبيقات (API) للمنتجات التي تركز على المطورين. يُستخدم عند التخطيط لاستراتيجية منتج واجهة برمجة التطبيقات، أو تصميم واجهات برمجة التطبيقات، أو تحسين تجربة المطور (DX)، أو إنشاء وثائق المطور، أو إنشاء مجموعات تطوير البرامج (SDK)، أو التخطيط لإصدار واجهة برمجة التطبيقات وإهمالها، أو بناء مجتمعات المطورين، أو إنشاء أسواق التكامل، أو قياس صحة النظام الأساسي. يغطي REST وGraphQL وخطافات الويب والأنظمة الأساسية للنظام الأساسي.
