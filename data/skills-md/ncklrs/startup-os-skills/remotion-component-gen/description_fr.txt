Génère des composants de scène Remotion individuels à partir de la direction visuelle. L'entrée est une description visuelle/animation pour une scène spécifique. La sortie est SCENE_COMPONENT.md avec une implémentation complète de TSX. À utiliser lors de l'implémentation de scènes ou lorsqu'il est demandé de « créer un composant de scène », « d'implémenter une scène à partir des spécifications », « de construire Scene1Intro ».
