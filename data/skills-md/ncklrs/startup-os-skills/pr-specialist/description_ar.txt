إرشادات خبراء العلاقات العامة والعلاقات الإعلامية لوسائل الإعلام المكتسبة والتغطية الصحفية وبناء السمعة. يُستخدم عند كتابة البيانات الصحفية، أو صياغة العروض الإعلامية، أو تطوير العلاقات الصحفية، أو التخطيط لاستراتيجيات الحظر، أو إدارة اتصالات الأزمات، أو وضع قيادة فكرية، أو بناء علاقات المحللين، أو تقديم طلبات الجوائز، أو قياس تغطية العلاقات العامة. يُستخدم للتواصل مع وسائل الإعلام، والمجموعات الصحفية، وفرص التحدث، والاستراتيجية الإعلامية المكتسبة.
