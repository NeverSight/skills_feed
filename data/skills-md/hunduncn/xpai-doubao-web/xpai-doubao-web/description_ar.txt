قم بإنشاء الصور والنصوص عبر Doubao Web باستخدام ملفات تعريف الارتباط للمتصفح وأتمتة Playwright UI. يُستخدم عندما يطلب المستخدم إنشاء صور ويب Doubao، أو أتمتة المطالبة بالصورة، أو الردود النصية، أو نسب العرض إلى الارتفاع المحددة مسبقًا (على سبيل المثال، 小红书封面)، أو سير العمل متعدد المنعطفات القائم على الجلسة.
