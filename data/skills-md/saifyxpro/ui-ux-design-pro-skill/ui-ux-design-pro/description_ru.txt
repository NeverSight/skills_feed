Старший эксперт по UI/UX-дизайну для создания управляемых данными производственных интерфейсов премиум-класса. Используйте, когда вам нужно: 1. Проектировать сложные приложения (панели мониторинга, SaaS, инструменты искусственного интеллекта) с нуля. 2. Создавать комплексные системы дизайна (токены, палитры, типографика). 3. Аудит существующего пользовательского интерфейса на предмет качества, доступности и «крафта». 4. Поиск проверенных реальных шаблонов проектирования и деталей реализации. Триггер: «разработать...», «проверить это...», «создать систему дизайна», «найти значки», «панель управления финансовыми технологиями», «целевую страницу».
