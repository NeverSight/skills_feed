データ駆動型のプレミアムなプロダクション インターフェイスを構築する上級レベルの UI/UX デザイン エキスパート。必要な場合に使用します。 1. 複雑なアプリケーション (ダッシュボード、SaaS、AI ツール) を最初から設計する 2. 包括的なデザイン システム (トークン、パレット、タイポグラフィ) を生成する 3. 品質、アクセシビリティ、および「クラフト」について既存の UI を監査する 4. 実証済みの実際のデザイン パターンと実装の詳細を検索する トリガー: 「デザインを...」、「これを監査する...」、「デザイン システムを作成する」、「アイコンを検索」、「フィンテック」ダッシュボード」、「ランディング ページ」
