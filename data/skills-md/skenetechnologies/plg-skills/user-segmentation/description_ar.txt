عندما يريد المستخدم تقسيم المستخدمين إلى شرائح للحصول على تجارب مخصصة - بما في ذلك المجموعات السلوكية، أو تسجيل المشاركة، أو تسجيل مخاطر التوقف، أو تحسين برنامج المقارنات الدولية. يُستخدم أيضًا عندما يقول المستخدم "شرائح المستخدمين"، أو "التحليل الجماعي"، أو "المستخدمون المتميزون"، أو "المستخدمون المعرضون للخطر"، أو "تحليل RFM"، أو "تسجيل المستخدم". بالنسبة للمبيعات التي يقودها المنتج، راجع المبيعات التي يقودها المنتج. للاحتفاظ، راجع تحليل الاستبقاء.
