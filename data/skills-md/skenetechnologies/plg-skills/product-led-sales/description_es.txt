Cuando el usuario quiera superponer las ventas a un movimiento PLG, crear una puntuación PQL, diseñar transferencias de ventas a partir de señales de uso del producto o planificar un modelo de ventas híbrido PLG +. Utilícelo también cuando el usuario diga "ventas basadas en productos", "PQL", "PQA", "cuándo agregar ventas a PLG" o "PLG empresarial". Para conocer una estrategia PLG más amplia, consulte estrategia-plg. Para conocer los ingresos de expansión, consulte ingresos de expansión.
