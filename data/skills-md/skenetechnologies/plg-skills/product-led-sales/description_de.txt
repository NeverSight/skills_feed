Wenn der Benutzer Verkäufe auf eine PLG-Bewegung schichten möchte, eine PQL-Bewertung erstellen, Verkaufsübergaben anhand von Produktnutzungssignalen entwerfen oder ein hybrides PLG + Verkaufsmodell planen möchte. Wird auch verwendet, wenn der Benutzer „produktgesteuerte Verkäufe“, „PQL“, „PQA“, „wann Verkäufe zu PLG hinzufügen sollen“ oder „Unternehmens-PLG“ sagt. Eine umfassendere PLG-Strategie finden Sie unter plg-strategy. Informationen zu Expansionserlösen finden Sie unter Expansionserlöse.
