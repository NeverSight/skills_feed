Auditoría analítica integral del código base del sitio web para identificar elementos rastreables y evaluar la preparación analítica. Utilícelo cuando los usuarios quieran "auditar mis análisis", "buscar elementos rastreables", "encontrar lo que puedo rastrear", "analizar mi sitio web en busca de oportunidades de seguimiento" o antes de implementar el seguimiento de GTM. Escanea HTML/JSX/TSX/Vue en busca de todos los elementos en los que se puede hacer clic (botones, enlaces, formularios, etc.), identifica el código de seguimiento existente, evalúa la estructura DOM para análisis y proporciona recomendaciones. Actúa como ingeniero frontend senior con experiencia en GA4.
