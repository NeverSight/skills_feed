Audit analytique complet de la base de code du site Web pour identifier les éléments traçables et évaluer l'état de préparation à l'analyse. À utiliser lorsque les utilisateurs souhaitent « auditer mes analyses », « rechercher des éléments traçables », « trouver ce que je peux suivre », « analyser mon site Web pour détecter des opportunités de suivi » ou avant de mettre en œuvre le suivi GTM. Analyse HTML/JSX/TSX/Vue à la recherche de tous les éléments cliquables (boutons, liens, formulaires, etc.), identifie le code de suivi existant, évalue la structure DOM à des fins d'analyse et fournit des recommandations. Agit en tant qu'ingénieur frontend senior avec une expertise GA4.
