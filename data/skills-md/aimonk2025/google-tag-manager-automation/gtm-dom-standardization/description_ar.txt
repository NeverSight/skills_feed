توحيد جميع المعرفات المرتبطة بالنقرات وفئات CSS عبر موقع الويب لتتبع التحليلات النظيفة. يُستخدم عندما يريد المستخدمون "توحيد فئات التحليلات"، أو "تنظيف معرفات التتبع"، أو "إعداد DOM لـ GTM"، أو "إصلاح تسمية التحليلات"، أو "جعل التتبع متسقًا". يقوم بمسح قاعدة التعليمات البرمجية بالكامل (HTML/JSX/TSX/Vue) ويطبق اصطلاح تسمية متسق - المعرفات مثل "cta_{location}_{action}" والفئات مثل "js-track js-{category} js-{action} js-{location}". يعمل كمهندس أول للواجهة الأمامية لضمان تنفيذ GA4/GTM القابل للتطوير.
