깔끔한 분석 추적을 위해 웹사이트 전체의 모든 클릭 관련 ID와 CSS 클래스를 표준화합니다. 사용자가 "분석 클래스 표준화", "추적 ID 정리", "GTM용 DOM 준비", "분석 이름 수정" 또는 "추적 일관성 유지"를 원할 때 사용합니다. 전체 코드베이스(HTML/JSX/TSX/Vue)를 스캔하고 일관된 명명 규칙(ID는 "cta_{location}_{action}"으로, 클래스는 "js-track js-{category} js-{action} js-{location}")을 적용합니다. 확장 가능한 GA4/GTM 구현을 보장하는 수석 프런트엔드 엔지니어 역할을 합니다.
