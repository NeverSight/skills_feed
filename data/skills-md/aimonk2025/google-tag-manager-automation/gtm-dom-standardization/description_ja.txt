クリーンな分析追跡のために、Web サイト全体でクリック関連のすべての ID と CSS クラスを標準化します。ユーザーが「分析クラスの標準化」、「トラッキング ID のクリーンアップ」、「GTM 用の DOM の準備」、「分析の名前付けの修正」、「トラッキングの一貫性の確保」を行う場合に使用します。コードベース全体 (HTML/JSX/TSX/Vue) をスキャンし、一貫した命名規則を適用します。ID は「cta_{location}_{action}」、クラスは「js-track js-{category} js-{action} js-{location}」です。上級フロントエンド エンジニアとして機能し、スケーラブルな GA4/GTM 実装を保証します。
