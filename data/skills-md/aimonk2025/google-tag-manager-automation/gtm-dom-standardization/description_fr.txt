Standardise tous les identifiants et classes CSS liés aux clics sur l’ensemble du site Web pour un suivi analytique propre. À utiliser lorsque les utilisateurs souhaitent « normaliser les classes d'analyse », « nettoyer les ID de suivi », « préparer le DOM pour GTM », « corriger la dénomination des analyses » ou « rendre le suivi cohérent ». Analyse l'intégralité de la base de code (HTML/JSX/TSX/Vue) et applique une convention de dénomination cohérente - ID comme "cta_{location}_{action}" et classes comme "js-track js-{category} js-{action} js-{location}". Agit en tant qu'ingénieur frontend principal assurant une mise en œuvre évolutive de GA4/GTM.
