Стандартизирует все идентификаторы кликов и классы CSS на веб-сайте для чистого аналитического отслеживания. Используйте, когда пользователи хотят «стандартизировать классы аналитики», «очистить идентификаторы отслеживания», «подготовить DOM для GTM», «исправить именование аналитики» или «сделать отслеживание единообразным». Сканирует всю базу кода (HTML/JSX/TSX/Vue) и применяет согласованное соглашение об именах — идентификаторы как «cta_{location}_{action}» и классы как «js-track js-{category} js-{action} js-{location}». Выполняет функции старшего фронтенд-инженера, обеспечивая масштабируемую реализацию GA4/GTM.
