Tests et validation complets du suivi GTM, y compris les tests automatisés sans tête Playwright, les tests de la console du navigateur, la validation du mode Aperçu GTM et la vérification GA4 DebugView. À utiliser lorsque les utilisateurs doivent « tester le suivi GTM », « valider les événements dataLayer », « déboguer GTM », « vérifier si le suivi fonctionne », « tests de suivi automatisés », « exécuter des tests de suivi sans ouvrir le navigateur » ou résoudre des problèmes de suivi. Donne la priorité aux tests automatisés plutôt qu’aux tests manuels lorsque cela est possible.
