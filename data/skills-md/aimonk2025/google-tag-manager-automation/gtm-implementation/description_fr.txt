Implémente un suivi GTM complet, y compris les événements dataLayer dans le code et la configuration GTM via l'API. À utiliser lorsque les utilisateurs doivent « mettre en œuvre le suivi GTM », « ajouter des événements dataLayer », « créer des variables et des balises GTM », « configurer le suivi CTA », « mettre en œuvre le suivi des événements » ou souhaitent exécuter un plan de suivi. Gère automatiquement l'implémentation du code (dataLayer.push) et la configuration du conteneur GTM (variables/déclencheurs/balises) via l'API. Prend en charge les mises à jour incrémentielles et les modèles spécifiques au framework (React, Next.js, Vue, etc.).
