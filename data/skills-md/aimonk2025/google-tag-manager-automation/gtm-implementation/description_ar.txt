ينفذ تتبع GTM الكامل بما في ذلك أحداث dataLayer في التعليمات البرمجية وتكوين GTM عبر واجهة برمجة التطبيقات. يُستخدم عندما يحتاج المستخدمون إلى "تنفيذ تتبع GTM"، أو "إضافة أحداث dataLayer"، أو "إنشاء متغيرات وعلامات GTM"، أو "إعداد تتبع CTA"، أو "تنفيذ تتبع الأحداث"، أو الرغبة في تنفيذ خطة تتبع. يتعامل مع تنفيذ التعليمات البرمجية (dataLayer.push) وتكوين حاوية GTM (المتغيرات/المشغلات/العلامات) تلقائيًا عبر واجهة برمجة التطبيقات. يدعم التحديثات المتزايدة والأنماط الخاصة بإطار العمل (React وNext.js وVue وما إلى ذلك).
