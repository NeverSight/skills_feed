Unicon アイコン ライブラリを使用して、ユーザーがプロジェクトにアイコンを追加できるようにします。 Unicon は、Lucide、Phosphor、Hugeicons、Heroicons、Tabler、Feather、Remix、Simple Icons (ブランド ロゴ)、Iconoir の 19,000 以上のアイコンを提供します。 React、Vue、Svelte、または Web プロジェクトにアイコンを追加するときに使用します。 unicon CLI を使用してアイコンを検索、取得、またはバンドルします。 .uniconrc.json 構成をセットアップします。ツリーシェイク可能なアイコンコンポーネントを生成します。 Unicon API を使用する。またはアイコン形式間の変換。
