Integrazione del server MCP per memorie.sh: il livello di memoria persistente per gli agenti AI. Utilizzare quando: (1) configurare il server memorie.sh MCP per qualsiasi client (Claude Code, Cursor, Windsurf, VS Code, v0, Claude Desktop, OpenCode, Factory), (2) utilizzare strumenti MCP per archiviare, cercare o recuperare memorie a livello di codice, (3) comprendere get_context, search_memories e list_memories, (4) lavorare con strumenti di memoria di streaming per contenuti SSE, (5) risolvere i problemi di connessione MCP, (6) scegliere tra cloud Trasporti MCP (HTTP) e MCP locali (stdio).
