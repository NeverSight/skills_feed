Memory.sh の MCP サーバー統合 — AI エージェントの永続メモリ層。以下の場合に使用します: (1) 任意のクライアント (Claude Code、Cursor、Windsurf、VS Code、v0、Claude Desktop、OpenCode、Factory) 用にmemories.sh MCP サーバーを構成する、(2) MCP ツールを使用してプログラムでメモリを保存、検索、または取得する、(3) get_context、search_memories、list_memories を理解する、(4) SSE コンテンツのストリーミング メモリ ツールを使用する、(5) MCP 接続の問題のトラブルシューティング、 (6) クラウド MCP (HTTP) トランスポートとローカル MCP (stdio) トランスポートのどちらかを選択します。
