تكامل خادم MCP لـ Memory.sh — طبقة الذاكرة المستمرة لوكلاء الذكاء الاصطناعي. يُستخدم عندما: (1) تكوين خادم Memes.sh MCP لأي عميل (Claude Code، Cursor، Windsurf، VS Code، v0، Claude Desktop، OpenCode، Factory)، (2) استخدام أدوات MCP لتخزين الذكريات أو البحث عنها أو استرجاعها برمجيًا، (3) فهم get_context vs search_memories vs list_memories، (4) العمل مع أدوات الذاكرة المتدفقة لمحتوى SSE، (5) استكشاف مشكلات اتصال MCP وإصلاحها، (6) الاختيار بين وسائل النقل السحابية MCP (HTTP) وMCP المحلية (stdio).
