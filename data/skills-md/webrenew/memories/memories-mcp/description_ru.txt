Интеграция сервера MCP для Memory.sh — уровня постоянной памяти для агентов ИИ. Используйте, когда: (1) Настройка сервера MCP Memory.sh для любого клиента (Claude Code, Cursor, Windsurf, VS Code, v0, Claude Desktop, OpenCode, Factory), (2) Использование инструментов MCP для программного хранения, поиска или извлечения воспоминаний, (3) Понимание get_context, search_memories и list_memories, (4) Работа с инструментами потоковой передачи памяти для содержимого SSE, (5) Устранение неполадок соединения MCP проблемы, (6) Выбор между облачным транспортом MCP (HTTP) и локальным транспортом MCP (stdio).
