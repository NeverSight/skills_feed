Guide du développeur pour contribuer à et étendre la base de code souvenirs.sh. À utiliser lorsque : (1) Comprendre l'architecture souvenirs.sh et la façon dont les packages se connectent, (2) Ajouter de nouvelles commandes CLI ou outils MCP, (3) Modifier la couche de stockage mémoire (SQLite/libSQL), (4) Travailler sur le tableau de bord Web (Next.js/Supabase), (5) Ajouter des cibles de nouvelle génération pour les outils d'IA, (6) Étendre la fonctionnalité de synchronisation ou d'intégration dans le cloud, (7) Déboguer les problèmes de construction, de test ou de déploiement dans le monorepo.
