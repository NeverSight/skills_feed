Развертывайте веб-приложения, API, статические сайты и периферийные функции на бесплатном уровне Cloudflare с помощью Wrangler и Cloudflared CLI. Используйте, когда пользователь хочет (1) разместить/развернуть веб-сайт, блог или веб-приложение, (2) создать бессерверные API или пограничные функции, (3) настроить базы данных (SQL или KV), (4) хранить файлы/объекты, (5) предоставить доступ к локальному хосту через туннель, (6) бесплатно развернуть любой личный проект в облаке. Охватывает рабочих, страницы, D1, KV, R2, прочные объекты, рабочий искусственный интеллект, туннели, маршрутизацию электронной почты. Все развертывания не выходят за пределы уровня бесплатного пользования (0 долларов США).
