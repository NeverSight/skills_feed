Wrangler と Cloudflared CLI を使用して、Web アプリ、API、静的サイト、エッジ機能を Cloudflare の無料枠にデプロイします。ユーザーが、(1) Web サイト、ブログ、または Web アプリをホスト/デプロイする、(2) サーバーレス API またはエッジ機能を作成する、(3) データベース (SQL または KV) をセットアップする、(4) ファイル/オブジェクトを保存する、(5) トンネル経由でローカルホストを公開する、(6) 個人プロジェクトをクラウドに無料でデプロイする場合に使用します。ワーカー、ページ、D1、KV、R2、耐久性オブジェクト、ワーカー AI、トンネル、電子メール ルーティングをカバーします。すべての展開は、$0 の無料利用枠の制限内に収まります。
