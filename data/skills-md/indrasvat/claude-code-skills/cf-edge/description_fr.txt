Déployez des applications Web, des API, des sites statiques et des fonctions périphériques sur l'offre gratuite de Cloudflare à l'aide des CLI Wrangler et Cloudflared. À utiliser lorsque l'utilisateur souhaite (1) héberger/déployer un site Web, un blog ou une application Web, (2) créer des API ou des fonctions de périphérie sans serveur, (3) configurer des bases de données (SQL ou KV), (4) stocker des fichiers/objets, (5) exposer localhost via un tunnel, (6) déployer gratuitement n'importe quel projet personnel dans le cloud. Couvre les travailleurs, les pages, D1, KV, R2, les objets durables, l'IA des travailleurs, les tunnels, le routage des e-mails. Tous les déploiements restent dans les limites du niveau gratuit de 0 $.
