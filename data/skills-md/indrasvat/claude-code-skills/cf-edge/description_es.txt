Implemente aplicaciones web, API, sitios estáticos y funciones perimetrales en el nivel gratuito de Cloudflare mediante las CLI de Wrangler y Cloudflared. Úselo cuando el usuario desee (1) alojar/implementar un sitio web, blog o aplicación web, (2) crear API sin servidor o funciones perimetrales, (3) configurar bases de datos (SQL o KV), (4) almacenar archivos/objetos, (5) exponer localhost a través de un túnel, (6) implementar cualquier proyecto personal en la nube de forma gratuita. Cubre trabajadores, páginas, D1, KV, R2, objetos duraderos, IA de trabajadores, túneles y enrutamiento de correo electrónico. Todas las implementaciones se mantienen dentro de los límites del nivel gratuito de $0.
