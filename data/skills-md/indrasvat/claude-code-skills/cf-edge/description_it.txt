Distribuisci app web, API, siti statici e funzioni edge al livello gratuito di Cloudflare utilizzando le CLI wrangler e cloudflared. Da utilizzare quando l'utente desidera (1) ospitare/distribuire un sito Web, un blog o un'app Web, (2) creare API serverless o funzioni edge, (3) configurare database (SQL o KV), (4) archiviare file/oggetti, (5) esporre localhost tramite tunnel, (6) distribuire gratuitamente qualsiasi progetto personale nel cloud. Copre lavoratori, pagine, D1, KV, R2, oggetti durevoli, intelligenza artificiale dei lavoratori, tunnel, routing della posta elettronica. Tutte le distribuzioni rimangono entro i limiti del livello gratuito di $ 0.
