انشر تطبيقات الويب وواجهات برمجة التطبيقات والمواقع الثابتة ووظائف الحافة إلى الطبقة المجانية من Cloudflare باستخدام واجهات سطر الأوامر الخاصة بـ wrangler وcloudflared. يُستخدم عندما يريد المستخدم (1) استضافة/نشر موقع ويب أو مدونة أو تطبيق ويب، (2) إنشاء واجهات برمجة التطبيقات أو وظائف الحافة بدون خادم، (3) إعداد قواعد البيانات (SQL أو KV)، (4) تخزين الملفات/الكائنات، (5) كشف المضيف المحلي عبر النفق، (6) نشر أي مشروع شخصي على السحابة مجانًا. يغطي العمال، الصفحات، D1، KV، R2، الكائنات المعمرة، الذكاء الاصطناعي للعمال، الأنفاق، توجيه البريد الإلكتروني. تظل جميع عمليات النشر ضمن حدود الطبقة المجانية البالغة 0 دولار.
