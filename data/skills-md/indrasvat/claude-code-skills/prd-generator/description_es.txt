Genere documentos completos de requisitos de productos con descubrimiento interactivo, seguimiento del progreso y soporte True Ralph Loop para una implementación autónoma. Úselo cuando el usuario desee (1) crear un PRD para un nuevo proyecto/función, (2) implementar un PRD de forma autónoma con nuevas sesiones de Claude, (3) realizar un seguimiento del progreso de la implementación, (4) recuperar el contexto después de la pérdida de la sesión. Crea docs/PRD.md y docs/PROGRESS.md.
