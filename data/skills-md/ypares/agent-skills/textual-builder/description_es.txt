Cree aplicaciones de interfaz de usuario de texto (TUI) utilizando el marco Textual Python (v0.86.0+). Úselo al crear aplicaciones basadas en terminales, crear prototipos de juegos de cartas o CLI interactivas, o cuando el usuario mencione Textual, TUI o UI de terminal. Incluye documentación de referencia completa, plantilla inicial para juegos de cartas y guías de estilo.
