Créez des applications d'interface utilisateur de texte (TUI) à l'aide du framework Textual Python (v0.86.0+). À utiliser lors de la création d'applications basées sur un terminal, du prototypage de jeux de cartes ou de CLI interactives, ou lorsque l'utilisateur mentionne Textual, TUI ou l'interface utilisateur du terminal. Comprend une documentation de référence complète, un modèle de démarrage de jeu de cartes et des guides de style.
