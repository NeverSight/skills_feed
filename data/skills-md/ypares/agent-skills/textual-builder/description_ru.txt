Создавайте приложения с текстовым пользовательским интерфейсом (TUI) с помощью платформы Textual Python (v0.86.0+). Используйте при создании приложений на базе терминала, прототипировании карточных игр или интерактивных интерфейсов командной строки, а также когда пользователь упоминает Textual, TUI или пользовательский интерфейс терминала. Включает подробную справочную документацию, стартовый шаблон карточной игры и руководства по стилю.
