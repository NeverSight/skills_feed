Руководство по созданию плагинов Nushell в Rust с использованием крейтов nu_plugin и nu_protocol. Используйте, когда пользователи хотят создавать собственные команды Nushell, расширять Nushell новыми функциями, создавать преобразования данных или интегрировать внешние инструменты/API в Nushell. Охватывает настройку проекта, реализацию команд, потоковую передачу данных, пользовательские значения и тестирование.
