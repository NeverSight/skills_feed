Guía para crear complementos de Nushell en Rust usando cajas nu_plugin y nu_protocol. Úselo cuando los usuarios quieran crear comandos Nushell personalizados, ampliar Nushell con nuevas funciones, crear transformaciones de datos o integrar herramientas/API externas en Nushell. Cubre la configuración del proyecto, la implementación de comandos, la transmisión de datos, los valores personalizados y las pruebas.
