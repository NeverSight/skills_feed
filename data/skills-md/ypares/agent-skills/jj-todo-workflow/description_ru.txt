Структурированный рабочий процесс фиксации TODO с использованием JJ (дзюдзюцу). Используйте для планирования задач в виде пустых коммитов с флагами [task:*], отслеживания прогресса при смене состояний, управления группами обеспечения доступности баз данных параллельных задач с проверкой зависимостей. Обеспечивает дисциплину завершения. Позволяет разделить работу между планировщиками и работниками. **Требуется навык работы с jj**
