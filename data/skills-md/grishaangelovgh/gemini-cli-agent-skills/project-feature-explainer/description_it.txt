Guida esperta per spiegare le caratteristiche del progetto. Utilizza questa competenza quando devi fornire una spiegazione completa del funzionamento di una funzionalità specifica, inclusi riepiloghi, approfondimenti, esempi di utilizzo e diagrammi di sequenza/flusso di lavoro. Questa abilità ha una directory di riferimenti che contiene istruzioni aggiuntive `checklist.md`, `example-output.md` e `explanation-template.md` che DEVONO essere utilizzate per ogni analisi.
