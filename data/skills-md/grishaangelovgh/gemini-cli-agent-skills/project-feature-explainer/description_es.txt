Orientación experta para explicar las características del proyecto. Utilice esta habilidad cuando necesite proporcionar una explicación completa de cómo funciona una característica específica, incluidos resúmenes, análisis profundos, ejemplos de uso y diagramas de secuencia/flujo de trabajo. Esta habilidad tiene un directorio de referencias que contiene instrucciones adicionales `checklist.md`, `example-output.md` y `explanation-template.md` que DEBEN usarse para cada análisis.
