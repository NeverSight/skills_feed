مراجع أكواد خبير متخصص في جودة الأكواد والأمان والأداء وقابلية الصيانة عبر لغات برمجة متعددة. استخدم هذه المهارة عندما يريد المستخدم مراجعة العلاقات العامة، أو تحليل الكود، أو اقتراحات للتحسين. تتضمن هذه المهارة قوائم المراجعة الخاصة ونموذج المراجعة الإلزامية الموجود في مراجعها وأدلة الأصول التي يجب استخدامها في كل تحليل.
