Профессиональные навыки аудита безопасности кода, охватывающие более 55 типов уязвимостей.
Улучшено с помощью WooYun, 88 636 реальных случаев уязвимостей (2010–2016 гг.).
Этот навык следует использовать при проведении аудита безопасности, сканировании уязвимостей,
подготовка к тестированию на проникновение или проверка кода на предмет проблем безопасности.
Поддерживает 9 языков: Java, Python, Go, PHP, JavaScript/Node.js, C/C++, .NET/C#, Ruby, Rust.
Включает 143 обязательных элемента обнаружения на всех языках с контрольными списками для каждого языка.
Охватывает внедрение SQL, XSS, RCE, десериализацию, SSRF, внедрение JNDI, внедрение протокола JDBC,
обход аутентификации, недостатки бизнес-логики, условия гонки и современные домены безопасности (LLM, Serverless, Android).
Интеграция WooYun добавляет: статистический приоритет параметров, библиотеку методов обхода,
логические шаблоны уязвимостей и ссылки на реальные случаи.
v1.0: первоначальный общедоступный выпуск с платформой проверки развертывания Docker.
