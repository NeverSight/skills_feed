55 種類以上の脆弱性をカバーする専門的なコード セキュリティ監査スキル。
WooYun 88,636 件の実世界の脆弱性ケース (2010 ～ 2016 年) で強化されました。
このスキルは、セキュリティ監査、脆弱性スキャン、
侵入テストの準備、またはセキュリティ問題のコード レビュー。
9 つの言語をサポートします: Java、Python、Go、PHP、JavaScript/Node.js、C/C++、.NET/C#、Ruby、Rust。
すべての言語にわたる 143 の必須検出項目と言語固有のチェックリストが含まれています。
SQL インジェクション、XSS、RCE、デシリアライゼーション、SSRF、JNDI インジェクション、JDBC プロトコル インジェクション、
認証バイパス、ビジネス ロジックの欠陥、競合状態、最新のセキュリティ ドメイン (LLM、サーバーレス、Android)。
WooYun の統合により、統計主導のパラメータ優先順位、バイパス技術ライブラリ、
ロジックの脆弱性パターン、および実際のケースの参照。
v1.0: Docker デプロイメント検証フレームワークを備えた最初のパブリック リリース。
