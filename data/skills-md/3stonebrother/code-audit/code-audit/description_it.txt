Competenze professionali di audit della sicurezza del codice che coprono oltre 55 tipi di vulnerabilità.
Migliorato con WooYun 88.636 casi di vulnerabilità nel mondo reale (2010-2016).
Questa abilità dovrebbe essere utilizzata durante l'esecuzione di controlli di sicurezza, scansione delle vulnerabilità,
preparazione ai test di penetrazione o revisione del codice per problemi di sicurezza.
Supporta 9 lingue: Java, Python, Go, PHP, JavaScript/Node.js, C/C++, .NET/C#, Ruby, Rust.
Include 143 elementi di rilevamento obbligatori in tutte le lingue con elenchi di controllo specifici per la lingua.
Copre l'iniezione SQL, XSS, RCE, deserializzazione, SSRF, iniezione JNDI, iniezione protocollo JDBC,
bypass dell'autenticazione, difetti della logica aziendale, condizioni di competizione e domini di sicurezza moderni (LLM, Serverless, Android).
L'integrazione di WooYun aggiunge: priorità dei parametri basati su statistiche, libreria di tecniche di bypass,
modelli di vulnerabilità logica e riferimenti a casi reali.
v1.0: rilascio pubblico iniziale con il framework di verifica della distribuzione Docker.
