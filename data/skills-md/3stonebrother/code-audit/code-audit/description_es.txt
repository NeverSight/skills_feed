Habilidad profesional de auditoría de seguridad de código que cubre más de 55 tipos de vulnerabilidad.
Mejorado con WooYun 88,636 casos de vulnerabilidad del mundo real (2010-2016).
Esta habilidad debe usarse al realizar auditorías de seguridad, escaneo de vulnerabilidades,
preparación de pruebas de penetración o revisión de código para problemas de seguridad.
Admite 9 idiomas: Java, Python, Go, PHP, JavaScript/Node.js, C/C++, .NET/C#, Ruby, Rust.
Incluye 143 elementos de detección obligatorios en todos los idiomas con listas de verificación específicas del idioma.
Cubre inyección SQL, XSS, RCE, deserialización, SSRF, inyección JNDI, inyección de protocolo JDBC,
omisión de autenticación, fallas en la lógica empresarial, condiciones de carrera y dominios de seguridad modernos (LLM, Serverless, Android).
La integración de WooYun agrega: prioridad de parámetros basada en estadísticas, biblioteca de técnicas de derivación,
patrones de vulnerabilidad lógica y referencias de casos reales.
v1.0: lanzamiento público inicial con el marco de verificación de implementación de Docker.
