Compétence professionnelle en audit de sécurité du code couvrant plus de 55 types de vulnérabilités.
Amélioré avec WooYun 88 636 cas de vulnérabilité réels (2010-2016).
Cette compétence doit être utilisée lors de la réalisation d'audits de sécurité, d'analyses de vulnérabilités,
préparation aux tests d'intrusion ou révision du code pour les problèmes de sécurité.
Prend en charge 9 langages : Java, Python, Go, PHP, JavaScript/Node.js, C/C++, .NET/C#, Ruby, Rust.
Comprend 143 éléments de détection obligatoires dans toutes les langues avec des listes de contrôle spécifiques à chaque langue.
Couvre l'injection SQL, XSS, RCE, la désérialisation, SSRF, l'injection JNDI, l'injection du protocole JDBC,
contournement d'authentification, failles de logique métier, conditions de concurrence et domaines de sécurité modernes (LLM, Serverless, Android).
L'intégration WooYun ajoute : priorité aux paramètres basés sur les statistiques, bibliothèque de techniques de contournement,
modèles de vulnérabilité logique et références de cas réels.
v1.0 : version publique initiale avec cadre de vérification du déploiement Docker.
