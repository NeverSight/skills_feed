55개 이상의 취약점 유형을 다루는 전문 코드 보안 감사 기술입니다.
WooYun 88,636건의 실제 취약점 사례(2010-2016)로 강화되었습니다.
이 기술은 보안 감사, 취약점 스캔,
침투 테스트 준비 또는 보안 문제에 대한 코드 검토.
9개 언어를 지원합니다: Java, Python, Go, PHP, JavaScript/Node.js, C/C++, .NET/C#, Ruby, Rust.
언어별 체크리스트와 함께 모든 언어에 걸쳐 143개의 필수 탐지 항목이 포함되어 있습니다.
SQL 주입, XSS, RCE, 역직렬화, SSRF, JNDI 주입, JDBC 프로토콜 주입,
인증 우회, 비즈니스 로직 결함, 경쟁 조건 및 최신 보안 도메인(LLM, 서버리스, Android).
WooYun 통합에는 통계 기반 매개변수 우선순위, 우회 기술 라이브러리,
논리 취약점 패턴 및 실제 사례 참조.
v1.0: Docker 배포 확인 프레임워크가 포함된 최초 공개 릴리스입니다.
