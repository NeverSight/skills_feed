مهارة تدقيق أمان التعليمات البرمجية الاحترافية التي تغطي أكثر من 55 نوعًا من الثغرات الأمنية.
معزز بـ WooYun 88,636 حالة ضعف في العالم الحقيقي (2010-2016).
يجب استخدام هذه المهارة عند إجراء عمليات التدقيق الأمني، وفحص الثغرات الأمنية،
التحضير لاختبار الاختراق، أو مراجعة التعليمات البرمجية لقضايا الأمان.
يدعم 9 لغات: Java، Python، Go، PHP، JavaScript/Node.js، C/C++، .NET/C#، Ruby، Rust.
يتضمن 143 عنصر كشف إلزاميًا عبر جميع اللغات مع قوائم مرجعية خاصة بكل لغة.
يغطي حقن SQL، XSS، RCE، إلغاء التسلسل، SSRF، حقن JNDI، حقن بروتوكول JDBC،
تجاوز المصادقة، وعيوب منطق الأعمال، وظروف السباق، ومجالات الأمان الحديثة (LLM، بدون خادم، Android).
يضيف تكامل WooYun: أولوية المعلمات المستندة إلى الإحصائية، ومكتبة تقنيات التجاوز،
أنماط الضعف المنطقي، ومراجع الحالة الحقيقية.
الإصدار 1.0: إصدار عام أولي مع إطار التحقق من نشر Docker.
