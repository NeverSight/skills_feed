Sur la base du framework Plugin jy-aiot (mine Hongyan), génère du code de module de collecte en mode hybride (config + extraits) à partir d’un document de protocole de collecte au format tableau Markdown. Permet d’implémenter rapidement des modules pour HTTP serveur/client, FTP, base de données, MQTT, Kafka, etc., et de simplifier le développement de composants de collecte sur protocoles personnalisés.
