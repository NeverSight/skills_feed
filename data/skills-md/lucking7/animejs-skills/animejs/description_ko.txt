Anime.js v4로 JavaScript 애니메이션을 만들 때, 웹 애니메이션을 구현할 때 또는 사용자가 anime.js/animejs를 언급할 때 사용합니다. v3 코드 증상에 대해 트리거합니다: animate() 대신 anime(), x 대신 translateX, outExpo 대신 easyOutExpo, createTimeline() 대신 anime.timeline(). 또한 CSS 애니메이션, SVG 애니메이션, 스크롤 기반 애니메이션, 드래그 상호 작용, 텍스트 분할, 레이아웃 FLIP, 스프링 물리학, JavaScript의 스태거 패턴에 대해 트리거됩니다.
