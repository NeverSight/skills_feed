Используйте при создании анимации JavaScript с помощью Anime.js v4, реализации веб-анимации или когда пользователь упоминает аниме.js/animejs. Триггеры по симптомам кода v3: Anime() вместо animate(), TranslateX вместо x, EasyOutExpo вместо outExpo, Anime.timeline() вместо createTimeline(). Также срабатывает: CSS-анимация, SVG-анимация, анимация с помощью прокрутки, взаимодействие перетаскивания, разделение текста, FLIP макета, пружинная физика, шаблоны смещения в JavaScript.
