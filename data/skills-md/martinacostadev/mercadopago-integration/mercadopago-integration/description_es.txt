Úselo cuando el usuario necesite: (1) Agregar procesamiento de pagos de MercadoPago a una aplicación Next.js, (2) Crear un flujo de pago con MercadoPago, (3) Configurar webhooks de pago para MercadoPago, (4) Crear páginas de éxito/error de pago, (5) Crear un carrito de compras con integración de pago, (6) Solucionar problemas de integración de MercadoPago (errores de devolución automática, fallas de webhook, discrepancias de hidratación, envíos dobles).
