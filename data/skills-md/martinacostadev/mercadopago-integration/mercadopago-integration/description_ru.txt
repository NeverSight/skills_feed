Интегрируйте MercadoPago Checkout Pro (на основе перенаправления) в приложения Next.js с любой базой данных PostgreSQL (Supabase, AWS RDS, Neon, PlanetScale, самостоятельный хостинг, Prisma, Drizzle или необработанный pg). Используйте, когда пользователю необходимо: (1) добавить обработку платежей MercadoPago в приложение Next.js, (2) создать поток оформления заказа с помощью MercadoPago, (3) настроить веб-перехватчики платежей для MercadoPago, (4) создать страницы успеха/неуспеха платежа, (5) создать корзину покупок с интеграцией платежей, (6) устранить проблемы интеграции MercadoPago (ошибки auto_return, сбои веб-перехватчика, несоответствия гидратации, двойные отправки). Запускается по запросам, упоминающим MercadoPago, Mercado Pago, интеграцию платежей с MP, обработку платежей в Аргентине/Латинской Америке или оформление заказа с помощью MercadoPago. Поддерживает все страны MercadoPago: Аргентина (ARS), Бразилия (BRL), Мексика (MXN), Колумбия (COP), Чили (CLP), Перу (PEN), Уругвай (UYU).
