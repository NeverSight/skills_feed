قم بدمج MercadoPago Checkout Pro (القائم على إعادة التوجيه) في تطبيقات Next.js مع أي قاعدة بيانات PostgreSQL (Supabase أو AWS RDS أو Neon أو PlanetScale أو الاستضافة الذاتية أو Prisma أو Drizzle أو Raw pg). يُستخدم عندما يحتاج المستخدم إلى: (1) إضافة معالجة دفع MercadoPago إلى تطبيق Next.js، (2) إنشاء تدفق الدفع باستخدام MercadoPago، (3) إعداد خطافات الويب للدفع لـ MercadoPago، (4) إنشاء صفحات نجاح/فشل الدفع، (5) إنشاء عربة تسوق مع تكامل الدفع، (6) استكشاف مشكلات تكامل MercadoPago وإصلاحها (أخطاء الإرجاع التلقائي، وفشل خطاف الويب، وعدم تطابق الترطيب، والنقر المزدوج التقديمات). يتم تشغيلها عند الطلب الذي يشير إلى MercadoPago، أو Mercado Pago، أو تكامل الدفع مع MP، أو معالجة الدفع في الأرجنتين/أمريكا اللاتينية، أو الدفع باستخدام MercadoPago. يدعم جميع بلدان MercadoPago: الأرجنتين (ARS)، البرازيل (BRL)، المكسيك (MXN)، كولومبيا (COP)، تشيلي (CLP)، بيرو (PEN)، أوروغواي (UYU).
