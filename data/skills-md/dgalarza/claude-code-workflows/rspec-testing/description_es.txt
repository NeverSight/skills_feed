Esta habilidad debe usarse al escribir, revisar o mejorar pruebas RSpec para aplicaciones Ruby on Rails. Utilice esta habilidad para todas las tareas de prueba, incluidas las especificaciones de modelos, especificaciones de controladores, especificaciones de sistemas, especificaciones de componentes, especificaciones de servicios y pruebas de integración. La habilidad proporciona las mejores prácticas integrales de RSpec de Better Specs y guías de thinkbot.
