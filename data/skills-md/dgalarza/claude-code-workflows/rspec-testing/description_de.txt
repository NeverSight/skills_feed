Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Verbessern von RSpec-Tests für Ruby on Rails-Anwendungen verwendet werden. Verwenden Sie diese Fähigkeit für alle Testaufgaben, einschließlich Modellspezifikationen, Controller-Spezifikationen, Systemspezifikationen, Komponentenspezifikationen, Servicespezifikationen und Integrationstests. Der Skill bietet umfassende RSpec-Best Practices aus Better Specs und Thoughtbot-Anleitungen.
