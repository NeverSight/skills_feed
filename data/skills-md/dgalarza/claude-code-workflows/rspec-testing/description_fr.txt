Cette compétence doit être utilisée lors de l'écriture, de la révision ou de l'amélioration des tests RSpec pour les applications Ruby on Rails. Utilisez cette compétence pour toutes les tâches de test, y compris les spécifications de modèle, les spécifications de contrôleur, les spécifications de système, les spécifications de composants, les spécifications de service et les tests d'intégration. La compétence fournit des bonnes pratiques RSpec complètes issues de Better Specs et de guides Thoughtbot.
