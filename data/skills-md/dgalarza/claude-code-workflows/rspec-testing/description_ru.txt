Этот навык следует использовать при написании, проверке или улучшении тестов RSpec для приложений Ruby on Rails. Используйте этот навык для всех задач тестирования, включая спецификации модели, спецификации контроллера, спецификации системы, спецификации компонентов, спецификации услуг и интеграционные тесты. Этот навык предоставляет исчерпывающие передовые методы RSpec из Better Specs и руководств thinkbot.
