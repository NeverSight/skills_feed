Этот навык следует использовать при создании коммитов Git, чтобы гарантировать, что они соответствуют спецификации обычных коммитов. Он содержит рекомендации по структуре, типам, областям сообщения о фиксации и лучшим практикам написания четких, последовательных и удобных для автоматизации сообщений о фиксации. Используйте при фиксации изменений кода или просмотре истории коммитов.
