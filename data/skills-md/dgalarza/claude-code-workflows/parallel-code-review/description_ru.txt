Этот навык следует использовать при выполнении комплексных проверок кода с использованием нескольких специализированных агентов проверки параллельно. Он предоставляет шаблоны для одновременного выполнения, отслеживания решений для предотвращения избыточности и консолидированной отчетности. Используйте, когда требуется тщательная проверка с разных точек зрения (безопасность, архитектура, производительность) или при проверке больших наборов изменений.
