Diese Fähigkeit sollte bei der Durchführung umfassender Codeüberprüfungen mit mehreren spezialisierten Überprüfungsagenten parallel eingesetzt werden. Es bietet Muster für die gleichzeitige Ausführung, Entscheidungsverfolgung zur Vermeidung von Redundanz und konsolidierte Berichterstattung. Verwenden Sie diese Option, wenn Sie eine umfassende Überprüfungsabdeckung aus mehreren Perspektiven (Sicherheit, Architektur, Leistung) benötigen oder wenn Sie große Änderungssätze überprüfen.
