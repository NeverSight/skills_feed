Esta habilidad se debe utilizar al realizar revisiones integrales de código utilizando varios agentes de revisión especializados en paralelo. Proporciona patrones para ejecución simultánea, seguimiento de decisiones para evitar redundancias e informes consolidados. Úselo cuando necesite una cobertura de revisión exhaustiva desde múltiples perspectivas (seguridad, arquitectura, rendimiento) o cuando revise grandes conjuntos de cambios.
