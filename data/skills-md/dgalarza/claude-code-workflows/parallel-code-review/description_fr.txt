Cette compétence doit être utilisée lors de la réalisation de révisions complètes de code en utilisant plusieurs agents de révision spécialisés en parallèle. Il fournit des modèles d'exécution simultanée, un suivi des décisions pour éviter la redondance et des rapports consolidés. À utiliser lorsque vous avez besoin d'une couverture de révision approfondie sous plusieurs perspectives (sécurité, architecture, performances) ou lors de la révision d'ensembles de modifications importants.
