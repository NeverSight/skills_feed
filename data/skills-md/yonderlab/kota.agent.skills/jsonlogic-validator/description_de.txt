Implementieren, validieren und testen Sie JSONLogic-Regeln für portable Geschäftslogik. Verwenden Sie es, wenn Sie mit der JSONLogic-Syntax arbeiten, Regeln für bedingte Logik erstellen, Regelstrukturen validieren, Regeln anhand von Daten testen oder Geschäftsanforderungen in JSONLogic konvertieren. Wird bei Anfragen zum „Schreiben von JSONLogic“, „Validieren von JSONLogic“, „Erstellen einer Regel“, „Geschäftslogik als JSON“, „Bedingte Logik“ oder jeder Erwähnung von JSONLogic-Regeln ausgelöst.
