قم بتطبيق أفضل ممارسات وضع إطار عمل React Router 7، بما في ذلك جلب البيانات من الخادم أولاً، وأدوات التحميل/الإجراءات الآمنة للنوع، واستراتيجيات الترطيب المناسبة، ومصادقة البرامج الوسيطة، والتعامل مع البيانات الوصفية، وخطافات useMatches/useRouteLoaderData، والحد الأقصى من أمان النوع. يُستخدم عند العمل مع وضع إطار عمل React Router 7، أو تنفيذ أدوات التحميل، أو الإجراءات، أو حماية المسار، أو مسارات التنقل، أو البث باستخدام Suspense/Await، أو معلمات بحث URL، أو التحقق من صحة النموذج، أو واجهة مستخدم متفائلة، أو مسارات الموارد (نقاط نهاية API)، أو تكوين المسار، أو إنشاء تطبيقات SSR.
