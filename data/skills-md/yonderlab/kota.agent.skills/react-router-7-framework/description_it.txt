Applica le migliori pratiche della modalità framework React Router 7, tra cui il recupero dei dati server-first, caricatori/azioni type-safe, strategie di idratazione adeguate, autenticazione middleware, gestione dei metadati, hook useMatches/useRouteLoaderData e massima sicurezza dei tipi. Da utilizzare quando si lavora con la modalità framework React Router 7, implementando caricatori, azioni, protezione del percorso, breadcrumb, streaming con Suspense/Await, parametri di ricerca URL, convalida del modulo, interfaccia utente ottimistica, percorsi di risorse (endpoint API), configurazione del percorso o creazione di applicazioni SSR.
