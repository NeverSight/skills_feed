Applica le migliori pratiche di React durante la scrittura o la revisione del codice React. Da utilizzare durante la creazione di componenti, la revisione di PR, il refactoring del codice React, la risoluzione di problemi di prestazioni, il debug di nuovi rendering, la strutturazione dello stato e del flusso di dati, la conversione di useEffect in caricatori, la creazione di moduli o la richiesta di "come dovrei strutturare questo componente".
