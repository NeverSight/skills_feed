قم بتطبيق أفضل ممارسات React عند كتابة كود React أو مراجعته. يُستخدم عند إنشاء مكونات، أو مراجعة العلاقات العامة، أو إعادة هيكلة تعليمات React البرمجية، أو إصلاح مشكلات الأداء، أو تصحيح أخطاء عمليات إعادة التصيير، أو هيكلة الحالة وتدفق البيانات، أو تحويل useEffect إلى أدوات تحميل، أو بناء النماذج، أو طرح السؤال "كيف يمكنني هيكلة هذا المكون".
