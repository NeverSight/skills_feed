Datadog CLI pour le débogage et le tri. Utilisez cette compétence lorsque vous devez :
recherchez les journaux Datadog, les métriques de requêtes, les journaux de fin en temps réel, tracez les requêtes distribuées,
enquêter sur les erreurs, comparer les périodes, trouver des modèles de journaux, vérifier l'état du service,
ou exporter des données d’observabilité. Les expressions de déclenchement incluent "journaux de recherche", "journaux de queue",
« métriques de requête », « vérifier Datadog », « rechercher des erreurs », « demande de trace », « comparer les erreurs »,
"quels services existent", "modèles de journalisation", "utilisation du processeur", "état de santé du service".
