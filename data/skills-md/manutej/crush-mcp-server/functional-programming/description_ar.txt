إتقان مبادئ وأنماط البرمجة الوظيفية بما في ذلك النقاء، والثبات، والتركيب، والوظائف ذات الترتيب الأعلى، وأنواع البيانات الجبرية، والوظائف، والأحادية، وإدارة التأثير. يُستخدم عند العمل مع النماذج الوظيفية، أو الوظائف النقية، أو هياكل البيانات غير القابلة للتغيير، أو تكوين الوظيفة، أو معالجة الأخطاء الآمنة من النوع، أو تنفيذ الأنماط الوظيفية مثل الخريطة/التصفية/التقليل، أو الكاري، أو التطبيق الجزئي، أو التكرار، أو التقييم البطيء، أو الشفافية المرجعية.
