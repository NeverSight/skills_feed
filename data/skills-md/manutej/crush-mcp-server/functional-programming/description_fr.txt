Maîtrisez les principes et modèles de programmation fonctionnelle, notamment la pureté, l'immuabilité, la composition, les fonctions d'ordre supérieur, les types de données algébriques, les foncteurs, les monades et la gestion des effets. À utiliser lorsque vous travaillez avec des paradigmes fonctionnels, des fonctions pures, des structures de données immuables, une composition de fonctions, une gestion des erreurs de type sécurisé ou lorsque vous implémentez des modèles fonctionnels tels que mapper/filtrer/réduire, curry, application partielle, récursivité, évaluation paresseuse ou transparence référentielle.
