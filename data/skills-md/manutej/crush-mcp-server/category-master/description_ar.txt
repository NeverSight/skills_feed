معرفة نظرية فئة الخبراء للتفكير الرياضي الصارم. يُستخدم عند العمل مع الهياكل الفئوية، أو العوامل الوظيفية، أو التحويلات الطبيعية، أو الإضافات، أو الحدود، أو الموضعات، أو الفئات الأحادية، أو الفئات الغنية، أو الفئات الأعلى، أو العمليات، أو أي بناء فئوي رسمي. مثالية للبراهين، ومطاردات المخططات، والخصائص العالمية، وظروف التماسك، والتجريدات الرياضية الأساسية.
