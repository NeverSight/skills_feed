Fachkundige Anleitung zur Recherche, Dokumentation und Integration von Model Context Protocol (MCP)-Servern und -Tools. Behandelt MCP-Architektur, Server-/Client-Implementierungsmuster, Tool-Erkennung, Integrationsworkflows, bewährte Sicherheitspraktiken und mehrsprachige SDK-Nutzung (Python, TypeScript, C#, Java, Rust). Ermöglicht die nahtlose Integration von MCP-Tools in Claude Code- und KI-Anwendungen.
