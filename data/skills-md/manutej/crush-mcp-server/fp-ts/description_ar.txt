أتقن مكتبة fp-ts للبرمجة الوظيفية المكتوبة في TypeScript، بما في ذلك Option وAman وTask وTaskEither وReader وState وIO وArray وRecord وتركيب الأنابيب/التدفق وترميز العمل والبصريات (العدسات/المنشورات) والتكامل مع نظام Effect-TS البيئي. يُستخدم عند العمل مع أنواع بيانات fp-ts، أو إنشاء مسارات وظيفية، أو التعامل مع التأثيرات وظيفيًا، أو تنفيذ أنماط أحادية، أو استخدام أدوات fp-ts المساعدة للتعليمات البرمجية الوظيفية الآمنة للنوع.
