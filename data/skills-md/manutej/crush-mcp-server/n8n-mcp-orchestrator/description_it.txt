Orchestrazione MCP (Model Context Protocol) esperta con automazione del flusso di lavoro n8n. Padroneggia l'integrazione MCP bidirezionale, esponi i flussi di lavoro n8n come strumenti di agenti AI, utilizza server MCP nei flussi di lavoro, crea sistemi ad agenti, orchestra flussi di lavoro multi-agente e crea pipeline di automazione basate sull'intelligenza artificiale pronte per la produzione con l'integrazione di Claude Code.
