Ein Code-Inspektionstool basierend auf der offiziellen Dokumentation von Huawei HarmonyOS NEXT und den ArkTS-Spezifikationen. Es erkennt automatisch Spezifikationsverstöße, Leistungsprobleme und potenzielle Fehler im Code und stellt so die Einhaltung der Best Practices für die HarmonyOS-Anwendungsentwicklung sicher.
