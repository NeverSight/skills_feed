خبراء في كتابة استعلامات SQL وتحسينها وتصميم مخطط قاعدة البيانات مع دعم PostgreSQL وMySQL وSQLite وSQL Server. يُستخدم عند العمل مع قواعد البيانات من أجل: (1) كتابة استعلامات SQL معقدة مع الصلات والاستعلامات الفرعية ووظائف النافذة، (2) تحسين الاستعلامات البطيئة وتحليل خطط التنفيذ، (3) تصميم مخططات قاعدة البيانات مع التسوية المناسبة، (4) إنشاء الفهارس وتحسين أداء الاستعلام، (5) كتابة عمليات الترحيل ومعالجة تغييرات المخطط، (6) تصحيح أخطاء SQL ومشكلات الاستعلام
