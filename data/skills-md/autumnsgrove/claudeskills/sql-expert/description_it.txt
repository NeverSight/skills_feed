Scrittura esperta di query SQL, ottimizzazione e progettazione di schemi di database con supporto per PostgreSQL, MySQL, SQLite e SQL Server. Utilizzare quando si lavora con database per: (1) Scrivere query SQL complesse con join, sottoquery e funzioni finestra, (2) Ottimizzare query lente e analizzare piani di esecuzione, (3) Progettare schemi di database con la corretta normalizzazione, (4) Creare indici e migliorare le prestazioni delle query, (5) Scrivere migrazioni e gestire modifiche allo schema, (6) Debug di errori SQL e problemi di query
