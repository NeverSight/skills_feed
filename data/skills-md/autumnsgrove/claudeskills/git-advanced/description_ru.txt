Расширенные операции и рабочие процессы Git, включая интерактивное перебазирование, разрешение конфликтов, манипулирование историей, разделение ошибок пополам, выборку вишен, восстановление повторного журнала и стратегии управления ветвями. Используется для: (1) интерактивного перебазирования и очистки коммитов, (2) разрешения сложных конфликтов слияния, (3) Git bisect для поиска ошибок, (4) переписывания и очистки истории, (5) реализации стратегии ветвей (Git Flow, на основе магистрали), (6) восстановления потерянных коммитов с помощью reflog
