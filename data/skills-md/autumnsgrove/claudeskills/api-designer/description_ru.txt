Проектируйте и документируйте API-интерфейсы RESTful и GraphQL с использованием спецификаций OpenAPI/Swagger, шаблонов аутентификации, стратегий управления версиями и лучших практик. Используется для: (1) создания спецификаций API, (2) проектирования конечных точек REST, (3) проектирования схемы GraphQL, (4) аутентификации и авторизации API, (5) стратегий управления версиями API, (6) создания документации.
