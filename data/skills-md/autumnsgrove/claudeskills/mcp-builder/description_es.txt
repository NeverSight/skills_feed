Guía completa para crear servidores Model Context Protocol (MCP) con soporte para herramientas, recursos, indicaciones y autenticación. Úselo cuando: (1) cree servidores MCP personalizados, (2) integre API externas con Claude, (3) cree servidores de herramientas para dominios especializados, (4) cree proveedores de recursos para documentación, (5) implemente autenticación y seguridad
