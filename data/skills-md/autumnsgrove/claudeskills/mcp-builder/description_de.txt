Umfassende Anleitung zum Erstellen von Model Context Protocol (MCP)-Servern mit Unterstützung für Tools, Ressourcen, Eingabeaufforderungen und Authentifizierung. Wird verwendet, wenn: (1) benutzerdefinierte MCP-Server erstellt werden, (2) externe APIs mit Claude integriert werden, (3) Tool-Server für spezielle Domänen erstellt werden, (4) Ressourcenanbieter für die Dokumentation erstellt werden, (5) Authentifizierung und Sicherheit implementiert werden
