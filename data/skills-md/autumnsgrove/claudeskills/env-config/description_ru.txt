Навыки настройки среды и управления секретами с использованием UV для проектов Python. Обрабатывает файлы .env, переменные среды, шифрование секретов, настройки нескольких сред и шаблоны безопасной конфигурации. Используйте при настройке сред проекта, управлении ключами API или реализации рекомендаций по настройке.
