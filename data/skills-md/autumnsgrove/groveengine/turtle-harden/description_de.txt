Härten Sie Code mit geduldiger, mehrschichtiger Verteidigung. Die Schildkröte trägt ihren Panzer – einen knochentiefen Schutz, der von innen herausgewachsen ist und nicht angeschnallt ist. Verwenden Sie diese Funktion, wenn Sie neue Funktionen erstellen, um sicherzustellen, dass sie von Natur aus sicher sind, oder wenn Sie vorhandenen Code auf schwerwiegende Schwachstellen überprüfen, die Standardprüfungen entgehen.
