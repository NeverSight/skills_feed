Erstellen Sie mit @autumnsgrove/groveengine/vineyard-Komponenten eine Vineyard-Präsentationsseite für jedes Grove-Anwesen. Vineyard ist die konsistente /vineyard-Route, auf der jedes Tool seine Funktionen vorführt, seine API dokumentiert und seine Roadmap zeigt. Verwenden Sie diese Option, wenn Sie die Weinbergseite einer Immobilie erstellen oder aktualisieren.
