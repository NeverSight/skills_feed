Создайте страницу-витрину Vineyard для любого объекта недвижимости Grove, используя компоненты @autumnsgrove/groveengine/vineyard. Vineyard — это последовательный маршрут /vineyard, где каждый инструмент демонстрирует свои функции, документирует свой API и показывает свою дорожную карту. Используйте при создании или обновлении страницы виноградника объекта недвижимости.
