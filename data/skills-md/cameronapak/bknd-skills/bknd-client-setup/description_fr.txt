À utiliser lors de la configuration du SDK Bknd dans une application frontale. Couvre l'initialisation de la classe API, le stockage des jetons, la gestion de l'état d'authentification, l'intégration de React avec BkndBrowserApp et le hook useApp, la configuration spécifique au framework (Vite, Next.js, autonome) et l'enregistrement du type TypeScript.
