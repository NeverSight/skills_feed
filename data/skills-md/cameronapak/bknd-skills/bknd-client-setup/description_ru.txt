Используйте при настройке Bknd SDK во внешнем приложении. Охватывает инициализацию классов API, хранение токенов, обработку состояния аутентификации, интеграцию React с BkndBrowserApp и перехватчик useApp, настройку для конкретной платформы (Vite, Next.js, автономный) и регистрацию типов TypeScript.
