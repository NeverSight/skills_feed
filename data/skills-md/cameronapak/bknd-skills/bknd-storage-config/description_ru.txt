Используйте при настройке серверных хранилищ для загрузки файлов. Охватывает хранилище, совместимое с S3 (AWS S3, Cloudflare R2, DigitalOcean Spaces), хранилище мультимедиа Cloudinary, адаптер локальной файловой системы для разработки, параметры конфигурации адаптера, переменные среды и настройку производственного хранилища.
