Используйте при защите определенных конечных точек API в Bknd. Охватывает защиту пользовательских триггеров HTTP, маршрутов плагинов, промежуточного программного обеспечения аутентификации для потоков, проверку разрешений в пользовательских конечных точках и доступ к конечным точкам на основе ролей.
