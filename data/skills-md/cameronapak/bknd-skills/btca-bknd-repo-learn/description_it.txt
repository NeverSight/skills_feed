Utilizza btca (Better Context App) per eseguire query e apprendere in modo efficiente dal framework backend bknd. Utilizzare quando si lavora con bknd per (1) comprendere le definizioni del modulo dati e dello schema, (2) implementare l'autenticazione e l'autorizzazione, (3) impostare la gestione dei file multimediali, (4) configurare gli adattatori (Node, Cloudflare, ecc.), (5) apprendere dal codice sorgente e dagli esempi di bknd, (6) eseguire il debug di problemi specifici di bknd
