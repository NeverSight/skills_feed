Utilisez btca (Better Context App) pour interroger et apprendre efficacement du framework backend bknd. À utiliser lorsque vous travaillez avec bknd pour (1) Comprendre les définitions de module de données et de schéma, (2) Implémenter l'authentification et l'autorisation, (3) Configurer la gestion des fichiers multimédias, (4) Configurer les adaptateurs (Node, Cloudflare, etc.), (5) Apprendre du code source et des exemples de bknd, (6) Déboguer les problèmes spécifiques à bknd.
