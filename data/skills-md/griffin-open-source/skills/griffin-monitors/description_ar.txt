يعلم وكلاء LLM كيفية إنشاء شاشات Griffin API فعالة لنقاط النهاية في قاعدة التعليمات البرمجية. يُستخدم عندما يريد المستخدم إضافة أو تصميم أجهزة مراقبة واجهة برمجة التطبيقات (API)، أو فحوصات السلامة، أو اختبارات نقطة النهاية المجدولة؛ عند تقييم نقاط النهاية التي يجب مراقبتها؛ أو عند كتابة تعريفات الشاشة في __griffin__ باستخدام Griffin Core DSL.
