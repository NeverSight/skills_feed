Обучает агентов LLM созданию эффективных мониторов API Griffin для конечных точек в базе кода. Используйте, когда пользователь хочет добавить или разработать мониторы API, проверки работоспособности или запланированные тесты конечных точек; при оценке того, какие конечные точки следует отслеживать; или при написании определений монитора в __griffin__ с использованием Griffin Core DSL.
