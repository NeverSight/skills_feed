Enseigne aux agents LLM comment créer des moniteurs d'API Griffin efficaces pour les points de terminaison dans une base de code. À utiliser lorsque l'utilisateur souhaite ajouter ou concevoir des moniteurs d'API, des contrôles de santé ou des tests de points de terminaison planifiés ; lors de l’évaluation des paramètres à surveiller ; ou lors de l'écriture de définitions de moniteur dans __griffin__ à l'aide de Griffin Core DSL.
