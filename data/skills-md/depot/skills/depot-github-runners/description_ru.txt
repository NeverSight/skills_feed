Настраивает средства запуска GitHub Actions, управляемые Depot, в качестве замены для средств запуска, размещенных на GitHub. Используйте при настройке или переносе рабочих процессов GitHub Actions для использования бегунов Depot, выбора размеров бегунов (ЦП/ОЗУ), настройки меток запуска, настройки бегунов ARM, Windows или macOS, устранения неполадок бегунов GitHub Actions, настройки фильтрации исходящего трафика, использования Depot Cache с GitHub Actions или запуска Dagger/Dependabot на бегунах Depot. Также используйте, когда пользователь упоминает метки бегунов depot-ubuntu, depot-windows, depot-macos или спрашивает о более быстрых/дешевых бегунах GitHub Actions.
