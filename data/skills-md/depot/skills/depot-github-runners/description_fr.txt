Configure les exécuteurs GitHub Actions gérés par Depot en remplacement immédiat des exécuteurs hébergés par GitHub. À utiliser lors de la configuration ou de la migration des flux de travail GitHub Actions pour utiliser les exécuteurs Depot, du choix des tailles des exécuteurs (CPU/RAM), de la configuration des étiquettes d'exécution, de la configuration des exécuteurs ARM ou Windows ou macOS, du dépannage des problèmes d'exécution GitHub Actions, de la configuration du filtrage de sortie, de l'utilisation du cache Depot avec GitHub Actions ou de l'exécution de Dagger/Dependabot sur les exécuteurs Depot. À utiliser également lorsque l'utilisateur mentionne les étiquettes des coureurs depot-ubuntu, depot-windows, depot-macos ou pose des questions sur les coureurs GitHub Actions plus rapides/moins chers.
