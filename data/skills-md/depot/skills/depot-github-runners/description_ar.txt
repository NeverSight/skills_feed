يقوم بتكوين مشغلات GitHub Actions المُدارة بواسطة المستودع كبديل مباشر للعدائين المستضافين بواسطة GitHub. يُستخدم عند إعداد أو ترحيل مسارات عمل GitHub Actions لاستخدام مشغلات Depot، أو اختيار أحجام المشغلات (CPU/RAM)، أو تكوين تسميات التشغيل، أو إعداد مشغلات ARM أو Windows أو macOS، أو استكشاف مشكلات مشغل GitHub Actions وإصلاحها، أو تكوين تصفية الخروج، أو استخدام Depot Cache مع GitHub Actions، أو تشغيل Dagger/Dependabot على مشغلات Depot. استخدمه أيضًا عندما يذكر المستخدم تسميات depot-ubuntu أو depot-windows أو depot-macos أو يسأل عن مشغلات GitHub Actions الأسرع/الأرخص.
