Configura i runner GitHub Actions gestiti dal depot come sostituto immediato dei runner ospitati su GitHub. Da utilizzare durante la configurazione o la migrazione dei flussi di lavoro di GitHub Actions per utilizzare i runner di Depot, scegliendo le dimensioni dei runner (CPU/RAM), configurando le etichette di esecuzione, configurando i runner ARM o Windows o macOS, risolvendo i problemi dei runner di GitHub Actions, configurando il filtro in uscita, utilizzando la cache del depot con GitHub Actions o eseguendo Dagger/Dependabot sui runner di Depot. Da utilizzare anche quando l'utente menziona le etichette dei runner depot-ubuntu, depot-windows, depot-macos o chiede informazioni sui runner GitHub Actions più veloci/economici.
