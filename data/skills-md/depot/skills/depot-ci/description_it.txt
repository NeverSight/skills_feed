Configura e gestisce Depot CI, un sostituto immediato per GitHub Actions che esegue i flussi di lavoro interamente all'interno di Depot. Da utilizzare durante la migrazione dei flussi di lavoro GitHub Actions a Depot CI, l'esecuzione di "depot ci migrate", la gestione dei segreti e delle variabili di Depot CI, l'esecuzione di flussi di lavoro con "depot ci run", il debug delle esecuzioni di Depot CI, la verifica della compatibilità del flusso di lavoro o la comprensione delle attuali limitazioni beta di Depot CI. Da utilizzare anche quando l'utente menziona la directory .depot/, i comandi ci depot o chiede informazioni sull'esecuzione di flussi di lavoro di GitHub Actions sull'infrastruttura di Depot senza runner ospitati su GitHub. NOTA: Depot CI è attualmente in versione beta con disponibilità limitata.
