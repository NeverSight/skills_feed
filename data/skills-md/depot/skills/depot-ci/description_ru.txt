Настраивает и управляет Depot CI — быстрой заменой GitHub Actions, которая полностью запускает рабочие процессы внутри Depot. Используйте при переносе рабочих процессов GitHub Actions в Depot CI, запуске depot cimigration, управлении секретами и переменными Depot CI, запуске рабочих процессов с помощью depot ci run, отладке запусков Depot CI, проверке совместимости рабочих процессов или понимании текущих ограничений бета-версии Depot CI. Также используйте, когда пользователь упоминает каталог .depot/, команды depot ci или спрашивает о запуске рабочих процессов GitHub Actions в инфраструктуре Depot без средств запуска, размещенных на GitHub. ПРИМЕЧАНИЕ. Depot CI в настоящее время находится на стадии бета-тестирования с ограниченной доступностью.
