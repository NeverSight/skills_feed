Configura ed esegue build di contenitori remoti di Depot utilizzando "depot build" e "depot bake". Da utilizzare durante la creazione di immagini Docker, la creazione di Dockerfile con Depot, il push di immagini nei registri, la creazione di immagini multipiattaforma/multi-arch (linux/amd64, linux/arm64), il debug degli errori di compilazione del contenitore, l'ottimizzazione della memorizzazione nella cache dei livelli Dockerfile, l'utilizzo di build docker-bake.hcl o docker-compose o la migrazione da "docker build" / "docker buildx build" a Depot. Da utilizzare anche quando l'utente menziona build del depot, build del depot, build di contenitori, build di immagini o chiede informazioni sulla cache di build di Depot, sul parallelismo di build o sul registro temporaneo.
