`depot build` 및 `depot baking`을 사용하여 Depot 원격 컨테이너 빌드를 구성하고 실행합니다. Docker 이미지 빌드, Depot으로 Dockerfile 생성, 레지스트리에 이미지 푸시, 다중 플랫폼/다중 아키텍처 이미지(linux/amd64, linux/arm64) 빌드, 컨테이너 빌드 실패 디버깅, Dockerfile 레이어 캐싱 최적화, docker-bake.hcl 또는 docker-compose 빌드 사용 또는 `docker build` / `docker buildx build`에서 Depot으로 마이그레이션할 때 사용합니다. 또한 사용자가 디포 빌드, 디포 베이크, 컨테이너 빌드, 이미지 빌드를 언급하거나 디포의 빌드 캐시, 빌드 병렬 처리 또는 임시 레지스트리에 대해 묻는 경우에도 사용합니다.
