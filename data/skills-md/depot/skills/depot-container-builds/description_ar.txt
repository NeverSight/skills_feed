تكوين وتشغيل إنشاءات حاوية Depot البعيدة باستخدام "depot build" و"depot Bake". يُستخدم عند إنشاء صور Docker، أو إنشاء Dockerfiles باستخدام Depot، أو دفع الصور إلى السجلات، أو إنشاء صور متعددة المنصات/متعددة القوس (linux/amd64، linux/arm64)، أو تصحيح أخطاء فشل إنشاء الحاوية، أو تحسين التخزين المؤقت لطبقة Dockerfile، أو استخدام docker-bake.hcl أو تصميمات docker-compose، أو الترحيل من `docker build` / `docker buildx build` إلى Depot. يُستخدم أيضًا عندما يذكر المستخدم إنشاء المستودع، أو خبز المستودع، أو إنشاءات الحاويات، أو إنشاءات الصور، أو يسأل عن ذاكرة التخزين المؤقت لبناء المستودع، أو توازي البناء، أو السجل المؤقت.
