Gestisce l'installazione della CLI di Depot, l'autenticazione, l'accesso, la configurazione del progetto, la gestione dell'organizzazione e l'accesso all'API. Da utilizzare durante l'installazione della CLI di Depot, l'accesso con `depot login`, la creazione o la gestione di progetti di Depot, la configurazione di token API o relazioni di trust OIDC, l'impostazione di depot.json, la gestione di organizzazioni, la reimpostazione delle cache di build o l'utilizzo di API/SDK di Depot. Da utilizzare anche quando l'utente chiede informazioni sui metodi di autenticazione del depot, sui tipi di token, sulle variabili di ambiente o sulla configurazione generale della piattaforma del depot che non è specifica per le build del contenitore, i runner di GitHub Actions o l'elemento della configurazione del depot.
