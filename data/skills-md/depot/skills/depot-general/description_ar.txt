يتعامل مع تثبيت Depot CLI والمصادقة وتسجيل الدخول وإعداد المشروع وإدارة المؤسسة والوصول إلى واجهة برمجة التطبيقات. يُستخدم عند تثبيت Depot CLI، أو تسجيل الدخول باستخدام "تسجيل الدخول إلى المستودع"، أو إنشاء مشاريع Depot أو إدارتها، أو تكوين الرموز المميزة لواجهة برمجة التطبيقات أو علاقات ثقة OIDC، أو إعداد depot.json، أو إدارة المؤسسات، أو إعادة تعيين ذاكرة التخزين المؤقت للإنشاء، أو استخدام Depot API/SDKs. يُستخدم أيضًا عندما يسأل المستخدم عن طرق مصادقة Depot، أو أنواع الرموز المميزة، أو متغيرات البيئة، أو إعداد نظام Depot الأساسي العام الذي لا يقتصر على إنشاءات الحاويات، أو مشغلات GitHub Actions، أو Depot CI.
