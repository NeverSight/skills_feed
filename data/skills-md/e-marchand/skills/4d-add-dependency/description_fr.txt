Ajoutez des dépendances à un projet 4D. À utiliser lorsque l'utilisateur souhaite ajouter un composant, une bibliothèque ou une dépendance à son projet 4D. Prend en charge les dépôts GitHub (format propriétaire/dépôt), les URL GitHub (avec extraction automatique des balises à partir des URL de version) et les chemins de dossiers locaux. Gère la configuration des dépendances.json et de l'environnement4d.json.
