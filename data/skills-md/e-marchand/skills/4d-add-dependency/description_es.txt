Agregar dependencias a un proyecto 4D. Utilícelo cuando el usuario quiera agregar un componente, biblioteca o dependencia a su proyecto 4D. Admite repositorios de GitHub (formato propietario/repositorio), URL de GitHub (con extracción automática de etiquetas de las URL de lanzamiento) y rutas de carpetas locales. Maneja la configuración de dependencies.json y Environment4d.json.
