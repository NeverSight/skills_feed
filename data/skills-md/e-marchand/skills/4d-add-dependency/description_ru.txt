Добавьте зависимости в 4D-проект. Используйте, когда пользователь хочет добавить компонент, библиотеку или зависимость в свой 4D-проект. Поддерживает репозитории GitHub (формат владельца/репозитория), URL-адреса GitHub (с автоматическим извлечением тегов из URL-адресов выпуска) и пути к локальным папкам. Обрабатывает конфигурацию Dependency.json и Environment4d.json.
