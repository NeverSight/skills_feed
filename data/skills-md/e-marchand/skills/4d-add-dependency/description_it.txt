Aggiungi dipendenze a un progetto 4D. Utilizzare quando l'utente desidera aggiungere un componente, una libreria o una dipendenza al proprio progetto 4D. Supporta repository GitHub (formato proprietario/repo), URL GitHub (con estrazione automatica dei tag dagli URL di rilascio) e percorsi di cartelle locali. Gestisce la configurazione di dependencies.json e Environment4d.json.
