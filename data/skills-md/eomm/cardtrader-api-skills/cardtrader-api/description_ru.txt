Взаимодействуйте с CardTrader REST API для управления торговыми картами. Осуществляет аутентификацию, операции на рынке (поиск продуктов, корзина, покупка), управление запасами (список/создание/обновление/удаление продуктов), управление заказами (список/обновление/отгрузка заказов) и операции со списком желаний. Используйте для интеграции рынка торговых карточек, синхронизации запасов, автоматизации выполнения заказов.
