Interagisci con l'API REST di CardTrader per la gestione delle carte collezionabili. Gestisce l'autenticazione, le operazioni del marketplace (ricerca di prodotti, carrello, acquisto), la gestione dell'inventario (elencare/creare/aggiornare/eliminare prodotti), la gestione degli ordini (elencare/aggiornare/spedire ordini) e le operazioni della lista dei desideri. Utilizzare per l'integrazione del mercato delle carte collezionabili, la sincronizzazione dell'inventario e l'automazione dell'evasione degli ordini.
