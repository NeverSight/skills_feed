التفاعل مع CardTrader REST API لإدارة بطاقات التداول. يتعامل مع المصادقة وعمليات السوق (البحث عن المنتجات، سلة التسوق، الشراء)، وإدارة المخزون (قائمة/إنشاء/تحديث/حذف المنتجات)، وإدارة الطلبات (قائمة/تحديث/طلبات الشحن)، وعمليات قائمة الرغبات. يُستخدم لتكامل سوق بطاقات التداول، ومزامنة المخزون، وأتمتة تنفيذ الطلبات.
