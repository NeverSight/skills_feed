构建、开发和部署 Power Apps Code Apps — 在 Power Platform 内运行的代码优先 Web 应用程序。在处理 Power Apps Code Apps 项目时使用此技能：搭建新应用程序、编写使用 @microsoft/power-apps SDK 的 React 组件、添加数据源和连接器（Dataverse、SQL、SharePoint、Office 365）、使用生成的 CRUD 服务和类型化模型、运行 pac CLI 命令（pac code init、pac code push、pac code add-data-source）、使用 power-apps-vite 插件配置 Vite 构建或部署到 Power Platform 环境。触发：power.config.json 文件、@microsoft/power-apps 导入、pac 代码命令、Power Apps Code Apps 项目。
