Erstellen, entwickeln und implementieren Sie Power Apps Code Apps – Code-First-Webanwendungen, die innerhalb der Power Platform ausgeführt werden. Verwenden Sie diese Fähigkeit, wenn Sie an Power Apps Code Apps-Projekten arbeiten: Gerüstbau für neue Apps, Schreiben von React-Komponenten, die das @microsoft/power-apps SDK verwenden, Hinzufügen von Datenquellen und Konnektoren (Dataverse, SQL, SharePoint, Office 365), Verwenden generierter CRUD-Dienste und typisierter Modelle, Ausführen von pac-CLI-Befehlen (pac code init, pac code push, pac code add-data-source), Konfigurieren von Vite-Builds mit dem Power-Apps-Vite-Plugin oder Bereitstellen auf Power Platform Umgebungen. Löst aus: power.config.json-Dateien, @microsoft/power-apps-Importe, Pac-Code-Befehle, Power Apps Code Apps-Projekte.
