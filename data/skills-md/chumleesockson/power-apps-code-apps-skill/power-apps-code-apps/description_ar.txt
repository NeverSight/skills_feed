قم ببناء وتطوير ونشر Power Apps Code Apps — تطبيقات الويب التي تعتمد التعليمات البرمجية أولاً والتي يتم تشغيلها داخل Power Platform. استخدم هذه المهارة عند العمل على مشاريع Power Apps Code Apps: إنشاء تطبيقات جديدة، وكتابة مكونات React التي تستخدم @microsoft/power-apps SDK، وإضافة مصادر البيانات والموصلات (Dataverse وSQL وSharePoint وOffice 365)، باستخدام خدمات CRUD التي تم إنشاؤها والنماذج المكتوبة، وتشغيل أوامر pac CLI (pac code init، pac code Push، pac code add-data-source)، وتكوين إصدارات Vite باستخدام power-apps-vite. البرنامج المساعد، أو النشر إلى بيئات Power Platform. المشغلات على: ملفات power.config.json، وعمليات استيراد @microsoft/power-apps، وأوامر pac code، ومشروعات Power Apps Code Apps.
