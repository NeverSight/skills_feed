Crea, sviluppa e distribuisci Power Apps Code Apps: applicazioni Web code-first eseguite all'interno di Power Platform. Utilizzare questa competenza quando si lavora su progetti Power Apps Code Apps: impalcatura di nuove app, scrittura di componenti React che utilizzano l'SDK @microsoft/power-apps, aggiunta di origini dati e connettori (Dataverse, SQL, SharePoint, Office 365), utilizzo di servizi CRUD generati e modelli tipizzati, esecuzione di comandi CLI pac (pac code init, pac code push, pac code add-data-source), configurazione di build Vite con il plug-in power-apps-vite o distribuzione su Power Platform ambienti. Trigger su: file power.config.json, importazioni @microsoft/power-apps, comandi codice pac, progetti Power Apps Code Apps.
