Power Apps コード アプリ (Power Platform 内で実行されるコードファースト Web アプリケーション) を構築、開発、デプロイします。 Power Apps Code Apps プロジェクトで作業する場合は、このスキルを使用します。新しいアプリのスキャフォールディング、@microsoft/power-apps SDK を使用する React コンポーネントの作成、データ ソースとコネクタ (Dataverse、SQL、SharePoint、Office 365) の追加、生成された CRUD サービスと型付きモデルの使用、pac CLI コマンド (pac code init、pac code Push、pac code add-data-source) の実行、power-apps-vite プラグインを使用した Vite ビルドの構成、またはへのデプロイPower Platform 環境。トリガー対象: power.config.json ファイル、@microsoft/power-apps インポート、pac コード コマンド、Power Apps Code Apps プロジェクト。
