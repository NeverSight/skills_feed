Создавайте, разрабатывайте и развертывайте приложения Power Apps Code — веб-приложения, ориентированные на код, которые работают внутри Power Platform. Используйте этот навык при работе над проектами Power Apps Code Apps: создание новых приложений, написание компонентов React, использующих SDK @microsoft/power-apps, добавление источников данных и соединителей (Dataverse, SQL, SharePoint, Office 365), использование сгенерированных служб CRUD и типизированных моделей, выполнение команд pac CLI (pac code init, pac code push, pac code add-data-source), настройка сборок Vite с помощью подключаемого модуля power-apps-vite или развертывание. в среду Power Platform. Запускается по: файлам power.config.json, импорту @microsoft/power-apps, командам pac-кода, проектам Power Apps Code Apps.
