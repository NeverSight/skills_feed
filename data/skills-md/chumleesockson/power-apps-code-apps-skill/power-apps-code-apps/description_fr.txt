Créez, développez et déployez des applications de code Power Apps : des applications Web axées sur le code qui s'exécutent au sein de Power Platform. Utilisez cette compétence lorsque vous travaillez sur des projets Power Apps Code Apps : échafaudage de nouvelles applications, écriture de composants React qui utilisent le SDK @microsoft/power-apps, ajout de sources de données et de connecteurs (Dataverse, SQL, SharePoint, Office 365), utilisation de services CRUD générés et de modèles typés, exécution de commandes CLI pac (pac code init, pac code push, pac code add-data-source), configuration des builds Vite avec le plug-in power-apps-vite ou déploiement sur Environnements Power Platform. Déclencheurs sur : fichiers power.config.json, importations @microsoft/power-apps, commandes de code pac, projets Power Apps Code Apps.
