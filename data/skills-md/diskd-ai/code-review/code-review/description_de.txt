Überprüfen Sie Code mit einem strukturierten, signalstarken Überprüfungsprozess. Verwenden Sie diese Funktion, wenn Sie aufgefordert werden, Code zu überprüfen, einen PR/MR zu überprüfen (GitHub-PRs über „gh“, GitLab-MRs über „glab“), eine Codebasis zu überprüfen, Feedback zu einer Änderungsliste zu geben, die Codequalität zu prüfen oder eine Codeüberprüfung durchzuführen. Unterstützt sowohl Diff/PR-Reviews als auch vollständige Codebase-Reviews. Erstellt umsetzbare Ergebnisse mit Schweregradbezeichnungen, genauen Datei- und Zeilenzitaten und einem endgültigen Urteil. Behandelt Design, Funktionalität, Komplexität, Tests, Benennung, Kommentare, Stil, Dokumentation, Sicherheit und Leistung.
