Esamina il codice con un processo di revisione strutturato e ad alto segnale. Da utilizzare quando viene richiesto di rivedere il codice, rivedere un PR/MR (PR di GitHub tramite "gh", MR di GitLab tramite "glab"), rivedere una base di codice, fornire feedback su un elenco di modifiche, verificare la qualità del codice o eseguire una revisione del codice. Supporta sia revisioni diff/PR che revisioni complete della base di codice. Produce risultati utilizzabili con etichette di gravità, citazioni esatte di file+riga e un verdetto finale. Copre progettazione, funzionalità, complessità, test, denominazione, commenti, stile, documentazione, sicurezza e prestazioni.
