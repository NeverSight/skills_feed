構造化されたシグナルの高いレビュー プロセスでコードをレビューします。コードのレビュー、PR/MR のレビュー (「gh」経由の GitHub PR、「glab」経由の GitLab MR)、コードベースのレビュー、変更リストに関するフィードバックの提供、コード品質の監査、またはコード レビューの実行を求められた場合に使用します。 diff/PR レビューと完全なコードベース レビューの両方をサポートします。重大度ラベル、正確なファイルと行の引用、および最終的な判定を含む実用的な結果を生成します。デザイン、機能、複雑さ、テスト、命名、コメント、スタイル、ドキュメント、セキュリティ、パフォーマンスをカバーします。
