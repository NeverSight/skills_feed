قم بمراجعة التعليمات البرمجية من خلال عملية مراجعة منظمة وعالية الإشارة. استخدمه عند مطالبتك بمراجعة التعليمات البرمجية، أو مراجعة PR/MR (GitHub PRs عبر `gh`، أو GitLab MRs عبر `glab`)، أو مراجعة قاعدة التعليمات البرمجية، أو تقديم تعليقات على قائمة التغيير، أو تدقيق جودة التعليمات البرمجية، أو إجراء مراجعة للتعليمات البرمجية. يدعم كلاً من مراجعات الفرق/العلاقات العامة ومراجعات قاعدة التعليمات البرمجية الكاملة. يُنتج نتائج قابلة للتنفيذ مع تصنيفات الخطورة، والاستشهادات الدقيقة للملف + الأسطر، والحكم النهائي. يغطي التصميم والوظائف والتعقيد والاختبارات والتسمية والتعليقات والأسلوب والوثائق والأمان والأداء.
