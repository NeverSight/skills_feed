Просматривайте код с помощью структурированного процесса проверки с высоким уровнем значимости. Используйте, когда вас просят просмотреть код, просмотреть PR/MR (запросы GitHub через `gh`, GitLab MR через `glab`), просмотреть базу кода, оставить отзыв о списке изменений, проверить качество кода или выполнить проверку кода. Поддерживает как обзоры различий/PR, так и полные обзоры кодовой базы. Предоставляет действенные выводы с метками серьезности, точными ссылками на файлы и строки и окончательным вердиктом. Охватывает дизайн, функциональность, сложность, тесты, именование, комментарии, стиль, документацию, безопасность и производительность.
