Révisez le code avec un processus de révision structuré et à signal élevé. À utiliser lorsqu'on vous demande de réviser le code, de réviser un PR/MR (PR GitHub via `gh`, GitLab MR via `glab`), de réviser une base de code, de donner votre avis sur une liste de modifications, d'auditer la qualité du code ou d'effectuer une révision de code. Prend en charge à la fois les révisions diff/PR et les révisions complètes de la base de code. Produit des résultats exploitables avec des étiquettes de gravité, des citations exactes de fichiers et de lignes et un verdict final. Couvre la conception, la fonctionnalité, la complexité, les tests, la dénomination, les commentaires, le style, la documentation, la sécurité et les performances.
