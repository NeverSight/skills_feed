Создавайте технические проектные документы для новых функций, систем, планов рефакторинга, проектов API и схем баз данных. Используйте, когда вас просят написать проектную документацию, TRD (документ с техническими требованиями), RFC или документ о проектировании системы. Запускается по таким запросам, как «написать проектную документацию для X», «спроектировать функцию Y», «создать техническую спецификацию» или «спланировать архитектуру для Z».
