قم بإنشاء مستندات التصميم الفني للميزات والأنظمة وخطط إعادة البناء الجديدة وتصميمات واجهة برمجة التطبيقات ومخططات قاعدة البيانات. يُستخدم عندما يُطلب منك كتابة مستند تصميم، أو TRD (مستند المتطلبات الفنية)، أو RFC، أو مستند تصميم النظام. مشغلات لطلبات مثل "كتابة مستند تصميم لـ X"، أو "تصميم ميزة Y"، أو "إنشاء مواصفات فنية"، أو "تخطيط البنية لـ Z".
