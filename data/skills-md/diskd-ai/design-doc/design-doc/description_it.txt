Crea documenti di progettazione tecnica per nuove funzionalità, sistemi, piani di refactoring, progettazioni API e schemi di database. Da utilizzare quando viene richiesto di scrivere un documento di progettazione, un TRD (documento sui requisiti tecnici), una RFC o un documento di progettazione del sistema. Si attiva su richieste come "scrivere un documento di progettazione per X", "progettare la funzionalità Y", "creare una specifica tecnica" o "pianificare l'architettura per Z".
