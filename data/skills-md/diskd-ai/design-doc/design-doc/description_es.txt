Cree documentos de diseño técnico para nuevas funciones, sistemas, planes de refactorización, diseños de API y esquemas de bases de datos. Úselo cuando se le solicite escribir un documento de diseño, TRD (documento de requisitos técnicos), RFC o documento de diseño de sistema. Se activan solicitudes como "escribir un documento de diseño para X", "diseñar la función Y", "crear una especificación técnica" o "planificar la arquitectura para Z".
