تكامل OpenAI API لبناء التطبيقات التي تعمل بالذكاء الاصطناعي. يُستخدم عند العمل مع واجهة برمجة تطبيقات Chat Completions API الخاصة بـ OpenAI، وPython SDK (openai)، وTypeScript SDK (openai)، واستخدام الأداة/الاستدعاء الوظيفي، ومدخلات الرؤية/الصورة، واستجابات البث، وإنشاء صور DALL-E، ونسخ الصوت Whisper، وتحويل النص إلى كلام، والتضمين، وواجهة برمجة تطبيقات المساعدين، والضبط الدقيق، أو أي مهمة تكامل OpenAI API. يتم تشغيله عند الإشارة إلى استخدام OpenAI أو GPT-4 أو GPT-4o أو GPT-5 أو o1 أو o3 أو o4 أو DALL-E أو Whisper أو Sora أو OpenAI SDK.
