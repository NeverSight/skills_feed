تكامل AI API معًا لبناء تطبيقات مدعومة بالذكاء الاصطناعي باستخدام نماذج مفتوحة المصدر. يُستخدم عند العمل مع Together's Chat Completions API، وPython SDK (معًا)، وTypeScript SDK (together-ai)، وأداة CLI، واستخدام الأداة/استدعاء الوظيفة، وفهم الرؤية/الصورة، وإنشاء الصور (FLUX، وStable Diffusion)، وإنشاء الفيديو (Veo، وSora، وKling)، والنسخ الصوتي (Whisper)، وتحويل النص إلى كلام، والاستجابات المتدفقة، والتضمين، وإعادة الترتيب، والضبط الدقيق، أو أي مهمة تكامل لواجهة برمجة تطبيقات AI معًا. يتم تشغيل الإشارات عند ذكر Together AI، أو Together API، أو GroqCloud، أو استدلال LLM مفتوح المصدر، أو إنشاء صور FLUX، أو نسخ Whisper عبر Together.
