超高速 LLM 推論を使用して AI を活用したアプリケーションを構築するための Groq API 統合。 Groq の Chat Completions API、Python SDK (groq)、TypeScript SDK (groq-sdk)、ツールの使用/関数呼び出し、ビジョン/画像処理、Whisper による音声転写、ストリーミング応答、テキスト読み上げ、Llama Guard によるコンテンツ モデレーション、バッチ処理、または Groq API 統合タスクを操作するときに使用します。 Groq、GroqCloud、または高速 LLM 推論のニーズに言及するとトリガーされます。
