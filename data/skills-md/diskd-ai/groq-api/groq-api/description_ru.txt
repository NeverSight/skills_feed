Интеграция Groq API для создания приложений на базе искусственного интеллекта со сверхбыстрым выводом LLM. Используйте при работе с Groq Chat Completions API, Python SDK (groq), TypeScript SDK (groq-sdk), использовании инструментов/вызове функций, обработке изображений/зрений, транскрипции звука с помощью Whisper, потоковой передаче ответов, преобразовании текста в речь, модерации контента с помощью Llama Guard, пакетной обработке или любой задаче интеграции API Groq. Запускается при упоминании Groq, GroqCloud или необходимости быстрого вывода LLM.
