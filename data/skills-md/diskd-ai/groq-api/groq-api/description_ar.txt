تكامل Groq API لبناء تطبيقات مدعومة بالذكاء الاصطناعي مع استدلال LLM فائق السرعة. يُستخدم عند العمل مع Groq's Chat Completions API، وPython SDK (groq)، وTypeScript SDK (groq-sdk)، واستخدام الأداة/استدعاء الوظيفة، ومعالجة الرؤية/الصورة، والنسخ الصوتي باستخدام Whisper، والاستجابات المتدفقة، وتحويل النص إلى كلام، والإشراف على المحتوى باستخدام Llama Guard، أو معالجة الدُفعات، أو أي مهمة تكامل Groq API. يتم تشغيل الإشارات عند ذكر احتياجات Groq أو GroqCloud أو الاستدلال السريع LLM.
