「red-cli」スタイルのインターフェースを使用して、Redmine REST API を介してコマンドラインから Redmine の問題/チケットを管理します。認証 (サーバー + API キー) に `~/.red/config.json` を使用し、`./.red/config.json` でオプションのローカル オーバーライドを行います。問題/プロジェクトの一覧表示、問題 (ジャーナル付き) の表示、ジャーナルノート/コメント (公開または非公開) の追加、問題の説明の更新、ジャーナル ID による既存のコメント (ジャーナル) の更新/削除、リストのフィルター/並べ替え/ページ付け、または Redmine CLI 設定 (プロジェクト ID/ユーザー ID、マルチインスタンス `--rid`) のトラブルシューティングが必要な場合に使用します。 Redmine、red-cli、issue list/view/note/edit/comment、`--rid`、`.red/config.json`、`api-key`、`project-id`、`user-id`、または CLI から Redmine を自動化するリクエストでトリガーします。
