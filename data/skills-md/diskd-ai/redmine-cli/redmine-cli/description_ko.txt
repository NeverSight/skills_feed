'red-cli' 스타일 인터페이스를 사용하여 Redmine REST API를 통해 명령줄에서 Redmine 문제/티켓을 관리하세요. `./.red/config.json`에서 선택적 로컬 재정의와 함께 인증(서버 + API 키)에 `~/.red/config.json`을 사용합니다. 이슈/프로젝트 나열, 이슈 보기(저널 포함), 저널 메모/댓글 추가(공개 또는 비공개), 이슈 설명 업데이트, 저널 ID별로 기존 댓글(저널) 업데이트/제거, 목록 필터링/정렬/페이지 매기기 또는 Redmine CLI 구성 문제 해결(프로젝트 ID/사용자 ID, 다중 인스턴스 `--rid`)이 필요할 때 사용합니다. Redmine, red-cli, 문제 목록/보기/노트/편집/설명, `--rid`, `.red/config.json`, `api-key`, `project-id`, `user-id` 또는 CLI에서 Redmine 자동화 요청에 대해 트리거됩니다.
