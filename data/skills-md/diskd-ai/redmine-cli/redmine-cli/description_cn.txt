使用“red-cli”风格的界面，通过 Redmine REST API 从命令行管理 Redmine 问题/票证。使用 `~/.red/config.json` 进行身份验证（服务器 + api-key），并在 `./.red/config.json` 中可选本地覆盖。当您需要列出问题/项目、查看问题（包含期刊）、添加期刊注释/评论（公共或私人）、更新问题描述、按期刊 ID 更新/删除现有评论（期刊）、过滤/排序/分页列表或对 Redmine CLI 配置（项目 ID/用户 ID、多实例 `--rid`）进行故障排除时使用。在 Redmine、red-cli、问题列表/视图/注释/编辑/评论、`--rid`、`.red/config.json`、`api-key`、`project-id`、`user-id` 上触发，或从 CLI 自动化 Redmine 的请求。
