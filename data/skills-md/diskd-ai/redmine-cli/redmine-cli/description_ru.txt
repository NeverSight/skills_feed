Управляйте проблемами/заявками Redmine из командной строки через Redmine REST API, используя интерфейс в стиле red-cli. Использует `~/.red/config.json` для аутентификации (сервер + API-ключ) с дополнительными локальными переопределениями в `./.red/config.json`. Используйте, когда вам нужно составить список задач/проектов, просмотреть проблемы (с журналами), добавить заметки/комментарии к журналу (публичные или частные), обновить описание проблемы, обновить/удалить существующие комментарии (журналы) по идентификатору журнала, отфильтровать/сортировать/разбить на страницы списки или устранить неполадки конфигурации Redmine CLI (идентификатор проекта/идентификатор пользователя, многоэкземплярный `--rid`). Запускается по Redmine, red-cli, списку проблем/просмотру/заметке/редактированию/комментарию, `--rid`, `.red/config.json`, `api-key`, `project-id`, `user-id` или запросам на автоматизацию Redmine из CLI.
