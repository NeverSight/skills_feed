قم بإدارة مشكلات/تذاكر Redmine من سطر الأوامر عبر Redmine REST API باستخدام واجهة بنمط `red-cli`. يستخدم `~/.red/config.json` للمصادقة (الخادم + مفتاح واجهة برمجة التطبيقات) مع التجاوزات المحلية الاختيارية في `./.red/config.json`. يُستخدم عندما تحتاج إلى سرد المشكلات/المشاريع، أو عرض المشكلات (مع المجلات)، أو إضافة ملاحظات/تعليقات دفتر اليومية (عامة أو خاصة)، أو تحديث وصف المشكلة، أو تحديث/إزالة التعليقات الموجودة (المجلات) حسب معرف المجلة، أو تصفية/فرز/صفحات الصفحات، أو استكشاف أخطاء تكوين Redmine CLI وإصلاحها (معرف المشروع/معرف المستخدم، المثيل المتعدد `--rid`). المشغلات على Redmine، أو red-cli، أو قائمة المشكلات/العرض/الملاحظة/التحرير/التعليق، أو `--rid`، أو `.red/config.json`، أو `api-key`، أو `project-id`، أو `user-id`، أو طلبات أتمتة Redmine من واجهة سطر الأوامر.
