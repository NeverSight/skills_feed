使用“red-cli”風格的界面，通過 Redmine REST API 從命令行管理 Redmine 問題/票證。使用 `~/.red/config.json` 進行身份驗證（服務器 + api-key），並在 `./.red/config.json` 中可選本地覆蓋。當您需要列出問題/項目、查看問題（包含期刊）、添加期刊註釋/評論（公共或私人）、更新問題描述、按期刊 ID 更新/刪除現有評論（期刊）、過濾/排序/分頁列表或對 Redmine CLI 配置（項目 ID/用戶 ID、多實例 `--rid`）進行故障排除時使用。在 Redmine、red-cli、問題列表/視圖/註釋/編輯/評論、`--rid`、`.red/config.json`、`api-key`、`project-id`、`user-id` 上觸發，或從 CLI 自動化 Redmine 的請求。
