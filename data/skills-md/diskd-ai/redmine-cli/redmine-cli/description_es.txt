Administre problemas/tickets de Redmine desde la línea de comando a través de la API REST de Redmine usando una interfaz estilo `red-cli`. Utiliza `~/.red/config.json` para la autenticación (servidor + clave de API) con anulaciones locales opcionales en `./.red/config.json`. Úselo cuando necesite enumerar problemas/proyectos, ver problemas (con revistas), agregar notas/comentarios de diarios (públicos o privados), actualizar la descripción del problema, actualizar/eliminar comentarios existentes (diarios) por ID de revista, filtrar/ordenar/paginar listas o solucionar problemas de configuración de CLI de Redmine (ID de proyecto/ID de usuario, instancia múltiple `--rid`). Activadores en Redmine, red-cli, lista de problemas/ver/nota/editar/comentario, `--rid`, `.red/config.json`, `api-key`, `project-id`, `user-id` o solicitudes para automatizar Redmine desde una CLI.
