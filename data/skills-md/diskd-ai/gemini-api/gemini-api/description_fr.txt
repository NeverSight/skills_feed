Intégration de l'API Google Gemini pour créer des applications basées sur l'IA. À utiliser lorsque vous travaillez avec l'API Gemini de Google, le SDK Python (google-genai), le SDK TypeScript (@google/genai), les entrées multimodales (image, vidéo, audio, PDF), les fonctionnalités de réflexion/raisonnement, les réponses en continu, les sorties structurées avec les schémas JSON, le chat multi-tours, les instructions système, la génération d'images (Nano Banana), la génération de vidéos (Veo), la génération de musique (Lyria), les intégrations, le traitement de documents/PDF ou toute tâche d'intégration de l'API Gemini. Se déclenche lors des mentions de l'utilisation du SDK Gemini, Gemini 3, Gemini 2.5, Google AI, Nano Banana, Veo, Lyria, google-genai ou @google/genai.
