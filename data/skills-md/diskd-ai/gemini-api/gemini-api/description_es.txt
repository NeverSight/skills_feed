Integración de la API de Google Gemini para crear aplicaciones impulsadas por IA. Úselo cuando trabaje con la API Gemini de Google, el SDK de Python (google-genai), el SDK de TypeScript (@google/genai), entradas multimodales (imagen, video, audio, PDF), funciones de pensamiento/razonamiento, transmisión de respuestas, salidas estructuradas con esquemas JSON, chat de varios turnos, instrucciones del sistema, generación de imágenes (Nano Banana), generación de videos (Veo), generación de música (Lyria), incrustaciones, procesamiento de documentos/PDF o cualquier tarea de integración de la API Gemini. Se activa cuando se menciona el uso de Gemini, Gemini 3, Gemini 2.5, Google AI, Nano Banana, Veo, Lyria, google-genai o @google/genai SDK.
