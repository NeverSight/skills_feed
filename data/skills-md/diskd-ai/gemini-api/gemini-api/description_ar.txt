تكامل Google Gemini API لإنشاء تطبيقات تعمل بالذكاء الاصطناعي. يُستخدم عند العمل مع Gemini API من Google، وPython SDK (google-genai)، وTypeScript SDK (@google/genai)، والمدخلات متعددة الوسائط (الصورة والفيديو والصوت وPDF)، وميزات التفكير/الاستدلال، والاستجابات المتدفقة، والمخرجات المنظمة باستخدام مخططات JSON، والدردشة متعددة المنعطفات، وتعليمات النظام، وتوليد الصور (Nano Banana)، وتوليد الفيديو (Veo)، وتوليد الموسيقى (Lyria)، والتضمين، ومعالجة المستندات/PDF، أو أي مهمة تكامل Gemini API. يتم تشغيله عند الإشارة إلى استخدام Gemini أو Gemini 3 أو Gemini 2.5 أو Google AI أو Nano Banana أو Veo أو Lyria أو google-genai أو @google/genai SDK.
