Интеграция Google Gemini API для создания приложений на базе искусственного интеллекта. Используйте при работе с Google Gemini API, Python SDK (google-genai), TypeScript SDK (@google/genai), мультимодальными входными данными (изображения, видео, аудио, PDF), функциями мышления/рассуждения, потоковыми ответами, структурированными выводами со схемами JSON, многооборотным чатом, системными инструкциями, генерацией изображений (Nano Banana), генерацией видео (Veo), генерацией музыки (Lyria), встраиванием, обработкой документов/PDF или любой задачей интеграции Gemini API. Запускается при упоминании об использовании Gemini, Gemini 3, Gemini 2.5, Google AI, Nano Banana, Veo, Lyria, google-genai или @google/genai SDK.
