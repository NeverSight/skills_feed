Google Gemini API-Integration zum Erstellen KI-gestützter Anwendungen. Verwendung bei der Arbeit mit Googles Gemini API, Python SDK (google-genai), TypeScript SDK (@google/genai), multimodalen Eingaben (Bild, Video, Audio, PDF), Denk-/Argumentationsfunktionen, Streaming-Antworten, strukturierten Ausgaben mit JSON-Schemas, Multi-Turn-Chat, Systemanweisungen, Bildgenerierung (Nano Banana), Videogenerierung (Veo), Musikgenerierung (Lyria), Einbettungen, Dokument-/PDF-Verarbeitung oder einer beliebigen Gemini-API-Integrationsaufgabe. Wird bei Erwähnungen der SDK-Nutzung von Gemini, Gemini 3, Gemini 2.5, Google AI, Nano Banana, Veo, Lyria, google-genai oder @google/genai ausgelöst.
