Da utilizzare quando si lavora con l'API Gemini di Google, Python SDK (google-genai), TypeScript SDK (@google/genai), input multimodali (immagine, video, audio, PDF), funzionalità di pensiero/ragionamento, risposte in streaming, output strutturati con schemi JSON, chat multi-turno, istruzioni di sistema, generazione di immagini (Nano Banana), generazione di video (Veo), generazione di musica (Lyria), incorporamenti, elaborazione di documenti/PDF o qualsiasi attività di integrazione dell'API Gemini. Si attiva quando vengono menzionate le menzioni di Gemini, Gemini 3, Gemini 2.5, Google AI, Nano Banana, Veo, Lyria, google-genai o l'utilizzo dell'SDK @google/genai.
