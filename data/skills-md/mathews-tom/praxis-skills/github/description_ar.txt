عمليات GitHub CLI عبر `gh` للمشكلات، وطلبات السحب، وCI/Actions، والإصدارات، وRepos، والبحث، وGists، وREST/GraphQL API. مخرجات منظمة باستخدام `--json` و`--jq` للتحليل. يغطي "إنشاء مشكلة gh/قائمة/عرض/تحرير/إغلاق"، و"gh pr create/review/merge/checks"، و"gh run list/view/rerun/watch"، و"gh Release create"، و"gh search repos/issues/prs/code"، و"gh api" لاستعلامات REST وGraphQL، وعمليات "gh gist". يتضمن معالجة الأخطاء لـ HTTP 401/403/404/422/429، واستكشاف أخطاء النطاق وإصلاحها، وإدارة حدود المعدل. عبارات التشغيل: "إنشاء مشكلة"، "تقديم خطأ"، "فتح تذكرة"، "إرسال علاقات عامة"، "رفع طلب سحب"، "التحقق من خط الأنابيب"، "عرض نتائج الاختبار"، "فشل CI"، "لماذا فشل CI"، "التحقق من حالة CI"، "دمج العلاقات العامة"، "إدارة الإصدارات"، "الاستعلام عن واجهة برمجة تطبيقات GitHub"، "مستودعات البحث"، "سير عمل الفرز"، "أتمتة عمليات GitHub". يتم تشغيله أيضًا عندما يقوم المستخدم بلصق عنوان URL لـ GitHub.
