Преобразуйте серверы MCP (Model Context Protocol) в навыки по требованию, чтобы значительно сократить использование активного контекстного окна. Схемы инструментов MCP потребляют токены на каждом ходу, независимо от того, используются они или нет; навыки загружаются только при необходимости. Анализирует определения инструментов MCP, классифицирует каждый инструмент по стратегии замены и генерирует полный набор навыков. Используйте этот навык всякий раз, когда пользователь упоминает: преобразовать MCP, MCP в навык, уменьшить размер контекста, слишком много инструментов, раздуть токен инструмента, уменьшить MCP, окно контекста заполнено, заменить MCP, миграцию MCP. Также срабатывает, когда у пользователя много активных серверов MCP и он хочет оптимизировать использование контекста, спрашивает об уменьшении накладных расходов на токены из определений инструментов или говорит, что мой контекст слишком велик или не хватает контекста, когда MCP активны.
