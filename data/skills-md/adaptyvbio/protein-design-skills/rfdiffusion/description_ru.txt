Создавайте белковые основы с помощью RFdiffusion, генеративной модели на основе диффузии для генерации белковых структур de novo. Используйте этот навык, когда: (1) проектируете связующие каркасы для целевого белка, (2) создаете новые белковые основы с нуля, (3) создаете каркасы функциональных мотивов в новых белках, (4) определяете остатки горячих точек для проектирования интерфейса, (5) создаете симметричные олигомеры.
Для проектирования последовательности после генерации основной цепи используйте протеинmpnn. Для проверки структуры используйте Alphafold или Chai. Для порогов контроля качества используйте белок-QC.
