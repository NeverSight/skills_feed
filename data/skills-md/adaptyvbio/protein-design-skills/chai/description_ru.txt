Прогнозирование структуры с использованием Chai-1, базовой модели молекулярной структуры. Используйте этот навык, когда: (1) прогнозируете структуры белково-белковых комплексов, (2) проверяете разработанные связующие, (3) прогнозируете комплексы белок-лиганд, (4) используете Chai API для высокопроизводительного прогнозирования, (5) нужна альтернатива AlphaFold2.
Для порогов контроля качества используйте белок-QC. Для прогнозирования AlphaFold2 используйте альфафолд. Для анализа на основе ESM используйте esm.
