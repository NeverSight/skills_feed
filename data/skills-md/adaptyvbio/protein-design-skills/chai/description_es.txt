Predicción de estructuras utilizando Chai-1, un modelo básico para la estructura molecular. Utilice esta habilidad cuando: (1) Prediga estructuras de complejos proteína-proteína, (2) Valide aglutinantes diseñados, (3) Prediga complejos proteína-ligando, (4) Utilice Chai API para predicción de alto rendimiento, (5) Necesita una alternativa a AlphaFold2.
Para los umbrales de control de calidad, utilice proteína-qc. Para la predicción AlphaFold2, utilice alphafold. Para análisis basado en ESM, utilice esm.
