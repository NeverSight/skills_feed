Predicción de estructuras utilizando Boltz-1/Boltz-2, un predictor de estructuras biomoleculares abierto. Utilice esta habilidad cuando: (1) Prediga estructuras de complejos de proteínas, (2) Valide aglutinantes diseñados, (3) Necesita una alternativa de código abierto a AF2, (4) Prediga complejos de proteína-ligando, (5) Utilice recursos de GPU locales.
Para los umbrales de control de calidad, utilice proteína-qc. Para la predicción AlphaFold2, utilice alphafold. Para la predicción de Chai, utilice chai.
