Prédiction de structure à l'aide de Boltz-1/Boltz-2, un prédicteur de structure biomoléculaire ouvert. Utilisez cette compétence lorsque : (1) prédire les structures de complexes protéiques, (2) valider les liants conçus, (3) avoir besoin d'une alternative open source à AF2, (4) prédire les complexes protéine-ligand, (5) utiliser les ressources GPU locales.
Pour les seuils QC, utilisez Protein-Qc. Pour la prédiction AlphaFold2, utilisez alphafold. Pour la prédiction Chai, utilisez chai.
