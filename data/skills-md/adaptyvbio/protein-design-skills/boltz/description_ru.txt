Прогнозирование структуры с использованием Boltz-1/Boltz-2, открытого предсказателя биомолекулярной структуры. Используйте этот навык, когда: (1) Прогнозирование структур белковых комплексов, (2) Проверка разработанных связующих, (3) Нужна альтернатива AF2 с открытым исходным кодом, (4) Прогнозирование комплексов белок-лиганд, (5) Использование локальных ресурсов графического процессора.
Для порогов контроля качества используйте белок-QC. Для прогнозирования AlphaFold2 используйте альфафолд. Для предсказания Чай используйте chai.
