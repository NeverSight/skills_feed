Полноатомный дизайн белка с использованием диффузионной модели BoltzGen. Используйте этот навык, когда: (1) с самого начала требуется проектирование с учетом боковой цепи, (2) проектирование вокруг небольших молекул или лигандов, (3) требуется диффузия всех атомов (а не только основной цепи), (4) требуется точная геометрия связывания, (5) использование конфигурации на основе YAML.
Для генерации только магистральной сети используйте RFDiffusion. Для проектирования только последовательностей используйте протеинmpnn. Для проверки структуры используйте Boltz.
