Руководство по выбору подходящего инструмента для разработки белкового связующего. Используйте этот навык, когда: (1) принимаете решение между BoltzGen, BindCraft или RFdiffusion, (2) планируете кампанию по разработке переплетного устройства, (3) понимаете компромиссы между различными подходами, (4) выбираете инструменты для конкретных типов целей.
Для конкретных параметров инструмента используйте индивидуальные навыки инструмента (больцген, биндкрафт, радиодиффузия и т. д.).
