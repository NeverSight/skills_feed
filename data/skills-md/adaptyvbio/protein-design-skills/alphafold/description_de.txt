Validieren Sie Proteindesigns mithilfe der AlphaFold2-Strukturvorhersage. Verwenden Sie diese Fähigkeit, wenn: (1) entworfene Sequenzen korrekt gefaltet werden, (2) komplexe Bindungs-Ziel-Strukturen vorhergesagt werden, (3) Konfidenzmetriken berechnet werden (pLDDT, pTM, ipTM), (4) Selbstkonsistenzvalidierung von Designs, (5) mehrkettige komplexe Vorhersage mit AlphaFold-Multimer.
Für eine schnellere Einzelkettenvorhersage verwenden Sie esm. Verwenden Sie für QC-Schwellenwerte protein-qc.
