Flujo de trabajo de auditoría de seguridad integral para revisar código confidencial e informar riesgos de seguridad creíbles con pruebas y requisitos previos claros para el atacante. Úselo cuando los usuarios pregunten "¿es esto seguro?", "verifique si hay vulnerabilidades" o quieran una revisión centrada en la seguridad de autenticación/autorización, pagos, secretos, PII, operaciones de archivos o funciones similares críticas para la seguridad (incluidas auditorías de archivos únicos o de fragmentos de código).
