Modèles de développement Apple multiplateformes pour le partage de code entre iOS, macOS, iPadOS, watchOS, tvOS et visionOS. Comprend une compilation conditionnelle, des extensions spécifiques à la plate-forme, des packages partagés et des modèles d'interface utilisateur adaptatifs. À utiliser lorsque l'utilisateur pose des questions sur les applications multiplateformes, le partage de code multiplateforme, les conditions de plateforme ou les applications Apple universelles.
