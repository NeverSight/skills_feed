Migration de Combine vers le framework Swift Observation et les modèles asynchrones/attente modernes. Couvre la conversion de Publisher vers AsyncSequence, la migration d'ObservableObject vers @Observable, les modèles de pontage et la modernisation du code réactif. À utiliser lorsque l'utilisateur pose des questions sur la migration de Combine, ObservableObject vers Observable, Publisher vers AsyncSequence ou sur la modernisation du code réactif.
