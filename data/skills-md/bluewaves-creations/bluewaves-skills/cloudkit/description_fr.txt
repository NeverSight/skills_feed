Framework CloudKit pour le stockage, la synchronisation et le partage de données iCloud. Couvre CKContainer, CKDatabase, CKRecord, CKQuery, les abonnements, CKShare et UICloudSharingController. À utiliser lorsque l'utilisateur pose des questions sur CloudKit, CKRecord, CKDatabase, la base de données publique iCloud, les abonnements CloudKit, CKShare, le partage d'enregistrements ou UICloudSharingController.
