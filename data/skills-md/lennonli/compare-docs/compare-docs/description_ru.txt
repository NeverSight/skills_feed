Сравнивает два документа (docx, md, txt, pdf) для выявления существенных различий, уделяя особое внимание финансовым данным, ключевым статьям и статистическим расхождениям. Создает форматированный отчет сравнения. Используйте, когда пользователь просит «сравнить документы», «проверить различия между файлами», «найти несоответствия», «比对文档», «查看差异», «比较实质性差异» или «核对合同».
