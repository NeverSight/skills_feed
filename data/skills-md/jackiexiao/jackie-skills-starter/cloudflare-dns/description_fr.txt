Configurez et dépannez les enregistrements DNS Cloudflare et les liaisons de domaine personnalisé Pages. À utiliser lorsqu'un utilisateur demande d'ajouter/modifier/supprimer des enregistrements DNS, de mapper un domaine personnalisé à Cloudflare Pages/Workers, de vérifier la propagation ou de corriger les erreurs de validation de domaine. Lors de la première utilisation, demandez la configuration du jeton API avec les autorisations de modification DNS avant d'apporter des modifications.
