قم بتكوين واستكشاف أخطاء سجلات Cloudflare DNS وروابط المجال المخصص للصفحات وإصلاحها. يُستخدم عندما يطلب المستخدم إضافة/تحرير/حذف سجلات DNS، أو تعيين مجال مخصص لصفحات/عاملي Cloudflare، أو التحقق من النشر، أو إصلاح أخطاء التحقق من صحة المجال. عند الاستخدام لأول مرة، اطلب إعداد رمز API المميز مع أذونات تحرير DNS قبل إجراء التغييرات.
