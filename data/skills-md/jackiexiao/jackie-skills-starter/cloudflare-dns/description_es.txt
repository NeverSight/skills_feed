Configure y solucione problemas de registros DNS de Cloudflare y enlaces de dominio personalizados de páginas. Úselo cuando un usuario solicite agregar/editar/eliminar registros DNS, asignar un dominio personalizado a páginas/trabajadores de Cloudflare, verificar la propagación o corregir errores de validación de dominio. En el primer uso, solicite la configuración del token API con permisos de edición de DNS antes de realizar cambios.
