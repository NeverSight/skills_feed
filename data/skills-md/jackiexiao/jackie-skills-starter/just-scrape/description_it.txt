Strumento CLI per web scraping, estrazione dati, ricerca e scansione basati sull'intelligenza artificiale tramite ScrapeGraph AI. Utilizzare quando l'utente ha bisogno di raschiare siti Web, estrarre dati strutturati da URL, convertire pagine in markdown, eseguire la scansione di siti multipagina, cercare informazioni sul Web, automatizzare le interazioni del browser (accesso, clic, compilare moduli), ottenere HTML non elaborato, scoprire mappe dei siti o generare schemi JSON. Trigger su attività che coinvolgono: (1) estrazione di dati da siti Web, (2) web scraping o scansione, (3) conversione di pagine Web in markdown, (4) ricerca Web basata sull'intelligenza artificiale con estrazione, (5) automazione del browser, (6) generazione di schemi di output per lo scraping. La CLI è just-scrape (pacchetto npm just-scrape).
