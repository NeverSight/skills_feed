ScrapeGraph AI를 통한 AI 기반 웹 스크래핑, 데이터 추출, 검색 및 크롤링을 위한 CLI 도구입니다. 사용자가 웹사이트 스크랩, URL에서 구조화된 데이터 추출, 페이지를 마크다운으로 변환, 다중 페이지 사이트 크롤링, 웹에서 정보 검색, 브라우저 상호 작용 자동화(로그인, 클릭, 양식 채우기), 원시 HTML 가져오기, 사이트맵 검색 또는 JSON 스키마 생성이 필요한 경우에 사용하세요. (1) 웹사이트에서 데이터 추출, (2) 웹 스크래핑 또는 크롤링, (3) 웹페이지를 마크다운으로 변환, (4) 추출을 통한 AI 기반 웹 검색, (5) 브라우저 자동화, (6) 스크래핑을 위한 출력 스키마 생성과 관련된 작업을 트리거합니다. CLI는 just-scrape(npm 패키지 just-scrape)입니다.
