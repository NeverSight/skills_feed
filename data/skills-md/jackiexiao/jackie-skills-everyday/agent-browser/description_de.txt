Browser-Automatisierungs-CLI des AI-Agenten. Verwenden Sie es, wenn Benutzer mit Websites interagieren müssen – durch Seiten navigieren, Formulare ausfüllen, auf Schaltflächen klicken, Screenshots machen, Daten extrahieren, Web-Apps testen oder beliebige Browseraufgaben automatisieren. Zu den Auslösern gehören „Website öffnen“, „Formular ausfüllen“, „Schaltfläche anklicken“, „Screenshot erstellen“, „Daten von der Seite entfernen“, „Diese Web-App testen“, „Auf der Website anmelden“, „Browseraktionen automatisieren“ oder jede Aufgabe, die eine programmatische Webinteraktion erfordert.
