أتمتة المتصفح المدعوم بالذكاء الاصطناعي CLI. يُستخدم عندما يحتاج المستخدمون إلى التفاعل مع مواقع الويب، مثل التنقل بين الصفحات، أو ملء النماذج، أو النقر على الأزرار، أو التقاط لقطات الشاشة، أو استخراج البيانات، أو اختبار تطبيقات الويب، أو تشغيل أي مهمة في المتصفح تلقائيًا. تتضمن المشغلات "فتح موقع ويب"، أو "ملء النموذج"، أو "زر النقر"، أو "لقطة الشاشة"، أو "استخراج البيانات من الصفحة"، أو "اختبار تطبيق الويب هذا"، أو "تسجيل الدخول إلى الموقع"، أو "أتمتة إجراءات المتصفح"، أو أي مهمة تتطلب تفاعلًا برمجيًا على الويب.
