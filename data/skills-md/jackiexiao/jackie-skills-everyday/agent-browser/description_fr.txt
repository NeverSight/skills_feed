CLI d'automatisation du navigateur basée sur l'IA. À utiliser lorsque les utilisateurs ont besoin d'interagir avec des sites Web : parcourir des pages, remplir des formulaires, cliquer sur des boutons, prendre des captures d'écran, extraire des données, tester des applications Web ou automatiser n'importe quelle tâche du navigateur. Les déclencheurs incluent « ouvrir un site Web », « remplir un formulaire », « cliquer sur un bouton », « capturer une capture d'écran », « extraire les données de la page », « tester cette application Web », « se connecter au site », « automatiser les actions du navigateur » ou toute tâche nécessitant une interaction Web programmatique.
