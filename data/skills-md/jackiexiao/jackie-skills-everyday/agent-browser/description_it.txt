CLI di automazione del browser basata sull'intelligenza artificiale. Utilizzalo quando gli utenti devono interagire con i siti Web: navigare tra le pagine, compilare moduli, fare clic sui pulsanti, acquisire screenshot, estrarre dati, testare app Web o automatizzare qualsiasi attività del browser. I trigger includono "apri sito Web", "compila modulo", "fai clic sul pulsante", "screenshot", "rimuovi dati dalla pagina", "prova questa app Web", "accedi al sito", "automatizza le azioni del browser" o qualsiasi attività che richieda un'interazione web programmatica.
