Opérations sur les fichiers du référentiel GitLab via API. Utilisez TOUJOURS cette compétence lorsque l'utilisateur souhaite : (1) lire le contenu d'un fichier à partir de GitLab, (2) créer/mettre à jour/supprimer des fichiers via l'API, (3) obtenir des informations sur la faute du fichier, (4) télécharger des fichiers bruts.
