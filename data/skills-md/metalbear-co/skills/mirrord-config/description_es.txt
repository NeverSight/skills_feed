Ayuda a los usuarios a generar, editar y validar archivos de configuración mirrord.json para mirrord (MetalBear). Úselo cuando el usuario desee conectar su proceso local a un entorno de Kubernetes, configurar funciones (env/fs/network) o necesite comentarios sobre un mirrord.json existente. Siempre garantiza que el JSON de salida sea válido y conforme al esquema.
