Помогает пользователям настроить Mirrord.json для ветвления базы данных, обеспечивая изолированные копии базы данных для безопасной разработки и тестирования. Используйте, когда пользователь хочет настроить ветвление MySQL или PostgreSQL, настроить режимы копирования, аутентификацию IAM или управлять ветвями базы данных.
