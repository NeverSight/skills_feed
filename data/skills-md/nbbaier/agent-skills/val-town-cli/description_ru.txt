Управляйте проектами Val Town с помощью vt CLI. Используйте при работе с Vals (бессерверные функции Val Town), синхронизации кода с Val Town, создании конечных точек HTTP, потоковой передаче журналов или управлении ветвями Val Town. Запускается при выполнении задач, связанных с разработкой города Вал, созданием/редактированием валов или когда пользователь упоминает «vt», «val Town» или «vals».
