Gérez les projets Val Town à l'aide de la CLI vt. À utiliser lorsque vous travaillez avec Vals (fonctions sans serveur de Val Town), synchronisez le code avec Val Town, créez des points de terminaison HTTP, diffusez des journaux en continu ou gérez des succursales de Val Town. Se déclenche sur les tâches impliquant le développement de Val Town, la création/édition de val, ou lorsque l'utilisateur mentionne « vt », « val town » ou « vals ».
