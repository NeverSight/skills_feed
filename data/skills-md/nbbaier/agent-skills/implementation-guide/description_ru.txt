Создавайте подробные руководства по реализации задач кодирования вместо того, чтобы писать код напрямую. Используйте, когда пользователь запрашивает подробную документацию по реализации, пошаговые руководства по разработке или когда он хочет реализовать функции самостоятельно с помощью таких инструментов, как Cursor. Создает исчерпывающие руководства с исходным контекстом, архитектурными решениями, контрольными точками с точками проверки и обоснованием рабочего процесса «сделай сам».
