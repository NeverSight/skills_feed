قم بإنشاء أدلة تنفيذ شاملة لمهام البرمجة بدلاً من كتابة التعليمات البرمجية مباشرة. يُستخدم عندما يطلب المستخدم وثائق تنفيذ مفصلة، ​​أو أدلة تطوير خطوة بخطوة، أو عندما يريد تنفيذ الميزات بنفسه باستخدام أدوات مثل المؤشر. ينشئ أدلة شاملة مع سياق الخلفية، وقرارات التصميم، والمعالم الرئيسية مع نقاط التحقق، والأساس المنطقي لسير عمل "البناء بنفسك".
