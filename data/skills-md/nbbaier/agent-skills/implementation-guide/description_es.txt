Genere guías de implementación completas para tareas de codificación en lugar de escribir código directamente. Úselo cuando el usuario solicite documentación de implementación detallada, guías de desarrollo paso a paso o cuando desee implementar funciones por sí mismo utilizando herramientas como Cursor. Crea guías exhaustivas con contexto de fondo, decisiones de arquitectura, hitos con puntos de verificación y fundamentos para un flujo de trabajo "constrúyalo usted mismo".
