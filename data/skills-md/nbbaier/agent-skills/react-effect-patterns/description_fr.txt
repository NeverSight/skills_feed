Lignes directrices pour une utilisation appropriée de React et des effets et pour éviter les effets inutiles. À utiliser lors de l'écriture, de la révision ou de la refactorisation de composants React qui utilisent useEffect, useState ou gèrent les effets secondaires. Déclencheurs sur des tâches impliquant React Effects, un état dérivé, des gestionnaires d'événements, une récupération de données ou une synchronisation de composants.
