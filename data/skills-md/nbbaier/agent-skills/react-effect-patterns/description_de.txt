Richtlinien für die ordnungsgemäße Verwendung von React useEffect und die Vermeidung unnötiger Effekte. Wird beim Schreiben, Überprüfen oder Refactoring von React-Komponenten verwendet, die useEffect, useState verwenden oder Nebenwirkungen verarbeiten. Löst Aufgaben aus, die Reaktionseffekte, abgeleitete Zustände, Ereignishandler, Datenabruf oder Komponentensynchronisierung umfassen.
