Руководство по созданию пользовательских субагентов Claude Code. Используйте, когда пользователь хочет создать новый агент (или обновить существующий агент), который обрабатывает определенные типы задач с помощью настраиваемых подсказок, ограничений инструментов и разрешений. Запускается по запросам на создание агентов, субагентов, пользовательских агентов или когда пользователю нужны специализированные помощники ИИ для конкретных рабочих процессов.
