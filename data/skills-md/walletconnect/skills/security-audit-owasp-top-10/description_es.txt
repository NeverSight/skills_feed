Realiza una auditoría de seguridad integral de cualquier base de código contra OWASP Top 10 2025. Úselo cuando el usuario solicita una auditoría de OWASP, una revisión de OWASP Top 10, una verificación de seguridad de OWASP o desea auditar el código contra las categorías de OWASP. No lo active para revisión de relaciones públicas, auditoría de npm/pip, cumplimiento de SOC2, preguntas de seguridad generales o modelado de amenazas.
