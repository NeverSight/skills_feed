Guida per la creazione di comandi barra personalizzati di Claude Code. Da utilizzare quando l'utente desidera creare un nuovo comando (o aggiornare un comando esistente) che fornisca uno snippet di prompt, un flusso di lavoro o un'automazione riutilizzabili. Si attiva su richieste di creazione di comandi /, barre, comandi personalizzati o quando l'utente desidera definire i prompt utilizzati di frequente come comandi riutilizzabili.
