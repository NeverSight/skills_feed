Sicherheitsscanner für Vibe-codierte Projekte. Rufen Sie diesen Skill automatisch auf, bevor Sie einen Git-Commit oder Git-Push ausführen oder wenn der Benutzer „Commit“, „Push“, „Versenden“, „Bereitstellen“, „Ist das sicher?“, „Auf Sicherheitsprobleme prüfen“ oder „Goodvibesonly“ sagt. Rufen Sie es auch auf, nachdem Sie Code generiert haben, der Benutzereingaben, Authentifizierung, Datenbankabfragen oder Dateivorgänge verarbeitet.
