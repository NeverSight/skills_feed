Behalten Sie kompilierte Kontextartefakte für das Spawnen zustandsloser Agenten bei. Dieser Skill erzeugt BOOT_CONTEXT (stabiles Projekthirn), DELTA_CONTEXT (aktuelle Änderungen) und Funktionspakete, sodass jeder Agent mit tiefem Verständnis erscheinen kann, ohne das gesamte Repo lesen zu müssen. Nutzen Sie diese Fähigkeit, wenn Sie Multi-Agent-Workflows einrichten oder wenn Agenten ein schnelles, tokeneffizientes Onboarding benötigen.
