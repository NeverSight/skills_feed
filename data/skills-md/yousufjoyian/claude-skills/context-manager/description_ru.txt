Поддерживать скомпилированные артефакты контекста для создания агентов без сохранения состояния. Этот навык создает BOOT_CONTEXT (стабильный мозг проекта), DELTA_CONTEXT (последние изменения) и пакеты функций, поэтому любой агент может создаваться с глубоким пониманием, не читая весь репозиторий. Используйте этот навык при настройке рабочих процессов с несколькими агентами или когда агентам требуется быстрая и эффективная адаптация с использованием токенов.
