Automazione completa del browser con Playwright. Rileva automaticamente i server di sviluppo, scrive script di test puliti su /tmp. Testa le pagine, compila moduli, acquisisci screenshot, controlla il design reattivo, convalida l'UX, testa i flussi di accesso, controlla i collegamenti, automatizza qualsiasi attività del browser. Utilizzare quando l'utente desidera testare siti Web, automatizzare le interazioni del browser, convalidare la funzionalità Web o eseguire qualsiasi test basato sul browser.
