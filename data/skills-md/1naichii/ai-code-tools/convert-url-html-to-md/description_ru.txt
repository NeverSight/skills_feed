Преобразование веб-страниц HTML в Markdown для извлечения документации. Используйте при получении документации с веб-сайтов, извлечении структурированного контента, очистке основного контента (clean=true) или получении полной страницы с навигационной/боковой панелью для обнаружения URL-адресов (clean=false). Идеально подходит для изучения библиотек и построения контекста из веб-источников.
