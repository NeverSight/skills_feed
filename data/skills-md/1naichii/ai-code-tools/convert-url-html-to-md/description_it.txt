Converti pagine Web HTML in Markdown per l'estrazione della documentazione. Da utilizzare quando si recupera documentazione da siti Web, si estrae contenuto strutturato, si ottiene contenuto principale pulito (clean=true) o si ottiene una pagina intera con nav/barra laterale per scoprire URL (clean=false). Ideale per l'apprendimento delle biblioteche e la creazione di contesto da fonti web.
