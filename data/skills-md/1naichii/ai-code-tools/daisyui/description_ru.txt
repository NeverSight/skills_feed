Библиотека компонентов CSS для Tailwind CSS 4, предоставляющая предварительно созданные компоненты пользовательского интерфейса с семантическими именами классов. Используйте при создании веб-интерфейсов с помощью HTML/Tailwind, которым необходимы: (1) быстрая разработка пользовательского интерфейса с единообразным стилем, (2) доступные шаблоны компонентов (кнопки, формы, модальные окна и т. д.), (3) цветовые системы с поддержкой тем, которые работают в светлых/темных режимах, (4) адаптивные макеты с утилитами Tailwind. Работает с DaisyUI v5+, для которого требуется Tailwind CSS v4+.
