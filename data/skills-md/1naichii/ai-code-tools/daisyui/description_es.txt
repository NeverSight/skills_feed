Biblioteca de componentes CSS para Tailwind CSS 4 que proporciona componentes de interfaz de usuario prediseñados con nombres de clases semánticos. Úselo al crear interfaces web con HTML/Tailwind que necesiten: (1) Desarrollo rápido de la interfaz de usuario con estilo consistente, (2) Patrones de componentes accesibles (botones, formularios, modales, etc.), (3) Sistemas de color basados ​​en temas que funcionan en modos claro/oscuro, (4) Diseños responsivos con utilidades Tailwind. Funciona con daisyUI v5+ que requiere Tailwind CSS v4+.
