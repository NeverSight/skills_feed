Создавайте современные монолитные приложения с помощью Inertia.js — объединяя серверные платформы (Laravel, Rails и т. д.) с интерфейсами React/Vue/Svelte без создания API. Используйте при создании страниц и макетов Inertia, работе с компонентом Link для навигации, создании форм с компонентом Form или перехватчиком useForm, обработке проверок и ошибок, управлении общими данными и реквизитами, реализации аутентификации и авторизации, использовании ручных посещений маршрутизатора, работе с частичными перезагрузками, настройке постоянных макетов или настройке настройки на стороне клиента.
