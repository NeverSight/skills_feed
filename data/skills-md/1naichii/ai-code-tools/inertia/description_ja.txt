Inertia.js を使用して最新のモノリス アプリケーションを構築します。API を構築せずに、サーバー側フレームワーク (Laravel、Rails など) と React/Vue/Svelte フロントエンドを組み合わせます。 Inertia ページとレイアウトの作成、ナビゲーション用の Link コンポーネントの操作、Form コンポーネントまたは useForm フックを使用したフォームの構築、検証とエラーの処理、共有データとプロパティの管理、認証と認可の実装、ルーターによる手動訪問の使用、部分リロードの操作、永続的なレイアウトのセットアップ、またはクライアント側セットアップの構成を行うときに使用します。
