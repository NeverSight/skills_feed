Crea moderne applicazioni monolitiche con Inertia.js, combinando framework lato server (Laravel, Rails, ecc.) con frontend React/Vue/Svelte senza creare API. Da utilizzare durante la creazione di pagine e layout Inertia, l'utilizzo del componente Link per la navigazione, la creazione di moduli con il componente Form o l'hook useForm, la gestione della convalida e degli errori, la gestione di dati e oggetti di scena condivisi, l'implementazione di autenticazione e autorizzazione, l'utilizzo di visite manuali con il router, l'utilizzo di ricariche parziali, l'impostazione di layout persistenti o la configurazione dell'impostazione lato client.
