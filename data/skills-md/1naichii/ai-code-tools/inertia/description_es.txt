Cree aplicaciones monolíticas modernas con Inertia.js, combinando marcos del lado del servidor (Laravel, Rails, etc.) con interfaces React/Vue/Svelte sin crear API. Úselo al crear páginas y diseños de Inercia, trabajar con el componente Enlace para la navegación, crear formularios con el componente Formulario o usar el gancho de formulario, manejar la validación y los errores, administrar datos y accesorios compartidos, implementar autenticación y autorización, usar visitas manuales con el enrutador, trabajar con recargas parciales, configurar diseños persistentes o configurar la configuración del lado del cliente.
