أنشئ تطبيقات متراصة حديثة باستخدام Inertia.js - حيث تجمع بين أطر العمل من جانب الخادم (Laravel وRails وما إلى ذلك) مع واجهات React/Vue/Svelte الأمامية دون إنشاء واجهات برمجة التطبيقات. يُستخدم عند إنشاء صفحات وتخطيطات القصور الذاتي، أو العمل مع مكون الارتباط للتنقل، أو إنشاء النماذج باستخدام مكون النموذج أو ربط useForm، أو التعامل مع التحقق من الصحة والأخطاء، أو إدارة البيانات والدعائم المشتركة، أو تنفيذ المصادقة والترخيص، أو استخدام الزيارات اليدوية مع جهاز التوجيه، أو العمل مع عمليات إعادة التحميل الجزئية، أو إعداد التخطيطات المستمرة، أو تكوين الإعداد من جانب العميل.
