Laravel Blade 템플릿을 위한 Lucide 아이콘 라이브러리 통합. Laravel 애플리케이션의 Lucide 아이콘, x-lucide- 접두사가 있는 Blade 구성 요소, Tailwind CSS를 사용한 아이콘 스타일, Blade의 동적 아이콘 렌더링 또는 SVG 아이콘이 필요한 모든 Laravel 보기 작업에 사용할 수 있습니다. 키워드에는 lucide 아이콘, 블레이드 아이콘, x-lucide, SVG 아이콘 Laravel, 블레이드-lucide-아이콘, mallardduck/blade-lucide-icons가 포함됩니다.
