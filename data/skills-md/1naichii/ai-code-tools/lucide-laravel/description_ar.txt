تكامل مكتبة أيقونات Lucide لقوالب Laravel Blade. يُستخدم عند العمل مع أيقونات Lucide في تطبيقات Laravel، أو مكونات Blade ذات البادئة x-lucide-، أو تصميم الأيقونات باستخدام Tailwind CSS، أو عرض الأيقونات الديناميكي في Blade، أو أي عمل عرض Laravel يتطلب أيقونات SVG. تتضمن الكلمات الرئيسية أيقونات لوسايد، أيقونات بليد، x-لوسيد، أيقونات SVG Laravel، أيقونات بليد لوسايد، أيقونات مالارد داك/بليد لوسايد.
