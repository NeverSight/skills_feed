Интеграция библиотеки значков Lucid для шаблонов Laravel Blade. Используйте при работе с значками Lucid в приложениях Laravel, компонентами Blade с префиксом x-lucide-, стилизацией значков с помощью Tailwind CSS, динамическим рендерингом значков в Blade или любой работой представления Laravel, требующей значков SVG. Ключевые слова включают значки люцида, значки лезвий, x-lucid, значки SVG Laravel, значки клинков-люцидов, кряквенные утки/иконки-лезвий-люцидов.
