Recupera il contenuto web con il rilevamento automatico della versione di markdown utilizzando curl. Da utilizzare quando Claude deve recuperare documentazione da siti Web che offrono sia formati HTML che markdown. Prima controlla se esiste una versione .md (più efficiente e più pulita), quindi torna all'HTML se non disponibile. Ideale per recuperare documentazione da siti come ui.shadcn.com, wiki GitHub o qualsiasi sito di documentazione che rispecchia il contenuto in formato markdown.
