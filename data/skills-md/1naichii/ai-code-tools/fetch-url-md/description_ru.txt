Получайте веб-контент с автоматическим определением версии уценки с помощью Curl. Используйте, когда Клоду необходимо получить документацию с веб-сайтов, которые предлагают форматы HTML и уценки. Сначала проверяет, существует ли версия .md (более эффективная и чистая), затем возвращается к HTML, если она недоступна. Идеально подходит для получения документации с таких сайтов, как ui.shadcn.com, вики GitHub или любого сайта документации, который отображает контент в формате уценки.
