Obtenga contenido web con detección automática de versiones de rebajas mediante curl. Úselo cuando Claude necesite recuperar documentación de sitios web que ofrecen formatos HTML y Markdown. Primero comprueba si existe una versión .md (más eficiente y limpia), luego recurre a HTML si no está disponible. Ideal para obtener documentación de sitios como ui.shadcn.com, wikis de GitHub o cualquier sitio de documentación que refleje el contenido en formato Markdown.
