جلب محتوى الويب من خلال الكشف التلقائي عن إصدار تخفيض السعر باستخدام الضفيرة. يُستخدم عندما يحتاج Claude إلى استرداد الوثائق من مواقع الويب التي تقدم تنسيقات HTML وmarkdown. يتحقق أولاً من وجود إصدار .md (أكثر كفاءة ونظافة)، ثم يعود إلى HTML إذا لم يكن متاحًا. مثالي لجلب الوثائق من مواقع مثل ui.shadcn.com، أو مواقع GitHub wiki، أو أي موقع توثيق يعكس المحتوى بتنسيق تخفيض السعر.
