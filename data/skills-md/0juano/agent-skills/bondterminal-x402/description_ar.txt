الاستعلام عن واجهة برمجة تطبيقات BondTerminal باستخدام الدفعات بدون مفتاح x402. لا حاجة إلى مفتاح API - ادفع 0.01 دولارًا أمريكيًا لكل طلب على الشبكة الرئيسية الأساسية. يُستخدم عندما يطلب المستخدمون بيانات السندات الأرجنتينية، أو التحليلات، أو التدفقات النقدية، أو السجل، أو riesgo país، أو عمليات البحث عن رقم ISIN/الشريط (على سبيل المثال AL30، GD30، US040114HS26). يدعم التلقائي 402 → الدفع → إعادة المحاولة.
