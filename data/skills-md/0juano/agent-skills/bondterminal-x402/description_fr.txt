Interrogez l’API BondTerminal à l’aide des paiements sans clé x402. Aucune clé API nécessaire : payez 0,01 USDC par requête sur le réseau principal de base. À utiliser lorsque les utilisateurs demandent des données sur les obligations argentines, des analyses, des flux de trésorerie, l'historique, le riesgo país ou des recherches d'ISIN/ticker (par exemple AL30, GD30, US040114HS26). Prend en charge le 402 automatique → paiement → réessayer.
