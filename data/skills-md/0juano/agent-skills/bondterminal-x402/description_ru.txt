Запросите API BondTerminal, используя платежи без ключа x402. Ключ API не требуется — платите 0,01 доллара США за запрос в основной сети Base. Используйте, когда пользователи запрашивают данные по аргентинским облигациям, аналитику, денежные потоки, историю, riesgo país или поиск ISIN/тикеров (например, AL30, GD30, US040114HS26). Поддерживает автоматическую 402 → оплата → повтор.
