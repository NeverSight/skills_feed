التخطيط الشامل للمشاريع ومولد الوثائق لمشاريع البرمجيات. يقوم بإنشاء مستندات المتطلبات المنظمة ومستندات تصميم النظام وخطط تقسيم المهام مع تتبع التنفيذ. يُستخدم عند بدء مشروع جديد، أو تحديد المواصفات، أو إنشاء تصميمات فنية، أو تقسيم الأنظمة المعقدة إلى مهام قابلة للتنفيذ. يدعم تنسيق قصة المستخدم، ومعايير القبول، وتصميم المكونات، ومواصفات واجهة برمجة التطبيقات (API)، وتحليل المهام الهرمية مع إمكانية تتبع المتطلبات.
