Комплексное планирование проектов и генератор документации для программных проектов. Создает структурированные документы с требованиями, документы по проектированию системы и планы разбивки задач с отслеживанием реализации. Используйте при запуске нового проекта, определении спецификаций, создании технических проектов или разбиении сложных систем на реализуемые задачи. Поддерживает формат пользовательской истории, критерии приемки, проектирование компонентов, спецификации API и иерархическую декомпозицию задач с возможностью отслеживания требований.
