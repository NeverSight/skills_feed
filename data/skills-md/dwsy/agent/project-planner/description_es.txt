Planificación integral de proyectos y generador de documentación para proyectos de software. Crea documentos de requisitos estructurados, documentos de diseño de sistemas y planes de desglose de tareas con seguimiento de implementación. Úselo al iniciar un nuevo proyecto, definir especificaciones, crear diseños técnicos o dividir sistemas complejos en tareas implementables. Admite formato de historias de usuario, criterios de aceptación, diseño de componentes, especificaciones API y descomposición jerárquica de tareas con trazabilidad de requisitos.
