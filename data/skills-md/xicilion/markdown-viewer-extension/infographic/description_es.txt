Crea hermosas infografías con plantillas prediseñadas. Lo mejor para tarjetas KPI, cronogramas, hojas de ruta, procesos paso a paso, comparaciones A vs B, análisis FODA, embudos, árboles organizativos, gráficos circulares/de barras. Úselo cuando necesite un impacto visual rápido con 4-8 elementos. Sintaxis simple similar a YAML. NO para análisis de datos complejos (use vega) o diagramas de flujo técnicos (use sirena).
