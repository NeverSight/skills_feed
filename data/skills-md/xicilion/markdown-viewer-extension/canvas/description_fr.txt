Créez des diagrammes basés sur des nœuds spatiaux avec un positionnement libre. Idéal pour les cartes mentales, les graphiques de connaissances, les cartes conceptuelles et les tableaux de planification où une disposition spatiale précise est importante. Utilisez le format JSON avec les coordonnées x/y. PAS pour les flux séquentiels (utilisez sirène) ou les graphiques de données (utilisez Vega).
