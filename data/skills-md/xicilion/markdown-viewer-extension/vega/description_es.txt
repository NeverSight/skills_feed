Cree gráficos basados ​​en datos con Vega-Lite (simple) y Vega (avanzado). Lo mejor para barras, líneas, dispersión, mapas de calor, gráficos de áreas y análisis de series múltiples. Úselo cuando tenga matrices de datos numéricos que necesiten visualización estadística. Vega para gráficos de radar y nubes de palabras. NO para diagramas de procesos (use sirena) o tarjetas KPI rápidas (use infografía).
