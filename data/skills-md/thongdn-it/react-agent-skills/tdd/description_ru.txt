Методология разработки через тестирование и рабочий процесс красно-зеленого рефакторинга (ранее — test-tdd). Этот навык следует использовать при практике TDD, предварительном написании тестов, разработке тестов перед реализацией или анализе подходов, основанных на тестировании. Запускается по принципу «сначала напишите тесты», «тестируйте перед написанием кода», «красно-зеленый рефакторинг», «разработку через тестирование». Этот навык НЕ охватывает специфику платформы Vitest (используйте навык vitest) или имитацию API с помощью MSW (используйте навык msw).
