أفضل ممارسات nuqs (حالة استعلام URL الآمنة للنوع) لتطبيقات Next.js. يجب استخدام هذه المهارة عند كتابة أو مراجعة أو إعادة بناء التعليمات البرمجية التي تستخدم nuqs لإدارة حالة عنوان URL. يتم تشغيل المهام التي تتضمن useQueryState، أو useQueryStates، أو معلمات البحث، أو حالة عنوان URL، أو معلمات الاستعلام، أو محللي nuqs، أو التوجيه Next.js مع الحالة.
