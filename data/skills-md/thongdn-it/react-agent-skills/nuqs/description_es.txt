Mejores prácticas de nuqs (estado de consulta de URL de tipo seguro) para aplicaciones Next.js. Esta habilidad se debe utilizar al escribir, revisar o refactorizar código que utilice nuqs para la gestión del estado de la URL. Activadores de tareas que involucran useQueryState, useQueryStates, parámetros de búsqueda, estado de URL, parámetros de consulta, analizadores nuqs o enrutamiento Next.js con estado.
