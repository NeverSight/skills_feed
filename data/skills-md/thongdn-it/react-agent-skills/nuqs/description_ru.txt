Рекомендации по nuqs (типобезопасному состоянию URL-запроса) для приложений Next.js. Этот навык следует использовать при написании, проверке или рефакторинге кода, использующего nuqs для управления состоянием URL-адресов. Запускается по задачам, связанным с useQueryState, useQueryStates, параметрами поиска, состоянием URL-адреса, параметрами запроса, анализаторами nuqs или маршрутизацией Next.js с состоянием.
