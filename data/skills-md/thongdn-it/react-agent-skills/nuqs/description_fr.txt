Meilleures pratiques nuqs (état de requête d’URL de type sécurisé) pour les applications Next.js. Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la refactorisation de code qui utilise des nuqs pour la gestion de l'état des URL. Déclencheurs sur des tâches impliquant useQueryState, useQueryStates, les paramètres de recherche, l'état de l'URL, les paramètres de requête, les analyseurs nuqs ou le routage Next.js avec état.
