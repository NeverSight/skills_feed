Modèles de cadre de test Vitest pour la configuration des tests, les tests asynchrones, la simulation avec vi.*, les instantanés et les performances des tests (anciennement test-vitest). Cette compétence doit être utilisée lors de l'écriture ou du débogage des tests Vitest. Cette compétence ne couvre PAS la méthodologie TDD (utilisez la compétence test-tdd), la simulation d'API avec MSW (utilisez la compétence test-msw) ou les API spécifiques à Jest.
