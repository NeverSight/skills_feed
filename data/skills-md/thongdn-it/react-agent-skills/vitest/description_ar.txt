أنماط إطار اختبار Vitest لإعداد الاختبار، والاختبار غير المتزامن، والاستهزاء باستخدام vi.*، واللقطات، وأداء الاختبار (المعروف سابقًا باسم test-vitest). يجب استخدام هذه المهارة عند كتابة أو تصحيح اختبارات Vitest. لا تغطي هذه المهارة منهجية TDD (استخدم مهارة اختبار tdd)، أو محاكاة واجهة برمجة التطبيقات باستخدام MSW (استخدم مهارة اختبار msw)، أو واجهات برمجة التطبيقات الخاصة بـ Jest.
