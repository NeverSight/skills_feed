Шаблоны среды тестирования Vitest для настройки тестов, асинхронного тестирования, макетирования с помощью vi.*, снимков и производительности тестов (ранее — test-vitest). Этот навык следует использовать при написании или отладке тестов Vitest. Этот навык НЕ охватывает методологию TDD (используйте навык test-tdd), имитацию API с помощью MSW (используйте навык test-msw) или API, специфичные для Jest.
