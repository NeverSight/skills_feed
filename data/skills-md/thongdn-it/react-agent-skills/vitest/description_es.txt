Patrones del marco de prueba de Vitest para configuración de pruebas, pruebas asíncronas, burlas con vi.*, instantáneas y rendimiento de pruebas (anteriormente test-vitest). Esta habilidad debe usarse al escribir o depurar pruebas de Vitest. Esta habilidad NO cubre la metodología TDD (use la habilidad test-tdd), la simulación de API con MSW (use la habilidad test-msw) ni las API específicas de Jest.
