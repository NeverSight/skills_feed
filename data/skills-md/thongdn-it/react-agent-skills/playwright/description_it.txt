Drammaturgo che testa le migliori pratiche per le applicazioni Next.js (in precedenza test-drammaturgo). Questa abilità dovrebbe essere utilizzata durante la scrittura, la revisione o il debug dei test E2E con Playwright. Trigger su attività che coinvolgono selettori di test, test instabili, stato di autenticazione, simulazione dell'API, test di idratazione, esecuzione parallela, configurazione CI o debug di errori di test.
