Playwright teste les meilleures pratiques pour les applications Next.js (anciennement test-playwright). Cette compétence doit être utilisée lors de l'écriture, de la révision ou du débogage de tests E2E avec Playwright. Se déclenche sur des tâches impliquant des sélecteurs de tests, des tests irréguliers, un état d'authentification, une simulation d'API, des tests d'hydratation, une exécution parallèle, une configuration CI ou des échecs de tests de débogage.
