Dramaturgo que prueba las mejores prácticas para aplicaciones Next.js (anteriormente test-playwright). Esta habilidad debe usarse al escribir, revisar o depurar pruebas E2E con Playwright. Activadores de tareas que involucran selectores de pruebas, pruebas inestables, estado de autenticación, simulación de API, pruebas de hidratación, ejecución paralela, configuración de CI o fallas de pruebas de depuración.
