Рекомендации по тестированию драматургов для приложений Next.js (ранее — test-playwright). Этот навык следует использовать при написании, просмотре или отладке тестов E2E с помощью Playwright. Запускается по задачам, включающим селекторы тестов, нестабильные тесты, состояние аутентификации, имитацию API, гидратационное тестирование, параллельное выполнение, конфигурацию CI или отладку сбоев тестов.
