Best Practices für Playwright-Tests für Next.js-Anwendungen (ehemals test-playwright). Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Debuggen von E2E-Tests mit Playwright verwendet werden. Wird bei Aufgaben ausgelöst, die Testselektoren, Flaky-Tests, Authentifizierungsstatus, API-Mocking, Hydratationstests, parallele Ausführung, CI-Konfiguration oder das Debuggen von Testfehlern umfassen.
