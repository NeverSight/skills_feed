同時レンダリング、サーバー コンポーネント、アクション、フック、およびメモ化に関する React 19 のパフォーマンス最適化ガイドライン (以前の React-19)。このスキルは、React 19 コンポーネントを作成するとき、同時機能を使用するとき、または再レンダリングを最適化するときに使用する必要があります。このスキルでは、App Router、next.config.js、Next.js キャッシュなどの Next.js 固有の機能はカバーされていません (nextjs-16-app-router スキルを使用してください)。 React Hook Form を使用したクライアント側のフォーム検証には、react-hook-form スキルを使用します。
