React 19 pautas de optimización del rendimiento para renderizado simultáneo, componentes del servidor, acciones, enlaces y memorización (anteriormente reaccionar-19). Esta habilidad debe usarse al escribir componentes de React 19, usar funciones concurrentes u optimizar renderizados. Esta habilidad NO cubre funciones específicas de Next.js como App Router, next.config.js o el almacenamiento en caché de Next.js (use la habilidad nextjs-16-app-router). Para la validación del formulario del lado del cliente con React Hook Form, utilice la habilidad reaccionar-hook-form.
