Linee guida per l'ottimizzazione delle prestazioni di React 19 per rendering simultaneo, componenti server, azioni, hook e memorizzazione (in precedenza react-19). Questa abilità dovrebbe essere utilizzata durante la scrittura di componenti React 19, l'utilizzo di funzionalità simultanee o l'ottimizzazione dei nuovi rendering. Questa competenza NON copre le funzionalità specifiche di Next.js come App Router, next.config.js o la memorizzazione nella cache di Next.js (usa la competenza nextjs-16-app-router). Per la convalida del modulo lato client con React Hook Form, utilizza la capacità react-hook-form.
