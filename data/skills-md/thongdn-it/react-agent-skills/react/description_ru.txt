Рекомендации по оптимизации производительности React 19 для одновременного рендеринга, серверных компонентов, действий, перехватчиков и мемоизации (ранее — React-19). Этот навык следует использовать при написании компонентов React 19, использовании параллельных функций или оптимизации повторного рендеринга. Этот навык НЕ охватывает специфичные для Next.js функции, такие как App Router, next.config.js или кеширование Next.js (используйте навык nextjs-16-app-router). Для проверки формы на стороне клиента с помощью React Hook Form используйте навык React-Hook-Form.
