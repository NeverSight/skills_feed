Directives d'optimisation des performances de React 19 pour le rendu simultané, les composants serveur, les actions, les hooks et la mémorisation (anciennement React-19). Cette compétence doit être utilisée lors de l'écriture de composants React 19, de l'utilisation de fonctionnalités simultanées ou de l'optimisation des rendus. Cette compétence ne couvre PAS les fonctionnalités spécifiques à Next.js telles que la mise en cache App Router, next.config.js ou Next.js (utilisez la compétence nextjs-16-app-router). Pour la validation de formulaire côté client avec React Hook Form, utilisez la compétence React-hook-form.
