Linee guida per l'ottimizzazione delle prestazioni del router app Next.js 16 (in precedenza nextjs-16-app-router). Questa competenza deve essere utilizzata durante la scrittura del codice Next.js 16, la configurazione della memorizzazione nella cache, l'implementazione dei componenti server in Next.js, l'impostazione del routing del router dell'app o la configurazione di next.config.js. Questa competenza NON copre i modelli React 19 generici (utilizzare la competenza react-19) o il rendering del server non Next.js.
