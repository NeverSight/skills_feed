Richtlinien zur Leistungsoptimierung des Next.js 16 App Routers (früher nextjs-16-app-router). Diese Fähigkeit sollte beim Schreiben von Next.js 16-Code, beim Konfigurieren des Cachings, beim Implementieren von Serverkomponenten in Next.js, beim Einrichten des App Router-Routings oder beim Konfigurieren von next.config.js verwendet werden. Dieser Skill deckt NICHT generische React 19-Muster (verwenden Sie den React-19-Skill) oder Nicht-Next.js-Server-Rendering ab.
