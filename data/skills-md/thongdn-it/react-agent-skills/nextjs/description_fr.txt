Directives d'optimisation des performances du routeur d'applications Next.js 16 (anciennement nextjs-16-app-router). Cette compétence doit être utilisée lors de l'écriture du code Next.js 16, de la configuration de la mise en cache, de l'implémentation des composants serveur dans Next.js, de la configuration du routage du routeur d'application ou de la configuration de next.config.js. Cette compétence ne couvre PAS les modèles génériques React 19 (utilisez la compétence React-19) ou le rendu du serveur non Next.js.
