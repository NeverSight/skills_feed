Рекомендации по оптимизации производительности маршрутизатора приложений Next.js 16 (ранее — nextjs-16-app-router). Этот навык следует использовать при написании кода Next.js 16, настройке кэширования, реализации серверных компонентов в Next.js, настройке маршрутизации App Router или настройке next.config.js. Этот навык НЕ охватывает общие шаблоны React 19 (используйте навык React-19) или серверный рендеринг, отличный от Next.js.
