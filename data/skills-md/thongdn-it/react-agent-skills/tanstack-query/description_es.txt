Optimización del rendimiento de TanStack Query v5 para recuperación de datos, almacenamiento en caché, mutaciones y patrones de consulta. Esta habilidad se debe utilizar al utilizar useQuery, useMutation, queryClient, patrones de captación previa o el almacenamiento en caché de TanStack Query. Esta habilidad NO cubre la generación de enlaces de consulta desde OpenAPI (use la habilidad orval) ni la burla de respuestas de API en las pruebas (use la habilidad test-msw).
