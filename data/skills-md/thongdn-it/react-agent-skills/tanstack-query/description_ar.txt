تحسين أداء TanStack Query v5 لجلب البيانات والتخزين المؤقت والطفرات وأنماط الاستعلام. يجب استخدام هذه المهارة عند استخدام useQuery، أو useMutation، أو queryClient، أو أنماط الجلب المسبق، أو التخزين المؤقت لاستعلام TanStack. لا تغطي هذه المهارة إنشاء روابط استعلام من OpenAPI (استخدم المهارة orval) أو الاستهزاء باستجابات واجهة برمجة التطبيقات في الاختبارات (استخدم مهارة اختبار msw).
