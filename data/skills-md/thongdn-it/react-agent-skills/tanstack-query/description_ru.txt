Оптимизация производительности TanStack Query v5 для выборки данных, кэширования, мутаций и шаблонов запросов. Этот навык следует использовать при использовании useQuery, useMutation, queryClient, шаблонов предварительной выборки или кэширования запросов TanStack. Этот навык НЕ охватывает создание перехватчиков запросов из OpenAPI (используйте навык orval) или имитацию ответов API в тестах (используйте навык test-msw).
