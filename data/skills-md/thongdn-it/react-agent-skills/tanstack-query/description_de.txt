TanStack Query v5 Leistungsoptimierung für Datenabruf, Caching, Mutationen und Abfragemuster. Diese Fähigkeit sollte verwendet werden, wenn useQuery, useMutation, queryClient, Prefetch-Muster oder TanStack Query-Caching verwendet werden. Dieser Skill deckt NICHT das Generieren von Abfrage-Hooks aus OpenAPI (Orval-Skill verwenden) oder das Verspotten von API-Antworten in Tests (Test-MSW-Skill verwenden) ab.
