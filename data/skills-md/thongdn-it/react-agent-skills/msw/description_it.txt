Best practice MSW (Mock Service Worker) per il mocking delle API nei test (in precedenza test-msw). Questa competenza dovrebbe essere utilizzata durante la configurazione di MSW, la scrittura di gestori di richieste o la simulazione di API HTTP. Questa competenza NON copre i modelli generali di test (utilizzare le competenze test-vitest o test-tdd) o la metodologia di test.
