Meilleures pratiques MSW (Mock Service Worker) pour la simulation d'API dans les tests (anciennement test-msw). Cette compétence doit être utilisée lors de la configuration de MSW, de l'écriture de gestionnaires de requêtes ou de la simulation d'API HTTP. Cette compétence ne couvre PAS les modèles de test généraux (utiliser les compétences test-vitest ou test-tdd) ou la méthodologie de test.
