Рекомендации MSW (Mock Service Worker) для имитации API в тестах (ранее — test-msw). Этот навык следует использовать при настройке MSW, написании обработчиков запросов или имитации HTTP API. Этот навык НЕ охватывает общие схемы тестирования (использование навыков test-vitest или test-tdd) или методологию тестирования.
