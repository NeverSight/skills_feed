Mejores prácticas de MSW (Mock Service Worker) para burlarse de API en pruebas (anteriormente test-msw). Esta habilidad debe usarse al configurar MSW, escribir controladores de solicitudes o burlarse de las API HTTP. Esta habilidad NO cubre patrones de prueba generales (use las habilidades test-vitest o test-tdd) ni la metodología de la prueba.
