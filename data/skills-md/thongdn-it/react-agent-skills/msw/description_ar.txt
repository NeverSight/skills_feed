أفضل ممارسات MSW (Mock Service Worker) للسخرية من واجهة برمجة التطبيقات (API) في الاختبارات (المعروفة سابقًا باسم test-msw). يجب استخدام هذه المهارة عند إعداد MSW، أو كتابة معالجات الطلبات، أو الاستهزاء بواجهات برمجة تطبيقات HTTP. لا تغطي هذه المهارة أنماط الاختبار العامة (استخدم مهارات test-vitest أو test-tdd) أو منهجية الاختبار.
