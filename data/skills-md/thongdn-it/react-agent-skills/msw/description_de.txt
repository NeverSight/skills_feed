MSW (Mock Service Worker) Best Practices für API-Mocking in Tests (ehemals test-msw). Diese Fähigkeit sollte beim Einrichten von MSW, beim Schreiben von Anforderungshandlern oder beim Verspotten von HTTP-APIs verwendet werden. Diese Fertigkeit deckt NICHT allgemeine Testmuster (verwenden Sie Test-Vitest- oder Test-TDD-Fähigkeiten) oder Testmethodik ab.
