Pautas de mejores prácticas de diseño de UI/UX y frontend (anteriormente frontend-design). Esta habilidad debe utilizarse al escribir, revisar o diseñar código de interfaz para garantizar la accesibilidad, el rendimiento y la usabilidad. Activadores de tareas que involucran estructura HTML, estilo CSS, diseños responsivos, diseño de formularios, animaciones o mejoras de accesibilidad.
