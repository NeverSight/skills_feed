Linee guida sulle migliori pratiche per UI/UX e progettazione frontend (in precedenza frontend-design). Questa competenza dovrebbe essere utilizzata durante la scrittura, la revisione o la progettazione del codice frontend per garantire accessibilità, prestazioni e usabilità. Attiva attività che coinvolgono struttura HTML, stile CSS, layout reattivi, progettazione di moduli, animazioni o miglioramenti dell'accessibilità.
