Lignes directrices sur les meilleures pratiques en matière de conception UI/UX et frontend (anciennement frontend-design). Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la conception de code frontend pour garantir l'accessibilité, les performances et la convivialité. Déclencheurs sur des tâches impliquant la structure HTML, le style CSS, les mises en page réactives, la conception de formulaires, les animations ou les améliorations d'accessibilité.
