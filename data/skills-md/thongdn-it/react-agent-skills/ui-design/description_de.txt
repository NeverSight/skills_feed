Best Practices-Richtlinien für UI/UX und Frontend-Design (ehemals Frontend-Design). Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Entwerfen von Frontend-Code verwendet werden, um Zugänglichkeit, Leistung und Benutzerfreundlichkeit sicherzustellen. Wird bei Aufgaben ausgelöst, die HTML-Struktur, CSS-Stil, responsive Layouts, Formulardesign, Animationen oder Verbesserungen der Barrierefreiheit betreffen.
