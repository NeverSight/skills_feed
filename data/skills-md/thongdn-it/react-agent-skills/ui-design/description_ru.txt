Рекомендации по передовому опыту в области UI/UX и дизайна интерфейса (ранее — дизайн интерфейса). Этот навык следует использовать при написании, проверке или разработке кода внешнего интерфейса, чтобы обеспечить доступность, производительность и удобство использования. Запускается при выполнении задач, связанных со структурой HTML, стилем CSS, адаптивными макетами, дизайном форм, анимацией или улучшениями специальных возможностей.
