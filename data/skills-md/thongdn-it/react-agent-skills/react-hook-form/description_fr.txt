Optimisation des performances de React Hook Form pour la validation de formulaire côté client à l'aide de useForm, useWatch, useController et useFieldArray. Cette compétence doit être utilisée lors de la création de formulaires contrôlés côté client avec la bibliothèque React Hook Form. Cette compétence ne couvre PAS les actions du serveur React 19, useActionState ou la gestion des formulaires côté serveur (utilisez la compétence React-19 pour ceux-ci).
