Optimización del rendimiento de React Hook Form para la validación de formularios del lado del cliente mediante useForm, useWatch, useController y useFieldArray. Esta habilidad debe usarse al crear formularios controlados del lado del cliente con la biblioteca React Hook Form. Esta habilidad NO cubre las acciones del servidor React 19, useActionState o el manejo de formularios del lado del servidor (use la habilidad reaccionar-19 para eso).
