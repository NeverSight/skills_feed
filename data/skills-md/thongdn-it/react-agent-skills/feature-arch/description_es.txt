Reaccionar pautas de arquitectura basada en características para aplicaciones escalables (anteriormente arquitectura de características). Esta habilidad debe usarse al escribir, revisar o refactorizar el código de React para garantizar la organización adecuada de las funciones. Activadores de tareas que involucran la estructura del proyecto, la organización de funciones, los límites de los módulos, las importaciones de funciones cruzadas, los patrones de obtención de datos o la composición de componentes.
