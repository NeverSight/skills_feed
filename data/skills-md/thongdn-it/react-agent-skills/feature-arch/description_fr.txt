Directives d'architecture basées sur les fonctionnalités de React pour les applications évolutives (anciennement architecture de fonctionnalités). Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la refactorisation du code React pour garantir une bonne organisation des fonctionnalités. Se déclenche sur des tâches impliquant la structure du projet, l'organisation des fonctionnalités, les limites des modules, les importations inter-fonctionnalités, les modèles de récupération de données ou la composition des composants.
