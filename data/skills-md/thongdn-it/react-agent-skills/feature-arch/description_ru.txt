Рекомендации по архитектуре React на основе функций для масштабируемых приложений (ранее — архитектура функций). Этот навык следует использовать при написании, проверке или рефакторинге кода React, чтобы обеспечить правильную организацию функций. Запускается по задачам, связанным со структурой проекта, организацией функций, границами модулей, импортом перекрестных функций, шаблонами выборки данных или составом компонентов.
