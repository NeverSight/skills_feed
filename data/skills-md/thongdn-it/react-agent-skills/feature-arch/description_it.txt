Linee guida sull'architettura basata su funzionalità di React per applicazioni scalabili (in precedenza architettura di funzionalità). Questa competenza dovrebbe essere utilizzata durante la scrittura, la revisione o il refactoring del codice React per garantire la corretta organizzazione delle funzionalità. Trigger su attività che coinvolgono la struttura del progetto, l'organizzazione delle funzionalità, i limiti dei moduli, le importazioni di funzionalità incrociate, i modelli di recupero dei dati o la composizione dei componenti.
