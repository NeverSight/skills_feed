Mejores prácticas de validación de esquemas de Zod para seguridad de tipos, análisis y manejo de errores. Esta habilidad debe usarse al definir esquemas z.object, usando validaciones z.string, safeParse o z.infer. Esta habilidad NO cubre los patrones de integración de React Hook Form (use la habilidad reaccionar-hook-form) ni la generación de clientes OpenAPI (use la habilidad orval).
