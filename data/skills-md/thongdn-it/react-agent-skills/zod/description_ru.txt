Лучшие практики проверки схемы Zod для обеспечения безопасности типов, синтаксического анализа и обработки ошибок. Этот навык следует использовать при определении схем z.object с использованием проверок z.string, SafeParse или z.infer. Этот навык НЕ охватывает шаблоны интеграции React Hook Form (используйте навык React Hook-Form) или генерацию клиента OpenAPI (используйте навык orval).
