أفضل ممارسات التحقق من صحة مخطط Zod لسلامة النوع والتحليل ومعالجة الأخطاء. يجب استخدام هذه المهارة عند تحديد مخططات z.object، باستخدام عمليات التحقق من صحة z.string، أو SafeParse، أو z.infer. لا تغطي هذه المهارة أنماط تكامل React Hook Form (استخدم مهارة نموذج الخطاف التفاعلي) أو إنشاء عميل OpenAPI (استخدم المهارة Orval).
