Best practice per la convalida dello schema Zod per l'indipendenza dai tipi, l'analisi e la gestione degli errori. Questa competenza dovrebbe essere utilizzata quando si definiscono schemi z.object, utilizzando convalide z.string, safeParse o z.infer. Questa competenza NON copre i modelli di integrazione React Hook Form (utilizzare la competenza react-hook-form) o la generazione di client OpenAPI (utilizzare la competenza orval).
