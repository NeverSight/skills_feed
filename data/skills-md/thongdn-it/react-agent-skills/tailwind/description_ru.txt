Рекомендации по оптимизации производительности Tailwind CSS v4 и передовому опыту (ранее — стиль Tailwindcss-v4). Этот навык следует использовать при написании, проверке или рефакторинге кода Tailwind CSS v4, чтобы обеспечить оптимальную производительность сборки, минимальный вывод CSS и правильное использование функций v4. Запускается при выполнении задач, связанных с конфигурацией Tailwind, директивой @theme, служебными классами, адаптивным дизайном, темным режимом, запросами контейнера или оптимизацией генерации CSS.
