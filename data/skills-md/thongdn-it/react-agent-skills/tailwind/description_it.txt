Ottimizzazione delle prestazioni di Tailwind CSS v4 e linee guida sulle best practice (in precedenza stile tailwindcss-v4). Questa competenza deve essere utilizzata durante la scrittura, la revisione o il refactoring del codice Tailwind CSS v4 per garantire prestazioni di compilazione ottimali, output CSS minimo e utilizzo corretto delle funzionalità v4. Trigger su attività che coinvolgono la configurazione di Tailwind, la direttiva @theme, le classi di utilità, la progettazione reattiva, la modalità oscura, le query sui contenitori o l'ottimizzazione della generazione di CSS.
