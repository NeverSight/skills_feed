Pautas de mejores prácticas y optimización del rendimiento de Tailwind CSS v4 (anteriormente estilo tailwindcss-v4). Esta habilidad debe usarse al escribir, revisar o refactorizar el código Tailwind CSS v4 para garantizar un rendimiento de compilación óptimo, una salida CSS mínima y un uso correcto de las funciones v4. Activadores de tareas que involucran la configuración de Tailwind, directiva @theme, clases de utilidad, diseño responsivo, modo oscuro, consultas de contenedores u optimización de generación de CSS.
