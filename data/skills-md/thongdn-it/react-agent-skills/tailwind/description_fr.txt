Lignes directrices sur l’optimisation des performances et les meilleures pratiques Tailwind CSS v4 (anciennement tailwindcss-v4-style). Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la refactorisation du code Tailwind CSS v4 pour garantir des performances de construction optimales, une sortie CSS minimale et une utilisation correcte des fonctionnalités v4. Déclencheurs sur des tâches impliquant la configuration Tailwind, la directive @theme, les classes utilitaires, la conception réactive, le mode sombre, les requêtes de conteneur ou l'optimisation de la génération CSS.
