Richtlinien zur Leistungsoptimierung und Best Practices für Tailwind CSS v4 (früher tailwindcss-v4-style). Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von Tailwind CSS v4-Code verwendet werden, um optimale Build-Leistung, minimale CSS-Ausgabe und korrekte Verwendung von v4-Funktionen sicherzustellen. Wird bei Aufgaben ausgelöst, die Tailwind-Konfiguration, @theme-Direktive, Dienstprogrammklassen, responsives Design, Dark Mode, Containerabfragen oder Optimierung der CSS-Generierung umfassen.
