Best Practices und Muster der shadcn/ui-Komponentenbibliothek (ehemals shadcn-ui). Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von shadcn/ui-Komponenten verwendet werden, um die richtige Architektur, Zugänglichkeit und Leistung sicherzustellen. Wird bei Aufgaben ausgelöst, die Radix-Primitive, Tailwind-Styling, Formularvalidierung mit React Hook Form, Datentabellen, Theming oder Komponentenzusammensetzungsmuster umfassen.
