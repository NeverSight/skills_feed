Mejores prácticas y patrones de la biblioteca de componentes shadcn/ui (anteriormente shadcn-ui). Esta habilidad debe usarse al escribir, revisar o refactorizar componentes shadcn/ui para garantizar una arquitectura, accesibilidad y rendimiento adecuados. Activadores de tareas que involucran primitivas de Radix, estilo Tailwind, validación de formularios con React Hook Form, tablas de datos, temas o patrones de composición de componentes.
