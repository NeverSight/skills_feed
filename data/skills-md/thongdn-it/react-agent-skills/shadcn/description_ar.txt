أفضل الممارسات والأنماط لمكتبة مكون shadcn/ui (shadcn-ui سابقًا). يجب استخدام هذه المهارة عند كتابة مكونات shadcn/ui أو مراجعتها أو إعادة هيكلتها لضمان التصميم المناسب وإمكانية الوصول والأداء. يتم تشغيل المهام التي تتضمن عناصر Radix الأولية، أو تصميم Tailwind، أو التحقق من صحة النموذج باستخدام React Hook Form، أو جداول البيانات، أو السمات، أو أنماط تكوين المكونات.
