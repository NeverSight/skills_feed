Лучшие практики и шаблоны библиотеки компонентов shadcn/ui (ранее shadcn-ui). Этот навык следует использовать при написании, проверке или рефакторинге компонентов shadcn/ui, чтобы обеспечить правильную архитектуру, доступность и производительность. Запускается по задачам, включающим примитивы Radix, стили Tailwind, проверку формы с помощью React Hook Form, таблицы данных, шаблоны тем или композиции компонентов.
