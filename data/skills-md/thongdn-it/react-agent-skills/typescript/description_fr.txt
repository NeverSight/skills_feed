Cette compétence doit être utilisée lorsque l'utilisateur demande « d'optimiser les performances de TypeScript », « d'accélérer la compilation tsc », de « configurer tsconfig.json », de « corriger les erreurs de type », « d'améliorer les modèles asynchrones » ou rencontre des erreurs TS (TS2322, TS2339, « n'est pas attribuable à »). Déclenche également le travail sur les fichiers .ts, .tsx, .d.ts impliquant des définitions de types, l'organisation des modules ou la gestion de la mémoire. Ne couvre PAS les bases de TypeScript, les modèles spécifiques au framework ou les tests.
