Этот навык следует использовать, когда пользователь просит «оптимизировать производительность TypeScript», «ускорить компиляцию tsc», «настроить tsconfig.json», «исправить ошибки типа», «улучшить асинхронные шаблоны» или сталкивается с ошибками TS (TS2322, TS2339, «не назначается»). Также триггеры срабатывают при работе с файлами .ts, .tsx, .d.ts, включая определения типов, организацию модулей или управление памятью. НЕ охватывает основы TypeScript, шаблоны, специфичные для платформы, или тестирование.
