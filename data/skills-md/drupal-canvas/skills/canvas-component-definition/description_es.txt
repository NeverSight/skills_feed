Comience aquí para cualquier tarea del componente React para hacer cumplir el contrato canónico del componente Canvas. Úselo para crear, modificar, refactorizar, revisar, migrar o validar el trabajo. Establece el contrato canónico del componente Canvas, asumiendo que los componentes del repositorio son objetivos de Canvas, y guía ya sea (1) transformando los componentes existentes para cumplir con los requisitos de Canvas o (2) creando nuevos componentes listos para Canvas.
