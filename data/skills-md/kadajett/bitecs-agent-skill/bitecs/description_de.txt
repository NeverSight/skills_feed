Umfassende Anleitung zur Verwendung von bitECS v0.4.0, einer flexiblen, minimalen, datenorientierten ECS-Bibliothek für TypeScript. Verwenden Sie diese Option, wenn Sie Spiele oder Simulationen mit der Entity Component System-Architektur erstellen, Entitätshierarchien erstellen, Komponenten definieren, Entitäten abfragen oder mit Serialisierung arbeiten.
