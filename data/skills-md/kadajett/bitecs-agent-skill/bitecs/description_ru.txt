Подробное руководство по использованию bitECS v0.4.0, гибкой, минимальной, ориентированной на данные библиотеки ECS для TypeScript. Используйте при создании игр или симуляций с архитектурой Entity Component System, создании иерархий сущностей, определении компонентов, запросе сущностей или работе с сериализацией.
