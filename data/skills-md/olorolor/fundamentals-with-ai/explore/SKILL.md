# Explore Skill

이 스킬은 `/explore <topic>` 요청 시 사용한다.

## 목표

- 특정 키워드의 핵심을 짧고 정확하게 설명
- 관련 개념, 전제 지식, 다음 학습 모듈을 연결
- 필요 시 학습 모드로 전환

## 출력 형식

- 기본 응답은 5문장 이내
- 아래 4블록 형식을 유지한다.
  1. 한 줄 정의
  2. 왜 중요한지 (학습자의 직무 맥락에 맞는 비유 1개)
  3. 자주 헷갈리는 개념 1개
  4. 학습자에게 던지는 질문 (사고를 자극하는 짧은 질문으로 끝낸다)

## 탐색 카테고리

- 모델/제품
- 도구/프레임워크
- 방법론(RAG, Fine-tuning, CoT 등)
- 인프라/배포
- Agent 생태계(MCP, Tool Use)
- 조직/전략 키워드

## 모드 전환 신호

아래 신호가 감지되면 학습 모드 전환을 제안한다.

### 명시적 신호

- "원리까지 자세히"
- "실무에 적용하고 싶다"
- "기초부터 차근차근"

### 암시적 신호

- 같은 개념에 대해 2회 이상 추가 질문을 하고 있음
- "왜?"를 반복적으로 물어보고 있음
- 구체적 사용 방법이나 구현 수준의 질문을 하고 있음
