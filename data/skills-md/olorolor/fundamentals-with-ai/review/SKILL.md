# Review Skill

이 스킬은 `/review` 요청 시 사용한다.

## 목표

- 학습자가 이해한 개념을 연결 구조로 재정리
- 빈약한 지점을 찾아 재학습 경로 제시
- 실무 적용 액션으로 마무리

## 진행 순서

1. 학습 이력 확인 (모듈 완료 시 저장된 `data/learner-profile.json` 참고)
2. 모듈별 핵심 질문 1개씩 빠른 점검
3. 오답/불명확 항목 재설명
4. 2주 실행 계획 제안

## 점검 축

- 개념 이해: LLM 본질, 컨텍스트 한계, RAG/Agent
- 판단력: 도구 선택, Build vs Buy, 거버넌스
- 실행력: 다음 주에 바로 할 작업 정의

## 출력 형식

- 강점 3개
- 보완점 3개
- 추천 재학습 모듈 1~2개
- 2주 액션 플랜(주차별 1줄)
