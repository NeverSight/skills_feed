Scanner complet de vulnérabilités de sécurité pour les projets Python, notamment les applications Flask, Django et FastAPI. Détecte les 10 principales vulnérabilités OWASP, les failles d'injection, la désérialisation non sécurisée, les problèmes d'authentification, les secrets codés en dur et les problèmes de sécurité spécifiques au framework. Audite les dépendances pour les CVE connus et génère des rapports de sécurité exploitables.
