Scanner completo delle vulnerabilità della sicurezza per progetti Python tra cui applicazioni Flask, Django e FastAPI. Rileva le 10 principali vulnerabilità OWASP, difetti di injection, deserializzazione non sicura, problemi di autenticazione, segreti codificati e problemi di sicurezza specifici del framework. Controlla le dipendenze per CVE noti e genera report di sicurezza utilizzabili.
