Комплексный сканер уязвимостей безопасности для проектов Python, включая приложения Flask, Django и FastAPI. Обнаруживает 10 наиболее уязвимых мест OWASP, ошибки внедрения, небезопасную десериализацию, проблемы аутентификации, жестко закодированные секреты и проблемы безопасности, специфичные для платформы. Аудит зависимостей на предмет известных CVE и создает действенные отчеты о безопасности.
