Публикуйте статьи Markdown в Substack как черновики. Используйте, когда пользователь хочет опубликовать файл Markdown в Substack или упоминает «发布到 Substack», «статья Substack», «опубликовать в Substack». Обрабатывает преобразование Markdown в HTML и сохраняет как черновик (никогда не публикует автоматически).
