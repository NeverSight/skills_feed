Преобразуйте диаграммы Mermaid и таблицы Markdown в изображения (PNG/SVG) для платформ, которые не поддерживают расширенное форматирование. Используйте, когда пользователь просит «конвертировать в изображение», «экспортировать в формате PNG», «преобразовать это в изображение» или если у него есть контент для X/Twitter, требующий визуального экспорта.
