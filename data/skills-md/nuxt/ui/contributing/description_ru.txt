Руководство по работе с пользовательским интерфейсом Nuxt. Предоставляет шаблоны структуры компонентов, темы Tailwind Variants, соглашения о тестировании Vitest и рекомендации по документации MDC. Используйте при создании новых компонентов, просмотре PR компонентов, изменении существующих компонентов, написании тестов или создании документации в этой базе кода.
