Extraiga automáticamente los repositorios de GitHub del contenido (artículos, capturas de pantalla, texto) y destaquelos. Casos de uso: (1) los usuarios solicitan analizar contenido y destacar proyectos de GitHub relacionados, (2) procesar artículos/blogs que mencionan herramientas o bibliotecas, (3) analizar capturas de pantalla que contienen referencias técnicas, (4) configurar tareas programadas para descubrir y destacar repositorios automáticamente.
