قم باستخراج مستودعات GitHub تلقائيًا من المحتوى (المقالات ولقطات الشاشة والنصوص) وتمييزها بنجمة. حالات الاستخدام: (1) يطلب المستخدمون تحليل المحتوى ومشاريع GitHub ذات الصلة بنجمة، (2) معالجة المقالات/المدونات التي تشير إلى الأدوات أو المكتبات، (3) تحليل لقطات الشاشة التي تحتوي على مراجع فنية، (4) إعداد المهام المجدولة لاكتشاف المستودعات وتمييزها تلقائيًا بنجمة.
