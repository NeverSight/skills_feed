Verwalten Sie die Installation, Synchronisierung, Deinstallation und Auflistung von Claude Code Skills und Commands und unterstützen Sie die Installation über lokale Pfade oder GitHub-Repositorys/Unterverzeichnisse. Dieser Skill sollte verwendet werden, wenn der Benutzer externe Skills/Befehle installieren, aus einem GitHub-Repository synchronisieren, lokale Verzeichnisse stapelweise installieren, installierte Elemente anzeigen oder unerwünschte Elemente deinstallieren muss.
