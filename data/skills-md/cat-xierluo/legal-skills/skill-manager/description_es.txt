Administre la instalación, sincronización, desinstalación y listado de habilidades y comandos de Claude Code, admitiendo la instalación desde rutas locales o repositorios/subdirectorios de GitHub. Esta habilidad debe usarse cuando el usuario necesita instalar habilidades/comandos externos, sincronizar desde un repositorio de GitHub, instalar directorios locales por lotes, ver elementos instalados o desinstalar elementos no deseados.
