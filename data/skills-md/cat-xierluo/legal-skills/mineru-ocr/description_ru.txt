Преобразуйте PDF-документы в формат Markdown с помощью API MinerU, который поддерживает распознавание оптического текста, таблиц и формул. Этот навык следует использовать, когда пользователю необходимо преобразовать документы PDF, Word, PPT или изображения в Markdown или упоминает PDF в Markdown, OCR, распознавание текста, распознавание таблиц, распознавание формул, преобразование документов, преобразование изображений в текст или преобразование отсканированных файлов.
