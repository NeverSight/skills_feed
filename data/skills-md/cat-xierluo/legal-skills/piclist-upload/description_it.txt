Utilizza il server HTTP PicList per caricare immagini locali nei file Markdown su un servizio di hosting di immagini e sostituirle con collegamenti cloud. Questa competenza deve essere utilizzata quando l'utente deve caricare immagini in Markdown, elaborare Markdown contenente riferimenti a immagini locali, elaborare in batch più file o directory Markdown o sostituire percorsi locali con collegamenti cloud per l'accesso tra dispositivi.
