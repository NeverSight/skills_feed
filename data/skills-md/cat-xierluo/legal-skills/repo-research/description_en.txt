GitHub repository deep research and integration analysis tool. Supports single/multiple repository studies, comparison with local projects, and heuristic integration suggestions. Offers topic-driven search mode: automatically searches for related repositories, clones, analyzes, and generates reports. Clones remote repositories to the local research/ directory, performs deep code analysis, architecture evaluation, dependency resolution, and generates structured research reports. Trigger conditions: when user provides GitHub URL requesting research/analysis/integration/comparison, or provides topic keywords to search and study related repositories.
