GitHub リポジトリの詳細な調査および統合分析ツール。単一/複数のリポジトリの調査、ローカル プロジェクトとの比較、ヒューリスティックな統合の提案をサポートします。トピック主導の検索モードを提供します。関連するリポジトリ、クローン、分析を自動的に検索し、レポートを生成します。リモート リポジトリのクローンをローカルの Research/ ディレクトリに作成し、詳細なコード分析、アーキテクチャの評価、依存関係の解決を実行し、構造化された調査レポートを生成します。トリガー条件: ユーザーが調査/分析/統合/比較をリクエストする GitHub URL を提供したとき、または関連リポジトリを検索および調査するためのトピック キーワードを提供したとき。
