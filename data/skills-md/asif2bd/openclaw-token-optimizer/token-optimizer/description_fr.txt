Réduisez l'utilisation des jetons OpenClaw et les coûts des API grâce au routage intelligent des modèles, à l'optimisation du rythme cardiaque, au suivi du budget et aux fonctionnalités natives 2026.2.15 (élagage de session, limites de taille d'amorçage, alignement TTL du cache). À utiliser lorsque les coûts des jetons sont élevés, que les limites de débit des API sont atteintes ou que vous hébergez plusieurs agents à grande échelle. Les 4 scripts exécutables (context_optimizer, model_router, heartbeat_optimizer, token_tracker) sont uniquement locaux : aucune requête réseau, aucun appel de sous-processus, aucune modification du système. Les fichiers de référence (PROVIDERS.md, config-patches.json) documentent les stratégies multi-fournisseurs facultatives qui nécessitent des clés API externes et un accès au réseau si vous choisissez de les utiliser. Voir SECURITY.md pour une description complète.
