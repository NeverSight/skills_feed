スマート モデル ルーティング、ハートビートの最適化、予算追跡、および 2026.2.15 のネイティブ機能 (セッション プルーニング、ブートストラップ サイズ制限、キャッシュ TTL アライメント) を通じて、OpenClaw トークンの使用量と API コストを削減します。トークンのコストが高い場合、API レート制限に達している場合、または複数のエージェントを大規模にホストしている場合に使用します。 4 つの実行可能スクリプト (context_optimizer、model_router、heartbeat_optimizer、token_tracker) はローカルのみです。ネットワーク リクエスト、サブプロセス呼び出し、システム変更はありません。参照ファイル (PROVIDERS.md、config-patches.json) には、外部 API キーとネットワーク アクセスを使用する場合に必要な、オプションのマルチプロバイダー戦略が文書化されています。詳細については、SECURITY.md を参照してください。
