يمكنك تقليل استخدام رمز OpenClaw وتكاليف واجهة برمجة التطبيقات (API) من خلال توجيه النموذج الذكي، وتحسين نبضات القلب، وتتبع الميزانية، وميزات 2026.2.15 الأصلية (تقليم الجلسة، وحدود حجم التمهيد، ومحاذاة TTL لذاكرة التخزين المؤقت). يُستخدم عندما تكون تكاليف الرمز المميز مرتفعة، أو يتم تجاوز حدود معدل واجهة برمجة التطبيقات (API)، أو استضافة وكلاء متعددين على نطاق واسع. البرامج النصية الأربعة القابلة للتنفيذ (context_optimizer، وmodel_router، وheartbeat_optimizer، وtoken_tracker) هي محلية فقط - لا توجد طلبات شبكة، ولا استدعاءات للعمليات الفرعية، ولا تعديلات على النظام. توثق الملفات المرجعية (PROVIDERS.md، config-patches.json) إستراتيجيات اختيارية متعددة الموفرين تتطلب مفاتيح API خارجية ووصولاً إلى الشبكة إذا اخترت استخدامها. راجع SECURITY.md للاطلاع على التفاصيل الكاملة.
