Запросите отслеживание ошибок Rollbar, чтобы исследовать ошибки, перечислять недавние проблемы, устранять элементы и получать контекст для исправления ошибок. Используйте, когда пользователь спрашивает о производственных ошибках, хочет исследовать ошибку, спрашивает «какие ошибки произошли за ночь», хочет пометить проблемы как решенные или нуждается в контексте для исправления ошибки с помощью Rollbar.
