تحليل البيانات والتصور ومهارة سرد القصص للسياقات المالية وسياقات RevOps. يُستخدم عندما: تحليل بيانات الإيرادات، أو إنشاء التنبؤات، أو التحليل الجماعي، أو نمذجة التغيير، أو تحليلات خطوط الأنابيب، أو إنشاء تقارير تعتمد على البيانات، أو إنشاء لوحات المعلومات، أو تنظيف البيانات الفوضوية، أو التحقق من سلامة المطالبات التحليلية، أو التصدير إلى Excel باستخدام الصيغ، أو استخراج البيانات من ملفات PDF. يتميز بتسجيل القرار، والتفسير المدرك للتحيز، والكشف التدريجي (مجموعة الشرائح -> تقرير مفصل -> دفتر ملاحظات كامل مع توثيق جميع القرارات).
