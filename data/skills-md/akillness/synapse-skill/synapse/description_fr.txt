Système d'orchestration d'agents multi-IA avec des modèles configurables et des flux de travail basés sur les rôles. À utiliser lorsque vous devez coordonner plusieurs agents IA (Claude, Gemini, Codex) pour des tâches complexes telles que la planification, la génération de code, l'analyse, la révision et l'exécution. Prend en charge les modèles de flux de travail agents : spécialistes parallèles, pipeline et orchestration en essaim. Compatible avec Claude Code, Cursor et OpenCode. Déclencheurs : « orchestrer les agents », « workflow multi-agents », « planifier et exécuter », « pipeline de révision de code », « exécuter synapse », « workflow agent ».
