دليل الخبراء لتخزين Supabase بما في ذلك إدارة المجموعة وعمليات الملفات وإنشاء عناوين URL وسياسات RLS. يُستخدم عند العمل مع تحميلات/تنزيلات الملفات، أو إنشاء مجموعات عامة أو خاصة، أو إنشاء عناوين URL موقعة، أو تنفيذ سياسات RLS للتخزين، أو التعامل مع التحميلات القابلة للاستئناف، أو تحويلات الصور، أو أي مهام متعلقة بتخزين Supabase.
