Esperto di sviluppo Supabase completo che copre Edge Functions, gestione dello schema di database, migrazioni, funzioni PostgreSQL e policy RLS. Utilizzare per qualsiasi sviluppo Supabase, tra cui TypeScript/Deno Edge Functions, gestione di schemi dichiarativi, formattazione SQL, creazione di migrazioni, creazione di funzioni di database con SECURITY INVOKER e implementazione di policy RLS con auth.uid() e auth.jwt().
