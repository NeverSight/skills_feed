خبير تطوير Supabase الشامل الذي يغطي وظائف Edge وإدارة مخطط قاعدة البيانات وعمليات الترحيل ووظائف PostgreSQL وسياسات RLS. يُستخدم لأي تطوير Supabase بما في ذلك وظائف TypeScript/Deno Edge، وإدارة المخطط التعريفي، وتنسيق SQL، وإنشاء الترحيل، وتأليف وظيفة قاعدة البيانات باستخدام SECURITY INVOKER، وتنفيذ سياسة RLS باستخدام auth.uid() وauth.jwt().
