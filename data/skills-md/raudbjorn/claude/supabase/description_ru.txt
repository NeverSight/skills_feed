Комплексный эксперт по разработке Supabase, охватывающий пограничные функции, управление схемой базы данных, миграцию, функции PostgreSQL и политики RLS. Используйте для любой разработки Supabase, включая функции TypeScript/Deno Edge, управление декларативной схемой, форматирование SQL, создание миграции, создание функций базы данных с помощью SECURITY INVOKER и реализацию политики RLS с помощью auth.uid() и auth.jwt().
