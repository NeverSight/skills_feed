Best Practices für die Entwicklung, Konfiguration und den Betrieb von Azure App Service-Web-Apps. Wird zum Überprüfen/Optimieren von App Service-Konfigurationen, zum Implementieren von Sicherheitsmustern (verwaltete Identität, Key Vault), zum Optimieren der Leistung (Kaltstarts, Skalierung), zum Einrichten von Produktionsbereitstellungen (Slots, CI/CD, Integritätsprüfungen), zur Kostenoptimierung oder zur Fehlerbehebung verwendet. Löst auf „Best Practices“, „Empfehlungen“, „Muster“ und „Wie soll ich konfigurieren“ aus.
