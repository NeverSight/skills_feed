Cette compétence doit être utilisée lorsque l'utilisateur souhaite que Claude Code agisse uniquement en tant que gestionnaire/architecte pendant que Gemini CLI effectue tout le travail de codage. Claude Code conduit Gemini comme un stagiaire - en émettant des tâches, en examinant les résultats, en demandant des correctifs - mais n'écrit jamais de code lui-même. À utiliser lorsque l'utilisateur dit « gérer Gemini », « mode architecte », « conduire Gemini » ou souhaite déléguer toute la mise en œuvre à Gemini.
