Utilice esta habilidad cuando administre extensiones OpenCode con OCX (OpenCode eXtensions). Esto incluye instalar componentes de registros, usar el modo fantasma para el desarrollo entre repositorios, auditar cambios con verificación SHA-256, administrar dependencias, configurar registros o realizar actualizaciones de componentes y administración de versiones. Invocar para tareas que involucran operaciones de inicio de ocx, adición de ocx, actualización de ocx, diferenciación de ocx, fantasma de ocx u operaciones de registro de OCX.
