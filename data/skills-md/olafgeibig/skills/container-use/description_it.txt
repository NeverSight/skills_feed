Utilizza questa competenza quando lavori con contenitori Apple (macchine virtuali Linux leggere) come sostituto nativo di Docker su macOS. Ciò include la creazione di immagini di contenitori, l'esecuzione di contenitori, la gestione del ciclo di vita dei contenitori, la configurazione della rete, la gestione dei volumi, il montaggio di file con l'inoltro SSH o l'esecuzione di build multipiattaforma. Richiama per attività che coinvolgono la CLI del contenitore, lo strumento contenitore Apple o i contenitori Linux su Apple Silicon macOS 26+.
