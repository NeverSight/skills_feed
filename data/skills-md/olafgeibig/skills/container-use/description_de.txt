Nutzen Sie diese Fähigkeit, wenn Sie mit Apple Containern (leichtgewichtige Linux-VMs) als nativen Docker-Ersatz unter macOS arbeiten. Dazu gehören das Erstellen von Container-Images, das Ausführen von Containern, das Verwalten des Container-Lebenszyklus, das Konfigurieren von Netzwerken, die Handhabung von Volumes, das Mounten von Dateien mit SSH-Weiterleitung oder das Durchführen von Builds für mehrere Plattformen. Aufrufen für Aufgaben mit der Container-CLI, dem Apple-Container-Tool oder Linux-Containern auf Apple Silicon macOS 26+.
