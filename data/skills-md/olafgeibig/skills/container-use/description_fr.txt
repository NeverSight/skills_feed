Utilisez cette compétence lorsque vous travaillez avec des conteneurs Apple (machines virtuelles Linux légères) en remplacement de Docker natif sur macOS. Cela inclut la création d'images de conteneurs, l'exécution de conteneurs, la gestion du cycle de vie des conteneurs, la configuration du réseau, la gestion des volumes, le montage de fichiers avec le transfert SSH ou l'exécution de builds multiplateformes. Appelez pour des tâches impliquant la CLI du conteneur, l’outil de conteneur Apple ou les conteneurs Linux sur Apple Silicon macOS 26+.
