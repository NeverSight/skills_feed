Используйте этот навык при работе с контейнерами Apple (облегченными виртуальными машинами Linux) в качестве встроенной замены Docker в macOS. Сюда входит создание образов контейнеров, запуск контейнеров, управление жизненным циклом контейнеров, настройка сети, обработка томов, монтирование файлов с перенаправлением SSH или выполнение многоплатформенных сборок. Вызов для задач, связанных с контейнерным CLI, контейнерным инструментом Apple или контейнерами Linux в Apple Silicon macOS 26+.
