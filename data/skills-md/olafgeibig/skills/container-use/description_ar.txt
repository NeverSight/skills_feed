استخدم هذه المهارة عند العمل مع Apple Containers (أجهزة Linux VM خفيفة الوزن) كبديل Docker أصلي على نظام التشغيل macOS. يتضمن ذلك إنشاء صور للحاويات، أو تشغيل الحاويات، أو إدارة دورة حياة الحاوية، أو تكوين الشبكات، أو التعامل مع وحدات التخزين، أو تحميل الملفات باستخدام إعادة توجيه SSH، أو تنفيذ إنشاءات متعددة المنصات. استدعاء المهام التي تتضمن حاوية CLI أو أداة حاوية Apple أو حاويات Linux على Apple Silicon macOS 26+.
