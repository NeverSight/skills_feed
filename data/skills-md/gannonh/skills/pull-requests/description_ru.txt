Используйте этот навык для рабочих процессов запросов на включение — создания запросов на ветку (ветвь, фиксация, отправка, открытие), проверки запросов на добавление (качество кода, покрытие тестированием, устранение проблем) или объединения запросов на добавление (проверка CI, слияние, очистка). Управляет полным жизненным циклом PR через gh CLI. Триггеры включали «создать PR», «открыть PR», «пересмотреть PR», «объединить PR».
