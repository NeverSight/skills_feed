Используйте этот навык при создании новых навыков Claude Code с нуля, редактировании существующих навыков для улучшения их описаний или структуры, а также при преобразовании субагентов Claude Code или косых команд в навыки. Это включает в себя разработку рабочих процессов навыков, написание файлов SKILL.md, организацию вспомогательных файлов с именами, раскрывающими намерения, а также использование инструментов CLI и соответствующих сценариев.
