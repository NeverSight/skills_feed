Utilizza questa abilità quando crei nuove abilità di Claude Code da zero, modifichi abilità esistenti per migliorarne le descrizioni o la struttura o converti gli agenti secondari di Claude Code o i comandi barra in abilità. Ciò include la progettazione di flussi di lavoro delle competenze, la scrittura di file SKILL.md, l'organizzazione di file di supporto con nomi che rivelino le intenzioni e l'utilizzo di strumenti CLI e script appropriati.
