استخدم هذه المهارة عند إنشاء مهارات Claude Code جديدة من البداية، أو تحرير المهارات الحالية لتحسين أوصافها أو بنيتها، أو تحويل وكلاء Claude Code الفرعيين أو أوامر الشرطة المائلة إلى مهارات. يتضمن ذلك تصميم سير عمل المهارات، وكتابة ملفات SKILL.md، وتنظيم الملفات الداعمة بأسماء تكشف النية، والاستفادة من أدوات واجهة سطر الأوامر (CLI) والبرمجة النصية المناسبة.
