Patrones de lenguaje de programación Zig actualizados para la versión 0.15.x. Úselo al escribir, revisar o depurar código Zig, trabajar con archivos build.zig y build.zig.zon o usar metaprogramación comptime. Es fundamental para evitar patrones obsoletos en los datos de entrenamiento, especialmente las API del sistema de compilación (root_module en lugar de root_source_file), las API de E/S (patrón de escritura en búfer), la inicialización del contenedor (.empty/.init), la selección del asignador (DebugAllocator) y las funciones de lenguaje eliminadas (async/await, usando el espacio de nombres).
