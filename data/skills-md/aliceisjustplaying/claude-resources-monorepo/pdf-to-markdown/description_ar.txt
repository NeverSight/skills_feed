قم بتحويل مستندات PDF بأكملها إلى Markdown نظيف ومنظم لتحميل السياق الكامل. استخدم هذه المهارة عندما يريد المستخدم استخراج كل النص من ملف PDF إلى السياق (وليس البحث/البحث)، أو عند مناقشة محتوى PDF أو تحليله بالكامل، أو عندما يذكر المستخدم "تحميل ملف PDF بأكمله"، أو "إحضار ملف PDF إلى السياق"، أو "قراءة ملف PDF بأكمله"، أو عندما يفقد الاستخراج/الجلب الجزئي سياقًا مهمًا. هذه هي الطريقة المفضلة لاستخراج نص PDF عبر طريقة صفحة تلو الأخرى أو طريقة grep.
