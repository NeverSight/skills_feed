Преобразуйте целые PDF-документы в чистый структурированный Markdown для полной загрузки контекста. Используйте этот навык, когда пользователь хочет извлечь ВЕСЬ текст из PDF-файла в контекст (не с помощью grep/поиска), при обсуждении или анализе содержимого PDF в полном объеме, когда пользователь упоминает «загрузить весь PDF-файл», «внести PDF-файл в контекст», «прочитать весь PDF-файл» или когда частичное извлечение/greping может пропустить важный контекст. Это предпочтительный метод извлечения текста PDF, а не постраничный подход или подход grep.
