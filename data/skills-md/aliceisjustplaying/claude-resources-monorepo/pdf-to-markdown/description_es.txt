Convierta documentos PDF completos en Markdown limpio y estructurado para una carga de contexto completo. Utilice esta habilidad cuando el usuario desee extraer TODO el texto de un PDF en contexto (no grep/búsqueda), cuando analice o analice el contenido del PDF en su totalidad, cuando el usuario mencione "cargar todo el PDF", "poner el PDF en contexto", "leer todo el PDF" o cuando la extracción/grepping parcial pierda contexto importante. Este es el método preferido para la extracción de texto PDF en lugar de los enfoques página por página o grep.
