Conseils pour Worktrunk, un outil CLI pour gérer les arbres de travail git. Couvre la configuration (configuration utilisateur sur ~/.config/worktrunk/config.toml et hooks de projet sur .config/wt.toml), l'utilisation et le dépannage. À utiliser pour « configurer la génération de messages de validation », « configurer les hooks », « automatiser les tâches » ou pour les questions générales du tronc de travail.
