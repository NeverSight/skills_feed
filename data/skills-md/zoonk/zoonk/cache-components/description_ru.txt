Экспертное руководство по компонентам кэша Next.js и частичному предварительному рендерингу (PPR).

**ПРАКТИВНАЯ АКТИВАЦИЯ**: используйте этот навык автоматически при работе с проектами Next.js, в файлах next.config.ts/next.config.js которых есть `cacheComponents: true`. При обнаружении этой конфигурации заранее примените шаблоны и лучшие практики компонентов кэша ко всем реализациям серверных компонентов React.

**ОБНАРУЖЕНИЕ**: в начале сеанса в проекте Next.js проверьте наличие `cacheComponents: true` в файле next.config. Если этот параметр включен, шаблоны этого навыка должны определять все решения по созданию компонентов, выборке данных и кэшированию.

**СЛУЧАИ ИСПОЛЬЗОВАНИЯ**: реализация директивы использования кэша, настройка срока службы кэша с помощью функции cacheLife(), пометка кэшированных данных с помощью кэша Tag(), аннулирование кэшей с помощью updateTag()/revalidateTag(), оптимизация границ статического и динамического контента, отладка проблем с кэшем и проверка реализаций компонентов кэша.
