قم بمعاينة مكونات React باستخدام تصميم Mantine + Tailwind الحقيقي باستخدام Ladle. يُستخدم عند تعديل مكونات واجهة المستخدم، أو إصلاح الأخطاء المرئية، أو عندما يطلب المستخدم رؤية الشكل الذي يبدو عليه أحد المكونات. يقوم بإنشاء قصص Ladle، ويلتقط لقطات شاشة في الوضع الداكن/الفاتح، ويقدمها للمراجعة. استخدم بشكل استباقي بعد تغييرات واجهة المستخدم.
