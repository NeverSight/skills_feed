Richtlinien zum Erstellen produktionsbereiter HTTP-APIs mit dem Deno- und Hono-Framework. Verwenden Sie diese Option, wenn Sie REST-APIs, Webdienste, Mikrodienste oder einen beliebigen HTTP-Server mit Deno-Laufzeit und Hono erstellen.
