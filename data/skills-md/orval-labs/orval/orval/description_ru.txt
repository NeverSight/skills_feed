Создавайте типобезопасные клиенты API, перехватчики TanStack Query/SWR, схемы Zod, макеты MSW, обработчики серверов Hono, серверы MCP и действия SolidStart из спецификаций OpenAPI с помощью Orval. Охватывает все клиенты (React/Vue/Svelte/Solid/Angular Query, Fetch, Axios), пользовательские мутаторы HTTP, шаблоны аутентификации, потоковую передачу NDJSON, программный API и расширенную настройку.
