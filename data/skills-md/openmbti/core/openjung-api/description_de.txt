Leitfaden zur Integration mit der OpenJung (OEJTS) Persönlichkeitstest-API – einer kostenlosen Open-Source-Alternative zu MBTI.
Verwenden Sie es, wenn Sie Anwendungen erstellen, die eine Persönlichkeitsbeurteilung erfordern, Tests im MBTI-Stil integrieren oder damit arbeiten
die OpenMBTI API-Endpunkte. Behandelt: (1) Testfragen abrufen, (2) Antworten einreichen und Ergebnisse berechnen,
(3) Das Bewertungssystem verstehen, (4) Testsitzungen aufzeichnen, (5) Gesamtstatistiken abrufen.
