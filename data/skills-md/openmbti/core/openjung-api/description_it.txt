Guida per l'integrazione con l'API del test della personalità OpenJung (OEJTS): un'alternativa gratuita e open source a MBTI.
Da utilizzare quando si creano applicazioni che necessitano di valutazione della personalità, si integrano test in stile MBTI o si lavora con
gli endpoint API OpenMBTI. Comprende: (1) recupero delle domande del test, (2) invio delle risposte e calcolo dei risultati,
(3) Comprendere il sistema di punteggio, (4) Registrare le sessioni di test, (5) Recuperare statistiche aggregate.
