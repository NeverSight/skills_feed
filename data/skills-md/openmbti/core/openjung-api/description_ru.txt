Руководство по интеграции с API личностного теста OpenJung (OEJTS) — бесплатной альтернативой MBTI с открытым исходным кодом.
Используйте при создании приложений, требующих оценки личности, интеграции тестов в стиле MBTI или работы с
конечные точки API OpenMBTI. Обложки: (1) Получение тестовых вопросов, (2) Отправка ответов и расчет результатов,
(3) Понимание системы оценок, (4) Запись тестовых сессий, (5) Получение совокупной статистики.
