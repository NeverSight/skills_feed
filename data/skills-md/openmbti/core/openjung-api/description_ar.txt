دليل التكامل مع واجهة برمجة تطبيقات اختبار الشخصية OpenJung (OEJTS) - بديل مجاني ومفتوح المصدر لـ MBTI.
يُستخدم عند إنشاء التطبيقات التي تحتاج إلى تقييم الشخصية، أو دمج اختبارات نمط MBTI، أو العمل معها
نقاط نهاية OpenMBTI API. يغطي: (1) جلب أسئلة الاختبار، (2) إرسال الإجابات وحساب النتائج،
(3) فهم نظام التسجيل، (4) تسجيل جلسات الاختبار، (5) استرجاع الإحصائيات الإجمالية.
