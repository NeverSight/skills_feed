Guía para la integración con la API de prueba de personalidad OpenJung (OEJTS): una alternativa gratuita y de código abierto a MBTI.
Úselo al crear aplicaciones que necesitan una evaluación de la personalidad, integrar pruebas de estilo MBTI o trabajar con
los puntos finales de la API de OpenMBTI. Cubre: (1) Obtener preguntas de la prueba, (2) Enviar respuestas y calcular resultados,
(3) Comprender el sistema de puntuación, (4) Grabar sesiones de prueba, (5) Recuperar estadísticas agregadas.
