Начните развертывать приложения в Google Cloud Run. Используйте, когда пользователь хочет развернуть свое приложение, поделиться своим приложением путем развертывания в облачной службе, сделать свое приложение доступным в Интернете или ему нужна помощь в выборе между методами развертывания Cloud Run (без сборки, пакеты сборки, Dockerfile или образ контейнера).
