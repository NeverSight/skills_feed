## How to Use

### 1. View Theme Showcase
Review the themes above or ask to see visual examples.

### 2. Select Theme
Choose a theme that matches your content's tone:
- **Professional/Corporate**: Modern Minimalist, Arctic Frost
- **Creative/Artistic**: Sunset Boulevard, Desert Rose
- **Tech/Development**: Tech Innovation, Midnight Galaxy
- **Nature/Wellness**: Forest Canopy, Botanical Garden, Ocean Depths
- **Luxury/Premium**: Golden Hour

### 3. Apply Theme
Provide the theme name when creating artifacts. The colors and fonts will be applied consistently throughout.

## Custom Theme Creation

For requirements not met by pre-set themes, provide:

1. **Mood/Feeling**: What emotion should the design evoke?
2. **Industry**: What sector is this for?
3. **Key Colors**: Any specific brand colors to incorporate?
4. **Typography Style**: Modern, classic, playful, professional?

A custom theme will be generated with:
- Primary, secondary, and accent colors
- Background color
- Header and body font pairings
- Contrast ratios for accessibility

## Application Guidelines

### Contrast Requirements
- Body text: minimum 4.5:1 contrast ratio
- Large text/headers: minimum 3:1 contrast ratio
- Interactive elements: minimum 3:1 contrast ratio

### Font Pairing Rules
- Headers: Display or serif fonts for impact
- Body: Sans-serif for readability
- Limit to 2-3 fonts per artifact

### Color Usage
- Primary: Main headings, key CTAs
- Secondary: Subheadings, navigation
- Accent: Highlights, links, buttons
- Background: Page/section backgrounds
