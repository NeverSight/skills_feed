URL が指定されている場合、Web コンテンツをクリーンなマークダウンとしてフェッチして抽出します。このスキルは、読み取り、分析、要約、または抽出が必要な URL (http/https リンク) をユーザーが提供するたびに使用します。生の HTML よりも 80% 少ないトークンで Web ページを Markdown に変換します。 JS を多用したサイト、ドキュメント、記事、ブログ投稿など、すべてのコンテンツ タイプを処理します。 3つの変換方法（自動、AI、ブラウザレンダリング）をサポートします。 URL を操作する場合は、web_fetch の代わりに常にこれを使用してください。これはより効率的で、よりクリーンな出力を提供します。
