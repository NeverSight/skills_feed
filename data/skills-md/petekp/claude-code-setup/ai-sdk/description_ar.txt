أجب عن الأسئلة حول AI SDK وساعد في إنشاء ميزات مدعومة بالذكاء الاصطناعي. يُستخدم عندما يقوم المطورون بما يلي: (1) طرح أسئلة حول وظائف AI SDK مثل generatorText أوstreamText أو ToolLoopAgent أو الأدوات، (2) الرغبة في إنشاء وكلاء الذكاء الاصطناعي أو روبوتات الدردشة أو ميزات إنشاء النص، (3) طرح أسئلة حول موفري الذكاء الاصطناعي (OpenAI أو Anthropic وما إلى ذلك) أو البث أو استدعاء الأدوات أو المخرجات المنظمة.
