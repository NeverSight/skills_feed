Valuta una base di codice per la compatibilità con gli agenti in base alle migliori pratiche degli agenti autonomi. Da utilizzare quando viene richiesto di "valutare gli agenti", "verificare la disponibilità dell'agente", "controllare l'esecuzione autonoma", "valutare la compatibilità dell'agente" o quando si esamina l'infrastruttura per il funzionamento automatico dell'agente. Da utilizzare anche quando viene richiesto di rendere una base di codice più adatta agli agenti IA o ai flussi di lavoro autonomi.
