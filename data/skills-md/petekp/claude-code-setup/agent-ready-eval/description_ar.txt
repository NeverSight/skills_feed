قم بتقييم قاعدة التعليمات البرمجية لملاءمة الوكيل بناءً على أفضل ممارسات الوكيل المستقل. استخدم عندما يُطلب منك "تقييم الوكلاء"، أو "التحقق من جاهزية الوكيل"، أو "تدقيق التنفيذ المستقل"، أو "تقييم مدى ملاءمة الوكيل"، أو عند مراجعة البنية التحتية لتشغيل الوكيل غير المراقب. استخدم أيضًا عند سؤالك عن جعل قاعدة التعليمات البرمجية أكثر ملاءمة لوكلاء الذكاء الاصطناعي أو سير العمل المستقل.
