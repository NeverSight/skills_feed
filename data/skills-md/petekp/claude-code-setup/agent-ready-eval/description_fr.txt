Évaluez une base de code pour la convivialité des agents en vous basant sur les meilleures pratiques des agents autonomes. À utiliser lorsqu'on vous demande « d'évaluer les agents », de « vérifier l'état de préparation des agents », d'« auditer l'exécution autonome », d'« évaluer la convivialité des agents » ou lors de l'examen de l'infrastructure pour le fonctionnement sans surveillance des agents. À utiliser également lorsqu'on vous demande de rendre une base de code plus adaptée aux agents IA ou aux flux de travail autonomes.
