Bewerten Sie eine Codebasis auf Agentenfreundlichkeit basierend auf Best Practices für autonome Agenten. Wird verwendet, wenn Sie aufgefordert werden, „für Agenten zu bewerten“, „die Agentenbereitschaft zu prüfen“, „auf autonome Ausführung zu prüfen“, „Agentenfreundlichkeit zu bewerten“ oder wenn Sie die Infrastruktur für den unbeaufsichtigten Agentenbetrieb überprüfen. Verwenden Sie es auch, wenn Sie gefragt werden, ob eine Codebasis besser für KI-Agenten oder autonome Arbeitsabläufe geeignet sein soll.
