Применяйте профессиональные принципы типографики для создания читаемых, иерархических и эстетически изысканных интерфейсов. Используйте при настройке масштабов шрифтов, выборе шрифтов, регулировке интервалов, разработке макетов с большим количеством текста или когда пользователь спрашивает о читаемости, сочетании шрифтов, высоте строки, размере или типографской иерархии.
