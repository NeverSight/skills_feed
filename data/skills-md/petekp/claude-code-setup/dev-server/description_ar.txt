ابدأ تشغيل خوادم التطوير باستخدام إدارة المنافذ الذكية. يُستخدم عندما يُطلب منك "بدء تشغيل خادم التطوير"، أو "تشغيل المطور"، أو "بدء التطوير"، أو "تشغيل الخادم"، أو "تشغيل التطبيق"، أو "تشغيل هذا"، أو "تمهيد الواجهة الأمامية"، أو أي طلب لتشغيل خادم تطوير محلي. يعالج تعارضات المنافذ، ويكتشف نوع المشروع، وينظف العمليات القديمة، ويفتح المتصفح تلقائيًا.
