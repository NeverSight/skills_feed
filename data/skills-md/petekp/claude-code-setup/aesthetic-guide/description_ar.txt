ابحث عن جمالية تصميم واجهة المستخدم وأنتج إرشادات تصميم شاملة وجاهزة للتنفيذ لوكلاء الترميز. يُستخدم عندما يذكر المستخدم عنصرًا جماليًا (وحشي، أو زجاجي، أو مستقبلي رجعي، أو حداثي سويسري، أو Apple HIG، أو neumorphism، أو Minimalism، أو cyberpunk، أو Material Design، أو art deco، أو vaporwave، وما إلى ذلك) ويريد دليل أسلوب كاملًا يحتوي على قيم CSS الدقيقة، ولوحات الألوان، وحالات المكونات، والرسوم المتحركة، والطباعة - مفصلة بما يكفي لوكيل الترميز لتنفيذ الجمالية بأمانة دون أي غموض.
