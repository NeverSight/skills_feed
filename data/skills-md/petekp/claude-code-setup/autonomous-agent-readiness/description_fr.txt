Évaluez l’état de préparation d’une base de code pour le développement d’agents autonomes et fournissez des recommandations personnalisées. À utiliser lorsqu'on vous le demande pour évaluer dans quelle mesure un projet prend en charge l'exécution d'agents sans surveillance, évaluer les pratiques de développement pour l'autonomie des agents, auditer l'infrastructure pour la fiabilité des agents ou améliorer une base de code pour les flux de travail d'agents autonomes. Se déclenche sur des requêtes telles que « évaluer ce projet pour déterminer l'état de préparation des agents », « dans quelle mesure cette base de code est-elle prête à être autonome », « évaluer l'infrastructure de l'agent » ou « améliorer les pratiques de développement pour les agents ».
