自律エージェント開発に対するコードベースの準備状況を評価し、カスタマイズされた推奨事項を提供します。プロジェクトがエージェントの無人実行をどの程度サポートしているかを評価したり、エージェントの自律性のための開発実践を評価したり、エージェントの信頼性を確保するためのインフラストラクチャを監査したり、自律エージェントのワークフローのコードベースを改善したりするように求められた場合に使用します。 「エージェントの準備状況についてこのプロジェクトを評価する」、「このコードベースがどの程度自律対応しているか」、「エージェントのインフラストラクチャを評価する」、または「エージェントの開発慣行を改善する」などのリクエストをトリガーします。
