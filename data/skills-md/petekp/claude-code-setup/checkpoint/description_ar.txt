دعم القرار المعرفي الذي يحلل السياق الحالي ويعرض خيارات الخطوة التالية الذكية للمستخدم. استخدم هذه المهارة عندما: (1) يستدعي المستخدم /نقطة التفتيش بشكل صريح، (2) يتم إكمال عمل مهم وتكون نقطة التفتيش ذات قيمة، (3) يوجد عدم يقين أو غموض حول المتطلبات أو النهج، (4) توسع تعقيد المهمة إلى ما هو أبعد من النطاق الأولي، (5) قبل الانتهاء أو الالتزام لضمان عدم تفويت أي شيء. تقوم هذه المهارة بإيقاف التنفيذ مؤقتًا، وتقييم الموقف بشكل كلي، وتقديم 2-5 خيارات مناسبة للسياق عبر AskUserQuestion، مع خيار وأساس منطقي موصى به.
