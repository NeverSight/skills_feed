التنقل بين إستراتيجية المنتج مفتوح المصدر وديناميكيات المجتمع والصيانة المستدامة. يُستخدم عند التخطيط لإصدارات برمجيات المصدر المفتوح، أو إدارة المساهمين، أو التعامل مع توقعات المجتمع، أو الموازنة بين المصالح التجارية والمجتمعية، أو عندما يحتاج المستخدم إلى حكمة تم اختبارها في المعركة بشأن البناء في العلن.
