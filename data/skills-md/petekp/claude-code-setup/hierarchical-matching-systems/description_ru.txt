Опыт проектирования, внедрения, проверки и отладки систем иерархического сопоставления. Используйте при работе с: (1) двусторонним сопоставлением (Гейл-Шепли, ординатор больницы, студент-школа), (2) задачами назначения/оптимизации (венгерский алгоритм, двустороннее сопоставление), (3) сопоставлением многоуровневой иерархии (организационные диаграммы, таксономии, вложенные категории), (4) разрешением сущностей и связью записей между иерархиями. Триггеры: отладка проблем с качеством сопоставления, проверка алгоритмов сопоставления, преобразование бизнес-требований в ограничения, проверка правильности сопоставления, разработка новых систем сопоставления, исправление нестабильных совпадений, устранение нарушений ограничений, диагностика несовпадения предпочтений.
