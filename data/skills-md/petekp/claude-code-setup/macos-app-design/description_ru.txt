Используйте при проектировании или создании собственных приложений macOS с помощью SwiftUI или AppKit. Запускается по структуре строки меню, сочетаниям клавиш, многооконному поведению, системе дизайна Liquid Glass, macOS Tahoe/Sequoia, навигации по боковой панели, дизайну панели инструментов, значкам приложений, символам SF или созданию ощущения «хорошего гражданина Mac».
