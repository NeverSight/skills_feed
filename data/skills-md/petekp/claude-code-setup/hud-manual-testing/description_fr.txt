Workflow de test manuel pour Claude HUD afin de vérifier les fonctionnalités de base. À utiliser lorsqu'on vous demande de « tester l'application », « vérifier que l'application fonctionne », « exécuter des tests manuels », « tester après des modifications » ou après avoir implémenté des fonctionnalités importantes. Effectue une réinitialisation complète, lance l'application et guide à travers la liste de contrôle de vérification.
