Critiquez les conceptions UI/UX pour leur clarté, leur hiérarchie, leur interaction, leur accessibilité et leur qualité de fabrication. À utiliser pour les revues de conception, les commentaires de relations publiques sur les modifications de l'interface utilisateur, l'évaluation des maquettes, la vérification si un composant est prêt à être expédié ou lorsqu'un retour honnête est nécessaire pour savoir si quelque chose répond à une barre haute.
