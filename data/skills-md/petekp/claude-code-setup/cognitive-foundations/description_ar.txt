تطبيق العلوم المعرفية وأبحاث HCI لتصميم القرارات. استخدمه عندما تحتاج إلى "السبب" العلمي وراء سهولة الاستخدام، أو شرح سلوك المستخدم، أو فهم حدود الإدراك/الذاكرة/الانتباه، أو تقييم الحمل المعرفي، أو تقييم توافق النموذج العقلي، أو التنبؤ بالأداء باستخدام قانون فيتس/هيك، أو ترسيخ قرارات الواجهة في البحث بدلاً من الرأي.
