Wenden Sie Kognitionswissenschaft und HCI-Forschung auf Designentscheidungen an. Verwenden Sie es, wenn Sie das wissenschaftliche „Warum“ hinter der Benutzerfreundlichkeit benötigen, das Benutzerverhalten erklären, Wahrnehmungs-/Gedächtnis-/Aufmerksamkeitsgrenzen verstehen, die kognitive Belastung bewerten, die Ausrichtung mentaler Modelle bewerten, die Leistung anhand des Fitts-/Hick-Gesetzes vorhersagen oder Schnittstellenentscheidungen eher in der Forschung als in der Meinung begründen möchten.
