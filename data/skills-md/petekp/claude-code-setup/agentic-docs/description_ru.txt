Пишите четкие, понятные комментарии к коду и документацию, которая находится рядом с кодом. Используйте при написании или проверке кода, для которого требуется встроенная документация, например заголовки файлов, документация по функциям, архитектурные решения или пояснительные комментарии. Хорошо работает как для читателей-людей, так и для помощников по программированию с использованием искусственного интеллекта, которые просматривают один файл за раз.
