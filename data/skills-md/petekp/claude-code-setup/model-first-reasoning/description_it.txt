Applica il Model-First Reasoning (MFR) alle attività di generazione del codice. Utilizzare quando l'utente richiede "prima il modello", "MFR", "modellazione formale prima della codifica", "modello poi implementazione" o quando le attività coinvolgono logica complessa, macchine a stati, sistemi di vincoli o qualsiasi implementazione che richiede garanzie di correttezza formale. Applica una rigorosa separazione tra le fasi di modellazione e implementazione.
