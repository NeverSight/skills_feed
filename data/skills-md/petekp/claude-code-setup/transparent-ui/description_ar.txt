قم بإنشاء واجهات تصحيح الأخطاء المستقلة التي تكشف عن الأعمال الداخلية للأنظمة المعقدة من خلال التصور التفاعلي. يُستخدم عندما يريد المستخدم فهم كيفية عمل شيء ما، أو تصحيح الحالة الداخلية، أو تصور تدفق البيانات، أو معرفة ما يحدث عندما يتفاعل مع النظام، أو إنشاء لوحة تصحيح لأي آلية معقدة. مشغلات لطلبات مثل "لا أفهم كيف يعمل هذا"، أو "أرني ما يحدث"، أو "تصور جهاز الحالة"، أو "إنشاء عرض تصحيح لهذا"، أو "مساعدتي في رؤية تدفق البيانات"، أو "اجعل هذا شفافًا"، أو أي طلب لفهم سلوك النظام الداخلي أو تصحيح أخطائه أو تصوره. ينطبق على أجهزة الحالة، أو أنظمة العرض، أو تدفقات الأحداث، أو الخوارزميات، أو الرسوم المتحركة، أو خطوط أنابيب البيانات، أو حسابات CSS، أو استعلامات قاعدة البيانات، أو أي نظام به أعمال داخلية غير واضحة.
