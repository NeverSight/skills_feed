Создавайте автономные интерфейсы отладки, которые раскрывают внутреннюю работу сложных систем посредством интерактивной визуализации. Используйте, когда пользователь хочет понять, как что-то работает, отладить внутреннее состояние, визуализировать поток данных, посмотреть, что происходит при взаимодействии с системой, или создать панель отладки для любого сложного механизма. Запускается по таким запросам, как «Я не понимаю, как это работает», «покажите мне, что происходит», «визуализируйте конечный автомат», «постройте представление отладки для этого», «помогите мне увидеть поток данных», «сделайте это прозрачным» или любой запрос на понимание, отладку или визуализацию внутреннего поведения системы. Применяется к конечным автоматам, системам рендеринга, потокам событий, алгоритмам, анимации, конвейерам данных, вычислениям CSS, запросам к базе данных или любой системе с неочевидной внутренней работой.
