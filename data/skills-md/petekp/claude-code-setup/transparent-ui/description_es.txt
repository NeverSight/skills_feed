Cree interfaces de depuración independientes que revelen el funcionamiento interno de sistemas complejos a través de visualización interactiva. Úselo cuando el usuario quiera comprender cómo funciona algo, depurar el estado interno, visualizar el flujo de datos, ver qué sucede cuando interactúa con el sistema o crear un panel de depuración para cualquier mecanismo complejo. Activadores de solicitudes como "No entiendo cómo funciona esto", "muéstrame qué está sucediendo", "visualiza la máquina de estado", "crea una vista de depuración para esto", "ayúdame a ver el flujo de datos", "haz esto transparente" o cualquier solicitud para comprender, depurar o visualizar el comportamiento interno del sistema. Se aplica a máquinas de estado, sistemas de representación, flujos de eventos, algoritmos, animaciones, canalizaciones de datos, cálculos CSS, consultas de bases de datos o cualquier sistema con funcionamiento interno no obvio.
