Содействовать методическому рассмотрению предложений (технические проекты, спецификации продукта, запросы функций). Используйте, когда вас просят «рассмотреть это предложение», «оставить отзыв об этом документе», «помогите мне просмотреть этот RFC» или когда представлен документ, требующий структурированной обратной связи. Обрабатывает файлы уценки, суть/проблемы/PR GitHub и другие текстовые форматы. Разумно разбивает предложения на части, прогнозирует реакцию рецензента и обеспечивает обратную связь, адаптированную к формату предложения.
