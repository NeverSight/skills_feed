Générez des profils de personnalité riches à partir des exportations de données de réseaux sociaux (Twitter/X, LinkedIn, Instagram). À utiliser lorsqu'un utilisateur souhaite analyser sa présence sur les réseaux sociaux, créer un profil de personnalité pour la personnalisation de l'IA, comprendre ses modèles de communication ou extraire des informations de son empreinte numérique. Se déclenche sur des requêtes telles que « analyser mes données Twitter », « créer un profil de personnalité », « que pouvez-vous apprendre sur moi grâce à mes publications », « personnaliser une IA pour moi » ou lorsque les utilisateurs fournissent des fichiers d'exportation de réseaux sociaux.
