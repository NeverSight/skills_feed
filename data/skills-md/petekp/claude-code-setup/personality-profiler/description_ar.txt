أنشئ ملفات شخصية غنية من خلال تصدير بيانات وسائل التواصل الاجتماعي (Twitter/X وLinkedIn وInstagram). يُستخدم عندما يريد المستخدم تحليل تواجده على وسائل التواصل الاجتماعي، أو إنشاء ملف تعريف شخصي لتخصيص الذكاء الاصطناعي، أو فهم أنماط الاتصال الخاصة به، أو استخلاص الرؤى من بصمته الرقمية. يتم تشغيل طلبات مثل "تحليل بيانات Twitter الخاصة بي" أو "إنشاء ملف تعريف شخصي" أو "ما الذي يمكنك معرفته عني من منشوراتي" أو "تخصيص الذكاء الاصطناعي لي" أو عندما يقدم المستخدمون ملفات تصدير الوسائط الاجتماعية.
