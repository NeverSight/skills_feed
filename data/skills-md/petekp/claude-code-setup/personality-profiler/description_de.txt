Generieren Sie umfassende Persönlichkeitsprofile aus Social-Media-Datenexporten (Twitter/X, LinkedIn, Instagram). Verwenden Sie es, wenn ein Benutzer seine Social-Media-Präsenz analysieren, ein Persönlichkeitsprofil für die KI-Personalisierung erstellen, seine Kommunikationsmuster verstehen oder Erkenntnisse aus seinem digitalen Fußabdruck gewinnen möchte. Wird bei Anfragen wie „Analysiere meine Twitter-Daten“, „Erstelle ein Persönlichkeitsprofil“, „Was kannst du aus meinen Beiträgen über mich erfahren“, „Personalisiere eine KI für mich“ oder wenn Benutzer Social-Media-Exportdateien bereitstellen, ausgelöst.
