Générez une invite d'amorçage intelligente pour poursuivre la conversation en cours dans une nouvelle session. À utiliser lorsque (1) approche des limites du contexte, (2) l'utilisateur dit « transfert », « bootstrap », « continuer plus tard », « enregistrer la session » ou similaire, (3) avant de fermer une session avec un travail inachevé, (4) l'utilisateur souhaite reprendre dans un environnement différent. Produit une invite prête pour le presse-papiers capturant le contexte essentiel tout en minimisant les jetons.
