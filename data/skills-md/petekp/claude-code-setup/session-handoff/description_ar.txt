قم بإنشاء مطالبة تمهيدية ذكية لمواصلة المحادثة الحالية في جلسة جديدة. يُستخدم عندما (1) يقترب من حدود السياق، (2) يقول المستخدم "handoff"، أو "bootstrap"، أو "يستمر لاحقًا"، أو "حفظ الجلسة"، أو ما شابه ذلك، (3) قبل إغلاق الجلسة بعمل غير مكتمل، (4) يريد المستخدم الاستئناف في بيئة مختلفة. يُخرج رسالة جاهزة للحافظة تلتقط السياق الأساسي مع تقليل الرموز المميزة.
