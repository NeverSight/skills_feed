Identifiez les signaux non évidents, les modèles cachés et les corrélations intelligentes dans les ensembles de données à l’aide de techniques d’analyse de données d’investigation. À utiliser lors de l'analyse des exportations de médias sociaux, des données utilisateur, des ensembles de données comportementales ou de toute donnée structurée pour laquelle des informations plus approfondies sont souhaitées. S'associe avec un profil de personnalité pour une extraction améliorée du signal. Se déclenche sur des requêtes telles que « quels modèles voyez-vous », « trouver des signaux cachés », « corréler ces ensembles de données », « qu'est-ce qui me manque dans ces données », « analyser des ensembles de données », « trouver des informations non évidentes » ou lorsque les utilisateurs souhaitent aller au-delà de l'analyse au niveau de la surface. Utilisez également de manière proactive lorsque vous remarquez des anomalies ou des corrélations intéressantes lors d'une tâche d'analyse de données.
