調査データ分析手法を使用して、データセット内の非明白なシグナル、隠れたパターン、および巧妙な相関関係を特定します。ソーシャル メディアのエクスポート、ユーザー データ、行動データセット、またはより深い洞察が必要な構造化データを分析するときに使用します。パーソナリティプロファイラーと組み合わせて信号抽出を強化します。 「どのようなパターンが見られるか」、「隠れたシグナルを見つける」、「これらのデータセットを関連付ける」、「このデータで何が欠けているか」、「データセット全体を分析する」、「明白でない洞察を見つける」などのリクエスト、またはユーザーが表面レベルの分析を超えたい場合にトリガーされます。また、データ分析タスク中に興味深い異常や相関関係に気づいた場合にも、積極的に使用してください。
