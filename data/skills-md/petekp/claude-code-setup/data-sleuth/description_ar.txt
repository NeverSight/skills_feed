تحديد الإشارات غير الواضحة والأنماط المخفية والارتباطات الذكية في مجموعات البيانات باستخدام تقنيات تحليل البيانات الاستقصائية. يُستخدم عند تحليل صادرات الوسائط الاجتماعية أو بيانات المستخدم أو مجموعات البيانات السلوكية أو أي بيانات منظمة تتطلب رؤى أعمق. أزواج مع ملف تعريف الشخصية لتحسين استخراج الإشارة. مشغلات لطلبات مثل "ما هي الأنماط التي تراها"، أو "العثور على إشارات مخفية"، أو "ربط مجموعات البيانات هذه"، أو "ما الذي أفتقده في هذه البيانات"، أو "التحليل عبر مجموعات البيانات"، أو "العثور على رؤى غير واضحة"، أو عندما يرغب المستخدمون في تجاوز التحليل على مستوى السطح. استخدم أيضًا بشكل استباقي عندما تلاحظ وجود حالات شاذة أو ارتباطات مثيرة للاهتمام أثناء أي مهمة لتحليل البيانات.
