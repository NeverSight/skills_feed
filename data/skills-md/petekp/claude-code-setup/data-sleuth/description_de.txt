Identifizieren Sie nicht offensichtliche Signale, versteckte Muster und clevere Korrelationen in Datensätzen mithilfe investigativer Datenanalysetechniken. Verwenden Sie es bei der Analyse von Social-Media-Exporten, Benutzerdaten, Verhaltensdatensätzen oder anderen strukturierten Daten, bei denen tiefere Einblicke gewünscht sind. Kombiniert mit dem Persönlichkeitsprofiler für eine verbesserte Signalextraktion. Wird auf Anfragen wie „Welche Muster sehen Sie“, „versteckte Signale finden“, „korrelieren Sie diese Datensätze“, „Was fehlt mir in diesen Daten“, „Datensatzübergreifend analysieren“, „nicht offensichtliche Erkenntnisse finden“ oder wenn Benutzer über die Analyse auf Oberflächenebene hinausgehen möchten, ausgelöst. Nutzen Sie es auch proaktiv, wenn Sie bei einer Datenanalyseaufgabe interessante Anomalien oder Zusammenhänge bemerken.
