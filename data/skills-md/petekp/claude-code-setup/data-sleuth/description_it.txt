Identifica segnali non ovvi, modelli nascosti e correlazioni intelligenti nei set di dati utilizzando tecniche di analisi investigativa dei dati. Da utilizzare quando si analizzano esportazioni di social media, dati utente, set di dati comportamentali o qualsiasi dato strutturato in cui si desiderano approfondimenti più approfonditi. Si abbina al profilatore della personalità per una migliore estrazione del segnale. Si attiva su richieste come "quali modelli vedi", "trova segnali nascosti", "correla questi set di dati", "cosa mi manca in questi dati", "analizza set di dati", "trova informazioni non ovvie" o quando gli utenti vogliono andare oltre l'analisi a livello superficiale. Utilizzalo anche in modo proattivo quando noti anomalie o correlazioni interessanti durante qualsiasi attività di analisi dei dati.
