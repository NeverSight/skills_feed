Úselo al analizar exportaciones de redes sociales, datos de usuarios, conjuntos de datos de comportamiento o cualquier dato estructurado donde se desee obtener conocimientos más profundos. Se combina con el perfilador de personalidad para mejorar la extracción de señales. Se activan solicitudes como "qué patrones ves", "encontrar señales ocultas", "correlacionar estos conjuntos de datos", "qué me falta en estos datos", "analizar entre conjuntos de datos", "encontrar información no obvia" o cuando los usuarios quieren ir más allá del análisis a nivel de superficie. Utilícelo también de forma proactiva cuando observe anomalías o correlaciones interesantes durante cualquier tarea de análisis de datos.
