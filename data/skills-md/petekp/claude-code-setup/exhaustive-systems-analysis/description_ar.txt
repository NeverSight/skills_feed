قم بإجراء تحليل شامل وعميق للنظام وأنظمته الفرعية لتحديد الأخطاء وظروف السباق والوثائق التي لا معنى لها والتعليمات البرمجية الميتة ومشكلات الصحة. يُستخدم عندما يُطلب منك "تدقيق هذا النظام"، أو "التحليل الشامل لـ X"، أو "التحليل للتأكد من صحته"، أو "استئصال المشكلات في..."، أو "التعمق في..."، أو "التحقق من صحة هذا الرمز"، أو "العثور على أخطاء في..."، أو عند مراجعة التعليمات البرمجية المكتوبة بواسطة الوكيل للتأكد من جاهزية الإنتاج. يقوم بتحليل الأنظمة تلقائيًا إلى أنظمة فرعية، ويطبق قوائم مراجعة التحليل المناسبة، وينتج نتائج منظمة مع تصنيف الخطورة.
