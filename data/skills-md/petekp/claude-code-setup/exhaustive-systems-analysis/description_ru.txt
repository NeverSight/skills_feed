Выполните всесторонний и глубокий анализ системы и ее подсистем для выявления ошибок, состояний гонки, устаревшей документации, неработающего кода и проблем с корректностью. Используйте, когда вас просят «проверить эту систему», «исчерпывающий анализ X», «проанализировать правильность», «искоренить проблемы в...», «углубиться в...», «проверить правильность этого кода», «найти ошибки в...» или при проверке готовности кода, написанного агентом. Автоматически разбивает системы на подсистемы, применяет соответствующие контрольные списки анализа и выдает структурированные результаты с классификацией серьезности.
