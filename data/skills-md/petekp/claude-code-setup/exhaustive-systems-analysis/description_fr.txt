À utiliser lorsqu'on vous demande « auditer ce système », « analyse exhaustive de X », « analyser l'exactitude », « éliminer les problèmes dans... », « approfondir en profondeur » , « vérifier que ce code est correct », « trouver des bogues dans » ou lors de l'examen du code écrit par l'agent pour être prêt à la production. Décompose automatiquement les systèmes en sous-systèmes, applique les listes de contrôle d'analyse appropriées et produit des résultats structurés avec classification de gravité.
