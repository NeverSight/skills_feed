Geführter Shape Up-Workflow für die Umsetzung von Projekten von der Idee bis zur funktionierenden Software. Orchestriert die /Shaping- und /Breadboarding-Fähigkeiten durch einen strukturierten Prozess: Frame, Shape, Breadboard, Slice, Build. Funktioniert sowohl für Greenfield-Projekte (0-1) als auch für bestehende Projekte. Verwenden Sie diese Option, wenn Sie ein neues Projekt oder eine neue Funktion starten, eine wesentliche Änderung an einer vorhandenen Codebasis planen, der Benutzer „Gestalten Sie dies“, „Lass uns gestalten“, „Gestalten“ sagt oder von der Idee zur Implementierung mit strukturierter Problem-/Lösungstrennung übergehen möchte. Leitet jede Phase proaktiv und schlägt die nächsten Schritte vor.
