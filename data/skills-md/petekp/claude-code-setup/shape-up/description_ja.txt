プロジェクトをアイデアから実用的なソフトウェアに移行するためのガイド付きシェイプアップ ワークフロー。構造化されたプロセス (フレーム、シェイプ、ブレッドボード、スライス、ビルド) を通じて、/shaping スキルと /breadboarding スキルを調整します。グリーンフィールド (0-1) と既存のプロジェクトの両方で機能します。新しいプロジェクトや機能を開始する場合、既存のコードベースへの大幅な変更を計画する場合、ユーザーが「これを形にする」、「形にしよう」、「形を整える」と言う場合、または構造化された問題と解決策を分離してアイデアから実装に移行したい場合に使用します。各フェーズを積極的にガイドし、次のステップを提案します。
