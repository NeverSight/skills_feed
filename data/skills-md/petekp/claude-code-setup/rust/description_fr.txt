Modèles Rust robustes pour les données sauvegardées sur fichiers, l'analyse, la persistance, les limites FFI et l'intégration du système. À utiliser lors de l'écriture de Rust qui gère les formats de fichiers, l'intégration de sous-processus, la gestion PID/processus, la sérialisation Serde ou les limites UniFFI. Couvre la sécurité UTF-8, les écritures atomiques, les machines à états et la gestion défensive des erreurs.
