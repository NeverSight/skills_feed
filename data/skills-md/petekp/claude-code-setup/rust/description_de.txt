Robuste Rust-Muster für dateigestützte Daten, Parsing, Persistenz, FFI-Grenzen und Systemintegration. Wird beim Schreiben von Rust verwendet, das Dateiformate, Unterprozessintegration, PID-/Prozessverwaltung, Serde-Serialisierung oder UniFFI-Grenzen verarbeitet. Behandelt UTF-8-Sicherheit, atomare Schreibvorgänge, Zustandsmaschinen und defensive Fehlerbehandlung.
