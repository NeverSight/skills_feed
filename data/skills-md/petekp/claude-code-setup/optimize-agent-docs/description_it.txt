Crea un livello di conoscenza ottimizzato per il recupero sulla documentazione dell'agente in file dot (.claude, .codex, .cursor, .aider). Da utilizzare quando viene richiesto di "ottimizzare i documenti", "migliorare la conoscenza dell'agente", "rendere i documenti più efficienti" o quando la documentazione si è accumulata e il recupero sembra inefficiente. Genera un manifest che mappa i contesti delle attività in blocchi di conoscenza, ottimizza la densità delle informazioni e crea artefatti compilati per un consumo efficiente degli agenti.
