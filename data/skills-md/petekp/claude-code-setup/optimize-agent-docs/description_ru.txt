Создайте уровень знаний, оптимизированный для поиска, поверх документации агента в точечных файлах (.claude, .codex, .cursor, .aider). Используйте, когда вас просят «оптимизировать документы», «улучшить знания агентов», «сделать документы более эффективными» или когда документация накопилась и поиск кажется неэффективным. Создает манифест, сопоставляющий контексты задач с фрагментами знаний, оптимизирует плотность информации и создает скомпилированные артефакты для эффективного использования агентов.
