Guía experta para configurar, personalizar y aprovechar creativamente OpenClaw, la puerta de enlace de IA autohospedada que conecta los LLM con los canales de mensajería (Telegram, WhatsApp, Discord, Slack, iMessage, etc.). Úselo cuando el usuario desee: (1) configurar o modificar su configuración de openclaw.json, (2) escribir o editar archivos de arranque (SOUL.md, USER.md, AGENTS.md, IDENTITY.md, TOOLS.md), (3) configurar canales de mensajería, (4) configurar modelos y proveedores, (5) crear enrutamiento multiagente, (6) crear habilidades, enlaces o trabajos cron, (7) solucionar problemas de OpenClaw, (8) Obtenga ideas creativas para aprovechar OpenClaw de formas no obvias.
