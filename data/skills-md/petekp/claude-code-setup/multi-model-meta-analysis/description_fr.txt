Synthétisez les résultats de plusieurs modèles d’IA dans une évaluation complète et vérifiée. À utiliser lorsque : (1) l'utilisateur colle les commentaires/analyses de plusieurs LLM (Claude, GPT, Gemini, etc.) sur le code ou un projet, (2) l'utilisateur souhaite consolider les sorties du modèle dans un seul document fiable, (3) l'utilisateur a besoin de revendications de modèle contradictoires résolues par rapport au code source réel. Cette compétence vérifie les affirmations du modèle par rapport à la base de code, résout les contradictions avec des preuves et produit une évaluation plus fiable que n'importe quel modèle unique.
