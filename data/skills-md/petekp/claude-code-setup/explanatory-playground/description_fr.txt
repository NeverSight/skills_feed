Créez des interfaces de débogage interactives qui révèlent le comportement interne du système. À utiliser lorsqu'on vous demande de « m'aider à comprendre comment cela fonctionne », « me montrer ce qui se passe », « visualiser l'état », « créer une vue de débogage », « je ne vois pas ce qui se passe » ou toute demande visant à rendre visible le comportement opaque du système. S'applique aux machines à états, aux flux de données, aux systèmes d'événements, aux algorithmes, aux cycles de rendu, aux animations, aux calculs CSS ou à tout mécanisme avec des composants internes cachés.
