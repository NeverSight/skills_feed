قم بمراجعة المهارات الحالية (العالمية وعلى مستوى المشروع) للتأكد من ملاءمة الوكيل والاتساق وأفضل الممارسات. يُستخدم عندما يُطلب منك "تدقيق مهاراتي"، أو "مراجعة إعداد المهارات"، أو "تحليل جودة المهارات"، أو "التحقق من صحة المهارات"، أو "تحسين مهاراتي"، أو عند الرغبة في تقييم النظام البيئي الشامل للمهارات. يقدم توصيات قابلة للتنفيذ لتحسين فعالية المهارات.
