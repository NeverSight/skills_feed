Проверяйте существующие навыки (глобальные и на уровне проекта) на предмет удобства для агентов, последовательности и передового опыта. Используйте, когда вас просят «проверить мои навыки», «проверить настройку навыков», «проанализировать качество навыков», «проверить состояние навыков», «улучшить мои навыки» или когда вы хотите оценить общую экосистему навыков. Содержит практические рекомендации по повышению эффективности навыков.
