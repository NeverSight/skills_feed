Erstellen Sie eine abrufoptimierte Wissensschicht über der Agentendokumentation in Dotfiles (.claude, .codex, .cursor, .aider). Verwenden Sie diese Option, wenn Sie aufgefordert werden, „Dokumente zu optimieren“, „das Wissen der Agenten zu verbessern“, „Dokumente effizienter zu gestalten“ oder wenn sich die Dokumentation angesammelt hat und der Abruf ineffizient erscheint. Erzeugt ein Manifest, das Aufgabenkontexte Wissensblöcken zuordnet, die Informationsdichte optimiert und kompilierte Artefakte für eine effiziente Agentennutzung erstellt.
