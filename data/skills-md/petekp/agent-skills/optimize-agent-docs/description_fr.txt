Créez une couche de connaissances optimisée pour la récupération sur la documentation de l'agent dans des fichiers dot (.claude, .codex, .cursor, .aider). À utiliser lorsqu'on vous demande « d'optimiser les documents », « d'améliorer les connaissances des agents », de « rendre les documents plus efficaces » ou lorsque la documentation s'est accumulée et que la récupération semble inefficace. Génère un manifeste mappant les contextes de tâches aux fragments de connaissances, optimise la densité des informations et crée des artefacts compilés pour une consommation efficace des agents.
