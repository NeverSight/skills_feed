أنشئ طبقة معرفة محسنة للاسترجاع عبر وثائق الوكيل في ملفات نقطية (.claude، .codex، .cursor، .aider). يُستخدم عندما يُطلب منك "تحسين المستندات"، أو "تحسين معرفة الوكيل"، أو "جعل المستندات أكثر كفاءة"، أو عندما تتراكم الوثائق ويصبح الاسترجاع غير فعال. يُنشئ سياقات مهام تعيين واضحة لأجزاء المعرفة، ويحسن كثافة المعلومات، وينشئ عناصر مجمعة لاستهلاك الوكيل بكفاءة.
