Cree una capa de conocimiento optimizada para la recuperación sobre la documentación del agente en archivos de puntos (.claude, .codex, .cursor, .aider). Úselo cuando se le solicite "optimizar documentos", "mejorar el conocimiento de los agentes", "hacer que los documentos sean más eficientes" o cuando se haya acumulado documentación y la recuperación parezca ineficiente. Genera un manifiesto de mapeo de contextos de tareas a fragmentos de conocimiento, optimiza la densidad de la información y crea artefactos compilados para un consumo eficiente de agentes.
