Aplique Model-First Reasoning (MFR) a las tareas de generación de código. Úselo cuando el usuario solicite "modelar primero", "MFR", "modelado formal antes de codificar", "modelar y luego implementar", o cuando las tareas involucren lógica compleja, máquinas de estados, sistemas de restricciones o cualquier implementación que requiera garantías formales de corrección. Hace cumplir una separación estricta entre las fases de modelado e implementación.
