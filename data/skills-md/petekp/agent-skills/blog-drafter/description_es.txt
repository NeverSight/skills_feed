Redacción de publicaciones de blog basadas en entrevistas para audiencias de productos técnicos. Úselo cuando el usuario quiera escribir una publicación de blog, un artículo o un ensayo y necesite ayuda para desarrollar su tesis, estructura y borrador inicial. Activadores al "escribir una publicación de blog", "redactar un artículo", "ayúdame a escribir sobre X", "redactor de blog" o cuando el usuario tiene un tema que desea convertir en contenido escrito. Realiza entrevistas estructuradas utilizando AskUserQuestion para extraer los conocimientos únicos del usuario antes de generar borradores.
