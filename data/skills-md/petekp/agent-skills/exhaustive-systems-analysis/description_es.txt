Realice un análisis exhaustivo y profundo de un sistema y sus subsistemas para identificar errores, condiciones de carrera, documentación obsoleta, código inactivo y problemas de corrección. Úselo cuando se le solicite "auditar este sistema", "análisis exhaustivo de X", "analizar la corrección", "eliminar problemas en...", "profundizar en...", "verificar que este código sea correcto", "encontrar errores en..." o cuando revise el código escrito por el agente para comprobar su preparación para la producción. Descompone automáticamente los sistemas en subsistemas, aplica listas de verificación de análisis apropiadas y produce hallazgos estructurados con clasificación de gravedad.
