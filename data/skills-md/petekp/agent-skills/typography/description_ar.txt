تطبيق مبادئ الطباعة الاحترافية لإنشاء واجهات سهلة القراءة وتسلسلية وجمالية. يُستخدم عند ضبط مقاييس الكتابة، أو اختيار الخطوط، أو ضبط التباعد، أو تصميم تخطيطات كثيفة النص، أو تنفيذ الطباعة في الوضع الداكن، أو عند السؤال عن سهولة القراءة، أو إقران الخطوط، أو ارتفاع الخط، أو القياس، أو التسلسل الهرمي المطبعي، أو الخطوط المتغيرة، أو تحميل الخط، أو ميزات OpenType.
