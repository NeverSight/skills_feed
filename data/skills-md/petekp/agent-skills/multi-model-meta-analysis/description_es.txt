Sintetice los resultados de múltiples modelos de IA en una evaluación integral y verificada. Úselo cuando: (1) el usuario pega comentarios/análisis de múltiples LLM (Claude, GPT, Gemini, etc.) sobre el código o un proyecto, (2) el usuario desea consolidar los resultados del modelo en un único documento confiable, (3) el usuario necesita reclamos de modelos conflictivos resueltos con el código fuente real. Esta habilidad verifica las afirmaciones del modelo con respecto al código base, resuelve contradicciones con evidencia y produce una evaluación más confiable que cualquier modelo individual.
