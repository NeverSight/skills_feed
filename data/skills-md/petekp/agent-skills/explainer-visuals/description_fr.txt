Créez des visuels explicatifs animés de haute qualité pour les essais et les articles de blog. À utiliser lorsque l'utilisateur souhaite visualiser des concepts, des processus, des données ou des idées avec des animations Web interactives. Se déclenche sur des requêtes telles que « créer un visuel pour », « animer ce concept », « créer un explicatif », « visualiser cette idée », « schématiser ce processus », « afficher ces données » ou lorsque le contenu de l'essai bénéficierait d'une explication visuelle. Gère les concepts abstraits (modèles mentaux, cadres), les processus techniques (algorithmes, systèmes) et la visualisation des données (tendances, comparaisons). Génère du HTML/CSS/JS autonome qui s'intègre directement dans le contenu Web.
