Alliance Canada (旧 Compute Canada) HPC クラスターで ML/AI ワークロードを実行するためのガイド。このスキルは、ユーザーが Alliance Canada、Compute Canada、Narval、Cedar、Graham、Trillium、Niagara、Fir、Nibi、Rorqual、Killarney、カナダ クラスターでの Slurm ジョブの送信、ML トレーニング用の sbatch、HPC での GPU ジョブ、クラスターでの virtualenv、または共有コンピューティング インフラストラクチャでのディープ ラーニングの実行に関連するものについて言及するたびに使用します。また、クラスター上の CUDA、Slurm 上の DeepSpeed/PyTorch DDP を使用した分散トレーニング、並列ファイルシステム上のデータセットの管理、HPC 上の HuggingFace、クラスター上の Apache Arrow または Parquet、共有コンピューティングの ML ベスト プラクティス、Apptainer、Singularity、HPC 上のコンテナー、vLLM、HPC 上の推論提供、またはクラスター上の LLM 推論についてユーザーが質問した場合にも使用します。
