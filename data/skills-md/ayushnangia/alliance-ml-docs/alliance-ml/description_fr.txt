Guide pour exécuter des charges de travail ML/AI sur les clusters HPC d'Alliance Canada (anciennement Calcul Canada). Utilisez cette compétence chaque fois que l'utilisateur mentionne Alliance Canada, Compute Canada, Narval, Cedar, Graham, Trillium, Niagara, Fir, Nibi, Rorqual, Killarney, la soumission de tâches Slurm sur des clusters canadiens, un lot pour la formation ML, des tâches GPU sur HPC, virtualenv sur des clusters ou tout ce qui concerne l'exécution d'un apprentissage en profondeur sur une infrastructure informatique partagée. À utiliser également lorsque l'utilisateur pose des questions sur CUDA sur les clusters, la formation distribuée avec DeepSpeed/PyTorch DDP sur Slurm, la gestion des ensembles de données sur des systèmes de fichiers parallèles, HuggingFace sur HPC, Apache Arrow ou Parquet sur les clusters, les meilleures pratiques de ML pour l'informatique partagée, Apptainer, Singularity, les conteneurs sur HPC, vLLM, l'inférence sur HPC ou l'inférence LLM sur les clusters.
