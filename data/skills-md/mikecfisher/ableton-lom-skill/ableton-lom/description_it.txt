Riferimento API Ableton Live Object Model (LOM) completo per lo sviluppo di script remoti in Python. Utilizza questa competenza quando lavori con Ableton Live Remote Scripts, superfici di controllo o qualsiasi codice Python che si interfaccia con l'API interna di Ableton Live. Copre l'API Live 12.3 tra cui: Song, Track, Clip, Device, DeviceParameter, Scene, ClipSlot, MixerDevice, Chain, RackDevice, Browser e tutte le classi di dispositivi specializzati (Simpler, Wavetable, Drift, ecc.). Fornisce tipi di proprietà, modalità di accesso (sola lettura/lettura-scrittura), stato osservabile, firme dei metodi e percorsi canonici. Essenziale per: (1) creare o estendere script remoti, (2) creare server MCP per Ableton, (3) sviluppare integrazioni con superfici di controllo, (4) automatizzare Live tramite Python.
