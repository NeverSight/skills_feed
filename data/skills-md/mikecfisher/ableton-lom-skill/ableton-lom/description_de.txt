Umfassende Ableton Live Object Model (LOM) API-Referenz für die Entwicklung von Remote-Skripten in Python. Nutzen Sie diese Fähigkeit, wenn Sie mit Ableton Live Remote-Skripten, Bedienoberflächen oder jedem anderen Python-Code arbeiten, der mit der internen API von Ableton Live verbunden ist. Deckt die Live 12.3-API ab, einschließlich: Song, Track, Clip, Device, DeviceParameter, Scene, ClipSlot, MixerDevice, Chain, RackDevice, Browser und alle speziellen Geräteklassen (Simpler, Wavetable, Drift usw.). Stellt Eigenschaftstypen, Zugriffsmodi (schreibgeschützt/Lese-Schreiben), beobachtbaren Status, Methodensignaturen und kanonische Pfade bereit. Unverzichtbar für: (1) Erstellen oder Erweitern von Remote-Skripten, (2) Erstellen von MCP-Servern für Ableton, (3) Entwickeln von Bedienoberflächenintegrationen, (4) Automatisieren von Live über Python.
