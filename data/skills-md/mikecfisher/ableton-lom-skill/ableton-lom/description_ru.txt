Комплексный справочник по API Ableton Live Object Model (LOM) для разработки удаленных сценариев на Python. Используйте этот навык при работе с удаленными сценариями Ableton Live, панелями управления или любым кодом Python, который взаимодействует с внутренним API Ableton Live. Охватывает API Live 12.3, включая Song, Track, Clip, Device, DeviceParameter, Scene, ClipSlot, MixerDevice, Chain, RackDevice, Browser и все специализированные классы устройств (Simpler, Wavetable, Drift и т. д.). Предоставляет типы свойств, режимы доступа (только чтение/чтение-запись), наблюдаемый статус, сигнатуры методов и канонические пути. Необходим для: (1) создания или расширения удаленных сценариев, (2) создания серверов MCP для Ableton, (3) разработки интеграции с поверхностью управления, (4) автоматизации Live с помощью Python.
