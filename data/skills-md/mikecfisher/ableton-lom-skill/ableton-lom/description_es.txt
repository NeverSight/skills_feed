Referencia completa de la API de Ableton Live Object Model (LOM) para desarrollar scripts remotos en Python. Utilice esta habilidad cuando trabaje con Ableton Live Remote Scripts, superficies de control o cualquier código Python que interactúe con la API interna de Ableton Live. Cubre la API de Live 12.3, que incluye: canción, pista, clip, dispositivo, parámetro de dispositivo, escena, ClipSlot, MixerDevice, Chain, RackDevice, navegador y todas las clases de dispositivos especializados (Simpler, Wavetable, Drift, etc.). Proporciona tipos de propiedades, modos de acceso (solo lectura/lectura-escritura), estado observable, firmas de métodos y rutas canónicas. Esencial para: (1) Crear o ampliar scripts remotos, (2) Crear servidores MCP para Ableton, (3) Desarrollar integraciones de superficies de control, (4) Automatizar Live a través de Python.
