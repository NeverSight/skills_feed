مرجع API الشامل لنموذج Ableton Live Object (LOM) لتطوير البرامج النصية عن بعد في Python. استخدم هذه المهارة عند العمل مع البرامج النصية لـ Ableton Live Remote أو أسطح التحكم أو أي رمز Python يتفاعل مع واجهة برمجة التطبيقات الداخلية لـ Ableton Live. يغطي Live 12.3 API بما في ذلك: الأغنية، والمسار، والمقطع، والجهاز، ومعلمة الجهاز، والمشهد، وClipSlot، وMixerDevice، وChain، وRackDevice، والمتصفح، وجميع فئات الأجهزة المتخصصة (Simpler، Wavetable، Drift، وما إلى ذلك). يوفر أنواع الخصائص، وأوضاع الوصول (للقراءة فقط/للقراءة والكتابة)، والحالة القابلة للملاحظة، وتوقيعات الطريقة، والمسارات الأساسية. ضروري لـ: (1) إنشاء البرامج النصية عن بعد أو توسيعها، (2) إنشاء خوادم MCP لـ Ableton، (3) تطوير عمليات تكامل سطح التحكم، (4) أتمتة البث المباشر عبر Python.
