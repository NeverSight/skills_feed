用於在 Python 中開發遠程腳本的綜合 Ableton Live 對像模型 (LOM) API 參考。在使用 Ableton Live 遠程腳本、控制界面或與 Ableton Live 內部 API 交互的任何 Python 代碼時使用此技能。涵蓋 Live 12.3 API，包括：Song、Track、Clip、Device、DeviceParameter、Scene、ClipSlot、MixerDevice、Chain、RackDevice、Browser 和所有專用設備類（Simpler、Wavetable、Drift 等）。提供屬性類型、訪問模式（只讀/讀寫）、可觀察狀態、方法簽名和規範路徑。對於以下用途至關重要：(1) 構建或擴展遠程腳本，(2) 為 Ableton 創建 MCP 服務器，(3) 開發控制界面集成，(4) 通過 Python 自動化直播。
