Référence complète de l'API Ableton Live Object Model (LOM) pour le développement de scripts distants en Python. Utilisez cette compétence lorsque vous travaillez avec des scripts à distance d'Ableton Live, des surfaces de contrôle ou tout code Python qui s'interface avec l'API interne d'Ableton Live. Couvre l'API Live 12.3, notamment : Song, Track, Clip, Device, DeviceParameter, Scene, ClipSlot, MixerDevice, Chain, RackDevice, Browser et toutes les classes de périphériques spécialisées (Simpler, Wavetable, Drift, etc.). Fournit les types de propriétés, les modes d'accès (lecture seule/lecture-écriture), l'état observable, les signatures de méthode et les chemins canoniques. Indispensable pour : (1) Créer ou étendre des scripts distants, (2) Créer des serveurs MCP pour Ableton, (3) Développer des intégrations de surfaces de contrôle, (4) Automatiser Live via Python.
