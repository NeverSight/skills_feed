Python でリモート スクリプトを開発するための包括的な Ableton Live Object Model (LOM) API リファレンス。このスキルは、Ableton Live リモート スクリプト、コントロール サーフェス、または Ableton Live の内部 API と接続する Python コードを操作するときに使用します。次の Live 12.3 API をカバーします: Song、Track、Clip、Device、DeviceParameter、Scene、ClipSlot、MixerDevice、Chain、RackDevice、Browser、およびすべての特殊なデバイス クラス (Simpler、Wavetable、Drift など)。プロパティ タイプ、アクセス モード (読み取り専用/読み取り/書き込み)、監視可能なステータス、メソッド シグネチャ、および正規パスを提供します。以下の場合に不可欠です: (1) リモート スクリプトの構築または拡張、(2) Ableton 用の MCP サーバーの作成、(3) コントロール サーフェス統合の開発、(4) Python による Live の自動化。
