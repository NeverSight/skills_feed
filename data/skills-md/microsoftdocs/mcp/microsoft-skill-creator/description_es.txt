Cree habilidades de agente para tecnologías de Microsoft utilizando las herramientas de Learn MCP. Úselo cuando los usuarios quieran crear una habilidad que enseñe a los agentes sobre cualquier tecnología, biblioteca, marco o servicio de Microsoft (Azure, .NET, M365, VS Code, Bicep, etc.). Investiga temas en profundidad y luego genera una habilidad híbrida que almacena conocimientos esenciales localmente y al mismo tiempo permite una investigación dinámica más profunda.
