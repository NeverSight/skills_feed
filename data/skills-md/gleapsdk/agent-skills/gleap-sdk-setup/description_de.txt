Integriert das Gleap-Kundenfeedback-SDK in Projekte. Erkennt die Plattform (JavaScript, iOS, Android, React Native, Flutter, Ionic Capacitor, Cordova, FlutterFlow) und führt Sie durch Installation, Initialisierung, Berechtigungen und allgemeine API-Nutzung wie Benutzeridentifikation und Ereignisverfolgung. Verwenden Sie diese Option, wenn Sie Gleap hinzufügen, das Feedback-SDK einrichten oder das Gleap-SDK integrieren.
