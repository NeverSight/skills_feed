Интегрирует SDK обратной связи с клиентами Gleap в проекты. Обнаруживает платформу (JavaScript, iOS, Android, React Native, Flutter, Ionic Capacitor, Cordova, FlutterFlow) и направляет процесс установки, инициализации, разрешений и общего использования API, такого как идентификация пользователя и отслеживание событий. Используйте при добавлении Gleap, настройке SDK обратной связи или интеграции Gleap SDK.
