يدمج SDK لتعليقات عملاء Gleap في المشاريع. يكتشف النظام الأساسي (JavaScript، وiOS، وAndroid، وReact Native، وFlutter، وIonic Capacitor، وCordova، وFlutterFlow) ويرشدك خلال التثبيت والتهيئة والأذونات والاستخدام الشائع لواجهة برمجة التطبيقات (API) مثل تحديد المستخدم وتتبع الأحداث. يُستخدم عند إضافة Gleap، أو إعداد SDK للتعليقات، أو دمج Gleap SDK.
