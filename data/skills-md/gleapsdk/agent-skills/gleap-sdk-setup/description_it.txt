Integra l'SDK per il feedback dei clienti Gleap nei progetti. Rileva la piattaforma (JavaScript, iOS, Android, React Native, Flutter, Ionic Capacitor, Cordova, FlutterFlow) e guida attraverso l'installazione, l'inizializzazione, le autorizzazioni e l'utilizzo comune delle API come l'identificazione dell'utente e il monitoraggio degli eventi. Da utilizzare quando si aggiunge Gleap, si configura l'SDK di feedback o si integra l'SDK di Gleap.
