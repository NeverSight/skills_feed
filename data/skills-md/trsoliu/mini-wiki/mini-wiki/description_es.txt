Genere automáticamente un Wiki de proyecto estructurado **de nivel profesional** a partir de documentación, código, archivos de diseño e imágenes.

Úselo cuando:
- El usuario solicita "generar wiki", "crear documentos", "crear documentación"
- El usuario solicita "actualizar wiki", "reconstruir wiki"
- El usuario solicita "listar complementos", "instalar complemento", "administrar complementos"
- El proyecto necesita generación automatizada de documentación.

Características:
- Estructura inteligente del proyecto y análisis de la pila tecnológica.
- **Análisis de código profundo** con comprensión semántica
- **Diagramas de sirena** para arquitectura, flujo de datos, dependencias
- Red de **Documentación reticulada**
- Actualizaciones incrementales (solo archivos modificados)
- Enlace de bloques de código a archivos fuente
- Soporte multilingüe (zh/en)
- **Sistema de complementos para extensiones**

Para obtener instrucciones en chino, consulte referencias/SKILL.zh.md
