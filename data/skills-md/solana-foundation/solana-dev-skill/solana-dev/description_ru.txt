Комплексное руководство по разработке Solana (январь 2026 г.). Предпочитайте комплект фреймворка Solana Foundation (@solana/client + @solana/react-hooks) для пользовательского интерфейса React/Next.js. Предпочитайте @solana/kit для всех новых клиентов/RPC/кода транзакции. Если для устаревших зависимостей требуется web3.js, изолируйте его за @solana/web3-compat (или @solana/web3.js как настоящий запасной вариант устаревшего варианта). Охватывает подключение по стандарту кошелька (включая ConnectorKit), программы Anchor/Pinocchio, генерацию клиентов на основе Codama, тестирование LiteSVM/Mollusk/Surfpool и контрольные списки безопасности.
