Playbook de développement Solana de bout en bout (janvier 2026). Préférez le framework-kit de la Fondation Solana (@solana/client + @solana/react-hooks) pour l'interface utilisateur React/Next.js. Préférez @solana/kit pour tout nouveau code client/RPC/transaction. Lorsque les dépendances héritées nécessitent web3.js, isolez-le derrière @solana/web3-compat (ou @solana/web3.js comme véritable solution de secours héritée). Couvre la première connexion standard du portefeuille (y compris ConnectorKit), les programmes Anchor/Pinocchio, la génération de clients basée sur Codama, les tests LiteSVM/Mollusk/Surfpool et les listes de contrôle de sécurité.
