Комплексный рабочий процесс исправления проблем GitHub с использованием gh, изменений локального кода, сборок/тестов и git push. Используйте, когда вас попросят получить номер проблемы, проверить проблему через gh, внедрить исправление, запустить сборки/тесты XcodeBuildMCP, зафиксировать с закрывающим сообщением и отправить.
