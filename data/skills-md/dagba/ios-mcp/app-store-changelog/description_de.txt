Erstellen Sie benutzerbezogene App Store-Versionshinweise, indem Sie alle benutzerrelevanten Änderungen seit dem letzten Git-Tag (oder einer angegebenen Referenz) sammeln und zusammenfassen. Verwenden Sie diese Option, wenn Sie dazu aufgefordert werden, ein umfassendes Release-Änderungsprotokoll, einen App Store-Text „Was ist neu“ oder Versionshinweise basierend auf dem Git-Verlauf oder Tags zu erstellen.
