Performances, sécurité et bonnes pratiques de Node.js et Fastify de Nearform. Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la refactorisation du code Node.js/Fastify pour garantir des performances, une sécurité et une maintenabilité optimales. Déclencheurs sur des tâches impliquant les API Node.js, les routes Fastify, les opérations de base de données, la gestion des erreurs ou l'optimisation du backend.
