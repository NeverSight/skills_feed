Производительность, безопасность Node.js и Fastify и лучшие практики от Nearform. Этот навык следует использовать при написании, проверке или рефакторинге кода Node.js/Fastify, чтобы обеспечить оптимальную производительность, безопасность и удобство обслуживания. Запускается при выполнении задач, связанных с API Node.js, маршрутами Fastify, операциями с базами данных, обработкой ошибок или оптимизацией серверной части.
