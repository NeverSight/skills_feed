Verwalten Sie MCP-Server in Codex schnell und sicher. Verwenden Sie diese Fähigkeit, wenn der Benutzer MCP-Server hinzufügen, aktualisieren, entfernen, überprüfen oder Fehler beheben möchte, insbesondere über eine GitHub-URL. Der Skill erzwingt die Bereichsauswahl (Projekt „./.codex/config.toml“ vs. global „~/.codex/config.toml“), prüft auf vorhandene Einträge und fordert den Benutzer auf, erforderliche Optionen auszuwählen, bevor er Änderungen vornimmt.
