Управляйте серверами MCP в Codex быстро и безопасно. Используйте этот навык, когда пользователь просит добавить, обновить, удалить, проверить серверы MCP или устранить неполадки, особенно по URL-адресу GitHub. Этот навык обеспечивает выбор области действия (проект `./.codex/config.toml` вместо глобального `~/.codex/config.toml`), проверяет наличие существующих записей и просит пользователя выбрать необходимые параметры перед внесением изменений.
