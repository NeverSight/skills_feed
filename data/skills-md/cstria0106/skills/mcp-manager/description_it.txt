Gestisci i server MCP nel Codex in modo rapido e sicuro. Utilizza questa capacità quando l'utente chiede di aggiungere, aggiornare, rimuovere, ispezionare o risolvere i problemi dei server MCP, in particolare da un URL GitHub. La capacità impone la selezione dell'ambito (progetto `./.codex/config.toml` vs globale `~/.codex/config.toml`), controlla le voci esistenti e chiede all'utente di scegliere le opzioni richieste prima di apportare modifiche.
