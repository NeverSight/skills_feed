Administre servidores MCP en Codex de forma rápida y segura. Utilice esta habilidad cuando el usuario solicite agregar, actualizar, eliminar, inspeccionar o solucionar problemas de servidores MCP, especialmente desde una URL de GitHub. La habilidad impone la selección de alcance (proyecto `./.codex/config.toml` vs global `~/.codex/config.toml`), verifica las entradas existentes y solicita al usuario que elija las opciones requeridas antes de realizar cambios.
