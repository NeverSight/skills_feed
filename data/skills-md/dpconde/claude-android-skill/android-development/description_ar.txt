قم بإنشاء تطبيقات Android ذات جودة إنتاجية باتباع إرشادات البنية الرسمية من Google وأفضل ممارسات NowInAndroid. يُستخدم عند إنشاء تطبيقات Android باستخدام Kotlin، أو Jetpack Compose، أو بنية MVVM، أو حقن تبعية Hilt، أو قاعدة بيانات الغرفة، أو المشروعات متعددة الوحدات. يتم تشغيله عند طلبات إنشاء مشاريع Android، أو شاشات، أو ViewModels، أو مستودعات، أو وحدات ميزات، أو عند السؤال عن أنماط بنية Android.
