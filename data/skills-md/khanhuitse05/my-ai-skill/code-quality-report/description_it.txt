Produce report strutturati di valutazione della qualità del codice per codebase frontend (mobile e Web) o backend (IEEE 730/1016/829, Clean Code, OWASP). Da utilizzare quando l'utente richiede un report sulla qualità del codice, una valutazione della qualità, una revisione della base di codice o desidera un report per frontend/backend o mobile/web/backend.
