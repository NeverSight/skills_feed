يُنتج تقارير تقييم جودة التعليمات البرمجية المنظمة للواجهة الأمامية (الجوال والويب) أو قواعد التعليمات البرمجية للواجهة الخلفية (IEEE 730/1016/829، Clean Code، OWASP). يُستخدم عندما يطلب المستخدم تقرير جودة التعليمات البرمجية، أو تقييم الجودة، أو مراجعة قاعدة التعليمات البرمجية، أو يريد تقريرًا للواجهة الأمامية مقابل الواجهة الخلفية أو الجوال مقابل الويب مقابل الواجهة الخلفية.
