Produit des rapports structurés d'évaluation de la qualité du code pour les bases de code Frontend (Mobile et Web) ou Backend (IEEE 730/1016/829, Clean Code, OWASP). À utiliser lorsque l'utilisateur demande un rapport sur la qualité du code, une évaluation de la qualité, une révision de la base de code ou souhaite un rapport pour le frontend par rapport au backend ou le mobile par rapport au Web par rapport au backend.
