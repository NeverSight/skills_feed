Convertissez des documents Markdown en livres blancs PDF professionnels avec un design de style Apple.
Prise en charge complète de la syntaxe Markdown (blocs de code, tableaux, guillemets, listes, etc.).
Générez automatiquement la couverture, la table des matières, les en-têtes et les pieds de page.
Cas d'utilisation : documents techniques, livres blancs, didacticiels, rapports ; tout Markdown nécessitant une mise en page professionnelle.
