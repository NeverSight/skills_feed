قم بتحويل مستندات Markdown إلى أوراق بيضاء احترافية بتنسيق PDF بتصميم على طراز Apple.
يدعم بناء جملة Markdown الكامل (كتل التعليمات البرمجية، والجداول، وعلامات الاقتباس، والقوائم، وما إلى ذلك).
يُنشئ تلقائيًا الغلاف وجدول المحتويات والرؤوس والتذييلات.
حالات الاستخدام: المستندات الفنية، والمستندات التقنية، والبرامج التعليمية، والتقارير - أي Markdown يحتاج إلى تخطيط احترافي.
