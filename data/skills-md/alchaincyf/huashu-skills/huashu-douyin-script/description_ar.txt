سير عمل إنشاء البرنامج النصي الفيروسي Douyin. عملية كاملة بدءًا من تفكيك مقاطع الفيديو المنافسة وحتى إنشاء البرنامج النصي: تنزيل فيديو Douyin ← تحليل فيديو Gemini ← استخراج الصيغة الفيروسية ← إنشاء البرنامج النصي + لوحة القصة ← تدقيق لغوي بتقنية الذكاء الاصطناعي.
استخدم هذه المهارة عندما يذكر المستخدم "نص Douyin"، أو "التفكيك الفيروسي"، أو "تحليل المنافس"، أو "نص المبيعات"، أو "مادة Qianchuan"، أو "نص البذر"، أو "تفكيك الفيديو"، أو "تحليل فيديو Douyin".
