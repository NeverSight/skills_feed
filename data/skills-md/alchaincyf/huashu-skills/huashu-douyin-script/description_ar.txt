سير عمل إنشاء البرنامج النصي الفيروسي Douyin. عملية كاملة بدءًا من تقسيم مقاطع الفيديو المنافسة إلى إنشاء البرنامج النصي: تنزيل فيديو Douyin ← تحليل فيديو Gemini ← استخراج الصيغة الفيروسية ← إنشاء البرنامج النصي + القصة المصورة ← تدقيق لغوي بتقنية الذكاء الاصطناعي.
استخدم هذه المهارة عندما يذكر المستخدمون "نص Douyin"، أو "الانهيار الفيروسي"، أو "تحليل المنافسين"، أو "نص المبيعات"، أو "مادة Qianchuan"، أو "نص البذر"، أو "تقسيم الفيديو"، أو "تحليل فيديو Douyin".
