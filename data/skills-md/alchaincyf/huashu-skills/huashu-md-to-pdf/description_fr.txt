Convertissez des documents Markdown en livres blancs PDF professionnels avec un design de style Apple.
Prend en charge la syntaxe Markdown complète (blocs de code, tableaux, guillemets, listes, etc.).
Génère automatiquement la couverture, la table des matières, les en-têtes et les pieds de page.
Cas d'utilisation : documents techniques, livres blancs, didacticiels, rapports ; tout Markdown nécessitant une mise en page soignée.
