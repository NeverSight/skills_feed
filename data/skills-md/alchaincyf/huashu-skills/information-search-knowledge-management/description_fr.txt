Effectue automatiquement une recherche multicanal, vérifie les informations et les enregistre dans la base de connaissances lorsqu'il s'agit de nouveaux produits/technologies. À utiliser pour discuter de nouveaux produits (publiés dans les 3 mois), pour vérifier les dernières informations ou pour rechercher des demandes d'utilisateurs. Donne la priorité aux médias technologiques faisant autorité et ignore les sources obsolètes.
