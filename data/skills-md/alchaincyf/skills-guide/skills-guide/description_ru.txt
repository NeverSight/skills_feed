Комплексное руководство по обнаружению, пониманию и установке навыков Claude Code с торговой площадкиkills.sh. Используйте этот навык, когда пользователи хотят (1) просмотреть или выполнить поиск рекомендаций в файлеkills.sh, (2) получить рекомендации по навыкам по ролям (разработчик, маркетолог, дизайнер, менеджер по продукту), (3) понять, как работает конкретный установленный навык, прочитав и объяснив его SKILL.md, (4) изучить анатомию и структуру навыков или (5) найти и установить навыки, соответствующие их конкретным потребностям рабочего процесса. Это НЕ для создания новых навыков (вместо этого используйте мастер-создатель).
