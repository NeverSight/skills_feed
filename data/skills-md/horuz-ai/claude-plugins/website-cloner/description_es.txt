Flujo de trabajo agente para clonar sitios web con fidelidad de píxeles perfecta utilizando subagentes especializados. Úselo cuando el usuario quiera clonar/copiar/replicar un sitio web, crear una página de destino basada en un sitio existente o necesita extraer y recrear el diseño de un sitio web. Incluye orquestación mediante comando de barra diagonal, cuatro subagentes especializados (captador de pantalla, extractor, clonador, revisor de control de calidad) y genera componentes de React con Tailwind CSS y animaciones de movimiento.
