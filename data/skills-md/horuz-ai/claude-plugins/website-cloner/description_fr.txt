Flux de travail agent pour cloner des sites Web avec une fidélité parfaite au pixel près à l'aide de sous-agents spécialisés. À utiliser lorsque l'utilisateur souhaite cloner/copier/répliquer un site Web, créer une page de destination basée sur un site existant ou doit extraire et recréer la conception d'un site Web. Comprend l'orchestration via la commande slash, quatre sous-agents spécialisés (capteur d'écran, extracteur, cloneur, qa-reviewer) et génère des composants React avec Tailwind CSS et des animations de mouvement.
