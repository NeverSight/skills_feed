سير عمل الوكيل لاستنساخ مواقع الويب بدقة مثالية للبيكسل باستخدام وكلاء فرعيين متخصصين. يُستخدم عندما يريد المستخدم استنساخ/نسخ/تكرار موقع ويب، أو إنشاء صفحة مقصودة بناءً على موقع موجود، أو يحتاج إلى استخراج تصميم موقع ويب وإعادة إنشائه. يتضمن التنسيق عبر أمر الشرطة المائلة، وأربعة وكلاء فرعيين متخصصين (مصور الشاشة، والمستخرج، والمستنسخ، ومراجع qa)، ومخرجات مكونات React باستخدام Tailwind CSS والرسوم المتحركة المتحركة.
