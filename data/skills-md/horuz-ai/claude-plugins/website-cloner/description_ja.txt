専用のサブエージェントを使用して、ピクセル完璧な忠実度で Web サイトのクローンを作成するためのエージェント ワークフロー。ユーザーが Web サイトのクローン/コピー/複製を行う場合、既存のサイトに基づいてランディング ページを作成する場合、または Web サイトのデザインを抽出して再作成する必要がある場合に使用します。スラッシュ コマンドによるオーケストレーション、4 つの特殊なサブエージェント (スクリーンショッター、エクストラクター、クローン作成者、QA レビューアー)、Tailwind CSS とモーション アニメーションを使用した React コンポーネントの出力が含まれます。
