Агентский рабочий процесс для клонирования веб-сайтов с точностью до пикселя с использованием специализированных субагентов. Используйте, когда пользователь хочет клонировать/копировать/реплицировать веб-сайт, создать целевую страницу на основе существующего сайта или ему необходимо извлечь и воссоздать дизайн веб-сайта. Включает оркестровку с помощью команды косой черты, четыре специализированных субагента (скриншоттер, экстрактор, клонер, qa-reviewer) и выводит компоненты React с CSS Tailwind и анимацией движения.
