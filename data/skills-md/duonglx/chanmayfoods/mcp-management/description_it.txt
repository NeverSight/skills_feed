Gestisci i server MCP (Model Context Protocol): scopri, analizza ed esegui strumenti/prompt/risorse dai server MCP configurati. Da utilizzare quando si lavora con le integrazioni MCP, è necessario scoprire le funzionalità MCP disponibili, filtrare gli strumenti MCP per attività specifiche, eseguire strumenti MCP a livello di programmazione, accedere alle richieste/risorse MCP o implementare la funzionalità client MCP. Supporta la selezione intelligente degli strumenti, la gestione multi-server e il rilevamento delle capacità efficiente in base al contesto.
