Этот навык следует использовать, когда пользователь говорит «гобелен <URL>», «сплетите <URL>», «помогите мне спланировать <URL>», «извлеките и спланируйте <URL>», «сделайте этот <URL> действенным» или хочет извлечь контент из URL-адреса и создать план действий. Автоматически определяет тип контента (видео YouTube, статьи, PDF) и организует полный рабочий процесс извлечения в план.
