Этот навык следует использовать, когда пользователь предоставляет URL-адрес YouTube и хочет «загрузить расшифровку», «получить титры», «получить субтитры», «расшифровать видео» или извлечь текстовый контент из видео YouTube. Обрабатывает ручные субтитры, автоматически создаваемые субтитры и транскрипцию Whisper в качестве резервного варианта.
