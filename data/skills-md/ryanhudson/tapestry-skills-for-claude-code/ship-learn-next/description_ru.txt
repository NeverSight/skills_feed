Этот навык следует использовать, когда пользователь хочет «превратить это в план», «сделать это действенным», «создать план действий», «реализовать совет» или когда у пользователя есть обучающий контент (расшифровка, статья, руководство) и он хочет конкретных шагов по реализации. Преобразует пассивный контент в готовые к отправке итерации с помощью платформы Ship-Learn-Next.
