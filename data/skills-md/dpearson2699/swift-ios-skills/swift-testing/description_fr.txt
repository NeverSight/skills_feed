Guide du framework de tests Swift pour l'écriture de tests avec @Test, @Suite, #expect, #require, confirmation, tests paramétrés, balises de test, traits, withKnownIssue, tests d'interface utilisateur XCTest, XCUITest, plan de test, moquerie, tests doubles, architecture testable, tests d'instantanés, modèles de test asynchrones, organisation de tests et développement piloté par les tests dans Swift. À utiliser lors de l'écriture ou de la migration de tests avec le framework Swift Testing, de la mise en œuvre de tests paramétrés, de l'utilisation de caractéristiques de test, de la conversion de XCTest en Swift Testing ou de la configuration d'une organisation de test et de modèles moqueurs.
