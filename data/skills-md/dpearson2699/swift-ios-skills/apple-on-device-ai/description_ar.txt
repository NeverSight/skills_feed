يُستخدم عند دمج إطار عمل Foundation Models، أو تنفيذ الذكاء الاصطناعي على الجهاز باستخدام Apple Intelligence، أو إنشاء ميزات الذكاء الاصطناعي لاستدعاء الأدوات، أو العمل مع مخططات الإنشاء الموجهة، أو تحويل النماذج باستخدام Core ML والأدوات الأساسية، أو تشغيل LLMs مفتوحة المصدر على Apple Silicon. يغطي النماذج الأساسية (LanguageModelSession، @Generable، @Guide، SystemLanguageModel، الإخراج المنظم، استدعاء الأدوات)، Core ML (الأدوات الأساسية، تحويل النماذج، القياس الكمي، التحويل، التقليم، المحرك العصبي، MLTensor)، MLX Swift (استدلال المحولات، الذاكرة الموحدة)، وllama.cpp (GGUF، LLM عبر الأنظمة الأساسية).
