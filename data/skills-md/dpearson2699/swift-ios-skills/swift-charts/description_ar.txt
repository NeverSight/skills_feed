تنفيذ أو مراجعة أو تحسين تصورات البيانات باستخدام Swift Charts. يُستخدم عند إنشاء مخططات شريطية أو خطية أو مساحية أو نقطية أو دائرية أو دائرية الشكل؛ عند إضافة تحديد المخطط أو التمرير أو التعليقات التوضيحية؛ عند رسم الوظائف باستخدام BarPlot أو LinePlot أو AreaPlot أو PointPlot المتجه؛ عند تخصيص المحاور أو المقاييس أو وسائل الإيضاح أو تجميع نمط المقدمة؛ أو عند إنشاء تصورات متخصصة مثل الخرائط الحرارية، أو مخططات جانت، أو الأشرطة المكدسة/المجمعة، أو خطوط المؤشرات، أو خطوط العتبة.
