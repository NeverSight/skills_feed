تنفيذ أو مراجعة أو تحسين استمرارية البيانات باستخدام SwiftData. يُستخدم عند تحديد فئات @Model باستخدام @Attribute، أو @Relationship، أو @Transient، أو @Unique، أو @Index؛ عند الاستعلام باستخدام @Query أو #Predicate أو FetchDescriptor أو SortDescriptor؛ عند تكوين ModelContainer وModelContext لـ SwiftUI أو العمل في الخلفية باستخدام @ModelActor؛ عند التخطيط لعمليات ترحيل المخطط باستخدام VersionedSchema وSchemaMigrationPlan؛ عند إعداد مزامنة CloudKit مع ModelConfiguration؛ أو عند التعايش مع البيانات الأساسية أو الترحيل منها.
