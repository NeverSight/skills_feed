تنفيذ أو مراجعة أو تحسين الرسوم المتحركة والانتقالات الخاصة بـ SwiftUI. يُستخدم عند إضافة رسوم متحركة ضمنية أو صريحة باستخدام withAnimation، أو تكوين الرسوم المتحركة الربيعية (.smooth، .snappy، .bouncy)، أو الرسوم المتحركة لمرحلة البناء أو الإطارات الرئيسية باستخدام PhaseAnimator/KeyframeAnimator، أو إنشاء انتقالات البطل باستخدام GeometryEffect المطابق أو TransitionSource المتطابق، أو إضافة تأثيرات رمز SF (ارتداد، نبض، متغير اللون، تنفس، تدوير، تذبذب)، تنفيذ انتقال مخصص أو أنواع CustomAnimation، أو ضمان الرسوم المتحركة تحترم إمكانية الوصولReduceMotion.
