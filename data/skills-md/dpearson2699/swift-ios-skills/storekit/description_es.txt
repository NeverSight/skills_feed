Implemente, revise o mejore las compras y suscripciones dentro de la aplicación usando StoreKit 2. Úselo al crear muros de pago con SubscriptionStoreView o ProductView, procesar transacciones con API de productos y transacciones, verificar derechos, manejar flujos de compras (consumibles, no consumibles, renovables automáticamente), implementar códigos de oferta u ofertas promocionales/recuperables/introductorias, administrar el estado de la suscripción y el estado de renovación, configurar pruebas de StoreKit con archivos de configuración o integrar Family Sharing, Ask to Buy, reembolso manejo y lógica de reintento de facturación.
