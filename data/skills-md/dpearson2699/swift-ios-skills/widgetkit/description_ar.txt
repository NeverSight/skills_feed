تنفيذ أو مراجعة أو تحسين الأدوات والأنشطة المباشرة وعناصر التحكم باستخدام WidgetKit وActivityKit. يُستخدم عند إنشاء عناصر واجهة مستخدم للشاشة الرئيسية أو شاشة القفل أو وضع الاستعداد مع موفري المخطط الزمني؛ عند إنشاء عناصر واجهة مستخدم تفاعلية باستخدام إجراءات Button/Toggle وAppIntent؛ عند إضافة أنشطة مباشرة باستخدام تخطيطات الجزيرة الديناميكية (مدمجة، وبسيطة، وموسعة)؛ عند إنشاء عناصر واجهة مستخدم مركز التحكم باستخدام ControlWidgetButton/ControlWidgetToggle؛ عند تكوين عائلات عناصر واجهة المستخدم، أو تحديث الميزانيات، أو الروابط العميقة، أو عمليات إعادة التحميل المستندة إلى الدفع، أو عرض Liquid Glass؛ أو عند إعداد ملحقات عناصر واجهة المستخدم ومجموعات التطبيقات والاستحقاقات.
