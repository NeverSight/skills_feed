تنفيذ الخرائط وميزات الموقع أو مراجعتها أو تحسينها في تطبيقات iOS/macOS باستخدام MapKit وCoreLocation. يُستخدم عند العمل مع طرق عرض الخريطة، أو التعليقات التوضيحية، أو العلامات، أو الخطوط المتعددة، أو تتبع موقع المستخدم، أو الترميز الجغرافي، أو الترميز الجغرافي العكسي، أو البحث/الإكمال التلقائي، أو الاتجاهات والمسارات، أو تحديد الموقع الجغرافي، أو مراقبة المنطقة، أو تدفقات CLLocationUpdate غير المتزامنة، أو تدفقات ترخيص الموقع. قم بتشغيل أي مهمة تتضمن الخرائط أو الإحداثيات أو العناوين أو الأماكن أو الاتجاهات أو حسابات المسافة أو الميزات المستندة إلى الموقع في تطبيقات Swift.
