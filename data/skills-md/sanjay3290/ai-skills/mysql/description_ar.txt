تنفيذ استعلامات SQL للقراءة فقط مقابل قواعد بيانات MySQL المتعددة. يُستخدم عندما: (1) الاستعلام عن قواعد بيانات MySQL، (2) استكشاف مخططات/جداول قاعدة البيانات، (3) تشغيل استعلامات SELECT لتحليل البيانات، (4) التحقق من محتويات قاعدة البيانات. يدعم اتصالات قاعدة بيانات متعددة مع أوصاف للاختيار التلقائي الذكي. يحظر جميع عمليات الكتابة (INSERT، UPDATE، DELETE، DROP، وما إلى ذلك) من أجل السلامة.
