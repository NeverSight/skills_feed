Выполнение SQL-запросов только для чтения к нескольким базам данных MySQL. Используйте при: (1) запросе к базам данных MySQL, (2) изучении схем/таблиц базы данных, (3) выполнении запросов SELECT для анализа данных, (4) проверке содержимого базы данных. Поддерживает несколько подключений к базе данных с описаниями для интеллектуального автоматического выбора. Блокирует все операции записи (INSERT, UPDATE, DELETE, DROP и т. д.) в целях безопасности.
