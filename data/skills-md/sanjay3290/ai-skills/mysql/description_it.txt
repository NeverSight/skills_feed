Esegui query SQL di sola lettura su più database MySQL. Da utilizzare quando: (1) si eseguono query sui database MySQL, (2) si esplorano schemi/tabelle di database, (3) si eseguono query SELECT per l'analisi dei dati, (4) si controlla il contenuto del database. Supporta più connessioni a database con descrizioni per la selezione automatica intelligente. Blocca tutte le operazioni di scrittura (INSERT, UPDATE, DELETE, DROP, ecc.) per sicurezza.
