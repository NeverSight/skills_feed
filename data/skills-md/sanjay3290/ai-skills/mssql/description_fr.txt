Exécutez des requêtes SQL en lecture seule sur plusieurs bases de données Microsoft SQL Server. À utiliser lorsque : (1) interroger des bases de données MSSQL/SQL Server, (2) explorer des schémas/tables de base de données, (3) exécuter des requêtes SELECT pour l'analyse des données, (4) vérifier le contenu de la base de données. Prend en charge plusieurs connexions de bases de données avec des descriptions pour une sélection automatique intelligente. Bloque toutes les opérations d'écriture (INSERT, UPDATE, DELETE, DROP, etc.) pour des raisons de sécurité.
