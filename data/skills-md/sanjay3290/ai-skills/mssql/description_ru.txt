Выполнение SQL-запросов только для чтения к нескольким базам данных Microsoft SQL Server. Используйте, когда: (1) отправляете запросы к базам данных MSSQL/SQL Server, (2) изучаете схемы/таблицы базы данных, (3) выполняете запросы SELECT для анализа данных, (4) проверяете содержимое базы данных. Поддерживает несколько подключений к базе данных с описаниями для интеллектуального автоматического выбора. Блокирует все операции записи (INSERT, UPDATE, DELETE, DROP и т. д.) в целях безопасности.
