Ejecute consultas SQL de solo lectura en múltiples bases de datos de Microsoft SQL Server. Úselo cuando: (1) consulte bases de datos de MSSQL/SQL Server, (2) explore esquemas/tablas de bases de datos, (3) ejecute consultas SELECT para análisis de datos, (4) verifique el contenido de la base de datos. Admite múltiples conexiones de bases de datos con descripciones para una selección automática inteligente. Bloquea todas las operaciones de escritura (INSERT, UPDATE, DELETE, DROP, etc.) por seguridad.
