Führen Sie schreibgeschützte SQL-Abfragen für mehrere PostgreSQL-Datenbanken aus. Wird verwendet, wenn: (1) PostgreSQL-Datenbanken abgefragt werden, (2) Datenbankschemata/-tabellen untersucht werden, (3) SELECT-Abfragen zur Datenanalyse ausgeführt werden, (4) Datenbankinhalte überprüft werden. Unterstützt mehrere Datenbankverbindungen mit Beschreibungen für eine intelligente automatische Auswahl. Blockiert aus Sicherheitsgründen alle Schreibvorgänge (INSERT, UPDATE, DELETE, DROP usw.).
