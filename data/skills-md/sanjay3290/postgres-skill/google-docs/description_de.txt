Interagieren Sie mit Google Docs – erstellen Sie Dokumente, suchen Sie nach Titel, lesen Sie Inhalte und bearbeiten Sie Text.
Wird verwendet, wenn der Benutzer Folgendes wünscht: ein Google-Dokument erstellen, ein Dokument suchen, Dokumentinhalt lesen, Text zu einem Dokument hinzufügen,
oder Text in einem Dokument ersetzen. Leichte Alternative zum vollständigen Google Workspace MCP-Server mit
eigenständige OAuth-Authentifizierung.
