Создавайте обработчики HTTP Chi в соответствии с соглашениями модульной архитектуры GO (DTO запроса/ответа, оркестровка вариантов использования, обработка ошибок, аннотации Swagger, Fx DI). Используйте при создании обработчиков конечных точек HTTP во внутреннем/модулях/<модуле>/http/chi/handler/ для операций REST (список, создание, обновление, удаление, получение), которым необходимо декодировать запросы, вызывать варианты использования, отображать ответы и обрабатывать ошибки с надлежащим протоколированием и отслеживанием.
