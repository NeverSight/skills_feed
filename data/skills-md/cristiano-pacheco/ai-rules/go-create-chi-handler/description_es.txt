Genere controladores Chi HTTP siguiendo las convenciones de arquitectura modular GO (DTO de solicitud/respuesta, orquestación de casos de uso, manejo de errores, anotaciones de arrogancia, Fx DI). Úselo al crear controladores de puntos finales HTTP en internal/modules/<module>/http/chi/handler/ para operaciones REST (Listar, Crear, Actualizar, Eliminar, Obtener) que necesitan decodificar solicitudes, llamar a casos de uso, asignar respuestas y manejar errores con un registro y seguimiento adecuados.
