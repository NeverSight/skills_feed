Создавайте реализации кэша Go в соответствии с соглашениями о модульной архитектуре GO. Используйте при создании слоев кэша во внутреннем/modules/<module>/cache/ — кэширование состояния пользователя, кэширование сеанса, данные ограничения скорости, временное хранилище данных или любой кэш домена, который использует Redis для быстрого доступа к данным с поддержкой TTL.
