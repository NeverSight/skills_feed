قم بإنشاء تطبيقات Go Cache باتباع اصطلاحات البنية المعيارية لـ GO. يُستخدم عند إنشاء طبقات ذاكرة التخزين المؤقت في Internal/modules/<module>/cache/ - التخزين المؤقت لحالة المستخدم، أو التخزين المؤقت للجلسة، أو بيانات تحديد المعدل، أو تخزين البيانات المؤقتة، أو أي ذاكرة تخزين مؤقت للمجال تستخدم Redis للوصول السريع إلى البيانات مع دعم TTL.
