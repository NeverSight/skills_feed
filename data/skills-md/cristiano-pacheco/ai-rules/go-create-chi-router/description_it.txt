Genera implementazioni di router Chi seguendo le convenzioni dell'architettura modulare GO (router Chi dal pacchetto bricks, Fx DI con interfaccia chi.Route, endpoint REST). Da utilizzare durante la creazione della registrazione della route HTTP per le risorse in internal/modules/<module>/http/chi/router/ incluse le route REST (GET, POST, PUT, DELETE) per operazioni CRUD, endpoint personalizzati, API con versione e gruppi di route.
