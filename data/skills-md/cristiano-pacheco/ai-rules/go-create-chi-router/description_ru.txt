Создавайте реализации маршрутизатора Chi в соответствии с соглашениями о модульной архитектуре GO (маршрутизатор Chi из пакета блоков, Fx DI с интерфейсом chi.Route, конечные точки REST). Используйте при создании регистрации HTTP-маршрута для ресурсов в Internal/modules/<module>/http/chi/router/, включая маршруты REST (GET, POST, PUT, DELETE) для операций CRUD, пользовательских конечных точек, API с поддержкой версий и групп маршрутов.
