Générez des implémentations de routeur Chi en suivant les conventions d'architecture modulaire GO (routeur Chi à partir du package de briques, Fx DI avec interface chi.Route, points de terminaison REST). À utiliser lors de la création d'un enregistrement de route HTTP pour les ressources dans internal/modules/<module>/http/chi/router/, y compris les routes REST (GET, POST, PUT, DELETE) pour les opérations CRUD, les points de terminaison personnalisés, les API versionnées et les groupes de routes.
