Générez des modèles Go GORM en suivant les conventions d'architecture modulaire Go. À utiliser lors de la création ou de la mise à jour de modèles de persistance dans internal/modules/<module>/model/, y compris le mappage de tables, les types de pointeurs nullables, les balises d'index, les types spécifiques à PostgreSQL et les horodatages. Utilisez toujours cette compétence lorsqu'on vous demande de créer un modèle, d'ajouter une structure GORM, de mapper une table de base de données ou de générer des fichiers de modèle.
