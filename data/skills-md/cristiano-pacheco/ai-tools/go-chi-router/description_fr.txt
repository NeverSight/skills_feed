Générez des implémentations de routeur Chi en suivant les conventions d'architecture modulaire Go (routeur Chi à partir du package de briques, Fx DI avec interface chi.Route, points de terminaison REST). Utilisez toujours cette compétence lors de la création ou de la modification de l'enregistrement d'une route HTTP dans internal/modules/<module>/http/chi/router/, y compris tout nouveau fichier de routeur, les routes CRUD (GET, POST, PUT, DELETE), les points de terminaison d'action personnalisés, les API versionnées, les groupes de routes avec middleware ou le câblage des routeurs dans le fx.go d'un module.
