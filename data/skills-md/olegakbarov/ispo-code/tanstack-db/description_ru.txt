Экспертное руководство по реактивной клиентской базе данных TanStack DB с коллекциями, динамическими запросами, оптимистическими мутациями и механизмами синхронизации (ElectricSQL, RxDB, PowerSync). Используйте, когда пользователь говорит «tanstack db», «/tanstack-db», «сначала локальный», «живой запрос», «коллекция», «оптимистическая мутация», «электрический», «механизм синхронизации» или спрашивает о реактивных данных, синхронизации в автономном режиме или в реальном времени.
