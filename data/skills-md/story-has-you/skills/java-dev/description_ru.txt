Комплексные навыки разработки Java на основе рекомендаций Alibaba по кодированию Java (Songshan Edition). Используйте при написании, проверке или рефакторинге кода Java, чтобы обеспечить соответствие лучшим отраслевым практикам. Срабатывает при: (1) написании нового кода Java (файлы .java), (2) анализе существующего кода Java, (3) рефакторинге проектов Java, (4) проектировании базы данных с помощью MySQL, (5) разработке и реализации API, (6) модульном тестировании, (7) параллельном программировании, (8) реализации безопасности или любых задачах разработки Java, требующих соблюдения стандартов кодирования.
