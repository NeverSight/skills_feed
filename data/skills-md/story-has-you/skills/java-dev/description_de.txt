Umfassende Kenntnisse in der Java-Entwicklung basierend auf den Alibaba Java Coding Guidelines (Songshan Edition). Verwenden Sie es beim Schreiben, Überprüfen oder Refactoring von Java-Code, um die Einhaltung der Best Practices der Branche sicherzustellen. Auslöser sind: (1) Schreiben von neuem Java-Code (.java-Dateien), (2) Überprüfen von vorhandenem Java-Code, (3) Refactoring von Java-Projekten, (4) Datenbankdesign mit MySQL, (5) API-Design und -Implementierung, (6) Unit-Tests, (7) gleichzeitige Programmierung, (8) Sicherheitsimplementierung oder alle Java-Entwicklungsaufgaben, die die Einhaltung von Codierungsstandards erfordern.
