Competenze complete di sviluppo Java basate sulle linee guida di codifica Java di Alibaba (edizione Songshan). Da utilizzare durante la scrittura, la revisione o il refactoring del codice Java per garantire la conformità con le migliori pratiche del settore. Si attiva su: (1) Scrittura di nuovo codice Java (file .java), (2) Revisione del codice Java esistente, (3) Refactoring di progetti Java, (4) Progettazione di database con MySQL, (5) Progettazione e implementazione di API, (6) Test unitari, (7) Programmazione simultanea, (8) Implementazione della sicurezza o qualsiasi attività di sviluppo Java che richieda il rispetto degli standard di codifica.
