Compétence complète en développement Java basée sur les directives de codage Java d'Alibaba (édition Songshan). À utiliser lors de l'écriture, de la révision ou de la refactorisation du code Java pour garantir la conformité aux meilleures pratiques du secteur. Déclencheurs sur : (1) l'écriture d'un nouveau code Java (fichiers .java), (2) la révision du code Java existant, (3) la refactorisation de projets Java, (4) la conception de bases de données avec MySQL, (5) la conception et la mise en œuvre d'API, (6) les tests unitaires, (7) la programmation simultanée, (8) la mise en œuvre de la sécurité ou toute tâche de développement Java nécessitant le respect des normes de codage.
