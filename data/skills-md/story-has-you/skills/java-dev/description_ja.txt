Alibaba Java コーディング ガイドライン (松山版) に基づいた包括的な Java 開発スキル。 Java コードを作成、レビュー、またはリファクタリングするときに使用して、業界のベスト プラクティスに準拠していることを確認します。トリガー: (1) 新しい Java コード (.java ファイル) の作成、(2) 既存の Java コードのレビュー、(3) Java プロジェクトのリファクタリング、(4) MySQL を使用したデータベース設計、(5) API 設計と実装、(6) 単体テスト、(7) 同時プログラミング、(8) セキュリティ実装、またはコーディング標準への準拠が必要な Java 開発タスク。
