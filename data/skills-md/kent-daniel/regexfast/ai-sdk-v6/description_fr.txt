Guide pour créer des applications basées sur l'IA à l'aide du Vercel AI SDK v6. À utiliser lors du développement avec generateText, streamText, useChat, les appels d'outils, les agents, la génération de sorties structurées, l'intégration MCP ou toute fonctionnalité basée sur LLM dans les applications TypeScript/JavaScript. Couvre les implémentations de React, Next.js, Vue, Svelte et Node.js.
