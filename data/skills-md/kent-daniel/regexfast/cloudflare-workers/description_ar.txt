أنشئ تطبيقات ذات حالة بدون خادم باستخدام Cloudflare Workers والكائنات الدائمة. يُستخدم عند إنشاء تطبيقات تعاونية في الوقت الفعلي، أو أنظمة دردشة، أو ألعاب متعددة اللاعبين، أو خوادم WebSocket، أو محددات الأسعار، أو أي تطبيق يتطلب التنسيق بين العملاء، أو الحالة المستمرة، أو المهام المجدولة مع منصة الحوسبة المتطورة الخاصة بـ Cloudflare.
