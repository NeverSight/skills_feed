Utilisez TickTick via l'interface de ligne de commande « ticktick », y compris l'authentification, les flux de lecture/requête et les mutations sécurisées pour les tâches, les projets, les dossiers, les colonnes, les balises, les habitudes, les informations utilisateur, les analyses de focus et les charges utiles de synchronisation. À utiliser lorsque les utilisateurs demandent à exécuter les commandes du terminal TickTick, à analyser la sortie JSON de la CLI TickTick, à résoudre les ID ou à corriger les échecs de configuration/d'authentification de la CLI TickTick.
