Patrones de prueba de backend de Python con pytest para aplicaciones FastAPI. Úselo al escribir pruebas de Python: pruebas unitarias para servicios y repositorios, pruebas de integración para puntos finales API con httpx.AsyncClient, creación de accesorios, configuración de fábrica con factory_boy, pruebas asíncronas con pytest-asyncio, estrategias de burla y pruebas parametrizadas. Cubre la organización de las pruebas (pruebas/unidad, pruebas/integración), la jerarquía del concurso y los requisitos de cobertura. NO cubre pruebas de interfaz (use patrones de prueba de reacción) ni pruebas de navegador E2E (use pruebas e2e).
