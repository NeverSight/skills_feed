Шаблоны внутреннего тестирования Python с помощью pytest для приложений FastAPI. Используйте при написании тестов Python: модульные тесты для сервисов и репозиториев, интеграционные тесты для конечных точек API с httpx.AsyncClient, создание приспособлений, заводскую настройку с помощью Factory_boy, асинхронное тестирование с pytest-asyncio, стратегии макетирования и параметризованные тесты. Охватывает организацию тестирования (тесты/модуль, тесты/интеграция), иерархию конфликтов и требования к покрытию. НЕ охватывает тесты внешнего интерфейса (используйте шаблоны реагирования) или тесты браузера E2E (используйте e2e-тестирование).
