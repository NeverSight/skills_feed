قائمة مراجعة مراجعة التعليمات البرمجية التي تركز على الأمان وأنماط المسح الآلي. يُستخدم عند مراجعة طلبات السحب بحثًا عن مشكلات أمنية، أو تدقيق رمز المصادقة/الترخيص، أو التحقق من أهم 10 نقاط ضعف في OWASP، أو التحقق من صحة المدخلات. يغطي منع حقن SQL، وحماية XSS، ورموز CSRF، ومراجعة تدفق المصادقة، واكتشاف الأسرار، وفحص ثغرات التبعية، وأنماط الترميز الآمنة لـ Python (FastAPI) وReact. لا يغطي أمان النشر (استخدم أفضل ممارسات عامل الإرساء) أو التعامل مع الحوادث (استخدم الاستجابة للحوادث).
