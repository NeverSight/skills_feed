تخطيط المشروع وتفصيل الميزات لمشاريع Python/React الكاملة. يُستخدم أثناء مرحلة التخطيط عند تقسيم طلبات الميزات أو قصص المستخدمين أو متطلبات المنتج إلى خطط التنفيذ. يرشد تحديد الملفات والوحدات المتأثرة، ويحدد معايير القبول، ويقيم المخاطر، ويقدر التعقيد العام. ينتج خرائط الوحدة وتقييمات المخاطر ومعايير القبول. لا يغطي قرارات الهندسة المعمارية (استخدم بنية النظام)، أو التنفيذ (استخدم خبير الواجهة الخلفية python أو خبير الواجهة الأمامية التفاعلية)، أو تحليل المهام الذرية (استخدم تحليل المهام).
