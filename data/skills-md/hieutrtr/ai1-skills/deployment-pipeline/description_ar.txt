إجراءات النشر وتكوين خطوط أنابيب CI/CD لمشاريع Python/React. يُستخدم عند النشر إلى التدريج أو الإنتاج، أو إنشاء خطوط أنابيب CI/CD باستخدام إجراءات GitHub، أو استكشاف أخطاء فشل النشر وإصلاحها، أو التخطيط لعمليات التراجع. يغطي مراحل خط الأنابيب (الإنشاء/الاختبار/التدريج/الإنتاج)، وتعزيز البيئة، والتحقق من صحة ما قبل النشر، والفحوصات الصحية، ونشر الكناري، وإجراءات التراجع، وسير عمل إجراءات GitHub. لا يغطي بناء صورة Docker (استخدم أفضل ممارسات عامل الإرساء) أو الاستجابة للحوادث (استخدم الاستجابة للحوادث).
