Процедуры развертывания и настройка конвейера CI/CD для проектов Python/React. Используйте при развертывании в промежуточной или рабочей среде, создании конвейеров CI/CD с помощью действий GitHub, устранении сбоев развертывания или планировании откатов. Охватывает этапы конвейера (сборка/тестирование/промежуточное исполнение/производство), продвижение среды, проверку перед развертыванием, проверки работоспособности, канареечное развертывание, процедуры отката и рабочие процессы GitHub Actions. НЕ охватывает создание образов Docker (используйте лучшие практики Docker) или реагирование на инциденты (используйте реагирование на инциденты).
