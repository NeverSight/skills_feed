Декомпозируйте цели высокого уровня на атомарные задачи реализации для проектов Python/React. Используйте при разбивке больших функций, изменениях нескольких файлов или задачах, требующих более 3 шагов. Создает независимо проверяемые задачи с условиями выполнения, путями к файлам, оценками сложности и явным упорядочением. Создает постоянные файлы задач (task_plan.md, Progress.md) для отслеживания состояния в контекстных окнах. НЕ охватывает высокоуровневое планирование (используйте планировщик проектов) или архитектурные решения (используйте системную архитектуру).
