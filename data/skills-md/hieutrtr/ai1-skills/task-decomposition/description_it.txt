Scomponi gli obiettivi di alto livello in attività di implementazione atomiche per progetti Python/React. Da utilizzare quando si suddividono funzionalità di grandi dimensioni, modifiche a più file o attività che richiedono più di 3 passaggi. Produce attività verificabili in modo indipendente con condizioni di completamento, percorsi di file, stime di complessità e ordinamento esplicito. Crea file di attività persistenti (task_plan.md, progress.md) per tenere traccia dello stato nelle finestre di contesto. NON copre la pianificazione di alto livello (utilizzare il pianificatore di progetto) o le decisioni sull'architettura (utilizzare l'architettura di sistema).
