Docker-Containerisierungsmuster für Python/React-Projekte. Verwenden Sie es, wenn Sie Docker-Dateien erstellen oder ändern, die Bildgröße optimieren, Docker Compose für die lokale Entwicklung einrichten oder die Containersicherheit erhöhen. Behandelt mehrstufige Builds für Python (python:3.12-slim) und React (node:20-alpine -> nginx:alpine), Layer-Optimierung, .dockerignore, Nicht-Root-Benutzer, Sicherheitsscans mit Trivy, Docker Compose für Entwickler (Backend + Frontend + PostgreSQL + Redis) und Image-Tagging-Strategie. Deckt NICHT die Deployment-Orchestrierung ab (verwenden Sie die Deployment-Pipeline).
