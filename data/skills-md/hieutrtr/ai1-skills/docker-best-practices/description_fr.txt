Modèles de conteneurisation Docker pour les projets Python/React. À utiliser lors de la création ou de la modification de fichiers Docker, de l'optimisation de la taille de l'image, de la configuration de Docker Compose pour le développement local ou du renforcement de la sécurité des conteneurs. Couvre les builds en plusieurs étapes pour Python (python : 3.12-slim) et React (node : 20-alpine -> nginx : alpine), l'optimisation des couches, .dockerignore, l'utilisateur non root, l'analyse de sécurité avec Trivy, Docker Compose pour le développement (backend + frontend + PostgreSQL + Redis) et la stratégie de balisage d'images. Ne couvre PAS l’orchestration du déploiement (utilisez le pipeline de déploiement).
