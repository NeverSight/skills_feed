Modelli di containerizzazione Docker per progetti Python/React. Da utilizzare durante la creazione o la modifica di Dockerfile, l'ottimizzazione delle dimensioni dell'immagine, la configurazione di Docker Compose per lo sviluppo locale o il rafforzamento della sicurezza del contenitore. Copre build multi-fase per Python (python:3.12-slim) e React (node:20-alpine -> nginx:alpine), ottimizzazione dei livelli, .dockerignore, utente non root, scansione di sicurezza con Trivy, Docker Compose per sviluppatori (backend + frontend + PostgreSQL + Redis) e strategia di tagging delle immagini. NON copre l'orchestrazione della distribuzione (utilizzare la pipeline di distribuzione).
