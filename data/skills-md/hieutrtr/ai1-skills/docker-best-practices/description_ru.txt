Шаблоны контейнеризации Docker для проектов Python/React. Используйте при создании или изменении файлов Dockerfile, оптимизации размера образа, настройке Docker Compose для локальной разработки или усилении безопасности контейнера. Охватывает многоэтапные сборки для Python (python:3.12-slim) и React (node:20-alpine -> nginx:alpine), оптимизацию слоев, .dockerignore, пользователя без полномочий root, сканирование безопасности с помощью Trivy, Docker Compose для разработчиков (бэкэнд + интерфейс + PostgreSQL + Redis) и стратегию тегирования изображений. НЕ охватывает оркестрацию развертывания (используйте конвейер развертывания).
