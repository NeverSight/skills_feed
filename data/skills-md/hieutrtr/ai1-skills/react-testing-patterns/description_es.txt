Reaccione los patrones de prueba de componentes y enlaces con Testing Library y Vitest. Úselo al escribir pruebas para componentes de React, enlaces personalizados o lógica de recuperación de datos. Cubre pruebas de representación de componentes, simulación de interacción del usuario, pruebas de estado asíncrono, MSW para simulación de API, pruebas de enlace con renderHook, afirmaciones de accesibilidad y pautas de prueba de instantáneas. NO cubre pruebas E2E (use e2e-testing) ni la aplicación del flujo de trabajo TDD (use tdd-workflow).
