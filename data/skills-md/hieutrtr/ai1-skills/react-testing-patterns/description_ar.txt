تفاعل مع أنماط اختبار المكونات والربط باستخدام مكتبة الاختبار وVitest. يُستخدم عند كتابة اختبارات لمكونات React أو الخطافات المخصصة أو منطق جلب البيانات. يغطي اختبارات عرض المكونات، ومحاكاة تفاعل المستخدم، واختبار حالة عدم المزامنة، وMSW للسخرية من واجهة برمجة التطبيقات (API)، واختبار الربط باستخدام renderHook، وتأكيدات إمكانية الوصول، وإرشادات اختبار اللقطة. لا يغطي اختبارات E2E (استخدم اختبار e2e) أو فرض سير عمل TDD (استخدم سير عمل tdd).
