コンポーネントに反応し、テスト ライブラリと Vitest を使用してテスト パターンをフックします。 React コンポーネント、カスタム フック、またはデータ フェッチ ロジックのテストを作成するときに使用します。コンポーネントのレンダリング テスト、ユーザー インタラクション シミュレーション、非同期状態テスト、API モック用の MSW、renderHook を使用したフック テスト、アクセシビリティ アサーション、およびスナップショット テストのガイドラインをカバーします。 E2E テスト (e2e-testing を使用) または TDD ワークフローの強制 (tdd-workflow を使用) は対象外です。
