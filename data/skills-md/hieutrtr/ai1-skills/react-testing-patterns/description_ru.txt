Реагируйте на компоненты и подключайте шаблоны тестирования с помощью библиотеки тестирования и Vitest. Используйте при написании тестов для компонентов React, пользовательских перехватчиков или логики извлечения данных. Охватывает тесты рендеринга компонентов, моделирование взаимодействия с пользователем, тестирование асинхронного состояния, MSW для макетирования API, тестирование перехвата с помощью renderHook, утверждения доступности и рекомендации по тестированию моментальных снимков. НЕ охватывает тесты E2E (используйте e2e-тестирование) или соблюдение рабочего процесса TDD (используйте tdd-workflow).
