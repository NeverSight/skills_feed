أنماط تنفيذ الواجهة الأمامية لـ React/TypeScript. يُستخدم أثناء مرحلة التنفيذ عند إنشاء أو تعديل مكونات React أو الروابط المخصصة أو الصفحات أو منطق جلب البيانات باستخدام استعلام TanStack أو النماذج أو التوجيه. يغطي بنية المكونات، وقواعد الخطافات، وتصميم الخطاف المخصص (useAuth، useDebounce، usePagination)، واصطلاحات الوضع الصارم لـ TypeScript، ومعالجة النماذج، ومتطلبات إمكانية الوصول، وبنية المشروع. لا يغطي الاختبار (استخدم أنماط اختبار التفاعل)، أو اختبار E2E (استخدم اختبار e2e)، أو النشر.
