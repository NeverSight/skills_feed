Modèles d'implémentation de l'interface React/TypeScript. À utiliser pendant la phase d'implémentation lors de la création ou de la modification de composants React, de hooks personnalisés, de pages, de logique de récupération de données avec TanStack Query, de formulaires ou de routage. Couvre la structure des composants, les règles de hooks, la conception de hooks personnalisés (useAuth, useDebounce, usePagination), les conventions en mode strict TypeScript, la gestion des formulaires, les exigences d'accessibilité et la structure du projet. Ne couvre PAS les tests (utilisez les modèles de test de réaction), les tests E2E (utilisez les tests e2e) ou le déploiement.
