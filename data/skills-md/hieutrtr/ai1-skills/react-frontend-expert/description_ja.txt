React/TypeScript フロントエンド実装パターン。 React コンポーネント、カスタム フック、ページ、TanStack Query を使用したデータ フェッチ ロジック、フォーム、またはルーティングを作成または変更する実装フェーズ中に使用します。コンポーネント構造、フック ルール、カスタム フック設計 (useAuth、useDebounce、usePagination)、TypeScript の厳密モード規則、フォーム処理、アクセシビリティ要件、プロジェクト構造について説明します。テスト (react-testing-patterns を使用)、E2E テスト (e2e-testing を使用)、またはデプロイメントは対象外です。
