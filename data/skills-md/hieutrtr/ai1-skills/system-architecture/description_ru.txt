Руководство по системной архитектуре для полнофункциональных проектов Python/React. Используйте на этапе проектирования при принятии архитектурных решений — границ компонентов, проектирования уровня обслуживания, шаблонов потоков данных, планирования схемы базы данных и анализа технологических компромиссов. Охватывает архитектуру уровней FastAPI (маршруты/сервисы/репозитории/модели), иерархию компонентов React, управление состоянием и сквозные проблемы (аутентификация, ошибки, ведение журнала). Создает архитектурную документацию и ADR. НЕ охватывает реализацию (используйте python-backend-expert или response-frontend-expert) или разработку контракта API (используйте api-design-patterns).
