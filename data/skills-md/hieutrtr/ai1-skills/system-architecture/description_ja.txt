Python/React フルスタック プロジェクトのシステム アーキテクチャ ガイダンス。設計段階で、コンポーネントの境界、サービス層の設計、データ フロー パターン、データベース スキーマの計画、テクノロジ トレードオフ分析などのアーキテクチャ上の決定を行うときに使用します。 FastAPI レイヤー アーキテクチャ (ルート/サービス/リポジトリ/モデル)、React コンポーネント階層、状態管理、および横断的な問題 (認証、エラー、ロギング) をカバーします。アーキテクチャ文書と ADR を作成します。実装 (python-backend-expert または React-frontend-expert を使用) または API コントラクト設計 (api-design-patterns を使用) はカバーしません。
