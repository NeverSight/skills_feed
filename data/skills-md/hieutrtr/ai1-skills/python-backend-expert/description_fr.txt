Modèles d'implémentation backend Python pour les applications FastAPI avec SQLAlchemy 2.0, Pydantic v2 et les modèles asynchrones. À utiliser pendant la phase de mise en œuvre lors de la création ou de la modification de points de terminaison FastAPI, de modèles Pydantic, de modèles SQLAlchemy, de couches de service ou de classes de référentiel. Couvre la gestion des sessions asynchrones, l'injection de dépendances via Depends(), la gestion des erreurs en couches et les migrations Alembic. Ne couvre PAS les tests (utilisez les modèles pytest), le déploiement (utilisez le pipeline de déploiement) ou les mécanismes du framework FastAPI comme le middleware et les WebSockets (utilisez les modèles fastapi).
