Patrones de implementación de backend de Python para aplicaciones FastAPI con SQLAlchemy 2.0, Pydantic v2 y patrones asíncronos. Úselo durante la fase de implementación al crear o modificar puntos finales FastAPI, modelos Pydantic, modelos SQLAlchemy, capas de servicios o clases de repositorio. Cubre la gestión de sesiones asíncronas, la inyección de dependencias mediante Depends(), el manejo de errores en capas y las migraciones de Alembic. NO cubre pruebas (use patrones de pytest), implementación (use canalización de implementación) ni mecánicas del marco FastAPI como middleware y WebSockets (use patrones fastapi).
