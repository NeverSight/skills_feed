أنماط تنفيذ الواجهة الخلفية لـ Python لتطبيقات FastAPI مع SQLAlchemy 2.0 وPydantic v2 وأنماط غير متزامنة. يُستخدم أثناء مرحلة التنفيذ عند إنشاء أو تعديل نقاط نهاية FastAPI أو نماذج Pydantic أو نماذج SQLAlchemy أو طبقات الخدمة أو فئات المستودع. يغطي إدارة الجلسة غير المتزامنة، وحقن التبعية عبر Depends()، ومعالجة الأخطاء ذات الطبقات، والترحيلات Alembic. لا يغطي الاختبار (استخدم أنماط pytest)، أو النشر (استخدم خط أنابيب النشر)، أو آليات إطار عمل FastAPI مثل البرامج الوسيطة وWebSockets (استخدم أنماط fastapi).
