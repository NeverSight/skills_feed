مهارة تطوير Flutter و Dart المتخصصة لبناء تطبيقات جميلة وفعالة وقابلة للصيانة. يُستخدم عندما يطلب المستخدمون مهام تطوير Flutter/Dart بما في ذلك إنشاء التطبيقات أو عناصر واجهة المستخدم أو الشاشات أو تنفيذ الميزات أو تصحيح الأخطاء أو الاختبار أو إدارة الحالة أو التنقل أو السمات أو التخطيطات أو العمل مع مشاريع Flutter. يغطي الأنظمة الأساسية للجوال والويب وسطح المكتب مع أفضل الممارسات الحديثة وتصميم المواد وواجهة المستخدم سريعة الاستجابة ومعايير جودة التعليمات البرمجية.
