Лучший автономный конвейер разработки. Сочетает в себе wavybaby (проверка CoVe, обнаружение навыков, инструменты MCP) + GSD (дорожные карты, этапы, планы, обнаружение, отслеживание состояния) + Ralph (автономный контур с автоматическими выключателями). Создает PRD, оснащает себя лучшими инструментами, загружает полную структуру GSD .planning/, затем запускает Ralph для автономного выполнения каждого плана с использованием кода, проверенного CoVe, до тех пор, пока не будет завершена контрольная точка.
