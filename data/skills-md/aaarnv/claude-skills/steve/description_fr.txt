Le pipeline de développement autonome ultime. Combine wavybaby (vérification CoVe, découverte de compétences, outils MCP) + GSD (feuilles de route, phases, plans, découverte, suivi d'état) + Ralph (boucle autonome avec disjoncteurs). Génère un PRD, s'équipe des meilleurs outils, démarre une structure GSD .planning/ complète, puis exécute Ralph pour exécuter de manière autonome chaque plan avec du code vérifié par CoVe jusqu'à ce que le jalon soit terminé.
