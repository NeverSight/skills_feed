Configura un circuito de desarrollo autónomo de Ralph para cualquier proyecto. Primero genera un PRD completo a partir de la descripción del usuario y luego deriva un plan de tareas a partir de él. Envuelve Claude Code en un bucle inteligente mientras es verdadero con disyuntores, detección de salida, persistencia de sesión y seguimiento del progreso. Úselo cuando desee que Claude trabaje de forma autónoma en una lista de tareas hasta que termine.
