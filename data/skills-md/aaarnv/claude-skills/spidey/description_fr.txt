Met en place une boucle de développement autonome Ralph pour tout projet. Génère d'abord un PRD complet à partir de la description de l'utilisateur, puis en dérive un plan de tâches. Enveloppe Claude Code dans une boucle intelligente while-true avec disjoncteurs, détection de sortie, persistance de session et suivi de la progression. À utiliser lorsque vous souhaitez que Claude travaille de manière autonome sur une liste de tâches jusqu'à ce qu'il ait terminé.
