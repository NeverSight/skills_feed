Richtet für jedes Projekt eine autonome Ralph-Entwicklungsschleife ein. Erstellt zunächst eine vollständige PRD aus der Beschreibung des Benutzers und leitet daraus dann einen Aufgabenplan ab. Schließt Claude Code in eine intelligente While-True-Schleife mit Leistungsschaltern, Beendigungserkennung, Sitzungspersistenz und Fortschrittsverfolgung ein. Verwenden Sie diese Option, wenn Sie möchten, dass Claude eine Aufgabenliste selbstständig durcharbeitet, bis er fertig ist.
