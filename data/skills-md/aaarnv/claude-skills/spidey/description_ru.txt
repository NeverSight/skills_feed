Устанавливает автономный цикл разработки Ralph для любого проекта. Сначала генерирует полный PRD на основе описания пользователя, затем на его основе составляет план задач. Обертывает Claude Code в интеллектуальный цикл while-true с автоматическими выключателями, обнаружением выхода, сохранением сеанса и отслеживанием прогресса. Используйте, если вы хотите, чтобы Клод самостоятельно работал над списком задач до тех пор, пока не закончит их.
