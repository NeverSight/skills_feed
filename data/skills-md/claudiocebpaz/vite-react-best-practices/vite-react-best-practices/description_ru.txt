Комплексные рекомендации по производительности, архитектуре и развертыванию React и Vite SPA. Используйте этот навык при создании, проверке или рефакторинге приложений React, созданных с помощью Vite (SPA). Охватывает конфигурации сборки, специфичные для Vite, требования к статическому хостингу и основные шаблоны производительности React.
