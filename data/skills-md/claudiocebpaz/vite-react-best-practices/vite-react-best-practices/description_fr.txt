Directives complètes sur les performances, l'architecture et le déploiement de React et Vite SPA. Utilisez cette compétence lors de la création, de la révision ou de la refactorisation d'applications React créées avec Vite (SPA). Couvre les configurations de build spécifiques à Vite, les exigences d'hébergement statique et les principaux modèles de performances de React.
