Configuration et utilisation de PyTiDB (pytidb) pour TiDB à partir de Python. Couvre la connexion, la modélisation de table (TableModel), CRUD, le SQL brut, les transactions, la recherche vectorielle/texte intégral/hybride, l'intégration automatique, les fonctions d'intégration personnalisées et les modèles/extraits de référence (vecteur/hybride/image) ainsi que des exemples orientés agent (RAG/mémoire/text2sql).
