Guía para usar el controlador TiDB Cloud Serverless (Beta) en entornos Node.js, sin servidor y de borde. Úselo cuando se conecte a TiDB Cloud Starter/Essential a través de HTTP con @tidbcloud/serverless, o cuando se integre con adaptadores sin servidor Prisma/Kysely/Drizzle en Vercel/Cloudflare/Netlify/Deno/Bun. Utilice esta habilidad para configurar controladores sin servidor y obtener orientación sobre el tiempo de ejecución perimetral.
