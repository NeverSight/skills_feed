Anleitung zur Verwendung des TiDB Cloud Serverless Driver (Beta) in Node.js-, serverlosen und Edge-Umgebungen. Wird verwendet, wenn eine Verbindung zu TiDB Cloud Starter/Essential über HTTP mit @tidbcloud/serverless hergestellt wird oder wenn die serverlosen Adapter Prisma/Kysely/Drizzle in Vercel/Cloudflare/Netlify/Deno/Bun integriert werden. Verwenden Sie diesen Skill für die Einrichtung serverloser Treiber und die Anleitung zur Edge-Laufzeit.
