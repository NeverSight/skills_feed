Conseils pour l'utilisation du pilote TiDB Cloud Serverless (bêta) dans les environnements Node.js, sans serveur et Edge. À utiliser lors de la connexion à TiDB Cloud Starter/Essential via HTTP avec @tidbcloud/serverless, ou lors de l'intégration avec les adaptateurs sans serveur Prisma/Kysely/Drizzle dans Vercel/Cloudflare/Netlify/Deno/Bun. Utilisez cette compétence pour la configuration du pilote sans serveur et les conseils d’exécution Edge.
