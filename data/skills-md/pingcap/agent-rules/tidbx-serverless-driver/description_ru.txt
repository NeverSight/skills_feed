Руководство по использованию бессерверного драйвера TiDB Cloud (бета) в Node.js, бессерверных и пограничных средах. Используйте при подключении к TiDB Cloud Starter/Essential через HTTP с помощью @tidbcloud/serverless или при интеграции с бессерверными адаптерами Prisma/Kysely/Drizzle в Vercel/Cloudflare/Netlify/Deno/Bun. Используйте этот навык для настройки бессерверных драйверов и управления работой периферийных устройств.
