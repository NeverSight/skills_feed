كتابة ومراجعة وتكييف SQL لـ TiDB مع المعالجة الصحيحة لاختلافات TiDB-v-MySQL (نوع VECTOR + فهارس/وظائف المتجهات، البحث عن النص الكامل، AUTO_RANDOM، المعاملات المتفائلة/المتشائمة، المفاتيح الخارجية، طرق العرض، قيود DDL، وميزات MySQL غير المدعومة مثل الإجراءات/المشغلات/الأحداث/الهندسة/المكانية). يُستخدم عند إنشاء SQL الذي يجب تشغيله على TiDB، أو ترحيل MySQL SQL إلى TiDB، أو تصحيح أخطاء توافق TiDB SQL.
