Приложение «Эксперт по безопасности» для обнаружения уязвимостей, проверки кода и руководства по практическим навыкам безопасности. OWASP Top 10, аутентификация, авторизация, криптография, раскрытие секретов. Использование для аудита безопасности, проверки разумного кода, концепции защищенных функций или устранения ошибок.
