استخدم Shopify Polaris Web Components (s-* العناصر المخصصة) لواجهة مستخدم App Home. يُستخدم عند إنشاء أسطح App Home (وليس التطبيقات المضمنة)، وتصميم واجهة المستخدم باستخدام s-page، وs-section، وs-stack، وs-box، وs-button، ومكونات s-* الأخرى. لا تستخدم @shopify/polaris React - يتطلب تطبيق App Home مكونات ويب.
