Подготавливает ветки запросов на включение, сохраняя изменения, создавая ветки функций из основных, просматривая изменения, выполняя проверки качества кода только для измененных файлов, генерируя обычные сообщения о фиксации и отправляя их. Используйте при подготовке PR, создании ветки для запроса на включение или фиксации изменений после обычных коммитов.
