beautiful-mermaid 라이브러리를 사용하여 아름다운 인어 다이어그램을 SVG 또는 ASCII 아트로 렌더링하세요.
15개 이상의 테마, 5가지 다이어그램 유형(플로우차트, 시퀀스, 상태, 클래스, ER) 및 초고속 렌더링을 지원합니다.

다음과 같은 경우에 이 기술을 사용하십시오.
1. 사용자가 "인어 다이어그램 렌더링"을 요청하거나 .mmd 파일을 제공합니다.
2. 사용자가 "플로우차트/시퀀스 다이어그램/상태 다이어그램 생성"을 요청합니다.
3. 사용자가 "테마를 적용"하거나 "다이어그램을 아름답게 꾸미고" 싶어합니다.
4. 사용자는 "여러 다이어그램을 일괄 처리"해야 합니다.
5. 사용자가 "ASCII 다이어그램" 또는 "터미널 친화적인 다이어그램"을 언급합니다.
6. 사용자가 아키텍처, 워크플로우 또는 데이터 모델을 시각화하기를 원합니다.
