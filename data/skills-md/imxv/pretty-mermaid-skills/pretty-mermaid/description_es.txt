Represente hermosos diagramas de sirenas como arte SVG o ASCII utilizando la biblioteca beautiful-mermaid.
Admite más de 15 temas, 5 tipos de diagramas (diagrama de flujo, secuencia, estado, clase, ER) y renderizado ultrarrápido.

Utilice esta habilidad cuando:
1. El usuario solicita "presentar un diagrama de sirena" o proporciona archivos .mmd
2. El usuario solicita "crear un diagrama de flujo/diagrama de secuencia/diagrama de estado"
3. El usuario quiere "aplicar un tema" o "embellecer un diagrama"
4. El usuario necesita "procesar por lotes varios diagramas"
5. El usuario menciona "diagrama ASCII" o "diagrama compatible con terminal"
6. El usuario quiere visualizar arquitectura, flujos de trabajo o modelos de datos.
