Définissez les profils clients idéaux et les personnalités des acheteurs avec des cadres structurés. À utiliser lorsque l'utilisateur pose des questions sur le profil client idéal, l'ICP, la personnalité de l'acheteur, la définition du public cible, la segmentation de la clientèle, la recherche client, les entretiens avec les utilisateurs ou la création de personnalité. Les expressions déclencheurs incluent « profil client idéal », « ICP », « personnalité de l'acheteur », « qui est mon client », « public cible », « profil client », « personnalité », « segmentation client », « recherche utilisateur », « entretien client », « qui dois-je cibler ».
