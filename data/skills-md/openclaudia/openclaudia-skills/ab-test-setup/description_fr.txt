Concevoir, planifier et analyser des tests A/B avec une rigueur statistique. À utiliser lorsque l'utilisateur pose des questions sur les tests A/B, les tests fractionnés, la conception des expériences, la signification statistique, le calcul de la taille de l'échantillon, la durée des tests, les tests multivariés ou les expériences de conversion. Les expressions de déclenchement incluent « test A/B », « test fractionné », « expérience », « signification statistique », « taille de l'échantillon », « durée du test », « quelle version gagne », « expérience de conversion », « test d'hypothèse », « test de variantes ».
