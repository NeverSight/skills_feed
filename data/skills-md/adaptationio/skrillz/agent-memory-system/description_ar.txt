بنية الذاكرة المستمرة لوكلاء الذكاء الاصطناعي عبر الجلسات. الذاكرة العرضية (الأحداث الماضية)، الذاكرة الإجرائية (المهارات المكتسبة)، الذاكرة الدلالية (الرسم البياني المعرفي)، الذاكرة قصيرة المدى (السياق النشط). يُستخدم عند تنفيذ المثابرة عبر الجلسات، أو تعلم المهارات، أو الحفاظ على السياق، أو التخصيص، أو بناء أنظمة ذكاء اصطناعي قابلة للتكيف حقًا مع ذاكرة طويلة المدى.
