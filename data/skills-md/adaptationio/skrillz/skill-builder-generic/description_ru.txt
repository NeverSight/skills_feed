Универсальное руководство по созданию готовых навыков Claude Code для любого проекта. Включает 6-этапный рабочий процесс (понимание, планирование, инициализация, редактирование, упаковка, итерация), дизайн с прогрессивным раскрытием информации, шаблоны исходных материалов YAML, сценарии проверки, справочные шаблоны организации и 10 инноваций, проверенных сообществом. Используйте при создании новых навыков Claude Code, преобразовании документации в навыки, улучшении существующих навыков или изучении лучших практик развития навыков для любой области.
