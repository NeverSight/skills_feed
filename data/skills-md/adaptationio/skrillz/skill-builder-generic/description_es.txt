Guía universal para crear habilidades de Claude Code listas para producción para cualquier proyecto. Incluye un flujo de trabajo de 6 pasos (comprender, planificar, inicializar, editar, empaquetar, iterar), diseño de divulgación progresiva, plantillas YAML, scripts de validación, patrones de organización de referencia y 10 innovaciones probadas por la comunidad. Úselo al crear nuevas habilidades de Claude Code, convertir documentación en habilidades, mejorar las habilidades existentes o aprender las mejores prácticas de desarrollo de habilidades para cualquier dominio.
