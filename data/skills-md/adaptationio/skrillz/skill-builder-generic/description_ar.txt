دليل عالمي لإنشاء مهارات Claude Code الجاهزة للإنتاج لأي مشروع. يتضمن سير عمل مكون من 6 خطوات (الفهم، والتخطيط، والتهيئة، والتحرير، والحزم، والتكرار)، وتصميم الكشف التدريجي، وقوالب YAML الأمامية، والبرامج النصية للتحقق من الصحة، وأنماط التنظيم المرجعية، و10 ابتكارات مثبتة من قبل المجتمع. يُستخدم عند إنشاء مهارات Claude Code جديدة، أو تحويل الوثائق إلى مهارات، أو تحسين المهارات الحالية، أو تعلم أفضل ممارسات تطوير المهارات لأي مجال.
