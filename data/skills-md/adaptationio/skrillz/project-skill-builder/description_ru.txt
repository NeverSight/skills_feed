Создайте навыки Claude Code для конкретного проекта, адаптированные к конкретным базам кода, доменам или организациям. Рабочий процесс для анализа контекста проекта, выявления возможностей развития навыков, разработки шаблонов для конкретного проекта и формирования индивидуальных навыков. Используйте при создании навыков для конкретных проектов, сборе знаний о проекте, построении рабочих процессов для конкретной команды или разработке наборов инструментов для конкретных предметных областей.
