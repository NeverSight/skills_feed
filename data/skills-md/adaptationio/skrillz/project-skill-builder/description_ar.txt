قم بإنشاء مهارات Claude Code الخاصة بالمشروع والمصممة خصيصًا لقواعد تعليمات برمجية أو مجالات أو مؤسسات معينة. سير العمل لتحليل سياق المشروع، وتحديد فرص المهارات، وتصميم أنماط خاصة بالمشروع، وبناء مهارات مخصصة. يُستخدم عند إنشاء مهارات لمشاريع محددة، أو الحصول على المعرفة بالمشروع، أو بناء مهام سير عمل خاصة بفريق معين، أو تطوير مجموعات أدوات المهارات الخاصة بالمجال.
