خبرة شاملة في المنتجات الإنسانية تغطي نماذج Claude وClaude API وPython SDK وAgent SDK وClaude Code وModel context Protocol. ست إمكانات متكاملة مع الوثائق الكاملة والمراجع القابلة للبحث وأمثلة التعليمات البرمجية وأنماط التكامل بين المنتجات. يُستخدم عند العمل مع Claude API، أو وكلاء البناء، أو استخدام SDKs، أو التطوير باستخدام Claude Code، أو دمج خوادم MCP، أو تعلم المنتجات الإنسانية، أو تحسين التكاليف، أو تنفيذ الميزات الإنسانية، أو إدارة السياق، أو استخدام Opus 4.5، أو تنفيذ أنماط الأدوات المتقدمة.
