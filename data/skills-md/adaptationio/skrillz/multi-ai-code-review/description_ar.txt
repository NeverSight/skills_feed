مراجعة التعليمات البرمجية متعددة وجهات النظر باستخدام Claude وGemini وCodex كوكلاء متخصصين. تحليل خماسي الأبعاد (الأمان، والأداء، وقابلية الصيانة، والصحة، والأسلوب) مع إجماع LLM كحكم، وتسجيل الجودة، وتكامل CI/CD. يُستخدم عند مراجعة العلاقات العامة، أو تدقيق جودة التعليمات البرمجية، أو إعداد إصدارات الإنتاج، أو إنشاء سير عمل مراجعة التعليمات البرمجية.
