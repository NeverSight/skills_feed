أنماط AWS Boto3 SDK لإدارة مجموعة Amazon EKS ومجموعات العقد ورموز المصادقة المميزة وتكامل عميل Kubernetes. يُستخدم عند العمل مع مجموعات EKS، أو إدارة مجموعات العقد، أو إنشاء kubeconfig، أو إنشاء رموز المصادقة المميزة، أو دمج عميل Kubernetes Python، أو إدارة ملفات تعريف Fargate، أو تنفيذ مصادقة IRSA.
