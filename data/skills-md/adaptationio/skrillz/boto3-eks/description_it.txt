Modelli SDK AWS Boto3 per la gestione dei cluster Amazon EKS, i gruppi di nodi, i token di autenticazione e l'integrazione del client Kubernetes. Da utilizzare quando si lavora con cluster EKS, si gestiscono gruppi di nodi, si genera kubeconfig, si creano token di autenticazione, si integra il client Kubernetes Python, si gestiscono profili Fargate o si implementa l'autenticazione IRSA.
