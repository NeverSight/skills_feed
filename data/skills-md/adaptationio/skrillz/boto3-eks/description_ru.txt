Шаблоны AWS Boto3 SDK для управления кластерами Amazon EKS, группами узлов, токенами аутентификации и интеграцией клиентов Kubernetes. Используйте при работе с кластерами EKS, управлении группами узлов, создании kubeconfig, создании токенов аутентификации, интеграции клиента Kubernetes Python, управлении профилями Fargate или реализации аутентификации IRSA.
