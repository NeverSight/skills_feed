Систематический исследовательский рабочий процесс, организующий исследовательские операции из нескольких источников для всестороннего исследования предметной области. Последовательный рабочий процесс от поиска в Интернете через исследование GitHub и анализ документации до синтеза исследований. Используйте при исследовании новых областей, сборе закономерностей, изучении технологий или проведении комплексных исследований из нескольких источников для развития навыков.
