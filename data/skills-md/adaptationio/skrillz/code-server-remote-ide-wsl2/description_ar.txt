قم بإعداد VS Code IDE آمن في المتصفح باستخدام خادم التعليمات البرمجية على WSL2، ويمكن الوصول إليه من الهاتف المحمول/الكمبيوتر اللوحي عبر ngrok/Cloudflare/Tailscale. ميزات IDE كاملة مع دعم الامتداد وإدارة الموارد وتحسين الأداء. استخدمه عندما تحتاج إلى الوصول إلى IDE عن بعد، أو رمز VS في المتصفح، أو التطوير عن بعد، أو بيئة الترميز الكاملة على الهاتف المحمول/الكمبيوتر اللوحي.
