Роли IAM для сервисных учетных записей (IRSA) для разрешений AWS на уровне модуля EKS. Используйте при настройке доступа к IAM модуля, настройке интеграции сервисов AWS, реализации безопасности с минимальными привилегиями, устранении неполадок доверительных отношений OIDC или развертывании контроллеров AWS.
