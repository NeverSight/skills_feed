Convalida completa dei test per le competenze di Claude Code attraverso test funzionali, convalida di esempi, test di integrazione, test di regressione e test di casi limite. Operazioni di test basate su attività con esecuzione automatizzata di esempi, test manuali degli scenari e reporting dei test. Da utilizzare durante il test della funzionalità delle competenze, la convalida dell'esecuzione corretta degli esempi, la garanzia del funzionamento dell'integrazione, la prevenzione di regressioni o l'esecuzione di un controllo completo della qualità funzionale.
