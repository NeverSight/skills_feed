Validation complète des tests pour les compétences de Claude Code via des tests fonctionnels, une validation d'exemples, des tests d'intégration, des tests de régression et des tests de cas extrêmes. Opérations de test basées sur des tâches avec exécution automatisée d'exemples, tests de scénarios manuels et rapports de tests. À utiliser pour tester la fonctionnalité des compétences, valider l'exécution correcte des exemples, garantir le bon fonctionnement de l'intégration, prévenir les régressions ou effectuer une assurance qualité fonctionnelle complète.
