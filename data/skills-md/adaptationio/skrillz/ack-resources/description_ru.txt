Контроллеры AWS для Kubernetes (ACK) для управления ресурсами AWS, нативного Kubernetes. Используйте при управлении ресурсами AWS через kubectl, реализации GitOps для инфраструктуры, создании платформ самообслуживания разработчиков, интеграции сервисов AWS с рабочими нагрузками EKS или внедрении существующих ресурсов AWS в Kubernetes.
