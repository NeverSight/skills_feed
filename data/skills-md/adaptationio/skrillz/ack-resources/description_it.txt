Controller AWS per Kubernetes (ACK) per la gestione delle risorse AWS native di Kubernetes. Da utilizzare durante la gestione delle risorse AWS tramite kubectl, l'implementazione di GitOps per l'infrastruttura, la creazione di piattaforme per sviluppatori self-service, l'integrazione di servizi AWS con carichi di lavoro EKS o l'adozione di risorse AWS esistenti in Kubernetes.
