Память Amazon Bedrock AgentCore для постоянного хранения данных агента во время сеансов. Эпизодическая память для обучения на основе взаимодействий, кратковременная для контекста сеанса. Используйте при создании агентов, которые запоминают предпочтения пользователя, учатся на основе разговоров или поддерживают контекст между сеансами.
