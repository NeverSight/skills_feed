Механизм автоматического обновления документации для навыков антропологического эксперта. Пятиэтапный рабочий процесс: от обнаружения обновлений, получения и обработки документации до интеграции и проверки навыков. Используйте при обновлении документации Anthropic, проверке новых выпусков, получении последних версий документации, поддержании актуальности информации для экспертов по антропологии или синхронизации с изменениями продукта Anthropic.
