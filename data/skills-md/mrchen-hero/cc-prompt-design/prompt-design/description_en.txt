A prompt design, optimization, and quality-check tool based on Anthropic’s official guidelines. Applicable scenarios:
(1) Creating new prompts – user says “Help me create/design an XXX prompt”
(2) Optimizing existing prompts – user says “Optimize/improve this prompt”
(3) Checking prompt quality – user says “Check/review this prompt”
(4) Understanding prompt structure – user asks “How do I design a prompt?” or “Best practices for prompts”
