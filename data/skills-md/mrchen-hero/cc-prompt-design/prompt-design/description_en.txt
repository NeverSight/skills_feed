A prompt design, optimization, and quality-checking tool based on Anthropic's official guidelines. Applicable scenarios:
(1) Creating new prompts – user says "help me create/design a prompt for XXX"
(2) Optimizing existing prompts – user says "optimize/improve this prompt"
(3) Checking prompt quality – user says "check/review this prompt"
(4) Understanding prompt structure – user asks "how to design prompts" or "prompt best practices"
