Una herramienta rápida de diseño, optimización y control de calidad basada en las pautas oficiales de Anthropic. Escenarios aplicables:
(1) Creación de nuevos mensajes: el usuario dice "ayúdame a crear/diseñar un mensaje para XXX"
(2) Optimización de mensajes existentes: el usuario dice "optimizar/mejorar este mensaje"
(3) Comprobación de la calidad del mensaje: el usuario dice "verificar/revisar este mensaje"
(4) Comprender la estructura de los mensajes: el usuario pregunta "cómo diseñar mensajes" o "mejores prácticas de mensajes"
