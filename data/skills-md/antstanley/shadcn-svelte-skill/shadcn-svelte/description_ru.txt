Экспертное руководство по shadcn-svelte, порту shadcn/ui для Svelte 5. Используйте этот навык при работе с компонентами shadcn-svelte, установке компонентов пользовательского интерфейса через CLI, реализации форм с помощью Superforms/Formsnap, добавлении темного режима с помощью mode-watcher, создании таблиц данных с помощью TanStack Table, создании тем с помощью цветов Tailwind CSS v4 и OKLCH или создании реестров пользовательских компонентов. Срабатывает при упоминании shadcn-svelte, компонентов пользовательского интерфейса Bits или библиотек компонентов пользовательского интерфейса SvelteKit.
