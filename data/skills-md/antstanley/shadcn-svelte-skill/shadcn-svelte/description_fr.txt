Conseils d'experts pour shadcn-svelte, un port Svelte 5 de shadcn/ui. Utilisez cette compétence lorsque vous travaillez avec des composants shadcn-svelte, installez des composants d'interface utilisateur via CLI, implémentez des formulaires avec Superforms/Formsnap, ajoutez un mode sombre avec mode-watcher, créez des tables de données avec TanStack Table, thématisez avec les couleurs Tailwind CSS v4 et OKLCH ou créez des registres de composants personnalisés. Se déclenche sur les mentions de shadcn-svelte, de composants Bits UI ou de bibliothèques de composants SvelteKit UI.
