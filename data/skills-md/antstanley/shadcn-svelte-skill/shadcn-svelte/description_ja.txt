shadcn/ui の Svelte 5 ポートである shadcn-svelte に関する専門家によるガイダンス。このスキルは、shadcn-svelte コンポーネントの操作、CLI を介した UI コンポーネントのインストール、Superforms/Formsnap を使用したフォームの実装、mode-watcher を使用したダーク モードの追加、TanStack Table を使用したデータ テーブルの作成、Tailwind CSS v4 および OKLCH カラーを使用したテーマ設定、またはカスタム コンポーネント レジストリの構築を行うときに使用します。 shadcn-svelte、Bits UI コンポーネント、または SvelteKit UI コンポーネント ライブラリの言及をトリガーします。
