تدقيق شامل لتحسين محركات البحث للتجارة الإلكترونية لصفحات المنتج وصفحات المجموعة وتحسين محركات البحث الفنية وتحليل ملف السجل وأبحاث المنافسين. يُستخدم عندما يطلب المستخدم تدقيق تحسين محركات البحث، أو مراجعة تحسين محركات البحث للتجارة الإلكترونية، أو تحسين صفحة المجموعة، أو تحسين محركات البحث لصفحة المنتج، أو تحليل الزحف، أو يريد تحسين التصنيفات العضوية.
