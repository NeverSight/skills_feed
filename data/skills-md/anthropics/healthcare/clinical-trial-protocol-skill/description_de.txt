Erstellen Sie klinische Studienprotokolle für medizinische Geräte oder Medikamente. Diese Fähigkeit sollte verwendet werden, wenn Benutzer sagen: „Erstellen Sie ein klinisches Studienprotokoll“, „Generieren Sie ein Protokoll für [Gerät/Arzneimittel]“, „Helfen Sie mir beim Entwerfen einer klinischen Studie“, „Recherchieren Sie ähnliche Studien für [Intervention]“ oder wenn Sie eine FDA-Einreichungsdokumentation für Prüfpräparate entwickeln.
