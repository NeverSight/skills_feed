Entwickeln oder verbessern Sie eine unternehmensspezifische Fähigkeit zur Datenanalyse, indem Sie Stammwissen von Analysten extrahieren.
BOOTSTRAP-MODUS – Auslöser: „Erstellen Sie einen Datenkontext-Skill“, „Richten Sie eine Datenanalyse für unser Lager ein“, „Helfen Sie mir, einen Skill für unsere Datenbank zu erstellen“, „Generieren Sie einen Daten-Skill für [Unternehmen]“ → Erkennt Schemata, stellt Schlüsselfragen und generiert einen ersten Skill mit Referenzdateien
ITERATIONSMODUS – Auslöser: „Kontext zu [Domäne] hinzufügen“, „Der Skill benötigt weitere Informationen zu [Thema]“, „Datenskill mit [Metriken/Tabellen/Terminologie] aktualisieren“, „[Domäne]-Referenz verbessern“ → Lädt vorhandenen Skill, stellt gezielte Fragen, fügt Referenzdateien an/aktualisiert
Verwenden Sie es, wenn Datenanalysten möchten, dass Claude das spezifische Data Warehouse, die Terminologie, die Metrikdefinitionen und die allgemeinen Abfragemuster ihres Unternehmens versteht.
