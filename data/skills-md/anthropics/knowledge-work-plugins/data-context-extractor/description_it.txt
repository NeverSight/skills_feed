Genera o migliora una capacità di analisi dei dati specifica dell'azienda estraendo la conoscenza tribale dagli analisti.
MODALITÀ BOOTSTRAP - Trigger: "Crea una competenza sul contesto dati", "Imposta l'analisi dei dati per il nostro magazzino", "Aiutami a creare una competenza per il nostro database", "Genera una competenza sui dati per [azienda]" → Scopre schemi, pone domande chiave, genera competenza iniziale con file di riferimento
MODALITÀ ITERAZIONE - Trigger: "Aggiungi contesto su [dominio]", "La competenza richiede più informazioni su [argomento]", "Aggiorna la competenza dati con [metriche/tabelle/terminologia]", "Migliora il riferimento [dominio]" → Carica competenza esistente, pone domande mirate, aggiunge/aggiorna file di riferimento
Da utilizzare quando gli analisti di dati desiderano che Claude comprenda il data warehouse, la terminologia, le definizioni delle metriche e i modelli di query comuni specifici della loro azienda.
