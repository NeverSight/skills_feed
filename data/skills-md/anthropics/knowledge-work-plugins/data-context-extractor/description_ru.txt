Создайте или улучшите навыки анализа данных, специфичные для компании, путем извлечения племенных знаний от аналитиков.
РЕЖИМ ЗАГРУЗКИ — триггеры: «Создать навык контекста данных», «Настроить анализ данных для нашего хранилища», «Помогите мне создать навык для нашей базы данных», «Создать навык данных для [компании]» → Обнаруживает схемы, задает ключевые вопросы, генерирует начальный навык с помощью справочных файлов.
РЕЖИМ ИТЕРАЦИИ — триггеры: «Добавить контекст о [домене]», «Навыку требуется дополнительная информация о [теме]», «Обновить навык данных с помощью [показателей/таблиц/терминологии]», «Улучшить ссылку на [домен]» → Загружает существующий навык, задает целевые вопросы, добавляет/обновляет справочные файлы.
Используйте, когда аналитики данных хотят, чтобы Клод понял конкретное хранилище данных их компании, терминологию, определения метрик и общие шаблоны запросов.
