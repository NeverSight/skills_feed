Générez ou améliorez une compétence d'analyse de données spécifique à l'entreprise en extrayant les connaissances tribales des analystes.
MODE BOOTSTRAP - Déclencheurs : "Créer une compétence de contexte de données", "Configurer l'analyse des données pour notre entrepôt", "Aidez-moi à créer une compétence pour notre base de données", "Générer une compétence de données pour [entreprise]" → Découvre les schémas, pose des questions clés, génère la compétence initiale avec des fichiers de référence
MODE ITÉRATION - Déclencheurs : "Ajouter du contexte sur [domaine]", "La compétence a besoin de plus d'informations sur [sujet]", "Mettre à jour la compétence de données avec [métriques/tables/terminologie]", "Améliorer la référence [domaine]" → Charge la compétence existante, pose des questions ciblées, ajoute/met à jour les fichiers de référence
À utiliser lorsque les analystes de données souhaitent que Claude comprenne l'entrepôt de données spécifique à leur entreprise, la terminologie, les définitions de métriques et les modèles de requêtes courants.
