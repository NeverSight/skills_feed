الاستعداد لمكالمة المبيعات مع سياق الحساب، وأبحاث الحضور، وجدول الأعمال المقترح. يعمل بشكل مستقل مع مدخلات المستخدم وأبحاث الويب، ويتم تعزيزه عند توصيل نظام إدارة علاقات العملاء (CRM) أو البريد الإلكتروني أو الدردشة أو النصوص. قم بتشغيل "إعدادي لمكالمتي مع [الشركة]"، أو "سأعقد اجتماعًا مع [الشركة] أعدني"، أو "أعد الاتصال بـ [الشركة]"، أو "جهزني لـ [الاجتماع]".
