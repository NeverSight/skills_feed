Guidez les utilisateurs dans la création d’un nouveau plugin à partir de zéro lors d’une session de cowork. À utiliser lorsque les utilisateurs souhaitent créer un plugin, créer un plugin, créer un nouveau plugin, développer un plugin, échafauder un plugin, démarrer un plugin à partir de zéro ou concevoir un plugin. Cette compétence nécessite le mode Cowork avec accès au répertoire de sorties pour livrer le fichier .plugin final.
