Guida gli utenti attraverso la creazione di un nuovo plug-in da zero in una sessione di cowork. Da utilizzare quando gli utenti desiderano creare un plug-in, creare un plug-in, creare un nuovo plug-in, sviluppare un plug-in, strutturare un plug-in, avviare un plug-in da zero o progettare un plug-in. Questa abilità richiede la modalità Cowork con accesso alla directory degli output per la consegna del file .plugin finale.
