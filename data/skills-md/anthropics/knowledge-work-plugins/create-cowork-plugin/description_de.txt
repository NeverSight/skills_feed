Führen Sie Benutzer durch die Erstellung eines neuen Plugins von Grund auf in einer Cowork-Sitzung. Verwenden Sie diese Option, wenn Benutzer ein Plugin erstellen, ein Plugin erstellen, ein neues Plugin erstellen, ein Plugin entwickeln, ein Plugin aufbauen, ein Plugin von Grund auf neu starten oder ein Plugin entwerfen möchten. Dieser Skill erfordert den Cowork-Modus mit Zugriff auf das Ausgabeverzeichnis für die Bereitstellung der endgültigen .plugin-Datei.
