Преобразуйте выходные файлы лабораторных приборов (PDF, CSV, Excel, TXT) в формат JSON аллотропной простой модели (ASM) или плоский 2D CSV. Используйте этот навык, когда ученым необходимо стандартизировать данные приборов для систем LIMS, озер данных или последующего анализа. Поддерживает автоматическое определение типов инструментов. Выходные данные включают полный ASM JSON, плоский CSV для удобного импорта и экспортируемый код Python для инженеров данных. Общие триггеры включают преобразование файлов приборов, стандартизацию лабораторных данных, подготовку данных для загрузки в системы LIMS/ELN или создание кода синтаксического анализатора для производственных конвейеров.
