قم بتحويل ملفات مخرجات أدوات المختبر (PDF، وCSV، وExcel، وTXT) إلى تنسيق Allotrope Simple Model (ASM) JSON أو تنسيق CSV ثنائي الأبعاد مسطح. استخدم هذه المهارة عندما يحتاج العلماء إلى توحيد بيانات الأجهزة لأنظمة LIMS أو بحيرات البيانات أو التحليل النهائي. يدعم الكشف التلقائي عن أنواع الأدوات. تتضمن المخرجات ASM JSON الكامل، وCSV المسطح لسهولة الاستيراد، ورمز Python القابل للتصدير لمهندسي البيانات. تتضمن المشغلات الشائعة تحويل ملفات الأجهزة، أو توحيد بيانات المختبر، أو إعداد البيانات للتحميل إلى أنظمة LIMS/ELN، أو إنشاء كود محلل لخطوط أنابيب الإنتاج.
