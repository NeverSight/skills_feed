Converti file di output degli strumenti di laboratorio (PDF, CSV, Excel, TXT) nel formato JSON Allotrope Simple Model (ASM) o CSV 2D appiattito. Utilizza questa competenza quando gli scienziati devono standardizzare i dati strumentali per sistemi LIMS, data lake o analisi downstream. Supporta il rilevamento automatico dei tipi di strumenti. Gli output includono JSON ASM completo, CSV appiattito per una facile importazione e codice Python esportabile per gli ingegneri dei dati. I trigger comuni includono la conversione di file di strumenti, la standardizzazione dei dati di laboratorio, la preparazione dei dati per il caricamento sui sistemi LIMS/ELN o la generazione di codice parser per le pipeline di produzione.
