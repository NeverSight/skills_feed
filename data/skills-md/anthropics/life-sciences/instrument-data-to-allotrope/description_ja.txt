実験室機器の出力ファイル (PDF、CSV、Excel、TXT) を同素体単純モデル (ASM) JSON 形式またはフラット化された 2D CSV に変換します。科学者が LIMS システム、データ レイク、またはダウンストリーム分析用に機器データを標準化する必要がある場合に、このスキルを使用します。機器タイプの自動検出をサポートします。出力には、完全な ASM JSON、簡単にインポートできるフラット化された CSV、データ エンジニア向けのエクスポート可能な Python コードが含まれます。一般的なトリガーには、機器ファイルの変換、ラボ データの標準化、LIMS/ELN システムにアップロードするためのデータの準備、または実稼働パイプライン用のパーサー コードの生成が含まれます。
