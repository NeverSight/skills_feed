Convertissez les fichiers de sortie des instruments de laboratoire (PDF, CSV, Excel, TXT) au format JSON Allotrope Simple Model (ASM) ou CSV 2D aplati. Utilisez cette compétence lorsque les scientifiques ont besoin de normaliser les données d'instruments pour les systèmes LIMS, les lacs de données ou les analyses en aval. Prend en charge la détection automatique des types d'instruments. Les sorties incluent du JSON ASM complet, du CSV aplati pour une importation facile et du code Python exportable pour les ingénieurs de données. Les déclencheurs courants incluent la conversion de fichiers d'instruments, la normalisation des données de laboratoire, la préparation des données à télécharger vers les systèmes LIMS/ELN ou la génération de code d'analyseur pour les pipelines de production.
