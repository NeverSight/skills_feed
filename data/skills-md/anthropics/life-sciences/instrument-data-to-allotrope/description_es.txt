Convierta archivos de salida de instrumentos de laboratorio (PDF, CSV, Excel, TXT) al formato JSON de modelo alótropo simple (ASM) o CSV 2D aplanado. Utilice esta habilidad cuando los científicos necesiten estandarizar datos de instrumentos para sistemas LIMS, lagos de datos o análisis posteriores. Admite la detección automática de tipos de instrumentos. Los resultados incluyen ASM JSON completo, CSV aplanado para una fácil importación y código Python exportable para ingenieros de datos. Los desencadenantes comunes incluyen la conversión de archivos de instrumentos, la estandarización de datos de laboratorio, la preparación de datos para cargarlos en sistemas LIMS/ELN o la generación de código de analizador para canales de producción.
