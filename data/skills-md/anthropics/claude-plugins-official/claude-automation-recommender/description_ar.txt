قم بتحليل قاعدة التعليمات البرمجية والتوصية بعمليات التشغيل الآلي لـ Claude Code (الخطافات، والوكلاء الفرعيون، والمهارات، والمكونات الإضافية، وخوادم MCP). يُستخدم عندما يطلب المستخدم توصيات التشغيل الآلي، أو يريد تحسين إعداد Claude Code الخاص به، أو يذكر تحسين سير عمل Claude Code، أو يسأل عن كيفية إعداد Claude Code لأول مرة لمشروع، أو يريد معرفة ميزات Claude Code التي يجب عليه استخدامها.
