Создает автономные шаблоны PPT НАВЫКИ (не презентации) на основе предоставленных пользователем шаблонов PowerPoint. Используйте ТОЛЬКО в том случае, если пользователь хочет создать навык многократного использования из своего шаблона. Вместо этого для создания реальных презентаций используйте навык pptx.
