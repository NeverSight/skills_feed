Заполняет шаблоны презентаций инвестиционно-банковской деятельности данными из исходных файлов. Используйте, когда: пользователь предоставляет шаблон PowerPoint для заполнения, у пользователя есть исходные данные (Excel/CSV) для заполнения слайдов, пользователь упоминает о заполнении или заполнении шаблона презентации или пользователю необходимо перенести данные в существующие макеты слайдов. Не для создания презентаций с нуля.
