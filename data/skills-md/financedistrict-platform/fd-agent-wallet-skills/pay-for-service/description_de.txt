Greifen Sie über das x402-Zahlungsprotokoll auf kostenpflichtige API-Endpunkte und Inhalte zu. Verwenden Sie diese Option, wenn Sie oder der Benutzer eine kostenpflichtige API aufrufen, auf geschützte Inhalte zugreifen, eine x402-Zahlung vornehmen, einen kostenpflichtigen Dienst nutzen oder kostenpflichtige Inhalte abrufen möchten. Behandelt „diese kostenpflichtige Ressource abrufen“, „auf x402-Inhalte zugreifen“ und „für diesen API-Aufruf bezahlen“.
