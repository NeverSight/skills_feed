Получите доступ к конечным точкам и контенту платных API с помощью платежного протокола x402. Используйте, когда вы или пользователь хотите вызвать платный API, получить доступ к закрытому контенту, совершить платеж x402, использовать платную службу или получить контент, требующий оплаты. Охватывает «получить этот платный ресурс», «получить доступ к контенту x402», «оплатить этот вызов API».
