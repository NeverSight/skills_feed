Комплексный набор инструментов для разработки с использованием библиотеки CocoIndex. Используйте, когда пользователям необходимо создавать конвейеры (потоки) преобразования данных, писать пользовательские функции или управлять потоками через CLI или API. Охватывает создание рабочих процессов ETL для обработки данных ИИ, включая встраивание документов в векторные базы данных, построение графиков знаний, создание поисковых индексов или обработку потоков данных с инкрементными обновлениями.
