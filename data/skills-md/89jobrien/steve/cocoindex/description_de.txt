Umfassendes Toolkit für die Entwicklung mit der CocoIndex-Bibliothek. Verwenden Sie diese Option, wenn Benutzer Datentransformationspipelines (Flows) erstellen, benutzerdefinierte Funktionen schreiben oder Flows über CLI oder API betreiben müssen. Behandelt die Erstellung von ETL-Workflows für die KI-Datenverarbeitung, einschließlich der Einbettung von Dokumenten in Vektordatenbanken, der Erstellung von Wissensgraphen, der Erstellung von Suchindizes oder der Verarbeitung von Datenströmen mit inkrementellen Aktualisierungen.
