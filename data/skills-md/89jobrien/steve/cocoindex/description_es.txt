Completo kit de herramientas para desarrollar con la biblioteca CocoIndex. Úselo cuando los usuarios necesiten crear canalizaciones (flujos) de transformación de datos, escribir funciones personalizadas u operar flujos a través de CLI o API. Cubre la creación de flujos de trabajo ETL para el procesamiento de datos de IA, incluida la incorporación de documentos en bases de datos vectoriales, la creación de gráficos de conocimiento, la creación de índices de búsqueda o el procesamiento de flujos de datos con actualizaciones incrementales.
