متخصص في التوثيق الشامل يغطي وثائق API والكتابة الفنية ووثائق التصميم وأدلة الترحيل وإنشاء سجل التغيير. يُستخدم عند إنشاء مواصفات OpenAPI/Swagger، أو إنشاء مجموعات SDK، أو كتابة أدلة المستخدم، أو إنشاء ملفات README، أو توثيق البنية، أو كتابة مواصفات التصميم، أو إنشاء ADRs، أو كتابة أدلة الترحيل، أو إنشاء سجلات التغيير من التزامات git. يتعامل مع الإصدارات والأمثلة وتجربة المطور والوثائق التي تواجه المستخدم.
