Шаблоны автоматизации рабочих процессов n8n и интеграция API. Этот навык следует использовать при создании рабочих процессов n8n, использовании веб-перехватчиков, управлении рабочими процессами через REST API или интеграции n8n с серверами MCP. Охватывает структуру JSON рабочего процесса, шаблоны узлов и лучшие практики автоматизации.
