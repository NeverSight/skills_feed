Modelli di automazione del flusso di lavoro n8n e integrazione API. Questa competenza dovrebbe essere utilizzata durante la creazione di flussi di lavoro n8n, l'utilizzo di webhook, la gestione di flussi di lavoro tramite API REST o l'integrazione di n8n con server MCP. Copre la struttura JSON del flusso di lavoro, i modelli di nodo e le migliori pratiche di automazione.
