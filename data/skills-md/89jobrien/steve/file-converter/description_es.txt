Esta habilidad maneja conversiones de formatos de archivos entre documentos (PDF, DOCX, Markdown, HTML, TXT), archivos de datos (JSON, CSV, YAML, XML, TOML) e imágenes (PNG, JPG, WebP, SVG, GIF). Utilícelo cuando el usuario solicite convertir, transformar o exportar archivos entre formatos. Genera código de conversión dinámicamente según la solicitud específica.
