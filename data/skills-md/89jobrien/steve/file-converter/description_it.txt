Questa competenza gestisce le conversioni dei formati di file tra documenti (PDF, DOCX, Markdown, HTML, TXT), file di dati (JSON, CSV, YAML, XML, TOML) e immagini (PNG, JPG, WebP, SVG, GIF). Da utilizzare quando l'utente richiede la conversione, la trasformazione o l'esportazione di file tra formati. Genera il codice di conversione in modo dinamico in base alla richiesta specifica.
