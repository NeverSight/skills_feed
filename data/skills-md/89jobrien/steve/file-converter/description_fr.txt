Cette compétence gère les conversions de formats de fichiers entre les documents (PDF, DOCX, Markdown, HTML, TXT), les fichiers de données (JSON, CSV, YAML, XML, TOML) et les images (PNG, JPG, WebP, SVG, GIF). À utiliser lorsque l'utilisateur demande la conversion, la transformation ou l'exportation de fichiers entre formats. Génère un code de conversion dynamiquement en fonction de la demande spécifique.
