Integración de componentes shadcn-vue para Inertia Rails Vue 3 (NO Nuxt): formularios, diálogos, tablas, brindis, modo oscuro y más. Úselo al crear una interfaz de usuario con componentes shadcn-vue en una aplicación Inertia + Vue o al adaptar ejemplos de shadcn-vue de Nuxt. Conecte las entradas de shadcn-vue a Inertia Form mediante el atributo de nombre y la ranura de alcance #default. Los brindis flash requieren la configuración del inicializador Rails flash_keys.
