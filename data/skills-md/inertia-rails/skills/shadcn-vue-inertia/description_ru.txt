интеграция компонентов shadcn-vue для Inertia Rails Vue 3 (НЕ Nuxt): формы, диалоги, таблицы, всплывающие уведомления, темный режим и многое другое. Используйте при создании пользовательского интерфейса с компонентами shadcn-vue в приложении Inertia + Vue или при адаптации примеров shadcn-vue из Nuxt. Подключите входные данные shadcn-vue к форме инерции через атрибут имени и слот с областью действия #default. Для флэш-тостов требуется конфигурация инициализатора Rails flash_keys.
