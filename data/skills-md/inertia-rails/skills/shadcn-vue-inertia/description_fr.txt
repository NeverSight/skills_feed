Intégration de composants shadcn-vue pour Inertia Rails Vue 3 (PAS Nuxt) : formulaires, boîtes de dialogue, tableaux, toasts, mode sombre, et plus encore. À utiliser lors de la création d'une interface utilisateur avec des composants shadcn-vue dans une application Inertia + Vue ou lors de l'adaptation d'exemples shadcn-vue de Nuxt. Câblez les entrées shadcn-vue à Inertia Form via l'attribut name et l'emplacement de portée #default. Les toasts Flash nécessitent la configuration de l'initialiseur Rails flash_keys.
