Intégration de composants shadcn-svelte (bits-ui) pour Inertia Rails Svelte (PAS SvelteKit) : formulaires, boîtes de dialogue, tableaux, toasts, mode sombre, etc. À utiliser lors de la création d'une interface utilisateur avec des composants shadcn-svelte dans une application Inertia + Svelte ou lors de l'adaptation d'exemples shadcn-svelte à partir de SvelteKit. Câblez les entrées shadcn-svelte à Inertia Form via l'attribut name et la syntaxe {#snippet}. Les toasts Flash nécessitent la configuration de l'initialiseur Rails flash_keys.
