항상 `render inertia: { key: data }`를 사용하여 데이터를 props로 전달합니다. 인스턴스 변수는 자동 전달되지 않습니다(alba-inertia만이 이를 수행합니다). Inertia.js용 Rails 컨트롤러 패턴: 렌더링 관성, 속성 유형(지연, 선택 사항, 병합, 스크롤), 공유 데이터, 플래시, PRG 리디렉션, 유효성 검사 오류. 데이터를 로드하고, 레코드를 표시하고, 관성 응답을 제공하는 컨트롤러를 작성할 때 사용합니다. 중요: 외부 URL(Stripe/OAuth)은 반드시 inertia_location을 사용해야 하며 절대 리디렉션_to를 사용해서는 안 됩니다.
