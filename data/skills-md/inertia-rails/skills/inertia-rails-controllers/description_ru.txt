ВСЕГДА `рендеринг инерции: {key: data}` для передачи данных в качестве реквизита — переменные экземпляра НЕ передаются автоматически (это делает только alba-inertia). Шаблоны контроллера Rails для Inertia.js: инерция рендеринга, типы свойств (отложенный, необязательный, слияние, прокрутка), общие данные, флэш-память, перенаправления PRG, ошибки проверки. Используйте при написании контроллеров, которые загружают данные, отображают записи или предоставляют ответы по инерции. ВАЖНО: внешние URL-адреса (Stripe/OAuth) ДОЛЖНЫ использовать inertia_location, НИКОГДА не перенаправлять_to.
