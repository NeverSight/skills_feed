Интеграция компонентов shadcn/ui для Inertia Rails React (НЕ Next.js): формы, диалоговые окна, таблицы, всплывающие уведомления, темный режим, палитра команд и многое другое. Используйте при создании пользовательского интерфейса с компонентами shadcn/ui в приложении Inertia или при адаптации примеров shadcn из Next.js. НИКОГДА не реагируйте-hook-form/zod — подключайте входные данные shadcn к форме инерции через атрибут имени. Для флэш-тостов требуется конфигурация инициализатора Rails flash_keys.
