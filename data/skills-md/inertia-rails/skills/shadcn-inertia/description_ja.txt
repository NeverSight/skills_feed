Inertia Rails React (Next.js ではありません) の shadcn/ui コンポーネントの統合: フォーム、ダイアログ、テーブル、トースト、ダーク モード、コマンド パレットなど。 Inertia アプリで shadcn/ui コンポーネントを使用して UI を構築する場合、または Next.js から shadcn サンプルを適応させる場合に使用します。絶対にreact-hook-form/zod — shadcn入力をname属性を介してInertia Formに接続しないでください。 Flash トーストには Rails flash_keys イニシャライザ設定が必要です。
