Integración de componentes shadcn/ui para Inertia Rails React (NO Next.js): formularios, diálogos, tablas, brindis, modo oscuro, paleta de comandos y más. Úselo al crear una interfaz de usuario con componentes shadcn/ui en una aplicación Inertia o al adaptar ejemplos de shadcn de Next.js. NUNCA reaccione-hook-form/zod: entradas de sombreado de cables a Inertia Form a través del atributo de nombre. Los brindis flash requieren la configuración del inicializador Rails flash_keys.
