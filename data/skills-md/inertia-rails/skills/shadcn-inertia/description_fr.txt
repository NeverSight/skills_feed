Intégration de composants shadcn/ui pour Inertia Rails React (PAS Next.js) : formulaires, boîtes de dialogue, tableaux, toasts, mode sombre, palette de commandes, etc. À utiliser lors de la création d'une interface utilisateur avec des composants shadcn/ui dans une application Inertia ou lors de l'adaptation d'exemples shadcn à partir de Next.js. NE JAMAIS réagir-hook-form/zod — câbler les entrées shadcn à Inertia Form via l'attribut name. Les toasts Flash nécessitent la configuration de l'initialiseur Rails flash_keys.
