Сериализаторы Alba + Typelizer для типобезопасных свойств Inertia Rails с автоматически генерируемыми типами TypeScript. Используйте при сериализации моделей для ответов Inertia, настройке ресурсов Alba, создании типов TypeScript из Ruby или использовании параметров свойств Inertia (отложить, один раз, слияние, прокрутка) с атрибутами Alba. Заменяет as_json структурированными автоматически типизированными ресурсами ApplicationResource, страницами и общими ресурсами реквизита. Когда этот параметр активен, он ОТМЕНЯЕТ шаблон `render inertia: { ... }` из других навыков — вместо этого используйте рендеринг переменных экземпляра на основе соглашений.
