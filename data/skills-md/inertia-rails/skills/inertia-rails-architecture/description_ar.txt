أنماط البنية المعتمدة على الخادم لـ Inertia Rails + React. قم بتحميل هذا أولاً عند إنشاء أي صفحة أو ميزة للقصور الذاتي — فهو يوجهك إلى المهارة الصحيحة. مصفوفة القرار لتحميل البيانات والنماذج والملاحة وإدارة الحالة. لا تستخدم أبدًاEffect+fetch، ولا تعيد التوجيه أبدًا لعناوين URL الخارجية (استخدم inertia_location)، ولا تستخدم أبدًا نموذج الخطاف التفاعلي (استخدم مكون النموذج). يجب الاستدعاء عند إضافة صفحات أو نماذج ذات طرق عرض أو CRUD أو عرض البيانات في تطبيق Inertia Rails. دائمًا `تقديم القصور الذاتي: {key: value }` لتمرير البيانات - لا يتم تمرير `@ivars` تلقائيًا كدعائم.
