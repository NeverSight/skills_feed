Шаблоны серверной архитектуры для Inertia Rails + React. Загрузите это ПЕРВОЕ при создании любой страницы или функции Inertia — оно направляется к нужному навыку. Матрица решений для загрузки данных, форм, навигации, управления состоянием. НИКОГДА не используйтеEffect+fetch, НИКОГДА не перенаправляйте_to для внешних URL-адресов (используйте inertia_location), НИКОГДА не реагируйте на форму (используйте компонент Form). ДОЛЖЕН вызываться при добавлении страниц, моделей с представлениями, CRUD или отображении данных в приложении Inertia Rails. ВСЕГДА `рендерите инерцию: {key: value}` для передачи данных — `@ivars` НЕ передаются автоматически в качестве реквизита.
