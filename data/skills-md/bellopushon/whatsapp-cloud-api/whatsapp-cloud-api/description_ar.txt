مرجع WhatsApp Cloud API الرسمي لبناء تكاملات المراسلة. يغطي إرسال الرسائل (النصية، والوسائط، والقوالب، والتفاعلية)، وتلقي خطافات الويب، ودورة حياة المحادثة، وإدارة أرقام الهاتف، ومعالجة الأخطاء. يُستخدم عند إنشاء تكاملات WhatsApp، أو إرسال الرسائل، أو معالجة خطافات الويب، أو العمل مع Meta WhatsApp Business Platform API.
