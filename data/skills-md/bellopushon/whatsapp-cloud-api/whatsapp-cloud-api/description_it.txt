Riferimento ufficiale API WhatsApp Cloud per la creazione di integrazioni di messaggistica. Riguarda l'invio di messaggi (di testo, multimediali, modelli, interattivi), la ricezione di webhook, il ciclo di vita della conversazione, la gestione dei numeri di telefono e la gestione degli errori. Da utilizzare quando si creano integrazioni WhatsApp, si inviano messaggi, si elaborano webhook o si lavora con l'API Meta WhatsApp Business Platform.
