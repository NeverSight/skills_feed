Refactoring aufgeblähter AGENTS.md, CLAUDE.md oder ähnlicher Agent-Anweisungsdateien, um den Grundsätzen der progressiven Offenlegung zu folgen. Teilt monolithische Dateien in organisierte, verknüpfte Dokumentationen auf. Verwenden Sie diese Option, wenn (1) die Agentenkontextdateien zu groß oder unhandlich sind, (2) der Projektkontext von den ausführbaren Fähigkeiten getrennt werden muss, (3) eine modulare Dokumentationsstruktur erstellt werden soll, (4) die vorhandene Dokumentation für eine bessere Organisation umgestaltet werden soll oder (5) eine neue Agentenkontextdokumentation von Grund auf erstellt werden soll.
