Выполните рефакторинг раздутых файлов AGENTS.md, CLAUDE.md или подобных файлов инструкций для агентов, чтобы они соответствовали принципам постепенного раскрытия информации. Разбивает монолитные файлы на организованную связанную документацию. Используйте, когда (1) файлы контекста агента слишком велики или громоздки, (2) необходимо отделить контекст проекта от исполняемых навыков, (3) хотите создать модульную структуру документации, (4) провести рефакторинг существующей документации для лучшей организации или (5) создать новую документацию контекста агента с нуля.
