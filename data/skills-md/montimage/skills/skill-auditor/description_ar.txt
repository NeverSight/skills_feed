قم بتحليل مهارات الوكيل فيما يتعلق بالمخاطر الأمنية والأنماط الضارة والمخاطر المحتملة قبل التثبيت. يُستخدم عندما يُطلب منك "تدقيق مهارة"، أو "التحقق مما إذا كانت المهارة آمنة"، أو "تحليل أمان المهارة"، أو "مراجعة مخاطر المهارة"، أو "هل يجب علي تثبيت هذه المهارة"، أو "هل هذه المهارة آمنة"، أو عند تقييم أي دليل مهارة من أجل الثقة والأمان. يتم تشغيله أيضًا عندما يقوم المستخدم بلصق أمر تثبيت المهارات مثل "npx Skills add https://github.com/org/repo --skill name". يُنتج تقريرًا أمنيًا شاملاً يتضمن حكمًا واضحًا بالتثبيت/الرفض.
