Фундаментальные стандарты и соглашения кодирования Java, применимые ко всему коду Java. Сначала загрузите этот навык при запуске любой задачи разработки Java. Охватывает импорт, конструкторы, возможность обнуления, неизменяемость, исключения, Javadoc, соглашения об именах, безопасность, управление ресурсами и шаблоны параллелизма.
