Normes et conventions fondamentales de codage Java qui s'appliquent à tout le code Java. Chargez d'abord cette compétence lorsque vous démarrez une tâche de développement Java. Couvre les importations, les constructeurs, la nullité, l'immuabilité, les exceptions, Javadoc, les conventions de dénomination, la sécurité, la gestion des ressources et les modèles de concurrence.
