Комплексные навыки работы с анимационной платформой Rive, включая написание сценариев (Luau), интеграцию во время выполнения (React/Next.js), 
конечные автоматы, привязка данных и полный API. Используйте этот навык, когда пользователям нужно создавать интерактивные 
анимацию с помощью Rive, интегрировать Rive в приложения React/Next.js, писать сценарии Rive (Node, Layout, 
Converter, PathEffect protocols), control animations via state machines, implement scroll-based animations, 
или работайте с API рисования Rive (Path, Paint, Renderer). Запускается по: "rive", "rive анимация", "rive script", 
«luau», «@rive-app/react-canvas», «анимация конечного автомата», «интерактивная анимация», «анимация прокрутки с помощью rive».
