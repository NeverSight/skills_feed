Patrones de accesibilidad web para sitios de noticias, herramientas periodísticas y plataformas académicas. Úselo al crear interfaces accesibles, auditar sitios existentes para cumplir con las WCAG, escribir texto alternativo para imágenes de noticias, crear visualizaciones de datos accesibles o garantizar que el contenido llegue a todos los lectores, incluidos aquellos que utilizan tecnologías de asistencia. Esencial para desarrolladores de salas de redacción y cualquiera que publique contenido web.
