Auditoría de seguridad previa al despliegue de aplicaciones web. Úselo cuando revise el código antes de enviarlo, audite una aplicación existente o cuando los usuarios mencionen "revisión de seguridad", "listo para implementar", "pasar a producción" o expresen preocupación por las vulnerabilidades. Cubre autenticación, validación de entradas, gestión de secretos, seguridad de bases de datos y conceptos básicos de cumplimiento.
