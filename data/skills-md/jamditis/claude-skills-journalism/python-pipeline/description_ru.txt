Конвейеры обработки данных Python с модульной архитектурой. Используйте при построении рабочих процессов обработки контента, реализации шаблонов диспетчеров, интеграции API-интерфейсов Google Sheets/Drive или создании систем пакетной обработки. Охватывает шаблоны из проектов Rosen-Scraper, Image-analyzer и Social-Scraper.
