Структурированный рабочий процесс для проверки фактов в журналистике. Используйте при проверке заявлений для публикации, оценке утверждений для статей для проверки фактов или построении процессов проверки перед публикацией. Включает извлечение претензий, сбор доказательств, рейтинговые шкалы и протоколы исправлений.
