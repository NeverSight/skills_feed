Методология эффективной разработки программного обеспечения с помощью искусственного интеллекта. Используйте, когда помогаете пользователям создавать программное обеспечение с помощью помощников по кодированию на основе искусственного интеллекта, отлаживать код, сгенерированный искусственным интеллектом, планировать функции реализации искусственного интеллекта, управлять контролем версий в рабочих процессах искусственного интеллекта или когда пользователи упоминают «кодирование вибрации», Cursor, Windsurf или аналогичные инструменты кодирования искусственного интеллекта. Предоставляет стратегии планирования, тестирования, отладки и итерации кода, написанного с помощью LLM.
