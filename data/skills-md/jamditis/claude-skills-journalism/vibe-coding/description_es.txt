Metodología para el desarrollo eficaz de software asistido por IA. Úselo cuando ayude a los usuarios a crear software con asistentes de codificación de IA, depure código generado por IA, planifique funciones para la implementación de IA, administre el control de versiones en flujos de trabajo de IA o cuando los usuarios mencionen "codificación de vibración", Cursor, Windsurf o herramientas de codificación de IA similares. Proporciona estrategias para planificar, probar, depurar e iterar código escrito con asistencia de LLM.
