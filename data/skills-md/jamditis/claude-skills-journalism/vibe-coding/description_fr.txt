Méthodologie pour un développement logiciel efficace assisté par IA. À utiliser pour aider les utilisateurs à créer des logiciels avec des assistants de codage d'IA, à déboguer le code généré par l'IA, à planifier des fonctionnalités pour la mise en œuvre de l'IA, à gérer le contrôle de version dans les flux de travail d'IA, ou lorsque les utilisateurs mentionnent le « codage dynamique », Cursor, Windsurf ou des outils de codage d'IA similaires. Fournit des stratégies pour la planification, les tests, le débogage et l'itération sur le code écrit avec l'assistance LLM.
