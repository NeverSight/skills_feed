Methodik für eine effektive KI-gestützte Softwareentwicklung. Verwenden Sie diese Option, wenn Sie Benutzern helfen, Software mit KI-Codierungsassistenten zu erstellen, KI-generierten Code zu debuggen, Funktionen für die KI-Implementierung zu planen, die Versionskontrolle in KI-Workflows zu verwalten oder wenn Benutzer „Vibe Coding“, Cursor, Windsurf oder ähnliche KI-Codierungstools erwähnen. Bietet Strategien zum Planen, Testen, Debuggen und Iterieren von Code, der mit LLM-Unterstützung geschrieben wurde.
