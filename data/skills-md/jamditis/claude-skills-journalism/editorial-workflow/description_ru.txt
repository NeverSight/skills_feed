Управляйте редакционными рабочими процессами для редакций и публикаций. Используйте при отслеживании заданий по написанию статей, управлении сроками, координации редакционных календарей или установлении протоколов передачи сообщений между репортерами и редакторами. Включает шаблоны для отслеживания заданий, редакционные календари и документацию по рабочим процессам.
