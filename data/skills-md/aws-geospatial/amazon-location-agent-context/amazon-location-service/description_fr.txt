Intègre les API Amazon Location Service pour les applications AWS. Utilisez cette compétence lorsque les utilisateurs souhaitent ajouter des cartes (MapLibre interactive ou images statiques) ; géocoder les adresses en coordonnées ou inverser les coordonnées en adresses ; calculer des itinéraires, des temps de trajet ou des zones de service ; recherchez des lieux et des entreprises via une recherche textuelle, une recherche à proximité ou des suggestions de saisie semi-automatique ; récupérer des informations détaillées sur le lieu, y compris les heures, les contacts et les adresses ; surveiller les limites géographiques avec des barrières géographiques ; ou suivre les emplacements des appareils. Couvre l'authentification, l'intégration du SDK et toutes les fonctionnalités d'Amazon Location Service.
