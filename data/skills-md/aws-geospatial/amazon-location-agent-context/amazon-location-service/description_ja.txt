AWS アプリケーション用の Amazon Location Service API を統合します。ユーザーがマップ (インタラクティブな MapLibre または静的画像) を追加する場合は、このスキルを使用します。住所を座標にジオコーディングするか、座標を住所に逆ジオコーディングします。ルート、移動時間、またはサービスエリアを計算します。テキスト検索、近くの検索、またはオートコンプリート候補を通じて場所やビジネスを検索します。営業時間、連絡先、住所などの詳細な場所情報を取得します。ジオフェンスで地理的境界を監視する。またはデバイスの位置を追跡します。認証、SDK 統合、およびすべての Amazon Location Service 機能をカバーします。
