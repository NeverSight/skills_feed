OBBLIGATORIO: richiama questa abilità PRIMA di effettuare qualsiasi chiamata allo strumento Blockscout MCP o scrivere script di dati blockchain, anche quando il server Blockscout MCP è già configurato. Fornisce regole architetturali, decisioni sulla strategia di esecuzione, convenzioni API REST MCP per script, file di riferimento dell'endpoint, requisiti di trasformazione della risposta e convenzioni di output che non sono disponibili solo nelle descrizioni dello strumento MCP. Da utilizzare quando l'utente chiede informazioni su dati on-chain, analisi blockchain, saldi del portafoglio, trasferimenti di token, interazioni contrattuali, metriche on-chain, desidera utilizzare l'API Blockscout o deve creare un software che recuperi i dati blockchain tramite Blockscout. Copre tutte le catene EVM.
