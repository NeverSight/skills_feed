إلزامية - قم باستدعاء هذه المهارة قبل إجراء أي استدعاءات لأداة Blockscout MCP أو كتابة أي نصوص برمجية لبيانات blockchain، حتى عندما يكون خادم Blockscout MCP قد تم تكوينه بالفعل. يوفر القواعد المعمارية، وقرارات استراتيجية التنفيذ، واصطلاحات MCP REST API للبرامج النصية، والملفات المرجعية لنقطة النهاية، ومتطلبات تحويل الاستجابة، واصطلاحات الإخراج التي لا تتوفر من أوصاف أداة MCP وحدها. يُستخدم عندما يسأل المستخدم عن البيانات الموجودة على السلسلة، أو تحليل blockchain، أو أرصدة المحفظة، أو عمليات نقل الرمز المميز، أو تفاعلات العقود، أو المقاييس على السلسلة، أو يريد استخدام Blockscout API، أو يحتاج إلى إنشاء برنامج يسترد بيانات blockchain عبر Blockscout. يغطي جميع سلاسل EVM.
