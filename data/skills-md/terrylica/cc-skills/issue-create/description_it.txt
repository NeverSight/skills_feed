Crea ticket GitHub ben formattati con suggerimenti di etichette intelligenti basati sull'intelligenza artificiale e rilevamento del tipo di contenuto. Da utilizzare ogni volta che l'utente desidera creare una segnalazione di bug, una richiesta di funzionalità, una domanda o un problema di documentazione su GitHub oppure dice "presenta un problema", "crea un problema" o "crea un problema gh". NON utilizzare per la gestione dei problemi esistenti, l'organizzazione delle gerarchie dei problemi (utilizzare invece il flusso di lavoro dei problemi) o per la creazione di PR.
