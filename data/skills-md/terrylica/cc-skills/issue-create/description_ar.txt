قم بإنشاء مشكلات GitHub جيدة التنسيق باستخدام اقتراحات التصنيفات الذكية المدعومة بالذكاء الاصطناعي واكتشاف نوع المحتوى. يُستخدم عندما يريد المستخدم إنشاء تقرير خطأ أو طلب ميزة أو سؤال أو مشكلة وثائق على GitHub، أو يقول "أرسل مشكلة" أو "إنشاء مشكلة" أو "إنشاء مشكلة gh". لا تستخدم لإدارة المشكلات الحالية، أو تنظيم التسلسل الهرمي للقضايا (استخدم سير عمل المشكلات بدلاً من ذلك)، أو لإنشاء العلاقات العامة.
