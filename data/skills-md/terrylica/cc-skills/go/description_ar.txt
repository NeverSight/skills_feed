قم بتنفيذ سير عمل التطوير رباعي المراحل المستند إلى ADR (الاختبار المبدئي، والتنفيذ، والتنسيق، والإصدار). يُستخدم عندما يقول المستخدم "itp go"، أو "بدء سير العمل"، أو "تنفيذ هذه الميزة"، أو "بدء المهمة"، أو الإشارة إلى سير عمل ITP. يُستخدم أيضًا عندما يكون لدى المستخدم خطة معتمدة ويريد التنفيذ المنظم مع تتبع ADR. لا تستخدمه لإجراء تعديلات بسيطة لمرة واحدة، أو إصلاحات سريعة، أو مهام لا تحتاج إلى تتبع ADR أو تنفيذ مرحلي.
