Этот навык следует использовать, когда пользователь просит «уточнить выходные данные DSPy», «обеспечить соблюдение ограничений», «использовать dspy.Refine», «выбрать лучший выходной результат», «использовать dspy.BestOfN», упоминает «проверку выходных данных», «проверку ограничений», «генерацию нескольких попыток», «функцию вознаграждения» или ему необходимо улучшить качество выходных данных посредством итеративного уточнения или выбора лучшего из N с помощью пользовательских ограничений.
