Contraintes avisées pour l'écriture de code maintenable et testable. Appliquez les principes MTE, évitez la sur-ingénierie, guidez la refactorisation et fournissez des listes de contrôle de révision du code. À utiliser lorsque les utilisateurs écrivent du code, refactorisent ou ont besoin d'une révision de code. Déclencheurs sur des mots-clés tels que « qualité du code », « refactor », « review », « MTE », « 代码质量 », « 重构 », « 审查 ».
