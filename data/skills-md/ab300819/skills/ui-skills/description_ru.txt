Упрямые ограничения для создания лучших интерфейсов с агентами. Применяйте при написании кода пользовательского интерфейса, реализации проектов или проверке компонентов внешнего интерфейса. Используйте в рабочем процессе DevDocs на этапе разработки для задач, связанных с пользовательским интерфейсом. Запускается по таким ключевым словам, как «UI», «界面», «前端», «组件», «анимация», «动画», «макет», «布局».
