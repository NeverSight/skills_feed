قم بإنشاء وإدارة ملفات تكوين Home Assistant YAML بما في ذلك عمليات التشغيل الآلي والبرامج النصية والقوالب والمخططات ولوحات معلومات Lovelace وتنظيم الملفات. يُستخدم عند العمل مع ملفات تكوين Home Assistant (.yaml، .yml) أو مناقشة عمليات التشغيل الآلي لـ HA، أو البرامج النصية، أو أجهزة الاستشعار، أو لوحات المعلومات.
