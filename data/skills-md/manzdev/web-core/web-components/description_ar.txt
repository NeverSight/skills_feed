قم بإنشاء أو تعديل HTML/CSS/JavaScript (بدون إطار عمل) باستخدام وحدات ES المعيارية (ESM)، وتقنيات الفانيليا، والتعليمات البرمجية النظيفة والقابلة للقراءة، والجاهزة للإنتاج (بدون نقل). مفيد للصفحات خفيفة الوزن وعالية الأداء وعناصر واجهة المستخدم الخالية من التبعية وإعادة هيكلة JavaScript القياسية والنماذج الأولية السريعة دون استخدام أطر عمل مثل React أو Angular أو Vue.
