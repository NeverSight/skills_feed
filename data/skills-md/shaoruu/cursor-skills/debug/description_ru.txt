Отладка кода путем добавления структурированного журналирования. Запускает локальный HTTP-сервер для браузерного/удаленного ведения журнала и, при необходимости, средство просмотра журнала в реальном времени. Используйте при отладке проблем, отслеживании ошибок или когда пользователь говорит /debug.
