Бессерверная база данных SQLite для структурированных данных на периферии. Загружайте при построении реляционных схем, выполнении SQL-запросов, управлении миграциями, выполнении операций CRUD, использовании JOIN/агрегаций, обработке столбцов JSON или принудительном использовании внешних ключей с помощью D1.
