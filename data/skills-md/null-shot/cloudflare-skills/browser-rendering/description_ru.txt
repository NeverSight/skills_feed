Автоматизация Headless Chrome для очистки веб-страниц, создания снимков экрана, PDF-файлов и периферийного тестирования. Загружайте при создании снимков экрана страниц, создании PDF-файлов, очистке динамического контента, извлечении структурированных данных или автоматизации взаимодействия с браузером. Поддерживает REST API, «Кукловод», «Драматург» и «Работник сцены».
