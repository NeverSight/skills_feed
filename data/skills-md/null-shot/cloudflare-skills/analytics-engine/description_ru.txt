Записывайте и запрашивайте данные о событиях высокой мощности в любом масштабе с помощью SQL. Загружайте при отслеживании пользовательских событий, показателей выставления счетов, аналитики по каждому арендатору, A/B-тестирования, использования API или пользовательской телеметрии. Используйте writeDataPoint для неблокирующей записи и API SQL для агрегирования.
