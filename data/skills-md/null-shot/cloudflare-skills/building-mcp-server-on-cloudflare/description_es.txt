Crea servidores MCP (Protocolo de contexto modelo) remotos en Cloudflare Workers
con herramientas, autenticación OAuth e implementación de producción. Genera
código de servidor, configura proveedores de autenticación y lo implementa en trabajadores.

Úselo cuando: el usuario quiera "construir un servidor MCP", "crear herramientas MCP", "remoto
MCP", "implementar MCP", agregar "OAuth a MCP" o menciona el protocolo de contexto modelo
en Cloudflare. También se activa en "autenticación MCP" o "implementación de MCP".
