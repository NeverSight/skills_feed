إنشاء ومراجعة كائنات Cloudflare المتينة المدعومة بـ SQLite. يُستخدم عند إنشاء تنسيق محدد (غرف الدردشة، والألعاب متعددة اللاعبين، وأنظمة الحجز)، أو تنفيذ أساليب RPC، أو واجهة برمجة تطبيقات تخزين SQLite، أو استرداد PITR، أو الإنذارات، أو إسبات WebSocket، أو روابط الخدمة، أو مراجعة كود DO للحصول على أفضل الممارسات. تغطية شاملة لاستعلامات SQL والمعاملات وواجهة برمجة تطبيقات مساحة الاسم والاختبار.
