Verwenden Sie diese Option, wenn Sie mit Speakeasy einen Terraform-Anbieter aus einer OpenAPI-Spezifikation generieren. Behandelt Entitätsanmerkungen, CRUD-Zuordnung, Typinferenz, Workflow-Konfiguration und Veröffentlichung. Löst auf „Terraform-Anbieter“, „Terraform generieren“, „Terraform-Anbieter erstellen“, „CRUD-Zuordnung“, „x-speakeasy-entity“, „Terraform-Ressource“ und „Terraform-Registrierung“ aus.
