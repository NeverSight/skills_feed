À utiliser lors de l'implémentation de hooks de code SDK pour une logique personnalisée (pas de configuration de spécifications ou de remplacements d'exécution).
Couvre les hooks du cycle de vie du SDK : BeforeRequest, AfterSuccess, AfterError pour les en-têtes personnalisés,
télémétrie, signature HMAC et transformation requête/réponse.
