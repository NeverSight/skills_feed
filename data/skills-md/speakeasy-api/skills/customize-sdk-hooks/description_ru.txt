Используйте при реализации перехватчиков кода SDK для пользовательской логики (не конфигурации спецификации или переопределений во время выполнения).
Охватывает перехватчики жизненного цикла SDK: BeforeRequest, AfterSuccess, AfterError для пользовательских заголовков,
телеметрия, подпись HMAC и преобразование запросов/ответов.
