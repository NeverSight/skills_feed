カスタム ロジックの SDK コード フックを実装する場合に使用します (仕様設定やランタイム オーバーライドではありません)。
SDK ライフサイクル フックをカバーします: カスタム ヘッダーの BeforeRequest、AfterSuccess、AfterError、
テレメトリ、HMAC 署名、リクエスト/レスポンス変換。

「SDK フック」、「フックの追加」、「BeforeRequestHook」、「SDK のカスタム ロジック」でトリガーします。
