Guía de referencia para las mejores prácticas de especificación de OpenAPI, convenciones de nomenclatura y expresión de patrones complejos de API REST como polimorfismo, enumeraciones, carga de archivos y eventos enviados por el servidor. Úselo al escribir o mejorar las especificaciones de OpenAPI para garantizar que sigan las convenciones establecidas y generen SDK de calidad.
