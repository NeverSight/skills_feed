Используйте при создании, применении или проверке файлов наложения, включая расширения x-speakeasy. Охватывает синтаксис наложения, таргетинг JSONPath, повторные попытки, разбиение на страницы, именование, группировку, открытые перечисления, глобальные заголовки, настраиваемую безопасность. Триггеры «создать наложение», «применить наложение», «файл наложения», «x-speakeasy», «добавить расширение», «настроить повторы», «добавить нумерацию страниц», «наложение для повторных попыток».
