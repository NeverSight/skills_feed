Verwenden Sie diese Option, wenn Sie mehrere SDK-Varianten aus einem Repository generieren. Behandelt die Multi-Target-Konfiguration von Workflow.yaml, Gen.yaml pro Ziel, Monorepo-Struktur und Versionierungsstrategien. Löst auf „Multi-Target-SDK“, „Mehrere SDK-Varianten“, „Azure SDK-Variante“, „GCP SDK“, „Cloud-spezifisches SDK“, „SDK-Monorepo“ aus.
