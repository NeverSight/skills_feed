Используйте при создании нескольких вариантов SDK из одного репозитория. Охватывает многоцелевую конфигурацию workflow.yaml, gen.yaml для каждой цели, структуру монорепозитория, стратегии управления версиями. Запускается по «многоцелевому SDK», «несколько вариантов SDK», «вариант Azure SDK», «GCP SDK», «облачный SDK», «монорепозиторий SDK».
