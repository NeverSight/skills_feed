Используйте при НАСТРОЙКЕ существующего SDK, а НЕ для первоначальной генерации. Охватывает конфигурацию gen.yaml для всех языков: TypeScript, Python, Go, Java, C#, PHP, Ruby. Также рассматриваются переопределения времени выполнения (повторные попытки, тайм-ауты, выбор сервера) в коде приложения. Триггеры по «настройка SDK», «gen.yaml», «параметры SDK», «конфигурация SDK», «конфигурация SDK», «переопределение времени выполнения», «конфигурация клиента SDK», «переопределение таймаута», «конфигурация каждого вызова». Для создания НОВОГО SDK вместо этого используйте start-new-sdk-project.
