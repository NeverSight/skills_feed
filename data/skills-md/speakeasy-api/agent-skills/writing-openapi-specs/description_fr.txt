Guide de référence sur les meilleures pratiques de spécification OpenAPI, les conventions de dénomination et l'expression de modèles d'API REST complexes tels que le polymorphisme, les énumérations, les téléchargements de fichiers et les événements envoyés par le serveur. À utiliser lors de l'écriture ou de l'amélioration des spécifications OpenAPI pour garantir qu'elles respectent les conventions établies et génèrent des SDK de qualité.
