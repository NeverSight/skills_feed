既存の SDK を構成するときに使用します。初期生成には使用しません。すべての言語の gen.yaml 設定をカバーします: TypeScript、Python、Go、Java、C#、PHP、Ruby。アプリケーション コードでのランタイム オーバーライド (再試行、タイムアウト、サーバーの選択) についても説明します。 「SDK の構成」、「gen.yaml」、「SDK オプション」、「SDK 構成」、「SDK 構成」、「ランタイム オーバーライド」、「SDK クライアント構成」、「オーバーライド タイムアウト」、「呼び出しごとの構成」でトリガーされます。新しい SDK を生成する場合は、代わりに start-new-sdk-project を使用してください。
