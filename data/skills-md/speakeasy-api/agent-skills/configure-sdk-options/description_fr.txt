À utiliser lors de la CONFIGURATION d'un SDK existant - PAS pour la génération initiale. Couvre la configuration de gen.yaml pour tous les langages : TypeScript, Python, Go, Java, C#, PHP, Ruby. Couvre également les remplacements d'exécution (nouvelles tentatives, délais d'attente, sélection du serveur) dans le code de l'application. Déclencheurs sur « configurer le SDK », « gen.yaml », « options du SDK », « configuration du SDK », « configuration du SDK », « remplacement de l'exécution », « configuration du client SDK », « remplacement du délai d'attente », « configuration par appel ». Pour la génération d’un NOUVEAU SDK, utilisez plutôt start-new-sdk-project.
