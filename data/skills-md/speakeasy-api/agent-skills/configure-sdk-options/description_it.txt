Da utilizzare durante la CONFIGURAZIONE di un SDK esistente, NON per la generazione iniziale. Copre la configurazione gen.yaml per tutti i linguaggi: TypeScript, Python, Go, Java, C#, PHP, Ruby. Copre anche le sostituzioni di runtime (tentativi, timeout, selezione del server) nel codice dell'applicazione. Si attiva su "configura SDK", "gen.yaml", "opzioni SDK", "configurazione SDK", "configurazione SDK", "override runtime", "configurazione client SDK", "override timeout", "configurazione per chiamata". Per la generazione del NUOVO SDK, utilizzare invece start-new-sdk-project.
