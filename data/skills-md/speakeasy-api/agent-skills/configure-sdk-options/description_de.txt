Verwenden Sie diese Option, wenn Sie ein vorhandenes SDK KONFIGURIEREN – NICHT für die Erstgenerierung. Deckt die gen.yaml-Konfiguration für alle Sprachen ab: TypeScript, Python, Go, Java, C#, PHP, Ruby. Deckt auch Laufzeitüberschreibungen (Wiederholungsversuche, Zeitüberschreitungen, Serverauswahl) im Anwendungscode ab. Löst auf „SDK konfigurieren“, „gen.yaml“, „SDK-Optionen“, „SDK-Konfiguration“, „SDK-Konfiguration“, „Laufzeitüberschreibung“, „SDK-Client-Konfiguration“, „Timeout überschreiben“, „Konfiguration pro Aufruf“ aus. Verwenden Sie für die NEUE SDK-Generierung stattdessen start-new-sdk-project.
