Используйте при создании поставщика Terraform на основе спецификации OpenAPI с помощью Speakeasy. Охватывает аннотации сущностей, сопоставление CRUD, определение типа, настройку рабочего процесса и публикацию. Триггеры по «поставщику терраформирования», «созданию терраформирования», «созданию провайдера терраформирования», «сопоставлению CRUD», «x-speakeasy-entity», «ресурсу терраформирования», «реестру терраформирования».
