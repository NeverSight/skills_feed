Guide complet pour styliser les composants React avec Vanilla Extract (intégration Vite, styles typés, thèmes, composition, API globales, recettes, thèmes d'exécution dynamique et saupoudrages facultatifs). Utilisez cette compétence lorsque les tâches utilisateur impliquent l'implémentation ou la refactorisation du style des composants React avec Vanilla Extract, en particulier lorsque vous mettez l'accent sur la génération CSS sans exécution, une forte ergonomie TypeScript, des variantes évolutives et des modèles de thème sûrs.
