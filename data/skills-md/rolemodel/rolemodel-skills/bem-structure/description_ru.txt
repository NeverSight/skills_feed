Экспертное руководство по написанию, рефакторингу и структурированию CSS с использованием методологии BEM (модификатор блочного элемента). Обеспечивает правильные соглашения об именах классов CSS, структуру компонентов и интеграцию системы проектирования Optics для поддерживаемых и масштабируемых таблиц стилей.
