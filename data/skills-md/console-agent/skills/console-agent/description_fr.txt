Créez des agents IA avec console.agent() - le jQuery des agents IA. Déposez console.agent(...) n'importe où dans votre code pour des flux de travail agents avec la simplicité de console.log(). À utiliser lors de l'ajout de fonctionnalités d'agent IA, du débogage avec l'IA, de l'audit de sécurité, de la journalisation intelligente ou de l'analyse d'exécution.
