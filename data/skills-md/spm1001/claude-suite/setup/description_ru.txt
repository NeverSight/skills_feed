Установите поведенческие навыки из трусов. Используйте FIRST при подключении нового компьютера или после новой установки Claude Code. Создает символические ссылки для жизненного цикла сеанса, утилит и, при необходимости, предлагает репозитории инструментов (todoist-gtd, garde-manger). Запускается по командам «помогите мне настроить», «установить навыки», «/setup». (пользователь)
