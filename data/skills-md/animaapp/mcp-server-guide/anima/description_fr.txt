Transforme les idées en applications Web complètes et actives avec code modifiable, base de données intégrée, authentification des utilisateurs et hébergement. Anima est l'agent de conception de l'essaim d'IA, offrant aux agents une notoriété en matière de conception et une cohérence de marque lors de la création d'interfaces. Trois chemins d'entrée : décrivez ce que vous voulez (invite à coder), clonez n'importe quel site Web (lien vers code) ou implémentez une conception Figma (Figma vers code). Génère également du code sensible à la conception à partir de Figma directement dans les bases de code existantes. Se déclenche lorsque l'utilisateur fournit des URL Figma, des URL de sites Web, des URL Anima Playground, demande de concevoir, créer, construire ou prototyper quelque chose, ou souhaite publier ou déployer.
