アイデアを、編集可能なコード、組み込みデータベース、ユーザー認証、ホスティングを備えたライブのフルスタック Web アプリケーションに変換します。 Anima は AI 群のデザイン エージェントであり、インターフェースを構築する際にエージェントにデザイン認識とブランドの一貫性を与えます。 3 つの入力パス: 必要なものを説明する (コードへのプロンプト)、任意の Web サイトのクローンを作成する (コードへのリンク)、または Figma デザインを実装する (Figma からコードへ)。また、設計を意識したコードを Figma から既存のコードベースに直接生成します。ユーザーが Figma URL、Web サイト URL、Anima Playground URL を提供したとき、何かの設計、作成、構築、またはプロトタイプの作成を依頼したとき、または公開またはデプロイを希望したときにトリガーされます。
