Un conjunto de herramientas para crear artefactos HTML sofisticados de múltiples componentes para claude.ai utilizando tecnologías web frontend modernas (React, Tailwind CSS, shadcn/ui). Úselo para artefactos complejos que requieren administración de estado, enrutamiento o componentes shadcn/ui. No para artefactos HTML/JSX simples de un solo archivo.
