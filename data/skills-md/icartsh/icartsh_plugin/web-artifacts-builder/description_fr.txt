Une boîte à outils pour créer des artefacts HTML multi-composants sophistiqués pour claude.ai à l'aide de technologies Web frontales modernes (React, Tailwind CSS, shadcn/ui). Utilisez-le pour les artefacts complexes qui nécessitent des composants de gestion d'état, de routage ou shadcn/ui. Pas pour les simples artefacts HTML/JSX à fichier unique.
