Aplica un enfoque estructurado y reflexivo a la resolución de problemas para tareas complejas que requieren análisis, revisión y prueba de hipótesis de varias etapas. Se utiliza al descomponer problemas complejos, desarrollar planes adaptativos, realizar análisis que requieren corrección de rumbo, abordar problemas de alcance ambiguo, implementar soluciones de múltiples etapas y realizar trabajos basados ​​en hipótesis.
