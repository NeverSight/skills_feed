Casos de uso: (1) Generar archivos README con insignias y secciones, (2) Crear automáticamente registros de cambios a partir del historial de git, (3) Crear tablas de contenido, (4) Escribir pautas de contribución, (5) Dar formato a documentos técnicos, (6) Código de documento con resaltado de sintaxis
