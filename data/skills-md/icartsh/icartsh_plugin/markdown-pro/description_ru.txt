НАВЫКИ документирования Markdown профессионального уровня для создания доработанных файлов README, журналов изменений, руководств по участию и технической документации.  
Варианты использования: (1) Создание файлов README со значками и разделами, (2) Автоматическое создание журналов изменений из истории git, (3) Создание оглавлений, (4) Написание рекомендаций по участию, (5) Форматирование технической документации, (6) Документирование кода с подсветкой синтаксиса
