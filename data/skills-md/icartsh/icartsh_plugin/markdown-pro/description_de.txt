Eine Markdown-Dokumentationskompetenz auf professionellem Niveau zum Erstellen ausgefeilter README-Dateien, Änderungsprotokolle, Beitragsleitfäden und technischer Dokumentation.  
Anwendungsfälle: (1) READMEs mit Abzeichen und Abschnitten generieren, (2) Änderungsprotokolle automatisch aus dem Git-Verlauf erstellen, (3) Inhaltsverzeichnisse erstellen, (4) Beitragsrichtlinien schreiben, (5) technische Dokumente formatieren, (6) Code mit Syntaxhervorhebung dokumentieren
