Une COMPÉTENCE de documentation Markdown de niveau professionnel pour créer des fichiers README, des journaux de modifications, des guides de contribution et une documentation technique raffinés.  
Cas d'utilisation : (1) Générer des README avec des badges et des sections, (2) Créer automatiquement des journaux de modifications à partir de l'historique git, (3) Créer des tables des matières, (4) Rédiger des directives de contribution, (5) Formater des documents techniques, (6) Code de document avec coloration syntaxique
