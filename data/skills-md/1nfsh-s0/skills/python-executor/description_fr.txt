Exécutez du code Python dans un environnement sandbox sécurisé via [inference.sh](https://inference.sh). Préinstallé : NumPy, Pandas, Matplotlib, requêtes, BeautifulSoup, Selenium, Playwright, MoviePy, Pillow, OpenCV, trimesh et plus de 100 autres bibliothèques. Utiliser pour : le traitement de données, le web scraping, la manipulation d'images, la création de vidéos, le traitement de modèles 3D, la génération de PDF, les appels d'API, les scripts d'automatisation. Déclencheurs : python, exécution de code, exécution de script, web scraping, analyse de données, traitement d'image, montage vidéo, modèles 3D, automatisation, pandas, matplotlib
