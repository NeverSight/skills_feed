تصميم مكونات React باتباع إرشادات بنية Tetris. يُستخدم عند إنشاء مكونات جديدة، أو إعادة هيكلة المكونات، أو تحسين الأداء، أو تحسين بنية المكونات. يتم التشغيل التلقائي لعبارات مثل "إنشاء مكون"، أو "إعادة بناء هذا المكون"، أو "تحسين العرض"، أو "تحسين تصميم المكون". يؤكد على دمج المكونات والأنماط الموحدة والاستخدام الفعال للخطاف.
