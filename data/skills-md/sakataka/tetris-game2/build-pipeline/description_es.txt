Ejecute un proceso de compilación completo con detección de código inactivo, formateo, linting, verificación de tipos, pruebas y compilación de producción. Úselo cuando el usuario mencione la compilación, la ejecución del proceso completo, la verificación de la calidad del código o la preparación para la implementación. Se activa automáticamente en frases como "construir el proyecto", "ejecutar todas las comprobaciones", "prepararse para la producción" o "validar la calidad del código".
