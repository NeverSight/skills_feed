Compréhension de l'interface utilisateur multimodale et planification en une seule étape via les API Responses compatibles OpenAI. À utiliser lorsque vous avez besoin d'AIQuery/AIAssert et de plan-next pour extraire les coordonnées des éléments de l'interface utilisateur, valider les assertions de l'interface utilisateur, résumer les captures d'écran ou décider de la prochaine action de l'interface utilisateur à partir d'une image. Les agents externes gèrent l'exécution via adb/hdc et des boucles en plusieurs étapes. Par défaut, les modèles Doubao mais peuvent être pointés vers d'autres fournisseurs multimodaux via l'URL de base, la clé API et le nom du modèle.
