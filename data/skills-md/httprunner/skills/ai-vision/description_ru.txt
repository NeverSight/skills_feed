Понимание мультимодального пользовательского интерфейса и пошаговое планирование с помощью API-интерфейсов ответов, совместимых с OpenAI. Используйте, когда вам нужны AIQuery/AIAssert и plan-next для извлечения координат элементов пользовательского интерфейса, проверки утверждений пользовательского интерфейса, обобщения снимков экрана или выбора следующего действия пользовательского интерфейса на основе изображения. Внешние агенты обрабатывают выполнение через adb/hdc и многошаговые циклы. По умолчанию используются модели Doubao, но на других мультимодальных поставщиков можно указать с помощью базового URL-адреса, ключа API и названия модели.
