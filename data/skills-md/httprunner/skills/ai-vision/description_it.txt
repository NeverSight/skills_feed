Comprensione dell'interfaccia utente multimodale e pianificazione in un unico passaggio tramite API di risposta compatibili con OpenAI. Utilizzalo quando hai bisogno di AIQuery/AIAssert e plan-next per estrarre le coordinate degli elementi dell'interfaccia utente, convalidare le asserzioni dell'interfaccia utente, riepilogare schermate o decidere la successiva azione dell'interfaccia utente da un'immagine. Gli agenti esterni gestiscono l'esecuzione tramite adb/hdc e cicli multi-step. Per impostazione predefinita sono modelli Doubao ma possono essere indirizzati ad altri fornitori multimodali tramite URL di base, chiave API e nome del modello.
