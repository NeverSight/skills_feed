Comprensión de la interfaz de usuario multimodal y planificación en un solo paso a través de API de respuestas compatibles con OpenAI. Úselo cuando necesite AIQuery/AIAssert y plan-next para extraer coordenadas de elementos de la interfaz de usuario, validar afirmaciones de la interfaz de usuario, resumir capturas de pantalla o decidir la siguiente acción de la interfaz de usuario a partir de una imagen. Los agentes externos manejan la ejecución a través de adb/hdc y bucles de varios pasos. El valor predeterminado es el modelo Doubao, pero se puede apuntar a otros proveedores multimodales a través de la URL base, la clave API y el nombre del modelo.
