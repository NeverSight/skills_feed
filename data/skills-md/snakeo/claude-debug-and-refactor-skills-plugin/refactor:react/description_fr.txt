Refactorisez le code React et TypeScript pour améliorer la maintenabilité, la lisibilité et les performances. Cette compétence transforme les composants React complexes en code propre et bien structuré suivant les modèles React 19 modernes. Il traite du gonflement des composants, du perçage des accessoires, des rendus inutiles et de l'utilisation inappropriée des crochets. Tire parti des fonctionnalités de React 19, notamment le compilateur React pour la mémorisation automatique, les actions pour la gestion des formulaires, useOptimistic pour un retour immédiat sur l'interface utilisateur, le hook use() pour les données asynchrones et les composants serveur pour des performances optimales.
