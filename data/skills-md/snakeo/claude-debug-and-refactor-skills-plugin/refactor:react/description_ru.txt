Выполните рефакторинг кода React и TypeScript, чтобы улучшить удобство обслуживания, читаемость и производительность. Этот навык преобразует сложные компоненты React в чистый, хорошо структурированный код, соответствующий современным шаблонам React 19. Он устраняет раздувание компонентов, сверление опор, ненужные повторные рендеринги и неправильное использование хуков. Использует функции React 19, включая компилятор React для автоматического запоминания, действия для обработки форм, useOptimistic для немедленной обратной связи пользовательского интерфейса, ловушку use() для асинхронных данных и серверные компоненты для оптимальной производительности.
