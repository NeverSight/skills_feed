Refactor React- und TypeScript-Code zur Verbesserung der Wartbarkeit, Lesbarkeit und Leistung. Diese Fähigkeit wandelt komplexe React-Komponenten in sauberen, gut strukturierten Code um, der modernen React-19-Mustern folgt. Es behebt das Aufblähen von Komponenten, das Bohren von Stützen, unnötiges erneutes Rendern und die unsachgemäße Verwendung von Haken. Nutzt React 19-Funktionen, darunter den React Compiler für die automatische Speicherung, Aktionen für die Formularverarbeitung, useOptimistic für sofortiges UI-Feedback, den use()-Hook für asynchrone Daten und Serverkomponenten für optimale Leistung.
