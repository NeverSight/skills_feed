React および TypeScript コードをリファクタリングして、保守性、可読性、パフォーマンスを向上させます。このスキルは、複雑な React コンポーネントを、最新の React 19 パターンに従って、クリーンで適切に構造化されたコードに変換します。コンポーネントの肥大化、プロップの穴あけ、不必要な再レンダリング、不適切なフックの使用に対処します。自動メモ化のための React コンパイラー、フォーム処理のためのアクション、即時 UI フィードバックのための useOptimistic、非同期データのための use() フック、最適なパフォーマンスのためのサーバー コンポーネントなどの React 19 機能を活用します。
