Систематически устраняйте проблемы с инфраструктурой как кодом Terraform. Этот навык помогает диагностировать и решать проблемы, специфичные для Terraform, включая конфликты блокировки состояния, сбои аутентификации поставщика, циклы зависимости ресурсов, обнаружение смещения состояния, сбои импорта, конфликты версий модулей и ошибки планирования/применения. Обеспечивает отладку TF_LOG, использование консоли terraform, команды манипулирования состоянием и лучшие практики CI/CD для автоматизации инфраструктуры.
