Eseguire il debug sistematico dei problemi dell'infrastruttura come codice di Terraform. Questa competenza aiuta a diagnosticare e risolvere problemi specifici di Terraform, inclusi conflitti di blocco dello stato, errori di autenticazione del provider, cicli di dipendenza delle risorse, rilevamento della deviazione dello stato, errori di importazione, conflitti di versione del modulo ed errori di pianificazione/applicazione. Fornisce debug TF_LOG, utilizzo della console Terraform, comandi di manipolazione dello stato e best practice CI/CD per l'automazione dell'infrastruttura.
