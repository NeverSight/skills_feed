Refactorisez le code Flutter/Dart pour améliorer la maintenabilité, la lisibilité et les performances. Cette compétence applique les fonctionnalités de Dart 3 telles que les enregistrements, les modèles et les classes scellées, implémente une gestion d'état appropriée avec Riverpod ou BLoC et utilise Freezed pour les modèles immuables. Il résout les widgets monolithiques, les constructeurs const manquants, l'utilisation inappropriée de BuildContext et l'imbrication profonde. Appliquez-le lorsque vous remarquez que les widgets en font trop, des problèmes de performances dus à des reconstructions inutiles ou des modèles Dart 2 hérités.
