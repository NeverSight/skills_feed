Refactorice el código Flutter/Dart para mejorar la mantenibilidad, la legibilidad y el rendimiento. Esta habilidad aplica características de Dart 3 como registros, patrones y clases selladas, implementa una gestión de estado adecuada con Riverpod o BLoC y utiliza Freezed para modelos inmutables. Aborda widgets monolíticos, constructores constantes faltantes, uso inadecuado de BuildContext y anidamiento profundo. Aplíquelo cuando note que los widgets hacen demasiado, problemas de rendimiento debido a reconstrucciones innecesarias o patrones heredados de Dart 2.
