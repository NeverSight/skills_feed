تصحيح مشكلات React بشكل منهجي. يُستخدم عند مواجهة أخطاء مكون مثل "لا يمكن قراءة خاصية غير محددة"، أو حلقات إعادة عرض لا نهائية مع "عدد كبير جدًا من عمليات إعادة العرض"، أو عمليات الإغلاق التي لا معنى لها في الخطافات، أو تحذيرات دعم المفاتيح، أو تسرب الذاكرة مع useEffect، أو عدم تطابق الماء في تطبيقات SSR/SSG، أو انتهاكات قاعدة الخطاف، أو تحذيرات التبعية المفقودة، أو اختناقات الأداء. يغطي المكونات الوظيفية والخطافات وأنماط React 18/19 الحديثة بما في ذلك مكونات الخادم والميزات المتزامنة.
