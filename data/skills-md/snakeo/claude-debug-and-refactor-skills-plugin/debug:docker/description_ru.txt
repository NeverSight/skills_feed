Отлаживайте контейнеры, образы и инфраструктуру Docker с помощью систематических методов диагностики. Этот навык предоставляет исчерпывающие рекомендации по устранению неполадок кодов выхода контейнеров, уничтожений OOM, сбоев сборки образа, проблем с сетью, проблем с монтированием томов и ошибок разрешений. Охватывает четырехэтапную методологию отладки: от быстрой оценки до глубокого анализа, основные команды Docker, методы отладки контейнеров для минимальных образов и устранение неполадок для конкретных платформ для Windows, Mac и Linux.
