Déboguez les conteneurs, les images et l'infrastructure Docker avec des techniques de diagnostic systématiques. Cette compétence fournit des conseils complets pour dépanner les codes de sortie de conteneur, les suppressions de MOO, les échecs de création d'image, les problèmes de mise en réseau, les problèmes de montage de volume et les erreurs d'autorisation. Couvre la méthodologie de débogage en quatre phases, de l'évaluation rapide à l'analyse approfondie, les commandes Docker essentielles, les techniques de débogage des conteneurs pour les images minimales et le dépannage spécifique à la plate-forme pour Windows, Mac et Linux.
