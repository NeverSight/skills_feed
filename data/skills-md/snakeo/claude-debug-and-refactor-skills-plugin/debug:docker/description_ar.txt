تصحيح أخطاء حاويات Docker والصور والبنية التحتية باستخدام تقنيات التشخيص المنهجية. توفر هذه المهارة إرشادات شاملة لاستكشاف أخطاء رموز خروج الحاوية وإصلاحها، وحالات قتل OOM، وفشل إنشاء الصورة، ومشكلات الشبكة، ومشكلات تحميل وحدة التخزين، وأخطاء الأذونات. يغطي منهجية تصحيح الأخطاء المكونة من أربع مراحل بدءًا من التقييم السريع وحتى التحليل العميق وأوامر Docker الأساسية وتقنيات حاوية تصحيح الأخطاء للحد الأدنى من الصور واستكشاف الأخطاء وإصلاحها الخاصة بالنظام الأساسي لأنظمة التشغيل Windows وMac وLinux.
