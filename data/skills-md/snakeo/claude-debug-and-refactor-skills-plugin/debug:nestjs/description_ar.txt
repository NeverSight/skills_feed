تصحيح مشكلات NestJS بشكل منهجي. يُستخدم عند مواجهة أخطاء حقن التبعيات مثل "يتعذر على Nest حل التبعيات"، أو مشكلات استيراد الوحدة، أو التبعيات الدائرية بين الخدمات أو الوحدات، أو مشكلات الحراسة والاعتراض، أو مشكلات تكوين الديكور، أو أخطاء اتصال الخدمات الصغيرة، أو فشل بوابة WebSocket، أو أخطاء التحقق من صحة التوجيه، أو أي مشكلات وقت تشغيل خاصة بـ NestJS تتطلب التشخيص.
