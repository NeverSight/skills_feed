قم بإعادة تصميم كود Scikit-Learn والتعلم الآلي لتحسين قابلية الصيانة والتكرار والالتزام بأفضل الممارسات. تعمل هذه المهارة على تحويل كود ML العامل إلى خطوط أنابيب جاهزة للإنتاج تمنع تسرب البيانات وتضمن نتائج قابلة للتكرار. وهو يعالج المعالجة المسبقة خارج خطوط الأنابيب، ومعلمات الحالة العشوائية المفقودة، والتحقق المتبادل غير المناسب، والمحولات المخصصة التي لا تتبع اصطلاحات sklearn API. ينفذ أنماط خطوط الأنابيب ومحولات الأعمدة المناسبة، والضبط المنهجي للمعلمات الفائقة، ومقاييس التقييم المناسبة.
