Eseguire il debug sistematico dei problemi Next.js. Da utilizzare quando si riscontrano errori SSR, mancate corrispondenze di idratazione come "Il contenuto del testo non corrisponde", problemi di routing con App Router o Pages Router, errori di build, problemi di importazione dinamica, errori di routing API, problemi middleware, problemi di memorizzazione nella cache e riconvalida o colli di bottiglia delle prestazioni. Copre sia le architetture Pages Router che App Router.
