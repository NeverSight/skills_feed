تصحيح مشكلات Next.js بشكل منهجي. يُستخدم عند مواجهة أخطاء SSR، أو عدم تطابق الماء مثل "محتوى النص غير متطابق"، أو مشكلات التوجيه مع جهاز توجيه التطبيق أو جهاز توجيه الصفحات، أو حالات فشل الإنشاء، أو مشكلات الاستيراد الديناميكي، أو أخطاء توجيه واجهة برمجة التطبيقات، أو مشكلات البرامج الوسيطة، أو مشكلات التخزين المؤقت وإعادة التحقق، أو اختناقات الأداء. يغطي كلاً من بنيات جهاز توجيه الصفحات وجهاز توجيه التطبيقات.
