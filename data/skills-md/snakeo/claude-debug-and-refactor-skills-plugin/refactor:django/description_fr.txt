Refactorisez le code Django/Python pour améliorer la maintenabilité, la lisibilité et le respect des meilleures pratiques. Transforme les vues volumineuses, les requêtes N+1 et les modèles obsolètes en code Django propre et moderne. Applique les fonctionnalités de Python 3.12+ telles que la syntaxe des paramètres de type et le décorateur @override, les modèles Django 5+ comme GeneratedField et les vues asynchrones, l'architecture de la couche de service et les conventions PEP 8. Identifie et corrige les anti-modèles, y compris les valeurs par défaut mutables, les exceptions simples et l'utilisation inappropriée de l'ORM.
