تكوينات Refactor Docker لتحسين الأمان والأداء وقابلية الصيانة. يحول ملفات Dockerfiles وملفات docker-compose غير الآمنة إلى حاويات جاهزة للإنتاج باتباع أفضل الممارسات لعام 2025. ينفذ عمليات إنشاء متعددة المراحل، ومستخدمين غير جذريين، وإصدارات الصور المثبتة، والفحوصات الصحية، وإدارة الأسرار، وتجزئة الشبكة. يعمل على إصلاح الأنماط المضادة الشائعة مثل التشغيل كجذر، وبيانات الاعتماد المشفرة، وdockerignore المفقودة، والصور المتضخمة.
