Refactorice las configuraciones de Docker para mejorar la seguridad, el rendimiento y la mantenibilidad. Transforma archivos Dockerfiles y archivos docker-compose inseguros en contenedores listos para producción siguiendo las mejores prácticas de 2025. Implementa compilaciones de varias etapas, usuarios no root, versiones de imágenes fijadas, controles de estado, administración de secretos y segmentación de red. Corrige antipatrones comunes, como ejecutar como root, credenciales codificadas, falta .dockerignore e imágenes infladas.
