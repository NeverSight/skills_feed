Eseguire il debug di applicazioni ASP.NET Core e .NET con approcci diagnostici sistematici. Questa competenza copre la risoluzione dei problemi relativi agli errori del contenitore di inserimento delle dipendenze, ai problemi della pipeline del middleware, ai problemi di query di Entity Framework Core, agli errori di associazione della configurazione, ai problemi di autenticazione/autorizzazione e agli errori di avvio. Include strumenti di debug di Visual Studio e VS Code, dotnet-trace, dotnet-dump, contatori dotnet, configurazione di Serilog, integrazione di Application Insights e metodologia di debug in quattro fasi.
