تصحيح أخطاء تطبيقات ASP.NET Core و.NET باستخدام أساليب التشخيص المنهجية. تغطي هذه المهارة استكشاف أخطاء حاوية حقن التبعية وإصلاحها، ومشكلات مسارات البرامج الوسيطة، ومشكلات استعلام Entity Framework Core، وفشل ربط التكوين، ومشكلات المصادقة/الترخيص، وفشل بدء التشغيل. يتضمن تصحيح أخطاء Visual Studio وVS Code، وتتبع dotnet، وdotnet-dump، وأدوات عدادات dotnet، وتكوين Serilog، وتكامل Application Insights، ومنهجية تصحيح الأخطاء على أربع مراحل.
