Отлаживайте приложения ASP.NET Core и .NET с помощью систематических подходов к диагностике. Этот навык охватывает устранение ошибок контейнера внедрения зависимостей, проблем с конвейером промежуточного программного обеспечения, проблем с запросами Entity Framework Core, сбоев привязки конфигурации, проблем с аутентификацией/авторизацией и сбоев запуска. Включает инструменты отладки Visual Studio и VS Code, dotnet-trace, dotnet-dump, dotnet-counters, конфигурацию Serilog, интеграцию Application Insights и методологию четырехфазной отладки.
