Depure aplicaciones ASP.NET Core y .NET con enfoques de diagnóstico sistemáticos. Esta habilidad cubre la resolución de problemas de errores de contenedor de inyección de dependencia, problemas de canalización de middleware, problemas de consulta de Entity Framework Core, fallas de enlace de configuración, problemas de autenticación/autorización y fallas de inicio. Incluye depuración de Visual Studio y VS Code, herramientas dotnet-trace, dotnet-dump, dotnet-counters, configuración de Serilog, integración de Application Insights y metodología de depuración de cuatro fases.
