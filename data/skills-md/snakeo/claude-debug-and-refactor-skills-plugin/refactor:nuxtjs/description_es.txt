Refactorice el código Nuxt.js/Vue para mejorar la mantenibilidad, la legibilidad y el cumplimiento de las mejores prácticas. Identifica y corrige infracciones DRY, componentes de gran tamaño, anidamiento profundo, infracciones de SRP, antipatrones de recuperación de datos con useFetch/useAsyncData/$fetch, organización componible deficiente y lógica mixta de negocios/presentación. Aplica patrones de Nuxt 3 que incluyen importaciones automáticas, recuperación adecuada de datos, componentes componibles de responsabilidad única, integración de TypeScript, configuración de tiempo de ejecución, rutas de servidor Nitro, capas Nuxt, patrones de middleware, administración del estado de Pinia y optimizaciones de rendimiento.
