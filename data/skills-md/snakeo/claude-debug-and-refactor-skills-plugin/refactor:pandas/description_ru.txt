Рефакторинг кода Pandas для улучшения удобства обслуживания, читаемости и производительности. Выявляет и исправляет циклы/.iterrows(), которые необходимо векторизовать, чрезмерное использование .apply() там, где существуют векторизованные альтернативы, шаблоны цепочки индексации, использование inplace=True, неэффективные типы dtypes, отсутствующие возможности цепочки методов, сложные фильтры, операции слияния без проверки и шаблоны SettingWithCopyWarning. Применяет функции Pandas 2.0+, включая серверную часть PyArrow, копирование при записи, векторизованные операции, цепочку методов, .query()/.eval(), оптимизированные типы данных и шаблоны конвейеров.
