Refactorisez le code Express.js/Node.js pour améliorer la maintenabilité, la lisibilité et le respect des meilleures pratiques. Transforme l'enfer des rappels, les gros gestionnaires de routes et les modèles obsolètes en code JavaScript/TypeScript propre et moderne. Applique l'architecture asynchrone/attente, contrôleur-service-référentiel, les modèles de middleware appropriés et les modules ESM. Identifie et corrige les anti-modèles, notamment le blocage de la boucle d'événements, la gestion inappropriée des erreurs, forEach avec des rappels asynchrones et les fuites de mémoire.
