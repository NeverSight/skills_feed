Eseguire il debug sistematico dei problemi di Rails. Da utilizzare quando si riscontrano errori ActiveRecord come RecordNotFound, problemi di routing, problemi di query N+1 rilevati da Bullet, problemi di pipeline delle risorse, errori di migrazione, conflitti di gem, errori ActionController, problemi di token CSRF o qualsiasi errore dell'applicazione Ruby on Rails che richiede una diagnosi.
