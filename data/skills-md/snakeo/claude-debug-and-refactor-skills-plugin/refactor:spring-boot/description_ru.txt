Выполните рефакторинг кода Spring Boot и Java, чтобы улучшить удобство обслуживания, читабельность и соответствие передовым корпоративным практикам. Этот навык превращает беспорядочные приложения Spring Boot в чистые, хорошо структурированные решения, соответствующие принципам SOLID и соглашениям Spring Boot 3.x. Он касается толстых контроллеров, неправильных границ транзакций, антишаблонов внедрения полей и разрозненной конфигурации. Использует функции Java 21+, включая шаблоны записей, сопоставление шаблонов для коммутатора, виртуальные потоки и упорядоченные коллекции.
