Refaktorieren Sie PHP/Laravel-Code, um die Wartbarkeit, Lesbarkeit und Einhaltung von Best Practices zu verbessern. Diese Fähigkeit transformiert Code mithilfe moderner PHP 8.3+/8.4+-Funktionen wie Eigenschafts-Hooks und typisierte Konstanten, Laravel 11+-Muster einschließlich Aktionen und vereinfachter Struktur sowie SOLID-Prinzipien. Es befasst sich mit fetten Controllern, Codeduplizierung, N+1-Abfragen und fehlenden Typdeklarationen. Wenden Sie es an, wenn Sie Geschäftslogik in Controllern oder älteren PHP-Mustern bemerken.
