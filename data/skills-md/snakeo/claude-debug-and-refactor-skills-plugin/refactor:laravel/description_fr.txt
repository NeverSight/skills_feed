Refactorisez le code PHP/Laravel pour améliorer la maintenabilité, la lisibilité et le respect des meilleures pratiques. Cette compétence transforme le code à l'aide des fonctionnalités modernes de PHP 8.3+/8.4+ telles que les hooks de propriété et les constantes typées, les modèles Laravel 11+ comprenant les actions et la structure simplifiée, ainsi que les principes SOLID. Il traite les gros contrôleurs, la duplication de code, les requêtes N+1 et les déclarations de type manquantes. Appliquez-le lorsque vous remarquez une logique métier dans les contrôleurs ou les modèles PHP hérités.
