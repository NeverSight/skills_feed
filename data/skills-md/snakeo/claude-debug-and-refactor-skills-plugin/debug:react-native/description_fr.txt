Déboguez systématiquement les problèmes de React Native. À utiliser lorsque vous rencontrez des erreurs de module natif telles que « Le module natif ne peut pas être nul », des problèmes de bundle Metro, notamment des conflits de ports et une corruption du cache, des échecs de construction spécifiques à la plate-forme pour iOS CocoaPods ou Android Gradle, des problèmes de communication de pont, des échecs de compilation de bytecode du moteur Hermes, des erreurs fatales d'écran rouge ou des problèmes de migration de nouvelle architecture avec les TurboModules et le moteur de rendu Fabric.
