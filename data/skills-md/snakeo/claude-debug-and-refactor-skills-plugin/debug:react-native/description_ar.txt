تصحيح مشكلات React Native بشكل منهجي. يُستخدم عند مواجهة أخطاء الوحدة الأصلية مثل "الوحدة الأصلية لا يمكن أن تكون خالية"، أو مشكلات حزمة Metro بما في ذلك تعارضات المنافذ وفساد ذاكرة التخزين المؤقت، أو فشل البنية الخاصة بالنظام الأساسي لنظام iOS CocoaPods أو Android Gradle، أو مشكلات اتصال الجسر، أو فشل تجميع الكود الثانوي لمحرك Hermes، أو الأخطاء الفادحة للشاشة الحمراء، أو مشكلات ترحيل البنية الجديدة مع TurboModules وعارض النسيج.
