تصحيح مشكلات Nuxt.js بشكل منهجي. يُستخدم عند مواجهة أخطاء SSR، أو مشكلات خادم Nitro، أو عدم تطابق الماء مثل "عدم تطابق النص/العقدة"، أو مشكلات قابلة للتركيب مع useFetch أو useAsyncData، أو فشل تهيئة البرنامج المساعد، أو تعارضات الوحدة النمطية، أو مشكلات الاستيراد التلقائي، أو أخطاء وقت التشغيل الخاصة بـ Vue في سياق Nuxt.
