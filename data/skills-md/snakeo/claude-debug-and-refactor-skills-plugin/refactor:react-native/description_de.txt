Refactor React Native- und TypeScript-Code zur Verbesserung der Wartbarkeit, Lesbarkeit und Leistung für plattformübergreifende mobile Anwendungen. Diese Fähigkeit wandelt komplexen mobilen Code in saubere, gut strukturierte Lösungen um, die den React Native New Architecture-Mustern folgen, einschließlich Fabric, TurboModules und JSI. Es befasst sich mit FlatList-Leistungsproblemen, Prop Drilling, plattformspezifischer Codeorganisation und Inline-Stilen. Nutzt Expo SDK 52+-Funktionen, React Navigation v7 und Reanimated für flüssige 60-fps-Animationen.
