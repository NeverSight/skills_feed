Выполните рефакторинг кода React Native и TypeScript, чтобы улучшить удобство обслуживания, читаемость и производительность кроссплатформенных мобильных приложений. Этот навык преобразует сложный мобильный код в чистые, хорошо структурированные решения в соответствии с шаблонами новой архитектуры React Native, включая Fabric, TurboModules и JSI. Он решает проблемы производительности FlatList, детализацию свойств, организацию кода для конкретной платформы и встроенные стили. Использует функции Expo SDK 52+, React Navigation v7 и Reanimated для плавной анимации со скоростью 60 кадров в секунду.
