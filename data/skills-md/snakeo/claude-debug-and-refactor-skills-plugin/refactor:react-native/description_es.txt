Refactorice el código React Native y TypeScript para mejorar la mantenibilidad, la legibilidad y el rendimiento de las aplicaciones móviles multiplataforma. Esta habilidad transforma código móvil complejo en soluciones limpias y bien estructuradas que siguen los patrones de nueva arquitectura de React Native, incluidos Fabric, TurboModules y JSI. Aborda problemas de rendimiento de FlatList, perforación de accesorios, organización de código específica de la plataforma y estilos en línea. Aprovecha las funciones de Expo SDK 52+, React Navigation v7 y Reanimated para lograr animaciones fluidas a 60 fps.
