Refactoring del codice React Native e TypeScript per migliorare manutenibilità, leggibilità e prestazioni per applicazioni mobili multipiattaforma. Questa competenza trasforma codice mobile complesso in soluzioni pulite e ben strutturate seguendo i modelli React Native New Architecture tra cui Fabric, TurboModules e JSI. Risolve problemi di prestazioni di FlatList, perforazione degli oggetti di scena, organizzazione del codice specifico della piattaforma e stili in linea. Sfrutta le funzionalità Expo SDK 52+, React Navigation v7 e Reanimated per animazioni fluide a 60 fps.
