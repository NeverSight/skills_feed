Refactorisez le code React Native et TypeScript pour améliorer la maintenabilité, la lisibilité et les performances des applications mobiles multiplateformes. Cette compétence transforme le code mobile complexe en solutions propres et bien structurées suivant les modèles de nouvelle architecture React Native, notamment Fabric, TurboModules et JSI. Il résout les problèmes de performances de FlatList, l’exploration des accessoires, l’organisation du code spécifique à la plate-forme et les styles en ligne. Tire parti des fonctionnalités Expo SDK 52+, React Navigation v7 et Reanimated pour des animations fluides à 60 ips.
