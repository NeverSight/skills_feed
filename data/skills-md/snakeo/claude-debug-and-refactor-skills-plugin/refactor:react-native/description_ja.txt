React Native および TypeScript コードをリファクタリングして、クロスプラットフォームのモバイル アプリケーションの保守性、可読性、パフォーマンスを向上させます。このスキルは、複雑なモバイル コードを、ファブリック、ターボモジュール、JSI などの React Native 新しいアーキテクチャ パターンに従って、クリーンで適切に構造化されたソリューションに変換します。これは、FlatList のパフォーマンスの問題、プロパティのドリルリング、プラットフォーム固有のコード構成、およびインライン スタイルに対処します。 Expo SDK 52+ の機能、React Navigation v7、および Reanimated を活用して、スムーズな 60fps アニメーションを実現します。
