Refactor React Native وTypeScript لتحسين قابلية الصيانة وقابلية القراءة والأداء لتطبيقات الهاتف المحمول عبر الأنظمة الأساسية. تعمل هذه المهارة على تحويل كود الهاتف المحمول المعقد إلى حلول نظيفة ومنظمة بشكل جيد تتبع أنماط React Native New Architecture بما في ذلك Fabric وTurboModules وJSI. وهو يعالج مشكلات أداء FlatList، وحفر الدعامات، وتنظيم التعليمات البرمجية الخاصة بالمنصة، والأنماط المضمنة. يستفيد من ميزات Expo SDK 52+ وReact Navigation v7 وReanimated للحصول على رسوم متحركة سلسة بمعدل 60 إطارًا في الثانية.
