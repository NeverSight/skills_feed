يرشد تنظيم الملفات والمجلدات باستخدام بنية الشجرة النمطية الكسورية: الدلائل المتشابهة ذاتيًا، والتغليف، والمجلدات/المشتركة، ونمط المكتبة المصغرة، وتسمية حالة الكباب. يُستخدم عند إنشاء الملفات أو نقلها أو إعادة تسميتها، أو استخراج الوحدات النمطية إلى أدلة فرعية، أو تحديد المكان الذي تنتمي إليه التعليمات البرمجية المشتركة، أو التحقق من حدود الاستيراد.
