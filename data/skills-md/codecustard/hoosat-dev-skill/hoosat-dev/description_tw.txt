全面的 Hoosat 區塊鏈開發工具包，用於構建交易、集成錢包、創建 dApp 以及與 Hoosat 網絡（使用 Blake3 和 Hoohash 的 Kaspa 分叉）交互。在進行 Hoosat 區塊鏈開發時使用，包括：(1) 構建和廣播 HTN 交易，(2) 使用 hoosat: 前綴生成地址，(3) 使用 hoosat-sdk 或 hoosat-sdk-web 創建 dApp，(4) 集成 Hoosat 錢包，(5) 使用 Hoosat SDK（Node.js hoosat-sdk、瀏覽器 hoosat-sdk-web、Motoko） hoosat-mo for IC），(6) 使用 proxy.hoosat.net 上的 Hoosat REST API，(7) 使用 HRC20 代幣。支持 JavaScript/TypeScript（Node.js 和瀏覽器）、Go 和 Motoko（互聯網計算機）開發。
