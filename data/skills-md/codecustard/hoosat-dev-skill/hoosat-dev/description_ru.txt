Комплексный набор инструментов разработки блокчейна Hoosat для создания транзакций, интеграции кошельков, создания dApps и взаимодействия с сетью Hoosat (форк Kaspa с использованием Blake3 и Hoohash). Используйте при разработке блокчейна Hoosat, включая: (1) Создание и трансляцию HTN-транзакций, (2) Генерацию адресов с префиксом hoosat:, (3) Создание dApps с использованием hoosat-sdk или hoosat-sdk-web, (4) Интеграцию кошельков Hoosat, (5) Использование Hoosat SDK (Node.js hoosat-sdk, Браузер hoosat-sdk-web, Motoko hoosat-mo для IC), (6) Использование Hoosat REST API на proxy.hoosat.net, (7) Работа с токенами HRC20. Поддерживает разработку JavaScript/TypeScript (Node.js и браузер), Go и Motoko (Интернет-компьютер).
