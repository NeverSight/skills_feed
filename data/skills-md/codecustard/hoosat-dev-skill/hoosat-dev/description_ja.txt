トランザクションの構築、ウォレットの統合、dApp の作成、および Hoosat ネットワーク (Blake3 と Hoohash を使用した Kaspa フォーク) との対話のための包括的な Hoosat ブロックチェーン開発ツールキット。以下を含む Hoosat ブロックチェーン開発を行うときに使用します: (1) HTN トランザクションの構築とブロードキャスト、(2) hoosat: プレフィックスを使用したアドレスの生成、(3) hoosat-sdk または hoosat-sdk-web を使用した dApp の作成、(4) Hoosat ウォレットの統合、(5) Hoosat SDK の使用 (Node.js hoosat-sdk、ブラウザ hoosat-sdk-web、Motoko) IC 用 hoosat-mo)、(6) proxy.hoosat.net での Hoosat REST API の使用、(7) HRC20 トークンの使用。 JavaScript/TypeScript (Node.js およびブラウザ)、Go、および Motoko (インターネット コンピューター) の開発をサポートします。
