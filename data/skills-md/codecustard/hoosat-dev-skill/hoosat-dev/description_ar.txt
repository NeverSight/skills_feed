مجموعة أدوات تطوير Hoosat blockchain الشاملة لبناء المعاملات ودمج المحافظ وإنشاء التطبيقات اللامركزية والتفاعل مع شبكة Hoosat (مفترق Kaspa باستخدام Blake3 وHoohash). يُستخدم عند العمل مع تطوير Hoosat blockchain بما في ذلك: (1) إنشاء وبث معاملات HTN، (2) إنشاء عناوين باستخدام hoosat: البادئة، (3) إنشاء تطبيقات dApps باستخدام hoosat-sdk أو hoosat-sdk-web، (4) دمج محافظ Hoosat، (5) استخدام Hoosat SDK (Node.js hoosat-sdk، Browser hoosat-sdk-web، Motoko hoosat-mo لـ IC)، (6) استخدام Hoosat REST API على proxy.hoosat.net، (7) العمل مع رموز HRC20. يدعم تطوير JavaScript/TypeScript (Node.js والمتصفح)، وGo، وMotoko (كمبيوتر الإنترنت).
