Completo kit de herramientas de desarrollo de blockchain de Hoosat para crear transacciones, integrar billeteras, crear dApps e interactuar con la red Hoosat (una bifurcación de Kaspa que utiliza Blake3 y Hoohash). Úselo cuando trabaje con el desarrollo de blockchain de Hoosat, que incluye: (1) Creación y transmisión de transacciones HTN, (2) Generación de direcciones con prefijo hoosat:, (3) Creación de dApps usando hoosat-sdk o hoosat-sdk-web, (4) Integración de billeteras Hoosat, (5) Uso de SDK de Hoosat (Node.js hoosat-sdk, Browser hoosat-sdk-web, Motoko hoosat-mo para IC), (6) Usar la API REST de Hoosat en proxy.hoosat.net, (7) Trabajar con tokens HRC20. Admite el desarrollo de JavaScript/TypeScript (Node.js y navegador), Go y Motoko (computadora de Internet).
