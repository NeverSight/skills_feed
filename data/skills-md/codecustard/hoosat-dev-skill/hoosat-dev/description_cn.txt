全面的 Hoosat 区块链开发工具包，用于构建交易、集成钱包、创建 dApp 以及与 Hoosat 网络（使用 Blake3 和 Hoohash 的 Kaspa 分叉）交互。在进行 Hoosat 区块链开发时使用，包括：(1) 构建和广播 HTN 交易，(2) 使用 hoosat: 前缀生成地址，(3) 使用 hoosat-sdk 或 hoosat-sdk-web 创建 dApp，(4) 集成 Hoosat 钱包，(5) 使用 Hoosat SDK（Node.js hoosat-sdk、浏览器 hoosat-sdk-web、Motoko） hoosat-mo for IC），(6) 使用 proxy.hoosat.net 上的 Hoosat REST API，(7) 使用 HRC20 代币。支持 JavaScript/TypeScript（Node.js 和浏览器）、Go 和 Motoko（互联网计算机）开发。
