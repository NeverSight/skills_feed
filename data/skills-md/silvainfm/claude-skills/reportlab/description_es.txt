Biblioteca Python para generación y creación programática de PDF. Úselo cuando el usuario desee generar archivos PDF desde cero, crear facturas, informes, certificados, etiquetas o documentos personalizados con control preciso sobre el diseño, las fuentes, los gráficos, las tablas y los cuadros. Admite tanto dibujo de bajo nivel (Canvas) como documentos de alto nivel (Platypus).
