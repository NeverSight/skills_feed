Framework Python rapide pour créer des applications Web interactives, des tableaux de bord et des visualisations de données sans HTML/CSS/JavaScript. À utiliser lorsque l'utilisateur souhaite créer des applications de données, des démos ML, des tableaux de bord, des outils d'exploration de données ou des visualisations interactives. Transforme les scripts Python en applications Web en quelques minutes avec des mises à jour automatiques de l'interface utilisateur.
