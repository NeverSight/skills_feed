Libreria DataFrame velocissima scritta in Rust per manipolazione e analisi dei dati ad alte prestazioni. Da utilizzare quando l'utente desidera trasformazioni di dati incredibilmente veloci, lavora con set di dati di grandi dimensioni, pipeline di valutazione lente o ha bisogno di prestazioni migliori rispetto ai panda. Ideale per ETL, gestione dati, aggregazioni, join e lettura/scrittura di file CSV, Parquet e JSON.
