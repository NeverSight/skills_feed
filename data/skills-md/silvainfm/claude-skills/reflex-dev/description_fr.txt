Guide pour créer des applications Web full-stack à l'aide de Reflex, un framework Python qui se compile pour le frontend React et le backend FastAPI. À utiliser lors de la création, de la modification ou du débogage d'applications Reflex : couvre la gestion de l'état, les gestionnaires d'événements, les composants, le routage, le style et les modèles d'intégration de données.
