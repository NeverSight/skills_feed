قاعدة بيانات تحليلية سريعة قيد التشغيل لاستعلامات SQL على DataFrames وCSV وParquet وملفات JSON والمزيد. يُستخدم عندما يريد المستخدم إجراء تحليلات SQL على ملفات البيانات أو Python DataFrames (pandas، Polars)، أو تشغيل مجموعات معقدة، أو صلات، أو وظائف نافذة، أو الاستعلام عن مصادر بيانات خارجية دون تحميلها في الذاكرة. الأفضل لأحمال العمل التحليلية واستعلامات OLAP واستكشاف البيانات.
