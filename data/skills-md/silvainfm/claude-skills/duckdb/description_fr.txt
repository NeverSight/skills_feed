Base de données analytique rapide en cours de processus pour les requêtes SQL sur les fichiers DataFrames, CSV, Parquet, JSON, etc. À utiliser lorsque l'utilisateur souhaite effectuer des analyses SQL sur des fichiers de données ou des DataFrames Python (pandas, Polars), exécuter des agrégations, des jointures ou des fonctions de fenêtre complexes, ou interroger des sources de données externes sans les charger en mémoire. Idéal pour les charges de travail analytiques, les requêtes OLAP et l'exploration de données.
