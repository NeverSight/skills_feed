Database analitico in-process veloce per query SQL su DataFrames, CSV, Parquet, file JSON e altro ancora. Da utilizzare quando l'utente desidera eseguire analisi SQL su file di dati o dataframe Python (panda, polari), eseguire aggregazioni complesse, join o funzioni finestra oppure eseguire query su origini dati esterne senza caricare in memoria. Ideale per carichi di lavoro analitici, query OLAP ed esplorazione dei dati.
