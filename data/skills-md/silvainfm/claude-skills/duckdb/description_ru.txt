Быстрая аналитическая база данных для SQL-запросов к DataFrames, CSV, Parquet, файлам JSON и т. д. Используйте, когда пользователь хочет выполнить SQL-аналитику для файлов данных или фреймов данных Python (Pandas, Polars), запустить сложные агрегаты, объединения или оконные функции или запросить внешние источники данных без загрузки в память. Лучше всего подходит для аналитических рабочих нагрузок, запросов OLAP и исследования данных.
