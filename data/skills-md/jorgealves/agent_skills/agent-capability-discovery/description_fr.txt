Analyse tous les répertoires de compétences du référentiel pour générer une carte globale complète des capacités, des entrées et des sorties des agents. À utiliser lorsque vous avez besoin de comprendre tout le potentiel de votre bibliothèque d'agents ou lorsqu'un agent principal doit décider quelle compétence de sous-agent invoquer pour une tâche complexe.
