Сканирует все каталоги навыков в репозитории, чтобы создать полную глобальную карту возможностей, входных и выходных данных агента. Используйте, когда вам нужно понять весь потенциал вашей библиотеки агентов или когда главному агенту необходимо решить, какой навык субагента использовать для выполнения сложной задачи.
