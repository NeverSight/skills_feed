Анализирует структуру проекта Java и генерирует исчерпывающую документацию, включая технологический стек, архитектуру, модули, конфигурацию, внутренние SDK и рекомендации по разработке. Используйте этот навык, когда вам нужно понять кодовую базу Java/Spring Boot, привлечь новых разработчиков, создать проектную документацию или предоставить контекст для разработки с помощью ИИ. Триггеры включают «анализ структуры проекта», «создание проектной документации», «какова архитектура проекта» или начало работы над незнакомой кодовой базой.
