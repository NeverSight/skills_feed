Java プロジェクトの構造を分析し、技術スタック、アーキテクチャ、モジュール、構成、内部 SDK、開発ガイドラインなどの包括的なドキュメントを生成します。 Java/Spring Boot コードベースを理解する必要がある場合、新しい開発者をオンボーディングする必要がある場合、プロジェクト ドキュメントを生成する必要がある場合、または AI 支援開発のコンテキストを提供する必要がある場合に、このスキルを使用します。トリガーには、「プロジェクト構造の分析」、「プロジェクト ドキュメントの生成」、「プロジェクト アーキテクチャとは何か」、または不慣れなコードベースで作業を開始するときなどが含まれます。
