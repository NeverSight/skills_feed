يقوم بتحليل هيكل مشروع Java وإنشاء وثائق شاملة بما في ذلك المكدس التكنولوجي والهندسة المعمارية والوحدات النمطية والتكوين ومجموعات SDK الداخلية وإرشادات التطوير. استخدم هذه المهارة عندما تحتاج إلى فهم قاعدة بيانات Java/Spring Boot، أو تأهيل مطورين جدد، أو إنشاء وثائق المشروع، أو توفير سياق للتطوير بمساعدة الذكاء الاصطناعي. تتضمن المشغلات "تحليل بنية المشروع"، أو "إنشاء وثائق المشروع"، أو "ما هي بنية المشروع"، أو عند بدء العمل على قاعدة تعليمات برمجية غير مألوفة.
