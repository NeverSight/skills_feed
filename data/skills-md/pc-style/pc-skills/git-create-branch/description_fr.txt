Créez une nouvelle branche Git avec les conventions de dénomination appropriées. À utiliser lorsque l'utilisateur demande à créer une branche, à démarrer une nouvelle branche de fonctionnalités ou à passer à une nouvelle branche. Gère la création de branche à partir du HEAD actuel ou à partir d'une branche de base spécifique, avec une dénomination conventionnelle (feature/, fix/, hotfix/, etc.).
