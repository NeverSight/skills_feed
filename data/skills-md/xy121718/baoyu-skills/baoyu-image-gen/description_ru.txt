Генерация изображений ИИ с помощью API OpenAI, Google, DashScope и Replication. Поддерживает преобразование текста в изображение, эталонные изображения, соотношение сторон. Последовательный по умолчанию; параллельная генерация доступна по запросу. Используйте, когда пользователь просит сгенерировать, создать или нарисовать изображения.
