Создавайте интерфейсные интерфейсы промышленного уровня с помощью современных React, Tailwind CSS и shadcn/ui. Пять режимов: создание проектов, создание компонентов, настройка тем и токенов проектирования, рефакторинг стилей и аудит кодовых баз. Кодирует лучшие практики для серверных компонентов React 19 (зависит от платформы), конфигурации TailwindCSS v4 CSS-first, shadcn/ui с примитивами Radix, современного CSS (контейнерные запросы, :has(), переходы между представлениями, анимацию на основе прокрутки), типографики Monaspace и Vite 6. Заменяет навыки дизайна внешнего интерфейса. Используйте при создании, стилизации, оформлении тем или улучшении любого внешнего проекта. НЕ для серверных API, проектирования баз данных, DevOps, платформ тестирования, библиотек управления состоянием, маршрутизации или полной настройки платформы SSR.
