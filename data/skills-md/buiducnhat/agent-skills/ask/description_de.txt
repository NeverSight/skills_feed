Strukturierte Klärung und Anforderungserfassung durch fokussierten Dialog. Verwenden Sie diese Option, wenn eine Aufgabe nicht eindeutig oder unzureichend spezifiziert ist oder eine Benutzereingabe erfordert, bevor eine Aktion ausgeführt werden kann. Planen oder implementieren Sie nichts – stellen Sie nur Fragen, um die benötigten Informationen zu sammeln. Wird ausgelöst bei: „Fragen Sie mich“, „Stellen Sie Fragen zu“, „Anforderungen klären“, „Anforderungen sammeln“, „Sie müssen fragen“ oder wenn der Benutzer vor Beginn der Arbeit ausdrücklich eine Frage-und-Antwort-Sitzung wünscht.
