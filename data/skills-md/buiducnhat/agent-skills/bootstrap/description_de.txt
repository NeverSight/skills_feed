Initialisieren und stützen Sie ein neues Projekt mit einer strukturierten Anforderungserfassung. Verwenden Sie diese Option, wenn Sie ein Projekt von Grund auf neu starten, nachdem Sie einen Framework-Initialisierer (npm init, create-next-app usw.) ausgeführt haben oder wenn ein vorhandenes Minimalprojekt eine ordnungsgemäße Struktur, Dokumentation und Konventionen benötigt. Wird ausgelöst bei: „Bootstrap-Projekt“, „Neues Projekt starten“, „Gerüstprojekt“, „Projekt initialisieren“, „Neues Projekt einrichten“ oder wenn der Benutzer von Null zu einer gut strukturierten Projektgrundlage übergehen möchte.
