Feishu All-in-One Toolbox: Spreadsheet, Calendar, Task Management, Group Chat Management, Whiteboard Operations, PlantUML Diagrams, File Management, Asset Upload/Download, Document Comments, Knowledge Base, Search, User Directory. Used when the user requests to operate Feishu sheets, view calendars, create tasks, manage group chats, operate whiteboards, generate PlantUML, manage files, upload assets, view comments, view knowledge base, search messages, look up user info, or query departments. Covers all feishu-cli functions except document read/write/import/export and permission management.
