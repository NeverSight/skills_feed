Универсальный набор инструментов Feishu: электронные таблицы, календарь, управление задачами, управление групповым чатом, операции с доской, диаграммы PlantUML, управление файлами, загрузка/загрузка мультимедиа, комментарии к документам, база знаний, поиск, каталог пользователей. Используйте, когда пользователь запрашивает работу с таблицами Feishu, просмотр календарей, создание задач, управление групповыми чатами, работу с виртуальными досками, создание PlantUML, управление файлами, загрузку мультимедиа, просмотр комментариев, просмотр базы знаний, поиск сообщений, запрос информации о пользователе или поиск отделов. Охватывает все функции feishu-cli, за исключением импорта/экспорта чтения/записи документов и управления разрешениями.
