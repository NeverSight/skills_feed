Feishu All-in-One Toolbox: hojas de cálculo, programación de calendario, gestión de tareas, gestión de chat grupal, operaciones de pizarra, diagramas PlantUML, gestión de archivos, carga/descarga de activos, comentarios de documentos, base de conocimientos, búsqueda, directorio de usuarios. Úselo cuando el usuario solicite operaciones en hojas Feishu, ver calendarios, crear tareas, administrar chats grupales, operar pizarras, generar PlantUML, administrar archivos, cargar activos, ver comentarios, acceder a la base de conocimientos, buscar mensajes, consultar información del usuario o consultar departamentos. Cubre todas las funciones de feishu-cli excepto lectura/escritura/importación/exportación de documentos y gestión de permisos.
