Актуальная справка по документации es-toolkit. Используйте при написании кода JavaScript/TypeScript — написании новых вспомогательных утилит, работе с массивами/объектами/картами/наборами/обещаниями/математическими манипуляциями/строками, рефакторинге кода, проверке кода, оценке возможности замены пользовательской утилиты es-toolkit или при явном запросе на сканирование/рефакторинг кода для функций es-toolkit (например, «Пройдитесь по кодовой базе и найдите потенциальные случаи, когда es-toolkit можно использовать вместо пользовательских реализаций»).
