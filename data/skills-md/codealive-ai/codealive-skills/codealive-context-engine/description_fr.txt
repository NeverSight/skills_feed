Recherche de code sémantique et questions-réponses sur la base de code basée sur l'IA dans des référentiels indexés. À utiliser pour comprendre le code au-delà des fichiers locaux, explorer les dépendances, découvrir des modèles inter-projets, planifier des fonctionnalités, déboguer ou intégrer. Des requêtes telles que « Comment fonctionne X ? », « Montrez-moi les modèles Y », « Comment la bibliothèque Z est-elle utilisée ? ». Fournit une recherche (rapide, renvoie les emplacements des fichiers) et une discussion avec la base de code (plus lente, coûte plus cher, mais renvoie des réponses synthétisées).
