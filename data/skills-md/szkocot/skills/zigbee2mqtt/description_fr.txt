Conseils d'experts pour le développement et la configuration de Zigbee2MQTT. À utiliser lorsque vous travaillez avec la base de code zigbee2mqtt, les fichiers de configuration (configuration.yaml, devices.yaml, groups.yaml), les convertisseurs de périphériques, la conception de sujets/charges utiles MQTT ou le dépannage de l'intégration de périphériques Zigbee. Déclenche des tâches impliquant la configuration du pont Zigbee vers MQTT, les flux de travail de couplage d'appareils, le développement de convertisseurs et l'intégration de Home Assistant.
