高品質の git コミットを作成します。意図した変更をレビュー/ステージングし、論理コミットに分割し、明確なコミット メッセージを書き込みます (従来のコミットに続き、絵文字もサポート)。このスキルは、ユーザーがコードのコミット、コミット メッセージの作成、変更のステージング、または作業を複数のコミットに分割することを要求する場合に使用します。ユーザーがキーワード【重大な変更】を言及した場合、コミットメッセージはこのスキルのBREAKING CHANGE仕様に従う必要があり、感嘆符形式を使用します。 git ステージング領域内のファイルのコミットを優先します。ステージング領域が空の場合のみ、作業ツリー全体を考慮してください。ユーザーがキーワード【分類】を言及した場合、このスキルのマルチコミット分割仕様に従い、ファイルタイプ、ビジネスモジュール、変更タイプ、変更スコープの4つの次元から慎重に複数のコミットに分割します。
