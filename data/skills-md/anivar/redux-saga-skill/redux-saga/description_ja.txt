Redux アプリケーションでジェネレーターベースの副作用ミドルウェアを構築、テスト、デバッグするための Redux-Saga のベスト プラクティス、パターン、および API ガイダンス。エフェクト クリエーター、フォーク モデル、チャネル、redux-saga-test-plan によるテスト、同時実行性、キャンセル、最新の Redux Toolkit 統合について説明します。ベースライン: redux-saga 1.4.2。トリガー: saga ファイル、redux-saga インポート、ジェネレーターベースのミドルウェア、「saga」、「takeEvery」、「takelatest」、「fork モデル」、または「チャネル」の言及。
