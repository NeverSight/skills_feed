Meilleures pratiques, modèles et conseils d'API de Redux-Saga pour la création, le test et le débogage d'un middleware d'effets secondaires basé sur un générateur dans les applications Redux. Couvre les créateurs d'effets, le modèle de fork, les canaux, les tests avec redux-saga-test-plan, la simultanéité, l'annulation et l'intégration moderne de Redux Toolkit. Référence : redux-saga 1.4.2. Déclencheurs sur : les fichiers saga, les importations redux-saga, le middleware basé sur un générateur, les mentions de "saga", "takeEvery", "takeLatest", "fork model" ou "channels".
