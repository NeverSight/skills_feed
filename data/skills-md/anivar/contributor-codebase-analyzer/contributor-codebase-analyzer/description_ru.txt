Глубокий анализ кода с периодическим сохранением. В режиме участника считываются все изменения коммитов для определения ежегодных проверок, показателей точности и готовности к продвижению. Режим Codebase отображает структуру репозитория, отношения между репозиториями и управление предприятием. Работает с GitHub (gh) и GitLab (glab). Сохраняет контрольные точки постепенно для возобновления между сеансами.
