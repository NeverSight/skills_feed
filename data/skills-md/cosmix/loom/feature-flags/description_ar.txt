تتميز بأنماط العلامات لعمليات الإطلاق التي يتم التحكم فيها، واختبار A/B، ومفاتيح التوقف، وتكوين وقت التشغيل. يُستخدم عند تنفيذ تبديل الميزات، أو علامات الميزات، أو عمليات الطرح التدريجي، أو إصدارات الكناري، أو نسبة عمليات الطرح، أو عمليات التشغيل المظلمة، أو استهداف المستخدم، أو اختبارات A/B، أو التجارب، أو قواطع الدائرة، أو مفاتيح إيقاف الطوارئ، أو تبديل النماذج، أو أعلام البنية التحتية.
