Визуализация наблюдаемости с помощью Grafana и стека LGTM. Дизайн информационной панели, конфигурация панели, оповещения, переменные/шаблоны и источники данных.

КОГДА ИСПОЛЬЗУЕТСЯ: Создание информационных панелей Grafana, настройка панелей и визуализаций, написание запросов LogQL/TraceQL, настройка источников данных Grafana, настройка переменных и шаблонов информационной панели, создание оповещений Grafana.
НЕ ИСПОЛЬЗУЙТЕ: для написания запросов PromQL (используйте /prometheus), для стратегии правил оповещений (используйте /prometheus), для общей архитектуры наблюдения (используйте старшего инженера-программиста с акцентом на инфраструктуру).

ТРИГГЕРЫ: графана, панель мониторинга, панель, визуализация, logql, Traceql, Loki, Tempo, Mimir, источник данных, аннотация, переменная, шаблон, строка, статистика, график, таблица, тепловая карта, датчик, гистограмма, круговая диаграмма, временные ряды, панель журналов, панель трассировок, стек LGTM.
