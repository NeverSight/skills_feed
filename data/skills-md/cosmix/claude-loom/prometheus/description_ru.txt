Мониторинг и оповещение Prometheus для обеспечения возможности наблюдения в облаке.

КОГДА ИСПОЛЬЗУЕТСЯ: написание запросов PromQL, настройка целей очистки Prometheus, создание правил оповещений, настройка правил записи, оснащение приложений метриками Prometheus, настройка обнаружения сервисов.
НЕ ИСПОЛЬЗУЙТЕ: для создания информационных панелей (используйте /grafana), для анализа журналов (используйте /logging-observability), для общей архитектуры наблюдаемости (используйте старшего инженера-программиста с акцентом на инфраструктуру).

ТРИГГЕРЫ: метрики, prometheus, promql, счетчик, датчик, гистограмма, сводка, оповещение, alertmanager, правило оповещения, правило записи, очистка, цель, метка, обнаружение сервисов, перемаркировка, экспортер, инструментарий, замедление, бюджет ошибок.
