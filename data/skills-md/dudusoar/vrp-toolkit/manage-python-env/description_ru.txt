Краткий справочник по операциям uv (быстрого менеджера пакетов Python) для сохранения токенов. Используйте при создании виртуальных сред, установке пакетов, управлении зависимостями или когда пользователь спрашивает о командах uv. Предоставляет краткие шаблоны для настройки проектов Python и управления пакетами.
