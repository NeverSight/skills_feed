Référence rapide pour les opérations uv (gestionnaire de packages Python rapide) pour enregistrer les jetons. À utiliser lors de la création d'environnements virtuels, de l'installation de packages, de la gestion des dépendances ou lorsque l'utilisateur pose des questions sur les commandes uv. Fournit des modèles concis pour la configuration du projet Python et la gestion des packages.
