Integrieren Sie mithilfe von OpenStreetMap-Daten reale Straßennetze in VRP-Probleme. Verwenden Sie diese Funktion, wenn Sie echte Kartendaten laden, Instanzen von tatsächlichen Standorten erstellen, netzwerkbasierte Entfernungen berechnen oder Tutorials mit realen Szenarien erstellen. Führt durch die Installation, das Laden von Bereichen, das Extrahieren von Knoten, das Berechnen von Entfernungsmatrizen und das Erstellen von PDPTW-Instanzen aus Kartendaten.
