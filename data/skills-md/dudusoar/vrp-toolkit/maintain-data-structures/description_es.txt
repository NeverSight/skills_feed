Documentación de referencia completa para todas las estructuras de datos en el proyecto del kit de herramientas VRP. Úselo cuando necesite comprender definiciones, atributos, formatos o representaciones de estructuras de datos para evitar leer código repetidamente. Cubre la capa de problema (instancia, solución, nodo), capa de algoritmo (solucionador, operadores, ALNS), capa de datos (OSMnx, matrices de distancia) y formatos de tiempo de ejecución (rutas como listas, ventanas de tiempo como tuplas).
