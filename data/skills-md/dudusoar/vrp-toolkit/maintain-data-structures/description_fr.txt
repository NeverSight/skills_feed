Documentation de référence complète pour toutes les structures de données du projet de boîte à outils VRP. À utiliser lorsque vous avez besoin de comprendre les définitions, les attributs, les formats ou les représentations de la structure de données pour éviter de lire le code à plusieurs reprises. Couvre la couche de problèmes (instance, solution, nœud), la couche d'algorithme (solveur, opérateurs, ALNS), la couche de données (OSMnx, matrices de distance) et les formats d'exécution (itinéraires sous forme de listes, fenêtres horaires sous forme de tuples).
