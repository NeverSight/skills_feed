Erstellen Sie einen präzisen Sitzungskontext, indem Sie wichtige Informationen aus Projektprotokollen (CLAUDE.md, TASK_BOARD.md, MIGRATION_LOG.md, DEBUG_LOG.md, GIT_LOG.md, SKILLS_LOG.md) extrahieren, um einen tokeneffizienten Projektstatus bereitzustellen. Verwenden Sie es, wenn Sie mit der Arbeit beginnen, nach einer Pause zurückkehren oder einen schnellen Projektüberblick benötigen, ohne vollständige Dateien lesen zu müssen.
