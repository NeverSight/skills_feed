Систематический рабочий процесс и справочник API для интеграции игровой площадки Streamlit с модулями vrp-toolkit. Используйте при разработке функций игровой площадки, подключении пользовательского интерфейса к API-интерфейсам vrp-toolkit или при обнаружении несоответствия подписи API. Предоставляет таблицу сопоставления интерфейсов, интеграцию тестов контракта и руководство по устранению неполадок, позволяющее избежать исследования исходного кода с интенсивным использованием токенов.
