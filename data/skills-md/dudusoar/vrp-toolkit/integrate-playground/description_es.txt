Flujo de trabajo sistemático y referencia API para integrar el parque infantil Streamlit con los módulos vrp-toolkit. Úselo cuando desarrolle funciones de juegos, conecte la interfaz de usuario a las API de vrp-toolkit o encuentre discrepancias en las firmas de API. Proporciona una tabla de mapeo de interfaces, integración de pruebas de contratos y una guía de solución de problemas para evitar la exploración del código fuente con uso intensivo de tokens.
