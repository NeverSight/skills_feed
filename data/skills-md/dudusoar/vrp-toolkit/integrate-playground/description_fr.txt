Flux de travail systématique et référence API pour l'intégration du terrain de jeu Streamlit avec les modules vrp-toolkit. À utiliser lors du développement de fonctionnalités de terrain de jeu, de la connexion de l'interface utilisateur aux API de vrp-toolkit ou de la rencontre d'incompatibilités de signature d'API. Fournit un tableau de mappage d'interface, une intégration de tests contractuels et un guide de dépannage pour éviter l'exploration du code source à forte intensité de jetons.
