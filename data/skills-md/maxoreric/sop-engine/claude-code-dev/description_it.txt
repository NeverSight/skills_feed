Esperto di sviluppo di plugin di codice Claude. Attivato quando l'utente desidera creare un plugin, un'abilità, un subagente, un comando barra o un hook. Utilizzato anche per rispondere alle domande sullo sviluppo del codice Claude. Parole chiave: crea plugin, scrivi abilità, scrivi agente, crea hook, scrivi comando, sviluppo plugin.
