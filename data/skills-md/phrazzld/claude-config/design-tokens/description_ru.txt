Примените шаблоны токенов дизайна с помощью директивы @theme Tailwind CSS 4: переменные CSS, семантическое именование, цветовые системы, типографские масштабы, интервалы, темный режим. Используйте при проектировании систем пользовательского интерфейса, проверке согласованности дизайна или установлении принципов бренда. Интегрируется с навыками внешнего дизайна для эстетичного исполнения.
