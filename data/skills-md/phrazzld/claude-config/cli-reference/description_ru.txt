Команды CLI для всех служб инфраструктуры. Вызов когда:
- Настройка любого внешнего сервиса (Vercel, Convex, Stripe, Sentry, PostHog, GitHub)
- Прежде чем предлагать какое-либо действие на панели управления
- Установка переменных среды на разных платформах.
Ключевые слова: env var, настройка, панель мониторинга, настройки, производство, развертывание, CLI.
