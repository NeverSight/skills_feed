Umsetzbares Designsystem, das das offizielle Frontend-Design-Plugin ergänzt.
Das Plugin bietet Philosophie; Dieser Skill stellt ausführbare Muster bereit.

Aufrufen, wenn:
- Erstellen von Webkomponenten, Seiten oder Anwendungen
- Entwerfen einer Benutzeroberfläche, die auffallen muss (nicht generisch)
- Implementierung von visuellem Feinschliff und Mikrointeraktionen
- Auswahl von Typografie, Farbpaletten oder räumlichen Layouts

KRITISCH: Vermeiden Sie eine „KI-Slop“-Ästhetik. Treffen Sie kreative, unerwartete Entscheidungen.
OBLIGATORISCH: Konsultieren Sie Gemini vor jeglicher Frontend-Arbeit.
