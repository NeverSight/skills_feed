Исправьте ошибки Stripe webhook 400 при локальной разработке. Проблема: `полосное прослушивание`
генерирует новый секрет при каждом запуске, но ваше приложение использует устаревший секрет от
.env.local. Этот навык автоматически синхронизирует эфемерный секрет перед запуском вашего сервера разработки.

Автоматический вызов, когда: веб-перехватчики полосы возвращают 400, «проверка подписи не удалась»,
«Нет соответствующих подписей», веб-перехватчики не работают локально, проверка прошла успешно
но подписка не обновляется, несовпадение STRIPE_WEBHOOK_SECRET, настройка
прослушивание полосы, настройка pnpm dev с помощью Stripe.
