Appliquer les normes de documentation : commentez pourquoi pas quoi, un minimum de commentaires (préférez un code clair), maintenez README avec un démarrage rapide, mettez à jour les documents avec les modifications majeures. À utiliser lors de la rédaction de commentaires, de la création de documents, de la révision de la documentation ou de la discussion de ce qu'il faut documenter.
