أفضل ممارسات Ruby and Rails للخدمات والاختبار والهندسة المعمارية. استخدم عندما:
- كتابة أو مراجعة كود Ruby/Rails
- تصميم كائنات الخدمة أو تنظيم منطق الأعمال
- إعداد اختبارات RSpec باستخدام FactoryBot
- بناء واجهات برمجة التطبيقات مع تسلسل JSON
- تحسين أداء ريلز (N+1، التخزين المؤقت)
الكلمات الرئيسية: روبي، ريلز، RSpec، FactoryBot، كائن الخدمة، المعلمات القوية،
استعلام N+1، Sidekiq، RuboCop، Thor CLI
