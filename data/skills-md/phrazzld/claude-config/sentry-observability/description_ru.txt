Отслеживание ошибок Sentry и наблюдение за производством. Вызов когда:
- Новый проект Next.js без отслеживания ошибок → Предложить настройку
- Развертывание в Vercel → «Добавить наблюдаемость перед развертыванием».
- Обсуждение обработки ошибок, try/catch → «Отслеживать их в производстве»
- Производственная ошибка, пользователи сообщают об ошибках → Сортировка с помощью скриптов.
- Настройка мониторинга, настройка оповещений, проверка обработки личных данных.
Скрипты:Detect_sentry.sh, init_sentry.sh, list_issues.sh, triage_score.sh
Ключевые слова: Sentry, производственные ошибки, отслеживание ошибок, сортировка, оповещения,
наблюдаемость, воспроизведение сеанса, редактирование личных данных, исходные карты, мониторинг
