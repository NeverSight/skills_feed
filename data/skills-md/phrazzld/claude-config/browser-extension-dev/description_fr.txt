Développement d'extensions de navigateur avec sécurité et prise en charge de plusieurs navigateurs. À utiliser lorsque :
- Création d'extensions Chrome, Firefox ou Safari
- Demander des autorisations dans le manifeste
- Implémentation de scripts de contenu ou de travailleurs en arrière-plan
- Gestion de la compatibilité entre navigateurs
- Planification des mises à jour des extensions
Mots clés : extension de navigateur, Manifest V3, script de contenu, script d'arrière-plan,
autorisations, extension Chrome, module complémentaire Firefox, API WebExtensions
