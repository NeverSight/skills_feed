تطوير امتدادات المتصفح مع دعم الأمان والمتصفحات المتعددة. استخدم عندما:
- إنشاء ملحقات Chrome أو Firefox أو Safari
- طلب الأذونات في البيان
- تنفيذ نصوص المحتوى أو العاملين في الخلفية
- التعامل مع التوافق عبر المتصفحات
- تحديثات تمديد التخطيط
الكلمات الرئيسية: ملحق المتصفح، Manifest V3، البرنامج النصي للمحتوى، البرنامج النصي للخلفية،
الأذونات، ملحق Chrome، ملحق Firefox، WebExtensions API
