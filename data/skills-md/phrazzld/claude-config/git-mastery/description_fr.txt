Application du workflow Git et bonnes pratiques. À utiliser lorsque :
- Rédiger des messages de commit ou réviser les commits
- Créer des succursales ou choisir des stratégies de fusion
- Mise en place de pipelines CI/CD avec des opérations git
- Résoudre les conflits de fusion ou de rebasage
- Configuration des paramètres du référentiel ou des git hooks
- Optimisation des performances des grands référentiels
Mots-clés : git, commit, branche, fusion, rebase, PR, pull request, basé sur le tronc,
commits conventionnels, commits atomiques, git hooks, CODEOWNERS
