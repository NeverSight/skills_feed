تطبيق سير عمل Git وأفضل الممارسات. استخدم عندما:
- كتابة رسائل الالتزام أو مراجعة الالتزامات
- إنشاء فروع أو اختيار استراتيجيات الدمج
- إعداد خطوط أنابيب CI/CD مع عمليات git
- حل تعارضات الدمج أو إعادة التأسيس
- تكوين إعدادات المستودع أو خطافات git
- تحسين أداء المستودعات الكبيرة
الكلمات الرئيسية: git، الالتزام، الفرع، الدمج، rebase، العلاقات العامة، طلب السحب، المستند إلى صندوق الأمتعة،
الإرتباطات التقليدية، الإرتكابات الذرية، خطافات git، CODEOWNERS
