Applicazione del flusso di lavoro Git e best practice. Utilizzare quando:
- Scrivere messaggi di commit o rivedere i commit
- Creazione di rami o scelta di strategie di fusione
- Configurazione di pipeline CI/CD con operazioni git
- Risolvere conflitti di unione o ribasamento
- Configurazione delle impostazioni del repository o degli hook git
- Ottimizzazione delle prestazioni dei repository di grandi dimensioni

commit convenzionali, commit atomici, hook git, CODEOWNERS
