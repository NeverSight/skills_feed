- Escribir mensajes de confirmación o revisar confirmaciones.
- Crear sucursales o elegir estrategias de fusión.
- Configuración de canalizaciones de CI/CD con operaciones de git
- Resolver conflictos de fusión o rebase
- Configuración de ajustes del repositorio o ganchos de git
- Optimización del rendimiento de repositorios grandes
