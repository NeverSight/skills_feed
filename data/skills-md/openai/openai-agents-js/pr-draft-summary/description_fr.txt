Créez un titre de PR et un brouillon de description une fois les modifications substantielles du code terminées. Déclenchez-vous lors de la conclusion d'une modification modérée ou plus importante (code d'exécution, tests, configuration de construction, documents avec impact sur le comportement) et vous avez besoin du bloc de résumé prêt pour les relations publiques avec un résumé des modifications et un brouillon de texte PR.
