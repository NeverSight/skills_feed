قم بتحليل تنفيذ الفرع الرئيسي وتكوينه للعثور على الوثائق المفقودة أو غير الصحيحة أو القديمة في المستندات/. استخدمه عندما يُطلب منك تدقيق تغطية المستند، أو مزامنة المستندات مع التعليمات البرمجية، أو اقتراح تحديثات/تغييرات في البنية. قم فقط بتحديث المستندات الإنجليزية (docs/src/content/docs/**) ولا تلمس المستندات المترجمة مطلقًا ضمن docs/src/content/docs/ja أو ko أو zh. قدم تقريرًا واطلب الموافقة قبل تحرير المستندات.
