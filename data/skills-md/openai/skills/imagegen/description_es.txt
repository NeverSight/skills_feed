Úselo cuando el usuario solicite generar o editar imágenes a través de OpenAI Image API (por ejemplo: generar imagen, editar/pintar/máscara, eliminación o reemplazo de fondo, fondo transparente, fotografías de productos, arte conceptual, portadas o variantes por lotes); ejecute la CLI incluida (`scripts/image_gen.py`) y requiera `OPENAI_API_KEY` para llamadas en vivo.
