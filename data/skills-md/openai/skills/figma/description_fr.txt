Utilisez le serveur Figma MCP pour récupérer le contexte de conception, les captures d'écran, les variables et les actifs de Figma, et pour traduire les nœuds Figma en code de production. Déclenchez lorsqu'une tâche implique des URL Figma, des ID de nœud, une implémentation de la conception au code ou la configuration et le dépannage de Figma MCP.
