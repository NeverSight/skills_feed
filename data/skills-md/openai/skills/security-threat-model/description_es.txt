Modelado de amenazas basado en repositorio que enumera límites de confianza, activos, capacidades de atacantes, rutas de abuso y mitigaciones, y escribe un modelo de amenazas Markdown conciso. Se activa solo cuando el usuario solicita explícitamente modelar una ruta o base de código de amenazas, enumerar rutas de amenazas/abuso o realizar un modelado de amenazas de AppSec. No lo active para resúmenes de arquitectura general, revisión de código o trabajos de diseño que no sean de seguridad.
