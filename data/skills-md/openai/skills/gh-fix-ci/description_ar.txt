يُستخدم عندما يطلب المستخدم تصحيح أو إصلاح عمليات فحص GitHub PR الفاشلة التي يتم تشغيلها في إجراءات GitHub؛ استخدم `gh` لفحص الشيكات والسجلات، وتلخيص سياق الفشل، وصياغة خطة الإصلاح، وتنفيذها فقط بعد الحصول على موافقة صريحة. تعامل مع مقدمي الخدمة الخارجيين (على سبيل المثال Buildkite) باعتبارهم خارج النطاق وقم بالإبلاغ فقط عن عنوان URL الخاص بالتفاصيل.
