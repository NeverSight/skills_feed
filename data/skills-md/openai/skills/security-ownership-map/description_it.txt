Analizza i repository git per creare una topologia di proprietà di sicurezza (persone-file), calcolare il fattore bus e la proprietà del codice sensibile ed esportare CSV/JSON per database a grafo e visualizzazione. Si attiva solo quando l'utente desidera esplicitamente un'analisi della proprietà orientata alla sicurezza o dei fattori bus basata sulla cronologia git (ad esempio: codice sensibile orfano, manutentori della sicurezza, controlli di realtà di CODEOWNERS per i rischi, hotspot sensibili o cluster di proprietà). Non attivare per elenchi generali di manutentori o domande sulla proprietà non di sicurezza.
