Analice repositorios de Git para crear una topología de propiedad de seguridad (personas a archivos), cómputo del factor de bus y propiedad de código confidencial, y exporte CSV/JSON para bases de datos de gráficos y visualización. Se activa solo cuando el usuario desea explícitamente una propiedad orientada a la seguridad o un análisis de factor de bus basado en el historial de git (por ejemplo: código sensible huérfano, mantenedores de seguridad, verificaciones de realidad de CODEOWNERS para detectar riesgos, puntos de acceso sensibles o clústeres de propiedad). No active listas generales de mantenedores ni preguntas que no estén relacionadas con la propiedad de seguridad.
