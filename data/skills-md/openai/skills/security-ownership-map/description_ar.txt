قم بتحليل مستودعات git لإنشاء هيكل ملكية الأمان (من الأشخاص إلى الملف)، وحساب عامل الناقل وملكية التعليمات البرمجية الحساسة، وتصدير ملف CSV/JSON لقواعد بيانات الرسم البياني والتصور. يتم التشغيل فقط عندما يريد المستخدم صراحةً ملكية موجهة نحو الأمان أو تحليل عامل ناقل يرتكز على سجل git (على سبيل المثال: التعليمات البرمجية الحساسة المعزولة، أو مشرفو الأمان، أو عمليات التحقق الواقعية من CODEOWNERS للمخاطر، أو نقاط الاتصال الحساسة، أو مجموعات الملكية). لا تقم بتشغيل قوائم المشرف العامة أو أسئلة الملكية غير الأمنية.
