استخدمه عند العمل مع OpenAI API (Responses API) أو ميزات النظام الأساسي OpenAI (الأدوات، البث، Realtime API، المصادقة، النماذج، حدود الأسعار، MCP) وتحتاج إلى وثائق موثوقة وحديثة (المخططات، الأمثلة، الحدود، حالات الحافة). تفضيل أدوات خادم OpenAI Developer Documentation MCP عند توفرها؛ وإلا قم بتوجيه المستخدم لتمكين `openaiDeveloperDocs`.
