Guía para usar DynamoDB-Toolbox v2: la biblioteca TypeScript liviana para DynamoDB con creación de consultas con seguridad de tipos y validación de esquemas. Úselo al implementar operaciones de DynamoDB con el patrón .build(), definir tablas/entidades/esquemas o trabajar con comandos como GetItem, PutItem, UpdateItem, DeleteItem, Query, Scan, BatchGet, BatchWrite o Transactions. Activadores en importaciones de DynamoDB-Toolbox, definiciones de entidad/tabla o patrones de comando .build().
