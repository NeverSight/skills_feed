دليل لاستخدام DynamoDB-Toolbox v2 - مكتبة TypeScript خفيفة الوزن لـ DynamoDB مع بناء استعلام آمن للنوع والتحقق من صحة المخطط. يُستخدم عند تنفيذ عمليات DynamoDB باستخدام نمط .build()، أو تحديد الجداول/الكيانات/المخططات، أو العمل مع أوامر مثل GetItem، أو PutItem، أو UpdateItem، أوDeleteItem، أوQuery، أو Scan، أو BatchGet، أو BatchWrite، أو Transactions. المشغلات عند استيراد DynamoDB-Toolbox، أو تعريفات الكيان/الجدول، أو أنماط أوامر .build().
