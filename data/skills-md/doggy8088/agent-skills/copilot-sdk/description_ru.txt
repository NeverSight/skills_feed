Этот навык дает рекомендации по созданию агентов и приложений с помощью GitHub Copilot SDK. ВАЖНО. При использовании SDK с TypeScript/Node.js проект ДОЛЖЕН использовать ESM (модули ECMAScript). CommonJS (require/module.exports) НЕ поддерживается. Его следует использовать, когда пользователь хочет создавать, изменять или работать с программным обеспечением, использующим GitHub Copilot SDK в TypeScript (только ESM), Python, Go или .NET. Навыки охватывают шаблоны использования SDK, конфигурацию проекта на основе ESM, конфигурацию CLI, пользовательские инструменты, серверы MCP и специальные агенты.
