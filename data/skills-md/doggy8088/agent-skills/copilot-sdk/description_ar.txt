توفر هذه المهارة إرشادات لإنشاء الوكلاء والتطبيقات باستخدام GitHub Copilot SDK. هام - عند استخدام SDK مع TypeScript/Node.js، يجب أن يستخدم المشروع ESM (وحدات ECMAScript). CommonJS (require/module.exports) غير مدعوم. يجب استخدامه عندما يريد المستخدم إنشاء أو تعديل أو العمل على برنامج يستخدم GitHub Copilot SDK في TypeScript (ESM فقط)، أو Python، أو Go، أو .NET. تغطي المهارة أنماط استخدام SDK، وتكوين المشروع القائم على ESM، وتكوين CLI، والأدوات المخصصة، وخوادم MCP، والوكلاء المخصصين.
