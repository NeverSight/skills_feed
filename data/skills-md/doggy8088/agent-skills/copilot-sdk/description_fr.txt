Cette compétence fournit des conseils pour créer des agents et des applications avec le SDK GitHub Copilot. IMPORTANT - Lors de l'utilisation du SDK avec TypeScript/Node.js, le projet DOIT utiliser ESM (modules ECMAScript). CommonJS (require/module.exports) n'est PAS pris en charge. Il doit être utilisé lorsque l'utilisateur souhaite créer, modifier ou travailler sur un logiciel qui utilise le SDK GitHub Copilot dans TypeScript (ESM uniquement), Python, Go ou .NET. La compétence couvre les modèles d'utilisation du SDK, la configuration de projet basée sur ESM, la configuration CLI, les outils personnalisés, les serveurs MCP et les agents personnalisés.
