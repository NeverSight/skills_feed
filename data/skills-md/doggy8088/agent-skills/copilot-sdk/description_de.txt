Dieser Skill bietet Anleitungen zum Erstellen von Agenten und Anwendungen mit dem GitHub Copilot SDK. WICHTIG – Bei Verwendung des SDK mit TypeScript/Node.js MUSS das Projekt ESM (ECMAScript-Module) verwenden. CommonJS (require/module.exports) wird NICHT unterstützt. Es sollte verwendet werden, wenn der Benutzer Software erstellen, ändern oder daran arbeiten möchte, die das GitHub Copilot SDK in TypeScript (nur ESM), Python, Go oder .NET verwendet. Die Fertigkeit umfasst SDK-Nutzungsmuster, ESM-basierte Projektkonfiguration, CLI-Konfiguration, benutzerdefinierte Tools, MCP-Server und benutzerdefinierte Agenten.
