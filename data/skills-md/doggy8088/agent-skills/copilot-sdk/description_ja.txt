このスキルは、GitHub Copilot SDK を使用してエージェントとアプリケーションを作成するためのガイダンスを提供します。重要 - TypeScript/Node.js で SDK を使用する場合、プロジェクトは ESM (ECMAScript モジュール) を使用する必要があります。 CommonJS (require/module.exports) はサポートされていません。これは、ユーザーが TypeScript (ESM のみ)、Python、Go、または .NET で GitHub Copilot SDK を使用するソフトウェアを作成、変更、または作業する場合に使用する必要があります。このスキルは、SDK の使用パターン、ESM ベースのプロジェクト構成、CLI 構成、カスタム ツール、MCP サーバー、およびカスタム エージェントをカバーします。
