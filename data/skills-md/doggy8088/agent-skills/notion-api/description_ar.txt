توفر هذه المهارة تعليمات شاملة للتفاعل مع Notion API عبر استدعاءات REST. يجب استخدام هذه المهارة عندما يطلب المستخدم التفاعل مع Notion، بما في ذلك قراءة أو إنشاء أو تحديث أو حذف الصفحات أو قواعد البيانات أو الكتل أو التعليقات أو أي محتوى Notion آخر. تغطي المهارة المصادقة وجميع نقاط النهاية المتاحة وترقيم الصفحات ومعالجة الأخطاء وأفضل الممارسات.
