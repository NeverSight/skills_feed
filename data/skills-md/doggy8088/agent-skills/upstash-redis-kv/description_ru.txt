Чтение и запись в хранилище значений ключей, совместимое с Upstash Redis, через REST API. Используйте, когда есть необходимость сохранить или получить данные «ключ-значение», использовать функции Redis (кэширование, счетчики, списки, наборы, хэши, отсортированные наборы и т. д.) для текущего взаимодействия или когда пользователь явно просит использовать Upstash или Redis.
