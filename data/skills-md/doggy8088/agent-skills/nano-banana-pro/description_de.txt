Erstellen und bearbeiten Sie Bilder mit der Nano Banana Pro (Gemini 3 Pro Image) API von Google. Verwenden Sie diese Option, wenn der Benutzer Bilder generieren, erstellen, bearbeiten, modifizieren, ändern, ändern oder aktualisieren möchte. Wird auch verwendet, wenn der Benutzer auf eine vorhandene Bilddatei verweist und sie auf irgendeine Weise ändern möchte (z. B. „Dieses Bild ändern“, „Hintergrund ändern“, „X durch Y ersetzen“). Unterstützt sowohl die Text-zu-Bild-Generierung als auch die Bild-zu-Bild-Bearbeitung mit konfigurierbarer Auflösung (1K Standard, 2K oder 4K für hohe Auflösung). Lesen Sie NICHT zuerst die Bilddatei – verwenden Sie diesen Skill direkt mit dem Parameter --input-image.
