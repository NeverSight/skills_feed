Recherche de code sémantique à l'aide de mgrep pour une exploration efficace de la base de code. Cette compétence doit être utilisée lors de la recherche ou de l'exploration de bases de code contenant plus de 30 fichiers non ignorés et/ou structures de répertoires imbriquées. Il fournit une recherche sémantique en langage naturel qui complète grep/ripgrep traditionnel pour rechercher des fonctionnalités, comprendre l'intention et explorer du code inconnu.
