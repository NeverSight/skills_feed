Семантический поиск кода с использованием mgrep для эффективного исследования базы кода. Этот навык следует использовать при поиске или изучении баз кода с более чем 30 неигнорируемыми файлами и/или вложенными структурами каталогов. Он обеспечивает семантический поиск на естественном языке, который дополняет традиционный grep/ripgrep для поиска функций, понимания намерений и изучения незнакомого кода.
