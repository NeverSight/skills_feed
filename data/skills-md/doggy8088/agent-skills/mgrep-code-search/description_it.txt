Ricerca del codice semantico utilizzando mgrep per un'esplorazione efficiente della base di codice. Questa abilità dovrebbe essere utilizzata durante la ricerca o l'esplorazione di codebase con più di 30 file non gitignored e/o strutture di directory annidate. Fornisce la ricerca semantica in linguaggio naturale che integra il tradizionale grep/ripgrep per trovare funzionalità, comprendere l'intento ed esplorare codice sconosciuto.
