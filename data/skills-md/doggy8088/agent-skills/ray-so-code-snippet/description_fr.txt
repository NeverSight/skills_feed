Générez de superbes images d'extraits de code à l'aide de ray.so. Cette compétence doit être utilisée lorsque l'utilisateur demande à créer une image de code, une capture d'écran de code, une image d'extrait de code ou souhaite rendre son code joli pour le partage. Enregistre les images localement dans le répertoire de travail actuel ou dans un chemin spécifié par l'utilisateur.
