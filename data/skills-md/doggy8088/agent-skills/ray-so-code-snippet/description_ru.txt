Создавайте красивые изображения фрагментов кода с помощью ray.so. Этот навык следует использовать, когда пользователь просит создать изображение кода, снимок экрана кода, изображение фрагмента кода или хочет, чтобы его код выглядел красиво, чтобы можно было поделиться им. Сохраняет изображения локально в текущий рабочий каталог или по указанному пользователем пути.
