توفر هذه المهارة تعليمات للتفاعل مع Todoist REST API v2 باستخدام curl وjq. وهو يغطي المصادقة وعمليات CRUD للمهام/المشاريع/الأقسام/التسميات/التعليقات، ومعالجة ترقيم الصفحات، ويتطلب التأكيد قبل الإجراءات المدمرة. استخدم هذه المهارة عندما يريد المستخدم قراءة بيانات تودويست أو إنشائها أو تحديثها أو حذفها عبر واجهة برمجة التطبيقات.
