Genere y edite imágenes utilizando el modelo GPT Image 1.5 de OpenAI. Se utiliza cuando el usuario solicita generar, crear, editar, modificar, cambiar, alterar o actualizar imágenes. También se utiliza cuando el usuario hace referencia a un archivo de imagen existente y solicita modificarlo de alguna manera (por ejemplo, "modificar esta imagen", "cambiar el fondo", "reemplazar X con Y"). Admite la generación de texto a imagen y la edición de imágenes con máscara opcional. NO lea el archivo de imagen primero; use esta habilidad directamente con el parámetro --input-image.
