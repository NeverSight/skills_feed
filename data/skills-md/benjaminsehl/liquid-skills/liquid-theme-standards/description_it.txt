Standard di codifica CSS, JavaScript e HTML per i temi Shopify Liquid. Copre la denominazione BEM all'interno dei tag dei fogli di stile, i token di progettazione, le proprietà personalizzate CSS, i componenti Web per i temi, i CSS difensivi e il miglioramento progressivo. Da utilizzare quando si scrive CSS/JS/HTML in file .liquid o file di risorse del tema.
