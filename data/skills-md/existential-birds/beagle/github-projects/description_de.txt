GitHub-Projektverwaltung über gh CLI zum Erstellen von Projekten, Verwalten von Elementen, Feldern und Workflows. Verwenden Sie es, wenn Sie mit GitHub Projects (v2) arbeiten, Probleme/PRs zu Projekten hinzufügen, benutzerdefinierte Felder erstellen, Projektelemente verfolgen oder Projektworkflows automatisieren. Auslöser für GH-Projekt, Projektboard, Kanban, GitHub-Projekt, Projektelemente.
