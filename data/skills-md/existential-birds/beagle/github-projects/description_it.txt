Gestione dei progetti GitHub tramite gh CLI per la creazione di progetti, la gestione di elementi, campi e flussi di lavoro. Da utilizzare quando si lavora con GitHub Projects (v2), si aggiungono problemi/PR ai progetti, si creano campi personalizzati, si tengono traccia degli elementi del progetto o si automatizzano i flussi di lavoro del progetto. Trigger su progetto gh, scheda del progetto, kanban, progetto GitHub, elementi del progetto.
