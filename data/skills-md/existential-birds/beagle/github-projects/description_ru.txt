Управление проектами GitHub через интерфейс командной строки gh для создания проектов, управления элементами, полями и рабочими процессами. Используйте при работе с проектами GitHub (v2), добавлении задач/запросов в проекты, создании настраиваемых полей, отслеживании элементов проекта или автоматизации рабочих процессов проекта. Триггеры по проекту gh, доске проекта, канбану, проекту GitHub, элементам проекта.
