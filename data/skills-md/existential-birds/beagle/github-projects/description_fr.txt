Gestion de projets GitHub via gh CLI pour créer des projets, gérer des éléments, des champs et des flux de travail. À utiliser lorsque vous travaillez avec des projets GitHub (v2), en ajoutant des problèmes/PR aux projets, en créant des champs personnalisés, en suivant les éléments du projet ou en automatisant les flux de travail du projet. Déclencheurs sur le projet gh, le tableau de projet, le kanban, le projet GitHub, les éléments du projet.
