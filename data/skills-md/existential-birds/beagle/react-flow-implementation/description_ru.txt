Правильно реализует пользовательский интерфейс React Flow на основе узлов, используя @xyflow/react. Используйте при создании блок-схем, диаграмм, визуальных редакторов или приложений на основе узлов с помощью React. Охватывает узлы, края, ручки, пользовательские компоненты, управление состоянием и управление видовым окном.
