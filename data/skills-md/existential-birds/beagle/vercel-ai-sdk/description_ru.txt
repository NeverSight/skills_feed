Vercel AI SDK для создания интерфейсов чата с потоковой передачей. Используйте при реализации перехватчика useChat, обработки вызовов инструментов, потоковой передачи ответов или создания пользовательского интерфейса чата. Триггеры по useChat, @ai-sdk/react, UIMessage, ChatStatus,streamText, toUIMessageStreamResponse, addToolOutput, onToolCall, sendMessage.
