Meilleures pratiques de React Router v7 pour le routage basé sur les données. À utiliser lors de l'implémentation d'itinéraires, de chargeurs, d'actions, de composants de formulaire, d'extracteurs, de gardes de navigation, d'itinéraires protégés ou de paramètres de recherche d'URL. Déclencheurs sur createBrowserRouter, RouterProvider, useLoaderData, useActionData, useFetcher, NavLink, Outlet.
