أفضل ممارسات React Router v7 للتوجيه المبني على البيانات. يُستخدم عند تنفيذ المسارات أو أدوات التحميل أو الإجراءات أو مكونات النموذج أو أدوات الجلب أو حراس التنقل أو المسارات المحمية أو معلمات البحث في عنوان URL. يتم تشغيلها على createBrowserRouter وRouterProvider وuseLoaderData وuseActionData وuseFetcher وNavLink وOutlet.
