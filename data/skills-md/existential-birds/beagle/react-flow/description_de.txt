React Flow (@xyflow/react) zur Workflow-Visualisierung mit benutzerdefinierten Knoten und Kanten. Verwenden Sie es beim Erstellen von Diagrammvisualisierungen, beim Erstellen benutzerdefinierter Workflow-Knoten, beim Implementieren von Kantenbeschriftungen oder beim Steuern des Ansichtsfensters. Trigger auf ReactFlow, @xyflow/react, Handle, NodeProps, EdgeProps, useReactFlow, fitView.
