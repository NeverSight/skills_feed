React Flow (@xyflow/react) для визуализации рабочего процесса с настраиваемыми узлами и краями. Используйте при построении визуализаций графов, создании пользовательских узлов рабочего процесса, реализации меток ребер или управлении областью просмотра. Триггеры на ReactFlow, @xyflow/react, Handle, NodeProps, EdgeProps, useReactFlow, fitView.
