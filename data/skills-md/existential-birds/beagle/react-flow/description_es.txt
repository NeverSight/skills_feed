React Flow (@xyflow/react) para visualización del flujo de trabajo con nodos y bordes personalizados. Úselo al crear visualizaciones de gráficos, crear nodos de flujo de trabajo personalizados, implementar etiquetas de borde o controlar la ventana gráfica. Activadores en ReactFlow, @xyflow/react, Handle, NodeProps, EdgeProps, useReactFlow, fitView.
