Mejores prácticas para usar Cursor: reglas, comandos, habilidades, subagentes, archivos ignorados, seguridad del agente, flujos de trabajo y recursos de la comunidad. Úselo al configurar el Cursor, inicializar o crear la carpeta .cursor, escribir .cursor/rules o AGENTS.md, crear comandos o habilidades, configurar .cursorignore, trabajar con el Agente, descubrir reglas o MCP (por ejemplo, cursor.directory), hacer que las bases de código sean compatibles con el cursor o preguntar sobre los flujos de trabajo del Cursor, TDD, comandos git o bases de código grandes.
