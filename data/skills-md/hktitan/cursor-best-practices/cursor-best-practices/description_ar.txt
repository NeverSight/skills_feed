أفضل الممارسات لاستخدام المؤشر — القواعد والأوامر والمهارات والوكلاء الفرعيين وتجاهل الملفات وأمان الوكيل ومسارات العمل وموارد المجتمع. يُستخدم عند إعداد المؤشر، أو تهيئة مجلد .cursor أو إنشائه، أو كتابة .cursor/rules أو AGENTS.md، أو إنشاء أوامر أو مهارات، أو تكوين .cursorignore، أو العمل مع Agent، أو اكتشاف القواعد أو MCPs (مثل cursor.directory)، أو جعل قواعد التعليمات البرمجية متوافقة مع المؤشر، أو السؤال عن سير عمل المؤشر، أو TDD، أو أوامر git، أو قواعد التعليمات البرمجية الكبيرة.
