Управление жизненным циклом ресурсов Scala с помощью Cats Effect Resource и IO. Используйте при определении безопасного получения/выпуска, составления ресурсов (включая параллельное получение) или разработки ресурсобезопасных API и поведения отмены для файлов, потоков, пулов, клиентов и фоновых волокон.
