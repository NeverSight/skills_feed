Управление состоянием Compose с сильным предпочтением поднятия состояния для Kotlin Compose (Android, Multiplatform, Compose for Web). Используйте для рефакторинга или нового пользовательского интерфейса, который требует четкого владения состоянием, однонаправленного потока данных, сохраненных решений о состоянии или рекомендаций по запоминанию/сохранению/rememberSaveable/rememberSerializable, а также для разработки компонуемых объектов без сохранения состояния с обратными вызовами событий.
