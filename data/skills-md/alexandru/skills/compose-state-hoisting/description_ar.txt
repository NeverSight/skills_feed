إنشاء إدارة الحالة مع تفضيل قوي لرفع الحالة لـ Kotlin Compose (Android، Multiplatform، Compose for Web). يُستخدم لإعادة البناء أو واجهة المستخدم الجديدة التي تحتاج إلى ملكية حالة واضحة، أو تدفق بيانات أحادي الاتجاه، أو قرارات الحالة المحفوظة، أو إرشادات حول التذكر/الاحتفاظ/التذكرالحفظ/التذكرSerializable، ولتصميم العناصر القابلة للتركيب عديمة الحالة مع عمليات الاسترجاعات للحدث.
