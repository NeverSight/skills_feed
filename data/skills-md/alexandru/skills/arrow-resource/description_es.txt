Kotlin + Arrow Gestión del ciclo de vida de recursos con `Resource`, `ResourceScope` y `resourceScope`. Úselo para diseñar adquisiciones/liberaciones seguras de archivos, transmisiones, grupos/conexiones de bases de datos, clientes HTTP o partes de varias partes; componer recursos (incluida la adquisición paralela); o integrar Resource con errores tipográficos y cancelación.
