Kotlin + Arrow إدارة دورة حياة الموارد باستخدام `Resource` و`ResourceScope` و`resourceScope`. يُستخدم لتصميم الحصول/الإصدار الآمن للملفات أو التدفقات أو تجمعات/اتصالات قاعدة البيانات أو عملاء HTTP أو الأجزاء متعددة الأجزاء؛ تكوين الموارد (بما في ذلك الاستحواذ الموازي) ؛ أو دمج المورد مع الأخطاء المكتوبة والإلغاء.
