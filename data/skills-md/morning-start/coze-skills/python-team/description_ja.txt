自己学習、PM、アーキテクト、上級開発者のコ​​ラボレーションを通じて、自然言語要件から完全な Python プロジェクト成果物を自動的に生成します。機能拡張、プロジェクトのリファクタリング、スキルの呼び出しをサポートします。これにより、Web 検索、ナレッジ統合、バージョン管理、Python 3.11 以降の機能、UV パッケージ管理、loguru ログ記録、適応型プロジェクト スケール (フォルダーまたは単一ファイル) が可能になります。データベースの設計と実装 (SQLite、PostgreSQL、MongoDB、ベクター データベース、グラフ データベース)、データ層の抽象化 (リポジトリ パターン)、およびデータベースの切り替えをサポートします。ソフトウェア要件の明確化、ラピッドプロトタイピング、プロジェクトの初期化、機能拡張、コードのリファクタリングに適用されます。
