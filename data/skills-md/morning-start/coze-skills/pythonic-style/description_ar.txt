تحسين أسلوب كود Python والإرشاد إلى الأساليب Pythonic؛ يعتمد على محتوى «One Python Craftsman» و«Friendly Python»، ويشمل تسمية المتغيرات والتحكم في التدفق وأنواع البيانات والحاويات وتصميم الدوال ومعالجة الاستثناءات والـ decorators وعمليات الملفات ومبادئ SOLID؛ يوفر أنماط تصميم وقوائم مراجعة وأكثر من 140 قالباً عملياً.
