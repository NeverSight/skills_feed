Руководство по оптимизации стиля кода Python и Pythonic Idioms; основан на полном содержании «One Python Craftsman» и философии «Friendly Python», охватывающем именование переменных, поток управления, типы данных, типы контейнеров, проектирование функций, обработку исключений, декораторы, файловые операции, принципы SOLID; предоставление удобных для пользователя и удобных в обслуживании шаблонов проектирования, контрольных списков проверки и более 140 практических шаблонов.
