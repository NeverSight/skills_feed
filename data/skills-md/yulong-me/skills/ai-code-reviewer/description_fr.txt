Révision automatisée du code basée sur l'IA qui s'exécute sur des hooks Git avec une conception de divulgation progressive. À utiliser lors de la configuration de la révision automatisée du code pour un projet, de l'installation de hooks git pour la révision du code, de la création ou de la modification de règles de révision ou de la configuration du comportement de révision. Se déclenche sur des requêtes telles que « configurer la révision du code AI », « installer des crochets de révision », « créer des règles de révision » ou « configurer le réviseur de code ».
