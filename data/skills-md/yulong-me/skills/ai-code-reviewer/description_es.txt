Revisión de código automatizada impulsada por IA que se ejecuta en ganchos de git con un diseño de divulgación progresiva. Úselo al configurar la revisión de código automatizada para un proyecto, instalar ganchos de git para la revisión de código, crear o modificar reglas de revisión o configurar el comportamiento de revisión. Se activan solicitudes como "configurar la revisión del código de IA", "instalar ganchos de revisión", "crear reglas de revisión" o "configurar el revisor de código".
