مراجعة أكواد برمجية آلية مدعومة بالذكاء الاصطناعي والتي يتم تشغيلها على خطافات git مع تصميم الكشف التدريجي. يُستخدم عند إعداد المراجعة التلقائية للتعليمات البرمجية لمشروع ما، أو تثبيت git Hooks لمراجعة التعليمات البرمجية، أو إنشاء قواعد المراجعة أو تعديلها، أو تكوين سلوك المراجعة. تشغيل طلبات مثل "إعداد مراجعة كود الذكاء الاصطناعي" أو "تثبيت روابط المراجعة" أو "إنشاء قواعد المراجعة" أو "تكوين مراجع الكود".
