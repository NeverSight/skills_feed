Автоматизированная проверка кода на базе искусственного интеллекта, выполняемая с помощью git-хуков и с прогрессивным дизайном раскрытия информации. Используйте при настройке автоматической проверки кода для проекта, установке перехватчиков git для проверки кода, создании или изменении правил проверки или настройке поведения проверки. Запускается по таким запросам, как «настроить проверку кода AI», «установить средства проверки», «создать правила проверки» или «настроить средство проверки кода».
