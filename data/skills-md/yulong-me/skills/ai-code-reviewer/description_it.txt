Revisione automatizzata del codice basata sull'intelligenza artificiale eseguita su hook git con design di divulgazione progressiva. Da utilizzare durante la configurazione della revisione automatizzata del codice per un progetto, l'installazione di hook git per la revisione del codice, la creazione o la modifica di regole di revisione o la configurazione del comportamento di revisione. Si attiva su richieste come "imposta la revisione del codice AI", "installa hook di revisione", "crea regole di revisione" o "configura il revisore del codice".
