Conseils d'experts pour le développement du framework Web ElysiaJS. À utiliser lors de la création d'API REST, de services GraphQL ou d'applications WebSocket avec Elysia sur Bun. Couvre le routage, les hooks de cycle de vie, la validation TypeBox, les clients de type sécurisé Eden, l'authentification avec JWT/Bearer, tous les plugins officiels (OpenAPI, CORS, JWT, static, cron, GraphQL, tRPC), les modèles de test et le déploiement en production. Suppose que la compétence Bun-Expert est active pour l'expertise d'exécution Bun.
