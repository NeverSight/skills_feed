Экспертное руководство по разработке веб-фреймворка ElysiaJS. Используйте при создании API REST, сервисов GraphQL или приложений WebSocket с помощью Elysia on Bun. Охватывает маршрутизацию, перехватчики жизненного цикла, проверку TypeBox, типобезопасные клиенты Eden, аутентификацию с помощью JWT/Bearer, все официальные плагины (OpenAPI, CORS, JWT, static, cron, GraphQL, tRPC), шаблоны тестирования и производственное развертывание. Предполагается, что навык эксперта по булочкам активен для опыта выполнения Bun.
