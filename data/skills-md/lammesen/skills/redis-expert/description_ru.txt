Эксперт по базам данных Redis для кэширования, структур данных, Pub/Sub, потоков, сценариев Lua,
и оптимизация производительности с использованием собственного клиента Bun.redis.
Используйте при работе с Redis, стратегиями кэширования, хранилищами ключей, данными в реальном времени,
векторный поиск или базы данных документов. Работает вместе с навыком эксперта по булочкам.
