Conseils d'experts sur la base de données PostgreSQL à l'aide du client Bun.sql. Fournit des modèles complets
pour les requêtes, la conception de schémas, les opérations JSON/JSONB, la recherche en texte intégral, l'indexation, PL/pgSQL,
pgvector et optimisation des performances. À utiliser lorsque vous travaillez avec des bases de données PostgreSQL,
écrire des requêtes SQL, optimiser les performances, concevoir des schémas ou implémenter une base de données
fonctionnalités. Complète les compétences Bun-Expert pour l'intégration Bun.sql.
