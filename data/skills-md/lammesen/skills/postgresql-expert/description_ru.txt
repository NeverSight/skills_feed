Экспертное руководство по работе с базами данных PostgreSQL с использованием клиента Bun.sql. Предоставляет комплексные шаблоны
для запросов, проектирование схемы, операции JSON/JSONB, полнотекстовый поиск, индексирование, PL/pgSQL,
pgvector и оптимизация производительности. Используйте при работе с базами данных PostgreSQL,
написание SQL-запросов, оптимизация производительности, разработка схем или внедрение базы данных
особенности. Дополняет навыки эксперта по Bun для интеграции Bun.sql.
