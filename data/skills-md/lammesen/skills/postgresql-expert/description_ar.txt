توجيهات الخبراء لقاعدة بيانات PostgreSQL باستخدام عميل Bun.sql. يوفر أنماط شاملة
للاستعلامات، وتصميم المخطط، وعمليات JSON/JSONB، والبحث عن النص الكامل، والفهرسة، وPL/pgSQL،
pgvector وتحسين الأداء. يُستخدم عند العمل مع قواعد بيانات PostgreSQL،
كتابة استعلامات SQL أو تحسين الأداء أو تصميم المخططات أو تنفيذ قاعدة البيانات
الميزات. يكمل مهارة خبير الكعكة لتكامل Bun.sql.
