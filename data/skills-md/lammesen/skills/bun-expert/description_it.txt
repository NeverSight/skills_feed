Competenza completa sul runtime di Bun che copre tutte le principali funzionalità.
Da utilizzare quando si lavora con progetti Bun, si esegue la migrazione da Node.js o si sfruttano API specifiche di Bun.
Si attiva per: Bun.serve, Bun.file, bun:test, bun:sqlite, bun install, bun build,
bunfig.toml, TypeScript in Bun, operazioni del gestore pacchetti, configurazione del bundler,
e domande sulla compatibilità di Node.js.
