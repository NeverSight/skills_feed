تحويل الصور وتغيير حجمها وضغطها وتحسينها عبر التنسيقات (HEIC وPNG وJPEG وWebP وAVIF وGIF وTIFF وBMP). يُستخدم عند العمل مع ملفات الصور لتحويل التنسيق، أو تغيير الحجم/تصغير الحجم، أو الضغط/التحسين، أو معالجة الدفعات، أو وضع العلامات المائية، أو تجريد البيانات التعريفية، أو أي مهمة معالجة للصور. تشغيل الطلبات التي تتضمن ملفات صور، أو معالجة الصور، أو تحسين صور الويب.
