Подробное руководство по глубокому обучению с помощью Keras 3 (мульти-бэкенд: JAX, TensorFlow, PyTorch). Используйте при построении нейронных сетей, CNN для компьютерного зрения, RNN/трансформеров для НЛП, прогнозирования временных рядов или генеративных моделей (VAE, GAN). Охватывает построение моделей (последовательные/функциональные/подклассовые API), пользовательские циклы обучения, увеличение данных, трансферное обучение и передовые методы производства.
