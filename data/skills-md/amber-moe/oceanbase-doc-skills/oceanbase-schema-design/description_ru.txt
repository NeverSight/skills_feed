Лучшие практики проектирования схем для базы данных OceanBase (режимы MySQL и Oracle). Охватывает проектирование таблиц, стратегии секционирования, группы таблиц, дизайн индексов и принципы оптимизации схемы. Активируется для проектирования схемы, проектирования таблиц, проектирования разделов, проектирования индекса, проектирования базы данных, DDL.
