Лучшие практики SQL-оптимизации для базы данных OceanBase (режимы MySQL и Oracle). Охватывает оптимизацию запросов, использование индексов, анализ плана выполнения, медленную настройку запросов и методы оптимизации производительности. Активируется для оптимизации SQL, производительности запросов, проектирования индексов, плана выполнения, медленных запросов, производительности базы данных.
