Hilft Benutzern, Agent-Fähigkeiten auf dem Marktplatz zu entdecken, zu suchen und zu installieren. 
Verwenden Sie diese Option, wenn der Benutzer einen Skill finden, neue Funktionen entdecken, bestimmte Tools (insbesondere Sandboxes) finden oder einen Skill installieren möchte.
Dieser Skill sollte verwendet werden, wenn der Benutzer nach Funktionen sucht, die möglicherweise als installierbarer Skill vorhanden sind.
Unterstützt auch die progressive Offenlegung, um bei Bedarf Sandbox-bezogene Fähigkeiten zu empfehlen.
用于帮助用户在市场中搜索、发现和安装 Agent 插件（Skills）.
当用户想要查找功能、安装特定工具或寻找沙箱（Sandbox）时使用.
Auslöser: „查找插件“, „搜索技能“, „搜索 X 技能“, „安装技能“, „安装„Nach Skill suchen“, „Skill installieren“, „Ich brauche einen Skill für X“, „Sandbox“, „Aio Sandbox“
