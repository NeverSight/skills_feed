Навыки мониторинга общественного мнения; Конечный результат: отчет об общественном мнении. Используйте этот навык, когда пользователь спрашивает: «Каково общественное мнение по поводу X/инцидента?» «Как настроение?» «Запустить анализ настроений» или запросы на сканирование по ключевому слову/платформе и создание отчета. Соглашение: любое намерение, связанное с общественным мнением, запускает весь рабочий процесс (сканирование → анализ настроений → отчет). Этот навык позволяет ползать; анализ настроений осуществляется основным Агентом автономно с помощью подсказок; отчет генерируется функциейgenerate_report.
