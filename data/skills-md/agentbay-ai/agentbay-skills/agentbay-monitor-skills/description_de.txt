Fähigkeit zur Beobachtung der öffentlichen Meinung; Endergebnis: ein Meinungsbericht. Verwenden Sie diese Fähigkeit, wenn der Benutzer fragt: „Wie ist die öffentliche Meinung zu X/Vorfall?“ „Wie ist die Stimmung?“ „Führen Sie eine Stimmungsanalyse durch“ oder fordern Sie das Crawlen nach Schlüsselwort/Plattform an und erstellen Sie einen Bericht. Zustimmung: Jede Absicht im Zusammenhang mit der öffentlichen Meinung löst den gesamten Workflow aus (Crawling → Stimmungsanalyse → Bericht). Das Kriechen wird von dieser Fertigkeit gehandhabt; Die Stimmungsanalyse wird vom Hauptagenten autonom über Eingabeaufforderungen durchgeführt. Der Bericht wird von generic_report generiert.
