Для любого запроса на «запуск/выполнение/оценку кода» (например, запуск этого кода, выполнение сценария, построение графика с помощью Python, запуск кода/построение/выполнение сценария). Создайте песочницу code_latest с помощью AgentBay SDK, вызовите run_code для выполнения и возврата результатов. Поддерживает Python, JavaScript, R, Java.
