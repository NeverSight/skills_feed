مراجعة كود العقد الذكي ومنهجية التدقيق الأمني ​​لـ Solidity. يُستخدم عند مراجعة أو تدقيق أو تقييم أمان كود Solidity. يوفر عملية مراجعة منظمة وتصنيف الخطورة ومجالات التفتيش الرئيسية وتكامل OWASP SCWE. تشغيل المهام التي تتضمن مراجعة التعليمات البرمجية، أو التدقيق الأمني، أو تقييم الثغرات الأمنية، أو مراجعة العقود الذكية، أو التحقق من أفضل الممارسات.
