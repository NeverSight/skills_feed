Проверка кода смарт-контракта и методология аудита безопасности для Solidity. Используйте при просмотре, аудите или оценке безопасности кода Solidity. Обеспечивает структурированный процесс проверки, классификацию серьезности, ключевые области проверки и интеграцию OWASP SCWE. Запускается при выполнении задач, связанных с проверкой кода, аудитом безопасности, оценкой уязвимостей, проверкой смарт-контрактов или проверкой лучших практик.
