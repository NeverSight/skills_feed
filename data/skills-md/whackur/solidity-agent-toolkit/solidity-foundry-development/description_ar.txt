سير عمل تطوير المسبك لعقود Solidity الذكية. يُستخدم عند البناء أو الاختبار أو النشر باستخدام المسبك (الصياغة أو الصب أو السندان). يغطي إعداد المشروع، وتكوين Foundry.toml، وأنماط الاختبار، واختبار الزغب، والاختبار الثابت، واختبار الشوكة، وأكواد الغش، والبرامج النصية للنشر، وتصحيح الأخطاء. يتم تشغيل المهام التي تتضمن بناء الصياغة، أو اختبار الصياغة، أو صياغة النص، أو الصب، أو السندان، أو تطوير Solidity القائم على المسبك.
