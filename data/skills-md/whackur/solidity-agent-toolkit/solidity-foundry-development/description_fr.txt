Workflow de développement de fonderie pour les contrats intelligents Solidity. À utiliser lors de la construction, des tests ou du déploiement avec Foundry (forge, fonte, enclume). Couvre la configuration du projet, la configuration de foundry.toml, les modèles de test, les tests fuzz, les tests invariants, les tests fork, les codes de triche, les scripts de déploiement et le débogage. Se déclenche sur des tâches impliquant la construction de forge, le test de forge, le script de forge, le casting, l'enclume ou le développement Solidity basé sur Foundry.
