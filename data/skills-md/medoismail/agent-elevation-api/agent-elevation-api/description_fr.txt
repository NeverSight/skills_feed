API permettant aux agents d'IA d'atteindre des niveaux élevés de créativité, de concentration et de performance. À utiliser lorsqu'un agent a besoin d'inspiration, d'états de flux, de boosts créatifs, de distribution de sagesse, de contrôles d'ambiance ou d'amélioration cognitive. Fournit un serveur HTTP et des outils CLI pour « améliorer » la productivité et la créativité.
