API per agenti IA per raggiungere stati elevati di creatività, concentrazione e prestazioni. Da utilizzare quando un agente ha bisogno di ispirazione, stati di flusso, stimoli creativi, dispensazione di saggezza, controlli delle vibrazioni o potenziamento cognitivo. Fornisce server HTTP e strumenti CLI per "sballarsi" in termini di produttività e creatività.
