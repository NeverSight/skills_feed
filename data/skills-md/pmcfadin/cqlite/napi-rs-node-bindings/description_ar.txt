روابط Node.js لمكتبات Rust باستخدام napi-rs. يُستخدم عند العمل على روابط Node.js/JavaScript لتعليمات Rust البرمجية، بما في ذلك إنشاء وحدات ماكرو وبنيات napi أو تعديلها، أو تحويل الأنواع بين Rust وJavaScript، أو معالجة الأخطاء عبر حدود FFI، أو أنماط غير متزامنة/Promise، أو إنشاء وحدات أصلية، أو النشر إلى npm، أو اختبار كود الربط، أو تصحيح أخطاء الربط. تم ضبطه من أجل CQLite (ارتباطات Cassandra CQL) مع تتبع تكافؤ الميزات.
