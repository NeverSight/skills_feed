Aide à l'intégration .NET dans vvvv gamma : packages NuGet, références de bibliothèque, configuration de projet .csproj, interopérabilité de type vectoriel et modèles asynchrones. À utiliser lors de l'ajout de packages NuGet, de la configuration des paramètres de build, du référencement de bibliothèques .NET externes ou de l'utilisation de conversions de type System.Numerics/Stride.
