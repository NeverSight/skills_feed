Explique les concepts de base de vvvv gamma : types de données, modèle d'exécution basé sur des trames, broches, blocs-notes, liens, navigateur de nœuds, compilation en direct (projet source vs workflows de référence binaire), structure de document .vl, types de fichiers (.vl/.sdsl/.cs/.csproj), aperçu de l'écosystème et détection du runtime AppHost. À utiliser lorsque l'utilisateur pose des questions sur les bases de vvvv, comment fonctionne vvvv, le modèle de rechargement en direct, quand patcher par rapport au code ou a besoin d'un aperçu de l'environnement de programmation visuel.
