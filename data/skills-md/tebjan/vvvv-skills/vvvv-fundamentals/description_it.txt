Spiega i concetti fondamentali di vvvv gamma: tipi di dati, modello di esecuzione basato su frame, pin, pad, collegamenti, browser dei nodi, compilazione live (progetto sorgente e flussi di lavoro di riferimento binario), struttura del documento .vl, tipi di file (.vl/.sdsl/.cs/.csproj), panoramica dell'ecosistema e rilevamento del runtime di AppHost. Da utilizzare quando l'utente chiede informazioni sulle nozioni di base di vvvv, come funziona vvvv, il modello di ricarica in tempo reale, quando applicare patch o codice o ha bisogno di una panoramica dell'ambiente di programmazione visiva.
