Помогает создавать расширения редактора гаммы vvvv — именование файлов .HDE.vl, регистрацию командного узла с помощью сочетаний клавиш, типы окон SkiaWindow/SkiaWindowTopMost, стыковку с WindowFactory и доступ через API к наведенным/выбранным узлам через узлы сеанса VL.Lang. Используйте при создании плагинов редактора, пользовательских окон инструментов или автоматизации рабочих процессов редактора.
