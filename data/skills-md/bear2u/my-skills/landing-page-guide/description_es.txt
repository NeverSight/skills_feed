Guía completa para crear páginas de destino efectivas usando Next.js o React. Esta habilidad debe usarse cuando los usuarios solicitan crear páginas de destino, páginas de marketing o páginas de productos que requieren los 11 elementos esenciales para páginas de destino de alta conversión. Diseñado específicamente para Next.js 14+ App Router con componentes de interfaz de usuario ShadCN.
