Esta habilidad se debe utilizar al redactar problemas de GitHub para el repositorio de Positron. Proporciona flujos de trabajo para buscar duplicados, seleccionar etiquetas apropiadas, recopilar un contexto completo mediante preguntas y escribir temas concisos y sin complicaciones que describan con precisión lo que se necesita o lo que está mal. La habilidad prepara problemas para el envío manual por parte del usuario. Utilice esta habilidad cuando el usuario solicite redactar o preparar un número para Positron.
