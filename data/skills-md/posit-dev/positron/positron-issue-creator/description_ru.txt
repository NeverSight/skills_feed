Этот навык следует использовать при написании выпусков GitHub для репозитория Positron. Он обеспечивает рабочие процессы для поиска дубликатов, выбора подходящих ярлыков, сбора полного контекста с помощью вопросов и написания кратких, без лишних слов вопросов, которые точно описывают, что нужно, а что нет. Навык подготавливает задачи для отправки пользователем вручную. Используйте этот навык, когда пользователь просит написать или подготовить выпуск для Positron.
