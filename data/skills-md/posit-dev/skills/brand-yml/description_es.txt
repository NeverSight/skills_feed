Cree y utilice archivos brand.yml para lograr una marca coherente en todas las aplicaciones Shiny y documentos Quarto. Úselo cuando trabaje con estilos de marca, colores, fuentes, logotipos o identidad corporativa en proyectos Shiny o Quarto. Cubre: (1) Creación de nuevos archivos _brand.yml a partir de pautas de marca, (2) Aplicación de brand.yml a aplicaciones Shiny para R con bslib, (3) Aplicación de brand.yml a aplicaciones Shiny para Python con ui.Theme, (4) Uso de brand.yml en documentos, presentaciones, paneles y archivos PDF de Quarto, (5) Modificación de archivos brand.yml existentes, (6) Solución de problemas de integración de marca. Incluye especificaciones completas y guías de integración específicas del marco.
