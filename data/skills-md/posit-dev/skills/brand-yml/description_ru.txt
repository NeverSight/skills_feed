Создавайте и используйте файлы Brand.yml для единообразного фирменного оформления приложений Shiny и документов Quarto. Используйте при работе со стилем бренда, цветами, шрифтами, логотипами или фирменным стилем в проектах Shiny или Quarto. Охватывает: (1) Создание новых файлов _brand.yml на основе рекомендаций по бренду, (2) Применение Brand.yml к приложениям Shiny для R с помощью bslib, (3) Применение Brand.yml к приложениям Shiny для Python с помощью ui.Theme, (4) Использование Brand.yml в документах Quarto, презентациях, информационных панелях и PDF-файлах, (5) Изменение существующих файлов Brand.yml, (6) Устранение проблем с интеграцией бренда. Включает полные спецификации и руководства по интеграции для конкретной платформы.
