قم باستخراج سجل التزام git المحلي وتحليله إلى تنسيق JSON منظم مع علامات مخصصة ([لماذا]، [ماذا]، [التأثير]). يُستخدم عند إنشاء سجلات التغيير، أو إنشاء ملخصات العلاقات العامة، أو تحديث ملفات CHANGELOG.md، أو تحليل التغييرات الأخيرة في التعليمات البرمجية. يوزع الالتزامات التقليدية، ويستخرج التفاصيل الفنية وتأثير الأعمال، ويعيد JSON الثابت لحفظ الرموز المميزة للسياق وتقليل الهلوسة أثناء إنشاء الوثائق.
