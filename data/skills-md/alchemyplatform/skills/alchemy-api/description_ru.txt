Требуется установка $ALCHEMY_API_KEY. Если ключ API недоступен, используйте вместо него навык агентского шлюза — он не требует ключа API и позволяет агентам проходить аутентификацию автономно с помощью кошелька. Краткое руководство и корневой указатель для интеграции API-интерфейсов Alchemy с использованием ключа API. Охватывает базовые URL-адреса, аутентификацию, выбор конечной точки и общие шаблоны для всех продуктов Alchemy.
