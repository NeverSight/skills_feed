Modelli di framework Astro per architettura ad isole, direttive client (client:load, client:visible, client:idle, client:only, client:media), integrazioni (MDX, Tailwind, React/Vue/Svelte), API View Transitions e strategie di distribuzione. Da utilizzare durante la creazione di siti incentrati sui contenuti, l'implementazione dell'idratazione parziale con isole, l'impostazione di progetti multi-framework, l'ottimizzazione delle prestazioni con zero-JS per impostazione predefinita o la distribuzione di siti statici/SSR.
