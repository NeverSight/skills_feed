Шаблоны инфраструктуры Astro для Islands Architecture, клиентские директивы (client:load, client:visible, client:idle, client:only, client:media), интеграция (MDX, Tailwind, React/Vue/Svelte), View Transitions API и стратегии развертывания. Используйте при создании сайтов, ориентированных на контент, реализации частичной гидратации с помощью островов, настройке проектов с несколькими платформами, оптимизации производительности с нулевым JS по умолчанию или развертывании статических сайтов или сайтов SSR.
