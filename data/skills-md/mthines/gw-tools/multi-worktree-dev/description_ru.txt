Расширенные шаблоны для одновременной разработки нескольких рабочих деревьев Git. Используйте этот навык при управлении зависимостями между рабочими деревьями, синхронизации файлов, работе с базами данных для нескольких ветвей, выполнении параллельных тестов или оркестрации сервисов между рабочими деревьями. Запускается при выполнении задач, включающих многоотраслевую разработку, совместное использование зависимостей, изоляцию базы данных или оркестровку сервисов с помощью gw.
