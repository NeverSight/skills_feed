Настраивайте и оптимизируйте инструменты gw для различных типов проектов и потребностей команды. Используйте этот навык при настройке gw для новых проектов, настройке автоматического копирования файлов, устранении проблем с конфигурацией или настройке gw для Next.js, API-интерфейсов Node.js, монорепозиториев или React SPA. Запускается при выполнении задач, связанных с .gw/config.json, шаблонами автоматического копирования, файлами среды или командами инициализации gw.
