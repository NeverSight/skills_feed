قم بتكوين أدوات gw وتحسينها لأنواع المشاريع المختلفة واحتياجات الفريق. استخدم هذه المهارة عند إعداد gw للمشاريع الجديدة، أو تكوين ملفات النسخ التلقائي، أو استكشاف مشكلات التكوين وإصلاحها، أو تخصيص gw لـ Next.js، أو Node.js APIs، أو monorepos، أو React SPAs. يتم تشغيلها عند تنفيذ المهام التي تتضمن ‎.gw/config.json، أو أنماط النسخ التلقائي، أو ملفات البيئة، أو أوامر gw init.
