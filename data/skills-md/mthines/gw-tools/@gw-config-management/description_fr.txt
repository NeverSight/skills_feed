Configurez et optimisez les outils gw pour différents types de projets et besoins de l'équipe. Utilisez cette compétence lors de la configuration de gw pour de nouveaux projets, de la configuration de fichiers de copie automatique, du dépannage de problèmes de configuration ou de la personnalisation de gw pour Next.js, les API Node.js, les monorepos ou les React SPA. Se déclenche sur des tâches impliquant .gw/config.json, des modèles de copie automatique, des fichiers d'environnement ou des commandes gw init.
