Освойте рабочие деревья Git и рабочие процессы gw-tools для параллельной разработки. Используйте этот навык при создании рабочих деревьев, одновременном управлении несколькими ветвями, навигации между рабочими деревьями, устранении проблем с рабочими деревьями или настройке рабочих процессов ветвей функций. Запускается при выполнении задач, связанных с командами рабочего дерева git, изоляцией ветвей, параллельной разработкой или использованием gw CLI.
