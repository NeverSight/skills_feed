Maîtrisez les arbres de travail Git et les workflows gw-tools pour le développement parallèle. Utilisez cette compétence lors de la création d'arbres de travail, de la gestion simultanée de plusieurs branches, de la navigation entre les arbres de travail, du dépannage des problèmes d'arbre de travail ou de la configuration de flux de travail de branche de fonctionnalités. Se déclenche sur des tâches impliquant des commandes git worktree, l'isolation de branche, le développement parallèle ou l'utilisation de gw CLI.
