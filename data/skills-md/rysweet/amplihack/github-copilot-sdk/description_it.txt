Conoscenza completa di GitHub Copilot SDK per incorporare i flussi di lavoro agenti di Copilot in applicazioni Python, TypeScript, Go e .NET. Si attiva automaticamente per l'integrazione dell'SDK di Copilot, l'utilizzo di CopilotClient, la gestione delle sessioni, le risposte in streaming, gli strumenti personalizzati e le connessioni al server MCP.
