Всестороннее знание GitHub Copilot SDK для внедрения агентных рабочих процессов Copilot в приложения Python, TypeScript, Go и .NET. Автоматически активируется для интеграции Copilot SDK, использования CopilotClient, управления сеансами, потоковой передачи ответов, пользовательских инструментов и подключений к серверу MCP.
