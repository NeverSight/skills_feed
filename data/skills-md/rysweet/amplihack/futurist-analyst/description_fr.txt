Analyse les événements du point de vue du futur en utilisant la planification de scénarios, l'analyse des tendances, les signaux faibles,
les moteurs du changement et les méthodes de prévision (exploratoire, normative, rétrospective).
Fournit des informations sur les futurs possibles, les tendances émergentes, les forces perturbatrices, la prospective stratégique et les scénarios alternatifs.
À utiliser dans les cas suivants : planification stratégique, tendances émergentes, évaluation technologique, planification à long terme, navigation dans l'incertitude.
Évalue : les tendances, les signaux faibles, les facteurs de changement, les futurs plausibles, les options stratégiques, les plages d'incertitude.
