Installation du SDK .NET et du runtime sur Windows, macOS et Linux. Gère la détection de version, les installateurs spécifiques à la plate-forme (WinGet, Homebrew, apt, dnf), la sélection du SDK par rapport au runtime, l'installation hors ligne, la configuration de Docker et le dépannage. S'active automatiquement pour l'installation, la configuration, la gestion des versions et le déploiement multiplateforme .NET.
