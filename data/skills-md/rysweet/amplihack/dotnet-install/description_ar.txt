.NET SDK والتثبيت في وقت التشغيل عبر أنظمة التشغيل Windows وmacOS وLinux. يتعامل مع الكشف عن الإصدار، والمثبتات الخاصة بالنظام الأساسي (WinGet، وHomebrew، وapt، وdnf)، واختيار SDK مقابل وقت التشغيل، والتثبيت دون اتصال بالإنترنت، وإعداد Docker، واستكشاف الأخطاء وإصلاحها. يتم التنشيط تلقائيًا لتثبيت .NET، والإعداد، وإدارة الإصدارات، والنشر متعدد الأنظمة الأساسية.
