.NET SDK и установка во время выполнения в Windows, macOS и Linux. Обеспечивает определение версии, установщики для конкретной платформы (WinGet, Homebrew, apt, dnf), выбор SDK или среды выполнения, автономную установку, настройку Docker и устранение неполадок. Автоматически активируется для установки .NET, настройки, управления версиями и развертывания на нескольких платформах.
