Adoptez les workflows agentiques GitHub (gh-aw) dans n'importe quel référentiel. Examine les flux de travail existants, identifie les lacunes, crée des flux de travail agents et gère les problèmes de CI/fusion. S'active automatiquement pour l'adoption de gh-aw, la configuration de flux de travail agent ou les demandes d'automatisation de flux de travail.
