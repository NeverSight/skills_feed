Автоматически генерирует диаграммы потока кода на основе анализа модулей Python.
Обнаруживает, когда архитектурные диаграммы устаревают (код изменился, а диаграмма — нет).
Используйте при создании новых модулей, проверке PR на предмет влияния на архитектуру или проверке актуальности диаграммы.
Создает диаграммы-русалки, показывающие импорт, зависимости и отношения модулей.
