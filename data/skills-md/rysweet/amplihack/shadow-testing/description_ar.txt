ينشئ بيئات حاوية معزولة لاختبار التغييرات المحلية المتاحة قبل الدفع.
يُستخدم عند اختبار تغييرات المكتبة، أو تنسيق الريبو المتعدد، أو التحقق من صحة "العمل على جهازي" ← "العمل في CI".
يوفر لقطات من حزمة git، وخادم git المضمن، وإعادة كتابة عنوان URL الانتقائي، وعزل ذاكرة التخزين المؤقت لمدير الحزم.
يعمل مع أي وكيل ترميز عبر واجهة سطر الأوامر (CLI) المستقلة أو البرامج النصية لـ Shell أو Docker Compose.
