Анализирует код для выявления непроверенных функций, областей с низким покрытием и отсутствующих крайних случаев.
Используйте при проверке покрытия тестированием или планировании улучшений тестирования.
Генерирует конкретные предложения по тестированию с примерами шаблонов в соответствии с пирамидой тестирования amplihack (60% модульного тестирования, 30% интеграции, 10% E2E).
Можно использовать Cover.py для проектов Python.
