Конвертируйте документы (PDF, Word, Excel, PowerPoint, изображения, HTML) в Markdown с помощью microsoft/markitdown. Используйте для анализа документов, извлечения контента, предварительной обработки для LLM или пакетного преобразования документов. Поддерживает изображения с описаниями OCR/LLM, аудиотранскрипцией и ZIP-архивами.
