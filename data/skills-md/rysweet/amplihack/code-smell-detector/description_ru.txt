Определяет антипаттерны, характерные для философии amplihack.
Используйте при проверке кода на наличие проблем с качеством или при рефакторинге.
Обнаруживает: чрезмерную абстракцию, сложное наследование, большие функции (>50 строк), тесную связь, отсутствие __all__ экспорта.
Предоставляет конкретные исправления и объяснения для каждого запаха.
