Crea rutas de aprendizaje personalizadas para tecnologías, marcos o conceptos.
Úselo para sesiones interactivas con el usuario solo para incorporar nuevas tecnologías, desarrollar habilidades en hackathon o planificar el desarrollo personal. No apto para uso en desarrollo o investigación automatizados.
Secuencia de recursos (documentos, tutoriales, ejercicios) según el nivel de habilidad actual y los objetivos de aprendizaje.
Se adapta al estilo de aprendizaje: práctico, primero la teoría, basado en proyectos.
