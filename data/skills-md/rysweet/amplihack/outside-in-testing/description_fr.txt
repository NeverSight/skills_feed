Génère des tests agents externes à l'aide du framework gadugi-agentic-test pour les applications CLI, TUI, Web et Electron.
À utiliser lorsque vous avez besoin de tests basés sur le comportement qui vérifient les interfaces externes sans connaissances en matière d'implémentation interne.
Crée des scénarios de test YAML que les agents d'IA exécutent, observent et valident par rapport aux résultats attendus.
Prend en charge la complexité progressive, depuis les simples tests de fumée jusqu'aux flux de travail avancés en plusieurs étapes.
