テキスト コンテンツを、プロフェッショナルなタイポグラフィと複数のビジュアル スタイルを備えた洗練された雑誌スタイルの HTML ページに変換します。トリガー シナリオ: (1) ユーザーがテキスト/記事を雑誌スタイルの HTML に変換したい、(2) ユーザーが「雑誌のレイアウト」、「雑誌のデザイン」、「記事のレイアウト」、「プロのタイポグラフィー」、「テキストの美化」、または「雑誌のレイアウト」について言及する、(3) ユーザーがエレガントにタイプセットされた HTML ページを必要とする、(4) ユーザーがデザイン コンテンツを PDF としてエクスポートする必要がある。 Tailwind CSS を使用して 12 のユニークな視覚スタイルをサポートします。
