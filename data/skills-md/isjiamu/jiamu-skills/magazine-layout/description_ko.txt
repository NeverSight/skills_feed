전문적인 타이포그래피와 다양한 시각적 스타일을 사용하여 텍스트 콘텐츠를 세련된 잡지 스타일의 HTML 페이지로 변환합니다. 트리거 시나리오: (1) 사용자가 텍스트/기사를 잡지 스타일 HTML로 변환하려고 함, (2) 사용자가 "잡지 레이아웃", "잡지 디자인", "기사 레이아웃", "전문 타이포그래피", "텍스트 미화" 또는 "잡지 레이아웃"을 언급함, (3) 사용자에게 우아하게 조판된 HTML 페이지가 필요함, (4) 사용자가 디자인 콘텐츠를 PDF로 내보내야 함. Tailwind CSS를 사용하여 12가지 고유한 시각적 스타일을 지원합니다.
