Intégrez et implémentez des applications et des écrans à l'aide de jetons, de styles et de composants gts-central-library. À utiliser lors de l'ajout du système de conception aux applications Vite, Next.js, Lovable ou React ; migration à partir de systèmes de jetons shadcn ou parallèles ; configuration de Tailwind v4 avec GTS CSS ; implémenter des mises en page de style configurateur ; ou dépanner les incompatibilités de jeton/thème/style.
