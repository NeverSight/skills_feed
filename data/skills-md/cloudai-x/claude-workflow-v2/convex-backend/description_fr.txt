Directives de développement backend convexes. À utiliser lors de l'écriture de fonctions Convex, de schémas, de requêtes, de mutations, d'actions ou de tout code backend dans un projet Convex. Se déclenche sur des tâches impliquant des opérations de base de données convexes, des abonnements en temps réel, le stockage de fichiers ou des fonctions sans serveur.
