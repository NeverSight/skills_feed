Guía completa para interactuar con el Indexador de Pools de Liquidez Hydric (Envio/HyperIndex). Utilice esta habilidad cuando necesite (1) consultar datos del fondo de liquidez en tiempo real, como TVL, volumen, tarifas o rendimientos/APY, (2) obtener metadatos y precios de tokens entre cadenas, (3) datos de protocolo agregados (Uniswap, etc.), (4) recuperar datos históricos de series de tiempo para análisis genéricos o (5) comprender las reglas de seguridad de valor específicas del indexador con seguimiento y sin seguimiento.
