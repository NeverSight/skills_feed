Этот навык следует использовать, когда пользователь просит «создать перехватчик», «добавить перехватчик PreToolUse/PostToolUse/Stop», «проверить использование инструмента», «реализовать перехватчики на основе подсказок», «использовать ${CLAUDE_PLUGIN_ROOT}», «настроить автоматизацию на основе событий», «блокировать опасные команды» или упоминает события перехвата (PreToolUse, PostToolUse, Stop, SubagentStop, SessionStart, SessionEnd, UserPromptSubmit, PreCompact, Уведомление). Содержит подробное руководство по созданию и реализации подключаемых модулей Claude Code с упором на расширенный API-интерфейс перехватчиков на основе подсказок.
