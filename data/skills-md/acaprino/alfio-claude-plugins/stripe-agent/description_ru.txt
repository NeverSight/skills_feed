Комплексный агент интеграции Stripe для платежей, подписок, выставления счетов и управления торговыми площадками. Используйте, когда Клоду необходимо работать с API-интерфейсом Stripe для создания клиентов, управления подписками, обработки платежей, обработки сеансов оформления заказа, настройки продуктов/цен, управления веб-перехватчиками, торговых площадок Connect, расчетных счетов, расчета налогов, предотвращения мошенничества или выполнения любых задач, связанных с платежами. Срабатывает при упоминании Stripe, платежей, подписок, выставления счетов, оформления заказов, счетов, намерений оплаты, регулярных платежей, Connect, торговой площадки, SCA, 3D Secure или споров.
