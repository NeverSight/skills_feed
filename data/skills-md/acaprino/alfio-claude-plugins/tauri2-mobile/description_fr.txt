Conseils d'experts pour développer, tester et déployer des applications mobiles avec Tauri 2. À utiliser lorsque vous travaillez avec le développement mobile Tauri 2 pour Android/iOS, y compris la configuration du projet, les modèles backend Rust, l'intégration frontend, l'utilisation des plugins (biométrique, géolocalisation, notifications, IAP), les tests d'émulateur/ADB, la signature de code et le déploiement Play Store/App Store.
