Habilidad de refactorización sistemática de código que transforma código complejo y difícil de entender en código claro, bien documentado y mantenible, preservando al mismo tiempo la corrección. Úselo cuando los usuarios solicitan código "legible", "mantenible" o "limpio", durante revisiones de código que señalan problemas de comprensión, para modernización de código heredado o en contextos educativos/de incorporación. Aplica patrones de refactorización estructurados con validación.
