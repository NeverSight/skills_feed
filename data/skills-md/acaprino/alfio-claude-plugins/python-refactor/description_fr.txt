Compétence en refactorisation systématique du code qui transforme un code complexe et difficile à comprendre en un code clair, bien documenté et maintenable tout en préservant son exactitude. À utiliser lorsque les utilisateurs demandent un code « lisible », « maintenable » ou « propre », lors de révisions de code signalant des problèmes de compréhension, pour la modernisation du code existant ou dans des contextes éducatifs/d'intégration. Applique des modèles de refactoring structurés avec validation.
