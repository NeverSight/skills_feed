Навыки мультиплатформенного и многоканального уведомления для агентов кода ИИ. Отправляет уведомления (звуковые, оповещения macOS, Telegram, электронную почту, Slack, Discord), когда агенту требуется взаимодействие с пользователем или он выполняет задачу. Поддерживает Claude Code, GitHub Copilot CLI, Cursor, Codex и Aider.
