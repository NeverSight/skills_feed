Plattformübergreifende und kanalübergreifende Benachrichtigungsfähigkeit für KI-Code-Agenten. Sendet Benachrichtigungen (Sound, macOS-Benachrichtigung, Telegram, E-Mail, Slack, Discord), wenn der Agent eine Benutzerinteraktion benötigt oder eine Aufgabe abschließt. Unterstützt Claude Code, GitHub Copilot CLI, Cursor, Codex und Aider.
