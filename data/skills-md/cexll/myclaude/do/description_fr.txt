Cette compétence doit être utilisée pour le développement de fonctionnalités structurées avec une compréhension de la base de code. Se déclenche sur la commande /do. Fournit un flux de travail en 7 phases (découverte, exploration, clarification, architecture, mise en œuvre, révision, résumé) utilisant codeagent-wrapper pour orchestrer l'explorateur de code, l'architecte de code, le réviseur de code et développer des agents en parallèle.
