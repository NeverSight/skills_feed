Этот навык следует использовать для разработки структурированных функций с пониманием кодовой базы. Запускается по команде /do. Обеспечивает 7-этапный рабочий процесс (обнаружение, исследование, уточнение, архитектура, реализация, обзор, сводка) с использованием codeagent-wrapper для параллельной организации проводника кода, архитектора кода, проверяющего кода и разработки агентов.
