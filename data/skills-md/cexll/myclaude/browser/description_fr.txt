Cette compétence doit être utilisée pour les tâches d'automatisation du navigateur à l'aide du protocole Chrome DevTools (CDP). Se déclenche lorsque les utilisateurs doivent lancer Chrome avec débogage à distance, parcourir des pages, exécuter JavaScript dans le contexte du navigateur, capturer des captures d'écran ou sélectionner de manière interactive des éléments DOM. Aucun serveur MCP requis.
