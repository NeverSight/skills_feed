Этот навык следует использовать для задач автоматизации браузера с использованием протокола Chrome DevTools (CDP). Срабатывает, когда пользователям необходимо запустить Chrome с удаленной отладкой, перемещаться по страницам, выполнять JavaScript в контексте браузера, делать снимки экрана или интерактивно выбирать элементы DOM. Сервер MCP не требуется.
