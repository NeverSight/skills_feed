Cette compétence doit être utilisée lors de la génération de cas de test complets à partir de documents PRD ou d'exigences des utilisateurs. Se déclenche lorsque les utilisateurs demandent la génération de scénarios de test, la planification du contrôle qualité, la création de scénarios de test ou ont besoin d'une documentation de test structurée. Produit des cas de test détaillés couvrant les scénarios fonctionnels, les cas extrêmes, la gestion des erreurs et les transitions d’état.
