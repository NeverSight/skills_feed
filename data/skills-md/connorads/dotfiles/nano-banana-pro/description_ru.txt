Создавайте и редактируйте изображения с помощью API Google Nano Banana Pro (Gemini 3 Pro Image). Используйте, когда пользователь запрашивает создание, создание, редактирование, изменение, изменение или обновление изображений. Также используйте, когда пользователь ссылается на существующий файл изображения и просит изменить его каким-либо образом (например, «изменить это изображение», «изменить фон», «заменить X на Y»). Поддерживает как генерацию текста в изображение, так и редактирование изображения в изображение с настраиваемым разрешением (1K по умолчанию, 2K или 4K для высокого разрешения). НЕ читайте сначала файл изображения — используйте этот навык напрямую с параметром --input-image.
