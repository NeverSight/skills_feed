Получайте актуальную документацию по программным библиотекам, платформам и компонентам через API Context7. Этот навык следует использовать при поиске документации для любой библиотеки или платформы программирования, поиске примеров кода для конкретных API или функций, проверке правильности использования функций библиотеки или получении текущей информации о библиотечных API, которые могли измениться со времени обучения.
