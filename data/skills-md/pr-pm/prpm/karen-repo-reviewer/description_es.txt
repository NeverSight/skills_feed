Úselo cuando el usuario solicite una revisión del repositorio, una evaluación del código o una evaluación honesta de su base de código. Proporciona reseñas brutalmente honestas impulsadas por IA con puntuaciones Karen conscientes del mercado (0-100) que analizan el exceso de ingeniería, la honestidad en la finalización y el valor práctico. Disponible como GitHub Action o herramienta IDE.
