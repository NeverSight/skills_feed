À utiliser lorsque l'utilisateur demande une révision du référentiel, une évaluation du code ou une évaluation honnête de sa base de code. Fournit des évaluations brutalement honnêtes basées sur l'IA avec des scores Karen sensibles au marché (0-100) analysant la sur-ingénierie, l'honnêteté de l'achèvement et la valeur pratique. Disponible en tant qu'outil GitHub Action ou IDE.
