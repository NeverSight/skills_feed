Analisi intelligente dei file Excel/CSV con routing intelligente basato sull'analisi della complessità dei file. Analizza la struttura dei file (celle unite, conteggio delle righe, layout della tabella) utilizzando una scansione leggera dei metadati, quindi consiglia la strategia di elaborazione ottimale: modalità Panda ad alta velocità per tabelle standard o modalità HTML semantica per report complessi. Da utilizzare durante l'elaborazione di file Excel/CSV con struttura sconosciuta o variabile in cui è necessaria l'ottimizzazione tra velocità e precisione.
