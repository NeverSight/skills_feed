Интеллектуальный анализ файлов Excel/CSV с интеллектуальной маршрутизацией на основе анализа сложности файлов. Анализирует структуру файла (объединенные ячейки, количество строк, макет таблицы) с помощью облегченного сканирования метаданных, а затем рекомендует оптимальную стратегию обработки — либо высокоскоростной режим Pandas для стандартных таблиц, либо режим семантического HTML для сложных отчетов. Используйте при обработке файлов Excel/CSV с неизвестной или меняющейся структурой, когда необходима оптимизация между скоростью и точностью.
