Konventionelle Commits v1.0.0-Standards für Git-Nachrichten. Wird verwendet, wenn (1) Git-Commits erstellt werden, (2) Commit-Nachrichten geschrieben oder entworfen werden, (3) das Format von Commit-Nachrichten überprüft wird, (4) Commit-Konventionen erklärt werden oder (5) die Compliance von Commit-Nachrichten überprüft wird.
