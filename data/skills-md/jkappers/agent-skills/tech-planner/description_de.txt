Wird verwendet, wenn (1) die technische Machbarkeit einer Funktionsspezifikation bewertet wird, (2) Technologie- oder Framework-Entscheidungen getroffen werden, (3) die Systemarchitektur für eine neue Funktion definiert wird, (4) Architekturentscheidungen als ADRs dokumentiert werden, (5) Integrationspunkte und Datenfluss für eine vorgeschlagene Änderung analysiert werden.
