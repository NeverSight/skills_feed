Da utilizzare durante la scrittura, la revisione o il miglioramento di qualsiasi prompt LLM o istruzioni dell'agente AI. Richiamare prompt di sistema, competenze di Claude Code, file CLAUDE.md, regole, comandi barra o domande generali di ingegneria dei prompt. Copre l'economia dei token, il linguaggio imperativo, il determinismo, la formattazione, i modificatori di enfasi e gli anti-pattern. Si applica a tutte le piattaforme LLM (Claude, GPT, Gemini, Llama).
