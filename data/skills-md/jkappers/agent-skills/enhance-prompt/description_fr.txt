À utiliser lors de la rédaction, de la révision ou de l'amélioration d'une invite LLM ou d'instructions d'agent IA. Invoquez les invites système, les compétences Claude Code, les fichiers CLAUDE.md, les règles, les commandes slash ou les questions générales d'ingénierie. Couvre l'économie des jetons, le langage impératif, le déterminisme, le formatage, les modificateurs d'accentuation et les anti-modèles. S'applique à toutes les plateformes LLM (Claude, GPT, Gemini, Llama).
