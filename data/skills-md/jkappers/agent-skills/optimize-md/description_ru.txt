Оптимизирует документы уценки для повышения эффективности, ясности и потребления токенов. Используйте, когда (1) файл уценки требует оптимизации для использования в качестве контекста LLM, (2) сокращение количества токенов в документации без потери смысла, (3) преобразование подробной документации в краткий справочный материал, (4) улучшение структуры и возможности сканирования файлов уценки или (5) подготовка документов с рекомендациями или знаниями для использования агентом.
