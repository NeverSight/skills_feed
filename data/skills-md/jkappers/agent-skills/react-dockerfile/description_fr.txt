Créez des Dockerfiles multi-étapes optimisés et sécurisés pour les applications React (Vite, CRA, Next.js statique). À utiliser lors de (1) la création d'un nouveau fichier Docker pour un projet React, (2) la conteneurisation d'une application React/Vite, (3) l'optimisation d'un fichier Docker React existant, (4) la configuration de Docker pour React avec Nginx ou (5) l'utilisateur mentionne React et Docker/conteneur ensemble.
