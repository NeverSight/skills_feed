Crea file Dockerfile multi-fase ottimizzati e sicuri per le applicazioni React (Vite, CRA, Next.js statico). Da utilizzare quando (1) si crea un nuovo Dockerfile per un progetto React, (2) si containerizza un'applicazione React/Vite, (3) si ottimizza un React Dockerfile esistente, (4) si configura Docker per React con Nginx o (5) l'utente menziona React e Docker/contenitore insieme.
