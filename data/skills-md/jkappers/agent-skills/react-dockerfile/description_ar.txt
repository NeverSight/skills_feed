أنشئ ملفات Dockerfiles محسّنة وآمنة ومتعددة المراحل لتطبيقات React (Vite وCRA وNext.js static). يُستخدم عند (1) إنشاء ملف Dockerfile جديد لمشروع React، أو (2) وضع حاوية لتطبيق React/Vite، أو (3) تحسين ملف React Dockerfile الحالي، أو (4) إعداد Docker لـ React مع Nginx، أو (5) يذكر المستخدم React وDocker/container معًا.
