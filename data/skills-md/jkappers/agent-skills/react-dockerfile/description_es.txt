Cree Dockerfiles de varias etapas optimizados y seguros para aplicaciones React (Vite, CRA, Next.js estático). Úselo al (1) crear un nuevo Dockerfile para un proyecto de React, (2) contener una aplicación React/Vite, (3) optimizar un React Dockerfile existente, (4) configurar Docker para React con Nginx o (5) el usuario menciona React y Docker/contenedor juntos.
