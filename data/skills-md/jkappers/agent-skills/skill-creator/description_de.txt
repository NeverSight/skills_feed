Leitfaden zum Erstellen effektiver Agentenkompetenzen gemäß dem offenen Standard für Agentenkompetenzen. Dieser Skill sollte verwendet werden, wenn Benutzer einen neuen Skill erstellen (oder einen vorhandenen Skill aktualisieren) möchten, der die Agentenfunktionen um Spezialwissen, Arbeitsabläufe oder Tool-Integrationen erweitert.
