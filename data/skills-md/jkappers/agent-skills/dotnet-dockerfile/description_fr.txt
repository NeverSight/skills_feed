Créez des fichiers Dockerfiles à plusieurs étapes optimisés et sécurisés pour les applications .NET Core et ASP.NET Core. À utiliser lors de (1) la création d'un nouveau fichier Docker pour un projet .NET, (2) la conteneurisation d'une application .NET, (3) l'optimisation d'un fichier Docker .NET existant, (4) la configuration de Docker pour les applications .NET 6/7/8+, ou (5) l'utilisateur mentionne .NET et Docker/conteneur ensemble.
