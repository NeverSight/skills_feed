Создавайте оптимизированные, безопасные многоэтапные файлы Dockerfile для приложений .NET Core и ASP.NET Core. Используйте при (1) создании нового файла Dockerfile для проекта .NET, (2) контейнеризации приложения .NET, (3) оптимизации существующего файла Docker .NET, (4) настройке Docker для приложений .NET 6/7/8+ или (5) когда пользователь упоминает .NET и Docker/контейнер вместе.
