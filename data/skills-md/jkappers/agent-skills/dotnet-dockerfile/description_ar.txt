قم بإنشاء ملفات Dockerfiles محسنة وآمنة ومتعددة المراحل لتطبيقات .NET Core وASP.NET Core. يُستخدم عند (1) إنشاء ملف Dockerfile جديد لمشروع .NET، أو (2) وضع تطبيق .NET في حاوية، أو (3) تحسين ملف Dockerfile.NET موجود، أو (4) إعداد Docker لتطبيقات .NET 6/7/8+، أو (5) يذكر المستخدم .NET وDocker/container معًا.
