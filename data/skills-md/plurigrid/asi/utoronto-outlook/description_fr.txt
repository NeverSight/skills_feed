Accès au courrier électronique Outlook sans tête de l'Université de Toronto via IMAP/SMTP avec OAuth2. Utilise l'ID client pré-autorisé de Thunderbird pour contourner les exigences de consentement de l'administrateur (AADSTS65002). Flux de code de l'appareil pour l'authentification initiale, trousseau macOS pour le cache de jetons.
