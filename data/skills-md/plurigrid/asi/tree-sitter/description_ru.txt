Анализ кода на основе AST с использованием Tree-Sitter. Используйте для анализа структуры кода, извлечения символов, поиска шаблонов с помощью запросов Tree-Sitter, анализа сложности и понимания архитектуры кода. Поддерживает Python, JavaScript, TypeScript, Go, Rust, C, C++, Swift, Java, Kotlin, Julia и другие.
