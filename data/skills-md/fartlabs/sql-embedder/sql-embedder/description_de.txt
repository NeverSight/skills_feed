Anweisungen zum Arbeiten mit von SQL-Embedder generierten Dateien. Verwenden Sie diese Option, wenn Sie .sql.ts-Dateien bearbeiten oder damit arbeiten, die den Kommentar „Diese Datei wurde von sql-embedder generiert“ enthalten. Bearbeiten Sie diese generierten Dateien NIEMALS direkt – bearbeiten Sie stattdessen die Quell-SQL-Datei und generieren Sie sie neu.
