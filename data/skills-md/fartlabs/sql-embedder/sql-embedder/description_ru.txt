Инструкции по работе с файлами, сгенерированными sql-embedder. Используйте при редактировании или работе с файлами .sql.ts, содержащими комментарий «Этот файл был создан с помощью sql-embedder». НИКОГДА не редактируйте эти сгенерированные файлы напрямую — вместо этого отредактируйте исходный файл .sql и создайте заново.
