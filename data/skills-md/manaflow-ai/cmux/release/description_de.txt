Bereiten Sie ein Cmux-Release durchgängig vor und versenden Sie es: Wählen Sie die nächste Version aus, kuratieren Sie benutzerbezogene Änderungsprotokolleinträge, aktualisieren Sie Versionen, öffnen und überwachen Sie eine Release-PR, führen Sie veröffentlichte Artefakte zusammen, kennzeichnen Sie sie und überprüfen Sie sie. Verwenden Sie diese Option, wenn Sie zum Ausschneiden, Vorbereiten, Veröffentlichen oder Markieren einer neuen Veröffentlichung aufgefordert werden.
