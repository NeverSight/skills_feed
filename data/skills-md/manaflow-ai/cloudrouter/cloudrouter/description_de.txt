Verwalten Sie Cloud-Entwicklungs-Sandboxen mit Cloudrouter. Erstellen, synchronisieren und greifen Sie auf Remote-VMs mit GPU-Unterstützung, Docker und Browser-Automatisierung zu. Verwenden Sie es, wenn Sie aufgefordert werden, eine Sandbox zu erstellen, eine Entwicklungsumgebung einzurichten, Code in der Cloud auszuführen, GPUs zu verwenden, einen Browser zu automatisieren oder mit Remote-VMs zu interagieren.
