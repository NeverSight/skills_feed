Directrices para crear habilidades efectivas. Úselo cuando el usuario desee crear una nueva habilidad (o actualizar una existente) para ampliar las capacidades de Claude, incorporando conocimientos especializados, flujos de trabajo o integraciones de herramientas. Se activa cuando el usuario menciona "crear una habilidad", "escribir una habilidad", "desarrollo de habilidades" o "creación de habilidades".
