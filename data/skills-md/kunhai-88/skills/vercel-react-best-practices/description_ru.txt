Руководство по оптимизации производительности для React и Next.js от Vercel. Используйте его при написании, проверке или рефакторинге кода React/Next.js, чтобы обеспечить оптимальные шаблоны производительности. Запускается: компонентами React, страницами Next.js, получением данных, оптимизацией пакетов, улучшением производительности и т. д.
