Набор инструментов для создания сложных, многокомпонентных HTML-артефактов claude.ai с использованием современных веб-технологий внешнего интерфейса (React, Tailwind CSS, shadcn/ui). Для сложных артефактов, которым требуются компоненты управления состоянием, маршрутизации или shadcn/ui, а не для простых однофайловых артефактов HTML/JSX.
