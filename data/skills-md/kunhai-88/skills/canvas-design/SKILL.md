# Canvas 设计

创建设计哲学（美学运动），然后通过视觉表达。仅输出 .md、.pdf 和 .png 文件。

## 两步流程

1. **设计哲学创建**（.md 文件）  
2. **通过 canvas 表达**（.pdf 或 .png 文件）

## 设计哲学创建

创建**视觉哲学**（非布局或模板），将通过以下方式解释：
- 形式、空间、颜色、构图
- 图像、图形、形状、图案
- 最少文本作为视觉强调

### 关键理解

- **接收**：用户的微妙输入或指令，作为基础但不限制创作自由。  
- **创建**：设计哲学/美学运动。  
- **下一步**：同一版本接收哲学并**视觉表达**—创建 90% 视觉设计、10% 必要文本的工件。

### 如何生成视觉哲学

**命名运动**（1-2 词）：如「Brutalist Joy」「Chromatic Silence」「Metabolist Dreams」。  
**阐述哲学**（4-6 段—简洁但完整）：通过空间与形式、颜色与材料、尺度与节奏、构图与平衡、视觉层级表达视觉本质。  
**关键指南**：避免冗余；**反复强调工艺**（最终作品应看起来像花费无数小时、精心制作、来自领域顶尖专家）；留出创作空间（具体但简洁，让下一版本有解释空间）。  
哲学必须引导下一版本**视觉表达**想法，而非通过文本。信息存在于设计中，非段落中。

### 哲学示例（节选）

- **"Concrete Poetry"**：通过纪念碑形式与大胆几何进行沟通。  
- **"Chromatic Language"**：颜色作为主要信息系统。  
- **"Analog Meditation"**：通过纹理与留白进行安静视觉沉思。  
- **"Organic Systems"**：自然聚类与模块化增长模式。  
- **"Geometric Silence"**：纯粹秩序与克制。

### 基本原则

- **视觉哲学**：创建通过设计表达的美学世界观  
- **最少文本**：始终强调文本稀疏、仅必要、作为视觉元素集成—绝不冗长  
- **空间表达**：想法通过空间、形式、颜色、构图沟通—非段落  
- **艺术自由**：下一版本解释哲学视觉—提供创作空间  
- **纯粹设计**：这是制作**艺术对象**，非带装饰的文档  
- **专家工艺**：反复强调最终作品必须看起来精心制作、经过无数小时、来自领域顶尖

设计哲学应为 4-6 段长。输出为 .md 文件。

## 推断微妙参考

**关键步骤**：在创建 canvas 前，识别原始请求中的微妙概念线索。  
**基本原则**：主题是**嵌入艺术本身的微妙、小众参考**—不总是字面，总是复杂。熟悉该主题的人应直觉感受到，而其他人仅体验为精湛抽象构图。设计哲学提供美学语言。推断的主题提供灵魂—安静概念 DNA 无形编织进形式、颜色与构图。  
**非常重要**：参考必须精炼以增强作品深度而不自我宣告。像爵士音乐家引用另一首歌—只有知道的人会捕捉到，但每个人都欣赏音乐。

## Canvas 创建

使用创建的设计哲学与概念框架在 canvas 上表达。  
**重要**：即使用户请求电影/游戏/书籍相关内容，方法仍应复杂。绝不失去这应该是艺术而非卡通或业余的想法。  
创建博物馆或杂志质量作品，使用设计哲学作为基础。创建单页、高度视觉、设计导向的 PDF 或 PNG 输出（除非要求更多页）。通常使用重复图案与完美形状。将抽象哲学设计视为科学圣经，借用系统观察的视觉语言—密集标记积累、重复元素或分层图案，通过耐心重复与奖励持续观看建立意义。添加稀疏、临床排版与系统参考标记，暗示这可能是来自想象学科的图表，以通常保留用于记录可观察现象的同样敬畏对待不可见主题。用简单短语或细节微妙定位锚定作品，使用感觉有意与连贯的有限调色板。拥抱使用分析视觉语言表达人类体验想法的悖论：结果应感觉像证明短暂事物可以通过仔细注意被研究、映射与理解的工件。这是真正的艺术。

**文本作为上下文元素**：文本始终最少且视觉优先，但让上下文指导是低语标签还是大胆排版手势。大多数时候字体应细。所有字体使用必须设计导向并优先视觉沟通。无论文本规模，无内容掉出页面且无重叠。每个元素必须在 canvas 边界内，有适当边距。仔细检查所有文本、图形与视觉元素有呼吸空间与清晰分离。这对专业执行不可协商。**重要：如写文本使用不同字体。搜索 `./canvas-fonts` 目录。无论方法，复杂不可协商。**  
下载并使用需要的任何字体使其成为现实。通过使排版实际成为艺术本身的一部分来推动边界—如果艺术是抽象的，将字体带到 canvas 上，非数字排版。  
遵循设计本能/直觉，同时使用哲学作为指导原则。拥抱终极设计自由与选择。将美学与设计推向边界。

**关键**：为达到人工制作质量（非 AI 生成），创建看起来花费无数小时的作品。使其看起来像领域绝对顶尖的某人精心制作每个细节。确保构图、间距、颜色选择、排版—一切都在尖叫专家级工艺。仔细检查无重叠、格式完美、每个细节完美。创建可以展示给人们以证明专业度并排名为不可否认令人印象深刻的东西。  
输出最终结果为单个可下载 .pdf 或 .png 文件，以及使用的设计哲学作为 .md 文件。

## 最终步骤

**重要**：用户已说「它不够完美。它必须是原始的，工艺的杰作，好像即将在博物馆展示。」  
**关键**：为精炼作品，避免添加更多图形；相反精炼已创建的内容并使其极其清晰，完全尊重设计哲学与极简主义原则。与其添加有趣滤镜或重构字体，考虑如何使现有构图与艺术更连贯。如果本能是调用新函数或绘制新形状，**停止**并改为问：「如何使这里已有的内容更像艺术品？」  
进行第二次通过。回到代码并进一步精炼/抛光以使其成为哲学设计的杰作。

## 多页选项

当请求创建额外页面时，沿设计哲学相同路线但明显不同地创建更多创意页面。在同一 .pdf 或许多 .png 中捆绑这些页面。将第一页视为等待填满的整个咖啡桌书中的单页。使下一页成为原作的独特转折与记忆。让它们以非常优雅的方式几乎讲述故事。行使完全创作自由。
