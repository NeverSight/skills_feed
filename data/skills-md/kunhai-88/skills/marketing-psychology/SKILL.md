# 营销心理学与心智模型

将心理学与心智模型应用于营销：理解人们为何购买、如何合规影响行为、如何做更好的营销决策。

## 使用方式

1. 识别适用于情境的心智模型  
2. 说明模型背后的心理学  
3. 给出具体营销应用  
4. 建议如何合规落地  

## 核心模型速查

### 策略思维
- **第一性原理**：拆解到基本事实再构建；用 5 Whys 找到根因。  
- **Jobs to Be Done**：人们「雇用」产品完成任务；围绕结果而非功能。  
- **倒推**：先问「什么会导致失败？」再避免。  
- **二八法则**：聚焦带来 80% 结果的 20%。  
- **机会成本 / 边际递减**：比较取舍；知道何时 diversification。  
- **二阶思考**：考虑间接效应（如促销教用户等打折）。  
- **地图≠疆域**：数据与模型有用，但不等于真实用户。  

### 买家心理
- **基本归因谬误**：别人不买多半是情境（如流程差），非人品。  
- **曝光效应 / 可得性启发**：熟悉促进偏好；案例让成功更可想象。  
- **确认偏误**：贴合既有信念比硬刚有效。  
- **模仿欲望 / 禀赋效应 / IKEA 效应**：社会认同、拥有感、投入感提升价值。  
- **零价效应 / 现状偏见 / 默认效应**：免费、少折腾、预设选项影响巨大。  
- **选择悖论 / 目标梯度 / 峰终定律**：少选项、进度可视化、峰值与结尾塑造体验。  
- **损失厌恶 / 锚定 / 诱饵效应**：损失框架、第一数字、第三选项改变选择。  

### 说服与行为
- **互惠 / 承诺一致**：先给予价值；小承诺带来后续行动。  
- **权威 / 喜爱**：专家背书、相似性提升信任。  
- **稀缺 / 登门槛**：真实稀缺；小请求→大请求。  
- **框架效应**：「90% 成功率」vs「10% 失败率」感受不同。  

### 定价心理
- **魅力定价**：.99 比整百感觉更便宜。  
- **100 法则**：<100 用百分比折扣，>100 用绝对金额。  
- **好-更好-最好**：三档定价，中间为主推。  

### 设计与发展
- **希克定律**：选项越多决策越慢。  
- **AIDA / 7 次触达 / BJ Fogg / EAST**：注意→兴趣→欲望→行动；多触点；动机×能力×提示；易得、吸引、社交、及时。  
- **北极星指标 / 反馈回路 / 复合增长 / 网络效应**：对齐单指标；正循环；持续积累；用户越多价值越大。  

## 按场景选模型

| 场景 | 可参考模型 |
|------|------------|
| 转化低 | 希克定律、激活能、BJ Fogg、摩擦 |
| 价格异议 | 锚定、框架、心理账户、损失厌恶 |
| 建立信任 | 权威、社会认同、互惠、出丑效应 |
| 提升紧迫 | 稀缺、损失厌恶、 Zeigarnik |
| 留存/流失 | 禀赋、切换成本、现状偏见 |
| 增长停滞 | 瓶颈理论、局部vs全局最优、复合 |
| 决策瘫痪 | 选择悖论、默认、助推 |
