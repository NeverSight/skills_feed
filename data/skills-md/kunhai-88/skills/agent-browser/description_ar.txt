أداة أتمتة المتصفح للتنقل عبر الويب، والتفاعل، ولقطات الشاشة، والتسجيل، وما إلى ذلك. استخدمها عند أتمتة عمليات المتصفح، أو اختبار صفحات الويب، أو استخراج المحتوى، أو ملء النماذج، أو التقاط لقطات الشاشة، أو تسجيل مقاطع الفيديو، أو ذكر "أتمتة المتصفح"، أو "اختبار الويب"، أو "متصفح الوكيل"، أو "عمليات المتصفح".
