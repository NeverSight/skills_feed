Utilisé lorsque l'utilisateur a besoin d'écrire, de réécrire ou d'améliorer une copie marketing pour n'importe quelle page, y compris la page d'accueil, la page de destination, la page de tarification, la page de fonctionnalités, la page à propos, la page produit. Mots déclencheurs : « rédiger une copie », « améliorer cette copie », « réécrire cette page », « copie marketing », « suggestions de titres », « copie CTA ». La copie de l'e-mail est en séquence d'e-mails, la copie contextuelle est en popup-cro.
