Utilisé lorsque l'utilisateur a besoin d'écrire, de réécrire ou d'améliorer une copie marketing pour n'importe quelle page, y compris la page d'accueil, la page de destination, la page de tarification, la page de fonctionnalités, la page à propos ou la page produit.  
Mots déclencheurs : « rédiger une copie », « améliorer cette copie », « réécrire cette page », « copie marketing », « suggestions de titres », « copie CTA ».  
Copie de l'e-mail : voir la séquence d'e-mails ; copie contextuelle : voir popup-cro.
