Используется при создании масштабных SEO-страниц с помощью шаблонов + данных.  
Триггерные термины: программатик SEO, страницы шаблонов, масштабированные страницы, страницы каталогов, страницы геолокации, [ключевое слово]+[город], страницы сравнения, страницы интеграции, создание больших объемов страниц для SEO.  
Чтобы провести аудит существующего SEO, см. SEO-аудит.
