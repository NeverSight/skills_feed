Коллекция ресурсов для написания различных внутренних коммуникаций в предпочтительном для компании формате. Клод должен использовать этот навык всякий раз, когда его просят написать какой-либо тип внутренней коммуникации (отчеты о статусе, обновления руководства, обновления 3P, краткие сведения о компании, часто задаваемые вопросы, отчеты об инцидентах, обновления проекта и т. д.).
