À utiliser pour optimiser les conversions sur n'importe quelle page marketing : page d'accueil, page de destination, page de tarification, page de fonctionnalités, blog. Mots déclencheurs : CRO, optimisation du taux de conversion, cette page ne convertit pas, améliore les conversions, pourquoi cette page ne fonctionne-t-elle pas. Flux d'inscription : signup-flow-cro ; activation post-inscription : onboarding-cro ; formulaires sans inscription : form-cro ; popups : popup-cro.
