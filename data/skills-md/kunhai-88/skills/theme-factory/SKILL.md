# Theme Factory 技能

提供精选的专业字体与颜色主题集合，每个包含精心选择的调色板与字体配对。选择主题后，可应用于任何工件。

## 用途

为演示幻灯片套牌或其他工件应用一致、专业的样式。每个主题包括：
- 带十六进制代码的连贯调色板
- 标题与正文的互补字体配对
- 适合不同上下文与受众的独特视觉身份

## 使用说明

要应用样式到幻灯片套牌或其他工件：

1. **显示主题展示**：显示 `theme-showcase.pdf` 文件以允许用户查看所有可用主题。不对其做任何修改；仅显示文件供查看。
2. **询问选择**：询问要应用到套牌的主题
3. **等待选择**：获得关于所选主题的明确确认
4. **应用主题**：选择主题后，将所选主题的颜色与字体应用到套牌/工件

## 可用主题

以下 10 个主题可用，每个在 `theme-showcase.pdf` 中展示：

1. **Ocean Depths** — 专业与平静的海洋主题
2. **Sunset Boulevard** — 温暖与充满活力的日落颜色
3. **Forest Canopy** — 自然与接地气的大地色调
4. **Modern Minimalist** — 干净与当代的灰度
5. **Golden Hour** — 丰富与温暖的秋季调色板
6. **Arctic Frost** — 凉爽与清脆的冬季主题
7. **Desert Rose** — 柔和与精致的尘土色调
8. **Tech Innovation** — 大胆与现代的科技美学
9. **Botanical Garden** — 新鲜与有机的花园颜色
10. **Midnight Galaxy** — 戏剧性与宇宙的深色调

## 主题详情

每个主题在 `themes/` 目录中定义，包含完整规格：
- 带十六进制代码的连贯调色板
- 标题与正文的互补字体配对
- 适合不同上下文与受众的独特视觉身份

## 应用流程

选择首选主题后：
1. 从 `themes/` 目录读取相应主题文件
2. 在整个套牌中一致应用指定颜色与字体
3. 确保适当对比与可读性
4. 在所有幻灯片中保持主题视觉身份

## 创建自己的主题

处理现有主题都不适合工件的情况，基于提供的输入生成新主题。生成与上述类似的新主题。给主题一个描述字体/颜色组合代表什么的类似名称。使用任何提供的基本描述选择适当颜色/字体。生成主题后，显示以供审查与验证。之后，如上所述应用主题。
