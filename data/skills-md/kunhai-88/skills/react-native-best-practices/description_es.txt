Proporcione una guía de optimización del rendimiento de React Native que cubra FPS, TTI, tamaño de paquete, pérdidas de memoria, re-renderizados y animaciones. Aplicable a tareas que involucran optimización de Hermes, bloqueo de subprocesos JS, sobrecarga de puentes, FlashList, módulos nativos o depuración de tartamudeos y caídas de fotogramas.
