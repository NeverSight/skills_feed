Fournissez un guide d'optimisation des performances de React Native couvrant les FPS, le TTI, la taille du bundle, les fuites de mémoire, les nouveaux rendus et les animations. Applicable aux tâches impliquant l'optimisation Hermes, le blocage de threads JS, la surcharge de pont, FlashList, les modules natifs ou le débogage des bégaiements et des pertes de trames.
