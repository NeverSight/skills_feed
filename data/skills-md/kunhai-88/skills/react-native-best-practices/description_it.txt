Fornire una guida all'ottimizzazione delle prestazioni di React Native che copra FPS, TTI, dimensioni del bundle, perdite di memoria, re-render e animazioni. Adatto per attività che coinvolgono l'ottimizzazione di Hermes, il blocco dei thread JS, l'overhead del bridge, FlashList, moduli nativi o il debug di stutter e cadute di frame.
