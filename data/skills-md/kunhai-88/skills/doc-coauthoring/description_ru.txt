Направляйте пользователей через структурированный рабочий процесс для совместного создания документов. Используйте это, когда они хотят написать документы, предложения, технические спецификации, записи решений или аналогичный структурированный контент. Рабочий процесс помогает им эффективно передавать контекст, итеративно уточнять и проверять, работает ли документ для читателей. Триггерные слова: написание документов, создание предложений, составление спецификаций или аналогичные задачи с документами.
