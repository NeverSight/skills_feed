Guíe a los usuarios a través de un flujo de trabajo estructurado para la colaboración de documentos. Úselo cuando quieran escribir documentos, propuestas, especificaciones técnicas, registros de decisiones o contenido estructurado similar. El flujo de trabajo ayuda a los usuarios a transmitir contexto de manera eficiente, refinar el contenido de forma iterativa y validar que el documento funcione para los lectores. Palabras desencadenantes: escribir documentos, crear propuestas, redactar especificaciones o tareas de documentos similares.
