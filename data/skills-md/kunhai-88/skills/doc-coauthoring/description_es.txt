Guíe a los usuarios a través de un flujo de trabajo estructurado para la creación colaborativa de documentos. Úselo cuando quieran escribir documentos, propuestas, especificaciones técnicas, registros de decisiones o contenido estructurado similar. El flujo de trabajo les ayuda a transmitir contexto de manera eficiente, refinar de forma iterativa y verificar que el documento funcione para los lectores. Palabras desencadenantes: escribir documentos, crear propuestas, redactar especificaciones o tareas de documentos similares.
