يدير مكونات ومشاريع shadcn - إضافة واجهة المستخدم والبحث عنها وإصلاحها وتصحيح الأخطاء والتصميم وإنشاءها. يوفر سياق المشروع ومستندات المكونات وأمثلة الاستخدام. ينطبق عند العمل مع shadcn/ui، أو سجلات المكونات، أو الإعدادات المسبقة، أو رموز --preset، أو أي مشروع يحتوي على ملفمكونات.json. يتم أيضًا تشغيل "shadcn init" أو "إنشاء تطبيق باستخدام --preset" أو "التبديل إلى --preset".
