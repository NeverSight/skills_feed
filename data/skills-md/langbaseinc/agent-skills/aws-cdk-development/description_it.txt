Esperto di AWS Cloud Development Kit (CDK) per la creazione di infrastrutture cloud con TypeScript/Python. Da utilizzare quando si creano stack CDK, si definiscono costrutti CDK, si implementa l'infrastruttura come codice o quando l'utente menziona CDK, CloudFormation, IaC, cdk synth, cdk deploy o desidera definire l'infrastruttura AWS a livello di codice. Copre la struttura dell'app CDK, i modelli di costruzione, la composizione dello stack e i flussi di lavoro di distribuzione.
