خبير AWS Cloud Development Kit (CDK) لبناء البنية التحتية السحابية باستخدام TypeScript/Python. يُستخدم عند إنشاء مجموعات CDK، أو تحديد بنيات CDK، أو تنفيذ البنية التحتية كرمز، أو عندما يذكر المستخدم CDK، أو CloudFormation، أو IaC، أو cdk Synth، أو نشر cdk، أو يريد تعريف البنية التحتية لـ AWS برمجيًا. يغطي بنية تطبيق CDK، وأنماط البناء، وتكوين المكدس، وسير عمل النشر.
