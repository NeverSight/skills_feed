AWS Cloud Development Kit (CDK)-Experte für den Aufbau einer Cloud-Infrastruktur mit TypeScript/Python. Verwenden Sie diese Option, wenn Sie CDK-Stacks erstellen, CDK-Konstrukte definieren, Infrastruktur als Code implementieren oder wenn der Benutzer CDK, CloudFormation, IaC, cdk synth, cdkployment erwähnt oder die AWS-Infrastruktur programmgesteuert definieren möchte. Behandelt die CDK-App-Struktur, Konstruktionsmuster, Stack-Zusammensetzung und Bereitstellungsworkflows.
