Эксперт по AWS Cloud Development Kit (CDK) для создания облачной инфраструктуры с помощью TypeScript/Python. Используйте при создании стеков CDK, определении конструкций CDK, реализации инфраструктуры в виде кода, а также когда пользователь упоминает CDK, CloudFormation, IaC, синтезатор cdk, развертывание cdk или хочет определить инфраструктуру AWS программным способом. Охватывает структуру приложения CDK, шаблоны построения, композицию стека и рабочие процессы развертывания.
