Well-Architected Framework를 기반으로 하는 AWS 서버리스 및 이벤트 기반 아키텍처 전문가입니다. 서버리스 API, Lambda 함수, REST API, 마이크로서비스 또는 비동기 워크플로를 구축할 때 사용합니다. TypeScript/Python, API 게이트웨이(REST/HTTP), DynamoDB, Step Functions, EventBridge, SQS, SNS 및 서버리스 패턴으로 Lambda를 다룹니다. 사용자가 서버리스, Lambda, API 게이트웨이, 이벤트 기반, 비동기 처리, 대기열, 게시/구독을 언급하거나 AWS 모범 사례를 사용하여 확장 가능한 서버리스 애플리케이션을 구축하려는 경우 필수적입니다.
