Biblioteca TypeScript para la interacción blockchain del protocolo NEAR. Utilice esta habilidad al escribir código que interactúe con el protocolo NEAR, incluida la visualización de datos de contratos, la llamada a métodos de contratos, el envío de tokens NEAR, la creación de transacciones, la creación de envoltorios de contratos con seguridad de tipos, la integración de billeteras (Wallet Selector, HOT Connect), los ganchos y proveedores de React (@near-kit/react), la administración de claves, las pruebas con sandbox, las metatransacciones (NEP-366) y la firma de mensajes (NEP-413).
