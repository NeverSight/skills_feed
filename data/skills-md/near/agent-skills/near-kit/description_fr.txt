Bibliothèque TypeScript pour l'interaction blockchain du protocole NEAR. Utilisez cette compétence lors de l'écriture de code qui interagit avec le protocole NEAR, notamment l'affichage des données du contrat, l'appel des méthodes de contrat, l'envoi de jetons NEAR, la création de transactions, la création de wrappers de contrat de type sécurisé, l'intégration de portefeuilles (Wallet Selector, HOT Connect), les hooks et les fournisseurs React (@near-kit/react), la gestion des clés, les tests avec sandbox, les méta-transactions (NEP-366) et la signature de messages (NEP-413).
