Библиотека TypeScript для взаимодействия с блокчейном протокола NEAR. Используйте этот навык при написании кода, который взаимодействует с протоколом NEAR, включая просмотр данных контракта, вызов методов контракта, отправку токенов NEAR, создание транзакций, создание типобезопасных оболочек контрактов, интеграцию кошельков (Wallet Selector, HOT Connect), перехватчиков и поставщиков React (@near-kit/react), управление ключами, тестирование в песочнице, метатранзакции (NEP-366) и подписание сообщений (NEP-413).
