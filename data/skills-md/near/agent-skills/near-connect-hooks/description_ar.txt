خطافات تفاعلية لتكامل محفظة NEAR Protocol باستخدام @hot-labs/near-connect. يُستخدم عند إنشاء تطبيقات React/Next.js التي تحتاج إلى اتصال NEAR بالمحفظة، أو مكالمات العقود الذكية (طرق العرض والتغيير)، أو عمليات نقل الرمز المميز، أو إدارة مفاتيح الوصول، أو توقيع الرسائل NEP-413. مشغلات للاستعلامات حول خطافات المحفظة NEAR، أو إعداد NearProvider، أو خطاف useNearWallet، أو تكامل NEAR dApp React.
