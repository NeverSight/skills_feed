موصل محفظة خفيف الوزن بدون اعتماد لـ NEAR blockchain باستخدام عزل آمن لصندوق الحماية. يُستخدم عند العمل مع تكامل محفظة NEAR، أو دعم محافظ متعددة (HOT، أو Meteor، أو Intear، أو MyNearWallet، أو Nightly، أو NEAR Mobile، أو Unity، أو OKX، أو WalletConnect)، أو توقيع المعاملات، أو توقيع الرسائل، أو تنفيذ ميزات اتصال المحفظة في NEAR dApps. يُستخدم أيضًا عند إنشاء عمليات تكامل مخصصة للمحفظة أو تصحيح البرامج النصية لمنفذ المحفظة.
