Near-api-js v7 を使用した開発ガイド (NEAR ブロックチェーン インタラクション用の JavaScript/TypeScript ライブラリ)。  次の場合に使用します: (1) NEAR ブロックチェーンと対話するアプリの構築、(2) トランザクションの作成/署名、(3) スマート コントラクトの呼び出し、(4) アカウントとキーの管理、(5) NEAR RPC API の操作、(6) NEAR での FT/NFT トークンの処理、(7) NEAR 暗号化操作の使用 (KeyPair、署名)、(8) NEAR ユニット間の変換 (yocto、ガス)、(9) リレーラーとのガスレス/メタ トランザクション、 (10) ウォレットの統合とセッション管理、(11) 認証のための NEP-413 メッセージ署名、(12) FT 契約のストレージ デポジット管理。 NEAR ブロックチェーン開発タスクをトリガーします。
