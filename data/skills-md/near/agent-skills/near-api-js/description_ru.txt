Руководство по разработке с использованием почти api-js v7 — библиотеки JavaScript/TypeScript для взаимодействия с блокчейном NEAR.  Используйте при: (1) создании приложений, которые взаимодействуют с блокчейном NEAR, (2) создании/подписании транзакций, (3) вызове смарт-контрактов, (4) управлении учетными записями и ключами, (5) работе с NEAR RPC API, (6) обработке токенов FT/NFT на NEAR, (7) использовании криптографических операций NEAR (KeyPair, подписывание), (8) преобразовании между единицами NEAR (yocto, gas), (9) безгазовых/метатранзакциях с ретрансляторами, (10) интеграция кошелька и управление сеансами, (11) подписание сообщений NEP-413 для аутентификации, (12) управление депозитами для контрактов FT. Запускается при выполнении любых задач разработки блокчейна NEAR.
