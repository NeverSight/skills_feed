Guide de développement avec Near-api-js v7 - la bibliothèque JavaScript/TypeScript pour l'interaction NEAR blockchain.  À utiliser lorsque : (1) créer des applications qui interagissent avec la blockchain NEAR, (2) créer/signer des transactions, (3) appeler des contrats intelligents, (4) gérer des comptes et des clés, (5) travailler avec l'API NEAR RPC, (6) gérer des jetons FT/NFT sur NEAR, (7) utiliser des opérations cryptographiques NEAR (KeyPair, signature), (8) convertir entre des unités NEAR (yocto, gas), (9) transactions sans gaz/méta avec des relais, (10) intégration de portefeuille et gestion de session, (11) signature de message NEP-413 pour l'authentification, (12) gestion de dépôt de stockage pour les contrats FT. Se déclenche sur toutes les tâches de développement de blockchain NEAR.
