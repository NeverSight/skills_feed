Leitfaden für die Entwicklung mit near-api-js v7 – der JavaScript/TypeScript-Bibliothek für die NEAR-Blockchain-Interaktion.  Verwendung bei: (1) Erstellen von Apps, die mit der NEAR-Blockchain interagieren, (2) Erstellen/Signieren von Transaktionen, (3) Aufrufen von Smart Contracts, (4) Verwalten von Konten und Schlüsseln, (5) Arbeiten mit der NEAR RPC-API, (6) Umgang mit FT/NFT-Tokens auf NEAR, (7) Verwenden von kryptografischen NEAR-Operationen (KeyPair, Signieren), (8) Konvertieren zwischen NEAR-Einheiten (Yocto, Gas), (9) gaslose/Meta-Transaktionen mit Relayern, (10) Wallet Integration und Sitzungsverwaltung, (11) NEP-413-Nachrichtensignierung zur Authentifizierung, (12) Speicherkautionsverwaltung für FT-Verträge. Wird bei allen NEAR-Blockchain-Entwicklungsaufgaben ausgelöst.
