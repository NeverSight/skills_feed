Навыки написания приложений TypeScript SDK. Используйте при создании браузерных приложений JavaScript/TypeScript, мобильных приложений React Native или серверных серверных частей Node.js/Deno с помощью Appwrite. Охватывает аутентификацию на стороне клиента (электронная почта, OAuth, анонимно), запросы к базе данных, загрузку файлов, подписки в реальном времени и администрирование на стороне сервера с помощью ключей API для управления пользователями, администрирования базы данных, хранения и функций.
