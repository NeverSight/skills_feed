Инструмент CLI для создания мультимедиа с использованием искусственного интеллекта с использованием Google Imagen 4, Veo 3.1 и Gemini TTS. Используйте, когда пользователь хочет (1) генерировать изображения из текстовых подсказок, (2) редактировать существующие изображения с помощью искусственного интеллекта, (3) объяснять содержимое изображения, (4) создавать видео из текста или изображений, (5) создавать повествование/голосовой звук с настройками персонажей. Запускается по таким запросам, как «создать изображение...», «создать видео...», «создать голос, который говорит...», «отредактировать это изображение, чтобы...», «описать это изображение».
