Leitet systematisches verifizierungsbasiertes Debugging für JavaScript/TypeScript (Node.js, Bun, Browser). Verwenden Sie dies, wenn: (1) der Benutzer unerwartetes Verhalten, Fehler oder Bugs meldet, (2) Datenfluss- oder Zustandsänderungen nachverfolgt werden müssen, ohne Grundursachen anzunehmen, (3) das Problem die genaue Identifizierung der Stellen erfordert, an denen der Code vom erwarteten Verhalten abweicht.
