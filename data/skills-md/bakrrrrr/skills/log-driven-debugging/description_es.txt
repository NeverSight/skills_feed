Guía la depuración sistemática basada en verificación para JavaScript/TypeScript (Node.js, Bun, navegador). Úselo cuando: (1) El usuario informa comportamientos inesperados, errores o fallas, (2) Necesita rastrear el flujo de datos o los cambios de estado sin asumir las causas fundamentales, (3) El problema requiere identificar exactamente dónde el código se desvía del comportamiento esperado.
