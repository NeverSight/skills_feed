Guide le débogage systématique basé sur la vérification pour JavaScript/TypeScript (Node.js, Bun, navigateur). À utiliser lorsque : (1) l'utilisateur signale un comportement inattendu, des erreurs ou des bogues, (2) doit suivre le flux de données ou les changements d'état sans supposer les causes profondes, (3) le problème nécessite d'identifier exactement où le code s'écarte du comportement attendu.
