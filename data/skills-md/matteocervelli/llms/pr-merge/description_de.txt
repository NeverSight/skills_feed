Führen Sie GitHub-Pull-Anfragen mithilfe einer umfassenden Validierung vor dem Zusammenführen sicher zusammen. Verwenden Sie diese Fähigkeit, wenn der Benutzer darum bittet, eine PR zusammenzuführen, einen PR-Workflow abzuschließen oder zu prüfen, ob eine PR zur Zusammenführung bereit ist. Validiert CI/CD-Status, Überprüfungen und Konflikte, bevor es mit der entsprechenden Strategie zusammengeführt wird.
