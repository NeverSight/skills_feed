Fusionnez en toute sécurité les requêtes pull GitHub à l’aide d’une validation complète avant fusion. Utilisez cette compétence lorsque l'utilisateur demande à fusionner une PR, à terminer un flux de travail de PR ou à vérifier si une PR est prête à fusionner. Valide le statut CI/CD, les révisions et les conflits avant de fusionner avec la stratégie appropriée.
