Guide pour créer de nouvelles compétences à partir de zéro. À utiliser lorsque l'utilisateur souhaite créer, écrire ou créer une nouvelle compétence, ou pose des questions sur la structure des compétences, les meilleures pratiques ou le format SKILL.md. Ne PAS utiliser pour installer des compétences existantes (utilisez le gestionnaire de compétences) ou pour synchroniser les compétences avec les IDE (utilisez la synchronisation des compétences).
