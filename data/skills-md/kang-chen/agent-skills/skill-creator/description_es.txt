Guía para crear nuevas habilidades desde cero. Úselo cuando el usuario quiera crear, escribir o crear una nueva habilidad, o pregunte sobre la estructura de las habilidades, las mejores prácticas o el formato SKILL.md. NO lo use para instalar habilidades existentes (use el administrador de habilidades) o sincronizar habilidades con IDE (use sincronización de habilidades).
