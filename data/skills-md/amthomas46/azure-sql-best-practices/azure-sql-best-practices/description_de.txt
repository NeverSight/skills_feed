Best Practices für Azure SQL-Datenbank zur Optimierung von T-SQL-Code, Datenbankkonfiguration, Indizierungsstrategien und Anwendungsmustern. Basierend auf der Microsoft SQL Assessment API, den SSDT-Codeanalyseregeln, den Leistungsrichtlinien für die Azure SQL-Datenbank und den offiziellen Best Practices von Microsoft. Verwenden Sie diese Fähigkeit, wenn Sie Code schreiben, überprüfen oder umgestalten, der mit Azure SQL-Datenbank interagiert.
