Habilidades de mejores prácticas de Azure SQL Database para optimizar el código T-SQL, la configuración de la base de datos, las estrategias de indexación y los patrones de aplicaciones. Basado en la API de evaluación de Microsoft SQL, las reglas de análisis de código SSDT, la guía de rendimiento de la base de datos SQL de Azure y las mejores prácticas oficiales de Microsoft. Utilice esta habilidad al escribir, revisar o refactorizar código que interactúe con Azure SQL Database.
