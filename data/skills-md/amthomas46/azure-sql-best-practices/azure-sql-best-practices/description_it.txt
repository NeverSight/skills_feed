Competenza sulle best practice per il database SQL di Azure per l'ottimizzazione del codice T-SQL, della configurazione del database, delle strategie di indicizzazione e dei modelli di applicazione. Basato sull'API Microsoft SQL Assessment, sulle regole di analisi del codice SSDT, sulle indicazioni sulle prestazioni del database SQL di Azure e sulle best practice ufficiali di Microsoft. Utilizzare questa competenza durante la scrittura, la revisione o il refactoring del codice che interagisce con il database SQL di Azure.
