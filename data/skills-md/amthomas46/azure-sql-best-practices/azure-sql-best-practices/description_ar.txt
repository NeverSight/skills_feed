مهارة أفضل ممارسات قاعدة بيانات Azure SQL لتحسين تعليمات T-SQL البرمجية وتكوين قاعدة البيانات واستراتيجيات الفهرسة وأنماط التطبيق. استنادًا إلى واجهة برمجة تطبيقات Microsoft SQL Assessment API وقواعد تحليل كود SSDT وتوجيهات أداء قاعدة بيانات Azure SQL وأفضل ممارسات Microsoft الرسمية. استخدم هذه المهارة عند كتابة أو مراجعة أو إعادة بناء التعليمات البرمجية التي تتفاعل مع قاعدة بيانات Azure SQL.
