Utilise correctement le package uv Python et le gestionnaire de projet pour les dépendances, les venvs et les scripts. À utiliser lors de la création ou de la modification de projets Python, de l'ajout de dépendances, de l'exécution de scripts avec des dépôts en ligne, de la gestion d'environnements virtuels, de l'épinglage de versions Python, de l'exécution d'outils CLI à partir de PyPI, de la configuration de l'interpréteur IDE Python ou de l'utilisation d'uv dans CI (par exemple GitHub Actions) ou de conteneurs Docker. À utiliser lorsque l'utilisateur mentionne uv, pyproject.toml, uv.lock, uv run, uv add, uv sync, .venv, l'interpréteur Python, poésie, pipenv, conda, CI, Docker, GitHub Actions ou demande d'utiliser uv au lieu de pip ou de poésie.
