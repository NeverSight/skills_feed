Utilizza correttamente il pacchetto uv Python e il project manager per dipendenze, venv e script. Da utilizzare durante la creazione o la modifica di progetti Python, l'aggiunta di dipendenze, l'esecuzione di script con dipendenze in linea, la gestione di ambienti virtuali, il pinning di versioni Python, l'esecuzione di strumenti CLI da PyPI, l'impostazione dell'interprete IDE Python o l'utilizzo di uv in CI (ad esempio GitHub Actions) o contenitori Docker. Da utilizzare quando l'utente menziona uv, pyproject.toml, uv.lock, uv run, uv add, uv sync, .venv, interprete Python, poetry, pipenv, conda, CI, Docker, GitHub Actions o chiede di utilizzare uv invece di pip o poetry.
