Правильно использует пакет uv Python и менеджер проектов для зависимостей, venvs и скриптов. Используйте при создании или изменении проектов Python, добавлении зависимостей, запуске сценариев со встроенными параметрами, управлении виртуальными средами, закреплении версий Python, запуске инструментов CLI из PyPI, настройке интерпретатора IDE Python или использовании uv в CI (например, GitHub Actions) или контейнерах Docker. Используйте, когда пользователь упоминает uv, pyproject.toml, uv.lock, uv run, uv add, uv sync, .venv, интерпретатор Python, поэзию, Pipenv, conda, CI, Docker, GitHub Actions или просит использовать uv вместо pip или поэзии.
