يستخدم حزمة uv Python ومدير المشروع بشكل صحيح للتبعيات وvenvs والبرامج النصية. يُستخدم عند إنشاء مشاريع Python أو تعديلها، أو إضافة التبعيات، أو تشغيل البرامج النصية باستخدام deps المضمنة، أو إدارة البيئات الافتراضية، أو تثبيت إصدارات Python، أو تشغيل أدوات CLI من PyPI، أو تعيين مترجم IDE Python، أو استخدام الأشعة فوق البنفسجية في CI (مثل GitHub Actions) أو حاويات Docker. يُستخدم عندما يذكر المستخدم uv، أو pyproject.toml، أو uv.lock، أو uv run، أو uv add، أو uv sync، أو .venv، أو Python Interpreter، أو Poetry، أو Pipenv، أو conda، أو CI، أو Docker، أو GitHub Actions، أو يطلب استخدام uv بدلاً من النقطة أو الشعر.
