Crea e gestisce pipeline dlt (strumento di caricamento dati) da API, database e altre fonti. Utilizzare quando l'utente desidera creare o eseguire il debug di pipeline; utilizzare fonti verificate (ad esempio Salesforce, GitHub, Stripe) o API REST dichiarative o Python personalizzato; configurare le destinazioni (ad esempio DuckDB, BigQuery, Snowflake); implementare il caricamento incrementale; o modificare la configurazione e i segreti del file .dlt. Da utilizzare quando l'utente menziona l'inserimento di dati, la pipeline dlt, dlt init, rest_api_source, carico incrementale o dashboard della pipeline.
