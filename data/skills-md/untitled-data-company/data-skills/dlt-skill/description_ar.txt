يقوم بإنشاء وصيانة خطوط أنابيب dlt (أداة تحميل البيانات) من واجهات برمجة التطبيقات وقواعد البيانات والمصادر الأخرى. يُستخدم عندما يريد المستخدم إنشاء خطوط الأنابيب أو تصحيح أخطائها؛ استخدام مصادر تم التحقق منها (مثل Salesforce أو GitHub أو Stripe) أو REST API التعريفية أو Python المخصصة؛ تكوين الوجهات (مثل DuckDB، وBigQuery، وSnowflake)؛ تنفيذ التحميل المتزايد. أو تحرير التكوين والأسرار .dlt. يُستخدم عندما يذكر المستخدم استيعاب البيانات، أو خط أنابيب dlt، أو dlt init، أو Rest_api_source، أو التحميل المتزايد، أو لوحة معلومات خط الأنابيب.
