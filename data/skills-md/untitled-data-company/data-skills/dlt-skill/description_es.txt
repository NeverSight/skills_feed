Crea y mantiene canalizaciones dlt (herramienta de carga de datos) a partir de API, bases de datos y otras fuentes. Úselo cuando el usuario quiera construir o depurar tuberías; utilice fuentes verificadas (por ejemplo, Salesforce, GitHub, Stripe) o API REST declarativa o Python personalizado; configurar destinos (por ejemplo, DuckDB, BigQuery, Snowflake); implementar carga incremental; o editar la configuración y los secretos de .dlt. Úselo cuando el usuario mencione la ingesta de datos, la canalización dlt, el inicio de dlt, rest_api_source, la carga incremental o el panel de canalización.
