API、データベース、その他のソースから dlt (データ ロード ツール) パイプラインを作成および維持します。ユーザーがパイプラインを構築またはデバッグしたい場合に使用します。検証済みのソース (Salesforce、GitHub、Stripe など)、宣言型 REST API、またはカスタム Python を使用します。宛先を構成します (DuckDB、BigQuery、Snowflake など)。インクリメンタルロードを実装します。または、.dlt 設定とシークレットを編集します。ユーザーがデータ インジェスト、dlt パイプライン、dlt init、rest_api_source、増分ロード、またはパイプライン ダッシュボードについて言及する場合に使用します。
