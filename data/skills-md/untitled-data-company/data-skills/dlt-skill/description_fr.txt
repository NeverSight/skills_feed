Crée et gère des pipelines dlt (outil de chargement de données) à partir d'API, de bases de données et d'autres sources. À utiliser lorsque l'utilisateur souhaite créer ou déboguer des pipelines ; utiliser des sources vérifiées (par exemple Salesforce, GitHub, Stripe) ou une API REST déclarative ou Python personnalisé ; configurer les destinations (par exemple DuckDB, BigQuery, Snowflake) ; mettre en œuvre un chargement incrémentiel ; ou modifiez la configuration et les secrets .dlt. À utiliser lorsque l'utilisateur mentionne l'ingestion de données, le pipeline dlt, dlt init, rest_api_source, la charge incrémentielle ou le tableau de bord du pipeline.
