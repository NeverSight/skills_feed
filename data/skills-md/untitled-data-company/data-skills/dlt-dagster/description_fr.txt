Exécute les pipelines dlt dans Dagster en tant qu'actifs définis par logiciel (Composant ou Pythonic @dlt_assets) ou en tant qu'actif Dagster standard unique. À utiliser lors de l'orchestration de dlt avec Dagster ; échafaudageloads.py/defs.yaml ; emplois/horaires ; secrets/environnement ; incrémentiel/remplissage via apply_hints ; parallélisation (un actif par ressource) ; Déploiement Dagster Cloud ; ou calcul externe (ECS, Fargate - voir intégrations dagster). Déclencheurs : dagster-dlt, dlt sur Dagster, déployer dlt avec Dagster, actif Dagster standard, calcul externe.
