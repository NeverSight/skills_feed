Limpie los cambios de código generados por IA revisando las diferencias con el principal y eliminando artefactos estilísticos o estructurales (comentarios excesivos, comprobaciones defensivas innecesarias, ruido de prueba/captura, cualquier conversión, estilo inconsistente). Úselo cuando se le solicite "eliminar el código de IA", "humanizar" los cambios o alinear las ediciones de una rama con las convenciones de base de código existentes.
