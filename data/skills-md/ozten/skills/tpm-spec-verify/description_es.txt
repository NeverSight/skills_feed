Enriquecer una Fase Sepc/PRD con Requisitos de Calidad (Q-nnn) y Criterios de Aceptación (AC-nnnn). Úselo cuando el usuario desee agregar una perspectiva de control de calidad, definir criterios de prueba, identificar requisitos no funcionales, agregar pasos de verificación o preparar una fase PRD para la planificación de pruebas.
