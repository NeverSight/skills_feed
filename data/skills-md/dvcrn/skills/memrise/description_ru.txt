Используйте интерфейс командной строки Memrise для курсов сообщества, включая настройку аутентификации, поиск курса/уровня/слова, поиск в пуле и добавление элементов. Используйте, когда в запросах упоминаются команды memcli или пакет npm memrise-cli (который устанавливает двоичный файл memcli), особенно для построения команд, флагов устранения неполадок и удобного для сценариев вывода JSON.
