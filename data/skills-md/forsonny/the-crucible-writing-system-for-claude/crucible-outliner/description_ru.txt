Генератор поглавных набросков для историй, запланированных с помощью Crucible Structure. Используйте, когда пользователь завершил документы планирования Crucible (тезисы, карты нитей, точки штамповки и т. д.) и хочет создать подробные схемы глав. Запускается по таким запросам, как «составить план моей книги», «создать план главы», «составить план книги 1», «превратить мой план Crucible в главы» или когда у пользователя есть документы по планированию Crucible и он хочет начать писать. Работает для отдельных книг или серий из нескольких книг.
