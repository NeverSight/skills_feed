Интерактивная система планирования для эпических фэнтезийных романов, использующая структуру Crucible — 36-битную повествовательную структуру с тремя переплетенными сюжетными линиями (квест, огонь, созвездие), пятью форс-пойнтами и двигателем милосердия. Используйте, когда пользователь хочет спланировать фэнтезийный роман, предоставляет предпосылку/синопсис истории, просит «спланировать мою фэнтезийную книгу», хочет создать документы планирования для эпического фэнтези или упоминает структуру Горнила. Помогает пользователям задавать вопросы с несколькими вариантами ответов для создания 7 категорий документов планирования (всего 14 файлов) на основе простой предпосылки.
