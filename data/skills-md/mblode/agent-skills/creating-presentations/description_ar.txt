ينشئ عروضًا تقديمية جريئة وبسيطة ومظلمة أولاً باستخدام أقواس سردية منظمة ونسخة شرائح قوية وتصميم مرئي عالي التباين وملاحظات المتحدث الحوارية. يتكيف مع المحادثات المباشرة أو العروض التقديمية غير المتزامنة للمستثمرين. يُستخدم عند إنشاء عرض تقديمي، أو إنشاء مجموعة شرائح، أو كتابة شرائح، أو طرح سؤال "مخطط عرض تقديمي حول..."، أو "كتابة شرائح لـ..."، أو "تصميم مجموعة شرائح لـ..."، أو إنشاء مجموعة عروض تقديمية للمستثمرين.
