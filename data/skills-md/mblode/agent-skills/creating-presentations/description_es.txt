Crea presentaciones atrevidas, minimalistas y oscuras con arcos narrativos estructurados, textos de diapositivas impactantes, diseño visual de alto contraste y notas conversacionales del orador. Se adapta para charlas en vivo o presentaciones asíncronas para inversores. Úselo al crear una presentación, estructurar una presentación, escribir diapositivas, preguntar "delinear una presentación sobre...", "escribir diapositivas para...", "diseñar una presentación para..." o crear una presentación para inversionistas.
