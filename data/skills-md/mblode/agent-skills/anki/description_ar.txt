يُنشئ بطاقات Anki التعليمية بتنسيق TSV من مادة المصدر باستخدام أفضل ممارسات التكرار المتباعد. يدعم أنواع البطاقات الأساسية والمعكوسة (ثنائية الاتجاه) وClose بكمية وصعوبة قابلة للتكوين. استخدمها عند إنشاء البطاقات التعليمية، أو إنشاء بطاقات Anki، أو إنشاء بطاقات دراسية، أو تحويل الملاحظات إلى بطاقات تعليمية، أو تحويل النص إلى بطاقات تعليمية، أو إنشاء مجموعة تكرار متباعدة.
