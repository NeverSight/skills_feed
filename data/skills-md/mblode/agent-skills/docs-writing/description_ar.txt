يكتب ويراجع الوثائق الفنية باستخدام Diataxis والوضوح على غرار الشريط والقواعد الثمانية. 52 قاعدة عبر 9 فئات تغطي الصوت والبنية والوضوح وأمثلة التعليمات البرمجية والتنسيق والملاحة وقابلية المسح الضوئي ونظافة المحتوى والمراجعة. يُستخدم عند كتابة المستندات، أو إنشاء ملفات README، أو توثيق واجهات برمجة التطبيقات، أو كتابة البرامج التعليمية، أو إنشاء موقع مستندات، أو تدقيق جودة الوثائق.
