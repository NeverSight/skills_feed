Supporta un progetto CLI TypeScript pronto per la produzione con ESM, tsdown, vitest, biome, changeset, azioni GitHub e una definizione di competenza dell'agente. Da utilizzare durante la creazione di un nuovo strumento CLI, l'avvio di un progetto TypeScript, l'impalcatura di una CLI di un nodo, l'avvio di un nuovo pacchetto npm o la richiesta di "impalcatura di un progetto CLI".
