Élabore un projet CLI TypeScript prêt pour la production avec ESM, tsdown, vitest, biome, changesets, GitHub Actions et une définition de compétence d'agent. À utiliser lors de la création d'un nouvel outil CLI, du démarrage d'un projet TypeScript, de l'échafaudage d'un nœud CLI, du démarrage d'un nouveau package npm ou de la demande de « échafaudage d'un projet CLI ».
