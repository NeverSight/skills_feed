Desarrolla un proyecto CLI de TypeScript listo para producción con ESM, tsdown, vitest, biome, conjuntos de cambios, acciones de GitHub y una definición de habilidad del agente. Úselo al crear una nueva herramienta CLI, iniciar un proyecto de TypeScript, aplicar soporte técnico a una CLI de nodo, iniciar un nuevo paquete npm o preguntar "ampliar un proyecto CLI".
