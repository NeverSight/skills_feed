Guida la creazione di competenze per agenti di best practice seguendo le specifiche del formato aperto. Copre il frontmatter, la struttura delle directory, la divulgazione progressiva, i file di riferimento, le cartelle delle regole e la convalida. Da utilizzare quando si crea una nuova competenza, si crea SKILL.md, si imposta una competenza di controllo basata su regole, si struttura un pacchetto di competenze o si chiede "come scrivere una competenza".
