Направляет создание передовых навыков агентов в соответствии со спецификацией открытого формата. Охватывает вступление, структуру каталогов, постепенное раскрытие, справочные файлы, папки правил и проверку. Используйте при создании нового навыка, создании SKILL.md, настройке навыка аудита на основе правил, структурировании набора навыков или при запросе «как написать навык».
