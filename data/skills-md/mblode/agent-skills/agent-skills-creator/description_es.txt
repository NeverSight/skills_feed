Guía la creación de habilidades de agentes de mejores prácticas siguiendo la especificación de formato abierto. Cubre temas preliminares, estructura de directorios, divulgación progresiva, archivos de referencia, carpetas de reglas y validación. Úselo al crear una nueva habilidad, crear SKILL.md, configurar una habilidad de auditoría basada en reglas, estructurar un paquete de habilidades o preguntar "cómo escribir una habilidad".
