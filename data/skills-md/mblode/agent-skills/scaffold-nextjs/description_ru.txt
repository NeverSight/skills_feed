Создает готовое к работе турборепозиторий Next.js с использованием TypeScript, Tailwind CSS, Shadcn CLI, компонентов пользовательского интерфейса Blode с сайта ui.blode.co, развертывания blode-icons-react, Biome, Ultracite и Vercel. Используйте при создании нового приложения Next.js, загрузке турборепозитория, создании веб-проекта, запуске нового веб-сайта или при запросе «создать проект Next.js».
