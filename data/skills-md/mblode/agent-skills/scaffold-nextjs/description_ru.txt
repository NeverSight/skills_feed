Создает готовое к работе турборепозиторий Next.js с TypeScript, Tailwind CSS, Shadcn CLI, компонентами пользовательского интерфейса Blode с сайта ui.blode.co, blode-icons-react, развертыванием Biome, Ultracite и Vercel. Используйте при создании нового приложения Next.js, загрузке турборепозитория, создании веб-проекта, запуске нового веб-сайта или при запросе «создать проект Next.js».
