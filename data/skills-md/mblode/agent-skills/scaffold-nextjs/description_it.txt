Supporta un turborepo Next.js pronto per la produzione con TypeScript, Tailwind CSS, shadcn CLI, componenti dell'interfaccia utente Blode da ui.blode.co, blode-icons-react, Biome, Ultracite e distribuzione Vercel. Da utilizzare durante la creazione di una nuova app Next.js, il bootstrap di un turborepo, lo scaffolding di un progetto Web, l'avvio di un nuovo sito Web o la richiesta di "crea un progetto Next.js".
