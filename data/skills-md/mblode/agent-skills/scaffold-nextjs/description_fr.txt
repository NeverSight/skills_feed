Élabore un turborepo Next.js prêt pour la production avec TypeScript, Tailwind CSS, shadcn CLI, les composants Blode UI de ui.blode.co, blode-icons-react, Biome, Ultracite et Vercel. À utiliser lors de la création d'une nouvelle application Next.js, du démarrage d'un turborepo, de l'échafaudage d'un projet Web, du démarrage d'un nouveau site Web ou de la demande de « créer un projet Next.js ».
