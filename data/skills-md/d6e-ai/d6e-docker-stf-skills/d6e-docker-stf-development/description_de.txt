Erstellt benutzerdefinierte Docker-basierte State Transition Functions (STFs) für D6E-Plattform-Workflows. Verwenden Sie es, wenn Sie eine containerisierte Geschäftslogik für D6E erstellen, Datenverarbeitungsschritte implementieren oder Workflow-Funktionen erstellen, die Datenbankzugriff benötigen. Verarbeitet JSON-Eingabe/-Ausgabe, SQL-API-Integration und mehrsprachige Implementierungen (Python, Node.js, Go).
