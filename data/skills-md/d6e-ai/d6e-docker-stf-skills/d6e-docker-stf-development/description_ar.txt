إنشاء وظائف نقل الحالة (STFs) المستندة إلى Docker المخصصة لسير عمل النظام الأساسي D6E. يُستخدم عند إنشاء منطق الأعمال المحتوي على حاوية لـ D6E، أو تنفيذ خطوات معالجة البيانات، أو إنشاء وظائف سير العمل التي تحتاج إلى الوصول إلى قاعدة البيانات. يتعامل مع إدخال/إخراج JSON، وتكامل SQL API، والتطبيقات متعددة اللغات (Python، Node.js، Go).
