Создает пользовательские функции перехода состояний (STF) на основе Docker для рабочих процессов платформы D6E. Используйте при построении контейнерной бизнес-логики для D6E, реализации этапов обработки данных или создании функций рабочего процесса, которым требуется доступ к базе данных. Обрабатывает ввод/вывод JSON, интеграцию SQL API и многоязычные реализации (Python, Node.js, Go).
