Crea funzioni di transizione di stato (STF) personalizzate basate su Docker per i flussi di lavoro della piattaforma D6E. Da utilizzare durante la creazione di logica aziendale containerizzata per D6E, l'implementazione di passaggi di elaborazione dei dati o la creazione di funzioni del flusso di lavoro che richiedono l'accesso al database. Gestisce input/output JSON, integrazione API SQL e implementazioni multilingue (Python, Node.js, Go).
