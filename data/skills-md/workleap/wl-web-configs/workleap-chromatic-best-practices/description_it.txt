Controlla e aggiorna i repository per seguire le migliori pratiche cromatiche di Workleap per il controllo dei costi degli snapshot e l'ottimizzazione della CI.

Usa questa abilità quando:
(1) Controllo di un archivio per la conformità alle migliori pratiche cromatiche
(2) Implementazione delle ottimizzazioni dei costi cromatici in un progetto
(3) Correzione dei modelli di disabilitazione di TurboSnap nel codice
(4) Configurazione di chromatic.config.json con l'opzione untraced
(5) Aggiornamento dei flussi di lavoro CI per l'esecuzione cromatica condizionale
(6) Refactoring delle importazioni di file barile nei file di anteprima di Storybook
(7) Revisione dei PR per l'impatto dei costi cromatici
(8) Impostazione di Chromatic in un nuovo progetto Turborepo
(9) Verifica dell'utilizzo cromatico locale che dovrebbe essere rimosso
