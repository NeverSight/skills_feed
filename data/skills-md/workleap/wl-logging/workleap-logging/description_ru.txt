Руководство по библиотеке журналирования Workleap (@workleap/logging), которая обеспечивает структурированное, составное ведение журналов для интерфейсных приложений TypeScript.

Используйте этот навык, когда:
(1) Настройка входа в интерфейсное приложение Workleap.
(2) Создание или настройка средств ведения журнала (BrowserConsoleLogger, CompositeLogger)
(3) Понимание уровней журнала (отладка, информация, предупреждение, ошибка, критический)
(4) Создание сложных записей журнала с цепочками сегментов (withText, withObject, withError)
(5) Использование областей журналирования для группировки связанных записей журнала.
(6) Стилизация вывода журнала в консоли браузера
(7) Создание нескольких регистраторов для отправки журналов в разные места назначения.
(8) Фильтрация журналов по уровню серьезности.
(9) Интеграция ведения журнала с LogRocket или другими инструментами телеметрии.
(10) Проверка изменений, связанных с журналированием, в запросах на включение
(11) Вопросы о передовых методах ведения журналов, характерных для wl-logging.
