Guide du système de construction de Turborepo monorepo. Déclencheurs sur : turbo.json, pipelines de tâches,
dependOn, mise en cache, cache distant, CLI "turbo", --filter, --affected, optimisation CI, environnement
variables, packages internes, structure monorepo/meilleures pratiques et limites.

À utiliser lorsque l'utilisateur : configure les tâches/workflows/pipelines, crée des packages, configure
monorepo, partage le code entre les applications, exécute les packages modifiés/affectés, débogue le cache,
ou possède des répertoires d'applications/packages.
