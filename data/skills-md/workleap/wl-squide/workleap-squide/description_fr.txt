Squide est un shell d'application modulaire React pour les applications Web Workleap. Utilisez cette compétence lorsque :
(1) Création ou modification d'applications ou de modules hôtes Squide
(2) Enregistrement d'itinéraires, d'éléments de navigation ou de gestionnaires de requêtes MSW
(3) Travailler avec FireflyRuntime, initializeFirefly ou AppRouter
(4) Mise en place des intégrations : TanStack Query, i18next, LaunchDarkly, Honeycomb, MSW, Storybook
(5) Mise en œuvre d'inscriptions différées ou d'éléments de navigation conditionnels
(6) Récupération de données mondiales publiques/protégées avec usePublicDataQueries/useProtectedDataQueries
(7) Utilisation des hooks Squide : useNavigationItems, useRenderedNavigationItems, useIsBootstrapping, useLogger, useEventBusListener, useEventBusDispatcher, useEnvironmentVariable, useFeatureFlag
(8) Implémentation de limites d'erreur dans les applications modulaires
(9) Questions sur les modèles d'architecture modulaire dans les applications React
