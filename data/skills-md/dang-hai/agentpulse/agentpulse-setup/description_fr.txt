Configurez AgentPulse dans un projet React ou Electron. À utiliser lorsque l'utilisateur souhaite ajouter AgentPulse,
rendre leur application contrôlable par MCP ou intégrer avec Claude Code/Claude Desktop.
Il s'agit d'une compétence de configuration UNIQUE. Pour obtenir de l'aide sur l'écriture des hooks useExpose, utilisez plutôt la compétence "agentpulse".
