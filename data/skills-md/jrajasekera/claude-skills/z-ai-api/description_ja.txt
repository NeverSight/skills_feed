GLM モデルを使用してアプリケーションを構築するための Z.ai API 統合。 Z.ai/ZhipuAI API を使用して次の目的で作業する場合に使用します: (1) GLM-4.7/4.6/4.5 モデルによるチャット完了、(2) GLM-4.6V によるビジョン/マルチモーダル タスク、(3) GLM-Image または CogView-4 による画像生成、(4) CogVideoX-3 または Vidu モデルによるビデオ生成、(5) GLM-ASR-2512 による音声転写、(6) 関数呼び出しおよびツールの使用、(7) Web 検索の統合、(8) 翻訳、スライド/ポスター生成エージェント。トリガー: Z.ai、ZhipuAI、GLM、BigModel、Zhipu、CogVideoX、CogView、Vidu。
