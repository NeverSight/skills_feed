Intégration de l'API Z.ai pour créer des applications avec des modèles GLM. À utiliser lorsque vous travaillez avec les API Z.ai/ZhipuAI pour : (1) les complétions de chat avec les modèles GLM-4.7/4.6/4.5, (2) les tâches de vision/multimodales avec GLM-4.6V, (3) la génération d'images avec GLM-Image ou CogView-4, (4) la génération de vidéo avec les modèles CogVideoX-3 ou Vidu, (5) la transcription audio avec GLM-ASR-2512, (6) l'appel de fonction et l'utilisation d'outils, (7) Intégration de recherche Web, (8) Traduction, agents de génération de diapositives/affiches. Déclencheurs : Z.ai, ZhipuAI, GLM, BigModel, Zhipu, CogVideoX, CogView, Vidu.
