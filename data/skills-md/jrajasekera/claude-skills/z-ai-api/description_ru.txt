Интеграция API Z.ai для создания приложений с использованием моделей GLM. Используйте при работе с API-интерфейсами Z.ai/ZhipuAI для: (1) Завершения чата с моделями GLM-4.7/4.6/4.5, (2) Vision/мультимодальных задач с GLM-4.6V, (3) Генерации изображений с помощью GLM-Image или CogView-4, (4) Генерации видео с моделями CogVideoX-3 или Vidu, (5) Транскрипции аудио с GLM-ASR-2512, (6) Вызова функций и использование инструментов, (7) интеграция веб-поиска, (8) агенты перевода, создания слайдов/плакатов. Триггеры: Z.ai, ZhipuAI, GLM, BigModel, Zhipu, CogVideoX, CogView, Vidu.
