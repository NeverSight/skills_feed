تكامل Z.ai API لبناء التطبيقات باستخدام نماذج GLM. يُستخدم عند العمل مع واجهات برمجة التطبيقات Z.ai/ZhipuAI من أجل: (1) عمليات إكمال الدردشة باستخدام نماذج GLM-4.7/4.6/4.5، (2) مهام الرؤية/متعددة الوسائط باستخدام GLM-4.6V، (3) إنشاء الصور باستخدام GLM-Image أو CogView-4، (4) إنشاء الفيديو باستخدام نماذج CogVideoX-3 أو Vidu، (5) نسخ الصوت باستخدام GLM-ASR-2512، (6) استدعاء الوظائف واستخدام الأدوات، (7) تكامل بحث الويب، (8) الترجمة، وكلاء إنشاء الشرائح/الملصقات. المشغلات: Z.ai، ZhipuAI، GLM، BigModel، Zhipu، CogVideoX، CogView، Vidu.
