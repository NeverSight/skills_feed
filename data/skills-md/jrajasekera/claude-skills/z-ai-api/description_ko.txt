GLM 모델을 사용하여 애플리케이션을 구축하기 위한 Z.ai API 통합입니다. Z.ai/ZhipuAI API 작업 시 사용: (1) GLM-4.7/4.6/4.5 모델을 사용한 채팅 완료, (2) GLM-4.6V를 사용한 비전/멀티모달 작업, (3) GLM-Image 또는 CogView-4를 사용한 이미지 생성, (4) CogVideoX-3 또는 Vidu 모델을 사용한 비디오 생성, (5) GLM-ASR-2512를 사용한 오디오 전사, (6) 함수 호출 및 도구 사용, (7) 웹 검색 통합, (8) 번역, 슬라이드/포스터 생성 에이전트. 트리거: Z.ai, ZhipuAI, GLM, BigModel, Zhipu, CogVideoX, CogView, Vidu.
