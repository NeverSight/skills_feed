Integración API Z.ai para crear aplicaciones con modelos GLM. Úselo cuando trabaje con las API Z.ai/ZhipuAI para: (1) Finalización de chat con modelos GLM-4.7/4.6/4.5, (2) Tareas de visión/multimodales con GLM-4.6V, (3) Generación de imágenes con GLM-Image o CogView-4, (4) Generación de video con modelos CogVideoX-3 o Vidu, (5) Transcripción de audio con GLM-ASR-2512, (6) Llamada de funciones y uso de herramientas. (7) Integración de búsqueda web, (8) Agentes de traducción y generación de diapositivas/pósteres. Activadores: Z.ai, ZhipuAI, GLM, BigModel, Zhipu, CogVideoX, CogView, Vidu.
