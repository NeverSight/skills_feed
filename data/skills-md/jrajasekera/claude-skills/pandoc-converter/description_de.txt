Konvertieren Sie Dokumente zwischen Formaten mit Pandoc. Verwenden Sie diese Option, wenn der Benutzer Dateien zwischen Formaten wie Markdown, docx, html, pdf, latex, epub, rtf, csv, xlsx oder pptx konvertieren möchte. Wird bei Anfragen wie „In Word konvertieren“, „Als PDF exportieren“, „Dieses Markdown in HTML umwandeln“ oder „CSV in eine Tabelle konvertieren“ ausgelöst.
