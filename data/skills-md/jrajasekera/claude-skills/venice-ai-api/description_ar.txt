تكامل واجهة برمجة تطبيقات Venice.ai لتطبيقات الذكاء الاصطناعي التي تعطي الأولوية للخصوصية. يُستخدم عند إنشاء تطبيقات باستخدام واجهة برمجة تطبيقات Venice.ai لإكمال الدردشة أو إنشاء الصور أو إنشاء الفيديو أو تحويل النص إلى كلام أو تحويل الكلام إلى نص أو التضمين. يتم تشغيلها على Venice، أو Venice.ai، أو الذكاء الاصطناعي غير الخاضع للرقابة، أو الذكاء الاصطناعي الذي يعطي الأولوية للخصوصية، أو عندما يحتاج المستخدمون إلى واجهة برمجة تطبيقات متوافقة مع OpenAI مع نماذج غير خاضعة للرقابة.
