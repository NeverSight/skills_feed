تكامل OpenRouter API للوصول الموحد إلى أكثر من 400 نموذج LLM من أكثر من 70 مزودًا. يُستخدم عند إنشاء التطبيقات التي تحتاج إلى استدعاء واجهة برمجة التطبيقات الخاصة بـ OpenRouter لإكمال الدردشة أو البث أو استدعاء الأدوات أو المخرجات المنظمة أو توجيه النموذج. يتم تشغيلها على OpenRouter، أو توجيه النماذج، أو النماذج المتعددة، أو احتياطيات الموفر، أو عندما يحتاج المستخدمون إلى الوصول إلى موفري LLM متعددين من خلال واجهة برمجة تطبيقات واحدة.
