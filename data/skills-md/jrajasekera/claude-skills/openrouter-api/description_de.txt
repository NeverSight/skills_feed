OpenRouter-API-Integration für einheitlichen Zugriff auf über 400 LLM-Modelle von über 70 Anbietern. Wird zum Erstellen von Anwendungen verwendet, die die OpenRouter-API für Chat-Abschlüsse, Streaming, Tool-Aufrufe, strukturierte Ausgaben oder Modellrouting aufrufen müssen. Löst auf OpenRouter, Modell-Routing, Multi-Modell, Anbieter-Fallbacks aus oder wenn Benutzer über eine einzige API auf mehrere LLM-Anbieter zugreifen müssen.
