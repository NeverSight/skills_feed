Intégration de l'API OpenRouter pour un accès unifié à plus de 400 modèles LLM provenant de plus de 70 fournisseurs. À utiliser lors de la création d'applications qui doivent appeler l'API d'OpenRouter pour la réalisation de discussions, le streaming, l'appel d'outils, les sorties structurées ou le routage de modèles. Déclencheurs sur OpenRouter, le routage de modèles, le multimodèle, les solutions de secours de fournisseur ou lorsque les utilisateurs doivent accéder à plusieurs fournisseurs LLM via une seule API.
