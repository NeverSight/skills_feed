Интеграция OpenRouter API для унифицированного доступа к более чем 400 моделям LLM от более чем 70 поставщиков. Используйте при создании приложений, которым необходимо вызывать API OpenRouter для завершения чата, потоковой передачи, вызова инструментов, структурированных выходных данных или маршрутизации модели. Запускается по OpenRouter, маршрутизации модели, мультимодели, резервным поставщикам или когда пользователям необходимо получить доступ к нескольким поставщикам LLM через один API.
