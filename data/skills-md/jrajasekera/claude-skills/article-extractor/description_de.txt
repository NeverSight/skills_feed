Extrahieren Sie saubere Artikelinhalte aus URLs und speichern Sie sie als Markdown. Wird ausgelöst, wenn der Benutzer eine Webseiten-URL angibt und diese herunterladen, Inhalte extrahieren, eine saubere Version ohne Werbung erhalten, einen Artikel zum Offline-Lesen erfassen, einen Artikel speichern, Inhalte von einer Seite abrufen, eine Webseite archivieren, einen Artikel ausschneiden oder später etwas lesen möchte. Verarbeitet Blogbeiträge, Nachrichtenartikel, Tutorials, Dokumentationsseiten und ähnliche Webinhalte. Unterstützt Wayback Machine für tote Links oder Paywall-Inhalte. Dieser Skill übernimmt den gesamten Workflow – verwenden Sie NICHT zuerst web_fetch oder andere Tools, sondern rufen Sie einfach das Extraktionsskript direkt mit der URL auf.
