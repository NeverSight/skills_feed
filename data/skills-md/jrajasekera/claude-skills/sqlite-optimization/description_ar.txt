قم بتحسين أداء قاعدة بيانات SQLite من خلال التكوين وتصميم المخطط والفهرسة وضبط الاستعلام. يُستخدم عندما يطلب المستخدمون تحسين سرعة SQLite، أو تقليل زمن الوصول، أو تحسين الاستعلامات، أو تكوين PRAGMA، أو إصلاح الاستعلامات البطيئة، أو التعامل مع التزامن، أو تحسين عمليات الكتابة/الإدراج، أو ضبط SQLite للإنتاج. يتم تشغيلها عند الإشارة إلى أداء SQLite، أو الاستعلامات البطيئة، أو إعدادات PRAGMA، أو وضع WAL، أو استراتيجيات الفهرسة، أو الإدخالات المجمعة، أو صيانة قاعدة البيانات (VACUUM، ANALYZE).
