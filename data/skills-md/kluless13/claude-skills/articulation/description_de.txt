Rahmen für die intelligente Formulierung von Ideen beim Schreiben und Sprechen. Basierend auf den Methoden von Dan Koe.

AUSLÖSER:
– Wenn der Benutzer darum bittet, einen Beitrag, Thread oder Newsletter zu schreiben
- Wenn sich der Benutzer auf einen Podcast oder eine Präsentation vorbereitet
- Wenn der Benutzer eine komplexe Idee erklären möchte
- Wenn der Benutzer sagt: „Hilf mir beim Artikulieren“ oder „Wie erkläre ich?“
- Beim Verfassen von Inhalten, die überzeugend und klar sein müssen

FÄHIGKEITEN:
- Strukturieren Sie Ideen mithilfe von Problem-Amplify-Solution (Anfänger)
- Wenden Sie das Pyramidenprinzip für logische Argumente an (mittel)
- Domänenübergreifende Synthese für einzigartige Perspektiven (fortgeschritten)
- Erstellen Sie eine „Greatest Hits“-Bibliothek mit Kernideen
