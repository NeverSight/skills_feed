Marco para articular ideas de forma inteligente al escribir y hablar. Basado en los métodos de Dan Koe.

DESENCADENADORES:
- Cuando el usuario solicita escribir una publicación, hilo o boletín
- Cuando el usuario se está preparando para un podcast o una presentación
- Cuando el usuario quiere explicar una idea compleja.
- Cuando el usuario dice "ayúdame a articular" o "cómo te explico"
- Al redactar contenido que debe ser convincente y claro.

CAPACIDADES:
- Estructurar ideas usando Problema-Amplificar-Solución (principiante)
- Aplicar el principio de la pirámide para argumentos lógicos (intermedio)
- Síntesis entre dominios para perspectivas únicas (avanzada)
- Crear una biblioteca de ideas centrales de "grandes éxitos"
