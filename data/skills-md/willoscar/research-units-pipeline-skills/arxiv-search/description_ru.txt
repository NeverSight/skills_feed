Извлекайте метаданные статьи из arXiv с помощью запросов по ключевым словам и сохраняйте результаты в формате JSONL (`papers/papers_raw.jsonl`).
**Триггер**: arXiv, arxiv, поиск по документам, поиск метаданных, поиск по литературе, поиск по статьям, получение метаданных, автономный импорт.
**Используйте, когда**: необходим первоначальный набор документов (опрос/снимок, этап C1) из arXiv (онлайн-поиск или автономный импорт экспорта).
**Пропустить, если**: `papers/papers_raw.jsonl` уже существует или источник данных не arXiv.
**Сеть**: для онлайн-поиска требуется сеть; offline `--input <export.*>` нет.
**Ограждение**: только метаданные; не пишите длинную прозу в `output/`.
