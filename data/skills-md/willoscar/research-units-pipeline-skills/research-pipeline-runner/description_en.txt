Run this repo’s Units+Checkpoints research pipelines end-to-end (survey/review/tutorial/systematic review/referee), with workspaces + checkpoints.  
**Trigger**: run pipeline, kickoff, continue, auto-run, write one, survey/review/research/tutorial/systematic review/referee.  
**Use when**: user wants an end-to-end run (creates `workspaces/<name>/`, generates/executes `UNITS.csv`, pauses at HUMAN checkpoints and waits).  
**Skip if**: user explicitly wants manual step-by-step execution (via `unit-executor`) or you should not auto-advance to prose stage.  
**Network**: depends on selected pipeline (arXiv/PDF/citation verification may need network; offline import supported where available).  
**Guardrail**: checkpoints must be respected (no prose without Approve); must pause and wait at HUMAN units; forbidden to create workspace artifacts in repo root.
