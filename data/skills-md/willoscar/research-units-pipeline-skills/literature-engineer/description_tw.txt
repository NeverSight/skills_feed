多途徑文獻擴展+元數據標準化，用於證據優先調查。
生成一個具有穩定 ID 和出處的大型候選池（`papers/papers_raw.jsonl`，目標 ≥1200），為重複數據刪除/排名 + 引文生成做好準備。
**觸發器**：證據收集者、文獻工程師、文獻增強、多路認知、滾雪球、引用、參考文獻、元信息增強、出處。
**使用時**：需要把候選文獻補充到≥1200篇並補齊可補充元（調查管道的階段C1，寫作前置證據）。
**如果**則跳過：已經有高質量`papers/papers_raw.jsonl`（≥1200且每條都有穩定+標識來源記錄）。
**網絡**：可離線（靠進口）；雪崩/在線搜索需要網絡。
**Guardrail**：不允許編造論文；每條記錄必須帶穩定標識（arXiv id / DOI / 可信 URL）和出處；不寫輸出/散文。
