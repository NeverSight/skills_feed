多途径文献扩展+元数据标准化，用于证据优先调查。
生成一个具有稳定 ID 和出处的大型候选池（`papers/papers_raw.jsonl`，目标 ≥1200），为重复数据删除/排名 + 引文生成做好准备。
**触发器**：证据收集者、文献工程师、文献增强、多路认知、滚雪球、引用、参考文献、元信息增强、出处。
**使用时**：需要把候选文献补充到≥1200篇并补齐可补充元（调查管道的阶段C1，写作前置证据）。
**如果**则跳过：已经有高质量`papers/papers_raw.jsonl`（≥1200且每条都有稳定+标识来源记录）。
**网络**：可离线（靠进口）；雪崩/在线搜索需要网络。
**Guardrail**：不允许编造论文；每条记录必须带稳定标识（arXiv id / DOI / 可信 URL）和出处；不写输出/散文。
