Armar un proyecto LaTeX (`latex/main.tex`, cableado de bibliografía, estructura) a partir de un borrador de Markdown existente y `citations/ref.bib`.
**Activador**: andamio de látex, md→tex, LaTeX 项目骨架, 生成 main.tex.
**Usar cuando**: 需要 LaTeX/PDF 交付（例如 arxiv-survey-latex pipeline），且 draft 已生成/已进入写作阶段。
**Omitir si**: 还没有 `output/DRAFT.md`（或你不需要 LaTeX 交付）。
**Red**: ninguna.
**Guardrail**: 移除 descuento 残留（`##`, `**`, `[@...]`）；bibliografía 指向 `citations/ref.bib`；不在此步骤改写内容。
