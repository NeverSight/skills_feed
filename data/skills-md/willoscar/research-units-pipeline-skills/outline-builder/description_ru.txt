Преобразуйте таксономию (`outline/taxonomy.yml`) в структуру, состоящую только из маркеров (`outline/outline.yml`) с разделами/подразделами.
**Триггер**: построитель контуров, контур маркеров, контур.yml, 大纲生成, только маркеры.
**Используйте, когда**: структура 阶段（НЕ ПРОЗА），已有 таксономия，需要生成可映射/可写作的章节与小节骨架（每小节≥3 маркера）。
**Пропустить, если**: 已经有批准过且可映射的 набросок (避免无意义 отток)。
**Сеть**: нет.
**Ограждение**: только пули；移除 TODO/模板语句；每小节至少 3 个可检查 пули。
