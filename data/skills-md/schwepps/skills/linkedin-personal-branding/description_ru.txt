Комплексный анализ личного бренда LinkedIn, оптимизация профиля и навыки улучшения видимости с использованием инструментов браузера Claude для Chrome. Используйте, когда пользователи запрашивают анализ профиля LinkedIn, аудит личного бренда, рекомендации по оптимизации профиля, улучшение видимости LinkedIn, оптимизацию заголовков, обзор раздела «О компании», рекомендации по стратегии контента, анализ вовлеченности или улучшение индекса социальных продаж. Работает с Claude для Chrome для анализа фотографий профиля, баннеров, заголовков, разделов «О нас», опыта, навыков, рекомендаций, избранного контента, активности/публикаций и взаимодействия с сетью непосредственно из браузера пользователя.
