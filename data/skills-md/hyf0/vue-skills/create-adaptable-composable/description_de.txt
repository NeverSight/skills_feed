Erstellen Sie ein Vue-Composable in Bibliotheksqualität, das möglicherweise reaktive Eingaben (MaybeRef / MaybeRefOrGetter) akzeptiert, sodass Aufrufer einen einfachen Wert, eine Referenz oder einen Getter übergeben können. Normalisieren Sie Eingaben mit toValue()/toRef() innerhalb reaktiver Effekte (watch/watchEffect), um das Verhalten vorhersehbar und reaktiv zu halten. Verwenden Sie diese Fähigkeit, wenn der Benutzer nach der Erstellung anpassbarer oder wiederverwendbarer Composables fragt.
