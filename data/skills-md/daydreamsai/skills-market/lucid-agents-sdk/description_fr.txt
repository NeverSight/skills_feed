Compétence pour travailler avec le SDK Lucid Agents - un framework TypeScript pour la construction
et monétiser les agents d’IA. Utilisez cette compétence lors de la construction ou de la modification d'agents lucides
projets, travaillant avec les points d'entrée des agents, les paiements, l'identité ou la communication A2A.

Activer lorsque : Construire ou modifier des projets Lucid Agents, travailler avec l'agent
points d'entrée, paiements, identité ou communication A2A, se développant dans le
monorepo lucid-agents, création de nouveaux modèles ou fonctionnalités CLI, ou questions sur
l'architecture ou API de Lucid Agents.
