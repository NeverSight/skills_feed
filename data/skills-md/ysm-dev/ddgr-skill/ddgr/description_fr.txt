Recherchez sur le Web à l'aide de DuckDuckGo via l'outil CLI ddgr. À utiliser lorsque l'utilisateur demande à effectuer une recherche sur le Web, à rechercher quelque chose en ligne, à trouver des informations sur Internet, à obtenir des réponses instantanées ou à avoir besoin de données actuelles/en temps réel. À utiliser également pour les recherches DuckDuckGo Bang (par exemple, "! W" pour Wikipedia, "! So" pour StackOverflow).
