Durchsuchen Sie das Web mit DuckDuckGo über das ddgr-CLI-Tool. Verwenden Sie diese Option, wenn der Benutzer das Internet durchsuchen, etwas online nachschlagen, Informationen im Internet suchen, sofortige Antworten erhalten möchte oder aktuelle/Echtzeitdaten benötigt. Auch für DuckDuckGo Bang-Suchen verwenden (z. B. „!w“ für Wikipedia, „!so“ für StackOverflow).
