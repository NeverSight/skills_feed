Выполняйте поиск в Интернете с помощью DuckDuckGo с помощью инструмента CLI ddgr. Используйте, когда пользователь просит выполнить поиск в Интернете, поискать что-либо в Интернете, найти информацию в Интернете, получить мгновенные ответы или ему нужны текущие данные/данные в реальном времени. Также используйте для поиска DuckDuckGo Bang (например, «!w» для Википедии, «!so» для StackOverflow).
