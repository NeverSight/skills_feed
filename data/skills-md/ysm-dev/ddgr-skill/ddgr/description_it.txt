Cerca sul Web utilizzando DuckDuckGo tramite lo strumento CLI ddgr. Da utilizzare quando l'utente chiede di effettuare ricerche sul Web, cercare qualcosa online, trovare informazioni su Internet, ottenere risposte immediate o necessita di dati correnti/in tempo reale. Utilizzare anche per le ricerche DuckDuckGo Bang (ad esempio, "!w" per Wikipedia, "!so" per StackOverflow).
