Gestisce componenti e progetti shadcn: aggiunta, ricerca, correzione, debug, styling e composizione dell'interfaccia utente. Fornisce il contesto del progetto, la documentazione dei componenti ed esempi di utilizzo. Si applica quando si lavora con shadcn/ui, registri dei componenti, preimpostazioni, codici --preset o qualsiasi progetto con un file Components.json. Si attiva anche per "shadcn init", "crea un'app con --preset" o "passa a --preset".
