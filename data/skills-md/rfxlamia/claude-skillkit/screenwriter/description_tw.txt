將創意轉化為針對 AI 影片生成管道進行最佳化的專業、可立即製作的劇本。將原始概念轉換為結構化的逐場景敘述，具有豐富的視覺描述、正確的劇本格式和 XML 標記輸出，以便與圖像/視訊生成工具（imagine、arch-v）無縫整合。
使用時間：將故事創意轉換為劇本格式，為 AI 視訊管道準備內容，為 5-10 分鐘的短片構建敘事，為圖像生成生成視覺豐富的場景描述。
工作流程：原始想法 → 場景分解 → 視覺增強 → 專業格式化 → XML 標記的 Markdown 輸出
輸出：Markdown 文檔，包含 XML 包裝的場景、豐富的視覺描述、適當的劇本元素（sluglines、動作、對話）以及用於管道處理的元資料。
