工作流程：原始想法 → 场景分解 → 视觉增强 → 专业格式化 → XML 标记的 Markdown 输出
输出：Markdown 文档，包含 XML 包装的场景、丰富的视觉描述、适当的剧本元素（sluglines、动作、对话）以及用于管道处理的元数据。
