قم بتحليل مستودعات Flux CD GitOps لمعرفة البنية والتحقق من الصحة والامتثال لواجهة برمجة التطبيقات (API) وأفضل الممارسات. استخدم هذه المهارة عندما يطلب المستخدم تحليل مستودع GitOps أو مراجعته أو تدقيقه أو التحقق من صحته أو فحصه. استخدمه أيضًا عندما يذكر المستخدمون بنية Flux repo، أو أفضل ممارسات GitOps، أو التحقق من صحة البيان، أو واجهات برمجة التطبيقات المهملة، أو تنظيم المستودع - حتى لو لم يقولوا صراحةً "تحليل".
