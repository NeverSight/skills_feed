Herunterladen von Plugins/Modulen für iDempiere (OSGi-Bundles) mit Maven/Tycho: Erstellen Sie ein neues Plugin, erweitern Sie iDempiere für den Kern (Prozesse, Callouts, Modellvalidatoren, Eventhandler, ZK-Formulare, Integrationen), laden Sie es herunter und öffnen Sie es (Feature/Update-Site). Wenn Sie die Arbeit nutzen, müssen Sie die Funktion per Plugin anpassen oder ändern.
