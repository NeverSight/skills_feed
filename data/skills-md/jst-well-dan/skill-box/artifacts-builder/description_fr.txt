Suite d'outils permettant de créer des artefacts HTML claude.ai élaborés et multi-composants à l'aide de technologies Web frontales modernes (React, Tailwind CSS, shadcn/ui). À utiliser pour les artefacts complexes nécessitant des composants de gestion d'état, de routage ou shadcn/ui - pas pour de simples artefacts HTML/JSX à fichier unique.
