Organice, confirme y envíe cambios de git con mensajes de confirmación convencionales. Úselo cuando el usuario quiera confirmar y enviar cambios, mencione enviar a control remoto o solicite guardar y enviar su trabajo. También se activa cuando el usuario dice "enviar cambios", "confirmar y enviar", "enviar esto", "enviar a github" o solicitudes de flujo de trabajo de git similares.
