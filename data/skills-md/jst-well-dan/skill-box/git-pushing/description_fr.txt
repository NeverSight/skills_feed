Organisez, validez et poussez les modifications git avec des messages de validation conventionnels. À utiliser lorsque l'utilisateur souhaite valider et transmettre des modifications, mentionne le transfert à distance ou demande de sauvegarder et de transmettre son travail. S'active également lorsque l'utilisateur dit « pousser les modifications », « valider et pousser », « pousser ceci », « pousser vers github » ou des requêtes de flux de travail git similaires.
