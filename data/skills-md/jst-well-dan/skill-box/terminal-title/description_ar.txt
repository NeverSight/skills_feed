يقوم تلقائيًا بتحديث عنوان النافذة الطرفية ليعكس المهمة الحالية عالية المستوى. يُستخدم في بداية كل جلسة Claude Code عندما يقدم المستخدم المطالبة الأولى، وعندما يتحول المستخدم إلى مهمة جديدة عالية المستوى بشكل واضح. يساعد المطورين على إدارة محطات Claude Code المتعددة من خلال توفير تعريف واضح وسهل لما تعمل عليه كل محطة.
