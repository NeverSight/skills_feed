Déployez des sites statiques sur Cloudflare Pages avec des domaines personnalisés et CI/CD. À utiliser lorsque l'utilisateur souhaite déployer un site sur Cloudflare Pages, ajouter un domaine personnalisé à un projet Pages, configurer GitHub Actions CI/CD pour Cloudflare Pages, restaurer un déploiement ou vérifier l'état du déploiement. Déclencheurs sur « déployer sur Cloudflare », « Pages Cloudflare », « ajouter un domaine personnalisé », « déployer des pages » ou tout workflow d'hébergement de pages Cloudflare.
