Stellen Sie statische Websites auf Cloudflare Pages mit benutzerdefinierten Domänen und CI/CD bereit. Verwenden Sie diese Option, wenn der Benutzer eine Site auf Cloudflare Pages bereitstellen, eine benutzerdefinierte Domäne zu einem Pages-Projekt hinzufügen, GitHub Actions CI/CD für Cloudflare Pages einrichten, ein Rollback einer Bereitstellung durchführen oder den Bereitstellungsstatus überprüfen möchte. Wird bei „Auf Cloudflare bereitstellen“, „Cloudflare Pages“, „Benutzerdefinierte Domäne hinzufügen“, „Seiten bereitstellen“ oder einem beliebigen Cloudflare Pages-Hosting-Workflow ausgelöst.
