Развертывайте статические сайты на страницах Cloudflare с помощью пользовательских доменов и CI/CD. Используйте, когда пользователь хочет развернуть сайт на Cloudflare Pages, добавить личный домен в проект Pages, настроить GitHub Actions CI/CD для Cloudflare Pages, откатить развертывание или проверить статус развертывания. Запускается при «развертывании в Cloudflare», «Страницах Cloudflare», «добавлении пользовательского домена», «развертывании страниц» или любом рабочем процессе хостинга Cloudflare Pages.
