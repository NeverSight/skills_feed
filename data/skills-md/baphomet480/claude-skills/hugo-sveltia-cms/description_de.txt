Starten Sie neue Hugo-Sites mit Sveltia CMS und Basecoat UI oder konvertieren Sie bestehende Sites (beliebiges SSG oder CMS) in Hugo + Sveltia CMS. Verwenden Sie diesen Skill immer dann, wenn der Benutzer Hugo, Sveltia CMS, Decap CMS-Migration, TinaCMS-Migration, statische Site-CMS-Einrichtung, Headless CMS für Hugo erwähnt oder einer Hugo-Site eine Content-Management-Schnittstelle hinzufügen möchte. Wird auch beim Konvertieren von WordPress, Jekyll, Eleventy, TinaCMS oder anderen Websites in Hugo oder beim Einrichten einer Git-basierten Inhaltsverwaltung ausgelöst. Deckt den gesamten Workflow vom Gerüstbau bis zur Bereitstellung von Cloudflare Pages mit GitHub-OAuth-Authentifizierung ab.
