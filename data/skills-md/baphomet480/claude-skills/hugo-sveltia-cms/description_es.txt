Arranque nuevos sitios Hugo con Sveltia CMS y Basecoat UI, o convierta sitios existentes (cualquier SSG o CMS) a Hugo + Sveltia CMS. Utilice esta habilidad siempre que el usuario mencione Hugo, Sveltia CMS, migración de Decap CMS, migración de TinaCMS, configuración de CMS de sitio estático, CMS sin cabeza para Hugo o desee agregar una interfaz de administración de contenido a un sitio de Hugo. También se activa al convertir WordPress, Jekyll, Eleventy, TinaCMS u otros sitios a Hugo, o al configurar la administración de contenido basada en Git. Cubre el flujo de trabajo completo desde el scaffolding hasta la implementación de Cloudflare Pages con autenticación GitHub OAuth.
