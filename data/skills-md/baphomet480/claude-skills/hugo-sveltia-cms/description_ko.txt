Sveltia CMS 및 Basecoat UI를 사용하여 새로운 Hugo 사이트를 부트스트랩하거나 기존 사이트(SSG 또는 CMS)를 Hugo + Sveltia CMS로 변환하세요. 사용자가 Hugo, Sveltia CMS, Decap CMS 마이그레이션, TinaCMS 마이그레이션, 정적 사이트 CMS 설정, Hugo용 헤드리스 CMS를 언급하거나 Hugo 사이트에 콘텐츠 관리 인터페이스를 추가하려고 할 때마다 이 기술을 사용하세요. WordPress, Jekyll, Eleventy, TinaCMS 또는 기타 사이트를 Hugo로 변환하거나 Git 기반 콘텐츠 관리를 설정할 때도 트리거됩니다. 스캐폴딩부터 GitHub OAuth 인증을 사용한 Cloudflare Pages 배포까지의 전체 워크플로를 다룹니다.
