Sveltia CMS と Basecoat UI を使用して新しい Hugo サイトをブートストラップするか、既存のサイト (SSG または CMS) を Hugo + Sveltia CMS に変換します。ユーザーが Hugo、Sveltia CMS、Decap CMS 移行、TinaCMS 移行、静的サイト CMS セットアップ、Hugo 用ヘッドレス CMS について言及した場合、または Hugo サイトにコンテンツ管理インターフェイスを追加したい場合には、このスキルを使用します。 WordPress、Jekyll、イレブンティ、TinaCMS、またはその他のサイトを Hugo に変換するとき、または Git ベースのコンテンツ管理を設定するときにもトリガーされます。スキャフォールディングから、GitHub OAuth 認証を使用した Cloudflare Pages のデプロイまでの完全なワークフローをカバーします。
