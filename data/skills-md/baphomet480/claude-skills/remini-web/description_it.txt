Applica i miglioramenti dell'intelligenza artificiale di Remini tramite un account Web Remini registrato da file di immagine locali. Da utilizzare quando l'utente richiede di migliorare, eseguire l'upscaling, ripristinare, correggere la sfocatura o ritoccare le foto con Remini Web o desidera un flusso di lavoro di automazione del browser ripetibile per i download di output di Remini.
