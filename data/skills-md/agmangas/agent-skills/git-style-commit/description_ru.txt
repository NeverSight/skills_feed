Анализируйте историю git на предмет стиля фиксации, изменений стадий и фиксации без принудительной отправки. Используйте, когда пользователь хочет зафиксировать изменения с помощью сообщения, соответствующего существующему стилю фиксации его репозитория, или когда он просит зафиксировать проиндексированные/непроиндексированные изменения.
