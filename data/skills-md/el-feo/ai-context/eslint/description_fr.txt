Agent ESLint complet pour la qualité du code JavaScript/TypeScript. À utiliser lors de la configuration d'ESLint, de la configuration des règles de peluchage, de l'analyse du code pour détecter les problèmes, de la correction des violations ou de l'intégration d'ESLint dans les flux de travail de développement. Se déclenche sur des requêtes impliquant la qualité du code, le peluchage, l'analyse statique ou la configuration ESLint pour les projets JavaScript, TypeScript, React ou Node.js.
