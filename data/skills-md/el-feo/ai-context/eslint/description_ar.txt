وكيل ESLint الشامل لجودة كود JavaScript/TypeScript. يُستخدم عند إعداد ESLint، أو تكوين قواعد الفحص، أو تحليل التعليمات البرمجية للمشكلات، أو إصلاح الانتهاكات، أو دمج ESLint في سير عمل التطوير. يتم تشغيل الطلبات التي تتضمن جودة التعليمات البرمجية أو الفحص أو التحليل الثابت أو تكوين ESLint لمشروعات JavaScript أو TypeScript أو React أو Node.js.
