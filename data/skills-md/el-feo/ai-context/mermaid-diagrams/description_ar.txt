دليل شامل لإنشاء الرسوم البيانية البرمجية باستخدام بناء جملة حورية البحر. يُستخدم عندما يحتاج المستخدمون إلى إنشاء برامج أو تصورها أو توثيقها من خلال الرسوم البيانية بما في ذلك الرسوم التخطيطية للفئات (نمذجة المجال، التصميم الموجه للكائنات)، الرسوم البيانية التسلسلية (تدفقات التطبيق، تفاعلات واجهة برمجة التطبيقات، تنفيذ التعليمات البرمجية)، المخططات الانسيابية (العمليات، الخوارزميات، رحلات المستخدم)، الرسوم البيانية لعلاقة الكيان (مخططات قاعدة البيانات)، الرسوم التخطيطية لهندسة C4 (سياق النظام، الحاويات، المكونات)، الرسوم التخطيطية للحالة، الرسوم البيانية git، المخططات الدائرية، مخططات جانت، أو أي نوع مخطط آخر. تتضمن المشغلات طلبات "الرسم التخطيطي"، أو "التصور"، أو "النموذج"، أو "التخطيط"، أو "إظهار التدفق"، أو عند شرح بنية النظام، أو تصميم قاعدة البيانات، أو بنية التعليمات البرمجية، أو تدفقات المستخدم/التطبيق.
