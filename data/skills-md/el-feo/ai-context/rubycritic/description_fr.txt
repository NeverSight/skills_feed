Intégrez RubyCritic pour analyser la qualité du code Ruby et maintenir des normes élevées tout au long du développement. À utiliser lorsque vous travaillez sur des projets Ruby pour vérifier les odeurs, la complexité et la duplication du code. Les déclencheurs incluent la création/la modification de fichiers Ruby, la refactorisation du code, l'examen de la qualité du code ou lorsque l'utilisateur demande une analyse du code ou des contrôles de qualité.
