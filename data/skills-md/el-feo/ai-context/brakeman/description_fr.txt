Scanner de vulnérabilités de sécurité d'analyse statique pour les applications Ruby on Rails. À utiliser pour analyser le code Rails pour détecter des problèmes de sécurité, exécuter des audits de sécurité, examiner le code pour détecter les vulnérabilités, configurer l'analyse de sécurité dans CI/CD, gérer les avertissements de sécurité ou enquêter sur des types de vulnérabilités spécifiques (injection SQL, XSS, injection de commandes, etc.). À utiliser également lors de la configuration de Brakeman, de la réduction des faux positifs ou de l'intégration à des flux de travail automatisés.
