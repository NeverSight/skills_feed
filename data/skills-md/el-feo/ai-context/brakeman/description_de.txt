Statischer Analyse-Sicherheitsschwachstellenscanner für Ruby on Rails-Anwendungen. Verwenden Sie es, wenn Sie Rails-Code auf Sicherheitsprobleme analysieren, Sicherheitsüberprüfungen durchführen, Code auf Schwachstellen überprüfen, Sicherheitsscans in CI/CD einrichten, Sicherheitswarnungen verwalten oder bestimmte Schwachstellentypen untersuchen (SQL-Injection, XSS, Command-Injection usw.). Verwenden Sie es auch, wenn Sie Brakeman konfigurieren, Fehlalarme reduzieren oder in automatisierte Arbeitsabläufe integrieren.
