Configurazione completa di PostgreSQL e analisi dell'utilizzo per le applicazioni Rails. Da utilizzare quando Claude Code deve analizzare una base di codice Rails per individuare problemi di prestazioni del database, opportunità di ottimizzazione o violazioni delle best practice. Rileva N+1 query, indici mancanti, configurazioni di database non ottimali, anti-pattern e fornisce consigli attuabili. Ideale per controlli delle prestazioni, attività di ottimizzazione o quando gli utenti chiedono di "analizzare il database", "controllare N+1 query", "ottimizzare PostgreSQL", "esaminare le prestazioni del database" o "suggerire miglioramenti del database".
