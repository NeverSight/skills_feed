تكوين PostgreSQL الشامل وتحليل الاستخدام لتطبيقات Rails. يُستخدم عندما يحتاج Claude Code إلى تحليل قاعدة تعليمات Rails لمشاكل أداء قاعدة البيانات، أو فرص التحسين، أو انتهاكات أفضل الممارسات. يكتشف استعلامات N+1، والفهارس المفقودة، وتكوينات قاعدة البيانات دون المستوى الأمثل، والأنماط المضادة، ويقدم توصيات قابلة للتنفيذ. مثالي لعمليات تدقيق الأداء، أو مهام التحسين، أو عندما يطلب المستخدمون "تحليل قاعدة البيانات"، أو "التحقق من استعلامات N+1"، أو "تحسين PostgreSQL"، أو "مراجعة أداء قاعدة البيانات"، أو "اقتراح تحسينات لقاعدة البيانات".
