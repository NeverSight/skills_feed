Analyse complète de la configuration et de l'utilisation de PostgreSQL pour les applications Rails. À utiliser lorsque Claude Code doit analyser une base de code Rails pour détecter des problèmes de performances de base de données, des opportunités d'optimisation ou des violations des meilleures pratiques. Détecte les requêtes N+1, les index manquants, les configurations de bases de données sous-optimales, les anti-modèles et fournit des recommandations exploitables. Idéal pour les audits de performances, les tâches d'optimisation ou lorsque les utilisateurs demandent « analyser la base de données », « vérifier les requêtes N+1 », « optimiser PostgreSQL », « examiner les performances de la base de données » ou « suggérer des améliorations à la base de données ».
