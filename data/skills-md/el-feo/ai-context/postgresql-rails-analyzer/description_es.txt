Análisis integral de configuración y uso de PostgreSQL para aplicaciones Rails. Úselo cuando Claude Code necesite analizar una base de código Rails para detectar problemas de rendimiento de la base de datos, oportunidades de optimización o violaciones de las mejores prácticas. Detecta consultas N+1, índices faltantes, configuraciones de bases de datos subóptimas, antipatrones y proporciona recomendaciones prácticas. Ideal para auditorías de rendimiento, tareas de optimización o cuando los usuarios solicitan "analizar la base de datos", "comprobar consultas N+1", "optimizar PostgreSQL", "revisar el rendimiento de la base de datos" o "sugerir mejoras en la base de datos".
