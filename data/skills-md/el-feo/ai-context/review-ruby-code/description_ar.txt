مراجعة شاملة لرموز Ruby and Rails باستخدام قواعد Sandi Metz ومبادئ SOLID. تشغيل Rubycritic وSimplecov تلقائيًا، وتحليل الملفات التي تم تغييرها في الفرع الحالي مقابل الفرع الأساسي، وتحديد انتهاكات OOP، وأنماط Rails المضادة، ومشكلات الأمان، وفجوات تغطية الاختبار. مخرجات REVIEW.md مع روابط الملفات المتوافقة مع VSCode. يُستخدم عند مراجعة كود Ruby/Rails، أو إجراء مراجعات الكود، أو التحقق من مشكلات التصميم، أو عندما يذكر المستخدم مراجعة الكود، أو مراجعة طلب السحب، أو تحليل جودة الكود.
