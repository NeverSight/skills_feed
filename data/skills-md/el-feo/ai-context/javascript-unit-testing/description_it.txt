Scrittura di unit test di alta qualità per JavaScript e TypeScript utilizzando Jest. Copre la struttura dei test (modello AAA, denominazione USE), l'interruzione delle dipendenze (stub, mock, inserimento delle dipendenze), il test del codice asincrono (promesse, callback, timer), l'evitamento di test instabili e lo sviluppo basato sui test. Da utilizzare durante la scrittura di test, il debug di errori di test, il refactoring di test per la manutenibilità o domande su Jest, TDD, mock, stub o best practice di test.
