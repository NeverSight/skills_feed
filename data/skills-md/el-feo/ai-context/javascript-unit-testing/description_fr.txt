Rédaction de tests unitaires de haute qualité pour JavaScript et TypeScript à l'aide de Jest. Couvre la structure de test (modèle AAA, dénomination USE), la rupture des dépendances (stubs, mocks, injection de dépendances), le test du code asynchrone (promesses, rappels, minuteries), l'évitement des tests irréguliers et le développement piloté par les tests. À utiliser lors de l'écriture de tests, du débogage des échecs de tests, de la refactorisation des tests pour la maintenabilité ou des questions sur Jest, TDD, les simulations, les stubs ou les meilleures pratiques de test.
