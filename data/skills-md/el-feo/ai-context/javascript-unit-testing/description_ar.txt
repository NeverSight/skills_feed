كتابة اختبارات وحدة عالية الجودة لـ JavaScript وTypeScript باستخدام Jest. يغطي بنية الاختبار (نمط AAA، تسمية الاستخدام)، كسر التبعيات (بذرة، نماذج، حقن التبعية)، اختبار التعليمات البرمجية غير المتزامنة (الوعود، عمليات الاسترجاعات، الموقتات)، وتجنب الاختبارات غير المستقرة، والتطوير القائم على الاختبار. يُستخدم عند كتابة الاختبارات، أو تصحيح أخطاء فشل الاختبار، أو إعادة بناء الاختبارات من أجل قابلية الصيانة، أو الأسئلة حول Jest أو TDD أو النماذج النموذجية أو كعب الروتين أو اختبار أفضل الممارسات.
