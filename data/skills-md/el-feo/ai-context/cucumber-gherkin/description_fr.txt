Tests BDD complets avec la syntaxe Cucumber et Gherkin. À utiliser lors de l'écriture de fichiers de fonctionnalités (.feature), de définitions d'étapes, de hooks ou de la mise en œuvre d'un développement piloté par le comportement. Couvre les mots-clés Gherkin (Fonctionnalité, Scénario, Étant donné/Quand/Alors, Arrière-plan, Aperçu du scénario, Règle), les modèles de définition d'étape pour Ruby/JavaScript/Java/Python, les hooks (Avant/Après/AvantTout/AprèsTout), les balises, les tableaux de données, les chaînes de documentation et les meilleures pratiques. Déclencheurs sur concombre, cornichon, BDD, fichiers de fonctionnalités, définitions d'étapes, tests d'acceptation, spécifications exécutables.
