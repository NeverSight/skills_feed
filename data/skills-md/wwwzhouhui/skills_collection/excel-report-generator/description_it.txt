Genera automaticamente report Excel da origini dati tra cui CSV, database o strutture dati Python. Supporta report di analisi dei dati, report aziendali, esportazione di dati e generazione di report basati su modelli utilizzando Panda e openpyxl. Attiva quando gli utenti menzionano Excel, foglio di calcolo, generazione di report, esportazione di dati o reporting aziendale.
