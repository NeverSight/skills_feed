Générez automatiquement des rapports Excel à partir de sources de données, notamment CSV, bases de données ou structures de données Python. Activez-le lorsque les utilisateurs mentionnent Excel, une feuille de calcul, la génération de rapports, l'exportation de données ou le reporting commercial.
