Generación de imágenes y vídeo con IA mediante jimeng-mcp-server. Usar cuando pidan imagen a partir de texto, combinar imágenes, vídeo a partir de texto o animar una imagen. Soporta: texto→imagen, composición de imágenes, texto→vídeo, imagen→vídeo. Requiere jimeng-mcp-server en local o vía SSE/HTTP.
