استخدم هذه المهارة عند إجراء مراجعات الأمان أو اختبار الاختراق أو نمذجة التهديدات أو أعمال الامتثال أو الاستجابة للحوادث. يتم التنشيط عند الإشارة إلى التدقيق الأمني، أو الضعف، أو اختبار الاختراق، أو pentest، أو OWASP، أو CVE، أو المراجعة الأمنية، أو نموذج التهديد، أو الثقة المعدومة، أو SOC 2، أو HIPAA، أو القانون العام لحماية البيانات (GDPR)، أو الامتثال، أو الاستجابة للحوادث، أو SBOM، أو أمان سلسلة التوريد، أو إدارة الأسرار، أو أمان المصادقة.
