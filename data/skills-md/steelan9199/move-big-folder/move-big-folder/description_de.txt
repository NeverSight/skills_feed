Migrieren Sie große Ordner vom Laufwerk C auf ein anderes Laufwerk (D/E/F usw.), behalten Sie den Zugriff auf den ursprünglichen Pfad über symbolische Links bei und geben Sie Speicherplatz auf dem Laufwerk C frei. Verwenden Sie diese Fähigkeit, wenn der Benutzer „Laufwerk C voll“, „Laufwerk C rot“, „Große Ordner migrieren“, „Speicherplatz auf Laufwerk C freigeben“ und „Symlink-Migration“ erwähnt.
