## 功能概述

用于生成图片提示词, 生成的图片会作为公众号文章的封面

## 提示词模板

```
一个未来主义风格的亚克力徽章，3D超写实渲染。徽章中心是透明的，可以看到背景。徽章边缘有明亮的霓虹蓝色和紫色辉光。徽章正面有两行文字，上面一行是“牙叔”，下面一行是“教程”，字体为现代无衬线体，带有轻微的发光效果。整体采用电影级照明，光影对比强烈，背景为深色，突出徽章的科技感和立体感。
图片宽高比:`21:9`, 2K
```

## 封面文字提取原则

封面应该突出**核心价值/用户收益**，而非产品名称。遵循以下优先级：

| 优先级 | 内容类型 | 示例 |
|--------|----------|------|
| 1 | **痛点/解决方案** | "告别复制粘贴"、"Markdown秒发" |
| 2 | **核心功能** | "自动同步"、"一键发布" |
| 3 | **用户收益** | "效率翻倍"、"10分钟→1分钟" |
| 4 | 产品名称 | 放正文介绍，**不要放封面** |

### 提取技巧
- 问：用户看完能获得什么？→ 提取这个结果
- 问：解决了什么痛点？→ 提取这个痛点/解决方案
- 产品名、技术细节 → 省略或放正文

## 工作流程

1. 分析用户提供的标题/文案，提取**核心价值**（见上方原则）
2. 把提示词模板中的文字替换为提取的核心价值
3. 优化提示词
   1. 考虑一下未来生成的图片的排版
   2. 要显示的文字，如果不是重要的字，可以进行适当的删减，要精简
   3. 提示词要包含图片宽高比和2K
4. 把替换后的提示词发给用户，并说明提取思路
5. 提示用户打开即梦`https://jimeng.jianying.com/ai-tool/home?activeTab=activity`, 输入提示词生成`封面图片`.
