Aide à estimer et à calculer les coûts des ressources Azure. Utilisez cette compétence lorsque les utilisateurs posent des questions sur la tarification Azure, l’estimation des coûts, les coûts de dimensionnement des ressources, la comparaison des niveaux de tarification, la budgétisation des déploiements Azure ou la compréhension de la facturation Azure. Les déclencheurs incluent des questions telles que « combien cela coûtera-t-il dans Azure », « estimer les coûts Azure », « comparer les tarifs Azure », « budget pour les ressources Azure ».
