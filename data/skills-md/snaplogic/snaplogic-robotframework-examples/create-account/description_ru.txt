Создает тестовые примеры Robot Framework для создания учетной записи SnapLogic. Используйте, когда пользователь хочет создать учетные записи (Oracle, PostgreSQL, Snowflake, Kafka, S3 и т. д.), ему необходимо знать, какие переменные среды нужно настроить, или он хочет просмотреть примеры тестовых примеров учетных записей.
