Руководства по настройке контейнеров баз данных для тестирования в проекте SnapLogic. Используйте, когда пользователь хочет запустить/остановить базы данных (Oracle, PostgreSQL, MySQL, SQL Server, DB2, Teradata, Snowflake), загрузить тестовые данные или устранить неполадки с подключениями к базе данных.
