Создает тестовые сценарии Robot Framework для создания и выполнения задач, запускаемых SnapLogic. Используйте, когда пользователь хочет создать триггерные задачи для выполнения конвейера по требованию, выполнить триггерные задачи с параметрами или хочет просмотреть примеры тестовых сценариев триггерных задач.
