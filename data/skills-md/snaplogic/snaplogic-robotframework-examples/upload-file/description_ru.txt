Создает тестовые примеры Robot Framework для загрузки файлов в SnapLogic SLDB. Используйте, когда пользователь хочет загрузить файлы (JSON, CSV, библиотеки выражений, конвейеры, файлы JAR и т. д.), ему необходимо знать, какой путь назначения использовать, или он хочет увидеть примеры тестовых случаев загрузки файлов.
