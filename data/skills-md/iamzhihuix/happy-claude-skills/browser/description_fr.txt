Outils minimaux du protocole Chrome DevTools pour l'automatisation et le scraping du navigateur. À utiliser lorsque vous devez démarrer Chrome, parcourir des pages, exécuter JavaScript, prendre des captures d'écran ou sélectionner de manière interactive des éléments DOM. Les déclencheurs incluent « parcourir le site Web », « gratter la page », « prendre une capture d'écran », « automatiser le navigateur », « extraire le DOM », « scraping Web ».
