Aktualisieren Sie Laravel-Anwendungen jeweils um eine Hauptversion (9→10, 10→11, 11→12). Verwenden Sie diese Option, wenn der Benutzer seine Laravel-Framework-Version aktualisieren möchte. Erkennt automatisch die aktuelle Version aus Composer.json, identifiziert wichtige Änderungen und wendet erforderliche Codekorrekturen an.
