Предоставляет комплексное руководство по действиям GitHub, включая создание рабочих процессов, действия, секреты и автоматизацию. Используйте, когда пользователь спрашивает о действиях GitHub, ему необходимо создать рабочие процессы GitHub, автоматизировать процессы GitHub или настроить CI/CD с помощью действий GitHub.
