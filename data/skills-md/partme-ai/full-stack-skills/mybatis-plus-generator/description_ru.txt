Предоставляет комплексное руководство по созданию кода MyBatis-Plus, включая Entity, Mapper, Service, ServiceImpl, 
Контроллер, DTO, VO, BO и другие связанные объекты из таблиц базы данных. Используйте ТОЛЬКО когда пользователь явно 
упоминает MyBatis-Plus, mybatis-plus-generator или хочет сгенерировать код с использованием платформы MyBatis-Plus. 
Этот навык автоматически генерирует стандартные методы CRUD и пользовательские методы на основе требований пользователя для 
Проекты MyBatis-Plus. Поддерживает архитектуры MVC и DDD, языки Java и Kotlin. НЕ запускайте для 
генерация общего кода, JPA/Hibernate или другие платформы ORM.
