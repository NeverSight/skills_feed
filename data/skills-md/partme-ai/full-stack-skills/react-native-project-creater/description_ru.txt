Обеспечивает создание проекта для React Native одной командой, включая инициализацию проекта, настройку и генерацию шаблона. Используйте, когда пользователь спрашивает о создании проектов React Native, ему необходимо инициализировать новый проект React Native или создать структуру проекта React Native.
