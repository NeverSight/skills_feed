Предоставляет комплексное руководство по микросервисам Spring Cloud, включая обнаружение сервисов, управление конфигурацией, балансировку нагрузки, автоматические выключатели, шлюзы API и распределенную трассировку. Используйте, когда пользователь спрашивает о Spring Cloud, ему необходимо создать микросервисы, реализовать обнаружение сервисов или работать с компонентами Spring Cloud.
