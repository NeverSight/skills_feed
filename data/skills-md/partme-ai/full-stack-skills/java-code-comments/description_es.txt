Proporciona una guía completa para agregar comentarios de código Java siguiendo los estándares y las mejores prácticas de la industria. 
Esta habilidad ayuda a agregar comentarios a nivel de clase, comentarios a nivel de método y comentarios a nivel de campo al código Java. 
Úselo cuando el usuario quiera agregar comentarios al código Java, necesite documentar clases/métodos/campos de Java, 
quiere mejorar la documentación del código o necesita generar comentarios de JavaDoc. Esta habilidad cubre Controlador, 
Servicio, ServiceImpl, Mapper, Modelo, Entidad, BO (Business Object), DTO, VO y otros Java comunes 
tipos de componentes. La habilidad sigue un flujo de trabajo sistemático: escanear código base, identificar componentes, crear 
lista de tareas pendientes y agregue comentarios en orden (comentarios de clase → comentarios de método → comentarios de campo).
