Содержит подробные рекомендации по добавлению комментариев к коду Java в соответствии с отраслевыми стандартами и передовыми практиками. 
Этот навык помогает добавлять комментарии на уровне класса, комментарии на уровне метода и комментарии на уровне поля в код Java. 
Используйте, когда пользователь хочет добавить комментарии к коду Java, ему необходимо документировать классы/методы/поля Java, 
хочет улучшить документацию по коду или нуждается в создании комментариев JavaDoc. Этот навык охватывает Контроллера, 
Service, ServiceImpl, Mapper, Model, Entity, BO (бизнес-объект), DTO, VO и другие распространенные Java. 
типы компонентов. Навык соответствует систематическому рабочему процессу: сканирование базы кода, определение компонентов, создание 
список дел и добавляйте комментарии по порядку (комментарии класса → комментарии к методам → комментарии к полям).
