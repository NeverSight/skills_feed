يوفر إرشادات شاملة لإنشاء مخططات حورية البحر. Mermaid هي أداة رسم تخطيطي ورسم بياني تعتمد على JavaScript وتستخدم تعريفات نصية مستوحاة من Markdown وعارضًا لإنشاء وتعديل الرسوم البيانية المعقدة. الغرض الرئيسي من Mermaid هو مساعدة التوثيق على مواكبة التطور. تعد Mermaid مناسبة بشكل خاص للاستخدام في مستندات Markdown وGitHub وGitLab والويكي والمدونات وغيرها من المنصات المستندة إلى Markdown. يُستخدم عندما يريد المستخدم رسم أو إنشاء أو إنشاء أو إنشاء أو إنشاء أو تصور أي رسم تخطيطي أو مخطط أو رسم بياني أو مخطط انسيابي أو مخطط معماري أو مخطط تسلسلي أو مخطط فئة أو مخطط حالة أو مخطط جانت أو خريطة ذهنية أو مخطط زمني أو أي مخطط مرئي آخر بتنسيق مناسب لـ Markdown. تغطي هذه المهارة جميع أنواع مخططات حورية البحر التي يزيد عددها عن 23 بما في ذلك المخططات الانسيابية، والرسوم البيانية التسلسلية، والرسوم البيانية للفصل، ومخططات الحالة، ومخططات ER، ومخططات جانت، والمخططات الدائرية، والمخططات الرباعية، والرسوم البيانية Git، ومخططات C4، والخرائط الذهنية، والجداول الزمنية، والمزيد. استخدم هذه المهارة دائمًا عندما يذكر المستخدم حورية البحر، أو يحتاج إلى مخططات لتوثيق Markdown، أو يريد مخططات سريعة يتم عرضها مباشرة في عارضات Markdown.
