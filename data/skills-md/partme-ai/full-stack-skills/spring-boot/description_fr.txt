Fournit des conseils complets pour le développement de Spring Boot, y compris la création de projets, la configuration automatique, l'injection de dépendances, le développement Web, l'accès aux données, la sécurité, les tests et le déploiement. À utiliser lorsque l'utilisateur pose des questions sur Spring Boot, doit créer des applications Spring Boot, configurer Spring Boot ou implémenter des fonctionnalités Spring Boot.
