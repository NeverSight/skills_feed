Обеспечивает создание проекта для uni-приложения одной командой с использованием официального интерфейса командной строки быстрого запуска, включая инициализацию проекта, настройку и выбор шаблона. Используйте, когда пользователь запрашивает создание проекта uni-приложения с помощью одной команды, ему необходимо инициализировать новый проект uni-app или создать структуру проекта uni-app.
