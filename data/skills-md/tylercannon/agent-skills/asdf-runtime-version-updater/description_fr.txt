Mettez à jour les versions d'exécution du langage dans les fichiers .tool-versions (asdf) et propagez les modifications aux fichiers Docker, aux workflows GitHub Actions et à d'autres fichiers de référence de version. À utiliser lorsque : (1) la mise à jour des versions d'exécution du langage d'un projet, (2) la synchronisation des versions entre les versions .tool, les fichiers Docker et les configurations CI/CD, ou (3) la migration vers une nouvelle version d'exécution.
