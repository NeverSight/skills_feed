Aggiorna le versioni del runtime del linguaggio nei file .tool-versions (asdf) e propaga le modifiche ai file Docker, ai flussi di lavoro di GitHub Actions e ad altri file che fanno riferimento alla versione. Da utilizzare quando: (1) si aggiornano le versioni runtime della lingua di un progetto, (2) si sincronizzano le versioni tra versioni .tool, Dockerfile e configurazioni CI/CD o (3) si esegue la migrazione a una nuova versione runtime.
