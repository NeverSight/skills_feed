Обновляйте версии среды выполнения языка в файлах .tool-versions (asdf) и распространяйте изменения в файлы Dockerfiles, рабочие процессы GitHub Actions и другие файлы, ссылающиеся на версию. Используйте при: (1) обновлении версий среды выполнения проекта, (2) синхронизации версий между версиями .tool, файлами Dockerfile и конфигурациями CI/CD или (3) переходе на новую версию среды выполнения.
