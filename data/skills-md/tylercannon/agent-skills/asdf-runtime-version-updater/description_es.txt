Actualice las versiones del lenguaje de ejecución en archivos .tool-versions (asdf) y propague los cambios a Dockerfiles, flujos de trabajo de GitHub Actions y otros archivos de referencia de versiones. Úselo cuando: (1) actualice las versiones de tiempo de ejecución del lenguaje de un proyecto, (2) sincronice versiones entre versiones de .tool, Dockerfiles y configuraciones de CI/CD, o (3) migre a una nueva versión de tiempo de ejecución.
