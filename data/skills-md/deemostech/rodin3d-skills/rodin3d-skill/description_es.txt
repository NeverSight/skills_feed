Convierte imágenes de entrada o mensajes a modelos 3D utilizando la API Hyper3D Rodin Gen-2. Utilice esta habilidad cuando los usuarios quieran generar modelos 3D a partir de imágenes o texto, como diseños de productos, elementos arquitectónicos o reconstrucciones de objetos. Esta habilidad maneja la comunicación API, el sondeo del estado de las tareas y la recuperación de modelos 3D.
