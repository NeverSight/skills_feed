Erstellen Sie eine Kapitelliste für ein Bilibili-Video.  
Auslöseszenarien:  
(1) Es müssen Bilibili-Kapitel für ein Video erstellt werden  
(2) Der Benutzer sagt „In Bilibili-Format konvertieren“, „Kapitel generieren“ oder „Bilibili-Kapitel generieren“  
(3) Es müssen Videosegmente aus Untertiteln generiert werden  
(4) Behandeln Sie die Segmentmarkierungen der Video-Fortschrittsleiste
