Guide la création de serveurs MCP (Model Context Protocol) de haute qualité qui permettent les LLM
d'interagir avec des services externes grâce à des outils bien conçus. À utiliser lors de la création de MCP
serveurs pour intégrer des API ou des services externes, que ce soit en Python (FastMCP) ou
Nœud/TypeScript (SDK MCP). Couvre la conception d'outils, l'authentification, le déploiement de Docker,
et la création d'évaluations. PAS lors de la consommation de serveurs MCP existants (utilisez le serveur directement).
