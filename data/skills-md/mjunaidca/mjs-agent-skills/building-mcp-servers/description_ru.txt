Руководит созданием высококачественных серверов MCP (Model Context Protocol), поддерживающих LLM.
взаимодействовать с внешними сервисами посредством хорошо продуманных инструментов. Используйте при построении MCP
серверы для интеграции внешних API или сервисов, будь то на Python (FastMCP) или
Node/TypeScript (MCP SDK). Охватывает разработку инструментов, аутентификацию, развертывание Docker,
и создание оценок. НЕ при использовании существующих серверов MCP (используйте сервер напрямую).
