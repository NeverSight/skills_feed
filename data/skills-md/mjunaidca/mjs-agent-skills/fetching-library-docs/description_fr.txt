Outil de récupération de documentation d'API de bibliothèque efficace en jetons utilisant Context7 MCP avec 77 % d'économies de jetons.
Récupère des exemples de code, des références API et des modèles d'utilisation pour les bibliothèques publiées (React,
Next.js, Prisma, etc.). À utiliser lorsque les utilisateurs demandent "comment utiliser la bibliothèque X", ont besoin d'exemples de code,
souhaitez la syntaxe de l'API ou apprenez l'API officielle d'un framework. Déclencheurs : "Montrez-moi les hooks React",
"Syntaxe de requête Prisma", "API de routage Next.js". PAS pour explorer les composants internes du dépôt/le code source
(utilisez la recherche avec deepwiki) ou des fichiers locaux.
