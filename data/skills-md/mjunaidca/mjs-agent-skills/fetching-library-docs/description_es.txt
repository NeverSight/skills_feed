Recuperador de documentación API de biblioteca con uso eficiente de tokens que utiliza Context7 MCP con un ahorro de tokens del 77 %.
Obtiene ejemplos de código, referencias de API y patrones de uso para bibliotecas publicadas (React,
Next.js, Prisma, etc.). Úselo cuando los usuarios pregunten "¿cómo uso la biblioteca X?", necesiten ejemplos de código,
desea la sintaxis de la API o está aprendiendo la API oficial de un marco. Activadores: "Muéstrame los ganchos de React",
"Sintaxis de consulta Prisma", "API de enrutamiento Next.js". NO para explorar el código fuente/internos del repositorio
(use researching-with-deepwiki) o archivos locales.
