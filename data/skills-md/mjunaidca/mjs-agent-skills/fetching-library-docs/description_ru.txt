Средство выборки документации API библиотеки с эффективным использованием токенов с использованием Context7 MCP с экономией токенов 77%.
Извлекает примеры кода, ссылки на API и шаблоны использования опубликованных библиотек (React,
Next.js, Prisma и т. д.). Используйте, когда пользователи спрашивают: «Как мне использовать библиотеку X», нужны примеры кода,
хотите синтаксис API или изучаете официальный API платформы. Триггеры: «Покажи мне хуки React»,
«Синтаксис запроса Prisma», «API маршрутизации Next.js». НЕ для изучения внутреннего устройства/исходного кода репо
(используйте исследование с помощью Deepwiki) или локальные файлы.
