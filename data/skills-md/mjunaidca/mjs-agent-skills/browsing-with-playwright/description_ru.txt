Автоматизация браузера с помощью Playwright MCP. Навигация по веб-сайтам, заполнение форм, нажатие элементов,
делать снимки экрана и извлекать данные. Используйте, когда задачи требуют просмотра веб-страниц, отправки формы,
парсинг веб-страниц, тестирование пользовательского интерфейса или любое взаимодействие с браузером. НЕ при получении только статики
содержимое (вместо этого используйте curl/wget).
