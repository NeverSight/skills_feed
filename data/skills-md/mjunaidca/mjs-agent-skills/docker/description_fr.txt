Conteneurisation Docker de niveau production pour les applications Python et Node.js.
Cette compétence doit être utilisée lorsque les utilisateurs demandent de conteneuriser des applications, de créer des Dockerfiles,
dockeriser des projets ou configurer Docker Compose. Détecte automatiquement la structure du projet,
analyse .env pour les secrets, valide la sécurité et génère des Dockerfiles testés.
