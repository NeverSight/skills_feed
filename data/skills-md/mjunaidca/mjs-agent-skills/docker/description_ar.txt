حاوية Docker على مستوى الإنتاج لتطبيقات Python وNode.js.
يجب استخدام هذه المهارة عندما يطلب المستخدمون وضع التطبيقات في حاويات، وإنشاء ملفات Dockerfiles،
إرساء المشاريع، أو إعداد Docker Compose. يكتشف تلقائيًا هيكل المشروع،
يقوم بتحليل .env بحثًا عن الأسرار والتحقق من صحة الأمان وإنشاء ملفات Dockerfiles التي تم اختبارها.
