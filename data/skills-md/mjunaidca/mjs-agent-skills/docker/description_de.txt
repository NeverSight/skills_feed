Docker-Containerisierung in Produktionsqualität für Python- und Node.js-Anwendungen.
Diese Fähigkeit sollte verwendet werden, wenn Benutzer Anwendungen in Container umwandeln, Docker-Dateien erstellen usw.
Dockerisieren Sie Projekte oder richten Sie Docker Compose ein. Erkennt automatisch die Projektstruktur,
analysiert .env auf Geheimnisse, validiert die Sicherheit und generiert getestete Docker-Dateien.
