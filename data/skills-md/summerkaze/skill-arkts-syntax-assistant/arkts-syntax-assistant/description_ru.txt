Руководство по синтаксису, миграции и оптимизации ArkTS для разработки HarmonyOS/OpenHarmony.
Запускается по: файлам .ets, ключевым словам ArkTS, контексту HarmonyOS/OpenHarmony, пакетам @ohos.
Помощь со справочником по синтаксису, миграцией TypeScript, оптимизацией производительности,
ошибки компиляции, управление состоянием, разработка компонентов и вопросы, связанные с языком.
