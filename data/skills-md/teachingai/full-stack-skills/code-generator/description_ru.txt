Предоставляет комплексные рекомендации по созданию кода, включая генерацию на основе шаблонов, создание шаблонов кода и автоматическое создание кода. Используйте, когда пользователь спрашивает о генерации кода, ему необходимо сгенерировать код из шаблонов, создать каркасы кода или автоматизировать создание кода.
