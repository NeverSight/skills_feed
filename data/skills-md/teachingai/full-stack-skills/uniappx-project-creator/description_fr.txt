Fournit la création de projets en une seule commande pour uni-app-x, y compris l'installation, la configuration et la génération de modèles de Vue 3 + TypeScript + Vite. À utiliser lorsque l'utilisateur demande à créer un projet uni-app-x avec une seule commande, doit initialiser un nouveau projet uni-app-x ou générer une structure de projet uni-app-x.
