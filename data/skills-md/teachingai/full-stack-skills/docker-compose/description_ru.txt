Предоставляет комплексное руководство по Docker Compose, включая многоконтейнерные приложения, определение сервисов, сети и тома. Используйте, когда пользователь спрашивает о Docker Compose, ему необходимо организовать несколько контейнеров, определить службы docker-compose или управлять многоконтейнерными приложениями.
