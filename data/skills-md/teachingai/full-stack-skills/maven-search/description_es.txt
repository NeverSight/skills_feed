Proporciona una guía completa para buscar y recuperar componentes de Maven desde el repositorio central de Maven (https://repo1.maven.org/maven2/). Esta habilidad permite buscar por ID de grupo, ID de artefacto, versión y otras coordenadas, recuperar metadatos de componentes (archivos POM, JAR, fuentes, Javadoc), consultar el historial de versiones y analizar dependencias. Úselo cuando el usuario necesite buscar, verificar o recuperar dependencias de Maven, verificar versiones de componentes, analizar árboles de dependencia o trabajar con coordenadas de Maven.
