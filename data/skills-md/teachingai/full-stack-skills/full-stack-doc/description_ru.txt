Предоставляет комплексные рекомендации по созданию документации по продуктам на китайском языке, включая создание, форматирование и передовые методы документации по продуктам. Используйте, когда пользователь спрашивает о документации по продукту на китайском языке, ему необходимо создать документацию по продукту или создать документацию на китайском языке.
