Une compétence complète pour utiliser agent-browser, un outil CLI pour l'automatisation du navigateur conçu pour les agents IA, développé par Vercel Labs. Cette compétence couvre l'installation, les commandes principales, les sélecteurs (réfs, CSS, XPath, localisateurs sémantiques), le mode agent, les sessions, les options et les meilleures pratiques. Utilisez cette compétence chaque fois que l'utilisateur a besoin d'automatiser les interactions du navigateur via des commandes CLI, en particulier pour les agents IA qui doivent interagir avec des pages Web.
