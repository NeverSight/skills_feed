Una habilidad integral para usar agente-navegador, una herramienta CLI para la automatización del navegador diseñada para agentes de IA, desarrollada por Vercel Labs. Esta habilidad cubre la instalación, los comandos principales, los selectores (referencias, CSS, XPath, localizadores semánticos), el modo de agente, las sesiones, las opciones y las mejores prácticas. Utilice esta habilidad siempre que el usuario necesite automatizar las interacciones del navegador mediante comandos CLI, especialmente para agentes de IA que necesiten interactuar con páginas web.
