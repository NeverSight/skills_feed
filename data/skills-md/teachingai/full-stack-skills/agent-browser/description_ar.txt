مهارة شاملة لاستخدام متصفح الوكيل، وهي أداة CLI لأتمتة المتصفح مصممة لعملاء الذكاء الاصطناعي، تم تطويرها بواسطة Vercel Labs. تغطي هذه المهارة التثبيت والأوامر الأساسية والمحددات (المراجع وCSS وXPath ومحددات المواقع الدلالية) ووضع الوكيل والجلسات والخيارات وأفضل الممارسات. استخدم هذه المهارة عندما يحتاج المستخدم إلى أتمتة تفاعلات المتصفح عبر أوامر واجهة سطر الأوامر (CLI)، خاصة لوكلاء الذكاء الاصطناعي الذين يحتاجون إلى التفاعل مع صفحات الويب.
