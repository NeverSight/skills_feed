Eine umfassende Fähigkeit zur Verwendung von Agent-Browser, einem von Vercel Labs entwickelten CLI-Tool zur Browser-Automatisierung für KI-Agenten. Diese Fertigkeit umfasst Installation, Kernbefehle, Selektoren (Refs, CSS, XPath, semantische Locators), Agentenmodus, Sitzungen, Optionen und Best Practices. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer Browserinteraktionen über CLI-Befehle automatisieren muss, insbesondere für KI-Agenten, die mit Webseiten interagieren müssen.
