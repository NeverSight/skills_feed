يوفر إرشادات شاملة لتطوير Vue 2.x بما في ذلك Options API والمكونات والتوجيهات وخطافات دورة الحياة والخصائص المحسوبة والمراقبين وإدارة حالة Vuex وVue Router. يُستخدم عندما يسأل المستخدم عن Vue 2، أو يحتاج إلى إنشاء مكونات Vue 2، أو تنفيذ ربط البيانات التفاعلية، أو التعامل مع اتصالات المكونات، أو العمل مع أدوات النظام البيئي Vue 2.
