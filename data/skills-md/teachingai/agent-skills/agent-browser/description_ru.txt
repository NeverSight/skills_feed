Комплексные навыки использования агента-браузера — инструмента CLI для автоматизации браузера, предназначенного для агентов искусственного интеллекта и разработанного Vercel Labs. Этот навык охватывает установку, основные команды, селекторы (ссылки, CSS, XPath, семантические локаторы), режим агента, сеансы, параметры и лучшие практики. Используйте этот навык всякий раз, когда пользователю необходимо автоматизировать взаимодействие с браузером с помощью команд CLI, особенно для агентов ИИ, которым необходимо взаимодействовать с веб-страницами.
