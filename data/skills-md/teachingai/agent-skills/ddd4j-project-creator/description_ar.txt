مهارة شاملة لإنشاء مشاريع ddd4j (التصميم المستند إلى المجال لـ Java) الجديدة استنادًا إلى إطار عمل ddd4j-boot. 
يُستخدم فقط عندما يذكر المستخدم صراحةً إنشاء مشروع ddd4j، أو تهيئة مشروع ddd4j، أو الإعداد 
مشروع التمهيد ddd4j. يدعم ثلاثة أنواع من المشاريع: متراصة أحادية الوحدة، متراصة متعددة الوحدات، والخدمات الصغيرة. 
إنشاء سقالات المشروع على أساس بنية ddd4j-boot مع DDD، والهندسة المعمارية السداسية، والهندسة المعمارية النظيفة، 
وأنماط كولا V5. لا تقم بتشغيل مشاريع DDD العامة أو مشاريع JPA/Hibernate أو أطر عمل ORM الأخرى 
بدون ذكر ddd4j بشكل صريح.
