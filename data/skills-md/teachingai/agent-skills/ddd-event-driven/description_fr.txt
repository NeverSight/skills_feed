Fournit des conseils complets pour l’architecture basée sur les événements, notamment les événements de domaine, le sourcing d’événements, le CQRS et les modèles d’événements. À utiliser lorsque l'utilisateur pose des questions sur l'architecture basée sur les événements, doit implémenter des systèmes pilotés par les événements ou travailler avec des événements de domaine.
