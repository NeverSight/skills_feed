Fornisce una guida completa per la ricerca e il recupero dei componenti Maven dal Maven Central Repository (https://repo1.maven.org/maven2/). Questa competenza consente la ricerca per groupId, artifactId, versione e altre coordinate, il recupero dei metadati dei componenti (file POM, JAR, origini, Javadoc), l'esecuzione di query sulla cronologia delle versioni e l'analisi delle dipendenze. Da utilizzare quando l'utente deve trovare, verificare o recuperare le dipendenze Maven, controllare le versioni dei componenti, analizzare gli alberi delle dipendenze o lavorare con le coordinate Maven.
