يوفر إرشادات شاملة للبحث عن مكونات Maven واسترجاعها من مستودع Maven المركزي (https://repo1.maven.org/maven2/). تتيح هذه المهارة البحث حسب معرف المجموعة، ومعرف الأداة، والإصدار، والإحداثيات الأخرى، واسترداد البيانات التعريفية للمكونات (ملفات POM، وJARs، والمصادر، وJavadoc)، والاستعلام عن سجل الإصدارات، وتحليل التبعيات. يُستخدم عندما يحتاج المستخدم إلى البحث عن تبعيات Maven أو التحقق منها أو استردادها، أو التحقق من إصدارات المكونات، أو تحليل أشجار التبعية، أو العمل مع إحداثيات Maven.
