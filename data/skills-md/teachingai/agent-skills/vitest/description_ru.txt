Предоставляет комплексное руководство по платформе тестирования Vitest, включая быстрое выполнение тестов, интеграцию Vite, тестирование компонентов, макетирование и настройку. Используйте, когда пользователь спрашивает о Vitest, ему необходимо написать быстрые модульные тесты, протестировать компоненты Vue/React или настроить Vitest с помощью проектов Vite.
