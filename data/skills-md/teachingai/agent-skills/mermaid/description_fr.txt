Fournit des conseils complets pour créer des diagrammes Mermaid. Mermaid est un outil de création de diagrammes et de graphiques basé sur JavaScript qui utilise des définitions de texte inspirées de Markdown et un moteur de rendu pour créer et modifier des diagrammes complexes. L'objectif principal de Mermaid est d'aider la documentation à suivre le développement. Mermaid est particulièrement adapté à une utilisation dans les documents Markdown, GitHub, GitLab, les wikis, les blogs et autres plateformes basées sur Markdown. À utiliser lorsque l'utilisateur souhaite dessiner, créer, générer, réaliser, construire ou visualiser un diagramme, un graphique, un organigramme, un diagramme d'architecture, un diagramme de séquence, un diagramme de classes, un diagramme d'état, un diagramme de Gantt, une carte mentale, une chronologie ou tout autre diagramme visuel au format compatible Markdown. Cette compétence couvre plus de 23 types de diagrammes Sirène, notamment les organigrammes, les diagrammes de séquence, les diagrammes de classes, les diagrammes d'état, les diagrammes ER, les diagrammes de Gantt, les diagrammes circulaires, les diagrammes à quadrants, les graphiques Git, les diagrammes C4, les cartes mentales, les chronologies, etc. Utilisez toujours cette compétence lorsque l'utilisateur mentionne Mermaid, a besoin de diagrammes pour la documentation Markdown ou souhaite des diagrammes rapides qui s'affichent directement dans les moteurs de rendu Markdown.
