Mermaid — это инструмент для создания диаграмм и диаграмм на основе JavaScript, который использует текстовые определения на основе Markdown и средство визуализации для создания и изменения сложных диаграмм. Основная цель Mermaid — помочь документации идти в ногу с развитием.
