Предоставляет комплексное руководство по платформе Angular, включая компоненты, модули, сервисы, внедрение зависимостей, маршрутизацию, формы и интеграцию TypeScript. Используйте, когда пользователь спрашивает об Angular, ему необходимо создавать приложения Angular, реализовывать компоненты Angular или работать с функциями Angular.
