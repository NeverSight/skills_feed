Предоставляет комплексные рекомендации по службе хранилища Azure, включая хранилище BLOB-объектов, общие файловые ресурсы, очереди и управление учетными записями хранения. Используйте, когда пользователь спрашивает о хранилище Azure, ему необходимо хранить данные в Azure, настраивать хранилище Azure или работать со службами хранилища Azure.
