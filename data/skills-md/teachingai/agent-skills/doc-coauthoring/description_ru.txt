Направляйте пользователей через структурированный рабочий процесс для совместного редактирования документации. Используйте, когда пользователь хочет написать документацию, предложения, технические спецификации, документы для принятия решений или аналогичный структурированный контент. Этот рабочий процесс помогает пользователям эффективно передавать контекст, уточнять контент посредством итерации и проверять, работает ли документ для читателей. Активируется, когда пользователь упоминает написание документации, создание предложений, составление спецификаций или аналогичные задачи по документированию.
