Fournit des conseils complets pour l'outil de gestion de base de données Navicat, y compris la connexion à la base de données, l'exécution de requêtes, la gestion des données et l'administration de la base de données. À utiliser lorsque l'utilisateur pose des questions sur Navicat, doit gérer des bases de données avec Navicat, exécuter des requêtes SQL ou utiliser les fonctionnalités de Navicat.
