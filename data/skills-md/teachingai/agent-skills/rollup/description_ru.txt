Предоставляет подробное руководство по сборщику Rollup, включая настройку, плагины, разделение кода, встряхивание дерева и объединение библиотек. Используйте, когда пользователь спрашивает о накопительном пакете, ему необходимо объединить библиотеки, оптимизировать выходные данные или настроить накопительный пакет для производственных сборок.
