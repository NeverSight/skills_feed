Предоставляет комплексные рекомендации по созданию чистой архитектуры, включая разделение слоев, правила зависимостей и архитектурные шаблоны. Используйте, когда пользователь спрашивает о чистой архитектуре, хочет реализовать принципы чистой архитектуры или структурировать приложения с чистой архитектурой.
