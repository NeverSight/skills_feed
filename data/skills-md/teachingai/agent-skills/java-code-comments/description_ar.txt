يوفر إرشادات شاملة لإضافة تعليقات تعليمات Java البرمجية وفقًا لمعايير الصناعة وأفضل الممارسات. 
تساعد هذه المهارة في إضافة تعليقات على مستوى الفصل الدراسي، وتعليقات على مستوى الطريقة، وتعليقات على مستوى الحقل إلى تعليمات Java البرمجية. 
يُستخدم عندما يريد المستخدم إضافة تعليقات إلى تعليمات Java البرمجية، ويحتاج إلى توثيق فئات/طرق/حقول Java، 
يريد تحسين وثائق التعليمات البرمجية، أو يحتاج إلى إنشاء تعليقات JavaDoc. تغطي هذه المهارة وحدة التحكم، 
الخدمة، ServiceImpl، Mapper، Model، Entity، BO (Business Object)، DTO، VO، وغيرها من تطبيقات Java الشائعة 
أنواع المكونات. تتبع المهارة سير عمل منهجي: مسح قاعدة التعليمات البرمجية، وتحديد المكونات، والإنشاء 
قائمة المهام، وإضافة التعليقات بالترتيب (تعليقات الفصل → تعليقات الطريقة → التعليقات الميدانية).
