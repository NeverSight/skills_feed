Fournit des conseils complets pour ajouter des commentaires de code Java conformément aux normes et bonnes pratiques du secteur. 
Cette compétence permet d'ajouter des commentaires au niveau de la classe, des commentaires au niveau de la méthode et des commentaires au niveau du champ au code Java. 
À utiliser lorsque l'utilisateur souhaite ajouter des commentaires au code Java, doit documenter les classes/méthodes/champs Java, 
souhaite améliorer la documentation du code ou doit générer des commentaires JavaDoc. Cette compétence couvre Contrôleur, 
Service, ServiceImpl, Mapper, Model, Entity, BO (Business Object), DTO, VO et autres Java courants 
types de composants. La compétence suit un flux de travail systématique : analyser la base de code, identifier les composants, créer 
liste de tâches et ajoutez des commentaires dans l'ordre (commentaires de classe → commentaires de méthode → commentaires de champ).
