Démarrez une nouvelle session d’apprentissage sur un sujet. À utiliser lorsque l'utilisateur souhaite apprendre quelque chose de nouveau, commencer à étudier, apprendre un sujet ou démarrer une conversation pédagogique. Déclencheurs sur « apprends-moi », « apprenons », « démarrer la session », « étudier [sujet] », « expliquer [sujet] ».
