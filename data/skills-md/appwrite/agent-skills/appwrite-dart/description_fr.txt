Compétence Appwrite Dart SDK. À utiliser lors de la création d'applications Flutter (mobiles, Web, de bureau) ou d'applications Dart côté serveur avec Appwrite. Couvre l'authentification côté client (e-mail, OAuth), les requêtes de base de données, les téléchargements de fichiers avec gestion native des fichiers, les abonnements en temps réel et l'administration côté serveur via des clés API pour la gestion des utilisateurs, l'administration de la base de données, le stockage et les fonctions.
