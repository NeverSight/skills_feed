Compétence Appwrite Swift SDK. À utiliser lors de la création d'applications natives iOS, macOS, watchOS ou tvOS, ou d'applications Swift côté serveur avec Appwrite. Couvre l'authentification côté client (e-mail, OAuth), les requêtes de base de données, les téléchargements de fichiers, les abonnements en temps réel avec async/await et l'administration côté serveur via des clés API pour la gestion des utilisateurs, l'administration de la base de données, le stockage et les fonctions.
