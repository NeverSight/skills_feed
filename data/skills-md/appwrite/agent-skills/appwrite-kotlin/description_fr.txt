Compétence Appwrite Kotlin SDK. À utiliser lors de la création d'applications Android natives ou de backends Kotlin/JVM côté serveur avec Appwrite. Couvre l'authentification côté client (e-mail, OAuth avec intégration d'activité), les requêtes de base de données, les téléchargements de fichiers, les abonnements en temps réel avec prise en charge de coroutine et l'administration côté serveur via des clés API pour la gestion des utilisateurs, l'administration de la base de données, le stockage et les fonctions.
