Appwrite مهارة TypeScript SDK. يُستخدم عند إنشاء تطبيقات JavaScript/TypeScript قائمة على المتصفح، أو تطبيقات React Native للهواتف المحمولة، أو واجهات Node.js/Deno الخلفية من جانب الخادم مع Appwrite. يغطي المصادقة من جانب العميل (البريد الإلكتروني، OAuth، المجهول)، واستعلامات قاعدة البيانات، وتحميل الملفات، والاشتراكات في الوقت الفعلي، والإدارة من جانب الخادم عبر مفاتيح API لإدارة المستخدم، وإدارة قاعدة البيانات، والتخزين، والوظائف.
