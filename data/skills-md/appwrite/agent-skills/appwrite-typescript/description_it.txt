Competenza SDK Appwrite TypeScript. Da utilizzare durante la creazione di app JavaScript/TypeScript basate su browser, app mobili React Native o backend Node.js/Deno lato server con Appwrite. Copre l'autenticazione lato client (e-mail, OAuth, anonima), query di database, caricamenti di file, abbonamenti in tempo reale e amministrazione lato server tramite chiavi API per la gestione degli utenti, l'amministrazione del database, l'archiviazione e le funzioni.
