أتمتة iOS Simulator باستخدام ax CLI لإيماءات اللمس وإدخال النص وأزرار الأجهزة ولقطات الشاشة وتسجيل الفيديو وفحص إمكانية الوصول. يُستخدم عند أتمتة تفاعلات iOS Simulator، أو كتابة اختبارات واجهة المستخدم، أو التقاط لقطات الشاشة/الفيديو، أو فحص عناصر إمكانية الوصول. المشغلات في التشغيل الآلي لـ iOS Simulator، أو استخدام واجهة سطر الأوامر (CLI)، أو أوامر النقر/التمرير/الإيماءة للمحاكاة، أو مهام اختبار إمكانية الوصول.
