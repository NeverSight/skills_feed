Безопасные шаблоны CLI 1Password для чтения секретов, обнаружения хранилищ/предметов и передачи учетных данных другим инструментам. Используйте при чтении из 1Password, ротации секретов или передаче учетных данных в wrangler/kubectl/и т. д. Запускается по операциям CLI, 1Password, секретной ротации или задачам конвейера учетных данных.
