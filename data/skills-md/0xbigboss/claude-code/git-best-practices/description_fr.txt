Modèles de workflow Git pour les validations, les branchements, les relations publiques et la gestion de l'historique dans des référentiels hétérogènes. À utiliser lors de la création de commits, de la gestion de branches, de l'ouverture de demandes d'extraction ou de la réécriture de l'historique. Ne pas utiliser pour des tâches d'implémentation non-git ou des décisions de politique de publication spécifiques au dépôt sans documentation du référentiel.
