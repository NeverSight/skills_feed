Patrones de flujo de trabajo de Git para confirmaciones, bifurcaciones, relaciones públicas y gestión del historial en repositorios heterogéneos. Úselo al crear confirmaciones, administrar sucursales, abrir solicitudes de extracción o reescribir el historial. No lo use para tareas de implementación que no sean de git o decisiones de políticas de lanzamiento específicas del repositorio sin documentación del repositorio.
