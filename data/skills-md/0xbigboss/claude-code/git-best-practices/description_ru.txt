Шаблоны рабочих процессов Git для коммитов, ветвления, PR и управления историей в гетерогенных репозиториях. Используйте при создании коммитов, управлении ветвями, открытии запросов на включение или переписывании истории. Не используйте для задач реализации, не связанных с Git, или для принятия решений по политике выпуска, специфичных для репозитория, без документации репозитория.
