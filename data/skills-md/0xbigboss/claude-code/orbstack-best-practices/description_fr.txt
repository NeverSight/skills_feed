Modèles pour les machines virtuelles OrbStack Linux et Docker sur macOS. Couvre les commandes orbctl/orb, le cycle de vie de la machine, l'initialisation du cloud, la mise en réseau, le partage de fichiers et l'accès SSH. Doit être utilisé lorsque vous travaillez avec OrbStack, les commandes orbctl ou les machines virtuelles Linux sur macOS.
