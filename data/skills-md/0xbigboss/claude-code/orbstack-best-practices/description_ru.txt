Шаблоны для виртуальных машин OrbStack Linux и Docker на macOS. Охватывает команды orbctl/orb, жизненный цикл машины, облачную инициализацию, работу в сети, общий доступ к файлам и доступ по SSH. Необходимо использовать при работе с OrbStack, командами orbctl или виртуальными машинами Linux в macOS.
