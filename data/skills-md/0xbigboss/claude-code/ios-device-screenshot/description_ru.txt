Делайте снимки экрана с физических устройств iOS, подключенных через USB, с помощью pymobiledevice3. Используйте при съемке снимков экрана с реальных iPhone/iPad (не симуляторов), отладке на устройстве или при необходимости высококачественного захвата устройства. Запускается по снимкам экрана физического устройства iOS, использованию pymobiledevice3 или задачам захвата устройства, подключенного через USB.
