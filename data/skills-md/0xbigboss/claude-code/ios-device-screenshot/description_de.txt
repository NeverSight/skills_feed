Machen Sie mit pymobiledevice3 Screenshots von physischen iOS-Geräten, die über USB angeschlossen sind. Verwenden Sie diese Option, wenn Sie Screenshots von echten iPhones/iPads (keine Simulatoren) aufnehmen, auf dem Gerät debuggen oder Geräteaufnahmen mit hoher Wiedergabetreue benötigen. Wird bei Screenshots von physischen iOS-Geräten, bei der Nutzung von pymobiledevice3 oder bei Erfassungsaufgaben für über USB angeschlossene Geräte ausgelöst.
