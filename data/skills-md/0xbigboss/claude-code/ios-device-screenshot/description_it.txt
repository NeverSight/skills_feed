Acquisisci screenshot da dispositivi iOS fisici collegati tramite USB utilizzando pymobiledevice3. Da utilizzare quando si acquisiscono screenshot da iPhone/iPad reali (non simulatori), si esegue il debug sul dispositivo o si necessitano acquisizioni di dispositivi ad alta fedeltà. Si attiva sugli screenshot del dispositivo iOS fisico, sull'utilizzo di pymobiledevice3 o sulle attività di acquisizione del dispositivo connesso via USB.
