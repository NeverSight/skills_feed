Anleitung zum Generieren von Commit-Meldungen auf Spanisch, gefolgt von herkömmlichen Commits mit Emojis. Sie müssen wissen, dass der Benutzer ein Commit-Konto erstellt, eine Commit-Nachricht generiert, Änderungen vorgenommen hat, oder sobald Sie einen Codewechsel abgeschlossen haben und den Moment haben, ihn auf Git zu registrieren. Sie werden auch mit „commit“, „commitear“, „guardar cambios“, „registrar cambios“ aktiviert oder wollen in der Git-Historie einen Punkt erstellen.
