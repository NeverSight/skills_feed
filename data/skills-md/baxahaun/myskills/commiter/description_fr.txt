Guide pour générer des messages de commit en espagnol suivant Conventional Commits limité aux Emojis. Utilisez cette compétence toujours pour que l'utilisateur fasse un commit, génère un message de commit, commette des changements, ou lorsqu'il complète un changement de code et s'enregistre à ce moment-là dans git. Il est également activé avec 'commit', 'commitear', 'guardar change', 'registrar change' ou toute autre intention de créer un point dans l'historique de git.
