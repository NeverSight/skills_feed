Bereitstellen und Festschreiben von Änderungen mit einer gut gestalteten Festschreibungsnachricht. TRIGGER, wenn: Der Benutzer fragt, ob er Änderungen an Git festschreiben, bereitstellen oder speichern möchte. Beispiele: „dieses festschreiben“, „die Änderungen festschreiben“, „bereitstellen und festschreiben“. NICHT AUSLÖSEN, wenn: Der Benutzer fragt nach einem Push, einem PR oder einer Änderung eines vorherigen Commits.
