قم بتحليل محادثات ترميز الذكاء الاصطناعي متعددة المنعطفات لإنشاء مطالبات محسّنة لمرة واحدة تلتقط جوهر العمل التكراري. 
يُستخدم عندما: (1) استغرقت المحادثة عدة تكرارات لتحقيق النتيجة المرجوة، (2) إذا كنت تريد إنشاء مطالبة قابلة لإعادة الاستخدام لمهام مستقبلية مماثلة، 
(3) أنت بحاجة إلى فهم المعلومات المفقودة من الطلب الأولي، (4) التخطيط لمهمة معقدة والرغبة في طرح أسئلة توضيحية مقدمًا.
يدعم بيانات جلسة OpenCode وClaude Code وGemini CLI وCodex CLI.
