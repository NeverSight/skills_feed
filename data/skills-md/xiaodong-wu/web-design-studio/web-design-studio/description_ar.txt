قم بإنشاء صفحات ويب كاملة مع تصميم الواجهة الأمامية على مستوى الإنتاج والصور التي يتم إنشاؤها تلقائيًا. استخدم هذه المهارة عندما يطلب المستخدم إنشاء مواقع ويب أو صفحات مقصودة أو لوحات معلومات أو أي واجهات ويب تحتاج إلى صور. تجمع هذه المهارة بين خبرة تصميم الواجهة الأمامية والتوليد التلقائي لصور الذكاء الاصطناعي لإنشاء صفحات ويب كاملة الوظائف ومذهلة بصريًا مع تضمين جميع الصور. ينشئ كود HTML/CSS/JS مبتكر ومصقول مع صور تم إنشاؤها بواسطة الذكاء الاصطناعي والتي تتوافق مع الاتجاه الجمالي.
