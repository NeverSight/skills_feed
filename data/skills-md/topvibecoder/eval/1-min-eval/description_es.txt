Evaluación rápida de la base de código utilizando Claude CLI. Escanea el código para extraer el árbol de repositorio y los archivos fuente, luego ejecuta evaluaciones de métricas paralelas (impacto, técnica, creatividad, presentación, diseño_indicación) a través de claude -p. Se envía automáticamente a la API de clasificación de TopVibeCoder y realiza un seguimiento del progreso a lo largo del tiempo. Úselo al evaluar aplicaciones LLM, proyectos de hackathon o calidad del código.
