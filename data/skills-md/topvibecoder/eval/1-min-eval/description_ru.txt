Быстрая оценка кодовой базы с использованием Claude CLI. Сканирует код для извлечения дерева репозитория и исходных файлов, затем запускает параллельные оценки показателей (воздействие, технические характеристики, креативность, презентация, Prompt_design) с помощью claude -p. Автоматически отправляет данные в API рейтинга TopVibeCoder и отслеживает прогресс с течением времени. Используйте при оценке приложений LLM, проектов хакатонов или качества кода.
