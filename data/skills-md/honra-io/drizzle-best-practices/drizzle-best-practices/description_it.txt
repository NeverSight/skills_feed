Best practice di Drizzle ORM per progetti TypeScript che utilizzano PostgreSQL. Utilizza questa competenza durante la scrittura, la revisione o l'ottimizzazione di schemi, query, relazioni, migrazioni o configurazioni di database Drizzle ORM destinati a Postgres. Applica queste linee guida ogni volta che vedi importazioni di drizzle-orm, definizioni di pgTable, file di configurazione di drizzle-kit o modelli di query relazionali. Da utilizzare anche durante l'impostazione di un nuovo progetto con Drizzle + Postgres, la migrazione da un altro ORM o la risoluzione di errori di tipo negli schemi Drizzle.
