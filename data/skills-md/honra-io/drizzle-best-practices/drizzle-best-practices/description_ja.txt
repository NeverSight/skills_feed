PostgreSQL を使用した TypeScript プロジェクトの Drizzle ORM ベスト プラクティス。このスキルは、Postgres を対象とした Drizzle ORM スキーマ、クエリ、関係、移行、またはデータベース構成を作成、レビュー、または最適化するときに使用します。 drizzle-orm インポート、pgTable 定義、drizzle-kit 構成ファイル、またはリレーショナル クエリ パターンを参照する場合は常に、これらのガイドラインを適用してください。 Drizzle + Postgres を使用して新しいプロジェクトをセットアップする場合、別の ORM から移行する場合、または Drizzle スキーマのタイプ エラーのトラブルシューティングを行う場合にも使用します。
