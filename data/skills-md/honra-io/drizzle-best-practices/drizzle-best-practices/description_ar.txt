أفضل ممارسات Drizzle ORM لمشاريع TypeScript باستخدام PostgreSQL. استخدم هذه المهارة عند كتابة أو مراجعة أو تحسين مخططات Drizzle ORM أو الاستعلامات أو العلاقات أو عمليات الترحيل أو تكوينات قاعدة البيانات التي تستهدف Postgres. قم بتطبيق هذه الإرشادات عندما ترى عمليات استيراد drizzle-orm أو تعريفات pgTable أو ملفات تكوين drizzle-kit أو أنماط الاستعلام العلائقية. استخدمه أيضًا عند إعداد مشروع جديد باستخدام Drizzle + Postgres، أو الترحيل من ORM آخر، أو استكشاف أخطاء الكتابة وإصلاحها في مخططات Drizzle.
