PostgreSQL을 사용하는 TypeScript 프로젝트에 대한 ORM 모범 사례를 드리즐합니다. Postgres를 대상으로 하는 Drizzle ORM 스키마, 쿼리, 관계, 마이그레이션 또는 데이터베이스 구성을 작성, 검토 또는 최적화할 때 이 기술을 사용하세요. drizzle-orm 가져오기, pgTable 정의, drizzle-kit 구성 파일 또는 관계형 쿼리 패턴을 볼 때마다 이러한 지침을 적용하세요. 또한 Drizzle + Postgres를 사용하여 새 프로젝트를 설정하거나, 다른 ORM에서 마이그레이션하거나, Drizzle 스키마의 유형 오류를 해결할 때 사용합니다.
