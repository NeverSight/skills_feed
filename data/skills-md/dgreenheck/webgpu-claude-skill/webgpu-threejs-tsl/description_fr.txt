Guide complet pour développer des applications Three.js compatibles WebGPU à l'aide de TSL (Three.js Shading Language). Couvre la configuration du moteur de rendu WebGPU, la syntaxe TSL et les matériaux de nœuds, les shaders de calcul, les effets de post-traitement et l'intégration WGSL. Utilisez cette compétence lorsque vous travaillez avec le WebGPU Three.js, les shaders TSL, les matériaux de nœuds ou le calcul GPU dans Three.js.
