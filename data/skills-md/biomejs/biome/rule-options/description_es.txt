Guía para implementar opciones configurables para reglas y asistencias de pelusa. Úselo cuando las reglas necesiten un comportamiento configurable por el usuario. Ejemplos:<example>El usuario quiere agregar opciones a una regla lint</example><example>El usuario necesita implementar la deserialización JSON para la configuración de la regla</example><example>El usuario está probando el comportamiento de la regla con diferentes opciones</example>
