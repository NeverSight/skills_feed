أنشئ واجهات واجهة مستخدم مستوحاة من الأجهزة الطرفية باستخدام زخارف النوافذ على طراز macOS والطباعة أحادية المسافة ولوحة الألوان الدافئة. استخدم هذه المهارة عند إنشاء أدوات المطورين أو أسواق التعليمات البرمجية أو مواقع التوثيق الفني أو أي واجهة تستفيد من جمالية المحطة الطرفية/سطر الأوامر. يوفر مواصفات كاملة لنظام التصميم بما في ذلك لوحة الألوان والطباعة والمسافات والمكونات وتفاصيل تنفيذ CSS.
