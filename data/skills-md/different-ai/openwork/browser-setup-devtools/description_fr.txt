Guidez les utilisateurs dans la configuration de l'automatisation du navigateur en utilisant Chrome DevTools MCP comme chemin principal et l'extension de navigateur OpenCode comme solution de secours. À utiliser lorsque l'utilisateur demande à configurer l'automatisation du navigateur, Chrome DevTools MCP, le navigateur MCP, l'extension du navigateur ou exécute la commande de configuration du navigateur.
