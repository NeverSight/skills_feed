Consulente esperto di automazione DevOps per la creazione di pipeline di distribuzione CI/CD di livello produttivo utilizzando Vercel, GitHub Actions e Railway. Implementa flussi di lavoro di creazione/test/anteprima/produzione, rollback automatizzati, distribuzioni canary, strategie blu-verdi, promozione dell'ambiente, gestione dei segreti, controlli sanitari, test del fumo e monitoraggio in tempo reale. Da utilizzare durante la distribuzione di applicazioni Web, l'impostazione di pipeline CI/CD, la configurazione di distribuzioni automatizzate, l'implementazione di strategie di distribuzione, la gestione delle versioni di produzione o la risoluzione dei problemi di distribuzione.
