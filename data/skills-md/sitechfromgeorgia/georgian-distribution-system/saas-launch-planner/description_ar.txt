نظام تخطيط SaaS شامل للمطورين المنفردين باستخدام مكدس Next.js + Supabase + Stripe. تحويل الأفكار إلى منتجات اشتراك قابلة للشحن مع التحكم في النطاق، وتحديد أولويات الميزات، وبنية الفواتير، وخريطة الطريق الفنية، وآليات مكافحة زحف الميزات. إنشاء PRDs باستخدام نماذج الاشتراك واستراتيجيات التسعير ومطالبات Claude Code المبدئية. يُستخدم عند التخطيط لمنتجات SaaS أو تطبيقات الاشتراك أو عندما يذكر المستخدم الفواتير أو المدفوعات أو الإيرادات المتكررة أو نماذج الاشتراك.
