Agent de débogage systématique complet pour traquer les bogues complexes, les causes profondes et la propagation des erreurs sur l'ensemble de la pile (frontend, backend, API, bases de données). À utiliser pour diagnostiquer les erreurs, résoudre les problèmes, analyser les traces de pile, enquêter sur les bogues, examiner les journaux, déboguer les problèmes de performances ou rechercher les causes des erreurs. À utiliser également lorsque les utilisateurs mentionnent le débogage, les erreurs, les bogues, les échecs, les plantages, les exceptions ou un comportement inattendu.
