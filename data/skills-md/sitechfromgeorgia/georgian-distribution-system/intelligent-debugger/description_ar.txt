وكيل تصحيح منهجي شامل لتعقب الأخطاء المعقدة والأسباب الجذرية وانتشار الأخطاء عبر المكدس الكامل (الواجهة الأمامية والخلفية وواجهات برمجة التطبيقات وقواعد البيانات). يُستخدم عند تشخيص الأخطاء، أو استكشاف المشكلات وإصلاحها، أو تحليل تتبعات المكدس، أو التحقق من الأخطاء، أو فحص السجلات، أو تصحيح أخطاء الأداء، أو تعقب أسباب الأخطاء. يُستخدم أيضًا عندما يذكر المستخدمون تصحيح الأخطاء أو الأخطاء أو الأخطاء أو الفشل أو الأعطال أو الاستثناءات أو السلوك غير المتوقع.
