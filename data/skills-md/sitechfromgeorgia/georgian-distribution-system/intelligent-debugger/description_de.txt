Umfassender systematischer Debugging-Agent zur Suche nach komplexen Fehlern, Grundursachen und Fehlerausbreitung über den gesamten Stack (Frontend, Backend, APIs, Datenbanken). Wird zum Diagnostizieren von Fehlern, Beheben von Problemen, Analysieren von Stack-Traces, Untersuchen von Fehlern, Untersuchen von Protokollen, Debuggen von Leistungsproblemen oder Aufspüren von Fehlerursachen verwendet. Verwenden Sie es auch, wenn Benutzer Debugging, Fehler, Bugs, Ausfälle, Abstürze, Ausnahmen oder unerwartetes Verhalten erwähnen.
