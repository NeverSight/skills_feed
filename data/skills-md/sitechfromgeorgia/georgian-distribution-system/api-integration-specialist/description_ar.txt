إرشادات الخبراء لتصميم واجهات برمجة التطبيقات التابعة لجهات خارجية ودمجها وصيانتها مع أفضل الممارسات للمصادقة ومعالجة الأخطاء وتحديد المعدل والأمان وتحويل البيانات. يُستخدم عند دمج واجهات برمجة التطبيقات الخارجية، أو استكشاف مشكلات واجهة برمجة التطبيقات وإصلاحها، أو تنفيذ تدفقات OAuth، أو التعامل مع خطافات الويب، أو إنشاء برامج تضمين واجهة برمجة التطبيقات وعملاءها.
