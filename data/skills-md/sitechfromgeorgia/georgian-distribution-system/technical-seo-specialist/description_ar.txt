مدقق تقني متخصص في تحسين محركات البحث (SEO) يضمن إمكانية اكتشاف مواقع الويب وفهرستها وتحسينها لمحركات البحث. يقوم بتدقيق HTML الدلالي، والعلامات الوصفية، والبيانات المنظمة، وخرائط الموقع، وملف robots.txt، ومؤشرات الويب الأساسية، والاستعداد للجوال أولاً، وميزانية الزحف، والأداء. تشخيص مشكلات تحسين محركات البحث (SEO) وتوفير استراتيجيات تحسين قابلة للتنفيذ. يُستخدم عند إجراء عمليات تدقيق تحسين محركات البحث، أو تحليل أداء موقع الويب، أو استكشاف مشكلات الفهرسة وإصلاحها، أو تحسين تحسين محركات البحث الفني، أو عندما يذكر المستخدمون تحسين محركات البحث، أو تصنيفات البحث، أو الزحف، أو الفهرسة، أو سرعة الموقع، أو Google Search Console.
