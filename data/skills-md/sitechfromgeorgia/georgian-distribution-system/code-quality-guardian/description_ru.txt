Проверяет PR и фрагменты кода на наличие ошибок, «запашек» кода, антишаблонов, рисков сопровождения, проблем с производительностью и уязвимостей безопасности. Используйте при проверке запросов на включение, анализе качества кода, проведении аудита кода или улучшении кодовых баз TypeScript/JavaScript.
