Créez et modifiez des fichiers de spécification de flux de travail Arazzo (.arazzo.yaml) pour les tests et l'orchestration des API. À utiliser lors de la création de workflows de test d'API, de la définition de séquences d'API en plusieurs étapes, de l'écriture de tests d'intégration d'API ou de la documentation des modèles d'utilisation des API. Les déclencheurs incluent : « arazzo », « spécifications du flux de travail », « flux de travail API », « test API en plusieurs étapes », « .arazzo.yaml », « appels d'orchestre » ou toute demande visant à décrire des flux de travail API avec des assertions.
