Создавайте и редактируйте файлы спецификации рабочего процесса Arazzo (.arazzo.yaml) для тестирования и оркестрации API. Используйте при построении рабочих процессов тестирования API, определении многоэтапных последовательностей API, написании интеграционных тестов API или документировании шаблонов использования API. Триггеры включают в себя: «arazzo», «спецификация рабочего процесса», «рабочий процесс API», «многоэтапный тест API», «.arazzo.yaml», «оркестрация вызовов» или любой запрос на описание рабочих процессов API с утверждениями.
