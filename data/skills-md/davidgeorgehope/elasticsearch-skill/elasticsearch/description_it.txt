Interagisci con Elasticsearch e Kibana tramite API REST utilizzando curl. Da utilizzare durante l'esecuzione di query, l'indicizzazione, la gestione degli indici, il controllo dell'integrità del cluster, la scrittura di aggregazioni, la distribuzione di dashboard o la risoluzione dei problemi di Elasticsearch. Richiede l'URL del cluster e la chiave API. Copre: ricerca (Query DSL), operazioni CRUD, gestione degli indici, mappature, aggregazioni, integrità del cluster, ILM, ES|QL, API Kibana (dashboard, visualizzazioni dati, oggetti salvati), modelli di dati OpenTelemetry e modelli comuni di risoluzione dei problemi.
