Interactúe con Elasticsearch y Kibana a través de la API REST usando curl. Úselo al realizar consultas, indexar, administrar índices, verificar el estado del clúster, escribir agregaciones, implementar paneles o solucionar problemas de Elasticsearch. Requiere URL del clúster y clave API. Cubre: búsqueda (Query DSL), operaciones CRUD, administración de índices, asignaciones, agregaciones, estado del clúster, ILM, ES|QL, API de Kibana (paneles, vistas de datos, objetos guardados), patrones de datos de OpenTelemetry y patrones de solución de problemas comunes.
