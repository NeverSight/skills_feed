Взаимодействуйте с Elasticsearch и Kibana через REST API, используя Curl. Используйте при запросе, индексировании, управлении индексами, проверке работоспособности кластера, написании агрегатов, развертывании информационных панелей или устранении неполадок Elasticsearch. Требуется URL-адрес кластера и ключ API. Охватывает: поиск (Query DSL), операции CRUD, управление индексами, сопоставления, агрегации, работоспособность кластера, ILM, ES|QL, Kibana API (панели мониторинга, представления данных, сохраненные объекты), шаблоны данных OpenTelemetry и общие шаблоны устранения неполадок.
