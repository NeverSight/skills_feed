تفاعل مع Elasticsearch وKibana عبر REST API باستخدام cur. يُستخدم عند الاستعلام أو الفهرسة أو إدارة المؤشرات أو التحقق من صحة المجموعة أو كتابة التجميعات أو نشر لوحات المعلومات أو استكشاف أخطاء Elasticsearch وإصلاحها. يتطلب عنوان URL للمجموعة ومفتاح API. يغطي: البحث (الاستعلام DSL)، وعمليات CRUD، وإدارة الفهرس، والتعيينات، والتجميعات، وصحة المجموعة، وILM، وES|QL، وKibana API (لوحات المعلومات، وطرق عرض البيانات، والكائنات المحفوظة)، وأنماط بيانات OpenTelemetry، وأنماط استكشاف الأخطاء وإصلاحها الشائعة.
