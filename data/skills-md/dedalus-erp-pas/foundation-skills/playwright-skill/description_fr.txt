Automatisation complète du navigateur et tests Web avec Playwright. Détecte automatiquement les serveurs de développement, gère le cycle de vie du serveur, écrit des scripts de test propres dans /tmp. Testez les pages, remplissez des formulaires, prenez des captures d'écran, vérifiez la conception réactive, validez l'UX, testez les flux de connexion, vérifiez les liens, déboguez les applications Web dynamiques, automatisez n'importe quelle tâche du navigateur. À utiliser lorsque l'utilisateur souhaite tester des sites Web, automatiser les interactions du navigateur, valider les fonctionnalités Web ou effectuer des tests basés sur le navigateur.
