Umfassender Best Practices-Leitfaden für React und Next.js mit Themen zur Leistungsoptimierung, Komponentenarchitektur, Shadcn/UI-Mustern, Bewegungsanimationen und modernen React 19+-Mustern. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von React/Next.js-Code verwendet werden. Wird bei Aufgaben ausgelöst, die React-Komponenten, Next.js-Seiten, Datenabruf, UI-Komponenten, Animationen oder Codequalitätsverbesserungen umfassen.
