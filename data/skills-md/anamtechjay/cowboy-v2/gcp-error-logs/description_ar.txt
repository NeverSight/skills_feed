جلب وتحليل سجلات الأخطاء من Google Cloud Logging for Cloud Functions. تجميع الأخطاء حسب النمط، وتحديد التكرار، وتوفير رؤى السبب الجذري. يُستخدم عند تصحيح أخطاء Google Cloud، أو التحقيق في حالات فشل Cloud Functions، أو تحليل مشكلات الإنتاج، أو عندما يذكر المستخدم سجلات Google Cloud، أو أخطاء GCP، أو مشكلات Cloud Functions.
