العمل مع MongoDB (قاعدة بيانات المستندات، ومستندات BSON، وخطوط أنابيب التجميع، وسحابة Atlas) وPostgreSQL (قاعدة بيانات علائقية، واستعلامات SQL، وpsql CLI، وpgAdmin). يُستخدم عند تصميم مخططات قاعدة البيانات، أو كتابة الاستعلامات والتجميعات، أو تحسين الفهارس للأداء، أو تنفيذ عمليات ترحيل قاعدة البيانات، أو تكوين النسخ المتماثل والتقسيم، أو تنفيذ إستراتيجيات النسخ الاحتياطي والاستعادة، أو إدارة مستخدمي قاعدة البيانات والأذونات، أو تحليل أداء الاستعلام، أو إدارة قواعد بيانات الإنتاج. | يحتوي هذا المستند على أحدث البيانات وقاعدة البيانات وSQL والاستعلام والخادم الحقيقي والمخطط والترحيل.
