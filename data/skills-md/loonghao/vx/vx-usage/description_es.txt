Esta habilidad enseña a los agentes de codificación de IA cómo utilizar vx, el administrador de herramientas de desarrollo universal.
Utilice esta habilidad siempre que el proyecto del usuario utilice vx (tenga el directorio vx.toml o .vx/),
o cuando el usuario menciona vx, gestión de versiones de herramientas o configuración de desarrollo multiplataforma.
vx administra de forma transparente Node.js, Python, Go, Rust y más de 50 herramientas más con configuración cero.
