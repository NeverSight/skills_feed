Интеграция X (Twitter) API v2 для публикации твитов, ответов, цитат, управления публикациями, отправки личных сообщений, поиска твитов и получения временной шкалы/активности пользователей.

Используйте, когда пользователю необходимо взаимодействовать с API X (Twitter):
- Контент публикации: твиты, ответы, цитаты, публикации в СМИ.
- Управляйте публикациями: удаляйте, ретвитьте, ставьте лайки.
- Прямые сообщения: отправляйте личные сообщения с дополнительными мультимедийными вложениями.
- Поиск: поиск твитов по ключевому слову, хэштегу, пользователю, с фильтрами.
- Получение данных: недавняя активность пользователя, домашняя временная шкала.
- Извлечение идентификаторов твитов из URL-адресов.

Требуются учетные данные OAuth 1.0a: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_SECRET.
