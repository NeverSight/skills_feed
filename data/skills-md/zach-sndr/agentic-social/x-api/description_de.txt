X (Twitter) API v2-Integration zum Posten von Tweets, Antworten, Zitaten, Verwalten von Beiträgen, Senden von DMs, Durchsuchen von Tweets und Abrufen der Zeitleiste/Benutzeraktivität.

Wird verwendet, wenn der Benutzer mit der X-API (Twitter) interagieren muss:
- Inhalte posten: Tweets, Antworten, Zitate, Beiträge mit Medien
- Beiträge verwalten: löschen, retweeten, liken
- Direktnachrichten: Senden Sie Direktnachrichten mit optionalen Medienanhängen
- Suche: Durchsuchen Sie Tweets nach Schlüsselwort, Hashtag, Benutzer und Filtern
- Daten abrufen: letzte Aktivität des Benutzers, Home-Timeline
- Extrahieren Sie Tweet-IDs aus URLs

Erfordert OAuth 1.0a-Anmeldeinformationen: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_SECRET.
