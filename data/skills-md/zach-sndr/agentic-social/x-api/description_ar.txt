تكامل X (Twitter) API v2 لنشر التغريدات والردود والاقتباسات وإدارة المنشورات وإرسال الرسائل المباشرة والبحث في التغريدات واسترداد الجدول الزمني/نشاط المستخدم.

يُستخدم عندما يحتاج المستخدم إلى التفاعل مع X (Twitter) API:
- نشر المحتوى: التغريدات والردود والاقتباسات والمشاركات مع الوسائط
- إدارة المنشورات: حذف، إعادة تغريد، إعجاب
- الرسائل المباشرة: أرسل رسائل مباشرة مع مرفقات الوسائط الاختيارية
- البحث: ابحث في التغريدات حسب الكلمة الرئيسية، الهاشتاج، المستخدم، باستخدام المرشحات
- استرداد البيانات: نشاط المستخدم الأخير، الجدول الزمني للمنزل
- استخراج معرفات التغريدة من عناوين URL

يتطلب بيانات اعتماد OAuth 1.0a: X_API_KEY، X_API_SECRET، X_ACCESS_TOKEN، X_ACCESS_SECRET.
