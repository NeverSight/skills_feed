트윗 게시, 답글, 인용문, 게시물 관리, DM 보내기, 트윗 검색, 타임라인/사용자 활동 검색을 위한 X(Twitter) API v2 통합입니다.

사용자가 X(Twitter) API와 상호작용해야 할 때 사용합니다.
- 게시 내용: 트윗, 답글, 인용문, 미디어가 포함된 게시물
- 게시물 관리 : 삭제, 리트윗, 좋아요
- 다이렉트 메시지: 옵션 미디어 첨부 파일을 사용하여 DM을 보냅니다.
- 검색: 키워드, 해시태그, 사용자, 필터로 트윗을 검색합니다.
- 데이터 검색: 사용자의 최근 활동, 홈 타임라인
- URL에서 트윗 ID 추출

OAuth 1.0a 자격 증명이 필요합니다: X_API_KEY, X_API_SECRET, X_ACCESS_TOKEN, X_ACCESS_SECRET.
