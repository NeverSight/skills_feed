ツイートの投稿、返信、引用、投稿の管理、DM の送信、ツイートの検索、タイムライン/ユーザー アクティビティの取得のための X (Twitter) API v2 統合。

ユーザーが X (Twitter) API と対話する必要がある場合に使用します。
- 投稿内容: ツイート、返信、引用、メディアへの投稿
- 投稿の管理: 削除、リツイート、いいね！
- ダイレクトメッセージ: オプションのメディア添付ファイルを含む DM を送信します
- 検索: キーワード、ハッシュタグ、ユーザー、フィルターを使用してツイートを検索します
- データの取得: ユーザーの最近のアクティビティ、ホーム タイムライン
- URLからツイートIDを抽出

OAuth 1.0a 認証情報が必要です: X_API_KEY、X_API_SECRET、X_ACCESS_TOKEN、X_ACCESS_SECRET。
