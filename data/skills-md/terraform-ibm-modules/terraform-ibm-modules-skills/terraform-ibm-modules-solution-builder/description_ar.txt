سير العمل لإنشاء حل terraform الذي يتكون من وحدة واحدة أو عدة وحدات Terraform IBM (TIM). يتم استخدامه عند العمل مع البنية الأساسية لـ IBM Cloud كتعليمات برمجية، أو وحدات Terraform، أو أتمتة البنية التحتية، أو توفير الموارد السحابية. يوفر سير العمل لاكتشاف الوحدة النمطية وأنماط التركيب وإنشاء التعليمات البرمجية والتحقق من الصحة. ضروري للمهام التي تتضمن IBM Cloud VPC، أو الحوسبة، أو الشبكات، أو الأمان، أو قواعد البيانات، أو إمكانية المراقبة، أو أي نشر لخدمة IBM Cloud. تشغيل الكلمات الأساسية مثل "terraform"، أو "IBM Cloud"، أو "infrastructure"، أو "IaC"، أو "modules"، أو "deploy"، أو "provision"، أو خدمات IBM Cloud محددة (VPC، VSI، OpenShift، وما إلى ذلك).
