Workflows de génération d'une solution Terraform qui sont la composition d'un ou plusieurs modules Terraform IBM (TIM). À utiliser lorsque vous travaillez avec l'infrastructure IBM Cloud en tant que code, modules Terraform, automatisation de l'infrastructure ou provisionnement de ressources cloud. Fournit des flux de travail pour la découverte de modules, les modèles de composition, la génération de code et la validation. Indispensable pour les tâches impliquant IBM Cloud VPC, le calcul, la mise en réseau, la sécurité, les bases de données, l'observabilité ou tout déploiement de service IBM Cloud. Déclencheurs sur des mots clés tels que "terraform", "IBM Cloud", "infrastructure", "IaC", "modules", "deploy", "provision" ou des services IBM Cloud spécifiques (VPC, VSI, OpenShift, etc.).
