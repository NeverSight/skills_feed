Рабочие процессы для создания решения terraform, состоящего из одного или нескольких модулей Terraform IBM (TIM). Используйте при работе с инфраструктурой IBM Cloud в качестве кода, модулей Terraform, автоматизации инфраструктуры или предоставления облачных ресурсов. Предоставляет рабочие процессы для обнаружения модулей, шаблонов композиции, генерации кода и проверки. Необходим для задач, связанных с IBM Cloud VPC, вычислениями, сетями, безопасностью, базами данных, наблюдением или развертыванием любого сервиса IBM Cloud. Запускается по таким ключевым словам, как «terraform», «IBM Cloud», «инфраструктура», «IaC», «модули», «развертывание», «предоставление» или конкретным службам IBM Cloud (VPC, VSI, OpenShift и т. д.).
