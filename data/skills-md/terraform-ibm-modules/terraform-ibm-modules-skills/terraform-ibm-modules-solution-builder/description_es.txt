Flujos de trabajo para generar soluciones terraform que son la composición de uno o varios Módulos Terraform IBM (TIM). Úselo cuando trabaje con infraestructura de IBM Cloud como código, módulos Terraform, automatización de infraestructura o aprovisionamiento de recursos en la nube. Proporciona flujos de trabajo para el descubrimiento de módulos, patrones de composición, generación de código y validación. Esencial para tareas que involucran IBM Cloud VPC, computación, redes, seguridad, bases de datos, observabilidad o cualquier implementación de servicios de IBM Cloud. Activadores de palabras clave como "terraform", "IBM Cloud", "infraestructura", "IaC", "módulos", "implementar", "aprovisionamiento" o servicios específicos de IBM Cloud (VPC, VSI, OpenShift, etc.).
