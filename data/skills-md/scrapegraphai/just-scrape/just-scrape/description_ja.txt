ScrapeGraph AI を介した AI を活用した Web スクレイピング、データ抽出、検索、およびクロールのための CLI ツール。ユーザーが Web サイトのスクレイピング、URL からの構造化データの抽出、ページのマークダウンへの変換、複数ページのサイトのクロール、Web での情報の検索、ブラウザー操作の自動化 (ログイン、クリック、フォーム入力)、生の HTML の取得、サイトマップの検出、または JSON スキーマの生成を行う必要がある場合に使用します。 (1) Web サイトからのデータの抽出、(2) Web スクレイピングまたはクローリング、(3) Web ページのマークダウンへの変換、(4) 抽出を伴う AI を活用した Web 検索、(5) ブラウザーの自動化、(6) スクレイピング用の出力スキーマの生成を含むタスクのトリガー。 CLI は just-scrape (npm package just-scrape) です。
