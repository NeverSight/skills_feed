وكيل تداول العملات المشفرة المدعوم بالذكاء الاصطناعي عبر اللغة الطبيعية. يُستخدم عندما يريد المستخدم تداول العملات المشفرة (شراء/بيع/مبادلة الرموز المميزة)، أو التحقق من أرصدة المحفظة، أو عرض أسعار الرموز المميزة، أو نقل العملات المشفرة، أو إدارة NFTs، أو استخدام الرافعة المالية، أو الرهان على Polymarket، أو نشر الرموز المميزة، أو إعداد استراتيجيات التداول الآلية. يدعم Base وEthereum وPolygon وSolana وUnichain. تشمل القدرات الشاملة التداول وإدارة المحافظ وأبحاث السوق وعمليات NFT وأسواق التنبؤ والتداول بالرافعة المالية وعمليات DeFi والأتمتة.
