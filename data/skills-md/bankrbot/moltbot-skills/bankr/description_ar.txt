وكيل تداول العملات المشفرة المدعوم بالذكاء الاصطناعي وبوابة LLM عبر اللغة الطبيعية. يُستخدم عندما يريد المستخدم تداول العملات المشفرة، أو التحقق من أرصدة المحفظة، أو عرض أسعار الرمز المميز، أو نقل التشفير، أو إدارة NFTs، أو استخدام الرافعة المالية، أو الرهان على Polymarket، أو نشر الرموز المميزة، أو إعداد التداول الآلي، أو التوقيع على المعاملات الأولية وإرسالها، أو الوصول إلى نماذج LLM من خلال بوابة Bankr LLM الممولة من محفظة Bankr الخاصة بك. يدعم Base وEthereum وPolygon وSolana وUnichain.
