Создавайте ончейн-приложения с помощью компонентов React и утилит TypeScript из OnchainKit от Coinbase. Используйте, когда пользователи хотят создавать криптокошельки, обменивать токены, выпускать NFT, создавать платежи, отображать идентификационные данные блокчейна или разрабатывать какие-либо функции онлайн-приложений. Поддерживает подключение к кошельку, построение транзакций, операции с токенами, управление идентификацией и полные рабочие процессы разработки онлайн-приложений.
