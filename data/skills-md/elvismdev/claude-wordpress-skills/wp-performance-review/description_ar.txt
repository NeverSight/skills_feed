مراجعة كود أداء WordPress وتحليل التحسين. استخدمه عند مراجعة كود PHP الخاص بـ WordPress لمشكلات الأداء، أو تدقيق السمات/المكونات الإضافية لقابلية التوسع، أو تحسين WP_Query، أو تحليل إستراتيجيات التخزين المؤقت، أو التحقق من التعليمات البرمجية قبل الإطلاق، أو اكتشاف الأنماط المضادة، أو عندما يذكر المستخدم "مراجعة الأداء"، أو "تدقيق التحسين"، أو "WordPress البطيء"، أو "الاستعلامات البطيئة"، أو "حركة المرور العالية"، أو "حجم WordPress"، أو "مراجعة التعليمات البرمجية"، أو "المهلة"، أو "500 خطأ"، أو "نفاد الذاكرة"، أو "لن يتم تحميل الموقع". يكتشف الأنماط المضادة في استعلامات قاعدة البيانات، والخطافات، والتخزين المؤقت للكائنات، وAJAX، وتحميل القالب.
