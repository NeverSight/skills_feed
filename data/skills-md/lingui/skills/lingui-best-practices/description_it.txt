Implementa l'internazionalizzazione con Lingui nelle applicazioni React e JavaScript. Da utilizzare quando si aggiunge i18n, si traduce l'interfaccia utente, si lavora con Trans/useLingui/Plural, si estraggono messaggi, si compilano cataloghi o quando l'utente menziona Lingui, internazionalizzazione, i18n, traduzioni, impostazioni locali, estrazione di messaggi, ICU MessageFormat o si lavora con file .po.
