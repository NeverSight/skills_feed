Материал 3 Экспрессивный (М3Е). Комплексное руководство по системе выразительного дизайна для Flutter с поддержкой платформ для настольных компьютеров Android и Linux. Охватывает цветовые маркеры, типографские масштабы, характеристики движения, маркеры формы, интервалы и улучшения компонентов для создания эмоционально привлекательных пользовательских интерфейсов. Включает руководство по переходу со стандартного M3 и примечания по интеграции для конкретной платформы.
