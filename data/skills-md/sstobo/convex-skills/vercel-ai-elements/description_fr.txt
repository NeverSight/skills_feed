Cette compétence fournit une documentation complète pour les 23 composants Vercel AI Elements organisés par catégorie (Message, Conversation, Entrée/Interaction, Affichage de contenu, Traitement IA, Fonctionnalités avancées). À utiliser lorsque les utilisateurs posent des questions sur la création de chatbots IA, ont besoin de documentation sur les composants, souhaitent des références API pour Vercel AI Elements ou ont besoin d'exemples d'intégration avec le SDK AI.
