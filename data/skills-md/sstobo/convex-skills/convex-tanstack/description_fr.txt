Guide complet pour créer des applications full-stack avec Convex et TanStack Start. Cette compétence doit être utilisée lorsque vous travaillez sur des projets qui utilisent Convex comme base de données principale avec TanStack Start (méta-framework React). Couvre la conception de schémas, les requêtes, les mutations, les actions, l'authentification avec Better Auth, le routage, les modèles de récupération de données, le SSR, le stockage de fichiers, la planification, les agents d'IA et les modèles frontaux. Utilisez-le lors de la mise en œuvre de fonctionnalités, de problèmes de débogage ou si vous avez besoin de conseils sur les meilleures pratiques de Convex + TanStack Start.
