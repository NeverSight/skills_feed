Комплексное руководство по созданию полнофункциональных приложений с помощью Convex и TanStack Start. Этот навык следует использовать при работе над проектами, в которых Convex используется в качестве серверной базы данных с TanStack Start (метафреймворк React). Охватывает проектирование схемы, запросы, изменения, действия, аутентификацию с помощью Better Auth, маршрутизацию, шаблоны выборки данных, SSR, хранилище файлов, планирование, агенты искусственного интеллекта и шаблоны внешнего интерфейса. Используйте это при реализации функций, устранении проблем или при необходимости получить рекомендации по передовым практикам Convex + TanStack Start.
