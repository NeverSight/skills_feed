دليل شامل لبناء تطبيقات متكاملة باستخدام Convex وTanStack Start. يجب استخدام هذه المهارة عند العمل على المشاريع التي تستخدم Convex كقاعدة بيانات خلفية مع TanStack Start (إطار عمل React التعريفي). يغطي تصميم المخطط والاستعلامات والطفرات والإجراءات والمصادقة باستخدام Better Auth والتوجيه وأنماط جلب البيانات وSSR وتخزين الملفات والجدولة ووكلاء الذكاء الاصطناعي وأنماط الواجهة الأمامية. استخدم هذا عند تنفيذ الميزات أو تصحيح الأخطاء أو الحاجة إلى إرشادات حول أفضل ممارسات Convex + TanStack Start.
