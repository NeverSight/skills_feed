Umfassende Anleitung zum Erstellen von Full-Stack-Anwendungen mit Convex und TanStack Start. Diese Fähigkeit sollte bei der Arbeit an Projekten verwendet werden, die Convex als Backend-Datenbank mit TanStack Start (React-Meta-Framework) verwenden. Behandelt Schemadesign, Abfragen, Mutationen, Aktionen, Authentifizierung mit Better Auth, Routing, Datenabrufmuster, SSR, Dateispeicherung, Planung, KI-Agenten und Frontend-Muster. Verwenden Sie dies, wenn Sie Funktionen implementieren, Probleme debuggen oder Anleitung zu Best Practices für Convex + TanStack Start benötigen.
