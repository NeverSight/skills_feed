Guía completa para crear aplicaciones full-stack con Convex y TanStack Start. Esta habilidad debe usarse cuando se trabaja en proyectos que usan Convex como base de datos backend con TanStack Start (metamarco de React). Cubre diseño de esquemas, consultas, mutaciones, acciones, autenticación con Better Auth, enrutamiento, patrones de obtención de datos, SSR, almacenamiento de archivos, programación, agentes de IA y patrones de interfaz. Úselo cuando implemente funciones, depure problemas o necesite orientación sobre las mejores prácticas de Convex + TanStack Start.
