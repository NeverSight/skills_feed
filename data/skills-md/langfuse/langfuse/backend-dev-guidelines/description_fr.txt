Guide complet de développement backend pour le monorepo Next.js 14/tRPC/Express/TypeScript de Langfuse. À utiliser lors de la création de routeurs tRPC, de points de terminaison d'API publics, de processeurs de file d'attente BullMQ, de services ou lors de l'utilisation de procédures tRPC, de routes API Next.js, d'accès à la base de données Prisma, de requêtes d'analyse ClickHouse, de files d'attente Redis, d'instrumentation OpenTelemetry, de validation Zod v4, de configuration env.mjs, de modèles d'isolation de locataire ou de modèles asynchrones. Couvre l'architecture en couches (procédures tRPC → services, processeurs de file d'attente → services), le système de base de données double (PostgreSQL + ClickHouse), le filtrage projectId pour l'isolation multi-tenant, la gestion des erreurs traceException, les modèles d'observabilité et les stratégies de test (Jest pour le Web, vitest pour le travailleur).
