Refactorisez le code Django/Python pour améliorer la maintenabilité, la lisibilité et le respect des meilleures pratiques. Transforme les vues volumineuses en architecture propre avec des cas d'utilisation et des services. Applique les principes SOLID, les modèles Clean Code, les fonctionnalités Python 3.12+ telles que la syntaxe des paramètres de type et le décorateur @override, les modèles Django 5+ comme GeneratedField et les vues asynchrones. Corrige les requêtes N+1, extrait la logique métier des vues, sépare les sérialiseurs de lecture/écriture et convertit la gestion des erreurs basée sur les exceptions en valeurs de retour explicites. À utiliser lors de la refactorisation du code Django, de l'application d'une architecture propre ou de la modernisation de projets Django existants.
