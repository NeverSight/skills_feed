Créez des API Django REST Framework de qualité production en utilisant les principes Clean Architecture et SOLID. Couvre l'architecture en couches (vues, cas d'utilisation, services, modèles), l'optimisation des requêtes (prévention N+1), la pagination/filtrage, l'authentification JWT, les autorisations et le déploiement en production. À utiliser lors de la création de nouvelles API Django, de la mise en œuvre d'une conception basée sur le domaine, de l'optimisation des requêtes ou de la configuration de l'authentification. Applique les modèles Python 3.12+ et Django 5+.
