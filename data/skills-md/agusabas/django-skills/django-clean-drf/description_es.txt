Cubre arquitectura en capas (vistas, casos de uso, servicios, modelos), optimización de consultas (prevención N+1), paginación/filtrado, autenticación JWT, permisos e implementación de producción. Úselo al crear nuevas API de Django, implementar un diseño basado en dominios, optimizar consultas o configurar la autenticación. Aplica los patrones Python 3.12+ y Django 5+.
