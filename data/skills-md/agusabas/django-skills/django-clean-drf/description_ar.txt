قم بإنشاء واجهات برمجة تطبيقات Django REST Framework ذات جودة إنتاجية باستخدام مبادئ Clean Architecture وSOLID. يغطي البنية ذات الطبقات (العروض، وحالات الاستخدام، والخدمات، والنماذج)، وتحسين الاستعلام (منع N+1)، وترقيم الصفحات/التصفية، ومصادقة JWT، والأذونات، ونشر الإنتاج. يُستخدم عند إنشاء واجهات برمجة تطبيقات Django الجديدة، أو تنفيذ التصميم المستند إلى المجال، أو تحسين الاستعلامات، أو تكوين المصادقة. يطبق أنماط Python 3.12+ وDjango 5+.
