Crea API Django REST Framework di qualità produttiva utilizzando l'architettura pulita e i principi SOLID. Copre architettura a più livelli (viste, casi d'uso, servizi, modelli), ottimizzazione delle query (prevenzione N+1), impaginazione/filtro, autenticazione JWT, autorizzazioni e distribuzione di produzione. Da utilizzare durante la creazione di nuove API Django, l'implementazione di una progettazione basata sul dominio, l'ottimizzazione delle query o la configurazione dell'autenticazione. Applica i modelli Python 3.12+ e Django 5+.
