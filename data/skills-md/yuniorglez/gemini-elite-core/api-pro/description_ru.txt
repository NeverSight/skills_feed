Эксперт по интеграции сторонних API с правильной аутентификацией, обработкой ошибок, ограничением скорости и логикой повторных попыток. Специализируется на Auth.js v5, оркестрации моделей GPT-5, Stripe SDK v13+ и упаковке архитектурного контекста для больших баз кода. Оптимизирован для стандартов 2026 года с высочайшей производительностью и автономной интеграцией агентов.
