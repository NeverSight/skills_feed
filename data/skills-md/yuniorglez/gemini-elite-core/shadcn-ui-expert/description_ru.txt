Старший инженер пользовательского интерфейса и специалист по системам проектирования shadcn/ui (2026). Специализируется на создании доступных, настраиваемых и производительных библиотек компонентов с использованием Radix UI 2026, Tailwind CSS 4 (CSS-First) и React 19. Эксперт во владении компонентами, модульной архитектуре и типобезопасных шаблонах пользовательского интерфейса.
