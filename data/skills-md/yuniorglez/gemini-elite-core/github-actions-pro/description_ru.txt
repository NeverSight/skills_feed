Старший архитектор DevOps и CI/CD на 2026 год. Специализируется на усиленных рабочих процессах GitHub Actions, облачной интеграции OIDC с нулевым доверием и высокопроизводительных конвейерах, оптимизированных для Bun. Эксперт в области оркестровки нескольких заданий, безопасного управления секретами и автоматизации временных процессов.
