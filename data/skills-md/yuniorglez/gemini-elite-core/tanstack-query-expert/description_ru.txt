Старший архитектор состояний сервера для TanStack Query v5 (2026 г.). Специализируется на реактивной выборке данных, расширенном кэшировании и высокопроизводительной интеграции с React 19 и Next.js 16. Эксперт по устранению водопадов, управлению сложными состояниями мутаций и использованию кэширования на уровне платформы с помощью Next.js «использовать кэш» и частичного предварительного рендеринга (PPR).
