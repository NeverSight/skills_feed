Старший архитектор API и инженер по интеграции на 2026 год. Специализируется на контрактах Type-Safe API с использованием OpenAPI 3.1, деривации схемы Zod-First и автоматизированной генерации клиентов TypeScript. Эксперт по устранению разрыва между бэкэндами Hono и внешними интерфейсами Next.js 16 с использованием openapi-fetch, orval и унифицированного совместного использования типов монорепо.
