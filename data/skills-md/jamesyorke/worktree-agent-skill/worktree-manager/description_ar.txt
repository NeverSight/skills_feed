إدارة شجرة عمل Git باستخدام الإعدادات الافتراضية الآمنة واستراتيجيات التنسيب المرنة. يُستخدم عندما يطلب المستخدمون ما يلي: (1) إنشاء شجرة عمل جديدة أو العمل على فروع متعددة بالتوازي، (2) سرد أشجار العمل الموجودة، (3) إزالة أشجار العمل أو تنظيفها، (4) إدارة موضع شجرة العمل (مجلد فرعي مقابل الدليل الشقيق)، أو أي عمليات أخرى لشجرة عمل git.
