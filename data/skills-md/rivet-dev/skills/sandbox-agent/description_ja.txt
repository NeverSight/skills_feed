サンドボックス エージェントの展開、構成、統合 - サンドボックス環境で AI コーディング エージェント (Claude Code、Codex、OpenCode、Amp) を調整するためのユニバーサル API。ローカルまたはクラウド サンドボックス (E2B、Daytona、Docker) でサンドボックス エージェント サーバーをセットアップする場合、SDK または API を介してエージェント セッションを作成および管理する場合、エージェント イベントをストリーミングして人間参加型の対話を処理する場合、エージェントをコーディングするためのチャット UI を構築する場合、またはエージェント応答のユニバーサル スキーマを理解する場合に使用します。
