نشر Sandbox Agent وتكوينه ودمجه - وهو واجهة برمجة تطبيقات عالمية لتنظيم وكلاء ترميز الذكاء الاصطناعي (Claude Code وCodex وOpenCode وAmp) في البيئات المعزولة. يُستخدم عند إعداد خادم Sandbox-agent محليًا أو في Sandboxes السحابية (E2B وDaytona وDocker)، أو إنشاء جلسات الوكيل وإدارتها عبر SDK أو API، أو تدفق أحداث الوكيل والتعامل مع التفاعلات البشرية داخل الحلقة، أو إنشاء واجهات مستخدم للدردشة لوكلاء الترميز، أو فهم المخطط العام لاستجابات الوكيل.
