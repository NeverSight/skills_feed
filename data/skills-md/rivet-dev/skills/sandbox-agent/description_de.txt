Stellen Sie Sandbox Agent bereit, konfigurieren und integrieren Sie ihn – eine universelle API zur Orchestrierung von KI-Coding-Agents (Claude Code, Codex, OpenCode, Amp) in Sandbox-Umgebungen. Verwenden Sie es, wenn Sie einen Sandbox-Agent-Server lokal oder in Cloud-Sandboxen (E2B, Daytona, Docker) einrichten, Agent-Sitzungen über SDK oder API erstellen und verwalten, Agent-Ereignisse streamen und Human-in-the-Loop-Interaktionen verwalten, Chat-Benutzeroberflächen für die Codierung von Agenten erstellen oder das universelle Schema für Agent-Antworten verstehen.
