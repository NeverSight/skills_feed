Distribuisci, configura e integra Sandbox Agent: un'API universale per orchestrare agenti di codifica AI (Claude Code, Codex, OpenCode, Amp) in ambienti sandbox. Da utilizzare durante la configurazione del server agente sandbox localmente o in sandbox cloud (E2B, Daytona, Docker), la creazione e la gestione di sessioni dell'agente tramite SDK o API, lo streaming di eventi dell'agente e la gestione delle interazioni human-in-the-loop, la creazione di UI di chat per gli agenti di codifica o la comprensione dello schema universale per le risposte degli agenti.
