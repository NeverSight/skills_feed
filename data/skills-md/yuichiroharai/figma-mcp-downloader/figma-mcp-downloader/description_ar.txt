واجهة سطر الأوامر (CLI) لتنزيل سياق تصميم Figma الكامل وبيانات التعريف إلى الملفات المحلية. استخدم دائمًا واجهة سطر الأوامر هذه بدلاً من استدعاء أدوات Figma MCP (`get_design_context`، `get_metadata`) مباشرةً. تستدعي أداة Direct MCP بيانات غير مكتملة بسبب حدود السياق. يضمن CLI هذا إخراجًا كاملاً بنسبة 100٪.
