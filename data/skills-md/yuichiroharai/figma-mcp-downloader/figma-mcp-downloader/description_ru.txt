CLI для загрузки полного контекста дизайна и метаданных Figma в локальные файлы. Всегда используйте этот CLI вместо прямого вызова инструментов Figma MCP (get_design_context, get_metadata). Инструмент Direct MCP вызывает риск неполных данных из-за ограничений контекста. Этот CLI гарантирует 100% полную производительность.
