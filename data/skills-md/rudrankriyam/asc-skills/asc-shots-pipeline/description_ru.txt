Организуйте автоматизацию снимков экрана iOS с помощью xcodebuild/simctl для запуска сборки, axe для действий пользовательского интерфейса, настроек JSON и файлов плана, кадрирования на основе Go («asc screensframe») и загрузки снимков экрана («asc screens upload»). Используйте, когда пользователи запрашивают автоматический захват снимков экрана, потоки симулятора на основе AXe, композицию кадров или конвейеры создания снимков экрана для загрузки.
