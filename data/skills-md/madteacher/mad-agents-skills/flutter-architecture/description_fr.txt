Guide complet pour l'architecture d'applications Flutter suivant le modèle MVVM et les meilleures pratiques avec une organisation de projet axée sur les fonctionnalités. À utiliser lorsque vous travaillez avec des projets Flutter pour structurer correctement le code, implémenter des couches d'architecture propres (interface utilisateur, données, domaine), appliquer les modèles de conception recommandés et organiser les projets en utilisant une approche axée sur les fonctionnalités pour des applications évolutives et maintenables.
