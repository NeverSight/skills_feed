Guía completa para usar la biblioteca de bases de datos de deriva en aplicaciones Dart (CLI, del lado del servidor, sin Flutter). Úselo al crear aplicaciones Dart que necesitan almacenamiento de base de datos SQLite local o conexión PostgreSQL con consultas con seguridad de tipos, flujos reactivos, migraciones y operaciones CRUD eficientes. Incluye configuración con el paquete sqlite3, compatibilidad con PostgreSQL con drift_postgres, agrupación de conexiones y patrones del lado del servidor.
