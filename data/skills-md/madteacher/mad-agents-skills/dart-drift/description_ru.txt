Полное руководство по использованию библиотеки базы данных Drift в приложениях Dart (CLI, на стороне сервера, без Flutter). Используйте при создании приложений Dart, которым требуется локальное хранилище базы данных SQLite или соединение PostgreSQL с типобезопасными запросами, реактивными потоками, миграциями и эффективными операциями CRUD. Включает настройку с помощью пакета sqlite3, поддержку PostgreSQL с помощьюdrift_postgres, пул соединений и шаблоны на стороне сервера.
