Guida completa per l'utilizzo della libreria di database Drift nelle applicazioni Dart (CLI, lato server, non Flutter). Da utilizzare durante la creazione di app Dart che richiedono l'archiviazione del database SQLite locale o una connessione PostgreSQL con query indipendenti dai tipi, flussi reattivi, migrazioni e operazioni CRUD efficienti. Include la configurazione con il pacchetto sqlite3, supporto PostgreSQL con drift_postgres, pool di connessioni e modelli lato server.
