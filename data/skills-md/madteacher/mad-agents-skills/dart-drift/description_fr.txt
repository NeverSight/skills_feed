Guide complet d'utilisation de la bibliothèque de bases de données Drift dans les applications Dart (CLI, côté serveur, non-Flutter). À utiliser lors de la création d'applications Dart nécessitant un stockage de base de données SQLite local ou une connexion PostgreSQL avec des requêtes de type sécurisé, des flux réactifs, des migrations et des opérations CRUD efficaces. Comprend la configuration avec le package sqlite3, la prise en charge de PostgreSQL avec drift_postgres, le regroupement de connexions et les modèles côté serveur.
