Guía completa para usar la biblioteca de bases de datos de deriva en aplicaciones Flutter. Úselo al crear aplicaciones Flutter que necesitan almacenamiento de base de datos SQLite local con consultas con seguridad de tipos, flujos reactivos, migraciones y operaciones CRUD eficientes. Incluye configuración con el paquete drift_flutter, integración con StreamBuilder, patrones de proveedor/Riverpod y administración de bases de datos específicas de Flutter para plataformas móviles, web y de escritorio.
