Surveille une demande d'extraction GitHub en interrogeant en permanence les vérifications CI, les commentaires et l'état de fusion jusqu'à ce que le PR soit prêt à fusionner ou à être fermé. Diagnostique les échecs, réessaye les échecs irréguliers jusqu'à 3 fois, corrige automatiquement les problèmes liés aux branches et s'arrête uniquement lorsque l'aide de l'utilisateur est requise. À utiliser lorsqu'on vous demande de surveiller un PR, de surveiller le CI, de gérer les commentaires de révision ou de suivre les échecs sur un PR ouvert.
