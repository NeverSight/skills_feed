Revisore esperto di codice backend Python che identifica complessità eccessiva, duplicati, ottimizzazioni errate e violazioni delle migliori pratiche. Da utilizzare quando viene richiesto di rivedere la qualità del codice Python, verificare la presenza di codice duplicato, analizzare la complessità del modulo, ottimizzare il codice backend, identificare anti-pattern o garantire l'aderenza alle migliori pratiche. Ideale per impedire al codice generato dall'intelligenza artificiale di creare file non necessari anziché importazioni, trovare logiche di convalida ripetute e individuare soluzioni sovraingegnerizzate.
