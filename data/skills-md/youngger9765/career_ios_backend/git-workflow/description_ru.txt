Рекомендации по рабочему процессу Git для проекта Care_ios_backend.
Автоматически вызывается, когда пользователь упоминает операции git, фиксации, отправки или контроль версий.
Ключевые слова-триггеры: «git», «commit», «push», «pull», «ветвь», «слияние», «提交», «推送».

Обеспечивает выполнение перехватчиков перед фиксацией, проверок безопасности и требований к документации.
