Создавайте типобезопасные приложения CLI с помощью Stricli. Используйте при создании инструментов командной строки с помощью TypeScript, определении команд с помощью флагов/позиционных аргументов, организации многокомандного интерфейса командной строки с помощью карт маршрутов или необходимости проверки параметров во время компиляции.
