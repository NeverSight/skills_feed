Paga per endpoint API protetti x402 con USDC. Da utilizzare quando si chiamano API che restituiscono HTTP 402 Payment Required, si integrano pagamenti negli agenti, si gestiscono requisiti di pagamento x402, si creano agenti autonomi che pagano per l'accesso API o si scoprono servizi a pagamento tramite Bazaar. Supporta le reti EVM (Base, Ethereum, Avalanche) e Solana.
