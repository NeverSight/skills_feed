ادفع مقابل نقاط نهاية API المحمية بـ x402 باستخدام USDC. يُستخدم عند الاتصال بواجهات برمجة التطبيقات التي تُرجع طلب الدفع عبر HTTP 402، أو دمج المدفوعات في الوكلاء، أو التعامل مع متطلبات الدفع x402، أو إنشاء وكلاء مستقلين يدفعون مقابل الوصول إلى واجهة برمجة التطبيقات، أو اكتشاف الخدمات المدفوعة عبر Bazaar. يدعم شبكات EVM (Base وEthereum وAvalanche) وSolana.
