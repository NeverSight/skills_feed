Meilleures pratiques du framework d'interface utilisateur GPUI pour la création d'applications de bureau. À utiliser lors de l'écriture de code GPUI, de la création de composants d'interface utilisateur, de la gestion d'états/d'événements, de tâches asynchrones, d'animations, de listes, de formulaires, de tests ou de l'utilisation du code GUI Rust de style Zed.
