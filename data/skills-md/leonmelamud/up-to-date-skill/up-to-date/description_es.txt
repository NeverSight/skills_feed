Obtiene y valida la documentación API actual de terceros, verifica la compatibilidad de la versión del SDK y verifica las firmas de métodos y las formas de los parámetros correctos antes de escribir o depurar el código de integración. Úselo al escribir, editar o refactorizar código que llame a cualquier SDK, API o método de biblioteca de terceros; problemas de depuración en los que una llamada API tiene éxito pero no se produce el efecto secundario esperado; instalar o importar paquetes externos; o cuando algo "no funciona" y el código involucra un servicio externo.
