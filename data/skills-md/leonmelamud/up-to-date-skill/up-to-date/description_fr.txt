Récupère et valide la documentation actuelle de l'API tierce, vérifie la compatibilité des versions du SDK et vérifie les signatures de méthode et les formes de paramètres correctes avant d'écrire ou de déboguer le code d'intégration. À utiliser lors de l'écriture, de la modification ou de la refactorisation de code qui appelle un SDK, une API ou une méthode de bibliothèque tiers ; problèmes de débogage où un appel d'API réussit mais l'effet secondaire attendu ne se produit pas ; installer ou importer des packages externes ; ou lorsque quelque chose « ne fonctionne pas » et que le code implique un service externe.
