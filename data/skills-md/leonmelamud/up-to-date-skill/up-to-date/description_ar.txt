جلب وثائق API الحالية لجهة خارجية والتحقق من صحتها، والتحقق من توافق إصدار SDK، والتحقق من صحة توقيعات الطريقة وأشكال المعلمات قبل كتابة كود التكامل أو تصحيح أخطاءه. يُستخدم عند كتابة التعليمات البرمجية أو تحريرها أو إعادة هيكلتها والتي تستدعي أي طريقة SDK أو API أو مكتبة تابعة لجهة خارجية؛ تصحيح الأخطاء حيث ينجح استدعاء API ولكن لا يحدث التأثير الجانبي المتوقع؛ تثبيت أو استيراد الحزم الخارجية؛ أو عندما يكون هناك شيء "لا يعمل" ويتضمن الكود خدمة خارجية.
