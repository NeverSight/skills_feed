Sviluppare uno strumento automatizzato per la generazione e il mantenimento dei documenti di sviluppo. Dopo aver completato lo sviluppo dei requisiti, produce automaticamente i documenti dei requisiti (PRD) e i documenti dell'interfaccia API; quando il codice viene aggiornato, mantiene automaticamente CHANGELOG e API CHANGELOG. Condizioni di attivazione: si attiva automaticamente quando l'utente dice "genera documenti", "scrivi documenti", "aggiorna documenti" o menziona PRD, documentazione API, registro delle modifiche o documenti dei requisiti.
