Genera e mantieni automaticamente gli strumenti di documentazione di sviluppo. Dopo aver completato lo sviluppo dei requisiti, genera automaticamente i documenti dei requisiti (PRD) e i documenti dell'interfaccia API; dopo gli aggiornamenti del codice, mantiene automaticamente CHANGELOG e API CHANGELOG. Tempi di attivazione: attivato automaticamente quando l'utente dice "genera documentazione", "scrivi documentazione", "aggiorna documentazione" o menziona PRD, documentazione API, registro delle modifiche o documenti dei requisiti.
