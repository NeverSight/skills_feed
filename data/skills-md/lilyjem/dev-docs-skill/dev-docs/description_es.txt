Generar y mantener automáticamente herramientas de documentación de desarrollo. Después de completar el desarrollo de requisitos, genere automáticamente documentos de requisitos (PRD) y documentos de interfaz API; después de las actualizaciones del código, mantenga automáticamente CHANGELOG y API CHANGELOG. Tiempo de activación: se activa automáticamente cuando el usuario dice "generar documentación", "escribir documentación", "actualizar documentación" o menciona PRD, documentación de API, registro de cambios o documentos de requisitos.
