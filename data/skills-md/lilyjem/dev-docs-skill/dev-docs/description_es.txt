Desarrollar una herramienta automatizada para generar y mantener documentos de desarrollo. Después de completar el desarrollo de requisitos, genera automáticamente documentos de requisitos (PRD) y documentos de interfaz API; cuando se actualiza el código, mantiene automáticamente CHANGELOG y API CHANGELOG. Condiciones de activación: se activa automáticamente cuando el usuario dice "generar documentos", "escribir documentos", "actualizar documentos" o menciona PRD, documentación API, registro de cambios o documentos de requisitos.
