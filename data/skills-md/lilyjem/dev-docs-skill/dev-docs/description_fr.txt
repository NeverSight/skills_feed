Générez et maintenez automatiquement les outils de documentation de développement. Après avoir terminé le développement des exigences, générer automatiquement les documents d'exigences (PRD) et les documents d'interface API ; après les mises à jour du code, maintenez automatiquement CHANGELOG et API CHANGELOG. Moment du déclenchement : déclenché automatiquement lorsque l'utilisateur dit « générer la documentation », « écrire la documentation », « mettre à jour la documentation » ou mentionne le PRD, la documentation de l'API, le journal des modifications ou les documents d'exigences.
