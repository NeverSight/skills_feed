Développer un outil automatisé pour générer et maintenir des documents de développement. Une fois le développement des exigences terminé, il produit automatiquement des documents d'exigences (PRD) et des documents d'interface API ; lorsque le code est mis à jour, il maintient automatiquement CHANGELOG et API CHANGELOG. Conditions de déclenchement : s'active automatiquement lorsque l'utilisateur dit « générer des documents », « rédiger des documents », « mettre à jour des documents » ou mentionne PRD, la documentation de l'API, le journal des modifications ou les documents d'exigences.
