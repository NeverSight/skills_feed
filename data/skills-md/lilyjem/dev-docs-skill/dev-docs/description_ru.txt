Автоматически создавайте и обслуживайте инструменты разработки документации. После завершения разработки требований автоматически генерировать документы требований (PRD) и документы интерфейса API; после обновления кода автоматически поддерживать CHANGELOG и API CHANGELOG. Время срабатывания: автоматически срабатывает, когда пользователь говорит «создать документацию», «написать документацию», «обновить документацию» или упоминает PRD, документацию API, журнал изменений или документы с требованиями.
