Entwickeln Sie ein automatisiertes Tool zur Erstellung und Pflege von Entwicklungsdokumenten. Nach Abschluss der Anforderungsentwicklung werden automatisch Anforderungsdokumente (PRD) und API-Schnittstellendokumente erstellt. Wenn der Code aktualisiert wird, werden CHANGELOG und API CHANGELOG automatisch verwaltet. Auslösende Bedingungen: Wird automatisch aktiviert, wenn der Benutzer „Dokumente generieren“, „Dokumente schreiben“, „Dokumente aktualisieren“ sagt oder PRD, API-Dokumentation, Änderungsprotokoll oder Anforderungsdokumente erwähnt.
