Una habilidad para operar la API web de Slack. Publica y recupera mensajes, recupera hilos, busca publicaciones y recupera información del usuario.
Cuándo usarlo: (1) Cuando desea publicar un mensaje en Slack (2) Cuando desea recuperar el historial de mensajes de un canal
(3) Cuando desea recuperar respuestas en un hilo (4) Cuando desea buscar sus propias publicaciones (5) Cuando desea recuperar información del usuario
Palabras clave de activación: Slack, enviar mensaje, obtener hilo, historial del canal, obtener usuario
