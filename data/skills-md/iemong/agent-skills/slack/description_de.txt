Eine Fähigkeit zum Betreiben der Slack-Web-API. Es postet und ruft Nachrichten ab, ruft Threads ab, durchsucht Beiträge und ruft Benutzerinformationen ab.
Verwendungszweck: (1) Wenn Sie eine Nachricht auf Slack posten möchten (2) Wenn Sie den Nachrichtenverlauf eines Kanals abrufen möchten
(3) Wenn Sie Antworten in einem Thread abrufen möchten (4) Wenn Sie nach Ihren eigenen Beiträgen suchen möchten (5) Wenn Sie Benutzerinformationen abrufen möchten
Trigger-Schlüsselwörter: Slack, Nachricht senden, Thread abrufen, Kanalverlauf, Benutzer abrufen
