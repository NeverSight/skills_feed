A skill for operating the Slack Web API. It posts and retrieves messages, fetches threads, searches posts, and retrieves user information.
When to use: (1) When you want to post a message to Slack (2) When you want to retrieve a channel’s message history
(3) When you want to retrieve replies in a thread (4) When you want to search for your own posts (5) When you want to retrieve user information
Trigger keywords: Slack, send message, get thread, channel history, get user
