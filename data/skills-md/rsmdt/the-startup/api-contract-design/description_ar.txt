أنماط تصميم REST وGraphQL API ومواصفات OpenAPI/Swagger واستراتيجيات الإصدار وأنماط المصادقة. يُستخدم عند تصميم واجهات برمجة التطبيقات، أو مراجعة عقود واجهة برمجة التطبيقات، أو تقييم تقنيات واجهة برمجة التطبيقات، أو تنفيذ نقاط نهاية واجهة برمجة التطبيقات. يغطي تصميم العقد أولاً، ونمذجة الموارد، ومعالجة الأخطاء، وترقيم الصفحات، والأمن.
