قم بتشغيل كود Python في السحابة باستخدام حاويات بدون خادم، ووحدات معالجة الرسومات، والقياس التلقائي. يُستخدم عند نشر نماذج تعلم الآلة، أو تشغيل المهام المجمعة، أو جدولة المهام، أو تقديم واجهات برمجة التطبيقات مع تسريع وحدة معالجة الرسومات، أو قياس أعباء العمل كثيفة الحوسبة. يتم تشغيلها بناءً على طلبات البنية التحتية لوحدة معالجة الرسومات بدون خادم، أو استدلال LLM، أو التدريب على النماذج/الضبط الدقيق، أو معالجة البيانات المتوازية، أو وظائف cron في السحابة، أو نشر نقاط نهاية ويب Python.
