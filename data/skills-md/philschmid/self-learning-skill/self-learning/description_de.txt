Autonomer Skill-Generator, der neue Technologien aus dem Web lernt. Verwenden Sie diese Option, wenn Benutzer mehr über eine neue Bibliothek/ein neues Framework/ein neues Tool erfahren möchten, einen Skill für eine unbekannte Technologie erstellen müssen, die Nutzungsmuster einer Technologie recherchieren und dokumentieren möchten oder mit „/learn <topic>“ aufrufen möchten. Dieser Skill verwendet Websuch- und Browser-Tools, um Dokumentation zu entdecken, zu extrahieren und zu einem wiederverwendbaren Skill zusammenzufassen.
