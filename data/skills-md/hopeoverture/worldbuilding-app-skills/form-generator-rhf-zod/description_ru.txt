Этот навык следует использовать при создании форм React с помощью React Hook Form, проверки Zod и компонентов shadcn/ui. Применяется при создании форм сущностей, редакторов символов, форм местоположения, форм ввода данных или любой формы, требующей проверки клиента и сервера. Термины триггера включают форму создания, форму генерации, форму сборки, форму React Hook, RHF, проверку Zod, компонент формы, форму сущности, форму символа, ввод данных, схему формы.
