Questa abilità dovrebbe essere utilizzata quando si generano moduli React con React Hook Form, convalida Zod e componenti shadcn/ui. Si applica durante la creazione di moduli di entità, editor di caratteri, moduli di posizione, moduli di immissione dati o qualsiasi modulo che richieda la convalida client e server. I termini di attivazione includono creare modulo, generare modulo, creare modulo, modulo React Hook, RHF, convalida Zod, componente modulo, modulo entità, modulo carattere, immissione dati, schema modulo.
