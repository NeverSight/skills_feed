Cette compétence doit être utilisée lors de la génération de formulaires React avec les composants React Hook Form, Zod validation et shadcn/ui. S'applique lors de la création de formulaires d'entité, d'éditeurs de caractères, de formulaires de localisation, de formulaires de saisie de données ou de tout formulaire nécessitant une validation client et serveur. Les termes déclencheurs incluent créer un formulaire, générer un formulaire, créer un formulaire, React Hook Form, RHF, validation Zod, composant de formulaire, formulaire d'entité, formulaire de caractère, saisie de données, schéma de formulaire.
