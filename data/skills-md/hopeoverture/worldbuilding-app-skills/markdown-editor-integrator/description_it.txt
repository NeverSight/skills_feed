Questa competenza deve essere utilizzata durante l'installazione e la configurazione della funzionalità dell'editor markdown utilizzando @uiw/react-md-editor. Si applica quando si aggiungono funzionalità di modifica del testo RTF, supporto per il markdown, editor WYSIWYG, modifica dei contenuti con anteprima o funzionalità di formattazione del testo. I termini di attivazione includono editor di markdown, editor di testo RTF, editor di testo, aggiunta di markdown, installazione di editor di markdown, componente di markdown, WYSIWYG, editor di contenuti, formattazione del testo, anteprima dell'editor.
