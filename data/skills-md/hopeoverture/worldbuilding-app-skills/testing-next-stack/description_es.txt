Apoya la configuración de pruebas integral para aplicaciones Next.js, incluidas las pruebas unitarias de Vitest, las pruebas de componentes de la biblioteca de pruebas de React y los flujos Playwright E2E con pruebas de accesibilidad a través de axe-core. Esta habilidad debe usarse al configurar la infraestructura de prueba, generar archivos de prueba, crear utilidades de prueba, agregar comprobaciones de accesibilidad o configurar marcos de prueba para proyectos Next.js. Los términos desencadenantes incluyen pruebas de configuración, pruebas de andamio, vitest, RTL, dramaturgo, pruebas e2e, pruebas de componentes, pruebas unitarias, pruebas de accesibilidad, pruebas a11y, axe-core, configuración de prueba.
