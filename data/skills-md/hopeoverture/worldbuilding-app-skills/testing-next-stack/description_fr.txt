Échafaudage une configuration de test complète pour les applications Next.js, y compris les tests unitaires Vitest, les tests de composants React Testing Library et les flux Playwright E2E avec tests d'accessibilité via axe-core. Cette compétence doit être utilisée lors de la configuration de l'infrastructure de test, de la génération de fichiers de test, de la création d'utilitaires de test, de l'ajout de contrôles d'accessibilité ou de la configuration de cadres de test pour les projets Next.js. Les termes déclencheurs incluent les tests de configuration, les tests d'échafaudage, les tests vitest, RTL, dramaturge, e2e, les tests de composants, les tests unitaires, les tests d'accessibilité, les tests a11y, axe-core, la configuration des tests.
