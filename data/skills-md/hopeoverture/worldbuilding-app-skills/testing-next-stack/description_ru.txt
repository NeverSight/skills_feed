Создает комплексную настройку тестирования для приложений Next.js, включая модульные тесты Vitest, тесты компонентов библиотеки тестирования React и потоки Playwright E2E с тестированием доступности через axe-core. Этот навык следует использовать при настройке тестовой инфраструктуры, создании тестовых файлов, создании тестовых утилит, добавлении проверок доступности или настройке платформ тестирования для проектов Next.js. Триггерные термины включают тестирование установки, тесты каркаса, vitest, RTL, драматургические тесты, тесты e2e, тесты компонентов, модульные тесты, тестирование доступности, тесты a11y, axe-core, тестовую конфигурацию.
