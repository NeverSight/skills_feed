Scaffolds umfassendes Test-Setup für Next.js-Anwendungen, einschließlich Vitest-Unit-Tests, React Testing Library-Komponententests und Playwright E2E-Flows mit Barrierefreiheitstests über Axe-Core. Diese Fähigkeit sollte beim Einrichten einer Testinfrastruktur, beim Generieren von Testdateien, beim Erstellen von Testdienstprogrammen, beim Hinzufügen von Barrierefreiheitsprüfungen oder beim Konfigurieren von Testframeworks für Next.js-Projekte verwendet werden. Zu den Auslöserbegriffen gehören Setup-Tests, Scaffold-Tests, Vitest, RTL, Playwright, E2E-Tests, Komponententests, Unit-Tests, Barrierefreiheitstests, A11y-Tests, Axe-Core und Testkonfiguration.
