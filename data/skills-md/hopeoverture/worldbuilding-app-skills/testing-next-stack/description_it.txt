Supporta una configurazione di test completa per le applicazioni Next.js, inclusi test unitari Vitest, test dei componenti della libreria React Testing e flussi Playwright E2E con test di accessibilità tramite axe-core. Questa competenza dovrebbe essere utilizzata durante la configurazione dell'infrastruttura di test, la generazione di file di test, la creazione di utilità di test, l'aggiunta di controlli di accessibilità o la configurazione di framework di test per progetti Next.js. I termini trigger includono test di configurazione, test di scaffold, vitest, RTL, drammaturgo, test e2e, test dei componenti, test unitari, test di accessibilità, test a11y, axe-core, configurazione del test.
