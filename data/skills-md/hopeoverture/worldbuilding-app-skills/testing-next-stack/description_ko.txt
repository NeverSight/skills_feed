Vitest 단위 테스트, React 테스트 라이브러리 구성 요소 테스트 및 Playwright E2E 흐름을 포함한 Next.js 애플리케이션에 대한 포괄적인 테스트 설정을 도끼 코어를 통한 접근성 테스트와 함께 제공합니다. 이 기술은 테스트 인프라 설정, 테스트 파일 생성, 테스트 유틸리티 생성, 접근성 검사 추가 또는 Next.js 프로젝트용 테스트 프레임워크 구성 시 사용해야 합니다. 트리거 용어에는 설정 테스트, 스캐폴드 테스트, vitest, RTL, 극작가, e2e 테스트, 구성 요소 테스트, 단위 테스트, 접근성 테스트, a11y 테스트, axe-core, 테스트 구성이 포함됩니다.
