Этот навык следует использовать при настройке инструментов обеспечения качества кода с помощью плоской конфигурации ESLint v9, форматирования Prettier, перехватчиков Husky git, поэтапных проверок перед фиксацией и рабочего процесса GitHub Actions CI lint. Применяйте при инициализации linting, добавлении форматирования кода, настройке перехватчиков перед фиксацией, настройке контрольных параметров качества или проверке CI lint для проектов Next.js или React.
