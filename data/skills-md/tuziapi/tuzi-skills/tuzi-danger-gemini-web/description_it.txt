Genera immagini e testo tramite API Web Gemini decodificata. Supporta la generazione di testo, la richiesta di immagine, l'input visivo di immagini di riferimento e il dialogo a più turni. Utilizzato quando altre competenze necessitano di un backend per la generazione di immagini o quando gli utenti richiedono "genera immagini con Gemini" o "generazione di testo Gemini".
