Documentazione completa della piattaforma e-commerce Salla. Utilizza questa competenza quando lavori con le API Salla (prodotti, ordini, spedizioni, clienti, pagamenti, tasse, coupon, impostazioni del negozio, webhook), moduli JavaScript del negozio Salla (autenticazione, carrello, prodotto, valuta, eventi, moduli, valutazioni, liste dei desideri), sviluppo di temi Twilight/Twig (layout, pagine, componenti, struttura dei file), sviluppo di app Salla (OAuth, funzioni dell'app, comandi CLI, processo di revisione) o qualsiasi domanda di integrazione relativa a Salla.
