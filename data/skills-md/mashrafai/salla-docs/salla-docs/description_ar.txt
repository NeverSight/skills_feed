وثائق منصة التجارة الإلكترونية الشاملة Salla. استخدم هذه المهارة عند العمل مع واجهات برمجة تطبيقات Salla (المنتجات، الطلبات، الشحن، العملاء، المدفوعات، الضرائب، القسائم، إعدادات المتجر، خطافات الويب)، وحدات JavaScript لواجهة متجر Salla (المصادقة، عربة التسوق، المنتج، العملة، الأحداث، النماذج، التقييمات، قوائم الرغبات)، تطوير سمة Twilight/Twig (التخطيطات، الصفحات، المكونات، بنية الملف)، تطوير تطبيق Salla (OAuth، وظائف التطبيق، أوامر CLI، عملية المراجعة)، أو أي سؤال تكامل متعلق بـ Salla.
