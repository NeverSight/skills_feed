Полная документация по платформе электронной коммерции Salla. Используйте этот навык при работе с API-интерфейсами Salla (продукты, заказы, доставка, клиенты, платежи, налоги, купоны, настройки магазина, веб-перехватчики), модулями JavaScript витрины Salla (аутентификация, корзина, продукт, валюта, события, формы, рейтинги, списки желаний), разработкой тем Twilight/Twig (макеты, страницы, компоненты, файловая структура), разработкой приложений Salla (OAuth, функции приложения, команды CLI, процесс проверки) или любыми вопросами интеграции, связанными с Salla.
