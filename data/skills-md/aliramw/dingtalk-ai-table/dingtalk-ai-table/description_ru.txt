Таблица DingTalk AI (многомерная таблица) Навыки работы. Используйте интерфейс командной строки mcporter для подключения к серверу DingTalk MCP для создания таблиц, управления таблицами данных, полевых операций и записей CRUD. Требуется настройка учетных данных DINGTALK_MCP_URL. Варианты использования: создание таблиц AI, управление структурами таблиц данных, пакетный импорт/экспорт данных, автоматизация управления запасами/проектами и другие задачи работы с таблицами.
