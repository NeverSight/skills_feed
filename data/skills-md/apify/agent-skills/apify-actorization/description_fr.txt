Convertissez les projets existants en Apify Actors - programmes cloud sans serveur. Actorisez JavaScript/TypeScript (SDK avec Actor.init/exit), Python (gestionnaire de contexte asynchrone) ou n'importe quel langage (wrapper CLI). À utiliser lors de la migration du code vers Apify, de l'encapsulation des outils CLI en tant qu'acteurs ou de l'ajout du SDK Actor à des projets existants.
