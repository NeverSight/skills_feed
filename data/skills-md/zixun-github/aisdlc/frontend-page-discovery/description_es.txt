Úselo cuando el código de front-end necesite producir evidencia para "lista de páginas → puntos de características → procesos de negocios → reglas de negocios" y el proyecto tiene múltiples puntos de entrada de enrutamiento, enrutamiento/permisos/menús de backend dinámicos complejos y el equipo tiende a hacer suposiciones o pasar por alto cadenas de evidencia.
