Организует реализацию плагина JUCE на этапах 1–3 (Foundation, DSP, GUI), используя делегирование субагента с автоматической проверкой после каждого этапа. Используйте при внедрении плагинов после завершения планирования или при возобновлении с помощью команды /continue. Вызывается командой /implement.
