Orchestrator para el flujo de trabajo de maqueta de la interfaz de usuario de WebView: delega la iteración del diseño al agente de diseño de interfaz de usuario y el andamiaje de implementación al agente de finalización de interfaz de usuario. Úselo cuando el usuario mencione el diseño de la interfaz de usuario, la maqueta, la interfaz WebView o solicite "diseño de interfaz de usuario para [complemento]".
