Оркестратор для рабочего процесса макета пользовательского интерфейса WebView — делегирует итерацию проектирования ui-design-agent, а создание каркасов реализации — ui-finalization-agent. Используйте, когда пользователь упоминает дизайн пользовательского интерфейса, макет, интерфейс WebView или запрашивает «разработать пользовательский интерфейс для [плагина]».
