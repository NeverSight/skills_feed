Validiert und konfiguriert alle für das Plugin Freedom System erforderlichen Abhängigkeiten. Dies ist eine EIGENSTÄNDIGE Fertigkeit, die VOR Beginn der Plugin-Workflows ausgeführt wird. Es prüft auf Python, Build-Tools, CMake, JUCE und Pluginval und installiert optional fehlende Abhängigkeiten mit Zustimmung des Benutzers. Die Konfiguration wird in .claude/system-config.json zur Verwendung durch andere Skills gespeichert. Wird verwendet, wenn der Benutzer Setup, Installation, Abhängigkeiten oder fehlende Tools erwähnt oder wenn der SessionStart-Hook Konfigurationsprobleme erkennt.
