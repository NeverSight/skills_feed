Проверяет и настраивает все зависимости, необходимые для системы свободы плагинов. Это ОТДЕЛЬНЫЙ навык, который запускается ДО начала рабочих процессов плагина. Он проверяет наличие Python, инструментов сборки, CMake, JUCE и плагинвалов, а также при необходимости устанавливает недостающие зависимости с одобрения пользователя. Конфигурация сохраняется в файле .claude/system-config.json для использования другими навыками. Используйте, когда пользователь упоминает настройку, установку, зависимости, отсутствующие инструменты или когда перехватчик SessionStart обнаруживает проблемы с конфигурацией.
