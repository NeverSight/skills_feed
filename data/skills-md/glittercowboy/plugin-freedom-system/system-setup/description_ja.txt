Plugin Freedom System に必要なすべての依存関係を検証して構成します。これは、プラグイン ワークフローが開始される前に実行されるスタンドアロン スキルです。 Python、ビルド ツール、CMake、JUCE、pluginval をチェックし、オプションでユーザーの承認を得て不足している依存関係をインストールします。構成は、他のスキルで使用できるように .claude/system-config.json に保存されます。ユーザーがセットアップ、インストール、依存関係、不足しているツールについて言及する場合、または SessionStart フックが構成の問題を検出する場合に使用します。
