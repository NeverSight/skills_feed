Valide et configure toutes les dépendances requises pour le Plugin Freedom System. Il s'agit d'une compétence AUTONOME qui s'exécute AVANT le début des flux de travail du plugin. Il vérifie Python, les outils de construction, CMake, JUCE et pluginval, et installe éventuellement les dépendances manquantes avec l'approbation de l'utilisateur. La configuration est enregistrée dans .claude/system-config.json pour être utilisée par d'autres compétences. À utiliser lorsque l'utilisateur mentionne la configuration, l'installation, les dépendances, les outils manquants ou lorsque le hook SessionStart détecte des problèmes de configuration.
