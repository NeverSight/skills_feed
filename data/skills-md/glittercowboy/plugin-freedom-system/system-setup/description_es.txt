Valida y configura todas las dependencias requeridas para el Plugin Freedom System. Esta es una habilidad INDEPENDIENTE que se ejecuta ANTES de que comiencen los flujos de trabajo del complemento. Comprueba Python, herramientas de compilación, CMake, JUCE y pluginval, y opcionalmente instala las dependencias que faltan con la aprobación del usuario. La configuración se guarda en .claude/system-config.json para que la utilicen otras habilidades. Úselo cuando el usuario mencione la configuración, la instalación, las dependencias, las herramientas faltantes o cuando el enlace SessionStart detecte problemas de configuración.
