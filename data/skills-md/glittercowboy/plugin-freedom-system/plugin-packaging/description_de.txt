Erstellen Sie gebrandete PKG-Installationsprogramme für die Plugin-Verteilung. Wird verwendet, wenn der Benutzer anfordert, ein Plugin zu verpacken, ein Installationsprogramm zu erstellen, ein Plugin mit anderen zu teilen oder erwähnt, dass er ein Plugin an jemanden verteilen/senden möchte. Wird durch den Befehl /package oder eine natürliche Sprache wie „Installationsprogramm für TapeAge erstellen“ oder „Paket GainKnob“ aufgerufen.
