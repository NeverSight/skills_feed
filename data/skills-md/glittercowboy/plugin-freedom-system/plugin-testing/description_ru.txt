Проверяет аудиоплагины с помощью автоматических тестов, плагинов или ручного тестирования DAW. Используйте для РУЧНОГО тестирования с помощью команды /test. Примечание. Автоматическая проверка теперь встроена в рабочий процесс (агент проверки запускается после каждого этапа). Этот навык предназначен только для ручного тестирования, после исправления ошибок или когда пользователь упоминает тестирование, проверку, проверку, подключаемость, стабильность, автоматические тесты, запуск тестов, проверку плагина или обеспечение качества.
