Convalida i plug-in audio tramite test automatizzati, pluginval o test DAW manuali. Utilizzare per test MANUALI tramite il comando /test. Nota: la convalida automatica è ora integrata nel flusso di lavoro (l'agente di convalida viene eseguito dopo ogni fase). Questa competenza è riservata ai test manuali, dopo la correzione di bug o quando l'utente menziona test, convalida, convalida, pluginval, stabilità, test automatizzati, esegui test, controlla plug-in o controllo qualità.
