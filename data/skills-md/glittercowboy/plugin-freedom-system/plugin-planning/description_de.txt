Orchestriert die Forschung und Planung der Stufe 0 für JUCE-Plugins – erstellt Architektur.md- und plan.md-Verträge durch Subagentendelegation. Verwenden Sie es, wenn ein kreativer Auftrag vorliegt und das Plugin eine DSP-Architekturspezifikation, eine Komplexitätsbewertung oder eine Implementierungsplanung benötigt. Aufruf über den Befehl /plan, natürliche Sprache (Planarchitektur, Forschungs-DSP) oder als erster Implementierungsschritt.
