Создайте серверы протокола контекста модели (MCP), которые предоставляют Клоду инструменты, ресурсы и подсказки. Используйте при создании пользовательских интеграций, API, источников данных или любого сервера, с которым Клод должен взаимодействовать через протокол MCP. Поддерживает реализации TypeScript и Python.
