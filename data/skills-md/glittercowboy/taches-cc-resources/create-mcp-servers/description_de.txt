Erstellen Sie Model Context Protocol (MCP)-Server, die Claude Tools, Ressourcen und Eingabeaufforderungen zur Verfügung stellen. Verwenden Sie es beim Erstellen benutzerdefinierter Integrationen, APIs, Datenquellen oder anderer Server, mit denen Claude über das MCP-Protokoll interagieren soll. Unterstützt sowohl TypeScript- als auch Python-Implementierungen.
