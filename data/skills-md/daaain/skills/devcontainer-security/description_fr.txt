Guide de configuration de conteneurs de développement VS Code sécurisés pour les agents de codage. À utiliser lors de la création ou du renforcement d'un DevContainer pour mettre en sandbox Claude Code ou d'autres agents de codage, la configuration des proxys de socket Docker, la gestion des vecteurs d'échappement VS Code IPC, la configuration de la prise en charge de git worktree ou la vérification des contrôles de sécurité. Couvre le modèle de menace, l'architecture de défense à trois couches, la configuration de Node.js/pnpm et les tests de vérification.
