Conseils d'experts pour la bibliothèque Streamdown de Vercel - un remplacement de démarque de réaction optimisé pour le streaming pour les applications d'IA.
À utiliser lorsque : (1) Rendu de la démarque générée par l'IA à partir de useChat/streamText,
(2) Création d'interfaces utilisateur de chat avec des réponses en continu,
(3) Migration de React-markdown vers un rendu adapté au streaming,
(4) Configuration des blocs de code (Shiki), mathématiques (KaTeX), diagrammes (Mermaid),
(5) Gestion des démarques incomplètes pendant le streaming AI (remend préprocesseur),
(6) Personnalisation du style de démarque avec les variables Tailwind/CSS,
(7) Sécurisation de la sortie de l'IA avec rehype-harden (protocoles lien/image).
Déclencheurs : Streamdown, démarque de streaming, démarque de chat AI, remplacement de démarque de réaction,
Réponse aux éléments AI, démarque incomplète, correction, thèmes Shiki, diagrammes sirène,
Mathématiques KaTeX, rehype-harden, isAnimating, streaming markdown.
