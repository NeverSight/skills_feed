إرشادات الخبراء للتدقيق وإعادة البناء وتصميم دفاتر ملاحظات Jupyter للتعلم الآلي مع أنماط جودة الإنتاج. يُستخدم عندما: (1) تحليل بنية دفتر الملاحظات وتحديد الأنماط المضادة، (2) اكتشاف مشكلات تسرب البيانات وقابلية التكرار، (3) إعادة هيكلة دفاتر الملاحظات الفوضوية إلى خطوط أنابيب معيارية، (4) إنشاء قوالب لسير عمل تعلم الآلة (EDA، التصنيف، التجارب)، (5) إضافة أدوات إمكانية التكرار (البذر، التسجيل، التقاط البيئة)، (6) تحويل دفاتر الملاحظات إلى نصوص Python النصية، (7) إنشاء تقارير ملخص التجربة. المشغلات على: دفتر ML، تدقيق Jupyter، إعادة بناء دفتر الملاحظات، تسرب البيانات، قالب التجربة، أفضل ممارسات ipynb، دفتر الملاحظات إلى البرنامج النصي، إمكانية التكرار.
