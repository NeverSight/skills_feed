SOTA Docker/Compose アーキテクチャ、実装、リファクタリング、セキュリティ強化。 Dockerfile、docker-compose ファイル、buildx/bake config、.dockerignore、および build/test/scan/publish の CI パイプラインの作成または書き換えなどのコンテナ化タスクに取り組むときに使用します。セキュリティ、正確性、サイズ/パフォーマンス、ベスト プラクティス (最小権限、非ルート、最小限のイメージ、ピン留めされたベース イメージ、BuildKit シークレット、ヘルスチェック) に関して既存のコンテナ設定を監査します。 Docker のビルド/実行の問題をデバッグします。または、正しいネットワーク、ボリューム、シークレット、およびリソース制限を使用して、サービス (DB/キャッシュ/キュー) にわたる開発と本番のワークフローを構成するように設計します。
