Экспертное руководство по созданию приложений Dash с помощью Dash Mantine Components (DMC) v2.4.0.
Используйте при создании информационных панелей, форм и приложений визуализации данных с помощью DMC.
Обложки: темы MantineProvider, реквизиты стиля (m, p, c, bg, w, h), API стилей,
обратные вызовы (базовые, сопоставление с образцом ALL/MATCH/ALLSMALLER, клиентские, фоновые),
многостраничные приложения с Dash Pages, диаграммами (LineChart, BarChart, DonutChart),
средства выбора даты, модальные окна и все более 90 компонентов.
Запускается по: Dash-mantine-Components, DMC, MantineProvider, dmc.Button, dmc.Select,
dmc.Modal, dmc.BarChart, тема Mantine, компоненты пользовательского интерфейса Dash, обратные вызовы Dash,
многостраничное приложение Dash, обратные вызовы с сопоставлением шаблонов, обратные вызовы на стороне клиента, AppShell.
