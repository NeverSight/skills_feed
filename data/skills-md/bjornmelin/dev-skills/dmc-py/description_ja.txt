Dash Mantine Components (DMC) v2.4.0 を使用して Dash アプリケーションを構築するための専門家によるガイダンス。
DMC を使用してダッシュボード、フォーム、データ視覚化アプリを作成するときに使用します。
対象: MantineProvider テーマ、スタイル プロップ (m、p、c、bg、w、h)、スタイル API、
コールバック (基本、パターンマッチング ALL/MATCH/ALLSMALLER、クライアントサイド、バックグラウンド)、
ダッシュ ページ、チャート (LineChart、BarChart、DoughnutChart) を備えた複数ページのアプリ
日付ピッカー、モーダル、および 90 以上のコンポーネントすべて。
トリガー対象:dash-mantine-components、DMC、MantineProvider、dmc.Button、dmc.Select、
dmc.Modal、dmc.BarChart、Mantine テーマ、Dash UI コンポーネント、Dash コールバック、
マルチページ Dash アプリ、パターン マッチング コールバック、クライアント側コールバック、AppShell。
