Плоское представление JSON, вывод схемы и инструмент редактирования. Используйте, когда пользователь предоставляет файл или данные JSON и хочет изучить его структуру, вывести схему или изменить поля. Запускается при: просмотре json, редактировании json, схеме json, структуре json, установке поля json, jstool, анализе json, плоском представлении, книге заказов, плоском json.
