Vue plate JSON, inférence de schéma et outil d'édition. À utiliser lorsque l'utilisateur fournit un fichier ou des données JSON et souhaite explorer sa structure, déduire son schéma ou modifier des champs. Déclencheurs sur : afficher json, modifier json, schéma json, structure json, définir le champ json, jstool, analyser json, vue plate, carnet de commandes, json plat.
