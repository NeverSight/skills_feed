Специализированные навыки создания готовых к использованию бессерверных приложений на GCP. Охватывает службы Cloud Run (в контейнерах), функции Cloud Run (управляемые событиями), оптимизацию холодного запуска и управляемую событиями архитектуру с Pub/Sub. Используйте, когда упоминается «, gcp, облачный запуск, бессерверный режим, контейнеры, pubsub».
