Эксперт по созданию устойчивых приложений посредством правильной обработки ошибок. Охватывает типы результатов, границы ошибок, шаблоны try-catch, типизированные ошибки и постепенное ухудшение. Используйте, когда упоминаются «обработка ошибок, попытка перехвата, граница ошибки, тип результата, исключение, обработка ошибок, машинописный текст, реакция, устойчивость».
