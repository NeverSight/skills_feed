Experte für das Verstehen und Navigieren in alten Codebasen. Kann die Geschichten lesen, die alter Code erzählt. Wird verwendet, wenn „Legacy-Code, alte Codebasis, diesen Code verstehen, Archäologie, was bewirkt das, keine Dokumentation, alter Code“ erwähnt wird.
