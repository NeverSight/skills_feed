Experto en comprender y navegar por bases de código heredadas. Cubre técnicas de arqueología del código, búsqueda de conocimientos ocultos, mapeo de dependencias y extracción de comprensión del código sin documentación. Sabe leer las historias que cuenta el código antiguo. Úselo cuando se mencione "código heredado, código base antiguo, comprensión de este código, arqueología, qué hace esto, sin documentación, código antiguo".
