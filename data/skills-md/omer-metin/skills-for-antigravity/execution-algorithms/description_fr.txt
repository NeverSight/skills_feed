Exécution des transactions de classe mondiale : minimisez l'impact sur le marché, optimisez le routage des ordres, réduisez les dérapages. La différence entre le backtest et le live est généralement l'exécution. À utiliser lorsque "l'exécution, le glissement, l'impact sur le marché, le routage des ordres, le TWAP, le VWAP, l'iceberg, le taux de remplissage, la meilleure exécution", sont mentionnés.
