Créez des applications Slack à l'aide du framework Bolt sur Python, JavaScript et Java. Couvre Block Kit pour les interfaces utilisateur riches, les composants interactifs, les commandes slash, la gestion des événements, les flux d'installation OAuth et l'intégration de Workflow Builder. Concentrez-vous sur les meilleures pratiques pour les applications Slack prêtes pour la production. À utiliser lorsque "slack bot, slack app, bolt framework, block kit, slash command, slack webhook, slack workflow, slack interactive, slack oauth", est mentionné.
