أنشئ تطبيقات Slack باستخدام إطار عمل Bolt عبر Python وJavaScript وJava. يغطي Block Kit لواجهات المستخدم الغنية والمكونات التفاعلية وأوامر الشرطة المائلة ومعالجة الأحداث وتدفقات تثبيت OAuth وتكامل Workflow Builder. ركز على أفضل الممارسات لتطبيقات Slack الجاهزة للإنتاج. يُستخدم عند "Slack bot، Slack app، Bolt Framework، Block Kit، Slash Command، Slack webhook، Slack Workflow، Slack Interactive، Slack Oauth،" المذكورة.
