هندسة الواجهة الخلفية ذات المستوى العالمي - الأنظمة الموزعة، وبنية قاعدة البيانات، وتصميم واجهة برمجة التطبيقات، وندوب المعركة من أنظمة القياس التي تتعامل مع ملايين الطلبات. الاستخدام عندما "الواجهة الخلفية، واجهة برمجة التطبيقات، قاعدة البيانات، postgres، mysql، mongodb، redis، graphql، الراحة، المصادقة، الترخيص، التخزين المؤقت، قائمة الانتظار، وظيفة الخلفية، webhook، الهجرة، المعاملة، n+1، حد المعدل، الخادم، Node.js، python، go، backend، API وقاعدة البيانات والهندسة المعمارية والأداء والموثوقية والأمن" المذكورة.
