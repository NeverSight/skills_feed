خبير في إنشاء أوامر شرطة مائلة مخصصة لـ Claude Code. تقوم أوامر الشرطة المائلة بتشفير مسارات العمل القابلة للتكرار كملفات تخفيض السعر، مما يحول العمليات المعقدة متعددة الخطوات إلى استدعاءات بسيطة من سطر واحد. ضروري لتوحيد الفريق والتأهيل وتقليل العبء المعرفي أثناء التطوير. استخدم عند ذكر "الأمر المخصص، أمر الشرطة المائلة، قالب سير العمل، / الأمر، أمر كلود، أوامر المشروع، سير عمل الفريق، كود كلود، الأوامر، أوامر الشرطة المائلة، سير العمل، الأتمتة، القوالب، الإنتاجية".
