Эксперт по созданию пользовательских слэш-команд для Claude Code. Команды слэша кодируют повторяющиеся рабочие процессы в виде файлов уценки, превращая сложные многоэтапные процессы в простые однострочные вызовы. Необходим для стандартизации команды, адаптации и снижения когнитивной нагрузки во время разработки. Используйте, когда упоминаются «пользовательская команда, косая черта, шаблон рабочего процесса, /команда, команда claude, команды проекта, командные рабочие процессы, claude-код, команды, косая черта-команды, рабочий процесс, автоматизация, шаблоны, производительность».
