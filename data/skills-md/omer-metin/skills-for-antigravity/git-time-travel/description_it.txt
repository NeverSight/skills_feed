Esperto nella navigazione e nella manipolazione della cronologia git. Riguarda la ricerca di bug con bisect, la comprensione dell'evoluzione del codice, il recupero del lavoro perduto e la riscrittura della cronologia in modo sicuro. Comprende che Git History è la tua macchina del tempo. Utilizzare quando viene menzionato "git History, git bisect, trova quando è stato introdotto il bug, ripristina eliminato, riscrivi la cronologia, git colpa, commit perso".
