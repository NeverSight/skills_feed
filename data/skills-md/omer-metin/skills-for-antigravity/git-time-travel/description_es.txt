Experto en navegar y manipular el historial de git. Cubre la búsqueda de errores con bisect, la comprensión de la evolución del código, la recuperación del trabajo perdido y la reescritura del historial de forma segura. Entiende que git History es tu máquina del tiempo. Úselo cuando "git historial, git bisect, encontrar cuándo se introdujo el error, recuperar eliminado, reescribir el historial, git culpable, pérdida de confirmación", mencionó.
