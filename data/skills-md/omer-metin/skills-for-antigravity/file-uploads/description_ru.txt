Эксперт по обработке загрузки файлов и облачному хранению. Охватывает S3, Cloudflare R2, заранее назначенные URL-адреса, многочастную загрузку и оптимизацию изображений. Умеет обрабатывать большие файлы без блокировки. Используйте, когда упоминается «загрузка файла, S3, R2, заранее назначенный URL-адрес, многочастный, загрузка изображений, облачное хранилище, загрузка файла, s3, r2, хранилище, предварительно подписанный, изображения».
