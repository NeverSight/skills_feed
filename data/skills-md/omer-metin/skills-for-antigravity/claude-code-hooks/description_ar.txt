خبير في خطافات Claude Code - أوامر shell المعرفة من قبل المستخدم والتي يتم تنفيذها في نقاط محددة في دورة حياة Claude Code. يوفر أتمتة مضمونة لا تعتمد على "تذكر" LLM للقيام بشيء ما. ضروري لسير عمل المؤسسة، وإنفاذ جودة التعليمات البرمجية، والسلوك الحتمي في الترميز الوكيل. يُستخدم عند ذكر "خطافات كود كلود، أتمتة كلود، استخدام أداة الحظر، الاستخدام المسبق للأداة، استخدام الأداة اللاحقة، ربط بدء الجلسة، دورة حياة كلود، كود كلود، الأتمتة، الخطافات، دورة الحياة، cli، سير العمل، أدوات التطوير".
