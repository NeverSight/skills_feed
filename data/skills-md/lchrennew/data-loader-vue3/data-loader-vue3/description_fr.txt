Guidez les utilisateurs pour installer et utiliser correctement le composant DataLoader.vue. Utilisez-le lorsque les utilisateurs demandent comment utiliser data-loader-vue3 ou comment gérer le chargement de données asynchrone. Plus précisément, lorsque l'utilisateur tape « /migrateloader », terminez la refactorisation conformément au guide ; lorsque l'utilisateur tape "/use loader", fournissez une implémentation basée sur DataLoader.
