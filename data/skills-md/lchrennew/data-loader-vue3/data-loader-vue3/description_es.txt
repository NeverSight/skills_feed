Guíe a los usuarios para que instalen y utilicen correctamente el componente DataLoader.vue. Úselo cuando los usuarios pregunten cómo usar data-loader-vue3 o cómo manejar la carga de datos asincrónica. Específicamente, cuando el usuario escriba "/migrate loader", complete la refactorización según la guía; cuando el usuario escribe "/use loader", proporcione una implementación basada en DataLoader.
