Esta habilidad debe usarse cuando se trabaja en proyectos de Godot Engine. Proporciona conocimiento especializado de los formatos de archivos de Godot (.gd, .tscn, .tres), patrones de arquitectura (basados ​​en componentes, basados ​​en señales, basados ​​en recursos), errores comunes, herramientas de validación, plantillas de código y flujos de trabajo CLI. El comando `godot` está disponible para ejecutar el juego, validar scripts, importar recursos y exportar compilaciones. Utilice esta habilidad para tareas relacionadas con el desarrollo de juegos Godot, depuración de archivos de escenas/recursos, implementación de sistemas de juegos o creación de nuevos componentes Godot.
