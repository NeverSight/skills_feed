Этот навык следует использовать при работе над проектами платформы Valdi. Valdi — это кроссплатформенная среда пользовательского интерфейса Snapchat, которая компилирует компоненты TypeScript/TSX в собственные представления для iOS, Android и macOS. Он предоставляет специализированные знания по архитектуре компонентов, системе сборки Bazel, собственным привязкам, элементам TSX и кроссплатформенной разработке. Используйте этот навык при реализации компонентов Valdi, отладке проблем сборки или создании новых проектов Valdi.
