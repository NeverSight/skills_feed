Esta habilidad debe utilizarse cuando se trabaja en proyectos del marco Valdi. Valdi es el marco de interfaz de usuario multiplataforma de Snapchat que compila componentes TypeScript/TSX en vistas nativas para iOS, Android y macOS. Proporciona conocimientos especializados para la arquitectura de componentes, el sistema de compilación Bazel, enlaces nativos, elementos TSX y desarrollo multiplataforma. Utilice esta habilidad al implementar componentes de Valdi, depurar problemas de compilación o crear nuevos proyectos de Valdi.
