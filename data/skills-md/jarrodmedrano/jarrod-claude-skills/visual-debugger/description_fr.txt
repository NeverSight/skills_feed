Débogage visuel interactif à l’aide de l’automatisation du navigateur Playwright. À utiliser lorsqu'on vous le demande pour inspecter visuellement les sites Web/applications, tester les éléments de l'interface utilisateur, vérifier la fonctionnalité des boutons, vérifier la conception réactive, déboguer les problèmes de mise en page, surveiller les erreurs de la console ou effectuer des tests visuels basés sur le navigateur. Les déclencheurs courants incluent les demandes pour vérifier si les boutons fonctionnent, vérifier le chargement des pages, tester les formulaires, inspecter les éléments, déboguer les pages, naviguer vers les URL et vérifier le comportement, ou toute demande impliquant une inspection et des tests visuels du navigateur.
