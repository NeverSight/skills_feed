Направляйте пользователей через структурированный рабочий процесс для совместной написания документов. Используйте это, когда пользователь хочет написать документ, предложение, техническую спецификацию, запись решения или аналогичный структурированный контент. Рабочий процесс помогает пользователям эффективно передавать контекст, улучшать контент и проверять эффективность документа для читателей. Запускайте его, когда пользователь упоминает написание документа, создание предложения, составление спецификации или аналогичные задачи, связанные с документами.
