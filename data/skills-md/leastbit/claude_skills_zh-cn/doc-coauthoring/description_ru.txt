Направляйте пользователей через структурированный рабочий процесс для совместной написания документов. Используйте его, когда пользователь хочет создавать документы, предложения, технические спецификации, записи решений или аналогичный структурированный контент. Этот рабочий процесс помогает пользователям эффективно передавать контекст, уточнять контент посредством итерации и проверять, подходит ли документ читателям. Срабатывает, когда пользователь упоминает написание документа, создание предложения, составление спецификации или аналогичные задачи по документированию.
