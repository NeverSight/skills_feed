Guíe a los usuarios a través de un flujo de trabajo estructurado para escribir documentos de forma colaborativa. Úselo cuando el usuario quiera crear documentos, propuestas, especificaciones técnicas, registros de decisiones o contenido estructurado similar. Este flujo de trabajo ayuda a los usuarios a transmitir contexto de manera eficiente, refinar el contenido mediante iteración y validar que el documento funciona para sus lectores. Se activa cuando el usuario menciona escribir un documento, crear una propuesta, redactar una especificación o tareas de documentación similares.
