Guíe a los usuarios a través de un flujo de trabajo estructurado para escribir documentos de forma colaborativa. Úselo cuando el usuario quiera escribir un documento, propuesta, especificación técnica, registro de decisión o contenido estructurado similar. El flujo de trabajo ayuda a los usuarios a transmitir contexto de manera eficiente, iterar para mejorar el contenido y validar la efectividad del documento para los lectores. Active esto cuando el usuario mencione escribir un documento, crear una propuesta, redactar una especificación o tareas similares relacionadas con documentos.
