Un conjunto de herramientas para crear artefactos HTML claude.ai complejos y de múltiples componentes utilizando tecnologías web frontend modernas (React, Tailwind CSS, shadcn/ui). Adecuado para artefactos complejos que requieren administración de estado, enrutamiento o componentes shadcn/ui, no para artefactos HTML/JSX simples de un solo archivo.
