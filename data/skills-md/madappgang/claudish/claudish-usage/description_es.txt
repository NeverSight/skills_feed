CRÍTICO: Guía para usar Claudish CLI SOLAMENTE a través de subagentes para ejecutar Claude Code con cualquier modelo de IA (OpenRouter, Gemini, OpenAI, modelos locales). NUNCA ejecute Claudish directamente en el contexto principal a menos que el usuario lo solicite explícitamente. Úselo cuando el usuario mencione modelos de IA externos, Claudish, OpenRouter, Gemini, OpenAI, Ollama o modelos alternativos. Incluye patrones obligatorios de delegación de subagentes, guía de selección de agentes, instrucciones basadas en archivos y reglas estrictas para evitar la contaminación de la ventana de contexto.
