قم بإجراء تحليل Kepner-Tregoe (KT) لحل المشكلات واتخاذ القرار (PSDM) باستخدام العمليات العقلانية الأربع - تقييم الموقف، وتحليل المشكلات، وتحليل القرار، وتحليل المشكلات المحتملة. يُستخدم عند إجراء تحليل السبب الجذري المنظم، أو اتخاذ قرارات معقدة، أو تقييم البدائل باستخدام معايير مرجحة، أو إجراء تحليل مواصفات IS/IS NOT، أو توقع مخاطر التنفيذ، أو استكشاف المشكلات المعقدة وإصلاحها، أو عندما يذكر المستخدم "Kepner-Tregoe"، أو "طريقة KT"، أو "IS/IS NOT"، أو "تقييم الموقف"، أو "تحليل القرار"، أو "الضرورات والرغبات"، أو "تحليل المشكلات المحتملة"، أو يحتاج إلى منهجية منهجية لحل المشكلات. يتضمن مصفوفات المواصفات وتسجيل القرارات ونماذج الجودة وإنشاء التقارير الاحترافية.
