قم بإجراء تحليل شجرة الأخطاء (FTA) لتحديد أسباب فشل النظام وتحليلها بشكل منهجي باستخدام البوابات المنطقية المنطقية. الطريقة الاستنتاجية من أعلى إلى أسفل لهندسة السلامة والموثوقية. يُستخدم عند تحليل فشل النظام، أو تقييم التصميمات المهمة للسلامة، أو حساب احتمالات الفشل، أو تحديد الحد الأدنى من مجموعات القطع، أو تقييم فعالية التكرار، أو عندما يذكر المستخدم "شجرة الأخطاء"، أو "اتفاقية التجارة الحرة"، أو "تحليل فشل النظام"، أو "مجموعات القطع الدنيا"، أو "تحليل السلامة"، أو "احتمال الفشل"، أو "بوابات و/أو"، أو يحتاج إلى تتبع مسارات الفشل من الحدث الأعلى إلى الأحداث الأساسية. يدعم تحليل الهيكل النوعي وحسابات الاحتمالية الكمية.
