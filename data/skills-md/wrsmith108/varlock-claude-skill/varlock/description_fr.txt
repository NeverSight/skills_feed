Gestion sécurisée des variables d’environnement avec Varlock. À utiliser lors de la gestion de secrets, de clés API, d'informations d'identification ou de toute configuration sensible. Garantit que les secrets ne sont jamais exposés dans les terminaux, les journaux, les traces ou le contexte de Claude. Les expressions de déclenchement incluent « variables d'environnement », « secrets », « .env », « clé API », « informations d'identification », « sensible », « Varlock ».
