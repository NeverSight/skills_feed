إدارة واستكشاف أخطاء عمليات نشر Coolify وإصلاحها باستخدام واجهة سطر الأوامر (CLI) وواجهة برمجة التطبيقات (API) الرسمية. استخدم هذه المهارة عندما يحتاج المستخدم إلى مساعدة في إدارة خادم Coolify، أو استكشاف أخطاء WordPress وإصلاحها على Coolify، أو تصحيح مشكلات الخدمة، أو التحقق من شهادات SSL، أو الوصول إلى الحاويات، أو إدارة التطبيقات وقواعد البيانات من خلال Coolify. مفيد بشكل خاص لتشخيص الخدمات وإصلاح مشكلات htaccess ومشكلات REST API وتنفيذ عمليات النشر.
