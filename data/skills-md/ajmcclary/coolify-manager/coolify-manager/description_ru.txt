Управляйте развертываниями Coolify и устраняйте неполадки с помощью официального интерфейса командной строки и API. Используйте этот навык, когда пользователю нужна помощь в управлении сервером Coolify, устранении неполадок WordPress в Coolify, отладке проблем со службами, проверке SSL-сертификатов, доступе к контейнерам или управлении приложениями и базами данных через Coolify. Особенно полезно для диагностики отключенных служб, устранения проблем с .htaccess, проблем REST API и выполнения операций развертывания.
