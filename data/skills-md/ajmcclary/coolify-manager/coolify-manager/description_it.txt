Gestisci e risolvi i problemi delle distribuzioni Coolify utilizzando la CLI e l'API ufficiali. Utilizza questa competenza quando l'utente ha bisogno di aiuto con la gestione del server Coolify, la risoluzione dei problemi di WordPress su Coolify, il debug dei problemi del servizio, il controllo dei certificati SSL, l'accesso ai contenitori o la gestione di applicazioni e database tramite Coolify. Particolarmente utile per diagnosticare servizi inattivi, risolvere problemi .htaccess, problemi API REST ed eseguire operazioni di distribuzione.
