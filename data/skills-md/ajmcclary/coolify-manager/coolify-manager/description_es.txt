Administre y solucione problemas de implementaciones de Coolify utilizando la CLI y API oficiales. Utilice esta habilidad cuando el usuario necesite ayuda con la administración del servidor Coolify, la resolución de problemas de WordPress en Coolify, la depuración de problemas del servicio, la verificación de certificados SSL, el acceso a contenedores o la administración de aplicaciones y bases de datos a través de Coolify. Particularmente útil para diagnosticar servicios inactivos, solucionar problemas de .htaccess, problemas de API REST y realizar operaciones de implementación.
