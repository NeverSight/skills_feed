Платформа Alchemy IaC для TypeScript. Используйте, когда пользователь упоминает Alchemy, хочет настроить инфраструктуру, развернуть Cloudflare Workers, настроить базы данных, KV, R2, очереди, использовать привязки и секреты, настроить режим разработки, использовать адаптеры платформы (Vite, Astro, React Router, SvelteKit, Nuxt, TanStack Start), создавать собственные ресурсы или работать с любым поставщиком Alchemy.
