Directives d'optimisation des performances React et Vite. À utiliser lors de l'écriture, de la révision ou de l'optimisation des composants React créés avec Vite. Déclencheurs sur des tâches impliquant la configuration de Vite, l'optimisation de build, le fractionnement de code, le chargement paresseux, le HMR, la taille du bundle ou les performances de React.
