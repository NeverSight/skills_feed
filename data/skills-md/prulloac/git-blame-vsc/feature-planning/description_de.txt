Ordnen Sie Aufgaben von einer Funktionsaufschlüsselung zu einer optimalen Ausführungsreihenfolge zu, identifizieren Sie Abhängigkeiten und Parallelisierungsmöglichkeiten und erstellen Sie eine agentenbereite Ausführungssequenz. Verwenden Sie diese Funktion, wenn Sie einen Funktionsausfall haben und die richtige Reihenfolge zum Erstellen von Aufgaben bestimmen müssen, die parallel ausgeführt werden können.
