Guida completa per la creazione, la modifica e la convalida dei file di configurazione devcontainer.json. Da utilizzare quando si lavora con le configurazioni del contenitore di sviluppo per qualsiasi scenario, inclusa la creazione di nuovi file devcontainer.json, la modifica di configurazioni esistenti, la risoluzione dei problemi del contenitore di sviluppo, l'aggiunta di funzionalità o script del ciclo di vita, la configurazione di porte e variabili di ambiente, la configurazione di ambienti multi-contenitore con Docker Compose o la comprensione delle proprietà del contenitore di sviluppo e del relativo utilizzo.
