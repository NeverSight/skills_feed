Guide complet pour créer, modifier et valider les fichiers de configuration devcontainer.json. À utiliser lorsque vous travaillez avec des configurations de conteneur de développement pour n'importe quel scénario, y compris la création de nouveaux fichiers devcontainer.json, la modification de configurations existantes, le dépannage des problèmes de conteneur de développement, l'ajout de fonctionnalités ou de scripts de cycle de vie, la configuration de ports et de variables d'environnement, la configuration d'environnements multi-conteneurs avec Docker Compose ou la compréhension des propriétés de devcontainer et de leur utilisation.
