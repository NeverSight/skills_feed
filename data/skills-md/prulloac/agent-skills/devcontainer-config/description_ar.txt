دليل شامل لإنشاء ملفات التكوين devcontainer.json وتحريرها والتحقق من صحتها. يُستخدم عند العمل مع تكوينات Development Container لأي سيناريو بما في ذلك إنشاء ملفات devcontainer.json جديدة، أو تعديل التكوينات الحالية، أو استكشاف مشكلات حاوية التطوير وإصلاحها، أو إضافة ميزات أو نصوص برمجية لدورة الحياة، أو تكوين المنافذ ومتغيرات البيئة، أو إعداد بيئات متعددة الحاويات باستخدام Docker Compose، أو فهم خصائص devcontainer واستخدامها.
