قم بتحليل وإدارة تعليقات طلب سحب GitHub من خلال التصنيف الآلي وتقييم الخطورة والتعامل مع الاستجابة الذكية. يُستخدم عند العمل مع تعليقات العلاقات العامة لفهم أنماط الملاحظات أو تحديد أولويات المشكلات أو معالجة مخاوف المراجعين تلقائيًا. الاستفادة من GitHub MCP Server أو GitHub CLI (gh) أو GitHub REST API بترتيب الأسبقية.
