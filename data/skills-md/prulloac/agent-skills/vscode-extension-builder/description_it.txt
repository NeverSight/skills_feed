Guida per la creazione di estensioni/plugin di Visual Studio Code. Utilizzalo quando gli utenti desiderano creare estensioni VS Code, aggiungere funzionalità a VS Code, creare supporto linguistico, aggiungere temi, creare visualizzazioni Web, implementare debugger o qualsiasi attività di sviluppo di plug-in VS Code. Aiuta a navigare nella documentazione dell'API VS Code Extension e fornisce indicazioni sulle funzionalità di estensione, sulla configurazione del progetto e sulle best practice.
