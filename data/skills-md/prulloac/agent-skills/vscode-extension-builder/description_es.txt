Guía para crear extensiones/complementos de Visual Studio Code. Úselo cuando los usuarios quieran crear extensiones de VS Code, agregar funcionalidad a VS Code, crear soporte de idiomas, agregar temas, crear vistas web, implementar depuradores o cualquier tarea de desarrollo de complementos de VS Code. Ayuda a navegar por la documentación de la API de VS Code Extension y proporciona orientación sobre las capacidades de extensión, la configuración del proyecto y las mejores prácticas.
