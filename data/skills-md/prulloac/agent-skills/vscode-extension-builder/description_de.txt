Leitfaden zum Erstellen von Visual Studio Code-Erweiterungen/Plugins. Verwenden Sie diese Option, wenn Benutzer VS-Code-Erweiterungen erstellen, Funktionen zu VS-Code hinzufügen, Sprachunterstützung erstellen, Themen hinzufügen, Webansichten erstellen, Debugger implementieren oder eine VS-Code-Plugin-Entwicklungsaufgabe ausführen möchten. Hilft beim Navigieren in der VS Code Extension API-Dokumentation und bietet Anleitungen zu Erweiterungsfunktionen, Projekteinrichtung und Best Practices.
