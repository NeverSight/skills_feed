Руководство по созданию расширений/плагинов Visual Studio Code. Используйте, когда пользователи хотят создавать расширения VS Code, добавлять функциональные возможности к VS Code, создавать языковую поддержку, добавлять темы, создавать веб-представления, реализовывать отладчики или решать любые задачи по разработке плагинов VS Code. Помогает ориентироваться в документации VS Code Extension API и предоставляет рекомендации по возможностям расширения, настройке проекта и передовым практикам.
