Guide de création d’extensions/plugins Visual Studio Code. À utiliser lorsque les utilisateurs souhaitent créer des extensions VS Code, ajouter des fonctionnalités à VS Code, créer une prise en charge linguistique, ajouter des thèmes, créer des vues Web, implémenter des débogueurs ou toute tâche de développement de plug-in VS Code. Aide à parcourir la documentation de l’API VS Code Extension et fournit des conseils sur les capacités d’extension, la configuration du projet et les meilleures pratiques.
