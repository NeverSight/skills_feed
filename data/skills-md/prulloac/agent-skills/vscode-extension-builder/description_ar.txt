دليل لإنشاء ملحقات/مكونات إضافية لـ Visual Studio Code. يُستخدم عندما يرغب المستخدمون في إنشاء ملحقات VS Code، أو إضافة وظائف إلى VS Code، أو إنشاء دعم اللغة، أو إضافة سمات، أو إنشاء عروض ويب، أو تنفيذ مصححات الأخطاء، أو أي مهمة تطوير مكون إضافي لـ VS Code. يساعد على التنقل في وثائق VS Code Extension API ويوفر إرشادات حول إمكانيات الامتداد وإعداد المشروع وأفضل الممارسات.
