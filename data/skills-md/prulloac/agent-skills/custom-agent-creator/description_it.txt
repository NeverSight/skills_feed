Capacità di creare agenti personalizzati per VS Code Copilot o OpenCode, aiutando gli utenti a configurare e generare file di agenti con formattazione e configurazioni adeguate. Da utilizzare quando gli utenti desiderano creare assistenti IA specializzati per VS Code Copilot (file .agent.md) o OpenCode (configurazioni agente JSON/markdown) con strumenti, prompt, modelli e comportamenti specifici. Se l'utente non è specifico riguardo alla piattaforma di destinazione, chiedigli di specificare Copilot o OpenCode.
