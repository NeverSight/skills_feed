مهارة إنشاء وكلاء مخصصين لـ VS Code Copilot أو OpenCode، مما يساعد المستخدمين على تكوين وإنشاء ملفات الوكيل بالتنسيق والتكوينات المناسبة. يُستخدم عندما يريد المستخدمون إنشاء مساعدين متخصصين للذكاء الاصطناعي لـ VS Code Copilot (ملفات .agent.md) أو OpenCode (تكوينات وكيل JSON/markdown) باستخدام أدوات ومطالبات ونماذج وسلوكيات محددة. إذا لم يكن المستخدم محددًا بشأن النظام الأساسي المستهدف، فاطلب منه تحديد Copilot أو OpenCode.
