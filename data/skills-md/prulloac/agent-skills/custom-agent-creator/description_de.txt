Fähigkeit, benutzerdefinierte Agenten für VS Code Copilot oder OpenCode zu erstellen und Benutzern dabei zu helfen, Agentendateien mit der richtigen Formatierung und Konfiguration zu konfigurieren und zu generieren. Verwenden Sie diese Option, wenn Benutzer spezielle KI-Assistenten für VS Code Copilot (.agent.md-Dateien) oder OpenCode (JSON/Markdown-Agent-Konfigurationen) mit bestimmten Tools, Eingabeaufforderungen, Modellen und Verhaltensweisen erstellen möchten. Wenn der Benutzer keine genauen Angaben zur Zielplattform macht, bitten Sie ihn, Copilot oder OpenCode anzugeben.
