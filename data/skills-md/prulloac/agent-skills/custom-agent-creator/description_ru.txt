Навыки создания пользовательских агентов для VS Code Copilot или OpenCode, помогающие пользователям настраивать и генерировать файлы агентов с правильным форматированием и конфигурациями. Используйте, когда пользователи хотят создать специализированных помощников искусственного интеллекта для VS Code Copilot (файлы .agent.md) или OpenCode (конфигурации агента JSON/markdown) с конкретными инструментами, подсказками, моделями и поведением. Если пользователь не уточняет целевую платформу, попросите его указать Copilot или OpenCode.
