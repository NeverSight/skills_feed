Habilidad para crear agentes personalizados para VS Code Copilot u OpenCode, ayudando a los usuarios a configurar y generar archivos de agentes con el formato y las configuraciones adecuados. Úselo cuando los usuarios quieran crear asistentes de IA especializados para VS Code Copilot (archivos .agent.md) u OpenCode (configuraciones de agente JSON/markdown) con herramientas, indicaciones, modelos y comportamientos específicos. Si el usuario no es específico acerca de la plataforma de destino, pídale que especifique Copilot u OpenCode.
