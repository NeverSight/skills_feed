Навыки внесения изменений в git в соответствии с рекомендациями репозитория. Используйте, когда необходимо зафиксировать проиндексированные или непроиндексированные изменения путем чтения файлов CONTRIBUTING на предмет стандартов, анализа изменений с помощью статуса git и различий, группировки связанных файлов и выполнения коммитов в соответствии с рекомендациями или обычных коммитов в качестве резервного варианта.
