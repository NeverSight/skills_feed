Compétence pour valider les modifications de Git en suivant les directives du référentiel. À utiliser lorsque vous avez besoin de valider des modifications planifiées ou non en lisant les fichiers CONTRIBUANTS pour les normes, en analysant les modifications avec git status et diff, en regroupant les fichiers associés et en effectuant des validations conformément aux directives ou aux validations conventionnelles comme solution de secours.
