Fähigkeit zum Festschreiben von Git-Änderungen gemäß den Repository-Richtlinien. Verwenden Sie diese Option, wenn Sie gestaffelte oder nicht bereitgestellte Änderungen festschreiben müssen, indem Sie CONTRIBUTING-Dateien für Standards lesen, Änderungen mit Git-Status und Diff analysieren, verwandte Dateien gruppieren und Festschreibungen gemäß Richtlinien oder herkömmlichen Festschreibungen als Fallback durchführen.
