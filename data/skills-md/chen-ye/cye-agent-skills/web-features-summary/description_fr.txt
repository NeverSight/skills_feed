Cette compétence doit être utilisée lorsque l'utilisateur effectue du « développement frontend », travaille avec « HTML » ou « CSS » ou écrit « JavaScript » (y compris « JS côté serveur », « Node.js » ou « script JS »). Il fournit des conseils sur les fonctionnalités Web modernes qui ont récemment atteint le statut de référence multi-navigateurs.
