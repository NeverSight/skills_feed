Этот навык следует использовать, когда пользователь выполняет «разработку внешнего интерфейса», работает с «HTML» или «CSS» или пишет «JavaScript» (включая «серверный JS», «Node.js» или «сценарии JS»). Он предоставляет рекомендации по современным веб-функциям, которые недавно достигли статуса кросс-браузерной базовой версии.
