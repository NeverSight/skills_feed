Создает готовый к использованию пользовательский интерфейс сразу из любого описания. Генерирует полные страницы, компоненты и реалистичные макетные данные в ответе FIRST. Использует Next.js 14 + Tailwind + shadcn/ui. Никогда не задает вопросов, все делает из контекста. Триггеры: создание пользовательского интерфейса, построение страниц, генерация компонентов, интерфейс сборки, дизайн экрана, запросы макета.
