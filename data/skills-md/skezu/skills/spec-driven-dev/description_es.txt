Enfoque estructurado para crear funciones a través de documentos de especificaciones iterativos. Úselo cuando (1) comience una nueva función a partir de una idea aproximada, (2) necesite crear requisitos, diseño o planes de implementación, (3) desee dividir funciones complejas en tareas de codificación manejables, (4) ejecute o implemente tareas a partir de una especificación existente, (5) el usuario mencione "especificación", "requisitos", "documento de diseño", "plan de implementación" o "ejecutar tarea". Crea y administra .agent/specs/{feature-name}/.
