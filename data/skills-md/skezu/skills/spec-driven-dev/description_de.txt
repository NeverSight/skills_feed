Strukturierter Ansatz zum Erstellen von Features durch iterative Spezifikationsdokumente. Verwenden Sie diese Option, wenn (1) eine neue Funktion ausgehend von einer groben Idee gestartet wird, (2) Anforderungen, Design- oder Implementierungspläne erstellt werden müssen, (3) komplexe Funktionen in überschaubare Codierungsaufgaben zerlegt werden sollen, (4) Aufgaben aus einer vorhandenen Spezifikation ausgeführt oder implementiert werden, (5) der Benutzer „Spezifikation“, „Anforderungen“, „Entwurfsdokument“, „Implementierungsplan“ oder „Aufgabe ausführen“ erwähnt. Erstellt und verwaltet .agent/specs/{feature-name}/.
