Créez des diagrammes Excalidraw sous forme de fichiers JSON pour les organigrammes, les parcours utilisateur, les architectures système, les wireframes et la documentation visuelle. À utiliser lorsque l'utilisateur demande à créer des diagrammes, des organigrammes, des représentations visuelles, des diagrammes d'architecture, des cartes mentales ou toute visualisation compatible Excalidraw. Génère des fichiers .excalidraw.json qui peuvent être ouverts dans excalidraw.com ou dans toute application compatible Excalidraw.
