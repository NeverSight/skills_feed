HostelOS/Almanik PMS 的專業知識 - 使用 Node.js、Express、SQLite 和 vanilla JavaScript 建立的旅館物業管理系統。在以下工作中使用：(1) 使用 Express.js 和 SQLite 進行後端 API 開發，(2) 資料庫模式設計和遷移，(3) 使用基於會話的身份驗證和 CASL 進行 RBAC 身份驗證，(4) 預訂狀態機和業務邏輯，(5) 財務指標計算（ADR、RevPAB、入住率、LOS），(6) 使用 vanQA與 OTA 的 iCal 同步（Booking.com、Airbnb），(9) 結帳/入住工作流程，(10) 旅遊佣金和賓客管理。觸發器：「HostelOS」、「Almanik」、「PMS」、「旅舍管理」、「預訂」、「結帳」、「入住」、「camas」、「huespedes」、「ADR」、「ocupacion」、「metricas hoteleras」、「旅遊佣金」、「錢箱」、「前台」。
