HostelOS/Almanik PMS 的专业知识 - 使用 Node.js、Express、SQLite 和 vanilla JavaScript 构建的旅馆物业管理系统。在以下工作中使用：(1) 使用 Express.js 和 SQLite 进行后端 API 开发，(2) 数据库模式设计和迁移，(3) 使用基于会话的身份验证和 CASL 进行 RBAC 身份验证，(4) 预订状态机和业务逻辑，(5) 财务指标计算（ADR、RevPAB、入住率、LOS），(6) 使用 vanilla JS 的前端 SPA，(7) QA 测试和调试 PMS 功能，(8) 与 OTA 的 iCal 同步（Booking.com、Airbnb），(9) 结账/入住工作流程，(10) 旅游佣金和宾客管理。触发器：“HostelOS”、“Almanik”、“PMS”、“旅舍管理”、“预订”、“结帐”、“入住”、“camas”、“huespedes”、“ADR”、“ocupacion”、“metricas hoteleras”、“旅游佣金”、“钱箱”、“前台”。
