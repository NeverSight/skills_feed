Node.js、Express、SQLite、バニラ JavaScript で構築されたホステル プロパティ管理システムである HostelOS/Almanik PMS に関する専門知識。次の作業に使用します: (1) Express.js と SQLite を使用したバックエンド API 開発、(2) データベース スキーマの設計と移行、(3) セッションベースの認証と CASL を使用した RBAC 認証、(4) 予約ステート マシンとビジネス ロジック、(5) 財務指標の計算 (ADR、RevPAB、占有、LOS)、(6) バニラ JS を使用したフロントエンド SPA、(7) QA テストと PMS 機能のデバッグ、 (8) iCal と OTA (Booking.com、Airbnb) の同期、(9) チェックアウト/チェックインのワークフロー、(10) ツアーコミッションとゲスト管理。トリガー: 「HostelOS」、「Almanik」、「PMS」、「ホステル管理」、「予約」、「チェックアウト」、「チェックイン」、「カマス」、「ウエスペデス」、「ADR」、「オキュパシオン」、「メトリカス ホテルエラ」、「ツアー コミッション」、「金庫」、「フロント デスク」。
