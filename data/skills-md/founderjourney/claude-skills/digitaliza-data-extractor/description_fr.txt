Extrayez et préparez les données client pour les cartes numériques de style digitalizaweb.vercel.app LinkTree.
À utiliser lorsque : (1) Traitement de dossiers de clients de restaurants/entreprises contenant des captures d'écran, du code HTML récupéré ou des données LinkTree,
(2) Extraction des couleurs de marque à partir de logos/images,
(3) Génération de JSON prêt pour Digitaliza avec slug, nom, liens, couleurs et configuration de thème,
(4) Traitement par lots de plusieurs dossiers clients pour plus de 100 projets de restaurants,
(5) L'utilisateur mentionne "digitaliza", "tarjeta digital", "linktree", "extraer datos de cliente" ou "procesar carpeta de restaurante".
