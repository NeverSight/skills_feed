Аналитика умных денег в реальном времени для Polymarket, Hyperliquid и Meteora через API MetEngine x402 с оплатой за запрос. Используйте, когда задачи требуют оценки кошелька, обнаружения инсайдеров, анализа позиций, рыночного потока или аналитики LP/AMM на этих платформах, особенно когда для доступа требуется подпись платежа Solana USDC вместо ключей API.
