Crea commit git ben formattati con formato commit ed emoji convenzionali. Esegue automaticamente i controlli pre-commit (lint, build, documenti), analizza le modifiche, suggerisce la suddivisione quando appropriato e genera messaggi di commit convenzionali con prefisso emoji. Da utilizzare durante la creazione di commit in qualsiasi repository per garantire una cronologia dei commit coerente e professionale.
