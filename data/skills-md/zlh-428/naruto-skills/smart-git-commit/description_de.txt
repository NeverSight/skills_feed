Erstellen Sie gut formatierte Git-Commits mit herkömmlichem Commit-Format und Emoji. Führt automatisch Pre-Commit-Prüfungen durch (Lint, Build, Docs), analysiert Änderungen, schlägt bei Bedarf eine Aufteilung vor und generiert konventionelle Commit-Nachrichten mit Emoji-Präfix. Verwenden Sie diese Option beim Erstellen von Commits in einem beliebigen Repository, um einen konsistenten, professionellen Commit-Verlauf sicherzustellen.
