Cree confirmaciones de git bien formateadas con emoji y formato de confirmación convencional. Ejecuta automáticamente comprobaciones previas a la confirmación (lint, build, docs), analiza cambios, sugiere dividir cuando sea apropiado y genera mensajes de confirmación convencionales con prefijo emoji. Úselo al crear confirmaciones en cualquier repositorio para garantizar un historial de confirmaciones profesional y coherente.
