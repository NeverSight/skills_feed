Создавайте хорошо отформатированные коммиты git с использованием обычного формата коммитов и смайлов. Автоматически запускает проверки перед фиксацией (lint, build, docs), анализирует изменения, предлагает разделение, когда это необходимо, и генерирует обычные сообщения о фиксации с префиксом смайликов. Используйте при создании коммитов в любом репозитории, чтобы обеспечить единообразную и профессиональную историю коммитов.
