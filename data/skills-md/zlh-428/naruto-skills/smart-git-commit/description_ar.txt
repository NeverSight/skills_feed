قم بإنشاء التزامات git منسقة بشكل جيد باستخدام تنسيق الالتزام التقليدي والرموز التعبيرية. يقوم تلقائيًا بتشغيل فحوصات ما قبل الالتزام (lint، build، docs)، ويحلل التغييرات، ويقترح التقسيم عندما يكون ذلك مناسبًا، وينشئ رسائل التزام تقليدية مسبوقة برموز تعبيرية. يُستخدم عند إنشاء التزامات في أي مستودع لضمان سجل التزام متسق واحترافي.
