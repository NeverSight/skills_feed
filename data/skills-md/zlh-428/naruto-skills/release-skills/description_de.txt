Release-Workflow für das Naruto-Skills-Plugin. Verwenden Sie diese Option, wenn der Benutzer „Release“, „发布“, „push“, „推送“, „new version“, „新版本“, „bump version“, „更新版本“ sagt oder Änderungen auf Remote veröffentlichen möchte. Analysiert Änderungen seit dem letzten Tag, aktualisiert CHANGELOG (EN/CN), erhöht die Marketplace.json-Version, schreibt fest und erstellt ein Versions-Tag. MUSS vor jedem Git-Push mit nicht festgeschriebenen Fähigkeitsänderungen verwendet werden.
