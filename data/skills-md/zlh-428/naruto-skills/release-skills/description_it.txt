Flusso di lavoro di rilascio per il plugin Naruto-Skills. Da utilizzare quando l'utente dice "rilascio", "发布", "push", "推送", "nuova versione", "新版本", "versione bump", "更新版本" o desidera pubblicare le modifiche sul telecomando. Analizza le modifiche dall'ultimo tag, aggiorna CHANGELOG (EN/CN), aumenta la versione di marketplace.json, esegue il commit e crea il tag della versione. DEVE essere utilizzato prima di qualsiasi push git con modifiche alle abilità non vincolate.
