Aktualisieren Sie OpenClaw jede Nacht automatisch. Verwenden Sie diese Option, wenn der Benutzer möchte, dass sein Agent auf dem neuesten Stand bleibt 
mit der neuesten OpenClaw-Version. Führt einen Cron-Job aus, der das Gateway stoppt, das Update ausführt und neu startet.
Wird ausgelöst, wenn der Benutzer Folgendes erwähnt: „Aktualisieren Sie sich selbst“, „Automatisches Update“, „Nächtliches Update“, „Auf dem Laufenden bleiben“, 
„Selbstaktualisierung“ oder jede Variante der Aktualisierung des Agenten.
