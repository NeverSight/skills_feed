Система продуктивности Claude Code, основанная на советах команды Claude Code. Предоставляет команды для параллельных рабочих деревьев, проверки планов, обслуживания CLAUDE.md, поиска технических долгов, автоматического исправления, обработки кода, элегантных рефакторингов, визуальных объяснений и оркестровки субагентов. Используйте, если хотите более эффективно работать с Claude Code.
