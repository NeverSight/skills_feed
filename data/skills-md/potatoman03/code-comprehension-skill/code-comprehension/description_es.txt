Cuestione a los usuarios sobre el código generado por IA para desarrollar la comprensión y mejorar las habilidades de indicaciones.
Se activa: /comprensión de código con --setup, --quiz, --status, --mode, --skip, --config, --reset.
También se activa automáticamente después de generar código para realizar un seguimiento de los cambios en las pruebas.
Ayuda a los usuarios a comprender su código base y detectar problemas antes de comprometerse.
