قم باختبار المستخدمين حول التعليمات البرمجية التي تم إنشاؤها بواسطة الذكاء الاصطناعي لبناء الفهم وتحسين مهارات الطلب.
يتم تشغيله على: / فهم الكود مع --setup، --quiz، --status، --mode، --skip، --config، --reset.
يتم تنشيطه تلقائيًا أيضًا بعد إنشاء التعليمات البرمجية لتتبع التغييرات للاختبار.
يساعد المستخدمين على فهم قاعدة التعليمات البرمجية الخاصة بهم والتعرف على المشكلات قبل الالتزام.
