Опросите пользователей о коде, сгенерированном ИИ, чтобы лучше понять и улучшить навыки подсказок.
Запускается по: /code-comprehension с --setup, --quiz, --status, --mode, --skip, --config, --reset.
Также активируется автоматически после генерации кода для отслеживания изменений в опросе.
Помогает пользователям понять свою кодовую базу и выявить проблемы перед фиксацией.
