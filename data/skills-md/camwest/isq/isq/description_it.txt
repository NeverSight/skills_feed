Utilizza la CLI isq per la gestione immediata dei problemi GitHub, Linear e JIRA offline-first. Utilizza questa abilità quando l'utente desidera elencare problemi, creare problemi, commentare problemi, iniziare a lavorare su un problema (crea git worktree), gestire obiettivi (traguardi/progetti), sincronizzare repository o lavorare con problemi offline. isq fornisce letture inferiori al millisecondo da una cache SQLite locale e si integra con git worktrees per flussi di lavoro di sviluppo senza soluzione di continuità.
