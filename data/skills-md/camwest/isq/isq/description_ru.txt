Используйте интерфейс командной строки isq для мгновенного автономного управления проблемами GitHub, Linear и JIRA. Используйте этот навык, когда пользователь хочет составить список проблем, создать проблемы, прокомментировать проблемы, начать работу над проблемой (создает рабочее дерево git), управлять целями (этапами/проектами), синхронизировать репозитории или работать с проблемами в автономном режиме. isq обеспечивает чтение из локального кэша SQLite за доли миллисекунды и интегрируется с рабочими деревьями git для упрощения рабочих процессов разработки.
