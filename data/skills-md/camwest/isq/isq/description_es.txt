Utilice la CLI de isq para una gestión instantánea y fuera de línea de problemas de GitHub, Linear y JIRA. Utilice esta habilidad cuando el usuario quiera enumerar problemas, crear problemas, comentar problemas, comenzar a trabajar en un problema (crea un árbol de trabajo de git), administrar objetivos (hitos/proyectos), sincronizar repositorios o trabajar con problemas sin conexión. isq proporciona lecturas de menos de milisegundos desde un caché SQLite local y se integra con árboles de trabajo de git para flujos de trabajo de desarrollo fluidos.
