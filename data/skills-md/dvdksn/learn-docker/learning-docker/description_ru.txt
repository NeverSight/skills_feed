Загружается автоматически, когда пользователь просит изучить Docker (например, «Научи меня Docker», «Проведи меня через Docker», «Я хочу изучить Docker», «Урок по Docker»). Интерактивное руководство, в котором Клод выступает в роли инструктора учебного курса по программированию, обучая шаг за шагом с контрольными точками и проверкой.
