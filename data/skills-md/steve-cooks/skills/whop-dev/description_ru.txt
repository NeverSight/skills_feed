Создавайте приложения Next.js для платформы Whop. Для пустых репозиториев помогает пользователям перейти от идеи к развернутому приложению посредством обнаружения и настройки. Для существующих проектов представлены рекомендации по аутентификации, платежам, веб-перехватчикам и пользовательскому интерфейсу. Триггеры «создайте мне приложение Whop», @whop/sdk,verifyUserToken, checkAccess, потоки оформления заказа, обработку веб-перехватчиков, страницы Experience/Dashboard, Frosted UI. Делегирует сопутствующие навыки (дизайн интерфейса, лучшие практики vercel-react, лучшие практики supabase-postgres, рекомендации по веб-дизайну) для проектирования, производительности и базы данных.
