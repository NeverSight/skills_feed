Cree aplicaciones Next.js para la plataforma Whop. Para repositorios vacíos, guía a los usuarios desde la idea hasta la aplicación implementada a través del descubrimiento y la configuración. Para proyectos existentes, proporciona mejores prácticas para autenticación, pagos, webhooks y UI. Activadores en "crearme una aplicación Whop", @whop/sdk, verificarUserToken, checkAccess, flujos de pago, manejo de webhooks, páginas de experiencia/panel, interfaz de usuario escarchada. Delega habilidades complementarias (diseño frontend, mejores prácticas de vercel-react, mejores prácticas de supabase-postgres, pautas de diseño web) para el diseño, el rendimiento y la base de datos.
