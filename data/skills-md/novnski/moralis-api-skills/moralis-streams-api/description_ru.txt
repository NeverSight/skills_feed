Мониторинг событий блокчейна в реальном времени с помощью веб-хуков. Используйте, когда пользователь спрашивает о настройке веб-перехватчиков, потоковой передаче событий в реальном времени, мониторинге адресов кошельков, отслеживании передачи токенов в режиме реального времени, создании/обновлении/удалении потоков, добавлении/удалении адресов из потоков или получении событий блокчейна по мере их возникновения. Поддерживает все цепочки EVM. НЕ для запроса исторического или текущего состояния блокчейна — вместо этого используйтеmoralis-data-api.
