Interrogez les données de la blockchain Web3 à partir de l'API Moralis. À utiliser lorsque l'utilisateur pose des questions sur les données du portefeuille (soldes, jetons, NFT, historique des transactions, rentabilité, valeur nette), les données sur les jetons (prix, métadonnées, paires DEX, analyses, scores de sécurité), les données NFT (métadonnées, transferts, caractéristiques, rareté, prix plancher), les positions DeFi, les données d'entité/label pour les échanges et les fonds, ou les données de bloc et de transaction. Prend en charge les chaînes EVM (Ethereum, Polygon, BSC, Arbitrum, Base, Optimism, Avalanche, etc.) et Solana. PAS pour le streaming en temps réel - utilisez plutôt moralis-streams-api.
