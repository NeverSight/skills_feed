Interroga i dati blockchain Web3 dall'API Moralis. Da utilizzare quando l'utente chiede informazioni sui dati del portafoglio (saldi, token, NFT, cronologia delle transazioni, redditività, patrimonio netto), dati sui token (prezzi, metadati, coppie DEX, analisi, punteggi di sicurezza), dati NFT (metadati, trasferimenti, caratteristiche, rarità, prezzi minimi), posizioni DeFi, dati su entità/etichette per scambi e fondi o dati su blocchi e transazioni. Supporta catene EVM (Ethereum, Polygon, BSC, Arbitrum, Base, Optimism, Avalanche, ecc.) e Solana. NON per lo streaming in tempo reale: usa invece moralis-streams-api.
