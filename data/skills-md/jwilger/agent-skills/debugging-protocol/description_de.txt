Systematisches 4-Phasen-Debugging: Den Fehler verstehen, Hypothesen aufstellen, jeweils eine Änderung testen und mit Zuversicht beheben. Aktivieren Sie diese Option, wenn Tests unerwartet fehlschlagen, Fehler auftreten, das Verhalten falsch ist oder etwas, das zuvor funktioniert hat, jetzt nicht mehr funktioniert. Auslöser bei: „Debug“, „Warum schlägt das fehl“, „Testfehler“, „unerwarteter Fehler“, „Fehler“, „kaputt“.
