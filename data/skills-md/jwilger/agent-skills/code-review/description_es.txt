Protocolo de revisión de código de tres etapas: cumplimiento de especificaciones, calidad del código e integridad del dominio. Actívelo al revisar el código, preparar relaciones públicas, evaluar la calidad de la implementación o verificar que el código coincida con los requisitos. Activadores en: "revisar este código", "preparar relaciones públicas", "verificar implementación", "calidad del código", "coincide esto con las especificaciones".
