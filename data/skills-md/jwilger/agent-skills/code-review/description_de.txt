Dreistufiges Code-Review-Protokoll: Spezifikationskonformität, Codequalität und Domänenintegrität. Aktivieren Sie diese Funktion, wenn Sie Code überprüfen, PRs vorbereiten, die Implementierungsqualität bewerten oder überprüfen, ob der Code den Anforderungen entspricht. Wird ausgelöst bei: „Diesen Code überprüfen“, „PR vorbereiten“, „Implementierung prüfen“, „Codequalität“, „Entspricht dies der Spezifikation“.
