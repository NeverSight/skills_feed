Трехэтапный протокол проверки кода: соответствие спецификациям, качество кода и целостность домена. Активируйте при проверке кода, подготовке запросов на запросы, оценке качества реализации или проверке соответствия кода требованиям. Срабатывает по: «проверьте этот код», «подготовьте PR», «проверьте реализацию», «качество кода», «соответствует ли это спецификации».
