Scomposizione del lavoro, ordinamento delle dipendenze e monitoraggio dello stato per le attività software. Attivalo quando suddividi le funzionalità in attività, gestisci elementi di lavoro, monitora le dipendenze, crea storie o epiche o chiedi su cosa lavorare dopo. Funziona con qualsiasi strumento di attività: attività native di cablaggio, dot CLI, problemi di GitHub o monitoraggio basato su file.
