Format de point de contrôle structuré pour demander une contribution humaine. Lorsqu'un agent a besoin d'une décision, il doit s'arrêter, présenter le contexte, afficher les options et attendre. Activez-le lorsque vous déléguez à des sous-agents, exécutez des tâches en arrière-plan ou atteignez tout point de décision nécessitant un jugement humain.
