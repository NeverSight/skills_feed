Strukturiertes Prüfpunktformat zum Anfordern menschlicher Eingaben. Wenn ein Agent eine Entscheidung benötigt, muss er anhalten, den Kontext präsentieren, Optionen anzeigen und warten. Aktivieren Sie diese Funktion, wenn Sie an Subagenten delegieren, Hintergrundaufgaben ausführen oder einen Entscheidungspunkt erreichen, der menschliches Urteilsvermögen erfordert.
