Conseils d'experts pour le développement .NET MAUI Blazor Hybrid combinant des applications mobiles/de bureau natives avec l'interface utilisateur Web Blazor.
À utiliser lorsque vous travaillez avec : (1) les applications .NET MAUI Blazor Hybrid, (2) les composants BlazorWebView, (3) les applications multiplateformes partageant l'interface utilisateur Blazor,
(4) Intégration de la plateforme dans Blazor Hybrid, (5) MVVM avec les composants Blazor, (6) Navigation entre les pages MAUI et les composants Razor,
(7) Bibliothèques de classes Razor partagées (RCL) pour MAUI et Web. S'applique à tout projet .NET MAUI utilisant Blazor pour l'interface utilisateur.
