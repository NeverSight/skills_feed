ネイティブ モバイル/デスクトップ アプリと Blazor Web UI を組み合わせた .NET MAUI Blazor ハイブリッド開発の専門家によるガイダンス。
作業する場合に使用します: (1) .NET MAUI Blazor Hybrid アプリ、(2) BlazorWebView コンポーネント、(3) Blazor UI を共有するクロスプラットフォーム アプリ、
(4) Blazor Hybrid でのプラットフォーム統合、(5) Blazor コンポーネントとの MVVM、(6) MAUI ページと Razor コンポーネント間のナビゲーション、
(7) MAUI および Web 用の共有 Razor クラス ライブラリ (RCL)。 Blazor for UI を使用するすべての .NET MAUI プロジェクトに適用されます。
