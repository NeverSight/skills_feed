Orientación experta para el desarrollo híbrido .NET MAUI Blazor que combina aplicaciones nativas móviles/de escritorio con la interfaz de usuario web de Blazor.
Úselo cuando trabaje con: (1) aplicaciones .NET MAUI Blazor Hybrid, (2) componentes BlazorWebView, (3) aplicaciones multiplataforma que comparten la interfaz de usuario de Blazor,
(4) Integración de plataforma en Blazor Hybrid, (5) MVVM con componentes Blazor, (6) Navegación entre páginas MAUI y componentes Razor,
(7) Bibliotecas de clases Razor compartidas (RCL) para MAUI y Web. Se aplica a cualquier proyecto .NET MAUI que utilice Blazor para UI.
