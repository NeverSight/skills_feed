تطوير React 19+ الحديث مع مكونات الخادم والإجراءات والخطافات وتكامل TypeScript وتحسين الأداء. يُستخدم عند إنشاء تطبيقات الويب React، أو تنفيذ مكونات الخادم، أو استخدام الإجراءات للتعامل مع النماذج، أو العمل مع الخطافات الجديدة (استخدم، useActionState، useOptimistic، useFormStatus)، أو إعداد مشاريع React باستخدام Vite أو Next.js، أو تحسين أداء React.
