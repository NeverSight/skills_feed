Développement moderne de React 19+ avec composants serveur, actions, hooks, intégration TypeScript et optimisation des performances. À utiliser lors de la création d'applications Web React, de l'implémentation de composants serveur, de l'utilisation d'actions pour la gestion des formulaires, de l'utilisation de nouveaux hooks (use, useActionState, useOptimistic, useFormStatus), de la configuration de projets React avec Vite ou Next.js ou de l'optimisation des performances de React.
