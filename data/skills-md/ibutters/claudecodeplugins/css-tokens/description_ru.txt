Создавайте систему токенов, используя пользовательские свойства CSS для единообразного стиля. Используйте для создания цветовых палитр, интервалов, систем типографики и тем (темный/светлый режим). Запускается по запросам маркеров дизайна, переменных CSS, тем, цветовых систем, масштабов интервалов или единообразного стиля.
