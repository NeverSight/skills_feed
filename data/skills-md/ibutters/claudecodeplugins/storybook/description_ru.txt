Сборник рассказов 8 для документации и тестирования компонентов React. Используйте для создания историй, документирования компонентов с помощью элементов управления/действий, визуального тестирования и документации MDX. Запускается по запросам историй из сборника рассказов, документации по компонентам, визуального тестирования или интерактивных демонстраций компонентов.
