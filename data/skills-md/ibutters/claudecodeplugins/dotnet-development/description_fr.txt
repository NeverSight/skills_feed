Directives complètes de développement .NET couvrant les principes DDD, SOLID, les API REST ASP.NET Core et les meilleures pratiques C#.
À utiliser lorsque vous travaillez avec : (1) des fichiers de code C# (.cs), (2) des fichiers de projet .NET (.csproj, .sln), (3) des applications ASP.NET Core,
(4) Implémentations de conception pilotée par domaine, (5) Développement d'API REST, (6) Entity Framework Core,
(7) Tests unitaires/d'intégration dans .NET. Déclencheurs spécifiques pour : les classes héritant d’AggregateRoot/Entity,
des services avec des dépendances IRepository, des contrôleurs héritant de ControllerBase ou des fichiers dans des dossiers Domaine/Application/Infrastructure.
