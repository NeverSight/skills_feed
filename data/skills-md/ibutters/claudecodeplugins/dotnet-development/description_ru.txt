Комплексные рекомендации по разработке .NET, охватывающие принципы DDD, SOLID, API REST ASP.NET Core и лучшие практики C#.
Используйте при работе с: (1) файлами кода C# (.cs), (2) файлами проекта .NET (.csproj, .sln), (3) приложениями ASP.NET Core,
(4) Реализация доменно-ориентированного проектирования, (5) Разработка REST API, (6) Entity Framework Core,
(7) Модульное/интеграционное тестирование в .NET. В частности, триггеры для: классов, наследующих AggregateRoot/Entity,
сервисы с зависимостями IRepository, контроллеры, наследующие ControllerBase, или файлы в папках Domain/Application/Infrastructure.
