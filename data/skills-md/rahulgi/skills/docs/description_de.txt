Agent-First-Projektdokumentation. Erstellt und pflegt eine für KI-Agenten entwickelte Dokumentationsarchitektur – ROADMAP, Designdokumente, Protokolle und mehr. Verwenden Sie diese Option, wenn der Benutzer „/docs“ sagt, die Einrichtung einer Projektdokumentation wünscht, Dokumente aktualisieren möchte oder wenn er ein neues Projekt startet, dem die Dokumentationsstruktur fehlt. Verwenden Sie es auch, wenn der Benutzer eine Funktion planen (ein Designdokument erstellen) oder überprüfen möchte, was getan wurde (Protokolle/Roadmap aktualisieren).
