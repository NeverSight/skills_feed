Documentation du projet axé sur l'agent. Élabore et maintient une architecture de documentation conçue pour les agents IA : ROADMAP, documents de conception, journaux, etc. À utiliser lorsque l'utilisateur dit "/docs", demande de configurer la documentation du projet, souhaite mettre à jour la documentation ou lors du démarrage d'un nouveau projet dépourvu de structure de documentation. À utiliser également lorsque l'utilisateur demande de planifier une fonctionnalité (crée un document de conception) ou de revoir ce qui a été fait (mises à jour des journaux/feuille de route).
