Documentación del proyecto del agente primero. Desarrolla y mantiene una arquitectura de documentación diseñada para agentes de IA: ROADMAP, documentos de diseño, registros y más. Úselo cuando el usuario dice "/docs", solicita configurar la documentación del proyecto, desea actualizar los documentos o cuando inicia un nuevo proyecto que carece de estructura de documentación. Utilícelo también cuando el usuario solicite planificar una función (crea un documento de diseño) o revisar lo que se ha hecho (actualiza registros/hoja de ruta).
