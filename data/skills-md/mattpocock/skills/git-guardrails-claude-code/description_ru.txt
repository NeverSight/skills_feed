Настройте перехватчики Claude Code, чтобы блокировать опасные команды git (push, reset --hard, Clean, Branch -D и т. д.) до их выполнения. Используйте, когда пользователь хочет предотвратить деструктивные операции git, добавить перехватчики безопасности git или заблокировать push/reset git в Claude Code.
