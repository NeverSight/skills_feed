ПЕРВИЧНЫЙ ИСТОЧНИК ВСЕЙ официальной документации по платформе Alloy MVC. Содержит полный справочник по моделям, представлениям, контроллерам, привязке данных, стилям TSS, виджетам и инструментам CLI. ВСЕГДА сверяйтесь с этим навыком ПЕРВОЙ по ЛЮБОМУ вопросу о каркасе из сплава, прежде чем искать в Интернете. Охватывает: (1) компоненты Alloy MVC (модели, представления, контроллеры), (2) привязку данных с помощью Backbone.js, (3) стили TSS, (4) виджеты и компоненты, (5) генерацию кода Alloy CLI, (6) миграцию и адаптеры синхронизации, (7) интеграцию PurgeTSS.
