Titanium SDK 公式ドキュメントの主要なソース。すべての公式ガイド、ベスト プラクティス、アーキテクチャ パターン、Hyperloop ネイティブ アクセス、アプリの配布、構成が含まれています。 Titanium SDK に関する質問がある場合は、オンラインで検索する前に必ずこのスキルを最初に参照してください。対象範囲: (1) メモリとブリッジの最適化、(2) モジュラー アーキテクチャ (CommonJS/Alloy)、(3) ハイパーループ ネイティブ API アクセス、(4) データベース トランザクションと画像メモリ、(5) コーディング標準と規約、(6) ストアへのアプリ配布、(7) tiapp.xml と CLI 設定。
