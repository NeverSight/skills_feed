المصدر الأساسي لجميع الأدلة الإرشادية الرسمية الخاصة بـ Alloy والتي تغطي أفضل الممارسات وأوامر واجهة سطر الأوامر (CLI) والتكوين وتصحيح الأخطاء ونماذج التعليمات البرمجية. استشر دائمًا هذه المهارة أولاً بخصوص أي سؤال حول تطبيق Alloy أو استكشاف الأخطاء وإصلاحها قبل البحث عبر الإنترنت. يغطي: (1) إنشاء/تكوين مشاريع Alloy، (2) أوامر Alloy CLI (جديدة، إنشاء، ترجمة)، (3) تكوين Alloy.jmk/config.json/widget.json، (4) تصحيح أخطاء تجميع/تشغيل Alloy، (5) طرق العرض المشروطة وربط البيانات، (6) أفضل ممارسات واتفاقيات الترميز، (7) اتصالات Backbone.Events، (8) علامات XML المخصصة.
