Руководство по интеграции Claude Code CLI в агентском/безголовом режиме: необходимые флаги потока-json, протокол NDJSON, утверждения инструмента control_request/control_response и управление процессами. Используйте при создании хоста, который управляет Claude CLI или при устранении неполадок с запросами разрешений и утверждениями инструментов.
