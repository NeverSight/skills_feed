Conseils pour l'intégration de Claude Code CLI en mode agent/headless : indicateurs stream-json requis, protocole NDJSON, approbations de l'outil control_request/control_response et gestion des processus. À utiliser lors de la création d'un hôte qui pilote Claude CLI ou lors du dépannage des invites d'autorisation et des approbations d'outils.
