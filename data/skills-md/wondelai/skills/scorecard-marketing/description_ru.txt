Система генерации лидов с использованием воронок викторин/оценки (Scorecard Marketing). Используйте при создании целевых страниц для привлечения потенциальных клиентов, воронок викторин, онлайн-оценок, анкет для определения потенциальных клиентов или любого «лид-магнита», который собирает данные, одновременно обеспечивая ценность. На основе методологии Дэниела Пристли, которая обеспечивает конверсию 30–50 % по сравнению с 3–10 % для традиционных лид-магнитов PDF.
