Web-Typografie-Prinzipien basierend auf „On Web Typography“ von Jason Santa Maria (A Book Apart, 2014). Verwenden Sie diese Option, wenn Sie Schriftarten auswählen, Textkörper festlegen, Typhierarchien erstellen, Schriftarten koppeln oder Typografie in CSS implementieren. Wird bei Anfragen nach Schriftartenauswahl, lesbarem Text, typografischer Hierarchie, Schriftartenpaarung oder „Text besser aussehen“ ausgelöst.
