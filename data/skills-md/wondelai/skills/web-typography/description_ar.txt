مبادئ طباعة الويب تعتمد على "On Web Typography" بقلم جيسون سانتا ماريا (A Book Apart، 2014). يُستخدم عند تحديد المحارف، أو إعداد النص الأساسي، أو إنشاء تسلسلات هرمية للأنواع، أو إقران الخطوط، أو تنفيذ الطباعة في CSS. يتم تفعيلها بناءً على طلبات اختيار الخط، أو النص المقروء، أو التسلسل الهرمي المطبعي، أو إقران الخطوط، أو "جعل النص يبدو أفضل".
