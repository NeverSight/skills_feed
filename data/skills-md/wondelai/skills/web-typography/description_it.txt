Principi di tipografia web basati su "On Web Typography" di Jason Santa Maria (A Book Apart, 2014). Da utilizzare quando si selezionano caratteri tipografici, si imposta il corpo del testo, si creano gerarchie di tipi, si abbinano caratteri o si implementa la tipografia nei CSS. Si attiva su richieste di selezione di caratteri, testo leggibile, gerarchia tipografica, abbinamento di caratteri o "miglioramento dell'aspetto del testo".
