Praktisches UI-Designsystem basierend auf Refactoring UI (Wathan & Schoger). Zur Verwendung beim Erstellen oder Verbessern von Benutzeroberflächen, Webkomponenten, Dashboards, Landingpages oder anderen visuellen Designarbeiten. Gilt für HTML/CSS, React, Tailwind oder jede Frontend-Ausgabe. Wird bei Anfragen nach UI-Design, Stil, Layout, Farbschemata, Typografie, visueller Hierarchie oder „besserem Aussehen“ von Schnittstellen ausgelöst.
