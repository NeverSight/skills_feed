Práctico sistema de diseño de UI basado en Refactoring UI (Wathan & Schoger). Úselo al crear o mejorar interfaces de usuario, componentes web, paneles de control, páginas de destino o cualquier trabajo de diseño visual. Se aplica a HTML/CSS, React, Tailwind o cualquier salida de frontend. Se activan solicitudes de diseño de interfaz de usuario, estilo, diseño, combinaciones de colores, tipografía, jerarquía visual o hacer que las interfaces "se vean mejor".
