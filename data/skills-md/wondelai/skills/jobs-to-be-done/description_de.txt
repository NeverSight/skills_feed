Strategischer Rahmen für die Entdeckung und Gestaltung von Produktinnovationen basierend auf Clayton Christensens Jobs to Be Done (JTBD)-Theorie aus „Competing Against Luck“. Verwenden Sie es, wenn Sie Folgendes benötigen: (1) die wahren Beweggründe der Kunden verstehen, (2) ein neues Produkt oder eine neue Funktion entwerfen, (3) Kundeninterviews durchführen, (4) den Wettbewerb durch die „Job“-Linse analysieren, (5) diagnostizieren, warum sich ein Produkt nicht verkauft oder Kunden abwandern, (6) eine Positionierungsstrategie erstellen, (7) eine arbeitsplatzorientierte Organisation aufbauen.
