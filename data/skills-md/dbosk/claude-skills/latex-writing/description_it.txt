Guida la creazione di documenti LaTeX seguendo le migliori pratiche e la semantica corretta 
markup. Utilizzare in modo proattivo quando: (1) si scrive o si modifica file .tex, (2) si scrive 
o modificare file di programmazione alfabetizzata .nw, (3) è l'abilità di programmazione alfabetizzata 
attivo e che lavora con file .nw, (4) l'utente menziona LaTeX, BibTeX o 
formattazione dei documenti, (5) revisione della qualità del codice LaTeX. Garantisce il corretto utilizzo di 
ambienti semantici (descrizione vs itemize), csquotes (\enquote{} not 
``...'') e intelligentef (\cref{} non \S\ref{}).
