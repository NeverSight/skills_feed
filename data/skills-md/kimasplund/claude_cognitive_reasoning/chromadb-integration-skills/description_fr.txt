Modèles d'intégration universels ChromaDB pour la recherche sémantique, le stockage persistant et la correspondance de modèles pour tous les types d'agents. À utiliser lorsque les agents doivent stocker/rechercher des ensembles de données volumineux, créer des bases de connaissances, effectuer une analyse sémantique ou maintenir une mémoire persistante entre les sessions.
