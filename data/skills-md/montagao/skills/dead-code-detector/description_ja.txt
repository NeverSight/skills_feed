TypeScript/JavaScript プロジェクト内の無効なコードを見つけて削除します。未使用のエクスポート (他にインポートされていない関数、型、定数)、未使用の npm 依存関係 (package.json 内のパッケージがインポートされていない)、および到達不能なコード パターン (リターン後のコード、常に false の条件、古い TODO) を検出します。 「デッドコードの検索」、「未使用のエクスポートのチェック」、「未使用のコードのクリーンアップ」、「未使用の依存関係の検索」、「デッドコードのコードベースの監査」、「削除できるコード」、または大規模なリファクタリングの前に使用します。
