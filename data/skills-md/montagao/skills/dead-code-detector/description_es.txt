Detecta exportaciones no utilizadas (funciones, tipos, constantes nunca importadas en otros lugares), dependencias de npm no utilizadas (paquetes en package.json nunca importados) y patrones de código inalcanzables (código después de devoluciones, condiciones siempre falsas, TODO obsoletos). Úselo cuando: "buscar código inactivo", "verificar exportaciones no utilizadas", "limpiar código no utilizado", "encontrar dependencias no utilizadas", "auditar la base de código en busca de código inactivo", "qué código puedo eliminar" o antes de refactorizaciones importantes.
