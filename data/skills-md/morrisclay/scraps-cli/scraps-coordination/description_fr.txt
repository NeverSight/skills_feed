Permet une collaboration multi-agent sécurisée sur les référentiels git partagés à l'aide des primitives de coordination scraps.sh. Utilisez cette compétence lorsque plusieurs agents IA doivent travailler simultanément sur la même base de code, lorsque vous devez revendiquer des fichiers avant de les modifier pour éviter les conflits, ou lors de la coordination du travail avec d'autres agents.
