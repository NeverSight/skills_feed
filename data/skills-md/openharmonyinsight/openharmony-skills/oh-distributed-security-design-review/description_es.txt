Habilidad especializada para revisión de código de seguridad en sistemas distribuidos OpenHarmony. Se activa cuando el usuario solicita "revisar la implementación de la seguridad del código", "revisión de la seguridad del código", "revisión del código de seguridad" o solicitudes similares de revisión de la seguridad del código del sistema distribuido. Esta habilidad proporciona una guía de revisión detallada para 18 reglas de diseño de seguridad de negocios distribuidos de OpenHarmony, que cubren control de autorización, máquinas de estado, transferencia de datos, gestión de permisos, relaciones de confianza y otras áreas de seguridad. Utilice esta habilidad para realizar revisiones de seguridad específicas para sistemas distribuidos OpenHarmony además de las reglas generales de ciberseguridad.
