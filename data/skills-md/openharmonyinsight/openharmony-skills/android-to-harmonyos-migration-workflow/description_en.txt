Multi-Agent Collaborative Android-to-HarmonyOS Code-Migration Workflow (Iterative). Triggered when a user needs to migrate an Android project to HarmonyOS, providing:  
1) Analyzer Agent scans code structure  
2) Planner Agent drafts migration plan  
3) Translator Agent performs code conversion  
4) Validator Agent compiles, packages, and on-device validates the first screen via ohos-app-build-debug skill  
5) Tester Agent verifies functionality (executed after each module migration)  
6) After all modules migrate, Feature Comparator Agent checks for missing features  
7) UI Comparator Agent verifies UI consistency  
8) Feeds unfinished parts back to Planner Agent for further iterations.  
Includes code-analysis scripts, API-mapping tables, component-conversion patterns, and automated validation tools.  
Trigger keywords: migrate Android to HarmonyOS, Android migration, HarmonyOS migration, Java/Kotlin to ArkTS, Activity to Page, feature comparison, UI comparison.
