Comando combinado de compilación y ejecución de OpenHarmony XTS. Compilación y ejecución integrales del proyecto de prueba XTS. Utilice esta habilidad cuando necesite el flujo de prueba XTS completo: (1) cree el proyecto de prueba XTS para generar archivos HAP, (2) ejecute las pruebas XTS y muestre los resultados. Admite pasar parámetros --package y --api a la etapa de ejecución.
