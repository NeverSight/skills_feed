Автоматически создавайте структуру кода HarmonyOS на основе проектных документов, включая определения интерфейса IDL, структуру каталогов, файлы заголовков, шаблоны исходных файлов и конфигурации сборки. Перед генерацией он анализирует существующий код OpenHarmony, чтобы изучить текущий стиль кода и архитектурные нормы. Подходит для запросов пользователей: (1) генерировать структуру кода, (2) создавать IDL-интерфейсы, (3) генерировать конфигурации BUILD.gn, (4) создавать сервисный код, (5) инициализировать структуру проекта, (6) ссылаться на существующий стиль кода. Ключевые слова: генерация кода, IDL, BUILD.gn, Bundle.json, профиль SA, стиль кода OpenHarmony, генерация кода, определение интерфейса, конфигурация сборки, стиль кода OH.
