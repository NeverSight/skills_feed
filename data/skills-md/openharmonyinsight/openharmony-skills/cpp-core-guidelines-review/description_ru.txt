Параллельная проверка кода C++ Core Guidelines с использованием нескольких специализированных субагентов. Используйте при проверке кода, модулей или файлов C++ на соответствие основным рекомендациям C++ для выявления нарушений. Каждый субагент проверяет соответствие определенному разделу рекомендаций (Функции, Классы, Управление ресурсами и т. д.) и выводит результаты в отдельные файлы уценки в каталоге review/, после чего следует сводная сводка.
