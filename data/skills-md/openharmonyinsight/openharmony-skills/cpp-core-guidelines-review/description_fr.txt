Révision du code des lignes directrices parallèles C++ Core à l’aide de plusieurs sous-agents spécialisés. À utiliser lors de la révision du code, des modules ou des fichiers C++ par rapport aux directives de base C++ pour identifier les violations. Chaque sous-agent examine une section de directives spécifique (fonctions, classes, gestion des ressources, etc.) et génère les résultats dans des fichiers de démarques séparés dans le répertoire review/, suivis d'un résumé consolidé.
