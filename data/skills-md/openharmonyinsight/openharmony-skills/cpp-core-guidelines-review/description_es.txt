Revisión paralela del código de las directrices básicas de C++ mediante el uso de múltiples subagentes especializados. Úselo al revisar código, módulos o archivos de C++ según las pautas básicas de C++ para identificar infracciones. Cada subagente revisa una sección de pautas específica (Funciones, Clases, Gestión de recursos, etc.) y envía los hallazgos a archivos de rebajas separados en el directorio review/, seguido de un resumen consolidado.
