CLI di automazione del browser per agenti AI. Utilizzare quando l'utente deve interagire con i siti Web, ad esempio navigare nelle pagine, compilare moduli, fare clic su pulsanti, acquisire schermate, estrarre dati, testare app Web o automatizzare qualsiasi attività del browser. I trigger includono richieste di "aprire un sito Web", "compilare un modulo", "fare clic su un pulsante", "fare uno screenshot", "estrarre dati da una pagina", "testare questa app Web", "accedere a un sito", "automatizzare le azioni del browser" o qualsiasi attività che richieda un'interazione Web programmatica.
