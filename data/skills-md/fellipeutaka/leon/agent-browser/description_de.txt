Browser-Automatisierungs-CLI für KI-Agenten. Verwenden Sie es, wenn der Benutzer mit Websites interagieren muss, einschließlich Seitennavigation, Ausfüllen von Formularen, Klicken auf Schaltflächen, Erstellen von Screenshots, Extrahieren von Daten, Testen von Web-Apps oder Automatisieren von Browseraufgaben. Zu den Auslösern gehören Anfragen zum „Öffnen einer Website“, „Ausfüllen eines Formulars“, „Klicken auf eine Schaltfläche“, „Erstellen eines Screenshots“, „Scrapen von Daten von einer Seite“, „Testen dieser Web-App“, „Anmelden bei einer Website“, „Automatisieren von Browseraktionen“ oder jede andere Aufgabe, die eine programmatische Webinteraktion erfordert.
