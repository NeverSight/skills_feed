Écrivez, déboguez et maintenez les tests de bout en bout de Playwright pour les applications Web. À utiliser lorsque vous travaillez avec des fichiers de test Playwright, configurez playwright.config.ts, écrivez l'automatisation du navigateur, déboguez des tests E2E instables, configurez l'authentification pour les tests, la simulation/interception d'API, les tests de régression visuelle, les tests d'accessibilité ou l'intégration CI/CD pour les tests de navigateur. Déclencheurs : Playwright, test E2E, de bout en bout, test du navigateur, @playwright/test, playwright.config, modèle d'objet de page, dispositif de test, instantané visuel, visionneuse de traces.
