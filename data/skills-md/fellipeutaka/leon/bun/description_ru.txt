Среда выполнения Bun, менеджер пакетов, сборщик пакетов и средство запуска тестов.
Используйте при запуске сценариев с помощью Bun, управлении пакетами, обслуживании HTTP с помощью Bun.serve,
запросы к базам данных с помощью Bun.sql/bun:sqlite/Bun.redis, сценарии оболочки с помощью $,
использование S3/файлового ввода-вывода, написание тестов с помощью Bun:test, объединение или компиляция в исполняемый файл,
или используя любой API, специфичный для Bun (spawn, glob, semver, FFI, Workers, плагины, HTMLRewriter).
