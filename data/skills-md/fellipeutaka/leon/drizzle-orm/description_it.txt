Drizzle ORM: ORM TypeScript leggero e indipendente dai tipi per database SQL. Dichiarazione di schema, query CRUD, join, relazioni, migrazioni con Drizzle Kit e seeding del database.
Utilizzare quando si definiscono schemi di database, si scrivono query (seleziona/inserisci/aggiorna/elimina), si impostano migrazioni, si configura drizzle.config.ts, si stabiliscono connessioni al database,
convalidare i dati con drizzle-zod/valibot o lavorare con PostgreSQL, MySQL, SQLite, Turso, Bun SQL, Neon, Supabase, PGlite,
Expo SQLite, Cloudflare D1, PlanetScale, SingleStore, MSSQL, CockroachDB.
