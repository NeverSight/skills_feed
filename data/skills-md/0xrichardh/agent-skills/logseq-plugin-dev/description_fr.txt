Guide complet et ressources pour développer des plugins Logseq. Utilisez cette compétence lorsqu'on vous demande de : (1) Créer un nouveau plugin Logseq, (2) Implémenter des fonctionnalités à l'aide de l'API du plugin Logseq (éditeur, application, base de données, interface utilisateur), (3) Déboguer ou refactoriser les plugins Logseq existants, (4) Configurer un environnement de développement pour les plugins Logseq.
