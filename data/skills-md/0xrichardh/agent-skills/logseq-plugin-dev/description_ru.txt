Комплексное руководство и ресурсы для разработки плагинов Logseq. Используйте этот навык, когда вас попросят: (1) создать новый плагин Logseq, (2) реализовать функции с помощью API плагина Logseq (редактор, приложение, база данных, пользовательский интерфейс), (3) отладить или реорганизовать существующие плагины Logseq, (4) настроить среду разработки для плагинов Logseq.
