Guida per l'interpretazione dei file SPAA (Stack Profile for Agentic Analysis).
Fornisce informazioni sul formato del file, nonché suggerimenti su come utilizzarlo per identificare le prestazioni
colli di bottiglia, perdite di memoria o opportunità di ottimizzazione.
Da utilizzare quando l'utente tenta di leggere un file .spaa per comprendere le prestazioni di un'applicazione.
