Отправка электронной почты от нескольких поставщиков для приложений Cloudflare Workers и Node.js.

Создавайте системы транзакционной электронной почты с помощью Resend (поддержка React Email), SendGrid (корпоративный масштаб),
Mailgun (веб-перехватчики для разработчиков) или SMTP2Go (надежный ретранслятор). Включает шаблоны шаблонов, вебхук.
проверка, обработка вложений и устранение ошибок. Используйте при отправке писем через API, обработке
отказы/жалобы или миграция между поставщиками услуг.
