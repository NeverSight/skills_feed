Stockez des objets avec le stockage compatible S3 de R2 à la périphérie de Cloudflare. À utiliser dans les cas suivants : téléchargement/téléchargement de fichiers, configuration de CORS, génération d'URL présignées, téléchargements en plusieurs parties, gestion des métadonnées ou dépannage de R2_ERROR, échecs CORS, problèmes d'URL présignées, erreurs de quota, limites de débit 429, métadonnées list() manquantes ou pannes de plate-forme. Empêche 13 erreurs documentées, notamment la limitation du débit r2.dev, les limites d'écriture simultanée, les autorisations de jeton API et la confusion du format CORS.
