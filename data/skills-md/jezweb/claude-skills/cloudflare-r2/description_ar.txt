قم بتخزين الكائنات باستخدام وحدة تخزين R2 المتوافقة مع S3 على حافة Cloudflare. يُستخدم عندما: تحميل/تنزيل الملفات، أو تكوين CORS، أو إنشاء عناوين URL مُوقعة مسبقًا، أو تحميلات متعددة الأجزاء، أو إدارة البيانات التعريفية، أو استكشاف أخطاء R2_ERROR وإصلاحها، أو فشل CORS، أو مشكلات URL المُوقعة مسبقًا، أو أخطاء الحصص، أو حدود المعدل 429، أو فقدان بيانات تعريف list()، أو انقطاع النظام الأساسي. يمنع 13 خطأ موثقًا بما في ذلك تحديد معدل r2.dev، وحدود الكتابة المتزامنة، وأذونات الرمز المميز لواجهة برمجة التطبيقات، والارتباك في تنسيق CORS.
