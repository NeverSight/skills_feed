قم بتخزين بيانات القيمة الأساسية على مستوى العالم باستخدام شبكة Cloudflare KV. يُستخدم عندما: تخزين استجابات واجهة برمجة التطبيقات (API) مؤقتًا، أو تخزين التكوين، أو إدارة تفضيلات المستخدم، أو التعامل مع انتهاء صلاحية TTL، أو استكشاف أخطاء KV_ERROR وإصلاحها، أو حدود معدل 429، أو الاتساق النهائي، أو أخطاء ذاكرة التخزين المؤقت، أو مشكلات أنواع wrangler، أو تكوين الربط عن بُعد.
