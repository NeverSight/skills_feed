Настройте Tailwind v4 с пользовательским интерфейсом в стиле shadcn/ui. Рабочий процесс: установите зависимости, настройте переменные CSS со встроенным @theme, настройте темный режим, проверьте. Используйте при инициализации проектов React с помощью Tailwind v4, настройке тем shadcn/ui или исправлении неработающих цветов, ошибок tw-animate-css, конфликтов встроенного темного режима @theme, нарушения @apply, проблем миграции v3.
