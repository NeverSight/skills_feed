قم بإعداد Tailwind v4 مع واجهة مستخدم ذات طابع shadcn/ui. سير العمل: تثبيت التبعيات، وتكوين متغيرات CSS باستخدام @theme المضمن، وإعداد الوضع المظلم، والتحقق. يُستخدم عند تهيئة مشاريع React باستخدام Tailwind v4، أو إعداد سمات shadcn/ui، أو إصلاح الألوان التي لا تعمل، أو أخطاء tw-animate-css، أو تعارض @theme في الوضع الداكن المضمن، أو @apply Break، أو مشكلات ترحيل v3.
