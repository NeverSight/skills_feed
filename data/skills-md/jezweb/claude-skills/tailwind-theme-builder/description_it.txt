Configura Tailwind v4 con l'interfaccia utente a tema shadcn/ui. Flusso di lavoro: installa le dipendenze, configura le variabili CSS con @theme inline, imposta la modalità oscura, verifica. Da utilizzare durante l'inizializzazione di progetti React con Tailwind v4, l'impostazione di temi shadcn/ui o la correzione di colori non funzionanti, errori tw-animate-css, conflitti in modalità oscura in linea @theme, interruzioni di @apply, problemi di migrazione v3.
