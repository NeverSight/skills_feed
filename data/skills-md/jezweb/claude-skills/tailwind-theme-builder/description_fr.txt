Configurez Tailwind v4 avec une interface utilisateur sur le thème shadcn/ui. Workflow : installer les dépendances, configurer les variables CSS avec @theme en ligne, configurer le mode sombre, vérifier. À utiliser lors de l'initialisation de projets React avec Tailwind v4, de la configuration du thème shadcn/ui ou de la correction des couleurs qui ne fonctionnent pas, des erreurs tw-animate-css, des conflits de mode sombre en ligne @theme, de la rupture @apply, des problèmes de migration v3.
