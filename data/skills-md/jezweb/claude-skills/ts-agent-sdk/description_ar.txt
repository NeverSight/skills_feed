قم بإنشاء حزم TypeScript SDK المكتوبة لوكلاء الذكاء الاصطناعي للتفاعل مع خوادم MCP. يحول أوامر تجعيد JSON-RPC إلى استدعاءات الوظائف النظيفة. يُنشئ تلقائيًا الأنواع وأساليب العميل وأمثلة البرامج النصية من تعريفات أداة MCP.

يُستخدم عند إنشاء تطبيقات تدعم MCP، أو تحتاج إلى وصول برمجي مكتوب إلى أدوات MCP، أو إنشاء برامج نصية لأتمتة الوكيل قابلة لإعادة الاستخدام.
