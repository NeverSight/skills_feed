Создавайте типизированные SDK TypeScript для агентов ИИ для взаимодействия с серверами MCP. Преобразует команды Curl JSON-RPC в чистые вызовы функций. Автоматически генерирует типы, клиентские методы и примеры сценариев на основе определений инструментов MCP.

Используйте при создании приложений с поддержкой MCP, когда требуется типизированный программный доступ к инструментам MCP или при создании многократно используемых сценариев автоматизации агентов.
