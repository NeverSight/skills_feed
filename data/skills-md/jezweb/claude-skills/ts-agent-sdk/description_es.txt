Genere SDK de TypeScript escritos para que los agentes de IA interactúen con los servidores MCP. Convierte comandos curl JSON-RPC en llamadas a funciones limpias. Genera automáticamente tipos, métodos de cliente y scripts de ejemplo a partir de definiciones de herramientas MCP.

Úselo cuando cree aplicaciones habilitadas para MCP, necesite acceso programático escrito a herramientas MCP o cree scripts de automatización de agentes reutilizables.
