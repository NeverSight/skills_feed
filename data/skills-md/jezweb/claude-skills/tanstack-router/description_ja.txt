TanStack Router を使用して、タイプ セーフなファイルベースの React ルーティングを構築します。クライアント側のナビゲーション、ルート ローダー、TanStack Query の統合をサポートします。検証構造の損失、パラメータ解析のバグ、SSR ストリーミングのクラッシュなど、文書化された 20 件のエラーを防止します。

ファイルベースのルーティング パターンの実装、TypeScript ルーティングを使用した SPA の構築、または devtools の依存関係エラー、タイプ セーフティの問題、Vite バンドルの問題、または Docker デプロイメントの問題のトラブルシューティングを行う場合に使用します。
