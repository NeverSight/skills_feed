تنفيذ مصادقة OAuth 2.0 باستخدام GitHub وMicrosoft Entra (Azure AD) في Cloudflare Workers
والبيئات الطرفية الأخرى. يغطي المراوغات الخاصة بموفر الخدمة، والرؤوس المطلوبة، ومتطلبات النطاق،
والتعامل مع الرمز المميز بدون MSAL.

يُستخدم عندما: تنفيذ GitHub OAuth، ومصادقة Microsoft/Azure AD، والتعامل مع عمليات رد اتصال OAuth،
أو استكشاف أخطاء 403 وإصلاحها في تدفقات OAuth.
