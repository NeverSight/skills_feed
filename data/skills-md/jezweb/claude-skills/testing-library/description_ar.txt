اختبار مكونات React باستخدام أنماط مكتبة الاختبار. يغطي الاستعلامات (getBy/findBy/queryBy)، وتفاعلات حدث المستخدم، والاختبار غير المتزامن (findBy vs waitFor)، واختبار إمكانية الوصول، وتكامل MSW للسخرية من واجهة برمجة التطبيقات (API).

يُستخدم عندما: اختبار مكونات React، أو محاكاة تفاعلات المستخدم، أو اختبار النماذج، أو الاستهزاء باستدعاءات واجهة برمجة التطبيقات (API) باستخدام MSW، أو كتابة اختبارات يمكن الوصول إليها. الكلمات الرئيسية: مكتبة الاختبار، مكتبة اختبار التفاعل، getByRole، حدث المستخدم، waitFor، MSW، الشاشة.
