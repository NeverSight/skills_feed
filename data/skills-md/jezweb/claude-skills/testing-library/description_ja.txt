テスト ライブラリ パターンを使用して React コンポーネントをテストします。クエリ (getBy/findBy/queryBy)、ユーザー イベント インタラクション、非同期テスト (findBy と waitFor)、アクセシビリティ テスト、および API モックのための MSW 統合をカバーします。

React コンポーネントのテスト、ユーザー インタラクションのシミュレーション、フォームのテスト、MSW による API 呼び出しのモック、またはアクセス可能なテストの作成の場合に使用します。キーワード: テスト ライブラリ、反応テスト ライブラリ、getByRole、ユーザー イベント、waitFor、MSW、画面。
