Gemini File Search を使用してドキュメント Q&A を構築する - 自動チャンク化、埋め込み、引用を備えたフルマネージド RAG。 100 を超えるファイル形式をアップロードし、自然言語でクエリを実行します。

次の場合に使用します: ドキュメント Q&A、検索可能なナレッジ ベース、セマンティック検索。トラブルシューティング: ドキュメントの不変性、ストレージ クォータ (3 倍)、チャンキング構成、メタデータ制限 (最大 20)、ポーリング タイムアウト、表示名のドロップ (Blob アップロード)、グラウンディングの喪失 (JSON モード)、ツールの競合 (googleSearch + fileSearch)。
