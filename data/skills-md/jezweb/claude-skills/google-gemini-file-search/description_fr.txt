Créez des questions et réponses sur des documents avec Gemini File Search - RAG entièrement géré avec segmentation, intégration et citations automatiques. Téléchargez plus de 100 formats de fichiers, interrogez en langage naturel.

À utiliser dans les cas suivants : questions-réponses sur les documents, bases de connaissances consultables, recherche sémantique. Dépannage : immuabilité du document, quota de stockage (3x), configuration de segmentation, limites de métadonnées (20 max), délais d'attente d'interrogation, displayName supprimé (téléchargements Blob), mise à la terre perdue (mode JSON), conflits d'outils (googleSearch + fileSearch).
