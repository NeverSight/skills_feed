أنشئ مكونات إضافية آمنة لـ WordPress باستخدام الخطافات وتفاعلات قاعدة البيانات وواجهة برمجة تطبيقات الإعدادات وأنواع المنشورات المخصصة وواجهة برمجة تطبيقات REST. يغطي أنماط الهندسة البسيطة وOOP وPSR-4 بالإضافة إلى Security Trinity. يتضمن تغييرات عاجلة في WordPress 6.7-6.9.

يُستخدم عند إنشاء مكونات إضافية أو استكشاف أخطاء حقن SQL وإصلاحها أو ثغرات XSS أو CSRF أو REST API أو أخطاء wpdb::prepare أو حالات nonce edge أو ترحيل WordPress 6.8+ bcrypt.
