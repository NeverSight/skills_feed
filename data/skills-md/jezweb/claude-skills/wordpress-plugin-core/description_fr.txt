Créez des plugins WordPress sécurisés avec des hooks, des interactions de base de données, une API de paramètres, des types de publication personnalisés et une API REST. Couvre les modèles d'architecture Simple, POO et PSR-4 ainsi que Security Trinity. Inclut les modifications de rupture de WordPress 6.7-6.9.

À utiliser lors de la création de plugins ou du dépannage d'injection SQL, de XSS, CSRF, de vulnérabilités de l'API REST, d'erreurs wpdb::prepare, de cas extrêmes occasionnels ou de migration bcrypt WordPress 6.8+.
