قم بإعداد Sveltia CMS - خليفة CMS خفيف الوزن مدعوم من Git لـ Decap/Netlify CMS (حزمة 300 كيلو بايت، أكثر من 270 إصلاحًا). الإطار الملحد لهوغو، جيكل، 11ty، أسترو. يمنع 10 أخطاء موثقة.

يُستخدم عند إضافة CMS إلى مواقع ثابتة، أو الترحيل من Decap CMS، أو إصلاح أخطاء OAuth، أو تحليل YAML، أو المنطقة الزمنية للتاريخ، أو تحميل خط القانون العام لحماية البيانات (GDPR)، أو أخطاء CORS/COOP.
