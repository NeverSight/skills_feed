Créez des applications React full-stack avec TanStack Start sur Cloudflare Workers. Routage de type sécurisé, fonctions serveur, SSR/streaming, intégration D1/KV/R2.

À utiliser lors de la création d'applications React full-stack avec SSR, lors de la migration de Next.js ou de Vinxi vers Vite (v1.121.0+).
