قم بإعداد Tailwind v4 مع shadcn/ui باستخدام النمط المضمن @theme وبنية CSS المتغيرة. نمط من أربع خطوات: متغيرات CSS، ورسم خرائط Tailwind، والأنماط الأساسية، والوضع المظلم التلقائي. يمنع 8 أخطاء موثقة.

يُستخدم عند تهيئة مشاريع React باستخدام Tailwind v4، أو إصلاح الألوان التي لا تعمل، أو أخطاء tw-animate-css، أو @theme المضمّن في الوضع الداكن، أو @apply Break، أو مشكلات الترحيل v3.
