Добавьте аутентификацию OAuth на серверы MCP в Cloudflare Workers. Использует @cloudflare/workers-oauth-provider с Google OAuth для аутентификации, совместимой с Claude.ai. Предотвращает 9 задокументированных ошибок, включая ошибки аудитории RFC 8707, сбои подключения Claude.ai и уязвимости CSRF.

Используйте при создании серверов MCP, которым требуется аутентификация пользователей, реализации динамической регистрации клиентов (DCR) для Claude.ai или замене статических токенов аутентификации потоками OAuth. Включает обходные пути для несоответствий URI производственного перенаправления и проблем с циклом повторной аутентификации.
