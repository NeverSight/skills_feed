Создавайте типобезопасные API с помощью Hono для Cloudflare Workers, Deno, Bun, Node.js. Маршрутизация, промежуточное программное обеспечение, проверка (Zod/Valibot), RPC, потоковая передача (SSE), WebSocket, безопасность (CSRF, secureHeaders).

Используйте при: создании API-интерфейсов Hono, потоковой передаче SSE, WebSocket, проверке, RPC. Устранение неполадок: перехватчики проверки, типы RPC, цепочки промежуточного программного обеспечения, требуемый алгоритм проверки JWT (v4.11.4+), ошибки, используемые в теле.
