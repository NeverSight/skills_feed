Créez des serveurs MCP sur Cloudflare Workers - la seule plate-forme avec prise en charge officielle de MCP à distance. Basé sur TypeScript avec OAuth, Durable Objects et hibernation WebSocket. Empêche 24 erreurs documentées.

À utiliser lorsque : le déploiement de serveurs MCP distants, la mise en œuvre d'OAuth ou le dépannage d'incompatibilités de chemin d'URL, d'exportations McpAgent, de problèmes CORS, de délais d'attente IoContext.
