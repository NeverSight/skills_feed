Создавайте серверы MCP на Cloudflare Workers — единственной платформе с официальной удаленной поддержкой MCP. На основе TypeScript с OAuth, Durable Objects и спящим режимом WebSocket. Предотвращает 24 задокументированные ошибки.

Используйте при развертывании удаленных серверов MCP, реализации OAuth или устранении неполадок несоответствия URL-путей, экспорта McpAgent, проблем CORS, тайм-аутов IoContext.
