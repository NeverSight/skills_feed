Инструменты для создания, аудита и поддержания навыков Claude Code. Включает /create-skill
для создания шаблонов, /review-skill для проверок качества и команды /audit для массовой проверки.

Используйте при: создании новых навыков, поддержании качества навыков или создании форка репозитория claude-skills.
