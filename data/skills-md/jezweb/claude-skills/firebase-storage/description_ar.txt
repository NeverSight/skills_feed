البناء باستخدام Firebase Cloud Storage - تحميل الملفات وتنزيلها والوصول الآمن. يُستخدم عندما: تحميل الصور/الملفات، أو إنشاء عناوين URL للتنزيل، أو تنفيذ منتقي الملفات، أو إعداد قواعد أمان التخزين، أو استكشاف أخطاء التخزين/غير المصرح به، أو أخطاء كورس، أو تجاوز الحصة النسبية، أو أخطاء التحميل الفاشلة. يمنع 9 أخطاء موثقة.
