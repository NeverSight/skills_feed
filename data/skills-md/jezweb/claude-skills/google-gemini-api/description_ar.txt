قم بدمج Gemini API مع @google/genai SDK (لم يتم إهمال @google/geneative-ai). إنشاء النص، الوسائط المتعددة (الصور/الفيديو/الصوت/ملفات PDF)، استدعاء الوظائف، وضع التفكير، البث. 1M رموز الإدخال. يمنع 14 خطأ موثقا.

يُستخدم عندما: تكامل الجوزاء، والذكاء الاصطناعي متعدد الوسائط، والاستدلال باستخدام وضع التفكير. استكشاف الأخطاء وإصلاحها: إهمال SDK، لم يتم العثور على النموذج، نافذة السياق، أخطاء استدعاء الوظيفة، تلف الدفق، إعدادات الأمان، حدود المعدل.
