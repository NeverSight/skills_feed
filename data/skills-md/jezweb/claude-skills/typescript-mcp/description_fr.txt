Créez des serveurs MCP avec TypeScript sur Cloudflare Workers. Couvre les outils, les ressources, les invites, les tâches, l'authentification (clés API, OAuth, Zero Trust) et les intégrations de services Cloudflare. Empêche 20 erreurs documentées.

À utiliser lors de l'exposition d'API à des LLM ou du dépannage d'erreurs de syntaxe d'exportation, de fuites de transport, de bogues de réutilisation d'instances de serveur, de mauvaises configurations CORS ou d'erreurs de validation de tâches.
