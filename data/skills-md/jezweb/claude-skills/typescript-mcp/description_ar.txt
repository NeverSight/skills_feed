يُستخدم عند تعريض واجهات برمجة التطبيقات إلى LLMs أو استكشاف أخطاء بناء جملة التصدير وإصلاحها، أو تسرب النقل، أو أخطاء إعادة استخدام مثيل الخادم، أو تكوينات CORS الخاطئة، أو أخطاء التحقق من صحة المهمة.
