Создавайте серверы MCP с помощью TypeScript на Cloudflare Workers. Охватывает инструменты, ресурсы, подсказки, задачи, аутентификацию (ключи API, OAuth, Zero Trust) и интеграцию сервисов Cloudflare. Предотвращает 20 документированных ошибок.

Используйте при предоставлении API для LLM или устранении синтаксических ошибок экспорта, утечек транспорта, ошибок повторного использования экземпляра сервера, неправильных конфигураций CORS или ошибок проверки задач.
