Costruisci server MCP con TypeScript su Cloudflare Workers. Copre strumenti, risorse, richieste, attività, autenticazione (chiavi API, OAuth, Zero Trust) e integrazioni del servizio Cloudflare. Previene 20 errori documentati.

Da utilizzare quando si espongono le API a LLM o si risolvono errori di sintassi di esportazione, perdite di trasporto, bug di riutilizzo delle istanze del server, configurazioni errate di CORS o errori di convalida delle attività.
