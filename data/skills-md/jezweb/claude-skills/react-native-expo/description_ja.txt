Expo SDK 52-54 を使用して React Native 0.76 以降のアプリを構築します。必須の新しいアーキテクチャ (0.82+/SDK 55+)、React 19 の変更、SDK 54 の重大な変更 (expo-av、expo-file-system、Reanimated v4)、および Swift iOS テンプレートをカバーします。文書化された 16 件のエラーを防止します。

Expo アプリの構築、新しいアーキテクチャへの移行、SDK 54 以降へのアップグレード、または Fabric、TurboModule、propTypes、expo 更新のクラッシュ、または Swift AppDelegate エラーの修正を行う場合に使用します。
