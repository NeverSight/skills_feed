أنشئ تطبيقات React Native الإصدار 0.76+ باستخدام Expo SDK 52-54. يغطي البنية الجديدة الإلزامية (0.82+/SDK 55+)، وتغييرات React 19، وتغييرات كسر SDK 54 (expo-av، وexpo-file-system، وReanimated v4)، وقالب Swift iOS. يمنع 16 خطأ موثقا.

يُستخدم عند إنشاء تطبيقات Expo، أو الترحيل إلى New Architecture، أو الترقية إلى SDK 54+، أو إصلاح Fabric، أو TurboModule، أوpropTypes، أو تعطل تحديثات expo، أو أخطاء Swift AppDelegate.
