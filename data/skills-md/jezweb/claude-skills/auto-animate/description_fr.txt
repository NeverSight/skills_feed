Animations sans configuration pour React, Vue, Solid, Svelte, Preact avec @formkit/auto-animate (3,28 Ko). Empêche 15 erreurs documentées, notamment les bogues React 19 StrictMode, les importations SSR, les parents conditionnels, les problèmes de fenêtre d'affichage, les conflits de glisser-déposer et les bogues de transformation CSS.

À utiliser lorsque : l'animation de listes/accordéons/toasts, le dépannage d'erreurs d'animation SSR, les problèmes de React 19 StrictMode ou le besoin de transitions accessibles avec préférences automatiques de mouvement réduit.
