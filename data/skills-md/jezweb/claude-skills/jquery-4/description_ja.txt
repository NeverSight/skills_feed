WordPress および従来の Web プロジェクトで jQuery 3.x を 4.0.0 に安全に移行します。すべての重大な変更をカバーします: 削除された API ($.isArray、$.trim、$.parseJSON、$.type)、フォーカス イベント順序の変更、スリム ビルドの違い、ES モジュールの移行、および信頼できるタイプのサポート。

次の場合に使用します: jQuery を 4.0 にアップグレードする場合、「$.isArray は関数ではありません」エラーを修正する場合、WordPress jQuery の移行、レガシー JavaScript を更新する場合、またはフォーカス/ブラー イベント順序の問題のトラブルシューティングを行う場合。
