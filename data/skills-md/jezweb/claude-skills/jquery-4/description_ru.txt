Безопасно перенесите jQuery 3.x на 4.0.0 в WordPress и устаревшие веб-проекты. Охватывает все критические изменения: удаленные API ($.isArray, $.trim, $.parseJSON, $.type), изменения порядка событий фокуса, тонкие различия в сборках, миграцию ES-модулей и поддержку доверенных типов.

Используйте при обновлении jQuery до версии 4.0, исправлении ошибок «$.isArray не является функцией», миграции jQuery WordPress, обновлении устаревшего JavaScript или устранении проблем с порядком событий фокуса/размытия.
