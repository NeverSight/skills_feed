Vercel AI SDK v6 を使用して React チャット インターフェイスを構築します。 useChat/useCompletion/useObject フック、メッセージ パーツ構造、ツール承認ワークフロー、および 18 の UI エラー ソリューションについて説明します。 React Strict Mode、同時リクエスト、古いクロージャ、ツール承認のエッジケースに関する文書化された問題を防止します。

AI チャット UI の実装、v5→v6 の移行、「useChat がストリームの解析に失敗した」、「本体の値が古い」、「反応の最大更新深度」、「未定義のプロパティを読み取れない (「状態」を読み取る)」、またはツールの承認ワークフロー エラーのトラブルシューティングを行う場合に使用します。
