أرسل رسائل البريد الإلكتروني والرسائل النصية القصيرة الخاصة بالمعاملات عبر SMTP2GO API. يغطي المصادقة ونقاط نهاية /email/send و/email/mime وإدارة القالب والمرفقات (base64/URL) وخطافات الويب لأحداث التسليم والإحصائيات وعمليات القمع. يُستخدم عند إرسال رسائل البريد الإلكتروني من Cloudflare Workers، وإنشاء الإشعارات، وتتبع حالة التسليم، والتعامل مع الارتدادات. يمنع أخطاء المصادقة، ومشكلات ترميز المرفقات.
