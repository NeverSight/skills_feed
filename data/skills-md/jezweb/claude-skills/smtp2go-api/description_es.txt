Envíe correos electrónicos y SMS transaccionales a través de la API SMTP2GO. Cubre autenticación, puntos finales /email/send y /email/mime, administración de plantillas, archivos adjuntos (base64/URL), webhooks para eventos de entrega, estadísticas y supresiones. Úselo al enviar correos electrónicos desde trabajadores de Cloudflare, crear notificaciones, rastrear el estado de entrega y manejar rebotes. Previene errores de autenticación y problemas de codificación de archivos adjuntos.
