Подготавливайте и публикуйте выпуски GitHub. Обеззараживает код для публичного выпуска (сканирование секретов, личные артефакты, проверка LICENSE/README), создает теги версий и публикует через gh CLI. Запускайте с помощью «выпуска», «опубликации», «открытого исходного кода», «подготовки к выпуску», «создания выпуска» или «выпуска GitHub».
