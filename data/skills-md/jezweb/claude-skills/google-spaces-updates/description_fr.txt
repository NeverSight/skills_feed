Publiez les mises à jour de l'équipe sur Google Chat Spaces via un webhook. Notifications de déploiement, corrections de bugs, annonces de fonctionnalités, questions. Lit la configuration à partir de .claude/settings.json, inclut le contexte git.

À utiliser lorsque : "publier à l'équipe", "notifier l'équipe", après les déploiements, terminer les fonctionnalités, corriger les bugs, poser des questions à l'équipe.
