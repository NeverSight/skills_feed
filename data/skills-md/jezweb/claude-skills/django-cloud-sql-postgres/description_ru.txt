Разверните Django в Google App Engine Standard с помощью Cloud SQL PostgreSQL. Охватывает соединения сокетов Unix, прокси-сервер Cloud SQL Auth для локальной разработки, конфигурацию Gunicorn и готовые к работе настройки.

Используйте при развертывании Django в App Engine, настройке Cloud SQL PostgreSQL, настройке соединений сокетов Unix или устранении неполадок «Нет такого файла или каталога», «Отказано в соединении» или «FATAL: не удалось выполнить аутентификацию пароля».
