انشر Django على Google App Engine Standard باستخدام Cloud SQL PostgreSQL. يغطي اتصالات مقبس Unix، وCloud SQL Auth Proxy للتطوير المحلي، وتكوين Gunicorn، وإعدادات الاستعداد للإنتاج.

يُستخدم عندما: نشر Django إلى App Engine، أو تكوين Cloud SQL PostgreSQL، أو إعداد اتصالات مقبس Unix، أو استكشاف الأخطاء وإصلاحها "لا يوجد مثل هذا الملف أو الدليل"، أو "تم رفض الاتصال"، أو "فادح: فشلت مصادقة كلمة المرور".
