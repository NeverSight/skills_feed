Déployez Django sur Google App Engine Standard avec Cloud SQL PostgreSQL. Couvre les connexions socket Unix, le proxy d'authentification Cloud SQL pour le développement local, la configuration Gunicorn et les paramètres prêts pour la production.

À utiliser lors du déploiement de Django sur App Engine, de la configuration de Cloud SQL PostgreSQL, de la configuration de connexions socket Unix ou du dépannage « Aucun fichier ou répertoire de ce type », « Connexion refusée » ou « FATAL : échec de l'authentification par mot de passe ».
