Cloud SQL PostgreSQL を使用して Google App Engine Standard に Django をデプロイします。 Unix ソケット接続、ローカル開発用の Cloud SQL Auth Proxy、Gunicorn 構成、本番環境に対応した設定について説明します。

Django を App Engine にデプロイする場合、Cloud SQL PostgreSQL を構成する場合、Unix ソケット接続をセットアップする場合、または「そのようなファイルまたはディレクトリはありません」、「接続が拒否されました」、または「致命的: パスワード認証に失敗しました」のトラブルシューティングを行う場合に使用します。
