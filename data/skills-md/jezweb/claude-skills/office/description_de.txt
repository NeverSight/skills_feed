Generieren Sie Office-Dokumente (DOCX, XLSX, PDF, PPTX) mit TypeScript. Reine JS-Bibliotheken, die überall funktionieren: Claude Code CLI, Cloudflare Workers, Browser. Verwendet docx (Word), xlsx/SheetJS (Excel), pdf-lib (PDF), pptxgenjs (PowerPoint).

Verwendung bei: Erstellen von Rechnungen, Berichten, Tabellenkalkulationen, Präsentationen, Ausfüllen von Formularen, Exportieren von Daten in Office-Formate oder Beheben von Fehlern wie „Packer.toBuffer“, „XLSX.utils“, „PDFDocument“ und „pptxgenjs“.
