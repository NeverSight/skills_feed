Cree sitios web compatibles con WCAG 2.1 AA con HTML semántico, ARIA adecuado, gestión de enfoque y compatibilidad con lectores de pantalla. Incluye contraste de color (texto 4.5:1), navegación con teclado, etiquetas de formulario y regiones en vivo.

Úselo al implementar interfaces accesibles, solucionar problemas del lector de pantalla, navegación con el teclado o solucionar problemas de "falta el esquema de enfoque", "se requiere etiqueta de aria", "contraste insuficiente".
