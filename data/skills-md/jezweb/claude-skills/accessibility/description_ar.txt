أنشئ مواقع ويب متوافقة مع WCAG 2.1 AA باستخدام HTML الدلالي وARIA المناسب وإدارة التركيز ودعم قارئ الشاشة. يتضمن تباين الألوان (نص 4.5:1)، والتنقل عبر لوحة المفاتيح، وتسميات النماذج، والمناطق المباشرة.

يُستخدم عند تنفيذ واجهات يمكن الوصول إليها، أو إصلاح مشكلات قارئ الشاشة، أو التنقل عبر لوحة المفاتيح، أو استكشاف الأخطاء وإصلاحها "مخطط التركيز مفقود"، "يتطلب تسمية aria"، "التباين غير كافٍ".
