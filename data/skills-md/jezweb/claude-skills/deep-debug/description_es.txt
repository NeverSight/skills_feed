Investigación de múltiples agentes para detectar errores persistentes. Úselo cuando: vaya en círculos durante la depuración, necesite investigar las interacciones entre el navegador y la API, haya errores complejos que se resistan a la depuración normal o cuando los síntomas no coincidan con las expectativas. Lanza agentes paralelos con diferentes perspectivas y utiliza herramientas de Chrome para recopilar pruebas.
