Costruisci server MCP in Python con FastMCP. Flusso di lavoro: definizione di strumenti e risorse, creazione di server, test locale, distribuzione su FastMCP Cloud o Docker. Da utilizzare durante la creazione di server MCP, l'esposizione di strumenti/risorse/prompt a LLM, la creazione di integrazioni Claude o la risoluzione di problemi di server a livello di modulo FastMCP, archiviazione, durata di vita, middleware, OAuth o errori di distribuzione.
