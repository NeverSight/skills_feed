Vercel AI SDK v6 安定版を使用してバックエンド AI を構築します。出力 API (generateObject/streamObject を置き換える)、音声合成、文字起こし、埋め込み、MCP ツールとセキュリティ ガイダンスをカバーします。 v4→v5 の移行と 15 のエラー解決策と回避策が含まれています。

AI SDK v5/v6 の実装、バージョンの移行、AI_APICallError のトラブルシューティング、ワーカーの起動の問題、出力 API エラー、Gemini キャッシュの問題、Anthropic ツールのエラー、MCP ツール、またはストリーム再開の失敗の場合に使用します。
