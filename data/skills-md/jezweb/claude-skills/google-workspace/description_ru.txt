Создавайте интеграцию с API Google Workspace (Gmail, Календарь, Диск, Таблицы, Документы, Чат, Meet, Формы, Задачи, Admin SDK). Охватывает OAuth 2.0, сервисные учетные записи, ограничения скорости, пакетные операции и шаблоны Cloudflare Workers.

Используйте при создании серверов MCP, инструментов автоматизации или интеграции с любым API Google Workspace, а также при устранении ошибок OAuth, ошибок ограничения скорости 429, проблем с областью действия или ошибок, связанных с API.
