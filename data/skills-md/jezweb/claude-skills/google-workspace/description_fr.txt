Créez des intégrations avec les API Google Workspace (Gmail, Calendar, Drive, Sheets, Docs, Chat, Meet, Forms, Tasks, Admin SDK). Couvre OAuth 2.0, les comptes de service, les limites de débit, les opérations par lots et les modèles Cloudflare Workers.

À utiliser lors de la création de serveurs MCP, d'outils d'automatisation ou d'intégrations avec n'importe quelle API Google Workspace, ou lors du dépannage d'erreurs OAuth, d'erreurs de limite de débit 429, de problèmes de portée ou de pièges spécifiques à l'API.
