Настройте бессерверную Postgres с помощью Neon или Vercel Postgres для Cloudflare Workers/Edge. Включает пул соединений, ветвление в стиле git и интеграцию Drizzle ORM.

Используйте при настройке Edge Postgres, устранении неполадок «TCP не поддерживается», исчерпании пула соединений, ошибках конфигурации SSL или проблемах с транзакциями Node v20.
