أضف أتمتة Chrome بدون رأس مع محرك الدمى/الكاتب المسرحي على Cloudflare Workers. يُستخدم عندما: التقاط لقطات شاشة، أو إنشاء ملفات PDF، أو تجريف الويب، أو الزحف إلى المواقع، أو أتمتة المتصفح، أو استكشاف أخطاء XPath وإصلاحها، أو مهلة المتصفح، أو أخطاء الربط التي لم يتم تمريرها، أو حدود الجلسة، أو أخطاء page.evaluate __name، أو مشكلات مهلة waitForSelector.
