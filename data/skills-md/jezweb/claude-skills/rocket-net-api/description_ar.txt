قم ببناء عمليات تكامل مع واجهة برمجة تطبيقات استضافة WordPress الخاصة بـ Rocket.net. إدارة المواقع والمجالات والنسخ الاحتياطية والمكونات الإضافية والموضوعات وذاكرة التخزين المؤقت لـ CDN وحسابات FTP والمزيد بشكل برمجي.

يُستخدم عندما: إنشاء أدوات إدارة استضافة WordPress، أو أتمتة نشر الموقع، أو إنشاء بوابات الموزعين، أو إدارة مواقع WordPress المتعددة، أو التكامل مع خدمات استضافة Rocket.net، أو أتمتة سير عمل النسخ الاحتياطي، أو إنشاء لوحات تحكم مخصصة.
