Создавайте целостные наборы значков SVG для конкретных проектов для веб-сайтов и приложений. Используйте этот навык всякий раз, когда пользователю нужны собственные значки, набор значков для веб-сайта или приложения, значки для клиентского проекта или когда он упоминает о необходимости значков SVG, которые выглядели бы согласованно друг с другом. Также срабатывает, когда пользователь описывает проект, и значки, естественно, будут частью результата — например. «Я создаю сайт для сантехника» подразумевает, что ему понадобятся значки сервисов. Запуск по: «значки для», «набор значков», «пользовательские значки», «значки SVG», «сделай мне значки», «мне нужны значки», «значки веб-сайта», «значки проекта» или любой запрос на согласованные визуальные ресурсы для веб-проекта. Этот навык создает отдельные файлы SVG с единым механизмом стилей, а не с помощью общего поиска в библиотеке значков.
