Generieren Sie zusammenhängende, projektspezifische SVG-Symbolsätze für Websites und Anwendungen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer benutzerdefinierte Symbole, einen Symbolsatz für eine Website oder App, Symbole für ein Kundenprojekt oder Erwähnungen benötigt, die SVG-Symbole benötigen, die zusammen einheitlich aussehen. Wird auch ausgelöst, wenn der Benutzer ein Projekt beschreibt und Symbole natürlich Teil des Ergebnisses sind – z. B. „Ich erstelle eine Website für einen Klempner“ bedeutet, dass er Servicesymbole benötigt. Auslöser bei: „Symbole für“, „Symbolsatz“, „benutzerdefinierte Symbole“, „SVG-Symbole“, „Make-me-Symbole“, „Ich benötige Symbole“, „Website-Symbole“, „Projektsymbole“ oder jede Anfrage nach konsistenten visuellen Assets für ein Webprojekt. Diese Fähigkeit erzeugt einzelne SVG-Dateien mit einer konsistenten Stil-Engine, nicht mit generischen Symbolbibliothekssuchen.
