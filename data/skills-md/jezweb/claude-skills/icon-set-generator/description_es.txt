Genere conjuntos de iconos SVG coherentes y específicos del proyecto para sitios web y aplicaciones. Utilice esta habilidad siempre que el usuario necesite íconos personalizados, un conjunto de íconos para un sitio web o aplicación, íconos para un proyecto de cliente o mencione la necesidad de íconos SVG que se vean consistentes juntos. También se activa cuando el usuario describe un proyecto y los íconos naturalmente serían parte del entregable, por ejemplo. "Estoy construyendo un sitio para un plomero" implica que necesitarán íconos de servicio. Activar: "íconos para", "conjunto de iconos", "íconos personalizados", "íconos SVG", "hacerme íconos", "Necesito íconos", "íconos de sitio web", "íconos de proyecto" o cualquier solicitud de recursos visuales consistentes para un proyecto web. Esta habilidad produce archivos SVG individuales con un motor de estilo consistente, no búsquedas genéricas en bibliotecas de íconos.
