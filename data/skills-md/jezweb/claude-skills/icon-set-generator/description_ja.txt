Web サイトやアプリケーション向けに、まとまりのあるプロジェクト固有の SVG アイコン セットを生成します。ユーザーがカスタム アイコン、Web サイトまたはアプリ用のアイコン セット、クライアント プロジェクト用のアイコン、または一貫して見える SVG アイコンを必要とするメンションを必要とする場合には、このスキルを使用します。また、ユーザーがプロジェクトについて説明したときにもトリガーされ、アイコンは当然成果物の一部になります。 「配管工向けのサイトを構築しています」ということは、サービス アイコンが必要であることを暗示しています。トリガー: 「アイコン」、「アイコン セット」、「カスタム アイコン」、「SVG アイコン」、「アイコンを作成」、「アイコンが必要」、「Web サイト アイコン」、「プロジェクト アイコン」、または Web プロジェクトの一貫したビジュアル アセットに対するリクエスト。このスキルは、汎用のアイコン ライブラリの検索ではなく、一貫したスタイル エンジンを使用して個々の SVG ファイルを生成します。
