Маршруты Scaffold Hono API для работников Cloudflare. Создает файлы маршрутов, промежуточное программное обеспечение, типизированные привязки, проверку Zod, обработку ошибок и документацию API_ENDPOINTS.md. Используйте после настройки проекта с помощью Cloudflare-worker-builder или Vite-Flare-Starter, когда вам нужно добавить маршруты API, создать конечные точки или создать документацию по API.
