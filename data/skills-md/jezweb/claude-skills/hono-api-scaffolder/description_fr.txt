Routes d'API Scaffold Hono pour Cloudflare Workers. Produit des fichiers de route, un middleware, des liaisons typées, la validation Zod, la gestion des erreurs et la documentation API_ENDPOINTS.md. À utiliser après la configuration d'un projet avec cloudflare-worker-builder ou vite-flare-starter, lorsque vous devez ajouter des routes API, créer des points de terminaison ou générer de la documentation API.
