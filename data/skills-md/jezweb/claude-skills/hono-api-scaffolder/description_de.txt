Scaffold Hono API-Routen für Cloudflare Workers. Erstellt Routendateien, Middleware, typisierte Bindungen, Zod-Validierung, Fehlerbehandlung und API_ENDPOINTS.md-Dokumentation. Verwenden Sie es, nachdem ein Projekt mit cloudflare-worker-builder oder vite-flare-starter eingerichtet wurde, wenn Sie API-Routen hinzufügen, Endpunkte erstellen oder API-Dokumentation generieren müssen.
