مسارات Scaffold Hono API لعمال Cloudflare. يُنتج ملفات المسار والبرامج الوسيطة والارتباطات المكتوبة والتحقق من صحة Zod ومعالجة الأخطاء ووثائق API_ENDPOINTS.md. يُستخدم بعد إعداد المشروع باستخدام cloudflare-worker-builder أو vite-flare-starter، عندما تحتاج إلى إضافة مسارات API، أو إنشاء نقاط نهاية، أو إنشاء وثائق API.
