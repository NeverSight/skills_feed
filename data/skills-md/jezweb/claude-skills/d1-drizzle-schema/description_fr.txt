Générez des schémas Drizzle ORM pour les bases de données Cloudflare D1 avec des modèles spécifiques à D1 corrects. Produit des fichiers de schéma, des commandes de migration, des exportations de types et une documentation DATABASE_SCHEMA.md. Gère les bizarreries de D1 : clés étrangères toujours appliquées, pas de types BOOLEAN/DATETIME natifs, limite de 100 paramètres liés, JSON stocké sous forme de TEXTE. À utiliser lors de la création d’une nouvelle base de données, de l’ajout de tables ou de l’échafaudage d’une couche de données D1.
