Декларативное тестирование на основе агентов с использованием спецификаций тестов YAML. Тесты выполняются в субагентах для сохранения
основной контекст при выполнении множества тестов. Поддерживает серверы MCP, API и автоматизацию браузера.

Используйте при: тестировании серверов MCP, запуске интеграционных тестов, проверке поведения инструмента после изменений,
или создание наборов регрессионных тестов. Ключевые слова: yaml-тесты, агентное тестирование, mcp-тест, интеграционные тесты.
