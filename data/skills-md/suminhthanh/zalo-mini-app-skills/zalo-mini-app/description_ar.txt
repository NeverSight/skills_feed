إنشاء تطبيقات Zalo Mini - تطبيقات ويب خفيفة الوزن تعمل داخل تطبيق Zalo الفائق. توفر هذه المهارة مكونات ZaUI (الزر، والإدخال، والمشروط، وعلامات التبويب، والصور الرمزية، وما إلى ذلك)، وواجهات برمجة تطبيقات JavaScript (التفويض، وgetUserInfo، وgetPhoneNumber، وgetLocation، والتخزين، والكاميرا)، وCheckout SDK للمدفوعات، وإرشادات التصميم، وأدوات التطوير. يُستخدم عند إنشاء تطبيقات صغيرة، أو استخدام مكونات ZaUI، أو استدعاء Zalo APIs، أو دمج المدفوعات، أو تحويل تطبيقات الويب إلى Mini Apps، أو اتباع معايير تصميم Zalo.
