Cree miniaplicaciones Zalo: aplicaciones web livianas que se ejecutan dentro de la súper aplicación Zalo. Esta habilidad proporciona componentes ZaUI (botón, entrada, modal, pestañas, avatar, etc.), API de JavaScript (autorizar, getUserInfo, getPhoneNumber, getLocation, almacenamiento, cámara), SDK de pago para pagos, pautas de diseño y herramientas de desarrollo. Úselo al crear miniaplicaciones, utilizar componentes ZaUI, llamar a las API de Zalo, integrar pagos, convertir aplicaciones web en miniaplicaciones o seguir los estándares de diseño de Zalo.
