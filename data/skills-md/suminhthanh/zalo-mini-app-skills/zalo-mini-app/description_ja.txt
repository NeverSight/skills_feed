Zalo Mini Apps を構築します - Zalo スーパーアプリ内で実行される軽量 Web アプリです。このスキルは、ZaUI コンポーネント (ボタン、入力、モーダル、タブ、アバターなど)、JavaScript API (authorize、getUserInfo、getPhoneNumber、getLocation、Storage、Camera)、支払い用の Checkout SDK、設計ガイドライン、および開発ツールを提供します。ミニアプリの構築、ZaUI コンポーネントの使用、Zalo API の呼び出し、支払いの統合、Web アプリからミニアプリへの変換、または Zalo 設計標準に従うときに使用します。
