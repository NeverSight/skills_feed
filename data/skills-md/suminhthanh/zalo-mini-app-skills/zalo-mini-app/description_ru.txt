Создайте мини-приложения Zalo — легкие веб-приложения, работающие внутри суперприложения Zalo. Этот навык предоставляет компоненты ZaUI (кнопка, ввод, модальные окна, вкладки, аватар и т. д.), API JavaScript (authorize, getUserInfo, getPhoneNumber, getLocation, Storage, Camera), Checkout SDK для платежей, рекомендации по дизайну и инструменты разработки. Используйте при создании мини-приложений, использовании компонентов ZaUI, вызове API-интерфейсов Zalo, интеграции платежей, преобразовании веб-приложений в мини-приложения или следовании стандартам проектирования Zalo.
