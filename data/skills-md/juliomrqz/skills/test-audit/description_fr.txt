Auditer les suites de tests (unité, intégration, fonctionnalité/E2E) pour en vérifier l'exactitude, la qualité et les meilleures pratiques. À utiliser lorsque l'utilisateur souhaite réviser, auditer, valider ou améliorer des tests dans n'importe quel langage ou framework de programmation (Jest, pytest, RSpec, JUnit, PHPUnit, etc.). Déclencheurs sur des requêtes telles que « auditer mes tests », « examiner la qualité des tests », « vérifier la couverture de mes tests », « mes tests sont-ils bons » ou toute demande visant à évaluer l'efficacité des tests.
