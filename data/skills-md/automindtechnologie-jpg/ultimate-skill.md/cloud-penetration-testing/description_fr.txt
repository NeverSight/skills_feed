Cette compétence doit être utilisée lorsque l'utilisateur demande « d'effectuer des tests d'intrusion dans le cloud », « d'évaluer la sécurité Azure ou AWS ou GCP », « d'énumérer les ressources cloud », « d'exploiter les mauvaises configurations du cloud », de « tester la sécurité O365 », « d'extraire les secrets des environnements cloud » ou « d'auditer l'infrastructure cloud ». Il fournit des techniques complètes d’évaluation de la sécurité sur les principales plates-formes cloud.
