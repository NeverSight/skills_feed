Automatisation complète du navigateur avec Playwright. Détecte automatiquement les serveurs de développement, écrit des scripts de test propres dans /tmp. Testez les pages, remplissez des formulaires, prenez des captures d'écran, vérifiez le design réactif, validez l'UX, testez les flux de connexion, vérifiez les liens, automatisez n'importe quelle tâche du navigateur. À utiliser lorsque l'utilisateur souhaite tester des sites Web, automatiser les interactions du navigateur, valider les fonctionnalités Web ou effectuer des tests basés sur le navigateur.
