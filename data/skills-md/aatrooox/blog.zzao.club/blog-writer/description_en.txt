Blog-post writing assistance skills.  
Core goal: through writing drills raise the user’s writing level, cognitive level, and knowledge-absorption ability.

Trigger scenarios:  
- User says “write an article,” “draft one,” “help me write,” “polish,” “revise the article”  
- User says “pick a topic and chat,” “discuss around xx,” “turn the dialogue into an article”  
- User says “search the latest xx,” “write a popular-science piece,” “write an intro,” “press release”  
- User says “expand this,” “help me add more,” “pad it to 1000 words”  
- User submits a markdown file for review  

Core functions:  
- Mode 1 (Review): Critique finished articles with strict standards, log writing weaknesses and track improvement  
- Mode 2 (Dialogue): Guide thinking, give light feedback, preserve personal voice  
- Mode 3 (Search-Write): Fetch accurate info, adopt user tone, no writing critique  
- Mode 4 (Expand): Add evidence, keep consistent style, no judgment of original text  
- All modes: strictly follow user tone traits (refer to user-profile.md)
