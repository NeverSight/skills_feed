Desarrollo frontend multiplataforma uni-app con Vue 3 + TypeScript. Úselo para: (1) Crear nuevos proyectos o páginas de una sola aplicación, (2) Integrar bibliotecas de UI (uView Plus, uni-ui, TuniaoUI), (3) Implementar sistemas de diseño multitema, (4) Crear miniprogramas WeChat, H5 y otras plataformas, (5) Configurar temas SCSS y estilos globales, (6) Configurar el sistema de compilación Vite. Incluye plantillas reutilizables, patrones de componentes y mejores prácticas aplicables a cualquier proyecto de una sola aplicación.
