Meilleures pratiques Tailwind CSS v4 pour les thèmes, les jetons de conception, le mode sombre, les dégradés, la conception réactive et les animations. À utiliser lors de l'écriture, de la révision ou de la refactorisation du code CSS Tailwind. Se déclenche sur des tâches impliquant le style, le CSS, les couleurs, la thématique, le mode sombre, la conception réactive, les dégradés, les animations, les transitions, la mise en page ou les décisions de style de l'interface utilisateur. À utiliser également lors de la configuration du système de conception d'un nouveau projet ou de l'examen de la qualité du code CSS/Tailwind.
