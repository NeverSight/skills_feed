Erstellen und verwalten Sie Claude Code-Hooks zur Automatisierung von Arbeitsabläufen, zur Validierung von Eingaben und zur Anpassung des Verhaltens. Wird verwendet, wenn Benutzer das Erstellen von Hooks, das Einrichten der Automatisierung für die Tool-Nutzung (PreToolUse, PostToolUse), das Hinzufügen von Sitzungskontexten (SessionStart), das Validieren von Eingabeaufforderungen (UserPromptSubmit), das Steuern des Stoppverhaltens (Stop, SubagentStop) oder das Konfigurieren der Berechtigungsbehandlung (PermissionRequest) verwenden.
