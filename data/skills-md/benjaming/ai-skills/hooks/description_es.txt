Cree y administre enlaces de Claude Code para automatizar flujos de trabajo, validar entradas y personalizar el comportamiento. Úselo cuando los usuarios soliciten crear enlaces, configurar la automatización para el uso de herramientas (PreToolUse, PostToolUse), agregar contexto de sesión (SessionStart), validar mensajes (UserPromptSubmit), controlar el comportamiento de detención (Stop, SubagentStop) o configurar el manejo de permisos (PermissionRequest).
