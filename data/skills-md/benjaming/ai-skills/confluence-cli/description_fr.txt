Utilisez confluence-cli (package NPM) pour gérer le contenu, les pages et les espaces Confluence à partir de la ligne de commande. Idéal pour les flux de travail de documentation, les opérations de contenu en masse, la migration de pages et lorsque les utilisateurs demandent des interactions Confluence basées sur CLI. Déclenchez des requêtes telles que « utiliser Confluence CLI », « créer des pages Confluence via CLI », « migrer le contenu Confluence », « automatiser les workflows de documentation » ou lorsque les utilisateurs souhaitent scripter des opérations Confluence.
