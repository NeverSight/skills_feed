Utilisez Atlassian CLI (acli) pour gérer les éléments de travail, les projets et les workflows Jira à partir de la ligne de commande. Idéal pour les opérations en masse, l'automatisation, les scripts et lorsque les utilisateurs demandent des interactions Jira basées sur CLI. Déclenchez des requêtes telles que « utiliser Jira CLI », « créer des tickets Jira via CLI », « mettre à jour en masse des tickets Jira », « automatiser les workflows Jira » ou lorsque les utilisateurs souhaitent créer des scripts pour des opérations Jira.
