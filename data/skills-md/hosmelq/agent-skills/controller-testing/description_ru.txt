Пишет тесты функций Pest для HTTP-контроллеров Laravel с повторяемыми шаблонами для веб-сеансов и потоков API/JSON. Активируется для тестов функций, ориентированных на контроллер (особенно в тестах/Функция/Http/Контроллеры/**), матриц действий CRUD, проверок привязки вложенных маршрутов, результатов авторизации (403 против 404), наборов проверочных данных и утверждений постоянства.
