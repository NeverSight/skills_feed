Scrive test delle funzionalità Pest per i controller HTTP Laravel con modelli ripetibili per flussi Web/sessione e API/JSON. Si attiva per test di funzionalità incentrati sul controller (in particolare in tests/Feature/Http/Controllers/**), matrici di azioni CRUD, controlli di associazione di route nidificate, risultati di autorizzazione (403 vs 404), set di dati di convalida e asserzioni di persistenza.
