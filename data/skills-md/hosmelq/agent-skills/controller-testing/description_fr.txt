Écrit des tests de fonctionnalités Pest pour les contrôleurs HTTP Laravel avec des modèles reproductibles pour les flux Web/session et API/JSON. S'active pour les tests de fonctionnalités axés sur le contrôleur (en particulier sous tests/Feature/Http/Controllers/**), les matrices d'actions CRUD, les vérifications de liaison de route imbriquées, les résultats d'autorisation (403 contre 404), les ensembles de données de validation et les assertions de persistance.
