Ein Claude-Code-Skill, der Ihnen das Suchen, Durchsuchen und Installieren von Skills aus einer Datenbank mit mehr als 31.767 Community-Skills mit intelligenten Downloads auf Ordnerebene ermöglicht. Unterstützt SVN-Export, Git-Sparse-Checkout und HTTP-Fallback-Methoden für die vollständige Installation des Skill-Ordners.
