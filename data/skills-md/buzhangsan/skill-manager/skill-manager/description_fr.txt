Une compétence Claude Code qui vous permet de rechercher, parcourir et installer des compétences à partir d'une base de données de plus de 31 767 compétences communautaires avec des téléchargements intelligents au niveau des dossiers. Prend en charge l'exportation SVN, l'extraction sparse Git et les méthodes de secours HTTP pour une installation complète du dossier de compétences.
