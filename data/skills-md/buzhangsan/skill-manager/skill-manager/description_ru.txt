Навык Claude Code, который позволяет вам искать, просматривать и устанавливать навыки из базы данных, содержащей более 31 767 навыков сообщества, с интеллектуальными загрузками на уровне папок. Поддерживает экспорт SVN, разреженную проверку Git и резервные методы HTTP для полной установки папки навыков.
