مجموعة أدوات Valyu API الكاملة لوكلاء الذكاء الاصطناعي. استخدم هذه المهارة عندما يُطلب منك إجراء بحث في الوقت الفعلي عبر الويب، أو المصادر الأكاديمية، أو الطبية، أو مصادر النقل، أو المصادر المالية، أو استخراج المحتوى من عناوين URL، أو الإجابات المدعومة بالذكاء الاصطناعي مع الاستشهادات، أو التقارير البحثية العميقة الشاملة.
