Ultracite est un préréglage de peluchage et de formatage sans configuration pour les projets JavaScript/TypeScript. À utiliser lorsque : (1) Configuration ou initialisation d'Ultracite dans un projet (ultracite init), (2) Exécution de commandes de peluchage ou de formatage (vérification, correction, docteur), (3) Écriture ou révision de code JS/TS dans un projet qui utilise Ultracite - pour suivre ses normes de code, (4) Dépannage des problèmes de peluchage/formattage, (5) L'utilisateur mentionne "ultracite", "lint", "format", "qualité du code" ou 'biome/eslint/oxlint' dans un projet avec Ultracite installé.
