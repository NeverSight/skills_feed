Crée des animations performantes et des interactions basées sur les gestes dans les applications React Native (Expo) à l'aide de React Native Reanimated v4 et de React Native Gesture Handler (GestureDetector / API hook). À utiliser lors de l'implémentation de mouvements d'interface utilisateur, de transitions, d'animations de mise en page/entrée/sortie, de transitions/animations de style CSS, de gestes interactifs (panoramique/pincement/glisser/glisser), d'animations liées au défilement, de worklets/valeurs partagées ou de débogage de performances d'animation et de problèmes de thread.
