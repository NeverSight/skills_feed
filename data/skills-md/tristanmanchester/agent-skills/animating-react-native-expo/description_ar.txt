ينشئ رسومًا متحركة عالية الأداء وتفاعلات تعتمد على الإيماءات في تطبيقات React Native (Expo) باستخدام React Native Reanimated v4 وReact Native Gesture Handler (GestureDetector / Hook API). يُستخدم عند تنفيذ حركة واجهة المستخدم، أو الانتقالات، أو الرسوم المتحركة للتخطيط/الدخول/الخروج، أو الانتقالات/الرسوم المتحركة بنمط CSS، أو الإيماءات التفاعلية (التحريك/الضغط/السحب/السحب)، أو الرسوم المتحركة المرتبطة بالتمرير، أو العمل الصغير/القيم المشتركة، أو تصحيح أخطاء أداء الرسوم المتحركة ومشكلات الترابط.
