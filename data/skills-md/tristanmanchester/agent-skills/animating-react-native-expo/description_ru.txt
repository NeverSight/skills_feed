Создает производительную анимацию и взаимодействие на основе жестов в приложениях React Native (Expo) с использованием React Native Reanimated v4 и React Native Gesture Handler (GestureDetector/hook API). Используйте при реализации движения пользовательского интерфейса, переходов, анимации макета/входа/выхода, переходов/анимации в стиле CSS, интерактивных жестов (панорамирование/сведение/перетаскивание), анимации со ссылкой на прокрутку, рабочих листов/общих значений или отладки производительности анимации и проблем с потоками.
