React Native Reanimated v4 と React Native Gesture Handler (GestureDetector / フック API) を使用して、React Native (Expo) アプリでパフォーマンスの高いアニメーションとジェスチャー駆動のインタラクションを構築します。 UI モーション、トランジション、アニメーションのレイアウト/開始/終了、CSS スタイルのトランジション/アニメーション、インタラクティブ ジェスチャ (パン/ピンチ/スワイプ/ドラッグ)、スクロールリンク アニメーション、ワークレット/共有値を実装する場合、またはアニメーションのパフォーマンスとスレッドの問題をデバッグする場合に使用します。
