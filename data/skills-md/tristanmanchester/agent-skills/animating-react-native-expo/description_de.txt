Erstellt leistungsstarke Animationen und gestengesteuerte Interaktionen in React Native (Expo)-Apps mit React Native Reanimated v4 und React Native Gesture Handler (GestureDetector/Hook-API). Verwenden Sie diese Option zum Implementieren von UI-Bewegungen, Übergängen, Layout-/Eingabe-/Beendigungsanimationen, Übergängen/Animationen im CSS-Stil, interaktiven Gesten (Schwenken/Kneifen/Wischen/Ziehen), scrollverknüpften Animationen, Worklets/gemeinsamen Werten oder dem Debuggen von Animationsleistungs- und Threading-Problemen.
