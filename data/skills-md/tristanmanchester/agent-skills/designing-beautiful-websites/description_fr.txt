Conçoit l'UX/UI pour les sites Web et les applications Web : stratégie UX, architecture de l'information, flux d'utilisateurs, wireframes, systèmes d'interface utilisateur, spécifications des composants et conception visuelle soignée. À utiliser lorsque l'utilisateur demande à concevoir, repenser, moderniser, critiquer ou améliorer un site Web, une page de destination, une interface d'application Web, un système de conception, un guide de style ou un kit d'interface utilisateur.
