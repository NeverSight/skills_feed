تصميمات UX/UI لمواقع الويب وتطبيقات الويب: استراتيجية UX، وهندسة المعلومات، وتدفقات المستخدم، والإطارات السلكية، وأنظمة واجهة المستخدم، ومواصفات المكونات، والتصميم المرئي المصقول. يُستخدم عندما يطلب المستخدم تصميم أو إعادة تصميم أو تحديث أو نقد أو تحسين موقع ويب أو صفحة مقصودة أو واجهة تطبيق ويب أو نظام تصميم أو دليل نمط أو مجموعة أدوات واجهة المستخدم.
