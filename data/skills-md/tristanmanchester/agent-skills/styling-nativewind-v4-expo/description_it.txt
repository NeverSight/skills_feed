Configura e utilizza NativeWind v4 (Tailwind CSS v3) nelle app Expo React Native, incluso Expo Router. Configura tailwind.config.js, global.css, babel.config.js (jsxImportSource + nativewind/babel), metro.config.js (con NativeWind + input) e app.json (web bundler metro). Risolve i problemi relativi a "nomeclasse non applicabile", compilazione CLI Tailwind e cache Metro. Implementa componenti/varianti riutilizzabili, modalità oscura + temi tramite variabili CSS (vars/useColorScheme) e stili di componenti di terze parti (remapProps/cssInterop). Da utilizzare quando si lavora su progetti Expo utilizzando NativeWind v4, utilità className in stile Tailwind o durante il debug della configurazione di NativeWind.
