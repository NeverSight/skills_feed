يقوم بإعداد واستخدام NativeWind v4 (Tailwind CSS v3) في تطبيقات Expo React Native، بما في ذلك Expo Router. تكوين tailwind.config.js، وglobal.css، وbabel.config.js (jsxImportSource + أصليwind/babel)، وmetro.config.js (مع NativeWind + input)، وapp.json (مجمع حزم الويب Metro). استكشاف أخطاء "اسم الفئة غير مطبق"، وتجميع Tailwind CLI، ومشكلات ذاكرة التخزين المؤقت للمترو وإصلاحها. ينفذ مكونات/متغيرات قابلة لإعادة الاستخدام، والوضع المظلم + السمات عبر متغيرات CSS (vars/useColorScheme)، وتصميم مكونات الطرف الثالث (remapProps/cssInterop). يُستخدم عند العمل على مشاريع Expo باستخدام NativeWind v4، أو الأدوات المساعدة لنمط Tailwind className، أو عند تصحيح أخطاء تكوين NativeWind.
