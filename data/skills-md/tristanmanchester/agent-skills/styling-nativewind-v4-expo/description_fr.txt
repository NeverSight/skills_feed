Configure et utilise NativeWind v4 (Tailwind CSS v3) dans les applications Expo React Native, y compris Expo Router. Configure tailwind.config.js, global.css, babel.config.js (jsxImportSource + nativewind/babel), metro.config.js (withNativeWind + input) et app.json (web bundler metro). Résout les problèmes de « className not apply », de compilation Tailwind CLI et de cache Metro. Implémente des composants/variantes réutilisables, le mode sombre + la thématique via des variables CSS (vars/useColorScheme) et le style de composants tiers (remapProps/cssInterop). À utiliser lorsque vous travaillez sur des projets Expo à l'aide de NativeWind v4, des utilitaires className de style Tailwind ou lors du débogage de la configuration NativeWind.
