Configura y utiliza NativeWind v4 (Tailwind CSS v3) en aplicaciones Expo React Native, incluido Expo Router. Configura tailwind.config.js, global.css, babel.config.js (jsxImportSource + Nativewind/babel), metro.config.js (con NativeWind + entrada) y app.json (paquete web metro). Soluciona problemas de “nombre de clase no aplicable”, compilación de Tailwind CLI y problemas de caché de Metro. Implementa componentes/variantes reutilizables, modo oscuro + tematización a través de variables CSS (vars/useColorScheme) y estilo de componentes de terceros (remapProps/cssInterop). Úselo cuando trabaje en proyectos de Expo usando NativeWind v4, utilidades de nombre de clase estilo Tailwind o cuando depure la configuración de NativeWind.
