Настраивает и использует NativeWind v4 (Tailwind CSS v3) в приложениях Expo React Native, включая Expo Router. Настраивает Tailwind.config.js, global.css, Babel.config.js (jsxImportSource + Nativewind/babel), Metro.config.js (withNativeWind + input) и App.json (Metro веб-пакета). Устраняет неполадки «имя класса не применяется», компиляцию интерфейса командной строки Tailwind и проблемы с кэшем Metro. Реализует повторно используемые компоненты/варианты, темный режим + оформление тем с помощью переменных CSS (vars/useColorScheme) и стилизацию сторонних компонентов (remapProps/cssInterop). Используйте при работе над проектами Expo с использованием NativeWind v4, утилит className в стиле Tailwind или при отладке конфигурации NativeWind.
