Richtet NativeWind v4 (Tailwind CSS v3) in Expo React Native-Apps, einschließlich Expo Router, ein und verwendet es. Konfiguriert tailwind.config.js, global.css, babel.config.js (jsxImportSource + nativewind/babel), metro.config.js (withNativeWind + input) und app.json (Web-Bundler-Metro). Behebt „className not apply“, Tailwind CLI-Kompilierung und Metro-Cache-Probleme. Implementiert wiederverwendbare Komponenten/Varianten, Dark Mode + Theming über CSS-Variablen (vars/useColorScheme) und Komponentenstil von Drittanbietern (remapProps/cssInterop). Wird verwendet, wenn Sie an Expo-Projekten mit NativeWind v4, ClassName-Dienstprogrammen im Tailwind-Stil arbeiten oder wenn Sie die NativeWind-Konfiguration debuggen.
