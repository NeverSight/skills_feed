Этот навык следует использовать при работе с HZL для отслеживания задач, когда пользователь просит «разбить работу на задачи», «отслеживать задачи с помощью HZL», «заявить задачу», «прогресс контрольной точки», «завершить задачу» или при работе над проектом, использующим HZL. Содержит рекомендации по эффективным шаблонам управления задачами для агентов ИИ.
