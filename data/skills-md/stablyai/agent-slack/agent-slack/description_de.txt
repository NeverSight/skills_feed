Slack-Automatisierungs-CLI für KI-Agenten. Verwenden Sie, wenn:
- Lesen einer Slack-Nachricht oder eines Slack-Threads (angegebene URL oder Kanal+TS)
- Herunterladen von Slack-Anhängen (Snippets, Bilder, Dateien) in lokale Pfade
- Durchsuchen von Slack-Nachrichten oder -Dateien
- Senden einer Antwort oder Hinzufügen/Entfernen einer Reaktion
- Abrufen eines Slack-Canvas als Markdown
- Nach Slack-Benutzern suchen
Auslöser: „Slack-Nachricht“, „Slack-Thread“, „Slack-URL“, „Slack-Link“, „Slack lesen“, „Auf Slack antworten“, „Slack durchsuchen“
