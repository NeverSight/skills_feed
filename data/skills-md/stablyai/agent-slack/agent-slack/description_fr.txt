CLI d’automatisation Slack pour les agents IA. À utiliser lorsque :
- Lecture d'un message ou d'un fil de discussion Slack (à partir d'une URL ou d'un canal + ts)
- Téléchargement des pièces jointes Slack (extraits, images, fichiers) vers des chemins locaux
- Recherche de messages ou de fichiers Slack
- Envoyer une réponse ou ajouter/supprimer une réaction
- Récupérer un canevas Slack en tant que démarque
- Recherche d'utilisateurs Slack
Déclencheurs : « message Slack », « fil de discussion Slack », « URL Slack », « lien Slack », « lire Slack », « répondre sur Slack », « rechercher Slack »
