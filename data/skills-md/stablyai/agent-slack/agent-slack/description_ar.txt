Slack Automation CLI لوكلاء الذكاء الاصطناعي. استخدم عندما:
- قراءة رسالة أو موضوع Slack (مع إعطاء عنوان URL أو قناة + ts)
- تنزيل مرفقات Slack (المقتطفات والصور والملفات) على المسارات المحلية
- البحث في رسائل أو ملفات Slack
- إرسال رد أو إضافة/إزالة رد فعل
- جلب قماش Slack كتخفيض
- البحث عن مستخدمي سلاك
