Экспертный помощник по настройке Stable Playwright SDK. Используйте этот навык, когда
установка Stable SDK в новый проект, переход с @playwright/test,
или настроить Stable Reporter для CI/CD. Запускается при выполнении таких задач, как «настройка
стабильно», «установить стабильный SDK» или «настроить драматурга со стабильно».
