Experten-Setup-Assistent für das Stably Playwright SDK. Benutze diese Fähigkeit, wenn
Installation des Stably SDK in einem neuen Projekt, Migration von @playwright/test,
oder Konfigurieren von Stably Reporter für CI/CD. Wird bei Aufgaben wie „setup
stabil“, „stabil sdk installieren“ oder „playwright mit stabil konfigurieren“.
