Эксперт-помощник для инструмента Stable CLI. Используйте этот навык при работе с
стабильно команды для создания, запуска и исправления тестов драматурга с использованием ИИ.
Запускается при таких задачах, как «создать стабильные тесты», «исправить неудачные тесты»,
«запустить стабильный тест» или «использовать стабильный интерфейс».
