Reglas de IA y referencia del SDK para escribir pruebas con Stably Playwright SDK.

características. Activadores de tareas que involucran toMatchScreenshotPrompt, agent.act(),
page.extract(), o al decidir entre los métodos Playwright y Stably SDK.
