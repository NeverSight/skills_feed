Règles d'IA et référence du SDK pour l'écriture de tests avec le SDK Stably Playwright.
Utilisez cette compétence lors de l'écriture ou de la modification de tests Playwright avec Stably AI
fonctionnalités. Déclencheurs sur des tâches impliquant toMatchScreenshotPrompt, agent.act(),
page.extract(), ou lors du choix entre les méthodes Playwright et Stably SDK.
Inclut les meilleures pratiques pour les assertions d'IA, l'extraction et les agents autonomes.
