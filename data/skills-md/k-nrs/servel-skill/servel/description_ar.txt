منصة نشر ذاتية الاستضافة عبر Docker Swarm. ينشر التطبيقات، ويدير أكثر من 45 نوعًا من أنواع البنية التحتية (قواعد البيانات، وقوائم الانتظار، وذاكرات التخزين المؤقت، والأنظمة الأساسية)، ويتعامل مع الأسرار، والمجالات، والنسخ الاحتياطية، وعمليات الخادم. المشغلات عند النشر، أو البنية التحتية، أو postgres، أو redis، أو supabase، أو النسخ الاحتياطي، أو الاستعادة، أو السجلات، أو الأسرار، أو SSL، أو المجالات، أو وضع dev، أو CI/CD، أو التنبيهات، أو traefik، أو التوجيه، أو إدارة الخادم.
