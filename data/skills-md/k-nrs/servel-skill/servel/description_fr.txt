Plateforme de déploiement auto-hébergée via Docker Swarm. Déploie des applications, gère plus de 45 types d'infrastructures (bases de données, files d'attente, caches, plates-formes), gère les secrets, les domaines, les sauvegardes et les opérations de serveur. Déclencheurs sur le déploiement, l'infrastructure, postgres, redis, supabase, la sauvegarde, la restauration, les journaux, les secrets, SSL, les domaines, le mode développement, CI/CD, les alertes, le trafic, le routage ou la gestion du serveur.
