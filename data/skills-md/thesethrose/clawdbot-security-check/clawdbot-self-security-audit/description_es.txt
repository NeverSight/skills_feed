Realice una auditoría de seguridad integral de solo lectura de la configuración propia de Clawdbot. Esta es una habilidad basada en conocimientos que le enseña a Clawdbot a identificar oportunidades de refuerzo y generar informes. Úselo cuando el usuario solicite "ejecutar verificación de seguridad", "auditar clawdbot", "verificar refuerzo de seguridad" o "qué vulnerabilidades tengo". Esta habilidad solo LEE la configuración y genera informes; nunca modifica la configuración ni ejecuta correcciones automáticamente. Diseñado para ser extensible: se pueden agregar nuevas comprobaciones actualizando el conocimiento de esta habilidad.
