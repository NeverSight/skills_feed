Выполняйте поиск в Интернете, получайте быстрые ответы ИИ с проверяемыми ссылками и извлекайте веб-контент с помощью инструментов JSON CLI на основе схемы You.com, оптимизированных для агентов ИИ на основе bash (OpenClaw, Claude Code, Codex, Cursor и т. д.). Быстрее, чем встроенные поисковые API, с одновременным сканированием в реальном времени, мгновенным извлечением контента и ответами, подтвержденными цитированием. Обнаружение схемы с помощью флага --schema позволяет создавать программные запросы.
