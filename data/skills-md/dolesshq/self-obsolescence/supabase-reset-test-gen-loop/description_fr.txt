Exécutez de manière itérative une réinitialisation Supabase locale, des tests de base de données et une génération de type jusqu'à ce qu'ils réussissent, en corrigeant les problèmes entre les exécutions. À utiliser lorsqu'on vous demande de réinitialiser la base de données Supabase locale, d'exécuter « supabase test db » et de régénérer les types (« npm run gen-types ») dans une boucle.
