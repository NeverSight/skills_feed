Führen Sie iterativ einen lokalen Supabase-Reset, Datenbanktests und Typgenerierung durch, bis sie erfolgreich sind, und beheben Sie Probleme zwischen den Läufen. Verwenden Sie diese Option, wenn Sie aufgefordert werden, die lokale Supabase-Datenbank zurückzusetzen, „supabase test db“ auszuführen und Typen („npm run gen-types“) in einer Schleife neu zu generieren.
