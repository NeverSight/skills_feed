Итеративно запускайте локальный сброс Supabase, тесты базы данных и генерацию типов до тех пор, пока они не будут успешными, устраняя проблемы между запусками. Используйте, когда вас попросят сбросить локальную базу данных Supabase, запустить `supabase test db` и повторно сгенерировать типы (`npm run gen-types`) в цикле.
