Управляйте собственным браузером Chrome пользователя через расширение Playwriter с фрагментами кода Playwright в локальной песочнице js с отслеживанием состояния через интерфейс playwriter. Автоматизируйте веб-взаимодействия, делайте снимки экрана, проверяйте деревья доступности, отлаживайте и профилируйте веб-приложения. Запустите команду «Навык драматурга», чтобы прочитать полную актуальную информацию о навыках.
