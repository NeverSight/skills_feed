قم بإنشاء أوامر شرطة مائلة مخصصة لـ Claude Code من خلال تدفق الأسئلة الذكي من 5 إلى 7. ينشئ أوامر قوية لأبحاث الأعمال، وتحليل المحتوى، والامتثال للرعاية الصحية، وتكامل واجهة برمجة التطبيقات (API)، وأتمتة الوثائق، وتحسين سير العمل. إخراج الأوامر المنظمة للأوامر التي تم إنشاؤها/ مع إرشادات التحقق من الصحة والتثبيت.
