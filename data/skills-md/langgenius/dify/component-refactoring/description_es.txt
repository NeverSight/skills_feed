Refactorice componentes de React de alta complejidad en la interfaz de Dify. Úselo cuando `pnpm analyse-component --json` muestre complejidad > 50 o lineCount > 300, cuando el usuario solicite división de código, extracción de enlaces o reducción de complejidad, o cuando `pnpm analyse-component` advierta que se refactorice antes de realizar la prueba; evítelo para componentes simples/bien estructurados, envoltorios de terceros o cuando el usuario desee explícitamente realizar pruebas sin refactorizar.
