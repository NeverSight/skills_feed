Руководство по реализации шаблонов API oRPC с приоритетом контракта во внешнем интерфейсе Dify. Срабатывает при создании новых контрактов API, добавлении конечных точек службы, интеграции TanStack Query с типизированными контрактами или переносе вызовов устаревших служб в oRPC. Используйте для всей работы на уровне API в каталогах web/contract и web/service.
