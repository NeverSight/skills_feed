دليل لتنفيذ أنماط واجهة برمجة التطبيقات (API) الخاصة بعقد oRPC في الواجهة الأمامية لـ Dify. يتم التشغيل عند إنشاء عقود API جديدة، أو إضافة نقاط نهاية الخدمة، أو دمج TanStack Query مع العقود المكتوبة، أو ترحيل استدعاءات الخدمة القديمة إلى oRPC. يستخدم لجميع أعمال طبقة API في أدلة الويب/العقد والويب/الخدمة.
