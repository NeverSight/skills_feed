Guida per l'implementazione di modelli API contract-first oRPC nel frontend Dify. Si attiva durante la creazione di nuovi contratti API, l'aggiunta di endpoint di servizio, l'integrazione di TanStack Query con contratti tipizzati o la migrazione delle chiamate di servizio legacy a oRPC. Utilizzare per tutto il lavoro del livello API nelle directory web/contratto e web/servizi.
