Guía para implementar patrones de API de contrato primero oRPC en la interfaz de Dify. Se activa al crear nuevos contratos API, agregar puntos finales de servicio, integrar TanStack Query con contratos escritos o migrar llamadas de servicio heredadas a oRPC. Úselo para todo el trabajo de la capa API en directorios web/contratos y web/servicios.
