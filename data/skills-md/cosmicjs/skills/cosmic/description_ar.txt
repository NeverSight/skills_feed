أنشئ تطبيقات باستخدام نظام إدارة المحتوى Cosmic Headless CMS باستخدام JavaScript SDK. يغطي الكائنات والوسائط وتوليد الذكاء الاصطناعي (النصوص والصور والفيديو) ونمذجة المحتوى باستخدام حقول التعريف والاستعلامات. يُستخدم عند العمل مع Cosmic CMS، أو @cosmicjs/sdk، أو المحتوى بدون رأس، أو إنشاء تطبيقات JAMstack.
