Добавляет компонент нижнего колонтитула «Запрос сводки AI» с кликабельными значками платформы AI (ChatGPT, Claude, Gemini, Grok, Perplexity), которые предварительно заполняют подсказки для пользователей, чтобы получить сводки AI веб-сайта. При необходимости создается файл llms.txt для улучшения обнаружения ИИ. Используйте, когда пользователи хотят добавить кнопки интеграции платформы AI или сделать свой веб-сайт удобным для искусственного интеллекта.
