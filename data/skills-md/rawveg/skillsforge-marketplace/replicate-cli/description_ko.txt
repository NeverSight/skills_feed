이 기술은 Replicate CLI를 사용하여 AI 모델을 실행하고, 예측을 생성하고, 배포를 관리하고, 모델을 미세 조정하는 데 대한 포괄적인 지침을 제공합니다. 사용자가 이미지 생성 모델, 언어 모델 또는 Replicate에서 호스팅되는 모든 ML 모델 실행을 포함하여 명령줄을 통해 Replicate의 AI 모델 플랫폼과 상호 작용하려는 경우 이 기술을 사용합니다. 이 기술은 사용자가 Replicate에서 모델 실행, 예측 생성, 배포 관리, 모델 미세 조정 또는 CLI를 통한 Replicate API 작업에 관해 질문할 때 사용해야 합니다.
