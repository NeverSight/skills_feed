このスキルは、Replicate CLI を使用して AI モデルを実行し、予測を作成し、デプロイメントを管理し、モデルを微調整するための包括的なガイダンスを提供します。このスキルは、ユーザーがコマンド ライン経由で Replicate の AI モデル プラットフォームと対話する場合 (画像生成モデル、言語モデル、または Replicate でホストされている ML モデルの実行など) に使用します。このスキルは、ユーザーが Replicate でのモデルの実行、予測の作成、デプロイメントの管理、モデルの微調整、または CLI を介した Replicate API の操作について質問するときに使用する必要があります。
