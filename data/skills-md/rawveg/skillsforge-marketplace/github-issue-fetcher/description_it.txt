Questa abilità dovrebbe essere utilizzata quando gli utenti hanno bisogno di recuperare problemi di GitHub dal repository corrente, incluso far funzionare il numero successivo, recuperare campi di problemi specifici (corpo, numero, titolo, etichette, ecc.) o interrogare problemi con ordinamento e filtro personalizzati. Si attivano richieste come "ottieni il prossimo numero di GitHub", "qual è il numero del prossimo numero", "mostrami il corpo del problema" o "trova i problemi con l'etichetta X".
