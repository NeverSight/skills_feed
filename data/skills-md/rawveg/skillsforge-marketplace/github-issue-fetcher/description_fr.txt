Cette compétence doit être utilisée lorsque les utilisateurs ont besoin de récupérer des problèmes GitHub à partir du référentiel actuel, notamment pour travailler sur le problème suivant, récupérer des champs de problème spécifiques (corps, numéro, titre, étiquettes, etc.) ou interroger des problèmes avec un tri et un filtrage personnalisés. Se déclenche sur des requêtes telles que « obtenir le prochain numéro de problème GitHub », « quel est le prochain numéro de problème », « me montrer le corps du problème » ou « rechercher des problèmes avec l'étiquette X ».
