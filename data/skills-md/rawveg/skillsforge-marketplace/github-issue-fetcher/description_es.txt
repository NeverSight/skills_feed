Esta habilidad debe usarse cuando los usuarios necesitan recuperar problemas de GitHub del repositorio actual, lo que incluye obtener el siguiente problema en el que trabajar, recuperar campos de problemas específicos (cuerpo, número, título, etiquetas, etc.) o consultar problemas con clasificación y filtrado personalizados. Activadores de solicitudes como "obtener el próximo número de GitHub", "cuál es el próximo número de problema", "muéstrame el cuerpo del problema" o "buscar problemas con la etiqueta X".
