Этот навык следует использовать, когда пользователям необходимо получить задачи GitHub из текущего репозитория, в том числе получить следующую задачу для работы, получить определенные поля задачи (тело, номер, заголовок, метки и т. д.) или запросить проблемы с помощью настраиваемой сортировки и фильтрации. Запускается по таким запросам, как «получить следующую задачу GitHub», «какой номер следующей проблемы», «покажи мне тело задачи» или «найти проблемы с меткой X».
