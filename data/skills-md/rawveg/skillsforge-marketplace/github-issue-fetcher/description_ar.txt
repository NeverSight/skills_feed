يجب استخدام هذه المهارة عندما يحتاج المستخدمون إلى جلب مشكلات GitHub من المستودع الحالي، بما في ذلك الحصول على الإصدار التالي للعمل عليه، واسترداد حقول المشكلات المحددة (النص، والرقم، والعنوان، والتسميات، وما إلى ذلك)، أو الاستعلام عن المشكلات باستخدام الفرز والتصفية المخصصة. مشغلات لطلبات مثل "الحصول على إصدار GitHub التالي"، أو "ما هو رقم الإصدار التالي"، أو "إظهار نص المشكلة"، أو "البحث عن المشكلات المتعلقة بالتسمية X".
