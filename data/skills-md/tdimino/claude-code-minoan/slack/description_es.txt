Integración del espacio de trabajo de Slack: 7 scripts bajo demanda (publicar, leer, buscar, reaccionar, cargar, canales, usuarios) + Puente de sesión (conecte cualquier sesión de Claude Code a Slack mediante escucha en segundo plano + inbox.jsonl). El demonio, el motor del alma y el sistema de memoria viven en Claudicle (github.com/tdimino/claudicle).
