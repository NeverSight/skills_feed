Создавайте и редактируйте высококачественные изображения с помощью искусственного интеллекта Nano Banana Pro (Gemini 3 Pro Image) от Google. Используйте этот навык при работе с задачами создания или редактирования изображений AI, включая (1) создание изображений из текстовых подсказок с профессиональным качеством, (2) редактирование существующих изображений с помощью инструкций на естественном языке, (3) многоэтапную итеративную обработку изображений, (4) фотографии продуктов и маркетинговые визуальные эффекты, (5) создание инфографики и диаграмм с текстом, (6) создание фотореалистичных и художественных изображений, (7) рабочие процессы пакетной генерации изображений, (8) тестирование и устранение неполадок API генерации изображений Gemini. Поддерживает разрешение до 4K, несколько форматов изображения, эталонные изображения и расширенные методы подсказок для получения результатов кинематографического качества.
