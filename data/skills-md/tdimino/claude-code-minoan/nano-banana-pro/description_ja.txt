Google の Nano Banana Pro (Gemini 3 Pro Image) AI モデルを使用して高品質の画像を生成および編集します。このスキルは、(1) プロの品質でテキスト プロンプトから画像を作成する、(2) 自然言語命令を使用して既存の画像を編集する、(3) 複数回の反復的な画像の改良、(4) 製品の写真とマーケティング ビジュアル、(5) テキストを使用したインフォグラフィックと図の作成、(6) フォトリアリスティックで芸術的な画像の生成、(7) バッチ画像生成のワークフロー、(8) Gemini 画像生成 API のテストとトラブルシューティングなどの AI 画像生成または編集タスクを行うときに使用します。最大 4K の解像度、複数のアスペクト比、参照画像、映画品質の結果をもたらす高度なプロンプト技術をサポートします。
