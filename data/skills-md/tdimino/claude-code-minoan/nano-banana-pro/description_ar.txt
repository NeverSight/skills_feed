قم بإنشاء وتحرير صور عالية الجودة باستخدام نموذج الذكاء الاصطناعي Nano Banana Pro (Gemini 3 Pro Image) من Google. استخدم هذه المهارة عند العمل مع إنشاء صور الذكاء الاصطناعي أو مهام التحرير بما في ذلك (1) إنشاء صور من المطالبات النصية بجودة احترافية، (2) تحرير الصور الموجودة بتعليمات اللغة الطبيعية، (3) تحسين الصور التكراري متعدد الدورات، (4) تصوير المنتجات والمرئيات التسويقية، (5) إنشاء الرسوم البيانية والرسوم البيانية مع النص، (6) إنشاء صور واقعية وفنية، (7) سير عمل إنشاء الصور دفعة واحدة، (8) اختبار واستكشاف أخطاء Gemini image generator API. يدعم دقة تصل إلى 4K ونسب العرض إلى الارتفاع المتعددة والصور المرجعية وتقنيات المطالبة المتقدمة للحصول على نتائج بجودة سينمائية.
