Создавайте агенты ИИ с помощью Claude Agent SDK. Охватывает функции запросов, ClaudeSDKClient, пользовательские инструменты, серверы MCP, перехватчики, разрешения, субагенты и управление браузером (расширение Chrome, навыки работы с браузером). Используйте при создании агентов, добавлении инструментов, настройке поведения агентов или автоматизации задач браузера.
