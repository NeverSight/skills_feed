Créez des agents IA à l'aide du SDK Claude Agent. Couvre les fonctions de requête, ClaudeSDKClient, les outils personnalisés, les serveurs MCP, les hooks, les autorisations, les sous-agents et le contrôle du navigateur (extension Chrome, compétence de navigateur de développement). À utiliser lors de la création d'agents, de l'ajout d'outils, de la configuration du comportement des agents ou de l'automatisation des tâches du navigateur.
