قم ببناء وكلاء الذكاء الاصطناعي باستخدام Claude Agent SDK. يغطي وظائف الاستعلام، ClaudeSDKClient، والأدوات المخصصة، وخوادم MCP، والخطافات، والأذونات، والوكلاء الفرعيين، والتحكم في المتصفح (امتداد Chrome، ومهارة متصفح المطورين). يُستخدم عند إنشاء وكلاء، أو إضافة أدوات، أو تكوين سلوك الوكيل، أو أتمتة مهام المتصفح.
