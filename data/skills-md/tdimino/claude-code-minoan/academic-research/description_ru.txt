Этот навык следует использовать для поиска научных статей, обзоров литературы и синтеза исследований. Сочетает Exa MCP (категория Research_paper, фильтрация arxiv.org) с arxiv-mcp-server для поиска, загрузки и глубокого анализа статей. Используйте при поиске статей, проведении обзоров литературы, анализе тенденций исследований или обобщении результатов нескольких статей.
