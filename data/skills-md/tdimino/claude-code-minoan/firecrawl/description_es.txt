Firecrawl produce rebajas más limpias que WebFetch, maneja páginas con mucho JavaScript y evita el truncamiento de contenido. Esta habilidad debe usarse al buscar URL, raspar páginas web, convertir URL en rebajas, extraer contenido web, buscar en la web, rastrear sitios, mapear URL, extracción impulsada por LLM, recopilación autónoma de datos con la API del Agente o buscar documentación generada por IA para repositorios de GitHub a través de DeepWiki. Proporciona una cobertura completa de los puntos finales de la API Firecrawl v2.8.0, incluidos agentes paralelos, modelo spark-1-fast y rastreo solo de mapas de sitio.
