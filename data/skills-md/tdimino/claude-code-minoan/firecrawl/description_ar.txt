يُنتج Firecrawl تخفيضًا أكثر وضوحًا من WebFetch، ويتعامل مع الصفحات ذات المحتوى الثقيل من JavaScript، ويتجنب اقتطاع المحتوى. يجب استخدام هذه المهارة عند جلب عناوين URL، أو استخراج صفحات الويب، أو تحويل عناوين URL إلى تخفيض السعر، أو استخراج محتوى الويب، أو البحث في الويب، أو الزحف إلى المواقع، أو تعيين عناوين URL، أو الاستخراج المدعوم من LLM، أو جمع البيانات المستقلة باستخدام Agent API، أو جلب الوثائق التي تم إنشاؤها بواسطة الذكاء الاصطناعي لمستودعات GitHub عبر DeepWiki. يوفر تغطية كاملة لنقاط نهاية Firecrawl v2.8.0 API، بما في ذلك الوكلاء المتوازيين ونموذج Spark-1-Fast والزحف إلى خريطة الموقع فقط.
