Firecrawl создает более чистую разметку, чем WebFetch, обрабатывает страницы с большим количеством JavaScript и избегает усечения контента. Этот навык следует использовать при получении URL-адресов, парсинге веб-страниц, преобразовании URL-адресов в уценку, извлечении веб-контента, поиске в Интернете, сканировании сайтов, сопоставлении URL-адресов, извлечении с помощью LLM, автономном сборе данных с помощью Agent API или получении документации, созданной искусственным интеллектом, для репозиториев GitHub через DeepWiki. Обеспечивает полный охват конечных точек API Firecrawl v2.8.0, включая параллельные агенты, модель Spark-1-Fast и сканирование только карты сайта.
