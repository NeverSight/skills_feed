Расширенный поиск Exa AI с 5 специализированными скриптами для нейронного веб-поиска, извлечения контента, обнаружения похожих страниц, быстрого исследования с цитатами и асинхронного профессионального исследования со структурированным выводом. Этот навык следует использовать при выполнении веб-поиска, извлечении контента из URL-адресов, поиске похожих страниц или проведении исследований с помощью искусственного интеллекта. Обеспечивает полный доступ ко всем конечным точкам API Exa, включая /search, /contents, /findSimilar, /answer и /research/v1.
