Sekundäre lokale LLM-Inferenz-Engine über llama.cpp. Diese Fähigkeit sollte verwendet werden, wenn GGUF-Modelle direkt ausgeführt werden, LoRA-Adapter für Kothar geladen werden, die Inferenzgeschwindigkeit bewertet wird oder Modelle über einen Lama-Server bereitgestellt werden. Ergänzt Ollama (das primär für RLAMA und den allgemeinen Gebrauch bleibt).
