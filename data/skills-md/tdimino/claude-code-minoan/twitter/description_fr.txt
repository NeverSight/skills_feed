Intégration Twitter/X avec trois modes : recherche/recherche API v2 officielle via x-search (paiement à l'utilisation, 0,005 $/lecture), publication/lecture basée sur la session via bird CLI (gratuit, cookies du navigateur) et archivage des signets via Smaug. Cette compétence doit être utilisée lors de la recherche de tweets, de recherches de sujets sur X, de publication, de surveillance de comptes ou d'archivage de signets.
