Integrazione Twitter/X con tre modalità: ricerca/ricerca API ufficiale v2 tramite x-search (pay-per-use, $ 0,005/lettura), pubblicazione/lettura basata sulla sessione tramite bird CLI (gratuito, cookie del browser) e archiviazione dei segnalibri tramite Smaug. Questa abilità dovrebbe essere utilizzata durante la ricerca di tweet, la ricerca di argomenti su X, la pubblicazione, il monitoraggio di account o l'archiviazione di segnalibri.
