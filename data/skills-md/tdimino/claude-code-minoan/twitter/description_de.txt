Twitter/X-Integration mit drei Modi: offizielle API v2-Suche/Recherche über x-search (Pay-per-Use, 0,005 $/Lesung), sitzungsbasiertes Posten/Lesen über Bird CLI (kostenlos, Browser-Cookies) und Lesezeichenarchivierung über Smaug. Diese Fähigkeit sollte beim Durchsuchen von Tweets, beim Recherchieren von Themen auf X, beim Posten, beim Überwachen von Konten oder beim Archivieren von Lesezeichen verwendet werden.
