Интеграция Twitter/X с тремя режимами: официальный поиск/исследование API v2 через x-search (плата за использование, 0,005 доллара США/чтение), публикация/чтение на основе сеанса через Bird CLI (бесплатно, файлы cookie браузера) и архивирование закладок через Smaug. Этот навык следует использовать при поиске твитов, исследовании тем на X, публикации сообщений, отслеживании учетных записей или архивировании закладок.
