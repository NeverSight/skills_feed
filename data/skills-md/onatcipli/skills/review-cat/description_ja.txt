App Store のレビュー分析、競合他社のインテリジェンス、iOS/macOS アプリの自動応答生成。 Apple App Store のレビューを使用して次のタスクのいずれかを行う場合は、このスキルを使用します。(1) App Store Connect API を介して独自のアプリの顧客レビューを取得および分析する、(2) パブリック RSS フィードまたは iTunes Search API を介して競合他社のアプリのレビューを取得する、(3) レビューに対するセンチメント分析、テーマ抽出、またはバグ/機能リクエストの分類を実行する、(4) 顧客レビューに対するコンテキストを認識した開発者の返信を生成する、(5) 評価、レビューのセンチメントを比較する、 (6) 長期にわたる評価傾向とレビュー量の追跡、(7) レビューデータからの概要や実用的な洞察レポートの作成、(8) App Store でのアプリの名前またはカテゴリによる検索。
