Marco completo para preparar proyectos de GitHub para un lanzamiento sostenible de código abierto. Cubre el escaneo de seguridad con Git History Cleaner, fundamentos legales, gobernanza, incorporación de contribuyentes, expectativas de los mantenedores y configuración de los patrocinadores de GitHub. Úselo al lanzar un proyecto públicamente, preparar un repositorio privado para código abierto o reforzar un repositorio público existente para mantenimiento a largo plazo.
