Framework completo per preparare progetti GitHub per un rilascio open source sostenibile. Copre la scansione della sicurezza con Git History Cleaner, i fondamenti legali, la governance, l'onboarding dei contributori, le aspettative dei manutentori e la configurazione degli sponsor GitHub. Da utilizzare quando si lancia pubblicamente un progetto, si prepara un repository privato per l'open source o si rafforza un repository pubblico esistente per la manutenzione a lungo termine.
