Полная основа для подготовки проектов GitHub к устойчивому выпуску с открытым исходным кодом. Охватывает сканирование безопасности с помощью Git History Cleaner, правовые основы, управление, регистрацию участников, ожидания сопровождающих и настройку спонсоров GitHub. Используйте при публичном запуске проекта, подготовке частного репозитория для открытого исходного кода или усилении существующего общедоступного репозитория для долгосрочного обслуживания.
