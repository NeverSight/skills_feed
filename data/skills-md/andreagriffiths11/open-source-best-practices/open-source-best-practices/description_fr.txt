Cadre complet pour préparer les projets GitHub pour une version open source durable. Couvre l'analyse de sécurité avec Git History Cleaner, les fondements juridiques, la gouvernance, l'intégration des contributeurs, les attentes des responsables et la configuration des sponsors GitHub. À utiliser lors du lancement public d'un projet, de la préparation d'un dépôt privé pour l'open source ou du renforcement d'un dépôt public existant pour une maintenance à long terme.
