Komplettes Framework zur Vorbereitung von GitHub-Projekten für eine nachhaltige Open-Source-Veröffentlichung. Behandelt Sicherheitsscans mit Git History Cleaner, rechtliche Grundlagen, Governance, Einbindung von Mitwirkenden, Erwartungen an Betreuer und Einrichtung von GitHub-Sponsoren. Verwenden Sie es, wenn Sie ein Projekt öffentlich starten, ein privates Repo für Open Source vorbereiten oder ein vorhandenes öffentliches Repo für die langfristige Wartung härten.
