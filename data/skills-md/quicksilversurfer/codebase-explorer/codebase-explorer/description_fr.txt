Une méthode structurée pour explorer et comprendre des bases de code inconnues. Utilisez cette compétence chaque fois que l'utilisateur demande à explorer, comprendre, auditer, réviser ou parcourir une base de code dans laquelle il n'a jamais travaillé auparavant. Se déclenche également lorsque l'utilisateur dit des choses telles que "aidez-moi à comprendre ce code", "que fait ce dépôt", "guidez-moi à travers cette base de code", "explorez ce projet", "j'ai hérité de ce code", "intégrez-moi à ce système", ou pose des questions sur l'architecture, la structure ou la santé d'une base de code existante. Déclenchez même s'ils abandonnent simplement un chemin de dépôt et disent "qu'est-ce que c'est". Cette compétence produit une compréhension structurée – non pas des résumés, mais des cartes navigables du fonctionnement réel du code.
