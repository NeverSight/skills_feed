Eine strukturierte Methode zum Erkunden und Verstehen unbekannter Codebasen. Verwenden Sie diese Fähigkeit immer dann, wenn der Benutzer eine Codebasis erforschen, verstehen, prüfen, überprüfen oder navigieren möchte, in der er noch nie gearbeitet hat. Wird auch ausgelöst, wenn der Benutzer Dinge sagt wie „Hilf mir, diesen Code zu verstehen“, „Was macht dieses Repo“, „Führe mich durch diese Codebasis“, „Entdecke dieses Projekt“, „Ich habe diesen Code geerbt“, „Mich in dieses System integrieren“ oder nach der Architektur, Struktur oder dem Zustand einer vorhandenen Codebasis fragt. Wird auch dann ausgelöst, wenn sie einfach einen Repo-Pfad löschen und sagen: „Was ist das?“. Diese Fähigkeit erzeugt ein strukturiertes Verständnis – keine Zusammenfassungen, sondern navigierbare Karten darüber, wie Code tatsächlich funktioniert.
