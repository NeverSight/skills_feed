Diseñe e implemente arquitecturas LLM de múltiples agentes utilizando el patrón orquestador-subagente. Úselo cuando: (1) Decidir si usar sistemas de múltiples agentes o de un solo agente, (2) Implementar aislamiento de contexto para operaciones de gran volumen, (3) Paralelizar tareas de investigación independientes, (4) Crear agentes especializados con conjuntos de herramientas enfocados, (5) Crear subagentes de verificación para garantizar la calidad, o (6) Analizar límites de descomposición centrados en el contexto.
