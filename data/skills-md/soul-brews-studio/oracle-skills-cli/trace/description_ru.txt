Найдите проекты в истории git, репозиториях, документации и Oracle. Используйте, когда пользователь спрашивает «отследить», «найти проект», «где находится [проект]», «история поиска», «копать», «сеансы», «прошлые сеансы». Поддерживает --oracle (быстрый), --smart (по умолчанию), --deep (5 субагентов), --dig (сессионный золотодобытчик).
