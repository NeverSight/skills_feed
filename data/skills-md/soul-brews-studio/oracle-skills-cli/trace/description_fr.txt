Recherchez des projets dans l'historique Git, les dépôts, les documents et Oracle. À utiliser lorsque l'utilisateur demande « trace », « rechercher un projet », « où se trouve [projet] », « historique de recherche », « creuser », « sessions », « sessions passées ». Prend en charge --oracle (rapide), --smart (par défaut), --deep (5 sous-agents), --dig (session goldminer).
