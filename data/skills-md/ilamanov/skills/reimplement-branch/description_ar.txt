أعد تنفيذ الفرع الحالي على فرع جديد باستخدام سجل التزام git نظيف وذو جودة سردية. يُستخدم عندما يريد المستخدم تنظيف الالتزامات الفوضوية، أو إنشاء سجل التزام بنمط البرنامج التعليمي، أو إعداد فرع للمراجعة باستخدام التزامات منطقية قائمة بذاتها. تشغيل طلبات مثل "تنظيف التزاماتي" أو "إعادة تنفيذ هذا الفرع" أو "إنشاء سجل نظيف" أو "جعل التزاماتي قابلة للمراجعة".
