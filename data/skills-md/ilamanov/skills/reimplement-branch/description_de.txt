Implementieren Sie den aktuellen Branch in einem neuen Branch mit einem sauberen Git-Commit-Verlauf in narrativer Qualität. Verwenden Sie diese Option, wenn der Benutzer chaotische Commits bereinigen, einen Commit-Verlauf im Tutorial-Stil erstellen oder einen Zweig mit logischen, eigenständigen Commits für die Überprüfung vorbereiten möchte. Wird bei Anfragen wie „meine Commits bereinigen“, „diesen Zweig neu implementieren“, „einen sauberen Verlauf erstellen“ oder „meine Commits überprüfbar machen“ ausgelöst.
