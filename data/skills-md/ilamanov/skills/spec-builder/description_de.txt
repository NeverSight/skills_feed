Verwandeln Sie vage Produkt- oder Feature-Ideen durch einen interaktiven Interviewprozess in konkrete, detaillierte Spezifikationsdokumente. Verwenden Sie diese Option, wenn der Benutzer eine Idee konkretisieren, eine Spezifikation erstellen, Anforderungen schreiben, ein Produkt/eine Funktion/einen Prototyp planen oder von „Ich habe diese Idee...“ zu einem konkreten Dokument übergehen möchte. Funktioniert für Softwareprodukte, physische Produkte, Dienstleistungen oder jedes Konzept, das einer Spezifikation bedarf.
