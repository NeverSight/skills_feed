Leitet den Evaluation-Driven-Development-Prozess (EDD) für Agentenkompetenzen. Verwenden Sie diese Option, wenn Sie Arbeitsabläufe zum Testen von Fertigkeiten einrichten, Szenarien zur Fertigkeitsbewertung erstellen oder Claude A/B-Feedbackschleifen zur Fertigkeitsvalidierung einrichten. Bietet Entwicklungsmethodik, keine inhaltliche Anleitung.
