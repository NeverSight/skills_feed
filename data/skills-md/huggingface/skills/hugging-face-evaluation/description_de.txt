Fügen Sie Bewertungsergebnisse in Hugging Face-Modellkarten hinzu und verwalten Sie sie. Unterstützt das Extrahieren von Bewertungstabellen aus README-Inhalten, das Importieren von Bewertungen aus der Artificial Analysis API und das Ausführen benutzerdefinierter Modellbewertungen mit vLLM/lighteval. Funktioniert mit dem Modellindex-Metadatenformat.
