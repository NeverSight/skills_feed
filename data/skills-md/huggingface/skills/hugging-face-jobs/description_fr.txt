Cette compétence doit être utilisée lorsque les utilisateurs souhaitent exécuter une charge de travail sur l'infrastructure Hugging Face Jobs. Couvre les scripts UV, les tâches basées sur Docker, la sélection du matériel, l'estimation des coûts, l'authentification avec des jetons, la gestion des secrets, la configuration des délais d'attente et la persistance des résultats. Conçu pour les charges de travail de calcul à usage général, notamment le traitement des données, l'inférence, les expériences, les tâches par lots et toutes les tâches basées sur Python. Doit être invoqué pour des tâches impliquant du calcul cloud, des charges de travail GPU ou lorsque les utilisateurs mentionnent l'exécution de tâches sur l'infrastructure Hugging Face sans configuration locale.
