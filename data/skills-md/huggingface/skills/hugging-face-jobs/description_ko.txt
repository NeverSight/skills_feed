이 기술은 사용자가 Hugging Face Jobs 인프라에서 워크로드를 실행하려고 할 때 사용해야 합니다. UV 스크립트, Docker 기반 작업, 하드웨어 선택, 비용 추정, 토큰을 사용한 인증, 비밀 관리, 시간 초과 구성 및 결과 지속성을 다룹니다. 데이터 처리, 추론, 실험, 배치 작업 및 모든 Python 기반 작업을 포함한 범용 컴퓨팅 워크로드용으로 설계되었습니다. 클라우드 컴퓨팅, GPU 워크로드와 관련된 작업 또는 사용자가 로컬 설정 없이 Hugging Face 인프라에서 작업 실행을 언급하는 경우 호출되어야 합니다.
