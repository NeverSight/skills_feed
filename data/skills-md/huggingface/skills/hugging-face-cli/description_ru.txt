Выполняйте операции Hugging Face Hub с помощью интерфейса командной строки hf. Используйте, когда пользователю необходимо загружать модели/наборы данных/пространства, загружать файлы в репозитории Hub, создавать репозитории, управлять локальным кешем или запускать вычислительные задания в инфраструктуре HF. Охватывает аутентификацию, передачу файлов, создание репозитория, операции с кэшем и облачные вычисления.
