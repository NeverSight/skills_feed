Ejecute operaciones de Hugging Face Hub utilizando la CLI `hf`. Úselo cuando el usuario necesite descargar modelos/conjuntos de datos/espacios, cargar archivos a repositorios de Hub, crear repositorios, administrar caché local o ejecutar trabajos informáticos en infraestructura HF. Cubre autenticación, transferencias de archivos, creación de repositorios, operaciones de caché y computación en la nube.
