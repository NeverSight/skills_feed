قم بتنفيذ عمليات Hugging Face Hub باستخدام `hf` CLI. يُستخدم عندما يحتاج المستخدم إلى تنزيل النماذج/مجموعات البيانات/المساحات، أو تحميل الملفات إلى مستودعات Hub، أو إنشاء مستودعات إعادة الشراء، أو إدارة ذاكرة التخزين المؤقت المحلية، أو تشغيل مهام الحوسبة على البنية التحتية لـ HF. يغطي المصادقة ونقل الملفات وإنشاء المستودع وعمليات ذاكرة التخزين المؤقت والحوسبة السحابية.
