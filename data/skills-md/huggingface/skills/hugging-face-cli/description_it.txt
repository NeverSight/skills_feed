Esegui le operazioni di Hugging Face Hub utilizzando la CLI `hf`. Da utilizzare quando l'utente deve scaricare modelli/set di dati/spazi, caricare file nei repository Hub, creare repository, gestire la cache locale o eseguire processi di calcolo sull'infrastruttura HF. Copre l'autenticazione, i trasferimenti di file, la creazione di repository, le operazioni di cache e il cloud computing.
