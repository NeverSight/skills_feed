Débogueur interactif pour les applications Deno/TypeScript utilisant le protocole V8 Inspector. Cette compétence doit être utilisée lors de l'enquête sur les problèmes dans les applications Deno, notamment les fuites de mémoire, les goulots d'étranglement des performances, les conditions de concurrence, les plantages ou tout comportement d'exécution nécessitant un débogage étape par étape, une analyse du tas ou un profilage du processeur. Fournit des outils clients CDP, des analyseurs de tas/CPU et un suivi des enquêtes.
