دليل لكتابة كود Rust الاصطلاحي استنادًا إلى دليل أفضل ممارسات Apollo GraphQL. استخدم هذه المهارة عندما: (1) كتابة كود أو وظائف Rust جديدة، (2) مراجعة كود Rust الحالي أو إعادة هيكلته، (3) الاختيار بين أنماط الاقتراض أو الاستنساخ أو الملكية، (4) تنفيذ معالجة الأخطاء مع أنواع النتائج، (5) تحسين كود Rust للأداء، (6) كتابة الاختبارات أو الوثائق لمشاريع Rust.
