Guía para escribir código idiomático de Rust basado en el manual de mejores prácticas de Apollo GraphQL. Utilice esta habilidad cuando: (1) escriba nuevo código o funciones de Rust, (2) revise o refactorice el código de Rust existente, (3) decida entre patrones de préstamo o clonación o propiedad, (4) implemente el manejo de errores con tipos de resultados, (5) optimice el código de Rust para el rendimiento, (6) escriba pruebas o documentación para proyectos de Rust.
