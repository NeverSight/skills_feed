Guide pour écrire du code Rust idiomatique basé sur le manuel des meilleures pratiques d'Apollo GraphQL. Utilisez cette compétence lorsque : (1) vous écrivez du nouveau code ou de nouvelles fonctions Rust, (2) révisez ou refactorisez le code Rust existant, (3) décidez entre l'emprunt, le clonage ou les modèles de propriété, (4) implémentez la gestion des erreurs avec les types de résultats, (5) optimisez le code Rust pour les performances, (6) rédigez des tests ou de la documentation pour les projets Rust.
