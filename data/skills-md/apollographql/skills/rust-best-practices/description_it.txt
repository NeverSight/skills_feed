Guida per scrivere codice Rust idiomatico basata sul manuale delle migliori pratiche di Apollo GraphQL. Usa questa abilità quando: (1) scrivi nuovo codice o funzioni Rust, (2) rivedi o refactoring il codice Rust esistente, (3) decidi tra prendere in prestito o clonare o modelli di proprietà, (4) implementare la gestione degli errori con i tipi di risultato, (5) ottimizzare il codice Rust per le prestazioni, (6) scrivere test o documentazione per progetti Rust.
