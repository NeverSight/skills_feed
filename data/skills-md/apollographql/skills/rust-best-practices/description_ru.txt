Руководство по написанию идиоматического кода Rust, основанное на справочнике лучших практик Apollo GraphQL. Используйте этот навык, когда: (1) пишете новый код или функции Rust, (2) просматриваете или рефакторите существующий код Rust, (3) выбираете между шаблонами заимствования и клонирования или владения, (4) реализуете обработку ошибок с помощью типов результатов, (5) оптимизируете код Rust для повышения производительности, (6) пишете тесты или документацию для проектов Rust.
