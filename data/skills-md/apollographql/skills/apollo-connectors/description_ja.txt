ユーザーが @source および @connect ディレクティブと選択マッピングを使用して REST API を GraphQL スーパーグラフに統合するための Apollo コネクタ スキーマを作成できるようにします。このスキルは、ユーザーが次の場合に使用します: (1) 「コネクタ」、「Apollo コネクタ」、または「REST コネクタ」について言及する場合、(2) REST API を GraphQL に統合したい場合、(3) @source または @connect ディレクティブを参照する場合、(4) 「# AI 友人への注意: これは Apollo コネクタ スキーマです」を含むファイルを扱う場合。
