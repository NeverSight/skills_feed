Guida per la progettazione di schemi GraphQL seguendo le migliori pratiche del settore. Utilizzare questa competenza quando: (1) si progetta un nuovo schema o API GraphQL, (2) si esamina lo schema esistente per miglioramenti, (3) si decide sulle strutture dei tipi o sull'annullamento dei valori nulli, (4) si implementano modelli di impaginazione o di errore, (5) si garantisce la sicurezza nella progettazione dello schema.
