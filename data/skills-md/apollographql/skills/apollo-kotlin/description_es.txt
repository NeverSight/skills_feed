Guía para crear aplicaciones con Apollo Kotlin, la biblioteca cliente GraphQL para Android y Kotlin. Utilice esta habilidad cuando: (1) configure Apollo Kotlin en un proyecto Gradle para Android, Kotlin/JVM o KMP, (2) configure la descarga de esquemas y codegen para servicios GraphQL, (3) configure un `ApolloClient` con autenticación, interceptores y almacenamiento en caché, (4) escriba consultas, mutaciones o suscripciones,
