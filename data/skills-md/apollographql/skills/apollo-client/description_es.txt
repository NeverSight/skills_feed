Guía para construir aplicaciones React con Apollo Client 4.x. Utilice esta habilidad cuando: (1) configure el Cliente Apollo en un proyecto de React, (2) escriba consultas GraphQL o mutaciones con ganchos, (3) configure el almacenamiento en caché o políticas de caché, (4) administre el estado local con variables reactivas, (5) solucione errores o problemas de rendimiento del Cliente Apollo.
