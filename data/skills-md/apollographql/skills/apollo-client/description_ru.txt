Руководство по созданию приложений React с помощью Apollo Client 4.x. Используйте этот навык при: (1) настройке клиента Apollo в проекте React, (2) написании запросов или мутаций GraphQL с помощью перехватчиков, (3) настройке кэширования или политик кэширования, (4) управлении локальным состоянием с помощью реактивных переменных, (5) устранении ошибок клиента Apollo или проблем с производительностью.
