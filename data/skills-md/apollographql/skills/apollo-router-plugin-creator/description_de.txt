Anleitung zum Schreiben nativer Rust-Plugins für Apollo Router. Verwenden Sie diese Fähigkeit, wenn: (1) Benutzer ein neues Router-Plugin erstellen möchten, (2) Benutzer Service-Hooks hinzufügen möchten (router_service, supergraph_service, Execution_service, subgraph_service), (3) Benutzer ein vorhandenes Router-Plugin ändern möchten, (4) Benutzer Router-Plugin-Muster oder den Anforderungslebenszyklus verstehen müssen. (5) löst bei Anfragen wie „Neues Plugin erstellen“, „Router-Plugin hinzufügen“, „X-Plugin ändern“ oder „subgraph_service-Hook hinzufügen“ aus.
