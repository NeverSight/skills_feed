Руководство по написанию собственных плагинов Rust для Apollo Router. Используйте этот навык, когда: (1) пользователи хотят создать новый плагин маршрутизатора, (2) пользователи хотят добавить сервисные перехватчики (router_service, supergraph_service, Execution_service, subgraph_service), (3) пользователи хотят изменить существующий плагин маршрутизатора, (4) пользователям необходимо понимать шаблоны плагинов маршрутизатора или жизненный цикл запроса. (5) срабатывает по таким запросам, как «создать новый плагин», «добавить плагин маршрутизатора», «изменить плагин X» или «добавить перехватчик subgraph_service».
