Guía para crear habilidades efectivas para Apollo GraphQL y el desarrollo de GraphQL. Utilice esta habilidad cuando: (1) los usuarios quieran crear una nueva habilidad, (2) los usuarios quieran actualizar una habilidad existente, (3) los usuarios pregunten sobre la estructura de habilidades o las mejores prácticas, (4) los usuarios necesiten ayuda para escribir archivos SKILL.md.
