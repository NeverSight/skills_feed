دليل لاستخدام Apollo MCP Server لتوصيل وكلاء الذكاء الاصطناعي بواجهات برمجة تطبيقات GraphQL. استخدم هذه المهارة عندما: (1) إعداد أو تكوين Apollo MCP Server، (2) تعريف أدوات MCP من عمليات GraphQL، (3) استخدام أدوات الاستبطان (الاستبطان، البحث، التحقق من الصحة، التنفيذ)، (4) استكشاف أخطاء اتصال خادم MCP أو مشكلات تنفيذ الأداة وإصلاحها.
