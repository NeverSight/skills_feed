Руководство по использованию сервера Apollo MCP для подключения агентов искусственного интеллекта к API GraphQL. Используйте этот навык при: (1) установке или настройке сервера Apollo MCP, (2) определении инструментов MCP на основе операций GraphQL, (3) использовании инструментов самоанализа (самоанализ, поиск, проверка, выполнение), (4) устранении неполадок подключения к серверу MCP или проблем с выполнением инструментов.
