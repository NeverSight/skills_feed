Guide d'utilisation d'Apollo MCP Server pour connecter des agents IA aux API GraphQL. Utilisez cette compétence lors de : (1) l'installation ou la configuration du serveur Apollo MCP, (2) la définition d'outils MCP à partir d'opérations GraphQL, (3) l'utilisation d'outils d'introspection (introspecter, rechercher, valider, exécuter), (4) le dépannage de problèmes de connectivité du serveur MCP ou d'exécution d'outils.
