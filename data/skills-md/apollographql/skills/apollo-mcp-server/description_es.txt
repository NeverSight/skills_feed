Guía para usar Apollo MCP Server para conectar agentes de IA con API GraphQL. Utilice esta habilidad cuando: (1) instale o configure el servidor Apollo MCP, (2) defina herramientas MCP a partir de operaciones GraphQL, (3) use herramientas de introspección (introspección, búsqueda, validación, ejecución), (4) solucione problemas de conectividad del servidor MCP o problemas de ejecución de herramientas.
