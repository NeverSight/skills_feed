Guía compatible con la versión para configurar y ejecutar Apollo Router para supergrafos GraphQL federados. Genera YAML correcto tanto para el enrutador v1.x como para el v2.x. Utilice esta habilidad cuando: (1) configure el enrutador Apollo para ejecutar un supergrafo, (2) configure enrutamiento, encabezados o CORS, (3) implemente complementos personalizados (scripts Rhai o coprocesadores), (4) configure telemetría (rastreo, métricas, registro), (5) solucione problemas de rendimiento o conectividad del enrutador.
