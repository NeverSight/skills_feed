Umfassendes Fachwissen für die Arbeit mit dem GenAIScript-Framework von Microsoft – einem JavaScript/TypeScript-basierten System zur Erstellung automatisierbarer LLM-Eingabeaufforderungen und KI-Workflows. Verwenden Sie es beim Erstellen, Debuggen oder Optimieren von GenAIScript-Skripten, beim Implementieren von Eingabeaufforderungen als Code, beim Arbeiten mit Tools und Agents, beim Verarbeiten von Dateien (PDF, CSV, DOCX), beim Definieren von Schemata oder beim Erstellen von KI-Automatisierungsworkflows.
