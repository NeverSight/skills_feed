Competenza completa per lavorare con il framework GenAIScript di Microsoft: un sistema basato su JavaScript/TypeScript per la creazione di prompt LLM automatizzabili e flussi di lavoro AI. Da utilizzare durante la creazione, il debug o l'ottimizzazione degli script GenAIScript, l'implementazione di prompt-as-code, l'utilizzo di strumenti e agenti, l'elaborazione di file (PDF, CSV, DOCX), la definizione di schemi o la creazione di flussi di lavoro di automazione AI.
