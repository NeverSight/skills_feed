خبرة شاملة للعمل مع إطار عمل GenAIScript من Microsoft - وهو نظام يستند إلى JavaScript/TypeScript لإنشاء مطالبات LLM قابلة للأتمتة وسير عمل الذكاء الاصطناعي. يُستخدم عند إنشاء البرامج النصية GenAIScript أو تصحيح الأخطاء أو تحسينها، أو تنفيذ المطالبات كرمز، أو العمل مع الأدوات والوكلاء، أو معالجة الملفات (PDF، CSV، DOCX)، أو تحديد المخططات، أو إنشاء سير عمل أتمتة الذكاء الاصطناعي.
