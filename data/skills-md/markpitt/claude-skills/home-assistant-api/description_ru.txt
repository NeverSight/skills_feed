Организует доступ к REST API Home Assistant для программного управления устройствами умного дома. Направляет запросы к специализированным файлам ресурсов в зависимости от типа задачи — аутентификация, управление состоянием, вызовы служб, типы сущностей или расширенные запросы. Предоставляет интеллектуальные таблицы решений для выбора подходящих конечных точек и управления интеграцией.
