Orchestriert den Zugriff auf die Home Assistant-REST-API zur programmgesteuerten Steuerung von Smart-Home-Geräten. Leitet Anfragen basierend auf dem Aufgabentyp – Authentifizierung, Statusverwaltung, Serviceaufrufe, Entitätstypen oder erweiterte Abfragen – an spezielle Ressourcendateien weiter. Bietet intelligente Entscheidungstabellen zur Auswahl geeigneter Endpunkte und zur Verwaltung von Integrationen.
