Achemine les requêtes vers des fichiers de ressources spécialisés en fonction du type de tâche : authentification, gestion de l'état, appels de service, types d'entités ou requêtes avancées. Fournit des tableaux de décision intelligents pour sélectionner les points de terminaison appropriés et gérer les intégrations.
