Orchestra l'accesso all'API REST Home Assistant per il controllo programmatico dei dispositivi domestici intelligenti. Instrada le richieste a file di risorse specializzati in base al tipo di attività: autenticazione, gestione dello stato, chiamate di servizio, tipi di entità o query avanzate. Fornisce tabelle decisionali intelligenti per la selezione degli endpoint appropriati e la gestione delle integrazioni.
