ينسق الوصول إلى Home Assistant REST API للتحكم البرمجي في الأجهزة المنزلية الذكية. توجيه الطلبات إلى ملفات الموارد المتخصصة بناءً على نوع المهمة - المصادقة، أو إدارة الحالة، أو مكالمات الخدمة، أو أنواع الكيانات، أو الاستعلامات المتقدمة. يوفر جداول قرارات ذكية لاختيار نقاط النهاية المناسبة وإدارة عمليات التكامل.
