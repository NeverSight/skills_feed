Организует комплексные модели когнитивного мышления для решения сложных проблем. Анализирует задачи для выбора оптимальных паттернов из фундаментальных, рассуждений, творческих, метакогнитивных, специализированных и нейроотличных категорий. При необходимости объединяет несколько шаблонов и проверяет выходные данные перед ответом.
