Compétences complètes pour travailler avec l'API REST Azure DevOps sur tous les services, y compris les tableaux (éléments de travail, requêtes, backlogs), les dépôts (Git, demandes d'extraction, validations), les pipelines (builds, versions, déploiements), les plans de test, les artefacts, les organisations, les projets, la sécurité, les extensions, etc. À utiliser lors de la mise en œuvre d’intégrations Azure DevOps, de l’automatisation des flux de travail DevOps ou de la création d’applications qui interagissent avec les services Azure DevOps.
