مهارة شاملة للعمل مع Azure DevOps REST API عبر جميع الخدمات بما في ذلك اللوحات (عناصر العمل والاستعلامات والتراكمات) وRepos (Git وطلبات السحب والالتزامات) وخطوط الأنابيب (عمليات الإنشاء والإصدارات وعمليات النشر) وخطط الاختبار والعناصر والمؤسسات والمشاريع والأمان والإضافات والمزيد. يُستخدم عند تنفيذ عمليات تكامل Azure DevOps، أو أتمتة سير عمل DevOps، أو إنشاء تطبيقات تتفاعل مع خدمات Azure DevOps.
