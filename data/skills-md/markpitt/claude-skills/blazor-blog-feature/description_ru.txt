Добавляет полную функцию блога в существующее статическое веб-приложение Blazor WebAssembly с серверной частью функций Azure и общим файловым ресурсом Azure для хранилища с уценкой. Используйте при реализации функций блога в проектах .NET Blazor WASM с инфраструктурой Azure. Включает список публикаций, страницы с подробными сведениями, рендеринг уценки, интеграцию с хранилищем Azure.
