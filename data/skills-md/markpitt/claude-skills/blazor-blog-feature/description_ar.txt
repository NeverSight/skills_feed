يضيف ميزة مدونة كاملة إلى تطبيق الويب الثابت Blazor WebAssembly Static Web App مع الواجهة الخلفية لـ Azure Functions وAzure File Share لتخزين تخفيض السعر. يُستخدم عند تنفيذ وظائف المدونة في مشاريع .NET Blazor WASM مع البنية الأساسية لـ Azure. يتضمن قائمة المنشورات وصفحات التفاصيل وعرض تخفيض السعر وتكامل Azure Storage.
