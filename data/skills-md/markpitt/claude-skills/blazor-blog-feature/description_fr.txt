Ajoute une fonctionnalité de blog complète à une application Web statique Blazor WebAssembly existante avec le backend Azure Functions et Azure File Share pour le stockage markdown. À utiliser lors de l'implémentation de la fonctionnalité de blog dans les projets .NET Blazor WASM avec l'infrastructure Azure. Comprend la liste des articles, les pages de détails, le rendu markdown et l'intégration du stockage Azure.
