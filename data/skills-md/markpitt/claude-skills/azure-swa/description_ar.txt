خبرة شاملة لتطبيقات الويب Azure Static بما في ذلك البنية والتكوين وتكامل واجهة برمجة التطبيقات مع وظائف Azure والمصادقة والتوجيه والنشر وCI/CD. يُستخدم عند إنشاء مشاريع Azure Static Web Apps أو تكوينها أو نشرها أو استكشاف أخطاءها وإصلاحها باستخدام أطر عمل مثل React أو Angular أو Vue أو Blazor أو Vanilla JavaScript.
