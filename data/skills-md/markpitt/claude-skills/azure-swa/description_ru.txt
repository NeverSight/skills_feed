Комплексный опыт работы со статическими веб-приложениями Azure, включая архитектуру, настройку, интеграцию API с функциями Azure, аутентификацию, маршрутизацию, развертывание и CI/CD. Используйте при создании, настройке, развертывании или устранении неполадок проектов статических веб-приложений Azure с такими платформами, как React, Angular, Vue, Blazor или ванильный JavaScript.
