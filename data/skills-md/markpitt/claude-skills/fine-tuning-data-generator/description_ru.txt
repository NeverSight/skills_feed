Создает комплексные синтетические наборы данных для точной настройки в формате ChatML (JSONL) для использования с Unsloth, Axolotl и аналогичными платформами обучения. Собирает требования, создает наборы данных с разнообразными примерами, проверяет качество и предоставляет рекомендации по интеграции инфраструктуры.
