الاستعلام عن قاعدة بيانات Apple Health SQLite للعناصر الحيوية والنشاط والنوم والتدريبات. يدعم تنسيقات إخراج Markdown وJSON وFHIR R4. يجب استخدام هذه المهارة عند تحليل المقاييس الصحية، أو إنشاء التقارير الصحية، أو الإجابة على الأسئلة المتعلقة باللياقة البدنية أو أنماط النوم، أو تصدير البيانات الصحية بتنسيقات قياسية.
