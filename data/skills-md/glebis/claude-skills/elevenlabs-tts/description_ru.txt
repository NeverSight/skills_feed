Этот навык преобразует текст в высококачественные аудиофайлы с помощью API ElevenLabs. Используйте этот навык, когда пользователи запрашивают преобразование текста в речь, звуковое сопровождение или синтез голоса с настраиваемыми голосовыми параметрами (стабильность, повышение сходства) и голосовыми пресетами (Рэйчел, Адам, Белла, Элли, Джош, Арнольд, Ава).
