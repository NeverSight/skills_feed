Esta habilidad analiza transcripciones de reuniones para extraer decisiones, elementos de acción, opiniones, preguntas y terminología utilizando Cerebras AI (llama-3.3-70b). Utilice esta habilidad cuando el usuario solicite analizar una transcripción, extraer elementos de acción de reuniones, encontrar decisiones en conversaciones, crear glosarios a partir de discusiones o resumir puntos clave de reuniones grabadas.
