Ищите и загружайте изображения через API пользовательского поиска Google с выбором на основе LLM. Этот навык следует использовать при поиске изображений для статей, презентаций, исследовательских документов или при дополнении заметок Obsidian соответствующими визуальными эффектами. Поддерживает простые запросы, пакетную обработку из конфигурации JSON, автоматическое создание конфигурации из терминов и полное обогащение примечаний с автоматической вставкой изображений под заголовками.
