Публикуйте файлы или заметки Obsidian как GitHub Gists. Используйте, когда пользователь хочет публично поделиться кодом/заметками, создать фрагменты кода, которыми можно быстро поделиться, или опубликовать уценку на GitHub. Триггеры включают «опубликовать как суть», «создать суть», «поделиться на github», «извлечь суть из этого».
