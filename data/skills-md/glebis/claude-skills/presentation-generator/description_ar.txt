قم بإنشاء عروض تقديمية بتنسيق HTML تفاعلية مع تصميم وحشي جديد وزخارف فنية ASCII وألوان العلامة التجارية للوكالة. يقوم بإخراج HTML (تفاعلي مع التنقل)، وPNG (شرائح فردية عبر Playwright)، وPDF. مراجع مهارة وكالة العلامة التجارية للألوان والطباعة. يُستخدم عند إنشاء العروض التقديمية أو مجموعات الشرائح أو مواد العرض التقديمي أو الملخصات المرئية.
