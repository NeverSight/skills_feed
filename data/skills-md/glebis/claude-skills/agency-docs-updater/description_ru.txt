Сквозной конвейер для публикации совещаний лаборатории Claude Code. Автоматически находит/создает стенограмму Fathom, загружает видео, загружает на YouTube, генерирует проверенное фактами резюме на русском языке, создает документацию MDX и отправляет документы агентства для развертывания Vercel. Одиночный вызов заменяет более 5 шагов вручную.
