Совершайте телефонные звонки с помощью естественных голосов искусственного интеллекта (ElevenLabs) и отправляйте SMS с помощью Twilio CLI. Используйте этот навык, когда пользователь хочет позвонить, отправить текстовое сообщение или использовать для звонков голос, сгенерированный искусственным интеллектом. Требуется проверка подлинности Twilio CLI и ключ API ElevenLabs.
