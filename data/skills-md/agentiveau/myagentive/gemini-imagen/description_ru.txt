Создавайте изображения из текстовых подсказок с помощью API Gemini Imagen от Google. Этот навык следует использовать, когда пользователь запрашивает создание, генерацию или визуализацию изображения на основе текстовых описаний (например, «создать изображение...», «создать изображение, показывающее...», «сделать мне изображение для...»).
