Generieren Sie Bilder aus Textaufforderungen mit der Gemini Imagen API von Google. Diese Fähigkeit sollte verwendet werden, wenn der Benutzer die Erstellung, Generierung oder Visualisierung von Bildern aus Textbeschreibungen anfordert (z. B. „Erstellen Sie ein Bild von …“, „Erzeugen Sie ein Bild, das ... zeigt“, „Erstellen Sie mir ein Bild für ...“).
