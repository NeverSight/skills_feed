Этот навык позволяет управлять устройствами Android через ADB и API специальных возможностей. Используйте этот навык, когда пользователи хотят автоматизировать задачи телефона Android, удаленно управлять приложениями, нажимать кнопки, вводить текст, перемещаться по пользовательскому интерфейсу или выполнять какие-либо действия на подключенном устройстве Android. Требуется ADB и подключенное устройство с включенной отладкой по USB.
