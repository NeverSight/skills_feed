Esta habilidad permite el control de dispositivos Android a través de ADB y la API de accesibilidad. Utilice esta habilidad cuando los usuarios quieran automatizar tareas del teléfono Android, controlar aplicaciones de forma remota, tocar botones, escribir texto, navegar por la interfaz de usuario o realizar cualquier acción en un dispositivo Android conectado. Requiere ADB y un dispositivo conectado con la depuración USB habilitada.
