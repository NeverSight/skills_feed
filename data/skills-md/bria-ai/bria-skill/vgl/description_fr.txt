Écrivez des invites JSON VGL (Visual Generation Language) structurées pour les modèles de génération d'images FIBO de Bria. Utilisez cette compétence lors de la création de descriptions d'images détaillées au format JSON pour la génération de texte en image, l'édition d'images, l'inpainting, l'outpainting, la génération d'arrière-plan ou le sous-titrage. Les déclencheurs incluent des demandes d'écriture d'invites structurées, de création de VGL JSON, de description d'images pour la génération d'IA ou de travail avec le format structured_prompt de Bria/FIBO. À utiliser également lors de la conversion de requêtes d'images en langage naturel en schéma JSON déterministe requis par les modèles FIBO.
