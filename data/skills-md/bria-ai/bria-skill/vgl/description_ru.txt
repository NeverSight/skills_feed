Напишите структурированный VGL (язык визуальной генерации) JSON, запрашивающий модели генерации изображений FIBO от Bria. Используйте этот навык при создании подробных описаний изображений в формате JSON для преобразования текста в изображение, редактирования изображений, закрашивания, закрашивания, создания фона или надписей. Триггеры включают запросы на написание структурированных подсказок, создание VGL JSON, описание изображений для генерации ИИ или работу с форматом структурированных_подсказок Bria/FIBO. Также используйте при преобразовании запросов изображений на естественном языке в детерминированную схему JSON, необходимую для моделей FIBO.
