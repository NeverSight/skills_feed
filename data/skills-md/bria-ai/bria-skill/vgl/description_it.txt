Scrivi prompt JSON VGL (Visual Generation Language) strutturati per i modelli di generazione di immagini FIBO di Bria. Utilizza questa competenza quando crei descrizioni dettagliate di immagini in formato JSON per la generazione di testo in immagine, la modifica delle immagini, l'inpainting, l'outpainting, la generazione di sfondo o i sottotitoli. I trigger includono richieste di scrittura di prompt strutturati, creazione di VGL JSON, descrizione di immagini per la generazione di intelligenza artificiale o utilizzo del formato Structured_Prompt di Bria/FIBO. Da utilizzare anche per convertire le richieste di immagini in linguaggio naturale nello schema JSON deterministico richiesto dai modelli FIBO.
