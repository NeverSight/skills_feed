كتابة مطالبات VGL (لغة الجيل المرئي) JSON المنظمة لنماذج إنشاء الصور FIBO الخاصة بـ Bria. استخدم هذه المهارة عند إنشاء أوصاف صور تفصيلية بتنسيق JSON لإنشاء نص إلى صورة، أو تحرير الصور، أو الرسم الداخلي، أو الرسم الخارجي، أو إنشاء الخلفية، أو التعليق التوضيحي. تتضمن المشغلات طلبات لكتابة مطالبات منظمة، أو إنشاء VGL JSON، أو وصف صور لإنشاء الذكاء الاصطناعي، أو العمل مع تنسيق Bria/FIBO's Structured_prompt. يُستخدم أيضًا عند تحويل طلبات صور اللغة الطبيعية إلى مخطط JSON الحتمي الذي تتطلبه نماذج FIBO.
