Bria の FIBO 画像生成モデル用に構造化された VGL (Visual Generation Language) JSON プロンプトを作成します。このスキルは、テキストから画像の生成、画像の編集、インペイント、アウトペイント、背景の生成、またはキャプションを目的として JSON 形式で詳細な画像の説明を作成するときに使用します。トリガーには、構造化プロンプトの作成、VGL JSON の作成、AI 生成用の画像の記述、または Bria/FIBO の Structured_prompt 形式での作業などのリクエストが含まれます。自然言語画像リクエストを FIBO モデルで必要な決定論的な JSON スキーマに変換する場合にも使用します。
