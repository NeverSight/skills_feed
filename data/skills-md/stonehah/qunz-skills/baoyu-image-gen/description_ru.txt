Генерация изображений ИИ с помощью API OpenAI, Google, DashScope, Replication и APIMart. Поддерживает преобразование текста в изображение, эталонные изображения, соотношения сторон. Последовательный по умолчанию; параллельная генерация доступна по запросу. Используйте, когда пользователь просит сгенерировать, создать или нарисовать изображения.
