Azure Durable Functions と .NET 分離ワーカーおよび Durable Task Scheduler バックエンドを使用して、耐久性とフォールト トレラントなワークフローを構築します。サーバーレス オーケストレーション、アクティビティ、エンティティを作成する場合、または関数チェーン、ファンアウト/ファンイン、非同期 HTTP API、人間による対話、モニタリング、ステートフル アグリゲーターなどのパターンを実装する場合に使用します。永続的な実行、状態の永続性、または組み込みの HTTP 管理 API と Azure 統合による分散調整を必要とする Azure Functions アプリに適用されます。
