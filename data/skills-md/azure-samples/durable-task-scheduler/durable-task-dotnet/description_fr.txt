Créez des workflows durables et tolérants aux pannes dans .NET à l’aide du SDK Durable Task avec Azure Durable Task Scheduler. À utiliser lors de la création d'orchestrations, d'activités, d'entités ou de la mise en œuvre de modèles tels que le chaînage de fonctions, la distribution/l'entrée, l'interaction humaine ou les agents avec état. S’applique à toute application .NET nécessitant une exécution durable, une persistance d’état ou des transactions distribuées sans dépendance à Azure Functions.
