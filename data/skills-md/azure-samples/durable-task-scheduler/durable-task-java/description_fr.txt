Créez des workflows durables et tolérants aux pannes en Java à l'aide du SDK Durable Task avec Azure Durable Task Scheduler. À utiliser lors de la création d'orchestrations, d'activités ou de la mise en œuvre de modèles tels que le chaînage de fonctions, la distribution/l'entrée, l'interaction humaine ou la surveillance. S'applique à toute application Java nécessitant une exécution durable, une persistance d'état ou des transactions distribuées sans dépendance à Azure Functions.
