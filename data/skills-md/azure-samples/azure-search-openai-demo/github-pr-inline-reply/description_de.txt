Antworten Sie auf Inline-PR-Rezensionskommentare zu GitHub-Pull-Requests mithilfe der GitHub-API. Verwenden Sie diese Fähigkeit, wenn Sie auf einzelne Bewertungskommentare zu einer PR antworten, Feedback bestätigen oder Kommentare als gelöst markieren müssen, indem Sie direkte Antworten in Kommentarthreads veröffentlichen.
