أكمل مرجع تطوير Frappe Framework لإنشاء تطبيقات ERPNext. يُستخدم عند إنشاء DocTypes، أو وحدات تحكم، أو خطافات، أو REST APIs، أو نصوص برمجية للنماذج، أو وظائف خلفية، أو أي تطوير Frappe/ERPNext مخصص. يحتوي على وثائق API كاملة للمستندات، وقواعد البيانات، ووحدات التحكم، والخطافات، وREST API، والبرامج النصية للنماذج، والاختبار، والأنماط الشائعة.
