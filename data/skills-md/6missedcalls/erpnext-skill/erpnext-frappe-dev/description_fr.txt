Référence complète de développement Frappe Framework pour la création d'applications ERPNext. À utiliser lors de la création de DocTypes personnalisés, de contrôleurs, de hooks, d'API REST, de scripts de formulaire, de tâches en arrière-plan ou de tout développement Frappe/ERPNext. Contient une documentation complète sur l'API pour les documents, la base de données, les contrôleurs, les hooks, l'API REST, les scripts de formulaire, les tests et les modèles courants.
