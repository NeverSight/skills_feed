Полное руководство по разработке Frappe Framework для создания приложений ERPNext. Используйте при создании пользовательских типов документов, контроллеров, перехватчиков, API REST, сценариев форм, фоновых заданий или любой разработки Frappe/ERPNext. Содержит полную документацию по API для документов, баз данных, контроллеров, перехватчиков, REST API, сценариев форм, тестирования и общих шаблонов.
