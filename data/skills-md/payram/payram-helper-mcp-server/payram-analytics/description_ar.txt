الاستعلام عن تحليلات PayRam عبر أدوات MCP لاسترداد المدفوعات ونمو المستخدم وملخصات المعاملات ومقاييس المشروع. يغطي الاكتشاف وجلب بيانات الرسم البياني ومسارات العمل الجاهزة للتشغيل للوحات المعلومات والتقارير. يُستخدم عند إنشاء لوحات معلومات تحليلية، أو إنشاء تقارير، أو مراقبة اتجاهات الدفع، أو أتمتة الرؤى من PayRam.
