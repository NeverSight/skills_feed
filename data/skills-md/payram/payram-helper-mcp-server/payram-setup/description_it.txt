Distribuisci e configura il tuo server gateway di pagamento crittografico self-hosted PayRam. Installa su VPS, imposta il database PostgreSQL, configura l'account root, i dettagli del nodo, i portafogli e gli hot wallet. Completa la distribuzione e l'onboarding in meno di 10 minuti. Nessuna registrazione richiesta: completamente ospitato in autonomia. Da utilizzare quando si distribuisce PayRam per la prima volta, si imposta l'infrastruttura del server o si configura il dashboard PayRam.
