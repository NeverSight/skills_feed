Обращайтесь к комментариям проверки запроса на извлечение GitHub с помощью gh CLI. Используйте, когда вас просят обработать отзывы по связям с общественностью, особенно смешанные отзывы людей и ботов с искусственным интеллектом (например, подсказки CodeRabbit), оценить достоверность, применить исправления и подготовить краткие ответы с доказательствами.
