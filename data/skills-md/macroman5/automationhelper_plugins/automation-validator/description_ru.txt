Проверяет JSON рабочего процесса автоматизации перед развертыванием для Power Automate, n8n, Make, Zapier и других платформ. Проверяет синтаксис, структуру, рекомендации и потенциальные проблемы. Анализирует файлы JSON рабочего процесса на предмет соответствия платформе, отсутствия обработки ошибок, проблем с производительностью и безопасности. Используйте, когда пользователь хочет проверить, просмотреть или проверить рабочий процесс перед развертыванием/импортом.
