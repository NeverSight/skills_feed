Générateur de workflow pour Power Automate, n8n, Make, Zapier et d'autres plateformes. Génère un flux de travail JSON complet et prêt pour la production à partir de plans ou d'exigences de mise en œuvre. Utilise le sous-agent Flow-Builder pour créer un JSON valide spécifique à la plate-forme avec tous les déclencheurs, actions, gestion des erreurs et configurations. Se déclenche lorsque l'utilisateur a un plan/des exigences et souhaite générer un workflow JSON, ou dit « créer ce workflow », « créer le flux », « générer JSON ». Sortie prête à être importée dans la plateforme cible.
