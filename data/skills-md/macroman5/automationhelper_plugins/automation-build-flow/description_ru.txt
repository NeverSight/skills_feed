Построитель рабочих процессов для Power Automate, n8n, Make, Zapier и других платформ. Генерирует полный, готовый к использованию рабочий процесс в формате JSON на основе планов внедрения или требований. Использует субагент построителя потоков для создания допустимого JSON для конкретной платформы со всеми триггерами, действиями, обработкой ошибок и конфигурациями. Срабатывает, когда у пользователя есть план/требования и он хочет сгенерировать рабочий процесс в формате JSON или говорит «постройте этот рабочий процесс», «создайте поток», «сгенерируйте JSON». Вывод готов к импорту в целевую платформу.
