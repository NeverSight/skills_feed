Generador de flujos de trabajo para Power Automate, n8n, Make, Zapier y otras plataformas. Genera JSON de flujo de trabajo completo y listo para producción a partir de planes o requisitos de implementación. Utiliza el subagente generador de flujo para crear JSON válido específico de la plataforma con todos los activadores, acciones, manejo de errores y configuraciones. Se activa cuando el usuario tiene un plan/requisitos y desea generar un flujo de trabajo JSON, o dice "construir este flujo de trabajo", "crear el flujo", "generar JSON". Salida lista para importar a la plataforma de destino.
