Power Automate、n8n、Make、Zapier およびその他のプラットフォーム用のワークフロー ビルダー。実装計画または要件から、実稼働対応の完全なワークフロー JSON を生成します。フロービルダー サブエージェントを使用して、すべてのトリガー、アクション、エラー処理、構成を含む有効なプラットフォーム固有の JSON を作成します。ユーザーに計画/要件があり、ワークフロー JSON を生成したい場合、または「このワークフローを構築する」、「フローを作成する」、「JSON を生成する」と指示した場合にトリガーされます。出力をターゲット プラットフォームにインポートする準備ができました。
