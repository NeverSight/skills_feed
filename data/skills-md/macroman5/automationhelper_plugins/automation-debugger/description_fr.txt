Débogueur d'erreurs de plate-forme d'automatisation experte pour Power Automate, n8n, Make, Zapier et d'autres plates-formes de flux de travail. Analyse les définitions de flux JSON avec des messages d'erreur, recherche la documentation officielle et génère un JSON fixe complet prêt à être copié-collé. Se déclenche lorsque l'utilisateur fournit des fichiers JSON d'erreur, un flux de travail JSON avec des erreurs, des messages d'erreur, des demandes de débogage ou un contenu d'automatisation défaillant. Renvoie un rapport de débogage structuré avec analyse des causes profondes et travail JSON fixe.
