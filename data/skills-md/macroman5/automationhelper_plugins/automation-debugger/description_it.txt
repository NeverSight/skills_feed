Debugger di errori della piattaforma di automazione esperta per Power Automate, n8n, Make, Zapier e altre piattaforme di flusso di lavoro. Analizza le definizioni del flusso JSON con messaggi di errore, ricerca la documentazione ufficiale e genera JSON fisso completo pronto per il copia-incolla. Si attiva quando l'utente fornisce file JSON di errore, flusso di lavoro JSON con errori, messaggi di errore, richieste di debug o contenuto di automazione non riuscito. Restituisce un report di debug strutturato con analisi della causa principale e JSON funzionante e corretto.
