Экспертный отладчик ошибок платформы автоматизации для Power Automate, n8n, Make, Zapier и других платформ рабочих процессов. Анализирует определения потока JSON с сообщениями об ошибках, изучает официальную документацию и генерирует полный исправленный JSON, готовый для копирования и вставки. Срабатывает, когда пользователь предоставляет файлы JSON с ошибками, JSON рабочего процесса с ошибками, сообщения об ошибках, запросы на отладку или сбойный контент автоматизации. Возвращает структурированный отчет об отладке с анализом первопричин и рабочим исправленным JSON.
