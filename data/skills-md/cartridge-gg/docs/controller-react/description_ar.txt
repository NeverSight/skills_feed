قم بدمج وحدة التحكم في الخرطوشة في تطبيقات React باستخدام starknet-react. يُستخدم عند إنشاء تطبيقات ويب React/Next.js باستخدام وحدة التحكم، أو إعداد موفر StarknetConfig، أو استخدام خطافات مثل useConnect/useAccount، أو تنفيذ مكونات اتصال المحفظة. يغطي إعداد ControllerConnector وتكوين الموفر وأنماط تنفيذ المعاملات.
