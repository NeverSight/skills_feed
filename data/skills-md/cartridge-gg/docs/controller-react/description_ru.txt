Интегрируйте контроллер картриджей в приложения React с помощью starknet-react. Используйте при создании веб-приложений React/Next.js с помощью контроллера, настройке поставщика StarknetConfig, использовании таких перехватчиков, как useConnect/useAccount, или реализации компонентов подключения к кошельку. Охватывает настройку ControllerConnector, конфигурацию поставщика и шаблоны выполнения транзакций.
