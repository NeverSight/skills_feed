Intégrez le contrôleur de cartouche dans les applications React à l'aide de starknet-react. À utiliser lors de la création d'applications Web React/Next.js avec Controller, de la configuration du fournisseur StarknetConfig, de l'utilisation de hooks tels que useConnect/useAccount ou de l'implémentation de composants de connexion de portefeuille. Couvre la configuration de ControllerConnector, la configuration du fournisseur et les modèles d’exécution de transactions.
