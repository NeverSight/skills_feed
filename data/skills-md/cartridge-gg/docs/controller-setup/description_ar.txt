دمج محفظة Cartridge Controller في تطبيقات Starknet. يُستخدم عند إعداد وحدة التحكم لأول مرة، أو تثبيت الحزم، أو تكوين السلاسل/نقاط نهاية RPC، أو استكشاف مشكلات التكامل الأساسية وإصلاحها. يغطي التثبيت، وإنشاء مثيل لوحدة التحكم، واختيار ControllerConnector مقابل SessionConnector، وتكوين السلسلة، وتوافق الحزمة.
