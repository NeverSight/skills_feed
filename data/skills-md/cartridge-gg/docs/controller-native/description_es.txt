Integre el controlador de cartucho en aplicaciones móviles nativas (iOS, Android, React Native) y contenedores web (Capacitor). Úselo cuando cree juegos o aplicaciones móviles que necesiten la funcionalidad de billetera del controlador con firma de par de claves local. Cubre SessionConnector para flujos de autenticación nativos, enlaces FFI de Controller.c, configuración de clave de acceso con Apple App Site Association y configuración específica de la plataforma.
