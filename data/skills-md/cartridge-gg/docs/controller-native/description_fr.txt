Intégrez Cartouche Controller dans les applications mobiles natives (iOS, Android, React Native) et les wrappers Web (Capacitor). À utiliser lors de la création de jeux ou d'applications mobiles nécessitant la fonctionnalité de portefeuille Controller avec signature de paire de clés locale. Couvre SessionConnector pour les flux d'authentification natifs, les liaisons Controller.c FFI, la configuration du mot de passe avec Apple App Site Association et la configuration spécifique à la plate-forme.
