Интегрируйте Cartridge Controller в собственные мобильные приложения (iOS, Android, React Native) и веб-оболочки (Capacitor). Используйте при создании мобильных игр или приложений, которым требуется функциональность кошелька контроллера с подписью локальной пары ключей. Охватывает SessionConnector для собственных потоков аутентификации, привязки FFI Controller.c, настройку пароля с помощью Apple App Site Association и настройку для конкретной платформы.
