قم بدمج وحدة التحكم في الخرطوشة في خدمات الواجهة الخلفية باستخدام وضع Node.js أو Rust أو بدون رأس. يُستخدم عند إنشاء تطبيقات من جانب الخادم، أو الواجهات الخلفية للألعاب، أو الروبوتات الآلية، أو أي بيئة غير متصفحية تحتاج إلى تنفيذ معاملات Starknet. يغطي SessionProvider لـ Node.js وإعداد Rust SDK ووحدة التحكم بدون رأس مع مفاتيح التوقيع المخصصة.
