Интегрируйте Cartridge Controller во внутренние службы с помощью Node.js, Rust или автономного режима. Используйте при создании серверных приложений, игровых серверов, автоматических ботов или любой небраузерной среды, в которой необходимо выполнять транзакции Starknet. Охватывает SessionProvider для Node.js, настройку Rust SDK и автономный контроллер с настраиваемыми ключами подписи.
