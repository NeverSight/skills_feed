معالجة المستندات لـ OpenClaw - تحويل ملفات PDF ومستندات Office واستخراجها والتعرف الضوئي على الحروف وتنقيحها وتوقيعها ووضع علامة مائية عليها باستخدام واجهة برمجة تطبيقات Nutrient DWS. استخدمه عندما يُطلب منك تحويل المستندات (DOCX/XLSX/PPTX إلى PDF، أو PDF إلى صور أو تنسيقات Office)، أو استخراج نص أو جداول من ملفات PDF، أو تطبيق OCR على المستندات الممسوحة ضوئيًا، أو تنقيح المعلومات الحساسة أو معلومات تحديد الهوية الشخصية (PII)، أو إضافة علامات مائية، أو التوقيع الرقمي على المستندات. مشغلات عند "التحويل إلى PDF"، أو "استخراج النص"، أو "OCR this"، أو "تنقيح معلومات تحديد الهوية الشخصية"، أو "العلامة المائية"، أو "توقيع المستند"، أو أي طلب لمعالجة المستندات.
