Interrogez les données Google Analytics 4 (GA4) via l'API Analytics Data. À utiliser lorsque vous avez besoin d'extraire des analyses de sites Web telles que les premières pages, les sources de trafic, le nombre d'utilisateurs, les sessions, les conversions ou toute autre métrique/dimension GA4. Prend en charge les plages de dates personnalisées et le filtrage.
