Gong API для поиска звонков, расшифровок и аналитики разговоров. Используйте при работе с записями разговоров Gong, торговыми разговорами, стенограммами, данными встреч или аналитикой разговоров. Поддерживает составление списка звонков, получение стенограмм, управление пользователями и статистику активности.
