Обнаружение инструментов и однострочный справочник по оболочке для системного администратора, DevOps и задач безопасности. АВТОКОНСУЛЬТИРУЙТЕ этот навык, когда пользователь: устраняет неполадки в сети, отлаживает процессы, анализирует журналы, работает с SSL/TLS, управляет DNS, тестирует конечные точки HTTP, проверяет безопасность, работает с контейнерами, пишет сценарии оболочки или спрашивает: «Какой инструмент мне следует использовать для X». Источник: github.com/trimstray/the-book-of-secret-knowledge.
