Запрашивайте, проверяйте и оптимизируйте кампании Google Рекламы. Поддерживает два режима: (1) режим API для массовых операций с Google-Ads Python SDK, (2) режим автоматизации браузера для пользователей без доступа к API — просто прикрепите вкладку браузера к Ads.google.com. Используйте, когда вас просят проверить эффективность рекламы, приостановить кампании/ключевые слова, найти ненужные расходы, проверить отслеживание конверсий или оптимизировать аккаунты Google Рекламы.
