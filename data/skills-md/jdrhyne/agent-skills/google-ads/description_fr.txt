Interrogez, auditez et optimisez les campagnes Google Ads. Prend en charge deux modes : (1) le mode API pour les opérations groupées avec le SDK Python google-ads, (2) le mode d'automatisation du navigateur pour les utilisateurs sans accès à l'API : attachez simplement un onglet de navigateur à ads.google.com. À utiliser lorsque vous y êtes invité pour vérifier les performances des annonces, suspendre des campagnes/mots clés, rechercher des dépenses inutiles, auditer le suivi des conversions ou optimiser les comptes Google Ads.
