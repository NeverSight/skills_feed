Система долговременной памяти AI Agent с сохранением между сеансами и проектами.
Триггеры:
- /remember — хранить воспоминания
- /recall — Поиск воспоминаний.
- /forget – удалить/архивировать воспоминания.
- /memory-status — Проверить статус
- Когда необходимо сохранить важные идеи разговора
- При совместном использовании пользовательских настроек в разных проектах
