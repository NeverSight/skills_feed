Desarrolle rutas RSSHub desde la URL de cualquier sitio web. Úselo cuando los usuarios quieran crear fuentes RSS para sitios web, crear rutas RSSHub o enviar PR de ruta al proyecto RSSHub. Maneja el flujo de trabajo completo: analiza la página de destino, genera código de ruta (namespace.ts, controlador de ruta, radar.ts), ejecuta el servidor de desarrollo local y prueba, formatea/lint y envía PR a DIYgod/RSSHub. Úselo también cuando los usuarios pregunten sobre los patrones de desarrollo de rutas RSSHub, problemas de depuración de rutas o convenciones de proyectos RSSHub.
