Développez des itinéraires RSSHub à partir de n’importe quelle URL de site Web. À utiliser lorsque les utilisateurs souhaitent créer des flux RSS pour des sites Web, créer des itinéraires RSSHub ou soumettre des PR d'itinéraire au projet RSSHub. Gère l'intégralité du flux de travail : analysez la page cible, générez le code de route (namespace.ts, gestionnaire de route, radar.ts), exécutez le serveur de développement local et testez, formatez/lint et soumettez les relations publiques à DIYgod/RSSHub. À utiliser également lorsque les utilisateurs posent des questions sur les modèles de développement de routes RSSHub, les problèmes de route de débogage ou les conventions de projet RSSHub.
