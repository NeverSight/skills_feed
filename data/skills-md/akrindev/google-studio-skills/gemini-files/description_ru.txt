Загружайте файлы и управляйте ими с помощью Google Gemini File API через скрипты/. Используйте для загрузки изображений, аудио, видео, PDF-файлов и других файлов для использования с моделями Gemini. Поддерживает загрузку файлов, проверку статуса и управление файлами. Триггеры «загрузить файл», «файл API», «загрузить изображение», «загрузить PDF», «загрузить видео», «управление файлами».
