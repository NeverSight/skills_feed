Генерируйте текстовый контент с использованием моделей Google Gemini с помощью скриптов/. Используйте для генерации текста, мультимодальных подсказок с изображениями, режима мышления для сложных рассуждений, вывода в формате JSON и обоснования поиска Google для получения информации в реальном времени. Триггеры: «Сгенерировать с помощью Gemini», «Использовать Gemini для текста», «Генерирование текста AI», «Мультимодальная подсказка», «Режим мышления Близнецов», «Обоснованный ответ».
