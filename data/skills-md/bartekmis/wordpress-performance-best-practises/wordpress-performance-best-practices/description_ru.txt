Рекомендации по оптимизации производительности WordPress для разработки плагинов, тем и пользовательского кода. Этот навык следует использовать при написании, проверке или рефакторинге PHP-кода WordPress, чтобы обеспечить оптимальные шаблоны производительности. Запускается при выполнении задач, связанных с WP_Query, запросами к базе данных, кешированием, перехватами, REST API или разработкой тем/плагинов WordPress.
