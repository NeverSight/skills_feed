Richtlinien zur WordPress-Leistungsoptimierung für die Entwicklung von Plugins, Themes und benutzerdefiniertem Code. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Refactoring von WordPress-PHP-Code verwendet werden, um optimale Leistungsmuster sicherzustellen. Löst Aufgaben aus, die WP_Query, Datenbankabfragen, Caching, Hooks, REST API oder die Entwicklung von WordPress-Themes/Plugins umfassen.
