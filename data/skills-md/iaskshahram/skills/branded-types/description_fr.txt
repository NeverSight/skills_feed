Implémentez des types de marque (nominaux/opaques) dans TypeScript pour éviter le mélange accidentel de types structurellement identiques. À utiliser lors de l'écriture d'ID de type sécurisé (UserId, PostId), de chaînes validées (E-mail, URL), de nombres spécifiques à une unité (Mètres, Secondes) ou de tout scénario nécessitant une saisie nominale dans le système de type structurel de TypeScript.
