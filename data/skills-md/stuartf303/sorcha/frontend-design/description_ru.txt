Стилизует компоненты Blazor WASM с помощью CSS и шаблонов адаптивного дизайна с помощью MudBlazor Material Design.
Используйте при: создании новых компонентов, стилизации существующих компонентов, реализации адаптивных макетов, добавлении анимации/переходов или работе с библиотекой компонентов MudBlazor.
