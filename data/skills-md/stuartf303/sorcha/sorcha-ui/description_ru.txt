Создает страницы Sorcha.UI Blazor WASM с сопровождающими тестами Playwright E2E, используя инфраструктуру тестирования Docker.
Используйте, когда: работаете над Sorcha.UI, создаете новые страницы, заменяете страницы шаблонов, добавляете функции пользовательского интерфейса или тестируете функциональность пользовательского интерфейса с помощью Docker.
