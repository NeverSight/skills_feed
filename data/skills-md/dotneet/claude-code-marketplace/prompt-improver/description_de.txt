Eine Fähigkeit zur Verbesserung von Eingabeaufforderungen durch Anwendung allgemeiner Best Practices für LLM/Agenten. Wenn der Benutzer eine Eingabeaufforderung gibt, gibt dieser Skill eine verbesserte Version aus, identifiziert fehlende Informationen und stellt spezifische Verbesserungspunkte bereit. Wird verwendet, wenn der Benutzer „diese Eingabeaufforderung verbessern“, „diese Eingabeaufforderung überprüfen“ oder „diese Eingabeaufforderung verbessern“ möchte.
