Una capacità di migliorare i suggerimenti applicando le migliori pratiche generali LLM/agente. Quando l'utente fornisce una richiesta, questa abilità produce una versione migliorata, identifica le informazioni mancanti e fornisce punti di miglioramento specifici. Da utilizzare quando l'utente chiede di "migliorare questo prompt", "rivedere questo prompt" o "migliorare questo prompt".
