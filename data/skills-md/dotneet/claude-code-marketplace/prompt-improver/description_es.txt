Una habilidad para mejorar las indicaciones mediante la aplicación de las mejores prácticas generales de LLM/agente. Cuando el usuario proporciona un mensaje, esta habilidad genera una versión mejorada, identifica la información faltante y proporciona puntos de mejora específicos. Úselo cuando el usuario solicite "mejorar este mensaje", "revisar este mensaje" o "mejorar este mensaje".
