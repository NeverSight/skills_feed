Une compétence pour améliorer les invites en appliquant les meilleures pratiques générales LLM/agent. Lorsque l'utilisateur fournit une invite, cette compétence génère une version améliorée, identifie les informations manquantes et fournit des points d'amélioration spécifiques. À utiliser lorsque l'utilisateur demande « d'améliorer cette invite », de « revoir cette invite » ou de « améliorer cette invite ».
