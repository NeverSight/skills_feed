مهارة إجراء عمليات البحث على الويب والبحث والاستدلال باستخدام Perplexity API. يتعامل مع استرجاع معلومات الويب في الوقت الفعلي، وتحليل البحث العميق، ومهام التفكير المنطقي المتقدمة. يُستخدم عندما يطلب المستخدم إجراء عمليات بحث على الويب أو إجراء بحث أو قول أشياء مثل "البحث" أو "البحث عن" أو "أحدث المعلومات" أو "البحث" أو "التحليل".
