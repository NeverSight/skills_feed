Une compétence pour récupérer la dernière documentation de la bibliothèque à l'aide de Context7. À utiliser lorsque l'utilisateur demande comment utiliser une bibliothèque, demande des exemples de code ou demande « d'utiliser context7 ». Empêche les hallucinations basées sur des données d'entraînement obsolètes et fournit des informations API à jour.
