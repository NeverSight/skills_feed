Utilizza Chrome DevTools MCP per estrarre e analizzare il contenuto degli articoli della pagina web. Usalo quando gli utenti richiedono di ottenere contenuti di pagine Web, leggere articoli online, estrarre testo da siti Web, acquisire istantanee di pagine Web o analizzare la struttura di pagine Web. Supporta più formati di estrazione tra cui testo semplice, HTML e contenuto strutturato. Ottimizzato appositamente per siti Web con restrizioni di sicurezza come gli account ufficiali di WeChat.
