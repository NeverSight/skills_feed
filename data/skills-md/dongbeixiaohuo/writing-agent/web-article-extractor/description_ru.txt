Используйте Chrome DevTools MCP для извлечения и анализа содержимого статей веб-страниц. Используйте его, когда пользователи запрашивают содержимое веб-страницы, читают онлайн-статьи, извлекают текст с веб-сайтов, делают снимки веб-страниц или анализируют структуру веб-страницы. Поддерживает несколько форматов извлечения, включая обычный текст, HTML и структурированный контент. Специально оптимизирован для веб-сайтов с ограничениями безопасности, таких как официальные учетные записи WeChat.
