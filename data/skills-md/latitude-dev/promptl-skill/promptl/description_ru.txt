Руководство по синтаксису PromptL для написания подсказок на платформе Latitude. Этот навык следует использовать при написании, просмотре или редактировании подсказок PromptL. Запускается по задачам, связанным с созданием подсказок, настройкой параметров LLM, использованием переменных, условий, циклов, цепочек, инструментов, агентов или любой разработкой подсказок в Latitude.
