PromptL-Syntaxanleitung zum Schreiben von Eingabeaufforderungen in der Latitude-Plattform. Diese Fähigkeit sollte beim Schreiben, Überprüfen oder Bearbeiten von PromptL-Eingabeaufforderungen verwendet werden. Löst Aufgaben aus, die das Erstellen von Eingabeaufforderungen, das Konfigurieren von LLM-Parametern, die Verwendung von Variablen, Bedingungen, Schleifen, Ketten, Tools, Agenten oder jegliche Eingabeaufforderungstechnik in Latitude umfassen.
