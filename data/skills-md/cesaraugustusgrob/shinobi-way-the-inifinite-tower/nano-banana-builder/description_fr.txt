Créez des applications Web complètes optimisées par les API de génération d'images Nano Banana et Nano Banana Pro de Google Gemini. À utiliser lors de la création de générateurs d'images Next.js, d'éditeurs, de galeries ou de toute application Web intégrant des modèles gemini-2.5-flash-image ou gemini-3-pro-image-preview. Couvre les composants React, les actions du serveur, les routes API, le stockage, la limitation de débit et les modèles de déploiement de production.
