Рабочий процесс контроля версий Git для команд разработчиков модов Hytale. Охватывает настройку репозитория, стратегии ветвления, соглашения о фиксации, запросы на включение и лучшие практики совместной работы. Используйте при работе с командами, управлении версиями модов или настройке системы контроля версий.
