Используйте этот навык, когда пользователю необходимо настроить Stripe, реализовать подписки, управлять выставлением счетов, настраивать планы платежей, управлять неудачными платежами, заниматься возвратами средств, настраивать сбор налогов или устранять проблемы с платежами. Охватывает конфигурацию Stripe, жизненный цикл подписки, напоминание, соблюдение налоговых требований и операции по выставлению счетов для SaaS с начальной загрузкой.
