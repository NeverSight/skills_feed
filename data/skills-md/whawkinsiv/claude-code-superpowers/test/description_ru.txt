Используйте этот навык, когда пользователю необходимо протестировать функции перед развертыванием, создать тестовые сценарии, найти крайние случаи или проверить исправления ошибок. Охватывает рабочие процессы ручного тестирования, кроссбраузерное тестирование, выявление крайних случаев и контрольные списки тестирования для основателей, не являющихся техническими специалистами.
