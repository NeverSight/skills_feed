Utilisez cette compétence lorsque l'application de l'utilisateur semble lente, que la base de code semble gonflée ou après un travail de développement important. Optimise sur quatre dimensions : vitesse (chargement de la page, réponse de l'API), code (fichiers inutilisés, code mort), base de données (données orphelines, hygiène du schéma) et dépendances (ballonnement du package, taille du bundle).
