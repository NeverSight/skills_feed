Utilice esta habilidad cuando la aplicación del usuario se sienta lenta, el código base se sienta inflado o después de un trabajo de desarrollo significativo. Optimiza en cuatro dimensiones: velocidad (carga de página, respuesta de API), código (archivos no utilizados, código inactivo), base de datos (datos huérfanos, higiene de esquema) y dependencias (inflación de paquetes, tamaño de paquete).
