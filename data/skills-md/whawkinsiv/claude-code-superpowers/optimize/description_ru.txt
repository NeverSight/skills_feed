Используйте этот навык, когда приложение пользователя кажется медленным, кодовая база раздутой или после значительной работы по разработке. Оптимизируется по четырем измерениям: скорость (загрузка страницы, ответ API), код (неиспользуемые файлы, мертвый код), база данных (потерянные данные, гигиена схемы) и зависимости (раздувание пакета, размер пакета).
