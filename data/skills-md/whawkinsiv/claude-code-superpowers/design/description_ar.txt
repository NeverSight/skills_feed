استخدم هذه المهارة عندما يحتاج المستخدم إلى إنشاء نظام تصميم، أو إنشاء أنماط واجهة المستخدم، أو اختيار الألوان والطباعة، أو ضمان الاتساق البصري عبر تطبيق SaaS الخاص به. يعمل في وضع الإنشاء (تطبيقات جديدة)، أو وضع التعديل (تحديث الأنظمة الحالية)، أو وضع العمل ضمن (بناء الميزات التي تتبع الأنماط المحددة).
