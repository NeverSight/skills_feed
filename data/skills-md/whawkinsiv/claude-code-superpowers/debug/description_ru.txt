Используйте этот навык, когда функции не работают, пользователи сообщают об ошибках, развертывания завершаются неудачно или тесты не проходят. Направляет систематическую отладку: воспроизводит ошибки, собирает диагностическую информацию, читает сообщения об ошибках и работает с инструментами искусственного интеллекта для эффективного устранения проблем.
