Conseils d'experts sur les composants basés sur le style SwiftUI suivant les modèles Apple (ButtonStyle, LabelStyle). À utiliser dans les cas suivants : création de composants avec des protocoles de style, implémentation de modèles de configuration, ajout de styles basés sur l'environnement, création de bibliothèques de composants extensibles ou révision de l'architecture des composants.
