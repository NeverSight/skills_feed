مجموعة أدوات لوكلاء الذكاء الاصطناعي لتصحيح أخطاء كود Python بشكل مستقل دون دعم IDE. يُستخدم عندما يحتاج الوكيل إلى تصحيح التعليمات البرمجية، أو إدراج نقاط التوقف برمجيًا، أو تحليل حالة التنفيذ، أو اختبار الفرضيات، أو أتمتة سير عمل تصحيح الأخطاء. يدعم pdb، وipdb، وdebugpy مع إدراج نقاط توقف برمجية، ونقاط توقف مشروطة، واقتراحات نقاط توقف قائمة على AST، وسير عمل تصحيح الأخطاء الآلي للوكلاء.
