Toolkit für KI-Agenten zum autonomen Debuggen von Python-Code ohne IDE-Unterstützung. Verwenden Sie diesen Agenten, wenn er Code debuggen, Haltepunkte programmgesteuert einfügen, den Ausführungsstatus analysieren, Hypothesen testen oder Debugging-Workflows automatisieren muss. Unterstützt pdb, ipdb, debugpy mit programmgesteuerter Haltepunkteinfügung, bedingte Haltepunkte, AST-basierte Haltepunktvorschläge und automatisierte Debugging-Workflows für Agenten.
