Kit de herramientas para que los agentes de IA depuren código Python de forma autónoma sin soporte IDE. Utilícelo cuando el agente necesite depurar código, insertar puntos de interrupción mediante programación, analizar el estado de ejecución, probar hipótesis o automatizar flujos de trabajo de depuración. Admite pdb, ipdb, debugpy con inserción programática de puntos de interrupción, puntos de interrupción condicionales, sugerencias de puntos de interrupción basadas en AST y flujos de trabajo de depuración automatizados para agentes.
