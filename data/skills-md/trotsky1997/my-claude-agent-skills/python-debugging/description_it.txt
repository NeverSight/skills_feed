Toolkit per agenti AI per eseguire il debug del codice Python in modo autonomo senza supporto IDE. Da utilizzare quando l'agente deve eseguire il debug del codice, inserire punti di interruzione a livello di codice, analizzare lo stato di esecuzione, testare ipotesi o automatizzare i flussi di lavoro di debug. Supporta pdb, ipdb, debugpy con inserimento di punti di interruzione programmatici, punti di interruzione condizionali, suggerimenti di punti di interruzione basati su AST e flussi di lavoro di debug automatizzati per gli agenti.
