Набор инструментов для агентов ИИ для автономной отладки кода Python без поддержки IDE. Используйте, когда агенту необходимо отладить код, программно вставить точки останова, проанализировать состояние выполнения, проверить гипотезы или автоматизировать рабочие процессы отладки. Поддерживает pdb, ipdb, debugpy с программной вставкой точек останова, условные точки останова, предложения точек останова на основе AST и автоматизированные рабочие процессы отладки для агентов.
