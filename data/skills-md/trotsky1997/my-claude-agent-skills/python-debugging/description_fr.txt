Boîte à outils permettant aux agents IA de déboguer le code Python de manière autonome sans prise en charge de l'IDE. À utiliser lorsque l'agent doit déboguer du code, insérer des points d'arrêt par programme, analyser l'état d'exécution, tester des hypothèses ou automatiser les flux de travail de débogage. Prend en charge pdb, ipdb, debugpy avec insertion de points d'arrêt programmatiques, points d'arrêt conditionnels, suggestions de points d'arrêt basées sur AST et flux de travail de débogage automatisés pour les agents.
