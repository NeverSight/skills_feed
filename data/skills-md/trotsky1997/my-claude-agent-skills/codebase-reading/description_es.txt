Metodología sistemática para leer y comprender grandes bases de código de manera eficiente. Úselo cuando (1) comprenda rápidamente una base de código nueva o desconocida, (2) se prepare para modificar o ampliar el código existente de forma segura, (3) depure problemas complejos que requieren una comprensión profunda del código, (4) incorpore nuevos miembros del equipo a una base de código, (5) realice auditorías de código o revisiones de seguridad, (6) refactorice código heredado con confianza, (7) cree documentación para sistemas existentes, (8) rastree flujos de ejecución y transformaciones de datos
