Méthodologie systématique pour lire et comprendre efficacement de grandes bases de code. À utiliser lorsque (1) comprendre rapidement une base de code nouvelle ou inconnue, (2) préparer la modification ou l'extension du code existant en toute sécurité, (3) déboguer des problèmes complexes nécessitant une compréhension approfondie du code, (4) intégrer de nouveaux membres de l'équipe dans une base de code, (5) effectuer des audits de code ou des examens de sécurité, (6) refactoriser le code existant en toute confiance, (7) créer de la documentation pour les systèmes existants, (8) suivre les flux d'exécution et les transformations de données.
