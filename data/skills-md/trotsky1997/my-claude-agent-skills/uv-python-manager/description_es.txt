Guía completa para usar uv, un paquete Python y administrador de proyectos extremadamente rápido escrito en Rust. Úselo cuando trabaje con proyectos de Python para (1) instalar y administrar paquetes de Python, (2) crear y administrar entornos virtuales, (3) instalar y cambiar entre versiones de Python, (4) ejecutar scripts de Python con administración de dependencias, (5) administrar proyectos de Python con archivos de bloqueo, (6) reemplazar pip, pip-tools, pipx, poesía, pyenv, flujos de trabajo virtualenv o cualquier tarea de administración de paquetes de Python.
