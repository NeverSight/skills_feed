Rust で書かれた非常に高速な Python パッケージおよびプロジェクト マネージャーである uv を使用するための包括的なガイド。 (1) Python パッケージのインストールと管理、(2) 仮想環境の作成と管理、(3) Python バージョンのインストールと切り替え、(4) 依存関係管理を使用した Python スクリプトの実行、(5) ロックファイルを使用した Python プロジェクトの管理、(6) pip、pip-tools、pipx、poetry、pyenv、virtualenv ワークフロー、または任意の Python パッケージ管理タスクの置き換えのために Python プロジェクトを操作するときに使用します。
