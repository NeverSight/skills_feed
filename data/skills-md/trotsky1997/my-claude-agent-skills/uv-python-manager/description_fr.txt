Guide complet pour utiliser uv, un package Python extrêmement rapide et un gestionnaire de projet écrit en Rust. À utiliser lorsque vous travaillez avec des projets Python pour (1) installer et gérer des packages Python, (2) créer et gérer des environnements virtuels, (3) installer et basculer entre les versions de Python, (4) exécuter des scripts Python avec gestion des dépendances, (5) gérer des projets Python avec des fichiers de verrouillage, (6) remplacer pip, pip-tools, pipx, poésie, pyenv, virtualenv workflows ou toute tâche de gestion de packages Python.
