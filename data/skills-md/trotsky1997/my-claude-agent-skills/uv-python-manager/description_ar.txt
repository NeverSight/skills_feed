دليل شامل لاستخدام الأشعة فوق البنفسجية، وهي حزمة Python سريعة للغاية ومدير مشروع مكتوب بلغة Rust. يُستخدم عند العمل مع مشاريع Python من أجل (1) تثبيت حزم Python وإدارتها، (2) إنشاء بيئات افتراضية وإدارتها، (3) التثبيت والتبديل بين إصدارات Python، (4) تشغيل برامج Python النصية مع إدارة التبعية، (5) إدارة مشاريع Python باستخدام ملفات القفل، (6) استبدال النقطة، أو أدوات النقطة، أو pipx، أو الشعر، أو pyenv، أو مسارات عمل Virtualenv، أو أي مهام لإدارة حزم Python.
