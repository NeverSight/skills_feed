Напишите эффективные подсказки для создания мультимодального видео с помощью искусственного интеллекта Jimeng Seedance 2.0. Используйте, когда пользователи хотят создавать видеоподсказки с использованием текста, изображений, видео и аудиовходов с помощью справочной системы @. Охватывает движения камеры, репликацию эффектов, расширение видео, редактирование, подбор музыкального ритма, рекламу электронной коммерции, короткие драмы и образовательный контент.
