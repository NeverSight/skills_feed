Invoque OpenAI Codex CLI pour un deuxième avis indépendant d’une autre famille de modèles. À utiliser lorsque (1) vous examinez des plans, des différences, du code ou des décisions d'architecture avant d'agir, (2) effectuez une nouvelle transmission du code ou d'une base de code complète pour les bogues, les cas extrêmes ou les exigences manquantes, (3) vérifiez l'intégrité du travail avant de le partager avec les utilisateurs ou les parties prenantes, (4) souhaitez une perspective concurrente sur les compromis ou les choix de conception.
