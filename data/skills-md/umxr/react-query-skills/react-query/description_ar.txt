أفضل الممارسات والأنماط لـ TanStack Query (React Query) من أدلة TkDodo الرسمية. يُستخدم عند كتابة كود React Query أو مراجعته أو تصحيحه، بما في ذلك الاستعلامات والطفرات والتخزين المؤقت والتحديثات الإيجابية وتكامل TypeScript. المشغلات على أنماط useQuery، أو useMutation، أو useInfiniteQuery، أو QueryClient، أو queryKey، أو staleTime، أو CacheTime، أو gcTime، أو validateQueries، أو React Query.
