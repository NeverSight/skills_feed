Meilleures pratiques et modèles TanStack Query (React Query) issus des guides faisant autorité de TkDodo. À utiliser lors de l'écriture, de la révision ou du débogage du code React Query, y compris les requêtes, les mutations, la mise en cache, les mises à jour optimistes et l'intégration de TypeScript. Déclencheurs sur les modèles useQuery, useMutation, useInfiniteQuery, QueryClient, queryKey, staleTime, cacheTime, gcTime, invalidateQueries ou React Query.
