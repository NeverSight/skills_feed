Best Practices und Muster für TanStack Query (React Query) aus den maßgeblichen Leitfäden von TkDodo. Wird zum Schreiben, Überprüfen oder Debuggen von React Query-Code verwendet, einschließlich Abfragen, Mutationen, Caching, optimistischen Aktualisierungen und TypeScript-Integration. Löst die Muster „useQuery“, „useMutation“, „useInfiniteQuery“, „QueryClient“, „queryKey“, „staleTime“, „cacheTime“, „gcTime“, „invalidateQueries“ oder „React Query“ aus.
