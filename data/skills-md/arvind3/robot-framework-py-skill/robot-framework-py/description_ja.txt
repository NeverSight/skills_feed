API、Web、モバイル、データレイヤーにわたる Python 中心のテスト自動化のためのロボット フレームワーク スキル。用途: .robot スイート、.resource ファイル、RequestsLibrary API テスト、SeleniumLibrary UI テスト、ブラウザ (Playwright) テスト、AppiumLibrary モバイル テスト、DatabaseLibrary データ チェック、カスタム Python キーワード ライブラリの作成またはリファクタリング。有効なロボット フレームワーク セクション、再利用可能なリソースの階層化、決定論的なアサーション、およびガードレールを強制します。
