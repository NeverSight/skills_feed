Навыки Robot Framework для автоматизации тестирования на основе Python на уровнях API, Интернета, мобильных устройств и данных. ИСПОЛЬЗОВАНИЕ ДЛЯ: создания или рефакторинга наборов .robot, файлов .resource, тестов API RequestsLibrary, тестов пользовательского интерфейса SeleniumLibrary, тестов браузера (драматурга), мобильных тестов AppiumLibrary, проверок данных DatabaseLibrary и пользовательских библиотек ключевых слов Python; обеспечение соблюдения допустимых разделов Robot Framework, многоразового использования ресурсов, детерминированных утверждений и ограждений.
