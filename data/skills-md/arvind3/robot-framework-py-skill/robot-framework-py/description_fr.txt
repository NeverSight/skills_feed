Compétence Robot Framework pour l'automatisation des tests centrés sur Python sur les couches API, Web, mobile et données. UTILISER POUR : créer ou refactoriser des suites .robot, des fichiers .resource, des tests d'API RequestsLibrary, des tests d'interface utilisateur SeleniumLibrary, des tests de navigateur (Playwright), des tests mobiles AppiumLibrary, des vérifications de données DatabaseLibrary et des bibliothèques de mots clés Python personnalisées ; en appliquant des sections Robot Framework valides, une superposition de ressources réutilisables, des assertions déterministes et des garde-fous.
