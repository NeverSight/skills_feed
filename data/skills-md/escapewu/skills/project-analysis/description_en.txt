Project analysis tool for analyzing the system architecture and module data flow of a codebase. Use this skill when users need to understand project structure, generate architecture diagrams, analyze data flow between modules, or produce sequence diagrams. Outputs visual charts in Mermaid syntax. Applicable scenarios: (1) project architecture review, (2) module dependency analysis, (3) data flow tracking, (4) onboarding new team members, (5) technical documentation generation
