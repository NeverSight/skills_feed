Compound-Engineering-Schleife – 5 Stufen zur Reduzierung von Nacharbeiten. Verwenden Sie es, wenn: (1) eine neue Arbeit beginnt (Brainstorming); (2) Planung der Umsetzung (Plan); (3) Code schreiben (Arbeit); (4) Evaluierung der Ergebnisse (Überprüfung); (5) Erfassen von Lektionen (zusammengesetzt). NICHT für: Tagesrhythmus (täglich verwenden); Nur Aufgabenverwaltung (Aufgabe verwenden).
