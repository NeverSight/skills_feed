Выполняет функции рекрутера и менеджера по агентским навыкам. Обнаруживает возможности черезkills.sh и GitHub, выполняет глубокую оценку качества, проверяет наличие угроз безопасности (быстрое внедрение), управляет совместимостью и выводит удобочитаемые рекомендации + действия JSON, поддающиеся машинному анализу. Интегрирует официальный интерфейс командной строки навыков («проверка навыков npx» / «обновление») для упрощения обслуживания установленных навыков.
