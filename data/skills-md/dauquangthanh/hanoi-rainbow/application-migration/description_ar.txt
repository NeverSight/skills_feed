يرشد مشاريع ترحيل التطبيقات الشاملة بما في ذلك تحديث النظام القديم، والترحيل السحابي، وتحديثات مكدس التكنولوجيا، وترحيل قاعدة البيانات، وتحويل البنية. يغطي التقييم والتخطيط واستراتيجيات الترحيل (التين الخانق، والانفجار الكبير، والمراحل)، وإدارة المخاطر، وترحيل البيانات، والاختبار، والتحويل. يُستخدم عند ترحيل التطبيقات، أو تحديث الأنظمة القديمة، أو الانتقال إلى السحابة، أو تغيير مجموعات التكنولوجيا، أو عندما يذكر المستخدمون "الترحيل"، أو "التحديث"، أو "إعادة النظام الأساسي"، أو "الرفع والتحويل"، أو "إعادة البناء"، أو "النمط الخانق"، أو "التحول القديم".
