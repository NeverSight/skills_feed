Effectue des examens complets du code frontend, y compris l'analyse des composants React/Vue/Angular, l'évaluation de la qualité TypeScript/JavaScript, l'examen CSS/style, l'optimisation des performances, la conformité en matière d'accessibilité, les vulnérabilités de sécurité et la validation des meilleures pratiques. Produit des rapports d'examen détaillés avec des problèmes spécifiques, des évaluations de gravité et des recommandations concrètes. À utiliser lors de la révision du code front-end, de l'analyse des composants React/Vue/Angular, de la vérification de la qualité JavaScript/TypeScript, de la validation de CSS/SCSS, de l'évaluation des performances Web ou lorsque les utilisateurs mentionnent « réviser le code front-end », « vérifier les composants React », « analyser JavaScript », « réviser TypeScript », « valider l'accessibilité » ou « qualité du code front-end ».
