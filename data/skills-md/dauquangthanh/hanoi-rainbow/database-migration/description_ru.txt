Управляет проектами миграции баз данных, включая изменения ядра (MySQL на PostgreSQL, Oracle на PostgreSQL, SQL Server на PostgreSQL), обновления версий, миграцию в облако (локальную в RDS/Cloud SQL/базу данных Azure), миграцию схем, миграцию с нулевым временем простоя, настройку репликации и стратегии миграции данных. Охватывает однородную и гетерогенную миграцию, процессы ETL, процедуры переключения и планы отката. Используйте при миграции баз данных, изменении механизмов баз данных, обновлении версий баз данных, перемещении баз данных в облако или когда пользователи упоминают «миграцию базы данных», «миграцию БД», «миграцию PostgreSQL», «MySQL в Postgres», «миграцию Oracle», «обновление базы данных» или «миграцию облачной базы данных».
