Guide les projets de migration de bases de données, y compris les modifications du moteur (MySQL vers PostgreSQL, Oracle vers PostgreSQL, SQL Server vers PostgreSQL), les mises à niveau de version, les migrations cloud (sur site vers RDS/Cloud SQL/Azure Database), les migrations de schémas, les migrations sans temps d'arrêt, la configuration de la réplication et les stratégies de migration de données. Couvre les migrations homogènes et hétérogènes, les processus ETL, les procédures de basculement et les plans de restauration. À utiliser lors de la migration de bases de données, de la modification de moteurs de bases de données, de la mise à niveau des versions de bases de données, du déplacement de bases de données vers le cloud ou lorsque les utilisateurs mentionnent « migration de base de données », « migration de base de données », « migration PostgreSQL », « MySQL vers Postgres », « migration Oracle », « mise à niveau de base de données » ou « migration de base de données cloud ».
