Guide la refactorisation systématique du code pour améliorer la qualité, la maintenabilité et la conception du code. Identifie les odeurs de code, applique des modèles de refactoring, garantit la couverture des tests et suit des pratiques de refactoring sûres. Produit un code plus propre et plus maintenable tout en préservant les fonctionnalités. À utiliser pour améliorer la qualité du code, éliminer la dette technique, préparer l'ajout de fonctionnalités, traiter les odeurs de code ou lorsque les utilisateurs mentionnent la refactorisation, le nettoyage du code, l'amélioration de la conception, la réduction de la complexité ou la modernisation du code existant.
