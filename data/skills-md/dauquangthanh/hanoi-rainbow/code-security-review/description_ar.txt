إجراء مراجعات شاملة لرموز الأمان بما في ذلك اكتشاف الثغرات الأمنية (OWASP Top 10, CWE)، وعيوب المصادقة/الترخيص، وهجمات الحقن، ومشكلات التشفير، والكشف عن البيانات الحساسة، وأمن واجهة برمجة التطبيقات (API)، ونقاط ضعف التبعية، والتكوينات الأمنية الخاطئة، والتحقق من صحة الامتثال (PCI-DSS، وGDPR، وHIPAA). يُنتج تقارير تقييم أمني مفصلة مع مراجع CVE، ونتائج CVSS، وسيناريوهات الاستغلال، وإرشادات العلاج. يُستخدم عند مراجعة أمان التعليمات البرمجية، أو إجراء عمليات تدقيق الأمان، أو التحقق من نقاط الضعف، أو التحقق من صحة الضوابط الأمنية، أو تقييم المخاطر الأمنية، أو عندما يذكر المستخدمون "مراجعة الأمان"، أو "فحص الثغرات الأمنية"، أو "تدقيق الأمان"، أو "اختبار الاختراق"، أو "OWASP"، أو "تقييم الأمان"، أو "الترميز الآمن"، أو "الامتثال الأمني".
