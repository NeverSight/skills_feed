يوفر إرشادات شاملة لـ Oracle Cloud Infrastructure (OCI) بما في ذلك مثيلات الحوسبة والشبكات (VCN وموازنات التحميل وVPN) والتخزين (الكتلة والكائن والملف) وخدمات قاعدة البيانات (قاعدة البيانات الذاتية وMySQL وNoSQL) وتنسيق الحاويات (OKE) وإدارة الهوية والوصول (IAM) وإدارة الموارد وتحسين التكلفة والبنية التحتية كرمز (موفر Terraform OCI وإدارة الموارد). ينتج كود البنية التحتية والبرامج النصية للنشر وأدلة التكوين والرسوم البيانية المعمارية. يُستخدم عند تصميم بنية OCI، أو توفير الموارد السحابية، أو الترحيل إلى Oracle Cloud، أو تنفيذ أمان OCI، أو إعداد قواعد بيانات OCI، أو نشر التطبيقات المعبأة في حاويات على OKE، أو إدارة موارد OCI، أو عندما يذكر المستخدمون "Oracle Cloud"، أو "OCI"، أو "Autonomous Database"، أو "VCN"، أو "OKE"، أو "OCI Terraform"، أو "Resource Manager"، أو "Oracle Cloud Infrastructure"، أو "ترحيل OCI".
