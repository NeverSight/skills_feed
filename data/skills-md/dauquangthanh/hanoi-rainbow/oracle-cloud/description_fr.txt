Fournit des conseils complets sur Oracle Cloud Infrastructure (OCI), y compris les instances de calcul, la mise en réseau (VCN, équilibreurs de charge, VPN), le stockage (bloc, objet, fichier), les services de base de données (base de données autonome, MySQL, NoSQL), l'orchestration de conteneurs (OKE), la gestion des identités et des accès (IAM), la gestion des ressources, l'optimisation des coûts et l'infrastructure en tant que code (fournisseur Terraform OCI, Resource Manager). Produit du code d'infrastructure, des scripts de déploiement, des guides de configuration et des diagrammes architecturaux. À utiliser lors de la conception d'une architecture OCI, du provisionnement de ressources cloud, de la migration vers Oracle Cloud, de la mise en œuvre de la sécurité OCI, de la configuration de bases de données OCI, du déploiement d'applications conteneurisées sur OKE, de la gestion des ressources OCI ou lorsque les utilisateurs mentionnent « Oracle Cloud », « OCI », « Autonomous Database », « VCN », « OKE », « OCI Terraform », « Resource Manager », « Oracle Cloud Infrastructure » ou « OCI migration ».
