Предоставляет комплексное руководство по Oracle Cloud Infrastructure (OCI), включая вычислительные экземпляры, сети (VCN, балансировщики нагрузки, VPN), хранилище (блочное, объектное, файловое), службы баз данных (автономная база данных, MySQL, NoSQL), оркестровку контейнеров (OKE), управление идентификацией и доступом (IAM), управление ресурсами, оптимизацию затрат и инфраструктуру как код (поставщик Terraform OCI, Resource Manager). Создает код инфраструктуры, сценарии развертывания, руководства по настройке и архитектурные схемы. Используйте при проектировании архитектуры OCI, предоставлении облачных ресурсов, переходе на Oracle Cloud, реализации безопасности OCI, настройке баз данных OCI, развертывании контейнерных приложений в OKE, управлении ресурсами OCI или когда пользователи упоминают «Oracle Cloud», «OCI», «Автономную базу данных», «VCN», «OKE», «OCI Terraform», «Диспетчер ресурсов», «Облачная инфраструктура Oracle» или «Миграция OCI».
