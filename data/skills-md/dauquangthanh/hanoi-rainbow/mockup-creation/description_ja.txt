NuxtJS 4 (Vue) または Next.js (React)、TypeScript、TailwindCSS v4 を使用して、インタラクティブで本番環境に対応した UI モックアップとプロトタイプを作成します。 Web モックアップ、プロトタイプ、ランディング ページ、ダッシュボード、管理パネル、またはインタラクティブな UI デモを構築するときに使用します。ユーザーが「モックアップの作成」、「プロトタイプの構築」、「インタラクティブなデモ」、「UI プロトタイプ」、「デザインからコードまで」、または最新のツールを使用した迅速なフロントエンド開発が必要な場合にトリガーされます。 Vue ベースのプロジェクトには NuxtJS を推奨します。ユーザーが ReactJS について言及する場合は Next.js を使用します。
