Bietet umfassende Anleitung zur Google Cloud Platform (GCP), einschließlich Compute Engine, Cloud Storage, Cloud SQL, BigQuery, GKE (Google Kubernetes Engine), Cloud Functions, Cloud Run, VPC-Netzwerk, Lastausgleich, IAM, Cloud Build, Infrastruktur als Code (Terraform, Deployment Manager), Sicherheitskonfiguration, Kostenoptimierung und Bereitstellung in mehreren Regionen. Erstellt Infrastrukturcode, Bereitstellungsskripte, Konfigurationshandbücher und Architekturentwürfe. Verwendung bei der Bereitstellung in Google Cloud, beim Entwerfen der GCP-Infrastruktur, bei der Migration zur GCP, beim Konfigurieren von GCE-Instanzen, beim Einrichten von Cloud Storage, beim Verwalten von Cloud SQL-Datenbanken, beim Arbeiten mit BigQuery, beim Bereitstellen in GKE oder wenn Benutzer „Google Cloud“, „GCP“, „Compute Engine“, „Cloud Storage“, „BigQuery“, „GKE“, „Cloud Run“, „Cloud Functions“, „VPC“, „Cloud SQL“ oder „Google Cloud Platform“ erwähnen.
