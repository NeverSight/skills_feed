Предоставляет комплексное руководство по Google Cloud Platform (GCP), включая Compute Engine, Cloud Storage, Cloud SQL, BigQuery, GKE (Google Kubernetes Engine), Cloud Functions, Cloud Run, сети VPC, балансировку нагрузки, IAM, Cloud Build, инфраструктуру как код (Terraform, Deployment Manager), настройку безопасности, оптимизацию затрат и развертывание в нескольких регионах. Создает код инфраструктуры, сценарии развертывания, руководства по настройке и проекты архитектуры. Используйте при развертывании в Google Cloud, проектировании инфраструктуры GCP, переходе на GCP, настройке экземпляров GCE, настройке облачного хранилища, управлении базами данных Cloud SQL, работе с BigQuery, развертывании в GKE или когда пользователи упоминают «Google Cloud», «GCP», «Compute Engine», «Cloud Storage», «BigQuery», «GKE», «Cloud Run», «Cloud Functions», «VPC», «Cloud SQL» или «Google Cloud Platform».
