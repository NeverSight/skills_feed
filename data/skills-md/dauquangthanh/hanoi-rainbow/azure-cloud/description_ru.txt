Предоставляет комплексное руководство по Microsoft Azure, включая виртуальные машины Azure, хранилище Azure (блобы, файлы, диски), базу данных SQL Azure, службу приложений Azure, функции Azure, AKS (служба Azure Kubernetes), Azure DevOps, шаблоны ARM, Bicep, Terraform for Azure, Azure Active Directory, Azure Key Vault, Azure Monitor, оптимизацию затрат и развертывание в нескольких регионах. Создает инфраструктуру в виде кода (Terraform/Bicep/ARM), сценариев развертывания, конфигураций безопасности и проектов архитектуры. Используйте при развертывании в Azure, проектировании инфраструктуры Azure, миграции в Microsoft Azure, настройке виртуальных машин, настройке хранилища Azure, управлении SQL Azure, работе с AKS или когда пользователи упоминают Azure, Microsoft Cloud, Azure Portal, шаблоны ARM, Bicep, функции Azure, службу приложений или Azure DevOps.
