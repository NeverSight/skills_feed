يوفر إرشادات شاملة لـ Microsoft Azure بما في ذلك أجهزة Azure الافتراضية، وتخزين Azure (Blob، والملفات، والأقراص)، وقاعدة بيانات Azure SQL، وخدمة تطبيقات Azure، وAzure Functions، وAKS (خدمة Azure Kubernetes)، وAzure DevOps، وقوالب ARM، وBicep، وTerraform for Azure، وAzure Active Directory، وAzure Key Vault، وAzure Monitor، وتحسين التكلفة، والنشر متعدد المناطق. إنتاج البنية التحتية كرمز (Terraform/Bicep/ARM)، والبرامج النصية للنشر، وتكوينات الأمان، والتصميمات المعمارية. يُستخدم عند النشر إلى Azure، أو تصميم البنية الأساسية لـ Azure، أو الترحيل إلى Microsoft Azure، أو تكوين الأجهزة الافتراضية، أو إعداد Azure Storage، أو إدارة Azure SQL، أو العمل مع AKS، أو عندما يذكر المستخدمون Azure، أو Microsoft Cloud، أو Azure Portal، أو قوالب ARM، أو Bicep، أو Azure Functions، أو App Service، أو Azure DevOps.
