Fournit des conseils complets sur Microsoft Azure, notamment les machines virtuelles Azure, le stockage Azure (Blob, fichiers, disques), Azure SQL Database, Azure App Service, Azure Functions, AKS (Azure Kubernetes Service), Azure DevOps, les modèles ARM, Bicep, Terraform pour Azure, Azure Active Directory, Azure Key Vault, Azure Monitor, l'optimisation des coûts et le déploiement multirégional. Produit une infrastructure sous forme de code (Terraform/Bicep/ARM), des scripts de déploiement, des configurations de sécurité et des conceptions d'architecture. À utiliser lors du déploiement sur Azure, de la conception de l'infrastructure Azure, de la migration vers Microsoft Azure, de la configuration des machines virtuelles, de la configuration du stockage Azure, de la gestion d'Azure SQL, de l'utilisation d'AKS ou lorsque les utilisateurs mentionnent Azure, Microsoft Cloud, Azure Portal, les modèles ARM, Bicep, Azure Functions, App Service ou Azure DevOps.
