Azure Virtual Machines、Azure Storage (Blob、ファイル、ディスク)、Azure SQL Database、Azure App Service、Azure Functions、AKS (Azure Kubernetes Service)、Azure DevOps、ARM テンプレート、Bicep、Terraform for Azure、Azure Active Directory、Azure Key Vault、Azure Monitor、コストの最適化、マルチリージョン展開などの包括的な Microsoft Azure ガイダンスを提供します。コードとしてのインフラストラクチャ (Terraform/Bicep/ARM)、デプロイメント スクリプト、セキュリティ構成、およびアーキテクチャ設計を生成します。 Azure へのデプロイ、Azure インフラストラクチャの設計、Microsoft Azure への移行、VM の構成、Azure Storage のセットアップ、Azure SQL の管理、AKS の操作を行う場合、またはユーザーが Azure、Microsoft Cloud、Azure Portal、ARM テンプレート、Bicep、Azure Functions、App Service、または Azure DevOps について言及する場合に使用します。
