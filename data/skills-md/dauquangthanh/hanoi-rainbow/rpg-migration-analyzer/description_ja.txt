AS/400 および IBM i システムのレガシー RPG (レポート プログラム ジェネレーター) プログラムを分析して、最新の Java アプリケーションに移行します。 RPG III/IV/ILE ソース コードからビジネス ロジックを抽出し、データ構造 (D 仕様)、ファイル操作 (F 仕様)、プログラムの依存関係 (CALLB/CALLP) を識別し、RPG 構造を Java の同等のものに変換します。移行レポート、複雑さの見積もり、POJO クラス、JPA エンティティ、サービス メソッドを使用した Java 実装戦略を生成します。 AS/400 または IBM i レガシー システムの最新化、RPG ソース ファイル (.rpg、.rpgle、.RPGLE) の分析、RPG から Java への変換、データ仕様の Java クラスへのマッピング、レガシー システム移行の計画を立てる場合、またはユーザーが RPG 分析、レポート プログラム ジェネレーター、RPG III/IV/ILE、AS/400 の最新化、IBM i の移行、パック 10 進数変換、またはメインフレーム アプリケーションの書き換えに言及する場合に使用します。
