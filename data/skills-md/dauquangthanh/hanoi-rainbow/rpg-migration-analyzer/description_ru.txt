Анализирует устаревшие программы RPG (Report Program Generator) из систем AS/400 и IBM i на предмет миграции в современные приложения Java. Извлекает бизнес-логику из исходного кода RPG III/IV/ILE, идентифицирует структуры данных (D-спецификации), файловые операции (F-спецификации), зависимости программ (CALLB/CALLP) и преобразует конструкции RPG в эквиваленты Java. Создает отчеты о миграции, оценки сложности и стратегии реализации Java с помощью классов POJO, сущностей JPA и методов обслуживания. Используйте при модернизации устаревших систем AS/400 или IBM i, анализе исходных файлов RPG (.rpg, .rpgle, .RPGLE), преобразовании RPG в Java, сопоставлении спецификаций данных с классами Java, планировании миграции устаревших систем или когда пользователь упоминает анализ RPG, генератор программ отчетов, RPG III/IV/ILE, модернизацию AS/400, миграцию IBM i, преобразование упакованных десятичных чисел или переписывание приложений для мэйнфреймов.
