최신 Java 애플리케이션으로 마이그레이션하기 위해 AS/400 및 IBM i 시스템의 레거시 RPG(Report Program Generator) 프로그램을 분석합니다. RPG III/IV/ILE 소스 코드에서 비즈니스 로직을 추출하고, 데이터 구조(D-사양), 파일 작업(F-사양), 프로그램 종속성(CALLB/CALLP)을 식별하고 RPG 구성을 Java 해당 항목으로 변환합니다. POJO 클래스, JPA 엔터티 및 서비스 메서드를 사용하여 마이그레이션 보고서, 복잡성 추정 및 Java 구현 전략을 생성합니다. AS/400 또는 IBM i 레거시 시스템 현대화, RPG 소스 파일(.rpg, .rpgle, .RPGLE) 분석, RPG를 Java로 변환, 데이터 사양을 Java 클래스로 매핑, 레거시 시스템 마이그레이션 계획 또는 사용자가 RPG 분석, 보고서 프로그램 생성기, RPG III/IV/ILE, AS/400 현대화, IBM i 마이그레이션, 팩형 10진수 변환 또는 메인프레임 애플리케이션 재작성을 언급할 때 사용합니다.
