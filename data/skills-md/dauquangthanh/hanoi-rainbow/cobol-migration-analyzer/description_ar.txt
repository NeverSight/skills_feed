يحلل برامج COBOL القديمة ووظائف JCL للمساعدة في الترحيل إلى تطبيقات Java الحديثة. يستخرج منطق الأعمال ويحدد التبعيات وينشئ تقارير الترحيل وينشئ استراتيجيات تنفيذ Java. يُستخدم عند العمل مع ترحيل الحاسوب المركزي، أو تحليل COBOL، أو تحديث النظام القديم، أو سير عمل JCL، أو عندما يذكر المستخدمون تحويل COBOL إلى Java، أو تحليل ملفات .cbl/.CBL/.cob، أو العمل مع دفاتر النسخ، أو التخطيط لتطبيقات خدمة Java من برامج COBOL.
