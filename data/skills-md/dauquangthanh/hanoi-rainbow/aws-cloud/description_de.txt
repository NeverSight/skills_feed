Bietet umfassende AWS-Anleitungen (Amazon Web Services), einschließlich EC2, S3, RDS, Lambda, ECS/EKS, CloudFormation, API Gateway, CloudFront, Cloud-Migration von vor Ort/GCP/Azure, Sicherheitskonfiguration (IAM, KMS, Security Hub), Kostenoptimierung (Sparpläne, reservierte Instanzen) und Bereitstellung in mehreren Regionen. Erstellt Infrastruktur als Code (Terraform/CloudFormation/CDK), Bereitstellungsskripte, Sicherheitskonfigurationen und Architekturentwürfe. Verwendung bei der Bereitstellung in AWS, dem Entwerfen der AWS-Infrastruktur, der Migration zu AWS, der Konfiguration von EC2-Instanzen, der Einrichtung von S3-Buckets, der Verwaltung von RDS-Datenbanken, der Bereitstellung von Containern auf ECS/EKS, der Erstellung serverloser Anwendungen oder wenn Benutzer AWS-, Amazon Cloud-, EC2-, S3-, Lambda-, EKS-, CloudFormation-, CDK- oder AWS-Dienste erwähnen.
