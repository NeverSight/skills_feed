يوفر إرشادات AWS (Amazon Web Services) الشاملة بما في ذلك EC2 وS3 وRDS وLambda وECS/EKS وCloudFormation وAPI Gateway وCloudFront والترحيل السحابي من الموقع المحلي/GCP/Azure وتكوين الأمان (IAM وKMS وSecurity Hub)، وتحسين التكلفة (Savings Plans، والمثيلات المحجوزة)، والنشر متعدد المناطق. إنتاج البنية التحتية كرمز (Terraform/CloudFormation/CDK)، والبرامج النصية للنشر، وتكوينات الأمان، والتصميمات المعمارية. يُستخدم عند النشر إلى AWS، أو تصميم البنية التحتية لـ AWS، أو الترحيل إلى AWS، أو تكوين مثيلات EC2، أو إعداد مجموعات S3، أو إدارة قواعد بيانات RDS، أو نشر الحاويات على ECS/EKS، أو إنشاء تطبيقات بدون خادم، أو عندما يذكر المستخدمون خدمات AWS أو Amazon Cloud أو EC2 أو S3 أو Lambda أو EKS أو CloudFormation أو CDK أو AWS.
