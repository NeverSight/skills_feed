Предоставляет комплексное руководство по AWS (веб-сервисы Amazon), включая EC2, S3, RDS, Lambda, ECS/EKS, CloudFormation, API Gateway, CloudFront, миграцию в облако из локальной среды/GCP/Azure, настройку безопасности (IAM, KMS, Security Hub), оптимизацию затрат (планы экономии, зарезервированные экземпляры) и развертывание в нескольких регионах. Создает инфраструктуру в виде кода (Terraform/CloudFormation/CDK), сценариев развертывания, конфигураций безопасности и проектов архитектуры. Используйте при развертывании на AWS, проектировании инфраструктуры AWS, миграции на AWS, настройке экземпляров EC2, настройке сегментов S3, управлении базами данных RDS, развертывании контейнеров на ECS/EKS, создании бессерверных приложений или когда пользователи упоминают сервисы AWS, Amazon Cloud, EC2, S3, Lambda, EKS, CloudFormation, CDK или AWS.
