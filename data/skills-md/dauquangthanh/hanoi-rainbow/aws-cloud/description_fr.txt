Fournit des conseils AWS (Amazon Web Services) complets, notamment EC2, S3, RDS, Lambda, ECS/EKS, CloudFormation, API Gateway, CloudFront, la migration cloud depuis le site/GCP/Azure, la configuration de la sécurité (IAM, KMS, Security Hub), l'optimisation des coûts (plans d'économies, instances réservées) et le déploiement multirégional. Produit une infrastructure sous forme de code (Terraform/CloudFormation/CDK), des scripts de déploiement, des configurations de sécurité et des conceptions d'architecture. À utiliser lors du déploiement sur AWS, de la conception de l'infrastructure AWS, de la migration vers AWS, de la configuration des instances EC2, de la configuration des compartiments S3, de la gestion des bases de données RDS, du déploiement de conteneurs sur ECS/EKS, de la création d'applications sans serveur ou lorsque les utilisateurs mentionnent les services AWS, Amazon Cloud, EC2, S3, Lambda, EKS, CloudFormation, CDK ou AWS.
