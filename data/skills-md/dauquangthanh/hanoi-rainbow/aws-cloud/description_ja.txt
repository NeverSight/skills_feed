EC2、S3、RDS、Lambda、ECS/EKS、CloudFormation、API Gateway、CloudFront、オンプレミス/GCP/Azure からのクラウド移行、セキュリティ構成 (IAM、KMS、セキュリティ ハブ)、コストの最適化 (節約プラン、リザーブド インスタンス)、マルチリージョン展開などの包括的な AWS (アマゾン ウェブ サービス) ガイダンスを提供します。コードとしてのインフラストラクチャ (Terraform/CloudFormation/CDK)、デプロイメント スクリプト、セキュリティ構成、およびアーキテクチャ設計を生成します。 AWS へのデプロイ、AWS インフラストラクチャの設計、AWS への移行、EC2 インスタンスの構成、S3 バケットのセットアップ、RDS データベースの管理、ECS/EKS へのコンテナのデプロイ、サーバーレス アプリケーションの構築を行う場合、またはユーザーが AWS、Amazon Cloud、EC2、S3、Lambda、EKS、CloudFormation、CDK、または AWS のサービスについて言及する場合に使用します。
