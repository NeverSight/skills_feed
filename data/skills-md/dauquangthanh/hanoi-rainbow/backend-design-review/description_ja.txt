API 設計品質、データベース アーキテクチャ検証、マイクロサービス パターン評価、統合戦略評価、セキュリティ設計レビュー、スケーラビリティ分析を含む、包括的なバックエンド設計レビューを実施します。 API 仕様 (REST、GraphQL、gRPC)、データベース スキーマ、サービス境界、認証/認可フロー、キャッシュ戦略、メッセージ キュー、およびデプロイメント アーキテクチャを評価します。設計上の欠陥、セキュリティの脆弱性、パフォーマンスのボトルネック、およびスケーラビリティの問題を特定します。重大度別の調査結果、アーキテクチャ図、実装の推奨事項を含む詳細な設計レビュー レポートを作成します。バックエンド システム設計のレビュー、API 仕様の検証、データベース スキーマの評価、マイクロサービス アーキテクチャの評価、統合パターンのレビューを行う場合、またはユーザーがバックエンド設計レビュー、API 設計検証、データベース設計レビュー、マイクロサービス評価、バックエンド アーキテクチャ評価について言及する場合に使用します。
