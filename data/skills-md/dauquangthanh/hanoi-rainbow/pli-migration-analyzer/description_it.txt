Analizza i programmi PL/I (Programming Language One) legacy per assistere nella migrazione alle moderne applicazioni Java. Estrae logica aziendale, strutture dati, definizioni di procedure e operazioni sui file dal codice PL/I. Genera report sulla migrazione e crea strategie di implementazione Java. Da utilizzare quando si lavora con la migrazione del mainframe, l'analisi PL/I, la modernizzazione del sistema legacy o quando gli utenti menzionano la conversione da PL/I a Java, analizzano file .pli/.PLI/.pl1, lavorano con procedure PL/I o pianificano implementazioni di servizi Java da programmi PL/I.
