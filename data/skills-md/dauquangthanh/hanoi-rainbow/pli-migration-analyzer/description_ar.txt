يحلل برامج PL/I (لغة البرمجة الأولى) القديمة للمساعدة في الترحيل إلى تطبيقات Java الحديثة. يستخرج منطق الأعمال وهياكل البيانات وتعريفات الإجراءات وعمليات الملفات من كود PL/I. يُنشئ تقارير الترحيل وينشئ استراتيجيات تنفيذ Java. يُستخدم عند العمل مع ترحيل الحاسوب المركزي، أو تحليل PL/I، أو تحديث النظام القديم، أو عندما يذكر المستخدمون تحويل PL/I إلى Java، أو تحليل ملفات .pli/.PLI/.pl1، أو العمل مع إجراءات PL/I، أو التخطيط لتطبيقات خدمة Java من برامج PL/I.
