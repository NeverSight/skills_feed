従来の PL/I (Programming Language One) プログラムを分析して、最新の Java アプリケーションへの移行を支援します。ビジネス ロジック、データ構造、プロシージャ定義、およびファイル操作を PL/I コードから抽出します。移行レポートを生成し、Java 実装戦略を作成します。メインフレームの移行、PL/I 分析、レガシー システムの最新化を扱う場合、またはユーザーが PL/I から Java への変換、.pli/.PLI/.pl1 ファイルの分析、PL/I プロシージャの操作、または PL/I プログラムからの Java サービス実装の計画について言及する場合に使用します。
