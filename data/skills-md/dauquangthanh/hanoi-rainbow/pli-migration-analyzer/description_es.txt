Analiza programas PL/I (Programming Language One) heredados para ayudar con la migración a aplicaciones Java modernas. Extrae lógica empresarial, estructuras de datos, definiciones de procedimientos y operaciones de archivos del código PL/I. Genera informes de migración y crea estrategias de implementación de Java. Utilícelo cuando trabaje con migración de mainframe, análisis PL/I, modernización de sistemas heredados o cuando los usuarios mencionen la conversión de PL/I a Java, analicen archivos .pli/.PLI/.pl1, trabajen con procedimientos PL/I o planifiquen implementaciones de servicios Java desde programas PL/I.
