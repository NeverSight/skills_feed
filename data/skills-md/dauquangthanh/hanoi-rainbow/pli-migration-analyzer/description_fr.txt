Analyse les anciens programmes PL/I (Programming Language One) pour faciliter la migration vers des applications Java modernes. Extrait la logique métier, les structures de données, les définitions de procédures et les opérations sur les fichiers du code PL/I. Génère des rapports de migration et crée des stratégies de mise en œuvre Java. À utiliser lorsque vous travaillez avec une migration mainframe, une analyse PL/I, une modernisation de systèmes existants ou lorsque les utilisateurs mentionnent une conversion PL/I vers Java, analysent des fichiers .pli/.PLI/.pl1, travaillent avec des procédures PL/I ou planifient des implémentations de services Java à partir de programmes PL/I.
