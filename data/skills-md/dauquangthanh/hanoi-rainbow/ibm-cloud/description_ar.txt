يوفر إرشادات شاملة لمنصة IBM Cloud بما في ذلك خدمات الحوسبة (VPC، Virtual Servers، IKS، OpenShift، Code Engine، Cloud Functions)، التخزين (Object Storage، Block Storage، File Storage)، قواعد البيانات (Db2، Cloudant، PostgreSQL، MySQL، MongoDB، Redis)، أمان IAM (مجموعات الوصول، معرفات الخدمة، Key Protect، Secrets Manager)، الشبكات (VPC، موازنات التحميل، Direct Link)، أتمتة CLI، البنية التحتية Terraform/Schematics الكود والمراقبة وتحسين التكلفة. يغطي توفير البنية التحتية، ونشر التطبيقات، وتكوين الأمان، والتوفر العالي للمناطق المتعددة، وأفضل الممارسات التشغيلية. يُستخدم عند العمل مع خدمات IBM Cloud، أو نشر البنية التحتية السحابية، أو إدارة الموارد السحابية، أو تكوين الأمان والشبكات، أو عندما يذكر المستخدمون "IBM Cloud"، أو "IKS"، أو "Code Engine"، أو "Db2"، أو "Cloudant"، أو "VPC"، أو "توفير السحابة"، أو "IBM Kubernetes"، أو "OpenShift"، أو "Terraform IBM"، أو "Schematics"، أو "IBM cloud Platform".
