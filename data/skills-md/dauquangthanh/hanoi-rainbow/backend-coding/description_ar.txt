إرشادات تطوير الواجهة الخلفية المتخصصة التي تغطي Node.js وPython وJava وGo وتصميم واجهة برمجة التطبيقات (REST/GraphQL/gRPC) وأنماط قاعدة البيانات والمصادقة والتخزين المؤقت وقوائم انتظار الرسائل والخدمات الصغيرة والاختبار. يُنتج أكواد خلفية جاهزة للإنتاج وقابلة للتطوير وآمنة مع أفضل ممارسات الصناعة. يُستخدم عند إنشاء واجهات برمجة التطبيقات، أو تنفيذ منطق الأعمال، أو تصميم نماذج البيانات، أو دمج الخدمات، أو عندما يذكر المستخدمون تطوير الواجهة الخلفية، أو التعليمات البرمجية من جانب الخادم، أو واجهات برمجة التطبيقات، أو قواعد البيانات، أو الخدمات الصغيرة.
