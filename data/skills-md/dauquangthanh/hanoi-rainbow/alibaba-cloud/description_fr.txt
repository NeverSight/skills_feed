Fournit des conseils complets sur Alibaba Cloud (Aliyun), notamment ECS, ApsaraDB, OSS, SLB, VPC, RAM, ACK (Kubernetes), Function Compute, API Gateway, CDN et des services de surveillance. Couvre le provisionnement de l'infrastructure avec Terraform/ROS, la conception de l'architecture cloud, les meilleures pratiques de sécurité, l'optimisation des coûts et les stratégies de migration. Produit du code d'infrastructure, des scripts de déploiement, des diagrammes d'architecture et des procédures opérationnelles. À utiliser lorsque vous travaillez avec les services Alibaba Cloud, concevez une architecture cloud sur Aliyun, migrez vers Alibaba Cloud, configurez une infrastructure cloud chinoise, mettez en œuvre des déploiements multirégionaux en Chine ou lorsque les utilisateurs mentionnent Alibaba Cloud, Aliyun, ECS, OSS, ApsaraDB, ACK, RDS, SLB ou le cloud computing chinois.
