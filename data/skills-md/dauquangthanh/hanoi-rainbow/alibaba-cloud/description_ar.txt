يوفر إرشادات شاملة لـ Alibaba Cloud (Aliyun) بما في ذلك ECS وApsaraDB وOSS وSLB وVPC وRAM وACK (Kubernetes) وحساب الوظائف وبوابة API وCDN وخدمات المراقبة. يغطي توفير البنية التحتية باستخدام Terraform/ROS، وتصميم البنية السحابية، وأفضل ممارسات الأمان، وتحسين التكلفة، واستراتيجيات الترحيل. ينتج كود البنية التحتية والبرامج النصية للنشر ومخططات الهندسة المعمارية والإجراءات التشغيلية. يُستخدم عند العمل مع خدمات Alibaba Cloud، أو تصميم البنية السحابية على Aliyun، أو الانتقال إلى Alibaba Cloud، أو إعداد البنية التحتية السحابية الصينية، أو تنفيذ عمليات نشر متعددة المناطق في الصين، أو عندما يذكر المستخدمون Alibaba Cloud، أو Aliyun، أو ECS، أو OSS، أو ApsaraDB، أو ACK، أو RDS، أو SLB، أو الحوسبة السحابية الصينية.
