Предоставляет комплексное руководство по Alibaba Cloud (Aliyun), включая ECS, ApsaraDB, OSS, SLB, VPC, RAM, ACK (Kubernetes), Function Compute, API Gateway, CDN и услуги мониторинга. Охватывает предоставление инфраструктуры с помощью Terraform/ROS, проектирование облачной архитектуры, передовые методы обеспечения безопасности, оптимизацию затрат и стратегии миграции. Создает код инфраструктуры, сценарии развертывания, диаграммы архитектуры и рабочие процедуры. Используйте при работе с облачными сервисами Alibaba, проектировании облачной архитектуры на Aliyun, переходе на Alibaba Cloud, настройке китайской облачной инфраструктуры, реализации развертываний в нескольких регионах в Китае или когда пользователи упоминают Alibaba Cloud, Aliyun, ECS, OSS, ApsaraDB, ACK, RDS, SLB или китайские облачные вычисления.
