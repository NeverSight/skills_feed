Effectue des examens complets du code backend, y compris la conception des API (REST/GraphQL/gRPC), les modèles de base de données, l'authentification/autorisation, les stratégies de mise en cache, les files d'attente de messages, l'architecture des microservices, les vulnérabilités de sécurité et l'optimisation des performances pour Node.js, Python, Java, Go et C#. Produit des rapports d'examen détaillés avec des problèmes spécifiques, des évaluations de gravité et des recommandations concrètes. À utiliser lors de l'examen du code côté serveur, de l'analyse des implémentations d'API, de la vérification des requêtes de base de données, de la validation des flux d'authentification, de l'évaluation de l'architecture des microservices ou lorsque les utilisateurs mentionnent « examiner le code backend », « vérifier la conception de l'API », « analyser le code du serveur », « valider les modèles de base de données », « audit de sécurité », « examen des performances » ou « qualité du code backend ».
