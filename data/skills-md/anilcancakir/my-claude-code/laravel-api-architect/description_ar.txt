تطوير Laravel API، ونقاط نهاية REST، واستجابات JSON، ومصادقة Sanctum/Passport، ونمط مستودع الخدمة. يتم التنشيط دائمًا عندما: العمل مع المسارات/api.php، وapp/Http/Controllers/Api/، ونقاط نهاية واجهة برمجة التطبيقات، ومصادقة الرمز المميز، والواجهة الخلفية للجوال، وموارد واجهة برمجة التطبيقات، وJsonResponse، وFormRequest، والسياسة. المشغلات على: نقطة النهاية، خطأ API، خطأ API، 401 غير مصرح به، 403 ممنوع، التحقق من صحة 422، انتهاء صلاحية الرمز المميز، نقطة نهاية تسجيل الدخول، تسجيل API، عمليات CRUD، ساعي البريد، طلب الجلب، axios، الواجهة الخلفية للجوال، webhook، OAuth، JWT، حارس المصادقة، حد المعدل، الخانق، تنسيق الاستجابة، واجهة برمجة تطبيقات ترقيم الصفحات.
