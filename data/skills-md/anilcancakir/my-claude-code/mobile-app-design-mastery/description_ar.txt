تصميم واجهة مستخدم لتطبيقات الهاتف المحمول على مستوى الإنتاج استنادًا إلى مبادئ إعادة هيكلة واجهة المستخدم. يتم التنشيط دائمًا لـ: تطبيق Flutter، وتطبيق الهاتف المحمول، وتطبيق iOS، وتطبيق Android، وواجهة مستخدم الهاتف المحمول، وشاشات التطبيق، والتنقل عبر الهاتف المحمول، والأوراق السفلية، ونماذج الهاتف المحمول، وأهداف اللمس، وطباعة الهاتف المحمول، ونظام ألوان التطبيق، وبطاقات الهاتف المحمول، وطرق عرض القائمة، ونماذج الهاتف المحمول، وأشرطة علامات التبويب، وأشرطة التطبيقات، وأزرار الإجراءات العائمة. يوفر سير عمل التصميم المحمول، والتباعد المحسّن باللمس، ومقياس الطباعة المحمول، والأنماط المتوافقة مع النظام الأساسي. اللغة التركية: mobil uygulama tasarımı، Flutter tasarım، uygulama arayüzü، mobil UI، telefon uygulaması، Android tasarım، iOS tasarım. اللغة الإنجليزية: تصميم التطبيق، واجهة الهاتف المحمول، سهولة اللمس، الشعور الأصلي، تجربة مستخدم الهاتف المحمول.
