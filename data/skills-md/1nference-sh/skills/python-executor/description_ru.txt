Выполняйте код Python в безопасной изолированной среде через [inference.sh](https://inference.sh). Предустановленные: NumPy, Pandas, Matplotlib, Requests, BeautifulSoup, Selenium, Playwright, MoviePy, Pillow, OpenCV, Trimesh и еще более 100 библиотек. Использование для: обработки данных, очистки веб-страниц, манипулирования изображениями, создания видео, обработки 3D-моделей, создания PDF-файлов, вызовов API, сценариев автоматизации. Триггеры: python, выполнение кода, запуск сценария, парсинг веб-страниц, анализ данных, обработка изображений, редактирование видео, 3D-модели, автоматизация, pandas, matplotlib.
