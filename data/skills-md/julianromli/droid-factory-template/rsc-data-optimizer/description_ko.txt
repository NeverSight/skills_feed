느린 클라이언트 측을 변환하여 Next.js 앱 라우터 데이터 가져오기 최적화 
RSC(React Server Components)를 사용하여 빠른 서버 측 가져오기로 가져오기.

다음과 같은 경우에 사용하세요:
- 사용자가 로딩 스피너로 인해 초기 페이지 로드가 느리다고 보고합니다.
- 페이지는 데이터 가져오기를 위해 useEffect + useState를 사용합니다.
- StoreContext/useStore 패턴으로 인해 워터폴 가져오기가 발생합니다.
- SEO 개선 필요(초기 HTML에 없는 내용)
- "클라이언트 사용" 페이지를 서버 구성 요소로 변환
