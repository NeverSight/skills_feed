تحليل وتنفيذ الرسوم المتحركة الهادفة لواجهة المستخدم لمشاريع Next.js + Tailwind + React. يُستخدم عندما يطلب المستخدم إضافة رسوم متحركة أو تحسين حركة واجهة المستخدم أو تحريك الصفحات/المكونات أو تحسين التعليقات المرئية. مشغلات على "إضافة رسوم متحركة"، و"واجهة مستخدم متحركة"، و"تصميم الحركة"، و"تأثيرات التمرير"، و"رسوم متحركة للتمرير"، و"انتقالات الصفحة"، و"التفاعلات الدقيقة".
