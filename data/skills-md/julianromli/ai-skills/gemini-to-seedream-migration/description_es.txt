Migre la generación de imágenes de IA de Google Gemini 2.5 Flash a BytePlus SeeDream v4.5. 
Úselo cuando: (1) El usuario desee cambiar de Gemini a SeeDream/BytePlus para la generación de imágenes,
(2) El usuario pregunta sobre la migración de las API de generación de imágenes o la sustitución de Gemini por BytePlus.
(3) El usuario necesita optimización de costos o mejor calidad de imagen para las imágenes generadas por IA,
(4) El usuario menciona SeeDream, BytePlus o desea la migración de SDK a API REST para la generación de imágenes
