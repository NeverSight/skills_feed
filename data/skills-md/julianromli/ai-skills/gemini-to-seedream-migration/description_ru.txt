Перенесите генерацию изображений с помощью ИИ из Google Gemini 2.5 Flash в BytePlus SeeDream v4.5. 
Используйте, когда: (1) Пользователь хочет переключиться с Gemini на SeeDream/BytePlus для создания изображений,
(2) Пользователь спрашивает о переносе API создания изображений или замене Gemini на BytePlus,
(3) Пользователю необходима оптимизация затрат или улучшение качества изображений, созданных с помощью искусственного интеллекта.
(4) Пользователь упоминает SeeDream, BytePlus или хочет выполнить миграцию API SDK на REST для создания изображений.
