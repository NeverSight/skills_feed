Клонируйте/реплицируйте веб-сайты в готовый к использованию код Next.js 16 с помощью Firecrawl MCP.
Используйте, когда пользователь просит: клонировать веб-сайт, клонировать Vibe, скопировать целевую страницу, скопировать дизайн веб-сайта,
перестроить этот сайт, воссоздать эту страницу, клонировать определенные разделы (герой, цены, нижний колонтитул и т. д.).
Триггеры: «клонировать этот сайт», «клонировать vibe [url]», «реплицировать эту целевую страницу»,
«перестроить этот сайт в Next.js», «клонировать главный раздел из [url]», «скопировать этот дизайн».
