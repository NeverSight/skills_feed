Klonen/replizieren Sie Websites mit Firecrawl MCP in produktionsbereiten Next.js 16-Code.
Verwenden Sie diese Option, wenn der Benutzer Folgendes auffordert: Website klonen, Vibe-Klonen, Zielseite replizieren, Website-Design kopieren,
Erstellen Sie diese Website neu, erstellen Sie diese Seite neu, klonen Sie bestimmte Abschnitte (Hero, Preise, Fußzeile usw.).
Auslöser: „Diese Website klonen“, „Vibe-Klon [URL]“, „Diese Zielseite replizieren“,
„Diese Seite in Next.js neu erstellen“, „Heldenabschnitt von [URL] klonen“, „Dieses Design kopieren“.
