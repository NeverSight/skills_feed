Firecrawl MCP を使用して、Web サイトを本番環境に対応した Next.js 16 コードに複製/複製します。
ユーザーが次のことを要求する場合に使用します: Web サイトのクローン、バイブのクローン、ランディング ページの複製、Web サイトのデザインのコピー、
このサイトを再構築し、このページを再作成し、特定のセクション (ヒーロー、価格設定、フッターなど) を複製します。
トリガー: 「この Web サイトのクローンを作成」、「vibe clone [url]」、「このランディング ページを複製」、
「Next.js でこのサイトを再構築する」、「[URL] からヒーロー セクションを複製する」、「このデザインをコピーする」。
