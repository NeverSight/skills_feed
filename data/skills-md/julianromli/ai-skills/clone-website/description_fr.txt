Clonez/répliquez des sites Web dans du code Next.js 16 prêt pour la production à l'aide de Firecrawl MCP.
À utiliser lorsque l'utilisateur demande : cloner un site Web, cloner une ambiance, répliquer une page de destination, copier la conception d'un site Web,
reconstruire ce site, recréer cette page, cloner des sections spécifiques (héros, tarification, pied de page, etc.).
Déclencheurs : "cloner ce site Web", "cloner l'ambiance [url]", "répliquer cette page de destination",
"reconstruisez ce site dans Next.js", "clonez la section héros à partir de [url]", "copiez ce design".
