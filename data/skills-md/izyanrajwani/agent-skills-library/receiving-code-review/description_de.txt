Bewertet und reagiert auf eingehendes Code-Review-Feedback zu PRs (Rezensentenkommentare, angeforderte Änderungen), insbesondere wenn Vorschläge unklar, technisch fragwürdig oder den Umfang erweiternd sind. Nutzen Sie die Überprüfungsvorschläge vor der Implementierung, um sie auf die Absicht abzustimmen und die Änderungen minimal zu halten.
