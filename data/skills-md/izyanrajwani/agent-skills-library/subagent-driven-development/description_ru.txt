Последовательное выполнение субагента с двухэтапной проверкой планов реализации. Используйте при выполнении многозадачных планов в текущем сеансе, когда задачам необходим свежий контекст субагента, чтобы избежать загрязнения, когда между задачами требуются формальные циклы проверки (соответствие спецификациям, а затем качество кода) или когда вам нужна проверка каждой задачи на основе различий перед продолжением.
