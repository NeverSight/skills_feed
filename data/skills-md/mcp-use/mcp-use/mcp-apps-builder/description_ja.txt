**すべての MCP サーバー作業に必須** - mcp 使用フレームワークのベスト プラクティスとパターン。

**最初にお読みください** 次のような MCP サーバーが動作する前に
- 新しい MCP サーバーの作成
- 既存の MCP サーバーの変更 (ツール、リソース、プロンプト、ウィジェットの追加/更新)
- MCP サーバーの問題またはエラーのデバッグ
- 品質、セキュリティ、またはパフォーマンスに関して MCP サーバー コードをレビューする
- MCP 開発または MCP の使用パターンに関する質問に答える
-server.tool()、server.resource()、server.prompt()、またはウィジェットに何らかの変更を加える

このスキルには、重要なアーキテクチャ上の決定、セキュリティ パターン、および一般的な落とし穴が含まれています。
MCP 機能を実装する前に、必ず関連する参照ファイルを参照してください。
