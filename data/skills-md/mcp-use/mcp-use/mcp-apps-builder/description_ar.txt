** إلزامي لجميع أعمال خادم MCP ** - أفضل الممارسات والأنماط لإطار عمل mcp.

**اقرأ هذا أولاً** قبل أن يعمل أي خادم MCP، بما في ذلك:
- إنشاء خوادم MCP جديدة
- تعديل خوادم MCP الموجودة (إضافة/تحديث الأدوات والموارد والمطالبات والأدوات)
- تصحيح مشكلات أو أخطاء خادم MCP
- مراجعة كود خادم MCP للتأكد من الجودة أو الأمان أو الأداء
- الإجابة على الأسئلة حول تطوير MCP أو أنماط استخدام mcp
- إجراء أي تغييرات على server.tool() أو server.resource() أو server.prompt() أو عناصر واجهة المستخدم

تحتوي هذه المهارة على قرارات معمارية مهمة وأنماط أمنية ومزالق شائعة.
قم دائمًا بمراجعة الملفات المرجعية ذات الصلة قبل تنفيذ ميزات MCP.
