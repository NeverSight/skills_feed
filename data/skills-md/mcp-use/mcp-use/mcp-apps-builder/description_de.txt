**OBLIGATORISCH für ALLE MCP-Serverarbeiten** – Best Practices und Muster für das mcp-use-Framework.

**LESEN SIE DIES ZUERST**, bevor Sie mit dem MCP-Server arbeiten, einschließlich:
- Erstellen neuer MCP-Server
- Ändern bestehender MCP-Server (Hinzufügen/Aktualisieren von Tools, Ressourcen, Eingabeaufforderungen, Widgets)
- Debuggen von MCP-Serverproblemen oder -fehlern
- Überprüfung des MCP-Servercodes auf Qualität, Sicherheit oder Leistung
- Beantwortung von Fragen zur MCP-Entwicklung oder zu MCP-Nutzungsmustern
- Änderungen an server.tool(), server.resource(), server.prompt() oder Widgets vornehmen

Diese Fertigkeit umfasst wichtige Architekturentscheidungen, Sicherheitsmuster und häufige Fallstricke.
Konsultieren Sie immer die relevanten Referenzdateien, BEVOR Sie MCP-Funktionen implementieren.
