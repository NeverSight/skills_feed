**OBLIGATOIRE pour TOUS les travaux sur le serveur MCP** - meilleures pratiques et modèles de framework d'utilisation mcp.

**LISEZ CECI EN PREMIER** avant tout travail sur le serveur MCP, notamment :
- Création de nouveaux serveurs MCP
- Modification des serveurs MCP existants (ajout/mise à jour d'outils, ressources, invites, widgets)
- Débogage des problèmes ou des erreurs du serveur MCP
- Examen du code du serveur MCP pour la qualité, la sécurité ou les performances
- Répondre aux questions sur le développement de MCP ou les modèles d'utilisation de mcp
- Apporter TOUTE modification à server.tool(), server.resource(), server.prompt() ou aux widgets

Cette compétence contient des décisions d'architecture critiques, des modèles de sécurité et des pièges courants.
Consultez toujours les fichiers de référence pertinents AVANT de mettre en œuvre les fonctionnalités MCP.
