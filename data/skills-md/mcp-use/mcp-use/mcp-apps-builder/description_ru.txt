**ОБЯЗАТЕЛЬНО для ВСЕХ работ с сервером MCP** — лучшие практики и шаблоны использования платформы mcp.

**ПРОЧИТАЙТЕ ЭТО ПЕРВЫМ** перед началом работы с любым сервером MCP, включая:
- Создание новых серверов MCP.
- Модификация существующих серверов MCP (добавление/обновление инструментов, ресурсов, подсказок, виджетов)
- Отладка проблем или ошибок сервера MCP.
- Проверка кода сервера MCP на предмет качества, безопасности и производительности.
- Отвечаем на вопросы о разработке MCP или шаблонах использования mcp.
- Внесение ЛЮБЫХ изменений в server.tool(), server.resource(), server.prompt() или виджеты.

Этот навык содержит важные архитектурные решения, шаблоны безопасности и распространенные ошибки.
Всегда просматривайте соответствующие справочные файлы ПЕРЕД внедрением функций MCP.
