Créez des serveurs MCP (Model Context Protocol) avec le framework mcp-use.
À utiliser lors de la création de serveurs MCP, de la définition d'outils/ressources/invites, de l'utilisation de mcp-use,
lors du démarrage de projets MCP, ou lorsque l'utilisateur mentionne le développement, les outils, les ressources ou les invites MCP.
