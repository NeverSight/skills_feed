Инструменты CLI для поиска документации Svelte 5 и анализа кода. ДОЛЖЕН использоваться при создании, редактировании или анализе любого компонента Svelte (.svelte) или модуля Svelte (.svelte.ts/.svelte.js). Если возможно, для получения оптимальных результатов этот навык следует выполнять в агенте svelte-file-editor.
