Outils CLI pour la recherche de documentation et l'analyse de code Svelte 5. DOIT être utilisé lors de la création, de la modification ou de l'analyse d'un composant Svelte (.svelte) ou d'un module Svelte (.svelte.ts/.svelte.js). Si possible, cette compétence doit être exécutée au sein de l'agent svelte-file-editor pour des résultats optimaux.
