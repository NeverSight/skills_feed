Herramientas CLI para búsqueda de documentación y análisis de código de Svelte 5. DEBE usarse siempre que se cree, edite o analice cualquier componente Svelte (.svelte) o módulo Svelte (.svelte.ts/.svelte.js). Si es posible, esta habilidad debe ejecutarse dentro del agente editor de archivos esbeltos para obtener resultados óptimos.
