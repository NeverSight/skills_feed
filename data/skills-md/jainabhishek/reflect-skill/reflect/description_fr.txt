Compétence auto-améliorable qui analyse les conversations pour les corrections et les préférences, puis les conserve dans des fichiers de compétences avec une gestion de versions Git en option. À utiliser lorsque : (1) l'utilisateur exécute "/reflect" ou "/reflect [skill-name]" pour extraire manuellement les apprentissages, (2) l'utilisateur dit "reflect on" ou "reflect off" pour activer la réflexion automatique, (3) l'utilisateur dit "reflect status" pour vérifier le mode, (4) l'utilisateur souhaite que Claude se souvienne des corrections pour les sessions futures. Prend en charge les modes manuel et automatique (basé sur le crochet).
