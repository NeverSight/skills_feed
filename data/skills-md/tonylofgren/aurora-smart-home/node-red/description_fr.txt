À utiliser lorsque l'utilisateur mentionne « node-red » n'importe où dans sa demande (y compris des mots composés comme « node-redflöde », « node-red-flow »). Le terme « Node-RED » est un nom de produit qui apparaît inchangé dans toutes les langues. PAS pour : les automatisations YAML (utilisez la compétence home-assistant), le micrologiciel de l'appareil (utilisez la compétence esphome).
