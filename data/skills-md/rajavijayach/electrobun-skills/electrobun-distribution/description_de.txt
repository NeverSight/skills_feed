Verpackung, Codesignierung, Beglaubigung und Verteilung für Electrobun-Desktopanwendungen. Diese Fähigkeit umfasst das Erstellen von Produktionspaketen, das Erstellen von Installationsprogrammen und verteilbaren Paketen, das Signieren von Code für Windows und macOS, die Apple-Beglaubigung für Gatekeeper, die Implementierung von automatischen Updates, Delta-Updates, Update-Server, plattformübergreifende Build-Prozesse, CI/CD-Integration, App-Symbole und -Ressourcen, Versionsverwaltung, Release-Workflows, Windows SmartScreen-Anforderungen, macOS DMG-Erstellung, Linux-Paketformate (deb, rpm, AppImage) und Best Practices für die Verteilung. Verwenden Sie es, wenn Sie eine App für die Produktion vorbereiten, automatische Updates implementieren, Codesignaturzertifikate einrichten, Verteilungsprobleme beheben, Installationsprogramme erstellen, Update-Server konfigurieren, für mehrere Plattformen erstellen oder neue Versionen veröffentlichen. Zu den Auslösern gehören „Build“, „Package“, „Distribute“, „Code Sign“, „Notarize“, „Installer“, „Auto-Update“, „Release“, „Production Build“, „DMG“, „Updater“, „Delta Update“ oder „Certificate“.
