Electrobun デスクトップ アプリケーション用の高度なタイプセーフ RPC パターン。双方向の main ↔ Webview 通信、TypeScript によるタイプ セーフティ、エラー処理と検証、パフォーマンスの最適化、ストリーミング データ パターン、バッチ操作、再試行戦略、イベント ベースの通信、共有型定義、RPC ミドルウェア、要求/応答パターン、複雑なデータ同期をカバーします。このスキルは、メイン プロセスと Webview プロセス間の複雑な通信を実装する場合、完全な IntelliSense を使用したタイプ セーフ RPC が必要な場合、大規模なデータ転送の処理、リアルタイム更新の実装、プロセス間のタイプ セーフ API の構築、RPC の問題のデバッグ、RPC パフォーマンスの最適化、またはストリーミング、バッチ処理、パブリッシュ/サブスクライブなどの高度なパターンの実装を行う場合に使用します。トリガーには、「RPC」、「メイン Web ビュー通信」、「タイプ セーフ RPC」、「双方向 RPC」、「RPC パフォーマンス」、「RPC エラー処理」、「共有タイプ」、「プロセス通信」、または「IPC パターン」が含まれます。
