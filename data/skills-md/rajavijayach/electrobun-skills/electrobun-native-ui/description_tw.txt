Electrobun 桌面應用程式的本機 UI 集成，包括 ApplicationMenu、ContextMenu、系統托盤、本機對話框、鍵盤快捷鍵和特定於平台的 UI 模式。此技能涵蓋建立具有子選單和加速器的應用程式選單、右鍵單擊觸發的上下文選單、帶有選單的系統托盤圖示、文件/資料夾對話框、訊息框、通知系統、全域鍵盤快捷鍵、選單項目角色、動態選單更新、特定於平台的選單約定（macOS 選單列、Windows 系統選單）、拖放整合和本機主題。在實作應用程式選單、新增系統托盤功能、建立上下文選單、顯示檔案選擇器、實作鍵盤快速鍵、顯示通知或對話方塊或建置平臺本機 UI 體驗時使用。觸發器包括「選單」、「托盤圖示」、「上下文選單」、「檔案對話方塊」、「捷徑」、「加速器」、「本機對話方塊」、「系統托盤」、「通知」、「功能表列」或「右鍵選單」。
