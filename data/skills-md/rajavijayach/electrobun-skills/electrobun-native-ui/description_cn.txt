Electrobun 桌面应用程序的本机 UI 集成，包括 ApplicationMenu、ContextMenu、系统托盘、本机对话框、键盘快捷键和特定于平台的 UI 模式。该技能涵盖创建带有子菜单和加速器的应用程序菜单、右键单击触发的上下文菜单、带有菜单的系统托盘图标、文件/文件夹对话框、消息框、通知系统、全局键盘快捷键、菜单项角色、动态菜单更新、特定于平台的菜单约定（macOS 菜单栏、Windows 系统菜单）、拖放集成和本机主题。在实现应用程序菜单、添加系统托盘功能、创建上下文菜单、显示文件选择器、实现键盘快捷键、显示通知或对话框或构建平台本机 UI 体验时使用。触发器包括“菜单”、“托盘图标”、“上下文菜单”、“文件对话框”、“快捷方式”、“加速器”、“本机对话框”、“系统托盘”、“通知”、“菜单栏”或“右键菜单”。
