تكامل واجهة المستخدم الأصلية لتطبيقات سطح المكتب Electrobun بما في ذلك ApplicationMenu وContextMenu وعلبة النظام ومربعات الحوار الأصلية واختصارات لوحة المفاتيح وأنماط واجهة المستخدم الخاصة بالنظام الأساسي. تغطي هذه المهارة إنشاء قوائم التطبيقات مع القوائم الفرعية والمسرعات، وقوائم السياق التي يتم تشغيلها عن طريق النقر بزر الماوس الأيمن، وأيقونات علبة النظام مع القوائم، ومربعات حوار الملفات/المجلدات، ومربعات الرسائل، وأنظمة الإشعارات، واختصارات لوحة المفاتيح العامة، وأدوار عناصر القائمة، وتحديثات القائمة الديناميكية، واصطلاحات القائمة الخاصة بالنظام الأساسي (شريط قوائم macOS، وقائمة نظام Windows)، وتكامل السحب والإفلات، والسمات الأصلية. يُستخدم عند تنفيذ قوائم التطبيقات، أو إضافة وظائف علبة النظام، أو إنشاء قوائم السياق، أو إظهار منتقي الملفات، أو تنفيذ اختصارات لوحة المفاتيح، أو عرض الإشعارات أو مربعات الحوار، أو إنشاء تجارب واجهة مستخدم أصلية للنظام الأساسي. تتضمن المشغلات "القائمة"، أو "رمز الدرج"، أو "قائمة السياق"، أو "مربع حوار الملف"، أو "الاختصارات"، أو "المسرّع"، أو "الحوار الأصلي"، أو "علبة النظام"، أو "الإشعارات"، أو "شريط القوائم"، أو "قائمة النقر بزر الماوس الأيمن".
