ApplicationMenu、ContextMenu、システム トレイ、ネイティブ ダイアログ、キーボード ショートカット、プラットフォーム固有の UI パターンなどの Electrobun デスクトップ アプリケーションのネイティブ UI 統合。このスキルでは、サブメニューとアクセラレータを備えたアプリケーション メニュー、右クリックによってトリガーされるコンテキスト メニュー、メニュー付きのシステム トレイ アイコン、ファイル/フォルダー ダイアログ、メッセージ ボックス、通知システム、グローバル キーボード ショートカット、メニュー項目の役割、動的メニュー更新、プラットフォーム固有のメニュー規則 (macOS メニュー バー、Windows システム メニュー)、ドラッグ アンド ドロップ統合、およびネイティブ テーマの作成をカバーします。アプリケーション メニューの実装、システム トレイ機能の追加、コンテキスト メニューの作成、ファイル ピッカーの表示、キーボード ショートカットの実装、通知やダイアログの表示、またはプラットフォーム ネイティブ UI エクスペリエンスの構築時に使用します。トリガーには、「メニュー」、「トレイ アイコン」、「コンテキスト メニュー」、「ファイル ダイアログ」、「ショートカット」、「アクセラレータ」、「ネイティブ ダイアログ」、「システム トレイ」、「通知」、「メニュー バー」、または「右クリック メニュー」が含まれます。
