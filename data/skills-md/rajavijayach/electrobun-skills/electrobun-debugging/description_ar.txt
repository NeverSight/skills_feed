تطوير سير العمل وتصحيح الأخطاء واستكشاف الأخطاء وإصلاحها لتطبيقات Electrobun لسطح المكتب. تغطي هذه المهارة تصحيح أخطاء العملية الرئيسية (Bun) وعمليات عرض الويب، وتكامل Chrome DevTools، واستراتيجيات تسجيل وحدة التحكم، ومعالجة الأخطاء، وملفات تعريف الأداء، واكتشاف تسرب الذاكرة، واستكشاف أخطاء البناء وإصلاحها، وأخطاء وقت التشغيل الشائعة، وإعداد بيئة التطوير، وتكوين إعادة التحميل السريع، وخرائط المصدر، وتصحيح أخطاء نقاط التوقف، وفحص الشبكة، وتصحيح أخطاء WebView على منصات مختلفة، وتصحيح أخطاء الوحدة الأصلية، وأساليب تصحيح الأخطاء المنهجية. يُستخدم عند مواجهة حالات فشل البناء، أو أخطاء وقت التشغيل، أو الأعطال، أو مشكلات الأداء، أو تصحيح أخطاء اتصالات RPC، أو فحص Webview DOM، أو تحديد ملفات تعريف استخدام وحدة المعالجة المركزية/الذاكرة، أو استكشاف المشكلات الخاصة بالنظام الأساسي وإصلاحها، أو إعداد سير عمل التطوير. تتضمن المشغلات "التصحيح" أو "الخطأ" أو "التعطل" أو "استكشاف الأخطاء وإصلاحها" أو "أدوات تطوير البرامج" أو "الفحص" أو "نقطة التوقف" أو "التوصيف" أو "مشكلة في الأداء" أو "خطأ في البناء" أو "لا يعمل" أو "التسجيل".
