قم بتطوير عمليات تكامل Home Assistant المخصصة وتدفقات التكوين والكيانات والأنظمة الأساسية. يُستخدم عند العمل مع البيان.json، أو المكونات المخصصة، أو config_flow.py، أو الفئات الأساسية للكيانات، أو سجل الجهاز. يتم التنشيط بناءً على الكلمات الرئيسية: التكامل، والمكون المخصص، وتدفق التكوين، والكيان، والنظام الأساسي، وManifest.json، وdevice_info.
