Разрабатывайте надстройки Home Assistant с помощью Docker, Supervisor API и многоархитектурных сборок. Используйте при создании надстроек, настройке Dockerfiles, настройке входа или публикации в репозитории. Активируется по ключевым словам: надстройка, аддон, супервизор, hassio, ingress, bashio, docker.
