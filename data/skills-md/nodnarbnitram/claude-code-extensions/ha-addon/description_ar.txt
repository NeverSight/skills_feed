قم بتطوير الوظائف الإضافية لـ Home Assistant باستخدام Docker وSupervisor API والإصدارات متعددة الأقواس. يُستخدم عند إنشاء الوظائف الإضافية، أو تكوين ملفات Dockerfiles، أو إعداد الدخول، أو النشر إلى المستودعات. يتم التنشيط عند الكلمات الرئيسية: الوظيفة الإضافية، الملحق، المشرف، هاسيو، الدخول، باشيو، عامل الإرساء.
