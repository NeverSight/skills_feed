Erstellen Sie ein Gerüst und automatisieren Sie Grafana-Plugin-Projekte mit @grafana/create-plugin. Zum Erstellen von Panel-Plugins, Datenquellen-Plugins, App-Plugins oder Backend-Plugins. Übernimmt die Projektgerüsterstellung, die Einrichtung der Docker-Entwicklungsumgebung und die Plugin-Konfiguration.
