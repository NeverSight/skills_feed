سقالة وأتمتة مشاريع البرنامج المساعد Grafana باستخدام @grafana/create-plugin. يُستخدم عند إنشاء المكونات الإضافية للوحة، أو المكونات الإضافية لمصدر البيانات، أو المكونات الإضافية للتطبيقات، أو المكونات الإضافية للواجهة الخلفية. يتعامل مع سقالات المشروع وإعداد بيئة تطوير Docker وتكوين المكونات الإضافية.
