المساعدة في تفاعلات Kubernetes بما في ذلك تصحيح الأخطاء (سجلات kubectl، والوصف، وexec، وإعادة توجيه المنفذ)، وإدارة الموارد (عمليات النشر، والخدمات، وخرائط التكوين، والأسرار)، وعمليات المجموعة (القياس، والطرح، وإدارة العقدة). يُستخدم عند العمل مع kubectl أو pods أو عمليات النشر أو الخدمات أو استكشاف مشكلات Kubernetes وإصلاحها.
