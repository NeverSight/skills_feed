Développement d'applications multiplateformes Tauri v2 avec backend Rust. À utiliser lors de la configuration de tauri.conf.json, de l'implémentation de commandes Rust (#[tauri::command]), de la configuration de modèles IPC (invocation, émission, canaux), de la configuration des autorisations/capacités, du dépannage des problèmes de build ou du déploiement d'applications de bureau/mobiles. Déclencheurs sur Tauri, src-tauri, invoquer, émettre, capacités.json.
