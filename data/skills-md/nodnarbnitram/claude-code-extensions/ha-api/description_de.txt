Integrieren Sie die REST- und WebSocket-APIs von Home Assistant. Verwenden Sie diese Option, wenn Sie API-Aufrufe durchführen, Entitätszustände verwalten, Dienste aufrufen, Ereignisse abonnieren oder die Authentifizierung einrichten. Wird mit den Schlüsselwörtern REST API, WebSocket, API-Endpunkt, Serviceaufruf, Zugriffstoken, Inhabertoken und subscribe_events aktiviert.
