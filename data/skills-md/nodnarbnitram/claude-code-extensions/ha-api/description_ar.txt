التكامل مع Home Assistant REST وWebSocket APIs. يُستخدم عند إجراء مكالمات API، أو إدارة حالات الكيان، أو الاتصال بالخدمات، أو الاشتراك في الأحداث، أو إعداد المصادقة. يتم التنشيط على الكلمات الرئيسية REST API وWebSocket ونقطة نهاية API ومكالمة الخدمة ورمز الوصول ورمز Bearer المميز وsubscribe_events.
