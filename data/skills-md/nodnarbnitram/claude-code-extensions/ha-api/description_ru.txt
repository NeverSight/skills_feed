Интеграция с Home Assistant REST и API WebSocket. Используйте при выполнении вызовов API, управлении состояниями объектов, вызове служб, подписке на события или настройке аутентификации. Активируется по ключевым словам REST API, WebSocket, конечная точка API, вызов службы, токен доступа, токен носителя, subscribe_events.
