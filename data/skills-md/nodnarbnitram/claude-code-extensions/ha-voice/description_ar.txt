قم بتكوين Home Assistant لمساعدة التحكم الصوتي من خلال المسارات والأهداف وكلمات التنبيه ومعالجة الكلام. يُستخدم عند إعداد التحكم الصوتي، أو إنشاء أهداف مخصصة، أو تكوين TTS/STT، أو إنشاء أقمار صناعية صوتية. يتم التنشيط عند الكلمات الرئيسية: المساعدة، التحكم الصوتي، كلمة التنبيه، الهدف، الجملة، TTS، STT، Piper، Whisper، Wyoming.
