Лучшие практики и рекомендации по моделированию авторизации OpenFGA. Этот навык следует использовать при разработке, проверке или рефакторинге моделей авторизации OpenFGA. Запускается по задачам, связанным с моделями OpenFGA, определениями отношений, структурами разрешений, файлами .fga, тестовыми файлами .fga.yaml или использованием OpenFGA SDK в JavaScript, TypeScript, Go, Python, Java или .NET.
