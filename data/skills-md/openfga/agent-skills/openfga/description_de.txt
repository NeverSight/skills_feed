Best Practices und Richtlinien für die OpenFGA-Autorisierungsmodellierung. Diese Fähigkeit sollte beim Verfassen, Überprüfen oder Refactoring von OpenFGA-Autorisierungsmodellen verwendet werden. Wird bei Aufgaben ausgelöst, die OpenFGA-Modelle, Beziehungsdefinitionen, Berechtigungsstrukturen, .fga-Dateien, .fga.yaml-Testdateien oder die Verwendung des OpenFGA SDK in JavaScript, TypeScript, Go, Python, Java oder .NET betreffen.
