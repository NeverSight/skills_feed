Создает оценщики на основе кода для агентов, отслеживаемых с помощью LangSmith. Используйте при построении пользовательской логики оценки, шаблонах использования инструментов тестирования или программной оценке выходных данных агента. Запускается по запросам на оценку агентов, создание оценщиков или проведение экспериментов с наборами данных LangSmith.
