Fähigkeit zur Sicherheitsüberprüfung für React Native-Anwendungen. Verwenden Sie es, wenn Sie Code auf Schwachstellen überprüfen, durchgesickerte Geheimnisse (API-Schlüssel, Token, Anmeldeinformationen) erkennen, offengelegte persönliche Daten (PII) identifizieren, unsicheren Speicher überprüfen, Authentifizierungsabläufe validieren, die Netzwerksicherheit überprüfen und die Einhaltung von Best Practices für die mobile Sicherheit (OWASP MASVS) sicherstellen. Deckt sowohl JavaScript/TypeScript als auch nativen iOS/Android-Code ab.
