Conseils d'experts pour créer des applications mobiles avec React Native et Expo. À utiliser lorsque l'utilisateur demande à créer, modifier, déboguer ou concevoir des applications mobiles, implémenter des fonctionnalités natives (caméra, notifications, stockage, navigation), configurer des projets React Native, travailler avec Expo ou des flux de travail React Native, intégrer des API d'appareil, gérer la gestion de l'état des applications ou optimiser les performances mobiles. Déclencheurs sur le développement d'applications mobiles, React Native, Expo, développement multiplateforme iOS/Android.
