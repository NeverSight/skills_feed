Kompetente Anleitung zum Erstellen mobiler Anwendungen mit React Native und Expo. Wird verwendet, wenn der Benutzer mobile Apps erstellen, ändern, debuggen oder entwerfen, native Funktionen (Kamera, Benachrichtigungen, Speicher, Navigation) implementieren, React Native-Projekte einrichten, mit Expo oder reinen React Native-Workflows arbeiten, Geräte-APIs integrieren, die App-Statusverwaltung verwalten oder die mobile Leistung optimieren möchte. Auslöser für die Entwicklung mobiler Apps, React Native, Expo und plattformübergreifende iOS/Android-Entwicklung.
