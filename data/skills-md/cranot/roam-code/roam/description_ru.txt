Понимание кодовой базы через CLI перемещаемого кода. Используйте при изучении баз кода, планировании изменений, отладке сбоев, оценке PR-риска или проверке работоспособности архитектуры. Запускается при: понимании структуры проекта, проверках безопасности перед изменениями, поиске символов/файлов, анализе радиуса взрыва, затронутых тестах, оценке работоспособности, руководстве по рефакторингу, проверке кода. Требуется установленный роум-код («pip install roam-code») и проиндексированный проект («roam init»).
