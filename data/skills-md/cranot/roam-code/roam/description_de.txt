Codebasis-Verständnis über Roam-Code-CLI. Verwenden Sie es, wenn Sie Codebasen erkunden, Änderungen planen, Fehler debuggen, PR-Risiken bewerten oder den Zustand der Architektur überprüfen. Auslöser sind: Verständnis der Projektstruktur, Sicherheitsüberprüfungen vor Änderungen, Auffinden von Symbolen/Dateien, Analyse des Explosionsradius, betroffene Tests, Gesundheitsbewertung, Refactoring-Anleitung, Codeüberprüfung. Erfordert die Installation von Roam-Code („pip install roam-code“) und ein indiziertes Projekt („roam init“).
