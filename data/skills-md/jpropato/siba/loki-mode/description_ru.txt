Мультиагентная автономная система запуска для Claude Code. Запускается в «Режиме Локи». Управляет более чем 100 специализированными агентами, занимающимися разработкой, контролем качества, DevOps, безопасностью, данными и машинным обучением, бизнес-операциями, маркетингом, кадрами и успехом клиентов. Превращает PRD в полностью развернутый, приносящий доход продукт без вмешательства человека. Функции: Инструмент задач для отправки субагентов, параллельная проверка кода с участием трех специализированных рецензентов, сортировка проблем по степени серьезности, распределенная очередь задач с обработкой недоставленных писем, автоматическое развертывание среди облачных провайдеров, A/B-тестирование, циклы обратной связи с клиентами, реагирование на инциденты, автоматические выключатели и самовосстановление. Управляет ограничениями скорости через распределенные контрольные точки состояния и автоматически возобновляет работу с экспоненциальной задержкой. Требуется флаг --dangerously-skip-permissions.
