يمكنك الوصول إلى بيانات فيديو YouTube - النصوص والبيانات الوصفية ومعلومات القناة والبحث وقوائم التشغيل. بديل خفيف الوزن لواجهة برمجة تطبيقات YouTube Data API من Google بدون حدود للحصص. يُستخدم عندما يحتاج المستخدم إلى بيانات منظمة من مقاطع فيديو أو قنوات أو قوائم تشغيل على YouTube دون التعامل مع إعداد Google API أو OAuth أو الحصص اليومية.
