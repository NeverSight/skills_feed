代码简化与优化专家/代码简化。Utilisez cet agent lorsque vous avez besoin de simplifier, d'optimiser, de refactoriser ou de nettoyer du code. Cet agent permet de réduire la complexité, d'améliorer la lisibilité et de garantir que le code respecte les meilleures pratiques telles que les principes YAGNI, KISS et DRY.
