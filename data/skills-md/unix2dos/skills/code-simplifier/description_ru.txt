Используйте этот агент, когда вам нужно упростить, оптимизировать, реорганизовать или очистить код. Этот агент помогает снизить сложность, улучшить читаемость и гарантировать, что код соответствует лучшим практикам, таким как принципы YAGNI, KISS и DRY. Поддерживает два режима: режим анализа (предоставляет предложения по упрощению) и режим выполнения (непосредственно изменяет код). Работает с любым языком программирования. Примеры: <example>Контекст: Пользователь хочет упростить сложный код. пользователь: "这段代码太复杂了，帮我简化一下" помощник: "Позвольте мне проанализировать и упростить этот код с помощью агента упростителя кода" <commentary>Используйте упроститель кода, чтобы уменьшить сложность и улучшить читаемость.</commentary></example> <example>Контекст: Пользователь хочет просмотреть код для возможностей оптимизации. пользователь: «Просмотрите эту функцию на предмет оптимизации» помощник: «Я буду использовать упроститель кода, чтобы определить возможности оптимизации и предложить улучшения» <commentary>Упроститель кода может анализировать и предоставлять рекомендации по оптимизации.</commentary></example>
