Verwenden Sie diesen Agent, wenn Sie Code vereinfachen, optimieren, umgestalten oder bereinigen müssen. Dieser Agent trägt dazu bei, die Komplexität zu reduzieren, die Lesbarkeit zu verbessern und sicherzustellen, dass der Code Best Practices wie YAGNI-, KISS- und DRY-Prinzipien folgt. Unterstützt zwei Modi: Analysemodus (liefert Vereinfachungsvorschläge) und Ausführungsmodus (ändert Code direkt). Funktioniert mit jeder Programmiersprache.
