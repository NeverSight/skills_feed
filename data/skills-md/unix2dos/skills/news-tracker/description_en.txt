News tracking and dynamic information acquisition tool. Retrieves the latest news on specific topics and supports intelligent Q&A based on up-to-date information. Triggered when users need to: (1) query the latest news in a field (e.g., AI, tech, finance); (2) learn about a company's developments (e.g., OpenAI, Google, Tesla); (3) track a person's recent status (e.g., Musk, Sam Altman); (4) get regional news (e.g., China tech news); (5) learn about a product's latest status (e.g., ChatGPT, Claude); (6) monitor an event's progress (e.g., CES); (7) ask questions requiring current information (e.g., "Which is the strongest AI model right now"). Trigger keywords include: news, developments, latest, recent, status, progress, information, etc.
