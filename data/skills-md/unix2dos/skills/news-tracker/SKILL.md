# News Tracker

获取特定主题的最新新闻和动态信息。

## 执行流程

```
1. 解析用户查询 → 识别主题类型 + 判断意图
2. 构造搜索查询 → 执行 2-3 次 search_web
3. 去重并评分 → 按相关性排序
4. 格式化输出 → 新闻列表 或 智能问答
```

## Step 1: 主题类型识别

解析用户查询，识别主题类型：

| 类型 | 识别特征 | 搜索关键词模式 |
|------|----------|----------------|
| **领域** | 行业/学科名词 | `{主题} 新闻 {时间}` + `{主题} latest news` |
| **公司** | 企业/组织名称 | `{公司名} 新闻` + `{公司英文名} news` |
| **人物** | 人名 | `{人名} 最新动态` + `{人名英文} latest` |
| **地区** | 国家/地区名 | `{地区} {领域}新闻 {时间}` |
| **产品** | 产品/服务名称 | `{产品名} 更新` + `{产品名} release` |
| **事件** | 活动/事件名 | `{事件名} {年份}` + `{事件英文名}` |

## Step 2: 意图识别

判断用户意图，决定输出格式：

| 意图 | 识别特征 | 输出格式 |
|------|----------|----------|
| **新闻聚合** | 含"新闻/动态/资讯"等词，无疑问句式 | 新闻列表（10 条） |
| **智能问答** | 疑问句，问"是什么/哪个/怎么样/多少" | 先回答问题，附信息来源 |

## Step 3: 搜索执行

### 3.1 时间范围（硬性规则）

| 主题特性 | 默认范围 | 用户指定时优先用户要求 |
|----------|----------|------------------------|
| 热点人物/公司 | 7 天 | ✅ |
| 行业领域 | 14 天 | ✅ |
| 特定事件 | 事件前后 1 周 | ✅ |
| 技术产品 | 14 天 | ✅ |

### 3.2 搜索策略（必须执行）

**执行 2-3 次 `search_web` 调用**，覆盖中英文和不同角度：

```
第 1 次搜索：{主题} 新闻 {当前月份}
第 2 次搜索：{主题英文} latest news {year}
第 3 次搜索（可选）：{主题} {相关子领域} 动态
```

**示例 - 用户查询 "AI 最新新闻"：**
```
search_web("AI 新闻 2026年1月")
search_web("artificial intelligence latest news 2026")
search_web("AI 大模型 发布")
```

### 3.3 结果处理

1. **去重**：合并来自不同搜索的相同新闻（URL 或标题相似度 > 80%）
2. **排序**：按相关性评分降序排列
3. **数量**：保留 **10 条**（结果不足时保留全部）

## Step 4: 相关性评分标准

| 星级 | 标准 |
|------|------|
| ⭐⭐⭐⭐⭐ | 主题完全匹配 + 来源权威 + 3天内 |
| ⭐⭐⭐⭐ | 主题匹配 + 来源可靠 + 7天内 |
| ⭐⭐⭐ | 主题相关 + 来源一般 或 超过7天 |
| ⭐⭐ | 主题部分相关 或 来源存疑 |
| ⭐ | 勉强相关，仅作补充参考 |

**权威来源列表**（优先采用）：
- 科技类：TechCrunch、The Verge、Wired、36氪、虎嗅、极客公园
- 综合类：Reuters、BBC、新华网、财联社、界面新闻
- AI专业：arXiv、Hacker News、机器之心、量子位

## 输出格式

### 新闻列表格式

```markdown
# {主题} 最新新闻

> 📅 时间范围：{起始日期} ~ {结束日期} | 共 {N} 条

---

### 1. {新闻标题}

- **摘要**：{2-3句话的新闻摘要}
- **来源**：{来源网站名称}
- **时间**：{YYYY-MM-DD}
- **相关性**：{⭐评分}
- **链接**：[阅读原文]({URL})

### 2. ...
```

### 智能问答格式

```markdown
## 回答

{基于最新信息的直接回答，1-3 段}

## 关键数据

| 指标 | 数值 | 来源 |
|------|------|------|
| ... | ... | ... |

## 信息来源

1. [{新闻标题} - {来源}]({URL})
2. ...
```

## 异常处理

| 情况 | 处理方式 |
|------|----------|
| **0 条结果** | 提示无结果 + 给出 2-3 个替代关键词建议 |
| **< 5 条结果** | 正常输出 + 提示"结果较少，可扩大范围" |
| **搜索工具失败** | 提示网络问题，建议稍后重试 |
| **结果全为旧闻（>30天）** | 提示"近期无新动态" + 展示最近的 3 条 |
