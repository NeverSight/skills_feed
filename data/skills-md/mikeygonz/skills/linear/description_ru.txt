Управляйте заявками Linear с помощью своего ИИ-агента — читайте, создавайте, обновляйте и сортируйте проблемы.
Автоматически извлекает связанные страницы исследований Notion для полного контекста дизайна.

Используйте, когда:
- Пользователь указывает идентификатор заявки (например, «ENG-123»).
- Пользователь спрашивает «что у меня на борту» или «покажи мои билеты».
- Пользователь хочет создать новую задачу
- Пользователь просит обновить статус/приоритет заявки.
