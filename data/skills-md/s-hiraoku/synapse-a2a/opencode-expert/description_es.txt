Esta habilidad proporciona una guía completa para usar OpenCode, el agente de codificación de IA de código abierto. Utilice esta habilidad cuando trabaje con comandos de OpenCode CLI, atajos de teclado, agentes (compilación/planificación), comandos de barra diagonal, herramientas, habilidades, servidores MCP o configuración. Se activa automáticamente cuando se detectan preguntas o tareas específicas de OpenCode.
