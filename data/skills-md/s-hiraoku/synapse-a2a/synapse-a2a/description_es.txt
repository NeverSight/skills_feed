Esta habilidad proporciona una guía integral para la comunicación entre agentes utilizando el marco Synapse A2A. Utilice esta habilidad al enviar mensajes a otros agentes, enrutar patrones de @agent, comprender los niveles de prioridad, manejar operaciones del protocolo A2A, administrar el historial de tareas, configurar ajustes o utilizar funciones de seguridad de archivos para la coordinación de múltiples agentes. Se activa automáticamente cuando se detecta comunicación del agente, tareas del protocolo A2A, operaciones de historial u operaciones de seguridad de archivos.
