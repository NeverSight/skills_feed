Questa competenza gestisce tutta la documentazione per il progetto Synapse A2A. Dovrebbe essere utilizzato quando le modifiche al codice richiedono aggiornamenti della documentazione, quando si controlla la coerenza della documentazione o quando viene richiamato manualmente tramite /synapse-docs. Si attiva automaticamente quando si modificano file nelle directory synapse/, plugins/ o templates/ e quando README.md, guide/ o docs/ necessitano di aggiornamenti.
