قم بإجراء اختبارات جودة التعليمات البرمجية (ruff وmypy وpytest) وقم بتبسيط التعليمات البرمجية بشكل اختياري. يجب استخدام هذه المهارة عندما يريد المستخدم التحقق من جودة التعليمات البرمجية، أو تشغيل أجهزة الفحص، أو إجراء الاختبارات، أو تبسيط التعليمات البرمجية التي تم تعديلها مؤخرًا. يتم تشغيله بواسطة أوامر /lint أو /check أو /code-quality.
