Cette compétence fournit des capacités systématiques de détection et de découverte de bogues pour les extensions VS Code. À utiliser pour rechercher des bogues cachés, analyser le code à la recherche de problèmes potentiels, enquêter sur des comportements suspects, effectuer des audits de code ou rechercher de manière proactive des bogues avant qu'ils ne se manifestent. Couvre les modèles d'analyse statique, les techniques d'analyse dynamique, la détection d'odeur de code et l'enquête systématique sur la base de code.
