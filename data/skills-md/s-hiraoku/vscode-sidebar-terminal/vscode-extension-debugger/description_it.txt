Questa competenza fornisce indicazioni di livello esperto per il debug e la correzione dei bug nelle estensioni VS Code. Da utilizzare quando si esaminano errori di runtime, si correggono perdite di memoria, si risolvono problemi di WebView, si esegue il debug di problemi di attivazione, si correggono errori di tipo TypeScript o si risolvono errori di comunicazione delle estensioni. Copre flussi di lavoro di debug sistematici, modelli di bug comuni, analisi delle cause principali e strategie di prevenzione.
