Cette compétence fournit des conseils de niveau expert pour le débogage et la correction des bogues dans les extensions VS Code. À utiliser pour enquêter sur les erreurs d'exécution, corriger les fuites de mémoire, résoudre les problèmes de WebView, déboguer les problèmes d'activation, corriger les erreurs de type TypeScript ou dépanner les échecs de communication d'extension. Couvre les flux de travail de débogage systématique, les modèles de bogues courants, l'analyse des causes profondes et les stratégies de prévention.
