Этот навык обеспечивает экспертное руководство по разработке через тестирование (TDD) при разработке расширений VS Code в соответствии с методологией t-wada. Используйте при написании тестов перед реализацией, создании комплексных наборов тестов, реализации циклов Red-Green-Refactor или улучшении тестового покрытия для компонентов расширения, таких как WebViews, менеджеры терминалов и логика активации.
