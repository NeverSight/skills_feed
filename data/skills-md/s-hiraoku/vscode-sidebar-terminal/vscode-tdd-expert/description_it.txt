Questa competenza fornisce una guida di livello esperto per lo sviluppo basato sui test (TDD) nello sviluppo di estensioni VS Code seguendo la metodologia t-wada. Da utilizzare quando si scrivono test prima dell'implementazione, si creano suite di test complete, si implementano cicli Red-Green-Refactor o si migliora la copertura dei test per componenti di estensione come WebView, gestori terminali e logica di attivazione.
