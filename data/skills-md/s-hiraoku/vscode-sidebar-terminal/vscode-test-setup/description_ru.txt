Этот навык предоставляет исчерпывающие рекомендации по настройке и настройке тестовых сред для проектов расширения VS Code. Используйте при инициализации новой инфраструктуры тестирования, настройке средств запуска тестов (Vitest), настройке конвейеров тестирования CI/CD, интеграции инструментов покрытия (v8/c8) или устранении проблем с конфигурацией тестов.
