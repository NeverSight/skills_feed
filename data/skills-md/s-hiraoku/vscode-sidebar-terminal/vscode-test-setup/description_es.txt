Esta habilidad proporciona una guía completa para instalar y configurar entornos de prueba para proyectos de extensión de VS Code. Úselo al inicializar una nueva infraestructura de prueba, configurar ejecutores de pruebas (Vitest), configurar canales de prueba de CI/CD, integrar herramientas de cobertura (v8/c8) o solucionar problemas de configuración de pruebas.
