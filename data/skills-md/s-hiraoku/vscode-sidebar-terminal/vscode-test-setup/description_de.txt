Dieser Skill bietet umfassende Anleitungen zum Einrichten und Konfigurieren von Testumgebungen für VS-Code-Erweiterungsprojekte. Verwenden Sie es zum Initialisieren einer neuen Testinfrastruktur, zum Konfigurieren von Testläufern (Vitest), zum Einrichten von CI/CD-Testpipelines, zum Integrieren von Abdeckungstools (v8/c8) oder zur Fehlerbehebung bei Testkonfigurationsproblemen.
