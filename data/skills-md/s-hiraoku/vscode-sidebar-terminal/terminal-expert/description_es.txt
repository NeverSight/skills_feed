Esta habilidad proporciona una guía unificada de nivel experto para la implementación de terminales en extensiones de VS Code. Cubre la API y los complementos de xterm.js, la arquitectura del terminal VS Code, la integración de PTY, la persistencia de la sesión, el manejo de entrada (teclado/IME/mouse), la integración del shell con secuencias OSC y la optimización del rendimiento. Úselo al implementar cualquier característica relacionada con el terminal.
