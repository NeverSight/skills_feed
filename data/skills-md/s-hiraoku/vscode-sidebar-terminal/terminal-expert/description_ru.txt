Этот навык обеспечивает единое руководство экспертного уровня по реализации терминала в расширениях VS Code. Охватывает API и надстройки xterm.js, архитектуру терминала VS Code, интеграцию PTY, сохранение сеансов, обработку ввода (клавиатура/IME/мышь), интеграцию оболочки с последовательностями OSC и оптимизацию производительности. Используйте при реализации любых функций, связанных с терминалом.
