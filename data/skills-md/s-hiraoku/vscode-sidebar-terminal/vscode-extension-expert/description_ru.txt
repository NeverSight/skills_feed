Этот навык обеспечивает экспертное руководство по разработке расширений VS Code. Используйте при реализации новых функций расширения, отладке кода расширения, разработке пользовательских интерфейсов WebView, реализации функций протокола языкового сервера или оптимизации производительности расширения. Охватывает события активации, точки вклада, шаблоны API VS Code, лучшие практики безопасности, стратегии тестирования и рабочие процессы публикации.
