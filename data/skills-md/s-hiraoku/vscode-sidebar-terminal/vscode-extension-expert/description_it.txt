Questa competenza fornisce indicazioni di livello esperto per lo sviluppo dell'estensione VS Code. Da utilizzare durante l'implementazione di nuove funzionalità di estensione, il debug del codice dell'estensione, la progettazione di interfacce utente WebView, l'implementazione di funzionalità del protocollo Language Server o l'ottimizzazione delle prestazioni dell'estensione. Copre eventi di attivazione, punti di contribuzione, modelli API VS Code, best practice sulla sicurezza, strategie di test e flussi di lavoro di pubblicazione.
