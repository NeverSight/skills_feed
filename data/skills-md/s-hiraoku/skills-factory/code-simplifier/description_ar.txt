يبسط ويحسن التعليمات البرمجية من أجل الوضوح والاتساق وقابلية الصيانة مع الحفاظ على جميع الوظائف. يُستخدم عند مراجعة التعليمات البرمجية التي تم تعديلها مؤخرًا، أو إعادة البناء لسهولة القراءة، أو تطبيق معايير ترميز المشروع، أو تقليل التعقيد، أو تحسين تنظيم التعليمات البرمجية. يتم تشغيل المهام التي تتضمن تنظيف التعليمات البرمجية، أو تناسق النمط، أو إزالة التجريدات الزائدة عن الحاجة، أو تبسيط المنطق المتداخل.
