قم بكتابة كود SwiftUI أو مراجعته أو تحسينه باتباع أفضل الممارسات لإدارة الحالة وتكوين العرض والأداء وواجهات برمجة التطبيقات الحديثة وتزامن Swift واعتماد iOS 26+ Liquid Glass. يُستخدم عند إنشاء ميزات SwiftUI جديدة، أو إعادة هيكلة طرق العرض الحالية، أو مراجعة جودة التعليمات البرمجية، أو اعتماد أنماط SwiftUI الحديثة.
