Guide de style de projet de composant GPUI basé sur des modèles de code de composant GPUI. À utiliser lors de l'écriture de nouveaux composants, de la révision du code ou de la garantie de la cohérence avec les implémentations de composants GPU existantes. Couvre la structure des composants, les implémentations de traits, les conventions de dénomination et les modèles d'API observés dans la base de code réelle.
