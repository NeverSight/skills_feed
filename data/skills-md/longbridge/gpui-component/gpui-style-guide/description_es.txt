Guía de estilo del proyecto de componentes GPUI basada en patrones de código de componentes gpui. Úselo al escribir nuevos componentes, revisar código o garantizar la coherencia con las implementaciones de componentes gpui existentes. Cubre la estructura de los componentes, las implementaciones de rasgos, las convenciones de nomenclatura y los patrones de API observados en el código base real.
