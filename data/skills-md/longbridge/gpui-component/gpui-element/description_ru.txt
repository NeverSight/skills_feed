Реализация пользовательских элементов с использованием низкоуровневого API-интерфейса элемента GPUI (по сравнению с высокоуровневыми API-интерфейсами Render/RenderOnce). Используйте, когда вам нужен максимальный контроль над макетом, предварительной прорисовкой и фазами рисования для сложных, критически важных для производительности пользовательских компонентов пользовательского интерфейса, чего невозможно достичь с помощью признаков Render/RenderOnce.
