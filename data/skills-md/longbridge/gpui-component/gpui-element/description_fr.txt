Implémentation d'éléments personnalisés à l'aide de l'API Element de bas niveau de GPUI (par rapport aux API Render/RenderOnce de haut niveau). À utiliser lorsque vous avez besoin d'un contrôle maximal sur les phases de mise en page, de pré-peinture et de peinture pour des composants d'interface utilisateur personnalisés complexes et critiques en termes de performances qui ne peuvent pas être obtenus avec les traits Render/RenderOnce.
