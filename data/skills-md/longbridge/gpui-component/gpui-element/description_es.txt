Implementación de elementos personalizados utilizando la API Element de bajo nivel de GPUI (frente a las API Render/RenderOnce de alto nivel). Úselo cuando necesite el máximo control sobre las fases de diseño, prepintado y pintado para componentes de interfaz de usuario personalizados complejos y críticos para el rendimiento que no se pueden lograr con los rasgos Render/RenderOnce.
