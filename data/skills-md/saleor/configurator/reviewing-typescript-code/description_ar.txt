أنماط جودة كود TypeScript لكتابة ومراجعة التعليمات البرمجية. يغطي سلامة النوع والكود النظيف والأنماط الوظيفية واستخدام Zod ومعالجة الأخطاء. المشغلات على: إضافة كيان، إنشاء خدمة، إضافة مستودع، إنشاء مقارنة، إضافة منسق، مرحلة النشر، استعلام GraphQL، طفرة GraphQL، طريقة التمهيد، دعم الفرق، معالج الأوامر، مخطط Zod، فئة الخطأ، تنفيذ الميزة، إضافة وظيفة، كود إعادة التصنيع، كود نظيف، أنماط وظيفية، تقليل مرشح الخريطة، يرضي المشغل، حماية النوع، مراجعة الكود، مراجعة العلاقات العامة، التحقق من التنفيذ، كود التدقيق، إصلاح الأنواع.
