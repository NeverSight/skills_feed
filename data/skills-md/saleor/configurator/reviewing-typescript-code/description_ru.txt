Шаблоны качества кода TypeScript для написания и проверки кода. Охватывает типобезопасность, чистый код, функциональные шаблоны, использование Zod и обработку ошибок. Срабатывает при: добавлении сущности, создании службы, добавлении репозитория, создании компаратора, добавлении средства форматирования, этапе развертывания, запросе GraphQL, мутации GraphQL, методе начальной загрузки, поддержке различий, обработчике команд, схеме Zod, классе ошибок, реализации функции, добавлении функции, коде рефакторинга, чистом коде, функциональных шаблонах, сокращении фильтра карты, удовлетворении оператора, защите типа, проверке кода, PR-проверке, проверке реализации, коде аудита, типах исправлений.
