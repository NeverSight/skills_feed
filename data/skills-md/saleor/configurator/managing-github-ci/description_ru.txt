Настраивает рабочие процессы GitHub Actions и конвейеры CI/CD. Управляет автоматическими выпусками с помощью наборов изменений, проверки связей с общественностью и хуков Husky. Устраняет сбои CI. Запускается по: действиям GitHub, конвейеру CI, рабочему процессу, автоматизации выпуска, перехватчикам Husky, gh CLI, сбою рабочего процесса.
