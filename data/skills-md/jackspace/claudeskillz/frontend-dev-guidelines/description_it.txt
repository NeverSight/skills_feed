Linee guida per lo sviluppo del frontend per applicazioni React/TypeScript. Modelli moderni tra cui Suspense, caricamento lento, useSuspenseQuery, organizzazione dei file con directory delle funzionalità, stile MUI v7, router TanStack, ottimizzazione delle prestazioni e best practice TypeScript. Da utilizzare durante la creazione di componenti, pagine, funzionalità, recupero di dati, stili, routing o utilizzo del codice frontend.
