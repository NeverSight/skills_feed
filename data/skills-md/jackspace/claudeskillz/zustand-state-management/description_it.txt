Configurazione testata in produzione per la gestione dello stato Zustand nelle applicazioni React con TypeScript.
Questa competenza fornisce modelli completi per la creazione di uno stato globale scalabile e indipendente dai tipi.

Utilizzare quando: impostazione dello stato globale in React, migrazione da Redux o API Context, implementazione
persistenza dello stato con localStorage, configurazione di TypeScript con Zustand, utilizzo del pattern a fette
per negozi modulari, aggiunta di middleware devtools per il debug, gestione dell'idratazione Next.js SSR,
o riscontrare errori di idratazione, problemi di inferenza di TypeScript o problemi middleware persistenti.

errori di sintassi, errori persistenti di esportazione del middleware, cicli di rendering infiniti e pattern di sezioni
errori di inferenza di tipo.
