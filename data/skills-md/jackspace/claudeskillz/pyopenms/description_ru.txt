Интерфейс Python для OpenMS для анализа данных масс-спектрометрии. Используйте для рабочих процессов протеомики и метаболомики ЖХ-МС/МС, включая обработку файлов (mzML, mzXML, mzTab, FASTA, pepXML, protXML, mzIdentML), обработку сигналов, обнаружение признаков, идентификацию пептидов и количественный анализ. Применяется при работе с данными масс-спектрометрии, анализе экспериментов по протеомике или обработке наборов данных по метаболомике.
