Этот навык следует использовать для задач машинного обучения временных рядов, включая классификацию, регрессию, кластеризацию, прогнозирование, обнаружение аномалий, сегментацию и поиск по сходству. Используйте при работе с временными данными, последовательными шаблонами или наблюдениями с индексацией по времени, требующими специализированных алгоритмов, выходящих за рамки стандартных подходов ML. Особенно подходит для одномерного и многомерного анализа временных рядов с помощью API, совместимых с scikit-learn.
