TheSys C1 Generative UI API を React アプリケーションに統合するための本番対応スキル。このスキルは、プレーン テキスト応答の代わりに対話型コンポーネント (フォーム、グラフ、テーブル) をストリーミングする AI を活用したインターフェイスを構築するときに使用する必要があります。 Vite+React、Next.js、Cloudflare Workers と OpenAI、Anthropic Claude、Cloudflare Workers AI の完全な統合パターンをカバーします。 Zod スキーマを使用したツール呼び出し、テーマ設定、スレッド管理、実稼働デプロイメントが含まれます。 12 以上の一般的な統合エラーを防止し、チャット インターフェイス、データ視覚化、動的フォーム用の作業テンプレートを提供します。このスキルは、会話型 UI、AI アシスタント、検索インターフェイス、またはストリーミング LLM 応答を備えたリアルタイム生成ユーザー インターフェイスを必要とするアプリケーションを実装するときに使用します。
