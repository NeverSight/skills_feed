Biblioteca completa de Python para astronomía y astrofísica. Esta habilidad debe usarse cuando se trabaja con datos astronómicos, incluidas coordenadas celestes, unidades físicas, archivos FITS, cálculos cosmológicos, sistemas horarios, tablas, sistemas de coordenadas mundiales (WCS) y análisis de datos astronómicos. Úselo cuando las tareas impliquen transformaciones de coordenadas, conversiones de unidades, manipulación de archivos FITS, cálculos de distancias cosmológicas, conversiones de escalas de tiempo o procesamiento de datos astronómicos.
