Этот навык дает исчерпывающие знания по интеграции объектного хранилища Vercel Blob в приложения Vercel. Его следует использовать при настройке загрузки файлов, управлении изображениями и документами, реализации ресурсов, доставляемых через CDN, или обработке больших файлов в бессерверных и пограничных функциях Next.js.

Используйте этот навык, когда:
- Настройка загрузки файлов для приложений Next.js (изображения, PDF-файлы, видео)
- Реализация загрузки на стороне клиента с заранее заданными URL-адресами.
- Управление пользовательским контентом (аватары, вложения, медиа)
- Создание функций управления файлами (список, загрузка, удаление)
- Миграция с Cloudflare R2 на Vercel Blob
- Возникновение таких ошибок, как «BLOB_READ_WRITE_TOKEN не установлен», «превышен предел размера файла» или «ошибки токена загрузки клиента».
- Требуется простое объектное хранилище с автоматическим распределением CDN.

Ключевые слова: vercel blob, @vercel/blob, хранилище vercel, загрузка файла vercel, vercel cdn, хранилище BLOB-объектов vercel, загрузка клиента vercel, заранее заданный URL-адрес vercel, загрузка файла nextjs, загрузка изображений vercel, загрузка PDF, загрузка видео, пользовательские загрузки, многочастная загрузка, потоковая загрузка, blob cdn, ресурсы vercel, загрузка файла
