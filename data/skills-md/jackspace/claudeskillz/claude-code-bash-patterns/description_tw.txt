在 Claude Code 中有效使用 Bash 工具的全面知識。這個技能
應該在編排 CLI 工具、配置掛鉤、設置自動化時使用
工作流程、管理 git 操作、處理多命令模式或遇到
Bash 工具錯誤。

涵蓋：PreToolUse 掛鉤、命令鏈模式、git 工作流程自動化、CLI 工具
集成、自定義命令 (.claude/commands/)、安全配置、白名單、
會話持久性、輸出處理、錯誤預防和常見問題故障排除。

使用場合：設置 Claude Code 掛鉤、配置 bash 權限、創建自定義
命令、自動化 git 工作流程、編排多個 CLI 工具、調試 bash
命令失敗、實施安全防護、記錄命令執行或阻止
危險操作。
