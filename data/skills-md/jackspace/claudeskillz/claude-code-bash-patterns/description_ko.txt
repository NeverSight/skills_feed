CLI 도구 조정, 후크 구성, 자동화 설정 시 사용해야 합니다.
워크플로, Git 작업 관리, 다중 명령 패턴 처리 또는 발생
Bash 도구 오류.

내용: PreToolUse 후크, 명령 체이닝 패턴, git 워크플로 자동화, CLI 도구
통합, 사용자 정의 명령(.claude/commands/), 보안 구성, 허용 목록,
세션 지속성, 출력 처리, 오류 방지 및 일반적인 문제 해결.

사용 시기: Claude Code 후크 설정, bash 권한 구성, 사용자 정의 생성
명령, git 워크플로 자동화, 여러 CLI 도구 조정, bash 디버깅
명령 실패, 보안 가드 구현, 명령 실행 로깅 또는 방지
