Umfassende Kenntnisse zur effektiven Nutzung des Bash-Tools in Claude Code. Diese Fähigkeit
sollte beim Orchestrieren von CLI-Tools, beim Konfigurieren von Hooks und beim Einrichten der Automatisierung verwendet werden
Workflows, Verwalten von Git-Operationen, Umgang mit Mustern mit mehreren Befehlen oder Begegnungen
Fehler im Bash-Tool.

Behandelt: PreToolUse-Hooks, Befehlsverkettungsmuster, Git-Workflow-Automatisierung, CLI-Tool
Integration, benutzerdefinierte Befehle (.claude/commands/), Sicherheitskonfigurationen, Zulassungslisten,
Sitzungspersistenz, Ausgabeverarbeitung, Fehlervermeidung und Fehlerbehebung bei häufigen Problemen.

Wird verwendet, wenn: Claude Code-Hooks eingerichtet, Bash-Berechtigungen konfiguriert und benutzerdefinierte erstellt werden
Befehle, Automatisierung von Git-Workflows, Orchestrierung mehrerer CLI-Tools, Debuggen von Bash
Befehlsfehler, Implementierung von Sicherheitsmaßnahmen, Protokollierung der Befehlsausführung oder Verhinderung
gefährliche Operationen.
