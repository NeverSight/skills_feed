在 Claude Code 中有效使用 Bash 工具的全面知识。这个技能
应该在编排 CLI 工具、配置挂钩、设置自动化时使用
工作流程、管理 git 操作、处理多命令模式或遇到
Bash 工具错误。

涵盖：PreToolUse 挂钩、命令链模式、git 工作流程自动化、CLI 工具
集成、自定义命令 (.claude/commands/)、安全配置、白名单、
会话持久性、输出处理、错误预防和常见问题故障排除。

使用场合：设置 Claude Code 挂钩、配置 bash 权限、创建自定义
命令、自动化 git 工作流程、编排多个 CLI 工具、调试 bash
命令失败、实施安全防护、记录命令执行或阻止
危险操作。
