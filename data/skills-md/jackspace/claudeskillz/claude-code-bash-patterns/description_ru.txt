Всесторонние знания для эффективного использования инструмента Bash в Claude Code. Этот навык
следует использовать при оркестрации инструментов CLI, настройке перехватчиков, настройке автоматизации
рабочие процессы, управление операциями git, обработка шаблонов с несколькими командами или столкновение с
Ошибки инструмента Bash.

Охватывает: перехватчики PreToolUse, шаблоны цепочек команд, автоматизацию рабочих процессов git, инструмент CLI.
интеграция, пользовательские команды (.claude/commands/), конфигурации безопасности, белый список,
сохранение сеанса, обработка вывода, предотвращение ошибок и устранение распространенных проблем.

Используйте, когда: настройка перехватчиков Claude Code, настройка разрешений bash, создание пользовательских
команды, автоматизация рабочих процессов git, оркестровка нескольких инструментов CLI, отладка bash
сбои команд, внедрение средств безопасности, протоколирование выполнения команд или предотвращение
опасные операции.
