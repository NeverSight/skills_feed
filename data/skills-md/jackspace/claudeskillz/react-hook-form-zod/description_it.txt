Questa competenza fornisce una conoscenza completa per la creazione di moduli validati e indipendenti dai tipi in React utilizzando React Hook Form e la convalida dello schema Zod.

Utilizzare quando: creare moduli con convalida in React, integrare la convalida dello schema Zod con React Hook Form, utilizzare componenti shadcn/ui Form o Field, implementare la convalida lato client e server con un unico schema, gestire scenari di convalida complessi (oggetti nidificati, array, campi condizionali, convalida asincrona), creare moduli o procedure guidate a più passaggi, implementare campi di moduli dinamici con useFieldArray, ottimizzare le prestazioni e i nuovi rendering dei moduli, garantire la gestione degli errori dei moduli accessibili o il debug del modulo problemi di convalida.

Parole chiave: react-hook-form, useForm, convalida zod, zodResolver, @hookform/resolvers, schema del modulo, registro, handleSubmit, formState, useFieldArray, useWatch, useController, Controller, modulo shadcn, componente campo, convalida del server client, convalida nidificata, convalida del campo dell'array, campi dinamici, modulo a più passaggi, convalida asincrona, zod refine, z.infer, gestione degli errori del modulo, da non controllato a controllato, risolutore non trovato, errore di convalida dello schema
