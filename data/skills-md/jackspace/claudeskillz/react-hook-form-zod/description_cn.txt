该技能提供了使用 React Hook Form 和 Zod 模式验证在 React 中构建类型安全、经过验证的表单的全面知识。

使用时机：在 React 中构建带有验证的表单、将 Zod 模式验证与 React Hook Form 集成、使用 shadcn/ui 表单或字段组件、使用单一模式实现客户端和服务器端验证、处理复杂的验证场景（嵌套对象、数组、条件字段、异步验证）、构建多步骤表单或向导、使用 useFieldArray 实现动态表单字段、优化表单性能和重新渲染、确保可访问的表单错误处理或调试表单验证问题。

关键词：react-hook-form、useForm、zod 验证、zodResolver、@hookform/resolvers、表单模式、注册、handleSubmit、formState、useFieldArray、useWatch、useController、控制器、shadcn 表单、字段组件、客户端服务器验证、嵌套验证、数组字段验证、动态字段、多步表单、异步验证、zod 精炼、z.infer、表单错误处理、不受控到受控、未找到解析器、模式验证错误
