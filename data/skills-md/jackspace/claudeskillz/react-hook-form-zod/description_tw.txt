該技能提供了使用 React Hook Form 和 Zod 模式驗證在 React 中構建類型安全、經過驗證的表單的全面知識。

使用時機：在 React 中構建帶有驗證的表單、將 Zod 模式驗證與 React Hook Form 集成、使用 shadcn/ui 表單或字段組件、使用單一模式實現客戶端和服務器端驗證、處理複雜的驗證場景（嵌套對象、數組、條件字段、異步驗證）、構建多步驟表單或嚮導、使用 useFieldArray 實現動態表單字段、優化表單性能和重新渲染、確保可訪問的表單錯誤處理或調試表單驗證問題。

關鍵詞：react-hook-form、useForm、zod 驗證、zodResolver、@hookform/resolvers、表單模式、註冊、handleSubmit、formState、useFieldArray、useWatch、useController、控制器、shadcn 表單、字段組件、客戶端服務器驗證、嵌套驗證、數組字段驗證、動態字段、多步表單、異步驗證、zod 精煉、z.infer、表單錯誤處理、不受控到受控、未找到解析器、模式驗證錯誤
