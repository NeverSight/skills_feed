이 기술은 better-chatbot 프로젝트(https://github.com/cgoinglove/better-chatbot)를 위한 프로젝트별 코딩 규칙, 아키텍처 원칙, 저장소 구조 표준, 테스트 패턴 및 기여 지침을 제공합니다. 더 나은 챗봇에 기여하거나 함께 작업할 때 이 기술을 사용하여 디자인 철학을 이해하고 코드가 확립된 패턴을 따르도록 하세요.

포함 내용: API 아키텍처 심층 분석, 3계층 도구 시스템(MCP/워크플로/기본값), 구성 요소 디자인 패턴, 데이터베이스 저장소 패턴, 아키텍처 원칙(점진적 향상, 방어 프로그래밍, 스트리밍 우선), 기능 추가를 위한 실용적인 템플릿(도구, 경로, 저장소).

사용 시기: 더 나은 챗봇 저장소에서 작업, 기능/수정 기여, 아키텍처 결정 이해, 서버 작업 유효성 검사기 따르기, 도구/워크플로 구현, Playwright 테스트 설정, API 경로 추가, 데이터베이스 쿼리 설계, UI 구성 요소 구축, 다중 AI 공급자 통합 처리
