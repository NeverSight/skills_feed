Машинное обучение на Python с помощью scikit-learn. Используйте при работе с обучением с учителем (классификация, регрессия), обучением без учителя (кластеризация, уменьшение размерности), оценкой модели, настройкой гиперпараметров, предварительной обработкой или построением конвейеров машинного обучения. Предоставляет исчерпывающую справочную документацию по алгоритмам, методам предварительной обработки, конвейерам и передовым практикам.
