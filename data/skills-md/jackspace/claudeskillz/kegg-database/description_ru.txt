Прямой доступ к KEGG через REST API (только для академического использования). Анализ путей, картирование генов, метаболические пути, взаимодействие лекарств, преобразование идентификаторов. Для рабочих процессов Python с несколькими базами данных отдавайте предпочтение биосервисам. Используйте это для прямой работы HTTP/REST или управления, специфичного для KEGG.
