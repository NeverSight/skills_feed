يجب استخدام هذه المهارة عند العمل مع مصفوفات البيانات المشروحة في لغة بايثون، وخاصةً لتحليل جينوم الخلية الواحدة، أو إدارة القياسات التجريبية باستخدام البيانات الوصفية، أو التعامل مع مجموعات البيانات البيولوجية واسعة النطاق. يُستخدم عندما تتضمن المهام كائنات AnnData، أو ملفات h5ad، أو بيانات RNA-seq أحادية الخلية، أو التكامل مع أدوات scanpy/scverse.
