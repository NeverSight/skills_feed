Этот навык предоставляет готовые к использованию компоненты пользовательского интерфейса чата искусственного интеллекта, созданные на основе shadcn/ui для диалоговых интерфейсов искусственного интеллекта. Используйте при создании интерфейсов чата в стиле ChatGPT с потоковыми ответами, отображением вызовов инструментов/функций, визуализацией рассуждений или цитатами из источников. Предоставляет более 30 компонентов, включая сообщение, диалог, ответ, блок кода, рассуждение, инструмент, действия и источники, оптимизированные для Vercel AI SDK v5.

Предотвращает распространенные ошибки настройки с помощью Next.js App Router, Tailwind v4, интеграции shadcn/ui, миграции AI SDK v5, шаблонов композиции компонентов, совместимости с браузером голосового ввода, проблем адаптивного дизайна и оптимизации потоковой передачи.

Ключевые слова: ai-elements, vercel-ai-sdk, shadcn, чат-бот, разговорный-ai, потоковый-ui, чат-интерфейс, ai-chat, компоненты сообщений, разговорный интерфейс, вызов инструментов, отображение рассуждений, цитирование источников, потоковая передача уценки, вызов функций, ответы ИИ, быстрый ввод, подсветка кода, веб-предварительный просмотр, навигация по ветвям, отображение мышления, Стиль недоумения, Клод-артефакты
