Verwenden Sie diesen Skill für Next.js App Router-Muster, Serverkomponenten, Serveraktionen, Cache-Komponenten und Optimierungen auf Framework-Ebene. Behandelt Next.js 16 Breaking Changes, einschließlich asynchroner Parameter, Proxy.ts-Migration, Cache-Komponenten mit „Cache verwenden“ und React 19.2-Integration. Verwenden Sie für die Bereitstellung auf Cloudflare Workers stattdessen den Skill cloudflare-nextjs. Dieser Skill ist bereitstellungsunabhängig und funktioniert mit Vercel, AWS, selbst gehosteten oder jeder anderen Plattform.
