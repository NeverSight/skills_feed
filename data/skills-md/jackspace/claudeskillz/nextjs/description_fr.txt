Utilisez cette compétence pour les modèles de routeur d'application Next.js, les composants de serveur, les actions de serveur, les composants de cache et les optimisations au niveau du framework. Couvre les modifications récentes de Next.js 16, notamment les paramètres asynchrones, la migration proxy.ts, les composants de cache avec « utiliser le cache » et l'intégration de React 19.2. Pour le déploiement sur Cloudflare Workers, utilisez plutôt la compétence cloudflare-nextjs. Cette compétence est indépendante du déploiement et fonctionne avec Vercel, AWS, auto-hébergé ou n'importe quelle plate-forme.
