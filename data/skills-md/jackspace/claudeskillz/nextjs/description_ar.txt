استخدم هذه المهارة لأنماط جهاز توجيه تطبيق Next.js ومكونات الخادم وإجراءات الخادم ومكونات ذاكرة التخزين المؤقت والتحسينات على مستوى إطار العمل. يغطي Next.js 16 من التغييرات العاجلة بما في ذلك المعلمات غير المتزامنة، وترحيل proxy.ts، ومكونات ذاكرة التخزين المؤقت مع "استخدام ذاكرة التخزين المؤقت"، وتكامل React 19.2. للنشر في Cloudflare Workers، استخدم مهارة cloudflare-nextjs بدلاً من ذلك. هذه المهارة لا تعتمد على النشر وتعمل مع Vercel أو AWS أو الاستضافة الذاتية أو أي نظام أساسي.

الكلمات الرئيسية: Next.js 16، Next.js App Router، Next.js Pages Router، مكونات الخادم، مكونات خادم React، إجراءات الخادم، مكونات ذاكرة التخزين المؤقت، استخدام ذاكرة التخزين المؤقت، Next.js 16 التغييرات العاجلة، المعلمات غير المتزامنة nextjs، ترحيل proxy.ts، React 19.2، البيانات التعريفية Next.js، Next.js SEO، إنشاء بيانات ميتاداتا، إنشاء ثابت، عرض ديناميكي، تدفق SSR، التشويق، المسارات الموازية، مسارات الاعتراض، مجموعات المسارات، البرامج الوسيطة Next.js، مسارات واجهة برمجة التطبيقات Next.js، معالجات الطريق، revalidatePath، revalidateTag، التالي/الملاحة، useSearchParams، Turbopack، next.config
