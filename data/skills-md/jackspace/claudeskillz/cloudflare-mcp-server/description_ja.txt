Cloudflare Worker上でModel Context Protocol (MCP)サーバーを構築する場合は、このスキルを使用します。このスキルは、TypeScript を使用したリモート MCP サーバーのデプロイ、OAuth 認証 (GitHub、Google、Azure など) の実装、ステートフル MCP サーバーのデュラブル オブジェクトの使用、コスト最適化のための WebSocket ハイバネーションの実装、またはデュアル トランスポート メソッド (SSE + Streamable HTTP) の構成を行うときに使用する必要があります。このスキルは、McpAgent クラスのエクスポートの問題、OAuth リダイレクト URI の不一致、WebSocket 状態の損失、Durable Objects バインディング エラー、CORS 構成の間違いなど、15 以上の一般的なエラーを防止します。基本的な MCP サーバー、OAuth プロキシ統合、Durable Objects を備えたステートフル サーバー、および完全な wrangler.jsonc 構成用の実稼働テスト済みテンプレートが含まれています。トークン検証、DCR を使用したリモート OAuth、OAuth プロキシ (workers-oauth-provider)、完全な OAuth プロバイダー実装の 4 つの認証パターンをすべてカバーします。 Worker オブジェクトと Durable Objects の基本を備えた自己完結型。トークン効率: 最大 87% の節約 (40,000 → 5,000 トークン)。本番環境はCloudflareの公式MCPサーバーでテストされました。
