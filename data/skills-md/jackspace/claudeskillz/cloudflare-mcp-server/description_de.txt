Verwenden Sie diese Fähigkeit, wenn Sie Model Context Protocol (MCP)-Server auf Cloudflare Workers erstellen. Diese Fähigkeit sollte verwendet werden, wenn Sie Remote-MCP-Server mit TypeScript bereitstellen, OAuth-Authentifizierung (GitHub, Google, Azure usw.) implementieren, langlebige Objekte für zustandsbehaftete MCP-Server verwenden, den WebSocket-Ruhezustand zur Kostenoptimierung implementieren oder duale Transportmethoden (SSE + Streamable HTTP) konfigurieren. Der Skill verhindert mehr als 15 häufige Fehler, darunter Probleme beim Export der McpAgent-Klasse, Nichtübereinstimmungen des OAuth-Umleitungs-URI, Verlust des WebSocket-Status, Fehler bei der Bindung dauerhafter Objekte und Fehler bei der CORS-Konfiguration. Enthält produktionsgetestete Vorlagen für grundlegende MCP-Server, OAuth-Proxy-Integration, Stateful-Server mit dauerhaften Objekten und vollständige wrangler.jsonc-Konfigurationen. Deckt alle vier Authentifizierungsmuster ab: Token-Validierung, Remote-OAuth mit DCR, OAuth-Proxy (Workers-OAuth-Provider) und vollständige OAuth-Provider-Implementierung. Eigenständig mit den Grundlagen von Arbeitern und langlebigen Objekten. Token-Effizienz: ~87 % Ersparnis (40.000 → 5.000 Token). Die Produktion wurde auf den offiziellen MCP-Servern von Cloudflare getestet.
