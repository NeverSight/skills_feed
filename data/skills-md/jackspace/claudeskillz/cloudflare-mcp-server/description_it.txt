Questa competenza deve essere utilizzata durante la distribuzione di server MCP remoti con TypeScript, l'implementazione dell'autenticazione OAuth (GitHub, Google, Azure e così via), l'utilizzo di oggetti durevoli per server MCP con stato, l'implementazione dell'ibernazione WebSocket per l'ottimizzazione dei costi o la configurazione di metodi di trasporto doppio (SSE + Streamable HTTP). La competenza previene oltre 15 errori comuni, tra cui problemi di esportazione della classe McpAgent, mancate corrispondenze degli URI di reindirizzamento OAuth, perdita dello stato di WebSocket, errori di associazione di oggetti durevoli ed errori di configurazione CORS. Include modelli testati in produzione per server MCP di base, integrazione proxy OAuth, server con stato con oggetti durevoli e configurazioni complete di wrangler.jsonc.
