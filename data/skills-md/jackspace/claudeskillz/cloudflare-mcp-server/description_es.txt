Utilice esta habilidad al crear servidores Model Context Protocol (MCP) en Cloudflare Workers. Esta habilidad se debe utilizar al implementar servidores MCP remotos con TypeScript, implementar la autenticación OAuth (GitHub, Google, Azure, etc.), usar Durable Objects para servidores MCP con estado, implementar la hibernación de WebSocket para optimizar costos o configurar métodos de transporte dual (SSE + Streamable HTTP). La habilidad previene más de 15 errores comunes, incluidos problemas de exportación de clases McpAgent, discrepancias de URI de redireccionamiento de OAuth, pérdida de estado de WebSocket, errores de vinculación de objetos duraderos y errores de configuración de CORS. Incluye plantillas probadas en producción para servidores MCP básicos, integración de proxy OAuth, servidores con estado con objetos duraderos y configuraciones completas de wrangler.jsonc. Cubre los 4 patrones de autenticación: validación de token, OAuth remoto con DCR, proxy OAuth (trabajadores-proveedor-oauth) e implementación completa del proveedor OAuth. Autónomo con conceptos básicos de trabajadores y objetos duraderos. Eficiencia de tokens: ~87% de ahorro (40k → 5k tokens). Producción probada en los servidores MCP oficiales de Cloudflare.
