مكتبة Python للعمل مع ملفات DICOM (التصوير الرقمي والاتصالات في الطب). استخدم هذه المهارة عند قراءة بيانات التصوير الطبي أو كتابتها أو تعديلها بتنسيق DICOM، أو استخراج بيانات البكسل من الصور الطبية (CT، أو التصوير بالرنين المغناطيسي، أو الأشعة السينية، أو الموجات فوق الصوتية)، أو إخفاء هوية ملفات DICOM، أو العمل مع بيانات تعريف DICOM وعلاماتها، أو تحويل صور DICOM إلى تنسيقات أخرى، أو التعامل مع بيانات DICOM المضغوطة، أو معالجة مجموعات بيانات التصوير الطبي. ينطبق على المهام التي تتضمن تحليل الصور الطبية وأنظمة PACS وسير عمل الأشعة وتطبيقات تصوير الرعاية الصحية.
