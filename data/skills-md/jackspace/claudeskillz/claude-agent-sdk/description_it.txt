Questa competenza fornisce una conoscenza completa per lavorare con l'SDK dell'agente Anthropic Claude. Dovrebbe essere utilizzato durante la creazione di agenti IA autonomi, la creazione di flussi di lavoro di ragionamento in più fasi, l'orchestrazione di agenti secondari specializzati, l'integrazione di strumenti personalizzati e server MCP o l'implementazione di sistemi di agenti pronti per la produzione con le funzionalità di Claude Code.

Da utilizzare durante la creazione di agenti di codifica, sistemi SRE, revisori della sicurezza, operatori di risposta agli incidenti, bot di revisione del codice o qualsiasi sistema autonomo che richieda l'interazione programmatica con la CLI di Claude Code, sessioni persistenti, orchestrazione degli strumenti e controllo granulare delle autorizzazioni.
