Этот навык дает всесторонние знания для работы с Anthropic Claude Agent SDK. Его следует использовать при создании автономных агентов ИИ, создании многоэтапных рабочих процессов рассуждения, оркестрации специализированных субагентов, интеграции пользовательских инструментов и серверов MCP или внедрении готовых к работе агентных систем с возможностями Claude Code.

Используйте при создании агентов кодирования, систем SRE, аудиторов безопасности, реагирования на инциденты, ботов проверки кода или любой автономной системы, требующей программного взаимодействия с Claude Code CLI, постоянных сеансов, оркестрации инструментов и детального контроля разрешений.

Ключевые слова: Claude Agent sdk, @anthropic-ai/claude-agent-sdk, query(), createSdkMcpServer, AgentDefinition, инструмент(), субагенты Claude, серверы mcp, автономные агенты, агентные циклы, управление сеансами, разрешенный режим, canUseTool, многоагентная оркестровка, SettingSources, CLI не найден, длина контекста превышена
