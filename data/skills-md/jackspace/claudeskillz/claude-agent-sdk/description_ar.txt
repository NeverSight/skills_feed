توفر هذه المهارة معرفة شاملة للعمل مع Anthropic Claude Agent SDK. ويجب استخدامه عند إنشاء وكلاء الذكاء الاصطناعي المستقلين، أو إنشاء مسارات عمل منطقية متعددة الخطوات، أو تنسيق الوكلاء الفرعيين المتخصصين، أو دمج الأدوات المخصصة وخوادم MCP، أو تنفيذ أنظمة وكيلة جاهزة للإنتاج بقدرات Claude Code.

يُستخدم عند إنشاء وكلاء الترميز، أو أنظمة SRE، أو مدققي الأمان، أو المستجيبين للحوادث، أو روبوتات مراجعة التعليمات البرمجية، أو أي نظام مستقل يتطلب تفاعلًا برمجيًا مع Claude Code CLI، والجلسات المستمرة، وتنسيق الأدوات، والتحكم الدقيق في الأذونات.
