توفر هذه المهارة أتمتة شاملة لإعداد وتكوين مستودع GitHub.
يجب استخدامه عند إنشاء مشاريع جديدة، وإعداد خطوط أنابيب CI/CD، وتكوين المشكلة
القوالب، أو تمكين الفحص الأمني، أو ترحيل المشاريع الحالية إلى أتمتة GitHub.

تمنع المهارة 18 خطأ موثقًا في بناء جملة GitHub Actions YAML، وتكوين سير العمل،
إصدار بنية القالب، وإعداد Dependabot، والفحص الأمني لـ CodeQL. وتشمل 12
قوالب سير العمل التي تم اختبارها من خلال الإنتاج، و4 قوالب للإصدارات، وتكوينات الأمان، والأتمتة
البرامج النصية لإعداد المشروع السريع.

يُستخدم عندما: إعداد GitHub Actions CI/CD، وإنشاء قوالب المشكلات/العلاقات العامة، وتمكين Dependabot،
تكوين فحص CodeQL، وأتمتة إعداد مستودع GitHub، وإصلاح أخطاء بناء جملة YAML،
دمج الفحص الأمني، أو النشر إلى Cloudflare Workers عبر GitHub Actions، أو
تنفيذ مصفوفات اختبار متعددة الأطر.
