このスキルは、GitHub リポジトリのセットアップと構成を包括的に自動化します。
新しいプロジェクトの作成、CI/CD パイプラインのセットアップ、問題の構成時に使用する必要があります。
テンプレート、セキュリティ スキャンの有効化、または既存のプロジェクトの GitHub オートメーションへの移行。

このスキルは、GitHub Actions YAML 構文、ワークフロー設定、
問題テンプレートの構造、Dependabot のセットアップ、CodeQL セキュリティ スキャン。 12個含まれています
実稼働環境でテストされたワークフロー テンプレート、4 つの問題テンプレート、セキュリティ構成、および自動化
プロジェクトを迅速にセットアップするためのスクリプト。

次の場合に使用します: GitHub Actions CI/CD のセットアップ、Issue/PR テンプレートの作成、Dependabot の有効化、
CodeQL スキャンの構成、GitHub リポジトリのセットアップの自動化、YAML 構文エラーの修正、
セキュリティスキャンの統合、GitHub Actions経由のCloudflare Workersへのデプロイ、または
マルチフレームワークのテストマトリックスの実装。
