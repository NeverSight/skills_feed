Utilisez cette compétence lors de la création d'applications d'IA avec le SDK OpenAI Agents pour JavaScript/TypeScript. La compétence couvre à la fois les agents textuels et les agents vocaux en temps réel, y compris les flux de travail multi-agents (transferts), les outils avec les schémas Zod, les garde-fous d'entrée/sortie, les sorties structurées, le streaming, les modèles humains dans la boucle et les intégrations de framework pour Cloudflare Workers, Next.js et React. Il évite plus de 9 erreurs courantes, notamment les erreurs de type de schéma Zod, les échecs de traçage MCP, les boucles infinies, les échecs d'appel d'outil et les incompatibilités de schéma. La compétence comprend des modèles complets pour tous les types d'agents, des modèles de gestion des erreurs et des stratégies de débogage.
