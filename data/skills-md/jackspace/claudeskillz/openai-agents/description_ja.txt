このスキルは、OpenAI Agents SDK for JavaScript/TypeScript を使用して AI アプリケーションを構築するときに使用します。このスキルは、マルチエージェント ワークフロー (ハンドオフ)、Zod スキーマを備えたツール、入出力ガードレール、構造化出力、ストリーミング、人間参加型パターン、Cloudflare Workers、Next.js、および React のフレームワーク統合を含む、テキストベースのエージェントとリアルタイム音声エージェントの両方をカバーします。 Zod スキーマ タイプ エラー、MCP トレースの失敗、無限ループ、ツール呼び出しの失敗、スキーマの不一致など、9 つ以上の一般的なエラーを防止します。このスキルには、すべてのエージェント タイプの包括的なテンプレート、エラー処理パターン、デバッグ戦略が含まれています。

キーワード: OpenAI エージェント SDK、@openai/agents、@openai/agents-realtime、openai エージェント JavaScript、openai エージェント typescript、テキスト エージェント、音声エージェント、リアルタイム エージェント、マルチエージェント ワークフロー、エージェント ハンドオフ、エージェント ツール、zod スキーマ エージェント、構造化出力エージェント、エージェント ストリーミング、エージェント ガードレール、入力ガードレール、出力ガードレール、人間参加者、cloudflare ワーカー エージェント、 nextjs openai エージェント、react openai エージェント、hono エージェント、エージェントのデバッグ、Zod スキーマ タイプ エラー、MCP トレースの失敗、エージェントの無限ループ、ツール呼び出しの失敗、スキーマの不一致エージェント
