توفر هذه المهارة معرفة شاملة للعمل مع واجهة برمجة التطبيقات للرسائل الإنسانية (Claude API). يجب استخدامه عند دمج نماذج Claude في التطبيقات، أو تنفيذ استجابات التدفق، أو تمكين التخزين المؤقت الفوري لتوفير التكاليف، أو إضافة استخدام الأداة (استدعاء الوظيفة)، أو معالجة الصور بقدرات الرؤية، أو استخدام وضع التفكير الموسع.

يُستخدم عند إنشاء روبوتات الدردشة أو مساعدي الذكاء الاصطناعي أو أدوات إنشاء المحتوى أو أي تطبيق يتطلب فهم لغة كلود. يغطي كلاً من التطبيقات من جانب الخادم (Node.js وCloudflare Workers وNext.js) والوصول المباشر إلى واجهة برمجة التطبيقات (API).

الكلمات الرئيسية: claude api، anthropic api، messages api، @anthropic-ai/sdk، تدفق كلود، التخزين المؤقت الفوري، استخدام الأداة، الرؤية، التفكير الموسع، كلود 3.5 السوناتة، كلود 3.7 السوناتة، كلود السوناتة 4، استدعاء الوظائف، SSE، حدود المعدل، 429 خطأ
