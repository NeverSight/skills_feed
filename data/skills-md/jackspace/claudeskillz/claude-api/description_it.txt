Questa competenza fornisce una conoscenza completa per lavorare con l'API Anthropic Messages (API Claude). Dovrebbe essere utilizzato quando si integrano i modelli Claude nelle applicazioni, si implementano risposte in streaming, si abilita la memorizzazione nella cache immediata per risparmi sui costi, si aggiunge l'uso di strumenti (chiamata di funzione), si elaborano immagini con funzionalità di visione o si utilizza la modalità di pensiero esteso.

Da utilizzare durante la creazione di chatbot, assistenti IA, strumenti di generazione di contenuti o qualsiasi applicazione che richieda la comprensione della lingua di Claude. Copre sia le implementazioni lato server (Node.js, Cloudflare Workers, Next.js) che l'accesso diretto all'API.

Parole chiave: claude api, antropic api, message api, @anthropic-ai/sdk, streaming di claude, memorizzazione nella cache dei prompt, utilizzo dello strumento, visione, pensiero esteso, sonetto di Claude 3.5, sonetto di Claude 3.7, sonetto di Claude 4, chiamata di funzione, SSE, limiti di velocità, errori 429
