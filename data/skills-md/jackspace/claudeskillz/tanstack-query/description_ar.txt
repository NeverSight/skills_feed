توفر هذه المهارة معرفة شاملة لإدارة حالة خادم TanStack Query v5 (React Query) في تطبيقات React. يجب استخدامه عند إعداد جلب البيانات باستخدام useQuery، أو تنفيذ الطفرات باستخدام useMutation، أو تكوين QueryClient، أو إدارة استراتيجيات التخزين المؤقت، أو الترحيل من الإصدار 4 إلى الإصدار 5، أو تنفيذ التحديثات المتفائلة، أو استخدام استعلامات لا نهائية، أو مواجهة أخطاء الاستعلام/الطفرة.

يُستخدم عندما: تهيئة استعلام TanStack في مشاريع React، أو تكوين إعدادات QueryClient، أو إنشاء روابط استعلام مخصصة، أو تنفيذ التحولات مع معالجة الأخطاء، أو إعداد تحديثات متفائلة، أو استخدام useInfiniteQuery لترقيم الصفحات، أو الترحيل من React Query v4 إلى v5، أو تصحيح مشكلات البيانات التي لا معنى لها، أو إصلاح مشكلات التخزين المؤقت، أو حل تغييرات الفصل v5، أو تنفيذ الاستعلامات المعلقة، أو إعداد أدوات تطوير الاستعلام.

الكلمات الرئيسية: استعلام TanStack، React Query، useQuery، useMutation، useInfiniteQuery، useSuspenseQuery، QueryClient، QueryClientProvider، جلب البيانات، حالة الخادم، التخزين المؤقت، staleTime، gcTime، إبطال الاستعلام، الجلب المسبق، التحديثات المتفائلة، الطفرات، مفاتيح الاستعلام، وظائف الاستعلام، حدود الخطأ، تشويق، React Query DevTools، ترحيل v5، v4 إلى الإصدار 5، طلب الشلالات، إعادة جلب الخلفية، إعادة تسمية ذاكرة التخزين المؤقت، إعادة تسمية حالة التحميل، الحالة المعلقة، مطلوب الأوليPageParam، إزالة keepPreviousData، placeholderData، إزالة عمليات رد الاتصال للاستعلام، إزالة onSuccess، إزالة onError، بناء جملة الكائن مطلوب
