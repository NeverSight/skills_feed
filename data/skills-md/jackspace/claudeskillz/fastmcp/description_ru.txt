Используйте этот навык при создании серверов MCP (Model Context Protocol) с помощью FastMCP на Python. FastMCP — это платформа для создания серверов, которые предоставляют инструменты, ресурсы и подсказки таким специалистам, как Клод. Навыки охватывают создание сервера, определения инструментов/ресурсов, серверные части хранилища (память/диск/Redis/DynamoDB), продолжительность жизни сервера, систему промежуточного программного обеспечения (8 встроенных типов), состав сервера (импорт/монтирование), прокси-сервер OAuth, шаблоны аутентификации, значки, интеграцию OpenAPI, конфигурацию клиента, развертывание в облаке (FastMCP Cloud), обработку ошибок и производственные шаблоны. Он предотвращает более 25 распространенных ошибок, включая неправильную конфигурацию хранилища, проблемы со сроком службы, ошибки порядка промежуточного программного обеспечения, циклический импорт, проблемы сервера на уровне модулей, путаницу с асинхронностью и ожиданием, уязвимости безопасности OAuth и сбои при развертывании облака. Включает шаблоны для базовых серверов, серверных частей хранилища, промежуточного программного обеспечения, состава серверов, прокси-сервера OAuth, интеграции API, тестирования и автономных производственных архитектур.
