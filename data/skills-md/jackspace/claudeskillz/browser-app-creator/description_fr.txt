Crée des applications Web HTML/CSS/JS complètes à fichier unique avec persistance localStorage, interface utilisateur optimisée pour le TDAH (boutons de plus de 60 px), mode sombre et fonctionnalité hors ligne. À utiliser lorsque l'utilisateur dit « créer une application », « créer un outil », « créer un tableau de bord » ou demande une interface basée sur un navigateur.
