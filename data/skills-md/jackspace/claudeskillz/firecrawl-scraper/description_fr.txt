Domaine de connaissances complet pour l'API Firecrawl v2 - web scraping et exploration qui convertit les sites Web en démarques ou en données structurées prêtes pour LLM.

À utiliser dans les cas suivants : scraping de sites Web, exploration de sites entiers, extraction de contenu Web, conversion de code HTML en markdown, création de web scrapers, gestion de contenu JavaScript dynamique, contournement de la protection anti-bot, extraction de données structurées à partir de pages Web ou lorsque vous rencontrez un « contenu qui ne se charge pas », des « problèmes de rendu JavaScript » ou un « blocage par détection de robot ».
