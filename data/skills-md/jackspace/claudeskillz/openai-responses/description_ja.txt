このスキルは、エージェント アプリケーションを構築するための統合ステートフル API である OpenAI の Responses API を操作するための包括的な知識を提供します。これは、ターン間で推論を保持する AI エージェントを構築する場合、外部ツール用の MCP サーバーを統合する場合、組み込みツール (コード インタープリター、ファイル検索、Web 検索、画像生成) を使用する場合、ステートフルな会話を管理する場合、バックグラウンド処理を実装する場合、またはチャット完了 API から移行する場合に使用する必要があります。

エージェント ワークフロー、メモリを備えた会話型 AI、ツールベースのアプリケーション、RAG システム、データ分析エージェント、または永続的な状態を持つ OpenAI の推論モデルを必要とするアプリケーションを構築するときに使用します。 Node.js SDK と Cloudflare Workers 実装の両方をカバーします。
