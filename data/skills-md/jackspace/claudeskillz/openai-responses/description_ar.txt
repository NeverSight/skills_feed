توفر هذه المهارة معرفة شاملة للعمل مع واجهة برمجة تطبيقات الاستجابة الخاصة بـ OpenAI، وهي واجهة برمجة التطبيقات ذات الحالة الموحدة لبناء التطبيقات الوكيلة. يجب استخدامه عند إنشاء وكلاء الذكاء الاصطناعي الذين يحافظون على التفكير المنطقي عبر المنعطفات، أو دمج خوادم MCP للأدوات الخارجية، أو استخدام الأدوات المضمنة (مترجم التعليمات البرمجية، أو البحث عن الملفات، أو بحث الويب، أو إنشاء الصور)، أو إدارة المحادثات ذات الحالة، أو تنفيذ معالجة الخلفية، أو الترحيل من Chat Completions API.

يُستخدم عند إنشاء مسارات عمل وكيلة، أو الذكاء الاصطناعي للمحادثة مع الذاكرة، أو التطبيقات القائمة على الأدوات، أو أنظمة RAG، أو وكلاء تحليل البيانات، أو أي تطبيق يتطلب نماذج استدلال OpenAI ذات الحالة المستمرة. يغطي كلاً من تطبيقات Node.js SDK وCloudflare Workers.

الكلمات الرئيسية: استجابات API، استجابات openai، openai ذات الحالة، openai mcp، مترجم الكود openai، بحث الملفات openai، بحث الويب openai، إنشاء الصور openai، الحفاظ على الاستدلال، سير العمل الوكيل، حالة المحادثة، وضع الخلفية، ترحيل إكمالات الدردشة، gpt-5، مخرجات متعددة الأشكال
