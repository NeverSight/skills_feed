이 기술은 에이전트 애플리케이션 구축을 위한 통합 상태 저장 API인 OpenAI의 Responses API 작업에 대한 포괄적인 지식을 제공합니다. 차례대로 추론을 보존하는 AI 에이전트를 구축하고, 외부 도구용 MCP 서버를 통합하고, 내장 도구(코드 해석기, 파일 검색, 웹 검색, 이미지 생성)를 사용하고, 상태 저장 대화를 관리하고, 백그라운드 처리를 구현하거나, Chat Completions API에서 마이그레이션할 때 사용해야 합니다.

에이전트 워크플로, 메모리가 포함된 대화형 AI, 도구 기반 애플리케이션, RAG 시스템, 데이터 분석 에이전트 또는 지속적인 상태를 갖춘 OpenAI의 추론 모델이 필요한 모든 애플리케이션을 구축할 때 사용하세요. Node.js SDK와 Cloudflare Workers 구현을 모두 다룹니다.
