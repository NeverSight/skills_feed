Этот навык дает всесторонние знания для работы с OpenAI Responses API, унифицированным API с отслеживанием состояния для создания агентных приложений. Его следует использовать при создании агентов искусственного интеллекта, которые сохраняют рассуждения во время поворотов, интеграции серверов MCP для внешних инструментов, использовании встроенных инструментов (интерпретатор кода, поиск файлов, веб-поиск, создание изображений), управлении разговорами с сохранением состояния, реализации фоновой обработки или переходе с API завершения чата.

Используйте при создании агентных рабочих процессов, диалогового ИИ с памятью, приложений на основе инструментов, систем RAG, агентов анализа данных или любого приложения, требующего моделей рассуждения OpenAI с постоянным состоянием. Охватывает реализации Node.js SDK и Cloudflare Workers.
