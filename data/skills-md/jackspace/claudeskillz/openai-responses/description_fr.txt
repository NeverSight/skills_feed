Cette compétence fournit des connaissances complètes pour travailler avec l'API Responses d'OpenAI, l'API avec état unifiée pour la création d'applications agentiques. Il doit être utilisé lors de la création d'agents d'IA qui préservent le raisonnement d'un tour à l'autre, de l'intégration de serveurs MCP pour des outils externes, de l'utilisation d'outils intégrés (interprète de code, recherche de fichiers, recherche sur le Web, génération d'images), de la gestion de conversations avec état, de la mise en œuvre du traitement en arrière-plan ou de la migration depuis l'API de complétion de chat.

À utiliser lors de la création de flux de travail agentiques, d'IA conversationnelle avec mémoire, d'applications basées sur des outils, de systèmes RAG, d'agents d'analyse de données ou de toute application nécessitant les modèles de raisonnement d'OpenAI avec état persistant. Couvre à la fois les implémentations du SDK Node.js et de Cloudflare Workers.

Mots-clés : API de réponses, réponses openai, openai avec état, openai mcp, interpréteur de code openai, openai de recherche de fichiers, openai de recherche Web, openai de génération d'images, préservation du raisonnement, flux de travail agents, état de conversation, mode d'arrière-plan, migration de terminaisons de chat, gpt-5, sorties polymorphes
