Configuration testée en production pour Content Collections - un outil de création basé sur TypeScript qui transforme
fichiers de contenu local (Markdown/MDX) dans des collections de données de type sécurisé avec validation automatique.

À utiliser lorsque : créer des blogs, des sites de documentation ou des applications riches en contenu avec Vite + React,
configuration du contenu MDX avec les composants React, implémentation de schémas de contenu de type sécurisé avec Zod,
migrer depuis Contentlayer ou rencontrer des erreurs d'importation TypeScript avec des collections de contenu.

Couvre : configuration du plugin Vite, alias de chemin tsconfig, schémas de collection avec validation Zod,
Compilation MDX avec compileMDX, fonctions de transformation pour les propriétés calculées, plugins rehype/remark,
Intégration des composants React avec MDXContent, déploiement de Cloudflare Workers et optimisation de la version de production.
