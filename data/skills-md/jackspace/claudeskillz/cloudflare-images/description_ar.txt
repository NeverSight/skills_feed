توفر هذه المهارة معرفة شاملة لصور Cloudflare، والتي تغطي كلاً من واجهة برمجة تطبيقات الصور (التحميل/التخزين) وتحويلات الصور (تحسين أي صورة). يجب استخدامه عند تحميل الصور إلى Cloudflare، أو تغيير حجم الصور، أو تحسين تسليم الصور، أو تنفيذ التحميلات المباشرة للمنشئ، أو إنشاء متغيرات للصور، أو إنشاء عناوين URL موقعة للصور الخاصة، أو تحويل الصور عبر العمال، أو مواجهة أخطاء تحميل/تحويل الصور.

يُستخدم عندما: إعداد تخزين Cloudflare Images، أو تنفيذ الصور التي تم تحميلها بواسطة المستخدم، أو إنشاء صور سريعة الاستجابة، أو تحسين تنسيقات الصور (WebP/AVIF)، أو تغيير حجم الصور عبر URL أو Workers، أو تصحيح أخطاء CORS من خلال التحميلات المباشرة، أو التعامل مع أخطاء تحويل الصور (9401-9413)، أو تنفيذ عناوين URL الموقعة، أو إدارة متغيرات الصور، أو إنشاء شبكات CDN للصور.

الكلمات الرئيسية: صور cloudflare، تحميل الصور cloudflare، imagedelivery.net، تحويلات الصور cloudflare، /cdn-cgi/image/، التحميل المباشر للمنشئ، متغيرات الصور، عمال cf.image، صور عناوين url الموقعة، المتغيرات المرنة، تحويل webp avif، الصور المستجيبة cloudflare، خطأ 5408، خطأ 9401، خطأ 9403، تحميل CORS المباشر، بيانات متعددة الأجزاء / النموذج، تحسين الصورة com.cloudflare
