此技能提供有關 Cloudflare Images 的全面知識，涵蓋圖像 API（上傳/存儲）和圖像轉換（優化任何圖像）。當將圖像上傳到 Cloudflare、調整圖像大小、優化圖像交付、實現直接創建者上傳、創建圖像變體、為私有圖像生成簽名 URL、通過 Workers 轉換圖像或遇到圖像上傳/轉換錯誤時，應使用它。

使用場合：設置 Cloudflare Images 存儲、實現用戶上傳的圖像、創建響應式圖像、優化圖像格式 (WebP/AVIF)、通過 URL 或 Workers 調整圖像大小、通過直接上傳調試 CORS 錯誤、處理圖像轉換錯誤 (9401-9413)、實現簽名 URL、管理圖像變體或構建圖像 CDN。

關鍵詞：cloudflare 圖像、圖像上傳 cloudflare、imagedelivery.net、cloudflare 圖像轉換、/cdn-cgi/image/、直接創建者上傳、圖像變體、cf.image 工作人員、簽名 url 圖像、靈活變體、webp avif 轉換、響應式圖像 cloudflare、錯誤 5408、錯誤 9401、錯誤 9403、CORS 直接上傳、多部分/表單數據、圖像優化 cloudflare
