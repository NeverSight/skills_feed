このスキルは、画像 API (アップロード/ストレージ) と画像変換 (任意の画像の最適化) の両方をカバーする、Cloudflare 画像に関する包括的な知識を提供します。 Cloudflareへの画像のアップロード、画像のサイズ変更、画像配信の最適化、クリエイターによる直接アップロードの実装、画像バリアントの作成、プライベート画像の署名付きURLの生成、ワーカーを介した画像の変換、または画像のアップロード/変換エラーが発生した場合に使用する必要があります。

以下の場合に使用します: Cloudflare 画像ストレージの設定、ユーザーがアップロードした画像の実装、レスポンシブ画像の作成、画像形式の最適化 (WebP/AVIF)、URL またはワーカー経由での画像のサイズ変更、直接アップロードでの CORS エラーのデバッグ、画像変換エラー (9401 ～ 9413) の処理、署名付き URL の実装、画像バリアントの管理、または画像 CDN の構築。

キーワード: Cloudflare イメージ、イメージ アップロード Cloudflare、imagedelivery.net、cloudflare イメージ変換、/cdn-cgi/image/、作成者による直接アップロード、イメージ バリアント、cf.image ワーカー、署名付き URL イメージ、フレキシブル バリアント、webp avif 変換、レスポンシブ イメージ クラウドフレア、エラー 5408、エラー 9401、エラー 9403、CORS 直接アップロード、マルチパート/フォーム データ、イメージ最適化クラウドフレア
