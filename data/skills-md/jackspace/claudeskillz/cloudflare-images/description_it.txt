Questa competenza fornisce una conoscenza completa delle immagini Cloudflare, coprendo sia l'API delle immagini (caricamento/archiviazione) che le trasformazioni delle immagini (ottimizzazione di qualsiasi immagine). Dovrebbe essere utilizzato quando si caricano immagini su Cloudflare, si ridimensionano immagini, si ottimizza la consegna di immagini, si implementano caricamenti diretti del creatore, si creano varianti di immagini, si generano URL firmati per immagini private, si trasformano immagini tramite Worker o si riscontrano errori di caricamento/trasformazione di immagini.

Da utilizzare quando: configurazione dell'archiviazione di immagini Cloudflare, implementazione di immagini caricate dagli utenti, creazione di immagini reattive, ottimizzazione dei formati di immagine (WebP/AVIF), ridimensionamento di immagini tramite URL o Worker, debug di errori CORS con caricamenti diretti, gestione di errori di trasformazione di immagini (9401-9413), implementazione di URL firmati, gestione di varianti di immagine o creazione di CDN di immagini.

Parole chiave: immagini cloudflare, caricamento immagini cloudflare, imagedelivery.net, trasformazioni immagini cloudflare, /cdn-cgi/image/, caricamento diretto del creatore, varianti immagine, cf.image lavoratori, immagini URL firmate, varianti flessibili, conversione webp avif, immagini reattive cloudflare, errore 5408, errore 9401, errore 9403, caricamento diretto CORS, multipart/form-data, ottimizzazione immagine cloudflare
