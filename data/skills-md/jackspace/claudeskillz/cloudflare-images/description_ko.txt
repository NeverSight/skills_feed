이 기술은 이미지 API(업로드/저장)와 이미지 변환(모든 이미지 최적화)을 모두 다루는 Cloudflare 이미지에 대한 포괄적인 지식을 제공합니다. Cloudflare에 이미지 업로드, 이미지 크기 조정, 이미지 전달 최적화, 직접 작성자 업로드 구현, 이미지 변형 생성, 비공개 이미지에 대한 서명된 URL 생성, 작업자를 통한 이미지 변환 또는 이미지 업로드/변환 오류 발생 시 사용해야 합니다.

사용 시기: Cloudflare 이미지 저장소 설정, 사용자 업로드 이미지 구현, 반응형 이미지 생성, 이미지 형식 최적화(WebP/AVIF), URL 또는 작업자를 통한 이미지 크기 조정, 직접 업로드로 CORS 오류 디버깅, 이미지 변환 오류 처리(9401-9413), 서명된 URL 구현, 이미지 변형 관리 또는 이미지 CDN 구축.
