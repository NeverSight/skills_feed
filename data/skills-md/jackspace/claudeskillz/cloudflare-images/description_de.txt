Diese Fähigkeit vermittelt umfassende Kenntnisse für Cloudflare-Bilder und deckt sowohl die Bilder-API (Hochladen/Speichern) als auch Bildtransformationen (Optimierung jedes Bildes) ab. Es sollte verwendet werden, wenn Sie Bilder auf Cloudflare hochladen, die Größe von Bildern ändern, die Bildbereitstellung optimieren, direkte Uploads durch Ersteller implementieren, Bildvarianten erstellen, signierte URLs für private Bilder generieren, Bilder über Worker transformieren oder auf Fehler beim Hochladen/Transformieren von Bildern stoßen.

Verwendung bei: Einrichten des Cloudflare-Bildspeichers, Implementieren von vom Benutzer hochgeladenen Bildern, Erstellen responsiver Bilder, Optimieren von Bildformaten (WebP/AVIF), Ändern der Bildgröße über URL oder Worker, Debuggen von CORS-Fehlern mit direkten Uploads, Behandeln von Bildtransformationsfehlern (9401-9413), Implementieren signierter URLs, Verwalten von Bildvarianten oder Erstellen von Bild-CDNs.
