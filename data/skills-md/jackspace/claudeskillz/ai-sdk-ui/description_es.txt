Enlaces frontend React para interfaces de chat, finalizaciones y UI de transmisión impulsadas por IA con Vercel AI SDK v5.
Incluye ganchos useChat, useCompletion y useObject para crear aplicaciones interactivas de IA.

Úselo cuando: cree interfaces de chat de React, implemente finalizaciones de IA en la interfaz de usuario, transmita respuestas de IA al frontend,
manejar el estado de los mensajes de chat, crear aplicaciones de IA Next.js, administrar archivos adjuntos con IA o encontrar
errores como "useChat no pudo analizar la transmisión", "useChat no responde", transmisiones no cerradas o problemas de transmisión.
