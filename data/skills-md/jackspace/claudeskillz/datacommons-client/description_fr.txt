Travaillez avec Data Commons, une plateforme fournissant un accès programmatique aux données statistiques publiques provenant de sources mondiales. Utilisez cette compétence lorsque vous travaillez avec des données démographiques, des indicateurs économiques, des statistiques de santé, des données environnementales ou tout autre ensemble de données publiques disponibles via Data Commons. Applicable pour interroger les statistiques démographiques, les chiffres du PIB, les taux de chômage, la prévalence des maladies, la résolution des entités géographiques et explorer les relations entre les entités statistiques.
