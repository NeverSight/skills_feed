Kit completo di strumenti di intelligenza artificiale per il settore sanitario per lo sviluppo, il test e l'implementazione di modelli di machine learning con dati clinici. Questa competenza dovrebbe essere utilizzata quando si lavora con cartelle cliniche elettroniche (EHR), attività di previsione clinica (mortalità, riammissione, raccomandazione sui farmaci), sistemi di codifica medica (ICD, NDC, ATC), segnali fisiologici (EEG, ECG), set di dati sanitari (MIMIC-III/IV, eICU, OMOP) o implementando modelli di deep learning per applicazioni sanitarie (RETAIN, SafeDrug, Transformer, GNN).
