Génère automatiquement des suites de tests complètes (unité, intégration, E2E) basées sur le code et les modèles de tests antérieurs. À utiliser lorsque l'utilisateur dit « écrire des tests », « tester ceci », « ajouter une couverture » ou après avoir corrigé des bogues pour créer des tests de régression. Élimine les frictions lors des tests pour les utilisateurs atteints de TDAH.
