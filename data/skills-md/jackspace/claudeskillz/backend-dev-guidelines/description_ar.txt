دليل شامل لتطوير الواجهة الخلفية لخدمات Node.js/Express/TypeScript الصغيرة. يُستخدم عند إنشاء مسارات أو وحدات تحكم أو خدمات أو مستودعات أو برامج وسيطة أو العمل مع Express APIs أو الوصول إلى قاعدة بيانات Prisma أو تتبع أخطاء Sentry أو التحقق من Zod أو unifiedConfig أو حقن التبعية أو الأنماط غير المتزامنة. يغطي البنية ذات الطبقات (المسارات ← وحدات التحكم ← الخدمات ← المستودعات)، ونمط BaseController، ومعالجة الأخطاء، ومراقبة الأداء، واستراتيجيات الاختبار، والهجرة من الأنماط القديمة.
