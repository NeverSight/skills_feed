Outil Python principal pour plus de 40 services bioinformatiques. Préféré pour les workflows multi-bases de données : UniProt, KEGG, ChEMBL, PubChem, Reactome, QuickGO. API unifiée pour les requêtes, le mappage des identifiants et l'analyse des parcours. Pour un contrôle REST direct, utilisez des compétences de base de données individuelles (uniprot-database, kegg-database).
