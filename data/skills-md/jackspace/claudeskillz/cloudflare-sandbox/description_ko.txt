이 기술은 에지의 전체 Linux 컨테이너에서 안전하고 격리된 코드 실행을 가능하게 하는 Cloudflare Sandboxes SDK로 애플리케이션을 구축하기 위한 포괄적인 지식을 제공합니다. 신뢰할 수 없는 코드 실행, Python/Node.js 스크립트 실행, git 작업 수행, AI 코드 실행 시스템 구축, 대화형 개발 환경 생성 또는 전체 OS 기능이 필요한 CI/CD 워크플로 구현 시 사용해야 합니다.

사용 시기: Cloudflare Sandbox 설정, Python/Node.js 코드 안전하게 실행, 상태 저장 개발 환경 관리, AI 코드 해석기 구현, 격리된 셸 명령 실행, 프로그래밍 방식으로 Git 저장소 처리, 채팅 기반 코딩 에이전트 구축, 임시 빌드 환경 생성, 시스템 도구(ffmpeg, imagemagick 등)를 사용하여 파일 처리, 컨테이너 수명 주기, 세션 관리 또는 상태 지속성과 관련된 문제가 발생할 때.
