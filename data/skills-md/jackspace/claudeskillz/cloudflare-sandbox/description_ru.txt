Этот навык дает всесторонние знания для создания приложений с помощью Cloudflare Sandboxes SDK, который обеспечивает безопасное, изолированное выполнение кода в полных контейнерах Linux на периферии. Его следует использовать при выполнении ненадежного кода, запуске сценариев Python/Node.js, выполнении операций git, построении систем выполнения кода AI, создании интерактивных сред разработки или реализации рабочих процессов CI/CD, требующих полных возможностей ОС.

Используйте в следующих случаях: настройка песочниц Cloudflare, безопасное выполнение кода Python/Node.js, управление средами разработки с отслеживанием состояния, внедрение интерпретаторов кода AI, изолированный запуск команд оболочки, программная обработка репозиториев git, создание агентов кодирования на основе чата, создание временных сред сборки, обработка файлов с помощью системных инструментов (ffmpeg, imagemagick и т. д.) или при возникновении проблем с жизненным циклом контейнера, управлением сеансами или сохранением состояния.

Ключевые слова: песочница Cloudflare, выполнение контейнера, выполнение кода, изолированная среда, устойчивые объекты, контейнер Linux, выполнение Python, выполнение узла, операции git, интерпретатор кода, агенты ИИ, управление сеансами, эфемерный контейнер, рабочая область, песочница SDK, @cloudflare/sandbox, exec(), getSandbox(), runCode(), gitCheckout(), контейнер ubuntu
