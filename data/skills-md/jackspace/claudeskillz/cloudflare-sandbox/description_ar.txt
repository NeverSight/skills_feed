توفر هذه المهارة معرفة شاملة لبناء التطبيقات باستخدام Cloudflare Sandboxes SDK، والتي تتيح تنفيذ تعليمات برمجية آمنة ومعزولة في حاويات Linux الكاملة على الحافة. يجب استخدامه عند تنفيذ تعليمات برمجية غير موثوقة، أو تشغيل البرامج النصية Python/Node.js، أو تنفيذ عمليات git، أو إنشاء أنظمة تنفيذ تعليمات برمجية للذكاء الاصطناعي، أو إنشاء بيئات تطوير تفاعلية، أو تنفيذ مسارات عمل CI/CD التي تتطلب إمكانات نظام التشغيل الكاملة.

يُستخدم عندما: إعداد Cloudflare Sandboxes، وتنفيذ كود Python/Node.js بأمان، وإدارة بيئات التطوير ذات الحالة، وتنفيذ مترجمات أكواد الذكاء الاصطناعي، وتشغيل أوامر Shell بشكل منفصل، والتعامل مع مستودعات git برمجيًا، وإنشاء وكلاء ترميز قائمين على الدردشة، وإنشاء بيئات إنشاء مؤقتة، ومعالجة الملفات باستخدام أدوات النظام (ffmpeg، imagemagick، وما إلى ذلك)، أو عند مواجهة مشكلات تتعلق بدورة حياة الحاوية، أو إدارة الجلسة، أو استمرار الحالة.

الكلمات الرئيسية: صندوق حماية Cloudflare، تنفيذ الحاوية، تنفيذ التعليمات البرمجية، بيئة معزولة، كائنات متينة، حاوية Linux، تنفيذ python، تنفيذ العقدة، عمليات git، مترجم التعليمات البرمجية، وكلاء الذكاء الاصطناعي، إدارة الجلسة، حاوية مؤقتة، مساحة عمل، sandbox SDK، @cloudflare/sandbox، exec()، getSandbox()، runCode()، gitCheckout()، حاوية أوبونتو
