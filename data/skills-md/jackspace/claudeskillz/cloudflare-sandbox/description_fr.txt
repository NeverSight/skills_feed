Cette compétence fournit des connaissances complètes pour créer des applications avec le SDK Cloudflare Sandboxes, qui permet une exécution de code sécurisée et isolée dans des conteneurs Linux complets à la périphérie. Il doit être utilisé lors de l'exécution de code non fiable, de l'exécution de scripts Python/Node.js, de l'exécution d'opérations git, de la création de systèmes d'exécution de code IA, de la création d'environnements de développement interactifs ou de la mise en œuvre de flux de travail CI/CD qui nécessitent toutes les capacités du système d'exploitation.

À utiliser dans les cas suivants : configuration de Cloudflare Sandbox, exécution de code Python/Node.js en toute sécurité, gestion d'environnements de développement avec état, implémentation d'interpréteurs de code IA, exécution de commandes shell de manière isolée, gestion des référentiels git par programme, création d'agents de codage basés sur le chat, création d'environnements de construction temporaires, traitement de fichiers avec des outils système (ffmpeg, imagemagick, etc.) ou lorsque vous rencontrez des problèmes avec le cycle de vie des conteneurs, la gestion des sessions ou la persistance de l'état.
