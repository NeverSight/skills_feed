Guide complet pour les API traditionnelles/sans état d'OpenAI : achèvements de chat (GPT-5, GPT-4o), intégrations,
Images (DALL-E 3), audio (Whisper + TTS) et modération. Inclut à la fois le SDK Node.js et le SDK basé sur la récupération
approches pour une compatibilité maximale.

À utiliser lorsque : l'intégration des API OpenAI, la mise en œuvre de complétions de chat avec GPT-5/GPT-4o, la génération de texte
avec le streaming, en utilisant des appels/outils de fonctions, en créant des sorties structurées avec des schémas JSON,
implémenter des intégrations pour RAG, générer des images avec DALL-E 3, transcrire de l'audio avec Whisper,
synthétiser la parole avec TTS, modérer le contenu, déployer sur Cloudflare Workers ou rencontrer
erreurs telles que limites de débit (429), clés API non valides (401), échecs d'appel de fonction, analyse en streaming
erreurs, incohérences de dimensions d'intégration ou limite de jetons dépassée.
