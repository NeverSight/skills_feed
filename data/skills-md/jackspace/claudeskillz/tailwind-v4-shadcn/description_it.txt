Configurazione testata in produzione per Tailwind CSS v4 con shadcn/ui, Vite e React.

Utilizzare quando: inizializzare progetti React con Tailwind v4, configurare shadcn/ui,
implementazione della modalità oscura, debug dei problemi delle variabili CSS, correzione del cambio di tema,
migrazione da Tailwind v3 o riscontro di problemi relativi a colori/temi.

Copertine: pattern in linea @theme, architettura variabile CSS, modalità oscura con
ThemeProvider, composizione dei componenti, configurazione di vite.config, trucchi comuni della v4,
e modelli testati in produzione.

Parole chiave: Tailwind v4, shadcn/ui, @tailwindcss/vite, @theme inline, modalità oscura,
Variabili CSS, wrapper hsl(), componenti.json, temi React, cambio tema,
colori non funzionanti, variabili interrotte, tema non applicabile, direttiva @plugin,
plugin di tipografia, plugin di moduli, lezione di prosa, @tailwindcss/typography,
@tailwindcss/forms
