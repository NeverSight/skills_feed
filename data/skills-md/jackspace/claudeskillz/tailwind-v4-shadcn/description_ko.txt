shadcn/ui, Vite 및 React를 사용하여 Tailwind CSS v4에 대한 프로덕션 테스트를 거친 설정입니다.

사용 시기: Tailwind v4로 React 프로젝트 초기화, shadcn/ui 설정,
다크 모드 구현, CSS 변수 문제 디버깅, 테마 전환 수정,
Tailwind v3에서 마이그레이션하거나 색상/테마 문제가 발생합니다.

커버: @theme 인라인 패턴, CSS 변수 아키텍처, 다크 모드
ThemeProvider, 구성요소 구성, vite.config 설정, 공통 v4 문제,
생산 테스트를 거친 패턴.

키워드: Tailwind v4, shadcn/ui, @tailwindcss/vite, @theme 인라인, 다크 모드,
CSS 변수, hsl() 래퍼, Components.json, React 테마, 테마 전환,
색상이 작동하지 않음, 변수가 깨짐, 테마가 적용되지 않음, @plugin 지시문,
타이포그래피 플러그인, 양식 플러그인, 산문 클래스, @tailwindcss/typography,
