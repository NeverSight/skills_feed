Проверенная на производстве настройка Tailwind CSS v4 с shadcn/ui, Vite и React.

Используйте при: инициализации проектов React с помощью Tailwind v4, настройке shadcn/ui,
реализация темного режима, отладка проблем с переменными CSS, исправление переключения тем,
при переходе с Tailwind v3 или при возникновении проблем с цветом/темами.

Обложки: встроенный шаблон @theme, переменная архитектура CSS, темный режим с
ThemeProvider, состав компонентов, настройка vite.config, распространенные ошибки v4,
и проверенные на производстве модели.

Ключевые слова: Tailwind v4, shadcn/ui, @tailwindcss/vite, встроенная @theme, темный режим,
Переменные CSS, оболочка hsl(), компоненты.json, темы React, переключение тем,
цвета не работают, переменные не работают, тема не применяется, директива @plugin,
плагин типографии, плагин форм, класс прозы, @tailwindcss/typography,
@tailwindcss/формы
