إعداد تم اختباره على مستوى الإنتاج لـ Tailwind CSS v4 مع shadcn/ui وVite وReact.

يُستخدم عندما: تهيئة مشاريع React باستخدام Tailwind v4، وإعداد shadcn/ui،
تنفيذ الوضع المظلم، وتصحيح مشكلات متغيرات CSS، وإصلاح تبديل السمات،
الترحيل من Tailwind v3، أو مواجهة مشاكل في اللون/السمات.

الأغطية: @theme نمط مضمن، بنية CSS متغيرة، الوضع المظلم مع
ThemeProvider، تكوين المكونات، إعداد vite.config، الأخطاء الشائعة في الإصدار 4،
والأنماط التي تم اختبارها في الإنتاج.

الكلمات الرئيسية: Tailwind v4، shadcn/ui، @tailwindcss/vite، @theme مضمن، الوضع المظلم،
متغيرات CSS، غلاف hsl()، المكونات.json، سمات React، تبديل السمات،
الألوان لا تعمل، المتغيرات معطلة، السمة لا تنطبق، توجيه @plugin،
البرنامج المساعد للطباعة، البرنامج المساعد للنماذج، فئة النثر، @tailwindcss / الطباعة،
