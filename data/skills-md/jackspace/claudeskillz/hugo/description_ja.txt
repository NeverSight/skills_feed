このスキルは、Hugo 静的サ​​イト ジェネレーターを使用して静的 Web サイトを構築するための包括的な知識を提供します。これは、Hugo プロジェクト (ブログ、ドキュメント サイト、ランディング ページ、ポートフォリオ) のセットアップ、カスタム スタイルのための Tailwind CSS v4 の統合、ヘッドレス CMS システム (Sveltia CMS または TinaCMS) の統合、静的アセットを使用した Cloudflare Workers へのデプロイ、テーマとテンプレートの設定、一般的な Hugo セットアップ エラーの防止の際に使用する必要があります。

次のシナリオに遭遇したときにこのスキルを使用します: 新しい Hugo サイトのスキャフォールディング、Hugo Extended エディションと Standard エディションの選択、Tailwind CSS v4 と Hugo Pipes の統合、hugo.yaml または Hugo.toml ファイルの構成、Git サブモジュールを介した PaperMod またはその他のテーマの統合、コンテンツ管理のための Sveltia CMS または TinaCMS のセットアップ、Cloudflare Workers または Pages へのデプロイ、baseURL 構成のトラブルシューティング、テーマのインストールの解決エラー、フロントマター形式の問題 (YAML と TOML) の修正、日付関連のビルド失敗の防止、Hugo を使用した PostCSS のセットアップ、または GitHub Actions を使用した CI/CD のセットアップ。
