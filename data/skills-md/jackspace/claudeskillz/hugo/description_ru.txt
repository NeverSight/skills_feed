Этот навык дает исчерпывающие знания по созданию статических веб-сайтов с помощью генератора статических сайтов Hugo. Его следует использовать при настройке проектов Hugo (блоги, сайты документации, целевые страницы, портфолио), интеграции Tailwind CSS v4 для индивидуального оформления, интеграции автономных систем CMS (Sveltia CMS или TinaCMS), развертывании в Cloudflare Workers со статическими активами, настройке тем и шаблонов и предотвращении распространенных ошибок установки Hugo.

Используйте этот навык при работе со следующими сценариями: создание новых сайтов Hugo, выбор между расширенной и стандартной редакциями Hugo, интеграция Tailwind CSS v4 с Hugo Pipes, настройка файлов humo.yaml или humo.toml, интеграция PaperMod или других тем через подмодули Git, настройка Sveltia CMS или TinaCMS для управления контентом, развертывание в Cloudflare Workers или Pages, устранение неполадок в конфигурации baseURL, решение проблемы установки темы. ошибок, исправление проблем с форматом фронтмена (YAML и TOML), предотвращение сбоев сборки, связанных с датой, настройка PostCSS с помощью Hugo или настройка CI/CD с помощью GitHub Actions.
