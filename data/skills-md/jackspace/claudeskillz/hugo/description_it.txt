Questa competenza fornisce una conoscenza completa per la creazione di siti Web statici con il generatore di siti statici Hugo. Dovrebbe essere utilizzato durante l'impostazione di progetti Hugo (blog, siti di documentazione, pagine di destinazione, portfolio), l'integrazione di Tailwind CSS v4 per uno stile personalizzato, l'integrazione di sistemi CMS headless (Sveltia CMS o TinaCMS), la distribuzione ai dipendenti Cloudflare con risorse statiche, la configurazione di temi e modelli e la prevenzione degli errori comuni di installazione di Hugo.

Utilizzare questa competenza quando si incontrano questi scenari: impalcatura di nuovi siti Hugo, scelta tra le edizioni Hugo Extended e Standard, integrazione di Tailwind CSS v4 con Hugo Pipes, configurazione di file Hugo.yaml o Hugo.toml, integrazione di PaperMod o altri temi tramite sottomoduli Git, impostazione di Sveltia CMS o TinaCMS per la gestione dei contenuti, distribuzione su Cloudflare Workers o Pages, risoluzione dei problemi di configurazione di baseURL, risoluzione degli errori di installazione del tema, correzione problemi di formato frontmatter (YAML vs TOML), prevenzione di errori di compilazione legati alla data, configurazione di PostCSS con Hugo o configurazione di CI/CD con GitHub Actions.
