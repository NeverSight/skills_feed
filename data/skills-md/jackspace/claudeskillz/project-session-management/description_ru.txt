Управляет состоянием сеанса и передачей контекста для многосессионных проектов с помощью протокола передачи обслуживания сеанса. Создает и поддерживает SESSION.md для отслеживания хода выполнения фазы, контрольных точек git и следующих действий при очистке контекста. Интегрируется с навыками планирования проектов для преобразования IMPLEMENTATION_PHASES.md в отслеживаемое состояние сеанса. Используйте при запуске новых проектов после планирования, возобновлении работы после очистки контекста или управлении сложными многоэтапными реализациями.
Ключевые слова: управление сеансами, SESSION.md, протокол передачи сеанса, передача контекста, многосессионные проекты, отслеживание фаз, контрольные точки git, отслеживание состояния сеанса, возобновление работы, очистка контекста, отслеживание прогресса фазы, этапы реализации, этап проверки, этап отладки, отслеживание следующих действий, непрерывность работы, восстановление сеанса, управление контекстом, отслеживание поэтапного внедрения
