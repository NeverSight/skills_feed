Utilizzare questa competenza per elaborare e analizzare set di dati tabulari di grandi dimensioni (miliardi di righe) che superano la RAM disponibile. Vaex eccelle nelle operazioni DataFrame out-of-core, nella valutazione pigra, nelle aggregazioni veloci, nella visualizzazione efficiente di big data e nell'apprendimento automatico su set di dati di grandi dimensioni. Applicalo quando gli utenti devono lavorare con file CSV/HDF5/Arrow/Parquet di grandi dimensioni, eseguire statistiche rapide su set di dati di grandi dimensioni, creare visualizzazioni di big data o creare pipeline ML che non rientrano nella memoria.
