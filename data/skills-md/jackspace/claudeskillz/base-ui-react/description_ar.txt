الإعداد الذي تم اختباره على الإنتاج لواجهة المستخدم الأساسية (@base-ui-components/react) - مكتبة المكونات غير المصممة لواجهة المستخدم المتعددة اللغات (MUI)
يوفر مكونات React يمكن الوصول إليها وقابلة للتخصيص باستخدام نمط عرض الدعائم. هذه المهارة
يجب استخدامه عند إنشاء واجهات مستخدم يمكن الوصول إليها مع التحكم الكامل في التصميم، والانتقال من Radix UI،
أو تحتاج إلى مكونات ذات تكامل واجهة المستخدم العائمة لتحديد المواقع الذكية.

يُستخدم عندما: إعداد واجهة المستخدم الأساسية في مشاريع Vite + React، والانتقال من Radix UI إلى Base UI،
تنفيذ المكونات التي يمكن الوصول إليها (Dialog، Select، Popover، Tooltip، NumberField، Accordion)،
مواجهة مشكلات تحديد المواقع مع النوافذ المنبثقة، والحاجة إلى واجهة برمجة تطبيقات دعم العرض بدلاً من نمط الطفل،
البناء باستخدام Tailwind v4 + shadcn/ui، أو النشر إلى Cloudflare Workers.

⚠️ حالة الإصدار التجريبي: واجهة المستخدم الأساسية هي v1.0.0-beta.4. الإصدار 1.0 المستقر المتوقع في الربع الرابع من عام 2025. توفر هذه المهارة
الحلول البديلة لمشكلات بيتا المعروفة والإرشادات حول استقرار واجهة برمجة التطبيقات.

الكلمات الرئيسية: base-ui، @base-ui-components/react، واجهة المستخدم الأساسية mui، المكونات غير المصممة، المكونات التي يمكن الوصول إليها،
عرض الدعائم، بديل الجذر، ترحيل الجذر، واجهة المستخدم العائمة، نمط الموضع، واجهة المستخدم بدون رأس،
مربع حوار يمكن الوصول إليه، تحديد يمكن الوصول إليه، نافذة منبثقة يمكن الوصول إليها، تلميح أدوات يمكن الوصول إليه، أكورديون يمكن الوصول إليه،
حقل الرقم، مكونات التفاعل، مكونات الريح الخلفية، تفاعل Vite، واجهة المستخدم لعمال Cloudflare،
