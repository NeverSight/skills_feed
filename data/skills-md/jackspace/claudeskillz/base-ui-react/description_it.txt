Configurazione testata in produzione per Base UI (@base-ui-components/react): libreria di componenti senza stile di MUI
che fornisce componenti React accessibili e personalizzabili utilizzando il modello degli oggetti di rendering. Questa abilità
dovrebbe essere utilizzato durante la creazione di interfacce utente accessibili con controllo completo dello stile, migrazione da Radix UI,
o che necessitano di componenti con integrazione dell'interfaccia utente mobile per il posizionamento intelligente.

Utilizzare quando: configurazione dell'interfaccia utente di base nei progetti Vite + React, migrazione da Radix UI a Base UI,
implementare componenti accessibili (Dialog, Select, Popover, Tooltip, NumberField, Accordion),
riscontrando problemi di posizionamento con i popup, necessitando dell'API render prop invece del pattern asChild,
creazione con Tailwind v4 + shadcn/ui o distribuzione su Cloudflare Workers.

⚠️ STATO BETA: l'interfaccia utente di base è v1.0.0-beta.4. La versione stabile 1.0 è prevista per il quarto trimestre del 2025. Questa competenza fornisce
soluzioni alternative per problemi beta noti e indicazioni sulla stabilità dell'API.
