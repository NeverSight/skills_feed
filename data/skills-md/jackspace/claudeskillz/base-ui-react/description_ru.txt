Проверенная на производстве установка для Base UI (@base-ui-comComponents/react) — библиотека компонентов без стилей MUI.
который предоставляет доступные, настраиваемые компоненты React с использованием шаблона рендеринга реквизита. Этот навык
следует использовать при создании доступных пользовательских интерфейсов с полным контролем над стилем, при переходе с пользовательского интерфейса Radix,
или вам нужны компоненты с интеграцией плавающего пользовательского интерфейса для интеллектуального позиционирования.

Используйте при: настройке базового пользовательского интерфейса в проектах Vite + React, переходе с пользовательского интерфейса Radix на базовый интерфейс,
реализация доступных компонентов (Dialog, Select, Popover, Tooltip, NumberField, Accordion),
возникают проблемы с позиционированием всплывающих окон, требуется API рендеринга вместо шаблона asChild,
сборка с помощью Tailwind v4 + shadcn/ui или развертывание в Cloudflare Workers.

⚠️ БЕТА-СТАТУС: базовый пользовательский интерфейс — v1.0.0-beta.4. Стабильная версия 1.0 ожидается в четвертом квартале 2025 года. Этот навык обеспечивает
обходные пути для известных проблем бета-версии и рекомендации по стабильности API.
