Base UI の実稼働テスト済みセットアップ (@base-ui-components/react) - MUI のスタイルなしコンポーネント ライブラリ
render props パターンを使用して、アクセス可能でカスタマイズ可能な React コンポーネントを提供します。このスキル
完全なスタイル コントロールを備えたアクセシブルな UI を構築する場合、Radix UI から移行する場合に使用する必要があります。
または、スマートな位置決めのためのフローティング UI 統合を備えたコンポーネントが必要です。

次の場合に使用します: Vite + React プロジェクトでの Base UI のセットアップ、Radix UI から Base UI への移行、
アクセシブルなコンポーネント (ダイアログ、選択、ポップオーバー、ツールチップ、ナンバーフィールド、アコーディオン) の実装、
ポップアップで位置の問題が発生し、asChild パターンの代わりにレンダリング プロップ API が必要になりました。
Tailwind v4 + shadcn/ui で構築するか、Cloudflare Workers にデプロイします。

⚠️ ベータステータス: 基本 UI は v1.0.0-beta.4 です。安定版 v1.0 は 2025 年第 4 四半期に予定されています。このスキルは以下を提供します。
既知のベータ版の問題の回避策と API の安定性に関するガイダンス。
