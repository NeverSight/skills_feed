經過生產測試的 Base UI 設置 (@base-ui-components/react) - MUI 的無樣式組件庫
它使用渲染道具模式提供可訪問的、可定制的 React 組件。這個技能
在構建具有完整樣式控制的可訪問 UI 時應使用，從 Radix UI 遷移，
或者需要具有浮動 UI 集成的組件以進行智能定位。

使用時機：在Vite + React項目中設置Base UI，從Radix UI遷移到Base UI，
實現可訪問的組件（Dialog、Select、Popover、Tooltip、NumberField、Accordion），
遇到彈出窗口的定位問題，需要 render prop API 而不是 asChild 模式，
使用 Tailwind v4 + shadcn/ui 進行構建，或部署到 Cloudflare Workers。

⚠️ 測試​​版狀態：基本 UI 是 v1.0.0-beta.4。穩定版 v1.0 預計於 2025 年第 4 季度發布。此技能提供
已知測試版問題的解決方法以及 API 穩定性指南。
