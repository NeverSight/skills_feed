Dominio di conoscenza completo per Cloudflare Hyperdrive: connessione di Cloudflare Workers ai database PostgreSQL e MySQL esistenti con pooling di connessioni globale, memorizzazione nella cache delle query e latenza ridotta.

Utilizzare quando: si connette Worker a database esistenti, si esegue la migrazione da PostgreSQL/MySQL a Cloudflare, si imposta il pool di connessioni, si configurano collegamenti Hyperdrive, si utilizzano driver node-postgres/postgres.js/mysql2, si integra Drizzle ORM o Prisma ORM o si riscontra "Impossibile acquisire una connessione dal pool", "TLS non supportato dal database", "connessione rifiutata", "nodejs_compat mancante", "Generazione di codice da stringhe non consentite" o errori di configurazione Hyperdrive.
