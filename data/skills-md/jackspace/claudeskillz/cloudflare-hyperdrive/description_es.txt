Dominio de conocimiento completo para Cloudflare Hyperdrive: conecta Cloudflare Workers a bases de datos PostgreSQL y MySQL existentes con agrupación de conexiones global, almacenamiento en caché de consultas y latencia reducida.

Úselo cuando: conecte Workers a bases de datos existentes, migre PostgreSQL/MySQL a Cloudflare, configure un grupo de conexiones, configure enlaces Hyperdrive, use controladores node-postgres/postgres.js/mysql2, integre Drizzle ORM o Prisma ORM, o encuentre "Error al adquirir una conexión del grupo", "TLS no es compatible con la base de datos", "conexión rechazada", "nodejs_compat falta", "Generación de código a partir de cadenas disallowed" o errores de configuración del Hyperdrive.
