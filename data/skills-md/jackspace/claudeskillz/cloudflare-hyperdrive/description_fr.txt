Domaine de connaissances complet pour Cloudflare Hyperdrive : connexion de Cloudflare Workers aux bases de données PostgreSQL et MySQL existantes avec un pool de connexions global, une mise en cache des requêtes et une latence réduite.

À utiliser lors de : la connexion de Workers à des bases de données existantes, la migration de PostgreSQL/MySQL vers Cloudflare, la configuration du pool de connexions, la configuration des liaisons Hyperdrive, l'utilisation de pilotes node-postgres/postgres.js/mysql2, l'intégration de Drizzle ORM ou Prisma ORM, ou la rencontre de "Échec de l'acquisition d'une connexion à partir du pool", "TLS non pris en charge par la base de données", "connexion refusée", "nodejs_compat manquant", "Génération de code à partir de chaînes interdit", ou des erreurs de configuration Hyperdrive.
