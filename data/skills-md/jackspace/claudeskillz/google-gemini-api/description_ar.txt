الدليل الكامل لـ Google Gemini API باستخدام حزمة SDK الحالية الصحيحة (@google/genai v1.27+، وليس
تم إهماله @google/geneative-ai). يغطي إنشاء النص، والمدخلات متعددة الوسائط (نص + صور + فيديو +
الصوت + ملفات PDF)، واستدعاء الوظائف، ووضع التفكير، والبث، وتعليمات النظام بدقة 2025
معلومات النموذج (Gemini 2.5 Pro/Flash/Flash-Lite مع 1 مليون رمز إدخال، وليس 2 مليون).

يُستخدم عندما: دمج Gemini API، وتنفيذ تطبيقات الذكاء الاصطناعي متعددة الوسائط، واستخدام وضع التفكير لـ
المنطق المعقد، استدعاء الوظائف مع التنفيذ المتوازي، الاستجابات المتدفقة، النشر إلى
عمال Cloudflare، أو إنشاء تطبيقات الدردشة، أو مواجهة تحذيرات إهمال SDK، نافذة السياق
أخطاء، أو أخطاء لم يتم العثور على النموذج، أو فشل استدعاء الوظيفة، أو أخطاء التنسيق المتعدد الوسائط.
