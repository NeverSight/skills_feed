Previeni i timeout delle richieste nelle sessioni di Claude Code suddividendo operazioni lunghe, implementando checkpoint di avanzamento, utilizzando processi in background e ottimizzando i modelli di utilizzo degli strumenti. Da utilizzare quando si eseguono operazioni in blocco, si elaborano set di dati di grandi dimensioni, si eseguono comandi con esecuzione prolungata, si scaricano più file o si eseguono attività che superano i limiti di timeout di 2 minuti. Implementa strategie come la scomposizione delle attività, l'elaborazione incrementale, l'esecuzione parallela, i modelli di ripristino del checkpoint e la gestione efficiente delle risorse per garantire il completamento affidabile di flussi di lavoro complessi.
