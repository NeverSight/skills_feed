Génère des clients API prêts pour la production avec des types TypeScript, une logique de nouvelle tentative, une limitation de débit, une authentification (OAuth, clés API), une gestion des erreurs et des réponses fictives. À utiliser lorsque l'utilisateur dit « intégrer l'API », « client API », « se connecter au service » ou demande l'intégration de services tiers.
