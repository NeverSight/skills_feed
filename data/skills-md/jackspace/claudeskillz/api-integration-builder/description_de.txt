Erzeugt produktionsbereite API-Clients mit TypeScript-Typen, Wiederholungslogik, Ratenbegrenzung, Authentifizierung (OAuth, API-Schlüssel), Fehlerbehandlung und Scheinantworten. Wird verwendet, wenn der Benutzer „API integrieren“, „API-Client“, „Mit Dienst verbinden“ sagt oder die Integration von Drittanbieterdiensten anfordert.
