Создает готовые к использованию клиенты API с типами TypeScript, логикой повторных попыток, ограничением скорости, аутентификацией (OAuth, ключи API), обработкой ошибок и ложными ответами. Используйте, когда пользователь говорит «интегрировать API», «API-клиент», «подключиться к сервису» или запрашивает интеграцию стороннего сервиса.
