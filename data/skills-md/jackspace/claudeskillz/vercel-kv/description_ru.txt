Этот навык дает всесторонние знания для интеграции Vercel KV (Redis-совместимого хранилища ключей и значений на базе Upstash) в приложения Vercel. Его следует использовать при настройке Vercel KV для приложений Next.js, реализации шаблонов кэширования, управления сеансами или обработки ограничения скорости в пограничных и бессерверных функциях.

Используйте этот навык, когда:
- Настройка Vercel KV для приложений Next.js.
- Реализация стратегий кэширования (кэш страниц, кеш API, кеш данных)
- Управление сеансами пользователей или токенами аутентификации в бессерверных средах.
- Ограничение скорости создания API или функций.
- Хранение временных данных с TTL (время жизни)
- Миграция с Cloudflare KV на Vercel KV
- Встречаются такие ошибки, как «KV_REST_API_URL не установлен», «превышен предел скорости» или «ошибки сериализации JSON».
- Нужен Redis-совместимый API с строгой согласованностью (по сравнению с возможной согласованностью).
