Используйте этот навык при создании серверов MCP (Model Context Protocol) с TypeScript на Cloudflare Workers. Этот навык предоставляет проверенные на производстве шаблоны для реализации инструментов, ресурсов и подсказок с использованием официального @modelcontextprotocol/sdk. Он предотвращает более 10 распространенных ошибок, включая проблемы с синтаксисом экспорта, сбои проверки схемы, утечки памяти из-за незакрытых транспортов, неправильные настройки CORS и уязвимости аутентификации. Этот навык следует использовать, когда разработчикам нужны серверы MCP без отслеживания состояния для интеграции API, использования внешних инструментов или развертывания периферийных устройств без серверов. Для агентов с отслеживанием состояния с WebSockets и постоянным хранилищем вместо этого рассмотрите SDK Cloudflare Agents. Поддерживает несколько методов аутентификации (ключи API, OAuth, нулевое доверие), интеграцию сервисов Cloudflare (D1, KV, R2, Vectorize) и комплексные стратегии тестирования. Проверено производство: экономия токенов составила ~70 % по сравнению с ручным внедрением.
