Utilice esta habilidad al crear servidores MCP (Protocolo de contexto modelo) con TypeScript en Cloudflare Workers. Esta habilidad proporciona patrones probados en producción para implementar herramientas, recursos e indicaciones utilizando el @modelcontextprotocol/sdk oficial. Previene más de 10 errores comunes, incluidos problemas de sintaxis de exportación, fallas de validación de esquemas, pérdidas de memoria por transportes no cerrados, configuraciones incorrectas de CORS y vulnerabilidades de autenticación. Esta habilidad debe usarse cuando los desarrolladores necesitan servidores MCP sin estado para integraciones de API, exposición a herramientas externas o implementaciones perimetrales sin servidor. Para agentes con estado con WebSockets y almacenamiento persistente, considere el SDK de agentes de Cloudflare. Admite múltiples métodos de autenticación (claves API, OAuth, Zero Trust), integraciones de servicios Cloudflare (D1, KV, R2, Vectorize) y estrategias de prueba integrales. Producción probada con ahorros simbólicos de ~70 % frente a la implementación manual.
