Cloudflare Workers 上で TypeScript を使用して MCP (Model Context Protocol) サーバーを構築する場合は、このスキルを使用します。このスキルは、公式 @modelcontextprotocol/sdk を使用してツール、リソース、プロンプトを実装するための実稼働テスト済みのパターンを提供します。エクスポート構文の問題、スキーマ検証の失敗、閉じられていないトランスポートからのメモリ リーク、CORS の構成ミス、認証の脆弱性など、10 以上の一般的なエラーを防ぎます。このスキルは、開発者が API 統合、外部ツールの公開、またはサーバーレス エッジ展開のためにステートレス MCP サーバーを必要とする場合に使用する必要があります。 WebSocket と永続ストレージを備えたステートフル エージェントの場合は、代わりに Cloudflare Agents SDK を検討してください。複数の認証方法 (API キー、OAuth、ゼロ トラスト)、Cloudflare サービス統合 (D1、KV、R2、Vectorize)、および包括的なテスト戦略をサポートします。実稼働テストでは、手動実装と比較してトークンを最大 70% 節約できました。
