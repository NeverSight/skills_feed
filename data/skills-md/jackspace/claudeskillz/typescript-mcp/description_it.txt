Utilizza questa competenza durante la creazione di server MCP (Model Context Protocol) con TypeScript su Cloudflare Workers. Questa competenza fornisce modelli testati in produzione per l'implementazione di strumenti, risorse e istruzioni utilizzando il @modelcontextprotocol/sdk ufficiale. Previene oltre 10 errori comuni tra cui problemi di sintassi di esportazione, errori di convalida dello schema, perdite di memoria dovute a trasporti non chiusi, configurazioni errate CORS e vulnerabilità di autenticazione. Questa competenza dovrebbe essere utilizzata quando gli sviluppatori necessitano di server MCP stateless per integrazioni API, esposizione a strumenti esterni o distribuzioni edge serverless. Per gli agenti con stato con WebSocket e archiviazione persistente, considera invece Cloudflare Agents SDK. Supporta più metodi di autenticazione (chiavi API, OAuth, Zero Trust), integrazioni di servizi Cloudflare (D1, KV, R2, Vectorize) e strategie di test complete. Produzione testata con un risparmio simbolico di circa il 70% rispetto all'implementazione manuale.
