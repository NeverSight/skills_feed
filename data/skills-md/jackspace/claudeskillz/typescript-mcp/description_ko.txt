Cloudflare Workers에서 TypeScript를 사용하여 MCP(모델 컨텍스트 프로토콜) 서버를 구축할 때 이 기술을 사용하세요. 이 기술은 공식 @modelcontextprotocol/sdk를 사용하여 도구, 리소스 및 프롬프트를 구현하기 위한 프로덕션 테스트 패턴을 제공합니다. 내보내기 구문 문제, 스키마 유효성 검사 실패, 닫히지 않은 전송으로 인한 메모리 누수, CORS 구성 오류, 인증 취약성을 비롯한 10개 이상의 일반적인 오류를 방지합니다. 개발자가 API 통합, 외부 도구 노출 또는 서버리스 에지 배포를 위해 상태 비저장 MCP 서버가 필요한 경우 이 기술을 사용해야 합니다. WebSocket과 영구 저장소를 갖춘 상태 저장 에이전트의 경우 대신 Cloudflare Agents SDK를 고려하세요. 다양한 인증 방법(API 키, OAuth, Zero Trust), Cloudflare 서비스 통합(D1, KV, R2, Vectorize) 및 포괄적인 테스트 전략을 지원합니다. 수동 구현에 비해 ~70%의 토큰 절약으로 프로덕션 테스트를 거쳤습니다.
