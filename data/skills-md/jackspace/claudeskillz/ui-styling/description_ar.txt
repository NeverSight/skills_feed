قم بإنشاء واجهات مستخدم جميلة ويمكن الوصول إليها باستخدام مكونات shadcn/ui (مبنية على Radix UI + Tailwind)، وتصميم الأداة المساعدة Tailwind CSS أولاً، والتصميمات المرئية القائمة على القماش. يُستخدم عند إنشاء واجهات المستخدم، أو تنفيذ أنظمة التصميم، أو إنشاء تخطيطات سريعة الاستجابة، أو إضافة مكونات يمكن الوصول إليها (مربعات الحوار، والقوائم المنسدلة، والنماذج، والجداول)، أو تخصيص السمات والألوان، أو تنفيذ الوضع الداكن، أو إنشاء تصميمات وملصقات مرئية، أو إنشاء أنماط تصميم متسقة عبر التطبيقات.
