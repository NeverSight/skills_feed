توفر هذه المهارة معرفة شاملة لتطوير المكونات الإضافية لـ WordPress، والتي تغطي الأنماط الأساسية وأفضل ممارسات الأمان وتفاعلات قاعدة البيانات والخطافات/المرشحات وواجهة برمجة تطبيقات الإعدادات وأنواع المنشورات المخصصة وREST API وAJAX. يجب استخدام هذه المهارة عند إنشاء مكونات WordPress الإضافية، أو استكشاف مشكلات الأمان وإصلاحها، أو تنفيذ أنواع/تصنيفات المنشورات المخصصة، أو إنشاء واجهات الإدارة، أو العمل مع قاعدة بيانات WordPress.

يُستخدم عندما: إنشاء مكونات WordPress الإضافية الجديدة، وتنفيذ noces/sanitization/escaping، والعمل مع $wpdb والبيانات المعدة، وإنشاء صفحات API للإعدادات، وتسجيل أنواع المنشورات المخصصة أو التصنيفات، وتنفيذ نقاط نهاية REST API، والتعامل مع طلبات AJAX، وتصحيح مشكلات تنشيط/إلغاء تنشيط المكونات الإضافية، ومنع ثغرات حقن SQL/XSS/CSRF.

الكلمات الرئيسية: تطوير البرنامج المساعد ووردبريس، أمان ووردبريس، خطافات ووردبريس، مرشحات ووردبريس، قاعدة بيانات ووردبريس، إعداد wpdb، sanitize_text_field، esc_html، wp_nonce، نوع المنشور المخصص، Register_post_type، إعدادات API، بقية API، admin-ajax، حقن Wordpress sql، wordpress xss، wordpress csrf، رأس البرنامج المساعد، ربط التنشيط، ربط التنشيط، معايير ترميز ووردبريس، ووردبريس بنية البرنامج المساعد
