此技能提供了实施 Cloudflare Turnstile（验证码替代机器人保护系统）的全面知识。当将机器人防护集成到表单、登录页面、注册流程或任何需要防止垃圾邮件/机器人的面向用户的功能时，应该使用它。 Turnstile 在后台运行隐形挑战，在阻止自动流量的同时保持出色的用户体验。

使用场合：向表单添加机器人保护、实现登录安全、保护 API 端点免遭滥用、从 reCAPTCHA/hCaptcha 迁移、使用 Turnstile 遇到 CSP 错误、处理令牌验证失败、使用 Turnstile 实施 E2E 测试、与 React/Next.js/Hono 应用程序集成或调试错误代码 100*、300*、600*。
