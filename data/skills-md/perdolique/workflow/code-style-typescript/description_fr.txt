Guide de style de code TypeScript et conventions de formatage. À utiliser lors de l'écriture de code TypeScript, de la révision de fichiers TypeScript, de la refactorisation du code .ts, du formatage de TypeScript ou lorsque vous travaillez avec des interfaces, des classes, des fonctions ou tout autre fichier .ts de TypeScript. Appliquez ces règles lors de la génération de code, de la révision du code et lorsque l'utilisateur mentionne le style TypeScript, le formatage, les conventions, les points-virgules ou la qualité du code.
