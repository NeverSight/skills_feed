Guía de estilo de código TypeScript y convenciones de formato. Úselo al escribir código TypeScript, revisar archivos TypeScript, refactorizar código .ts, formatear TypeScript o cuando trabaje con interfaces, clases, funciones o cualquier archivo .ts de TypeScript. Aplique estas reglas durante la generación y revisión del código y cuando el usuario mencione el estilo, el formato, las convenciones, el punto y coma o la calidad del código de TypeScript.
