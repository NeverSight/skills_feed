Используйте при фиксации кода, промежуточной фиксации изменений, написании сообщений фиксации, запросе фиксации git или когда пользователь упоминает фиксацию, сообщение фиксации, обычные фиксации, семантические фиксации, git commit, этапные изменения, создание фиксации. Поддерживает отслеживание версий пакетов монорепо.
