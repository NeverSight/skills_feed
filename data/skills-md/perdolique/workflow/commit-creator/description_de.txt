Generieren Sie semantische Commit-Nachrichten nach dem herkömmlichen Commit-Format. Wird verwendet, wenn Code festgeschrieben, Änderungen bereitgestellt, Festschreibungsnachrichten geschrieben, Git-Festschreibungen angefordert werden oder wenn der Benutzer Festschreibung, Festschreibungsnachricht, herkömmliche Festschreibungen, semantische Festschreibungen, Git-Festschreibung, Phasenänderungen und Commit erstellen erwähnt. Unterstützt die Versionsverfolgung von Monorepo-Paketen.
