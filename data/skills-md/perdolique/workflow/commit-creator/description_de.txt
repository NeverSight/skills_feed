Generieren Sie semantische Commit-Nachrichten nach dem herkömmlichen Commit-Format. Wird verwendet, wenn Sie Code festschreiben, Änderungen bereitstellen, Festschreibungsnachrichten schreiben, Git-Festschreibung anfordern oder wenn der Benutzer Festschreibung, Festschreibungsnachricht, herkömmliche Festschreibungen, semantische Festschreibungen, Git-Festschreibung, Phasenänderungen und Commit erstellen erwähnt. Unterstützt die Versionsverfolgung von Monorepo-Paketen.
