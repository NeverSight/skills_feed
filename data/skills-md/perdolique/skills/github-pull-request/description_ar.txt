قم بإنشاء طلبات سحب GitHub مباشرةً عبر واجهة برمجة التطبيقات (API) من تغييرات التعليمات البرمجية. يحلل التزامات الفرع، وينشئ عناوين وأوصاف العلاقات العامة، وينشئ علاقات عامة على GitHub بشكل افتراضي. يُستخدم عندما يريد المستخدم إنشاء علاقات عامة، أو فتح علاقات عامة، أو إجراء علاقات عامة، أو إرسال علاقات عامة، أو تحديث العلاقات العامة، أو طلب سحب مذكور، أو علاقات عامة، أو طلب دمج، أو مراجعة التعليمات البرمجية، أو إعداد العلاقات العامة، أو سير عمل GitHub. يُستخدم أيضًا عندما يطلب المستخدم إنشاء محتوى علاقات عامة للنسخ/العرض (الوضع الاحتياطي).
