Создавайте запросы на извлечение GitHub напрямую через API на основе изменений кода. Анализирует фиксации ветвей, генерирует заголовки и описания PR, а также по умолчанию создает PR на GitHub. Используйте, когда пользователь хочет создать PR, открыть PR, сделать PR, отправить PR, обновить PR или упоминает запрос на включение, PR, запрос на слияние, проверку кода, подготовку PR, рабочий процесс GitHub. Также используйте, когда пользователь просит создать PR-контент для копирования/просмотра (резервный режим).
