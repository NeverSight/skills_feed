以專業 UI/UX 設計師和前端工程師的思維方式創建大膽、高端、生產級的前端界面。當用戶要求設計或重新設計必須讓人感覺有意為之、獨特且非通用的 Web 界面（組件、頁面、儀表板或應用程序）時，可以使用此技能。
該模型應積極從廣泛的現實世界設計參考（例如現代產品界面、編輯佈局、設計系統、實驗性網頁設計、Pinterest、Awwwards、Dribbble 等平台上看到的趨勢）中汲取靈感，而不復制任何單一來源。
輸出必須是真實的、可實施的代碼，但應避免僵化、類似模板或過度對稱的“AI 外觀”組件。佈局、間距、排版、層次結構和交互模式應該讓人感覺是設計的，而不是生成的。
優先考慮強烈的視覺指導、深思熟慮的用戶體驗決策和受控實驗，而不是安全或傳統的 UI 模式。
