Recupera, riepiloga e gestisci i segnalibri X/Twitter tramite bird CLI o X API v2. Da utilizzare quando: (1) l'utente dice "controlla i miei segnalibri", "cosa ho aggiunto ai segnalibri", "digest dei segnalibri", "riepiloga i miei segnalibri", "x segnalibri", "segnalibri di Twitter", (2) l'utente desidera un riepilogo periodico dei tweet salvati, (3) l'utente desidera classificare, cercare o analizzare i propri segnalibri, (4) digest dei segnalibri pianificati tramite cron. Autenticazione: CLI bird con cookie del browser, OPPURE X API v2 con token OAuth 2.0.
