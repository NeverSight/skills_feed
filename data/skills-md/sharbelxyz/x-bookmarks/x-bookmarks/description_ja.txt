Bird CLI または X API v2 を介して X/Twitter ブックマークを取得、要約、管理します。次の場合に使用します: (1) ユーザーが「ブックマークを確認する」、「何をブックマークしたか」、「ブックマーク ダイジェスト」、「ブックマークを要約する」、「x ブックマーク」、「twitter ブックマーク」、(2) ユーザーが保存されたツイートの定期的なダイジェストを希望する、(3) ユーザーがブックマークの分類、検索、または分析を希望する、(4) cron 経由でスケジュールされたブックマーク ダイジェスト。認証: ブラウザ Cookie を使用した Bird CLI、または OAuth 2.0 トークンを使用した X API v2。
