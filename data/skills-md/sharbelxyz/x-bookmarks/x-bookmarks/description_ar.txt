جلب إشارات X/Twitter وتلخيصها وإدارتها عبر Bird CLI أو X API v2. يُستخدم عندما: (1) يقول المستخدم "التحقق من إشاراتي المرجعية"، "ماذا قمت بوضع إشارة مرجعية"، "ملخص الإشارات المرجعية"، "تلخيص إشاراتي المرجعية"، "x الإشارات المرجعية"، "إشارات تويتر المرجعية"، (2) يريد المستخدم ملخصًا دوريًا للتغريدات المحفوظة، (3) يريد المستخدم تصنيف إشاراته المرجعية أو البحث عنها أو تحليلها، (4) ملخصات الإشارات المرجعية المجدولة عبر cron. المصادقة: Bird CLI مع ملفات تعريف الارتباط للمتصفح، أو X API v2 مع رموز OAuth 2.0.
