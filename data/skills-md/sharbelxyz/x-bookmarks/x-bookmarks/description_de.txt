X/Twitter-Lesezeichen über Bird CLI oder X API v2 abrufen, zusammenfassen und verwalten. Wird verwendet, wenn: (1) der Benutzer „Meine Lesezeichen prüfen“, „Was habe ich als Lesezeichen gesetzt“, „Lesezeichenübersicht“, „Meine Lesezeichen zusammenfassen“, „x Lesezeichen“, „Twitter-Lesezeichen“ sagt, (2) der Benutzer eine regelmäßige Zusammenfassung der gespeicherten Tweets wünscht, (3) der Benutzer seine Lesezeichen kategorisieren, durchsuchen oder analysieren möchte, (4) geplante Lesezeichenübersichten über cron. Authentifizierung: Bird CLI mit Browser-Cookies oder X API v2 mit OAuth 2.0-Tokens.
