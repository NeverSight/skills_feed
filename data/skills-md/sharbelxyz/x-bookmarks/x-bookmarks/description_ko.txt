Bird CLI 또는 X API v2를 통해 X/Twitter 북마크를 가져오고 요약하고 관리합니다. 사용 시기: (1) 사용자가 "내 북마크 확인", "내가 북마크한 내용", "북마크 다이제스트", "내 북마크 요약", "x 북마크", "트위터 북마크"라고 말하고, (2) 사용자가 저장된 트윗을 주기적으로 요약하고 싶어하고, (3) 사용자가 자신의 북마크를 분류, 검색 또는 분석하고 싶어하고, (4) cron을 통해 예약된 북마크 다이제스트를 원합니다. 인증: 브라우저 쿠키가 포함된 Bird CLI, OAuth 2.0 토큰이 포함된 X API v2.
