Obtenga, resuma y administre marcadores de X/Twitter a través de bird CLI o X API v2. Úselo cuando: (1) el usuario dice "revisar mis marcadores", "qué marqué como favorito", "resumen de marcadores", "resumir mis marcadores", "x marcadores", "marcadores de Twitter", (2) el usuario quiere un resumen periódico de los tweets guardados, (3) el usuario quiere categorizar, buscar o analizar sus marcadores, (4) resúmenes de marcadores programados a través de cron. Autenticación: CLI de bird con cookies del navegador, O X API v2 con tokens OAuth 2.0.
