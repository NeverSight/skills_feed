Récupérez, résumez et gérez les signets X/Twitter via bird CLI ou X API v2. À utiliser lorsque : (1) l'utilisateur dit "vérifier mes favoris", "qu'est-ce que j'ai ajouté aux favoris", "résumé des favoris", "résumer mes favoris", "x favoris", "signets Twitter", (2) l'utilisateur souhaite un résumé périodique des tweets enregistrés, (3) l'utilisateur souhaite catégoriser, rechercher ou analyser ses favoris, (4) les résumés de favoris programmés via cron. Auth : bird CLI avec cookies de navigateur, OU X API v2 avec jetons OAuth 2.0.
