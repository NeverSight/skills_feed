Получайте, суммируйте и управляйте закладками X/Twitter через Bird CLI или X API v2. Используйте, когда: (1) пользователь говорит «проверьте мои закладки», «что я добавил в закладки», «дайджест закладок», «подведите итоги по моим закладкам», «x закладок», «закладки Twitter», (2) пользователь хочет периодически получать дайджесты сохраненных твитов, (3) пользователь хочет классифицировать, искать или анализировать свои закладки, (4) запланированные дайджесты закладок через cron. Аутентификация: Bird CLI с файлами cookie браузера, OR X API v2 с токенами OAuth 2.0.
