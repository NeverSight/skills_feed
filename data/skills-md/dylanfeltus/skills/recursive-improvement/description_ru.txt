Шаблон для создания выходных данных более высокого качества путем итерации по явным критериям оценки. Используйте для заголовков, призывов к действию, текста целевой страницы, социального контента, текста рекламы — всего, где качество имеет значение. Создать → Оценить → Диагностировать → Улучшить → Повторить.
