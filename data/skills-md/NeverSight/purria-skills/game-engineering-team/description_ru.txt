Инженерный совет ААА-класса по созданию игр серийного качества. Используйте при реализации игровых систем, написании игрового кода, проектировании архитектуры данных, создании компонентов пользовательского интерфейса, создании учебных пособий, оптимизации производительности или любых технических задачах разработки игр. Охватывает шаблоны программирования игр, реализацию казино/карточных игр, системы вознаграждений, разработку игрового пользовательского интерфейса, дизайн учебных пособий, архитектуру кода, инфраструктуру данных, безопасность и контроль качества. Запускается по запросам на игровой код, реализацию системы, рефакторинг, оптимизацию производительности, дизайн данных или решения по технической архитектуре.
