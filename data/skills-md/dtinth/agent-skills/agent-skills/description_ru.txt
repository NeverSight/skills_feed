Используйте это, когда: (1) Создание нового навыка, специфичного для проекта, для документирования шаблонов проекта, (2) Улучшение или обновление существующего навыка проекта, (3) Написание описаний навыков, которые будут активировать других агентов, (4) Организация контента навыков для постепенного раскрытия или (5) Изучение того, какие навыки, специфичные для проекта, существуют в .claude/skills/. Гарантирует, что навыки соответствуют соглашениям проекта и обеспечивают эффективное постепенное раскрытие информации.
