Ищет и фильтрует трассировки MLflow с помощью CLI или Python API. Используйте, когда пользователь запрашивает поиск трассировок, фильтрацию трассировок по статусу/тегам/метаданным/времени выполнения, трассировки запросов или неудачные трассировки отладки. Триггеры по «поиску трассировок», «найти неудачные трассировки», «фильтровать трассировки по», «следы медленнее», «запросить трассировки MLflow».
