استخدم هذا عندما تحتاج إلى تحسين أو تحسين أداء وكيل LLM الحالي - بما في ذلك تحسين دقة اختيار الأداة، أو جودة الإجابة، أو تقليل التكاليف، أو إصلاح المشكلات التي يقدم فيها الوكيل استجابات خاطئة/غير كاملة. يقوم بتقييم الوكلاء بشكل منهجي باستخدام تقييم MLflow مع مجموعات البيانات، والهدافين، والتتبع. يغطي سير عمل التقييم الشامل أو المكونات الفردية (إعداد التتبع، وإنشاء مجموعة البيانات، وتعريف الهداف، وتنفيذ التقييم).
