قم بتشغيل أوامر Firefox mach حاول باستخدام إشارات تم تكوينها مسبقًا لاختبار تكامل نظام التشغيل على مجموعات العاملين ألفا لنظامي التشغيل Windows وLinux. يُستخدم عند اختبار تغييرات Firefox على أنظمة التشغيل Windows 10 أو Windows 11 أو Ubuntu 24.04 أو الأجهزة العاملة أو تكوينات ARM64 أو AMD. مشغلات على "تكامل نظام التشغيل"، "محاولة Mach"، "اختبار Windows"، "اختبار Linux"، "صورة ألفا".
