Führen Sie Firefox-Mach-Try-Befehle mit vorkonfigurierten Flags für Betriebssystemintegrationstests in Windows- und Linux-Alpha-Worker-Pools aus. Verwenden Sie diese Option, wenn Sie Firefox-Änderungen anhand von Windows 10-, Windows 11-, Ubuntu 24.04-, Hardware-Worker-, ARM64- oder AMD-Konfigurationen testen. Löst auf „OS-Integration“, „Mach Try“, „Windows Testing“, „Linux Testing“ und „Alpha Image“ aus.
