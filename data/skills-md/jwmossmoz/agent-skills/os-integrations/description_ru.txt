Запускайте команды Firefox mach try с предварительно настроенными флагами для тестирования интеграции с ОС в альфа-пулах рабочих групп Windows и Linux. Используйте при тестировании изменений Firefox на конфигурациях Windows 10, Windows 11, Ubuntu 24.04, аппаратных средств, ARM64 или AMD. Триггеры по «интеграции ОС», «попытке машины», «тестированию Windows», «тестированию Linux», «альфа-образу».
