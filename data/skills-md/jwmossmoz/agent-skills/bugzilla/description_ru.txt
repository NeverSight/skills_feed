Взаимодействуйте с Mozilla Bugzilla (bugzilla.mozilla.org) через REST API. Используйте, когда пользователь запрашивает поиск ошибок, просмотр сведений об ошибках, создание ошибок, обновление ошибок, добавление комментариев или прикрепление файлов. Запускается по «bugzilla», «bmo», «сообщить об ошибке», «отчет об ошибке», «ошибка Mozilla».
