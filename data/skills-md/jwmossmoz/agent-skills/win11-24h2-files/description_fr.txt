Interrogez les informations du fichier de mise à jour cumulative de Windows 11 version 24H2. Recherchez des DLL, des pilotes et des fichiers système dans tous les niveaux de correctifs. Comparez les versions de fichiers entre les versions, suivez l'historique des versions et identifiez ce qui a changé dans chaque mise à jour. À utiliser lorsque : (1) rechercher quelle version de Windows a introduit une version de fichier, (2) comparer les modifications de fichiers entre les correctifs, (3) rechercher les versions de DLL/pilote pour des versions spécifiques, (4) déboguer les régressions de mise à jour Windows, (5) l'utilisateur pose des questions sur les fichiers Windows 11 24H2, les mises à jour de la base de connaissances ou les versions de build.
