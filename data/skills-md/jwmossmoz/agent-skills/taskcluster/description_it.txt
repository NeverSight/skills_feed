Interagisci con l'elemento CI di Mozilla Taskcluster utilizzando la CLI di taskcluster. Esegui query sullo stato delle attività, visualizza i registri, scarica gli artefatti, riattiva le attività e gestisci i gruppi di attività. Da utilizzare quando si lavora con attività CI da firefox-ci-tc.services.mozilla.com o si esegue il debug di problemi del pool di lavoro. Per le operazioni del pool di lavoratori (elenco dei lavoratori, terminazione dei lavoratori, annullamento di attività in blocco), utilizzare i comandi API nativi di taskcluster (vedere riferimenti/worker-pools.md). Trigger su "taskcluster", "task status", "task log", "artifacts", "retrigger", "task group", "worker pool".
