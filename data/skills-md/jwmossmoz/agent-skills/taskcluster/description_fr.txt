Interagissez avec Mozilla Taskcluster CI à l’aide de la CLI taskcluster. Interrogez l’état des tâches, affichez les journaux, téléchargez des artefacts, redéclenchez les tâches et gérez les groupes de tâches. À utiliser lorsque vous travaillez avec des tâches CI à partir de firefox-ci-tc.services.mozilla.com ou lorsque vous déboguez des problèmes de pool de travailleurs. Pour les opérations du pool de travailleurs (lister les travailleurs, terminer les travailleurs, annulation groupée de tâches), utilisez les commandes API natives du taskcluster (voir references/worker-pools.md). Déclencheurs sur « groupe de tâches », « état de la tâche », « journal des tâches », « artefacts », « redéclenchement », « groupe de tâches », « pool de travailleurs ».
