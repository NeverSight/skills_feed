Взаимодействуйте с Mozilla Taskcluster CI с помощью интерфейса командной строки Taskcluster. Запрашивайте состояние задач, просматривайте журналы, загружайте артефакты, перезапускайте задачи и управляйте группами задач. Используйте при работе с задачами CI из firefox-ci-tc.services.mozilla.com или при отладке проблем с рабочим пулом. Для операций с пулом рабочих (список рабочих, завершение рабочих процессов, массовая отмена задач) используйте собственные команды API Taskcluster (см. ссылки/worker-pools.md). Триггеры по «кластеру задач», «статусу задачи», «журналу задач», «артефактам», «повторному триггеру», «группе задач», «пулу рабочих».
