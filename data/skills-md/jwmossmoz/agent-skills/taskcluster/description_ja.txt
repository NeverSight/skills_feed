taskcluster CLI を使用して Mozilla Taskcluster CI と対話します。タスクのステータスのクエリ、ログの表示、アーティファクトのダウンロード、タスクの再トリガー、およびタスク グループの管理を行います。 firefox-ci-tc.services.mozilla.com からの CI タスクを操作する場合、またはワーカー プールの問題をデバッグする場合に使用します。ワーカー プールの操作 (ワーカーのリスト、ワーカーの終了、タスクの一括キャンセル) には、ネイティブの taskcluster API コマンドを使用します (references/worker-pools.md を参照)。 「タスククラスター」、「タスクステータス」、「タスクログ」、「アーティファクト」、「再トリガー」、「タスクグループ」、「ワーカープール」でトリガーします。
