Interactúe con Mozilla Taskcluster CI mediante la CLI de taskcluster. Consulte el estado de las tareas, vea registros, descargue artefactos, vuelva a activar tareas y administre grupos de tareas. Úselo cuando trabaje con tareas de CI desde firefox-ci-tc.services.mozilla.com o depure problemas del grupo de trabajadores. Para las operaciones del grupo de trabajadores (enumerar trabajadores, despedir trabajadores, cancelar tareas en masa), utilice los comandos nativos de la API del cluster de tareas (consulte references/worker-pools.md). Activadores en "taskcluster", "estado de tarea", "registro de tareas", "artefactos", "reactivación", "grupo de tareas", "grupo de trabajadores".
