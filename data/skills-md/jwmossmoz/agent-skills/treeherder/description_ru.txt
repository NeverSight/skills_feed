Запросите Firefox Treeherder для получения результатов заданий CI, используя Treeherder-cli (основной) и Lumberjackth (вторичный). Используйте после фиксации земли, чтобы проверить результаты тестирования/сборки. Триггеры по «деревопастуху», «результатам работы», «проверочным тестам», «статусу ci».
