Запросите Redash Mozilla (sql.telemetry.mozilla.org) для получения данных телеметрии из BigQuery. Используйте при запросе телеметрии пользователя Firefox, распределении ОС, разбивке архитектуры или запуске пользовательского SQL в хранилище данных Mozilla.
