Исследуйте сбои задач Taskcluster, связанные с рабочими образами. Сравнивайте выполненные и невыполненные задачи, извлекайте версии образов, находите работающие рабочие процессы и отлаживайте виртуальные машины Windows с помощью Azure CLI. Используйте при отладке сбоев CI после обновлений образа, расследовании периодических сбоев тестов или сравнении рабочих конфигураций. Триггеры по «исследованию изображения», «образу работника», «сравнению задач», «почему это не удается», «регрессии изображения», «отладке работника».
