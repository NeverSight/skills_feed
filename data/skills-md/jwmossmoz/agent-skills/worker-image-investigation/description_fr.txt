Examinez les échecs de tâches Taskcluster liés aux images de travailleur. Comparez les tâches qui réussissent et celles qui échouent, extrayez les versions d'image, recherchez les nœuds de calcul en cours d'exécution et déboguez les machines virtuelles Windows via Azure CLI. À utiliser lors du débogage des échecs de CI après des mises à niveau d'images, de l'enquête sur les échecs de tests intermittents ou de la comparaison des configurations de travail. Déclencheurs sur "l'enquête sur l'image", "l'image du travailleur", "comparer les tâches", "pourquoi cet échec", "la régression de l'image", "le débogage du travailleur".
