Untersuchen Sie Taskcluster-Aufgabenfehler im Zusammenhang mit Worker-Images. Vergleichen Sie bestandene und fehlgeschlagene Aufgaben, extrahieren Sie Image-Versionen, finden Sie laufende Worker und debuggen Sie Windows-VMs über Azure CLI. Verwenden Sie diese Option zum Debuggen von CI-Fehlern nach Image-Upgrades, zur Untersuchung zeitweiliger Testfehler oder zum Vergleichen von Worker-Konfigurationen. Löst auf „Bilduntersuchung“, „Arbeiterbild“, „Aufgaben vergleichen“, „Warum schlägt das fehl“, „Bildregression“ und „Arbeiter-Debug“ aus.
