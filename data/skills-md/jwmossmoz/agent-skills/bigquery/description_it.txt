Esegui query sui dati di telemetria di Mozilla direttamente da BigQuery utilizzando la CLI bq. Da utilizzare quando l'utente desidera eseguire SQL rispetto alla telemetria di Firefox, analizzare la distribuzione delle versioni di Windows, contare DAU/MAU/WAU, eseguire query sulle metriche di Glean o esaminare le popolazioni di utenti.
