bq CLI を使用して、BigQuery から Mozilla テレメトリ データを直接クエリします。ユーザーが Firefox テレメトリに対して SQL を実行する場合、Windows バージョン分布を分析する場合、DAU/MAU/WAU をカウントする場合、Glean メトリックをクエリする場合、またはユーザー人口を調査する場合に使用します。 「bigquery」、「bq」、「テレメトリ クエリ」、「DAU」、「MAU」、「Windows ディストリビューション」、「macOS ディストリビューション」、「Darwin バージョン」、「Linux ディストリビューション」、「カーネル バージョン」、「クライアント数」、「ユーザー数」、「収集メトリクス クエリ」、「baseline_clients」でトリガーされます。
