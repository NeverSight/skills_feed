mozilla-platform-ops/worker-images で GitHub Actions ワークフローをトリガーして、Firefox CI ワーカー イメージを構築します。
Windows イメージ ビルド用の FXCI Azure ワークフローをサポートします (信頼できるものと信頼できないもの)。

次の場合に使用します。
- ユーザーがワーカー イメージを構築したい
- ユーザーが「FXCI Azure」、「ワーカー イメージ ビルド」、または「win11-64-24h2-alpha」などの特定のプール名に言及している
- ユーザーは Windows 10、Windows 11、または Windows Server 2022 のイメージ ビルドをトリガーしたい
- ユーザーがワーカー イメージのビルドのステータスを確認するように要求する

トリガー: 「ビルドイメージ」、「ワーカーイメージ」、「FXCI Azure」、「トリガービルド」、「イメージビルドステータス」
