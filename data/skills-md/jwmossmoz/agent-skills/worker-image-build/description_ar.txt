أنشئ صورًا عاملة لـ Firefox CI عن طريق تشغيل سير عمل GitHub Actions في mozilla-platform-ops/worker-images.
يدعم سير عمل FXCI Azure لإنشاءات صور Windows (الموثوقة وغير الموثوقة).

استخدم عندما:
- يريد المستخدم بناء صورة العامل
- يذكر المستخدم "FXCI Azure"، أو "إنشاء صورة العامل"، أو أسماء مجموعات محددة مثل "win11-64-24h2-alpha"
- يريد المستخدم تشغيل إنشاءات الصور لنظام التشغيل Windows 10 أو Windows 11 أو Windows Server 2022
- يطلب المستخدم التحقق من حالة بناء صورة العامل

المشغلات: "صورة البناء"، "صورة العامل"، "FXCI Azure"، "إنشاء المشغل"، "حالة إنشاء الصورة"
