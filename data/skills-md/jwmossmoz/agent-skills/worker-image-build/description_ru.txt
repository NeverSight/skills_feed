Создавайте рабочие образы Firefox CI, запуская рабочие процессы GitHub Actions в mozilla-platform-ops/worker-images.
Поддерживает рабочие процессы FXCI Azure для сборок образов Windows (доверенных и ненадежных).

Используйте, когда:
- Пользователь хочет создать рабочий образ.
– Пользователь упоминает «FXCI Azure», «сборку рабочего образа» или конкретные имена пулов, например «win11-64-24h2-alpha».
- Пользователь хочет запустить сборки образа для Windows 10, Windows 11 или Windows Server 2022.
- Пользователь просит проверить статус сборки рабочего образа.

Триггеры: «образ сборки», «рабочий образ», «FXCI Azure», «сборка триггера», «статус сборки образа».
