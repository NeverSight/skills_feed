mozilla-platform-ops/worker-images에서 GitHub Actions 워크플로를 트리거하여 Firefox CI 작업자 이미지를 빌드하세요.
Windows 이미지 빌드(신뢰할 수 있는 것과 신뢰할 수 없는 것)를 위한 FXCI Azure 워크플로를 지원합니다.

다음과 같은 경우에 사용하세요.
- 작업자 이미지를 구축하려는 사용자
- 사용자가 "FXCI Azure", "작업자 이미지 빌드" 또는 "win11-64-24h2-alpha"와 같은 특정 풀 이름을 언급합니다.
- 사용자가 Windows 10, Windows 11 또는 Windows Server 2022용 이미지 빌드를 트리거하려고 합니다.
- 사용자가 작업자 이미지 빌드 상태를 확인하도록 요청합니다.

트리거: "빌드 이미지", "작업자 이미지", "FXCI Azure", "트리거 빌드", "이미지 빌드 상태"
