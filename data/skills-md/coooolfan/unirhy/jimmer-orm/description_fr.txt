Assistant de développement Jimmer ORM, aidant à écrire le code de mappage d'entités et les requêtes DSL. Applicable à : (1) définir ou modifier des entités Jimmer (interfaces annotées avec @Entity) ; (2) écrire le code de requête Jimmer DSL ; (3) configuration des associations d'entités (@ManyToOne, @OneToMany, @ManyToMany) ; (4) en utilisant des prédicats dynamiques, des jointures de tables dynamiques ou des sous-requêtes implicites ; (5) édition de fichiers Java/Kotlin contenant du code lié à Jimmer.
