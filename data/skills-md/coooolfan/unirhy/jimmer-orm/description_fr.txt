Assistant au développement Jimmer ORM : aide à écrire le mapping d’entités et les requêtes DSL. Pour : (1) définir ou modifier des entités Jimmer (interface avec @Entity) ; (2) écrire du code de requête Jimmer DSL ; (3) configurer les associations (@ManyToOne, @OneToMany, @ManyToMany) ; (4) prédicats dynamiques, jointures dynamiques ou sous-requêtes implicites ; (5) éditer des fichiers Java/Kotlin contenant du code Jimmer.
