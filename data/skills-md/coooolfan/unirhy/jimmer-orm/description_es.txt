Asistente para Jimmer ORM: entidades y consultas DSL. Usar para (1) definir o modificar entidades Jimmer (interface con @Entity), (2) escribir consultas DSL, (3) configurar asociaciones (@ManyToOne, @OneToMany, @ManyToMany), (4) predicados dinámicos, joins dinámicos o subconsultas implícitas, (5) editar archivos Java/Kotlin con código Jimmer.
