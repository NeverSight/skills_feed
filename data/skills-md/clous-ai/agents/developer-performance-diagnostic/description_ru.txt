Этот навык следует использовать, когда пользователь просит «диагностировать проблемы с производительностью», «проанализировать производительность команды», «исследовать низкую производительность», «выявить закономерности производительности» или «оценить здоровье команды». Анализирует данные о производительности для выявления закономерностей, системных проблем и индивидуальных проблем с помощью рекомендаций на основе данных.
