Erstellen Sie Backend-Logik mit Shopify-Funktionen. Verwenden Sie diese Fähigkeit zum Erstellen benutzerdefinierter Rabatte, zur Anpassung von Lieferungen, zur Zahlungsanpassung, zur Warenkorb- und Checkout-Validierung sowie zur Bestellweiterleitung. Funktionen werden mithilfe von WebAssembly auf der Shopify-Infrastruktur ausgeführt. Unterstützt Rust und JavaScript.
