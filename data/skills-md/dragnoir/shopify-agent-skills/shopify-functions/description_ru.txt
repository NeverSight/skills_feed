Создавайте внутреннюю логику с помощью Shopify Functions. Используйте этот навык для создания пользовательских скидок, настройки доставки, настройки оплаты, проверки корзины и оформления заказа, а также маршрутизации заказов. Функции выполняются в инфраструктуре Shopify с использованием WebAssembly. Поддерживает Rust и JavaScript.
