ينشئ صفحات وثائق للأدلة والبرامج التعليمية ومقالات قاعدة المعرفة ومحتوى استكشاف الأخطاء وإصلاحها في المستندات/. يُستخدم عند إضافة أدلة إرشادية، أو كتابة مقالات قاعدة المعارف، أو إنشاء مستندات لاستكشاف الأخطاء وإصلاحها، أو إضافة صفحات للبدء/، أو التطبيقات/، أو قواعد البيانات/، أو قاعدة المعرفة/، أو عمليات التكامل/. ليس لصفحات الخدمة - استخدم وثائق إضافة الخدمة للمستندات/الخدمات/.
