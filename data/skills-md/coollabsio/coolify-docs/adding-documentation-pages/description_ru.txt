Создает страницы документации для руководств, учебных пособий, статей базы знаний и содержимого по устранению неполадок в docs/. Используйте при добавлении практических руководств, написании статей базы знаний, создании документации по устранению неполадок, добавлении страниц для начала работы/, приложений/, баз данных/, базы знаний/ или интеграций/. НЕ для страниц сервисов — используйте документацию по добавлению сервисов для docs/services/.
