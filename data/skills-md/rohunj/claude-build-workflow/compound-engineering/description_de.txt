Compound Engineering-Workflow für KI-gestützte Entwicklung. Verwenden Sie es, wenn Sie Funktionen planen, Arbeiten ausführen, Code überprüfen oder Erkenntnisse kodifizieren. Folgt der Schleife „Plan → Arbeit → Überprüfung → Verbund“, wobei jede technische Einheit die nachfolgende Arbeit erleichtert. Auslöser für: diese Funktion planen, implementieren, diesen Code überprüfen, Erkenntnisse zusammenfassen, Implementierungsplan erstellen, systematische Entwicklung.
