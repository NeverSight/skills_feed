Wird zum Übersetzen von UX-Spezifikationen in Build-Reihenfolge-Eingabeaufforderungen für UI-Generierungstools verwendet. Wird ausgelöst, wenn der Benutzer über eine UX-Spezifikation, ein PRD oder ein detailliertes Feature-Dokument verfügt und sequentielle, eigenständige Eingabeaufforderungen für Tools wie v0, Bolt oder Claude Frontend-Design benötigt.
