TDD 가드 후크에 의해 시행되는 엄격한 TDD(테스트 기반 개발) 방법론을 사용하여 NuxtUI로 Vue 3 구성 요소를 구축하기 위한 가이드입니다. Poche 프로젝트를 위한 새로운 UI 구성 요소(원자, 분자, 유기체)를 구현하거나, 상호 작용 테스트를 통해 스토리북 스토리를 만들거나, RED-GREEN-REFACTOR 주기 내에서 작업할 때 이 기술을 사용하세요. 사용자가 "TDD", "테스트 우선", "컴포넌트 생성", "컴포넌트 빌드", "[ComponentName] 구현"을 언급하거나 UI 기능을 추가할 때 특히 유용합니다.
