Guía para crear componentes de Vue 3 con NuxtUI utilizando una estricta metodología de desarrollo basado en pruebas (TDD) aplicada mediante un gancho de protección TDD. Utilice esta habilidad al implementar nuevos componentes de la interfaz de usuario (átomos, moléculas, organismos) para el proyecto Poche, crear historias de libros de cuentos con pruebas de interacción o trabajar dentro del ciclo ROJO-VERDE-REFACTOR. Particularmente útil cuando el usuario menciona "TDD", "probar primero", "crear un componente", "construir un componente", "implementar [Nombre del componente]" o al agregar funcionalidad de interfaz de usuario.
