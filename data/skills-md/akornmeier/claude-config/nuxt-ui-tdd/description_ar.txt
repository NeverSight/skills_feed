دليل لبناء مكونات Vue 3 باستخدام NuxtUI باستخدام منهجية صارمة للتطوير القائم على الاختبار (TDD) يتم فرضها بواسطة خطاف حماية TDD. استخدم هذه المهارة عند تنفيذ مكونات واجهة المستخدم الجديدة (الذرات والجزيئات والكائنات الحية) لمشروع Poche، أو إنشاء قصص القصص القصيرة مع اختبارات التفاعل، أو العمل ضمن دورة RED-GREEN-REFACTOR. مفيد بشكل خاص عندما يذكر المستخدم "TDD"، أو "الاختبار أولاً"، أو "إنشاء مكون"، أو "إنشاء مكون"، أو "تنفيذ [ComponentName]"، أو عند إضافة وظيفة واجهة المستخدم.
