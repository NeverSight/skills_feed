Leitfaden zur Verwendung von Docker – einer Containerisierungsplattform zum Erstellen, Ausführen und Bereitstellen von Anwendungen in isolierten Containern. Verwendung beim Containerisieren von Anwendungen, Erstellen von Docker-Dateien, Arbeiten mit Docker Compose, Verwalten von Bildern/Containern, Konfigurieren von Netzwerk und Speicher, Optimieren von Builds, Bereitstellen in der Produktion oder Implementieren von CI/CD-Pipelines mit Docker.
