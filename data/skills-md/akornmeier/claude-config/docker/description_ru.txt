Руководство по использованию Docker — платформы контейнеризации для создания, запуска и развертывания приложений в изолированных контейнерах. Используйте при контейнеризации приложений, создании файлов Dockerfile, работе с Docker Compose, управлении образами/контейнерами, настройке сети и хранилища, оптимизации сборок, развертывании в рабочей среде или реализации конвейеров CI/CD с помощью Docker.
