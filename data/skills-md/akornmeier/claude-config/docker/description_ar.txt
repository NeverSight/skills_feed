دليل استخدام Docker - منصة حاويات لبناء التطبيقات وتشغيلها ونشرها في حاويات معزولة. يُستخدم عند تخزين التطبيقات في حاويات، أو إنشاء ملفات Dockerfiles، أو العمل مع Docker Compose، أو إدارة الصور/الحاويات، أو تكوين الشبكات والتخزين، أو تحسين البنيات، أو النشر إلى الإنتاج، أو تنفيذ خطوط أنابيب CI/CD باستخدام Docker.
