Подробное руководство по PostgreSQL psql — интерактивный терминальный клиент для PostgreSQL. Используйте при подключении к базам данных PostgreSQL, выполнении запросов, управлении базами данных/таблицами, настройке параметров подключения, форматировании вывода, написании сценариев, управлении транзакциями и использовании расширенных функций psql для администрирования и разработки баз данных.
