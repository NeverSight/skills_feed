Руководство по внедрению Cloudflare Browser Rendering — API-интерфейса для автоматизации браузера без управления для создания снимков экрана, PDF-файлов, очистки веб-страниц и тестирования. Используйте при автоматизации браузеров, создании снимков экрана, создании PDF-файлов, очистке динамического контента, извлечении структурированных данных или тестировании веб-приложений. Поддерживает REST API, рабочие привязки (кукловод/драматург), серверы MCP и автоматизацию на базе искусственного интеллекта. (проект)
