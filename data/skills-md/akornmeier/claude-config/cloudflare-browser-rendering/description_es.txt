Guía para implementar Cloudflare Browser Rendering: una API de automatización de navegador sin cabeza para capturas de pantalla, archivos PDF, web scraping y pruebas. Úselo para automatizar navegadores, tomar capturas de pantalla, generar archivos PDF, extraer contenido dinámico, extraer datos estructurados o probar aplicaciones web. Admite API REST, Workers Bindings (titiritero/dramaturgo), servidores MCP y automatización impulsada por IA. (proyecto)
