دليل لتنفيذ Cloudflare Browser Rendering - واجهة برمجة التطبيقات لأتمتة المتصفح بدون رأس للحصول على لقطات الشاشة وملفات PDF وتجريد الويب والاختبار. يُستخدم عند أتمتة المتصفحات، أو التقاط لقطات الشاشة، أو إنشاء ملفات PDF، أو استخراج المحتوى الديناميكي، أو استخراج البيانات المنظمة، أو اختبار تطبيقات الويب. يدعم REST API، وربطات العمال (محرك الدمى/الكاتب المسرحي)، وخوادم MCP، والأتمتة التي تعمل بالذكاء الاصطناعي.
