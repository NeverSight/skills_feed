Guida per l'implementazione del rendering del browser Cloudflare: un'API di automazione del browser headless per screenshot, PDF, web scraping e test. Da utilizzare per automatizzare i browser, acquisire screenshot, generare PDF, raschiare contenuti dinamici, estrarre dati strutturati o testare applicazioni web. Supporta API REST, associazioni di lavoratori (burattinaio/drammaturgo), server MCP e automazione basata sull'intelligenza artificiale. (progetto)
