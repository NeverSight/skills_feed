Guide d'implémentation de Cloudflare Browser Rendering - une API d'automatisation de navigateur sans tête pour les captures d'écran, les PDF, le web scraping et les tests. À utiliser pour automatiser les navigateurs, prendre des captures d'écran, générer des PDF, récupérer du contenu dynamique, extraire des données structurées ou tester des applications Web. Prend en charge l'API REST, les liaisons de travailleurs (marionnettiste/dramaturge), les serveurs MCP et l'automatisation basée sur l'IA. (projet)
