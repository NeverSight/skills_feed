Руководство Effect-TS (Effect) для TypeScript: проектирование и реализация кода на основе Effect, моделирование ожидаемых ошибок и дефектов, управление зависимостями с помощью Context/Layer/Effect.Service, обработка жизненных циклов ресурсов с помощью Scope, запуск эффектов на границе программы, использование Effect.gen, проверка данных с помощью Effect Schema и время тестирования с помощью TestClock. Используйте, когда вас просят создать, реорганизовать, просмотреть или объяснить код эффекта или при преобразовании кода Promise/async в эффект.
