Conseils Effect-TS (Effect) pour TypeScript : conception et implémentation de code basé sur les effets, modélisation des erreurs attendues par rapport aux défauts, gestion des dépendances avec Context/Layer/Effect.Service, gestion des cycles de vie des ressources avec Scope, exécution des effets en périphérie du programme, utilisation d'Effet.gen, validation des données avec Effect Schema et test du temps avec TestClock. À utiliser lorsqu'on vous demande de créer, refactoriser, réviser ou expliquer le code d'effet ou lors de la conversion du code Promise/async en effet.
