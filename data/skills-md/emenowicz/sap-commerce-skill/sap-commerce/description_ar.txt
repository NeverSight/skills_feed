يوفر إرشادات تطوير شاملة لـ SAP Commerce Cloud (Hybris سابقًا) بما في ذلك نمذجة نظام النوع، وهندسة طبقة الخدمة، وإدارة البيانات باستخدام ImpEx وFlexibleSearch، وتخصيص OCC API، وأنماط تسريع B2C/B2B، وCronJobs، والعمليات التجارية، وبحث Solr، والترقيات، والتخزين المؤقت، وتكوين Backoffice. يُستخدم عندما يطلب المستخدم "إنشاء ملحقات SAP Commerce"، و"تحديد أنواع العناصر في items.xml"، و"كتابة نصوص ImpEx النصية"، و"تنفيذ مكونات طبقة الخدمة (الواجهات/الخدمات/DAOs)"، و"تخصيص واجهات برمجة تطبيقات OCC REST"، و"العمل مع استعلامات البحث المرنة"، و"تخصيص مسرعات B2C أو B2B"، و"تكوين حبوب الربيع"، و"إنشاء CronJobs أو المهام المجدولة"، و"تحديد العمليات التجارية أو تدفقات الطلبات"، و"التكوين". بحث Solr أو فهرسته"، أو "إعداد العروض الترويجية أو القسائم"، أو "تكوين التخزين المؤقت"، أو "تخصيص Backoffice"، أو يذكر "تطوير Hybris" أو "منصة SAP Commerce Cloud"، أو يسأل عن استكشاف مشكلات SAP Commerce وإصلاحها.
