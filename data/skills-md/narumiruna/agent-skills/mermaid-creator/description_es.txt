Cree diagramas de sirena para documentación técnica, diseño de sistemas y modelado de datos. Úselo al crear diagramas de flujo, diagramas de secuencia, diagramas de clases, diagramas de estado, diagramas ER, diagramas de Gantt, gráficos git o cualquier otro tipo de diagrama compatible con Mermaid. Admite tanto la creación de nuevos diagramas a partir de descripciones como la conversión de diagramas a formato SVG para incrustarlos en presentaciones o documentos.
