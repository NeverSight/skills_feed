Prisma CLI-Befehlsreferenz, die alle verfügbaren Befehle, Optionen und Verwendungsmuster abdeckt. Wird zum Ausführen von Prisma-CLI-Befehlen, zum Einrichten von Projekten, zum Generieren von Clients, zum Ausführen von Migrationen oder zum Verwalten von Datenbanken verwendet. Löst auf „Prisma Init“, „Prisma Generate“, „Prisma Migrate“, „Prisma DB“, „Prisma Studio“ aus.
