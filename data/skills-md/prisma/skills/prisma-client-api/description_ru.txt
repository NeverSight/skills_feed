Справочник по клиентскому API Prisma, охватывающий запросы моделей, фильтры, операторы и методы клиента. Используйте при написании запросов к базе данных, использовании операций CRUD, фильтрации данных или настройке клиента Prisma. Запускается по «запросу призмы», «findMany», «созданию», «обновлению», «удалению», «$transaction».
