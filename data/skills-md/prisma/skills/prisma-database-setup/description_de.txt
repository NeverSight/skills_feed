Anleitungen zur Konfiguration von Prisma mit verschiedenen Datenbankanbietern (PostgreSQL, MySQL, SQLite, MongoDB usw.). Verwenden Sie diese Option, wenn Sie ein neues Projekt einrichten, Datenbanken ändern oder Verbindungsprobleme beheben. Löst auf „Postgres konfigurieren“, „Mit MySQL verbinden“, „Mongodb einrichten“, „SQLite-Setup“ aus.
