Руководство по настройке и работе Prisma Postgres в консоли, интерфейсе командной строки create-db, Management API и Management API SDK. Используйте при создании баз данных Prisma Postgres, работе в Prisma Console, подготовке с помощью create-db/create-pg/create-postgres или интеграции программной подготовки с токенами служб или OAuth.
