Video- und Audioverarbeitung mit FFmpeg. Zur Formatkonvertierung, Größenänderung, Komprimierung, Audioextraktion und Vorbereitung von Assets für Remotion. Zu den Auslösern gehören das Konvertieren von GIF in MP4, das Ändern der Videogröße, das Extrahieren von Audio, das Komprimieren von Dateien oder jede beliebige Medientransformationsaufgabe.
