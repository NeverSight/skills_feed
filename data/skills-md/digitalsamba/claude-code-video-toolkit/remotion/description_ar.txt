قم بإنشاء مقاطع فيديو برمجية باستخدام React with Remotion. يُستخدم عند إنشاء تركيبات فيديو أو رسوم متحركة أو عرض مقاطع فيديو برمجيًا. تتضمن المشغلات إنشاء محتوى فيديو باستخدام React، أو تحريك العناصر إطارًا بإطار، أو عرض مقاطع الفيديو إلى MP4/WebM/GIF، أو استخدام الرسوم المتحركة الزنبركية/الاستكمالية، أو إنشاء تطبيقات تحرير الفيديو، أو نشر عرض الفيديو إلى AWS Lambda.
