Verwenden Sie diese Fähigkeit, wenn der Benutzer Erinnerungen speichern, merken, abrufen oder organisieren möchte. Wird ausgelöst durch: „Merken Sie sich das“, „Speichern Sie das“, „Notieren Sie sich das“, „Worüber haben wir gesprochen?“, „Überprüfen Sie Ihre Notizen“, „Räumen Sie Erinnerungen auf“. Auch bei der Entdeckung wertvoller, erhaltenswerter Erkenntnisse proaktiv einsetzen.
