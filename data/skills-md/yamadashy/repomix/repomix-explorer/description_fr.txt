Utilisez cette compétence lorsque l'utilisateur souhaite analyser ou explorer une base de code (référentiel distant ou référentiel local) à l'aide de Repomix. Déclencheurs sur : "analyser ce dépôt", "explorer la base de code", "quelle est la structure", "trouver des modèles dans le dépôt", "combien de fichiers/jetons". Exécute repomix CLI pour compresser les référentiels, puis analyse le résultat.
