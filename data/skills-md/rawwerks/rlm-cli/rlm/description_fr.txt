CLI des modèles de langage récursifs (RLM) - permet aux LLM de traiter de manière récursive de grands contextes en décomposant les entrées et en s'appelant eux-mêmes sur les parties. À utiliser pour l'analyse de code, les révisions de différences, l'exploration de la base de code. Déclencheurs sur "rlm request", "rlm complete", "rlm search", "rlm index".
