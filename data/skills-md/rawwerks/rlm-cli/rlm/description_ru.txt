CLI рекурсивных языковых моделей (RLM) — позволяет LLM рекурсивно обрабатывать большие контексты путем разложения входных данных и вызова самих себя по частям. Используйте для анализа кода, проверки различий, исследования кодовой базы. Триггеры: «запрос rlm», «rlm завершен», «поиск rlm», «индекс rlm».
