Tailwind CSS、Font Awesome、Google Fonts を使用して、HTML スライド プレゼンテーション (1 スライド = 1 HTML ファイル、1280x720 ピクセル) を生成します。ユーザーが新しいプレゼンテーションデッキまたはスライド HTML ファイルを作成するように要求する場合に使用します。デザインガイドライン、20種類のレイアウトパターン、コンポーネントライブラリ、PPTX変換互換性ルールを網羅。スタイルの選択 (クリエイティブ、エレガント、モダン、プロフェッショナル、ミニマリスト) とテーマの選択 (マーケティング、ポートフォリオ、ビジネス、テクノロジー、教育) をサポートします。
