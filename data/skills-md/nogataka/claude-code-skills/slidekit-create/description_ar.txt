قم بإنشاء عروض تقديمية لشرائح HTML (شريحة واحدة = ملف HTML واحد، 1280 × 720 بكسل) باستخدام Tailwind CSS وFont Awesome وGoogle Fonts. يُستخدم عندما يطلب المستخدم إنشاء مجموعة عرض تقديمي جديدة أو ملفات HTML للشرائح. يغطي إرشادات التصميم و20 نمط تخطيط ومكتبة المكونات وقواعد توافق تحويل PPTX. يدعم اختيار الأسلوب (الإبداعي والأنيق والحديث والمهني والبسيط) واختيار الموضوع (التسويق والمحفظة والأعمال التجارية والتكنولوجيا والتعليم).
