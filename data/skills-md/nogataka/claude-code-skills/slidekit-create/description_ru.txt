Создавайте слайд-презентации в формате HTML (1 слайд = 1 HTML-файл, 1280x720 пикселей) с помощью Tailwind CSS, Font Awesome и Google Fonts. Используйте, когда пользователь просит создать новую презентацию или HTML-файлы слайдов. Охватывает рекомендации по проектированию, 20 шаблонов компоновки, библиотеку компонентов и правила совместимости преобразования PPTX. Поддерживает выбор стиля (креативный, элегантный, современный, профессиональный, минималистский) и выбор темы (маркетинг, портфолио, бизнес, технологии, образование).
