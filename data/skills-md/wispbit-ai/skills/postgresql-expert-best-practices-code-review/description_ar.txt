أفضل ممارسات تصميم قاعدة بيانات PostgreSQL وترحيلها وتحسين الأداء. يجب استخدام هذه المهارة عند كتابة أو مراجعة أو إعادة بناء مخططات قاعدة البيانات أو عمليات الترحيل أو أنماط الاستعلام. يتم تفعيلها عند تنفيذ المهام التي تتضمن قواعد بيانات PostgreSQL، أو تصميم المخطط، أو تحسين الترحيل، أو نمذجة البيانات.
