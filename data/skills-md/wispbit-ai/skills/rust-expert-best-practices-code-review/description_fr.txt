Meilleures pratiques Rust et directives de qualité du code pour écrire du code Rust idiomatique, sûr et performant. Cette compétence doit être utilisée lors de l’écriture, de la révision ou de la refactorisation du code Rust. Déclencheurs sur des tâches impliquant la programmation Rust, la révision du code, la gestion des erreurs, la sécurité des types ou l'optimisation des performances.
