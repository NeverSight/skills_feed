Le migliori pratiche di Rust e le linee guida sulla qualità del codice per scrivere codice Rust idiomatico, sicuro e performante. Questa abilità dovrebbe essere utilizzata durante la scrittura, la revisione o il refactoring del codice Rust. Trigger su attività che coinvolgono la programmazione Rust, la revisione del codice, la gestione degli errori, l'indipendenza dai tipi o l'ottimizzazione delle prestazioni.
