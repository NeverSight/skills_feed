Рекомендации по миграции SQLAlchemy ORM и Alembic для создания безопасных и производительных схем баз данных. Этот навык следует использовать при написании, проверке или рефакторинге моделей SQLAlchemy, миграции Alembic или шаблонов запросов к базе данных. Запускается при выполнении задач, связанных с SQLAlchemy ORM, миграцией Alembic, изменением схемы базы данных или оптимизацией запросов.
