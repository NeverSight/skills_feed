Meilleures pratiques Python pour écrire du code de qualité production. Cette compétence doit être utilisée lors de l’écriture, de la révision ou de la refactorisation du code Python. Déclencheurs sur des tâches impliquant le développement Python, les modèles de gestion des erreurs, les opérations de dictionnaire et l'amélioration de la qualité du code.
