Synchronisez les fichiers .env du référentiel racine git avec les arbres de travail. À utiliser lorsqu'on vous demande de synchroniser l'environnement, de copier l'environnement, le fichier d'environnement ou lorsque vous travaillez dans un arbre de travail git pour lequel il manque un fichier .env. Détecte automatiquement les fichiers .env manquants dans les arbres de travail.
