Синхронизируйте файлы .env из корневого репозитория git с рабочими деревьями. Используйте, когда вас просят синхронизировать env, скопировать env, файл среды или при работе с рабочим деревом git, в котором отсутствует файл .env. Автоматически обнаруживает отсутствующий файл .env в рабочих деревьях.
