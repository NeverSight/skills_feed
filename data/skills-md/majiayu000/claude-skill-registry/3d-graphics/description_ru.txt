3D веб-графика с помощью Three.js (WebGL/WebGPU). Возможности: сцены, камеры, геометрия, материалы, освещение, анимация, загрузка модели (GLTF/FBX), материалы PBR, тени, постобработка (цветение, SSAO, SSR), пользовательские шейдеры, создание экземпляров, LOD, физика, VR/XR. Действия: создавать, строить, анимировать, рендерить 3D-сцены/модели. Ключевые слова: Three.js, WebGL, WebGPU, 3D-графика, сцена, камера, геометрия, материал, свет, анимация, GLTF, FBX, OrbitControls, PBR, отображение теней, постобработка, цветение, SSAO, шейдер, создание экземпляров, LOD, WebXR, VR, AR, конфигуратор продукта, визуализация данных, архитектурное пошаговое руководство, интерактивное 3D, холст. Используйте при: создании 3D-визуализации, создании приложений WebGL/WebGPU, загрузке 3D-моделей, добавлении анимации, реализации VR/XR, создании интерактивной графики, создании конфигураторов продуктов.
