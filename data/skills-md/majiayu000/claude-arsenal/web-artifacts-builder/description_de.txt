Suite von Tools zum Erstellen aufwändiger, mehrkomponentiger claude.ai-HTML-Artefakte unter Verwendung moderner Frontend-Webtechnologien (React, Tailwind CSS, shadcn/ui). Wird für komplexe Artefakte verwendet, die Statusverwaltung, Routing oder Shadcn/UI-Komponenten erfordern – nicht für einfache HTML/JSX-Artefakte mit einer einzelnen Datei.
