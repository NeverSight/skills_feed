Bewertet Code anhand der App Store-Überprüfungsrichtlinien von Apple. Nutzen Sie diese Fähigkeit, wenn Sie iOS-, macOS-, tvOS-, watchOS- oder visionOS-App-Code (Swift, Objective-C, React Native oder Expo) überprüfen, um potenzielle Ablehnungsprobleme im App Store vor der Einreichung zu identifizieren. Wird bei Aufgaben ausgelöst, die die Vorbereitung von App-Reviews, Compliance-Prüfungen oder die Einreichungsbereitschaft im App Store umfassen.
