Évalue le code par rapport aux directives d'examen de l'App Store d'Apple. Utilisez cette compétence lors de l'examen du code d'une application iOS, macOS, tvOS, watchOS ou visionOS (Swift, Objective-C, React Native ou Expo) pour identifier les problèmes potentiels de rejet de l'App Store avant la soumission. Se déclenche sur des tâches impliquant la préparation de l'examen de l'application, la vérification de la conformité ou la préparation à la soumission sur l'App Store.
