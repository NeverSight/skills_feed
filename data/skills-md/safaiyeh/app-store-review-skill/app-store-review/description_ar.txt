يقيم الكود وفقًا لإرشادات مراجعة متجر تطبيقات Apple. استخدم هذه المهارة عند مراجعة رمز تطبيق iOS أو macOS أو tvOS أو watchOS أو VisionOS (Swift أو Objective-C أو React Native أو Expo) لتحديد مشكلات الرفض المحتملة في App Store قبل الإرسال. تشغيل المهام التي تتضمن إعداد مراجعة التطبيق، أو التحقق من الامتثال، أو جاهزية الإرسال في متجر التطبيقات.
