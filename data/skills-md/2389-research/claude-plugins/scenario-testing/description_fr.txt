Déclencheurs sur « tests d'écriture », « ajouter une couverture de test », « valider la fonctionnalité », « test d'intégration », « de bout en bout », « test e2e », « simulation », « test unitaire ». Applique des tests basés sur des scénarios avec des dépendances réelles dans le répertoire .scratch/.
