Diese Fähigkeit sollte beim Erstellen von iOS-Apps mit xtool (Xcode-freie iOS-Entwicklung), beim Erstellen von xtool-Projekten, beim Hinzufügen von App-Erweiterungen oder beim Konfigurieren von xtool.yml verwendet werden. Löst auf „xtool“, „SwiftPM iOS“, „iOS unter Linux“, „iOS unter Windows“, „Xcode-free“, „App-Erweiterung“, „Widget-Erweiterung“, „Share-Erweiterung“ aus. Behandelt Projekteinrichtung, App-Erweiterungen und Bereitstellung.
