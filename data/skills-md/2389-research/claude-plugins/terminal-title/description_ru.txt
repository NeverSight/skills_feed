Этот навык следует использовать для обновления заголовка окна терминала с учетом контекста. Запускается автоматически при запуске сеанса через перехватчик. Также срабатывает при изменении темы во время разговора (отладка до документов, интерфейс до серверной части). Обновляет заголовок с помощью смайликов + проекта + текущей темы.
