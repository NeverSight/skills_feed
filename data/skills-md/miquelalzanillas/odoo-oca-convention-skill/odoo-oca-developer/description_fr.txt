Conseils d'experts pour le développement, la migration et la maintenance des modules Odoo (versions 14.0-19.0) conformément aux conventions OCA. À utiliser lorsque vous travaillez avec des modules Odoo pour créer de nouveaux modules à partir du modèle OCA, migrer les modules entre les versions à l'aide d'OpenUpgrade, étendre les modules Odoo de base, garantir la conformité OCA et structurer correctement les fichiers des modules. Comprend les normes de codage OCA, les modèles de structure de module, les modèles de migration OpenUpgrade et les outils de validation.
