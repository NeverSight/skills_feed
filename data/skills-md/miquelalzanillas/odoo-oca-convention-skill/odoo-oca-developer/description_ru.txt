Экспертное руководство по разработке, миграции и поддержке модулей Odoo (версии 14.0–19.0) в соответствии с соглашениями OCA. Используйте при работе с модулями Odoo для создания новых модулей из шаблона OCA, переноса модулей между версиями с помощью OpenUpgrade, расширения основных модулей Odoo, обеспечения соответствия OCA и правильного структурирования файлов модулей. Включает стандарты кодирования OCA, шаблоны структуры модулей, шаблоны миграции OpenUpgrade и инструменты проверки.
