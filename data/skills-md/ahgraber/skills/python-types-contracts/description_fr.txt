À utiliser lors de la définition ou de l'évolution d'interfaces publiques, de limites de schéma ou d'utilisation pydantique en Python. À utiliser également lorsque les annotations manquent sur les API publiques, que les modèles pydantiques apparaissent partout plutôt qu'aux limites de confiance, que les modifications de contrat manquent de conseils de migration ou que les types Any/object sont surutilisés au-delà des limites des modules.
