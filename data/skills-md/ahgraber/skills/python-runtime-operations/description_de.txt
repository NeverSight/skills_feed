Wird zum Erstellen oder Überprüfen des Laufzeitverhaltens von Diensten, Jobs oder CLIs in Python verwendet – zum Entwerfen von Startvalidierungen, Abschaltsequenzen, Beobachtbarkeit und strukturierter Protokollierung. Wird auch verwendet, wenn der Start aufgrund einer späten Konfiguration abstürzt, beim Herunterfahren verwaiste Prozesse zurückbleiben, Terminalstatus implizit sind oder Protokolle keine Struktur aufweisen.
