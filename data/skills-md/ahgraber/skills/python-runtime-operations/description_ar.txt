يُستخدم عند إنشاء أو مراجعة سلوك وقت تشغيل الخدمة أو الوظيفة أو واجهة سطر الأوامر (CLI) في Python - تصميم التحقق من صحة بدء التشغيل، وتسلسلات إيقاف التشغيل، وإمكانية المراقبة، والتسجيل المنظم. يُستخدم أيضًا عند تعطل بدء التشغيل بسبب التكوين المتأخر، أو ترك إيقاف التشغيل عمليات معزولة، أو عندما تكون الحالات الطرفية ضمنية، أو تفتقر السجلات إلى البنية.
