Verwenden Sie es beim Erstellen oder Überprüfen externer API-Integrationen in Python – beim Entwerfen von Clientgrenzen, beim Definieren von Outbound-Zuverlässigkeitsrichtlinien oder beim Strukturieren von Vertragstests.
