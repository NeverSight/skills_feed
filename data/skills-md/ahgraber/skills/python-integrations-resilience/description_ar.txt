يُستخدم عند إنشاء أو مراجعة عمليات تكامل واجهة برمجة التطبيقات الخارجية في Python - تصميم حدود العميل، أو تحديد سياسة الموثوقية الصادرة، أو هيكلة اختبارات العقود. استخدمه أيضًا عندما تتسرب تفاصيل SDK الخاصة بالموفر إلى منطق المجال، أو تفتقر المكالمات الصادرة إلى سياسة المهلة/إعادة المحاولة، أو عندما لا يتم اختبار مسارات الفشل.
