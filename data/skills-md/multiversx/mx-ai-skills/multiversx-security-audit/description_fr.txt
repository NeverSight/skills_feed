Méthodologie complète d'audit de sécurité pour les contrats intelligents MultiversX. Couvre la création de contexte, l'analyse du point d'entrée, les modèles d'analyse statique et l'analyse Semgrep automatisée. À utiliser lors de la réalisation d'audits de sécurité, de révisions de code ou de configuration d'une détection automatisée des vulnérabilités.
