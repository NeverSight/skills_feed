Комплексный набор инструментов для анализа кода для смарт-контрактов MultiversX. Охватывает дифференциальный анализ (сравнение версий, безопасность обновления), проверку исправлений (проверка исправлений, регрессионное тестирование) и анализ вариантов (нахождение похожих ошибок в базе кода). Используйте при просмотре запросов на запросы, проверке обновлений безопасности или поиске вариантов ошибок.
