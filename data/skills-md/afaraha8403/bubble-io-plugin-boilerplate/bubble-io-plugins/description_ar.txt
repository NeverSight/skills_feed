قواعد تطوير البرنامج المساعد Bubble.io ومرجع API ومعايير الترميز. يُستخدم عند العمل على أي مهمة في هذا الريبو: الكتابة أو المراجعة أو إعادة البناء أو إنشاء initize.js، أو update.js، أو Preview.js، أو header.html، أو إجراءات العناصر، أو الإجراءات من جانب العميل، أو الإجراءات من جانب الخادم (SSA)، أو كود Plugin API v4 غير المتزامن/الانتظار، أو JSDoc، أو ملفات الإعداد، أو README، أو CHANGELOG، أو أوصاف السوق، أو تلميحات الأدوات الميدانية. يُستخدم أيضًا في عمليات تدقيق الأمان ومراجعة التعليمات البرمجية وتصحيح الأخطاء ونشر المكونات الإضافية. يغطي المثيلات/الخصائص/كائنات السياق، وواجهات BubbleThing/BubbleList، وتعليق تحميل البيانات، وقواعد DOM/canvas، والرؤوس المشتركة للعنصر مقابل الرؤوس المشتركة، والحالات المكشوفة، ومعالجة الأحداث، ومعايير ESLint، وحدود الفقاعات الصلبة.
