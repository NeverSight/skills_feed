Правила разработки плагинов Bubble.io, справочник API и стандарты кодирования. Используйте при работе над любой задачей в этом репозитории: написание, проверка, рефакторинг или создание инициализации.js, update.js, предварительного просмотра.js, header.html, действий с элементами, действий на стороне клиента, действий на стороне сервера (SSA), асинхронного/ожидающего кода Plugin API v4, JSDoc, файлов настройки, README, CHANGELOG, описаний торговой площадки или всплывающих подсказок полей. Также используйте для аудита безопасности, проверки кода, отладки и публикации плагинов. Охватывает объекты экземпляра/свойств/контекста, интерфейсы BubbleThing/BubbleList, приостановку загрузки данных, правила DOM/canvas, элементы и общие заголовки, открытые состояния, обработку событий, стандарты ESLint и жесткие ограничения Bubble.
