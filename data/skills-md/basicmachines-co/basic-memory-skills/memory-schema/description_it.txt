Gestione del ciclo di vita dello schema per la memoria di base: scopri note non schematizzate, deduci schemi, crea e modifica definizioni di schema, convalida note e rileva deviazioni. Da utilizzare quando si lavora con tipi di note strutturate (Attività, Persona, Riunione, ecc.) per mantenere la coerenza nel grafico della conoscenza.
