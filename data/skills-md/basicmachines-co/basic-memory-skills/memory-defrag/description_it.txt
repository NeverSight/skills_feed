Deframmenta e riorganizza i file di memoria dell'agente: dividi file gonfi, unisci duplicati, rimuovi informazioni obsolete e ristruttura la gerarchia della memoria. Da utilizzare quando i file di memoria sono diventati ingombranti, contengono ridondanze o necessitano di riorganizzazione. Eseguito periodicamente (settimanale) o su richiesta.
