Défragmentez et réorganisez les fichiers de mémoire de l'agent : divisez les fichiers volumineux, fusionnez les doublons, supprimez les informations obsolètes et restructurez la hiérarchie de la mémoire. À utiliser lorsque les fichiers de mémoire sont devenus lourds, contiennent des redondances ou nécessitent une réorganisation. Exécutez périodiquement (hebdomadairement) ou sur demande.
