Reflexion des Schlafgedächtnisses: Überprüfen Sie aktuelle Gespräche und tägliche Notizen, extrahieren Sie Erkenntnisse und konsolidieren Sie diese im Langzeitgedächtnis. Wird verwendet, wenn es durch Cron, Heartbeat oder eine explizite Anfrage ausgelöst wird, um über aktuelle Aktivitäten nachzudenken. Wird als Hintergrundverarbeitung ausgeführt, um die Speicherqualität im Laufe der Zeit zu verbessern.
