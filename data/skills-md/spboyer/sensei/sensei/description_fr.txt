**COMPÉTENCE DE FLUX DE TRAVAIL** - Améliorez de manière itérative la conformité des compétences à l'aide du modèle de boucle Ralph. UTILISER POUR : exécuter sensei, aide sensei, améliorer les compétences, corriger le frontmatter, la conformité des compétences, l'audit du frontmatter, améliorer les déclencheurs, ajouter des anti-déclencheurs, améliorer les compétences par lots, vérifier les jetons de compétence, marquer les compétences. NE PAS UTILISER POUR : la création de nouvelles compétences à partir de zéro (utilisez le créateur de compétences), l'écriture du contenu/du corps des compétences, l'édition générale de démarques ou des fichiers non SKILL.md. INVOKES : outils de comptage de jetons, exécuteurs de tests, commandes git. POUR LES OPÉRATIONS SIMPLES : utilisez le token CLI directement pour les décomptes/vérifications.
