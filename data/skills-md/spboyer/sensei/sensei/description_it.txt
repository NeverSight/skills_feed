** COMPETENZA DEL FLUSSO DI LAVORO ** - Migliora in modo iterativo la conformità alle questioni fondamentali delle competenze utilizzando il modello di loop Ralph. UTILIZZARE PER: eseguire sensei, aiutare sensei, migliorare abilità, correggere frontmatter, conformità abilità, controllo frontmatter, migliorare trigger, aggiungere anti-trigger, miglioramento batch di abilità, controllare token abilità, punteggio abilità. NON UTILIZZARE PER: creazione di nuove competenze da zero (utilizzare skill-creator), scrittura del contenuto/corpo delle competenze, modifica generale del markdown o file non SKILL.md. INVOCA: strumenti di conteggio dei token, test runner, comandi git. PER OPERAZIONI SINGOLE: utilizzare direttamente la CLI del token per conteggi/controlli.
