IntelliJ-IDEA MCP proporciona potentes funciones IDE que incluyen ejecución de pruebas, análisis de código, refactorización, búsqueda y navegación de proyectos. Úselo cuando necesite un análisis preciso del código Java (evitando falsos positivos de LSP), ejecutar pruebas a través de configuraciones IDEA, refactorizar símbolos o explorar la estructura base del código. Comandos clave: ejecutar_ejecutar_configuración (ejecutar pruebas), get_file_problems (errores/advertencias precisos), search_in_files_by_text (código de búsqueda), list_directory_tree (ver estructura), get_file_text_by_path (leer archivos), rename_refactoring (refactorización segura), ejecutar_terminal_command (ejecutar comandos de shell).
