IntelliJ-IDEA MCP предоставляет мощные функции IDE, включая запуск тестов, анализ кода, рефакторинг, поиск и навигацию по проекту. Используйте это, когда вам нужен точный анализ кода Java (избегая ложных срабатываний LSP), запуск тестов с помощью конфигураций IDEA, рефакторинг символов или исследование структуры базы кода. Ключевые команды: Execute_run_configuration (запуск тестов), get_file_problems (точные ошибки/предупреждения), search_in_files_by_text (код поиска), list_directory_tree (структура просмотра), get_file_text_by_path (чтение файлов), rename_refactoring (безопасный рефакторинг), Execute_terminal_command (запуск команд оболочки).
