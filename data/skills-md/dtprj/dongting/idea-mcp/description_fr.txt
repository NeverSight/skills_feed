IntelliJ-IDEA MCP fournit de puissantes fonctionnalités IDE, notamment l'exécution de tests, l'analyse de code, la refactorisation, la recherche et la navigation dans le projet. Utilisez-le lorsque vous avez besoin d'une analyse précise du code Java (en évitant les faux positifs LSP), d'exécuter des tests via des configurations IDEA, de refactoriser des symboles ou d'explorer la structure de la base de code. Commandes clés : exécuter_run_configuration (exécuter des tests), get_file_problems (erreurs/avertissements précis), search_in_files_by_text (code de recherche), list_directory_tree (afficher la structure), get_file_text_by_path (lire les fichiers), rename_refactoring (refactoring sécurisé), execute_terminal_command (exécuter des commandes shell).
