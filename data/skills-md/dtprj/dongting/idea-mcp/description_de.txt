IntelliJ-IDEA MCP bietet leistungsstarke IDE-Funktionen, darunter das Ausführen von Tests, Code-Analyse, Refactoring, Suche und Projektnavigation. Verwenden Sie dies, wenn Sie eine genaue Java-Code-Analyse (Vermeidung falsch positiver LSP-Ergebnisse), die Durchführung von Tests über IDEA-Konfigurationen, die Umgestaltung von Symbolen oder die Untersuchung der Codebasisstruktur benötigen. Schlüsselbefehle: „execute_run_configuration“ (Tests ausführen), „get_file_problems“ (genaue Fehler/Warnungen), „search_in_files_by_text“ (Suchcode), „list_directory_tree“ (Struktur anzeigen), „get_file_text_by_path“ (Dateien lesen), „rename_refactoring“ (sicheres Refactoring), „execute_terminal_command“ (Shell-Befehle ausführen).
