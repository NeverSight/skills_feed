IntelliJ-IDEA MCP 提供强大的 IDE 功能，包括运行测试、代码分析、重构、搜索和项目导航。当您需要准确的 Java 代码分析（避免 LSP 误报）、通过 IDEA 配置运行测试、重构符号或探索代码库结构时，请使用此功能。关键命令：execute_run_configuration（运行测试）、get_file_problems（准确的错误/警告）、search_in_files_by_text（搜索代码）、list_directory_tree（查看结构）、get_file_text_by_path（读取文件）、rename_refactoring（安全重构）、execute_terminal_command（运行 shell 命令）。
