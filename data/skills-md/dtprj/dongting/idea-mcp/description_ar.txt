يوفر IntelliJ-IDEA MCP ميزات IDE قوية بما في ذلك تشغيل الاختبارات وتحليل التعليمات البرمجية وإعادة البناء والبحث والتنقل في المشروع. استخدم هذا عندما تحتاج إلى تحليل دقيق لكود Java (تجنب النتائج الإيجابية الخاطئة لـ LSP)، أو إجراء اختبارات عبر تكوينات IDEA، أو إعادة بناء الرموز، أو استكشاف بنية قاعدة التعليمات البرمجية. الأوامر الرئيسية: Execute_run_configuration (اختبارات التشغيل)، get_file_problems (الأخطاء/التحذيرات الدقيقة)، search_in_files_by_text (رمز البحث)، list_directory_tree (بنية العرض)، get_file_text_by_path (قراءة الملفات)، rename_refactoring (إعادة البناء الآمن)، Execute_terminal_command (تشغيل أوامر Shell).
