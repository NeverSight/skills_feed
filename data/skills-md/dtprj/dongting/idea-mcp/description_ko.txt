IntelliJ-IDEA MCP는 테스트 실행, 코드 분석, 리팩토링, 검색 및 프로젝트 탐색을 포함한 강력한 IDE 기능을 제공합니다. 정확한 Java 코드 분석(LSP 거짓 긍정 방지)이 필요할 때, IDEA 구성을 통해 테스트 실행, 기호 리팩터링 또는 코드베이스 구조 탐색이 필요할 때 이 기능을 사용하세요. 주요 명령: run_run_configuration(테스트 실행), get_file_problems(정확한 오류/경고), search_in_files_by_text(검색 코드), list_directory_tree(구조 보기), get_file_text_by_path(파일 읽기), rename_refactoring(안전한 리팩토링), run_terminal_command(셸 명령 실행).
