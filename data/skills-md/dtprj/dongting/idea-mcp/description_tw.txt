IntelliJ-IDEA MCP 提供強大的 IDE 功能，包括運行測試、代碼分析、重構、搜索和項目導航。當您需要準確的 Java 代碼分析（避免 LSP 誤報）、通過 IDEA 配置運行測試、重構符號或探索代碼庫結構時，請使用此功能。關鍵命令：execute_run_configuration（運行測試）、get_file_problems（準確的錯誤/警告）、search_in_files_by_text（搜索代碼）、list_directory_tree（查看結構）、get_file_text_by_path（讀取文件）、rename_refactoring（安全重構）、execute_terminal_command（運行 shell 命令）。
