IntelliJ-IDEA MCP は、テストの実行、コード分析、リファクタリング、検索、プロジェクト ナビゲーションなどの強力な IDE 機能を提供します。これは、正確な Java コード分析 (LSP 誤検知の回避)、IDEA 構成によるテストの実行、シンボルのリファクタリング、またはコードベース構造の調査が必要な場合に使用します。主要なコマンド:execute_run_configuration (テストの実行)、get_file_problems (正確なエラー/警告)、search_in_files_by_text (コードの検索)、list_directory_tree (構造の表示)、get_file_text_by_path (ファイルの読み取り)、rename_refactoring (安全なリファクタリング)、execute_terminal_command (シェル コマンドの実行)。
