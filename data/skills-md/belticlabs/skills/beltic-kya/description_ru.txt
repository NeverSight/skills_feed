Руководство по работе с экосистемой Beltic KYA (Знай своего агента) — системой доверия на основе учетных данных для агентов ИИ. Используйте, когда: (1) работаете в любом репозитории Beltic (beltic-spec,beltic-cli,beltic-sdk, fact-python, kya-platform, Wizard, NASA), (2) реализуете подписание/проверку учетных данных агента, (3) используете @belticlabs/kya SDK или Beltic-sdk Python, (4) понимаете сертификацию безопасности агента, (5) работаете с проверяемыми учетными данными для ИИ. Запускается по: командам Beltic CLI, учетным данным агента, подписям HTTP-сообщений (RFC 9421), оценкам безопасности, проверке уровня KYB, проверке цепочки доверия.
