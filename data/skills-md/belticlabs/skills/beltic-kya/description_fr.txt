Conseils pour travailler avec l'écosystème Beltic KYA (Know Your Agent) - un cadre de confiance basé sur les informations d'identification pour les agents IA. À utiliser lorsque : (1) vous travaillez dans n'importe quel référentiel Beltic (beltic-spec, beltic-cli, beltic-sdk, fact-python, kya-platform, assistant, nasa), (2) implémentez la signature/vérification des informations d'identification de l'agent, (3) utilisez le SDK @belticlabs/kya ou beltic-sdk Python, (4) comprenez la certification de sécurité des agents, (5) travaillez avec des informations d'identification vérifiables pour l'IA. Déclencheurs sur : les commandes Beltic CLI, les informations d'identification de l'agent, les signatures de messages HTTP (RFC 9421), les scores de sécurité, la vérification du niveau KYB, la validation de la chaîne de confiance.
