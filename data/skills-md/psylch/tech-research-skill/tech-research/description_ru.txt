Комплексное техническое исследование путем объединения нескольких источников разведки — Grok (обсуждения разработчиков X/Twitter с помощью автоматизации браузера), DeepWiki (анализ репозитория GitHub с помощью искусственного интеллекта) и WebSearch. Отправляет параллельные субагенты для каждого источника и объединяет результаты в единый отчет. Этот навык следует использовать при оценке технологий, сравнении библиотек/фреймворков, исследовании репозиториев GitHub, оценке настроений разработчиков или исследовании технических решений по архитектуре. Триггерные фразы включают «техническое исследование», «изучите эту технологию», «技术调研», «调研一下», «сравнить библиотеки», «оценить структуру», «исследовать репозиторий».
