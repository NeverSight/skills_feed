Настраивает полную среду ИИ-кодирования на базе терминала, основанную на терминале Ghostty, с помощью Fish Shell, yazi, lazygit, Neovim (LazyVim), fzf, zoxy, atuin и вспомогательных инструментов. Включает оптимизацию шрифтов CJK для китайских, японских и корейских пользователей на macOS с Apple Silicon. Этот навык следует использовать при настройке или изменении среды разработки на основе Ghostty, настройке инструментов терминала для рабочих процессов кодирования Vibe или устранении неполадок Ghostty/Fish/yazi/lazygit.
