Konfiguriert eine vollständige terminalbasierte KI-Codierungsumgebung, die sich auf das Ghostty-Terminal mit Fish Shell, Yazi, Lazygit, Neovim (LazyVim), FZF, Zoxide, Atuin und unterstützenden Tools konzentriert. Beinhaltet CJK-Schriftoptimierung für chinesische/japanische/koreanische Benutzer auf macOS mit Apple Silicon. Diese Fähigkeit sollte beim Einrichten oder Ändern einer Ghostty-basierten Entwicklungsumgebung, beim Konfigurieren von Terminal-Tools für Vibe-Coding-Workflows oder bei der Fehlerbehebung bei Ghostty/Fish/Yazi/Lazygit-Problemen verwendet werden.
