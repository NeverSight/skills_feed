Лучшие практики оптимизации токенов для экономичного использования Claude Code. Автоматически применяет эффективные стратегии чтения файлов, выполнения команд и обработки вывода. Включает руководство по выбору модели (Opus для обучения, Sonnet для разработки/отладки). Предпочитает команды bash чтению файлов.
