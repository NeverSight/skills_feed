Best Practices zur Token-Optimierung für eine kostengünstige Nutzung von Claude Code. Wendet automatisch effiziente Strategien zum Lesen von Dateien, zur Befehlsausführung und zur Ausgabeverarbeitung an. Enthält Anleitungen zur Modellauswahl (Opus zum Lernen, Sonnet zum Entwickeln/Debuggen). Bevorzugt Bash-Befehle gegenüber dem Lesen von Dateien.
