ابحث في GitHub عن أمثلة التعليمات البرمجية الواقعية وأنماط التنفيذ. يُستخدم عندما يريد المستخدم العثور على أمثلة للتعليمات البرمجية على GitHub، أو البحث في مستودعات GitHub، أو اكتشاف كيفية قيام الآخرين بتنفيذ الميزات، أو تعلم أنماط استخدام المكتبة، أو البحث في الأساليب المعمارية. جلب أفضل النتائج باستخدام التصنيف الذكي (النجوم، الحداثة، اللغة)، واستخراج البيانات الفعلية (الواردات، وأنماط بناء الجملة، والمقاييس)، وإرجاع تخفيض السعر النظيف للتحليل وتحديد النمط.
