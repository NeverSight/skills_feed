Recherchez sur GitHub des exemples de code réels et des modèles d'implémentation. À utiliser lorsque l'utilisateur souhaite trouver des exemples de code sur GitHub, rechercher des référentiels GitHub, découvrir comment d'autres implémentent des fonctionnalités, apprendre des modèles d'utilisation de bibliothèque ou rechercher des approches architecturales. Récupère les meilleurs résultats avec un classement intelligent (étoiles, récence, langue), extrait des données factuelles (importations, modèles de syntaxe, métriques) et renvoie une démarque propre pour l'analyse et l'identification des modèles.
