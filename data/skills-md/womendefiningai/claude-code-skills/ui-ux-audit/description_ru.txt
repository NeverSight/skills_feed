Обязательный рабочий процесс аудита изменений пользовательского интерфейса/UX, который в первую очередь считывает текущее состояние, проверяет наличие избыточности, соблюдает философию чистого дизайна и выявляет подлинные пробелы перед реализацией. Вызывается автоматически, когда пользователь упоминает пользовательский интерфейс, UX, дизайн, макет, домашнюю страницу, улучшения страницы, визуальные изменения или модификации интерфейса.
