Obligatorischer Audit-Workflow für UI/UX-Änderungen, der ZUERST den aktuellen Status liest, auf Redundanz prüft, die Clean-Design-Philosophie respektiert und echte Lücken vor der Implementierung identifiziert. Wird automatisch aufgerufen, wenn der Benutzer Benutzeroberfläche, UX, Design, Layout, Homepage, Seitenverbesserungen, visuelle Änderungen oder Schnittstellenmodifikationen erwähnt.
