FORGE (إطار عمل للهندسة التوليدية المرنة المنسقة) - إطار تطوير موحد يعتمد على الذكاء الاصطناعي ويجمع بين مسارات عمل رشيقة متعددة الوكلاء، وحلقات التكرار المستقلة، والذاكرة المستمرة، وهندسة Claude Code Skills، وتكامل خدمة MCP، وأتمتة سير العمل n8n. يُستخدم عندما: إنشاء مشاريع برمجية شاملة، أو تخطيط البنية، أو تشغيل حلقات التطوير المستقلة، أو إنشاء خوادم MCP، أو تنسيق سير عمل n8n، أو إعداد خطوط أنابيب CI/CD، أو إدارة فرق التطوير متعددة الوكلاء، أو أي مهمة تطوير منظمة تعتمد على الذكاء الاصطناعي. المشغلات: "تشكيل"، "تطوير مستقل"، "حلقة الوكيل"، "التخطيط السريع"، "متعدد الوكلاء"، "خط أنابيب التطوير"، "مشروع السقالة"، "تشغيل الصياغة"، "الطيار الآلي".
