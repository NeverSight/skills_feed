FORGE (Framework for Orchestrated Resilient Generative Engineering) — Framework di sviluppo unificato basato sull'intelligenza artificiale che combina flussi di lavoro agili multi-agente, cicli di iterazione autonomi, memoria persistente, architettura Claude Code Skills, integrazione del servizio MCP e automazione del flusso di lavoro n8n. Utilizzare quando: creazione di progetti software end-to-end, pianificazione dell'architettura, esecuzione di cicli di sviluppo autonomi, creazione di server MCP, orchestrazione di flussi di lavoro n8n, impostazione di pipeline CI/CD, gestione di team di sviluppo multi-agente o qualsiasi attività di sviluppo strutturata basata sull'intelligenza artificiale.
