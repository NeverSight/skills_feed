## Workflow

1. Analyze the topic provided as argument
2. Identify the 2-3 most relevant agents
3. Create a shared brief for each agent
4. Launch agents in parallel (via Task tool)
5. Collect independent analyses
6. Synthesize into a unified report:
   - Points of consensus
   - Points of divergence with pros/cons
