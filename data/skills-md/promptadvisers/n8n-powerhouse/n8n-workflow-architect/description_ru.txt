Консультант по стратегической архитектуре автоматизации. Используйте его, когда пользователи хотят спланировать решения по автоматизации, оценить свой технологический стек (Shopify, Zoho, HubSpot и т. д.), сделать выбор между n8n и Python/Claude Code или получить рекомендации по готовому к использованию проектированию автоматизации. Вызывает режим планирования для сложных архитектурных решений.
