Co-Autor einer Pantagruel-Spezifikation (.pant) durch geführte Gespräche. Verwenden Sie diese Option, wenn der Benutzer eine Pantagruel-Spezifikationsdatei schreiben, entwerfen oder iterieren möchte. Hilft Benutzern, gründlich über Domänen, Regeln, Invarianten, Aktionen und den Anfangszustand nachzudenken, indem sie klärende Fragen stellen, anstatt zu raten. Wird ausgelöst, wenn der Benutzer erwähnt, dass er eine Spezifikation schreibt, ein System modelliert oder mit .pant-Dateien arbeitet.
