Esamina le modifiche al codice utilizzando la CLI CodeRabbit: supporta i file non salvati (modalità attività) o tutti i file PR rispetto al ramo principale (modalità pr). Rileva bug, problemi di sicurezza e problemi di qualità del codice prima del commit o durante la revisione delle richieste pull. Da utilizzare quando: (1) rivedere le modifiche non confermate prima del commit (modalità attività), (2) rivedere tutti i file modificati in un PR rispetto al ramo principale (modalità pr), (3) lavorare su attività secondarie e si desidera verificare l'avanzamento, (4) è necessario feedback sul codice in corso di lavorazione, (5) preparare il PR per l'unione, (6) quando è necessaria la revisione di CodeRabbit, (7) per il rilevamento di bug e la scansione di sicurezza o (8) per la valutazione automatizzata della qualità del codice. Trigger: revisione del codice, verifica della qualità del codice, revisione delle modifiche, revisione del codice, revisione delle PR, verifica della presenza di bug, scansione di sicurezza, revisione senza commit, finalizzazione del codice, revisione pre-commit.
