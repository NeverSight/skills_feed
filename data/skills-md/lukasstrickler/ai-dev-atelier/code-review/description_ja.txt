CodeRabbit CLI を使用してコード変更を確認します - コミットされていないファイル (タスク モード) またはメイン ブランチと比較してすべての PR ファイル (PR モード) をサポートします。プルリクエストをコミットする前、またはプルリクエストをレビューするときに、バグ、セキュリティ問題、コード品質の問題を捕捉します。次の場合に使用します: (1) コミット前にコミットされていない変更をレビューする (タスク モード)、(2) メイン ブランチに対して PR 内の変更されたすべてのファイルをレビューする (PR モード)、(3) サブタスクに取り組んでおり、進捗状況を確認したい、(4) 進行中のコードに関するフィードバックが必要、(5) PR のマージの準備、(6) CodeRabbit レビューが必要な場合、(7) バグ検出とセキュリティ スキャンの場合、または (8) 自動コード品質評価の場合。トリガー: コードのレビュー、コードの品質のチェック、変更のレビュー、コード レビュー、PR のレビュー、バグのチェック、セキュリティ スキャン、コミットされていないレビュー、コードの完成、コミット前レビュー。
