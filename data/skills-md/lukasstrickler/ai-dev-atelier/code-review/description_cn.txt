使用 CodeRabbit CLI 检查代码更改 - 支持未提交的文件（任务模式）或所有 PR 文件与主分支（pr 模式）。在提交之前或审查拉取请求时捕获错误、安全问题和代码质量问题。在以下情况下使用：(1) 在提交之前审查未提交的更改（任务模式），(2) 针对主分支审查 PR 中所有已更改的文件（pr 模式），(3) 处理子任务并想要检查进度，(4) 需要有关正在进行的代码的反馈，(5) 准备 PR 进行合并，(6) 需要 CodeRabbit 审查时，(7) 用于错误检测和安全扫描，或 (8) 用于自动代码质量评估。触发器：审查代码、检查代码质量、审查更改、代码审查、审查 PR、检查错误、安全扫描、审查未提交、最终确定代码、预提交审查。
