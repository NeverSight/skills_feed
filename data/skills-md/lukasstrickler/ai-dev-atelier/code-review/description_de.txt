Überprüfen Sie Codeänderungen mit der CodeRabbit-CLI – unterstützt nicht festgeschriebene Dateien (Aufgabenmodus) oder alle PR-Dateien im Vergleich zum Hauptzweig (PR-Modus). Erkennt Fehler, Sicherheitsprobleme und Probleme mit der Codequalität vor dem Festschreiben oder beim Überprüfen von Pull-Anfragen. Verwenden Sie dies, wenn: (1) nicht festgeschriebene Änderungen vor dem Festschreiben überprüft werden (Aufgabenmodus), (2) alle geänderten Dateien in einem PR anhand des Hauptzweigs überprüft werden (PR-Modus), (3) an Unteraufgaben gearbeitet wird und der Fortschritt überprüft werden soll, (4) Feedback zum in Arbeit befindlichen Code benötigt wird, (5) PR für die Zusammenführung vorbereitet wird, (6) wenn eine CodeRabbit-Überprüfung erforderlich ist, (7) zur Fehlererkennung und Sicherheitsüberprüfung oder (8) zur automatisierten Codequalitätsbewertung.
