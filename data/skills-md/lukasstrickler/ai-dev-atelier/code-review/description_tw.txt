使用 CodeRabbit CLI 檢查代碼更改 - 支持未提交的文件（任務模式）或所有 PR 文件與主分支（pr 模式）。在提交之前或審查拉取請求時捕獲錯誤、安全問題和代碼質量問題。在以下情況下使用：(1) 在提交之前審查未提交的更改（任務模式），(2) 針對主分支審查 PR 中所有已更改的文件（pr 模式），(3) 處理子任務並想要檢查進度，(4) 需要有關正在進行的代碼的反饋，(5) 準備 PR 進行合併，(6) 需要 CodeRabbit 審查時，(7) 用於錯誤檢測和安全掃描，或 (8) 用於自動代碼質量評估。觸發器：審查代碼、檢查代碼質量、審查更改、代碼審查、審查 PR、檢查錯誤、安全掃描、審查未提交、最終確定代碼、預提交審查。
