Revise los cambios de código usando CodeRabbit CLI: admite archivos no confirmados (modo tarea) o todos los archivos PR frente a la rama principal (modo PR). Detecta errores, problemas de seguridad y problemas de calidad del código antes de realizar la confirmación o al revisar las solicitudes de extracción. Úselo cuando: (1) Revisar cambios no confirmados antes de confirmar (modo tarea), (2) Revisar todos los archivos modificados en un PR en comparación con la rama principal (modo PR), (3) Trabajar en subtareas y desea verificar el progreso, (4) Necesita comentarios sobre el código de trabajo en progreso, (5) Preparar PR para la fusión, (6) Cuando se necesita la revisión de CodeRabbit, (7) Para detección de errores y escaneo de seguridad, o (8) Para evaluación automatizada de la calidad del código. Activadores: revisar código, verificar la calidad del código, revisar cambios, revisar código, revisar relaciones públicas, verificar errores, escaneo de seguridad, revisar no confirmados, finalizar código, revisión previa a la confirmación.
