CodeRabbit CLI를 사용하여 코드 변경 사항 검토 - 커밋되지 않은 파일(작업 모드) 또는 모든 PR 파일 대 기본 분기(pr 모드)를 지원합니다. 풀 요청을 커밋하거나 검토하기 전에 버그, 보안 문제 및 코드 품질 문제를 포착합니다. 다음의 경우에 사용하십시오: (1) 커밋하기 전에 커밋되지 않은 변경 사항 검토(작업 모드), (2) 메인 브랜치에 대해 PR에서 변경된 모든 파일 검토(pr 모드), (3) 하위 작업을 수행하고 진행 상황을 확인하려는 경우, (4) 진행 중인 코드에 대한 피드백이 필요한 경우, (5) 병합을 위한 PR 준비, (6) CodeRabbit 검토가 필요한 경우, (7) 버그 감지 및 보안 검사의 경우 또는 (8) 자동화된 코드 품질 평가의 경우. 트리거: 코드 검토, 코드 품질 확인, 변경 사항 검토, 코드 검토, PR 검토, 버그 확인, 보안 검색, 커밋되지 않은 검토, 코드 마무리, 커밋 전 검토.
