Examinez les modifications de code à l'aide de CodeRabbit CLI - prend en charge les fichiers non validés (mode tâche) ou tous les fichiers PR par rapport à la branche principale (mode pr). Détecte les bogues, les problèmes de sécurité et les problèmes de qualité du code avant de s'engager ou lors de l'examen des demandes d'extraction. À utiliser lorsque : (1) Examiner les modifications non validées avant de les valider (mode tâche), (2) Examiner tous les fichiers modifiés dans un PR par rapport à la branche principale (mode pr), (3) Travailler sur des sous-tâches et vouloir vérifier la progression, (4) Besoin de commentaires sur le code du travail en cours, (5) Préparation du PR pour la fusion, (6) Lorsqu'une révision de CodeRabbit est nécessaire, (7) Pour la détection de bogues et l'analyse de sécurité, ou (8) Pour l'évaluation automatisée de la qualité du code. Déclencheurs : révision du code, vérification de la qualité du code, révision des modifications, révision du code, révision des relations publiques, vérification des bogues, analyse de sécurité, révision non validée, finalisation du code, révision préalable à la validation.
