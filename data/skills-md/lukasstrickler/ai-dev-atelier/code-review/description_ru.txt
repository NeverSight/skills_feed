Просмотр изменений кода с помощью CodeRabbit CLI — поддерживает незафиксированные файлы (режим задач) или все файлы PR по сравнению с основной веткой (режим pr). Выявляет ошибки, проблемы безопасности и проблемы с качеством кода перед фиксацией или при проверке запросов на включение. Используйте, когда: (1) просмотр незафиксированных изменений перед фиксацией (режим задач), (2) проверка всех измененных файлов в PR по сравнению с основной веткой (режим PR), (3) работа над подзадачами и желание проверить прогресс, (4) необходимость обратной связи по незавершенному коду, (5) подготовка PR к слиянию, (6) когда требуется проверка CodeRabbit, (7) для обнаружения ошибок и сканирования безопасности или (8) для автоматической оценки качества кода. Триггеры: проверка кода, проверка качества кода, проверка изменений, проверка кода, проверка PR, проверка на наличие ошибок, сканирование безопасности, проверка незавершенности, завершение кода, проверка перед фиксацией.
