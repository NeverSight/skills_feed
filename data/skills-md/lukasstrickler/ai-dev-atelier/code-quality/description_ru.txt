Выполняйте комплексные проверки качества кода, включая проверку типов TypeScript, анализ ESLint, форматирование Prettier и проверку Markdown. Автоматически устраняет проблемы форматирования в режиме агента или обеспечивает проверки конвейеров CI только для чтения. Используйте, когда: (1) перед фиксацией изменений кода, (2) в конвейерах CI/CD для автоматизированных шлюзов качества, (3) после внесения значительных изменений в код, (4) при подготовке кода к проверке, (5) при обеспечении соответствия кода стандартам качества, (6) для проверки типов, анализа, форматирования и проверки уценки, (7) в перехватчиках перед фиксацией или (8) для автоматических шлюзов качества перед слиянием. Триггеры: финализация, качество кода, проверка типа, проверка, форматирование, проверка кода, проверка качества, проверка запуска, предварительная фиксация, перед фиксацией, проверки CI, проверка кода.
