git push/gh pr create 대신 Graphite CLI(gt)를 사용하여 누적된 PR을 관리하세요. Graphite 저장소를 자동으로 감지하고 유용한 대안을 통해 충돌하는 명령을 차단합니다. 사용 시기: (1) Graphite 저장소에서 git push 또는 gh pr create를 실행하려는 경우, (2) 기능에 대한 새 분기 만들기, (3) 검토를 위해 코드 제출, (4) 검토 가능한 청크로 분할해야 하는 대규모 변경 사항, (5) Hook이 git 명령을 차단하고 그에 상응하는 gt 명령을 제안합니다. 해당되지 않음: 저장소가 Graphite로 초기화되지 않음, git add/commit/status/log. 트리거: git push 차단됨, gh pr 생성 차단됨, 분기 생성, PR 제출, PR 누적, 대규모 PR 분할, gt 생성, gt 제출, 그래파이트 워크플로.
