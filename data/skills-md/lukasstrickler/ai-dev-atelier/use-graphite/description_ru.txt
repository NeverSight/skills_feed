Управляйте составными PR с помощью Graphite CLI (gt) вместо git push/gh pr create. Автоматически обнаруживает репозитории Graphite и блокирует конфликтующие команды с помощью полезных альтернатив. Используйте, когда: (1) собираетесь запустить git push или gh pr create в репозитории Graphite, (2) создаете новую ветку для функции, (3) отправляете код на проверку, (4) большие изменения, которые следует разделить на проверяемые фрагменты, (5) Hook блокирует вашу команду git и предлагает эквивалент gt. НЕ для: репозитории, не инициализированные с помощью Graphite, git add/commit/status/log. Триггеры: git pushblock, gh pr, создать заблокированный, создать ветку, отправить PR, сложенные PR, разделить большой PR, gt create, gt submit, графитовый рабочий процесс.
