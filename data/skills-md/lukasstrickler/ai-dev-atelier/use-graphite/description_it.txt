Gestisci PR in pila con Graphite CLI (gt) invece di git push/gh pr create. Rileva automaticamente i repository Graphite e blocca i comandi in conflitto con alternative utili. Da utilizzare quando: (1) si sta per eseguire git push o gh pr create in un repository Graphite, (2) si crea un nuovo ramo per una funzionalit√†, (3) si invia codice per la revisione, (4) modifiche di grandi dimensioni che dovrebbero essere suddivise in blocchi rivedibili, (5) Hook blocca il comando git e suggerisce l'equivalente gt. NON per: repository non inizializzati con Graphite, git add/commit/status/log. Trigger: git push bloccato, gh pr create bloccato, crea ramo, invia PR, PR impilati, dividi PR di grandi dimensioni, gt crea, gt invia, flusso di lavoro in grafite.
