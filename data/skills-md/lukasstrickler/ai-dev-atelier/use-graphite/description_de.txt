Verwalten Sie gestapelte PRs mit Graphite CLI (gt) anstelle von git push/gh pr create. Erkennt Graphite-Repos automatisch und blockiert widersprüchliche Befehle mit hilfreichen Alternativen. Verwenden Sie diese Option, wenn: (1) git push oder gh pr create in einem Graphite-Repository ausgeführt werden soll, (2) ein neuer Zweig für eine Funktion erstellt wird, (3) Code zur Überprüfung übermittelt wird, (4) große Änderungen vorgenommen werden, die in überprüfbare Blöcke aufgeteilt werden sollten, (5) Hook Ihren Git-Befehl blockiert und ein GT-Äquivalent vorschlägt. NICHT für: Repos, die nicht mit Graphite initialisiert wurden, git add/commit/status/log. Auslöser: git push blockiert, gh pr create blockiert, Zweig erstellen, PR senden, gestapelte PRs, große PR teilen, gt erstellen, gt senden, Graphite-Workflow.
