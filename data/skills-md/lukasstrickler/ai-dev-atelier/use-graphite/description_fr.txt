Gérez les PR empilés avec Graphite CLI (gt) au lieu de git push/gh pr create. Détecte automatiquement les dépôts Graphite et bloque les commandes conflictuelles avec des alternatives utiles. À utiliser lorsque : (1) sur le point d'exécuter git push ou gh pr create dans un dépôt Graphite, (2) création d'une nouvelle branche pour une fonctionnalité, (3) soumission de code pour révision, (4) modifications importantes qui doivent être divisées en morceaux révisables, (5) Hook bloque votre commande git et suggère un équivalent gt. PAS pour : les dépôts non initialisés avec Graphite, git add/commit/status/log. Déclencheurs : git push bloqué, gh pr create bloqué, création de branche, soumission de PR, PR empilés, division de grands PR, gt create, gt submit, workflow graphite.
