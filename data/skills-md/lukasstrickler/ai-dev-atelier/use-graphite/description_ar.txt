يمكنك إدارة العلاقات العامة المكدسة باستخدام Graphite CLI (gt) بدلاً من إنشاء git Push/gh pr. يكتشف تلقائيًا مستودعات الجرافيت ويحظر الأوامر المتعارضة باستخدام بدائل مفيدة. يُستخدم عندما: (1) على وشك تشغيل git Push أو gh pr create في Graphite repo، (2) إنشاء فرع جديد لميزة ما، (3) إرسال التعليمات البرمجية للمراجعة، (4) تغييرات كبيرة يجب تقسيمها إلى أجزاء قابلة للمراجعة، (5) يقوم Hook بحظر أمر git الخاص بك ويقترح مكافئ gt. ليس من أجل: إعادة الشراء التي لم تتم تهيئتها باستخدام Graphite، git add/commit/status/log. المشغلات: تم حظر دفع git، إنشاء gh pr، إنشاء فرع، إرسال علاقات عامة، علاقات عامة مكدسة، تقسيم العلاقات العامة الكبيرة، إنشاء gt، إرسال gt، سير عمل الجرافيت.
