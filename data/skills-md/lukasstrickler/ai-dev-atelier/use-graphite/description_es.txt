Administre relaciones públicas apiladas con Graphite CLI (gt) en lugar de git push/gh pr create. Detecta automáticamente repositorios de Graphite y bloquea comandos conflictivos con alternativas útiles. Úselo cuando: (1) esté a punto de ejecutar git push o gh pr create en un repositorio de Graphite, (2) cree una nueva rama para una función, (3) envíe código para revisión, (4) cambios grandes que deben dividirse en partes revisables, (5) Hook bloquea su comando git y sugiere un equivalente de gt. NO para: repositorios no inicializados con Graphite, git add/commit/status/log. Activadores: git push bloqueado, gh pr create bloqueado, crear rama, enviar PR, PR apilados, dividir PR grandes, gt create, gt submit, flujo de trabajo de grafito.
