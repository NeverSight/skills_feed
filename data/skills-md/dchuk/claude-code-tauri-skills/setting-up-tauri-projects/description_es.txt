Ayuda a los usuarios a crear e inicializar nuevos proyectos Tauri v2 para crear aplicaciones móviles y de escritorio multiplataforma. Cubre los requisitos previos del sistema y los requisitos de configuración para macOS, Windows y Linux. Guía a través de la creación de proyectos mediante create-tauri-app o la inicialización manual de Tauri CLI. Explica la estructura del directorio del proyecto y los archivos de configuración. Admite interfaces basadas en JavaScript, TypeScript, React, Vue, Svelte, Angular, SolidJS y Rust.
