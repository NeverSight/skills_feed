사용자가 크로스 플랫폼 데스크톱 및 모바일 애플리케이션을 구축하기 위한 새로운 Tauri v2 프로젝트를 생성하고 초기화하는 데 도움이 됩니다. macOS, Windows 및 Linux에 대한 시스템 전제 조건 및 설정 요구 사항을 다룹니다. create-tauri-app 또는 수동 Tauri CLI 초기화를 사용하여 프로젝트 생성을 안내합니다. 프로젝트 디렉터리 구조와 구성 파일을 설명합니다. 바닐라 JavaScript, TypeScript, React, Vue, Svelte, Angular, SolidJS 및 Rust 기반 프런트엔드를 지원합니다.
