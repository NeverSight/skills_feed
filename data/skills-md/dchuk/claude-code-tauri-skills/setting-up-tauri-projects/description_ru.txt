Помогает пользователям создавать и инициализировать новые проекты Tauri v2 для создания кроссплатформенных настольных и мобильных приложений. Охватывает предварительные требования к системе и требования к настройке для macOS, Windows и Linux. Помогает создать проект с помощью create-tauri-app или ручной инициализации Tauri CLI. Объясняет структуру каталогов проекта и файлы конфигурации. Поддерживает стандартный JavaScript, TypeScript, React, Vue, Svelte, Angular, SolidJS и интерфейсы на основе Rust.
