يساعد المستخدمين على إنشاء وبدء تشغيل مشاريع Tauri v2 الجديدة لبناء تطبيقات سطح المكتب والهواتف المحمولة عبر الأنظمة الأساسية. يغطي متطلبات النظام الأساسية ومتطلبات الإعداد لأنظمة التشغيل macOS وWindows وLinux. يرشدك خلال عملية إنشاء المشروع باستخدام تطبيق create-tauri أو التهيئة اليدوية لـ Tauri CLI. يشرح بنية دليل المشروع وملفات التكوين. يدعم الفانيليا JavaScript وTypeScript وReact وVue وSvelte وAngular وSolidJS والواجهات الأمامية القائمة على Rust.
