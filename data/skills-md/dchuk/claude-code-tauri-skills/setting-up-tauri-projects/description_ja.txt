ユーザーがクロスプラットフォームのデスクトップおよびモバイル アプリケーションを構築するための新しい Tauri v2 プロジェクトを作成および初期化できるようにします。 macOS、Windows、Linux のシステム前提条件とセットアップ要件について説明します。 create-tauri-app または手動による Tauri CLI 初期化を使用したプロジェクトの作成をガイドします。プロジェクトのディレクトリ構造と設定ファイルについて説明します。バニラ JavaScript、TypeScript、React、Vue、Svelte、Angular、SolidJS、および Rust ベースのフロントエンドをサポートします。
