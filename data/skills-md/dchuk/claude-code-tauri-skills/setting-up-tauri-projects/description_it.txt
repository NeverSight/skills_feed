Aiuta gli utenti a creare e inizializzare nuovi progetti Tauri v2 per la creazione di applicazioni desktop e mobili multipiattaforma. Copre i prerequisiti di sistema e i requisiti di configurazione per macOS, Windows e Linux. Guida attraverso la creazione del progetto utilizzando create-tauri-app o l'inizializzazione manuale della CLI Tauri. Spiega la struttura delle directory del progetto e i file di configurazione. Supporta frontend JavaScript, TypeScript, React, Vue, Svelte, Angular, SolidJS e basati su Rust.
