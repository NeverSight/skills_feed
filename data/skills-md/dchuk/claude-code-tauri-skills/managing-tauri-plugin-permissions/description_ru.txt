Помогает пользователям настраивать разрешения, возможности и безопасность плагина Tauri. Охватывает возможности, специфичные для платформы, разрешения, ориентированные на окна, использование официальных разрешений плагинов и разрешений сообщества, а также написание пользовательских разрешений плагинов с областями действия.
