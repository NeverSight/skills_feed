Подробное руководство по созданию мини-приложений Telegram с помощью React с использованием @tma.js/sdk-react. Охватывает инициализацию SDK, монтаж компонентов, сигналы, темы, обработку кнопки «Назад», управление областью просмотра, данные инициализации, глубокие ссылки и макет среды для разработки. Используйте при создании или отладке мини-приложений Telegram с помощью React.
