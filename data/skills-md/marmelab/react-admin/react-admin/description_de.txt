Diese Fähigkeit sollte beim Erstellen, Ändern oder Debuggen einer React-Admin-Anwendung verwendet werden – einschließlich der Erstellung von Ressourcen, Listen, Formularen, Datenabruf, Authentifizierung, Beziehungen zwischen Entitäten, benutzerdefinierten Seiten oder jeder mit React-Admin erstellten CRUD-Administratorschnittstelle.
