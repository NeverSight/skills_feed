Выполняйте код Python в безопасной изолированной среде через [inference.sh](https://inference.sh).
Предустановленные: NumPy, Pandas, Matplotlib, запросы, BeautifulSoup, Selenium,
Playwright, MoviePy, Pillow, OpenCV, Trimesh и еще более 100 библиотек.
Использование для: обработки данных, парсинга веб-страниц, манипулирования изображениями, создания видео,
Обработка 3D-моделей, генерация PDF, вызовы API, скрипты автоматизации.
Триггеры: Python, выполнение кода, запуск сценария, парсинг веб-страниц, анализ данных,
обработка изображений, редактирование видео, 3D-модели, автоматизация, pandas, matplotlib
