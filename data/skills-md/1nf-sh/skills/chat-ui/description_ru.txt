Строительные блоки пользовательского интерфейса чата для React/Next.js из ui.inference.sh.
Компоненты: контейнер, сообщения, ввод, индикаторы набора текста, аватары.
Возможности: интерфейсы чата, списки сообщений, обработка ввода, потоковая передача.
Использование для: создания пользовательских интерфейсов чата, интерфейсов обмена сообщениями, помощников с искусственным интеллектом.
Триггеры: пользовательский интерфейс чата, компонент чата, список сообщений, ввод в чат, чат shadcn,
чат реагирования, интерфейс чата, пользовательский интерфейс обмена сообщениями, пользовательский интерфейс беседы, строительные блоки чата
