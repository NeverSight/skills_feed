In Batterien enthaltene Agentenkomponente für React/Next.js von ui.inference.sh.
Eine Komponente mit integrierter Laufzeit, Tools, Streaming, Genehmigungen und Widgets.
Fähigkeiten: Drop-in-Agent, Human-in-the-Loop, clientseitige Tools, Formularausfüllung.
Verwendung für: Erstellen von KI-Chat-Schnittstellen, Agenten-Benutzeroberflächen, SaaS-Copiloten, Assistenten.
Auslöser: Agentenkomponente, Agenten-Benutzeroberfläche, Chat-Agent, Shadcn-Agent, Reaktionsagent,
Agenten-Benutzeroberfläche, KI-Assistenten-Benutzeroberfläche, Copilot-Benutzeroberfläche, Inferenz-Benutzeroberfläche, Mensch in der Schleife
