عناصر واجهة المستخدم التعريفية من JSON لـ React/Next.js من ui.inference.sh.
عرض واجهات مستخدم تفاعلية غنية من استجابات الوكيل المنظمة.
القدرات: النماذج، الأزرار، البطاقات، التخطيطات، المدخلات، الاختيارات، مربعات الاختيار.
يُستخدم من أجل: واجهات المستخدم التي أنشأها الوكيل، والنماذج الديناميكية، وعرض البيانات، والبطاقات التفاعلية.
المشغلات: عناصر واجهة المستخدم، واجهة المستخدم التعريفية، واجهة مستخدم json، عارض عناصر واجهة المستخدم، عناصر واجهة مستخدم الوكيل،
واجهة المستخدم الديناميكية، عناصر واجهة المستخدم للنموذج، عناصر واجهة المستخدم للبطاقة، عناصر واجهة المستخدم shadcn، واجهة المستخدم للإخراج المنظم
