JavaScript/TypeScript SDK для inference.sh — запускайте приложения искусственного интеллекта, создавайте агенты, интегрируйте более 150 моделей.
Пакет: @inferencesh/sdk (установка npm). Полная поддержка TypeScript, потоковая передача, загрузка файлов.
Создавайте агентов с помощью шаблонов или специальных шаблонов, API для создания инструментов, навыков и одобрения людей.
Использование для: интеграции JavaScript, TypeScript, Node.js, React, Next.js, интерфейсных приложений.
