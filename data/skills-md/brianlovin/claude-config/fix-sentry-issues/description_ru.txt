Используйте Sentry MCP для обнаружения, сортировки и устранения производственных проблем с помощью анализа первопричин. Используйте, когда вас просят исправить проблемы Sentry, сортировать производственные ошибки, исследовать всплески ошибок или убрать шум Sentry. Требуется сервер Sentry MCP. Срабатывает при «исправлении часового», «ошибках сортировки», «производственных ошибках», «проблемах с часовым».
