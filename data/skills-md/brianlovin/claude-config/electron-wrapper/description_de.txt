Integrieren Sie eine Bun-Web-App in eine Electron-Desktop-App mit nativer Fensterverwaltung, automatischen Updates, Code-Signierung und CI/CD-Verteilung. Verwenden Sie diese Option, wenn der Benutzer eine native Desktop-Anwendung von einem vorhandenen Bun-basierten Webserver erstellen, sie für macOS/Windows paketieren, automatische Aktualisierungen einrichten oder Electron UX-Anliegen wie Drag-Regionen und Ampeln bearbeiten möchte. Wird auch zum Schneiden von Releases oder zum Markieren von Versionen für Electron-Apps verwendet.
