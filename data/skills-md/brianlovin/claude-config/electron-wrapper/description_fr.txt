Intégrez une application Web Bun dans une application de bureau Electron avec gestion native des fenêtres, mises à jour automatiques, signature de code et distribution CI/CD. À utiliser lorsque l'utilisateur souhaite créer une application de bureau native à partir d'un serveur Web existant basé sur Bun, la conditionner pour macOS/Windows, configurer la mise à jour automatique ou gérer les problèmes d'Electron UX tels que les régions de déplacement et les feux de circulation. À utiliser également lors de la suppression de versions ou du marquage de versions pour les applications Electron.
