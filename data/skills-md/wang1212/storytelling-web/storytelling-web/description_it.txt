Crea pagine Web visive esplorative straordinarie e ricche di animazioni da contenuti di testo. Supporta sia la modalità Diapositive (stile presentazione con navigazione tramite diapositive) sia la modalità Scorrimento (narrativa basata sullo scorrimento con animazioni progressive). Da utilizzare quando l'utente desidera trasformare il testo in esperienze Web coinvolgenti con un design visivo distintivo, animazioni fluide e interazioni coinvolgenti. Genera HTML a file singolo con dipendenza zero.
