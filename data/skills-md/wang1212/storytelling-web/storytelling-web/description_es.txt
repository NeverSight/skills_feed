Cree impresionantes páginas web visuales exploratorias ricas en animaciones a partir de contenido de texto. Admite tanto el modo Diapositivas (estilo de presentación con navegación por diapositivas) como el modo Desplazamiento (narrativa basada en desplazamiento con animaciones progresivas). Úselo cuando el usuario quiera transformar el texto en experiencias web inmersivas con un diseño visual distintivo, animaciones fluidas e interacciones atractivas. Genera HTML de un solo archivo sin dependencia.
