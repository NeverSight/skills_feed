Diese Fähigkeit sollte verwendet werden, wenn React-Komponenten mit TypeScript erstellt werden, Hooks eingegeben werden, Ereignisse verarbeitet werden oder wenn React TypeScript, React 19 und Serverkomponenten erwähnt werden. Behandelt typsichere Muster für React 18–19, einschließlich allgemeiner Komponenten, ordnungsgemäßer Ereignistypisierung und Routing-Integration (TanStack Router, React Router).
