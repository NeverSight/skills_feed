Этот навык следует использовать при создании компонентов React с помощью TypeScript, перехватах ввода, обработке событий или когда упоминаются React TypeScript, React 19, серверные компоненты. Охватывает типобезопасные шаблоны для React 18–19, включая универсальные компоненты, правильную типизацию событий и интеграцию маршрутизации (TanStack Router, React Router).
