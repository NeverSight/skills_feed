توثيق احتياجات بيانات الواجهة الأمامية لمطوري الواجهة الخلفية. يُستخدم عندما تحتاج الواجهة الأمامية إلى توصيل متطلبات واجهة برمجة التطبيقات (API) إلى الواجهة الخلفية، أو عندما يقول المستخدم "متطلبات الواجهة الخلفية"، أو "ما البيانات التي أحتاجها"، أو "متطلبات واجهة برمجة التطبيقات"، أو عندما يصف احتياجات البيانات لواجهة المستخدم.
