Создает комплексные документы передачи обслуживания для плавной передачи сеансов агента ИИ. Срабатывает, когда: (1) пользователь запрашивает передачу обслуживания/сохранение памяти/контекста, (2) окно контекста приближается к емкости, (3) основная веха задачи завершена, (4) завершение рабочего сеанса, (5) пользователь говорит «сохранить состояние», «создать передачу обслуживания», «Мне нужно сделать паузу», «контекст заполняется», (6) возобновляет работу с «передачей нагрузки», «возобновить с», «продолжить с того места, где мы остановились». Активно предлагает передачу после существенной работы (множественное редактирование файлов, сложная отладка, решения по архитектуре). Решает проблему длительного исчерпания контекста агента, позволяя новым агентам продолжать работу с нулевой неоднозначностью.
