دليل شامل للكتابة وتصحيح الأخطاء والحفاظ على اختبارات الكاتب المسرحي في TypeScript. يُستخدم عند كتابة اختبارات Playwright جديدة، أو مراجعة كود الاختبار، أو إصلاح الاختبارات غير المستقرة، أو تصحيح أخطاء الاختبار، أو إعداد البنية الأساسية للاختبار، أو تنفيذ نموذج كائن الصفحة، أو تكوين خطوط أنابيب CI/CD، أو تحسين أداء الاختبار، أو إصلاح مشكلات المحدد، أو التعامل مع المصادقة، أو السخرية من واجهات برمجة التطبيقات، أو تنظيم مجموعات الاختبار، أو استكشاف مشكلات Playwright وإصلاحها. يغطي E2E، والمكونات، وواجهة برمجة التطبيقات (API)، واختبار الانحدار البصري.
