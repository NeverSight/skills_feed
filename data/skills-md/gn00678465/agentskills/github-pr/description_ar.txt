المساعدة في إنشاء أو تعديل طلبات سحب GitHub. قم تلقائيًا بإنشاء عناوين وأوصاف العلاقات العامة الصينية التقليدية بناءً على الالتزامات في الفرع الحالي. مناسب لطلبات مثل "إنشاء علاقات عامة" أو "إنشاء طلب سحب" أو "ساعدني في فتح علاقات عامة" أو "تعديل محتوى العلاقات العامة". استخدم GitHub CLI (gh) لإجراء عمليات العلاقات العامة.
