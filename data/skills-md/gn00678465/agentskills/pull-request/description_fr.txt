Aider à créer, modifier, afficher ou gérer les demandes de tirage GitHub (PR). Analysez automatiquement les validations et les modifications des branches pour générer des titres et des descriptions de relations publiques en chinois traditionnel conformes. Les cas d'utilisation incluent : (1) la création d'un nouveau PR à partir de la branche actuelle, (2) la modification du titre, de la description, des réviseurs ou des étiquettes d'un PR existant, (3) la vérification du statut du PR d'une branche (ouverte, fusionnée, fermée), (4) la consolidation automatique de plusieurs modifications pour générer des résumés. Applicable aux scénarios contenant des requêtes telles que « créer une PR », « créer une demande d'extraction », « aidez-moi à ouvrir une PR » ou « modifier le contenu d'une PR ». Les opérations sont effectuées via la CLI GitHub (gh).
