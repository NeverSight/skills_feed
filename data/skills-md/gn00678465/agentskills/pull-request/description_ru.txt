Помогите создавать, редактировать, просматривать или управлять запросами на извлечение GitHub (PR). Автоматически анализируйте фиксации и изменения ветвей, чтобы создавать соответствующие традиционным китайским PR-заголовкам и описаниям. Варианты использования включают в себя: (1) создание нового PR на основе текущей ветки, (2) изменение названия, описания, рецензентов или меток существующего PR, (3) проверка статуса PR ветки (открытый, объединенный, закрытый), (4) автоматическое объединение нескольких изменений для создания сводок. Применимо к запросам, содержащим такие фразы, как «создать PR», «создать запрос на включение», «помогите мне открыть PR» или «отредактировать PR-контент». Операции выполняются через GitHub CLI (gh).
