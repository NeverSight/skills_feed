GitHub PR(Pull Request) 생성, 수정, 보기 또는 관리를 지원합니다. 분기 커밋 및 변경 사항을 자동으로 분석하여 호환되는 중국어 번체 PR 제목 및 설명을 생성합니다. 사용 사례에는 (1) 현재 분기에서 새 PR 생성, (2) 기존 PR의 제목, 설명, 검토자 또는 레이블 수정, (3) 분기의 PR 상태(열림, 병합, 닫힘) 확인, (4) 여러 변경 사항 자동 요약이 포함됩니다. "PR 생성", "풀 요청 생성", "PR 열기 지원" 또는 "PR 콘텐츠 수정"이 포함된 요청에 적합합니다. 작업은 GitHub CLI(gh)를 통해 수행됩니다.
