Assist in creating, modifying, viewing, or managing GitHub Pull Requests (PRs). Automatically analyze branch commits and changes to generate compliant traditional Chinese PR titles and descriptions. Use cases include: (1) creating a new PR from the current branch, (2) modifying an existing PR’s title, description, reviewers, or labels, (3) checking a branch’s PR status (open, merged, closed), (4) automatically summarizing multiple changes. Applicable to requests containing phrases like “create PR,” “create pull request,” “open a PR for me,” or “modify PR content.” Operations are performed via the GitHub CLI (gh).
