Helfen Sie beim Erstellen, Ändern, Anzeigen oder Verwalten von GitHub Pull Requests (PRs). Analysieren Sie automatisch Branch-Commits und -Änderungen, um konforme PR-Titel und -Beschreibungen im traditionellen Chinesisch zu generieren. Zu den Anwendungsfällen gehören: (1) Erstellen eines neuen PR aus dem aktuellen Zweig, (2) Ändern des Titels, der Beschreibung, der Prüfer oder Beschriftungen eines vorhandenen PR, (3) Überprüfen des PR-Status eines Zweigs (offen, zusammengeführt, geschlossen), (4) automatische Zusammenfassung mehrerer Änderungen. Gilt für Anfragen, die Phrasen wie „PR erstellen“, „Pull-Anfrage erstellen“, „Hilf mir beim Öffnen einer PR“ oder „PR-Inhalt ändern“ enthalten. Operationen werden über die GitHub CLI (gh) ausgeführt.
