GitHub プル リクエスト (PR) の作成、変更、表示、管理を支援します。ブランチのコミットと変更を自動的に分析し、準拠した繁体字中国語の PR タイトルと説明を生成します。ユースケースには、(1) 現在のブランチから新しい PR を作成する、(2) 既存の PR のタイトル、説明、レビュー担当者、またはラベルを変更する、(3) ブランチの PR ステータス (オープン、マージ、クローズ) を確認する、(4) 複数の変更を自動的に要約するなどが含まれます。 「PR の作成」、「プル リクエストの作成」、「PR を開くのを手伝ってください」、「PR コンテンツの変更」などを含むリクエストに適用されます。操作は GitHub CLI (gh) を介して実行されます。
