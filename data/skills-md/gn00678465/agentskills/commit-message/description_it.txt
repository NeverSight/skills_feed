Analizza le modifiche git staged e genera automaticamente un messaggio di commit in cinese tradizionale secondo le specifiche dei commit convenzionali. Adatto per generare messaggi di commit conformi quando richiesto, ad esempio "scrivimi un messaggio di commit", "genera un commit" o "invia modifiche". Controllerà automaticamente se sei sul ramo principale/master e fornirà suggerimenti appropriati.
