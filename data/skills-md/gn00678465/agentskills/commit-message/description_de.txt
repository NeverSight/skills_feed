Analysieren Sie von Git bereitgestellte Änderungen und generieren Sie automatisch eine Commit-Nachricht in traditionellem Chinesisch gemäß der Spezifikation für konventionelle Commits. Geeignet zum Generieren konformer Commit-Nachrichten auf Anfrage, z. B. „Schreiben Sie mir eine Commit-Nachricht“, „Generieren Sie ein Commit“ oder „Senden Sie Änderungen“. Es prüft automatisch, ob Sie sich im Haupt-/Master-Zweig befinden und macht entsprechende Vorschläge.
