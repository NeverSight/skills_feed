قم بتحليل التغييرات المرحلية لـ git وقم تلقائيًا بإنشاء رسالة التزام صينية تقليدية وفقًا لمواصفات الالتزامات التقليدية. مناسب لإنشاء رسائل التزام متوافقة عند الطلب، مثل "اكتب لي رسالة التزام" أو "إنشاء التزام" أو "إرسال تغييرات". سوف يتحقق تلقائيًا مما إذا كنت في الفرع الرئيسي/الرئيسي ويقدم الاقتراحات المناسبة.
