قم بإنشاء التزامات git الذرية باستخدام التحليل الإرشادي وتنسيق الالتزام التقليدي وقواعد التدريج والتحليل العميق الاختياري والدفع الاختياري. يُستخدم عندما يطلب المستخدم صياغة رسالة التزام، أو تنفيذ تغييرات، أو تنفيذ/تنفيذ تعديلات الجلسة فقط، أو تشغيل سير عمل التزام بعلامات مثل --all، أو --deep، أو --push.
