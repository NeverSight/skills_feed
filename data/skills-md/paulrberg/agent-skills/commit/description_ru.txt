Создавайте атомарные коммиты git с помощью эвристического анализа, обычного форматирования коммитов, правил промежуточного хранения, дополнительного глубокого анализа и дополнительной push-уведомлений. Используйте, когда пользователь запрашивает создание сообщения о фиксации, фиксацию изменений, редактирование сеанса только на этапе/фиксации или запуск рабочего процесса фиксации с такими флагами, как --all, --deep или --push.
