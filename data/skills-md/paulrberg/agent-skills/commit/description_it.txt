Crea commit atomic git con analisi euristica, formattazione dei commit convenzionali, regole di staging, analisi approfondita opzionale e push opzionale. Da utilizzare quando l'utente chiede di creare un messaggio di commit, eseguire il commit di modifiche, mettere in stage/solo commit le modifiche della sessione o eseguire un flusso di lavoro di commit con flag come --all, --deep o --push.
