Этот навык следует использовать, когда пользователь просит «создать justfile», «написать только рецепты», «настроить только параметры», «добавить только модули», «использовать только атрибуты», «настроить автоматизацию задач», упоминает justfile, просто запуск команд или автоматизацию задач с помощью just.
