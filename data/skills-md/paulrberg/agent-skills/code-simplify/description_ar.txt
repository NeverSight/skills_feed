تبسيط التعليمات البرمجية الموجودة وإعادة هيكلتها لسهولة القراءة وقابلية الصيانة دون تغيير السلوك. يُستخدم عندما يطلب المستخدم "تبسيط التعليمات البرمجية"، أو "تنظيف التعليمات البرمجية"، أو "إعادة البناء من أجل الوضوح"، أو "تقليل التعقيد"، أو "تحسين إمكانية القراءة"، أو "تسهيل صيانته"، أو يطلب تحسين التعليمات البرمجية التي تم تعديلها مؤخرًا.
