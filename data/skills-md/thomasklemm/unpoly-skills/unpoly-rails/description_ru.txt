Интеграция Ruby on Rails для Unpoly. Используйте при работе с драгоценным камнем unpoly-rails или при создании приложений Unpoly-powered Rails. Охватывает серверные помощники (up?, up.target, up.layer.accept, up.layer.dismiss, up.layer.open, up.validate?, up.cache.expire, up.context, up.emit, up.safe_callback, Fresh_when, render_nothing), помощники представления Rails (link_to, form_with, button_to с атрибутами Unpoly), флэш-сообщения с [up-hungry], турбо-сосуществование. (отключение Turbo Drive в Rails 7+), настройка CSP с помощью csp_meta_tag и глобальная конфигурация для всех.
