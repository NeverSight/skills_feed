Intégration Ruby on Rails pour Unpoly. À utiliser lorsque vous travaillez avec la gemme unpoly-rails ou lorsque vous créez des applications Rails alimentées par Unpoly. Couvre les assistants côté serveur (up?, up.target, up.layer.accept, up.layer.dismiss, up.layer.open, up.validate?, up.cache.expire, up.context, up.emit, up.safe_callback, fresh_when, render_nothing), les assistants d'affichage Rails (link_to, form_with, button_to avec attributs Unpoly), les messages flash avec [up-hungry], la coexistence Turbo (désactivation de Turbo Drive dans Rails 7+), configuration CSP avec csp_meta_tag et configuration globale de suivi.
