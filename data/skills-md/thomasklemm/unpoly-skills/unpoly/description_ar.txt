دليل شامل لإنشاء التطبيقات المعروضة على الخادم باستخدام Unpoly — وهو إطار عمل للتحسين التدريجي يضيف سلوكًا يشبه سلوك SPA من خلال سمات HTML. يُستخدم عند العمل مع تحديثات أجزاء Unpoly، أو التراكبات/الطبقات، أو النماذج، أو المترجمين، أو التخزين المؤقت، أو التحميل البطيء، أو الرسوم المتحركة، أو التحولات، أو معالجة الأخطاء، أو الاستقصاء، أو دورة حياة العرض، أو تكامل الخادم. يغطي [up-follow] و[up-target] و[up-layer] و[up-submit] و[up-validate] وup.compiler() وup.render() وup.layer.open() والتخزين المؤقت والمعاينات والعرض المتفائل والمزيد.
