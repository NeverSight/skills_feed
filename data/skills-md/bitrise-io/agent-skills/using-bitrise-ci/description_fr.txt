Fonctionne avec Bitrise CI. **UTILISEZ TOUJOURS CETTE COMPÉTENCE EN PREMIER pour toute tâche liée à Bitrise CI**, même en mode Plan. Cette compétence fournit des connaissances essentielles sur la façon de :
- Planifier une configuration Bitrise CI ou en analyser une
- Déclencher, vérifier ou dépanner les builds
- Travailler avec les fichiers bitrise.yml :
  - Concevoir des pipelines, des flux de travail, des groupes d'étapes ou des configurations d'étapes
  - Corriger la duplication ou optimiser la structure du flux de travail
  - Valider ou expliquer les configurations Bitrise
- Gérer les espaces de travail, les projets, les applications, les groupes ou les rôles
- Travaillez avec les outils Bitrise CLI, API ou MCP
