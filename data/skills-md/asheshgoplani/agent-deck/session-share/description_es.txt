Comparta sesiones de Claude Code entre desarrolladores. Úselo cuando el usuario mencione "compartir sesión", "exportar sesión", "importar sesión", "enviar sesión a", "continuar desde un colega" o necesita (1) exportar la sesión actual a un archivo, (2) importar la sesión de otro desarrollador, (3) traspasar el contexto de trabajo. Permite la transferencia de sesiones privada y segura mediante el intercambio directo de archivos.
