Partagez des sessions Claude Code entre développeurs. À utiliser lorsque l'utilisateur mentionne « partager la session », « exporter la session », « importer la session », « envoyer la session à », « continuer depuis un collègue » ou doit (1) exporter la session en cours vers un fichier, (2) importer la session d'un autre développeur, (3) transférer le contexte de travail. Permet un transfert de session privé et sécurisé via le partage direct de fichiers.
