مدير الجلسة الطرفية لوكلاء ترميز الذكاء الاصطناعي. يُستخدم عندما يذكر المستخدم "agent-deck"، أو "session"، أو "sub-agent"، أو "MCP Attach"، أو "git Worktree"، أو يحتاج إلى (1) إنشاء/بدء/إيقاف/إعادة تشغيل/جلسات شوكة، (2) إرفاق/فصل MCPs، (3) إدارة المجموعات/ملفات التعريف، (4) الحصول على مخرجات الجلسة، (5) تكوين مجموعة الوكيل، (6) استكشاف المشكلات وإصلاحها، (7) تشغيل الوكلاء الفرعيين، أو (8) إنشاء/إدارة شجرة العمل جلسات. يغطي أوامر CLI واختصارات TUI وخيارات config.toml والأتمتة.
