استخدم AwesomeAssertions لتأكيدات الاختبار بطلاقة وقابلة للقراءة.
استخدمه عندما تحتاج إلى كتابة تأكيدات واضحة، أو مقارنة الكائنات، أو التحقق من المجموعات، أو التعامل مع المطابقة المعقدة.
يغطي واجهة برمجة التطبيقات (API) الكاملة بما في ذلك يجب () وBeEquivalentTo () وContain () و ThrowAsync () وما إلى ذلك.
