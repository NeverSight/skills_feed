Verwenden Sie AwesomeAssertions für flüssige, lesbare Testaussagen.
Verwenden Sie es, wenn Sie klare Aussagen schreiben, Objekte vergleichen, Sammlungen überprüfen oder komplexe Übereinstimmungen verarbeiten müssen.
Deckt die vollständige API ab, einschließlich Should(), BeEquivalentTo(), Contain(), ThrowAsync() usw.
