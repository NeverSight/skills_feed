Используйте AwesomeAssertions для плавных и читаемых тестовых утверждений.
Используйте его, когда вам нужно писать четкие утверждения, сравнивать объекты, проверять коллекции или обрабатывать сложные сопоставления.
Охватывает весь API, включая Must(), BeEquiваленто(), Contain(), ThrowAsync() и т. д.
