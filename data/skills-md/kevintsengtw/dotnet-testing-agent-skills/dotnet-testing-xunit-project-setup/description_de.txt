Spezialkenntnisse in der Erstellung und Konfiguration von xUnit-Testprojekten.
Wird verwendet, wenn ein Testprojekt erstellt, die Projektstruktur eingerichtet, NuGet-Pakete konfiguriert und Testordner organisiert werden müssen.
Behandelt csproj-Konfiguration, Paketverwaltung, Projektstruktur, xunit.runner.json-Einstellungen usw.
