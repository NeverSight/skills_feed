Competenze specializzate nella creazione e configurazione di progetti di test xUnit.
Utilizzato quando è necessario creare un progetto di test, impostare la struttura del progetto, configurare i pacchetti NuGet e organizzare le cartelle di test.
Copre la configurazione di csproj, la gestione dei pacchetti, la struttura del progetto, le impostazioni di xunit.runner.json, ecc.
