Специализированные навыки создания и настройки тестовых проектов xUnit.
Используется, когда необходимо создать тестовый проект, настроить структуру проекта, настроить пакеты NuGet и организовать тестовые папки.
Охватывает конфигурацию csproj, управление пакетами, структуру проекта, настройки xunit.runner.json и т. д.
