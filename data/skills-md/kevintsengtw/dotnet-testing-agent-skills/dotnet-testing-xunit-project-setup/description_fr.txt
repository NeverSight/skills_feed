Compétences spécialisées dans la création et la configuration de projets de test xUnit.
Utilisé lorsqu'un projet de test doit être créé, la structure du projet configurée, les packages NuGet configurés et les dossiers de test organisés.
Couvre la configuration de csproj, la gestion des packages, la structure du projet, les paramètres xunit.runner.json, etc.
