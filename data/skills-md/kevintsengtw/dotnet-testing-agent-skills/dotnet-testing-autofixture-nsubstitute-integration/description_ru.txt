Руководство по интеграции AutoFixture и NSubstitute — реализация автоматического издевательства.  
Охватывает AutoNSubstituteDataAttribute, механизм заморозки, стратегию жадного построения.  
Включает пользовательскую обработку особых зависимостей, таких как IMapper (AutoMapper/Mapster).  
Подходит для сложных тестов внедрения зависимостей, макетной проверки и настройки поведения.
