Специализированные навыки для сравнения сложных объектов и глубокой проверки.
Используется, когда необходимо глубокое сравнение объектов, исключение определенных свойств, обработка циклических ссылок или проверка DTO/сущностей.
Охватывает BeEquiваленто, исключение, включение, пользовательские правила сравнения и т. д.
