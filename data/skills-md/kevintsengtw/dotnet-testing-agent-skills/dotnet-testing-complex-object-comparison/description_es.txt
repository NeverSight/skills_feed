Habilidades especializadas para manejar la comparación de objetos complejos y la validación profunda.
Se utiliza cuando se necesita una comparación profunda de objetos, excluir propiedades específicas, manejar referencias circulares o validar DTO/entidades.
Cubre BeEquivalentTo, Excluyendo, Incluyendo, reglas de comparación personalizadas, etc.
