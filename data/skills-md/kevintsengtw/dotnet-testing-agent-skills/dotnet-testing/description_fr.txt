Centre de présentation et d'orientation des principes fondamentaux des tests .NET. Déclenché lorsque les utilisateurs posent des questions de test générales telles que « comment écrire des tests .NET », « démarrer avec les tests .NET », « de quels outils de test ai-je besoin », « les meilleures pratiques de test » ou « apprendre les tests à partir de zéro ». Il recommande des ensembles appropriés de sous-compétences basées sur des besoins spécifiques, couvrant 19 principes fondamentaux, notamment les bases des tests, les données de test, les assertions, les simulations et les scénarios spéciaux.
