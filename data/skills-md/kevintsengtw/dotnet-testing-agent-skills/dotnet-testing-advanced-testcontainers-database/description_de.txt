Spezialkenntnisse im Testen von Container-Datenbanken mit Testcontainern.
Wird verwendet, wenn das tatsächliche Datenbankverhalten getestet werden muss, indem SQL Server/PostgreSQL/MySQL-Container verwendet und EF Core/Dapper getestet werden.
Deckt den Start von Containern, Datenbankmigrationen, Testisolation, Containerfreigabe usw. ab.
