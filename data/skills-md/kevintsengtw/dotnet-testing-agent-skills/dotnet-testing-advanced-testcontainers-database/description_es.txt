Habilidades especializadas en pruebas de bases de datos en contenedores con Testcontainers.
Se utiliza cuando es necesario probar el comportamiento real de la base de datos, empleando contenedores SQL Server/PostgreSQL/MySQL y probando EF Core/Dapper.
Cubre el inicio de contenedores, migraciones de bases de datos, aislamiento de pruebas, uso compartido de contenedores, etc.
