Базовые навыки автоматического создания тестовых данных с помощью AutoFixture.  
Используйте его, когда вам нужно быстро создать тестовые объекты, сократить шаблонный код и добиться анонимности тестирования.  
Охватывает Fixture.Create, CreateMany, обработку циклических ссылок и интеграцию с xUnit.
