Эксперт по рефакторингу кода для улучшения качества, читаемости, удобства сопровождения и производительности кода. Специализируется на шаблонах рефакторинга Java и Python, устранении «запашков» кода и применении принципов чистого кода. Используйте при рефакторинге кода, улучшении существующих реализаций или устранении технического долга.
