اكتشاف العلامة التجارية وتطوير مفهوم الشعار. يُستخدم عند تبادل الأفكار حول الشعار، أو استكشاف الاتجاهات المرئية، أو تحليل شعارات المنافسين، أو تطوير مفاهيم الشعار قبل الإنشاء. يتم تشغيلها عند "أفكار الشعار"، أو "مفهوم الشعار"، أو "هوية العلامة التجارية"، أو "العصف الذهني للشعار"، أو "شعارات المنافسين"، أو "استكشاف الهوية المرئية"، أو أي تخطيط مسبق للشعار.
