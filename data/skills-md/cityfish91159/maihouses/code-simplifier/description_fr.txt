Simplifiez et affinez le code pour améliorer la clarté, la cohérence et la maintenabilité tout en préservant toutes les fonctionnalités.  
Par défaut, concentrez-vous sur le code le plus récemment modifié, sauf indication contraire.

Déclencheurs :
- Après avoir terminé la mise en œuvre d'une fonctionnalité, un nettoyage est nécessaire
- Lorsque le code s'avère trop complexe ou difficile à comprendre
- Refactorisation du code existant pour une meilleure lisibilité
- L'utilisateur mentionne "simplifier", "simplifier", "nettoyer" ou "refactoriser".
