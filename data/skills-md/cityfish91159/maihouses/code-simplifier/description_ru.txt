Упростите и уточните код, чтобы повысить ясность, последовательность и удобство сопровождения, сохраняя при этом всю функциональность.  
По умолчанию фокусируется на самом последнем измененном коде, если не указано иное.

Триггеры:
- После завершения реализации функции необходима очистка.
- Когда код оказывается слишком сложным или трудным для понимания.
- Рефакторинг существующего кода для лучшей читаемости.
– Пользователь упоминает слова «упростить», «упростить», «очистку» или «рефакторинг».
