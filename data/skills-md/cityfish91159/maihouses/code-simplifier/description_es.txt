Simplifique y refine el código para mejorar la claridad, la coherencia y la facilidad de mantenimiento, preservando al mismo tiempo toda la funcionalidad.  
De forma predeterminada, céntrese en el código modificado más recientemente, a menos que se le indique lo contrario.

Desencadenantes:
- Después de completar la implementación y limpieza de una función, es necesario
- Cuando se descubre que el código es demasiado complejo o difícil de entender.
- Refactorización del código existente para una mejor legibilidad
- El usuario menciona "simplificar", "simplificar", "limpiar" o "refactorizar".
