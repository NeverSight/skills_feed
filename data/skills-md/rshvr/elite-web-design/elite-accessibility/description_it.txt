Accessibilità del movimento e conformità WCAG per le animazioni web. CRITICO: questa abilità dovrebbe essere caricata ogni volta che si implementano le animazioni. Copre la preferenza per il movimento ridotto (modelli CSS e GSAP), requisiti di contrasto WCAG, APCA, gestione della messa a fuoco, navigazione tramite tastiera e considerazioni sullo screen reader. Utilizzare quando viene chiesto su: movimento ridotto, sensibilità al movimento, disturbi vestibolari, accessibilità, a11y, WCAG, rapporti di contrasto, stili di messa a fuoco o come rendere accessibili le animazioni.
