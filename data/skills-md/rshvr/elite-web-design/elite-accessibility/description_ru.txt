Доступность движения и соответствие WCAG для веб-анимации. ВАЖНО: этот навык следует загружать при каждой реализации анимации. Обложки предпочитают ограниченное движение (шаблоны CSS и GSAP), требования к контрастности WCAG, APCA, управление фокусом, навигацию с помощью клавиатуры и особенности чтения с экрана. Используйте, когда вас спрашивают о: уменьшении движения, чувствительности к движению, вестибулярных расстройствах, доступности, a11y, WCAG, коэффициентах контрастности, стилях фокусировки или обеспечении доступности анимации.
