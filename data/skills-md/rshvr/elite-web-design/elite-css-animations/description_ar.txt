تقنيات الرسوم المتحركة الأصلية لـ CSS، بما في ذلك واجهة برمجة تطبيقات الرسوم المتحركة التي تعتمد على التمرير، وانتقالات العرض، وقاعدة @property، والتأثيرات المرئية. استخدمه عند سؤالك عن: الرسوم المتحركة CSS، الرسوم المتحركة التي تعتمد على التمرير، المخطط الزمني للتمرير CSS، عرض التحولات، انتقالات الصفحة، @property، تحريك متغيرات CSS، الرسوم المتحركة لمسار المقطع، مرشح الخلفية، وضع المزج المختلط، أقنعة CSS، الرسوم المتحركة CSS فقط، الرسوم المتحركة بدون JavaScript، أو عندما لا تكون هناك حاجة/مطلوب GSAP. يتضمن دليل القرار لـ CSS مقابل GSAP.
