Cree agentes de IA de Pydantic de nivel de producción con arquitectura en capas. Úselo al crear agentes de IA con sistemas de herramientas, inyección de dependencia, transmisión de respuestas, soporte de múltiples proveedores o administración del estado de la conversación. Incluye scripts de andamiaje para una configuración rápida del proyecto y patrones arquitectónicos integrales para servicios web FastAPI y aplicaciones CLI.
