Создавайте ИИ-агенты Pydantic промышленного уровня с многоуровневой архитектурой. Используйте при создании агентов ИИ с системами инструментов, внедрением зависимостей, потоковой передачей ответов, поддержкой нескольких поставщиков или управлением состоянием диалога. Включает сценарии создания шаблонов для быстрой настройки проекта и комплексные архитектурные шаблоны для веб-сервисов FastAPI и приложений CLI.
