قم بالتشغيل بعد إجراء تغييرات Docyrus API لاكتشاف الأخطاء ومشكلات الأداء ومشاكل جودة التعليمات البرمجية. يُستخدم عند تنفيذ أو تعديل التعليمات البرمجية التي تستخدم خطافات مجموعة Docyrus (.list، .get، .create، .update، .delete)، أو مكالمات RestApiClient المباشرة، أو حمولات الاستعلام باستخدام المرشحات/الحسابات/الصيغ/childQueries/pivots، أو تكامل TanStack Query مع مصادر بيانات Docyrus. يتم تشغيل المهام التي تتضمن منطق Docyrus API أو جلب البيانات أو الطفرات أو إنشاء حمولة الاستعلام.
