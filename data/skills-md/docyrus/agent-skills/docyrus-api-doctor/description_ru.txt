Запускайте после внесения изменений в Docyrus API, чтобы выявить ошибки, проблемы с производительностью и качеством кода. Используйте при реализации или изменении кода, который использует перехватчики сбора Docyrus (.list, .get, .create, .update, .delete), прямые вызовы RestApiClient, полезные данные запросов с фильтрами/вычислениями/формулами/childQueries/основными точками или интеграцию запросов TanStack с источниками данных Docyrus. Запускается при выполнении задач, включающих логику API Docyrus, выборку данных, мутации или построение полезных данных запроса.
