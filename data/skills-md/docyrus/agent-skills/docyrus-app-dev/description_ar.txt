أنشئ تطبيقات ويب React TypeScript باستخدام Docyrus كواجهة خلفية. يُستخدم عند إنشاء أو تعديل التطبيقات التي تصادق مع Docyrus OAuth2، أو جلب/تحوير البيانات عبر مكتبة @docyrus/api-client، أو استخدام المجموعات التي تم إنشاؤها تلقائيًا لعمليات CRUD، أو إنشاء استعلامات باستخدام عوامل التصفية والتجميعات والصيغ والمحاور والاستعلامات الفرعية مقابل مصادر بيانات Docyrus. تشغيل المهام التي تتضمن @docyrus/api-client، أو @docyrus/signin، أو مجموعات Docyrus، أو استعلامات مصدر البيانات، أو تطوير تطبيق React المدعوم من Docyrus.
