Docyrus をバックエンドとして使用して React TypeScript Web アプリケーションを構築します。 Docyrus OAuth2 で認証するアプリを作成または変更する場合、@docyrus/api-client ライブラリを介してデータをフェッチ/変更する場合、CRUD 操作に自動生成されたコレクションを使用する場合、または Docyrus データ ソースに対してフィルター、集計、数式、ピボット、および子クエリを使用したクエリを構築する場合に使用します。 @docyrus/api-client、@docyrus/signin、Docyrus コレクション、データ ソース クエリ、または Docyrus を利用した React アプリ開発に関連するタスクをトリガーします。
