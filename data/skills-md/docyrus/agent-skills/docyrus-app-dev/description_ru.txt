Создавайте веб-приложения React TypeScript, используя Docyrus в качестве серверной части. Используйте при создании или изменении приложений, которые проходят аутентификацию с помощью Docyrus OAuth2, извлекают/изменяют данные через библиотеку @docyrus/api-client, используют автоматически создаваемые коллекции для операций CRUD или создают запросы с фильтрами, агрегатами, формулами, сводными данными и дочерними запросами к источникам данных Docyrus. Запускается по задачам, связанным с @docyrus/api-client, @docyrus/signin, коллекциями Docyrus, запросами к источникам данных или разработкой приложений React на основе Docyrus.
