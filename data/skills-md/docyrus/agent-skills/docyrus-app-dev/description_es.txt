Cree aplicaciones web React TypeScript utilizando Docyrus como backend. Úselo al crear o modificar aplicaciones que se autentican con Docyrus OAuth2, recuperar/mutar datos a través de la biblioteca @docyrus/api-client, usar colecciones generadas automáticamente para operaciones CRUD o crear consultas con filtros, agregaciones, fórmulas, pivotes y consultas secundarias contra fuentes de datos de Docyrus. Activadores de tareas que involucran @docyrus/api-client, @docyrus/signin, colecciones de Docyrus, consultas de fuentes de datos o desarrollo de aplicaciones React respaldadas por Docyrus.
