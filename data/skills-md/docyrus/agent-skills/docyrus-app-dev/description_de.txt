Erstellen Sie React TypeScript-Webanwendungen mit Docyrus als Backend. Verwenden Sie diese Option, wenn Sie Apps erstellen oder ändern, die sich mit Docyrus OAuth2 authentifizieren, Daten über die @docyrus/api-client-Bibliothek abrufen/ändern, automatisch generierte Sammlungen für CRUD-Vorgänge verwenden oder Abfragen mit Filtern, Aggregationen, Formeln, Pivots und untergeordneten Abfragen für Docyrus-Datenquellen erstellen. Wird bei Aufgaben ausgelöst, die @docyrus/api-client, @docyrus/signin, Docyrus-Sammlungen, Datenquellenabfragen oder die von Docyrus unterstützte React-App-Entwicklung betreffen.
