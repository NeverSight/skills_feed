استخدم Docyrus CLI ("docyrus") للتفاعل مع منصة Docyrus من المحطة. يُستخدم عندما يطلب المستخدم المصادقة أو إدراج التطبيقات أو الاستعلام عن سجلات البيانات أو إدارتها (`ds`) أو إدارة كائنات مخطط مصدر بيانات تطبيق المطورين (`studio`) أو إرسال طلبات واجهة برمجة التطبيقات (API) أو تبديل البيئات/المستأجرين/الحسابات أو اكتشاف مواصفات OpenAPI عبر أداة سطر الأوامر `docyrus`. تشغيل المهام التي تتضمن أوامر docyrus CLI، أو عمليات Docyrus المستندة إلى المحطة الطرفية، أو `docyrus ds list`، أو `docyrus studio`، أو `docyrus Discover`، أو `docyrus auth`، أو `docyrus env`، أو مسارات عمل Docyrus المستندة إلى shell.
