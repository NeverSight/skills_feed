Docyrus CLI(`docyrus`)를 사용하여 터미널에서 Docyrus 플랫폼과 상호 작용합니다. 사용자가 `docyrus` 명령줄 도구를 통해 인증, 앱 나열, 데이터 레코드 쿼리/관리(`ds`), 개발 앱 데이터 소스 스키마 개체 관리(`studio`), API 요청 전송, 환경/테넌트/계정 전환 또는 OpenAPI 사양 검색을 요청할 때 사용합니다. docyrus CLI 명령, 터미널 기반 Docyrus 작업, `docyrus ds list`, `docyrus studio`, `docyrus discover`, `docyrus auth`, `docyrus env` 또는 셸 기반 Docyrus 워크플로와 관련된 작업을 트리거합니다.
