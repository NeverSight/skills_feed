使用 Docyrus CLI (`docyrus`) 从终端与 Docyrus 平台进行交互。当用户要求进行身份验证、列出应用程序、查询/管理数据记录（“ds”）、管理开发应用程序数据源架构对象（“studio”）、发送 API 请求、切换环境/租户/帐户或通过“docyrus”命令行工具发现 OpenAPI 规范时使用。在涉及 docyrus CLI 命令、基于终端的 Docyrus 操作、“docyrus ds list”、“docyrus studio”、“docyrus discovery”、“docyrus auth”、“docyrus env”或基于 shell 的 Docyrus 工作流程的任务上触发。
