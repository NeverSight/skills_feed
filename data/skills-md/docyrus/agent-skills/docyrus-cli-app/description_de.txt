Verwenden Sie die Docyrus-CLI („docyrus“), um vom Terminal aus mit der Docyrus-Plattform zu interagieren. Wird verwendet, wenn der Benutzer über das Befehlszeilentool „docyrus“ die Authentifizierung, das Auflisten von Apps, das Abfragen/Verwalten von Datensätzen („ds“), das Verwalten von Schemaobjekten der Entwicklungs-App-Datenquellen („Studio“), das Senden von API-Anfragen, das Wechseln von Umgebungen/Mandanten/Konten oder das Ermitteln von OpenAPI-Spezifikationen anfordert. Wird bei Aufgaben ausgelöst, die Docyrus-CLI-Befehle, terminalbasierte Docyrus-Vorgänge, „docyrus ds list“, „docyrus studio“, „docyrus discover“, „docyrus auth“, „docyrus env“ oder Shell-basierte Docyrus-Workflows umfassen.
