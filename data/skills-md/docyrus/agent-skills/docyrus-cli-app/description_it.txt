Utilizza la CLI Docyrus ("docyrus") per interagire con la piattaforma Docyrus dal terminale. Da utilizzare quando l'utente richiede di autenticarsi, elencare app, interrogare/gestire record di dati (`ds`), gestire oggetti dello schema dell'origine dati dell'app di sviluppo (`studio`), inviare richieste API, cambiare ambiente/tenant/account o scoprire le specifiche OpenAPI tramite lo strumento da riga di comando "docyrus". Attiva attività che coinvolgono comandi CLI di docyrus, operazioni Docyrus basate su terminale, "docyrus ds list", "docyrus studio", "docyrus discover", "docyrus auth", "docyrus env" o flussi di lavoro Docyrus basati su shell.
