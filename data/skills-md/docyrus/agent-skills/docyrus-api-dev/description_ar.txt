قم بتطوير التطبيقات باستخدام Docyrus API مع مكتبات @docyrus/api-client و@docyrus/signin. يُستخدم عند إنشاء تطبيقات تصادق مع Docyrus OAuth2 (PKCE، وiframe، وبيانات اعتماد العميل، ورمز الجهاز)، أو إجراء استدعاءات REST API إلى نقاط نهاية مصدر بيانات Docyrus، أو إنشاء حمولات استعلام باستخدام عوامل التصفية والتجميعات والصيغ والمحاور والاستعلامات الفرعية. تشغيل المهام التي تتضمن تكامل Docyrus API، أو استخدام @docyrus/api-client، أو مصادقة @docyrus/signin، أو إنشاء استعلام مصدر البيانات، أو استهلاك نقطة نهاية Docyrus REST.
