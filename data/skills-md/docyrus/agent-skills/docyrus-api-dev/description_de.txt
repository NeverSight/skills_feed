Entwickeln Sie Anwendungen mithilfe der Docyrus-API mit den Bibliotheken @docyrus/api-client und @docyrus/signin. Verwenden Sie diese Option, wenn Sie Apps erstellen, die sich mit Docyrus OAuth2 (PKCE, Iframe, Client-Anmeldeinformationen, Gerätecode) authentifizieren, REST-API-Aufrufe an Docyrus-Datenquellenendpunkte durchführen oder Abfragenutzlasten mit Filtern, Aggregationen, Formeln, Pivots und untergeordneten Abfragen erstellen. Wird bei Aufgaben ausgelöst, die die Integration der Docyrus-API, die Verwendung von @docyrus/api-client, die Authentifizierung mit @docyrus/signin, die Erstellung von Datenquellenabfragen oder die Nutzung von Docyrus-REST-Endpunkten betreffen.
