Разрабатывайте приложения с использованием API Docyrus с библиотеками @docyrus/api-client и @docyrus/signin. Используйте при создании приложений, которые проходят аутентификацию с помощью Docyrus OAuth2 (PKCE, iframe, учетные данные клиента, код устройства), выполняют вызовы REST API к конечным точкам источника данных Docyrus или создают полезные данные запросов с фильтрами, агрегатами, формулами, сводными данными и дочерними запросами. Запускается при выполнении задач, связанных с интеграцией API Docyrus, использованием @docyrus/api-client, аутентификацией @docyrus/signin, построением запросов к источнику данных или использованием конечной точки Docyrus REST.
