Desarrolle aplicaciones utilizando la API de Docyrus con las bibliotecas @docyrus/api-client y @docyrus/signin. Úselo al crear aplicaciones que se autentican con Docyrus OAuth2 (PKCE, iframe, credenciales de cliente, código de dispositivo), realice llamadas de API REST a puntos finales de fuentes de datos de Docyrus o construya cargas útiles de consultas con filtros, agregaciones, fórmulas, pivotes y consultas secundarias. Activadores de tareas que involucran la integración de la API de Docyrus, el uso de @docyrus/api-client, la autenticación de @docyrus/signin, la creación de consultas de fuentes de datos o el consumo de puntos finales REST de Docyrus.
