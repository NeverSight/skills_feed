@docyrus/api-client 및 @docyrus/signin 라이브러리와 함께 Docyrus API를 사용하여 애플리케이션을 개발합니다. Docyrus OAuth2(PKCE, iframe, 클라이언트 자격 증명, 장치 코드)로 인증하는 앱을 구축하거나 Docyrus 데이터 소스 엔드포인트에 대한 REST API 호출을 수행하거나 필터, 집계, 수식, 피벗 및 하위 쿼리를 사용하여 쿼리 페이로드를 구성할 때 사용합니다. Docyrus API 통합, @docyrus/api-client 사용, @docyrus/signin 인증, 데이터 소스 쿼리 구축 또는 Docyrus REST 엔드포인트 소비와 관련된 작업을 트리거합니다.
