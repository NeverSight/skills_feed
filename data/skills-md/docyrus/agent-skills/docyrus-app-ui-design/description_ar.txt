تصميم وبناء مكونات واجهة المستخدم على مستوى الإنتاج لتطبيقات Docyrus React باستخدام مكتبات المكونات المفضلة. يُستخدم عند إنشاء لوحات المعلومات أو النماذج أو جداول البيانات أو التخطيطات أو أي عناصر واجهة مستخدم. تشغيل المهام التي تتضمن اختيار المكونات، أو تصميم واجهة المستخدم، أو إنشاء لوحة المعلومات، أو تصميم التخطيط، أو shadcn، أو diceui، أو animate-ui، أو docyrus-ui، أو مكونات reui، أو طلبات إنشاء واجهات المستخدم.
