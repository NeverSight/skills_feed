استخدم أدوات Docyrus Architect MCP لإدارة مصادر البيانات والحقول والتعدادات والتطبيقات وبيانات الاستعلام في منصة Docyrus. يُستخدم عندما يطلب المستخدم إنشاء أو تحديث أو حذف أو الاستعلام عن مصادر البيانات أو الحقول أو خيارات التعداد أو التطبيقات عبر خادم docyrus-architect MCP. يُستخدم أيضًا عند إنشاء التقارير أو لوحات المعلومات أو إجراء تحليل البيانات الذي يتطلب الاستعلام عن مصادر بيانات Docyrus باستخدام عوامل التصفية أو التجميعات أو الصيغ أو المحاور أو الاستعلامات الفرعية.
