Используйте инструменты MCP Docyrus Architect для управления источниками данных, полями, перечислениями, приложениями и данными запросов на платформе Docyrus. Используйте, когда пользователь запрашивает создание, обновление, удаление или запрос источников данных, полей, параметров перечисления или приложений через сервер MCP docyrus-architect. Также используйте при создании отчетов, информационных панелей или выполнении анализа данных, который требует запроса к источникам данных Docyrus с помощью фильтров, агрегатов, формул, сводных данных или дочерних запросов.
