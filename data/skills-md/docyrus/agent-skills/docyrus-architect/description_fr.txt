Utilisez les outils MCP de Docyrus Architect pour gérer les sources de données, les champs, les énumérations, les applications et les données de requête dans la plateforme Docyrus. À utiliser lorsque l'utilisateur demande à créer, mettre à jour, supprimer ou interroger des sources de données, des champs, des options d'énumération ou des applications via le serveur MCP docyrus-architect. À utiliser également lors de la création de rapports, de tableaux de bord ou d'analyses de données nécessitant l'interrogation de sources de données Docyrus avec des filtres, des agrégations, des formules, des pivots ou des requêtes enfants.
