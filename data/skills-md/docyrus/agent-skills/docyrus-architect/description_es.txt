Utilice las herramientas MCP de Docyrus Architect para administrar fuentes de datos, campos, enumeraciones, aplicaciones y consultar datos en la plataforma Docyrus. Úselo cuando el usuario solicite crear, actualizar, eliminar o consultar fuentes de datos, campos, opciones de enumeración o aplicaciones a través del servidor MCP de docyrus-architect. Utilícelo también al crear informes, paneles o realizar análisis de datos que requieran consultar fuentes de datos de Docyrus con filtros, agregaciones, fórmulas, pivotes o consultas secundarias.
