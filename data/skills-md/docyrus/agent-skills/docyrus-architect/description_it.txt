Utilizza gli strumenti MCP di Docyrus Architect per gestire origini dati, campi, enumerazioni, app e dati di query nella piattaforma Docyrus. Da utilizzare quando l'utente richiede di creare, aggiornare, eliminare o eseguire query su origini dati, campi, opzioni di enumerazione o app tramite il server MCP docyrus-architect. Da utilizzare anche quando si creano report, dashboard o si eseguono analisi dei dati che richiedono l'esecuzione di query su origini dati Docyrus con filtri, aggregazioni, formule, pivot o query secondarie.
