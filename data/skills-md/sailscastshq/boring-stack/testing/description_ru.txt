Шаблоны тестирования для The Boring JavaScript Stack — модульное тестирование с помощью средства запуска тестов Node.js, сквозное тестирование с помощью Playwright и интеграционное тестирование с помощью inertia-sails/test. Используйте этот навык при написании, настройке или отладке тестов в приложении Sails.js + Inertia.js.
