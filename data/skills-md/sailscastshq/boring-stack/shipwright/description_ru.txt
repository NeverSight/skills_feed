Система сборки Shipwright для The Boring JavaScript Stack — конвейер ресурсов на основе Rsbuild, заменяющий Grunt, с плагинами платформы для React/Vue/Svelte, Tailwind CSS, поддержкой SSR и сервером разработки с HMR. Используйте этот навык при настройке сборок, управлении ресурсами или отладке сервера разработки.
