Patrones de autenticación para The Boring JavaScript Stack: autenticación basada en sesión con contraseña, enlaces mágicos, claves de acceso (WebAuthn), autenticación de dos factores (TOTP/correo electrónico/códigos de respaldo), restablecimiento de contraseña y OAuth. Utilice esta habilidad al implementar o modificar cualquier flujo de autenticación en una aplicación Sails.js.
