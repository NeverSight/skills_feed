Шаблоны аутентификации для The Boring JavaScript Stack — аутентификация на основе сеанса с паролем, магическими ссылками, ключами доступа (WebAuthn), двухфакторная аутентификация (TOTP/электронная почта/резервные коды), сброс пароля и OAuth. Используйте этот навык при реализации или изменении любого потока аутентификации в приложении Sails.js.
