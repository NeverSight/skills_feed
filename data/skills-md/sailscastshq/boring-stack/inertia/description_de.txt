Inertia.js serverseitiges Rendering, Weiterleitungen, gemeinsame Daten, Formulare, Validierung und clientseitige Muster für The Boring JavaScript Stack. Verwenden Sie diese Fähigkeit, wenn Sie Seiten erstellen, Formularübermittlungen bearbeiten, Weiterleitungen verwalten, Daten teilen oder mit einer beliebigen Inertia.js-Funktion in einer Sails.js-Anwendung arbeiten.
