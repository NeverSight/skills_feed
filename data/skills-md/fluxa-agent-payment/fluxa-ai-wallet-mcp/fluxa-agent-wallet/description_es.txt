Integración de FluxA Agent Wallet a través de CLI. Permite a los agentes realizar pagos x402 para API pagas, enviar pagos en USDC a cualquier billetera y crear enlaces de pago para recibir pagos, todo a través de una herramienta CLI independiente de Node.js. Úselo cuando el usuario pregunte sobre pagos criptográficos, x402, transferencias USDC, enlaces de pago o interacción con FluxA Agent Wallet.
