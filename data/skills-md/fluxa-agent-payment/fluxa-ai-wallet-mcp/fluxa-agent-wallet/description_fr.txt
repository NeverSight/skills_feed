Intégration du portefeuille d'agents FluxA via CLI. Permet aux agents d'effectuer des paiements x402 pour les API payantes, d'envoyer des paiements USDC vers n'importe quel portefeuille et de créer des liens de paiement pour recevoir des paiements, le tout via un outil CLI Node.js autonome. À utiliser lorsque l'utilisateur pose des questions sur les paiements cryptés, x402, les transferts USDC, les liens de paiement ou l'interaction avec le portefeuille d'agent FluxA.
