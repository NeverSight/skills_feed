تكامل محفظة وكيل FluxA عبر CLI. يمكّن الوكلاء من إجراء دفعات x402 لواجهات برمجة التطبيقات المدفوعة، وإرسال دفعات USDC إلى أي محفظة، وإنشاء روابط دفع لتلقي المدفوعات - كل ذلك من خلال أداة Node.js CLI المستقلة. يُستخدم عندما يسأل المستخدم عن مدفوعات التشفير، أو x402، أو تحويلات USDC، أو روابط الدفع، أو التفاعل مع FluxA Agent Wallet.
