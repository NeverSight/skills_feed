FluxA Agent Wallet-Integration über CLI. Ermöglicht es Agenten, x402-Zahlungen für kostenpflichtige APIs zu tätigen, USDC-Auszahlungen an jede Wallet zu senden und Zahlungslinks für den Empfang von Zahlungen zu erstellen – alles über ein eigenständiges Node.js-CLI-Tool. Verwenden Sie diese Option, wenn der Benutzer nach Krypto-Zahlungen, x402, USDC-Überweisungen, Zahlungslinks oder der Interaktion mit der FluxA Agent Wallet fragt.
