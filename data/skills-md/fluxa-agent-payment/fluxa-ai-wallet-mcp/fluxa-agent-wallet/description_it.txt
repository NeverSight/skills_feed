Integrazione del portafoglio FluxA Agent tramite CLI. Consente agli agenti di effettuare pagamenti x402 per API a pagamento, inviare pagamenti USDC a qualsiasi portafoglio e creare collegamenti di pagamento per ricevere pagamenti, il tutto tramite uno strumento CLI Node.js autonomo. Da utilizzare quando l'utente richiede informazioni su pagamenti crittografici, x402, trasferimenti USDC, collegamenti di pagamento o interazione con il portafoglio dell'agente FluxA.
