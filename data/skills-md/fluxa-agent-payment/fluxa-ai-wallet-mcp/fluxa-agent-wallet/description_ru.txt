Интеграция агентского кошелька FluxA через CLI. Позволяет агентам совершать платежи x402 за платные API, отправлять выплаты в долларах США на любой кошелек и создавать платежные ссылки для получения платежей — и все это с помощью автономного инструмента CLI Node.js. Используйте, когда пользователь спрашивает о криптовалютных платежах, x402, переводах USDC, платежных ссылках или взаимодействии с кошельком агента FluxA.
