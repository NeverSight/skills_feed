Извлекайте форматирование из существующих документов Word и создавайте новые документы того же формата, но с другим содержимым. Используйте этот навык, когда пользователям необходимо создать несколько документов с единообразным форматированием, копировать шаблоны документов или поддерживать корпоративные стандарты документов для различного контента.
