Extrahieren Sie Formatierungen aus vorhandenen Word-Dokumenten und erstellen Sie neue Dokumente mit demselben Format, aber unterschiedlichem Inhalt. Nutzen Sie diese Fähigkeit, wenn Benutzer mehrere Dokumente mit konsistenter Formatierung erstellen, Dokumentvorlagen replizieren oder unternehmensweite Dokumentstandards für verschiedene Inhalte beibehalten müssen.
