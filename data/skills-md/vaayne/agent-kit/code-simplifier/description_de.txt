Wird ausgelöst bei „diesen Code vereinfachen“, „bereinigen“, „zur besseren Lesbarkeit umgestalten“, „diesen sauberer machen“ oder nach der Implementierung von Funktionen, wenn eine Codepolitur erforderlich ist.
