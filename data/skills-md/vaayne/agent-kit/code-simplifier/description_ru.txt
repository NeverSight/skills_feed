Упрощает и совершенствует код для обеспечения ясности, последовательности и удобства сопровождения, сохраняя при этом функциональность. Срабатывает при «упростить этот код», «очистить», «рефакторить для удобства чтения», «сделать это чище» или после реализации функций, когда требуется полировка кода.
