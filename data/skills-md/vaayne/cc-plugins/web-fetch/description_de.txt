Rufen Sie mit der Jina Reader API saubere Inhalte aus URLs ab und extrahieren Sie sie. Verwenden Sie diese Option, wenn Benutzer Webseiteninhalte lesen, Artikeltext extrahieren oder URL-Inhalte zur Analyse abrufen müssen. Löst aus bei „Diese Seite abrufen“, „Diese URL lesen“, „Inhalt extrahieren von“, „Inhalt abrufen von“, „Was sagt diese Seite“.
