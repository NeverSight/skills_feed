Übertragen Sie den Kontext auf eine neue fokussierte Sitzung. Verwenden Sie diese Option, wenn Sie eine neue Arbeitsphase beginnen, an eine neue Sitzung übergeben oder wenn der Kontext für die Fortsetzung zusammengefasst werden muss. Wird bei „Übergabe“, „Kontext übertragen“, „Neue Sitzung mit Kontext starten“ oder wenn der Benutzer die Arbeit in einer neuen Sitzung fortsetzen möchte, ausgelöst.
