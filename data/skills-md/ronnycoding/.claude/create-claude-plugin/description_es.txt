Guía para crear complementos de Claude Code que combinan habilidades, agentes, comandos, enlaces y servidores MCP para su distribución. Cubre la estructura del complemento, el formato Marketplace.json, la instalación, las pruebas y la publicación. Úselo cuando el usuario quiera crear complementos, distribuir habilidades/agentes, crear mercados o mencione la creación/empaquetado/publicación de complementos.
