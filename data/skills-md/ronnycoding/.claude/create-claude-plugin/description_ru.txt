Руководство по созданию плагинов Claude Code, объединяющих навыки, агентов, команды, перехватчики и серверы MCP для распространения. Охватывает структуру плагина, формат marketplace.json, установку, тестирование и публикацию. Используйте, когда пользователь хочет создавать плагины, распространять навыки/агентов, создавать торговые площадки или упоминает о создании/упаковке/публикации плагинов.
