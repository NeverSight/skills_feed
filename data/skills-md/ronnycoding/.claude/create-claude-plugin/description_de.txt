Leitfaden zum Erstellen von Claude Code-Plugins, die Fähigkeiten, Agenten, Befehle, Hooks und MCP-Server zur Verteilung bündeln. Behandelt die Plugin-Struktur, das Marketplace.json-Format, die Installation, das Testen und die Veröffentlichung. Verwenden Sie diese Option, wenn der Benutzer Plugins erstellen, Fähigkeiten/Agenten verteilen, Marktplätze aufbauen möchte oder die Erstellung/Verpackung/Veröffentlichung von Plugins erwähnt.
