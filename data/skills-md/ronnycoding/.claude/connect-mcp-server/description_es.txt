Guía para conectar servidores MCP (Model Context Protocol) a Claude Code con transportes HTTP, stdio y SSE. Cubre instalación, configuración, autenticación, variables de entorno y seguridad. Úselo cuando el usuario desee conectar servidores MCP, agregar integraciones, configurar servicios externos o mencione MCP, servidores, integraciones o herramientas externas.
