Leitfaden zum Verbinden von MCP-Servern (Model Context Protocol) mit Claude Code über HTTP-, STDIO- und SSE-Transporte. Behandelt Installation, Konfiguration, Authentifizierung, Umgebungsvariablen und Sicherheit. Verwenden Sie diese Option, wenn der Benutzer MCP-Server verbinden, Integrationen hinzufügen, externe Dienste konfigurieren möchte oder MCP, Server, Integrationen oder externe Tools erwähnt.
