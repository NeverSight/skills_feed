Руководство по подключению серверов MCP (Model Context Protocol) к Claude Code с помощью протоколов HTTP, stdio и SSE. Охватывает установку, настройку, аутентификацию, переменные среды и безопасность. Используйте, когда пользователь хочет подключить серверы MCP, добавить интеграции, настроить внешние службы или упоминает MCP, серверы, интеграции или внешние инструменты.
