Guida per la connessione dei server MCP (Model Context Protocol) a Claude Code con trasporti HTTP, stdio e SSE. Copre installazione, configurazione, autenticazione, variabili di ambiente e sicurezza. Da utilizzare quando l'utente desidera connettere server MCP, aggiungere integrazioni, configurare servizi esterni o menzionare MCP, server, integrazioni o strumenti esterni.
