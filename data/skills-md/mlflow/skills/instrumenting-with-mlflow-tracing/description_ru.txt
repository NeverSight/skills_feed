Инструментирует код Python и TypeScript с трассировкой MLflow для обеспечения наблюдаемости. Запускается по вопросам о добавлении трассировки, инструментировании агентов/приложений LLM, начале работы с трассировкой MLflow или отслеживании конкретных платформ (LangGraph, LangChain, OpenAI, DSPy, CrewAI, AutoGen). Примеры: «Как добавить трассировку?», «Как инструментировать мой агент?», «Как отследить мое приложение LangChain?», «Начало работы с трассировкой MLflow», «Отследить мое приложение TypeScript».
