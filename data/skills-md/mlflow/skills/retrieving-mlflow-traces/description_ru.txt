Извлекает трассировки MLflow с помощью CLI или Python API. Используйте, когда пользователь запрашивает трассировку по идентификатору, поиск трассировки, фильтрацию трассировки по состоянию/тегам/метаданным/времени выполнения, трассировку запроса или отладку неудачной трассировки. Триггеры по «получить трассировку», «поиск трассировок», «найти неудавшиеся трассировки», «фильтровать трассировки по», «трассировки медленнее», «запросить трассировки MLflow».
