Diese Fähigkeit sollte für Browser-Automatisierungsaufgaben mithilfe des Chrome DevTools Protocol (CDP) verwendet werden. Wird ausgelöst, wenn Benutzer Chrome mit Remote-Debugging starten, durch Seiten navigieren, JavaScript im Browserkontext ausführen, Screenshots erfassen oder interaktiv DOM-Elemente auswählen müssen. Kein MCP-Server erforderlich.
