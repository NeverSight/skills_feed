Conseils d'un spécialiste Web UI/UX pour la conception, la révision et le peaufinage de l'interface utilisateur des produits Web (mise en page, convivialité, microcopie, accessibilité, comportement réactif, formulaires, navigation). À utiliser lorsqu'on vous le demande pour améliorer l'UI/UX, auditer une page, concevoir un écran, créer une spécification de composant ou générer des conseils en-tête/manifeste/icône HTML pour une application Web ; applicable en option à Unreal UI (UMG) pour les heuristiques UX.
