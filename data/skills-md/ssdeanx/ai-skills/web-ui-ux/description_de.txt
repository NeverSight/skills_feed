Fachliche Anleitung für Web-UI/UX zum Entwerfen, Überprüfen und Verfeinern der Benutzeroberfläche von Webprodukten (Layout, Benutzerfreundlichkeit, Mikrokopie, Zugänglichkeit, Reaktionsverhalten, Formulare, Navigation). Verwenden Sie es, wenn Sie aufgefordert werden, UI/UX zu verbessern, eine Seite zu prüfen, einen Bildschirm zu entwerfen, eine Komponentenspezifikation zu erstellen oder HTML-Kopf-/Manifest-/Symbolanleitungen für eine Web-App zu generieren. optional anwendbar auf Unreal UI (UMG) für UX-Heuristiken.
