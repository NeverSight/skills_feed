Développe des ressources pour FiveM à l'aide du framework QBCore. Couvre la création de ressources, l'utilisation des objets de base, la gestion des joueurs, les rappels, les événements, les éléments, les tâches, les gangs, la base de données (oxmysql) et les meilleures pratiques. À utiliser lorsque l'utilisateur travaille avec des scripts FiveM, QBCore, Lua pour les serveurs QBCore, ou mentionne « QBCore.Functions », « GetCoreObject », « CitizenID » ou tout autre système du framework QBCore.
