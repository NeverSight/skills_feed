تطوير الموارد لـ FiveM باستخدام ESX Framework (Legacy). يغطي إنشاء الموارد، والكائن الأساسي (ESX)، ووظائف xPlayer، والأحداث، وعمليات الاسترجاعات، والعناصر، والوظائف، وقاعدة البيانات (oxmysql)، وأفضل الممارسات. يُستخدم عندما يعمل المستخدم مع FiveM أو ESX أو ESX Legacy أو يذكر `ESX.GetCoreObject` أو `xPlayer` أو `ESX.GetPlayerFromId` أو `esx:`.
