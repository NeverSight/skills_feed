إرشادات تطوير Flutter الشاملة التي تغطي الهندسة المعمارية وإدارة الحالة وتحسين الأداء والاختبار وأفضل الممارسات عبر الأنظمة الأساسية. يُستخدم عند إنشاء تطبيقات Flutter أو إعادة هيكلتها أو مراجعتها لمنصات الأجهزة المحمولة والويب وسطح المكتب. يغطي أنماط عناصر واجهة المستخدم، وإدارة حالة Riverpod/BLoC، والتنقل باستخدام go_router، وتكامل Firebase، وقنوات النظام الأساسي، والرسم المخصص، والرسوم المتحركة، وCI/CD، ونشر الإنتاج.
