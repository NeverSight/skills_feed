Используйте, когда пользователь запрашивает автономную работу для решения нескольких проблем. Управляет параллельными работниками с помощью инструмента Task, контролирует с помощью TaskOutput, обрабатывает циклы SLEEP/WAKE и работает до тех пор, пока область действия не будет завершена без вмешательства пользователя.
