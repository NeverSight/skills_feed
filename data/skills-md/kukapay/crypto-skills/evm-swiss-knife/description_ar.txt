تفاعل مع سلاسل الكتل المتوافقة مع EVM باستخدام أداة Foundry cast للاستعلام عن الأرصدة واستدعاء العقود وإرسال المعاملات واستكشاف blockchain. يُستخدم عند الحاجة إلى التفاعل مع شبكات Ethereum Virtual Machine عبر سطر الأوامر، بما في ذلك قراءة حالة العقد أو إرسال الأموال أو تنفيذ وظائف العقد أو فحص بيانات blockchain.
