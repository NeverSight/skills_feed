قم بتحسين الوثائق لمساعدي ترميز الذكاء الاصطناعي وماجستير القانون. يعمل على تحسين المستندات الخاصة بـ Claude وCopilot وأدوات الذكاء الاصطناعي الأخرى من خلال تحسين c7score وإنشاء llms.txt وإعادة الهيكلة المبنية على الأسئلة وتسجيل الجودة تلقائيًا. استخدمه عندما يُطلب منك تحسين الوثائق أو تحسينها أو تحسينها لمساعدي الذكاء الاصطناعي أو LLMs أو c7score أو context7 أو عند إنشاء ملفات llms.txt. استخدم أيضًا لتحليل جودة الوثائق، أو تحسين README، أو التأكد من أن المستندات تتبع أفضل الممارسات لأنظمة استرجاع LLM.
