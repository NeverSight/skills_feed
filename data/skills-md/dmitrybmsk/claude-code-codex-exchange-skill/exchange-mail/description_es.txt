Gestión completa de correo electrónico para Microsoft Exchange/Outlook. Úselo cuando Claude necesite enumerar correos electrónicos no leídos, leer el contenido del correo electrónico, responder correos electrónicos, marcar correos electrónicos como leídos o archivar correos electrónicos. Admite operaciones por lotes para correos electrónicos externos/internos. Activadores: revisar mi correo electrónico, correos electrónicos no leídos, responder a correos electrónicos, archivar correos electrónicos externos, marcar como leídos.
