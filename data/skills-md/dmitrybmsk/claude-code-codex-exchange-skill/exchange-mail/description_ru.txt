Полное управление электронной почтой для Microsoft Exchange/Outlook. Используйте, когда Клоду нужно составить список непрочитанных писем, прочитать содержимое электронной почты, ответить на электронные письма, пометить электронные письма как прочитанные или заархивировать электронные письма. Поддерживает пакетные операции для внешних/внутренних писем. Триггеры: проверить мою электронную почту, непрочитанные электронные письма, ответить на электронное письмо, заархивировать внешние электронные письма, пометить как прочитанные.
