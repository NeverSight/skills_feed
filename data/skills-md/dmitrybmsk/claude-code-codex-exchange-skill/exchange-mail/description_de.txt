Vollständige E-Mail-Verwaltung für Microsoft Exchange/Outlook. Verwenden Sie diese Funktion, wenn Claude ungelesene E-Mails auflisten, E-Mail-Inhalte lesen, auf E-Mails antworten, E-Mails als gelesen markieren oder E-Mails archivieren muss. Unterstützt Stapelvorgänge für externe/interne E-Mails. Auslöser: Meine E-Mails prüfen, ungelesene E-Mails, auf E-Mails antworten, externe E-Mails archivieren, als gelesen markieren.
