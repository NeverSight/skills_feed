Persistentes Speichersystem für KI-Agenten gemäß Model Context Protocol (MCP). Verwendung zum Speichern von Langzeitgedächtnissen über Sitzungen hinweg, zur semantischen Suche nach vergangenem Wissen, zum Erstellen von Wissensdiagrammen, zum automatischen Einfügen von Kontext, zum Deduplizieren von Erinnerungen und zum Synchronisieren mit dem Cloud-Speicher. Unverzichtbar für Agenten, die sich Entscheidungen, Lösungen, Präferenzen und erlernte Muster im Laufe der Zeit merken müssen.
