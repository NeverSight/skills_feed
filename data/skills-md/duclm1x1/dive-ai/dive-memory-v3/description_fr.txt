Système de mémoire persistante pour les agents IA suivant le Model Context Protocol (MCP). Utilisation pour stocker des souvenirs à long terme au fil des sessions, recherche sémantique de connaissances passées, création de graphiques de connaissances, injection automatique de contexte, déduplication de mémoires, synchronisation avec le stockage cloud. Indispensable pour les agents qui doivent mémoriser les décisions, les solutions, les préférences et les modèles appris au fil du temps.
