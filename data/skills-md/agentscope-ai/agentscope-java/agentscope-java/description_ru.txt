Экспертные навыки разработчика Java для платформы AgentScope Java — реактивной многоагентной системы, управляемой сообщениями, построенной на базе Project Reactor. Используйте при работе с реактивным программированием, интеграцией LLM, оркестровкой агентов, многоагентными системами или когда пользователь упоминает AgentScope, ReActAgent, Mono/Flux, Project Reactor или разработку агентов Java. Специализируется на неблокирующем коде, интеграции инструментов, перехватчиках, конвейерах и готовых к использованию агентских приложениях.
