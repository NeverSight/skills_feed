Competenze di sviluppatore Java esperto per il framework Java AgentScope: un sistema multi-agente reattivo e basato su messaggi basato su Project Reactor. Da utilizzare quando si lavora con programmazione reattiva, integrazione LLM, orchestrazione di agenti, sistemi multi-agente o quando l'utente menziona AgentScope, ReActAgent, Mono/Flux, Project Reactor o lo sviluppo di agenti Java. Specializzato in codice non bloccante, integrazione di strumenti, hook, pipeline e applicazioni di agenti pronti per la produzione.
