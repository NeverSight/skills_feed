Hub Canonical Playwright pour les tests E2E et l'automatisation ad hoc du navigateur. À utiliser lorsque l'utilisateur mentionne explicitement « Playwright », « @playwright/test », « npx playwright », « playwright.config.ts », « PWDEBUG », « trace viewer » ou « toHaveScreenshot ». Évitez de l'utiliser pour l'automatisation générique du navigateur, sauf si Playwright est demandé, et évitez de l'utiliser pour du pur scraping Web.
