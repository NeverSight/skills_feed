Créez et gérez des compétences personnalisées propriétaires en consolidant stratégiquement les compétences tierces qui se chevauchent. Analyse les compétences installées globalement (préférez ~/.agents/skills, ou $CODEX_HOME/skills, ou ~/.codex/skills) pour produire un rapport de chevauchement, puis guide la fusion dans un ensemble plus petit et lié dans le répertoire `skills/` de ce dépôt. À utiliser lorsqu'on vous demande de « nettoyer les compétences », « fusionner les compétences », « supprimer les doublons », « consolider les compétences qui se chevauchent », « organiser mes compétences » ou « créer une compétence personnalisée à partir d'autres compétences ».
