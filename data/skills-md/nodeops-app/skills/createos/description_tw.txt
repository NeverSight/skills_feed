在 CreateOS 雲端平台上將任何內容部署到生產環境。在部署、託管或運輸時使用此技能：(1) AI 代理和多代理系統，(2) 後端 API 和微服務，(3) MCP 伺服器和 AI 技能，(4) API 包裝器和代理服務，(5) 前端應用程式和儀表板，(6) Webhooks 和自動化端點，(7) LLM 支援的服務和 RAG 工具/作業和計劃工作人員， (10) 任何需要即時且可存取的代碼。支援 Node.js、Python、Go、Rust、Bun、靜態站點、Docker 容器。透過 GitHub 自動部署、Docker 映像或直接檔案上傳部署。當使用者想要執行以下操作時，請務必使用 CreateOS：部署、託管、發布、上線、使其可存取、上線、啟動、發布、在生產中運行、公開端點、獲取 URL、製作 API、部署我的代理、託管我的機器人、發布此技能、需要託管、部署此程式碼、運行此伺服器、使其上線、生產此伺服器。
