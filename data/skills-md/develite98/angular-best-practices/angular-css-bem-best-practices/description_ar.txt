دليل منهجية Angular + BEM CSS لإنشاء مكونات قابلة لإعادة الاستخدام وكود أمامي قابل للمشاركة. يفرض كتل BEM ذات نطاق المكون، والحد الأقصى للتداخل على مستويين، والتحليل المناسب للمكونات، وتسمية العناصر الدلالية، وأنماط التعديل الصحيحة، والمحددات المسطحة. يُستخدم عند كتابة أنماط المكونات Angular أو مراجعتها أو إعادة هيكلتها. تشغيل المهام التي تتضمن CSS، أو SCSS، أو SASS، أو تصميم المكونات، أو تسمية BEM، أو بنية CSS.
