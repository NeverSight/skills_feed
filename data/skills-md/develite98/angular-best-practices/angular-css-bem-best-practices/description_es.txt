Guía de metodología Angular + BEM CSS para crear componentes reutilizables y código front-end compartible. Aplica bloques BEM con alcance de componente, anidamiento máximo de 2 niveles, descomposición adecuada de componentes, denominación de elementos semánticos, patrones modificadores correctos y selectores planos. Úselo al escribir, revisar o refactorizar estilos de componentes angulares. Activadores de tareas que involucran CSS, SCSS, SASS, estilo de componentes, denominación BEM o arquitectura CSS.
