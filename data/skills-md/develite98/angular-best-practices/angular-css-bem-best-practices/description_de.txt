Angular + BEM CSS-Methodikleitfaden zum Erstellen wiederverwendbarer Komponenten und gemeinsam nutzbarem Front-End-Code. Erzwingt komponentenbezogene BEM-Blöcke, maximal zweistufige Verschachtelung, ordnungsgemäße Komponentenzerlegung, Benennung semantischer Elemente, korrekte Modifikatormuster und flache Selektoren. Wird beim Schreiben, Überprüfen oder Refactoring von Angular-Komponentenstilen verwendet. Wird bei Aufgaben ausgelöst, die CSS, SCSS, SASS, Komponentenstil, BEM-Benennung oder CSS-Architektur betreffen.
