Руководство по методологии Angular + BEM CSS для создания повторно используемых компонентов и общего внешнего кода. Обеспечивает соблюдение блоков BEM на уровне компонентов, максимальное двухуровневое вложение, правильную декомпозицию компонентов, семантическое наименование элементов, правильные шаблоны модификаторов и плоские селекторы. Используйте при написании, просмотре или рефакторинге стилей компонентов Angular. Запускается при выполнении задач, связанных с CSS, SCSS, SASS, стилем компонентов, именованием BEM или архитектурой CSS.
