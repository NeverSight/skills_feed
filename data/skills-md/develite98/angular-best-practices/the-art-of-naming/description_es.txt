Guía completa de convenciones de nomenclatura para aplicaciones TypeScript y Angular. Cubre reglas de mayúsculas y minúsculas (camelCase, PascalCase, UPPER_CASE), convenciones de prefijos (I para interfaces, _ para privados, T para genéricos), denominación booleana, el principio S-I-D (corto, intuitivo, descriptivo), duplicación de contexto y patrones de denominación estructurados (P/HC/LC para variables, A/HC/LC para funciones). Úselo al escribir, revisar o refactorizar código TypeScript. Activadores de tareas que involucran nombres, variables, funciones, interfaces o estilo de código.
