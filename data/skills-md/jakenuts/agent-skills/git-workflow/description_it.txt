Supporti automatizzati del flusso di lavoro git per attività di sviluppo comuni come la creazione di rami di funzionalità, la pulizia di rami uniti e il rebasing interattivo. Da utilizzare quando l'utente menziona la ramificazione di git, la pulizia dei rami, il flusso di lavoro delle funzionalità o l'automazione di git. Nessun prerequisito richiesto: utilizza comandi git nativi.
