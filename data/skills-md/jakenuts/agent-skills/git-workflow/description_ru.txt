Автоматизированные помощники по рабочим процессам git для распространенных задач разработки, таких как создание функциональных ветвей, очистка объединенных ветвей и интерактивное перебазирование. Используйте, когда пользователь упоминает ветвление git, очистку ветвей, рабочий процесс функций или автоматизацию git. Никаких предварительных условий не требуется — используются собственные команды git.
