Assistants de flux de travail Git automatisés pour les tâches de développement courantes telles que la création de branches de fonctionnalités, le nettoyage des branches fusionnées et le rebasage interactif. À utiliser lorsque l'utilisateur mentionne le branchement git, le nettoyage de branche, le workflow de fonctionnalités ou l'automatisation git. Aucun prérequis requis - utilise les commandes git natives.
