Ищите и анализируйте производственные журналы DealerVision с помощью API SolarWinds Observability. Используйте при исследовании ошибок, отладке проблем, проверке работоспособности системы или когда пользователь упоминает журналы, SolarWinds, производственные ошибки или мониторинг системы. Требуется установка инструмента CLI `logs`.
