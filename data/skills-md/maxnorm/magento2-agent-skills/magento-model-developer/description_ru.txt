Разрабатывает и реализует архитектуру уровня данных для Magento 2. Используется при создании моделей данных, разработке схем баз данных, реализации репозиториев или работе со структурами EAV/плоских таблиц. Мастера проектирования сущностей, шаблонов репозиториев, коллекций и оптимизации баз данных.
