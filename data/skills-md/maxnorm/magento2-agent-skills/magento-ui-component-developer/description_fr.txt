Crée des interfaces d'administration et des composants frontend sophistiqués à l'aide du framework de composants d'interface utilisateur de Magento. À utiliser lors de la création de grilles d'administration, de formulaires, de modaux ou d'interfaces basées sur les données. Maîtrisez la configuration XML, les modèles KnockoutJS, les fournisseurs de données et les implémentations de formulaires/grilles complexes.
