Создает сложные интерфейсы администратора и компоненты внешнего интерфейса, используя структуру компонентов пользовательского интерфейса Magento. Используйте при создании административных сеток, форм, модальных окон или интерфейсов, управляемых данными. Владеет конфигурацией XML, шаблонами KnockoutJS, поставщиками данных и сложными реализациями форм/сеток.
