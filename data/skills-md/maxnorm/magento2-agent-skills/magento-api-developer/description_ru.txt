Разрабатывает и реализует API-интерфейсы REST и GraphQL для Magento 2. Используйте при разработке API, создании сервисных контрактов, создании решений для автономной коммерции или интеграции с внешними системами. Основные контракты на обслуживание, объекты передачи данных, аутентификация и архитектура API корпоративного уровня.
