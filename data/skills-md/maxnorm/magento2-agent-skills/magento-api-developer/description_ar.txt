يصمم وينفذ واجهات برمجة تطبيقات REST وGraphQL لـ Magento 2. يُستخدم عند تطوير واجهات برمجة التطبيقات، أو إنشاء عقود الخدمة، أو إنشاء حلول تجارية مقطوعة الرأس، أو التكامل مع الأنظمة الخارجية. عقود الخدمة الرئيسية، وكائنات نقل البيانات، والمصادقة، وبنية واجهة برمجة التطبيقات (API) على مستوى المؤسسات.
