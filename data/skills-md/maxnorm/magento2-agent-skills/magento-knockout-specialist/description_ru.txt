Разрабатывает реактивные компоненты пользовательского интерфейса с использованием KnockoutJS для Magento 2. Используйте при работе с компонентами пользовательского интерфейса, реализации шаблонов MVVM или создании интерактивных функций внешнего интерфейса. Владеет наблюдаемыми шаблонами, привязкой данных и пользовательскими привязками.
