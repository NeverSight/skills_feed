使用 x402 保護的 API 進行神經網絡搜索和內容提取。比 WebSearch 更適合深入研究，比 WebFetch 更適合被阻止的網站。

用於：
- 深網研究和調查
- 查找與參考 URL 相似的頁面
- 從網頁中提取乾淨的文本
- 抓取阻止標準獲取器的網站
- 獲得事實問題的直接答案
- 需要多個來源的研究

觸發因素：
- “研究”、“調查”、“深入研究”、“尋找來源”
- “類似於”、“類似頁面”、“更像這樣”
- “抓取”、“從中提取內容”、“從中獲取文本”
- “被阻止的網站”、“無法訪問”、“付費專區”
- “這是什麼”，“解釋一下”，“回答這個”

對這些端點使用 mcp__x402__fetch。更喜歡 Exa 進行語義/神經搜索，更喜歡 Firecrawl 進行直接抓取。
