Ищите места, компании и местоположения с помощью API Google Maps через x402.

ИСПОЛЬЗОВАТЬ ДЛЯ:
- Поиск предприятий по названию или типу
- Поиск близлежащих мест
- Получение подробной информации о месте (адрес, время работы, отзывы)
- Поиск ресторанов, магазинов, услуг в районе
- Получение бизнес-рейтингов и отзывов

ТРИГГЕРЫ:
- «найти», «искать», «найти», «рядом»
- «рестораны рядом», «отели внутри», «магазины вокруг»
- «детали бизнеса», «часы работы», «отзывы о»
- «места», «что рядом», «как добраться»

Используйте mcp__x402__fetch для конечных точек Google Maps. Выберите частичный (0,02 доллара США) или полный (0,05–0,08 доллара США) в зависимости от потребностей в данных.
