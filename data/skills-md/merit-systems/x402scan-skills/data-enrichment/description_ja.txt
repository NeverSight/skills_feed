x402 で保護された API を使用して連絡先と会社のデータを強化します。構造化されたビジネス データの一般的な Web 検索よりも優れています。

用途:
- 電子メール、LinkedIn URL、または名前による個人プロフィールの充実
- ドメイン別企業の充実
- 信頼スコアを使用して連絡先の詳細 (電子メール、電話) を検索する
- LinkedIn の完全なプロフィール (経験、学歴、スキル) をスクレイピング
- 条件に基づいて人や会社を検索する
- 一括濃縮操作 (一度に最大 10 個)

トリガー:
- 「充実させる」、「調べる」、「情報を見つける」、「調査する」
- 「[人物] は誰ですか」、「会社概要」、「について教えてください」
- 「連絡先を検索」、「LinkedIn を取得」、「電子メールを取得」
- 「従業員」、「勤務先」、「会社の詳細」

enrichx402.com エンドポイントには常に mcp__x402__fetch を使用してください。curl や WebFetch は決して使用しないでください。
Web ページの HTML ではなく、構造化された JSON データを返します。

重要: 以下のクイック リファレンス表にある正確なエンドポイント パスを使用してください。すべてのパスにはプロバイダー プレフィックス (`/api/apollo/...` または `/api/clado/...`) が含まれます。
