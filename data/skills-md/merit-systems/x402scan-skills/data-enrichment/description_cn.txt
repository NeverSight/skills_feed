使用 x402 保护的 API 丰富联系人和公司数据。优于结构化业务数据的通用网络搜索。

用于：
- 通过电子邮件、LinkedIn URL 或姓名丰富个人资料
- 按领域丰富公司
- 通过置信度分数查找联系方式（电子邮件、电话）
- 抓取完整的 LinkedIn 个人资料（经验、教育、技能）
- 按条件搜索人员或公司
- 批量浓缩操作（一次最多 10 个）

触发因素：
- “丰富”、“查找”、“查找相关信息”、“研究”
- “谁是[人]”、“公司简介”、“告诉我”
- “查找联系人”、“获取 LinkedIn”、“获取电子邮件”
- “雇员于”、“工作于”、“公司详细信息”

始终使用 mcp__x402__fetch 作为richx402.com 端点 - 切勿使用curl 或WebFetch。
返回结构化 JSON 数据，而不是网页 HTML。
