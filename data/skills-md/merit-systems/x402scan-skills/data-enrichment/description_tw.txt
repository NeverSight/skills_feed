使用 x402 保護的 API 豐富聯繫人和公司數據。優於結構化業務數據的通用網絡搜索。

用於：
- 通過電子郵件、LinkedIn URL 或姓名豐富個人資料
- 按領域豐富公司
- 通過置信度分數查找聯繫方式（電子郵件、電話）
- 抓取完整的 LinkedIn 個人資料（經驗、教育、技能）
- 按條件搜索人員或公司
- 批量濃縮操作（一次最多 10 個）

觸發因素：
- “豐富”、“查找”、“查找相關信息”、“研究”
- “誰是[人]”、“公司簡介”、“告訴我”
- “查找聯繫人”、“獲取 LinkedIn”、“獲取電子郵件”
- “僱員於”、“工作於”、“公司詳細信息”

始終使用 mcp__x402__fetch 作為richx402.com 端點 - 切勿使用curl 或WebFetch。
返回結構化 JSON 數據，而不是網頁 HTML。
