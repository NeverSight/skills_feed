Ищите и отслеживайте социальные сети с помощью API X/Twitter (через Grok) и Reddit.

ИСПОЛЬЗОВАТЬ ДЛЯ:
- Поиск сообщений X/Twitter по ключевым словам или хэштегам.
- Поиск пользователей X/Twitter по критериям
- Получение последних сообщений пользователя
- Поиск сообщений и обсуждений Reddit.
- Получение комментариев из тем Reddit.
- Мониторинг и исследование социальных сетей.

ТРИГГЕРЫ:
- «twitter», «X», «твиты», «публикации на X»
- «реддит», «субреддит», «обсуждение на реддите»
- «что говорят люди», «социальные сети», «настроения»
- «трендовые», «вирусные», «популярные сообщения»
- «сообщения пользователя», «хронология», «недавняя активность»

Используйте mcp__x402__fetch для конечных точек Grok (X) и Reddit. Все конечные точки стоят 0,02 доллара США за вызов.
