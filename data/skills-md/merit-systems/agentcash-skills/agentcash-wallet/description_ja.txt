AgentCash ウォレットを管理し、自動支払いを使用して x402 で保護された API を呼び出します。 API キーやサブスクリプションはなく、資金提供されたウォレット (Base 上の USDC) だけです。

用途:
- API呼び出し前にウォレット残高を確認する
- 招待コードを無料クレジットと引き換える
- USDCの入金アドレスの取得
- x402 で保護されたオリジンでのエンドポイントの検出と価格設定
- Agentcash CLI 経由で有料 API リクエストを行う
- 支払い失敗のトラブルシューティング

トリガー:
- 「残高」、「ウォレット」、「資金」、「クレジット」、「x402」
- 「引き換え」、「招待コード」、「プロモーションコード」
- 「入金」、「資金追加」、「チャージ」
- 「検出」、「エンドポイント」、「API の種類」、「価格」
・「残高不足」「支払い失敗」
