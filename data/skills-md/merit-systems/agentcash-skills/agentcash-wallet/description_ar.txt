قم بإدارة محفظة AgentCash الخاصة بك واتصل بأي واجهة برمجة تطبيقات محمية بـ x402 مع الدفع التلقائي. لا توجد مفاتيح API، ولا اشتراكات - مجرد محفظة ممولة (USDC on Base).

استخدام ل:
- التحقق من رصيد المحفظة قبل مكالمات API
- استرداد رموز الدعوة للحصول على أرصدة مجانية
- الحصول على عنوان الإيداع لUSDC
- اكتشاف نقاط النهاية والتسعير على أي أصل محمي بنظام x402
- تقديم طلبات API المدفوعة عبر AgentCash CLI
- استكشاف أخطاء فشل الدفع وإصلاحها
