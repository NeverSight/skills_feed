使用 X/Twitter（通过 Grok）和 Reddit API 搜索和监控社交媒体。

用于：
- 通过关键字或主题标签搜索 X/Twitter 帖子
- 按标准查找 X/Twitter 用户
- 获取用户最近的帖子
- 搜索 Reddit 帖子和讨论
- 从 Reddit 线程获取评论
- 社交媒体监测和研究

触发因素：
- “twitter”、“X”、“推文”、“X 上的帖子”
- “reddit”、“subreddit”、“reddit 讨论”
-“人们在说什么”、“社交媒体”、“情绪”
- “热门帖子”、“病毒式传播”、“热门帖子”
- “用户的帖子”、“时间线”、“最近的活动”

对 Grok (X) 和 Reddit 端点使用“npx agentcash fetch”。所有端点每次调用费用为 0.02 美元。
