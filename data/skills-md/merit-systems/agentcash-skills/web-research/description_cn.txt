使用 x402 保护的 API 进行神经网络搜索和内容提取。比 WebSearch 更适合深入研究，比 WebFetch 更适合被阻止的网站。

用于：
- 深网研究和调查
- 查找与参考 URL 相似的页面
- 从网页中提取干净的文本
- 抓取阻止标准获取器的网站
- 获得事实问题的直接答案
- 需要多个来源的研究

触发因素：
- “研究”、“调查”、“深入研究”、“寻找来源”
- “类似于”、“类似页面”、“更像这样”
- “抓取”、“从中提取内容”、“从中获取文本”
- “被阻止的网站”、“无法访问”、“付费专区”
- “这是什么”，“解释一下”，“回答这个”

对 stableenrich.dev 端点使用“npx agentcash fetch”。更喜欢 Exa 进行语义/神经搜索，更喜欢 Firecrawl 进行直接抓取。
