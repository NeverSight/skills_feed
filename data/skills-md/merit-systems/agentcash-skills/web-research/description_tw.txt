使用 x402 保護的 API 進行神經網路搜尋和內容擷取。比 WebSearch 更適合深入研究，比 WebFetch 更適合被封鎖的網站。

用於：
- 深網研究與調查
- 尋找與參考 URL 相似的頁面
- 從網頁中提取乾淨的文本
- 抓取封鎖標準取得器的網站
- 獲得事實問題的直接答案
- 需要多個來源的研究

觸發因素：
- “研究”、“調查”、“深入研究”、“尋找來源”
- “類似”、“類似頁”、“更像這樣”
- “抓取”、“從中提取內容”、“從中獲取文字”
- “被封鎖的網站”、“無法存取”、“付費專區”
- “這是什麼”，“解釋一下”，“回答這個”

對 stableenrich.dev 端點使用「npx agentcash fetch」。偏好 Exa 進行語義/神經搜索，偏好 Firecrawl 進行直接抓取。
