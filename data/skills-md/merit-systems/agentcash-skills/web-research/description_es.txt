Búsqueda web neuronal y extracción de contenido mediante API protegidas por x402. Mejor que WebSearch para investigaciones profundas y WebFetch para sitios bloqueados.

UTILIZAR PARA:
- Investigación e investigación en la web profunda.
- Encontrar páginas similares a una URL de referencia
- Extracción de texto limpio de páginas web.
- Sitios de scraping que bloquean los buscadores estándar
- Obtener respuestas directas a preguntas objetivas.
- Investigación que requiere múltiples fuentes.

DESENCADENADORES:
- "investigación", "investigación", "inmersión profunda", "encontrar fuentes"
- "similar a", "páginas como", "más como esta"
- "raspar", "extraer contenido de", "obtener el texto de"
- "sitio bloqueado", "no puedo acceder", "muro de pago"
- "qué es", "explica", "responde esto"

Utilice `npx agentcash fetch` para los puntos finales de stableenrich.dev. Prefiera Exa para búsqueda semántica/neural, Firecrawl para raspado directo.
