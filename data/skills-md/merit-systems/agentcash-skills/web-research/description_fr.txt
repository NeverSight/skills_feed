Recherche sur le Web neuronal et extraction de contenu à l'aide d'API protégées par x402. Mieux que WebSearch pour une recherche approfondie et WebFetch pour les sites bloqués.

UTILISER POUR :
- Recherche et enquête sur le Web profond
- Trouver des pages similaires à une URL de référence
- Extraire du texte propre à partir de pages Web
- Sites de scraping qui bloquent les récupérateurs standard
- Obtenir des réponses directes à des questions factuelles
- Recherches nécessitant plusieurs sources

DÉCLENCHEURS :
- "rechercher", "enquêter", "approfondir", "trouver des sources"
- "similaire à", "pages comme", "plus comme ça"
- "gratter", "extraire le contenu de", "obtenir le texte de"
- "site bloqué", "accès impossible", "paywall"
- "qu'est-ce que c'est", "expliquez", "répondez à ceci"

Utilisez `npx agentcash fetch` pour les points de terminaison stableenrich.dev. Préférez Exa pour la recherche sémantique/neurale, Firecrawl pour le scraping direct.
