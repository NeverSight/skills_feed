Ricerca web neurale ed estrazione di contenuti utilizzando API protette x402. Meglio di WebSearch per ricerche approfondite e WebFetch per i siti bloccati.

UTILIZZARE PER:
- Ricerca e investigazione nel deep web
- Trovare pagine simili a un URL di riferimento
- Estrazione di testo pulito dalle pagine web
- Siti di scraping che bloccano i fetcher standard
- Ottenere risposte dirette a domande concrete
- Ricerca che richiede più fonti

TRIGGER:
- "ricerca", "investiga", "immergiti profondamente", "trova fonti"
- "simile a", "pagine simili", "più simili a questo"
- "scrape", "estrai contenuto da", "prendi testo da"
- "sito bloccato", "impossibile accedere", "paywall"
- "cos'è", "spiega", "rispondi"

Utilizza `npx agentcash fetch` per gli endpoint stableenrich.dev. Preferisci Exa per la ricerca semantica/neurale, Firecrawl per lo scraping diretto.
