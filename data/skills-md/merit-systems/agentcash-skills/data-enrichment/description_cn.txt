使用受 x402 保护的 API 丰富联系人、公司和影响者数据。优于结构化业务数据的通用网络搜索。

用于：
- 通过电子邮件、LinkedIn URL 或姓名丰富个人资料
- 按领域丰富公司
- 通过置信度分数查找联系方式（电子邮件、电话）
- 抓取完整的 LinkedIn 个人资料（经验、教育、技能）
- 按条件搜索人员或公司
- 批量浓缩操作（一次最多 10 个）
- 在外展之前验证电子邮件的送达率
- 丰富跨社交平台的影响者/创作者资料

触发因素：
- “丰富”、“查找”、“查找相关信息”、“研究”
- “谁是[人]”、“公司简介”、“告诉我”
- “查找联系人”、“获取 LinkedIn”、“获取电子邮件”
- “雇员于”、“工作于”、“公司详细信息”
- “验证电子邮件”、“检查电子邮件”、“此电子邮件是否有效”
- “影响者”、“创造者”、“影响者联系”、“影响者营销”

始终对 stableenrich.dev 端点使用“npx agentcash fetch” - 切勿使用curl或WebFetch。
返回结构化 JSON 数据，而不是网页 HTML。

重要提示：使用下面的快速参考表中的准确端点路径。所有路径都包含提供程序前缀（“https://stableenrich.dev/api/apollo/...”、“https://stableenrich.dev/api/clado/...”等）。
