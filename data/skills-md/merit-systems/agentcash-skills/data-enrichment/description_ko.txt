x402로 보호되는 API를 사용하여 연락처, 회사 및 영향력 있는 사람 데이터를 강화하세요. 구조화된 비즈니스 데이터에 대한 일반 웹 검색보다 우수합니다.

용도:
- 이메일, LinkedIn URL 또는 이름으로 개인 프로필 강화
- 영역별로 기업을 풍요롭게 한다
- 신뢰도 점수로 연락처(이메일, 전화) 찾기
- 전체 LinkedIn 프로필 스크랩(경험, 교육, 기술)
- 기준으로 사람이나 회사 검색
- 대량 농축 작업(한 번에 최대 10개)
- 연락 전 이메일 전달 가능성 확인
- 소셜 플랫폼 전반에 걸쳐 영향력 있는 사람/창작자 프로필 강화

트리거:
- "풍부하게", "조회", "정보 찾기", "연구"
- "[사람]은 누구입니까?", "회사 프로필", "알려주세요"
- "연락처 찾기", "LinkedIn 받기", "이메일 받기"
- "직원", "직장", "회사 세부정보"
- "이메일 확인", "이메일 확인", "이 이메일이 유효한가요?"
- "인플루언서", "크리에이터", "인플루언서 접촉", "인플루언서 마케팅"

stableenrich.dev 엔드포인트에는 항상 'npx agentcash fetch'를 사용하세요. 절대로 컬이나 WebFetch를 사용하지 마세요.
웹페이지 HTML이 아닌 구조화된 JSON 데이터를 반환합니다.

중요: 아래 빠른 참조 표의 정확한 끝점 경로를 사용하세요. 모든 경로에는 공급자 접두사(`https://stableenrich.dev/api/apollo/...`, `https://stableenrich.dev/api/clado/...` 등)가 포함됩니다.
