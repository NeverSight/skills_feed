使用受 x402 保護的 API 豐富聯絡人、公司和影響者資料。優於結構化業務資料的通用網路搜尋。

用於：
- 透過電子郵件、LinkedIn URL 或姓名豐富個人資料
- 按領域豐富公司
- 透過置信度分數尋找聯絡資訊（電子郵件、電話）
- 抓取完整的 LinkedIn 個人資料（經驗、教育、技能）
- 依條件搜尋人員或公司
- 大量濃縮作業（一次最多 10 個）
- 在外展前驗證電子郵件的送達率
- 豐富跨社群平台的影響者/創作者資料

觸發因素：
- “豐富”、“查找”、“查找相關資訊”、“研究”
- “誰是[人]”、“公司簡介”、“告訴我”
- “尋找聯絡人”、“獲取 LinkedIn”、“獲取電子郵件”
- “僱員於”、“工作於”、“公司詳細資料”
- “驗證電子郵件”、“檢查電子郵件”、“此電子郵件是否有效”
- “影響者”、“創造者”、“影響者聯繫”、“影響者行銷”

始終對 stableenrich.dev 端點使用「npx agentcash fetch」 - 切勿使用curl或WebFetch。
返回結構化 JSON 數據，而不是網頁 HTML。

重要提示：使用下面的快速參考表中的準確端點路徑。所有路徑都包含提供程式前綴（「https://stableenrich.dev/api/apollo/...」、「https://stableenrich.dev/api/clado/...」等）。
