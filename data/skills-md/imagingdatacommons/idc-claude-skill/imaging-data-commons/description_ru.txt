Запросите и загрузите общедоступные данные изображений рака из NCI Imaging Data Commons с помощью idc-index. Используйте для доступа к крупномасштабным наборам данных радиологии (КТ, МРТ, ПЭТ) и патологии для обучения или исследований ИИ. Аутентификация не требуется. Запрос по метаданным, визуализация в браузере, проверка лицензий.
