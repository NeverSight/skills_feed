Vérifie une demande d'extraction GitHub pour les commentaires de révision non résolus, les échecs de vérification de statut et les descriptions PR incomplètes. Attend la fin des vérifications en attente, catégorise les problèmes comme étant exploitables ou informatifs, et éventuellement les corrige et les résout. À utiliser lorsque l'utilisateur souhaite vérifier un PR, répondre aux commentaires de révision ou préparer un PR pour la fusion.
