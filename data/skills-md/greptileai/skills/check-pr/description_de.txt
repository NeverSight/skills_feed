Überprüft eine GitHub-Pull-Anfrage auf ungelöste Überprüfungskommentare, fehlgeschlagene Statusprüfungen und unvollständige PR-Beschreibungen. Wartet auf den Abschluss ausstehender Prüfungen, kategorisiert Probleme als umsetzbar oder informativ und behebt sie optional. Verwenden Sie diese Option, wenn der Benutzer eine PR überprüfen, auf Bewertungsfeedback eingehen oder eine PR für die Zusammenführung vorbereiten möchte.
