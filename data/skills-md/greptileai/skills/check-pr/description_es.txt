Comprueba una solicitud de extracción de GitHub en busca de comentarios de revisión no resueltos, comprobaciones de estado fallidas y descripciones de relaciones públicas incompletas. Espera a que se completen las comprobaciones pendientes, clasifica los problemas como procesables o informativos y, opcionalmente, los repara y resuelve. Úselo cuando el usuario quiera verificar un PR, abordar los comentarios de la revisión o preparar un PR para fusionarlo.
