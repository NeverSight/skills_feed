Écrivez et révisez des animations React Native et des graphiques 2D hautes performances à l'aide de React-Native-Reanimated (v4+) et @shopify/react-native-skia (scènes Canvas, effets d'exécution/shaders). À utiliser pour : les interactions gestuelles, les transitions ressort/synchronisation, les animations de mise en page/de montage, les transitions/animations CSS réanimées, les dessins Skia, les uniformes de shader animés, l'interpolation de chemin/vecteur, les panneaux de réglage en mode développement (curseurs) et le diagnostic des erreurs d'animation (blocages de thread JS, rendus excessifs, allocations par image).
