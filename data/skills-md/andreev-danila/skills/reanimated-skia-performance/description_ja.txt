React-native-reanimated (v4+) と @shopify/react-native-skia (キャンバス シーン、ランタイム エフェクト/シェーダー) を使用して、高性能 React Native アニメーションと 2D グラフィックを作成およびレビューします。用途: ジェスチャー駆動のインタラクション、スプリング/タイミング トランジション、レイアウト/マウント アニメーション、再アニメーション化された CSS トランジション/アニメーション、Skia 描画、アニメーション化されたシェーダー ユニフォーム、パス/ベクトル補間、開発モード調整パネル (スライダー)、およびアニメーション ジャンクの診断 (JS スレッドの停止、過剰な再レンダリング、フレームごとの割り当て)。
