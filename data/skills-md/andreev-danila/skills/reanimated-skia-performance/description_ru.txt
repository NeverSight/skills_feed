Создавайте и проверяйте высокопроизводительную анимацию и 2D-графику React Native с помощью React-native-reanimated (v4+) и @shopify/react-native-skia (сцены Canvas, эффекты/шейдеры времени выполнения). Используется для: взаимодействия с помощью жестов, переходов пружины/времени, анимации макета/монтирования, реанимированных переходов/анимаций CSS, рисунков Skia, анимированных форм шейдеров, интерполяции пути/вектора, панелей настройки режима разработки (ползунков) и диагностики ошибок анимации (остановки потоков JS, чрезмерные повторные рендеринги, покадровое распределение).
