À utiliser pour écrire des requêtes BigQuery, optimiser les performances des requêtes, analyser les plans d'exécution ou éviter les pièges SQL courants. Couvre les requêtes paramétrées, les UDF, les scripts, les fonctions de fenêtre (QUALIFY, ROW_NUMBER, RANK, LEAD/LAG), les fonctions JSON, les opérations ARRAY/STRUCT, les fonctionnalités spécifiques à BigQuery (EXCEPT, REPLACE, SAFE_*), les problèmes de réexécution CTE, NOT IN avec NULL, les performances DML, Standard vs Legacy SQL et les meilleures pratiques en matière de performances.
