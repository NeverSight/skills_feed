Analyse de projet itérative en plusieurs phases avec des diagrammes Mermaid. Génère des rapports d'architecture, des rapports de conception, des rapports d'analyse de méthodes. À utiliser pour analyser les bases de code, comprendre la structure du projet, réviser l'architecture, explorer les modèles de conception ou documenter les composants du système. Déclencheurs sur « analyser le projet », « rapport d'architecture », « analyse de conception », « structure du code », « présentation du système ».
