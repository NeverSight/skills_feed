Un conjunto de herramientas para crear artefactos HTML complejos de claude.ai de múltiples componentes utilizando tecnologías web frontend modernas (React, Tailwind CSS, shadcn/ui). Ideal para artefactos sofisticados que requieren administración de estado, enrutamiento o componentes shadcn/ui; no está destinado a artefactos HTML/JSX simples de un solo archivo.
