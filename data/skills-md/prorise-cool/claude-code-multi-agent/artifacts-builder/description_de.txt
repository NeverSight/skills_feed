Ein Toolkit zum Erstellen komplexer mehrkomponentiger claude.ai-HTML-Artefakte unter Verwendung moderner Frontend-Webtechnologien (React, Tailwind CSS, shadcn/ui). Geeignet für anspruchsvolle Artefakte, die Statusverwaltung, Routing oder Shadcn/UI-Komponenten erfordern; Nicht für einfache Einzeldatei-HTML/JSX-Artefakte gedacht.
