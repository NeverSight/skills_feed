Ein Toolkit zum Erstellen komplexer mehrkomponentiger claude.ai-HTML-Artefakte unter Verwendung moderner Frontend-Webtechnologien (React, Tailwind CSS, shadcn/ui). Geeignet für komplexe Artefakte, die Statusverwaltung, Routing oder Shadcn/UI-Komponenten erfordern; Nicht geeignet für einfache Einzeldatei-HTML/JSX-Artefakte.
