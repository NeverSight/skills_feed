Une boîte à outils pour créer des artefacts HTML claude.ai multi-composants complexes à l'aide des technologies Web frontales modernes (React, Tailwind CSS, shadcn/ui). Convient aux artefacts sophistiqués nécessitant des composants de gestion d'état, de routage ou shadcn/ui ; non destiné aux simples artefacts HTML/JSX à fichier unique.
