Готовый навык API Seedance 2.0 — всего один ключ API для создания видео AI. Создает раскадровки, генерирует эталонные изображения с помощью Seedream 4.5, отправляет видеозадания и опрашивает результаты. Поддерживает как MCP, так и автономный режим сценариев Python. Используйте, когда пользователь упоминает просмотр, видео AI, раскадровку или создание видео.
