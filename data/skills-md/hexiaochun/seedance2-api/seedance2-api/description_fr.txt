Compétence API Seedance 2.0 prête à l'emploi : une seule clé API pour générer des vidéos IA. Crée des storyboards, génère des images de référence avec Seedream 4.5, soumet des tâches vidéo et interroge les résultats. Prend en charge le mode script MCP et Python autonome. À utiliser lorsque l'utilisateur mentionne le semis, la vidéo IA, le storyboard ou la génération de vidéo.
