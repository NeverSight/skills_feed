Concevez et créez des interfaces utilisateur d'application macOS natives. Utilisez cette compétence chaque fois que l'utilisateur demande à créer une application de bureau, une application macOS, une interface de style Mac, une interface utilisateur de style Apple, un utilitaire système ou tout ce qui devrait ressembler à une application Mac native. Déclenchez également lorsque les utilisateurs mentionnent « sensation native », « conception d'applications de bureau », « modèles de conception Apple », « disposition de la barre latérale », « feux de circulation » ou souhaitent créer des outils/utilitaires qui semblent appartenir à macOS. Cette compétence couvre la mise en page, la composition, les modèles d'interaction, les animations, le mode clair/sombre et tous les détails subtils qui donnent à une application l'impression de l'avoir conçue par Apple.
