プロフェッショナルな実稼働グレードのソフトウェアを構築するための、決定的な UI デザイン スキル。このスキルは、ユーザーが Web アプリケーション UI (ダッシュボード、SaaS アプリ、ランディング ページ、請求ページ、分析ビュー、モバイル アプリ、または任意の「vibecoded」インターフェイス) を作成、再設計、洗練、または強化したい場合に使用します。カラー理論 (4 レイヤー システム、OKLCH、ダーク モード、HSB パレット)、レイアウト、タイポグラフィ、アイコン、マイクロ インタラクション、コンポーネント デザイン、価格設定ページ、分析、カード、スペース、インタラクティブなフィードバック、遊び心のあるデザイン、プレゼンテーション テクニック、デザイン思考にわたる 25 以上のデザイン法則をカバーします。クリーンで最新の信頼できる変換ソフトウェアを作成します。
