مهارة تصميم واجهة المستخدم النهائية لبناء برامج احترافية على مستوى الإنتاج. استخدم هذه المهارة عندما يريد المستخدم إنشاء أو إعادة تصميم أو صقل أو رفع مستوى أي واجهة مستخدم لتطبيق ويب - لوحات المعلومات، أو تطبيقات SaaS، أو الصفحات المقصودة، أو صفحات الفوترة، أو طرق عرض التحليلات، أو تطبيقات الهاتف المحمول، أو أي واجهة "مشفره حيويًا". يغطي أكثر من 25 قانونًا للتصميم عبر نظرية الألوان (نظام 4 طبقات، OKLCH، الوضع المظلم، لوحات HSB)، التخطيط، الطباعة، الأيقونات، التفاعلات الدقيقة، تصميم المكونات، صفحات التسعير، التحليلات، البطاقات، المسافات، التعليقات التفاعلية، التصميم المرح، تقنيات العرض، والتفكير التصميمي. تنتج برامج نظيفة وحديثة وجديرة بالثقة والتي تقوم بالتحويل.
