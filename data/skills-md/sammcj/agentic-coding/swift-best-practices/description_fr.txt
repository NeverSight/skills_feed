Cette compétence doit être utilisée lors de l'écriture ou de la révision du code Swift pour les projets iOS ou macOS. Appliquez les meilleures pratiques Swift 6+ modernes, les modèles de concurrence, les directives de conception d'API et les stratégies de migration. Couvre l'asynchrone/l'attente, les acteurs, MainActor, Sendable, les lancements tapés et les modifications avec rupture de Swift 6.
