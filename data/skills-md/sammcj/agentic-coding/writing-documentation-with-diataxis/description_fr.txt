Applique le framework Diataxis pour créer ou améliorer la documentation technique. À utiliser lorsqu'on vous demande de rédiger des didacticiels, des guides pratiques, des documents de référence ou des explications de haute qualité, lors de l'examen de la qualité de la documentation ou lorsque vous décidez du type de documentation à créer. Aide à identifier les types de documentation à l’aide des dimensions action/cognition et acquisition/application.
