React Native Testing Library (RNTL) v13 および v14 (`@testing-library/react-native`) を使用してテストを作成します。 React Native コンポーネント テストを作成、レビュー、修正するときに使用します。対象となるのは、レンダリング、スクリーン、クエリ (getBy/getAllBy/queryBy/findBy)、Jest マッチャー、userEvent、fireEvent、waitFor、および非同期パターンです。 v13 (React 18、同期レンダリング) および v14 (React 19+、非同期レンダリング) をサポートします。トリガー: React Native コンポーネントのテスト ファイル、RNTL インポート、「テスト ライブラリ」、「書き込みテスト」、「コンポーネント テスト」、または「RNTL」の言及。
