اكتب الاختبارات باستخدام الإصدارين 13 و14 من مكتبة React Native Testing Library (RNTL) (`@testing-library/react-native`). يُستخدم عند كتابة اختبارات مكونات React Native أو مراجعتها أو إصلاحها. يغطي: العرض، والشاشة، والاستعلامات (getBy/getAllBy/queryBy/findBy)، ومطابقات Jest، وuserEvent، وfireEvent، وwaitFor، وأنماط غير متزامنة. يدعم الإصدار 13 (React 18، عرض المزامنة) والإصدار 14 (React 19+، عرض غير متزامن). المشغلات على: ملفات الاختبار لمكونات React Native، أو عمليات استيراد RNTL، أو الإشارات إلى "مكتبة الاختبار"، أو "اختبارات الكتابة"، أو "اختبارات المكونات"، أو "RNTL".
