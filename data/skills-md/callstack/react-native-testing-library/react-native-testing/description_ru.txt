Пишите тесты, используя React Native Testing Library (RNTL) v13 и v14 (@testing-library/react-native). Используйте при написании, проверке или исправлении тестов компонентов React Native. Охватывает: рендеринг, экран, запросы (getBy/getAllBy/queryBy/findBy), сопоставления Jest, userEvent, fireEvent, waitFor и асинхронные шаблоны. Поддерживает v13 (React 18, синхронный рендеринг) и v14 (React 19+, асинхронный рендеринг). Срабатывает при: тестовых файлах для компонентов React Native, импорте RNTL, упоминаниях «библиотеки тестирования», «тестов записи», «тестов компонентов» или «RNTL».
