Schreiben Sie Tests mit der React Native Testing Library (RNTL) v13 und v14 („@testing-library/react-native“). Zum Schreiben, Überprüfen oder Reparieren von React Native-Komponententests verwenden. Deckt ab: Rendern, Bildschirm, Abfragen (getBy/getAllBy/queryBy/findBy), Jest-Matcher, userEvent, fireEvent, waitFor und asynchrone Muster. Unterstützt v13 (React 18, synchrones Rendern) und v14 (React 19+, asynchrones Rendern). Wird ausgelöst durch: Testdateien für React Native-Komponenten, RNTL-Importe, Erwähnungen von „Testbibliothek“, „Tests schreiben“, „Komponententests“ oder „RNTL“.
