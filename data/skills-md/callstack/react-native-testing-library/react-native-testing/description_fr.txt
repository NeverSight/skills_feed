Écrivez des tests à l'aide de React Native Testing Library (RNTL) v13 et v14 ($`@testing-library/react-native`). À utiliser lors de l'écriture, de la révision ou de la correction des tests de composants React Native. Couvre : rendu, écran, requêtes (getBy/getAllBy/queryBy/findBy), matchers Jest, userEvent, fireEvent, waitFor et modèles asynchrones. Prend en charge la v13 (React 18, rendu synchronisé) et la v14 (React 19+, rendu asynchrone). Déclencheurs sur : les fichiers de tests des composants React Native, les importations RNTL, les mentions de « bibliothèque de tests », « tests d'écriture », « tests de composants » ou « RNTL ».
