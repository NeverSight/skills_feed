ادفع فرع الميزات الحالي وافتح/حدّث طلب سحب مسودة لأفضل الممارسات عبر GitHub CLI (gh)، بما في ذلك عنوان/نص العلاقات العامة عالي الجودة المرتبط بـ PRD مع خطوات الاختبار + التحقق. يُستخدم عند دفع فرع، أو إنشاء/تحديث العلاقات العامة، أو صياغة وصف العلاقات العامة، أو تشغيل إنشاء/تحرير gh pr. المشغلات: open-pr، open pr، Push، PR، سحب الطلب، إنشاء PR، مسودة PR، وصف pr، gh pr create، gh pr تحرير، جاهز للمراجعة.
