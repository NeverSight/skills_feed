Poussez la branche de fonctionnalités actuelle et ouvrez/mettez à jour une demande d'extraction de brouillon de bonnes pratiques via GitHub CLI (gh), y compris un titre/corps de PR de haute qualité lié au PRD avec des étapes de test et de vérification. À utiliser lors de la poussée d'une branche, de la création/mise à jour d'un PR, de la rédaction d'une description de PR ou de l'exécution de gh pr create/edit. Déclencheurs : open-pr, open pr, push, PR, pull request, create PR, draft PR, pr description, gh pr create, gh pr edit, prêt pour révision.
