قم بإجراء مراجعة منظمة تركز على الصحة وأفضل ممارسات الأمان والأخطاء وحالات الحافة. قم بمراجعة إما GitHub PR (المفضل، عبر gh) أو فرق محلي مقابل قاعدة عند عدم وجود علاقات عامة. تتبع التحديث: قم بوضع علامة على PRD كمراجع عند LGTM؛ إذا تم دمج العلاقات العامة، ضع علامة "مدمج"، وأعد تسمية PRD إلى "تم-*"، وقم بتحديث حالة المهام/todo.md إلى ✅. المشغلات: المراجعة، المراجعة الذاتية، مراجعة التعليمات البرمجية، مراجعة الأمان، حالات الحافة، الدمج المسبق، gh pr diff، gh pr checks.
