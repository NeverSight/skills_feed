Запускается, когда: (1) используется @rive-app/react-canvas или @rive-app/react-webgl,
(2) Используя хук useRive или RiveComponent,
(3) Создание компонентов React с анимацией Rive,
(4) Пользователь упоминает «Rive React» или «useRive»,
(5) Реализация анимации прокрутки или параллакса с помощью Rive.

Лучшие практики по интеграции анимации Rive в приложения React
используя официальный пакет Rive React. Охватывает крючки, макет и анимацию прокрутки.
