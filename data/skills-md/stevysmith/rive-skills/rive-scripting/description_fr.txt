Déclenchez lorsque : (1) Vous travaillez avec des scripts Rive ou du code Luau,
(2) Le code contient les protocoles Node, Layout, Converter ou PathEffect,
(3) L'utilisateur mentionne "Rive scripting" ou "Rive Luau",
(4) Dessiner avec les API Path, Paint, Renderer,
(5) Les fichiers ont l'extension .lua dans un contexte de projet Rive.

Meilleures pratiques pour Rive Scripting - Scripts basés sur Luau pour les procédures
graphiques, mises en page personnalisées, convertisseurs de données et effets de chemin qui s'exécutent
dans les animations Rive.
