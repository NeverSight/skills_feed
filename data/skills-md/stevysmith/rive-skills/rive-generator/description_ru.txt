Срабатывает, когда: (1) Пользователь хочет создать анимацию Rive программным способом;
(2) Пользователь упоминает «создать файл .riv» или «создать анимацию Rive»,
(3) Пользователь хочет увидеть визуальную анимацию, созданную из кода.
(4) Работа с проектом Rive-генератора.

Создавайте файлы .riv программно с помощью TypeScript. Создает статические фигуры
и анимации. Требуется пакет npm @stevysmith/rive-generator.
