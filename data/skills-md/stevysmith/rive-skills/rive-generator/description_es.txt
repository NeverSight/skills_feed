Se activa cuando: (1) El usuario desea crear una animación de Rive mediante programación,
(2) El usuario menciona "generar archivo .riv" o "crear animación Rive",
(3) El usuario quiere ver una animación visual creada a partir de código,
(4) Trabajar con el proyecto del generador de ríos.

Genere archivos .riv mediante programación utilizando TypeScript. Crea formas estáticas
y animaciones. Requiere el paquete npm @stevysmith/rive-generator.
