Si attiva quando: (1) l'utente desidera creare un'animazione Rive a livello di codice,
(2) L'utente menziona "genera file .riv" o "crea animazione Rive",
(3) L'utente desidera vedere un'animazione visiva creata dal codice,
(4) Lavorare al progetto rive-generator.

Genera file .riv a livello di codice utilizzando TypeScript. Crea forme statiche
e animazioni. Richiede il pacchetto npm @stevysmith/rive-generator.
