Wird ausgelöst, wenn: (1) der Benutzer programmgesteuert eine Rive-Animation erstellen möchte,
(2) Der Benutzer erwähnt „.riv-Datei generieren“ oder „Rive-Animation erstellen“,
(3) Der Benutzer möchte eine aus Code erstellte visuelle Animation sehen.
(4) Arbeiten mit dem Rive-Generator-Projekt.

Generieren Sie .riv-Dateien programmgesteuert mit TypeScript. Erstellt statische Formen
und Animationen. Erfordert das NPM-Paket @stevysmith/rive-generator.
