Se activa cuando: (1) Se cargan o incrustan archivos .riv en la web,
(2) Usando paquetes @rive-app/canvas o @rive-app/webgl,
(3) Controlar las máquinas de estado de Rive desde JavaScript,
(4) Leer/escribir entradas de Rive o manejar eventos,
(5) El usuario menciona "tiempo de ejecución de Rive" o "animación de Rive" en el contexto web.

Mejores prácticas para incrustar y controlar animaciones de Rive en la web
utilizando el tiempo de ejecución oficial de Rive JavaScript/TypeScript.
