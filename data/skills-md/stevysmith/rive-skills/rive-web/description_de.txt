Wird ausgelöst, wenn: (1) .riv-Dateien im Web geladen oder eingebettet werden,
(2) Verwendung von @rive-app/canvas- oder @rive-app/webgl-Paketen,
(3) Steuerung von Rive-Zustandsmaschinen über JavaScript,
(4) Rive-Eingaben lesen/schreiben oder Ereignisse verarbeiten,
(5) Der Benutzer erwähnt „Rive Runtime“ oder „Rive Animation“ im Webkontext.
