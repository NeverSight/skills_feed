Создает и поддерживает архитектуру программного обеспечения в виде кода с помощью LikeC4 DSL. Используйте, когда в запросах упоминаются диаграммы архитектуры, представления контекста/контейнера/компонента C4, системные ландшафты, карты зависимостей, карты интеграции или архитектура, созданная из кода. Применяется для создания новых моделей `.c4`/`.likec4`, обновления существующих моделей, проверки с помощью интерфейса командной строки LikeC4 и подготовки результатов предварительного просмотра/сборки/экспорта.
