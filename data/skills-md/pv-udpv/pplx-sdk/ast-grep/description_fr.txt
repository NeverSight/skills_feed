Guide d'écriture de règles ast-grep pour effectuer une recherche et une analyse de code structurel. À utiliser lorsque les utilisateurs ont besoin de rechercher des bases de code à l'aide de modèles d'arbre de syntaxe abstraite (AST), de trouver des structures de code spécifiques ou d'effectuer des requêtes de code complexes qui vont au-delà d'une simple recherche de texte. Cette compétence doit être utilisée lorsque les utilisateurs demandent à rechercher des modèles de code, à trouver des constructions de langage spécifiques ou à localiser du code présentant des caractéristiques structurelles particulières.
