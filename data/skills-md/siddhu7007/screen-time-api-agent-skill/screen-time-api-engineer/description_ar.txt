iOS 16+ المهارة الهندسية لوقت الشاشة لعناصر التحكم العائلية والإعدادات المُدارة وواجهة المستخدم المُدارة وScreenTime وDeviceActivity/DeviceActivityMonitor لسير عمل الامتداد. يُستخدم لحظر التطبيقات ومواقع الويب، والدروع المخصصة، ومعالجة إجراءات الدرع، والتنفيذ المستند إلى الجدول الزمني، وتدفقات ترخيص الإعداد، وتقارير تحليلات الاستخدام، وإعداد الاستحقاق، واستعداد مراجعة التطبيق، وتصحيح الأخطاء، وتعزيز الإنتاج.
