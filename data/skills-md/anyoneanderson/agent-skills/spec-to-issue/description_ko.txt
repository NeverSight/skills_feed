사양 문서에서 GitHub 이슈 생성 — 사양에서 구조화된 기능 이슈를 자동 생성합니다.

.specs/{feature}/에 있는 사양 문서(requirement.md, design.md, task.md)를 분석하고
gh issue create를 통해 구조화된 기능 문제를 생성합니다.
spec-generator로 생성된 사양과 함께 사용하는 것이 가장 좋습니다.

영어 트리거:
- "사양에서 이슈 생성", "사양을 이슈로 등록"
- "사양을 GitHub 문제로 변환", "사양을 문제로 게시"
- 사양 생성기 이후: "이것을 문제로 삼으세요."

일본어트리거:
- 「仕様書をIssueにして」 「Issue에 登録して」「spec으로부터 Issue작성」
- 「仕様書에서 Issue生成」 「specをIssueに変換」
- 사양 생성기完了後に 「이슈에 대해」 「이슈에 대해」
