Interagissez avec l'API Up Banking pour interroger les comptes, les transactions, les catégories et les balises. À utiliser lorsque l'utilisateur souhaite :
- Vérifier les soldes des comptes ou répertorier les comptes
- Rechercher ou filtrer les transactions (par date, catégorie, tag, statut)
- Analyser les modèles de dépenses ou générer des résumés
- Catégoriser les transactions ou gérer les balises
- Configurer ou gérer des webhooks pour les notifications en temps réel
Nécessite la variable d'environnement UP_API_TOKEN.
