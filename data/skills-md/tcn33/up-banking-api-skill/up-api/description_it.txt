Interagisci con l'API Up Banking per interrogare conti, transazioni, categorie e tag. Utilizzare quando l'utente desidera:
- Controllare i saldi dei conti o elencare i conti
- Cerca o filtra le transazioni (per data, categoria, tag, stato)
- Analizzare i modelli di spesa o generare riepiloghi
- Classificare le transazioni o gestire i tag
- Configura o gestisci webhook per notifiche in tempo reale
Richiede la variabile di ambiente UP_API_TOKEN.
