Взаимодействуйте с API Up Banking для запроса счетов, транзакций, категорий и тегов. Используйте, когда пользователь хочет:
- Проверить баланс счетов или составить список счетов
- Поиск или фильтрация транзакций (по дате, категории, тегу, статусу)
- Анализировать структуру расходов или составлять сводки
- Классифицировать транзакции или управлять тегами
- Настройка или управление веб-перехватчиками для уведомлений в реальном времени.
Требуется переменная среды UP_API_TOKEN.
