تفاعل مع Up Banking API للاستعلام عن الحسابات والمعاملات والفئات والعلامات. استخدم عندما يريد المستخدم:
- التحقق من أرصدة الحسابات أو قائمة الحسابات
- بحث أو تصفية المعاملات (حسب التاريخ والفئة والعلامة والحالة)
- تحليل أنماط الإنفاق أو إنشاء ملخصات
- تصنيف المعاملات أو إدارة العلامات
- قم بإعداد أو إدارة خطافات الويب للإشعارات في الوقت الفعلي
يتطلب متغير البيئة UP_API_TOKEN.
