Interagieren Sie mit der Up Banking API, um Konten, Transaktionen, Kategorien und Tags abzufragen. Verwenden Sie es, wenn der Benutzer Folgendes möchte:
- Kontostände prüfen oder Konten auflisten
- Transaktionen suchen oder filtern (nach Datum, Kategorie, Tag, Status)
- Analysieren Sie Ausgabenmuster oder erstellen Sie Zusammenfassungen
- Kategorisieren Sie Transaktionen oder verwalten Sie Tags
- Richten Sie Webhooks für Echtzeitbenachrichtigungen ein oder verwalten Sie sie
Erfordert die Umgebungsvariable UP_API_TOKEN.
