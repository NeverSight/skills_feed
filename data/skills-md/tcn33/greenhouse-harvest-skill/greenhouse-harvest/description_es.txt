Interfaz con la API Greenhouse Harvest v3 para consultar datos de reclutamiento y contratación de forma conversacional. Úselo cuando el usuario necesite recuperar o analizar información de su ATS de Greenhouse, incluidas solicitudes de candidatos, cronogramas de entrevistas, métricas de canalización, ofertas de trabajo, desempeño del reclutador u otros datos de adquisición de talento. Admite consultas como "¿Cuántos candidatos presentaron su solicitud esta semana?", "¿En qué estado se encuentra Jane Smith?", "¿Cuántas entrevistas están reservadas para mañana?", "¿Cuál es la tasa de paso desde la pantalla del teléfono hasta el sitio?" y solicitudes conversacionales similares para análisis de reclutamiento.
