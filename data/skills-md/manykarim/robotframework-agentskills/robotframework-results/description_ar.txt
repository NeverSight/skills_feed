يؤدي تحليل نتائج ملف robot Framework.xml إلى ملخصات JSON وتفاصيل المجموعة/الاختبارات التفصيلية وإحصائيات العلامات والأهمية وأخطاء التنفيذ ورسائل الاختبار الفاشلة والأخطاء على مستوى الكلمة الرئيسية والتوقيت. يُستخدم عندما يُطلب منك قراءة/دمج مخرجات xml، أو حساب عدد النجاح/الفشل، أو إحصائيات العلامات، أو الأهمية، أو رسائل الخطأ، أو الوقت المنقضي، أو أبطأ الاختبارات/الكلمات الرئيسية، أو دمج/دمج مخرجات متعددة عبر إعادة التشغيل.
