Profils comportementaux d'agent qui standardisent le comportement des différents LLM. Chargez cette compétence lorsque vous devez : (1) adopter un mode comportemental spécifique pour une tâche, (2) basculer entre les modes créatif/strict/bavard, (3) garantir un comportement cohérent entre différents modèles. Les profils définissent la personnalité, les heuristiques de décision, le style de communication et les normes de qualité.
