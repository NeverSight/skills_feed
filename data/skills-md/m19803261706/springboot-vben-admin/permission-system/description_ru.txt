Спецификация разработки структуры разрешений для системы управления чрезвычайными ситуациями. Этот навык необходимо использовать при создании новых функциональных модулей, настройке разрешений меню или настройке разрешений данных. При выполнении команд CX создание SQL-кода для миграции разрешений меню является обязательным.
