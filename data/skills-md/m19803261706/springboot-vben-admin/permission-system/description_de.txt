Entwicklungsspezifikation für das Notfallmanagementsystem-Berechtigungsframework. Diese Fähigkeit muss immer dann verwendet werden, wenn neue Funktionsmodule erstellt, Menüberechtigungen konfiguriert oder Datenberechtigungen festgelegt werden. Beim Ausführen von CX-Befehlen ist die Generierung von SQL für die Migration von Menüberechtigungen obligatorisch.
