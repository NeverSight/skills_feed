Mapa WMS Componentes de React (WMS 地圖 React 元件). Úselo al crear editores de mapas de almacenes (倉儲地圖編輯器), planos de planta interactivos, visualización de ubicación o interfaces de gestión espacial. Basado en ReactFlow/XYFlow. Palabras clave: WMS, 倉儲地圖, mapa de almacén, plano de planta, ReactFlow, XYFlow, 地圖編輯, espacial, ubicación
