Компоненты CMS React (CMS React 元件). Используйте при создании интерфейсов администрирования CMS с управлением статьями (文章管理), рабочими процессами утверждения (審核流程), действиями на основе разрешений или интеграцией мезонинного пользовательского интерфейса. Охватывает StandardCMSTable, модальные окна и управление этапами статьи. Ключевые слова: CMS, React, 文章管理, 審核, 表格, таблица, модальный интерфейс, Mezzanine UI, статья, рабочий процесс.
