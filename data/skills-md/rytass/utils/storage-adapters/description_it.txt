Adattatori di archiviazione file (檔案儲存適配器). Da utilizzare quando si lavora con AWS S3, Google Cloud Storage (GCS), Cloudflare R2, BLOB di Azure o archiviazione di file locale (本地儲存). Riguarda il caricamento di file (檔案上傳), il download (下載), gli URL prefirmati (簽署 URL), le operazioni batch (批量操作) e l'integrazione NestJS.
