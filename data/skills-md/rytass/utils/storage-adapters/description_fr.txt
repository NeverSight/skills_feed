Adaptateurs de stockage de fichiers (檔案儲存適配器). À utiliser lorsque vous travaillez avec AWS S3, Google Cloud Storage (GCS), Cloudflare R2, Azure Blob ou le stockage de fichiers local (本地儲存). Couvre le téléchargement de fichiers (檔案上傳), le téléchargement (下載), les URL présignées (URL 簽署), les opérations par lots (批量操作) et l'intégration NestJS.
