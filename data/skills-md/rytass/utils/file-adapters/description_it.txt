Utilità di elaborazione dei file di immagine (圖像檔案處理工具). Da utilizzare quando si lavora con il ridimensionamento delle immagini (圖片縮放), la conversione del formato (格式轉換), le filigrane (浮水印) o le pipeline di elaborazione (處理管道). Copre l'integrazione Sharp, la gestione buffer/stream (緩衝/串流處理), l'elaborazione batch (批次處理) e l'integrazione NestJS (NestJS 整合).
