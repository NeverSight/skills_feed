Utilitaires de traitement de fichiers image (圖像檔案處理工具). À utiliser lorsque vous travaillez avec le redimensionnement d'images (圖片縮放), la conversion de format (格式轉換), les filigranes (浮水印) ou les pipelines de traitement (處理管道). Couvre l'intégration Sharp, la gestion des tampons/flux (緩衝/串流處理), le traitement par lots (批次處理) et l'intégration NestJS (NestJS 整合).
