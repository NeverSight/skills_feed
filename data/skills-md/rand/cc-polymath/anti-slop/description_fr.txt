Boîte à outils complète pour détecter et éliminer les « AI slop » : des modèles génériques et de mauvaise qualité générés par l'IA dans le langage naturel, le code et la conception. À utiliser pour réviser ou améliorer la qualité du contenu, empêcher les modèles d'IA génériques, nettoyer le contenu existant ou appliquer des normes de qualité dans le travail d'écriture, de code ou de conception.
