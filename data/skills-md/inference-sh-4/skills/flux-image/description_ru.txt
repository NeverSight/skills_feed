Создавайте изображения с помощью моделей FLUX (Black Forest Labs) через интерфейс командной строки inference.sh. Модели: FLUX Dev LoRA, FLUX.2 Klein LoRA с индивидуальной адаптацией стиля. Возможности: преобразование текста в изображение, изображение в изображение, точная настройка LoRA, пользовательские стили. Триггеры: Flux, Flux.2, Flux dev, Flux Schnell, Flux Pro, лаборатории Шварцвальда, Flux Image, Flux AI, Flux Model, Flux Lora
