Vollständige Java Spring Boot-Kenntnisse zum Erstellen von Unternehmensanwendungen.
Beinhaltet eine modulare Architektur mit optionalen Komponenten:
- PostgreSQL-Datenbank mit JPA/Hibernate + Flyway-Migration
- Redis-Caching (optional)
- Kafka/RabbitMQ-Messaging (optional, wählen Sie eines aus)
- JWT + OAuth2-Authentifizierung (optional OAuth2)
- RBAC-Autorisierung (optional)
- TDD mit Mockito
- Spec-First-Entwicklung mit OpenSpec
