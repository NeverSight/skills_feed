مجموعة مهارات Java Spring Boot الكاملة لبناء تطبيقات المؤسسات.
يتضمن بنية معيارية مع مكونات اختيارية:
- قاعدة بيانات PostgreSQL مع ترحيل JPA/Hibernate + Flyway
- التخزين المؤقت Redis (اختياري)
- رسائل Kafka/RabbitMQ (اختياري، اختر واحدًا)
- مصادقة JWT + OAuth2 (OAuth2 اختياري)
- ترخيص RBAC (اختياري)
- TDD مع Mockito
- تطوير المواصفات أولاً باستخدام OpenSpec
