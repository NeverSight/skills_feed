Conjunto completo de habilidades de Java Spring Boot para crear aplicaciones empresariales.
Incluye arquitectura modular con componentes opcionales:
- Base de datos PostgreSQL con JPA/Hibernate + migración Flyway
- Almacenamiento en caché de Redis (opcional)
- Mensajería Kafka/RabbitMQ (opcional, elige uno)
- Autenticación JWT + OAuth2 (OAuth2 opcional)
- Autorización RBAC (opcional)
- TDD con Mockito
- Desarrollo Spec-First con OpenSpec
