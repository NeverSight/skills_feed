Полный набор навыков Java Spring Boot для создания корпоративных приложений.
Включает модульную архитектуру с дополнительными компонентами:
- База данных PostgreSQL с миграцией JPA/Hibernate + Flyway
- Кэширование Redis (необязательно)
- Обмен сообщениями Kafka/RabbitMQ (необязательно, выберите один)
- Аутентификация JWT + OAuth2 (необязательно OAuth2)
- Авторизация RBAC (необязательно)
- TDD с Mockito
- Специализированная разработка с OpenSpec
