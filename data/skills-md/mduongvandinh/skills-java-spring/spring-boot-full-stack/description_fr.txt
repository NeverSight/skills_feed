Ensemble complet de compétences Java Spring Boot pour créer des applications d’entreprise.
Inclut une architecture modulaire avec des composants optionnels :
- Base de données PostgreSQL avec migration JPA/Hibernate + Flyway
- Mise en cache Redis (facultatif)
- Messagerie Kafka/RabbitMQ (facultatif, choisissez-en une)
- Authentification JWT + OAuth2 (OAuth2 en option)
- Autorisation RBAC (facultatif)
-TDD avec Mockito
- Développement Spec-First avec OpenSpec
