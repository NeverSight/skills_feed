エンタープライズ アプリケーションを構築するための完全な Java Spring Boot スキル セット。
オプションのコンポーネントを備えたモジュール式アーキテクチャが含まれています。
- JPA/Hibernate + Flyway 移行による PostgreSQL データベース
- Redis キャッシュ (オプション)
- Kafka/RabbitMQ メッセージング (オプション、1 つ選択)
- JWT + OAuth2 認証 (オプションの OAuth2)
- RBAC 認証 (オプション)
- Mockito を使用した TDD
- OpenSpec を使用した仕様優先の開発
