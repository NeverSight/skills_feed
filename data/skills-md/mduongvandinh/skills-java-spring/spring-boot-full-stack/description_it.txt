Set completo di competenze Java Spring Boot per la creazione di applicazioni aziendali.
Include architettura modulare con componenti opzionali:
- Database PostgreSQL con migrazione JPA/Hibernate + Flyway
- Cache Redis (opzionale)
- Messaggistica Kafka/RabbitMQ (opzionale, scegline uno)
- Autenticazione JWT + OAuth2 (OAuth2 opzionale)
- Autorizzazione RBAC (facoltativa)
- TDD con Mockito
- Sviluppo Spec-First con OpenSpec
