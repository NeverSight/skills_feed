Optimisation des coûts AWS et workflows FinOps. À utiliser pour rechercher des ressources inutilisées, analyser les opportunités d'instances réservées, détecter les anomalies de coûts, redimensionner les instances, évaluer les instances Spot, migrer vers des instances de nouvelle génération, mettre en œuvre les meilleures pratiques FinOps, optimiser les coûts de stockage/réseau/base de données et gérer les opérations financières dans le cloud. Comprend des scripts d'analyse automatisés et une documentation de référence complète.
