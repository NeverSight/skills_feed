Проектирование конвейера CI/CD, оптимизация, сканирование безопасности DevSecOps и устранение неполадок. Используйте для создания рабочих процессов, отладки сбоев конвейера, внедрения SAST/DAST/SCA, оптимизации производительности сборки, реализации стратегий кэширования, настройки развертываний, защиты конвейеров с помощью OIDC/управления секретами, а также устранения распространенных проблем в GitHub Actions, GitLab CI и других платформах.
