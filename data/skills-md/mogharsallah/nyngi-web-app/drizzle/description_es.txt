Documentación de Drizzle ORM que cubre consultas, operaciones CRUD, definiciones de esquemas, migraciones, almacenamiento en caché (50 temas), tipos personalizados y conexiones de bases de datos. Incluye integraciones para PostgreSQL (Neon, Vercel, Supabase, AWS Data API, PlanetScale, Prisma), MySQL (AWS Data API, PlanetScale, TiDB) y SQLite (Bun, Cloudflare D1/Durable Objects, Expo, Turso, OP SQLite). Úselo cuando trabaje con consultas de Drizzle, esquemas de bases de datos, migraciones, SQL con seguridad de tipos, patrones ORM o se conecte a bases de datos compatibles.
