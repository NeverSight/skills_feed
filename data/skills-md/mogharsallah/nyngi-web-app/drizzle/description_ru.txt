Документация по ORM Drizzle, охватывающая запросы, операции CRUD, определения схем, миграции, кэширование (50 тем), пользовательские типы и подключения к базе данных. Включает интеграцию с PostgreSQL (Neon, Vercel, Supabase, AWS Data API, PlanetScale, Prisma), MySQL (AWS Data API, PlanetScale, TiDB) и SQLite (Bun, Cloudflare D1/Durable Objects, Expo, Turso, OP SQLite). Используйте при работе с запросами Drizzle, схемами баз данных, миграциями, типобезопасным SQL, шаблонами ORM или подключением к поддерживаемым базам данных.
