وثائق Drizzle ORM التي تغطي الاستعلامات وعمليات CRUD وتعريفات المخطط وعمليات الترحيل والتخزين المؤقت (50 موضوعًا) والأنواع المخصصة واتصالات قاعدة البيانات. يتضمن عمليات تكامل لـ PostgreSQL (Neon وVercel وSupabase وAWS Data API وPlanetScale وPrisma) وMySQL (AWS Data API وPlanetScale وTiDB) وSQLite (Bun وCloudflare D1/Durable Objects وExpo وTurso وOP SQLite). يُستخدم عند العمل مع استعلامات Drizzle أو مخططات قاعدة البيانات أو عمليات الترحيل أو SQL الآمنة للنوع أو أنماط ORM أو الاتصال بقواعد البيانات المدعومة.
