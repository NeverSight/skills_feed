Разрабатывайте высококачественные и доступные компоненты React, используя shadcn-ui, Tailwind CSS и Radix UI. Используйте при создании форм, макетов, диалоговых окон, таблиц или любых компонентов пользовательского интерфейса. Поддерживает Next.js, Vite, Remix, Astro и другие. Интегрируется с сервером shadcn MCP для обнаружения и установки компонентов.
