Guida per la migrazione da Lovable Cloud al tuo progetto Supabase. Da utilizzare quando gli utenti chiedono di esportare dati da Lovable Cloud, disconnettere Lovable Cloud, migrare al proprio Supabase, estrarre dati dal database Lovable gestito, passare da Lovable Cloud a Supabase esterno o limitazioni di Lovable Cloud (nessun accesso SQL, nessuna autenticazione personalizzata, nessun modello email, permanente una volta abilitato, nessun accesso diretto alla dashboard).
