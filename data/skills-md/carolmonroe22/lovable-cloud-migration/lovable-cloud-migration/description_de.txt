Leitfaden für die Migration von Lovable Cloud zu Ihrem eigenen Supabase-Projekt. Verwenden Sie diese Option, wenn Benutzer Fragen zum Exportieren von Daten aus Lovable Cloud, zum Trennen der Lovable Cloud-Verbindung, zur Migration zu ihrer eigenen Supabase, zum Abrufen von Daten aus der verwalteten Lovable-Datenbank, zum Wechsel von Lovable Cloud zu externer Supabase oder zu Lovable Cloud-Einschränkungen (kein SQL-Zugriff, keine benutzerdefinierte Authentifizierung, keine E-Mail-Vorlagen, permanent nach Aktivierung, kein direkter Dashboard-Zugriff) stellen.
