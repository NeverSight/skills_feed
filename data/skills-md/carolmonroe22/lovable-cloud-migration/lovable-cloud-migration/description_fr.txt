Guide pour migrer de Lovable Cloud vers votre propre projet Supabase. À utiliser lorsque les utilisateurs posent des questions sur l'exportation de données depuis Lovable Cloud, la déconnexion de Lovable Cloud, la migration vers leur propre Supabase, l'extraction de données de la base de données Lovable gérée, le passage de Lovable Cloud à une Supabase externe ou les limitations de Lovable Cloud (pas d'accès SQL, pas d'authentification personnalisée, pas de modèles de courrier électronique, permanent une fois activé, pas d'accès direct au tableau de bord).
