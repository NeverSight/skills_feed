Руководство по переходу из Lovable Cloud в ваш собственный проект Supabase. Используйте, когда пользователи спрашивают об экспорте данных из Lovable Cloud, отключении Lovable Cloud, миграции на свою собственную Supabase, получении данных из управляемой базы данных Lovable, переключении с Lovable Cloud на внешнюю Supabase или ограничениях Lovable Cloud (нет доступа к SQL, нет пользовательской аутентификации, нет шаблонов электронной почты, постоянный после включения, нет прямого доступа к панели управления).
