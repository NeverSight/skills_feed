Аудит TypeScript/типизированного кода на наличие проблем с безопасностью типов. Выявляет любые злоупотребления, отсутствующие средства защиты типов и возможности сделать недопустимые состояния непредставимыми. Только для чтения. Триггеры: типы рецензий, безопасность типов, типы проверок, проверка машинописного текста.
