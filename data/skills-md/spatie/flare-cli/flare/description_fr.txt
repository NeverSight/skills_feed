Gérez le suivi des erreurs Flare et la surveillance des performances à l’aide de la CLI Flare. À utiliser lorsque l'utilisateur souhaite répertorier, trier, résoudre, répéter ou déboguer les erreurs ; gérer des projets; créer des projets et récupérer des clés API ; vérifier le nombre d'erreurs ; enquêter sur les itinéraires lents, les requêtes, les tâches ou les commandes ; afficher les données et les traces de performances agrégées ; ou interagissez avec flareapp.io à partir de la ligne de commande.
