Divide-and-Conquer-Implementierung anhand von Spezifikationen/Plänen. Zerlegt ein Referenzdokument in unabhängige Aufgaben, weist jede einem Builder-Agenten zu, führt sie in parallelen Wellen unter Berücksichtigung von Abhängigkeiten aus und integriert dann die Ergebnisse. Verwenden Sie diese Option, wenn Sie eine Spezifikation, ein PRD, einen Plan oder eine große Funktion schnell und mit paralleler Ausführung implementieren möchten.
