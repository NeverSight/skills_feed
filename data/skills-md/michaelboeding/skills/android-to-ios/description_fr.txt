Utilisez le code Android/Kotlin comme source de vérité et implémentez la fonctionnalité équivalente dans iOS/Swift. Comprend le comportement des fonctionnalités, les structures de données et la logique d'Android, puis crée un code iOS idiomatique qui correspond aux modèles existants de la base de code cible. À utiliser lors du portage de fonctionnalités d'Android vers iOS ou pour garantir la cohérence de la plate-forme.
