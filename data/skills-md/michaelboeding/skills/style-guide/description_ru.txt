Анализируйте базу кода, чтобы извлечь из нее соглашения, шаблоны и стиль. Создает специализированные агенты-анализаторы, каждый из которых фокусируется на одном аспекте (структуре, именовании, шаблонах, тестировании, интерфейсе). Создает подробное руководство по стилю, на которое могут ссылаться другие навыки. Используйте при начале работы с незнакомой кодовой базой или для создания явной документации неявных соглашений.
