Analysez une base de code pour en extraire les conventions, les modèles et le style. Génère des agents d'analyse spécialisés qui se concentrent chacun sur un aspect (structure, dénomination, modèles, tests, frontend). Génère un guide de style complet auquel d’autres compétences peuvent faire référence. À utiliser lorsque vous commencez à travailler sur une base de code inconnue ou pour créer une documentation explicite de conventions implicites.
