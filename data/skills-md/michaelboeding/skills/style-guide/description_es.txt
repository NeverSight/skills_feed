Analice una base de código para extraer sus convenciones, patrones y estilo. Genera agentes analizadores especializados, cada uno de los cuales se centra en un aspecto (estructura, denominación, patrones, pruebas, interfaz). Genera una guía de estilo completa a la que otras habilidades pueden hacer referencia. Úselo cuando comience a trabajar en una base de código desconocida o para crear documentación explícita de convenciones implícitas.
