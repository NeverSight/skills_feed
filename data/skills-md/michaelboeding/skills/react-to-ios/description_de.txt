Verwenden Sie React/React Native-Code als Quelle der Wahrheit und implementieren Sie die entsprechende Funktion in iOS/Swift. Versteht das Funktionsverhalten, die Komponenten, die Statusverwaltung und die Logik von React und erstellt dann idiomatischen iOS-Code, der den vorhandenen Mustern der Zielcodebasis entspricht. Wird verwendet, wenn Sie Funktionen von React/React Native auf natives iOS portieren oder native Alternativen zu Webkomponenten erstellen.
