Utilisez le code iOS/Swift comme source de vérité et implémentez la fonctionnalité équivalente dans Android/Kotlin. Comprend le comportement des fonctionnalités, les structures de données et la logique d'iOS, puis crée un code Android idiomatique qui correspond aux modèles existants de la base de code cible. À utiliser lors du portage de fonctionnalités d'iOS vers Android ou pour garantir la cohérence de la plate-forme.
