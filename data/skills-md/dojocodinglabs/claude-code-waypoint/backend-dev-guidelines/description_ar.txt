دليل شامل لتطوير الواجهة الخلفية لوظائف Supabase Edge + PostgreSQL. يُستخدم عند العمل مع Supabase (قاعدة بيانات، أو مصادقة، أو تخزين، أو الوقت الفعلي)، أو Edge Functions، أو PostgreSQL، أو Row-Level Security (RLS)، أو إعادة إرسال البريد الإلكتروني، أو دفعات الشريط، أو أنماط الواجهة الخلفية لـ TypeScript. يغطي تصميم قاعدة البيانات، وتدفقات المصادقة، وأنماط وظائف الحافة، وسياسات RLS، وتكامل البريد الإلكتروني، ومعالجة الدفع، والنشر إلى Supabase.
