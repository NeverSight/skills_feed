Guide complet de développement backend pour Supabase Edge Functions + PostgreSQL. À utiliser lorsque vous travaillez avec Supabase (base de données, authentification, stockage, temps réel), Edge Functions, PostgreSQL, Row-Level Security (RLS), Renvoi d'e-mails, Paiements Stripe ou modèles de backend TypeScript. Couvre la conception de la base de données, les flux d'authentification, les modèles de fonctions Edge, les politiques RLS, l'intégration de la messagerie électronique, le traitement des paiements et le déploiement sur Supabase.
