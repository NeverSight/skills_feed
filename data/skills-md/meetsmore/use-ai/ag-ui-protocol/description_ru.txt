Справочник по протоколу AG-UI (взаимодействие агента и пользователя) для создания интерфейсов агентов AI. Используйте при реализации событий AG-UI (RUN_STARTED, TEXT_MESSAGE_*, Tool_CALL_*, STATE_*), создании агентов, взаимодействующих с внешними интерфейсами, реализации потоковых ответов, управлении состоянием с помощью снимков/дельт, жизненных циклах вызовов инструментов или отладке потоков событий AG-UI.
