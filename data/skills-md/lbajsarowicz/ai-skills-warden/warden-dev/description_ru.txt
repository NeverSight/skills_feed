Управляйте локальными средами разработки на базе Docker с помощью Warden. Используйте при настройке проектов PHP/Node (Magento, Laravel, Symfony, WordPress, Drupal), импорте баз данных, запуске команд оболочки, отладке с помощью Xdebug или управлении службами Docker. Запускается по «надзирателю», «локальной среде», «оболочке docker-compose», «импорту базы данных», «доступу к оболочке».
