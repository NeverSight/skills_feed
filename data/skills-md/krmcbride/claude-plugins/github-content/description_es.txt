Obtenga problemas, relaciones públicas, contenidos del repositorio y código de GitHub utilizando la CLI de gh. Úselo cuando el usuario comparte URL de GitHub (problemas, relaciones públicas, repositorios, archivos fuente) o pregunta sobre el contenido de GitHub. La CLI de gh proporciona contenido completo que la búsqueda web a menudo pasa por alto debido a la representación de JavaScript.
