Récupérez les problèmes GitHub, les PR, le contenu du dépôt et le code à l'aide de la CLI gh. À utiliser lorsque l'utilisateur partage des URL GitHub (problèmes, PR, dépôts, fichiers sources) ou pose des questions sur le contenu GitHub. La CLI gh fournit un contenu complet que la récupération Web manque souvent en raison du rendu JavaScript.
