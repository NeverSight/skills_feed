Извлекайте проблемы GitHub, PR, содержимое репозитория и код с помощью gh CLI. Используйте, когда пользователь делится URL-адресами GitHub (проблемы, PR, репозитории, исходные файлы) или спрашивает о содержимом GitHub. Интерфейс командной строки gh предоставляет полный контент, который часто отсутствует при загрузке из Интернета из-за рендеринга JavaScript.
