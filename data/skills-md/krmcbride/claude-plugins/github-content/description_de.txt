Rufen Sie GitHub-Probleme, PRs, Repo-Inhalte und Code mit der gh-CLI ab. Verwenden Sie diese Option, wenn der Benutzer GitHub-URLs (Issues, PRs, Repos, Quelldateien) teilt oder nach GitHub-Inhalten fragt. Die gh-CLI stellt vollständige Inhalte bereit, die beim Webabruf aufgrund der JavaScript-Wiedergabe häufig fehlen.
