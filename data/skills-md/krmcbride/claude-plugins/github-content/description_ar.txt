قم بإحضار مشكلات GitHub، والعلاقات العامة، ومحتويات الريبو، والتعليمات البرمجية باستخدام gh CLI. يُستخدم عندما يشارك المستخدم عناوين URL لـ GitHub (المشاكل، أو العلاقات العامة، أو مستودعات الشراء، أو الملفات المصدر) أو يسأل عن محتوى GitHub. يوفر gh CLI محتوى كاملًا غالبًا ما يفتقده جلب الويب بسبب عرض JavaScript.
