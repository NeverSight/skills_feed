Организуйте весь рабочий процесс Platonic Coding, от концептуального проектирования до спецификаций RFC, руководств по реализации, реализации кода и проверки соответствия спецификациям. Всегда показывает текущую фазу; использует интерактивный чат на этапе 0, вызывает спецификации platonic-code на этапе 1, platonic-impl-guide на этапе 2, агенты кодирования на этапе 3 и проверку платонического кода на этапе 4.
