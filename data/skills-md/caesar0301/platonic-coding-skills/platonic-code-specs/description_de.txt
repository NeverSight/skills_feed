Verwalten Sie RFC-ähnliche Spezifikationen mit Vorlagen, Validierung und dynamischer Generierung von Verlaufs-, Index- und Benennungsdateien. Wird zum Initialisieren von Spezifikationssystemen, zum Validieren von RFC-Dokumenten, zum Überprüfen der Taxonomiekonformität oder zum Generieren von Spezifikationsindizes und Terminologiereferenzen verwendet.
