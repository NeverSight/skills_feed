Professionelle PDF-Lösung. Erstellen Sie PDFs mit HTML+Paged.js (wissenschaftliche Arbeiten, Berichte, Dokumente). Verarbeiten Sie vorhandene PDFs mit Python (lesen, extrahieren, zusammenführen, teilen, Formulare ausfüllen). Unterstützt KaTeX-Matheformeln, Mermaid-Diagramme, dreizeilige Tabellen, Zitate und andere akademische Elemente. Verwenden Sie diese Fähigkeit auch, wenn der Benutzer explizit LaTeX (.tex) oder eine native LaTeX-Kompilierung anfordert.
