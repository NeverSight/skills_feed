Инструкции по использованию библиотеки ModelMix Node.js для взаимодействия с несколькими поставщиками AI LLM через единый интерфейс. Используйте при интеграции моделей ИИ (OpenAI, Anthropic, Google, Groq, Perplexity, Grok и т. д.), объединении моделей с резервными моделями, получении структурированного JSON из LLM, добавлении инструментов MCP, потоковой передаче ответов или управлении рабочими процессами ИИ с несколькими поставщиками в Node.js.
