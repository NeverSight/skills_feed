Проверка синтаксиса вики-разметки Jira, шаблоны и рекомендации по форматированию. Используйте, когда: (1) написание описаний проблем или комментариев Jira, (2) преобразование Markdown в вики-разметку Jira, (3) запрос отчетов об ошибках или шаблонов запросов функций, (4) проверка синтаксиса Jira перед отправкой, (5) такие ключевые слова, как «формат jira», «вики-разметка», «синтаксис jira», «формат для jira», (6) обеспечение использования контента h2./h3. заголовки вместо Markdown ##, (7) Проверка блоков кода с использованием {code:lang}, а не тройных обратных кавычек, (8) Любая задача, связанная с форматированием текста Jira
