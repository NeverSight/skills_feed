Jira-Wiki-Markup-Syntaxvalidierung, Vorlagen und Formatierungsanleitungen. Wird verwendet, wenn: (1) Beschreibungen oder Kommentare zu Jira-Problemen geschrieben werden, (2) Markdown in Jira-Wiki-Markup konvertiert wird, (3) Vorlagen für Fehlerberichte oder Funktionsanforderungen angefordert werden, (4) Jira-Syntax vor der Übermittlung validiert wird, (5) Schlüsselwörter wie „Jira-Format“, „Wiki-Markup“, „Jira-Syntax“, „Format für Jira“ verwendet werden, (6) Sicherstellen, dass der Inhalt h2./h3 verwendet. Überschriften anstelle von Markdown ##, (7) Beim Überprüfen von Codeblöcken wird {code:lang} verwendet, keine dreifachen Backticks, (8) Jede Aufgabe, die Jira-Textformatierung beinhaltet
