مهارة الوكيل: تكامل Jira الشامل من خلال نصوص Python خفيفة الوزن. يتم التشغيل تلقائيًا عندما يذكر المستخدم عناوين URL لـ Jira مثل "https://jira.*/browse/*" أو "https://*.atlassian.net/browse/*" أو مفاتيح الإصدار مثل "PROJ-123". يُستخدم عند البحث عن المشكلات (JQL)، أو الحصول على/تحديث تفاصيل المشكلة، أو إنشاء المشكلات، أو نقل الحالة، أو إضافة التعليقات، أو تسجيل سجلات العمل، أو إدارة الدورات السريعة واللوحات، أو إنشاء روابط المشكلات، أو تنسيق علامات Jira wiki. إذا فشلت المصادقة، فاعرض تكوين بيانات الاعتماد بشكل تفاعلي. يدعم كلاً من Jira Cloud والخادم/مركز البيانات مع الكشف التلقائي عن المصادقة. بواسطة نتريسرش.
