エージェント スキル: 軽量の Python スクリプトによる包括的な Jira 統合。ユーザーが「https://jira.*/browse/*」、「https://*.atlassian.net/browse/*」などの Jira URL に言及するか、「PROJ-123」などのキーを発行すると、自動的にトリガーされます。課題の検索 (JQL)、課題の詳細の取得/更新、課題の作成、ステータスの移行、コメントの追加、作業ログの記録、スプリントとボードの管理、課題リンクの作成、Jira Wiki マークアップのフォーマットを行うときに使用します。認証が失敗した場合は、対話的に資格情報を構成するよう提案します。自動認証検出により、Jira Cloud とサーバー/データ センターの両方をサポートします。ネットリサーチによる。
