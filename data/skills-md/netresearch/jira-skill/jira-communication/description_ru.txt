Операции API Jira с помощью скриптов Python CLI. АВТОМАТИЧЕСКИ ЗАПУСКАЕТСЯ, когда пользователь упоминает URL-адреса Jira (https://jira.*/browse/*, https://*.atlassian.net/browse/*), выдает ключи (PROJ-123) или спрашивает о проблемах Jira. Используйте, когда Клоду нужно: (1) найти проблемы с помощью запросов JQL, (2) получить или обновить сведения о проблеме, (3) создать новые задачи, (4) перейти к статусу задачи (например, «Сделать» → «Готово»), (5) добавить комментарии, (6) записать рабочее время (рабочие журналы), (7) составить список спринтов и проблем спринта, (8) составить список досок и проблем с досками, (9) создать или составить список ссылок на задачи, (10) обнаружить доступные Jira поля, (11) Получить информацию о профиле пользователя, (12) Загрузить вложения из задач. Если аутентификация не удалась, предложите интерактивную настройку учетных данных через jira-setup.py. Поддерживает как Jira Cloud, так и Server/Data Center с автоматическим обнаружением аутентификации.
