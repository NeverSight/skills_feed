通过 Python CLI 脚本进行 Jira API 操作。当用户使用时自动触发
提及 Jira URL（https://jira.*/browse/*、https://*.atlassian.net/browse/*），
问题密钥 (PROJ-123)，或询问有关 Jira 的问题。当克劳德需要时使用：
(1) 使用 JQL 查询搜索问题，(2) 获取或更新问题详细信息，
(3) 创建新问题，(4) 转换问题状态（例如“待办事项”→“完成”），
(5) 添加评论，(6) 记录工作时间（工作日志），(7) 列出冲刺和冲刺问题，
(8) 列出看板和看板问题，(9) 创建或列出问题链接，
(10) 发现可用的 Jira 字段，(11) 获取用户个人资料信息，
(12)下载问题的附件。
如果身份验证失败，请通过 jira-setup.py 提供交互式凭据设置。
支持 Jira Cloud 和服务器/数据中心，并具有自动身份验证检测功能。
