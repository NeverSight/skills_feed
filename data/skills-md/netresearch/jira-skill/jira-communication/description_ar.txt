عمليات Jira API عبر البرامج النصية Python CLI. يتم التشغيل تلقائيًا عندما يذكر المستخدم عناوين URL لـ Jira (https://jira.*/browse/*، https://*.atlassian.net/browse/*)، أو مفاتيح الإصدار (PROJ-123)، أو يسأل عن مشكلات Jira. يُستخدم عندما يحتاج Claude إلى: (1) البحث عن المشكلات باستخدام استعلامات JQL، (2) الحصول على تفاصيل المشكلة أو تحديثها، (3) إنشاء مشكلات جديدة، (4) حالة مشكلة النقل (على سبيل المثال، "المهام" → "تم")، (5) إضافة تعليقات، (6) تسجيل وقت العمل (سجلات العمل)، (7) سرد الدورات المتكررة ومشكلات الدورات المتكررة، (8) سرد اللوحات ومشكلات اللوحة، (9) إنشاء روابط المشكلات أو إدراجها، (10) اكتشاف حقول Jira المتاحة، (11) الحصول على معلومات ملف تعريف المستخدم، (12) تنزيل المرفقات من المشكلات. إذا فشلت المصادقة، فاعرض إعدادًا تفاعليًا لبيانات الاعتماد عبر jira-setup.py. يدعم كلاً من Jira Cloud والخادم/مركز البيانات مع الكشف التلقائي عن المصادقة.
