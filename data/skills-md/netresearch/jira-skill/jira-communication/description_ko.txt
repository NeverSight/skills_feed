Python CLI 스크립트를 통한 Jira API 작업. 사용자가 자동으로 트리거
Jira URL(https://jira.*/browse/*, https://*.atlassian.net/browse/*)을 언급합니다.
키(PROJ-123)를 발행하거나 Jira 문제에 대해 질문합니다. Claude가 다음을 수행해야 할 때 사용하세요.
(1) JQL 쿼리로 문제 검색, (2) 문제 세부 정보 가져오기 또는 업데이트,
(3) 새로운 이슈 생성, (4) 전환 이슈 상태(예: "To Do" → "Done"),
(5) 댓글 추가, (6) 작업 시간 기록(작업 로그), (7) 스프린트 및 스프린트 문제 나열,
(8) 보드 및 보드 이슈 나열, (9) 이슈 링크 생성 또는 나열,
(10) 사용 가능한 Jira 필드 검색, (11) 사용자 프로필 정보 가져오기,
(12) 이슈에서 첨부 파일을 다운로드합니다.
인증이 실패하면 jira-setup.py를 통해 대화형 자격 증명 설정을 제공하세요.
자동 인증 감지 기능을 통해 Jira Cloud와 서버/데이터 센터를 모두 지원합니다.
