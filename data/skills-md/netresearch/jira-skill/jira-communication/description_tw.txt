通過 Python CLI 腳本進行 Jira API 操作。當用戶使用時自動觸發
提及 Jira URL（https://jira.*/browse/*、https://*.atlassian.net/browse/*），
問題密鑰 (PROJ-123)，或詢問有關 Jira 的問題。當克勞德需要時使用：
(1) 使用 JQL 查詢搜索問題，(2) 獲取或更新問題詳細信息，
(3) 創建新問題，(4) 轉換問題狀態（例如“待辦事項”→“完成”），
(5) 添加評論，(6) 記錄工作時間（工作日誌），(7) 列出衝刺和衝刺問題，
(8) 列出看板和看板問題，(9) 創建或列出問題鏈接，
(10) 發現可用的 Jira 字段，(11) 獲取用戶個人資料信息，
(12)下載問題的附件。
如果身份驗證失敗，請通過 jira-setup.py 提供交互式憑據設置。
支持 Jira Cloud 和服務器/數據中心，並具有自動身份驗證檢測功能。
