Просматривайте запросы на включение путем интеграции операций GitHub с линейным отслеживанием проблем. Используйте, когда: (1) проверяется соответствие изменений PR требованиям к линейной проблеме, (2) проверяется, учитывает ли PR все критерии приемки из линейных проблем, (3) создается комплексный PR-обзор, который связывает изменения кода с требованиями проекта, (4) выявляются пробелы между изменениями PR и требованиями к проблеме или (5) публикуются результаты проверки как в PR-запросах GitHub, так и в линейных задачах. Сочетает операции CLI GitHub с функциями Linear MCP для сквозных рабочих процессов проверки PR.
