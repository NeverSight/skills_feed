Examinez les demandes d'extraction en intégrant les opérations GitHub au suivi des problèmes linéaire. À utiliser lorsque : (1) valider les modifications de PR par rapport aux exigences du problème linéaire, (2) vérifier si le PR répond à tous les critères d'acceptation des problèmes linéaires, (3) créer des revues de PR complètes qui lient les modifications de code aux exigences du projet, (4) identifier les écarts entre les modifications de PR et les exigences du problème, ou (5) publier les résultats de la révision dans les PR GitHub et les problèmes linéaires. Combine les opérations GitHub CLI avec les fonctions Linear MCP pour les flux de travail de révision des relations publiques de bout en bout.
