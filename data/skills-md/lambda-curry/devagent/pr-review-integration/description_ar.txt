قم بمراجعة طلبات السحب من خلال دمج عمليات GitHub مع تتبع المشكلات الخطي. يُستخدم عندما: (1) التحقق من صحة تغييرات العلاقات العامة مقابل متطلبات المشكلات الخطية، (2) التحقق مما إذا كانت العلاقات العامة تتناول جميع معايير القبول من المشكلات الخطية، (3) إنشاء مراجعات علاقات عامة شاملة تربط تغييرات التعليمات البرمجية بمتطلبات المشروع، (4) تحديد الفجوات بين تغييرات العلاقات العامة ومتطلبات المشكلات، أو (5) نشر نتائج المراجعة لكل من العلاقات العامة في GitHub والقضايا الخطية. يجمع بين عمليات GitHub CLI ووظائف Linear MCP لسير عمل مراجعة العلاقات العامة الشاملة.
