Hook git après le paiement pour la configuration automatique de l'arbre de travail. À utiliser lorsque : (1) Configuration de la configuration automatique pour les nouveaux arbres de travail git, (2) Création de hooks post-checkout qui détectent de nouveaux arbres de travail et exécutent des tâches de configuration, (3) Configuration des arbres de travail pour copier automatiquement les fichiers d'environnement et installer les dépendances.
