Utilice las funciones del servidor Linear MCP para interactuar con problemas, proyectos, equipos y comentarios de Linear. Úselo cuando: (1) Recuperar requisitos de problemas y criterios de aceptación de Linear, (2) Validar cambios de relaciones públicas con requisitos de problemas de Linear, (3) Crear o actualizar problemas y comentarios de Linear, (4) Verificar el estado, las etiquetas o las relaciones de los problemas, (5) Enumerar problemas por equipo, asignado o proyecto, o (6) Cualquier operación de gestión de proyectos de Linear. Requiere servidor Linear MCP configurado y autenticado.
