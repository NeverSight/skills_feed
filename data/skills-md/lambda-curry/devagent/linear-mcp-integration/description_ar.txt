استخدم وظائف خادم Linear MCP للتفاعل مع المشكلات والمشاريع والفرق والتعليقات الخطية. يُستخدم عندما: (1) جلب متطلبات المشكلة ومعايير القبول من Linear، (2) التحقق من صحة تغييرات العلاقات العامة مقابل متطلبات المشكلة الخطية، (3) إنشاء المشكلات والتعليقات الخطية أو تحديثها، (4) التحقق من حالة المشكلة أو التصنيفات أو العلاقات، (5) إدراج المشكلات حسب الفريق أو المكلف أو المشروع، أو (6) أي عملية إدارة مشروع خطي. يتطلب تكوين خادم Linear MCP ومصادقته.
