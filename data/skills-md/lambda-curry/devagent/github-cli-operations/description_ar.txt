استخدم GitHub CLI (gh) للتفاعل مع مستودعات GitHub وطلبات السحب والمشكلات وسير العمل. يُستخدم عندما: (1) قراءة تفاصيل العلاقات العامة أو الاختلافات أو بيانات التعريف، (2) إدراج العلاقات العامة والمشكلات أو تصفيتها، (3) إنشاء تعليقات العلاقات العامة أو الارتباط بالمشكلات، (4) التحقق من حالة CI أو إمكانية الدمج، (5) استخراج مراجع المشكلات من أوصاف العلاقات العامة، أو (6) أي عملية مستودع GitHub عبر سطر الأوامر. يتطلب تثبيت GitHub CLI والمصادقة عليه.
