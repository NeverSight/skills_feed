Utilisez GitHub CLI (gh) pour interagir avec les référentiels GitHub, les demandes d'extraction, les problèmes et les flux de travail. À utiliser lorsque : (1) lire les détails, les différences ou les métadonnées des PR, (2) répertorier ou filtrer les PR et les problèmes, (3) créer des commentaires de PR ou des liens vers des problèmes, (4) vérifier l'état ou la fusion des CI, (5) extraire des références de problèmes à partir des descriptions de PR, ou (6) toute opération de référentiel GitHub via la ligne de commande. Nécessite GitHub CLI installé et authentifié.
