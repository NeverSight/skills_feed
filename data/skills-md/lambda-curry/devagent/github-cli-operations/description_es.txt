Utilice GitHub CLI (gh) para interactuar con repositorios, solicitudes de extracción, problemas y flujos de trabajo de GitHub. Úselo cuando: (1) Leer detalles, diferencias o metadatos de PR, (2) Listar o filtrar PR y problemas, (3) Crear comentarios de PR o vincular a problemas, (4) Verificar el estado o la capacidad de combinación de CI, (5) Extraer referencias de problemas de las descripciones de PR, o (6) Cualquier operación del repositorio de GitHub a través de la línea de comando. Requiere GitHub CLI instalado y autenticado.
