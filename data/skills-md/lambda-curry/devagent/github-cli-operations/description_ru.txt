Используйте GitHub CLI (gh) для взаимодействия с репозиториями GitHub, запросами на включение, проблемами и рабочими процессами. Используйте, когда: (1) чтение сведений о PR, различиях или метаданных, (2) составление списка или фильтрация PR и проблем, (3) создание комментариев к PR или ссылок на проблемы, (4) проверка статуса CI или возможности объединения, (5) извлечение ссылок на проблемы из описаний PR или (6) любая операция с репозиторием GitHub через командную строку. Требуется установленный и проверенный GitHub CLI.
