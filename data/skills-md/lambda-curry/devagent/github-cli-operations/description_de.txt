Verwenden Sie GitHub CLI (gh), um mit GitHub-Repositorys, Pull-Anfragen, Problemen und Workflows zu interagieren. Wird verwendet, wenn: (1) PR-Details, Unterschiede oder Metadaten lesen, (2) PRs und Probleme auflisten oder filtern, (3) PR-Kommentare erstellen oder mit Problemen verknüpfen, (4) CI-Status oder Zusammenführungsfähigkeit prüfen, (5) Problemreferenzen aus PR-Beschreibungen extrahieren oder (6) jede GitHub-Repository-Operation über die Befehlszeile. Erfordert die Installation und Authentifizierung der GitHub-CLI.
