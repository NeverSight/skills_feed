Utilizza GitHub CLI (gh) per interagire con repository GitHub, richieste pull, problemi e flussi di lavoro. Da utilizzare quando: (1) leggere dettagli PR, differenze o metadati, (2) elencare o filtrare PR e problemi, (3) creare commenti PR o collegare problemi, (4) controllare lo stato o la fusione dei CI, (5) estrarre riferimenti ai problemi dalle descrizioni PR o (6) qualsiasi operazione del repository GitHub tramite riga di comando. Richiede la CLI GitHub installata e autenticata.
