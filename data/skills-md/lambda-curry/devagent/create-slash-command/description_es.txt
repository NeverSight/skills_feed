Cree nuevos comandos de barra diagonal para los flujos de trabajo de DevAgent. Úselo cuando necesite crear un nuevo archivo de comando en .agents/commands/ y vincularlo simbólicamente a .cursor/commands/ para la integración de Cursor IDE. Esta habilidad maneja el flujo de trabajo completo de creación de comandos, incluida la generación de archivos, la creación de enlaces simbólicos y la validación de estructuras.
