Создайте новые команды с косой чертой для рабочих процессов DevAgent. Используйте, когда вам нужно создать новый командный файл в .agents/commands/ и связать его с .cursor/commands/ для интеграции Cursor IDE. Этот навык управляет полным рабочим процессом создания команд, включая генерацию файлов, создание символических ссылок и проверку структуры.
