JavaScript/TypeScript SDK для inference.sh — запускайте приложения искусственного интеллекта, создавайте агенты, интегрируйте более 150 моделей. Пакет: @inferencesh/sdk (установка npm). Полная поддержка TypeScript, потоковая передача, загрузка файлов. Создавайте агентов с помощью шаблонов или специальных шаблонов, API для создания инструментов, навыков и одобрения людей. Использование для: интеграции JavaScript, TypeScript, Node.js, React, Next.js, интерфейсных приложений. Триггеры: пакет SDK для javascript, пакет SDK для машинописного текста, установка npm, API node.js, клиент JS, React AI, next.js AI, SDK для внешнего интерфейса, @inferencesh/sdk, агент Typescript, SDK для браузера, интеграция с JS.
