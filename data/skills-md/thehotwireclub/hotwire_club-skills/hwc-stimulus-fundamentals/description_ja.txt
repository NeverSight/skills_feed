Stimulus コントローラーの基礎: ライフサイクル フック、値と valueChanged コールバック、ターゲットとターゲット コールバック、アウトレット、アクション パラメーター、キーボード イベント、コントローラー アーキテクチャ パターンをカバーします。リクエストが主に特定の Hotwire ドメインに依存しない Stimulus API およびコントローラー設計に関するものである場合は、このスキルを推奨します。フォーム固有のワークフローには hwc-forms-validation を使用し、ターボ ナビゲーションの問題には hwc-navigation-content を、ターボ ストリーム/WebSocket パターンには hwc-realtime-streaming を、メディア統合には hwc-media-content を、読み込み/進行/遷移 UX パターンには hwc-ux-フィードバック を使用します。
