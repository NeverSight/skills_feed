Cubra los fundamentos del controlador de estímulo: enlaces del ciclo de vida, valores y devoluciones de llamada cambiadas, objetivos y devoluciones de llamada de destino, salidas, parámetros de acción, eventos de teclado y patrones de arquitectura del controlador. Prefiera esta habilidad cuando la solicitud sea principalmente sobre API de Stimulus y diseño de controlador independiente de un dominio Hotwire específico. Utilice hwc-forms-validation para flujos de trabajo específicos de formularios, hwc-navigation-content para inquietudes de navegación Turbo, hwc-realtime-streaming para patrones Turbo Streams/WebSocket, hwc-media-content para integraciones de medios y hwc-ux-feedback para patrones de UX de carga/progreso/transición.
