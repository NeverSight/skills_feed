Реализуйте поведение Hotwire в реальном времени: турбопотоки через WebSocket/SSE, настраиваемые действия потока, встроенные теги потока, обновления списков в реальном времени и синхронизацию состояний между вкладками. Предпочитайте этот навык, когда основной проблемой являются push-обновления или оркестровка действий потока. Используйте hwc-navigation-content для потоков пагинации/вкладки/ленивой навигации на основе извлечения, hwc-forms-validation для жизненного цикла и проверки формы, hwc-media-content для загрузки/воспроизведения мультимедиа, hwc-ux-feedback для общей загрузки/прогресса/переходов и hwc-стимуляции-фундаментальные основы для непотоковых основ Stimulus.
