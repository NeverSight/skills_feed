Cree flujos de navegación Hotwire y descubrimiento de contenido: paginación Turbo Frame, navegación con pestañas, carga diferida, filtrado/búsqueda por facetas, ciclo de vida de caché, restauración de desplazamiento y control de visitas/renderizado. Prefiere esta habilidad cuando el problema principal es el estado de navegación de solicitud/respuesta y el comportamiento del historial del navegador. Utilice hwc-forms-validation para la validación de formularios y flujos de edición en línea, hwc-realtime-streaming para actualizaciones push de WebSocket/Turbo Stream, hwc-media-content para funciones de imagen/video/audio, hwc-ux-feedback para pulido genérico de carga/progreso/transición y hwc-stimulus-fundamentals para API de estímulo no centradas en la navegación.
