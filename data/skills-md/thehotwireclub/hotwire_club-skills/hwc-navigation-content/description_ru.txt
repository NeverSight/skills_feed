Создавайте навигацию Hotwire и потоки обнаружения контента: нумерацию страниц Turbo Frame, навигацию с вкладками, отложенную загрузку, фасетную фильтрацию/поиск, жизненный цикл кэша, восстановление прокрутки и управление посещением/рендерингом. Предпочитайте этот навык, когда основной проблемой является состояние навигации по запросу/ответу и поведение истории браузера. Используйте hwc-forms-validation для проверки формы и потоков встроенного редактирования, hwc-realtime-streaming для push-обновлений WebSocket/Turbo Stream, hwc-media-content для функций изображения/видео/аудио, hwc-ux-feedback для общей загрузки/прогресса/полировки перехода и hwc-стимуляции-основы для API-интерфейсов Stimulus, не ориентированных на навигацию.
