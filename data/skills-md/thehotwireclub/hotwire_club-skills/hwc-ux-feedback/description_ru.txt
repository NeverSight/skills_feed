Внедряйте сквозные шаблоны обратной связи Hotwire UX: состояния загрузки, индикаторы занятости, индикаторы выполнения, оптимистичный пользовательский интерфейс, перехват рендеринга и переходы между представлениями и страницами. Отдавайте предпочтение этому навыку, когда основной целью является воспринимаемая производительность и обратная связь с пользователем, независимо от конкретной области функций. Используйте hwc-forms-validation для корректности формы и поведения проверки, hwc-navigation-content для механики навигации/истории/кэша, hwc-realtime-streaming для оркестровки push/stream, hwc-media-content для поведения, специфичного для медиа, и hwc-стимуляции-фундаментальные принципы для базовых вопросов API Stimulus.
