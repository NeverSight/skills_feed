横断的な Hotwire UX フィードバック パターンを実装します: ロード状態、ビジー インジケーター、プログレス バー、オプティミスティック UI、レンダリング インターセプト、ビュー/ページ遷移。単一の機能ドメインに関係なく、パフォーマンスとユーザーからのフィードバックを認識することが中心的な目標である場合は、このスキルを推奨します。フォームの正確性と検証動作には hwc-forms-validation を、ナビゲーション/履歴/キャッシュの仕組みには hwc-navigation-content を、プッシュ/ストリーム オーケストレーションには hwc-realtime-streaming を、メディア固有の動作には hwc-media-content を、基本的な Stimulus API の質問には hwc-stimulus-fundamentals を使用します。
