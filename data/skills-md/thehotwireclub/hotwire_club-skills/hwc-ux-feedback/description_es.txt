Implemente patrones transversales de retroalimentación de Hotwire UX: estados de carga, indicadores de ocupación, barras de progreso, interfaz de usuario optimista, intercepción de renderizado y transiciones de vista/página. Prefiera esta habilidad cuando el objetivo principal sea el rendimiento percibido y los comentarios de los usuarios, independientemente de un único dominio de funciones. Utilice hwc-forms-validation para corrección de formularios y comportamiento de validación, hwc-navigation-content para mecánicas de navegación/historial/caché, hwc-realtime-streaming para orquestación de transmisión/inserción, hwc-media-content para comportamientos específicos de medios y hwc-stimulus-fundamentals para preguntas básicas de Stimulus API.
