Управляйте рабочими процессами форм Hotwire: жизненный цикл отправки формы, встроенное редактирование, ошибки проверки, ввод/автозаполнение, модальные формы и внешние элементы управления формами. Предпочитайте этот навык, когда основной проблемой является корректность и UX взаимодействия формы. Используйте hwc-navigation-content для навигации по страницам, вкладкам и фильтрам, hwc-realtime-streaming для трансляции через WebSocket/Turbo Stream, hwc-media-content для поведения изображения/видео/аудио, hwc-ux-feedback для общей загрузки/полировки перехода и hwc-стимуляции-основы для API-интерфейсов Stimulus уровня платформы, не привязанных к формам.
