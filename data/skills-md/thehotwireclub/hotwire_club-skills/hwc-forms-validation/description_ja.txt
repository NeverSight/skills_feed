Hotwire フォーム ワークフローの処理: フォーム送信ライフサイクル、インライン編集、検証エラー、先行入力/オートコンプリート、モーダル フォーム、外部フォーム コントロール。中心的な問題がフォーム操作の正確性と UX である場合は、このスキルを優先します。ページネーション/タブ/フィルターのナビゲーションには hwc-navigation-content を使用し、WebSocket/Turbo Stream ブロードキャストには hwc-realtime-streaming を使用し、画像/ビデオ/オーディオの動作には hwc-media-content を使用し、一般的な読み込み/遷移の磨きには hwc-ux-フィードバックを使用し、フォームに関連付けられていないフレームワーク レベルの Stimulus API には hwc-stimulus-fundamentals を使用します。
