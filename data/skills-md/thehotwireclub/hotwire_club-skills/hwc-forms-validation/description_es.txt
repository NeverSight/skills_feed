Maneje los flujos de trabajo de formularios de Hotwire: ciclo de vida de envío de formularios, edición en línea, errores de validación, escritura anticipada/autocompletar, formularios modales y controles de formulario externos. Prefiere esta habilidad cuando el problema central es la corrección y la UX de la interacción de formularios. Utilice hwc-navigation-content para navegación por paginación/pestañas/filtro, hwc-realtime-streaming para transmisión WebSocket/Turbo Stream, hwc-media-content para comportamiento de imagen/video/audio, hwc-ux-feedback para pulido genérico de carga/transición y hwc-stimulus-fundamentals para API de estímulo a nivel de marco no vinculadas a formularios.
