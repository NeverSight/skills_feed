Управление функциями Hotwire, требующими большого количества мультимедиа: загрузка изображений/видео/аудио, предварительный просмотр, элементы управления воспроизведением, отслеживание прогресса и интеграция сторонних мультимедиа (например, WaveSurfer, Swiper, Picture-in-Picture, Blurhash). Предпочитайте этот навык, когда основной проблемой является рендеринг мультимедиа, состояние воспроизведения или интеграция медиа-библиотеки. Используйте hwc-realtime-streaming для обновлений Turbo Stream, отправляемых сервером, hwc-navigation-content для потоков разбивки на страницы/вкладок/ленивой навигации, не связанных с медиа, hwc-forms-validation для проверки формы и поведения встроенного редактирования, hwc-ux-feedback для общих шаблонов загрузки/перехода и hwc-стимуляции-основы для примитивов Stimulus, не специфичных для мультимедиа.
