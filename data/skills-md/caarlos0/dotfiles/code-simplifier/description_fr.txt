Simplifiez et affinez le code pour plus de clarté, de cohérence et de maintenabilité tout en préservant les fonctionnalités. À utiliser lorsqu'on vous demande de « simplifier », « nettoyer » ou « refactoriser » du code, après avoir écrit du code complexe qui pourrait bénéficier d'une simplification, ou lorsque le code est devenu difficile à suivre.
