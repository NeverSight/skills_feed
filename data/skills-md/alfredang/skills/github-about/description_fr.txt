Mise à jour automatique de la description du dépôt GitHub, de l'URL du site en direct et des sujets. Analyse votre base de code pour générer une description convaincante, détecte les URL de déploiement et ajoute des sujets pertinents. À utiliser lors de la configuration d'un dépôt, après le déploiement ou pour peaufiner votre présence sur GitHub. À utiliser lorsque l'utilisateur dit « mettre à jour github about », « mettre à jour le dépôt à propos de », « définir la description du dépôt », « ajouter des sujets », « mettre à jour la section à propos de », « github à propos de » ou toute référence à la mise à jour de la page à propos/description/sujets/d'accueil d'un référentiel GitHub.
