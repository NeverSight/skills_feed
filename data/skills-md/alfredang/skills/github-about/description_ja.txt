GitHub リポジトリの説明、ライブ サイト URL、トピックを自動更新します。コードベースを分析して説得力のある説明を生成し、デプロイ URL を検出して、関連トピックを追加します。リポジトリのセットアップ時、デプロイ後、または GitHub のプレゼンスを高めるために使用します。ユーザーが「github について更新する」、「リポジトリについて更新する」、「リポジトリの説明を設定する」、「トピックを追加する」、「セクションについて更新する」、「github について」、または GitHub リポジトリの about/description/topics/homepage の更新について言及する場合に使用します。
