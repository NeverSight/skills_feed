Автоматическое обновление описания репозитория GitHub, URL-адреса действующего сайта и тем. Анализирует вашу кодовую базу для создания убедительного описания, определяет URL-адреса развертывания и добавляет соответствующие темы. Используйте при настройке репозитория, после развертывания или для улучшения вашего присутствия на GitHub. Используйте, когда пользователь говорит «обновить информацию о GitHub», «обновить репозиторий», «установить описание репозитория», «добавить темы», «обновить раздел о GitHub», «о GitHub» или любую ссылку на обновление репозитория GitHub about/description/topics/homepage.
