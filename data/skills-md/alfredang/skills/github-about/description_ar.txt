التحديث التلقائي لوصف GitHub repo وعنوان URL المباشر للموقع والموضوعات. يحلل قاعدة التعليمات البرمجية الخاصة بك لإنشاء وصف مقنع، ويكتشف عناوين URL للنشر، ويضيف الموضوعات ذات الصلة. استخدمه عند إعداد الريبو، أو بعد النشر، أو لتحسين تواجدك على GitHub. يُستخدم عندما يقول المستخدم "تحديث جيثب حول"، أو "تحديث الريبو حول"، أو "تعيين وصف الريبو"، أو "إضافة موضوعات"، أو "تحديث حول القسم"، أو "جيثب حول"، أو أي إشارة إلى تحديث حول/وصف/موضوعات/الصفحة الرئيسية لمستودع GitHub.
