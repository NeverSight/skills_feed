Развертывание на страницах GitHub с помощью автоматически создаваемого рабочего процесса действий. Определяет тип проекта, создает рабочий процесс развертывания, автоматически включает страницы и действия GitHub через API. Используйте при развертывании статических сайтов, приложений React, Vite или Next.js на страницах GitHub.
