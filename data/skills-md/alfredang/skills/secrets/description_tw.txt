跨所有平台實施安全機密管理。切勿在原始碼中對 OAuth2 機密、API 金鑰、令牌、密碼或憑證進行硬編碼。將所有機密儲存在 .env 檔案中，從環境變數加載，並確保 .env 被 gitignored。在以下情況下使用此技能：(1) 編寫使用 API 金鑰、OAuth2 用戶端機密、令牌或憑證的任何程式碼，(2) 設定身份驗證或第三方集成，(3) 建立需要環境配置的新項目，(4) 檢查與機密相關的程式碼是否有安全問題，(5) 使用機密配置 CI/CD 管道或 Docker 部署。觸發器：API 金鑰、OAuth、客戶端金鑰、令牌、憑證、.env、環境變數、金鑰、密碼、驗證設定、第三方整合。
