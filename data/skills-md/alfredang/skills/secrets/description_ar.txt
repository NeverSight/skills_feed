فرض إدارة الأسرار الآمنة عبر جميع المنصات. لا تقم مطلقًا بترميز أسرار OAuth2 أو مفاتيح API أو الرموز المميزة أو كلمات المرور أو بيانات الاعتماد في التعليمات البرمجية المصدر. قم بتخزين جميع الأسرار في ملفات ‎.env، وتحميلها من متغيرات البيئة، وتأكد من تجاهل ‎.env. استخدم هذه المهارة عندما: (1) كتابة أي تعليمات برمجية تستخدم مفاتيح API أو أسرار عميل OAuth2 أو الرموز المميزة أو بيانات الاعتماد، (2) إعداد المصادقة أو عمليات تكامل الجهات الخارجية، (3) إنشاء مشاريع جديدة تحتاج إلى تكوين البيئة، (4) مراجعة التعليمات البرمجية لمشكلات الأمان المتعلقة بالأسرار، (5) تكوين خطوط أنابيب CI/CD أو عمليات نشر Docker باستخدام الأسرار. المشغلات: مفتاح API، OAuth، سر العميل، الرمز المميز، بيانات الاعتماد، .env، متغيرات البيئة، السر، كلمة المرور، إعداد المصادقة، تكامل الطرف الثالث.
