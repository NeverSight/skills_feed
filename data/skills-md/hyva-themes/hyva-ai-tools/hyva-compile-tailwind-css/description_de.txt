Kompilieren Sie Tailwind CSS für Hyvä-Themes in Magento 2. Diese Fähigkeit sollte verwendet werden, wenn der Benutzer Stile erstellen, CSS generieren, Tailwind kompilieren, Tailwind ausführen oder Produktions-/Entwicklungs-Stylesheets für ein Hyvä-Theme erstellen möchte. Wird bei Ausdrücken wie „Rückenwind kompilieren“, „Stile erstellen“, „CSS generieren“, „Rückenwind ausführen“, „CSS erstellen“ oder „NPM-Build für Theme“ ausgelöst.
