Примените компоненты на основе шаблонов пользовательского интерфейса Hyva к теме Hyvä. Этот навык следует использовать, когда пользователь хочет добавить, установить или применить компонент пользовательского интерфейса Hyva (например, верхний и нижний колонтитулы, галерею, меню, мини-корзину и т. д.) к своей теме Hyvä. В нем перечислены доступные компоненты, не относящиеся к CMS, и их варианты, отображаются инструкции README компонента и копируются файлы компонентов в каталог темы.
