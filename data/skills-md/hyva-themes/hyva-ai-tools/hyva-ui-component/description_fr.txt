Appliquez des composants basés sur un modèle Hyva UI à un thème Hyvä. Cette compétence doit être utilisée lorsque l'utilisateur souhaite ajouter, installer ou appliquer un composant de l'interface utilisateur Hyva (tel qu'un en-tête, un pied de page, une galerie, un menu, un mini-cart, etc.) à son thème Hyvä. Il répertorie les composants non CMS disponibles et leurs variantes, affiche les instructions README des composants et copie les fichiers des composants dans le répertoire du thème.
