Applica componenti basati su modelli dell'interfaccia utente Hyva a un tema Hyvä. Questa abilità dovrebbe essere utilizzata quando l'utente desidera aggiungere, installare o applicare un componente dell'interfaccia utente Hyva (come intestazione, piè di pagina, galleria, menu, minicart, ecc.) al proprio tema Hyvä. Elenca i componenti non CMS disponibili e le relative varianti, visualizza le istruzioni README dei componenti e copia i file dei componenti nella directory del tema.
