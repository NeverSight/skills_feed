Cree tipos de campos personalizados y controladores de campos para los componentes de Hyvä CMS. Úselo cuando el usuario mencione Hyvä, Hyva o CMS junto con un campo personalizado, entrada personalizada o selector modal (por ejemplo, campo personalizado hyva, campo cms personalizado, selector modal hyva, entrada hyva personalizada, entrada cms personalizada). No lo utilice para trabajos de interfaz de usuario o formularios genéricos fuera de Hyvä CMS.
