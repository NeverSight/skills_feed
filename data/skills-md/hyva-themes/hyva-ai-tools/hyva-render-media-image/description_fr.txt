Générez un code d'image réactif pour les modèles de thème Hyvä à l'aide du modèle de vue Media. Cette compétence doit être utilisée lorsque l'utilisateur souhaite restituer des images dans un modèle Hyvä, créer des éléments d'image réactifs, ajouter des images de héros, des images de produits ou toute image nécessitant des points d'arrêt réactifs. Les expressions de déclenchement incluent « rendu de l'image », « ajouter une image au modèle », « image réactive », « élément d'image », « image de héros », « bannière réactive », « image pour mobile et ordinateur de bureau » ou « image de bannière ».
