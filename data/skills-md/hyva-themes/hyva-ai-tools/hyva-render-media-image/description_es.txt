Genere código de imagen responsivo para plantillas de temas Hyvä utilizando el modelo de vista de medios. Esta habilidad debe usarse cuando el usuario desea renderizar imágenes en una plantilla Hyvä, crear elementos de imagen responsivos, agregar imágenes principales, imágenes de productos o cualquier imagen que necesite puntos de interrupción responsivos. Las frases de activación incluyen "renderizar imagen", "agregar imagen a la plantilla", "imagen adaptable", "elemento de imagen", "imagen principal", "banner adaptable", "imagen para dispositivos móviles y de escritorio" o "imagen de banner".
