Videz tous les composants Hyvä CMS des modules actifs. Cette compétence doit être utilisée lorsque l'utilisateur souhaite répertorier tous les composants CMS, afficher les composants disponibles, déboguer les configurations des composants ou voir la sortie composants.json fusionnée. Les phrases de déclenchement incluent « lister les composants cms », « vider les composants », « afficher tous les composants », « afficher les composants cms », « vidage des composants.json ».
