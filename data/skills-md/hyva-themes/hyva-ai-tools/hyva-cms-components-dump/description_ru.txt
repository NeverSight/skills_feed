Выгрузите все компоненты Hyvä CMS из активных модулей. Этот навык следует использовать, когда пользователь хочет составить список всех компонентов CMS, просмотреть доступные компоненты, отладить конфигурации компонентов или просмотреть объединенные выходные данныеComponents.json. Триггерные фразы включают «перечислить компоненты cms», «компоненты дампа», «показать все компоненты», «просмотреть компоненты cms», «дамп Components.json».
