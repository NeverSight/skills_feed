Créez un nouveau module Magento 2 dans app/code/. Cette compétence doit être utilisée lorsque l'utilisateur souhaite créer un module, échafauder un nouveau module, générer un modèle standard de module ou configurer un module personnalisé. Il gère la génération d'enregistrement.php, composer.json, module.xml avec des dépendances configurables. Les phrases de déclenchement incluent « créer un module », « nouveau module », « module d'échafaudage », « générer un module ».
