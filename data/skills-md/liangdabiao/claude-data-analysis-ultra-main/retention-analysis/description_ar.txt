قم بتحليل الاحتفاظ بالمستخدمين وتغييرهم باستخدام تحليل البقاء والتحليل الجماعي والتعلم الآلي. حساب معدلات الاستبقاء، وبناء منحنيات البقاء، والتنبؤ بمخاطر التوقف عن العمل، وإنشاء استراتيجيات تحسين الاستبقاء. يُستخدم عند العمل مع بيانات اشتراك المستخدم، أو معلومات العضوية، أو عندما يذكر المستخدم الاحتفاظ أو التوقف عن العمل أو تحليل البقاء أو القيمة الدائمة للعميل.
