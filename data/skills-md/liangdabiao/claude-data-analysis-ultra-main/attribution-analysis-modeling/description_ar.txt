قم بإجراء تحليل إسناد اللمس المتعدد باستخدام سلاسل ماركوف وقيم شابلي ونماذج الإسناد المخصصة. استخدمه عندما تحتاج إلى تحليل فعالية قناة التسويق، أو حساب إسناد التحويل، أو تحسين ميزانيات التسويق، أو فهم مسارات رحلة العميل. يدعم تحليل انتقال القناة وحساب عائد الاستثمار ورؤى تحسين التسويق مع دعم اللغة الصينية.
