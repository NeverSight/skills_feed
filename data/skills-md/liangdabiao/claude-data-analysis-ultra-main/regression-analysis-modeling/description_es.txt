Realice análisis de regresión integrales y modelos predictivos utilizando regresión lineal, árboles de decisión y bosques aleatorios. Úselo cuando necesite predecir valores continuos como precios de la vivienda, pronósticos de ventas, predicciones de demanda o cualquier variable objetivo numérica. Incluye ingeniería de funciones automatizada, comparación de modelos y visualización con soporte para idioma chino.
