قم بإجراء تحليل الانحدار الشامل والنمذجة التنبؤية باستخدام الانحدار الخطي وأشجار القرار والغابات العشوائية. يُستخدم عندما تحتاج إلى التنبؤ بالقيم المستمرة مثل أسعار المساكن أو توقعات المبيعات أو توقعات الطلب أو أي متغيرات رقمية مستهدفة. يتضمن هندسة الميزات الآلية ومقارنة النماذج والتصور مع دعم اللغة الصينية.
