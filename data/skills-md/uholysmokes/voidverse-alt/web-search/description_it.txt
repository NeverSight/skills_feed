Implementa funzionalità di ricerca Web utilizzando z-ai-web-dev-sdk. Utilizza questa capacità quando l'utente deve effettuare ricerche sul Web, recuperare informazioni correnti, trovare contenuti pertinenti o creare applicazioni con funzionalità di ricerca Web in tempo reale. Restituisce risultati di ricerca strutturati con URL, snippet e metadati.
