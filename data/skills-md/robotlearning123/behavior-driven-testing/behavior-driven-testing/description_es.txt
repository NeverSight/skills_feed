Metodología de prueba sistemática para una cobertura exhaustiva de sucursales, identificación de casos extremos y prevención de errores de producción. Úselo cuando la revisión de relaciones públicas revele una cobertura de prueba incompleta, cuando las pruebas pasan pero los usuarios informan errores, cuando los cambios de código interrumpen las funciones existentes, cuando se verifican todas las ramas y casos extremos antes de la fusión, cuando se analizan problemas de "funciona en mi máquina", cuando se planifica una estrategia de prueba para nuevas funciones o cuando se depuran pruebas inestables y condiciones de carrera.
