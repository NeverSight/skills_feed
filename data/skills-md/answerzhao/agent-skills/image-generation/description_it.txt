Implementa le funzionalità di generazione di immagini AI utilizzando z-ai-web-dev-sdk. Utilizza questa competenza quando l'utente deve creare immagini da descrizioni di testo, generare contenuti visivi, creare opere d'arte, progettare risorse o creare applicazioni con la creazione di immagini basata sull'intelligenza artificiale. Supporta più dimensioni di immagine e restituisce immagini con codifica base64. Include anche lo strumento CLI per la generazione rapida di immagini.
