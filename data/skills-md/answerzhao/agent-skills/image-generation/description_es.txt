Implemente capacidades de generación de imágenes de IA utilizando z-ai-web-dev-sdk. Utilice esta habilidad cuando el usuario necesite crear imágenes a partir de descripciones de texto, generar contenido visual, crear obras de arte, diseñar activos o crear aplicaciones con creación de imágenes impulsada por IA. Admite múltiples tamaños de imagen y devuelve imágenes codificadas en base64. También incluye la herramienta CLI para la generación rápida de imágenes.
