تنفيذ إمكانات إنشاء صور الذكاء الاصطناعي باستخدام z-ai-web-dev-sdk. استخدم هذه المهارة عندما يحتاج المستخدم إلى إنشاء صور من الأوصاف النصية، أو إنشاء محتوى مرئي، أو إنشاء عمل فني، أو تصميم الأصول، أو إنشاء تطبيقات باستخدام إنشاء الصور المدعومة بالذكاء الاصطناعي. يدعم أحجام الصور المتعددة ويعيد الصور المشفرة باستخدام Base64. يتضمن أيضًا أداة CLI لإنشاء الصور بسرعة.
