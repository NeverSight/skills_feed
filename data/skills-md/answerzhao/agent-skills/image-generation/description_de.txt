Implementieren Sie Funktionen zur KI-Bildgenerierung mit dem z-ai-web-dev-sdk. Nutzen Sie diese Fähigkeit, wenn der Benutzer Bilder aus Textbeschreibungen erstellen, visuelle Inhalte generieren, Grafiken erstellen, Assets entwerfen oder Anwendungen mit KI-gestützter Bilderstellung erstellen muss. Unterstützt mehrere Bildgrößen und gibt Base64-codierte Bilder zurück. Enthält außerdem ein CLI-Tool zur schnellen Bildgenerierung.
