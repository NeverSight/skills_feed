Реализуйте возможности извлечения содержимого веб-страниц с помощью z-ai-web-dev-sdk. Используйте этот навык, когда пользователю необходимо очистить веб-страницы, извлечь содержимое статей, получить метаданные страницы или создать приложения, обрабатывающие веб-контент. Поддерживает автоматическое извлечение контента с получением заголовка, HTML и времени публикации.
