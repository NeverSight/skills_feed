Extrayez des systèmes de conception UI/UX réutilisables à partir des bases de code frontend : jetons de conception, styles globaux, composants, modèles d'interaction et modèles de page. À utiliser lors de l'analyse de n'importe quel référentiel frontend (React/Vue/Angular/Next/Vite/etc.) pour documenter ou migrer l'UI/UX pour une réutilisation dans plusieurs projets. Concentrez-vous uniquement sur l'UI/UX ; ignorez explicitement la logique métier et les workflows de domaine.
