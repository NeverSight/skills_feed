Erstellen Sie Commits gemäß Commit-Konventionen (automatische Kategorisierung und Aufteilung von Änderungen). Wird verwendet, wenn der Benutzer „festschreiben“, „Änderungen festschreiben“, „/festschreiben“ usw. sagt. Folgt herkömmlichen Commits-basierten Konventionen und generiert automatisch den entsprechenden Typ/Bereich/Betreff. Sollen Änderungen in mehrere logische Einheiten aufgeteilt werden, werden diese automatisch auf mehrere Commits aufgeteilt.
