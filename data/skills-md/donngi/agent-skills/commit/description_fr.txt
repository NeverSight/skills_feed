Créez des commits en suivant les conventions de commit (catégorisation et fractionnement automatiques des modifications). Utilisé lorsque l'utilisateur dit « valider », « valider les modifications », « /commit », etc. Suit les conventions basées sur les commits conventionnels, générant automatiquement le type/portée/sujet approprié. Si les modifications doivent être divisées en plusieurs unités logiques, elles sont automatiquement divisées en plusieurs validations.
