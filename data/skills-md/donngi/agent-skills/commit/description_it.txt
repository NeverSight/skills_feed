Crea commit seguendo le convenzioni di commit (categorizzazione e suddivisione automatica delle modifiche). Utilizzato quando l'utente dice "commit", "commit change", "/commit", ecc. Segue le convenzioni convenzionali basate sui commit, generando automaticamente il tipo/ambito/oggetto appropriato. Se le modifiche devono essere suddivise in più unità logiche, le suddivide automaticamente in diversi commit.
