Générez un modèle de référence avec une explication complète propriété par propriété pour une seule ressource Terraform AWS. Lorsque l'utilisateur fournit le nom de la ressource (par exemple, aws_s3_bucket, aws_lambda_function), génèrez un modèle contenant une liste d'attributs exacte basée sur le schéma du fournisseur ainsi que des explications tirées de la documentation officielle d'AWS. Utilisez-le pour les requêtes telles que « Créer un modèle pour aws_xxx » ou « Montrez-moi toutes les propriétés de aws_xxx ».
