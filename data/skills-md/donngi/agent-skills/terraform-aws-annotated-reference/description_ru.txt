Создайте справочный шаблон с полным описанием каждого свойства для одного ресурса Terraform AWS. Когда пользователь указывает имя ресурса (например, aws_s3_bucket, aws_lambda_function), выведите шаблон, содержащий точный список атрибутов на основе схемы поставщика, а также пояснения, взятые из официальной документации AWS. Используйте это для таких запросов, как «Создайте шаблон для aws_xxx» или «Покажите мне все свойства aws_xxx».
