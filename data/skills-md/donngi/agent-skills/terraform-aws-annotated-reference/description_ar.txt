قم بإنشاء قالب مرجعي يتضمن شرحًا كاملاً لكل خاصية على حدة لمورد Terraform AWS واحد. عندما يقدم المستخدم اسم المورد (على سبيل المثال، aws_s3_bucket، aws_lambda_function)، قم بإخراج قالب يحتوي على قائمة سمات دقيقة بناءً على مخطط الموفر بالإضافة إلى التوضيحات المستمدة من وثائق AWS الرسمية. استخدم هذا لطلبات مثل "إنشاء قالب لـ aws_xxx" أو "أرني كل خاصية لـ aws_xxx".
