Genera un modello di riferimento con una spiegazione completa proprietà per proprietà per una singola risorsa AWS Terraform. Quando l'utente fornisce il nome della risorsa (ad esempio, aws_s3_bucket, aws_lambda_function), genera un modello che contiene un elenco esatto di attributi basato sullo schema del provider più spiegazioni tratte dalla documentazione ufficiale di AWS. Utilizzalo per richieste come "Crea un modello per aws_xxx" o "Mostrami tutte le proprietà di aws_xxx".
