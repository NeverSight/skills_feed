汎用X/Twitter調査エージェント。 X を検索して、リアルタイムの視点、開発ディスカッション、製品フィードバック、文化的見解、ニュース速報、専門家の意見を検索します。 Web リサーチ エージェントのように機能しますが、ソースとして X を使用します。次の場合に使用します: (1) ユーザーが「x リサーチ」、「x を検索」、「twitter で検索」、「みんなが何と言っているか」、「twitter が何と言っているか」、「x をチェック」、「x 検索」、「/x-research」と言う場合、(2) ユーザーが最近の X の議論が有用なコンテキストを提供する何かに取り組んでいる場合 (新しいライブラリのリリース、API の変更、製品の発売、文化的イベント、業界のドラマ)、(3) ユーザーが検索したいもの開発者/専門家/コミュニティがトピックについて考えます。ツイートの投稿やアカウント管理には適していません。注: 現在、最近の検索 (過去 7 日間) が使用されています。フルアーカイブ検索は同じ従量課金制の X API プランで利用できますが、このスキルにはまだ実装されていません。
