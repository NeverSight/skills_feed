Agent de recherche X/Twitter à usage général. Recherche dans X des perspectives en temps réel, des discussions de développement, des commentaires sur les produits, des points de vue culturels, des dernières nouvelles et des avis d'experts. Fonctionne comme un agent de recherche Web mais utilise X comme source. À utiliser lorsque : (1) l'utilisateur dit "x recherche", "recherche x", "recherche sur Twitter", "de quoi disent les gens", "que dit Twitter", "vérifie x pour", "x recherche", "/x-research", (2) l'utilisateur travaille sur quelque chose où le discours récent de X fournirait un contexte utile (nouvelles versions de bibliothèque, modifications d'API, lancements de produits, événements culturels, drame de l'industrie), (3) l'utilisateur veut savoir ce que les développeurs/experts/communauté pensent d'un sujet. PAS pour : publier des tweets ou gérer un compte. Remarque : utilise actuellement la recherche récente (7 derniers jours). La recherche dans les archives complètes est disponible sur le même plan API X payant à l'utilisation, mais n'est pas encore implémentée dans cette compétence.
