Улучшение пользовательского интерфейса/UX на основе исследований с использованием Mobbin. Используйте, когда пользователь явно запрашивает исследование шаблонов проектирования («использовать Mobbin», «исследовать шаблоны проектирования», «найти шаблоны UX для X»). Требуется учетная запись Mobbin и автоматизация браузера (MCP Claude-in-Chrome). Ищет в Mobbin соответствующие шаблоны пользовательского интерфейса, извлекает принципы проектирования, генерирует документ спецификации для утверждения, а затем реализует. НЕ для общей работы с пользовательским интерфейсом — только тогда, когда пользователю нужен дизайн, основанный на исследованиях.
