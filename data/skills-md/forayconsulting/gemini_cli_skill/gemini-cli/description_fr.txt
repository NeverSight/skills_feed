Utilisez Gemini CLI de Google comme un puissant outil auxiliaire pour la génération, la révision, l'analyse et la recherche sur le Web de code. À utiliser lorsque les tâches bénéficient d'une seconde perspective d'IA, d'informations Web actuelles via la recherche Google, d'une analyse de l'architecture de la base de code ou de la génération de code parallèle. À utiliser également lorsque l'utilisateur demande explicitement des opérations Gemini.
