Создайте или заново создайте монорепозиторий Bun с полным стеком, используя только официальные генераторы: Bun create elysia@latest для бэкэнда и Bun create vite@latest --templateact-ts для интерфейса SPA, затем примените базовую внутреннюю проводку (Better Auth, Drizzle, MCP, OpenAPI, CORS и настройки безопасности по умолчанию). Используйте при создании или сбросе монорепозитория рабочего пространства Bun до последних версий генератора.
