Создайте функции Nango (TypeScript createAction/createSync/createOnEvent) с помощью Nango Runner SDK. Включает проверки проекта/корня, соглашения о конечных точках, повторы/разбиение на страницы, обнаружение удаления, метаданные и рабочий процесс пробного запуска/тестирования, согласованный с документацией.
