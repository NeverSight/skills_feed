Exécute les opérations de l'API locale AdsPower via la CLI du navigateur Adspower (aucun MCP requis). À utiliser lorsque l'utilisateur demande à créer ou à gérer des navigateurs, des groupes, des proxys AdsPower ou à vérifier l'état ; exécutez les commandes Node CLI qui appellent les mêmes gestionnaires que le serveur MCP.
