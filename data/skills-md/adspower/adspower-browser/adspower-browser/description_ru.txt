Запускает операции AdsPower Local API через CLI браузера AdsPower (MCP не требуется). Используйте, когда пользователь запрашивает создание браузеров, групп, прокси-серверов AdsPower или управление ими или проверку статуса; запускайте команды Node CLI, которые вызывают те же обработчики, что и сервер MCP.
