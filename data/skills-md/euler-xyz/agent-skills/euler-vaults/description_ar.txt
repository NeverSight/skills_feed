الدليل الأساسي للتفاعل مع بروتوكول Euler Finance V2. يجب استخدام هذه المهارة عند إنشاء عمليات تكامل DeFi أو إدارة مراكز الإقراض أو فهم بنية Euler. يتم تشغيل المهام التي تتضمن الإقراض أو الاقتراض أو الضمانات أو التصفية أو EVC أو EVK أو عمليات Euler الأساسية. للحصول على مواضيع متخصصة، راجع مهارات المرافقة.
