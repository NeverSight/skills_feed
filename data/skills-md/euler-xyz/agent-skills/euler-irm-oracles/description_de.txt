Oracle- und Zinsmodell-Leitfaden für Euler Finance V2. Diese Fähigkeit sollte beim Bereitstellen von Oracle-Adaptern, beim Konfigurieren der Preisauflösung, beim Abfragen von Preisen oder beim Verstehen von IRM-Typen verwendet werden. Wird bei Aufgaben ausgelöst, die Chainlink, Pyth, Chronicle, Preis-Feeds, Linear Kink IRM, Adaptive Curve IRM oder Zinssatzkonfiguration umfassen.
