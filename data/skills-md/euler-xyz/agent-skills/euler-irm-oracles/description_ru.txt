Руководство по Oracle и модели процентных ставок для Euler Finance V2. Этот навык следует использовать при развертывании адаптеров Oracle, настройке разрешения цен, запросе цен или понимании типов IRM. Запускается по задачам, включающим Chainlink, Pyth, Chronicle, потоки цен, Linear Kink IRM, Adaptive Curve IRM или конфигурацию процентной ставки.
