دليل Oracle ونموذج سعر الفائدة لـ Euler Finance V2. يجب استخدام هذه المهارة عند نشر محولات Oracle أو تكوين تحليل الأسعار أو الاستعلام عن الأسعار أو فهم أنواع إدارة حقوق المعلومات (IRM). تشغيل المهام التي تتضمن Chainlink، أو Pyth، أو Chronicle، أو خلاصات الأسعار، أو Linear Kink IRM، أو Adaptive Curve IRM، أو تكوين سعر الفائدة.
