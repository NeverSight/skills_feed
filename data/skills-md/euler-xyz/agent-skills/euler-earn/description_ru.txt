Руководство по агрегированию доходности EulerEarn для Euler Finance. Этот навык следует использовать при создании хранилищ агрегирования доходности, управлении стратегиями, настройке ролей или использовании PublicAllocator. Запускается при выполнении задач, связанных с хранилищами EulerEarn, распределением стратегий, оптимизацией доходности или управлением метахранилищами.
