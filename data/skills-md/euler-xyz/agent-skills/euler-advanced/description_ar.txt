دليل الميزات المتقدمة لبروتوكول Euler Finance V2. يجب استخدام هذه المهارة عند تنفيذ خطافات الخزنة، أو القروض السريعة، أو تحويل الديون، أو آليات تدفق الرسوم، أو مكافآت ترخيص EUL. يتم تشغيل المهام التي تتضمن الخطافات أو الأوصياء المؤقتين أو التحكم في الوصول أو القروض السريعة أو pullDebt أو FeeFlowController أو RewardToken.
