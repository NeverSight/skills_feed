Инструменты разработчика и руководство по доступу к данным для Euler Finance V2. Этот навык следует использовать при запросе данных хранилища через контракты Lens, получении исторических данных из подграфов, доступе к интерфейсам контрактов или развертывании хранилищ через Euler Creator. Запускается при выполнении задач, включающих VaultLens, OracleLens, запросы подграфов, ABI или развертывание без кода.
