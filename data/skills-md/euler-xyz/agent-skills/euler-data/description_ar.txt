أدوات المطور ودليل الوصول إلى البيانات لـ Euler Finance V2. يجب استخدام هذه المهارة عند الاستعلام عن بيانات المخزن عبر عقود Lens، أو جلب البيانات التاريخية من الرسوم البيانية الفرعية، أو الوصول إلى واجهات العقود، أو نشر الخزائن عبر Euler Creator. يتم تشغيل المهام التي تتضمن VaultLens، أو OracleLens، أو استعلامات الرسم البياني الفرعي، أو واجهات التطبيقات الثنائية (ABIs)، أو النشر بدون تعليمات برمجية.
