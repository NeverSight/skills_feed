Flusso di lavoro di aggiornamento del set di dati end-to-end con passaggi di creazione PR, istantanea, prato, giardino e grafica. Da utilizzare quando l'utente desidera aggiornare un set di dati, aggiornare i dati, eseguire l'aggiornamento ETL o menziona l'aggiornamento delle versioni del set di dati.
