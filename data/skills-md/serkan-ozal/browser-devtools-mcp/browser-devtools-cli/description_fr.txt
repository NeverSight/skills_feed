Automatise les interactions du navigateur à l'aide de Playwright pour les tests Web, le débogage et l'automatisation. À utiliser lorsque l'utilisateur a besoin de naviguer sur des sites Web, de prendre des captures d'écran, de remplir des formulaires, de cliquer sur des éléments, d'extraire le contenu d'une page (HTML/texte), d'auditer l'accessibilité (arborescence ARIA/AX), de mesurer les performances de Web Vitals, de surveiller les journaux de la console et les requêtes HTTP, de simuler les réponses de l'API, d'exécuter du JavaScript dans le navigateur, d'inspecter les composants React ou de comparer l'interface utilisateur avec les conceptions Figma.
