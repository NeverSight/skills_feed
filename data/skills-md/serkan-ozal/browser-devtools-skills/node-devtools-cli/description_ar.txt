واجهة سطر الأوامر (CLI) لتصحيح أخطاء عمليات الواجهة الخلفية لـ Node.js مع فحص غير محظور. يُستخدم عندما يحتاج المستخدم إلى الاتصال بعمليات Node.js (عن طريق PID أو الاسم أو Docker أو المنفذ)، أو تعيين نقاط التتبع/نقاط التسجيل/نقاط الاستثناء، أو التقاط مكدسات الاستدعاءات والمتغيرات المحلية، أو تشغيل JavaScript في سياق العملية، أو فحص سجلات وحدة التحكم. يتطلب الخفي. الاتصال قبل أوامر التصحيح الأخرى.
