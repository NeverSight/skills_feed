CLI для отладки серверных процессов Node.js с неблокирующей проверкой. Используйте, когда пользователю необходимо подключиться к процессам Node.js (по PID, имени, Docker или порту), установить точки трассировки/журнала/точки исключений, захватить стеки вызовов и локальные переменные, запустить JavaScript в контексте процесса или проверить журналы консоли. Требуется демон; подключайтесь перед другими командами отладки.
