CLI para depurar procesos backend de Node.js con inspección sin bloqueo. Úselo cuando el usuario necesite conectarse a procesos de Node.js (por PID, nombre, Docker o puerto), establecer puntos de seguimiento/puntos de registro/puntos de excepción, capturar pilas de llamadas y variables locales, ejecutar JavaScript en el contexto del proceso o inspeccionar registros de la consola. Requiere demonio; conectarse antes que otros comandos de depuración.
