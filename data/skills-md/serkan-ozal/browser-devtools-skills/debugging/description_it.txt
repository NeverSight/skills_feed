Esegui il debug di applicazioni Web e Node.js utilizzando l'ispezione della console, l'analisi di rete e il debug del codice non bloccante. Da utilizzare quando l'utente segnala bug, desidera eseguire il debug di JavaScript (browser o backend), ispezionare richieste di rete, risolvere problemi di pagina/API, tracciare chiamate di funzioni o monitorare eccezioni.
