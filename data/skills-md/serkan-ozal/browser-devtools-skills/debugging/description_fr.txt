Déboguez les applications Web et Node.js à l'aide de l'inspection de la console, de l'analyse du réseau et du débogage de code non bloquant. À utiliser lorsque l'utilisateur signale des bogues, souhaite déboguer JavaScript (navigateur ou backend), inspecter les requêtes réseau, résoudre les problèmes de page/API, suivre les appels de fonction ou surveiller les exceptions.
