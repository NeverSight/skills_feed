تصحيح أخطاء تطبيقات الويب وNode.js باستخدام فحص وحدة التحكم وتحليل الشبكة وتصحيح أخطاء التعليمات البرمجية غير المحظورة. يُستخدم عندما يقوم المستخدم بالإبلاغ عن الأخطاء، أو يريد تصحيح أخطاء JavaScript (المتصفح أو الواجهة الخلفية)، أو فحص طلبات الشبكة، أو استكشاف مشكلات الصفحة/واجهة برمجة التطبيقات وإصلاحها، أو تتبع استدعاءات الوظائف، أو مراقبة الاستثناءات.
