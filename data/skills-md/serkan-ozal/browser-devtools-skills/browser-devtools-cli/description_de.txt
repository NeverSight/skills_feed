Automatisiert Browserinteraktionen mithilfe von Playwright für Webtests, Debugging und Automatisierung. Wird verwendet, wenn der Benutzer auf Websites navigieren, Screenshots erstellen, Formulare ausfüllen, auf Elemente klicken, Seiteninhalte (HTML/Text) extrahieren, die Zugänglichkeit prüfen (ARIA/AX-Baum), die Leistung von Web Vitals messen, Konsolenprotokolle und HTTP-Anfragen überwachen, API-Antworten simulieren, JavaScript im Browser ausführen, React-Komponenten überprüfen, die Benutzeroberfläche mit Figma-Designs vergleichen oder nicht blockierendes Debugging mit Tracepoints, Logpoints und Ausnahmeüberwachung durchführen muss.
