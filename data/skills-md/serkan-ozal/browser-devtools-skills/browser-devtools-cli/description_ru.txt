Автоматизирует взаимодействие с браузером с помощью Playwright для веб-тестирования, отладки и автоматизации. Используйте, когда пользователю необходимо перемещаться по веб-сайтам, делать снимки экрана, заполнять формы, щелкать элементы, извлекать содержимое страницы (HTML/текст), проверять доступность (дерево ARIA/AX), измерять производительность Web Vitals, отслеживать журналы консоли и HTTP-запросы, имитировать ответы API, выполнять JavaScript в браузере, проверять компоненты React, сравнивать пользовательский интерфейс с дизайном Figma или выполнять неблокирующую отладку с помощью точек трассировки, точек журнала и мониторинга исключений.
