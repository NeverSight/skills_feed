Web テスト、デバッグ、自動化に Playwright を使用してブラウザーの操作を自動化します。ユーザーが Web サイトの移動、スクリーンショットの撮影、フォームの入力、要素のクリック、ページ コンテンツ (HTML/テキスト) の抽出、アクセシビリティの監査 (ARIA/AX ツリー)、Web Vitals のパフォーマンスの測定、コンソール ログと HTTP リクエストの監視、API 応答のモック、ブラウザーでの JavaScript の実行、React コンポーネントの検査、UI と Figma デザインの比較、またはトレースポイント、ログポイント、および例外監視によるノンブロッキング デバッグの実行が必要な場合に使用します。
