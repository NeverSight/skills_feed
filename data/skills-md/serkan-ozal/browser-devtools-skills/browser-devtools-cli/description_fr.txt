Automatise les interactions du navigateur à l'aide de Playwright pour les tests Web, le débogage et l'automatisation. À utiliser lorsque l'utilisateur doit naviguer sur des sites Web, prendre des captures d'écran, remplir des formulaires, cliquer sur des éléments, extraire le contenu d'une page (HTML/texte), auditer l'accessibilité (arborescence ARIA/AX), mesurer les performances de Web Vitals, surveiller les journaux de la console et les requêtes HTTP, simuler les réponses de l'API, exécuter JavaScript dans le navigateur, inspecter les composants React, comparer l'interface utilisateur avec les conceptions Figma ou effectuer un débogage non bloquant avec des points de trace, des points de journalisation et une surveillance des exceptions.
