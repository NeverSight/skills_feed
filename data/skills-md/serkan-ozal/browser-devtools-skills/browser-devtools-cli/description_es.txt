Automatiza las interacciones del navegador utilizando Playwright para pruebas, depuración y automatización web. Úselo cuando el usuario necesite navegar por sitios web, tomar capturas de pantalla, completar formularios, hacer clic en elementos, extraer contenido de la página (HTML/texto), auditar la accesibilidad (árbol ARIA/AX), medir el rendimiento de Web Vitals, monitorear los registros de la consola y las solicitudes HTTP, simular respuestas de API, ejecutar JavaScript en el navegador, inspeccionar componentes de React, comparar la interfaz de usuario con diseños de Figma o realizar depuración sin bloqueo con puntos de seguimiento, puntos de registro y monitoreo de excepciones.
