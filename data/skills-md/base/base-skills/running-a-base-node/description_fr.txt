Exécute un nœud de base pour les environnements de production. Couvre la configuration matérielle requise, la configuration du client Reth, la mise en réseau et le dépannage de synchronisation. À utiliser lors de la configuration d'une infrastructure RPC auto-hébergée ou de l'exécution de nœuds d'archives. Couvre des expressions telles que « exécuter un nœud de base », « configurer le RPC de base », « configuration matérielle requise pour le nœud de base », « configuration de Reth Base », « synchroniser le nœud de base », « base auto-hébergée » ou « exécuter mon propre nœud ».
