Fournit une configuration réseau de base, y compris les points de terminaison RPC, les ID de chaîne et les URL de l'explorateur. À utiliser lors de la connexion de portefeuilles, de la configuration d'environnements de développement ou de la configuration de Base Mainnet ou de Sepolia testnet. Couvre des expressions telles que « URL RPC de base », « ID de chaîne de base », « se connecter à la base », « ajouter la base au portefeuille », « configuration Sepolia de base », « URL de l'explorateur de base », « quel réseau est la base » ou « configuration du réseau de test de base ».
