認証と支払いのために Base Account SDK を統合します。 Base (SIWB) でのサインイン、Base Pay、Paymasters、サブアカウント、支出権限、プロリンク、バッチ トランザクションをカバーします。ウォレット認証、USDC 支払い、スポンサー付きトランザクション、スマート ウォレット機能、定期購読、共有可能な支払いリンク、または Base 上のオンチェーン インタラクションを備えたアプリを構築するときに使用します。 「Base でサインインを追加」、「SIWB ボタン」、「USDC 支払いを受け入れる」、「Base Pay」、「ペイマスター設定」、「ガススポンサーシップ」、「スマートウォレット」、「サブアカウント」、「支出許可」、または「支払いリンク」などのフレーズをカバーします。
