Déploie des contrats intelligents sur Base à l’aide de Foundry. Couvre les commandes de création de forge, la vérification du contrat, la configuration du robinet testnet via CDP et la configuration des clés API BaseScan. À utiliser lors du déploiement de contrats Solidity sur Base Mainnet ou Sepolia testnet. Couvre des expressions telles que « déployer le contrat sur la base », « forger la création sur la base », « vérifier le contrat sur BaseScan », « obtenir testnet ETH », « Base Sepolia faucet », « comment déployer sur la base » ou « publier mon contrat ».
