Интегрирует SDK базовой учетной записи для аутентификации и платежей. Охватывает вход в систему с помощью Base (SIWB), Base Pay, Paymasters, дополнительных учетных записей, разрешений на расходы, Prolink и пакетных транзакций. Используйте при создании приложений с аутентификацией кошелька, платежами в долларах США, спонсируемыми транзакциями, функциями смарт-кошелька, повторяющимися подписками, общими платежными ссылками или любым взаимодействием в цепочке на Base. Охватывает такие фразы, как «добавить вход с помощью Base», «кнопка SIWB», «принимать платежи в долларах США», «базовая оплата», «настройка платежного мастера», «спонсорство газа», «умный кошелек», «дополнительная учетная запись», «разрешения на расходы» или «ссылка для оплаты».
