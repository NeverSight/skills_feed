Intègre le SDK de compte de base pour l'authentification et les paiements. Couvre la connexion avec la base (SIWB), le salaire de base, les payeurs, les sous-comptes, les autorisations de dépenses, les Prolinks et les transactions par lots. À utiliser lors de la création d'applications avec authentification de portefeuille, paiements USDC, transactions sponsorisées, fonctionnalités de portefeuille intelligent, abonnements récurrents, liens de paiement partageables ou toute interaction en chaîne sur Base. Couvre des expressions telles que « ajouter une connexion avec Base », « bouton SIWB », « accepter les paiements USDC », « Base Pay », « configuration du payeur », « parrainage de gaz », « portefeuille intelligent », « sous-compte », « autorisations de dépenses » ou « lien de paiement ».
