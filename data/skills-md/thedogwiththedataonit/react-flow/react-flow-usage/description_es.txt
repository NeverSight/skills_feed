Patrones completos de React Flow (@xyflow/react) y mejores prácticas para crear interfaces de usuario basadas en nodos, editores de flujo de trabajo y diagramas interactivos. Úselo cuando trabaje con React Flow para (1) crear editores de flujo o interfaces basadas en nodos, (2) crear nodos y bordes personalizados, (3) implementar flujos de trabajo de arrastrar y soltar, (4) optimizar el rendimiento para gráficos grandes, (5) administrar el estado del flujo y las interacciones, (6) implementar el diseño o posicionamiento automático, o (7) la integración de TypeScript con React Flow.
