Modelli e best practice completi di React Flow (@xyflow/react) per la creazione di interfacce utente basate su nodi, editor di flussi di lavoro e diagrammi interattivi. Utilizzare quando si lavora con React Flow per (1) creare editor di flusso o interfacce basate su nodi, (2) creare nodi e bordi personalizzati, (3) implementare flussi di lavoro drag-and-drop, (4) ottimizzare le prestazioni per grafici di grandi dimensioni, (5) gestire lo stato del flusso e le interazioni, (6) implementare il layout o il posizionamento automatico o (7) l'integrazione di TypeScript con React Flow.
