Erstellen und verwalten Sie Claude-Code-Fähigkeiten gemäß den Best Practices von Anthropic. Verwenden Sie diese Funktion, wenn Sie neue Skills erstellen, „skill-rules.json“ ändern, Triggermuster verstehen, mit Hooks arbeiten, die Aktivierung von Skills debuggen oder progressive Offenlegung implementieren. Behandelt die Skill-Struktur, YAML-Frontmaterie, Triggertypen (Schlüsselwörter, Absichtsmuster, Dateipfade, Inhaltsmuster), Durchsetzungsebenen (Blockieren, Vorschlagen, Warnen), Hook-Mechanismen (UserPromptSubmit, PreToolUse), Sitzungsverfolgung und die 500-Zeilen-Regel.
