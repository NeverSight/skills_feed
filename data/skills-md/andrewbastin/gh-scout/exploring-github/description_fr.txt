Explorez et comprenez les référentiels GitHub à l'aide de la CLI gh. À utiliser lorsqu'on vous demande de parcourir les dépôts, de lire des fichiers, de rechercher du code, d'analyser des problèmes/PR/commits/releases ou de comprendre les bases de code sur GitHub. Remplace les outils GitHub basés sur MCP par zéro dépendance au-delà de gh.
