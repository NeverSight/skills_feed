Analysieren, planen, überprüfen und optimieren Sie jede Codebasis in vier Modi: Discovery (Architektur und Risiken verstehen), Review (Änderungen validieren und Breaking Changes erkennen), Optimierung (Engpässe und Schwachstellen finden), Implementierungsplanung (Schritt-für-Schritt-Anleitung erstellen). Funktioniert mit React, Django, Rails, Go, Rust und über 30 Frameworks. Verwenden Sie es, wenn Sie die Codebasisstruktur analysieren, die Funktionssicherheit bewerten, Sicherheitsprobleme finden, Implementierungen planen oder Leistungsprobleme entdecken.
