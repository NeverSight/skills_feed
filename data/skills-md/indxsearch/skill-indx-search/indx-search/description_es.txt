Habilidad de integración de Indx Search para agentes de codificación de IA. Úselo al crear funciones de búsqueda con Indx, un motor de búsqueda de alto rendimiento que utiliza reconocimiento de patrones en lugar de tokenizadores o lematizadores. Cubre la integración de C# NuGet (IndxSearchLib) y HTTP API (IndxCloudApi), configuración de campos, consultas, filtros, mejoras, ajuste de cobertura y patrones de UX de búsqueda.
