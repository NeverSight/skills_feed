Навыки интеграции Indx Search для агентов кодирования ИИ. Используйте при создании функций поиска с помощью Indx — высокопроизводительной поисковой системы, использующей распознавание образов вместо токенизаторов или стеммеров. Охватывает интеграцию C# NuGet (IndxSearchLib) и HTTP API (IndxCloudApi), настройку полей, запросы, фильтры, повышения, настройку покрытия и шаблоны пользовательского интерфейса поиска.
