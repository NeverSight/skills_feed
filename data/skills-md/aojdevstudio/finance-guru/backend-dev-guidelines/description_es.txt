Guía completa de desarrollo de backend para microservicios Node.js/Express/TypeScript. Úselo al crear rutas, controladores, servicios, repositorios, middleware o trabajar con API Express, acceso a la base de datos Prisma, seguimiento de errores Sentry, validación Zod, unifiedConfig, inyección de dependencia o patrones asíncronos. Cubre arquitectura en capas (rutas → controladores → servicios → repositorios), patrón BaseController, manejo de errores, monitoreo del rendimiento, estrategias de prueba y migración desde patrones heredados.
