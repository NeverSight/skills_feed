Mémoire persistante axée sur l'agent pour stocker, récupérer et conserver des notes lisibles par l'homme au cours des conversations. L'agent doit capturer de manière proactive les décisions importantes et le travail effectué ; les utilisateurs peuvent également demander à se souvenir, enregistrer, noter, rappeler ou réviser un travail antérieur (par exemple, « se souvenir de ceci », « enregistrer ceci », « noter ceci », « me rappeler… », « qu'avons-nous décidé de… », « vérifier vos notes », « nettoyer les souvenirs »).
