Applique automatiquement les normes de codage Python 3.11+ modernes, la conformité PEP 8 et les meilleures pratiques en matière d'indication de type. Cette compétence doit être utilisée lors de l'écriture, de la révision ou de la refactorisation du code Python pour garantir la cohérence avec PEP 8, les astuces de type appropriées, les docstrings de style Google et les idiomes Python modernes.
