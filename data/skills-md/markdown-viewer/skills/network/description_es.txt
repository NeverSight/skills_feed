Cree diagramas de topología de red utilizando el formato XML de drawio con íconos de dispositivos estándar de la industria. Lo mejor para diagramas LAN/WAN, redes empresariales, infraestructura en la nube y diagramas específicos de proveedores (Cisco, Arista, Fortinet). Construido sobre drawio con plantillas de red especializadas. NO para gráficos de dependencia abstractos (use Graphviz) o diagramas de flujo simples (use sirena).
