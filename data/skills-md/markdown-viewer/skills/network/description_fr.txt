Créez des diagrammes de topologie de réseau à l'aide du format XML drawio avec des icônes de périphériques conformes aux normes de l'industrie. Idéal pour les diagrammes LAN/WAN, les réseaux d'entreprise, l'infrastructure cloud et les diagrammes spécifiques aux fournisseurs (Cisco, Arista, Fortinet). Construit sur drawio avec des pochoirs de réseau spécialisés. PAS pour les graphiques de dépendances abstraits (utilisez graphviz) ou les organigrammes simples (utilisez sirène).
