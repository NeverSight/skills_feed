Cree gráficos complejos dirigidos/no dirigidos con un control de diseño preciso. Ideal para árboles de dependencia, organigramas, topologías de red y relaciones de módulos. Úselo cuando necesite enrutamiento de bordes detallado o diseños jerárquicos con muchos niveles. NO para diagramas de flujo simples (use sirena) o diagramas de datos (use vega).
