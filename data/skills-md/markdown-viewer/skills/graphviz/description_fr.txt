Créez des graphiques complexes orientés/non orientés avec un contrôle de mise en page précis. Idéal pour les arbres de dépendances, les organigrammes, les topologies de réseau et les relations entre modules. À utiliser lorsque vous avez besoin d'un routage de périphérie précis ou de dispositions hiérarchiques à plusieurs niveaux. PAS pour les organigrammes simples (utilisez sirène) ou les graphiques de données (utilisez Vega).
