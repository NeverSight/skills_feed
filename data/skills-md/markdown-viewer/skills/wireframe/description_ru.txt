Создавайте каркасы и макеты пользовательского интерфейса, используя формат XML drawio с библиотеками компонентов пользовательского интерфейса для конкретной платформы. Лучше всего подходит для макетов веб-страниц, макетов мобильных приложений iOS/Android и проектов на основе Bootstrap. Создан на основе drawio с трафаретами, специфичными для макета. НЕ для простых блок-схем (используйте русалку) или визуализации данных (используйте вегу).
