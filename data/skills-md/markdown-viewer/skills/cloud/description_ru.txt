Создавайте диаграммы облачной архитектуры, используя формат XML drawio с официальными значками поставщиков облачных услуг. Лучше всего подходит для диаграмм облачной архитектуры AWS, Azure, GCP, Alibaba с соответствующими значками служб и соглашениями о макете. Создано на основе drawio с трафаретами, специфичными для облаков. НЕ для простых блок-схем (используйте русалку) или топологии сети без облачных сервисов (используйте навыки работы в сети).
