قم بإنشاء مخططات معمارية سحابية باستخدام تنسيق drawio XML مع أيقونات موفر السحابة الرسمية. الأفضل للرسومات التخطيطية لبنية AWS وAzure وGCP وAlibaba Cloud مع أيقونات الخدمة المناسبة واتفاقيات التخطيط. مبني على رسم استنسل خاص بالسحابة. ليس للمخططات الانسيابية البسيطة (استخدم حورية البحر) أو طوبولوجيا الشبكة بدون الخدمات السحابية (استخدم مهارة الشبكة).
