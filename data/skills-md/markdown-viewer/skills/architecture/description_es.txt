Cree diagramas de arquitectura en capas utilizando plantillas HTML/CSS con capas codificadas por colores y diseños de cuadrícula. Lo mejor para visualizar capas del sistema (Usuario → Aplicación → Datos → Infraestructura), arquitectura de microservicios y diseño de aplicaciones empresariales. NO para diagramas personalizados con píxeles perfectos (use drawio), diagramas de flujo simples (use sirena) o visualización de datos (use vega).
