Créez de superbes infographies avec des modèles prédéfinis. Idéal pour les cartes KPI, les calendriers, les feuilles de route, les processus étape par étape, les comparaisons A et B, l'analyse SWOT, les entonnoirs, les arbres organisationnels, les diagrammes circulaires/à barres. À utiliser lorsque vous avez besoin d'un impact visuel rapide avec 4 à 8 éléments. Syntaxe simple de type YAML. PAS pour l'analyse de données complexes (utilisez Vega) ou les organigrammes techniques (utilisez Mermaid).
