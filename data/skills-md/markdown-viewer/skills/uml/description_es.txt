Cree diagramas UML (Lenguaje de modelado unificado) utilizando el formato XML de drawio. Lo mejor para el modelado de software, incluidos diagramas de clase, secuencia, actividad, máquina de estado, componente, caso de uso e implementación. Construido sobre drawio con formas y notación específicas de UML. NO para diagramas de flujo simples (use sirena), arquitectura de sistema en capas (use arquitectura) o diagramas de datos (use vega).
