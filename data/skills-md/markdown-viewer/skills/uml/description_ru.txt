Создавайте диаграммы UML (унифицированный язык моделирования), используя формат XML drawio. Лучше всего подходит для моделирования программного обеспечения, включая диаграммы классов, последовательностей, действий, конечных автоматов, компонентов, вариантов использования и развертывания. Создан на основе drawio с формами и обозначениями, специфичными для UML. НЕ для простых блок-схем (используйте «русалку»), многоуровневой системной архитектуры (используйте архитектуру) или диаграмм данных (используйте вегу).
