Gestisce automaticamente blog, libri e contenuti di pagine per progetti di generatori di siti statici GitSite. Utilizza questa competenza quando gli utenti devono creare post di blog, capitoli di libri, libri o pagine statiche. Crea automaticamente la struttura di directory corretta, genera file modello Markdown con frontmatter completo e fornisce script Python per la gestione dei contenuti. Adatto per progetti di siti Web statici realizzati con la CLI GitSite.
