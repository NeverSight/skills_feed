Supprimez le « slop » généré par l'IA des bases de code en supprimant les commentaires redondants et le code inutilisé, en renforçant le formatage et en normalisant les modèles de style (par exemple, la cohérence de la classe Tailwind, les jetons de couleur de l'interface utilisateur Nuxt). À utiliser lorsqu'un utilisateur demande de nettoyer le code, de supprimer les commentaires inutiles, de réduire les importations/variables inutilisées ou d'aligner les conventions de l'interface utilisateur Tailwind/Nuxt.
