Entfernen Sie KI-generierte „Slop“ aus Codebasen, indem Sie überflüssige Kommentare und ungenutzten Code löschen, die Formatierung verschärfen und Stilmuster normalisieren (z. B. Tailwind-Klassenkonsistenz, Nuxt-UI-Farbtokens). Verwenden Sie diese Option, wenn ein Benutzer den Code bereinigen, unnötige Kommentare entfernen, nicht verwendete Importe/Variablen reduzieren oder Tailwind/Nuxt-Benutzeroberflächenkonventionen angleichen möchte.
