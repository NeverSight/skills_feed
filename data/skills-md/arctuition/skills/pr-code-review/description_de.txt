Führen Sie GitHub-Pull-Request-Codeüberprüfungen mit der gh-CLI durch. Verwenden Sie diese Option, wenn Sie aufgefordert werden, eine PR zu überprüfen, PR-Unterschiede zu überprüfen, Inline-Überprüfungskommentare zu bestimmten Zeilen zu hinterlassen oder eine auf dem Schweregrad basierende Zusammenfassung (hoch/mittel/niedrig) der Ergebnisse zu erstellen.
