يُستخدم عند كتابة الاختبارات الأكثر أهمية أو مراجعتها أو إعادة هيكلتها. قم بالتحميل عندما ترى ملفات *.test.ts أو *.spec.ts، أو كتل وصف متداخلة، أو تأكيدات فضفاضة (toBeTruthy)، أو الإفراط في الاستهزاء، أو الاختبارات البطيئة، أو احتياجات الاختبار غير المتزامنة. يغطي نمط AAA، والاختبارات ذات المعلمات، واختبار التسلسل الهرمي المزدوج، والتنظيف الوهمي، وعزل الاختبار، وتحسين الأداء. تشمل الكلمات الرئيسية vitest، وtesting، وTDD، والتأكيدات، وmocks، وStubs، وFakes، وSpies، وBeforeEach، وdescription، وIt، وتوقع، وvi.mock، وit.each.
