Оптимизация производительности React и лучшие практики. Используйте при написании компонентов React, перехватчиков или JSX; рефакторинг кода React; оптимизация повторного рендеринга, мемоизации или управления состоянием; проверка кода React на наличие проблем с производительностью; исправление несоответствий гидратации; или реализация переходов, ленивой инициализации или зависимостей эффектов. Охватывает функции React 19+, включая useEffectEvent, компонент Activity и реквизиты ref.
