Réagissez à l’optimisation des performances et aux meilleures pratiques. À utiliser lors de l'écriture de composants React, de hooks ou de JSX ; refactorisation du code React ; optimiser les rendus, la mémorisation ou la gestion de l'état ; examiner le code React pour les problèmes de performances ; corriger les déséquilibres d’hydratation ; ou implémenter des transitions, une initialisation paresseuse ou des dépendances d'effets. Couvre les fonctionnalités de React 19+, notamment useEffectEvent, le composant Activity et les accessoires de référence.
