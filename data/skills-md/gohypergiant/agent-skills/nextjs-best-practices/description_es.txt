Optimización del rendimiento y mejores prácticas de Next.js. Úselo al escribir código Next.js (App Router o Pages Router); implementar componentes de servidor, acciones de servidor o rutas API; optimizar la serialización de RSC, la obtención de datos o la representación del lado del servidor; revisar el código Next.js para detectar problemas de rendimiento; arreglar la autenticación en Acciones del Servidor; o implementar límites de suspensión, obtención de datos en paralelo o solicitar deduplicación.
