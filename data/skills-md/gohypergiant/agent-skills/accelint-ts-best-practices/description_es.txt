Estándares integrales de codificación TypeScript/JavaScript que se centran en la seguridad de tipos, la programación defensiva y la corrección del código. Úselo cuando (1) escriba o revise código TS/JS, (2) corrija errores de tipo o evite any/enum/null, (3) implemente el flujo de control, la gestión de estado o el manejo de errores, (4) aplique un patrón de valor cero o inmutabilidad, (5) revise el código para antipatrones de TypeScript. Cubre convenciones de nomenclatura, diseño de funciones, valores de retorno, iteración limitada y validación de entradas. Para optimizar el rendimiento, utilice la habilidad accelint-ts-rendimiento. Para la documentación, utilice la habilidad accelint-ts-documentation.
