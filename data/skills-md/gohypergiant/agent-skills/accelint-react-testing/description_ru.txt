Используйте при написании, проверке или рефакторинге тестов компонентов React с помощью библиотеки тестирования. Загружайтесь, когда видите файлы render(), screen, fireEvent, userEvent, waitFor или *.test.tsx. Охватывает приоритет запроса (getByRole > getByLabelText > getByText), шаблоны тестирования, ориентированные на пользователя, асинхронные утилиты, пользовательскую визуализацию с поставщиками и утверждения, ориентированные на доступность. Ключевые слова включают RTL, библиотеку тестирования, экран, getByRole, findBy, queryBy, userEvent, waitFor, toBeInTheDocument, библиотеку тестирования/реагировать, библиотеку тестирования/пользовательское событие, jest-dom.
