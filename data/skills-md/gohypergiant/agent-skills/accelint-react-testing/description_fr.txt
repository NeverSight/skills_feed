À utiliser lors de l'écriture, de la révision ou de la refactorisation de tests de composants React avec la bibliothèque de tests. Chargez lorsque vous voyez des fichiers render(), screen, fireEvent, userEvent, waitFor ou *.test.tsx. Couvre la priorité des requêtes (getByRole > getByLabelText > getByText), les modèles de test centrés sur l'utilisateur, les utilitaires asynchrones, les rendus personnalisés avec les fournisseurs et les assertions axées sur l'accessibilité. Les mots clés incluent RTL, Testing Library, screen, getByRole, findBy, queryBy, userEvent, waitFor, toBeInTheDocument, testing-library/react, testing-library/user-event, jest-dom.
