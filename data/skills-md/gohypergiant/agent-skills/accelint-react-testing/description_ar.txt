يُستخدم عند كتابة اختبارات مكونات React أو مراجعتها أو إعادة هيكلتها باستخدام مكتبة الاختبار. قم بالتحميل عندما ترى ملفات render() أو screen أو fireEvent أو userEvent أو waitFor أو *.test.tsx. يغطي أولوية الاستعلام (getByRole > getByLabelText > getByText)، وأنماط الاختبار التي تركز على المستخدم، والأدوات المساعدة غير المتزامنة، والعروض المخصصة مع مقدمي الخدمة، وتأكيدات إمكانية الوصول أولاً. تشمل الكلمات الرئيسية RTL، ومكتبة الاختبار، والشاشة، وgetByRole، وfindBy، وqueryBy، وuserEvent، وwaitFor، وtoBeInTheDocument، وtest-library/react، وtest-library/user-event، وjest-dom.
