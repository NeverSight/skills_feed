تحسين أداء Next.js وأفضل الممارسات. يُستخدم عند كتابة كود Next.js (جهاز توجيه التطبيقات أو جهاز توجيه الصفحات)؛ وتنفيذ مكونات الخادم أو إجراءات الخادم أو مسارات واجهة برمجة التطبيقات؛ تحسين تسلسل RSC أو جلب البيانات أو العرض من جانب الخادم؛ ومراجعة التعليمات البرمجية Next.js لمشاكل الأداء؛ إصلاح المصادقة في إجراءات الخادم؛ أو تنفيذ حدود التشويق، أو جلب البيانات المتوازية، أو طلب إلغاء البيانات المكررة.
