Optimisation des performances Next.js et bonnes pratiques. À utiliser lors de l'écriture du code Next.js (App Router ou Pages Router) ; implémenter des composants serveur, des actions serveur ou des routes API ; optimiser la sérialisation RSC, la récupération de données ou le rendu côté serveur ; examiner le code Next.js pour détecter les problèmes de performances ; correction de l'authentification dans les actions du serveur ; ou la mise en œuvre de limites de suspension, de récupération de données parallèle ou de demande de déduplication.
