Da utilizzare durante la scrittura, la revisione o il refactoring dei test vitest. Carica quando vedi file *.test.ts o *.spec.ts, blocchi descrittivi nidificati, asserzioni vaghe (toBeTruthy), test eccessivi, test lenti o esigenze di test asincroni. Copre il modello AAA, i test parametrizzati, la gerarchia dei doppi test, la pulizia simulata, l'isolamento dei test e l'ottimizzazione delle prestazioni. Le parole chiave includono vitest, testing, TDD, assertions, mocks, stubs, fakes, spies, beforeEach, description, it, wait, vi.mock, it.each.
