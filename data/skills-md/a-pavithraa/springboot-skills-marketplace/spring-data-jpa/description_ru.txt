Реализуйте репозитории, сущности и запросы Spring Data JPA, следуя современным передовым практикам. Используйте для создания репозиториев (только для совокупных корней), написания запросов (@Query, проекций DTO), пользовательских репозиториев (Criteria API, массовые операции), служб запросов CQRS, отношений сущностей и оптимизации производительности. Охватывает шаблоны от простых репозиториев до расширенных CQRS с подробным руководством по борьбе с шаблонами.
