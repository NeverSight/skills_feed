Migrieren Sie Spring Boot-Projekte auf Version 4.0 mit Java 25, einschließlich Spring Modulith 2.0- und Testcontainers 2.x-Upgrades. Wird verwendet, wenn der Benutzer ein Upgrade von Spring Boot, eine Migration auf Java 25, ein Update von Abhängigkeiten auf Spring Boot 4 anfordert, die Jackson 3-Migration erwähnt, nach Starter-Umbenennungen (web→webmvc, aop→aspectj) fragt, Testanmerkungen korrigiert (@MockBean→@MockitoBean) oder Hilfe bei der Kompatibilität mit Spring Modulith 2.0 oder Testcontainers 2.x benötigt. Analysiert die Codebasis auf Migrationsprobleme und führt Sie durch Änderungen mit spezifischen Dateiverweisen.
