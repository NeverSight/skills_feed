Spring Boot プロジェクトを Java 25 を使用したバージョン 4.0 (Spring Modulith 2.0 および Testcontainers 2.x アップグレードを含む) に移行します。ユーザーが Spring Boot のアップグレード、Java 25 への移行、Spring Boot 4 への依存関係の更新を要求する場合、Jackson 3 への移行について言及する場合、スターターの名前変更 (web→webmvc、aop→aspectj) について尋ねる場合、テスト アノテーションの修正 (@MockBean→@MockitoBean) について尋ねる場合、または Spring Modulith 2.0 または Testcontainers 2.x の互換性に関するサポートが必要な場合に使用します。移行の問題についてコードベースを分析し、特定のファイル参照を使用して変更をガイドします。
