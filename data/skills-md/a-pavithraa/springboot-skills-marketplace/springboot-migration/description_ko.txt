Spring Modulith 2.0 및 Testcontainers 2.x 업그레이드를 포함하여 Java 25를 사용하여 Spring Boot 프로젝트를 버전 4.0으로 마이그레이션합니다. 사용자가 Spring Boot 업그레이드, Java 25로 마이그레이션, Spring Boot 4로 종속성 업데이트, Jackson 3 마이그레이션 언급, 시작 이름 변경(web→webmvc, aop→aspectj), 테스트 주석 수정(@MockBean→@MockitoBean)에 대해 요청하거나 Spring Modulith 2.0 또는 Testcontainers 2.x 호환성에 대한 도움이 필요할 때 사용합니다. 마이그레이션 문제에 대한 코드베이스를 분석하고 특정 파일 참조를 통해 변경 사항을 안내합니다.
