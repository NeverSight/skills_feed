ترحيل مشاريع Spring Boot إلى الإصدار 4.0 مع Java 25، بما في ذلك ترقيات Spring Modulith 2.0 وTestcontainers 2.x. يُستخدم عندما يطلب المستخدم ترقية Spring Boot، أو الترحيل إلى Java 25، أو تحديث التبعيات إلى Spring Boot 4، أو ذكر ترحيل Jackson 3، أو السؤال عن إعادة تسمية المبتدئين (web→webmvc، aop→aspectj)، أو إصلاح التعليقات التوضيحية للاختبار (@MockBean→@MockitoBean)، أو يحتاج إلى مساعدة في توافق Spring Modulith 2.0 أو Testcontainers 2.x. يحلل قاعدة التعليمات البرمجية لمشكلات الترحيل ويوجه خلال التغييرات باستخدام مراجع ملفات محددة.
