Migrazione dei progetti Spring Boot alla versione 4.0 con Java 25, inclusi gli aggiornamenti Spring Modulith 2.0 e Testcontainers 2.x. Da utilizzare quando l'utente richiede l'aggiornamento di Spring Boot, la migrazione a Java 25, l'aggiornamento delle dipendenze a Spring Boot 4, menziona la migrazione di Jackson 3, chiede informazioni sulla ridenominazione dello starter (web→webmvc, aop→aspectj), corregge le annotazioni di test (@MockBean→@MockitoBean) o ha bisogno di aiuto con la compatibilità di Spring Modulith 2.0 o Testcontainers 2.x. Analizza la codebase per individuare eventuali problemi di migrazione e guida attraverso le modifiche con riferimenti a file specifici.
