Перенесите проекты Spring Boot на версию 4.0 с Java 25, включая обновления Spring Modulith 2.0 и Testcontainers 2.x. Используйте, когда пользователь запрашивает обновление Spring Boot, переход на Java 25, обновление зависимостей до Spring Boot 4, упоминает миграцию Jackson 3, спрашивает о стартовых переименованиях (web→webmvc, aop→aspectj), исправлении тестовых аннотаций (@MockBean→@MockitoBean) или нуждается в помощи по совместимости Spring Modulith 2.0 или Testcontainers 2.x. Анализирует кодовую базу на предмет проблем миграции и направляет изменения с помощью конкретных ссылок на файлы.
