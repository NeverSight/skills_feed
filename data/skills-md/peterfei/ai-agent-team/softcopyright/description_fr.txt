Outil de génération de matériel d'application d'enregistrement de droits d'auteur de logiciel intelligent. Analyse automatiquement le code source du projet pour générer des manuels de logiciels et des documents de code source qui répondent aux exigences des applications de droits d'auteur sur les logiciels. Prend en charge la recherche par mot-clé, l'analyse intelligente du code source, la sortie formatée et l'exportation PDF.
