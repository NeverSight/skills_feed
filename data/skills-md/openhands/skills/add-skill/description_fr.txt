Ajoutez une compétence externe d'un référentiel GitHub à l'espace de travail actuel. À utiliser lorsque les utilisateurs souhaitent importer, installer ou ajouter une compétence à partir d'une URL GitHub (par exemple, `/add-skill https://github.com/OpenHands/skills/tree/main/skills/codereview` ou "ajouter la compétence de révision de code depuis https://github.com/OpenHands/skills/"). Gère la récupération des fichiers de compétences et leur placement dans .openhands/skills/.
