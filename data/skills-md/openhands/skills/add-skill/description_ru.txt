Добавьте внешний навык из репозитория GitHub в текущую рабочую область. Используйте, когда пользователи хотят импортировать, установить или добавить навык из URL-адреса GitHub (например, `/add-skill https://github.com/OpenHands/skills/tree/main/skills/codereview` или «добавить навык codereview из https://github.com/OpenHands/skills/»). Обрабатывает получение файлов навыков и размещение их в .openhands/skills/.
