Fügen Sie einen externen Skill aus einem GitHub-Repository zum aktuellen Arbeitsbereich hinzu. Verwenden Sie diese Option, wenn Benutzer einen Skill von einer GitHub-URL importieren, installieren oder hinzufügen möchten (z. B. „/add-skill https://github.com/OpenHands/skills/tree/main/skills/codereview“ oder „den Codereview-Skill von https://github.com/OpenHands/skills/ hinzufügen“). Behandelt das Abrufen der Skill-Dateien und deren Platzierung in .openhands/skills/.
