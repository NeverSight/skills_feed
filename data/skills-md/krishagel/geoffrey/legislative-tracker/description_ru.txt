Отслеживайте законодательство штата Вашингтон в области образования для K-12, используя данные комитетов через SOAP API. Получает ВСЕ счета от комитетов по образованию — без фильтрации по ключевым словам, которая может пропустить счета. Используйте, когда: проверка законодательного органа, отслеживание законопроектов, обновлений законодательства, утренний брифинг законодательного раздела.
