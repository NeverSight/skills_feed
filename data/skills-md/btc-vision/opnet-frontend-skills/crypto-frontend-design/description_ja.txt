暗号、Web3、DeFi、エアドロップ、ダッシュボード、およびトークン プロジェクト フロントエンド用の CSS ファースト ディレクティブ。スタイルシート、レイアウト、アニメーション、テーマ、レスポンシブ HTML を生成します。ランディング ページ、ダッシュボード、エアドロップ クレーム、ステーキング UI、DEX フロントエンド、ブリッジ UI、NFT ギャラリー、トークンの起動、または任意の暗号 Web インターフェイスでトリガーされます。マトリックス スタイル、サイバーパンク、ネオン光彩、ハッカー ターミナル、またはダーク フィンテック ダッシュボードの美学でもトリガーされます。マルチテーマ CSS、モバイル/タブレット/デスクトップのサポート、絵文字なし。 OPNet フロントエンドの場合は、opnet 開発スキル (TypeScript、React ロジック、ウォレット、プロバイダーを所有する) と併用します。このスキルはビジュアルを所有します。 opnet-development はロジックを所有しています。コードを生成する前に、常にユーザー コンテキスト (画像、テーマ、ブランド カラー、視聴者) を収集します。
