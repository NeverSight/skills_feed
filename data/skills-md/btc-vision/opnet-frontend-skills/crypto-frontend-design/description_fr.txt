Directive CSS-first pour les frontends de projets crypto, web3, DeFi, airdrop, tableau de bord et jetons. Produit des feuilles de style, une mise en page, une animation, des thèmes et du HTML réactif. Déclencheurs sur les pages de destination, les tableaux de bord, les réclamations de largage aérien, les interfaces utilisateur de jalonnement, les interfaces DEX, les interfaces utilisateur de pont, les galeries NFT, les lancements de jetons ou toute interface Web cryptographique. Se déclenche également sur l'esthétique du tableau de bord de style matriciel, cyberpunk, néon, pirate informatique ou sombre fintech. CSS multi-thème, support mobile/tablette/ordinateur de bureau, zéro emoji. Pour les interfaces OPNet, utilisez avec les compétences de développement Opnet (qui possèdent TypeScript, la logique React, le portefeuille, les fournisseurs). Cette compétence possède des visuels ; opnet-development possède une logique. Rassemble toujours le contexte utilisateur (images, thème, couleurs de la marque, audience) avant de générer du code.
