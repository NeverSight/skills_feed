Primera directiva CSS para interfaces de proyectos criptográficos, web3, DeFi, airdrop, paneles y tokens. Produce hojas de estilo, diseños, animaciones, temas y HTML responsivo. Activadores en páginas de destino, paneles, reclamos de lanzamiento aéreo, interfaces de usuario de participación, interfaces de usuario de DEX, interfaces de usuario de puente, galerías de NFT, lanzamientos de tokens o cualquier interfaz web criptográfica. También se activa con la estética de estilo matricial, cyberpunk, brillo de neón, terminal de piratas informáticos o tablero oscuro de tecnología financiera. CSS multitema, soporte para dispositivos móviles/tabletas/escritorio, cero emojis. Para interfaces OPNet, utilícelo junto con la habilidad de desarrollo de opnet (que posee TypeScript, lógica React, billetera y proveedores). Esta habilidad posee imágenes; opnet-development posee lógica. Siempre recopila el contexto del usuario (imágenes, tema, colores de marca, audiencia) antes de generar código.
