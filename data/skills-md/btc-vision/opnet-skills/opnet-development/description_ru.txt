Используйте OPNet — уровень консенсуса Bitcoin L1 для не требующих доверия смарт-контрактов. Используйте при создании смарт-контрактов AssemblyScript, библиотек TypeScript, интерфейсов React или плагинов Node для OPNet. Триггеры для разработки смарт-контрактов Биткойн, токенов OP20, NFT OP721, контрактов WebAssembly, шаблонов проверки-не-хранения, эпохального майнинга и вопросов архитектуры OPNet.
