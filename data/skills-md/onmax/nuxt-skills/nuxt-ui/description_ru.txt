Используйте при создании стилизованного пользовательского интерфейса с компонентами @nuxt/ui v4 (кнопка, модальное окно, форма, таблица и т. д.) — предоставляет готовые к использованию компоненты с темами Tailwind Variants. Используйте навыки vue для необработанных шаблонов компонентов и reka-ui для обезглавленных примитивов.
