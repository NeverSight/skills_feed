Boîte à outils complète pour la création vidéo programmatique avec Remotion + React. Couvre les animations, le timing, le rendu (CLI/Node.js/Lambda/Cloud Run), les légendes, la 3D, les graphiques, les effets de texte, les transitions et la gestion des médias. À utiliser lors de l'écriture de code Remotion, de la création de pipelines de génération vidéo ou de la création de modèles vidéo basés sur les données.
