Управляйте браузером Chrome программно с помощью chrome-devtools-mcp. Используйте, когда пользователь просит автоматизировать Chrome, отладить веб-страницы, сделать снимки экрана, оценить JavaScript, проверить сетевые запросы или взаимодействовать с DevTools браузера. Также используйте, когда вас спрашивают об автоматизации браузера, очистке веб-страниц или тестировании веб-сайтов.
