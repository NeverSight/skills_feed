التحكم في متصفح Chrome برمجيًا باستخدام chrome-devtools-mcp. يُستخدم عندما يطلب المستخدم تشغيل Chrome تلقائيًا، أو تصحيح أخطاء صفحات الويب، أو التقاط لقطات شاشة، أو تقييم JavaScript، أو فحص طلبات الشبكة، أو التفاعل مع DevTools للمتصفح. استخدمه أيضًا عند سؤالك عن أتمتة المتصفح أو تجريف الويب أو اختبار مواقع الويب.
