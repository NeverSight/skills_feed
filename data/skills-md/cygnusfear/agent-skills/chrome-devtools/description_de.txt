Steuern Sie den Chrome-Browser programmgesteuert mit chrome-devtools-mcp. Verwenden Sie es, wenn der Benutzer Chrome automatisieren, Webseiten debuggen, Screenshots erstellen, JavaScript auswerten, Netzwerkanfragen überprüfen oder mit den DevTools des Browsers interagieren möchte. Verwenden Sie es auch, wenn Sie nach Browser-Automatisierung, Web Scraping oder dem Testen von Websites gefragt werden.
