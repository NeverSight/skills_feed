Controle el navegador Chrome mediante programación usando chrome-devtools-mcp. Úselo cuando el usuario solicite automatizar Chrome, depurar páginas web, tomar capturas de pantalla, evaluar JavaScript, inspeccionar solicitudes de red o interactuar con DevTools del navegador. Utilícelo también cuando se le pregunte sobre la automatización del navegador, el web scraping o la prueba de sitios web.
