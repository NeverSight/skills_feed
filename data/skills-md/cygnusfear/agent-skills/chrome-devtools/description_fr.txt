Contrôlez le navigateur Chrome par programmation à l'aide de chrome-devtools-mcp. À utiliser lorsque l'utilisateur demande à automatiser Chrome, à déboguer des pages Web, à prendre des captures d'écran, à évaluer JavaScript, à inspecter les requêtes réseau ou à interagir avec les DevTools du navigateur. À utiliser également lorsque vous êtes interrogé sur l'automatisation du navigateur, le web scraping ou le test de sites Web.
