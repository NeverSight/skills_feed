Controlla il browser Chrome in modo programmatico utilizzando chrome-devtools-mcp. Da utilizzare quando l'utente chiede di automatizzare Chrome, eseguire il debug di pagine Web, acquisire screenshot, valutare JavaScript, ispezionare le richieste di rete o interagire con gli strumenti di sviluppo del browser. Utilizzare anche quando viene chiesto informazioni sull'automazione del browser, sul web scraping o sul test dei siti Web.
