Diese Fähigkeit sollte verwendet werden, wenn während der Codeimplementierung technische Entscheidungen getroffen werden. Der Archivar erzwingt die Entscheidungsdokumentation als Standardpraxis und stellt sicher, dass jede technische Entscheidung eine Begründung enthält und in die Architecture Decision Records (ADRs) integriert ist. Verwenden Sie es, wenn Sie Code schreiben, bei dem Sie zwischen Alternativen wählen, Technologien auswählen, Architekturen entwerfen oder Kompromisse eingehen müssen.
