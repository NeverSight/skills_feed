Esta habilidad debe usarse cuando el usuario solicita "usar el oráculo" o "preguntar al oráculo" para investigaciones, análisis o preguntas arquitectónicas profundas. Oracle se destaca en la investigación de múltiples fuentes que combina la exploración de la base de código y las búsquedas web, para luego sintetizar los hallazgos en respuestas procesables. Úselo para preguntas complejas que requieren investigación en múltiples fuentes, análisis arquitectónico, planes de refactorización, depuración de misterios y revisiones de código.
