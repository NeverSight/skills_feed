Этот навык следует использовать, когда пользователь просит «использовать оракула» или «спросить оракула» для глубокого исследования, анализа или вопросов архитектуры. Oracle превосходно справляется с исследованиями из нескольких источников, сочетая исследование кодовой базы и веб-поиск, а затем синтезируя результаты в практические ответы. Используйте для решения сложных вопросов, требующих изучения нескольких источников, архитектурного анализа, планов рефакторинга, загадок отладки и проверок кода.
