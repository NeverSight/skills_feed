À utiliser lors de l'examen du code Rust pour en vérifier la qualité, lors de l'écriture d'un nouveau code Rust qui doit suivre des modèles professionnels, ou lorsque l'utilisateur demande à juger, auditer ou améliorer le code Rust par rapport aux meilleures pratiques. Couvre la conception de types, les signatures de fonctions, l'architecture des traits, la gestion des erreurs, la visibilité, les macros, les tests et les modèles de performances.
