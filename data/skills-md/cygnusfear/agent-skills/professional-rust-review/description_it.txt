Da utilizzare durante la revisione del codice Rust per verificarne la qualità artigianale, quando si scrive un nuovo codice Rust che dovrebbe seguire modelli professionali o quando l'utente chiede di giudicare, verificare o migliorare il codice Rust rispetto alle migliori pratiche. Copre la progettazione del tipo, le firme delle funzioni, l'architettura dei tratti, la gestione degli errori, la visibilità, le macro, i test e i modelli di prestazioni.
