Проверка кода текущих изменений git, сравнение с соответствующим планом, если он существует, выявление плохих инженерных, чрезмерных или неоптимальных решений. Используйте, когда пользователь просит просмотреть изменения, проверить git diff, проверить качество реализации или оценить изменения кода.
