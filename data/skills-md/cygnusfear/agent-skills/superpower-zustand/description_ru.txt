ОБЯЗАТЕЛЬНО для создания магазинов Zustand. Этот навык необходим, когда пользователи запрашивают управление состоянием, создание магазинов или упоминают Zustand. НЕ создавайте магазины Zustand без этого навыка — все магазины должны использовать необходимый шаблон StoreBuilder с промежуточным программным обеспечением immer и разделением заводских шаблонов.
