Diese Fähigkeit sollte verwendet werden, wenn benutzerdefinierte Agenten für Claude Code erstellt, spezielle KI-Assistenten konfiguriert werden oder wenn der Benutzer Fragen zur Agentenerstellung, Agentenkonfiguration oder zum Delegieren von Aufgaben an Mitarbeiter stellt.
