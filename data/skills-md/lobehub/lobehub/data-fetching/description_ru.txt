Руководство по архитектуре получения данных с использованием уровня обслуживания + магазина Zustand + SWR. Используйте при реализации выборки данных, создании сервисов, работе с перехватчиками хранилища или переходе с useEffect. Запускается при загрузке данных, вызовах API, создании службы или задачах извлечения данных из хранилища.
