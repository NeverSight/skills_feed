Guide pour l’ajout d’une nouvelle documentation de fournisseur d’IA. À utiliser lors de l'ajout de documentation pour un nouveau fournisseur d'IA (comme OpenAI, Anthropic, etc.), y compris les documents d'utilisation, les variables d'environnement, la configuration Docker et les ressources d'image.
