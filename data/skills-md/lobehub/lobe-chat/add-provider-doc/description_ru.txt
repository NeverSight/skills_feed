Руководство по добавлению документации нового поставщика ИИ. Используйте при добавлении документации для нового поставщика ИИ (например, OpenAI, Anthropic и т. д.), включая документы об использовании, переменные среды, конфигурацию Docker и ресурсы изображений. Запускает задачи документации поставщика.
