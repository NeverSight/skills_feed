Überprüfen Sie den Code zur Einhaltung des Government of Canada Official Languages ​​Act. Prüft auf hartcodierte Zeichenfolgen, Wörterbuchparität zwischen Englisch-/Französisch-Übersetzungsdateien, länderspezifisches Routing, Datums-/Zahlenformatierung und Übersetzungen von Barrierefreiheitsattributen. Verwenden Sie diese Option, wenn Sie Code auf zweisprachige Unterstützung, i18n-Konformität, Französisch/Englisch-Übersetzungsabdeckung oder OLA-Anforderungen überprüfen.
