13 个联赛的足球数据 — 积分榜、赛程表、比赛统计数据、预期进球、转会、球员资料。零配置，无 API 密钥。涵盖英超、西甲、德甲、意甲、法甲、美国职业足球大联盟、欧冠、世界杯、英冠、荷甲、英超、巴西甲级联赛、欧洲杯。

使用场合：用户询问足球排名、赛程、比赛统计数据、预期进球、阵容、球员价值、转会、伤病新闻、联赛表、每日赛程或球员档案。
不要在以下情况下使用：用户询问美式橄榄球 (NFL)、篮球 (NBA)、棒球或任何非足球运动。请勿用于实时/实时比分——赛后数据更新。不要对非英超联赛使用 get_season_leaders 或 get_missing_players （它们返回空）。不要将 get_event_xg 用于前 5 名之外的联赛（EPL、西甲、德甲、意甲、法甲）。
