13 個聯賽的足球數據 — 積分榜、賽程表、比賽統計數據、預期進球、轉會、球員資料。零配置，無 API 金鑰。涵蓋英超、西甲、德甲、義甲、法甲、美國職業足球大聯盟、歐冠、世界盃、英冠、荷甲、英超、巴西甲級聯賽、歐洲盃。

使用場合：用戶詢問足球排名、賽程、比賽統計數據、預期得分、陣容、球員價值、轉會、傷病新聞、聯賽表、每日賽程或球員檔案。
不要在以下情況下使用：使用者詢問美式橄欖球 (NFL)、籃球 (NBA)、棒球或任何非足球運動。請勿用於即時/即時比分－賽後數據更新。不要對非英超聯賽使用 get_season_leaders 或 get_missing_players （它們返回空）。不要將 get_event_xg 用於前 5 名之外的聯賽（EPL、西甲、德甲、意甲、法甲）。
