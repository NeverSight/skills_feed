Données NBA via les points de terminaison publics ESPN : scores, classements, effectifs, calendriers, résumés de matchs, leaders statistiques et actualités. Zéro configuration, pas de clés API.

À utiliser lorsque : l'utilisateur pose des questions sur les scores de la NBA, les classements, les listes d'équipes, les horaires, les statistiques de jeu, les scores des box ou les actualités de la NBA.
Ne pas utiliser lorsque : l'utilisateur pose des questions sur la WNBA (utilisez les données wnba), le basket-ball dans d'autres ligues, le basket-ball universitaire ou d'autres sports. Ne pas utiliser pour la lecture en direct : mises à jour des données après la lecture.
