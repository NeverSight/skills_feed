Données NFL via les points de terminaison publics ESPN : scores, classements, effectifs, calendriers, résumés de matchs, dirigeants et actualités. Zéro configuration, pas de clés API.

À utiliser lorsque : l'utilisateur pose des questions sur les scores de la NFL, les classements, les listes d'équipes, les horaires, les statistiques de jeu, les scores des box ou les actualités de la NFL.
Ne pas utiliser lorsque : l'utilisateur pose des questions sur le football (utiliser les données sur le football), le football universitaire ou d'autres sports. Ne pas utiliser pour la lecture en direct : mises à jour des données après la lecture.
