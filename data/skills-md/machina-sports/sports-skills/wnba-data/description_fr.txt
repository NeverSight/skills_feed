Données WNBA via les points de terminaison publics ESPN : scores, classements, listes, calendriers, résumés de matchs, leaders statistiques et actualités. Zéro configuration, pas de clés API.

À utiliser lorsque : l'utilisateur pose des questions sur les scores de la WNBA, les classements, les listes d'équipes, les horaires, les statistiques de jeu, les scores des box ou les actualités de la WNBA.
Ne pas utiliser lorsque : l'utilisateur pose des questions sur la NBA (utilisez les données nba), le basket-ball masculin, le basket-ball universitaire ou d'autres sports. Ne pas utiliser pour la lecture en direct : mises à jour des données après la lecture.
