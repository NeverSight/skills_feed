Marchés de prédiction sportive Polymarket : cotes en direct, prix, carnets de commandes, événements, séries et recherche de marché. Aucune authentification requise. Couvre la NFL, la NBA, la MLB, le football, le tennis, le cricket, le MMA et l'esport. Prend en charge la moneyline, les spreads, les totaux et les accessoires des joueurs.

À utiliser lorsque : l'utilisateur pose des questions sur les cotes des paris sportifs, les prévisions de marchés, les probabilités de gain, le sentiment du marché ou des questions sur "qui est favorisé pour gagner".
Ne pas utiliser lorsque : l'utilisateur pose des questions sur les résultats, les scores ou les statistiques réels du match – utilisez plutôt football-data ou fastf1. Ne pas utiliser pour les données de correspondance historiques. Ne l'utilisez pas pour les actualités, utilisez plutôt les actualités sportives. Ne confondez pas avec Kalshi – Polymarket se concentre sur les marchés de prédiction crypto-natifs avec une couverture sportive plus approfondie ; Kalshi est une bourse réglementée par les États-Unis avec une structure de marché différente.
