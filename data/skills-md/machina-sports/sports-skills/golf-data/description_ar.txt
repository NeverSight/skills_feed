بيانات جولة PGA، وLPGA، وDP World Tour عبر نقاط النهاية العامة لـ ESPN - لوحات المتصدرين للبطولات، وبطاقات الأداء، وجداول المواسم، وملفات تعريف/نظرات عامة للاعبي الجولف، والأخبار. تكوين صفر، لا توجد مفاتيح API.

يُستخدم عندما: يسأل المستخدم عن نتائج لعبة الجولف، أو لوحات المتصدرين للبطولات، أو بطاقات الأداء، أو جدول جولات PGA، أو ملفات تعريف لاعبي الجولف، أو إحصائيات موسم لاعبي الجولف، أو نتائج LPGA، أو أخبار الجولف.
لا تستخدم عندما: يسأل المستخدم عن الرياضات الأخرى.
