Теннисные данные ATP и WTA через общедоступные конечные точки ESPN — результаты турниров, календари сезонов, рейтинги игроков, профили игроков и новости. Нулевая конфигурация, никаких ключей API.

Используйте, когда: пользователь спрашивает о теннисных результатах, результатах матчей, турнирных сетках, рейтингах ATP/WTA, информации о теннисистах или теннисных новостях.
Не использовать, когда: пользователь спрашивает о других видах спорта. Не используйте для получения данных по точкам в реальном времени — результаты обновляются после каждого сета/матча.
